## 常见问题

**Q: 创建图表后我可以修改数据标签吗？**  
A: 可以，您可以访问图表的 `DataLabels` 集合，随时更新文本、位置或格式。

**Q: 如何为图表设置自定义图例文本？**  
A: 在创建图表对象后，使用 `chart.getLegend().setText("Your Legend Text")`。

**Q: 是否可以为工作簿中的所有图表应用自定义主题颜色方案？**  
A: 当然。加载或创建 `Theme` 对象，修改其配色方案，然后在添加图表之前将其分配给工作簿。

**Q: 导出图表为 SVG 或 PDF 是否需要许可证？**  
A: 试用版可用于评估，但生产级导出需要商业许可证。

**Q: Aspose.Cells for Java 支持哪些图表类型？**  
A: 超过 50 种图表类型，包括柱形图、折线图、饼图、气泡图、雷达图、股票图和透视图表等。
