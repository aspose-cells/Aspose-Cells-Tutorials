# 适用于 .NET 应用程序的 Excel 计算引擎

## 概述

Aspose.Cells for .NET 提供强大的计算引擎，允许开发人员在 Excel 电子表格中处理复杂的公式计算。这个强大的组件使您能够配置计算行为、实现自定义计算逻辑并优化大型工作簿的性能。

## 主要特点

- **自定义计算引擎**：针对专门的公式实现自己的计算逻辑
- **计算监控**：跟踪和控制公式计算以实现性能优化
- **递归计算选项**：配置如何处理循环引用
- **日期系统配置**：在 1900 和 1904 日期系统之间切换
- **循环引用检测**：识别并解决公式中的循环依赖关系
- **性能优化**：微调大型工作簿的计算设置
