# Excel 中的数字和显示格式

## 介绍

当我们想到 Excel 时，我们经常会想到充满数据的电子表格。但是，您是否曾尝试过让这些数据在视觉上更具吸引力或更容易理解？这就是数字格式发挥作用的地方。使用 Aspose.Cells for .NET，您可以自动执行格式化过程，这可以为您节省大量时间，尤其是在处理大型数据集时。让我们深入研究如何以编程方式使用这些格式，重点关注两个关键领域：内置数字格式和自定义显示格式。

## 使用内置数字格式

使用内置数字格式非常简单！假设您希望 Excel 工作表始终显示货币值。您可以使用 Aspose.Cells 自动应用货币或百分比等格式，而不必手动格式化每个单元格。此功能不仅可以增强演示效果，还可以消除人为错误。想象一下，在发送数据之前，知道所有数据都已正确格式化，您会感到多么轻松。这就像拥有一个专门负责电子表格美观的私人助理！有关更多信息，请查看我们的教程[以编程方式使用 Excel 中的内置数字格式](./using-built-in-number-formats/).

## 自定义显示格式

如果您想为 Excel 表格添加个性化元素，自定义显示格式是最佳选择。此功能允许您创建用户定义的数字格式以满足您的特定要求。您可能想知道它是如何工作的 - 可以这样想：就像画家为他们的杰作选择正确的颜色一样，您可以选择数据的呈现方式。无论是独特的日期格式还是显示百分比的专门方式，Aspose.Cells 都使您能够轻松做出这些选择。要了解如何利用这种功能，请不要错过我们关于[使用用户定义数字自定义显示格式](./customizing-display-formats-with-user-defined-numbers/).
