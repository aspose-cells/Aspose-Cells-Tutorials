## 安装说明  
1. 确保您已设置 .NET 开发环境，例如 Visual Studio。  
2. 通过 NuGet 包管理器安装 Aspose.Cells for .NET。在包管理器控制台中使用以下命令：  
   ```
   Install-Package Aspose.Cells
   ```
3. 在您的项目中添加对 Aspose.Cells 命名空间的引用：  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Drawing; 
   ```
4. 准备源 Excel 文件并更新 `sourceDir` 和 `outputDir` 提供的代码示例中的路径。  
5. 将提供的代码片段复制并粘贴到您的项目中，以操作 Excel 文件中形状的 Z 顺序。  
6. 构建并运行您的应用程序以查看输出 Excel 文件中反映的更改。
