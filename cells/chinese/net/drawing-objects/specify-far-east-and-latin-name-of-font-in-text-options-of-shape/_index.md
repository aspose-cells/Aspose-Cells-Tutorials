# 使用远东和拉丁字体创建Excel工作簿

这个例子展示了如何使用Aspose.Cells为 .NET创建Excel工作簿,添加一个文本框到一个工作表,并为该文本框内的文本设置远东和拉丁字体名称.该代码显示了初始化工作空间,操纵工作表元素,并将工作簿保存以针对各种语言要求的特定字体格式.

## 装备说明
1. 安装 .NET 的Aspose.Cells: 
   * 在视觉工作室项目中使用NuGet包管理器:
     ```
     Install-Package Aspose.Cells
     ```
   * 或从 文件下载[目标网站](https://www.aspose.com/downloads/cells/net)现在,我们要去.

2. 添加引用:确保您的项目引用Aspose.Cells图书馆.

3. 设置输出目录:更新`输出`编程中可包含输出路径的变量,其中将保存Excel文件.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
