# 使用远东和拉丁字体创建 Excel 工作簿

此示例演示如何使用 Aspose.Cells for .NET 创建 Excel 工作簿、向工作表添加文本框，并为该文本框中的文本设置远东和拉丁字体名称。代码展示了如何初始化工作区、操作工作表元素以及如何使用根据不同语言需求定制的特定字体格式保存工作簿。

## 安装说明
1. 安装 Aspose.Cells for .NET： 
   - 在 Visual Studio 项目中使用 NuGet 包管理器：
     ```
     Install-Package Aspose.Cells
     ```
   - 或者从 [Aspose 网站](https://www。aspose.com/downloads/cells/net).

2. 添加引用：确保您的项目引用 Aspose.Cells 库。

3. 设置输出目录：更新 `outputDir` 代码中的变量具有保存 Excel 文件的所需输出路径。

4. 运行示例：将提供的 C# 代码复制到您的项目中，编译并运行它。您应该会看到输出确认，并且将在指定目录中创建包含已配置文本框的 Excel 文件。
