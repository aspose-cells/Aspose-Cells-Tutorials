# 使用 Aspose.Cells 修改 Excel 中的图表数据标签形状
在本教程中，我们将探索如何使用 Aspose.Cells for .NET 修改 Excel 图表中数据标签的形状类型。图表中的数据标签有助于提供更佳的可视化效果，而更改其形状则可以提高可读性。本示例演示了如何访问现有的 Excel 图表，检索系列数据标签，并将其形状类型更改为气泡椭圆（楔形椭圆标注）。只需几行 C# 代码，即可高效提升 Excel 图表的视觉吸引力。
