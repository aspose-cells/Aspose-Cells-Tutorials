## 安装说明：  
要使用此示例，请按照以下步骤操作：  

1. 通过 NuGet 安装 Aspose.Cells for .NET：  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. 在您的 C# 项目中添加所需的命名空间：  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. 运行提供的代码来加载 ODS 文件，访问图表并将其字幕打印到控制台。
