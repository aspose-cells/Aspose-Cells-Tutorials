## 安装说明：  

要使用 Aspose.Cells for .NET，请按照以下步骤操作：  

1. 通过 NuGet 包管理器安装  
- 打开 Visual Studio。  
- 导航到工具 → NuGet 包管理器 → 管理解决方案的 NuGet 包。  
- 搜索 Aspose.Cells。  
- 单击“安装”将包添加到您的项目中。  

或者，使用 NuGet 包管理器控制台安装它：  

```powershell
Install-Package Aspose.Cells
```

2. 在 C# 代码中添加命名空间  
```csharp
using Aspose.Cells;
```
