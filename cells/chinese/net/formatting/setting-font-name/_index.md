# 使用 Aspose.Cells 在 Excel 中设置字体名称

在 Excel 文件中设置字体名称对于维护专业且一致的文档格式至关重要。使用 Aspose.Cells for .NET，您可以轻松创建 Excel 工作簿、添加工作表并修改单元格样式。本教程将指导您如何在 C# 中为特定单元格设置自定义字体名称。按照分步指南操作，轻松增强您的 Excel 文档格式。

## 安装说明  

要使用 Aspose.Cells for .NET，请按照以下步骤操作：  

1. 通过 NuGet 安装：  
   在 Visual Studio 中打开 NuGet 包管理器并运行：  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. 从 Aspose 网站下载：  
   - 获取最新版本 [Aspose.Cells 下载](https://releases。aspose.com/cells/net/).  

3. 添加对 Aspose.Cells 的引用：  
   - 在您的 .NET 项目中，添加对 `Aspose。Cells.dll`.  

4. 申请许可证（可选）：  
   - 如果使用许可版本，请按如下方式设置许可证文件：  
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Cells.lic");
   ```
