# تحويل المخطط إلى PDF في .NET

## مقدمة
هل ترغب في تحويل مخططات من جداول بيانات Excel إلى صيغة PDF باستخدام .NET؟ أنت في المكان المناسب! في هذا الدليل، سنستكشف تفاصيل استخدام Aspose.Cells لتحقيق ذلك. سواء كنت مبرمجًا محترفًا أو جديدًا، سيساعدك دليلنا خطوة بخطوة على إتمام العملية بسهولة.

## المتطلبات الأساسية
قبل أن نبدأ هذه الرحلة المستنيرة، هناك بعض المتطلبات الأساسية التي يجب عليك التحقق منها من قائمتك:
### 1. تم تثبيت .NET Framework أو .NET Core
تأكد من تثبيت .NET Framework أو .NET Core على جهازك. هذا الدليل مناسب لكلا البيئتين، فلا داعي للقلق إذا فضّلت إحداهما على الأخرى!
### 2. مكتبة Aspose.Cells
يحدث السحر بفضل مكتبة Aspose.Cells، التي يجب عليك تضمينها في مشروعك. يمكنك تنزيلها من [موقع Aspose](https://releases.aspose.com/cells/net/).
### 3. فهم أساسيات برمجة C#
إذا كان لديك فهم أساسي للغة C#، فهذا رائع! ستجد سهولة في فهم الأمثلة التي نقدمها. إذا كنت مبتدئًا، فلا تقلق كثيرًا؛ فنحن نجعل الأمور بسيطة ومباشرة.
### 4. إعداد Visual Studio
سواء كنت تستخدم Visual Studio أو أي بيئة تطوير متكاملة أخرى، تأكد من أن بيئة التطوير الخاصة بك مهيأة بالكامل لكتابة وتشغيل تطبيقات .NET.
## استيراد الحزم
لبدء عملية التحويل، عليك استيراد الحزم اللازمة إلى مشروعك. إليك كيفية القيام بذلك:
### افتح مشروعك
قم بتشغيل Visual Studio وافتح المشروع الذي تريد تنفيذ هذه الوظيفة فيه.
### تثبيت حزمة Aspose.Cells NuGet
يمكنك بسهولة إضافة مكتبة Aspose.Cells عبر مدير الحزم NuGet. إليك الطريقة:
- انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- حدد "إدارة حزم NuGet".
- ابحث عن "Aspose.Cells" واضغط على زر التثبيت.
سيضمن لك هذا توفر جميع الفصول والأساليب التي تحتاجها في متناول يدك!

```csharp
using System;
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Charts;
```

الآن، لنبدأ بشرح تحويل مخطط بياني إلى صيغة PDF باستخدام Aspose.Cells. سنشرح كل خطوة بشكل منهجي، لتعرف ما يحدث بالضبط.
## الخطوة 1: إعداد دليل المستندات الخاص بك
أولاً، عليك تحديد مسار حفظ مستند Excel. هنا، ستُوجِّه مكتبة Aspose.Cells للعثور على ملف .xls.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
```
هذا الخط يحدد `dataDir` المتغير إلى موقع ملف Excel الخاص بك. تأكد من استبدال `"Your Document Directory"` مع مسارك الفعلي.
## الخطوة 2: تحميل ملف Excel
بعد تحديد المجلد، حان وقت تحميل ملف Excel الذي يحتوي على المخططات البيانية. إليك كيفية القيام بذلك:
```csharp
// قم بتحميل ملف Excel الذي يحتوي على المخططات البيانية
Workbook workbook = new Workbook(dataDir + "Sample1.xls");
```
من خلال القيام بذلك، فإنك تقوم بإنشاء مثيل جديد من `Workbook` وأطلب منه تحميل ملف Excel النموذجي. تأكد من تطابق اسم الملف وامتداده مع ملفك الفعلي.
## الخطوة 3: الوصول إلى ورقة العمل الصحيحة
قد تحتوي ملفات Excel على عدة أوراق عمل، لذا عليك تحديد الورقة التي تريد العمل عليها. هنا، نصل إلى ورقة العمل الأولى:
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```
استخدام الفهرس `0` يقوم بجلب ورقة العمل الأولى. اضبط الفهرس إذا كان مخططك على ورقة أخرى.
## الخطوة 4: الوصول إلى الرسم البياني
الآن بعد أن حصلت على ورقة العمل، فلنأخذ الرسم البياني الذي تريد تحويله:
```csharp
// الوصول إلى الرسم البياني الأول داخل ورقة العمل
Chart chart = worksheet.Charts[0];
```
يصل هذا السطر إلى أول مخطط في ورقة العمل. إذا كان لديك عدة مخططات وترغب في تحويل مخطط آخر، فما عليك سوى زيادة الفهرس.
## الخطوة 5: تحويل الرسم البياني إلى PDF
بعد أن أصبحت مخططاتك جاهزة، حان وقت تحويلها إلى صيغة PDF. إليك الطريقة:
```csharp
// احفظ الرسم البياني بتنسيق PDF
chart.ToPdf(dataDir + "Output-Chart_out.pdf");
```
أمر التحقق هذا يُلزم Aspose.Cells بحفظ الرسم البياني كملف PDF في مسار الإخراج المُحدد. وهكذا! أصبح الرسم البياني الآن بصيغة PDF.
## الخطوة 6: حفظ الرسم البياني في مجرى الذاكرة
إذا كنت تفضل حفظ الرسم البياني ليس في ملف ولكن في مجرى ذاكرة (على سبيل المثال، إذا كنت تخطط لتنزيله بشكل ديناميكي)، فيمكنك القيام بذلك باستخدام الكود التالي:
```csharp
// احفظ الرسم البياني بتنسيق PDF في التدفق
MemoryStream ms = new MemoryStream();
chart.ToPdf(ms);
```
من خلال القيام بذلك، يمكنك حفظ الرسم البياني في `MemoryStream` بدلاً من إرسالها مباشرةً إلى ملف. يُعد هذا مفيدًا بشكل خاص لتطبيقات الويب التي تتطلب إنشاء ملفات ديناميكيًا.
