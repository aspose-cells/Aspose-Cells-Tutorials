# حساب الصيغ في Excel برمجيًا

## مقدمة
في عالمنا اليوم الذي يعتمد على البيانات، تُمكّن أتمتة المهام من توفير الوقت وتعزيز الكفاءة، خاصةً عند التعامل مع جداول البيانات. إذا سبق لك التعامل مع صيغ معقدة في Excel، فأنت تُدرك أهمية إتقانها. باستخدام Aspose.Cells لـ .NET، يُمكنك حساب الصيغ برمجيًا وإدارة ملفات Excel بسهولة. في هذا البرنامج التعليمي، سنشرح كل خطوة من خطوات إنشاء ملف Excel، وإضافة القيم والصيغ، ثم حسابها باستخدام لغة C#. هيا بنا!

## استيراد الحزم
قبل كتابة أي كود، تأكد من استيراد المساحات الأساسية اللازمة لـ Aspose.Cells في ملف C# الخاص بك:
```csharp
using System.IO;
using Aspose.Cells;
```
يتيح لك هذا الوصول إلى الوظائف التي توفرها مكتبة Aspose.Cells للتعامل مع ملفات Excel.
## الخطوة 1: تعيين دليل المستندات
ابدأ بتحديد المسار الذي تريد حفظ مستند Excel فيه. من الضروري التأكد من وجود هذا الدليل، أو إنشائه إذا لم يكن موجودًا.
```csharp
// المسار إلى دليل المستندات
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
في هذه الخطوة، ستتحقق من وجود الدليل. إذا لم يكن موجودًا، فأنت تُنشئه. تساعدك هذه الخطوة البسيطة على تجنب الأخطاء عند محاولة حفظ ملف Excel لاحقًا.
## الخطوة 2: إنشاء كائن مصنف
## إنشاء مصنف جديد
الآن بعد أن تم تعيين الدليل الخاص بك، فلنقم بإنشاء كائن مصنف يمثل ملف Excel الخاص بك:
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
هذا السطر يُنشئ ببساطة مصنفًا جديدًا في الذاكرة. تخيّل الأمر كما لو كنت تفتح ملف Excel فارغًا، حيث يمكنك البدء بإضافة البيانات والصيغ.
## الخطوة 3: إضافة ورقة عمل جديدة
## العمل مع أوراق العمل
في مصنفنا، نريد إضافة ورقة عمل جديدة لمعالجة بياناتنا. إليك كيفية القيام بذلك:
```csharp
// إضافة ورقة عمل جديدة إلى كائن Excel
int sheetIndex = workbook.Worksheets.Add();
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```
أولاً، أضف ورقة عمل جديدة، وستظهر لك تلقائيًا فهرسها. بعد ذلك، استرجاع ورقة العمل هذه من خلال فهرسها. الأمر أشبه بفتح علامة تبويب جديدة في مصنف Excel!
## الخطوة 4: إدراج القيم في الخلايا
## ملء البيانات
الآن بعد أن قمنا بإنشاء ورقة العمل الخاصة بنا، نحتاج إلى إضافة بعض البيانات إليها:
```csharp
// إضافة قيمة إلى الخلية "A1"
worksheet.Cells["A1"].PutValue(1);
// إضافة قيمة إلى الخلية "A2"
worksheet.Cells["A2"].PutValue(2);
// إضافة قيمة إلى الخلية "A3"
worksheet.Cells["A3"].PutValue(3);
```
في هذه الخطوة، تُدخل القيم في الخلايا الثلاث الأولى (A1، A2، A3) من ورقة العمل. هذه العملية تُشبه إدخال القيم مباشرةً في ورقة Excel. 
## الخطوة 5: إضافة صيغة
## تلخيص القيم
بعد إدخال القيم، حان الوقت لإضافة صيغة لحساب مجموع هذه الخلايا. إليك الطريقة:
```csharp
// إضافة صيغة SUM إلى الخلية "A4"
worksheet.Cells["A4"].Formula = "=SUM(A1:A3)";
```
يُضيف هذا السطر من التعليمات البرمجية صيغة SUM إلى الخلية A4، والتي ستجمع القيم من A1 إلى A3. يشبه الأمر كتابة صيغة في Excel، ولكن برمجيًا!
## الخطوة 6: احسب الصيغة
## إجراء الحساب
الآن حانت لحظة الحقيقة! علينا حساب نتائج الصيغ التي أدخلناها:
```csharp
// حساب نتائج الصيغ
workbook.CalculateFormula();
```
عن طريق الاتصال `CalculateFormula()`أنت تطلب من المصنف معالجة جميع الصيغ فيه. هذا يشبه الضغط على زر "Enter" بعد كتابة صيغة في خلية Excel.
## الخطوة 7: استرداد القيمة المحسوبة
## قراءة النتيجة
بمجرد حساب الصيغ، يمكننا استرجاع القيمة من A4:
```csharp
// احصل على القيمة المحسوبة للخلية
string value = worksheet.Cells["A4"].Value.ToString();
```
في هذه الخطوة، ستحصل على نتيجة صيغة الجمع. هذا سيعطيك المجموع ١ + ٢ + ٣، أي ٦!
## الخطوة 8: حفظ ملف Excel
## الكتابة على القرص
وأخيرًا، احفظ المصنف في الدليل المحدد، حتى تتمكن من الوصول إليه لاحقًا:
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "output.xls");
```
يحفظ هذا الكود ملف إكسل باسم "output.xls" في المجلد الذي حددته. يشبه الأمر النقر على "حفظ باسم" في إكسل واختيار مكان حفظ الملف.
