## إعداد Aspose.Cells لـ .NET
ابدأ بإضافة حزمة NuGet اللازمة إلى مشروعك، كما هو موضح أعلاه. بعد التثبيت، قم بتشغيل نسخة مصنف جديدة لبدء العمل مع ملفات Excel برمجيًا.

```csharp
using Aspose.Cells;

// إنشاء كائن مصنف يمثل ملف Excel.
Workbook workbook = new Workbook();
```

## دليل التنفيذ
### الميزة 1: تهيئة المصنف والوصول إلى ورقة العمل
**ملخص:** توضح هذه الميزة كيفية إنشاء مصنف جديد، والوصول إلى ورقة العمل الافتراضية الخاصة به، وتعيين عرض الأعمدة.

#### الخطوة 1: إنشاء مصنف جديد
```csharp
// إنشاء مصنف جديد
Workbook workbook = new Workbook();
```

#### الخطوة 2: الوصول إلى ورقة العمل الافتراضية
```csharp
// احصل على ورقة العمل الأولى (الافتراضية) في المصنف
Worksheet worksheet = workbook.Worksheets[0];
```

#### الخطوة 3: تعيين عرض الأعمدة
```csharp
// تعيين عرض الأعمدة للأعمدة A وB وC
worksheet.Cells.SetColumnWidth(0, 24);
worksheet.Cells.SetColumnWidth(1, 24);
worksheet.Cells.SetColumnWidth(2, 24);
```

### الميزة 2: إدخال البيانات في الخلايا
**ملخص:** إدخال البيانات في خلايا محددة باستخدام هذه الميزة.

#### الخطوة 1: الوصول إلى ورقة العمل والخلايا
```csharp
// إنشاء مصنف عمل جديد والوصول إلى ورقة العمل الأولى
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
Cells cells = worksheet.Cells;
```

#### الخطوة 2: إدخال البيانات في الخلايا
```csharp
// إدخال الرؤوس والبيانات في خلايا محددة
cells["A1"].PutValue("KPIs");
cells["B1"].PutValue("UA Contract Size Group 4");

// مثال على إدخال القيم الرقمية والنسبية
cells["B2"].PutValue(19551794);
cells["B3"].PutValue(11.8070745566204);
```

### الميزة 3: إضافة أيقونات شرطية إلى الخلايا
**ملخص:** قم بتعزيز تقاريرك عن طريق إضافة إشارات مرئية من خلال الرموز الشرطية.

#### الخطوة 1: تحضير بيانات الصورة
```csharp
// احصل على بيانات صورة الرمز لأنواع مختلفة باستخدام واجهة برمجة تطبيقات Aspose.Cells
byte[] imagedata = ConditionalFormattingIcon.GetIconImageData(IconSetType.TrafficLights31, 0);
MemoryStream stream = new MemoryStream(imagedata);
```

#### الخطوة 2: إدراج الرموز في الخلايا
```csharp
// إضافة أيقونات إلى خلايا محددة في ورقة العمل
worksheet.Pictures.Add(1, 1, stream); // أيقونة إشارة المرور إلى الخلية B2
```

### الميزة 4: حفظ المصنف
**ملخص:** وأخيرًا، قم بحفظ المصنف الخاص بك في الدليل المحدد.

#### الخطوة 1: تحديد دليل الإخراج وحفظه
```csharp
// عنصر نائب لمسار دليل الإخراج
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// حفظ ملف Excel
countbook.Save(outputDir + "outputAddConditionalIconsSet.xlsx");
```
