# قراءة جدول الأرقام برمجيًا في .NET

## مقدمة
في عالمنا الرقمي اليوم، تُعَد إدارة البيانات مهارة حيوية، وتحتل جداول البيانات الصدارة في تنظيم البيانات. ولكن ماذا لو كنت بحاجة إلى العمل باستخدام جدول بيانات Numbers — تلك الملفات التي تم إنشاؤها بواسطة تطبيق Numbers من Apple — باستخدام .NET؟ لا تقلق؛ فأنت لست وحدك! في هذا البرنامج التعليمي، سنشرح عملية قراءة جدول بيانات Numbers برمجيًا باستخدام Aspose.Cells for .NET. ستتعلم كيفية تحميل ملف Numbers وتحويله إلى PDF.
## المتطلبات الأساسية
قبل أن نبدأ، هناك بعض الأشياء التي تحتاج إلى وضعها في مكانها:
1. Aspose.Cells لـ .NET: تأكد من تثبيت مكتبة Aspose.Cells. يمكنك تنزيلها[هنا](https://releases.aspose.com/cells/net/).
2. Visual Studio: يوصى بتثبيت Visual Studio (أو أي IDE آخر متوافق مع .NET) على جهازك.
3. المعرفة الأساسية بلغة C#: إن القليل من المعرفة ببرمجة C# سوف يساعدك على المتابعة بسلاسة.
4. دليل المستندات الخاص بك: ستحتاج إلى دليل يتم تخزين ملف Numbers الخاص بك فيه، بالإضافة إلى موقع لحفظ ملف PDF المُحوّل.
بمجرد تغطية هذه المتطلبات الأساسية، ستكون جاهزًا للبدء!
## استيراد الحزم
للبدء، نحتاج إلى استيراد الحزم اللازمة إلى مشروع C# الخاص بنا. هذه خطوة بالغة الأهمية لأنها تسمح لنا بالاستفادة من الوظائف التي توفرها مكتبة Aspose.Cells.
1. افتح مشروع C# الخاص بك في Visual Studio.
2. أضف مرجعًا إلى مكتبة Aspose.Cells:
   - إذا كنت تستخدم NuGet، فما عليك سوى تشغيل الأمر التالي في وحدة التحكم في إدارة الحزم:
```
 Install-Package Aspose.Cells
 ```
3. استيراد المساحات الأساسية اللازمة في الكود الخاص بك:
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```
الآن بعد أن قمنا باستيراد الحزم اللازمة، دعنا ننتقل إلى الدليل خطوة بخطوة لقراءة جدول بيانات Numbers.
## الخطوة 1: تحديد أدلة المصدر والإخراج
في هذه الخطوة، سنقوم بإعداد الدلائل التي يوجد بها ملف Numbers المصدر والمكان الذي تريد حفظ ملف PDF الناتج فيه.
```csharp
// دليل المصدر
string sourceDir = "Your Document Directory"; // قم بتحديث هذا إلى الدليل الفعلي الخاص بك
// دليل الإخراج
string outputDir = "Your Document Directory"; // قم بتحديث هذا إلى الدليل الفعلي الخاص بك
```
 هنا، نقوم بتعريف متغيرين للسلسلة،`sourceDir` و`outputDir` لتحديد مواقع ملفات الإدخال والإخراج. تأكد من استبدال`"Your Document Directory"` مع المسارات الفعلية على نظامك.
## الخطوة 2: إعداد خيارات التحميل لتنسيق الأرقام
بعد ذلك، سنحدد خيارات التحميل لقراءة جدول بيانات Numbers. هذه الخطوة ضرورية لأنها تخبر Aspose بكيفية تفسير ملف Numbers.
```csharp
// حدد خيارات التحميل؛ نريد تحميل جدول بيانات الأرقام
LoadOptions opts = new LoadOptions(LoadFormat.Numbers);
```
 نحن ننشئ`LoadOptions` الكائن وتحديد التنسيق كـ`LoadFormat.Numbers`يخبر هذا مكتبة Aspose.Cells بأننا نعمل على ملف Numbers. 
## الخطوة 3: تحميل جدول بيانات الأرقام في مصنف
الآن، حان الوقت لتحميل جدول الأرقام الفعلي إلى`Workbook` هدف.
```csharp
// قم بتحميل جدول بيانات الأرقام في المصنف باستخدام خيارات التحميل المذكورة أعلاه
Workbook wb = new Workbook(sourceDir + "sampleNumbersByAppleInc.numbers", opts);
```
 نحن ننشئ مثالا`Workbook` الكائن ومرر مسار الملف لملف Numbers مع خيارات التحميل الخاصة بنا. تأكد من أن اسم الملف (`sampleNumbersByAppleInc.numbers`) يتطابق مع الاسم الفعلي لملف الأرقام الخاص بك.
## الخطوة 4: احفظ المصنف بتنسيق PDF
بمجرد تحميل ملف Numbers بنجاح، فإن الخطوة التالية هي حفظه بتنسيق مختلف - على وجه التحديد، PDF.
```csharp
// حفظ المصنف بصيغة PDF
wb.Save(outputDir + "outputNumbersByAppleInc.pdf", SaveFormat.Pdf);
```
 هنا، نسميها`Save` الطريقة على`Workbook` الكائن، مع تحديد مسار ملف الإخراج والتنسيق الذي نريد حفظه به. في هذه الحالة، نقوم بحفظه بتنسيق PDF. تأكد من أن اسم ملف الإخراج (`outputNumbersByAppleInc.pdf`) فريد من نوعه ولا يستبدل أي ملفات موجودة.
## الخطوة 5: تأكيد النجاح
وأخيرًا، دعنا نضيف رسالة للتأكيد على نجاح عمليتنا.
```csharp
Console.WriteLine("ReadNumbersSpreadsheet executed successfully.\r\n");
```
سيطبع هذا السطر من التعليمات البرمجية رسالة نجاح على وحدة التحكم بمجرد الانتهاء من كل شيء. من الجيد دائمًا الحصول على تعليقات، أليس كذلك؟
