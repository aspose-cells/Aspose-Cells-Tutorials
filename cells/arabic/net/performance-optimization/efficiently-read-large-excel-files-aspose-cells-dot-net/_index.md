## دليل التنفيذ
### تكوين LoadOptions لتفضيلات الذاكرة
لقراءة ملفات Excel الكبيرة بكفاءة، قم بتكوين `LoadOptions` تفضيل تحسين الذاكرة.

#### الخطوة 1: إعداد مشروعك ومساحة الاسم
```csharp
using System;
using Aspose.Cells;

namespace EfficientExcelReading
{
    public class ReadingLargeExcelFiles
    {
        public static void Run()
        {
            // إعداد مسار دليل المصدر
            string sourceDir = RunExamples.Get_SourceDirectory();
            
            // حدد LoadOptions باستخدام تفضيلات الذاكرة
            LoadOptions opt = new LoadOptions { MemorySetting = MemorySetting.MemoryPreference };
            
            // إنشاء مثيل للمصنف باستخدام الإعدادات المُحسّنة
            tWorkbook wb = new tWorkbook(sourceDir + "sampleReadingLargeExcelFiles.xlsx", opt);
            
            Console.WriteLine("Efficiently reading large Excel files executed successfully.");
        }
    }
}
```
#### شرح الأجزاء الرئيسية:
- **خيارات التحميل**:قم بتكوين هذا الكائن لتحديد تفضيلات الذاكرة، وهو أمر ضروري للتعامل مع مجموعات البيانات الكبيرة.
- **إعدادات الذاكرة.تفضيلات الذاكرة**:يعمل هذا الإعداد على تحسين استخدام الذاكرة أثناء تحميل الملف.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من إعداد مسار دليل المصدر الخاص بك بشكل صحيح؛ وإلا، فقد تواجه `FileNotFoundException`.
- راقب استخدام الموارد لتجنب تجاوز سعة الذاكرة عند معالجة ملفات كبيرة جدًا.
