## دليل التنفيذ

### تعيين الحد الأقصى لعدد الصفوف للصيغ المشتركة

#### ملخص
توضح هذه الميزة كيفية تعيين حد لعدد الصفوف التي يمكن أن تمتد إليها صيغة مشتركة في ورقة عمل Excel.

**الخطوة 1: إنشاء كائن مصنف**

ابدأ بتهيئة ملف جديد `Workbook` الكائن الذي يمثل ملف Excel الخاص بك.

```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY"; // قم بتحديد دليل الإخراج الخاص بك

// تهيئة المصنف
Workbook wb = new Workbook();
```

**الخطوة 2: تكوين الحد الأقصى للصفوف للصيغ المشتركة**

يستخدم `wb.Settings.MaxRowsOfSharedFormula` لتحديد الحد الأقصى لعدد الصفوف التي يمكن أن تغطيها الصيغة المشتركة. هنا، قمنا بضبطه على 5.

```csharp
// تعيين الحد الأقصى للصفوف للصيغ المشتركة
wb.Settings.MaxRowsOfSharedFormula = 5;
```

**الخطوة 3: احفظ مصنفك**

وأخيرًا، احفظ المصنف الخاص بك لتطبيق هذه الإعدادات.

```csharp
// الوصول إلى ورقة العمل الأولى وحفظها
Worksheet ws = wb.Worksheets[0];
wb.Save(outputDir + "outputMaxRowsSharedFormula.xlsx");
```

### تطبيق صيغة مشتركة على صفوف متعددة

#### ملخص
تعلم كيفية استخدام `SetSharedFormula` طريقة لتطبيق الصيغ عبر خلايا متعددة بكفاءة.

**الخطوة 1: إعداد المصنف وورقة العمل الخاصة بك**

كما في السابق، قم بتهيئة المصنف الخاص بك والوصول إلى ورقة العمل الأولى الخاصة به.

```csharp
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```

**الخطوة 2: تطبيق صيغة مشتركة**

ولإثبات ذلك، دعونا نطبق `Sum` الصيغة من A1 إلى A2 عبر 100 صف بدءًا من الخلية D1.

```csharp
Cell cell = ws.Cells["D1"];
cell.SetSharedFormula("=Sum(A1:A2)", 100, 1);
```

**الخطوة 3: احفظ مصنفك**

تأكد من حفظ التغييرات لرؤية تأثيرات تطبيق الصيغة المشتركة.

```csharp
wb.Save(outputDir + "outputApplySharedFormula.xlsx");
```
