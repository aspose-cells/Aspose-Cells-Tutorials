**ما سوف تتعلمه:**
- إعداد Aspose.Cells في مشروع .NET
- إنشاء ملف Excel يحتوي على 10000 صف و30 عمودًا
- قياس الوقت المستغرق لإنشاء ملفات Excel كبيرة الحجم

## دليل التنفيذ

### إنشاء ملف Excel باستخدام Aspose.Cells
#### ملخص
يوضح هذا القسم كيفية إنشاء ملف Excel بسيط باستخدام Aspose.Cells، وملئه ببيانات العينة.

**خطوات التنفيذ:**
1. **تهيئة المصنف وورقة العمل**
   ```csharp
   Workbook workbook = new Workbook();
   Worksheet ws = workbook.Worksheets[0];
   ```
2. **ملء ورقة العمل بالبيانات**
   - استخدم الحلقات المتداخلة لملء الخلايا:
     ```csharp
     for (int row = 0; row < 10000; row++)
     {
         for (int col = 0; col < 30; col++)
         {
             ws.Cells[row, col].PutValue(row.ToString() + "," + col.ToString());
         }
     }
     ```
3. **حفظ المصنف**
   - حدد دليل الإخراج واحفظه:
     ```csharp
     string outputDir = "YOUR_OUTPUT_DIRECTORY";
     workbook.Save(outputDir + "CellsSample_out.xlsx"); // تم التحديث إلى .xlsx لتنسيقات Excel الحديثة
     ```

### قياس وقت إنشاء الملف
#### ملخص
قم بقياس المدة التي يستغرقها إنشاء ملف Excel، مما يساعدك على تحسين الأداء.

**خطوات التنفيذ:**
1. **توقيت البدء**
   - سجل وقت البدء قبل إنشاء المصنف:
     ```csharp
     DateTime start = DateTime.Now;
     ```
2. **إنشاء مصنف وملء البيانات**
   - اتبع الخطوات المماثلة المذكورة أعلاه لتعبئة البيانات.
3. **توقيت النهاية**
   - التقاط وقت النهاية بعد حفظ الملف:
     ```csharp
     DateTime end = DateTime.Now;
     TimeSpan time = end - start;
     Console.WriteLine("Time taken: " + time.TotalSeconds + " seconds"); // الإخراج إلى وحدة التحكم للرؤية
     ```
