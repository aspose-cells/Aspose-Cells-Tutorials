### دليل التنفيذ

دعونا نقسم التنفيذ إلى أقسام قابلة للإدارة لكل ميزة:

#### إنشاء مصنف وتكوينه باستخدام Aspose.Cells

**ملخص**

إنشاء مصنف هو خطوتك الأولى نحو أتمتة Excel. سيرشدك هذا القسم إلى كيفية تهيئة مصنف، والوصول إلى أوراق عمله، وحفظه بالتنسيق المناسب.

##### الخطوة 1: تهيئة المصنف

```csharp
using System;
using Aspose.Cells;

string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// إنشاء مثيل جديد من مصنف
Workbook workbook = new Workbook();
```

ال `Workbook` تُمثل الفئة ملف Excel الخاص بك. بإنشاء مثيل، فأنت تُجهّز نفسك للعمل مع هذا الملف برمجيًا.

##### الخطوة 2: الوصول إلى ورقة العمل الأولى

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

يحتوي كل مصنف على مجموعة من أوراق العمل. هنا، نصل إلى ورقة العمل الأولى حسب الفهرس. `0`.

##### الخطوة 3: حفظ المصنف

```csharp
// احفظ المصنف بصيغة xlsx
workbook.Save(outputDir + "outputCreateWorkbook.xlsx");
```

تكتب هذه الخطوة تغييراتك إلى ملف Excel.

#### إضافة وتكوين شكل مربع نص مع النص

**ملخص**

إضافة أشكال مثل مربعات النص تُحسّن المظهر البصري لجداول بياناتك. يوضح هذا القسم كيفية إضافة شكل مربع نص وتخصيص محتواه وحجم خطه.

##### الخطوة 1: إنشاء مربع نص

```csharp
using Aspose.Cells.Drawing;

// إضافة مربع نص إلى ورقة العمل
TextBox textbox = worksheet.Shapes.AddTextBox(0, 0, 0, 0, 100, 700);
textbox.Text = "Aspose File Format APIs";
textbox.Font.Size = 44;
```

ال `AddTextBox` تتيح لك هذه الطريقة تحديد الموضع والحجم. هنا، نحدد نصًا وحجم خط مخصصين.

##### الخطوة 2: حفظ المصنف

```csharp
// حفظ التغييرات مع إضافة مربع النص
workbook.Save(outputDir + "outputAddTextbox.xlsx");
```

تأكد من حفظ التغييرات بعد إضافة الأشكال.

#### تطبيق نمط WordArt المحدد مسبقًا على نص مربع النص

**ملخص**

حسّن عرض النص بتطبيق أنماط مُعدّة مسبقًا، مثل WordArt. يوضح هذا القسم كيفية تطبيق نمط على النص داخل شكل مربع النص.

##### الخطوة 1: تعيين نمط WordArt

```csharp
FontSetting fntSetting = textbox.GetCharacters()[0] as FontSetting;
fntSetting.SetWordArtStyle(PresetWordArtStyle.WordArtStyle3);
```

يستخدم `SetWordArtStyle` لتطبيق الأنماط المحددة مسبقًا، مما يعزز جماليات النص.

##### الخطوة 2: حفظ المصنف

```csharp
// احفظ المصنف مع تطبيق نمط WordArt عليه
workbook.Save(outputDir + "outputSetPresetWordArtStyle.xlsx");
```

قم بإتمام تغييراتك عن طريق حفظ المصنف.
