**ملخص:**

#### التنفيذ خطوة بخطوة:
##### تهيئة المصنف وورقة العمل
ابدأ بإعداد مصنفك وورقة عملك. ستكون هذه هي لوحتك للعمل على البيانات.
```csharp
// بداية سابقة: 1
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

```

##### ملء الخلايا في حلقة
املأ خلاياك بحلقات متداخلة، كما هو موضح أدناه. لاحظ الكفاءة التي تحققت باستخدام Aspose.Cells.
```csharp
DateTime start = DateTime.Now;

for (int i = 0; i < 1000; i++)
{
    for (int j = 0; j < 20; j++)
    {
        // تعيين القيم في الخلايا بكفاءة
        ws.Cells[i, j].PutValue("Row " + (i + 1).ToString() + " Col " + (j + 1).ToString());
    }
}

DateTime end = DateTime.Now;
TimeSpan time = end - start;
Console.WriteLine("File Created! Time consumed (Seconds): " + time.TotalSeconds.ToString());
// نهاية: 1
```
**شرح المعلمات:**
- `PutValue`: يحدد قيمة الخلية. تم تحسينه لتحسين الأداء.
- `DateTime` و `TimeSpan`:يستخدم لقياس وقت التنفيذ، مع تسليط الضوء على سرعة Aspose.Cells.
