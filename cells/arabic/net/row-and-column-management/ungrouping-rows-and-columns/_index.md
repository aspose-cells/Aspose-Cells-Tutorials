# فك تجميع الصفوف والأعمدة في Excel باستخدام Aspose.Cells

## مقدمة
عند التعامل مع ملفات Excel، قد تجد نفسك في مواقف تحتاج فيها إلى فك تجميع الصفوف والأعمدة. سواء كنت تُنظّف جدول بيانات أو تُعيد تنسيق البيانات لعرض أفضل، فإن Aspose.Cells for .NET أداة رائعة تُبسّط العملية. في هذا البرنامج التعليمي، سأرشدك خلال خطوات فك تجميع الصفوف والأعمدة في Excel باستخدام Aspose.Cells. في النهاية، ستكتسب فهمًا متينًا لكيفية التعامل مع ملفات Excel برمجيًا.
## المتطلبات الأساسية
قبل التعمق في الكود، تأكد من إعداد كل شيء. إليك ما ستحتاجه:
1. فيجوال ستوديو: يجب أن يكون لديك إصدار فعال من فيجوال ستوديو مثبتًا على جهازك. إذا لم يكن لديك الإصدار بعد، يمكنك تنزيله من [موقع Visual Studio](https://visualstudio.microsoft.com/).
2. Aspose.Cells لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.Cells. يمكنك الحصول عليها من [صفحة إصدارات Aspose](https://releases.aspose.com/cells/net/)تأكد من حصولك على التراخيص اللازمة، والتي يمكن شراؤها أو الحصول عليها من خلال [رخصة مؤقتة](https://purchase.aspose.com/temporary-license/).

بمجرد أن يكون كل شيء جاهزًا، يمكننا الانتقال إلى الجزء الممتع: الكود!
## استيراد الحزم
للبدء، عليك استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك الطريقة:
1. افتح مشروعك في Visual Studio.
2. أضف مرجعًا إلى مكتبة Aspose.Cells. يمكنك القيام بذلك بالنقر بزر الماوس الأيمن على المراجع في مشروعك واختيار "إضافة مرجع". انتقل إلى المكان الذي حفظت فيه مكتبة Aspose.Cells DLL.
3. في أعلى ملف C# الخاص بك، أضف ما يلي باستخدام التوجيهات:
```csharp
using System.IO;
using Aspose.Cells;
```
الآن بعد إعداد كل شيء، دعنا ننتقل إلى الخطوات اللازمة لإلغاء تجميع الصفوف والأعمدة في ورقة Excel الخاصة بك. 
## الخطوة 1: تحديد دليل المستندات
أولاً، عليك تحديد المجلد الذي يحتوي على ملف Excel. يمكنك إعداده كما يلي:
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
```
يستبدل `"Your Document Directory"` مع المسار الفعلي على جهاز الكمبيوتر الخاص بك حيث يتم حفظ ملف Excel. 
## الخطوة 2: إنشاء تدفق ملف
بعد ذلك، عليك إنشاء مسار ملف لفتح ملف Excel. إليك كيفية القيام بذلك:
```csharp
// إنشاء مجرى ملف يحتوي على ملف Excel الذي سيتم فتحه
FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open);
```
هنا، تقوم بفتح الملف المسمى `book1.xls`تأكد من وجود هذا الملف في الدليل المحدد، وإلا فسوف تواجه خطأ عدم العثور على الملف.
## الخطوة 3: إنشاء كائن مصنف
الآن، لنحمّل ملف Excel إلى كائن مصنف. هذا يسمح لك بالتعامل مع المصنف برمجيًا:
```csharp
// إنشاء كائن مصنف
// فتح ملف Excel من خلال تدفق الملف
Workbook workbook = new Workbook(fstream);
```
باستخدام هذا السطر من التعليمات البرمجية، قمت بتحميل ملف Excel إلى الذاكرة بنجاح وأصبحت جاهزًا للعمل معه.
## الخطوة 4: الوصول إلى ورقة العمل
بعد الحصول على المصنف، الخطوة التالية هي الوصول إلى ورقة العمل المحددة التي تريد فك تجميع الصفوف والأعمدة فيها. إليك كيفية القيام بذلك:
```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];
```
في هذه الحالة، نصل إلى ورقة العمل الأولى. إذا كانت بياناتك موجودة في ورقة عمل مختلفة، يمكنك تغيير الفهرس وفقًا لذلك.
## الخطوة 5: إلغاء تجميع الصفوف
الآن يأتي الجزء المثير! لنفكّ تجميع الصفوف الستة الأولى (من الصف ٠ إلى الصف ٥). استخدم الكود التالي:
```csharp
// إلغاء تجميع أول ستة صفوف (من 0 إلى 5)
worksheet.Cells.UngroupRows(0, 5);
```
هذه الطريقة تزيل أي تجميع مُطبّق على الصفوف المُحددة. الأمر بهذه السهولة!
## الخطوة 6: إلغاء تجميع الأعمدة
كما هو الحال مع الصفوف، يمكنك أيضًا فصل الأعمدة. إليك كيفية فصل الأعمدة الثلاثة الأولى (من العمود ٠ إلى العمود ٢):
```csharp
// إلغاء تجميع الأعمدة الثلاثة الأولى (من 0 إلى 2)
worksheet.Cells.UngroupColumns(0, 2);
```
## الخطوة 7: حفظ ملف Excel المعدّل
بعد فك تجميع الصفوف والأعمدة، الخطوة التالية هي حفظ التغييرات في ملف Excel. يمكنك القيام بذلك باستخدام `Save` طريقة:
```csharp
// حفظ ملف Excel المعدل
workbook.Save(dataDir + "output.xls");
```
في هذا المثال، نقوم بحفظ الملف المعدل باسم `output.xls`يمكنك تغيير اسم الملف إلى أي اسم تفضله.
## الخطوة 8: إغلاق مجرى الملف
أخيرًا، لتحرير الموارد، يجب عليك إغلاق مجرى الملف:
```csharp
// إغلاق مجرى الملف لتحرير كافة الموارد
fstream.Close();
```
هذه ممارسة جيدة للتأكد من أن تطبيقك لا يحتفظ بمقابض الملفات لفترة أطول من اللازم.
