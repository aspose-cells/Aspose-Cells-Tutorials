# احصل على عرض وارتفاع الورقة لطباعة ورقة العمل

## مقدمة
تتطلب طباعة المستندات بدقة معرفة أبعاد الورق. إذا كنت مطورًا أو تعمل على تطبيق يتعامل مع ملفات Excel، فقد تحتاج إلى معرفة كيفية الحصول على عرض وارتفاع الورق عند طباعة أوراق العمل. لحسن الحظ، يوفر Aspose.Cells for .NET طريقة فعّالة لإدارة مستندات Excel برمجيًا. في هذه المقالة، سنرشدك خلال عملية تحديد حجم الورق، باستخدام أمثلة بسيطة لتوضيح المفاهيم الأساسية. 
## المتطلبات الأساسية
قبل الخوض في التفاصيل الفنية، دعونا نوضح بعض الأساسيات. لمتابعة هذا البرنامج التعليمي بنجاح، ستحتاج إلى:
### 1. المعرفة الأساسية بلغة C#
يجب أن يكون لديك فهم جيد لبرمجة C#، حيث سنعمل ضمن بيئة .NET.
### 2. مكتبة Aspose.Cells
تأكد من تثبيت مكتبة Aspose.Cells في مشروعك. إذا لم تقم بذلك بعد، يمكنك تنزيل أحدث إصدار من [صفحة تنزيل Aspose.Cells](https://releases.aspose.com/cells/net/).
### 3. بيئة التطوير المتكاملة لبرنامج Visual Studio
من المفيد استخدام Visual Studio لتشغيل وإدارة مشاريع C#. أي إصدار يدعم .NET سيعمل بكفاءة.
### 4. ترخيص Aspose صالح
مع أنه يمكنك تجربة Aspose.Cells، فكّر في شراء ترخيص إذا كنت تستخدمه لمشاريع طويلة الأمد. يمكنك شراؤه من خلال [هذا الرابط](https://purchase.aspose.com/buy) أو استكشاف [رخصة مؤقتة](https://purchase.aspose.com/temporary-license/) لمراحل الاختبار القصيرة.
بمجرد أن تكون كل الأمور جاهزة، دعنا ننتقل إلى الكود!
## استيراد الحزم
الخطوة الأولى في رحلتنا تتضمن استيراد مساحات الأسماء الأساسية. هذا أمر بالغ الأهمية، إذ يتيح لنا الوصول إلى الفئات والأساليب التي سنستخدمها لمعالجة ملفات Excel. إليك كيفية القيام بذلك:
```csharp
using System;
using System.IO;
using Aspose.Cells;
```
تأكد من تضمين هذا السطر في أعلى ملف .cs. الآن وقد أصبحت عمليات الاستيراد جاهزة، لنبدأ بإنشاء مصنفنا والوصول إلى ورقة العمل.
## الخطوة 1: إنشاء مصنفك
نبدأ بإنشاء مثيل لـ `Workbook` يشكل هذا الأساس لمعالجة ملفات Excel الخاصة بنا.
```csharp
Workbook wb = new Workbook();
```
يخبر هذا السطر البرنامج بتهيئة مصنف جديد، مما يهيئنا للتعمق في أوراق العمل الخاصة بنا.
## الخطوة 2: الوصول إلى ورقة العمل الأولى
بعد ذلك، سنصل إلى ورقة العمل الأولى في مصنفنا الجديد. الأمر بسيط جدًا:
```csharp
Worksheet ws = wb.Worksheets[0];
```
هنا، نصل إلى الورقة الأولى (المفهرسة عند ٠) في مصنفنا. هنا سنضبط أحجام الورق.
## ضبط حجم الورق واسترجاع الأبعاد
الآن ندخل في جوهر العملية - تحديد حجم الورق واسترجاع أبعاده! لنشرح ذلك خطوة بخطوة.
## الخطوة 3: اضبط حجم الورق على A2
دعونا أولاً نضبط حجم الورق على A2 ونطبع أبعاده.
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA2;
Console.WriteLine("PaperA2: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
بعد هذا الإعداد، نستخدم `Console.WriteLine` لعرض الأبعاد. عند تشغيل هذا الأمر، سترى العرض والارتفاع بالبوصة لحجم ورق A2.
## الخطوة 4: ضبط حجم الورق إلى A3
الآن حان وقت A3! نكرر العملية ببساطة:
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA3;
Console.WriteLine("PaperA3: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
ها هو! سيُطبع الإعلان الارتفاع والعرض المحددين لورق A3.
## الخطوة 5: اضبط حجم الورق على A4
باتباع نفس النمط، دعنا نتحقق من كيفية قياس A4:
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA4;
Console.WriteLine("PaperA4: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
وهذا يزودنا بالأبعاد الخاصة بـ A4، وهو أحد أحجام الورق الأكثر استخدامًا.
## الخطوة 6: ضبط حجم الورق إلى Letter
لاستكمال استكشافنا لحجم الورق، دعنا نضبطه على حجم Letter:
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperLetter;
Console.WriteLine("PaperLetter: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
مرة أخرى، سنرى العرض والارتفاع المحددين لحجم الحرف.
