## إعداد Aspose.Cells لـ .NET

### معلومات التثبيت

ثبّت Aspose.Cells في مشروع .NET الخاص بك عبر مدير حزم NuGet. استخدم الأوامر التالية حسب بيئة التطوير الخاصة بك:

**.NET CLI**
```bash
dotnet add package Aspose.Cells
```

**وحدة تحكم مدير الحزم**
```plaintext
PM> Install-Package Aspose.Cells
```

### الحصول على الترخيص

ابدأ بفترة تجريبية مجانية أو احصل على ترخيص مؤقت لاستكشاف جميع الميزات. للمشاريع الجارية، فكّر في الشراء من موقعهم الرسمي.

### التهيئة والإعداد الأساسي

بمجرد التثبيت، قم بتهيئة مشروعك عن طريق إنشاء ملف جديد `Workbook` مثال:

```csharp
using Aspose.Cells;

// تهيئة المصنف
Workbook wb = new Workbook();
```

## دليل التنفيذ

### إنشاء خلية وتصميمها

تعرف على كيفية إنشاء مصنف Excel والوصول إلى خلايا محددة وتطبيق أنماط مخصصة.

#### ملخص

سنبدأ بإنشاء مصنف، والوصول إلى الخلية "B5"، وإضافة محتوى نصي، وتصميمه باستخدام لون الخط الأحمر.

#### التنفيذ خطوة بخطوة

1. **إنشاء مصنف والوصول إلى الخلية**
   
   قم بتهيئة المصنف الخاص بك وحدد ورقة العمل:
   
   ```csharp
   using Aspose.Cells;
   using System.Drawing;
   
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";
   
   Workbook wb = new Workbook();
   Worksheet ws = wb.Worksheets[0];
   Cell cell = ws.Cells["B5"];
   ```

2. **تعيين قيمة الخلية والنمط**
   
   أضف نصًا إلى الخلية وقم بتطبيق لون الخط الأحمر:
   
   ```csharp
   cell.PutValue("This is some text.");
   Style st = cell.GetStyle();
   st.Font.Color = Color.Red;
   cell.SetStyle(st);
   ```

#### خيارات تكوين المفاتيح
- **لون الخط**:تخصيص مع أي `System.Drawing.Color` قيمة.
- **قيمة الخلية**: يستخدم `.PutValue()` لمختلف أنواع البيانات.

### تصدير المصنف بصيغة HTML باستخدام CSS منفصل

تعرف على كيفية تصدير مصنف مصمم إلى تنسيق HTML، مما يتيح تنسيق CSS منفصلاً لكل ورقة عمل.

#### ملخص

سوف نقوم بتصدير المصنف المصمم إلى تنسيق HTML وتكوينه بحيث يكون CSS منفصلاً عن المحتوى.

#### التنفيذ خطوة بخطوة

1. **تصدير المصنف**
   
   بعد إعداد نمط الخلية الخاص بك، استخدم `HtmlSaveOptions` لتحديد كيفية رغبتك في الحصول على إخراج HTML:
   
   ```csharp
   HtmlSaveOptions opts = new HtmlSaveOptions();
   opts.ExportWorksheetCSSSeparately = true;
   wb.Save(outputDir + "outputExportWorksheetCSSSeparately.html", opts);
   ```

#### خيارات تكوين المفاتيح
- **تصدير ورقة عمل CSS بشكل منفصل**: تم الضبط على `true` لملفات CSS المنفصلة.
