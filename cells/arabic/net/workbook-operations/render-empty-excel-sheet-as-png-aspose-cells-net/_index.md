## دليل التنفيذ

الآن بعد أن أصبح لديك الإعداد اللازم، دعنا نعرض ورقة عمل فارغة كصورة PNG.

### عرض ورقة عمل فارغة كصورة PNG

هذه الميزة مفيدة لإنشاء تمثيلات مرئية لأوراق العمل بدون بيانات. إليك كيفية تنفيذها:

#### الخطوة 1: إنشاء مصنف وتكوينه

إنشاء مثيل جديد لمصنف عمل يتضمن ورقة عمل افتراضية واحدة.
```csharp
// تهيئة مثيل مصنف جديد
Workbook wb = new Workbook();

// الوصول إلى ورقة العمل الأولى (الافتراضية)
Worksheet ws = wb.Worksheets[0];
```

#### الخطوة 2: إعداد خيارات الصورة

تكوين `ImageOrPrintOptions` لتحديد PNG كتنسيق الإخراج والتأكد من إنشاء صورة للأوراق الفارغة.
```csharp
// تكوين خيارات الصورة أو الطباعة
ImageOrPrintOptions opts = new ImageOrPrintOptions {
    // تم ضبط تنسيق الإخراج على PNG
    ImageType = Drawing.ImageType.Png,
    
    // تأكد من إنتاج صورة حتى للأوراق الفارغة
    OutputBlankPageWhenNothingToPrint = true
};
```

#### الخطوة 3: عرض ورقة العمل

يستخدم `SheetRender` لإنشاء الصورة وحفظها في دليل الإخراج المحدد.
```csharp
// تحويل ورقة العمل إلى ملف PNG
SheetRender sr = new SheetRender(ws, opts);
sr.ToImage(0, "YOUR_OUTPUT_DIRECTORY\OutputBlankPageWhenNothingToPrint.png");
```

يؤدي مقتطف التعليمات البرمجية هذا إلى إنشاء صورة لورقة العمل الفارغة وحفظها باسم `OutputBlankPageWhenNothingToPrint.png` في دليل الإخراج الخاص بك.
