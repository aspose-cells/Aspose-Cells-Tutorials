## دليل التنفيذ

### نظرة عامة على إشارات مرجعية PDF ذات وجهات محددة

تتيح لك إشارات مرجعية PDF التنقل بسهولة عبر المستندات. في هذا الدليل، سننشئ وجهات مرجعية مُسمّاة ترتبط مباشرةً بخلايا مُحددة في ورقة Excel عند تصديرها كملف PDF. يُعدّ هذا مفيدًا بشكل خاص لإنشاء تقارير ووثائق ديناميكية.

#### التنفيذ خطوة بخطوة

##### 1. إعداد مصنف العمل الخاص بك

ابدأ بتحميل ملف Excel المصدر الخاص بك:

```csharp
// تحميل ملف Excel المصدر
tWorkbook wb = new Workbook("samplePdfBookmarkEntry_DestinationName.xlsx");
```

##### 2. الوصول إلى أوراق العمل والخلايا

قم بالوصول إلى ورقة العمل والخلايا المطلوبة حيث تريد إنشاء الإشارات المرجعية.

```csharp
// الوصول إلى ورقة العمل الأولى
tWorksheet ws = wb.Worksheets[0];

// الوصول إلى الخلية C5
tCell cellC5 = ws.Cells["C5"];
```

##### 3. إنشاء إدخالات الإشارة المرجعية

تعريف أ `PdfBookmarkEntry` لكل إشارة مرجعية تحتوي على نص ووجهة:

```csharp
// إنشاء إشارة مرجعية ووجهة للخلية C5
tPdfBookmarkEntry bookmarkEntry = new PdfBookmarkEntry();
bookmarkEntry.Text = "Main Section";
bookmarkEntry.Destination = cellC5;
bookmarkEntry.DestinationName = "AsposeCells--" + cellC5.Name;

// الوصول إلى خلايا أخرى وإنشاء إشارات مرجعية فرعية على نحو مماثل
tCell cellG56 = ws.Cells["G56"];
tPdfBookmarkEntry subbookmark1 = new PdfBookmarkEntry();
subbookmark1.Text = "Subsection 1";
subbookmark1.Destination = cellG56;
subbookmark1.DestinationName = "AsposeCells--" + cellG56.Name;

// كرر ذلك للخلايا الإضافية حسب الحاجة
```

##### 4. تنظيم الإشارات المرجعية الفرعية

أضف إشاراتك المرجعية الفرعية إلى قائمة وقم بتعيينها للإشارة المرجعية الرئيسية:

```csharp
ArrayList list = new ArrayList { subbookmark1 /*، أضف إشارات مرجعية فرعية أخرى هنا */ };
bookmarkEntry.SubEntry = list;
```

##### 5. تكوين خيارات حفظ PDF

يثبت `PdfSaveOptions` لتضمين الإشارات المرجعية وحفظ المصنف الخاص بك بتنسيق PDF:

```csharp
// تكوين خيارات حفظ Pdf
tPdfSaveOptions opts = new PdfSaveOptions();
opts.Bookmark = bookmarkEntry;

// احفظ المصنف مع الإشارات المرجعية بتنسيق PDF
wb.Save("outputPdfBookmarkEntry_DestinationName.pdf", opts);
```
