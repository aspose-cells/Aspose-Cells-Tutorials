## دليل التنفيذ
### الميزة 1: تهيئة المصنف وورقة العمل
**ملخص:** إن إنشاء مصنف جديد والوصول إلى أوراق العمل الخاصة به هي الخطوة الأولى للتعامل مع بيانات Excel برمجيًا.
#### الخطوة 1: إنشاء مصنف جديد
لإنشاء مثيل جديد من `Workbook`، قم بإنشائه ببساطة:
```csharp
Workbook workbook = new Workbook();
```
يؤدي هذا إلى تهيئة مصنف فارغ باستخدام ورقة عمل واحدة بشكل افتراضي.
#### الخطوة 2: الوصول إلى ورقة العمل الأولى
يمكنك الوصول إلى أوراق العمل باستخدام فهرسها. ورقة العمل الأولى موجودة عند الفهرس ٠:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
### الميزة 2: تعيين قيم الخلايا وتحديد النطاق المسمى
**ملخص:** يعد تعيين قيم الخلايا وإنشاء نطاقات مسماة أمرًا ضروريًا لتنظيم البيانات داخل ملفات Excel.
#### الخطوة 1: تعيين قيم الخلايا
تعيين قيم لخلايا محددة باستخدام مؤشرات الصفوف والأعمدة الخاصة بها:
```csharp
worksheet.Cells[0, 2].Value = 1; // المجموعات '1' في C1
document.Cells[1, 2].Value = 2; // المجموعات '2' في C2
```
#### الخطوة 2: تحديد نطاق مسمى
يمكنك إنشاء نطاق وتسميته للإشارة إليه بسهولة:
```csharp
Range namedRange = worksheet.Cells.CreateRange(0, 2, 3, 1);
namedRange.Name = "NamedRange";
```
يؤدي هذا إلى إنشاء نطاق من C1 إلى C3.
### الميزة 3: قص الخلايا وإدراجها في النطاق
**ملخص:** يتيح لك قص الخلايا وإدراجها إعادة تنظيم بياناتك بكفاءة داخل ورقة العمل.
#### الخطوة 1: إنشاء نطاق للعمود C
حدد العمود الذي تريد قطعه:
```csharp
Range cutRange = worksheet.Cells.CreateRange("C:C");
```
#### الخطوة 2: إدراج الخلايا المقطوعة
قص الخلايا وإدراجها، وتحريك الخلايا الموجودة حسب الضرورة:
```csharp
worksheet.Cells.InsertCutCells(cutRange, 0, 1, ShiftType.Right);
workbook.Save("outputDir/CutAndPasteCells.xlsx");
```
يؤدي هذا إلى قطع العمود C وإدراجه بدءًا من B1.
