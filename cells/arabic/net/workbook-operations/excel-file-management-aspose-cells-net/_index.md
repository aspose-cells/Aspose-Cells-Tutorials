### دليل التنفيذ

#### إنشاء مصنف Excel وتهيئته

**ملخص**:توضح هذه الميزة كيفية البدء بإنشاء مثيل جديد من `Workbook`، الذي يمثل ملف Excel الخاص بك.

**التنفيذ خطوة بخطوة**:

1. **تحديد دليل الإخراج**
   - قم بإعداد متغير لتخزين المسار الذي تريد حفظ الملفات أو تحميلها فيه.
   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";
   ```

2. **إنشاء مثيل مصنف**
   - إنشاء مثيل `Workbook` لإنشاء مستند Excel فارغ.
   ```csharp
   Workbook workbook = new Workbook();
   ```
   في هذه المرحلة، سيكون لديك كائن مصنف أساسي قابل للتعديل.

#### إضافة أجزاء XML مخصصة إلى المصنف

**ملخص**تعرّف على كيفية تضمين بيانات XML مخصصة في ملف Excel. قد يكون هذا مفيدًا لربط البيانات الوصفية أو المعلومات المنظمة الإضافية بمصنف عملك.

1. **قم بإعداد بيانات XML الخاصة بك**
   - تعريف محتوى XML كسلسلة.
   ```csharp
   string booksXML = @"<catalog>
      <book>
         <title>Complete C#</title>
         <price>44</price>
      </book>
      ...
   </catalog>";
   ```

2. **إضافة XML مخصص إلى المصنف**
   - استخدم `Add` الطريقة على `ContentTypeProperties` لربط XML الخاص بك باسم فريد.
   ```csharp
   workbook.ContentTypeProperties.Add("BookStore", booksXML);
   ```

#### حفظ المصنف

**ملخص**:وأخيرًا، دعنا نحفظ تعديلاتنا في ملف Excel على القرص.

1. **تحديد مسار الملف**
   - تأكد من أن لديك مسارًا صالحًا لحفظ ملفك.
   
2. **حفظ مثيل المصنف**
   ```csharp
   workbook.Save(outputDir + "outputUsingCustomXmlParts.xlsx");
   ```
   تم الآن حفظ المصنف الخاص بك كملف `.xlsx` الملف في الموقع المحدد.
