### إنشاء ملف ODS وحفظه باستخدام مواصفات ODF 1.2 الافتراضية

#### ملخص

تتيح لك هذه الميزة إنشاء ملف ODS بسيط باستخدام Aspose.Cells مع إعدادات مواصفات ODF 1.2 الافتراضية.

#### التنفيذ خطوة بخطوة

##### الخطوة 1: إعداد مسارات الدليل

قم بتحديد أدلة المصدر والإخراج الخاصة بك:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY"; // قم بتعيين مسار دليل المصدر الخاص بك هنا
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // قم بتعيين مسار دليل الإخراج الخاص بك هنا
```

##### الخطوة 2: إنشاء مصنف جديد

تهيئة مثيل مصنف جديد:
```csharp
Workbook workbook = new Workbook();
```

##### الخطوة 3: الوصول إلى ورقة العمل وتعديلها

قم بالوصول إلى ورقة العمل الأولى وأدخل البيانات في الخلية A1:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Welcome to Aspose!");
```

##### الخطوة 4: تكوين خيارات الحفظ وحفظ الملف

إعداد خيارات حفظ ODS لمواصفات ODF 1.2 الافتراضية وحفظ الملف:
```csharp
OdsSaveOptions options = new OdsSaveOptions();
workbook.Save(outputDir + "/ODF1.2_out.ods", options);
```

### إنشاء ملف ODS وحفظه باستخدام مواصفات ODF 1.1

#### ملخص

توضح هذه الميزة كيفية حفظ ملف ODS باستخدام Aspose.Cells مع الالتزام الصارم بمواصفات ODF 1.1.

#### التنفيذ خطوة بخطوة

##### الخطوة 1: إعداد مسارات الدليل

تأكد من تعريف أدلة المصدر والإخراج بشكل صحيح:
```csharp

```

##### الخطوة 2: إنشاء مصنف جديد

قم بتهيئة مثيل المصنف تمامًا كما في السابق:
```csharp
Workbook workbook = new Workbook();
```

##### الخطوة 3: الوصول إلى ورقة العمل وتعديلها

قم بالوصول إلى ورقة العمل وإدراج البيانات في الخلية A1:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
Cell cell = worksheet.Cells["A1"];
```

##### الخطوة 4: تكوين خيارات الحفظ لـ ODF 1.1 وحفظ الملف

إعداد خيارات حفظ ODS مع الامتثال الصارم لـ ODF 1.1:

## قسم الأسئلة الشائعة

1. **ما هو ODS؟**
   - ODS (OpenDocument Spreadsheet) هو تنسيق ملف XML مفتوح يستخدمه تطبيقات جداول البيانات، وخاصة تلك المستندة إلى LibreOffice و OpenOffice.

2. **كيف أقوم بتثبيت Aspose.Cells لـ .NET؟**
   - استخدم مدير الحزم NuGet أو .NET CLI كما هو موضح في هذا البرنامج التعليمي.

3. **ما هي مواصفات ODF؟**
   - ODF (تنسيق OpenDocument) هو معيار لملفات المستندات، بما في ذلك جداول البيانات والمستندات النصية والعروض التقديمية.

4. **هل يمكنني استخدام Aspose.Cells مع تنسيقات جداول البيانات الأخرى؟**
   - نعم، يدعم Aspose.Cells تنسيقات متعددة مثل XLSX، CSV، PDF، وما إلى ذلك.

5. **ماذا لو لم يتم حفظ ملف ODS بشكل صحيح؟**
   - تأكد من صحة مسارات الدليل وامتلاكك أذونات الكتابة اللازمة. تحقق من وجود أي استثناءات في الكود.
