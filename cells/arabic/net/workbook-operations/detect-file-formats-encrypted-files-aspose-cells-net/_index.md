### الخطوة 2: فتح الملف المشفر وقراءته
افتح ملفك المشفر باستخدام دفق. هنا، سنستخدم اسم ملف نموذجي. `encryptedBook1.out.tmp`.
```csharp
public static void Run()
{
    string sourceDir = "Your Source Directory Path";
    var filename = sourceDir + "encryptedBook1.out.tmp";

    // فتح الملف في وضع القراءة فقط
    using (Stream stream = File.Open(filename, FileMode.Open))
    {
        // اكتشاف التنسيق باستخدام كلمة مرور معروفة
        FileFormatInfo fileFormatInfo = FileFormatUtil.DetectFileFormat(stream, "1234"); 

        Console.WriteLine("File Format: " + fileFormatInfo.FileFormatType);
    }
}
```
### توضيح:
- **تدفق**:يوفر التدفق طريقة لقراءة بيانات الملف. هنا، نفتح الملف باستخدام `File.Open`.
- **FileFormatUtil.DetectFileFormat**:تأخذ هذه الطريقة الدفق وكلمة المرور (`"1234"`), اكتشاف التنسيق دون فك تشفيره بالكامل.
#### حدود:
- **تدفق**:تدفق الملف للمستند المشفر الخاص بك.
- **كلمة المرور**سلسلة نصية تُمثل كلمة المرور المُستخدمة لتشفير المستند. من الضروري أن يُحدد Aspose.Cells تنسيق الملف بشكل صحيح.
### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من أن المسار إلى دليل المصدر صحيح ويمكن الوصول إليه.
- تأكد من أن كلمة المرور المقدمة تتطابق مع كلمة المرور المستخدمة أثناء التشفير، وإلا فسوف تفشل عملية الاكتشاف.
