## دليل التنفيذ
### فتح كائن مصنف وإنشاء مثيل له
#### ملخص
في هذا القسم، سنوضح كيفية فتح ملف Excel باستخدام FileStream وإنشاء مثيل له `Workbook` كائن من Aspose.Cells.

#### الخطوة 1: إنشاء تدفق ملف لملف Excel
ابدأ بإنشاء FileStream للوصول إلى ملف Excel الخاص بك:
```csharp
using System.IO;
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";

// إنشاء FileStream لفتح ملف Excel
FileStream fstream = new FileStream(sourceDir + "/book1.xls", FileMode.Open);
```

#### الخطوة 2: إنشاء كائن مصنف
استخدم FileStream لإنشاء `Workbook` هدف:
```csharp
// إنشاء كائن مصنف باستخدام مجرى الملف
Workbook workbook = new Workbook(fstream);

// تذكر إغلاق FileStream بعد الاستخدام
fstream.Close();
```
تضمن هذه الخطوة أن يتم تحميل ملف Excel الخاص بك في الذاكرة، وجاهزًا للتعامل معه.

### الوصول إلى إمكانية رؤية ورقة العمل وتعديلها
#### ملخص
بعد ذلك، سنستكشف كيفية الوصول إلى ورقة عمل في ملف Excel وتغيير رؤيتها باستخدام Aspose.Cells.

#### الخطوة 1: افتح المصنف
أعد فتح المصنف كما هو موضح سابقًا:
```csharp
FileStream fstream = new FileStream(sourceDir + "/book1.xls", FileMode.Open);
Workbook workbook = new Workbook(fstream);
```

#### الخطوة 2: الوصول إلى ورقة العمل الأولى
قم بالوصول إلى ورقة العمل الأولى في ملف Excel الخاص بك:
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```

#### الخطوة 3: تعديل إمكانية رؤية ورقة العمل
تغيير إمكانية رؤية ورقة العمل التي تم الوصول إليها:
```csharp
// تعيين رؤية ورقة العمل إلى مخفية
worksheet.IsVisible = false;
```

#### الخطوة 4: حفظ المصنف المعدل
وأخيرًا، احفظ التغييرات مرة أخرى في ملف Excel:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/output.out.xls");

// إغلاق تدفق الملفات
fstream.Close();
```
