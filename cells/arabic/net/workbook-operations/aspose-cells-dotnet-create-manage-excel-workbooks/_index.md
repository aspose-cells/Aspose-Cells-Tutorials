## دليل التنفيذ

سنقوم بتقسيم البرنامج التعليمي إلى أقسام منطقية استنادًا إلى الميزات.

### إنشاء مصنف عمل

يُعدّ مصنف العمل الكائن الأساسي الذي يُمثّل ملف Excel. إليك كيفية إنشائه:

#### الخطوة 1: إنشاء مصنف جديد
```csharp
// إنشاء مثيل جديد لفئة Workbook
Workbook workbook = new Workbook();
```

هذا `Workbook` يمثل الكائن ملف Excel الخاص بك، مما يسمح لك بإضافة أوراق العمل والبيانات.

### إضافة ورقة عمل

إن إضافة ورقة عمل أمر سهل بمجرد حصولك على مثيل لمصنف العمل:

#### الخطوة 2: إضافة ورقة عمل جديدة
```csharp
// إضافة ورقة عمل جديدة إلى المصنف
int sheetIndex = workbook.Worksheets.Add();
```

تعيد هذه الطريقة فهرسًا للوصول إلى ورقة العمل المضافة حديثًا.

#### الخطوة 3: الوصول إلى ورقة العمل الخاصة بك وتسميتها

للعمل مع ورقة العمل الجديدة، يمكنك الوصول إليها من خلال فهرسها وتعيين اسم لها:

```csharp
// الحصول على مرجع إلى ورقة العمل التي تم إنشاؤها حديثًا
Worksheet newSheet = workbook.Worksheets[sheetIndex];

// تعيين اسم مخصص لسهولة التعرف عليه
newSheet.Name = "My Worksheet";
```

### حفظ المصنف

وأخيرًا، احفظ تعديلاتك مرة أخرى في ملف Excel:

#### الخطوة 4: تحديد مسار الإخراج وحفظه

```csharp
using System.IO;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// تأكد من وجود دليل الإخراج قبل الحفظ
bool isExists = Directory.Exists(outputDir);
if (!isExists)
    Directory.CreateDirectory(outputDir);

// تحديد المسار لملف الإخراج
string outputPath = Path.Combine(outputDir, "output.out.xls");

// حفظ المصنف في موقع محدد
workbook.Save(outputPath);
```

تضمن هذه الخطوة حفظ المصنف الخاص بك بالتنسيق والموقع المطلوبين.
