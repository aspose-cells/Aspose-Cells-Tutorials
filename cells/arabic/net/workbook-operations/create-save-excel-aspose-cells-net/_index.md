# دليل إنشاء ملفات Excel وحفظها باستخدام Aspose.Cells لـ .NET

**الدروس المستفادة:**
- التحقق من وجود الدليل وإنشائه
- إنشاء مثيلات لكتاب العمل باستخدام Aspose.Cells لـ .NET
- إدراج البيانات في خلايا المصنف
- تقنيات حفظ الملفات الآمنة

قبل البدء، تأكد من أن إعدادك يلبي المتطلبات الأساسية التالية:

## دليل التنفيذ

### إنشاء دليل

#### ملخص
تضمن هذه الميزة وجود الدليل المستهدف قبل تنفيذ عمليات الملف، مما يمنع حدوث أخطاء أثناء حفظ الملف.

##### الخطوة 1: التحقق من الدليل وإنشائه
```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY"; // قم بتحديد مسار دليل المصدر الخاص بك هنا
bool IsExists = Directory.Exists(SourceDir);
if (!IsExists)
{
    Directory.CreateDirectory(SourceDir); 
}
```
- **توضيح:** يتحقق هذا الكود من وجود دليل محدد ويقوم بإنشائه باستخدام `Directory.CreateDirectory` إذا لم يكن كذلك.

### إنشاء مصنف وحفظه باستخدام Aspose.Cells

#### ملخص
تعلم كيفية إنشاء مصنف Excel، وملئه بالبيانات، وحفظه في الموقع المطلوب.

##### الخطوة 2: إنشاء كائن مصنف
```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY"; // قم بتحديد مسار دليل الإخراج الخاص بك هنا

Workbook workbook = new Workbook(); 
Worksheet worksheet = workbook.Worksheets[0];
```
- **توضيح:** جديد `Workbook` يتم إنشاء الكائن، ويمكننا الوصول إلى الورقة الأولى.

##### الخطوة 3: إضافة البيانات إلى الخلايا
```csharp
// إضافة أنواع مختلفة من القيم إلى الخلايا
worksheet.Cells["A1"].PutValue("Hello World"); // قيمة السلسلة
worksheet.Cells["A2"].PutValue(20.5);          // قيمة مزدوجة
worksheet.Cells["A3"].PutValue(15);            // قيمة صحيحة
worksheet.Cells["A4"].PutValue(true);          // القيمة المنطقية

// إضافة قيمة التاريخ/الوقت وتعيين تنسيق عرضها
DateTime now = DateTime.Now;
worksheet.Cells["A5"].PutValue(now);
Style style = worksheet.Cells["A5"].GetStyle();
style.Number = 15;                             // تنسيق الأرقام للتاريخ
worksheet.Cells["A5"].SetStyle(style);
```
- **توضيح:** يقوم الكود بملء أنواع مختلفة من البيانات في الخلايا، بما في ذلك التاريخ المنسق.

##### الخطوة 4: حفظ ملف Excel
```csharp
workbook.Save(Path.Combine(outputDir, "output.out.xls"));
```
- **توضيح:** سيؤدي هذا إلى حفظ مصنفك في الدليل المحدد. تأكد من `outputDir` تم تعريفه بشكل صحيح.
