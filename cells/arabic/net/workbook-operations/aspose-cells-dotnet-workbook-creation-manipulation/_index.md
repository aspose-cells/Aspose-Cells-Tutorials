## دليل التنفيذ

### الميزة: إنشاء المصنفات ومعالجتها
يوضح هذا القسم كيفية إنشاء مصنف Excel، ومعالجة أوراق العمل الخاصة به، وإضافة قيم إلى الخلايا، وإدراج صور مع مراجع الخلايا، وحفظ المصنف.

#### إنشاء مصنف جديد
ابدأ بإنشاء حساب جديد `Workbook` هذا الكائن سيكون لوحتك لجميع العمليات:

```csharp
// إنشاء مصنف جديد
Workbook workbook = new Workbook();
```

#### الوصول إلى أوراق العمل وإضافة القيم
قم بالوصول إلى مجموعة خلايا ورقة العمل الأولى لبدء إدخال البيانات:

```csharp
// احصل على مجموعة خلايا ورقة العمل الأولى
Cells cells = workbook.Worksheets[0].Cells;

// إضافة قيم سلسلة إلى خلايا محددة
cells["A1"].PutValue("A1");
cells["C10"].PutValue("C10");
```

#### إدراج صورة مع مراجع الخلايا
أضف صورة إلى ورقتك وقم بالإشارة إليها عبر صيغ الخلايا:

```csharp
// أضف صورة فارغة في الموضع D1
Picture pic = workbook.Worksheets[0].Shapes.AddPicture(0, 3, 10, 6, null);

// حدد الصيغة للصورة التي تشير إلى الخلايا A1:C10
cells["D1"].Formula = "=OFFSET($A$1:$C$10, ROW()-ROW(A1), COLUMN()-COLUMN(A1))";
pic.Formula = "=OFFSET($A$1:$C$10, 0, 3)";

// تحديث قيمة الأشكال المحددة لتعكس التغييرات
table.Links[2].LinkSource = "path_to_your_image.jpg";
workbook.Worksheets[0].Shapes.UpdateSelectedValue();
```

#### حفظ المصنف
احفظ المصنف الخاص بك في موقع محدد:

```csharp
// احفظ المصنف في دليل الإخراج

```
