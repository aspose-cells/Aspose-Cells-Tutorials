### فتح ملف CSV باستخدام Aspose.Cells
#### ملخص
فتح ملف CSV باستخدام Aspose.Cells سهل للغاية. تتعامل المكتبة مع مختلف التكوينات والتنسيقات بسلاسة.

#### التنفيذ خطوة بخطوة
1. **إعداد خيارات التحميل**

   ابدأ بإنشاء خيارات تحميل خاصة بتنسيق CSV:

   ```csharp
   using Aspose.Cells;

   // إنشاء LoadOptions المحددة بواسطة LoadFormat.
   LoadOptions loadOptions = new LoadOptions(LoadFormat.Csv);
   ```

2. **تحميل ملف CSV إلى كائن مصنف**

   استخدم `Workbook` الفئة لفتح الملف الخاص بك:

   ```csharp
   string dataDir = "path/to/your/directory/";
   Workbook workbook = new Workbook(dataDir + "Book_CSV.csv", loadOptions);
   Console.WriteLine("CSV file opened successfully!");
   ```

#### شرح المعلمات
- **LoadFormat.Csv**:يحدد أن تنسيق الملف هو CSV.
- **كتاب العمل**:يمثل ملف Excel في Aspose.Cells، والذي يمكنه أيضًا التعامل مع ملفات CSV.
