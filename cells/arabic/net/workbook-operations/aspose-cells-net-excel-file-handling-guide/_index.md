## دليل التنفيذ

### فتح ملفات Excel عبر المسار

**ملخص:**
فتح ملف Excel باستخدام مسار مباشر أمر سهل. هذه الطريقة مثالية في الحالات التي يتوفر فيها الوصول إلى نظام الملفات.

**التنفيذ خطوة بخطوة:**

#### الخطوة 1: تحديد مسار الملف
```csharp
// قم بتحديد الدليل الذي يحتوي على ملفاتك.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// حدد المسار إلى ملف Excel.
string filePath = dataDir + "Book1.xls";
```

#### الخطوة 2: إنشاء كائن مصنف
```csharp
// قم بإنشاء كائن مصنف جديد وافتح ملف Excel المحدد.
Workbook workbook = new Workbook(filePath);
Console.WriteLine("Workbook opened using path successfully!");
```
**توضيح:** ال `Workbook` تُمثل الفئة ملف Excel. بتمرير مسار الملف، يُمكنك تحميله بسهولة إلى الذاكرة.

### فتح ملفات Excel عبر Stream

**ملخص:**
يُعد استخدام التدفقات مفيدًا عند التعامل مع ملفات لا يمكن الوصول إليها مباشرة على القرص أو للتطبيقات المستندة إلى الشبكة.

#### الخطوة 1: إنشاء FileStream
```csharp
// فتح تدفق لملف Excel موجود.
using (FileStream fstream = new FileStream(dataDir + "Book2.xls", FileMode.Open))
{
    // قم بتحميل المصنف من الدفق.
    Workbook workbookStream = new Workbook(fstream);
    Console.WriteLine("Workbook opened using stream successfully!");
}
```
**توضيح:** توفر التدفقات طريقة أكثر مرونة للتعامل مع عمليات الملفات، وخاصة عند التعامل مع ملفات كبيرة أو موارد الشبكة.

### فتح ملفات Excel المشفرة

**ملخص:**
تتطلب معالجة ملفات Excel المشفرة تحديد كلمة المرور أثناء عملية التحميل.

#### الخطوة 1: تعيين LoadOptions
```csharp
// قم بتحديد خيارات التحميل وتعيين كلمة المرور.
LoadOptions loadOptions = new LoadOptions();
loadOptions.Password = "1234";

// افتح مصنفًا مشفرًا باستخدام كلمة المرور المحددة.
Workbook wbEncrypted = new Workbook(dataDir + "encryptedBook.xls", loadOptions);
Console.WriteLine("Encrypted excel file opened successfully!");
```
**توضيح:** `LoadOptions` يسمح لك بتوفير المعلمات الضرورية مثل كلمات المرور، مما يضمن الوصول الآمن إلى الملفات المحمية.
