## دليل التنفيذ

يرشدك هذا القسم خلال عملية تصدير الصفوف المرئية من ورقة عمل Excel باستخدام **Aspose.Cells لـ .NET**.

### الخطوة 1: تحميل المصنف وورقة عمل Access

قم بتحميل مصنف Excel الخاص بك والوصول إلى ورقة العمل الأولى الخاصة به:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleExportVisibleRowsData.xlsx");
Worksheet worksheet = workbook.Worksheets[0]; // الوصول إلى ورقة العمل الأولى
```

### الخطوة 2: تصدير الصفوف المرئية مع الخيارات

التركيز على تصدير الصفوف المرئية فقط وإدراج أسماء الأعمدة:

```csharp
// تعيين خيارات التصدير لتشمل الصفوف والرؤوس المرئية فقط
ExportTableOptions exportOptions = new ExportTableOptions();
exportOptions.PlotVisibleRows = true; // تضمين الصفوف المرئية فقط
exportOptions.ExportColumnName = true; // تضمين رؤوس الأعمدة في التصدير

// تصدير نطاق محدد من الخلايا، بدءًا من A1
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, 10, 4, exportOptions);
```
