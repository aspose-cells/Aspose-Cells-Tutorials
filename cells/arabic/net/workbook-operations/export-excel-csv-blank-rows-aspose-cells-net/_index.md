# كيفية تصدير ملفات Excel بتنسيق CSV مع فواصل الصفوف الفارغة باستخدام Aspose.Cells لـ .NET

**ما سوف تتعلمه:**
- إعداد Aspose.Cells لـ .NET
- فتح مصنفات Excel وتكوينها
- تصدير جداول Excel إلى CSV مع الاحتفاظ بالصفوف الفارغة
- التطبيقات العملية لهذه الميزة

## دليل التنفيذ

بعد إعداد Aspose.Cells، دعنا نصدر ملف Excel بتنسيق CSV مع الاحتفاظ بالفواصل للصفوف الفارغة.

### فتح وتكوين المصنف (H2)

#### الخطوة 1: تحميل ملف Excel الخاص بك
حدد المسار إلى دليل المصدر الذي يوجد به ملف Excel. أنشئ `Workbook` كائن لفتحه:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string filePath = System.IO.Path.Combine(SourceDir, "Book1.xlsx");
Workbook wb = new Workbook(filePath);
```

#### الخطوة 2: تكوين خيارات الحفظ
يثبت `TxtSaveOptions` لتخصيص حفظ CSV والتأكد من الاحتفاظ بالفواصل للصفوف الفارغة:
```csharp
TxtSaveOptions options = new TxtSaveOptions();
options.KeepSeparatorsForBlankRow = true; // يحتفظ بالفاصل في الصفوف الفارغة.
```

#### الخطوة 3: حفظ المصنف بتنسيق CSV
احفظ المصنف الخاص بك بالخيارات المحددة في دليل الإخراج المحدد:
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
wb.Save(System.IO.Path.Combine(outputDir, "output.csv"), options);
```
