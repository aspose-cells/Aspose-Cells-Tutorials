## دليل التنفيذ

### تصدير المصنف إلى HTML باستخدام خطوط الشبكة

في هذا القسم، سنركز على تصدير مصنف والتأكد من تضمين خطوط الشبكة في ملف HTML الناتج.

#### تهيئة المصنف وورقة العمل

أولاً، قم بإنشاء ملف جديد `Workbook` الكائن والوصول إلى ورقة العمل الأولى الخاصة به:

```csharp
// إنشاء كائن مصنف جديد
Workbook wb = new Workbook();

// الوصول إلى ورقة العمل الأولى
Worksheet ws = wb.Worksheets[0];
```

#### ملء البيانات للتوضيح

لمحاكاة سيناريو في العالم الحقيقي، دعنا نملأ ورقة العمل ببيانات العينة:

```csharp
// املأ ورقة العمل بقيم صحيحة
for (int r = 0; r < 10; r++) {
    for (int c = 0; c < 10; c++) {
        ws.Cells[r, c].PutValue(r * 1);
    }
}
```

#### تكوين خيارات تصدير HTML

إعداد `HtmlSaveOptions` لتضمين خطوط الشبكة في مخرجات HTML الخاصة بك:

```csharp
// إعداد خيارات حفظ HTML
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.ExportGridLines = true;
```

#### الحفظ بصيغة HTML مع خطوط الشبكة

وأخيرًا، قم بحفظ المصنف كملف HTML باستخدام الخيارات المحددة:

```csharp
// حفظ المصنف في HTML باستخدام خطوط الشبكة
wb.Save("YOUR_OUTPUT_DIRECTORY/outputExportToHTMLWithGridLines.html", opts);
```
