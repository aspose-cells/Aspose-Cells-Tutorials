## دليل التنفيذ

### الميزة 1: تحميل المصنف من دليل المصدر
#### ملخص
إن تحميل مصنف Excel إلى الذاكرة هو الخطوة الأولى في معالجته وتحويله.
##### الخطوة 1: إعداد الدلائل ومسار الملف
```csharp
using System;
using Aspose.Cells;
// تحديد مسارات لدلائل المصدر والإخراج
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string excelFileName = "sampleFitAllWorksheetColumns.xlsx";
// قم بتحميل المصنف من مسار الملف المحدد
Workbook book = new Workbook(SourceDir + "/" + excelFileName);
```
**توضيح**: يستبدل `YOUR_SOURCE_DIRECTORY` مع مسار الدليل الفعلي. هذا المقطع من التعليمات البرمجية يُهيئ `Workbook` الكائن عن طريق تحميل ملف Excel، مما يجعله جاهزًا لمزيد من المعالجة.

### الميزة 2: تكوين خيارات حفظ PDF
#### ملخص
قم بتخصيص كيفية حفظ المصنف بتنسيق PDF لضمان احتواء كافة الأعمدة على صفحة واحدة لكل ورقة.
##### الخطوة 2: إعداد خيارات حفظ ملف Pdf
```csharp
// تهيئة خيارات حفظ ملف Pdf
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.AllColumnsInOnePagePerSheet = true;
```
**توضيح**:بالضبط `AllColumnsInOnePagePerSheet` إذا كنت تريد ضبط القيمة true، فأنت تطلب من Aspose.Cells ضبط عرض الأعمدة بحيث تتناسب جميع الأعمدة مع صفحة واحدة لكل ورقة في ملف PDF.

### الميزة 3: حفظ المصنف بتنسيق PDF باستخدام الخيارات المهيئة
#### ملخص
احفظ المصنف المحمّل في ملف PDF باستخدام الخيارات المكوّنة.
##### الخطوة 3: تحديد الناتج والحفظ
```csharp
using System.IO;
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
string pdfFileName = "outputFitAllWorksheetColumns.pdf";
// احفظ المصنف بتنسيق PDF باستخدام خيارات الحفظ المحددة
book.Save(OutputDir + "/" + pdfFileName, saveOptions);
```
**توضيح**: يستبدل `YOUR_OUTPUT_DIRECTORY` مع مسار الإخراج المطلوب. يحفظ هذا الكود المصنف بتنسيق PDF، مع تطبيق إعدادات لتناسب جميع الأعمدة في صفحة واحدة.
