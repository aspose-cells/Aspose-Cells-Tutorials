## دليل التنفيذ

### الميزة: تحميل المصنف بخيارات محددة

تتيح لك هذه الميزة تحميل مصنف Excel مع استبعاد بيانات الرسم البياني، وبالتالي تحسين عملية التحميل.

#### الخطوة 1: تحديد أدلة المصدر والإخراج

ابدأ بتحديد الدلائل الخاصة بملفات المصدر والإخراج:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: تكوين خيارات التحميل

إنشاء مثيل لـ `LoadOptions` وتعيين مرشح لاستبعاد بيانات الرسم البياني باستخدام العمليات الثنائية:

```csharp
LoadOptions options = new LoadOptions();
options.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.Chart);
```

- **لماذا؟** يضمن هذا التكوين تحميل البيانات الضرورية فقط (باستثناء المخططات)، مما يقلل من استخدام الذاكرة ووقت التحميل.

#### الخطوة 3: تحميل المصنف

استخدم الخيارات المحددة لتحميل المصنف الخاص بك:

```csharp
Workbook workbook = new Workbook(SourceDir + "sampleLoadTemplateWithoutCharts.xlsx", options);
```

- **ماذا يحدث؟** يتم فتح المصنف بقيود محددة، مع تجاهل أي بيانات مخطط مضمنة فيه.

#### الخطوة 4: حفظ المصنف

بعد التحميل، احفظ المصنف بالتنسيق المطلوب، مثل PDF:

```csharp
workbook.Save(OutputDir + "outputLoadTemplateWithoutCharts.pdf", SaveFormat.Pdf);
```

- **فائدة:** تضمن هذه الخطوة إمكانية مشاركة البيانات أو توزيعها بسهولة دون الحاجة إلى معلومات مخطط غير ضرورية.
