## دليل التنفيذ

### نظرة عامة على ميزة تجاهل الأخطاء

الهدف الأساسي هنا هو تحويل ملف Excel إلى PDF مع تجاهل أخطاء العرض. هذا يضمن عدم تعطل عملية التحويل بأكملها بسبب أي مشاكل بسيطة في جدول البيانات.

### تنفيذ خيارات حفظ PDF

دعونا نستعرض كل خطوة من الخطوات المتضمنة في تنفيذ هذه الميزة:

#### الخطوة 1: تحميل المصنف الخاص بك

أولاً، قم بتحميل مصنف Excel الذي تريد تحويله:
```csharp
string sourceDir = "path/to/your/source/directory";
Workbook wb = new Workbook(sourceDir + "sampleErrorExcel2Pdf.xlsx");
```

#### الخطوة 2: تكوين خيارات حفظ PDF

قم بإعداد `PdfSaveOptions` كائن لتجاهل الأخطاء أثناء التحويل:
```csharp
// تحديد خيارات حفظ ملف PDF - تجاهل الخطأ
PdfSaveOptions opts = new PdfSaveOptions();
opts.IgnoreError = true;
```

- **حدود**:
  - `IgnoreError`:علم منطقي، عندما يتم ضبطه على `true`يسمح للمكتبة بتجاوز الأخطاء التي واجهتها أثناء العرض.

#### الخطوة 3: التحويل والحفظ

وأخيرًا، احفظ المصنف الخاص بك بتنسيق PDF باستخدام الخيارات التالية:
```csharp
string outputDir = "path/to/your/output/directory";
wb.Save(outputDir + "outputErrorExcel2Pdf.pdf", opts);
```
