# تحويل Excel إلى HTML مع عرض تقديمي محسّن في .NET باستخدام Aspose.Cells

## دليل التنفيذ

### الخطوة 1: تحميل ملف Excel
ابدأ بتحميل ملف Excel المصدر الخاص بك إلى `Workbook` الكائن. هذه الخطوة بالغة الأهمية لأنها تقوم بإعداد البيانات التي سيتم تحويلها.

```csharp
// حدد الدلائل الخاصة بك
string sourceDir = "path_to_source_directory";
string outputDir = "path_to_output_directory";

// تحميل ملف Excel
Workbook workbook = new Workbook(sourceDir + "sampleUsePresentationPreferenceOption.xlsx");
```

### الخطوة 2: تكوين خيارات حفظ HTML
لتحسين العرض، سنستخدم `HtmlSaveOptions`يتيح لنا هذا تعيين التفضيلات التي تعمل على تحسين طريقة عرض البيانات في HTML.

```csharp
// إنشاء كائن HtmlSaveOptions
HtmlSaveOptions options = new HtmlSaveOptions();

// تعيين خيار تفضيل العرض التقديمي
options.PresentationPreference = true;
```

### الخطوة 3: حفظ ملف Excel بصيغة HTML
وأخيرًا، احفظ مصنفك في ملف HTML مع إعدادات عرض تقديمي محسّنة.

```csharp
// حفظ ملف Excel إلى HTML باستخدام الخيار المحدد
workbook.Save(outputDir + "outputUsePresentationPreferenceOption.html", options);

Console.WriteLine("Conversion executed successfully.");
```
