## دليل التنفيذ

يقوم هذا القسم بتقسيم العملية إلى ميزتين رئيسيتين: تحميل ملف HTML وتكوين خيارات التحميل.

### الميزة 1: تحميل ملف HTML وحفظه كملف Excel

#### ملخص:
قم بتحويل ملف HTML الموجود إلى مصنف Excel يعمل بكامل طاقته باستخدام Aspose.Cells لـ .NET. 

##### التنفيذ خطوة بخطوة:

**1. تعيين أدلة المصدر والإخراج:**
ابدأ بتحديد الدلائل التي يوجد بها ملف HTML المصدر والمكان الذي تريد حفظ ملف Excel الناتج فيه.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**2. حدد المسار إلى ملف HTML الخاص بك:**
قم بإنشاء مسار لملف HTML المصدر الخاص بك باستخدام `System.IO.Path.Combine`.

```csharp
string filePath = System.IO.Path.Combine(SourceDir, "Book1.html");
```

**3. تكوين خيارات التحميل:**
إنشاء مثيل `HtmlLoadOptions` الصف مع `LoadFormat.Html`تشير هذه الخطوة إلى أنك تقوم بتحميل مستند HTML.

```csharp
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
```

**4. إنشاء كائن مصنف:**
استخدم `Workbook` منشئ لفتح ملفك باستخدام المسار المحدد وخيارات التحميل.

```csharp
Workbook wb = new Workbook(filePath, loadOptions);
```

**5. حفظ كملف Excel:**
وأخيرًا، احفظ المصنف في دليل الإخراج المطلوب.

```csharp
string outputFilePath = System.IO.Path.Combine(outputDir, "output.xlsx");
wb.Save(outputFilePath);
```

### الميزة 2: تكوين خيارات التحميل لملفات HTML

#### ملخص:
تعرف على كيفية تعديل خيارات التحميل لتخصيص كيفية التعامل مع ملف HTML عند تحويله إلى مصنف Excel.

##### التنفيذ خطوة بخطوة:

**1. تعيين دليل المصدر:**
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
```

**2. قم بتحديد خيارات المسار والتحميل باستخدام التكوينات:**
استخدم نفس إعداد المسار كما في السابق ولكن قم بتكوين خيارات تحميل إضافية إذا لزم الأمر، مثل التعرف على محتوى HTML كمصنف كامل.

```csharp
string filePath = System.IO.Path.Combine(SourceDir, "Book1.html");
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
loadOptions.IsRecognizeAsSingleFile = true;  // خيار التكوين المثال
```

**3. إنشاء مصنف وحفظه:**
قم بإنشاء مصنف باستخدام هذه الخيارات المكوّنة ثم احفظه.

```csharp
Workbook wb = new Workbook(filePath, loadOptions);
string outputFilePath = System.IO.Path.Combine(SourceDir, "output.xlsx");
b.Save(outputFilePath);
```

#### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من تحديد مسار ملف HTML الخاص بك بشكل صحيح.
- تحقق من وجود أي مشكلات متعلقة بالترخيص قد تؤثر على عملية التحميل.
