## دليل التنفيذ
سنستكشف كيفية تحميل المصنفات بإعدادات محددة والوصول إلى خصائصها.

### تحميل المصنفات بإعدادات محددة
توضح هذه الميزة تحميل مصنفات Excel باستخدام Aspose.Cells، مع التركيز على `PageSetup.IsAutomaticPaperSize` ملكية.

#### ملخص
قم بتحميل مصنفين مختلفين - أحدهما تم تعيين حجم الورق التلقائي فيه على false والآخر تم تعيينه على true - ثم قم بالوصول إلى خصائص PageSetup الخاصة بهما.

#### التنفيذ خطوة بخطوة
1. **تحميل مصنف مع تعيين حجم الورق التلقائي على "خطأ"**
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   // قم بتحميل المصنف حيث تم تعيين حجم الورق التلقائي على "خطأ"
   Workbook wb1 = new Workbook(SourceDir + "/samplePageSetupIsAutomaticPaperSize-False.xlsx");

   // الوصول إلى ورقة العمل الأولى
   Worksheet ws11 = wb1.Worksheets[0];

   // طباعة خاصية IsAutomaticPaperSize
   Console.WriteLine("First Worksheet of First Workbook - IsAutomaticPaperSize: " + ws11.PageSetup.IsAutomaticPaperSize);
   ```
2. **تحميل مصنف مع تعيين حجم الورق التلقائي على "صحيح"**
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   // قم بتحميل المصنف حيث تم تعيين حجم الورق التلقائي على "صحيح"
   Workbook wb2 = new Workbook(SourceDir + "/samplePageSetupIsAutomaticPaperSize-True.xlsx");

   // الوصول إلى ورقة العمل الأولى
   Worksheet ws12 = wb2.Worksheets[0];

   // طباعة خاصية IsAutomaticPaperSize
   Console.WriteLine("First Worksheet of Second Workbook - IsAutomaticPaperSize: " + ws12.PageSetup.IsAutomaticPaperSize);
   ```

#### توضيح
- **حدود**: ال `Workbook` يقوم المنشئ بأخذ مسار الملف لتحميل مصنف Excel.
- **قيم الإرجاع**: ال `PageSetup.IsAutomaticPaperSize` تعيد الخاصية قيمة منطقية تشير إلى ما إذا كان حجم الورق مضبوطًا تلقائيًا أم لا.
