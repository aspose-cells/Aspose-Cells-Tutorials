### عملية التحويل خطوة بخطوة
**1. افتح ملف القالب**
ابدأ بتحميل ملف Excel المصدر الخاص بك:
```csharp
// مسار دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// تهيئة كائن مصنف باستخدام مسار ملف Excel
Workbook workbook = new Workbook(sourceDir + "Book1.xlsx");
```
*توضيح*: ال `Workbook` تُمثل الفئة ملف Excel. بتمرير مسار الملف إلى مُنشئه، تُحمّل جدول البيانات إلى الذاكرة.

**2. احفظ بصيغة DOCX**
تحويل ملف Excel المحمّل وحفظه كمستند Word:
```csharp
// مسار دليل الإخراج
string outputDir = RunExamples.Get_OutputDirectory();

// احفظ المصنف بتنسيق DOCX
workbook.Save(outputDir + "Book1.docx", SaveFormat.Docx);
```
*توضيح*: ال `Save` تُصدّر الطريقة مصنف العمل الحالي إلى تنسيقات مختلفة. هنا، `SaveFormat.Docx` يحدد أننا نريد مستند Word.
