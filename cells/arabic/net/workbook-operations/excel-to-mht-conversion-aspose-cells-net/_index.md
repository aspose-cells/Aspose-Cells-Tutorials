## دليل التنفيذ

### الخطوة 1: تحديد مسارات الملفات
قم بتحديد المسارات لملف Excel المصدر وملف MHT الناتج.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

string filePath = SourceDir + "/Book1.xlsx"; // مسار ملف Excel المدخل
string outputPath = outputDir + "/Book1.out.mht"; // مسار ملف MHT الناتج
```

### الخطوة 2: تكوين خيارات حفظ HTML
قم بتكوين خيارات الحفظ لتحويل ملف Excel إلى تنسيق MHTML.
```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.MHTML);
```
ال `HtmlSaveOptions` توفر الفئة تكوينات لحفظ المصنفات بتنسيقات HTML. الإعداد `SaveFormat.MHTML` يجمع كل الموارد (الصور، CSS) في ملف واحد.

### الخطوة 3: تحميل مصنف Excel
قم بتحميل مصنف Excel الخاص بك باستخدام المسار المحدد مسبقًا.
```csharp
Workbook workbook = new Workbook(filePath);
```
ال `Workbook` تُمثِّل الفئة في Aspose.Cells مستند Excel كاملاً. يسمح تحميلها بمعالجة البيانات داخلها.

### الخطوة 4: حفظ باسم MHT
احفظ المصنف في مسار الإخراج المطلوب باستخدام الخيارات التي تم تكوينها.
```csharp
workbook.save(outputPath, saveOptions);
```
تؤدي هذه الخطوة إلى تحويل ملف Excel وحفظه بتنسيق MHTML، مع الحفاظ على تخطيطه وأسلوبه للاستخدام على الويب.

### نصائح استكشاف الأخطاء وإصلاحها
