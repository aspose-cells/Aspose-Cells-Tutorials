## دليل التنفيذ

### تحويل المصنف إلى PDF/A

#### ملخص

في هذا القسم، سنوضح كيفية تحويل مصنف Excel إلى ملف PDF متوافق. سنركز على ضبط مستوى التوافق إلى PDF/A-1b للأرشفة طويلة الأمد.

#### التنفيذ خطوة بخطوة

**الخطوة 1: إنشاء المصنف وتعبئته**

ابدأ بإنشاء مثيل لـ `Workbook` الفئة التي تمثل ملف Excel الخاص بك:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // استبدله بدليل المصدر الخاص بك
string outputDir = @"YOUR_OUTPUT_DIRECTORY"; // استبدله بدليل الإخراج المطلوب

// إنشاء كائن مصنف جديد
Workbook workbook = new Workbook();

// إدراج قيمة في الخلية A1 من ورقة العمل الأولى
workbook.Worksheets[0].Cells[0, 0].PutValue("Testing PDF/A");
```

**الخطوة 2: تكوين خيارات حفظ PDF**

بعد ذلك، قم بتكوين خيارات الحفظ لتحديد إعدادات التوافق:

```csharp
using Aspose.Cells.Rendering;

// إنشاء مثيل لـ PdfSaveOptions
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

// تعيين مستوى التوافق لملف PDF الناتج
pdfSaveOptions.Compliance = PdfCompliance.PdfA1b;
```

**الخطوة 3: الحفظ بتنسيق PDF**

وأخيرًا، احفظ المصنف الخاص بك في ملف PDF باستخدام الخيارات التالية:

```csharp
// حفظ المصنف بتنسيق PDF مع الخيارات المحددة
workbook.Save(outputDir + "/output.pdf", pdfSaveOptions);
```
