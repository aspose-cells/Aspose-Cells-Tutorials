## دليل التنفيذ

### تحميل ملف Excel
**ملخص**:تعرف على كيفية تحميل ملف Excel موجود في `Workbook` الكائن. وهذا ضروري لمعالجة البيانات المخزنة في ملفات Excel ومعالجتها.

#### التنفيذ خطوة بخطوة
1. **إعداد دليل المصدر**
   قم بتحديد مكان وجود ملف Excel المصدر الخاص بك:
   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   ```
2. **تحميل ملف Excel**
   تهيئة `Workbook` الكائن عن طريق تحميل ملف Excel موجود:
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "sampleExpandTextFromRightToLeft.xlsx");
   ```
   - **توضيح**: ال `Workbook` يقوم المنشئ بأخذ مسار الملف وتحميله في الذاكرة، مما يسمح لك بالتفاعل مع محتوياته برمجيًا.

### حفظ المصنف بصيغة HTML
**ملخص**يوضح هذا القسم كيفية حفظ مصنف Excel بتنسيق HTML. يُعدّ هذا القسم مفيدًا بشكل خاص لتطبيقات الويب التي تتطلب عرض البيانات خارج برنامج Excel.

#### التنفيذ خطوة بخطوة
1. **إعداد دليل الإخراج**
   حدد المكان الذي تريد حفظ ملف HTML الناتج فيه:
   ```csharp
   string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
   ```
2. **حفظ المصنف بتنسيق HTML**
   احفظ المصنف مع معلومات الإصدار المرفقة باسم الملف:
   ```csharp
   workbook.Save(OutputDir + "outputExpandTextFromRightToLeft_" + CellsHelper.GetVersion() + ".html", SaveFormat.Html);
   ```
   - **توضيح**: ال `Save` تكتب الطريقة محتوى المصنف في ملف HTML. `CellsHelper.GetVersion()` يوفر تفاصيل الإصدار للتمييز بين المخرجات المختلفة.
