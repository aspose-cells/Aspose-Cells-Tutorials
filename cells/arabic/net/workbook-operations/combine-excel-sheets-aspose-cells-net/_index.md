## دليل التنفيذ

دعونا نقسم العملية إلى خطوات مميزة من أجل الوضوح.

### تحميل المصنف

#### ملخص
تحميل مصنف Excel من دليل محدد.

#### خطوات التنفيذ
1. **تعيين دليل المصدر**
   قم بتحديد المسار الذي يتواجد فيه ملف Excel الخاص بك.
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   ```
2. **تحميل المصنف**
   إنشاء جديد `Workbook` كائن لتحميل ملف Excel الخاص بك.
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "/book1.xls");
   ```

### تهيئة خيارات حفظ النص

#### ملخص
قم بتكوين كيفية حفظ كل ورقة عمل بتنسيق نصي، باستخدام القيم المنفصلة بعلامات التبويب (TSV).

#### خطوات التنفيذ
1. **إنشاء خيارات حفظ النص**
   إنشاء مثيل `TxtSaveOptions` لتحديد الفاصل.
   ```csharp
   TxtSaveOptions opts = new TxtSaveOptions();
   opts.Separator = '\t'; // استخدم علامة التبويب كفاصل لتنسيق TSV
   ```

### تحويل ودمج أوراق العمل إلى تنسيق نصي

#### ملخص
قم بتحويل كل ورقة عمل إلى تنسيق نصي ودمجها في مصفوفة بايت واحدة.

#### خطوات التنفيذ
1. **تهيئة مجموعة البايتات**
   قم بإعداد مجموعة بايتات فارغة لحمل البيانات المجمعة من جميع أوراق العمل.
   ```csharp
   byte[] workbookData = new byte[0];
   ```
2. **التكرار من خلال أوراق العمل**
   قم بالمرور على كل ورقة عمل، وحفظها كنص ودمج الناتج.
   ```csharp
   for (int idx = 0; idx < workbook.Worksheets.Count; idx++) {
       workbook.Worksheets.ActiveSheetIndex = idx;
       
       using (MemoryStream ms = new MemoryStream()) {
           workbook.Save(ms, opts);
           
           ms.Position = 0;
           byte[] sheetData = ms.ToArray();
           
           byte[] combinedArray = new byte[workbookData.Length + sheetData.Length];
           Array.Copy(workbookData, 0, combinedArray, 0, workbookData.Length);
           Array.Copy(sheetData, 0, combinedArray, workbookData.Length, sheetData.Length);
           
           workbookData = combinedArray;
       }
   }
   ```

### حفظ بيانات المصنف المجمعة في ملف

#### ملخص
احفظ بيانات النص المجمعة من كافة أوراق العمل في ملف واحد.

#### خطوات التنفيذ
1. **تعيين دليل الإخراج**
   قم بتحديد المكان الذي سيتم حفظ ملف النص الناتج فيه.
   ```csharp
   string OutputDir = "YOUR_OUTPUT_DIRECTORY";
   ```
2. **الكتابة إلى الملف**
   يستخدم `File.WriteAllBytes` لحفظ مجموعة البايتات كملف `.txt` ملف.
   ```csharp
   File.WriteAllBytes(OutputDir + "/out.txt", workbookData);
   ```
