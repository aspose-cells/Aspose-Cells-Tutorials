## دليل التنفيذ

### نظرة عامة على ميزة التحويل

يتضمن تحويل JSON إلى CSV باستخدام Aspose.Cells قراءة ملف JSON واستيراد بياناته إلى مصنف Excel قبل حفظه كملف CSV. تضمن هذه العملية الحفاظ على الهيكل الهرمي لـ JSON بتنسيق مسطح يشبه الجدول.

#### الخطوة 1: قراءة ملف JSON

```csharp
// دليل المصدر الذي يوجد فيه ملف JSON الخاص بك
string sourceDir = RunExamples.Get_SourceDirectory();
string jsonFilePath = sourceDir + "SampleJson.json";

// قراءة محتوى ملف JSON
string jsonString = File.ReadAllText(jsonFilePath);
```

هنا، `File.ReadAllText` يقرأ محتوى JSON كاملاً إلى سلسلة نصية. هذه هي خطوتنا الأولى نحو التحويل.

#### الخطوة 2: إنشاء مصنف وتكوينه

```csharp
// تهيئة مصنف فارغ
Workbook workbook = new Workbook();

// الوصول إلى مجموعة خلايا ورقة العمل الأولى
Cells cells = workbook.Worksheets[0].Cells;

// تكوين JsonLayoutOptions لإعدادات الاستيراد
JsonLayoutOptions options = new JsonLayoutOptions
{
    ConvertNumericOrDate = true,
    ArrayAsTable = true,
    IgnoreArrayTitle = true,
    IgnoreObjectTitle = true
};
```

ال `JsonLayoutOptions` توفر الفئة إعدادات متنوعة لتخصيص عملية التحويل. على سبيل المثال، `ConvertNumericOrDate` يضمن تفسير القيم الرقمية والتاريخية بشكل صحيح.

#### الخطوة 3: استيراد بيانات JSON

```csharp
// استيراد البيانات من سلسلة JSON إلى خلايا المصنف بدءًا من الصف 0 والعمود 0
JsonUtility.ImportData(jsonString, cells, 0, 0, options);
```

`JsonUtility.ImportData` تقوم الطريقة باستيراد بيانات JSON إلى ورقة العمل ونطاق الخلايا المحددين باستخدام التكوين المقدم.

#### الخطوة 4: الحفظ بتنسيق CSV

```csharp
// تحديد دليل الإخراج لحفظ ملف CSV
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "SampleJson_out.csv");
```

وأخيرًا، احفظ المصنف الخاص بك بتنسيق CSV. `Save` الطريقة متعددة الاستخدامات وتدعم تنسيقات مختلفة بما في ذلك CSV.

### نصائح استكشاف الأخطاء وإصلاحها

- **لم يتم العثور على الملف**:تأكد من أن المسار إلى ملف JSON الخاص بك صحيح.
