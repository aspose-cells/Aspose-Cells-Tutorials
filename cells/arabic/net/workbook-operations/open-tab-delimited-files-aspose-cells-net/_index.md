   ```csharp

   ```

2. **إنشاء مصنف وفتحه**
   استخدم خيارات التحميل المحددة لإنشاء `Workbook` هدف.
   ```csharp
   string dataDir = "path_to_your_directory"; // تحديث هذا المسار

   // إنشاء مصنف بملف مفصول بعلامات التبويب
   ```

- **تنسيق التحميل.مفصول بعلامات تبويب**: يشير إلى تنسيق ملف الإدخال.
