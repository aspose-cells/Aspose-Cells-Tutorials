## دليل التنفيذ

اتبع الخطوات التالية لتحويل ملف XLSB إلى XLSM باستخدام Aspose.Cells.

### فتح وتحميل المصنف

**ملخص:** قم بتحميل ملف XLSB الحالي الخاص بك في الذاكرة عن طريق تحديد دليل المصدر.

#### الخطوة 1: تعيين دليل المصدر
```csharp
// استبدل 'YOUR_SOURCE_DIRECTORY' بالمسار الفعلي الذي توجد به ملفات المصدر الخاصة بك.
string sourceDir = "YOUR_SOURCE_DIRECTORY";
```

#### الخطوة 2: تحديد مسار المصنف وتحميله
```csharp
// حدد مسار الملف الخاص بمصنف XLSB الذي تريد تحويله.
string filePath = Path.Combine(sourceDir, "sampleConvertXLSBToXLSM.xlsb");
// قم بتحميل المصنف من المسار المحدد.
Workbook workbook = new Workbook(filePath);
```

### حفظ المصنف بتنسيق مختلف

**ملخص:** بعد تحميل المصنف الخاص بك، قم بحفظه كملف XLSM.

#### الخطوة 1: تعيين دليل الإخراج
```csharp
// استبدل 'YOUR_OUTPUT_DIRECTORY' بالمسار الفعلي الذي تريد حفظ ملفات الإخراج فيه.
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: حفظ المصنف بتنسيق XLSM
```csharp
// قم بتحديد اسم الملف الجديد والمسار لمصنف XLSM.
string outputFilePath = Path.Combine(outputDir, "outputConvertXLSBToXLSM.xlsm");
// احفظ المصنف بتنسيق XLSM.
workbook.Save(outputFilePath, SaveFormat.Xlsm);
```
