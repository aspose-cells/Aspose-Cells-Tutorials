# تحويل مصنفات Excel إلى ملفات PDF مصممة باستخدام Aspose.Cells لـ .NET

## مقدمة

قد يكون تحويل مصنفات Excel إلى PDF مع الحفاظ على سلامة عرضها التقديمي أمرًا صعبًا، خاصةً عند الحفاظ على الخطوط والأنماط والأحرف الخاصة. يوضح هذا الدليل الشامل كيفية استخدام **Aspose.Cells لـ .NET** لإنشاء مصنف Excel وتنسيقه قبل تحويله إلى مستند PDF باستخدام خيارات تنسيق محددة.

### ما سوف تتعلمه

- إنشاء وتصميم مصنف Excel باستخدام C#.
- حفظ مصنفات Excel بتنسيق PDF، مع أو بدون خيارات استبدال الخط.

دعونا نراجع المتطلبات الأساسية قبل البدء!

## دليل التنفيذ
سنقوم بتقسيم التنفيذ إلى ميزات رئيسية من أجل الوضوح.

### الميزة 1: إنشاء وتنسيق مصنف العمل
توضح هذه الميزة كيفية إنشاء مصنف Excel والوصول إلى ورقة العمل الخاصة به وتطبيق أنماط الخطوط باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تهيئة المصنف
ابدأ بإنشاء حساب جديد `Workbook` هدف:
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

#### الخطوة 2: الوصول إلى خلايا ورقة العمل وتنسيقها
قم بالوصول إلى ورقة العمل الأولى، واسترجاع الخلايا، وتطبيق الأنماط:
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];

// الوصول إلى الخلايا A1 وB1
Cell cell1 = worksheet.Cells["A1"];
Cell cell2 = worksheet.Cells["B1"];

// تعيين نمط الخط إلى Times New Roman لكلا الخليتين
Style style = cell1.GetStyle();
style.Font.Name = "Times New Roman";
cell1.SetStyle(style);
cell2.SetStyle(style);

// إضافة القيم، بما في ذلك الواصلة غير المنقطعة في B1
cell1.PutValue("Hello without Non-Breaking Hyphen");
cell2.PutValue("Hello" + Convert.ToChar(8209) + " with Non-Breaking Hyphen");

// ضبط الأعمدة تلقائيًا لتناسب حجم المحتوى
worksheet.AutoFitColumns();
```
**الاعتبارات الرئيسية**: 
- **تصميم الخط**:تعمل إعدادات الخط الصحيحة على تعزيز إمكانية القراءة والاحترافية.
- **الأحرف غير القابلة للكسر**: يستخدم `Convert.ToChar(8209)` للواصلات غير القابلة للكسر، ومنع انقطاع الأسطر في النقاط غير المرغوب فيها.

### الميزة 2: حفظ المصنف بصيغة PDF دون خيارات استبدال الأحرف
يوضح هذا القسم كيفية حفظ مصنف Excel بتنسيق PDF دون خيارات استبدال الخط.
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();

// حفظ المصنف بصيغة PDF
workbook.Save(outputDir + "/SampleOutput_out.pdf");
```
**توضيح**:تحافظ هذه الطريقة على الخطوط الأصلية عندما يكون ذلك ممكنًا، وهي مثالية للمستندات التي تتطلب سلامة الخطوط.

### الميزة 3: حفظ المصنف بصيغة PDF مع خيارات استبدال الأحرف
لمزيد من التحكم في استبدال الخط أثناء التحويل:
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();

// تهيئة PdfSaveOptions مع تمكين حبيبات استبدال الخط
PdfSaveOptions opts = new PdfSaveOptions();
opts.IsFontSubstitutionCharGranularity = true;

// احفظ المصنف بصيغة PDF باستخدام هذه الخيارات
workbook.Save(outputDir + "/SampleOutput2_out.pdf", opts);
```
**تكوين المفتاح**:تمكين `IsFontSubstitutionCharGranularity` يسمح بالتحكم الدقيق في استبدال الخط، وهو أمر ضروري للمستندات التي تحتاج إلى تمثيل أحرف محددة.

### نصائح استكشاف الأخطاء وإصلاحها
- **الخطوط المفقودة**:تأكد من تثبيت كافة خطوط المصنف على نظامك.
- **مسارات غير صحيحة**:تأكد من وجود أدلة المصدر والإخراج مع الأذونات المناسبة.
