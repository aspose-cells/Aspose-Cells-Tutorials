### التعامل مع ملفات المصنف
#### فتح وإغلاق مصنف Excel
**ملخص:**
تعلم كيفية فتح مصنف Excel الحالي باستخدام تدفق الملفات لإدارة الموارد بكفاءة.

```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// إنشاء مجرى ملف يحتوي على ملف Excel الذي سيتم فتحه
using (FileStream fstream = new FileStream(sourceDir + "/book1.xls", FileMode.Open))
{
    // إنشاء كائن مصنف عن طريق فتح ملف Excel من خلال مجرى الملف
    Workbook workbook = new Workbook(fstream);
    // تضمن عبارة الاستخدام تحرير الموارد بعد الاستخدام.
}
```
**توضيح:**
- **تدفق الملفات:** إدارة عمليات الملفات، وضمان فتح ملف Excel بشكل آمن وفعال.
- **كائن المصنف:** يمثل مستند Excel المفتوح لإجراء عمليات مختلفة.

#### فك تجميع الصفوف والأعمدة
**ملخص:**
اكتشف كيفية إلغاء تجميع صفوف وأعمدة محددة في ورقة عمل Excel لتنظيم البيانات بشكل مرن.

```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مصنف من ملف المصدر
Workbook workbook = new Workbook(sourceDir + "/book1.xls");

// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];

// إلغاء تجميع أول ستة صفوف (من 0 إلى 5)
worksheet.Cells.UngroupRows(0, 5);

// إلغاء تجميع الأعمدة الثلاثة الأولى (من 0 إلى 2)
worksheet.Cells.UngroupColumns(0, 2);

// حفظ ملف Excel المعدّل في دليل الإخراج
workbook.Save(outputDir + "/output.xls");
```
**توضيح:**
- **طرق UngroupRows/UngroupColumns:** تعديل بنية ورقة العمل عن طريق عكس عمليات التجميع.
- **حفظ التغييرات:** تأكد من حفظ التغييرات عن طريق حفظ المصنف بعد التعديل.
