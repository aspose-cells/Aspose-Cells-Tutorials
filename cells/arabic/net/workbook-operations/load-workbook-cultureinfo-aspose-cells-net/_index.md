# كيفية تحميل مصنف بتنسيق رقمي محدد في CultureInfo باستخدام Aspose.Cells .NET

## مقدمة

هل واجهتَ مشاكل عند تحميل ملفات Excel بسبب تنسيق الأرقام الإقليمي؟ يُعالج هذا البرنامج التعليمي هذه المشكلة من خلال شرح كيفية استخدام Aspose.Cells لـ .NET لتحميل المصنفات مع مراعاة إعدادات الثقافة المحددة. سواءً كنتَ تتعامل مع أرقام بتنسيقات مختلفة عبر المناطق، سيُوضح لك هذا الدليل كيفية إدارة هذه الاختلافات بسلاسة.

في هذه المقالة، سنتعمق في تحميل ملفات Excel باستخدام أداة مخصصة `CultureInfo` تنسيق الأرقام في C#. ستتعلم أساسيات إعداد Aspose.Cells لـ .NET وتهيئته للتعامل مع التنسيق الإقليمي بفعالية. بنهاية هذا البرنامج التعليمي، ستكون قد أتقنت:

- تحميل المصنفات ذات التنسيقات الخاصة بالمنطقة
- تكوين CultureInfo لتحليل البيانات بدقة
- استخدام LoadOptions في Aspose.Cells

## دليل التنفيذ

### نظرة عامة على تحميل المصنفات باستخدام Custom CultureInfo

في هذا القسم، سنركز على تحميل مصنف مع مراعاة معلومات الثقافة الخاصة بتنسيقات الأرقام. يُعد هذا مفيدًا بشكل خاص عند التعامل مع بيانات دولية تتبع قواعد تنسيق إقليمية مختلفة.

#### التنفيذ خطوة بخطوة

##### إعداد معلومات الثقافة
أولاً، قم بإنشاء وتكوين `CultureInfo` كائن يطابق الإعدادات المطلوبة:

```csharp
var culture = new CultureInfo("en-GB");
culture.NumberFormat.NumberDecimalSeparator = ",";
culture.DateTimeFormat.DateSeparator = "-";
culture.DateTimeFormat.ShortDatePattern = "dd-MM-yyyy";
```

هنا، نحدد أن الأرقام يجب أن تستخدم الفاصلة كفاصل عشري وتعديل تنسيقات التاريخ وفقًا لذلك.

##### تكوين LoadOptions
بعد ذلك، قم بتكوين `LoadOptions` للاستفادة من هذه المعلومات الثقافية:

```csharp
LoadOptions options = new LoadOptions(LoadFormat.Html);
options.CultureInfo = culture;
```

تضمن هذه الخطوة أن يقوم Aspose.Cells بقراءة بياناتك باستخدام الإعدادات الثقافية المحددة.

##### تحميل المصنف
أخيرًا، قم بتحميل المصنف الخاص بك باستخدام الخيارات المكوّنة التالية:

```csharp
using (var workbook = new Workbook(inputStream, options))
{
    var cell = workbook.Worksheets[0].Cells["A1"];
    Assert.AreEqual(CellValueType.IsNumeric, cell.Type);
    Assert.AreEqual(1234.56, cell.DoubleValue);
}
```

يوضح مقتطف التعليمات البرمجية هذا كيفية قراءة قيمة رقمية منسقة بالثقافة المحددة.

##### نصائح استكشاف الأخطاء وإصلاحها
- **تأكد من صحة سلاسل الثقافة**:تحقق مرة أخرى من `CultureInfo` سلاسل لتتناسب مع المعايير الإقليمية.
- **التحقق من صحة تنسيقات الملفات**:تأكد من أن ملفات الإدخال موجودة في تنسيقات مدعومة مثل HTML أو Excel.
