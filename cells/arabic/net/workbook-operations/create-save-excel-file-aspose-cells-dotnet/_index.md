## دليل التنفيذ

اتبع الخطوات التالية لإنشاء ملف Excel وتخصيصه:

### إنشاء مصنف جديد
قم بإنشاء مصنف Excel جديد على النحو التالي:
```csharp
// إنشاء مثيل لـ Workbook (ملف Excel)
Workbook workbook = new Workbook();
```

### إزالة ورقة العمل الافتراضية
قم بإزالة ورقة العمل الافتراضية إذا لم تكن هناك حاجة إليها:
```csharp
// إزالة ورقة العمل الافتراضية التي يتم إنشاؤها عند إنشاء مصنف جديد
workbook.Worksheets.RemoveAt(0);
```

### إضافة وتسمية أوراق متعددة
أضف خمس أوراق عمل إلى المصنف الخاص بك وقم بتسميتها بالتسلسل.
```csharp
// أضف 5 أوراق عمل وقم بتسميتها
for (int i = 0; i < 5; i++) {
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = "Sheet" + (i + 1).ToString();
}
```

### ملء الأوراق بالبيانات
قم بملء كل ورقة عمل بالبيانات في شبكة.
```csharp
// ملء الأوراق بالبيانات
for (int i = 0; i < workbook.Worksheets.Count; i++) {
    Worksheet ws = workbook.Worksheets[i];
    for (int row = 0; row < 150; row++) {
        for (int col = 0; col < 56; col++) {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

### حفظ المصنف
احفظ المصنف الخاص بك في الدليل المحدد.
```csharp
// حفظ المصنف
string outputFilePath = System.IO.Path.Combine(outputDir, "ACellsSample_out.xlsx");
workbook.Save(outputFilePath);
```
