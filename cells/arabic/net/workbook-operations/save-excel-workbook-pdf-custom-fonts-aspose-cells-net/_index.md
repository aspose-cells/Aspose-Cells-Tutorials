## دليل التنفيذ
الآن، دعنا نستعرض عملية حفظ مصنف بتنسيق PDF باستخدام خطوط مخصصة خطوة بخطوة.

### حفظ المصنف بصيغة PDF باستخدام الخطوط المخصصة
تتيح لك هذه الميزة تخصيص طريقة عرض مصنفات Excel الخاصة بك في ملفات PDF من خلال تحديد إعدادات الخطوط لكل منها. هذا يضمن ظهور جميع الخطوط المستخدمة في مستندك بشكل صحيح في ملف الإخراج.

#### تكوين إعدادات الخط المخصص
أولاً، قم بإعداد دليل للخطوط المخصصة وتكوين Aspose.Cells لاستخدام هذه الخطوط:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

IndividualFontConfigs fontConfigs = new IndividualFontConfigs();
fontConfigs.SetFontFolder(SourceDir + "/CustomFonts", false); // قم بتكوين المجلد الذي سيتم تخزين الخطوط المخصصة فيه.
```
#### خيارات التحميل باستخدام الخطوط المخصصة
قم بتطبيق هذه التكوينات لتحميل الخيارات عند فتح مصنف:
```csharp
LoadOptions opts = new LoadOptions(LoadFormat.Xlsx);
opts.FontConfigs = fontConfigs; // تعيين إعدادات الخط المكوّنة لخيارات التحميل.

Workbook wb = new Workbook(SourceDir + "/sampleSpecifyIndividualOrPrivateSetOfFontsForWorkbookRendering.xlsx", opts); // قم بتحميل ملف Excel الخاص بك باستخدام الخطوط المخصصة.
```
#### حفظ كملف PDF
أخيرًا، احفظ المصنف المحمّل بتنسيق PDF مع التأكد من استخدام كافة الخطوط المحددة:
```csharp
wb.Save(outputDir + "/outputSpecifyIndividualOrPrivateSetOfFontsForWorkbookRendering.pdf", SaveFormat.Pdf);
```
**نصائح استكشاف الأخطاء وإصلاحها**:إذا لم تظهر الخطوط المخصصة بشكل صحيح:
- تأكد من أن ملفات الخطوط موجودة في التنسيقات المدعومة (على سبيل المثال، .ttf، .otf).
- تأكد من أن المسار إلى دليل الخط المخصص لديك صحيح.
