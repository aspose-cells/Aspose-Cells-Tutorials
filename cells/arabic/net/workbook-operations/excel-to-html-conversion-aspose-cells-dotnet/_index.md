# تحويل Excel إلى HTML باستخدام Aspose.Cells .NET

## دليل التنفيذ

### الميزة 1: تحميل وحفظ مصنف Excel

توضح هذه الميزة كيفية تحميل مصنف Excel من دليل مصدر محدد وحفظه بتنسيق HTML مع خيارات مخصصة.

#### ملخص
يضمن تحميل مصنفات العمل وحفظها بكفاءة تبادل البيانات بشكل سلس بين التطبيقات بتنسيقات مختلفة.

#### خطوات:

**الخطوة 1**:قم بتحديد دليل المصدر والإخراج الخاص بك.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**الخطوة 2**:قم بتحميل مصنف Excel باستخدام Aspose.Cells.
```csharp
// تحميل مصنف موجود من ملف
Workbook workbook = new Workbook(SourceDir + "sampleChangeHtmlLinkTarget.xlsx");
```
*توضيح*: ال `Workbook` يتم استخدام الفئة لتحميل ملفات Excel ومعالجتها.

**الخطوة 3**:قم بتكوين خيارات حفظ HTML مع أهداف ارتباط محددة.
```csharp
// قم بتهيئة HtmlSaveOptions وتعيين LinkTargetType
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.LinkTargetType = HtmlLinkTargetType.Self; // تفتح الروابط في نفس النافذة/علامة التبويب
```
*تكوين المفتاح*: `HtmlLinkTargetType.Self` يضمن فتح جميع الروابط الموجودة داخل ملف HTML في علامة تبويب المتصفح الحالية.

**الخطوة 4**:احفظ المصنف كملف HTML.
```csharp
// حفظ المصنف باستخدام خيارات HTML المحددة
workbook.Save(OutputDir + "outputChangeHtmlLinkTarget.html", opts);
```
*غاية*: ال `Save` تكتب الطريقة المصنف إلى تنسيق محدد، في هذه الحالة، HTML.

### الميزة 2: تكوين خيارات حفظ HTML

ترتكز هذه الميزة على تخصيص إعدادات حفظ HTML لمصنف Excel.

#### ملخص
يتيح تخصيص خيارات الحفظ الحصول على إخراج مخصص يلبي متطلبات التطبيق المحددة.

#### خطوات:

**الخطوة 1**:إنشاء وتكوين `HtmlSaveOptions`.
```csharp
// إنشاء مثيل HtmlSaveOptions
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.LinkTargetType = HtmlLinkTargetType.Self;
```
*توضيح*:ضبط خيارات حفظ HTML مثل `LinkTargetType` يتيح لك التحكم في كيفية عرض بياناتك في المتصفح.

**الخطوة 2**:حفظ مع الخيارات المكوّنة.
```csharp
// على افتراض أن المصنف تم تحميله بالفعل باسم "مصنف"
workbook.Save(OutputDir + "outputChangeHtmlLinkTarget.html", opts);
```
