## دليل التنفيذ

سنقوم بتقسيم كل ميزة إلى خطوات مميزة لمساعدتك على تنفيذها بشكل فعال.

### فتح ملف Excel وحفظه

يوضح هذا القسم كيفية فتح ملف Excel موجود وحفظه بعد إجراء التعديلات عليه.

**ملخص:**

فتح ملف هو الخطوة الأولى لمعالجة أي بيانات داخل Excel. بعد تعديله، يضمن الحفظ الحفاظ على جميع التغييرات.

**خطوات التنفيذ:**

1. **فتح ملف Excel موجود:**

   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   
   // تحميل المصنف الموجود
   Workbook workbook = new Workbook(SourceDir + "sampleSortData.xlsx");
   ```

2. **حفظ المصنف في دليل محدد:**

   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";

   // حفظ المصنف المعدل
   workbook.Save(outputDir + "outputSaveExcel.xlsx");
   ```

### مجموعة أوراق العمل والخلايا في Access

يعد الوصول إلى أوراق عمل محددة وخلاياها أمرًا بالغ الأهمية لمعالجة البيانات المستهدفة.

**ملخص:**

انتقل عبر مصنف للوصول إلى أوراق فردية والتلاعب بقيم الخلايا حسب الحاجة.

**خطوات التنفيذ:**

1. **افتح المصنف:**

   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   
   Workbook workbook = new Workbook(SourceDir + "sampleSortData.xlsx");
   ```

2. **مجموعة أوراق العمل والخلايا المحددة للوصول:**

   ```csharp
   // الوصول إلى ورقة العمل الأولى في المصنف
   Worksheet worksheet = workbook.Worksheets[0];
   
   // الحصول على مجموعة كافة الخلايا من ورقة العمل
   Cells cells = worksheet.Cells;
   ```

### تهيئة وتكوين DataSorter

يتيح تكوين معلمات الفرز تنظيم البيانات بكفاءة.

**ملخص:**

ال `DataSorter` يتيح لك الكائن فرز البيانات استنادًا إلى مفاتيح محددة وتفضيلات الترتيب.

**خطوات التنفيذ:**

1. **تهيئة كائن DataSorter:**

   ```csharp
   Workbook workbook = new Workbook();
   DataSorter sorter = workbook.DataSorter; // الحصول على كائن DataSorter
   ```

2. **تعيين معلمات الفرز:**

   ```csharp
   // تعيين ترتيب الفرز والمفاتيح للأعمدة
   sorter.Order1 = SortOrder.Ascending;
   sorter.Key1 = 0; // فهرس العمود الأول
   sorter.Order2 = SortOrder.Ascending;
   sorter.Key2 = 1; // فهرس العمود الثاني
   ```

### تحديد منطقة الخلية وإجراء الفرز

يعد تحديد نطاق من الخلايا لفرز البيانات ضمن حدود محددة أمرًا ضروريًا للفرز الدقيق.

**ملخص:**

استخدام `CellArea`يمكنك تحديد النطاق الدقيق الذي يحتاج إلى الفرز، مما يضمن الدقة في مهام معالجة البيانات الخاصة بك.

**خطوات التنفيذ:**

1. **تحديد منطقة الخلية (النطاق):**

   ```csharp
   Workbook workbook = new Workbook();
   DataSorter sorter = workbook.DataSorter;

   // إنشاء منطقة خلية للفرز
   CellArea ca = new CellArea { StartRow = 1, EndRow = 9, StartColumn = 0, EndColumn = 2 };
   ```

2. **تنفيذ الفرز:**

   ```csharp
   // فرز النطاق المحدد في ورقة العمل الأولى
   sorter.Sort(workbook.Worksheets[0].Cells, ca);
   ```
