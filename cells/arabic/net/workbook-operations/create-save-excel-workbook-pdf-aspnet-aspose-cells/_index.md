### إنشاء مصنف وحفظه بتنسيق PDF
**ملخص**
توضح هذه الميزة كيفية إنشاء مثيل `Workbook` الكائن وحفظه كمستند PDF باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تهيئة المصنف

```csharp
// استيراد مساحات الأسماء الضرورية
using Aspose.Cells;

// حدد مسار دليل المصدر الخاص بك
string SourceDir = "YOUR_SOURCE_DIRECTORY";
// حدد مسار دليل الإخراج الخاص بك
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء مثيل جديد لفئة Workbook
Workbook workbook = new Workbook();
```

#### الخطوة 2: الحفظ بتنسيق PDF

```csharp
// احفظ المصنف بتنسيق PDF في الموقع المحدد
workbook.Save(OutputDir + "/output.pdf", SaveFormat.Pdf);
```

**توضيح**: 
- `SaveFormat.Pdf` يُحدد أنك تريد حفظ الملف بتنسيق PDF. تأكد من ضبط المسار بشكل صحيح لمجلد الكتابة في تطبيقك.

### العمل مع HttpResponse لتنزيل الملف
**ملخص**
يوضح هذا القسم كيفية استخدام `HttpResponse` كائن لتحفيز تنزيل الملف، مع التركيز بشكل خاص على ملفات PDF التي تم إنشاؤها باستخدام Aspose.Cells.

#### الخطوة 1: تحضير كائن الاستجابة

```csharp
// استيراد مساحات الأسماء الضرورية
using System.Web;
using Aspose.Cells;

// افترض أن كائن HttpResponse متاح في سياق ASP.NET الخاص بك
HttpResponse response = HttpContext.Current.Response;

// إنشاء مصنف عمل موجود أو استخدامه
Workbook workbook = new Workbook();
```

#### الخطوة 2: إعداد التخلص من المحتوى وحفظه في الاستجابة

```csharp
if (response != null)
{
    // تكوين رأس HTTP لتنزيل الملف
    response.AddHeader("Content-Disposition", "attachment; filename=\"output.pdf\"");

    // احفظ المصنف مباشرة في HttpResponse OutputStream
    workbook.Save(response.OutputStream, new PdfSaveOptions());
    
    // أكمل عملية الرد
    response.End();
}
```

**توضيح**: 
- `response.AddHeader` ويضمن أن المتصفحات تتعامل مع الإخراج باعتباره تنزيل ملف.
- `PdfSaveOptions` يوفر تكوينات إضافية لحفظ ملفات PDF.
