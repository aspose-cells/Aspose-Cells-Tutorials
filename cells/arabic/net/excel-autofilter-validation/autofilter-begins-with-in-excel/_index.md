# يبدأ التصفية التلقائية في Excel

## مقدمة

عندما يتعلق الأمر بالعمل مع البيانات، فقد أثبت Excel جدارته كتطبيق أساسي للعديد من الصناعات والأغراض. ومن أهم ميزاته ميزة التصفية التلقائية (AutoFilter)، التي تُسهّل عملية غربلة مجموعات البيانات الضخمة. إذا كنت تستخدم Aspose.Cells لـ .NET، يمكنك الاستفادة من هذه الميزة برمجيًا وتحسين مهام إدارة البيانات لديك بشكل ملحوظ. في هذا الدليل، سنشرح لك عملية تطبيق ميزة تُصفّي صفوف Excel بناءً على بدايتها بسلسلة نصية معينة.

## استيراد الحزم

للبدء، قم باستيراد المساحات الأساسية اللازمة في أعلى ملف C# الخاص بك:

```csharp
using System.IO;
using Aspose.Cells;
using System;
```

يؤدي هذا إلى استيراد وظيفة Aspose.Cells الأساسية إلى جانب ميزات النظام الأساسية التي سنعتمد عليها للتفاعل مع وحدة التحكم.

بعد إعداد بيئتك واستيراد الحزم اللازمة، لنبدأ بتقسيم ميزة التصفية التلقائية إلى خطوات سهلة. سنُنفّذ مرشحًا يستخرج الصفوف التي تبدأ بـ "Ba".

## الخطوة 1: تحديد أدلة المصدر والإخراج

أولاً، دعنا نحدد مكان وجود ملف Excel المدخل لدينا، بالإضافة إلى المكان الذي نريد حفظ مخرجاتنا المفلترة فيه:

```csharp
// دليل المصدر
string sourceDir = "Your Document Directory\\";

// دليل الإخراج
string outputDir = "Your Document Directory\\";
```

الشرح: هنا، استبدل `"Your Document Directory\\"` مع المسار الفعلي لمجلداتك. تأكد من إنهاء مسارات المجلدات بعلامة عكسية مزدوجة (`\\`) لتجنب أي مشاكل في المسار.

## الخطوة 2: إنشاء كائن المصنف

بعد ذلك، سنقوم بإنشاء كائن Workbook الذي يشير إلى ملف Excel الخاص بنا:

```csharp
// إنشاء كائن مصنف يحتوي على بيانات عينة
Workbook workbook = new Workbook(sourceDir + "sourseSampleCountryNames.xlsx");
```

شرح: يقوم هذا السطر بتهيئة مثيل جديد لمصنف العمل باستخدام مسار الملف المحدد. `Workbook` تعتبر الفئة أساسية لأنها تمثل ملف Excel بأكمله.

## الخطوة 3: الوصول إلى ورقة العمل الأولى

الآن، نحتاج إلى الوصول إلى ورقة العمل المحددة التي نريد العمل عليها:

```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];
```

الشرح: `Worksheets` تتيح لنا المجموعة الوصول إلى الأوراق الفردية. باستخدام `[0]` يشير إلى ورقة العمل الأولى في ملف Excel الخاص بك، وهي ممارسة شائعة بشكل عام عند العمل مع ملف مكون من ورقة واحدة.

## الخطوة 4: إعداد التصفية التلقائية

هنا يبدأ السحر! سننشئ نطاق تصفية تلقائية لبياناتنا:

```csharp
// إنشاء مرشح تلقائي عن طريق إعطاء نطاق الخلايا
worksheet.AutoFilter.Range = "A1:A18";
```

الشرح: `AutoFilter.Range` تتيح لك الخاصية تحديد الصفوف المراد تصفيتها. في هذه الحالة، نقوم بتصفية الصفوف ضمن النطاق A1 إلى A18، والتي يُفترض أنها تحتوي على بياناتنا.

## الخطوة 5: تطبيق شرط الفلتر

الخطوة التالية هي تحديد شرط الفلترة. نريد عرض الصفوف التي تبدأ قيم أعمدتها الأولى بـ "Ba" فقط:

```csharp
// تهيئة الفلتر للصفوف التي تبدأ بسلسلة "Ba"
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");
```

الشرح: `Custom` تُعرّف الطريقة منطق التصفية لدينا. الوسيطة الأولى (`0`) يشير إلى أننا نقوم بالتصفية بناءً على العمود الأول (أ)، و `FilterOperatorType.BeginsWith` يحدد شرطنا للبحث عن الصفوف التي تبدأ بـ "Ba".

## الخطوة 6: تحديث الفلتر

بعد تطبيق شرط التصفية الخاص بنا، نحتاج إلى التأكد من تحديث Excel ليعكس التغييرات:

```csharp
// قم بتحديث الفلتر لإظهار/إخفاء الصفوف المفلترة
worksheet.AutoFilter.Refresh();
```

شرح: يستدعي هذا السطر تحديثًا للتصفية التلقائية لضمان توافق الصفوف المرئية مع معايير التصفية المطبقة. يشبه الأمر الضغط على زر التحديث في Excel.

## الخطوة 7: حفظ ملف Excel المعدّل

الآن حان الوقت لحفظ التغييرات التي أجريناها:

```csharp
// حفظ ملف Excel المعدل
workbook.Save(outputDir + "outSourseSampleCountryNames.xlsx");
```

الشرح: `Save` تعيد هذه الطريقة كتابة المصنف المُعدَّل إلى مسار الإخراج المُحدَّد. يندرج هذا ضمن كتابة المُرشِّحات المُحدَّدة في ملف جديد، بحيث تبقى بياناتك الأصلية سليمة.

## الخطوة 8: تأكيد الإخراج

وأخيرًا، دعونا نؤكد أن عمليتنا كانت ناجحة:

```csharp
Console.WriteLine("AutofilterBeginsWith executed successfully.\r\n");
```

الشرح: يقوم هذا السطر البسيط بإخراج رسالة تأكيد إلى وحدة التحكم، لإعلامك بأن عملية التصفية اكتملت دون أخطاء.

## خاتمة

في عالمٍ تُصبح فيه إدارة البيانات مُرهقة، يُمكّنك إتقان ميزات مثل AutoFilter في Excel عبر Aspose.Cells لـ .NET من معالجة البيانات بكفاءة وفعالية. لقد تعلمتَ كيفية تصفية صفوف Excel التي تبدأ بحرف "Ba"، بتطبيق الطريقة خطوة بخطوة. مع الممارسة، ستتمكن من تكييف هذه الطريقة لتلبية احتياجات تصفية البيانات المختلفة في مشاريعك الحالية.
