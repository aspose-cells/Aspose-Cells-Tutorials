### ملخص

### التنفيذ خطوة بخطوة

#### 1. قم بإعداد مشروعك
قم بإنشاء مشروع C# جديد وتأكد من تثبيت Aspose.Cells عبر CLI أو Package Manager.

#### 2. اكتب الكود لتعطيل أداة التحقق من التوافق

فيما يلي كود التنفيذ لتعطيل أداة التحقق من التوافق:

```csharp
using System;
using Aspose.Cells;

namespace Aspose.Cells.Examples.CSharp.Articles
{
    public class DisableCompatibilityChecker
    {
        public static void Run()
        {
            // مسار دليل المصدر
            string sourceDir = RunExamples.Get_SourceDirectory();

            // مسار دليل الإخراج
            string outputDir = RunExamples.Get_OutputDirectory();

            // فتح ملف Excel موجود
            Workbook workbook = new Workbook(sourceDir + "sampleDisableCompatibilityChecker.xlsx");

            // تعطيل أداة التحقق من التوافق
            workbook.Settings.CheckCompatibility = false;

            // حفظ ملف Excel المعدل
            workbook.Save(outputDir + "outputDisableCompatibilityChecker.xlsx");

            Console.WriteLine("DisableCompatibilityChecker executed successfully.\r\n");
        }
    }
}
```

#### شرح الكود
- **كتاب العمل الصفي**:يمثل مستند Excel.
- **التحقق من خاصية التوافق**:ضبط هذا على `false` تعطيل التحقق من التوافق.
- **طريقة الحفظ**:يكتب التغييرات مرة أخرى إلى الملف.
