## دليل التنفيذ

في هذا القسم، سنقوم بتفصيل الخطوات اللازمة لحماية أعمدة محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET.

### إنشاء مصنف وورقة عمل
ابدأ بإنشاء مصنف جديد والحصول على ورقة العمل الأولى. هنا ستُطبّق إعدادات حماية الأعمدة.

```csharp
// إنشاء مصنف جديد.
Workbook wb = new Workbook();

// احصل على ورقة العمل الأولى.
Worksheet sheet = wb.Worksheets[0];
```

### فتح جميع الأعمدة مبدئيًا
لتتأكد من حماية أعمدة محددة فقط لاحقًا، قم بإلغاء قفل كافة الأعمدة في ورقة العمل مبدئيًا.

**خطوة بخطوة:**
1. **تعريف النمط و StyleFlag**:ستساعدك هذه الكائنات في إدارة أنماط الأعمدة والأعلام الخاصة بالقفل/إلغاء القفل.
   ```csharp
   Style style;
   StyleFlag flag = new StyleFlag { Locked = true };
   ```
2. **تكرار الأعمدة**:قم بالتكرار عبر كل الأعمدة الممكنة (0-255) لفتحها.
   ```csharp
   for (int i = 0; i <= 255; i++)
   {
       style = sheet.Cells.Columns[(byte)i].Style;
       style.IsLocked = false;
       sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
   }
   ```

### قفل أعمدة محددة
الآن بعد أن تم إلغاء قفل جميع الأعمدة، قم بقفل الأعمدة التي تريد حمايتها.
1. **الحصول على نمط لعمود الهدف**:على سبيل المثال، قفل العمود الأول.
   ```csharp
   style = sheet.Cells.Columns[0].Style;
   style.IsLocked = true;
   ```
2. **تطبيق النمط المقفل**:استخدم `ApplyStyle` الطريقة مع علم النمط لقفل الأعمدة المطلوبة.
   ```csharp
   sheet.Cells.Columns[0].ApplyStyle(style, flag);
   ```

### حماية ورقة العمل
أخيرًا، قم بحماية ورقة العمل بأكملها لفرض أقفال الأعمدة بشكل فعال.
```csharp
// حماية ورقة العمل.
sheet.Protect(ProtectionType.All);

// احفظ ملف Excel.
string dataDir = "your_directory_path";
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```
