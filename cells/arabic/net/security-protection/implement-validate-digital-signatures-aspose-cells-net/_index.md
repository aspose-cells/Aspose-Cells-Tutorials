## دليل التنفيذ
في هذا القسم، سنستكشف كيفية تنفيذ التوقيعات الرقمية باستخدام Aspose.Cells. يُقسّم الدليل إلى أقسام منطقية للتوضيح.

### تعيين التوقيعات الرقمية
**ملخص:**
إضافة توقيع رقمي تضمن بقاء مستند Excel الخاص بك دون أي تغيير منذ لحظة التوقيع. تتضمن هذه العملية إنشاء توقيع معتمد على الشهادة وتعيينه لملفك.
#### الخطوة 1: جهّز بيئتك
تأكد من حصولك على شهادة رقمية. يمكنك استخدام الشهادات ذاتية التوقيع لأغراض الاختبار.
```csharp
string certPath = "path_to_your_certificate.pfx";
string password = "your_password";
```
#### الخطوة 2: تحميل المصنف وتوقيعه
قم بتحميل ملف Excel الخاص بك، وقم بإنشاء `DigitalSignature` الكائن باستخدام شهادتك، وقم بتعيينه إلى المصنف:
```csharp
Workbook workbook = new Workbook("input.xlsx");
DigitalSignature signature = new DigitalSignature(certPath, password);
workbook.DigitalSignatures.Add(signature);

// حفظ المصنف الموقع
workbook.Save("signed_output.xlsx");
```
**توضيح:**
- `certPath`:المسار إلى ملف الشهادة الخاص بك.
- `password`:كلمة المرور للوصول إلى شهادتك.
#### الخطوة 3: التحقق من تعيين التوقيع
يمكنك التحقق مما إذا كان قد تم إضافة التوقيع بنجاح عن طريق إدراج جميع التوقيعات في مصنف:
```csharp
foreach (DigitalSignature ds in workbook.DigitalSignatures)
{
    Console.WriteLine("Signed By: " + ds.SignerName);
}
```
### التحقق من صحة التوقيعات الرقمية
**ملخص:**
بعد التوقيع، من المهم التحقق من صحة التوقيع الرقمي لضمان سلامة المستند ومصداقيته.
#### الخطوة 1: تحميل المصنف الموقّع
```csharp
Workbook signedWorkbook = new Workbook("signed_output.xlsx");
```
#### الخطوة 2: التحقق من صحة التوقيع
يتيح لك Aspose.Cells تكرار التوقيعات للتحقق من صحتها:
```csharp
foreach (DigitalSignature ds in signedWorkbook.DigitalSignatures)
{
    Console.WriteLine($"Signature Validity: {ds.IsValid}");
}
```
**نصيحة لاستكشاف الأخطاء وإصلاحها:** إذا فشل التوقيع في التحقق من صحته، فتأكد من عدم انتهاء صلاحية شهادتك ومن الإشارة إليها بشكل صحيح.
