- التحقق من حالة التوقيع لمشروع VBA

## دليل التنفيذ

### التحقق من توقيع مشروع VBA

تتيح لك هذه الميزة التحقق مما إذا كان مشروع VBA المضمن في ملف Excel موقّعًا، مما يضمن صحته وسلامته.

#### تحميل المصنف

ابدأ بتحميل مصنف Excel الخاص بك باستخدام Aspose.Cells:
```csharp
// قم بتحميل المصنف من دليل المصدر المحدد
Workbook workbook = new Workbook(SourceDir + "sampleCheckVbaProjectSigned.xlsm");
```

#### التحقق من حالة التوقيع

بمجرد التحميل، تحقق مما إذا كان مشروع VBA موقّعًا:
```csharp
// تحقق مما إذا كان مشروع VBA موقّعًا
bool isSigned = workbook.VbaProject.IsSigned;

// إخراج النتيجة (لأغراض العرض التوضيحي)
Console.WriteLine("VBA Project is Signed: " + isSigned);
```

#### توضيح
- **حدود:** ال `Workbook` يأخذ المنشئ مسار الملف كحجة.
- **قيم العودة:** `isSigned` يقوم بإرجاع قيمة منطقية تشير إلى حالة التوقيع.
