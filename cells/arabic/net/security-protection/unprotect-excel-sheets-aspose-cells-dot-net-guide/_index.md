## دليل التنفيذ

اتبع الخطوات التالية للوصول إلى ورقة العمل وإلغاء حمايتها:

### الوصول إلى ورقة العمل وإلغاء حمايتها

#### الخطوة 1: إنشاء كائن مصنف

ابدأ بإنشاء `Workbook` الكائن من الملف الموجود لديك:

```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// تحميل المصنف من القرص
Workbook workbook = new Workbook(sourceDir + "/book1.xls");
```

**توضيح:** يقوم هذا السطر بتهيئة المصنف باستخدام ملف Excel المحدد.

#### الخطوة 2: الوصول إلى ورقة العمل

استرداد ورقة العمل التي ترغب في إلغاء حمايتها:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

**توضيح:** ال `Worksheets[0]` تتمكن الطريقة من الوصول إلى ورقة العمل الأولى في المصنف.

#### الخطوة 3: إلغاء حماية ورقة العمل

إزالة الحماية من ورقة العمل:

```csharp
// إلغاء حماية ورقة العمل دون تحديد كلمة مرور
worksheet.Unprotect();
```

**توضيح:** يؤدي هذا الإجراء إلى إزالة أي حماية موجودة، مما يسمح بالوصول الكامل إلى ورقة العمل.

#### الخطوة 4: حفظ المصنف

احفظ التغييرات مرة أخرى على القرص:

```csharp
workbook.Save(outputDir + "/output.xls", SaveFormat.Excel97To2003);
```

**توضيح:** ال `save` تكتب الطريقة المصنف المحدث بتنسيق Excel 97-2003.

### تحميل وحفظ المصنف
