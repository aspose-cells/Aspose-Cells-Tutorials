## دليل التنفيذ

### فتح جميع الأعمدة
يتيح إلغاء قفل جميع الأعمدة للمستخدمين تحرير أي خلية داخل تلك الأعمدة، مما يوفر المرونة عند التعامل مع مجموعات البيانات الكبيرة.

#### ملخص
توضح هذه الميزة كيفية إلغاء قفل كل عمود في ورقة العمل باستخدام Aspose.Cells لـ .NET.

#### خطوات التنفيذ
**الخطوة 1: تهيئة المصنف وورقة العمل**
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook();
Worksheet sheet = wb.Worksheets[0];
```

**الخطوة 2: فتح الأعمدة**
قم بالمرور عبر كل عمود، ثم اضبط `IsLocked` قم بتعديل الخاصية إلى false، ثم قم بتطبيق النمط.
```csharp
Style style;
StyleFlag flag;

for (int i = 0; i <= 255; i++) {
    style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false;
    
    flag = new StyleFlag();
    flag.Locked = true;
    
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
}
```

#### توضيح
- `style.IsLocked` يتحكم في حالة قفل العمود.
- `StyleFlag` يحدد الخصائص التي سيتم تطبيقها أثناء التصميم.

### قفل صف معين
قد يؤدي قفل صفوف معينة إلى منع التحرير غير المقصود في مناطق البيانات الهامة، مثل العناوين أو الصيغ.

#### ملخص
ترتكز هذه الميزة على قفل الصف الأول فقط في ورقة العمل الخاصة بك.

#### خطوات التنفيذ
**الخطوة 1: الحصول على نمط الصف الأول**
```csharp
Style style = sheet.Cells.Rows[0].GetStyle();
style.IsLocked = true;
```

**الخطوة 2: تطبيق نمط القفل على الصف**
```csharp
flag = new StyleFlag();
flag.Locked = true;

sheet.Cells.ApplyRowStyle(0, style, flag);
```

#### توضيح
- يتم تحقيق القفل عن طريق الضبط `IsLocked` إلى صحيح وتطبيقه مع `ApplyRowStyle`.

### حماية ورقة العمل
تضمن الحماية بقاء بنية ورقة العمل سليمة، مما يحمي سلامة البيانات.

#### ملخص
توضح هذه الميزة كيفية حماية ورقة عمل بأكملها باستخدام أنواع الحماية المختلفة.

#### خطوات التنفيذ
**الخطوة 1: تطبيق الحماية**
```csharp
sheet.Protect(ProtectionType.All);
```

**الخطوة 2: حفظ المصنف**
```csharp
wb.Save(outputDir + "output.out.xls", SaveFormat.Excel97To2003);
```

#### توضيح
- `Protect` تعمل الطريقة على تأمين ورقة العمل ضد التغييرات غير المصرح بها.
- اختر المناسب `ProtectionType` بناءً على احتياجاتك.
