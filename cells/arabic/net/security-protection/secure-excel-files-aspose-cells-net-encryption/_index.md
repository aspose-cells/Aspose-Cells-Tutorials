## دليل التنفيذ (H2)

### تعيين نوع التشفير القوي
**ملخص:** تعمل هذه الميزة على تعزيز أمان ملفات Excel الخاصة بك من خلال تطبيق خوارزمية تشفير قوية.

#### الخطوة 1: تحديد مسارات المصدر والإخراج
ابدأ بتحديد المسارات لملف Excel المصدر والمكان الذي تريد حفظ الإصدار المشفر فيه:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: فتح ملف Excel موجود
قم بتحميل المصنف من مسار محدد باستخدام Aspose.Cells للتعامل مع الملفات بسلاسة.

```csharp
Workbook workbook = new Workbook(SourceDir + "sampleSettingStrongEncryptionType.xlsx");
```

#### الخطوة 3: تكوين خيارات التشفير
اضبط التشفير لاستخدام مُزوِّد تشفير قوي بطول مفتاح 128 بت. تضمن هذه الطريقة أمانًا عاليًا لبياناتك:

```csharp
workbook.SetEncryptionOptions(EncryptionType.StrongCryptographicProvider, 128);
```
- **حدود**: 
  - `EncryptionType.StrongCryptographicProvider`:يحدد نوع المزوّد.
  - `128`:يمثل طول المفتاح بالبتات.

#### الخطوة 4: تعيين كلمة مرور المصنف
تأمين المصنف الخاص بك عن طريق تعيين كلمة مرور:

```csharp
workbook.Settings.Password = "1234";
```
تعتبر هذه الخطوة ضرورية لمنع الوصول غير المصرح به إلى الملف.

#### الخطوة 5: حفظ المصنف المشفر
وأخيرًا، احفظ ملف Excel المشفر والمحمي بكلمة مرور:

```csharp
workbook.Save(OutputDir + "outputSettingStrongEncryptionType.xlsx");
```
