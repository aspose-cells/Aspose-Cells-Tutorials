## دليل التنفيذ

### التحقق من حماية كلمة المرور لأوراق العمل

تتيح لك هذه الميزة تحديد ما إذا كانت أي ورقة عمل داخل ملف Excel محمية بكلمة مرور.

#### الخطوة 1: تحميل المصنف الخاص بك
قم بتحميل المصنف الذي تريد التحقق من الحماية منه:
```csharp
// دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// إنشاء مثيل لـ Workbook وتحميل جدول بيانات
var book = new Workbook(sourceDir + "sampleCheckIfPasswordProtected.xlsx");
```

#### الخطوة 2: الوصول إلى ورقة العمل
قم بالوصول إلى ورقة العمل التي تريد التحقق من الحماية لها:
```csharp
// الوصول إلى ورقة العمل المحمية
var sheet = book.Worksheets[0];
```

#### الخطوة 3: التحقق من حماية كلمة المرور
تحديد ما إذا كانت ورقة العمل محمية بكلمة مرور باستخدام `IsProtectedWithPassword`:
```csharp
if (sheet.Protection.IsProtectedWithPassword)
{
    Console.WriteLine("Worksheet is Password Protected");
}
else
{
    Console.WriteLine("Worksheet is Not Password Protected");
}

Console.WriteLine("CheckIfPasswordProtected executed successfully.");
```

**توضيح:**
- **حدود**: ال `Workbook` و `Worksheets` تدير الفئات محتوى ملف Excel.
- **قيم الإرجاع**:قيمة منطقية تشير إلى حالة حماية كلمة المرور.
