### تشفير ملفات Excel باستخدام Aspose.Cells لـ .NET
**ملخص:**
قم بتشفير ملفات Excel لحماية معلوماتك الحساسة من الوصول غير المصرح به. يوضح هذا القسم كيفية تطبيق أنواع تشفير مختلفة باستخدام Aspose.Cells.

#### الخطوة 1: إعداد مشروعك وتحميل المصنف
```csharp
// تأكد من تعيين مسارات الدليل هذه بشكل صحيح في بيئتك.
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

Workbook workbook = new Workbook(SourceDir + "/Book1.xls");
```

#### الخطوة 2: تحديد خيارات التشفير
اختر بين أنواع التشفير XOR و Strong Cryptographic Provider:
```csharp
// استخدم تشفير XOR بطول مفتاح يبلغ 40.
workbook.SetEncryptionOptions(EncryptionType.XOR, 40);

// بدلاً من ذلك، استخدم تشفير RC4 قويًا بطول مفتاح يبلغ 128 بت.
workbook.SetEncryptionOptions(EncryptionType.StrongCryptographicProvider, 128);
```

#### الخطوة 3: تعيين كلمة مرور الملف
```csharp
// حماية ملف Excel الخاص بك عن طريق تعيين كلمة مرور.
workbook.Settings.Password = "1234";
```

#### الخطوة 4: حفظ المصنف المشفر
```csharp
// احفظ المصنف المشفر الخاص بك في دليل الإخراج.
workbook.Save(OutputDir + "/encryptedBook1.out.xls");
```

### حماية كلمة المرور للتعديل باستخدام Aspose.Cells
**ملخص:**
منع التعديلات غير المصرح بها عن طريق تعيين كلمة المرور المطلوبة للتحرير.

#### الخطوة 1: تحميل المصنف الموجود
