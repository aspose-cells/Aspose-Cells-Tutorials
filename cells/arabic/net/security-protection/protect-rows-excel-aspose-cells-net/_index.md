## دليل التنفيذ

### فتح الأعمدة
أولاً، لنفتح جميع الأعمدة باستثناء العمود الذي نريد حمايته. هذا يضمن إمكانية تعديل صفوف محددة فقط.

#### الخطوة 1: تكرار الأعمدة وإلغاء قفلها

```csharp
// تحديد كائن النمط لفتح القفل
Style style;
// قم بتعريف العلم لتطبيق الأنماط
StyleFlag flag;

for (int i = 0; i <= 255; i++)
{
    // احصل على نمط العمود الحالي
    style = sheet.Cells.Columns[(byte)i].GetStyle();
    // تعيين السمة المقفلة إلى false
    style.IsLocked = false;
    
    // إنشاء كائن StyleFlag جديد
    flag = new StyleFlag { Locked = true };
    
    // تطبيق النمط غير المؤمَّن على جميع الأعمدة
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
}
```

### قفل وحماية صفوف محددة
بعد ذلك، نركز على حماية صفوف محددة مع ترك صفوف أخرى قابلة للوصول.

#### الخطوة 2: قفل الصف الأول

```csharp
// احصل على نمط الصف الأول
style = sheet.Cells.Rows[0].GetStyle();
// تعيين السمة المقفلة إلى true
style.IsLocked = true;

// تطبيق إعداد القفل باستخدام StyleFlag
flag.Locked = true;
sheet.Cells.ApplyRowStyle(0, style, flag);
```

### حماية ورقة العمل
أخيرًا، قم بحماية ورقة العمل لضمان عدم تمكن المستخدمين غير المصرح لهم من تجاوز أقفال الصفوف.

#### الخطوة 3: تطبيق الحماية

```csharp
// قفل جميع العناصر الموجودة على الورقة
sheet.Protect(ProtectionType.All);

// حفظ المصنف
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```
