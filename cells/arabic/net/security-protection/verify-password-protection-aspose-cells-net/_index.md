## دليل التنفيذ

يتناول هذا القسم التحقق من حماية كلمة المرور على أوراق العمل.

### التحقق من حماية ورقة العمل

#### ملخص

سنقوم بالتحقق مما إذا كانت ورقة العمل محمية بكلمة مرور والتأكد من دقتها باستخدام Aspose.Cells لـ .NET.

#### تعليمات خطوة بخطوة

**1. قم بتحميل المصنف**

ابدأ بتحميل ملف Excel الخاص بك:

```csharp
string sourceDir = "path_to_your_directory";
var book = new Workbook(sourceDir + "sampleVerifyPasswordUsedToProtectWorksheets.xlsx");
```
*توضيح*: ال `Workbook` يقوم الفصل بتحميل ملفات Excel ومعالجتها.

**2. الوصول إلى ورقة العمل**

قم بالوصول إلى ورقة العمل المحددة للتحقق من:

```csharp
var sheet = book.Worksheets[0];
```
*توضيح*:يؤدي هذا إلى الوصول إلى ورقة العمل الأولى حسب الفهرس.

**3. التحقق من حالة الحماية**

تحديد ما إذا كانت ورقة العمل محمية بكلمة مرور:

```csharp
if (sheet.Protection.IsProtectedWithPassword)
{
    // انتقل إلى التحقق من كلمة المرور
}
else
{
    Console.WriteLine("Worksheet is not protected.");
}
```
*توضيح*: ال `IsProtectedWithPassword` تشير الخاصية إلى وجود الحماية.

**4. التحقق من كلمة المرور**

إذا كانت محمية، تحقق من كلمة المرور المقدمة:

```csharp
if (sheet.Protection.VerifyPassword("1234"))
{
    Console.WriteLine("Specified password has matched");
}
else
{
    Console.WriteLine("Specified password has not matched");
}
```
*توضيح*: `VerifyPassword` يتحقق من صحة كلمة المرور المقدمة.
