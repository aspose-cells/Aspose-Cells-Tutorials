## دليل التنفيذ

### حماية ورقة عمل Excel بكلمة مرور

لحماية ورقة العمل الخاصة بك باستخدام Aspose.Cells في C#، اتبع الخطوات التالية:

#### الخطوة 1: تحميل المصنف

أولاً، يجب عليك تحميل المصنف الحالي الذي تريد تطبيق الحماية عليه.

```csharp
// تحديد مسار الملف لمستند Excel
cstring dataDir = "path_to_your_file_directory";
FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open);

// إنشاء كائن مصنف وفتح ملف Excel من خلال الدفق
Workbook excel = new Workbook(fstream);
```

#### الخطوة 2: الوصول إلى ورقة العمل

انتقل إلى ورقة العمل المحددة التي ترغب في حمايتها. في هذا المثال، سنصل إلى ورقة العمل الأولى.

```csharp
Worksheet worksheet = excel.Worksheets[0];
```

#### الخطوة 3: تطبيق حماية كلمة المرور

بعد ذلك، استخدم `Protect` طريقة لتأمين ورقة العمل بكلمة مرور. `ProtectionType.All` تضمن الحجة حماية جميع جوانب ورقة العمل.

```csharp
// حماية ورقة العمل باستخدام "aspose" ككلمة مرور
tworksheet.Protect(ProtectionType.All, "aspose", null);
```

#### الخطوة 4: حفظ المصنف المحمي

وأخيرًا، احفظ المصنف الخاص بك للحفاظ على التغييرات.

```csharp
// حفظ ملف Excel المعدّل بالتنسيق الافتراضي (Excel97To2003)
extcel.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);

// أغلق دائمًا تدفق الملفات لتحرير الموارد
fstream.Close();
```

### نصائح استكشاف الأخطاء وإصلاحها

- **أخطاء مسار الملف:** تأكد من أن مسارات الملفات الخاصة بك صحيحة ويمكن الوصول إليها.
- **قضايا الترخيص:** تأكد مرة أخرى من تطبيق ترخيصك بشكل صحيح إذا كنت لا تستخدم إصدارًا تجريبيًا.
- **لم يتم تطبيق حماية كلمة المرور:** التحقق من `ProtectionType` وتأكد من ضبطه على `All`.
