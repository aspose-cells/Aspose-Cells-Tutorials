# استخدام تنسيقات الأرقام المضمنة في Excel برمجيًا

## مقدمة
في هذا البرنامج التعليمي، سنشرح لك كيفية استخدام تنسيقات الأرقام المضمنة في Excel باستخدام Aspose.Cells لـ .NET. سنغطي كل شيء، بدءًا من إعداد بيئة العمل وصولًا إلى تطبيق تنسيقات مختلفة مثل التواريخ والنسب المئوية والعملات. سواء كنت محترفًا متمرسًا أو مبتدئًا في بيئة .NET، سيساعدك هذا الدليل على تنسيق خلايا Excel بسهولة تامة.
## المتطلبات الأساسية
قبل الغوص، تأكد من أن لديك ما يلي:
- تم تثبيت مكتبة Aspose.Cells لـ .NET. يمكنك [قم بتحميله هنا](https://releases.aspose.com/cells/net/).
- معرفة عملية بلغة C# وبرمجة .NET الأساسية.
- Visual Studio أو أي .NET IDE مثبت على جهازك.
- ترخيص Aspose صالح أو [رخصة مؤقتة](https://purchase.aspose.com/temporary-license/).
- تم تثبيت إطار عمل .NET (الإصدار 4.0 أو أعلى).
  
إذا فاتك أيٌّ مما سبق، فاتبع الروابط المُرفقة لإعداد كل شيء. هل أنت مستعد؟ لنبدأ الجزء المُمتع!
## استيراد الحزم
قبل أن نبدأ بالبرنامج التعليمي، تأكد من استيراد المساحات الأساسية اللازمة للعمل مع Aspose.Cells لـ .NET:
```csharp
using System.IO;
using Aspose.Cells;
using System;
```
بعد استيراد هذه الملفات، ستكون جاهزًا للتعامل مع ملفات Excel برمجيًا. الآن، لنبدأ بالدليل خطوة بخطوة!
## الخطوة 1: إنشاء مصنف Excel الخاص بك أو الوصول إليه
في هذه الخطوة، ستنشئ مصنفًا جديدًا. تخيل هذا كأنك تفتح ملف إكسل جديدًا، لكنك تفعل ذلك من خلال الكود!
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
	System.IO.Directory.CreateDirectory(dataDir);
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
هنا، نقوم ببساطة بإنشاء مثيل جديد `Workbook` هذا الكائن بمثابة ملف Excel جاهز لمعالجة البيانات. يمكنك أيضًا تحميل ملف موجود عن طريق تحديد مساره.
## الخطوة 2: الوصول إلى ورقة العمل
يمكن أن تحتوي مصنفات Excel على عدة أوراق عمل. في هذه الخطوة، سنصل إلى ورقة العمل الأولى في مصنفك:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
نصل الآن إلى ورقة العمل الأولى في المصنف. إذا كنت بحاجة إلى التعامل مع أوراق عمل إضافية، يمكنك الرجوع إليها باستخدام فهرسها أو اسمها.
## الخطوة 3: إضافة البيانات إلى الخلايا
لنبدأ بإضافة بعض البيانات إلى خلايا محددة. أولًا، سنُدرج تاريخ النظام الحالي في الخلية "A1":
```csharp
worksheet.Cells["A1"].PutValue(DateTime.Now);
```
يُدرج هذا السطر التاريخ الحالي في الخلية A1. رائع، أليس كذلك؟ تخيّل القيام بذلك يدويًا لمئات الخلايا - سيكون الأمر مُرهقًا للغاية. الآن، لننتقل إلى التنسيق!
## الخطوة 4: تنسيق التاريخ في الخلية "A1"
الآن، لنُنسّق هذا التاريخ بصيغة أسهل للقراءة، مثل "15-أكتوبر-24". وهنا يبرز Aspose.Cells حقًا:
1. استرداد نمط الخلية:
```csharp
Style style = worksheet.Cells["A1"].GetStyle();
```
هنا، نأخذ نمط الخلية A1. تخيل هذا كأننا نأخذ نمط الخلية قبل إجراء أي تعديلات.
2. تعيين تنسيق التاريخ:
```csharp
style.Number = 15;
```
ضبط `Number` الخاصية 15 تُطبّق تنسيق التاريخ المطلوب. هذا رمز تنسيق رقمي مُدمج لعرض التواريخ بالتنسيق "يوم-شهر-سنة".
3. تطبيق النمط على الخلية:
```csharp
worksheet.Cells["A1"].SetStyle(style);
```
يُطبّق هذا السطر تغييرات النمط على الخلية. الآن، بدلًا من تنسيق التاريخ الافتراضي، سترى تنسيقًا أسهل استخدامًا مثل "15-أكتوبر-24".
## الخطوة 5: إضافة وتنسيق النسبة المئوية في الخلية "A2"
لننتقل الآن إلى تنسيق النسب المئوية. لنفترض أنك تريد إدراج قيمة وعرضها كنسبة مئوية. في هذه الخطوة، سنضيف قيمة رقمية إلى الخلية "A2" وننسقها كنسبة مئوية:
1. إدراج القيمة الرقمية:
```csharp
worksheet.Cells["A2"].PutValue(20);
```
يُدرج هذا الرقم ٢٠ في الخلية A2. قد تتساءل: "هذا مجرد رقم عادي، كيف أحوّله إلى نسبة مئوية؟" حسنًا، نحن على وشك الوصول إلى ذلك.
2. استرداد النمط وتعيين تنسيق النسبة المئوية:
```csharp
style = worksheet.Cells["A2"].GetStyle();
style.Number = 9;  // التنسيق كنسبة مئوية
worksheet.Cells["A2"].SetStyle(style);
    ```
Setting the `Number` property to 9 applies the built-in percentage format. Now the value in A2 will be displayed as "2000%." (Yes, 20 is treated as 2000% in percentage formatting).
## Step 6: Add and Format Currency in Cell "A3"
Now, let’s add a numeric value in cell A3 and format it as currency. This is a common use case for financial reports.
1. Insert Numeric Value:
```csharp
worksheet.Cells["A3"].PutValue(2546);
```
هنا، نضيف ٢٥٤٦ إلى الخلية A٣. بعد ذلك، سنُنسّق هذا الرقم ليظهر كعملة.
2. استرداد النمط وتعيين تنسيق العملة:
```csharp
style = worksheet.Cells["A3"].GetStyle();
style.Number = 6;  // تنسيق كعملة
worksheet.Cells["A3"].SetStyle(style);
```
ضبط `Number` الخاصية 6 تُطبّق تنسيق العملة. الآن، ستظهر القيمة في الخلية A3 كـ "2,546.00"، مع إضافة فواصل ورقمين عشريين.
## الخطوة 7: حفظ ملف Excel
الآن بعد أن قمنا بتطبيق كل سحر التنسيق، فقد حان الوقت لحفظ الملف:
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```
يحفظ هذا السطر ملف Excel بتنسيق Excel 97-2003. يمكنك تغيير `SaveFormat` لتناسب احتياجاتك. وهكذا، تكون قد أنشأت ملف Excel ونسقته برمجيًا!
## خاتمة
تهانينا! لقد تعلمتَ بنجاح كيفية استخدام Aspose.Cells لـ .NET لتطبيق تنسيقات الأرقام المُدمجة على خلايا ملف Excel. من التواريخ إلى النسب المئوية والعملات، تناولنا بعضًا من أكثر احتياجات التنسيق شيوعًا لمعالجة بيانات Excel. الآن، بدلًا من تنسيق الخلايا يدويًا، يمكنك أتمتة العملية بأكملها، مما يوفر عليك الوقت ويقلل الأخطاء.
## الأسئلة الشائعة
### هل يمكنني تطبيق تنسيقات الأرقام المخصصة باستخدام Aspose.Cells لـ .NET؟
نعم! بالإضافة إلى التنسيقات المضمنة، يدعم Aspose.Cells أيضًا تنسيقات أرقام مخصصة. يمكنك إنشاء تنسيقات محددة للغاية باستخدام `Custom` الممتلكات في `Style` فصل.
### كيف يمكنني تنسيق خلية كعملة برمز محدد؟
لتطبيق رمز عملة محدد، يمكنك استخدام التنسيق المخصص عن طريق ضبط `Style.Custom` ملكية.
### هل يمكنني تنسيق الصفوف أو الأعمدة بأكملها؟
بالتأكيد! يمكنك تطبيق الأنماط على الصفوف أو الأعمدة بأكملها باستخدام `Rows` أو `Columns` المجموعات في `Worksheet` هدف.
### كيف يمكنني تنسيق خلايا متعددة في وقت واحد؟
يمكنك استخدام `Range` كائن لتحديد خلايا متعددة وتطبيق الأنماط عليها جميعًا مرة واحدة.
### هل أحتاج إلى تثبيت Microsoft Excel لاستخدام Aspose.Cells؟
لا، يعمل Aspose.Cells بشكل مستقل عن Microsoft Excel، لذا لا تحتاج إلى تثبيت Excel على جهازك.
