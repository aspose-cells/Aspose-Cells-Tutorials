# إدارة الدليل الرئيسي وملفات Excel في .NET باستخدام Aspose.Cells

- افتح ملفات Excel وقم بإدارتها باستخدام FileStream.
- تعديل خصائص مصنف Excel مثل عرض الأعمدة باستخدام Aspose.Cells.

## دليل التنفيذ

### إدارة الدليل

**ملخص**:تتحقق هذه الميزة من وجود دليل وتقوم بإنشائه إذا كان مفقودًا.

#### التنفيذ خطوة بخطوة

##### التحقق من وجود الدليل

```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
bool isExists = Directory.Exists(SourceDir);
```

هنا، `Directory.Exists` يتحقق من وجود المسار المحدد. تُرجع هذه الطريقة قيمة منطقية.

##### إنشاء الدليل إذا لم يكن موجودًا

```csharp
if (!isExists)
{
    Directory.CreateDirectory(SourceDir);
}
```

`Directory.CreateDirectory` يقوم بإنشاء الدليل وجميع الدلائل الفرعية الضرورية على طول المسار.

### معالجة تدفق الملفات

**ملخص**:يوضح كيفية فتح ملف Excel باستخدام FileStream والتأكد من إصدار الموارد بشكل صحيح.

#### التنفيذ خطوة بخطوة

##### إنشاء تدفق ملف لملف Excel

```csharp
string SourceFile = Path.Combine("YOUR_SOURCE_DIRECTORY", "book1.xls");
FileStream fstream = new FileStream(SourceFile, FileMode.Open);
```

`FileStream` يستخدم لفتح الملف في `Open` وضع.

##### إغلاق تدفق الملفات

```csharp
fstream.Close();
```

يؤدي إغلاق الدفق إلى تحرير موارد النظام المرتبطة به، مما يمنع تسرب الذاكرة.

### عمليات المصنف باستخدام Aspose.Cells

**ملخص**:توضح هذه الميزة كيفية تحميل مصنف Excel، وتعديل خصائص مثل عرض الأعمدة، وحفظ التغييرات.

#### التنفيذ خطوة بخطوة

##### تحميل وفتح مصنف

```csharp
using (FileStream fstream = new FileStream(inputFilePath, FileMode.Open))
{
    Workbook workbook = new Workbook(fstream);
}
```

ال `Workbook` يقوم المُنشئ بتهيئة كائن لعمليات ملف Excel. باستخدام `using` تضمن العبارة إغلاق الدفق تلقائيًا.

##### الوصول إلى خصائص ورقة العمل وتعديلها

```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Cells.StandardWidth = 20.5;
```

يتيح لك الوصول إلى ورقة العمل الأولى تعديل عرض الأعمدة، مما يؤدي إلى تحسين إمكانية القراءة.

##### حفظ المصنف

```csharp
workbook.Save(outputFilePath);
```

ال `Save` تكتب الطريقة جميع التغييرات مرة أخرى إلى موقع ملف Excel المحدد.
