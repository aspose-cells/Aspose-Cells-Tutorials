## دليل التنفيذ

### الميزة 1: إنشاء مصنف العمل وتحميله

#### ملخص
ابدأ بإنشاء `Workbook` كائن لتحميل ملف Excel الخاص بك، مما يسمح لك بمعالجة البيانات برمجيًا.

**التنفيذ خطوة بخطوة**

##### الخطوة 1: تحديد دليل المصدر
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
```
*توضيح:* حدد الدليل الذي يوجد فيه ملف Excel المصدر الخاص بك.

##### الخطوة 2: إنشاء المصنف وتحميله
```csharp
Workbook workbook = new Workbook(sourceDir + "/sampleNamesTable.xlsx");
```
*توضيح:* هذا الخط ينشئ `Workbook` الكائن ويُحمّل 'sampleNamesTable.xlsx'. يجمع مسار الملف بين الدليل المُحدد واسم الملف.

### الميزة 2: الوصول إلى نطاق مُسمّى في ورقة عمل Excel

#### ملخص
يمكنك الوصول إلى نطاقات محددة مسماة ضمن مصنف Excel الخاص بك لإجراء عمليات على أقسام البيانات المستهدفة.

**التنفيذ خطوة بخطوة**

##### الخطوة 1: تهيئة WorkbookDesigner
```csharp
WorkbookDesigner designer = new WorkbookDesigner(workbook);
```
*توضيح:* ال `WorkbookDesigner` تسمح الفئة بالتعامل المتقدم مع مصنفات العمل، مثل الوصول إلى النطاقات المسماة.

##### الخطوة 2: استرداد النطاق المسمى
```csharp
var range = designer.Workbook.Worksheets.GetRangeByName("Names");
```
*توضيح:* استخدم هذه الطريقة للوصول إلى النطاق المُسمّى "الأسماء" في مصنفك. هذا النطاق جاهز الآن لمزيد من المعالجة.

### الميزة 3: تصدير البيانات من نطاق مُسمى باستخدام الخيارات

#### ملخص
تصدير البيانات بكفاءة عن طريق تخطي العناوين وتكوين خيارات التصدير باستخدام `ExportTableOptions`.

**التنفيذ خطوة بخطوة**

##### الخطوة 1: تكوين خيارات التصدير
```csharp
ExportTableOptions options = new ExportTableOptions();
options.ExportColumnName = true;
```
*توضيح:* عن طريق الإعداد `ExportColumnName` ل `true`سيتم تخطي الصف الأول (المفترض أنه عناوين) أثناء التصدير.

##### الخطوة 2: تصدير البيانات من النطاق المسمى
```csharp
var dataTable = range.ExportDataTable(options);
```
*توضيح:* تقوم هذه الطريقة بتصدير البيانات إلى `DataTable`، مع حذف أسماء الأعمدة كعناوين، مما يجعلها مثالية لمزيد من المعالجة أو التحليل.
