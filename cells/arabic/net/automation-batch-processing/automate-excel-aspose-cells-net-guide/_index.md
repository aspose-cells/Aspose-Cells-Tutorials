```csharp

```

### الوصول إلى أوراق العمل والخلايا
**ملخص:** بعد إنشاء المصنف، يمكنك الوصول إلى أوراق العمل والخلايا المحددة الخاصة به للتحكم في محتواها.
```csharp

// خلية الوصول "A1"

```

### تعيين قيم الخلايا
**ملخص:** عيّن قيمًا للخلية المستهدفة. هذا المثال يضيف نصًا إلى الخلية "A1".
```csharp
// إضافة قيمة إلى الخلية "A1"

```

### تطبيق إعدادات النمط
**ملخص:** قم بتخصيص الأنماط مثل محاذاة النص وتقليصه ليتناسب.
```csharp
// استرداد وتعديل إعدادات النمط للخلية
```

### حفظ المصنف
**ملخص:** احفظ المصنف الخاص بك بالتنسيق المطلوب، مثل Excel 97-2003 أو التنسيقات الأحدث.
