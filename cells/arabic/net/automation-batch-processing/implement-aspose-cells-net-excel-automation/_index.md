## دليل التنفيذ

### إنشاء مصنف Excel وحفظه

#### ملخص
سيوضح لك هذا القسم كيفية إنشاء مثيل جديد لمصنف عمل، وملئه بالبيانات، وحفظه كملف Excel.

**الخطوة 1: إنشاء كائن مصنف جديد**

ابدأ بإنشاء حساب جديد `Workbook` هذا الكائن يمثل ملف Excel الخاص بك في الذاكرة.
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
Workbook workbook = new Workbook();
```

**الخطوة 2: حفظ المصنف في ملف**

احفظ المصنف الخاص بك في ملف Excel، مع تحديد المسار المطلوب.
```csharp
workbook.Save(outputDir + "SomeExcelFile.xlsx");
```
*المعايير والغرض*: ال `Save` تكتب هذه الطريقة بيانات المصنف المخزنة في الذاكرة إلى القرص كملف .xlsx. يمكنك تحديد تنسيقات مختلفة مثل XLS أو CSV عن طريق تعديل الامتداد.

### إضافة ارتباطات تشعبية إلى ورقة عمل

#### ملخص
الارتباطات التشعبية ضرورية لإنشاء نقاط بيانات مترابطة داخل ملفات Excel. إليك كيفية إضافتها باستخدام Aspose.Cells.

**الخطوة 1: إنشاء مصنف والحصول على ورقة العمل الأولى**

ابدأ بمصنف عمل موجود، أو قم بإنشاء مصنف جديد إذا لزم الأمر.
```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**الخطوة 2: إضافة ارتباط تشعبي إلى الخلية A5**

قم بربط الخلية A5 بملف Excel آخر موجود في دليل الإخراج الخاص بك.
```csharp
worksheet.Hyperlinks.Add("A5", 1, 1, outputDir + "SomeExcelFile.xlsx");
```
*المعايير والغرض*: ال `Hyperlinks.Add` تتطلب هذه الطريقة مرجع الخلية والأبعاد (الصفوف × الأعمدة) لوضع الارتباط التشعبي. ثم تحدد مسار الملف المستهدف.

**الخطوة 3: تعيين نص العرض للارتباط التشعبي**

قم بتحديد النص الذي يظهر للمستخدمين كنص قابل للنقر.
```csharp
worksheet.Hyperlinks[0].TextToDisplay = "Link To External File";
```

**الخطوة 4: احفظ المصنف الذي يحتوي على الارتباط التشعبي المضاف**

احفظ تعديلاتك في ملف جديد.
```csharp
workbook.Save(outputDir + "outputAddingLinkToExternalFile.xlsx");
```
