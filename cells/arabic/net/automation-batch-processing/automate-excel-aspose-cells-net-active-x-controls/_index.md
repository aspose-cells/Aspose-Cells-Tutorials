# أتمتة مصنفات Excel باستخدام Aspose.Cells لـ .NET: إنشاء عناصر تحكم ActiveX وإدارتها

في عالمنا اليوم الذي يعتمد على البيانات، يُمكن لإنشاء مصنفات Excel وإدارتها بكفاءة برمجيًا توفير الوقت وتعزيز الإنتاجية. باستخدام Aspose.Cells لـ .NET، يُمكن للمطورين أتمتة إنشاء ملفات Excel ودمج العناصر التفاعلية مثل عناصر تحكم ActiveX بسلاسة. سيرشدك هذا البرنامج التعليمي خلال إنشاء مصنف Excel، وإضافة عنصر تحكم ActiveX بزر تبديل، وحفظه بتنسيق XLSX باستخدام Aspose.Cells.

**ما سوف تتعلمه:**
- كيفية إنشاء مصنف Excel جديد باستخدام Aspose.Cells لـ .NET.
- إضافة عناصر تحكم ActiveX إلى أوراق العمل.
- حفظ المصنف الخاص بك بالتنسيق المطلوب.

دعونا نستكشف كيفية الاستفادة من هذه الميزات لتبسيط مهام معالجة ملفات Excel. قبل البدء في التنفيذ، دعونا نتأكد من تغطية جميع المتطلبات الأساسية.

### دليل التنفيذ
ينقسم هذا الدليل إلى أقسام توضح الميزات المحددة لـ Aspose.Cells لـ .NET.

#### إنشاء مصنف وورقة عمل Access
**ملخص:**
سنبدأ بإنشاء مصنف Excel والوصول إلى ورقة العمل الأولى فيه. يُشكّل هذا أساسًا لعمليات أخرى، مثل إضافة عناصر تحكم أو تعديل بيانات.

**التنفيذ خطوة بخطوة:**

**1. إنشاء كائن مصنف جديد**

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook(); // الخطوة 1: إنشاء كائن مصنف جديد.
```

يؤدي هذا إلى تهيئة مصنف Excel جديد وفارغ.

**2. الوصول إلى ورقة العمل الأولى**

```csharp
Worksheet sheet = wb.Worksheets[0]; // الخطوة 2: قم بالوصول إلى ورقة العمل الأولى في المصنف.
```
ال `Worksheets` تتيح لك المجموعة التفاعل مع جميع أوراق العمل في المصنف. هنا، نصل إلى أول ورقة عمل بالمؤشر (0).

#### إضافة عنصر تحكم ActiveX إلى ورقة العمل
**ملخص:**
بعد ذلك، دعنا نعمل على تحسين ورقة العمل الخاصة بنا عن طريق إضافة عنصر تحكم ActiveX التفاعلي Toggle Button.

**التنفيذ خطوة بخطوة:**

**1. إضافة زر تبديل عنصر التحكم ActiveX**

```csharp
using Aspose.Cells.Drawing;
using Aspose.Cells.Drawing.ActiveXControls;

Workbook wb = new Workbook(); // إعادة إنشاء كائن مصنف جديد.
Worksheet sheet = wb.Worksheets[0]; // قم بالوصول إلى ورقة العمل الأولى في المصنف مرة أخرى.

Shape s = sheet.Shapes.AddActiveXControl(ControlType.ToggleButton, 4, 0, 100, 30); 
// أضف زر تبديل لعنصر تحكم ActiveX. المعلمات: نوع العنصر (ToggleButton)، الموضع (x: 4، y: 0)، العرض: 100، الارتفاع: 30.
```

يؤدي مقتطف التعليمات البرمجية هذا إلى إنشاء شكل في ورقة العمل التي تستضيف عنصر تحكم ActiveX.

**2. تكوين الخلية المرتبطة لعنصر التحكم ActiveX**

```csharp
ActiveXControl c = s.ActiveXControl; // الوصول إلى كائن عنصر التحكم ActiveX من الشكل.
c.LinkedCell = "A1"; // قم بتعيين خاصية الخلية المرتبطة لعنصر التحكم ActiveX إلى 'A1'.
```
يتيح ربط خلية إمكانية تشغيل وظيفة تفاعلية، مثل تحديث البيانات عند النقر فوق زر التبديل.

#### حفظ المصنف بتنسيق XLSX
**ملخص:**
وأخيرًا، سنحفظ مصنفنا مع جميع التعديلات بتنسيق ملف XLSX.

**التنفيذ خطوة بخطوة:**

```csharp
wb.Save(outputDir + "/outputAddActiveXControls.xlsx", SaveFormat.Xlsx); 
// احفظ المصنف بتنسيق XLSX. يجمع مسار الحفظ بين دليل الإخراج واسم الملف.
```

تضمن هذه الخطوة تخزين المصنف الخاص بك على القرص، مع الاحتفاظ بكل التغييرات التي تم إجراؤها برمجيًا.
