## دليل التنفيذ
سنستكشف الميزات الرئيسية: معالجة المصنف وإدارة مسار الملف.
### معالجة المصنف
ركز على فتح ملف Excel، وإدراج الصفوف في ورقة العمل، وحفظ المصنف المعدل.
#### الخطوة 1: فتح ملف Excel موجود باستخدام FileStream
افتح ملف Excel الموجود باستخدام `FileStream`، مما يسمح بعمليات القراءة أو الكتابة المباشرة:
```csharp
// افتح ملف Excel المصدر
FileStream fstream = new FileStream(dataDir, FileMode.Open);
```
#### الخطوة 2: إنشاء كائن مصنف من مجرى الملفات
إنشاء `Workbook` كائن لتمثيل مصنف Excel بأكمله في الذاكرة:
```csharp
// قم بتحميل المصنف باستخدام تدفق الملف
Workbook workbook = new Workbook(fstream);
```
#### الخطوة 3: الوصول إلى ورقة العمل الأولى في المصنف
استهداف التعديلات بدقة من خلال الوصول إلى أوراق العمل المحددة:
```csharp
// احصل على ورقة العمل الأولى من المصنف
Worksheet worksheet = workbook.Worksheets[0];
```
#### الخطوة 4: إدراج الصفوف في ورقة العمل
إدراج صفوف متعددة في فهرس محدد، ونقل البيانات الموجودة إلى الأسفل دون الكتابة فوقها:
```csharp
// إدراج 10 صفوف بدءًا من مؤشر الصف 2 (الصف الثالث)
worksheet.Cells.InsertRows(2, 10);
```
#### الخطوة 5: حفظ ملف Excel المعدّل في موقع جديد
احفظ التغييرات في موقع ملف جديد، مع الحفاظ على البيانات الأصلية وتخزين التعديلات بشكل منفصل:
```csharp
// حفظ المصنف المعدل في دليل الإخراج
workbook.Save(outputFilePath);
```
#### الخطوة 6: إغلاق FileStream لتحرير الموارد
أغلق دائمًا التدفقات بعد العمليات لتحرير موارد النظام:
```csharp
// إغلاق مجرى الملفstream.Close();
```
