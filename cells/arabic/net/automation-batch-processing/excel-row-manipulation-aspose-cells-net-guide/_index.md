## دليل التنفيذ

### الميزة: نسخ صف في ورقة عمل Excel

#### ملخص

تتيح لك هذه الميزة نسخ صف من موضع إلى آخر داخل نفس ورقة العمل، مما يضمن تضمين كافة العناصر مثل البيانات والتنسيق والصور وكائنات الرسم.

#### التنفيذ خطوة بخطوة

**1. قم بتحميل مصنف العمل الخاص بك**
ابدأ بتحميل ملف Excel الحالي لديك:
```csharp
// استبدله بمسار دليل المصدر الفعلي الخاص بك
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; 
Workbook excelWorkbook1 = new Workbook(SourceDir + "book1.xls");
```

**2. الوصول إلى ورقة العمل**
قم بالوصول إلى ورقة العمل التي تريد معالجتها، على سبيل المثال، الورقة الأولى:
```csharp
Worksheet wsTemplate = excelWorkbook1.Worksheets[0];
```

**3. انسخ الصف**
استخدم `CopyRow` طريقة لنسخ البيانات من صف إلى آخر. هنا، نقوم بنسخ الصف الثاني (الفهرس ١) إلى الصف السادس عشر (الفهرس ١٥):
```csharp
wsTemplate.Cells.CopyRow(wsTemplate.Cells, 1, 15);
```

**4. احفظ مصنفك**
وأخيرًا، احفظ التغييرات:
```csharp
excelWorkbook1.Save(SourceDir + "output.xls");
```
