## دليل التنفيذ

في هذا القسم، سنتعرف على كيفية إعداد فرز البيانات وتنفيذه باستخدام Aspose.Cells.

### الخطوة 1: تحضير كتاب العمل الخاص بك

ابدأ بتحميل ملف Excel الخاص بك إلى `Workbook` الكائن. يمثل هذا الكائن المصنف بأكمله داخل تطبيقك.

```csharp
// تحميل ملف Excel الحالي.
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "book1.xls");
```

### الخطوة 2: تكوين DataSorter

بعد ذلك، قم بتكوين `DataSorter` الكائن. سيحدد هذا كيفية فرز البيانات ونوعها.

```csharp
// قم بالوصول إلى DataSorter من المصنف.
DataSorter sorter = workbook.DataSorter;

// تعيين ترتيب الفرز للمفتاح الأول تنازليًا.
sorter.Order1 = SortOrder.Descending;
sorter.Key1 = 0; // فهرس العمود الأول

// تعيين ترتيب الفرز للمفتاح الثاني تصاعديًا.
sorter.Order2 = SortOrder.Ascending;
sorter.Key2 = 1; // فهرس العمود الثاني
```

### الخطوة 3: تحديد منطقة الخلية

قم بتحديد نطاق الخلايا التي تريد فرزها باستخدام `CellArea` هدف.

```csharp
// حدد منطقة الخلية للفرز.
CellArea ca = new CellArea();
ca.StartRow = 0;
ca.EndRow = 13; // يتضمن الصفوف من 0 إلى 13
ca.StartColumn = 0;
ca.EndColumn = 1; // يتضمن العمودين 0 و1
```

### الخطوة 4: تنفيذ الفرز

تنفيذ عملية الفرز على ورقة العمل المحددة.

```csharp
// قم بتطبيق الفرز على منطقة الخلية المحددة في الورقة الأولى.
sorter.Sort(workbook.Worksheets[0].Cells, ca);
```
