### إنشاء مثيلات وتكوين WorkbookDesigner
#### ملخص:
ال `WorkbookDesigner` تعتبر الفئة محورية في معالجة قوالب Excel باستخدام العلامات الذكية، مما يسمح لك بإنشاء تقارير ديناميكية بسلاسة.
##### الخطوة 1: تحديد ملف المصمم ومجموعة البيانات
```csharp
public static Stream DesignerFile { get; set; }
public static System.Data.SqlClient.SqlConnection Dataset { get; set; }
```
**توضيح**:تعتبر هذه الخصائص بمثابة عناصر نائبة لملف القالب واتصال قاعدة البيانات، على التوالي.
##### الخطوة 2: تنفيذ طريقة التشغيل
```csharp
public static void Run()
{
    if (DesignerFile != null && Dataset != null)
    {
        WorkbookDesigner designer = new WorkbookDesigner();
        designer.Workbook = new Workbook(DesignerFile);
        designer.SetDataSource(Dataset);
        designer.Process();
    }
}
```
**توضيح**:تضمن هذه الطريقة توفر القالب ومصدر البيانات، ثم تقوم بمعالجة العلامات الذكية لإنتاج المستند النهائي.
