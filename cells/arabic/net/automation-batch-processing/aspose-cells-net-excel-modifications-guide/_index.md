## دليل التنفيذ
### الميزة: تحميل وتعديل ملف Excel
تُظهر هذه الميزة كيفية تحميل مصنف Excel الحالي، وإدراج أعمدة في كل ورقة عمل، وحفظ الإصدار المحدث.

#### ملخص
سنستكشف كيفية تحميل مصنف، والتكرار خلال أوراق العمل الخاصة به، وإدراج أعمدة جديدة، وتعيين قيم الرأس، وحفظ التغييرات بكفاءة باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تحميل المصنف
ابدأ بإنشاء مثيل لـ `Workbook` مع مسار ملف Excel المصدر الخاص بك:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string inputFile = SourceDir + "/Sample.xls";

// قم بإنشاء كائن مصنف لتحميل ملف Excel.
Workbook workbook = new Workbook(inputFile);
```

#### الخطوة 2: إدراج الأعمدة وتعيين الرؤوس
قم بالتكرار خلال كل ورقة عمل وإدراج الأعمدة:
```csharp
for (int i = 0; i < workbook.Worksheets.Count; i++)
{
    Worksheet ws = workbook.Worksheets[i];
    Cells cells = ws.Cells;

    // قم بإدراج عشرة أعمدة جديدة في بداية كل ورقة عمل.
    for (int c = 0; c < 10; c++)
    {
        cells.InsertColumn(c); // إدراج عمود جديد
        cells[0, c].PutValue("Column" + c.ToString()); // تعيين أسماء الرؤوس
    }
}
```
**لماذا هذا النهج؟**
يضمن إدراج الأعمدة قبل تعيين القيم محاذاة جميع الرؤوس بشكل صحيح وسهولة التعرف عليها.

#### الخطوة 3: حفظ المصنف المعدل
بمجرد اكتمال التعديلات، احفظ المصنف الخاص بك في ملف جديد:
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = outputDir + "/output_out.xls";

// احفظ ملف Excel المعدّل.
workbook.Save(outputFile);
```
