# تغيير اتجاه علامة التجزئة في الرسم البياني في Excel باستخدام Aspose.Cells 

يوضح هذا المثال كيفية تعديل اتجاه علامات التجزئة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. بتعديل اتجاه نص علامات التجزئة لمحور الفئة، يمكنك تحسين سهولة قراءة بيانات الرسم البياني. يقوم الكود بتحميل مصنف Excel موجود، والوصول إلى ورقة العمل الأولى، واسترجاع الرسم البياني، وضبط اتجاه علامة التجزئة أفقيًا، ثم حفظ المصنف المُعدَّل. يضمن هذا النهج تحكمًا دقيقًا في اتجاه علامات التجزئة في تطبيقات .NET.

## تعليمات التثبيت  
لتشغيل هذا المثال، اتبع الخطوات التالية:  

1. تثبيت Aspose.Cells لـ .NET  
   - تنزيل من [Aspose.Cells لـ .NET](https://releases.aspose.com/cells/net/)  
   - أو قم بالتثبيت عبر NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. قم بإعداد مشروعك  
   - إنشاء مشروع .NET جديد.  
   - أضف مرجعًا إلى Aspose.Cells.dll.  

3. تشغيل الكود  
   - يستبدل `"Your Document Directory"` مع مسار الملف الفعلي.  
   - قم بتنفيذ البرنامج لتحديث اتجاه علامة التجزئة في الرسم البياني.
