## تعليمات التثبيت:  

لاستخدام Aspose.Cells لـ .NET، اتبع الخطوات التالية:  

1. التثبيت عبر مدير الحزم NuGet  
- افتح Visual Studio.  
- انتقل إلى الأدوات → مدير حزمة NuGet → إدارة حزم NuGet للحل.  
- ابحث عن Aspose.Cells.  
- انقر فوق "تثبيت" لإضافة الحزمة إلى مشروعك.  

بدلاً من ذلك، قم بتثبيته باستخدام وحدة التحكم في إدارة الحزم NuGet:  

```powershell
Install-Package Aspose.Cells
```

2. إضافة مساحة اسمية في كود C# الخاص بك  
```csharp
using Aspose.Cells;
```

3. تفعيل الترخيص (اختياري، للميزات الكاملة)  
إذا كان لديك ترخيص، قم بتطبيقه باستخدام الكود التالي:  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
