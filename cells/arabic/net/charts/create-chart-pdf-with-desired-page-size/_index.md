# إعداد PDF من مخطط Excel مع حجم صفحة محدد

## تعليمات التثبيت  
للاستخدام Aspose.Cells ل .NET، اتبع هذه الخطوات:  

1. إثباط عبر NuGet:  
   افتح مدير الحزمات NuGet في Visual Studio و تشغيل:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. تحميل DLL:  
   احصل على أحدث نسخة من [Aspose.Cells لصفحة تنزيل .NET](https://releases.aspose.com/cells/net/). . .  

3. إدراج مساحة الأسماء:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
