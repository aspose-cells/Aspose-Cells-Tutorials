# العمل مع ألوان Excel برمجيًا

## مقدمة
هل ترغب في تحسين ملفات Excel لديك بإضافة لمسة جمالية باستخدام الألوان؟ سواء كنت تعمل على تقارير أو لوحات معلومات أو أي مستندات تعتمد على البيانات، فإن الألوان أداة فعّالة لتحسين سهولة القراءة والتفاعل. في هذا البرنامج التعليمي، سنتعمق في عالم Aspose.Cells for .NET، وهي مكتبة رائعة تتيح لك التعامل مع ملفات Excel برمجيًا. بنهاية هذا الدليل، ستتمكن من تغيير ألوان الخلايا في جداول بيانات Excel بسهولة.

## المتطلبات الأساسية
قبل أن نبدأ، هناك بعض الأشياء التي تحتاج إلى وضعها في مكانها:

1. Microsoft Visual Studio: ستكون هذه بيئة التطوير الخاصة بك لكتابة كود C#.
2. Aspose.Cells لـ .NET: يجب تثبيت مكتبة Aspose.Cells. يمكنك تنزيلها. [هنا](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم الأمثلة بشكل أفضل.
4. .NET Framework: تأكد من تثبيت .NET Framework أيضًا.

## استيراد الحزم
لبدء استخدام Aspose.Cells، ستحتاج إلى استيراد مساحات الأسماء اللازمة في الكود. إليك كيفية القيام بذلك:

```csharp
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

ستتيح لك هذه المساحات الوصول إلى الفئات والطرق التي ستحتاجها للتعامل مع ملفات Excel.

## الخطوة 1: إعداد دليل المستندات الخاص بكإنشاء دليل العمل الخاص بك

أولاً، أنت بحاجة إلى مكان لتخزين مستندات Excel. إليك كيفية إنشاء مجلد برمجيًا إذا لم يكن موجودًا بالفعل:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";

// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
 System.IO.Directory.CreateDirectory(dataDir);
```

في هذه القطعة، استبدل `"Your Document Directory"` بالمسار المُفضّل لديك. هذا يضمن لك مساحة عمل مُنظّمة.

## الخطوة 2: إنشاء كائن مصنف إنشاء مصنف جديد

بعد ذلك، دعنا نقوم بإنشاء مصنف جديد حيث سنعمل مع الألوان:

```csharp
// إنشاء كائن مصنف 
Workbook workbook = new Workbook();
```

يؤدي هذا السطر إلى إنشاء مثيل جديد لفئة Workbook، مما يوفر لك لوحة قماشية جديدة للعمل عليها.

## الخطوة 3: إضافة ورقة عمل جديدةإضافة ورقة عمل إلى مصنفك

الآن بعد أن أصبح لديك مصنف عمل جاهز، تحتاج إلى إضافة ورقة عمل إليه:

```csharp
// إضافة ورقة عمل جديدة إلى كائن المصنف
int i = workbook.Worksheets.Add();
```

هنا، نقوم ببساطة بإضافة ورقة عمل جديدة وتخزين فهرس الورقة المضافة حديثًا.

## الخطوة 4: الوصول إلى ورقة العمل الجديدة الحصول على مرجع لورقة العمل

الآن، دعونا نلقي نظرة على مرجع ورقة العمل التي أنشأناها للتو:

```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[i];
```

باستخدام هذا المرجع، يمكنك البدء في التعامل مع ورقة العمل بشكل مباشر.

## الخطوة 5: تحديد نمط وتطبيقه على الخلية A1: قم بتحديد نمط الخلية الأولى

حان وقت الألوان! لنُنشئ نمطًا للخلية A1:

```csharp
// قم بتحديد نمط واحصل على نمط الخلية A1
Style style = worksheet.Cells["A1"].GetStyle();

// تعيين لون المقدمة إلى الأصفر
style.ForegroundColor = Color.Yellow;

// تعيين نمط الخلفية إلى شريط عمودي
style.Pattern = BackgroundType.VerticalStripe;

// تطبيق النمط على الخلية A1
worksheet.Cells["A1"].SetStyle(style);
```

في هذه الخطوة، نحصل على النمط الحالي للخلية A1، ونغير لونها الأمامي إلى الأصفر، ونُنشئ نمطًا للخطوط العمودية، ثم نُعيد تطبيق النمط على الخلية. ها هي خليتك الملونة الأولى!

## الخطوة 6: تحديد نمط وتطبيقه على الخلية A2إبراز الخلية A2

الآن، لنُضِف بعض الألوان إلى الخلية A2. ستكون زرقاء على خلفية صفراء.

```csharp
// احصل على نمط الخلية A2
style = worksheet.Cells["A2"].GetStyle();

// تعيين لون المقدمة إلى اللون الأزرق
style.ForegroundColor = Color.Blue;

// تعيين لون الخلفية إلى الأصفر
style.BackgroundColor = Color.Yellow;

// تعيين نمط الخلفية إلى شريط عمودي
style.Pattern = BackgroundType.VerticalStripe;

// تطبيق النمط على الخلية A2
worksheet.Cells["A2"].SetStyle(style);
```

هنا، نُزيّن الخلية A2 بلون أزرق في المقدمة، ولون أصفر في الخلفية، ونستخدم أيضًا نمط الخطوط العمودية. ورقة Excel الخاصة بك تبدو نابضة بالحياة!

## الخطوة 7: احفظ مصنفكلا تنسى الحفظ!

وأخيرًا وليس آخرًا، دعنا نحفظ مصنفنا في ملف:

```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```

هذا يحفظ ملف إكسل الملون في المجلد المحدد. تذكر دائمًا حفظ عملك، فلا ترغب في إضاعة كل هذا الجهد!
