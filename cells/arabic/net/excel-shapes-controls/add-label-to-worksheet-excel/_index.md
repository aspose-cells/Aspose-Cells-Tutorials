## استيراد الحزم
قبل كتابة أي كود، يجب عليك استيراد الحزم المطلوبة إلى مشروع C# الخاص بك. إليك ما تحتاجه:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
يضمن هذا أن يتمكن مشروعك من الوصول إلى الوظائف الأساسية لـ Aspose.Cells بالإضافة إلى الفئات الإضافية اللازمة للتعامل مع الأشكال، بما في ذلك العلامات.

دعنا نوضح عملية إضافة تسمية إلى ورقة العمل الخاصة بك. وسنرشدك خلال كل خطوة، حتى تشعر بالراحة عند القيام بذلك بنفسك.
## الخطوة 1: إعداد الدليل

أول شيء عليك القيام به هو إعداد دليل لحفظ ملف الإخراج. هذا هو المكان الذي سيتواجد فيه ملف Excel الناتج.
```csharp
string dataDir = "Your Document Directory";
bool IsExists = Directory.Exists(dataDir);
if (!IsExists)
{
    Directory.CreateDirectory(dataDir);
}
```
هنا، يمكنك التحقق من وجود الدليل الذي تريد حفظ الملف فيه. إذا لم يكن موجودًا، يمكنك إنشاء الدليل. وهذا يمنع حدوث أخطاء عند محاولة حفظ الملفات لاحقًا.
## الخطوة 2: إنشاء مصنف جديد

بمجرد إعداد الدليل، فإن الخطوة التالية هي إنشاء مصنف Excel جديد.
```csharp
Workbook workbook = new Workbook();
```
يؤدي هذا إلى إنشاء مصنف عمل جديد في الذاكرة. يمكنك اعتبار الأمر بمثابة فتح ورقة Excel فارغة حيث ستضيف البيانات والأشكال والمزيد.
## الخطوة 3: الوصول إلى ورقة العمل الأولى

في ملف Excel، يمكنك الحصول على أوراق عمل متعددة. في هذا المثال، سنعمل على ورقة العمل الأولى.
```csharp
Worksheet sheet = workbook.Worksheets[0];
```
 ال`Worksheets[0]`يسترجع ورقة العمل الأولى في المصنف. يمكنك الرجوع إلى ورقة العمل هذه من خلال فهرسها أو من خلال اسمها.
## الخطوة 4: إضافة تسمية إلى ورقة العمل

الآن، دعنا نضيف تسمية إلى ورقة العمل. التسمية هي في الأساس مربع نص يمكن وضعه بحرية.
```csharp
Aspose.Cells.Drawing.Label label = sheet.Shapes.AddLabel(2, 0, 2, 0, 60, 120);
```
يضيف هذا الخط تسمية جديدة إلى ورقة العمل في الصف 2، العمود 0، بعرض 60 وارتفاع 120. تحدد المعلمات موضع التسمية وحجمها.
## الخطوة 5: تعيين نص الملصق

يمكنك إضافة نص إلى الملصق لجعله ذا معنى. دعنا نضفي عليه تعليقًا توضيحيًا.
```csharp
label.Text = "This is a Label";
```
هنا، كل ما عليك فعله هو تعيين عنوان التسمية. سيظهر هذا النص داخل التسمية في ورقة Excel الخاصة بك.
## الخطوة 6: ضبط موضع الملصق

بعد ذلك، قد ترغب في تحديد كيفية تصرف العلامة عند تغيير حجم الخلايا. سنقوم بتعيين نوع الموضع.
```csharp
label.Placement = PlacementType.FreeFloating;
```
 عن طريق ضبط نوع التنسيب إلى`FreeFloating`، تأكد من أن موضع العلامة مستقل عن تغيير حجم الخلية أو حركتها. ستبقى العلامة في المكان الذي وضعتها فيه.
## الخطوة 7: احفظ المصنف

وأخيرًا، دعنا نحفظ المصنف مع إضافة التسمية.
```csharp
workbook.Save(dataDir + "book1.out.xls");
```
 يحفظ هذا الأمر المصنف في الدليل المخصص لك باسم الملف`book1.out.xls`يمكنك فتح هذا الملف في Excel لرؤية الملصق أثناء العمل!
