# إضافة تسمية إلى ورقة العمل في Excel

## استيراد الحزم
قبل كتابة أي شيفرة برمجية، عليك استيراد الحزم المطلوبة إلى مشروع C# الخاص بك. إليك ما تحتاجه:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
يضمن هذا أن يتمكن مشروعك من الوصول إلى الوظائف الأساسية لـ Aspose.Cells بالإضافة إلى الفئات الإضافية اللازمة للتعامل مع الأشكال، بما في ذلك العلامات.

دعونا نشرح عملية إضافة تسمية إلى ورقة العمل. سنرشدك خلال كل خطوة، لتتمكن من القيام بها بنفسك.
## الخطوة 1: إعداد الدليل

أول ما عليك فعله هو إنشاء مجلد لحفظ ملف الإخراج. هذا هو المكان الذي سيُحفظ فيه ملف Excel المُنشأ.
```csharp
string dataDir = "Your Document Directory";
bool IsExists = Directory.Exists(dataDir);
if (!IsExists)
{
    Directory.CreateDirectory(dataDir);
}
```
هنا، يمكنك التحقق من وجود المجلد الذي تريد حفظ الملف فيه. إذا لم يكن موجودًا، فأنشئ المجلد. هذا يمنع حدوث أخطاء عند محاولة حفظ الملفات لاحقًا.
## الخطوة 2: إنشاء مصنف جديد

بمجرد إعداد الدليل، فإن الخطوة التالية هي إنشاء مصنف Excel جديد.
```csharp
Workbook workbook = new Workbook();
```
سيؤدي هذا إلى إنشاء مصنف جديد في الذاكرة. تخيل الأمر كما لو أنك تفتح ورقة Excel فارغة، حيث ستضيف بيانات وأشكالًا وغيرها.
## الخطوة 3: الوصول إلى ورقة العمل الأولى

في ملف Excel، يمكنك إنشاء عدة أوراق عمل. في هذا المثال، سنعمل على ورقة العمل الأولى.
```csharp
Worksheet sheet = workbook.Worksheets[0];
```
ال `Worksheets[0]` يسترجع أول ورقة عمل في المصنف. يمكنك الرجوع إلى هذه الورقة بفهرسها أو باسمها.
## الخطوة 4: إضافة تسمية إلى ورقة العمل

الآن، لنُضِف تسميةً إلى ورقة العمل. التسمية هي في الأساس مربع نص يُمكن وضعه بحرية.
```csharp
Aspose.Cells.Drawing.Label label = sheet.Shapes.AddLabel(2, 0, 2, 0, 60, 120);
```
يضيف هذا السطر تسمية جديدة إلى ورقة العمل في الصف 2، العمود 0، بعرض 60 وارتفاع 120. تحدد المعلمات موضع التسمية وحجمها.
## الخطوة 5: تعيين نص الملصق

يمكنك إضافة نص إلى الملصق لجعله أكثر وضوحًا. لنضف إليه تعليقًا توضيحيًا.
```csharp
label.Text = "This is a Label";
```
هنا، ما عليك سوى تعيين عنوان الملصق. سيظهر هذا النص داخل الملصق في ورقة Excel.
## الخطوة 6: ضبط موضع الملصق

بعد ذلك، قد ترغب في تحديد كيفية ظهور التسمية عند تغيير حجم الخلايا. سنحدد نوع الموضع.
```csharp
label.Placement = PlacementType.FreeFloating;
```
عن طريق ضبط نوع التنسيب إلى `FreeFloating`، تضمن أن يكون موضع الملصق مستقلاً عن تغيير حجم الخلية أو حركتها. سيبقى الملصق في مكانه.
## الخطوة 7: حفظ المصنف

وأخيرًا، دعنا نحفظ المصنف مع إضافة التسمية.
```csharp
workbook.Save(dataDir + "book1.out.xls");
```
يحفظ هذا الأمر المصنف في الدليل المخصص لك باسم الملف `book1.out.xls`يمكنك فتح هذا الملف في Excel لرؤية الملصق أثناء العمل!
