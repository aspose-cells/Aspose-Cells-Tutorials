# إضافة عنصر التحكم الدوار إلى ورقة العمل في Excel

## مقدمة
إذا كنتَ تتعمق في عالم أتمتة Excel باستخدام .NET، فربما لاحظتَ الحاجة إلى عناصر تحكم أكثر تفاعلية في جداول بياناتك. أحد هذه العناصر هو Spinner، الذي يُتيح للمستخدمين زيادة أو إنقاص قيمة بسهولة. في هذا البرنامج التعليمي، سنستكشف كيفية إضافة عنصر تحكم Spinner إلى ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. سنُقسّم العملية إلى خطوات سهلة الفهم لتتمكن من متابعتها بسلاسة. 
## المتطلبات الأساسية
قبل أن ننتقل إلى الكود، دعنا نتأكد من إعداد كل شيء للحصول على تجربة سلسة:
1. Aspose.Cells لـ .NET: تأكد من تثبيت مكتبة Aspose.Cells. إذا لم تقم بتثبيتها بعد، يمكنك تنزيل أحدث إصدار من [رابط التحميل](https://releases.aspose.com/cells/net/).
2. Visual Studio: يجب أن يكون لديك تثبيت عمل لبرنامج Visual Studio أو أي .NET IDE آخر تفضله.
3. المعرفة الأساسية بلغة C#: ستساعدك معرفة برمجة C# على فهم مقتطفات الشيفرة البرمجية بسهولة. إذا كنت مبتدئًا، فلا تقلق! سأشرح لك كل جزء.
## استيراد الحزم
لاستخدام Aspose.Cells في مشروعك، عليك استيراد مساحات الأسماء اللازمة. إليك كيفية إعداد بيئتك:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
تتيح لك مساحات الأسماء هذه الوصول إلى الوظائف الأساسية لـ Aspose.Cells، بما في ذلك معالجة المصنف وإمكانيات الرسم للأشكال مثل Spinner.
بعد أن استعرضنا المتطلبات الأساسية واستوردنا الحزم اللازمة، لنبدأ بالدليل خطوة بخطوة. كل خطوة واضحة وموجزة ليسهل عليك تنفيذها.
## الخطوة 1: إعداد دليل المشروع الخاص بك
قبل البدء بالبرمجة، يُنصح بتنظيم ملفاتك. لنُنشئ مجلدًا لملفات Excel.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
هنا، نحدد مسارًا لمجلد المستندات. إذا لم يكن المجلد موجودًا، فننشئه. هذا يضمن وجود عنوان رئيسي مُحدد لجميع ملفاتنا المُولّدة.
## الخطوة 2: إنشاء مصنف جديد
الآن حان الوقت لإنشاء مصنف Excel حيث سنضيف عنصر التحكم Spinner الخاص بنا.
```csharp
// إنشاء مصنف جديد.
Workbook excelbook = new Workbook();
```
ال `Workbook` تُمثل الفئة ملف Excel. بإنشاء مثيل لها، ننشئ مصنفًا جديدًا جاهزًا للتعديل.
## الخطوة 3: الوصول إلى ورقة العمل الأولى
سنضيف Spinner الخاص بنا إلى ورقة العمل الأولى في المصنف.
```csharp
// احصل على ورقة العمل الأولى.
Worksheet worksheet = excelbook.Worksheets[0];
```
يصل هذا السطر إلى ورقة العمل الأولى (الفهرس ٠) من مصنفنا. يمكنك إنشاء عدة أوراق عمل، ولكن في هذا المثال، سنبسط الأمر.
## الخطوة 4: العمل مع الخلايا
الآن، لنبدأ العمل على خلايا ورقة العمل. سنحدد بعض القيم والأنماط.
```csharp
// احصل على خلايا ورقة العمل.
Cells cells = worksheet.Cells;
// أدخل قيمة السلسلة في الخلية A1.
cells["A1"].PutValue("Select Value:");
// تعيين لون الخط للخلية.
cells["A1"].GetStyle().Font.Color = Color.Red;
// تعيين خط النص عريضًا.
cells["A1"].GetStyle().Font.IsBold = true;
// أدخل القيمة في الخلية A2.
cells["A2"].PutValue(0);
```
هنا، نقوم بملء الخلية A1 بموجه، ونضع لونًا أحمر، ونجعل النص غامقًا. كما قمنا بتعيين قيمة أولية للخلية A2 إلى 0، والتي سيتم ربطها بأداة التدوير.
## الخطوة 5: تصميم خلية A2
بعد ذلك، دعنا نطبق بعض الأنماط على الخلية A2 لجعلها أكثر جاذبية بصريًا.
```csharp
// ضبط لون التظليل إلى الأسود مع خلفية صلبة.
cells["A2"].GetStyle().ForegroundColor = Color.Black;
cells["A2"].GetStyle().Pattern = BackgroundType.Solid;
// تعيين لون الخط للخلية.
cells["A2"].GetStyle().Font.Color = Color.White;
// تعيين خط النص عريضًا.
cells["A2"].GetStyle().Font.IsBold = true;
```
نضيف خلفية سوداء بنمط موحد إلى الخلية A2، ونجعل لون الخط أبيض. هذا التباين سيجعلها بارزة في ورقة العمل.
## الخطوة 6: إضافة عنصر التحكم الدوار
الآن، نحن جاهزون لإضافة عنصر التحكم Spinner إلى ورقة العمل الخاصة بنا.
```csharp
// أضف عنصر التحكم الدوار.
Aspose.Cells.Drawing.Spinner spinner = excelbook.Worksheets[0].Shapes.AddSpinner(1, 0, 1, 0, 20, 18);
```
يضيف هذا السطر عنصر تحكم Spinner إلى ورقة العمل. تحدد المعلمات موضع وحجم Spinner (الصف، العمود، العرض، الارتفاع).
## الخطوة 7: تكوين خصائص الدوار
دعونا نقوم بتخصيص سلوك Spinner ليناسب احتياجاتنا.
```csharp
// ضبط نوع وضع الدوار.
spinner.Placement = PlacementType.FreeFloating;
// تعيين الخلية المرتبطة بالتحكم.
spinner.LinkedCell = "A2";
// تعيين الحد الأقصى للقيمة.
spinner.Max = 10;
// تعيين الحد الأدنى للقيمة.
spinner.Min = 0;
// تعيين تغيير الزيادة لعنصر التحكم.
spinner.IncrementalChange = 2;
// ضبط التظليل ثلاثي الأبعاد.
spinner.Shadow = true;
```
هنا، نُعيّن خصائص Spinner. نربطه بالخلية A2، مما يسمح له بالتحكم في القيمة المعروضة هناك. تُحدّد القيم الدنيا والقصوى النطاق الذي يُمكن لـ Spinner العمل ضمنه، بينما يُحدّد التغيير التدريجي مقدار تغيّر القيمة مع كل نقرة. إضافة تظليل ثلاثي الأبعاد يُضفي عليه مظهرًا أنيقًا.
## الخطوة 8: حفظ ملف Excel
أخيرًا، دعنا نحفظ مصنف Excel الخاص بنا مع تضمين Spinner.
```csharp
// احفظ ملف الاكسل.
excelbook.Save(dataDir + "book1.out.xls");
```
يحفظ هذا الأمر المصنف في المجلد المحدد. يمكنك تغيير اسم الملف حسب الحاجة.
