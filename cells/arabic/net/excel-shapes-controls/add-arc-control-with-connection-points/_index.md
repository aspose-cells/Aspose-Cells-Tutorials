# إضافة التحكم بالقوس مع نقاط الاتصال

## مقدمة
عند إنشاء تقارير إكسل جذابة بصريًا، تلعب الرسوم التوضيحية دورًا حيويًا. سواء كنت تُعدّ تقريرًا ماليًا أو تحليلًا تفصيليًا لمشروع، فإن استخدام أشكال مثل الأقواس يُضفي عمقًا ووضوحًا على عرض بياناتك. اليوم، سنتعمق في كيفية استخدام Aspose.Cells لـ .NET لإضافة عناصر تحكم أقواس مع نقاط اتصال في جداول بيانات إكسل. لذا، إذا تساءلت يومًا عن كيفية تحسين جداول بياناتك أو جعل بياناتك أكثر وضوحًا، فتابع القراءة!
## المتطلبات الأساسية
قبل أن نخوض في عالم البرمجة، لنتأكد من جاهزيتك. إليك ما تحتاجه:
1. إطار عمل .NET: تأكد من تثبيت إصدار متوافق. يعمل Aspose.Cells مع إصدارات متعددة، بما في ذلك .NET Core.
2. Aspose.Cells لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.Cells وتثبيتها. يمكنك الحصول عليها بسهولة من [رابط التحميل](https://releases.aspose.com/cells/net/).
3. بيئة تطوير متكاملة جيدة: Visual Studio، الرفيق المخلص لأي مطور .NET، سيساعدك على تبسيط تجربة الترميز الخاصة بك.
4. المعرفة الأساسية بلغة C#: إذا كنت تعرف طريقك حول لغة C#، فستجد أن هذا البرنامج التعليمي يسير بسلاسة.
5. الوصول إلى دليل مستنداتك: اعرف مكان حفظ ملفات Excel. هذا ضروري لتنظيم مخرجاتك بكفاءة.
## استيراد الحزم
الخطوة التالية هي التأكد من استيراد الحزم الصحيحة إلى مشروعك. يحتوي Aspose.Cells لـ .NET على وظائف متنوعة، لذا سنبسطه. إليك ما ستحتاج إلى تضمينه:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
ستتيح لك هذه المساحات الوصول إلى جميع ميزات الرسم ووظائف إدارة الخلايا التي ستستخدمها خلال هذا الدليل.
## الخطوة 1: إعداد دليل المستندات الخاص بك
أولاً، لنُنشئ مجلداً لحفظ ملفات إكسل الجديدة. إليك الطريقة:
```csharp
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
يتحقق هذا الكود من وجود المجلد المحدد. إن لم يكن موجودًا، يُنشئ واحدًا. بسيط، أليس كذلك؟ من الجيد دائمًا تخصيص مكان لملفاتك لتجنب الفوضى.
## الخطوة 2: إنشاء مصنف
الآن بعد أن أصبح الدليل جاهزًا، فلنقم بإنشاء مصنف Excel جديد.
```csharp
Workbook excelbook = new Workbook();
```
عن طريق الاتصال `Workbook` من خلال إنشاء ملف Excel جديد، فإنك تقول في الأساس: "مرحبًا، فلنبدأ ملف Excel جديدًا!" سيكون هذا هو القماش لجميع الأشكال والبيانات الخاصة بك.
## الخطوة 3: إضافة شكل القوس الأول
هنا تبدأ المتعة! لنُضيف شكل قوسنا الأول.
```csharp
Aspose.Cells.Drawing.ArcShape arc1 = excelbook.Worksheets[0].Shapes.AddArc(2, 0, 2, 0, 130, 130);
```
يضيف هذا السطر من التعليمات البرمجية شكل قوس إلى ورقة العمل الأولى. تحدد المعلمات إحداثيات القوس والزوايا التي تحدد انحنائه. 
## الخطوة 4: تخصيص مظهر القوس
إن شكل القوس الفارغ يشبه لوحة قماشية بدون طلاء - فهو يحتاج إلى القليل من الذوق!
### تعيين لون تعبئة القوس
```csharp
arc1.Fill.FillType = FillType.Solid;
arc1.Fill.SolidFill.Color = Color.Blue;
```
هذا يجعل القوس أزرقًا ثابتًا. يمكنك تغيير اللون إلى أي درجة لونية تريدها عن طريق تبديل `Color.Blue` للحصول على لون آخر.
### تعيين موضع القوس
```csharp
arc1.Placement = PlacementType.FreeFloating;
```
يتيح ضبط الموضع على "FreeFloating" للقوس التحرك بشكل مستقل عن حدود الخلية، مما يمنحك المرونة في تحديد الموضع.
### ضبط وزن الخط ونمطه
```csharp
arc1.Line.Weight = 1;      
arc1.Line.DashStyle = MsoLineDashStyle.Solid;
```
هنا، يمكنك تحديد وزن الخط ونمطه، مما يجعله أكثر بروزًا وجاذبية بصريًا.
## الخطوة 5: إضافة شكل قوس آخر
لماذا نتوقف عند واحد فقط؟ لنُضِف شكل قوس آخر لإثراء عرضنا المرئي في إكسل.
```csharp
Aspose.Cells.Drawing.ArcShape arc2 = excelbook.Worksheets[0].Shapes.AddArc(9, 0, 2, 0, 130, 130);
```
مثل القوس الأول، تمت إضافة هذا القوس في موضع مختلف - وهذا هو المكان الذي يحدث فيه سحر التصميم!
## الخطوة 6: تخصيص القوس الثاني
دعونا نعطي قوسنا الثاني بعض الشخصية أيضًا!
### تغيير لون خط القوس
```csharp
arc2.Line.FillType = FillType.Solid;
arc2.Line.SolidFill.Color = Color.Blue;
```
نحن نحافظ على الاتساق مع اللون الأزرق، ولكن يمكنك دائمًا المزج والمطابقة لمعرفة ما يناسب تصميمك بشكل أفضل!
### تعيين خصائص مشابهة للقوس الأول
تأكد من تكرار تلك الاختيارات الجمالية:
```csharp
arc2.Placement = PlacementType.FreeFloating;
arc2.Line.Weight = 1;           
arc2.Line.DashStyle = MsoLineDashStyle.Solid;
```
هنا، عليك ببساطة التأكد من أن القوس الثاني يتطابق مع القوس الأول، مما يؤدي إلى إنشاء مظهر متماسك في جميع أنحاء ورقة العمل الخاصة بك.
## الخطوة 7: احفظ مصنفك
لا تكتمل أي تحفة فنية دون حفظها، أليس كذلك؟ حان الوقت لكتابة أقواسك في ملف إكسل.
```csharp
excelbook.Save(dataDir + "book1.out.xls");
```
يحفظ هذا السطر الأقواس التي قمت بإنشائها حديثًا في ملف Excel يسمى "book1.out.xls" في الدليل المخصص لك.
## خاتمة
تهانينا! لقد أتقنتَ للتو أساسيات إضافة عناصر تحكم القوس مع نقاط الاتصال في جداول بيانات Excel باستخدام Aspose.Cells لـ .NET. لا تُضفي هذه الميزة جمالاً على جداول بياناتك فحسب، بل تُسهّل أيضاً استيعاب البيانات المعقدة. سواءً كنتَ مطوراً محترفاً أو مبتدئاً، فإن هذه العناصر المرئية قادرة على تحويل تقاريرك من مملة إلى رائعة.

Aspose.Cells هي مكتبة .NET قوية تسمح للمطورين بإنشاء ملفات Excel ومعالجتها برمجيًا.

### ما نوع الملفات التي يمكنني إنشاؤها باستخدام Aspose.Cells؟
يمكنك إنشاء ومعالجة تنسيقات Excel المختلفة بما في ذلك XLS وXLSX وCSV والمزيد.
