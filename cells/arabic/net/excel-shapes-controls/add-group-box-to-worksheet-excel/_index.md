# إضافة مربع المجموعة إلى ورقة العمل في Excel

## مقدمة
عندما يتعلق الأمر بعرض البيانات، يُعدّ Excel الخيار الأمثل. إضافة عناصر تفاعلية، مثل مربعات المجموعات، تجعل جداول بياناتك أكثر جاذبية وسهولة في الاستخدام. اليوم، نتعمق في عالم Aspose.Cells لـ .NET، وهي مكتبة فعّالة تُساعدك على التعامل مع جداول بيانات Excel بسهولة. ولكن لا تقلق إذا لم تكن خبيرًا في البرمجة، فهذا الدليل يُفصّل كل شيء في خطوات بسيطة. هل أنت مستعد لتطوير مهاراتك في Excel؟ هيا بنا!
## المتطلبات الأساسية
قبل أن ننتقل إلى الكود، هناك بعض الأشياء التي ستحتاجها:
1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك؛ فهو المكان الذي ستكتب فيه كود .NET.
2. Aspose.Cells لـ .NET: تحتاج إلى تنزيل هذه المكتبة. يمكنك العثور عليها [هنا](https://releases.aspose.com/cells/net/). 
3. المعرفة الأساسية بلغة C#: على الرغم من أنني سأشرح كل شيء خطوة بخطوة، إلا أن القليل من الفهم بلغة C# سيساعدك على المتابعة.
## استيراد الحزم
لأي مشروع، ستحتاج أولًا إلى استيراد الحزم اللازمة. هنا، سيكون Aspose.Cells هو محور تركيزك. إليك كيفية القيام بذلك:
## الخطوة 1: افتح مشروعك في Visual Studio
قم بتشغيل Visual Studio وافتح مشروعك الحالي أو قم بإنشاء مشروع جديد. 
## الخطوة 2: إضافة مرجع إلى Aspose.Cells
- انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- حدد "إدارة حزم NuGet".
- ابحث عن "Aspose.Cells" وثبّته. سيُمكّنك هذا من استخدام جميع الفئات والأساليب التي تُوفّرها مكتبة Aspose.Cells.
## الخطوة 3: تضمين استخدام التوجيه
في أعلى ملف C# الخاص بك، قم بتضمين مساحة اسم Aspose.Cells:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
يتيح لك هذا الوصول إلى الفئات اللازمة للعمل مع ملفات Excel.
بعد أن انتهينا من الإعداد، لننتقل إلى جوهر الدرس: إضافة مربع مجموعة بأزرار اختيارية إلى ورقة عمل Excel. سنُقسّم هذه العملية إلى عدة خطوات للتوضيح.
## الخطوة 1: إعداد دليل المستندات الخاص بك
قبل إنشاء أي ملف إكسل، عليك تحديد مكان حفظه. لنقم بإنشاء مجلد إذا لم يكن موجودًا مسبقًا.
```csharp
// المسار إلى دليل المستندات
string dataDir = "Your Document Directory"; // حدد المسار المطلوب
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
يتحقق هذا الكود من وجود المجلد الذي سيُحفظ فيه ملف Excel. إذا لم يكن موجودًا، يُنشئ مجلدًا - يشبه تجهيز مساحة العمل قبل البدء بالمشروع!
## الخطوة 2: إنشاء مصنف جديد
بعد ذلك، ستحتاج إلى إنشاء مصنف Excel حيث ستضيف مربع المجموعة الخاص بك.
```csharp
// إنشاء مصنف جديد.
Workbook excelbook = new Workbook();
```
يُنشئ هذا السطر نسخة جديدة من مصنف. تخيل هذا كأنك تفتح ملف Excel جديدًا فارغًا جاهزًا للتعديل.
## الخطوة 3: إضافة مربع المجموعة
الآن، دعونا نضيف صندوق المجموعة هذا. 
```csharp
// أضف مربع المجموعة إلى ورقة العمل الأولى.
GroupBox box = excelbook.Worksheets[0].Shapes.AddGroupBox(1, 0, 1, 0, 300, 250);
```
هنا، تُضيف مربع مجموعة عند إحداثيات مُحددة في ورقة العمل الأولى. تُحدد المعلمات موضع وحجم المربع، تمامًا مثل وضع الأثاث في الغرفة!
## الخطوة 4: تعيين تسمية توضيحية لمربع المجموعة
الآن، دعونا نعطي صندوق مجموعتك عنوانًا!
```csharp
// تعيين تسمية توضيحية لمربع المجموعة.
box.Text = "Age Groups";
box.Placement = PlacementType.FreeFloating;
```
سلسلة "الفئات العمرية" تُحدد التسمية التي تظهر في مربع المجموعة. ضبط `Placement` مثل `FreeFloating` يسمح للصندوق بأن يكون متحركًا - المرونة هي المفتاح!
## الخطوة 5: اجعل مربع المجموعة ثنائي الأبعاد
رغم أن المظهر ثلاثي الأبعاد قد يبدو رائعًا، إلا أننا نهدف إلى مظهر كلاسيكي هنا.
```csharp
// اجعله صندوقًا ثنائي الأبعاد.
box.Shadow = false;
```
يقوم هذا الكود بإزالة تأثير الظل، مما يعطي الصندوق مظهرًا مسطحًا - مثل ورقة بسيطة!
## الخطوة 6: إضافة أزرار الراديو
دعونا نضفي بعض الإثارة على الأمور من خلال إضافة بعض أزرار الاختيار لإدخال المستخدم.
## الخطوة 6.1: إضافة زر الاختيار الأول
```csharp
// إضافة زر الاختيار.
Aspose.Cells.Drawing.RadioButton radio1 = excelbook.Worksheets[0].Shapes.AddRadioButton(3, 0, 2, 0, 30, 110);
// تعيين سلسلة النص الخاصة به.
radio1.Text = "20-29";
// تعيين الخلية A1 كخلية مرتبطة لزر الاختيار.
radio1.LinkedCell = "A1";
```
أنشئ زر اختيار للفئة العمرية ٢٠-٢٩، واربطه بالخلية A1 في ورقة العمل. هذا يعني أنه عند تحديد هذا الزر، تعكس الخلية A1 هذا الاختيار!
## الخطوة 6.2: تخصيص زر الراديو الأول
الآن دعونا نعطيه بعض الأناقة.
```csharp
// جعل زر الراديو ثلاثي الأبعاد.
radio1.Shadow = true;
// ضبط وزن زر الراديو.
radio1.Line.Weight = 4;
// تعيين نمط الشرطة لزر الاختيار.
radio1.Line.DashStyle = MsoLineDashStyle.Solid;
```
بإضافة ظل وتعديل نمط الخط، نُحسّن رؤية الزر. الأمر أشبه بإضافة زخارف لجعله بارزًا في الصفحة!
## الخطوة 6.3: كرر ذلك لمزيد من أزرار الاختيار
كرر هذه العملية للمجموعات العمرية الإضافية:
```csharp
// زر الراديو الثاني
Aspose.Cells.Drawing.RadioButton radio2 = excelbook.Worksheets[0].Shapes.AddRadioButton(6, 0, 2, 0, 30, 110);
radio2.Text = "30-39";
radio2.LinkedCell = "A1";
radio2.Shadow = true;
radio2.Line.Weight = 4;
radio2.Line.DashStyle = MsoLineDashStyle.Solid;
// زر الراديو الثالث
Aspose.Cells.Drawing.RadioButton radio3 = excelbook.Worksheets[0].Shapes.AddRadioButton(9, 0, 2, 0, 30, 110);
radio3.Text = "40-49";
radio3.LinkedCell = "A1";
radio3.Shadow = true;
radio3.Line.Weight = 4;
radio3.Line.DashStyle = MsoLineDashStyle.Solid;
```
يُمكِّن كل زر اختيار فئات عمرية مختلفة، ويرتبط بالخلية A1 نفسها. هذا يُسهِّل عملية الاختيار ويسهلها.
## الخطوة 7: تجميع الأشكال
وبعد أن أصبح كل شيء في مكانه، فلنبدأ في ترتيب الأشياء عن طريق تجميع أشكالنا. 
```csharp
// احصل على الأشكال.
Aspose.Cells.Drawing.Shape[] shapeobjects = new Shape[] { box, radio1, radio2, radio3 };
// تجميع الأشكال.
Aspose.Cells.Drawing.GroupShape group = excelbook.Worksheets[0].Shapes.Group(shapeobjects);
```
تجمع هذه الخطوة كل شيء في وحدة واحدة متماسكة. يشبه الأمر وضع إطار حول مجموعتك الفنية، فهو يربطها معًا بشكل جميل!
## الخطوة 8: حفظ ملف Excel
وأخيرا، دعونا نحفظ تحفتنا الفنية!
```csharp
// احفظ ملف الاكسل.
excelbook.Save(dataDir + "book1.out.xls");
```
هذا السطر من التعليمات البرمجية يُدخل تغييراتك إلى ملف إكسل جديد باسم "book1.out.xls" في المجلد المُحدد. وكأنك تُغلق ظرفًا، أصبح عملك الآن مُخزّنًا بأمان!
## خاتمة
هذا كل ما في الأمر - دليل شامل لإضافة مربع مجموعة وأزرار اختيار إلى ورقة عمل Excel باستخدام Aspose.Cells لـ .NET! مع كل خطوة، ستتعلم كيفية التعامل مع Excel برمجيًا، مما يفتح لك آفاقًا لا حصر لها لتخصيص التقارير وتصور البيانات وغيرها. يكمن جمال البرمجة في إمكانية أتمتة المهام وإنشاء واجهات سهلة الاستخدام بسهولة نسبية - تخيل الإمكانات!
