# إضافة مربع الاختيار إلى ورقة العمل في Excel

## مقدمة
عندما يتعلق الأمر بإدارة البيانات في Excel، فهناك عدد لا يحصى من الوظائف والطرق التي يمكنها تبسيط مهامك وتحسين جداول البيانات الخاصة بك. إحدى هذه الميزات هي مربع الاختيار - وهي أداة صغيرة أنيقة تسمح للمستخدمين بإجراء اختيارات ثنائية مباشرة داخل جداول عمل Excel الخاصة بهم. في هذا الدليل، سنرشدك خلال عملية إضافة مربع اختيار إلى ورقة عمل Excel باستخدام مكتبة Aspose.Cells لـ .NET. لذا، استعد لرحلة مثيرة في عالم أتمتة Excel!
## المتطلبات الأساسية
قبل أن نتعمق في التفاصيل الدقيقة للترميز، دعنا نتأكد من أنك تمتلك كل ما تحتاجه للبدء. فيما يلي المتطلبات الأساسية:
- فيجوال ستوديو: نفترض أن لديك بيئة عمل تم إعدادها باستخدام Visual Studio. إذا لم يكن الأمر كذلك، فيمكنك تنزيله بسهولة من[Visual Studio](https://visualstudio.microsoft.com/vs/).
- .NET Framework: تأكد من تثبيت .NET Framework على نظامك. تحقق من توافق Aspose.Cells مع إصدار .NET الخاص بك.
-  Aspose.Cells لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.Cells والإشارة إليها في مشروعك. يمكنك تنزيلها من[هنا](https://releases.aspose.com/cells/net/).
- الفهم الأساسي لبرمجة C#: سيساعدك الفهم الأساسي لبرمجة C# على متابعة الأمثلة بسهولة أكبر.
بعد التحقق من هذه المتطلبات الأساسية في قائمتك، فلنبدأ!
## استيراد الحزم
قبل أن نبدأ في كتابة التعليمات البرمجية، نحتاج إلى استيراد الحزم اللازمة إلى مشروع C# الخاص بنا. تعد مكتبة Aspose.Cells ضرورية لمهمتنا، واستيرادها سهل للغاية. ما عليك سوى اتباع الخطوات التالية:
### إنشاء مشروع C# جديد
- افتح Visual Studio وقم بإنشاء تطبيق وحدة تحكم C# جديد.
### إضافة مرجع إلى Aspose.Cells
- انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- حدد "إدارة حزم NuGet".
- في مدير الحزم NuGet، ابحث عن "Aspose.Cells" وقم بتثبيته.
### استيراد مساحة الاسم
في الجزء العلوي من ملف Program.cs، قم بتضمين المرجع التالي إلى مساحة اسم Aspose.Cells:
```csharp
using System.IO;
using Aspose.Cells;
```
الآن، أنت جاهز تمامًا لبدء البرمجة!

الآن سنبدأ في العمل. فيما يلي تعليمات خطوة بخطوة حول كيفية إضافة مربع اختيار إلى ورقة عمل Excel باستخدام Aspose.Cells.
## الخطوة 1: إعداد الدليل
أولاً، نحتاج إلى التأكد من وجود الدليل لحفظ ملف Excel. هذه خطوة بالغة الأهمية لأنها تمنع حدوث أخطاء وقت التشغيل عندما نحاول حفظ ملفنا.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
## الخطوة 2: إنشاء مصنف جديد
بعد ذلك، نحتاج إلى إنشاء مثيل جديد لكتاب العمل. سيعمل هذا المثيل كأساس لملف Excel بأكمله.
```csharp
// إنشاء مصنف جديد.
Workbook excelBook = new Workbook();
```
## الخطوة 3: إضافة مربع اختيار إلى ورقة العمل
 الآن، دعنا نضيف مربع اختيار إلى ورقة العمل الأولى من المصنف الخاص بنا. يمكنك تحديد موضع وحجم مربع الاختيار باستخدام`Add` طريقة:
```csharp
// أضف مربع اختيار إلى ورقة العمل الأولى في المصنف.
int index = excelBook.Worksheets[0].CheckBoxes.Add(5, 5, 100, 120);
```
## الخطوة 4: الحصول على كائن مربع الاختيار
بمجرد إضافة مربع الاختيار، نحتاج إلى استرداد كائن مربع الاختيار لإجراء المزيد من التخصيصات.
```csharp
// احصل على كائن مربع الاختيار.
Aspose.Cells.Drawing.CheckBox checkbox = excelBook.Worksheets[0].CheckBoxes[index];
```
## الخطوة 5: تعيين نص مربع الاختيار
ما معنى مربع الاختيار بدون تسمية؟ دعنا نضفي على مربع الاختيار بعض النصوص حتى يعرف المستخدمون ما هو المقصود به!
```csharp
// تعيين سلسلة النص الخاصة به.
checkbox.Text = "Click it!";
```
## الخطوة 6: ربط مربع الاختيار بخلية
يتيح لنا ربط مربع الاختيار بخلية معينة تتبع حالته بسهولة. في هذه الحالة، سنربطه بالخلية B1.
```csharp
// ضع قيمة في الخلية B1.
excelBook.Worksheets[0].Cells["B1"].PutValue("LnkCell");
// تعيين الخلية B1 كخلية مرتبطة بمربع الاختيار.
checkbox.LinkedCell = "B1";
```
## الخطوة 7: تعيين قيمة مربع الاختيار الافتراضية
إذا كنت تريد أن يتم تحديد مربع الاختيار افتراضيًا عند فتح الملف، فيمكنك القيام بذلك بسهولة أيضًا!
```csharp
// قم بتحديد مربع الاختيار افتراضيا.
checkbox.Value = true;
```
## الخطوة 8: حفظ ملف Excel
وأخيرًا، بعد كل هذه الخطوات، حان الوقت لحفظ تحفتنا الفنية في الدليل المحدد. 
```csharp
// احفظ ملف Excel.
excelBook.Save(dataDir + "book1.out.xls");
```
وبهذا تكون قد أنشأت ملف Excel يحتوي على مربع اختيار فعال!
