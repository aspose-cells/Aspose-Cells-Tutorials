# إضافة مربع الاختيار إلى ورقة العمل في Excel

## مقدمة
عندما يتعلق الأمر بإدارة البيانات في Excel، هناك العديد من الوظائف والأساليب التي تُسهّل مهامك وتُحسّن أداء جداول بياناتك. من هذه الميزات مربع الاختيار، وهو أداة صغيرة أنيقة تُتيح للمستخدمين اختيار خيارات ثنائية مباشرةً داخل أوراق عمل Excel. في هذا الدليل، سنشرح لك عملية إضافة مربع اختيار إلى ورقة عمل Excel باستخدام مكتبة Aspose.Cells لـ .NET. لذا، استعد لرحلة شيّقة في عالم أتمتة Excel!
## المتطلبات الأساسية
قبل أن نتعمق في أساسيات البرمجة، لنتأكد من امتلاكك كل ما تحتاجه للبدء. إليك المتطلبات الأساسية:
- فيجوال ستوديو: نفترض أن لديك بيئة عمل مُجهزة بفيجوال ستوديو. إذا لم يكن الأمر كذلك، يمكنك تنزيله بسهولة من [فيجوال ستوديو](https://visualstudio.microsoft.com/vs/).
- إطار عمل .NET: تأكد من تثبيت إطار عمل .NET على نظامك. تحقق من توافق Aspose.Cells مع إصدار .NET لديك.
- Aspose.Cells لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.Cells والإشارة إليها في مشروعك. يمكنك تنزيلها من [هنا](https://releases.aspose.com/cells/net/).
- الفهم الأساسي لبرمجة C#: إن الفهم الأساسي لبرمجة C# سيساعدك على متابعة الأمثلة بسهولة أكبر.
بعد التحقق من هذه المتطلبات الأساسية في قائمتك، فلنبدأ!
## استيراد الحزم
قبل البدء بالبرمجة، علينا استيراد الحزم اللازمة إلى مشروع C#. مكتبة Aspose.Cells أساسية لمهمتنا، واستيرادها سهل للغاية. ما عليك سوى اتباع الخطوات التالية:
### إنشاء مشروع C# جديد
- افتح Visual Studio وقم بإنشاء تطبيق وحدة تحكم C# جديد.
### إضافة مرجع إلى Aspose.Cells
- انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- حدد "إدارة حزم NuGet".
- في مدير الحزم NuGet، ابحث عن "Aspose.Cells" وقم بتثبيته.
### استيراد مساحة الاسم
في أعلى ملف Program.cs، قم بتضمين المرجع التالي إلى مساحة اسم Aspose.Cells:
```csharp
using System.IO;
using Aspose.Cells;
```
الآن، أنت جاهز تمامًا لبدء البرمجة!

الآن، لنبدأ العمل. فيما يلي تعليمات خطوة بخطوة حول كيفية إضافة مربع اختيار إلى ورقة عمل Excel باستخدام Aspose.Cells.
## الخطوة 1: إعداد الدليل
أولاً، علينا التأكد من وجود مجلد حفظ ملف Excel. هذه خطوة بالغة الأهمية لأنها تمنع حدوث أخطاء وقت التشغيل عند محاولة حفظ الملف.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
## الخطوة 2: إنشاء مصنف جديد
بعد ذلك، علينا إنشاء نسخة جديدة من مصنف العمل. سيُشكّل هذا أساسًا لملف Excel بأكمله.
```csharp
// إنشاء مصنف جديد.
Workbook excelBook = new Workbook();
```
## الخطوة 3: إضافة مربع اختيار إلى ورقة العمل
الآن، لنُضِف مربع اختيار إلى ورقة العمل الأولى من مصنفنا. يمكنك تحديد موضع وحجم مربع الاختيار باستخدام `Add` طريقة:
```csharp
// أضف مربع اختيار إلى ورقة العمل الأولى في المصنف.
int index = excelBook.Worksheets[0].CheckBoxes.Add(5, 5, 100, 120);
```
## الخطوة 4: الحصول على كائن مربع الاختيار
بمجرد إضافة مربع الاختيار، نحتاج إلى استرداد كائن مربع الاختيار لإجراء المزيد من التخصيصات.
```csharp
// احصل على كائن مربع الاختيار.
Aspose.Cells.Drawing.CheckBox checkbox = excelBook.Worksheets[0].CheckBoxes[index];
```
## الخطوة 5: تعيين نص مربع الاختيار
ما فائدة مربع الاختيار بدون تسمية؟ لنُضيف نصًا إلى مربع الاختيار ليفهم المستخدمون معناه!
```csharp
// تعيين سلسلة النص الخاصة به.
checkbox.Text = "Click it!";
```
## الخطوة 6: ربط مربع الاختيار بخلية
ربط مربع الاختيار بخلية محددة يُسهّل علينا تتبع حالته. في هذه الحالة، سنربطه بالخلية B1.
```csharp
// ضع قيمة في الخلية B1.
excelBook.Worksheets[0].Cells["B1"].PutValue("LnkCell");
// قم بتعيين الخلية B1 كخلية مرتبطة بمربع الاختيار.
checkbox.LinkedCell = "B1";
```
## الخطوة 7: تعيين قيمة مربع الاختيار الافتراضية
إذا كنت تريد أن يتم تحديد مربع الاختيار بشكل افتراضي عند فتح الملف، فيمكنك القيام بذلك بسهولة أيضًا!
```csharp
// قم بتحديد مربع الاختيار افتراضيًا.
checkbox.Value = true;
```
## الخطوة 8: حفظ ملف Excel
وأخيرًا، بعد كل هذه الخطوات، حان الوقت لحفظ تحفتنا الفنية في الدليل المحدد. 
```csharp
// احفظ ملف الاكسل.
excelBook.Save(dataDir + "book1.out.xls");
```
وهكذا، تكون قد أنشأت ملف Excel يحتوي على مربع اختيار فعال!
