# إضافة شكل بيضاوي إلى ورقة العمل في Excel

إنشاء ملفات Excel رائعة وتفاعلية لا يقتصر على الأرقام والصيغ فحسب. فالأشكال البيضاوية تُضفي لمسة جمالية أو تُوفر عناصر وظيفية في أوراق العمل. في هذا البرنامج التعليمي، سنستكشف كيفية استخدام Aspose.Cells لـ .NET لإضافة أشكال بيضاوية إلى ورقة عمل Excel برمجيًا. سواء كنت ترغب في إضافة لمسة جمالية أو وظائف إضافية، فلدينا دليل خطوة بخطوة يُفصّل كل شيء.

## استيراد الحزم
قبل كتابة أي شيفرة برمجية، تأكد من تضمين مساحات الأسماء المطلوبة. إليك مقتطف شيفرة C# لضمان استخدامك للمكتبات الصحيحة:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
## الخطوة 1: إعداد الدليل الخاص بك
الخطوة الأولى لإضافة شكل بيضاوي إلى ورقة Excel هي تحديد مكان حفظ ملف Excel. لنحدد مسار المجلد ونتأكد من وجوده قبل حفظ العمل.

سننشئ مسارًا للدليل ونتحقق من وجوده. إذا لم يكن المجلد موجودًا، فسيتم إنشاؤه.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
تعتبر هذه الخطوة بالغة الأهمية لأنها تضمن حفظ ملفك في موقع مناسب، وعدم مواجهة مشكلات مسار الملف لاحقًا.
## الخطوة 2: تهيئة مصنف جديد
بعد ذلك، علينا إنشاء مصنف جديد لإضافة الأشكال البيضاوية إليه. هذا المصنف عبارة عن ملف Excel، ويمكننا إضافة محتوى أو أشكال إليه.

في هذه الخطوة، نقوم بإنشاء مثيل جديد `Workbook` الكائن الذي سيعمل كحاوية لملف Excel الخاص بنا.
```csharp
// إنشاء مصنف جديد.
Workbook excelbook = new Workbook();
```
## الخطوة 3: أضف الشكل البيضاوي الأول
الآن يأتي الجزء الممتع: إضافة شكل بيضاوي إلى ورقة العمل. يمكن أن يمثل هذا الشكل البيضاوي عنصرًا بصريًا، مثل زر أو تمييز. سنبدأ بإضافة الشكل البيضاوي الأول إلى ورقة العمل الأولى من مصنفنا.

هنا نستخدم `Shapes.AddOval()` طريقة لإنشاء شكل بيضاوي على ورقة العمل في صف وعمود محددين.
```csharp
// أضف شكلًا بيضاويًا.
Aspose.Cells.Drawing.Oval oval1 = excelbook.Worksheets[0].Shapes.AddOval(2, 0, 2, 0, 130, 160);
```
المعلمات بالداخل `AddOval()` هي كما يلي:
- الرقمان الأولان يمثلان الصف والعمود للزاوية العلوية اليسرى من الشكل البيضاوي.
- الرقمان التاليان يمثلان ارتفاع وعرض الشكل البيضاوي.
## الخطوة 4: تعيين موضع الشكل البيضاوي ونمطه
بمجرد إنشاء الشكل البيضاوي، يمكننا تحديد موضعه، وسمك الخط، ونمط الشرطة. `Placement` تحدد الخاصية كيفية سلوك الشكل البيضاوي عند تغيير حجم الخلايا أو نقلها في ورقة العمل.

نجعل الشكل البيضاوي عائمًا بحرية ونضبط مظهره.
```csharp
// ضبط موضع البيضاوي.
oval1.Placement = PlacementType.FreeFloating;
// ضبط وزن الخط.
oval1.Line.Weight = 1;
// تعيين نمط الشرطة للشكل البيضاوي.
oval1.Line.DashStyle = MsoLineDashStyle.Solid;
```
يتيح هذا للشكل البيضاوي التحرك بحرية داخل ورقة العمل، كما يتم ضبط وزن الخط ونمطه لتحقيق الاتساق البصري.
## الخطوة 5: إضافة شكل بيضاوي (دائرة) آخر
لماذا نتوقف عند واحد؟ في هذه الخطوة، سنضيف شكلًا بيضاويًا آخر، وهذه المرة سننشئ دائرة مثالية بجعل الارتفاع والعرض متساويين.

نقوم بإنشاء شكل بيضاوي آخر، ونضعه في مكان مختلف، ونتأكد من أنه ذو شكل دائري من خلال ضبط الارتفاع والعرض بشكل متساوي.
```csharp
// أضف شكلًا بيضاويًا (دائريًا) آخر.
Aspose.Cells.Drawing.Oval oval2 = excelbook.Worksheets[0].Shapes.AddOval(9, 0, 2, 15, 130, 130);
```
## الخطوة 6: تصميم الشكل البيضاوي الثاني
تمامًا كما في السابق، سنقوم بتعديل موضع ووزن ونمط الشرطة لهذا الشكل البيضاوي الثاني (أو الدائرة).

نطبق خصائص مماثلة على الشكل البيضاوي الثاني لتتناسب مع أسلوب الشكل الأول.
```csharp
// ضبط موضع البيضاوي.
oval2.Placement = PlacementType.FreeFloating;
// ضبط وزن الخط.
oval2.Line.Weight = 1;
// تعيين نمط الشرطة للشكل البيضاوي.
oval2.Line.DashStyle = MsoLineDashStyle.Solid;
```
## الخطوة 7: حفظ المصنف
أخيرًا، علينا حفظ المصنف الذي يحتوي على الأشكال البيضاوية التي أضفناها للتو. حفظ الملف يضمن حفظ جميع تغييراتنا.

نقوم بحفظ المصنف في مسار الدليل الذي حددناه سابقًا.
```csharp
// احفظ ملف الاكسل.
excelbook.Save(dataDir + "book1.out.xls");
```
وهذا كل شيء! لقد أضفت الأشكال البيضاوية بنجاح إلى ورقة عمل Excel وحفظت الملف.
