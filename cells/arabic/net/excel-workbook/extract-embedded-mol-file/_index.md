# استخراج ملف Mol المضمن

## مقدمة

هل وجدت نفسك يومًا في حاجة إلى استخراج ملفات مضمنة، وخاصة ملفات MOL، من جدول بيانات Excel؟ إنها مهمة صعبة، أليس كذلك؟ ولكن لا تقلق! بمساعدة Aspose.Cells for .NET، يمكننا تحويل هذه المهمة المعقدة إلى نزهة في الحديقة. في هذا البرنامج التعليمي، سنرشدك خطوة بخطوة حول كيفية استخراج ملفات MOL من ملف Excel باستخدام مكتبة Aspose.Cells القوية.

## المتطلبات الأساسية

قبل أن نتعمق في عملية الاستخراج، دعنا نتأكد من أنك مجهز بالكامل لمتابعة العملية. إليك ما تحتاجه:

- المعرفة الأساسية بلغة C#: إن القليل من المعرفة بلغة C# سوف يساعدك كثيرًا. حتى إذا كنت مبتدئًا، فيجب أن تكون قادرًا على مواكبة التقدم.
- Visual Studio: قم بتثبيت Visual Studio على نظامك. فهو ضروري لكتابة وتنفيذ كود C#.
- Aspose.Cells لـ .NET: إذا لم تقم بتنزيله بعد، فتوجه إلى[صفحة تحميل Aspose.Cells](https://releases.aspose.com/cells/net/) واحصل على الإصدار الأحدث.
- .NET Framework: تأكد من أن لديك إصدارًا متوافقًا من .NET Framework مثبتًا.
-  ملف Excel يحتوي على كائنات MOL مضمنة: بالنسبة لمثالنا، سنستخدم`EmbeddedMolSample.xlsx`تأكد من أن هذا الملف جاهز للاستخراج.

## استيراد الحزم

الآن بعد أن أصبح لدينا كل ما نحتاجه، حان الوقت لإعداد مشروعنا. إليك كيفية استيراد الحزم اللازمة في مشروع C# الخاص بك:

### إنشاء مشروع جديد

افتح Visual Studio واختر إنشاء تطبيق وحدة تحكم C# جديد.

### إضافة حزمة NuGet إلى Aspose.Cells

في مشروعك الذي تم إنشاؤه حديثًا، ستحتاج إلى إضافة حزمة Aspose.Cells. يمكنك القيام بذلك عبر مدير الحزم NuGet:

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. ابحث عن "Aspose.Cells" وانقر على "تثبيت".

### استيراد مساحة اسم Aspose.Cells

```csharp
using Aspose.Cells.Drawing;
using Aspose.Cells.WebExtensions;

```

يجب أن يكون مشروعك الآن قادرًا على الاستفادة من وظائف مكتبة Aspose.Cells.

## الخطوة 1: إعداد البيئة

الآن بعد أن قمت باستيراد الحزم المطلوبة، فلنبدأ في إعداد بيئتنا لاستخراج ملفات MOL.

```csharp
//الدلائل
string SourceDir = "Your Document Directory";
string outputDir = "Your Document Directory";

```

يؤدي هذا إلى تهيئة المصنف باستخدام ملف Excel الذي يحتوي على ملفات MOL المضمنة.


دعونا نقوم بتقسيم عملية الاستخراج إلى خطوات سهلة المتابعة.

## الخطوة 2: تحميل المصنف

 بمجرد حصولك على`workbook` بعد إعداد ملف Excel الخاص بنا كعينة، فإن الخطوة التالية هي تحميل المصنف والاستعداد للاستخراج:

```csharp
Workbook workbook = new Workbook(SourceDir + "EmbeddedMolSample.xlsx");
```

 في هذه الخطوة، نقوم بإنشاء مثيل جديد لـ`Workbook` الفئة، التي تعمل كجسر لمحتوى ملف Excel الخاص بك. يتم تحميل الملف هنا حتى نتمكن لاحقًا من التكرار عبر الأوراق والعثور على كائنات MOL المضمنة.

## الخطوة 3: التكرار خلال أوراق العمل

الآن بعد تحميل المصنف، حان الوقت للبحث بشكل أعمق. تحتاج إلى المرور عبر كل ورقة عمل في المصنف للعثور على أي كائنات مضمنة:

```csharp
foreach (Worksheet sheet in workbook.Worksheets)
{
    OleObjectCollection oles = sheet.OleObjects;
    // متابعة معالجة كائنات OLE...
}
```

 مع هذه القطعة، نستخدم`foreach` حلقة للانتقال عبر كل ورقة في مصنفنا. من خلال الوصول إلى`OleObjects` من خلال المجموعة، يمكننا الوصول إلى كافة الكائنات المضمنة في تلك الورقة المعينة. 

## الخطوة 4: استخراج كائنات OLE

وهنا يحدث السحر! تحتاج إلى تكرار كل كائن OLE لاستخراج ملفات MOL وحفظها:

```csharp
var index = 1;
foreach (OleObject ole in oles)
{
    string fileName = outputDir + "OleObject" + index + ".mol";
    FileStream fs = File.Create(fileName);
    fs.Write(ole.ObjectData, 0, ole.ObjectData.Length);
    fs.Close();
    index++;
}
```

في هذا النهج:
- نحن نحتفظ بسجل للمؤشر لتسمية ملفات الإخراج بشكل تسلسلي.
- بالنسبة لكل كائن OLE، نقوم بإنشاء ملف جديد باستخدام FileStream.
- نكتب بعد ذلك البيانات المضمنة في هذا الملف ونغلق التدفق.

## الخطوة 5: تأكيد التنفيذ

بعد الانتهاء من منطق الاستخراج الخاص بك، من الأفضل أن تؤكد نجاح تنفيذ عملية الاستخراج الخاصة بك:

يقوم هذا الخط البسيط بإخراج رسالة إلى وحدة التحكم عند اكتمال عملية الاستخراج بأكملها بسلاسة.
