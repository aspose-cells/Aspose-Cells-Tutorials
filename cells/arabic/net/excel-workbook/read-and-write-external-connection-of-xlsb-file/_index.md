# قراءة وكتابة اتصال خارجي لملف XLSB

## مقدمة

أهلاً بكم يا مُحبي Excel! هل أنتم مستعدون للغوص في عالم ملفات XLSB والاتصالات الخارجية المُثير؟ إذا سبق لكم العمل مع Excel، فأنتم تُدركون مدى قوته وتعدد استخداماته، ولكن في بعض الأحيان، قد تكون له بعض العيوب، خاصةً عند التعامل مع الاتصالات بقواعد البيانات الخارجية. وهنا يأتي دور Aspose.Cells for .NET! تُتيح لكم هذه المكتبة الرائعة التعامل مع ملفات Excel بسهولة ويسر. في هذه المقالة، سنشرح لكم خطوة بخطوة كيفية قراءة وكتابة الاتصالات الخارجية في ملف XLSB.

## المتطلبات الأساسية

قبل أن نبدأ، إليك بعض الأشياء التي ستحتاجها:

1. المعرفة الأساسية بلغة C#: معرفة أساسيات لغة البرمجة C# ستساعدك كثيرًا. إذا كنت مرتاحًا في كتابة شيفرة C#، فأنت جاهز!
2. Aspose.Cells لـ .NET: ستحتاج إلى تثبيت هذه المكتبة في بيئة التطوير لديك. إذا لم تقم بذلك بعد، يمكنك تنزيلها بسهولة. [هنا](https://releases.aspose.com/cells/net/). 
3. بيئة التطوير المتكاملة (IDE): ستحتاج إلى محرر أكواد أو بيئة تطوير متكاملة (IDE) مثل Visual Studio. 
4. ملف XLSB نموذجي: سنعمل على ملف XLSB نموذجي يحتوي على اتصالات خارجية. يمكنك إنشاء ملفك الخاص أو تنزيل نماذج عبر الإنترنت. 

بمجرد توفر كل هذه المتطلبات الأساسية لديك، سنكون جاهزين للانتقال إلى الكود!

## استيراد الحزم

لنبدأ باستيراد الحزم اللازمة في مشروعنا. تأكد من إضافة مرجع إلى Aspose.Cells في مشروعك. إليك كيفية القيام بذلك بلغة C#:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

تتيح لك عمليات الاستيراد هذه الوصول إلى الوظائف التي ستحتاج إليها للتعامل مع ملفات Excel واتصالاتها.

## الخطوة 1: إعداد الدلائل

قبل العمل على ملف XLSB، عليك إعداد مجلدي المصدر والإخراج. هذه الخطوة أساسية لتنظيم ملفاتك.

```csharp
// دليل المصدر
string sourceDir = "Your Document Directory";
// دليل الإخراج
string outputDir = "Your Output Directory";
```

في مقتطف الكود هذا، `"Your Document Directory"` و `"Your Document Directory"` يُفترض أن تقوم هذه الطرق بجلب المسارات الخاصة بملفات الإدخال والإخراج. يمكنك استبدال هذه الطرق بمسارات مُبرمجة مسبقًا إذا رغبت في ذلك.

## الخطوة 2: تحميل ملف XLSB

الآن حان وقت تحميل ملف Excel إلى مصنف. يتيح لك هذا إمكانية التعامل مع البيانات داخله.

```csharp
// تحميل ملف Excel Xlsb المصدر
Workbook wb = new Workbook(sourceDir + "sampleExternalConnection_XLSB.xlsb");
```

فقط تأكد من أن ملف XLSB الخاص بك موجود في المكان المحدد `sourceDir`إذا كان المسار صحيحًا، فسيكون المصنف الخاص بك جاهزًا للتلاعب!

## الخطوة 3: الوصول إلى الاتصالات الخارجية

بعد تحميل المصنف، يمكنك الوصول إلى اتصالاته الخارجية. في حالتنا، سنسترد اتصال قاعدة البيانات الأول.

```csharp
// اقرأ أول اتصال خارجي وهو في الواقع اتصال قاعدة بيانات
Aspose.Cells.ExternalConnections.DBConnection dbCon = wb.DataConnections[0] as Aspose.Cells.ExternalConnections.DBConnection;
```

هنا، `DataConnections[0]` يشير إلى أول اتصال في المصنف. نعرضه كـ `DBConnection` نوع لمزيد من التلاعب.

## الخطوة 4: استخراج معلومات الاتصال

الآن وقد أصبح لديك وصول إلى اتصال قاعدة البيانات، لنستخرج تفاصيله ونطبعها في وحدة التحكم. سيساعدنا هذا على رؤية ما نعمل عليه بدقة.

```csharp
// اطبع الاسم والأوامر ومعلومات الاتصال الخاصة باتصال قاعدة البيانات
Console.WriteLine("Connection Name: " + dbCon.Name);
Console.WriteLine("Command: " + dbCon.Command);
Console.WriteLine("Connection Info: " + dbCon.ConnectionInfo);
```

تتيح لك هذه الخطوة التحقق من الحالة الحالية لاتصالك الخارجي. سترى اسم الاتصال، والأمر المستخدم، ومعلومات الاتصال مطبوعةً على لوحة التحكم.

## الخطوة 5: تعديل الاتصال

أحيانًا، يكفي تغيير الاسم لتوضيح الأمور أو لمواكبة قواعد التسمية الجديدة. لنُغيّر اسم الاتصال الحالي إلى اسم أكثر ملاءمة.

```csharp
// تعديل اسم الاتصال
dbCon.Name = "NewCust";
```

هذا السطر من التعليمات البرمجية يُغيّر اسم اتصالك من اسمه الأصلي إلى "NewCust". إنه تغيير بسيط ولكنه مؤثر!

## الخطوة 6: حفظ التغييرات

بعد إجراء التعديلات، ستحتاج إلى حفظها في ملف XLSB. إليك كيفية القيام بذلك:

```csharp
// حفظ ملف Excel Xlsb
wb.Save(outputDir + "outputExternalConnection_XLSB.xlsb");
```

يحفظ هذا السطر من التعليمات البرمجية المصنف المحدث الخاص بك، مع الاحتفاظ بكل التغييرات، بما في ذلك اسم الاتصال الجديد.

## الخطوة 7: تأكيد التنفيذ

أخيرًا، لنتأكد من أن كل شيء سار بسلاسة. رسالة بسيطة في النهاية ستوضح ذلك.

```csharp
Console.WriteLine("ReadAndWriteExternalConnectionOfXLSBFile executed successfully.\r\n");
```

ستطمئنك هذه الرسالة إلى أن الكود الخاص بك تم تشغيله بنجاح، وتم حفظ جميع التغييرات كما هو متوقع.

## خاتمة

ها قد انتهيت! لقد تعلمت بنجاح كيفية قراءة وكتابة الاتصالات الخارجية في ملفات XLSB باستخدام Aspose.Cells لـ .NET. باتباع هذه الخطوات، يمكنك إدارة اتصالات بيانات Excel بفعالية، مما يجعل عملك على Excel أكثر سلاسة وأقل إزعاجًا. تذكر، سواء كنت تتعامل مع مجموعات بيانات ضخمة أو تنظم معلوماتك فحسب، فإن Aspose.Cells يوفر لك الأدوات اللازمة.
