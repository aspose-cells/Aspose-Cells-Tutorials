# فرز البيانات في عمود باستخدام قائمة الفرز المخصصة في Excel

## مقدمة

سيرشدك هذا البرنامج التعليمي خلال عملية إعداد مشروعك وتحميل ملف Excel وفرز البيانات ضمن نطاق محدد باستخدام ترتيب فرز مخصص. باتباع هذا الدليل، ستكتسب خبرة عملية يمكنها تعزيز مهاراتك في إدارة البيانات وقابلية استخدام مكتبة Aspose.Cells.

## المتطلبات الأساسية

قبل الغوص في البرنامج التعليمي، دعونا نحدد بعض المتطلبات الأساسية لضمان تجربة تعليمية سلسة.

### المعرفة الأساسية بلغة C#

على الرغم من أن البرنامج التعليمي مصمم لإرشادك خلال كل خطوة، فإن الحصول على فهم أساسي لـ C# سيجعل من السهل استيعاب المفاهيم المقدمة.

### بيئة تطوير .NET

تأكد من إعداد بيئة تطوير .NET صالحة للعمل. يمكنك استخدام Visual Studio أو أي بيئة تطوير متكاملة أخرى تدعم تطوير .NET.

### Aspose.Cells لحزمة .NET NuGet

تحتاج إلى تثبيت مكتبة Aspose.Cells لـ .NET في مشروعك. يمكنك إضافتها بسهولة عبر NuGet Package Manager. 

إليك كيفية القيام بذلك:

1. افتح مشروعك في Visual Studio.
2. انتقل إلى "أدوات" > "مدير حزم NuGet" > "إدارة حزم NuGet للحل".
3.  بحث عن`Aspose.Cells` وتثبيت الإصدار الأحدث.

### ملف Excel الأساسي للاختبار

ستحتاج إلى ملف Excel نموذجي للعمل عليه. يمكنك إنشاء ملف Excel بسيط يحتوي على أسماء دول عشوائية ورموزها.

## استيراد الحزم

للبدء، دعنا نستورد الحزم اللازمة إلى مشروعك. فيما يلي مقتطف يوضح كيفية إعداد الكود الخاص بك:

```csharp
using System;
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

بعد استيراد الحزم، أصبحنا جاهزين للمضي قدمًا.

## الخطوة 1: تحديد أدلة المصدر والإخراج 

الخطوة الأولى هي تحديد مكان ملف الإدخال والمكان الذي تريد حفظ الإخراج (الملف المفرز). تحتاج إلى تحديد مسارين: أحدهما لملف Excel المصدر والآخر لحفظ الإخراج بعد الفرز.

```csharp
string sourceDir = "Your Document Directory\\";
string outputDir = "Your Document Directory\\";
```

## الخطوة 2: تحميل ملف Excel المصدر

بعد ذلك، سنقوم بتحميل ملف Excel الذي يحتوي على البيانات التي تريد فرزها. يتم ذلك عن طريق إنشاء مثيل لـ`Workbook` الفئة وتمرير مسار ملف المصدر الخاص بك.

```csharp
Workbook wb = new Workbook(sourceDir + "sampleSortData_CustomSortList.xlsx");
```

## الخطوة 3: الوصول إلى ورقة العمل الأولى 

بمجرد تحميل الملف، نحتاج إلى الوصول إلى ورقة العمل المحددة التي تحتوي على البيانات التي نعتزم فرزها. في هذه الحالة، نستهدف ورقة العمل الأولى.

```csharp
Worksheet ws = wb.Worksheets[0];
```

## الخطوة 4: تحديد منطقة الخلية المراد فرزها

 نحتاج إلى تحديد نطاق الخلايا التي سنقوم بفرزها. في هذا المثال، سنقوم بفرز الخلايا من A1 إلى A40. استخدم`CellArea.CreateCellArea` طريقة لتحديد مساحة الخلية.

```csharp
CellArea ca = CellArea.CreateCellArea("A1", "A40");
```

## الخطوة 5: إنشاء قائمة فرز مخصصة

قبل الفرز، نحتاج إلى تحديد المعايير التي سنستخدمها للفرز المخصص. يمكنك تعريف قائمة الفرز كمجموعة من السلاسل. ستحدد قائمة الفرز المخصصة ترتيب الفرز.

```csharp
string[] customSortList = new string[] { "USA,US", "Brazil,BR", "China,CN", "Russia,RU", "Canada,CA" };
```

## الخطوة 6: إضافة مفتاح الفرز وإجراء الفرز

الآن حان وقت الفرز! سنستخدم فئة DataSorter لهذا الغرض. أنشئ مفتاحًا للفرز استنادًا إلى القائمة المخصصة لدينا وقم بتنفيذ عملية الفرز.

```csharp
wb.DataSorter.AddKey(0, SortOrder.Ascending, customSortList);
wb.DataSorter.Sort(ws.Cells, ca);
```

## الخطوة 7: احفظ ملف Excel الناتج

بعد اكتمال الفرز، تكون الخطوة الأخيرة هي حفظ التغييرات في ملف Excel جديد. حدد اسم ملف الإخراج واحفظ المصنف.

```csharp
wb.Save(outputDir + "outputSortData_CustomSortList.xlsx");
```

## الخطوة 8: تأكيد التنفيذ الناجح

للتأكد من أن كل شيء سار بسلاسة، يمكنك طباعة رسالة تأكيد على وحدة التحكم. يساعد هذا في تصحيح الأخطاء ويمنحك الرضا عن نجاح العملية.

```csharp
Console.WriteLine("SortDataInColumnWithCustomSortList executed successfully.\r\n");
```

## خاتمة

والآن، لقد نجحت في فرز البيانات في عمود Excel باستخدام قائمة فرز مخصصة مع Aspose.Cells for .NET. يساعد الفرز في إضفاء الهيكل والوضوح على بياناتك، مما يسهل تحليلها وتفسيرها. آمل أن يساعدك هذا الدليل في الارتقاء بمهاراتك إلى المستوى التالي ويساعدك في إدراك مدى قوة Aspose.Cells في أداء المهام المرتبطة بـ Excel.
