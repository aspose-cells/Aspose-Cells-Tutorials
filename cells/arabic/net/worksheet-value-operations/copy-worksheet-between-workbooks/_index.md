# نسخ ورقة عمل من مصنف إلى آخر باستخدام Aspose.Cells

## مقدمة
هل تحتاج إلى طريقة لنقل البيانات بكفاءة من مصنف Excel إلى آخر في تطبيق .NET الخاص بك؟ يُعد نسخ ورقة عمل من مصنف إلى آخر مفيدًا للغاية، سواء كنت تدير التقارير، أو تُنشئ قوالب، أو تُنظم البيانات تلقائيًا. لحسن الحظ، مع Aspose.Cells لـ .NET، هذه العملية سهلة وفعّالة. في هذا البرنامج التعليمي، سنستكشف كيفية نسخ ورقة عمل بسلاسة من مصنف إلى آخر، مما يمنحك تحكمًا كاملاً في إدارة البيانات.
في هذه المقالة، سنغطي كل ما تحتاج لمعرفته للبدء. بدءًا من إعداد Aspose.Cells لـ .NET في مشروعك، وصولًا إلى دليل شامل خطوة بخطوة، ستكتسب المهارات اللازمة لتطبيق هذه الميزة بسلاسة.
## المتطلبات الأساسية
قبل الغوص في الأمر، دعنا نتأكد من أنك مجهز بكل الأدوات اللازمة:
1. مكتبة Aspose.Cells لـ .NET: هذه المكتبة أساسية للعمل مع ملفات Excel في .NET. يمكنك تنزيلها. [هنا](https://releases.aspose.com/cells/net/).
2. Visual Studio: سوف نستخدم Visual Studio (أو IDE مماثل) لكتابة وتشغيل كود .NET.
3. ترخيص Aspose: إذا كنت تريد تجنب قيود التقييم، ففكر في [التقدم بطلب للحصول على نسخة تجريبية مجانية](https://releases.aspose.com/) أو أ [رخصة مؤقتة](https://purchase.aspose.com/temporary-license/).
## استيراد الحزم
للبدء، قم باستيراد المساحات الأساسية اللازمة إلى مشروعك:
```csharp
using System.IO;
using Aspose.Cells;
using System;
```
ستوفر هذه المساحات الاسمية إمكانية الوصول إلى الفئات اللازمة لإنشاء مصنفات وأوراق عمل Excel وتحريرها ومعالجتها.
في هذا الدليل، سنُقسّم كل جزء من العملية إلى خطوات واضحة وسهلة التنفيذ. لنبدأ كل خطوة على حدة!
## الخطوة 1: تعيين مسار الدليل
قبل إنشاء الملفات وحفظها، حدد المجلد الذي ستحفظ فيه مصنفاتك. هذا يُسهّل الوصول إليها لاحقًا.
```csharp
// قم بتعيين المسار إلى دليل المستندات الخاص بك.
string dataDir = "Your Document Directory";
```
ال `dataDir` يخزن المتغير المسار إلى الدليل. تأكد من استبدال `"Your Document Directory"` مع مسار الدليل الفعلي الخاص بك.
## الخطوة 2: إنشاء المصنف وورقة العمل الأولى
الآن، دعنا ننشئ مصنفًا جديدًا باستخدام ورقة عمل واحدة ونضيف إليه بعض البيانات.
```csharp
// إنشاء مصنف جديد.
Workbook excelWorkbook0 = new Workbook();
// قم بالوصول إلى ورقة العمل الأولى في المصنف.
Worksheet ws0 = excelWorkbook0.Worksheets[0];
```
هنا، نقوم بإنشاء كائن مصنف `excelWorkbook0` واسترجاع ورقة العمل الأولى `ws0` للتلاعب بالبيانات.
## الخطوة 3: إضافة بيانات الرأس إلى ورقة العمل
لنملأ ورقة العمل الأولى بصفوف العناوين. ستكون هذه البيانات بمثابة نموذج لتوضيح عملية النسخ.
```csharp
// ملء صفوف الرأس (A1:A4).
for (int i = 0; i < 5; i++)
{
    ws0.Cells[i, 0].PutValue($"Header Row {i}");
}
```
باستخدام حلقة، نملأ الصفوف الخمسة الأولى في العمود A بأسماء العناوين. هذا يوضح مكان بداية كل قسم جديد في ورقة العمل.
## الخطوة 4: ملء صفوف البيانات التفصيلية
الآن، لنُضِف بعض البيانات المُفصَّلة لإضفاء سياق على ورقة العمل. هذا مُفيدٌ بشكلٍ خاص لمحاكاة تقرير أو ورقة تحليل بيانات.
```csharp
// ملء صفوف التفاصيل (A5:A999).
for (int i = 5; i < 1000; i++)
{
    ws0.Cells[i, 0].PutValue($"Detail Row {i}");
}
```
تملأ هذه الحلقة الصفوف من A5 إلى A999 برسالة بسيطة، تحاكي المحتوى التفصيلي الموجود عادةً في جداول البيانات.
## الخطوة 5: تكوين إعداد الصفحة للطباعة
يتيح لنا Aspose.Cells تحديد إعدادات الطباعة لورقة العمل. هنا، سنضبط الصفوف الخمسة الأولى للتكرار في كل صفحة مطبوعة، وهو أمر مفيد بشكل خاص للتقارير.
```csharp
// قم بتكوين إعداد الصفحة لتكرار صفوف الرأس في كل صفحة.
PageSetup pagesetup = ws0.PageSetup;
pagesetup.PrintTitleRows = "$1:$5";
```
عن طريق الإعداد `PrintTitleRows` ل `$1:$5`نضمن طباعة الصفوف الخمسة الأولى (رؤوس الصفحات) في كل صفحة. هذه الميزة مثالية للحفاظ على السياق عند طباعة مجموعات بيانات كبيرة.
## الخطوة 6: إنشاء المصنف الثاني
الآن، لنُنشئ مُصنّفًا ثانيًا للصق ورقة العمل المنسوخة. سيكون هذا المُصنّف هو الوجهة لنقل ورقة العمل.
```csharp
// إنشاء مصنف آخر.
Workbook excelWorkbook1 = new Workbook();
// قم بالوصول إلى ورقة العمل الأولى في المصنف.
Worksheet ws1 = excelWorkbook1.Worksheets[0];
```
هنا، نقوم بالتهيئة `excelWorkbook1` كمصنف عمل وجهتنا واسترجاع ورقة العمل الأولى الخاصة به، `ws1`، حيث سنقوم بلصق المحتوى المنسوخ.
## الخطوة 7: تسمية ورقة عمل الوجهة
لتسهيل التعرف عليه، دعنا نعيد تسمية ورقة العمل الأولى في المصنف الثاني.
```csharp
// إعادة تسمية ورقة العمل.
ws1.Name = "MySheet";
```
إعادة التسمية `ws1` ل `"MySheet"` يجعل من السهل التمييز بين ورقة العمل في المصنف الجديد، خاصة عند التعامل مع أوراق متعددة.
## الخطوة 8: نسخ البيانات من ورقة العمل المصدر
الآن، ننتقل إلى الحدث الرئيسي: نسخ بيانات ورقة العمل من المصنف الأول إلى الثاني. يُبسط Aspose.Cells هذه العملية باستخدام `Copy` طريقة.
```csharp
// انسخ البيانات من ورقة العمل الأولى في المصنف الأول إلى ورقة العمل الأولى في المصنف الثاني.
ws1.Copy(ws0);
```
ال `Copy` تنقل الطريقة كل المحتوى والتنسيق من `ws0` ل `ws1`هذه الطريقة فعالة، حيث تقوم بمعالجة كافة البيانات في أمر واحد.
## الخطوة 9: حفظ المصنف النهائي
بمجرد ضبط كل شيء، احفظ مصنف الوجهة في الدليل المحدد.
```csharp
// احفظ المصنف الثاني.
excelWorkbook1.Save(dataDir + "CopyWorksheetFromWorkbookToOther_out.xls");
```
ال `Save` طريقة الحفظ `excelWorkbook1` كملف إكسل في الدليل المحدد. اسم الملف هنا هو `"CopyWorksheetFromWorkbookToOther_out.xls"`.
