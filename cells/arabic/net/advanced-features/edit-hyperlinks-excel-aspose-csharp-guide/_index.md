## دليل التنفيذ

### تحرير الارتباطات التشعبية في المصنفات

يوضح هذا القسم كيفية تحرير الارتباطات التشعبية الموجودة ضمن ورقة عمل باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تحميل المصنف

أولاً، قم بإنشاء مثيل لـ `Workbook` الفئة وتحميل ملف Excel المستهدف الخاص بك:

```csharp
// تحميل المصنف من مسار الملف
Workbook workbook = new Workbook("sampleEditingHyperlinksOfWorksheet.xlsx");
```

#### الخطوة 2: الوصول إلى ورقة العمل

يمكنك الوصول إلى ورقة العمل المطلوبة باستخدام الفهرس أو الاسم. هنا، نصل إلى ورقة العمل الأولى:

```csharp
// احصل على ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```

#### الخطوة 3: تحرير الارتباطات التشعبية

قم بالتكرار على كل ارتباط تشعبي في ورقة العمل وتعديل خصائصه:

```csharp
// تكرار جميع الارتباطات التشعبية في ورقة العمل
for (int i = 0; i < worksheet.Hyperlinks.Count; i++)
{
    // الوصول إلى رابط تشعبي محدد
    Hyperlink hl = worksheet.Hyperlinks[i];

    // تحديث العنوان
    hl.Address = "http://www.aspose.com";

    // تعديل النص المعروض للرابط التشعبي
    hl.TextToDisplay += "_Modified";
}
```

#### الخطوة 4: حفظ المصنف

بعد إجراء التغييرات، احفظ المصنف في ملف جديد:

```csharp
// حفظ المصنف المحدث
tworkbook.Save("outputEditingHyperlinksOfWorksheet.xlsx");
```
