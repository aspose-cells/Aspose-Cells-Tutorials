## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى ميزتين رئيسيتين: تحميل الروابط الخارجية للمصنف والوصول إليها والتكرار من خلال هذه الروابط لعرض المعلومات.

### الميزة 1: تحميل المصنف والوصول إليه

**ملخص:** تُظهر لك هذه الميزة كيفية تحميل مصنف Excel من دليل محدد والوصول إلى ارتباطاته الخارجية باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: إعداد دليل المصدر

قم بتحديد دليل المصدر الذي يوجد به ملف Excel الخاص بك:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
```

#### الخطوة 2: تحميل المصنف

قم بتحميل المصنف الذي يحتوي على الروابط الخارجية المخفية:

```csharp
Workbook workbook = new Workbook(SourceDir + "/sampleCheckHiddenExternalLinks.xlsx");
```

#### الخطوة 3: الوصول إلى مجموعة الروابط الخارجية

الوصول إلى مجموعة الروابط الخارجية في المصنف:

```csharp
ExternalLinkCollection links = workbook.Worksheets.ExternalLinks;
```

الآن، `links` يحتوي على كافة كائنات الارتباط الخارجي الموجودة ضمن المصنف الخاص بك.

### الميزة 2: تكرار وعرض معلومات الروابط الخارجية

**ملخص:** يوضح هذا القسم كيفية تكرار كل رابط خارجي وعرض مصدر بياناته وحالة الإحالة وخصائص الرؤية.

#### الخطوة 1: التكرار من خلال الروابط الخارجية

قم بالمرور على كل رابط خارجي في المجموعة:

```csharp
for (int i = 0; i < links.Count; i++)
{
    Console.WriteLine("Data Source: " + links[i].DataSource);
    Console.WriteLine("Is Referred: " + links[i].IsReferred);
    Console.WriteLine("Is Visible: " + links[i].IsVisible);
    Console.WriteLine();
}
```

توفر هذه الحلقة رؤى تفصيلية حول خصائص كل رابط، مثل مصدر البيانات وحالة الرؤية.
