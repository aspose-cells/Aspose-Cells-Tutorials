## دليل التنفيذ: إزالة المقطع

اتبع الخطوات التالية لإزالة الشرائح من ملف Excel:

### الخطوة 1: تحميل المصنف

إنشاء مثيل لـ `Workbook` وحمل ملف Excel الذي يحتوي على المقطع:

```csharp
// تحديد مسار دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// تحميل المصنف باستخدام الشرائح
Workbook wb = new Workbook(sourceDir + "sampleRemovingSlicer.xlsx");
```

### الخطوة 2: الوصول إلى ورقة العمل

ادخل إلى ورقة العمل التي تحتوي على المُقطّع. افترض أنه موجود في الورقة الأولى:

```csharp
// احصل على مرجع إلى ورقة العمل الأولى
Worksheet ws = wb.Worksheets[0];
```

### الخطوة 3: إزالة القطاعة

حدد موقع المقطع المطلوب وقم بإزالته باستخدام الفهرس الخاص به داخل `Slicers` مجموعة:

```csharp
// الوصول إلى أول شريحة في المجموعة
Aspose.Cells.Slicers.Slicer slicer = ws.Slicers[0];

// إزالة المقطعة من ورقة العمل
ws.Slicers.Remove(slicer);
```

### الخطوة 4: احفظ مصنفك

احفظ المصنف الخاص بك للاحتفاظ بالتغييرات التي أجريتها عن طريق إزالة المقطع:

```csharp
// تحديد مسار دليل الإخراج
string outputDir = RunExamples.Get_OutputDirectory();

// حفظ المصنف المحدث
wb.Save(outputDir + "outputRemovingSlicer.xlsx", SaveFormat.Xlsx);

Console.WriteLine("RemovingSlicer executed successfully.");
```
