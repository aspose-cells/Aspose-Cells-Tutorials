## دليل التنفيذ
سنقوم بتفصيل الخطوات اللازمة لإدارة الروابط الخارجية بكفاءة.

### تحديد النطاقات واسترجاعها باستخدام الروابط الخارجية
#### ملخص
يوضح هذا القسم كيفية تكرار النطاقات المسماة في ملف Excel لتحديد النطاقات المرتبطة خارجيًا.

#### التنفيذ خطوة بخطوة
**1. قم بتحميل مصنف العمل الخاص بك**
قم بتحميل ملف Excel المصدر الخاص بك:
```csharp
string sourceDir = "YourSourceDirectoryPath";
Workbook workbook = new Workbook(sourceDir + "SampleExternalReferences.xlsx");
```
**2. التكرار عبر النطاقات المسماة**
قم بالوصول إلى كل نطاق مسمى وتحقق من وجود روابط خارجية:
```csharp
foreach (Name namedRange in workbook.Worksheets.Names)
{
    ReferredArea[] referredAreas = namedRange.GetReferredAreas(true);
    
    if (referredAreas != null)
    {
        foreach (var referredArea in referredAreas)
        {
            // اطبع تفاصيل كل رابط خارجي
            Console.WriteLine("IsExternalLink: " + referredArea.IsExternalLink);
            Console.WriteLine("SheetName: " + referredArea.SheetName);
            Console.WriteLine("ExternalFileName: " + referredArea.ExternalFileName);
            // يمكن طباعة المعلومات الإضافية هنا حسب الحاجة
        }
    }
}
```
**شرح المعلمات الرئيسية:**
- **`GetReferredAreas(true)`**:يستعيد المناطق المرتبطة بالنطاق المسمى. `true` تضمن المعلمة تضمين المراجع الخارجية.
- **`IsExternalLink`**: يشير إلى ما إذا كانت المنطقة المشار إليها عبارة عن رابط خارجي.
