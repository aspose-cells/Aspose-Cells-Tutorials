## دليل التنفيذ

دعونا نقسم عملية العثور على اسم العنصر الجذر لخريطة XML إلى خطوات قابلة للإدارة.

### تحميل ملف Excel

ابدأ بتحميل ملف Excel الذي يحتوي على خريطة XML:

```csharp
// مسار دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// تحميل ملف Excel النموذجي
Workbook workbook = new Workbook(sourceDir + "sampleRootElementNameOfXmlMap.xlsx");
```

**لماذا:** يعد تحميل المصنف أمرًا ضروريًا للوصول إلى محتوياته، بما في ذلك أي خرائط XML مرتبطة به.

### الوصول إلى خريطة XML

بعد ذلك، قم باسترداد خريطة XML الأولى من المصنف:

```csharp
// احصل على أول كائن XmlMap من المجموعة
XmlMap xmlMap = workbook.Worksheets.XmlMaps[0];
```

**لماذا:** يمكن أن يحتوي Excel على خرائط XML متعددة؛ ويتطلب الوصول إليها الفهرسة في مجموعتها.

### استخراج اسم العنصر الجذر

أخيرًا، اطبع اسم العنصر الجذر لخريطة XML الخاصة بك:

```csharp
// طباعة اسم العنصر الجذر في وحدة التحكم
Console.WriteLine("Root Element Name Of Xml Map: " + xmlMap.RootElementName);
```

**لماذا:** ال `RootElementName` توفر الخاصية طريقة سريعة لتحديد العقدة الرئيسية في بنية XML الخاصة بك، وهي مفيدة للمعالجة الإضافية.

- **مشاكل مسار الملف**:تأكد من أن مسار الملف صحيح ويمكن الوصول إليه.
