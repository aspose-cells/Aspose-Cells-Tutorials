الآن بعد أن أصبح الإعداد جاهزًا، دعنا نتعمق في تنفيذ الوظيفة لقراءة وكتابة الجداول باستخدام مصدر بيانات QueryTable.

أولاً، يجب عليك تحميل ملف Excel الذي يحتوي على الجدول المرتبط بالاستعلام:


#### تحديد مصدر بيانات جدول الاستعلام

حدد ListObject في ورقة العمل الخاصة بك والذي يتوافق مع الجدول الذي تريد تعديله:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
ListObject table = worksheet.ListObjects[0];

if (table.DataSourceType == TableDataSourceType.QueryTable)
{
    // تنفيذ العمليات على جداول الاستعلام
}
```

#### تكوين خصائص الجدول

بالنسبة للجداول التي تحتوي على مصدر بيانات QueryTable، قد ترغب في عرض الإجماليات:

```csharp
// تمكين عرض الإجماليات للجدول
```

بعد إجراء التعديلات الخاصة بك، احفظ المصنف لتطبيق التغييرات:

