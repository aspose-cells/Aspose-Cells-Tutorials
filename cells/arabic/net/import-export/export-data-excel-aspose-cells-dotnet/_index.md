### تصدير البيانات من Excel (H2)

الهدف الرئيسي هو استخراج البيانات بكفاءة من ورقة عمل Excel وتصديرها إلى جدول بيانات. لنرَ كيف يُمكن تحقيق ذلك باستخدام Aspose.Cells.

#### الخطوة 1: إعداد البيئة الخاصة بك

تحديد المسارات وإنشاء تدفق ملف لملف Excel:
```csharp
// المسار إلى دليل المستندات الخاص بك.
string dataDir = "path/to/your/excel/files/";

// قم بإنشاء FileStream لفتح ملف Excel.
FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open);

// إنشاء كائن مصنف باستخدام مجرى الملف.
Workbook workbook = new Workbook(fstream);
```

#### الخطوة 2: الوصول إلى ورقة العمل وتصدير البيانات

قم بالوصول إلى ورقة العمل الخاصة بك وقم بتصدير نطاق البيانات المطلوب إلى جدول البيانات:
```csharp
// قم بالوصول إلى ورقة العمل الأولى في المصنف.
Worksheet worksheet = workbook.Worksheets[0];

// تصدير محتويات الصفوف والأعمدة المحددة إلى جدول البيانات.
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, 7, 2, true);

System.Console.WriteLine("Number of Rows in Data Table: " + dataTable.Rows.Count);
```

#### توضيح
- **طريقة ExportDataTable**:تعمل هذه الطريقة على تصدير البيانات من النطاق المحدد (صف البداية، عمود البداية، إجمالي الصفوف، إجمالي الأعمدة) إلى جدول البيانات.
- **حدود**:
  - `startRow`:مؤشر الصف الأولي.
  - `startColumn`:مؤشر العمود الافتتاحي.
  - `totalRows`:عدد الصفوف المراد تصديرها.
  - `totalColumns`:عدد الأعمدة المراد تصديرها.
  - `convertStringToNumeric`:يقوم بتحويل السلاسل التي تمثل الأرقام إلى أنواع بيانات رقمية.

#### الخطوة 3: تنظيف الموارد

تأكد دائمًا من إغلاق أي تدفقات ملفات مفتوحة للموارد المجانية:
```csharp
// أغلق FileStream بعد الاستخدام.
fstream.Close();
```
