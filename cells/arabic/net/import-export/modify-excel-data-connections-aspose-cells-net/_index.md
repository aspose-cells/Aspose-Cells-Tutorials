# تعديل اتصالات بيانات Excel باستخدام Aspose.Cells .NET

## مقدمة

في عالمنا اليوم الذي يعتمد على البيانات، تُعدّ إدارة اتصالات بيانات Excel وتعديلها بكفاءة أمرًا بالغ الأهمية لضمان سلاسة تكامل البيانات وإعداد التقارير. إذا واجهتَ صعوبة في تحديث أو تعديل اتصالات البيانات الموجودة في ملفات Excel باستخدام .NET، فهذا البرنامج التعليمي مُصمّم خصيصًا لك. بالاستفادة من مكتبة Aspose.Cells .NET القوية، سنستكشف كيفية إنشاء اتصالات البيانات والوصول إليها وتعديلها بسهولة داخل مصنفات Excel.

**ما سوف تتعلمه:**
- كيفية إنشاء كائن مصنف والوصول إلى اتصالات البيانات الخاصة به.
- تقنيات لتعديل خصائص اتصالات البيانات، مثل الأسماء ومسارات الملفات.
- طرق لتغيير معلمات اتصال قاعدة البيانات بما في ذلك أنواع الأوامر وعبارات SQL.
- خطوات لحفظ التعديلات الخاصة بك مرة أخرى في المصنف.

دعونا نتعمق في المتطلبات الأساسية اللازمة للبدء في استخدام Aspose.Cells .NET.

قبل أن نبدأ، تأكد من أن لديك ما يلي:

## دليل التنفيذ

### إنشاء مصنف العمل والوصول إلى اتصالات البيانات

**ملخص:**
ابدأ بإنشاء `Workbook` كائن من ملف Excel موجود. هذه هي الخطوة الأولى للوصول إلى أي اتصالات بيانات ضمن هذا المصنف.

#### الخطوة 1: إنشاء كائن مصنف
لإنشاء `Workbook` الكائن، الاستخدام:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "sampleModifyingExistingDataConnection.xlsx");
```

يقوم هذا السطر بقراءة ملف Excel الخاص بك في التطبيق، مما يسمح لك بالتعامل معه برمجيًا.

#### الخطوة 2: الوصول إلى اتصال البيانات
قم بالوصول إلى اتصال البيانات الأول باستخدام:

```csharp
ExternalConnection conn = workbook.DataConnections[0];
```

### تعديل خصائص اتصال البيانات

**ملخص:**
بمجرد الوصول إليه، قم بتعديل خصائص مثل اسم الاتصال ومسار ملف ODC لتناسب احتياجاتك.

#### الخطوة 1: تغيير الاسم والمسار
لتغيير هذه الخصائص:

```csharp
conn.Name = "MyConnectionName";
conn.OdcFile = @"C:\\Users\\MyDefaultConnection.odc";
```

### تعديل معلمات DBConnection

**ملخص:**
بالنسبة لاتصالات قاعدة البيانات، يمكنك ضبط المعلمات مثل نوع الأمر، وأمر SQL، وسلسلة الاتصال.

#### الخطوة 1: الإرسال إلى DBConnection
أولاً، قم بإرسال اتصال البيانات الخاص بك:

```csharp
DBConnection dbConn = (DBConnection)workbook.DataConnections[0];
```

#### الخطوة 2: تعديل معلمات الاتصال
ثم قم بتحديث المعلمات الضرورية:

```csharp
dbConn.CommandType = OLEDBCommandType.SqlStatement;
dbConn.Command = "SELECT * FROM AdminTable";
dbConn.ConnectionInfo = "Server=myServerAddress;Database=myDataBase;User ID=myUsername;Password=myPassword;Trusted_Connection=False";
```

### حفظ المصنف

**ملخص:**
بعد إجراء التعديلات، احفظ المصنف الخاص بك للحفاظ على التغييرات.

#### الخطوة 1: حفظ المصنف المعدّل
يستخدم:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outputModifyingExistingDataConnection.xlsx");
```
