## دليل التنفيذ
### تحميل ملف Excel
قم بتحميل ملف Excel الخاص بك باستخدام `Workbook` فصل.
**الخطوة 1: تحميل ملف Excel النموذجي**
```csharp
// دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// تحميل ملف Excel النموذجي
Workbook wb = new Workbook(sourceDir + "sampleExportTableAsHtmlString.xlsx");
```
### الوصول إلى ورقة العمل
يمكنك الوصول إلى ورقة عمل محددة في مصنف Excel الخاص بك على النحو التالي:
**الخطوة 2: الوصول إلى ورقة العمل الأولى**
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet ws = wb.Worksheets[0];
```
### تكوين خيارات التصدير
قم بتكوين خيارات التصدير لتحديد تصدير البيانات كسلاسل HTML.
**الخطوة 3: تكوين ExportTableOptions**
```csharp
// حدد خيارات جدول التصدير وقم بتعيين ExportAsHtmlString إلى true
ExportTableOptions opts = new ExportTableOptions();
opts.ExportColumnName = false;
opts.ExportAsHtmlString = true;
```
### تصدير البيانات
تصدير البيانات من نطاق الخلايا المحدد إلى جدول البيانات.
**الخطوة 4: تصدير الخلايا إلى جدول البيانات**
```csharp
// تصدير بيانات الخلايا إلى جدول البيانات باستخدام خيارات جدول التصدير المحددة
DataTable dt = ws.Cells.ExportDataTable(0, 0, 3, 3, opts);
```
