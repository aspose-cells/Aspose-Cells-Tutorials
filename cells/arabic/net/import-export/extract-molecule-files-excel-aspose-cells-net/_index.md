## دليل التنفيذ

### استخراج ملفات الجزيئات المضمنة من Excel

#### ملخص
تتيح لك هذه الميزة استرجاع البيانات برمجيًا `.mol` الملفات المُخزّنة ككائنات OleObjects داخل مصنف Excel باستخدام Aspose.Cells لـ .NET. إليك الطريقة:

#### الخطوة 1: تحميل المصنف
ابدأ بتحميل المصنف الخاص بك الذي يحتوي على الجزيئات المضمنة.

```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY"; // استبدله بمسار دليل المصدر الخاص بك
string outputDir = @"YOUR_OUTPUT_DIRECTORY";  // استبدله بمسار دليل الإخراج الخاص بك

Workbook workbook = new Workbook(sourceDir + "EmbeddedMolSample.xlsx");
```

#### الخطوة 2: التكرار على أوراق العمل وOleObjects
قم بالتنقل عبر كل ورقة عمل في المصنف للوصول إلى الكائنات المضمنة.

```csharp
var index = 1;
foreach (Worksheet sheet in workbook.Worksheets)
{
    OleObjectCollection oles = sheet.OleObjects; // الحصول على جميع كائنات Ole من ورقة العمل
    
    foreach (OleObject ole in oles)
    {
        string fileName = outputDir + "OleObject" + index + ".mol";
        
        using (FileStream fs = File.Create(fileName))
        {
            fs.Write(ole.ObjectData, 0, ole.ObjectData.Length); // كتابة بيانات الكائن المضمن في ملف
        }
        index++;
    }
}
```

#### توضيح
- **كتاب العمل**:يمثل مصنف Excel الخاص بك ويعمل كنقطة دخول للتعامل معه.
- **مجموعة OleObject**:مجموعة من كائنات OLE في كل ورقة عمل.
- **تدفق الملفات**:تستخدم لإنشاء الملفات حيث يتم استخراجها `.mol` تمت كتابة البيانات.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تعيين المسارات بشكل صحيح لكل من الدليل المصدر والدليل الناتج.
