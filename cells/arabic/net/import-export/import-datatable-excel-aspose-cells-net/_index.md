## دليل التنفيذ
ينقسم هذا الدليل إلى قسمين رئيسيين: إنشاء وتعبئة `DataTable`، متبوعًا باستيراد هذه البيانات إلى ورقة عمل Excel باستخدام Aspose.Cells لـ .NET.

### إنشاء جدول البيانات وتعبئته
#### ملخص
يوضح هذا القسم كيفية إنشاء `DataTable` كائن، أضف أعمدة، ثم املأه بصفوف من البيانات. هذا ضروري لتحضير بياناتك قبل تصديرها إلى Excel.

#### خطوات:
**1. تحديد دليل المصدر**
ابدأ بتحديد الدلائل لملفات الإدخال والإخراج، على الرغم من أن هذا المثال لا يستخدمها بشكل مباشر داخل هذه العمليات.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**2. إنشاء كائن جدول البيانات**
إنشاء مثيل `DataTable` الكائن المسمى "المنتجات".
```csharp
DataTable dataTable = new DataTable("Products");
```

**3. إضافة أعمدة إلى جدول البيانات**
أضف الأعمدة الضرورية، مع تحديد أنواع البيانات لكل منها.
```csharp
dataTable.Columns.Add("Product ID", typeof(Int32));
dataTable.Columns.Add("Product Name", typeof(string));
dataTable.Columns.Add("Units In Stock", typeof(Int32));
```

**4. ملء الصفوف بالبيانات**
إنشاء صفوف وتعيين قيم لها قبل إضافتها إلى `DataTable`.
```csharp
// الصف الأول
DataRow dr = dataTable.NewRow();
dr[0] = 1;
dr[1] = "Aniseed Syrup";
dr[2] = 15;
dataTable.Rows.Add(dr);

// الصف الثاني
dr = dataTable.NewRow();
dr[0] = 2;
dr[1] = "Boston Crab Meat";
dr[2] = 123;
dataTable.Rows.Add(dr);
```

### استيراد جدول البيانات إلى ورقة عمل Excel
#### ملخص
يوضح هذا القسم كيفية استيراد البيانات المأهولة `DataTable` في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET، مما يوضح تصدير البيانات بسلاسة.

#### خطوات:
**1. تهيئة المصنف وورقة العمل**
إنشاء مثيل جديد لمصنف عمل والحصول على مرجع إلى ورقة العمل الأولى الخاصة به.
```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**2. تكوين خيارات الاستيراد**
تعيين خيارات الاستيراد لتضمين أسماء الحقول في ورقة Excel.
```csharp
ImportTableOptions options = new ImportTableOptions();
options.IsFieldNameShown = true;
```

**3. استيراد بيانات جدول البيانات**
استخدم `ImportData` طريقة تصدير البيانات بدءًا من الخلية A1.
```csharp
worksheet.Cells.ImportData(dataTable.DefaultView, 0, 0, options);
```

**4. احفظ ملف Excel**
حدد دليل الإخراج واسم الملف لحفظ مستند Excel.
```csharp
workbook.Save(outputDir + "output.xls");
```
