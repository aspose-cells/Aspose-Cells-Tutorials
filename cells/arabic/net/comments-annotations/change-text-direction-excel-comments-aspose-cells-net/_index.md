# تغيير اتجاه النص في تعليقات Excel باستخدام Aspose.Cells .NET

## مقدمة

هل ترغب في تخصيص اتجاه النص في التعليقات داخل ملفات Excel باستخدام C#؟ مع Aspose.Cells لـ .NET، أصبح تغيير اتجاه النص أمرًا سهلاً، خاصةً عند التعامل مع مستندات متعددة اللغات. سيرشدك هذا البرنامج التعليمي خلال تعديل اتجاه نص التعليق من اليسار إلى اليمين (LTR) إلى اليمين إلى اليسار (RTL)، والعكس.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Cells لـ .NET
- خطوات تغيير اتجاه النص في تعليقات Excel
- أفضل الممارسات لتحسين التنفيذ الخاص بك

هل أنت مستعد لتحسين ملفات Excel لديك باستخدام توجيهات نصية مخصصة؟ لنبدأ!

### المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

- **المكتبات**ثبّت Aspose.Cells لـ .NET. سنشرح طرق التثبيت أدناه.
- **إعداد البيئة**:بيئة تطوير تدعم تطبيقات .NET (على سبيل المثال، Visual Studio).
- **معرفة**:فهم أساسيات لغة C# والتعرف على كيفية التعامل مع ملفات Excel.

**استخدام .NET CLI:**
```bash
dotnet add package Aspose.Cells
```

**استخدام وحدة تحكم إدارة الحزم:**

### الحصول على الترخيص

تقدم Aspose نسخة تجريبية مجانية تتيح لك اختبار كامل إمكانيات مكتباتها. لمواصلة الاستخدام، فكّر في الحصول على ترخيص مؤقت أو شراء اشتراك للمشاريع طويلة الأجل.

```csharp

```

الآن دعنا نقوم بإعداد مصنف Excel وتعديل بعض التعليقات!

## دليل التنفيذ

### إنشاء مصنف وإضافة تعليقات

سنبدأ بإنشاء مصنف Excel جديد وإضافة نص إلى خلية.

**ملخص:**
يوضح هذا القسم كيفية إنشاء مصنف وإضافة نص إلى ورقة عمل وإضافة تعليقات.

```csharp
// إنشاء مصنف جديد
var wb = new Workbook();

// احصل على ورقة العمل الأولى
var sheet = wb.Worksheets[0];

// أضف بعض النص في الخلية A1
sheet.Cells["A1"].PutValue("Here");
```

### إضافة التعليقات وتكوينها

الآن، دعونا نضيف تعليقًا إلى خليتنا ونقوم بتكوين محاذاة النص الخاصة بها.

**إضافة تعليق:**
```csharp
// أضف تعليقًا إلى الخلية A1
var comment = sheet.Comments[sheet.Comments.Add("A1"]);
```

**تكوين محاذاة النص واتجاهه:**

- **المحاذاة الرأسية**:مركز النص عموديا.
- **المحاذاة الأفقية**:محاذاة النص إلى اليمين.
- **اتجاه النص**:يتم الضبط من اليسار إلى اليمين (LTR) إلى اليمين إلى اليسار (RTL).

```csharp
// ضبط المحاذاة الرأسية
comment.CommentShape.TextVerticalAlignment = TextAlignmentType.Center;

// ضبط المحاذاة الأفقية
comment.CommentShape.TextHorizontalAlignment = TextAlignmentType.Right;

// تغيير اتجاه النص إلى من اليمين إلى اليسار
comment.CommentShape.TextDirection = TextDirectionType.RightToLeft;
```

**نصيحة لاستكشاف الأخطاء وإصلاحها:** تأكد من أن الخلية التي تضيف إليها التعليقات ليست مقفلة أو محمية، حيث يمكن أن يؤدي هذا إلى منع التعديلات.

### حفظ مصنفك

وأخيرًا، احفظ التغييرات لتتمكن من رؤيتها تنعكس في ملف Excel:

```csharp
// حفظ ملف Excel
wb.Save("outputChangeTextDirection.xlsx");

Console.WriteLine("ChangeTextDirection executed successfully.\r\n");
```
