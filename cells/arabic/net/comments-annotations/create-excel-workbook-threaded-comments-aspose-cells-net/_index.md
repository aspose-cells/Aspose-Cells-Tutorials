## دليل التنفيذ

### إنشاء مصنف وحفظه باستخدام التعليقات المترابطة

#### ملخص
في هذا القسم، سنقوم بإنشاء مصنف Excel وإضافة تعليقات مترابطة باستخدام Aspose.Cells لـ .NET.

#### تعليمات خطوة بخطوة
**1. تهيئة المصنف**
ابدأ بإنشاء مثيل جديد لـ `Workbook`:
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء مثيل جديد للمصنف
Workbook workbook = new Workbook();
```

**2. أضف مؤلفًا للتعليقات المترابطة**
قم بتحديد وإضافة مؤلف التعليق إلى `ThreadedCommentAuthors` مجموعة:
```csharp
// أضف مؤلفًا للتعليق المترابط
int authorIndex = workbook.Worksheets.ThreadedCommentAuthors.Add("Aspose Test", "", "");
ThreadedCommentAuthor author = workbook.Worksheets.ThreadedCommentAuthors[authorIndex];
```

**3. إدراج تعليق مترابط**
أضف تعليقًا مترابطًا إلى الخلية A1 في ورقة العمل الأولى:
```csharp
// أضف تعليقًا مترابطًا إلى الخلية A1 في ورقة العمل الأولى
workbook.Worksheets[0].Comments.AddThreadedComment("A1", "Test Threaded Comment", author);
```

**4. احفظ المصنف**
وأخيرًا، احفظ المصنف الخاص بك في دليل الإخراج المطلوب:
```csharp
// حفظ المصنف في دليل الإخراج
workbook.Save(outputDir + "/AddThreadedComments_out.xlsx");
```
