## دليل التنفيذ

### تحرير التعليقات المترابطة في ورقة عمل Excel

ترتكز هذه الميزة على الوصول إلى التعليقات المترابطة وتعديلها داخل خلية محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تحميل المصنف

ابدأ بتحميل ملف Excel الحالي. يتم ذلك باستخدام `Workbook` الفئة التي تمثل مصنف Excel بأكمله:

```csharp
// تعيين مسارات لدلائل المصدر والإخراج
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

// تحميل المصنف من دليل محدد
Workbook workbook = new Workbook(SourceDir + "ThreadedCommentsSample.xlsx");
```

#### الخطوة 2: الوصول إلى التعليقات المترابطة

قم بالوصول إلى ورقة العمل الأولى واسترجاع التعليقات المترابطة لخلية معينة، مثل `A1`يمكنك استهداف أي خلية عن طريق تغيير مرجعها:

```csharp
// احصل على ورقة العمل الأولى من المصنف
Worksheet worksheet = workbook.Worksheets[0];

// استرداد جميع التعليقات المترابطة للخلية A1
ThreadedComment comment = worksheet.Comments.GetThreadedComments("A1")[0];
```

#### الخطوة 3: تحديث التعليق

بمجرد وصولك إلى تعليق محدد مترابط، قم بتحديث محتواه حسب الحاجة:

```csharp
// تعديل ملاحظة التعليق المترابط
comment.Notes = "Updated Comment";
```

#### الخطوة 4: حفظ التغييرات

بعد إجراء التحديثات، احفظ المصنف للاحتفاظ بالتغييرات. يمكنك تحديد اسم ملف جديد أو استبدال الملف الأصلي:

```csharp
// احفظ المصنف المحدث باسم ملف جديد
workbook.Save(OutputDir + "EditThreadedComments.xlsx");
```

### تحميل وحفظ مصنف Excel

تعتبر هذه الميزة عرضًا سريعًا لكيفية تحميل ملف Excel الحالي، وإجراء العمليات عليه، وحفظه مرة أخرى.

// حفظ المصنف المعدل في ملف جديد
