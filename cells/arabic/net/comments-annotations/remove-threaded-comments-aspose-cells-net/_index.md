## دليل التنفيذ

### نظرة عامة على إزالة التعليقات المترابطة

يوضح هذا القسم كيفية إزالة التعليقات المترابطة من مصنفات Excel باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تحميل المصنف

ابدأ بتحميل ملف المصنف الخاص بك:

```csharp
string sourceDir = "path_to_your_directory";
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```

**لماذا هذا مهم:** يعد تحميل المصنف أمرًا ضروريًا للوصول إلى محتوياته والتلاعب بها.

#### الخطوة 2: الوصول إلى ورقة العمل

قم بالوصول إلى ورقة العمل المحددة التي تحتوي على تعليقاتك:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
CommentCollection comments = worksheet.Comments;
```

**توضيح:** إن استهداف ورقة عمل محددة يسمح بإدارة التعليقات عليها بشكل فعال.

#### الخطوة 3: إزالة التعليقات المترابطة

إزالة التعليقات من خلية معينة، مثل "A1":

```csharp
// احصل على مؤلف التعليق الأول في A1 (خطوة اختيارية إذا كنت تريد التعامل مع المؤلفين)
ThreadedCommentAuthor author = worksheet.Comments.GetThreadedComments("A1")[0].Author;

// إزالة التعليق في A1
comments.RemoveAt("A1");

// اختياريا قم بإزالة المؤلف أيضًا
ThreadedCommentAuthorCollection authors = workbook.Worksheets.ThreadedCommentAuthors;
authors.RemoveAt(authors.IndexOf(author));
```

**الرؤية الرئيسية:** `RemoveAt` يزيل التعليقات بكفاءة من خلال مراجع الخلايا الخاصة بها.

#### الخطوة 4: حفظ المصنف

وأخيرًا، احفظ المصنف المعدّل:

```csharp
string outDir = "output_directory_path";
workbook.Save(outDir + "ThreadedCommentsSample_Out.xlsx");
```

**غاية:** يضمن الحفظ استمرار جميع التغييرات في ملف جديد أو موجود.
