## دليل التنفيذ

### الوصول إلى التعليقات المترابطة في Excel
#### ملخص
في هذا القسم، سنتمكن من الوصول إلى التعليقات المترابطة وقراءتها من خلية في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. تُعد هذه الميزة مفيدة بشكل خاص لاستخراج التعليقات التفصيلية أو الملاحظات التعاونية المضمنة في جداول البيانات.

#### التنفيذ خطوة بخطوة
**1. قم بتحميل المصنف**
ابدأ بتحميل المصنف الذي يحتوي على جدول البيانات الذي تريد العمل به:
```csharp
string sourceDir = "path/to/your/source/directory/";
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```

**2. الوصول إلى ورقة العمل**
انتقل إلى ورقة العمل التي تريد قراءة التعليقات منها. في هذا المثال، نصل إلى ورقة العمل الأولى:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

**3. استرداد التعليقات المترابطة**
جلب التعليقات المترابطة لخلية معينة باستخدام `GetThreadedComments` طريقة:
```csharp
ThreadedCommentCollection threadedComments = worksheet.Comments.GetThreadedComments("A1");
```

**4. عرض تفاصيل التعليق**
قم بالتكرار عبر المجموعة لعرض تفاصيل كل تعليق، مثل الملاحظات ومعلومات المؤلف:
```csharp
foreach (ThreadedComment comment in threadedComments)
{
    Console.WriteLine("Comment: " + comment.Notes);
    Console.WriteLine("Author: " + comment.Author.Name);
}
```

**5. التنفيذ والتحقق**
قم بتشغيل الكود للتأكد من تنفيذه بنجاح، والتحقق من قراءة التعليقات بشكل صحيح.
