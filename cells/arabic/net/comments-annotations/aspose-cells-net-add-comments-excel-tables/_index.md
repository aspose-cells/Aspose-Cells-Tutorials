### إضافة تعليقات إلى كائن القائمة
**ملخص:**
تعرف على كيفية إضافة تعليقات برمجيًا إلى كائن القائمة الأول في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تحميل المصنف الخاص بك
قم بتحميل مصنف Excel الحالي الخاص بك:
```csharp
string dataDir = "path/to/your/files/";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### الخطوة 2: الوصول إلى ورقة العمل وقائمة الكائنات
قم بالوصول إلى ورقة العمل الأولى ثم احصل على أول كائن قائمة بداخلها:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
ListObject lstObj = worksheet.ListObjects[0];
```

#### الخطوة 3: إضافة تعليق إلى كائن القائمة
قم بتعيين التعليق المطلوب لكائن القائمة:
```csharp
lstObj.Comment = "This is an Aspose.Cells comment.";
```

#### الخطوة 4: احفظ مصنفك
احفظ المصنف الخاص بك مع التعليق المضاف:
```csharp
workbook.Save(dataDir + "SetCommentOfTableOrListObject_out.xlsx", SaveFormat.Xlsx);
```

### نصائح استكشاف الأخطاء وإصلاحها:
- يضمن `source.xlsx` موجود في الدليل المحدد.
- تأكد من وجود كائن قائمة واحد على الأقل في ورقة العمل الخاصة بك.
