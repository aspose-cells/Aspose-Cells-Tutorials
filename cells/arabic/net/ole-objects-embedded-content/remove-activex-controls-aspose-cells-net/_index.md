## دليل التنفيذ

### إزالة عناصر تحكم ActiveX من مصنفات Excel
يرشدك هذا القسم إلى كيفية إزالة عناصر تحكم ActiveX باستخدام C# وAspose.Cells.

#### الخطوة 1: تحميل ملف Excel
قم بتحميل المصنف الذي يحتوي على عنصر تحكم ActiveX. استبدل `sourceDir` مع المسار إلى ملفك:
```csharp
// دليل المصدر
string sourceDir = "path_to_your_source_directory";

// إنشاء مصنف من ملف موجود
Workbook wb = new Workbook(sourceDir + "sampleUpdateActiveXComboBoxControl.xlsx");
```

#### الخطوة 2: الوصول إلى عنصر التحكم ActiveX وإزالته
قم بالوصول إلى الشكل الذي يحتوي على عنصر التحكم ActiveX الخاص بك، ثم قم بإزالته.
```csharp
// الوصول إلى الشكل الأول من ورقة العمل الأولى
Shape shape = wb.Worksheets[0].Shapes[0];

if (shape.ActiveXControl != null)
{
    // إزالة عنصر التحكم ActiveX الخاص بالشكل
    shape.RemoveActiveXControl();
}
```
**المعلمات موضحة:**
- `Workbook`:يمثل مصنف Excel.
- `Worksheet.Shapes`:الوصول إلى الأشكال، بما في ذلك عناصر تحكم ActiveX، في ورقة العمل.

#### الخطوة 3: حفظ المصنف المعدل
احفظ المصنف الخاص بك للحفاظ على التغييرات:
```csharp
// دليل الإخراج
string outputDir = "path_to_your_output_directory";

// حفظ المصنف المعدل
wb.Save(outputDir + "RemoveActiveXControl_our.xlsx");
```
**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن مسار الملف صحيح ويمكن الوصول إليه.
- تأكد من عدم وجود مشكلات تتعلق بأذونات الكتابة في دليل الحفظ الخاص بك.
