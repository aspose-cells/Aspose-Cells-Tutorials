## دليل التنفيذ

### إضافة أجزاء XML مخصصة

أولاً، لنُنشئ مُصنّف Excel ونُضيف إليه أجزاء XML مُخصّصة. يُمكن استخدام هذه الأجزاء لتمثيلات بيانات مُختلفة وإضافات منطق الأعمال في تطبيقك.

**الخطوة 1: إنشاء مصنف**

ابدأ بإنشاء مثيل جديد لـ `Workbook` فصل:

```csharp
// تهيئة كائن مصنف جديد
Workbook wb = new Workbook();
```

**الخطوة 2: إضافة أجزاء XML مخصصة**

سنضيف أجزاء XML مخصصة باستخدام مصفوفات البايتات. عمليًا، استبدلها ببيانات XML ومخططك الفعلي.

```csharp
byte[] btsData = { 1, 2, 3 };
byte[] btsSchema = { 1, 2, 3 };

// إضافة أربعة أجزاء XML مخصصة إلى المصنف
wb.CustomXmlParts.Add(btsData, btsSchema);
wb.CustomXmlParts.Add(btsData, btsSchema);
wb.CustomXmlParts.Add(btsData, btsSchema);
wb.CustomXmlParts.Add(btsData, btsSchema);
```

**الخطوة 3: تعيين معرفات لأجزاء XML المخصصة**

تعيين معرفات ذات معنى لكل جزء XML مخصص لسهولة التعرف عليه:

```csharp
wb.CustomXmlParts[0].ID = "Fruit";
wb.CustomXmlParts[1].ID = "Color";
wb.CustomXmlParts[2].ID = "Sport";
wb.CustomXmlParts[3].ID = "Shape";
```

### تحديد أجزاء XML المخصصة حسب المعرف

الآن، دعنا ننفذ الوظيفة لتحديد جزء XML مخصص استنادًا إلى معرفه.

**الخطوة 4: تحديد معرف البحث**

حدد جزء XML الذي تريد استرداده:

```csharp
String srchID = "Fruit"; // قم بتغيير هذه القيمة حسب الحاجة
```

**الخطوة 5: استرداد جزء XML المخصص**

استخدم `SelectByID` طريقة للعثور على جزء XML المخصص المطلوب وإرجاعه.

```csharp
Aspose.Cells.Markup.CustomXmlPart cxp = wb.CustomXmlParts.SelectByID(srchID);
```

**الخطوة 6: إخراج النتيجة**

تحقق مما إذا تم العثور على جزء XML وعرض الرسالة:

```csharp
if (cxp == null)
{
    Console.WriteLine("Not Found: CustomXmlPart ID " + srchID);
}
else
{
    Console.WriteLine("Found: CustomXmlPart ID " + srchID);
}

Console.WriteLine("AddCustomXMLPartsAndSelectThemByID executed successfully.");
```
