### نظرة عامة: إدراج كائن OLE في Excel

تتيح لك هذه الميزة تضمين الصور أو الملفات الأخرى مباشرةً في جداول بيانات Excel باستخدام لغة C#. إليك كيفية تحقيق ذلك خطوة بخطوة:

#### الخطوة 1: تحضير ملفاتك (H3)

أولاً، تأكد من إمكانية الوصول إلى الصورة والملف اللذين تريد تضمينهما. في هذا المثال، نستخدم صورة شعار وملف Excel.

```csharp
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// إنشاء الدليل إذا لم يكن موجودًا
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

#### الخطوة 2: تحميل بيانات الصورة والكائن (H3)

قراءة بيانات ملف الصورة والكائن في مصفوفات البايت.

```csharp
// قراءة الصورة في مجرى ثم في مجموعة بايتات
string ImageUrl = dataDir + "logo.jpg";
FileStream fs = File.OpenRead(ImageUrl);
byte[] imageData = new Byte[fs.Length];
fs.Read(imageData, 0, imageData.Length);
fs.Close();

// اقرأ ملف الكائن (على سبيل المثال، ملف Excel آخر) بنفس الطريقة
string path = dataDir + "book1.xls";
fs = File.OpenRead(path);
byte[] objectData = new Byte[fs.Length];
fs.Read(objectData, 0, objectData.Length);
fs.Close();
```

#### الخطوة 3: إضافة كائن OLE إلى ورقة العمل (H3)

قم بتضمين صورتك وملفك في ورقة العمل.

```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet sheet = workbook.Worksheets[0];

// أضف كائن Ole إلى ورقة العمل بالصورة المعروضة في MS Excel
sheet.OleObjects.Add(14, 3, 200, 220, imageData);

// تعيين بيانات كائن ole المضمنة
sheet.OleObjects[0].ObjectData = objectData;
```

#### الخطوة 4: حفظ المصنف (H3)

وأخيرًا، احفظ المصنف الخاص بك ليعكس هذه التغييرات.

```csharp
workbook.Save(dataDir + "output.out.xls");
```
