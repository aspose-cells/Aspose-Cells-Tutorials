# تصدير نطاق من الخلايا إلى صورة باستخدام Aspose.Cells

## مقدمة
عند العمل مع ملفات Excel، تُعدّ إمكانية تحويل نطاقات خلايا محددة إلى صور مفيدة للغاية. تخيّل أنك تحتاج إلى مشاركة جزء مهم من جدول بياناتك دون إرسال المستند بأكمله، وهنا يأتي دور Aspose.Cells لـ .NET! في هذا الدليل، سنشرح لك خطوة بخطوة عملية تصدير نطاق خلايا إلى صورة، مما يضمن لك إتقان كل جزء من العملية دون أي تعقيدات تقنية.
## المتطلبات الأساسية
قبل الخوض في البرنامج التعليمي، هناك بعض المتطلبات الأساسية لضمان إعداد كل شيء بشكل صحيح:
1. Visual Studio: تأكد من تثبيت Visual Studio على نظامك.
2. Aspose.Cells لـ .NET: قم بتنزيل هذه المكتبة من [موقع Aspose](https://releases.aspose.com/cells/net/)يمكنك أيضًا بدء تجربة مجانية إذا كنت ترغب في استكشاف إمكانياتها قبل الالتزام بها.
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة بلغة C# وإطار عمل .NET على فهم الكود بشكل أفضل.
4. ملف Excel نموذجي: في هذا البرنامج التعليمي، سنستخدم ملفًا باسم `sampleExportRangeOfCellsInWorksheetToImage.xlsx`يمكنك إنشاء ملف Excel بسيط لأغراض الاختبار.
الآن بعد أن قمنا بتغطية المتطلبات الأساسية، دعنا ننتقل مباشرة إلى الكود!
## استيراد الحزم
للبدء، علينا استيراد مساحات الأسماء الأساسية. إليك كيفية القيام بذلك:
```csharp
using System.IO;
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using Aspose.Cells.Rendering;
using System;
```
ستسمح لنا هذه الحزم بالعمل مع المصنفات وأوراق العمل وإدارة عرض نطاقات الخلايا الخاصة بنا.
## الخطوة 1: إعداد مسارات الدليل الخاصة بك
قد يبدو إعداد المجلدات أمرًا مملًا، ولكنه بالغ الأهمية. تضمن هذه الخطوة أن يعرف برنامجك مكان العثور على الملفات ومكان حفظ الصور المُصدَّرة.
```csharp
// دليل المصدر
string sourceDir = "Your Document Directory";
// دليل الإخراج
string outputDir = "Your Document Directory";
```
يستبدل `"Your Document Directory"` مع المسار الفعلي لملفاتك. قد يكون هذا مسارًا على محرك الأقراص المحلي أو دليل شبكة.
## الخطوة 2: إنشاء مصنف من ملف المصدر
الخطوة التالية هي إنشاء `Workbook` الكائن الذي يعمل كنقطة دخول إلى ملف Excel.
```csharp
// إنشاء مصنف من ملف المصدر.
Workbook workbook = new Workbook(sourceDir + "sampleExportRangeOfCellsInWorksheetToImage.xlsx");
```
هنا نقوم بإنشاء جديد `Workbook` على سبيل المثال، أدخل المسار الكامل لملف Excel الذي تريد العمل عليه. هذه الخطوة تفتح الملف وتُهيئه للعمل.
## الخطوة 3: الوصول إلى ورقة العمل الأولى
بمجرد حصولنا على المصنف، نحتاج إلى الوصول إلى ورقة العمل التي تحتوي على البيانات التي نرغب في تصديرها.
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```
ال `Worksheets` المجموعة مفهرسة بـ 0، مما يعني أن `Worksheets[0]` تظهر لنا الورقة الأولى. يمكنك تعديل الفهرس إذا أردت ورقة أخرى.
## الخطوة 4: تعيين منطقة الطباعة
بعد ذلك، علينا تحديد المنطقة التي نريد تصديرها كصورة. يتم ذلك بتحديد منطقة الطباعة في ورقة العمل.
```csharp
// قم بتعيين منطقة الطباعة بالنطاق المطلوب
worksheet.PageSetup.PrintArea = "D8:G16";
```
في هذه الحالة، نحدد أننا نريد تصدير الخلايا من D8 إلى G16. عدّل مراجع هذه الخلايا بناءً على البيانات التي تريد التقاطها.
## الخطوة 5: تكوين الهوامش
لنتأكد من خلو الصورة المُصدّرة من أي مسافات غير ضرورية. سنُعيّن جميع الهوامش إلى صفر.
```csharp
// تعيين جميع الهوامش إلى 0
worksheet.PageSetup.LeftMargin = 0;
worksheet.PageSetup.RightMargin = 0;
worksheet.PageSetup.TopMargin = 0;
worksheet.PageSetup.BottomMargin = 0;
```
تعتبر هذه الخطوة ضرورية لضمان أن الصورة الناتجة تتناسب تمامًا مع المظهر العام دون أي فوضى حولها.
## الخطوة 6: تعيين خيارات الصورة
بعد ذلك، نضبط خيارات عرض الصورة، بما في ذلك الدقة ونوع الصورة.
```csharp
// تعيين خيار OnePagePerSheet على true
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.OnePagePerSheet = true;
options.ImageType = ImageType.Jpeg;
options.HorizontalResolution = 200;
options.VerticalResolution = 200;
```
هنا، نؤكد على ضرورة أن تكون الصورة بصيغة JPEG بدقة 200 نقطة في البوصة. يمكنك تعديل الدقة حسب احتياجاتك.
## الخطوة 7: تحويل ورقة العمل إلى صورة
الآن يأتي الجزء المثير: تحويل ورقة العمل إلى صورة!
```csharp
// التقط صورة لورقة العمل الخاصة بك
SheetRender sr = new SheetRender(worksheet, options);
sr.ToImage(0, outputDir + "outputExportRangeOfCellsInWorksheetToImage.jpg");
```
نحن ننشئ `SheetRender` المثيل والدعوة `ToImage` لإنشاء الصورة من الصفحة الأولى من ورقة العمل المحددة. تُحفظ الصورة في مجلد الإخراج باسم الملف المحدد.
## الخطوة 8: تأكيد التنفيذ
وأخيرًا، من الجيد دائمًا تقديم تعليقات بعد اكتمال العملية، حتى نتمكن من طباعة رسالة على وحدة التحكم.
```csharp
Console.WriteLine("ExportRangeOfCellsInWorksheetToImage executed successfully.\r\n");
```
تعتبر هذه الخطوة ضرورية لتأكيد نجاح العملية، خاصة عند تشغيل الكود في تطبيق وحدة التحكم.
