## دليل التنفيذ
سنركز على تحديد لغة ملف Excel باستخدام خصائص المستند المُدمجة. تتيح هذه الميزة للمستخدمين تحديد اللغات الأساسية المستخدمة في مستنداتهم لتحسين إمكانية الوصول والترجمة.

### الخطوة 1: إنشاء كائن مصنف
ابدأ بإنشاء كائن مصنف جديد، والذي يمثل ملف Excel الخاص بك.

```csharp
// تهيئة مكتبة Aspose.Cells
Workbook wb = new Workbook();
```

يؤدي هذا السطر إلى إنشاء مصنف فارغ حيث يمكنك إضافة البيانات أو الأوراق أو الخصائص حسب الحاجة.

### الخطوة 2: الوصول إلى خصائص المستند المضمنة
لتغيير إعدادات اللغة، قم بالوصول إلى مجموعة خصائص المستند المضمنة في مصنف العمل الخاص بك:

```csharp
// الوصول إلى خصائص المستند المضمنة
Aspose.Cells.Properties.BuiltInDocumentPropertyCollection bdpc = wb.BuiltInDocumentProperties;
```

هنا، `bdpc` هي مجموعة تحتوي على خصائص مستند مختلفة مثل اسم المؤلف والعنوان واللغة.

### الخطوة 3: ضبط اللغة
حدد اللغات المستخدمة في ملف Excel. هذا يُساعد المستخدمين الذين يستخدمون برامج قراءة الشاشة أو أدوات الترجمة على فهم المحتوى بشكل أفضل.

```csharp
// ضبط اللغة إلى الألمانية والفرنسية

```

في هذه الخطوة، قمنا بتعيين اللغتين الألمانية والفرنسية كلغتين أساسيتين لمستندنا.

### الخطوة 4: احفظ مصنفك
أخيرًا، احفظ مصنفك مع هذه الخصائص. هذا يضمن الحفاظ على جميع الإعدادات:

