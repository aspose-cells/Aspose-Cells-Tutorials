## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى ميزتين رئيسيتين: إعداد معاودة الاتصال بالتحذير وتحميل ملف Excel مع معالجة التحذيرات.

### الميزة 1: استدعاء التحذير

**ملخص**

تتضمن هذه الميزة إنشاء فئة تنفذ `IWarningCallback` لاعتراض التحذيرات أثناء تحميل المصنفات، وخاصة لإدارة الأسماء المكررة المحددة أو المشكلات الأخرى.

#### الخطوة 1: تنفيذ واجهة IWarningCallback

إنشاء فئة باسم `WarningCallback` على النحو التالي:
```csharp
using System;
using Aspose.Cells;

namespace AsposeCellsExamples
{
    private class استدعاء تحذيري : IWarningCallback
    {
        public void Warning(WarningInfo warningInfo)
        {
            if (warningInfo.WarningType == WarningType.DuplicateDefinedName)
            {
                Console.WriteLine("Duplicate Defined Name Warning: " + warningInfo.Description);
            }
        }
    } // WarningCallback
}
```
**توضيح**: ال `Warning` تلتقط هذه الطريقة التحذيرات وتعالجها. هنا، تتحقق تحديدًا من الأسماء المُعرّفة المكررة.

### الميزة 2: تحميل ملف Excel مع معالجة التحذيرات

**ملخص**

في هذه الميزة، نقوم بتحميل مصنف Excel أثناء استخدام استدعاء التحذير المخصص للتعامل مع أي مشكلات قد تنشأ.

#### الخطوة 1: تحديد أدلة المصدر والإخراج

إعداد مسارات الدليل الخاص بك:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";
```
تأكد من أن هذه المسارات تشير إلى أدلة صالحة على نظامك.

#### الخطوة 2: تكوين LoadOptions باستخدام استدعاء التحذير

يخلق `LoadOptions` وتعيين استدعاء التحذير:
```csharp
LoadOptions options = new LoadOptions();
options.WarningCallback = new WarningCallback();
```

#### الخطوة 3: تحميل المصنف وحفظ الناتج

وأخيرًا، قم بتحميل المصنف وحفظه في الدليل المحدد:
```csharp
Workbook book = new Workbook(SourceDir + "/sampleDuplicateDefinedName.xlsx", options);
book.Save(OutputDir + "/outputDuplicateDefinedName.xlsx");
```
**توضيح**يقوم هذا الكود بتحميل ملف إكسل بالتحذيرات المحتملة التي تتم معالجتها بواسطة دالة الاستدعاء المخصصة لدينا. ثم يحفظ المصنف المُعالج.
