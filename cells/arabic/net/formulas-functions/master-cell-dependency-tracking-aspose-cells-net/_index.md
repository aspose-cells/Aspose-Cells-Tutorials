## دليل التنفيذ

### تحميل المصنف

قم بتحميل المصنف الخاص بك لتحديد ملف Excel:
```csharp
// تحميل مصنف موجود من مسار محدد
Workbook workbook = new Workbook("Book1.xlsx");
```
#### ملخص
يؤدي هذا إلى تهيئة `Workbook` الكائن، الذي يوفر إمكانية الوصول إلى أوراق العمل والخلايا.

### الوصول إلى الخلايا وتتبع التبعيات
حدد ورقة العمل والخلية لتتبع التبعيات:
```csharp
// احصل على ورقة العمل الأولى في المصنف
Worksheet worksheet = workbook.Worksheets[0];

// الوصول إلى خلية محددة
Cell targetCell = worksheet.Cells["B2"];
```
#### ملخص
الوصول إلى `Cells` مجموعة من ورقة العمل المحددة لتحديد الخلية المستهدفة.

### الحصول على المعالين
استخدم `GetDependents` طريقة استرجاع الخلايا التابعة:
```csharp
// احصل على جميع الخلايا التابعة لـ 'B2'
Cell[] dependents = targetCell.GetDependents(true);

foreach (Cell c in dependents)
{
    Console.WriteLine(c.Name); // إخراج أسماء الخلايا التابعة
}
```
#### ملخص
`GetDependents(true)` العائدات `Cell` الكائنات المتأثرة بالتغييرات في الخلية المحددة.
