# تحويل المخطط إلى PDF

## مقدمة

عند التعامل مع جداول البيانات، غالبًا ما تلعب المخططات البيانية دورًا حاسمًا في عرض البيانات بفعالية. سواء كنت تُعدّ تقريرًا، أو تُقدّم عرضًا تقديميًا، أو تُسهّل تحليل البيانات، فإن تحويل هذه المخططات إلى صيغة PDF يُضفي لمسةً احترافية. سنشرح لك هنا خطوات تحويل مخطط Excel إلى صيغة PDF باستخدام Aspose.Cells for .NET، وهي مكتبة فعّالة مُصمّمة لتبسيط عمليات معالجة Excel.

## المتطلبات الأساسية

قبل البدء في البرنامج التعليمي، تأكد من إعداده بشكل صحيح. إليك ما تحتاجه:

### إطار عمل .NET
تأكد من تثبيت إطار عمل .NET على جهازك. Aspose.Cells متوافق مع إصدارات مختلفة، ولكنه يعمل بشكل أفضل مع الإصدارات الأحدث.

### مكتبة Aspose.Cells
ستحتاج إلى مكتبة Aspose.Cells لـ .NET. يمكنك تنزيلها من [هنا](https://releases.aspose.com/cells/net/)تأتي المكتبة مع واجهة برمجة تطبيقات غنية تحتوي على جميع الوظائف التي تحتاجها لعمليات معالجة Excel.

### فيجوال ستوديو
يعد تثبيت Visual Studio أمرًا ضروريًا، لأنه يعتبر بيئة تطوير متكاملة رائعة لكتابة كود .NET الخاص بك بسلاسة.

### المعرفة الأساسية بلغة C#
إن بعض المعرفة بلغة البرمجة C# سوف تساعدك على فهم أجزاء التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

لاستخدام Aspose.Cells بنجاح في مشروعك، عليك استيراد الحزم اللازمة. إليك كيفية القيام بذلك:

### إنشاء مشروع جديد

ابدأ بإنشاء مشروع C# جديد في Visual Studio:

1. افتح Visual Studio.
2. انقر فوق "إنشاء مشروع جديد".
3. حدد "تطبيق وحدة التحكم (.NET Core)" أو "تطبيق وحدة التحكم (.NET Framework)" بناءً على متطلباتك.
4. قم بتسمية مشروعك وانقر على "إنشاء".

### إضافة مرجع Aspose.Cells

بعد إنشاء مشروعك، يجب عليك إضافة مرجع إلى مكتبة Aspose.Cells:

1. في مستكشف الحلول، انقر بزر الماوس الأيمن فوق مشروعك.
2. اختر "إدارة حزم NuGet".
3. ابحث عن “Aspose.Cells” وقم بتثبيته.

بمجرد تضمين المكتبة في مشروعك، ستكون جاهزًا للانتقال إلى الكود.

### استيراد مساحات الأسماء المطلوبة

في الجزء العلوي من `Program.cs` الملف، أضف المساحات التالية:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using Aspose.Cells.Charts;
using System.IO;
```

إليك كيفية تحويل مخطط Excel إلى PDF بطريقة منهجية. اتبع الخطوات خطوة بخطوة!

## الخطوة 1: إعداد أدلة الإخراج والمصدر

لبدء الكود الخاص بك، ستحتاج أولاً إلى تحديد المكان الذي ستحفظ فيه مخرجاتك ومكان مستند المصدر الخاص بك.

```csharp
// دليل الإخراج
string outputDir = "Your Output Directory";

// دليل المصدر
string sourceDir = "Your Document Directory";
```

تأكد من الاستبدال `"Your Output Directory"` و `"Your Document Directory"` مع المسار الفعلي الذي توجد به ملفاتك.

## الخطوة 2: تحميل مصنف Excel

الآن، لنحمّل ملف Excel الذي يحتوي على المخططات التي تريد تحويلها. الأمر بسيط جدًا:

```csharp
// تحميل ملف Excel الذي يحتوي على الرسوم البيانية
Workbook workbook = new Workbook(sourceDir + "sampleChartToPdf.xlsx");
```

يقوم هذا الكود بتهيئة مصنف جديد وتحميل ملف Excel المحدد. تأكد من أن اسم الملف يطابق اسم الملف الموجود في مجلد المصدر.

## الخطوة 3: الوصول إلى ورقة العمل

بعد ذلك، عليك الوصول إلى ورقة العمل التي تحتوي على المخطط الذي ترغب في تحويله. إليك كيفية القيام بذلك:

```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```

يتيح لك هذا الكود الوصول إلى ورقة العمل الأولى في المصنف الخاص بك، مما يسمح لك بالعمل معها.

## الخطوة 4: الوصول إلى الرسم البياني 

بمجرد حصولك على ورقة العمل، حان الوقت للوصول إلى المخطط المحدد الذي تريد تحويله:

```csharp
// الوصول إلى المخطط الأول داخل ورقة العمل
Chart chart = worksheet.Charts[0];
```

يلتقط هذا الخط أول مخطط في ورقة العمل. إذا كانت ورقة العمل تحتوي على عدة مخططات وتحتاج إلى استهداف مخطط محدد، فاضبط الفهرس وفقًا لذلك.

## الخطوة 5: تحويل الرسم البياني إلى PDF

الآن يأتي الجزء المثير: تحويل المخطط إلى صيغة PDF. يمكنك إما حفظه في ملف أو في ذاكرة.

### الخيار 1: حفظ الرسم البياني في ملف

لحفظ الرسم البياني مباشرة في ملف PDF، استخدم الكود التالي:

```csharp
// احفظ الرسم البياني بتنسيق pdf
chart.ToPdf(outputDir + "outputChartToPdf.pdf");
```

تأكد فقط من وجود دليل الإخراج بالفعل لتجنب أي أخطاء.

### الخيار 2: حفظ الرسم البياني في مجرى الذاكرة

إذا كنت ترغب في معالجة ملف PDF بشكل أكبر أو تحتاج إلى استخدامه على الفور في تطبيقك، فقد يكون حفظه في مجرى الذاكرة هو الخيار الأفضل:

```csharp
// احفظ الرسم البياني بتنسيق pdf في التدفق
MemoryStream ms = new MemoryStream();
chart.ToPdf(ms);
```

هنا، يمكنك حفظ ملف PDF في مجرى الذاكرة، والذي يمكن استخدامه وفقًا لاحتياجات تطبيقك.

## الخطوة 6: عرض رسالة النجاح

وأخيرًا، من الجيد دائمًا الإشارة إلى نجاح عمليتك. يمكنك ببساطة طباعة رسالة نجاح على لوحة التحكم:

```csharp
Console.WriteLine("ChartToPdf executed successfully.");
```

## خاتمة

ها قد انتهيت! باستخدام Aspose.Cells لـ .NET، أصبح تحويل مخططات Excel إلى تنسيقات PDF سهلاً للغاية. سواءً اخترت الحفظ في ملف أو في ذاكرة، تضمن لك المكتبة مرونة وسهولة في الاستخدام. لم لا تجربها؟ ستبدو تقاريرك أكثر وضوحًا مع مخططات PDF بتنسيق احترافي!
