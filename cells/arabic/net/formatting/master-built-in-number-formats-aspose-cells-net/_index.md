## دليل التنفيذ
دعنا نقسم التنفيذ إلى أجزاء قابلة للإدارة، مع التركيز على تطبيق تنسيقات الأرقام المضمنة على أنواع مختلفة من البيانات.

### إعداد مصنف العمل الخاص بك

#### ملخص
ابدأ بإنشاء ملف Excel جديد واحصل على مراجع لأوراق العمل الخاصة به. هذه الخطوة أساسية للتعامل مع أنماط الخلايا بفعالية.

**إنشاء مصنف عمل**
```csharp
// إنشاء مثيل جديد للمصنف
Workbook workbook = new Workbook();

// الوصول إلى ورقة العمل الأولى في المصنف
Worksheet worksheet = workbook.Worksheets[0];
```

### تنسيق التواريخ

#### ملخص
عرض التواريخ بتنسيق سهل الاستخدام ضروري للوضوح. لنطبّق تنسيق "يوم-شهر-سنة" على خلية.

**تطبيق تنسيق التاريخ**
```csharp
// أدخل التاريخ الحالي في الخلية A1
worksheet.Cells["A1"].PutValue(DateTime.Now);

// استرجاع وتعديل نمط الخلية
Style style = worksheet.Cells["A1"].GetStyle();
style.Number = 15; // التنسيق المدمج لـ "d-mmm-yy"
worksheet.Cells["A1"].SetStyle(style);
```

### تنسيق النسب المئوية

#### ملخص
إن تحويل القيم الرقمية إلى نسب مئوية قد يعزز تفسير البيانات، وخاصة في التقارير المالية.

**تطبيق تنسيق النسبة المئوية**
```csharp
// إدراج قيمة رقمية في الخلية A2
worksheet.Cells["A2"].PutValue(20);

// تعديل نمط عرض النسبة المئوية
style = worksheet.Cells["A2"].GetStyle();
style.Number = 9; // تنسيق مدمج للنسب المئوية
worksheet.Cells["A2"].SetStyle(style);
```

### تنسيق العملة

#### ملخص
غالبًا ما تتطلب البيانات المالية تنسيق العملة لضمان الاتساق عبر التقارير.

**تطبيق تنسيق العملة**
```csharp
// إدراج قيمة رقمية في الخلية A3
worksheet.Cells["A3"].PutValue(2546);

// تعيين نمط عرض العملة
style = worksheet.Cells["A3"].GetStyle();
style.Number = 6; // تنسيق مدمج للعملة
worksheet.Cells["A3"].SetStyle(style);
```

### حفظ مصنفك
وأخيرًا، احفظ المصنف الخاص بك في ملف Excel:
```csharp
// احفظ المصنف بتنسيق Excel97To2003
workbook.Save("path/to/your/book1.out.xls", SaveFormat.Excel97To2003);
```
