### ضبط حجم الخط في الخلية (H2)
#### ملخص:
يُمكن لتخصيص مظهر النص بتحديد أحجام خطوط مُحددة داخل خلية Excel أن يُحسّن الوضوح. إليك كيفية تحقيق ذلك باستخدام Aspose.Cells لـ .NET.
##### الخطوة 1: جهّز بيئتك
ابدأ بإعلان دليل المصدر ودليل الإخراج.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
##### الخطوة 2: إضافة ورقة عمل والوصول إلى الخلايا
أضف ورقة عمل جديدة إلى المصنف الخاص بك وقم بالوصول إلى الخلية المطلوبة.
```csharp
int i = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[i];
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
cell.PutValue("Hello Aspose!");
```
##### الخطوة 3: تعيين حجم الخط
احصل على نمط الخلية، وعدّل حجم الخط، ثم قم بتطبيقه مرة أخرى.
```csharp
Style style = cell.GetStyle();
style.Font.Size = 14; // قم بتعيين حجم الخط المطلوب هنا
cell.SetStyle(style);
```
##### الخطوة 4: احفظ مصنفك
وأخيرًا، احفظ المصنف الخاص بك لمراقبة التغييرات.
```csharp
workbook.Save(outputDir + "SetFontSizeExample.out.xls", SaveFormat.Excel97To2003);
```
