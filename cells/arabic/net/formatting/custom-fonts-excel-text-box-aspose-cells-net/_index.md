## دليل التنفيذ

الآن بعد أن قمنا بإعداد بيئتنا، دعنا نتعمق في تنفيذ إعدادات الخط المخصصة لمربعات النص.

### إضافة مربع نص إلى ورقة عمل Excel

**ملخص**سنضيف مربع نص ونُهيئ خطوطه باستخدام Aspose.Cells. تتيح لك هذه الميزة تحديد خطوط مختلفة لمجموعات الأحرف اللاتينية والشرقية في مربع النص نفسه.

#### الخطوة 1: إنشاء مصنف فارغ

ابدأ بإنشاء مصنف جديد والوصول إلى ورقة العمل الأولى الخاصة به:

```csharp
// إنشاء مصنف جديد.
Workbook wb = new Workbook();

// الوصول إلى ورقة العمل الأولى.
Worksheet ws = wb.Worksheets[0];
```

#### الخطوة 2: إضافة مربع نص إلى ورقة العمل

بعد ذلك، قم بإضافة مربع نص عند الإحداثيات المحددة ضمن ورقة العمل.

```csharp
// أضف مربع نص داخل ورقة العمل.
int idx = ws.TextBoxes.Add(5, 5, 50, 200);
Aspose.Cells.Drawing.TextBox tb = ws.TextBoxes[idx];
```

#### الخطوة 3: تعيين أسماء النصوص والخطوط

قم بتعيين نص مربع النص وتحديد الخطوط المخصصة لكل من أحرف الشرق الأقصى واللاتينية.

```csharp
// تعيين نص مربع النص.
tb.Text = "こんにちは世界";

// حدد أسماء الخطوط.
tb.TextOptions.LatinName = "Comic Sans MS";
tb.TextOptions.FarEastName = "KaiTi";
```

#### الخطوة 4: احفظ مصنفك

وأخيرًا، احفظ المصنف الخاص بك في ملف إخراج.

```csharp
// احفظ ملف Excel الناتج.
wb.Save("outputSpecifyFarEastAndLatinNameOfFontInTextOptionsOfShape.xlsx", SaveFormat.Xlsx);
```
