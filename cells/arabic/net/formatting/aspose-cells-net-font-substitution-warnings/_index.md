### تنفيذ استدعاء تحذير استبدال الخط
#### ملخص
الهدف هنا هو إنشاء آلية تنبهك عندما تقوم المكتبة باستبدال خط أثناء التحويل، مما يضمن أن يتطابق الناتج الخاص بك مع التوقعات.

#### التنفيذ خطوة بخطوة
**إنشاء فئة Callback**
تعريف فئة التنفيذ `IWarningCallback` للتعامل مع التحذيرات أثناء العمليات مثل التحويلات:
```csharp
using Aspose.Cells;
using System.Diagnostics;

public class GetWarningsForFontSubstitution : IWarningCallback
{
    // طريقة لالتقاط وتسجيل تحذيرات استبدال الخط.
    public void Warning(WarningInfo info)
    {
        if (info.WarningType == WarningType.FontSubstitution)
        {
            Debug.WriteLine("WARNING INFO: " + info.Description);
        }
    }
}
```

**توضيح:** تستمع هذه الفئة لأحداث التحذير أثناء التحويل. إذا كان نوع الحدث هو `FontSubstitution`، فإنه يسجل رسالة مفصلة باستخدام `Debug.WriteLine`.

### تحويل كتاب العمل إلى PDF مع تحذيرات استبدال الخط
#### ملخص
بعد أن أصبحت مكالمة التحذير جاهزة، فلنستخدمها لتحويل مصنف Excel إلى ملف PDF أثناء التقاط تحذيرات استبدال الخط.

**تنفيذ التحويل**
إنشاء فئة ثابتة وطريقة للتعامل مع عملية التحويل:
```csharp
using Aspose.Cells;
using System.IO;

public static class ConvertWorkbookToPdfWithWarnings
{
    public static void Run()
    {
        // قم بتحديد مصدرك ومخرجاتك.
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string OutputDir = "YOUR_OUTPUT_DIRECTORY";

        // قم بتحميل مصنف Excel من الدليل المحدد.
        Workbook workbook = new Workbook(SourceDir + "sampleGetWarningsForFontSubstitution.xlsx");

        // قم بإنشاء مثيل لـ PdfSaveOptions لتخصيص خيارات الحفظ.
        PdfSaveOptions options = new PdfSaveOptions();

        // قم بتعيين استدعاء التحذير الخاص بنا للتعامل مع تحذيرات استبدال الخط.
        options.WarningCallback = new GetWarningsForFontSubstitution();

        // احفظ المصنف كملف PDF، باستخدام الخيارات المحددة.
        workbook.Save(OutputDir + "outputGetWarningsForFontSubstitution.pdf", options);
    }
}
```

**توضيح:** يقوم هذا الكود بتحميل ملف Excel وإعداده `PdfSaveOptions` لاستخدام استدعاء التحذير المخصص لدينا. عند استدعاء `workbook.Save`يتم التقاط أي تحذيرات لاستبدال الخط بواسطة معاودة الاتصال، مما يسمح بالتحكم بشكل أفضل في جودة الإخراج.
