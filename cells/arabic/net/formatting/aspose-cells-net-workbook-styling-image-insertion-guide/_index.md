# أتمتة Excel باستخدام Aspose.Cells: تنسيق المصنفات وإدراج الصور

## إتقان Aspose.Cells .NET: دليل شامل لتصميم المصنفات وإدراج الصور

### مقدمة

هل تحتاج إلى أتمتة إنشاء مصنفات Excel، أو تنسيق الخلايا بدقة، أو إدراج الصور بسلاسة؟ سواء كنت مطورًا تُحسّن أدوات إعداد التقارير أو محللًا يسعى إلى عروض تقديمية جذابة بصريًا للبيانات، فإن إتقان هذه المهام يُحدث نقلة نوعية في طريقة تعاملك مع جداول البيانات برمجيًا. سيرشدك هذا الدليل إلى كيفية استخدام Aspose.Cells لـ .NET لإنشاء مصنفات وتنسيقها، وإدراج الصور بسهولة.

#### ما سوف تتعلمه:
- **تهيئة المصنف**:فهم أساسيات إنشاء مصنف جديد.
- **تقنيات تصفيف الخلايا**:تطبيق الأنماط مثل ألوان الخلفية على الخلايا بشكل فعال.
- **إدراج الصورة**:تعرف على كيفية إضافة الصور داخل خلايا جدول البيانات الخاص بك.
- **التطبيقات العملية**:اكتشف حالات الاستخدام الواقعية لهذه الميزات.

دعونا نتعمق في المتطلبات الأساسية اللازمة قبل أن نبدأ في الترميز!

## دليل التنفيذ

سنقوم بتقسيم دليلنا إلى قسمين رئيسيين: **تنسيق المصنف** و **إدراج الصورة**.

### تهيئة المصنف وتنسيق الخلايا

#### ملخص
توضح هذه الميزة إنشاء مصنف، والوصول إلى الخلايا، وتطبيق الأنماط عليها. وهي ضرورية لإنشاء تقارير أو لوحات معلومات جذابة بصريًا برمجيًا.

##### الخطوة 1: إنشاء مصنف جديد
إنشاء مثيل جديد `Workbook` هدف.
```csharp
using Aspose.Cells;

// إنشاء مصنف جديد
Workbook workbook = new Workbook();
```

##### الخطوة 2: الوصول إلى الخلايا وتطبيق الأنماط
قم بالوصول إلى مجموعة الخلايا في ورقة العمل الأولى وإنشاء الأنماط.
```csharp
Cells cells = workbook.Worksheets[0].Cells;
Style st = workbook.CreateStyle();
st.Pattern = BackgroundType.Solid;
st.ForegroundColor = Color.Yellow;

// إضافة قيم السلسلة إلى الخلايا وتعيين الأنماط
cells["A1"].PutValue("A1");
cells["A1"].SetStyle(st, true);

st.ForegroundColor = Color.Red;
cells["C10"].PutValue("C10");
cells["C10"].SetStyle(st, true);
```

##### الخطوة 3: حفظ المصنف
قم بتحديد دليل الإخراج وحفظ المصنف المصمم الخاص بك.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/WorkbookInitializationAndStyling.xlsx");
```

### إضافة الصور وتصميمها في خلايا المصنف

#### ملخص
تعرف على كيفية إضافة الصور داخل الخلايا، وتعيين الصيغ التي تشير إلى هذه الصور، وضبط أحجامها للحصول على عرض تقديمي ديناميكي.

##### الخطوة 1: تحضير المصنف وورقة العمل
إنشاء مصنف والوصول إلى مجموعة الأشكال الخاصة به.
```csharp
using Aspose.Cells;
using System.IO;

// إنشاء مصنف عمل موجود أو إنشاء مصنف جديد
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
ShapeCollection shapes = sheet.Shapes;
```

##### الخطوة 2: إضافة صورة إلى الخلية D1
إنشاء مجرى للصورة وإضافتها إلى خلية محددة.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
byte[] imagedata = ConditionalFormattingIcon.GetIconImageData(IconSetType.TrafficLights31, 0);
MemoryStream stream = new MemoryStream(imagedata);

// أضف صورة إلى الخلية D1 (في مؤشر الصف 5، ومؤشر العمود 5)
Picture pic = shapes.AddPicture(5, 5, stream, 600, 600);
```

##### الخطوة 3: حفظ المصنف بالصور
قم بتحديد دليل الإخراج وحفظ المصنف الخاص بك.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/AddPictureToCell.xlsx");
```
