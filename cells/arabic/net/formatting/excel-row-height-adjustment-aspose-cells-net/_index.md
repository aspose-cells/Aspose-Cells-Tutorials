### دليل التنفيذ
سنرشدك خلال تعديل ارتفاعات الصفوف خطوة بخطوة.

#### نظرة عامة على تعديل ارتفاع الصف
يؤدي تعديل ارتفاع الصف إلى تحسين رؤية البيانات وعرضها، خاصةً عندما يختلف المحتوى عبر الخلايا.

##### الخطوة 1: افتح المصنف الخاص بك
قم بتحميل ملف Excel الخاص بك إلى `Workbook` كائن باستخدام مجرى ملف.
```csharp
using System.IO;
using Aspose.Cells;

namespace AsposeCellsExamples
{
    public class SettingHeightOfRowExample
    {
        public static void Run()
        {
            // حدد المسار إلى دليل المستند الخاص بك
            string dataDir = "path_to_your_directory";
            
            // افتح مجرى ملف لمستند Excel الخاص بك
            using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
            {
                // إنشاء كائن مصنف باستخدام مجرى الملف المفتوح
                Workbook workbook = new Workbook(fstream);

                // الوصول إلى ورقة العمل وتعديلها...
            }
        }
    }
}
```

##### الخطوة 2: الوصول إلى ورقة العمل
قم بالوصول إلى ورقة العمل المحددة التي تريد ضبط ارتفاع الصف فيها.
```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];
```

##### الخطوة 3: تعيين ارتفاع الصف
استخدم `SetRowHeight` طريقة لتغيير ارتفاع صف معين. هنا، قمنا بتعيين ارتفاع الصف الثاني إلى ١٣ نقطة.
```csharp
// ضبط ارتفاع الصف الثاني (المؤشر 1) إلى 13 نقطة
worksheet.Cells.SetRowHeight(1, 13);
```

##### الخطوة 4: احفظ مصنفك
بعد إجراء التغييرات، احفظ مصنفك مرة أخرى في ملف أو قم ببثه حسب الحاجة.
```csharp
// حفظ ملف Excel المعدل
workbook.Save(dataDir + "output.out.xls");
```
