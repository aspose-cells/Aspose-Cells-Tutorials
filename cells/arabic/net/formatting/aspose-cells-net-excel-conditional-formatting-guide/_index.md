# إتقان التنسيق الشرطي في Aspose.Cells .NET لـ Excel

## مقدمة

حوّل جداول بيانات Excel لديك إلى بيانات ديناميكية وجذابة بصريًا باستخدام Aspose.Cells لـ .NET. سيرشدك هذا الدليل الشامل خلال عملية تطبيق قواعد التنسيق الشرطي المتقدمة لتحسين سهولة الاستخدام والجماليات في جداول بياناتك.

**ما سوف تتعلمه:**
- إنشاء مثيل لمصنف عمل وورقة عمل في Excel
- إضافة قواعد التنسيق الشرطي إلى الخلايا
- تخصيص ألوان الخلفية للبيانات المميزة
- حفظ ملف Excel المُنسّق

هل أنت مستعد للارتقاء بعرض بياناتك؟ لنُهيئ بيئتك ونبدأ البرمجة!

## دليل التنفيذ
سنقوم بتقسيم كل ميزة من ميزات التنسيق الشرطي باستخدام Aspose.Cells لـ .NET إلى خطوات قابلة للإدارة.

### إنشاء مصنف وورقة عمل
**ملخص:** يوضح هذا القسم كيفية إنشاء مصنف Excel جديد والوصول إلى ورقة العمل الأولى الخاصة به.

#### الخطوة 1: إنشاء مصنف جديد
```csharp
// تهيئة كائن المصنف.
Workbook workbook = new Workbook();
```
- **المعايير والغرض**: ال `Workbook` يقوم المُنشئ بتهيئة ملف Excel جديد. افتراضيًا، يُنشئ ورقة عمل فارغة.

#### الخطوة 2: الوصول إلى ورقة العمل الأولى
```csharp
// قم بالوصول إلى ورقة العمل الأولى في المصنف.
Worksheet sheet = workbook.Worksheets[0];
```
ال `Worksheets[0]` يصل الفهرس إلى ورقة العمل الأولية التي تم إنشاؤها باستخدام المصنف.

### إضافة قواعد التنسيق الشرطي
**ملخص:** تعرف على كيفية تحديد قواعد التنسيق الشرطي لنطاقات خلايا محددة ضمن ورقة العمل.

#### الخطوة 1: إضافة قاعدة تنسيق شرطي جديدة
```csharp
// إضافة قاعدة تنسيق شرطية جديدة.
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcs = sheet.ConditionalFormattings[index];
```
- **غاية**: `ConditionalFormattings.Add()` ينشئ قاعدة جديدة ويعيد فهرسها.

#### الخطوة 2: تحديد منطقة الخلية
```csharp
// إعداد مناطق الخلايا لتطبيق التنسيق الشرطي.
CellArea ca = new CellArea();
ca.StartRow = 0;
c.EndRow = 0;
ca.StartColumn = 0;
c.EndColumn = 0;
fcs.AddArea(ca);

c = new CellArea();
ca.StartRow = 1;
c.EndRow = 1;
c.StartColumn = 1;
c.EndColumn = 1;
fcs.AddArea(c);
```
- **غاية**: `CellArea` تحدد الكائنات المكان الذي سيتم فيه تطبيق التنسيق الشرطي.

#### الخطوة 3: إضافة الشروط
```csharp
// تحديد شروط قاعدة التنسيق.
int conditionIndex = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "=A2", "100");
```
- **غاية**: `AddCondition()` يضيف قاعدة جديدة استنادًا إلى قيم الخلايا.

### تعيين لون الخلفية للتنسيق الشرطي
**ملخص:** قم بتخصيص مظهر الخلايا التي تلبي شروطًا معينة عن طريق تغيير لون خلفيتها.

#### الخطوة 1: تعيين لون الخلفية
```csharp
// تغيير لون الخلفية إلى اللون الأحمر إذا تم استيفاء الشرط.
FormatCondition fc = fcs[conditionIndex];
fc.Style.BackgroundColor = Color.Red;
```
- **غاية**: `Style.BackgroundColor` تعيين لون الخلفية للخلايا التي تفي بالقاعدة الشرطية.

### حفظ ملف Excel
**ملخص:** تعرف على كيفية حفظ المصنف الخاص بك بعد تطبيق كافة قواعد التنسيق.

#### الخطوة 1: حفظ المصنف
```csharp
// حدد دليل الإخراج واسم الملف.
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "output.xls");
```
- **غاية**: `Save()` يكتب المصنف إلى مسار محدد باسم ملف معين.

   - تأكد من استيراد كافة المساحات الأساسية بشكل صحيح ومن تثبيت المكتبة بشكل صحيح في مشروعك.
