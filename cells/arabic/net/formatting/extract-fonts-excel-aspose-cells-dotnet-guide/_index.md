## دليل التنفيذ

في هذا القسم، سنقوم بتفصيل عملية استخراج بيانات الخط من مصنفات Excel.

### تحميل المصنف
أولاً، تأكد من إمكانية الوصول إلى ملف مصنفك. قد يكون هذا مصنفًا جديدًا أو مصنفًا موجودًا مُحمّلًا من القرص.

#### الخطوة 1: إعداد دليل البيانات
```csharp
string dataDir = "path_to_your_directory";

// قم بتحميل مصنف المصدر.
Workbook wb = new Workbook(dataDir + "sampleGetFonts.xlsx");
```

### استخراج الخطوط
الآن، دعونا نركز على استخراج كافة الخطوط المستخدمة داخل المصنف.

#### الخطوة 2: الحصول على جميع الخطوط الموجودة في المصنف
```csharp
// استرداد مجموعة من كائنات الخط من المصنف.
Aspose.Cells.Font[] fonts = wb.GetFonts();

// قم بتكرار كل خط وطباعة تفاصيله.
foreach (var font in fonts)
{
    Console.WriteLine($"Font Name: {font.Name}, Style: {font.Style}");
}
```

- **كتاب العمل**: يُمثل ملف Excel. تحميل مصنف هو الخطوة الأولى للوصول إلى خصائص أي مستند.
