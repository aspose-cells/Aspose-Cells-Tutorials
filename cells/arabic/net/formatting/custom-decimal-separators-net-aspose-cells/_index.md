## دليل التنفيذ

### تخصيص الفواصل العشرية والمجموعية

**ملخص:**
يؤدي تخصيص الفواصل العشرية والمجموعية إلى تحسين قابلية قراءة البيانات وتلبية معايير التنسيق المحددة التي تتطلبها مناطق أو شركات مختلفة.

#### الخطوة 1: تكوين الإعدادات
ابدأ بتحديد تنسيقات الأرقام المطلوبة للمصنف بأكمله:
```csharp
// تحديد الفواصل العشرية المخصصة ومجموعات الفواصل
workbook.Settings.NumberDecimalSeparator = '.';
workbook.Settings.NumberGroupSeparator = ' ';
```
**توضيح:** ال `NumberDecimalSeparator` يتم ضبطه على النقطة (.) كما هو مستخدم بشكل شائع في العديد من المناطق. `NumberGroupSeparator` يتم تكوينه كمساحة (' ')، والتي يمكن تعديلها بناءً على التفضيلات الإقليمية.

#### الخطوة 2: تطبيق الأنماط المخصصة
بمجرد تحديد الفواصل، قم بتطبيق نمط مخصص على خلاياك:
```csharp
Worksheet worksheet = workbook.Worksheets[0];

// تعيين قيمة الخلية وتطبيق النمط
Cell cell = worksheet.Cells["A1"];
cell.PutValue(123456.789);

Style style = cell.GetStyle();
style.Custom = "#,##0.000;[Red]#,##0.000"; // سلسلة تنسيق مخصصة
cell.SetStyle(style);
```
**توضيح:** التنسيق المخصص `#,##0.000` يضمن ثلاثة أماكن عشرية ويجمع الأرقام باستخدام الفواصل المحددة.

#### الخطوة 3: ملاءمة الأعمدة تلقائيًا
لضمان عرض بياناتك بشكل جيد، قم بضبط الأعمدة تلقائيًا:
```csharp
worksheet.AutoFitColumns();
```
تعمل هذه الطريقة على ضبط عرض الأعمدة لتناسب محتواها تلقائيًا.

#### الخطوة 4: الحفظ بتنسيق PDF
وأخيرًا، احفظ المصنف بتنسيق PDF باستخدام الإعدادات المخصصة الخاصة بك:
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY/CustomSeparator_out.pdf");
```
