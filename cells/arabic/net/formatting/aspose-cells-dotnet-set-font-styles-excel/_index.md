## دليل التنفيذ

### إنشاء مصنف وورقة عمل

ابدأ بإنشاء مصنف جديد وإضافة ورقة عمل:

```csharp
// إنشاء كائن مصنف جديد.
Workbook workbook = new Workbook();

// إضافة ورقة عمل جديدة.
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

### الوصول إلى أنماط الخلايا وتعديلها

جوهر هذا البرنامج التعليمي هو تعديل نمط الخط. إليك الطريقة:

#### ضبط وزن الخط إلى عريض

لجعل النص غامقًا، قم بالوصول إلى كائن نمط الخلية المطلوبة:

```csharp
// الوصول إلى الخلية "A1".
Aspose.Cells.Cell cell = worksheet.Cells["A1"];

// إضافة قيمة إلى الخلية.
cell.PutValue("Hello Aspose!");

// احصل على كائن النمط المرتبط بالخلية.
Style style = cell.GetStyle();

// ضبط وزن الخط إلى غامق.
style.Font.IsBold = true;

// قم بتطبيق النمط مرة أخرى على الخلية.
cell.SetStyle(style);
```

#### شرح الكود
- **الحصول على النمط ()**:استرجاع إعدادات النمط الحالية للخلية.
- **الخط عريض**:خاصية تتحكم في جرأة النص. ضبطها على `true` يتم تطبيق التنسيق الغامق.

### حفظ ملف Excel

وأخيرًا، احفظ المصنف الخاص بك للحفاظ على التغييرات:

```csharp
string outputPath = "Path_to_output_directory\\styledWorkbook.xls";
workbook.Save(outputPath, SaveFormat.Excel97To2003);
```
