## دليل التنفيذ

ينقسم هذا القسم إلى ميزتين رئيسيتين: إنشاء الخلايا وتنسيقها، وإدارة المصنفات وأوراق العمل.

### إنشاء خلية Excel وتنسيقها

#### ملخص

تعرف على كيفية إنشاء خلية في مصنف Excel الخاص بك، وإدراج القيم، وتطبيق تنسيقات الأرقام لتحسين إمكانية القراءة، واسترداد بيانات الخلايا المنسقة وغير المنسقة.

**الخطوة 1: إنشاء مصنف وورقة عمل Access**

إنشاء جديد `Workbook` الكائن والوصول إلى ورقة العمل الأولى:

```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**الخطوة 2: إدراج القيمة في الخلية**

قم بالوصول إلى الخلية A1 وأدخل قيمة عددية:

```csharp
Cell cell = worksheet.Cells["A1"];
cell.PutValue(0.012345);
```

**الخطوة 3: تطبيق تنسيق الأرقام**

تنسيق الخلية لعرض منزلتين عشريتين فقط باستخدام `Style`:

```csharp
Style style = cell.GetStyle();
style.Number = 2; // تنسيق '0.00'
cell.SetStyle(style);
```

**الخطوة 4: استرداد القيم المنسقة وغير المنسقة**

احصل على كلا الإصدارين من قيمة الخلية للمقارنة:

```csharp
string formattedValue = cell.GetStringValue(CellValueFormatStrategy.CellStyle);
string unformattedValue = cell.GetStringValue(CellValueFormatStrategy.None);
```

### إدارة المصنفات وأوراق العمل

#### ملخص

اكتشف كيفية إنشاء أوراق العمل والوصول إليها والتلاعب بها داخل مصنف Excel.

**الخطوة 1: إنشاء مصنف جديد**

تهيئة `Workbook` الكائن كما هو موضح سابقًا.

**الخطوة 2: الوصول إلى ورقة العمل حسب الفهرس**

يمكنك الوصول إلى ورقة العمل الأولى باستخدام فهرسها:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
Console.WriteLine("Accessed Worksheet: " + worksheet.Name);
```

**الخطوة 3: التعامل مع الخلايا في ورقة العمل**

إنشاء وتعيين قيم للخلايا الجديدة، مثل وضع "Hello World" في الخلية A2:

```csharp
cell = worksheet.Cells["A2"];
cell.PutValue("Hello World");
```
