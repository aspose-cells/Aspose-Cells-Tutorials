## تعليمات التثبيت  
لاستخدام Aspose.Cells لـ .NET في مشروعك، اتبع الخطوات التالية:  

1. التثبيت عبر NuGet  
   افتح وحدة التحكم Package Manager في Visual Studio وقم بتشغيل:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. تنزيل ملف DLL  
   بدلاً من ذلك، قم بتنزيل الإصدار الأحدث من [تنزيلات Aspose.Cells لـ .NET](https://releases.aspose.com/cells/net/) والإشارة إلى DLL في مشروعك.  

3. أضف مرجعًا إلى مشروعك  
   - في Visual Studio، انقر بزر الماوس الأيمن فوق مشروعك في مستكشف الحلول.  
   - انقر فوق إضافة مرجع > استعراض > حدد ملف DLL الذي تم تنزيله.  
   - انقر فوق موافق لإضافة المرجع.  

4. ابدأ باستخدام Aspose.Cells  
   أضف مساحة الأسماء التالية في بداية ملف C# الخاص بك:  
   ```csharp
   using Aspose.Cells;
   ```
