## دليل التنفيذ

في هذا القسم، سنتناول كيفية تطبيق التنسيق الشرطي باستخدام أنماط الخطوط المخصصة.

### إعداد التنسيق الشرطي

#### ملخص
يتيح لك التنسيق الشرطي التمييز بصريًا بين البيانات في جدول بيانات بناءً على معايير محددة. سنركز على تحسين الخطوط لشروط محددة.

#### التنفيذ خطوة بخطوة

1. **تهيئة المصنف وورقة العمل**
   
   ```csharp
   Workbook workbook = new Workbook();
   Worksheet sheet = workbook.Worksheets[0];
   ```

2. **إضافة قاعدة التنسيق الشرطي**

   أضف تنسيقًا شرطيًا فارغًا إلى ورقة العمل الخاصة بك:

   ```csharp
   int index = sheet.ConditionalFormattings.Add();
   FormatConditionCollection fcs = sheet.ConditionalFormattings[index];
   ```

3. **تحديد نطاق الهدف**

   حدد الخلايا التي يجب تنسيقها بشكل مشروط:

   ```csharp
   CellArea ca = new CellArea();
   ca.StartRow = 0;
   ca.EndRow = 9; // تعديل وفقًا لنطاق بياناتك
   ca.StartColumn = 0;
   ca.EndColumn = 4;
   fcs.AddArea(ca);
   ```

4. **تطبيق أنماط الخطوط المخصصة**

   قم بتكوين أنماط الخطوط مثل المائل، والغامق، والمشطوب، والمسطر:

   ```csharp
   FormatCondition fc = fcs[0];
   fc.Style.Font.IsItalic = true; // تعيين الخط إلى الخط المائل
   fc.Style.Font.IsBold = true;   // تعيين الخط إلى غامق
   fc.Style.Font.IsStrikeout = true; // يطبق تأثير الشطب
   fc.Style.Font.Underline = FontUnderlineType.Double; // ضع خطًا مزدوجًا أسفل النص
   fc.Style.Font.Color = Color.Black; // تعيين لون الخط إلى الأسود
   ```

5. **احفظ مصنفك**

   بعد تطبيق التنسيق، احفظ المصنف الخاص بك:

   ```csharp
   workbook.Save(outputDir + "output.xlsx");
   ```
