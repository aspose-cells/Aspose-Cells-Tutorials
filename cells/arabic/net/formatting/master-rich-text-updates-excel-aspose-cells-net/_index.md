## دليل التنفيذ

الآن، لنبدأ بتطبيق ميزة تحديث النص الغني. سنُقسّم هذا الدليل إلى أقسام منطقية لتسهيل فهمه.

### تحميل خلايا النص الغني والوصول إليها

#### ملخص
لتحديث خلية تحتوي على محتوى نص غني في ملف Excel، قم أولاً بتحميل المصنف الخاص بك والوصول إلى ورقة العمل والخلية المحددة التي تحتاج إلى التحديثات.
```csharp
// تحديد أدلة المصدر والإخراج
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// قم بتحميل المصنف الذي يحتوي على ملف Excel الخاص بك
Workbook workbook = new Workbook(sourceDir + "sampleUpdateRichTextCells.xlsx");

// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];

// احصل على الخلية A1 التي تحتوي على نص غني
Cell cell = worksheet.Cells["A1"];
```

#### توضيح
- **كتاب العمل:** يمثل ملف Excel بأكمله.
- **ورقة عمل:** ورقة واحدة ضمن المصنف الخاص بك، يمكن الوصول إليها عن طريق الفهرس أو الاسم.
- **خلية:** الخلية المحددة التي تريد إجراء التحديثات عليها.

### تحديث إعدادات الخط في خلايا النص الغني

#### ملخص
لتغيير إعدادات الخط لمحتوى النص الغني داخل خلية، قم باستردادها وتعديلها `FontSetting` أشياء.
```csharp
Console.WriteLine("Before updating the font settings....");

// الحصول على جميع الأحرف في الخلية كمجموعة من FontSettings
FontSetting[] fnts = cell.GetCharacters();

// قم بالمرور عبر كل إعداد خط لطباعة اسم الخط الحالي
for (int i = 0; i < fnts.Length; i++)
{
    Console.WriteLine(fnts[i].Font.Name);
}

// تحديث اسم الخط الأول في FontSetting
fnts[0].Font.Name = "Arial";

// تطبيق التغييرات مرة أخرى على الخلية
cell.SetCharacters(fnts);

Console.WriteLine();

Console.WriteLine("After updating the font settings....");

// استرداد إعدادات الخطوط المحدثة
fnts = cell.GetCharacters();

// اطبع أسماء الخطوط الجديدة
for (int i = 0; i < fnts.Length; i++)
{
    Console.WriteLine(fnts[i].Font.Name);
}
```

#### توضيح
- **الحصول على الأحرف ():** يسترجع مجموعة من `FontSetting` الكائنات التي تمثل أجزاء النص الغني داخل الخلية.
- **SetCharacters(إعداد الخط[]):** يتم تطبيق إعدادات الخط المعدلة على الخلية مرة أخرى.
- **نصيحة لاستكشاف الأخطاء وإصلاحها:** تأكد من تطبيق التغييرات باستخدام `SetCharacters()`؛ وإلا فلن تستمر التعديلات.
