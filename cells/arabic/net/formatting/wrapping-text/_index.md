## تعليمات التثبيت:  
لاستخدام Aspose.Cells لـ .NET، اتبع الخطوات التالية:  

1. التثبيت عبر NuGet:  
   - افتح Visual Studio.  
   - انتقل إلى الأدوات → مدير حزمة NuGet → إدارة حزم NuGet للحل.  
   - ابحث عن Aspose.Cells وانقر فوق تثبيت.  

2. التثبيت عبر وحدة تحكم إدارة الحزم:  
   - افتح الأدوات ← مدير الحزم NuGet ← وحدة تحكم مدير الحزم.  
   - قم بتشغيل الأمر:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. إضافة مرجع يدويًا:  
   - قم بتنزيل Aspose.Cells لـ .NET DLL من [إصدارات Aspose](https://releases.aspose.com/).  
   - قم بإضافة Aspose.Cells.dll الذي قمت بتنزيله إلى مراجع مشروعك.
