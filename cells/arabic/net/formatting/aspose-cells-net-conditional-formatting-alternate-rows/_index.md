## دليل التنفيذ

سنقوم بتقسيم العملية إلى خطوات قابلة للإدارة لمساعدتك على تطبيق التنسيق الشرطي بشكل فعال.

### تطبيق التنسيق الشرطي على الصفوف البديلة (H2)

تتيح لنا هذه الميزة التمييز بصريًا بين الصفوف، مما يُسهّل قراءة البيانات وتحليلها. لنستعرض كل خطوة:

#### الخطوة 1: إنشاء مثيل مصنف جديد

ابدأ بإنشاء مثيل جديد لـ `Workbook`. يمثل هذا ملف Excel الخاص بك:

```csharp
using System;
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// تهيئة مثيل مصنف جديد
Workbook book = new Workbook();
```

#### الخطوة 2: الوصول إلى ورقة العمل الأولى

انتقل إلى ورقة العمل الأولى في المصنف الخاص بك حيث ستطبق التنسيق:

```csharp
// احصل على ورقة العمل الأولى في المصنف
Worksheet sheet = book.Worksheets[0];
```

#### الخطوة 3: إضافة التنسيق الشرطي

تعريف أ `CellArea` وأضفه إلى `ConditionalFormattings` مجموعة. هذا يحدد مكان تطبيق التنسيق الشرطي:

```csharp
// قم بتحديد منطقة خلية تتراوح من A1 إلى I20
int idx = sheet.ConditionalFormattings.Add();
FormatConditionCollection conditionCollection = sheet.ConditionalFormattings[idx];
CellArea area = CellArea.CreateCellArea("A1", "I20");
conditionCollection.AddArea(area);
```

#### الخطوة 4: تعيين صيغة للتنسيق الشرطي

أضف شرط نوع التعبير واضبط الصيغة لتطبيق التظليل استنادًا إلى أرقام الصفوف:

```csharp
// أضف شرطًا باستخدام صيغة لتظليل الصف بالتناوب
idx = conditionCollection.AddCondition(FormatConditionType.Expression);
FormatCondition formatCondition = conditionCollection[idx];
formatCondition.Formula1 = @"=MOD(ROW(),2)=0";
```

#### الخطوة 5: تكوين النمط

تخصيص لون الخلفية ونمطها `Style` مرتبط بالتنسيق الشرطي الخاص بك:

```csharp
// تعيين نمط الصفوف المتناوبة
dateCondition.Style.BackgroundColor = Color.Blue;
dateCondition.Style.Pattern = BackgroundType.Solid;
```

#### الخطوة 6: احفظ مصنفك

وأخيرًا، احفظ المصنف على القرص بالتنسيق المطبق:

```csharp
// حفظ المصنف المنسق
book.Save(outputDir + "/output_out.xlsx");
```
