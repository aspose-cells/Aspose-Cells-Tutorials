## دليل التنفيذ

سنغطي الميزات الرئيسية لتخصيص تنسيقات الأرقام باستخدام Aspose.Cells.

### إضافة تنسيق تاريخ مخصص
**ملخص**:تعلم كيفية تنسيق التواريخ في خلايا Excel باستخدام نمط مخصص.
1. **إنشاء ورقة عمل أو الوصول إليها**
   ```csharp
   int sheetIndex = workbook.Worksheets.Add();
   Worksheet worksheet = workbook.Worksheets[sheetIndex];
   ```
2. **تعيين تاريخ النظام الحالي باستخدام تنسيق مخصص**
   أضف التاريخ الحالي إلى الخلية "A1" وقم بتطبيق تنسيق عرض مخصص.
   ```csharp
   // أدخل تاريخ النظام الحالي في A1
   worksheet.Cells["A1"].PutValue(DateTime.Now);

   // استرداد كائن النمط للتخصيص
   Style style = worksheet.Cells["A1"].GetStyle();

   // تعيين تنسيق الرقم المخصص إلى "d-mmm-yy"
   style.Custom = "d-mmm-yy";

   // تطبيق النمط المخصص مرة أخرى على الخلية A1
   worksheet.Cells["A1"].SetStyle(style);
   ```

### تنسيق القيم الرقمية كنسبة مئوية
**ملخص**:عرض القيم الرقمية بتنسيق النسبة المئوية.
1. **إدراج وتنسيق القيمة**
   ```csharp
   // إضافة قيمة عددية إلى الخلية A2
   worksheet.Cells["A2"].PutValue(20);

   // جلب النمط للتنسيق
   Style style = worksheet.Cells["A2"].GetStyle();

   // تطبيق تنسيق الأرقام المخصص كنسبة مئوية
   style.Custom = "0.0%";

   // تعيين النمط المنسق مرة أخرى إلى الخلية A2
   worksheet.Cells["A2"].SetStyle(style);
   ```

### تطبيق تنسيق العملة
**ملخص**:إظهار الأرقام بتنسيق العملة، مع تنسيق محدد للقيم السلبية.
1. **إدراج وتنسيق قيمة العملة**
   ```csharp
   // إضافة قيمة إلى الخلية A3
   worksheet.Cells["A3"].PutValue(2546);

   // الوصول إلى كائن النمط
   Style style = worksheet.Cells["A3"].GetStyle();

   // تعيين تنسيق العملة المخصص
   style.Custom = "\u00a3#,##0;[Red]$-#,##0";

   // تطبيق على الخلية A3
   worksheet.Cells["A3"].SetStyle(style);
   ```
