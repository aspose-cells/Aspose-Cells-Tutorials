## استيراد الحزم

بعد أن انتهينا من الإعداد، لنستورد الحزم اللازمة. هذه الخطوة بالغة الأهمية لأنها تتيح لنا الوصول إلى فئات وأساليب Aspose.Cells اللازمة للتعامل مع ملفات Excel.

```csharp
using System;
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Charts;
```

هذه العبارات تُعلم المُجمِّع بأننا سنستخدم فئات مكتبة Aspose.Cells. تأكد من إدراجها في بداية الكود لتجنب أي مشاكل لاحقًا!

الآن، لنُقسّم العملية إلى خطوات سهلة. سنتّبعها خطوةً بخطوة، لضمان وضوح كل شيء.

## الخطوة 1: تحديد أدلة المصدر والإخراج

```csharp
string sourceDir = "Your Document Directory";
string outputDir = "Your Output Directory";
```

أولاً، علينا تحديد مكان ملف المصدر ومكان حفظ ملف الإخراج. استبدل "دليل مستنداتك" و"دليل الإخراج" بمسارات مجلداتك الفعلية. اعتبر هذه المجلدات بمثابة قاعدتك الرئيسية ومنصة الإطلاق التي تتواجد فيها ملفاتك.

## الخطوة 2: تحميل المصنف

```csharp
Workbook workbook = new Workbook(sourceDir + "sampleChangeChartSizeAndPosition.xlsx");
```

هنا، نقوم بإنشاء مثيل جديد لـ `Workbook` صفّ وحمّل ملف إكسل إليه. تخيّل المصنف كدفتر ملاحظات رقمي يحتوي على جميع أوراقك ومخططاتك. المعلمة التي نمررها هي المسار الكامل لملف إكسل، لذا تأكد من تضمين اسم الملف!

## الخطوة 3: الوصول إلى ورقة العمل

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

الآن بعد أن قمنا بتحميل المصنف الخاص بنا، نحتاج إلى الوصول إلى ورقة العمل المحددة التي نريد العمل بها، والتي في هذه الحالة هي ورقة العمل الأولى (الفهرس) `[0]`). كما هو الحال مع الانتقال إلى الصفحة الصحيحة في الكتاب، تساعدنا هذه الخطوة على التركيز على الورقة المطلوبة لتحريراتنا.

## الخطوة 4: تحميل الرسم البياني

```csharp
Chart chart = worksheet.Charts[0];
```

بعد استرجاع ورقة العمل، ننتقل مباشرةً إلى الرسم البياني! نلتقط الرسم البياني الأول (مرة أخرى، الفهرس) `[0]`هذا أشبه باختيار العمل الفني الذي ترغب بتحسينه. تأكد من وجود مخططك في ورقة العمل، وإلا ستُترك حائرًا!

## الخطوة 5: تغيير حجم الرسم البياني

```csharp
chart.ChartObject.Width = 400;
chart.ChartObject.Height = 300;
```

حان وقت تغيير أبعاد الرسم البياني! هنا، نضبط العرض على `400` بكسل والارتفاع إلى `300` تعديل الحجم أشبه باختيار الإطار المثالي لعملك الفني - سواء كان كبيرًا جدًا أو صغيرًا جدًا، فلن يتناسب مع الغرفة.

## الخطوة 6: إعادة وضع الرسم البياني

```csharp
chart.ChartObject.X = 250;
chart.ChartObject.Y = 150;
```

الآن بعد أن أصبح لدينا الحجم المناسب، لنحرك الرسم البياني! بتغيير `X` و `Y` الخصائص، نحن في الأساس نعيد وضع المخطط على ورقة العمل. تخيل الأمر كما لو أنك تسحب صورتك المؤطرة إلى مكان جديد على الحائط لإبراز جمالها بشكل أفضل!

## الخطوة 7: حفظ المصنف

```csharp
workbook.Save(outputDir + "outputChangeChartSizeAndPosition.xlsx");
```

أخيرًا، نحفظ تغييراتنا في ملف إكسل جديد. اختر اسمًا مناسبًا للملف المُصدَّر للحفاظ على تنظيمه. يشبه الأمر التقاط صورة لغرفتك المُرتَّبة بشكل جميل بعد نقل الأثاث، مع الحفاظ على التصميم الجديد!

## الخطوة 8: تأكيد النجاح

```csharp
Console.WriteLine("ChangeChartSizeAndPosition executed successfully.");
```

ولختام الموضوع، نُقدّم ملاحظات حول نجاح العملية. هذه ممارسة رائعة، تُتيح لك إنهاء مهمتك بوضوح وثقة، تمامًا كما تُعجب بعملك بعد إعادة ترتيب الأثاث!
