# إنشاء مخطط مخصص

## مقدمة

إنشاء مخططات بيانية مخصصة في Excel باستخدام مكتبة Aspose.Cells لـ .NET ليس سهلاً فحسب، بل هو طريقة رائعة لعرض بياناتك بفعالية. تستطيع المخططات البيانية تحويل البيانات العادية إلى قصص شيقة، مما يُسهّل على المحللين وصناع القرار استخلاص الرؤى. في هذا البرنامج التعليمي، نتعمق في كيفية إنشاء مخططات بيانية مخصصة داخل تطبيقاتك. لذا، إذا كنت ترغب في تحسين تقاريرك أو إضافة لمسة جمالية لعرض بياناتك، فأنت في المكان المناسب!

## المتطلبات الأساسية

قبل الخوض في تفاصيل إنشاء المخططات البيانية، دعونا نتأكد من تجهيز كل شيء. إليك ما تحتاجه:

1. Visual Studio أو أي IDE متوافق مع .NET: سيكون هذا هو الملعب الخاص بك لكتابة واختبار الكود الخاص بك.
2. مكتبة Aspose.Cells لـ .NET: تأكد من تثبيت هذه المكتبة. يمكنك تنزيلها. [هنا](https://releases.aspose.com/cells/net/).
3. الفهم الأساسي للغة C#: سيكون من المفيد لك فهم المفاهيم الأساسية للغة C#، حيث سنستخدمها في أمثلة التعليمات البرمجية الخاصة بنا.
4. مجموعة بيانات نموذجية: لإنشاء المخططات البيانية، من الضروري وجود بعض البيانات. سنستخدم مجموعة بيانات بسيطة في مثالنا، ولكن يمكنك تعديلها لتناسب احتياجاتك.

## استيراد الحزم

للبدء، ستحتاج إلى استيراد مساحة اسم Aspose.Cells اللازمة في تطبيق C#. إليك كيفية القيام بذلك:

```csharp
using System;
using System.IO;

using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
using Aspose.Cells.Charts;
```

الآن بعد أن تم وضع الهيكل الأساسي، دعنا ننتقل إلى الدليل خطوة بخطوة حول إنشاء مخطط مخصص.

## الخطوة 1: إعداد دليل الإخراج الخاص بك

أولاً، ستحتاج إلى إنشاء مجلد لحفظ ملف Excel. هذه الخطوة ضرورية لضمان معرفة تطبيقك بمكان حفظ منتجه النهائي.

```csharp
// دليل الإخراج
string outputDir = "Your Output Directory"; // قم بتغيير هذا إلى المسار المطلوب
```

بدلاً من "مجلد الإخراج"، يمكنك تحديد المسار الفعلي الذي تريد حفظ ملف Excel فيه. تأكد من وجود هذا المجلد على نظامك، وإلا ستواجه أخطاءً لاحقًا.

## الخطوة 2: إنشاء كائن مصنف

الآن، قد ترغب في بدء الأمور عن طريق إنشاء مثيل جديد لـ `Workbook` هذه هي اللبنة الأساسية لأي عمليات Excel باستخدام Aspose.Cells.

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

يقوم هذا السطر من التعليمات البرمجية بتهيئة مصنف جديد، وستكون جاهزًا لبدء إضافة البيانات والمخططات!

## الخطوة 3: الوصول إلى ورقة العمل

بعد ذلك، عليك الحصول على مرجع لورقة العمل التي ستُخزَّن فيها بياناتك. في هذه الحالة، سنعمل مع ورقة العمل الأولى في المصنف.

```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا
Worksheet worksheet = workbook.Worksheets[0];
```

يصل هذا السطر إلى ورقة العمل الأولى (الفهرس ٠). يتيح لك Aspose.Cells استخدام أوراق عمل متعددة، ما يتيح لك الاختيار وفقًا لذلك.

## الخطوة 4: إضافة بيانات العينة إلى ورقة العمل


بعد تجهيز ورقة العمل، حان الوقت لإضافة بعض البيانات النموذجية إلى خلاياك. ستساعدنا مجموعة بيانات بسيطة على تصوّر المخططات البيانية بشكل أكثر فعالية.

```csharp
// إضافة قيم العينة إلى الخلايا
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["A4"].PutValue(110);
worksheet.Cells["B1"].PutValue(260);
worksheet.Cells["B2"].PutValue(12);
worksheet.Cells["B3"].PutValue(50);
worksheet.Cells["B4"].PutValue(100);
```

هنا، نضع قيمًا في النطاقات من A1 إلى B4. لا تتردد في تعديل هذه القيم لاختبار سيناريوهات بيانات مختلفة.

## الخطوة 5: إضافة مخطط إلى ورقة العمل

الآن وصلنا إلى الجزء المثير: إضافة مخطط يُمثل البيانات التي أدخلناها بصريًا. يمكنك الاختيار من بين أنواع المخططات المتنوعة المتوفرة في Aspose.Cells.

```csharp
// إضافة مخطط إلى ورقة العمل
int chartIndex = worksheet.Charts.Add(ChartType.Column, 5, 0, 25, 10);
```

في هذا السطر، نضيف مخططًا عموديًا. يمكنك أيضًا استخدام أنواع أخرى من المخططات، مثل المخططات الخطية أو الدائرية أو الشريطية، حسب احتياجاتك.

## الخطوة 6: الوصول إلى مثيل الرسم البياني

بعد إضافة الرسم البياني، نحتاج إلى الرجوع إليه لتعديله بشكل أكبر. إليك الطريقة:

```csharp
// الوصول إلى مثيل الرسم البياني المضاف حديثًا
Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];
```

في هذه المرحلة، لديك `chart` كائن يسمح لك بتعديل خصائصه حسب الحاجة.

## الخطوة 7: إضافة سلسلة البيانات إلى الرسم البياني

الآن، عليك تحديد مصدر جلب بيانات المخطط. يتم ذلك بإضافة سلسلة بيانات إلى Aspose.Cells.

```csharp
// إضافة NSeries (مصدر بيانات الرسم البياني) إلى الرسم البياني
chart.NSeries.Add("A1:B4", true);
```

يربط هذا الخط بشكل فعال الرسم البياني الخاص بك بنقاط البيانات التي وضعتها في الخلايا، مما يسمح للرسم البياني بعرض هذه القيم.

## الخطوة 8: تخصيص نوع السلسلة

يمكنك تخصيص مخططك بشكل أكبر بتغيير نوع أي سلسلة. على سبيل المثال، لنُغيّر السلسلة الثانية إلى مخطط خطي لتحسين وضوح الصورة.

```csharp
// تعيين نوع الرسم البياني لسلسلة NSeries الثانية ليتم عرضه كرسم بياني خطي
chart.NSeries[1].Type = Aspose.Cells.Charts.ChartType.Line;
```

يتيح هذا إنشاء مخططات ذات أنواع مختلطة، مما يوفر فرص تصور فريدة.

## الخطوة 9: حفظ المصنف

بعد كل هذه الإعدادات، حان وقت حفظ ملف Excel. إليك كيفية القيام بذلك:

```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputHowToCreateCustomChart.xlsx");
```

تأكد من إضافة اسم الملف مع `.xlsx` تمديد للتأكد من حفظ المصنف بشكل صحيح.
