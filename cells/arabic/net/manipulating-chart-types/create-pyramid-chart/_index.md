# إنشاء مخطط هرمي

## استيراد الحزم

قبل البدء بالبرمجة، لنستورد مساحات الأسماء اللازمة. هذه الخطوة أساسية لأنها تتيح لنا استخدام الفئات والأساليب التي توفرها مكتبة Aspose.Cells.

```csharp
using System;
using System.IO;

using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
using Aspose.Cells.Charts;
```

تغطي هذه المساحات الأسماء الوظائف الأساسية التي سنستخدمها في هذا البرنامج التعليمي، مثل إنشاء مصنفات العمل، ومعالجة أوراق العمل، وإضافة المخططات البيانية.

حسنًا، لنُقسّم عملية إنشاء مخطط الهرم إلى خطوات بسيطة. بنهاية هذا الدليل، سيكون لديك مثال عملي شامل.

## الخطوة 1: تحديد دليل الإخراج

أولاً، علينا تحديد مكان حفظ ملف الإخراج (ملف إكسل الذي يحتوي على مخطط الهرم). يشبه الأمر اختيار مساحة عمل قبل بدء مشروع.

```csharp
// دليل الإخراج
string outputDir = "Your Output Directory";
```

تأكد من استبدال `"Your Output Directory"` بمسار صحيح على جهاز الكمبيوتر الخاص بك. هذا المسار هو المكان الذي سيتم فيه حفظ ملف Excel الذي تم إنشاؤه.

## الخطوة 2: إنشاء كائن مصنف

الآن، لنُنشئ نسخة جديدة من مصنف. تخيّل المصنف كلوحة فارغة يمكنك رسم بياناتك عليها.

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

يقوم هذا السطر بإنشاء مصنف جديد جاهز لإدخال البيانات وتصورها.

## الخطوة 3: الحصول على مرجع لورقة العمل

يحتوي كل مصنف على ورقة عمل واحدة على الأقل. سنشير هنا إلى ورقة العمل الأولى التي سنعمل عليها.

```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[0];
```

بالرجوع إليه `Worksheets[0]`، نحن نتفاعل بشكل مباشر مع الورقة الأولى، حيث سنضيف بياناتنا والمخطط البياني.

## الخطوة 4: إضافة بيانات العينة إلى الخلايا

لإنشاء أي مخطط بياني، ستحتاج إلى بعض البيانات. لنملأ بعض القيم النموذجية في ورقة العمل.

```csharp
// إضافة قيم العينة إلى الخلايا
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```

هنا، نقوم بإدراج القيم في الخلايا من A1 إلى A3 (التسميات أو مستويات الهرم) ومن B1 إلى B3 (القيم المقابلة لتلك المستويات).

## الخطوة 5: إضافة مخطط هرمي إلى ورقة العمل

الآن، لنُضِف مخططنا الهرمي. هنا تبدأ المفاجأة!

```csharp
// إضافة مخطط إلى ورقة العمل
int chartIndex = worksheet.Charts.Add(Aspose.Cells.Charts.ChartType.Pyramid, 5, 0, 25, 10);
```

في هذا السطر، نحدد نوع الرسم البياني على النحو التالي `Pyramid` وحدد موضعها في ورقة العمل باستخدام فهارس الصفوف والأعمدة. هذا أشبه بتأطير صورة على الحائط - عليك اختيار المكان الأنسب لظهورها!

## الخطوة 6: الوصول إلى الرسم البياني المضاف حديثًا

بعد إضافة الرسم البياني، نحتاج إلى الوصول إليه لإعداده.

```csharp
// الوصول إلى مثيل الرسم البياني المضاف حديثًا
Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];
```

يضمن هذا الخط أننا نعمل مع نموذج الرسم البياني الصحيح الذي أنشأناه للتو.

## الخطوة 7: إضافة سلسلة البيانات إلى الرسم البياني

لكي يعرض الرسم البياني البيانات، نحتاج إلى تعيين مصدر البيانات الخاص به استنادًا إلى الخلايا التي ملأناها مسبقًا.

```csharp
// إضافة SeriesCollection (مصدر بيانات الرسم البياني) إلى الرسم البياني الذي يتراوح من الخلية "A1" إلى "B3"
chart.NSeries.Add("A1:B3", true);
```

في هذا الجزء، نقوم بربط البيانات الموجودة في الخلايا A1 إلى B3، مما يسمح لمخطط الهرم الخاص بنا بتصور هذه المعلومات.

## الخطوة 8: حفظ ملف Excel

أخيرًا، حان وقت حفظ تحفتنا الفنية. لنكتب مصنف Excel إلى ملف.

```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputHowToCreatePyramidChart.xlsx");
```

سيؤدي هذا الإجراء إلى إنشاء ملف Excel باسم `outputHowToCreatePyramidChart.xlsx` في دليل الإخراج المحدد الخاص بك.

## الخطوة 9: تأكيد وحدة التحكم

وأخيرًا وليس آخرًا، دعنا نضيف بعض التعليقات في وحدة التحكم لتأكيد تنفيذ كل شيء بسلاسة.

```csharp
Console.WriteLine("HowToCreatePyramidChart executed successfully.");
```

سيعلمك هذا السطر أن مهمة إنشاء مخطط الهرم الخاص بك قد اكتملت دون أي عوائق.
