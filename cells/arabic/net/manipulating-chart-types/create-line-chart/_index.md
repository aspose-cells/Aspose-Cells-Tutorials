# إنشاء مخطط خطي

## مقدمة

هل أنت مستعد لعرض بياناتك بوضوح مذهل؟ تُعد المخططات الخطية طريقة رائعة لعرض الاتجاهات بمرور الوقت أو العلاقة بين متغيرين. سواء كنت تدير بيانات لمشروع تجاري أو تُحلل مقاييس شخصية، فإن إمكانية إنشاء مخططات خطية برمجيًا توفر لك الوقت وتتيح لك مرونة أكبر. في هذا الدليل، سنشرح لك كل خطوة من خطوات إنشاء مخطط خطي باستخدام Aspose.Cells لـ .NET. هل أنت مستعد للبدء؟ هيا بنا!

## المتطلبات الأساسية

قبل أن ننتقل إلى التفاصيل الدقيقة لإنشاء مخطط خطي، دعنا نتأكد من أنك مجهز للمتابعة:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك لأنه أحد بيئات التطوير المتكاملة الأكثر شيوعًا لتطوير .NET.
2. مكتبة Aspose.Cells لـ .NET: ستحتاج إلى مكتبة Aspose.Cells، والتي يمكنك تنزيلها من [هنا](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة البرمجة C#: ستساعدك المعرفة بلغة البرمجة C# على فهم الأمثلة ومقاطع التعليمات البرمجية بشكل أفضل.
4. .NET Framework أو .NET Core: إعداد أساسي لأي إطار عمل حيث سيكون هذا هو الأساس لتطبيقاتنا.

بمجرد الانتهاء من هذه المتطلبات الأساسية، ستكون جاهزًا لإنشاء بعض المخططات البيانية!

## استيراد الحزم

بعد أن أعددنا بيئتنا، نحتاج إلى استيراد الحزم اللازمة في شيفرة C#. وكما هو الحال عند جمع أدواتك قبل بدء أي مشروع، يُعد استيراد الحزم أمرًا أساسيًا لضمان توفر كل ما تحتاجه.

إليك كيفية القيام بذلك:

```csharp
using System;
using System.IO;

using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
using Aspose.Cells.Charts;
```

هذا الخط يستورد `Aspose.Cells` مساحة الاسم، التي تحتوي على جميع الفئات والطرق التي سنستخدمها لإنشاء مخطط الخط الخاص بنا.

الآن، لنُقسّم العملية بأكملها إلى خطوات بسيطة وسهلة الفهم. ستُرشدك كل خطوة خلال التسلسل المنطقي لإنشاء مخطط خطي باستخدام Aspose.Cells لـ .NET.

## الخطوة 1: إعداد دليل الإخراج

الخطوة الأولى هي تحديد مكان حفظ ملف الإخراج. الأمر أشبه بإعداد مساحة العمل قبل البدء بالعمل. 

```csharp
// دليل الإخراج
string outputDir = "Your Output Directory";
```
يستبدل `"Your Output Directory"` مع المسار الفعلي الذي تريد حفظ ملف Excel الناتج فيه.

## الخطوة 2: إنشاء كائن المصنف

بعد ذلك، علينا إنشاء نسخة جديدة من مصنف العمل. اعتبر مصنف العمل بمثابة لوحة فنية تتدفق فيها إبداعاتك. 

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
يقوم هذا السطر بإنشاء مصنف جديد يحتوي على كافة بياناتك ومرئياتك.

## الخطوة 3: الوصول إلى ورقة العمل

في مصنفنا الجديد، نحتاج إلى مرجع لورقة العمل التي سنُدخل بياناتنا فيها. إذا كان مصنف العمل هو لوحتنا، فإن ورقة العمل هي لوحة الألوان.

```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[0];
```
هنا نصل إلى ورقة العمل الأولى (الفهرس) `0`).

## الخطوة 4: إضافة قيم العينة إلى الخلايا

الآن يأتي الجزء الممتع! سنُدخل بعض القيم النموذجية في ورقة العمل. ستكون هذه البيانات أساسًا لمخططنا الخطي. 

```csharp
// إضافة قيم العينة إلى الخلايا
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```
في هذه المقتطفات، نضيف قيمًا إلى الخلايا في العمودين A وB. يمثل العمود A قيم المحور X، بينما يمثل العمود B قيم المحور Y.

## الخطوة 5: إضافة مخطط خطي إلى ورقة العمل

بعد ذلك، سنضيف مخططنا الخطي إلى ورقة العمل. هنا ستنبض بياناتك بالحياة!

```csharp
// إضافة مخطط إلى ورقة العمل
int chartIndex = worksheet.Charts.Add(Aspose.Cells.Charts.ChartType.Line, 5, 0, 25, 10);
```
هنا، نضيف مخططًا خطيًا في الموقع المحدد. تُحدد المعلمات (5، 0، 25، 10) موضع المخطط وحجمه داخل ورقة العمل.

## الخطوة 6: الوصول إلى مثيل الرسم البياني الجديد

بمجرد إضافة الرسم البياني الخاص بنا، حان الوقت للحصول على كائن الرسم البياني الذي تم إنشاؤه حديثًا. 

```csharp
// الوصول إلى مثيل الرسم البياني المضاف حديثًا
Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];
```
يربطنا هذا الكود بالرسم البياني حتى نتمكن من التحكم فيه بشكل أكبر.

## الخطوة 7: إضافة SeriesCollection إلى الرسم البياني

الآن، علينا تحديد البيانات التي سيعرضها مخططنا البياني. هنا، نُحدد مصدر بيانات مخططنا البياني الخطي بإضافة مجموعة سلاسل.

```csharp
// إضافة SeriesCollection (مصدر بيانات الرسم البياني) إلى الرسم البياني الذي يتراوح من الخلية "A1" إلى "B3"
chart.NSeries.Add("A1:B3", true);
```
في هذا المثال، نطلب من الرسم البياني استخدام القيم الموجودة في الخلايا من A1 إلى B3.

## الخطوة 8: حفظ ملف Excel

النهاية الرائعة! بعد كل هذا الجهد، حان وقت حفظ ملف Excel ورؤية مخططك الخطي قيد التنفيذ.

```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputHowToCreateLineChart.xlsx");
```
يحفظ هذا السطر المصنف الخاص بك في دليل الإخراج المحدد باسم `outputHowToCreateLineChart.xlsx`.

## الخطوة 9: التنفيذ والتحقق

أخيرًا، يمكنك الآن تشغيل الكود الخاص بك والتأكد من إنشاء مخطط الخط بنجاح في دليل الإخراج الخاص بك! 

```csharp
Console.WriteLine("HowToCreateLineChart executed successfully.");
```
سيؤدي هذا إلى إخراج رسالة في وحدة التحكم الخاصة بك، لإعلامك بأن كل شيء يسير بسلاسة.
