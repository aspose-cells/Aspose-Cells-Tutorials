## دليل التنفيذ

### تهيئة المصنف والوصول إلى ورقة العمل

**ملخص:**  
البدء بتهيئة ملف جديد `Workbook` يُعدّ تحديد الكائن أمرًا بالغ الأهمية. يمكنك أيضًا تحميل ملفات Excel الموجودة بسهولة. بعد ذلك، يمكنك الوصول إلى أوراق عمل محددة داخل مصنفك.

#### تهيئة المصنف
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string dataDir = SourceDir + "/sample.xlsx";
Workbook workbook = new Workbook(dataDir);
```

**توضيح:**  
- **دليل المصدر**:استبدله بمسار الدليل الفعلي الخاص بك.
- **دليل البيانات**:المسار إلى ملف Excel الخاص بك.

#### الوصول إلى ورقة العمل
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
- `Worksheets[0]` يسترجع أول ورقة عمل في المصنف. غيّر فهرس الأوراق الأخرى.

### تجميع الصفوف

**ملخص:**  
قم بتجميع الصفوف في ورقة Excel لتنظيم البيانات بشكل هرمي.

#### تنفيذ تجميع الصفوف
```csharp
worksheet.Cells.GroupRows(0, 5, true);
```

**توضيح:**
- **ابدأ الصف**:مؤشر الصف الابتدائي (0).
- **العدد الإجمالي**:عدد الصفوف المتتالية للتجميع (6 في هذه الحالة).
- **مستوى المخطط التفصيلي**: تعيين `true` لإظهار مستوى المخطط التفصيلي.

### تجميع الأعمدة

**ملخص:**  
وبالمثل، يمكن أن يساعد تجميع الأعمدة في تلخيص البيانات وإدارتها بكفاءة.

#### تنفيذ تجميع الأعمدة
```csharp
worksheet.Cells.GroupColumns(0, 2, true);
```

**توضيح:**
- **عمود البداية**:مؤشر العمود الابتدائي (0).
- **العدد الإجمالي**:عدد الأعمدة المتتالية التي سيتم تجميعها (3 في هذه الحالة).
- **مستوى المخطط التفصيلي**: تعيين `true` لعرض مستوى المخطط التفصيلي.

### إعداد عمود الملخص

**ملخص:**  
قم بإضافة معلومات الملخص بسهولة عن طريق تعيين عمود الملخص على الجانب الأيمن من البيانات المجمعة.

#### تنفيذ عمود الملخص
```csharp
worksheet.Outline.ملخصالعمود الأيمن = true;
```

- **SummaryColumnRight**: تم الضبط على `true` لعرض عمود الملخص على الجانب الأيمن للمجموعة.

### حفظ المصنف

**ملخص:**  
بعد إجراء التعديلات، قم بحفظ المصنف الخاص بك بكفاءة باستخدام Aspose.Cells.

#### تنفيذ حفظ المصنف
```csharp
string دليل الإخراج = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/output.xls");
```

- **outputDir**:قم بتحديد المكان الذي تريد حفظ الملف المعدل فيه.
- تأكد من وجود الدليل قبل الحفظ.
