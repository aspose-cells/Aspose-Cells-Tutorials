## دليل التنفيذ

### الميزة 1: تحميل ملف القالب
#### ملخص
يعد تحميل ملف Excel الخطوة الأولى قبل التعامل مع جداول البيانات المحورية باستخدام Aspose.Cells.

```csharp
// قم بتحديد دليل المصدر الخاص بك حيث يقع "samplePivotTable.xlsx".
string SourceDir = @"YOUR_SOURCE_DIRECTORY";

// تهيئة كائن المصنف وتحميل ملف Excel الموجود.
Workbook wb = new Workbook(SourceDir + "samplePivotTable.xlsx");
```

### الميزة 2: الوصول إلى جدول البيانات المحوري وتعيين صفحة تصفية التقارير
#### ملخص
يمكنك الوصول إلى جداول محورية محددة داخل المصنف الخاص بك لتعيين صفحة تصفية التقارير لتحسين تصفية البيانات.

```csharp
// احصل على الجدول المحوري الأول في ورقة العمل.
PivotTable pt = wb.Worksheets[1].PivotTables[0];

// قم بتعيين حقل المحور لإظهار صفحة تصفية التقرير.
pt.ShowReportFilterPage(pt.PageFields[0]);
```

### الميزة 3: عرض صفحة تصفية التقارير حسب الفهرس والاسم
#### ملخص
تتيح لك هذه الميزة إعداد صفحة تصفية التقرير باستخدام كل من الفهرس والاسم، مما يوفر لك المرونة في إدارة تكوينات جدول المحور الخاص بك.

```csharp
// تعيين مؤشر الموضع لإظهار صفحات مرشح التقرير.
pt.ShowReportFilterPageByIndex(pt.PageFields[0].Position);

// بدلاً من ذلك، استخدم اسم حقل الصفحة لتكوين مرشحات التقرير.
pt.ShowReportFilterPageByName(pt.PageFields[0].Name);
```

### الميزة 4: حفظ ملف الإخراج
#### ملخص
بعد إجراء التغييرات، احفظ مصنفك. يساعدك هذا الدليل على حفظ ملف Excel المعدّل بكفاءة.

```csharp
// قم بتحديد دليل الإخراج الخاص بالملف المحفوظ.
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// حفظ التعديلات في ملف Excel جديد.
wb.Save(outputDir + "outputSamplePivotTable.xlsx");
```
