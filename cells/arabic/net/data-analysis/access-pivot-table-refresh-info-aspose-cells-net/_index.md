## دليل التنفيذ

### الوصول إلى معلومات تحديث جدول المحور

#### ملخص

تتيح لك هذه الميزة استرداد من قام بتحديث جدول محوري آخر مرة ومتى تم التحديث بشكل برمجي، مما يوفر رؤى قيمة حول سلامة بياناتك.

#### إعداد مشروعك
1. **تحميل المصنف:**
   قم بتحميل مصنف Excel الذي يحتوي على جدول المحور المستهدف باستخدام `Workbook` فصل.
   ```csharp
   Workbook workbook = new Workbook("sourcePivotTable.xlsx");
   ```
2. **الوصول إلى ورقة العمل والجدول المحوري:**
   قم بالوصول إلى ورقة العمل ثم إلى الجدول المحوري المحدد داخلها.
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   PivotTable pivotTable = worksheet.PivotTables[0];
   ```
3. **استرداد معلومات التحديث:**
   يستخدم `RefreshedByWho` و `RefreshDate` للحصول على معلومات التحديث التفصيلية.
   ```csharp
   string refreshByWho = pivotTable.RefreshedByWho;
   DateTime refreshDate = pivotTable.RefreshDate;
   
   Console.WriteLine("Pivot table refreshed by: " + refreshByWho);
   Console.WriteLine("Last refresh date: " + refreshDate);
   ```

#### توضيح
- **`RefreshedByWho`:** إرجاع اسم المستخدم للشخص الذي قام بتحديث جدول المحور آخر مرة.
- **`RefreshDate`:** يوفر الطابع الزمني لوقت آخر تحديث لجدول المحور.
