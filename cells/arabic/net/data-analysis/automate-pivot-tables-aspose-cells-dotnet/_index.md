# أتمتة جداول البيانات المحورية في Excel باستخدام Aspose.Cells لـ .NET

## مقدمة
هل ترغب في تبسيط أتمتة تحميل وتعديل جداول البيانات المحورية في مصنفات Excel باستخدام C#؟ مع مكتبة Aspose.Cells، تُصبح إدارة ملفات Excel سلسة، مما يُمكّن المطورين من معالجة البيانات بكفاءة. سيرشدك هذا الدليل الشامل خلال عملية تحميل مصنف موجود، والوصول إلى جدول بيانات محوري، وتكوين حقوله، وحفظ التغييرات - كل ذلك باستخدام Aspose.Cells لـ .NET.

**ما سوف تتعلمه:**
- كيفية تحميل مصنف Excel من دليل
- الوصول إلى جداول البيانات المحورية وتعديلها في المصنف
- تكوين تنسيقات عرض البيانات داخل جداول البيانات المحورية
- حفظ التغييرات مرة أخرى في ملف Excel جديد

دعنا نتعمق في إعداد بيئتك حتى تتمكن من البدء في تنفيذ هذه الميزات القوية.

## دليل التنفيذ

### تحميل مصنف Excel
**ملخص:**
تتيح لك هذه الميزة تحميل مصنف Excel موجود من نظام الملفات لديك إلى بيئة Aspose.Cells. إليك كيفية القيام بذلك:

#### الخطوة 1: إعداد مسارات الدليل
أولاً، قم بتحديد الدليل المصدر والدليل الناتج الذي سيتم قراءة ملفاتك منه وحفظها.
```csharp
string SourceDir = @"C:\\Your\\Source\\Directory";
string outputDir = @"C:\\Your\\Output\\Directory";
```

#### الخطوة 2: تحميل المصنف
تحميل ملف Excel إلى `Workbook` الكائن. هذه الخطوة تقوم بتهيئة مثيل المصنف بالملف المحدد.
```csharp
Workbook workbook = new Workbook(SourceDir + "Book1.xls");
```

### الوصول إلى حقول البيانات وتكوينها في جدول محوري
**ملخص:**
بمجرد تحميل المصنف، يمكنك الوصول إلى ورقة العمل الأولى والجدول المحوري المطلوب لتعديل إعدادات عرض البيانات الخاصة به.

#### الخطوة 3: احصل على ورقة العمل الأولى
استرجاع ورقة العمل الأولى من المصنف.
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

#### الخطوة 4: الوصول إلى جدول المحور
الوصول إلى جدول PivotTable المحدد ضمن ورقة العمل. هنا، نستخدم الفهرس `pivotIndex` لتحديد جدول البيانات المحوري الذي تريد تعديله.
```csharp
int pivotIndex = 0;
PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
```

#### الخطوة 5: تعديل تنسيق عرض البيانات
قم بضبط كيفية عرض البيانات في حقول بيانات الجدول المحوري. هنا، قمنا بضبطها لعرضها كنسبة مئوية من حقل أساسي محدد.
```csharp
PivotFieldCollection pivotFields = pivotTable.DataFields;
PivotField pivotField = pivotFields[0];
pivotField.DataDisplayFormat = PivotFieldDataDisplayFormat.PercentageOf;
pivotField.BaseFieldIndex = 1;
pivotField.BaseItemPosition = PivotItemPosition.Next;
pivotField.Number = 10; // تعيين تنسيق الرقم
```

### حفظ ملف Excel
**ملخص:**
بعد إجراء التعديلات، قد ترغب في حفظ المصنف الخاص بك كملف جديد.

#### الخطوة 6: حفظ المصنف
احفظ المصنف المحدث في دليل الإخراج المخصص لك.
```csharp
workbook.Save(outputDir + "output.xls");
```
