## إعداد Aspose.Cells لـ .NET
لنُهيئ الأرضية لمهمة إدارة السيناريوهات. ابدأ بتحميل مصنف Excel الحالي والوصول إلى أوراق العمل الخاصة به.

### تحميل المصنف والوصول إلى ورقة العمل
ابدأ بإنشاء مثيل جديد لـ `Workbook` الفئة وتحميل الملف المطلوب:

```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// تحميل مصنف موجود
Workbook workbook = new Workbook(sourceDir + "sampleCreateManipulateRemoveScenarios.xlsx");
Worksheet worksheet = workbook.Worksheets[0];
```

### إنشاء سيناريو
بعد تحميل المصنف، يمكنك الآن إضافة السيناريوهات وإدارتها:

```csharp
// إضافة سيناريو جديد
int i = worksheet.Scenarios.Add("MyScenario");

// الوصول إلى السيناريو الذي تم إنشاؤه حديثًا
Scenario scenario = worksheet.Scenarios[i];

// إضافة تعليق إلى السيناريو الخاص بنا من أجل فهم أفضل
scenario.Comment = "Test scenario is created.";
```

### إدارة خلايا إدخال السيناريو
لتجعل السيناريوهات الخاصة بك ديناميكية، تحتاج إلى إدارة خلايا الإدخال بشكل فعال:

```csharp
// احصل على مجموعة من خلايا الإدخال
ScenarioInputCellCollection sic = scenario.InputCells;

// أضف خلية إدخال في B4 بقيمة افتراضية
sic.Add(3, 1, "1100000");
```

### حفظ عملك
لا تنسى حفظ التغييرات مرة أخرى في الملف:
