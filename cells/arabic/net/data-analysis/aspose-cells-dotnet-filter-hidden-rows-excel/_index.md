## دليل التنفيذ

سنقوم بتقسيم هذا التنفيذ إلى أقسام منطقية لضمان فهم سلس لكل ميزة.

### تطبيق المرشحات التلقائية في ملفات Excel باستخدام C#

#### ملخص
يُركز هذا القسم على تحميل ملف Excel وتطبيق مُرشِّح تلقائي. سنستعيد بعد ذلك فهارس الصفوف المخفية بعد تحديث المُرشِّح.

#### خطوات

**الخطوة 1: تحميل ملف Excel**

```csharp
// قم بتحديد دليل المصدر الخاص بك وقم بتحميل ملف Excel النموذجي
string sourceDir = "PathToYourDirectory\\";
Workbook wb = new Workbook(sourceDir + "sampleGetAllHiddenRowsIndicesAfterRefreshingAutoFilter.xlsx");
```

- **توضيح**:هنا، نقوم بتهيئة `Workbook` الكائن الذي يحتوي على المسار إلى ملف Excel الخاص بنا.

**الخطوة 2: الوصول إلى التصفية التلقائية وتطبيقها**

```csharp
// الوصول إلى ورقة العمل الأولى في المصنف
Worksheet ws = wb.Worksheets[0];

// تطبيق التصفية التلقائية على فهرس العمود 0 (العمود الأول)
ws.AutoFilter.AddFilter(0, "Orange");
```

- **توضيح**:نقوم بالوصول إلى ورقة العمل الأولى وتطبيق مرشح لإظهار الصفوف التي يحتوي العمود الأول فيها على "برتقالي" فقط.

**الخطوة 3: تحديث التصفية التلقائية واسترداد الصفوف المخفية**

```csharp
// تحديث الفلتر التلقائي والحصول على مؤشرات الصفوف المخفية
int[] rowIndices = ws.AutoFilter.Refresh(true);

Console.WriteLine("Printing Rows Indices, Cell Names, and Values Hidden By AutoFilter.");
```

- **توضيح**: ال `Refresh(true)` تقوم الطريقة بتحديث الفلتر وإرجاع مجموعة من مؤشرات الصفوف المخفية بسبب الفلتر.

**الخطوة 4: طباعة تفاصيل الصف المخفي**

```csharp
for (int i = 0; i < rowIndices.Length; i++)
{
    int r = rowIndices[i];
    Cell cell = ws.Cells[r, 0];
    Console.WriteLine($"{r}\t{cell.Name}\t{cell.StringValue}");
}
```

- **توضيح**:قم بالتنقل عبر مؤشرات الصفوف المخفية وطباعة التفاصيل مثل مؤشر الصف واسم الخلية والقيمة.
