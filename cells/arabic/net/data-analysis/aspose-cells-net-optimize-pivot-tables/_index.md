الآن بعد أن قمت بالإعداد، دعنا ننتقل إلى تفاصيل التنفيذ.

### تخصيص خيارات عرض جدول المحور

يرشدك هذا القسم خلال تخصيص كيفية عرض جداول البيانات المحورية الخاصة بك باستخدام Aspose.Cells لـ .NET.

#### الإشارة إلى قيم الخلايا الفارغة

للتحكم في ما إذا كان سيتم عرض الخلايا الفارغة أم لا في جدول محوري، استخدم `DisplayNullString` ملكية:
```csharp
// الوصول إلى ورقة العمل الأولى وجدولها المحوري الأول
PivotTable pt = wb.Worksheets[0].PivotTables[0];

// اضبط على "صحيح" لعرض سلاسل فارغة للخلايا الفارغة
pt.DisplayNullString = true;
```

#### تكوين السلاسل الفارغة

حدد السلسلة التي تريد إظهارها حيث تكون الخلية فارغة `NullString`:
```csharp
// تعيين نص مخصص للقيم الفارغة
pt.NullString = "null";
pt.CalculateData();
```

#### تحديث البيانات عند فتح الملف

التحكم فيما إذا كان يجب على جدول المحور تحديث البيانات عند فتح الملف باستخدام:
```csharp
pt.RefreshDataOnOpeningFile = false;
```

### حفظ مصنفك

أخيرًا، احفظ المصنف الخاص بك بإعدادات جدول المحور المحدثة:
```csharp
wb.Save("outputSettingPivotTableOption.xlsx");
Console.WriteLine("Pivot table options set successfully.");
```
