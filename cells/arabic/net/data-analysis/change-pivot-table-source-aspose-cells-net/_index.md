### ملخص

يرشدك هذا القسم خلال تعديل بيانات المصدر لجدول محوري موجود في ملف Excel. سنحمّل المصنف، ونصل إلى أوراق العمل الخاصة به، ونحدّث خلايا محددة ببيانات جديدة، ثم نحفظ التغييرات.

#### الخطوة 1: تحميل المصنف

ابدأ بتحميل ملف Excel الخاص بك إلى `Workbook` هدف:

```csharp
// المسار إلى دليل المستندات.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// إنشاء FileStream لملف Excel
FileStream fstream = new FileStream(InputPath, FileMode.Open);

// فتح ملف Excel باستخدام FileStream
Workbook workbook = new Workbook(fstream);
```

#### الخطوة 2: الوصول إلى البيانات وتعديلها

ادخل إلى ورقة العمل التي تحتوي على نطاق بيانات جدولك المحوري. حدّثها بالقيم الجديدة حسب الحاجة:

```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];

// تحديث الخلايا ببيانات جديدة لمصدر المحور
worksheet.Cells["A9"].PutValue("Golf");
worksheet.Cells["B9"].PutValue("Qtr4");
worksheet.Cells["C9"].PutValue(7000);
```

#### الخطوة 3: تحديث النطاق المسمى

قم بتعديل النطاق المسمى ليعكس بياناتك المحدثة:

```csharp
// تحديث النطاق المسمى "DataSource"
Range range = worksheet.Cells.CreateRange(0, 0, 9, 3);
range.Name = "DataSource";
```

#### الخطوة 4: حفظ التغييرات

وأخيرًا، احفظ المصنف ببيانات المصدر المحدثة:

```csharp
// حفظ ملف Excel المعدل

// إغلاق FileStream لتحرير الموارد
```
