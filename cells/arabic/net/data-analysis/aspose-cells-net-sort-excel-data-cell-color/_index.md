## دليل التنفيذ

في هذا القسم، سنشرح عملية فرز البيانات حسب لون الخلية خطوة بخطوة.

### إنشاء مصنف وتحميله

ابدأ بإنشاء مثيل لـ `Workbook` الصف وتحميل ملف Excel الخاص بك:
```csharp
// إنشاء كائن مصنف وتحميل ملف القالب
Workbook workbook = new Workbook(sourceDir + "sampleBackGroundFile.xlsx");
```
يقوم هذا الكود بتهيئة مصنف جديد وتحميل البيانات من ملف Excel موجود موجود في دليل المصدر الخاص بك.

### تهيئة DataSorter

بعد ذلك، قم بإنشاء مثيل `DataSorter` الصف للتحضير للفرز:
```csharp
// إنشاء كائن فرز البيانات
DataSorter sorter = workbook.DataSorter;
```
ال `DataSorter` يعد ضروريًا لتحديد وتنفيذ عمليات الفرز على بياناتك.

### إضافة مفتاح فرز حسب لون الخلية

حدد كيفية فرز البيانات. هنا، نضيف مفتاحًا بناءً على لون الخلية:
```csharp
// أضف مفتاحًا للعمود الثاني للون الأحمر
csorter.AddKey(1, SortOnType.CellColor, SortOrder.Descending, Color.Red);
```
تخبر هذه الخطوة المُصنف بإعطاء الأولوية للصفوف التي تحتوي الخلايا الموجودة في العمود الثاني فيها على خلفية حمراء وفرزها بترتيب تنازلي.

### تنفيذ عملية الفرز

بعد إعداد المفاتيح، قم بإجراء الفرز:
```csharp
// فرز البيانات على أساس المفتاح
sorter.Sort(workbook.Worksheets[0].Cells, CellArea.CreateCellArea("A2", "C6"));
```
يقوم هذا الأمر بفرز الصفوف داخل منطقة الخلية المحددة (من A2 إلى C6) استنادًا إلى معاييرنا.

### حفظ البيانات المصنفة

وأخيرًا، احفظ المصنف الذي قمت بفرزه:
```csharp
// حفظ ملف الإخراج
workbook.Save(outputDir + "outputsampleBackGroundFile.xlsx");
```
