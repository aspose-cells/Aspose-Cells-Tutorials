# تنفيذ المرشحات المخصصة في Excel باستخدام Aspose.Cells لـ .NET

## مقدمة
هل ترغب في أتمتة تصفية البيانات في Excel باستخدام C#؟ تتيح لك مكتبة Aspose.Cells for .NET القوية تصفية مجموعات البيانات الكبيرة بسهولة بناءً على معايير مخصصة مباشرةً من شفرتك البرمجية. سيرشدك هذا الدليل الشامل إلى كيفية تنفيذ المرشحات المخصصة في ملفات Excel باستخدام مكتبة Aspose.Cells.

**ما سوف تتعلمه:**
- تهيئة مصنف باستخدام بيانات العينة
- الوصول إلى أوراق العمل وإعداد المرشحات التلقائية
- تطبيق التصفية المخصصة مع `AutoFilter.Contains`
- تحديث المرشحات وحفظ التغييرات
بنهاية هذا الدليل، ستتمكن من تنفيذ وظائف Excel المتقدمة برمجيًا. لنستكشف المتطلبات الأساسية اللازمة قبل البدء.

## دليل التنفيذ
### تهيئة المصنف
**ملخص:**
ابدأ بإنشاء `Workbook` كائن من ملف Excel موجود يحتوي على بيانات نموذجية. هذا يُمثل نقطة انطلاق لتطبيق المرشحات.

#### الخطوة 1: إنشاء كائن مصنف
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// قم بتحميل المصنف ببيانات العينة
Workbook workbook = new Workbook(sourceDir + "/sourceSampleCountryNames.xlsx");
```
*ال `Workbook` يمثل الكائن ملف Excel. تأكد من استبدال `"YOUR_SOURCE_DIRECTORY"` مع مسار الدليل الفعلي الخاص بك.*

### إعداد الوصول إلى ورقة العمل والتصفية
**ملخص:**
قم بالوصول إلى ورقة عمل داخل المصنف وقم بإعداد نطاق التصفية التلقائية.

#### الخطوة 2: الوصول إلى ورقة العمل
```csharp
Worksheet worksheet = workbook.Worksheets[0]; // الوصول إلى ورقة العمل الأولى
worksheet.AutoFilter.Range = "A1:A18"; // ضبط نطاق الفلتر
```
*يقوم هذا الكود بالوصول إلى ورقة العمل الأولى في ملف Excel الخاص بك ويحدد نطاقًا لتطبيق المرشحات.*

### التصفية المخصصة باستخدام AutoFilter.Contains
**ملخص:**
تطبيق التصفية المخصصة باستخدام `Contains` عامل لإظهار الصفوف المطابقة لمعايير محددة.

#### الخطوة 3: تطبيق مرشح يحتوي على
```csharp
// استخدم مرشح "يحتوي" لإظهار الصفوف التي تحتوي على "Ba"
worksheet.AutoFilter.Custom(0, FilterOperatorType.Contains, "Ba");
```
*ال `Custom` تقوم الطريقة بتصفية البيانات بناءً على معايير محددة. هنا، تبحث عن الخلايا التي تحتوي على "Ba" في العمود A.*

### تحديث وحفظ المصنف
**ملخص:**
قم بتحديث "التصفية التلقائية" المطبقة للتأكد من سريان التغييرات وحفظ المصنف المعدّل.

#### الخطوة 4: التحديث والحفظ
```csharp
// قم بتحديث الفلتر لتطبيق التغييرات
worksheet.AutoFilter.Refresh();

// حفظ ملف Excel المعدل
workbook.Save(outputDir + "/outSourceSampleCountryNames.xlsx");
```
*يضمن التحديث أن يتم تطبيق تعديلات التصفية الخاصة بك بشكل صحيح قبل الحفظ.*
