## دليل التنفيذ

الآن بعد أن قمت بإعداد كل شيء، دعنا ننفذ شرائح في جدول محوري باستخدام Aspose.Cells لـ .NET.

### تحميل المصنف والوصول إليه

أولاً، قم بتحميل ملف Excel الذي يحتوي على جدول المحور:

```csharp
// مسار دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// تحميل المصنف
Workbook wb = new Workbook(sourceDir + "sampleCreateSlicerToPivotTable.xlsx");
```

#### الوصول إلى أوراق العمل والجداول المحورية

الوصول إلى ورقة العمل والجدول المحوري المحددين:

```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet ws = wb.Worksheets[0];

// الوصول إلى الجدول المحوري الأول في ورقة العمل
Aspose.Cells.Pivot.PivotTable pt = ws.PivotTables[0];
```

### إضافة شريحة إلى جدول المحور

الآن، أضف مقطعًا مرتبطًا بجدول المحور الخاص بك:

```csharp
// أضف المقطع في الخلية B22 مع الحقل الأساسي الأول لجدول المحور
int idx = ws.Slicers.Add(pt, "B22", pt.BaseFields[0]);

// قم بالوصول إلى المقطعة المضافة حديثًا من مجموعة المقطعات
Aspose.Cells.Slicers.Slicer slicer = ws.Slicers[idx];
```

#### توضيح:
- **`ws.Slicers.Add()`**:تضيف هذه الطريقة مقطعًا إلى ورقة العمل. 
  - `pt`:كائن الجدول المحوري.
  - "B22": الموضع الذي سيتم وضع المقطع فيه.
  - `pt.BaseFields[0]`: الحقل الأساسي الذي تستخدمه المقطعة.

### احفظ مصنفك

وأخيرًا، احفظ المصنف الخاص بك بالتنسيقات المطلوبة:

```csharp
// تحديد مسار دليل الإخراج
string outputDir = RunExamples.Get_OutputDirectory();

// حفظ بتنسيق XLSX
wb.Save(outputDir + "outputCreateSlicerToPivotTable.xlsx", SaveFormat.Xlsx);

// حفظ بتنسيق XLSB
wb.Save(outputDir + "outputCreateSlicerToPivotTable.xlsb", SaveFormat.Xlsb);
```
