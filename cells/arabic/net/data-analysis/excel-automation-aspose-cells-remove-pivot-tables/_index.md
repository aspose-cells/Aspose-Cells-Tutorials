## دليل التنفيذ
### الوصول إلى جدول محوري وإزالته حسب الكائن
توضح هذه الميزة كيفية الوصول إلى جدول محوري وإزالته في ورقة عمل Excel باستخدام مرجع الكائن الخاص به.

#### التنفيذ خطوة بخطوة
**1. إنشاء كائن مصنف**
قم بتحميل ملف Excel المصدر الخاص بك إلى `Workbook` فصل:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleRemovePivotTable.xlsx");
```

**2. الوصول إلى ورقة العمل والجدول المحوري**
الوصول إلى ورقة العمل المطلوبة وكائن الجدول المحوري:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
PivotTable pivotTable = worksheet.PivotTables[0];
```

**3. إزالة جدول المحور باستخدام مرجع الكائن**
استدعاء `Remove` الطريقة على كائن جدول المحور:
```csharp
worksheet.PivotTables.Remove(pivotTable);
```

**4. حفظ التغييرات في ملف جديد**
الحفاظ على التغييرات عن طريق حفظ المصنف:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/outputRemovePivotTable.xlsx");
```

### الوصول إلى جدول محوري وإزالته حسب الموضع
إذا كنت تفضل استخدام موضع الفهرس في جدول المحور، فإن هذه الطريقة تبسط عملية الإزالة.

#### التنفيذ خطوة بخطوة
**1. إنشاء كائن مصنف**
كما في السابق، قم بتحميل ملف Excel الخاص بك:
```csharp
Workbook workbook = new Workbook(SourceDir + "/sampleRemovePivotTable.xlsx");
```

**2. الوصول إلى جدول محوري وإزالته حسب الفهرس**
قم بإزالة جدول المحور مباشرةً باستخدام مؤشر موضعه:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.PivotTables.RemoveAt(0);
```

**3. حفظ التغييرات في ملف جديد**
احفظ المصنف المحدث بالتغييرات:
```csharp
workbook.Save(outputDir + "/outputRemovePivotTableByPosition.xlsx");
```
