### الوصول إلى جداول البيانات المحورية في ورقة العمل
لتحديث الجداول المتداخلة، قم بالوصول إلى ورقة العمل وحدد موقع جدول المحور الرئيسي:
```csharp
Worksheet ws = wb.Worksheets[0];

```

### تحديث جداول المحور الفرعية
بعد تحديد جدول المحور الرئيسي، قم باسترداد العناصر الفرعية الخاصة به وتحديثها:
```csharp
// احصل على جميع جداول المحور الفرعية للوالد
PivotTable[] ptChildren = ptParent.GetChildren();

// قم بالتنقل عبر كل جدول محوري فرعي لتحديثه
foreach (var ptChild in ptChildren)
{
    ptChild.RefreshData();
    ptChild.CalculateData();  // التأكد من حساب البيانات المحدثة
}
```
#### توضيح
- **احصل على الأطفال ()**:استرجاع جميع جداول المحور المتداخلة ضمن الجدول الرئيسي.
- **RefreshData() و CalculateData()**:تحديث وإعادة حساب البيانات في كل جدول محوري فرعي، مما يضمن الدقة.
