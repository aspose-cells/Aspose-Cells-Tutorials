## دليل التنفيذ

### إنشاء جدول PivotTable الأول الخاص بك باستخدام الفرز المخصص

لنبدأ بإنشاء وتخصيص جدول محوري باستخدام Aspose.Cells. سنستكشف كيفية إضافة حقول إلى مناطق مختلفة من الجدول المحوري وتطبيق ميزات الفرز.

#### الخطوة 1: تهيئة المصنف وورقة العمل
ابدأ بتحميل ملف Excel الخاص بك والإشارة إلى ورقة العمل التي تريد إنشاء جدول PivotTable فيها.
```csharp
// تهيئة المصنف باستخدام مسار الملف المصدر
Workbook wb = new Workbook(sourceDir + "SamplePivotSort.xlsx");

// الوصول إلى ورقة العمل الأولى
Worksheet sheet = wb.Worksheets[0];
```

#### الخطوة 2: إضافة جدول محوري إلى ورقة العمل
إنشاء جدول محوري جديد وتكوين نطاق البيانات الخاص به.
```csharp
// إضافة جدول محوري إلى ورقة العمل في موقع محدد
int index = sheet.PivotTables.Add("=Sheet1!A1:C10", "E3", "PivotTable2");

// الوصول إلى مثيل PivotTable المضاف حديثًا
PivotTable pivotTable = sheet.PivotTables[index];
```

#### الخطوة 3: تخصيص حقول الصفوف والأعمدة باستخدام الفرز
قم بتكوين حقول الصفوف للفرز، مع التأكد من عرض البيانات بترتيب ذي معنى.
```csharp
// عدم إظهار الإجماليات الكبرى من أجل الوضوح
pivotTable.RowGrand = false;
pivotTable.ColumnGrand = false;

// أضف الحقل الأول إلى منطقة الصف وقم بتمكين الفرز
pivotTable.AddFieldToArea(PivotFieldType.Row, 1);
PivotField rowField = pivotTable.RowFields[0];
rowField.IsAutoSort = true; // تمكين الفرز التلقائي
rowField.IsAscendSort = true; // فرز حسب الترتيب التصاعدي

// تكوين حقل العمود بتنسيق التاريخ والفرز
pivotTable.AddFieldToArea(PivotFieldType.Column, 0);
PivotField colField = pivotTable.ColumnFields[0];
colField.NumberFormat = "dd/mm/yyyy"; // تعيين تنسيق التاريخ
colField.IsAutoSort = true;
colField.IsAscendSort = true;
```

#### الخطوة 4: إضافة حقل البيانات وتحديث جدول البيانات المحوري
أضف حقل بيانات لإكمال الإعداد، ثم قم بتحديث البيانات وحسابها للحصول على نتائج محدثة.
```csharp
// إضافة الحقل الثالث إلى منطقة البيانات
pivotTable.AddFieldToArea(PivotFieldType.Data, 2);

// تحديث بيانات الجدول المحوري وحسابها
pivotTable.RefreshData();
pivotTable.CalculateData();
```

كرر الخطوات المماثلة لإنشاء جداول محورية إضافية مع فرز مخصص استنادًا إلى معايير محددة مثل "المأكولات البحرية" أو تواريخ معينة.
