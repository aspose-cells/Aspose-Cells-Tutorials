## دليل التنفيذ

### نظرة عامة على الترتيب في جداول البيانات المحورية

تتيح لك هذه الميزة تصنيف البيانات داخل جدول محوري، مما يوفر رؤى حول الموضع النسبي للقيم من الأكبر إلى الأصغر.

#### تحميل المصنف والوصول إليه

أولاً، قم بتحميل ملف Excel الموجود الذي يحتوي على جدول المحور الخاص بك:

```csharp
// دلائل للملفات المصدر والإخراج
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// تحميل مصنف باستخدام قالب PivotTable
Workbook workbook = new Workbook(sourceDir + "PivotTableSample.xlsx");
```

#### الوصول إلى جدول البيانات المحوري

قم بالوصول إلى جدول المحور المحدد الذي ترغب في تطبيق التصنيف عليه:

```csharp
// احصل على أول ورقة عمل تحتوي على جدول PivotTable
Worksheet worksheet = workbook.Worksheets[0];

// افترض أن الجدول المحوري موجود عند الفهرس 0
int pivotIndex = 0;
PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
```

#### تكوين تنسيق عرض البيانات

قم بتكوين ترتيب حقول البيانات داخل جدول المحور الخاص بك:

```csharp
// الوصول إلى مجموعة حقول البيانات من PivotTable
PivotFieldCollection pivotFields = pivotTable.DataFields;

// احصل على حقل البيانات الأول لتطبيق تنسيق الرتبة
PivotField pivotField = pivotFields[0];

// تعيين تنسيق العرض للترتيب من الأكبر إلى الأصغر
pivotField.DataDisplayFormat = PivotFieldDataDisplayFormat.RankLargestToSmallest;
```

#### حفظ التغييرات

بعد التكوين، احفظ المصنف الخاص بك:

```csharp
// حساب البيانات وحفظ المصنف مع التغييرات
pivotTable.CalculateData();
workbook.Save(outputDir + "PivotTableDataDisplayFormatRanking_out.xlsx");
```
