## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى قسمين رئيسيين: إنشاء مصنف وتطبيق وظائف توحيد PivotTable.

### الميزة 1: إنشاء مصنف والوصول إلى ورقة العمل

#### ملخص
إنشاء مصنفات من ملفات Excel موجودة أمرٌ أساسي لأتمتة إنشاء التقارير. تتيح لك هذه الميزة تحميل ملف موجود، والوصول إلى أوراق عمله، وحفظ التغييرات بكفاءة.

**التنفيذ خطوة بخطوة:**

##### الخطوة 1: تحديد مسارات الملفات
ابدأ بإعداد دليل المصدر الذي يوجد به ملف Excel ودليل الإخراج لحفظ التغييرات.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// المسار إلى ملف Excel المصدر
string filePath = Path.Combine(SourceDir, "Book.xlsx");
```

##### الخطوة 2: تحميل المصنف وورقة عمل Access
قم بتحميل المصنف الحالي والوصول إلى ورقة العمل الأولى الخاصة به.

```csharp
// تحميل مصنف موجود من الملف المحدد
Workbook workbook = new Workbook(filePath);

// الوصول إلى ورقة العمل الأولى في المصنف
Worksheet worksheet = workbook.Worksheets[0];
```

##### الخطوة 3: حفظ التغييرات في ملف جديد
بعد إجراء أي تعديلات، احفظ المصنف في ملف Excel جديد.

```csharp
// حفظ التغييرات في ملف جديد
string outputFilePath = Path.Combine(OutputDir, "output.xlsx");
workbook.Save(outputFilePath);
```

### الميزة 2: وظائف دمج الجداول المحورية

#### ملخص
تُعد جداول البيانات المحورية أدوات فعّالة لتلخيص البيانات. يُمكن لتطبيق دوال مثل Average وDistinctCount تحسين قدرات تحليل البيانات لديك.

**التنفيذ خطوة بخطوة:**

##### الخطوة 1: تحميل المصنف باستخدام PivotTable
ابدأ بتحميل المصنف الذي يحتوي على جدول PivotTable الخاص بك.

```csharp
string filePath = Path.Combine(SourceDir, "Book.xlsx");
Workbook workbook = new Workbook(filePath);
Worksheet worksheet = workbook.Worksheets[0];
```

##### الخطوة 2: الوصول إلى جدول البيانات المحوري وتكوينه
قم بالوصول إلى أول جدول محوري في ورقة العمل وقم بتطبيق وظائف التوحيد على حقول البيانات الخاصة به.

```csharp
PivotTable pivotTable = worksheet.PivotTables[0];

// تطبيق دالة المتوسط على حقل البيانات الأول
pivotTable.DataFields[0].Function = ConsolidationFunction.Average;

// تطبيق دالة DistinctCount على حقل البيانات الثاني
pivotTable.DataFields[1].Function = ConsolidationFunction.DistinctCount;
```

##### الخطوة 3: حساب التغييرات وحفظها
تأكد من حساب التغييرات وحفظها.

```csharp
pivotTable.CalculateData();
string outputFilePath = Path.Combine(OutputDir, "output.xlsx");
workbook.Save(outputFilePath);
```
