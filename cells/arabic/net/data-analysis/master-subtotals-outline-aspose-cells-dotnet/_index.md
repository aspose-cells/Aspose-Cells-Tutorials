## دليل التنفيذ

دعونا نقسم العملية إلى أجزاء قابلة للإدارة لتطبيق المجاميع الفرعية والتحكم في اتجاه المخطط التفصيلي.

### الخطوة 1: تهيئة المصنف وورقة العمل

أولاً، قم بإنشاء مثيل لـ `Workbook` عن طريق تحميل ملف Excel والوصول إلى ورقة العمل الأولى الخاصة به:

```csharp
// إنشاء مصنف من ملف Excel المصدر
Workbook workbook = new Workbook(sourceDir + "sampleApplyingSubtotalChangeSummaryDirection.xlsx");

// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```

### الخطوة 2: تحديد مساحة الخلية للمجموعات الفرعية

حدد نطاق الخلايا الذي تريد تطبيق المجاميع الفرعية عليه. هنا، نحدد `A2:B11`:

```csharp
// احصل على مجموعة الخلايا في ورقة العمل الأولى
Cells cells = worksheet.Cells;

// إنشاء منطقة خلية، على سبيل المثال، A2:B11
CellArea ca = CellArea.CreateCellArea("A2", "B11");
```

### الخطوة 3: تطبيق المجاميع الفرعية

استخدم `Subtotal` طريقة تطبيق المجاميع الفرعية وتحديد الأعمدة ووظائف التوحيد:

```csharp
// تطبيق المجموع الفرعي باستخدام دالة المجموع على العمود B
cells.Subtotal(ca, 0, ConsolidationFunction.Sum, new int[] { 1 }, true, false, true);
```
- **وظيفة التوحيد**:يحدد العملية (على سبيل المثال، المجموع).
- **فهرس الأعمدة**:يحدد الأعمدة التي سيتم تضمينها.

### الخطوة 4: تعيين اتجاه المخطط التفصيلي

التحكم في مكان ظهور صفوف الملخص باستخدام `SummaryRowBelow` ملكية:

```csharp
// تعيين اتجاه ملخص المخطط التفصيلي
worksheet.Outline.SummaryRowBelow = true;
```

يضمن هذا الإعداد وضع صفوف الملخص أسفل عناصر المجموعة، مما يعزز قابلية القراءة.

### الخطوة 5: حفظ التغييرات

وأخيرًا، احفظ المصنف المعدّل في ملف جديد:

```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputApplyingSubtotalChangeSummaryDirection.xlsx");
```
