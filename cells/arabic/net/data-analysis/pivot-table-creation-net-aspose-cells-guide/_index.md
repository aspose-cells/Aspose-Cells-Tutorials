## دليل التنفيذ
### إنشاء جدول محوري باستخدام Aspose.Cells
دعنا نستعرض عملية إنشاء جدول محوري خطوة بخطوة باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: تهيئة المصنف الخاص بك
أولاً، قم بإنشاء مثيل لـ `Workbook` الصف. هذا يمثل ملف Excel الخاص بك:

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

#### الخطوة 2: تحضير البيانات في ورقة العمل
قم بالوصول إلى ورقة العمل الأولى وقم بملئها بالبيانات اللازمة لجدول PivotTable الخاص بك:

```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا
Worksheet sheet = workbook.Worksheets[0];
Cells cells = sheet.Cells;

// تعيين قيم للخلايا
cells["A1"].PutValue("Sport");
cells["B1"].PutValue("Quarter");
cells["C1"].PutValue("Sales");

// إضافة بيانات العينة
string[] sports = { "Golf", "Golf", "Tennis", "Tennis", "Tennis", "Tennis", "Golf" };
string[] quarters = { "Qtr3", "Qtr4", "Qtr3", "Qtr4", "Qtr3", "Qtr4", "Qtr3" };
int[] sales = { 1500, 2000, 600, 1500, 4070, 5000, 6430 };

for (int i = 0; i < sports.Length; i++)
{
    cells[$"A{i + 2}"].PutValue(sports[i]);
cells[$"B{i + 2}"].PutValue(quarters[i]);
cells[$"C{i + 2}"].PutValue(sales[i]);
}
```

#### الخطوة 3: إنشاء جدول محوري وتكوينه
الآن، أضف جدولًا محوريًا إلى ورقة العمل الخاصة بك:

```csharp
// إضافة جدول محوري إلى ورقة العمل
PivotTableCollection pivotTables = sheet.PivotTables;
int index = pivotTables.Add("=A1:C8", "E3", "PivotTable2");

// الوصول إلى مثيل PivotTable المضاف حديثًا
PivotTable pivotTable = pivotTables[index];

// تكوين إعدادات PivotTable
pivotTable.RowGrand = false; // إخفاء الإجماليات الكبرى للصفوف

// سحب الحقول إلى المناطق المناسبة
pivotTable.AddFieldToArea(PivotFieldType.Row, 0);   // ملعب رياضي في منطقة التجديف
pivotTable.AddFieldToArea(PivotFieldType.Column, 1); // ربع الحقل في منطقة العمود
pivotTable.AddFieldToArea(PivotFieldType.Data, 2);   // حقل المبيعات في منطقة البيانات
```

#### الخطوة 4: حفظ المصنف
وأخيرًا، احفظ مصنفك لرؤية النتائج:

```csharp
// حفظ ملف Excel
cells.Workbook.Save("pivotTable_test_out.xls");
```
