# تطبيق التنسيق الشرطي في وقت التشغيل في Excel

## مقدمة

إنها أدوات فعّالة لتحليل البيانات وتصورها. ومن أبرز ميزات Excel التنسيق الشرطي، الذي يسمح للمستخدمين بتطبيق أنماط تنسيق محددة على الخلايا بناءً على قيمها. يُسهّل هذا تحديد الاتجاهات، وإبراز نقاط البيانات المهمة، أو ببساطة جعل البيانات أكثر قابلية للقراءة. إذا كنت ترغب في تطبيق التنسيق الشرطي في ملفات Excel برمجيًا، فأنت في المكان المناسب! في هذا الدليل، سنشرح كيفية تطبيق التنسيق الشرطي أثناء التشغيل باستخدام Aspose.Cells لـ .NET.

## المتطلبات الأساسية
قبل الغوص في الكود، دعنا نتأكد من أن لديك كل ما تحتاجه للبدء:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. يمكنك استخدام أي إصدار يدعم تطوير .NET.
2. Aspose.Cells لـ .NET: ستحتاج إلى تثبيت Aspose.Cells لـ .NET. يمكنك تنزيله من [موقع Aspose](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.
4. .NET Framework: تأكد من أن مشروعك يستهدف إصدارًا متوافقًا من .NET Framework.

الآن بعد أن قمنا بتغطية المتطلبات الأساسية، فلننتقل إلى الجزء الممتع!

## استيراد الحزم
لبدء استخدام Aspose.Cells، ستحتاج إلى استيراد مساحات الأسماء اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

```csharp
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

ستتيح لك هذه المساحات الاسمية الوصول إلى الفئات والطرق المطلوبة لمعالجة ملفات Excel وتطبيق التنسيق الشرطي.

الآن، دعونا نقوم بتقسيم عملية تطبيق التنسيق الشرطي إلى خطوات قابلة للإدارة.

## الخطوة 1: إعداد مشروعك
أولاً، عليك إنشاء مشروع C# جديد في Visual Studio. إليك الطريقة:

1. افتح Visual Studio وحدد ملف > جديد > مشروع.
2. اختر تطبيق وحدة التحكم (.NET Framework) وأعطي مشروعك اسمًا.
3. انقر فوق إنشاء.

## الخطوة 2: إضافة مرجع Aspose.Cells
بمجرد إعداد مشروعك، ستحتاج إلى إضافة مرجع إلى مكتبة Aspose.Cells:

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد إدارة حزم NuGet.
3. ابحث عن Aspose.Cells وقم بتثبيته.

سيسمح لك هذا باستخدام كافة الوظائف التي توفرها مكتبة Aspose.Cells.

## الخطوة 3: إنشاء كائن مصنف
الآن، لنُنشئ مصنفًا وورقة عمل جديدين. هنا تبدأ كل التفاصيل:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
string filePath = dataDir + "Book1.xlsx";

// إنشاء كائن مصنف
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

في هذه الخطوة، نقوم بتحديد الدليل الذي سيتم حفظ ملف Excel فيه، وإنشاء مصنف جديد، والوصول إلى ورقة العمل الأولى.

## الخطوة 4: إضافة التنسيق الشرطي
الآن، لنُضِف بعض التنسيق الشرطي. سنبدأ بإنشاء كائن تنسيق شرطي فارغ:

```csharp
// يضيف تنسيقًا شرطيًا فارغًا
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcs = sheet.ConditionalFormattings[index];
```

هنا، نضيف مجموعة تنسيق شرطية جديدة إلى ورقة العمل الخاصة بنا، والتي ستحمل قواعد التنسيق الخاصة بنا.

## الخطوة 5: تحديد نطاق التنسيق
بعد ذلك، علينا تحديد نطاق الخلايا التي سيُطبّق عليها التنسيق الشرطي. لنفترض أننا نريد تنسيق الصف الأول والعمود الثاني:

```csharp
// تعيين نطاق التنسيق الشرطي.
CellArea ca = new CellArea();
ca.StartRow =0;
ca.EndRow =0;
ca.StartColumn =0;
ca.EndColumn =0;
fcs.AddArea(ca);

ca = new CellArea();
ca.StartRow =1;
ca.EndRow =1;
ca.StartColumn =1;
ca.EndColumn =1;
fcs.AddArea(ca);
```

في هذا الكود، نُعرّف منطقتين للتنسيق الشرطي. المنطقة الأولى للخلية عند (0,0) والثانية للخلية عند (1,1). لا تتردد في تعديل هذه النطاقات وفقًا لاحتياجاتك الخاصة!

## الخطوة 6: إضافة شروط التنسيق الشرطي
الآن حان وقت تحديد شروط التنسيق. لنفترض أننا نريد تمييز الخلايا بناءً على قيمها:

```csharp
// يضيف الشرط.
int conditionIndex = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "=A2", "100");

// يضيف الشرط.
int conditionIndex2 = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "50", "100");
```

في هذه الخطوة، نضيف شرطين: أحدهما للقيم بين `A2` و `100`، وآخر للقيم بين `50` و `100`يتيح لك هذا إمكانية تمييز الخلايا بشكل ديناميكي استنادًا إلى قيمها.

## الخطوة 7: تعيين أنماط التنسيق
بعد ضبط شروطنا، يمكننا الآن ضبط أنماط التنسيق. لنُغيّر لون خلفية شروطنا:

```csharp
// تعيين لون الخلفية.
FormatCondition fc = fcs[conditionIndex];
fc.Style.BackgroundColor = Color.Red;
```

هنا، نُعيّن لون خلفية الشرط الأول إلى الأحمر. يُمكنك تخصيصه أكثر بتغيير لون الخط والحدود والأنماط الأخرى حسب الحاجة!

## الخطوة 8: حفظ ملف Excel
أخيرًا، حان وقت حفظ عملنا! سنحفظ المصنف في المجلد المحدد:

```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "output.xls");
```

هذا السطر من التعليمات البرمجية يحفظ ملف إكسل بالتنسيق الشرطي. تأكد من التحقق من الدليل المحدد لملف الإخراج!
