# قراءة وقت إنشاء التعليقات المترابطة في ورقة العمل

## مقدمة
عند العمل مع ملفات Excel، تُعدّ إدارة التعليقات جانبًا أساسيًا في تعاون البيانات وتلقي الملاحظات. إذا كنت تستخدم Aspose.Cells لـ .NET، فستجده فعالًا للغاية في التعامل مع وظائف Excel المختلفة، بما في ذلك التعليقات المترابطة. في هذا البرنامج التعليمي، سنركز على كيفية قراءة وقت إنشاء التعليقات المترابطة في ورقة عمل. سواء كنت مطورًا متمرسًا أو مبتدئًا، سيرشدك هذا الدليل خلال العملية خطوة بخطوة.
## المتطلبات الأساسية
قبل أن نتعمق في الكود، دعنا نتأكد من أن لديك كل ما تحتاجه للبدء:
1. Aspose.Cells لـ .NET: تأكد من تثبيت مكتبة Aspose.Cells. يمكنك تنزيلها من [موقع Aspose](https://releases.aspose.com/cells/net/).
2. Visual Studio: تثبيت عامل من Visual Studio أو أي .NET IDE آخر حيث يمكنك كتابة وتنفيذ كود C# الخاص بك.
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.
4. ملف إكسل: جهّز ملف إكسل مع بعض التعليقات المترابطة. في هذا المثال، سنستخدم ملفًا باسم `ThreadedCommentsSample.xlsx`.
الآن بعد أن قمنا بتغطية المتطلبات الأساسية، فلنبدأ في استيراد الحزم الضرورية.
## استيراد الحزم
لبدء استخدام Aspose.Cells، عليك استيراد مساحات الأسماء المطلوبة. إليك كيفية القيام بذلك:
### استيراد مساحة اسم Aspose.Cells
افتح مشروع C# الخاص بك في Visual Studio وأضف التوجيه التالي باستخدام في الجزء العلوي من ملف الكود الخاص بك:
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```
تتيح لك مساحة الاسم هذه الوصول إلى جميع الفئات والطرق التي توفرها مكتبة Aspose.Cells.
الآن بعد أن هيأنا المسرح، دعنا نقسم عملية قراءة وقت إنشاء التعليقات المترابطة إلى خطوات قابلة للإدارة.
## الخطوة 1: تحديد دليل المصدر
أولاً، عليك تحديد المجلد الذي يوجد فيه ملف Excel. هذا أمر بالغ الأهمية لأن البرنامج يحتاج إلى معرفة مكان البحث عن الملف.
```csharp
// دليل المصدر
string sourceDir = "Your Document Directory";
```
يستبدل `"Your Document Directory"` مع المسار الفعلي لملف Excel الخاص بك. قد يكون هذا شيئًا مثل `"C:\\Documents\\"`.
## الخطوة 2: تحميل المصنف
بعد ذلك، ستُحمّل مصنف Excel الذي يحتوي على التعليقات المترابطة. إليك كيفية القيام بذلك:
```csharp
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```
يؤدي هذا السطر من التعليمات البرمجية إلى إنشاء سطر جديد `Workbook` الكائن عن طريق تحميل ملف Excel المحدد. إذا لم يتم العثور على الملف، فسيتم طرح استثناء، لذا تأكد من صحة المسار.
## الخطوة 3: الوصول إلى ورقة العمل
بعد تحميل المصنف، الخطوة التالية هي الوصول إلى ورقة العمل المحددة التي تحتوي على التعليقات. في حالتنا، سنصل إلى ورقة العمل الأولى:
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```
يسترجع هذا السطر ورقة العمل الأولى (الفهرس ٠) من المصنف. إذا كانت تعليقاتك موجودة في ورقة عمل أخرى، فعدّل الفهرس وفقًا لذلك.
## الخطوة 4: الحصول على التعليقات المترابطة
الآن، حان وقت استرجاع التعليقات المترابطة من خلية محددة. في هذا المثال، سنحصل على التعليقات من الخلية A1:
```csharp
// احصل على تعليقات مترابطة
ThreadedCommentCollection threadedComments = worksheet.Comments.GetThreadedComments("A1");
```
يقوم هذا السطر بجلب جميع التعليقات المترابطة المرتبطة بالخلية A1. في حال عدم وجود أي تعليقات، ستكون المجموعة فارغة.
## الخطوة 5: تكرار التعليقات
بعد استرداد التعليقات المترابطة، يمكننا الآن المرور عليها وعرض التفاصيل، بما في ذلك وقت إنشائها:
```csharp
foreach (ThreadedComment comment in threadedComments)
{
    Console.WriteLine("Comment: " + comment.Notes);
    Console.WriteLine("Author: " + comment.Author.Name);
    Console.WriteLine("Created Time: " + comment.CreatedTime);
}
```
تمر هذه الحلقة عبر كل تعليق في `threadedComments` يقوم بتجميع وطباعة نص التعليق واسم المؤلف والوقت الذي تم إنشاء التعليق فيه.
## الخطوة 6: رسالة التأكيد
أخيرًا، بعد تنفيذ منطق قراءة التعليقات، يُنصح دائمًا بإرسال رسالة تأكيد. هذا يُساعد في تصحيح الأخطاء ويضمن تنفيذ الكود بنجاح.
```csharp
Console.WriteLine("ReadThreadedCommentCreatedTime executed successfully.");
```
## خاتمة
تهانينا! لقد نجحت في تعلم كيفية قراءة وقت إنشاء التعليقات المترابطة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. هذه الوظيفة مفيدة للغاية لتتبع الملاحظات والتعاون في مستندات Excel. ببضعة أسطر فقط من التعليمات البرمجية، يمكنك استخراج معلومات قيّمة تُحسّن عمليات تحليل البيانات وإعداد التقارير.
