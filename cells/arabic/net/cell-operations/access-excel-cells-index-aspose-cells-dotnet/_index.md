## دليل التنفيذ

### إنشاء كائن مصنف
#### ملخص
إنشاء مثيل لـ `Workbook` الفئة هي الخطوة الأولى، وتمثل ملف Excel الذي ستتعامل معه.

**الخطوة 1: تحميل ملف Excel**
حدد الدليل الذي يحتوي على ملف Excel الخاص بك وقم بتحميله في `Workbook` هدف:
```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// قم بإنشاء كائن مصنف جديد عن طريق تحميل ملف Excel.
Workbook workbook = new Workbook(sourceDir + "sampleAccessCellByRowAndColumnIndex.xlsx");
```
يقوم الكود أعلاه بتهيئة `workbook` مع البيانات من ملف Excel المحدد الخاص بك، وجاهزة لمزيد من العمليات.

### الوصول إلى الخلايا في ورقة العمل
#### ملخص
بمجرد تحميل المصنف الخاص بك، يصبح الوصول إلى خلايا معينة من خلال مؤشراتها أمرًا سهلاً.

**الخطوة 1: الوصول إلى ورقة العمل الأولى**
تتكون مصنفات العمل من عدة أوراق عمل. يمكنك الوصول إليها باستخدام الفهرسة الصفرية:
```csharp
// الوصول إلى ورقة العمل الأولى.
Worksheet worksheet = workbook.Worksheets[0];
```

**الخطوة 2: الوصول إلى خلية محددة**
استرداد خلية حسب مؤشرات الصفوف والأعمدة (فهرسة الصفر):
```csharp
// الوصول إلى خلية محددة باستخدام مؤشرات الصفوف والأعمدة الخاصة بها.
Cell cell = worksheet.Cells[5, 2]; // الصف السادس، العمود الثالث.

// إخراج اسم الخلية وقيمتها.
Console.WriteLine("Cell Name: " + cell.Name + " Value: " + cell.StringValue);
```
