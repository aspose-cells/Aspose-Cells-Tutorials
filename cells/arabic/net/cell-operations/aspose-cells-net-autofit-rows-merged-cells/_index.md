## دليل التنفيذ

سنقوم بإرشادك خلال عملية تركيب الصفوف تلقائيًا في الخلايا المدمجة باستخدام C#.

### إنشاء الخلايا ودمجها

#### ملخص

أولاً، قم بإنشاء نطاق من الخلايا ودمجها لإعداد ورقة العمل الخاصة بك قبل تطبيق إعدادات التجهيز التلقائي.

**الخطوة 1: إنشاء مثيل للمصنف وورقة العمل**

```csharp
// دليل الإخراج
string outputDir = RunExamples.Get_OutputDirectory();

// إنشاء مصنف جديد
Workbook wb = new Workbook();

// احصل على ورقة العمل الأولى (الافتراضية)
Worksheet _worksheet = wb.Worksheets[0];
```

#### الخطوة 2: إنشاء النطاق ودمجه

إنشاء نطاق من الخلايا ليتم دمجها لتمثيل البيانات الموحدة.

```csharp
// إنشاء نطاق A1:B1
Range range = _worksheet.Cells.CreateRange(0, 0, 1, 2);

// دمج الخلايا
range.Merge();
```

### إدراج خلايا القيمة والنمط

#### ملخص

بعد الدمج، قم بإدراج النص في الخلية المدمجة وقم بتطبيق التصميم لضمان إمكانية القراءة.

**الخطوة 3: إضافة النص والأسلوب**

أدرج جملة طويلة لإظهار إمكانيات التجهيز التلقائي. فعّل التفاف النص وحدد الأنماط لمزيد من الوضوح.

```csharp
// إدراج قيمة في الخلية المدمجة A1
_worksheet.Cells[0, 0].Value = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog....end";

// إنشاء كائن نمط
Aspose.Cells.Style style = _worksheet.Cells[0, 0].GetStyle();

// تعيين التفاف النص على
style.IsTextWrapped = true;

// تطبيق النمط على الخلية
_worksheet.Cells[0, 0].SetStyle(style);
```

### صفوف التجهيز التلقائي

#### ملخص

استخدم Aspose.Cells `AutoFitterOptions` لضبط ارتفاعات الصفوف للخلايا المندمجة.

**الخطوة 4: تكوين وتطبيق AutoFit**

قم بتكوين خيارات الملاءمة التلقائية المخصصة للخلايا المدمجة، مما يضمن أن كل سطر من النص يتناسب تمامًا مع الخلية.

```csharp
// إنشاء كائن لـ AutoFitterOptions
AutoFitterOptions options = new AutoFitterOptions();

// تعيين الملاءمة التلقائية للخلايا المندمجة
options.AutoFitMergedCellsType = AutoFitMergedCellsType.EachLine;

// ضبط الصفوف تلقائيًا في الورقة (بما في ذلك الخلايا المدمجة)
_worksheet.AutoFitRows(options);
```

### حفظ ومراجعة

#### ملخص

وأخيرًا، احفظ المصنف الخاص بك لمراجعة التغييرات.

```csharp
// حفظ ملف Excel
wb.Save(outputDir + "AutofitRowsforMergedCells.xlsx");

```
