## دليل التنفيذ

### تحديد أيام لحفظ التاريخ في المصنفات المشتركة

في مصنفات العمل المشتركة، يُعدّ تتبع المراجعات أمرًا أساسيًا للتحرير التعاوني. باستخدام Aspose.Cells، يمكنك تحديد مدة حفظ هذه السجلات.

#### إنشاء مصنف مشترك وتكوينه

**الخطوة 1: إنشاء مصنف فارغ**

```csharp
// إنشاء مثيل جديد للمصنف
Workbook wb = new Workbook();
```

**الخطوة 2: مشاركة المصنف**

تمكين المشاركة للسماح لمستخدمين متعددين بالتحرير:

```csharp
// تمكين الإعدادات المشتركة
wb.Settings.Shared = true;
```

**الخطوة 3: تحديث أيام الحفاظ على سجل المراجعة**

حدد عدد الأيام التي تريد الاحتفاظ بسجل المراجعة فيها:

```csharp
// تعيين أيام لحفظ سجلات المراجعة
wb.Worksheets.RevisionLogs.DaysPreservingHistory = 7;
```

يضمن هذا الإعداد تسجيل التغييرات الخاصة بالأيام السبعة الأخيرة فقط، مما يحافظ على سجلاتك موجزة وذات صلة.

**الخطوة 4: حفظ المصنف**

وأخيرًا، احفظ المصنف بالإعدادات المحدثة:

```csharp
// تحديد دليل الإخراج
string outputDir = RunExamples.Get_OutputDirectory();

// حفظ الملف
wb.Save(outputDir + "outputShared_DaysPreservingHistory.xlsx");
```

#### نصائح استكشاف الأخطاء وإصلاحها

- **تأكد من مشاركة المصنف**:إذا لم تعكس التغييرات، فتأكد من ذلك `wb.Settings.Shared` تم ضبطه على true.
- **التحقق من قيمة الأيام**: يضمن `DaysPreservingHistory` هو عدد صحيح موجب.
