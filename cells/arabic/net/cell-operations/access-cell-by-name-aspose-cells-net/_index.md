#### الخطوة 1: تحميل المصنف (H3)

ابدأ بتحميل المصنف الخاص بك من دليل محدد:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleAccessCellUsingCellName.xlsx");
```
*لماذا هذه الخطوة؟*:يعد تحميل المصنف أمرًا ضروريًا للوصول إلى أي بيانات داخل ملفات Excel.

#### الخطوة 2: الوصول إلى ورقة العمل (H3)

استرجع ورقة العمل التي تريد العمل عليها. هنا، نصل إلى ورقة العمل الأولى:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
*لماذا هذه الخطوة؟*:تعتبر أوراق العمل بمثابة حاويات للخلايا؛ لذا فإن الوصول إليها ضروري قبل التعامل مع بيانات الخلايا.

#### الخطوة 3: الوصول إلى الخلية حسب الاسم (H3)

الوصول إلى خلية محددة باستخدام اسمها. على سبيل المثال، للوصول إلى الخلية "C6":

```csharp
Cell cell = worksheet.Cells["C6"];
```
*لماذا هذه الخطوة؟*:يؤدي استخدام أسماء الخلايا إلى تحسين قابلية قراءة التعليمات البرمجية وإمكانية صيانتها.
