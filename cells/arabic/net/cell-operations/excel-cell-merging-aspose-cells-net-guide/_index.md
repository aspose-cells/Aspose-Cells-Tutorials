### دليل التنفيذ

#### دمج الخلايا في ورقة العمل

**ملخص:**
دمج الخلايا يُوحّد البيانات لتحسين قابلية القراءة والعرض. يرشدك هذا القسم إلى كيفية دمج خلايا مُحددة باستخدام Aspose.Cells.

1. **إنشاء مصنف جديد**
   ابدأ بإنشاء مثيل لـ `Workbook` الفئة التي تمثل ملف Excel.
   
   ```csharp
   Workbook workbook = new Workbook();
   ```

2. **الوصول إلى ورقة العمل**
   قم بالوصول إلى ورقة العمل الأولى من المصنف الخاص بك:
   
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **تعديل ودمج الخلايا**
   أضف قيمة إلى خلية محددة، ثم قم بدمج الخلايا في النطاق المطلوب.
   
   ```csharp
   // تعيين قيمة "A1"
   Cell cell = worksheet.Cells["A1"];
   cell.PutValue("Visit Aspose!");

   // دمج الخلايا من A1 إلى C1 (فهرس قائم على 0)
   worksheet.Cells.Merge(0, 0, 1, 3);
   ```

4. **احفظ مصنفك**
   احفظ المصنف بالتنسيق المطلوب:
   
   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "/merged_cells_output.xls", SaveFormat.Excel97To2003);
   ```

#### إنشاء الدلائل لعمليات الملفات

**ملخص:**
تأكد من وجود مجلد لحفظ ملفات Excel. تحقق من المجلدات وأنشئها إذا لم تكن موجودة.
