## دليل التنفيذ

### الميزة 1: تهيئة المصنف والتحقق من صحة البيانات لخلية واحدة

#### ملخص

تعلم كيفية تهيئة مصنف والتحقق من صحة البيانات في خلايا محددة باستخدام Aspose.Cells.

**الخطوة 1: استيراد المكتبات الضرورية**

تأكد من استيراد مكتبات Aspose.Cells المطلوبة:

```java
import com.aspose.cells.*;
```

**الخطوة 2: تهيئة المصنف**

قم بتحميل ملف Excel الخاص بك إلى كائن مصنف.

```csharp
Workbook workbook = new Workbook("YOUR_SOURCE_DIRECTORY/sampleDataValidationRules.xlsx");
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("C1");
```

**الخطوة 3: التحقق من صحة بيانات الخلية**

تحقق مما إذا كانت البيانات الموجودة في خلية معينة تلبي معايير التحقق.

```csharp
// القيمة 3 خارج نطاق التحقق (10 إلى 20)
cell.putValue(3);
System.out.println("Is 3 a Valid Value for this Cell: " + cell.getValidationValue());

// القيمة 15 تقع ضمن نطاق التحقق (10 إلى 20)
cell.putValue(15);
System.out.println("Is 15 a Valid Value for this Cell: " + cell.getValidationValue());

// القيمة 30 خارج نطاق التحقق (10 إلى 20)
cell.putValue(30);
System.out.println("Is 30 a Valid Value for this Cell: " + cell.getValidationValue());
```

### الميزة 2: التحقق من صحة البيانات لخلية أخرى ذات نطاق قواعد مختلف

#### ملخص

تطبيق قواعد التحقق من صحة البيانات المختلفة على خلية أخرى.

**الخطوة 1: تهيئة المصنف والخلية المستهدفة**

قم بتحميل المصنف وحدد خلية هدف جديدة:

```csharp
Workbook workbook2 = new Workbook("YOUR_SOURCE_DIRECTORY/sampleDataValidationRules.xlsx");
Worksheet worksheet2 = workbook2.getWorksheets().get(0);
Cell cell2 = worksheet2.getCells().get("D1");
```

**الخطوة 2: التحقق من صحة البيانات**

أدخل قيمة وتحقق مما إذا كانت تلبي معايير التحقق.

```csharp
// أدخل الرقم الكبير 12345678901 في الخلية D1، والذي يجب أن يجتاز التحقق بسبب نطاقه (من 1 إلى 999999999999)
cell2.putValue(12345678901);
System.out.println("Is 12345678901 a Valid Value for this Cell: " + cell2.getValidationValue());
```

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن ملف Excel الخاص بك يحتوي على قواعد التحقق الصحيحة.
- تأكد مرة أخرى من النطاق والمعايير المحددة في عمليات التحقق الخاصة بك.
