## دليل التنفيذ

دعونا نقسم تنفيذ التحقق من صحة بيانات الوقت إلى خطوات قابلة للإدارة.

### الخطوة 1: إنشاء المصنف وتكوينه

ابدأ بإنشاء مصنف Excel وتكوين ورقة العمل الأولى الخاصة به للتحضير للتحقق من الصحة:

**إنشاء وتكوين المصنف**
```csharp
// إنشاء مثيل جديد للمصنف
Workbook workbook = new Workbook();

// الوصول إلى ورقة العمل الأولى في المصنف
Cells cells = workbook.Worksheets[0].Cells;

// إعداد التعليمات للمستخدمين
cells["A1"].PutValue("Please enter Time b/w 09:00 and 11:30 'o Clock");

// ضبط ارتفاع الصف وعرض العمود لتحسين الرؤية
cells.SetRowHeight(0, 31);
cells.SetColumnWidth(0, 35);
```

### الخطوة 2: إضافة التحقق من صحة بيانات الوقت

تتضمن الوظيفة الأساسية إعداد قواعد التحقق من صحة البيانات لضمان وقوع إدخالات الوقت بين ساعات محددة.

**إضافة التحقق من الوقت**
```csharp
// الوصول إلى مجموعة التحقق من صحة ورقة العمل الأولى
ValidationCollection validations = workbook.Worksheets[0].Validations;

// تحديد منطقة الخلية للتحقق (الصف 0، العمود 1)
CellArea ca = new CellArea { StartRow = 0, EndRow = 0, StartColumn = 1, EndColumn = 1 };

// إضافة وتكوين التحقق من الوقت
Validation validation = validations[validations.Add(ca)];
validation.Type = ValidationType.Time;
validation.Operator = OperatorType.Between;
validation.Formula1 = "09:00";
validation.Formula2 = "11:30";

// تكوين رسائل الخطأ للإدخالات غير الصالحة
validation.ShowError = true;
validation.AlertStyle = ValidationAlertType.Information;
validation.ErrorTitle = "Time Error";
validation.ErrorMessage = "Enter a Valid Time";

// إعداد رسالة الإدخال وتجاهل الخلايا الفارغة
validation.InputMessage = "Time Validation Type";
validation.IgnoreBlank = true;
validation.ShowInput = true;

// إضافة منطقة التحقق للعمود 1
validation.AddArea(ca);
```
