### إعداد التحقق من التاريخ
أضف التحقق من التاريخ إلى ملف Excel الخاص بك باستخدام Aspose.Cells:

#### الخطوة 1: تحديد منطقة الخلية للتحقق
حدد منطقة الخلية التي تريد تطبيق التحقق عليها.
```csharp
// إنشاء منطقة خلية للتحقق
CellArea ca = new CellArea();
ca.StartRow = 0;
ca.EndRow = 0;
cStartColumn = 1; // استهداف العمود ب
ca.EndColumn = 1;
```

#### الخطوة 2: تكوين إعدادات التحقق
أضف إعدادات التحقق وقم بتكوينها للتأكد من قيام المستخدمين بإدخال التواريخ ضمن نطاق محدد.
```csharp
// احصل على مجموعة التحقق من الصحة من ورقة العمل
ValidationCollection validations = sheet.Validations;

// إضافة كائن التحقق الجديد إلى المجموعة
Validation validation = validations[validations.Add(ca)];

// تعيين نوع التحقق إلى التاريخ
validation.Type = ValidationType.Date;
validation.Operator = OperatorType.Between;
validation.Formula1 = "1/1/1970";  // تاريخ البدء
validation.Formula2 = "12/31/1999"; // تاريخ الانتهاء

// تمكين عرض الخطأ
validation.ShowError = true;
validation.AlertStyle = ValidationAlertType.Stop;

// تخصيص رسالة الخطأ
customize the validation.ErrorTitle to "Date Error";
validation.ErrorMessage = "Enter a Valid Date";

// اختياري: تعيين رسالة الإدخال للإرشاد
validation.InputMessage = "Please enter dates between 1/1/1970 and 12/31/1999";
validation.ShowInput = true;
```
