# تغيير نظام التاريخ في Excel إلى 1904 باستخدام Aspose.Cells .NET

## مقدمة

هل تواجه صعوبة في استخدام نظام التاريخ الافتراضي ١٨٩٩ في مصنفات Excel؟ غالبًا ما يكون التبديل إلى نظام التاريخ ١٩٠٤ ضروريًا للتوافق أو لتلبية متطلبات إقليمية محددة. سيرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Cells .NET لتغيير نظام التاريخ في مصنفك بسهولة.

### ما سوف تتعلمه:
- كيفية تحويل نظام التاريخ في Excel من 1899 إلى 1904.
- خطوات تحميل وحفظ مصنف Excel بالإعدادات الجديدة.
- الميزات الرئيسية لـ Aspose.Cells .NET للتعامل مع ملفات Excel.

لنستعرض كيفية تطبيق هذه التغييرات بسلاسة. تأكد من استيفاء جميع المتطلبات الأساسية قبل المتابعة.

## دليل التنفيذ

سنقوم بتقسيم هذا الدليل إلى قسمين رئيسيين استنادًا إلى الوظيفة.

### تغيير نظام تاريخ مصنف Excel

#### ملخص
تعمل هذه الميزة على تغيير نظام التاريخ الخاص بمصنف Excel من الافتراضي (1899) إلى 1904، وهو أمر ضروري للتوافق أو المتطلبات الإقليمية المحددة.

##### التنفيذ خطوة بخطوة:

**1. افتح ملف Excel**
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "sampleImplement1904DateSystem.xlsx");
```
هنا، `Workbook` يتم تهيئة المسار باستخدام مسار ملف موجود لتحميل مستند Excel الخاص بك.

**2. تغيير نظام التاريخ**
```csharp
workbook.Settings.Date1904 = true;
```
يضبط هذا السطر نظام التاريخ للمصنف إلى عام 1904 عن طريق تعديل `Date1904` ملكية.

**3. احفظ المصنف المحدث**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outputImplement1904DateSystem_1904DateSystem.xlsx");
```
يتم حفظ المصنف باسم جديد يعكس تكوين نظام التاريخ المحدث الخاص به.

### تحميل وحفظ المصنف

#### ملخص
تعرف على كيفية تحميل ملف Excel بكفاءة من دليل وحفظه في مكان آخر باستخدام Aspose.Cells.

##### التنفيذ خطوة بخطوة:

**1. افتح ملف Excel**
```csharp
Workbook workbook = new Workbook(SourceDir + "sampleImplement1904DateSystem.xlsx");
```
هذه الخطوة مشابهة لمثالنا السابق، حيث نفتح المصنف للتلاعب به.

**2. احفظ المصنف**
```csharp
workbook.Save(outputDir + "outputSaveWorkbook.xlsx");
```
هنا، يتم حفظ المصنف في موقع جديد باسم ملف محدد.
