## دليل التنفيذ

سوف يرشدك هذا القسم خلال إنشاء مراقب الحساب المخصص واستخدامه.

### إنشاء فئة مراقبة حسابات مخصصة

الهدف هنا هو إنشاء فئة تُقاطع حسابات الصيغ لخلايا مُحددة. لنبدأ بخطوات التنفيذ:

#### تحديد فئة مراقبة الحسابات المخصصة

ابدأ بالتعريف `clsCalculationMonitor`، وراثة من `AbstractCalculationMonitor`:

```csharp
using System;
using Aspose.Cells;

class clsCalculationMonitor : AbstractCalculationMonitor
{
    public override void BeforeCalculate(int sheetIndex, int rowIndex, int colIndex)
    {
        // تحويل مؤشرات الخلايا إلى اسم (على سبيل المثال، A1، B2)
        string cellName = CellsHelper.CellIndexToName(rowIndex, colIndex);

        // حساب المقاطعة للخلية المحددة "B8"
        if (cellName == "B8")
        {
            this.Interrupt("Interrupt/Cancel the formula calculation");
        }
    }
}
```

**توضيح:**
- **طريقة BeforeCalculate**: يتم استدعاؤه قبل حساب كل خلية. يتحقق مما إذا كانت الخلية الحالية `"B8"` ويقاطع حسابها.

### تكوين حساب صيغة المصنف باستخدام شاشة مخصصة

توضح هذه الميزة كيفية تحميل مصنف Excel، وتكوين خيارات الحساب المخصصة، وتنفيذ الصيغ باستخدام هذه الإعدادات.

#### تحميل المصنف وإعداد خيارات الحساب

```csharp
public static void Run()
{
    // تحديد دليل المصدر لملف Excel
    string SourceDir = @"YOUR_SOURCE_DIRECTORY";

    // تحميل ملف Excel
    Workbook wb = new Workbook(SourceDir + "sampleCalculationMonitor.xlsx");

    // إعداد خيارات الحساب باستخدام شاشة مخصصة
    CalculationOptions opts = new CalculationOptions();
    opts.CalculationMonitor = new clsCalculationMonitor();

    // حساب صيغ المصنف باستخدام الخيارات المحددة
    wb.CalculateFormula(opts);
}
```

**توضيح:**
- **تحميل المصنف**:يفتح ملف Excel من دليل محدد.
- **تعيين شاشة مخصصة**:يربط مراقب الحساب المخصص بخيارات الحساب.
- **طريقة حساب الصيغة**:تنفيذ كافة صيغ المصنف، مع الالتزام بمنطق المراقبة المخصص.
