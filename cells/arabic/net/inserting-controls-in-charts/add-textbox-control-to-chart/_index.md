# إضافة عنصر تحكم مربع النص إلى الرسم البياني

## مقدمة

إنشاء مخططات ديناميكية وجذابة بصريًا في Excel طريقة رائعة لعرض البيانات بفعالية. من الميزات الرائعة التي يمكنك استخدامها إضافة مربع نص إلى المخطط. مع Aspose.Cells لـ .NET، تصبح هذه المهمة سهلة وممتعة! في هذا الدليل، سنشرح لك عملية دمج مربع نص في مخططك خطوة بخطوة. سواء كنت مطورًا محترفًا أو مبتدئًا، سيوفر لك هذا البرنامج التعليمي جميع الأدوات اللازمة لتحسين مخططات Excel الخاصة بك. فهل أنت مستعد للبدء؟

## المتطلبات الأساسية

قبل أن ننتقل إلى البرمجة، هناك بعض الأشياء التي يجب أن تكون موجودة لديك:

- فهم أساسيات لغة C#: سيكون من المفيد فهم أساسيات برمجة C#. لا تقلق، لستَ بحاجة إلى أن تكون خبيرًا، يكفي أن تكون متمكنًا من قواعد اللغة.
- مكتبة Aspose.Cells المُثبّتة: تأكد من تثبيت مكتبة Aspose.Cells لـ .NET. يمكنك تنزيلها من [هنا](https://releases.aspose.com/cells/net/) إذا لم تكن قد فعلت ذلك بالفعل.
- Visual Studio: من الضروري أن تكون على دراية بـ Visual Studio أو أي بيئة تطوير متكاملة تفضل استخدامها لإطار عمل .NET.
- ملف إكسل موجود: في هذا المثال، سنعمل مع ملف إكسل موجود باسم "sampleAddingTextBoxControlInChart.xls". يمكنك إنشاء ملف أو تنزيل نموذج منه.

الآن بعد أن أصبح كل شيء في مكانه، دعنا ننتقل إلى جزء الترميز!

## استيراد الحزم

أولاً، علينا استيراد مساحات أسماء Aspose.Cells اللازمة إلى مشروع C#. يمكنك القيام بذلك بسهولة عن طريق تضمين الأسطر التالية في أعلى ملف الكود الخاص بك:

```csharp
using System;
using System.IO;

using Aspose.Cells;
using System.Drawing;
```

## الخطوة 1: تحديد دليل المصدر والإخراج

قبل البدء بالعمل على ملف Excel، من المهم تحديد مكان ملف الإدخال ومكان حفظ ملف الإخراج. هذا يُساعد في تنظيم مشروعك.

```csharp
// دليل المصدر
string sourceDir = "Your Document Directory";

// دليل الإخراج
string outputDir = "Your Output Directory";
```
يستبدل `"Your Document Directory"` و `"Your Output Directory"` مع المسارات الفعلية على نظامك.

## الخطوة 2: افتح ملف Excel الموجود

بعد ذلك، علينا فتح ملف إكسل الذي يحتوي على المخطط الذي نريد تعديله. سيسمح لنا هذا بجلب المخطط وإجراء التعديلات.

```csharp
// افتح الملف الموجود.
Workbook workbook = new Workbook(sourceDir + "sampleAddingTextBoxControlInChart.xls");
```
يقوم هذا السطر بتهيئة كائن مصنف جديد بالملف المحدد لدينا.

## الخطوة 3: الوصول إلى الرسم البياني في ورقة العمل

بما أن المخططات البيانية في Excel تُخزَّن داخل ورقة عمل، فعلينا أولاً الوصول إلى ورقة العمل ثم الحصول على المخطط المطلوب. في هذا المثال، سنصل إلى المخطط الأول في ورقة العمل الأولى.

```csharp
// احصل على مخطط المصمم في الورقة الأولى.
Worksheet sheet = workbook.Worksheets[0];
Aspose.Cells.Charts.Chart chart = sheet.Charts[0];
```
من خلال تغيير قيمة الفهرس، يمكنك تحديد أوراق عمل أو مخططات مختلفة إذا كان ملفك يحتوي على المزيد.

## الخطوة 4: إضافة مربع نص جديد إلى الرسم البياني

الآن، أصبحنا جاهزين لإضافة مربع النص. سنحدد موقعه وحجمه عند إنشائه.

```csharp
// أضف مربع نص جديد إلى الرسم البياني.
Aspose.Cells.Drawing.TextBox textbox0 = chart.Shapes.AddTextBoxInChart(400, 1100, 350, 2550);
```
في هذا الأمر، تُحدد المعلمات موقع (س، ص) وحجم (عرض، ارتفاع) مربع النص في المخطط. عدّل هذه القيم وفقًا لاحتياجاتك الخاصة في التخطيط.

## الخطوة 5: تعيين النص لمربع النص

بعد تثبيت مربع النص، حان وقت ملئه بالمحتوى. يمكنك إضافة أي نص تراه ضروريًا للمخطط.

```csharp
// إملأ النص.
textbox0.Text = "Sales By Region";
```
لا تتردد في استبدال "المبيعات حسب المنطقة" بأي نص ذي صلة ببياناتك.

## الخطوة 6: ضبط خصائص مربع النص

الآن، لنجعل مربع النص يبدو رائعًا! يمكنك تخصيص خصائص متنوعة، مثل لون الخط وحجمه ونمطه.

```csharp
// ضبط لون الخط.
textbox0.Font.Color = Color.Maroon; // تغيير إلى اللون المطلوب

// ضبط الخط إلى غامق.
textbox0.Font.IsBold = true;

// ضبط حجم الخط.
textbox0.Font.Size = 14;

// تعيين سمة الخط إلى مائل.
textbox0.Font.IsItalic = true;
```

يقوم كل من هذه الأسطر بتعديل مظهر النص داخل مربع النص الخاص بك، مما يعزز الرؤية والجاذبية.

## الخطوة 7: تنسيق مظهر مربع النص

من الضروري أيضًا تنسيق خلفية وحدود مربع النص، مما يجعله بارزًا على الرسم البياني.

```csharp
// احصل على تنسيق التعبئة لمربع النص.
Aspose.Cells.Drawing.FillFormat fillformat = textbox0.Fill;

// احصل على نوع تنسيق الخط لمربع النص.
Aspose.Cells.Drawing.LineFormat lineformat = textbox0.Line;

// ضبط وزن الخط.
lineformat.Weight = 2;

// تعيين نمط الشرطة إلى صلب.
lineformat.DashStyle = Aspose.Cells.Drawing.MsoLineDashStyle.Solid;
```

تتيح لك هذه الخيارات تعيين خلفية التعبئة لمربع النص وتخصيص حدوده.

## الخطوة 8: حفظ ملف Excel المعدّل

الخطوة الأخيرة هي حفظ التغييرات التي أجريتها على ملف Excel جديد. هذا يضمن بقاء ملفك الأصلي سليمًا.

```csharp
// احفظ ملف الاكسل.
workbook.Save(outputDir + "outputAddingTextBoxControlInChart.xls");
```
يستبدل `"outputAddingTextBoxControlInChart.xls"` مع أي اسم ملف تفضله.

## خاتمة

تهانينا! لقد نجحت في إضافة عنصر تحكم مربع نص إلى مخطط باستخدام Aspose.Cells لـ .NET. هذا التغيير البسيط والفعال يجعل مخططاتك أكثر إفادة وجاذبية بصريًا. يُعد تمثيل البيانات أساسًا للتواصل الفعال، وباستخدام أدوات مثل Aspose، يمكنك تحسين هذا العرض التقديمي بأقل جهد.
