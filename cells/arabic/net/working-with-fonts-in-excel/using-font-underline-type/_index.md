# استخدام نوع الخط المسطر في Excel

## مقدمة
عند إنشاء جداول البيانات أو التعامل مع ملفات Excel في تطبيقات .NET، تُعدّ الكفاءة وسهولة الاستخدام أمرًا بالغ الأهمية. تُعدّ Aspose.Cells for .NET مكتبة فعّالة تُمكّن المطورين من العمل مع ملفات Excel بسلاسة. في هذا البرنامج التعليمي، سنستكشف كيفية استخدام نوع الخط المُسطّر في Excel باستخدام Aspose.Cells. سنُقدّم تعليمات سهلة المتابعة خطوة بخطوة، لضمان استيعابك للمفاهيم وتطبيقها في مشاريعك الخاصة دون أي متاعب!
## المتطلبات الأساسية
قبل الغوص في أمثلة التعليمات البرمجية الخاصة بنا، هناك بعض المتطلبات الأساسية لضمان أن بيئة التطوير الخاصة بك جاهزة للعمل.
### المعرفة الأساسية بلغة C#
يجب أن يكون لديك فهم أساسي لبرمجة C#. كما أن الإلمام بمبادئ البرمجة كائنية التوجه سيساعدك على استيعاب هذه المفاهيم بشكل أفضل.
### تم تثبيت Visual Studio
لتشغيل واختبار شفرتك البرمجية بفعالية، يُعد تثبيت Visual Studio أمرًا ضروريًا. يمكنك تنزيله من [موقع مايكروسوفت](https://visualstudio.microsoft.com/).
### Aspose.Cells لـ .NET
تأكد من تثبيت مكتبة Aspose.Cells لـ .NET. يمكنك تنزيلها من [صفحة إصدارات Aspose](https://releases.aspose.com/cells/net/) أو استخدم NuGet Package Manager في Visual Studio.
### إطار عمل .NET
تأكد من تثبيت إطار عمل .NET المناسب في مشروعك. يدعم Aspose.Cells إصدارات مختلفة؛ راجع وثائقه للتأكد من توافقها.
مع توفر هذه المتطلبات الأساسية، ستكون جاهزًا لإنشاء مستند Excel الأول الخاص بك بنص مسطر!
## استيراد الحزم
للبدء، ستحتاج إلى استيراد بعض مساحات الأسماء الأساسية إلى مشروع C# الخاص بك. إليك كيفية القيام بذلك:
```csharp
using System.IO;
using Aspose.Cells;
```
سيؤدي تضمين هذه المساحات إلى تمكينك من الوصول إلى جميع الفئات والطرق التي ستحتاجها للعمل مع ملفات Excel باستخدام Aspose.Cells.

الآن بعد أن قمنا بإعداد كل شيء، دعنا نقوم بتقسيم كل جانب من جوانب الكود المطلوب لتسطير النص في خلية Excel.
## الخطوة 1: إعداد دليل المستندات الخاص بك
قبل أي شيء، ستحتاج إلى مكان على محرك الأقراص لحفظ ملفات Excel. إليك كيفية إنشاء هذا المجلد:
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
يتحقق هذا المقطع من وجود الدليل المحدد. إذا لم يكن موجودًا، فسينشئه لك. استبدل `"Your Document Directory"` مع المسار الذي تريده.
## الخطوة 2: إنشاء كائن مصنف
بعد ذلك، ستحتاج إلى إنشاء نسخة جديدة من مصنف، وهو في الأساس ملف Excel الخاص بك. إليك الطريقة:
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
هذا السطر يُنشئ مُصنّف عمل جديد. تخيّل الأمر كما لو أنك تفتح صفحةً بيضاء حيث يمكنك البدء في صياغة تحفتك الفنية.
## الخطوة 3: إضافة ورقة عمل جديدة
بعد حصولك على مصنفك، ستحتاج إلى ورقة عمل للعمل عليها. لنُضِف واحدة:
```csharp
// إضافة ورقة عمل جديدة إلى كائن Excel
int i = workbook.Worksheets.Add();
```
يؤدي هذا إلى إضافة ورقة عمل جديدة إلى المصنف الخاص بك وتخزين فهرس الورقة المضافة حديثًا في المتغير `i`.
## الخطوة 4: الرجوع إلى ورقة العمل الجديدة
الآن، عليك الحصول على مرجع لورقة العمل التي أضفتها للتو. هذا يسمح لك بالتحكم بها:
```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[i];
```
من خلال هذه الخطوة، يمكنك توجيه الكود الخاص بك مباشرةً إلى ورقة العمل الجديدة، لتكون جاهزًا لإضافة المحتوى.
## الخطوة 5: الوصول إلى خلية محددة
الآن حان وقت تحديد مكان النص. في هذه الحالة، سنستخدم الخلية A1:
```csharp
// الوصول إلى الخلية "A1" من ورقة العمل
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
```
هنا، نقوم بالتقاط الخلية الموجودة في الموضع A1 حتى نتمكن من إدراج بعض النص.
## الخطوة 6: إضافة قيمة إلى الخلية
دعونا نضع بعض المحتوى في تلك الخلية:
```csharp
// إضافة بعض القيمة إلى الخلية "A1"
cell.PutValue("Hello Aspose!");
```
في هذه المرحلة، أصبح "مرحبًا Aspose!" محتوى الخلية A1. بسيط، أليس كذلك؟
## الخطوة 7: الحصول على نمط الخلية
لتسطير النص، ستحتاج إلى الوصول إلى خصائص نمطه. إليك كيفية استرجاع النمط الحالي للخلية:
```csharp
// الحصول على نمط الخلية
Style style = cell.GetStyle();
```
يقوم هذا السطر بجلب النمط الموجود المطبق على الخلية، مما يسمح لك بتعديله.
## الخطوة 8: ضبط الخط ليكون مسطرًا
والآن يأتي الجزء المثير! لنُحدِّث نمط الخط:
```csharp
// ضبط الخط المراد تسطيره
style.Font.Underline = FontUnderlineType.Single;
```
يُغيّر هذا خاصية تسطير الخط إلى تسطير واحد. يمكنك أيضًا استكشاف أنواع أخرى، ولكن في الوقت الحالي، لنجعل الأمر بسيطًا!
## الخطوة 9: تطبيق النمط على الخلية
لا يمكنك التوقف في منتصف الطريق! الآن عليك إعادة ضبط هذا النمط المُحدّث على جهازك:
```csharp
// تطبيق النمط على الخلية
cell.SetStyle(style);
```
ها هو! الآن، تعكس الخلية النمط الجديد بنص مسطر.
## الخطوة 10: حفظ المصنف
وأخيرًا، دعنا نحفظ تحفتك الفنية في ملف Excel:
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```
يحفظ هذا السطر المصنف بتنسيق Excel 97-2003. تأكد من ضبط اسم الملف ومساره بشكل صحيح في المكان الذي تريد حفظ الملف فيه.
## خاتمة
كما رأيت، العمل مع Aspose.Cells لـ .NET ليس قويًا فحسب، بل سهل الاستخدام أيضًا، مما يتيح لك إنشاء ملفات Excel ومعالجتها بسهولة. تسطير النص في الخلية ليس سوى غيض من فيض إمكانيات هذه المكتبة. سواء كنت تُنشئ تقارير معقدة أو تُدير مجموعات بيانات ضخمة، تُزودك Aspose.Cells بالأدوات اللازمة للنجاح في تطبيقات .NET.
