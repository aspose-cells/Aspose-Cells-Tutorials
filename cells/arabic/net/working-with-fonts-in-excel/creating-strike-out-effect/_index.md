# إنشاء تأثير الشطب على النص في Excel

## مقدمة
في برنامج إكسل، تُعدّ العناصر المرئية بنفس أهمية البيانات نفسها. سواءً كنت تُبرز تغييرات مهمة أو تُعلّم عناصر لم تعد ذات صلة، يُعدّ تأثير الشطب على النص طريقةً تقليديةً لإدارة التمثيل المرئي في جداول البيانات. في هذا الدليل، سنشرح لك عملية تطبيق تأثير الشطب على النص في إكسل باستخدام Aspose.Cells لـ .NET. لن يقتصر هذا البرنامج التعليمي على تغطية المتطلبات الأساسية فحسب، بل سيُقدّم أيضًا نهجًا خطوة بخطوة لضمان سهولة تطبيق هذا التأثير.
## المتطلبات الأساسية
قبل الغوص في البرنامج التعليمي، تأكد من استيفاء المتطلبات الأساسية التالية:
1. بيئة التطوير: يجب أن تكون لديك بيئة تطوير .NET مُعدّة. قد تكون هذه بيئة Visual Studio أو أي بيئة تطوير متكاملة أخرى تُفضّلها وتدعم تطوير .NET.
2. Aspose.Cells لـ .NET: تأكد من تثبيت Aspose.Cells في مشروعك. يمكنك تنزيله من الرابط التالي: [تنزيل Aspose.Cells](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: إن الفهم الأساسي لبرمجة C# مفيد حيث سيتم برمجة الأمثلة بلغة C#.
4. .NET Framework: تأكد من أن مشروعك يستهدف إصدار .NET Framework متوافق، عادةً .NET Core أو .NET Framework 4.5 وما فوق.
## استيراد الحزم
قبل كتابة أي شيفرة برمجية، عليك استيراد مساحات الأسماء المطلوبة من Aspose.Cells. هذا ضروري للوصول إلى الميزات المختلفة التي توفرها المكتبة. إليك كيفية استيراد مساحات الأسماء اللازمة:
```csharp
using System.IO;
using Aspose.Cells;
```
باستخدام هذه الواردات، سيكون لديك حق الوصول إلى فئات Workbook وWorksheet وStyle التي سيتم استخدامها خلال هذا البرنامج التعليمي.
بعد أن هيأنا المشهد، لنُقسّم العملية إلى خطوات سهلة. ستُرفق بكل خطوة تعليمات واضحة لإرشادك خلال إنشاء تأثير الشطب على النص في إكسل.
## الخطوة 1: تحديد دليل المستندات
ابدأ بتحديد المسار الذي ستُخزَّن فيه مستندات Excel. سيكون هذا هو موقع حفظ ملفات الإخراج.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
```
يستبدل `"Your Document Directory"` مع مسار الدليل الفعلي الذي تريد حفظ ملف Excel فيه. هذا يُهيئ الدليل لإخراجك.
## الخطوة 2: إنشاء الدليل
بعد ذلك، تأكد من وجود الدليل الذي حددته في الخطوة السابقة. إذا لم يكن موجودًا، يمكنك إنشاؤه برمجيًا.
```csharp
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
يتحقق هذا الكود من وجود الدليل، ويُنشئه إن لم يكن. يُساعد هذا في تجنب الأخطاء عند محاولة حفظ الملف لاحقًا.
## الخطوة 3: إنشاء كائن مصنف
الآن، حان وقت إنشاء مصنف جديد. هذا هو أساس ملف Excel الذي ستُضيف إليه البيانات وتُطبّق التنسيقات.
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
ال `Workbook` تُمثل الفئة ملف Excel. بإنشاء مثيل لهذه الفئة، فأنت تُنشئ مستند Excel جديدًا.
## الخطوة 4: إضافة ورقة عمل جديدة
يمكن أن يحتوي كل مصنف على عدة أوراق عمل. لنبدأ بإنشاء ورقة عمل جديدة في مصنفك.
```csharp
// إضافة ورقة عمل جديدة إلى كائن Excel
int i = workbook.Worksheets.Add();
```
ال `Add` طريقة `Worksheets` تضيف المجموعة ورقة عمل جديدة إلى المصنف وتعيد الفهرس الخاص بها. 
## الخطوة 5: الحصول على مرجع ورقة العمل الجديدة
بمجرد إنشاء ورقة العمل، ستحتاج إلى الرجوع إليها للعمليات المستقبلية.
```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[i];
```
هنا، يمكنك جلب ورقة العمل التي تم إنشاؤها حديثًا باستخدام الفهرس الخاص بها (`i`). هذا يتيح لك إمكانية التحكم في ورقة العمل.
## الخطوة 6: الوصول إلى الخلية
ستحتاج إلى الوصول إلى خلية محددة في ورقة العمل لتطبيق تنسيق الشطب. في هذا المثال، نستخدم الخلية `A1`.
```csharp
// الوصول إلى الخلية "A1" من ورقة العمل
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
```
في Excel، يُشار إلى الخلايا بمعرفات أعمدتها وصفوفها (مثل "A1"). نحصل على مرجع للخلية `A1` لمزيد من التلاعب.
## الخطوة 7: إضافة قيمة إلى الخلية
الآن، لنُدخل نصًا في الخلية. سنكتب "مرحبًا Aspose!" في الخلية `A1`.
```csharp
// إضافة بعض القيمة إلى الخلية "A1"
cell.PutValue("Hello Aspose!");
```
ال `PutValue` تُستخدم هذه الطريقة لتعيين قيمة نصية للخلية. يمكنك تعديل هذه القيمة لعرض أي قيمة.
## الخطوة 8: الحصول على نمط الخلية
الآن بعد أن أصبح لدينا نص في الخلية، حان الوقت للوصول إلى نمط الخلية لتطبيق التنسيق المطلوب، بما في ذلك تأثير الشطب.
```csharp
// الحصول على نمط الخلية
Style style = cell.GetStyle();
```
ال `GetStyle` تسترجع الطريقة النمط الحالي للخلية، مما يسمح لك بتعديل خصائص مثل نوع الخط وحجمه والتأثيرات.
## الخطوة 9: ضبط تأثير الشطب
لنُطبّق تأثير الشطب على النص في الخلية. سنُعدّل نمط خطّ الخلية.
```csharp
// ExStart:SetStrikeout
// ضبط تأثير الشطب على الخط
style.Font.IsStrikeout = true;
// ExEnd:SetStrikeout
```
عن طريق الإعداد `IsStrikeout` إذا كنت تريد تعيين هذا الخيار على true، فأنت تطلب من Excel أن يشطب بصريًا النص الموجود في الخلية المحددة - تمامًا مثل وضع علامة بصريًا على شيء ما في قائمة.
## الخطوة 10: تطبيق النمط على الخلية
بعد تعديل النمط، ستحتاج إلى تطبيقه مرة أخرى على الخلية ليعكس التغييرات.
```csharp
// تطبيق النمط على الخلية
cell.SetStyle(style);
```
ال `SetStyle` تقوم الطريقة بتحديث الخلية بالنمط الجديد، والذي يتضمن الآن تنسيق الشطب.
## الخطوة 11: حفظ ملف Excel
أخيرًا، حان وقت حفظ مصنفك في المجلد المحدد. في هذا المثال، نحفظ الملف باسم `book1.out.xls`.
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```
ال `Save` تكتب هذه الطريقة المصنف على القرص بتنسيق Excel 97-2003. يمكنك تحديد تنسيقات مختلفة إذا لزم الأمر.
## خاتمة
إنشاء تأثير شطب على النص في Excel باستخدام Aspose.Cells لـ .NET عملية سهلة وبسيطة، إذا شرحتها خطوة بخطوة. باتباع هذا الدليل، ستمتلك الآن المهارات اللازمة لتحسين جداول البيانات الخاصة بك باستخدام الإشارات البصرية، مما يجعل بياناتك ليس فقط غنية بالمعلومات، بل جذابة بصريًا أيضًا.
