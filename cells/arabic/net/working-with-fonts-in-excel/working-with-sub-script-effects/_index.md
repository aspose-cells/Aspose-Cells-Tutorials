# العمل مع تأثيرات النص الفرعي في Excel

## مقدمة
في برنامج إكسل، يُحدث التنسيق فرقًا كبيرًا في طريقة عرض بياناتك. ومن أساليب التنسيق التي غالبًا ما تُغفل، ولكنها تُعزز وضوح معلوماتك، تأثير المؤشر السفلي. يُعد هذا مفيدًا بشكل خاص للصيغ الكيميائية، والتعابير الرياضية، وحتى الحواشي السفلية. في هذا البرنامج التعليمي، سنستكشف كيفية تطبيق تنسيق المؤشر السفلي على خلايا مصنف إكسل باستخدام Aspose.Cells لـ .NET.
## المتطلبات الأساسية
قبل الغوص في البرنامج التعليمي، دعنا نتأكد من إعداد كل شيء لرحلة سلسة:
1. Aspose.Cells لـ .NET: تأكد من تثبيت مكتبة Aspose.Cells. إذا لم تكن قد قمت بذلك، يمكنك تنزيلها بسهولة من [رابط تحميل Aspose Cells](https://releases.aspose.com/cells/net/).
2. Visual Studio: ستحتاج إلى تثبيت Visual Studio أو أي .NET IDE متوافق لتشغيل عينات التعليمات البرمجية.
3. المعرفة الأساسية بلغة C#: ستكون المعرفة بلغة C# وبرمجة .NET مفيدة، على الرغم من أننا سنقوم بتقسيم الكود لتسهيل متابعته.
4. بيئة العمل: قم بإعداد دليل جاهز لحفظ ملفات الإخراج الخاصة بك، وتأكد من أن لديك أذونات الكتابة لهذا الموقع.
بعد استيفاء هذه الشروط الأساسية، فلنبدأ العمل!
## استيراد الحزم
لبدء استخدام Aspose.Cells، عليك استيراد مساحات الأسماء ذات الصلة. إليك كيفية القيام بذلك:
### إنشاء مشروع جديد
افتح بيئة التطوير المتكاملة (IDE) وأنشئ مشروع C# جديدًا. يمكنك اختيار تطبيق وحدة تحكم أو تطبيق نماذج Windows، حسب تفضيلاتك. في هذا البرنامج التعليمي، يعمل تطبيق وحدة التحكم بشكل مثالي.
### أضف مرجع Aspose.Cells
بعد ذلك، أضف مرجعًا إلى مكتبة Aspose.Cells في مشروعك. يمكنك القيام بذلك عبر مدير حزم NuGet:
- انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- حدد "إدارة حزم NuGet".
- بحث عن `Aspose.Cells` وتثبيته.
### استيراد مساحة الاسم
في أعلى ملف البرنامج الرئيسي الخاص بك (عادةً `Program.cs`), تتضمن مساحة الأسماء التالية:
```csharp
using System;
using System.IO;
using Aspose.Cells;
```
الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الكود!
## الخطوة 1: إعداد دليل الإخراج الخاص بك
أولاً، علينا تحديد مكان حفظ ملف إكسل الناتج. هذه الخطوة بسيطة ولكنها بالغة الأهمية.
```csharp
// دليل الإخراج
string outputDir = "Your Document Directory\\";
```
يستبدل `"Your Document Directory\\"` مع مسار الدليل الفعلي. هذا هو المكان الذي سيتم فيه تخزين ملف Excel المُنشأ.
## الخطوة 2: إنشاء كائن مصنف
بعد ذلك، سنقوم بإنشاء مثيل لـ `Workbook` هذه الفئة تمثل ملف Excel وتسمح لنا بالتعامل معه بسهولة.
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
عندما تقوم بإنشاء ملف جديد `Workbook`، فإنه يقوم تلقائيًا بإنشاء ملف Excel جديد باستخدام ورقة عمل واحدة.
## الخطوة 3: الوصول إلى ورقة العمل
الآن وقد أصبح لدينا مصنف العمل، لننتقل إلى ورقة العمل التي نريد إجراء التغييرات عليها. في هذه الحالة، سنعمل على ورقة العمل الأولى.
```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[0];
```
## الخطوة 4: الوصول إلى الخلية
بعد الحصول على ورقة العمل، حان الوقت للوصول إلى خلية محددة لتطبيق تنسيق المؤشر السفلي. سنستخدم الخلية "A1" في هذا المثال.
```csharp
// الوصول إلى الخلية "A1" من ورقة العمل
Cell cell = worksheet.Cells["A1"];
```
## الخطوة 5: إضافة قيمة إلى الخلية
قبل تنسيق الخلية، لنُدخل نصًا فيها. في هذه الحالة، سنكتب ببساطة "مرحبًا".
```csharp
// إضافة بعض القيمة إلى الخلية "A1"
cell.PutValue("Hello");
```
## الخطوة 6: اضبط الخط على منخفض
الآن يأتي الجزء الممتع! سنُعدِّل نمط خط الخلية لجعله خطًا سفليًا. وهنا يأتي السحر.
```csharp
// ضبط الخط السفلي
Style style = cell.GetStyle();
style.Font.IsSubscript = true;
cell.SetStyle(style);
```
في الكود أعلاه، نقوم أولاً باسترجاع النمط الحالي للخلية باستخدام `GetStyle()`. ثم قمنا بضبط `IsSubscript` ممتلكات `Font` الاعتراض على `true`وأخيرًا، نطبق هذا النمط المعدّل على الخلية مرة أخرى.
## الخطوة 7: حفظ ملف Excel
بعد تطبيق تأثير المؤشر السفلي، علينا حفظ التغييرات في ملف Excel. إليك الطريقة:
```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputSettingSubscriptEffect.xlsx");
```
تأكد من أن المسار الذي تقدمه صحيح حتى يتم حفظ الملف دون أي مشاكل.
## الخطوة 8: تأكيد التنفيذ الناجح
للتأكد من أن كل شيء يسير بسلاسة، يمكننا طباعة رسالة إلى وحدة التحكم.
```csharp
Console.WriteLine("SettingSubscriptEffect executed successfully.\r\n");
```
تؤكد هذه الرسالة البسيطة أن الكود الخاص بنا تم تنفيذه دون أي مشاكل.
