# العمل مع تأثيرات النص الفرعي في Excel

## مقدمة
عندما يتعلق الأمر ببرنامج Excel، يمكن أن يحدث التنسيق فرقًا كبيرًا في كيفية عرض بياناتك. أحد أنماط التنسيق التي غالبًا ما تمر دون أن يلاحظها أحد ولكنها يمكن أن تعزز وضوح معلوماتك هو تأثير الخط السفلي. وهذا مفيد بشكل خاص للصيغ الكيميائية أو التعبيرات الرياضية أو حتى الحواشي السفلية. في هذا البرنامج التعليمي، سنستكشف كيفية تطبيق تنسيق الخط السفلي على الخلايا في مصنف Excel باستخدام Aspose.Cells for .NET.
## المتطلبات الأساسية
قبل الخوض في البرنامج التعليمي، دعنا نتأكد من إعداد كل شيء لرحلة سلسة:
1. Aspose.Cells لـ .NET: تأكد من تثبيت مكتبة Aspose.Cells. إذا لم تقم بذلك، فيمكنك تنزيلها بسهولة من[رابط تحميل برنامج Aspose Cells](https://releases.aspose.com/cells/net/).
2. Visual Studio: ستحتاج إلى تثبيت Visual Studio أو أي .NET IDE متوافق لتشغيل عينات التعليمات البرمجية.
3. المعرفة الأساسية بلغة C#: ستكون المعرفة بلغة C# وبرمجة .NET مفيدة، على الرغم من أننا سنقوم بتقسيم الكود لتسهيل متابعته.
4. بيئة العمل: قم بإعداد دليل جاهز لحفظ ملفات الإخراج الخاصة بك، وتأكد من حصولك على أذونات الكتابة لهذا الموقع.
بعد استيفاء هذه الشروط الأساسية، فلنبدأ العمل!
## استيراد الحزم
للبدء في استخدام Aspose.Cells، تحتاج إلى استيراد المساحات ذات الصلة. وإليك كيفية القيام بذلك:
### إنشاء مشروع جديد
افتح بيئة التطوير المتكاملة الخاصة بك وقم بإنشاء مشروع C# جديد. يمكنك اختيار تطبيق وحدة التحكم أو تطبيق نماذج Windows، حسب تفضيلاتك. في هذا البرنامج التعليمي، يعمل تطبيق وحدة التحكم بشكل مثالي.
### إضافة مرجع Aspose.Cells
بعد ذلك، أضف مرجعًا إلى مكتبة Aspose.Cells في مشروعك. يمكنك القيام بذلك عبر NuGet Package Manager:
- انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- حدد "إدارة حزم NuGet".
-  بحث عن`Aspose.Cells` وتثبيته.
### استيراد مساحة الاسم
 في الجزء العلوي من ملف البرنامج الرئيسي الخاص بك (عادةً`Program.cs`), تتضمن مساحة الأسماء التالية:
```csharp
using System;
using System.IO;
using Aspose.Cells;
```
الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الكود!
## الخطوة 1: إعداد دليل الإخراج الخاص بك
أولاً، نحتاج إلى تحديد المكان الذي سيتم فيه حفظ ملف Excel الناتج. هذه الخطوة بسيطة ولكنها بالغة الأهمية.
```csharp
// دليل الإخراج
string outputDir = "Your Document Directory\\";
```
 يستبدل`"Your Document Directory\\"` مع مسار الدليل الفعلي الخاص بك. هذا هو المكان الذي سيتم فيه تخزين ملف Excel الناتج.
## الخطوة 2: إنشاء كائن مصنف
 بعد ذلك، سنقوم بإنشاء مثيل لـ`Workbook` هذه الفئة تمثل ملف Excel وتسمح لنا بالتعامل معه بسهولة.
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
 عندما تقوم بإنشاء ملف جديد`Workbook`، فإنه يقوم تلقائيًا بإنشاء ملف Excel جديد يحتوي على ورقة عمل واحدة.
## الخطوة 3: الوصول إلى ورقة العمل
الآن بعد أن أصبح لدينا المصنف، فلننتقل إلى ورقة العمل التي نريد إجراء التغييرات عليها. في هذه الحالة، سنعمل على ورقة العمل الأولى.
```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[0];
```
## الخطوة 4: الوصول إلى الخلية
بمجرد حصولنا على ورقة العمل، حان الوقت للوصول إلى خلية معينة حيث سنطبق تنسيق الخط السفلي. سنستخدم الخلية "A1" لهذا المثال.
```csharp
// الوصول إلى الخلية "A1" من ورقة العمل
Cell cell = worksheet.Cells["A1"];
```
## الخطوة 5: إضافة قيمة إلى الخلية
قبل تنسيق الخلية، دعنا ندرج بعض النصوص فيها. في هذه الحالة، سنكتب ببساطة "Hello".
```csharp
// إضافة بعض القيمة إلى الخلية "A1"
cell.PutValue("Hello");
```
## الخطوة 6: اضبط الخط ليصبح منخفضًا
الآن يأتي الجزء الممتع! سنقوم بتعديل نمط الخط في الخلية لجعله خطًا سفليًا. وهنا يحدث السحر.
```csharp
// ضبط الخط السفلي
Style style = cell.GetStyle();
style.Font.IsSubscript = true;
cell.SetStyle(style);
```
 في الكود أعلاه، نقوم أولاً باسترجاع النمط الحالي للخلية باستخدام`GetStyle()` . ثم قمنا بتعيين`IsSubscript` ممتلكات`Font` الاعتراض على`true`وأخيرًا، نطبق هذا النمط المعدّل مرة أخرى على الخلية.
## الخطوة 7: حفظ ملف Excel
بعد تطبيق تأثير الخط السفلي، نحتاج إلى حفظ التغييرات التي أجريناها في ملف Excel. إليك كيفية القيام بذلك:
```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputSettingSubscriptEffect.xlsx");
```
تأكد من أن المسار الذي قمت بتوفيره صحيح حتى يتم حفظ الملف دون أي مشاكل.
## الخطوة 8: تأكيد التنفيذ الناجح
للتأكد من أن كل شيء يسير بسلاسة، يمكننا طباعة رسالة على وحدة التحكم.
```csharp
Console.WriteLine("SettingSubscriptEffect executed successfully.\r\n");
```
تؤكد هذه الرسالة البسيطة أن الكود الخاص بنا تم تنفيذه دون أي عوائق.

والآن، لقد نجحت في إنشاء ملف Excel بتأثيرات الخط السفلي باستخدام Aspose.Cells for .NET. تسهل هذه المكتبة القوية التعامل مع ملفات Excel، مما يمنحك قدرًا كبيرًا من المرونة والتحكم في عرض البيانات. باستخدام تنسيق الخط السفلي، يمكنك جعل جداول Excel الخاصة بك أكثر إفادة وجاذبية بصريًا.
