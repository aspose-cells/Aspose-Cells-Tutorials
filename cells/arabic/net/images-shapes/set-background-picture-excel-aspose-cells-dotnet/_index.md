## دليل التنفيذ

دعونا نقسم التنفيذ إلى خطوات واضحة.

### إعداد هيكل مشروعك

قبل الغوص في الكود، تأكد من تنظيم دليل المشروع الخاص بك بالصور ومجلدات الإخراج الضرورية.

#### تعريف الدلائل

إعداد أدلة المصدر والإخراج في ملف C# الخاص بك:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

### إضافة صورة خلفية إلى ورقة Excel

إليك كيفية تعيين صورة خلفية لورقة العمل الأولى.

#### الخطوة 1: قم بتحميل المصنف وورقة عمل Access

ابدأ بإنشاء مثيل `Workbook` الكائن والوصول إلى ورقة العمل المطلوبة:

```csharp
// إنشاء مصنف جديد.
Workbook workbook = new Workbook();

// احصل على ورقة العمل الأولى.
Worksheet sheet = workbook.Worksheets[0];
```

#### الخطوة 2: تعيين صورة الخلفية

قراءة ملف الصورة كبايتات وتعيينه إلى ورقة العمل `BackgroundImage` ملكية:

```csharp
// تعيين صورة الخلفية للورقة.
sheet.BackgroundImage = File.ReadAllBytes(SourceDir + "/background.jpg");
```

تأكد من فاصل المسار الخاص بك (`/`) يتوافق مع نظام التشغيل الخاص بك (استخدم `\` (للويندوز).

#### الخطوة 3: احفظ مصنفك

وأخيرًا، احفظ المصنف بتنسيقي Excel وHTML:

```csharp
// احفظ ملف Excel.
workbook.Save(OutputDir + "/outputBackImageSheet.xlsx");

// احفظ ملف HTML.
workbook.Save(OutputDir + "/outputBackImageSheet.html", SaveFormat.Html);
```

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن مسار الصورة صحيح ويمكن الوصول إليه.
- تأكد من أن مشروعك لديه أذونات القراءة والكتابة المناسبة للمجلدات.
