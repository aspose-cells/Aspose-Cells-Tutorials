# كيفية إدراج صورة مرتبطة في ملف Excel باستخدام Aspose.Cells .NET

**ما سوف تتعلمه:**
- إدراج الصور المرتبطة بالويب في ملفات Excel.
- تكوين أبعاد الصورة.
- حفظ المصنف المعدل بكفاءة.

هل أنت مستعد لتحسين مشاريعك في Excel؟ لنبدأ بإعداد بيئتك!

## دليل التنفيذ

اتبع الخطوات التالية لإدراج صورة مرتبطة في ورقة Excel باستخدام Aspose.Cells لـ .NET.

### إدراج صورة مرتبطة

#### ملخص
أضف صورًا من عناوين الويب مباشرةً إلى ورقة عمل Excel. تتيح هذه الميزة تحديثات ديناميكية دون تضمين موارد ثابتة.

#### التنفيذ خطوة بخطوة

**1. إعداد دليل الإخراج**
حدد المكان الذي سيتم حفظ ملف الإخراج الخاص بك فيه:

```csharp
string outputDir = RunExamples.Get_OutputDirectory();
```

**2. تهيئة المصنف وورقة العمل**
إنشاء جديد `Workbook` الكائن والوصول إلى ورقة العمل الأولى:

```csharp
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

**3. إضافة صورة مرتبطة**
استخدم `AddLinkedPicture` طريقة تضمين صورة من عنوان URL للويب في الخلية B2 (1، 1 يعتمد على الفهرس):

```csharp
Aspose.Cells.Drawing.Picture pic = sheet.Shapes.AddLinkedPicture(1, 1, 100, 100, "http://www.aspose.com/Images/aspose-logo.jpg");
```
- **المعلمات موضحة:**
  - `row`: مؤشر الصف (على أساس 0)
  - `column`: فهرس العمود (يعتمد على 0)
  - `width`:عرض الصورة بالنقاط
  - `height`:ارتفاع الصورة بالنقاط
  - `webAddress`:رابط الصورة

**4. تكوين أبعاد الصورة**
ضبط الحجم باستخدام البوصات:

```csharp
pic.HeightInch = 1.04;
pic.WidthInch = 2.6;
```

**5. احفظ المصنف**
حفظ المصنف في الدليل المحدد:

```csharp
workbook.Save(outputDir + "outputInsertLinkedPicture.xlsx");
```

### نصائح استكشاف الأخطاء وإصلاحها
- **روابط الصور المكسورة:** تأكد من أن عنوان الويب الخاص بك صحيح ويمكن الوصول إليه.
- **الصورة لا يتم عرضها:** تأكد من أن Aspose.Cells يقوم بتحديث الصور المرتبطة بشكل صحيح.
