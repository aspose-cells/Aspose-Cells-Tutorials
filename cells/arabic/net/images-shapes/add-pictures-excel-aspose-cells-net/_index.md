### إضافة ورقة عمل وصورة جديدة

#### ملخص
تتيح لك هذه الميزة إدراج صورة في خلية محددة في ورقة العمل الخاصة بك، مما يعزز عرض البيانات.

#### التنفيذ خطوة بخطوة

**1. قم بإعداد مشروعك:**
تأكد من إضافة Aspose.Cells كتبعية في مشروعك.

**2. إنشاء المصنف أو الوصول إليه:**
```csharp
// إنشاء كائن مصنف جديد
Workbook workbook = new Workbook();
```

**3. إضافة ورقة عمل جديدة:**
```csharp
// إضافة ورقة عمل جديدة إلى المصنف
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

**4. إدراج الصورة في الموقع المطلوب:**
هنا نضيف صورة تقع في "logo.jpg" إلى الخلية F6.
```csharp
// حدد المسار إلى ملف صورتك
string dataDir = RunExamples.GetDataDir(typeof(AddingPictures));

// أضف الصورة إلى ورقة العمل في الموضع (5، 5) المقابل للخلية 'F6'
worksheet.Pictures.Add(5, 5, dataDir + "logo.jpg");
```

**5. احفظ مصنفك:**
```csharp
// احفظ المصنف بالصورة المضافة
workbook.Save(dataDir + "output.xls");
```
