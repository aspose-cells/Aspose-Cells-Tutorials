## دليل التنفيذ
### الميزة 1: تحميل مصنف Excel ومعالجته
#### ملخص
يوضح هذا القسم كيفية تحميل مصنف موجود، والوصول إلى أوراق عمل محددة، وتعديل كائنات مربع النص داخل تلك الأوراق.

#### تعليمات خطوة بخطوة
**الخطوة 1: تحميل المصنف**
ابدأ بتحميل مصنف المصدر الخاص بك باستخدام مسار الملف الخاص به:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "book1.xls");
```
*توضيح*: ال `Workbook` تُستخدم الفئة لفتح ملفات Excel ومعالجتها. هنا، يتم تحميل ملف موجود باسم `book1.xls`.

**الخطوة 2: الوصول إلى ورقة العمل**
الوصول إلى ورقة العمل الأولى داخل المصنف:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
*توضيح*يمكن الوصول إلى أوراق العمل من خلال فهرسها أو اسمها. في هذا المثال، نصل إلى الورقة الأولى.

**الخطوة 3: التعامل مع كائنات مربع النص**
الوصول إلى كائنات مربع النص وتعديلها حسب الحاجة:
```csharp
Aspose.Cells.Drawing.TextBox textbox0 = worksheet.TextBoxes[0];
string text0 = textbox0.Text; // استرجاع النص الموجود

Aspose.Cells.Drawing.TextBox textbox1 = worksheet.TextBoxes[1];
textbox1.Text = "This is an alternative text"; // تعديل النص
```
*توضيح*:يتم الوصول إلى مربعات النص بنفس طريقة الوصول إلى أوراق العمل. يمكنك قراءة أو ضبط إعداداتها. `Text` ملكية.

**الخطوة 4: حفظ المصنف**
وأخيرًا، احفظ التغييرات مرة أخرى في ملف:
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "output.out.xls");
```
*توضيح*: ال `Save` تكتب الطريقة جميع التعديلات مرة أخرى إلى ملف Excel.

### الميزة 2: الوصول إلى النص وقراءته من عناصر التحكم في مربع النص
#### ملخص
ترتكز هذه الميزة على الوصول إلى عناصر التحكم في مربع النص المحددة داخل ورقة العمل وقراءة محتواها.

**تعليمات خطوة بخطوة**
اتبع الخطوات المشابهة للميزة السابقة، مع التركيز فقط على استرداد النص:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "book1.xls");
Worksheet worksheet = workbook.Worksheets[0];

Aspose.Cells.Drawing.TextBox textbox0 = worksheet.TextBoxes[0];
string textContent = textbox0.Text;

Aspose.Cells.Drawing.TextBox textbox1 = worksheet.TextBoxes[1];
string anotherTextContent = textbox1.Text;
```
*توضيح*:يقوم هذا الكود باسترجاع وعرض محتوى مربعات النص المحددة.
