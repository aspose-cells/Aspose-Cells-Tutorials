# استخراج نقاط اتصال الشكل باستخدام Aspose.Cells لـ .NET

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Cells لـ .NET
- استخراج نقاط الاتصال من الأشكال داخل ورقة عمل Excel
- أفضل الممارسات لدمج هذا الحل في تطبيقات أوسع

## دليل التنفيذ
### استخراج نقاط اتصال الشكل
سوف يرشدك هذا القسم خلال استخراج نقاط الاتصال من الأشكال باستخدام Aspose.Cells لـ .NET.
#### الخطوة 1: إنشاء مصنف جديد والوصول إلى ورقة العمل
ابدأ بإنشاء مثيل `Workbook` كائن يمثل ملف Excel. ثم انتقل إلى ورقة العمل الأولى التي تحتوي على الشكل المطلوب.
```csharp
// إنشاء مصنف جديد.
Workbook workbook = new Workbook();

// احصل على ورقة العمل الأولى في الكتاب.
Worksheet worksheet = workbook.Worksheets[0];
```
#### الخطوة 2: إضافة شكل والوصول إليه
أضف مربع نص (أو أي شكل آخر) إلى المجموعة، ثم استرده من مجموعة الأشكال.
```csharp
// أضف مربع نص جديد إلى المجموعة.
int textboxIndex = worksheet.TextBoxes.Add(2, 1, 160, 200);

// قم بالوصول إلى مربع النص الخاص بك والذي يعد أيضًا كائنًا شكليًا من مجموعة الأشكال.
Shape shape = workbook.Worksheets[0].Shapes[textboxIndex];
```
#### الخطوة 3: استرداد نقاط الاتصال
استخدم `GetConnectionPoints` طريقة لجلب جميع نقاط اتصال الشكل.
```csharp
// احصل على جميع نقاط الاتصال بهذا الشكل
var connectionPoints = shape.GetConnectionPoints();

// عرض جميع نقاط الشكل
foreach (var pt in connectionPoints)
{
    System.Console.WriteLine(string.Format("X = {0}, Y = {1}", pt[0], pt[1]));
}
```
