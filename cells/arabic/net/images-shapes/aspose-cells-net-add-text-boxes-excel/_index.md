### إنشاء مصنف والوصول إليه

**ملخص**

يعد إنشاء مثيل لمصنف عمل أمرًا أساسيًا عند العمل مع Aspose.Cells، لأنه يسمح بإجراء تعديلات وإضافات أخرى مثل أوراق العمل أو مربعات النص.

#### الخطوة 1: إنشاء مثيل لفئة المصنف
```csharp
using System;
using Aspose.Cells;

public static void CreateAndAccessWorkbook()
{
    // إنشاء كائن من فئة Workbook
    Workbook workbook = new Workbook();
    
    // الوصول إلى ورقة العمل الأولى من المجموعة
    Worksheet sheet = workbook.Worksheets[0];
}
```
**توضيح:**  
- `Workbook` يتم إنشاؤه لإنشاء ملف Excel جديد.
- يتم الوصول إلى ورقة العمل الافتراضية باستخدام `Worksheets[0]`.

### إضافة مربع نص إلى ورقة عمل

**ملخص**

تتيح لك إضافة مربعات النص عرض محتوى أكثر ثراءً في أوراق العمل الخاصة بك، وهو أمر مفيد للتعليقات التوضيحية أو عرض البيانات التفاعلية.

#### الخطوة 2: إضافة مربع النص وتسميته
```csharp
using Aspose.Cells;
using Aspose.Cells.Drawing;

public static void AddTextBoxToWorksheet()
{
    Workbook workbook = new Workbook();
    Worksheet sheet = workbook.Worksheets[0];
    
    // أضف مربع نص في الموضع (10، 10) بحجم (100، 50)
    int idx = sheet.TextBoxes.Add(10, 10, 100, 50);
    
    // الوصول إلى مربع النص الذي تم إنشاؤه حديثًا وتسميته
    TextBox tb1 = sheet.TextBoxes[idx];
    tb1.Name = "MyTextBox";
    
    // تعيين النص لمربع النص
    tb1.Text = "This is MyTextBox";
}
```
**توضيح:**  
- `sheet.TextBoxes.Add()` يضع مربع نص جديد.
- المعلمات تحدد الموضع `(x, y)` والحجم `(width, height)`.
- يتم تسمية مربع النص باستخدام `.Name`، مما يسمح بالرجوع إليها في المستقبل.

### الوصول إلى مربع نص مسمى في ورقة عمل

**ملخص**

يضمن الوصول إلى مربعات النص المسماة إمكانية استرجاعها أو تعديلها لاحقًا بكفاءة دون الحاجة إلى إعادة التنقل عبر المجموعة بأكملها.

#### الخطوة 3: الاسترداد حسب الاسم
```csharp
using Aspose.Cells;
using Aspose.Cells.Drawing;

public static void AccessNamedTextBox()
{
    Workbook workbook = new Workbook();
    Worksheet sheet = workbook.Worksheets[0];
    
    int idx = sheet.TextBoxes.Add(10, 10, 100, 50);
    TextBox tb1 = sheet.TextBoxes[idx];
    tb1.Name = "MyTextBox";
    tb1.Text = "This is MyTextBox";

    // الوصول إلى مربع النص عبر اسمه
    TextBox tb2 = sheet.TextBoxes["MyTextBox"];
}
```
**توضيح:**  
- `sheet.TextBoxes["MyTextBox"]` يسترجع مربع نص باستخدام اسمه المخصص، مما يوضح المرونة في إدارة عناصر المصنف.
