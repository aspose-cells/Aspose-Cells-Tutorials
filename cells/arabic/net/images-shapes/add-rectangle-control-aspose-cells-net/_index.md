## دليل التنفيذ
الآن بعد أن قمنا بتغطية الإعداد، دعنا ننفذ إضافة عنصر تحكم مستطيل داخل مصنف Excel باستخدام C#.

### إنشاء عنصر تحكم مستطيل وتكوينه
#### ملخص
تتضمن إضافة عنصر التحكم المستطيل إنشاء شكل جديد في ورقة العمل وتخصيص خصائصه مثل الموضع والحجم وسمك الخط ونمط الشرطة.

#### دليل خطوة بخطوة
**1. إنشاء مصنف**
ابدأ بإنشاء مثيل لـ `Workbook` فصل:
```csharp
// إنشاء مثيل جديد للمصنف
Workbook excelbook = new Workbook();
```

**2. أضف شكل المستطيل**
استخدم `AddRectangle` طريقة إدراج شكل مستطيل في ورقة العمل الخاصة بك:
```csharp
// إضافة عنصر تحكم مستطيل في موضع وحجم محددين
Aspose.Cells.Drawing.RectangleShape rectangle = excelbook.Worksheets[0].Shapes.AddRectangle(3, 0, 2, 0, 70, 130);
```
- **حدود**:المعلمات `(3, 0, 2, 0, 70, 130)` قم بتحديد مؤشر الصف، ومؤشر العمود، والعرض والارتفاع للمستطيل بالنقاط.

**3. وضع المجموعة**
قم بتحديد المكان الذي يجب وضع المستطيل فيه داخل ورقة العمل:
```csharp
// تعيين الوضع إلى التعويم الحر
rectangle.Placement = نوع الموضع.FreeFloating;
```
- **PlacementType**:يسمح FreeFloating بالحركة دون الحاجة إلى محاذاة الخلايا.

**4. تخصيص المظهر**
قم بتكوين الخصائص المرئية مثل وزن الخط ونمط الشرطة لتحسين الرؤية:
```csharp
// تعديل مظهر المستطيل
rectangle.Line.Weight = 4; // ضبط وزن الخط
rectangle.Line.DashStyle = MsoLineDashStyle.Solid; // تحديد نمط الشرطة كنمط صلب
```
- **وزن**:يحدد سمك حدود الشكل.
- **داش ستايل**:يحدد نمط الشرطات والفجوات المستخدمة لرسم المسارات.

**5. احفظ المصنف**
أخيرًا، احفظ المصنف الخاص بك باستخدام عنصر التحكم المستطيل المضاف حديثًا:
```csharp
// حفظ التغييرات في ملف جديد
excelbook.Save(dataDir + "book1.out.xls");
```
