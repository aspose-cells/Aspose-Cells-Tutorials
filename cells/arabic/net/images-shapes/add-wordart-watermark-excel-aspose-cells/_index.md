# كيفية إضافة علامة مائية WordArt إلى ورقة عمل Excel باستخدام Aspose.Cells .NET

هل ترغب في تعزيز أمان واحترافية جداول بيانات Excel الخاصة بك بإضافة علامات مائية؟ مع Aspose.Cells لـ .NET، تُصبح إضافة علامة WordArt المائية إلى أوراق عملك سهلة وفعّالة. سواءً كنت تحمي معلومات سرية أو تُضفي طابعًا تجاريًا على مستنداتك، تُحسّن هذه الميزة جودة ملفات Excel الخاصة بك بأقل جهد.

## دليل التنفيذ

### الميزة: إنشاء مصنف عمل جديد

**ملخص:** 
إنشاء مثيل لـ `Workbook` الفئة هي الخطوة الأولى للتعامل مع ملفات Excel باستخدام Aspose.Cells. يمثل هذا الكائن مصنفك بأكمله.

#### الخطوة 1: إنشاء مثيل مصنف جديد
```csharp
using Aspose.Cells;

string outputDir = @"YOUR_OUTPUT_DIRECTORY";
Workbook workbook = new Workbook();
// تم إنشاء مثيل جديد من Workbook، جاهزًا للتعامل معه.
```

### الميزة: الوصول إلى ورقة العمل

**ملخص:** 
انتقل إلى ورقة العمل الأولى لإضافة علامة مائية. أوراق العمل مفهرسة صفريًا.

#### الخطوة 2: الوصول إلى ورقة العمل الأولى
```csharp
Worksheet sheet = workbook.Worksheets[0];
// يمكنك الوصول إلى ورقة العمل الأولى من المصنف هنا.
```

### الميزة: إضافة علامة مائية WordArt إلى ورقة العمل

**ملخص:** 
أضف شكل تأثير النص (WordArt) كعلامة مائية لتعزيز أمان مستندك أو علامتك التجارية.

#### الخطوة 3: إضافة شكل WordArt
```csharp
using Aspose.Cells.Drawing;

Aspose.Cells.Drawing.Shape wordart = sheet.Shapes.AddTextEffect(
    MsoPresetTextEffect.TextEffect1, // نوع تأثير النص المحدد مسبقًا
    "CONFIDENTIAL",                 // محتوى النص الخاص بـ WordArt
    "Arial Black",                  // اسم الخط
    50,                             // حجم الخط
    false,                          // هل الخط عريض؟
    true,                           // هل الخط مائل؟
    18,                             // موضع X
    8,                              // موضع Y
    1,                              // مقياس العرض
    1,                              // مقياس الارتفاع
    130,                            // زاوية الدوران
    800);                           // معرف الشكل (يتم إنشاؤه تلقائيًا)
```

#### الخطوة 4: تكوين خصائص WordArt

قم بضبط شفافية ووضوح العلامة المائية الخاصة بك للتأكد من أنها لا تعيق المحتوى.

```csharp
// ضبط مستوى الشفافية للحصول على مظهر دقيق.
FillFormat wordArtFormat = wordart.Fill;
wordArtFormat.Transparency = 0.9;

// جعل الحدود غير مرئية.
LineFormat lineFormat = wordart.Line;
lineFormat.IsVisible = false;
```

### الميزة: حفظ المصنف مع العلامة المائية

**ملخص:** 
احفظ تعديلاتك في دليل محدد، مع التأكد من الحفاظ على العلامة المائية الخاصة بك.

#### الخطوة 5: حفظ المصنف المعدل
```csharp
workbook.Save(outputDir + "outputAddWordArtWatermarkToWorksheet.xlsx");
// يتم حفظ المصنف مع العلامة المائية WordArt المضمنة.
```
