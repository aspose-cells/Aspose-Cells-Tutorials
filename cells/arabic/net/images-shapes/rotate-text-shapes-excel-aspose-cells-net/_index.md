## دليل التنفيذ
في هذا القسم، سنرشدك خلال عملية تدوير النص داخل شكل في ورقة عمل Excel.

### الخطوة 1: تحميل ملف Excel الخاص بك
ابدأ بتحميل ملف Excel الخاص بك:
```csharp
Workbook wb = new Workbook("sampleRotateTextWithShapeInsideWorksheet.xlsx");
```
تعمل هذه الخطوة على تهيئة كائن المصنف الذي يمثل مستند Excel الخاص بك.

### الخطوة 2: الوصول إلى ورقة العمل وتعديلها
قم بالوصول إلى ورقة العمل المطلوبة حيث تريد معالجة الأشكال والنص:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell b4 = ws.Cells["B4"];
b4.PutValue("Text is not rotating with shape because RotateTextWithShape is false.");
```

### الخطوة 3: تكوين خصائص الشكل
قم بالوصول إلى الشكل الأول في ورقة العمل الخاصة بك لتعديل خصائص النص الخاصة به:
```csharp
Shape sh = ws.Shapes[0];
Aspose.Cells.Drawing.Texts.ShapeTextAlignment shapeTextAlignment = sh.TextBody.TextAlignment;
shapeTextAlignment.RotateTextWithShape = false; // اضبط هذا على "صحيح" إذا كنت تريد تدوير النص مع الشكل.
```
يحدد هذا التكوين ما إذا كان النص يدور مع الشكل.

### الخطوة 4: حفظ التغييرات
بعد إجراء التغييرات، احفظ المصنف:
```csharp
wb.Save("outputRotateTextWithShapeInsideWorksheet.xlsx");
Console.WriteLine("Rotation executed successfully.");
```
