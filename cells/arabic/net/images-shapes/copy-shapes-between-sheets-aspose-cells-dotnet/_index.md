## دليل التنفيذ

في هذا القسم، سنشرح عملية نسخ الأشكال بين أوراق العمل خطوة بخطوة.

### الخطوة 1: فتح مصنف موجود
ابدأ بإنشاء مصنف من ملف Excel المصدر. هنا يمكنك الوصول إلى الأشكال المراد نسخها.
```csharp
// إنشاء كائن مصنف وفتح ملف القالب
Workbook workbook = new Workbook(sourceDir + "sampleCopyControls.xlsx");
```

### الخطوة 2: الوصول إلى الأشكال في ورقة العمل المصدر
يمكنك الوصول إلى مجموعة الأشكال من ورقة العمل المصدر. هنا، نستهدف ورقة العمل "Sheet1" لاسترجاع أشكالها.
```csharp
// احصل على الأشكال من ورقة عمل "التحكم"
Aspose.Cells.Drawing.ShapeCollection shapes = workbook.Worksheets["Sheet1"].Shapes;
```

### الخطوة 3: نسخ الأشكال المحددة
الآن، لننسخ أشكالًا محددة (مثل مربع نص أو شكل بيضاوي) إلى ورقة عمل أخرى. سنضيف هذه النسخ في مواقع محددة.
```csharp
// نسخ مربع النص إلى ورقة عمل النتيجة
workbook.Worksheets["Result"].Shapes.AddCopy(shapes[0], 5, 0, 2, 0);

// انسخ الشكل البيضاوي إلى ورقة عمل النتيجة
workbook.Worksheets["Result"].Shapes.AddCopy(shapes[1], 10, 0, 2, 0);
```
- **حدود**: ال `AddCopy` تعتمد الطريقة على معلمات الموضع والحجم. عدّلها حسب احتياجاتك.

### الخطوة 4: حفظ المصنف
وأخيرًا، احفظ المصنف للحفاظ على التغييرات التي أجريتها.
```csharp
// حفظ ورقة العمل
workbook.Save(outputDir + "outputCopyControls.xlsx");
```
