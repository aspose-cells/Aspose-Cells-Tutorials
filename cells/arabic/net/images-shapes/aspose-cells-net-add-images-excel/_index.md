# إضافة الصور وتحديد موضعها في Excel باستخدام Aspose.Cells .NET: دليل شامل

**مقدمة**

يُعدّ تحسين مصنفات Excel باستخدام الصور أمرًا بالغ الأهمية عند إنشاء عروض تقديمية أو تقارير أو لوحات معلومات تعتمد على البيانات وتتطلب سياقًا بصريًا. **Aspose.Cells لـ .NET**يمكنك أتمتة هذه العملية بكفاءة. سواء كنت مطورًا يهدف إلى إنشاء تقارير ديناميكية أو محللًا يسعى إلى جعل جداول البيانات أكثر إفادة، سيرشدك هذا البرنامج التعليمي خلال خطوات إضافة الصور وتحديد مواقعها في مصنفات Excel باستخدام Aspose.Cells.

**ما سوف تتعلمه:**
- تهيئة وإعداد Aspose.Cells لـ .NET
- إضافة أوراق عمل جديدة إلى مصنف Excel
- تضمين الصور في خلايا ورقة عمل محددة
- تعيين مواضع البكسل المطلقة للصور داخل خلية
- حفظ التغييرات مرة أخرى في ملف Excel

قبل الغوص في الأمر، تأكد من استيفاء هذه المتطلبات الأساسية.

## دليل التنفيذ
دعونا نتعمق في كل ميزة خطوة بخطوة:

### إضافة ورقة عمل جديدة
**ملخص**
إضافة أوراق العمل ضرورية لتنظيم البيانات في Excel. توضح هذه الميزة كيفية القيام بذلك برمجيًا.

#### الخطوة 1: إنشاء ورقة عمل جديدة والإشارة إليها
```csharp
int sheetIndex = workbook.Worksheets.Add(); // إضافة ورقة عمل جديدة
Worksheet worksheet = workbook.Worksheets[sheetIndex]; // قم بالإشارة إلى ورقة العمل المضافة حديثًا
```

### إضافة صورة إلى خلية في ورقة العمل
**ملخص**
يمكن أن يؤدي تضمين الصور داخل الخلايا إلى توفير سياق أساسي أو عناصر العلامة التجارية في تقارير Excel الخاصة بك.

#### الخطوة 1: تحديد مسار الصورة وإضافتها إلى ورقة العمل
```csharp
using System.IO;

string imagePath = Path.Combine(SourceDir, "logo.jpg");
int pictureIndex = worksheet.Pictures.Add(5, 5, imagePath); // وضع الصورة في الخلية F6 (الصف 5، العمود 5)
```

#### الخطوة 2: الوصول إلى الصورة المضافة حديثًا
```csharp
Aspose.Cells.Drawing.Picture picture = worksheet.Pictures[pictureIndex];
```

### وضع الصورة بالبكسل
**ملخص**
للحصول على تحكم دقيق في وضع الصورة داخل خلية، يمكنك تعيين مواضع البكسل المطلقة.

#### الخطوة 1: تعيين مواضع البكسل للصورة
```csharp
picture.Left = 60; // تعيين الموضع الأيسر للصورة بالبكسل
picture.Top = 10; // تعيين الموضع العلوي للصورة بالبكسل
```

### حفظ المصنف في ملف
**ملخص**
تأكد من حفظ المصنف الخاص بك مع كافة التعديلات بشكل صحيح.

#### الخطوة 1: تحديد مسار الإخراج وحفظه
```csharp
string outputPath = Path.Combine(outputDir, "book1.out.xls"); // تحديد مسار ملف الإخراج
workbook.Save(outputPath); // حفظ المصنف
```

## قسم الأسئلة الشائعة
1. **كيف أقوم بتثبيت Aspose.Cells على بيئة Linux؟**
   - بإمكانك استخدام .NET Core لتشغيل تطبيقات C#، بما في ذلك تلك التي تحتوي على حزمة Aspose.Cells.
2. **هل يمكنني إضافة صور متعددة إلى ورقة عمل واحدة؟**
   - نعم يمكنك الاتصال `worksheet.Pictures.Add` عدة مرات لصور ومواضع مختلفة.
3. **ما هي تنسيقات الصور التي يدعمها Aspose.Cells؟**
   - يتم دعم التنسيقات الشائعة مثل JPEG، PNG، BMP، وما إلى ذلك.
4. **كيف يمكنني التأكد من حفظ المصنف الخاص بي بشكل صحيح؟**
   - تأكد من أن مسار دليل الإخراج صحيح وأنه يحتوي على أذونات الكتابة.
5. **هل يمكنني تغيير حجم الصورة برمجيًا؟**
   - نعم، استخدم خصائص مثل `picture.WidthScale` و `picture.HeightScale`.
