### التلاعب بموضع ترتيب Z للأشكال

#### ملخص
يتيح لك فهم ترتيب Z والتحكم فيه التحكم في الأشكال التي تظهر في الأعلى في السيناريوهات المتداخلة. تُعد هذه الميزة بالغة الأهمية عند التعامل مع أوراق عمل معقدة تحتوي على كائنات رسومية متعددة.

#### الوصول إلى مواضع الأشكال وتعديلها (H3)

لإرسال شكل إلى الأمام أو الخلف، اتبع الخطوات التالية:

```csharp
// تحميل ملف Excel المصدر
Workbook workbook = new Workbook("sampleToFrontOrBack.xlsx");

// الوصول إلى ورقة العمل الأولى
Worksheet sheet = workbook.Worksheets[0];

// الوصول إلى الأشكال المحددة عن طريق الفهرس
Shape shape1 = sheet.Shapes[0];
Shape shape4 = sheet.Shapes[3];

// طباعة موضع ترتيب Z الحالي للشكل
Console.WriteLine("Z-Order Shape 1: " + shape1.ZOrderPosition);

// انقل هذا الشكل إلى الأمام
shape1.ToFrontOrBack(2);

// التحقق من موضع Z-Order الجديد
Console.WriteLine("New Z-Order Shape 4: " + shape4.ZOrderPosition);

// أرسل شكلًا آخر إلى الخلف
shape4.ToFrontOrBack(-2);
```

**توضيح**: 
- `ToFrontOrBack(int value)`:تضبط هذه الطريقة ترتيب Z بناءً على المعلمة. العدد الصحيح الموجب يُحرك الشكل للأمام، بينما العدد السالب يُرجعه للخلف.

#### حفظ التغييرات (H3)

بعد معالجة الأشكال، احفظ التغييرات للتأكد من الحفاظ عليها:

