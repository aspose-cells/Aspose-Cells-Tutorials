## تعليمات التثبيت
1. تثبيت Aspose.Cells لـ .NET: استخدم NuGet Package Manager لتثبيت مكتبة Aspose.Cells في مشروعك.
   ```
   Install-Package Aspose.Cells
   ```
2. إضافة استخدام التوجيهات: قم بتضمين المساحات الأساسية الضرورية في ملف التعليمات البرمجية الخاص بك.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. إضافة كود العينة: انسخ الكود المقدم والصقه في مشروعك، مع التأكد من تحديد مسار دليل المستند الخاص بك.
4. شغّل الكود: نفّذ مشروعك للوصول إلى تسمية كائن Ole وتعديلها في ملف Excel. تحقق من وحدة التحكم لرسائل الإخراج التي تؤكد نجاح تنفيذ العملية.
