## تعليمات التثبيت
1. قم بتثبيت Aspose.Cells لـ .NET: 
   * استخدم مدير الحزمات NuGet في مشروع Visual Studio الخاص بك:
     ```
     Install-Package Aspose.Cells
     ```
   * أو تنزيل DLL من [الموقع الإلكتروني](https://www.aspose.com/downloads/cells/net). . .

2. إضافة الإشارات: تأكد من إشارات مشروعك إلى مكتبة Aspose.Cells.

3. إعداد دليل الخروج: تحديث `النتائج`المتغير في الرمز مع المسار المطلوب للخروج حيث سيتم حفظ ملف Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
