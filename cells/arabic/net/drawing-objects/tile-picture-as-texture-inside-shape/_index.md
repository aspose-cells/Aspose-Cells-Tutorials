## تعليمات التثبيت
1. تأكد من أن لديك بيئة .NET على جهازك
2. قم بتنزيل وتثبيت مكتبة Aspose.Cells من موقع Aspose الرسمي أو باستخدام NuGet Package Manager:
   ```
   Install-Package Aspose.Cells
   ```
3. Create a new C# project or open your existing .NET project.
4. إضافة إشارة إلى مكتبة Aspose.Cells في مشروعك.
5. نسخ رمز المثال المقدم في مشروعك، واستبدال مسارات المجلد المدخل / الخارجي حسب الحاجة.
6. قم بتجميع وتشغيل مشروعك لرؤية ملء التركيب المطبق على الأشكال في ملف Excel.
