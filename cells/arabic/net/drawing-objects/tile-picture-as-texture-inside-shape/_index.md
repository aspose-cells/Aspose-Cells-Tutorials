## تعليمات التثبيت
1. تأكد من إعداد بيئة .NET على جهازك.
2. قم بتنزيل مكتبة Aspose.Cells وتثبيتها من موقع Aspose الرسمي أو باستخدام NuGet Package Manager:
   ```
   Install-Package Aspose.Cells
   ```
3. قم بإنشاء مشروع C# جديد أو افتح مشروع .NET الحالي لديك.
4. أضف مرجعًا إلى مكتبة Aspose.Cells في مشروعك.
5. انسخ الكود المثال المقدم إلى مشروعك، مع استبدال مسارات دليل الإدخال/الإخراج حسب الحاجة.
6. قم بتجميع مشروعك وتشغيله لرؤية تعبئة نسيج البلاط المطبق على الأشكال في ملف Excel.
