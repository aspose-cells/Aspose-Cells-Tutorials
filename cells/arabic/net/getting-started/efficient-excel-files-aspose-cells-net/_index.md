### إنشاء ملفات Excel متعددة

تتيح لك هذه الميزة إنشاء العديد من مصنفات Excel بكفاءة:

#### الخطوة 1: تهيئة المصنف

ابدأ بإنشاء مثيل جديد لـ `Workbook`يمثل هذا الكائن ملف Excel بأكمله.

```csharp
Workbook workbook = new Workbook();
```

#### الخطوة 2: تكوين أوراق العمل

يمكن أن يحتوي كل مصنف على عدة أوراق عمل. هنا، سنضيف خمس أوراق لكل مصنف:

```csharp
// إزالة ورقة العمل الافتراضية.
workbook.Worksheets.RemoveAt(0);

for (int sheetIndex = 0; sheetIndex < 5; sheetIndex++)
{
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = sheetIndex.ToString();
    
    for (int row = 0; row < 150; row++)
    {
        for (int col = 0; col < 50; col++)
        {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

#### الخطوة 3: حفظ المصنف

أخيرًا، احفظ كل مصنف كملف Excel في دليل الإخراج المطلوب:

```csharp
string filePath = System.IO.Path.Combine(outputDir, $"{fileIndex}_out.xls");
workbook.Save(filePath);
```
