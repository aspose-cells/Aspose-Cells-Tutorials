## دليل التنفيذ

### الميزة 1: تحميل أوراق العمل وتكرارها

**ملخص**:يوضح هذا القسم كيفية تحميل مصنف Excel والوصول إلى أوراق العمل الخاصة به والتكرار عليها باستخدام مكتبة Aspose.Cells.

#### تعليمات خطوة بخطوة

##### الوصول إلى أوراق العمل في مصنف

```csharp
using System;
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";

// تحميل ملف Excel المصدر
Workbook wb = new Workbook(SourceDir + "/sampleRemoveExistingPrinterSettingsOfWorksheets.xlsx");

// احصل على عدد أوراق المصنف
int sheetCount = wb.Worksheets.Count;

// تكرار جميع الأوراق
for (int i = 0; i < sheetCount; i++)
{
    // الوصول إلى ورقة العمل i-th
    Worksheet ws = wb.Worksheets[i];
    
    // قم بإجراء العمليات على كل ورقة عمل هنا
}
```

**توضيح**:هنا، نقوم بتحميل مصنف Excel واستخدام حلقة بسيطة للوصول إلى كل ورقة عمل. `Workbook` توفر الفئة خصائص مثل `Worksheets`، مما يسمح لنا بالتكرار خلال كافة الأوراق.

### الميزة 2: الوصول إلى إعدادات إعداد الصفحة وتعديلها

**ملخص**:تركز هذه الميزة على الوصول إلى إعدادات إعداد الصفحة لكل ورقة عمل وإزالة تكوينات الطابعة الموجودة إذا كانت موجودة.

#### تعليمات خطوة بخطوة

##### تعديل تكوينات إعداد الصفحة

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";

// تحميل ملف Excel المصدر
Workbook wb = new Workbook(SourceDir + "/sampleRemoveExistingPrinterSettingsOfWorksheets.xlsx");

// احصل على عدد أوراق المصنف
int sheetCount = wb.Worksheets.Count;

// تكرار جميع الأوراق
for (int i = 0; i < sheetCount; i++)
{
    // الوصول إلى ورقة العمل i-th
    Worksheet ws = wb.Worksheets[i];
    
    // إعداد صفحة ورقة عمل Access
    PageSetup ps = ws.PageSetup;
    
    // تحقق مما إذا كانت إعدادات الطابعة لهذه ورقة العمل موجودة
    if (ps.PrinterSettings != null)
    {
        // قم بإزالة إعدادات الطابعة عن طريق تعيينها على القيمة null
        ps.PrinterSettings = null;
    }
}
```

**توضيح**يوضح هذا المقطع كيفية الانتقال إلى إعدادات صفحة كل ورقة عمل وإزالة إعدادات الطابعة الحالية. `PageSetup` يوفر الكائن إمكانية الوصول إلى العديد من التكوينات المتعلقة بالطباعة، مما يسمح بالتحكم الدقيق في إخراج المستند.

### الميزة 3: حفظ المصنف

**ملخص**بعد إجراء التغييرات، من الضروري حفظ مصنفك. يتناول هذا القسم حفظ ملف Excel المُعدَّل.

#### تعليمات خطوة بخطوة

##### حفظ التعديلات

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// تحميل ملف Excel المصدر
Workbook wb = new Workbook(SourceDir + "/sampleRemoveExistingPrinterSettingsOfWorksheets.xlsx");

// حفظ المصنف بعد التعديلات
wb.Save(OutputDir + "/outputRemoveExistingPrinterSettingsOfWorksheets.xlsx");
```

**توضيح**: ال `Save` طريقة `Workbook` تُعيد الفئة كتابة جميع التغييرات إلى ملف Excel. تأكد من تحديد دليل الإخراج بشكل صحيح لحفظ ناجح.
