# تخصيص إعدادات الاتجاه للنص في Excel

## مقدمة
عند العمل مع جداول البيانات، يُعدّ العرض التقديمي أمرًا بالغ الأهمية. ربما واجهتَ مواقف لا يُجدي فيها اتجاه النص الافتراضي نفعًا. سواءً كان ذلك لإضافة نص أكثر في خلية ضيقة، أو لإضافة لمسة جمالية، أو لتحسين سهولة القراءة، يُمكن لتخصيص اتجاه النص أن يُحسّن ملفات Excel. في هذا البرنامج التعليمي، سنتناول بالتفصيل كيفية تعديل اتجاه النص في Excel باستخدام Aspose.Cells لـ .NET، مُقدّمًا لك دليلًا عمليًا ومباشرًا.

## المتطلبات الأساسية

قبل أن نبدأ رحلتنا في عالم التعامل مع Excel، لنتأكد من إعداد كل شيء بشكل صحيح. إليك ما تحتاجه للبدء:

- Visual Studio: تأكد من تثبيت Visual Studio على جهازك. إنه بيئة التطوير المتكاملة الأكثر شيوعًا لتطوير .NET.
- Aspose.Cells لمكتبة .NET: قم بتنزيل أحدث إصدار من Aspose.Cells من [موقع](https://releases.aspose.com/cells/net/). تعتبر هذه المكتبة ضرورية لمهامنا في قراءة وكتابة وتعديل ملفات Excel.
- .NET Framework: تأكد من تثبيت .NET Framework، حيث يعمل Aspose.Cells بشكل أساسي داخل هذه البيئة.
  
بمجرد إعداد هذه الأدوات، ستكون جاهزًا لإطلاق العنان للفنان الذي بداخلك في مجال جداول البيانات!

## استيراد الحزم

لبدء البرمجة، عليك استيراد مساحات الأسماء اللازمة من مكتبة Aspose.Cells. سيُتيح لك هذا الوصول إلى جميع الفئات والأساليب التي ستستخدمها. إليك كيفية القيام بذلك:

### إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع تطبيق وحدة تحكم جديدًا. سيكون هذا المشروع بمثابة ساحة لعب لتجربة وظائف Aspose.Cells.

### تثبيت حزمة Aspose.Cells NuGet

لإضافة مكتبة Aspose.Cells إلى مشروعك بسرعة، استخدم مدير حزم NuGet. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول، ثم اختر "إدارة حزم NuGet". ابحث عن "Aspose.Cells" وثبّته.

### أضف التوجيه باستخدام

الآن بعد تثبيت الحزمة، تأكد من تضمين التوجيه التالي في بداية ملفك `Program.cs` ملف:

```csharp
using System.IO;
using Aspose.Cells;
```

مع توفر هذه الحزم، أصبحنا جاهزين للتعمق في الترميز الفعلي!

الآن، لنبدأ بتخصيص اتجاه النص في إكسل باستخدام Aspose.Cells. فيما يلي الخطوات مُقسّمة إلى أجزاء يسهل التعامل معها:

## الخطوة 1: إعداد دليل المستندات 

أولاً، علينا إنشاء مجلد لحفظ ملفات إكسل. هذا يُبقي مساحة العمل مُنظمة.

```csharp
string dataDir = "Your Document Directory";

// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

هنا، يمكنك تعريف متغير سلسلة `dataDir` لتحديد مسار مستنداتك. يتحقق الكود من وجود الدليل؛ وإن لم يكن، يُنشئه. الأمر أشبه بالتأكد من نظافة مساحة العمل قبل بدء أي مشروع!

## الخطوة 2: إنشاء مصنف جديد

بعد ذلك، سنقوم بإنشاء مصنف جديد يمثل ملف Excel الخاص بنا.

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

من خلال إنشاء مثيل `Workbook` في الصف، أنت تُنشئ مصنف Excel جديدًا. تخيل هذا كفتح صفحة فارغة حيث يمكنك البدء بتلوين بياناتك!

## الخطوة 3: الوصول إلى ورقة العمل

الآن بعد أن أصبح لدينا المصنف، نحتاج إلى الوصول إلى ورقة العمل المحددة التي نريد تعديلها. 

```csharp
// الحصول على مرجع ورقة العمل
Worksheet worksheet = workbook.Worksheets[0];
```

يمكن أن يحتوي كل مصنف على عدة أوراق عمل. هنا، نصل إلى الورقة الأولى باستخدام `Worksheets[0]`إنه مثل اختيار الصفحة التي تريد العمل عليها في دفتر ملاحظاتك!

## الخطوة 4: الحصول على مرجع الخلية

لننتقل الآن إلى استرجاع الخلية التي نريد تخصيص النص فيها.

```csharp
// الوصول إلى الخلية "A1" من ورقة العمل
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
```

نحن نحصل على المرجع للخلية `A1`هذه هي الخلية التي سنتحكم بها. تخيّل أنها تُحدد بدقة نقطة البداية على لوحتك!

## الخطوة 5: إضافة قيمة إلى الخلية

بعد ذلك، سنضع بعض النص في الخلية لرؤية التغييرات أثناء العمل.

```csharp
// إضافة بعض القيمة إلى الخلية "A1"
cell.PutValue("Visit Aspose!");
```

هنا، نضع ببساطة نص "زيارة Aspose!" في الخلية المحددة. الأمر أشبه بكتابة عنوانك على لوحة الرسم!

## الخطوة 6: تخصيص نمط الخلية

الآن يأتي الجزء المثير - تخصيص اتجاه النص داخل الخلية.

```csharp
// ضبط المحاذاة الأفقية للنص في الخلية "A1"
Style style = cell.GetStyle();

// ضبط دوران النص (داخل الخلية) إلى 25
style.RotationAngle = 25;

cell.SetStyle(style);
```

نقوم باسترجاع نمط الخلية، ثم نقوم بتعديله `RotationAngle` حتى ٢٥ درجة. هذا يُغيّر النص قليلاً، مُضيفاً لمسةً من الأناقة. تماماً كما لو كنت تُميل لوحة الرسم لإضفاء منظورٍ مختلف!

## الخطوة 7: حفظ ملف Excel

أخيرًا، حان الوقت لحفظ ملف Excel المخصص بشكل جميل.

```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```

هنا، نحفظ مصنف العمل في المجلد المخصص له بتنسيق Excel 97-2003. تخيل هذا كإطار حماية حول تحفتك الفنية!
