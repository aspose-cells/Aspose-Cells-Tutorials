# تدوير النص وتغيير اتجاهه في Excel

## مقدمة
عند العمل مع ملفات Excel برمجيًا، غالبًا ما نواجه تحدي عرض البيانات بالتنسيق المطلوب. هل سبق لك أن رغبت في تغيير اتجاه النص في خلية Excel؟ ربما تحتاج إلى قراءة النص من اليمين إلى اليسار، خاصةً إذا كنت تعمل بلغات مثل العربية أو العبرية. أو ربما تبحث فقط عن طريقة لتحسين المظهر المرئي لجداول البيانات. مهما كان سببك، يوفر Aspose.Cells for .NET حلاً بسيطًا للتحكم في اتجاه النص في ملفات Excel. في هذا البرنامج التعليمي، سنشرح الخطوات اللازمة لتدوير النص وتغيير اتجاهه في Excel باستخدام Aspose.Cells.
## المتطلبات الأساسية
قبل أن نتعمق في جزء الترميز، تأكد من أن لديك بعض الأشياء جاهزة:
1. Visual Studio: تأكد من تثبيت Visual Studio على جهاز الكمبيوتر لديك. مكتبة Aspose.Cells تعمل معه بشكل جيد.
2. مكتبة Aspose.Cells: ستحتاج إلى مكتبة Aspose.Cells لـ .NET. يمكنك تنزيلها من [موقع](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: إن الإلمام ببرمجة C# سوف يسهل عليك متابعة البرنامج التعليمي.
4. .NET Framework: تأكد من أن مشروعك يستهدف .NET Framework، حيث تم تصميم Aspose.Cells للعمل ضمن تلك البيئة.
بمجرد أن تكون كل المتطلبات الأساسية جاهزة، فأنت جاهز للبدء!
## استيراد الحزم
الآن، لنُجهّز مشروعنا باستيراد الحزم المطلوبة. إليك الطريقة:
### إنشاء مشروع جديد
- افتح Visual Studio، ثم قم بإنشاء مشروع جديد.
- حدد تطبيق وحدة التحكم من القوالب، وأعطه اسمًا مناسبًا مثل "ExcelTextDirectionDemo".
### إضافة مكتبة Aspose.Cells
- انقر بزر الماوس الأيمن فوق المشروع في مستكشف الحلول واختر إدارة حزم NuGet.
- ابحث عن Aspose.Cells وقم بتثبيته.
### استيراد مساحات الأسماء الضرورية
الآن حان وقت إدخال مساحات الأسماء اللازمة. في أعلى `Program.cs` الملف، يتضمن ما يلي:
```csharp
using System.IO;
using Aspose.Cells;
```
بهذا، أنت جاهز لبدء تعديل ملفات Excel! والآن، لنبدأ البرمجة الفعلية.
## الخطوة 1: إعداد دليل المستندات الخاص بك
لضمان حفظ ملف إكسل في المكان الصحيح، علينا تحديد مجلد. إليك كيفية القيام بذلك:
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory"; // ضبط مسار الدليل الخاص بك
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

يُحدد هذا الكود دليلاً لحفظ ملف إكسل. يتحقق من وجود الدليل ويُنشئه إن لم يكن. تأكد من استبدال `"Your Document Directory"` مع مسار صالح.
## الخطوة 2: إنشاء كائن مصنف
الآن، لنُنشئ مُصنّف Excel جديدًا. هنا سنُتلاعب بالخلايا.
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

من خلال إنشاء `Workbook` الكائن، فأنت تبدأ في الأساس بملف Excel جديد فارغ يمكنك تعديله.
## الخطوة 3: الحصول على مرجع ورقة العمل
الآن قم بالوصول إلى ورقة العمل التي تريد إجراء التغييرات عليها.
```csharp
// الحصول على مرجع ورقة العمل
Worksheet worksheet = workbook.Worksheets[0];
```

ال `Worksheet` يشير الكائن إلى ورقة العمل الأولى في مصنفك. يمكنك الوصول إلى أوراق العمل الأخرى بتغيير الفهرس.
## الخطوة 4: الوصول إلى خلية محددة
دعونا نركز على خلية محددة، في هذه الحالة، "A1". 
```csharp
// الوصول إلى الخلية "A1" من ورقة العمل
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
```

يتيح لك هذا السطر من التعليمات البرمجية الوصول إلى الخلية "A1"، والتي سنقوم بتعديلها قريبًا.
## الخطوة 5: إضافة قيمة إلى الخلية
حان الوقت لوضع بعض البيانات في خلايانا.
```csharp
// إضافة بعض القيمة إلى الخلية "A1"
cell.PutValue("Visit Aspose!");
```

هنا، نضيف ببساطة النص "زيارة Aspose!" إلى الخلية "A1". يمكنك تغيير هذا النص إلى أي شيء تريده.
## الخطوة 6: إعداد نمط النص
الآن يأتي الجزء الذي نقوم فيه بتغيير اتجاه النص. 
```csharp
// ضبط المحاذاة الأفقية للنص في الخلية "A1"
Style style = cell.GetStyle();
```

يؤدي هذا إلى استرجاع نمط الخلية الحالي، مما يمهد الطريق للتعديلات.
## الخطوة 7: تغيير اتجاه النص 
هنا يأتي السحر! يمكنك تغيير اتجاه النص كما يلي:
```csharp
// ضبط اتجاه النص من اليمين إلى اليسار
style.TextDirection = TextDirectionType.RightToLeft;
```

يقوم هذا السطر بتعيين اتجاه النص من اليمين إلى اليسار، وهو أمر ضروري للغات مثل العربية أو العبرية. 
## الخطوة 8: تطبيق النمط على الخلية
بعد تغيير نمط اتجاه النص، قم بتطبيق هذه التغييرات مرة أخرى على الخلية:
```csharp
cell.SetStyle(style);
```

قم بتطبيق النمط المعدل على الخلية مرة أخرى، مع التأكد من أنه يعكس اتجاه النص الجديد.
## الخطوة 9: حفظ ملف Excel
وأخيرًا، دعنا نحفظ التغييرات في ملف Excel جديد.
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```

يحفظ هذا الكود المصنف باسم الملف المحدد في المجلد المحدد. التنسيق المحدد هو Excel 97-2003.
## خاتمة
وها أنت ذا! لقد تعلمت بنجاح كيفية تدوير وتغيير اتجاه النص في خلية Excel باستخدام Aspose.Cells لـ .NET. أليس من المذهل كيف يمكن لبضعة أسطر من التعليمات البرمجية أن تُغير تصميم جدول بياناتك ولغته بالكامل؟ إن القدرة على التعامل مع ملفات Excel برمجيًا تفتح آفاقًا واسعة من الإمكانيات، من أتمتة التقارير إلى تحسين عرض البيانات.
