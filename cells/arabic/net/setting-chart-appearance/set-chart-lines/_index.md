# تعيين خطوط الرسم البياني

## مقدمة

إنشاء مخططات بيانية جذابة بصريًا وغنية بالمعلومات أمرٌ أساسي لتمثيل البيانات. سواءً كنت محلل بيانات، أو مدير أعمال، أو مجرد شخصٍ يعشق تنظيم البيانات، فإن المخططات البيانية تُحسّن بشكل كبير طريقة عرض معلوماتك. سيشرح لك هذا البرنامج التعليمي عملية إعداد خطوط المخططات البيانية باستخدام Aspose.Cells for .NET، وهي مكتبة فعّالة للتعامل مع ملفات Excel. في النهاية، ستتعلم كيفية إنشاء مخططات بيانية رائعة مليئة بالتخصيصات لإبراز بيانات Excel الخاصة بك!

## المتطلبات الأساسية

قبل الخوض في جزء الترميز، تأكد من أنك مجهز بما يلي:

- Visual Studio: تأكد من تثبيت Visual Studio. يُنصح بشدة باستخدام أحدث إصدار للاستفادة من جميع الميزات.
- .NET Framework: يجب أن يعتمد مشروعك على .NET Framework (أو .NET Core) حيث ستنفذ Aspose.Cells.
- Aspose.Cells لـ .NET: قم بتنزيل Aspose.Cells وتثبيته من [موقع Aspose](https://releases.aspose.com/cells/net/).
- الفهم الأساسي للغة البرمجة C#: إن الإلمام بلغة البرمجة C# سيكون مفيدًا أثناء الترميز.

## استيراد الحزم

لبدء استخدام Aspose.Cells، ستحتاج إلى استيراد مساحات الأسماء اللازمة إلى مشروعك. سيُمكّنك هذا من الوصول إلى جميع الميزات والوظائف الرائعة التي يُقدمها Aspose.Cells. إليك كيفية استيراد الحزم في ملف C# الخاص بك:

```csharp
using Aspose.Cells;
using Aspose.Cells.Charts;
using System.Drawing;
```

دعنا نقسم العملية إلى خطوات قابلة للإدارة حتى تتمكن من متابعتها بسهولة.

## الخطوة 1: تحديد دليل الإخراج الخاص بك

أولاً، ستحتاج إلى مكان لحفظ ملف إكسل الذي أنشأته حديثاً. حدّد مسار الإخراج في أعلى الكود كما يلي:

```csharp
// دليل الإخراج
string outputDir = "Your Output Directory";
```

الشرح: استبدل "دليل الإخراج الخاص بك" بالمسار الذي تريد أن يحفظ فيه Aspose.Cells الملف، مثل `C:\\MyExcelFiles\\`.

## الخطوة 2: إنشاء كائن مصنف

الآن، سنقوم بإنشاء كائن مصنف، والذي سيكون بمثابة حاوية لجدول البيانات الخاص بك.

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```

الشرح: هذا السطر ينشئ مثيلًا لـ `Workbook` من مكتبة Aspose.Cells. الأمر أشبه بفتح ملف Excel جديد فارغ، حيث يمكنك البدء بإضافة أوراقك وبياناتك.

## الخطوة 3: الرجوع إلى ورقة العمل

بعد ذلك، ستحتاج إلى العمل على ورقة عمل محددة في مصنفك. سنختار ورقة العمل الأولى.

```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[0];
```

التوضيح: يتم فهرسة أوراق العمل بدءًا من 0، لذا `worksheets[0]` يشير إلى ورقة العمل الأولى.

## الخطوة 4: إضافة قيم العينة إلى الخلايا

دعونا نملأ بعض الخلايا بالبيانات التي سنستخدمها لاحقًا لإنشاء الرسم البياني الخاص بنا.

```csharp
// إضافة قيم العينة إلى الخلايا
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(60);
worksheet.Cells["B2"].PutValue(32);
worksheet.Cells["B3"].PutValue(50);
```

توضيح: هنا نملأ الخلايا من "A1" إلى "A3" ومن "B1" إلى "B3" ببعض القيم الرقمية. سيتم رسم هذه القيم في مخططنا لاحقًا.

## الخطوة 5: إضافة مخطط إلى ورقة العمل

الآن، حان وقت إنشاء مخطط بياني! سنضيف مخططًا بيانيًا عموديًا.

```csharp
// إضافة مخطط إلى ورقة العمل
int chartIndex = worksheet.Charts.Add(Aspose.Cells.Charts.ChartType.Column, 5, 0, 25, 10);
```

توضيح: يُضيف هذا الخط مخططًا عموديًا عند إحداثيات مُحددة في ورقة العمل. تُحدد المعلمات مكان رسم المخطط على الشبكة.

## الخطوة 6: الوصول إلى الرسم البياني المضاف حديثًا

أنت الآن بحاجة إلى الرجوع إلى الرسم البياني الذي قمت بإنشائه للتو.

```csharp
// الوصول إلى مثيل الرسم البياني المضاف حديثًا
Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];
```

الشرح: يتيح لك هذا التحكم في مثيل الرسم البياني مما يسمح لك بتخصيصه وتصميمه بشكل أكبر.

## الخطوة 7: إضافة سلسلة البيانات إلى الرسم البياني

دعونا نضيف سلسلة البيانات إلى مخططنا.

```csharp
// إضافة SeriesCollection (مصدر بيانات الرسم البياني) إلى الرسم البياني الذي يتراوح من الخلية "A1" إلى "B3"
chart.NSeries.Add("A1:B3", true);
```

شرح: يُوجِّه هذا السطر الرسم البياني لسحب البيانات من النطاق المُحدَّد. يُحدِّد المُعامل الثاني ما إذا كانت نطاقات البيانات تتضمن فئات.

## الخطوة 8: تخصيص مظهر الرسم البياني

الآن، حان وقت المرح - تخصيص مخططك! لنُغيّر بعض الألوان.

```csharp
// ضبط لون المقدمة لمنطقة الرسم البياني
chart.PlotArea.Area.ForegroundColor = Color.Blue;

// تعيين لون المقدمة لمنطقة الرسم البياني
chart.ChartArea.Area.ForegroundColor = Color.Yellow;

// ضبط لون المقدمة لمنطقة المجموعة الأولى
chart.NSeries[0].Area.ForegroundColor = Color.Red;

// ضبط لون المقدمة لمنطقة نقطة المجموعة من السلسلة الأولى
chart.NSeries[0].Points[0].Area.ForegroundColor = Color.Cyan;

// ملء منطقة المجموعة من السلسلة الثانية بتدرج لوني
chart.NSeries[1].Area.FillFormat.SetOneColorGradient(Color.Lime, 1, Aspose.Cells.Drawing.GradientStyleType.Horizontal, 1);
```

شرح: هنا، يمكنك تخصيص ألوان مختلف مكونات الرسم البياني لجعله جذابًا بصريًا. يستهدف كل سطر مناطق مختلفة من الرسم البياني.

## الخطوة 9: تطبيق أنماط الخطوط

بعد ذلك، يمكنك تعديل أنماط الخطوط لسلسلة البيانات الخاصة بك لجعل الرسم البياني الخاص بك ليس جميلًا فحسب، بل احترافيًا أيضًا.

```csharp
// تطبيق نمط الخط المنقط على أسطر SeriesCollection
chart.NSeries[0].Border.Style = Aspose.Cells.Drawing.LineType.Dot;

// تطبيق نمط علامة مثلثة على علامات البيانات في SeriesCollection
chart.NSeries[0].Marker.MarkerStyle = Aspose.Cells.Charts.ChartMarkerType.Triangle;

// تعيين وزن جميع الأسطر في SeriesCollection إلى متوسط
chart.NSeries[1].Border.Weight = Aspose.Cells.Drawing.WeightType.MediumLine;
```

شرح: يُخصّص الكود أعلاه حدود سلسلة الرسم البياني، بمنحها خطًا منقّطًا، بل ويُغيّر علامات نقاط البيانات إلى مثلثات. الأمر كله يتعلق بتلك اللمسة الشخصية!

## الخطوة 10: احفظ مصنفك

الآن، دعنا نحفظ عملك الشاق في ملف Excel.

```csharp
// حفظ ملف Excel
workbook.Save(outputDir + "outputSettingChartLines.xlsx");
```

شرح: هذا السطر يحفظ مصنفك بالاسم المحدد في مجلد الإخراج الذي حددته. يمكنك الآن فتحه ورؤية مخططك الرائع!

## الخطوة 11: تأكيد التنفيذ

وأخيرا، دعونا نؤكد أن كل شيء سار بسلاسة.

```csharp
Console.WriteLine("SettingChartLines executed successfully.");
```

الشرح: رسالة بسيطة لإعلامك بأن الكود الخاص بك تم تنفيذه دون أي مشاكل.
