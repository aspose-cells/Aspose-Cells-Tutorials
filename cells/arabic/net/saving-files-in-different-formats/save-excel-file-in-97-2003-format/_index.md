# حفظ ملف Excel بتنسيق 97-2003

## مقدمة
يُمكن أن يُحدث إنشاء ملفات Excel وإدارتها برمجيًا نقلة نوعية، خاصةً للشركات التي تعتمد بشكل كبير على معالجة البيانات. تُعد Aspose.Cells إحدى الأدوات الرائعة المتاحة لمطوري .NET. إنها متعددة الاستخدامات وقوية، تُساعدك على تبسيط سير العمل وأتمتة المهام باستخدام جداول البيانات. إذا كنت ترغب في حفظ ملفات Excel بتنسيق 97-2003 الكلاسيكي، فأنت في المكان المناسب! هيا بنا نبدأ.
## المتطلبات الأساسية
قبل أن نتعمق في التفاصيل، هناك بعض المتطلبات الأساسية التي ستحتاج إلى التحقق منها في قائمتك:
1. الفهم الأساسي لـ .NET: ستكون المعرفة بـ C# أو VB.NET مفيدة للغاية.
2. Aspose.Cells لـ .NET: تأكد من تثبيت مكتبة Aspose.Cells في مشروعك. إذا لم تكن قد قمت بذلك بعد، يمكنك [قم بتحميله هنا](https://releases.aspose.com/cells/net/).
3. Visual Studio: بيئة تطوير مثل Visual Studio أو أي IDE متوافق مع .NET من شأنها تسهيل عملية الترميز والتصحيح.
4. مدير حزمة NuGet: لتسهيل تثبيت Aspose.Cells في مشروعك. 
بمجرد إعدادك لهذه المتطلبات الأساسية، سنكون جاهزين للانطلاق!
## استيراد الحزم
لبدء استخدام Aspose.Cells، ستحتاج أولًا إلى استيراد مساحات الأسماء اللازمة إلى مشروعك. سيُتيح لك هذا الوصول إلى الفئات والأساليب اللازمة للتعامل مع ملفات Excel. إليك الطريقة:
### افتح مشروعك
افتح مشروع .NET الخاص بك في Visual Studio.
### تثبيت Aspose.Cells
إذا لم تقم بالفعل بتثبيت حزمة Aspose.Cells، فيمكنك القيام بذلك عبر NuGet. 
1. انتقل إلى الأدوات -> مدير حزمة NuGet -> إدارة حزم NuGet للحل.
2. ابحث عن Aspose.Cells.
3. انقر فوق تثبيت.
### استيراد مساحة الاسم
في أعلى ملف C# الخاص بك، قم بتضمين السطر التالي:
```csharp
using System.IO;
using Aspose.Cells;
```
أنت الآن جاهز لبدء الترميز!
في هذا القسم، سنرشدك خلال عملية حفظ ملف Excel بتنسيق 97-2003 (.xls) باستخدام Aspose.Cells. سنشرحها في خطوات سهلة.
## الخطوة 1: إعداد دليل المستندات
أولاً وقبل كل شيء! ستحتاج إلى تحديد المجلد الذي ستحفظ فيه ملف Excel.
```csharp
string dataDir = "Your Document Directory";
```
- `"Your Document Directory"`:استبدل سلسلة النص هذه بالمسار الفعلي الذي تريد حفظ ملف Excel فيه. قد يكون مثل: `"C:\\ExcelFiles\\"`.
## الخطوة 2: إنشاء كائن مصنف جديد
بعد ذلك، دعنا ننشئ مثيلًا جديدًا لـ `Workbook` الصف. هنا يحدث كل السحر!
```csharp
Workbook workbook = new Workbook();
```
- `Workbook`:تمثل هذه الفئة ملف Excel الذي تعمل عليه. بإنشاء مثيل لها، ستنشئ في الأساس مصنفًا فارغًا جديدًا.
## الخطوة 3: احفظ المصنف بتنسيق 97-2003
هذه هي اللحظة التي كنت تنتظرها! حان وقت حفظ مصنفك. هناك طريقتان للقيام بذلك.
### حفظ بسيط
استخدم الكود التالي لحفظ ملفك مباشرة في المسار المحدد.
```csharp
workbook.Save(dataDir + "output.xls");
```
### حفظ بالتنسيق المحدد
يمكنك أيضًا تحديد تنسيق الحفظ صراحةً:
```csharp
workbook.Save(dataDir + "output.xls", SaveFormat.Excel97To2003);
```
- `output.xls`هذا هو اسم الملف الذي تحفظه. يمكنك إعادة تسميته حسب رغبتك.
- `SaveFormat.Excel97To2003`:يضمن هذا حفظ ملفك بتنسيق Excel 97-2003.
