## دليل التنفيذ

دعونا نصل إلى الحد الأقصى لنطاق العرض الخاص بورقة العمل خطوة بخطوة.

### ملخص

يتيح الوصول إلى أقصى نطاق عرض معرفة أي جزء من ورقة Excel مرئي. يُعد هذا مفيدًا لمجموعات البيانات الكبيرة حيث قد يتم عرض مجموعة فرعية فقط في أي وقت.

#### الخطوة 1: إنشاء كائن مصنف

إنشاء مثيل لـ `Workbook` الفئة لتحميل ملف Excel الخاص بك:

```csharp
// دليل المصدر
total_sourceDir = RunExamples.Get_SourceDirectory();

// إنشاء كائن مصنف
Workbook workbook = new Workbook(sourceDir + "sampleAccessingMaximumDisplayRangeofWorksheet.xlsx");
```

#### الخطوة 2: الوصول إلى ورقة العمل

استرجع ورقة العمل التي تريد العمل عليها. عادةً، هذه هي الورقة الأولى:

```csharp
// الوصول إلى المصنف الأول
Worksheet worksheet = workbook.Worksheets[0];
```

#### الخطوة 3: استرداد نطاق العرض الأقصى

استخدم `MaxDisplayRange` ممتلكات `Cells` مجموعة للحصول على النطاق:

```csharp
// الوصول إلى الحد الأقصى لنطاق العرض
Range range = worksheet.Cells.MaxDisplayRange;
```

#### الخطوة 4: إخراج النتيجة

اطبع أو استخدم معلومات نطاق العرض الأقصى حسب الحاجة:

```csharp
// طباعة خاصية "النطاق الأقصى للعرض يشير إلى"
Console.WriteLine("Maximum Display Range: " + range.RefersTo);
Console.WriteLine("AccessingMaximumDisplayRangeofWorksheet executed successfully.");
```
