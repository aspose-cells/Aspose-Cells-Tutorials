## دليل التنفيذ

### إنشاء نطاق من الخلايا وتسميته في ورقة عمل Excel
ستوضح لك هذه الميزة كيفية إنشاء نطاق محدد داخل ورقة عمل وتعيين اسم له لسهولة الرجوع إليه.

#### ملخص
ستتعلم كيفية تعريف نطاق خلايا من A1 إلى C10 وتسمية هذا النطاق باستخدام مرجع الورقة، مما يجعل بياناتك أكثر سهولة في الوصول إليها.

#### خطوات التنفيذ

##### الخطوة 1: تهيئة المصنف
إنشاء مثيل لـ `Workbook` وهو يمثل ملف Excel.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مصنف جديد
Workbook workbook = new Workbook();
```

##### الخطوة 2: الوصول إلى مجموعة أوراق العمل والخلايا
احصل على إمكانية الوصول إلى ورقة العمل الأولى في مصنفك ومجموعة خلاياها.
```csharp
// احصل على ورقة العمل الأولى من المصنف
Worksheet sheet = workbook.Worksheets[0];

// الوصول إلى مجموعة خلايا ورقة العمل
Cells cells = sheet.Cells;
```

##### الخطوة 3: إنشاء نطاق من الخلايا
قم بتحديد نطاق داخل الخلايا، مع تحديد مواضع البداية والنهاية المطلوبة.
```csharp
// إنشاء نطاق من الخلايا من A1 إلى C10
Range localRange = cells.CreateRange("A1", "C10");
```

##### الخطوة 4: تعيين اسم مع مرجع الورقة
قم بتسمية النطاق الذي تم إنشاؤه لتسهيل التعرف عليه والرجوع إليه داخل الصيغ أو البرامج النصية.
```csharp
// تعيين اسم مع مرجع الورقة إلى النطاق الذي تم إنشاؤه
localRange.Name = "Sheet1!local";
```

##### الخطوة 5: احفظ مصنفك
حافظ على تغييراتك عن طريق حفظ المصنف في الدليل المحدد.
```csharp
// حفظ المصنف في دليل الإخراج المحدد
workbook.Save(Path.Combine(outputDir, "outputWorksheetNamedRange.xlsx"));
```
