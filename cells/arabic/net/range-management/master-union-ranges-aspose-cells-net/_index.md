## دليل التنفيذ
في هذا القسم، سنرشدك خلال عملية توحيد النطاقات وتطبيق الأنماط.

### استرجاع النطاقات المسماة
أولاً، قم بالوصول إلى النطاقات المسماة داخل مصنف Excel الخاص بك:
```csharp
// افتح ملف Excel الموجود.
Workbook workbook = new Workbook("sampleUnionOfRanges.xlsx");

// احصل على النطاقات المسماة من ورقة العمل الأولى.
Range[] ranges = workbook.Worksheets[0].GetNamedRanges();
```
**توضيح**: ال `GetNamedRanges` تسترجع الطريقة جميع النطاقات المسماة المحددة في ورقة العمل المحددة، مما يسمح بالتلاعب بها.

### إنشاء الأنماط وتطبيقها
لتمييز النطاقات الموحدة بصريًا، قم بتطبيق نمط مخصص:
```csharp
// إنشاء كائن نمط جديد.
Style style = workbook.CreateStyle();

// تعيين لون الخلفية إلى اللون الأحمر مع نوع النمط الصلب.
style.ForegroundColor = Color.Red;
style.Pattern = BackgroundType.Solid;

// قم بتهيئة StyleFlag لتحديد عناصر الخلية التي سيتم تصميمها.
StyleFlag flag = new StyleFlag();
flag.CellShading = true; // نحن نطبق التظليل
```

### أداء عملية الاتحاد
الآن، قم بإجراء عملية الاتحاد على النطاقات المحددة لديك:
```csharp
// قم بإنشاء ArrayList لتخزين نتيجة عملية الاتحاد.
ArrayList al = ranges[0].Union(ranges[1]);
```
**توضيح**: ال `Union` تجمع هذه الطريقة نطاقات متعددة في مجموعة نطاقات واحدة. نستخدم `ArrayList` هنا من أجل البساطة، ولكن قم بتعديل هذا حسب الحاجة.

### تطبيق الأنماط على النطاقات الموحدة
بمجرد التوحيد، قم بتطبيق الأنماط:
```csharp
foreach (Range rng in al)
{
    // قم بتطبيق النمط الذي تم إنشاؤه مسبقًا على كل نطاق.
    rng.ApplyStyle(style, flag);
}
```
**توضيح**: ال `ApplyStyle` تستخدم الطريقة كائنات النمط المخصصة لدينا والأعلام لتنسيق كل خلية ضمن النطاقات الموحدة.

### حفظ المصنف
وأخيرًا، احفظ التغييرات:
```csharp
// احفظ المصنف باستخدام النطاقات المصممة.
workbook.Save("outputUnionOfRanges.xlsx");
```
