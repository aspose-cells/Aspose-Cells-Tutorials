## دليل التنفيذ

### إنشاء نطاق وتسميته

أولاً، لنركز على إنشاء نطاق وتسميته ضمن ورقة عمل Excel. تتيح لك هذه الميزة الرجوع بسهولة إلى أقسام محددة في ورقة العمل دون الحاجة إلى حفظ مراجع الخلايا.

#### تهيئة المصنف وورقة العمل
```csharp
// فتح ملف Excel من خلال إنشاء مثيل مصنف جديد
Workbook workbook = new Workbook();

// الوصول إلى ورقة العمل الأولى في ملف Excel الذي تم إنشاؤه حديثًا
Worksheet worksheet = workbook.Worksheets[0];
```

هنا نقوم بإنشاء جديد `Workbook` كائن يمثل ملف Excel بأكمله. ثم نصل إلى ورقة العمل الأولى الخاصة به.

#### تعريف وتسمية النطاق
```csharp
// إنشاء نطاق من الخلايا من B4 إلى G14
Range range = worksheet.Cells.CreateRange("B4", "G14");

// تعيين اسم النطاق المسمى إلى 'TestRange'
range.Name = "TestRange";
```

في هذه الخطوة، نقوم بتعريف نطاق خلية يمتد من B4 إلى G14 ونقوم بتعيين اسم له، `TestRange`تساعد تسمية النطاقات على تعزيز الوضوح عند العمل مع مجموعات البيانات المعقدة.

### تصميم النطاق المسمى

بعد إنشاء نطاقك المُسمّى، يمكنك تطبيق أنماط مخصصة لجعله مميزًا بصريًا. هذا مفيد بشكل خاص لتسليط الضوء على أقسام البيانات المهمة.

#### إنشاء النمط وتطبيقه
```csharp
// إنشاء وتكوين نمط للنطاق بلون خلفية ثابت
Style st = workbook.CreateStyle();
st.Pattern = BackgroundType.Solid;
st.ForegroundColor = System.Drawing.Color.Yellow;

// تطبيق النمط الذي تم إنشاؤه على النطاق المحدد
range.SetStyle(st);
```

هنا، نقوم بإنشاء `Style` الكائن وتهيئته بخلفية صفراء صلبة. ثم نُطبّق هذا النمط على النطاق المُسمّى، مما يُحسّن وضوحه.

### احفظ مصنفك

وأخيرًا، احفظ تعديلاتك مرة أخرى في ملف Excel:
```csharp
// حفظ ملف Excel المعدّل في دليل الإخراج المخصص
workbook.Save("outputCreateNamedRangeofCells.xlsx");
```

تضمن هذه الخطوة أن يتم الاحتفاظ بجميع التغييرات في ملف جديد يسمى `outputCreateNamedRangeofCells.xlsx`.
