### إنشاء نطاق الاتحاد واستخدامه في مصنف Excel

#### ملخص

يتيح لك إنشاء نطاق موحد إدارة نطاقات خلايا متعددة كما لو كانت خلية واحدة. وهذا مفيد بشكل خاص لضبط القيم في أعمدة مختلفة بكفاءة.

#### التنفيذ خطوة بخطوة

##### 1. إنشاء كائن المصنف

ابدأ بإنشاء مثيل لـ `Workbook` فصل:

```csharp
using Aspose.Cells;

// تعريف الدلائل
cstring sourceDir = "YOUR_SOURCE_DIRECTORY";
cstring outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مصنف جديد
Workbook workbook = new Workbook();
```

##### 2. إنشاء نطاق الاتحاد

بعد ذلك، قم بإنشاء نطاق اتحاد يمتد عبر الخلايا عبر أعمدة مختلفة:

```csharp
// إنشاء نطاق اتحاد لـ A1:A10 وC1:C10 على "sheet1"
UnionRange unionRange = workbook.Worksheets.CreateUnionRange("sheet1!A1:A10,sheet1!C1:C10", 0);
```

- **حدود:** الخيط `"sheet1!A1:A10,sheet1!C1:C10"` يحدد نطاقات الخلايا التي سيتم تضمينها في الاتحاد.
- **فهرس ورقة العمل:** `0` يشير إلى ورقة العمل الأولى (`"sheet1"`).

##### 3. تعيين القيم

تعيين قيمة لجميع الخلايا ضمن نطاق الاتحاد:

```csharp
// تعيين "ABCD" كقيمة لنطاق الاتحاد
unionRange.Value = "ABCD";
```

##### 4. احفظ المصنف

وأخيرًا، احفظ التغييرات في ملف الإخراج:

```csharp
// حفظ المصنف في الدليل المحدد
workbook.Save(outputDir + "CreateUnionRange_out.xlsx");
```

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من تنسيق اسم الورقة وعناوين النطاق بشكل صحيح.
- تأكد من وجود أدلة لمسارات المصدر والإخراج قبل الحفظ.
