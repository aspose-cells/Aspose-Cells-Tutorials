## دليل التنفيذ

### إنشاء وإدارة نطاقات السماح بالتحرير

#### ملخص
تتيح هذه الميزة للمستخدمين تحديد مناطق قابلة للتحرير داخل ورقة عمل Excel المحمية، وهي مثالية للمواقف التي تحتاج فيها حقول بيانات معينة فقط إلى التعديل من قبل المستخدمين النهائيين مع الحفاظ على أمان بقية الورقة.

#### التنفيذ خطوة بخطوة

**1. إعداد الدلائل**
أولاً، تأكد من أن الدلائل الخاصة بالمصدر والإخراج جاهزة:
```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// تحقق مما إذا كان دليل الإخراج موجودًا؛ قم بإنشائه إذا لم يكن موجودًا
bool isExists = Directory.Exists(outputDir);
if (!isExists)
    Directory.CreateDirectory(outputDir);
```
يتحقق مقتطف التعليمات البرمجية هذا من وجود الدلائل المحددة ويقوم بإنشائها إذا لزم الأمر، مما يضمن التعامل السلس مع الملفات.

**2. تهيئة المصنف**
إنشاء مثيل جديد لمصنف Excel:
```csharp
using Aspose.Cells;

// إنشاء كائن مصنف جديد
Workbook book = new Workbook();
```
سنقوم هنا بإنشاء مصنف Excel فارغًا ليكون بمثابة مستند العمل الخاص بنا.

**3. إضافة نطاق السماح بالتحرير**
الوصول إلى المناطق القابلة للتحرير في ورقة العمل وتكوينها:
```csharp
Worksheet sheet = book.Worksheets[0];
ProtectedRangeCollection allowRanges = sheet.AllowEditRanges;

// أضف نطاقًا محميًا جديدًا بمعلمات محددة: الاسم، ومؤشر الصف/العمود الابتدائي، والحجم بالصفوف/الأعمدة
int idx = allowRanges.Add("r2", 1, 1, 3, 3);
ProtectedRange protected_range = allowRanges[idx];

// تعيين كلمة مرور لهذا النطاق القابل للتحرير المحدد
protected_range.Password = "123";
```
يُعرّف هذا الكود نطاقًا قابلًا للتعديل باسم "r2" يبدأ من الصف والعمود الثاني، ويمتد على ثلاثة صفوف وأعمدة. ثم يُعيّن كلمة مرور لتقييد الوصول.

**4. حماية ورقة العمل**
قم بتأمين ورقة العمل الخاصة بك عن طريق تمكين الحماية:
```csharp
// تطبيق الحماية مع تمكين جميع الأنواع المتاحة
sheet.Protect(ProtectionType.All);
```
من خلال استدعاء هذه الطريقة، فإننا نضمن عدم إمكانية إجراء أي تعديلات خارج نطاقات التحرير المسموح بها المحددة.

**5. حفظ مصنفك**
أخيرًا، احفظ المصنف الخاص بك في دليل الإخراج المخصص:
```csharp
book.Save(Path.Combine(outputDir, "protectedrange.out.xls"));
```
تنهي هذه الخطوة عمليتنا عن طريق كتابة جميع التغييرات في ملف Excel باسم "protectedrange.out.xls" في الموقع المحدد.

**1. هل يمكنني تغيير كلمة المرور لنطاق التحرير المسموح به الحالي؟**
