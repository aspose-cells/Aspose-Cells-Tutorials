# تحديد حقول الصيغة عند استيراد البيانات إلى ورقة Excel

## مقدمة

عندما يتعلق الأمر بالتعامل مع ملفات Excel برمجيًا، فإن Aspose.Cells for .NET هي أداة لا تقدر بثمن. فهي توفر وظائف قوية لإنشاء جداول بيانات Excel وتعديلها والتلاعب بها بسهولة. ومن بين الميزات المثيرة للاهتمام التي تقدمها القدرة على تحديد حقول الصيغة أثناء استيراد البيانات إلى ورقة Excel. تخيل أنك تعمل على تقرير مالي وتحتاج إلى حساب الإجماليات تلقائيًا استنادًا إلى إدخال المستخدم. سيرشدك هذا البرنامج التعليمي خطوة بخطوة لتحقيق ذلك تمامًا من خلال نهج واضح ومباشر.

## المتطلبات الأساسية

قبل الغوص في الكود، دعنا نتأكد من أن لديك كل ما تحتاجه. 

1. Visual Studio أو أي بيئة تطوير متكاملة (IDE) لـ .NET: تأكد من أن لديك بيئة تطوير متكاملة مناسبة لكتابة وتشغيل كود C#.
2.  Aspose.Cells لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.Cells والرجوع إليها في مشروعك. يمكنك تنزيلها من[إصدارات Aspose](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة بلغة C# ومفاهيم البرمجة الموجهة للكائنات على فهم الأمثلة بشكل أفضل.
4. .NET Framework: يفترض هذا البرنامج التعليمي أنك تستخدم .NET Framework 4.5 أو أعلى.

بمجرد الانتهاء من المتطلبات الأساسية، دعنا ننتقل إلى استيراد بعض البيانات إلى ورقة Excel مع حقول الصيغة المحددة.

## استيراد الحزم

قبل أن تبدأ في كتابة الكود الخاص بك، ستحتاج إلى استيراد مساحة اسم Aspose.Cells الضرورية. يتم ذلك عادةً في الجزء العلوي من ملف C# الخاص بك:

```csharp
using Aspose.Cells;
using System;
using System.Collections.Generic;
```

يتيح لك هذا استخدام الفئات والطرق التي توفرها مكتبة Aspose.Cells دون الحاجة إلى إضافة بادئة لها باستخدام مساحة الاسم في كل مرة.

دعونا نقسم العملية بأكملها إلى خطوات قابلة للإدارة:

## الخطوة 1: تحديد دليل الإخراج

أولاً، عليك تحديد المكان الذي تريد حفظ ملف Excel فيه. إليك كيفية القيام بذلك:

```csharp
static string outputDir = "Your Document Directory"; // حدد دليل المستند الخاص بك هنا
```

 يستبدل`"Your Document Directory"` مع مسار الملف الفعلي الخاص بك. هذا هو المكان الذي سيتم فيه حفظ ملف Excel الناتج.

## الخطوة 2: إنشاء فئة محددة للمستخدم لعناصر البيانات

بعد ذلك، سنقوم بتعريف فئة لتنظيم البيانات التي نخطط لاستيرادها.

```csharp
class DataItems
{
    public int Number1 { get; set; }
    public int Number2 { get; set; }
    public string Formula1 { get; set; }
    public string Formula2 { get; set; }
}
```

 هذا`DataItems` ستحتوي الفئة على الأعداد الصحيحة الخام والصيغ التي سنكتبها في ورقة Excel. 

## الخطوة 3: تهيئة قائمة لاحتواء عناصر البيانات

 سنستخدم قائمة لحفظ مثيلات متعددة من`DataItems` فصل.

```csharp
List<DataItems> dis = new List<DataItems>();
```

## الخطوة 4: إضافة عناصر البيانات إلى القائمة

الآن، دعنا نضيف بعض الإدخالات إلى قائمتنا. سيحتوي كل إدخال على رقمين وصيغتين.

```csharp
// قم بتحديد كل عنصر بيانات وإضافته
DataItems di = new DataItems();
di.Number1 = 2002;
di.Number2 = 3502;
di.Formula1 = "=SUM(A2,B2)";
di.Formula2 = "=HYPERLINK(\"https://www.aspose.com\"،\"موقع Aspose\")";
dis.Add(di);

// كرر ذلك للحصول على عناصر بيانات إضافية
```

 تأكد من تخصيص كل منها`DataItems` مثال بقيم وصيغ فريدة.

## الخطوة 5: إنشاء مصنف وورقة عمل Access

بعد ذلك، قم بإنشاء المصنف والوصول إلى ورقة العمل الأولى التي سنقوم فيها في النهاية باستيراد البيانات.

```csharp
Workbook wb = new Workbook(); // إنشاء مصنف جديد
Worksheet ws = wb.Worksheets[0]; // الوصول إلى ورقة العمل الأولى
```

## الخطوة 6: تحديد خيارات استيراد الجدول

وهنا يحدث السحر. يتعين عليك تحديد الحقول في بياناتك التي تتوافق مع الصيغ. 

```csharp
ImportTableOptions opts = new ImportTableOptions();
opts.IsFormulas = new bool[] { false, false, true, true };
```

 في هذا المثال، يحتوي الحقلان الأخيران على صيغ، وهو ما يشار إليه بواسطة`true` في حين تم تعيين الحقلين الأولين على`false`.

## الخطوة 7: استيراد الكائنات المخصصة

الآن بعد أن تم إعداد كل شيء، فلنقم باستيراد قائمة عناصر البيانات إلى ورقة العمل.

```csharp
ws.Cells.ImportCustomObjects(dis, 0, 0, opts);
```

يقوم هذا الخط باستيراد البيانات فعليًا بدءًا من الخلية A1.

## الخطوة 8: حساب الصيغ

بما أننا قمنا باستيراد بعض الصيغ، فمن الضروري أن نقوم بحسابها.

```csharp
wb.CalculateFormula();
```

تضمن هذه الطريقة تقييم الصيغ الخاصة بك استنادًا إلى تبعياتها.

## الخطوة 9: ملاءمة الأعمدة تلقائيًا

لتتأكد من أن بياناتك قابلة للعرض، يمكنك ضبط الأعمدة تلقائيًا استنادًا إلى المحتوى.

```csharp
ws.AutoFitColumns();
```

تعمل هذه الخطوة على تحسين تخطيط ملف Excel. 

## الخطوة 10: احفظ ملف Excel الخاص بك

وأخيرًا، حان الوقت لحفظ ملف Excel الذي قمت بإنشائه حديثًا. 

```csharp
wb.Save(outputDir + "outputSpecifyFormulaFieldsWhileImportingDataToWorksheet.xlsx");
```

تأكد من أن اسم ملف الإخراج الخاص بك ذو صلة ووصفي!

## الخطوة 11: التحقق من التنفيذ

كطريقة بسيطة للتأكد من أن كل شيء يعمل بشكل صحيح، قد ترغب في طباعة رسالة.

```csharp
Console.WriteLine("SpecifyFormulaFieldsWhileImportingDataToWorksheet executed successfully.");
```

ويمنحك هذا ردود فعل فورية بأن الكود يعمل دون أي مشاكل.

## خاتمة

والآن، لقد نجحت في استيراد البيانات إلى ورقة Excel باستخدام Aspose.Cells for .NET وحقول الصيغة المحددة. باتباع هذه الخطوات، يمكنك تطبيق تقنيات مماثلة لأتمتة مهام معالجة البيانات المخصصة لاحتياجاتك. سواء كنت تقوم بتحليل الأرقام للتقارير أو مجرد صيانة البيانات، فإن إتقان فن معالجة Excel باستخدام Aspose يعد مهارة تستحق اكتسابها.
