# ضبط تنسيق حقل الصفحة برمجيًا في .NET

## مقدمة
إنشاء ملفات Excel ومعالجتها باستخدام الشيفرة البرمجية أمرٌ مُمكِّن، خاصةً عند الحاجة إلى تحليل مجموعات بيانات ضخمة. من الأدوات الرائعة التي يُمكنك استخدامها Aspose.Cells for .NET، والتي تُتيح لك التفاعل برمجيًا مع ملفات Excel وإنشاء هياكل تقارير مُعقدة. في هذا البرنامج التعليمي، سنتعمق في كيفية إعداد تنسيقات حقول الصفحات داخل PivotTable باستخدام هذه المكتبة القوية. سواءً كنتَ مُطورًا خبيرًا أو مبتدئًا، ستُصبح بنهاية هذا الدليل مُتمكنًا من كيفية التعامل مع PivotTables وإعداداتها المُختلفة في .NET.
## المتطلبات الأساسية
قبل أن نتعمق في البرمجة، لنتأكد من إعداد كل شيء بشكل صحيح. ستحتاج إلى ما يلي:
- Visual Studio: بيئة عمل يمكنك من خلالها كتابة وتنفيذ كود .NET الخاص بك.
- Aspose.Cells: يمكنك تنزيل المكتبة [هنا](https://releases.aspose.com/cells/net/).
- المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.
- ملف Excel: قم بإعداد ملف Excel (مثل `Book1.xls`) تحتوي على بيانات مناسبة لإنشاء جدول محوري. 
إذا لم تقم بذلك بالفعل، فاحصل على النسخة التجريبية المجانية من Aspose.Cells [هنا](https://releases.aspose.com/).
## استيراد الحزم
للبدء، ستحتاج إلى استيراد الحزم المناسبة في مشروعك. ابدأ بإضافة مراجع إلى مكتبة Aspose.Cells في مشروع C#. إليك كيفية القيام بذلك:
```csharp
using System.IO;
using Aspose.Cells;
using System.Drawing;
using Aspose.Cells.Pivot;
```
سيؤدي هذا إلى جلب جميع الفئات والطرق الضرورية اللازمة لمعالجة ملفات Excel باستخدام Aspose.Cells.
## الخطوة 1: إعداد مساحة العمل الخاصة بك
ابدأ بتحديد مجلد العمل الذي ستُخزَّن فيه ملفات Excel. على سبيل المثال، يمكنك إعلان متغير كالتالي:
```csharp
string dataDir = "Your Document Directory";
```
## تحميل المصنف
بعد ذلك، علينا تحميل قالب إكسل. هذه خطوة أساسية لأنها تُحدد سياق عملياتنا:
```csharp
Workbook workbook = new Workbook(dataDir + "Book1.xls");
```
يقوم هذا السطر بتحميل المصنف الموجود من الدليل المحدد.
## الخطوة 2: الوصول إلى ورقة العمل
بعد تحميل مصنفك، حان الوقت للوصول إلى ورقة العمل التي تحتوي على الجدول المحوري أو البيانات التي تريد تحليلها. إليك كيفية القيام بذلك:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
يؤدي هذا إلى التقاط أول ورقة عمل من المصنف المُحمّل. يمكنك تعديل الفهرس بسهولة إذا كنت تعمل على أوراق عمل متعددة.
## الخطوة 3: الوصول إلى الجدول المحوري
نواصل، لنصل إلى الجدول المحوري في ورقة العمل التي اخترناها. إذا كنت تستخدم جدولًا محوريًا واحدًا، يمكنك ضبط فهرسه إلى `0`:
```csharp
int pivotindex = 0;
// الوصول إلى جدول البيانات المحوري
PivotTable pivotTable = worksheet.PivotTables[pivotindex];
```
يقوم مقتطف التعليمات البرمجية هذا بتحديد أول جدول محوري في ورقة العمل. 
## الخطوة 4: تكوين الجدول المحوري
الآن يأتي الجزء المثير! لنُعِد جدول البيانات المحوري لعرض الإجماليات الكلية للصفوف:
```csharp
pivotTable.RowGrand = true;
```
يضمن هذا السطر أن يعرض تقريرك الإجمالي الكلي، والذي يمكن أن يكون بمثابة ملخص مفيد لتحليل البيانات.
## الخطوة 5: الوصول إلى حقول الصفوف وتكوينها
بعد ذلك، نحتاج إلى الوصول إلى حقول الصفوف في جدول البيانات المحوري:
```csharp
Aspose.Cells.Pivot.PivotFieldCollection pivotFields = pivotTable.RowFields;
```
تسمح لنا هذه المجموعة بالتعامل مع الحقول حسب الحاجة.
## تكوين حقل الصف الأول
هل تريد تحديد أنواع مجاميع فرعية محددة؟ لننتقل إلى الحقل الأول في مجموعتنا ونُهيئه:
```csharp
Aspose.Cells.Pivot.PivotField pivotField = pivotFields[0];
// تعيين المجموع الفرعي.
pivotField.SetSubtotals(Aspose.Cells.Pivot.PivotFieldSubtotalType.Sum, true);
pivotField.SetSubtotals(Aspose.Cells.Pivot.PivotFieldSubtotalType.Count, true);
```
عن طريق تمكين `Sum` و `Count` من خلال المجاميع الفرعية، يمكننا تلخيص البيانات بسرعة في تقريرنا.
## الخطوة 6: ضبط خيارات الفرز التلقائي
الآن، لنبدأ بفرز البيانات بذكاء. بهذه الطريقة، سيرتب جدولك المحوري البيانات بترتيب منطقي:
```csharp
// إعداد خيارات الفرز التلقائي.
pivotField.IsAutoSort = true;
pivotField.IsAscendSort = true;
pivotField.AutoSortField = -5; // استخدام حقل فرز محدد مسبقًا.
```
يتيح مقتطف التعليمات البرمجية هذا الفرز التلقائي ويحدد الترتيب التصاعدي. 
## الخطوة 7: ضبط خيارات العرض التلقائي
هل ترغب في تصفية بياناتك بشكل أكبر؟ خيار "العرض التلقائي" مفيد لعرض نقاط بيانات محددة في ظل ظروف محددة:
```csharp
// ضبط خيارات العرض التلقائي.
pivotField.IsAutoShow = true;
pivotField.IsAscendShow = false;
pivotField.AutoShowField = 0; // حدد الحقل لإظهاره تلقائيًا.
```
يضمن هذا أن يعرض جدول البيانات المحوري الخاص بك البيانات ذات الصلة فقط، مما يعزز الوضوح والتركيز.
## الخطوة 8: حفظ عملك
بعد كل هذه الإعدادات، لا ترغب بفقدان عملك! احفظ المصنف المُعدَّل كما يلي:
```csharp
workbook.Save(dataDir + "output.xls");
```
الآن، يمكنك العثور على ملف Excel الذي تم إنشاؤه حديثًا في دليل المستندات الخاص بك.
