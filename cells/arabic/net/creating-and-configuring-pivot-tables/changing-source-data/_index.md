# تغيير بيانات المصدر لجدول محوري برمجيًا في .NET

## مقدمة
في عالم تحليل البيانات، قلّما تجد أدواتٍ تضاهي مايكروسوفت إكسل. يعتمد عددٌ لا يُحصى من المستخدمين يوميًا على إكسل لإدارة البيانات وتحليلها، ولكن في الواقع، الأمر أكثر تعقيدًا بكثير من مجرد النقر والسحب. إذا كنت ترغب يومًا في معالجة ملفات إكسل برمجيًا، وتحديدًا لتغيير بيانات المصدر لجدول محوري، فأنت في المكان المناسب! في هذا الدليل، سنستكشف كيفية تحقيق ذلك باستخدام Aspose.Cells لـ .NET. سواءً كنت مطورًا متمرسًا أو مبتدئًا في عالم البرمجة، ستجد هذا الدليل التعليمي غنيًا بالمعلومات القيّمة سهلة الفهم.
## المتطلبات الأساسية
قبل أن نبدأ رحلتنا لتغيير بيانات المصدر لجدول محوري، دعنا نتأكد من إعداد كل شيء وتجهيزه:
1. Visual Studio: تأكد من تثبيت نسخة من Microsoft Visual Studio، حيث سنقوم بكتابة الكود الخاص بنا هنا.
2. مكتبة Aspose.Cells: ستحتاج إلى تنزيل مكتبة Aspose.Cells والإشارة إليها في مشروعك. يمكنك تنزيلها. [هنا](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: على الرغم من تبسيط هذا البرنامج التعليمي، فإن فهم لغة C# سيساعدك على فهم الكود بشكل أفضل.
4. ملف Excel: يجب أن يكون لديك ملف Excel نموذجي (مثل "Book1.xlsx") يحتوي على جدول محوري يمكننا التعامل معه.
حسنًا، مع التحقق من هذه المتطلبات الأساسية، يمكننا المضي قدمًا في استيراد الحزم الضرورية والحصول على الترميز!
## استيراد الحزم
أولاً، لنستورد الحزم التي نحتاجها. افتح مشروع C# في Visual Studio، وأضف التعليمات البرمجية التالية في أعلى ملف الكود:
```csharp
using System.IO;
using Aspose.Cells;
using System.Drawing;
```
ستتيح لك هذه المساحات الاسمية الوصول إلى الفئات الأساسية اللازمة للعمل مع ملفات Excel ومعالجة محتوياتها باستخدام Aspose.Cells.

الآن، لنُقسّم العملية إلى خطوات سهلة. سنشرح فتح ملف Excel، وتعديل ورقة العمل، وتغيير مصدر بيانات الجدول المحوري، وحفظ النتائج.
## الخطوة 1: تحديد دليل المستندات الخاص بك
أولاً، عليك تحديد مكان ملف Excel. عدّل `dataDir` متغير للإشارة إلى المجلد الذي يحتوي على "Book1.xlsx".
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
```
يقوم هذا السطر بإعداد الدليل الذي سيتم تخزين ملف Excel فيه، مما يجعل الوصول إليه أسهل لاحقًا.
## الخطوة 2: تحديد مسار الإدخال
بعد ذلك، دعنا ننشئ سلسلة لتحديد المسار الكامل لملف Excel المدخل الخاص بك:
```csharp
string InputPath = dataDir + "Book1.xlsx";
```
يساعد هذا في تبسيط الوصول إلى ملفك؛ فلن تضطر إلى الاستمرار في كتابة نفس المسار عدة مرات خلال الكود الخاص بك.
## الخطوة 3: إنشاء تدفق ملف
الآن حان وقت فتح ملف Excel. سننشئ `FileStream` الذي يسمح لك بقراءة محتوى ملف Excel:
```csharp
// إنشاء مجرى ملف يحتوي على ملف Excel الذي سيتم فتحه
FileStream fstream = new FileStream(InputPath, FileMode.Open);
```
يفتح هذا السطر الملف في وضع القراءة، مما يسمح لنا بالوصول إلى بياناته.
## الخطوة 4: تحميل المصنف
مع وجود تدفق الملف في مكانه، فإن الخطوة التالية هي تحميل المصنف:
```csharp
// فتح ملف Excel من خلال تدفق الملف
Workbook workbook = new Workbook(fstream);
```
يقوم هذا الأمر بأخذ ملف Excel الخاص بك وتحميله في `Workbook` بمجرد تحميل الملف، يمكنك التعامل معه حسب الحاجة.
## الخطوة 5: الوصول إلى ورقة العمل
حان وقت التعمق في التفاصيل. سننتقل إلى ورقة العمل الأولى في المصنف:
```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];
```
يتيح لك هذا الوصول المباشر إلى البيانات داخل ورقة العمل الأولى، مما يسهل تعديلها.
## الخطوة 6: ملء البيانات الجديدة
بعد ذلك، نريد إدراج بيانات جديدة في الخلايا. في هذا المثال، سنضيف بعض البيانات النموذجية:
```csharp
// ملء البيانات الجديدة في خلايا ورقة العمل
worksheet.Cells["A9"].PutValue("Golf");
worksheet.Cells["B9"].PutValue("Qtr4");
worksheet.Cells["C9"].PutValue(7000);
```
هنا، نضع القيم "Golf" و"Qtr4" و `7000` في خلايا محددة. يمكنك تغيير هذه القيم بما يناسب احتياجاتك.
## الخطوة 7: تغيير النطاق المسمى
الآن، سنغيّر النطاق المُسمّى الذي يشير إليه الجدول المحوري. يتضمن ذلك إنشاء نطاق أو تحديثه:
```csharp
// تغيير النطاق المسمى "مصدر البيانات"
Range range = worksheet.Cells.CreateRange(0,0,9,3);
range.Name = "DataSource";
```
من خلال تحديد نطاق جديد، فإننا نضمن أن الجدول المحوري يستخدم هذه البيانات الجديدة عند تحديثه.
## الخطوة 8: حفظ ملف Excel المعدّل
بعد كل هذه التغييرات، من الضروري حفظ عملك! لنحفظ المصنف المعدّل:
```csharp
// حفظ ملف Excel المعدل
workbook.Save(dataDir + "output.xls");
```
يؤدي هذا الأمر إلى حفظ المصنف في ملف جديد، حتى لا تتمكن من الكتابة فوق الملف الأصلي إلا إذا كنت تريد ذلك!
## الخطوة 9: إغلاق مجرى الملف
أخيرًا، من الضروري إغلاق مجرى الملف لتحرير أي موارد تستخدمها:
```csharp
// إغلاق مجرى الملف لتحرير كافة الموارد
fstream.Close();
```
تضمن هذه الخطوة عدم تسرب الذاكرة في تطبيقك وبقاءه فعالاً.
## خاتمة
تهانينا! لقد نجحتَ في تغيير بيانات المصدر لجدول محوري برمجيًا في .NET باستخدام Aspose.Cells. تتيح لك هذه الوظيفة إمكانياتٍ عديدة لأتمتة مهام Excel وتحسين سير عملك. سواءً كنت تُحدّث التقارير المالية، أو تتبّع بيانات المبيعات، أو حتى تُجري تعديلاتٍ على مجموعات البيانات، فإنّ القدرة على القيام بذلك برمجيًا تُوفّر عليك الكثير من الوقت وتُقلّل من خطر الأخطاء.
