# إنشاء جدول محوري جديد برمجيًا في .NET

## مقدمة
قد يبدو إنشاء جدول محوري مهمةً شاقة، خاصةً عند تنفيذه برمجيًا. لكن لا تقلق! مع Aspose.Cells لـ .NET، لا يقتصر إنشاء جدول محوري على البساطة فحسب، بل يُعدّ أيضًا أداةً فعّالة لتحليل البيانات. في هذا البرنامج التعليمي، سنرشدك خطوة بخطوة حول كيفية إنشاء جدول محوري جديد في تطبيق .NET. سواءً كنت تُضيف بيانات للمبيعات أو الرياضة أو أي مقياس أعمال آخر، سيساعدك هذا الدليل على تشغيل جداولك المحورية بسرعة فائقة.

## المتطلبات الأساسية
قبل البدء، تأكد من تجهيز كل شيء. إليك ما عليك فعله:

1. تثبيت .NET Framework: تأكد من تثبيت .NET Framework على جهازك. يدعم Aspose.Cells إصدارات مختلفة، ولكن يُفضل استخدام الإصدار الأحدث.
2. مكتبة Aspose.Cells: يجب أن يكون لديك مكتبة Aspose.Cells. يمكنك [قم بتحميله هنا](https://releases.aspose.com/cells/net/) أو احصل على [رخصة مؤقتة](https://purchase.aspose.com/temporary-license/) للتقييم.
3. إعداد IDE: قم بإعداد IDE متوافق مع C#، مثل Visual Studio، حيث يمكنك بدء مشروع جديد.
4. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على المتابعة دون الوقوع في فخ كبير.

هل أنت جاهز؟ رائع! لنبدأ باستيراد الحزم اللازمة.

## استيراد الحزم
أولاً، عليك استيراد مساحات الأسماء المطلوبة إلى مشروع C#. افتح ملف C# وأضف ما يلي باستخدام التوجيهات:

```csharp
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

تتيح لك هذه المساحات الوصول إلى وظائف المصنف وورقة العمل والجدول المحوري التي سنستخدمها خلال هذا البرنامج التعليمي.

## الخطوة 1: إنشاء كائن مصنف
إنشاء مصنف عمل هو بداية رحلتك. لنبدأ بإنشاء مصنف عمل جديد والوصول إلى ورقة العمل الأولى.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء كائن مصنف
Workbook workbook = new Workbook();

// الحصول على مرجع ورقة العمل المضافة حديثًا
Worksheet sheet = workbook.Worksheets[0];
```

في هذه الخطوة، نقوم بإنشاء `Workbook` مثال يمثل ملف Excel الخاص بنا ونأخذ ورقة العمل الأولى، والتي ستكون بمثابة ساحة اللعب الخاصة بنا للجدول المحوري.

## الخطوة 2: إدراج البيانات في الخلايا
الآن، لنملأ ورقة العمل ببعض البيانات النموذجية. سنُدخل صفوفًا لمختلف الرياضات، والأرباع، وأرقام المبيعات، لنوفر لجدولنا المحوري ملخصًا.

```csharp
Cells cells = sheet.Cells;

// تعيين القيمة للخلايا
Cell cell = cells["A1"];
cell.PutValue("Sport");
cell = cells["B1"];
cell.PutValue("Quarter");
cell = cells["C1"];
cell.PutValue("Sales");

// ملء خلية البيانات = cells["A2"];
cell.PutValue("Golf");
// ... المزيد من إدخالات البيانات
```

هنا، نُحدِّد عناوين الأعمدة ونُدرِج القيم أسفل كل عنوان. ستكون هذه البيانات مصدرًا لجدولنا المحوري، لذا تأكد من تنظيمها! اتبع هذه الخطوات لإنشاء مجموعة بيانات شاملة.

## الخطوة 3: إضافة جدول محوري
بعد تجهيز بياناتنا، حان وقت إنشاء الجدول المحوري. سنستخدم مجموعة الجداول المحورية من ورقة العمل لإضافة الجدول المحوري الجديد.

```csharp
Aspose.Cells.Pivot.PivotTableCollection pivotTables = sheet.PivotTables;

// إضافة جدول محوري إلى ورقة العمل
int index = pivotTables.Add("=A1:C8", "E3", "PivotTable2");
```

في هذا المقطع، نضيف جدولًا محوريًا إلى ورقة العمل يشير إلى نطاق بياناتنا (في هذه الحالة، الخلايا من A1 إلى C8). نضع الجدول المحوري بدءًا من الخلية E3، ونسميه "PivotTable2". الأمر بسيط جدًا، أليس كذلك؟

## الخطوة 4: تخصيص جدول المحور
الآن وقد أصبح لدينا جدولنا المحوري، فلنُخصّصه لعرض ملخصات مفيدة. يمكننا التحكم في ما يظهر في الصفوف والأعمدة ومناطق البيانات في الجدول المحوري.

```csharp
// الوصول إلى مثيل PivotTable المضاف حديثًا
Aspose.Cells.Pivot.PivotTable pivotTable = pivotTables[index];

// عدم إظهار الإجماليات الكبرى للصفوف.
pivotTable.RowGrand = false;

// سحب الحقل الأول إلى منطقة الصف.
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Row, 0);

// سحب الحقل الثاني إلى منطقة العمود.
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Column, 1);

// سحب الحقل الثالث إلى منطقة البيانات.
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Data, 2);
```

في هذه الخطوة، نطلب من الجدول المحوري إخفاء الإجماليات الكلية للصفوف، ثم نحدد الحقول التي تُدرج في مناطق الصفوف والأعمدة والبيانات. ستُعرض أسماء الرياضات في الصفوف، والأرباع في الأعمدة، بينما تُعرض أرقام المبيعات في الملخصات.

## الخطوة 5: حفظ المصنف
وأخيرًا، نريد حفظ مصنف العمل الذي أنشأناه حديثًا لرؤية ثمار عملنا.

```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "pivotTable_test_out.xls");
```

ما عليك سوى توفير المسار الصحيح، وسيتم حفظ مخرجات جدول البيانات المحوري الخاص بك في ملف Excel يمكنك فتحه ومراجعته.

## خاتمة
إنشاء جداول محورية برمجيًا باستخدام Aspose.Cells لـ .NET يوفر عليك الكثير من الوقت، خاصةً عند التعامل مع مجموعات بيانات ضخمة. لقد تعلمت كيفية إعداد مشروعك، واستيراد الحزم اللازمة، وتعبئة البيانات، وإنشاء جدول محوري قابل للتخصيص من البداية. لذا، في المرة القادمة التي تغرق فيها بالأرقام، تذكر هذا البرنامج التعليمي ودع Aspose.Cells يقوم بالعمل الشاق نيابةً عنك.
