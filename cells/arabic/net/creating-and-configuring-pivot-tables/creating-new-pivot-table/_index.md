# إنشاء جدول محوري جديد برمجيًا في .NET

## مقدمة
قد يبدو إنشاء جدول محوري مهمة شاقة، خاصة عندما تقوم بذلك برمجيًا. ولكن لا تخف! باستخدام Aspose.Cells for .NET، لا يصبح إنشاء جدول محوري أمرًا سهلاً فحسب، بل إنه أيضًا قوي جدًا لتحليل البيانات. في هذا البرنامج التعليمي، سنرشدك خطوة بخطوة حول كيفية إنشاء جدول محوري جديد في تطبيق .NET. سواء كنت تضيف بيانات للمبيعات أو الرياضة أو أي مقياس تجاري آخر، سيساعدك هذا الدليل على تشغيل جداولك المحورية في وقت قصير.

## المتطلبات الأساسية
قبل أن نبدأ، دعنا نتأكد من أن كل شيء جاهز. إليك ما عليك القيام به:

1. تثبيت .NET Framework: تأكد من تثبيت .NET Framework على جهازك. يدعم Aspose.Cells إصدارات مختلفة، ولكن من الأفضل الالتزام بالإصدار الأحدث.
2.  مكتبة Aspose.Cells: يجب أن يكون لديك مكتبة Aspose.Cells. يمكنك[تحميله هنا](https://releases.aspose.com/cells/net/)أو الحصول على[رخصة مؤقتة](https://purchase.aspose.com/temporary-license/) للتقييم.
3. إعداد IDE: قم بإعداد IDE متوافق مع C#، مثل Visual Studio، حيث يمكنك بدء مشروع جديد.
4. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على المتابعة دون الوقوع في فخ كبير.

هل كل شيء جاهز؟ رائع! دعنا ننتقل إلى استيراد الحزم اللازمة.

## استيراد الحزم
أولاً، عليك استيراد المساحات المطلوبة إلى مشروع C# الخاص بك. افتح ملف C# وأضف ما يلي باستخدام التوجيهات:

```csharp
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

توفر لك هذه المساحات الاسمية إمكانية الوصول إلى وظائف المصنف وورقة العمل والجدول المحوري التي سنستخدمها في هذا البرنامج التعليمي.

## الخطوة 1: إنشاء كائن مصنف
إن إنشاء مصنف عمل هو بداية رحلتك. فلنبدأ بإنشاء مصنف عمل جديد والوصول إلى ورقة العمل الأولى.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء كائن مصنف
Workbook workbook = new Workbook();

// الحصول على مرجع ورقة العمل المضافة حديثًا
Worksheet sheet = workbook.Worksheets[0];
```

 في هذه الخطوة، نقوم بإنشاء`Workbook`مثال يمثل ملف Excel الخاص بنا ونحصل على ورقة العمل الأولى، والتي ستكون بمثابة ساحة اللعب الخاصة بنا للجدول المحوري.

## الخطوة 2: إدراج البيانات في الخلايا
بعد ذلك، دعنا نملأ ورقة العمل الخاصة بنا ببعض البيانات النموذجية. سنقوم بإدخال صفوف لرياضات مختلفة، وأرباع عملات، وأرقام مبيعات لنوفر لجدولنا المحوري شيئًا لتلخيصه.

```csharp
Cells cells = sheet.Cells;

// تعيين القيمة للخلايا
Cell cell = cells["A1"];
cell.PutValue("Sport");
cell = cells["B1"];
cell.PutValue("Quarter");
cell = cells["C1"];
cell.PutValue("Sales");

// ملء خلية البيانات = cells["A2"];
cell.PutValue("Golf");
// ... المزيد من إدخالات البيانات
```

هنا، نقوم بتحديد عناوين الأعمدة وإدراج القيم أسفل كل عنوان. ستعمل هذه البيانات كمصدر لجدولنا المحوري، لذا تأكد من تنظيمها! اتبع هذه الخطوات وستتمكن من إنشاء مجموعة بيانات شاملة.

## الخطوة 3: إضافة جدول محوري
بعد أن أصبحت بياناتنا جاهزة، حان الوقت لإنشاء جدول محوري. سنستخدم مجموعة الجداول المحورية من ورقة العمل لإضافة جدولنا المحوري الجديد.

```csharp
Aspose.Cells.Pivot.PivotTableCollection pivotTables = sheet.PivotTables;

// إضافة جدول محوري إلى ورقة العمل
int index = pivotTables.Add("=A1:C8", "E3", "PivotTable2");
```

في هذا المقطع، نضيف جدولًا محوريًا إلى ورقة العمل التي تشير إلى نطاق البيانات لدينا (في هذه الحالة، الخلايا من A1 إلى C8). نضع الجدول المحوري بدءًا من الخلية E3، ونسميه "PivotTable2". الأمر بسيط للغاية، أليس كذلك؟

## الخطوة 4: تخصيص جدول المحور
الآن بعد أن أصبح لدينا جدول البيانات المحوري، فلنقم بتخصيصه لعرض ملخصات مفيدة. يمكننا التحكم في ما يظهر في الصفوف والأعمدة ومناطق البيانات في جدول البيانات المحوري.

```csharp
// الوصول إلى مثيل جدول البيانات المحوري المضاف حديثًا
Aspose.Cells.Pivot.PivotTable pivotTable = pivotTables[index];

// عدم إظهار الإجماليات الكبرى للصفوف.
pivotTable.RowGrand = false;

// سحب الحقل الأول إلى منطقة الصف.
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Row, 0);

// سحب الحقل الثاني إلى منطقة العمود.
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Column, 1);

// سحب الحقل الثالث إلى منطقة البيانات.
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Data, 2);
```

في هذه الخطوة، نطلب من الجدول المحوري إخفاء الإجماليات الكلية للصفوف ثم تحديد الحقول التي تدخل في مناطق الصفوف والأعمدة والبيانات. ستملأ أسماء الرياضات الصفوف، وستملأ الأرباع الأعمدة، وستوفر أرقام المبيعات الملخصات.

## الخطوة 5: احفظ المصنف
وأخيرًا، نريد حفظ مصنف العمل الذي أنشأناه حديثًا لرؤية ثمار عملنا.

```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "pivotTable_test_out.xls");
```

ما عليك سوى توفير المسار الصحيح، وسيتم حفظ مخرجات جدولك المحوري في ملف Excel يمكنك فتحه ومراجعته.

## خاتمة
إن إنشاء جداول محورية برمجيًا باستخدام Aspose.Cells for .NET يمكن أن يوفر لك الكثير من الوقت، وخاصة عند التعامل مع مجموعات بيانات كبيرة. لقد تعلمت كيفية إعداد مشروعك واستيراد الحزم الضرورية وملء البيانات وإنشاء جدول محوري قابل للتخصيص من البداية. لذا، في المرة القادمة التي تغرق فيها في الأرقام، تذكر هذا البرنامج التعليمي واترك لـ Aspose.Cells القيام بالعمل الشاق نيابة عنك.
