### تحميل ملف Excel وتعديله

**ملخص**:يوضح لك هذا القسم كيفية تحميل مصنف كبير، والتكرار خلال أوراق العمل الخاصة به، وإدراج صفوف في بداية كل ورقة، وملء هذه الصفوف الجديدة بالبيانات.

#### الخطوة 1: تحديد مسارات الإدخال والإخراج

حدد المجلدات لملف المصدر والإخراج. استبدل `"YOUR_SOURCE_DIRECTORY"` و `"YOUR_OUTPUT_DIRECTORY"` مع المسارات الفعلية على جهازك:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

string inputFile = SourceDir + "/Sample.xls";
string outputFile = outputDir + "/output_out.xls";
```

#### الخطوة 2: تحميل المصنف

استخدم Aspose.Cells لتحميل ملف Excel موجود. هذه الخطوة تُهيئ `Workbook` هدف:

```csharp
try {
    Workbook workbook = new Workbook(inputFile);
    DateTime start = DateTime.Now;
    
    // متابعة التعديلات...
} catch (Exception ex) {
    // التعامل مع الاستثناءات هنا
}
```

#### الخطوة 3: إدراج الصفوف وتعبئتها

كرّر كل ورقة عمل، وأدخل 100 صف في البداية. ثم املأ هذه الصفوف ببيانات مخصصة:

```csharp
for (int i = 0; i < workbook.getWorksheets().getCount(); i++) {
    Worksheet worksheet = workbook.getWorksheets().get(i);
    Cells cells = worksheet.getCells();

    // إدراج 100 صف في الفهرس 0.
    cells.insertRows(0, 100);

    for (int r = 0; r < 100; r++) {
        cells.get(r, 0).putValue("This is testing row #: " + r.ToString());
    }
}
```

#### الخطوة 4: حفظ المصنف المعدل

بعد إجراء التعديلات، احفظ المصنف في ملف جديد:

```csharp
workbook.save(outputFile);
DateTime end = DateTime.Now;
TimeSpan time = end - start;

// اختياريا تسجيل وقت المعالجة.
```
