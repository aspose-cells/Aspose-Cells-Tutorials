## دليل التنفيذ

### الميزة 1: دمج ملفات Excel المتعددة

توضح هذه الميزة كيفية دمج عدة ملفات .xls في إخراج واحد باستخدام Aspose.Cells.

#### الخطوة 1: تحديد أدلة المصدر والإخراج

تعيين المسارات لدلائل المصدر والوجهة الخاصة بك:

```csharp
string YOUR_SOURCE_DIRECTORY = "YOUR_SOURCE_DIRECTORY";
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: تحديد الملفات المراد دمجها

إنشاء مجموعة من مسارات الملفات التي تريد دمجها:

```csharp
String[] files = new String[2];
files[0] = YOUR_SOURCE_DIRECTORY + "/sampleMergeFiles_Book1.xls";
files[1] = YOUR_SOURCE_DIRECTORY + "/sampleMergeFiles_Book2.xls";
```

#### الخطوة 3: تنفيذ الدمج

يستخدم `CellsHelper.MergeFiles` لدمج ملفات Excel في مصنف واحد:

```csharp
string cacheFile = YOUR_OUTPUT_DIRECTORY + "/cacheMergeFiles.txt";
string dest = YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls";

CellsHelper.MergeFiles(files, cacheFile, dest);
```

### الميزة 2: إعادة تسمية الأوراق في ملف Excel المدمج

بعد دمج الملفات، قد ترغب في إعادة تسمية كل ورقة لتحسين التنظيم.

#### الخطوة 1: تحميل المصنف

قم بتحميل المصنف الذي سيتم إعادة تسمية الأوراق فيه:

```csharp
Workbook workbook = new Workbook(YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls");
```

#### الخطوة 2: إعادة تسمية الأوراق بشكل تسلسلي

قم بالتكرار خلال كل ورقة عمل وتعيين اسم جديد:

```csharp
int i = 1;
foreach (Worksheet sheet in workbook.Worksheets)
{
    sheet.Name = "Sheet" + i++;
}
```

#### الخطوة 3: حفظ المصنف

أخيرًا، احفظ التغييرات للحفاظ على الأوراق التي تمت إعادة تسميتها:

```csharp
workbook.Save(YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls");
```
