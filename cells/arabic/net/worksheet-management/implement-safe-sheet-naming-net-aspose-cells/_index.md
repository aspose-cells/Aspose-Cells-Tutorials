## دليل التنفيذ

هذا القسم يرشدك خلال الاستخدام `CreateSafeSheetName` لإدارة أسماء الأوراق بشكل فعال.

### اقتطاع واستبدال الأحرف غير الصالحة
1. **ملخص:**
   - ضمان الامتثال لقواعد تسمية Excel، وإزالة الأحرف غير الصالحة وتقليص الأسماء الطويلة.
2. **قطع الأسماء الطويلة:**
تقوم الطريقة تلقائيًا بتحديد الأسماء إلى 31 حرفًا:
```csharp
string name1 = CellsHelper.CreateSafeSheetName("this is first name which is created using CellsHelper.CreateSafeSheetName and truncated to 31 characters");
```
3. **استبدال الأحرف غير الصالحة:**
يستبدل الأحرف غير الصالحة بعلامة سفلية (`_`):
```csharp
string name2 = CellsHelper.CreateSafeSheetName("<> + (adj.Private ? \" Private\" : \")", '_');
```
4. **عرض النتائج:**
التحقق من النتائج باستخدام `Console.WriteLine()`:
```csharp
Console.WriteLine(name1);  // إخراج الاسم المقطوع
Console.WriteLine(name2);  // يُخرج اسمًا مُعقّمًا مع الشرطة السفلية
Console.WriteLine("CreateSafeSheetNames executed successfully.");
```
### نصائح استكشاف الأخطاء وإصلاحها
- **التحقق من طول الاسم:** تأكد من أن الأسماء تقع ضمن حدود Excel.
- **التحقق من صحة الأحرف:** قم بمراجعة الأحرف غير الصالحة في Excel للتحقق مسبقًا من أسماء الأوراق.
