### إزالة أوراق العمل باستخدام أسماء الأوراق
إزالة أوراق مُحددة قد يكون أمرًا بالغ الأهمية لإدارة البيانات. لنرَ كيف يعمل ذلك:

#### الخطوة 1: تحميل ملف Excel
ابدأ بتحميل ملف Excel الخاص بك باستخدام `FileStream`.

```csharp
string dataDir = "your_directory_path_here";

// إنشاء FileStream لفتح ملف Excel
using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
{
    // إنشاء كائن مصنف وتحميل الملف عبر التدفق
    Workbook workbook = new Workbook(fstream);
}
```
*لماذا تستخدم `FileStream`؟* إنه يسمح لك بإدارة الملفات بكفاءة، مما يضمن تحرير الموارد بعد اكتمال العمليات.

#### الخطوة 2: إزالة ورقة العمل
الآن، دعنا نزيل ورقة العمل حسب اسمها:

```csharp
// إزالة ورقة عمل باستخدام اسم الورقة الخاصة بها
workbook.Worksheets.RemoveAt("Sheet1");
```
تعمل هذه الطريقة على استهداف الورقة المحددة وحذفها بشكل مباشر، مما يؤدي إلى تحسين مهام إدارة الملفات.

#### الخطوة 3: حفظ التغييرات
وأخيرًا، احفظ المصنف الخاص بك للحفاظ على التغييرات:

```csharp
// حفظ المصنف المحدث
using (FileStream fstream = new FileStream(dataDir + "output.out.xls", FileMode.Create))
{
    workbook.Save(fstream);
}
```
