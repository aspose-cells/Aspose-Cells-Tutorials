## دليل التنفيذ
اتبع الخطوات التالية لنسخ الأوراق داخل مصنف:
### ملخص
تعمل هذه الميزة على تكرار ورقة موجودة وإضافتها كورقة جديدة، وهي مثالية لتنسيقات البيانات أو القوالب المتكررة.
#### الخطوة 1: افتح المصنف
قم بتحميل ملف Excel الخاص بك باستخدام Aspose.Cells:
```csharp
// قم بتحديد الدليل الذي يحتوي على ملفات Excel الخاصة بك.
string dataDir = "path_to_your_directory";

// تحميل مصنف موجود.
Workbook wb = new Workbook(dataDir + "book1.xls");
```
**توضيح**: ال `Workbook` يتم تهيئة الفئة عن طريق تحميل ملف، مما يسمح بالتلاعب البرمجي بمحتوياته.
#### الخطوة 2: الوصول إلى أوراق العمل
الوصول إلى كافة الأوراق داخل المصنف الخاص بك:
```csharp
// استرجاع كافة الأوراق من المصنف.
WorksheetCollection sheets = wb.Worksheets;
```
**توضيح**: ال `WorksheetCollection` يتيح الوصول إلى الأوراق الموجودة، مما يتيح إجراء عمليات مثل الإضافة أو النسخ.
#### الخطوة 3: نسخ الورقة
قم بتكرار ورقة موجودة لإنشاء ورقة جديدة:
```csharp
// أضف نسخة من "Sheet1" كصفحة عمل جديدة.
sheets.AddCopy("Sheet1");
```
**توضيح**: `AddCopy` يكرر الورقة المحددة، مع ترك الورقة الأصلية دون تغيير.
#### الخطوة 4: حفظ التغييرات
حفظ المصنف مع التغييرات:
```csharp
// احفظ المصنف المحدث في ملف جديد.
wb.Save(dataDir + "CopyWithinWorkbook_out.xls");
```
**توضيح**:تضمن هذه الخطوة إعادة كتابة التعديلات، مع الحفاظ على كافة التعديلات.
### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسار ملف Excel صحيح لتجنب `FileNotFoundException`.
