في هذا البرنامج التعليمي، سنتعمق في كيفية إضافة أوراق عمل إلى ملف Excel جديد باستخدام Aspose.Cells لـ .NET. سنشرح كل خطوة بشكل مبسط وجذاب لتتمكن من البدء بسرعة.

## استيراد الحزم
بعد إعداد مشروعك في Visual Studio، ستحتاج إلى استيراد مساحات الأسماء المطلوبة. سيؤدي ذلك إلى إتاحة فئات وطرق Aspose.Cells في مشروعك.
```csharp
using System.IO;
using Aspose.Cells;
```
الآن، دعنا ننتقل إلى دليلنا خطوة بخطوة.
سنبدأ بإنشاء ملف إكسل جديد، وإضافة ورقة عمل، وتسميتها، وأخيرًا حفظ الملف. سيتم تفصيل كل خطوة للتوضيح.
## الخطوة 1: إعداد مسار الدليل
أولاً، حدد مسار المجلد لحفظ ملف Excel. إذا لم يكن هذا المجلد موجودًا، فسيقوم البرنامج بإنشائه.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
```
يُحدد هذا السطر موقع حفظ ملف Excel. خصّص `"Your Document Directory"` إلى المسار الذي تختاره.
## الخطوة 2: التحقق من الدليل وإنشائه
في هذه الخطوة، سوف تقوم بالتحقق من وجود الدليل وإنشائه إذا لم يكن موجودًا.
```csharp
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);
```
فيما يلي تفصيل سريع:
- Directory.Exists(dataDir): يتحقق ما إذا كان الدليل المحدد موجودًا بالفعل.
- Directory.CreateDirectory(dataDir): إذا لم يكن موجودًا، فسوف يقوم هذا السطر بإنشائه.
## الخطوة 3: تهيئة مصنف جديد
الآن، نقوم بإنشاء كائن مصنف جديد، وهو في الأساس ملف Excel. 
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
ال `Workbook` الفئة أساسية في Aspose.Cells، فهي تُمثل ملف Excel بأكمله. بتهيئتها، نُهيئ ملفًا جديدًا للعمل عليه.
## الخطوة 4: إضافة ورقة عمل جديدة
بعد ذلك نضيف ورقة عمل جديدة إلى المصنف. 
```csharp
// إضافة ورقة عمل جديدة إلى كائن المصنف
int index = workbook.Worksheets.Add();
```
يؤدي هذا السطر من التعليمات البرمجية ما يلي:
- workbook.Worksheets.Add(): يضيف ورقة عمل جديدة إلى المصنف.
- int index: يخزن فهرس ورقة العمل المضافة حديثًا.
ال `Add()` تضيف الطريقة ورقة عمل فارغة، وهو أمر ضروري إذا كنت تريد أوراق عمل متعددة في ملف Excel واحد.
## الخطوة 5: الوصول إلى ورقة العمل المضافة حديثًا
الآن، دعنا نحصل على مرجع إلى ورقة العمل المضافة حديثًا باستخدام الفهرس الخاص بها.
```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[index];
```
في هذه الخطوة:
- workbook.Worksheets[index]: استرداد ورقة العمل باستخدام الفهرس الخاص بها.
- ورقة العمل ورقة العمل: متغير لتخزين المرجع إلى ورقة العمل الجديدة هذه.
باستخدام هذا المرجع، يمكنك الآن تخصيص ورقة العمل بطرق مختلفة.
## الخطوة 6: إعادة تسمية ورقة العمل
تسمية ورقة العمل باسم وصفي يُسهّل التعرف عليها. لنُغيّر اسمها إلى "ورقة العمل الخاصة بي".
```csharp
// تعيين اسم ورقة العمل المضافة حديثًا
worksheet.Name = "My Worksheet";
```
هنا:
- worksheet.Name: يحدد اسم ورقة العمل. 
بدلاً من اسم افتراضي مثل "Sheet1" أو "Sheet2"، يمكنك تعيين اسم مخصص، مما يجعل ملفك أكثر تنظيمًا.
## الخطوة 7: حفظ المصنف كملف Excel
وأخيرًا، قم بحفظ المصنف كملف Excel في الدليل المحدد.
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "output.xls");
```
في هذه الخطوة الأخيرة:
- dataDir + "output.xls": يجمع مسار الدليل الخاص بك مع اسم الملف، مما يؤدي إلى إنشاء مسار الملف الكامل.
- workbook.Save(): يحفظ المصنف في هذا المسار.
يؤدي هذا إلى حفظ ملف Excel مع كل التغييرات التي أجريتها - إضافة ورقة عمل، وتسميتها، وإعداد الدليل.
