## استيراد الحزم
بعد إعداد مشروعك في Visual Studio، ستحتاج إلى استيراد مساحات الأسماء المطلوبة. سيؤدي هذا إلى جعل فئات وطرق Aspose.Cells متاحة في مشروعك.
```csharp
using System.IO;
using Aspose.Cells;
```
الآن، دعونا ننتقل إلى دليلنا خطوة بخطوة.
سنبدأ بإنشاء ملف Excel جديد، وإضافة ورقة عمل، وتسميتها، وأخيرًا حفظ الملف. سيتم تقسيم كل خطوة إلى خطوات أكثر وضوحًا.
## الخطوة 1: إعداد مسار الدليل
أولاً، عليك تحديد مسار الدليل لحفظ ملف Excel. إذا لم يكن الدليل موجودًا، فسيقوم البرنامج بإنشائه.
```csharp
// المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
```
 يحدد هذا السطر الموقع الذي سيتم حفظ ملف Excel فيه. قم بتخصيص`"Your Document Directory"` إلى المسار الذي تختاره.
## الخطوة 2: التحقق من الدليل وإنشائه
في هذه الخطوة، سوف تقوم بالتحقق من وجود الدليل وتقوم بإنشائه إذا لم يكن موجودًا.
```csharp
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);
```
فيما يلي تفصيل سريع:
- Directory.Exists(dataDir): يتحقق ما إذا كان الدليل المحدد موجودًا بالفعل.
- Directory.CreateDirectory(dataDir): إذا لم يكن موجودًا، فسوف يقوم هذا السطر بإنشائه.
## الخطوة 3: تهيئة مصنف جديد
الآن، نقوم بإنشاء كائن مصنف جديد، والذي هو في الأساس ملف Excel. 
```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
```
 ال`Workbook` تُعد الفئة عنصرًا أساسيًا في Aspose.Cells، فهي تمثل ملف Excel بأكمله. ومن خلال تهيئتها، نقوم بإعداد ملف جديد للعمل معه.
## الخطوة 4: إضافة ورقة عمل جديدة
بعد ذلك نضيف ورقة عمل جديدة إلى المصنف. 
```csharp
// إضافة ورقة عمل جديدة إلى كائن المصنف
int index = workbook.Worksheets.Add();
```
يؤدي هذا السطر من التعليمات البرمجية ما يلي:
- workbook.Worksheets.Add(): يضيف ورقة عمل جديدة إلى المصنف.
- int index: يخزن فهرس ورقة العمل المضافة حديثًا.
 ال`Add()` تضيف الطريقة ورقة عمل فارغة، وهو أمر ضروري إذا كنت تريد أوراق عمل متعددة في ملف Excel واحد.
## الخطوة 5: الوصول إلى ورقة العمل المضافة حديثًا
الآن، دعونا نحصل على مرجع إلى ورقة العمل المضافة حديثًا باستخدام فهرسها.
```csharp
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاصة بها
Worksheet worksheet = workbook.Worksheets[index];
```
في هذه الخطوة:
- دفتر عمل.أوراق عمل[[الفهرس]: يسترجع ورقة العمل باستخدام الفهرس الخاص بها.
- ورقة عمل ورقة العمل: متغير لتخزين المرجع إلى ورقة العمل الجديدة هذه.
باستخدام هذا المرجع، يمكنك الآن تخصيص ورقة العمل بطرق مختلفة.
## الخطوة 6: إعادة تسمية ورقة العمل
إن إعطاء ورقة العمل اسمًا وصفيًا قد يسهل التعرف عليها. دعنا نغير اسمها إلى "ورقة العمل الخاصة بي".
```csharp
// تعيين اسم ورقة العمل المضافة حديثًا
worksheet.Name = "My Worksheet";
```
هنا:
- worksheet.Name: يحدد اسم ورقة العمل. 
بدلاً من اسم افتراضي مثل "Sheet1"، "Sheet2"، يمكنك تعيين اسم مخصص، مما يجعل ملفك أكثر تنظيمًا.
## الخطوة 7: احفظ المصنف كملف Excel
وأخيرًا، قم بحفظ المصنف كملف Excel في الدليل المحدد.
```csharp
// حفظ ملف Excel
workbook.Save(dataDir + "output.xls");
```
في هذه الخطوة الأخيرة:
- dataDir + "output.xls": يجمع مسار الدليل الخاص بك مع اسم الملف، مما يؤدي إلى إنشاء مسار الملف الكامل.
- workbook.Save(): يحفظ المصنف في هذا المسار.
يؤدي هذا إلى حفظ ملف Excel مع كل التغييرات التي أجريتها - إضافة ورقة عمل، وتسميتها، وإعداد الدليل.
