## دليل التنفيذ

في هذا القسم، سنستعرض الخطوات اللازمة لحذف صف من ورقة عمل Excel باستخدام Aspose.Cells.

### ملخص

يُعد حذف الصفوف أمرًا ضروريًا لتنظيف البيانات أو تعديل جدول البيانات ديناميكيًا. تساعد هذه الميزة في الحفاظ على جداول بيانات منظمة وفعالة برمجيًا.

#### الخطوة 1: تحميل المصنف الخاص بك

أولاً، قم بتحميل المصنف الذي يحتوي على الورقة التي تريد حذف صف منها:

```csharp
using System.IO;
using Aspose.Cells;

namespace AsposeExample
{
    public class DeleteRowExample
    {
        public void Run()
        {
            // تحديد مسار الملف
            string dataDir = "path/to/your/directory/";
            
            // افتح المصنف باستخدام FileStream
            using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
            {
                Workbook workbook = new Workbook(fstream);

                // انتقل إلى حذف الصف
            }
        }
    }
}
```

#### الخطوة 2: الوصول إلى ورقة العمل

قم بالوصول إلى ورقة العمل المحددة التي تريد إجراء الحذف فيها:

```csharp
// الوصول إلى ورقة العمل الأولى في المصنف
Worksheet worksheet = workbook.Worksheets[0];
```

#### الخطوة 3: حذف صف

الآن، احذف الصف المطلوب. في هذا المثال، سنحذف الصف الثالث (الفهرس) `2`):

```csharp
// حذف الصف الثالث من ورقة العمل
worksheet.Cells.DeleteRow(2);
```

#### الخطوة 4: حفظ التغييرات

وأخيرًا، احفظ المصنف الخاص بك للحفاظ على التغييرات:

```csharp

// حفظ ملف Excel المعدل

```
