## دليل التنفيذ
دعونا نلقي نظرة على عملية إدراج صف في ورقة عمل Excel.
### الخطوة 1: افتح ملف Excel (H3)
أولاً، عليك فتح ملف Excel باستخدام `FileStream`تتضمن هذه الخطوة قراءة مستند Excel الحالي لديك:
```csharp
using System.IO;

// المسار إلى دليل المستندات.
string dataDir = "your_data_directory_path/";

// إنشاء مجرى ملف يحتوي على ملف Excel الذي سيتم فتحه
FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open);

// فتح ملف Excel من خلال تدفق الملف
Workbook workbook = new Workbook(fstream);
```
### الخطوة 2: الوصول إلى ورقة العمل (H3)
بعد ذلك، انتقل إلى ورقة العمل المحددة التي تريد تعديلها. هذا المثال يُظهر ورقة العمل الأولى:
```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];
```
### الخطوة 3: إدراج صف في ورقة العمل (H3)
الآن، أدخل صفًا في الموضع المطلوب. الكود التالي يُدرج صفًا في الموضع الثالث (الفهرس ٢):
```csharp
// إدراج صف في ورقة العمل في الموضع الثالث
worksheet.Cells.InsertRow(2);
```
### الخطوة 4: حفظ وإغلاق مجرى الملف (H3)
أخيرًا، احفظ تعديلاتك وأغلق مجرى الملف للموارد المجانية:
```csharp
// حفظ ملف Excel المعدل
workbook.Save(dataDir + "output.out.xls");

// إغلاق مجرى الملف
fstream.Close();
```
