### نظرة عامة على اكتشاف أوراق العمل الفارغة

يساعد اكتشاف أوراق العمل الفارغة على إدارة مجموعات البيانات الكبيرة وتبسيطها. تُعد هذه الوظيفة أساسية لمهام مثل تنظيف البيانات وإنشاء التقارير.

#### الخطوة 1: تحميل المصنف الخاص بك
أولاً، قم بإنشاء مثيل لـ `Workbook` الفئة لتحميل ملف جدول البيانات الخاص بك:

```csharp
// تحميل المصنف الموجود
string sourceDir = RunExamples.Get_SourceDirectory();
var book = new Workbook(sourceDir + "sampleDetectEmptyWorksheets.xlsx");
```

#### الخطوة 2: التكرار خلال أوراق العمل

قم بالتنقل عبر كل ورقة عمل في المصنف والتحقق من المحتوى.

##### التحقق من الخلايا المأهولة
إذا تم ملء أي خلايا، فلن تكون الورقة فارغة:

```csharp
for (int i = 0; i < book.Worksheets.Count; i++)
{
    Worksheet sheet = book.Worksheets[i];
    
    if (sheet.Cells.MaxDataRow != -1)
    {
        Console.WriteLine(sheet.Name + " is not Empty because one or more Cells are Populated");
    }
}
```

##### التحقق من الأشكال
قد تحتوي الأوراق على أشكال، مما يجعلها غير فارغة:

```csharp
else if (sheet.Shapes.Count > 0)
{
    Console.WriteLine(sheet.Name + " is not Empty because there are one or more Shapes");
}
```

##### التحقق من الخلايا المهيئة

بالنسبة للأوراق الفارغة تمامًا، تحقق من الخلايا المهيئة:

```csharp
else
{
    Aspose.Cells.Range range = sheet.Cells.MaxDisplayRange;
    var rangeIterator = range.GetEnumerator();
    
    if (rangeIterator.MoveNext())
    {
        Console.WriteLine(sheet.Name + " is not Empty because one or more cells are Initialized");
```
