## دليل التنفيذ

الآن بعد أن قمنا بتغطية الإعداد، دعنا ننتقل إلى تنفيذ الحل.

### تحميل أوراق محددة

الهدف هو تحميل أوراق محددة فقط من ملف Excel مع تجاهل الأوراق الأخرى. إليك كيفية تحقيق ذلك:

#### الخطوة 1: تحديد خيارات التحميل

أولاً، قم بإنشاء `LoadOptions` كائن يحدد تنسيق المصنف الخاص بك ويعين مرشح تحميل مخصص.

```csharp
LoadOptions loadOptions = new LoadOptions(LoadFormat.Xlsx);
loadOptions.LoadFilter = new CustomLoad();
```

**توضيح**: ال `LoadOptions` توفر الفئة إعدادات لتحميل ملفات Excel. من خلال ضبط `LoadFilter`، يمكنك التحكم في الأوراق التي سيتم تحميلها استنادًا إلى معاييرك.

#### الخطوة 2: إنشاء مرشح تحميل مخصص

قم بتعريف مرشح مخصص عن طريق الوراثة من `LoadFilter`سيحدد هذا كيفية معالجة كل ورقة.

```csharp
class CustomLoad : LoadFilter
{
    public override void StartSheet(Worksheet sheet)
    {
        if (sheet.Name == "Sheet2")
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.All;
        }
        else
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.Structure;
        }
    }
}
```

**توضيح**: ال `StartSheet` تم تجاوز الطريقة لتحديد أنه يجب تحميل "Sheet2" فقط بكل البيانات، بينما يتم تجاهل الأوراق الأخرى خارج بنيتها.

#### الخطوة 3: تحميل المصنف

استخدم خيارات التحميل المحددة لإنشاء مثيل مصنف وتحميل الورقة المطلوبة.

```csharp
Workbook workbook = new Workbook(sourceDir + "sampleLoadSpecificSheets.xlsx", loadOptions);
```

**توضيح**: ال `Workbook` يقبل المنشئ كل من مسار الملف وخيارات التحميل، مما يسمح لك بتحديد الأوراق التي يجب تحميلها استنادًا إلى منطق الفلتر المخصص.

#### الخطوة 4: حفظ النتيجة

بعد المعالجة، احفظ المصنف الخاص بك مع التعديلات إذا لزم الأمر:

```csharp
workbook.Save(outputDir + "outputLoadSpecificSheets.xlsx");
```
