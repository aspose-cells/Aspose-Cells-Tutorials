# حماية صف محدد في ورقة عمل Excel

## مقدمة

في عالمنا المتسارع، أصبحت إدارة جداول البيانات بفعالية أكثر أهمية من أي وقت مضى. يُعدّ مايكروسوفت إكسل أداةً لا غنى عنها في العديد من القطاعات والمهن. ومع ذلك، مع مشاركة هذه المستندات، وخاصةً في البيئات التعاونية، تُصبح حماية معلومات مُحددة داخل جداول البيانات أمرًا بالغ الأهمية. إذًا، كيف يُمكنك إغلاق صف في إكسل لمنع التعديلات غير المرغوب فيها؟ حسنًا، إذا كنت تعمل باستخدام .NET، فأنت محظوظ! تُعدّ Aspose.Cells مكتبةً ممتازةً للتعامل مع ملفات إكسل برمجيًا، مما يُتيح لنا حماية صفوف مُحددة بكفاءة.

## المتطلبات الأساسية

قبل أن نبدأ، هناك بعض الأشياء التي ستحتاجها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. يمكنك استخدام أي إصدار يدعم تطوير .NET.
2. Aspose.Cells لـ .NET: ستحتاج إلى تثبيت مكتبة Aspose.Cells. تفضل بزيارة [هذا الرابط للتحميل](https://releases.aspose.com/cells/net/) الإصدار الأحدث.
3. المعرفة الأساسية بـ .NET: ستكون المعرفة بلغة C# ومفاهيم البرمجة الأساسية مفيدة لأننا سنعمل مع أجزاء صغيرة من التعليمات البرمجية.

بمجرد أن يكون كل شيء في مكانه، دعنا ننتقل إلى العمل!

## استيراد الحزم

قبل كتابة الكود، يجب علينا استيراد مساحات أسماء Aspose.Cells اللازمة. هذا يُهيئ تطبيقنا لاستخدام الفئات والأساليب التي توفرها مكتبة Aspose.Cells. إليك ما عليك فعله:

### إعداد مشروعك

1. إنشاء مشروع جديد:
   - افتح Visual Studio وأنشئ مشروع تطبيق وحدة تحكم جديد. سيستضيف هذا المشروع شيفرة معالجة Excel الخاصة بنا.

2. إضافة مرجع Aspose.Cells:
   - انقر بزر الماوس الأيمن على المشروع في مستكشف الحلول، ثم انتقل إلى "إدارة حزم NuGet"، وابحث عن "Aspose.Cells". انقر لتثبيته.

3. قم بتضمين مساحات الأسماء الضرورية في الكود الخاص بك:
```csharp
using System.IO;
using Aspose.Cells;
```

بعد أن انتهينا من إعداد كل شيء، لنبدأ بحماية صف معين في ورقة عمل Excel خطوة بخطوة. المثال الذي سنستخدمه يقفل الصف الأول، ولكن يمكنك تعديله لأي صف تريده.

## الخطوة 1: تحديد دليل المستندات

أولاً، علينا تحديد مجلد لتخزين ملف Excel. إليك الطريقة:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY"; // التغيير إلى المسار المطلوب.

// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي تريد حفظ ملف Excel الجديد فيه.

## الخطوة 2: إنشاء مصنف جديد

بعد ذلك، سننشئ مصنفًا جديدًا باستخدام Aspose.Cells. هذه هي لوحتك الفارغة لإنشاء جدول بيانات.

```csharp
// إنشاء مصنف جديد.
Workbook wb = new Workbook();
```

## الخطوة 3: إنشاء ورقة عمل والوصول إليها

الآن، دعونا ندخل إلى ورقة العمل الأولى في مصنفنا لإجراء التغييرات اللازمة.

```csharp
// إنشاء كائن ورقة عمل والحصول على الورقة الأولى.
Worksheet sheet = wb.Worksheets[0];
```

## الخطوة 4: فتح جميع الأعمدة

قبل قفل أي صف، يجب التأكد من فتح جميع الأعمدة. هذا يتيح لنا المرونة لحماية الصف المطلوب فقط.

```csharp
// تعريف كائن النمط.
Style style;
// تعريف كائن styleflag.
StyleFlag flag;
// قم بالمرور على جميع الأعمدة في ورقة العمل وإلغاء قفلها.
for (int i = 0; i <= 255; i++)
{
    style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false; // فتح العمود
    flag = new StyleFlag();
    flag.Locked = true; // تعيين العلم إلى صحيح للقفل
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag); // تطبيق النمط
}
```

## الخطوة 5: قفل الصف المطلوب

الآن، حان وقت قفل الصف الذي تريد حمايته. في هذه الحالة، نقوم بقفل الصف الأول.

```csharp
// احصل على نمط الصف الأول.
style = sheet.Cells.Rows[0].Style;
// قفله.
style.IsLocked = true;
// إنشاء العلم.
flag = new StyleFlag();
// ضبط إعداد القفل.
flag.Locked = true;
// قم بتطبيق النمط على الصف الأول.
sheet.Cells.ApplyRowStyle(0, style, flag);
```

## الخطوة 6: حماية ورقة العمل

بعد قفل الصف المطلوب، نحتاج إلى تفعيل الحماية في ورقة العمل. هنا تبدأ العملية!

```csharp
// حماية الورقة.
sheet.Protect(ProtectionType.All);
```

## الخطوة 7: حفظ المصنف

أخيرًا، حان وقت حفظ ملف Excel الجديد. يمكنك اختيار التنسيق الذي تريده.

```csharp
// احفظ ملف الاكسل.
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```

## الأسئلة الشائعة

### هل يمكنني حماية صفوف متعددة باستخدام نفس الطريقة؟  
نعم، يمكنك تكرار عملية القفل لأي صفوف أخرى بنفس الطريقة التي قمت بها للصف الأول.

### ماذا لو أردت حماية وفتح خلايا محددة بدلاً من الصفوف؟  
بإمكانك تحديد خلايا بشكل فردي وتطبيق أنماط القفل عليها، على غرار الطريقة التي قمت بها بقفل صف.

### هل استخدام Aspose.Cells مجاني؟  
Aspose.Cells هو منتج تجاري، ولكن يمكنك تجربته من خلال إصدار تجريبي مجاني متاح [هنا](https://releases.aspose.com/).

### هل أحتاج إلى اتصال بالإنترنت لاستخدام Aspose.Cells؟  
لا، Aspose.Cells عبارة عن مكتبة .NET ويمكنها العمل دون اتصال بالإنترنت بمجرد تثبيتها.

### أين يمكنني الحصول على الدعم لـ Aspose.Cells؟  
لأي استفسارات أو دعم يمكنك زيارة [منتدى دعم Aspose](https://forum.aspose.com/c/cells/9).
