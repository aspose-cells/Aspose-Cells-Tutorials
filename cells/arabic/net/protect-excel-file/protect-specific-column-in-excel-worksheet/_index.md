# حماية عمود محدد في ورقة عمل Excel

## مقدمة

في عالمٍ تزداد فيه إدارة البيانات تعقيدًا، فإن معرفة كيفية حماية أقسام محددة من مستنداتك يمكن أن تحمي معلوماتك المهمة من التغييرات غير المرغوبة. سواءً كنت طالبًا تُدير درجاتك، أو مدير مشروع يُتابع الميزانيات، أو مُحللًا يتعامل مع بيانات حساسة، فمن الضروري الحفاظ على أمان المعلومات المهمة مع السماح للآخرين باستخدام جدول البيانات. سيوضح هذا الدليل كيفية حماية أعمدة مُحددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET.

## المتطلبات الأساسية 

قبل الغوص في الكود، هناك بعض المتطلبات الأساسية التي تحتاج إلى الاهتمام بها:

1. Visual Studio: تأكد من تثبيت Microsoft Visual Studio (يفضل إصدار 2017 أو أحدث). سيُشكّل بيئة التطوير الخاصة بك. 
2. مكتبة Aspose.Cells: يجب أن تكون مكتبة Aspose.Cells مُنزّلة ومُشار إليها في مشروعك. يمكنك [تحميل المكتبة هنا](https://releases.aspose.com/cells/net/) إذا لم تكن قد فعلت ذلك بالفعل.
3. الفهم الأساسي للغة C#: على الرغم من أن أمثلة التعليمات البرمجية واضحة، فإن الحصول على معرفة أساسية بلغة C# سيساعدك على إجراء التعديلات حسب الضرورة.
4. .NET Framework: تأكد من أن مشروعك يستهدف .NET Framework حيث يتم دعم Aspose.Cells.

الآن، دعونا ننتقل إلى الجزء الممتع - البرمجة!

## استيراد الحزم

للبدء، عليك استيراد مساحات الأسماء اللازمة المتعلقة بـ Aspose.Cells. في أعلى ملف C#، أضف السطر التالي:

```csharp
using System.IO;
using Aspose.Cells;
```

تعتبر هذه المكتبة قوية وتسمح لك بإجراء عدد لا يحصى من العمليات، بما في ذلك حماية بياناتك داخل ملفات Excel، وهو ما نهدف إلى تحقيقه اليوم.

لنُقسّم هذا إلى عدة خطوات واضحة وموجزة. ستحمي أعمدة محددة، مما يسمح ببقاء بقية ورقة العمل قابلة للتعديل.

## الخطوة 1: إعداد دليل البيانات

أولاً، عليك تحديد مسار المجلد الذي ستحفظ فيه ملف Excel. يتضمن ذلك إنشاء مجلد إذا لم يكن موجودًا بالفعل. إليك كيفية القيام بذلك:

```csharp
// قم بتحديد المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// قم بإنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

إن مقتطف التعليمات البرمجية ينشئ دليلاً في المسار المحدد إذا لم يكن موجودًا بالفعل، مما يضمن لك وجود موقع آمن لملف الإخراج الخاص بك.

## الخطوة 2: إنشاء مصنف جديد

بعد ذلك، نحتاج إلى إنشاء مصنف جديد. يتيح لك Aspose.Cells إنشاء ملفات Excel ومعالجتها بسهولة. إليك كيفية القيام بذلك:

```csharp
// إنشاء مصنف جديد.
Workbook wb = new Workbook();
```

من خلال إنشاء مثيل جديد `Workbook` الهدف هو أن تبدأ بصفحة فارغة، جاهزة لتخصيص جدول البيانات الخاص بك.

## الخطوة 3: الوصول إلى ورقة العمل الأولى

بعد إنشاء المصنف، ستحتاج إلى الوصول إلى ورقة العمل الأولى التي ستقوم فيها بإجراء العمليات الخاصة بك:

```csharp
// إنشاء كائن ورقة عمل والحصول على الورقة الأولى.
Worksheet sheet = wb.Worksheets[0];
```

ال `Worksheet` يسمح لك الكائن بالتعامل مع ورقة عمل محددة في المصنف. في هذه الحالة، نستخدم الورقة الأولى.

## الخطوة 4: فتح جميع الأعمدة

لتعيين أعمدة محددة كمحمية، يجب عليك أولاً إلغاء قفل جميع الأعمدة في ورقة العمل. هذه الخطوة تُهيئها للتعديلات:

```csharp
// تعريف كائن النمط.
Style style;
// تعريف كائن علم النمط.
StyleFlag flag;
// قم بالمرور على جميع الأعمدة في ورقة العمل وإلغاء قفلها.
for (int i = 0; i <= 255; i++)
{
    style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false;
    flag = new StyleFlag();
    flag.Locked = true;
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
}
```

يتكرر هذا الكود في أول ٢٥٦ عمودًا. يفتح كل عمود بتعديل إعدادات النمط. `StyleFlag` يضمن إمكانية تطبيق الخاصية المقفلة لاحقًا.

## الخطوة 5: قفل العمود المطلوب

الآن، ستحتاج إلى قفل العمود الأول تحديدًا، مع ترك جميع الأعمدة الأخرى قابلة للتعديل. إليك كيفية القيام بذلك:

```csharp
// احصل على نمط العمود الأول.
style = sheet.Cells.Columns[0].Style;
// قفله.
style.IsLocked = true;
// إنشاء العلم.
flag = new StyleFlag();
// ضبط إعداد القفل.
flag.Locked = true;
// قم بتطبيق النمط على العمود الأول.
sheet.Cells.Columns[0].ApplyStyle(style, flag);
```

هنا، يقوم الكود بجلب نمط العمود الأول، ثم يُقفله، ثم يُطبّقه. النتيجة هي أن المستخدمين يستطيعون تعديل بقية الصفحة، لكنهم لن يتمكنوا من تعديل العمود الأول.

## الخطوة 6: حماية ورقة العمل

الخطوة التالية هي تفعيل الحماية لورقة العمل بأكملها. هنا ستُفعّل أقفال الأعمدة:

```csharp
// حماية الورقة.
sheet.Protect(ProtectionType.All);
```

ال `Protect` تضمن الطريقة تأمين جميع العناصر القابلة للتنفيذ في الورقة، باستثناء المناطق التي سمحت بها على وجه التحديد (مثل الأعمدة غير المؤمنة).

## الخطوة 7: حفظ المصنف

بمجرد تكوين كل شيء وتجهيزه، حان الوقت لحفظ المصنف الخاص بك، مع التأكد من تسجيل جميع التغييرات:

```csharp
// احفظ ملف الاكسل.
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```

يحفظ هذا الكود مصنفك بتنسيق Excel 97-2003 في المسار المحدد. تأكد من استبدال `dataDir` مع مسار الدليل الفعلي الخاص بك.

## خاتمة

باتباع الخطوات الموضحة أعلاه، نجحت في حماية أعمدة محددة في ورقة عمل Excel مع إبقاء أجزاء أخرى قابلة للتعديل. يتيح استخدام Aspose.Cells لـ .NET إمكانيات واسعة للتعامل مع ملفات Excel. تُعد هذه القدرة على حماية المعلومات الحساسة أمرًا بالغ الأهمية في بيئات العمل المشتركة.
