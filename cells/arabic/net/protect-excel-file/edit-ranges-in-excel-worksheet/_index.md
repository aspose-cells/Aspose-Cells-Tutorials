# تحرير النطاقات في ورقة عمل Excel

## مقدمة

عند تحرير جداول بيانات Excel، تُعد إمكانية حماية مناطق معينة مع السماح بالتعديل في مناطق أخرى من أهم الميزات المفيدة. يُعد هذا مفيدًا للغاية في البيئات التعاونية حيث يحتاج عدة مستخدمين إلى الوصول، ولكن عليهم تعديل خلايا محددة فقط. سنتناول اليوم كيفية استخدام Aspose.Cells لـ .NET لإدارة النطاقات القابلة للتحرير ضمن ورقة عمل Excel. هيا، استمتع بمشروبك المفضل في البرمجة، ولنبدأ!

## المتطلبات الأساسية

قبل أن نبدأ بالبرمجة، لنتأكد من جاهزيتك. إليك ما تحتاجه:

1. Visual Studio: تأكد من تثبيت Visual Studio. إصدار المجتمع يعمل بكفاءة.
2. مكتبة Aspose.Cells: أنت بحاجة إلى مكتبة Aspose.Cells لـ .NET. يمكنك [قم بتحميله هنا](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: إن الفهم الأساسي للغة C# سوف يساعدك كثيرًا.
4. إعداد المشروع: إنشاء تطبيق وحدة تحكم C# جديد في Visual Studio.

تمام، أنت جاهز! الآن، لنتعمق في جوهر البرمجة.

## استيراد الحزم

بعد إعداد مشروعك، تبدأ الخطوة الأولى باستيراد مساحة اسم Aspose.Cells اللازمة. للقيام بذلك، أضف السطر التالي في أعلى ملف الكود:

```csharp
using Aspose.Cells;
```

سيسمح لك هذا بالوصول إلى جميع الوظائف التي يوفرها Aspose.Cells في مشروعك.

## الخطوة 1: إعداد الدليل

قبل البدء بالعمل مع ملفات Excel، يُنصح بإنشاء مجلد لحفظ ملفاتك. تضمن هذه الخطوة أن يعرف تطبيقك مكان قراءة البيانات وكتابتها.

دعونا نضع الكود لإنشاء دليل (إذا لم يكن موجودًا بالفعل):

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

يستبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الذي تريد تخزين ملفاتك فيه. قد يكون هذا شيئًا مثل `@"C:\ExcelFiles\"`.

## الخطوة 2: إنشاء مصنف جديد

الآن وقد أصبح دليلك جاهزًا، لننشئ مصنف Excel جديدًا. هذا أشبه بفتح لوحة رسم فارغة قبل البدء بالرسم.

```csharp
// إنشاء مصنف جديد
Workbook book = new Workbook();
```

وبهذا، أصبح لديك مصنف العمل الفارغ جاهزًا للانطلاق!

## الخطوة 3: احصل على ورقة العمل الأولى

يحتوي كل مصنف افتراضيًا على ورقة عمل واحدة على الأقل. يجب عليك جلب هذه الورقة لإجراء عمليات عليها.

```csharp
// احصل على ورقة العمل الأولى (الافتراضية)
Worksheet sheet = book.Worksheets[0];
```

وهنا نصل إلى ورقة العمل الأولى، والتي تشبه فتح ورقة جديدة في دفتر ملاحظاتك.

## الخطوة 4: الحصول على نطاقات السماح بالتحرير

قبل أن نتمكن من إعداد النطاقات القابلة للتحرير، نحتاج إلى استرداد مجموعة النطاقات المحمية من ورقة العمل الخاصة بنا.

```csharp
// احصل على نطاقات التحرير المسموح بها
ProtectedRangeCollection allowRanges = sheet.AllowEditRanges;
```

هذا السطر يُظهر لك المجموعة التي ستُدير بها نطاقاتك المحمية. من الجيد معرفة ما هو متاح تحت الغطاء!

## الخطوة 5: تحديد وإنشاء نطاق محمي

في هذه المرحلة، نحن جاهزون لتحديد النطاق الذي تريد السماح بالتعديلات فيه. لنبدأ في إنشاء هذا النطاق.

```csharp
// تعريف النطاق المحمي
ProtectedRange proteced_range;

// إنشاء النطاق
int idx = allowRanges.Add("r2", 1, 1, 3, 3);
proteced_range = allowRanges[idx];
```

في الكود أعلاه، نُنشئ نطاقًا محميًا باسم "r2"، يسمح بالتعديل في الخلايا من الصف 1 (العمود 1) إلى الصف 3 (العمود 3) (وهذا يُترجم في لغة إكسل إلى كتلة من A1 إلى C3). يمكنك تعديل هذه المؤشرات حسب الحاجة.

## الخطوة 6: تعيين كلمة مرور 

تعيين كلمة مرور للنطاق المحمي يضمن أن يتمكن فقط من يملك كلمة المرور من تعديل المنطقة المحددة. هذه الخطوة تعزز أمان جدول البيانات الخاص بك.

```csharp
// حدد كلمة المرور
proteced_range.Password = "YOUR_PASSWORD";
```

يستبدل `"YOUR_PASSWORD"` بكلمة مرور من اختيارك. تذكر فقط، لا تجعل الأمر بسيطًا جدًا - فكّر فيه كقفل صندوق كنزك!

## الخطوة 7: حماية الورقة

الآن بعد أن قمنا بتحديد نطاقنا القابل للتحرير وتأمينه بكلمة مرور، فقد حان الوقت لحماية ورقة العمل بأكملها.

```csharp
// حماية الورقة
sheet.Protect(ProtectionType.All);
```

باستدعاء هذه الطريقة، تُغلق ورقة العمل بأكملها. لا يمكن تعديل سوى النطاقات المُحددة للتحرير.

## الخطوة 8: حفظ ملف Excel

لقد وصلنا أخيرًا إلى الخطوة الأخيرة في برنامجنا التعليمي - حفظ المصنف في الدليل المحدد!

```csharp
// حفظ ملف Excel
book.Save(dataDir + "protectedrange.out.xls");
```

سيؤدي هذا إلى حفظ المصنف المحمي الخاص بك باسم `protectedrange.out.xls` في الدليل المحدد الخاص بك.

## خاتمة

وها قد انتهيت! لقد أنشأتَ بنجاح ورقة عمل Excel باستخدام Aspose.Cells لـ .NET، وحددتَ نطاقات قابلة للتعديل، وعيّنتَ كلمة مرور، وحميتَ ورقة العمل - كل ذلك بخطوات بسيطة. الآن، يمكنك مشاركة مصنفك مع زملائك، مما يُعزز التعاون مع الحفاظ على أمان بياناتك الأساسية.
