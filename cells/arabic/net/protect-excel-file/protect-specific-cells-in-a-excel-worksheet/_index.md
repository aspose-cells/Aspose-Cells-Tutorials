# حماية خلايا محددة في ورقة عمل Excel

## مقدمة

إنشاء جداول بيانات Excel وإدارة حماية الخلايا قد يبدو مهمة شاقة، أليس كذلك؟ خاصةً عند محاولة ضمان إمكانية تعديل خلايا معينة فقط مع الحفاظ على أمان الخلايا الأخرى. مع Aspose.Cells لـ .NET، يمكنك بسهولة حماية خلايا محددة داخل جدول بيانات Excel ببضعة أسطر من التعليمات البرمجية!

في هذه المقالة، سنشرح لك خطوة بخطوة كيفية تطبيق حماية الخلايا باستخدام Aspose.Cells لـ .NET. بنهاية هذا الدليل، ستكون قد اكتسبت المعرفة اللازمة لحماية بيانات Excel بكفاءة.

## المتطلبات الأساسية

قبل التعمق في الكود، هناك بعض المتطلبات الأساسية التي يجب أن تكون موجودة:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك لأننا سنقوم بالبرمجة باستخدام C#.
2. Aspose.Cells لـ .NET: يجب تثبيت Aspose.Cells لـ .NET. إذا لم تقم بذلك بعد، فقم بتنزيله من [هنا](https://releases.aspose.com/cells/net/).
3. الفهم الأساسي لـ C#: ستساعدك المعرفة ببرمجة C# على فهم الأمثلة المقدمة بسهولة أكبر.

## استيراد الحزم

بعد أن تكون قد انتهيت من المتطلبات الأساسية، حان وقت استيراد الحزم اللازمة لمشروعك. في ملف C#، ستحتاج إلى تضمين مساحة الاسم التالية:

```csharp
using System.IO;
using Aspose.Cells;
```

تحتوي مساحة الاسم هذه على جميع الفئات والطرق اللازمة للعمل مع ملفات Excel وتنفيذ الوظائف التي نحتاجها.

لنبدأ بشرح عملية حماية خلايا محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. سنُقسّم الكود إلى عدة خطوات سهلة الفهم:

## الخطوة 1: إعداد دليل العمل الخاص بك

أول ما نريد فعله هو تحديد مكان ملفاتك. هذه الخطوة بسيطة، حيث ستحدد مجلدًا لملف Excel.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
هنا، نقوم بتعريف متغير سلسلة `dataDir` يشير إلى مجلد المستند المطلوب. نتحقق من وجود هذا المجلد. إذا لم يكن موجودًا، نقوم بإنشائه. هذا يضمن عدم مواجهة أي مشاكل عند حفظ ملف Excel لاحقًا.

## الخطوة 2: إنشاء مصنف جديد

بعد ذلك، دعنا نقوم بإنشاء مصنف جديد للعمل عليه.

```csharp
// إنشاء مصنف جديد.
Workbook wb = new Workbook();
```
لقد قمنا بإنشاء مثيل جديد `Workbook` اعتبر هذا بمثابة لوحة فارغة سترسم عليها بياناتك.

## الخطوة 3: الوصول إلى ورقة العمل

الآن بعد أن أصبح لدينا مصنف، فلننتقل إلى ورقة العمل الأولى حيث سنطبق إعدادات الحماية الخاصة بنا.

```csharp
// إنشاء كائن ورقة عمل والحصول على الورقة الأولى.
Worksheet sheet = wb.Worksheets[0];
```
هنا، نصل إلى أول ورقة عمل من كتابنا. هنا سيحدث كل السحر!

## الخطوة 4: فتح جميع الأعمدة

قبل قفل خلايا محددة، يجب فتح جميع الأعمدة في ورقة العمل. هذا يسمح بقفل الخلايا المحددة فقط لاحقًا.

```csharp
// تعريف كائن النمط.
Style style;
// تعريف كائن styleflag.
StyleFlag styleflag;

// قم بالمرور على جميع الأعمدة في ورقة العمل وإلغاء قفلها.
for (int i = 0; i <= 255; i++)
{
    style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false;
    styleflag = new StyleFlag();
    styleflag.Locked = true;
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, styleflag);
}
```
تتكرر هذه الحلقة على جميع الأعمدة (من ٠ إلى ٢٥٥) في ورقة العمل، مع فتح كل منها. بذلك، نُهيئ الظروف لقفل الخلايا التي نختارها لاحقًا فقط.

## الخطوة 5: قفل خلايا محددة

الآن نصل إلى الجزء المثير: قفل خلايا محددة! في هذا المثال، سنقفل الخلايا A1 وB1 وC1.

```csharp
// قم بإغلاق الخلايا الثلاث...أي A1، B1، C1.
style = sheet.Cells["A1"].GetStyle();
style.IsLocked = true;
sheet.Cells["A1"].SetStyle(style);

style = sheet.Cells["B1"].GetStyle();
style.IsLocked = true;
sheet.Cells["B1"].SetStyle(style);

style = sheet.Cells["C1"].GetStyle();
style.IsLocked = true;
sheet.Cells["C1"].SetStyle(style);
```
بالنسبة لكل خلية محددة، نسترد النمط الحالي ونضبط `IsLocked` الخاصية إلى true. الآن، هذه الخلايا الثلاث مقفلة ولا يمكن تعديلها بعد الآن.

## الخطوة 6: حماية ورقة العمل

قائمتنا شبه مكتملة! الخطوة الأخيرة هي حماية ورقة العمل نفسها.

```csharp
// وأخيرًا، قم بحماية الورقة الآن.
sheet.Protect(ProtectionType.All);
```
عن طريق الاتصال `Protect` باستخدام الطريقة الموجودة في ورقة العمل، نطبق إعدادات الحماية لدينا. باستخدام `ProtectionType.All`، نحن نحدد أن جميع جوانب الورقة ستكون محمية.

## الخطوة 7: حفظ ملف Excel

وأخيرًا، دعونا نحفظ عملنا اليدوي في ملف Excel.

```csharp
// احفظ ملف الاكسل.
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```
يحفظ هذا الأمر المصنف في المجلد المحدد باسم "output.out.xls". يمكنك الوصول إلى هذا الملف في أي وقت لمشاهدة الخلايا المحمية أثناء عملها.

## خاتمة

ها قد انتهيت! لقد نجحت في حماية خلايا محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. باتباع هذه الخطوات، ستتعلم كيفية إعداد بيئتك، وإنشاء مصنف Excel، وقفل الخلايا بشكل مشروط للحفاظ على سلامة البيانات. لذا، في المرة القادمة التي تفكر فيها بالسماح للآخرين بتعديل جداول بياناتك، تذكر التقنيات البسيطة التي يمكنك تطبيقها لحماية بياناتك المهمة!
