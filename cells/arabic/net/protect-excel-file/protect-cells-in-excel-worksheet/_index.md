# حماية الخلايا في ورقة عمل Excel

## مقدمة

في عالمنا الرقمي اليوم، أصبحت إدارة البيانات بأمان في جداول البيانات أكثر أهمية من أي وقت مضى. سواء كنت تتعامل مع معلومات حساسة أو ترغب ببساطة في ضمان سلامة تنسيقك، فإن حماية خلايا محددة في ورقة عمل Excel قد تُحدث فرقًا كبيرًا. لحسن الحظ، إذا كنت تستخدم .NET، فإن Aspose.Cells يُسهّل هذه العملية. في هذه المقالة، سنستكشف دليلًا سهلًا خطوة بخطوة لحماية الخلايا في ورقة عمل Excel، لضمان سلامة بياناتك.

## المتطلبات الأساسية

قبل الخوض في التفاصيل الدقيقة لحماية الخلايا، هناك بعض المتطلبات الأساسية التي يجب أن تكون موجودة لديك:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهاز الكمبيوتر الخاص بك. إنه بيئة التطوير المتكاملة الأساسية لتطوير .NET.
2. مكتبة Aspose.Cells: يجب أن تكون مكتبة Aspose.Cells متوفرة في مشروعك. يمكنك تثبيتها بسهولة عبر مدير حزم NuGet أو تنزيلها مباشرةً من [موقع Aspose.Cells](https://releases.aspose.com/cells/net/).
3. المعرفة الأساسية بلغة C#: إن معرفة القليل من لغة البرمجة C# سوف تساعدك على المتابعة بسلاسة.

## استيراد الحزم

الخطوة الأولى في رحلتنا هي استيراد الحزم المطلوبة إلى مشروعك. إليك كيفية القيام بذلك:

### إنشاء مشروع C# جديد

- افتح Visual Studio وقم بإنشاء مشروع تطبيق وحدة تحكم جديد (.NET Framework).
- قم بتسمية مشروعك بشيء ذي معنى (مثل "ProtectCellsExample").

### إضافة مرجع Aspose.Cells

- في مستكشف الحلول، انقر بزر الماوس الأيمن فوق مشروعك وحدد "إدارة حزم NuGet".
- ابحث عن "Aspose.Cells" وانقر على "تثبيت". ستتيح لك هذه المكتبة الوصول إلى جميع الطرق اللازمة لحماية خلاياك.

### استخدام مساحات الأسماء

بمجرد إضافة المرجع، تأكد من استيراد المساحات الأساسية اللازمة في أعلى ملف التعليمات البرمجية الخاص بك:

```csharp
using System.IO;
using Aspose.Cells;
```

الآن بعد أن وضعنا الأساس، دعونا ننتقل إلى الحدث الرئيسي.

دعونا نقوم بتحليل مثال الكود الذي يوضح كيفية حماية خلايا معينة في ورقة عمل Excel.

## الخطوة 1: إعداد دليل البيانات

عليك أولاً تحديد مكان حفظ ملف Excel. إليك كيفية تحديد ذلك:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // حدد مسار الدليل الخاص بك هنا
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

يتحقق هذا المقطع من وجود دليل محدد. إذا لم يكن موجودًا، فسيتم إنشاؤه. هذا ضروري لضمان وجود مجلد مخصص لملفك المحفوظ!

## الخطوة 2: إنشاء مصنف جديد

بعد ذلك، نحتاج إلى إنشاء مصنف جديد. يوفر Aspose.Cells طريقة بسيطة للقيام بذلك:

```csharp
Workbook wb = new Workbook();
```

يقوم هذا السطر بإنشاء مصنف جديد للعمل عليه.

## الخطوة 3: الوصول إلى ورقة العمل الأولى

في معظم الحالات، سوف تعمل في الورقة الأولى من المصنف الخاص بك:

```csharp
Worksheet sheet = wb.Worksheets[0]; // الوصول إلى ورقة العمل الأولى
```

الأمر بسيط جدًا! لديك الآن مرجع للورقة الأولى حيث ستقفل الخلايا.

## الخطوة 4: فتح جميع الأعمدة

لتتأكد من قفل خلايا محددة فقط، عليك أن تبدأ بإلغاء قفل جميع الأعمدة:

```csharp
for (int i = 0; i <= 255; i++)
{
    Style style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false; // فتح العمود
    StyleFlag styleflag = new StyleFlag();
    styleflag.Locked = true; // أشر إلى أننا نريد قفل هذا النمط
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, styleflag);
}
```

هذه الحلقة تمر عبر جميع الأعمدة الممكنة (حتى ٢٥٦ عمودًا) وتُفعّل أنماطها. بمعنى آخر، أنت تقول: "مرحبًا، جميعكم أحرار في التعديل!"

## الخطوة 5: قفل خلايا محددة

بعد فتح جميع الأعمدة، حان وقت قفل خلايا محددة. في مثالنا، نقوم بقفل الخلايا A1 وB1 وC1.

```csharp
style = sheet.Cells["A1"].GetStyle();
style.IsLocked = true; // القفل A1
sheet.Cells["A1"].SetStyle(style);

style = sheet.Cells["B1"].GetStyle();
style.IsLocked = true; // القفل B1
sheet.Cells["B1"].SetStyle(style);

style = sheet.Cells["C1"].GetStyle();
style.IsLocked = true; // القفل C1
sheet.Cells["C1"].SetStyle(style);
```

يتم الوصول إلى كل خلية على حدة، ونُعدِّل نمطها لقفلها. هذا أشبه بوضع قفل آمن على صندوق الكنز - مفاتيح محددة فقط هي التي تفتحه!

## الخطوة 6: حماية ورقة العمل

لفرض القفل، يجب حماية الورقة بأكملها. يمكن القيام بذلك باستخدام سطر التعليمات البرمجية التالي:

```csharp
sheet.Protect(ProtectionType.All);
```

عن طريق الاتصال `Protect` بهذه الطريقة، فأنت تطلب من Excel منع أي تعديلات ما لم تتم إزالة الحماية.

## الخطوة 7: حفظ المصنف

وأخيرًا، عليك حفظ عملك! إليك الطريقة:

```csharp
wb.Save(dataDir + "output.xls", SaveFormat.Excel97To2003);
```

هذا السطر يحفظ مصنفك كملف إكسل. تأكد من تحديد التنسيق المناسب!

## خاتمة

ها قد انتهيت! لقد تعلمت بنجاح حماية خلايا محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. ببضعة أسطر برمجية فقط، يمكنك حماية بياناتك، مع ضمان وصول الأشخاص المناسبين فقط إلى المعلومات المهمة وتعديلها. تذكر أن حماية الخلايا ليست سوى واحدة من الميزات العديدة التي يقدمها Aspose.Cells للمساعدة في إدارة ملفات Excel ومعالجتها بكفاءة.
