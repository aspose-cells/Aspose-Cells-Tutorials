## دليل التنفيذ

### قراءة معلومات Sparkline

#### ملخص
تتضمن قراءة معلومات المخطط الشريطي الوصول إلى المجموعات الموجودة وتفاصيلها داخل ورقة العمل.

**الخطوة 1: تهيئة المصنف وورقة العمل**

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook book = new Workbook(SourceDir + "/sampleUsingSparklines.xlsx");
Worksheet sheet = book.Worksheets[0];
```

**الخطوة 2: التكرار عبر مجموعات Sparkline**

```csharp
foreach (SparklineGroup g in sheet.SparklineGroups)
{
    Console.WriteLine("sparkline group: type:" + g.Type + ", sparkline items count:" + g.Sparklines.Count);
    
    foreach (Sparkline s in g.Sparklines)
    {
        Console.WriteLine("sparkline: row:" + s.Row + ", col:" + s.Column + ", dataRange:" + s.DataRange);
    }
}
```

في هذا الكود، `g.Type` و `g.Sparklines.Count` حدد نوع المجموعة وعدد المخططات البيانية. لكل مخطط بياني، يمكنك الوصول إلى موقعه (`Row`، `Column`) و `DataRange`.

### إضافة مخططات شرارة إلى ورقة عمل

#### ملخص
تتيح لك إضافة الرسوم البيانية الشريطية تصور اتجاهات البيانات برمجيًا.

**الخطوة 1: تحديد منطقة الخلية للمخططات البيانية الشريطية**

```csharp
CellArea ca = new CellArea();
ca.StartColumn = 4;
ca.EndColumn = 4;
ca.StartRow = 1;
ca.EndRow = 7;
```

**الخطوة 2: إضافة مجموعة Sparkline جديدة**

```csharp
int idx = sheet.SparklineGroups.Add(SparklineType.Column, "Sheet1!B2:D8", false, ca);
SparklineGroup group = sheet.SparklineGroups[idx];
```

هنا، `SparklineType.Column` يُحدد نوع الرسوم البيانية الشريطية المراد إضافتها. يتم تحديد نطاق البيانات ومنطقة العرض بواسطة مراجع الخلايا.

**الخطوة 3: تخصيص مظهر المخطط البياني**

```csharp
CellsColor clr = book.CreateCellsColor();
clr.Color = Color.Orange;
group.SeriesColor = clr;
```

يمكنك تخصيص اللون باستخدام `CellsColor`، تعزيز التمييز البصري.

**الخطوة 4: حفظ المصنف**

```csharp
book.Save(outputDir + "/outputUsingSparklines.xlsx");
```

يؤدي هذا إلى حفظ التغييرات التي أجريتها، والحفاظ على مؤشرات الشريط البياني المضافة حديثًا في دليل الإخراج المحدد.
