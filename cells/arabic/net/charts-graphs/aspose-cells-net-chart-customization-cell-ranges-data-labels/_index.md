## دليل التنفيذ

في هذا القسم، سنشرح كيفية تنفيذ تسميات البيانات التي تعرض نطاقات الخلايا داخل مخطط باستخدام Aspose.Cells.

### الخطوة 1: تحميل مصنف Excel

ابدأ بتحميل المصنف الخاص بك والوصول إلى ورقة العمل المطلوبة:

```csharp
// دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// إنشاء مصنف من ملف Excel المصدر
Workbook workbook = new Workbook(sourceDir + "sampleShowCellRangeAsDataLabels.xlsx");

// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```

### الخطوة 2: الوصول إلى تسميات بيانات الرسم البياني وتعديلها

بعد ذلك، قم بالوصول إلى الرسم البياني داخل ورقة العمل وقم بتكوين تسميات البيانات الخاصة به:

```csharp
// الوصول إلى الرسم البياني داخل ورقة العمل
Chart chart = worksheet.Charts[0];

// تكوين تسميات البيانات لإظهار نطاق الخلايا
DataLabels dataLabels = chart.NSeries[0].DataLabels;
dataLabels.LinkedSource = "=Sheet1!$B$2:$B$10"; // ربط نطاق الخلايا المحدد
dataLabels.ShowCellRange = true; // تمكين عرض نطاق الخلايا في تسميات البيانات

// حفظ التغييرات في مصنف جديد
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "outputShowCellRangeAsDataLabels.xlsx");
```

#### توضيح:
- **المصدر المرتبط:** تحدد هذه المعلمة نطاق خلايا Excel الذي يحتوي على القيم المعروضة كعناوين بيانات.
- **إظهار نطاق الخلية:** ضبط هذا على `true` يضمن ظهور نطاق الخلايا المحدد ضمن تسميات بيانات الرسم البياني.

### الخطوة 3: الحفظ والتحقق

وأخيرًا، احفظ المصنف الخاص بك مع التغييرات:
