# إنشاء مخطط هرمي في Excel باستخدام Aspose.Cells لـ .NET: دليل خطوة بخطوة

## مقدمة

حسّن مهاراتك في تصور البيانات بإنشاء مخططات هرمية ديناميكية مباشرةً من تطبيقات .NET. يرشدك هذا البرنامج التعليمي إلى كيفية إنشاء مخططات هرمية في ملفات Excel باستخدام مكتبة Aspose.Cells for .NET القوية. ستتعلم كيفية تهيئة مصنف، وإضافة بيانات نموذجية، وتكوين مخطط، وحفظ ملفك.

**ما سوف تتعلمه:**
- تهيئة مصنف Excel باستخدام Aspose.Cells
- ملء الخلايا ببيانات العينة
- إضافة مخطط هرمي وتخصيصه
- تعيين مصدر البيانات للرسم البياني الخاص بك
- حفظ المصنف في الدليل المحدد

هل أنت مستعد للبدء؟ لنبدأ بتجهيز كل شيء أولًا.

## دليل التنفيذ

اتبع الخطوات التالية لإنشاء مخطط هرمي.

### تهيئة المصنف وورقة العمل

**ملخص:**
سنبدأ بإنشاء مصنف Excel والوصول إلى ورقة العمل الأولى الخاصة به.

#### الخطوة 1: إنشاء مثيل مصنف

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// تهيئة كائن مصنف جديد
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

### إضافة بيانات العينة إلى الخلايا

**ملخص:**
بعد ذلك، قم بملء ورقة العمل ببيانات العينة لمخططنا.

#### الخطوة 2: ملء الخلايا

```csharp
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```

### إضافة مخطط هرمي إلى ورقة العمل

**ملخص:**
الآن، أضف مخططًا هرميًا لتوضيح البيانات.

#### الخطوة 3: إدراج مخطط هرمي

```csharp
using Aspose.Cells.Charts;

// إضافة مخطط هرمي إلى ورقة العمل
int chartIndex = worksheet.Charts.Add(ChartType.Pyramid, 5, 0, 25, 10);
Chart chart = worksheet.Charts[chartIndex];
```

### تعيين مصدر بيانات الرسم البياني

**ملخص:**
قم بتحديد نطاق البيانات الذي سيتم استخدامه لمخطط الهرم الخاص بنا.

#### الخطوة 4: تكوين بيانات الرسم البياني

```csharp
// تعيين نطاق مصدر البيانات للرسم البياني
chart.NSeries.Add("A1:B3", true);
```

### حفظ المصنف في ملف

**ملخص:**
وأخيرًا، احفظ المصنف الخاص بك باستخدام مخطط الهرم الذي قمت بإنشائه حديثًا.

#### الخطوة 5: حفظ ملف Excel

```csharp
workbook.Save(outputDir + "outputHowToCreatePyramidChart.xlsx");
```
