## دليل التنفيذ: تعديل تسميات بيانات المخطط الدائري

سنستعرض عملية تعديل تسميات البيانات في مخطط دائري باستخدام Aspose.Cells.

### ملخص

يتيح تعديل تسميات البيانات في المخططات الدائرية تمثيلًا نصيًا مخصصًا، مما يُحسّن الوضوح، ويوفر رؤىً مُحددة مباشرةً على المخطط. يتناول هذا القسم الوصول إلى هذه التسميات وتغييرها برمجيًا.

#### الخطوة 1: تحميل ملف Excel الخاص بك

أولاً، قم بتحميل مصنف Excel الذي يحتوي على الرسم البياني المطلوب:
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(sourceDir + "/sampleModifyPieChart.xlsx");
```
*توضيح:* ال `Workbook` تُستخدم الفئة لفتح ملف Excel موجود. استبدل `"YOUR_SOURCE_DIRECTORY"` مع المسار الفعلي لملفك.

#### الخطوة 2: الوصول إلى ورقة العمل والمخطط الخاصين بك

حدد ورقة العمل والمخطط الذي تريد تعديله:
```csharp
Worksheet sheet = workbook.Worksheets[1];
Chart chart = sheet.Charts[0];
```
*توضيح:* نقوم بالوصول إلى ورقة العمل الثانية (الفهرس 1) ونسترد المخطط الأول على تلك الورقة.

#### الخطوة 3: تعديل تسميات البيانات

الوصول إلى تسميات البيانات وتغييرها لنقطة محددة في مخططك الدائري:
```csharp
DataLabels datalabels = chart.NSeries[0].Points[2].DataLabels;
datalabels.Text = "United Kingdom, 400K ";
```
*توضيح:* هنا، `NSeries[0]` يستهدف سلسلة البيانات الأولى، و `Points[2]` يصل إلى النقطة الثالثة. ثم نحدد نصًا مخصصًا لعلامة البيانات الخاصة به.

#### الخطوة 4: حفظ التغييرات

وأخيرًا، احفظ المصنف الخاص بك مع التعديلات:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/outputModifyPieChart.xlsx");
```
*توضيح:* هذه الخطوة تُعيد كتابة التغييرات إلى ملف Excel في الدليل المُحدد. تأكد `"YOUR_OUTPUT_DIRECTORY"` تم تعريفه.
