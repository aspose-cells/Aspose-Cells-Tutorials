# كيفية تنسيق قيم سلسلة المخططات في Excel باستخدام Aspose.Cells .NET

**ما سوف تتعلمه:**

- إضافة سلسلة إلى مخطط وتعيين رمز تنسيق قيمها
- حفظ التغييرات مرة أخرى في ملف Excel

أولاً، دعونا نراجع المتطلبات الأساسية.

## دليل التنفيذ

دعونا نقسم العملية إلى خطوات مميزة لتسهيل التنفيذ.

### تحميل المصنف من الدليل

**ملخص:** ابدأ بتحميل مصنف Excel من الدليل المحدد.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
// تحميل ملف Excel المصدر 
Workbook wb = new Workbook(SourceDir + "sampleSeries_ValuesFormatCode.xlsx");
```

**توضيح:**
- `SourceDir` هو المسار إلى ملفات الإدخال الخاصة بك.
- ال `Workbook` يفتح المنشئ الملف المحدد.

### الوصول إلى ورقة العمل من المصنف

**ملخص:** استرداد ورقة العمل التي تحتاج إلى العمل بها.

```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = wb.Worksheets[0];
```

**توضيح:**
- يمكن أن تحتوي مصنفات العمل على عدة أوراق عمل. هنا، نصل إلى الورقة الأولى باستخدام فهرس `0`.

### الوصول إلى المخطط من ورقة العمل

**ملخص:** حدد المخطط ضمن ورقة العمل المحددة التي تريد التعامل معه.

```csharp
// الوصول إلى الرسم البياني الأول
Chart ch = worksheet.Charts[0];
```

**توضيح:**
- كما هو الحال في أوراق العمل، يمكن أن تحتوي ورقة العمل على عدة مخططات بيانية. هذا الكود يُمكّن من الوصول إلى المخطط الأول.

### إضافة سلسلة إلى الرسم البياني

**ملخص:** أضف سلسلة بيانات إلى الرسم البياني الخاص بك باستخدام مجموعة من القيم.

```csharp
// إضافة سلسلة باستخدام مجموعة من القيم
ch.NSeries.Add("{10000, 20000, 30000, 40000}", true);
```

**توضيح:**
- `NSeries.Add` يأخذ تمثيلًا نصيًا للأرقام وقيمة منطقية تشير إلى ما إذا كان النطاق حصريًا. هنا، يكون شاملًا.

### تعيين رمز تنسيق قيم السلسلة

**ملخص:** قم بتخصيص كيفية تنسيق القيم في سلسلة الرسم البياني الخاصة بك.

```csharp
// قم بالوصول إلى السلسلة وتعيين رمز تنسيق القيم الخاصة بها
Series srs = ch.NSeries[0];
srs.ValuesFormatCode = "$#,##0";
```

**توضيح:**
- `ValuesFormatCode` يسمح لك بتحديد تنسيق رقم مخصص، مثل العملة في هذا المثال (`"$#,##0"`).

### حفظ المصنف في الدليل

**ملخص:** حافظ على تغييراتك عن طريق حفظ المصنف في دليل الإخراج.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
// حفظ ملف Excel الناتج
wb.Save(outputDir + "outputSeries_ValuesFormatCode.xlsx");
```

**توضيح:**
- ال `Save` تكتب الطريقة المصنف المعدل إلى ملف جديد، مع الحفاظ على التغييرات التي أجريتها.
