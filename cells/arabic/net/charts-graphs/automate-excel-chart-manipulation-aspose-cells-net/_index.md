## دليل التنفيذ

في هذا القسم، سنستعرض الخطوات المطلوبة لقراءة المخططات ومعالجتها في ملف Excel 2016.

### الوصول إلى المخططات البيانية في ورقة العمل

نبدأ بتحميل مصنف المصدر الخاص بنا والوصول إلى ورقة العمل الأولى الخاصة به، والتي تحتوي على مخططاتنا:

```csharp
// تحميل ملف Excel
Workbook book = new Workbook("sampleReadAndManipulateExcel2016Charts.xlsx");

// الوصول إلى ورقة العمل الأولى
Worksheet sheet = book.Worksheets[0];
```

### أنواع مخططات القراءة

بعد ذلك، نقوم بتكرار كل مخطط في ورقة العمل لقراءة نوعه وطباعته:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    // احصل على الرسم البياني الحالي
    Chart ch = sheet.Charts[i];

    // طباعة نوع الرسم البياني
    Console.WriteLine(ch.Type);
}
```

### تعديل عناوين المخططات

يمكننا تغيير عنوان كل مخطط ليعكس نوعه:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    Chart ch = sheet.Charts[i];

    // تحديث عنوان الرسم البياني
    ch.Title.Text = "Chart Type is " + ch.Type.ToString();
}
```

### حفظ التغييرات

وأخيرًا، احفظ تعديلاتك مرة أخرى في ملف Excel جديد:

```csharp
book.Save("outputReadAndManipulateExcel2016Charts.xlsx");
Console.WriteLine("Manipulation completed successfully.");
```
