# كيفية إنشاء مصنف وإعداد مخطط باستخدام Aspose.Cells .NET

- إعداد أنواع مختلفة من المخططات البيانية باستخدام Aspose.Cells .NET.
- احفظ مصنفك بكفاءة.

دعونا نبدأ بالمتطلبات الأساسية المطلوبة قبل الغوص في التنفيذ.

### إنشاء مصنف جديد
**ملخص**:ابدأ بتهيئة مصنف Excel جديد، ليكون بمثابة الحاوية لبياناتك ومخططاتك.

```csharp
// إنشاء مصنف جديد
tWorkbook workbook = new tWorkbook(tFileFormatType.Xlsx);
```
هنا، `tFileFormatType.Xlsx` يحدد أننا نقوم بإنشاء ملف Excel بتنسيق XLSX، مما يضمن التوافق مع إصدارات Excel الحديثة.

### إضافة البيانات إلى ورقة العمل
**ملخص**املأ ورقة العمل بالبيانات اللازمة لإنشاء المخطط. إليك كيفية إضافة قيم محور الفئة وبيانات السلسلة:

```csharp
// الوصول إلى ورقة العمل الأولى
tWorksheet worksheet = workbook.Worksheets[0];

// إضافة بيانات للرسم البياني
tworksheet.Cells["A2"].PutValue("C1");
tworksheet.Cells["A3"].PutValue("C2");
tworksheet.Cells["A4"].PutValue("C3");

// السلسلة العمودية الأولى
tworksheet.Cells["B1"].PutValue("T1");
tworksheet.Cells["B2"].PutValue(6);
tworksheet.Cells["B3"].PutValue(3);
tworksheet.Cells["B4"].PutValue(2);

// السلسلة العمودية الثانية
tworksheet.Cells["C1"].PutValue("T2");
tworksheet.Cells["C2"].PutValue(7);
tworksheet.Cells["C3"].PutValue(2);
tworksheet.Cells["C4"].PutValue(5);

// السلسلة العمودية الثالثة
tworksheet.Cells["D1"].PutValue("T3");
tworksheet.Cells["D2"].PutValue(8);
tworksheet.Cells["D3"].PutValue(4);
tworksheet.Cells["D4"].PutValue(2);
```
كل `PutValue` تضيف طريقة الاتصال البيانات إلى خلية معينة، مما يضع الأساس لمخططك.

### إعداد وتكوين الرسم البياني
**ملخص**:بعد ملء ورقة العمل بالبيانات، قم بإنشاء مخطط عمودي وتكوينه.

```csharp
// إنشاء مخطط عمودي بسهولة
tint idx = tworksheet.Charts.Add(tChartType.Column, 6, 5, 20, 13);	tChart ch = tworksheet.Charts[idx];	ch.SetChartDataRange("A1:D4", true);
```
تضيف هذه القطعة مخططًا عموديًا إلى ورقة العمل وتحدد نطاق بياناتها من `A1` ل `D4`، التأكد من تضمين جميع البيانات المضافة في التصور.

### حفظ المصنف
**ملخص**أخيرًا، احفظ مصنفك بجميع الإعدادات. إليك كيفية القيام بذلك:

```csharp
// حفظ المصنف
tworkbook.Save(outputDir + "output_out.xlsx", tSaveFormat.Xlsx);
```
ال `Save` تكتب الطريقة مصنفك إلى ملف بالتنسيق المحدد (XLSX)، مما يجعله جاهزًا للاستخدام أو التوزيع.
