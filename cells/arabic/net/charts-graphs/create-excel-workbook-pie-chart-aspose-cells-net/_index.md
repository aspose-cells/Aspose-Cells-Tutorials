# إنشاء مصنف Excel باستخدام مخطط دائري باستخدام Aspose.Cells .NET

## مقدمة

في عالمنا اليوم الذي يعتمد على البيانات، يُعدّ التصور الفعّال للمعلومات أمرًا بالغ الأهمية. سواء كنت تُدير بيانات المبيعات أو تُحلّل مقاييس الأداء الإقليمية، يُمكن لمخطط دائري مُصمّم بإتقان في Excel أن يجعل رؤاك أكثر وضوحًا وتأثيرًا. قد يستغرق إنشاء هذه المخططات يدويًا وقتًا طويلاً. استخدم Aspose.Cells لـ .NET، وهي مكتبة فعّالة تُبسّط إنشاء تقارير Excel الديناميكية برمجيًا.

سيرشدك هذا البرنامج التعليمي خلال عملية إنشاء مصنف Excel من الصفر، وملئه بالبيانات، وإضافة مخطط دائري جذاب - كل ذلك باستخدام لغة C#. صُمم هذا الدليل خصيصًا لمن يرغبون في الاستفادة من Aspose.Cells لـ .NET، مما يجعل مهام تصور البيانات الخاصة بك سلسة وفعالة.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Cells في مشروع .NET الخاص بك.
- خطوات إنشاء مصنف Excel جديد وملئه ببيانات المبيعات النموذجية.
- تقنيات لإضافة مخطط دائري وتخصيصه باستخدام Aspose.Cells.
- أفضل الممارسات لتحسين الأداء عند التعامل مع مجموعات البيانات الكبيرة.

دعونا نبدأ بتغطية المتطلبات الأساسية التي ستحتاجها قبل البدء في هذه الرحلة.

## دليل التنفيذ

سنقسّم العملية إلى أقسام منطقية بناءً على الميزات. سيقدم كل قسم نظرة عامة، متبوعة بتعليمات خطوة بخطوة مع مقتطفات من التعليمات البرمجية.

### إنشاء مصنف وتعبئته

**ملخص**:توضح هذه الميزة كيفية إنشاء مصنف جديد، والوصول إلى ورقة العمل الأولى الخاصة به، وتعيين اسم الورقة، وملئها بالبيانات.

1. **إنشاء مصنف جديد**
   
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   Workbook workbook = new Workbook();
   ```

2. **الوصول إلى ورقة العمل الأولى واسم المجموعة**
   
   ```csharp
   Worksheet sheet = workbook.Worksheets[0];
   sheet.Name = "Data";
   ```

3. **ملء ورقة العمل بالبيانات**
   
   ```csharp
   Cells cells = sheet.Cells;
   cells["A1"].PutValue("Region");
   // ملء بيانات المنطقة
   cells["A2"].PutValue("France");
   // متابعة للمناطق الأخرى...

   cells["B1"].PutValue("Sale");
   // ملء أرقام المبيعات
   cells["B2"].PutValue(70000);
   ```

### إضافة ورقة مخطط وإنشاء مخطط دائري

**ملخص**:تعرف على كيفية إضافة ورقة مخطط جديدة، وإنشاء مخطط دائري، وتعيين خصائصه الأساسية.

1. **إضافة ورقة رسم بياني جديدة**
   
   ```csharp
   int sheetIndex = workbook.Worksheets.Add(SheetType.Chart);
   Worksheet chartSheet = workbook.Worksheets[sheetIndex];
   chartSheet.Name = "Chart";
   ```

2. **إنشاء مخطط دائري**
   
   ```csharp
   int chartIndex = chartSheet.Charts.Add(ChartType.Pie, 5, 0, 25, 10);
   Chart chart = chartSheet.Charts[chartIndex];
   ```

### تكوين خصائص الرسم البياني

**ملخص**:قم بتخصيص خصائص منطقة الرسم البياني والعنوان والسلسلة لمخططك الدائري.

1. **تكوين مساحة الرسم والعنوان**
   
   ```csharp
   chart.PlotArea.Area.ForegroundColor = Color.Coral;
   chart.Title.Text = "Sales By Region";
   chart.Title.Font.Color = Color.Blue;
   ```

2. **تعيين خصائص السلسلة**
   
   ```csharp
   chart.NSeries.Add("Data!B2:B8", true);
   chart.NSeries.CategoryData = "Data!A2:A8";
   chart.NSeries.IsColorVaried = true;
   ```

### إعداد تسميات البيانات لسلسلة المخططات

**ملخص**:قم بتعزيز مخططك الدائري عن طريق إضافة تسميات البيانات إلى كل سلسلة.

1. **إضافة تسميات البيانات**
   
   ```csharp
   for (int i = 0; i < chart.NSeries.Count; i++) {
       DataLabels datalabels = chart.NSeries[i].DataLabels;
       datalabels.Position = LabelPositionType.InsideBase;
       datalabels.ShowCategoryName = true;
       datalabels.ShowValue = true;
   }
   ```

### تخصيص منطقة الرسم البياني والأسطورة

**ملخص**:قم بتخصيص مخططك الدائري بشكل أكبر عن طريق ضبط خصائص منطقة المخطط والأسطورة.

1. **تخصيص منطقة الرسم البياني**
   
   ```csharp
   ChartArea chartarea = chart.ChartArea;
   chartarea.Area.Formatting = FormattingType.Custom;
   chartarea.Area.FillFormat.Texture = TextureType.BlueTissuePaper;
   ```

2. **تعديل خصائص الأسطورة**
   
   ```csharp
   Legend legend = chart.Legend;
   legend.Position = LegendPositionType.Left;
   legend.Font.IsBold = true;
   legend.Border.Color = Color.Blue;
   ```

### حفظ المصنف

**ملخص**:احفظ المصنف الخاص بك مع جميع المخططات والبيانات التي قمت بتكوينها.

```csharp
workbook.Save(outputDir + "outputHowToCreatePieChart.xlsx");
```
