## دليل التنفيذ
### تخصيص تسميات المخططات استنادًا إلى الإعدادات المحلية
الهدف الرئيسي هو تجاوز النص الافتراضي لعناوين المخططات الدائرية باستخدام إعدادات خاصة بكل ثقافة. إليك كيفية تحقيق ذلك:

#### 1. قم بتحميل مصنفك والوصول إلى المخطط
ابدأ بتحميل ملف Excel موجود يحتوي على مخطط دائري:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleCustomTextForLabels.xlsx");
```

قم بالوصول إلى ورقة العمل والمخطط الذي ترغب في تخصيصه:
```csharp
Worksheet sheet = book.Worksheets[0];
Chart chart = sheet.Charts[0];
```

#### 2. ضبط إعدادات العولمة
تجاوز `GetOtherName` طريقة لتوفير تسميات مخصصة استنادًا إلى إعدادات النظام المحلية:

```csharp
GlobalizationSettings globalSettings = new GlobalizationSettings();
globalSettings.ChartSettings = new CustomSettings();
book.Settings.GlobalizationSettings = globalSettings;
```

قم بتحديد فئة الإعدادات المخصصة الخاصة بك:
```csharp
class CustomSettings : ChartGlobalizationSettings
{
    public override string GetOtherName()
    {
        int lcid = System.Globalization.CultureInfo.CurrentCulture.LCID;
        switch (lcid)
        {
            case 1033: // إنجليزي
                return "Other";
            case 1036: // فرنسي
                return "Autre";
            case 1031: // الألمانية
                return "Andere";
            default:
                return base.GetOtherName();
        }
    }
}
```

#### 3. تحديث الرسم البياني وتقديمه
لتطبيق التغييرات، قم بتحديث الرسم البياني وتحويله إلى ملف صورة:

```csharp
chart.Calculate();
chart.ToImage(outputDir + "outputCustomTextForLabels.png", new ImageOrPrintOptions());
Console.WriteLine("CustomTextForLabels executed successfully.");
```
