### الوصول إلى خصائص المستند المخصصة
#### ملخص
خصائص المستند المخصصة هي بيانات وصفية مرتبطة بملف Excel، وهي مفيدة لتخزين معلومات إضافية، مثل تفاصيل المؤلف، وأرقام الإصدارات، والعلامات المخصصة. الوصول إلى هذه الخصائص برمجيًا يُحسّن سير عمل إدارة البيانات بشكل كبير.

#### التنفيذ خطوة بخطوة
**1. تحميل المصنف**
ابدأ بتحميل مصنف Excel الخاص بك من دليل محدد:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sample-document-properties.xlsx");
```

**2. استرداد خصائص المستند المخصصة**
الوصول إلى جميع خصائص المستند المخصصة المحددة في ملف Excel الخاص بك:
```csharp
Aspose.Cells.Properties.DocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```

**3. الوصول إلى خصائص محددة**
يمكنك استرجاع خصائص فردية باستخدام فهرسها أو اسمها. إليك كيفية الوصول إلى الخاصيتين الأوليين:
```csharp
// الوصول إلى خاصية المستند المخصص الأول
Aspose.Cells.Properties.DocumentProperty customProperty1 = customProperties[0];
object objectValue = customProperty1.Value;

// الوصول إلى نوع خاصية المستند المخصص الثاني والتحقق منه
Aspose.Cells.Properties.DocumentProperty customProperty2 = customProperties[1];
if (customProperty2.Type == Aspose.Cells.Properties.PropertyType.String)
{
    string value = customProperty2.Value.ToString();
}
```
#### توضيح
- **حدود**: ال `Workbook` يقوم الفصل بتحميل ملف Excel الخاص بك، و `CustomDocumentProperties` تتيح لك المجموعة التفاعل مع كافة الخصائص المحددة من قبل المستخدم.
  
- **قيم الإرجاع**:تعيد كل خاصية في المجموعة مثيلًا لـ `DocumentProperty`، الذي يحمل اسم وقيمة ونوع خاصية مستند مخصصة.

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تحديد مسار دليل المصدر الخاص بك بشكل صحيح.
- قم بمعالجة الاستثناءات عند الوصول إلى خصائص غير موجودة لمنع أخطاء وقت التشغيل.
