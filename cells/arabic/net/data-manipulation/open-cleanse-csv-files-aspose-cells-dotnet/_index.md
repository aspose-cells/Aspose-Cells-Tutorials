## دليل التنفيذ

سيرشدك هذا القسم خلال فتح ملف CSV وتنظيفه باستخدام Aspose.Cells.

### فتح ملف CSV

#### ملخص

يُسهّل Aspose.Cells فتح ملفات CSV. سنحمّل ملف CSV بإعدادات مُخصّصة للتعامل مع الأحرف غير الصالحة بفعالية.

#### التنفيذ خطوة بخطوة

1. **إعداد دليل المصدر:**
   
   ```csharp
   string sourceDir = RunExamples.Get_SourceDirectory();
   var filename = sourceDir + "[20180220142533][ASPOSE_CELLS_TEST].csv";
   ```

2. **تحميل ملف CSV باستخدام الخيارات المخصصة:**
   
   ```csharp
   var workbook = new Workbook(filename, new TxtLoadOptions()
   {
       Separator = ';',
       LoadFilter = new LoadFilter(LoadDataFilterOptions.CellData),
       CheckExcelRestriction = false,
       ConvertNumericData = false,
       ConvertDateTimeData = false
   });
   ```

3. **عرض معلومات ورقة العمل:**
   
   ```csharp
   Console.WriteLine(workbook.Worksheets[0].Name);
   Console.WriteLine("CSV file opened successfully!");
   ```

**المعلمات موضحة:**
- `Separator`:يحدد الفاصل المستخدم في ملف CSV الخاص بك.
- `LoadFilter`:يحدد البيانات التي سيتم تحميلها (على سبيل المثال، CellData).
- `CheckExcelRestriction`:يسمح بالتعامل مع الملفات الأكبر من قيود Excel.
