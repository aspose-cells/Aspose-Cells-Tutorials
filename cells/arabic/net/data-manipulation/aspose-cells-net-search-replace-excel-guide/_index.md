## دليل التنفيذ

### البحث والاستبدال في نطاق

تتيح لك هذه الميزة البحث بكفاءة عن بيانات محددة ضمن نطاق محدد في ورقة Excel واستبدالها ببيانات جديدة. لنشرح خطوات التنفيذ بالتفصيل.

#### ملخص

ستقوم بتكوين منطقة الخلية، وإعداد خيارات البحث، والتنقل بين الخلايا للبحث عن القيم واستبدالها، وحفظ المصنف المعدّل.

#### تنفيذ الكود

1. **تحديد الدلائل وتحميل المصنف**
   ابدأ بتحديد مجلدي المصدر والإخراج. ثم حمّل ملف Excel باستخدام `Workbook`.

   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string OutputDir = "YOUR_OUTPUT_DIRECTORY";

   Workbook workbook = new Workbook(SourceDir + "sampleSearchReplaceDataInRange.xlsx");
   Worksheet worksheet = workbook.Worksheets[0];
   ```

2. **تحديد النطاق وإعداد خيارات البحث**
   إنشاء `CellArea` لتحديد المكان الذي تريد البحث فيه، وتكوين خيارات البحث.

   ```csharp
   CellArea area = CellArea.CreateCellArea("E9", "H15");

   FindOptions opts = new FindOptions();
   opts.LookInType = LookInType.Values;
   opts.LookAtType = LookAtType.EntireContent;
   opts.SetRange(area);
   ```

3. **البحث عن البيانات واستبدالها**
   استخدم حلقة للعثور على كل ظهور لمصطلح البحث ضمن النطاق، واستبداله ببيانات جديدة.

   ```csharp
   Cell cell = null;

   while (true)
   {
       cell = worksheet.Cells.Find("search", cell, opts);
       if (cell == null) break;
       cell.PutValue("replace");
   }
   ```

4. **حفظ المصنف المعدل**
   وأخيرًا، احفظ التغييرات في ملف جديد في دليل الإخراج.

   ```csharp
   workbook.Save(OutputDir + "outputSearchReplaceDataInRange.xlsx");
   ```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن جميع مسارات الدليل صحيحة ويمكن الوصول إليها.
- التحقق مرة أخرى من تعريفات نطاق الخلايا في `CellArea.CreateCellArea`.
