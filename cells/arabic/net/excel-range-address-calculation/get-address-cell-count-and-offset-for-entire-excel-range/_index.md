# الحصول على العنوان وعدد الخلايا والإزاحة لنطاق Excel بأكمله

هل سبق لك أن وجدت نفسك تُعقّد بياناتك في Excel، أو تحتاج إلى الوصول بسرعة إلى نطاقات مُعينة، أو تُحاول معرفة عدد الخلايا التي تعمل عليها؟ حسنًا، أنت محظوظ! اليوم، نتعمق في عالم Aspose.Cells لـ .NET، وهي مكتبة رائعة تُمكّنك من التعامل مع ملفات Excel بسهولة. بنهاية هذا الدليل، ستتعلم كيفية الحصول على العنوان، وعدد الخلايا، وتحديد الإزاحات لنطاق كامل. اعتبر هذا دليلك لتصبح خبيرًا في Excel باستخدام C#!
إذن، اجلس واستمتع بمشروبك المفضل، ولنبدأ!

## استيراد الحزم
للبدء، نحتاج إلى استيراد بعض الحزم الأساسية. هذه هي العناصر الأساسية التي ستساعدنا على التفاعل مع ملفات Excel في .NET. إليك كيفية القيام بذلك:
### افتح مشروعك
افتح Visual Studio وأنشئ مشروع C# جديدًا. اختر تطبيق وحدة التحكم، لأننا سنشغّل الكود من وحدة التحكم.
### إضافة حزمة NuGet
قبل البدء بالبرمجة، لنُضِف حزمة Aspose.Cells. إليك الطريقة:
1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. في مدير الحزم NuGet، ابحث عن "Aspose.Cells".
4. انقر فوق "تثبيت" لإضافة الحزمة إلى مشروعك.
### استيراد مساحة الاسم
في الجزء العلوي من `Program.cs` الملف، استيراد مساحة اسم Aspose.Cells:
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

الآن، لنُقسّم الأمر إلى خطوات سهلة. سنُنشئ تطبيقًا بسيطًا يتفاعل مع Excel ويسترجع معلومات مفيدة حول نطاق مُحدد.
## الخطوة 1: إنشاء مصنف فارغ
في هذه الخطوة، سننشئ مصنفًا جديدًا. هذا المصنف هو في الأساس ملف Excel بأكمله.
```csharp
// إنشاء مصنف فارغ.
Workbook wb = new Workbook();
```
يقوم هذا السطر من التعليمات البرمجية بتهيئة مثيل جديد من مصنف، مما يمنحنا لوحة نظيفة للعمل بها.
## الخطوة 2: الوصول إلى ورقة العمل الأولى
بعد ذلك، نحتاج إلى الحصول على ورقة عمل محددة ضمن المصنف. افتراضيًا، يُعطينا إكسل ورقة عمل واحدة - كما توقعت - الأولى!
```csharp
// الوصول إلى ورقة العمل الأولى.
Worksheet ws = wb.Worksheets[0];
```
هنا، نقوم بالفهرسة في `Worksheets` مجموعة للاستيلاء على الورقة الأولى.
## الخطوة 3: إنشاء نطاق
الآن، لنُنشئ نطاقًا داخل ورقة العمل. يمكن أن يكون النطاق خلية واحدة أو مجموعة خلايا. سننشئ نطاقًا يمتد من A1 إلى B3.
```csharp
// إنشاء النطاق A1:B3.
Console.WriteLine("Creating Range A1:B3\n");
Range rng = ws.Cells.CreateRange("A1:B3");
```
ال `CreateRange` تُنشئ الطريقة النطاق المُحدد. ستلاحظ أننا قمنا بطباعة رسالة إلى وحدة التحكم لمتابعة ما يحدث.
## الخطوة 4: طباعة عنوان النطاق
لفهم مكان وجود بياناتنا، يمكننا استرداد عنوان النطاق:
```csharp
// طباعة عنوان النطاق وعدد الخلايا.
Console.WriteLine("Range Address: " + rng.Address);
```
من خلال هذا السطر، نعرض عنوان النطاق، والذي يجب أن يُخرج "A1:B3".
## الخطوة 5: طباعة الفاصل
من الضروري الحفاظ على نظافة مخرجات وحدة التحكم. لذا، نضيف فاصلًا صغيرًا.
```csharp
// تنسيق مخرجات وحدة التحكم.
Console.WriteLine("----------------------");
Console.WriteLine("");
```
## الخطوة 6: إنشاء نطاق جديد A1
الآن حان وقت التعمق في النطاق A1. إليك كيفية القيام بذلك:
```csharp
// إنشاء النطاق A1.
Console.WriteLine("Creating Range A1\n");
rng = ws.Cells.CreateRange("A1");
```
يؤدي هذا إلى إنشاء نطاق جديد يتكون فقط من الخلية A1.
## الخطوة 7: استرداد وطباعة الأوفست
لنستكشف بعض الميزات الرائعة للنطاق. على سبيل المثال، يمكننا تحديد الإزاحة من A1 إلى خلية أخرى.
```csharp
// طباعة نطاق الإزاحة، العمود بأكمله والصف بأكمله.
Console.WriteLine("Offset: " + rng.GetOffset(2, 2).Address);
```
ال `GetOffset` تسمح لنا هذه الطريقة بتحديد عدد الصفوف والأعمدة المراد نقلها من موضع البداية. في هذه الحالة، ننقل صفين للأسفل وعمودين للأعلى، مما يوصلنا إلى C3.
## الخطوة 8: طباعة العمود والصف بالكامل
الآن، دعونا نكتشف إلى أي عمود أو صف ينتمي A1:
```csharp
Console.WriteLine("Entire Column: " + rng.EntireColumn.Address);
Console.WriteLine("Entire Row: " + rng.EntireRow.Address);
```
ستقوم هذه المكالمات بإخراج العمود A بالكامل والصف 1 بالكامل، مما يساعدنا في تحديد جميع الخلايا المرتبطة بنطاقنا.
## الخطوة 9: فاصل آخر للوضوح
تمامًا كما كان من قبل، سنتأكد من تنسيق مخرجاتنا بشكل جيد:
```csharp
// تنسيق مخرجات وحدة التحكم.
Console.WriteLine("----------------------");
Console.WriteLine("");
```

أخيرًا، لنختتم الموضوع. سنضيف رسالة بسيطة للإشارة إلى انتهاء برنامجنا بنجاح.
