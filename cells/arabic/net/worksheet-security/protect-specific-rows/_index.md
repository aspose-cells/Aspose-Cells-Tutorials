# حماية صفوف محددة في ورقة العمل باستخدام Aspose.Cells

## مقدمة
في هذا البرنامج التعليمي، سنرشدك خلال عملية حماية صفوف محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET. سنشرح كل خطوة بالتفصيل، مغطيين المتطلبات الأساسية، واستيراد الحزم المطلوبة، وشرح الكود في تعليمات سهلة المتابعة. في النهاية، ستكون متمكنًا من تطبيق حماية الصفوف في تطبيقاتك الخاصة.
## المتطلبات الأساسية
قبل الخوض في التنفيذ، هناك بعض المتطلبات الأساسية التي يجب عليك تلبيتها لمتابعة هذا البرنامج التعليمي:
1. Aspose.Cells لـ .NET: ستحتاج إلى تثبيت Aspose.Cells لـ .NET. إذا لم تقم بتثبيته بعد، يمكنك الحصول على أحدث إصدار بزيارة موقع Aspose الإلكتروني.
2. فهم أساسيات C# و.NET: يفترض هذا البرنامج التعليمي أنك مُلِمٌّ بلغة C# ولديك معرفة أساسية ببرمجة .NET. إذا لم تكن مُلِمًّا بها، فقد ترغب في الاطلاع على بعض الموارد التمهيدية أولًا.
3. Visual Studio أو أي بيئة تطوير متكاملة لـ .NET: ستحتاج إلى بيئة تطوير متكاملة (IDE) مثل Visual Studio لتشغيل الكود. توفر هذه البيئة جميع الأدوات اللازمة وإمكانيات تصحيح الأخطاء.
4. ترخيص Aspose.Cells: لتجنب قيود الإصدار التجريبي، تأكد من حصولك على ترخيص Aspose.Cells ساري المفعول. يمكنك أيضًا استخدام ترخيص مؤقت إذا كنت قد بدأت للتو.
للحصول على معلومات مفصلة حول Aspose.Cells والتثبيت، يمكنك التحقق من [التوثيق](https://reference.aspose.com/cells/net/).
## استيراد الحزم
لبدء استخدام Aspose.Cells، عليك استيراد مساحات الأسماء اللازمة في مشروع C#. تتيح لك هذه المساحات الوصول إلى الفئات والأساليب اللازمة لمعالجة ملفات Excel.
إليك كيفية استيراد مساحات الأسماء المطلوبة:
```csharp
using System.IO;
using Aspose.Cells;
```
تعتبر هذه الواردات بالغة الأهمية لأنها توفر الوصول إلى وظائف Aspose.Cells وتسمح لك بالتفاعل مع ملفات Excel في مشروع .NET الخاص بك.
بعد إعداد المتطلبات الأساسية واستيراد البيانات اللازمة، حان وقت البدء بالعمل على الكود. سنُقسّم العملية إلى عدة خطوات لضمان الوضوح.
## الخطوة 1: إعداد دليل المشروع الخاص بك
في أي برنامج، يُعد تنظيم ملفاتك أمرًا بالغ الأهمية. أولًا، لنُنشئ مجلدًا لتخزين المصنف. نتحقق من وجود المجلد وننشئه إذا لزم الأمر.
```csharp
// قم بتحديد المسار إلى دليل المستندات.
string dataDir = "Your Document Directory";
// إنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
هنا، يمكنك تحديد مسار حفظ ملفات Excel. إذا لم يكن المجلد موجودًا، فسيتم إنشاؤه. هذه الخطوة ضرورية لضمان وجود مكان لحفظ مصنفك.
## الخطوة 2: إنشاء مصنف جديد
بعد ذلك، نقوم بإنشاء مصنف جديد باستخدام `Workbook` توفر هذه الفئة جميع الوظائف المطلوبة للعمل مع ملفات Excel.
```csharp
// إنشاء مصنف جديد.
Workbook wb = new Workbook();
```
في هذه المرحلة، أصبح لدينا الآن كتاب عمل جديد للعمل عليه.
## الخطوة 3: الوصول إلى ورقة العمل
نصل الآن إلى ورقة العمل الأولى من المصنف المُنشأ حديثًا. يمكن أن يحتوي المصنف على عدة أوراق عمل، ولكن في هذه الحالة، سنركز على الورقة الأولى.
```csharp
// إنشاء كائن ورقة عمل والحصول على الورقة الأولى.
Worksheet sheet = wb.Worksheets[0];
```
هنا، `Worksheets[0]` يشير إلى ورقة العمل الأولى في المصنف (والتي يتم فهرستها بدءًا من 0).
## الخطوة 4: فتح جميع الأعمدة
في Excel، تكون الخلايا مقفلة افتراضيًا عند حماية ورقة العمل. لحماية صفوف محددة، يجب عليك أولًا إلغاء قفل الأعمدة. في هذه الخطوة، نمرر حلقة التكرار عبر جميع الأعمدة ونفتحها.
```csharp
// تعريف كائن النمط.
Style style;
// تعريف كائن styleflag.
StyleFlag flag;
// قم بالمرور على جميع الأعمدة في ورقة العمل وإلغاء قفلها.
for (int i = 0; i <= 255; i++)
{
    style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false;
    flag = new StyleFlag();
    flag.Locked = true;
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
}
```
هنا، ننتقل إلى الأعمدة من ٠ إلى ٢٥٥ (إجمالي عدد الأعمدة في ورقة عمل Excel) ونفتحها. هذا يضمن إمكانية التفاعل مع الصفوف التي نريد حمايتها، بينما تبقى الصفوف الأخرى مقفلة.
## الخطوة 5: قفل الصف الأول
بعد فتح جميع الأعمدة، يمكننا الانتقال إلى حماية الصفوف. في هذه الخطوة، نقفل الصف الأول، مما يجعله غير قابل للتعديل بعد حماية الورقة.
```csharp
// احصل على نمط الصف الأول.
style = sheet.Cells.Rows[0].Style;
// قفله.
style.IsLocked = true;
// إنشاء العلم.
flag = new StyleFlag();
// ضبط إعداد القفل.
flag.Locked = true;
// قم بتطبيق النمط على الصف الأول.
sheet.Cells.ApplyRowStyle(0, style, flag);
```
يقوم هذا الرمز بقفل الصف الأول، مما يضمن بقائه محميًا بمجرد تطبيق الحماية على الورقة.
## الخطوة 6: حماية ورقة العمل
في هذه المرحلة، أصبحنا جاهزين لحماية ورقة العمل. تُطبّق هذه الخطوة إعدادات الحماية على ورقة العمل بأكملها، مع التأكد من عدم إمكانية تعديل أي خلايا مقفلة.
```csharp
// حماية الورقة.
sheet.Protect(ProtectionType.All);
```
عن طريق استخدام `ProtectionType.All`نضمن حماية جميع الخلايا، باستثناء الخلايا المفتوحة صراحةً (مثل الأعمدة). هذه هي الخطوة التي تُطبّق الحماية على ورقة العمل.
## الخطوة 7: حفظ ملف Excel
أخيرًا، بعد تطبيق الحماية، نحفظ المصنف. يمكنك تحديد التنسيق الذي تريد حفظ الملف به. في هذا المثال، نحفظ المصنف كملف Excel 97-2003.
```csharp
// احفظ ملف الاكسل.
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```
تؤدي هذه الخطوة إلى حفظ الملف في المسار المحدد، مما يكمل مهمة حماية صفوف محددة في ورقة العمل.
## خاتمة
حماية صفوف محددة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET عملية سهلة وبسيطة بمجرد شرحها خطوة بخطوة. بإلغاء قفل الأعمدة، وقفل صفوف محددة، وتطبيق إعدادات الحماية، تضمن أمان بياناتك وإمكانية تعديلها فقط عند الضرورة. غطّى هذا البرنامج التعليمي جميع الخطوات الأساسية، من إعداد دليل المشروع إلى حفظ المصنف النهائي.
سواءً كنت تُنشئ قوالب أو تقارير أو جداول بيانات تفاعلية، فإن استخدام حماية الصفوف يُعدّ طريقة بسيطة وفعّالة للحفاظ على التحكم في بياناتك. جرّب هذه العملية في مشاريعك الخاصة واستكشف الإمكانات الكاملة لـ Aspose.Cells لـ .NET.
## الأسئلة الشائعة
### هل يمكنني حماية صفوف متعددة في ورقة العمل؟  
نعم، يمكنك تطبيق نفس خطوات الحماية على صفوف متعددة عن طريق تعديل الحلقة أو تطبيق الأنماط على صفوف أخرى.
### ماذا سيحدث إذا لم أقم بإلغاء قفل أي أعمدة قبل حماية الورقة؟  
إذا لم تقم بإلغاء قفل الأعمدة، فسيتم قفلها عند حماية الورقة، ولن يتمكن المستخدمون من التفاعل معها.
### كيف يمكنني فتح خلايا محددة بدلاً من الأعمدة بأكملها؟  
يمكنك فتح خلايا معينة عن طريق الوصول إلى نمطها وتعيين `IsLocked` الممتلكات إلى `false`.
### هل يمكنني استخدام هذه الطريقة لحماية أوراق العمل بأكملها؟  
نعم، يمكنك حماية ورقة العمل بأكملها عن طريق تطبيق الحماية على جميع الخلايا وعدم ترك أي خلايا غير مقفلة.
### كيف يمكنني إلغاء حماية ورقة العمل؟  
يمكنك إزالة الحماية عن طريق الاتصال بـ `Unprotect` الطريقة الموجودة في ورقة العمل وتوفير كلمة مرور الحماية (إذا تم تعيين واحدة).
