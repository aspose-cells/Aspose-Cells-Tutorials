## دليل التنفيذ
### تحديد أسماء الوظائف عند طباعة المصنفات
#### ملخص
يرشدك هذا القسم خلال عملية طباعة مصنف Excel بالكامل وتحديد اسم المهمة لتمييز مهمة الطباعة.

#### خطوات
**1. إنشاء كائن مصنف**
أولاً، قم بتحميل ملف Excel المصدر الخاص بك:
```csharp
// مسار دليل المصدر
string sourceDir = RunExamples.Get_SourceDirectory();

// تحميل المصنف من الملف
Workbook workbook = new Workbook(sourceDir + "sampleSpecifyJobWhilePrinting.xlsx");
```

**2. تكوين الطابعة واسم المهمة**
قم بتحديد اسم الطابعة وعنوان المهمة للتعريف:
```csharp
string printerName = "doPDF 8"; // التغيير إلى الطابعة المثبتة لديك
string jobName = "My Job Name";
```

**3. عرض وطباعة المصنف**
يستخدم `WorkbookRender` لإدارة الطباعة:
```csharp
// إعداد خيارات العرض (يمكن إضافة تكوينات اختيارية هنا)
ImageOrPrintOptions options = new ImageOrPrintOptions();

// تهيئة عرض المصنف باستخدام المصنف والخيارات
WorkbookRender wr = new WorkbookRender(workbook, options);

try
{
    // الطباعة باستخدام الطابعة المحددة واسم المهمة
    wr.ToPrinter(printerName, jobName);
}
catch (Exception ex)
{
    Console.WriteLine("Error during printing: " + ex.Message);
}
```
### طباعة أوراق عمل محددة
#### ملخص
إذا كنت بحاجة إلى طباعة ورقة عمل محددة باسم مهمة مخصصة، فاتبع الخطوات التالية.

**1. الوصول إلى ورقة العمل**
حدد ورقة العمل من المصنف الخاص بك:
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];
```

**2. ورقة عمل العرض والطباعة**
يستخدم `SheetRender` للطباعة المستهدفة:
```csharp
// قم بتهيئة SheetRender باستخدام ورقة العمل والخيارات المحددة
SheetRender sr = new SheetRender(worksheet, options);

try
{
    // تنفيذ الطباعة على الطابعة المحددة باسم المهمة
    sr.ToPrinter(printerName, jobName);
}
catch (Exception ex)
{
    Console.WriteLine("Worksheet print error: " + ex.Message);
}
```
