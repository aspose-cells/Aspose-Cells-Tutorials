## دليل التنفيذ
### ضبط اتجاه الصفحة
الآن، لنبدأ بتطبيق الوظيفة الرئيسية: ضبط اتجاه الصفحة. يرشدك هذا الدليل إلى كيفية تعديل اتجاه ورقة العمل باستخدام Aspose.Cells لـ .NET.

#### الخطوة 1: إنشاء كائن مصنف
ابدأ بإنشاء مثيل لـ `Workbook` فصل:

```csharp
// إنشاء كائن مصنف جديد
class Program
{
    static void Main()
    {
        var workbook = new Workbook();
        // بقية الكود...
    }
}
```

يقوم هذا السطر بإنشاء مصنف فارغ حيث يمكنك إضافة أوراق العمل ومعالجتها حسب الحاجة.

#### الخطوة 2: الوصول إلى ورقة العمل
قم بالوصول إلى ورقة العمل الأولى في المصنف لتعديل إعداداتها:

```csharp
// احصل على ورقة العمل الأولى من المصنف
var worksheet = workbook.Worksheets[0];
```

ال `Worksheets` تتيح لك المجموعة الوصول إلى كل ورقة داخل المصنف الخاص بك.

#### الخطوة 3: ضبط نوع الاتجاه
لتغيير اتجاه الصفحة، استخدم `PageSetup.Orientation` الخاصية. هذا المثال يضبطها على الوضع الرأسي:

```csharp
// ضبط اتجاه الصفحة إلى عمودي
worksheet.PageSetup.Orientation = PageOrientationType.Portrait;
```

يمكنك أيضًا ضبطه على الوضع الأفقي باستخدام `PageOrientationType.Landscape`.

#### الخطوة 4: حفظ المصنف الخاص بك
وأخيرًا، احفظ المصنف الخاص بك بالإعدادات الجديدة المطبقة:

```csharp
// تحديد المسار لحفظ الملف
string dataDir = "/your/directory/path/here/";

// حفظ المصنف المحدث
class Program
{
    static void Main()
    {
        var workbook = new Workbook();
        // كود آخر...
        workbook.Save(dataDir + "PageOrientation_out.xls");
    }
}
```

تكتب هذه الخطوة كافة التغييرات إلى موقع محدد على القرص الخاص بك.
