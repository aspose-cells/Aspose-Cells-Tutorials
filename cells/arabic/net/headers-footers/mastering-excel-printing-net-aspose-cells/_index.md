### دليل التنفيذ

ينقسم هذا الدليل إلى أقسام استنادًا إلى الميزات المختلفة لـ Aspose.Cells لـ .NET.

#### الميزة 1: تحميل مصنف Excel والوصول إليه

**ملخص**:تعرف على كيفية تحميل مصنف Excel من دليل محدد والوصول إلى ورقة العمل الأولى الخاصة به.

##### الخطوة 1: تعيين دليل المصدر
حدد المسار الذي يوجد به ملف Excel الخاص بك:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // التحديث بالمسار الفعلي
```

##### الخطوة 2: تحميل المصنف
استخدم Aspose.Cells لتحميل المصنف:
```csharp
// تحميل ملف Excel المصدر
Workbook workbook = new Workbook(SourceDir + "SheetRenderSample.xlsx");
```
*توضيح*:هذا يقوم بتهيئة `Workbook` الكائن الذي يسمح بالتفاعل مع ملف Excel.

##### الخطوة 3: الوصول إلى ورقة العمل الأولى
قم بالوصول إلى ورقة العمل المطلوبة باستخدام الفهرس الخاص بها:
```csharp
// الوصول إلى ورقة العمل الأولى في المصنف
Worksheet worksheet = workbook.Worksheets[1];
```

#### الميزة 2: تكوين خيارات الصورة أو الطباعة لعرض الورقة

**ملخص**:قم بتخصيص إعدادات العرض للتحكم في كيفية طباعة أوراق Excel الخاصة بك.

##### الخطوة 1: تهيئة ImageOrPrintOptions
إنشاء مثيل لـ `ImageOrPrintOptions` لتعيين تكوينات محددة:
```csharp
using Aspose.Cells.Rendering;

ImageOrPrintOptions imgOpt = new ImageOrPrintOptions();
```

##### الخطوة 2: تعيين خيارات التكوين
اختياريًا، يمكنك تكوين إعدادات مثل عرض ورقة كاملة على صفحة واحدة.
```csharp
// مثال على التكوين
imgOpt.OnePagePerSheet = true; // يعرض كل محتوى ورقة واحدة على صفحة صورة واحدة
```

#### الميزة 3: عرض ورقة العمل على الطابعة باستخدام إعدادات إضافية

**ملخص**:أرسل ورقة العمل مباشرة إلى الطابعة، مع تطبيق الإعدادات المخصصة.

##### الخطوة 1: تكوين إعدادات الطابعة
يثبت `PrinterSettings` لتحديد الطابعة وعدد النسخ:
```csharp
using System.Drawing.Printing;

PrinterSettings printerSettings = new PrinterSettings();
printerSettings.PrinterName = "<PRINTER NAME>"; // التحديث باستخدام اسم الطابعة الخاصة بك
printerSettings.Copies = 2; // تعيين عدد النسخ المطلوبة
```

##### الخطوة 2: الإرسال إلى الطابعة
يستخدم `SheetRender` لإرسال ورقة العمل إلى الطابعة المحددة:
```csharp
SheetRender sheetRender = new SheetRender(worksheet, imgOpt);
sheetRender.ToPrinter(printerSettings); // طباعة ورقة العمل بالإعدادات المحددة
```
*توضيح*: ال `ToPrinter` ترسل الطريقة الورقة إلى الطابعة باستخدام الإعدادات المحددة.
