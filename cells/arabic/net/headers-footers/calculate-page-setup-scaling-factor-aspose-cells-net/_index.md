## دليل التنفيذ

### حساب عامل مقياس إعداد الصفحة

تساعدك هذه الميزة على تحديد مدى ملاءمة محتوى ورقة العمل للصفحة عند الطباعة.

#### الخطوة 1: الوصول إلى خصائص ورقة العمل وتعديلها

أولاً، قم بالوصول إلى ورقة العمل المطلوبة وإجراء التعديلات اللازمة:
```csharp
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.Worksheets[0];

// ضع بعض البيانات في خلايا محددة للتوضيح
worksheet.Cells["A4"].PutValue("Test");
worksheet.Cells["S4"].PutValue("Test");

// ضبط حجم الورق إلى A4
worksheet.PageSetup.PaperSize = PaperSizeType.PaperA4;

// قم بتكوين ورقة العمل لتناسب المحتوى على صفحة واحدة
worksheet.PageSetup.FitToPagesWide = 1;
```

#### الخطوة 2: إنشاء كائن SheetRender

استخدم `SheetRender` الفئة للتعامل مع إعدادات العرض:
```csharp
// تهيئة SheetRender باستخدام خيارات الطباعة الافتراضية
SheetRender sr = new SheetRender(worksheet, new ImageOrPrintOptions());
```

#### الخطوة 3: حساب وعرض عامل القياس

تحويل عامل القياس من قيمة مزدوجة إلى تنسيق النسبة المئوية لتسهيل التفسير:
```csharp
// تحويل مقياس الصفحة إلى سلسلة نسبية قابلة للقراءة
string strPageScale = sr.PageScale.ToString("0%");
Console.WriteLine($"Scaling Factor: {strPageScale}");
```

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من جميع المسارات (`SourceDir`، `outputDir`) تم ضبطها بشكل صحيح.
- إذا لم يكن القياس كما هو متوقع، فتأكد من ذلك مرة أخرى `FitToPagesWide` وتكوينات إعداد الصفحة الأخرى.
