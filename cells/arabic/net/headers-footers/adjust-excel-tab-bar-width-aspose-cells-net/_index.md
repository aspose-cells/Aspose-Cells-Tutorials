## دليل التنفيذ

### نظرة عامة: ضبط عرض شريط علامة تبويب الورقة

يُحسّن تخصيص عرض علامات تبويب الورقة في ملف Excel التنقل ويضمن رؤية كاملة لأسماء علامات التبويب. هذه الميزة مفيدة بشكل خاص للوحات المعلومات والتقارير والقوالب المشتركة.

#### الخطوة 1: تحميل ملف Excel الخاص بك

ابدأ بتحميل مصنف Excel حيث تريد ضبط عرض شريط علامات التبويب.

```csharp
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

*ملحوظة:* `RunExamples.GetDataDir` هي طريقة مساعدة لتحديد مسار مجلدك. عدّل هذا المسار حسب مكان تخزين ملفاتك.

#### الخطوة 2: تكوين إعدادات علامة تبويب الورقة

قم بتعيين رؤية علامات التبويب وتعديل عرضها حسب الحاجة.

```csharp
// تمكين عرض علامات التبويب
workbook.Settings.ShowTabs = true;

// تعيين عرض شريط علامة تبويب الورقة (بالبكسل)
workbook.Settings.SheetTabBarWidth = 800;
```

*توضيح:*
- `ShowTabs`:يحدد ما إذا كانت علامات التبويب مرئية أم لا.
- `SheetTabBarWidth`يُحدد عرض بكسل شريط علامات التبويب. اضبط هذه القيمة وفقًا لمتطلبات تخطيطك.

#### الخطوة 3: حفظ التغييرات

بعد إجراء التعديلات، احفظ المصنف للحفاظ على التغييرات.

```csharp
workbook.Save(dataDir + "output.xls");
```

### نصائح استكشاف الأخطاء وإصلاحها:

- تأكد من أن لديك أذونات الكتابة للدليل الذي تريد حفظ الملف فيه.
- إذا واجهت أخطاء أثناء تحميل الملفات، فتأكد من توافق المسار وتنسيق الملف (على سبيل المثال، `.xls` مقابل `.xlsx`).
