# كيفية تعيين حجم ورق مخصص في Aspose.Cells .NET لعرض ملفات PDF

## دليل التنفيذ
### تعيين حجم ورق مخصص
مع Aspose.Cells، يمكنك تخصيص حجم ورقة عملك بسهولة. يوضح هذا القسم كيفية تطبيق هذه الميزة في تطبيق .NET.

#### تهيئة مشروعك
ابدأ بإنشاء مثيل لـ `Workbook` الصف والوصول إلى ورقة العمل الأولى الخاصة به:
```csharp
using System;
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// إنشاء كائن مصنف
Workbook wb = new Workbook();

// الوصول إلى ورقة العمل الأولى
Worksheet ws = wb.Worksheets[0];
```

#### تكوين حجم الورق المخصص
لتعيين حجم ورق مخصص، استخدم `PageSetup.CustomPaperSize` الطريقة. إليك كيفية تحديد الأبعاد بالبوصة:
```csharp
// تعيين حجم الورق المخصص (6 بوصات × 4 بوصات)
ws.PageSetup.CustomPaperSize(6, 4);
```
تُعد هذه الميزة مفيدة بشكل خاص لتخصيص المستندات لتناسب تنسيقات الطباعة غير التقليدية.

#### ملء ورقة العمل وحفظها
أضف المحتوى إلى ورقة العمل الخاصة بك واحفظه بتنسيق PDF:
```csharp
// الوصول إلى الخلية B4 في ورقة العمل
Cell b4 = ws.Cells["B4"];

// أضف رسالة إلى الخلية B4 تشير إلى أبعاد صفحة PDF
b4.PutValue("Pdf Page Dimensions: 6.00 x 4.00 in");

// احفظ المصنف كملف PDF مع تحديد حجم الورق المخصص
wb.Save(outputDir + "outputCustomPaperSize.pdf");
```
