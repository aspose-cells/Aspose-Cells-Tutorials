## دليل التنفيذ

دعنا نقسم العملية إلى خطوات يمكن إدارتها، مع التركيز على تحميل مصنف، والوصول إلى الخلايا، وتطبيق الصيغ المشتركة، وحفظ التغييرات.

### تحميل مصنف Excel
**ملخص:**
تحميل ملف Excel موجود في `Workbook` يُعدّ إنشاء الكائن الخطوة الأولى في أتمتة Excel باستخدام Aspose.Cells. يتيح لك هذا معالجة البيانات برمجيًا.

#### التنفيذ خطوة بخطوة:
**استيراد الحزم المطلوبة:**
```java
import com.aspose.cells.Workbook;
```

**تحميل المصنف من الملف:**
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // قم بتعيين مسار دليل الملف الخاص بك هنا
String filePath = dataDir + "/input.xlsx";
Workbook workbook = new Workbook(filePath);
```

### الوصول إلى مجموعة الخلايا وتطبيق الصيغة المشتركة
**ملخص:**
يتيح لك الوصول إلى مجموعات الخلايا تطبيق الصيغ المشتركة عبر خلايا متعددة بكفاءة.

#### التنفيذ خطوة بخطوة:
**استيراد الحزم المطلوبة:**
```java
import com.aspose.cells.Cells;
```

**الحصول على مجموعة الخلايا وتعيين الصيغة المشتركة:**
```java
Cells cells = workbook.getWorksheets().get(0).getCells();
cells.get("B2").setSharedFormula("=A2*0.09", 13, 1);
```
- **المعلمات موضحة:** 
  - `setSharedFormula(String formula, int rowCount, int columnCount)`:يتم تطبيق الصيغة المشتركة على النطاق المحدد.
  - `formula` هو تعبير Excel الذي تريد تطبيقه.
  - `rowCount` و `columnCount` تحديد الحدود الرأسية والأفقية للصيغة.

### حفظ المصنف في ملف إخراج
**ملخص:**
بعد إجراء التعديلات، يؤدي حفظ المصنف إلى ضمان الحفاظ على التغييرات في ملف جديد.

#### التنفيذ خطوة بخطوة:
**استيراد الحزم المطلوبة:**
```java
import com.aspose.cells.SaveFormat;
```

**حفظ المصنف المعدل:**
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/SSharedFormula_out.xlsx", SaveFormat.XLSX);
```
