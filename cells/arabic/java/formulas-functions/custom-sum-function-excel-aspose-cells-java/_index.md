## دليل التنفيذ

### الميزة: محرك الحسابات المخصص
تتيح لك هذه الميزة تعديل كيفية عمل Excel مثل `SUM` تعمل داخل Aspose.Cells.

#### ملخص
من خلال توسيع محرك الحساب، يمكنك تخصيص سلوك وظائف محددة. يركز هذا البرنامج التعليمي على تعديل `SUM` وظيفة لإضافة قيمة ثابتة إضافية.

#### التنفيذ خطوة بخطوة
##### توسيع محرك الحسابات المجردة
1. **إنشاء فئة CustomEngine**
   ابدأ بإنشاء فئة تمتد `AbstractCalculationEngine`.
   
   ```java
   import com.aspose.cells.AbstractCalculationEngine;
   import com.aspose.cells.CalculationData;

   public class CustomEngine extends AbstractCalculationEngine {
       @Override
       public void calculate(CalculationData data) {
           // تحقق مما إذا كانت الدالة التي يتم حسابها هي 'SUM'.
           if (data.getFunctionName().toUpperCase().equals("SUM")) {
               // استرجاع القيمة المحسوبة الحالية وتعديلها.
               double val = (double) data.getCalculatedValue();
               val += 30;  // إضافة قيمة ثابتة 30
               data.setCalculatedValue(val);
           }
       }
   }
   ```
2. **شرح المعلمات**
   - `data.getFunctionName()`:استرجاع اسم الدالة التي يتم حسابها.
   - `data.getCalculatedValue()`: يقوم بجلب النتيجة المحسوبة الحالية.
   - `data.setCalculatedValue(double)`:تحديث بيانات الحساب بقيمة جديدة.
3. **نصائح استكشاف الأخطاء وإصلاحها**
   تأكد من أن أسماء الطرق والمنطق للتحقق من الوظائف لا تميز بين الأحرف الكبيرة والصغيرة لمنع حدوث أي أخطاء أثناء التنفيذ.
