## إعداد Aspose.Cells لـ Java

يمكنك دمج Aspose.Cells في مشروع Java الخاص بك بسهولة باستخدام Maven أو Gradle. إليك كيفية إعداده:

**مافن:**
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml

```

**جرادل:**


### التهيئة والإعداد الأساسي
بمجرد إضافة Aspose.Cells إلى مشروعك، قم بتهيئته في تطبيق Java الخاص بك على النحو التالي:
```java
// استيراد الفئات الضرورية من Aspose.Cells
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class ShowFormulas {
    public static void main(String[] args) throws Exception {
        // قم بتحديد المسار الذي توجد به ملفات Excel الخاصة بك
        String dataDir = "path/to/your/excel/files/";

        // تحميل مصنف موجود من القرص
        Workbook workbook = new Workbook(dataDir + "source.xlsx");
        
        // الوصول إلى ورقة العمل الأولى في المصنف
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // إظهار الصيغ داخل ورقة العمل هذه
        worksheet.setShowFormulas(true);
        
        // احفظ التغييرات مرة أخرى في ملف
        workbook.save(dataDir + "ShowFormulas_out.xlsx");
    }
}
```
