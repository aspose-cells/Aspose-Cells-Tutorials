## دليل التنفيذ

### إنشاء مصنف جديد

تمثل الفئة `Workbook` ملف Excel كامل في الذاكرة وتعد نقطة الدخول لجميع العمليات.

```java
import com.aspose.cells.Workbook;
// Create an empty workbook
Workbook workbook = new Workbook();
```

### الوصول إلى الورقة الأولى

توفر الفئة `Worksheet` إمكانية الوصول إلى ورقة واحدة داخل المصنف حيث يمكنك إضافة بيانات أو مخططات أو أشكال.

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Worksheets;
// Get the first worksheet from the workbook
Worksheet sheet = workbook.getWorksheets().get(0);
```

### إضافة شكل تسمية

تتيح لك الفئة `Shape` (وبشكل خاص شكل التسمية) إدراج رسومات نصية تطفو فوق الخلايا.

```java
import com.aspose.cells.Label;
import com.aspose.cells.MsoDrawingType;
// Add a label shape to the worksheet
Label label = (Label) sheet.getShapes().addShape(MsoDrawingType.LABEL, 2, 2, 2, 0, 60, 120);
```

### تعيين نص التسمية

تحدد طريقة `setText` لكائن `Label` السلسلة المرئية داخل الشكل.

```java
// Set text for the label
label.setText("This is a Label");
```

### تكوين نوع وضع التسمية

تتحكم تعداد `PlacementType` في سلوك التسمية عند إدراج أو حذف الصفوف أو الأعمدة.

```java
import com.aspose.cells.PlacementType;
// Configure label placement
label.setPlacement(PlacementType.FREE_FLOATING);
```

### تعيين لون التعبئة مع التدرج

تمكن الفئة `FillFormat` من تعبئة التدرج، مما يسمح لك بتطبيق انتقال لون سلس لخلفية التسمية.

```java
import com.aspose.cells.Color;
import com.aspose.cells.GradientStyleType;
// Set one-color gradient as fill for the label
label.getFill().setOneColorGradient(Color.getYellow(), 1, GradientStyleType.HORIZONTAL, 1);
```

### حفظ المصنف

تكتب طريقة `save` المصنف الموجود في الذاكرة إلى ملف فعلي. يمكنك اختيار صيغ الإخراج XLS أو XLSX أو CSV أو PDF.

```java
// Define output directory and save the workbook
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/AddingLabelControl_out.xls");
```

## المشكلات الشائعة والحلول

| المشكلة | الحل |
|-------|----------|
| **OutOfMemoryError** when processing large files | استخدم `Workbook.setMemorySetting(MemorySetting.MEMORY_PREFERENCE)` وعالج البيانات على دفعات. |
| **License not applied** | تحقق من مسار ملف الترخيص وتأكد من استدعاء `license.setLicense()` قبل أي عمليات على المصنف. |
| **Shape not appearing** | تأكد من أن إحداثيات الشكل وأبعاده ضمن نطاق الورقة المرئي. |
