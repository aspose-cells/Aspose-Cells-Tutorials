## دليل التنفيذ

### إنشاء عنصر تحكم ActiveX وإضافته إلى ورقة عمل

#### ملخص
تتضمن إضافة عنصر تحكم ActiveX، مثل زر التبديل، إنشائه ضمن مجموعة أشكال ورقة العمل. يرشدك هذا القسم خلال هذه العملية.

#### دليل خطوة بخطوة
**1. إنشاء مصنف والوصول إلى ورقة العمل الأولى**
قم بتهيئة المصنف الخاص بك والوصول إلى ورقة العمل الأولى الخاصة به:

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

// تهيئة المصنف
Workbook wb = new Workbook();

// احصل على ورقة العمل الأولى
Worksheet sheet = wb.getWorksheets().get(0);
```

**2. إضافة زر تبديل عنصر التحكم ActiveX**
أضف زر تبديل إلى ورقة العمل الخاصة بك:

```java
import com.aspose.cells.ControlType;
import com.aspose.cells.Shape;

// إضافة زر تبديل داخل مجموعة الأشكال في الموقع والحجم المحددين
Shape s = sheet.getShapes().addActiveXControl(
    ControlType.TOGGLE_BUTTON, 4, 0, 4, 0, 100, 30);
```

**3. تكوين عنصر تحكم ActiveX**
تعيين خصائص مثل ربط الخلايا لتعزيز التفاعل:

```java
import com.aspose.cells.ActiveXControl;

// الوصول إلى كائن عنصر التحكم ActiveX
ActiveXControl c = s.getActiveXControl();

// ربط عنصر التحكم بخلية
c.setLinkedCell("A1");
```

**4. احفظ المصنف**
احفظ المصنف الخاص بك بالتنسيق المطلوب:

```java
import com.aspose.cells.SaveFormat;

// تحديد دليل الإخراج
String dataDir = "path/to/your/directory/";

// حفظ المصنف كملف Excel
wb.save(dataDir + "AAXControl_out.xlsx", SaveFormat.XLSX);
```
