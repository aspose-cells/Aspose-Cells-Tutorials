### دليل التنفيذ

#### العثور على الخلايا باستخدام التعبيرات العادية

يرشدك هذا القسم خلال تنفيذ عمليات البحث عن الخلايا المستندة إلى regex في ورقة Excel.

**الخطوة 1: الوصول إلى ورقة العمل**

احصل على مرجع إلى ورقة العمل التي تريد إجراء البحث فيها:

```java
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**الخطوة 2: إعداد خيارات البحث**

قم بتكوين `FindOptions` لاستخدام التعبيرات العادية للبحث المبني على النمط:

```java
// إنشاء FindOptions وتعيين RegexKey إلى true
FindOptions opt = new FindOptions();
opt.setRegexKey(true); // تمكين البحث عن التعبيرات العادية
opt.setLookAtType(LookAtType.ENTIRE_CONTENT);
```

**الخطوة 3: إجراء البحث**

استخدم `find` الطريقة مع نمط التعابير العادية الخاص بك:

```java
// العثور على الخلايا المطابقة لـ regex المحدد
Cells cells = worksheet.getCells();
cells.find("abc[\\s]*$", null, opt); // البحث عن "abc" متبوعًا بأي أحرف مسافة بيضاء في نهاية محتوى الخلية
```

**توضيح:**
- `setRegexKey(true)`:تنشيط البحث عن التعبيرات العادية.
- `LookAtType.ENTIRE_CONTENT`:يضمن أن يتم أخذ محتوى الخلية بأكمله في الاعتبار أثناء البحث، وليس أجزاء منه فقط.
