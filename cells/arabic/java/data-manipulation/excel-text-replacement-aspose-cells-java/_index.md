## دليل التنفيذ
يقوم هذا القسم بتحليل عملية تنفيذ ميزة البحث والاستبدال ضمن نطاق محدد في ملف Excel.
### نظرة عامة على الميزات
الهدف هو تحديد موقع النص واستبداله بكفاءة فقط ضمن منطقة خلية محددة، مما يقلل من المعالجة غير الضرورية لمجموعات البيانات الكبيرة.
#### الخطوة 1: تحديد نطاق الخلايا
حدد النطاق المحدد الذي تريد أن تحدث فيه العملية:
```java
CellArea area = CellArea.createCellArea("E3", "H6"); // على سبيل المثال، من الخلية E3 إلى H6
```
#### الخطوة 2: تكوين خيارات البحث
قم بإعداد `FindOptions` لكيفية إجراء البحث:
```java
FindOptions opts = new FindOptions();
opts.setLookInType(LookInType.VALUES); // البحث داخل قيم الخلايا فقط
opts.setLookAtType(LookAtType.ENTIRE_CONTENT); // مطابقة المحتوى الكامل للخلايا
opts.setRange(area); // تحديد البحث في المنطقة المحددة
```
#### الخطوة 3: قم بإجراء البحث والاستبدال
تنفيذ حلقة للعثور على جميع الحالات واستبدالها:
```java
Cell cell = null;
do {
    cell = worksheet.getCells().find("search", cell, opts);
    if (cell == null) break;
    cell.putValue("replace"); // استبدال النص الموجود بـ "استبدال"
} while (true);
workbook.save("SRDataInRange_out.xlsx");
```
### خيارات تكوين المفاتيح
- **نوع البحث:** يقتصر البحث على القيم فقط.
- **نوع النظرة:** يتأكد من أن المطابقات دقيقة وليست جزئية.
#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من صحة بناء جملة نطاق الخلايا (`"startCell:endCell"`).
- التحقق من صحة ذلك `search` السلسلة موجودة في النطاق المحدد.
- التحقق من أذونات الملفات لقراءة/كتابة ملفات Excel.
