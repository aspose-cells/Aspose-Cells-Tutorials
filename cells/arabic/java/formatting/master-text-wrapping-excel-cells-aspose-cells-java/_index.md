## دليل التنفيذ

### إنشاء مصنف والوصول إلى الخلايا

أولاً، قم بإنشاء مثيل مصنف والوصول إلى خلاياه:

```java
// إنشاء كائن مصنف جديد
document = new Workbook();

// افتح ورقة العمل الأولى في المصنف
worksheet = document.getWorksheets().get(0);

// احصل على مجموعة الخلايا من ورقة العمل
cells = worksheet.getCells();
```

### تكوين عرض العمود وارتفاع الصف

اضبط عرض العمود وارتفاع الصف للتأكد من أن النص يناسب بشكل أنيق:

```java
// زيادة عرض العمود الأول
cells.setColumnWidth(0, 35);

// زيادة ارتفاع الصف الأول
cells.setRowHeight(0, 65);
```

### إضافة نص وتطبيق نمط الالتفاف

إضافة نص إلى خلية وتمكين التفاف النص:

```java
// إضافة نص إلى الخلية الأولى
cells.get(0, 0).setValue("I am using the latest version of Aspose.Cells to test this functionality");

// احصل على نمط الخلية
Style style = cells.get(0, 0).getStyle();

// تمكين التفاف النص لمحتوى الخلية
style.setTextWrapped(true);

// تطبيق النمط مرة أخرى على الخلية
cells.get(0, 0).setStyle(style);
```

### حفظ مصنفك

احفظ المصنف الخاص بك مع النص الملتف:

```java
// حفظ ملف Excel
document.save("WrapTextinCell_out.xls");
```
