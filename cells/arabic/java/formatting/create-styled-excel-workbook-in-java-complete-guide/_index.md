# إنشاء دفتر عمل Excel مُنسق في Java – دليل كامل

هل احتجت يوماً إلى **create styled excel workbook** من Java لكن شعرت بأن تفاصيل التنسيق تثقل كاهلك؟ لست وحدك. سواء كنت تستخرج البيانات من قاعدة بيانات أو تصدر تقريرًا، الحصول على المظهر الصحيح—رموز العملة، أقنعة التاريخ، رؤوس بالخط العريض—يمكن أن يشعرك وكأنه عبء.

في هذا الدرس سنستعرض حلًا عمليًا يضيف **adds custom formatting to excel**، ويسمح لك بـ **set date format excel column**، وأخيرًا **save workbook as xlsx** دون عناء. هل أنت مستعد لرؤيته عمليًا؟

## ما ستتعلمه

- كيفية إنشاء Aspose.Cells `Workbook` من الصفر.  
- طرق تطبيق مصفوفة أنماط تغطي الأعمدة العامة، والعملات، والتواريخ.  
- النداء الدقيق الذي **creates styled excel workbook** مع أسماء الأعمدة مدمجة.  
- كيفية حفظ الملف باستخدام طريقة **save workbook as xlsx**.  

لا ملفات تكوين خارجية، لا حيل XML غامضة—فقط كود Java بسيط يمكنك نسخه‑ولصقه، تشغيله، وتعديله.

### المتطلبات المسبقة

- Java 8+ (الكود يعمل على أي JDK حديث).  
- مكتبة Aspose.Cells للـ Java (يمكنك الحصول على أحدث JAR من Maven Central).  
- قليل من الإلمام بالمجموعات (`List`، `Map`).  

إذا كان أي من ذلك غير مألوف لك، توقف لحظة وقم بتثبيت Aspose.Cells JAR؛ باقي الأمور ستسير طبيعيًا.

## إنشاء دفتر عمل Excel مُنسق – إعداد المشروع

أولاً: أنشئ مشروع Maven (أو Gradle) جديد وأضف Aspose.Cells كاعتماد.

```xml
<!-- pom.xml snippet -->
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>23.12</version> <!-- use the latest version -->
</dependency>
```

> **نصيحة احترافية:** إذا كنت تستخدم Gradle، فإن نفس الإحداثيات تعمل مع `implementation 'com.aspose:aspose-cells:23.12'`.

لماذا نبدأ بإعداد Maven؟ فهو يجلب جميع الاعتمادات المتداخلة تلقائيًا، لذا لن تضيع وقتًا في البحث عن JARs المناسبة لاحقًا.

## إضافة تنسيق مخصص إلى أعمدة Excel

الآن بعد أن أصبح المشروع جاهزًا، دعنا نحدد الأنماط التي نريدها. في Aspose.Cells يمكنك تمرير مصفوفة من سلاسل التنسيق التي ترتبط مباشرة بكل عمود.

```java
// Define column styles: General, Currency, Date
String[] columnStyles = {
    "General",          // Column A – plain text or numbers
    "$#,##0.00",        // Column B – currency with two decimals
    "mm/dd/yyyy"        // Column C – US‑style date
};
```

خطوة **add custom formatting to excel** بسيطة بقدر بناء هذه المصفوفة النصية.  
كل إدخال يتبع نفس النمط الذي تستخدمه في مربع حوار “Custom” في Excel، لذا لن تحتاج لتعلم صياغة جديدة.

## تعيين تنسيق التاريخ لعمود Excel

إذا كانت بياناتك تتضمن تواريخ، فستحتاج إلى التأكد من ظهورها بشكل ثابت. الإدخال الثالث في `columnStyles` (`"mm/dd/yyyy"`) يفعل ذلك تمامًا—يخبر Aspose.Cells بأن يعامل العمود الثالث كتاريخ ويعرضه بترتيب الشهر/اليوم/السنة.

```java
// Example: Adding a row with a Date object
Map<String, Object> row = new HashMap<>();
row.put("Product", "Widget");
row.put("Price", 19.99);
row.put("ReleaseDate", new java.util.Date()); // Today’s date
dataRows.add(row);
```

عند إنشاء دفتر العمل، يضمن توجيه **set date format excel column** أن كل خلية `ReleaseDate` تظهر كـ `04/27/2026` (أو أي تاريخ اليوم). لا حاجة لمعالجة لاحقة.

## استيراد البيانات وتطبيق الأنماط

مع تعريف الأنماط، الخطوة التالية هي إمداد ورقة العمل ببياناتنا. توفر Aspose.Cells الدالة `importCustomObjects`، التي لا تُدرج البيانات فحسب، بل تُطبق أيضًا مصفوفة الأنماط التي أنشأناها.

```java
// Step 1: Simulate fetching rows from a DB
List<Map<String, Object>> dataRows = fetchRows(); // see helper below

// Step 2: Create a fresh workbook and grab the first sheet
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);

// Step 3: Import data with custom styles; true = first row has headers
worksheet.getCells().importCustomObjects(
        dataRows,        // data source
        0,               // start row (0‑based)
        0,               // start column (0‑based)
        columnStyles,    // style array from above
        true);           // first row contains column names
```

### لماذا `importCustomObjects`؟

- **سحر الاستدعاء الواحد:** يكتب البيانات *ويطبق* التنسيق في خطوة واحدة.  
- **الوعي بالرؤوس:** تمرير `true` يخبر الـ API بأن يعامل مفاتيح الخريطة الأولى كعناوين أعمدة، وهو مثالي لسيناريوهات **create styled excel workbook** حيث تريد عناوين أعمدة قابلة للقراءة.  
- **الأداء:** تحت الغطاء، تقوم Aspose بتجميع عمليات الكتابة، مما يجعلها أسرع من التكرار خلية بخلية.

> **احذر:** ترتيب المفاتيح في كل `Map` مهم. استخدم `LinkedHashMap` إذا كنت تحتاج إلى ترتيب أعمدة محدد.

## حفظ دفتر العمل كملف XLSX

كل هذا العمل سيكون بلا فائدة إذا لم تتمكن من حفظ الملف. السطر الأخير هو سطر واحد يحقق **save workbook as xlsx**.

```java
// Step 5: Persist the workbook
String outputPath = "output/DataTableStyled.xlsx";
workbook.save(outputPath);
System.out.println("Workbook saved to " + outputPath);
```

طريقة `save` تكتشف امتداد الملف تلقائيًا. بتحديد `.xlsx`، تضمن تنسيق Office Open XML الحديث، وهو ما تتوقعه معظم الأدوات اللاحقة.

## تشغيل والتحقق من النتيجة

قم بترجمة وتشغيل الفئة `ImportDataTableWithStyle`. بعد التنفيذ يجب أن ترى ملفًا في `output/DataTableStyled.xlsx`. افتحه في Excel (أو LibreOffice) وستلاحظ:

- العمود **A** يظهر نصًا عاديًا (`Product`).  
- العمود **B** يعرض أرقامًا مع علامة الدولار ومكانين عشريين (`$19.99`).  
- العمود **C** يقدم تواريخ بصيغة `mm/dd/yyyy` (`04/27/2026`).

## مثال كامل قابل للتنفيذ

فيما يلي الفئة الكاملة التي يمكنك نسخها مباشرة إلى بيئة التطوير المتكاملة الخاصة بك. تتضمن طريقة `fetchRows` وهمية تُعيد بيانات عينة، لذا لا تحتاج إلى قاعدة بيانات حقيقية للاختبار.

```java
import com.aspose.cells.*;
import java.util.*;

public class ImportDataTableWithStyle {
    public static void main(String[] args) throws Exception {
        // ---------- Step 1: Get data ----------
        List<Map<String, Object>> dataRows = fetchRows();

        // ---------- Step 2: Create workbook ----------
        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // ---------- Step 3: Define column styles ----------
        String[] columnStyles = { "General", "$#,##0.00", "mm/dd/yyyy" };

        // ---------- Step 4: Import data with styles ----------
        // true = first row contains column names
        worksheet.getCells().importCustomObjects(dataRows, 0, 0, columnStyles, true);

        // ---------- Step 5: Save as XLSX ----------
        String outPath = "output/DataTableStyled.xlsx";
        workbook.save(outPath);
        System.out.println("Workbook saved to " + outPath);
    }

    // Mock method – replace with real DB call if needed
    private static List<Map<String, Object>> fetchRows() {
        List<Map<String, Object>> rows = new ArrayList<>();

        Map<String, Object> row1 = new LinkedHashMap<>();
        row1.put("Product", "Gadget");
        row1.put("Price", 49.95);
        row1.put("ReleaseDate", new Date()); // today
        rows.add(row1);

        Map<String, Object> row2 = new LinkedHashMap<>();
        row2.put("Product", "Widget");
        row2.put("Price", 19.99);
        Calendar cal = Calendar.getInstance();
        cal.set(2025, Calendar.DECEMBER, 15);
        row2.put("ReleaseDate", cal.getTime());
        rows.add(row2);

        return rows;
    }
}
```

شغّلها، افتح الملف المُولد، وسترى نتيجة **create styled excel workbook** التي توقعتها.

## أسئلة شائعة وحالات خاصة

**ماذا لو احتجت إلى أكثر من ثلاثة أعمدة؟**
