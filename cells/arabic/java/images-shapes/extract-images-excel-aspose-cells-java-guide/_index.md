## إعداد Aspose.Cells لـ Java

لاستخدام Aspose.Cells في مشروع Java، قم بتضمينه كتبعيه:

**مافن**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**جرادل**
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## دليل التنفيذ

### استخراج الصور من ورقة عمل Excel
قم باستخراج الصور المضمنة وحفظها برمجيًا باستخدام الخطوات التالية:

#### الخطوة 1: الوصول إلى ورقة العمل
قم بتحميل المصنف الخاص بك والوصول إلى ورقة العمل المطلوبة.

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### الخطوة 2: استرداد كائن الصورة
حدد الصورة التي تريد استخراجها واسترجعها. هنا، نركز على الصورة الأولى في الورقة:

```java
Picture pic = worksheet.getPictures().get(0);
```

#### الخطوة 3: تكوين خيارات إخراج الصورة
قم بتحديد الطريقة التي تريد بها حفظ الصور المستخرجة باستخدام `ImageOrPrintOptions`.

```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // استبدل بمسار دليل الإخراج الفعلي الخاص بك
ImageOrPrintOptions printoption = new ImageOrPrintOptions();
printoption.setImageType(ImageType.JPEG);
```

#### الخطوة 4: حفظ الصورة المستخرجة
احفظ الصورة المستخرجة في الدليل المحدد.

```java
String fileName = "aspose-logo.jpg";
pic.toImage(outDir + "/" + fileName, printoption);
```
