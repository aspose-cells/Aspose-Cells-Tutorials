## دليل التنفيذ

### ملخص
في هذا القسم، سنوضح كيفية تعيين صورة خلفية لأول ورقة عمل في ملف Excel باستخدام Aspose.Cells.

#### الخطوة 1: تحديد مسارات الدليل
أولاً، قم بتحديد المكان الذي سيتم فيه تخزين صور الإدخال وملفات الإخراج:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; 
String outDir = "YOUR_OUTPUT_DIRECTORY";
```
تعتبر هذه المسارات ضرورية لتحديد موقع ملف الصورة وحفظ المصنف المعدل.

#### الخطوة 2: تحميل ملف صورة كبيانات بايت
بعد ذلك، حمّل صورة الخلفية إلى مصفوفة بايتات. تتضمن هذه الخطوة قراءة بيانات الصورة من ملف:
```java
String imagePath = dataDir + "background.png";
java.io.File file = new java.io.File(imagePath);
byte[] imageData = new byte[(int) file.length()];
try (java.io.FileInputStream fis = new java.io.FileInputStream(file)) {
    fis.read(imageData); // قم بتحميل الصورة إلى مجموعة بايتات.
}
```

#### الخطوة 3: تعيين صورة الخلفية لورقة العمل
الآن، قم بتطبيق الصورة المحملة كخلفية لورقة العمل الخاصة بك:
```java
dsheet.setBackgroundImage(imageData);
```
تقوم هذه الطريقة بتعيين بيانات الصورة لخلفية ورقة العمل.

#### الخطوة 4: حفظ المصنف
أخيرًا، احفظ المصنف الخاص بك بالإعدادات المحدثة في دليل الإخراج:
```java
workbook.save(outDir + "SBPforWorksheet.xlsx");
```
