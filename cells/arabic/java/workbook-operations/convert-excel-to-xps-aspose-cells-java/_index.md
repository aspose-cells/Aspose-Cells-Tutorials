# تحويل Excel إلى تنسيق XPS باستخدام Aspose.Cells لـ Java: دليل خطوة بخطوة

- تحميل ملف Excel إلى `Workbook` هدف

- عرض أوراق العمل الفردية أو المصنفات الكاملة بتنسيق XPS

## دليل التنفيذ

### تحميل ملف Excel

#### ملخص
الخطوة الأولى هي تحميل ملف Excel الخاص بك إلى `Workbook` الكائن الذي يعمل كنقطة دخول للوصول إلى بيانات Excel ومعالجتها.

**مقتطف من الكود**
```java
import com.aspose.cells.*;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "Book1.xls");
```
*توضيح:* يستبدل `"YOUR_DATA_DIRECTORY"` مع مسار دليل الملف الخاص بك. `Workbook` تعتبر الفئة أساسية للتفاعل مع ملفات Excel في Aspose.Cells.

### الوصول إلى أوراق العمل

#### ملخص
بمجرد تحميل الملف، يمكنك الوصول إلى أوراق عمل محددة لمزيد من المعالجة أو التحويل.

**مقتطف من الكود**
```java
Worksheet sheet = workbook.getWorksheets().get(0);
```
*توضيح:* يقوم هذا السطر بجلب أول ورقة عمل في مصنفك. يمكنك تكرار جميع الأوراق إذا لزم الأمر. `workbook.getWorksheets()`.

### تكوين خيارات الصورة والطباعة

#### ملخص
لتحويل إلى XPS، قم بإعداد `ImageOrPrintOptions` لتحديد تفاصيل الإخراج مثل التنسيق والجودة.

**مقتطف من الكود**
```java
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setSaveFormat(SaveFormat.XPS);
```
*توضيح:* هنا، نحدد تنسيق الحفظ كـ XPS باستخدام `SaveFormat.XPS`.

### عرض ورقة عمل Excel كملف XPS

#### ملخص
قم بتحويل ورقة العمل الخاصة بك إلى صورة XPS واحدة باستخدام خيارات الطباعة المخصصة.

**مقتطف من الكود**
```java
SheetRender sr = new SheetRender(sheet, options);
sr.toImage(0, "YOUR_OUTPUT_DIRECTORY" + "/ConvertingToXPS_out.xps");
```
*توضيح:* ال `SheetRender` يتم استخدام الفئة لعرض الورقة وفقًا للخيارات المحددة.

### حفظ مصنف كامل بتنسيق XPS

#### ملخص
احفظ المصنف بأكمله كملف XPS واحد عن طريق تحديد التنسيق المطلوب في طريقة الحفظ.

**مقتطف من الكود**
```java
workbook.save("YOUR_OUTPUT_DIRECTORY" + "/ConvertingToXPS_out.xps", SaveFormat.XPS);
```
*توضيح:* يعمل هذا النهج على تبسيط عملية حفظ أوراق متعددة في مستند XPS واحد، مع الحفاظ على بنية المصنف.
