## دليل التنفيذ

### إنشاء مصنفات العمل والوصول إلى أوراق العمل (H2)

#### ملخص
يُعد إنشاء مصنف Excel والوصول إلى أوراق عمله من المهام الأساسية عند استخدام جداول البيانات. إليك كيفية إنجاز هذه المهام باستخدام Aspose.Cells لـ Java:

##### الخطوة 1: إنشاء مصنف جديد

إنشاء مثيل جديد من `Workbook` الفئة لبدء مستند Excel الخاص بك.

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // قم بتحديد مسار الدليل الخاص بك هنا
dataDir += "/Data/";

// إنشاء مصنف جديد
Workbook workbook = new Workbook();
```

##### الخطوة 2: إضافة أوراق العمل والوصول إليها

يمكنك إضافة أوراق عمل جديدة إلى المصنف الخاص بك والوصول إليها برمجيًا.

```java
import com.aspose.cells.Worksheet;

int sheetIndex = workbook.getWorksheets().add(); // إضافة ورقة عمل
Worksheet worksheet = workbook.getWorksheets().get(sheetIndex); // الوصول إلى ورقة العمل المضافة
```

### تعيين قيمة الخلية ونمطها باستخدام خط التسطير (H2)

#### ملخص
يمكن أن يُحسّن تعديل قيم الخلايا وتطبيق أنماط مثل تسطير الخط سهولة قراءة جدول البيانات. لنرَ كيف يتم ذلك:

##### الخطوة 1: تعيين قيمة الخلية

قم بالوصول إلى خلايا محددة في ورقة العمل لتعيين قيمها.

```java
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;

Cells cells = worksheet.getCells(); // احصل على مجموعة الخلايا
cell = cells.get("A1"); // الوصول إلى الخلية "A1"
cell.setValue("Hello Aspose!"); // تعيين القيمة في الخلية
```

##### الخطوة 2: تطبيق نمط تسطير الخط

استخدم `Style` و `Font` فئات لتعديل مظهر الخلية.

```java
import com.aspose.cells.Style;
import com.aspose.cells.Font;
import com.aspose.cells.FontUnderlineType;

// احصل على النمط الحالي للخلية
Style style = cell.getStyle();
Font font = style.getFont();

// تطبيق نمط التسطير
font.setUnderline(FontUnderlineType.SINGLE);
style.setFont(font);

// تحديث الخلية بأسلوب جديد
cell.setStyle(style);
```

#### حفظ مصنفك

لا تنسى حفظ التغييرات في الملف.

```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // قم بتحديد مسار دليل الإخراج هنا
dataDir += "/SFUnderlineType_out.xls";
workbook.save(dataDir); // حفظ المصنف مع التغييرات
```
