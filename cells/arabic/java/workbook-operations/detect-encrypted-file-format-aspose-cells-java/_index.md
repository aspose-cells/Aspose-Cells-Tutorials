### اكتشاف تنسيق الملف (H2)

#### ملخص

باستخدام `FileFormatUtil` باستخدام فئة Aspose.Cells، يمكنك تحديد تنسيق الملف المشفّر بإدخال كلمة المرور الصحيحة. تُعد هذه الوظيفة أساسية عند التعامل مع أنواع مختلفة من الملفات المخزّنة بشكل آمن باستخدام التشفير.

#### التنفيذ خطوة بخطوة (العناوين الفرعية H3)

1. **حضّر بيئتك:**

   تأكد من أن مشروعك يتضمن التبعيات الضرورية كما هو موضح سابقًا.

2. **إعداد الدليل ومسار الملف:**

   قم بتحديد مسار الدليل الذي تتواجد فيه ملفاتك المشفرة.

   ```java
   String dataDir = "path_to_your_directory/";
   String filename = dataDir + "encryptedBook1.out.tmp";
   ```

3. **الكشف عن تنسيق الملف:**

   يستخدم `FileFormatUtil.detectFileFormat` لتحديد تنسيق الملف عن طريق توفير مسار الملف وكلمة المرور.

   ```java
   FileFormatInfo fileFormatInfo = FileFormatUtil.detectFileFormat(filename, "1234");
   ```

   - **حدود:** 
     - `filename`:المسار إلى ملفك المشفر.
     - `"1234"`:كلمة المرور لفك تشفير معلومات تنسيق الملف.

   - **قيمة الإرجاع:** أ `FileFormatInfo` كائن يحتوي على تفاصيل حول تنسيق الملف المكتشف.

4. **تحديد نوع تنسيق الملف:**

   تقييم نوع تنسيق الملف المُرجع باستخدام العبارات الشرطية:

   ```java
   if (fileFormatInfo.getFileFormatType() == FileFormatType.EXCEL_97_TO_2003) {
       System.out.println("File Format: EXCEL_97_TO_2003");
   } else if (fileFormatInfo.getFileFormatType() == FileFormatType.PPTX) {
       System.out.println("File Format: PPTX");
   } else if (fileFormatInfo.getFileFormatType() == FileFormatType.DOCX) {
       System.out.println("File Format: DOCX");
   }
   ```

#### نصائح استكشاف الأخطاء وإصلاحها

- **المشاكل الشائعة:** 
  - قد يؤدي مسار الملف أو كلمة المرور غير الصحيحة إلى حدوث أخطاء.
  - تأكد من تضمين مكتبة Aspose.Cells وتحديثها بشكل صحيح.
