## دليل التنفيذ

### عرض إصدار Aspose.Cells

#### ملخص
يعد استرجاع معلومات الإصدار وعرضها أمرًا بالغ الأهمية لاستكشاف الأخطاء وإصلاحها وضمان التوافق مع تنسيقات ملفات Excel المختلفة.

#### التنفيذ خطوة بخطوة

**1. استيراد الفئات المطلوبة**
```java
import com.aspose.cells.*;
```

**2. استرداد معلومات الإصدار وطباعتها**
إنشاء فصل دراسي `ShowAsposeCellsVersion` لتنفيذ هذه المهمة.
```java
public class ShowAsposeCellsVersion {
    public static void main(String[] args) throws Exception {
        // الخطوة 1: الحصول على إصدار Aspose.Cells لـ Java
        String version = CellsHelper.getVersion();
        
        // الخطوة 2: عرض الإصدار في وحدة التحكم
        System.out.println("Aspose.Cells for Java Version: " + version);
    }
}
```
- **توضيح:** `CellsHelper.getVersion()` يقوم بجلب إصدار المكتبة الحالي، والذي يتم طباعته باستخدام `System.out.println`.

### إنشاء مصنف عمل مشترك

#### ملخص
تتيح المصنفات المشتركة لمستخدمين متعددين الوصول إلى ملف Excel وتحريره في آنٍ واحد. هذه الميزة مفيدة بشكل خاص للمشاريع التعاونية.

#### التنفيذ خطوة بخطوة

**1. استيراد الفئات المطلوبة**
```java
import com.aspose.cells.*;
```

**2. إنشاء مصنف العمل وتكوينه**
إنشاء فصل دراسي `CreateSharedWorkbook` مع التنفيذ التالي:
```java
public class CreateSharedWorkbook {
    // مسار نائب لدليل الإخراج
    static String outDir = "YOUR_OUTPUT_DIRECTORY";

    public static void main(String[] args) throws Exception {
        // الخطوة 1: تهيئة كائن مصنف جديد
        Workbook wb = new Workbook();
        
        // الخطوة 2: تمكين الإعدادات المشتركة
        wb.getSettings().setShared(true);
        
        // الخطوة 3: احفظ المصنف كملف Excel
        wb.save(outDir + "outputSharedWorkbook.xlsx");
    }
}
```
- **توضيح:** ال `setShared(true)` تقوم الطريقة بتكوين المصنف ليكون مستندًا مشتركًا، والذي يتم حفظه بعد ذلك باستخدام المسار المحدد.
