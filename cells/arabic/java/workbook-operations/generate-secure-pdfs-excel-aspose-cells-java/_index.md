# إنشاء ملفات PDF آمنة من Excel باستخدام Aspose.Cells لـ Java

- كيفية إنشاء مستند PDF آمن من ملف Excel باستخدام Aspose.Cells.
- إعداد خيارات الأمان مثل كلمات مرور المستخدم والمالك.
- تقييد الأذونات لاستخراج المحتوى وطباعته في ملف PDF الناتج.
- تحسين الأداء عند العمل مع مجموعات بيانات كبيرة أو مستندات معقدة.

- **متطلبات المعرفة**:المعرفة بمفاهيم برمجة Java الأساسية، مثل الفئات والطرق وعمليات إدخال/إخراج الملفات.

### إعداد Aspose.Cells لـ Java
#### تثبيت Maven
لتضمين Aspose.Cells في مشروعك باستخدام Maven، أضف التبعية التالية إلى مشروعك `pom.xml`:

```xml

    <artifactId>aspose-cells</artifactId>

</dependency>
```

بالنسبة للمشاريع التي تستخدم Gradle، قم بتضمين هذا السطر في `build.gradle`:


### دليل التنفيذ
#### الميزة: إنشاء مستندات PDF آمنة
تتضمن هذه الميزة تحويل ملف Excel إلى مستند PDF مع إعدادات أمان مُحسّنة. لنشرح العملية خطوة بخطوة.

##### الخطوة 1: فتح ملف Excel
أولاً، حمّل مصنف Excel باستخدام Aspose.Cells. تأكد من تحديد المسار الصحيح لملف الإدخال:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "input.xlsx");
```

هنا، `Workbook` يمثل مستند Excel الذي ترغب في تحويله.

##### الخطوة 2: تكوين خيارات أمان PDF
بعد ذلك، قم بتكوين إعدادات الأمان باستخدام `PdfSaveOptions`. تسمح لك هذه الفئة بتحديد سمات الحماية المختلفة:

```java
// قم بإنشاء PdfSaveOptions لإدارة سمات الأمان الخاصة بملف PDF.
PdfSaveOptions saveOption = new PdfSaveOptions();

// إعداد خيارات الأمان لمستند PDF.
saveOption.setSecurityOptions(new PdfSecurityOptions());
```

##### الخطوة 3: تحديد كلمات المرور والأذونات
تعيين كلمات مرور المستخدم والمالك، إلى جانب قيود الأذونات:

```java
// قم بتحديد كلمة مرور المستخدم المطلوبة لفتح ملف PDF.
saveOption.getSecurityOptions().setUserPassword("user");

// قم بتحديد كلمة مرور للمالك للحصول على حقوق الوصول الكاملة.
saveOption.getSecurityOptions().setOwnerPassword("owner");

// تعطيل استخراج المحتوى من المستند.
saveOption.getSecurityOptions().setExtractContentPermission(false);

// تقييد أذونات الطباعة.
saveOption.getSecurityOptions().setPrintPermission(false);
```

- **كلمة مرور المستخدم**:مطلوب لفتح ملف PDF وعرضه.
- **كلمة مرور المالك**:يمنحك حق الوصول الكامل، مما يسمح لك بتعديل إعدادات الأمان.

##### الخطوة 4: الحفظ كملف PDF آمن
وأخيرًا، احفظ المصنف الخاص بك كمستند PDF آمن:

```java
// قم بتحديد مسار دليل الإخراج لحفظ ملف PDF المؤمن.
String outDir = "YOUR_OUTPUT_DIRECTORY";

// احفظ مصنف Excel كملف PDF آمن في موقع الإخراج المحدد.
workbook.save(outDir + "SecurePDFDocuments_out.pdf", saveOption);
```
