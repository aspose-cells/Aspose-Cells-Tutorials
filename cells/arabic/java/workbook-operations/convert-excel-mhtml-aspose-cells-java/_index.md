### إعداد Aspose.Cells لـ Java
لاستخدام Aspose.Cells في مشروعك، اتبع الخطوات التالية:

#### مافن
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

#### جرادل
قم بتضمين هذا السطر في `build.gradle` ملف:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

### دليل التنفيذ
#### تحميل ملف Excel
لتحميل ملف Excel، اتبع الخطوات التالية:
1. **إعداد دليل البيانات الخاص بك**:قم بتحديد المسار الذي سيتم تخزين ملفات Excel الخاصة بك فيه.
   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // استبدل بمسار دليل البيانات الفعلي الخاص بك
   ```
2. **إنشاء كائن مصنف**:يمثل هذا الكائن مصنف Excel الخاص بك.
   ```java
   String filePath = dataDir + "Book1.xlsx"; // المسار إلى ملف Excel
   Workbook wb = new Workbook(filePath); // تحميل ملف Excel
   ```
3. **لماذا تستخدم `Workbook`؟** ال `Workbook` تعتبر الفئة ضرورية لأنها تغلف جميع الأوراق وبياناتها، مما يسمح بالتلاعب بها بسهولة.

#### تحويل ملف Excel إلى تنسيق MHTML
الآن بعد أن قمنا بتحميل ملف Excel الخاص بنا، فلنحوله إلى MHTML:
1. **إعداد دليل الإخراج**:قم بتحديد المكان الذي تريد حفظ الملف المحول فيه.
   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY"; // استبدل بمسار دليل الإخراج الفعلي الخاص بك
   ```
2. **تحديد خيارات حفظ HTML**: يستخدم `HtmlSaveOptions` لتعيين تنسيق التحويل.
   ```java
   HtmlSaveOptions sv = new HtmlSaveOptions(SaveFormat.M_HTML); // MHTML هو تنسيق أرشيف الويب
   ```
3. **قم بإجراء التحويل**:احفظ المصنف الخاص بك بالتنسيق المطلوب.
   ```java
   wb.save(outDir + "/CToMHTMLFiles_out.mht", sv);
   ```
4. **لماذا `SaveFormat.M_HTML`؟** يضمن هذا الخيار حفظ ملف Excel الخاص بك بتنسيق MHTML، وهو تنسيق مناسب لعرض الويب والأرشفة.
