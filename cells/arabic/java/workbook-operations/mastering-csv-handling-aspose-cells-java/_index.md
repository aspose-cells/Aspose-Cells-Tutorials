## دليل التنفيذ

### فتح ملفات CSV باستخدام خيارات التحميل

#### ملخص
يرشدك هذا القسم إلى كيفية فتح ملف CSV أثناء تكوين خيارات التحميل للتعامل مع الأحرف غير الصالحة.

#### خطوات:
**1. قم بإعداد بيئتك**
ابدأ بتحديد الدليل الذي يتم تخزين ملفات CSV فيه وإنشاء مثيل له `LoadOptions`.
```java
// دليل المصدر
String dataDir = Utils.getSharedDataDir(OpeningCSVFilesAndReplacingInvalidCharacters.class) + "LoadingSavingConvertingAndManaging/";

// إنشاء خيارات التحميل لتنسيق CSV
LoadOptions loadOptions = new LoadOptions(LoadFormat.CSV);
```

**2. قم بتحميل ملف CSV**
استخدم `Workbook` الفئة لفتح ملفك باستخدام خيارات التحميل المحددة.
```java
// تحميل ملف CSV إلى كائن مصنف
Workbook workbook = new Workbook(dataDir + "[20180220142533][ASPOSE_CELLS_TEST].csv", loadOptions);

System.out.println("CSV file opened successfully!");
```

#### توضيح:
- `LoadOptions` يسمح لك بتحديد المعلمات التي تتحكم في كيفية تحميل ملف CSV.
- ال `Workbook` تمثل الفئة جدول البيانات بأكمله، مما يجعل من السهل التعامل مع البيانات وتنظيفها.

### استبدال الأحرف غير الصالحة
سوف يرشدك هذا القسم إلى كيفية استبدال الأحرف غير الصالحة بمجرد فتح ملف CSV الخاص بك.

**3. تحديد واستبدال الأحرف غير الصالحة**
```java
// الوصول إلى ورقة العمل الأولى
Worksheet worksheet = workbook.getWorksheets().get(0);

// احصل على مجموعة خلايا ورقة العمل
Cells cells = worksheet.getCells();

// قم بالتنقل عبر جميع الخلايا للعثور على الأحرف غير الصالحة واستبدالها
for (int row = 0; row < cells.getMaxDataRow() + 1; row++) {
    for (int column = 0; column < cells.getMaxDataColumn() + 1; column++) {
        Cell cell = cells.get(row, column);
        
        // التحقق مما إذا كانت القيمة تحتوي على أحرف غير صالحة
        String cellValue = cell.getStringValue();
        if (cellValue != null && !cellValue.matches("[\\w\\s.,-]+")) { // تعبيرات عادية للأحرف الصالحة
            // استبدل بعنصر نائب أو أي حرف مرغوب فيه
            cell.putValue("INVALID");
        }
    }
}

System.out.println("Invalid characters replaced!");
```

#### توضيح:
- يتكرر هذا الكود في كل خلية، ويتحقق من وجود أحرف غير صالحة باستخدام التعبيرات العادية، ثم يستبدلها.
- قم بتخصيص منطق الاستبدال ليناسب احتياجات تنظيف البيانات الخاصة بك.
