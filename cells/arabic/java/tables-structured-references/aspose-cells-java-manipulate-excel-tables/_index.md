## إعداد Aspose.Cells لـ Java

يعد إعداد Aspose.Cells في مشروعك أمرًا سهلاً باستخدام Maven أو Gradle.

### استخدام Maven

أضف التبعية التالية إلى ملفك `pom.xml` ملف:

```xml
<dependency>
  <groupId>com.aspose</groupId>
  <artifactId>aspose-cells</artifactId>
  <version>25.3</version>
</dependency>
```

قم بتضمين هذا السطر في `build.gradle` ملف:


## دليل التنفيذ

الآن بعد إعداد بيئتك، دعنا نصل إلى جدول Excel ونتعامل معه عبر خلية.

### الوصول إلى جدول Excel من خلية

#### 1. قم بتحميل المصنف

أولاً، قم بإنشاء `Workbook` مثال عن طريق تحميل ملف Excel موجود:

```java
String dataDir = "path/to/your/directory/";
Workbook workbook = new Workbook(dataDir + "ATFromCell.xlsx");
```

#### 2. الوصول إلى ورقة العمل والخلية

قم بالوصول إلى ورقة العمل الأولى والخلية المحددة ضمن الجدول الذي يهمك:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("D5");
```

#### 3. معالجة بيانات الجدول

استخدم `TableObject` لتعديل البيانات باستخدام إزاحات الصفوف والأعمدة:

```java
// ضع قيمة داخل الخلية D5
cell.putValue("D5 Data");

// الوصول إلى الجدول من هذه الخلية
ListObject table = cell.getTable();

// أضف بعض القيمة في موضع الإزاحة المحدد
table.putCellValue(1, 1, "Offset [2,2]");
```

#### 4. احفظ المصنف

وأخيرًا، احفظ التغييرات مرة أخرى في ملف Excel:

```java
workbook.save(dataDir + "ATFromCell_out.xlsx");
```
