في هذا البرنامج التعليمي، سنوضح كيفية تنفيذ وظيفة "إنشاء نطاق اتحاد" باستخدام Aspose.Cells في جافا. باتباع هذه الخطوات، يمكنك دمج مجموعات الخلايا غير المتجاورة بكفاءة ضمن ورقة عمل Excel.

## إعداد Aspose.Cells لـ Java

### تثبيت Aspose.Cells باستخدام Maven

لإضافة Aspose.Cells إلى مشروعك عبر Maven، قم بتضمين التبعية التالية في ملفك `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### تثبيت Aspose.Cells باستخدام Gradle

بالنسبة لأولئك الذين يستخدمون Gradle، أضف هذا السطر إلى `build.gradle` ملف:

```gradle
dependency 'com.aspose:aspose-cells:25.3'
```

## دليل التنفيذ

الآن بعد أن أصبح الإعداد جاهزًا، دعنا ننتقل إلى إنشاء نطاق اتحاد في Excel باستخدام Aspose.Cells Java.

### إنشاء كائنات المصنف وورقة العمل

أولاً، قم بإنشاء `Workbook` الكائن الذي يمثل ملف Excel الخاص بنا:

```java
// إنشاء مصنف جديد
Workbook workbook = new Workbook();
```

بعد ذلك، حدد ورقة العمل التي تريد إنشاء نطاق الاتحاد فيها. في هذا المثال، سنستخدم "sheet1".

### إنشاء نطاق الاتحاد

تتمثل الوظيفة الأساسية في إنشاء اتحاد بين النطاقات غير المتجاورة.

**إنشاء نطاق الاتحاد:**

```java
// تحديد نطاق الاتحاد ضمن الورقة 1
UnionRange unionRange = workbook.getWorksheets().createUnionRange("sheet1!A1:A10,sheet1!C1:C10", 0);
```

في هذه المقتطفة، `createUnionRange` يقبل سلسلة نصية تمثل نطاقات بتنسيق Excel وفهرسًا. هنا، يتم دمج "sheet1!A1:A10" و"sheet1!C1:C10" في نطاق اتحاد واحد.

### ضبط القيم في نطاق الاتحاد

بمجرد إنشائه، يمكنك تعيين قيم للاتحاد بأكمله:

```java
// تعيين القيمة "ABCD" لجميع الخلايا ضمن نطاق الاتحاد
unionRange.setValue("ABCD");
```

يقوم هذا الخط بتعيين السلسلة "ABCD" عبر كل خلية في نطاق الاتحاد المحدد لدينا.

### حفظ المصنف

وأخيرًا، احفظ المصنف الخاص بك للحفاظ على التغييرات:

```java
// حفظ المصنف مع التعديلات
String outputDir = Utils.Get_OutputDirectory();
workbook.save(outputDir + "CreateUnionRange_out.xlsx");
```

ال `save` تكتب الطريقة ملف Excel المحدث إلى الدليل المحدد.
