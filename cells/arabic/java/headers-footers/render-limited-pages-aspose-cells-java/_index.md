## دليل التنفيذ
### الخطوة 1: تحميل ملف Excel
أولاً، قم بتحميل ملف Excel الخاص بك باستخدام Aspose.Cells عن طريق إنشاء `Workbook` هدف.

#### تحميل المصنف
```java
Workbook wb = new Workbook("path/to/sampleImageOrPrintOptions_PageIndexPageCount.xlsx");
```
هنا نستخدم `new Workbook()` لفتح ملف موجود في المسار المحدد.

### الخطوة 2: الوصول إلى أوراق العمل
بعد ذلك، قم بالوصول إلى ورقة العمل المحددة التي تريد عرضها.

#### ورقة عمل الوصول
```java
Worksheet ws = wb.getWorksheets().get(0);
```
يسترجع هذا السطر أول ورقة عمل في المصنف. عدّله لاستهداف أي ورقة حسب فهرسها أو اسمها.

### الخطوة 3: ضبط خيارات الصورة/الطباعة
قم بتكوين خيارات العرض الخاصة بك، وتحديد الصفحات التي تريد عرضها كصور.

#### تكوين خيارات العرض
```java
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.setPageIndex(3); // بدءًا من الصفحة 4 (فهرس يعتمد على 0)
opts.setPageCount(4); // تقديم أربع صفحات متتالية
opts.setImageType(ImageType.PNG);
```
- `setPageIndex`:تحديد صفحة البداية.
- `setPageCount`:حدد عدد الصفحات التي سيتم عرضها.
- `setImageType`:اختر تنسيق الصور الناتجة.

### الخطوة 4: عرض الصفحات
إنشاء `SheetRender` الكائن واستخدامه لتحويل الصفحات إلى صور.

#### عرض الصفحات
```java
SheetRender sr = new SheetRender(ws, opts);

for (int i = opts.getPageIndex(); i < sr.getPageCount(); i++) {
    sr.toImage(i, "outputPath/outputImage-" + (i+1) + ".png");
}
```
هنا، نقوم بالمرور على نطاق الصفحات المحدد وتحويل كل صفحة إلى صورة.

### نصائح استكشاف الأخطاء وإصلاحها
- **فهرس الصفحة خارج النطاق**:تأكد من أن `setPageIndex` و `setPageCount` تقع ضمن العدد الإجمالي للصفحات.
- **أخطاء مسار الملف**:تحقق جيدًا من مسارات الملفات لكل من ملفات Excel المدخلة والصور الناتجة.
