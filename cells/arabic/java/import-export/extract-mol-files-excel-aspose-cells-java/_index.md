## إعداد Aspose.Cells لـ Java (H2)

إعداد Aspose.Cells في مشروع Java الخاص بك سهل للغاية. إليك كيفية القيام بذلك باستخدام Maven أو Gradle:

```xml

```

## دليل التنفيذ

في هذا القسم، سنتناول عملية استخراج ملفات .mol المضمنة من مصنفات Excel.

### نظرة عامة على الميزة
الوظيفة الأساسية هي الوصول إلى بيانات الجزيئات واستخراجها (بتنسيق .mol) من كائنات OLE داخل ملف Excel. يُعد هذا ضروريًا للكيميائيين أو العلماء الذين يحتاجون إلى دمج تحليل البيانات عبر منصات متعددة.

#### الخطوة 1: إعداد الدلائل
أولاً، قم بتحديد دليل البيانات الذي يوجد به مصنف Excel ودليل الإخراج الذي سيتم حفظ الملفات المستخرجة فيه.
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // استبدال بالمسار الفعلي
String outDir = "YOUR_OUTPUT_DIRECTORY"; // مسار دليل الإخراج المطلوب
```

#### الخطوة 2: تحميل المصنف
قم بتحميل ملف Excel باستخدام Aspose.Cells `Workbook` يؤدي هذا إلى تهيئة كائن المصنف الخاص بك لمزيد من المعالجة.
```java
Workbook workbook = new Workbook(dataDir + "/EmbeddedMolSample.xlsx");
```

#### الخطوة 3: الوصول إلى أوراق العمل وكائنات OLE
قم بالتكرار خلال كل ورقة عمل للوصول إلى كائنات OLE المضمنة، والتي تحتوي في هذا السياق على ملفات .mol.
```java
int index = 1;
for (Object obj : workbook.getWorksheets()) {
    Worksheet sheet = (Worksheet) obj; // إرسال الكائن إلى ورقة العمل
    OleObjectCollection oles = sheet.getOleObjects(); // الحصول على مجموعة من كائنات OLE

    for (Object obj2 : oles) {
        OleObject ole = (OleObject) obj2; // الوصول إلى كل كائن OLE
```

#### الخطوة 4: استخراج ملفات .mol وحفظها
بالنسبة لكل كائن OLE، استخرج البيانات المضمنة واحفظها كملف .mol في دليل الإخراج المحدد.
```java
String fileName = outDir + "/OleObject" + index + ".mol"; // قم بتحديد اسم ملف فريد لكل ملف .mol
FileOutputStream fos = new FileOutputStream(fileName); // إنشاء تدفق لكتابة البيانات
fos.write(ole.getObjectData()); // اكتب بيانات .mol المضمنة في الملف
fos.flush(); // تأكد من كتابة كافة البيانات
close(fos); // إغلاق مجرى الملف باستخدام try-with-resources
index++; // زيادة الفهرس لكائن OLE التالي
    }
}
```

### نصائح استكشاف الأخطاء وإصلاحها
- **استثناء عدم العثور على الملف**:تحقق من مسارات دليل الإدخال والإخراج لديك.
- **استثناء IO**:تأكد من أن لديك أذونات الكتابة في دليل الإخراج الخاص بك.
