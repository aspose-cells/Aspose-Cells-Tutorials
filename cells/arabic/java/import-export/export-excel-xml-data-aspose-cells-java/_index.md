## دليل التنفيذ
### تحميل مصنف المصدر
#### ملخص
الخطوة الأولى هي تحميل مصنف Excel من مجلد محدد. يتيح لك هذا الوصول إلى البيانات الموجودة فيه ومعالجتها.

#### مقتطف من الكود
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY/TechnicalArticles/";
Workbook wb = new Workbook(dataDir + "sample_Export-Xml-Data-linked.xlsx");
```
**توضيح**: 
- `dataDir` يحدد مكان وجود المصنف الخاص بك.
- ال `Workbook` يقوم الكائن بتحميل ملف Excel، مما يجعله جاهزًا للعمليات الإضافية.

### تصدير بيانات XML لخرائط XML
#### ملخص
بعد تحميل المصنف، يمكنك تصدير جميع بيانات XML المرتبطة بخرائط XML فيه. تُعد هذه الميزة أساسية لدمج بيانات XML أو تحليلها برمجيًا.

#### مقتطف من الكود
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.XmlMap;

String outDir = "YOUR_OUTPUT_DIRECTORY/";
for (int i = 0; i < wb.getWorksheets().getXmlMaps().getCount(); i++) {
    XmlMap map = wb.getWorksheets().getXmlMaps().get(i);
    wb.exportXml(map.getName(), outDir + map.getName() + ".xml");
}
```
**توضيح**: 
- كرر كل شيء `XmlMap` في المصنف.
- يستخدم `exportXml()` طريقة لتصدير البيانات من كل خريطة XML إلى دليل إخراج محدد.

#### خيارات تكوين المفاتيح
- تعديل `dataDir` و `outDir` المسارات وفقًا لهيكل الملف الخاص بك.
- تأكد من وجود الدلائل المحددة قبل تشغيل البرنامج لتجنب الاستثناءات.
