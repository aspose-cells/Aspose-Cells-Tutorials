### إعداد Aspose.Cells لـ Java
لاستخدام Aspose.Cells في مشروع Java الخاص بك، قم بإضافته عبر Maven أو Gradle:

**مافن**
```xml
<dependency>
  <groupId>com.aspose</groupId>
  <artifactId>aspose-cells</artifactId>
  <version>25.3</version>
</dependency>
```

**جرادل**
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

### دليل التنفيذ
#### تصدير شرائح Excel إلى PDF
**ملخص**
تتيح لك هذه الميزة تحويل ملف Excel الذي يحتوي على مخطط شرائح إلى مستند PDF، مع الحفاظ على عناصره التفاعلية وتخطيطه.

**تعليمات خطوة بخطوة**
1. **تحميل المصنف**:قم بتحميل ملف Excel الخاص بك باستخدام `Workbook`، مع تحديد المسار الذي يوجد فيه.
   
   ```java
   // قم بتحميل المصنف من الدليل المحدد
   Workbook workbook = new Workbook(dataDir + "/SampleSlicerChart.xlsx");
   ```

2. **حفظ كملف PDF**:استخدم `save` طريقة `Workbook` لتصدير الملف إلى تنسيق PDF، مع تحديد دليل الإخراج والتنسيق.
   
   ```java
   // احفظ المصنف بتنسيق PDF في دليل الإخراج
   workbook.save(outDir + "/SampleSlicerChart.pdf", SaveFormat.PDF);
   ```

**خيارات تكوين المفاتيح**
- **حفظ بتنسيق PDF**:يضمن هذا الثابت حفظ الملف كمستند PDF، مع الحفاظ على الدقة المرئية لمخططات Excel.
