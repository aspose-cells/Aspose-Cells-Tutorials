**الكلمات الرئيسية الأساسية:** Aspose.Cells Java، تغيير أهداف ارتباط HTML

## إعداد Aspose.Cells لـ Java

للبدء، قم بتضمين Aspose.Cells في مشروعك عبر Maven أو Gradle.

### تثبيت Maven

أضف التبعية التالية إلى ملفك `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### تثبيت Gradle

فيك `build.gradle`، قم بتضمين السطر التالي:

```gradle
implementation 'com.aspose:aspose-cells:25.3'
```

### خطوات الحصول على الترخيص

يتطلب Aspose.Cells ترخيصًا للحصول على الوظائف الكاملة، ولكن يمكنك البدء بإصدار تجريبي مجاني أو طلب ترخيص مؤقت:

- **نسخة تجريبية مجانية:** قم بتنزيل Aspose.Cells for Java واختباره لاستكشاف إمكانياته.
- **رخصة مؤقتة:** قم بالتقديم على موقع Aspose للحصول على ترخيص مؤقت يزيل قيود التقييم.
- **رخصة الشراء:** فكر في الشراء إذا وجدت أن المكتبة تناسب احتياجاتك.

### التهيئة الأساسية

```java
// استيراد الفئات الضرورية من حزمة Aspose.Cells
import com.aspose.cells.Workbook;
import com.aspose.cells.HtmlSaveOptions;

public class Main {
    public static void main(String[] args) throws Exception {
        // تهيئة كائن مصنف باستخدام مسار ملف Excel
        Workbook workbook = new Workbook("path/to/your/excel/file.xlsx");
        
        // إعداد HtmlSaveOptions للتصدير إلى تنسيق HTML
        HtmlSaveOptions options = new HtmlSaveOptions();
        // تكوين خيارات إضافية حسب الحاجة
        
        // حفظ المستند بتنسيق HTML
        workbook.save("output/path/outputFile.html", options);
    }
}
```

## دليل التنفيذ

الآن، دعنا نوضح كيفية تغيير أهداف ارتباط HTML باستخدام Aspose.Cells لـ Java.

### نظرة عامة على تغيير أهداف الارتباط

عند تصدير ملفات Excel إلى HTML، يمكن ضبط الروابط التشعبية لفتحها في النافذة نفسها أو في علامة تبويب جديدة. هذه الميزة أساسية للتحكم في تنقل المستخدم وتحسين محركات البحث.

#### الخطوة 1: إعداد مصنف العمل الخاص بك

ابدأ بتحميل ملف Excel الخاص بك إلى Aspose.Cells `Workbook` هدف:

```java
String dataDir = "your/data/directory/";
String inputPath = dataDir + "Sample1.xlsx";
Workbook workbook = new Workbook(inputPath);
```

#### الخطوة 2: تكوين خيارات حفظ HTML

استخدم `HtmlSaveOptions` الفئة لتحديد كيفية تصرف الروابط التشعبية في مستند HTML المُصدَّر:

```java
// استيراد HtmlLinkTargetType لتعيين أهداف الرابط
import com.aspose.cells.HtmlLinkTargetType;

HtmlSaveOptions opts = new HtmlSaveOptions();
opts.setLinkTargetType(HtmlLinkTargetType.SELF);  // تفتح الروابط في نفس النافذة/علامة التبويب
```

#### الخطوة 3: حفظ المصنف بصيغة HTML

وأخيرًا، احفظ المصنف الخاص بك باستخدام الخيارات التي قمت بتكوينها:

```java
String outputPath = dataDir + "CHLinkTarget.html";
workbook.save(outputPath, opts);

System.out.println("File saved at " + outputPath);
```

### نصائح استكشاف الأخطاء وإصلاحها

- **مشكلة شائعة:** الروابط لا تُحدَّث كما هو متوقع. تأكد من ذلك. `HtmlSaveOptions` تم تكوينه وتطبيقه بشكل صحيح أثناء عملية الحفظ.
- **مخاوف الأداء:** بالنسبة للملفات الكبيرة، فكر في تحسين الكود الخاص بك للتعامل مع الذاكرة بكفاءة.
