## دليل التنفيذ
### الميزة 1: إنشاء مصنف وتهيئته
**ملخص**:ابدأ بإنشاء كائن مصنف ليكون بمثابة الأساس لجميع العمليات اللاحقة.

#### خطوة بخطوة:
1. **استيراد الفئات المطلوبة**
    ```java
    import com.aspose.cells.Workbook;
    ```
2. **تهيئة مثيل المصنف**
    ```java
    String dataDir = "YOUR_DATA_DIRECTORY";
    Workbook wb = new Workbook();
    ```
تؤدي هذه الخطوة إلى إنشاء مصنف فارغ، جاهزًا لإجراء تعديلات إضافية مثل إضافة خرائط XML.

### الميزة 2: إضافة خريطة XML إلى المصنف
**ملخص**:دمج ملف XML خارجي في المصنف الخاص بك لمعالجة البيانات والتلاعب بها بشكل متقدم.

#### خطوة بخطوة:
1. **تحميل ملف XML**
    ```java
    wb.getWorksheets().getXmlMaps().add(dataDir + "sample.xml");
    ```
يضيف هذا الخط `sample.xml` من الدليل المحدد إلى مجموعة خرائط XML الخاصة بالمصنف، مما يتيح مهام استخراج البيانات وتحويلها.

### الميزة 3: حفظ المصنف بتنسيق XLSX
**ملخص**:قم بتصدير المصنف المعدّل إلى ملف XLSX، مع الحفاظ على جميع التغييرات التي أجريتها أثناء المعالجة.

#### خطوة بخطوة:
1. **تحديد دليل الإخراج**
    ```java
    String outDir = "YOUR_OUTPUT_DIRECTORY";
    ```
2. **احفظ المصنف بتنسيق XLSX**
    ```java
    wb.save(outDir + "AddXMLMapInsideWorkbook_out.xlsx");
    ```
تضمن هذه الخطوة أن المصنف الخاص بك، والذي تم إثرائه الآن بتعيينات بيانات XML، تم حفظه بشكل صحيح للمشاركة أو التحليل الإضافي.
