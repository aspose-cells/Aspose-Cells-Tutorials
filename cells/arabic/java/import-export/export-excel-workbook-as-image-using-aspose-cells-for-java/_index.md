# تصدير مصنف Excel كصورة باستخدام Aspose.Cells لـ Java

**ما سوف تتعلمه:**
- إعداد وتثبيت Aspose.Cells لـ Java
- تحميل مصنف Excel وتكوينه لعرض الصور
- تخصيص خيارات الإخراج مثل التنسيق والتخطيط
- الاستخدامات العملية لتصدير المصنفات كصور

من خلال اتباع هذا الدليل، سوف تتقن عملية تحويل ملفات Excel إلى صور باستخدام Aspose.Cells في Java.

```xml

```

## دليل التنفيذ

### تحميل المصنف

قم بتحميل مصنف Excel باستخدام `Workbook` فصل:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // استبدله بمسار دليل الإدخال الخاص بك
Workbook book = new Workbook(dataDir + "/book1.xlsx"); // تحميل المصنف
```
**توضيح**: ال `Workbook` يُعد الكائن أساسيًا للوصول إلى ملفات Excel ومعالجتها. هنا، نقوم بتحميل ملف باسم `book1.xlsx`.

### تكوين خيارات عرض الصورة

تكوين معلمات العرض باستخدام `ImageOrPrintOptions`:
```java
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setImageType(ImageType.TIFF); // تعيين تنسيق الإخراج إلى TIFF
options.setOnePagePerSheet(true); // عرض كل ورقة على صفحة واحدة
```
**توضيح**: `ImageOrPrintOptions` يسمح لك بتحديد معلمات مثل نوع الصورة وتخطيطها. هنا، نستخدم صيغة TIFF مع صورة واحدة لكل ورقة Excel.

### عرض المصنف

عرض المصنف كصورة:
```java
WorkbookRender render = new WorkbookRender(book, options); // تهيئة برنامج العرض باستخدام الخيارات
render.toImage("YOUR_OUTPUT_DIRECTORY/CWorkbooktoImage_out.tiff"); // حفظ الصورة الناتجة
```
**توضيح**: `WorkbookRender` يأخذ `Workbook` و `ImageOrPrintOptions`تحويل ملف Excel إلى صورة. حدد مكان الحفظ واسم الملف هنا.

### نصائح استكشاف الأخطاء وإصلاحها
- **خطأ عدم العثور على الملف**:تأكد من أن مسار دليل الإدخال الخاص بك صحيح.
- **تنسيق الصورة غير مدعوم**:تحقق مما إذا كان التنسيق المحدد في `setImageType()` يتم دعمه.
