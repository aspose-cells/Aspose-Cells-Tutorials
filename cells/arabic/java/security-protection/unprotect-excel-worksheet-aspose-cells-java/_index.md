## دليل التنفيذ

الآن، دعنا ننفذ الوظيفة لإلغاء حماية ورقة عمل Excel.

### نظرة عامة حول إلغاء حماية ورقة العمل

تتيح لك هذه الميزة إزالة الحماية عن ورقة عمل كانت مؤمّنة مسبقًا. تُعد هذه الميزة مفيدة عند الحاجة إلى إجراء تغييرات أو مشاركة بيانات دون قيود.

#### الخطوة 1: إنشاء كائن مصنف

أولاً، قم بإنشاء `Workbook` الكائن وتحميل ملف Excel المحمي الخاص بك:

```java
String dataDir = Utils.getSharedDataDir(UnprotectingSimplyProtectedWorksheet.class) + "Worksheets/";
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

#### الخطوة 2: الوصول إلى مجموعة أوراق العمل

استرداد مجموعة أوراق العمل من المصنف وتحديد الورقة التي تريد إلغاء حمايتها:

```java
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet worksheet = worksheets.get(0);
```

#### الخطوة 3: تعديل إعدادات الحماية (لإصدار Excel 2000 والإصدارات الأقدم)

إذا كنت تتعامل مع تنسيقات Excel القديمة، فاضبط إعدادات الحماية:

```java
Protection protection = worksheet.getProtection();
protection.setAllowEditingContent(false);
protection.setAllowEditingObject(false);
protection.setAllowEditingScenario(false);
```

#### الخطوة 4: إلغاء حماية ورقة العمل

قم بإزالة الحماية باستخدام `unprotect()` هذه الخطوة لا تتطلب أي معلمات إذا لم يتم تعيين كلمة مرور:

```java
worksheet.unprotect();
```

#### الخطوة 5: حفظ التغييرات في ملف جديد

وأخيرًا، احفظ التغييرات في ملف جديد:

```java
workbook.save(dataDir + "USPWorksheet_out.xls");
```
