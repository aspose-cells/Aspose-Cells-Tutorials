**مافن:**

```xml

    <artifactId>aspose-cells</artifactId>

```

**جرادل:**

```gradle

```

## دليل التنفيذ

### فتح ملفات Excel المشفرة

#### ملخص

سوف نقوم بفتح ملف Excel مشفر عن طريق تعيين كلمة مرور في `LoadOptions` وإنشاء `Workbook` كائن لتحميل الملف.

**الخطوة 1: إعداد خيارات التحميل**

أولاً، قم بإنشاء `LoadOptions` الكائن بالتنسيق المناسب. هذا يعني أننا نتعامل مع تنسيق Excel قديم (97-2003):

```java
// إنشاء LoadOptions لتنسيق Excel 97-2003
LoadOptions loadOptions = new LoadOptions(LoadFormat.EXCEL_97_TO_2003);
```

**الخطوة 2: تعيين كلمة المرور**

بعد ذلك، قم بتعيين كلمة المرور المطلوبة لفتح الملف المشفر:

```java
// حدد كلمة المرور لفتح المصنف
loadOptions.setPassword("1234");
```

#### توضيح

ال `LoadOptions` تتيح لك الفئة تحديد إعدادات تحميل الملفات. بتعيين كلمة مرور، تضمن وصول المستخدمين المصرح لهم فقط إلى محتويات ملفات Excel.

**الخطوة 3: تحميل المصنف**

وأخيرا، قم بإنشاء `Workbook` مثال مع مسار الملف وخيارات التحميل:

```java
// تهيئة كائن المصنف باستخدام مسار الملف المشفر وخيارات التحميل
new Workbook("path/to/your/encryptedBook.xls", loadOptions);
System.out.println("Encrypted workbook has been opened successfully.");
```

### نصائح استكشاف الأخطاء وإصلاحها

- **كلمة المرور غير صحيحة:** تأكد من إدخال كلمة المرور الصحيحة. كلمة مرور خاطئة ستؤدي إلى استثناء.
- **مشاكل مسار الملف:** تأكد من مسار الملف للتأكد من عدم وجود أخطاء مطبعية أو مراجع دليل غير صحيحة.
