## دليل التنفيذ
اتبع الخطوات التالية لتنفيذ تغييرات اتجاه النص في التعليقات.

### إضافة التعليقات وتعديلها
#### ملخص
قم بتعزيز وضوح المستند عن طريق إضافة تعليقات إلى الخلايا وتخصيص اتجاه النص الخاص بها، وهو أمر مفيد بشكل خاص للمستندات أو الأنظمة متعددة اللغات التي تتطلب تنسيقًا محددًا.

**الخطوة 1: إنشاء مصنف**
إنشاء مثيل لـ `Workbook` فصل:
```java
// إنشاء مصنف جديد
Workbook wb = new Workbook();
```

**الخطوة 2: الوصول إلى ورقة العمل**
قم بالوصول إلى ورقة العمل التي تريد إضافة التعليقات إليها:
```java
// احصل على ورقة العمل الأولى في المصنف
Worksheet sheet = wb.getWorksheets().get(0);
```

**الخطوة 3: إضافة تعليق**
أضف تعليقًا إلى الخلية "A1":
```java
// أضف تعليقًا إلى الخلية A1
Comment comment = sheet.getComments().get(sheet.getComments().add("A1"));
```
استرداد كائن التعليق عن طريق إضافته إلى الخلية الأولى.

**الخطوة 4: تكوين محاذاة النص واتجاهه**
تخصيص محاذاة النص وتعيين الاتجاه من اليمين إلى اليسار:
```java
// تعيين محاذاة النص الرأسية والأفقية
comment.getCommentShape().setTextVerticalAlignment(TextAlignmentType.CENTER);
comment.getCommentShape().setTextHorizontalAlignment(TextAlignmentType.RIGHT);

// تغيير اتجاه النص من اليمين إلى اليسار
comment.getCommentShape().setTextDirection(TextDirectionType.RIGHT_TO_LEFT);
```
ال `TextAlignmentType` و `TextDirectionType` تساعد العناصر في تكوين مظهر تعليقاتك.

**الخطوة 5: تعيين ملاحظة التعليق**
حدد محتوى التعليق:
```java
// أضف ملاحظة إلى التعليق
comment.setNote("This is my Comment Text. This is test");
```
