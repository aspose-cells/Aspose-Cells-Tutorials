قبل البدء بالبرمجة، لنبدأ بإعداد بيئتك لاستخدام Aspose.Cells لجافا. يمكنك دمج هذه المكتبة القوية بسهولة باستخدام Maven أو Gradle.

### إعداد Maven
أضف التبعية التالية في ملفك `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>

</dependency>
```

قم بتضمين هذا السطر في `build.gradle` ملف:


## دليل التنفيذ

دعنا نستعرض كيفية إعداد مصنف باستخدام وضع حساب الصيغة اليدوي باستخدام Aspose.Cells لـ Java.

### إنشاء المصنف وتعيين وضع الحساب

#### ملخص

يؤدي ضبط وضع حساب الصيغة إلى الوضع اليدوي إلى منع إعادة حساب الصيغ تلقائيًا، مما يسمح لك بتشغيل الحسابات عند الحاجة فقط. هذا يُحسّن الأداء بشكل ملحوظ في المصنفات الكبيرة.

#### التنفيذ خطوة بخطوة

##### الخطوة 1: إنشاء مصنف جديد
ابدأ من خلال تهيئة مثيل مصنف جديد:

```java
import com.aspose.cells.Workbook;

Workbook workbook = new Workbook();
```

##### الخطوة 2: ضبط وضع الحساب إلى يدوي
قم بتكوين وضع حساب الصيغة إلى الوضع اليدوي باستخدام `CalcModeType.MANUAL`:

```java
import com.aspose.cells.CalcModeType;
import com.aspose.cells.SaveFormat;

workbook.getSettings().getFormulaSettings().setCalculationMode(CalcModeType.MANUAL);
```

##### الخطوة 3: حفظ المصنف

وأخيرًا، احفظ المصنف الخاص بك في الموقع المطلوب بتنسيق XLSX:

```java
workbook.save("SFCalculationMode_out.xlsx", SaveFormat.XLSX);
```

### نصائح استكشاف الأخطاء وإصلاحها

- **أخطاء الحساب**:تأكد من صحة جميع الصيغ قبل الحفظ.
- **مشاكل مسار الملف**:تحقق مرة أخرى من مسار الملف المستخدم في `save` طريقة.
