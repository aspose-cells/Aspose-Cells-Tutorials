## دليل التنفيذ
### توسيع فئة الشخص
في هذا القسم، سنقوم بتوسيع `Person` فئة لإنشاء `Individual` الفئة التي تدير السمات والسلوكيات الإضافية.
#### ملخص:
ال `Individual` تمتد الفئة `Person`، عرض الميراث في Java لتحسين الوظائف عن طريق إضافة خصائص محددة مثل معلومات الزوج/الزوجة.
##### الخطوة 1: تحديد الفئة الفردية
ابدأ بإنشاء `Individual` الفئة، بما في ذلك الأعضاء الخاصة والمنشئين لتهيئة الكائنات:
```java
import java.util.ArrayList;
class Person {
    // نسخة مبسطة من فئة أساسية مثل Aspose.Person
    protected String name;
    protected int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
// فئة فردية تمتد للشخص
class Individual extends Person {
    private Person m_Wife; // عضو خاص لمعلومات الزوج

    // منشئ للفئة الفردية
    public Individual(String name, int age, Person wife) {
        super(name, age); // استدعاء منشئ الفئة العليا
        this.m_Wife = wife; // تهيئة m_Wife بالقيمة المقدمة
    }

    // طريقة الحصول على m_Wife
    public Person getWife() {
        return m_Wife;
    }
}
```
**توضيح**: 
- **منشئ الفئة العليا**: `super(name, age)` يقوم بتهيئة الفئة العليا `Person` صفات.
- **عضو خاص**: `m_Wife` يخزن معلومات الزوج/الزوجة، ويعرض التغليف.
##### الخطوة 2: الاستفادة من الفصل الفردي
قم بإنشاء مثيلات لفئتك الجديدة واستفد من وظائفها:
```java
public class Main {
    public static void main(String[] args) {
        Person wife = new Person("Jane", 30);
        Individual person = new Individual("John", 35, wife);

        System.out.println("Person's Wife: " + person.getWife().name); // المخرج: جين
    }
}
```
**توضيح**: 
- وهذا يوضح إنشاء `Person` الاعتراض على تمثيل الزوج وتمريره عند بناء `Individual`.
