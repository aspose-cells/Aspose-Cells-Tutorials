## دليل التنفيذ

### تحويل اسم الخلية إلى فهرس

تتيح لك هذه الميزة تحويل أسماء خلايا Excel (مثل "C6") إلى فهارس الصفوف والأعمدة الخاصة بها. لنشرح الخطوات بالتفصيل:

#### الخطوة 1: استيراد الفئات المطلوبة

ابدأ باستيراد الفئات الضرورية من Aspose.Cells:

```java
import com.aspose.cells.CellsHelper;
```

#### الخطوة 2: تنفيذ منطق التحويل

استخدم `CellsHelper.cellNameToIndex` طريقة إجراء التحويل:

```java
public class NameToIndex {
    public static void main(String[] args) throws Exception {
        // تحويل اسم الخلية "C6" إلى مؤشرات
        int[] cellIndices = CellsHelper.cellNameToIndex("C6");
        
        // إخراج النتائج
        System.out.println("Row Index of Cell C6: " + cellIndices[0]);
        System.out.println("Column Index of Cell C6: " + cellIndices[1]);
    }
}
```

**توضيح**: 
- `CellsHelper.cellNameToIndex` يأخذ سلسلة تمثل اسم خلية Excel ويعيد مصفوفة حيث يكون العنصر الأول هو فهرس الصف، والثاني هو فهرس العمود.

#### الخطوة 3: تشغيل الكود الخاص بك

قم بتجميع تطبيق جافا وتشغيله لمشاهدة عملية التحويل. يجب أن يظهر الناتج مشابهًا لما يلي:

