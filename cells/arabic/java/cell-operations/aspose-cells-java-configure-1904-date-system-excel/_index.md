# إتقان نظام التاريخ لعام 1904 في Excel باستخدام Aspose.Cells Java لإجراء عمليات فعالة على الخلايا

## مقدمة

قد تُشكّل إدارة البيانات التاريخية في Excel تحديًا نظرًا لاختلاف أنظمة التواريخ، مثل نظام 1904. مع Aspose.Cells لـ Java، يُمكنك بسهولة تهيئة جداول بيانات Excel ومعالجتها مع ضمان التوافق مع مختلف أنظمة التواريخ. سيُرشدك هذا البرنامج التعليمي خلال تهيئة مصنف جديد، وتفعيل نظام 1904، وحفظ التغييرات باستخدام Aspose.Cells لـ Java.

**ما سوف تتعلمه:**
- تهيئة مصنف Aspose.Cells في Java
- تمكين نظام التاريخ 1904 في ملفات Excel
- حفظ مصنفك باستخدام التكوينات المحدثة

دعونا نلقي نظرة على المتطلبات الأساسية اللازمة قبل البدء.

## المتطلبات الأساسية

لمتابعة هذا البرنامج التعليمي، تأكد من أن لديك:
- **مجموعة تطوير جافا (JDK)** مُثبَّت على جهازك. يُنصح باستخدام الإصدار 8 أو أعلى.
- **مافن** أو **جرادل** لإدارة التبعيات، اعتمادًا على إعداد مشروعك.
- المعرفة الأساسية بلغة جافا والتعرف على عمليات ملفات Excel.

## إعداد Aspose.Cells لـ Java

لاستخدام Aspose.Cells لجافا في مشاريعك، أضفها كاعتمادية. فيما يلي تعليمات لإعدادات Maven وGradle:

### **مافن**

أضف التبعية التالية إلى ملفك `pom.xml` ملف:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### **جرادل**

قم بتضمين هذا السطر في `build.gradle` ملف:

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

يقدم Aspose نسخة تجريبية مجانية، وترخيصًا مؤقتًا، وخيارات لشراء تراخيص للاستخدام التجاري. يمكنك البدء بـ [نسخة تجريبية مجانية](https://releases.aspose.com/cells/java/) أو الحصول على ترخيص مؤقت من [صفحة الترخيص المؤقت](https://purchase.aspose.com/temporary-license/).

لتهيئة Aspose.Cells في تطبيق Java الخاص بك، قم بتضمين بيان الاستيراد التالي:

```java
import com.aspose.cells.Workbook;
```

## دليل التنفيذ

### تهيئة وتحميل المصنف

#### ملخص

أولاً، قم بإنشاء مثيل جديد لـ `Workbook` وحمّل ملف Excel موجودًا. هذا الإعداد ضروري لمزيد من العمليات.

#### مقتطف من الكود

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // تأكد من أن المسار إلى ملف Excel الخاص بك صحيح
// تهيئة كائن مصنف باستخدام المسار إلى ملف Excel الخاص بك
Workbook workbook = new Workbook(dataDir + "/Mybook.xlsx");
```

- **حدود:**
  - `dataDir`:الدليل الذي توجد به ملفات Excel المصدرية الخاصة بك.
  - `"/Mybook.xlsx"`:اسم ملف Excel الذي ترغب في تحميله.

### تنفيذ نظام التاريخ 1904

#### ملخص

يُعد نظام التاريخ ١٩٠٤ أساسيًا للتوافق مع بعض التطبيقات. سنقوم هنا بتفعيله في مصنف Excel باستخدام Aspose.Cells.

#### مقتطف من الكود

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // تأكد من أن المسار إلى ملف Excel الخاص بك صحيح
// قم بتحميل المصنف من الدليل المحدد
Workbook workbook = new Workbook(dataDir + "/Mybook.xlsx");

// تمكين نظام التاريخ 1904
workbook.getSettings().setDate1904(true);
```

- **تكوين المفتاح:**
  - `getSettings()`:استرجاع إعدادات المصنف.
  - `setDate1904(true)`:تفعيل نظام التاريخ 1904.

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن مسار ملف Excel الخاص بك صحيح ويمكن الوصول إليه.
- تأكد من أنك قمت بتعيين الإصدار الصحيح من Aspose.Cells لتجنب مشكلات التوافق.

### حفظ المصنف

#### ملخص

بعد إجراء التغييرات، مثل تفعيل نظام التاريخ ١٩٠٤، من الضروري حفظ المصنف. تُنهي هذه الخطوة جميع التعديلات التي أجريتها.

#### مقتطف من الكود

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // تأكد من أن المسار إلى ملف Excel الخاص بك صحيح
String outDir = "YOUR_OUTPUT_DIRECTORY"; // حدد المكان الذي تريد حفظ المصنف المعدل فيه

// قم بتحميل وتعديل المصنف الخاص بك كما هو موضح في الخطوات السابقة
tWorkbook workbook = new Workbook(dataDir + "/Mybook.xlsx");
workbook.getSettings().setDate1904(true);

// حفظ التغييرات في ملف جديد
workbook.save(outDir + "/I1904DateSystem_out.xls");
```

- **حدود:**
  - `outDir`:الدليل الذي تريد حفظ المصنف الذي قمت بتعديله فيه.
  - `"/I1904DateSystem_out.xls"`: اسم ملف Excel الناتج.
