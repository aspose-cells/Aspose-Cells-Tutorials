## دليل التنفيذ

### دمج الخلايا

دمج الخلايا يجمع عدة خلايا متجاورة في خلية واحدة، وهو مفيد لإنشاء رؤوس أو تنظيم البيانات بكفاءة. إليك كيفية القيام بذلك باستخدام Aspose.Cells.

#### عملية خطوة بخطوة
**1. إنشاء مصنف جديد**  
ابدأ بإنشاء نسخة من الفئة `Workbook`، التي تمثل ملف Excel الخاص بك.
```java
// Initialize a workbook
Workbook workbook = new Workbook();
```

**2. الوصول إلى ورقة العمل**  
احصل على ورقة العمل الأولى من المصنف لإجراء العمليات.
```java
// Get the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**3. تعريف نطاق من الخلايا**  
حدد النطاق الذي تريد دمجه، مثل `A1:D4`، والذي سيصبح رأسك المدمج.
```java
// Create a cell range
Range range = worksheet.getCells().createRange("A1:D4");
```

**4. دمج النطاق المحدد**  
استدعِ طريقة `merge()` على النطاق المحدد لدمج الخلايا.
```java
// Merge the range into one cell
range.merge();
```

**5. حفظ المصنف**  
احفظ التغييرات بتحديد دليل الإخراج واسم الملف.
```java
// Specify the output directory
String outDir = "YOUR_OUTPUT_DIRECTORY";

// Save the workbook
workbook.save(outDir + "MURangeofCells_out.xlsx");
```

### إلغاء دمج الخلايا

إلغاء دمج الخلايا مهم عندما تحتاج إلى إرجاع التغييرات أو تعديل تخطيطات البيانات. اتبع هذه الخطوات لإلغاء دمج الخلايا التي تم دمجها مسبقًا.

#### عملية خطوة بخطوة
**1. تحميل المصنف**  
حمّل مصنفًا موجودًا يحتوي على نطاق خلايا مدمج.
```java
// Load the workbook with merged cells
Workbook workbook = new Workbook(outDir + "MURangeofCells_out.xlsx");
```

**2. الوصول إلى ورقة العمل مرة أخرى**  
أعد الوصول إلى ورقة العمل الأولى لإجراء عمليات إلغاء الدمج.
```java
// Get the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**3. تعريف نفس نطاق الخلايا**  
حدد النطاق الذي دمجته مسبقًا.
```java
// Create a cell range
Range range = worksheet.getCells().createRange("A1:D4");
```

**4. إلغاء دمج النطاق**  
استدعِ طريقة `unMerge()` لإرجاع الخلايا إلى حالتها الأصلية.
```java
// Unmerge the range
range.unMerge();
```

**5. حفظ التغييرات**  
احفظ المصنف مع الخلايا غير المدمجة.
```java
// Save the workbook with unmerged changes
workbook.save(outDir + "UnMURangeofCells_out.xlsx");
```
