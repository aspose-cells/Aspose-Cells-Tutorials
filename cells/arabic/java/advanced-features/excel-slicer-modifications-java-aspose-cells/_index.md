### تحميل ملف Excel

**ملخص:**

يؤدي تحميل مصنف Excel الحالي إلى إعداد الأساس لأي تعديلات تخطط لإجرائها.

#### الخطوة 2: إنشاء مصنف وتحميله

public class LoadExcelFile {

        return new Workbook(dataDir + "/sampleFormattingSlicer.xlsx");

- **توضيح:** ال `Workbook` يقوم المُنشئ بتحميل ملف Excel من المسار المُحدد. تأكد من ضبط دليل البيانات بشكل صحيح لتجنب الاستثناءات.

### ورقة عمل الوصول

**ملخص:**

يتيح لك الوصول إلى أوراق العمل داخل مصنف استهداف مناطق معينة للتعديلات، مثل أدوات التقطيع.

#### الخطوة 3: استرداد ورقة العمل الأولى

```java
public class AccessWorksheet {
    public static Worksheet getFirstWorksheet(Workbook wb) throws Exception {
        return wb.getWorksheets().get(0);
    }
}
```

- **توضيح:** تعمل هذه الطريقة على جلب ورقة العمل الأولى في المصنف الخاص بك، وهو المكان الذي سنطبق فيه تعديلات المقطع.

### تعديل خصائص المقطع

**ملخص:**

يؤدي تخصيص خصائص المقطع إلى تحسين عرض تقارير Excel وسهولة استخدامها.

#### الخطوة 4: تكوين المقطع

```java
public class ModifySlicerProperties {
    public static void configureSlicer(Worksheet ws) throws Exception {
        Slicer slicer = ws.getSlicers().get(0);
        
        // تعيين عدد الأعمدة التي يعرضها المقطع
        slicer.setNumberOfColumns(2);
        
        // تغيير نوع النمط للحصول على مظهر بصري أفضل
        slicer.setStyleType(SlicerStyleType.SLICER_STYLE_LIGHT_6);
    }
}
```

- **توضيح:** ال `Slicer` يتيح لك الكائن معالجة خصائص مثل عدد الأعمدة والأسلوب المرئي، مما يعزز كل من الوظيفة والمظهر.

### حفظ المصنف

**ملخص:**

يضمن حفظ التغييرات الحفاظ على جميع التعديلات لاستخدامها أو مشاركتها في المستقبل.

#### الخطوة 5: حفظ التغييرات

```java
public class SaveWorkbook {
    public static void saveModifiedWorkbook(Workbook wb) throws Exception {
        wb.save(outDir + "/outputFormattingSlicer.xlsx", SaveFormat.XLSX);

```

- **توضيح:** ال `save` تعيد هذه الطريقة كتابة المصنف إلى القرص، مع الحفاظ على جميع التغييرات. تأكد من تحديد دليل الإخراج بشكل صحيح.
