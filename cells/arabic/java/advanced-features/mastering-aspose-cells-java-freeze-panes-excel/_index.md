# كيفية استخدام Aspose.Cells Java لتجميد الأجزاء في Excel

## مقدمة
هل تواجه صعوبة في التنقل عبر جداول بيانات Excel الكبيرة؟ **ألواح التجميد** يُبقي الصفوف والأعمدة الأساسية مرئية، مما يجعل تحليل البيانات أكثر كفاءة. سيرشدك هذا البرنامج التعليمي خلال استخدام **Aspose.Cells لـ Java** لتجميد الألواح بشكل فعال.

### ما سوف تتعلمه
- كيفية تحميل مصنف Excel الحالي.
- تقنيات لتطبيق إعدادات لوحة التجميد.
- خطوات لحفظ المصنف المعدّل.

دعونا نبدأ بمراجعة المتطلبات الأساسية اللازمة لهذا البرنامج التعليمي.

## المتطلبات الأساسية
للمتابعة، تأكد من أن لديك:
- **مكتبة Aspose.Cells**:يجب أن يكون الإصدار 25.3 أو أحدث.
- معرفة أساسية ببرمجة Java و IDE مثل IntelliJ IDEA أو Eclipse.
- تم تثبيت Maven أو Gradle لإدارة التبعيات.

## إعداد Aspose.Cells لـ Java
قم بدمج المكتبة اللازمة في مشروعك باستخدام Maven أو Gradle.

### استخدام Maven
```xml
<dependency>
  <groupId>com.aspose</groupId>
  <artifactId>aspose-cells</artifactId>
  <version>25.3</version>
</dependency>
```

### استخدام Gradle
```gradle
implementation(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## دليل التنفيذ
### تحميل ملف Excel والوصول إليه
**ملخص**:يرشدك هذا القسم خلال عملية تحميل ملف Excel الحالي والوصول إلى ورقة العمل الأولى الخاصة به باستخدام Aspose.Cells Java.

#### الخطوة 1: استيراد الفئات المطلوبة
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.WorksheetCollection;
import com.aspose.cells.Worksheet;
```

#### الخطوة 2: تحميل المصنف
إنشاء `Workbook` مثال: بتوفير مسار ملف Excel. هذا ضروري للوصول إلى محتوياته ومعالجتها.
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "book.xls");
```
**توضيح**:المنشئ `new Workbook(filePath)` يقوم بتهيئة كائن المصنف، مما يسمح لنا بإجراء العمليات عليه.

#### الخطوة 3: الوصول إلى ورقة العمل الأولى
استرداد ورقة العمل الأولى من المصنف باستخدام مجموعة أوراق العمل الخاصة به. 
```java
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet worksheet = worksheets.get(0);
```
**توضيح**: ال `getWorksheets()` تقوم الطريقة بجلب جميع الأوراق والوصول إلى الفهرس `0` يعطينا الأول.

### تطبيق تجميد الأجزاء على ورقة العمل
**ملخص**:تعرف على كيفية إبقاء الصفوف والأعمدة المحددة مرئية أثناء التمرير عبر ورقة العمل الخاصة بك عن طريق تطبيق إعدادات تجميد الأجزاء.

#### الخطوة 4: ضبط أجزاء التجميد
قم بتطبيق ألواح التجميد باستخدام `freezePanes` طريقة.
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
worksheet.freezePanes(3, 2, 3, 2);
```
**توضيح**:المعلمات `(rowSplitIndex, columnSplitIndex, frozenRowCount, frozenColumnCount)` تحديد الصفوف والأعمدة التي تظل مرئية عند التمرير.

### حفظ ملف Excel المعدّل
**ملخص**:بعد تطبيق التغييرات، احفظ المصنف للاحتفاظ بتعديلاتك.

#### الخطوة 5: حفظ المصنف
اكتب المصنف المحدث مرة أخرى على القرص باستخدام المسار المحدد.
```java
workbook.save(outDir + "FreezePanes_out.xls");
```
**توضيح**: ال `save(filePath)` تلتزم الطريقة بجميع التغييرات التي تم إجراؤها على المصنف، مما يضمن تخزينها بشكل دائم في ملف Excel.
