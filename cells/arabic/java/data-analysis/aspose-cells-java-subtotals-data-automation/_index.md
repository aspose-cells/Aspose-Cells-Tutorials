# أتمتة إجماليات Excel الفرعية في Java باستخدام Aspose.Cells

- دليل خطوة بخطوة حول تطبيق المجاميع الفرعية برمجيًا

- أمثلة واقعية حيث تكون هذه الطريقة مفيدة

دعونا نستكشف كيفية الاستفادة من هذه القدرات في مشاريعك.

## إعداد Aspose.Cells لـ Java
لبدء استخدام Aspose.Cells لجافا في مشاريعك، عليك تضمينه في إعدادات البناء. إليك خطوات إعداده:
### مافن
أضف التبعية التالية في ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```
### جرادل
بالنسبة لأولئك الذين يستخدمون Gradle، قم بتضمين هذا في `build.gradle`:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```
### خطوات الحصول على الترخيص
يمكنك الحصول على ترخيص لـ Aspose.Cells لفتح الوظائف الكاملة:
- **نسخة تجريبية مجانية**:قم بتنزيل المكتبة واختبارها بإمكانيات محدودة.
- **رخصة مؤقتة**:يمكنك الحصول على هذا من موقع Aspose الإلكتروني إذا كنت بحاجة إلى أكثر مما هو متاح في النسخة التجريبية.
- **شراء**:شراء ترخيص تجاري للاستخدام غير المحدود.

## دليل التنفيذ
### ملخص
سيرشدك هذا القسم إلى كيفية تطبيق المجاميع الفرعية في جداول بيانات Excel باستخدام Aspose.Cells لجافا. تُعد المجاميع الفرعية أساسية لتلخيص البيانات في فئات، مما يُسهّل تحليل وتفسير مجموعات البيانات الكبيرة.
#### الخطوة 1: تحميل المصنف الخاص بك
ابدأ بتحميل المصنف الذي يحتوي على بياناتك:
```java
String sourceDir = "path/to/source/directory/";
Workbook workbook = new Workbook(sourceDir + "SampleSubtotal.xlsx");
```
#### الخطوة 2: الوصول إلى ورقة العمل
قم بالوصول إلى ورقة العمل التي تريد تطبيق المجاميع الفرعية عليها:
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```
#### الخطوة 3: تحديد مساحة الخلية للمجموع الفرعي
حدد نطاق الخلايا التي سيتم أخذها في الاعتبار للمجموع الفرعي:
```java
import com.aspose.cells.CellArea;
CellArea ca = CellArea.createCellArea("A2", "B11");
```
يركز هذا المثال على الأعمدة من A إلى B، والصفوف من 2 إلى 11.
#### الخطوة 4: تطبيق المجموع الفرعي
استخدم `subtotal` طريقة تطبيق المجاميع الفرعية:
```java
import com.aspose.cells.ConsolidationFunction;
worksheet.getCells().subtotal(ca, 0, ConsolidationFunction.SUM, new int[]{1}, true, false, true);
```
- **شرح المعلمات**:
  - **كاليفورنيا**:منطقة الخلية المحددة.
  - **0**:قم بالتجميع حسب العمود الأول في نطاقك (أ).
  - **دالة التوحيد.SUM**:تطبيق المجموع كدالة توحيد.
  - **int جديد[]{1}**:يحدد الأعمدة التي سيتم حساب المجموع الفرعي لها، وهنا العمود الثاني (ب).
  - **صحيح، خطأ، صحيح**:خيارات لمستويات المخطط التفصيلي والرؤية.
#### الخطوة 5: تحديد اتجاه ملخص المخطط التفصيلي
حدد المكان الذي يجب أن تظهر فيه صفوف الملخص:
```java
worksheet.getOutline().setSummaryRowBelow(true);
```
يؤدي هذا إلى وضع صف المجموع الفرعي أسفل كل مجموعة.
#### الخطوة 6: احفظ مصنفك
وأخيرًا، احفظ المصنف الخاص بك ليعكس التغييرات:
```java
String outputDir = "path/to/output/directory/";
workbook.save(outputDir + "ASubtotal_out.xlsx");
```
