### دليل التنفيذ
الآن بعد أن قمت بإعداد بيئتك، دعنا نركز على تنفيذ الإجماليات الفرعية.

#### نظرة عامة على إنشاء المجاميع الفرعية
يساعد التجميع الفرعي في تلخيص البيانات بتطبيق دالة تجميعية مثل المجموع أو المتوسط أو العد على نطاق. باستخدام Aspose.Cells، يتم ذلك برمجيًا باستخدام `subtotal` طريقة.

##### الخطوة 1: تهيئة المصنف ومجموعة الخلايا
ابدأ بتحميل المصنف الخاص بك والوصول إلى خلاياه:
```java
// تحميل ملف Excel
Workbook workbook = new Workbook(dataDir + "book1.xls");

// الوصول إلى مجموعة خلايا ورقة العمل الأولى
Cells cells = workbook.getWorksheets().get(0).getCells();
```

##### الخطوة 2: تحديد مساحة الخلية للمجموع الفرعي
حدد نطاق البيانات الذي تريد تطبيق المجموع الفرعي عليه:
```java
// حدد المنطقة من B3 إلى C19 (مؤشر قائم على 1)
CellArea ca = new CellArea();
ca.StartRow = 2; // الصف B3 في الفهرس القائم على الصفر
ca.EndRow = 18; // الصف C19 في الفهرس القائم على الصفر
ca.StartColumn = 1;
cac.EndColumn = 2;
```

##### الخطوة 3: تطبيق المجموع الفرعي
استخدم `subtotal` طريقة حساب وإدراج المجاميع الفرعية:
```java
// تطبيق المجموع الفرعي على العمود C (الفهرس 1) باستخدام دالة SUM
cells.subtotal(ca, 0, ConsolidationFunction.SUM, new int[] { 1 });
```
- **شرح المعلمات**:
  - `ca`:نطاق الخلايا.
  - `0`:يحدد موضع الصف الإجمالي.
  - `ConsolidationFunction.SUM`:يحدد الوظيفة التي سيتم تطبيقها (SUM في هذه الحالة).
  - `new int[]{1}`:فهرس العمود الذي يتم تطبيق المجموع الفرعي عليه.

##### الخطوة 4: الحفظ والإخراج
وأخيرًا، احفظ المصنف الخاص بك باستخدام المجاميع الفرعية الجديدة:
```java
// حفظ ملف Excel المعدل
dataDir + "CreatingSubtotals_out.xls";

// تأكيد النجاح
System.out.println("Process completed successfully");
```
