# تجميع حقول المحور في Excel باستخدام Aspose.Cells لـ Java

## مقدمة

هل تواجه صعوبة في تنظيم مجموعات البيانات الكبيرة داخل Excel؟ **Aspose.Cells لـ Java** يوفر حلاً فعّالاً لمعالجة بيانات Excel برمجياً. سيرشدك هذا البرنامج التعليمي خلال تحميل مصنف Excel، والوصول إلى أوراق العمل الخاصة به، وتعديل جداول البيانات المحورية، وتجميع الحقول حسب نطاقات التواريخ، وحفظ المصنف المعدّل.

من خلال اتباع هذا البرنامج التعليمي، سوف تصبح متمكنًا من:
- تحميل المصنفات باستخدام Aspose.Cells
- الوصول إلى جداول البيانات المحورية وتعديلها
- تجميع البيانات داخل حقول المحور
- تحديث بيانات الجدول المحوري وإعادة حسابها

لنبدأ بإعداد البيئة الخاصة بك لـ Aspose.Cells Java.

### المتطلبات الأساسية
قبل أن تبدأ، تأكد من أن لديك المتطلبات الأساسية التالية:

- **مجموعة تطوير جافا (JDK)**:تأكد من تثبيت JDK 8 أو أعلى على نظامك.
- **Aspose.Cells لـ Java**:قم بتضمين مكتبة Aspose.Cells في مشروعك باستخدام Maven أو Gradle:

**مافن**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**جرادل**
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

- **بيئة تطوير متكاملة**:استخدم بيئة تطوير متكاملة مثل IntelliJ IDEA أو Eclipse.
- **المعرفة الأساسية بلغة جافا**:يُفترض الإلمام بمفاهيم برمجة Java.

#### إعداد Aspose.Cells لـ Java
لبدء استخدام Aspose.Cells، اتبع الخطوات التالية:
1. **تثبيت Aspose.Cells** باستخدام Maven أو Gradle كما هو موضح أعلاه.

## دليل التنفيذ

### تحميل والوصول إلى المصنف
**ملخص**:ابدأ بتحميل مصنف موجود من الدليل المحدد لديك وقم بالوصول إلى أوراق العمل الخاصة به للتعامل مع البيانات.

- **تحميل المصنف**
  ```java
  import com.aspose.cells.Workbook;
  
  String dataDir = "YOUR_DATA_DIRECTORY";
  Workbook wb = new Workbook(dataDir + "/sampleGroupPivotFieldsInPivotTable.xlsx");
  ```

- **ورقة عمل الوصول**
  ```java
  import com.aspose.cells.Worksheet;

  Worksheet ws = wb.getWorksheets().get(1); // الوصول إلى ورقة العمل الثانية (يبدأ الفهرس من 0)
  ```

### الوصول إلى جدول محوري في ورقة العمل
**ملخص**:انتقل إلى الجدول المحوري ضمن ورقة العمل المحددة لإجراء المزيد من العمليات.

- **استرداد جدول محوري**
  ```java
  import com.aspose.cells.PivotTable;

  PivotTable pt = ws.getPivotTables().get(0); // الوصول إلى الجدول المحوري الأول في الورقة
  ```

### تعيين نطاق التاريخ للتجميع
**ملخص**:قم بتحديد نطاقات تاريخية محددة لتجميع البيانات بشكل فعال داخل حقول المحور الخاصة بك.

- **تحديد نطاق التاريخ**
  ```java
  import com.aspose.cells.DateTime;

  DateTime dtStart = new DateTime(2008, 1, 1); // تاريخ البدء: 1 يناير 2008
  DateTime dtEnd = new DateTime(2008, 9, 5);   // تاريخ الانتهاء: 5 سبتمبر 2008
  ```

### تحديد أنواع المجموعات لحقول المحور
**ملخص**:حدد أنواع التجميع التي سيتم تطبيقها على حقول المحور الخاصة بك.

- **تحديد أنواع المجموعة**
  ```java
  import com.aspose.cells.PivotGroupByType;
  import java.util.ArrayList;

  ArrayList groupTypeList = new ArrayList();
  groupTypeList.add(PivotGroupByType.MONTHS);
  groupTypeList.add(PivotGroupByType.QUARTERS);
  ```

### تطبيق التجميع على حقل المحور
**ملخص**:قم بتنفيذ عملية التجميع من خلال تطبيق الإعدادات المحددة على حقل المحور الخاص بك.

- **تطبيق التجميع**
  ```java
  pt.setManualGroupField(0, dtStart, dtEnd, groupTypeList, 1); // تطبيق التجميع باستخدام المعلمات المحددة
  ```

### تحديث بيانات جدول المحور وحسابها
**ملخص**:تأكد من أن بيانات جدول المحور تعكس التغييرات التي أجريتها عن طريق تحديثها وإعادة حسابها.

- **تحديث وإعادة الحساب**
  ```java
  pt.setRefreshDataFlag(true);
  pt.refreshData();
  pt.calculateData();
  pt.setRefreshDataFlag(false); // إعادة تعيين العلم بعد العملية
  ```

### حفظ المصنف في دليل الإخراج
**ملخص**:بمجرد الانتهاء من التعديلات، احفظ المصنف الخاص بك في دليل الإخراج المخصص.

- **حفظ المصنف المعدل**
  ```java
  String outDir = "YOUR_OUTPUT_DIRECTORY";
  wb.save(outDir + "/outputGroupPivotFieldsInPivotTable.xlsx");
  ```
