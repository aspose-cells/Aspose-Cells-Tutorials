## دليل التنفيذ

في هذا القسم، سنشرح كيفية إنشاء مرشح مخصص يحدد الصفوف التي تحتوي على نص معين باستخدام Aspose.Cells لـ Java.

### الخطوة 1: تحميل المصنف

ابدأ بتحميل ملف Excel الحالي. هذا هو مصدر بياناتك.

```java
Workbook workbook = new Workbook(srcDir + "sourseSampleCountryNames.xlsx");
```

### الخطوة 2: الوصول إلى ورقة العمل

قم بالوصول إلى ورقة العمل التي تحتوي على البيانات التي ترغب في تصفيتها:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

### الخطوة 3: إعداد التصفية التلقائية

إنشاء `AutoFilter` بتحديد نطاق. هذا يُحدد منطقة الورقة التي سيتم تصفيتها:

```java
worksheet.getAutoFilter().setRange("A1:A18");
```

### الخطوة 4: تطبيق الفلتر المخصص

استخدم `custom()` طريقة لتطبيق مرشح يبحث عن الصفوف التي تحتوي على نص معين، في هذه الحالة "Ba":

```java
worksheet.getAutoFilter().custom(0, FilterOperatorType.CONTAINS, "Ba");
```

**توضيح**: 
- المعلمة الأولى (`0`) يحدد فهرس العمود.
- `FilterOperatorType.CONTAINS` هل تم تطبيق منطق التصفية؟
- `"Ba"` هي السلسلة التي تبحث عنها ضمن النطاق المحدد.

### الخطوة 5: التحديث والحفظ

بعد إعداد الفلتر، قم بتحديثه لتطبيق التغييرات وحفظ المصنف الخاص بك:

```java
worksheet.getAutoFilter().refresh();
workbook.save(outDir + "outSourseSampleCountryNames.xlsx");
```

**نصائح استكشاف الأخطاء وإصلاحها**:إذا واجهت مشكلات مع مسارات الملفات أو نطاقات البيانات، فتأكد من تحديدها بشكل صحيح بالنسبة إلى بنية دليل مشروعك.
