# كيفية فك تجميع الصفوف والأعمدة في Excel باستخدام Aspose.Cells Java

## مقدمة

هل تواجه صعوبة في التعامل مع البيانات المجمعة في جداول بيانات Excel، مما يُعقّد عملية التحليل أو العرض؟ يُعد تجميع الصفوف والأعمدة ميزة شائعة في Excel لتبسيط العرض، ولكنه قد يتطلب أحيانًا إعادة ترتيبها. يرشدك هذا البرنامج التعليمي إلى كيفية فك تجميع هذه الصفوف والأعمدة بسهولة باستخدام Aspose.Cells لـ Java.

بحلول نهاية هذا الدليل، سوف تتعلم:
- كيفية إعداد البيئة الخاصة بك باستخدام Aspose.Cells.
- تعليمات خطوة بخطوة حول إلغاء تجميع الصفوف والأعمدة في ملفات Excel.
- التطبيقات العملية لهذه الوظائف.

دعونا نلقي نظرة على المتطلبات الأساسية اللازمة قبل البدء.

## المتطلبات الأساسية

قبل أن نبدأ في الترميز، تأكد من أن لديك ما يلي:

- **المكتبات المطلوبة**:يتطلب Aspose.Cells إصدار Java 25.3 أو إصدار أحدث.
- **إعداد البيئة**:فهم أساسي لـJava وIDE مثل IntelliJ IDEA أو Eclipse.
- **متطلبات المعرفة**:المعرفة بعمليات Excel وبرمجة Java.

## إعداد Aspose.Cells لـ Java

### معلومات التثبيت

لتضمين Aspose.Cells في مشروعك، اتبع الخطوات التالية:

**مافن**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**جرادل**
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

### خطوات الحصول على الترخيص

1. **نسخة تجريبية مجانية**:ابدأ بالتجربة المجانية لاستكشاف ميزات Aspose.Cells.
2. **رخصة مؤقتة**:اطلب ترخيصًا مؤقتًا للوصول إلى الميزات الكاملة أثناء التطوير.
3. **شراء**:فكر في الشراء إذا كنت بحاجة إلى خدمة طويلة الأمد دون انقطاع.

بمجرد التثبيت والترخيص، قم بتهيئة مشروعك عن طريق استيراد الفئات الضرورية:

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;
import com.aspose.cells.Cells;
```

## دليل التنفيذ

### فك تجميع الصفوف في Excel

يساعدك فصل الصفوف عن بعضها على العودة إلى التخطيط الأصلي إذا تم تجميعها لإجراء تحليل مفصل. اتبع هذه الخطوات لفصل الصفوف عن بعضها.

#### قم بتحميل مصنفك وورقة عملك
أولاً، قم بتحميل المصنف الخاص بك من ملف:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "BookStyles.xls");
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### إلغاء تجميع الصفوف
حدد الصفوف التي ترغب في إلغاء تجميعها وتنفيذها:

```java
Cells cells = worksheet.getCells();
cells.ungroupRows(0, 5); // إلغاء تجميع الصفوف من الفهرس 0 إلى 5
```

### فك تجميع الأعمدة في Excel
وبالمثل، قم بإلغاء تجميع الأعمدة إذا تم تجميعها لتحسين إدارة البيانات أو عرضها.

#### قم بتحميل مصنفك وورقة عملك
تأكد من تحميل المصنف الخاص بك:

```java
Workbook workbook = new Workbook(dataDir + "BookStyles.xls");
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### إلغاء تجميع الأعمدة
حدد نطاق مؤشرات الأعمدة التي تريد إلغاء تجميعها:

```java
Cells cells = worksheet.getCells();
cells.ungroupColumns(0, 2); // إلغاء تجميع الأعمدة من الفهرس 0 إلى 2
```

### احفظ التغييرات
بعد إجراء التعديلات، احفظ المصنف الخاص بك:

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "UngroupingRows_out.xls"); // للصفوف
workbook.save(outDir + "UngroupingColumns_out.xls"); // للأعمدة
```
