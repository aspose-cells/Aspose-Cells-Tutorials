## إعداد Aspose.Cells لـ Java

للبدء، قم بدمج مكتبة Aspose.Cells في مشروعك باستخدام Maven أو Gradle.

**مافن:**

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**جرادل:**

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

// إنشاء كائن مصنف يحتوي على بيانات عينة

## دليل التنفيذ

يقوم هذا القسم بتفصيل الخطوات اللازمة لتنفيذ مرشح "يبدأ بـ" باستخدام Aspose.Cells لـ Java.

### الوصول إلى ورقة العمل

أولاً، قم بالوصول إلى ورقة العمل التي تريد تطبيق التصفية التلقائية عليها:

```java
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.getWorksheets().get(0);
```

### إعداد نطاق التصفية التلقائية

حدّد نطاق الخلايا التي سيُطبّق عليها الفلتر. يُساعد هذا على تضييق نطاق البيانات بكفاءة.

```java
// إنشاء مرشح تلقائي عن طريق تحديد نطاق الخلية
worksheet.getAutoFilter().setRange("A1:A18");
```

### تطبيق مرشح "يبدأ بـ"

لتصفية الصفوف استنادًا إلى سلسلة تبدأ بأحرف معينة، استخدم `custom` طريقة.

```java
// تهيئة الفلتر للصفوف التي تبدأ بسلسلة "Ba"
worksheet.getAutoFilter().custom(0, FilterOperatorType.BEGINS_WITH, "Ba");
```

- **المعلمات موضحة:**
  - المعلمة الأولى (`0`) يمثل مؤشر العمود.
  - `BEGINS_WITH` يحدد نوع العملية.
  - `"Ba"` هو السلسلة المستخدمة للتصفية.

### تحديث التصفية التلقائية

بعد إعداد معايير التصفية، قم بتحديثها لتطبيق التغييرات:

```java
// قم بتحديث الفلتر لإظهار/إخفاء الصفوف المفلترة
worksheet.getAutoFilter().refresh();
```

### حفظ عملك

أخيرًا، احفظ ملف Excel المعدّل للحفاظ على التغييرات:

```java
// حفظ ملف Excel المعدل
workbook.save(outDir + "outSourceSampleCountryNames.xlsx");
```
