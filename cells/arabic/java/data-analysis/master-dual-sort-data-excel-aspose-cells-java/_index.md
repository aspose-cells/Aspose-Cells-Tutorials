# فرز البيانات ثنائيًا بكفاءة في Excel باستخدام Aspose.Cells لـ Java: دليل خطوة بخطوة

## مقدمة

يتطلب العمل مع مجموعات بيانات كبيرة في Excel غالبًا فرز البيانات بكفاءة بناءً على معايير متعددة. قد يكون الفرز اليدوي مُرهقًا، خاصةً مع ازدياد تعقيد مجموعة البيانات. **Aspose.Cells لـ Java** يوفر إمكانيات فرز فعّالة مباشرةً داخل تطبيقات جافا. يستكشف هذا البرنامج التعليمي تطبيق وظيفة الفرز المزدوج باستخدام Aspose.Cells لجافا، وهي ميزة تتيح فرزًا سلسًا متعدد المفاتيح.

### ما سوف تتعلمه
- قم بتهيئة كائن Workbook وتكوينه باستخدام Aspose.Cells.
- إعداد معايير الفرز الأولية والثانوية.
- تطبيق الفرز ضمن نطاق محدد من الخلايا.
- احفظ البيانات المصنفة مرة أخرى في ملف Excel.

دعونا نبدأ بتحديد المتطلبات الأساسية لهذا البرنامج التعليمي.

## إعداد Aspose.Cells لـ Java

لاستخدام Aspose.Cells لـ Java، أضف المكتبة إلى مشروعك:

أضف التبعية التالية في ملفك `pom.xml` ملف:

## دليل التنفيذ

### تهيئة المصنف وDataSorter
لفرز البيانات في Excel باستخدام Aspose.Cells لـ Java، ابدأ بإنشاء `Workbook` الكائن والوصول إليه `DataSorter`.

#### الخطوة 1: إنشاء كائن مصنف
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "book1.xls");
```
يقوم هذا الكود بتهيئة المصنف من ملف Excel الموجود في الدليل المحدد.

#### الخطوة 2: الوصول إلى DataSorter
```java
DataSorter sorter = workbook.getDataSorter();
```
ال `DataSorter` يعد الكائن ضروريًا لإعداد معلمات الفرز وتنفيذ عملية الفرز.

### تكوين الفرز الأساسي
قم بتعيين مفتاح الفرز الأساسي الخاص بك لتحديد كيفية فرز البيانات أولاً.

#### الخطوة 1: تعيين ترتيب المفتاح الأول
```java
sorter.setOrder1(SortOrder.DESCENDING);
```
`DESCENDING` يشير إلى أن العمود الأول سيتم فرزه بترتيب تنازلي.

#### الخطوة 2: تحديد فهرس عمود المفتاح الأول
```java
sorter.setKey1(0); // فرز حسب العمود الأول (العمود أ).
```

### تكوين الفرز الثانوي
أضف معيار فرز ثانويًا لتحقيق تحكم أكثر دقة في تنظيم بياناتك.

#### الخطوة 1: تعيين ترتيب المفتاح الثاني
```java
sorter.setOrder2(SortOrder.ASCENDING);
```
ال `ASCENDING` يقوم بترتيب المفتاح الثاني بترتيب تصاعدي، مكملاً للفرز الأساسي.

#### الخطوة 2: تحديد فهرس عمود المفتاح الثاني
```java
sorter.setKey2(1); // فرز حسب العمود الثاني (العمود ب).
```

### فرز البيانات في نطاق محدد
قم بتحديد النطاق الدقيق الذي تريد تطبيق معايير الفرز المزدوج ضمنه.

#### الخطوة 1: تحديد منطقة الخلية للفرز
```java
CellArea cellArea = new CellArea();
cellArea.StartRow = 0;
cellArea.StartColumn = 0;
cellArea.EndRow = 13;
cellArea.EndColumn = 1; // قم بضبط هذه المؤشرات استنادًا إلى نطاق بياناتك.
```

#### الخطوة 2: تنفيذ عملية الفرز
```java
sorter.sort(workbook.getWorksheets().get(0).getCells(), cellArea);
```
يقوم هذا الكود بفرز المنطقة المحددة في ورقة العمل الأولى من المصنف.

### حفظ المصنف المرتب
وأخيرًا، احفظ بياناتك المصنفة مرة أخرى في ملف Excel للحفاظ على التغييرات.

#### الخطوة 1: تحديد دليل الإخراج
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: حفظ المصنف
```java
workbook.save(outDir + "DataSorting_out.xls");
```
يقوم هذا الأمر بحفظ المصنف المعدل في دليل الإخراج المحدد.
