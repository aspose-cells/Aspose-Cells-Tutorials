## كيف تقوم بتحميل دفتر عمل Excel في Java؟

فئة `Workbook` تمثل ملف Excel محملاً في الذاكرة، وتوفر الوصول إلى أوراق العمل، الخلايا، والمخططات.

حمّل دفتر العمل باستخدام `new Workbook("path/to/file.xlsx")` – هذه السطر الواحد يقرأ كامل جدول البيانات إلى الذاكرة، مما يمنحك وصولًا برمجيًا إلى جميع أوراق العمل، الخلايا، والمخططات المدمجة. Aspose.Cells يكتشف تنسيق الملف تلقائيًا، لذا لا تحتاج إلى تحديد XLSX أو XLS أو CSV صراحةً.

## تحميل دفتر العمل من ملف

**نظرة عامة:**  
الخطوة الأولى هي تحميل دفتر عمل Excel. هذا يجهز البيئة للوصول إلى المخططات.

```java
import com.aspose.cells.Workbook;
// Load an Excel workbook from a specified directory.
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

**شرح:**  
- `Workbook` هي الكائن الأعلى مستوى الذي يمثل ملف Excel واحد في الذاكرة.  
- قدّم المسار الكامل لملف Excel الخاص بك عبر المتغير `dataDir` أو مسار مطلق.

## كيف تصل إلى ورقة عمل ومخطط محددين؟

`Worksheet` تمثل ورقة واحدة داخل دفتر العمل، تحتوي على صفوف، أعمدة، وكائنات مدمجة.  
`Chart` تمثل تمثيلًا رسوميًا للبيانات على ورقة العمل، يمكن تصييره أو تصديره.

احصل على ورقة العمل باستخدام `workbook.getWorksheets().get(0)` ثم استدعِ `getCharts().get(0)` للحصول على أول كائن مخطط – هذا النهج المباشر يعمل لأي فهرس مخطط تحتاجه. API تُعيد مثيل `Chart` جاهز للتصيير أو استخراج البيانات.

## الوصول إلى ورقة العمل والمخطط

**نظرة عامة:**  
بعد التحميل، وصول إلى ورقة العمل والمخطط المحددين اللذين تريد تحويلهما.

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;
// Access the first worksheet and its first chart.
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```

**شرح:**  
- `worksheet` هو كائن من النوع `Worksheet`.  
- `chart` يتم استرجاعه من مجموعة المخططات في ورقة العمل.

## كيف تقوم بتحويل مخطط إلى صورة SVG؟

`ImageOrPrintOptions` تُعرّف إعدادات التصيير مثل صيغة الإخراج، الدقة، والجودة لتحويل المخططات أو أوراق العمل إلى ملفات صور.

أنشئ مثيلًا من `ImageOrPrintOptions`، عيّن `setSaveFormat(SaveFormat.SVG)`، ثم استدعِ `chart.toImage(options, "output.svg")`. هذه الدعوة ذات السطر الواحد تكتب ملف SVG متوافق بالكامل يحافظ على الألوان، الخطوط، وملصقات البيانات كما تظهر في Excel.

## تحويل المخطط إلى صورة SVG

**نظرة عامة:**  
الخطوة الأخيرة تتضمن تحويل المخطط إلى صورة SVG لعرض عالي الجودة.

```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.SaveFormat;
// Convert and save the chart as an SVG image.
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setSaveFormat(SaveFormat.SVG);
String outDir = "YOUR_OUTPUT_DIRECTORY";
chart.toImage(outDir + "CCToImageinSVGFormat_out.svg", options);
```

**شرح:**  
- `ImageOrPrintOptions` يضبط كيفية حفظ المخطط.  
- تعيين الصيغة إلى SVG يخبر Aspose.Cells بإنشاء رسم متجه.  
- يمكن تضمين الملف الناتج مباشرةً في خلفيات HTML أو CSS.
