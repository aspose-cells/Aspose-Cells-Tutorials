## دليل التنفيذ
سنقوم بتقسيم العملية إلى ثلاث ميزات رئيسية: تحميل ملف Excel، والوصول إلى مخططات محددة، وتصديرها إلى ملف PDF بأحجام صفحات مخصصة.

### تحميل ملف Excel النموذجي
#### ملخص
تحميل ملف Excel هو الخطوة الأولى للتعامل مع محتوياته. سنستخدم Aspose.Cells لـ Java لتحميل مصنف يحتوي على بيانات مخطط بياني.

#### التنفيذ خطوة بخطوة
**1. استيراد الفئات الضرورية**
```java
import com.aspose.cells.Workbook;
```

**2. قم بتحميل المصنف**
يستبدل `"YOUR_DATA_DIRECTORY"` مع مسار الملف الفعلي الخاص بك:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // اضبط هذا على الدليل الخاص بك
Workbook wb = new Workbook(dataDir + "/sampleCreateChartPDFWithDesiredPageSize.xlsx");
```
يؤدي مقتطف التعليمات البرمجية هذا إلى تحميل مصنف Excel، مما يجعله جاهزًا للعمليات الإضافية.

### ورقة عمل ومخطط Access First
#### ملخص
يتيح لك الوصول إلى أوراق العمل والمخططات المحددة داخل مصنف التعامل معها أو تصديرها بشكل فردي.

#### التنفيذ خطوة بخطوة
**1. استرداد ورقة العمل**
```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;

Worksheet ws = wb.getWorksheets().get(0);
```
يقوم الكود أعلاه باسترجاع ورقة العمل الأولى من المصنف الخاص بك.

**2. الوصول إلى الرسم البياني**
```java
Chart ch = ws.getCharts().get(0);
```
الآن، `ch` يحتوي على مرجع للمخطط الأول في ورقة العمل المحددة.

### إنشاء مخطط PDF بحجم الصفحة المطلوب
#### ملخص
يعد تحويل المخططات إلى ملفات PDF ذات أحجام صفحات مخصصة أمرًا ضروريًا للعروض التقديمية والتقارير التي تتطلب تنسيقًا محددًا.

#### التنفيذ خطوة بخطوة
**1. تحديد دليل الإخراج**
```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // اضبط هذا على مسار الإخراج المطلوب
```

**2. تصدير الرسم البياني بصيغة PDF**
هنا نقوم بتحديد الأبعاد (7×7 بوصة) ومحاذاة الصفحة.
```java
import com.aspose.cells.PageLayoutAlignmentType;

ch.toPdf(outDir + "/outputCreateChartPDFWithDesiredPageSize.pdf", 7, 7,
         PageLayoutAlignmentType.CENTER, PageLayoutAlignmentType.CENTER);
```
تؤدي هذه الخطوة إلى تحويل المخطط إلى ملف PDF بحجم الصفحة والمحاذاة المطلوبة.
