# Altere a direção de etiqueta de gráfico no Excel usando Aspose.Cells 

Este exemplo demonstra como modificar a direção de etiquetas de marcação de gráfico em uma folha de trabalho do Excel usando Aspose.Cells para .NET. Ao ajustar a direção de texto das etiquetas de marcação de eixo de categoria, você pode melhorar a legibilidade dos dados do gráfico. O código carrega um livro de trabalho existente do Excel, acessa a primeira folha de trabalho, recupera o gráfico, define a direção do rótulo de marcação para horizontal e, em seguida, salva o livro de trabalho modificado. Esta abordagem garante um controle preciso sobre a orientação de rótulo de gráfico em seus aplicativos .NET.

## Instruções de instalação  
Para executar este exemplo, siga estes passos:  

1. Instalar Aspose.Cells para .NET  
   * Descarregar de [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   * Ou instalar através de NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Configure seu projeto  
   * Crie um novo projeto .NET.  
   * Adicionar uma referência a Aspose.Cells.dll.  

3. Executa o código  
   * Substitui`"O seu diretório de documentos"`com o caminho do arquivo real.  
   * Execute o programa para atualizar a direção de etiqueta de marca no gráfico.
