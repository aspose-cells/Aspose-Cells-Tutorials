# Alterar a direção do rótulo da escala do gráfico no Excel usando Aspose.Cells 

Este exemplo demonstra como modificar a direção dos rótulos de marcação de gráfico em uma planilha do Excel usando o Aspose.Cells para .NET. Ajustando a direção do texto dos rótulos de marcação do eixo de categorias, você pode melhorar a legibilidade dos dados do gráfico. O código carrega uma pasta de trabalho do Excel existente, acessa a primeira planilha, recupera o gráfico, define a direção do rótulo de marcação como horizontal e, em seguida, salva a pasta de trabalho modificada. Essa abordagem garante um controle preciso sobre a orientação dos rótulos do gráfico em seus aplicativos .NET.

## Instruções de instalação  
Para executar este exemplo, siga estas etapas:  

1. Instalar Aspose.Cells para .NET  
   - Baixar de [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   - Ou instale via NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Configure seu projeto  
   - Crie um novo projeto .NET.  
   - Adicione uma referência a Aspose.Cells.dll.  

3. Execute o código  
   - Substituir `"Your Document Directory"` com o caminho real do arquivo.  
   - Execute o programa para atualizar a direção do rótulo do tique no gráfico.
