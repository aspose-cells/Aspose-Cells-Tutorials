# Crie um gráfico do Excel com Aspose.Cells para .NET
Este exemplo demonstra como criar um livro de trabalho do Excel, preenchê-lo com dados de amostra e gerar um gráfico de coluna usando Aspose.Cells para .NET. O guia passo a passo abrange a criação de um livro de trabalho, adição de dados, configuração de rótulos de categoria e configuração do gráfico. Aspose.Cells fornece uma API poderosa para automatizar a criação e manipulação de arquivos do Excel em aplicativos .NET, facilitando a geração de gráficos dinâmicos programaticamente.

## Instruções de instalação  

Para usar Aspose.Cells para .NET, siga estes passos:  

1. Instalar através do Gerente de Pacotes NuGet:  
   * Abra o seu projeto .NET no Visual Studio.  
   * Vá para Ferramentas → Gerente de Pacotes NuGet → Gerencie Pacotes NuGet para Solução.  
   * Procure Aspose.Cells e instale a versão mais recente.  

   Alternativamente, instale usando o Console do Gestor de Pacotes:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Descarregar do site Aspose:  
   * Visite a página oficial de download: [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   * Descarregue o DLL e adicione uma referência a ele no seu projeto.  

3. Licença de aplicação (opcional):  
   * Obter uma licença de [Aspire a compra](https://purchase.aspose.com/buy)ou solicitar um [licença temporária](https://purchase.aspose.com/temporary-license/)\- Não .  
   * Aplique a licença no seu código:  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
