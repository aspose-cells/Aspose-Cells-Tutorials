# Crie um gráfico do Excel com Aspose.Cells para .NET
Este exemplo demonstra como criar uma pasta de trabalho do Excel, preenchê-la com dados de exemplo e gerar um gráfico de colunas usando o Aspose.Cells para .NET. O guia passo a passo aborda a criação de uma pasta de trabalho, a adição de dados, a definição de rótulos de categoria e a configuração do gráfico. O Aspose.Cells fornece uma API poderosa para automatizar a criação e a manipulação de arquivos do Excel em aplicativos .NET, facilitando a geração de gráficos dinâmicos programaticamente.

## Instruções de instalação  

Para usar o Aspose.Cells para .NET, siga estas etapas:  

1. Instalar via Gerenciador de Pacotes NuGet:  
   - Abra seu projeto .NET no Visual Studio.  
   - Acesse Ferramentas → Gerenciador de Pacotes NuGet → Gerenciar Pacotes NuGet para Solução.  
   - Procure por Aspose.Cells e instale a versão mais recente.  

   Como alternativa, instale-o usando o Console do Gerenciador de Pacotes:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Baixe do site da Aspose:  
   - Visite a página oficial de download: [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   - Baixe a DLL e adicione uma referência a ela no seu projeto.  

3. Aplicar licença (opcional):  
   - Obtenha uma licença de [Aspose Compra](https://purchase.aspose.com/buy) ou solicitar um [licença temporária](https://purchase.aspose.com/temporary-license/).  
   - Aplique a licença no seu código:  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
