# Criar PDF a partir de um gráfico do Excel com tamanho de página especificado

Exportar gráficos do Excel para um arquivo PDF com um layout de página preciso é um requisito crucial para relatórios e visualização de dados. Este tutorial demonstra como usar o Aspose.Cells para .NET para gerar um PDF a partir de um gráfico do Excel, especificando o tamanho e o alinhamento da página. Você aprenderá a carregar um arquivo do Excel, acessar uma planilha e seu gráfico e convertê-lo em um arquivo PDF bem formatado.

## Instruções de instalação  
Para usar o Aspose.Cells para .NET, siga estas etapas:  

1. Instalar via NuGet:  
   Abra o Gerenciador de Pacotes NuGet no Visual Studio e execute:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Baixe a DLL:  
   Obtenha a versão mais recente do [Página de download do Aspose.Cells para .NET](https://releases.aspose.com/cells/net/).  

3. Incluir o namespace:  
   Adicione o seguinte no seu arquivo C#:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
