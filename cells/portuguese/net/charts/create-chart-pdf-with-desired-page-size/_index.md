# Crie PDF a partir do gráfico do Excel com tamanho especificado da página

Exportar gráficos do Excel para um arquivo PDF com um layout de página preciso é um requisito crucial para relatórios e visualização de dados. Este tutorial demonstra como usar Aspose.Cells para .NET para gerar um PDF a partir de um gráfico Excel, especificando o tamanho e alinhamento da página. Você aprenderá como carregar um arquivo Excel, acessar uma folha de cálculo e seu gráfico e convertê-lo em um arquivo PDF bem formatado.

## Instruções de instalação  
Para usar Aspose.Cells para .NET, siga estes passos:  

1. Instalação através do NuGet:  
   Abra o Gerente de Pacotes NuGet no Visual Studio e execute:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Descarregar o DLL:  
   Obtenha a versão mais recente do [Aspose.Cells para página de download .NET](https://releases.aspose.com/cells/net/)\- Não .  

3. Incluir o espaço de nomes:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
