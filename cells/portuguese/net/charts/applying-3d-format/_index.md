# Crie e formate gráficos 3D no Excel usando Aspose.Cells 

Aspose.Cells para .NET é uma biblioteca poderosa que permite aos desenvolvedores criar, manipular e formatar arquivos do Excel programaticamente. Este exemplo demonstra como criar uma nova pasta de trabalho do Excel, inserir dados em uma planilha, adicionar um gráfico e aplicar formatação 3D à série de gráficos. Seguindo este guia, você aprenderá a configurar dados de categoria, configurar propriedades de gráfico e aprimorar a visualização com efeitos 3D usando C#.

## Instruções de instalação  
Para usar o Aspose.Cells para .NET, siga estas etapas:  

1. Instalar via Gerenciador de Pacotes NuGet:  
   - Abra o Visual Studio e vá para Ferramentas > Gerenciador de Pacotes NuGet > Gerenciar Pacotes NuGet para Solução.  
   - Procure por Aspose.Cells e instale-o.  

   Ou instale por meio do Console do Gerenciador de Pacotes NuGet:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Baixe a versão mais recente:  
   - Obtenha a versão mais recente no site da Aspose:  
     [Baixe Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  

3. Adicione referência em seu projeto:  
   - Após a instalação, inclua o seguinte namespace no seu código C#:  
     ```csharp
     using Aspose.Cells;
     using Aspose.Cells.Charts;
     using Aspose.Cells.Drawing;
     using System;
     using System.Drawing;
     ```
