# Redimensionar e reposicionar gráfico no Excel usando Aspose.Cells 
Aspose.Cells para .NET fornece uma API poderosa para trabalhar com arquivos do Excel programaticamente. Este exemplo demonstra como redimensionar e reposicionar um gráfico em uma pasta de trabalho do Excel usando o Aspose.Cells. Seja para ajustar as dimensões do gráfico ou movê-lo para uma posição específica, este tutorial o guia pelo processo com código C# fácil de entender. Utilizando o Aspose.Cells, você pode manipular gráficos do Excel perfeitamente, garantindo que seus relatórios e apresentações de dados visuais sejam formatados com precisão.

## Instruções de instalação:  

Para usar o Aspose.Cells para .NET, siga estas etapas:  

1. Instalar via Gerenciador de Pacotes NuGet  
- Abra o Visual Studio.  
- Navegue até Ferramentas → Gerenciador de Pacotes NuGet → Gerenciar Pacotes NuGet para Solução.  
- Pesquise por Aspose.Cells.  
- Clique em Instalar para adicionar o pacote ao seu projeto.  

Como alternativa, instale-o usando o Console do Gerenciador de Pacotes NuGet:  

```powershell
Install-Package Aspose.Cells
```

2. Adicione namespace ao seu código C#  
```csharp
using Aspose.Cells;
```

3. Ativar licença (opcional, para recursos completos)  
Se você tiver uma licença, aplique-a usando o seguinte código:  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
