# Obter legenda do gráfico do arquivo ODS usando Aspose.Cells  
Este exemplo demonstra como usar o Aspose.Cells para .NET para carregar um arquivo ODS, acessar um gráfico específico em uma planilha e recuperar seu subtítulo. Seguindo este guia, os desenvolvedores podem ler e extrair informações de gráficos de planilhas ODS com eficiência usando C#. O exemplo fornece uma abordagem clara e passo a passo para trabalhar com gráficos em arquivos Excel, garantindo uma implementação tranquila em aplicativos .NET.

## Instruções de instalação:  
Para usar este exemplo, siga estas etapas:  

1. Instalar o Aspose.Cells para .NET via NuGet:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Adicione o namespace necessário no seu projeto C#:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Execute o código fornecido para carregar um arquivo ODS, acessar um gráfico e imprimir sua legenda no console.
