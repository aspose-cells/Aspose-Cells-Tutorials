# Aplicar cores e fundos no Excel usando Aspose.Cells
  
Aspose.Cells para .NET permite que os desenvolvedores formatem programaticamente as folhas de trabalho do Excel aplicando cores e estilos de fundo às células. Este exemplo demonstra como criar um livro de trabalho do Excel, adicionar uma folha de trabalho, personalizar os estilos de células com diferentes cores de fundo e fundo e salvar o arquivo no formato Excel 97-2003.

## Instruções de instalação  
Para começar com Aspose.Cells para .NET, siga estes passos:  

1. Descarregar e instalar Aspose.Cells para .NET  
   * Baixe a versão mais recente de [Aspose.Cells Descarga](https://releases.aspose.com/cells/net/)\- Não .  
   * Instala-lo usando NuGet no Visual Studio:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

2. Adicionar Referências de Espaço de Nome  
   In your C# project, include:  
   ```csharp
   using Aspose.Cells;
   using System.Drawing;
   ```  

3. Aplicar cores e fundos para células do Excel  
   * Criar um novo .`Manual de trabalho`Objeto.  
   * Retirando o `Ficha de trabalho`e aplicar estilos personalizados a células específicas.  
   * Salvar o manual de trabalho modificado em `. xls`formato.
