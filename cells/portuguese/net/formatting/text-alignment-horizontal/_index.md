# Alinhamento de texto horizontal no Excel com Aspose.Cells

Este exemplo demonstra como usar o Aspose.Cells para .NET para criar um arquivo Excel e definir o alinhamento horizontal do texto em uma célula específica. O tutorial orienta você na inicialização de uma pasta de trabalho, na modificação das propriedades de uma célula e no salvamento do arquivo no formato Excel 97-2003. Perfeito para desenvolvedores que buscam aprimorar suas habilidades de manipulação de arquivos Excel usando o Aspose.Cells.

## Instruções de instalação
1. Baixe e instale o Aspose.Cells para .NET a partir do [Página de lançamentos do Aspose](https://releases.aspose.com/cells/net/).  
2. Adicione uma referência ao Aspose.Cells.dll no seu projeto C#.  
3. Use o Gerenciador de Pacotes NuGet para instalar o Aspose.Cells:  
   ```sh
   Install-Package Aspose.Cells
   ```
4. Importe os namespaces necessários no seu arquivo C#:  
   ```csharp
   using Aspose.Cells;
   ```
