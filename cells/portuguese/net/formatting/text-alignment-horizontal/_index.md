# Alinhamento horizontal de texto no Excel com Aspose.Cells

Este exemplo demonstra como usar Aspose.Cells para .NET para criar um arquivo Excel e definir o alinhamento horizontal de texto em uma célula específica. O tutorial guia você através da inicialização de um livro de trabalho, modificação das propriedades de uma célula e armazenamento do arquivo no formato Excel 97-2003. Perfeito para desenvolvedores que desejam melhorar suas habilidades de manipulação de arquivo Excel usando Aspose.Cells.

## Instruções de instalação
1. Descarregar e instalar Aspose.Cells para .NET do [Aspose libertações página](https://releases.aspose.com/cells/net/)\- Não .  
2. Add a reference to Aspose.Cells.dll in your C# project.  
3. Use o Gerente de Pacotes NuGet para instalar Aspose.Cells:  
   ```sh
   Install-Package Aspose.Cells
   ```
4. Import the required namespaces in your C# file:  
   ```csharp
   using Aspose.Cells;
   ```
