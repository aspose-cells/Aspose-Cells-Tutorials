## Instruções de instalação  
Para usar Aspose.Cells para .NET no seu projeto, siga estes passos:  

1. Instalação através de NuGet  
   Abra o Console do Gestor de Pacotes no Visual Studio e execute:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Descarregar o DLL  
   Alternativamente, baixe a versão mais recente de [Aspose.Cells para Downloads .NET](https://releases.aspose.com/cells/net/)e referir o DLL no seu projeto.  

3. Adicione uma referência no seu projeto  
   * No Visual Studio, clique com o botão direito no Solution Explorer.  
   * Clique em Adicionar Referência > Explorar > Selecionar o DLL baixado.  
   * Clique em OK para adicionar a referência.  

4. Comece a usar Aspose.Cells  
   Adicione o seguinte espaço de nomes no início do seu arquivo C#:  
   ```csharp
   using Aspose.Cells;
   ```
