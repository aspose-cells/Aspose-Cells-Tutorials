## Instruções de instalação
Para começar com Aspose.Cells para .NET, siga estes passos:  

1. Descarregue Aspose.Cells para .NET do repositório oficial: [Aspose.Cells Descarga](https://releases.aspose.com/cells/net/)  
2. Instalar através do Gerente de Pacotes NuGet:  
   * Studio Visual Aberto  
   * Execute o seguinte comando no Console do Gestor de Pacotes:  
     ```powershell
     Install-Package Aspose.Cells
     ```
3. Include the Aspose.Cells namespace in your C# project:  
   ```csharp
   using Aspose.Cells;
   ```  
4. Comece a codificar usando o exemplo fornecido para criar e salvar uma Tabela Pivot no formato ODS.
