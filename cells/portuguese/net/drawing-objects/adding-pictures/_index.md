# Crie e salve arquivos do Excel com Aspose.Cells
Este tutorial demonstra como utilizar o Aspose.Cells para .NET para criar um novo arquivo do Excel programaticamente. Você aprenderá a verificar a existência de um diretório, instanciar um objeto Workbook, adicionar uma planilha, inserir uma imagem em uma célula específica e salvar o arquivo no caminho desejado. Siga o trecho de código simples fornecido para começar a aprimorar suas tarefas de automação do Excel.

## Instruções de instalação
1. Instalar Aspose.Cells: Certifique-se de ter a biblioteca Aspose.Cells instalada. Você pode fazer isso usando o Gerenciador de Pacotes NuGet executando:
   ```
   Install-Package Aspose.Cells
   ```
2. Configure seu ambiente: Crie um diretório em sua máquina onde você deseja armazenar os arquivos Excel gerados. Substitua `"Your Document Directory"` no exemplo com seu caminho de diretório real.
3. Implementar o código: Copie o código de exemplo fornecido em seu aplicativo .NET. Certifique-se de atualizar o caminho da imagem (`dataDir + "logo.jpg"`) para apontar para uma imagem existente que você deseja inserir no arquivo Excel.
4. Execute o aplicativo: Execute seu aplicativo. O código criará um novo arquivo Excel chamado `output.xls` no diretório especificado, com uma imagem colocada na célula "F6".
