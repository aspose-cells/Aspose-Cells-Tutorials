# Crie e salve arquivos do Excel com Aspose.Cells
Este tutorial mostra como usar Aspose.Cells para .NET para criar um novo arquivo Excel programaticamente. Você aprenderá a verificar a existência de um diretório, instanciar um objeto de Workbook, adicionar uma folha de cálculo, inserir uma imagem em uma célula específica e salvar o arquivo para o seu caminho desejado. Siga o simples snippet de código fornecido para começar a melhorar suas tarefas de automação do Excel.

## Instruções de instalação
1. Instale Aspose.Cells: Certifique-se de ter a biblioteca Aspose.Cells instalada.
   ```
   Install-Package Aspose.Cells
   ```
2. Configurar o seu ambiente: Crie um diretório na sua máquina onde você deseja armazenar os arquivos do Excel gerados. Substitua `"O seu diretório de documentos"`no exemplo com o seu caminho de diretório real.
3. Implementar o Código: Copie o código de exemplo fornecido em sua aplicação .NET.`dadosDir + "logo.jpg"`) para apontar para uma imagem existente que deseja inserir no arquivo do Excel.
4. Execute a aplicação: execute a sua aplicação. O código criará um novo arquivo do Excel chamado `output.xls`no diretório especificado, com uma imagem colocada na célula "F6".
