# Crie uma pasta de trabalho do Excel com fontes do Extremo Oriente e Latinas

Este exemplo demonstra como usar o Aspose.Cells para .NET para criar uma pasta de trabalho do Excel, adicionar uma caixa de texto a uma planilha e definir nomes de fontes do Extremo Oriente e da América Latina para o texto dentro dessa caixa de texto. O código demonstra como inicializar o espaço de trabalho, manipular elementos da planilha e salvar a pasta de trabalho com formatação de fonte específica, adaptada para diversos requisitos de idioma.

## Instruções de instalação
1. Instalar Aspose.Cells para .NET: 
   - Use o Gerenciador de Pacotes NuGet no seu projeto do Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   - Ou baixe a DLL do [Site Aspose](https://www.aspose.com/downloads/cells/net).

2. Adicionar referências: certifique-se de que seu projeto faça referência à biblioteca Aspose.Cells.

3. Configurar diretório de saída: atualizar o `outputDir` variável no código com o caminho de saída desejado onde o arquivo Excel será salvo.

4. Execute o exemplo: copie o código C# fornecido para o seu projeto, compile e execute-o. Você deverá ver a confirmação da saída e o arquivo Excel com a caixa de texto configurada será criado no diretório especificado.
