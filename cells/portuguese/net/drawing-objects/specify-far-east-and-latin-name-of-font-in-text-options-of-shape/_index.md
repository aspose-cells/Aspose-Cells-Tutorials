# Crie Excel Workbook com fontes do Extremo Oriente e Latino

Este exemplo demonstra como usar Aspose.Cells para .NET para criar um livro de trabalho Excel, adicionar uma caixa de texto a uma folha de trabalho e definir nomes de fontes tanto do Extremo Oriente quanto do latim para o texto dentro dessa caixa de texto. O código mostra a inicialização do espaço de trabalho, a manipulação de elementos de folha de trabalho e a salvação do livro de trabalho com uma formatagem de fontes específica adaptada aos requisitos de linguagem diversos.

## Instruções de instalação
1. Instalar Aspose.Cells para .NET: 
   * Use o Gerente de Pacotes NuGet no seu projeto Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   * Ou baixar o DLL do [Aspose site](https://www.aspose.com/downloads/cells/net)\- Não .

2. Adicionar Referências: Certifique-se de que o seu projeto refere-se à biblioteca Aspose.Cells.

3. Configurar o diretório de saída: Atualizar o `saídaDir`A variável no código com o caminho de saída desejado onde o arquivo Excel será salvo.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
