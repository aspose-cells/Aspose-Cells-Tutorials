# Mecanismo de cálculo do Excel para aplicativos .NET

## Visão geral

Aspose.Cells para .NET oferece um mecanismo de cálculo robusto que permite aos desenvolvedores lidar com cálculos de fórmulas complexas em planilhas do Excel. Este poderoso componente permite configurar o comportamento do cálculo, implementar lógica de cálculo personalizada e otimizar o desempenho para pastas de trabalho grandes.

## Principais características

- **Motores de Cálculo Personalizados**: Implemente sua própria lógica de cálculo para fórmulas especializadas
- **Monitoramento de Cálculos**: Cálculos de fórmulas de rastreamento e controle para otimização de desempenho
- **Opções de cálculo recursivo**: Configurar como as referências circulares são tratadas
- **Configuração do sistema de data**:Alternar entre os sistemas de data de 1900 e 1904
- **Detecção de referência circular**: Identificar e resolver dependências circulares em fórmulas
- **Otimização de Desempenho**: Ajuste fino das configurações de cálculo para pastas de trabalho grandes
