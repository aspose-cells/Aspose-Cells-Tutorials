## Instrukcje instalacji
1. Zainstalowanie Aspose.Cells dla .NET: Użyj NuGet Package Manager do zainstalowania biblioteki Aspose.Cells w projekcie.
   ```
   Install-Package Aspose.Cells
   ```
2. Dodaj instrukcje używania: Włącz w plik kodu niezbędne przestrzenie nazw.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Dodaj kod próbkowy: Kopiować i wklej dostarczony kod do projektu, zapewniając określenie ścieżki katalogu dokumentów.
4. Uruchomić kod: Wykonaj projekt, aby uzyskać dostęp i zmienić etykietę obiektu Ole w pliku Excel. Sprawdź na konsoli wiadomości wyjściowe potwierdzające udane wykonanie procesu.
