# Tworzenie i zapisywanie plików Excel za pomocą Aspose.Cells
W tym tutoriale dowiemy się, jak użyć Aspose.Cells dla .NET do tworzenia nowego pliku Excel programatycznie. Nauczysz się sprawdzić istnienie katalogu, instansować obiekt Workbook, dodać kartę roboczą, wstawić obraz do określonej komórki i zapisać plik na żądaną drogę.

## Instrukcje instalacji
1. Zainstalowanie Aspose.Cells: Upewnij się, że masz zainstalowaną bibliotekę Aspose.Cells. Możesz to zrobić za pomocą pakietu NuGet Manager uruchamiając:
   ```
   Install-Package Aspose.Cells
   ```
2. Ustaw środowisko: utwórz katalog na komputerze, w którym chcesz przechowywać generowane pliki Excel.`"Wykładnik dokumentów"`w przykładzie z prawdziwą ścieżką katalogu.
3. Wdrożenie kodu: Kopiować podany przykład kodu do aplikacji .NET. Upewnij się, że aktualizujesz ścieżkę obrazu (`daneDir + "logo.jpg"`) wskazując na istniejący obraz, który chcesz wstawić do pliku Excel.
4. Uruchomić aplikację: uruchomić aplikację. Kod tworzy nowy plik Excel o nazwie `output.xls`w wypełnionym katalogu, z obrazem umieszczonym w komórce "F6".
