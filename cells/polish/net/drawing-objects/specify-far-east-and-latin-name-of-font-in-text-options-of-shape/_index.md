# Tworzenie podręcznika Excel z odległym Wschodem i latynskim czcionkiem

Ten przykład pokazuje, jak użyć Aspose.Cells dla .NET do tworzenia książeczki roboczej Excel, dodać pudełko do arkusza roboczej i ustawić zarówno Far East, jak i łacińskie nazwy czcionek dla tekstu w tej skrzynce roboczej. Kod pokazuje inicjalizację przestrzeni roboczej, manipulowanie elementami arkusza roboczej i zapisywanie książeczki roboczej z konkretnym formatowaniem czcionek dostosowanym do różnych wymagań językowych.

## Instrukcje instalacji
1. Zainstalowanie Aspose.Cells dla .NET: 
   * Użyj menedżera pakietu NuGet w projekcie Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   * Albo pobierz DLL z [Strona internetowa](https://www.aspose.com/downloads/cells/net)\- Nie .

2. Dodaj odniesienia: Upewnij się, że projekt odnosi się do biblioteki Aspose.Cells.

3. Ustawić katalog wyjściowy: Aktualizować `wydajność`zmienna w kodzie z pożądanym ścieżką wyjściową, w której zostanie zapisane plik Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
