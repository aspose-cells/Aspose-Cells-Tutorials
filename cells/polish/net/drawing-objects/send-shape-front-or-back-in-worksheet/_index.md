# Manipulacja z-porządu kształtów w Excel za pomocą Aspose.Cells
 
W tym przykładzie pokazano, jak używać Aspose.Cells dla .NET do ładowania książeczki roboczej Excel, manipulowania pozycjami w Z-pozicji kształtów w arkuszu roboczym i zapisywania zaktualizowanego książeczki roboczej.

## Instrukcje instalacji  
1. Upewnij się, że masz ustawione środowisko rozwoju .NET, takie jak Visual Studio.  
2. Zainstalować Aspose.Cells dla .NET za pośrednictwem pakietu NuGet Manager.  
   ```
   Install-Package Aspose.Cells
   ```
3. Dodaj odniesienie do przestrzeni nazw Aspose.Cells w projekcie:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Drawing; 
   ```
4. Przygotuj plik źródłowy i aktualizuj `źródłoDir`i `wydajność`ścieżki w przedstawionych przykładach kodu.  
5. Kopiować i wklejać dostarczony fragment kodu do projektu, aby manipulować Z-porządek kształtów w pliku Excel.  
6. Wykonaj i uruchomi aplikację, aby zobaczyć zmiany odzwierciedlane w pliku Excel wyjściowym.
