## Instrukcje instalacji
1. Pobierz i zainstaluj Aspose.Cells dla .NET z [Strona wydań Aspose](https://releases.aspose.com/cells/net/).  
2. Dodaj odwołanie do Aspose.Cells.dll w swoim projekcie C#.  
3. Użyj Menedżera pakietów NuGet, aby zainstalować Aspose.Cells:  
   ```sh
   Install-Package Aspose.Cells
   ```
4. Zaimportuj wymagane przestrzenie nazw do pliku C#:  
   ```csharp
   using Aspose.Cells;
   ```
