## Instrukcje instalacji
1. Zabierz i instaluj Aspose.Cells dla .NET z [Strona wypuścił Aspose](https://releases.aspose.com/cells/net/)\- Nie .  
2. Add a reference to Aspose.Cells.dll in your C# project.  
3. Użyj Menedżera pakietu NuGet do zainstalowania Aspose.Cells:  
   ```sh
   Install-Package Aspose.Cells
   ```
4. Import the required namespaces in your C# file:  
   ```csharp
   using Aspose.Cells;
   ```
