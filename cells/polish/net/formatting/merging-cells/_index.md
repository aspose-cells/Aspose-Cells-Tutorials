## Instrukcje instalacji  
Aby użyć Aspose.Cells dla .NET w projekcie, wykonaj następujące kroki:  

1. Instalacja za pośrednictwem NuGet  
   Otwórz konsolę zarządzającego pakietami w Visual Studio i uruchomi:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Pobierz DLL  
   Alternatywnie, pobierz najnowszą wersję z [Aspose.Cells dla pobrań .NET](https://releases.aspose.com/cells/net/)i odwołaj się do DLL w projekcie.  

3. Dodaj referencję do swojego projektu  
   * W Visual Studio kliknij prawym kliknięciem projekt w rozwiązania Explorer.  
   * Kliknij Dodaj odniesienie > przeglądanie > Wybierz pobrane DLL.  
   * Kliknij OK, aby dodać odniesienie.  

4. Zacznij używać Aspose.Cells  
   Dodaj następujące przestrzenie nazw na początku pliku C#:  
   ```csharp
   using Aspose.Cells;
   ```
