## Instrukcje instalacji:  

Aby użyć Aspose.Cells dla .NET w projekcie, wykonaj następujące kroki:  

1. Instalacja za pośrednictwem NuGet:  
   Otwórz konsolę zarządzającego pakietami w Visual Studio i uruchomi:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Zabierz DLL:  
   Alternatywnie, można pobrać najnowszą wersję Aspose.Cells dla wersji .NET z [Wystarczy pobrać](https://releases.aspose.com/cells/net/)\- Nie .

3. Dodaj odniesienie:  
   Jeśli załadowałeś DLL ręcznie, dodaj odniesienie do `Aspose.Cells.`w twoim projekcie .NET.

4. Włączyć przestrzeni nazwisk:  
   In your C# file, add:  
   ```csharp
   using Aspose.Cells;
   using System;
   using System.Drawing;
   ```

5. Wykorzystaj przykład:  
   Copy and paste the provided C# code into your project, update the output directory, and execute the script to generate an Excel file with a line chart.
