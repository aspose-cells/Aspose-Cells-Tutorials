## Installation Instructions:  

To use Aspose.Cells for .NET in your project, follow these steps:  

1. Install via NuGet:  
   Open the Package Manager Console in Visual Studio and run:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Download DLL:  
   Alternatively, you can download the latest Aspose.Cells for .NET version from [Aspose Downloads](https://releases.aspose.com/cells/net/).

3. Add Reference:  
   If you've downloaded the DLL manually, add a reference to `Aspose.Cells.dll` in your .NET project.

4. Include Namespace:  
   In your C# file, add:  
   ```csharp
   using Aspose.Cells;
   using System;
   using System.Drawing;
   ```

5. Run the Example:  
   Copy and paste the provided C# code into your project, update the output directory, and execute the script to generate an Excel file with a line chart.
