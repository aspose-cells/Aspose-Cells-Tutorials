## Installation Instructions

To get started with Aspose.Cells for .NET, follow these installation steps:  

1. Install via NuGet Package Manager:  
   - Open Visual Studio.  
   - Go to Tools > NuGet Package Manager > Manage NuGet Packages for Solution.  
   - Search for `Aspose.Cells` and click Install.  

2. Install via NuGet Package Manager Console:  
   - Open Package Manager Console from Tools > NuGet Package Manager > Package Manager Console.  
   - Run the command:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. Download the DLL Manually:  
   - Visit the [Aspose.Cells Download Page](https://releases.aspose.com/cells/net/).  
   - Download and extract the package.  
   - Add a reference to `Aspose.Cells.dll` in your project.
