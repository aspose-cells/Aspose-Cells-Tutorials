# Create Excel Workbook with Far East & Latin Fonts

This example demonstrates how to use Aspose.Cells for .NET to create an Excel workbook, add a textbox to a worksheet, and set both Far East and Latin font names for the text within that textbox. The code showcases initializing the workspace, manipulating worksheet elements, and saving the workbook with specific font formatting tailored for diverse language requirements.

## Installation Instructions
1. Install Aspose.Cells for .NET: 
   - Use NuGet Package Manager in your Visual Studio project:
     ```
     Install-Package Aspose.Cells
     ```
   - Or download the DLL from the [Aspose website](https://www.aspose.com/downloads/cells/net).

2. Add References: Ensure your project references the Aspose.Cells library.

3. Set Up Output Directory: Update the `outputDir` variable in the code with the desired output path where the Excel file will be saved.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
