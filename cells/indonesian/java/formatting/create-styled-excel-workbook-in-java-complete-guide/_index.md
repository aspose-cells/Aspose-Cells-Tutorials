# Buat Workbook Excel Bergaya di Java – Panduan Lengkap

Pernah perlu **create styled excel workbook** dari Java tetapi merasa terbebani oleh detail pemformatan? Anda tidak sendirian. Baik Anda mengambil data dari basis data atau mengekspor laporan, mendapatkan tampilan yang tepat—simbol mata uang, pola tanggal, header tebal—bisa terasa seperti pekerjaan rumah.  

Dalam tutorial ini kami akan membahas solusi praktis yang **adds custom formatting to excel**, memungkinkan Anda **set date format excel column**, dan akhirnya **save workbook as xlsx** tanpa susah payah. Siap melihatnya beraksi?

## Apa yang Akan Anda Pelajari

- Cara memulai `Workbook` Aspose.Cells dari awal.  
- Cara menerapkan array gaya yang mencakup kolom General, Currency, dan Date.  
- Panggilan tepat yang **creates styled excel workbook** dengan nama kolom yang sudah disertakan.  
- Cara menyimpan file menggunakan metode **save workbook as xlsx**.  

Tanpa file konfigurasi eksternal, tanpa trik XML yang rumit—hanya kode Java biasa yang dapat Anda salin‑tempel, jalankan, dan sesuaikan.

### Prasyarat

- Java 8+ (kode ini bekerja pada JDK terbaru mana pun).  
- Perpustakaan Aspose.Cells untuk Java (Anda dapat mengunduh JAR terbaru dari Maven Central).  
- Sedikit pengetahuan tentang koleksi (`List`, `Map`).  

Jika ada yang belum familiar, jeda sejenak dan instal JAR Aspose.Cells; sisanya akan mengalir dengan alami.

---

## Buat Workbook Excel Bergaya – Menyiapkan Proyek

Langkah pertama: buat proyek Maven (atau Gradle) baru dan tambahkan Aspose.Cells sebagai dependensi.

```xml
<!-- pom.xml snippet -->
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>23.12</version> <!-- use the latest version -->
</dependency>
```

> **Pro tip:** Jika Anda menggunakan Gradle, koordinat yang sama dapat dipakai dengan `implementation 'com.aspose:aspose-cells:23.12'`.

Mengapa memulai dengan pengaturan Maven? Karena secara otomatis menarik semua dependensi transitive, jadi Anda tidak perlu membuang waktu mencari JAR yang tepat nanti.

## Tambahkan Pemformatan Kustom ke Kolom Excel

Setelah proyek siap, mari definisikan gaya yang kita inginkan. Di Aspose.Cells Anda dapat memberikan array string format yang langsung dipetakan ke setiap kolom.

```java
// Define column styles: General, Currency, Date
String[] columnStyles = {
    "General",          // Column A – plain text or numbers
    "$#,##0.00",        // Column B – currency with two decimals
    "mm/dd/yyyy"        // Column C – US‑style date
};
```

Langkah **add custom formatting to excel** sesederhana membangun array string ini.  
Setiap entri mengikuti pola yang sama seperti yang Anda gunakan di dialog “Custom” Excel, jadi Anda tidak perlu mempelajari sintaks baru.

---

## Atur Format Tanggal pada Kolom Excel

Jika data Anda mencakup tanggal, Anda ingin memastikan tampilannya konsisten. Entri ketiga dalam `columnStyles` (`"mm/dd/yyyy"`) melakukan hal itu—memberitahu Aspose.Cells untuk memperlakukan kolom ketiga sebagai tanggal dan menampilkannya dalam urutan bulan/hari/tahun.

```java
// Example: Adding a row with a Date object
Map<String, Object> row = new HashMap<>();
row.put("Product", "Widget");
row.put("Price", 19.99);
row.put("ReleaseDate", new java.util.Date()); // Today’s date
dataRows.add(row);
```

Saat workbook dihasilkan, instruksi **set date format excel column** memastikan setiap sel `ReleaseDate` terlihat seperti `04/27/2026` (atau tanggal hari ini). Tidak perlu pemrosesan tambahan.

## Impor Data dan Terapkan Gaya

Dengan gaya yang sudah didefinisikan, langkah selanjutnya adalah memasukkan data ke dalam worksheet. Aspose.Cells menyediakan `importCustomObjects`, yang tidak hanya menyisipkan data tetapi juga menerapkan array gaya yang baru saja kita buat.

```java
// Step 1: Simulate fetching rows from a DB
List<Map<String, Object>> dataRows = fetchRows(); // see helper below

// Step 2: Create a fresh workbook and grab the first sheet
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);

// Step 3: Import data with custom styles; true = first row has headers
worksheet.getCells().importCustomObjects(
        dataRows,        // data source
        0,               // start row (0‑based)
        0,               // start column (0‑based)
        columnStyles,    // style array from above
        true);           // first row contains column names
```

### Mengapa `importCustomObjects`?

- **Keajaiban satu panggilan:** Ia menulis data *dan* menerapkan pemformatan sekaligus.  
- **Kesadaran header:** Menetapkan `true` memberi tahu API untuk memperlakukan kunci peta pertama sebagai header kolom, yang sangat cocok untuk skenario **create styled excel workbook** di mana Anda menginginkan judul kolom yang dapat dibaca.  
- **Kinerja:** Di balik layar Aspose mengelompokkan operasi penulisan, membuatnya lebih cepat dibandingkan menulis sel per sel.

> **Perhatian:** Urutan kunci dalam setiap `Map` penting. Gunakan `LinkedHashMap` jika Anda memerlukan urutan kolom yang deterministik.

## Simpan Workbook sebagai File XLSX

Semua kerja keras itu akan sia-sia jika Anda tidak dapat menyimpan file. Baris terakhir adalah satu baris kode yang **save workbook as xlsx**.

```java
// Step 5: Persist the workbook
String outputPath = "output/DataTableStyled.xlsx";
workbook.save(outputPath);
System.out.println("Workbook saved to " + outputPath);
```

Metode `save` secara otomatis mendeteksi ekstensi file. Dengan menentukan `.xlsx`, Anda menjamin format Office Open XML modern, yang merupakan harapan kebanyakan alat downstream.

## Contoh Lengkap yang Dapat Dijalankan

Di bawah ini adalah kelas lengkap yang dapat Anda salin langsung ke IDE. Ia menyertakan metode mock `fetchRows` yang mengembalikan data contoh, sehingga Anda tidak memerlukan basis data nyata untuk menguji.

```java
import com.aspose.cells.*;
import java.util.*;

public class ImportDataTableWithStyle {
    public static void main(String[] args) throws Exception {
        // ---------- Step 1: Get data ----------
        List<Map<String, Object>> dataRows = fetchRows();

        // ---------- Step 2: Create workbook ----------
        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // ---------- Step 3: Define column styles ----------
        String[] columnStyles = { "General", "$#,##0.00", "mm/dd/yyyy" };

        // ---------- Step 4: Import data with styles ----------
        // true = first row contains column names
        worksheet.getCells().importCustomObjects(dataRows, 0, 0, columnStyles, true);

        // ---------- Step 5: Save as XLSX ----------
        String outPath = "output/DataTableStyled.xlsx";
        workbook.save(outPath);
        System.out.println("Workbook saved to " + outPath);
    }

    // Mock method – replace with real DB call if needed
    private static List<Map<String, Object>> fetchRows() {
        List<Map<String, Object>> rows = new ArrayList<>();

        Map<String, Object> row1 = new LinkedHashMap<>();
        row1.put("Product", "Gadget");
        row1.put("Price", 49.95);
        row1.put("ReleaseDate", new Date()); // today
        rows.add(row1);

        Map<String, Object> row2 = new LinkedHashMap<>();
        row2.put("Product", "Widget");
        row2.put("Price", 19.99);
        Calendar cal = Calendar.getInstance();
        cal.set(2025, Calendar.DECEMBER, 15);
        row2.put("ReleaseDate", cal.getTime());
        rows.add(row2);

        return rows;
    }
}
```

Jalankan, buka file yang dihasilkan, dan Anda akan melihat hasil **create styled excel workbook** yang diharapkan.

## Pertanyaan Umum & Kasus Pojok

**Bagaimana jika saya membutuhkan lebih dari tiga kolom?**  
Cukup perpanjang array `columnStyles`—setiap entri dipetakan ke kolom secara berurutan. Untuk kolom yang ingin Anda biarkan tanpa perubahan, gunakan `"General"`.

**Apakah saya dapat menerapkan font atau warna?**  
Ya. Alih-alih string format, Anda dapat membuat objek `Style`, menetapkannya ke `StyleFlag`, dan memberikan array `Style[]` ke `importCustomObjects`. Itu merupakan eksplorasi lebih dalam, tetapi prinsipnya tetap sama.

**Bagaimana dengan kumpulan data besar?**  
Aspose.Cells menangani jutaan baris secara efisien, tetapi Anda mungkin ingin meningkatkan heap JVM (`-Xmx2…`).
