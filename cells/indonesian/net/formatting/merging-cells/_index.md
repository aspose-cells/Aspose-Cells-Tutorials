## Telepítési utasítások  
Untuk menggunakan Aspose.Cells for .NET di proyek Anda, ikuti langkah-langkah berikut:  

1. Instal melalui NuGet  
   Buka Konsol Manajer Paket di Visual Studio dan jalankan:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Unduh DLL  
   Atau, unduh versi terbaru dari [Aspose.Cells .NET-hez letöltések](https://releases.aspose.com/cells/net/) dan merujuk ke DLL di proyek Anda.  

3. Tambahkan Referensi di Proyek Anda  
   - Di Visual Studio, klik kanan proyek Anda di Solution Explorer.  
   - Klik Tambahkan Referensi > Telusuri > Pilih DLL yang diunduh.  
   - Klik OK untuk menambahkan referensi.  

4. Mulai Menggunakan Aspose.Cells  
   Tambahkan namespace berikut di awal file C# Anda:  
   ```csharp
   using Aspose.Cells;
   ```
