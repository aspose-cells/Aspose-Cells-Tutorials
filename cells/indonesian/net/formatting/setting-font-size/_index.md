## Instruksi Instalasi

Untuk memulai dengan Aspose.Cells untuk .NET, ikuti langkah-langkah instalasi ini:  

1. Instal melalui NuGet Package Manager:  
   * Buka Visual Studio.  
   * Pergi ke Alat > NuGet Package Manager > Mengelola NuGet Packages untuk Solution.  
   * Carilah`"Aspose. Sel"`dan klik Install.  

2. Instal melalui NuGet Package Manager Console:  
   * Buka Konsole Manajer Paket dari Alat > NuGet Manajer Paket > Konsole Manajer Paket.  
   * Luncurkan perintah:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. Unduh DLL secara manual:  
   * Kunjungi [Aspose.Cells Download Page](https://releases.aspose.com/cells/net/)Aku tidak tahu.  
   * Unduh dan ambil paketnya.  
   * Tambahkan referensi ke `Aspose.Cells. dll`dalam proyekmu.
