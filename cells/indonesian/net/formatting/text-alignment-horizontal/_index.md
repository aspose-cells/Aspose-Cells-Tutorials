Contoh ini menunjukkan cara menggunakan Aspose.Cells untuk .NET guna membuat file Excel dan mengatur perataan teks horizontal dalam sel tertentu. Tutorial ini memandu Anda dalam menginisialisasi buku kerja, memodifikasi properti sel, dan menyimpan file dalam format Excel 97-2003. ## Telepítési utasítások
1. Unduh dan instal Aspose.Cells untuk .NET dari [Aspose kiadási oldal](https://releases.aspose.com/cells/net/).  
2. Tambahkan referensi ke Aspose.Cells.dll di proyek C# Anda.  
3. Gunakan NuGet Package Manager untuk menginstal Aspose.Cells:  
   ```sh
   Install-Package Aspose.Cells
   ```
4. Impor namespace yang diperlukan dalam file C# Anda:  
   ```csharp
   using Aspose.Cells;
   ```
