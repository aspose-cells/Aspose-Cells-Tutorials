## Telepítési utasítások:  

Untuk menggunakan Aspose.Cells untuk .NET, ikuti langkah-langkah berikut:  

1. Instal melalui Pengelola Paket NuGet  
- Nyisd meg a Visual Studio-t.  
- Navigasi ke Alat → Manajer Paket NuGet → Kelola Paket NuGet untuk Solusi.  
- Keresd meg az Aspose.Cells-t.  
- Klik Instal untuk menambahkan paket ke proyek Anda.  

Atau, instal menggunakan Konsol Manajer Paket NuGet:  

```powershell
Install-Package Aspose.Cells
```

2. Tambahkan Namespace di Kode C# Anda  
```csharp
using Aspose.Cells;
```

3. Aktifkan Lisensi (Opsional, untuk Fitur Lengkap)  
Jika Anda memiliki lisensi, terapkan menggunakan kode berikut:  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
