# Mengubah arah Tick Label Chart di Excel menggunakan Aspose.Cells 

Contoh ini menunjukkan bagaimana mengubah arah label tik grafik dalam lembar kerja Excel menggunakan Aspose.Cells untuk .NET. Dengan menyesuaikan arah teks label tik kategori sumbu, Anda dapat meningkatkan kebacakan data grafik. Kode memuat buku kerja Excel yang ada, mengakses lembar kerja pertama, mengambil grafik, mengatur arah label tik ke horizontal, dan kemudian menyimpan buku kerja yang dimodifikasi. Pendekatan ini memastikan kontrol yang tepat atas orientasi label grafik dalam aplikasi .NET Anda.

## Instruksi Instalasi  
Untuk menjalankan contoh ini, ikuti langkah-langkah ini:  

1. Instal Aspose.Cells untuk .NET  
   * Unduh dari [Aspose.Cells untuk .NET](https://releases.aspose.com/cells/net/)  
   * Atau install melalui NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Buat Proyek Anda  
   * Buat proyek .NET baru.  
   * Tambahkan referensi ke Aspose.Cells.dll.  

3. Luncurkan Kode  
   * Ganti `"Directory Dokumen Anda"`dengan jalur file yang sebenarnya.  
   * Keluarkan program untuk memperbarui arah label tik di grafik.
