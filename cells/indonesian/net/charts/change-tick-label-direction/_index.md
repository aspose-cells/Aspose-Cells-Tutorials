# Ubah Arah Label Centang Bagan di Excel menggunakan Aspose.Cells 

Contoh ini menunjukkan cara mengubah arah label tanda centang pada bagan di lembar kerja Excel menggunakan Aspose.Cells untuk .NET. Dengan menyesuaikan arah teks label tanda centang pada sumbu kategori, Anda dapat meningkatkan keterbacaan data bagan. Kode tersebut memuat buku kerja Excel yang sudah ada, mengakses lembar kerja pertama, mengambil bagan, mengatur arah label tanda centang ke horizontal, lalu menyimpan buku kerja yang telah diubah. Pendekatan ini memastikan kontrol yang tepat atas orientasi label bagan di aplikasi .NET Anda.

## Telepítési utasítások  
Untuk menjalankan contoh ini, ikuti langkah-langkah berikut:  

1. Instal Aspose.Cells untuk .NET  
   - Letöltés innen [Aspose.Cells .NET-hez](https://releases.aspose.com/cells/net/)  
   - Atau instal melalui NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Projekt beállítása  
   - Buat proyek .NET baru.  
   - Tambahkan referensi ke Aspose.Cells.dll.  

3. Jalankan Kode  
   - Csere `"Your Document Directory"` dengan jalur berkas sebenarnya.  
   - Jalankan program untuk memperbarui arah label centang pada grafik.
