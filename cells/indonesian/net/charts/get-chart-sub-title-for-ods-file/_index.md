## Instruksi pemasangan:  
Untuk menggunakan contoh ini, ikuti langkah-langkah berikut:  

1. Instal Aspose.Cells untuk .NET melalui NuGet:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Tambahkan ruang nama yang dibutuhkan dalam proyek C# Anda:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Jalankan kode yang diberikan untuk memuat file ODS, mengakses grafik, dan cetak subtitle ke konsol.
