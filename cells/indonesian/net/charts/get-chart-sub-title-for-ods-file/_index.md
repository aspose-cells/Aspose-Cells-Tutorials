## Telepítési utasítások:  
Untuk menggunakan contoh ini, ikuti langkah-langkah berikut:  

1. Instal Aspose.Cells untuk .NET melalui NuGet:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Tambahkan namespace yang diperlukan dalam proyek C# Anda:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Jalankan kode yang disediakan untuk memuat berkas ODS, mengakses bagan, dan mencetak subjudulnya ke konsol.
