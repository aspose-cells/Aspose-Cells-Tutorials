# Buat PDF dari Excel Chart dengan ukuran halaman yang ditentukan

Mengekspor grafik dari Excel ke file PDF dengan tata letak halaman yang tepat adalah persyaratan penting untuk pelaporan dan visualisasi data. Tutorial ini menunjukkan cara menggunakan Aspose.Cells untuk .NET untuk menghasilkan PDF dari grafik Excel sambil menentukan ukuran halaman dan keselarasan. Anda akan belajar cara memuat file Excel, mengakses lembar kerja dan grafiknya, dan mengkonversi ke file PDF yang berformat baik.

## Instruksi Instalasi  
Untuk menggunakan Aspose.Cells untuk .NET, ikuti langkah-langkah ini:  

1. Instal melalui NuGet:  
   Buka NuGet Package Manager di Visual Studio dan jalankan:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Unduh DLL:  
   Dapatkan versi terbaru dari [Aspose.Cells untuk halaman download .NET](https://releases.aspose.com/cells/net/)Aku tidak tahu.  

3. Tambahkan ruang nama:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
