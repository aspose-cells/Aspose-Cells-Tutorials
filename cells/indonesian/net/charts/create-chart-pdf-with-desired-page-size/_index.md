# Buat PDF dari Bagan Excel dengan Ukuran Halaman Tertentu

Mengekspor bagan dari Excel ke berkas PDF dengan tata letak halaman yang tepat merupakan persyaratan penting untuk pelaporan dan visualisasi data. Tutorial ini menunjukkan cara menggunakan Aspose.Cells for .NET untuk membuat PDF dari bagan Excel sambil menentukan ukuran dan perataan halaman. Anda akan mempelajari cara memuat berkas Excel, mengakses lembar kerja dan bagannya, serta mengonversinya menjadi berkas PDF yang diformat dengan baik.

## Telepítési utasítások  
Untuk menggunakan Aspose.Cells untuk .NET, ikuti langkah-langkah berikut:  

1. Instal melalui NuGet:  
   Buka NuGet Package Manager di Visual Studio dan jalankan:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Unduh DLL:  
   Dapatkan versi terbaru dari [Aspose.Cells .NET letöltési oldal](https://releases.aspose.com/cells/net/).  

3. Sertakan Namespace:  
   Tambahkan yang berikut ini di berkas C# Anda:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
