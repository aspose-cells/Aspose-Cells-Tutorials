## Instruksi Instalasi  
1. Pastikan Anda memiliki lingkungan pengembangan .NET yang diatur, seperti Visual Studio.  
2. Instal Aspose.Cells untuk .NET melalui NuGet Package Manager. Gunakan perintah berikut di Konsole Manager Package:  
   ```
   Install-Package Aspose.Cells
   ```
3. Tambahkan referensi ke ruang nama Aspose.Cells di proyek Anda:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Drawing; 
   ```
4. Persiapkan file Excel sumber dan memperbarui `sumberDir`dan`outputDir`jalur dalam contoh kode yang diberikan.  
5. Salin dan tempel snippet kode yang diberikan ke dalam proyek Anda untuk memanipulasi urutan bentuk Z dalam file Excel Anda.  
6. Bangun dan jalankan aplikasi Anda untuk melihat perubahan yang tercermin dalam file Excel output.
