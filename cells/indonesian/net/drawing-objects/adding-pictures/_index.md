# Buat dan Simpan File Excel dengan Aspose.Cells
Tutorial ini menunjukkan cara menggunakan Aspose.Cells untuk .NET untuk membuat file Excel baru secara programatik. Anda akan belajar untuk memeriksa keberadaan direktori, menginstansiasi objek Workbook, menambahkan lembar kerja, memasukkan gambar ke dalam sel tertentu, dan menyimpan file ke jalur yang Anda inginkan. Ikuti snippet kode sederhana yang disediakan untuk mulai meningkatkan tugas otomatisasi Excel Anda.

## Instruksi Instalasi
1. Instal Aspose.Cells: Pastikan Anda memiliki perpustakaan Aspose.Cells terinstal. Anda dapat melakukan ini menggunakan NuGet Package Manager dengan menjalankan:
   ```
   Install-Package Aspose.Cells
   ```
2. Setup Environment Anda: Buat direktori di mesin Anda di mana Anda ingin menyimpan file Excel yang dihasilkan.`"Directory Dokumen Anda"`di contoh dengan jalur direktori Anda yang sebenarnya.
3. Implementasi Kode: Salin kode contoh yang diberikan ke dalam aplikasi .NET Anda.`dataDir + "logo.jpg"`) untuk menunjuk ke gambar yang ada yang ingin Anda masukkan ke file Excel.
4. Jalankan Aplikasi: Jalankan aplikasi Anda. Kode akan membuat file Excel baru bernama `output.xls`di direktori yang ditentukan, dengan gambar yang ditempatkan di sel "F6".
