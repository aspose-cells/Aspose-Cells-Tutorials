## Instruksi Instalasi
1. Instal Aspose.Cells untuk .NET: Gunakan NuGet Package Manager untuk menginstal perpustakaan Aspose.Cells di proyek Anda.
   ```
   Install-Package Aspose.Cells
   ```
2. Tambahkan Instruksi Penggunaan: Masukkan ruang nama yang diperlukan dalam file kode Anda.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Tambahkan Kode Contoh: Salin dan tempelkan kode yang diberikan ke dalam proyek Anda, memastikan untuk menentukan jalur direktori dokumen Anda.
4. Jalankan Kode: Eksekusi proyek Anda untuk mengakses dan memodifikasi label objek Ole dalam file Excel Anda. Periksa konsol untuk pesan output yang mengkonfirmasi eksekusi proses yang berhasil.
