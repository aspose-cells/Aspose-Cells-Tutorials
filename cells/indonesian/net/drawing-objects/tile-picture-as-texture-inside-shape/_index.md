## Instruksi Instalasi
1. Pastikan Anda memiliki lingkungan .NET yang disiapkan pada mesin Anda.
2. Unduh dan pasang perpustakaan Aspose.Cells dari situs web resmi Aspose atau menggunakan NuGet Package Manager:
   ```
   Install-Package Aspose.Cells
   ```
3. Create a new C# project or open your existing .NET project.
4. Tambahkan referensi ke perpustakaan Aspose.Cells dalam proyek Anda.
5. Salin kode contoh yang diberikan ke dalam proyek Anda, mengganti jalur direktori input/output sesuai kebutuhan.
6. Kompilasi dan jalankan proyek Anda untuk melihat isi tekstur lembaran diterapkan pada bentuk dalam file Excel.
