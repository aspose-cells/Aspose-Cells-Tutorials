## การแนะนำ

## คู่มือการใช้งาน

ให้เราแบ่งการใช้งานออกเป็นคุณสมบัติที่แตกต่างกัน: การสร้างเวิร์กบุ๊ก การเติมข้อมูล การเพิ่มแผนภูมิ และการบันทึกไฟล์

### คุณลักษณะที่ 1: การสร้างเวิร์กบุ๊กและการจัดการพื้นฐาน

#### ภาพรวม

การสร้างเวิร์กบุ๊ก Excel ใหม่เกี่ยวข้องกับการเริ่มต้นวัตถุ การเข้าถึงเวิร์กชีต และการเติมข้อมูลตัวอย่างลงในเซลล์ หัวข้อนี้จะแนะนำคุณตลอดขั้นตอนเหล่านี้โดยใช้ Aspose.Cells สำหรับ Java

#### การดำเนินการแบบทีละขั้นตอน

**สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กใหม่**

สร้างอินสแตนซ์ใหม่ของ `Workbook` ชั้นเรียนเพื่อเริ่มทำงานกับไฟล์ Excel ของคุณ:

```java
Workbook workbook = new Workbook();
```

**เข้าถึงแผ่นงานแรก**

ดึงข้อมูลและทำงานกับเวิร์กชีตแรกในเวิร์กบุ๊ก:

```java
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet sheet = worksheets.get(0);
```

**เติมข้อมูลตัวอย่างลงในเซลล์**

เติมเซลล์โดยใช้ `Cells` วัตถุ ต่อไปนี้เป็นวิธีการเติมข้อมูลในเซลล์เฉพาะ:

```java
Cells cells = sheet.getCells();

// เติมค่า 50 ลงในเซลล์ A1
cells.get("A1").setValue(50);

// ดำเนินการต่อสำหรับเซลล์อื่นๆ...
cells.get("A2").setValue(100);
cells.get("A3").setValue(150);
cells.get("B1").setValue(4);
cells.get("B2").setValue(20);
cells.get("B3").setValue(50);
```

**คำอธิบาย**:โค้ดนี้จะเริ่มต้นและเติมเวิร์กชีตด้วยค่าที่กำหนดไว้ล่วงหน้า เพื่อเตรียมพร้อมสำหรับการจัดการเพิ่มเติม เช่น การสร้างแผนภูมิ

### คุณลักษณะที่ 2: การเพิ่มแผนภูมิลงในเวิร์กชีต

#### ภาพรวม

การเพิ่มแผนภูมิเกี่ยวข้องกับการกำหนดประเภท ตั้งค่าแหล่งข้อมูล และกำหนดค่าชุดข้อมูล ซึ่งจะช่วยเพิ่มประสิทธิภาพการแสดงภาพข้อมูลโดยตรงภายในเวิร์กบุ๊กของคุณ

#### การดำเนินการแบบทีละขั้นตอน

**สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กใหม่**

หากยังไม่ได้สร้าง ให้เริ่มต้นใหม่ `Workbook` ตัวอย่าง:

```java
Workbook workbook = new Workbook();
```

**เข้าถึงแผ่นงานแรก**

ตรวจสอบให้แน่ใจว่าคุณกำลังทำงานบนเวิร์กชีตแรก:

```java
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet sheet = worksheets.get(0);
```

**เพิ่มข้อมูลตัวอย่างลงในเซลล์**

คล้ายกับก่อนหน้านี้ ให้ป้อนข้อมูลของคุณ:

```java
Cells cells = sheet.getCells();

cells.get("A1").setValue(50);
cells.get("A2").setValue(100);
cells.get("A3").setValue(150);
cells.get("B1").setValue(4);
cells.get("B2").setValue(20);
cells.get("B3").setValue(50);
```

**เข้าถึงคอลเลกชันแผนภูมิ**

ดึงข้อมูลหรือสร้างคอลเลกชันแผนภูมิภายในเวิร์กชีตของคุณ:

```java
ChartCollection charts = sheet.getCharts();
```

**เพิ่มและกำหนดค่าแผนภูมิใหม่**

สร้างแผนภูมิพีระมิดเป็นตัวอย่าง:

```java
int chartIndex = charts.add(ChartType.PYRAMID, 5, 0, 15, 5);
Chart chart = charts.get(chartIndex);

// ตั้งค่าแหล่งที่มาของข้อมูลสำหรับชุดแผนภูมิ
SeriesCollection serieses = chart.getNSeries();
serieses.add("A1:B3", true); // 'จริง' หมายถึงแถวแรกมีส่วนหัว
```

**คำอธิบาย**: เดอะ `add` วิธีการเพิ่มแผนภูมิใหม่ของประเภทที่ระบุในตำแหน่งที่กำหนด และ `getNSeries()` กำหนดค่าชุดข้อมูลที่เชื่อมโยงกับช่วงเซลล์

### คุณสมบัติที่ 3: การบันทึกไฟล์ Excel

#### ภาพรวม

เมื่อเวิร์กบุ๊กของคุณพร้อมแล้ว—ซึ่งเต็มไปด้วยข้อมูลและแผนภูมิ—ถึงเวลาบันทึกเวิร์กบุ๊กไว้เพื่อใช้หรือเผยแพร่ในภายหลัง หัวข้อนี้จะกล่าวถึงกลไกการบันทึกไฟล์โดยใช้ Aspose.Cells สำหรับ Java

#### การดำเนินการแบบทีละขั้นตอน

**สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กใหม่**

สมมติว่าสมุดงานของคุณได้รับการเติมข้อมูลแล้ว:

```java
Workbook workbook = new Workbook();
```

**บันทึกสมุดงาน**

ระบุไดเรกทอรีเอาท์พุตและชื่อไฟล์ที่จะบันทึกไฟล์ Excel ของคุณ:

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "CreateChart_out.xls");
```

**คำอธิบาย**: เดอะ `save` วิธีการรักษาสถานะปัจจุบันของเวิร์กบุ๊กของคุณไปยังเส้นทางที่ระบุ ทำให้คุณสามารถเข้าถึงได้ในรูปแบบ Excel มาตรฐาน
