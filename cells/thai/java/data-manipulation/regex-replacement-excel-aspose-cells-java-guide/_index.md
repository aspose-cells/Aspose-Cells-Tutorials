## คู่มือการใช้งาน

มาแบ่งกระบวนการใช้งานออกเป็นส่วนๆ ที่สามารถจัดการได้:

### การเริ่มต้นเวิร์กบุ๊กและการดำเนินการแทนที่ Regex

#### ภาพรวม
หัวข้อนี้สาธิตวิธีโหลดเวิร์กบุ๊ก Excel การแทนที่ข้อความตาม regex และการบันทึกการเปลี่ยนแปลง

#### การเริ่มต้นสมุดงาน
เริ่มต้นด้วยการโหลดไฟล์ Excel ของคุณ:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // อัปเดตด้วยเส้นทางไดเร็กทอรีต้นทางของคุณ

// โหลดเวิร์กบุ๊กจากไดเร็กทอรีที่คุณระบุ
Workbook workbook = new Workbook(dataDir + "/SampleRegexReplace.xlsx");
```
**ทำไม** การโหลดเวิร์กบุ๊กเป็นสิ่งสำคัญสำหรับการเข้าถึงเนื้อหาเพื่อปรับเปลี่ยน

#### กำหนดค่าตัวเลือกการแทนที่
ตั้งค่าตัวเลือกสำหรับการแทนที่ข้อความ:
```java
ReplaceOptions replace = new ReplaceOptions();
replace.setCaseSensitive(false);  // การทดแทนไม่ขึ้นอยู่กับความละเอียดอ่อนของตัวพิมพ์ใหญ่และเล็ก
replace.setMatchEntireCellContents(false);  // อนุญาตให้จับคู่บางส่วนภายในเนื้อหาเซลล์
replace.setRegexKey(true);  // เปิดใช้งานการจับคู่รูปแบบ regex
```
**ทำไม** การกำหนดค่าตัวเลือกเหล่านี้ทำให้แน่ใจว่ามีการแทนที่ข้อความที่แม่นยำตามความต้องการของคุณ

#### ดำเนินการแทนที่ตาม Regex
ดำเนินการแทนที่ข้อความ:
```java
// แทนที่อินสแตนซ์ทั้งหมดของ "\\bKIM\\b" ด้วย "^^^^TIM^^^"
workbook.replace("\\bKIM\\b", "^^^TIM^^^", replace);
```
**ทำไม** ขั้นตอนนี้ใช้ regex เพื่อค้นหาและแทนที่รูปแบบเฉพาะในเวิร์กบุ๊กของคุณ

#### บันทึกสมุดงานที่แก้ไขแล้ว
สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณ:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";  // อัปเดตด้วยเส้นทางไดเร็กทอรีเอาท์พุตของคุณ

// บันทึกสมุดงานที่แก้ไขไปยังไฟล์ใหม่
workbook.save(outDir + "/RegexReplace_out.xlsx");
```
**ทำไม** การบันทึกจะช่วยให้มั่นใจว่าการปรับเปลี่ยนทั้งหมดได้รับการเก็บไว้ และสามารถตรวจสอบหรือแชร์ได้

### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่ารูปแบบ regex ได้รับการหลบหนีอย่างถูกต้องสำหรับ Java
- ตรวจสอบเส้นทางไปยังไดเร็กทอรีต้นทางและปลายทางของคุณว่าถูกต้อง
