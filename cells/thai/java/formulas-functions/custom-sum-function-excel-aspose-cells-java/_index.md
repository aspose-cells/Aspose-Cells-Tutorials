## คู่มือการใช้งาน

### คุณสมบัติ: เครื่องมือคำนวณแบบกำหนดเอง
ฟีเจอร์นี้ช่วยให้คุณสามารถปรับเปลี่ยนการทำงานของ Excel ได้ เช่น `SUM` ดำเนินการภายใน Aspose.Cells

#### ภาพรวม
ด้วยการขยายกลไกการคำนวณ คุณสามารถปรับแต่งพฤติกรรมสำหรับฟังก์ชันเฉพาะต่างๆ ได้ บทช่วยสอนนี้มุ่งเน้นที่การปรับเปลี่ยน `SUM` ฟังก์ชั่นเพื่อเพิ่มค่าคงที่เพิ่มเติม

#### การดำเนินการแบบทีละขั้นตอน
##### การขยาย AbstractCalculationEngine
1. **สร้างคลาส CustomEngine**
   เริ่มต้นด้วยการสร้างคลาสที่ขยาย `AbstractCalculationEngine`-
   
   ```java
   import com.aspose.cells.AbstractCalculationEngine;
   import com.aspose.cells.CalculationData;

   public class CustomEngine extends AbstractCalculationEngine {
       @Override
       public void calculate(CalculationData data) {
           // ตรวจสอบว่าฟังก์ชันที่กำลังคำนวณคือ 'SUM' หรือไม่
           if (data.getFunctionName().toUpperCase().equals("SUM")) {
               // ดึงข้อมูลและแก้ไขค่าที่คำนวณในปัจจุบัน
               double val = (double) data.getCalculatedValue();
               val += 30;  // การบวกค่าคงที่ 30
               data.setCalculatedValue(val);
           }
       }
   }
   ```
2. **คำอธิบายพารามิเตอร์**
   - `data.getFunctionName()`: ดึงชื่อฟังก์ชันที่กำลังคำนวณ
   - `data.getCalculatedValue()`: ดึงผลลัพธ์ที่คำนวณในปัจจุบัน
   - `data.setCalculatedValue(double)`: อัปเดตข้อมูลการคำนวณด้วยค่าใหม่
3. **เคล็ดลับการแก้ไขปัญหา**
   ตรวจสอบให้แน่ใจว่าชื่อวิธีการและตรรกะในการตรวจสอบฟังก์ชันไม่คำนึงถึงตัวพิมพ์เล็ก/ใหญ่เพื่อป้องกันข้อผิดพลาดใดๆ ระหว่างการดำเนินการ
