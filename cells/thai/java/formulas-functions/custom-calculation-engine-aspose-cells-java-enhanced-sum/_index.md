## คู่มือการใช้งาน

ตอนนี้เราได้ตั้งค่าสภาพแวดล้อมเรียบร้อยแล้ว ให้เราลองใช้งานฟีเจอร์เครื่องคำนวณแบบกำหนดเอง

### การนำเครื่องมือคำนวณแบบกำหนดเองมาใช้

ส่วนนี้มุ่งเน้นที่การขยายความสามารถของ Aspose.Cells โดยปรับเปลี่ยนวิธีการคำนวณฟังก์ชัน SUM เราจะสร้าง `CustomEngine` คลาสโดยใช้วิธีการแทนที่เพื่อปรับแต่งพฤติกรรม

#### ภาพรวม

เราจะขยายเวลา `AbstractCalculationEngine` และแทนที่มัน `calculate` วิธีการปรับการดำเนินการ SUM โดยเพิ่มค่าคงที่ 30 ให้กับผลลัพธ์แต่ละรายการ

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดเครื่องมือกำหนดเอง**

สร้างคลาส Java ใหม่ชื่อ `CustomEngine`ซึ่งขยายออกไป `AbstractCalculationEngine`. แทนที่ `calculate` วิธีปรับเปลี่ยนฟังก์ชัน SUM:

```java
import com.aspose.cells.AbstractCalculationEngine;
import com.aspose.cells.CalculationData;

class CustomEngine extends AbstractCalculationEngine {
    public void calculate(CalculationData data) {
        if (data.getFunctionName().toUpperCase().equals("SUM")) {
            double val = (double) data.getCalculatedValue();
            val += 30; // บวก 30 ลงในผลลัพธ์รวม
            data.setCalculatedValue(val); // อัพเดทค่าที่คำนวณได้
        }
    }
}
```

**2. ใช้ Custom Engine ในเวิร์กบุ๊ก**

สร้างจุดเข้าสำหรับแอปพลิเคชันของคุณและสาธิตวิธีใช้เครื่องมือที่กำหนดเอง:

```java
import com.aspose.cells.*;

public class CustomCalculationEngineDemo {
    public static void main(String[] args) throws Exception {
        // เริ่มต้นสมุดงานใหม่
        Workbook workbook = new Workbook();

        Worksheet sheet = workbook.getWorksheets().get(0);

        Cell a1 = sheet.getCells().get("A1");
        a1.setFormula("=Sum(B1:B2)"); // ตั้งค่าสูตรเป็นช่วง SUM B1:B2

        sheet.getCells().get("B1").putValue(10); // กำหนดค่า 10 ให้กับเซลล์ B1
        sheet.getCells().get("B2").putValue(10); // กำหนดค่า 10 ให้กับเซลล์ B2

        // คำนวณโดยใช้เครื่องยนต์เริ่มต้น
        workbook.calculateFormula();
        String withoutCustomEngineResult = a1.getStringValue();

        // กำหนดค่าและใช้เครื่องมือคำนวณแบบกำหนดเอง
        CalculationOptions opts = new CalculationOptions();
        opts.setCustomEngine(new CustomEngine());
        workbook.calculateFormula(opts);
        String withCustomEngineResult = a1.getStringValue();

        System.out.println("Without Custom Engine: " + withoutCustomEngineResult);
        System.out.println("With Custom Engine: " + withCustomEngineResult);
    }
}
```
