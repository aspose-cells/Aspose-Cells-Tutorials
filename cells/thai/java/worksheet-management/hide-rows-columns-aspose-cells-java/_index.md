## คู่มือการใช้งาน

มาแบ่งกระบวนการออกเป็นขั้นตอนที่จัดการได้ พร้อมทั้งมีคำอธิบายโดยละเอียดและตัวอย่างโค้ด

### การสร้างอินสแตนซ์ของเวิร์กบุ๊กจากไฟล์ Excel

ในการทำงานกับไฟล์ Excel ที่มีอยู่:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "/book1.xls");
```
แทนที่ `"YOUR_DATA_DIRECTORY"` โดยใช้เส้นทางไฟล์ Excel ของคุณ ซึ่งจะโหลดไฟล์เข้าสู่หน่วยความจำเพื่อจัดการ

### การเข้าถึงเวิร์กชีตและเซลล์

เข้าถึงเวิร์กชีตที่เฉพาะเจาะจงและเซลล์ของมัน:
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Cells cells = worksheet.getCells();
```
ที่นี่เราเรียกค้นเวิร์กชีตแรก (ดัชนี 0) และรับมัน `Cells` คัดค้านให้ดำเนินการต่อไป

### การซ่อนแถว

หากต้องการซ่อนแถวในแผ่นงาน Excel ของคุณ:
```java
cells.hideRow(2); // ซ่อนแถวที่สาม (ตามดัชนี)
```
การ `hideRow()` วิธีนี้ใช้ดัชนีเริ่มจาก 0 ดังนั้น `hideRow(2)` ซ่อนแถวที่สาม

### การซ่อนคอลัมน์

ในทำนองเดียวกันการซ่อนคอลัมน์:
```java
cells.hideColumn(1); // ซ่อนคอลัมน์ที่สอง
```
คอลัมน์มีดัชนีเป็นศูนย์เช่นกัน `hideColumn(1)` โดยกำหนดเป้าหมายไปที่คอลัมน์ที่สอง

### การบันทึกสมุดงานที่แก้ไข

หลังจากทำการเปลี่ยนแปลงแล้ว ให้บันทึกสมุดงานของคุณ:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/HidingRowsandColumns_out.xls");
```
แทนที่ `"YOUR_OUTPUT_DIRECTORY"` ด้วยเส้นทางเอาต์พุตที่คุณต้องการเพื่อสรุปการเปลี่ยนแปลงในเอกสาร Excel
