# สร้างผลรวมย่อยใน Excel โดยใช้ Aspose.Cells สำหรับ Java: คู่มือฉบับสมบูรณ์

การสร้างผลรวมย่อยในเวิร์กบุ๊ก Excel เป็นงานสำคัญในการสรุปชุดข้อมูลขนาดใหญ่ได้อย่างมีประสิทธิภาพ ด้วยไลบรารี Aspose.Cells อันทรงพลังสำหรับ Java คุณสามารถทำให้กระบวนการนี้เป็นอัตโนมัติได้ด้วยโปรแกรม บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ Aspose.Cells เพื่อสร้างผลรวมย่อยภายในแอปพลิเคชัน Java ของคุณ

## สิ่งที่คุณจะได้เรียนรู้
- การตั้งค่า Aspose.Cells สำหรับ Java ในโครงการของคุณ
- คำแนะนำทีละขั้นตอนในการสร้างผลรวมย่อยในแผ่นงาน Excel
- กรณีการใช้งานจริงสำหรับการนำฟีเจอร์นี้ไปใช้
- เคล็ดลับประสิทธิภาพการทำงานและแนวทางปฏิบัติที่ดีที่สุดเมื่อใช้ Aspose.Cells

มาเจาะลึกข้อกำหนดเบื้องต้นก่อนที่เราจะเริ่มเขียนโค้ดกัน

### การตั้งค่า Aspose.Cells สำหรับ Java
Aspose.Cells สำหรับ Java เป็นไลบรารีที่มีประสิทธิภาพสำหรับการจัดการไฟล์ Excel คุณสามารถเพิ่มไลบรารีนี้ลงในโปรเจ็กต์ของคุณได้ดังนี้:

**การใช้ Maven:**

เพิ่มการอ้างอิงนี้ให้กับของคุณ `pom.xml` ไฟล์:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**การใช้ Gradle:**

รวมสิ่งต่อไปนี้ไว้ในของคุณ `build.gradle` ไฟล์:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

### คู่มือการใช้งาน
ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมของคุณเรียบร้อยแล้ว มาเน้นที่การนำผลรวมย่อยไปใช้กัน

#### ภาพรวมของการสร้างผลรวมย่อย
การรวมผลรวมย่อยช่วยในการสรุปข้อมูลด้วยการใช้ฟังก์ชันรวม เช่น sum, average หรือ count ในช่วงหนึ่ง ด้วย Aspose.Cells การดำเนินการนี้จะทำโดยการเขียนโปรแกรมโดยใช้ `subtotal` วิธี.

##### ขั้นตอนที่ 1: เริ่มต้นคอลเลกชันสมุดงานและเซลล์
เริ่มต้นด้วยการโหลดเวิร์กบุ๊กของคุณและเข้าถึงเซลล์ของมัน:
```java
// โหลดไฟล์ Excel
Workbook workbook = new Workbook(dataDir + "book1.xls");

// เข้าถึงคอลเลกชันเซลล์ของเวิร์กชีตแรก
Cells cells = workbook.getWorksheets().get(0).getCells();
```

##### ขั้นตอนที่ 2: กำหนดพื้นที่เซลล์สำหรับการรวมย่อย
ระบุช่วงข้อมูลที่คุณต้องการใช้ผลรวมย่อย:
```java
// กำหนดพื้นที่ตั้งแต่ B3 ถึง C19 (ดัชนีฐาน 1)
CellArea ca = new CellArea();
ca.StartRow = 2; // แถว B3 ในดัชนีฐานศูนย์
ca.EndRow = 18; // แถว C19 ในดัชนีฐานศูนย์
ca.StartColumn = 1;
cac.EndColumn = 2;
```

##### ขั้นตอนที่ 3: ใช้ผลรวมย่อย
ใช้ `subtotal` วิธีการคำนวณและแทรกผลรวมย่อย:
```java
// ใช้ผลรวมย่อยในคอลัมน์ C (ดัชนี 1) ด้วยฟังก์ชัน SUM
cells.subtotal(ca, 0, ConsolidationFunction.SUM, new int[] { 1 });
```
- **คำอธิบายพารามิเตอร์**-
  - `ca`ช่วงของเซลล์
  - `0`: ระบุตำแหน่งแถวทั้งหมด
  - `ConsolidationFunction.SUM`: กำหนดฟังก์ชันที่จะนำไปใช้ (SUM ในกรณีนี้)
  - `new int[]{1}`:ดัชนีคอลัมน์ที่ใช้การรวมผลรวมย่อย

##### ขั้นตอนที่ 4: บันทึกและส่งออก
สุดท้าย ให้บันทึกสมุดงานของคุณด้วยผลรวมย่อยใหม่:
```java
// บันทึกไฟล์ Excel ที่ถูกแก้ไข
dataDir + "CreatingSubtotals_out.xls";

// ยืนยันความสำเร็จ
System.out.println("Process completed successfully");
```
