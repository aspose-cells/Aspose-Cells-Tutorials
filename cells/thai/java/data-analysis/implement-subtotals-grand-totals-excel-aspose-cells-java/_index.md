## คู่มือการใช้งาน

### ภาพรวม

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการใช้ผลรวมย่อยและปรับแต่งป้ายกำกับในเวิร์กชีต Excel ของคุณโดยใช้ Aspose.Cells สำหรับ Java

### คำแนะนำทีละขั้นตอน

#### 1. โหลดสมุดงานต้นฉบับของคุณ

ขั้นแรก โหลดไฟล์ Excel ที่มีข้อมูลของคุณ:

```java
// เส้นทางไปยังไดเร็กทอรีเอกสาร
String dataDir = Utils.getSharedDataDir(ImplementSubtotalGrandTotallabels.class) + "TechnicalArticles/";

// โหลดสมุดงานต้นฉบับของคุณ
Workbook wb = new Workbook(dataDir + "sample.xlsx");
```

#### 2. ปรับแต่งป้ายผลรวมย่อยและผลรวมทั้งหมด

หากต้องการแปลฉลากเหล่านี้ ให้ตั้งค่าการตั้งค่าสากล:

```java
// ตั้งค่าการตั้งค่าสากลเพื่อเปลี่ยนแปลงชื่อผลรวมย่อยและผลรวมทั้งหมด
GlobalizationSettings gsi = new GlobalizationSettingsImp();
wb.getSettings().setGlobalizationSettings(gsi);
```

#### 3. เข้าถึงแผ่นงานของคุณ

เข้าถึงแผ่นงานเฉพาะที่คุณต้องการใช้ผลรวมย่อย:

```java
// เข้าถึงแผ่นงานแรก
Worksheet ws = wb.getWorksheets().get(0);
```

#### 4. ใช้ฟังก์ชันผลรวมย่อย

ใช้ `subtotal` วิธีการบนช่วงเซลล์ที่คุณต้องการ โดยระบุคอลัมน์ที่จะรวมผลรวมย่อย และใช้ฟังก์ชันการรวม เช่น `SUM`-

```java
// ใช้ผลรวมย่อยใน A1:B10 สำหรับคอลัมน์ 2, 3 และ 4 (การสร้างดัชนีเริ่มจาก 0)
CellArea ca = CellArea.createCellArea("A1", "B10");
ws.getCells().subtotal(ca, 0, ConsolidationFunction.SUM, new int[] { 2, 3, 4 });
```

#### 5. ปรับความกว้างของคอลัมน์

เพื่อให้มองเห็นได้ชัดเจนยิ่งขึ้น คุณสามารถปรับความกว้างของคอลัมน์ได้:

```java
// กำหนดความกว้างของคอลัมน์แรก
ws.getCells().setColumnWidth(0, 40);
```

#### 6. บันทึกสมุดงานของคุณ

สุดท้าย ให้บันทึกสมุดงานของคุณโดยใช้การเปลี่ยนแปลงทั้งหมด:

```java
// บันทึกไฟล์ excel เอาท์พุต
wb.save(dataDir + "ImplementTotallabels_out.xlsx");
```
