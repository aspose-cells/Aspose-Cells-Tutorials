## คู่มือการใช้งาน
### โหลดสมุดงาน (H2)
**ภาพรวม**:การโหลดเวิร์กบุ๊กที่มีอยู่เป็นสิ่งสำคัญสำหรับการเข้าถึงและการจัดการตารางสรุปข้อมูล
#### ขั้นตอนที่ 1: นำเข้าคลาสที่จำเป็น
```java
import com.aspose.cells.Workbook;
```
#### ขั้นตอนที่ 2: โหลดเวิร์กบุ๊ก
ระบุไดเรกทอรีที่ไฟล์ Excel ของคุณตั้งอยู่:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "/PivotTable.xls");
```
*คำอธิบาย*- `Workbook` หมายถึงไฟล์ Excel และการโหลดไฟล์จะทำให้คุณสามารถเข้าถึงแผ่นงานและตารางสรุปข้อมูลได้
### คอลเลกชันเวิร์กชีตและตารางสรุปข้อมูล Access (H2)
**ภาพรวม**: เข้าถึงเวิร์กชีตซึ่งมีตารางสรุปข้อมูลของคุณอยู่
#### ขั้นตอนที่ 1: นำเข้าคลาส
```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.PivotTableCollection;
```
#### ขั้นตอนที่ 2: ดึงข้อมูลเวิร์กชีตและตารางสรุปข้อมูล
เข้าถึงเวิร์กชีตแรกและตารางสรุปข้อมูล:
```java
Worksheet sheet = workbook.getWorksheets().get(0);
PivotTableCollection pivotTables = sheet.getPivotTables();
```
*คำอธิบาย*:เวิร์กชีตเป็นที่เก็บข้อมูล รวมถึงตารางสรุปข้อมูลที่สรุปข้อมูล
### การแก้ไขฟิลด์ข้อมูลตารางสรุปข้อมูล (H2)
**ภาพรวม**การปรับเปลี่ยนฟิลด์ข้อมูลในตารางสรุปมักจำเป็นเพื่อสะท้อนตรรกะทางธุรกิจหรือรายงานที่อัปเดต
#### ขั้นตอนที่ 1: ล้างข้อมูลที่มีอยู่
```java
import com.aspose.cells.PivotTable;
import com.aspose.cells.PivotFieldType;

PivotTable pivotTable = pivotTables.get(0);
pivotTable.getDataFields().clear();
```
*คำอธิบาย*ขั้นตอนนี้จะลบฟิลด์ข้อมูลที่มีอยู่ทั้งหมด ช่วยให้สามารถเพิ่มฟิลด์ข้อมูลใหม่ที่ตอบโจทย์ความต้องการปัจจุบันได้
#### ขั้นตอนที่ 2: เพิ่มฟิลด์ข้อมูลใหม่
```java
pivotTable.addFieldToArea(PivotFieldType.DATA, "Betrag Netto FW");
```
*คำอธิบาย*- `addFieldToArea` เพิ่มเขตข้อมูลเฉพาะลงในตารางสรุปข้อมูลของคุณ เพื่อเพิ่มความสามารถในการวิเคราะห์ข้อมูล
### รีเฟรชและคำนวณข้อมูลตารางสรุปข้อมูล (H2)
**ภาพรวม**:หลังจากปรับเปลี่ยนแล้ว การรีเฟรช และการคำนวณใหม่จะช่วยให้ตารางสรุปข้อมูลแสดงข้อมูลที่ถูกต้องแม่นยำ
#### ขั้นตอนที่ 1: รีเฟรชและคำนวณใหม่
```java
pivotTable.setRefreshDataFlag(false);
pivotTable.refreshData();
pivotTable.calculateData();
```
*คำอธิบาย*:กระบวนการนี้จะอัปเดตข้อมูลของตารางสรุปข้อมูลตามการเปลี่ยนแปลงที่เกิดขึ้นกับโครงสร้างหรือฟิลด์ข้อมูลต้นทาง
### บันทึกสมุดงานที่แก้ไขแล้ว (H2)
**ภาพรวม**สุดท้ายให้บันทึกสมุดงานของคุณพร้อมการแก้ไขทั้งหมด
#### ขั้นตอนที่ 1: ส่งออกสมุดงานที่อัปเดต
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/ClearPivotFields_out.xlsx");
```
*คำอธิบาย*การบันทึกไฟล์จะช่วยให้มั่นใจว่าการเปลี่ยนแปลงทั้งหมดได้รับการรักษาไว้และสามารถเข้าถึงได้เพื่อการใช้งานในอนาคต
