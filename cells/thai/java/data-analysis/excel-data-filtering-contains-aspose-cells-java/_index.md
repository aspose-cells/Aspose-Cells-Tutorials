หากต้องการใช้ Aspose.Cells ในโปรเจ็กต์ของคุณ คุณต้องรวม Aspose.Cells ไว้เป็นส่วนที่ต้องพึ่งพา นี่คือวิธีดำเนินการโดยใช้ Maven หรือ Gradle:

เพิ่มการอ้างอิงต่อไปนี้ให้กับของคุณ `pom.xml` ไฟล์:


## คู่มือการใช้งาน

ในส่วนนี้ เราจะแนะนำการสร้างตัวกรองแบบกำหนดเองเพื่อระบุแถวที่มีข้อความเฉพาะโดยใช้ Aspose.Cells สำหรับ Java

### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

เริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีอยู่ นี่คือแหล่งข้อมูลของคุณ

```java
Workbook workbook = new Workbook(srcDir + "sourseSampleCountryNames.xlsx");
```

### ขั้นตอนที่ 2: การเข้าถึงแผ่นงาน

เข้าถึงแผ่นงานที่มีข้อมูลที่คุณต้องการกรอง:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

### ขั้นตอนที่ 3: การตั้งค่าตัวกรองอัตโนมัติ

สร้าง `AutoFilter` โดยระบุช่วง ซึ่งจะกำหนดพื้นที่ของชีตของคุณที่จะถูกกรอง:

```java
worksheet.getAutoFilter().setRange("A1:A18");
```

### ขั้นตอนที่ 4: การใช้ตัวกรองแบบกำหนดเอง

ใช้ `custom()` วิธีการใช้ตัวกรองเพื่อค้นหาแถวที่มีข้อความเฉพาะ ในกรณีนี้คือ "Ba":

```java
worksheet.getAutoFilter().custom(0, FilterOperatorType.CONTAINS, "Ba");
```

**คำอธิบาย**- 
- พารามิเตอร์แรก (`0`) ระบุดัชนีคอลัมน์
- `FilterOperatorType.CONTAINS` เป็นตรรกะการกรองที่นำมาใช้
- `"Ba"` เป็นสตริงที่คุณกำลังค้นหาภายในช่วงที่ระบุ

### ขั้นตอนที่ 5: รีเฟรชและบันทึก

หลังจากตั้งค่าตัวกรองแล้ว ให้รีเฟรชเพื่อใช้การเปลี่ยนแปลงและบันทึกสมุดงานของคุณ:

```java
worksheet.getAutoFilter().refresh();
workbook.save(outDir + "outSourseSampleCountryNames.xlsx");
```

**เคล็ดลับการแก้ไขปัญหา**:หากคุณพบปัญหาเกี่ยวกับเส้นทางไฟล์หรือช่วงข้อมูล โปรดตรวจสอบให้แน่ใจว่าได้ระบุไว้อย่างถูกต้องตามโครงสร้างไดเร็กทอรีของโครงการของคุณ

ต่อไปนี้คือสถานการณ์จริงบางส่วนที่สามารถนำฟังก์ชันนี้ไปใช้:

2. **การจัดการข้อมูลลูกค้า**:แยกบันทึกลูกค้าที่มีคุณลักษณะเฉพาะในระบบ CRM
3. **การติดตามสินค้าคงคลัง**:ค้นหาสินค้าที่มีแท็กหรือตัวระบุเฉพาะได้อย่างรวดเร็ว
