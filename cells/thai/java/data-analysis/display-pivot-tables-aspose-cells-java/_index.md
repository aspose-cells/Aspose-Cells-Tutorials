- ขั้นตอนการนำเสนอตารางสรุปข้อมูลในรูปแบบตาราง

เมื่อสิ้นสุดบทช่วยสอนนี้ คุณจะเชี่ยวชาญการแสดงตารางสรุปข้อมูลในรูปแบบต่างๆ โดยใช้ Aspose.Cells Java มาเจาะลึกสิ่งที่คุณต้องมีเพื่อเริ่มต้นกันเลย

หากต้องการใช้ Aspose.Cells สำหรับ Java คุณจะต้องรวมไว้ในโปรเจ็กต์ของคุณ คุณมีสองตัวเลือก: Maven หรือ Gradle

เพิ่มการอ้างอิงต่อไปนี้ให้กับของคุณ `pom.xml` ไฟล์:

</dependency>

## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็น 3 ส่วน คือ รูปแบบย่อ รูปแบบโครงร่าง และรูปแบบตาราง

### แสดงตารางสรุปข้อมูลในรูปแบบกะทัดรัด

**ภาพรวม:** การแสดงตารางสรุปในรูปแบบกะทัดรัดช่วยประหยัดพื้นที่ในขณะที่ยังคงความชัดเจน

#### ขั้นตอนที่ 1: โหลดไฟล์ Excel
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```
*ทำไม* นี่จะโหลดไฟล์ Excel ต้นฉบับของคุณลงในหน่วยความจำ

#### ขั้นตอนที่ 2: เข้าถึงเวิร์กชีตและตารางสรุปข้อมูล
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### ขั้นตอนที่ 3: ตั้งค่ารูปแบบกะทัดรัด
```java
pivotTable.showInCompactForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/CompactForm.xlsx");
```
*ทำไม* การกำหนดค่านี้จะแสดงตารางสรุปในรูปแบบกะทัดรัดและบันทึกไว้

### แสดงตารางสรุปในรูปแบบเค้าร่าง

**ภาพรวม:** แบบโครงร่างเหมาะสำหรับข้อมูลแบบลำดับชั้น โดยอนุญาตให้ผู้ใช้ขยายหรือยุบรายละเอียดได้

#### ขั้นตอนที่ 1: โหลดสมุดงาน
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### ขั้นตอนที่ 2: เข้าถึงส่วนประกอบที่จำเป็น
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### ขั้นตอนที่ 3: กำหนดค่าแบบฟอร์มโครงร่าง
```java
pivotTable.showInOutlineForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/OutlineForm.xlsx");
```
*ทำไม* ขั้นตอนนี้จะกำหนดตารางสรุปให้เป็นรูปแบบโครงร่างและรับรองว่าข้อมูลจะได้รับการอัปเดต

### แสดงตารางสรุปในรูปแบบตาราง

**ภาพรวม:** รูปแบบตารางจะแสดงข้อมูลทั้งหมดเป็นแถว เหมาะสำหรับการวิเคราะห์โดยละเอียด

#### ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊ก
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### ขั้นตอนที่ 2: เข้าถึงส่วนประกอบ
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### ขั้นตอนที่ 3: ตั้งค่ารูปแบบตาราง
```java
pivotTable.showInTabularForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/TabularForm.xlsx");
```
*ทำไม* การกำหนดค่านี้จะแสดงตารางสรุปในรูปแบบตาราง

## การประยุกต์ใช้งานจริง

ต่อไปนี้เป็นกรณีการใช้งานจริงในการแสดงตารางสรุปในรูปแบบต่างๆ:

1. **รายงานทางการเงิน:** ใช้รูปแบบกะทัดรัดเพื่อสรุปข้อมูลทางการเงินได้อย่างรวดเร็ว
2. **การวิเคราะห์การขาย:** แบบฟอร์มโครงร่างสามารถช่วยเจาะลึกข้อมูลการขายตามลำดับชั้นได้
3. **การจัดการสินค้าคงคลัง:** รูปแบบตารางแสดงรายการสิ่งของโดยละเอียด

ความเป็นไปได้ในการบูรณาการได้แก่การเชื่อมต่อกับเครื่องมือ BI และแดชบอร์ดเพื่อการแสดงข้อมูลที่ได้รับการปรับปรุง
