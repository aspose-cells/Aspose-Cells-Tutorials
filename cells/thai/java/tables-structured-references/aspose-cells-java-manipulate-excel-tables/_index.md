## คู่มือการใช้งาน

ตอนนี้สภาพแวดล้อมของคุณได้รับการตั้งค่าแล้ว มาเข้าถึงและจัดการตาราง Excel ผ่านเซลล์กัน

### การเข้าถึงตาราง Excel จากเซลล์

#### 1. โหลดเวิร์กบุ๊ก

ขั้นแรกให้สร้าง `Workbook` ตัวอย่างเช่นโดยการโหลดไฟล์ Excel ที่มีอยู่:

```java
String dataDir = "path/to/your/directory/";
Workbook workbook = new Workbook(dataDir + "ATFromCell.xlsx");
```

#### 2. เข้าถึงเวิร์กชีตและเซลล์

เข้าถึงเวิร์กชีตแรกและเซลล์เฉพาะภายในตารางที่คุณสนใจ:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("D5");
```

#### 3. การจัดการข้อมูลตาราง

ใช้ `TableObject` การปรับเปลี่ยนข้อมูลโดยใช้การชดเชยแถวและคอลัมน์:

```java
// ใส่ค่าภายในเซลล์ D5
cell.putValue("D5 Data");

// เข้าถึงตารางจากเซลล์นี้
ListObject table = cell.getTable();

// เพิ่มค่าบางค่าที่ตำแหน่งออฟเซ็ตที่ระบุ
table.putCellValue(1, 1, "Offset [2,2]");
```

#### 4. บันทึกสมุดงาน

สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณกลับไปยังไฟล์ Excel:

```java
workbook.save(dataDir + "ATFromCell_out.xlsx");
```
