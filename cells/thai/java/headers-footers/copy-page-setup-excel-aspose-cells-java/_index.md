## คู่มือการใช้งาน
ตอนนี้เรามาดูคุณลักษณะหลักของการคัดลอกการตั้งค่าหน้าระหว่างเวิร์กชีตกัน

### ภาพรวม
การคัดลอกการตั้งค่าหน้ากระดาษเกี่ยวข้องกับการทำซ้ำการตั้งค่าต่างๆ เช่น ขนาดกระดาษและทิศทางจากเวิร์กชีตหนึ่งไปยังอีกเวิร์กชีตหนึ่ง วิธีนี้จะช่วยให้แน่ใจถึงความสม่ำเสมอในแผ่นงานหลายแผ่นในเวิร์กบุ๊ก

#### สร้างสมุดงานและแผ่นงาน (H3)
เริ่มต้นด้วยการสร้างเวิร์กบุ๊กใหม่และเพิ่มเวิร์กชีตทดสอบสองแผ่น:

```java
import com.aspose.cells.*;

public class CopyPageSetupSettingsFromSourceWorksheetToDestinationWorksheet {
    public static void main(String[] args) throws Exception {
        // การเริ่มต้นสมุดงาน
        Workbook wb = new Workbook();

        // เพิ่มแผ่นงาน
        wb.getWorksheets().add("TestSheet1");
        wb.getWorksheets().add("TestSheet2");

        System.out.println("Workbooks and worksheets created successfully.");
    }
}
```

#### ตั้งค่าขนาดกระดาษ (H3)
กำหนดขนาดกระดาษสำหรับ `TestSheet1` เพื่อสาธิตการตั้งค่าการคัดลอก:

```java
// เข้าถึง TestSheet1
Worksheet TestSheet1 = wb.getWorksheets().get("TestSheet1");

// ตั้งค่าขนาดกระดาษของ TestSheet1 เป็น PAPER_A_3_EXTRA_TRANSVERSE
TestSheet1.getPageSetup().setPaperSize(PaperSizeType.PAPER_A_3_EXTRA_TRANSVERSE);

System.out.println("Paper size set for TestSheet1.");
```

#### คัดลอกการตั้งค่าหน้า (H3)
ตอนนี้คัดลอกการตั้งค่าหน้าจาก `TestSheet1` ถึง `TestSheet2`-

```java
// เข้าถึง TestSheet2
Worksheet TestSheet2 = wb.getWorksheets().get("TestSheet2");

// คัดลอก PageSetup จาก TestSheet1 ไปยัง TestSheet2
TestSheet2.getPageSetup().copy(TestSheet1.getPageSetup(), new CopyOptions());

System.out.println("Page setup copied successfully.");
```
