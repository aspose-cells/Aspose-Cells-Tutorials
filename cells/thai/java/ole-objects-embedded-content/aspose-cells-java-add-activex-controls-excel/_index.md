# วิธีการเพิ่มตัวควบคุม ActiveX ลงใน Excel โดยใช้ Aspose.Cells Java: คู่มือฉบับสมบูรณ์

## คู่มือการใช้งาน

### การสร้างและการเพิ่มตัวควบคุม ActiveX ลงในเวิร์กชีต

#### ภาพรวม
การเพิ่มตัวควบคุม ActiveX เหมือนกับปุ่มสลับ เกี่ยวข้องกับการสร้างตัวควบคุมนั้นภายในคอลเลกชันรูปร่างของเวิร์กชีต หัวข้อนี้จะแนะนำคุณตลอดขั้นตอนนี้

#### คำแนะนำทีละขั้นตอน
**1. สร้างเวิร์กบุ๊กและเข้าถึงเวิร์กชีตแรก**
สร้างเวิร์กบุ๊กของคุณและเข้าถึงเวิร์กชีตแรก:

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

// เริ่มต้นสมุดงาน
Workbook wb = new Workbook();

// รับแผ่นงานแรก
Worksheet sheet = wb.getWorksheets().get(0);
```

**2. เพิ่มปุ่มสลับ ActiveX Control**
เพิ่มปุ่มสลับให้กับเวิร์กชีตของคุณ:

```java
import com.aspose.cells.ControlType;
import com.aspose.cells.Shape;

// เพิ่มปุ่มสลับภายในคอลเลกชันรูปทรงตามตำแหน่งและขนาดที่ระบุ
Shape s = sheet.getShapes().addActiveXControl(
    ControlType.TOGGLE_BUTTON, 4, 0, 4, 0, 100, 30);
```

**3. กำหนดค่าการควบคุม ActiveX**
ตั้งค่าคุณสมบัติเช่นการเชื่อมโยงเซลล์เพื่อปรับปรุงการโต้ตอบ:

```java
import com.aspose.cells.ActiveXControl;

// เข้าถึงวัตถุควบคุม ActiveX
ActiveXControl c = s.getActiveXControl();

// เชื่อมโยงการควบคุมกับเซลล์
c.setLinkedCell("A1");
```

**4. บันทึกสมุดงาน**
บันทึกสมุดงานของคุณในรูปแบบที่ต้องการ:

```java
import com.aspose.cells.SaveFormat;

// กำหนดไดเรกทอรีเอาท์พุต
String dataDir = "path/to/your/directory/";

// บันทึกสมุดงานเป็นไฟล์ Excel
wb.save(dataDir + "AAXControl_out.xlsx", SaveFormat.XLSX);
```

### เคล็ดลับการแก้ไขปัญหา
- ให้แน่ใจว่ามีการรวมสิ่งที่ต้องพึ่งพาเพื่อป้องกัน `ClassNotFoundException`-
- ตรวจสอบเส้นทางและสิทธิ์อนุญาตไดเร็กทอรีเมื่อบันทึกไฟล์
