# การวิเคราะห์ข้อมูล Excel Pivot


## บทนำสู่ Aspose.Cells สำหรับ Java

ก่อนที่จะเจาะลึกรายละเอียดการวิเคราะห์ข้อมูล เรามาทำความรู้จักกับ Aspose.Cells สำหรับ Java กันก่อน ไลบรารี Java นี้เป็นส่วนหนึ่งของกลุ่มผลิตภัณฑ์ Aspose.Cells ซึ่งมีชื่อเสียงในด้านความสามารถในการจัดการไฟล์ Excel Aspose.Cells สำหรับ Java ช่วยให้คุณสามารถสร้าง แก้ไข และจัดการเวิร์กบุ๊ก Excel เวิร์กชีต แผนภูมิ และตารางสรุปข้อมูลโดยการเขียนโปรแกรม

## ข้อกำหนดเบื้องต้น

หากต้องการปฏิบัติตามคู่มือนี้ คุณจะต้องมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา Java: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Java ไว้ในระบบของคุณแล้ว
-  Aspose.Cells สำหรับ Java: ดาวน์โหลดและรวมไลบรารี Aspose.Cells สำหรับ Java ไว้ในโปรเจ็กต์ของคุณ คุณสามารถค้นหาลิงก์ดาวน์โหลด[ที่นี่](https://releases.aspose.com/cells/java/).
- ข้อมูลตัวอย่าง: เตรียมข้อมูล Excel ที่คุณต้องการวิเคราะห์

## การสร้างเวิร์กบุ๊ก Excel ใหม่

เริ่มต้นด้วยการสร้างเวิร์กบุ๊ก Excel ใหม่โดยใช้ Aspose.Cells สำหรับ Java ซึ่งจะเป็นพื้นฐานสำหรับการวิเคราะห์ข้อมูลของเรา

```java
// โค้ด Java เพื่อสร้างเวิร์กบุ๊ก Excel ใหม่
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);
```

## การนำเข้าข้อมูลลงใน Excel

ตอนนี้เรามีเวิร์กบุ๊กเปล่าแล้ว เราสามารถนำเข้าข้อมูลของเราเข้าไปในเวิร์กบุ๊กนั้นได้ คุณสามารถอ่านข้อมูลจากแหล่งต่าง ๆ เช่น ฐานข้อมูล ไฟล์ CSV หรือแม้แต่การป้อนข้อมูลด้วยตนเอง

```java
// โค้ด Java สำหรับนำเข้าข้อมูลลงใน Excel
Cells cells = worksheet.getCells();
cells.importData(yourDataArray, 0, 0, importOptions);
```

## การสร้างตารางสรุปข้อมูล

ตารางสรุปข้อมูลเป็นวิธีที่มีประสิทธิภาพในการสรุปและวิเคราะห์ข้อมูลใน Excel มาสร้างตารางสรุปข้อมูลในเวิร์กบุ๊กของเราเพื่ออำนวยความสะดวกในการวิเคราะห์ข้อมูลกัน

```java
// โค้ด Java ในการสร้างตารางสรุปข้อมูล
PivotTableCollection pivotTables = worksheet.getPivotTables();
int index = pivotTables.add("A1", "E10", "PivotTable");
PivotTable pivotTable = pivotTables.get(index);
```

## การกำหนดฟิลด์ตารางสรุปข้อมูล

เพื่อดำเนินการวิเคราะห์ข้อมูลอย่างมีประสิทธิภาพ เราจำเป็นต้องระบุฟิลด์ที่จะรวมไว้ในตารางสรุปข้อมูลของเรา ฟิลด์เหล่านี้สามารถเป็นคอลัมน์จากข้อมูลที่เรานำเข้าได้

```java
// โค้ด Java สำหรับกำหนดฟิลด์ตารางสรุปข้อมูล
PivotFieldCollection pivotFields = pivotTable.getRowFields();
pivotFields.add(cells, 0); // เพิ่มคอลัมน์แรกเป็นฟิลด์แถว
```

## การรวบรวมข้อมูล

เมื่อตั้งค่าตารางสรุปข้อมูลเรียบร้อยแล้ว เราสามารถรวมและสรุปข้อมูลตามความต้องการได้ คุณสามารถระบุฟังก์ชันการรวมข้อมูล เช่น ผลรวม ค่าเฉลี่ย การนับ เป็นต้น

```java
// โค้ด Java สำหรับรวบรวมข้อมูลในตารางสรุปข้อมูล
pivotTable.addFieldToArea(0, PivotFieldType.DATA); // เพิ่มคอลัมน์แรกเป็นเขตข้อมูล
pivotTable.getDataFields().get(0).setFunction(PivotFieldFunctionType.SUM); // ใช้ฟังก์ชั่น SUM
```

## การปรับแต่งเค้าโครงตารางสรุปข้อมูล

เพื่อให้ตารางสรุปข้อมูลมีข้อมูลมากขึ้น เราสามารถปรับแต่งเค้าโครง เช่น การเพิ่มตัวกรอง การเรียงลำดับ และการเปลี่ยนแปลงตำแหน่งฟิลด์

```java
// รหัส Java สำหรับปรับแต่งเค้าโครงของตารางสรุปข้อมูล
pivotTable.addFieldToArea(1, PivotFieldType.PAGE); // เพิ่มคอลัมน์ที่ 2 เป็นฟิลด์หน้า (ตัวกรอง)
pivotTable.getField(1).setDisplayAutomaticSubtotals(false); // ปิดใช้งานผลรวมย่อยอัตโนมัติ
```

## การวิเคราะห์ข้อมูล

ตอนนี้เราสร้างและปรับแต่งตารางสรุปข้อมูลเรียบร้อยแล้ว ถึงเวลาวิเคราะห์ข้อมูล คุณสามารถใช้ตารางสรุปข้อมูลเพื่อสร้างข้อมูลเชิงลึก ตรวจจับแนวโน้ม และตัดสินใจอย่างรอบรู้

## บทสรุป

ในคู่มือนี้ เราได้ศึกษาวิธีดำเนินการวิเคราะห์ข้อมูลใน Excel โดยใช้ Aspose.Cells สำหรับ Java โดยเริ่มต้นด้วยการสร้างเวิร์กบุ๊กใหม่ นำเข้าข้อมูล และสร้างตารางสรุปข้อมูล จากนั้นจึงกำหนดฟิลด์ของตารางสรุปข้อมูล รวบรวมข้อมูล และปรับแต่งเค้าโครง ด้วยเครื่องมือเหล่านี้ คุณสามารถปลดล็อกศักยภาพทั้งหมดของการวิเคราะห์ข้อมูลใน Excel ด้วย Java ได้
