# เรียนรู้ระบบวันที่ 1904 ใน Excel โดยใช้ Aspose.Cells Java เพื่อการดำเนินการเซลล์อย่างมีประสิทธิภาพ

## การแนะนำ

การจัดการข้อมูลประวัติใน Excel อาจเป็นเรื่องท้าทายเนื่องจากระบบวันที่ที่แตกต่างกัน เช่น ระบบวันที่ 1904 ด้วย Aspose.Cells สำหรับ Java คุณสามารถกำหนดค่าและจัดการสเปรดชีต Excel ได้อย่างง่ายดายในขณะที่ยังคงความเข้ากันได้กับระบบวันที่ต่างๆ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการเริ่มต้นเวิร์กบุ๊กใหม่ การเปิดใช้งานระบบวันที่ 1904 และการบันทึกการเปลี่ยนแปลงของคุณโดยใช้ Aspose.Cells Java

**สิ่งที่คุณจะได้เรียนรู้:**
- การเริ่มต้นเวิร์กบุ๊ก Aspose.Cells ใน Java
- การเปิดใช้งานระบบวันที่ 1904 ในไฟล์ Excel
- การบันทึกสมุดงานของคุณด้วยการกำหนดค่าที่อัปเดต

มาเจาะลึกข้อกำหนดเบื้องต้นที่จำเป็นก่อนที่คุณจะเริ่มต้น

## ข้อกำหนดเบื้องต้น

หากต้องการทำตามบทช่วยสอนนี้ ให้แน่ใจว่าคุณมี:
- **ชุดพัฒนา Java (JDK)** ติดตั้งไว้ในเครื่องของคุณแล้ว แนะนำให้ใช้เวอร์ชัน 8 ขึ้นไป
- **เมเวน** หรือ **แกรเดิล** สำหรับการจัดการการอ้างอิงขึ้นอยู่กับการตั้งค่าโครงการของคุณ
- ความรู้พื้นฐานเกี่ยวกับ Java และความคุ้นเคยกับการดำเนินการไฟล์ Excel

## การตั้งค่า Aspose.Cells สำหรับ Java

หากต้องการใช้ Aspose.Cells สำหรับ Java ในโปรเจ็กต์ของคุณ ให้เพิ่มเป็นส่วนที่ต้องพึ่งพา ด้านล่างนี้เป็นคำแนะนำสำหรับการตั้งค่า Maven และ Gradle:

### **เมเวน**

เพิ่มการอ้างอิงต่อไปนี้ให้กับของคุณ `pom.xml` ไฟล์:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### **แกรเดิล**

รวมบรรทัดนี้ไว้ในของคุณ `build.gradle` ไฟล์:

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

#### การเริ่มต้นขั้นพื้นฐาน

หากต้องการเริ่มต้น Aspose.Cells ในแอปพลิเคชัน Java ของคุณ ให้รวมคำสั่งนำเข้านี้:

import com.aspose.cells.Workbook;

## คู่มือการใช้งาน

### เริ่มต้นและโหลดสมุดงาน

#### ภาพรวม

ขั้นแรกให้สร้างอินสแตนซ์ใหม่ของ `Workbook` และโหลดไฟล์ Excel ที่มีอยู่ การตั้งค่านี้จำเป็นสำหรับการจัดการเพิ่มเติม

#### ตัวอย่างโค้ด

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไฟล์ Excel ของคุณถูกต้อง
// สร้างวัตถุเวิร์กบุ๊กด้วยเส้นทางไปยังไฟล์ Excel ของคุณ
Workbook workbook = new Workbook(dataDir + "/Mybook.xlsx");
```

- **พารามิเตอร์:**
  - `dataDir`:ไดเร็กทอรีซึ่งไฟล์ Excel ต้นฉบับของคุณตั้งอยู่
  - `"/Mybook.xlsx"`: ชื่อไฟล์ Excel ที่คุณต้องการโหลด

### การนำระบบวันที่ปี 1904 มาใช้

#### ภาพรวม

ระบบวันที่ 1904 มีความสำคัญต่อความเข้ากันได้กับแอปพลิเคชันบางประเภท ที่นี่ เราจะเปิดใช้งานระบบนี้ในเวิร์กบุ๊ก Excel โดยใช้ Aspose.Cells

#### ตัวอย่างโค้ด

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไฟล์ Excel ของคุณถูกต้อง
// โหลดเวิร์กบุ๊กจากไดเร็กทอรีที่คุณระบุ
Workbook workbook = new Workbook(dataDir + "/Mybook.xlsx");

// เปิดใช้งานระบบวันที่ปี 1904
workbook.getSettings().setDate1904(true);
```

- **การกำหนดค่าคีย์:**
  - `getSettings()`: ดึงการตั้งค่าสมุดงาน
  - `setDate1904(true)`: เปิดใช้งานระบบวันที่ปี 1904

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ Excel ของคุณถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าคุณได้ตั้งค่า Aspose.Cells เวอร์ชันที่ถูกต้องเพื่อหลีกเลี่ยงปัญหาความเข้ากันได้

### บันทึกสมุดงาน

#### ภาพรวม

หลังจากทำการเปลี่ยนแปลง เช่น เปิดใช้งานระบบวันที่ปี 1904 จำเป็นต้องบันทึกเวิร์กบุ๊ก ขั้นตอนนี้จะทำให้การแก้ไขทั้งหมดเสร็จสิ้น

#### ตัวอย่างโค้ด

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // ตรวจสอบให้แน่ใจว่าเส้นทางไปยังไฟล์ Excel ของคุณถูกต้อง
String outDir = "YOUR_OUTPUT_DIRECTORY"; // ระบุตำแหน่งที่คุณต้องการบันทึกสมุดงานที่แก้ไข

// โหลดและปรับเปลี่ยนสมุดงานของคุณตามที่แสดงในขั้นตอนก่อนหน้า
tWorkbook workbook = new Workbook(dataDir + "/Mybook.xlsx");
workbook.getSettings().setDate1904(true);

// บันทึกการเปลี่ยนแปลงไปยังไฟล์ใหม่
workbook.save(outDir + "/I1904DateSystem_out.xls");
```

- **พารามิเตอร์:**
  - `outDir`:ไดเร็กทอรีที่คุณต้องการบันทึกสมุดงานที่คุณแก้ไข
  - `"/I1904DateSystem_out.xls"`: ชื่อไฟล์ Excel ที่ได้ผลลัพธ์

## การประยุกต์ใช้งานจริง

1. **การเก็บข้อมูลถาวร**:ใช้ฟีเจอร์นี้เมื่อจัดการข้อมูลประวัติที่ต้องการความเข้ากันได้กับระบบรุ่นเก่าที่ใช้ระบบวันที่ปี 1904
2. **ความเข้ากันได้ข้ามแพลตฟอร์ม**:รับรองการเปลี่ยนแปลงที่ราบรื่นระหว่างแพลตฟอร์มที่ระบบวันที่เริ่มต้นอาจแตกต่างกัน
3. **การรายงานทางการเงิน**:มีประโยชน์ในภาคการเงินเพื่อรักษาความสม่ำเสมอระหว่างซอฟต์แวร์เวอร์ชันต่างๆ

**2. ฉันจะมั่นใจได้อย่างไรว่าแอปพลิเคชันอื่น ๆ ที่ใช้ Aspose.Cells เข้ากันได้**
ตรวจสอบให้แน่ใจว่าคุณได้ตรวจสอบข้อกำหนดเฉพาะแอปพลิเคชันที่เกี่ยวข้องกับระบบวันที่และกำหนดค่าการตั้งค่าเวิร์กบุ๊กของคุณตามนั้นโดยใช้เมธอด Aspose.Cells

**5. ฉันจะแก้ไขปัญหาได้อย่างไรหากสมุดงานไม่บันทึกอย่างถูกต้อง?**
