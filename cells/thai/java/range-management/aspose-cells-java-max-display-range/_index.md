## คู่มือการใช้งาน
### การเข้าถึงช่วงการแสดงผลสูงสุดของแผ่นงาน
#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณสามารถดึงข้อมูลช่วงการแสดงผลสูงสุด ซึ่งถือเป็นสิ่งสำคัญในการทำความเข้าใจว่าสามารถดูข้อมูลได้ครั้งละเท่าใดในเวิร์กชีต Excel
#### การดำเนินการแบบทีละขั้นตอน
**1. โหลดสมุดงานของคุณ**
เริ่มต้นด้วยการโหลดเวิร์กบุ๊กของคุณโดยใช้ Aspose.Cells:
```java
String dataDir = "path/to/your/data/";
Workbook workbook = new Workbook(dataDir + "Book1.xlsx");
```
**2. เข้าถึงแผ่นงาน**
ดึงข้อมูลเวิร์กชีตแรกจากเวิร์กบุ๊ก:
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```
**3. ดึงข้อมูลช่วงการแสดงผลสูงสุด**
ใช้ `getMaxDisplayRange()` เพื่อให้ได้ช่วง:
```java
Range range = worksheet.getCells().getMaxDisplayRange();
System.out.println("Maximum Display Range: " + range.getRefersTo());
```
#### คำอธิบาย
- **`Workbook`-** แสดงถึงไฟล์ Excel ของคุณ
- **`Worksheet`-** เข้าถึงแผ่นงานที่ระบุในเวิร์กบุ๊ก
- **`getMaxDisplayRange()`-** ดึงข้อมูลช่วงการแสดงผล ซึ่งเป็นสิ่งสำคัญสำหรับแอพพลิเคชันที่ต้องการนำเสนอข้อมูลอย่างมีประสิทธิภาพ

## การพิจารณาประสิทธิภาพ
- **การจัดการหน่วยความจำ:** เพิ่มประสิทธิภาพการตั้งค่าหน่วยความจำ Java เมื่อประมวลผลไฟล์ขนาดใหญ่
