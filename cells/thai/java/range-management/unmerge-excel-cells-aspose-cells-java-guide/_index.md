## คู่มือการใช้งาน
ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมของคุณเรียบร้อยแล้ว มาดูขั้นตอนการตรวจจับและการแยกเซลล์กัน

### การตรวจจับเซลล์ที่รวมกัน
การทำความเข้าใจว่าเซลล์ใดถูกผสานกันนั้นมีความสำคัญก่อนที่จะพยายามแยกเซลล์ออกจากกัน โดยทำได้ดังนี้:

#### โหลดสมุดงาน
เริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีเซลล์ที่ผสานกัน
```java
import com.aspose.cells.Workbook;

Workbook workbook = new Workbook("path/to/your/excel/file.xlsx");
```

#### รับใบงาน
เข้าถึงเวิร์กชีตเฉพาะที่ประกอบด้วยเซลล์ที่ผสานกัน
```java
import com.aspose.cells.Worksheet;

Worksheet worksheet = workbook.getWorksheets().get("SheetName");
```

### การแยกเซลล์ออกจากกัน
เมื่อคุณระบุพื้นที่ที่รวมกันแล้ว ให้ดำเนินการยกเลิกการรวมกัน

#### ดึงข้อมูลพื้นที่เซลล์ที่รวมกัน
ดึงพื้นที่เซลล์ที่รวมกันทั้งหมดลงในรายการเพื่อประมวลผล:
```java
import com.aspose.cells.CellArea;
import java.util.ArrayList;

ArrayList<CellArea> mergedCells = worksheet.getCells().getMergedCells();
```

#### ยกเลิกการรวมพื้นที่เซลล์แต่ละเซลล์
ทำซ้ำตามรายการและยกเลิกการผสานแต่ละพื้นที่ที่ระบุ
```java
for (int i = 0; i < mergedCells.size(); i++) {
    CellArea cellArea = mergedCells.get(i);
    worksheet.getCells().unMerge(cellArea.StartRow, cellArea.StartColumn,
                                 cellArea.EndRow, cellArea.EndColumn);
}
```

### บันทึกสมุดงาน
หลังจากยกเลิกการผสานเซลล์แล้วให้บันทึกการเปลี่ยนแปลงของคุณ
```java
workbook.save("path/to/your/output/file.xlsx");
```
