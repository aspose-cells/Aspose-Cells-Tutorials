## การแนะนำ

## คู่มือการใช้งาน

### คุณสมบัติ: การสร้างสมุดงาน

สร้าง `Workbook` วัตถุที่จะแสดงไฟล์ Excel ของคุณ:

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีจริงของคุณ
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

### คุณสมบัติ: การเข้าถึงแผ่นงาน

เข้าถึงเวิร์กชีตเฉพาะโดยใช้ดัชนีหรือชื่อ:

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Workbook;

Worksheet worksheet = workbook.getWorksheets().get(0); // เข้าถึงแผ่นงานแรก
```

### คุณสมบัติ: แถวปรับพอดีอัตโนมัติ

ปรับความสูงของแถวให้พอดีโดยอัตโนมัติตามเนื้อหาโดยใช้ `autoFitRow`-

```java
// ปรับแถวที่ 2 (ดัชนี 1)
worksheet.autoFitRow(1);
```

### คุณสมบัติ: คอลัมน์ปรับพอดีอัตโนมัติ

ปรับคอลัมน์ให้พอดีอัตโนมัติเพื่อให้แน่ใจว่าเนื้อหาทั้งหมดสามารถมองเห็นได้โดยไม่ถูกตัดออก:

```java
// ปรับความกว้างของคอลัมน์แรก (ดัชนี 0)
worksheet.autoFitColumn(0);
```

### คุณสมบัติ: บันทึกสมุดงาน

บันทึกสมุดงานหลังจากปรับแต่ง:

```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอาท์พุตจริง
workbook.save(outDir + "AutoFitRowsandColumns_out.xls");
```
