## คู่มือการใช้งาน
มาสำรวจคุณลักษณะแต่ละอย่างทีละขั้นตอนกัน
### โหลดสมุดงาน
#### ภาพรวม
การโหลดเวิร์กบุ๊กเป็นขั้นตอนเริ่มต้นในการจัดการไฟล์ Excel ด้วย Aspose.Cells
**H3: การโหลดเวิร์กบุ๊กที่มีอยู่**
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีข้อมูลของคุณ
String filePath = dataDir + "/chart.xls";
Workbook workbook = new Workbook(filePath);
```
- `dataDir`: เส้นทางไปยังไดเร็กทอรีข้อมูลของคุณ
- `filePath`: ชื่อไฟล์ของสมุดงาน Excel ของคุณ
**คำอธิบาย**: เดอะ `Workbook` คลาสอนุญาตให้โหลดไฟล์ Excel ที่มีอยู่ ซึ่งจำเป็นสำหรับการเริ่มต้นการปรับเปลี่ยนใดๆ

### ใบงานการเข้าถึง
#### ภาพรวม
การเข้าถึงเวิร์กชีตที่เจาะจงภายในเวิร์กบุ๊กช่วยให้สามารถจัดการอย่างตรงเป้าหมายได้
**H3: การดึงข้อมูลเวิร์กชีตแรก**
```java
import com.aspose.cells.Worksheet;

Worksheet worksheet = workbook.getWorksheets().get(0);
```
- `workbook.getWorksheets()`ดึงข้อมูลเวิร์กชีททั้งหมดในเวิร์กบุ๊ก
- `.get(0)`: เข้าถึงเวิร์กชีทแรกโดยดัชนี
**คำอธิบาย**:แผ่นงานจะถูกสร้างดัชนีตั้งแต่ศูนย์ ทำให้สามารถเข้าถึงแผ่นงานเฉพาะใดๆ ได้ตามดัชนี

### แผนภูมิการโหลดจากเวิร์กชีต
#### ภาพรวม
การดึงข้อมูลแผนภูมิถือเป็นสิ่งสำคัญสำหรับการจัดการข้อมูล
**H3: การโหลดวัตถุแผนภูมิ**
```java
import com.aspose.cells.Chart;

Chart chart = worksheet.getCharts().get(0);
```
- `worksheet.getCharts()`:ดึงวัตถุแผนภูมิทั้งหมดภายในเวิร์กชีตที่เลือก
- `.get(0)`: เลือกวัตถุแผนภูมิแรกตามดัชนี
**คำอธิบาย**การดำเนินการนี้มีความสำคัญต่อการเข้าถึงและการจัดการแผนภูมิเฉพาะในแผ่นงาน Excel ของคุณ

### การวางตำแหน่งวัตถุแผนภูมิใหม่
#### ภาพรวม
การเปลี่ยนตำแหน่งแผนภูมิเกี่ยวข้องกับการเปลี่ยนตำแหน่งบนเวิร์กชีต
**H3: การเปลี่ยนแปลงตำแหน่งแผนภูมิ**
```java
chart.getChartObject().setX(250);
chart.getChartObject().setY(150);
```
- `setX(int x)`: ตั้งค่าตำแหน่งแนวนอนของแผนภูมิ
- `setY(int y)`: ปรับตำแหน่งแนวตั้ง
**คำอธิบาย**:วิธีการเหล่านี้ช่วยให้ควบคุมตำแหน่งที่แผนภูมิปรากฏบนเวิร์กชีตได้อย่างแม่นยำ ทำให้มั่นใจได้ว่าแผนภูมิจะตรงตามความต้องการด้านเค้าโครงของคุณ

### บันทึกสมุดงาน
#### ภาพรวม
หลังจากทำการปรับเปลี่ยนแล้ว การบันทึกเวิร์กบุ๊กถือเป็นสิ่งสำคัญสำหรับการรักษาการเปลี่ยนแปลง
**H3: การบันทึกสมุดงานที่แก้ไข**
```java
import com.aspose.cells.Workbook;

String outDir = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอาท์พุตของคุณ
workbook.save(outDir + "/CCPosition_out.xls");
```
- `outDir`: เส้นทางไปยังไดเร็กทอรีเอาท์พุตของคุณ
- `.save(String filePath)`: บันทึกสมุดงานไปยังไฟล์ที่ระบุ
**คำอธิบาย**: เดอะ `save` วิธีการนี้รับรองว่าการเปลี่ยนแปลงทั้งหมดจะถูกเขียนกลับไปยังไฟล์ Excel ทำให้สามารถใช้หรือแจกจ่ายต่อไปได้
