## คู่มือการใช้งาน

### การโหลดและการเข้าถึงข้อมูล

เริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีข้อมูลของคุณ ขั้นตอนนี้มีความสำคัญสำหรับการเข้าถึงและจัดการแผนภูมิภายในเวิร์กบุ๊ก:
```java
// โหลดไฟล์ตัวอย่าง Excel
Workbook wb = new Workbook(srcDir + "sampleHandleAutomaticUnitsOfChartAxisLikeMicrosoftExcel.xlsx");

// เข้าถึงแผ่นงานแรก
Worksheet ws = wb.getWorksheets().get(0);

// เข้าถึงแผนภูมิแรก
Chart ch = ws.getCharts().get(0);
```

การทำให้หน่วยแกนของแผนภูมิเป็นแบบอัตโนมัติช่วยให้แน่ใจว่าแผนภูมิของคุณเลียนแบบพฤติกรรมของ Excel ทำให้การแสดงข้อมูลมีความสม่ำเสมอและแม่นยำ

**เรนเดอร์แผนภูมิเป็น PDF:**
การส่งออกแผนภูมิเป็นรูปแบบต่างๆ อาจมีประโยชน์อย่างยิ่งสำหรับการนำเสนอหรือรายงาน ต่อไปนี้คือวิธีแสดงแผนภูมิเป็น PDF:
