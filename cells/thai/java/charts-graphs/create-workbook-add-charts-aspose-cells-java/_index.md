## คู่มือการใช้งาน

### คุณสมบัติ: การสร้างสมุดงานและการป้อนข้อมูล

#### ภาพรวม
การสร้างเวิร์กบุ๊กเป็นขั้นตอนแรกในการจัดการสเปรดชีตด้วยโปรแกรม หัวข้อนี้จะแนะนำคุณเกี่ยวกับการสร้างเวิร์กบุ๊กใหม่และเติมข้อมูลตัวอย่างลงไป

#### การดำเนินการแบบทีละขั้นตอน
##### 1. สร้างอินสแตนซ์ของวัตถุสมุดงาน
```java
import com.aspose.cells.Workbook;

// สร้างวัตถุสมุดงานใหม่
double value = 50;
workbook.getWorksheets().get(0).getCells().get("A1").setValue(value);
```
การ `Workbook` คลาสแสดงถึงไฟล์ Excel ที่ให้คุณสามารถดำเนินการกับเนื้อหาในไฟล์ได้

##### 2. เข้าถึงแผ่นงานแรก
```java
import com.aspose.cells.WorksheetCollection;

WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet sheet = worksheets.get(0);
```
การเข้าถึงเวิร์กชีตแรกเป็นสิ่งสำคัญเนื่องจากทำหน้าที่เป็นจุดป้อนข้อมูลของเรา

##### 3. ป้อนข้อมูลลงในเซลล์
```java
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;

Cells cells = sheet.getCells();

// ตั้งค่าสำหรับเซลล์ที่แตกต่างกัน
cells.get("A1").setValue(50);
cells.get("A2").setValue(100);
cells.get("A3").setValue(150);
cells.get("B1").setValue(4);
cells.get("B2").setValue(20);
cells.get("B3").setValue(180);
cells.get("C1").setValue(320);
cells.get("C2").setValue(110);
cells.get("C3").setValue(180);
cells.get("D1").setValue(40);
cells.get("D2").setValue(120);
cells.get("D3").setValue(250);
```
ที่นี่ เรากำลังเติมค่าเฉพาะลงในเซลล์ ซึ่งจะแสดงเป็นแผนภูมิในภายหลัง

### คุณสมบัติ: การเพิ่มแผนภูมิลงในเวิร์กชีต

#### ภาพรวม
การแสดงข้อมูลเป็นภาพสามารถช่วยเพิ่มความเข้าใจและข้อมูลเชิงลึกได้อย่างมาก มาเพิ่มแผนภูมิคอลัมน์ 3 มิติลงในเวิร์กชีตของเรากัน

#### การดำเนินการแบบทีละขั้นตอน
##### 1. การรวบรวมแผนภูมิการเข้าถึง
```java
import com.aspose.cells.ChartCollection;

ChartCollection charts = sheet.getCharts();
```
การ `ChartCollection` คลาสนี้ช่วยให้คุณสามารถจัดการแผนภูมิต่างๆ ภายในเวิร์กชีตได้

##### 2. เพิ่มแผนภูมิคอลัมน์ 3 มิติ
```java
import com.aspose.cells.Chart;
import com.aspose.cells.ChartType;

int chartIndex = charts.add(ChartType.COLUMN_3_D, 5, 0, 15, 5);
Chart chart = charts.get(chartIndex);
```
การเพิ่มแผนภูมิเกี่ยวข้องกับการระบุประเภทและตำแหน่งบนเวิร์กชีต

##### 3. ตั้งค่าแหล่งข้อมูลแผนภูมิ
```java
import com.aspose.cells.SeriesCollection;

SeriesCollection serieses = chart.getNSeries();
serieses.add("A1:B3", true);
```
การกำหนดช่วงข้อมูลสำหรับแผนภูมิของคุณถือเป็นสิ่งสำคัญเพื่อแสดงชุดข้อมูลของคุณอย่างแม่นยำ

### คุณสมบัติ: การบันทึกสมุดงาน
สุดท้ายให้บันทึกสมุดงานที่มีการเปลี่ยนแปลงทั้งหมด:
```java
import com.aspose.cells.SaveFormat;

String outDir = "YOUR_OUTPUT_DIRECTORY"; // กำหนดเส้นทางไดเรกทอรีเอาท์พุต
workbook.save(outDir + "/HTCCustomChart_out.xls", SaveFormat.EXCEL_97_TO_2003);
```
การดำเนินการนี้จะบันทึกสมุดงานของคุณในรูปแบบที่เข้ากันได้กับ Excel ทำให้พร้อมสำหรับการแชร์หรือการวิเคราะห์เพิ่มเติม
