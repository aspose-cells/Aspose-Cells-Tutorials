## คู่มือการใช้งาน
ตอนนี้เรามาดูวิธีการเปลี่ยนทิศทางป้ายเครื่องหมายในแผนภูมิโดยใช้ Aspose.Cells สำหรับ Java กัน

### การโหลดเวิร์กบุ๊กและแผนภูมิ
1. **การเริ่มต้นสมุดงาน**:โหลดไฟล์ Excel ของคุณที่มีแผนภูมิ
2. **เข้าถึงแผ่นงาน**: รับแผ่นงานที่มีแผนภูมิที่ต้องการ
3. **ดึงข้อมูลวัตถุแผนภูมิ**:แยกแผนภูมิเฉพาะที่คุณต้องการปรับเปลี่ยน

```java
import com.aspose.cells.*;

public class ChangeTickLabelDirection {
    public static void main(String[] args) throws Exception {
        // ตั้งค่าไดเรกทอรีสำหรับไฟล์ต้นฉบับและไฟล์เอาท์พุต
        String sourceDir = Utils.Get_SourceDirectory();
        String outputDir = Utils.Get_OutputDirectory();

        // โหลดเวิร์กบุ๊กจากไฟล์
        Workbook workbook = new Workbook(sourceDir + "SampleChangeTickLabelDirection.xlsx");
        
        // รับแผ่นงานแรกในสมุดงาน
        Worksheet worksheet = workbook.getWorksheets().get(0);
        
        // ดึงข้อมูลแผนภูมิจากแผ่นงาน
        Chart chart = worksheet.getCharts().get(0);
    }
}
```

### การเปลี่ยนทิศทางของป้ายเครื่องหมายถูก
4. **ปรับเปลี่ยนทิศทางของป้ายเครื่องหมายถูก**: ตั้งค่าประเภททิศทางสำหรับป้ายกำกับแกนหมวดหมู่

```java
// เปลี่ยนทิศทางของป้ายเครื่องหมายเป็นแนวนอน
chart.getCategoryAxis().getTickLabels().setDirectionType(ChartTextDirectionType.HORIZONTAL);
```

5. **บันทึกการเปลี่ยนแปลง**: ส่งออกเวิร์กบุ๊กที่แก้ไขไปยังไฟล์ใหม่

```java
// บันทึกการเปลี่ยนแปลงในไฟล์ Excel ใหม่
workbook.save(outputDir + "outputChangeTickLabelDirection.xlsx");
```

**พารามิเตอร์และการกำหนดค่า**
- `ChartTextDirectionType.HORIZONTAL`: ระบุว่าควรแสดงป้ายเครื่องหมายถูกในแนวนอน
  
### เคล็ดลับการแก้ไขปัญหา
