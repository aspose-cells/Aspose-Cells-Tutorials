## คู่มือการใช้งาน
เราจะแบ่งกระบวนการออกเป็นสามฟีเจอร์หลัก: การโหลดไฟล์ Excel การเข้าถึงแผนภูมิเฉพาะ และการส่งออกไปยัง PDF ด้วยขนาดหน้าแบบกำหนดเอง

### โหลดไฟล์ตัวอย่าง Excel
#### ภาพรวม
การโหลดไฟล์ Excel เป็นขั้นตอนแรกในการทำงานกับเนื้อหา เราจะใช้ Aspose.Cells สำหรับ Java เพื่อโหลดเวิร์กบุ๊กที่มีข้อมูลแผนภูมิ

#### การดำเนินการแบบทีละขั้นตอน
**1. นำเข้าคลาสที่จำเป็น**
```java
import com.aspose.cells.Workbook;
```

**2. โหลดสมุดงาน**
แทนที่ `"YOUR_DATA_DIRECTORY"` ด้วยเส้นทางไฟล์จริงของคุณ:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // ตั้งค่านี้ไปยังไดเร็กทอรีของคุณ
Workbook wb = new Workbook(dataDir + "/sampleCreateChartPDFWithDesiredPageSize.xlsx");
```
โค้ดสั้นๆ นี้โหลดเวิร์กบุ๊ก Excel ทำให้พร้อมสำหรับการดำเนินการเพิ่มเติม

### เข้าถึงเวิร์กชีตและแผนภูมิแรก
#### ภาพรวม
การเข้าถึงเวิร์กชีตและแผนภูมิที่เจาะจงภายในเวิร์กบุ๊กทำให้คุณสามารถจัดการหรือส่งออกทีละรายการได้

#### การดำเนินการแบบทีละขั้นตอน
**1. ดึงข้อมูลแผ่นงาน**
```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;

Worksheet ws = wb.getWorksheets().get(0);
```
โค้ดด้านบนจะดึงเวิร์กชีตแรกจากเวิร์กบุ๊กของคุณ

**2. การเข้าถึงแผนภูมิ**
```java
Chart ch = ws.getCharts().get(0);
```
ตอนนี้, `ch` มีการอ้างอิงถึงแผนภูมิแรกในเวิร์กชีตที่ระบุ

### สร้างแผนภูมิ PDF ด้วยขนาดหน้าตามต้องการ
#### ภาพรวม
การแปลงแผนภูมิเป็น PDF ด้วยขนาดหน้าแบบกำหนดเองถือเป็นสิ่งสำคัญสำหรับการนำเสนอและรายงานที่ต้องการการจัดรูปแบบเฉพาะ

#### การดำเนินการแบบทีละขั้นตอน
**1. กำหนดไดเรกทอรีผลลัพธ์**
```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // ตั้งค่านี้ให้เป็นเส้นทางเอาท์พุตที่คุณต้องการ
```

**2. ส่งออกแผนภูมิเป็น PDF**
ที่นี่เราจะระบุขนาด (7x7 นิ้ว) และการจัดตำแหน่งหน้า
```java
import com.aspose.cells.PageLayoutAlignmentType;

ch.toPdf(outDir + "/outputCreateChartPDFWithDesiredPageSize.pdf", 7, 7,
         PageLayoutAlignmentType.CENTER, PageLayoutAlignmentType.CENTER);
```
ขั้นตอนนี้จะแปลงแผนภูมิเป็นไฟล์ PDF ตามขนาดหน้าและการจัดตำแหน่งที่คุณต้องการ

- **ข้อผิดพลาดเกี่ยวกับใบอนุญาต:** ตรวจสอบซ้ำอีกครั้งว่าตั้งค่าใบอนุญาตถูกต้องแล้วก่อนดำเนินการ
