## วิธีโหลด Excel workbook ใน Java?

คลาส `Workbook` แทนไฟล์ Excel ที่โหลดเข้าสู่หน่วยความจำ ให้การเข้าถึง worksheets, cells, และ charts

โหลด workbook ด้วย `new Workbook("path/to/file.xlsx")` – บรรทัดเดียวนี้จะอ่านสเปรดชีตทั้งหมดเข้าสู่หน่วยความจำ ทำให้คุณสามารถเข้าถึง worksheets, cells, และแผนภูมิที่ฝังอยู่ได้โดยโปรแกรม Aspose.Cells จะตรวจจับรูปแบบไฟล์อัตโนมัติ ไม่จำเป็นต้องระบุ XLSX, XLS หรือ CSV อย่างชัดเจน

## โหลด Workbook จากไฟล์
**ภาพรวม:**  
ขั้นตอนแรกคือการโหลด Excel workbook ซึ่งเป็นการตั้งค่าสภาพแวดล้อมสำหรับการเข้าถึงแผนภูมิ

```java
import com.aspose.cells.Workbook;
// Load an Excel workbook from a specified directory.
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

**คำอธิบาย:**  
- คลาส `Workbook` เป็นอ็อบเจกต์ระดับบนสุดที่แทนไฟล์ Excel หนึ่งไฟล์ในหน่วยความจำ  
- ระบุพาธเต็มของไฟล์ Excel ของคุณผ่านตัวแปร `dataDir` หรือพาธแบบ absolute

## วิธีเข้าถึง worksheet และ chart เฉพาะ?

อ็อบเจกต์ `Worksheet` แทนชีตเดียวใน workbook ที่มีแถว, คอลัมน์, และอ็อบเจกต์ฝังอยู่  
อ็อบเจกต์ `Chart` แทนการแสดงผลข้อมูลบน worksheet ซึ่งสามารถเรนเดอร์หรือส่งออกได้

ดึง worksheet ด้วย `workbook.getWorksheets().get(0)` แล้วเรียก `getCharts().get(0)` เพื่อรับแผนภูมิแรก – วิธีนี้ใช้ได้กับดัชนีแผนภูมิใด ๆ ที่คุณต้องการ API จะคืนค่าอ็อบเจกต์ `Chart` พร้อมสำหรับการเรนเดอร์หรือดึงข้อมูล

## เข้าถึง Worksheet และ Chart
**ภาพรวม:**  
หลังจากโหลดแล้ว ให้เข้าถึง worksheet และ chart ที่ต้องการแปลง

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;
// Access the first worksheet and its first chart.
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```

**คำอธิบาย:**  
- `worksheet` เป็นอ็อบเจกต์ประเภท `Worksheet`  
- `chart` ถูกดึงมาจากคอลเลกชัน chart ของ worksheet

## วิธีแปลง chart เป็นภาพ SVG?

คลาส `ImageOrPrintOptions` กำหนดการตั้งค่าการเรนเดอร์ เช่น รูปแบบเอาต์พุต, ความละเอียด, และคุณภาพสำหรับการแปลง chart หรือ worksheet เป็นไฟล์ภาพ

สร้างอินสแตนซ์ `ImageOrPrintOptions` ตั้งค่า `setSaveFormat(SaveFormat.SVG)` แล้วเรียก `chart.toImage(options, "output.svg")` การเรียกนี้จะเขียนไฟล์ SVG ที่สอดคล้องเต็มรูปแบบซึ่งรักษาสี, ฟอนต์, และป้ายข้อมูลเหมือนใน Excel อย่างแม่นยำ

## แปลง Chart เป็นภาพ SVG
**ภาพรวม:**  
ขั้นตอนสุดท้ายคือการแปลง chart ให้เป็นภาพ SVG เพื่อการแสดงผลคุณภาพสูง

```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.SaveFormat;
// Convert and save the chart as an SVG image.
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setSaveFormat(SaveFormat.SVG);
String outDir = "YOUR_OUTPUT_DIRECTORY";
chart.toImage(outDir + "CCToImageinSVGFormat_out.svg", options);
```

**คำอธิบาย:**  
- `ImageOrPrintOptions` กำหนดวิธีการบันทึก chart  
- การตั้งค่าเป็น SVG บอก Aspose.Cells ให้สร้างกราฟิกเวกเตอร์  
- ไฟล์ที่ได้สามารถฝังโดยตรงใน HTML หรือเป็นพื้นหลัง CSS ได้
