# วิธีเพิ่มแผนภูมิใน Excel ด้วย Aspose.Cells for Java: คู่มือฉบับสมบูรณ์

## บทนำ

ในโลกที่ขับเคลื่อนด้วยข้อมูลในปัจจุบัน, **how to add chart** ไปยังสมุดงาน Excel อย่างรวดเร็วอาจเป็นความแตกต่างระหว่างรายงานแบบคงที่กับเรื่องราวภาพที่น่าสนใจ แผนภูมิดิจิทัลช่วยให้คุณเปลี่ยนตัวเลขดิบให้เป็นข้อมูลเชิงลึกที่ชัดเจนโดยไม่ต้องคัดลอก‑วางด้วยตนเอง คู่มือฉบับนี้จะพาคุณผ่านการสร้างแผนภูมิอัตโนมัติด้วย Aspose.Cells for Java เพื่อให้คุณสร้างกราฟที่ดูเป็นมืออาชีพโดยตรงจากโค้ด

## คำตอบสั้น
- **อะไรคือไลบรารีหลัก?** Aspose.Cells for Java.  
- **ต้องใช้ Maven artifact ใด?** `com.aspose:aspose-cells:25.3`.  
- **ต้องการใบอนุญาตสำหรับการพัฒนาไหม?** การทดลองใช้ฟรีทำงานสำหรับการทดสอบ; ใบอนุญาตแบบชำระเงินจะลบข้อจำกัดการประเมิน.  
- **สามารถสร้างแผนภูมิแบบเส้น, พาย, และแท่งได้หรือไม่?** ได้—รองรับประเภทแผนภูมิมากกว่า 20 ประเภทโดยตรง.  
- **การจัดการไฟล์ขนาดใหญ่มีประสิทธิภาพหรือไม่?** Aspose.Cells ประมวลผลสมุดงาน 500 หน้าโดยใช้หน่วยความจำน้อยกว่า 200 MB.

## การตั้งค่า Aspose.Cells for Java

### การตั้งค่า Maven
ใส่ dependency ต่อไปนี้ในไฟล์ `pom.xml` ของคุณเพื่อรวม Aspose.Cells:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### การตั้งค่า Gradle
สำหรับผู้ใช้ Gradle, เพิ่มบรรทัดนี้ในไฟล์ `build.gradle` ของคุณ:

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

- **การทดลองใช้ฟรี:** เริ่มต้นด้วยการทดลองใช้ฟรีเพื่อสำรวจคุณลักษณะ.  
- **ใบอนุญาตชั่วคราว:** ขอรับเพื่อการทดสอบระยะยาว.  
- **การซื้อ:** สำหรับแอปพลิเคชันเชิงพาณิชย์, แนะนำให้ซื้อใบอนุญาต.

หลังจากตั้งค่าลibrary แล้ว, คุณสามารถเริ่มต้นสร้างอ็อบเจกต์ workbook ได้ บรรทัดโค้ดแรกที่คุณจะเขียนจะสร้างอินสแตนซ์ `Workbook` ที่เป็นตัวแทนไฟล์ Excel ในหน่วยความจำ

## คู่มือการดำเนินการ

ตอนนี้เราจะตอบคำถามหลัก: **how to add chart** ใน workbook ของ Java

### วิธีเพิ่มแผนภูมิโดยโปรแกรมเมติกใน Java?
โหลดหรือสร้าง `Workbook`, เพิ่มข้อมูลลงใน worksheet, จากนั้นสร้างอ็อบเจกต์ `Chart` ที่เชื่อมโยงกับช่วงข้อมูลนั้น สุดท้ายบันทึก workbook กระบวนการแบบต้น‑ถึง‑ปลายนี้ใช้เพียงไม่กี่บรรทัดของโค้ดและทำงานกับประเภทแผนภูมิที่รองรับทั้งหมด  
`Workbook` เป็นตัวแทนไฟล์ Excel ในหน่วยความจำ  
อ็อบเจกต์ `Chart` กำหนดการแสดงผลภาพของข้อมูลภายใน worksheet

### การเพิ่มข้อมูลลงใน Worksheet ของคุณ

แรกเริ่ม, เราจะเติมข้อมูลตัวอย่างลงใน worksheet:

```java
// Obtain a reference to the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);
Cells cells = worksheet.getCells();

// Populate data in A1:B4 and C1:C4 as values and categories
cells.get("A1").setValue(50); // Add value to A1
cells.get("B1").setValue(60); // Add value to B1, etc.
```

### การแทรกแผนภูมิ

ต่อไป, เพิ่มแผนภูมิลงใน worksheet:

```java
// Access the charts collection of the worksheet
ChartCollection charts = worksheet.getCharts();

// Add a new chart (e.g., Column type) to the worksheet
int chartIndex = charts.add(ChartType.COLUMN, 5, 0, 15, 5);
Chart chart = charts.get(chartIndex);

// Configure the chart's data source and category
SeriesCollection nSeries = chart.getNSeries();
nSeries.add("A1:B4", true); // Specify data range
nSeries.setCategoryData("C1:C4"); // Set category range

System.out.println("Chart added successfully.");
```

### การบันทึก Workbook ของคุณ

สุดท้าย, บันทึก workbook ของคุณเป็นไฟล์:

```java
workbook.save("SettingChartsData_out.xls");
System.out.println("Workbook with chart is created successfully.");
```
