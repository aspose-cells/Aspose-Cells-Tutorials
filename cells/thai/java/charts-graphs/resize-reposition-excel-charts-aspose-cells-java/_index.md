# ปรับขนาดและจัดตำแหน่งแผนภูมิ Excel ใหม่ด้วย Aspose.Cells สำหรับ Java

**สิ่งที่คุณจะได้เรียนรู้:**

- เทคนิคการปรับขนาดแผนภูมิภายในสมุดงานของคุณ
- วิธีการวางตำแหน่งแผนภูมิใหม่บนเวิร์กชีต
- แนวทางปฏิบัติที่ดีที่สุดสำหรับการเพิ่มประสิทธิภาพการทำงาน
มาสำรวจข้อกำหนดเบื้องต้นที่จำเป็นก่อนที่จะเริ่มต้น

### การตั้งค่า Aspose.Cells สำหรับ Java
ก่อนที่จะทำงานกับแผนภูมิ ให้ตั้งค่า Aspose.Cells ในสภาพแวดล้อมการพัฒนาของคุณ:
#### การตั้งค่า Maven
เพิ่มการอ้างอิงต่อไปนี้ให้กับของคุณ `pom.xml`-
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```
#### การตั้งค่า Gradle
รวมบรรทัดนี้ไว้ในของคุณ `build.gradle` ไฟล์:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

Aspose.Cells เสนอให้ทดลองใช้งานฟรีเพื่อทดสอบความสามารถ โดยมีตัวเลือกในการขอรับใบอนุญาตชั่วคราวหรือซื้อใบอนุญาต เริ่มต้นด้วยการดาวน์โหลด [ทดลองใช้งานฟรี](https://releases.aspose.com/cells/java/) แล้วค่อยพิจารณาซื้อหรือขอใบอนุญาตชั่วคราวผ่าน [หน้าการซื้อ](https://purchase-aspose.com/buy).

### คู่มือการใช้งาน
ในหัวข้อนี้ เราจะสำรวจวิธีการโหลด ปรับขนาด และเปลี่ยนตำแหน่งแผนภูมิโดยใช้ Aspose.Cells สำหรับ Java
#### โหลดและปรับขนาดแผนภูมิ
การปรับขนาดแผนภูมิจะปรับรูปลักษณ์ให้เหมาะกับความต้องการในการนำเสนอข้อมูลของคุณ ดังต่อไปนี้:
##### ขั้นตอนที่ 1: สร้างอินสแตนซ์เวิร์กบุ๊ก
โหลดไฟล์ Excel ที่มีอยู่โดยการสร้างอินสแตนซ์ของ `Workbook`-
```java
String filePath = "YOUR_DATA_DIRECTORY/book1.xls";
Workbook workbook = new Workbook(filePath);
```
##### ขั้นตอนที่ 2: เข้าถึงแผ่นงานแรก
เราจะทำงานกับเวิร์กชีตแรกซึ่งมักใช้กันในกรณีการใช้งานมากมาย
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```
##### ขั้นตอนที่ 3: โหลดแผนภูมิ
เข้าถึงแผนภูมิที่คุณต้องการปรับขนาด ในตัวอย่างนี้ เราจะใช้แผนภูมิแรกบนแผ่นงาน
```java
Chart chart = worksheet.getCharts().get(0);
```
##### ขั้นตอนที่ 4: ปรับขนาดแผนภูมิ
กำหนดขนาดใหม่สำหรับความกว้างและความสูงของแผนภูมิของคุณ
```java
chart.getChartObject().setWidth(400); // ตั้งค่าความกว้างของแผนภูมิเป็น 400 หน่วย
chart.getChartObject().setHeight(300); // ตั้งค่าความสูงของแผนภูมิเป็น 300 หน่วย

// บันทึกการเปลี่ยนแปลง
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "ResizeChart_out.xls");
```
#### การวางตำแหน่งแผนภูมิใหม่
การเปลี่ยนตำแหน่งของแผนภูมิจะช่วยเพิ่มประสิทธิภาพเค้าโครงและการอ่านได้ ดังนี้:
##### ขั้นตอนที่ 1: โหลดไฟล์ Excel
โหลดสมุดงานของคุณ
```java
String filePath = "YOUR_DATA_DIRECTORY/book1.xls";
Workbook workbook = new Workbook(filePath);
```
##### ขั้นตอนที่ 2: เข้าถึงเวิร์กชีตและแผนภูมิ
เข้าถึงแผ่นงานและแผนภูมิที่จำเป็นคล้ายกับการปรับขนาด
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```
##### ขั้นตอนที่ 3: วางตำแหน่งแผนภูมิใหม่
ปรับพิกัด X และ Y เพื่อย้ายแผนภูมิของคุณภายในเวิร์กชีต
```java
chart.getChartObject().setX(250); // ตั้งค่าตำแหน่งแนวนอนเป็น 250 หน่วย
chart.getChartObject().setY(150); // ตั้งค่าตำแหน่งแนวตั้งเป็น 150 หน่วย

// บันทึกการเปลี่ยนแปลงในไฟล์ใหม่
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "RepositionChart_out.xls");
```
