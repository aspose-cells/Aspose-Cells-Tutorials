## คู่มือการใช้งาน
ทำตามขั้นตอนเหล่านี้เพื่อใช้การเปลี่ยนแปลงทิศทางข้อความในความคิดเห็น

### การเพิ่มและแก้ไขความคิดเห็น
#### ภาพรวม
ปรับปรุงความชัดเจนของเอกสารโดยการเพิ่มความคิดเห็นในเซลล์และปรับแต่งทิศทางของข้อความ ซึ่งมีประโยชน์อย่างยิ่งสำหรับเอกสารหลายภาษาหรือระบบที่ต้องการการจัดรูปแบบเฉพาะ

**ขั้นตอนที่ 1: สร้างตัวอย่างสมุดงาน**
สร้างอินสแตนซ์ของ `Workbook` ระดับ:
```java
// สร้างสมุดงานใหม่
Workbook wb = new Workbook();
```

**ขั้นตอนที่ 2: เข้าถึงแผ่นงาน**
เข้าถึงแผ่นงานที่คุณต้องการเพิ่มความคิดเห็น:
```java
// รับแผ่นงานแรกในสมุดงาน
Worksheet sheet = wb.getWorksheets().get(0);
```

**ขั้นตอนที่ 3: เพิ่มความคิดเห็น**
เพิ่มความคิดเห็นลงในเซลล์ "A1":
```java
// เพิ่มความคิดเห็นลงในเซลล์ A1
Comment comment = sheet.getComments().get(sheet.getComments().add("A1"));
```
ดึงข้อมูลวัตถุความคิดเห็นโดยการเพิ่มลงในเซลล์แรก

**ขั้นตอนที่ 4: กำหนดค่าการจัดตำแหน่งและทิศทางข้อความ**
ปรับแต่งการจัดตำแหน่งข้อความและกำหนดทิศทางจากขวาไปซ้าย:
```java
// ตั้งค่าการจัดตำแหน่งข้อความแนวตั้งและแนวนอน
comment.getCommentShape().setTextVerticalAlignment(TextAlignmentType.CENTER);
comment.getCommentShape().setTextHorizontalAlignment(TextAlignmentType.RIGHT);

// เปลี่ยนทิศทางข้อความจากขวาไปซ้าย
comment.getCommentShape().setTextDirection(TextDirectionType.RIGHT_TO_LEFT);
```
การ `TextAlignmentType` และ `TextDirectionType` enums ช่วยกำหนดค่าลักษณะที่ปรากฏของความคิดเห็นของคุณ

**ขั้นตอนที่ 5: ตั้งค่าหมายเหตุความคิดเห็น**
กำหนดเนื้อหาความคิดเห็น:
```java
// เพิ่มหมายเหตุลงในความคิดเห็น
comment.setNote("This is my Comment Text. This is test");
```

### การบันทึกงานของคุณ
หลังจากทำการเปลี่ยนแปลงแล้ว ให้บันทึกสมุดงานพร้อมความคิดเห็นที่อัปเดต:
```java
// กำหนดไดเรกทอรีเอาท์พุตและชื่อไฟล์
String dataDir = Utils.getSharedDataDir(ChangeTextDirectionofComment.class) + "TechnicalArticles/";

// บันทึกไฟล์ Excel พร้อมความคิดเห็นที่ปรับเปลี่ยน
wb.save(dataDir + "CTDOfComment_out.xlsx");
```
