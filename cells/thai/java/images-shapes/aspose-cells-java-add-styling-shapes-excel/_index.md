# วิธีการเพิ่มและกำหนดรูปแบบรูปร่างใน Excel โดยใช้ Aspose.Cells Java

## การแนะนำ

เพิ่มประสิทธิภาพเวิร์กชีต Excel ของคุณด้วยการเพิ่มรูปร่างที่กำหนดเองด้วยโปรแกรมด้วย `Aspose.Cells` สำหรับ Java บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการเพิ่มรูปทรงสี่เหลี่ยมผืนผ้า การกำหนดค่าสไตล์เส้น และการใช้การเติมแบบไล่ระดับ

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า Aspose.Cells ในโปรเจ็กต์ Java ของคุณ
- การเพิ่มรูปร่างสี่เหลี่ยมผืนผ้าลงในเวิร์กชีต Excel
- การกำหนดค่าสไตล์เส้นและการไล่ระดับสีสำหรับรูปร่าง
- บันทึกสมุดงานที่แก้ไข

เริ่มต้นด้วยการตรวจสอบให้แน่ใจว่าคุณปฏิบัติตามข้อกำหนดเบื้องต้นทั้งหมด

## ข้อกำหนดเบื้องต้น

ก่อนที่จะเจาะลึกโค้ด โปรดตรวจสอบให้แน่ใจว่า:
- **ห้องสมุด:** ไลบรารี Aspose.Cells (เวอร์ชัน 25.3 หรือใหม่กว่า) รวมอยู่ในโปรเจ็กต์ของคุณ
- **สิ่งแวดล้อม:** มีความคุ้นเคยกับสภาพแวดล้อมการพัฒนา Java เช่น Maven หรือ Gradle สำหรับการจัดการการอ้างอิง
- **ความรู้:** ความเข้าใจพื้นฐานเกี่ยวกับการเขียนโปรแกรม Java และการจัดการไฟล์ Excel

## การตั้งค่า Aspose.Cells สำหรับ Java

รวม Aspose.Cells เข้ากับโครงการ Java ของคุณโดยใช้เครื่องมือสร้างของคุณ:

**เมเวน:**
เพิ่มไปยังของคุณ `pom.xml`-
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**เกรเดิ้ล:**
รวมอยู่ในของคุณ `build.gradle` ไฟล์:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## คู่มือการใช้งาน

### การเพิ่มรูปร่างสี่เหลี่ยมผืนผ้าลงในเวิร์กชีต Excel

**ภาพรวม:** เรียนรู้วิธีการเพิ่มและจัดตำแหน่งรูปสี่เหลี่ยมผืนผ้าในเวิร์กชีตของคุณโดยใช้ Aspose.Cells

#### ขั้นตอนที่ 1: สร้างสมุดงานใหม่
```java
Workbook excelBook = new Workbook();
```
การดำเนินการนี้จะเริ่มต้นอินสแตนซ์เวิร์กบุ๊กใหม่ที่คุณจะเพิ่มรูปร่าง

#### ขั้นตอนที่ 2: เพิ่มรูปทรงสี่เหลี่ยมผืนผ้า
```java
import com.aspose.cells.RectangleShape;
import com.aspose.cells.MsoDrawingType;

RectangleShape rectangle = (RectangleShape) excelBook.getWorksheets().get(0)
        .getShapes().addShape(MsoDrawingType.RECTANGLE, 3, 2, 0, 0, 70, 130);
```
ที่นี่ สี่เหลี่ยมผืนผ้าจะถูกเพิ่มลงในเวิร์กชีตแรก พารามิเตอร์จะระบุประเภท ตำแหน่ง และขนาดของสี่เหลี่ยมผืนผ้า

#### ขั้นตอนที่ 3: กำหนดตำแหน่ง
```java
rectangle.setPlacement(com.aspose.cells.PlacementType.FREE_FLOATING);
```
การดำเนินการนี้จะกำหนดรูปร่างให้เป็นแบบลอยอิสระแทนที่จะยึดกับช่วงเซลล์เฉพาะ

### การกำหนดค่ารูปแบบเส้นของรูปทรง

**ภาพรวม:** ปรับแต่งรูปแบบเส้นและการเติมแบบไล่ระดับสีสำหรับรูปร่างสี่เหลี่ยมผืนผ้าของคุณ

#### ขั้นตอนที่ 1: กำหนดค่ารูปแบบเส้น
```java
import com.aspose.cells.LineFormat;
import com.aspose.cells.MsoLineStyle;

LineFormat linestyle = rectangle.getLine();
linestyle.setDashStyle(MsoLineStyle.THICK_THIN);
linestyle.setWeight(4);
```
การกระทำนี้จะกำหนดรูปแบบเส้นให้เป็นรูปแบบเส้นหนา-บาง และปรับน้ำหนักของเส้น

#### ขั้นตอนที่ 2: ใช้การเติมแบบไล่เฉดสี
```java
import com.aspose.cells.FillFormat;
import com.aspose.cells.GradientStyleType;

FillFormat fillformat = rectangle.getFill();
fillformat.setOneColorGradient(com.aspose.cells.Color.getBlue(), 1, 
    GradientStyleType.HORIZONTAL, 1);
```
มีการใช้เอฟเฟกต์ไล่เฉดสีกับการเติมสีสี่เหลี่ยมเพื่อเพิ่มความสวยงามให้กับภาพ

### การบันทึกสมุดงาน

สุดท้ายให้บันทึกสมุดงานของคุณพร้อมการกำหนดค่าทั้งหมด:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
excelBook.save(outDir + "/StyledRectangle_out.xls");
```

- **การออกแบบเทมเพลต:** สร้างเทมเพลตสำหรับรายงานหรือใบแจ้งหนี้ที่ต้องการองค์ประกอบกราฟิกโดยเฉพาะ
- **การสร้างรายงานอัตโนมัติ:** ปรับปรุงกระบวนการอัตโนมัติด้วยการเพิ่มและกำหนดรูปแบบรูปร่างด้วยโปรแกรม
