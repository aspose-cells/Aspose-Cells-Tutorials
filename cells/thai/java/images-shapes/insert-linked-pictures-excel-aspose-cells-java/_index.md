## คู่มือการใช้งาน

มาดูรายละเอียดขั้นตอนการแทรกภาพเชื่อมโยงลงในไฟล์ Excel ของคุณกัน

### การแทรกภาพเชื่อมโยงจากที่อยู่เว็บ

#### ขั้นตอนที่ 1: การตั้งค่าเวิร์กบุ๊ก
สร้างอินสแตนซ์เวิร์กบุ๊กใหม่ที่คุณจะแทรกภาพที่เชื่อมโยงของคุณ

```java
import com.aspose.cells.Workbook;

Workbook workbook = new Workbook();
```

#### ขั้นตอนที่ 2: การเพิ่มรูปภาพที่เชื่อมโยง
ใช้ `addLinkedPicture` วิธีการเพิ่มรูปภาพจากที่อยู่เว็บในเซลล์ B2 พารามิเตอร์จะระบุแถว คอลัมน์ และขนาดของรูปภาพ

```java
import com.aspose.cells.Picture;
import com.aspose.cells.Worksheet;

Worksheet worksheet = workbook.getWorksheets().get(0);
int pictureIndex = worksheet.getShapes().addLinkedPicture(1, 1, 100, 100,
        "http://www.aspose.com/Images/aspose-logo.jpg");
Picture pic = worksheet.getShapes().get(pictureIndex) instanceof Picture ? (Picture) worksheet.getShapes().get(pictureIndex) : null;
```

#### ขั้นตอนที่ 3: การกำหนดค่าแหล่งที่มาของภาพ
ตั้งค่า URL ของแหล่งที่มาของรูปภาพเพื่อให้แน่ใจว่ามีการเชื่อมโยงแบบไดนามิก

```java
pic.setSourceFullName("http://www.aspose.com/images/aspose-logo.gif");
```

#### ขั้นตอนที่ 4: การปรับขนาดภาพ
ปรับแต่งความสูงและความกว้างเพื่อให้แสดงในไฟล์ Excel ของคุณได้ดีขึ้น

```java
pic.setHeightInch(1.04);
pic.setWidthInch(2.6);
```

#### ขั้นตอนที่ 5: การบันทึกสมุดงานของคุณ
บันทึกสมุดงานของคุณเพื่อยืนยันการเปลี่ยนแปลง โดยให้แน่ใจว่ามีการรวมรูปภาพที่เชื่อมโยงไว้

```java
workbook.save("ILPfromWebAddress_out.xlsx");
```

### เคล็ดลับการแก้ไขปัญหา
- **รูปภาพไม่แสดง**:ให้แน่ใจว่า URL ถูกต้องและสามารถเข้าถึงได้
- **ปัญหาด้านความจำ**:ปรับขนาดรูปภาพเพื่อประสิทธิภาพที่ดีขึ้นกับไฟล์ Excel ขนาดใหญ่
