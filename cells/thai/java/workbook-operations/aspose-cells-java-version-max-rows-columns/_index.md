## คู่มือการใช้งาน
### การตรวจสอบ Aspose.Cells สำหรับเวอร์ชัน Java
#### ภาพรวม
การตรวจสอบเวอร์ชันของ Aspose.Cells เป็นสิ่งสำคัญสำหรับการดีบักและการรับรองความเข้ากันได้กับส่วนประกอบอื่น ๆ คุณสามารถนำไปใช้งานได้ดังนี้:

##### ขั้นตอนที่ 1: นำเข้าคลาสที่จำเป็น

```java
import com.aspose.cells.*;
```

##### ขั้นตอนที่ 2: ดึงข้อมูลและพิมพ์เวอร์ชัน
สร้างชั้นเรียน `AsposeCellsVersionCheck` เพื่อรวมฟังก์ชันการทำงานนี้ไว้

```java
public class AsposeCellsVersionCheck {
    public static void main(String[] args) throws Exception {
        String version = CellsHelper.getVersion();
        System.out.println("Aspose.Cells for Java Version: " + version);
    }
}
```

**คำอธิบาย**: เดอะ `getVersion()` วิธีการจาก `CellsHelper` คลาสดึงข้อมูลสตริงเวอร์ชันของ Aspose.Cells จากนั้นจะพิมพ์ไปที่คอนโซล

### จำนวนแถวและคอลัมน์สูงสุดในรูปแบบ XLS
#### ภาพรวม
การทำความเข้าใจข้อจำกัดของรูปแบบช่วยในการออกแบบแอปพลิเคชันที่สามารถจัดการชุดข้อมูลขนาดใหญ่ได้ ต่อไปนี้เป็นวิธีค้นหาจำนวนแถวและคอลัมน์สูงสุดสำหรับไฟล์ XLS:

##### ขั้นตอนที่ 1: นำเข้าคลาสที่จำเป็น

```java
import com.aspose.cells.*;
```

##### ขั้นตอนที่ 2: สร้างเวิร์กบุ๊กและดึงการตั้งค่า
นำฟังก์ชันนี้ไปใช้ใน `MaxRowsColsXLSFormat`-

```java
public class MaxRowsColsXLSFormat {
    public static void main(String[] args) throws Exception {
        Workbook wb = new Workbook(FileFormatType.EXCEL_97_TO_2003);
        int maxRows = wb.getSettings().getMaxRow() + 1;
        int maxCols = wb.getSettings().getMaxColumn() + 1;
        
        System.out.println("Maximum Rows: " + maxRows);
        System.out.println("Maximum Columns: " + maxCols);
    }
}
```

**คำอธิบาย**: การสร้าง `Workbook` กับ `FileFormatType.EXCEL_97_TO_2003` ช่วยให้เราเข้าถึงการตั้งค่าที่เฉพาะกับรูปแบบ XLS รวมถึงจำนวนแถวและคอลัมน์สูงสุด

### จำนวนแถวและคอลัมน์สูงสุดในรูปแบบ XLSX
#### ภาพรวม
คล้ายกับ XLS การทราบข้อจำกัดเหล่านี้สำหรับ XLSX ช่วยให้มั่นใจได้ว่าแอปพลิเคชันของคุณสามารถจัดการสเปรดชีตขนาดใหญ่ได้โดยไม่เกิดข้อผิดพลาด

##### ขั้นตอนที่ 1: นำเข้าคลาสที่จำเป็น

```java
import com.aspose.cells.*;
```

##### ขั้นตอนที่ 2: สร้างเวิร์กบุ๊กและดึงการตั้งค่า
ดำเนินการนี้ใน `MaxRowsColsXLSXFormat`-

```java
public class MaxRowsColsXLSXFormat {
    public static void main(String[] args) throws Exception {
        Workbook wb = new Workbook(FileFormatType.XLSX);
        int maxRows = wb.getSettings().getMaxRow() + 1;
        int maxCols = wb.getSettings().getMaxColumn() + 1;

        System.out.println("Maximum Rows: " + maxRows);
        System.out.println("Maximum Columns: " + maxCols);
    }
}
```

**คำอธิบาย**: โดยการเริ่มต้น `Workbook` กับ `FileFormatType.XLSX`คุณสามารถเข้าถึงการตั้งค่าเฉพาะ XLSX เพื่อกำหนดจำนวนแถวและคอลัมน์สูงสุดได้
