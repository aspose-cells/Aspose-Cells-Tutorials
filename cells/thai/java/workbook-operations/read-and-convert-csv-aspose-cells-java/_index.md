# การอ่านและการแปลงไฟล์ CSV อย่างมีประสิทธิภาพด้วย Aspose.Cells Java

## การแนะนำ
การจัดการไฟล์ CSV อาจเป็นเรื่องท้าทาย โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับการเข้ารหัสที่แตกต่างกัน หากคุณประสบปัญหาในการอ่าน CSV ที่ซับซ้อนหรือแปลงเป็นรูปแบบ Excel คู่มือนี้เหมาะสำหรับคุณ เราจะมาสำรวจความสามารถของ **Aspose.Cells สำหรับ Java** เพื่ออ่านและแปลงไฟล์ CSV ได้อย่างมีประสิทธิภาพ

ในบทช่วยสอนนี้ เราจะสาธิตวิธีการดังต่อไปนี้:
- อ่านไฟล์ CSV ที่เข้ารหัสหลายตัวได้อย่างง่ายดาย
- แปลงข้อมูล CSV เป็นรูปแบบ Excel ได้อย่างราบรื่น
- กำหนดค่าตัวเลือกการโหลดเพื่อประสิทธิภาพที่เหมาะสมที่สุด

## คู่มือการใช้งาน
### การอ่านและการแปลงไฟล์ CSV
#### ภาพรวม
เน้นการอ่านไฟล์ CSV ที่เข้ารหัสหลายตัวโดยใช้ Aspose.Cells Java จากนั้นแปลงเป็นรูปแบบ Excel

##### ขั้นตอนที่ 1: ตั้งค่าตัวเลือกโหลด
กำหนดค่าตัวเลือกการโหลดเพื่อจัดการการเข้ารหัสหลายรายการ:
```java
import com.aspose.cells.TxtLoadOptions;

// สร้างอ็อบเจ็กต์ TxtLoadOptions และตั้งค่าคุณสมบัติ MultiEncoded
TxtLoadOptions options = new TxtLoadOptions();
options.setMultiEncoded(true);
```
**คำอธิบาย:** โดยการตั้งค่า `setMultiEncoded(true)`Aspose.Cells สามารถจัดการไฟล์ CSV ที่มีรูปแบบการเข้ารหัสต่างๆ ได้อย่างราบรื่น

##### ขั้นตอนที่ 2: โหลดไฟล์ CSV
โหลดไฟล์ CSV ที่เข้ารหัสหลายรายการของคุณลงในอ็อบเจ็กต์เวิร์กบุ๊ก:
```java
import com.aspose.cells.Workbook;

// ระบุเส้นทางไปยังไฟล์ CSV
String dataDir = "path/to/your/csv/";

// โหลดไฟล์ CSV ด้วยตัวเลือกที่ระบุ
Workbook workbook = new Workbook(dataDir + "MultiEncoded.csv", options);
```
**คำอธิบาย:** การ `Workbook` คลาสแสดงถึงไฟล์ Excel ที่ใช้ที่นี่เพื่อโหลดไฟล์ CSV โดยใช้ตัวเลือกการโหลดที่กำหนดค่าของเรา

##### ขั้นตอนที่ 3: บันทึกเป็น XLSX
บันทึกข้อมูลที่โหลดลงในรูปแบบ XLSX:
```java
import com.aspose.cells.SaveFormat;

// บันทึกสมุดงานในรูปแบบ XLSX
workbook.save(dataDir + "ReadingCSVFile_out.xlsx", SaveFormat.XLSX);
```
**คำอธิบาย:** การ `SaveFormat.XLSX` พารามิเตอร์ระบุว่าเราต้องการบันทึกไฟล์ของเราเป็นเอกสาร Excel

- **ข้อผิดพลาดเส้นทาง:** ตรวจสอบเส้นทางไปยังไฟล์ CSV ของคุณอีกครั้งเพื่อให้แน่ใจว่าถูกต้องเมื่อเทียบกับไดเร็กทอรีการทำงานของโปรแกรม Java
