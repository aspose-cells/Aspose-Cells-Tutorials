## คู่มือการใช้งาน

เราจะแบ่งคุณลักษณะแต่ละอย่างออกเป็นขั้นตอนที่จัดการได้เพื่อความชัดเจน 

### คุณสมบัติ 1: โหลดไฟล์ Excel

**ภาพรวม**

ฟีเจอร์นี้ช่วยให้คุณโหลดเวิร์กบุ๊ก Excel ที่มีอยู่โดยใช้ Aspose.Cells เพื่อเตรียมพร้อมสำหรับการจัดการหรือวิเคราะห์เพิ่มเติม

#### ขั้นตอนที่ 1: สร้างวัตถุเวิร์กบุ๊กใหม่

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook wb = new Workbook(dataDir + "/sampleInlineCharts.xlsx");
```

**คำอธิบาย:**  
- **`Workbook` ระดับ**: หมายถึงไฟล์ Excel โหลดเวิร์กบุ๊กจากเส้นทางที่คุณระบุลงในหน่วยความจำ
- **พารามิเตอร์**:ตัวสร้างรับอาร์กิวเมนต์สตริงเพื่อระบุตำแหน่งของไฟล์

### คุณลักษณะที่ 2: ตั้งค่าพื้นที่การพิมพ์ในเวิร์กชีต

**ภาพรวม**

ที่นี่ เราจะกำหนดช่วงเซลล์เฉพาะภายในเวิร์กชีตที่ควรพิจารณาสำหรับการพิมพ์หรือการส่งออกงาน

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

```java
Workbook wb = new Workbook(dataDir + "/sampleInlineCharts.xlsx");
```

#### ขั้นตอนที่ 2: เข้าถึงแผ่นงานแรก

```java
Worksheet ws = wb.getWorksheets().get(0);
```

**คำอธิบาย:**  
- **`getWorksheets()`**: ส่งคืนคอลเลกชันของเวิร์กชีททั้งหมดในเวิร์กบุ๊ก
- **วิธีการเข้าถึง**: ใช้ `get(index)` เพื่อเข้าถึงแผ่นงานเฉพาะตามดัชนี

#### ขั้นตอนที่ 3: ตั้งค่าพื้นที่การพิมพ์

```java
ws.getPageSetup().setPrintArea("D2:M20");
```

**คำอธิบาย:**  
- **`setPrintArea(String range)`**: ระบุช่วงเซลล์สำหรับการดำเนินการพิมพ์/ส่งออก
- **พารามิเตอร์ช่วง**: สตริงที่ระบุเซลล์ที่จะรวม เช่น `"D2:M20"`-

### คุณสมบัติที่ 3: ส่งออกพื้นที่การพิมพ์ไปยัง HTML

**ภาพรวม**

คุณลักษณะนี้ส่งออกเฉพาะพื้นที่พิมพ์ที่กำหนดของเวิร์กชีตไปยังไฟล์ HTML

#### ขั้นตอนที่ 1: โหลดและตั้งค่าพื้นที่พิมพ์ (ตามที่อธิบายไว้ก่อนหน้านี้)

```java
Workbook wb = new Workbook(dataDir + "/sampleInlineCharts.xlsx");
Worksheet ws = wb.getWorksheets().get(0);
ws.getPageSetup().setPrintArea("D2:M20");
```

#### ขั้นตอนที่ 2: เริ่มต้น HtmlSaveOptions

```java
HtmlSaveOptions options = new HtmlSaveOptions();
options.setExportPrintAreaOnly(true);
```

**คำอธิบาย:**  
- **`HtmlSaveOptions`**: กำหนดค่าวิธีการบันทึกไฟล์ Excel ในรูปแบบ HTML
- **`setExportPrintAreaOnly(boolean value)`**: หากตั้งค่าเป็น `true`ส่งออกเฉพาะพื้นที่พิมพ์ที่ระบุเท่านั้น

#### ขั้นตอนที่ 3: บันทึกเป็น HTML

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
wb.save(outDir + "/outputInlineCharts.html", options);
```

**คำอธิบาย:**  
- **`save(String filePath, HtmlSaveOptions options)`**: บันทึกสมุดงานตามตัวเลือกที่ระบุ
- **ไดเรกทอรีผลลัพธ์**: ตรวจสอบให้แน่ใจว่าไดเร็กทอรีเอาต์พุตของคุณถูกตั้งค่าอย่างถูกต้อง

## การประยุกต์ใช้งานจริง

1. **ความสามารถในการพกพาข้อมูล:**
   - ส่งออกส่วนข้อมูลเฉพาะสำหรับแอปพลิเคชันหรือรายงานบนเว็บ

2. **การแบ่งปันข้อมูลแบบเลือก:**
   - แบ่งปันเฉพาะส่วนที่เกี่ยวข้องของรายงานกับผู้มีส่วนได้ส่วนเสียผ่านรูปแบบ HTML

3. **การบูรณาการเว็บ:**
   - ฝังข้อมูล Excel ลงในเว็บไซต์เพื่อให้แน่ใจว่าแสดงเฉพาะข้อมูลที่จำเป็นเท่านั้น

4. **การรายงานอัตโนมัติ:**
   - สร้างรายงานแบบไดนามิกที่เน้นที่ตัวชี้วัดหลัก ช่วยประหยัดเวลาในการแยกข้อมูลด้วยตนเอง

5. **การแสดงภาพข้อมูล:**
   - รวม HTML ที่ส่งออกกับไลบรารี JavaScript สำหรับการแสดงข้อมูลแบบโต้ตอบ

## การพิจารณาประสิทธิภาพ

- **เพิ่มประสิทธิภาพการใช้หน่วยความจำ:** ใช้ `Workbook.dispose()` เพื่อปลดปล่อยทรัพยากรเมื่อการดำเนินการเสร็จสิ้น
- **การประมวลผลแบบแบตช์:** จัดการชุดข้อมูลขนาดใหญ่โดยประมวลผลแบบเป็นชุด
- **การโหลดที่มีประสิทธิภาพ:** โหลดเฉพาะเวิร์กชีตที่จำเป็นหรือบางส่วนของเวิร์กบุ๊กหากเป็นไปได้ ซึ่งจะช่วยลดการใช้หน่วยความจำ
