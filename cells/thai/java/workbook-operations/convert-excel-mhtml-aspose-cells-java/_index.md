### คู่มือการใช้งาน
#### การโหลดไฟล์ Excel
หากต้องการโหลดไฟล์ Excel ให้ทำตามขั้นตอนเหล่านี้:
1. **ตั้งค่าไดเรกทอรีข้อมูลของคุณ**: กำหนดเส้นทางที่จัดเก็บไฟล์ Excel ของคุณ
   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีข้อมูลจริงของคุณ
   ```
2. **สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก**:วัตถุนี้แสดงถึงเวิร์กบุ๊ก Excel ของคุณ
   ```java
   String filePath = dataDir + "Book1.xlsx"; // เส้นทางไปยังไฟล์ Excel
   Workbook wb = new Workbook(filePath); // โหลดไฟล์ Excel
   ```
3. **เหตุใดจึงต้องใช้ `Workbook`-** การ `Workbook` คลาสมีความจำเป็น เนื่องจากคลาสจะรวมชีตทั้งหมดและข้อมูลไว้ ทำให้จัดการได้ง่าย

#### การแปลงไฟล์ Excel เป็นรูปแบบ MHTML
ตอนนี้เราได้โหลดไฟล์ Excel แล้ว มาแปลงเป็น MHTML กัน:
1. **ตั้งค่าไดเรกทอรีเอาท์พุต**: กำหนดว่าคุณต้องการบันทึกไฟล์ที่แปลงไว้ที่ไหน
   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอาท์พุตจริงของคุณ
   ```
2. **ระบุตัวเลือกการบันทึก HTML**: ใช้ `HtmlSaveOptions` สำหรับการตั้งค่ารูปแบบการแปลง
   ```java
   HtmlSaveOptions sv = new HtmlSaveOptions(SaveFormat.M_HTML); // MHTML เป็นรูปแบบไฟล์เก็บถาวรเว็บ
   ```
3. **ดำเนินการแปลง**:บันทึกสมุดงานของคุณในรูปแบบที่ต้องการ
   ```java
   wb.save(outDir + "/CToMHTMLFiles_out.mht", sv);
   ```
4. **ทำไม `SaveFormat.M_HTML`-** ตัวเลือกนี้จะช่วยให้แน่ใจว่าไฟล์ Excel ของคุณได้รับการบันทึกเป็น MHTML ซึ่งเป็นรูปแบบที่เหมาะสำหรับการดูและการเก็บถาวรบนเว็บ

2. **ไฟล์แนบอีเมล**:ส่งสเปรดชีตในรูปแบบที่เป็นมิตรต่ออีเมล

- **การประมวลผลแบบแบตช์**:จัดการชุดข้อมูลขนาดใหญ่เป็นกลุ่มแทนที่จะโหลดทุกอย่างลงในหน่วยความจำในครั้งเดียว
