**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

### แสดงเวอร์ชัน Aspose.Cells

**ภาพรวม:** การตรวจสอบเวอร์ชันของ Aspose.Cells จะช่วยให้แน่ใจถึงความเข้ากันได้และช่วยในการดีบัก

1. **นำเข้าคลาสที่จำเป็น:**

   ```java
   import com.aspose.cells.CellsHelper;
   ```

2. **พิมพ์เวอร์ชัน:**

   ใช้ `CellsHelper.getVersion()` เพื่อดึงข้อมูลและแสดงเวอร์ชันปัจจุบัน:

   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // กำหนดเส้นทางไดเร็กทอรีแหล่งที่มาของคุณ

   System.out.println("Aspose.Cells for Java Version: " + CellsHelper.getVersion());
   ```

### โหลดไฟล์ Excel

**ภาพรวม:** การโหลดไฟล์ Excel ลงใน Aspose.Cells ช่วยให้คุณสามารถจัดการและแปลงไฟล์ได้

1. **ตั้งค่าตัวแปรเส้นทาง:**

   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // กำหนดเส้นทางไดเร็กทอรีแหล่งที่มาของคุณ
   ```

2. **โหลดสมุดงาน:**

   สร้าง `Workbook` วัตถุที่ใช้เส้นทางไฟล์:

   ```java
   Workbook wb = new Workbook(dataDir + "/sampleRenderOfficeAdd-Ins.xlsx");
   ```

### แปลงและบันทึก Excel เป็น PDF

**ภาพรวม:** การแปลงเวิร์กบุ๊ก Excel เป็นรูปแบบ PDF เป็นเรื่องง่ายด้วย Aspose.Cells

1. **กำหนดไดเรกทอรีเอาท์พุต:**

   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY"; // กำหนดเส้นทางไดเร็กทอรีเอาท์พุตของคุณ
   ```

2. **บันทึกสมุดงานเป็น PDF:**

   บันทึกสมุดงานที่โหลดไว้ในรูปแบบ PDF พร้อมฝังข้อมูลเวอร์ชัน:

   ```java
   wb.save(outDir + "/output-" + CellsHelper.getVersion() + ".pdf");
   ```
