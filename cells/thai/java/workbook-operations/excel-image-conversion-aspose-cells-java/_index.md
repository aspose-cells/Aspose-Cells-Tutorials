# เรียนรู้การแปลง Excel เป็นรูปภาพใน Java ด้วย Aspose.Cells

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

### คุณสมบัติ: โหลดสมุดงาน
ในการเริ่มประมวลผลไฟล์ Excel คุณต้องโหลดไฟล์นั้นลงใน `Workbook` วัตถุ ขั้นตอนนี้จะเริ่มต้นเวิร์กบุ๊กจากไดเร็กทอรีที่คุณระบุ:
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีข้อมูลจริงของคุณ
Workbook book = new Workbook(dataDir + "/book1.xlsx");
```

### คุณสมบัติ: การเข้าถึงแผ่นงาน
เมื่อโหลดเสร็จแล้ว คุณสามารถเข้าถึงเวิร์กชีตใดๆ ภายในเวิร์กบุ๊กได้ ต่อไปนี้เป็นวิธีรับชีตแรก:
```java
import com.aspose.cells.Worksheet;

Worksheet sheet = book.getWorksheets().get(0);
```
วิธีการนี้ส่งคืน `Worksheet` วัตถุซึ่งช่วยให้สามารถดำเนินการเพิ่มเติมในแผ่นงานเฉพาะนี้ได้

### คุณสมบัติ: กำหนดค่าตัวเลือกภาพและการพิมพ์
เพื่อปรับปรุงคุณภาพเอาต์พุต ให้กำหนดค่าตัวเลือกการเรนเดอร์ภาพโดยใช้ `ImageOrPrintOptions`คลาสนี้ช่วยให้คุณสามารถตั้งค่าความละเอียดและเลือกรูปแบบได้:
```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.ImageType;

ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setHorizontalResolution(300);
options.setVerticalResolution(300);
options.setImageType(ImageType.JPEG);
options.setOnePagePerSheet(true); // แสดงแผ่นงานทั้งหมดเป็นภาพเดียว
```

### คุณสมบัติ: เรนเดอร์แผ่นงานเป็นรูปภาพ
สุดท้ายใช้ `SheetRender` การแปลงและบันทึกเวิร์กชีตของคุณเป็นภาพ:
```java
import com.aspose.cells.SheetRender;

String outDir = "YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอาท์พุตจริงของคุณ
SheetRender sr = new SheetRender(sheet, options);
sr.toImage(0, outDir + "/ConversionOptions_out.jpg");
```
โค้ดนี้จะแสดงหน้าแรกของแผ่นงานเป็นภาพ JPEG และบันทึกลงในไดเร็กทอรีเอาต์พุตที่คุณระบุ
