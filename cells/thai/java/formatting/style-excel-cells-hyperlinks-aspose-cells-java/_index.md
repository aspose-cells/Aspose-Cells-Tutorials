## คู่มือการใช้งาน

ในส่วนนี้เราจะแบ่งการใช้งานออกเป็นขั้นตอนที่จัดการได้เพื่อกำหนดรูปแบบเซลล์และเพิ่มไฮเปอร์ลิงก์โดยใช้ `Aspose-Cells for Java`.

### สร้างและกำหนดรูปแบบเซลล์

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณสร้างเซลล์ Excel ตั้งค่า และใช้รูปแบบเช่น สีแบบอักษรและขีดเส้นใต้

**ขั้นตอน:**
1. **สร้างวัตถุสมุดงาน**
   เริ่มต้นด้วยการสร้างอินสแตนซ์เวิร์กบุ๊กใหม่:
   ```java
   Workbook workbook = new Workbook();
   ```

2. **เข้าถึงคอลเลกชันแผ่นงาน**
   รับการอ้างอิงถึงเวิร์กชีตแรกในเวิร์กบุ๊กของคุณ:
   ```java
   WorksheetCollection worksheets = workbook.getWorksheets();
   Worksheet sheet = worksheets.get(0);
   ```

3. **รับและจัดสไตล์ให้กับเซลล์**
   เข้าถึงเซลล์ A1 ตั้งค่าและใช้ตัวเลือกการจัดรูปแบบเช่นสีแบบอักษรและขีดเส้นใต้:
   ```java
   Cells cells = sheet.getCells();
   Cell cell = cells.get("A1");
   cell.setValue("Visit Aspose");

   Style style = cell.getStyle();
   style.getFont().setColor(com.aspose.cells.Color.getBlue());
   style.getFont().setUnderline(FontUnderlineType.SINGLE);

   // นำรูปแบบไปใช้กับเซลล์
   cell.setStyle(style);
   ```

**ตัวเลือกการกำหนดค่าคีย์:**
- `setFontColor()`: กำหนดสีของข้อความ
- `setUnderline()`: เพิ่มรูปแบบการขีดเส้นใต้

### เพิ่มไฮเปอร์ลิงก์ไปยังเซลล์

#### ภาพรวม

คุณลักษณะนี้ช่วยให้คุณสามารถเพิ่มไฮเปอร์ลิงก์ภายในเวิร์กบุ๊ก Excel ของคุณได้ ซึ่งจะช่วยเพิ่มการโต้ตอบและประโยชน์ใช้สอย

**ขั้นตอน:**
1. **สร้างวัตถุสมุดงาน**
   คล้ายกับการกำหนดสไตล์เซลล์ เริ่มต้นด้วยการสร้างหรือใช้เวิร์กบุ๊กที่มีอยู่:
   ```java
   Workbook workbook = new Workbook();
   ```

2. **เข้าถึงคอลเลกชันแผ่นงาน**
   รับข้อมูลอ้างอิงสำหรับแผ่นงานที่คุณเลือก:
   ```java
   WorksheetCollection worksheets = workbook.getWorksheets();
   Worksheet sheet = worksheets.get(0);
   ```

3. **เพิ่มไฮเปอร์ลิงก์ไปที่เซลล์ A1**
   ใช้ `HyperlinkCollection` เพื่อเพิ่มไฮเปอร์ลิงก์ไปยังเซลล์ A1:
   ```java
   HyperlinkCollection hyperlinks = sheet.getHyperlinks();
   hyperlinks.add("A1", 1, 1, "http://www.aspose.com");
   ```

### บันทึกสมุดงาน

หลังจากกำหนดรูปแบบเซลล์และเพิ่มไฮเปอร์ลิงก์แล้ว ให้บันทึกสมุดงานของคุณ:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/StyledWorkbook.xls");
```
