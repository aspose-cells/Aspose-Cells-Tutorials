# ปรับแต่งเอฟเฟกต์ข้อความและเงาใน Excel ด้วย Aspose.Cells Java

## การจัดรูปแบบการนำเสนอใน Excel: เพิ่มเงาแบบไดนามิกให้กับรูปร่างและกล่องข้อความ

**สิ่งที่คุณจะได้เรียนรู้:**
- การนำเอฟเฟกต์ข้อความและเงาไปใช้งานใน Excel ด้วย Aspose.Cells

- การใช้งานจริงของการปรับปรุงข้อความแบบไดนามิก

## คู่มือการใช้งาน: การเพิ่มเอฟเฟกต์เงาให้กับข้อความใน Excel

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มเอฟเฟ็กต์เงาให้กับ TextBox ภายในเวิร์กชีต Excel

### ขั้นตอนที่ 1: สร้างและกำหนดค่าเวิร์กบุ๊ก

ตั้งค่าสมุดงานของคุณและเข้าถึงแผ่นงานแรก:

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

// การเริ่มต้นสมุดงาน
Workbook wb = new Workbook();

// เข้าถึงแผ่นงานแรก
Worksheet ws = wb.getWorksheets().get(0);
```

### ขั้นตอนที่ 2: เพิ่มกล่องข้อความพร้อมเอฟเฟกต์ข้อความ

เพิ่ม TextBox และตั้งค่าข้อความพร้อมเอฟเฟกต์เงา:

```java
import com.aspose.cells.TextBox;
import com.aspose.cells.PresetShadowType;

// เพิ่มกล่องข้อความที่พิกัดที่ระบุ
int idx = ws.getTextBoxes().add(2, 2, 100, 400);
TextBox tb = ws.getTextBoxes().get(idx);

// ตั้งค่าข้อความของ TextBox
tb.setText("This text has the following settings.\n\nText Effects > Shadow > Offset Bottom");

// ใช้เอฟเฟกต์เงาให้กับแต่ละข้อความที่รันในกล่องข้อความ
for (int i = 0; i < tb.getTextBody().getCount(); i++) {
    tb.getTextBody().get(i).getTextOptions().getShadow().setPresetType(PresetShadowType.OFFSET_BOTTOM);
}
```

### ขั้นตอนที่ 3: ปรับแต่งลักษณะข้อความ

ปรับแต่งสีและขนาดตัวอักษรเพื่อให้ข้อความของคุณโดดเด่น:

```java
import com.aspose.cells.Color;

// ตั้งค่าสีและขนาดของตัวอักษรของ TextBox
tb.getFont().setColor(Color.getRed());
tb.getFont().setSize(16);
```

### ขั้นตอนที่ 4: บันทึกสมุดงานของคุณ

สุดท้าย ให้บันทึกสมุดงานโดยใช้การตั้งค่าใหม่ที่ใช้:

```java
import com.aspose.cells.SaveFormat;

String dataDir = "path/to/your/directory/";
wb.save(dataDir + "STESOfShapeOrTextbox_out.xlsx", SaveFormat.XLSX);
```
