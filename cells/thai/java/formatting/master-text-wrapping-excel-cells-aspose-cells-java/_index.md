## คู่มือการใช้งาน

### การสร้างเวิร์กบุ๊กและการเข้าถึงเซลล์

ขั้นแรก ให้สร้างอินสแตนซ์เวิร์กบุ๊กและเข้าถึงเซลล์ของมัน:

```java
// สร้างวัตถุเวิร์กบุ๊กใหม่
document = new Workbook();

// เปิดเวิร์กชีตแรกในเวิร์กบุ๊ก
worksheet = document.getWorksheets().get(0);

// รับคอลเลกชันของเซลล์จากเวิร์กชีต
cells = worksheet.getCells();
```

### การกำหนดค่าความกว้างของคอลัมน์และความสูงของแถว

ปรับความกว้างของคอลัมน์และความสูงของแถวเพื่อให้แน่ใจว่าข้อความพอดี:

```java
// เพิ่มความกว้างของคอลัมน์แรก
cells.setColumnWidth(0, 35);

// เพิ่มความสูงของแถวแรก
cells.setRowHeight(0, 65);
```

### การเพิ่มข้อความและการใช้รูปแบบการห่อ

เพิ่มข้อความลงในเซลล์และเปิดใช้งานการตัดข้อความ:

```java
// เพิ่มข้อความลงในเซลล์แรก
cells.get(0, 0).setValue("I am using the latest version of Aspose.Cells to test this functionality");

// รับสไตล์ของเซลล์
Style style = cells.get(0, 0).getStyle();

// เปิดใช้งานการห่อข้อความสำหรับเนื้อหาของเซลล์
style.setTextWrapped(true);

// นำสไตล์กลับไปใช้กับเซลล์
cells.get(0, 0).setStyle(style);
```

### การบันทึกสมุดงานของคุณ

บันทึกสมุดงานของคุณด้วยข้อความที่ถูกห่อหุ้ม:

```java
// บันทึกไฟล์ Excel
document.save("WrapTextinCell_out.xls");
```
