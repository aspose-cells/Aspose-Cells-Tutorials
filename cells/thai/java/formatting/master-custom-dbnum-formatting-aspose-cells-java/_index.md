# เรียนรู้การจัดรูปแบบเลขทศนิยมแบบกำหนดเองด้วย Aspose.Cells ใน Java
ปลดล็อกพลังของการจัดรูปแบบตัวเลขแบบ Excel ในแอปพลิเคชัน Java ของคุณโดยใช้ Aspose.Cells คู่มือที่ครอบคลุมนี้จะแนะนำคุณเกี่ยวกับการใช้การจัดรูปแบบตัวเลขทศนิยม (DBNum) ที่กำหนดเอง ทำให้การนำเสนอข้อมูลของคุณแม่นยำและเป็นมืออาชีพ

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน
### ภาพรวมของการจัดรูปแบบตัวเลขทศนิยมแบบกำหนดเอง
การจัดรูปแบบตัวเลขทศนิยมแบบกำหนดเองช่วยให้คุณควบคุมวิธีการแสดงตัวเลขในแอปพลิเคชันของคุณได้ตามรูปแบบตัวเลขของ Excel คุณสมบัตินี้มีความสำคัญสำหรับแอปพลิเคชันที่ต้องการการแสดงข้อมูลที่แม่นยำ

#### การดำเนินการแบบทีละขั้นตอน
**1. สร้างเวิร์กบุ๊กและเข้าถึงเวิร์กชีต**
ขั้นแรก ให้สร้างอินสแตนซ์ของ `Workbook` ชั้นเรียนและเข้าถึงแผ่นงานแรก:
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class Main {
    public static void main(String[] args) throws Exception {
        // การเริ่มต้นสมุดงาน
        Workbook wb = new Workbook();
        
        // เข้าถึงแผ่นงานแรก
        Worksheet ws = wb.getWorksheets().get(0);
    }
}
```
**2. แทรกและจัดรูปแบบค่า**
แทรกค่าลงในเซลล์ A1 และใช้การจัดรูปแบบเลขทศนิยมแบบกำหนดเอง:
```java
import com.aspose.cells.Cell;
import com.aspose.cells.Style;

public class Main {
    public static void main(String[] args) throws Exception {
        // โค้ดก่อนหน้า...

        // เข้าถึงเซลล์ A1 และใส่ค่า 123
        Cell cell = ws.getCells().get("A1");
        cell.putValue(123);

        // ใช้การจัดรูปแบบเลขทศนิยมแบบกำหนดเอง
        Style st = cell.getStyle();
        st.setCustom("[DBNum2][$-804]General");

        // ตั้งค่ารูปแบบรูปแบบให้กับเซลล์
        cell.setStyle(st);
    }
}
```
**3. บันทึกสมุดงานเป็น PDF**
สุดท้ายให้บันทึกสมุดงานของคุณในรูปแบบ PDF:
```java
import com.aspose.cells.SaveFormat;

public class Main {
    public static void main(String[] args) throws Exception {
        // โค้ดก่อนหน้า...

        // กำหนดความกว้างของคอลัมน์และบันทึกสมุดงาน
        ws.getCells().setColumnWidth(0, 30);
        wb.save("outputDecimalNumberFormatting.pdf", SaveFormat.PDF);

        System.out.println("Custom Decimal Number Formatting Done Successfully");
    }
}
```
**เคล็ดลับการแก้ไขปัญหา**
- ตรวจสอบให้แน่ใจว่าไลบรารี Aspose.Cells ถูกเพิ่มลงในโปรเจ็กต์ของคุณอย่างถูกต้อง
- ตรวจสอบข้อผิดพลาดทางไวยากรณ์ในการจัดรูปแบบของสตริง
