ในบทความนี้ เราจะสำรวจวิธีการใช้ Aspose.Cells สำหรับ Java เพื่อ:
- โหลดเวิร์กบุ๊ก Excel จากไดเร็กทอรีที่ระบุ
- ส่งออกข้อมูล XML ทั้งหมดที่เชื่อมโยงกับแผนที่ XML ในเวิร์กบุ๊ก

หากทำตามคำแนะนำนี้ คุณจะปรับปรุงความสามารถในการจัดการและใช้ข้อมูลภายในไฟล์ Excel ได้ด้วยโปรแกรม มาเจาะลึกสิ่งที่คุณต้องการก่อนที่เราจะเริ่มกัน

## คู่มือการใช้งาน
### โหลดสมุดงานแหล่งที่มา
#### ภาพรวม
ขั้นตอนแรกคือโหลดเวิร์กบุ๊ก Excel จากไดเร็กทอรีที่ระบุ ซึ่งจะช่วยให้คุณเข้าถึงและจัดการข้อมูลภายในได้

#### ตัวอย่างโค้ด
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY/TechnicalArticles/";
Workbook wb = new Workbook(dataDir + "sample_Export-Xml-Data-linked.xlsx");
```
**คำอธิบาย**- 
- `dataDir` ระบุตำแหน่งของสมุดงานของคุณ
- การ `Workbook` วัตถุโหลดไฟล์ Excel ทำให้พร้อมสำหรับการดำเนินการต่อไป

### ส่งออกข้อมูล XML ของแผนที่ XML
#### ภาพรวม
เมื่อโหลดเวิร์กบุ๊กเสร็จแล้ว คุณสามารถส่งออกข้อมูล XML ทั้งหมดที่เกี่ยวข้องกับแผนที่ XML ภายในเวิร์กบุ๊กได้ ฟีเจอร์นี้มีความสำคัญอย่างยิ่งในการบูรณาการหรือวิเคราะห์ข้อมูล XML ด้วยโปรแกรม

#### ตัวอย่างโค้ด
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.XmlMap;

String outDir = "YOUR_OUTPUT_DIRECTORY/";
for (int i = 0; i < wb.getWorksheets().getXmlMaps().getCount(); i++) {
    XmlMap map = wb.getWorksheets().getXmlMaps().get(i);
    wb.exportXml(map.getName(), outDir + map.getName() + ".xml");
}
```
**คำอธิบาย**- 
- ทำซ้ำผ่านแต่ละ `XmlMap` ในสมุดงาน
- ใช้ `exportXml()` วิธีการส่งออกข้อมูลจากแต่ละ XML Map ไปยังไดเร็กทอรีเอาท์พุตที่ระบุ

#### ตัวเลือกการกำหนดค่าคีย์
- ปรับเปลี่ยน `dataDir` และ `outDir` เส้นทางตามโครงสร้างไฟล์ของคุณ
- ให้แน่ใจว่าไดเร็กทอรีที่ระบุมีอยู่ก่อนที่จะรันโปรแกรมเพื่อหลีกเลี่ยงข้อยกเว้น
