## คู่มือการใช้งาน

### การส่งออกช่วงเซลล์เป็นรูปภาพ

หัวข้อนี้จะอธิบายวิธีการส่งออกเซลล์เฉพาะจากเวิร์กชีต Excel ไปยังภาพ JPEG โดยใช้ Aspose.Cells สำหรับ Java

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

เริ่มต้นด้วยการโหลดไฟล์ Excel ที่คุณต้องการแยกช่วงเซลล์:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "book1.xlsx");
```

#### ขั้นตอนที่ 2: เข้าถึงแผ่นงาน

เข้าถึงแผ่นงานที่มีเซลล์ที่ต้องการ:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### ขั้นตอนที่ 3: ตั้งค่าพื้นที่การพิมพ์

ระบุส่วนของเวิร์กชีตที่จะส่งออกโดยตั้งค่าพื้นที่พิมพ์:

```java
worksheet.getPageSetup().setPrintArea("E8:H10");
```

ขั้นตอนนี้จะช่วยให้แน่ใจว่ามีการพิจารณาส่งออกเฉพาะช่วงที่ระบุเท่านั้น

#### ขั้นตอนที่ 4: กำหนดค่าตัวเลือกภาพ

กำหนดวิธีที่คุณต้องการให้ภาพของคุณออกมาโดยใช้ `ImageOrPrintOptions`-

```java
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setOnePagePerSheet(true);
options.setImageType(ImageType.JPEG);
```

ที่นี่ เราตั้งค่าการส่งออกเพื่อสร้างรูปภาพ JPEG หนึ่งภาพต่อแผ่น

#### ขั้นตอนที่ 5: สร้างและใช้ SheetRender

ใช้ประโยชน์ `SheetRender` ในการจัดการการเรนเดอร์เวิร์กชีตของคุณเป็นรูปภาพ:

```java
SheetRender sr = new SheetRender(worksheet, options);
sr.toImage(0, "YOUR_OUTPUT_DIRECTORY" + "ERangeofCells_out.jpg");
```

ขั้นตอนสุดท้ายนี้จะสร้างช่วงเซลล์ที่ระบุเป็นไฟล์ JPEG

### การตั้งค่าพื้นที่พิมพ์ของแผ่นงาน

การปรับพื้นที่การพิมพ์เป็นสิ่งสำคัญในการกำหนดว่าจะใช้ส่วนใดของเวิร์กชีตของคุณ ดังต่อไปนี้:

```java
worksheet.getPageSetup().setPrintArea("E8:H10");
```

การตั้งค่านี้จะทำให้คุณเน้นที่ช่วง E8:H10 ในระหว่างการส่งออกหรือการพิมพ์

### การปรับระยะขอบของแผ่นงาน

หากต้องการลบระยะห่างที่ไม่จำเป็นรอบพื้นที่พิมพ์ ให้ตั้งค่าระยะขอบทั้งหมดเป็นศูนย์:

```java
worksheet.getPageSetup()
    .setLeftMargin(0)
    .setRightMargin(0)
    .setTopMargin(0)
    .setBottomMargin(0);
```

ซึ่งรับประกันได้ว่าภาพที่ออกมาจะสะอาดโดยไม่มีขอบพิเศษใดๆ
