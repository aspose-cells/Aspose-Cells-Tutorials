# เรียนรู้การใช้งานเวิร์กบุ๊กที่แชร์กันใน Java ด้วย Aspose.Cells

## การแนะนำ

ในคู่มือนี้เราจะครอบคลุมถึง:

เมื่อสิ้นสุดหลักสูตร คุณจะมีข้อมูลเชิงปฏิบัติเกี่ยวกับการใช้คุณลักษณะเหล่านี้อย่างมีประสิทธิภาพ มาเริ่มด้วยข้อกำหนดเบื้องต้นกันก่อน

## คู่มือการใช้งาน
### แสดงเวอร์ชัน Aspose.Cells
**ภาพรวม**: ตรวจสอบเวอร์ชันของ Aspose.Cells สำหรับ Java ที่คุณใช้

```java
import com.aspose.cells.*;
```

#### ขั้นตอนที่ 2: พิมพ์ข้อมูลเวอร์ชัน
```java
String dataDir = "YOUR_DATA_DIRECTORY";
System.out.println("Aspose.Cells for Java Version: " + CellsHelper.getVersion());
```
### สร้างและแบ่งปันสมุดงาน
**ภาพรวม**: สร้างเวิร์กบุ๊กเปล่าและกำหนดค่าให้สามารถใช้ร่วมกันได้

#### ขั้นตอนที่ 1: เริ่มต้น Aspose.Cells และสร้างเวิร์กบุ๊กใหม่
```java
Workbook wb = new Workbook();
```

#### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการแชร์สมุดงาน
```java
WritableCellSettings writableCellSettings = new WritableCellSettings(wb.getWorksheets().get(0));
writableCellSettings.setShared(true);
wb.save("YOUR_OUTPUT_DIRECTORY/outputShared_Workbook.xlsx");
```
### อัปเดตวันเก็บรักษาประวัติการแก้ไขบันทึกในสมุดงานที่แชร์
**ภาพรวม**ปรับจำนวนวันที่สมุดงานที่ใช้ร่วมกันจะเก็บรักษาประวัติการแก้ไขไว้

#### ขั้นตอนที่ 1: เริ่มต้น Aspose.Cells และสร้างเวิร์กบุ๊กใหม่
```java

WritableCellSettings writableCellSettings = new WritableCellSettings(wb.getWorksheets().get(0));
```

#### ขั้นตอนที่ 2: กำหนดค่าการตั้งค่าบันทึกการแก้ไข
```java

writableCellSettings.getWorksheets().getRevisionLogs().setDaysPreservingHistory(7); // ตั้งไว้ 7 วัน
wb.save("YOUR_OUTPUT_DIRECTORY/outputShared_DaysPreservingHistory.xlsx");
```
