## คู่มือการใช้งาน

### การโหลดไฟล์ Excel ด้วยรหัสผ่าน
#### ภาพรวม
การโหลดไฟล์ Excel ที่ได้รับการป้องกันด้วยรหัสผ่านถือเป็นสิ่งสำคัญเมื่อต้องจัดการกับข้อมูลที่ละเอียดอ่อน Aspose.Cells มอบวิธีง่ายๆ เพื่อเปิดไฟล์เหล่านี้โดยระบุรหัสผ่านที่จำเป็น

#### การดำเนินการแบบทีละขั้นตอน
1. **ตั้งค่า LoadOptions**สร้าง `LoadOptions` วัตถุและตั้งรหัสผ่านที่จำเป็น
   ```java
   import com.aspose.cells.LoadOptions;
   import com.aspose.cells.Workbook;

   String dataDir = "YOUR_DATA_DIRECTORY";
   LoadOptions opts = new LoadOptions();
   opts.setPassword("1234"); // ตั้งรหัสผ่านเพื่อเปิดไฟล์
   ```
2. **โหลดสมุดงาน**: ใช้ `Workbook` คลาสที่มีตัวเลือกที่ระบุเพื่อโหลดไฟล์ Excel ของคุณ
   ```java
   Workbook workbook = new Workbook(dataDir + "Book1.xlsx", opts);
   ```
### การตรวจสอบรหัสผ่านเพื่อการแก้ไข
#### ภาพรวม
สิ่งสำคัญคือต้องทำการเปลี่ยนแปลงที่ได้รับอนุญาตเท่านั้นในไฟล์ Excel ที่ได้รับการป้องกัน Aspose.Cells ช่วยให้คุณตรวจสอบรหัสผ่านที่ใช้สำหรับแก้ไขไฟล์เหล่านี้ได้
#### การดำเนินการแบบทีละขั้นตอน
1. **โหลดสมุดงาน**:ขั้นแรกให้โหลดไฟล์ Excel ตามที่แสดงไว้ในส่วนก่อนหน้า
2. **ยืนยันรหัสผ่าน**: ใช้ `validatePassword` วิธีการตรวจสอบว่ารหัสผ่านที่ให้มาตรงกับรหัสผ่านในการแก้ไขหรือไม่
   ```java
   boolean ret = workbook.getSettings().getWriteProtection().validatePassword("567");
   // ตรวจสอบรหัสผ่านที่เป็นไปได้อื่น ๆ
   ret = workbook.getSettings().getWriteProtection().validatePassword("5678");
   ```
### เคล็ดลับการแก้ไขปัญหา
- **รหัสผ่านไม่ถูกต้อง**: ตรวจสอบให้แน่ใจว่าสะกดรหัสผ่านถูกต้องและตรงกับรหัสผ่านที่ใช้ในการรักษาความปลอดภัยไฟล์
- **ปัญหาเส้นทางไฟล์**:ตรวจสอบว่าเส้นทางไปยังไฟล์ Excel ของคุณถูกต้องและสามารถเข้าถึงได้โดยแอปพลิเคชัน Java ของคุณ
