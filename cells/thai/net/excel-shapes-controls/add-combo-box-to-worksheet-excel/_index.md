# เพิ่มกล่องคอมโบลงในเวิร์กชีตใน Excel

## การแนะนำ
การสร้างสเปรดชีต Excel แบบโต้ตอบสามารถปรับปรุงประสบการณ์ของผู้ใช้ได้อย่างมาก โดยเฉพาะอย่างยิ่งเมื่อคุณเพิ่มองค์ประกอบแบบฟอร์ม เช่น กล่องคอมโบ กล่องคอมโบช่วยให้ผู้ใช้สามารถเลือกตัวเลือกจากรายการที่กำหนดไว้ล่วงหน้า ทำให้การป้อนข้อมูลสะดวกและมีประสิทธิภาพมากขึ้น ด้วย Aspose.Cells สำหรับ .NET คุณสามารถสร้างกล่องคอมโบในแผ่นงาน Excel ได้โดยการเขียนโปรแกรมโดยไม่ต้องใช้ Excel โดยตรง ไลบรารีอันทรงพลังนี้ช่วยให้นักพัฒนาสามารถจัดการไฟล์ Excel ได้หลายวิธี รวมถึงความสามารถในการควบคุมแบบฟอร์มโดยอัตโนมัติ
ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับขั้นตอนการเพิ่มกล่องรวมข้อมูลลงในเวิร์กชีตใน Excel โดยใช้ Aspose.Cells สำหรับ .NET หากคุณต้องการสร้างสเปรดชีตแบบไดนามิกที่ใช้งานง่าย คู่มือนี้จะช่วยคุณเริ่มต้นได้

## แพ็คเกจนำเข้า
ในการเริ่มต้น คุณต้องนำเข้าเนมสเปซที่จำเป็นลงในโปรเจ็กต์ของคุณ นี่คือสิ่งที่คุณต้องการ:
```csharp
using System.IO;
using Aspose.Cells;
```
สิ่งเหล่านี้มีความจำเป็นสำหรับการโต้ตอบกับไฟล์ Excel และการจัดการองค์ประกอบแบบฟอร์ม เช่น กล่องคอมโบในเวิร์กบุ๊ก
มาแบ่งขั้นตอนการเพิ่มกล่องคอมโบออกเป็นหลายขั้นตอนง่าย ๆ เพื่อความเข้าใจง่าย
## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร
ขั้นตอนแรกคือการสร้างไดเรกทอรีที่จะบันทึกไฟล์ Excel ของคุณ คุณสามารถสร้างโฟลเดอร์ใหม่ได้หากยังไม่มีอยู่
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "Your Document Directory";
// สร้างไดเร็กทอรีหากยังไม่มีอยู่
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
- dataDir: ระบุตำแหน่งที่จะบันทึกไฟล์เอาต์พุต
- System.IO.Directory.Exists: ตรวจสอบว่าไดเร็กทอรีมีอยู่แล้วหรือไม่
- System.IO.Directory.CreateDirectory: สร้างไดเร็กทอรีถ้าขาดหายไป
## ขั้นตอนที่ 2: สร้างสมุดงานใหม่
ตอนนี้ ให้สร้างเวิร์กบุ๊ก Excel ใหม่ โดยคุณจะเพิ่มกล่องคอมโบ

```csharp
// สร้างสมุดงานใหม่
Workbook workbook = new Workbook();
```

- สมุดงาน สมุดงาน: เริ่มต้นอินสแตนซ์ใหม่ของคลาสสมุดงาน ซึ่งแสดงถึงไฟล์ Excel
## ขั้นตอนที่ 3: รับแผ่นงานและเซลล์
ขั้นตอนต่อไป คือเข้าถึงเวิร์กชีตแรกจากเวิร์กบุ๊กและดึงคอลเลกชันเซลล์ที่คุณจะป้อนข้อมูล

```csharp
// รับแผ่นงานแรก
Worksheet sheet = workbook.Worksheets[0];
// รับคอลเลกชันเซลล์ของเวิร์กชีต
Cells cells = sheet.Cells;
```

- แผ่นงานเวิร์กชีท: ดึงแผ่นงานแรกจากเวิร์กบุ๊ก
- เซลล์ เซลล์: รับคอลเลกชันของเซลล์จากเวิร์กชีต
## ขั้นตอนที่ 4: ป้อนค่าสำหรับกล่องคอมโบ
ตอนนี้เราต้องป้อนค่าบางค่าลงในเซลล์ ค่าเหล่านี้จะทำหน้าที่เป็นตัวเลือกสำหรับกล่องรวมข้อมูล

```csharp
// ป้อนค่า
cells["B3"].PutValue("Employee:");
// ตั้งค่าให้เป็นตัวหนา
cells["B3"].GetStyle().Font.IsBold = true;
// ป้อนค่าบางค่าที่แสดงช่วงอินพุตสำหรับกล่องคอมโบ
cells["A2"].PutValue("Emp001");
cells["A3"].PutValue("Emp002");
cells["A4"].PutValue("Emp003");
cells["A5"].PutValue("Emp004");
cells["A6"].PutValue("Emp005");
cells["A7"].PutValue("Emp006");
```

- cells["B3"].PutValue: วางป้ายชื่อ "พนักงาน" ไว้ในเซลล์ B3
- Font.IsBold = true: ตั้งค่าข้อความเป็นตัวหนาเพื่อให้โดดเด่น
- ช่วงอินพุต: ป้อนรหัสพนักงานหลายรหัสในเซลล์ A2 ถึง A7 รหัสเหล่านี้จะปรากฏในกล่องรวมแบบดรอปดาวน์
## ขั้นตอนที่ 5: เพิ่มกล่องคอมโบลงในเวิร์กชีต
ขั้นตอนต่อไปคือการเพิ่มตัวควบคุมกล่องรวมข้อมูลลงในเวิร์กชีตของคุณ กล่องรวมข้อมูลนี้จะให้ผู้ใช้เลือกรหัสพนักงานที่คุณป้อนไว้ก่อนหน้านี้ได้

```csharp
// เพิ่มกล่องคอมโบใหม่
Aspose.Cells.Drawing.ComboBox comboBox = sheet.Shapes.AddComboBox(2, 0, 2, 0, 22, 100);
```

- AddComboBox: เพิ่มกล่องคอมโบใหม่ลงในเวิร์กชีต ตัวเลข (2, 0, 2, 0, 22, 100) แสดงถึงตำแหน่งและขนาดของกล่องคอมโบ
## ขั้นตอนที่ 6: เชื่อมโยงกล่องคอมโบกับเซลล์และตั้งค่าช่วงอินพุต
ในการสร้างกล่องรวมให้ใช้งานได้ เราจำเป็นต้องลิงก์กล่องเข้ากับเซลล์เฉพาะ และกำหนดช่วงเซลล์ที่จะดึงตัวเลือกมาจากกล่องดังกล่าว

```csharp
// ตั้งค่าเซลล์ที่เชื่อมโยง
comboBox.LinkedCell = "A1";
// ตั้งค่าช่วงอินพุต
comboBox.InputRange = "A2:A7";
```

- LinkedCell: เชื่อมโยงการเลือกของกล่องคอมโบเข้ากับเซลล์ A1 ค่าที่เลือกจากกล่องคอมโบจะปรากฏในเซลล์นี้
- InputRange: กำหนดช่วงเซลล์ (A2:A7) ที่มีค่าที่จะปรากฏในตัวเลือกกล่องคอมโบ
## ขั้นตอนที่ 7: ปรับแต่งรูปลักษณ์ของกล่องคอมโบ
คุณสามารถปรับแต่งกล่องคอมโบเพิ่มเติมได้โดยการระบุจำนวนบรรทัดดรอปดาวน์และเปิดใช้งานการแรเงา 3 มิติเพื่อความสวยงามยิ่งขึ้น

```csharp
// ตั้งค่าจำนวนบรรทัดรายการที่จะแสดงในส่วนรายการของกล่องคอมโบ
comboBox.DropDownLines = 5;
// ตั้งค่ากล่องคอมโบด้วยการแรเงา 3 มิติ
comboBox.Shadow = true;
```

- DropDownLines: ควบคุมจำนวนตัวเลือกที่จะมองเห็นได้ในกล่องรวมแบบดรอปดาวน์ในครั้งเดียว
- เงา: เพิ่มเอฟเฟกต์เงา 3 มิติให้กับกล่องคอมโบ
## ขั้นตอนที่ 8: ปรับคอลัมน์ให้พอดีอัตโนมัติและบันทึกสมุดงาน
สุดท้าย ให้ปรับขนาดคอลัมน์ให้พอดีโดยอัตโนมัติเพื่อให้ได้เค้าโครงที่สะอาด และบันทึกเวิร์กบุ๊ก

```csharp
// คอลัมน์ปรับพอดีอัตโนมัติ
sheet.AutoFitColumns();
// บันทึกไฟล์
workbook.Save(dataDir + "book1.out.xls");
```

- AutoFitColumns: ปรับความกว้างของคอลัมน์โดยอัตโนมัติเพื่อให้พอดีกับเนื้อหา
- บันทึก: บันทึกเวิร์กบุ๊กเป็นไฟล์ Excel ในไดเร็กทอรีที่ระบุ
