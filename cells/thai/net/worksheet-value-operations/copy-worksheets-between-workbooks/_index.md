## แพ็คเกจนำเข้า
ในการเริ่มต้น คุณต้องนำเข้าเนมสเปซที่จำเป็นในโค้ดของคุณ โดยทำตามขั้นตอนดังต่อไปนี้:
```csharp

```
บรรทัดเดียวนี้จะทำให้คุณเข้าถึงฟีเจอร์อันทรงพลังทั้งหมดของ Aspose.Cells
ในบทช่วยสอนนี้ เราจะแบ่งงานออกเป็นขั้นตอนที่จัดการได้ แต่ละขั้นตอนจะต่อยอดจากขั้นตอนก่อนหน้า ดังนั้น คุณจะมีโค้ดที่ทำงานได้สมบูรณ์ในตอนท้าย
## ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสาร
ก่อนอื่นเรามาระบุเส้นทางที่เก็บไฟล์เวิร์กบุ๊กของเรา เส้นทางนี้จะแจ้งให้โปรแกรมทราบว่าจะค้นหาเวิร์กบุ๊กต้นฉบับได้จากที่ใด และจะบันทึกไฟล์ที่คัดลอกไว้ที่ไหน
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "Your Document Directory";
```
 ที่นี่แทนที่`"Your Document Directory"` ด้วยเส้นทางจริงที่คุณบันทึกไฟล์ไว้
## ขั้นตอนที่ 2: ตั้งค่าเส้นทางไฟล์อินพุต
ในขั้นตอนนี้ เราจะกำหนดเส้นทางไปยังเวิร์กบุ๊กต้นฉบับที่มีเวิร์กชีตที่เราต้องการคัดลอก สำหรับการสาธิต สมมติว่าไฟล์นี้มีชื่อว่า`book1.xls`.
```csharp
string inputPath = dataDir + "book1.xls";
```
 เส้นนี้รวม`dataDir` ด้วยชื่อไฟล์ สร้างเส้นทางที่สมบูรณ์ไปยัง`book1.xls`นี่คือสมุดงานที่มีแผ่นงานที่เราจะคัดลอก
## ขั้นตอนที่ 3: เปิดสมุดงานต้นฉบับ
ตอนนี้เรามาเปิดสมุดงานต้นฉบับกัน (`book1.xls` ) โดยการสร้าง`Workbook` วัตถุและส่งผ่านใน`inputPath` เป็นข้อโต้แย้ง
```csharp
// สร้างสมุดงาน
// เปิดไฟล์ลงในหนังสือเล่มแรก
Workbook sourceWorkbook = new Workbook(inputPath);
```
 ที่นี่เราจะเริ่มต้น`sourceWorkbook` เพื่อแสดงเวิร์กบุ๊กต้นฉบับของเรา อ็อบเจ็กต์นี้ช่วยให้เราเข้าถึงเวิร์กชีตทั้งหมดภายในไฟล์ได้
## ขั้นตอนที่ 4: สร้างสมุดงานปลายทาง
ในขั้นตอนนี้ เราจะสร้างเวิร์กบุ๊กใหม่เพื่อใช้เป็นปลายทางสำหรับเวิร์กชีตที่คัดลอกมา ซึ่งจะทำหน้าที่เป็นกระดานเปล่าสำหรับวางชีตที่คัดลอกมา
```csharp
// สร้างสมุดงานใหม่
Workbook destinationWorkbook = new Workbook();
```
 ของเรา`destinationWorkbook` จะว่างเปล่าตามค่าเริ่มต้น โดยมีเพียงเวิร์กชีตเดียว
## ขั้นตอนที่ 5: คัดลอกเวิร์กชีตไปยังเวิร์กบุ๊กใหม่
ตอนนี้มาถึงหัวใจหลักของบทช่วยสอนนี้แล้ว นั่นก็คือการคัดลอกเวิร์กชีต เราจะคัดลอกเวิร์กชีตแรกจากเวิร์กบุ๊กต้นฉบับแล้ววางลงในช่องเวิร์กชีตแรกของเวิร์กบุ๊กปลายทาง
```csharp
// คัดลอกแผ่นงานแรกของสมุดงานต้นฉบับไปยังสมุดงานปลายทาง
destinationWorkbook.Worksheets[0].Copy(sourceWorkbook.Worksheets[0]);
```
ในโค้ดนี้:
- `sourceWorkbook.Worksheets[0]` แสดงถึงเวิร์กชีทแรกในเวิร์กบุ๊กต้นฉบับของเรา
- `destinationWorkbook.Worksheets[0]` หมายถึงแผ่นงานแรกในสมุดงานปลายทาง
-  การ`.Copy` วิธีนี้ช่วยยกของหนักโดยโอนเวิร์กชีตจากสมุดงานหนึ่งไปยังอีกสมุดงานหนึ่งได้อย่างราบรื่น
## ขั้นตอนที่ 6: บันทึกสมุดงานปลายทาง
สุดท้ายนี้ ให้บันทึกเวิร์กบุ๊กปลายทางของเรา การดำเนินการนี้จะทำให้กระบวนการคัดลอกเสร็จสมบูรณ์ และสร้างไฟล์เอาต์พุตที่มีเวิร์กชีตที่คัดลอกมา
```csharp
// บันทึกไฟล์
destinationWorkbook.Save(dataDir + "CopyWorksheetsBetweenWorkbooks_out.xls");
```
 แทนที่`"CopyWorksheetsBetweenWorkbooks_out.xls"` ด้วยชื่อไฟล์เอาต์พุตที่คุณต้องการ ตอนนี้คุณจะมีไฟล์ใหม่ในไดเร็กทอรีที่คุณระบุพร้อมกับเวิร์กชีตที่คัดลอกมา
