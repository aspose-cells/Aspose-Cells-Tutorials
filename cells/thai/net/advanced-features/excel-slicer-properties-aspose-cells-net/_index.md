## คู่มือการใช้งาน

มาแยกรายละเอียดกระบวนการสร้างและเพิ่มประสิทธิภาพตัวแบ่งส่วนใน Excel โดยใช้ Aspose.Cells กัน

### การเพิ่ม Slicer ลงในตาราง Excel

#### ภาพรวม
เราเริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีอยู่ เข้าถึงเวิร์กชีตของไฟล์นั้น จากนั้นเพิ่มตัวแบ่งข้อมูลที่เชื่อมโยงกับตาราง วิธีนี้ทำให้ผู้ใช้สามารถกรองข้อมูลแบบไดนามิกตามเกณฑ์เฉพาะได้

#### การดำเนินการทีละขั้นตอน:

**1. โหลดเวิร์กบุ๊ก:**
```csharp
// โหลดไฟล์ Excel ตัวอย่างที่มีตาราง
Workbook workbook = new Workbook("sampleCreateSlicerToExcelTable.xlsx");
```
ที่นี่ เราโหลดเวิร์กบุ๊กที่มีอยู่ซึ่งประกอบด้วยเวิร์กชีตอย่างน้อยหนึ่งแผ่นพร้อมตารางข้อมูล

**2. เข้าถึงแผ่นงานและตาราง:**
```csharp
// เข้าถึงแผ่นงานแรก
Worksheet worksheet = workbook.Worksheets[0];

// เข้าถึงตารางแรกภายในเวิร์กชีต
ListObject table = worksheet.ListObjects[0];
```
สไนปเป็ตนี้จะเข้าถึงเวิร์กชีตแรกและวัตถุรายการแรก (ตาราง) ภายในนั้น

**3. เพิ่ม Slicer ลงในตาราง:**
```csharp
// เพิ่มตัวแบ่งสำหรับคอลัมน์เฉพาะ เช่น "หมวดหมู่" ที่ตำแหน่ง H5
int idx = worksheet.Slicers.Add(table, 0, "H5");
Slicer slicer = worksheet.Slicers[idx];
```
เราเพิ่มตัวแบ่งข้อมูลที่เชื่อมโยงกับคอลัมน์แรกของตารางและวางไว้โดยเริ่มจากเซลล์ H5

### การปรับแต่งคุณสมบัติของ Slicer

#### ภาพรวม
หลังจากเพิ่มเครื่องแบ่งส่วนแล้ว เราจะปรับแต่งคุณสมบัติต่างๆ เช่น ตำแหน่ง ขนาด ชื่อเรื่อง และอื่นๆ เพื่อให้ตรงกับความต้องการเฉพาะของผู้ใช้

**1. การจัดวางและขนาด:**
```csharp
// ปรับแต่งตำแหน่งและขนาดของเครื่องหั่น
slicer.Placement = PlacementType.FreeFloating;
slicer.RowHeightPixel = 50;
slicer.WidthPixel = 500;
```
การกำหนดค่านี้จะช่วยให้เครื่องตัดสามารถลอยได้อย่างอิสระภายในเวิร์กชีตและกำหนดขนาดเพื่อให้มองเห็นได้ชัดเจนยิ่งขึ้น

**2. อัปเดตชื่อเรื่องและข้อความทางเลือก:**
```csharp
// ตั้งชื่อเรื่องและข้อความทางเลือก
slicer.Title = "Aspose";
slicer.AlternativeText = "Alternate Text";
```
ชื่อเรื่องให้บริบท ในขณะที่ข้อความทางเลือกช่วยให้เข้าถึงได้ง่ายขึ้น

**3. กำหนดค่าความสามารถในการพิมพ์และสถานะการล็อค:**
```csharp
// ตัดสินใจว่าเครื่องหั่นนั้นสามารถพิมพ์ได้หรือล็อคได้
slicer.IsPrintable = false;
slicer.IsLocked = false;
```
การตั้งค่าเหล่านี้จะควบคุมการมองเห็นตัวแบ่งส่วนในเอกสารที่พิมพ์และความสามารถในการแก้ไข

### การรีเฟรชเครื่องตัด

เพื่อให้แน่ใจว่าการเปลี่ยนแปลงทั้งหมดจะมีผล โปรดรีเฟรชตัวแบ่งส่วน:
```csharp
// รีเฟรชตัวแบ่งเพื่ออัปเดตมุมมอง
slicer.Refresh();
```

### การบันทึกสมุดงาน

สุดท้าย ให้บันทึกสมุดงานของคุณด้วยตัวแบ่งข้อมูลที่อัปเดตแล้ว:
```csharp
// บันทึกสมุดงานที่แก้ไข
workbook.Save("outputChangeSlicerProperties.xlsx", SaveFormat.Xlsx);
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่าการเปลี่ยนแปลงทั้งหมดจะถูกเก็บรักษาไว้ในไฟล์ใหม่
