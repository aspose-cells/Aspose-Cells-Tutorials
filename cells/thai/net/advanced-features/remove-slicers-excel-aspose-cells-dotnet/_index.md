## คู่มือการใช้งาน: การลบ Slicer

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อลบตัวแบ่งส่วนออกจากไฟล์ Excel:

### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

สร้างอินสแตนซ์ของ `Workbook` และโหลดไฟล์ Excel ของคุณที่มีตัวแบ่งส่วน:

```csharp
// กำหนดเส้นทางไดเร็กทอรีแหล่งที่มา
string sourceDir = RunExamples.Get_SourceDirectory();

// โหลดเวิร์กบุ๊กด้วยตัวแบ่งส่วน
Workbook wb = new Workbook(sourceDir + "sampleRemovingSlicer.xlsx");
```

### ขั้นตอนที่ 2: เข้าถึงแผ่นงาน

เข้าถึงเวิร์กชีตที่มีตัวแบ่งส่วนของคุณ สมมติว่ามันอยู่ในชีตแรก:

```csharp
// รับการอ้างอิงถึงแผ่นงานแรก
Worksheet ws = wb.Worksheets[0];
```

### ขั้นตอนที่ 3: ถอดเครื่องหั่นออก

ค้นหาและลบเครื่องหั่นที่ต้องการโดยใช้ดัชนีภายใน `Slicers` ของสะสม:

```csharp
// เข้าถึงเครื่องตัดชิ้นแรกในคอลเลกชั่น
Aspose.Cells.Slicers.Slicer slicer = ws.Slicers[0];

// นำเครื่องตัดออกจากแผ่นงาน
ws.Slicers.Remove(slicer);
```

### ขั้นตอนที่ 4: บันทึกสมุดงานของคุณ

บันทึกสมุดงานของคุณเพื่อเก็บการเปลี่ยนแปลงที่เกิดขึ้นโดยการลบตัวแบ่งส่วน:

```csharp
// กำหนดเส้นทางไดเรกทอรีเอาท์พุต
string outputDir = RunExamples.Get_OutputDirectory();

// บันทึกสมุดงานที่อัพเดต
wb.Save(outputDir + "outputRemovingSlicer.xlsx", SaveFormat.Xlsx);

Console.WriteLine("RemovingSlicer executed successfully.");
```
