## คู่มือการใช้งาน

ในหัวข้อนี้ เราจะสำรวจวิธีการดึงและจัดการไฮเปอร์ลิงก์โดยใช้ Aspose.Cells สำหรับ .NET

### การดึงไฮเปอร์ลิงก์จากช่วง

#### ภาพรวม

การดึงไฮเปอร์ลิงก์ภายในช่วง Excel ช่วยให้คุณสามารถวิเคราะห์หรือแก้ไขไฮเปอร์ลิงก์ได้โดยอัตโนมัติ ตัวอย่างนี้สาธิตการแยกไฮเปอร์ลิงก์จากเซลล์ A2 ถึง B3

#### ขั้นตอนการดำเนินการ

1. **ตั้งค่าเส้นทางไดเรกทอรี**
   กำหนดเส้นทางสำหรับไดเร็กทอรีแหล่งที่มาและเอาต์พุตของคุณ
   ```csharp
   string sourceDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
   string outputDir = RunExamples.Get_OutputDirectory();
   ```

2. **โหลดสมุดงาน**
   เปิดไฟล์ Excel ที่มีอยู่ซึ่งประกอบด้วยไฮเปอร์ลิงก์
   ```csharp
   Workbook workbook = new Workbook(sourceDir + "HyperlinksSample.xlsx");
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **สร้างช่วงและดึงไฮเปอร์ลิงก์**
   กำหนดช่วงเซลล์และแยกไฮเปอร์ลิงก์จากช่วงเซลล์นั้น
   ```csharp
   Range range = worksheet.Cells.CreateRange("A2", "B3");
   Hyperlink[] hyperlinks = range.Hyperlinks;
   
   foreach (Hyperlink link in hyperlinks)
   {
       Console.WriteLine(link.Area + " : " + link.Address);
       // ทางเลือก: ลบไฮเปอร์ลิงก์
       link.Delete();
   }
   ```

4. **บันทึกการเปลี่ยนแปลง**
   บันทึกสมุดงานที่มีการเปลี่ยนแปลงไปยังไฟล์ใหม่
   ```csharp
   workbook.Save(outputDir + "HyperlinksSample_out.xlsx");
   ```

### การลบไฮเปอร์ลิงก์

การ `Delete()` วิธีนี้ใช้เพื่อลบไฮเปอร์ลิงก์จากช่วงที่ระบุ เพื่อลดความซับซ้อนของกระบวนการล้างข้อมูล หรือเตรียมไฟล์สำหรับการวิเคราะห์เพิ่มเติมโดยไม่ต้องมีลิงก์ภายนอก

## การประยุกต์ใช้งานจริง

1. **การทำความสะอาดข้อมูล:** ทำให้การลบไฮเปอร์ลิงก์ที่ล้าสมัยหรือไม่เกี่ยวข้องจากรายงานทางการเงินเป็นระบบอัตโนมัติ
2. **การตรวจสอบการปฏิบัติตาม:** ตรวจสอบให้แน่ใจว่าไฮเปอร์ลิงก์ทั้งหมดสอดคล้องกับนโยบายขององค์กรก่อนที่จะแชร์เอกสารกับภายนอก
3. **การบูรณาการกับระบบ CRM:** แยกและจัดการข้อมูลที่เกี่ยวข้องกับลูกค้าที่เชื่อมโยงผ่านแผ่นงาน Excel
4. **เครื่องมือสร้างรายงานอัตโนมัติ:** ปรับปรุงเครื่องมือการรายงานด้วยการผสานรวมฟีเจอร์การจัดการไฮเปอร์ลิงก์แบบไดนามิก
