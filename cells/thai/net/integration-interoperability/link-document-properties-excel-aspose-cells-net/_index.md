## คู่มือการใช้งาน

### คุณสมบัติ: การตั้งค่าลิงก์คุณสมบัติเอกสาร

คุณลักษณะนี้สาธิตวิธีการเชื่อมโยงคุณสมบัติเอกสารแบบกำหนดเองในไฟล์ Excel กับช่วงเนื้อหาที่เฉพาะเจาะจง

#### ภาพรวม

การเชื่อมโยงคุณสมบัติของเอกสารช่วยให้คุณสร้างการอ้างอิงแบบไดนามิกภายในสเปรดชีตของคุณ ทำให้การจัดการข้อมูลเป็นไปอย่างสัญชาตญาณและอัตโนมัติมากขึ้น ซึ่งอาจมีประโยชน์โดยเฉพาะในการติดตามเจ้าของหรือเวอร์ชันของชุดข้อมูลโดยตรงจากเนื้อหา

#### การดำเนินการแบบทีละขั้นตอน

##### 1. กำหนดค่าไดเรกทอรี

กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุตที่ไฟล์ Excel ของคุณจะอยู่:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**คำอธิบาย**ควรแทนที่ตัวแทนเหล่านี้ด้วยเส้นทางจริงไปยังระบบไฟล์ของโครงการของคุณ

##### 2. โหลดสมุดงาน

สร้างตัวอย่าง `Workbook` วัตถุที่จะทำงานกับไฟล์ Excel ที่มีอยู่:

```csharp
Workbook workbook = new Workbook(SourceDir + "sample-document-properties.xlsx");
```

**วัตถุประสงค์**:การดำเนินการนี้จะโหลดเอกสาร Excel ของคุณเข้าสู่หน่วยความจำ ทำให้คุณสามารถจัดการคุณสมบัติและเนื้อหาผ่านโปรแกรมได้

##### 3. ดึงข้อมูลคุณสมบัติที่กำหนดเอง

เข้าถึงคอลเลกชันคุณสมบัติเอกสารที่กำหนดเองภายในเวิร์กบุ๊ก:

```csharp
CustomDocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```

**ฟังก์ชันการทำงาน**- `customProperties` ให้สิทธิ์เข้าถึงข้อมูลเมตาที่ผู้ใช้กำหนดทั้งหมดซึ่งเชื่อมโยงกับไฟล์ Excel ของคุณ

##### 4. เพิ่มลิงก์ไปยังเนื้อหา

เชื่อมโยงคุณสมบัติกับช่วงที่ระบุในเวิร์กชีตของคุณ:

```csharp
customProperties.AddLinkToContent("Owner", "MyRange");
```

**พารามิเตอร์**-
- `"Owner"`: ชื่อของคุณสมบัติเอกสารที่กำหนดเอง
- `"MyRange"`:การอ้างอิงเซลล์หรือช่วงที่คุณสมบัติเชื่อมโยงอยู่ภายใน

##### 5. ยืนยันลิงก์

ตรวจสอบว่าคุณสมบัติที่กำหนดเองเชื่อมโยงสำเร็จหรือไม่:

```csharp
DocumentProperty customProperty1 = customProperties["Owner"];
bool isLinkedToContent = customProperty1.IsLinkedToContent;
string source = customProperty1.Source; // เช่น "A1"
```

**การตรวจสอบ**- `isLinkedToContent` ยืนยันว่ามีการสร้างลิงค์แล้วและ `source` ช่วยให้คุณทราบการอ้างอิงเซลล์หรือช่วงที่แน่นอน

##### 6. บันทึกไฟล์ที่แก้ไข

สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณลงในไฟล์ใหม่:

```csharp
workbook.Save(outputDir + "out_sample-document-properties.xlsx");
```

**ความสำคัญ**ขั้นตอนนี้จะช่วยให้แน่ใจว่าการปรับเปลี่ยนทั้งหมดจะยังคงอยู่ในไฟล์ Excel เอาท์พุต

#### เคล็ดลับการแก้ไขปัญหา

- **ข้อผิดพลาดไม่พบไฟล์**: ตรวจสอบเส้นทางที่ระบุไว้ใน `SourceDir` ถูกต้องครับ.
- **การเชื่อมโยงความล้มเหลว**: ตรวจสอบให้แน่ใจว่าช่วงที่คุณกำลังลิงก์นั้นมีอยู่และตรงกับโครงสร้างของเวิร์กบุ๊กของคุณ

## การประยุกต์ใช้งานจริง

1. **การติดตามข้อมูล**:เชื่อมโยงคุณสมบัติเช่น "เจ้าของ" หรือ "อัปเดตครั้งล่าสุด" ไปยังเซลล์ที่มีข้อมูลเมตา ช่วยให้ตรวจสอบอัตโนมัติได้
2. **การควบคุมเวอร์ชัน**:ใช้คุณสมบัติเอกสารที่เชื่อมโยงเพื่อติดตามประวัติเวอร์ชันโดยตรงภายในช่วง Excel
3. **แดชบอร์ดแบบกำหนดเอง**:สร้างแดชบอร์ดแบบไดนามิกที่อัปเดตตามการเปลี่ยนแปลงในพื้นที่เนื้อหาที่เจาะจง

## การพิจารณาประสิทธิภาพ

- **การจัดการหน่วยความจำ**:เมื่อทำงานกับไฟล์ Excel ขนาดใหญ่ ให้แน่ใจว่าคุณกำจัดทิ้ง `Workbook` วัตถุอย่างเหมาะสมเพื่อปลดปล่อยทรัพยากร
