## คู่มือการใช้งาน

### การเพิ่มการอ้างอิงที่ลงทะเบียนไปยังโครงการ VBA
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มข้อมูลอ้างอิงที่ลงทะเบียน เช่น `stdole` และ `Office` ไลบรารีที่ใช้ Aspose.Cells สำหรับ .NET

#### ภาพรวม
การเพิ่มการอ้างอิงที่ลงทะเบียนไว้เป็นสิ่งสำคัญเมื่อแมโครของคุณขึ้นอยู่กับ DLL ภายนอก ซึ่งจะทำให้แน่ใจว่าส่วนประกอบทั้งหมดจะพร้อมใช้งานระหว่างการทำงาน

#### ขั้นตอนการดำเนินการ
1. **การเริ่มต้นสมุดงาน**
   สร้างอินสแตนซ์ใหม่ของ `Workbook` คลาสที่แสดงไฟล์ Excel:
   ```csharp
   Workbook workbook = new Workbook();
   ```
2. **เข้าถึงโครงการ VBA**
   ดึงข้อมูลโครงการ VBA ที่เชื่อมโยงกับสมุดงานของคุณ:
   ```csharp
   VbaProject vbaProj = workbook.VbaProject;
   ```
3. **เพิ่มข้อมูลอ้างอิงที่ลงทะเบียน**
   เพิ่มการอ้างอิงที่ลงทะเบียนแล้ว `stdole` และ `Office` ไลบรารีที่ใช้ GUID ของตนเอง
   - **สำหรับไลบรารี stdole:**
     ```csharp
     vbaProj.References.AddRegisteredReference(
         "stdole",
         "*\\G{00020430-0000-0000-C000-000000000046}#2.0#0#C:\\Windows\\system32\\stdole2.tlb#OLE Automation"
     );
     ```
   - **สำหรับห้องสมุดสำนักงาน:**
     ```csharp
     vbaProj.References.AddRegisteredReference(
         "Office",
         "*\\G{2DF8D04C-5BFA-101B-BDE5-00AA0044DE52}#2.0#0#C:\\Program Files\\Common Files\\Microsoft Shared\\OFFICE14\\MSO.DLL#Microsoft Office 14.0 Object Library"
     );
     ```
   แต่ละ `AddRegisteredReference` การโทรต้องการชื่อไลบรารี, GUID, หมายเลขเวอร์ชัน และเส้นทางไฟล์
4. **บันทึกสมุดงาน**
   บันทึกสมุดงานของคุณด้วยโครงการ VBA ที่อัปเดต:
   ```csharp
   string outputPath = "YOUR_OUTPUT_DIRECTORY" + "Output_out.xlsm";
   workbook.Save(outputPath);
   ```
#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบว่าเส้นทางไปยังไฟล์ DLL หรือ TLB ถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบให้แน่ใจว่า GUID ตรงกับเวอร์ชันไลบรารีของคุณ

## การประยุกต์ใช้งานจริง
การรวมการอ้างอิงที่ลงทะเบียนไว้ใน Excel อาจมีความสำคัญสำหรับ:
1. **การรายงานอัตโนมัติ:** ใช้ `Office` การอ้างอิงห้องสมุดสำหรับงานการสร้างรายงานขั้นสูง
2. **การวิเคราะห์ข้อมูลอัตโนมัติ:** ใช้ประโยชน์จากไลบรารีภายนอก เช่น `stdole` เพื่อจัดการข้อมูลภายในแมโคร
3. **การบูรณาการกับระบบภายนอก:** เชื่อมต่อเวิร์กบุ๊ก Excel กับซอฟต์แวร์อื่นได้อย่างราบรื่นผ่านส่วนประกอบ COM

แอปพลิเคชันเหล่านี้สามารถเพิ่มประสิทธิภาพการผลิตได้อย่างมากด้วยการทำให้การทำงานซ้ำๆ เป็นแบบอัตโนมัติ
