## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณในการเปิดไฟล์ CSV และการล้างข้อมูลโดยใช้ Aspose.Cells

### การเปิดไฟล์ CSV

#### ภาพรวม

Aspose.Cells ช่วยให้การเปิดไฟล์ CSV เป็นไปอย่างราบรื่น เราจะโหลดไฟล์ CSV ที่มีการกำหนดค่าแบบกำหนดเองเพื่อจัดการกับอักขระที่ไม่ถูกต้องอย่างมีประสิทธิภาพ

#### การดำเนินการแบบทีละขั้นตอน

1. **ตั้งค่าไดเรกทอรีแหล่งที่มา:**
   
   ```csharp
   string sourceDir = RunExamples.Get_SourceDirectory();
   var filename = sourceDir + "[20180220142533][ASPOSE_CELLS_TEST].csv";
   ```

2. **โหลด CSV พร้อมตัวเลือกที่กำหนดเอง:**
   
   ```csharp
   var workbook = new Workbook(filename, new TxtLoadOptions()
   {
       Separator = ';',
       LoadFilter = new LoadFilter(LoadDataFilterOptions.CellData),
       CheckExcelRestriction = false,
       ConvertNumericData = false,
       ConvertDateTimeData = false
   });
   ```

3. **แสดงข้อมูลแผ่นงาน:**
   
   ```csharp
   Console.WriteLine(workbook.Worksheets[0].Name);
   Console.WriteLine("CSV file opened successfully!");
   ```

**คำอธิบายพารามิเตอร์:**
- `Separator`: กำหนดตัวแบ่งที่ใช้ใน CSV ของคุณ
- `LoadFilter`: ระบุข้อมูลที่จะโหลด (เช่น CellData)
- `CheckExcelRestriction`: ช่วยให้สามารถจัดการไฟล์ที่มีขนาดใหญ่กว่าข้อจำกัดของ Excel ได้

### การแทนที่อักขระที่ไม่ถูกต้อง

หากต้องการแทนที่อักขระที่ไม่ถูกต้อง ให้แก้ไข TxtLoadOptions ของคุณหรือประมวลผลข้อมูลหลังการโหลด วิธีนี้จะช่วยให้มั่นใจได้ว่าชุดข้อมูลจะสะอาดสำหรับการประมวลผลเพิ่มเติม

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้อง
- ตรวจสอบรูปแบบและโครงสร้าง CSV ก่อนที่จะโหลด
