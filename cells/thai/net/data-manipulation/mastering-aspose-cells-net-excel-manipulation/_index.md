## คู่มือการใช้งาน

### การตั้งค่าไดเรกทอรี
#### ภาพรวม
การสร้างไดเร็กทอรีต้นทางและปลายทางเป็นสิ่งสำคัญสำหรับการจัดระเบียบไฟล์อย่างมีประสิทธิภาพ หัวข้อนี้จะแสดงวิธีการตั้งค่าไดเร็กทอรีเหล่านี้โดยใช้ C#

#### การดำเนินการแบบทีละขั้นตอน
**1. กำหนดเส้นทางไดเรกทอรี**
```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY"; // ระบุไดเร็กทอรีแหล่งที่มาของคุณที่นี่
string OutputDir = "YOUR_OUTPUT_DIRECTORY"; // ระบุไดเรกทอรีเอาท์พุตของคุณที่นี่
```

**2. ตรวจสอบและสร้างไดเรกทอรี**
```csharp
if (!Directory.Exists(SourceDir))
{
    Directory.CreateDirectory(SourceDir); // สร้างไดเรกทอรีแหล่งที่มาหากไม่มีอยู่
}

if (!Directory.Exists(OutputDir))
{
    Directory.CreateDirectory(OutputDir); // สร้างไดเรกทอรีเอาท์พุตถ้ายังไม่มีอยู่
}
```

### การสร้างและการจัดการเวิร์กบุ๊ก
#### ภาพรวม
เรียนรู้วิธีการสร้างเวิร์กบุ๊ก Excel เพิ่มเวิร์กชีต และจัดการรูปภาพโดยใช้ Aspose.Cells

**3. สร้างเวิร์กบุ๊กตัวอย่าง**
```csharp
using Aspose.Cells;

Workbook workbook = new Workbook();
```

**4. เพิ่มแผ่นงาน**
```csharp
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

**5. แทรกภาพลงในเวิร์กชีต**
ให้แน่ใจว่าคุณมีไฟล์รูปภาพชื่อ `logo.jpg` ในไดเร็กทอรีแหล่งที่มาของคุณ
```csharp
string imagePath = Path.Combine(SourceDir, "logo.jpg");
int pictureIndex = worksheet.Pictures.Add(5, 5, imagePath);
Aspose.Cells.Drawing.Picture picture = worksheet.Pictures[pictureIndex];

// จัดวางภาพให้อยู่ในสัดส่วนที่เหมาะสม
picture.UpperDeltaX = 200; // ออฟเซ็ตแนวนอน
picture.UpperDeltaY = 200; // ออฟเซ็ตแนวตั้ง
```

**6. บันทึกสมุดงาน**
```csharp
string outputPath = Path.Combine(OutputDir, "book1.out.xls");
workbook.Save(outputPath);
```
