## คู่มือการใช้งาน

ในส่วนนี้เราจะแนะนำคุณเกี่ยวกับการสร้างตัวกรองอัตโนมัติสำหรับไฟล์ Excel ทีละขั้นตอน

### การสร้างสมุดงานและการเข้าถึงแผ่นงาน

**ภาพรวม:**
เริ่มต้นด้วยการโหลดเวิร์กบุ๊ก Excel ของคุณและเข้าถึงเวิร์กชีตที่คุณจะใช้ตัวกรอง

**ขั้นตอนการดำเนินการ:**
1. **สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก:**
   ```csharp
   string dataDir = "your_directory_path";
   Workbook workbook = new Workbook(dataDir + "book1.xls");
   ```
2. **การเข้าถึงแผ่นงาน:**
   เข้าถึงแผ่นงานแรกในไฟล์ Excel ของคุณ:
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

### การใช้ตัวกรองอัตโนมัติ

**ภาพรวม:**
ตั้งค่าและใช้ตัวกรองอัตโนมัติกับช่วงเซลล์ที่ระบุ

**ขั้นตอนการดำเนินการ:**
3. **สร้างและใช้ตัวกรองอัตโนมัติ:**
   กำหนดช่วงเซลล์สำหรับตัวกรองของคุณ:
   ```csharp
   worksheet.AutoFilter.Range = "A1:B1";
   ```
4. **บันทึกสมุดงานของคุณ:**
   บันทึกการเปลี่ยนแปลงไปยังไฟล์ใหม่หลังจากใช้ตัวกรอง:
   ```csharp
   workbook.Save(dataDir + "output.out.xls");
   ```

3. **การจัดการสินค้าคงคลัง:** จัดการระดับสต๊อกอย่างมีประสิทธิภาพด้วยการใช้ตัวกรองเพื่อดูรายการสินค้าคงคลังที่ต่ำ
