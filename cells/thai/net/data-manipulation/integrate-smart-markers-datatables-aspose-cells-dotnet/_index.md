## คู่มือการใช้งาน

คู่มือนี้แบ่งออกเป็นสองคุณลักษณะหลัก: การสร้าง DataTable และการใช้เครื่องหมายอัจฉริยะสำหรับการประมวลผล Excel

### การสร้างและการเติมข้อมูลใน DataTable

ขั้นตอนแรกเกี่ยวข้องกับการตั้งค่า `DataTable`การเพิ่มคอลัมน์และการเติมข้อมูล หัวข้อนี้จะกล่าวถึงกระบวนการดังกล่าวโดยละเอียด

#### ภาพรวม
สร้างสิ่งง่ายๆ `DataTable` ชื่อว่า "MyDataSource" ซึ่งมีคอลัมน์เดียวสำหรับสูตรทดสอบ แต่ละแถวจะมีสตริงที่ต่อกันเพื่อแสดงให้เห็นการจัดการสตริงพื้นฐานใน C#

```csharp
using System;
using System.Data;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// สร้างอินสแตนซ์ DataTable
table dt = new DataTable();
dt.Columns.Add("TestFormula");

// เติมข้อมูลตัวอย่างลงใน DataTable
for (int i = 1; i <= 5; i++)
{
    DataRow dr = dt.NewRow();
    // เชื่อมโยงค่าสตริงกับการจัดรูปแบบสำหรับ Excel
    dr["TestFormula"] = $'="{i:00}-This " & "is " & "concatenation"';
    dt.Rows.Add(dr);
}
dt.TableName = "MyDataSource";
```

#### คำอธิบาย:
- **ตารางข้อมูล**:วิธีที่ยืดหยุ่นในการแสดงข้อมูลในหน่วยความจำ ใช้เป็นแหล่งข้อมูลสำหรับ Excel
- **การสอดแทรกและการเชื่อมต่อสตริง**:สาธิตด้วย `+=` ผู้ดำเนินการ เทคนิคนี้มีประโยชน์สำหรับการสร้างสตริงที่ซับซ้อน

### การสร้างสมุดงานและการประมวลผลมาร์กเกอร์อัจฉริยะ

คุณลักษณะที่สองมุ่งเน้นไปที่การรวม DataTable เข้าในเวิร์กบุ๊ก Excel โดยใช้ตัวระบุอัจฉริยะของ Aspose.Cells

#### ภาพรวม
สร้างเวิร์กบุ๊กใหม่ แทรกมาร์กเกอร์อัจฉริยะที่อ้างอิง DataTable ของเรา ตั้งค่าแหล่งข้อมูล ประมวลผล และบันทึกเอาต์พุตเป็นไฟล์ Excel

```csharp
using Aspose.Cells;

// สร้างอินสแตนซ์เวิร์กบุ๊กใหม่
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
ws.Cells["A1"].PutValue("&=MyDataSource.TestFormula(Formula)");

// ตั้งค่าแหล่งข้อมูลสำหรับการประมวลผลมาร์กเกอร์อัจฉริยะ
WorkbookDesigner wd = new WorkbookDesigner(wb);
wd.SetDataSource(dt);
wd.Process();

// บันทึกสมุดงานไปยังไฟล์ Excel
wb.Save(outputDir + "outputUsingFormulaParameterInSmartMarkerField.xlsx");
```

#### คำอธิบาย:
- **สมุดงานและใบงาน**:แสดงถึงไฟล์ Excel ทั้งหมดและแผ่นงานแต่ละแผ่นตามลำดับ
- **มาร์กเกอร์อัจฉริยะ**: สัญลักษณ์ เช่น `&=` ในค่าเซลล์ที่สั่ง Aspose.Cells เกี่ยวกับวิธีการประมวลผลข้อมูลจาก DataTable

## การประยุกต์ใช้งานจริง

ต่อไปนี้เป็นกรณีการใช้งานจริงบางส่วนสำหรับการบูรณาการเครื่องหมายอัจฉริยะกับ DataTables:
1. **การสร้างรายงานอัตโนมัติ**:สร้างรายงาน Excel โดยละเอียดจากแบบสอบถามฐานข้อมูลได้อย่างง่ายดาย
2. **การวิเคราะห์ข้อมูล**:ใช้สเปรดชีตที่สร้างแบบไดนามิกเพื่อวิเคราะห์และแสดงภาพเมตริกทางธุรกิจ
3. **การประมวลผลใบแจ้งหนี้**:ทำให้การสร้างใบแจ้งหนี้เป็นระบบอัตโนมัติโดยป้อนข้อมูลลงในเทมเพลตที่ออกแบบไว้ล่วงหน้า
