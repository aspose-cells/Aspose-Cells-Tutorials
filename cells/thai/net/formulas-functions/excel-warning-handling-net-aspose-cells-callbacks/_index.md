## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นสองฟีเจอร์หลัก: การตั้งค่าการโทรกลับคำเตือน และการโหลดไฟล์ Excel พร้อมการจัดการคำเตือน

### คุณสมบัติ 1: การแจ้งเตือนการโทรกลับ

**ภาพรวม**

ฟีเจอร์นี้เกี่ยวข้องกับการสร้างคลาสที่ใช้งาน `IWarningCallback` เพื่อสกัดกั้นคำเตือนในระหว่างการโหลดเวิร์กบุ๊ก โดยเฉพาะการจัดการชื่อที่กำหนดซ้ำกันหรือปัญหาอื่นๆ

#### ขั้นตอนที่ 1: นำ IWarningCallback Interface มาใช้

สร้างคลาสที่มีชื่อว่า `WarningCallback` ดังต่อไปนี้:
```csharp
using System;
using Aspose.Cells;

namespace AsposeCellsExamples
{
    private class คำเตือนการโทรกลับ : IWarningCallback
    {
        public void Warning(WarningInfo warningInfo)
        {
            if (warningInfo.WarningType == WarningType.DuplicateDefinedName)
            {
                Console.WriteLine("Duplicate Defined Name Warning: " + warningInfo.Description);
            }
        }
    } // WarningCallback
}
```
**คำอธิบาย**: เดอะ `Warning` วิธีการนี้จะจับและประมวลผลคำเตือน โดยจะตรวจสอบชื่อที่กำหนดซ้ำโดยเฉพาะ

### คุณสมบัติที่ 2: โหลดไฟล์ Excel พร้อมการจัดการคำเตือน

**ภาพรวม**

ในฟีเจอร์นี้ เราจะโหลดเวิร์กบุ๊ก Excel ในขณะที่ใช้คอลแบ็กคำเตือนแบบกำหนดเองเพื่อจัดการกับปัญหาใดๆ ที่เกิดขึ้น

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต

ตั้งค่าเส้นทางไดเร็กทอรีของคุณ:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";
```
ตรวจสอบให้แน่ใจว่าเส้นทางเหล่านี้ชี้ไปยังไดเร็กทอรีที่ถูกต้องบนระบบของคุณ

#### ขั้นตอนที่ 2: กำหนดค่า LoadOptions ด้วยการแจ้งเตือนการโทรกลับ

สร้าง `LoadOptions` และกำหนดการโทรกลับคำเตือน:
```csharp
LoadOptions options = new LoadOptions();
options.WarningCallback = new WarningCallback();
```

#### ขั้นตอนที่ 3: โหลดเวิร์กบุ๊กและบันทึกเอาท์พุต

สุดท้ายโหลดเวิร์กบุ๊กและบันทึกลงในไดเร็กทอรีที่คุณระบุ:
```csharp
Workbook book = new Workbook(SourceDir + "/sampleDuplicateDefinedName.xlsx", options);
book.Save(OutputDir + "/outputDuplicateDefinedName.xlsx");
```
**คำอธิบาย**:โค้ดนี้จะโหลดไฟล์ Excel พร้อมคำเตือนที่อาจเกิดขึ้นซึ่งจัดการโดยคอลแบ็กที่กำหนดเองของเรา จากนั้นจะบันทึกสมุดงานที่ประมวลผลแล้ว
