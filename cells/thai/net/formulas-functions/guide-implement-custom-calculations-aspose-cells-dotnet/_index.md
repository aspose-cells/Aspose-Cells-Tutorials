## คู่มือการใช้งาน

เราจะแบ่งคู่มือนี้ออกเป็นสองส่วนหลักเพื่อความชัดเจน

### คุณสมบัติ 1: เครื่องมือคำนวณแบบกำหนดเอง

คุณสมบัตินี้ช่วยให้คุณสามารถแทนที่ `Calculate` วิธีการที่มีตรรกะแบบกำหนดเองสำหรับสูตรเฉพาะ

#### ภาพรวม
การสร้างเครื่องคำนวณแบบกำหนดเองช่วยให้คุณสามารถผสานตรรกะเฉพาะธุรกิจเข้ากับการคำนวณ Excel ได้อย่างราบรื่น ซึ่งมีประโยชน์อย่างยิ่งเมื่อฟังก์ชันมาตรฐานไม่ตรงตามความต้องการของคุณ

#### ขั้นตอนการดำเนินการ
##### ขั้นตอนที่ 1: กำหนดเครื่องมือคำนวณแบบกำหนดเองของคุณ
สร้างคลาสที่สืบทอดมาจาก `AbstractCalculationEngine` และแทนที่ `Calculate` วิธี:
```csharp
using Aspose.Cells;

public class ICustomEngine : AbstractCalculationEngine
{
    public override void Calculate(CalculationData data)
    {
        if (data.FunctionName == "MyCompany.CustomFunction")
        {
            // ตรรกะที่กำหนดเองที่นี่: ตั้งค่าค่าที่คำนวณได้
            data.CalculatedValue = "Aspose.Cells.";
        }
    }
}
```
**คำอธิบาย:**
- `AbstractCalculationEngine`:คลาสฐานสำหรับเครื่องยนต์ที่กำหนดเอง
- `Calculate`:วิธีการที่คุณฉีดตรรกะแบบกำหนดเองของคุณ

##### ขั้นตอนที่ 2: ใช้ Custom Engine ในการคำนวณ
รวมเครื่องยนต์ที่กำหนดเองเข้ากับการคำนวณเวิร์กบุ๊กของคุณ:
```csharp
using System;
using Aspose.Cells;

public class ImplementDirectCalculationOfCustomFunction
{
    public static void Run()
    {
        Workbook wb = new Workbook();
        Worksheet ws = wb.Worksheets[0];
        ws.Cells["A1"].PutValue("Welcome to ");
        
        CalculationOptions opts = new CalculationOptions();
        opts.CustomEngine = new ICustomEngine();

        object ret = ws.CalculateFormula("=A1 & MyCompany.CustomFunction()", opts);
    }
}
```
**คำอธิบาย:**
- `CalculationOptions`: กำหนดค่าการตั้งค่าการคำนวณ รวมถึงเครื่องยนต์แบบกำหนดเอง
- `CalculateFormula`:ประเมินสูตรโดยใช้ตรรกะที่กำหนดเองของคุณ

### คุณสมบัติ 2: ใช้งานการคำนวณโดยตรงของฟังก์ชันที่กำหนดเอง

ฟีเจอร์นี้สาธิตวิธีการใช้เครื่องคำนวณแบบกำหนดเองเพื่อคำนวณสูตรโดยตรง

#### ภาพรวม
การประเมินสูตรโดยตรงด้วยฟังก์ชันที่กำหนดเองช่วยลดความซับซ้อนของการคำนวณและเพิ่มความยืดหยุ่นในการประมวลผลข้อมูลภายในสเปรดชีต
