## คู่มือการใช้งาน
เราจะเน้นที่การระบุภาษาของไฟล์ Excel โดยใช้คุณสมบัติเอกสารในตัว คุณสมบัตินี้ช่วยให้ผู้ใช้สามารถกำหนดภาษาหลักที่ใช้ในเอกสารของตนได้ เพื่อให้เข้าถึงและแปลเป็นภาษาท้องถิ่นได้ดียิ่งขึ้น

### ขั้นตอนที่ 1: สร้างวัตถุเวิร์กบุ๊ก
เริ่มต้นด้วยการสร้างวัตถุเวิร์กบุ๊กใหม่ซึ่งแสดงถึงไฟล์ Excel ของคุณ

```csharp
// เริ่มต้นไลบรารี Aspose.Cells
Workbook wb = new Workbook();
```

บรรทัดนี้จะตั้งค่าเวิร์กบุ๊กว่าง ซึ่งคุณสามารถเพิ่มข้อมูล แผ่นงาน หรือคุณสมบัติตามต้องการได้

### ขั้นตอนที่ 2: เข้าถึงคุณสมบัติเอกสารในตัว
หากต้องการเปลี่ยนการตั้งค่าภาษา ให้เข้าถึงคอลเลกชันคุณสมบัติเอกสารในตัวของเวิร์กบุ๊กของคุณ:

```csharp
// การเข้าถึงคุณสมบัติเอกสารในตัว
Aspose.Cells.Properties.BuiltInDocumentPropertyCollection bdpc = wb.BuiltInDocumentProperties;
```

ที่นี่, `bdpc` เป็นคอลเลกชันที่เก็บคุณสมบัติของเอกสารต่าง ๆ เช่น ชื่อผู้เขียน ชื่อเรื่อง และภาษา

### ขั้นตอนที่ 3: ตั้งค่าภาษา
ระบุภาษาที่ใช้ในไฟล์ Excel ของคุณ ซึ่งจะช่วยให้ผู้ใช้ที่ใช้โปรแกรมอ่านหน้าจอหรือเครื่องมือแปลเข้าใจเนื้อหาได้ดีขึ้น:

```csharp
// ตั้งค่าภาษาเป็นภาษาเยอรมันและภาษาฝรั่งเศส
bdpc.Language = "German, French";
```

ในขั้นตอนนี้ เรากำหนดทั้งภาษาเยอรมันและภาษาฝรั่งเศสเป็นภาษาหลักสำหรับเอกสารของเรา

### ขั้นตอนที่ 4: บันทึกสมุดงานของคุณ
สุดท้าย ให้บันทึกสมุดงานของคุณโดยมีคุณสมบัติเหล่านี้ วิธีนี้จะช่วยให้มั่นใจว่าการตั้งค่าทั้งหมดจะยังคงอยู่:

```csharp
// บันทึกสมุดงานไปยังเส้นทางที่ระบุ
wb.Save(outputDir + "outputSpecifyLanguageOfExcelFileUsingBuiltInDocumentProperties.xlsx", SaveFormat.Xlsx);
```

ขั้นตอนนี้จะเขียนการเปลี่ยนแปลงลงใน `.xlsx` ไฟล์พร้อมใช้งานหรือแจกจ่าย

## การประยุกต์ใช้งานจริง
การระบุภาษาของไฟล์ Excel มีการใช้งานจริงหลายประการ:

1. **องค์กรที่มีหลายภาษา**: อำนวยความสะดวกในการเข้าถึงเอกสารระหว่างภูมิภาคต่างๆ
2. **การปฏิบัติตามและการแปล**:ให้แน่ใจว่าเอกสารเป็นไปตามข้อกำหนดด้านภาษาของถิ่น
3. **การทำงานร่วมกัน**:ปรับปรุงการทำงานร่วมกันระหว่างทีมงานต่างประเทศด้วยการกำหนดการตั้งค่าภาษาอย่างชัดเจน

การรวมฟีเจอร์นี้เข้ากับระบบอื่นๆ จะช่วยปรับปรุงเวิร์กโฟลว์อัตโนมัติ เช่น ระบบการจัดการเอกสารหรือเครือข่ายการจัดส่งเนื้อหา
