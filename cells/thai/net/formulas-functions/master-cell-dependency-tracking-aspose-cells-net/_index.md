## คู่มือการใช้งาน

### การโหลดสมุดงาน

โหลดเวิร์กบุ๊กของคุณเพื่อกำหนดไฟล์ Excel:
```csharp
// โหลดสมุดงานที่มีอยู่จากเส้นทางที่ระบุ
Workbook workbook = new Workbook("Book1.xlsx");
```
#### ภาพรวม
นี่คือการเริ่มต้น `Workbook` วัตถุที่ให้การเข้าถึงแผ่นงานและเซลล์

### การเข้าถึงเซลล์และการติดตามการอ้างอิง
เลือกแผ่นงานและเซลล์สำหรับการติดตามการอ้างอิง:
```csharp
// รับแผ่นงานแรกในสมุดงาน
Worksheet worksheet = workbook.Worksheets[0];

// เข้าถึงเซลล์ที่ต้องการ
Cell targetCell = worksheet.Cells["B2"];
```
#### ภาพรวม
เข้าถึง `Cells` การรวบรวมแผ่นงานที่ระบุเพื่อระบุเซลล์เป้าหมาย

### การรับผู้พึ่งพา
ใช้ `GetDependents` วิธีการดึงข้อมูลเซลล์ที่ขึ้นต่อกัน:
```csharp
// รับเซลล์ที่ขึ้นอยู่กับ 'B2' ทั้งหมด
Cell[] dependents = targetCell.GetDependents(true);

foreach (Cell c in dependents)
{
    Console.WriteLine(c.Name); // แสดงชื่อของเซลล์ที่ขึ้นต่อกัน
}
```
#### ภาพรวม
`GetDependents(true)` การกลับมา `Cell` วัตถุที่ได้รับผลกระทบจากการเปลี่ยนแปลงในเซลล์ที่ระบุ

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาทั่วไป**: ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกต้องหากคุณพบข้อผิดพลาด "ไม่พบไฟล์"
