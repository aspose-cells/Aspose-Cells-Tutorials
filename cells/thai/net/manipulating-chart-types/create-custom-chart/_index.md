# สร้างแผนภูมิที่กำหนดเอง

## การแนะนำ

การสร้างแผนภูมิแบบกำหนดเองใน Excel โดยใช้ไลบรารี Aspose.Cells สำหรับ .NET ไม่เพียงแต่จะตรงไปตรงมาเท่านั้น แต่ยังเป็นวิธีที่ยอดเยี่ยมในการแสดงภาพข้อมูลของคุณอย่างมีประสิทธิภาพ แผนภูมิสามารถเปลี่ยนข้อมูลธรรมดาให้กลายเป็นเรื่องราวที่น่าสนใจ ทำให้ผู้วิเคราะห์และผู้ตัดสินใจสามารถค้นหาข้อมูลเชิงลึกได้ง่ายขึ้น ในบทช่วยสอนนี้ เราจะเจาะลึกถึงวิธีสร้างแผนภูมิแบบกำหนดเองภายในแอปพลิเคชันของคุณ ดังนั้น หากคุณกำลังมองหาวิธีปรับปรุงรายงานของคุณหรือเพียงแค่เพิ่มความสวยงามให้กับการนำเสนอข้อมูลของคุณ คุณมาถูกที่แล้ว!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเจาะลึกรายละเอียดในการสร้างแผนภูมิ เรามาตรวจสอบกันก่อนว่าคุณจัดเตรียมทุกอย่างเรียบร้อยแล้ว นี่คือสิ่งที่คุณต้องการ:

1. Visual Studio หรือ IDE ที่เข้ากันได้กับ .NET ใดๆ นี่จะเป็นพื้นที่เล่นของคุณในการเขียนและทดสอบโค้ดของคุณ
2.  Aspose.Cells สำหรับไลบรารี .NET: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารีนี้แล้ว คุณสามารถดาวน์โหลดได้[ที่นี่](https://releases.aspose.com/cells/net/).
3. ความเข้าใจพื้นฐานเกี่ยวกับ C#: จะเป็นประโยชน์สำหรับคุณหากเข้าใจแนวคิดพื้นฐานของ C# เนื่องจากเราจะใช้แนวคิดดังกล่าวในตัวอย่างโค้ดของเรา
4. ชุดข้อมูลตัวอย่าง: ในการสร้างแผนภูมิ การมีข้อมูลบางอย่างถือเป็นสิ่งสำคัญ เราจะใช้ชุดข้อมูลที่เรียบง่ายในตัวอย่างของเรา แต่คุณสามารถปรับเปลี่ยนให้เหมาะกับความต้องการของคุณได้

## แพ็คเกจนำเข้า

ในการเริ่มต้น คุณจะต้องนำเข้าเนมสเปซ Aspose.Cells ที่จำเป็นในแอปพลิเคชัน C# ของคุณ โดยคุณสามารถทำได้ดังนี้:

```csharp
using System;
using System.IO;

using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
using Aspose.Cells.Charts;
```

ตอนนี้โครงสร้างพื้นฐานก็วางโครงร่างไว้เรียบร้อยแล้ว มาดูคำแนะนำทีละขั้นตอนในการสร้างแผนภูมิแบบกำหนดเองกัน

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอาต์พุตของคุณ

ขั้นแรก คุณต้องสร้างไดเรกทอรีที่จะบันทึกไฟล์ Excel ขั้นตอนนี้มีความสำคัญเพื่อให้แน่ใจว่าแอปพลิเคชันของคุณทราบว่าควรวางผลิตภัณฑ์ขั้นสุดท้ายไว้ที่ใด

```csharp
// ไดเรกทอรีผลลัพธ์
string outputDir = "Your Output Directory"; // เปลี่ยนสิ่งนี้เป็นเส้นทางที่คุณต้องการ
```

แทนที่จะใช้ "ไดเรกทอรีผลลัพธ์ของคุณ" คุณสามารถระบุเส้นทางจริงที่คุณต้องการบันทึกไฟล์ Excel ได้ ตรวจสอบให้แน่ใจว่ามีไดเรกทอรีนี้อยู่ในระบบของคุณ มิฉะนั้น คุณจะพบกับข้อผิดพลาดในภายหลัง

## ขั้นตอนที่ 2: การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก

 ตอนนี้ คุณจะต้องการเริ่มต้นด้วยการสร้างอินสแตนซ์ใหม่ของ`Workbook`คลาส นี่คือองค์ประกอบพื้นฐานสำหรับการดำเนินการ Excel ใดๆ ที่ใช้ Aspose.Cells

```csharp
// การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก
Workbook workbook = new Workbook();
```

บรรทัดโค้ดนี้จะเริ่มต้นเวิร์กบุ๊กใหม่ และคุณก็พร้อมที่จะเริ่มเพิ่มข้อมูลและแผนภูมิแล้ว!

## ขั้นตอนที่ 3: การเข้าถึงแผ่นงาน

ขั้นตอนต่อไป คุณต้องได้รับข้อมูลอ้างอิงถึงเวิร์กชีตที่ข้อมูลของคุณจะอยู่ในนั้น ในกรณีนี้ เราจะทำงานกับเวิร์กชีตแรกในเวิร์กบุ๊ก

```csharp
// การรับข้อมูลอ้างอิงของแผ่นงานที่เพิ่มใหม่
Worksheet worksheet = workbook.Worksheets[0];
```

บรรทัดนี้จะเข้าถึงเวิร์กชีตแรก (ดัชนี 0) Aspose.Cells ช่วยให้คุณสามารถมีเวิร์กชีตได้หลายแผ่น ดังนั้นคุณจึงเลือกได้ตามความเหมาะสม

## ขั้นตอนที่ 4: การเพิ่มข้อมูลตัวอย่างลงในเวิร์กชีต


เมื่อเตรียมเวิร์กชีตเสร็จแล้ว ก็ได้เวลาเพิ่มข้อมูลตัวอย่างลงในเซลล์ของคุณ ชุดข้อมูลที่เรียบง่ายจะช่วยให้เราแสดงภาพผ่านแผนภูมิได้อย่างมีประสิทธิภาพมากขึ้น

```csharp
// การเพิ่มค่าตัวอย่างลงในเซลล์
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["A4"].PutValue(110);
worksheet.Cells["B1"].PutValue(260);
worksheet.Cells["B2"].PutValue(12);
worksheet.Cells["B3"].PutValue(50);
worksheet.Cells["B4"].PutValue(100);
```

ที่นี่เราใส่ค่าในช่วง A1 ถึง B4 คุณสามารถปรับเปลี่ยนค่าเหล่านี้เพื่อทดสอบสถานการณ์ข้อมูลที่แตกต่างกันได้

## ขั้นตอนที่ 5: การเพิ่มแผนภูมิลงในเวิร์กชีต

ตอนนี้เรามาถึงส่วนที่น่าตื่นเต้นแล้ว นั่นคือการเพิ่มแผนภูมิที่จะแสดงข้อมูลที่เราเพิ่งป้อนเข้าไป คุณสามารถเลือกประเภทแผนภูมิต่างๆ ที่มีใน Aspose.Cells ได้

```csharp
// การเพิ่มแผนภูมิลงในเวิร์กชีต
int chartIndex = worksheet.Charts.Add(ChartType.Column, 5, 0, 25, 10);
```

ในบรรทัดนี้ เราจะเพิ่มแผนภูมิคอลัมน์ คุณสามารถใช้แผนภูมิประเภทอื่น เช่น แผนภูมิเส้น แผนภูมิวงกลม หรือแผนภูมิแท่ง ตามความต้องการของคุณก็ได้

## ขั้นตอนที่ 6: การเข้าถึงอินสแตนซ์แผนภูมิ

เมื่อเราเพิ่มแผนภูมิแล้ว เราจำเป็นต้องอ้างอิงแผนภูมินั้นเพื่อให้สามารถปรับเปลี่ยนเพิ่มเติมได้ ดังนี้:

```csharp
// การเข้าถึงอินสแตนซ์ของแผนภูมิที่เพิ่มใหม่
Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];
```

 ณ จุดนี้คุณมี`chart` วัตถุที่ให้คุณปรับเปลี่ยนคุณสมบัติตามต้องการ

## ขั้นตอนที่ 7: การเพิ่มชุดข้อมูลลงในแผนภูมิ

ตอนนี้ คุณต้องแจ้งแผนภูมิว่าจะดึงข้อมูลจากที่ใด ซึ่งทำได้โดยการเพิ่มชุดข้อมูลใน Aspose.Cells

```csharp
// การเพิ่ม NSeries (แหล่งข้อมูลแผนภูมิ) ลงในแผนภูมิ
chart.NSeries.Add("A1:B4", true);
```

เส้นนี้เชื่อมต่อแผนภูมิของคุณกับจุดข้อมูลที่คุณวางไว้ในเซลล์ได้อย่างมีประสิทธิภาพ ช่วยให้แผนภูมิแสดงค่าเหล่านี้ได้

## ขั้นตอนที่ 8: การปรับแต่งประเภทซีรีส์

คุณสามารถปรับแต่งแผนภูมิของคุณเพิ่มเติมได้โดยการเปลี่ยนประเภทของชุดข้อมูลใดๆ ตัวอย่างเช่น ลองเปลี่ยนชุดข้อมูลที่สองเป็นแผนภูมิเส้นเพื่อให้มองเห็นได้ชัดเจนยิ่งขึ้น

```csharp
// การตั้งค่าประเภทแผนภูมิของ 2nd NSeries ให้แสดงเป็นแผนภูมิเส้น
chart.NSeries[1].Type = Aspose.Cells.Charts.ChartType.Line;
```

วิธีนี้ช่วยให้สามารถใช้แผนภูมิแบบผสมได้ ซึ่งให้โอกาสในการแสดงภาพที่ไม่ซ้ำใคร

## ขั้นตอนที่ 9: การบันทึกสมุดงาน

หลังจากกำหนดค่าทั้งหมดเสร็จแล้ว ก็ถึงเวลาบันทึกไฟล์ Excel ของคุณ วิธีดำเนินการมีดังนี้:

```csharp
// การบันทึกไฟล์ Excel
workbook.Save(outputDir + "outputHowToCreateCustomChart.xlsx");
```

 ตรวจสอบให้แน่ใจว่าคุณได้เพิ่มชื่อไฟล์ด้วย`.xlsx` ส่วนขยายเพื่อให้แน่ใจว่าสมุดงานได้รับการบันทึกอย่างถูกต้อง

## บทสรุป

และแล้วคุณก็จะได้มัน! คุณเพิ่งสร้างแผนภูมิแบบกำหนดเองโดยใช้ Aspose.Cells สำหรับ .NET ด้วยโค้ดเพียงไม่กี่บรรทัด คุณก็สามารถสร้างภาพข้อมูลได้อย่างมีประสิทธิภาพ ทำให้รายงานและการนำเสนอน่าสนใจยิ่งขึ้น 

โปรดจำไว้ว่าพลังของแผนภูมิอยู่ที่ความสามารถในการบอกเล่าเรื่องราว ทำให้สามารถเข้าใจข้อมูลที่ซับซ้อนได้ในทันที ดังนั้น ลองทดลองใช้ชุดข้อมูลและแผนภูมิประเภทต่างๆ แล้วปล่อยให้ข้อมูลของคุณทำหน้าที่บอกเล่าเรื่องราว!
