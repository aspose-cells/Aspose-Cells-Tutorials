## คู่มือการใช้งาน

เมื่อตั้งค่า Aspose.Cells สำหรับ .NET เรียบร้อยแล้ว เรามารีเฟรชตารางสรุปข้อมูลแบบซ้อนกันทีละขั้นตอนกัน ซึ่งเกี่ยวข้องกับการระบุและอัปเดตตารางสรุปข้อมูลย่อยภายในตารางหลัก

### โหลดไฟล์ Excel
เริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีอยู่ซึ่งมีตารางสรุปข้อมูลของคุณ:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook wb = new Workbook(sourceDir + "sampleFindAndRefreshNestedOrChildrenPivotTables.xlsx");
```

### เข้าถึงตารางสรุปข้อมูลในเวิร์กชีต
หากต้องการรีเฟรชตารางที่ซ้อนกัน ให้เข้าถึงเวิร์กชีตและค้นหาตารางสรุปข้อมูลหลัก:
```csharp
Worksheet ws = wb.Worksheets[0];
PivotTable ptParent = ws.PivotTables[2];  // ตัวอย่าง: การเข้าถึงตารางสรุปข้อมูลที่สาม
```

### รีเฟรชตารางสรุปข้อมูลย่อย
เมื่อระบุตารางสรุปข้อมูลของผู้ปกครองแล้ว ให้ดึงตารางสรุปข้อมูลของบุตรหลานมาและรีเฟรช:
```csharp
// รับตารางสรุปข้อมูลย่อยทั้งหมดของผู้ปกครอง
PivotTable[] ptChildren = ptParent.GetChildren();

// วนซ้ำผ่านตารางสรุปข้อมูลย่อยแต่ละตารางเพื่อรีเฟรช
foreach (var ptChild in ptChildren)
{
    ptChild.RefreshData();
    ptChild.CalculateData();  // รับประกันว่าข้อมูลที่อัปเดตได้รับการคำนวณ
}
```
#### คำอธิบาย
- **รับลูก()**: ดึงข้อมูลตารางสรุปแบบซ้อนกันทั้งหมดภายใต้ตารางหลัก
- **RefreshData() และคำนวณข้อมูล()**:อัปเดตและคำนวณข้อมูลใหม่ในตารางสรุปข้อมูลย่อยแต่ละตารางเพื่อให้แน่ใจถึงความถูกต้อง

### เคล็ดลับการแก้ไขปัญหา
หากเกิดปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้องเมื่อโหลดเวิร์กบุ๊ก
- ตรวจสอบว่าดัชนีตารางสรุปที่ระบุมีอยู่ในเวิร์กชีตของคุณ
