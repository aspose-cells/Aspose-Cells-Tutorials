## คู่มือการใช้งาน

ในส่วนนี้ เราจะอธิบายวิธีการใช้ตัวกรองขั้นสูงโดยใช้ Aspose.Cells เราจะแนะนำคุณตลอดขั้นตอนการตั้งค่าและการใช้งาน

### การโหลดสมุดงานของคุณ

เริ่มต้นด้วยการโหลดเวิร์กบุ๊ก Excel ของคุณลงใน `Aspose.Cells.Workbook` วัตถุ:

```csharp
// ระบุแหล่งที่มาของไดเร็กทอรี
string sourceDir = RunExamples.Get_SourceDirectory();

// โหลดเวิร์กบุ๊กจากไฟล์
Workbook wb = new Workbook(sourceDir + "sampleAdvancedFilter.xlsx");
```

### การเข้าถึงและการกรองข้อมูล

ขั้นตอนต่อไป ให้เข้าถึงแผ่นงานที่คุณต้องการใช้ตัวกรอง เราจะใช้ `AdvancedFilter` วิธีการระบุเกณฑ์การกรอง

```csharp
// เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
Worksheet ws = wb.Worksheets[0];

// ใช้ตัวกรองขั้นสูงกับช่วง A5:D19 โดยมีเกณฑ์ที่ระบุใน A1:D2
// ตัวกรองจะถูกนำมาใช้ และจะรวมบันทึกทั้งหมดไว้ด้วย (ไม่ใช่แค่เฉพาะรายการเดียว)
ws.AdvancedFilter(true, "A5:D19", "A1:D2", "", false);
```

#### คำอธิบายพารามิเตอร์:

- **อยู่ในสถานที่**: ตั้งค่าเป็น `true` เพื่อกรองข้อมูลภายในช่วงเดิม
- **รายการช่วง**: ช่วงเป้าหมายที่คุณต้องการใช้ตัวกรอง (`"A5:D19"` ในตัวอย่างของเรา)
- **เกณฑ์ช่วง**: กำหนดเกณฑ์การกรอง (`"A1:D2"` ที่นี่).
- **ชื่อสำเนาแผ่นงาน**: ชื่อชีตใหม่หากกรองนอกตำแหน่ง (ปล่อยว่างไว้หากกรองในตำแหน่งเดิม)
- **มีเอกลักษณ์`: Set to `false` คือการรวมทุกรายการ ไม่ใช่เฉพาะรายการที่ไม่ซ้ำกันเท่านั้น

### การบันทึกสมุดงานของคุณ

หลังจากใช้ตัวกรองแล้ว ให้บันทึกสมุดงาน:

```csharp
// ระบุไดเรกทอรีเอาท์พุตและบันทึกสมุดงาน
string outputDir = RunExamples.Get_OutputDirectory();
wb.Save(outputDir + "outputAdvancedFilter.xlsx", SaveFormat.Xlsx);

Console.WriteLine("ApplyAdvancedFilterOfMicrosoftExcel executed successfully.\r\n");
```
