# วิธีการใช้ AutoFilter ใน Excel โดยใช้ Aspose.Cells สำหรับ .NET

## การแนะนำ

คุณกำลังมองหาวิธีเพิ่มประสิทธิภาพการวิเคราะห์ข้อมูลโดยการกรองแถวในไฟล์ Excel ด้วยโปรแกรมหรือไม่ ด้วยเครื่องมืออันทรงพลัง **Aspose.Cells สำหรับ .NET** ในไลบรารี คุณสามารถจัดการสมุดงานและใช้ตัวกรองอัตโนมัติได้อย่างง่ายดาย บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าสภาพแวดล้อม การเริ่มต้นสมุดงาน การเข้าถึงแผ่นงาน การสร้างตัวกรองอัตโนมัติแบบกำหนดเอง และการรีเฟรชเพื่อบันทึกการเปลี่ยนแปลง

### สิ่งที่คุณจะได้เรียนรู้:
- วิธีการติดตั้ง Aspose.Cells สำหรับ .NET
- การเริ่มต้นวัตถุเวิร์กบุ๊กจากไฟล์ Excel
- การเข้าถึงแผ่นงานเฉพาะในสมุดงาน
- การใช้งานและการประยุกต์ใช้ตัวกรองอัตโนมัติแบบกำหนดเอง
- การรีเฟรชฟิลเตอร์และการบันทึกสมุดงานที่อัปเดต

ก่อนที่จะเจาะลึกขั้นตอนต่างๆ เรามาตรวจสอบก่อนว่าคุณมีทุกสิ่งที่คุณต้องการแล้ว

- **Aspose.Cells สำหรับ .NET** ห้องสมุดที่ติดตั้งอยู่ในโครงการของคุณ

## การตั้งค่า Aspose.Cells สำหรับ .NET

### การติดตั้ง

คุณสามารถเพิ่มแพ็กเกจ Aspose.Cells ลงในโปรเจ็กต์ของคุณได้โดยใช้ **ตัวจัดการแพ็กเกจ NuGet** หรือว่า **.NET CLI**-

**.NET CLI**
```bash
dotnet add package Aspose.Cells
```

**คอนโซลตัวจัดการแพ็คเกจ**
```powershell
PM> NuGet\Install-Package Aspose.Cells
```

### การขอใบอนุญาต

Aspose.Cells สำหรับ .NET นำเสนอใบอนุญาตทดลองใช้งานฟรี ใบอนุญาตชั่วคราว และตัวเลือกการซื้อ:

- **ทดลองใช้งานฟรี**:ดาวน์โหลดไลบรารีเพื่อทดสอบความสามารถทั้งหมดโดยไม่มีข้อจำกัด
- **ใบอนุญาตชั่วคราว**:ขอใบอนุญาตชั่วคราวเพื่อประเมินผลระยะสั้นได้ที่เว็บไซต์ของตน
- **ซื้อ**:หากต้องการใช้ในระยะยาว ควรพิจารณาซื้อใบอนุญาต

### การเริ่มต้นขั้นพื้นฐาน

เมื่อติดตั้งแล้วให้เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Workbook` ชั้นเรียนและโหลดไฟล์ Excel ของคุณ:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";

// โหลดเวิร์กบุ๊กจากไดเร็กทอรีแหล่งที่มาที่ระบุพร้อมข้อมูลตัวอย่าง
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
```

## คู่มือการใช้งาน

### 1. การเริ่มต้นและการเปิดสมุดงาน

#### ภาพรวม
หัวข้อนี้จะกล่าวถึงวิธีการโหลดไฟล์ Excel ลงใน `Workbook` วัตถุที่ใช้ Aspose.Cells

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// โหลดเวิร์กบุ๊กจากไดเร็กทอรีแหล่งที่มาที่ระบุพร้อมข้อมูลตัวอย่าง
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
```

**คำอธิบาย**: เดอะ `Workbook` คลาสนี้แสดงถึงไฟล์ Excel ทั้งหมด โดยการระบุเส้นทาง คุณสามารถโหลดไฟล์ที่มีอยู่เพื่อการจัดการได้

### 2. การเข้าถึงเวิร์กชีตในเวิร์กบุ๊ก

#### ภาพรวม
เข้าถึงเวิร์กชีตแต่ละรายการภายในเวิร์กบุ๊กของคุณเพื่อใช้การดำเนินการเฉพาะ เช่น การกรอง

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// โหลดเวิร์กบุ๊กจากไดเร็กทอรีแหล่งที่มา
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");

// เข้าถึงเวิร์กชีตแรกโดยใช้ดัชนี
Worksheet worksheet = workbook.Worksheets[0];
```

**คำอธิบาย**: เดอะ `Worksheets` คอลเลกชันช่วยให้คุณเข้าถึงแผ่นงานแต่ละแผ่นได้ ดัชนี 0 สอดคล้องกับแผ่นงานแรก

### 3. การสร้างและการใช้ตัวกรองอัตโนมัติ

#### ภาพรวม
ตั้งค่าตัวกรองอัตโนมัติสำหรับช่วงเซลล์ที่ระบุ โดยใช้เกณฑ์ที่กำหนดเองเพื่อแสดงข้อมูลที่เกี่ยวข้อง

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// โหลดสมุดงานและเข้าถึงแผ่นงานแรก
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
Worksheet worksheet = workbook.Worksheets[0];

// กำหนดช่วงสำหรับตัวกรองอัตโนมัติ (เช่น A1:A18)
worksheet.AutoFilter.Range = "A1:A18";

// ใช้ตัวกรองแบบกำหนดเองเพื่อแสดงแถวที่มีค่าเริ่มต้นด้วย 'Ba'
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");
```

**คำอธิบาย**: เดอะ `AutoFilter` คุณสมบัตินี้ช่วยให้สามารถกำหนดช่วงและใช้ตัวกรองได้ สามารถใช้เมธอดแบบกำหนดเองเพื่อระบุเงื่อนไขได้

### 4. การรีเฟรชและบันทึกสมุดงาน

#### ภาพรวม
รีเฟรชตัวกรองของคุณเพื่อใช้การเปลี่ยนแปลงและบันทึกเวิร์กบุ๊กไปยังตำแหน่งไฟล์ใหม่

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// โหลดเวิร์กบุ๊ก เข้าถึงเวิร์กชีต และตั้งค่าตัวกรองอัตโนมัติ
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
Worksheet worksheet = workbook.Worksheets[0];
worksheet.AutoFilter.Range = "A1:A18";
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");

// รีเฟรชตัวกรองอัตโนมัติเพื่อใช้การเปลี่ยนแปลง
worksheet.AutoFilter.Refresh();

// บันทึกสมุดงานที่อัพเดตไปยังไดเร็กทอรีเอาท์พุตที่ระบุ
workbook.Save(outputDir + "/outSourceSampleCountryNames.xlsx");
```

**คำอธิบาย**: หลังจากใช้ฟิลเตอร์แล้ว ให้ใช้ `Refresh()` เพื่ออัปเดตแผ่นงาน สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณด้วย `Save()` วิธี.

## การประยุกต์ใช้งานจริง

1. **การรายงานข้อมูล**:กรองข้อมูลโดยอัตโนมัติสำหรับรายงานที่รวมเฉพาะประเทศหรือภูมิภาคที่เจาะจงเท่านั้น
2. **การจัดการสินค้าคงคลัง**:กรองรายการสินค้าตามชื่อสินค้าหรือหมวดหมู่ที่ขึ้นต้นด้วยตัวอักษรบางตัว
3. **การวิเคราะห์ทางการเงิน**:ใช้ตัวกรองอัตโนมัติเพื่อเน้นไปที่บันทึกทางการเงินที่ตรงตามเกณฑ์บางประการ เช่น ธุรกรรมที่ขึ้นต้นด้วยชื่อผู้ขายที่เจาะจง
