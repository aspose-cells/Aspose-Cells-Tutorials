# เรียนรู้การสร้างตารางสรุปข้อมูล Excel ด้วย Aspose.Cells ใน .NET: โหลด เรียงลำดับ และบันทึก

## การแนะนำ
คุณกำลังประสบปัญหาในการจัดการข้อมูลที่ซับซ้อนใน Excel หรือไม่? ใช้ Aspose.Cells สำหรับ .NET เพื่อจัดการงานวิเคราะห์ข้อมูลของคุณโดยอัตโนมัติและคล่องตัว บทช่วยสอนนี้เหมาะอย่างยิ่งสำหรับนักพัฒนาที่ต้องการปรับปรุงแอปพลิเคชันหรือผู้วิเคราะห์ธุรกิจที่ต้องการข้อมูลเชิงลึกที่แม่นยำ เรียนรู้การโหลดเวิร์กบุ๊ก กำหนดค่าฟีเจอร์ PivotTable ขั้นสูง เช่น ผลรวมรวมและผลรวมย่อย การเรียงลำดับอัตโนมัติ และการบันทึกการเปลี่ยนแปลง

**สิ่งที่คุณจะได้เรียนรู้:**
- โหลดและเข้าถึง Excel PivotTables ด้วย Aspose.Cells
- ตั้งค่าผลรวมรวมและผลรวมย่อยแบบแถวสำหรับการสรุปข้อมูลที่ได้รับการปรับปรุง
- กำหนดค่าตัวเลือกการเรียงลำดับอัตโนมัติและการแสดงข้อมูลอัตโนมัติเพื่อการแสดงข้อมูลที่ดีขึ้น
- บันทึกการปรับเปลี่ยนอย่างมีประสิทธิภาพกลับสู่ดิสก์

มาเจาะลึกฟังก์ชันอันทรงพลังเหล่านี้กันดีกว่า!

## คู่มือการใช้งาน
สำรวจคุณสมบัติแต่ละอย่างโดยละเอียดด้านล่าง

### โหลดและเข้าถึง PivotTable
#### ภาพรวม
การเข้าถึง PivotTable เป็นสิ่งสำคัญสำหรับการจัดการข้อมูล ต่อไปนี้เป็นวิธีการโหลดไฟล์ Excel และเรียกค้น PivotTable ที่ต้องการ

#### ทีละขั้นตอน
**1. โหลดเวิร์กบุ๊ก:**
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Pivot;
   
   string sourceDir = "YOUR_SOURCE_DIRECTORY";
   Workbook workbook = new Workbook(sourceDir + "Book1.xls");
   ```
**2. เข้าถึงเวิร์กชีตและตารางสรุปข้อมูล:**
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   int pivotIndex = 0;
   PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
   ```
### ตั้งค่าแถวยอดรวมทั้งหมดและยอดรวมย่อย
#### ภาพรวม
การกำหนดค่าผลรวมรวมและผลรวมย่อยแบบแถวช่วยให้สรุปข้อมูลได้อย่างมีประสิทธิภาพ

#### ทีละขั้นตอน
**1. เข้าถึงฟิลด์แถว:**
   ```csharp
   PivotFieldCollection pivotFields = pivotTable.RowFields;
   PivotField pivotField = pivotFields[0];
   ```
**2. กำหนดค่าผลรวมและผลรวมย่อย:**
   ```csharp
   // เปิดใช้งานผลรวมทั้งหมด
   pivotTable.RowGrand = true;

   // ตั้งค่าผลรวมย่อยสำหรับผลรวมและจำนวน
   pivotField.SetSubtotals(PivotFieldSubtotalType.Sum, true);
   pivotField.SetSubtotals(PivotFieldSubtotalType.Count, true);
   ```
### กำหนดค่าตัวเลือกการเรียงลำดับอัตโนมัติ
#### ภาพรวม
การเรียงลำดับอัตโนมัติจะจัดระเบียบข้อมูลอย่างไดนามิก ต่อไปนี้เป็นวิธีการกำหนดค่าฟีเจอร์นี้

#### ทีละขั้นตอน
**1. เปิดใช้งานการเรียงลำดับอัตโนมัติ:**
   ```csharp
   PivotField pivotField = pivotTable.RowFields[0];
   pivotField.IsAutoSort = true;
   pivotField.IsAscendSort = true; // ตั้งค่าลำดับการเรียงให้เรียงจากน้อยไปมาก
   ```
**2. กำหนดดัชนีฟิลด์การเรียงลำดับ:**
   ```csharp
   pivotField.AutoSortField = -5;
   ```
### กำหนดค่าตัวเลือกการแสดงอัตโนมัติ
#### ภาพรวม
คุณสมบัติการแสดงอัตโนมัติจะแสดงเฉพาะข้อมูลที่เกี่ยวข้องโดยอัตโนมัติ

#### ทีละขั้นตอน
**1. เปิดใช้งานการตั้งค่าการแสดงอัตโนมัติ:**
   ```csharp
   PivotField pivotField = pivotTable.RowFields[0];
   pivotField.IsAutoShow = true;
   ```
**2. กำหนดค่าเงื่อนไขการแสดง:**
   ```csharp
   pivotField.AutoShowField = 0; // โดยอิงตามดัชนีฟิลด์ข้อมูลเฉพาะ
   ```
### บันทึกไฟล์ Excel
#### ภาพรวม
หลังจากทำการเปลี่ยนแปลงแล้ว ให้บันทึกสมุดงานของคุณกลับลงในดิสก์

#### ทีละขั้นตอน
**1. บันทึกสมุดงาน:**
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "output.xls");
   ```
## การประยุกต์ใช้งานจริง
การใช้ Aspose.Cells อย่างเชี่ยวชาญในการสร้าง PivotTable มีประโยชน์ต่อสถานการณ์ต่างๆ มากมาย:

1. **การรายงานทางการเงิน:** สร้างรายงานรายไตรมาสอัตโนมัติเพื่อสรุปสุขภาพทางการเงิน
2. **การจัดการสินค้าคงคลัง:** จัดเรียงและกรองข้อมูลสต๊อกเพื่อระบุสินค้าที่มีสต๊อกต่ำ
3. **การวิเคราะห์การขาย:** เน้นผลิตภัณฑ์หรือภูมิภาคที่มีประสิทธิภาพสูงสุดโดยใช้การเรียงลำดับอัตโนมัติและผลรวมย่อย
4. **การวิเคราะห์ทรัพยากรบุคคล:** จัดทำสรุปผลการปฏิบัติงานของพนักงานตามแผนกหรือบทบาท

## การพิจารณาประสิทธิภาพ
มั่นใจถึงประสิทธิภาพที่เหมาะสมที่สุดด้วย Aspose.Cells:
- **การจัดการหน่วยความจำ:** กำจัดทิ้ง `Workbook` วัตถุเมื่อทำเพื่อปลดปล่อยทรัพยากร
- **การจัดการข้อมูลอย่างมีประสิทธิภาพ:** ประมวลผลเฉพาะฟิลด์ข้อมูลที่จำเป็นเพื่อลดเวลาในการโหลด
- **การประมวลผลแบบแบตช์:** หากทำงานกับไฟล์หลายไฟล์ ควรประมวลผลเป็นชุดแทนที่จะประมวลผลแบบต่อเนื่อง
