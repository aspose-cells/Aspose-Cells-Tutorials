## คู่มือการใช้งาน
หัวข้อนี้จะครอบคลุมการตั้งค่าความเข้ากันได้ของตารางสรุปข้อมูลใน .NET โดยใช้ Aspose.Cells
### การโหลดไฟล์ Excel และการเข้าถึงแผ่นงาน
โหลดไฟล์ Excel ที่มีอยู่ซึ่งประกอบด้วยตารางสรุปข้อมูลตัวอย่าง:
```csharp
// โหลดไฟล์ Excel ต้นฉบับที่มีตารางสรุปตัวอย่าง
Workbook wb = new Workbook("sample-pivot-table.xlsx");

// เข้าถึงเวิร์กชีตแรกที่มีข้อมูลตารางสรุปข้อมูล
Worksheet dataSheet = wb.Worksheets[0];
```
### การแก้ไขข้อมูลเซลล์
เมื่อคุณเข้าถึงเวิร์กชีตของคุณได้แล้ว ให้แก้ไขข้อมูลเซลล์ รวมถึงตั้งค่าสตริงยาว:
```csharp
Cells cells = dataSheet.Cells;
Cell cell = cells["B3"];
string longStr = "Very long text 1. very long text 2... End of text.";
cell.PutValue(longStr);

Console.WriteLine("Length of original data string: " + cell.StringValue.Length);
```
### การจัดการความเข้ากันได้ของตารางสรุปข้อมูล
เข้าถึงและปรับเปลี่ยนการตั้งค่าความเข้ากันได้ของตารางสรุปข้อมูล:
```csharp
// เข้าถึงเวิร์กชีตที่สองที่มีตารางสรุปข้อมูล
Worksheet pivotSheet = wb.Worksheets[1];
PivotTable pivotTable = pivotSheet.PivotTables[0];

// ตั้งค่าความเข้ากันได้กับ Excel 2003
pivotTable.IsExcel2003Compatible = true;
pivotTable.RefreshData();
pivotTable.CalculateData();

Cell b5 = pivotSheet.Cells["B5"];
Console.WriteLine("Length of cell B5 after setting IsExcel2003Compatible to True: " + b5.StringValue.Length);

// เปลี่ยนการตั้งค่าความเข้ากันได้และรีเฟรช
pivotTable.IsExcel2003Compatible = false;
pivotTable.RefreshData();
pivotTable.CalculateData();
b5 = pivotSheet.Cells["B5"];
Console.WriteLine("Length of cell B5 after setting IsExcel2003Compatible to False: " + b5.StringValue.Length);
```
### การปรับรูปแบบเซลล์
ปรับความสูงของแถวและความกว้างของคอลัมน์เพื่อให้มองเห็นได้ชัดเจนยิ่งขึ้น:
```csharp
pivotSheet.Cells.SetRowHeight(b5.Row, 100);
pivotSheet.Cells.SetColumnWidth(b5.Column, 65);

Style st = b5.GetStyle();
st.IsTextWrapped = true;
b5.SetStyle(st);

// บันทึกสมุดงานที่แก้ไขแล้ว
wb.Save("SpecifyCompatibility_out.xlsx", SaveFormat.Xlsx);
```

- ตรวจสอบการตั้งค่าความเข้ากันได้ของตารางสรุปข้อมูลหากพบปัญหาการตัดทอนข้อมูล
- ตรวจสอบการกำหนดค่ารูปแบบเซลล์อีกครั้งเพื่อหาปัญหาการห่อข้อความ

3. **การวิเคราะห์ข้อมูลอัตโนมัติ**:ใช้ตารางสรุปข้อมูลเพื่อสรุปชุดข้อมูลขนาดใหญ่โดยการใช้โปรแกรม
