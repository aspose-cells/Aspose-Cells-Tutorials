# จัดรูปแบบ PivotTables อัตโนมัติใน Excel ด้วย Aspose.Cells สำหรับ .NET

## การแนะนำ

เพิ่มความน่าสนใจให้กับรายงาน Excel ของคุณโดยเรียนรู้การจัดรูปแบบอัตโนมัติสำหรับ PivotTables โดยใช้ Aspose.Cells สำหรับ .NET คู่มือนี้จะช่วยให้คุณจัดรูปแบบงานโดยอัตโนมัติอย่างมีประสิทธิภาพ ทำให้การนำเสนอข้อมูลของคุณอ่านง่ายและดูเป็นมืออาชีพมากขึ้น

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า Aspose.Cells สำหรับ .NET
- การโหลดสมุดงานอย่างง่ายดาย
- การเข้าถึงแผ่นงานและตารางสรุปข้อมูล
- การใช้ตัวเลือกการจัดรูปแบบอัตโนมัติกับ PivotTables
- การบันทึกไฟล์ Excel ที่ถูกปรับเปลี่ยน

## ข้อกำหนดเบื้องต้น
ก่อนที่จะเริ่มต้น ให้แน่ใจว่าคุณมี:
- **ห้องสมุดที่จำเป็น**: Aspose.Cells สำหรับ .NET (เวอร์ชันที่เข้ากันได้)
- **การตั้งค่าสภาพแวดล้อม**:สภาพแวดล้อมการทำงานของ .NET พร้อมความรู้ C#
- **ข้อกำหนดเบื้องต้นของความรู้**:ความเข้าใจพื้นฐานเกี่ยวกับการพัฒนา .NET และการจัดการแพ็กเกจ NuGet

## การตั้งค่า Aspose.Cells สำหรับ .NET
ในการใช้ Aspose.Cells ในโปรเจ็กต์ของคุณ ให้ติดตั้งไลบรารีผ่าน:

**.NET CLI:**
```bash
dotnet add package Aspose.Cells
```

**คอนโซลตัวจัดการแพ็คเกจ:**
```powershell
PM> NuGet\Install-Package Aspose.Cells
```

### การขอใบอนุญาต
หากต้องการใช้งานฟังก์ชันเต็มรูปแบบหลังจากช่วงทดลองใช้งาน โปรดซื้อใบอนุญาตจากเว็บไซต์ของ Aspose หรือขอใบอนุญาตชั่วคราวเพื่อใช้ทดสอบ

## คู่มือการใช้งาน

### การโหลดสมุดงาน Excel
เริ่มต้นด้วยการโหลดเวิร์กบุ๊กที่คุณต้องการใช้การจัดรูปแบบอัตโนมัติ:
1. **ระบุไดเรกทอรีแหล่งที่มา:**
   ```csharp
   string sourceDir = "YOUR_SOURCE_DIRECTORY";
   ```
2. **โหลดสมุดงาน:**
   ```csharp
   string dataDir = Path.Combine(sourceDir, "Book1.xls");
   Workbook workbook = new Workbook(dataDir);
   ```

### การเข้าถึงเวิร์กชีตและตารางสรุปข้อมูล
เข้าถึงเวิร์กชีตเฉพาะและ PivotTables:
1. **เข้าถึงแผ่นงานที่ต้องการ:**
   ```csharp
   int pivotIndex = 0;
   Worksheet worksheet = workbook.Worksheets[pivotIndex];
   ```
2. **ดึงข้อมูล PivotTable:**
   ```csharp
   PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
   ```

### การจัดรูปแบบตารางสรุปข้อมูลอัตโนมัติ
ปรับปรุงรูปลักษณ์ด้วยการจัดรูปแบบอัตโนมัติ:
1. **เปิดใช้งานการจัดรูปแบบอัตโนมัติ:**
   ```csharp
   pivotTable.IsAutoFormat = true;
   ```
2. **ตั้งค่ารูปแบบอัตโนมัติ:**
   ```csharp
   pivotTable.AutoFormatType = PivotTableAutoFormatType.Report5;
   ```

### บันทึกสมุดงาน
รักษาการเปลี่ยนแปลงโดยบันทึกสมุดงานที่แก้ไข:
1. **กำหนดไดเรกทอรีเอาท์พุต:**
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```
2. **บันทึกไฟล์ที่แก้ไข:**
   ```csharp
   string outputFilePath = Path.Combine(outputDir, "output.xls");
   workbook.Save(outputFilePath);
   ```
