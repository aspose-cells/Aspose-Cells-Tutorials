## คู่มือการใช้งาน

เราจะแบ่งการใช้งานนี้ออกเป็นส่วนๆ ตามตรรกะเพื่อให้แน่ใจว่าเข้าใจคุณลักษณะแต่ละอย่างได้อย่างราบรื่น

### การใช้ตัวกรองอัตโนมัติในไฟล์ Excel โดยใช้ C#

#### ภาพรวม
หัวข้อนี้เน้นที่การโหลดไฟล์ Excel และการใช้ตัวกรองอัตโนมัติ จากนั้นเราจะดึงดัชนีของแถวที่ถูกซ่อนไว้หลังจากรีเฟรชตัวกรอง

#### ขั้นตอน

**ขั้นตอนที่ 1: โหลดไฟล์ Excel**

```csharp
// กำหนดไดเรกทอรีแหล่งที่มาของคุณและโหลดไฟล์ตัวอย่าง Excel
string sourceDir = "PathToYourDirectory\\";
Workbook wb = new Workbook(sourceDir + "sampleGetAllHiddenRowsIndicesAfterRefreshingAutoFilter.xlsx");
```

- **คำอธิบาย**: ที่นี่เราจะเริ่มต้นการ `Workbook` วัตถุที่มีเส้นทางไปยังไฟล์ Excel ตัวอย่างของเรา

**ขั้นตอนที่ 2: เข้าถึงและใช้ตัวกรองอัตโนมัติ**

```csharp
// เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
Worksheet ws = wb.Worksheets[0];

// ใช้ตัวกรองอัตโนมัติกับดัชนีคอลัมน์ 0 (คอลัมน์แรก)
ws.AutoFilter.AddFilter(0, "Orange");
```

- **คำอธิบาย**:เรากำลังเข้าถึงเวิร์กชีตแรกและใช้ตัวกรองเพื่อแสดงเฉพาะแถวที่คอลัมน์แรกมี "สีส้ม"

**ขั้นตอนที่ 3: รีเฟรชตัวกรองอัตโนมัติและดึงข้อมูลแถวที่ซ่อนอยู่**

```csharp
// รีเฟรชตัวกรองอัตโนมัติและรับดัชนีของแถวที่ซ่อนอยู่
int[] rowIndices = ws.AutoFilter.Refresh(true);

Console.WriteLine("Printing Rows Indices, Cell Names, and Values Hidden By AutoFilter.");
```

- **คำอธิบาย**: เดอะ `Refresh(true)` วิธีการอัปเดตตัวกรองและส่งคืนอาร์เรย์ของดัชนีแถวที่ถูกซ่อนเนื่องจากตัวกรอง

**ขั้นตอนที่ 4: พิมพ์รายละเอียดแถวที่ซ่อนไว้**

```csharp
for (int i = 0; i < rowIndices.Length; i++)
{
    int r = rowIndices[i];
    Cell cell = ws.Cells[r, 0];
    Console.WriteLine($"{r}\t{cell.Name}\t{cell.StringValue}");
}
```

- **คำอธิบาย**:วนซ้ำผ่านดัชนีแถวที่ซ่อนอยู่และพิมพ์รายละเอียดเช่นดัชนีแถว ชื่อเซลล์ และค่า

### การประยุกต์ใช้งานจริง

การกรองข้อมูลด้วยโปรแกรมสามารถใช้ได้ในสถานการณ์ต่างๆ ดังนี้:

1. **การล้างข้อมูล**:กรองแถวที่ไม่ต้องการออกโดยอัตโนมัติตามเกณฑ์ที่เฉพาะเจาะจง
2. **การสร้างรายงาน**:สร้างรายงานแบบไดนามิกโดยการกรองชุดข้อมูลก่อนการวิเคราะห์
3. **การบูรณาการกับตรรกะทางธุรกิจ**:ใช้ข้อมูลที่กรองแล้วเพื่อขับเคลื่อนการตัดสินใจทางธุรกิจหรือบูรณาการกับระบบอื่น ๆ เช่นซอฟต์แวร์ CRM

เมื่อทำงานกับไฟล์ Excel ขนาดใหญ่ ควรพิจารณาแนวทางปฏิบัติที่ดีที่สุดดังต่อไปนี้:

- **การประมวลผลแบบแบตช์**:ดำเนินการเป็นแถวแบบชุดหากสามารถทำได้เพื่อลดการใช้ทรัพยากรให้เหลือน้อยที่สุด
- **การกรองที่มีประสิทธิภาพ**:ใช้ตัวกรองเฉพาะเมื่อจำเป็นและจำกัดขอบเขตให้เฉพาะกับคอลัมน์ที่เกี่ยวข้อง
