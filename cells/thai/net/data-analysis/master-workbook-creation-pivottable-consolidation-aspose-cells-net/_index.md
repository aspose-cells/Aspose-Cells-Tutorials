# เรียนรู้การสร้างสมุดงานและการรวม PivotTable ด้วย Aspose.Cells .NET สำหรับการวิเคราะห์ข้อมูล

ปลดล็อกศักยภาพของ Aspose.Cells .NET โดยการสร้างเวิร์กบุ๊กจากไฟล์ Excel ที่มีอยู่และใช้ฟังก์ชันการรวมข้อมูลที่มีประสิทธิภาพ เช่น Average และ DistinctCount คู่มือที่ครอบคลุมนี้จะพาคุณผ่านแต่ละขั้นตอน เพื่อเพิ่มพูนทักษะการจัดการข้อมูลของคุณในสภาพแวดล้อม .NET

## การแนะนำ

ในโลกธุรกิจที่เปลี่ยนแปลงอย่างรวดเร็วในปัจจุบัน การจัดการและวิเคราะห์ชุดข้อมูลขนาดใหญ่ใน Excel อย่างมีประสิทธิภาพถือเป็นสิ่งสำคัญ ไม่ว่าจะเป็นการสร้างรายงานใหม่จากไฟล์ที่มีอยู่หรือการสรุปข้อมูลที่ซับซ้อนด้วย PivotTable การเชี่ยวชาญงานเหล่านี้สามารถปรับปรุงเวิร์กโฟลว์ให้มีประสิทธิภาพได้อย่างมาก บทช่วยสอนนี้จะเจาะลึกคุณลักษณะสำคัญสองประการของ Aspose.Cells .NET ได้แก่ การสร้างเวิร์กบุ๊กและการใช้ฟังก์ชันการรวมข้อมูลใน PivotTables

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีการสร้างเวิร์กบุ๊กจากไฟล์ Excel ที่มีอยู่โดยใช้ Aspose.Cells สำหรับ .NET
- การเข้าถึงแผ่นงานภายในสมุดงานที่สร้างขึ้น
- การใช้ฟังก์ชัน Average และ DistinctCount ในเขตข้อมูล PivotTable

## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นสองส่วนหลัก: การสร้างเวิร์กบุ๊กและการใช้ฟังก์ชันการรวม PivotTable

### คุณลักษณะที่ 1: การสร้างสมุดงานและการเข้าถึงแผ่นงาน

#### ภาพรวม
การสร้างเวิร์กบุ๊กจากไฟล์ Excel ที่มีอยู่ถือเป็นสิ่งสำคัญสำหรับการสร้างรายงานแบบอัตโนมัติ ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ที่มีอยู่ เข้าถึงเวิร์กชีต และบันทึกการเปลี่ยนแปลงได้อย่างมีประสิทธิภาพ

**การดำเนินการทีละขั้นตอน:**

##### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์
เริ่มต้นด้วยการตั้งค่าไดเร็กทอรีแหล่งที่มาที่ไฟล์ Excel ของคุณอยู่และไดเร็กทอรีเอาต์พุตสำหรับบันทึกการเปลี่ยนแปลง

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// เส้นทางไปยังไฟล์ Excel ต้นฉบับ
string filePath = Path.Combine(SourceDir, "Book.xlsx");
```

##### ขั้นตอนที่ 2: โหลดสมุดงานและเข้าถึงแผ่นงาน
โหลดเวิร์กบุ๊กที่มีอยู่และเข้าถึงเวิร์กชีตแรก

```csharp
// โหลดสมุดงานที่มีอยู่จากไฟล์ที่ระบุ
Workbook workbook = new Workbook(filePath);

// เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
Worksheet worksheet = workbook.Worksheets[0];
```

##### ขั้นตอนที่ 3: บันทึกการเปลี่ยนแปลงไปยังไฟล์ใหม่
หลังจากทำการปรับเปลี่ยนใดๆ แล้ว ให้บันทึกเวิร์กบุ๊กไปยังไฟล์ Excel ใหม่

```csharp
// บันทึกการเปลี่ยนแปลงไปยังไฟล์ใหม่
string outputFilePath = Path.Combine(OutputDir, "output.xlsx");
workbook.Save(outputFilePath);
```

### คุณลักษณะที่ 2: ฟังก์ชันการรวม PivotTable

#### ภาพรวม
PivotTables เป็นเครื่องมือที่มีประสิทธิภาพในการสรุปข้อมูล การใช้ฟังก์ชันเช่น Average และ DistinctCount สามารถเพิ่มความสามารถในการวิเคราะห์ข้อมูลของคุณได้

**การดำเนินการทีละขั้นตอน:**

##### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊กด้วย PivotTable
เริ่มต้นด้วยการโหลดเวิร์กบุ๊กที่มี PivotTable ของคุณ

```csharp
string filePath = Path.Combine(SourceDir, "Book.xlsx");
Workbook workbook = new Workbook(filePath);
Worksheet worksheet = workbook.Worksheets[0];
```

##### ขั้นตอนที่ 2: เข้าถึงและกำหนดค่า PivotTable
เข้าถึง PivotTable แรกในเวิร์กชีต และใช้ฟังก์ชันการรวมกับเขตข้อมูล

```csharp
PivotTable pivotTable = worksheet.PivotTables[0];

// ใช้ฟังก์ชันค่าเฉลี่ยกับฟิลด์ข้อมูลแรก
pivotTable.DataFields[0].Function = ConsolidationFunction.Average;

// ใช้ฟังก์ชัน DistinctCount กับฟิลด์ข้อมูลที่สอง
pivotTable.DataFields[1].Function = ConsolidationFunction.DistinctCount;
```

##### ขั้นตอนที่ 3: คำนวณและบันทึกการเปลี่ยนแปลง
ตรวจสอบให้แน่ใจว่าการเปลี่ยนแปลงได้รับการคำนวณและบันทึกแล้ว

```csharp
pivotTable.CalculateData();
string outputFilePath = Path.Combine(OutputDir, "output.xlsx");
workbook.Save(outputFilePath);
```
