# เรียนรู้การใช้งาน Subtotal และการควบคุมโครงร่างด้วย Aspose.Cells .NET

## การแนะนำ

การสรุปชุดข้อมูลขนาดใหญ่ได้อย่างมีประสิทธิภาพเป็นความท้าทายทั่วไปสำหรับผู้ใช้ Excel จำนวนมาก **Aspose.Cells สำหรับ .NET**การทำให้แอปพลิเคชันผลรวมย่อยและการควบคุมโครงร่างเป็นอัตโนมัติกลายเป็นเรื่องง่ายดาย ไม่ว่าคุณจะกำลังเตรียมรายงานทางการเงินหรือจัดการรายการสินค้าคงคลัง การเชี่ยวชาญฟังก์ชันเหล่านี้สามารถปรับปรุงความสามารถในการจัดการข้อมูลของคุณได้อย่างมาก

ในบทช่วยสอนนี้ เราจะมาเรียนรู้วิธีการใช้ผลรวมย่อยโดยใช้ฟังก์ชันการรวมเฉพาะกับ Aspose.Cells สำหรับ .NET และสาธิตการควบคุมตำแหน่งของแถวสรุป คุณจะได้เรียนรู้สิ่งต่อไปนี้:
- วิธีตั้งค่า Aspose.Cells ในโครงการ .NET ของคุณ
- กระบวนการนำผลรวมย่อยไปใช้และควบคุมโครงร่างทิศทางในไฟล์ Excel
- ตัวเลือกการกำหนดค่าที่สำคัญเพื่อปรับแต่งการนำเสนอข้อมูลของคุณ

ก่อนที่เราจะเริ่ม ให้แน่ใจว่าคุณได้ครอบคลุมข้อกำหนดเบื้องต้นที่จำเป็นแล้ว

## คู่มือการใช้งาน

มาแบ่งกระบวนการออกเป็นส่วนๆ ที่จัดการได้เพื่อใช้ผลรวมย่อยและควบคุมทิศทางโครงร่างกัน

### ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊กและเวิร์กชีต

ขั้นแรก ให้สร้างอินสแตนซ์ของ `Workbook` โดยการโหลดไฟล์ Excel และเข้าถึงเวิร์กชีตแรกของไฟล์:

```csharp
// สร้างสมุดงานจากไฟล์ Excel ต้นฉบับ
Workbook workbook = new Workbook(sourceDir + "sampleApplyingSubtotalChangeSummaryDirection.xlsx");

// เข้าถึงแผ่นงานแรก
Worksheet worksheet = workbook.Worksheets[0];
```

### ขั้นตอนที่ 2: กำหนดพื้นที่เซลล์สำหรับผลรวมย่อย

ระบุช่วงเซลล์ที่คุณต้องการใช้ผลรวมย่อย ที่นี่ เราจะระบุ `A2-B11`:

```csharp
// รับคอลเลกชันเซลล์ในเวิร์กชีตแรก
Cells cells = worksheet.Cells;

// สร้างพื้นที่เซลล์ เช่น A2:B11
CellArea ca = CellArea.CreateCellArea("A2", "B11");
```

### ขั้นตอนที่ 3: ใช้ผลรวมย่อย

การใช้ประโยชน์จาก `Subtotal` วิธีการใช้ผลรวมย่อยโดยระบุคอลัมน์และฟังก์ชันการรวม:

```csharp
// ใช้ผลรวมย่อยกับฟังก์ชัน Sum ในคอลัมน์ B
cells.Subtotal(ca, 0, ConsolidationFunction.Sum, new int[] { 1 }, true, false, true);
```
- **ฟังก์ชั่นการรวม**: กำหนดการดำเนินการ (เช่น Sum)
- **ดัชนีคอลัมน์**: ระบุคอลัมน์ที่จะรวม

### ขั้นตอนที่ 4: กำหนดทิศทางโครงร่าง

ควบคุมว่าแถวสรุปจะปรากฏที่ใดด้วย `SummaryRowBelow` คุณสมบัติ:

```csharp
// กำหนดทิศทางการสรุปโครงร่าง
worksheet.Outline.SummaryRowBelow = true;
```

การตั้งค่านี้จะช่วยให้แน่ใจว่าแถวสรุปจะอยู่ในตำแหน่งใต้รายการกลุ่ม ซึ่งจะช่วยเพิ่มการอ่านได้ง่าย

### ขั้นตอนที่ 5: บันทึกการเปลี่ยนแปลง

สุดท้าย ให้บันทึกสมุดงานที่คุณแก้ไขไปยังไฟล์ใหม่:

```csharp
// บันทึกไฟล์ Excel
workbook.Save(outputDir + "outputApplyingSubtotalChangeSummaryDirection.xlsx");
```

## การพิจารณาประสิทธิภาพ

เพื่อให้มั่นใจถึงประสิทธิภาพที่เหมาะสมที่สุด:
- ประมวลผลเฉพาะช่วงเซลล์ที่จำเป็นเมื่อใช้ผลรวมย่อย
- จัดการหน่วยความจำอย่างมีประสิทธิภาพด้วยการปล่อยทรัพยากรที่ไม่ได้ใช้งานในแอปพลิเคชัน .NET โดยใช้ `Dispose` วิธีการตามความเหมาะสม
- สำหรับชุดข้อมูลขนาดใหญ่ ควรพิจารณาแบ่งข้อมูลออกเป็นส่วนย่อยๆ หากเป็นไปได้
