# วิธีกำหนดรูปแบบตาราง Excel ใน HTML โดยใช้ Aspose.Cells .NET

## การแนะนำ

การแปลงข้อมูล Excel ให้เป็นรูปแบบที่เป็นมิตรกับเว็บจะช่วยเพิ่มการเข้าถึงและการใช้งาน บทช่วยสอนนี้สาธิตวิธีจัดรูปแบบตาราง Excel เมื่อแปลงเป็น HTML โดยใช้ Aspose.Cells สำหรับ .NET ซึ่งจะเปลี่ยนชีตคงที่ให้กลายเป็นเนื้อหาเว็บที่น่าสนใจ

**สิ่งที่คุณจะได้เรียนรู้:**
- การกำหนดรูปแบบเซลล์ตาราง Excel ด้วยคุณสมบัติ CSS เฉพาะ

- โดยใช้ `HtmlSaveOptions` สำหรับการจัดแต่งทรงขั้นสูง

## คู่มือการใช้งาน

### การจัดรูปแบบตาราง Excel
สร้างวัตถุเวิร์กบุ๊กเพื่อบรรจุข้อมูล Excel ของคุณ:
```csharp
// สร้างอินสแตนซ์สมุดงาน
Workbook wb = new Workbook();
```
เข้าถึงเวิร์กชีตแรกและกำหนดรูปแบบเซลล์ของมัน:
```csharp
// เข้าถึงแผ่นงานแรก
Worksheet ws = wb.Worksheets[0];

// เพิ่มข้อความลงในเซลล์ B5
Cell cell = ws.Cells["B5"];
cell.PutValue("This is some text.");

// สไตล์เซลล์ - เปลี่ยนสีตัวอักษรเป็นสีแดง
Style st = cell.GetStyle();
st.Font.Color = Color.Red;
cell.SetStyle(st);
```
### บันทึกเป็น HTML ด้วย CSS ที่กำหนดเอง
ใช้ `HtmlSaveOptions` เพื่อระบุรูปแบบที่กำหนดเอง:
```csharp
// กำหนดค่า HtmlSaveOptions และระบุรหัส CSS ของตาราง
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.TableCssId = "MyTest_TableCssId";

// บันทึกสมุดงานเป็นไฟล์ HTML ที่มีตารางแบบมีสไตล์
wb.Save("outputTableCssId.html", opts);
```
