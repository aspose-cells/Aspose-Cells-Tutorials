## คู่มือการใช้งาน

ในส่วนนี้ เราจะแยกย่อยกระบวนการการใช้งานการจัดรูปแบบตามเงื่อนไขโดยใช้แถบเฉียงย้อนกลับ

### การสร้างสมุดงานและแผ่นงานใหม่

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Workbook` และเข้าถึงแผ่นงานแรก:

```csharp
using Aspose.Cells;

// สร้างสมุดงานใหม่
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

### การเพิ่มการจัดรูปแบบตามเงื่อนไข

#### ขั้นตอนที่ 1: กำหนดช่วงรูปแบบ

ระบุช่วงที่คุณต้องการใช้การจัดรูปแบบตามเงื่อนไข:

```csharp
CellArea ca = new CellArea { StartRow = 0, EndRow = 5, StartColumn = 0, EndColumn = 3 };
```

#### ขั้นตอนที่ 2: ตั้งค่ากฎการจัดรูปแบบตามเงื่อนไข

เพิ่มกฎการจัดรูปแบบตามเงื่อนไขใหม่โดยใช้ `FormatConditionType` และระบุประเภทเงื่อนไข:

```csharp
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcs = sheet.ConditionalFormattings[index];
fcs.AddArea(ca);

// กำหนดเงื่อนไข (เช่น ค่าระหว่าง 50 ถึง 100)
int conditionIndex = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "50", "100");
```

#### ขั้นตอนที่ 3: ใช้รูปแบบแถบทแยงมุมย้อนกลับ

กำหนดค่ารูปแบบเพื่อรวมรูปแบบแถบทแยงกลับด้านพร้อมสีพื้นหน้าและพื้นหลังที่เฉพาะเจาะจง:

```csharp
FormatCondition fc = fcs[conditionIndex];
fc.Style.Pattern = BackgroundType.ReverseDiagonalStripe;
fc.Style.ForegroundColor = Color.FromArgb(255, 255, 0); // สีเหลือง
fc.Style.BackgroundColor = Color.FromArgb(0, 255, 255); // ไซยาน
```
