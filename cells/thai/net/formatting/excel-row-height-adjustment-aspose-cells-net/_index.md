### คู่มือการใช้งาน
เราจะแนะนำคุณปรับความสูงของแถวทีละขั้นตอน

#### ภาพรวมการปรับความสูงของแถว
การปรับความสูงของแถวจะช่วยเพิ่มการมองเห็นและการนำเสนอข้อมูล โดยเฉพาะเมื่อเนื้อหาแตกต่างกันในแต่ละเซลล์

##### ขั้นตอนที่ 1: เปิดสมุดงานของคุณ
โหลดไฟล์ Excel ของคุณลงใน `Workbook` วัตถุที่ใช้สตรีมไฟล์
```csharp
using System.IO;
using Aspose.Cells;

namespace AsposeCellsExamples
{
    public class SettingHeightOfRowExample
    {
        public static void Run()
        {
            // กำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ
            string dataDir = "path_to_your_directory";
            
            // เปิดสตรีมไฟล์สำหรับเอกสาร Excel ของคุณ
            using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
            {
                // สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กด้วยสตรีมไฟล์ที่เปิดอยู่
                Workbook workbook = new Workbook(fstream);

                // เข้าถึงและปรับเปลี่ยนแผ่นงาน...
            }
        }
    }
}
```

##### ขั้นตอนที่ 2: เข้าถึงแผ่นงาน
เข้าถึงแผ่นงานเฉพาะที่คุณต้องการปรับความสูงของแถว
```csharp
// การเข้าถึงเวิร์กชีตแรกในไฟล์ Excel
Worksheet worksheet = workbook.Worksheets[0];
```

##### ขั้นตอนที่ 3: ตั้งค่าความสูงของแถว
ใช้ `SetRowHeight` วิธีการเปลี่ยนความสูงของแถวที่ต้องการ ในที่นี้ เรากำหนดความสูงของแถวที่ 2 เป็น 13 จุด
```csharp
// ตั้งค่าความสูงของแถวที่ 2 (ดัชนี 1) เป็น 13 จุด
worksheet.Cells.SetRowHeight(1, 13);
```

##### ขั้นตอนที่ 4: บันทึกสมุดงานของคุณ
หลังจากทำการเปลี่ยนแปลงแล้ว ให้บันทึกสมุดงานของคุณกลับไปยังไฟล์หรือสตรีมตามต้องการ
```csharp
// การบันทึกไฟล์ Excel ที่แก้ไขแล้ว
workbook.Save(dataDir + "output.out.xls");
```

การปรับความสูงของแถวมีประโยชน์ในสถานการณ์ต่างๆ ดังนี้:
1. **รายงานทางการเงิน**:จัดตำแหน่งข้อความให้เหมาะสมเพื่อให้สามารถอ่านได้ดีขึ้น
2. **รายการสินค้าคงเหลือ**:ให้แน่ใจว่าชื่อและคำอธิบายผลิตภัณฑ์เหมาะสม
3. **ข้อมูลวิชาการ**:จัดระเบียบข้อมูลนักเรียนให้สอดคล้องกันในแต่ละแถว

คุณสามารถรวมฟังก์ชันนี้เข้ากับระบบอื่นๆ เช่น ฐานข้อมูลหรือเว็บเซอร์วิส เพื่อปรับความสูงของแถวแบบไดนามิกตามรายการข้อมูล
