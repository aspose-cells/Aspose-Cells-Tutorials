# วิธีการใช้ข้อความขีดฆ่าใน Excel โดยใช้ Aspose.Cells .NET กับ C#

ในโลกปัจจุบันที่ข้อมูลเป็นปัจจัยสำคัญ การปรับแต่งไฟล์ Excel ด้วยโปรแกรมสามารถประหยัดเวลาและเพิ่มประสิทธิภาพการทำงานได้ ไม่ว่าคุณจะกำลังเตรียมรายงานทางการเงินหรือทำเครื่องหมายข้อมูลที่ล้าสมัย การขีดฆ่าข้อความเป็นวิธีที่มีประสิทธิภาพในการสื่อสารการเปลี่ยนแปลงสถานะในรูปแบบภาพ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้เอฟเฟกต์การขีดฆ่าใน Excel โดยใช้ Aspose.Cells สำหรับ .NET กับ C# ด้วยการใช้ประโยชน์จากไลบรารีอันทรงพลังนี้ คุณจะได้รับความยืดหยุ่นในการสร้างระบบอัตโนมัติและปรับแต่งเอกสาร Excel ของคุณอย่างมีประสิทธิภาพ

**สิ่งที่คุณจะได้เรียนรู้:**

พร้อมที่จะเพิ่มพูนทักษะการจัดการ Excel ของคุณหรือยัง มาเจาะลึกข้อกำหนดเบื้องต้นกันก่อน

## คู่มือการใช้งาน

ตอนนี้คุณได้จัดเตรียมเครื่องมือที่จำเป็นเรียบร้อยแล้ว มาสำรวจวิธีการใช้เอฟเฟกต์การขีดฆ่าโดยใช้ C# กัน

### ขั้นตอนที่ 1: สร้างและกำหนดค่าเวิร์กบุ๊ก

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Workbook` คลาส นี่แสดงถึงไฟล์ Excel ของคุณ:

```csharp
// การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก
Workbook workbook = new Workbook();
```

### ขั้นตอนที่ 2: เพิ่มแผ่นงาน

เพิ่มเวิร์กชีตใหม่ลงในเวิร์กบุ๊กของคุณ ซึ่งคุณจะใช้เอฟเฟกต์การขีดฆ่า:

```csharp
// การเพิ่มเวิร์กชีตใหม่ลงในวัตถุ Excel
int i = workbook.Worksheets.Add();
```

### ขั้นตอนที่ 3: เข้าถึงเซลล์และตั้งค่า

เข้าถึงเซลล์ที่ต้องการในเวิร์กชีตนี้และตั้งค่าค่า:

```csharp
// การรับข้อมูลอ้างอิงของเวิร์กชีตที่เพิ่มใหม่โดยส่งดัชนีชีตของมัน
Worksheet worksheet = workbook.Worksheets[i];
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
cell.PutValue("Hello Aspose!");
```

### ขั้นตอนที่ 4: ใช้เอฟเฟกต์การขีดฆ่า

ดึงข้อมูลและปรับเปลี่ยนรูปแบบเพื่อใช้เอฟเฟกต์การขีดฆ่า:

```csharp
// การได้รับสไตล์ของเซลล์
Style style = cell.GetStyle();
style.Font.IsStrikeout = true; // การกำหนดเอฟเฟกต์การขีดฆ่าบนแบบอักษร
cell.SetStyle(style); // การนำรูปแบบไปใช้กับเซลล์
```

### ขั้นตอนที่ 5: บันทึกสมุดงาน

สุดท้าย ให้บันทึกสมุดงานของคุณด้วยการเปลี่ยนแปลงที่ใช้:

```csharp
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```
