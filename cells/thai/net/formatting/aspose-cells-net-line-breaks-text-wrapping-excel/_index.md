## คู่มือการใช้งาน

### การเพิ่มการแบ่งบรรทัดและการเปิดใช้งานการห่อข้อความ

**ภาพรวม:**
ในส่วนนี้เราจะเพิ่มการแบ่งบรรทัดที่ชัดเจนในข้อความในเซลล์และเปิดใช้งานการตัดข้อความเพื่อให้แสดงเนื้อหาเรียบร้อยใน Excel

#### ขั้นตอนที่ 1: สร้างสมุดงานและเข้าถึงแผ่นงาน

เริ่มต้นด้วยการสร้าง `Workbook` วัตถุและการเข้าถึงแผ่นงานแรกของมัน:
```csharp
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```
**คำอธิบาย:** การ `Workbook` แสดงถึงไฟล์ Excel ทั้งหมด ในขณะที่แต่ละไฟล์ `Worksheet` เป็นเหมือนแผ่นงานภายในสมุดงาน

#### ขั้นตอนที่ 2: ตั้งค่าเซลล์พร้อมการแบ่งบรรทัด

เข้าถึงเซลล์ที่ต้องการและตั้งค่าโดยใช้การแบ่งบรรทัดที่ชัดเจน (`\n`) สำหรับบรรทัดใหม่:
```csharp
Cell c5 = ws.Cells["C5"];
c5.PutValue("I am using\nThe latest version of \nAspose.Cells to \ntest this functionality");
```
**คำอธิบาย:** การ `PutValue` วิธีการกำหนดข้อความให้กับเซลล์ โดยที่ `\n` หมายถึงการแบ่งบรรทัด

#### ขั้นตอนที่ 3: เปิดใช้งานการห่อข้อความ

เพื่อให้แน่ใจว่าข้อความพอดีกับขอบเขตเซลล์ ให้เปิดใช้งานการตัดข้อความ:
```csharp
Style style = c5.GetStyle();
style.IsTextWrapped = true;
c5.SetStyle(style);
```
**คำอธิบาย:** การ `IsTextWrapped` คุณสมบัติจะกำหนดว่าเนื้อหาควรห่อหรือไม่ การตั้งค่าเป็น `true` ทำให้ข้อความปรับตามความกว้างของคอลัมน์

#### ขั้นตอนที่ 4: บันทึกสมุดงาน

สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณลงในไฟล์ Excel:
```csharp
string outputDir = "your/output/directory";
wb.Save(outputDir + "outputUseExplicitLineBreaks.xlsx");
Console.WriteLine("Workbook saved successfully.");
```
**คำอธิบาย:** การ `Save` วิธีการเขียนเวิร์กบุ๊กไปยังตำแหน่งที่ระบุบนดิสก์

### เคล็ดลับการแก้ไขปัญหา

- **ข้อความไม่ห่อ:** ตรวจสอบให้แน่ใจว่าได้เปิดใช้งานการตัดข้อความสำหรับเซลล์ที่จำเป็นแต่ละเซลล์
- **การแบ่งบรรทัดไม่ถูกต้อง:** ตรวจสอบว่าแทรกเส้นแบ่งบรรทัดอย่างถูกต้องโดยใช้ `\n`-
