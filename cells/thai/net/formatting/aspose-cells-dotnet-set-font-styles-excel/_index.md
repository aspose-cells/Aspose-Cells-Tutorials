## คู่มือการใช้งาน

### การสร้างสมุดงานและแผ่นงาน

เริ่มต้นด้วยการสร้างเวิร์กบุ๊กใหม่และเพิ่มเวิร์กชีต:

```csharp
// สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กใหม่
Workbook workbook = new Workbook();

// เพิ่มแผ่นงานใหม่
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

### การเข้าถึงและการปรับเปลี่ยนรูปแบบเซลล์

หัวใจหลักของบทช่วยสอนนี้คือการจัดการรูปแบบตัวอักษร ดังต่อไปนี้:

#### การกำหนดน้ำหนักแบบอักษรเป็นตัวหนา

หากต้องการทำให้ข้อความเป็นตัวหนา ให้เข้าถึงวัตถุสไตล์ของเซลล์ที่ต้องการ:

```csharp
// เข้าถึงเซลล์ "A1"
Aspose.Cells.Cell cell = worksheet.Cells["A1"];

// เพิ่มมูลค่าให้กับเซลล์
cell.PutValue("Hello Aspose!");

// รับวัตถุสไตล์ที่เชื่อมโยงกับเซลล์
Style style = cell.GetStyle();

// ตั้งค่าน้ำหนักตัวอักษรเป็นตัวหนา
style.Font.IsBold = true;

// นำสไตล์กลับไปใช้กับเซลล์
cell.SetStyle(style);
```

#### คำอธิบายรหัส
- **รับสไตล์()**: ดึงการตั้งค่ารูปแบบปัจจุบันของเซลล์
- **แบบอักษรตัวหนา**: คุณสมบัติที่ควบคุมความหนาของข้อความ ตั้งค่าเป็น `true` ใช้การจัดรูปแบบตัวหนา

### การบันทึกไฟล์ Excel

สุดท้าย ให้บันทึกสมุดงานของคุณเพื่อรักษาการเปลี่ยนแปลง:

```csharp
string outputPath = "Path_to_output_directory\\styledWorkbook.xls";
workbook.Save(outputPath, SaveFormat.Excel97To2003);
```

- **แผงข้อมูลการวิเคราะห์ข้อมูล**:การสร้างตัวชี้วัดที่สำคัญให้โดดเด่น
- **เครื่องมือทางการศึกษา**:การปรับปรุงการอ่านเนื้อหาการศึกษา

การเปลี่ยนแปลงเหล่านี้สามารถบูรณาการกับระบบอื่นๆ ได้ เพื่อให้มั่นใจว่าเอกสาร Excel ของคุณยังคงมีความคล่องตัวและให้ข้อมูลได้ดี
