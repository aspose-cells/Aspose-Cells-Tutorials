## คู่มือการใช้งาน
คู่มือนี้ครอบคลุมการตั้งค่าการโทรกลับคำเตือนสำหรับการแทนที่แบบอักษรและการแปลงเวิร์กบุ๊ก Excel เป็น PDF โดยมีคำเตือนเหล่านี้อยู่

### การนำการแจ้งเตือนการแทนที่แบบอักษรไปใช้
#### ภาพรวม
เป้าหมายที่นี่คือการสร้างกลไกที่จะแจ้งเตือนคุณทุกครั้งที่ไลบรารีทำการแทนที่แบบอักษรในระหว่างการแปลง เพื่อให้แน่ใจว่าเอาต์พุตของคุณตรงตามที่คาดหวัง

#### การดำเนินการแบบทีละขั้นตอน
**สร้างคลาสการโทรกลับ**
กำหนดคลาสที่ใช้งาน `IWarningCallback` เพื่อจัดการคำเตือนในระหว่างการดำเนินการ เช่น การแปลง:
```csharp
using Aspose.Cells;
using System.Diagnostics;

public class GetWarningsForFontSubstitution : IWarningCallback
{
    // วิธีการจับภาพและบันทึกคำเตือนการแทนที่แบบอักษร
    public void Warning(WarningInfo info)
    {
        if (info.WarningType == WarningType.FontSubstitution)
        {
            Debug.WriteLine("WARNING INFO: " + info.Description);
        }
    }
}
```

**คำอธิบาย:** คลาสนี้จะคอยฟังเหตุการณ์เตือนระหว่างการแปลง หากชนิดของเหตุการณ์เป็น `FontSubstitution`มันบันทึกข้อความโดยละเอียดโดยใช้ `Debug-WriteLine`.

### การแปลงสมุดงานเป็น PDF พร้อมคำเตือนการแทนที่แบบอักษร
#### ภาพรวม
เมื่อคอลแบ็กคำเตือนของเราพร้อมแล้ว เรามาใช้มันเพื่อแปลงเวิร์กบุ๊ก Excel เป็นไฟล์ PDF พร้อมกับบันทึกคำเตือนการแทนที่แบบอักษร

**การดำเนินการแปลง**
สร้างคลาสคงที่และวิธีการสำหรับการจัดการกระบวนการแปลง:
```csharp
using Aspose.Cells;
using System.IO;

public static class ConvertWorkbookToPdfWithWarnings
{
    public static void Run()
    {
        // กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุตของคุณ
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string OutputDir = "YOUR_OUTPUT_DIRECTORY";

        // โหลดเวิร์กบุ๊ก Excel จากไดเร็กทอรีที่ระบุ
        Workbook workbook = new Workbook(SourceDir + "sampleGetWarningsForFontSubstitution.xlsx");

        // สร้างอินสแตนซ์ของ PdfSaveOptions เพื่อปรับแต่งตัวเลือกการบันทึก
        PdfSaveOptions options = new PdfSaveOptions();

        // กำหนดการเรียกกลับคำเตือนของเราเพื่อจัดการคำเตือนการแทนที่แบบอักษร
        options.WarningCallback = new GetWarningsForFontSubstitution();

        // บันทึกสมุดงานเป็นไฟล์ PDF โดยใช้ตัวเลือกที่ระบุ
        workbook.Save(OutputDir + "outputGetWarningsForFontSubstitution.pdf", options);
    }
}
```

**คำอธิบาย:** โค้ดนี้โหลดไฟล์ Excel และตั้งค่า `PdfSaveOptions` เพื่อใช้การโทรกลับคำเตือนแบบกำหนดเองของเรา เมื่อโทร `workbook.Save`คำเตือนการแทนที่แบบอักษรใด ๆ จะถูกบันทึกโดยการโทรกลับ ช่วยให้สามารถควบคุมคุณภาพเอาต์พุตของคุณได้ดีขึ้น
