## คู่มือการใช้งาน

### การปรับแต่งตัวคั่นทศนิยมและกลุ่ม

**ภาพรวม:**
การกำหนดตัวคั่นทศนิยมและกลุ่มเองจะช่วยเพิ่มความสามารถในการอ่านข้อมูลและตรงตามมาตรฐานการจัดรูปแบบเฉพาะที่กำหนดโดยภูมิภาคหรือธุรกิจต่างๆ

#### ขั้นตอนที่ 1: กำหนดค่าการตั้งค่า
เริ่มต้นด้วยการระบุรูปแบบตัวเลขที่คุณต้องการสำหรับสมุดงานทั้งหมด:
```csharp
// กำหนดตัวคั่นทศนิยมและกลุ่มที่กำหนดเอง
workbook.Settings.NumberDecimalSeparator = '.';
workbook.Settings.NumberGroupSeparator = ' ';
```
**คำอธิบาย:** การ `NumberDecimalSeparator` ถูกกำหนดให้เป็นจุด (.) ตามที่ใช้กันทั่วไปในหลายภูมิภาค `NumberGroupSeparator` มีการกำหนดค่าให้เป็นช่องว่าง (' ') ซึ่งสามารถปรับเปลี่ยนได้ตามความต้องการของแต่ละภูมิภาค

#### ขั้นตอนที่ 2: ใช้รูปแบบที่กำหนดเอง
เมื่อกำหนดตัวคั่นแล้ว ให้ใช้รูปแบบที่กำหนดเองกับเซลล์ของคุณ:
```csharp
Worksheet worksheet = workbook.Worksheets[0];

// ตั้งค่าเซลล์และใช้รูปแบบ
Cell cell = worksheet.Cells["A1"];
cell.PutValue(123456.789);

Style style = cell.GetStyle();
style.Custom = "#,##0.000;[Red]#,##0.000"; // รูปแบบสตริงที่กำหนดเอง
cell.SetStyle(style);
```
**คำอธิบาย:** รูปแบบที่กำหนดเอง `#,##0.000` รับรองตำแหน่งทศนิยมสามตำแหน่งและจัดกลุ่มหลักโดยใช้ตัวคั่นที่กำหนด

#### ขั้นตอนที่ 3: ปรับคอลัมน์ให้พอดีอัตโนมัติ
เพื่อให้แน่ใจว่าข้อมูลของคุณถูกนำเสนออย่างดี ให้ปรับคอลัมน์ให้พอดีโดยอัตโนมัติ:
```csharp
worksheet.AutoFitColumns();
```
วิธีนี้จะปรับความกว้างของคอลัมน์ให้พอดีกับเนื้อหาโดยอัตโนมัติ

#### ขั้นตอนที่ 4: บันทึกเป็น PDF
สุดท้าย ให้บันทึกสมุดงานเป็น PDF โดยใช้การตั้งค่าแบบกำหนดเองของคุณ:
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY/CustomSeparator_out.pdf");
```
