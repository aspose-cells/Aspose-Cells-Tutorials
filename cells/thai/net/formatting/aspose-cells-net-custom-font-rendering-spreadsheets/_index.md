# เรนเดอร์สเปรดชีตด้วยแบบอักษรที่กำหนดเองโดยใช้ Aspose.Cells .NET: คู่มือฉบับสมบูรณ์

## การแนะนำ
ในยุคดิจิทัล การเรนเดอร์สเปรดชีตเป็นรูปภาพถือเป็นสิ่งสำคัญสำหรับรายงาน การนำเสนอ หรือการแบ่งปันข้อมูล การรับประกันว่าแบบอักษรมีความสม่ำเสมอและสวยงามอาจเป็นเรื่องท้าทาย โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับแบบอักษรที่ไม่รู้จักหรือขาดหายไป คู่มือนี้สาธิตวิธีการใช้ Aspose.Cells .NET เพื่อเรนเดอร์สเปรดชีตด้วยแบบอักษรเริ่มต้นที่กำหนดเอง เพื่อให้แน่ใจว่าผลลัพธ์จะมีความสม่ำเสมอ

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่าแบบอักษรเริ่มต้นสำหรับการเรนเดอร์สเปรดชีต

- การประยุกต์ใช้เทคนิคเหล่านี้ในโลกแห่งความเป็นจริง

ด้วย Aspose.Cells .NET คุณสามารถจัดการงานเหล่านี้ได้อย่างมีประสิทธิภาพ โดยรักษาความสมบูรณ์ของสเปรดชีตของคุณในทุกแพลตฟอร์ม มาเริ่มต้นด้วยข้อกำหนดเบื้องต้นกันก่อน

## คู่มือการใช้งาน

### คุณลักษณะที่ 1: ตั้งค่าแบบอักษรเริ่มต้นขณะเรนเดอร์สเปรดชีต

#### ภาพรวม
คุณลักษณะนี้ทำให้แน่ใจว่าการแสดงผลแบบอักษรสเปรดชีตมีความสม่ำเสมอ แม้ว่าแบบอักษรที่ระบุจะหายไปหรือไม่รู้จักก็ตาม

#### การดำเนินการแบบทีละขั้นตอน
**ขั้นตอนที่ 1: เตรียมสมุดงานของคุณ**
สร้างวัตถุเวิร์กบุ๊กและตั้งค่ารูปแบบเริ่มต้น:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook();
Style s = wb.DefaultStyle;
s.Font.Name = "Arial"; // ตั้งค่าแบบอักษรเริ่มต้นเบื้องต้น
wb.DefaultStyle = s;
```
**ขั้นตอนที่ 2: กำหนดค่าแผ่นงานของคุณ**
เข้าถึงเวิร์กชีตของคุณ ตั้งค่าเซลล์ และใช้รูปแบบ:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell cell = ws.Cells["A4"];
cell.PutValue("This text uses a custom default font.");

Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist"; // ใช้แบบอักษรที่ไม่มีอยู่โดยตั้งใจ
st.Font.Size = 20;
st.IsTextWrapped = true;
cell.SetStyle(st);

// ปรับความกว้างของคอลัมน์และความสูงของแถวเพื่อการแสดงผลที่ดีขึ้น:
ws.Cells.SetColumnWidth(0, 80);
ws.Cells.SetRowHeight(3, 60);
```
**ขั้นตอนที่ 3: เรนเดอร์ด้วยแบบอักษรที่กำหนดเอง**
ตั้งค่าตัวเลือกภาพเพื่อแสดงเวิร์กชีตของคุณโดยใช้แบบอักษรเริ่มต้นที่แตกต่างกัน:
```csharp
using Aspose.Cells.Rendering;

ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.OnePagePerSheet = true;
opts.ImageType = Drawing.ImageType.Png;

// เรนเดอร์โดยใช้ 'Arial' เป็นแบบอักษรเริ่มต้น
opts.DefaultFont = "Arial";
SheetRender sr = new SheetRender(ws, opts);
sr.ToImage(0, System.IO.Path.Combine(outputDir, "out_a.png"));

// เปลี่ยนเป็น 'Times New Roman'
opts.DefaultFont = "Times New Roman";
sr = new SheetRender(ws, opts);
sr.ToImage(0, System.IO.Path.Combine(outputDir, "times_new_roman_out.png"));
```
### คุณสมบัติ 2: ตั้งค่าความกว้างของคอลัมน์และความสูงของแถว

#### ภาพรวม
การปรับความกว้างของคอลัมน์และความสูงของแถวช่วยให้แสดงข้อมูลได้ชัดเจนและเป็นมืออาชีพ

**การดำเนินการแบบทีละขั้นตอน**
**ขั้นตอนที่ 1: ปรับขนาด**
เข้าถึงแผ่นงานและกำหนดขนาดที่เฉพาะเจาะจง:
```csharp
Worksheet ws = wb.Worksheets[0];
ws.Cells.SetColumnWidth(0, 80); // กำหนดความกว้างของคอลัมน์แรก
ws.Cells.SetRowHeight(3, 60);   // ตั้งค่าความสูงแถวที่สี่
```
