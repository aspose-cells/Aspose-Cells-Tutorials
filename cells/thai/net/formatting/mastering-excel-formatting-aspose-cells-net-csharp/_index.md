## คู่มือการใช้งาน

### การตั้งค่าสมุดงาน

#### ภาพรวม

ขั้นแรกเราจะสร้างเวิร์กบุ๊ก Excel ใหม่และเติมข้อมูลตัวอย่างลงไป

**ขั้นตอนที่ 1: สร้างสมุดงานใหม่**
```csharp
using Aspose.Cells;

namespace ExcelFormattingGuide
{
    class Program
    {
        static void Main(string[] args)
        {
            // เริ่มต้นสมุดงานใหม่
            var workbook = new Workbook();
            
            // เข้าถึงแผ่นงานแรก
            var sheet = workbook.Worksheets[0];
            
            // เพิ่มข้อมูลตัวอย่างลงในเซลล์
            sheet.Cells["A1"].PutValue("Month");
            sheet.Cells["B1"].PutValue("Sales");

            for (int i = 2; i <= 13; i++)
            {
                sheet.Cells[$"A{i}"].PutValue($"Month {i-1}");
                sheet.Cells[$"B{i}"].PutValue(i * 1000);
            }
        }
    }
}
```

**คำอธิบาย:** โค้ดนี้จะเริ่มต้นเวิร์กบุ๊กใหม่และเพิ่มข้อมูลตัวอย่างยอดขายรายเดือน `PutValue` วิธีการแทรกค่าลงในเซลล์ที่ระบุ

### การจัดรูปแบบเซลล์

#### ภาพรวม

ต่อไปเราจะใช้รูปแบบต่างๆ เพื่อปรับปรุงการอ่านข้อมูลของเรา

**ขั้นตอนที่ 2: ใช้สไตล์**
```csharp
// สร้างวัตถุสไตล์สำหรับส่วนหัว
Style headerStyle = workbook.CreateStyle();
headerStyle.ForegroundColor = System.Drawing.Color.FromArgb(124, 199, 72);
headerStyle.Pattern = BackgroundType.Solid;
headerStyle.Font.IsBold = true;
headerStyle.HorizontalAlignment = TextAlignmentType.Center;

// ใช้รูปแบบกับแถวแรก (ส่วนหัว)
Range headerRange = sheet.Cells.CreateRange("A1", "B1");
headerRange.ApplyStyle(headerStyle, new StyleFlag() { All = true });
```

**คำอธิบาย:** สไนปเป็ตนี้สร้างสไตล์ตัวหนาตรงกลางพร้อมพื้นหลังสีเขียวสำหรับส่วนหัว `ApplyStyle` วิธีนี้จะใช้รูปแบบนี้กับช่วงที่ระบุ

### การจัดรูปแบบตามเงื่อนไข

#### ภาพรวม

เพื่อเน้นตัวเลขยอดขายที่โดดเด่น เราจะใช้การจัดรูปแบบตามเงื่อนไข

**ขั้นตอนที่ 3: ใช้การจัดรูปแบบตามเงื่อนไข**
```csharp
// กำหนดกฎเพื่อเน้นเซลล์ที่มากกว่า $10,000
int index = sheet.ConditionalFormattings.Add();
var cfRule = sheet.ConditionalFormattings[index].AddCondition(FormatConditionType.CellValue, OperatorType.GreaterThan, "10000");
cfRule.Style.ForegroundColor = System.Drawing.Color.FromArgb(255, 192, 0);
cfRule.Style.Pattern = BackgroundType.Solid;
cfRule.Formula1 = "10000";

// ใช้กฎเกณฑ์กับข้อมูลการขาย
var range = sheet.Cells.CreateRange("B2", "B13");
sheet.ConditionalFormattings[index].AddArea(range);
```

**คำอธิบาย:** โค้ดนี้จะกำหนดกฎการจัดรูปแบบตามเงื่อนไขที่เน้นเซลล์ที่มียอดขายเกิน 10,000 ดอลลาร์เป็นสีส้ม
