## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณในการโหลดเวิร์กบุ๊กที่มีรูปแบบวันที่เฉพาะวัฒนธรรมโดยใช้ Aspose.Cells

### การกำหนดค่ารูปแบบวันที่เฉพาะวัฒนธรรม
เพื่อให้แน่ใจว่าแอปพลิเคชันของคุณแปลความวันที่จากตำแหน่งต่างๆ ได้อย่างถูกต้อง ให้กำหนดค่า `CultureInfo` การตั้งค่าให้ตรงกับรูปแบบที่คาดหวัง

#### การตั้งค่าตัวเลือกการโหลดด้วย CultureInfo
1. **สร้าง MemoryStream สำหรับข้อมูลอินพุต**:จำลองการอ่านข้อมูลจากไฟล์ HTML
2. **เขียนเนื้อหา HTML พร้อมวันที่**:รวมวันที่ในรูปแบบเฉพาะวัฒนธรรม
3. **กำหนดค่าการตั้งค่าวัฒนธรรม**-
   - ชุด `NumberDecimalSeparator`- `DateSeparator`, และ `ShortDatePattern`-
4. **ใช้ LoadOptions เพื่อระบุ CultureInfo**-

```csharp
using System;
using System.IO;
using System.Globalization;
using Aspose.Cells;

class LoadWorkbookWithSpecificCultureInfoDateFormat
{
    public static void Run()
    {
        using (var inputStream = new MemoryStream())
        {
            using (var writer = new StreamWriter(inputStream))
            {
                // เขียนเนื้อหา HTML ด้วยวันที่ในรูปแบบ "dd-MM-yyyy"
                writer.WriteLine("<html><head><title>Test Culture</title></head><body><table><tr><td>10-01-2016</td></tr></table></body></html>");
                writer.Flush();
                
                // กำหนดค่าการตั้งค่าวัฒนธรรมสำหรับรูปแบบวันที่ของสหราชอาณาจักร
                var culture = new CultureInfo("en-GB");
                culture.NumberFormat.NumberDecimalSeparator = ",";
                culture.DateTimeFormat.DateSeparator = "-";
                culture.DateTimeFormat.ShortDatePattern = "dd-MM-yyyy";

                // สร้าง LoadOptions ด้วยวัฒนธรรมที่ระบุ
                LoadOptions options = new LoadOptions(LoadFormat.Html);
                options.CultureInfo = culture;

                // โหลดสมุดงานโดยใช้ InputStream และ LoadOptions
                using (var workbook = new Workbook(inputStream, options))
                {
                    var cell = workbook.Worksheets[0].Cells["A1"];
                    
                    // ยืนยันว่าวันที่ได้รับการตีความอย่างถูกต้องเป็น DateTime
                    Console.WriteLine("Date Type: " + cell.Type == CellValueType.IsDateTime);
                    Console.WriteLine("Parsed Date: " + cell.DateTimeValue.ToString(culture));
                }
            }
        }
        
        Console.WriteLine("LoadWorkbookWithSpecificCultureInfoDateFormat executed successfully.");
    }
}
```

**พารามิเตอร์และวัตถุประสงค์:**
- **เมมโมรี่สตรีม**:จำลองการอ่านข้อมูลเหมือนกับว่ามาจากไฟล์
- **ข้อมูลวัฒนธรรม**: กำหนดค่าแอปพลิเคชันเพื่อตีความวันที่ใน `dd-MM-yyyy` รูปแบบมีความสำคัญต่อการจัดการวันที่ของสหราชอาณาจักร

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบการตั้งค่าวัฒนธรรมของคุณ (`DateSeparator`- `ShortDatePattern`) ให้ตรงกับที่ใช้ภายในเวิร์กบุ๊ก
- ตรวจสอบว่าอินพุต HTML มีการจัดรูปแบบอย่างถูกต้องและสามารถเข้าถึงได้โดย MemoryStream
