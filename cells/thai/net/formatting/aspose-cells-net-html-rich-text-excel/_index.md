## คู่มือการใช้งาน

ตอนนี้เรามีข้อกำหนดเบื้องต้นและการตั้งค่าพร้อมแล้ว มาดูขั้นตอนการใช้ฟีเจอร์ต่างๆ ของเราทีละขั้นตอนกัน

### การเพิ่มข้อความ HTML ที่อุดมไปด้วยข้อมูลลงในเซลล์

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณแทรกข้อความที่มีรูปแบบ HTML ลงในเซลล์ Excel ได้ โดยใช้แท็ก HTML คุณสามารถใช้รูปแบบต่างๆ เช่น ตัวหนา ตัวเอียง ขีดเส้นใต้ การเปลี่ยนแบบอักษร การปรับสี และอื่นๆ ภายในเนื้อหาเซลล์ได้

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊กและเวิร์กชีต**
เริ่มต้นด้วยการสร้างเวิร์กบุ๊กใหม่และเข้าถึงเวิร์กชีตแรก:

```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**ขั้นตอนที่ 2: อ้างอิงเซลล์เป้าหมาย**
รับการอ้างอิงไปยังเซลล์ที่คุณต้องการใช้การจัดรูปแบบ HTML ในตัวอย่างนี้ เราจะใช้เซลล์ "A1":

```csharp
Cell cell = worksheet.Cells["A1"];
```

**ขั้นตอนที่ 3: ตั้งค่าสตริง HTML สำหรับการจัดรูปแบบข้อความที่หลากหลาย**
กำหนดสตริง HTML ด้วยข้อความและรูปแบบที่คุณต้องการ:

```csharp
string htmlString = "<Font Style=\"FONT-WEIGHT: bold; FONT-STYLE: italic; TEXT-DECORATION: underline; FONT-FAMILY: Arial; FONT-SIZE: 11pt; COLOR: #ff0000;\">This is simple HTML formatted text.</Font>";
cell.HtmlString = htmlString;
```

**ขั้นตอนที่ 4: บันทึกสมุดงาน**
สุดท้าย ให้บันทึกสมุดงานของคุณไปยังไดเร็กทอรีที่ระบุ:

```csharp
workbook.Save("output_out.xlsx");
```

### การทำงานกับเวิร์กบุ๊กและวัตถุเวิร์กชีต

#### ภาพรวม
นอกเหนือจากการเพิ่มข้อความที่มีรูปแบบหลากหลายแล้ว สิ่งสำคัญคือต้องเข้าใจวิธีการสร้างและจัดการเวิร์กบุ๊กและเวิร์กชีตโดยใช้ Aspose.Cells

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊ก**
สร้างอินสแตนซ์ใหม่ของ `Workbook`-

```csharp
Workbook workbook = new Workbook();
```

**ขั้นตอนที่ 2: เข้าถึงแผ่นงาน**
ดึงข้อมูลคอลเลกชันของแผ่นงานในสมุดงานของคุณ:

```csharp
WorksheetCollection worksheets = workbook.Worksheets;
```

**ขั้นตอนที่ 3: อ้างอิงและแก้ไขเซลล์**
เข้าถึงเซลล์เฉพาะเพื่อดำเนินการตามต้องการ เช่น การเข้าถึงเซลล์ "A1":

```csharp
Cell cell = worksheets[0].Cells["A1"];
// ตอนนี้คุณสามารถดำเนินการต่างๆ บนเวิร์กชีตหรือเซลล์ที่นี่ได้
```

**ขั้นตอนที่ 4: บันทึกการเปลี่ยนแปลง**
หลังจากทำการเปลี่ยนแปลงของคุณแล้ว ให้บันทึกสมุดงาน:

```csharp
workbook.Save("output.xlsx");
```

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าแท็ก HTML ได้รับการจัดรูปแบบอย่างถูกต้องเพื่อหลีกเลี่ยงปัญหาการแสดงผลใน Excel
- ตรวจสอบเส้นทางไฟล์และการอนุญาตสำหรับการบันทึกสมุดงาน
