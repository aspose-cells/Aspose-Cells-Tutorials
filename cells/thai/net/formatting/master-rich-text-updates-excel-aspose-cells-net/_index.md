## คู่มือการใช้งาน

ตอนนี้เรามาลองใช้ฟีเจอร์การอัปเดตข้อความแบบสมบูรณ์กัน เราจะแบ่งคู่มือนี้ออกเป็นส่วนๆ เพื่อช่วยให้คุณทำตามได้ง่าย

### การโหลดและการเข้าถึงเซลล์ข้อความที่มีเนื้อหาหลากหลาย

#### ภาพรวม
หากต้องการอัปเดตเซลล์ที่มีเนื้อหาข้อความที่หลากหลายในไฟล์ Excel ขั้นแรกให้โหลดเวิร์กบุ๊กของคุณ จากนั้นเข้าถึงเวิร์กชีตและเซลล์เฉพาะที่ต้องมีการอัปเดต
```csharp
// กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// โหลดเวิร์กบุ๊กที่มีไฟล์ Excel ของคุณ
Workbook workbook = new Workbook(sourceDir + "sampleUpdateRichTextCells.xlsx");

// เข้าถึงแผ่นงานแรก
Worksheet worksheet = workbook.Worksheets[0];

// รับเซลล์ A1 ซึ่งประกอบด้วยข้อความที่หลากหลาย
Cell cell = worksheet.Cells["A1"];
```

#### คำอธิบาย
- **สมุดงาน:** แสดงถึงไฟล์ Excel ทั้งหมด
- **ใบงาน :** แผ่นงานเดียวภายในสมุดงานของคุณ เข้าถึงได้โดยดัชนีหรือชื่อ
- **เซลล์:** เซลล์เฉพาะที่คุณต้องการทำการอัปเดต

### การอัปเดตการตั้งค่าแบบอักษรในเซลล์ข้อความที่มีรูปแบบหลากหลาย

#### ภาพรวม
หากต้องการเปลี่ยนการตั้งค่าแบบอักษรของเนื้อหาข้อความที่มีรูปแบบหลากหลายภายในเซลล์ ให้ดึงข้อมูลและแก้ไข `FontSetting` วัตถุ
```csharp
Console.WriteLine("Before updating the font settings....");

// รับอักขระทั้งหมดในเซลล์เป็นอาร์เรย์ของ FontSettings
FontSetting[] fnts = cell.GetCharacters();

// วนซ้ำผ่าน FontSetting แต่ละรายการเพื่อพิมพ์ชื่อแบบอักษรปัจจุบัน
for (int i = 0; i < fnts.Length; i++)
{
    Console.WriteLine(fnts[i].Font.Name);
}

// อัปเดตชื่อฟอนต์ของ FontSetting แรก
fnts[0].Font.Name = "Arial";

// นำการเปลี่ยนแปลงกลับไปใช้กับเซลล์
cell.SetCharacters(fnts);

Console.WriteLine();

Console.WriteLine("After updating the font settings....");

// ดึงข้อมูล FontSettings ที่อัพเดตแล้ว
fnts = cell.GetCharacters();

// พิมพ์ชื่อแบบอักษรใหม่
for (int i = 0; i < fnts.Length; i++)
{
    Console.WriteLine(fnts[i].Font.Name);
}
```

#### คำอธิบาย
- **รับอักขระ():** ดึงข้อมูลอาร์เรย์ของ `FontSetting` วัตถุที่แสดงส่วนข้อความที่มีเนื้อหาหลากหลายภายในเซลล์
- **ตั้งค่าอักขระ (FontSetting[]):** นำการตั้งค่าแบบอักษรที่แก้ไขแล้วกลับไปใช้กับเซลล์
- **เคล็ดลับการแก้ไขปัญหา:** ตรวจสอบให้แน่ใจว่าคุณใช้การเปลี่ยนแปลงโดยใช้ `SetCharacters()`; มิฉะนั้น การปรับเปลี่ยนจะไม่ดำเนินต่อไป

### การบันทึกการเปลี่ยนแปลง

เมื่อทำการอัปเดตแล้ว ให้บันทึกสมุดงานของคุณ:
```csharp
// บันทึกสมุดงานที่อัปเดตไปยังไฟล์ใหม่
workbook.Save(outputDir + "outputUpdateRichTextCells.xlsx");

Console.WriteLine("UpdateRichTextCells executed successfully.");
```
