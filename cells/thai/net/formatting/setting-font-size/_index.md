# กําหนดขนาด Font ใน Excel โดยใช้ Aspose.Cells

## คําแนะนําการติดตั้ง

เพื่อเริ่มต้นกับ Aspose.Cells สําหรับ .NET ติดตามขั้นตอนการติดตั้งนี้  

1. โครงการผ่าน NuGet Package Manager:  
   * เปิด Visual Studio  
   * ไปยัง Tools > NuGet Package Manager > การจัดการ NuGet Packages for Solution  
   * ค้นหา`"อสโปส เซลล์"`แล้วคลิกติดตั้ง  

2. โกตั้งผ่าน NuGet Package Manager Console:  
   * เปิด คอนโซลผู้จัดการแพ็คเกจ จาก Tools > NuGet Package Manager > คอนโซลผู้จัดการแพ็คเกจ  
   * กําหนดการ:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. ดาวน์โหลด DLL ด้วยมือ  
   * ไปเยี่ยม [Aspose.Cells ดาวน์โหลดหน้า](https://releases.aspose.com/cells/net/). .  
   * ดาวน์โหลดและเอาแพ็คเกจออก  
   * เพิ่มความหมายกับ `Aspose.Cells.dll`ในโครงการของคุณ
