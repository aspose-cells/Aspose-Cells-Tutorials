## คู่มือการใช้งาน
### การเพิ่มตัวแบ่งหน้าแนวนอนและแนวตั้ง
การแทรกตัวแบ่งหน้าช่วยในการนำทางชุดข้อมูลขนาดใหญ่โดยการแบ่งข้อมูลออกเป็นส่วนๆ ที่จัดการได้ มาสำรวจวิธีการเพิ่มตัวแบ่งหน้าเหล่านี้ในเวิร์กชีต Excel ด้วยโปรแกรมกัน

#### ภาพรวม
เราจะใช้ Aspose.Cells สำหรับ .NET เพื่อแทรกตัวแบ่งหน้าทั้งสองประเภทในเวิร์กชีต Excel

#### การดำเนินการแบบทีละขั้นตอน
##### **1. เริ่มต้นการใช้งานสมุดงาน**
สร้างวัตถุเวิร์กบุ๊กใหม่:

```csharp
using System;
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY"; // ตั้งค่าไดเรกทอรีแหล่งที่มาของคุณที่นี่
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // ตั้งค่าไดเรกทอรีเอาท์พุตของคุณที่นี่

Workbook workbook = new Workbook();
```
##### **2. เข้าถึงแผ่นงาน**
เข้าถึงแผ่นงานแรกในสมุดงาน:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
##### **3. เพิ่มการแบ่งหน้า**
แทรกตัวแบ่งหน้าแนวนอนและแนวตั้งที่ตำแหน่งเซลล์ที่ระบุ:

```csharp
// แบ่งหน้าแนวนอนที่แถวที่ 30
worksheet.HorizontalPageBreaks.Add("Y30");

// แบ่งหน้าแนวตั้งที่คอลัมน์ 30
worksheet.VerticalPageBreaks.Add("X30");
```
**คำอธิบาย**: ที่นี่, `HorizontalPageBreaks` และ `VerticalPageBreaks` เป็นคอลเลกชันที่จัดการการหยุดพัก `Add` วิธีการระบุสตริงที่แสดงตำแหน่งเซลล์ (เช่น "Y30") โดยระบุตำแหน่งที่จะแทรกจุดแบ่ง
##### **4. บันทึกสมุดงาน**
บันทึกการเปลี่ยนแปลงของคุณโดยการเขียนเวิร์กบุ๊กไปยังไฟล์เอาท์พุต:

```csharp
string outputPath = System.IO.Path.Combine(outputDir, "AddingPageBreaks_out.xls");
workbook.Save(outputPath);
```
#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าการอ้างอิงเซลล์ เช่น "Y30" ถูกต้องและมีอยู่ในเวิร์กชีตของคุณ
- ตรวจสอบว่าคุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาต์พุต
### การสร้างตัวอย่างและการใช้ Workbook Object
การเข้าใจวิธีการทำงานกับวัตถุเวิร์กบุ๊กถือเป็นสิ่งสำคัญสำหรับการจัดการไฟล์ Excel ด้วยโปรแกรม
#### ภาพรวม
เรียนรู้การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก ดำเนินการพื้นฐาน และบันทึกการเปลี่ยนแปลงอย่างมีประสิทธิภาพ
##### **1. สร้างอินสแตนซ์เวิร์กบุ๊ก**
เริ่มต้นอินสแตนซ์ใหม่ของ `Workbook` ระดับ:

```csharp
using Aspose.Cells;

Workbook workbook = new Workbook();
```
##### **2. การเข้าถึงใบงาน**
เข้าถึงแผ่นงานเฉพาะตามดัชนีหรือชื่อ:

```csharp
Worksheet sheet = workbook.Worksheets[0];
```
##### **3. ปรับเปลี่ยนเนื้อหาแผ่นงาน**
เพิ่มข้อมูลลงในเซลล์ตามต้องการ:

```csharp
sheet.Cells["A1"].PutValue("Hello World!");
```
##### **4. บันทึกสมุดงานที่มีการเปลี่ยนแปลง**
คงไว้ซึ่งการเปลี่ยนแปลงโดยการบันทึกสมุดงาน:

```csharp
string outputFilePath = System.IO.Path.Combine(outputDir, "SampleWorkbook_out.xlsx");
workbook.Save(outputFilePath);
```
