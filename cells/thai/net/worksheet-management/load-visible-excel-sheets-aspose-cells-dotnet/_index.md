## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการใช้งานตรรกะเพื่อโหลดเฉพาะแผ่นงานที่มองเห็นได้โดยใช้ Aspose.Cells สำหรับ .NET
### ภาพรวม: การโหลดแผ่นงานที่มองเห็นได้เท่านั้น
เปิดเวิร์กบุ๊ก Excel ได้อย่างมีประสิทธิภาพด้วยการโหลดข้อมูลจากชีตที่มองเห็นได้ โดยไม่เปลี่ยนแปลงชีตที่ซ่อนอยู่ ซึ่งจะช่วยปรับปรุงทั้งประสิทธิภาพและการใช้หน่วยความจำ
#### ขั้นตอนที่ 1: สร้างเวิร์กบุ๊กตัวอย่างพร้อมแผ่นงานที่ซ่อนไว้
เริ่มต้นด้วยการสร้างเวิร์กบุ๊กตัวอย่างโดยมีแผ่นงานบางแผ่นถูกทำเครื่องหมายว่ามองไม่เห็น:
```csharp
string dataDir = "path_to_directory";
string sampleFile = "output.xlsx";
string samplePath = dataDir + sampleFile;
// สร้างสมุดงานใหม่และเพิ่มแผ่นงาน
Workbook createWorkbook = new Workbook();
createWorkbook.Worksheets["Sheet1"].Cells["A1"].Value = "Aspose";
createWorkbook.Worksheets.Add("Sheet2").Cells["A1"].Value = "Aspose";
createWorkbook.Worksheets.Add("Sheet3").Cells["A1"].Value = "Aspose";
// ซ่อนแผ่นที่สาม
createWorkbook.Worksheets["Sheet3"].IsVisible = false;
// บันทึกสมุดงาน
createWorkbook.Save(samplePath);
```
#### ขั้นตอนที่ 2: กำหนดตัวกรองโหลดแบบกำหนดเอง
สร้างตัวกรองการโหลดแบบกำหนดเองเพื่อระบุแผ่นงานที่จะโหลด:
```csharp
class CustomLoad : LoadFilter
{
    public override void StartSheet(Worksheet sheet)
    {
        if (sheet.IsVisible)
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.All;
        }
        else
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.Structure;
        }
    }
}
```
#### ขั้นตอนที่ 3: โหลดเวิร์กบุ๊กด้วยตัวกรองแบบกำหนดเอง
ใช้ตัวกรองโหลดแบบกำหนดเองเพื่อเปิดเฉพาะแผ่นงานที่มองเห็นได้เท่านั้น:
```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.LoadFilter = new CustomLoad();
Workbook loadWorkbook = new Workbook(samplePath, loadOptions);
// เนื้อหาเอาต์พุตของแผ่นงานที่โหลด
Console.WriteLine("Sheet1: A1: {0}", loadWorkbook.Worksheets["Sheet1"].Cells["A1"].Value);
Console.WriteLine("Sheet2: A1: {0}", loadWorkbook.Worksheets["Sheet2"].Cells["A1"].Value);
```
### เคล็ดลับการแก้ไขปัญหา
- ให้แน่ใจว่า `IsVisible` คุณสมบัติได้รับการตั้งค่าอย่างถูกต้องสำหรับแต่ละแผ่นงาน
- ตรวจสอบเส้นทางไฟล์ของคุณและให้แน่ใจว่าเวิร์กบุ๊กมีอยู่ในตำแหน่งที่ระบุ
