## คู่มือการใช้งาน

### คุณสมบัติ 1: รวมไฟล์ Excel หลายไฟล์

ฟีเจอร์นี้สาธิตวิธีการรวมไฟล์ .xls หลายไฟล์ให้เป็นผลลัพธ์เดียวโดยใช้ Aspose.Cells

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต

ตั้งค่าเส้นทางสำหรับไดเร็กทอรีต้นทางและปลายทางของคุณ:

```csharp
string YOUR_SOURCE_DIRECTORY = "YOUR_SOURCE_DIRECTORY";
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";
```

#### ขั้นตอนที่ 2: ระบุไฟล์ที่จะรวม

สร้างอาร์เรย์ของเส้นทางไฟล์ที่คุณต้องการผสาน:

```csharp
String[] files = new String[2];
files[0] = YOUR_SOURCE_DIRECTORY + "/sampleMergeFiles_Book1.xls";
files[1] = YOUR_SOURCE_DIRECTORY + "/sampleMergeFiles_Book2.xls";
```

#### ขั้นตอนที่ 3: ดำเนินการผสาน

ใช้ `CellsHelper.MergeFiles` เพื่อรวมไฟล์ Excel ของคุณเข้าในเวิร์กบุ๊กเดียว:

```csharp
string cacheFile = YOUR_OUTPUT_DIRECTORY + "/cacheMergeFiles.txt";
string dest = YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls";

CellsHelper.MergeFiles(files, cacheFile, dest);
```

### คุณสมบัติ 2: เปลี่ยนชื่อแผ่นงานในไฟล์ Excel รวม

หลังจากรวมไฟล์แล้ว คุณอาจต้องการเปลี่ยนชื่อชีตแต่ละชีตเพื่อให้จัดระเบียบได้ดีขึ้น

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

โหลดเวิร์กบุ๊กที่แผ่นงานจะถูกเปลี่ยนชื่อ:

```csharp
Workbook workbook = new Workbook(YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls");
```

#### ขั้นตอนที่ 2: เปลี่ยนชื่อแผ่นงานตามลำดับ

ทำซ้ำผ่านแต่ละเวิร์กชีตและกำหนดชื่อใหม่:

```csharp
int i = 1;
foreach (Worksheet sheet in workbook.Worksheets)
{
    sheet.Name = "Sheet" + i++;
}
```

#### ขั้นตอนที่ 3: บันทึกสมุดงาน

สุดท้าย ให้บันทึกการเปลี่ยนแปลงของคุณเพื่อรักษาแผ่นงานที่เปลี่ยนชื่อ:

```csharp
workbook.Save(YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls");
```

1. **การรวมรายงานทางการเงิน**:ผสานรายงานทางการเงินรายไตรมาสจากแผนกต่าง ๆ เข้าในสมุดงานเดียวเพื่อการวิเคราะห์ที่ครอบคลุม
2. **การจัดการโครงการ**รวมกำหนดเวลาของโครงการและผลงานส่งมอบระหว่างทีมเพื่อปรับปรุงการวางแผนและการติดตาม
3. **การรวมข้อมูล**:รวบรวมข้อมูลจากแหล่งต่าง ๆ เช่น ยอดขาย หรือคำติชมจากลูกค้า เพื่อการรายงานแบบรวม
