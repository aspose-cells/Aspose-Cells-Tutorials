## คู่มือการใช้งาน

ตอนนี้เราได้ครอบคลุมการตั้งค่าแล้ว มาดูการใช้งานโซลูชันกัน

### การโหลดแผ่นงานเฉพาะ

เป้าหมายคือการโหลดเฉพาะแผ่นงานที่ระบุจากไฟล์ Excel โดยไม่สนใจแผ่นงานอื่นๆ คุณสามารถทำได้ดังนี้:

#### ขั้นตอนที่ 1: กำหนดตัวเลือกการโหลด

ขั้นแรกให้สร้าง `LoadOptions` วัตถุที่ระบุรูปแบบของเวิร์กบุ๊กของคุณและกำหนดตัวกรองโหลดแบบกำหนดเอง

```csharp
LoadOptions loadOptions = new LoadOptions(LoadFormat.Xlsx);
loadOptions.LoadFilter = new CustomLoad();
```

**คำอธิบาย**: เดอะ `LoadOptions` คลาสนี้ให้การตั้งค่าสำหรับการโหลดไฟล์ Excel โดยการตั้งค่า `LoadFilter`คุณสามารถควบคุมแผ่นงานที่จะโหลดตามเกณฑ์ของคุณ

#### ขั้นตอนที่ 2: สร้างตัวกรองโหลดแบบกำหนดเอง

กำหนดตัวกรองแบบกำหนดเองโดยสืบทอดจาก `LoadFilter`นี้จะกำหนดวิธีการประมวลผลแต่ละแผ่น

```csharp
class CustomLoad : LoadFilter
{
    public override void StartSheet(Worksheet sheet)
    {
        if (sheet.Name == "Sheet2")
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.All;
        }
        else
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.Structure;
        }
    }
}
```

**คำอธิบาย**: เดอะ `StartSheet` วิธีการนี้ถูกเขียนทับเพื่อระบุว่าควรโหลดข้อมูลทั้งหมดเฉพาะ "Sheet2" เท่านั้น ในขณะที่แผ่นงานอื่นๆ จะถูกละเว้นนอกเหนือจากโครงสร้างของมัน

#### ขั้นตอนที่ 3: โหลดเวิร์กบุ๊ก

ใช้ตัวเลือกโหลดที่กำหนดเพื่อสร้างอินสแตนซ์เวิร์กบุ๊กและโหลดแผ่นงานที่คุณต้องการ

```csharp
Workbook workbook = new Workbook(sourceDir + "sampleLoadSpecificSheets.xlsx", loadOptions);
```

**คำอธิบาย**: เดอะ `Workbook` ตัวสร้างยอมรับทั้งเส้นทางไฟล์และตัวเลือกการโหลด ช่วยให้คุณระบุแผ่นงานที่จะโหลดตามลอจิกของตัวกรองแบบกำหนดเอง

#### ขั้นตอนที่ 4: บันทึกผลลัพธ์

หลังจากประมวลผลแล้ว ให้บันทึกสมุดงานของคุณด้วยการปรับเปลี่ยนหากจำเป็น:

```csharp
workbook.Save(outputDir + "outputLoadSpecificSheets.xlsx");
```

## การประยุกต์ใช้งานจริง

ต่อไปนี้คือสถานการณ์จริงบางสถานการณ์ที่การโหลดแผ่นงานเฉพาะอาจเป็นประโยชน์ได้:
1. **การวิเคราะห์ข้อมูล**:มุ่งเน้นเฉพาะข้อมูลที่เกี่ยวข้องโดยโหลดแผ่นงานที่จำเป็นสำหรับการวิเคราะห์
2. **การสร้างรายงาน**:สร้างรายงานโดยอิงตามชุดข้อมูลที่เลือกโดยไม่ต้องประมวลผลสมุดงานทั้งหมด
3. **การบูรณาการกับระบบอื่น ๆ**:ปรับปรุงกระบวนการรวบรวมข้อมูลโดยนำเข้าข้อมูลที่จำเป็นอย่างเลือกสรร
