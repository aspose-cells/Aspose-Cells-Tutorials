## คู่มือการใช้งาน

ในหัวข้อนี้ เราจะอธิบายวิธีการตรวจจับแมโครชีตนานาชาติโดยใช้ Aspose.Cells สำหรับ .NET

### ภาพรวม: การตรวจจับประเภทแผ่นงาน

เป้าหมายคือโหลดไฟล์ Excel และตรวจสอบว่ามีชีตแมโครระหว่างประเทศหรือไม่ เราจะทำสิ่งนี้โดยตรวจสอบประเภทของชีตแต่ละชีตในเวิร์กบุ๊ก

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก
เริ่มต้นด้วยการโหลดไฟล์ Excel ต้นฉบับของคุณลงใน `Workbook` วัตถุ:

```csharp
// เส้นทางไดเร็กทอรีแหล่งที่มา
string sourceDir = RunExamples.Get_SourceDirectory();

// โหลดไฟล์ Excel ต้นฉบับ
Workbook workbook = new Workbook(sourceDir + "InternationalMacroSheet.xlsm");
```

#### ขั้นตอนที่ 2: รับประเภทแผ่นงาน
ขั้นตอนต่อไปคือการดึงประเภทของเวิร์กชีตแรกเพื่อตรวจสอบว่าเป็นชีตแมโครระหว่างประเทศหรือไม่:

```csharp
// รับประเภทแผ่นงาน
SheetType sheetType = workbook.Worksheets[0].Type;
```

#### ขั้นตอนที่ 3: พิมพ์ประเภทแผ่นงาน
สุดท้ายส่งออกประเภทแผ่นงานที่ตรวจพบไปยังคอนโซล:

```csharp
// ประเภทแผ่นพิมพ์
Console.WriteLine("Sheet Type: " + sheetType);
```

### คำอธิบายพารามิเตอร์และวิธีการ

- `Workbook`: หมายถึงไฟล์ Excel โดยตัวสร้างจะใช้เส้นทางไฟล์เป็นพารามิเตอร์
- `Worksheets[0]`: เข้าถึงแผ่นงานแรกในเวิร์กบุ๊ก
- `sheetType`: การแจงนับที่อธิบายประเภทของเวิร์กชีต (เช่น เวิร์กชีต มาโครชีต)

### เคล็ดลับการแก้ไขปัญหาทั่วไป

- ตรวจสอบให้แน่ใจว่าไดเร็กทอรีแหล่งที่มาและเส้นทางไฟล์ของคุณถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- ตรวจสอบว่าคุณมีสิทธิ์ที่เหมาะสมในการเข้าถึงและอ่านไฟล์ Excel
