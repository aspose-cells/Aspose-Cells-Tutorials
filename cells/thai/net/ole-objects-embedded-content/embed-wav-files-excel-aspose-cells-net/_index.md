# วิธีการแทรกไฟล์ WAV เป็นวัตถุ OLE ใน Excel ด้วย Aspose.Cells .NET

ปรับปรุงเอกสาร Excel ของคุณโดยฝังไฟล์สื่อ เช่น ไฟล์เสียง ไว้ภายในเอกสารโดยตรง ไม่ว่าจะเป็นการสร้างงานนำเสนอ รายงาน หรือสเปรดชีตแบบโต้ตอบ การแทรกองค์ประกอบมัลติมีเดีย เช่น ไฟล์ WAV จะช่วยเพิ่มการมีส่วนร่วมของผู้ใช้ได้อย่างมาก ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับกระบวนการฝังไฟล์ WAV เป็นอ็อบเจกต์ OLE (Object Linking and Embedding) ในสเปรดชีต Excel โดยใช้ Aspose.Cells สำหรับ .NET

## คู่มือการใช้งาน

### การแทรกไฟล์ WAV เป็นวัตถุ OLE

เราจะดำเนินการแต่ละขั้นตอนในการแทรกไฟล์ WAV ลงใน Excel โดยใช้ Aspose.Cells

#### 1. เตรียมไฟล์ของคุณ

ตรวจสอบว่าคุณมีไฟล์ภาพและเสียงที่จำเป็นพร้อมแล้ว:
- `sampleInsertOleObject_WAVFile.jpg` (ภาพแสดงวัตถุ OLE ของคุณ)
- `sampleInsertOleObject_WAVFile.wav` (ไฟล์เสียงจริง)

#### 2. เริ่มต้นใช้งานเวิร์กบุ๊กและเวิร์กชีต

สร้างเวิร์กบุ๊ก Excel ใหม่และเข้าถึงเวิร์กชีตแรก
```csharp
// สร้างเวิร์กบุ๊กใหม่
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

#### 3. เพิ่มวัตถุ OLE

ใช้ Aspose.Cells เพื่อเพิ่มวัตถุ OLE ที่ฝังไฟล์ WAV ของคุณ:
```csharp
// กำหนดอาร์เรย์ไบต์สำหรับข้อมูลภาพและเสียง
byte[] imageData = File.ReadAllBytes("sampleInsertOleObject_WAVFile.jpg");
byte[] objectData = File.ReadAllBytes("sampleInsertOleObject_WAVFile.wav");

// เพิ่ม Ole Object ลงในเวิร์กชีตที่เซลล์ที่ระบุ
int idx = sheet.OleObjects.Add(3, 3, 200, 220, imageData);
OleObject ole = sheet.OleObjects[idx];
```

#### 4. กำหนดค่าคุณสมบัติ OLE

ตั้งค่าคุณสมบัติต่างๆ ให้กับวัตถุที่ฝังไว้เพื่อให้แน่ใจว่าทำงานได้อย่างถูกต้อง:
```csharp
// ตั้งค่ารูปแบบไฟล์และคุณสมบัติที่จำเป็นอื่น ๆ
ole.FileFormatType = FileFormatType.Ole10Native;
ole.ObjectData = objectData;
ole.ObjectSourceFullName = "sample.wav";
ole.ProgID = "Packager Shell Object";

Guid gu = new Guid("0003000c-0000-0000-c000-000000000046");
ole.ClassIdentifier = gu.ToByteArray();
```

#### 5. บันทึกสมุดงาน

สุดท้าย ให้บันทึกสมุดงานของคุณเพื่อยืนยันการเปลี่ยนแปลง:
```csharp
// บันทึกไฟล์ Excel
workbook.Save("outputInsertOleObject_WAVFile.xlsx");
Console.WriteLine("InsertOleObject_WAVFile executed successfully.");
```
