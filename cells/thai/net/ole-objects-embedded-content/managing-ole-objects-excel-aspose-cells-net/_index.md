**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับกระบวนการในการรับและตั้งค่าตัวระบุคลาสสำหรับวัตถุ OLE แบบฝังตัว

### รับตัวระบุคลาสจากวัตถุ OLE ที่ฝังตัว

**ภาพรวม**:ฟีเจอร์นี้ช่วยให้คุณสามารถดึงข้อมูลตัวระบุที่ไม่ซ้ำกัน (GUID) ของวัตถุที่ฝังไว้เฉพาะภายในไฟล์ Excel ของคุณได้

#### ขั้นตอนที่ 1: โหลดสมุดงานของคุณ
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook wb = new Workbook(sourceDir + "sampleGetSetClassIdentifierEmbedOleObject.xls");
```

#### ขั้นตอนที่ 2: เข้าถึงเวิร์กชีตและวัตถุ OLE
```csharp
Worksheet ws = wb.Worksheets[0];
OleObject oleObj = ws.OleObjects[0];
```

#### ขั้นตอนที่ 3: แปลงเป็น GUID และพิมพ์
```csharp
Guid guid = new Guid(oleObj.ClassIdentifier);
Console.WriteLine(guid.ToString().ToUpper());
```

### ตั้งค่าตัวระบุคลาสใหม่

**ภาพรวม**: แก้ไขตัวระบุคลาสของวัตถุ OLE ที่มีอยู่หากจำเป็น

#### ขั้นตอนที่ 1: กำหนด GUID ใหม่
```csharp
string newClassId = "Your-New-GUID-Here"; // แทนที่ด้วยสตริง GUID จริง
Guid newGuid = new Guid(newClassId);
```

#### ขั้นตอนที่ 2: กำหนดและบันทึกการเปลี่ยนแปลง
```csharp
oleObj.ClassIdentifier = newGuid.ToByteArray();
wb.Save("updatedWorkbook.xls");
```
