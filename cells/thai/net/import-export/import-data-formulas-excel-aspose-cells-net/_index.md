## คู่มือการใช้งาน

หัวข้อนี้ครอบคลุมการระบุรายการข้อมูลและการนำเข้าข้อมูลดังกล่าวในเวิร์กชีต Excel โดยใช้สูตร

### การระบุรายการข้อมูล

#### ภาพรวม

การสร้างและจัดระเบียบวัตถุข้อมูลที่กำหนดเองเป็นสิ่งสำคัญก่อนนำเข้า คุณลักษณะนี้เน้นที่การกำหนดวัตถุเหล่านี้โดยใช้คลาส C#

#### การดำเนินการแบบทีละขั้นตอน

**กำหนดคลาสที่ผู้ใช้กำหนด**

```csharp
using System;
using System.Collections.Generic;

class FeatureSpecifyDataItems
{
    class DataItems
    {
        public int Number1 { get; set; }
        public int Number2 { get; set; }
        public string Formula1 { get; set; }
        public string Formula2 { get; set; }
    }

    public static void Run()
    {
        List<DataItems> dis = new List<DataItems>();

        // กำหนดรายการข้อมูล
        DataItems di = new DataItems();
        di.Number1 = 2005;
        di.Number2 = 3505;
        di.Formula1 = "+=SUM(A5,B5)"; // สูตรการหาผลรวมของ A5 และ B5
        di.Formula2 = "+=HYPERLINK(\"https://www.aspose.com\", \"เว็บไซต์ Aspose\")";

        dis.Add(di);
    }
}
```

**คำอธิบาย**- 
- การ `DataItems` คลาสเก็บจำนวนเต็มและสูตร
- สูตรถูกกำหนดให้เป็นสตริงเพื่อความยืดหยุ่นในระหว่างการนำเข้า

### การนำเข้าข้อมูลลงในเวิร์กชีตด้วยสูตร

#### ภาพรวม

คุณลักษณะนี้สาธิตการนำเข้ารายการข้อมูลที่สร้างไว้ก่อนหน้านี้ไปยังเวิร์กชีต Excel โดยระบุว่าฟิลด์ใดที่ควรได้รับการปฏิบัติเป็นสูตร

#### การดำเนินการแบบทีละขั้นตอน

**นำเข้าวัตถุที่กำหนดเอง**

```csharp
using Aspose.Cells;

class FeatureImportDataWithFormulas
{
    string outputDir = "YOUR_OUTPUT_DIRECTORY";

    public static void Run()
    {
        Workbook wb = new Workbook();
        Worksheet ws = wb.Worksheets[0];

        ImportTableOptions opts = new ImportTableOptions();
        opts.IsFormulas = new bool[] { false, false, true, true };

        List<DataItems> dis = new List<DataItems>(); // ถือว่ารายการนี้ถูกกรอกตามที่แสดงไว้ด้านบน
        
        ws.Cells.ImportCustomObjects(dis, 0, 0, opts);
        wb.CalculateFormula();
        ws.AutoFitColumns();

        wb.Save(outputDir + "/outputSpecifyFormulaFieldsWhileImportingDataToWorksheet.xlsx");
    }
}
```

**คำอธิบาย**- 
- `ImportTableOptions` ระบุว่าฟิลด์ใดเป็นสูตร
- สูตรคำนวณโดยใช้ `wb-CalculateFormula()`.
- คอลัมน์จะถูกปรับให้พอดีโดยอัตโนมัติเพื่อให้สามารถอ่านได้ดีขึ้น

สำรวจกรณีการใช้งานจริงของฟังก์ชันนี้:

1. **การรายงานทางการเงิน**เติมข้อมูลทางการเงินจากการคำนวณและลิงก์ไปยังรายงานโดยละเอียดลงในแผ่นงาน Excel โดยอัตโนมัติ
2. **การวิเคราะห์ข้อมูล**:รวมชุดข้อมูลที่กำหนดเองลงในเทมเพลตการวิเคราะห์ โดยที่สูตรจะอัปเดตผลลัพธ์โดยอัตโนมัติตามการเปลี่ยนแปลงข้อมูล
3. **การจัดการสินค้าคงคลัง**:ใช้สูตรสำหรับการคำนวณแบบไดนามิก เช่น ระดับสต๊อกสินค้าหรือจุดสั่งซื้อใหม่ภายในสเปรดชีตสินค้าคงคลัง
