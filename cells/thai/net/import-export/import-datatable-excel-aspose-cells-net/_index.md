## คู่มือการใช้งาน
คู่มือนี้แบ่งออกเป็นสองส่วนหลัก: การสร้างและการเติมข้อมูล `DataTable`ตามด้วยการนำเข้าข้อมูลนี้ลงในเวิร์กชีต Excel โดยใช้ Aspose.Cells สำหรับ .NET

### การสร้างและการเติมข้อมูลใน DataTable
#### ภาพรวม
ส่วนนี้สาธิตวิธีการสร้าง `DataTable` เพิ่มอ็อบเจ็กต์ เพิ่มคอลัมน์ และเติมแถวข้อมูลลงไป ซึ่งถือเป็นสิ่งสำคัญในการเตรียมข้อมูลก่อนส่งออกไปยัง Excel

#### ขั้นตอน:
**1. กำหนดไดเรกทอรีแหล่งที่มา**
เริ่มต้นด้วยการระบุไดเรกทอรีสำหรับไฟล์อินพุตและเอาต์พุต แม้ว่าตัวอย่างนี้จะไม่ได้ใช้โดยตรงภายในการดำเนินการเหล่านี้ก็ตาม
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**2. สร้างวัตถุ DataTable**
สร้างตัวอย่าง `DataTable` วัตถุที่มีชื่อว่า "ผลิตภัณฑ์"
```csharp
DataTable dataTable = new DataTable("Products");
```

**3. เพิ่มคอลัมน์ลงใน DataTable**
เพิ่มคอลัมน์ที่จำเป็นโดยระบุประเภทข้อมูลสำหรับแต่ละคอลัมน์
```csharp
dataTable.Columns.Add("Product ID", typeof(Int32));
dataTable.Columns.Add("Product Name", typeof(string));
dataTable.Columns.Add("Units In Stock", typeof(Int32));
```

**4. เติมแถวด้วยข้อมูล**
สร้างแถวและกำหนดค่าให้กับแถวเหล่านั้นก่อนที่จะเพิ่มลงใน `DataTable`-
```csharp
// แถวแรก
DataRow dr = dataTable.NewRow();
dr[0] = 1;
dr[1] = "Aniseed Syrup";
dr[2] = 15;
dataTable.Rows.Add(dr);

// แถวที่ 2
dr = dataTable.NewRow();
dr[0] = 2;
dr[1] = "Boston Crab Meat";
dr[2] = 123;
dataTable.Rows.Add(dr);
```

### การนำเข้า DataTable ไปยังเวิร์กชีต Excel
#### ภาพรวม
ส่วนนี้แสดงวิธีการนำเข้าข้อมูลที่มี `DataTable` ลงในเวิร์กชีต Excel โดยใช้ Aspose.Cells สำหรับ .NET แสดงให้เห็นการส่งออกข้อมูลอย่างราบรื่น

#### ขั้นตอน:
**1. เริ่มต้นใช้งานเวิร์กบุ๊กและเวิร์กชีต**
สร้างอินสแตนซ์เวิร์กบุ๊กใหม่และรับการอ้างอิงไปยังเวิร์กชีตแรก
```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**2. กำหนดค่าตัวเลือกการนำเข้า**
ตั้งค่าตัวเลือกการนำเข้าเพื่อรวมชื่อฟิลด์ในแผ่นงาน Excel
```csharp
ImportTableOptions options = new ImportTableOptions();
options.IsFieldNameShown = true;
```

**3. นำเข้าข้อมูล DataTable**
ใช้ `ImportData` วิธีการส่งออกข้อมูลเริ่มต้นจากเซลล์ A1
```csharp
worksheet.Cells.ImportData(dataTable.DefaultView, 0, 0, options);
```

**4. บันทึกไฟล์ Excel**
ระบุไดเรกทอรีเอาท์พุตและชื่อไฟล์สำหรับบันทึกเอกสาร Excel
```csharp
workbook.Save(outputDir + "output.xls");
```
