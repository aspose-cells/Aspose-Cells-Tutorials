# การกรองอัตโนมัติเริ่มต้นด้วยใน Excel

## การแนะนำ

เมื่อต้องทำงานกับข้อมูล Excel ได้รับการยอมรับว่าเป็นแอปพลิเคชันที่ได้รับความนิยมสำหรับอุตสาหกรรมและวัตถุประสงค์ต่างๆ มากมาย หนึ่งในฟีเจอร์ที่ทรงพลังที่สุดคือ AutoFilter ซึ่งทำให้การกรองข้อมูลจำนวนมากเป็นเรื่องง่าย หากคุณใช้ Aspose.Cells สำหรับ .NET คุณสามารถใช้ประโยชน์จากฟังก์ชันนี้ผ่านโปรแกรมและปรับปรุงงานจัดการข้อมูลของคุณได้อย่างมาก ในคู่มือนี้ เราจะแนะนำคุณเกี่ยวกับกระบวนการนำฟีเจอร์ที่กรองแถว Excel โดยพิจารณาว่าแถวนั้นเริ่มต้นด้วยสตริงใด

## แพ็คเกจนำเข้า

ในการเริ่มต้น ให้นำเข้าเนมสเปซที่จำเป็นที่ด้านบนของไฟล์ C# ของคุณ:

```csharp
using System.IO;
using Aspose.Cells;
using System;
```

การดำเนินการนี้จะนำเข้าฟังก์ชันการทำงานของ Aspose.Cells หลักควบคู่ไปกับฟีเจอร์พื้นฐานของระบบที่เราจะใช้ในการโต้ตอบกับคอนโซล

ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมและนำเข้าแพ็คเกจที่จำเป็นแล้ว เรามาแบ่งคุณลักษณะตัวกรองอัตโนมัติออกเป็นขั้นตอนที่จัดการได้ เราจะใช้ตัวกรองที่แยกแถวที่ขึ้นต้นด้วย "Ba"

## ขั้นตอนที่ 1: กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต

ก่อนอื่น เรามากำหนดกันก่อนว่าไฟล์ Excel อินพุตของเราอยู่ที่ไหน รวมถึงตำแหน่งที่เราต้องการบันทึกเอาต์พุตที่ผ่านการกรองแล้ว:

```csharp
// ไดเรกทอรีแหล่งที่มา
string sourceDir = "Your Document Directory\\";

// ไดเรกทอรีผลลัพธ์
string outputDir = "Your Document Directory\\";
```

คำอธิบาย : ที่นี่ให้แทนที่ `"Your Document Directory\\"` ด้วยเส้นทางจริงไปยังไดเร็กทอรีของคุณ ตรวจสอบให้แน่ใจว่าได้สิ้นสุดเส้นทางไดเร็กทอรีด้วยแบ็กสแลชคู่ (`\\`) เพื่อหลีกเลี่ยงปัญหาเส้นทางใด ๆ

## ขั้นตอนที่ 2: สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก

ต่อไปเราจะสร้างวัตถุเวิร์กบุ๊กที่ชี้ไปยังไฟล์ Excel ของเรา:

```csharp
// การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กที่มีข้อมูลตัวอย่าง
Workbook workbook = new Workbook(sourceDir + "sourseSampleCountryNames.xlsx");
```

คำอธิบาย: บรรทัดนี้จะเริ่มต้นอินสแตนซ์เวิร์กบุ๊กใหม่โดยใช้เส้นทางไฟล์ที่ระบุ `Workbook` คลาสเป็นพื้นฐานเนื่องจากแสดงถึงไฟล์ Excel ทั้งหมด

## ขั้นตอนที่ 3: การเข้าถึงแผ่นงานแรก

ตอนนี้เราต้องเข้าถึงเวิร์กชีตเฉพาะที่เราต้องการใช้งาน:

```csharp
// การเข้าถึงเวิร์กชีตแรกในไฟล์ Excel
Worksheet worksheet = workbook.Worksheets[0];
```

คำอธิบาย : `Worksheets` การรวบรวมช่วยให้เราสามารถเข้าถึงแผ่นงานแต่ละแผ่นได้โดยใช้ `[0]` อ้างอิงถึงเวิร์กชีตแรกในไฟล์ Excel ของคุณ ซึ่งโดยทั่วไปแล้วถือเป็นแนวทางปฏิบัติทั่วไปเมื่อทำงานกับไฟล์แผ่นงานเดียว

## ขั้นตอนที่ 4: การตั้งค่าตัวกรองอัตโนมัติ

นี่คือจุดเริ่มต้นของเวทมนตร์! เราจะสร้างช่วงตัวกรองอัตโนมัติสำหรับข้อมูลของเรา:

```csharp
// การสร้างตัวกรองอัตโนมัติโดยกำหนดช่วงเซลล์
worksheet.AutoFilter.Range = "A1:A18";
```

คำอธิบาย : `AutoFilter.Range` คุณสมบัตินี้ช่วยให้คุณระบุแถวที่จะกรองได้ ในกรณีนี้ เราจะกรองแถวภายในช่วง A1 ถึง A18 ซึ่งถือว่ามีข้อมูลของเราอยู่

## ขั้นตอนที่ 5: ใช้เงื่อนไขตัวกรอง

ขั้นตอนต่อไปคือการกำหนดเงื่อนไขของตัวกรอง เราต้องการแสดงเฉพาะแถวที่มีค่าคอลัมน์แรกขึ้นต้นด้วย "Ba" เท่านั้น:

```csharp
// เริ่มต้นตัวกรองสำหรับแถวที่เริ่มต้นด้วยสตริง "Ba"
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");
```

คำอธิบาย : `Custom` วิธีการนี้จะกำหนดตรรกะการกรองของเรา อาร์กิวเมนต์แรก (`0`) ระบุว่าเรากำลังกรองตามคอลัมน์แรก (A) และ `FilterOperatorType.BeginsWith` กำหนดเงื่อนไขของเราในการค้นหาแถวที่เริ่มต้นด้วย "Ba"

## ขั้นตอนที่ 6: รีเฟรชตัวกรอง

หลังจากใช้เงื่อนไขตัวกรองแล้ว เราต้องตรวจสอบให้แน่ใจว่า Excel รีเฟรชเพื่อสะท้อนการเปลี่ยนแปลง:

```csharp
// รีเฟรชตัวกรองเพื่อแสดง/ซ่อนแถวที่ถูกกรอง
worksheet.AutoFilter.Refresh();
```

คำอธิบาย: บรรทัดนี้จะเรียกใช้การรีเฟรชตัวกรองอัตโนมัติเพื่อให้แน่ใจว่าแถวที่มองเห็นได้สอดคล้องกับเกณฑ์ตัวกรองที่ใช้ ซึ่งคล้ายกับการกดปุ่มรีเฟรชใน Excel

## ขั้นตอนที่ 7: บันทึกไฟล์ Excel ที่ปรับเปลี่ยนแล้ว

ตอนนี้ถึงเวลาที่จะบันทึกการเปลี่ยนแปลงที่เราได้ทำ:

```csharp
// การบันทึกไฟล์ Excel ที่แก้ไขแล้ว
workbook.Save(outputDir + "outSourseSampleCountryNames.xlsx");
```

คำอธิบาย : `Save` วิธีการนี้จะเขียนเวิร์กบุ๊กที่แก้ไขแล้วกลับไปยังเส้นทางเอาต์พุตที่ระบุ ซึ่งถือเป็นการเขียนตัวกรองที่คุณกำหนดไว้ในไฟล์ใหม่เพื่อให้ข้อมูลเดิมของคุณยังคงอยู่เหมือนเดิม

## ขั้นตอนที่ 8: การยืนยันผลลัพธ์

สุดท้ายเรามายืนยันว่าการดำเนินการของเราประสบความสำเร็จ:

```csharp
Console.WriteLine("AutofilterBeginsWith executed successfully.\r\n");
```

คำอธิบาย: บรรทัดง่ายๆ นี้จะส่งข้อความยืนยันไปยังคอนโซล เพื่อแจ้งให้คุณทราบว่ากระบวนการกรองเสร็จสมบูรณ์โดยไม่มีข้อผิดพลาด

## บทสรุป

ในโลกที่การจัดการข้อมูลอาจดูยุ่งยาก การเรียนรู้ฟีเจอร์ต่างๆ เช่น AutoFilter ใน Excel ผ่าน Aspose.Cells สำหรับ .NET จะช่วยให้คุณจัดการข้อมูลได้อย่างมีประสิทธิภาพและประสิทธิผล คุณได้เรียนรู้วิธีการกรองแถว Excel ที่ขึ้นต้นด้วย "Ba" แล้ว โดยนำวิธีการนี้ไปใช้ทีละขั้นตอน เมื่อฝึกฝนไปเรื่อยๆ คุณจะสามารถปรับวิธีการนี้ให้เหมาะกับความต้องการในการกรองข้อมูลต่างๆ ในโครงการที่กำลังดำเนินการอยู่ของคุณได้
