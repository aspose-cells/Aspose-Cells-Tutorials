# การสร้างแผนภูมิวงกลมด้วยเส้นผู้นำโดยใช้ Aspose.Cells .NET

## การแนะนำ

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า Aspose.Cells สำหรับ .NET ในสภาพแวดล้อมของคุณ

- การบันทึกแผนภูมิเป็นรูปภาพหรือภายในเวิร์กบุ๊ก Excel

ให้แน่ใจว่าคุณมีทุกอย่างพร้อมเพื่อปฏิบัติตามอย่างมีประสิทธิภาพ

## คู่มือการใช้งาน

### การสร้างเวิร์กบุ๊กและเวิร์กชีต
1. **การเริ่มต้นสมุดงาน**
   สร้างสมุดงานใหม่ในรูปแบบ XLSX:
   ```csharp
   Workbook workbook = new Workbook(FileFormatType.Xlsx);
   ```

2. **การเข้าถึงแผ่นงานแรก**
   ใช้แผ่นงานแรกในการป้อนข้อมูล:
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **การเพิ่มข้อมูลสำหรับแผนภูมิวงกลม**
   เติมแผ่นงานของคุณด้วยหมวดหมู่และค่า:
   ```csharp
   worksheet.Cells["A1"].PutValue("Retail");
   // เพิ่มชื่อหมวดหมู่ที่เหลือ...
   worksheet.Cells["B1"].PutValue(10.4);
   // เพิ่มค่าที่สอดคล้องกัน...
   ```

### การเพิ่มแผนภูมิวงกลมลงในเวิร์กชีต
1. **สร้างแผนภูมิวงกลม**
   สร้างแผนภูมิวงกลมและเพิ่มลงในคอลเล็กชันแผนภูมิของเวิร์กชีตของคุณ:
   ```csharp
   int id = worksheet.Charts.Add(ChartType.Pie, 3, 3, 23, 13);
   ```

2. **กำหนดค่าข้อมูลซีรีส์และหมวดหมู่**
   เชื่อมโยงข้อมูลชุดและหมวดหมู่:
   ```csharp
   Chart chart = worksheet.Charts[id];
   chart.NSeries.Add("B1:B16", true);
   chart.NSeries.CategoryData = "A1:A16";
   ```

3. **ปรับแต่งป้ายข้อมูล**
   ปิดการแสดงคำอธิบาย ตั้งค่าป้ายข้อมูลเพื่อแสดงชื่อหมวดหมู่และเปอร์เซ็นต์:
   ```csharp
   chart.ShowLegend = false;
   DataLabels dataLabels = chart.NSeries[0].DataLabels;
   dataLabels.ShowCategoryName = true;
   dataLabels.ShowPercentage = true;
   dataLabels.Position = LabelPositionType.OutsideEnd;
   ```

### การนำแนวทางผู้นำมาใช้
1. **เปิดใช้งานเส้นผู้นำ**
   เปิดใช้งานเส้นผู้นำเพื่อการเชื่อมต่อภาพที่ชัดเจนยิ่งขึ้น:
   ```csharp
   chart.NSeries[0].HasLeaderLines = true;
   ```

2. **ปรับตำแหน่งป้ายข้อมูล**
   เพิ่มความชัดเจนด้วยการปรับตำแหน่งฉลาก:
   ```csharp
   int DELTA = 100;
   foreach (var point in chart.NSeries[0].Points)
   {
       int X = point.DataLabels.X;
       if (X > 2000) 
           point.DataLabels.X += DELTA;
       else 
           point.DataLabels.X -= DELTA;
   }
   ```

### การบันทึกแผนภูมิและสมุดงาน
1. **บันทึกเป็นรูปภาพ**
   เรนเดอร์แผนภูมิเป็นไฟล์รูปภาพ:
   ```csharp
   ImageOrPrintOptions options = new ImageOrPrintOptions { ImageType = Drawing.ImageType.Png, HorizontalResolution = 200, VerticalResolution = 200 };
   chart.ToImage("output_out.png", options);
   ```

2. **บันทึกสมุดงาน**
   บันทึกเวิร์กบุ๊กเพื่อดูแผนภูมิภายใน Excel:
   ```csharp
   workbook.Save("output_out.xlsx");
   ```

## การประยุกต์ใช้งานจริง
- **รายงานทางการเงิน**:แสดงการจัดสรรงบประมาณอย่างชัดเจน
- **การวิเคราะห์การตลาด**:แสดงภาพข้อมูลส่วนแบ่งการตลาดอย่างมีประสิทธิผลในงานนำเสนอหรือรายงาน
- **การวิเคราะห์การขาย**:แสดงการกระจายการขายในแต่ละภูมิภาค/ผลิตภัณฑ์ได้อย่างง่ายดาย

ความเป็นไปได้ในการบูรณาการได้แก่ การส่งออกภาพข้อมูลเหล่านี้ไปยังแอปพลิเคชันเว็บหรือฝังไว้ในเครื่องมือสร้างรายงานอัตโนมัติ

- ใช้ลูปที่มีประสิทธิภาพและหลีกเลี่ยงการคำนวณที่ไม่จำเป็นภายในลูป
- ทำความสะอาดทรัพยากร เช่น วัตถุสมุดงานเป็นประจำเพื่อป้องกันการรั่วไหลของหน่วยความจำ
