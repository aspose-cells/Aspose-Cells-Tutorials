## คู่มือการใช้งาน

### คุณสมบัติ 1: แทรกข้อมูลลงในเวิร์กชีต Excel

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแทรกข้อมูลตามคอลัมน์ลงในเวิร์กชีต Excel โดยใช้ Aspose.Cells

#### กระบวนการทีละขั้นตอน

##### การตั้งค่าเวิร์กบุ๊กและเวิร์กชีต

เริ่มต้นด้วยการสร้างเวิร์กบุ๊กใหม่และเข้าถึงแผ่นงานแรก:

```csharp
var book = new Workbook();
var sheet = book.Worksheets[0];
var cells = sheet.Cells;
```

##### การแทรกข้อมูลแบบคอลัมน์

เติมข้อมูลในเวิร์กชีตของคุณโดยใช้ `PutValue` วิธีการนี้มีประสิทธิภาพสำหรับการป้อนข้อมูลแบบคอลัมน์

```csharp
// แทรกข้อมูลหมวดหมู่ลงในคอลัมน์ A
cells["A1"].PutValue("Category");
cells["A2"].PutValue("Fruit");
cells["A3"].PutValue("Fruit");
cells["A4"].PutValue("Fruit");
cells["A5"].PutValue("Fruit");
cells["A6"].PutValue("Vegetables");
// ดำเนินการเพิ่มข้อมูลตามต้องการ...

// แทรกข้อมูลอาหารลงในคอลัมน์ B
cells["B1"].PutValue("Food");
cells["B2"].PutValue("Apple");
// เพิ่มรายการที่เหลือในลักษณะเดียวกัน...

// แทรกข้อมูลต้นทุนลงในคอลัมน์ C
cells["C1"].PutValue("Cost");
cells["C2"].PutValue(2.2);
// ดำเนินการเพิ่มต้นทุนต่อไป...

// แทรกข้อมูลกำไรลงในคอลัมน์ D
cells["D1"].PutValue("Profit");
cells["D2"].PutValue(0.1);
// ต่อกำไรกันต่อ...
```

### คุณสมบัติที่ 2: สร้าง ListObject ในเวิร์กชีต

ListObjects มีวิธีการจัดการช่วงข้อมูลอย่างมีประสิทธิภาพ โดยเฉพาะอย่างยิ่งเมื่อจัดการกับตาราง

#### การสร้าง ListObject จากช่วงข้อมูล

ระบุช่วงที่มีส่วนหัวและข้อมูลของคุณ:

```csharp
var listObjects = sheet.ListObjects;
// เพิ่มรายการตามช่วงแหล่งข้อมูลโดยเปิดใช้งานส่วนหัว
int index = listObjects.Add(0, 0, 11, 3, true);
sheet.AutoFitColumns();
```

### คุณสมบัติที่ 3: สร้างแผนภูมิจากข้อมูลในเวิร์กชีต

การสร้างภาพข้อมูลของคุณเป็นสิ่งสำคัญสำหรับการวิเคราะห์ มาสร้างแผนภูมิคอลัมน์โดยใช้ Aspose.Cells กัน

#### การเพิ่มแผนภูมิคอลัมน์

เลือกช่วงที่มีข้อมูลของคุณและเพิ่มวัตถุแผนภูมิใหม่:

```csharp
index = sheet.Charts.Add(ChartType.Column, 21, 1, 35, 18);
var chart = sheet.Charts[index];
chart.SetChartDataRange("A1:D12", true);
chart.NSeries.CategoryData = "A2:B12";
```

### คุณสมบัติที่ 4: บันทึกไฟล์ Excel

สุดท้าย ให้บันทึกสมุดงานของคุณไปยังไดเร็กทอรีที่ระบุ:

```csharp
book.Save(outputDir + "/output_out.xlsx");
```
