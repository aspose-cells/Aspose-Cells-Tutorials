การสร้างแผนภูมิที่ดึงดูดสายตาและเกี่ยวข้องกับวัฒนธรรมถือเป็นสิ่งสำคัญเมื่อต้องนำเสนอข้อมูลในกลุ่มผู้ชมที่หลากหลาย บทช่วยสอนนี้จะกล่าวถึงการปรับแต่งป้ายชื่อแผนภูมิใน Excel โดยใช้ Aspose.Cells สำหรับ .NET ทำให้คุณสามารถปรับแต่งแผนภูมิสำหรับกลุ่มภาษาต่างๆ ได้อย่างราบรื่น

ในคู่มือนี้ เราจะมาสำรวจวิธีการใช้ Aspose.Cells ซึ่งเป็นไลบรารีอันทรงพลังที่ช่วยลดความซับซ้อนของงานอัตโนมัติของ Excel เพื่อปรับแต่งป้ายชื่อแผนภูมิวงกลมด้วยคำศัพท์เฉพาะทางวัฒนธรรม เมื่ออ่านบทช่วยสอนนี้จบ คุณจะ:

พร้อมที่จะเปลี่ยนแผนภูมิ Excel ของคุณให้เป็นภาพที่น่าสนใจสำหรับทั่วโลกหรือยัง มาเริ่มกันเลย!

## คู่มือการใช้งาน
### การปรับแต่งป้ายแผนภูมิตามตำแหน่ง
วัตถุประสงค์หลักคือการแทนที่ข้อความเริ่มต้นสำหรับป้ายแผนภูมิวงกลมโดยใช้การตั้งค่าเฉพาะวัฒนธรรม นี่คือวิธีที่คุณสามารถทำสิ่งนี้ได้:

#### 1. โหลดสมุดงานของคุณและเข้าถึงแผนภูมิ
เริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีอยู่ซึ่งประกอบด้วยแผนภูมิวงกลม:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleCustomTextForLabels.xlsx");
```

เข้าถึงแผ่นงานและแผนภูมิที่คุณต้องการปรับแต่ง:
```csharp
Worksheet sheet = book.Worksheets[0];
Chart chart = sheet.Charts[0];
```

#### 2. ตั้งค่าการตั้งค่าสากล
การแทนที่ `GetOtherName` วิธีการจัดทำป้ายชื่อแบบกำหนดเองตามตำแหน่งของระบบ:

```csharp
GlobalizationSettings globalSettings = new GlobalizationSettings();
globalSettings.ChartSettings = new CustomSettings();
book.Settings.GlobalizationSettings = globalSettings;
```

กำหนดคลาสการตั้งค่าที่กำหนดเองของคุณ:
```csharp
class CustomSettings : ChartGlobalizationSettings
{
    public override string GetOtherName()
    {
        int lcid = System.Globalization.CultureInfo.CurrentCulture.LCID;
        switch (lcid)
        {
            case 1033: // ภาษาอังกฤษ
                return "Other";
            case 1036: // ภาษาฝรั่งเศส
                return "Autre";
            case 1031: // เยอรมัน
                return "Andere";
            default:
                return base.GetOtherName();
        }
    }
}
```

#### 3. รีเฟรชและเรนเดอร์แผนภูมิ
หากต้องการนำการเปลี่ยนแปลงของคุณไปใช้ ให้รีเฟรชแผนภูมิและเรนเดอร์เป็นไฟล์รูปภาพ:

```csharp
chart.Calculate();
chart.ToImage(outputDir + "outputCustomTextForLabels.png", new ImageOrPrintOptions());
Console.WriteLine("CustomTextForLabels executed successfully.");
```

- **ความไม่ตรงกันทางวัฒนธรรม**: ตรวจสอบว่าการตั้งค่าตำแหน่งของระบบของคุณตรงกับที่คุณกำหนดเป้าหมาย
