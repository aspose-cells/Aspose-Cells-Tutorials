## คู่มือการใช้งาน

### การเปลี่ยนแปลงแหล่งข้อมูลแผนภูมิ

ฟีเจอร์นี้ช่วยให้คุณสามารถปรับเปลี่ยนแหล่งที่มาข้อมูลของแผนภูมิในเวิร์กบุ๊ก Excel ได้อย่างง่ายดาย

#### ภาพรวม
ในส่วนนี้ เราจะสาธิตวิธีการเปลี่ยนแหล่งข้อมูลโดยใช้ Aspose.Cells คุณจะได้เรียนรู้วิธีโหลดเวิร์กบุ๊กที่มีอยู่ เข้าถึงเวิร์กชีต และอัปเดตแผนภูมิ

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก**

ขั้นแรก ให้เริ่มต้นของคุณ `Workbook` วัตถุโดยการโหลดไฟล์ที่มีอยู่:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook(SourceDir + "/sampleChangeChartDataSource.xlsx");
```

**ขั้นตอนที่ 2: เข้าถึงและกำหนดค่าเวิร์กชีต**

เข้าถึงแผ่นงานแหล่งที่มาที่คุณจะคัดลอกข้อมูล:
```csharp
Worksheet source = wb.Worksheets[0];
Worksheet destination = wb.Worksheets.Add("DestSheet");

CopyOptions options = new CopyOptions();
options.ReferToDestinationSheet = true;

destination.Cells.CopyRows(source.Cells, 0, 0, source.Cells.MaxDisplayRange.RowCount, options);
```

**ขั้นตอนที่ 3: บันทึกสมุดงาน**

สุดท้ายให้บันทึกสมุดงานของคุณด้วยข้อมูลที่อัปเดต:
```csharp
wb.Save(outputDir + "/outputChangeChartDataSource.xlsx", SaveFormat.Xlsx);
```

### โหลดและเข้าถึงสมุดงาน Excel
การเข้าถึงเวิร์กบุ๊กที่มีอยู่เป็นเรื่องง่ายด้วย Aspose.Cells

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊กที่มีอยู่**
โหลดเวิร์กบุ๊กเพื่อเข้าถึงเวิร์กชีตของมัน:
```csharp
Workbook wb = new Workbook(SourceDir + "/sampleChangeChartDataSource.xlsx");
Worksheet sourceSheet = wb.Worksheets[0];
```

### เพิ่มและกำหนดค่าแผ่นงาน
การเพิ่มและกำหนดค่าเวิร์กชีตเป็นสิ่งสำคัญสำหรับการจัดการข้อมูล

**ขั้นตอนที่ 1: สร้างสมุดงานใหม่**
เริ่มต้นอินสแตนซ์เวิร์กบุ๊กใหม่:
```csharp
Workbook wb = new Workbook();
Worksheet destination = wb.Worksheets.Add("DestSheet");
```

**ขั้นตอนที่ 2: คัดลอกข้อมูลด้วยตัวเลือก**
ใช้ประโยชน์ `CopyOptions` เพื่อจัดการวิธีการคัดลอกข้อมูล:
```csharp
CopyOptions options = new CopyOptions();
options.ReferToDestinationSheet = true;
destination.Cells.CopyRows(source.Cells, 0, 0, source.Cells.MaxDisplayRange.RowCount, options);
```

**ขั้นตอนที่ 3: บันทึกสมุดงานใหม่**
บันทึกการเปลี่ยนแปลงของคุณลงในไฟล์:
```csharp
wb.Save(outputDir + "/outputWorkbook.xlsx", SaveFormat.Xlsx);
```
