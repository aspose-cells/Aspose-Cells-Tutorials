## การตั้งค่า Aspose.Cells สำหรับ .NET

หากต้องการติดตั้ง Aspose.Cells ให้ใช้ **.NET CLI** หรือ **คอนโซลตัวจัดการแพ็คเกจ**-

```bash

```

หรือในคอนโซลตัวจัดการแพ็คเกจ:

## คู่มือการใช้งาน

ในส่วนนี้ เราจะแนะนำขั้นตอนต่างๆ ที่จำเป็นในการอ่านและจัดการแผนภูมิในไฟล์ Excel 2016

### การเข้าถึงแผนภูมิในเวิร์กชีต

เราเริ่มต้นด้วยการโหลดเวิร์กบุ๊กต้นฉบับของเราและเข้าถึงเวิร์กชีตแรกซึ่งประกอบด้วยแผนภูมิของเรา:

```csharp
// โหลดไฟล์ Excel
Workbook book = new Workbook("sampleReadAndManipulateExcel2016Charts.xlsx");

// เข้าถึงแผ่นงานแรก
Worksheet sheet = book.Worksheets[0];
```

### แผนภูมิการอ่านประเภทต่างๆ

ต่อไปเราจะวนซ้ำผ่านแผนภูมิแต่ละรายการในเวิร์กชีตเพื่ออ่านประเภทของแผนภูมิและพิมพ์ออกมา:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    // รับแผนภูมิปัจจุบัน
    Chart ch = sheet.Charts[i];

    // พิมพ์ประเภทแผนภูมิ
    Console.WriteLine(ch.Type);
}
```

### การแก้ไขชื่อแผนภูมิ

เราสามารถเปลี่ยนชื่อแผนภูมิแต่ละรายการเพื่อสะท้อนถึงประเภทของแผนภูมินั้นๆ ได้:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    Chart ch = sheet.Charts[i];

    // อัปเดตชื่อแผนภูมิ
    ch.Title.Text = "Chart Type is " + ch.Type.ToString();
}
```

### การบันทึกการเปลี่ยนแปลง

สุดท้าย ให้บันทึกการปรับเปลี่ยนของคุณกลับไปยังไฟล์ Excel ใหม่:

```csharp
book.Save("outputReadAndManipulateExcel2016Charts.xlsx");
Console.WriteLine("Manipulation completed successfully.");
```
