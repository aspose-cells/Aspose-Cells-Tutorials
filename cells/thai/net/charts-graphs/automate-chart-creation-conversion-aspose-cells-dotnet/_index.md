## การตั้งค่า Aspose.Cells สำหรับ .NET
ในการเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าได้ติดตั้ง Aspose.Cells ไว้ในโปรเจ็กต์ของคุณแล้ว หากยังไม่ได้ติดตั้ง ให้ใช้หนึ่งในวิธีการติดตั้งแพ็คเกจที่กล่าวถึงข้างต้น เมื่อติดตั้งแล้ว ให้เริ่มต้นวัตถุ Workbook เพื่อโฮสต์ข้อมูลและแผนภูมิของคุณ

### การเริ่มต้นและการตั้งค่าเบื้องต้น
```csharp
using Aspose.Cells;

// สร้างอินสแตนซ์เวิร์กบุ๊กใหม่
Workbook workbook = new Workbook();
```
การเริ่มต้นใช้งานนี้จะตั้งค่าสมุดงานว่างสำหรับการเพิ่มแผ่นงานและข้อมูล

## คู่มือการใช้งาน
เราจะแบ่งการใช้งานออกเป็นคุณสมบัติที่แตกต่างกันเพื่อความชัดเจน

### การตั้งค่าเส้นทางไดเรกทอรี
ก่อนที่จะจัดการไฟล์ใดๆ ให้กำหนดไดเร็กทอรีแหล่งที่มาและเอาต์พุตของคุณ:
```csharp
using System.IO;

string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // แทนที่ด้วยเส้นทางจริง
string OutputDir = @"YOUR_OUTPUT_DIRECTORY"; // แทนที่ด้วยเส้นทางจริง
```
การตั้งค่านี้จะช่วยให้แน่ใจว่าแหล่งที่มาของข้อมูลถูกระบุตำแหน่งอย่างถูกต้องและไฟล์เอาท์พุตจะถูกบันทึกไว้ในไดเร็กทอรีที่ต้องการ

### การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก
ตามที่แสดงไว้ก่อนหน้านี้ การสร้าง `Workbook` วัตถุนี้ตรงไปตรงมา วัตถุนี้จะโฮสต์เวิร์กชีต ข้อมูล และแผนภูมิของคุณ

### การเพิ่มเวิร์กชีตและการเติมข้อมูล
หากต้องการแสดงข้อมูลผ่านแผนภูมิ ขั้นแรกให้ป้อนข้อมูลลงในเวิร์กชีต:
```csharp
// เพิ่มแผ่นงานใหม่ลงในสมุดงาน
int sheetIndex = workbook.Worksheets.Add();

// รับการอ้างอิงถึงแผ่นงานที่เพิ่มใหม่
Worksheet worksheet = workbook.Worksheets[sheetIndex];

// เติมค่าตัวอย่างลงในเซลล์
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].putValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```

### การเพิ่มและการกำหนดค่าแผนภูมิ
ตอนนี้เรามาเพิ่มแผนภูมิลงในเวิร์กชีตกัน:
```csharp
// เพิ่มแผนภูมิคอลัมน์ลงในเวิร์กชีตในตำแหน่งที่ระบุ
int chartIndex = worksheet.Charts.Add(Aspose.Cells.Charts.ChartType.Column, 5, 0, 15, 5);

// เข้าถึงอินสแตนซ์แผนภูมิที่เพิ่มใหม่
Chart chart = worksheet.Charts[chartIndex];

// ตั้งค่าช่วงข้อมูลสำหรับคอลเลกชันชุดแผนภูมิ (A1 ถึง B3)
chart.NSeries.Add("A1:B3", true);
```
ที่นี่ เราเพิ่มแผนภูมิคอลัมน์และกำหนดค่าช่วงข้อมูลเพื่อแสดงข้อมูลของคุณอย่างถูกต้องแม่นยำ

### การแปลงแผนภูมิเป็นรูปภาพ
สุดท้ายแปลงแผนภูมิเป็นไฟล์รูปภาพ:
```csharp
using System.Drawing.Imaging;

// แปลงแผนภูมิเป็นไฟล์รูปภาพในรูปแบบ EMF และบันทึกไว้
string outputPath = Path.Combine(OutputDir, "Chart.emf");
chart.ToImage(outputPath, ImageFormat.Emf);
```
การแปลงนี้ช่วยให้สามารถแชร์หรือฝังแผนภูมิในรายงานได้อย่างง่ายดาย
