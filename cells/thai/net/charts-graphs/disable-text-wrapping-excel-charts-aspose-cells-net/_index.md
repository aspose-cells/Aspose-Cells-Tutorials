## คู่มือการใช้งาน

เมื่อตั้งค่าทุกอย่างเรียบร้อยแล้ว ให้ปิดการใช้งานการห่อข้อความในป้ายข้อมูลแผนภูมิ Excel โดยใช้ Aspose.Cells สำหรับ .NET

### การโหลดและการเข้าถึงสมุดงาน
โหลดไฟล์ Excel ของคุณลงใน `Workbook` วัตถุ:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// โหลดไฟล์ตัวอย่าง Excel ไว้ในวัตถุเวิร์กบุ๊ก
Workbook workbook = new Workbook(SourceDir + "/sampleDisableTextWrappingForDataLabels.xlsx");
```

### การเข้าถึงแผ่นงานและแผนภูมิ
เข้าถึงแผ่นงานและแผนภูมิเฉพาะที่คุณต้องการปรับเปลี่ยน:
```csharp
// เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
Worksheet worksheet = workbook.Worksheets[0];

// เข้าถึงแผนภูมิแรกในเวิร์กชีต
Chart chart = worksheet.Charts[0];
```

### การปิดใช้งานการห่อข้อความสำหรับป้ายข้อมูล
ปิดใช้งานการห่อข้อความโดยการตั้งค่า `IsTextWrapped` เป็นเท็จ:
```csharp
foreach (var series in chart.NSeries)
{
    // ตั้งค่า IsTextWrapped เป็นเท็จเพื่อปิดการใช้งานการห่อข้อความ
    series.DataLabels.IsTextWrapped = false;
}
```

### การบันทึกสมุดงานที่แก้ไข
บันทึกการเปลี่ยนแปลงของคุณโดยการเขียนเวิร์กบุ๊กที่แก้ไขไปยังไฟล์ใหม่:
```csharp
// บันทึกสมุดงานที่มีการเปลี่ยนแปลงไปยังไฟล์ใหม่
workbook.Save(outputDir + "/outputDisableTextWrappingForDataLabels.xlsx");
```
