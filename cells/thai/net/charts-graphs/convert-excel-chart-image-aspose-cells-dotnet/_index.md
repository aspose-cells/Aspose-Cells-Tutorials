## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่า Aspose.Cells เรียบร้อยแล้ว มาดำเนินการใช้งานกันเลย

### ขั้นตอนที่ 1: เปิดไฟล์ Excel

ขั้นแรก เราต้องเปิดไฟล์ Excel ที่มีแผนภูมิของคุณ:

```csharp
// เปิดไฟล์ Excel ที่มีอยู่ซึ่งประกอบด้วยแผนภูมิคอลัมน์
Workbook workbook = new Workbook("sampleConvertingColumnChartToImage.xlsx");
```

สไนปเป็ตนี้จะสร้าง `Workbook` สร้างอ็อบเจ็กต์โดยโหลดไฟล์ Excel ตรวจสอบว่า "sampleConvertingColumnChartToImage.xlsx" อยู่ในไดเร็กทอรีของโครงการของคุณหรือระบุเส้นทางแบบสัมบูรณ์

### ขั้นตอนที่ 2: การเข้าถึงแผนภูมิ

ขั้นตอนต่อไปคือเข้าถึงแผนภูมิที่คุณต้องการแปลง:

```csharp
Worksheet ws = workbook.Worksheets[0];
Aspose.Cells.Charts.Chart chart = ws.Charts[0];
```

ที่นี่ เราถือว่าแผนภูมิอยู่ในเวิร์กชีตแรกและเป็นแผนภูมิแรกในชีตนั้น ปรับดัชนีตามโครงสร้างไฟล์เฉพาะของคุณ

### ขั้นตอนที่ 3: การแปลงแผนภูมิเป็นรูปภาพ

แปลงแผนภูมิเป็นรูปแบบภาพ:

```csharp
chart.ToImage("outputConvertingColumnChartToImage.jpeg", System.Drawing.Imaging.ImageFormat.Jpeg);
```

โค้ดนี้จะแปลงแผนภูมิแรกที่พบในเวิร์กบุ๊กเป็นรูปภาพ JPEG คุณสามารถเปลี่ยน "jpeg" เป็นรูปแบบอื่น เช่น PNG หากจำเป็น

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ Excel ของคุณถูกต้อง
