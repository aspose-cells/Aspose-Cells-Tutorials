## คู่มือการใช้งาน
มาแยกขั้นตอนการแปลงแผนภูมิวงกลมเป็นรูปภาพกัน

### การเปิดและการเข้าถึงไฟล์ Excel
หากต้องการแปลงแผนภูมิวงกลมจากไฟล์ Excel ก่อนอื่นคุณต้องเปิดไฟล์ดังกล่าว:
1. **ตั้งค่าไดเรกทอรีต้นทางและปลายทาง**-
   - กำหนดเส้นทางสำหรับแหล่งที่มา (ไฟล์ Excel) และไดเร็กทอรีเอาต์พุต
   
    ```csharp
    string sourceDir = RunExamples.Get_SourceDirectory();
    string outputDir = RunExamples.Get_OutputDirectory();
    ```
2. **โหลดสมุดงาน**-
   - ใช้ Aspose.Cells เพื่อโหลดเวิร์กบุ๊ก Excel ของคุณ

    ```csharp
    Workbook workbook = new Workbook(sourceDir + "sampleConvertingPieChartToImageFile.xlsx");
    Worksheet ws = workbook.Worksheets[0];
    ```

### การเข้าถึงและการแปลงแผนภูมิวงกลม
ตอนนี้คุณเข้าถึงเวิร์กชีตของคุณได้แล้ว มาแปลงแผนภูมิกัน:
1. **ดึงข้อมูลแผนภูมิ**-
   - ระบุแผนภูมิวงกลมในเวิร์กชีตของคุณ

    ```csharp
    Aspose.Cells.Charts.Chart chart = ws.Charts[0];
    ```
2. **แปลงแผนภูมิเป็นรูปภาพ**-
   - บันทึกแผนภูมิวงกลมเป็นไฟล์รูปภาพโดยใช้ `ToImage` วิธี.

    ```csharp

    ```
