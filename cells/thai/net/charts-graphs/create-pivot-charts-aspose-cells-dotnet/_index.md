## คู่มือการใช้งาน

ในส่วนนี้แบ่งคุณลักษณะแต่ละอย่างออกเป็นขั้นตอนที่จัดการได้ ช่วยให้คุณสร้างและกำหนดค่าแผนภูมิหมุนได้อย่างมีประสิทธิภาพ

### สร้างตัวอย่างและเปิดสมุดงาน

#### ภาพรวม
การสร้างใหม่ `Workbook` วัตถุเป็นขั้นตอนแรกในการจัดการไฟล์ Excel ด้วยโปรแกรม

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊กที่มีอยู่**
```csharp
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string fileName = "sampleCreatePivotChart.xlsx";

// สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กด้วยเส้นทางไปยังไฟล์ Excel ของคุณ
Workbook workbook = new Workbook(sourceDir + fileName);
```

- **พารามิเตอร์:** ผู้สร้างจะใช้เส้นทางไฟล์ของเอกสาร Excel
- **วัตถุประสงค์:** ขั้นตอนนี้เป็นการเตรียมสมุดงานสำหรับการดำเนินการเพิ่มเติม เช่น การเพิ่มแผ่นงานหรือแผนภูมิ

### เพิ่มและตั้งชื่อชีตใหม่

#### ภาพรวม
การเพิ่มแผ่นงานแผนภูมิถือเป็นสิ่งสำคัญในการโฮสต์แผนภูมิสรุปข้อมูล คุณสามารถทำได้ดังนี้:

**ขั้นตอนที่ 2: สร้างแผ่นงานแผนภูมิใหม่**
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// การเพิ่มแผ่นงานแผนภูมิใหม่ชื่อ 'PivotChart'
Worksheet sheet3 = workbook.Worksheets[workbook.Worksheets.Add(SheetType.Chart)];
sheet3.Name = "PivotChart";
```

- **พารามิเตอร์:** `SheetType.Chart` ระบุชนิดของแผ่นงาน
- **วัตถุประสงค์:** ขั้นตอนนี้จะเพิ่มพื้นที่เฉพาะสำหรับแผนภูมิหมุนของคุณ โดยมีการตั้งชื่อเพื่อให้ระบุได้ง่าย

### เพิ่มและกำหนดค่าแผนภูมิคอลัมน์

#### ภาพรวม
หากต้องการเพิ่มแผนภูมิคอลัมน์ที่ทำหน้าที่เป็นแผนภูมิหมุน ให้ทำตามขั้นตอนเหล่านี้:

**ขั้นตอนที่ 3: แทรกและกำหนดค่าแผนภูมิสรุปข้อมูล**
```csharp
Worksheet sheet3 = workbook.Worksheets[0];

// การเพิ่มแผนภูมิคอลัมน์ในตำแหน่งที่ระบุในเวิร์กชีต
int index = sheet3.Charts.Add(ChartType.Column, 0, 5, 28, 16);

// การตั้งค่าแหล่งข้อมูลสำหรับแผนภูมิสรุปข้อมูลเป็น 'PivotTable1'
sheet3.Charts[index].PivotSource = "PivotTable!PivotTable1";

// การกำหนดค่าว่าจะซ่อนปุ่มฟิลด์ Pivot หรือไม่ (ตั้งค่าเป็นเท็จที่นี่)
sheet3.Charts[index].HidePivotFieldButtons = false;
```

- **พารามิเตอร์:** การ `Add` วิธีการนี้ต้องใช้ประเภทและตำแหน่งของแผนภูมิ
- **วัตถุประสงค์:** การดำเนินการนี้จะสร้างแผนภูมิที่เชื่อมโยงกับตารางสรุปข้อมูล ซึ่งช่วยให้แสดงข้อมูลแบบไดนามิกได้

### บันทึกสมุดงาน

#### ภาพรวม
สุดท้าย ให้บันทึกการเปลี่ยนแปลงของคุณเพื่อคงไว้ในไฟล์ Excel

**ขั้นตอนที่ 4: บันทึกสมุดงานของคุณ**
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// บันทึกสมุดงานที่แก้ไขไปยังไดเร็กทอรีที่ระบุ
workbook.Save(outputDir + "outputCreatePivotChart.xlsx");
```

- **พารามิเตอร์:** การ `Save` วิธีนี้จะนำพาไปยังเส้นทางที่คุณต้องการเก็บไฟล์ Excel ของคุณ
- **วัตถุประสงค์:** ขั้นตอนนี้จะช่วยให้แน่ใจว่าการปรับเปลี่ยนทั้งหมดของคุณได้รับการจัดเก็บและสามารถเข้าถึงหรือแชร์ได้ตามต้องการ
