## คู่มือการใช้งาน

### การเปิดไฟล์ที่คั่นด้วยแท็บ
#### ภาพรวม
หัวข้อนี้สาธิตวิธีเปิดไฟล์ที่คั่นด้วยแท็บโดยใช้ Aspose.Cells เราจะมาสำรวจการกำหนดค่าที่จำเป็นและทำความเข้าใจบทบาทของแต่ละพารามิเตอร์

#### การดำเนินการแบบทีละขั้นตอน
1. **ระบุตัวเลือกการโหลด**
   ระบุว่าไฟล์ของคุณอยู่ในรูปแบบที่คั่นด้วยแท็บโดยใช้ `LoadOptions`-
   ```csharp
   // ระบุตัวเลือกการโหลดสำหรับไฟล์ที่คั่นด้วยแท็บ
   LoadOptions loadOptions = new LoadOptions(LoadFormat.TabDelimited);
   ```

2. **สร้างและเปิดสมุดงาน**
   ใช้ตัวเลือกโหลดที่ระบุเพื่อสร้าง `Workbook` วัตถุ.
   ```csharp
   string dataDir = "path_to_your_directory"; // อัพเดตเส้นทางนี้

   // สร้างเวิร์กบุ๊กด้วยไฟล์ที่คั่นด้วยแท็บ
   Workbook workbook = new Workbook(dataDir + "Book1TabDelimited.txt", loadOptions);

   Console.WriteLine("Tab delimited file opened successfully!");
   ```

#### คำอธิบายพารามิเตอร์
- **LoadFormat.TabDelimited**: ระบุรูปแบบของไฟล์อินพุต
- **ไดเรกทอรีข้อมูล + "Book1TabDelimited.txt"**: เส้นทางไปยังไฟล์ที่คั่นด้วยแท็บของคุณ

### ตัวเลือกการกำหนดค่าคีย์
คุณสามารถปรับแต่งเพิ่มเติมว่า Aspose.Cells จัดการไฟล์ของคุณอย่างไรโดยใช้วิธีต่างๆ `LoadOptions`ตัวอย่างเช่น ระบุตัวคั่นแบบกำหนดเองหากข้อมูลของคุณไม่ได้ถูกคั่นด้วยแท็บอย่างเคร่งครัดหรือจัดการการเข้ารหัสเฉพาะ
