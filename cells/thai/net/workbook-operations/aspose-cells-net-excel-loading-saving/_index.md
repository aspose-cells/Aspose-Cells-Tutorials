## คู่มือการใช้งาน

### การโหลดไฟล์ Excel
**ภาพรวม**:เรียนรู้วิธีโหลดไฟล์ Excel ที่มีอยู่ลงใน `Workbook` วัตถุ ซึ่งถือเป็นสิ่งสำคัญสำหรับการประมวลผลและจัดการข้อมูลที่จัดเก็บในไฟล์ Excel

#### การดำเนินการแบบทีละขั้นตอน
1. **ตั้งค่าไดเรกทอรีแหล่งที่มา**
   กำหนดว่าไฟล์ Excel ต้นฉบับของคุณอยู่ที่ใด:
   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   ```
2. **โหลดไฟล์ Excel**
   เริ่มต้น `Workbook` วัตถุโดยการโหลดไฟล์ Excel ที่มีอยู่:
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "sampleExpandTextFromRightToLeft.xlsx");
   ```
   - **คำอธิบาย**: เดอะ `Workbook` constructor จะนำเส้นทางของไฟล์และโหลดเข้าสู่หน่วยความจำ ทำให้คุณสามารถโต้ตอบกับเนื้อหาผ่านโปรแกรมได้

### การบันทึกสมุดงานเป็น HTML
**ภาพรวม**:ส่วนนี้จะแสดงตัวอย่างการบันทึกเวิร์กบุ๊ก Excel ในรูปแบบ HTML ซึ่งมีประโยชน์อย่างยิ่งสำหรับแอปพลิเคชันเว็บที่ต้องการนำเสนอข้อมูลภายนอกซอฟต์แวร์ Excel

#### การดำเนินการแบบทีละขั้นตอน
1. **ตั้งค่าไดเรกทอรีเอาต์พุต**
   กำหนดตำแหน่งที่จะบันทึกไฟล์ HTML เอาท์พุตของคุณ:
   ```csharp
   string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
   ```
2. **บันทึกสมุดงานในรูปแบบ HTML**
   บันทึกสมุดงานโดยมีข้อมูลเวอร์ชันผนวกเข้ากับชื่อไฟล์:
   ```csharp
   workbook.Save(OutputDir + "outputExpandTextFromRightToLeft_" + CellsHelper.GetVersion() + ".html", SaveFormat.Html);
   ```
   - **คำอธิบาย**: เดอะ `Save` วิธีการเขียนเนื้อหาของสมุดงานลงในไฟล์ HTML `CellsHelper.GetVersion()` ให้รายละเอียดเวอร์ชันเพื่อแยกแยะเอาท์พุตที่แตกต่างกัน

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกระบุอย่างถูกต้องและมีไดเร็กทอรีอยู่
- ตรวจสอบว่าโครงการของคุณมีใบอนุญาตที่ถูกต้องหากคุณพบข้อจำกัดกับเวอร์ชันทดลองใช้

2. **ระบบจัดการเนื้อหา**:ใช้การส่งออก HTML เพื่อแสดงเนื้อหาบนเว็บไซต์
