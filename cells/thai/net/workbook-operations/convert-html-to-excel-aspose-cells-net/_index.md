## คู่มือการใช้งาน

ในส่วนนี้จะแบ่งกระบวนการออกเป็นสองคุณลักษณะหลัก: การโหลดไฟล์ HTML และการกำหนดค่าตัวเลือกในการโหลด

### คุณสมบัติ 1: การโหลดและบันทึกไฟล์ HTML เป็น Excel

#### ภาพรวม:
แปลงไฟล์ HTML ที่มีอยู่ให้เป็นเวิร์กบุ๊ก Excel ที่มีฟังก์ชันครบถ้วนโดยใช้ Aspose.Cells สำหรับ .NET 

##### การดำเนินการทีละขั้นตอน:

**1. ตั้งค่าไดเร็กทอรีแหล่งที่มาและผลลัพธ์:**
เริ่มต้นด้วยการกำหนดไดเร็กทอรีที่ไฟล์ HTML ต้นฉบับของคุณตั้งอยู่และตำแหน่งที่คุณต้องการบันทึกไฟล์ Excel เอาท์พุต

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**2. กำหนดเส้นทางไปยังไฟล์ HTML ของคุณ:**
สร้างเส้นทางสำหรับไฟล์ HTML ต้นฉบับของคุณโดยใช้ `System-IO.Path.Combine`.

```csharp
string filePath = System.IO.Path.Combine(SourceDir, "Book1.html");
```

**3. กำหนดค่าตัวเลือกการโหลด:**
สร้างตัวอย่าง `HtmlLoadOptions` ชั้นเรียนด้วย `LoadFormat.Html`ขั้นตอนนี้ระบุว่าคุณกำลังโหลดเอกสาร HTML

```csharp
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
```

**4. สร้างวัตถุเวิร์กบุ๊ก:**
ใช้ `Workbook` ผู้สร้างเพื่อเปิดไฟล์ของคุณโดยใช้เส้นทางที่ระบุและโหลดตัวเลือก

```csharp
Workbook wb = new Workbook(filePath, loadOptions);
```

**5. บันทึกเป็นไฟล์ Excel:**
สุดท้ายให้บันทึกสมุดงานในไดเร็กทอรีเอาต์พุตที่ต้องการ

```csharp
string outputFilePath = System.IO.Path.Combine(outputDir, "output.xlsx");
wb.Save(outputFilePath);
```

### คุณสมบัติ 2: การกำหนดค่าตัวเลือกการโหลดสำหรับไฟล์ HTML

#### ภาพรวม:
เรียนรู้วิธีปรับแต่งตัวเลือกการโหลดเพื่อกำหนดวิธีการจัดการไฟล์ HTML ของคุณเมื่อแปลงเป็นเวิร์กบุ๊ก Excel

##### การดำเนินการทีละขั้นตอน:

**1. ตั้งค่าไดเรกทอรีแหล่งที่มา:**
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
```

**2. กำหนดเส้นทางและโหลดตัวเลือกพร้อมการกำหนดค่า:**
ใช้การตั้งค่าเส้นทางแบบเดียวกับก่อนหน้านี้แต่กำหนดค่าตัวเลือกการโหลดเพิ่มเติมหากจำเป็น เช่น การจดจำเนื้อหา HTML เป็นเวิร์กบุ๊กที่สมบูรณ์

```csharp
string filePath = System.IO.Path.Combine(SourceDir, "Book1.html");
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
loadOptions.IsRecognizeAsSingleFile = true;  // ตัวอย่างตัวเลือกการกำหนดค่า
```

**3. สร้างและบันทึกสมุดงาน:**
สร้างเวิร์กบุ๊กที่มีตัวเลือกการกำหนดค่าเหล่านี้และบันทึกไว้

```csharp
Workbook wb = new Workbook(filePath, loadOptions);
string outputFilePath = System.IO.Path.Combine(SourceDir, "output.xlsx");
b.Save(outputFilePath);
```

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าคุณได้ระบุเส้นทางไฟล์ HTML อย่างถูกต้อง
- ตรวจสอบปัญหาด้านใบอนุญาตที่อาจส่งผลต่อกระบวนการโหลด
