## คู่มือการใช้งาน

เราจะแบ่งคุณลักษณะแต่ละอย่างออกเป็นขั้นตอนที่ชัดเจนเพื่อช่วยให้คุณใช้งานได้อย่างมีประสิทธิผล

### เปิดและบันทึกไฟล์ Excel

หัวข้อนี้สาธิตวิธีเปิดไฟล์ Excel ที่มีอยู่และบันทึกหลังจากปรับเปลี่ยน

**ภาพรวม:**

การเปิดไฟล์เป็นขั้นตอนแรกในการประมวลผลข้อมูลใน Excel เมื่อแก้ไขแล้ว การบันทึกจะช่วยให้มั่นใจได้ว่าการเปลี่ยนแปลงทั้งหมดจะยังคงอยู่

**ขั้นตอนการดำเนินการ:**

1. **เปิดไฟล์ Excel ที่มีอยู่:**

   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   
   // โหลดสมุดงานที่มีอยู่
   Workbook workbook = new Workbook(SourceDir + "sampleSortData.xlsx");
   ```

2. **บันทึกสมุดงานในไดเร็กทอรีที่ระบุ:**

   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";

   // บันทึกสมุดงานที่แก้ไขแล้ว
   workbook.Save(outputDir + "outputSaveExcel.xlsx");
   ```

### เข้าถึงเวิร์กชีตและคอลเลกชันเซลล์

การเข้าถึงเวิร์กชีตและเซลล์ที่เจาะจงถือเป็นสิ่งสำคัญสำหรับการจัดการข้อมูลที่ต้องการ

**ภาพรวม:**

นำทางผ่านเวิร์กบุ๊กเพื่อเข้าถึงแผ่นงานแต่ละแผ่นและจัดการค่าเซลล์ตามต้องการ

**ขั้นตอนการดำเนินการ:**

1. **เปิดสมุดงาน:**

   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   
   Workbook workbook = new Workbook(SourceDir + "sampleSortData.xlsx");
   ```

2. **เข้าถึงเวิร์กชีตเฉพาะและคอลเลกชันเซลล์:**

   ```csharp
   // เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
   Worksheet worksheet = workbook.Worksheets[0];
   
   // รับคอลเลกชันเซลล์ทั้งหมดจากเวิร์กชีต
   Cells cells = worksheet.Cells;
   ```

### เริ่มต้นและกำหนดค่า DataSorter

การกำหนดค่าพารามิเตอร์การเรียงลำดับช่วยให้จัดระเบียบข้อมูลได้อย่างมีประสิทธิภาพ

**ภาพรวม:**

การ `DataSorter` วัตถุช่วยให้คุณสามารถเรียงลำดับข้อมูลตามคีย์ที่ระบุและการกำหนดลักษณะการสั่งซื้อ

**ขั้นตอนการดำเนินการ:**

1. **เริ่มต้นวัตถุ DataSorter:**

   ```csharp
   Workbook workbook = new Workbook();
   DataSorter sorter = workbook.DataSorter; // รับวัตถุ DataSorter
   ```

2. **ตั้งค่าพารามิเตอร์การเรียงลำดับ:**

   ```csharp
   // ตั้งค่าลำดับการเรียงลำดับและคีย์สำหรับคอลัมน์
   sorter.Order1 = SortOrder.Ascending;
   sorter.Key1 = 0; // ดัชนีคอลัมน์แรก
   sorter.Order2 = SortOrder.Ascending;
   sorter.Key2 = 1; // ดัชนีคอลัมน์ที่สอง
   ```

### ระบุพื้นที่เซลล์และดำเนินการเรียงลำดับ

การกำหนดช่วงของเซลล์เพื่อเรียงลำดับข้อมูลภายในขอบเขตที่กำหนดถือเป็นสิ่งสำคัญเพื่อการเรียงลำดับที่แม่นยำ

**ภาพรวม:**

โดยใช้ `CellArea`คุณสามารถระบุช่วงที่ต้องการการเรียงลำดับได้อย่างแม่นยำ เพื่อให้มั่นใจถึงความถูกต้องแม่นยำในงานการจัดการข้อมูลของคุณ

**ขั้นตอนการดำเนินการ:**

1. **กำหนดพื้นที่เซลล์ (ช่วง):**

   ```csharp
   Workbook workbook = new Workbook();
   DataSorter sorter = workbook.DataSorter;

   // สร้างพื้นที่เซลล์สำหรับการเรียงลำดับ
   CellArea ca = new CellArea { StartRow = 1, EndRow = 9, StartColumn = 0, EndColumn = 2 };
   ```

2. **ดำเนินการจัดเรียง:**

   ```csharp
   // เรียงลำดับช่วงที่ระบุในเวิร์กชีตแรก
   sorter.Sort(workbook.Worksheets[0].Cells, ca);
   ```
