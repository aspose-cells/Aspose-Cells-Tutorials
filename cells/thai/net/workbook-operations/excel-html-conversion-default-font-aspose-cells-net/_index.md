# เรียนรู้การตั้งค่าฟอนต์เริ่มต้นใน Excel เพื่อแปลง HTML ด้วย Aspose.Cells สำหรับ .NET

## การแนะนำ

การแปลงเวิร์กบุ๊ก Excel เป็นรูปแบบ HTML โดยยังคงรักษาการจัดวางตัวอักษรให้สม่ำเสมออาจเป็นเรื่องท้าทาย บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าแบบอักษรเริ่มต้นโดยใช้ Aspose.Cells สำหรับ .NET เพื่อให้แน่ใจว่าเอกสารที่แปลงแล้วของคุณดูสวยงามและเป็นมืออาชีพ การเชี่ยวชาญฟีเจอร์นี้จะช่วยให้คุณเอาชนะความท้าทายที่เกี่ยวข้องกับแบบอักษรที่ไม่รู้จักหรือไม่พร้อมใช้งานในกระบวนการแปลงได้

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีการตั้งค่าแบบอักษรเริ่มต้นเมื่อแปลงไฟล์ Excel เป็น HTML
- คำแนะนำทีละขั้นตอนในการใช้ Aspose.Cells สำหรับ .NET
- เทคนิคการจัดการแบบอักษรที่ไม่รู้จักให้สวยงามระหว่างการเรนเดอร์

มาเริ่มตั้งค่าสภาพแวดล้อมของคุณและเริ่มต้นสำรวจฟีเจอร์นี้กันเลย!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่ม ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- **สภาพแวดล้อม .NET**:มีการติดตั้ง .NET เวอร์ชันที่เข้ากันได้ (เช่น .NET Core หรือ .NET Framework)
- **Aspose.Cells สำหรับไลบรารี .NET**:ติดตั้ง Aspose.Cells ผ่าน NuGet
- **ความรู้พื้นฐานเกี่ยวกับ C#**ความคุ้นเคยกับแนวคิดการเขียนโปรแกรม C# จะเป็นประโยชน์

## การตั้งค่า Aspose.Cells สำหรับ .NET

ในการเริ่มต้น ให้ตั้งค่า Aspose.Cells ในสภาพแวดล้อมการพัฒนาของคุณโดยทำตามขั้นตอนเหล่านี้:

**การติดตั้งผ่าน CLI:**
```bash
dotnet add package Aspose.Cells
```

**การติดตั้งผ่านตัวจัดการแพ็คเกจ:**
```shell
PM> NuGet\Install-Package Aspose.Cells
```

### การขอใบอนุญาต
- **ทดลองใช้งานฟรี**:เริ่มต้นด้วยการทดลองใช้ฟรีเพื่อสำรวจคุณสมบัติต่างๆ
- **ใบอนุญาตชั่วคราว**:การขอใบอนุญาตชั่วคราวเพื่อวัตถุประสงค์ในการประเมินผล
- **ซื้อ**: โปรดพิจารณาซื้อใบอนุญาตเพื่อใช้ในการผลิต

เมื่อติดตั้งแล้ว ให้เริ่มต้นและตั้งค่าโครงการของคุณดังนี้:
```csharp
using Aspose.Cells;
```

## คู่มือการใช้งาน

### การตั้งค่าฟอนต์เริ่มต้นขณะเรนเดอร์

ฟีเจอร์นี้ช่วยให้แน่ใจว่าเวิร์กบุ๊ก Excel จะแสดงด้วยแบบอักษรเริ่มต้นเฉพาะเมื่อแปลงเป็น HTML ฟีเจอร์นี้มีประโยชน์โดยเฉพาะในการจัดการกรณีที่แบบอักษรบางตัวอาจไม่มีให้ใช้งานบนระบบเป้าหมาย

#### ขั้นตอนที่ 1: สร้างและเข้าถึงสมุดงาน

สร้างอินสแตนซ์ใหม่ของ `Workbook` และเข้าถึงแผ่นงานแรก:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// สร้างวัตถุเวิร์กบุ๊กและเข้าถึงเวิร์กชีตแรก
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```

#### ขั้นตอนที่ 2: ปรับเปลี่ยนรูปแบบเซลล์

เข้าถึงเซลล์เฉพาะ เพิ่มข้อความ และตั้งค่าแบบอักษรเป็นแบบอักษรที่ไม่รู้จักเพื่อสาธิต:
```csharp
// เข้าถึงเซลล์ B4 และเพิ่มข้อความภายในนั้น
Cell cell = ws.Cells["B4"];
cell.PutValue("This text has some unknown or invalid font which does not exist.");

// ตั้งค่าแบบอักษรของเซลล์ B4 เป็นแบบอักษรที่ไม่รู้จัก
Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist";
st.Font.Size = 20;
cell.SetStyle(st);
```

#### ขั้นตอนที่ 3: กำหนดตัวเลือกการบันทึก HTML

ตั้งค่าแบบอักษรเริ่มต้นในผลลัพธ์ HTML ของคุณ ที่นี่เราจะสาธิตด้วยแบบอักษรสามแบบที่แตกต่างกัน:

**บริการจัดส่งใหม่:**
```csharp
// บันทึกสมุดงานในรูปแบบ HTML โดยตั้งค่าฟอนต์เริ่มต้นเป็น Courier New
HtmlSaveOptions optsCourierNew = new HtmlSaveOptions();
optsCourierNew.DefaultFontName = "Courier New";
wb.Save(outputDir + "/out_courier_new_out.htm", optsCourierNew);
```

**อาเรียล:**
```csharp
// บันทึกเวิร์กบุ๊กในรูปแบบ HTML โดยตั้งค่าฟอนต์เริ่มต้นเป็น Arial
HtmlSaveOptions optsArial = new HtmlSaveOptions();
optsArial.DefaultFontName = "Arial";
wb.Save(outputDir + "/out_arial_out.htm", optsArial);
```

**ไทม์นิวโรมัน:**
```csharp
// บันทึกเวิร์กบุ๊กในรูปแบบ HTML โดยตั้งค่าฟอนต์เริ่มต้นเป็น Times New Roman
HtmlSaveOptions optsTimesNewRoman = new HtmlSaveOptions();
optsTimesNewRoman.DefaultFontName = "Times New Roman";
wb.Save(outputDir + "/times_new_roman_out.htm", optsTimesNewRoman);
```

### การสร้างสมุดงานและการจัดรูปแบบเซลล์

หัวข้อนี้ครอบคลุมถึงการสร้างเวิร์กบุ๊ก การเข้าถึงเวิร์กชีต เซลล์ และการใช้สไตล์:

#### ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊ก
สร้างใหม่ `Workbook` ตัวอย่าง:
```csharp
// สร้างวัตถุเวิร์กบุ๊ก
Workbook wb = new Workbook();
```

#### ขั้นตอนที่ 2: เข้าถึงเวิร์กชีตและเซลล์
เข้าถึงเวิร์กชีตแรกและเซลล์ B4 เพื่อเพิ่มข้อความและรูปแบบ:
```csharp
// เข้าถึงแผ่นงานแรกในสมุดงาน
Worksheet ws = wb.Worksheets[0];

// เข้าถึงเซลล์ B4 และเพิ่มข้อความภายในนั้น
Cell cell = ws.Cells["B4"];
cell.PutValue("This text has some unknown or invalid font which does not exist.");

// ตั้งค่าแบบอักษรของเซลล์ B4 เป็นแบบอักษรที่ไม่รู้จัก
Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist";
st.Font.Size = 20;
cell.SetStyle(st);
```

- **การพกพาเอกสาร**:จัดการสถานการณ์ที่สภาพแวดล้อมเป้าหมายขาดแบบอักษรเฉพาะ
