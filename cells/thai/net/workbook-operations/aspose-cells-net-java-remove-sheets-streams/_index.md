## คู่มือการใช้งาน

### คุณลักษณะที่ 1: การลบเวิร์กชีต Excel โดยใช้ดัชนี

**ภาพรวม**:ฟีเจอร์นี้ช่วยให้คุณลบเวิร์กชีตเฉพาะออกจากเวิร์กบุ๊ก Excel โดยใช้ดัชนีของเวิร์กบุ๊ก ฟีเจอร์นี้มีประโยชน์อย่างยิ่งสำหรับการทำงานอัตโนมัติในการจัดการข้อมูลในกรณีที่จำเป็นต้องลบชีตที่ไม่จำเป็น

#### คำแนะนำทีละขั้นตอน:

##### เปิดไฟล์ Excel
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.WorksheetCollection;

String SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/book1.xls");
```

##### เข้าถึงคอลเลกชันแผ่นงาน
```java
WorksheetCollection worksheets = workbook.getWorksheets();
```

##### ลบแผ่นงานแรกออก
- ใช้ดัชนีฐานศูนย์เพื่อระบุว่าจะลบแผ่นงานใด
```java
worksheets.removeAt(0);
```

##### บันทึกสมุดงานที่แก้ไขแล้ว
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outputDir + "/output.out.xls");
```

**เคล็ดลับการแก้ไขปัญหา**: ตรวจสอบให้แน่ใจว่าเส้นทางเวิร์กบุ๊กของคุณถูกต้องและคุณมีสิทธิ์การเขียนที่จำเป็นสำหรับไดเร็กทอรีเอาต์พุต

### คุณสมบัติ 2: การจัดการไฟล์ด้วยสตรีม

**ภาพรวม**:ฟีเจอร์นี้สาธิตวิธีการจัดการสตรีมไฟล์อย่างมีประสิทธิภาพเมื่อทำงานกับไฟล์ Excel เพื่อให้มั่นใจถึงการใช้ทรัพยากรอย่างเหมาะสมที่สุด

#### คำแนะนำทีละขั้นตอน:

##### เปิดสตรีมไฟล์
```java
import java.io.FileInputStream;

String filePath = SourceDir + "/book1.xls";
FileInputStream fstream = new FileInputStream(filePath);
```

##### โหลดเวิร์กบุ๊กจากสตรีมไฟล์
```java
Workbook workbook = new Workbook(fstream);
```

##### ปิดสตรีมไฟล์
- การปิดลำธารเป็นสิ่งสำคัญเพื่อปลดปล่อยทรัพยากร
```java
fstream.close();
```

**เคล็ดลับการแก้ไขปัญหา**:จัดการข้อยกเว้นอยู่เสมอและตรวจสอบให้แน่ใจว่าสตรีมถูกปิดใน `finally` บล็อกหรือใช้ try-with-resources เพื่อการจัดการทรัพยากรที่ดีขึ้น
