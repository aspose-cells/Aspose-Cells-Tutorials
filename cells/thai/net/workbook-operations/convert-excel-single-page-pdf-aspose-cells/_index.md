## คู่มือการใช้งาน

เราจะแบ่งกระบวนการนี้ออกเป็นหลายส่วนเพื่อให้จัดการได้ชัดเจน

### การเปิดไฟล์ Excel

คุณลักษณะนี้ช่วยให้คุณเปิดเวิร์กบุ๊ก Excel ที่มีอยู่โดยใช้ `Workbook` คลาสที่จัดทำโดย Aspose.Cells นี่คือวิธีการทำงาน:

**ขั้นตอนที่ 1**: กำหนดไดเร็กทอรีแหล่งที่มาและชื่อไฟล์ของคุณ

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string fileName = "sampleRenderOnePdfPagePerExcelWorksheet.xlsx";
```

**ขั้นตอนที่ 2**:โหลดสมุดงาน Excel

```csharp
Workbook workbook = new Workbook(SourceDir + fileName);
```

### การกำหนดค่าตัวเลือกการบันทึก PDF

เพื่อให้แน่ใจว่าแต่ละแผ่นงานจะแสดงบนหน้าเดียวใน PDF ของคุณ ให้กำหนดค่า `PdfSaveOptions`-

**ขั้นตอนที่ 1**: สร้างอินสแตนซ์ของ `PdfSaveOptions` และตั้งค่า `OnePagePerSheet` คุณสมบัติ.

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.OnePagePerSheet = true;
```

### การบันทึก Excel เป็น PDF พร้อมตัวเลือกเฉพาะ

เมื่อโหลดเวิร์กบุ๊กและกำหนดค่าตัวเลือกแล้ว ให้บันทึกเป็นไฟล์ PDF โดยใช้การตั้งค่าเหล่านี้

**ขั้นตอนที่ 1**: กำหนดไดเร็กทอรีเอาต์พุตและชื่อไฟล์สำหรับ PDF ที่ได้

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
string pdfFileName = "outputRenderOnePdfPagePerExcelWorksheet.pdf";
```

**ขั้นตอนที่ 2**: บันทึกสมุดงานด้วยตัวเลือกบันทึกที่คุณระบุ

```csharp
workbook.Save(outputDir + pdfFileName, pdfSaveOptions);
```

### เคล็ดลับการแก้ไขปัญหา

- **ข้อผิดพลาดไม่พบไฟล์**: ให้แน่ใจว่าคุณ `SourceDir` และเส้นทางไฟล์ได้รับการตั้งค่าอย่างถูกต้อง
- **ปัญหาเอาต์พุต PDF**: ตรวจสอบว่า `OnePagePerSheet` ได้รับการกำหนดค่าอย่างถูกต้องใน `PdfSaveOptions`-
