## คู่มือการใช้งาน

### คุณสมบัติ 1: เปิดและอ่านไฟล์ Excel
**ภาพรวม:** คุณลักษณะนี้สาธิตการเปิดไฟล์ Excel โดยใช้ Aspose.Cells เพื่อเข้าถึงวัตถุเวิร์กบุ๊ก

#### คำแนะนำทีละขั้นตอน
1. **การตั้งค่าสตรีมไฟล์**: ใช้ `FileStream` เพื่อเปิดไฟล์ Excel ที่คุณต้องการ
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   FileStream fstream = new FileStream(SourceDir + "/Book1.xlsx", FileMode.Open);
   Workbook workbook = new Workbook(fstream);
   ```
2. **การเข้าถึงสมุดงาน**:ตัวอย่างโค้ดด้านบนจะเริ่มต้น `Workbook` วัตถุที่ให้สิทธิ์ในการเข้าถึงเนื้อหาของไฟล์ Excel

#### แนวคิดหลัก
- **ไฟล์สตรีม**: จัดการการดำเนินการอินพุต/เอาท์พุตไฟล์
- **สมุดแบบฝึกหัด**: หมายถึงเอกสาร Excel ทั้งหมด

### คุณลักษณะที่ 2: เข้าถึงเวิร์กชีตในเวิร์กบุ๊ก
**ภาพรวม:** เรียนรู้วิธีการกำหนดเป้าหมายและทำงานกับเวิร์กชีตเฉพาะภายในเวิร์กบุ๊กของคุณ

#### คำแนะนำทีละขั้นตอน
1. **โหลดสมุดงาน**-
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "/Book1.xlsx");
   ```
2. **ใบงานการเข้าถึง**:เข้าถึงเวิร์กชีตเฉพาะโดยใช้ดัชนี
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

### คุณสมบัติที่ 3: คัดลอกการจัดรูปแบบตามเงื่อนไขจากเซลล์หนึ่งไปยังอีกเซลล์หนึ่ง
**ภาพรวม:** คุณสมบัตินี้ครอบคลุมการคัดลอกการตั้งค่าการจัดรูปแบบตามเงื่อนไขระหว่างช่วงเซลล์

#### คำแนะนำทีละขั้นตอน
1. **การเริ่มต้นสมุดงานและแผ่นงาน**-
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "/Book1.xlsx");
   Worksheet worksheet = workbook.Worksheets[0];
   int TotalRowCount = 0;
   ```
2. **คัดลอกรูปแบบลูป**: ทำซ้ำในเวิร์กชีตทั้งหมดเพื่อคัดลอกการจัดรูปแบบตามเงื่อนไข
   ```csharp
   for (int i = 0; i < workbook.Worksheets.Count; i++)
   {
       Worksheet sourceSheet = workbook.Worksheets[i];
       Range sourceRange = sourceSheet.Cells.MaxDisplayRange;
       Range destRange = worksheet.Cells.CreateRange(sourceRange.FirstRow + TotalRowCount, 
           sourceRange.FirstColumn, sourceRange.RowCount, sourceRange.ColumnCount);
       destRange.Copy(sourceRange);
       TotalRowCount += sourceRange.RowCount;
   }
   ```

#### แนวคิดหลัก
- **พิสัย**: หมายถึงกลุ่มของเซลล์ในเวิร์กบุ๊ก
- **สำเนา**: วิธีการจำลองการตั้งค่าการจัดรูปแบบ

### คุณสมบัติที่ 4: บันทึกไฟล์ Excel ที่ปรับเปลี่ยนแล้ว
**ภาพรวม:** เรียนรู้วิธีบันทึกการปรับเปลี่ยนของคุณกลับลงในไฟล์ Excel

#### คำแนะนำทีละขั้นตอน
1. **ดำเนินการปรับเปลี่ยน**:ใช้ขั้นตอนจากฟีเจอร์ก่อนหน้าเพื่อปรับเปลี่ยนเวิร์กบุ๊กของคุณ
   ```csharp
   int TotalRowCount = 0;
   for (int i = 0; i < workbook.Worksheets.Count; i++)
   {
       Worksheet sourceSheet = workbook.Worksheets[i];
       Range sourceRange = sourceSheet.Cells.MaxDisplayRange;
       Range destRange = workbook.Worksheets[0].Cells.CreateRange(sourceRange.FirstRow + TotalRowCount, 
           sourceRange.FirstColumn, sourceRange.RowCount, sourceRange.ColumnCount);
       destRange.Copy(sourceRange);
       TotalRowCount += sourceRange.RowCount;
   }
   ```
2. **บันทึกสมุดงาน**-
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "/output.xls");
   ```
