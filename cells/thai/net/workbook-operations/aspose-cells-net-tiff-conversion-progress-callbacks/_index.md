# เพิ่มประสิทธิภาพการแปลง Excel เป็น TIFF ด้วย Aspose.Cells .NET และ Progress Callbacks

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน
ให้เราแบ่งขั้นตอนการดำเนินการออกเป็นขั้นตอนเชิงตรรกะ เพื่อให้เกิดความชัดเจนและเข้าใจง่าย 
### 1. การตั้งค่าตัวเลือกการแปลง
#### ภาพรวม
เราจะเริ่มต้นด้วยการกำหนดค่า `ImageOrPrintOptions` คลาสซึ่งให้การตั้งค่าเฉพาะสำหรับงานการเรนเดอร์ภาพ
**คำแนะนำทีละขั้นตอน:**
##### กำหนดประเภทภาพ
ตั้งค่ารูปแบบเอาท์พุตเป็น TIFF:
```csharp
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.ImageType = ImageType.Tiff;
```
##### เพิ่มการโทรกลับความคืบหน้า
แนบตัวจัดการการโทรกลับเพื่อติดตามความคืบหน้าในการบันทึกหน้า:
```csharp
opts.PageSavingCallback = new TestTiffPageSavingCallback();
```
### 2. การนำ Page Saving Callback มาใช้
#### ภาพรวม
ปรับแต่งหน้าที่จะแสดงผลและติดตามความคืบหน้าของการเรนเดอร์ด้วยการโทรกลับ
**คำแนะนำทีละขั้นตอน:**
##### การสร้างคลาสการโทรกลับแบบกำหนดเอง
กำหนดคลาสการโทรกลับของคุณโดยการใช้งาน `IPageSavingCallback`-
```csharp
public class TestTiffPageSavingCallback : IPageSavingCallback
{
    public void PageStartSaving(PageStartSavingArgs args)
    {
        Console.WriteLine("Start saving page index {0} of pages {1}", args.PageIndex, args.PageCount);
        
        // อย่าพิมพ์หน้าก่อนดัชนี 2
        if (args.PageIndex < 2)
        {
            args.IsToOutput = false;
        }
    }

    public void PageEndSaving(PageEndSavingArgs args)
    {
        Console.WriteLine("End saving page index {0} of pages {1}", args.PageIndex, args.PageCount);

        // หยุดการส่งออกหลังจากดัชนีหน้า 8
        if (args.PageIndex >= 8)
        {
            args.HasMorePages = false;
        }
    }
}
```
### 3. การดำเนินการตามกระบวนการแปลง
#### ภาพรวม
สุดท้ายให้เรนเดอร์สมุดงานของคุณเป็นภาพ TIFF โดยใช้ `WorkbookRender`-
**คำแนะนำทีละขั้นตอน:**
##### สมุดงานการเรนเดอร์
แปลงและบันทึกเอกสารด้วยตัวเลือกที่กำหนดค่าไว้:
```csharp
WorkbookRender wr = new WorkbookRender(workbook, opts);
wr.ToImage("DocumentConversionProgressForTiff_out.tiff");
```
