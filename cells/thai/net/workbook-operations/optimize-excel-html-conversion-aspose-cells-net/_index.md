## คู่มือการใช้งาน

### ภาพรวม
ในส่วนนี้จะแนะนำคุณเกี่ยวกับการกำหนดค่าและการดำเนินการแปลง Excel เป็น HTML โดยมีคอลัมน์ที่ปรับขนาดได้โดยใช้ Aspose.Cells สำหรับ .NET

#### ขั้นตอนที่ 1: โหลดสมุดงานของคุณ
เริ่มต้นด้วยการโหลดเวิร์กบุ๊ก Excel ต้นฉบับที่คุณต้องการแปลง ซึ่งเกี่ยวข้องกับการตั้งค่าไดเร็กทอรีอินพุตและเอาท์พุตของคุณ:

```csharp
// ไดเรกทอรีอินพุต
string sourceDir = RunExamples.Get_SourceDirectory();

// ไดเรกทอรีผลลัพธ์
string outputDir = RunExamples.Get_OutputDirectory();
```

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือกการบันทึก HTML
สร้างอินสแตนซ์ของ `HtmlSaveOptions` เพื่อจัดการวิธีบันทึกไฟล์ Excel เป็น HTML ซึ่งรวมถึงการเปิดใช้งานคอลัมน์ที่ปรับขนาดได้และการส่งออกรูปภาพเป็น Base64

```csharp
// ระบุตัวเลือกการบันทึก HTML
HtmlSaveOptions options = new HtmlSaveOptions();

// ตั้งค่าคุณสมบัติสำหรับความกว้างที่ปรับขนาดได้
options.WidthScalable = true;

// ส่งออกรูปภาพเป็น Base64 เพื่อฝังไว้ใน HTML
options.ExportImagesAsBase64 = true;
```

#### ขั้นตอนที่ 3: ดำเนินการแปลง
สุดท้าย ให้บันทึกเวิร์กบุ๊กลงในไฟล์ HTML พร้อมตัวเลือกที่คุณกำหนดค่าไว้:

```csharp
// โหลดไฟล์ตัวอย่างต้นฉบับ
Workbook wb = new Workbook(sourceDir + "sampleForScalableColumns.xlsx");

// บันทึกสมุดงานในรูปแบบ HTML
wb.Save(outputDir + "outsampleForScalableColumns.html", options);
```
