## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นคุณสมบัติที่แตกต่างกันเพื่อให้ง่ายต่อการปฏิบัติตาม

### 1. ตั้งค่าไดเรกทอรี

**ภาพรวม**: กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุตสำหรับไฟล์ของคุณ

#### ขั้นตอนการดำเนินการ:
- **กำหนดเส้นทาง**-
  ```csharp
  string SourceDir = @"YOUR_SOURCE_DIRECTORY";
  string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
  ```
  - แทนที่ตัวแทนด้วยเส้นทางไดเร็กทอรีจริงที่ไฟล์ Excel ของคุณอยู่และที่คุณต้องการบันทึกไฟล์ SVG

### 2. โหลดสมุดงานจากเทมเพลต

**ภาพรวม**โหลดเวิร์กบุ๊ก Excel ที่มีอยู่โดยใช้เทมเพลต

#### ขั้นตอนการดำเนินการ:
- **โหลดสมุดงาน**-
  ```csharp
  string filePath = SourceDir + "Template.xlsx";
  Workbook book = new Workbook(filePath);
  ```
  - ให้แน่ใจว่า `filePath` ชี้ไปที่ไฟล์เทมเพลตของคุณ โค้ดนี้จะเริ่มต้นวัตถุเวิร์กบุ๊กจากไฟล์นี้

### 3. แปลงเวิร์กชีตเป็น SVG

**ภาพรวม**:แปลงแผ่นงานแต่ละแผ่นในเวิร์กบุ๊ก Excel เป็นรูปแบบ SVG

#### ขั้นตอนการดำเนินการ:
- **กำหนดค่าตัวเลือกภาพ**-
  ```csharp
  using Aspose.Cells.Rendering;

  ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
  imgOptions.SaveFormat = SaveFormat.Svg;
  imgOptions.OnePagePerSheet = true; // บันทึกแต่ละแผ่นเป็นหนึ่งหน้า
  ```

- **ทำซ้ำและแปลง**-
  ```csharp
  foreach (Worksheet sheet in book.Worksheets)
  {
      SheetRender sr = new SheetRender(sheet, imgOptions);
      for (int i = 0; i < sr.PageCount; i++)
      {
          string outputFilePath = OutputDir + sheet.Name + i + ".svg";
          sr.ToImage(i, outputFilePath); // บันทึกแต่ละหน้าเป็นไฟล์ SVG
      }
  }
  ```
  - ลูปนี้จะประมวลผลเวิร์กชีตแต่ละแผ่นและบันทึกเป็น SVG หน้าเดียว

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยง `DirectoryNotFoundException`-
- ตรวจสอบว่าไฟล์เทมเพลตของคุณมีอยู่ในเส้นทางที่ระบุก่อนที่จะโหลด
