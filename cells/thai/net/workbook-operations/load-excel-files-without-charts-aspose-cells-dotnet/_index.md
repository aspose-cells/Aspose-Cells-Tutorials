## การแนะนำ

พร้อมที่จะเปลี่ยนแปลงวิธีการจัดการไฟล์ Excel ของคุณหรือยัง มาเริ่มต้นด้วยข้อกำหนดเบื้องต้นกันก่อน

## คู่มือการใช้งาน

ในหัวข้อนี้เราจะแบ่งการใช้งานออกเป็นส่วนๆ ที่สามารถจัดการได้ เพื่อให้เข้าใจได้ชัดเจนยิ่งขึ้น

### ภาพรวมของคุณสมบัติ
ฟีเจอร์นี้ช่วยให้คุณโหลดเวิร์กบุ๊ก Excel ได้ในขณะที่ยกเว้นข้อมูลแผนภูมิโดยเฉพาะ ซึ่งมีประโยชน์อย่างยิ่งเมื่อต้องจัดการกับชุดข้อมูลขนาดใหญ่ที่ข้อมูลแผนภูมิอาจใช้ทรัพยากรและเวลาในการประมวลผลที่ไม่จำเป็น

### การดำเนินการแบบทีละขั้นตอน

#### **1. กำหนดไดเรกทอรีต้นทางและปลายทาง (H3)**

เริ่มต้นด้วยการตั้งค่าไดเร็กทอรีสำหรับไฟล์ต้นฉบับและปลายทางผลลัพธ์:

```csharp
// ระบุเส้นทางสำหรับไฟล์ของคุณ
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();
```
**คำอธิบาย**บรรทัดเหล่านี้กำหนดว่าไฟล์อินพุต Excel ของคุณอยู่ที่ไหน และคุณต้องการบันทึกเอาต์พุตที่ประมวลผลแล้วไว้ที่ใด

#### **2. กำหนดค่าตัวเลือกการโหลด (H3)**

ตั้งค่าตัวเลือกการโหลดเพื่อกรองข้อมูลแผนภูมิ:

```csharp
// สร้างตัวเลือกการโหลดด้วยตัวกรองเฉพาะสำหรับข้อมูล
LoadOptions loadOptions = new LoadOptions();
loadOptions.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.Chart);
```
**คำอธิบาย**: ที่นี่เราสร้าง `LoadOptions` และนำไปใช้ `LoadFilter` เพื่อไม่รวมข้อมูลแผนภูมิ (`~LoadDataFilterOptions.Chart`) วิธีนี้จะช่วยให้แน่ใจว่าแผนภูมิจะไม่ถูกโหลดเข้าสู่หน่วยความจำ

#### **3. โหลดเวิร์กบุ๊ก (H3)**

ตอนนี้โหลดสมุดงานของคุณโดยใช้ตัวเลือกเหล่านี้:

```csharp
// ใช้ตัวเลือกการโหลดเพื่อเปิดไฟล์ Excel โดยไม่ต้องโหลดแผนภูมิ
Workbook workbook = new Workbook(sourceDir + "sampleLoadExcelFileWithoutChart.xlsx", loadOptions);
```
**คำอธิบาย**: เดอะ `Workbook` คอนสตรัคเตอร์ยอมรับเส้นทางและ `LoadOptions`โหลดเฉพาะข้อมูลที่ระบุไว้โดยตัวกรองของคุณ

#### **4. บันทึกไฟล์ที่ประมวลผลแล้ว (H3)**

สุดท้ายให้บันทึกสมุดงานที่คุณประมวลผลแล้วในรูปแบบที่ต้องการ:

```csharp
// บันทึกสมุดงานเป็น PDF โดยไม่ต้องใช้แผนภูมิ
workbook.Save(outputDir + "outputLoadExcelFileWithoutChart.pdf", SaveFormat.Pdf);
```
**คำอธิบาย**: เดอะ `Save` วิธีการนี้จะส่งออกไฟล์ไปยังไดเร็กทอรีและรูปแบบที่ระบุ ในที่นี้ เราจะแปลงไฟล์เป็น PDF

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาทั่วไป**:หากเอาต์พุตของคุณไม่รวมแผนภูมิ ให้ตรวจสอบอีกครั้งว่าการตั้งค่าตัวกรองโหลดถูกนำไปใช้ถูกต้องหรือไม่
- **คอขวดของประสิทธิภาพ**:ให้แน่ใจว่าระบบของคุณมีทรัพยากรเพียงพอเมื่อประมวลผลไฟล์ขนาดใหญ่ แม้จะใช้วิธีการโหลดที่ปรับให้เหมาะสมแล้วก็ตาม
