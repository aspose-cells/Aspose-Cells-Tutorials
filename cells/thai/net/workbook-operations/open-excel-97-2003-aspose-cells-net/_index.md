## คู่มือการใช้งาน (H2)

### ภาพรวม

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการโหลดและการเข้าถึงข้อมูลจากไฟล์ Microsoft Excel 97-2003 โดยใช้ Aspose.Cells สำหรับ .NET เราจะเน้นที่การเริ่มต้นใช้งาน `Workbook` วัตถุซึ่งแสดงไฟล์ Excel และสาธิตวิธีจัดการสตรีมไฟล์อย่างมีประสิทธิภาพ

#### การดำเนินการทีละขั้นตอน (H3)

1. **ตั้งค่าโครงการของคุณ**

   ตรวจสอบให้แน่ใจว่าโครงการของคุณอ้างอิงไลบรารี Aspose.Cells ตามที่ระบุไว้ในส่วนการตั้งค่า

2. **เปิดไฟล์ Excel 97-2003**

   ด้านล่างนี้เป็นตัวอย่างที่แสดงวิธีเปิดไฟล์ Excel 97-2003:

   ```csharp
   using System;
   using System.IO;
   using Aspose.Cells;

   namespace Aspose.Cells.Examples.CSharp.Files.Handling
   {
       public class OpeningMicrosoftExcel972003Files
       {
           public static void Run()
           {
               // เส้นทางไปยังไดเร็กทอรีเอกสาร
               string dataDir = "your_directory_path/"; // อัปเดตด้วยเส้นทางไดเร็กทอรีจริงของคุณ

               // รับไฟล์ Excel เข้าสู่สตรีม
               using (FileStream stream = new FileStream(dataDir + "Book_Excel97_2003.xls", FileMode.Open))
               {
                   // สร้างอินสแตนซ์ของ LoadOptions ที่ระบุโดย LoadFormat
                   LoadOptions loadOptions1 = new LoadOptions(LoadFormat.Excel97To2003);

                   // สร้างวัตถุเวิร์กบุ๊กและเปิดไฟล์จากสตรีม
                   Workbook wbExcel97 = new Workbook(stream, loadOptions1);
                   Console.WriteLine("Microsoft Excel 97 - 2003 workbook opened successfully!");
               }
           }
       }
   }
   ```

#### ตัวเลือกการกำหนดค่าคีย์

- **ตัวเลือกการโหลด**: เดอะ `LoadOptions` คลาสช่วยให้คุณสามารถระบุรูปแบบของไฟล์ Excel ได้ ในกรณีนี้ เราใช้ `Excel97To2003`-
- **ไฟล์สตรีม**: การใช้ `FileStream`เรามั่นใจว่าทรัพยากรได้รับการจัดการอย่างมีประสิทธิภาพด้วยการกำจัดสตรีมหลังการใช้งาน

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกระบุอย่างถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าคุณมีสิทธิ์อนุญาตที่เหมาะสมในการอ่านไฟล์จากไดเร็กทอรี
- หากพบปัญหาในการโหลด ให้ยืนยันว่ารูปแบบไฟล์ Excel ตรงกัน `Excel97To2003`-
