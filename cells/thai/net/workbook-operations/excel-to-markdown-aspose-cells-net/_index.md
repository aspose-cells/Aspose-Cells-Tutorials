## คู่มือการใช้งาน

วิธีการแปลงไฟล์ Excel เป็นมาร์กดาวน์โดยใช้ Aspose.Cells มีดังนี้

### ขั้นตอนที่ 1: เปิดไฟล์ Excel
โหลดไฟล์ Excel ของคุณด้วย `Workbook` ชั้นเรียนเพื่อการเข้าถึงข้อมูลได้ง่าย

```csharp
// โหลดไฟล์ Excel
Workbook workbook = new Workbook("sourcePath\\Book1.xlsx");
```
**คำอธิบาย**:โค้ดนี้จะสร้างอินสแตนซ์ของ `Workbook` คลาสและโหลดไฟล์ Excel จากเส้นทางที่ระบุ

### ขั้นตอนที่ 2: แปลงเป็นมาร์กดาวน์
บันทึกสมุดงานที่คุณโหลดไว้ในรูปแบบมาร์กดาวน์โดยใช้ `Save` วิธี.

```csharp
// กำหนดไดเรกทอรีเอาท์พุตและแปลง
workbook.Save("outputPath\\Book1.md", SaveFormat.Markdown);
```
**คำอธิบาย**: เดอะ `Save` วิธีนี้ใช้พารามิเตอร์สองตัว ได้แก่ เส้นทางไฟล์สำหรับจัดเก็บมาร์กดาวน์และรูปแบบการบันทึกที่ต้องการ ที่นี่ `SaveFormat.Markdown` ระบุรูปแบบมาร์กดาวน์

### เคล็ดลับการแก้ไขปัญหา
- **ข้อผิดพลาดไม่พบไฟล์**ตรวจสอบเส้นทางไฟล์ของคุณอีกครั้ง
- **ปัญหาการอนุญาต**: ตรวจสอบให้แน่ใจว่าแอปพลิเคชันของคุณมีสิทธิ์การเขียนในไดเร็กทอรีเอาต์พุต
