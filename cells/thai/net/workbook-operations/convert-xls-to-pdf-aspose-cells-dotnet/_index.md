### ขั้นตอนที่ 1: การโหลดไฟล์ Excel
#### ภาพรวม
โหลดเอกสาร Excel ของคุณลงใน `Workbook` วัตถุ ที่นี่คือจุดที่ความมหัศจรรย์ทั้งหมดเริ่มต้น!

```csharp

```
**คำอธิบาย**: เดอะ `Workbook` คลาสแสดงถึงสเปรดชีตทั้งหมด ช่วยให้คุณสามารถจัดการและบันทึกในรูปแบบต่างๆ ได้

### ขั้นตอนที่ 2: บันทึกเป็น PDF
#### ภาพรวม
บันทึกไฟล์ Excel ที่โหลดไว้เป็นเอกสาร PDF

```csharp
// ระบุไดเร็กทอรีเอาท์พุตของคุณ

// บันทึกสมุดงานในรูปแบบ PDF

```
**คำอธิบาย**: เดอะ `Save` วิธีการของ `Workbook` คลาสนี้ช่วยให้บันทึกไฟล์ได้หลากหลายรูปแบบ รวมถึง PDF โดยที่นี่ เราจะระบุไดเร็กทอรีเอาต์พุตและชื่อไฟล์
