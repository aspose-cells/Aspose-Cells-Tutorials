## คู่มือการใช้งาน

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อสร้างและปรับแต่งไฟล์ Excel ของคุณ:

### การสร้างสมุดงานใหม่
สร้างเวิร์กบุ๊ก Excel ใหม่ดังนี้:
```csharp
// สร้างอินสแตนซ์ของเวิร์กบุ๊ก (ไฟล์ Excel)
Workbook workbook = new Workbook();
```

### การลบแผ่นงานเริ่มต้น
ลบแผ่นงานเริ่มต้นหากไม่จำเป็น:
```csharp
// ลบเวิร์กชีตเริ่มต้นที่ถูกสร้างขึ้นเมื่อมีการสร้างอินสแตนซ์เวิร์กบุ๊กใหม่
workbook.Worksheets.RemoveAt(0);
```

### การเพิ่มและการตั้งชื่อแผ่นงานหลายแผ่น
เพิ่มแผ่นงานห้าแผ่นลงในสมุดงานของคุณและตั้งชื่อตามลำดับ
```csharp
// เพิ่มแผ่นงาน 5 แผ่นและตั้งชื่อ
for (int i = 0; i < 5; i++) {
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = "Sheet" + (i + 1).ToString();
}
```

### การเติมข้อมูลลงในแผ่นงาน
เติมข้อมูลในแต่ละแผ่นงานในตาราง
```csharp
// เติมข้อมูลลงในแผ่นงาน
for (int i = 0; i < workbook.Worksheets.Count; i++) {
    Worksheet ws = workbook.Worksheets[i];
    for (int row = 0; row < 150; row++) {
        for (int col = 0; col < 56; col++) {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

### การบันทึกสมุดงาน
บันทึกสมุดงานของคุณไปยังไดเร็กทอรีที่ระบุ
```csharp
// บันทึกสมุดงาน
string outputFilePath = System.IO.Path.Combine(outputDir, "ACellsSample_out.xlsx");
workbook.Save(outputFilePath);
```
