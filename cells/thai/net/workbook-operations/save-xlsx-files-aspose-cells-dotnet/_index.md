## คู่มือการใช้งาน

เรามาดูขั้นตอนการบันทึกไฟล์ XLSX ทีละขั้นตอนกัน

### ขั้นตอนที่ 1: การตั้งค่าสมุดงานของคุณ

ขั้นแรก ให้สร้างหรือโหลดเวิร์กบุ๊กที่ข้อมูลจะถูกจัดการหรือบันทึก วิธีการมีดังนี้

#### สร้างสมุดงานใหม่
```csharp
// เริ่มต้นอินสแตนซ์เวิร์กบุ๊กใหม่
Workbook workbook = new Workbook();
```
โค้ดสั้นๆ นี้จะเริ่มต้นเวิร์กบุ๊กว่างซึ่งคุณสามารถเติมข้อมูลเข้าไปได้

### ขั้นตอนที่ 2: การกำหนดค่ากระบวนการบันทึก

ตอนนี้ กำหนดค่าวิธีการบันทึกไฟล์ของคุณ:

#### กำหนดค่าการตอบสนอง HTTP สำหรับการดาวน์โหลดไฟล์
หากใช้ ASP.NET และจำเป็นต้องส่งไฟล์เป็นการตอบกลับแบบดาวน์โหลดได้ ให้เริ่มต้น `HttpResponse`-
```csharp
HttpResponse Response = HttpContext.Current.Response;
```

#### บันทึกสมุดงานเป็น XLSX
ใช้โค้ดต่อไปนี้เพื่อบันทึกเวิร์กบุ๊กในรูปแบบ Excel 2007 xlsx:
```csharp
// ให้แน่ใจว่าคำตอบของคุณไม่ใช่ค่าว่างก่อนบันทึก
if (Response != null)
{
    // ระบุตัวเลือกการจัดการเนื้อหาและบันทึกสำหรับรูปแบบ Excel 2007
    workbook.Save(Response, "output.xlsx", ContentDisposition.Attachment, new OoxmlSaveOptions());
    Response.End(); // สิ้นสุดสตรีมการตอบสนอง HTTP
}
```

### คำอธิบายพารามิเตอร์โค้ด
- **`HttpResponse`**: จัดการวิธีการส่งไฟล์ไปยังลูกค้า
- **`ContentDisposition.Attachment`**:สั่งให้เบราว์เซอร์ปฏิบัติต่อเอกสารเป็นไฟล์ที่ดาวน์โหลดได้แทนที่จะแสดงแบบอินไลน์
- **`OoxmlSaveOptions`**:ให้ตัวเลือกที่เฉพาะเจาะจงสำหรับการบันทึกในรูปแบบ OLE2 เช่น XLSX

### เคล็ดลับการแก้ไขปัญหา
ปัญหาทั่วไปที่คุณอาจเผชิญ ได้แก่:
- **ข้อยกเว้นการอ้างอิงว่าง**: ทำให้มั่นใจ `HttpResponse` จะต้องมีการตั้งค่าเริ่มต้นให้ถูกต้องก่อนใช้งาน
- **ไฟล์ไม่สามารถดาวน์โหลด**ตรวจสอบว่าเส้นทางไฟล์และส่วนหัว HTTP ได้รับการกำหนดค่าอย่างถูกต้องสำหรับการดาวน์โหลดไคลเอนต์
