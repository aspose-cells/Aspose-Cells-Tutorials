## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการแปลงไฟล์ Excel ที่มีการเติมไล่ระดับสีเป็นรูปแบบ HTML

### การอ่านและการเตรียมหนังสือฝึกหัด

#### ภาพรวม
ขั้นแรก ให้อ่านไฟล์ Excel ต้นฉบับของคุณซึ่งมีเซลล์ที่มีการไล่ระดับสี
```csharp
// กำหนดไดเรกทอรีสำหรับไฟล์ต้นฉบับและไฟล์เอาท์พุต
double string sourceDir = RunExamples.Get_SourceDirectory();
double string outputDir = RunExamples.Get_OutputDirectory();

// โหลดเวิร์กบุ๊กจากเส้นทางที่ระบุ
Workbook book = new Workbook(sourceDir + "sampleRenderGradientFillToHTML.xlsx");
```

#### คำอธิบาย
- **ที่มาข้อมูล**:ไดเร็กทอรีที่มีไฟล์ Excel ของคุณ
- **เอาท์พุทไดเร็กทอรี**: ปลายทางสำหรับไฟล์ HTML ที่ถูกแปลง

### การบันทึกเวิร์กบุ๊กเป็น HTML

#### ภาพรวม
บันทึกสมุดงานของคุณในรูปแบบ HTML โดยคงการจัดรูปแบบภาพไว้:
```csharp
// บันทึกสมุดงานในรูปแบบ HTML book.Save(outputDir + "outputRenderGradientFillToHTML.html");
```

#### คำอธิบาย
- **วิธีการบันทึก**: ส่งออกเนื้อหาสมุดงานเป็นรูปแบบต่างๆ ที่นี่จะแปลงไฟล์ Excel ที่มีการเติมแบบไล่ระดับเป็นเอกสาร HTML

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบว่าเส้นทางไฟล์ถูกต้องและสามารถเข้าถึงได้
- เพิ่มประสิทธิภาพเวิร์กบุ๊กของคุณก่อนการแปลงหากคุณพบปัญหาด้านประสิทธิภาพเนื่องจากการลบข้อมูลที่ไม่จำเป็นออก

2. **การแบ่งปันข้อมูล**:แบ่งปันข้อมูลที่จัดรูปแบบกับผู้ใช้ที่ไม่มีการเข้าถึง Excel
