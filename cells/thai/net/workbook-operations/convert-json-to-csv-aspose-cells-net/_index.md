## คู่มือการใช้งาน

### ภาพรวมของฟีเจอร์การแปลง

การแปลง JSON เป็น CSV โดยใช้ Aspose.Cells เกี่ยวข้องกับการอ่านไฟล์ JSON และนำเข้าข้อมูลลงในเวิร์กบุ๊ก Excel ก่อนที่จะบันทึกเป็น CSV กระบวนการนี้จะช่วยให้มั่นใจได้ว่าโครงสร้างลำดับชั้นของ JSON จะถูกเก็บรักษาไว้ในรูปแบบตารางแบบแบน

#### ขั้นตอนที่ 1: อ่านไฟล์ JSON

```csharp
// ไดเร็กทอรีแหล่งที่มาที่ไฟล์ JSON ของคุณอยู่
string sourceDir = RunExamples.Get_SourceDirectory();
string jsonFilePath = sourceDir + "SampleJson.json";

// การอ่านเนื้อหาของไฟล์ JSON
string jsonString = File.ReadAllText(jsonFilePath);
```

ที่นี่, `File.ReadAllText` อ่านเนื้อหา JSON ทั้งหมดลงในสตริง นี่คือขั้นตอนแรกของเราในการแปลงข้อมูล

#### ขั้นตอนที่ 2: สร้างและกำหนดค่าเวิร์กบุ๊ก

```csharp
// เริ่มต้นสมุดงานว่างเปล่า
Workbook workbook = new Workbook();

// การเข้าถึงคอลเลกชันเซลล์ของเวิร์กชีตแรก
Cells cells = workbook.Worksheets[0].Cells;

// การกำหนดค่า JsonLayoutOptions สำหรับการตั้งค่าการนำเข้า
JsonLayoutOptions options = new JsonLayoutOptions
{
    ConvertNumericOrDate = true,
    ArrayAsTable = true,
    IgnoreArrayTitle = true,
    IgnoreObjectTitle = true
};
```

การ `JsonLayoutOptions` คลาสนี้มีการตั้งค่าต่างๆ เพื่อปรับแต่งกระบวนการแปลง ตัวอย่างเช่น `ConvertNumericOrDate` รับประกันว่าค่าตัวเลขและวันที่ได้รับการตีความอย่างถูกต้อง

#### ขั้นตอนที่ 3: นำเข้าข้อมูล JSON

```csharp
// การนำเข้าข้อมูลจากสตริง JSON เข้าสู่เซลล์เวิร์กบุ๊กโดยเริ่มจากแถว 0 คอลัมน์ 0
JsonUtility.ImportData(jsonString, cells, 0, 0, options);
```

`JsonUtility.ImportData` วิธีการนำเข้าข้อมูล JSON ลงในเวิร์กชีตและช่วงเซลล์ที่ระบุโดยใช้การกำหนดค่าที่ให้ไว้

#### ขั้นตอนที่ 4: บันทึกเป็น CSV

```csharp
// กำหนดไดเรกทอรีเอาท์พุตสำหรับการบันทึกไฟล์ CSV
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "SampleJson_out.csv");
```

สุดท้ายให้บันทึกสมุดงานของคุณในรูปแบบ CSV `Save` วิธีการนี้มีความหลากหลายและรองรับรูปแบบต่างๆ รวมถึง CSV
