## คู่มือการใช้งาน
ในส่วนนี้เราจะแนะนำการแปลงไฟล์ Excel เป็นรูปแบบ DOCX

### ภาพรวม
การแปลงไฟล์ Excel เป็นเอกสาร Word จะทำให้การนำเสนอข้อมูลคล่องตัวขึ้นและปรับปรุงการทำงานร่วมกันให้ดีขึ้นโดยทำให้การแก้ไขและแชร์ข้อมูลข้ามแพลตฟอร์มง่ายขึ้น

### กระบวนการแปลงทีละขั้นตอน
**1. เปิดไฟล์เทมเพลต**
เริ่มต้นด้วยการโหลดไฟล์ Excel ต้นฉบับของคุณ:
```csharp
// เส้นทางไดเร็กทอรีแหล่งที่มา
string sourceDir = RunExamples.Get_SourceDirectory();

// สร้างการเริ่มต้นวัตถุเวิร์กบุ๊กด้วยเส้นทางของไฟล์ Excel
Workbook workbook = new Workbook(sourceDir + "Book1.xlsx");
```
*คำอธิบาย*: เดอะ `Workbook` คลาสแสดงถึงไฟล์ Excel โดยการส่งเส้นทางของไฟล์ไปยังคอนสตรัคเตอร์ คุณจะโหลดสเปรดชีตลงในหน่วยความจำ

**2. บันทึกเป็น DOCX**
แปลงและบันทึกไฟล์ Excel ที่โหลดเป็นเอกสาร Word:
```csharp
// เส้นทางไดเรกทอรีเอาท์พุต
string outputDir = RunExamples.Get_OutputDirectory();

// บันทึกสมุดงานในรูปแบบ DOCX
workbook.Save(outputDir + "Book1.docx", SaveFormat.Docx);
```
*คำอธิบาย*: เดอะ `Save` วิธีการส่งออกเวิร์กบุ๊กปัจจุบันไปยังรูปแบบต่างๆ ที่นี่ `SaveFormat.Docx` ระบุว่าเราต้องการเอกสาร Word
