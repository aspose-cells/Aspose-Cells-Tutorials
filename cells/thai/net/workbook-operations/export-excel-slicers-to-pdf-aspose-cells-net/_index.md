## คู่มือการใช้งาน
### ส่งออกตัวแบ่ง Excel ไปยัง PDF
ฟีเจอร์นี้ช่วยให้คุณแปลงแผนภูมิสไลเซอร์ Excel เป็นเอกสาร PDF ได้โดยตรง โดยมีวิธีการทำงานดังนี้:
#### ขั้นตอนที่ 1: กำหนดเส้นทางไดเร็กทอรี
ขั้นแรก ตั้งค่าไดเร็กทอรีสำหรับไฟล์ต้นฉบับและไฟล์เอาต์พุตของคุณ แทนที่ `YOUR_SOURCE_DIRECTORY` และ `YOUR_OUTPUT_DIRECTORY` ด้วยเส้นทางจริงบนระบบของคุณ
```csharp
// คุณสมบัติ: ตั้งค่าเส้นทางไดเรกทอรี
string SourceDir = @"C:\\Path\\To\\Your\\ExcelFile";
string OutputDir = @"C:\\Path\\To\\Save\\PDF";
```
#### ขั้นตอนที่ 2: โหลดเวิร์กบุ๊ก
ขั้นตอนต่อไปคือโหลดไฟล์ Excel ของคุณโดยใช้ Aspose.Cells ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกต้องและสามารถเข้าถึงได้
```csharp
// โหลดสมุดงานที่มีอยู่จากไดเร็กทอรีที่ระบุ
Workbook workbook = new Workbook(SourceDir + "SampleSlicerChart.xlsx");
```
#### ขั้นตอนที่ 3: บันทึกเป็น PDF
สุดท้ายให้บันทึกเวิร์กบุ๊กที่โหลดเป็นเอกสาร PDF ในตำแหน่งเอาต์พุตที่คุณต้องการ
```csharp
// บันทึกสมุดงานเป็นไฟล์ PDF ในไดเร็กทอรีเอาท์พุตที่ระบุ
workbook.Save(OutputDir + "SampleSlicerChart.pdf", SaveFormat.Pdf);
```
### คำอธิบายเกี่ยวกับโค้ดสั้นๆ
- **สมุดแบบฝึกหัด**หมายถึงไฟล์ Excel อ็อบเจ็กต์นี้ช่วยให้คุณสามารถจัดการและบันทึกไฟล์ได้
- **บันทึกรูปแบบ PDF**: ระบุว่าควรบันทึกเอกสารในรูปแบบ PDF
กระบวนการง่ายๆ นี้จะส่งออกแผนภูมิสไลเซอร์ของคุณไปยัง PDF อย่างมีประสิทธิภาพ ทำให้พร้อมสำหรับการแชร์หรือการเก็บถาวร
