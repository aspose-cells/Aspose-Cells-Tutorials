## คู่มือการใช้งาน

ในส่วนนี้ เราจะมาสำรวจวิธีการบันทึกเวิร์กบุ๊ก Excel โดยใช้รูปแบบไฟล์ต่างๆ ด้วย Aspose.Cells สำหรับ .NET โดยฟีเจอร์แต่ละอย่างจะแบ่งออกเป็นขั้นตอนที่จัดการได้

### บันทึกเป็น XLS และ XLSX

#### ภาพรวม
การบันทึกเวิร์กบุ๊กของคุณในรูปแบบ Excel ดั้งเดิมหรือแบบใหม่จะช่วยให้มั่นใจในความเข้ากันได้ระหว่างระบบต่างๆ

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก**
```csharp
// กำหนดเส้นทางไฟล์ไปยังเวิร์กบุ๊กแหล่งที่มาของคุณ
string filePath = "path/to/your/Book1.xls";
Workbook workbook = new Workbook(filePath);
```

**ขั้นตอนที่ 2: บันทึกเป็น XLS หรือ XLSX**
```csharp
// ระบุไดเร็กทอรีเอาท์พุตและบันทึกในรูปแบบ Excel 97-2003
workbook.Save("output.xls");

// อีกวิธีหนึ่งคือบันทึกในรูปแบบ XLSX เพื่อความเข้ากันได้กับยุคใหม่
workbook.Save("output.xlsx");
```

### บันทึกเป็น PDF

#### ภาพรวม
การแปลงไฟล์ Excel เป็น PDF เป็นสิ่งสำคัญสำหรับการแชร์ข้อมูลในรูปแบบที่ไม่สามารถแก้ไขได้และสามารถเข้าถึงได้ทั่วไป

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก**
ใช้ขั้นตอนการโหลดแบบเดียวกันกับข้างต้น

**ขั้นตอนที่ 2: บันทึกเป็น PDF**
```csharp
// แปลงและบันทึกสมุดงานเป็นไฟล์ PDF
workbook.Save("output.pdf");
```

### บันทึกเป็น HTML

#### ภาพรวม
การแปลง HTML ช่วยให้สามารถฝังข้อมูล Excel ลงในหน้าเว็บหรือดูในเบราว์เซอร์ได้อย่างง่ายดาย

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก**
ใช้ขั้นตอนการโหลดแบบเดียวกันกับข้างต้น

**ขั้นตอนที่ 2: บันทึกเป็น HTML**
```csharp
// ส่งออกเนื้อหาสมุดงานไปยังไฟล์ HTML
workbook.Save("output.html");
```

### บันทึกเป็น ODS

#### ภาพรวม
รูปแบบ ODS เหมาะอย่างยิ่งสำหรับความเข้ากันได้กับสเปรดชีต OpenOffice หรือ LibreOffice

**ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก**
ใช้ขั้นตอนการโหลดแบบเดียวกันกับข้างต้น

**ขั้นตอนที่ 2: บันทึกเป็น ODS**
```csharp
// บันทึกสมุดงานในรูปแบบ ODS
workbook.Save("output.ods");
```

## การประยุกต์ใช้งานจริง

มาสำรวจสถานการณ์จริงบางอย่างที่ความสามารถในการบันทึกไฟล์เหล่านี้อาจมีค่าอย่างยิ่ง:

1. **การรายงานข้อมูล**:แปลงรายงาน Excel เป็น PDF เพื่อแจกจ่ายไปยังแผนกต่างๆ
2. **การบูรณาการเว็บไซต์**:ฝังแผ่นข้อมูลเวอร์ชัน HTML ลงในอินทราเน็ตขององค์กรหรือเว็บไซต์
3. **การแชร์ข้ามแพลตฟอร์ม**:ใช้รูปแบบ ODS เพื่อการแบ่งปันที่ราบรื่นกับผู้ใช้ที่ชื่นชอบซอฟต์แวร์โอเพ่นซอร์ส
