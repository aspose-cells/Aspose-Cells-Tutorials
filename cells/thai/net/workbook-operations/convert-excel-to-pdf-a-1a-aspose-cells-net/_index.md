## คู่มือการใช้งาน

หัวข้อนี้แบ่งออกเป็นขั้นตอนเชิงตรรกะสำหรับการแปลงไฟล์ Excel เป็นรูปแบบ PDF/A-1a

### ขั้นตอนที่ 1: สร้างเวิร์กบุ๊กและเข้าถึงเวิร์กชีต

**ภาพรวม**ขั้นตอนแรกเกี่ยวข้องกับการสร้างวัตถุเวิร์กบุ๊กและการเข้าถึงเวิร์กชีตซึ่งมีข้อมูลของคุณอยู่

```csharp
// เริ่มต้นสมุดงานใหม่
Workbook wb = new Workbook();

// เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
Worksheet ws = wb.Worksheets[0];
```

### ขั้นตอนที่ 2: เพิ่มข้อมูลลงในเซลล์

**ภาพรวม**:ที่นี่คุณจะได้เรียนรู้วิธีการเพิ่มข้อความหรือข้อมูลลงในเซลล์เฉพาะของแผ่นงาน Excel ของคุณ

```csharp
// เข้าถึงเซลล์ B5 และแทรกข้อความ
Cell cell = ws.Cells["B5"];
cell.PutValue("This PDF format is compatible with PDFA-1a.");
```

### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการบันทึก PDF

**ภาพรวม**:การกำหนดระดับความสอดคล้องสำหรับไฟล์ PDF ที่คุณส่งออกนั้นถือเป็นสิ่งสำคัญเพื่อให้เป็นไปตามมาตรฐานการเก็บถาวร

```csharp
// สร้างอินสแตนซ์ของ PdfSaveOptions และตั้งค่าการปฏิบัติตาม
PdfSaveOptions opts = new PdfSaveOptions();
opts.Compliance = PdfCompliance.PdfA1a;
```

### ขั้นตอนที่ 4: บันทึก Excel เป็น PDFA-1a

**ภาพรวม**สุดท้าย ให้บันทึกสมุดงานของคุณเป็นไฟล์ที่เข้ากันได้กับ PDF/A-1a

```csharp
// กำหนดไดเรกทอรีเอาท์พุตและชื่อไฟล์
string outputDir = RunExamples.Get_OutputDirectory();

// บันทึกสมุดงานเป็นเอกสาร PDF/A-1a
wb.Save(outputDir + "outputCompliancePdfA1a.pdf", opts);
```

**เคล็ดลับการแก้ไขปัญหา**:หากคุณพบปัญหา โปรดตรวจสอบให้แน่ใจว่าเส้นทางเอาต์พุตของคุณได้รับการระบุอย่างถูกต้องและสามารถเข้าถึงได้

- **การจัดการเอกสารทางกฎหมาย**:ให้แน่ใจว่าเอกสารทางกฎหมายได้รับการเก็บรักษาในรูปแบบที่ตรงตามข้อกำหนดด้านกฎระเบียบ
