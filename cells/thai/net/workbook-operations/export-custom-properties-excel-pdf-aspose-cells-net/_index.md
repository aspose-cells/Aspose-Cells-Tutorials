# วิธีการส่งออกคุณสมบัติที่กำหนดเองจาก Excel ไปยัง PDF โดยใช้ Aspose.Cells .NET

## คู่มือการใช้งาน

### คุณสมบัติ: ส่งออกคุณสมบัติที่กำหนดเองไปยัง PDF

คุณลักษณะนี้จะแสดงวิธีการส่งออกคุณสมบัติแบบกำหนดเองจากไฟล์ Excel ไปยัง PDF โดยใช้ Aspose.Cells สำหรับ .NET

#### ภาพรวม

การส่งออกคุณสมบัติที่กำหนดเองช่วยให้ผู้ใช้สามารถเก็บรักษาข้อมูลเมตาไว้ได้เมื่อเปลี่ยนรูปแบบข้อมูล ซึ่งถือเป็นสิ่งสำคัญในการรักษาบริบทและที่มาในเวิร์กโฟลว์การจัดทำเอกสาร

#### การดำเนินการแบบทีละขั้นตอน

**1. ตั้งค่าไดเรกทอรี**

กำหนดไดเร็กทอรีแหล่งที่มา (ที่เก็บไฟล์ Excel ของคุณ) และไดเร็กทอรีเอาต์พุต (สำหรับ PDF)

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // เส้นทางไดเรกทอรีอินพุต
string OutputDir = @"YOUR_OUTPUT_DIRECTORY"; // เส้นทางไดเรกทอรีเอาท์พุต
```

**2. โหลดสมุดงาน Excel**

โหลดเวิร์กบุ๊กที่มีคุณสมบัติแบบกำหนดเอง

```csharp
Workbook workbook = new Workbook(SourceDir + "sampleWithCustProps.xlsx");
```

**3. กำหนดค่าตัวเลือกการบันทึก PDF**

สร้างและกำหนดค่า `PdfSaveOptions` เพื่อรวมคุณสมบัติที่กำหนดเองใน PDF

```csharp
PdfSaveOptions pdfSaveOpt = new PdfSaveOptions();
pdfSaveOpt.CustomPropertiesExport = Rendering.PdfCustomPropertiesExport.Standard;
```

**4. ส่งออกสมุดงานเป็น PDF**

สุดท้าย ให้บันทึกสมุดงานเป็น PDF พร้อมคุณสมบัติแบบกำหนดเองรวมอยู่ด้วย

```csharp
workbook.Save(OutputDir + "outSampleWithCustProps.pdf", pdfSaveOpt);
```

### คุณสมบัติ: โหลดสมุดงานจากไฟล์

การโหลดไฟล์ Excel ลงในหน่วยความจำทำได้โดยตรงโดยใช้ Aspose.Cells

#### ภาพรวม

ฟังก์ชันนี้ช่วยให้คุณสามารถเปิดและจัดการไฟล์ Excel ที่มีอยู่ผ่านโปรแกรมได้

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดไดเรกทอรีแหล่งที่มา**

ตั้งค่าเส้นทางไดเร็กทอรีสำหรับไฟล์ต้นฉบับของคุณ

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // เส้นทางไดเรกทอรีอินพุต
```

**2. โหลดสมุดงาน**

โหลดไฟล์ Excel ลงใน `Workbook` วัตถุ.

```csharp
Workbook workbook = new Workbook(SourceDir + "sampleWithCustProps.xlsx");
```

### คุณสมบัติ: กำหนดค่าตัวเลือกการบันทึก PDF

การกำหนดค่าตัวเลือกการบันทึกจะปรับแต่งวิธีสร้างเอกสาร PDF จากไฟล์ Excel ของคุณ

#### ภาพรวม

ผ่าน `PdfSaveOptions`คุณสามารถควบคุมด้านต่างๆ เช่น การส่งออกคุณสมบัติแบบกำหนดเอง และการตั้งค่าเฉพาะ PDF อื่นๆ

#### การดำเนินการแบบทีละขั้นตอน

**1. เริ่มต้นใช้งาน PdfSaveOptions**

เริ่มต้นด้วยการกำหนดค่าเริ่มต้นสำหรับการบันทึกเป็น PDF

```csharp
PdfSaveOptions pdfSaveOpt = new PdfSaveOptions();
```

**2. ตั้งค่าตัวเลือกการส่งออกคุณสมบัติที่กำหนดเอง**

ตรวจสอบให้แน่ใจว่าคุณสมบัติที่กำหนดเองมาตรฐานจะถูกส่งออกไปยัง PDF ในระหว่างการแปลง

```csharp
pdfSaveOpt.CustomPropertiesExport = Rendering.PdfCustomPropertiesExport.Standard;
```
