# โหลดไฟล์ Excel โดยไม่ใช้ VBA Macro โดยใช้ Aspose.Cells สำหรับ .NET | คู่มือการดำเนินการเวิร์กบุ๊ก

## การแนะนำ
คุณกำลังประสบปัญหาในการใช้ไฟล์ Excel ที่มีแมโคร VBA หรือไม่? คำแนะนำที่ครอบคลุมของเราเกี่ยวกับการใช้ **Aspose.Cells สำหรับ .NET** จะปฏิวัติเวิร์กโฟลว์ของคุณโดยให้คุณโหลดไฟล์เหล่านี้ได้โดยไม่ต้องมีคอมโพเนนต์ VBA ที่ฝังอยู่ คุณลักษณะนี้ช่วยขจัดความซับซ้อนที่ไม่จำเป็นและเพิ่มประสิทธิภาพเมื่อต้องจัดการกับเวิร์กบุ๊กขนาดใหญ่หรือที่มีแมโคร

ในบทช่วยสอนนี้ คุณจะได้เรียนรู้วิธีการกำหนดค่า Aspose.Cells เพื่อไม่รวมแมโคร VBA ขณะโหลดเวิร์กบุ๊ก Excel ซึ่งจะช่วยประหยัดเวลาและทรัพยากรในแอปพลิเคชัน .NET ของคุณ ไม่ว่าคุณจะเป็นนักพัฒนาที่กำลังมองหาวิธีการประมวลผลข้อมูลที่มีประสิทธิภาพหรือผู้ที่กำลังหาทางปรับปรุงประสิทธิภาพของแอปพลิเคชัน คู่มือนี้เหมาะสำหรับคุณ

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีตั้งค่า Aspose.Cells สำหรับ .NET
- การกำหนดค่าตัวเลือกการโหลดเพื่อไม่รวมแมโคร VBA
- การโหลดเวิร์กบุ๊กโดยไม่จำเป็นต้องอาศัยส่วนประกอบ VBA
- บันทึกไฟล์ Excel ในรูปแบบเฉพาะโดยยังคงคุณสมบัติที่จำเป็นไว้

ก่อนที่จะเริ่มใช้งาน เรามาตรวจสอบให้แน่ใจก่อนว่าคุณเตรียมทุกอย่างพร้อมแล้ว

## คู่มือการใช้งาน
ในส่วนนี้ เราจะศึกษาวิธีการกำหนดค่าตัวเลือกการโหลดและจัดการเวิร์กบุ๊ก Excel โดยใช้ Aspose.Cells สำหรับ .NET

### คุณสมบัติ 1: การกำหนดค่า LoadOptions

#### ภาพรวม
ฟีเจอร์แรกมุ่งเน้นไปที่การกำหนดค่าตัวเลือกการโหลดเพื่อไม่รวมแมโคร VBA เมื่อโหลดเวิร์กบุ๊ก Excel ซึ่งมีประโยชน์อย่างยิ่งหากคุณจำเป็นต้องประมวลผลข้อมูลโดยไม่ต้องมีสคริปต์ฝังตัว

**การดำเนินการแบบทีละขั้นตอน**

1. **สร้างอินสแตนซ์ใหม่ของ LoadOptions**
   เริ่มต้นด้วยการสร้าง `LoadOptions` วัตถุโดยตั้งค่าให้ตรวจจับรูปแบบไฟล์โดยอัตโนมัติ
   
    ```csharp
    LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
    ```

2. **ไม่รวม VBA Macros โดยใช้ LoadFilter**
   กำหนดค่าตัวกรองเพื่อไม่รวมแมโคร VBA แต่อนุญาตประเภทข้อมูลอื่น

    ```csharp
    loadOptions.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.VBA);
    ```

### คุณสมบัติที่ 2: การโหลดเวิร์กบุ๊กโดยไม่ต้องใช้ VBA

#### ภาพรวม
ต่อไปเราจะสาธิตวิธีใช้ที่กำหนดค่าไว้ `LoadOptions` เพื่อเปิดเวิร์กบุ๊กโดยไม่รวมส่วนประกอบ VBA

**การดำเนินการแบบทีละขั้นตอน**

1. **กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต**
   ตรวจสอบให้แน่ใจว่าคุณได้ระบุเส้นทางไดเร็กทอรีของคุณที่เก็บไฟล์ Excel ของคุณและที่ที่ควรบันทึกเอาต์พุต
   
    ```csharp
    string sourceDir = "YOUR_SOURCE_DIRECTORY";
    string outputDir = "YOUR_OUTPUT_DIRECTORY";
    ```

2. **โหลดเวิร์กบุ๊กด้วย VBA ที่ถูกแยกออก**

    ```csharp
    Workbook workbook = new Workbook(sourceDir + "/sampleMacroEnabledWorkbook.xlsm", loadOptions);
    ```
   ตอนนี้เวิร์กบุ๊กโหลดโดยไม่มีแมโคร VBA แล้ว ขอบคุณการกำหนดค่าของเรา `loadOptions`-

### คุณสมบัติที่ 3: การบันทึกสมุดงานในรูปแบบเฉพาะ

#### ภาพรวม
สุดท้าย เราจะบันทึกเวิร์กบุ๊กที่แก้ไขแล้วในรูปแบบเฉพาะโดยยังคงคุณลักษณะที่ไม่ใช่ VBA ไว้

**การดำเนินการแบบทีละขั้นตอน**

1. **บันทึกสมุดงานในรูปแบบ XLSM**
   ใช้ `Save` วิธีการเก็บสมุดงานของคุณด้วยการตั้งค่าที่ต้องการ
   
    ```csharp
    workbook.Save(outputDir + "/OutputSampleMacroEnabledWorkbook.xlsm", SaveFormat.Xlsm);
    ```
