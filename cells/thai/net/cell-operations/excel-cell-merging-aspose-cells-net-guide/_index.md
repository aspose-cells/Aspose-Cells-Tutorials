### คู่มือการใช้งาน

#### การผสานเซลล์ในเวิร์กชีต

**ภาพรวม:**
การผสานเซลล์จะช่วยรวมข้อมูลเพื่อให้สามารถอ่านและนำเสนอได้ดีขึ้น หัวข้อนี้จะแนะนำคุณเกี่ยวกับการผสานเซลล์เฉพาะโดยใช้ Aspose.Cells

1. **สร้างสมุดงานใหม่**
   เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Workbook` คลาสซึ่งแสดงถึงไฟล์ Excel
   
   ```csharp
   Workbook workbook = new Workbook();
   ```

2. **เข้าถึงแผ่นงาน**
   เข้าถึงแผ่นงานแรกจากสมุดงานของคุณ:
   
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **การปรับเปลี่ยนและรวมเซลล์**
   เพิ่มค่าลงในเซลล์ที่ระบุ จากนั้นผสานเซลล์ในช่วงที่ต้องการ
   
   ```csharp
   // ตั้งค่าของ “A1”
   Cell cell = worksheet.Cells["A1"];
   cell.PutValue("Visit Aspose!");

   // รวมเซลล์จาก A1 ถึง C1 (ดัชนีฐาน 0)
   worksheet.Cells.Merge(0, 0, 1, 3);
   ```

4. **บันทึกสมุดงานของคุณ**
   บันทึกสมุดงานในรูปแบบที่คุณต้องการ:
   
   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "/merged_cells_output.xls", SaveFormat.Excel97To2003);
   ```

#### การสร้างไดเรกทอรีสำหรับการดำเนินการไฟล์

**ภาพรวม:**
ตรวจสอบให้แน่ใจว่าคุณมีไดเรกทอรีที่สามารถบันทึกไฟล์ Excel ได้ ตรวจสอบและสร้างไดเรกทอรีหากไม่มี

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// ตรวจสอบและสร้างไดเรกทอรีหากไม่มีอยู่
bool isExists = Directory.Exists(outputDir);
if (!isExists)
{
    Directory.CreateDirectory(outputDir);
}
```
