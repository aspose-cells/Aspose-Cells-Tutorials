## คู่มือการใช้งาน
ในส่วนนี้ เราจะแนะนำการเพิ่มและการจัดการคุณสมบัติ ContentType โดยใช้ Aspose.Cells สำหรับ .NET

### การเพิ่มคุณสมบัติ ContentType
Aspose.Cells ทำให้การเพิ่มคุณสมบัติแบบกำหนดเองที่สามารถใช้เพื่อวัตถุประสงค์ต่าง ๆ เช่น การกำหนดข้อมูลเมตาหรือการติดตามข้อมูลเพิ่มเติมเกี่ยวกับเวิร์กบุ๊ก Excel ของคุณเป็นเรื่องง่ายดาย

#### ภาพรวมทีละขั้นตอน
1. **สร้างสมุดงานใหม่:** เริ่มต้นอินสแตนซ์ใหม่ของ `Workbook` ระดับ.
2. **เพิ่มคุณสมบัติ ContentType:** ใช้ `ContentTypeProperties.Add()` วิธีการรวมคุณสมบัติที่กำหนดเอง
3. **กำหนดค่าคุณสมบัติที่สามารถยกเลิกได้:** ตั้งค่าว่าจะสามารถยกเลิกคุณสมบัติแต่ละอย่างได้หรือไม่

#### การนำโค้ดไปใช้
```csharp
using Aspose.Cells.WebExtensions;
using System;

namespace Aspose.Cells.Examples.CSharp._Workbook
{
    public class WorkingWithContentTypeProperties
    {
        public static void Run()
        {
            // เริ่มต้นเวิร์กบุ๊กใหม่ในรูปแบบ XLSX
            Workbook workbook = new Workbook(FileFormatType.Xlsx);
            
            // เพิ่มคุณสมบัติ ContentType ของสตริง "MK31"
            int index1 = workbook.ContentTypeProperties.Add("MK31", "Simple Data");
            workbook.ContentTypeProperties[index1].IsNillable = false;
            
            // เพิ่มคุณสมบัติ DateTime ContentType "MK32"
            int index2 = workbook.ContentTypeProperties.Add("MK32", DateTime.Now.ToString("yyyy-MM-dd'T'hh:mm:ss"), "DateTime");
            workbook.ContentTypeProperties[index2].IsNillable = true;

            // บันทึกสมุดงาน
            string outputDir = RunExamples.Get_OutputDirectory();
            workbook.Save(outputDir + "WorkingWithContentTypeProperties_out.xlsx");

            Console.WriteLine("ContentType Properties added successfully.");
        }
    }
}
```

### คำอธิบายพารามิเตอร์และวิธีการ
- **วิธีการเพิ่ม:** การ `Add` วิธีนี้ใช้ตัวระบุที่ไม่ซ้ำกัน ค่า และประเภทเนื้อหาที่เป็นทางเลือก
  - **พารามิเตอร์:**
    - ตัวระบุ (สตริง): ชื่อเฉพาะสำหรับคุณสมบัติ
    - ค่า (วัตถุ): ข้อมูลที่เกี่ยวข้องกับคุณสมบัตินี้
    - ประเภทเนื้อหา (ตัวเลือก, สตริง): ระบุประเภทข้อมูล เช่น "DateTime"
- **ไร้สาระได้:** บูลีนที่ระบุว่าสามารถปล่อยคุณสมบัติว่างไว้ได้หรือไม่

### เคล็ดลับการแก้ไขปัญหา
- ให้แน่ใจว่ามีตัวระบุที่ไม่ซ้ำกันสำหรับแต่ละคุณสมบัติ ContentType เพื่อหลีกเลี่ยงความขัดแย้ง
- ตรวจสอบว่าใช้ประเภทข้อมูลที่ถูกต้องเมื่อเพิ่มคุณสมบัติ
