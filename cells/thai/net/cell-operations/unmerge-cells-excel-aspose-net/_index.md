## คู่มือการใช้งาน: การแยกเซลล์ที่ผสานกัน

เมื่อตั้งค่าทุกอย่างเสร็จเรียบร้อยแล้ว ให้เราเน้นที่การแยกเซลล์ที่ผสานกันโดยใช้ Aspose.Cells

### ภาพรวม

การยกเลิกการผสานเซลล์ถือเป็นสิ่งสำคัญสำหรับงานจัดการข้อมูลซึ่งจำเป็นต้องใช้ค่าเซลล์แต่ละเซลล์ กระบวนการนี้ทำได้โดยตรงด้วย Aspose.Cells

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

เริ่มต้นด้วยการโหลดเวิร์กบุ๊ก Excel จากไดเร็กทอรีแหล่งที่มาของคุณ:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook wbk = new Workbook(SourceDir + "/sampleUnMergingtheMergedCells.xlsx");
```

**เหตุใดจึงก้าวขั้นนี้?** มันเริ่มต้นการ `Workbook` วัตถุกับไฟล์ Excel ที่คุณตั้งใจจะจัดการ

#### ขั้นตอนที่ 2: เข้าถึงแผ่นงาน

ขั้นตอนต่อไปคือเข้าถึงเวิร์กชีตที่มีเซลล์ที่ผสานกัน:

```csharp
Worksheet worksheet = wbk.Worksheets[0];
```

บรรทัดนี้จะดึงเวิร์กชีตแรก ปรับดัชนีหากชีตเป้าหมายของคุณแตกต่างกัน

#### ขั้นตอนที่ 3: แยกเซลล์ออกจากกัน

ใช้ `UnMerge` วิธีการแยกช่วงเซลล์ที่เฉพาะเจาะจง:

```csharp
Cells cells = worksheet.Cells;
cells.UnMerge(5, 2, 2, 3);
```

**คำอธิบายพารามิเตอร์:**
- **แถวเริ่มต้น (5)** และ **คอลัมน์เริ่มต้น (2)**: ระบุจุดเริ่มต้นของภูมิภาคที่รวมกัน
- **จำนวนแถวทั้งหมดที่จะยกเลิกการรวม (2)** และ **จำนวนคอลัมน์ทั้งหมดที่จะยกเลิกการรวม (3)**: กำหนดขนาดของพื้นที่ที่ต้องการยกเลิกการผสาน

#### ขั้นตอนที่ 4: บันทึกสมุดงาน

สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณกลับไปยังไฟล์:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
wbk.Save(outputDir + "/outputUnMergingtheMergedCells.xlsx");
```
