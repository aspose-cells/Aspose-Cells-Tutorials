## คู่มือการใช้งาน

### การย้ายช่วงเซลล์

ในส่วนนี้ เราจะใช้งานฟังก์ชันหลัก: การย้ายช่วงเซลล์

#### ภาพรวม

เป้าหมายคือการเปลี่ยนตำแหน่งพื้นที่เฉพาะภายในเวิร์กชีต Excel ซึ่งอาจมีประโยชน์สำหรับการจัดระเบียบข้อมูลหรือการปรับเค้าโครงแบบไดนามิก

#### การดำเนินการแบบทีละขั้นตอน

**1. กำหนดไดเรกทอรีต้นทางและปลายทาง**

ขั้นแรก ให้ระบุไดเร็กทอรีแหล่งที่มาของคุณ (ที่ไฟล์ Excel เริ่มต้นของคุณอยู่) และไดเร็กทอรีเอาต์พุต (ที่คุณจะบันทึกไฟล์ที่แก้ไข)

```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();
```

**2. เปิดสมุดงาน Excel**

โหลดเวิร์กบุ๊กโดยใช้ Aspose.Cells:

```csharp
Workbook workbook = new Workbook(sourceDir + "sampleMoveRangeOfCells.xlsx");
```

**3. เข้าถึงเซลล์เวิร์กชีต**

เข้าถึงเซลล์จากเวิร์กชีตแรก:

```csharp
Cells cells = workbook.Worksheets[0].Cells;
```

**4. สร้างพื้นที่เซลล์และย้ายมัน**

ระบุช่วงที่ต้องการย้าย (เช่น A1:C5) และเลื่อนไป 7 แถวและ 5 คอลัมน์

```csharp
CellArea ca = CellArea.CreateCellArea("A1", "C5");
cells.MoveRange(ca, 7, 5);
```

**5. บันทึกสมุดงานที่แก้ไขแล้ว**

สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณลงในไฟล์ใหม่:

```csharp
workbook.Save(outputDir + "outputMoveRangeOfCells.xlsx");
Console.WriteLine("MoveRangeOfCells executed successfully.");
```
