# เรียนรู้การทำงานอัตโนมัติของ Excel: สร้างเวิร์กบุ๊กและเพิ่มรายการกล่องโดยใช้ Aspose.Cells สำหรับ .NET

## การแนะนำ

คุณกำลังมองหาวิธีทำให้ Excel ทำงานอัตโนมัติอย่างมีประสิทธิภาพหรือไม่ ไม่ว่าจะเป็นการตั้งค่าสเปรดชีตที่ซับซ้อนหรือการเพิ่มองค์ประกอบแบบโต้ตอบ เช่น ListBox **ระบบอัตโนมัติของ Excel** สามารถประหยัดเวลาการทำงานด้วยมือได้หลายชั่วโมง ด้วย **Aspose.Cells สำหรับ .NET**คุณมีเครื่องมืออันทรงพลังที่จะช่วยให้งานเหล่านี้ง่ายขึ้น และทำให้สามารถสร้างและจัดการไฟล์ Excel ในแอปพลิเคชันของคุณได้อย่างราบรื่น

## คู่มือการใช้งาน

### การตั้งค่าเวิร์กบุ๊กและเวิร์กชีต

**ภาพรวม:**
ขั้นตอนแรกคือการสร้างเวิร์กบุ๊กใหม่และเข้าถึงเวิร์กชีตของเวิร์กบุ๊กนั้น ซึ่งถือเป็นรากฐานของงานอัตโนมัติของ Excel ของคุณ

#### สร้างสมุดงานใหม่
```csharp
Workbook workbook = new Workbook(); // สร้างวัตถุเวิร์กบุ๊กใหม่
```

ที่นี่เราจะสร้างตัวอย่าง `Workbook`ซึ่งแสดงถึงไฟล์ Excel ทั้งหมด

#### เข้าถึงแผ่นงานแรก
```csharp
Worksheet sheet = workbook.getWorksheets().get(0); // ดึงข้อมูลแผ่นงานแรก
```

การเข้าถึงเวิร์กชีตแรกช่วยให้คุณสามารถเริ่มกรอกข้อมูลและตัวควบคุมลงในเวิร์กชีตได้

#### รับคอลเลกชันเซลล์
```csharp
Cells cells = sheet.getCells(); // เข้าถึงเซลล์ทั้งหมดในเวิร์กชีต
```

คอลเลกชั่นนี้ช่วยให้เราจัดการแต่ละเซลล์หรือช่วงเซลล์ภายในชีตได้

### การเพิ่มข้อความและการจัดรูปแบบเซลล์

**ภาพรวม:**
ปรับปรุงแผ่นงาน Excel ของคุณด้วยการเพิ่มข้อความลงในเซลล์และใช้รูปแบบเช่นการจัดรูปแบบตัวหนาเพื่อเน้นย้ำ

#### ป้อนข้อความลงในเซลล์
```csharp
cells.get("B3").putValue("Choose Dept:");
```

โค้ดนี้จะป้อนสตริง "เลือกแผนก:" ลงในเซลล์ B3

#### ตั้งค่ารูปแบบเซลล์เป็นตัวหนา
```csharp
Style style = cells.get("B3").getStyle();
style.getFont().setBold(true);
cells.get("B3").setStyle(style);
```

ที่นี่ เราจะดึงและปรับเปลี่ยนรูปแบบของเซลล์ B3 เพื่อทำให้ข้อความเป็นตัวหนา เพื่อให้มองเห็นได้ชัดเจนมากขึ้น

### การป้อนค่ารายการและการเพิ่มตัวควบคุม ListBox

**ภาพรวม:**
เติมค่าเซลล์ด้วยรายการที่สามารถเลือกได้ผ่านการควบคุม ListBox ซึ่งจะเพิ่มการโต้ตอบให้กับแผ่นงานของคุณ

#### ป้อนค่ารายการลงในเซลล์
```csharp
cells.get("A2").putValue("Sales");
cells.get("A3").putValue("Finance");
// ต่อสำหรับแผนกอื่นๆ...
```

การดำเนินการนี้จะเติมเซลล์ด้วยชื่อแผนก และตั้งค่าตัวเลือกสำหรับ ListBox

#### เพิ่มและกำหนดค่าตัวควบคุม ListBox
```csharp
Aspose.Cells.Drawing.ListBox listBox = sheet.getShapes().addListBox(2, 0, 3, 0, 122, 100);
listBox.setPlacement(PlacementType.FreeFloating);
cells.get("A1").setValue(listBox.getName());
string tempLinkedCell = "A1";
listBox.setLinkedCell(tempLinkedCell);
listBox.setInputRange("A2:A7");
cells.get(tempLinkedCell).setValue(listBox.getName());
string tempInputRange = "A2:A7";
listBox.setInputRange(tempInputRange);
cells.get("A1").setFormula(RangeUtility.getReferenceFromHSSFRangeName(tempLinkedCell));
listBox.setSelectionType(SelectionType.Single);
listBox.setShadow(true);
```

ListBox จะถูกเพิ่มลงในเวิร์กชีต เชื่อมโยงกับเซลล์ A1 สำหรับเอาต์พุต และกำหนดค่าด้วยช่วงตัวเลือกต่างๆ

### การบันทึกสมุดงาน

**ภาพรวม:**
ตรวจสอบว่างานของคุณไม่สูญหายโดยการบันทึกสมุดงานไปยังไดเร็กทอรีที่ระบุ

#### บันทึกสมุดงาน
```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/book1.out.xls";
workbook.save(outputFilePath);
```

การดำเนินการนี้จะบันทึกไฟล์ Excel ของคุณพร้อมการเปลี่ยนแปลงทั้งหมดโดยใช้เส้นทางที่กำหนดไว้
