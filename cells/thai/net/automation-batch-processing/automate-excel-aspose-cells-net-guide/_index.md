## คู่มือการใช้งาน
มาแบ่งกระบวนการออกเป็นขั้นตอนที่จัดการได้เพื่อสร้างและกำหนดรูปแบบเวิร์กบุ๊ก Excel โดยใช้ Aspose.Cells สำหรับ .NET

### การสร้างสมุดงานใหม่
**ภาพรวม:** เริ่มต้นด้วยการสร้างตัวอย่าง `Workbook` วัตถุซึ่งแสดงถึงไฟล์ Excel ทั้งหมด
```csharp
// สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กใหม่
Workbook workbook = new Workbook();
```

### การเข้าถึงเวิร์กชีตและเซลล์
**ภาพรวม:** หลังจากสร้างเวิร์กบุ๊กแล้ว ให้เข้าถึงเวิร์กชีตและเซลล์ที่เจาะจงเพื่อจัดการเนื้อหา
```csharp
// รับการอ้างอิงถึงแผ่นงานแรก
Worksheet worksheet = workbook.Worksheets[0];

// เข้าถึงเซลล์ "A1"
Cell cell = worksheet.Cells["A1"];
```

### การตั้งค่าค่าเซลล์
**ภาพรวม:** ตั้งค่าสำหรับเซลล์เป้าหมาย ตัวอย่างนี้จะเพิ่มข้อความลงในเซลล์ "A1"
```csharp
// เพิ่มค่าลงในเซลล์ "A1"
cell.PutValue("Visit Aspose!");
```

### การใช้การตั้งค่าสไตล์
**ภาพรวม:** ปรับแต่งรูปแบบต่างๆ เช่น การจัดตำแหน่งข้อความและการย่อขนาดให้พอดี
```csharp
// ดึงข้อมูลและปรับเปลี่ยนการตั้งค่ารูปแบบสำหรับเซลล์
Style style = cell.GetStyle();
style.ShrinkToFit = true;
cell.SetStyle(style);
```

### การบันทึกสมุดงาน
**ภาพรวม:** บันทึกสมุดงานของคุณในรูปแบบที่ต้องการ เช่น Excel 97-2003 หรือรูปแบบใหม่กว่า
```csharp
// บันทึกสมุดงานเป็นไฟล์ Excel
workbook.Save("YOUR_OUTPUT_DIRECTORY/book1.out.xls", SaveFormat.Excel97To2003);
```
