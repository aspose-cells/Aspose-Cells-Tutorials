## คู่มือการใช้งาน

ทำตามขั้นตอนเหล่านี้เพื่อกรองเซลล์ว่างโดยอัตโนมัติโดยใช้ Aspose.Cells:

### การโหลดสมุดงาน Excel
สร้างและโหลด `Workbook` วัตถุ:
```csharp
// สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก
Workbook workbook = new Workbook(sourceDir + "sampleBlank.xlsx");
```
นี่เป็นการเริ่มต้นไฟล์เพื่อการจัดการ

### การเข้าถึงแผ่นงาน
เข้าถึงแผ่นงานที่ต้องการเพื่อใช้ตัวกรองอัตโนมัติ:
```csharp
// การเข้าถึงเวิร์กชีตแรกในไฟล์ Excel
Worksheet worksheet = workbook.Worksheets[0];
```
ดัชนี `0` หมายถึงแผ่นแรก ปรับเปลี่ยนตามความจำเป็น

### การใช้ตัวกรองอัตโนมัติกับเซลล์ว่าง
ใช้ `MatchBlanks()` การกรองเซลล์ว่าง:
```csharp
// ใช้ตัวกรองอัตโนมัติสำหรับช่องว่างในคอลัมน์แรก
worksheet.AutoFilter.MatchBlanks(0);
```
ปรับดัชนีสำหรับคอลัมน์ที่แตกต่างกัน

### สดชื่นและประหยัด
รีเฟรชเพื่อใช้การเปลี่ยนแปลง จากนั้นบันทึก:
```csharp
// รีเฟรชแผ่นงาน
dworksheet.AutoFilter.Refresh();

// บันทึกสมุดงานที่แก้ไขแล้ว
workbook.Save(outputDir + "outSampleBlank.xlsx");
```

### เคล็ดลับการแก้ไขปัญหา
- **ไม่พบไฟล์**: ตรวจสอบ `sourceDir` เส้นทาง.
- **ดัชนีอยู่นอกช่วง**: ตรวจสอบดัชนีแผ่นงานและคอลัมน์ว่าถูกต้อง
