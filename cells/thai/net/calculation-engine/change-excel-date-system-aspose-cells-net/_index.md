# เปลี่ยนระบบวันที่ของ Excel เป็น 1904 โดยใช้ Aspose.Cells .NET

## การแนะนำ

คุณกำลังประสบปัญหาในการใช้ระบบวันที่ 1899 ที่เป็นค่าเริ่มต้นในเวิร์กบุ๊ก Excel ของคุณหรือไม่ การเปลี่ยนไปใช้ระบบวันที่ 1904 มักจำเป็นสำหรับความเข้ากันได้หรือข้อกำหนดเฉพาะของแต่ละภูมิภาค บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ Aspose.Cells .NET เพื่อเปลี่ยนระบบวันที่ของเวิร์กบุ๊กของคุณได้อย่างง่ายดาย

### สิ่งที่คุณจะได้เรียนรู้:
- วิธีการเปลี่ยนระบบวันที่ของ Excel จากปี พ.ศ. 2442 ไปเป็น พ.ศ. 2447
- ขั้นตอนการโหลดและบันทึกเวิร์กบุ๊ก Excel ด้วยการตั้งค่าใหม่
- คุณลักษณะหลักของ Aspose.Cells .NET สำหรับการจัดการไฟล์ Excel

มาดูกันว่าคุณจะนำการเปลี่ยนแปลงเหล่านี้ไปใช้ได้อย่างราบรื่นอย่างไร ตรวจสอบให้แน่ใจว่าคุณปฏิบัติตามข้อกำหนดเบื้องต้นทั้งหมดก่อนที่เราจะดำเนินการต่อไป

## คู่มือการใช้งาน

เราจะแบ่งคู่มือนี้ออกเป็นสองส่วนหลักตามฟังก์ชันการทำงาน

### เปลี่ยนระบบวันที่สมุดงาน Excel

#### ภาพรวม
คุณลักษณะนี้จะเปลี่ยนระบบวันที่ของเวิร์กบุ๊ก Excel จากค่าเริ่มต้น (พ.ศ. 2442) เป็น พ.ศ. 2447 ซึ่งจำเป็นสำหรับความเข้ากันได้หรือข้อกำหนดเฉพาะของแต่ละภูมิภาค

##### การดำเนินการทีละขั้นตอน:

**1. เปิดไฟล์ Excel**
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "sampleImplement1904DateSystem.xlsx");
```
ที่นี่, `Workbook` จะเริ่มต้นด้วยเส้นทางไฟล์ที่มีอยู่เพื่อโหลดเอกสาร Excel ของคุณ

**2. เปลี่ยนระบบวันที่**
```csharp
workbook.Settings.Date1904 = true;
```
บรรทัดนี้จะกำหนดระบบวันที่ของสมุดงานเป็นปี 1904 โดยการแก้ไข `Date1904` คุณสมบัติ.

**3. บันทึกสมุดงานที่อัปเดต**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outputImplement1904DateSystem_1904DateSystem.xlsx");
```
เวิร์กบุ๊กจะถูกบันทึกด้วยชื่อใหม่ ซึ่งสะท้อนถึงการกำหนดค่าระบบวันที่อัปเดต

### โหลดและบันทึกสมุดงาน

#### ภาพรวม
เรียนรู้วิธีโหลดไฟล์ Excel จากไดเร็กทอรีและบันทึกไว้ที่อื่นอย่างมีประสิทธิภาพโดยใช้ Aspose.Cells

##### การดำเนินการทีละขั้นตอน:

**1. เปิดไฟล์ Excel**
```csharp
Workbook workbook = new Workbook(SourceDir + "sampleImplement1904DateSystem.xlsx");
```
ขั้นตอนนี้จะคล้ายกับตัวอย่างก่อนหน้าของเรา โดยที่เราเปิดเวิร์กบุ๊กเพื่อการจัดการ

**2. บันทึกสมุดงาน**
```csharp
workbook.Save(outputDir + "outputSaveWorkbook.xlsx");
```
ที่นี่สมุดงานจะถูกบันทึกไปยังตำแหน่งใหม่โดยมีชื่อไฟล์ที่ระบุ

## การประยุกต์ใช้งานจริง

1. **การปฏิบัติตามข้อกำหนดในระดับภูมิภาค**:การเปลี่ยนแปลงระบบวันที่เพื่อให้เป็นไปตามมาตรฐานและระเบียบข้อบังคับท้องถิ่น
2. **การโยกย้ายข้อมูล**:การรับประกันความสอดคล้องของข้อมูลระหว่างการโยกย้ายระหว่าง Excel เวอร์ชันต่างๆ หรือการตั้งค่าภูมิภาค
3. **ความสามารถในการทำงานร่วมกัน**:ปรับปรุงความเข้ากันได้เมื่อแชร์ไฟล์กับผู้ใช้ในภูมิภาคที่ใช้ระบบวันที่ 1904 ตามค่าเริ่มต้น

## การพิจารณาประสิทธิภาพ

- **การเพิ่มประสิทธิภาพการใช้ทรัพยากร**: ปิดเวิร์กบุ๊กทันทีหลังจากการประมวลผลเพื่อเพิ่มหน่วยความจำ
- **แนวทางปฏิบัติที่ดีที่สุด**:ใช้ Aspose.Cells ภายในบล็อก try-catch เพื่อจัดการข้อยกเว้นอย่างเหมาะสมและรับรองประสิทธิภาพการทำงานของแอปพลิเคชันที่ราบรื่น
