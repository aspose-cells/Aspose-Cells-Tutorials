# แยกภาพพื้นหลัง ODS โดยใช้ Aspose.Cells สำหรับ .NET: คำแนะนำทีละขั้นตอน

## การแนะนำ

### สิ่งที่คุณจะได้เรียนรู้:

## คู่มือการใช้งาน

### โหลดไฟล์ ODS
ฟีเจอร์นี้สาธิตวิธีโหลดไฟล์ OpenDocument Spreadsheet (ODS) โดยใช้ Aspose.Cells สำหรับ .NET

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีแหล่งที่มาและเอาต์พุต
```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```
แทนที่ `YOUR_SOURCE_DIRECTORY` และ `YOUR_OUTPUT_DIRECTORY` กับเส้นทางไดเร็กทอรีของคุณ

#### ขั้นตอนที่ 2: โหลดไฟล์ ODS ลงในวัตถุเวิร์กบุ๊ก
```csharp
Workbook workbook = new Workbook(sourceDir + "/GraphicBackground.ods");
```
ขั้นตอนนี้จะสร้าง `Workbook` วัตถุที่แสดงถึงไฟล์สเปรดชีตทั้งหมด

### การเข้าถึงใบงานและข้อมูลพื้นฐาน
การเข้าถึงเวิร์กชีตเฉพาะและดึงข้อมูลพื้นหลังเป็นเรื่องง่ายด้วย Aspose.Cells

#### ขั้นตอนที่ 3: เข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
เรากำลังเข้าถึงแผ่นงานแรกภายใน `Workbook`-

#### ขั้นตอนที่ 4: รับพื้นหลังหน้า ODS ของเวิร์กชีต
```csharp
OdsPageBackground background = worksheet.PageSetup.ODSPageBackground;
```
การ `OdsPageBackground` วัตถุประกอบด้วยข้อมูลเกี่ยวกับข้อมูลกราฟิกสำหรับหน้า

### บันทึกภาพพื้นหลัง
หากต้องการแยกและบันทึกภาพพื้นหลัง ให้แปลงเป็นบิตแมป จากนั้นบันทึกเป็นไฟล์ JPEG

#### ขั้นตอนที่ 5: แปลงข้อมูลกราฟิกเป็นวัตถุบิตแมป
```csharp
using System.Drawing;
using System.IO;

Bitmap image = new Bitmap(new MemoryStream(background.GraphicData));
```
ขั้นตอนนี้จะสร้าง `Bitmap` จากข้อมูลกราฟิก

#### ขั้นตอนที่ 6: บันทึกบิตแมปเป็นไฟล์ JPEG
```csharp
image.Save(outputDir + "/background.jpg");
```
รูปภาพจะถูกบันทึกไว้ในไดเร็กทอรีเอาท์พุตที่ระบุเป็น "background.jpg"
