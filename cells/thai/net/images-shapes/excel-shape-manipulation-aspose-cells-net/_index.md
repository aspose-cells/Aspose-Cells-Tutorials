## คู่มือการใช้งาน (H2)

ตอนนี้เรามาดูวิธีใช้ Aspose.Cells สำหรับ .NET เพื่อส่งรูปร่างในเวิร์กชีต Excel ของคุณไปที่ด้านหน้าหรือด้านหลังกัน เราจะเน้นที่คุณสมบัติหลักและขั้นตอนการใช้งาน

### การจัดการตำแหน่งลำดับ Z ของรูปทรง

#### ภาพรวม
การทำความเข้าใจและการจัดการตำแหน่งลำดับ Z ช่วยให้คุณควบคุมได้ว่ารูปร่างใดจะปรากฏอยู่ด้านบนในสถานการณ์ที่ทับซ้อนกัน คุณสมบัตินี้มีความสำคัญเมื่อต้องจัดการกับเวิร์กชีตที่ซับซ้อนซึ่งมีวัตถุกราฟิกหลายรายการ

#### การเข้าถึงและการปรับตำแหน่งรูปร่าง (H3)

หากต้องการส่งรูปร่างไปด้านหน้าหรือด้านหลัง ให้ทำตามขั้นตอนเหล่านี้:

```csharp
// โหลดไฟล์ Excel ต้นฉบับ
Workbook workbook = new Workbook("sampleToFrontOrBack.xlsx");

// เข้าถึงแผ่นงานแรก
Worksheet sheet = workbook.Worksheets[0];

// เข้าถึงรูปร่างที่เฉพาะเจาะจงโดยใช้ดัชนี
Shape shape1 = sheet.Shapes[0];
Shape shape4 = sheet.Shapes[3];

// พิมพ์ตำแหน่ง Z-Order ปัจจุบันของรูปร่าง
Console.WriteLine("Z-Order Shape 1: " + shape1.ZOrderPosition);

// ย้ายรูปร่างนี้ไปด้านหน้า
shape1.ToFrontOrBack(2);

// ตรวจสอบตำแหน่ง Z-Order ใหม่
Console.WriteLine("New Z-Order Shape 4: " + shape4.ZOrderPosition);

// ส่งอีกแบบไปด้านหลัง
shape4.ToFrontOrBack(-2);
```

**คำอธิบาย**- 
- `ToFrontOrBack(int value)`:วิธีการนี้จะปรับลำดับ Z ตามพารามิเตอร์ จำนวนเต็มบวกจะเลื่อนรูปร่างไปข้างหน้า ในขณะที่จำนวนเต็มลบจะเลื่อนรูปร่างไปข้างหลัง

#### การบันทึกการเปลี่ยนแปลง (H3)

หลังจากจัดการรูปร่างแล้ว ให้บันทึกการเปลี่ยนแปลงของคุณเพื่อให้แน่ใจว่าจะยังคงอยู่:

```csharp
// บันทึกไฟล์ Excel ที่ถูกแก้ไข
workbook.Save("outputToFrontOrBack.xlsx");
```

- **ให้แน่ใจว่าการจัดทำดัชนีถูกต้อง**:โปรดจำไว้ว่าการจัดทำดัชนีรูปร่างเริ่มต้นที่ 0 ตรวจสอบว่าคุณกำลังเข้าถึงรูปร่างที่ถูกต้อง
