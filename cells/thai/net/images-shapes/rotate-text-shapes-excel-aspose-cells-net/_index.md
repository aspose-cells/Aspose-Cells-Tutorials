# วิธีการหมุนข้อความด้วยรูปร่างใน Excel โดยใช้ Aspose.Cells สำหรับ .NET

## การแนะนำ
เมื่อทำงานกับไฟล์ Excel ด้วยโปรแกรม การหมุนข้อความภายในรูปร่างสามารถปรับปรุงความสวยงามและการจัดแนวข้อมูลของเอกสารของคุณได้อย่างมาก บทช่วยสอนนี้ให้คำแนะนำที่ครอบคลุมเกี่ยวกับวิธีการบรรลุผลดังกล่าวโดยใช้ Aspose.Cells สำหรับ .NET ซึ่งเป็นไลบรารีอันทรงพลังที่ออกแบบมาสำหรับการจัดการเอกสาร Excel

- วิธีการหมุนข้อความโดยมีหรือไม่มีการจัดแนวให้ตรงกับรูปร่างในเวิร์กชีต Excel
- คำแนะนำทีละขั้นตอนในการตั้งค่าและใช้งาน Aspose.Cells สำหรับ .NET
- การประยุกต์ใช้งานจริงของการหมุนข้อความภายในรูปทรง

พร้อมที่จะเพิ่มทักษะการจัดการ Excel ของคุณหรือยัง มาเริ่มกันเลย!

## คู่มือการใช้งาน
ในส่วนนี้เราจะแนะนำคุณเกี่ยวกับกระบวนการหมุนข้อความภายในรูปร่างในเวิร์กชีต Excel

### ขั้นตอนที่ 1: โหลดไฟล์ Excel ของคุณ
เริ่มต้นด้วยการโหลดไฟล์ Excel ตัวอย่างของคุณ:
```csharp
Workbook wb = new Workbook("sampleRotateTextWithShapeInsideWorksheet.xlsx");
```
ขั้นตอนนี้จะเริ่มต้นวัตถุเวิร์กบุ๊กที่แสดงเอกสาร Excel ของคุณ

### ขั้นตอนที่ 2: เข้าถึงและแก้ไขแผ่นงาน
เข้าถึงเวิร์กชีตที่ต้องการซึ่งคุณต้องการจัดการรูปร่างและข้อความ:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell b4 = ws.Cells["B4"];
b4.PutValue("Text is not rotating with shape because RotateTextWithShape is false.");
```

### ขั้นตอนที่ 3: กำหนดค่าคุณสมบัติรูปร่าง
เข้าถึงรูปร่างแรกในเวิร์กชีตของคุณเพื่อปรับเปลี่ยนคุณสมบัติข้อความ:
```csharp
Shape sh = ws.Shapes[0];
Aspose.Cells.Drawing.Texts.ShapeTextAlignment shapeTextAlignment = sh.TextBody.TextAlignment;
shapeTextAlignment.RotateTextWithShape = false; // ตั้งค่านี้เป็นจริงหากคุณต้องการหมุนข้อความตามรูปร่าง
```
การกำหนดค่านี้จะกำหนดว่าข้อความจะหมุนไปตามรูปร่างหรือไม่

### ขั้นตอนที่ 4: บันทึกการเปลี่ยนแปลงของคุณ
หลังจากทำการเปลี่ยนแปลงของคุณแล้ว ให้บันทึกสมุดงาน:
```csharp
wb.Save("outputRotateTextWithShapeInsideWorksheet.xlsx");
Console.WriteLine("Rotation executed successfully.");
```
