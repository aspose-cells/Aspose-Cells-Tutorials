## คู่มือการใช้งาน

เราจะแยกรายละเอียดฟีเจอร์ต่างๆ ของการแปลงแผ่นงาน Excel เป็นรูปภาพโดยใช้ส่วนตรรกะ

### โหลดและเปิดสมุดงาน Excel

**ภาพรวม:**
ขั้นตอนแรกในกระบวนการของเราคือการโหลดเวิร์กบุ๊ก Excel ที่มีอยู่จากไดเร็กทอรีที่ระบุ ซึ่งจะช่วยให้เราเข้าถึงข้อมูลที่ต้องการแปลงเป็นรูปภาพได้

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// โหลดไฟล์ Excel ลงในวัตถุเวิร์กบุ๊ก
Workbook book = new Workbook(SourceDir + "sampleConvertWorksheettoImageFile.xlsx");
```

**คำอธิบาย:**
- `Workbook`แสดงเวิร์กบุ๊กทั้งหมดและให้การเข้าถึงเวิร์กชีตของเวิร์กบุ๊กเหล่านั้น
- ผู้สร้างจะนำเส้นทางของไฟล์ Excel เป็นอาร์กิวเมนต์แล้วโหลดเข้าสู่หน่วยความจำ

### การเข้าถึงเวิร์กชีตจากเวิร์กบุ๊ก

**ภาพรวม:**
หลังจากเปิดเวิร์กบุ๊กแล้ว เราต้องระบุเวิร์กชีตที่ต้องการแปลง ส่วนนี้จะสาธิตการเข้าถึงชีตเฉพาะภายในเวิร์กบุ๊ก

```csharp
// เปิดไฟล์ Excel ลงในวัตถุเวิร์กบุ๊ก
Workbook book = new Workbook(SourceDir + "sampleConvertWorksheettoImageFile.xlsx");

// การเข้าถึงเวิร์กชีตแรกจากเวิร์กบุ๊ก
Worksheet sheet = book.Worksheets[0];
```

**คำอธิบาย:**
- `Worksheets`: คอลเลกชันภายใน `Workbook` ที่เก็บแผ่นงานทั้งหมด
- `sheet.Worksheets[0]`: ดึงข้อมูลเวิร์กชีทแรก (ดัชนี 0) ในเวิร์กบุ๊ก

### การกำหนดค่าตัวเลือกการพิมพ์ภาพ

**ภาพรวม:**
ก่อนการเรนเดอร์ เราจะกำหนดค่าว่าเวิร์กชีตจะถูกแปลงเป็นรูปภาพอย่างไร ซึ่งรวมถึงการตั้งค่ารูปแบบเอาต์พุตและตัวเลือกหน้า

```csharp
// กำหนดค่าตัวเลือกภาพหรือการพิมพ์สำหรับการเรนเดอร์
ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
imgOptions.OnePagePerSheet = true; // เรนเดอร์แผ่นงานทั้งหมดในหนึ่งหน้า
imgOptions.ImageType = Drawing.ImageType.Jpeg; // ตั้งค่าชนิดภาพเอาท์พุตเป็น JPEG
```

**คำอธิบาย:**
- `OnePagePerSheet`:ช่วยให้แน่ใจว่าแผ่นงานทั้งหมดจะถูกแสดงบนภาพเดียว
- `ImageType`: ระบุรูปแบบของภาพเอาต์พุต ในกรณีนี้คือ JPEG

### การเรนเดอร์แผ่นงานเป็นรูปภาพ

**ภาพรวม:**
ตอนนี้เราแปลงแผ่นงานที่ระบุเป็นรูปภาพโดยใช้ตัวเลือกที่ตั้งไว้ก่อนหน้านี้

```csharp
// สร้างวัตถุ SheetRender เพื่อเรนเดอร์เวิร์กชีตเป็นรูปภาพ
SheetRender sr = new SheetRender(sheet, imgOptions);
Bitmap bitmap = sr.ToImage(0); // เรนเดอร์หน้าแรกของแผ่นงานเป็นรูปภาพ
```

**คำอธิบาย:**
- `SheetRender`: จัดการการดำเนินการเรนเดอร์สำหรับเวิร์กชีต
- `ToImage(int pageIndex)`:แปลงหน้าเวิร์กชีตที่ระบุเป็นรูปภาพ

### การบันทึกภาพที่เรนเดอร์

**ภาพรวม:**
สุดท้ายให้บันทึกภาพที่สร้างขึ้นไปยังไดเร็กทอรีเอาต์พุตที่คุณต้องการ

```csharp
// บันทึกภาพที่แสดงผลไปยังไดเร็กทอรีเอาท์พุต
bitmap.Save(outputDir + "outputConvertWorksheettoImageFile.jpg");
```

**คำอธิบาย:**
- `Save(string path)`: เขียนไฟล์ภาพลงดิสก์ที่ตำแหน่งที่ระบุ

2. **การนำเสนอข้อมูล**:สร้างสื่อช่วยสื่อภาพสำหรับการนำเสนอโดยการแปลงชุดข้อมูลที่ซับซ้อน
3. **เอกสารประกอบ**:รวมตารางที่จัดรูปแบบเป็นรูปภาพคงที่ภายในเอกสารทางเทคนิค
4. **เนื้อหาเว็บไซต์**:แสดงข้อมูลทางการเงินหรือข้อมูลเชิงวิเคราะห์บนเว็บไซต์โดยไม่ต้องใช้ Excel
5. **การจัดเก็บถาวร**:รักษาสถานะที่แน่นอนของเวิร์กชีตในจุดหนึ่งในเวลา
