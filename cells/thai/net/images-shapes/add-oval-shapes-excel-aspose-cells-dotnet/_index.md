## Implementation Guide

### Adding an Oval Shape

This feature guides you through adding a basic oval shape to an Excel worksheet.

#### Overview
Adding ovals can enhance the visual appeal of your data presentation. In this section, we'll add and configure an oval in the first worksheet of our Excel file using Aspose.Cells.

#### Steps:

##### Step 1: Define Directory for Output

First, define where you want to save your output files:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
string dataDir = Path.Combine(outputDir, "OvalShapeExample");

if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
```

##### ขั้นตอนที่ 2: สร้างตัวอย่างสมุดงาน

สร้างอินสแตนซ์ของ `Workbook` ชั้นเรียนเพื่อเริ่มต้นทำงานกับไฟล์ Excel:

```csharp
Workbook excelbook = new Workbook();
```

##### ขั้นตอนที่ 3: เพิ่มรูปทรงวงรี

ใช้ `AddOval` วิธีการวางรูปวงรีในแผ่นงาน:

```csharp
// เพิ่มวงรีตามพิกัดและขนาดที่กำหนด
Oval oval1 = excelbook.Worksheets[0].Shapes.AddOval(2, 0, 2, 0, 130, 160);
```

##### ขั้นตอนที่ 4: กำหนดค่าตำแหน่ง

ตั้งค่าประเภทการวางเป็น `FreeFloating` เพื่อการควบคุมตำแหน่งที่มากขึ้น:

```csharp
oval1.Placement = PlacementType.FreeFloating;
```

##### ขั้นตอนที่ 5: ตั้งค่าคุณสมบัติเส้น

ปรับแต่งลักษณะของโครงร่างวงรีโดยการตั้งค่าน้ำหนักเส้นและสไตล์เส้นประ:

```csharp
// ตั้งค่าน้ำหนักเส้นและรูปแบบเส้นประ
oval1.Line.Weight = 1;
oval1.Line.DashStyle = MsoLineDashStyle.Solid;
```

##### ขั้นตอนที่ 6: บันทึกสมุดงาน

สุดท้าย ให้บันทึกสมุดงานของคุณไปยังไฟล์ในไดเร็กทอรีที่ระบุ:

```csharp
excelbook.Save(Path.Combine(dataDir, "OvalShapeExample.xls"));
```

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีทั้งหมดได้รับการตั้งค่าอย่างถูกต้องเพื่อป้องกันข้อผิดพลาดไม่พบไฟล์
- ตรวจสอบว่า Aspose.Cells ได้รับอนุญาตการใช้งานอย่างถูกต้อง หากคุณกำลังใช้ฟีเจอร์ต่างๆ เกินกว่าข้อจำกัดการทดลองใช้

### การเพิ่มรูปวงรีอีกอัน (วงกลม)

ตอนนี้เรามาเพิ่มรูปวงรีอีกรูปหนึ่ง ซึ่งกำหนดค่าเป็นวงกลม โดยมีคุณสมบัติที่แตกต่างกัน

#### ภาพรวม
การเพิ่มรูปทรงหลายรูปทรงสามารถช่วยสร้างภาพที่ซับซ้อนมากขึ้นได้ ในที่นี้ เราจะสาธิตการเพิ่มรูปวงรีวงกลมลงในเวิร์กชีตของคุณ

#### ขั้นตอน:

##### ขั้นตอนที่ 1: ตรวจสอบว่ามีไดเร็กทอรีอยู่

ขั้นตอนนี้จะคล้ายกับส่วนก่อนหน้า ตรวจสอบให้แน่ใจว่าไดเร็กทอรีของคุณถูกตั้งค่าอย่างถูกต้อง

```csharp
if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
```

##### ขั้นตอนที่ 2: สร้างเวิร์กบุ๊ก

สร้างใหม่ `Workbook` ตัวอย่างสำหรับการบวกรูปร่างนี้:

```csharp
Workbook excelbook = new Workbook();
```

##### ขั้นตอนที่ 3: เพิ่มรูปร่างวงกลม

เพิ่มวงรีอีกอันที่มีขนาดเพื่อให้ดูเป็นวงกลม:

```csharp
// เพิ่มรูปวงกลมที่มีพิกัดและขนาดต่างกัน
Oval oval2 = excelbook.Worksheets[0].Shapes.AddOval(9, 0, 2, 15, 130, 130);
```

##### ขั้นตอนที่ 4: กำหนดค่าตำแหน่ง

ตั้งค่าประเภทการวางสำหรับรูปร่างใหม่:

```csharp
oval2.Placement = PlacementType.FreeFloating;
```

##### ขั้นตอนที่ 5: ตั้งค่าคุณสมบัติเส้น

กำหนดน้ำหนักเส้นและสไตล์เส้นประเพื่อการปรับแต่ง:

```csharp
// ปรับแต่งคุณสมบัติของเส้น
oval2.Line.Weight = 1;
oval2.Line.DashStyle = MsoLineDashStyle.Solid;
```

##### ขั้นตอนที่ 6: บันทึกสมุดงานด้วยรูปร่างใหม่

บันทึกสมุดงานอีกครั้ง คราวนี้รวมทั้งรูปร่างทั้งสอง:

```csharp
excelbook.Save(Path.Combine(dataDir, "OvalShapeExampleWithCircle.xls"));
```

3. **การสร้างเทมเพลต**:สร้างเทมเพลตที่สามารถนำมาใช้ซ้ำได้สำหรับรายงานที่ต้องการองค์ประกอบภาพที่สอดคล้องกัน
