# วิธีการอัปเดตข้อความ SmartArt ในเวิร์กบุ๊ก Excel แบบอัตโนมัติโดยใช้ Aspose.Cells .NET

## การแนะนำ
การอัปเดตกราฟิก SmartArt ด้วยตนเองใน Excel อาจเป็นเรื่องน่าเบื่อ โดยเฉพาะเมื่อต้องจัดการกับชุดข้อมูลขนาดใหญ่หรือเอกสารหลายฉบับ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ Aspose.Cells สำหรับ .NET เพื่อทำให้กระบวนการนี้เป็นอัตโนมัติ ช่วยประหยัดเวลาและลดข้อผิดพลาด

**สิ่งที่คุณจะได้เรียนรู้:**
- โหลดเวิร์กบุ๊ก Excel และทำซ้ำผ่านเวิร์กชีต
- ระบุและแก้ไขรูปร่าง SmartArt ภายในแผ่นงาน Excel
- บันทึกสมุดงานที่อัปเดตโดยใช้การเปลี่ยนแปลงของคุณ

มาเริ่มตั้งค่าสภาพแวดล้อมของคุณเพื่อเริ่มต้นกันเลย

## คู่มือการใช้งาน
หัวข้อนี้จะกล่าวถึงฟังก์ชันหลักสามประการ ได้แก่ การโหลดและวนซ้ำผ่านเวิร์กชีต การจัดการรูปร่าง SmartArt และการบันทึกเวิร์กบุ๊กที่อัปเดต

### คุณลักษณะที่ 1: การโหลดเวิร์กบุ๊กและการวนซ้ำผ่านเวิร์กชีต
**ภาพรวม:**
เรียนรู้วิธีโหลดไฟล์ Excel และเข้าถึงเวิร์กชีตแต่ละแผ่นเพื่อจัดการเนื้อหา

#### การดำเนินการทีละขั้นตอน:
##### โหลดสมุดงาน
เริ่มต้นด้วยการสร้าง `Workbook` วัตถุที่มีเส้นทางไฟล์ต้นฉบับของคุณ:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook wb = new Workbook(SourceDir + "SmartArt.xlsx");
```

##### ทำซ้ำผ่านเวิร์กชีตและรูปร่าง
ใช้ลูปซ้อนกันเพื่อเข้าถึงเวิร์กชีตแต่ละแผ่นและรูปร่างของแผ่นงานนั้น และตั้งค่าข้อความทางเลือกสำหรับการปรับแต่ง:

```csharp
foreach (Worksheet worksheet in wb.Worksheets)
{
    foreach (Shape shape in worksheet.Shapes)
    {
        shape.AlternativeText = "ReplacedAlternativeText";
        
        if (shape.IsSmartArt)
        {
            // จัดการตรรกะเฉพาะ SmartArt ที่นี่
        }
    }
}
```

### คุณสมบัติ 2: การจัดการรูปทรง SmartArt
**ภาพรวม:**
เจาะลึกการประมวลผลและการอัปเดตข้อความภายในรูปร่าง SmartArt ด้วยโปรแกรม

#### การดำเนินการทีละขั้นตอน:
##### วนซ้ำผ่านรูปทรง SmartArt
ภายในลูปที่สร้างไว้ก่อนหน้า ให้เน้นที่รูปร่าง SmartArt เพื่อปรับเปลี่ยนเนื้อหา:

```csharp
foreach (Worksheet worksheet in wb.Worksheets)
{
    foreach (Shape shape in worksheet.Shapes)
    {
        if (shape.IsSmartArt)
        {
            foreach (Shape smartart in shape.GetResultOfSmartArt().GetGroupedShapes())
            {
                smartart.Text = "ReplacedText"; // อัพเดทข้อความ
            }
        }
    }
}
```

### คุณสมบัติที่ 3: การบันทึกสมุดงานด้วยข้อความ SmartArt ที่อัปเดต
**ภาพรวม:**
ตรวจสอบให้แน่ใจว่าการเปลี่ยนแปลงของคุณได้รับการบันทึกโดยการกำหนดค่าและบันทึกเวิร์กบุ๊กอย่างถูกต้อง

#### การดำเนินการทีละขั้นตอน:
##### บันทึกสมุดงาน
ใช้ `OoxmlSaveOptions` เพื่อระบุว่าควรพิจารณาการอัปเดต SmartArt:
```csharp
Aspose.Cells.OoxmlSaveOptions options = new Aspose.Cells.OoxmlSaveOptions();
options.UpdateSmartArt = true;
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
wb.Save(OutputDir + "outputSmartArt.xlsx", options);
```

2. **การอัพเดตเอกสารเป็นกลุ่ม:** แก้ไขไฟล์ Excel หลายไฟล์โดยให้แบรนด์มีความสอดคล้องหรือมีการเปลี่ยนแปลงข้อมูล
