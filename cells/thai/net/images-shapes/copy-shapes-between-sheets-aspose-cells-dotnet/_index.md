## คู่มือการใช้งาน

ในส่วนนี้เราจะแนะนำการคัดลอกรูปร่างระหว่างเวิร์กชีตทีละขั้นตอน

### ขั้นตอนที่ 1: เปิดเวิร์กบุ๊กที่มีอยู่
เริ่มต้นด้วยการสร้างวัตถุเวิร์กบุ๊กจากไฟล์ Excel ต้นฉบับของคุณ ที่นี่คุณจะสามารถเข้าถึงรูปร่างที่จะคัดลอกได้
```csharp
// สร้างวัตถุเวิร์กบุ๊กและเปิดไฟล์เทมเพลต
Workbook workbook = new Workbook(sourceDir + "sampleCopyControls.xlsx");
```

### ขั้นตอนที่ 2: เข้าถึงรูปร่างในเวิร์กชีตต้นฉบับ
เข้าถึงคอลเลกชันรูปร่างจากเวิร์กชีตต้นทาง ที่นี่ เรากำหนดเป้าหมายที่เวิร์กชีต "Sheet1" เพื่อดึงรูปร่างของมัน
```csharp
// รับรูปร่างจากเวิร์กชีต "การควบคุม"
Aspose.Cells.Drawing.ShapeCollection shapes = workbook.Worksheets["Sheet1"].Shapes;
```

### ขั้นตอนที่ 3: คัดลอกรูปร่างเฉพาะ
ตอนนี้ให้คัดลอกรูปร่างเฉพาะ (เช่น กล่องข้อความหรือวงรี) ไปยังเวิร์กชีตอื่น เราจะเพิ่มสำเนาเหล่านี้ในตำแหน่งที่ระบุ
```csharp
// คัดลอกกล่องข้อความไปยังแผ่นงานผลลัพธ์
workbook.Worksheets["Result"].Shapes.AddCopy(shapes[0], 5, 0, 2, 0);

// คัดลอกรูปทรงวงรีไปยังแผ่นงานผลลัพธ์
workbook.Worksheets["Result"].Shapes.AddCopy(shapes[1], 10, 0, 2, 0);
```
- **พารามิเตอร์**: เดอะ `AddCopy` วิธีนี้ใช้พารามิเตอร์สำหรับตำแหน่งและขนาด ปรับเปลี่ยนได้ตามความต้องการของคุณ

### ขั้นตอนที่ 4: บันทึกสมุดงาน
สุดท้ายให้บันทึกสมุดงานเพื่อรักษาการเปลี่ยนแปลงของคุณ
```csharp
// บันทึกใบงาน
workbook.Save(outputDir + "outputCopyControls.xlsx");
```
