## คู่มือการใช้งาน

ในส่วนนี้เราจะแนะนำวิธีการแปลงรูปร่าง SmartArt เป็นรูปร่างกลุ่มโดยใช้ `Aspose.Cells` ห้องสมุด.

### การระบุและการแปลงรูปทรง

#### ภาพรวม
การแปลงวัตถุ SmartArt เป็นรูปร่างกลุ่มทำให้การจัดการและปรับแต่งภายในไฟล์ Excel ของคุณง่ายขึ้น กระบวนการนี้เกี่ยวข้องกับการระบุวัตถุ SmartArt จากนั้นใช้เมธอด Aspose.Cells เพื่อดำเนินการแปลง

**ขั้นตอนที่ 1: โหลดสมุดงานของคุณ**
```csharp
// ไดเรกทอรีแหล่งที่มา
string sourceDir = RunExamples.Get_SourceDirectory();

// โหลดไฟล์ Excel ตัวอย่างรูปทรงสมาร์ทอาร์ต
Workbook wb = new Workbook(sourceDir + "sampleSmartArtShape_GetResultOfSmartArt.xlsx");
```

#### การเข้าถึงรูปทรง
**ขั้นตอนที่ 2: เข้าถึงเวิร์กชีตและรูปร่าง**
```csharp
// เข้าถึงแผ่นงานแรก
Worksheet ws = wb.Worksheets[0];

// เข้าถึงรูปร่างแรกในเวิร์กชีต
Shape sh = ws.Shapes[0];
```

#### การตรวจสอบ SmartArt
**ขั้นตอนที่ 3: ระบุว่ารูปร่างเป็น SmartArt หรือไม่**
ก่อนที่จะแปลง ให้ตรวจสอบว่ารูปร่างของคุณเป็นวัตถุ SmartArt หรือไม่
```csharp
// พิจารณาว่ารูปร่างเป็นศิลปะอัจฉริยะหรือไม่
Console.WriteLine("Is Smart Art Shape: " + sh.IsSmartArt);
```

#### การแปลงเป็นรูปร่างกลุ่ม
**ขั้นตอนที่ 4: แปลง SmartArt เป็นรูปร่างกลุ่ม**
```csharp
// ตรวจสอบว่ารูปร่างเป็นรูปร่างกลุ่มก่อนการแปลงหรือไม่
Console.WriteLine("Is Group Shape Before Conversion: " + sh.IsGroup);

// ทำการแปลงและตรวจสอบอีกครั้ง
Console.WriteLine("Is Group Shape After Conversion: " + sh.GetResultOfSmartArt().IsGroup);
```
