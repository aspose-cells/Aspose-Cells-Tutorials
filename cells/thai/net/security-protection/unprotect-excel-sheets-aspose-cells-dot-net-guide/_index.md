## คู่มือการใช้งาน

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อเข้าถึงและยกเลิกการป้องกันเวิร์กชีต:

### การเข้าถึงและการเลิกป้องกันแผ่นงาน

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก

เริ่มต้นด้วยการสร้าง `Workbook` วัตถุจากไฟล์ที่มีอยู่ของคุณ:

```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// โหลดเวิร์กบุ๊กจากดิสก์
Workbook workbook = new Workbook(sourceDir + "/book1.xls");
```

**คำอธิบาย:** บรรทัดนี้จะเริ่มต้นเวิร์กบุ๊กด้วยไฟล์ Excel ที่คุณระบุ

#### ขั้นตอนที่ 2: เข้าถึงแผ่นงาน

ดึงข้อมูลเวิร์กชีตที่คุณต้องการยกเลิกการป้องกัน:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

**คำอธิบาย:** การ `Worksheets[0]` วิธีการเข้าถึงแผ่นงานแรกในเวิร์กบุ๊ก

#### ขั้นตอนที่ 3: ยกเลิกการป้องกันแผ่นงาน

การนำการป้องกันออกจากแผ่นงาน:

```csharp
// ยกเลิกการป้องกันแผ่นงานโดยไม่ระบุรหัสผ่าน
worksheet.Unprotect();
```

**คำอธิบาย:** การดำเนินการนี้จะลบการป้องกันทั้งหมดที่มีอยู่ และอนุญาตให้เข้าถึงเวิร์กชีตได้อย่างสมบูรณ์

#### ขั้นตอนที่ 4: บันทึกสมุดงาน

บันทึกการเปลี่ยนแปลงของคุณกลับไปยังดิสก์:

```csharp
workbook.Save(outputDir + "/output.xls", SaveFormat.Excel97To2003);
```

**คำอธิบาย:** การ `save` วิธีการเขียนสมุดงานที่อัปเดตในรูปแบบ Excel 97-2003

### การโหลดและการบันทึกสมุดงาน

โหลดเวิร์กบุ๊ก แก้ไข และบันทึกการเปลี่ยนแปลงของคุณ:

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊กที่มีอยู่

```csharp
// โหลดสมุดงานใหม่อีกครั้งเพื่อจุดประสงค์ในการสาธิต
tWorkbook = new Workbook(sourceDir + "/book1.xls");
```

**คำอธิบาย:** วิธีนี้จะช่วยให้แน่ใจว่าคุณกำลังทำงานกับไฟล์เวอร์ชันล่าสุด

#### ขั้นตอนที่ 2: ยกเลิกการป้องกันและบันทึก

ทำซ้ำการยกเลิกการป้องกันและบันทึกตามที่อธิบายไว้ก่อนหน้านี้เพื่อใช้การเปลี่ยนแปลง

## การประยุกต์ใช้งานจริง

การปลดล็อคเวิร์กชีต Excel มีประโยชน์ในสถานการณ์ต่างๆ ดังนี้:
1. **การตรวจสอบข้อมูล:** เข้าถึงและตรวจสอบข้อมูลในแผ่นงานที่ได้รับการป้องกันได้อย่างรวดเร็ว
2. **การรายงานอัตโนมัติ:** สร้างรายงานอัตโนมัติจากชุดข้อมูลที่ถูกล็อค
3. **การแก้ไขแบบร่วมมือกัน:** แบ่งปันเวอร์ชันที่แก้ไขได้กับสมาชิกในทีมสำหรับโครงการร่วมกัน
