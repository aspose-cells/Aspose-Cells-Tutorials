## คู่มือการใช้งาน

### การยกเลิกการป้องกันเวิร์กชีตโดยไม่ต้องใช้รหัสผ่าน

**ภาพรวม:** ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์ Excel เข้าถึงเวิร์กชีตที่ต้องการ และยกเลิกการป้องกันแม้ว่าจะไม่ทราบรหัสผ่านก็ตาม

#### การดำเนินการทีละขั้นตอน:

**1. โหลดไฟล์ Excel**

ขั้นแรก โหลดเวิร์กบุ๊กของคุณจากไดเร็กทอรีแหล่งที่มา
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/book1.xls");
```
*คำอธิบาย:* บรรทัดนี้จะเริ่มต้น `Workbook` วัตถุโดยการโหลดไฟล์ Excel ที่มีอยู่

**2. การเข้าถึงและยกเลิกการป้องกันแผ่นงาน**

เข้าถึงเวิร์กชีตแรกและยกเลิกการป้องกัน
```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Unprotect();
```
*คำอธิบาย:* โดยการเข้าถึง `Worksheets[0]`คุณดึงแผ่นงานแรกออกมา `Unprotect()` วิธีการนี้จะลบการป้องกันทั้งหมดออก ทำให้สามารถปรับเปลี่ยนได้

**3. บันทึกสมุดงาน**

สุดท้าย ให้บันทึกสมุดงานที่ไม่ได้รับการป้องกันไปยังไดเร็กทอรีที่คุณต้องการ
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/output.out.xls", SaveFormat.Excel97To2003);
```
*คำอธิบาย:* บรรทัดนี้จะบันทึกเวิร์กบุ๊กในรูปแบบ Excel 97-2003 คุณสามารถเลือกรูปแบบอื่นที่ Aspose.Cells รองรับได้

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกต้อง
- ตรวจสอบสิทธิ์การอ่าน/เขียนบนไดเร็กทอรี

### การบันทึกไฟล์ Excel ในรูปแบบเฉพาะ

**ภาพรวม:** เรียนรู้วิธีการบันทึกไฟล์ Excel โดยใช้รูปแบบเฉพาะ ซึ่งมีประโยชน์อย่างยิ่งเมื่อต้องจัดการกับระบบเดิมหรือปัญหาความเข้ากันได้

#### การดำเนินการทีละขั้นตอน:

**1. โหลดเวิร์กบุ๊ก**

คล้ายกับคุณสมบัติการยกเลิกการป้องกัน:
```csharp
Workbook workbook = new Workbook(SourceDir + "/book1.xls");
```

**2. บันทึกในรูปแบบที่ต้องการ**

ระบุรูปแบบระหว่างการดำเนินการบันทึก
```csharp
workbook.Save(outputDir + "/output.out.xls", SaveFormat.Excel97To2003);
```
*คำอธิบาย:* `SaveFormat` ระบุประเภทไฟล์เอาต์พุตเพื่อให้แน่ใจว่าเข้ากันได้กับ Excel เวอร์ชันเก่ากว่า

## การประยุกต์ใช้งานจริง

ต่อไปนี้เป็นกรณีการใช้งานจริงบางส่วนสำหรับการยกเลิกการป้องกันและบันทึกไฟล์ Excel:

1. **การย้ายข้อมูล:** ยกเลิกการป้องกันแผ่นงานเพื่อย้ายข้อมูลระหว่างระบบที่แตกต่างกันโดยไม่มีข้อจำกัดด้านรหัสผ่าน
2. **การจัดการเทมเพลต:** ปรับเปลี่ยนไฟล์เทมเพลตที่ได้รับการป้องกันได้อย่างง่ายดายก่อนที่จะแจกจ่ายเป็นรูปแบบมาตรฐาน
3. **การสร้างรายงาน:** ทำให้การสร้างรายงานเป็นแบบอัตโนมัติโดยลบการป้องกันจากแหล่งข้อมูล
4. **โครงการความร่วมมือ:** แบ่งปันสมุดงานระหว่างทีม เพื่อให้แน่ใจว่าไม่มีการจำกัดรหัสผ่านที่จะขัดขวางการทำงานร่วมกัน
