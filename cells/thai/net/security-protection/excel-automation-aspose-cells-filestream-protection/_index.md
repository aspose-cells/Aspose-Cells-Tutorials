## คู่มือการใช้งาน

### คุณสมบัติ 1: การสร้างสตรีมไฟล์และการเริ่มต้นเวิร์กบุ๊ก

ฟีเจอร์นี้ช่วยให้คุณสามารถสร้างสตรีมไฟล์จากไฟล์ Excel ได้ ทำให้จัดการชุดข้อมูลขนาดใหญ่ได้อย่างมีประสิทธิภาพและง่ายยิ่งขึ้น

#### ขั้นตอนที่ 1: สร้าง FileStream
```csharp
using System.IO;
using Aspose.Cells;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";

// การสร้าง FileStream สำหรับไฟล์ Excel ที่ระบุ
FileStream fstream = new FileStream(SourceDir + "book1.xls", FileMode.Open);
```
*ทำไม* การใช้ FileStream ช่วยให้คุณจัดการไฟล์ได้อย่างมีประสิทธิภาพ โดยเฉพาะอย่างยิ่งกับชุดข้อมูลขนาดใหญ่

#### ขั้นตอนที่ 2: เริ่มต้นวัตถุสมุดงาน
```csharp
// การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กโดยใช้ FileStream
Workbook excel = new Workbook(fstream);

// การปิด FileStream เพื่อปลดปล่อยทรัพยากร
fstream.Close();
```
*คำอธิบาย*: เดอะ `Workbook` คลาสจะถูกเริ่มต้นด้วยสตรีมไฟล์ ซึ่งทำให้คุณสามารถจัดการไฟล์ Excel ผ่านโปรแกรมได้

### คุณลักษณะที่ 2: การตั้งค่าการป้องกันแผ่นงาน

การปกป้องเวิร์กชีตของคุณจะช่วยให้ข้อมูลมีความสมบูรณ์และป้องกันการเปลี่ยนแปลงที่ไม่ได้รับอนุญาต

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊กและเข้าถึงเวิร์กชีต
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";

// การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊กโดยการเปิดไฟล์ที่ระบุ
Workbook excel = new Workbook(SourceDir + "book1.xls");

// การเข้าถึงเวิร์กชีตแรกในเวิร์กบุ๊ก
Worksheet worksheet = excel.Worksheets[0];
```
*มันทำอะไรได้บ้าง?* ขั้นตอนนี้จะเตรียมเวิร์กชีตของคุณเพื่อใช้การตั้งค่าการป้องกัน

#### ขั้นตอนที่ 2: ใช้การตั้งค่าการป้องกัน
```csharp
// การใช้การตั้งค่าการป้องกันต่างๆ เพื่อจำกัดการกระทำของผู้ใช้
worksheet.Protection.AllowDeletingColumn = false;
worksheet.Protection.AllowDeletingRow = false;
worksheet.Protection.AllowEditingContent = false;

// อนุญาตให้ดำเนินการเฉพาะในขณะที่ปกป้องเวิร์กชีต
data cell formatting and hyperlink insertion are permitted.
worksheet.Protection.AllowFormattingCell = true;
worksheet.Protection.AllowInsertingHyperlink = true;

// บันทึกสมุดงานด้วยการตั้งค่าการป้องกัน
excel.Save(@"YOUR_OUTPUT_DIRECTORY\output.xls", SaveFormat.Excel97To2003);
```
*คำอธิบาย*การตั้งค่าเหล่านี้กำหนดสิ่งที่ผู้ใช้ทำได้และไม่สามารถทำได้ เพื่อสร้างสมดุลระหว่างความปลอดภัยและการใช้งาน
