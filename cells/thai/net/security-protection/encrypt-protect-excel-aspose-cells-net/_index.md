## คู่มือการใช้งาน
เราจะแบ่งกระบวนการใช้งานตามฟีเจอร์ต่างๆ: การเข้ารหัสไฟล์ Excel และการเพิ่มรหัสผ่านในการปรับเปลี่ยน

### การเข้ารหัสไฟล์ Excel ด้วย Aspose.Cells สำหรับ .NET
**ภาพรวม:**
เข้ารหัสไฟล์ Excel ของคุณเพื่อปกป้องข้อมูลสำคัญจากการเข้าถึงโดยไม่ได้รับอนุญาต หัวข้อนี้สาธิตวิธีการใช้ประเภทการเข้ารหัสต่างๆ โดยใช้ Aspose.Cells

#### ขั้นตอนที่ 1: ตั้งค่าโครงการของคุณและโหลดเวิร์กบุ๊ก
```csharp
// ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่าเส้นทางไดเร็กทอรีเหล่านี้อย่างถูกต้องในสภาพแวดล้อมของคุณ
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

Workbook workbook = new Workbook(SourceDir + "/Book1.xls");
```

#### ขั้นตอนที่ 2: ระบุตัวเลือกการเข้ารหัส
เลือกประเภทการเข้ารหัสระหว่าง XOR และ Strong Cryptographic Provider:
```csharp
// ใช้การเข้ารหัส XOR ด้วยความยาวคีย์ 40
workbook.SetEncryptionOptions(EncryptionType.XOR, 40);

// ทางเลือกอื่น คือ ใช้การเข้ารหัส RC4 ที่แข็งแกร่งด้วยความยาวคีย์ 128 บิต
workbook.SetEncryptionOptions(EncryptionType.StrongCryptographicProvider, 128);
```

#### ขั้นตอนที่ 3: ตั้งรหัสผ่านไฟล์
```csharp
// ปกป้องไฟล์ Excel ของคุณด้วยการตั้งรหัสผ่าน
workbook.Settings.Password = "1234";
```

#### ขั้นตอนที่ 4: บันทึกสมุดงานที่เข้ารหัส
```csharp
// บันทึกสมุดงานที่เข้ารหัสของคุณไปยังไดเร็กทอรีเอาท์พุต
workbook.Save(OutputDir + "/encryptedBook1.out.xls");
```

### การป้องกันด้วยรหัสผ่านสำหรับการแก้ไขด้วย Aspose.Cells
**ภาพรวม:**
ป้องกันการแก้ไขโดยไม่ได้รับอนุญาตโดยการตั้งรหัสผ่านที่จำเป็นสำหรับการแก้ไข

#### ขั้นตอนที่ 1: โหลดสมุดงานที่มีอยู่
```csharp
Workbook workbook = new Workbook(SourceDir + "/Book1.xls");
```

#### ขั้นตอนที่ 2: ตั้งรหัสผ่านการป้องกันการเขียน
```csharp
// กำหนดรหัสผ่านที่จำเป็นในการปรับเปลี่ยนไฟล์ Excel
workbook.Settings.WriteProtection.Password = "1234";
```

#### ขั้นตอนที่ 3: บันทึกสมุดงานที่ได้รับการป้องกัน
```csharp
// บันทึกสมุดงานของคุณโดยเปิดการใช้งานการป้องกันการแก้ไข
workbook.Save(OutputDir + "/SpecifyPasswordToModifyOption.out.xls");
```

- **ปัญหาทั่วไป:** หากคุณพบข้อผิดพลาดเกี่ยวกับไดเร็กทอรีหรือไฟล์ที่หายไป ให้ตรวจสอบอีกครั้ง `SourceDir` และ `OutputDir` เส้นทาง
