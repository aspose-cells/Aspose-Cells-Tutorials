# เรียนรู้ตารางแบบสอบถาม Excel ด้วย Aspose.Cells .NET

## การแนะนำ
ในโลกปัจจุบันที่ข้อมูลเป็นปัจจัยสำคัญในการจัดการและดึงข้อมูลจากไฟล์ Excel อย่างมีประสิทธิภาพ ไม่ว่าคุณจะเป็นนักพัฒนาที่มีประสบการณ์หรือเพิ่งเริ่มต้น การเรียนรู้วิธีจัดการเวิร์กบุ๊ก Excel ด้วยโปรแกรมจะช่วยลดขั้นตอนการทำงานของคุณได้อย่างมาก คู่มือนี้จะช่วยให้คุณเชี่ยวชาญศิลปะในการอ่าน แก้ไข และบันทึกตารางแบบสอบถาม Excel โดยใช้ Aspose.Cells สำหรับ .NET

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีการอ่านเวิร์กบุ๊ก Excel และเข้าถึงเวิร์กชีตของมัน
- การเข้าถึงตารางแบบสอบถามเฉพาะภายในเวิร์กชีต
- การอ่านและปรับเปลี่ยนคุณสมบัติของตารางแบบสอบถาม เช่น `AdjustColumnWidth` และ `PreserveFormatting`
- การบันทึกการเปลี่ยนแปลงที่ทำกับเวิร์กบุ๊ก Excel

พร้อมที่จะเริ่มหรือยัง เริ่มต้นด้วยการตั้งค่าเครื่องมือและสภาพแวดล้อมที่จำเป็น

## คู่มือการใช้งาน

### การอ่านสมุดงาน Excel
**ภาพรวม:** ฟีเจอร์นี้สาธิตวิธีการโหลดไฟล์ Excel และเข้าถึงเวิร์กชีตของไฟล์นั้น

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(sourceDir + "sampleReadingAndWritingQueryTable.xlsx");
```

#### ขั้นตอนที่ 2: เข้าถึงแผ่นงาน
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

### การเข้าถึงตารางแบบสอบถามในเวิร์กชีต
**ภาพรวม:** เรียนรู้วิธีการเข้าถึงตารางแบบสอบถามเฉพาะภายในเวิร์กชีต Excel

#### ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊กและเวิร์กชีต
```csharp
Workbook workbook = new Workbook(sourceDir + "sampleReadingAndWritingQueryTable.xlsx");
Worksheet worksheet = workbook.Worksheets[0];
```

#### ขั้นตอนที่ 2: เข้าถึงตารางแบบสอบถาม
```csharp
QueryTable qt = worksheet.QueryTables[0];
```

### การอ่านคุณสมบัติของตารางแบบสอบถาม
**ภาพรวม:** ฟีเจอร์นี้แสดงคุณสมบัติการอ่าน เช่น `AdjustColumnWidth` และ `PreserveFormatting`-

```csharp
bool adjustColumnWidth = qt.AdjustColumnWidth;
bool preserveFormatting = qt.PreserveFormatting;

// คำอธิบาย: AdjustColumnWidth จะปรับขนาดคอลัมน์โดยอัตโนมัติ และ PreserveFormatting จะรักษารูปแบบดั้งเดิมไว้
```

### การแก้ไขคุณสมบัติตารางแบบสอบถาม
**ภาพรวม:** เรียนรู้วิธีการปรับเปลี่ยนคุณสมบัติของตารางแบบสอบถาม

#### ขั้นตอนที่ 1: ตั้งค่าการรักษาการจัดรูปแบบ
```csharp
qt.PreserveFormatting = true;
```

### การบันทึกสมุดงาน Excel
**ภาพรวม:** คุณลักษณะนี้จะแสดงวิธีบันทึกการเปลี่ยนแปลงที่ทำกับเวิร์กบุ๊ก Excel

#### ขั้นตอนที่ 1: บันทึกสมุดงาน
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outputReadingAndWritingQueryTable.xlsx");
```
