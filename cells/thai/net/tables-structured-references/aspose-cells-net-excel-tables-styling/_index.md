# วิธีการสร้างและกำหนดรูปแบบตาราง Excel โดยใช้ Aspose.Cells สำหรับ .NET

## การแนะนำ
ในโลกปัจจุบันที่ข้อมูลเป็นปัจจัยสำคัญในการจัดการชุดข้อมูลจำนวนมากอย่างมีประสิทธิภาพ ถือเป็นเรื่องสำคัญสำหรับการวิเคราะห์และการรายงาน บทช่วยสอนนี้ให้คำแนะนำที่ครอบคลุมเกี่ยวกับการสร้างและกำหนดรูปแบบตาราง Excel โดยใช้ Aspose.Cells สำหรับ .NET ซึ่งเป็นเครื่องมือที่ขาดไม่ได้สำหรับนักพัฒนาที่ต้องการผสานรวมฟังก์ชันการทำงานของสเปรดชีตในแอปพลิเคชันของตนอย่างราบรื่น

## คู่มือการใช้งาน

### คุณลักษณะที่ 1: สร้างและกำหนดค่าสมุดงาน
คุณลักษณะนี้เกี่ยวข้องกับการสร้างเวิร์กบุ๊ก Excel การเพิ่มข้อมูลลงไป และการบันทึกไฟล์

#### ภาพรวม
เราจะเริ่มต้นด้วยการสร้างเวิร์กบุ๊กใหม่และเติมข้อมูลส่วนหัวและข้อมูลพนักงาน

#### การดำเนินการแบบทีละขั้นตอน

**ขั้นตอนที่ 1: เริ่มต้นเวิร์กบุ๊ก**
สร้างอินสแตนซ์ใหม่ของ `Workbook`-

```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY";

// สร้างอินสแตนซ์เวิร์กบุ๊กใหม่
Workbook workbook = new Workbook();
```

**ขั้นตอนที่ 2: เข้าถึงและเติมข้อมูลในเซลล์เวิร์กชีต**
เข้าถึงเวิร์กชีตแรกและเติมส่วนหัวลงไป

```csharp
Worksheet sheet = workbook.Worksheets[0];
Cells cells = sheet.Cells;

// กำหนดแถวส่วนหัว
string[] headers = { "Employee", "Quarter", "Product", "Continent", "Country", "Sale" };
for (int i = 0; i < headers.Length; i++)
{
    // ตั้งค่าสำหรับแต่ละเซลล์ส่วนหัวในแถวแรก
    cells["A1"].Offset[0, i].PutValue(headers[i]);
}
```

**ขั้นตอนที่ 3: เพิ่มแถวข้อมูล**
เติมแถวข้อมูลด้วยข้อมูลพนักงาน

```csharp
string[,] employeeData = {
    { "David", "China", "Asia", "2000" },
    // ...ข้อมูลเพิ่มเติม...
};

for (int i = 0; i < employeeData.GetLength(0); i++)
{
    for (int j = 0; j < employeeData.GetLength(1); j++)
    {
        cells["A" + (i + 2)].Offset[0, j].PutValue(employeeData[i, j]);
    }
}
```

**ขั้นตอนที่ 4: กำหนดค่าวัตถุรายการ**
สร้างและกำหนดรูปแบบตารางภายในเวิร์กชีต

```csharp
Aspose.Cells.Tables.ListObject listObject = sheet.ListObjects[sheet.ListObjects.Add("A1", "F" + (employeeData.GetLength(0) + 1), true)];
listObject.TableStyleType = Aspose.Cells.Tables.TableStyleType.TableStyleMedium10;
listObject.ShowTotals = true;

// ตั้งค่าการคำนวณผลรวมสำหรับคอลัมน์ 'ไตรมาส'
listObject.ListColumns[1].TotalsCalculation = Aspose.Cells.Tables.TotalsCalculation.Count;
```

**ขั้นตอนที่ 5: บันทึกสมุดงาน**
สุดท้าย ให้บันทึกสมุดงานของคุณไปยังไดเร็กทอรีที่ระบุ

```csharp
workbook.Save(Path.Combine(outputDir, "output.xlsx"));
```

### คุณลักษณะที่ 2: เพิ่มข้อมูลและกำหนดค่ารูปแบบตาราง
ส่วนนี้ปรับปรุงคุณสมบัติเดิมให้ดีขึ้นด้วยการใช้สไตล์เฉพาะเพื่อความสวยงามยิ่งขึ้น

#### ภาพรวม
คล้ายกับฟีเจอร์แรก เราจะเติมข้อมูลในเซลล์แต่มีการกำหนดค่ารูปแบบเพิ่มเติมเพื่อให้ดูสวยงามยิ่งขึ้น

#### การดำเนินการแบบทีละขั้นตอน
**ขั้นตอนที่ 1-4**
ขั้นตอนจะคล้ายกับการตั้งค่าของ Feature 1 เน้นที่การกำหนดค่า `TableStyleType` และ `ShowTotals`-

```csharp
// เพิ่มรายการวัตถุ (ตาราง) พร้อมรูปแบบ
Aspose.Cells.Tables.ListObject listObject = sheet.ListObjects.Add("A1", "F" + (employeeData.GetLength(0) + 1), true);
listObject.TableStyleType = Aspose.Cells.Tables.TableStyleType.TableStyleMedium10;
listObject.ShowTotals = true;

// กำหนดค่าคอลัมน์ 'ไตรมาส' สำหรับผลรวม
table.ListColumns[1].TotalsCalculation = Aspose.Cells.Tables.TotalsCalculation.Count;
```

**ขั้นตอนที่ 5: บันทึกสมุดงาน**
บันทึกสมุดงานเหมือนเดิม

```csharp
workbook.Save(Path.Combine(outputDir, "styled_output.xlsx"));
```
