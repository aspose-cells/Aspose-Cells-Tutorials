# วิธีการใช้และจัดรูปแบบความคิดเห็นของ Excel โดยใช้ Aspose.Cells สำหรับ .NET: คำแนะนำทีละขั้นตอน

## คู่มือการใช้งาน
ตอนนี้เรามาดูคุณลักษณะแต่ละอย่างทีละขั้นตอนกัน

### การสร้างสมุดงานและแผ่นงาน
**ภาพรวม**:ส่วนนี้จะกล่าวถึงวิธีการสร้างเวิร์กบุ๊กและเพิ่มเวิร์กชีต
1. **การเริ่มต้นสมุดงาน**
   - เริ่มต้นด้วยการสร้างความว่างเปล่า `Workbook` วัตถุ.
   ```csharp
   Workbook workbook = new Workbook();
   ```
2. **เพิ่มแผ่นงานใหม่**
   - ใช้ `Worksheets.Add()` วิธีการผนวกแผ่นงานใหม่
   ```csharp
   int sheetIndex = workbook.Worksheets.Add();
   Worksheet worksheet = workbook.Worksheets[sheetIndex];
   // ตอนนี้สมุดงานมีแผ่นงานหนึ่งแผ่น
   ```

### การเพิ่มความคิดเห็นลงในเซลล์
**ภาพรวม**:เรียนรู้วิธีการแทรกความคิดเห็นลงในเซลล์ที่เจาะจง
1. **เพิ่มความคิดเห็น**
   - ใช้ `Comments.Add()` วิธีการใส่ความคิดเห็นในเซลล์ "F5"
   ```csharp
   int commentIndex = worksheet.Comments.Add("F5");
   Comment comment = worksheet.Comments[commentIndex];
   ```
2. **ตั้งค่าหมายเหตุความคิดเห็น**
   - กำหนดข้อความให้กับความคิดเห็นของคุณโดยใช้ `Note` คุณสมบัติ.
   ```csharp
   comment.Note = "Hello Aspose!";
   ```

### การจัดรูปแบบการปรากฏของความคิดเห็น
**ภาพรวม**:ปรับแต่งลักษณะการแสดงความคิดเห็นเพื่อให้สามารถอ่านได้ดีขึ้น
1. **ปรับขนาดและรูปแบบตัวอักษร**
   - เปลี่ยนขนาดตัวอักษรและใช้การจัดรูปแบบตัวหนา
   ```csharp
   comment.Font.Size = 14;
   comment.Font.IsBold = true;
   ```
2. **ตั้งค่าขนาดเป็นเซนติเมตร**
   - ระบุความสูงและความกว้างเพื่อควบคุมพื้นที่ภาพ
   ```csharp
   comment.HeightCM = 10;
   comment.WidthCM = 2;
   ```

### การบันทึกสมุดงาน
**ภาพรวม**: รักษาการเปลี่ยนแปลงของคุณโดยการบันทึกสมุดงาน
1. **บันทึกการเปลี่ยนแปลง**
   - ใช้ `Workbook.Save()` วิธีการเขียนการเปลี่ยนแปลงลงในไฟล์
   ```csharp
   workbook.Save(outputDir + "book1.out.xls");
   ```
