## คู่มือการใช้งาน

### การแก้ไขความคิดเห็นแบบเธรดในเวิร์กชีต Excel

คุณลักษณะนี้มุ่งเน้นไปที่การเข้าถึงและแก้ไขความคิดเห็นแบบเธรดภายในเซลล์เฉพาะของเวิร์กชีต Excel โดยใช้ Aspose.Cells สำหรับ .NET

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊ก

เริ่มต้นด้วยการโหลดไฟล์ Excel ที่มีอยู่ของคุณ ซึ่งทำได้โดยใช้ `Workbook` คลาสซึ่งแสดงถึงเวิร์กบุ๊ก Excel ทั้งหมด:

```csharp
// ตั้งค่าเส้นทางสำหรับไดเร็กทอรีต้นทางและปลายทาง
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

// โหลดเวิร์กบุ๊กจากไดเร็กทอรีที่ระบุ
Workbook workbook = new Workbook(SourceDir + "ThreadedCommentsSample.xlsx");
```

#### ขั้นตอนที่ 2: การเข้าถึงความคิดเห็นแบบเธรด

เข้าถึงเวิร์กชีตแรกและดึงความคิดเห็นแบบเธรดสำหรับเซลล์ที่ระบุ เช่น `A1`คุณสามารถกำหนดเป้าหมายเซลล์ใดๆ ได้โดยการเปลี่ยนการอ้างอิง:

```csharp
// รับแผ่นงานแรกจากสมุดงาน
Worksheet worksheet = workbook.Worksheets[0];

// ดึงความคิดเห็นแบบเธรดทั้งหมดสำหรับเซลล์ A1
ThreadedComment comment = worksheet.Comments.GetThreadedComments("A1")[0];
```

#### ขั้นตอนที่ 3: อัปเดตความคิดเห็น

เมื่อคุณเข้าถึงความคิดเห็นแบบเธรดที่เจาะจงแล้ว ให้ปรับปรุงเนื้อหาตามต้องการ:

```csharp
// แก้ไขหมายเหตุของความคิดเห็นแบบเธรด
comment.Notes = "Updated Comment";
```

#### ขั้นตอนที่ 4: บันทึกการเปลี่ยนแปลง

หลังจากทำการอัปเดตแล้ว ให้บันทึกเวิร์กบุ๊กเพื่อคงการเปลี่ยนแปลงไว้ คุณสามารถระบุชื่อไฟล์ใหม่หรือเขียนทับไฟล์เดิมได้:

```csharp
// บันทึกสมุดงานที่อัปเดตด้วยชื่อไฟล์ใหม่
workbook.Save(OutputDir + "EditThreadedComments.xlsx");
```

### การโหลดและการบันทึกสมุดงาน Excel

ฟีเจอร์นี้เป็นการสาธิตด่วนเกี่ยวกับวิธีการโหลดไฟล์ Excel ที่มีอยู่ ดำเนินการ และบันทึกกลับคืน

#### ขั้นตอนที่ 1: โหลดเวิร์กบุ๊กที่มีอยู่

โหลดสมุดงานของคุณโดยใช้ `Workbook` ระดับ:

```csharp
// ระบุไดเรกทอรีสำหรับการโหลดและบันทึกสมุดงาน
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

// โหลดเวิร์กบุ๊กจากไดเร็กทอรีที่ระบุ
Workbook workbook = new Workbook(SourceDir + "ExistingWorkbook.xlsx");
```

#### ขั้นตอนที่ 2: บันทึกสมุดงาน

หลังจากดำเนินการใดๆ (แก้ไข เพิ่มข้อมูล) ให้บันทึกการเปลี่ยนแปลงของคุณ:

```csharp
// บันทึกสมุดงานที่แก้ไขไปยังไฟล์ใหม่
workbook.Save(OutputDir + "SavedWorkbook.xlsx");
```
