## คู่มือการใช้งาน

### การตั้งค่าจำนวนแถวสูงสุดสำหรับสูตรที่ใช้ร่วมกัน

#### ภาพรวม
ฟีเจอร์นี้สาธิตวิธีการตั้งค่าขีดจำกัดจำนวนแถวที่สามารถขยายได้โดยสูตรที่ใช้ร่วมกันในเวิร์กชีต Excel

**ขั้นตอนที่ 1: สร้างวัตถุเวิร์กบุ๊ก**

เริ่มต้นด้วยการเริ่มต้นใหม่ `Workbook` วัตถุซึ่งแสดงถึงไฟล์ Excel ของคุณ

```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY"; // กำหนดไดเรกทอรีเอาท์พุตของคุณ

// การเริ่มต้นสมุดงาน
Workbook wb = new Workbook();
```

**ขั้นตอนที่ 2: กำหนดค่าจำนวนแถวสูงสุดสำหรับสูตรที่ใช้ร่วมกัน**

ใช้ `wb.Settings.MaxRowsOfSharedFormula` เพื่อกำหนดจำนวนแถวสูงสุดที่สูตรที่ใช้ร่วมกันสามารถครอบคลุมได้ ที่นี่ เราตั้งค่าให้เป็น 5 แถว

```csharp
// ตั้งค่าจำนวนแถวสูงสุดสำหรับสูตรที่ใช้ร่วมกัน
wb.Settings.MaxRowsOfSharedFormula = 5;
```

**ขั้นตอนที่ 3: บันทึกสมุดงานของคุณ**

สุดท้ายนี้ ให้บันทึกสมุดงานของคุณเพื่อใช้การตั้งค่าเหล่านี้

```csharp
// เข้าถึงแผ่นงานแรกและบันทึก
Worksheet ws = wb.Worksheets[0];
wb.Save(outputDir + "outputMaxRowsSharedFormula.xlsx");
```

### การใช้สูตรที่ใช้ร่วมกันในหลายแถว

#### ภาพรวม
เรียนรู้วิธีการใช้งาน `SetSharedFormula` วิธีการใช้สูตรกับเซลล์หลายเซลล์อย่างมีประสิทธิภาพ

**ขั้นตอนที่ 1: ตั้งค่าสมุดงานและแผ่นงานของคุณ**

เช่นเดียวกับก่อนหน้านี้ ให้เริ่มต้นเวิร์กบุ๊กของคุณและเข้าถึงเวิร์กชีตแรก

```csharp
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```

**ขั้นตอนที่ 2: ใช้สูตรที่ใช้ร่วมกัน**

เพื่อสาธิตให้เราลองประยุกต์ใช้ `Sum` สูตรจาก A1 ถึง A2 ครอบคลุม 100 แถวเริ่มต้นที่เซลล์ D1

```csharp
Cell cell = ws.Cells["D1"];
cell.SetSharedFormula("=Sum(A1:A2)", 100, 1);
```

**ขั้นตอนที่ 3: บันทึกสมุดงานของคุณ**

ตรวจสอบให้แน่ใจว่าคุณบันทึกการเปลี่ยนแปลงของคุณเพื่อดูผลของแอปพลิเคชันสูตรที่แชร์

```csharp
wb.Save(outputDir + "outputApplySharedFormula.xlsx");
```
