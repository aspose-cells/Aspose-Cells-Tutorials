## คู่มือการใช้งาน

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อโหลดไฟล์ Excel ขนาดใหญ่โดยไม่ต้องพบกับปัญหา `OutOfMemoryException`-

### กำหนดค่าตัวเลือกการโหลดสำหรับไฟล์ขนาดใหญ่

การเพิ่มประสิทธิภาพการใช้หน่วยความจำถือเป็นสิ่งสำคัญเมื่อต้องจัดการกับชุดข้อมูลจำนวนมาก ดังต่อไปนี้:

#### ขั้นตอนที่ 1: ระบุเส้นทางและเริ่มต้น LoadOptions
```csharp
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
// สร้างอินสแตนซ์ของ LoadOptions
LoadOptions options = new LoadOptions();
```

#### ขั้นตอนที่ 2: ตั้งค่าหน่วยความจำ
โดยใช้ `MemorySetting.MemoryPreference` เพิ่มประสิทธิภาพการใช้หน่วยความจำ:
```csharp
options.MemorySetting = MemorySetting.MemoryPreference;
```

#### ขั้นตอนที่ 3: โหลดเวิร์กบุ๊กด้วยตัวเลือกที่ระบุ
โหลดไฟล์ Excel ขนาดใหญ่ของคุณเพื่อป้องกันข้อผิดพลาดหน่วยความจำไม่เพียงพอ:
```csharp
Workbook book = new Workbook(dataDir + "sample.xlsx", options);
Console.WriteLine("File has been loaded successfully");
```

### เคล็ดลับการแก้ไขปัญหา
- **ให้แน่ใจว่ามีหน่วยความจำเพียงพอ**: ตรวจสอบว่า RAM ของระบบของคุณเพียงพอสำหรับการประมวลผลไฟล์ขนาดใหญ่
- **เพิ่มประสิทธิภาพโครงสร้างข้อมูล**:ประมวลผลข้อมูลล่วงหน้าเพื่อลดขนาดก่อนโหลดหากเป็นไปได้
