## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการใช้งานการคัดลอกแถวและคอลัมน์ด้วย Aspose.Cells

### ภาพรวมของการวางแถว/คอลัมน์
เป้าหมายคือการคัดลอกข้อมูลจากเวิร์กชีตหนึ่งไปยังอีกเวิร์กชีตหนึ่งในขณะที่ปรับแต่งพฤติกรรมการวาง เราจะใช้ `CopyOptions` และ `PasteOptions` เพื่อจุดประสงค์นี้

#### ขั้นตอนที่ 1: โหลดไฟล์ Excel ต้นฉบับ
เริ่มต้นด้วยการโหลดไฟล์ Excel ต้นฉบับของคุณ:

```csharp
// กำหนดไดเรกทอรี
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// โหลดสมุดงาน
Workbook wb = new Workbook(sourceDir + "SamplePasteOptions.xlsx");
```

#### ขั้นตอนที่ 2: เข้าถึงเวิร์กชีตแหล่งที่มาและปลายทาง
เข้าถึงทั้งเวิร์กชีตแหล่งที่มาที่มีข้อมูลของคุณและสร้างแผ่นงานปลายทาง:

```csharp
// รับแผ่นงานแรกเป็นแหล่งที่มา
Worksheet source = wb.Worksheets[0];

// เพิ่มอีกแผ่นสำหรับวาง
Worksheet destination = wb.Worksheets.Add("DestSheet");
```

#### ขั้นตอนที่ 3: กำหนดค่า CopyOptions
ชุด `CopyOptions` เพื่ออ้างอิงแหล่งข้อมูลไปยังแผ่นงานปลายทาง:

```csharp
// ตั้งค่าตัวเลือกการคัดลอก
CopyOptions options = new CopyOptions();
options.ReferToDestinationSheet = true;
```

#### ขั้นตอนที่ 4: กำหนด PasteOptions
การกำหนดค่า `PasteOptions` สำหรับการวางพฤติกรรมที่กำหนดเอง:

```csharp
// ตั้งค่าตัวเลือกการวาง
PasteOptions pasteOptions = new PasteOptions();
pasteOptions.PasteType = PasteType.Values; // วางเฉพาะค่า
pasteOptions.OnlyVisibleCells = true;      // รวมเฉพาะเซลล์ที่มองเห็นได้เท่านั้น
```

#### ขั้นตอนที่ 5: คัดลอกแถวที่มีตัวเลือก
ดำเนินการคัดลอกโดยใช้ตัวเลือกที่กำหนด:

```csharp
// ดำเนินการคัดลอกแถว
destination.Cells.CopyRows(source.Cells, 0, 0, source.Cells.MaxDisplayRange.RowCount, options, pasteOptions);
```

### เคล็ดลับการแก้ไขปัญหา
- **ไม่พบไฟล์**: ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้องและสามารถเข้าถึงได้
- **ตัวเลือกที่ไม่ถูกต้อง**: ตรวจสอบอีกครั้ง `PasteType` และการกำหนดค่าอื่น ๆ เพื่อให้เข้ากันได้กับข้อมูลของคุณ
