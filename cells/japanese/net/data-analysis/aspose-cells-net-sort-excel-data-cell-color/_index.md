## 実装ガイド

このセクションでは、セルの色でデータを並べ替える手順を段階的に説明します。

### ワークブックの作成と読み込み

まず、 `Workbook` クラスを作成して Excel ファイルを読み込みます。
```csharp
// ワークブックオブジェクトを作成し、テンプレートファイルをロードする
Workbook workbook = new Workbook(sourceDir + "sampleBackGroundFile.xlsx");
```
このコードは新しいブックを初期化し、ソース ディレクトリにある既存の Excel ファイルからデータを読み込みます。

### DataSorter の初期化

次に、 `DataSorter` 仕分けの準備をするクラス：
```csharp
// データソーターオブジェクトのインスタンス化
DataSorter sorter = workbook.DataSorter;
```
その `DataSorter` データの並べ替え操作を定義および実行するために不可欠です。

### セルの色による並べ替えキーの追加

データの並べ替え方法を指定します。ここでは、セルの色に基づいてキーを追加します。
```csharp
// 赤色の2列目のキーを追加します
csorter.AddKey(1, SortOnType.CellColor, SortOrder.Descending, Color.Red);
```
このステップでは、2 番目の列のセルの背景が赤い行を優先し、降順で並べ替えるようにソート機能に指示します。

### ソート操作の実行

キーを設定したら、ソートを実行します。
```csharp
// キーに基づいてデータを並べ替える
sorter.Sort(workbook.Worksheets[0].Cells, CellArea.CreateCellArea("A2", "C6"));
```
このコマンドは、定義されたセル領域 (A2 から C6) 内の行を基準に基づいて並べ替えます。

### ソートされたデータの保存

最後に、並べ替えたワークブックを保存します。
```csharp
// 出力ファイルを保存する
workbook.Save(outputDir + "outputsampleBackGroundFile.xlsx");
```
上記のコードは、処理されたデータを指定された出力ディレクトリ内の新しい Excel ファイルに保存します。
