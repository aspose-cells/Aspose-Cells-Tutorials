# Aspose.Cells を使用して Excel のグラフの目盛りラベルの方向を変更する 

この例では、Aspose.Cells for .NET を使用して、Excel ワークシート内のグラフの目盛りラベルの方向を変更する方法を示します。カテゴリ軸の目盛りラベルのテキスト方向を調整することで、グラフデータの読みやすさを向上させることができます。このコードは、既存の Excel ワークブックを読み込み、最初のワークシートにアクセスしてグラフを取得し、目盛りラベルの方向を水平に設定し、変更後のワークブックを保存します。このアプローチにより、.NET アプリケーションでグラフラベルの方向を正確に制御できます。

## インストール手順  
この例を実行するには、次の手順に従います。  

1. Aspose.Cells for .NET をインストールする  
   - ダウンロードはこちら [Aspose.Cells .NET 版](https://releases.aspose.com/cells/net/)  
   - または NuGet 経由でインストールします:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. プロジェクトの設定  
   - 新しい .NET プロジェクトを作成します。  
   - Aspose.Cells.dll への参照を追加します。  

3. コードを実行する  
   - 交換する `"Your Document Directory"` 実際のファイル パスを使用します。  
   - プログラムを実行して、チャートの目盛りラベルの方向を更新します。
