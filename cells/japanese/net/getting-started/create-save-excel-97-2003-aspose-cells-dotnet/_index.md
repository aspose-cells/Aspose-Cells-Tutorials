## 実装ガイド

### Excel 97-2003 ワークブックの作成と保存

このセクションでは、ワークブックを作成し、古い Excel 形式で保存する手順を説明します。

#### 機能の概要

Aspose.Cellsを使えば、簡単にワークブックを新規作成したり、既存のワークブックを操作したりできます。ここでは、新しいワークブックを作成し、それをExcel 97-2003形式（`.xls`）。

#### ステップバイステップの実装

**1. ディレクトリパスを設定する**

ファイル処理用のソース ディレクトリと出力ディレクトリを定義します。

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**2. 新しいワークブックオブジェクトを作成する**

インスタンス化する `Workbook` Excel ファイルの作成を開始するためのクラスです。

```csharp
// 新しいワークブックオブジェクトを作成する
Workbook workbook = new Workbook();
```

**3. ワークブックをExcel 97-2003形式で保存する**

ブックを保存するには、デフォルト設定を使用するか、形式を明示的に指定する方法の 2 つがあります。

*デフォルト設定の使用:*

```csharp
// ワークブックをExcel 97-2003形式で保存します
workbook.Save(OutputDir + "/output.xls");
```

*保存形式を明示的に指定する:*

```csharp
// 明示的なフォーマット指定で保存
workbook.Save(OutputDir + "/output.xls", SaveFormat.Excel97To2003);
```

**パラメータとメソッドの目的**

- `SaveFormat.Excel97To2003`以前のバージョンの Excel との互換性を確保します。
- `OutputDir + "/output.xls"`: ワークブックを保存するファイル パスを指定します。

#### トラブルシューティングのヒント

- **ファイルパスエラー**ディレクトリ パスにタイプミスや権限の問題がないか再確認してください。
- **ライブラリが見つかりません**Aspose.Cells がプロジェクトに正しくインストールされ、参照されていることを確認します。

## Excel 97-2003形式の一般的な制限

Excel 97-2003 形式を使用する場合は、次の制限に注意してください。

1. **行数制限**最大 65,536 行 (新しい形式では 1,048,576 行)
2. **列制限**最大 256 列 (XLSX では 16,384 列)
3. **ファイルサイズ**通常、同等のXLSXファイルよりも大きい
4. **限定機能**Excel の最新機能の一部はサポートされていません
5. **書式設定の制限**新しい Excel 形式に比べて書式設定オプションが少ない
