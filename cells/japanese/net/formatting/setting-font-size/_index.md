## 設置指示

.NET の Aspose.Cells を始めると,次のインストール手順を実行してください.  

1. NuGet パッケージ管理者によるインストール:  
   * ビジュアル・スタジオを開け  
   * ツール > NuGet パッケージ管理者 > ソリューションのための NuGet パッケージ管理へ  
   * 検索する`"アスペス・セル"`インストールをクリックします.  

2. NuGet パッケージ管理者コンソールでインストールする:  
   * ツール> NuGet パッケージ管理者> パッケージ管理者コンソールからパケット管理者コンソールを開く.  
   * コマンドを実行する:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. DLLを手動でダウンロードする:  
   * 訪問する[Aspose.Cells ダウンロードページ](https://releases.aspose.com/cells/net/)やってくれ  
   * パッケージをダウンロードして取り出す  
   * 参照を追加する`Aspose.Cells.dll`プロジェクトに
