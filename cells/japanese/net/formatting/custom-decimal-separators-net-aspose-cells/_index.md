## 実装ガイド

### 小数点とグループ区切りのカスタマイズ

**概要：**
小数点とグループの区切りをカスタマイズすると、データの読みやすさが向上し、さまざまな地域や企業で必要な特定の書式設定標準を満たすことができます。

#### ステップ1: 設定を構成する
まず、ワークブック全体に必要な数値形式を指定します。
```csharp
// カスタムの小数点とグループ区切りを定義する
workbook.Settings.NumberDecimalSeparator = '.';
workbook.Settings.NumberGroupSeparator = ' ';
```
**説明：** その `NumberDecimalSeparator` 多くの地域で一般的に使用されているピリオド（.）に設定されています。 `NumberGroupSeparator` スペース (' ') として設定されており、地域の設定に基づいて調整できます。

#### ステップ2: カスタムスタイルを適用する
セパレーターを定義したら、セルにカスタム スタイルを適用します。
```csharp
Worksheet worksheet = workbook.Worksheets[0];

// セルの値を設定し、スタイルを適用する
Cell cell = worksheet.Cells["A1"];
cell.PutValue(123456.789);

Style style = cell.GetStyle();
style.Custom = "#,##0.000;[Red]#,##0.000"; // カスタムフォーマット文字列
cell.SetStyle(style);
```
**説明：** カスタムフォーマット `#,##0.000` 小数点以下 3 桁を確保し、定義された区切り文字を使用して数字をグループ化します。

#### ステップ3: 列の自動調整
データが適切に表示されるように、列を自動調整します。
```csharp
worksheet.AutoFitColumns();
```
このメソッドは、列の幅をその内容に合わせて自動的に調整します。

#### ステップ4: PDFとして保存
最後に、カスタム設定を使用してワークブックを PDF として保存します。
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY/CustomSeparator_out.pdf");
```
