### 四角形コントロールの作成と構成
#### 概要
四角形コントロールを追加するには、ワークシートに新しい図形を作成し、配置、サイズ、線の太さ、破線スタイルなどのプロパティをカスタマイズする必要があります。

#### ステップバイステップガイド
**1. ワークブックをインスタンス化する**
まず、 `Workbook` クラス：
```csharp
// 新しいワークブックインスタンスを作成する
Workbook excelbook = new Workbook();
```

**2. 長方形を追加する**
使用 `AddRectangle` ワークシートに長方形を挿入する方法:
```csharp
// 指定した位置とサイズで四角形コントロールを追加します
Aspose.Cells.Drawing.RectangleShape rectangle = excelbook.Worksheets[0].Shapes.AddRectangle(3, 0, 2, 0, 70, 130);
```
- **パラメータ**パラメータ `(3, 0, 2, 0, 70, 130)` 四角形の行インデックス、列インデックス、幅、高さをポイント単位で定義します。

**3. 配置を設定する**
ワークシート内で四角形を配置する場所を定義します。
```csharp
// 配置をフリーフローティングに設定する
rectangle.Placement = 配置タイプ.FreeFloating;
```
- **PlacementType**FreeFloating では、セルに揃えずに移動できます。

**4. 外観をカスタマイズする**
視認性を高めるために、線の太さや破線スタイルなどの視覚プロパティを構成します。
```csharp
// 長方形の外観を変更する
rectangle.Line.Weight = 4; // 線の太さを設定する
rectangle.Line.DashStyle = MsoLineDashStyle.Solid; // ダッシュスタイルを実線として定義します
```
- **重さ**図形の境界線の太さを決定します。
- **ダッシュスタイル**パスのストロークに使用するダッシュとギャップのパターンを設定します。

**5. ワークブックを保存する**
最後に、新しく追加された四角形コントロールを含むワークブックを保存します。
```csharp
// 変更を新しいファイルに保存する
excelbook.Save(dataDir + "book1.out.xls");
```
