# Cambiar la dirección de etiqueta de gráfico en Excel usando Aspose.Cells 

Este ejemplo demuestra cómo modificar la dirección de las etiquetas de marca de gráfico en una hoja de trabajo de Excel utilizando Aspose.Cells para .NET. Ajustando la dirección de texto de las etiquetas de marca de eje de categoría, se puede mejorar la legibilidad de los datos de la tabla. El código carga un libro de trabajo de Excel existente, accede a la primera hoja de trabajo, recupera el gráfico, establece la dirección de marca de marca de gráfico a horizontal, y luego guarda el libro de trabajo modificado. Este enfoque garantiza un control preciso sobre la orientación de la etiqueta de gráfico en sus aplicaciones .NET.
