# Cambiar la dirección de la etiqueta de marca de un gráfico en Excel usando Aspose.Cells 

Este ejemplo muestra cómo modificar la dirección de las etiquetas de marca de un gráfico en una hoja de cálculo de Excel con Aspose.Cells para .NET. Al ajustar la dirección del texto de las etiquetas de marca del eje de categorías, se mejora la legibilidad de los datos del gráfico. El código carga un libro de Excel existente, accede a la primera hoja, recupera el gráfico, establece la dirección de la etiqueta de marca en horizontal y, a continuación, guarda el libro modificado. Este enfoque garantiza un control preciso de la orientación de las etiquetas del gráfico en las aplicaciones .NET.

## Instrucciones de instalación  
Para ejecutar este ejemplo, siga estos pasos:  

1. Instalar Aspose.Cells para .NET  
   - Descargar desde [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   - O instalar mediante NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Configura tu proyecto  
   - Crear un nuevo proyecto .NET.  
   - Agregue una referencia a Aspose.Cells.dll.  

3. Ejecutar el código  
   - Reemplazar `"Your Document Directory"` con la ruta del archivo real.  
   - Ejecute el programa para actualizar la dirección de la etiqueta de marca en el gráfico.
