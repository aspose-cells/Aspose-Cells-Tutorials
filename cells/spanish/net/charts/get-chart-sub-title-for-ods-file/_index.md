# Obtener el subtítulo del gráfico del archivo ODS usando Aspose.Cells  
Este ejemplo demuestra cómo usar Aspose.Cells para .NET para cargar un archivo ODS, acceder a un gráfico específico dentro de una hoja de cálculo y recuperar su subtítulo. Siguiendo esta guía, los desarrolladores pueden leer y extraer información de gráficos de hojas de cálculo ODS de forma eficiente con C#. El ejemplo proporciona un enfoque claro y paso a paso para trabajar con gráficos en archivos de Excel, garantizando una implementación fluida en aplicaciones .NET.

## Instrucciones de instalación:  
Para utilizar este ejemplo, siga estos pasos:  

1. Instalar Aspose.Cells para .NET a través de NuGet:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Agregue el espacio de nombres requerido en su proyecto C#:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Ejecute el código proporcionado para cargar un archivo ODS, acceder a un gráfico e imprimir su subtítulo en la consola.
