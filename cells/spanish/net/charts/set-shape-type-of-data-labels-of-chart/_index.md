# Modificar la forma de la etiqueta de datos de un gráfico en Excel usando Aspose.Cells
En este tutorial, exploraremos cómo modificar el tipo de forma de las etiquetas de datos en un gráfico de Excel con Aspose.Cells para .NET. Las etiquetas de datos en los gráficos mejoran la visualización, y cambiar su forma puede mejorar la legibilidad. Este ejemplo muestra cómo acceder a un gráfico de Excel existente, recuperar las etiquetas de datos de la serie y cambiar su tipo de forma a un óvalo de burbuja de diálogo (llamada de elipse de cuña). Con unas pocas líneas de código en C#, puede mejorar el aspecto visual de sus gráficos de Excel de forma eficiente.

## Instrucciones de instalación:  

Para utilizar Aspose.Cells para .NET en su proyecto, siga estos pasos:  

1. Instalar mediante NuGet:  
   - Abra su proyecto de Visual Studio.  
   - Vaya a Herramientas > Administrador de paquetes NuGet > Administrar paquetes NuGet para la solución.  
   - Busque Aspose.Cells e instale la última versión.  

   O utilice la consola del administrador de paquetes:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Descargar desde el sitio web de Aspose:  
   - Visita [Descargar Aspose.Cells para .NET](https://releases.aspose.com/cells/net/) y obtener las últimas DLL.  
   - Agregue la referencia DLL descargada a su proyecto manualmente.  

3. Aplicar la Licencia (Opcional pero Recomendado):  
   - Si tienes licencia, aplícala mediante:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Cells.lic");
     ```
