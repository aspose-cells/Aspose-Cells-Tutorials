# Crear PDF a partir de un gráfico de Excel con un tamaño de página especificado

Exportar gráficos de Excel a un archivo PDF con un diseño de página preciso es fundamental para la generación de informes y la visualización de datos. Este tutorial muestra cómo usar Aspose.Cells para .NET para generar un PDF a partir de un gráfico de Excel, especificando el tamaño y la alineación de la página. Aprenderá a cargar un archivo de Excel, acceder a una hoja de cálculo y su gráfico, y convertirlo a un archivo PDF con un formato correcto.

## Instrucciones de instalación  
Para utilizar Aspose.Cells para .NET, siga estos pasos:  

1. Instalar mediante NuGet:  
   Abra el Administrador de paquetes NuGet en Visual Studio y ejecute:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Descargar la DLL:  
   Obtenga la última versión de [Página de descarga de Aspose.Cells para .NET](https://releases.aspose.com/cells/net/).  

3. Incluir el espacio de nombres:  
   Agregue lo siguiente en su archivo C#:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
