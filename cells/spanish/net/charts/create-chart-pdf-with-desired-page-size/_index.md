# Crea PDF desde Excel Chart con tamaño de página especificado

Exportar gráficos de Excel a un archivo PDF con un diseño preciso de la página es un requisito crucial para la presentación de informes y visualización de datos. Este tutorial demuestra cómo usar Aspose.Cells para .NET para generar un PDF desde un gráfico Excel al tiempo que especifica el tamaño y la alineación de la página. Aprenderá a cargar un archivo Excel, acceder a una hoja de cálculo y su gráfico y convertirlo en un archivo PDF bien formado.
