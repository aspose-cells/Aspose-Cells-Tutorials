# Crear un gráfico de Excel con Aspose.Cells para .NET
Este ejemplo muestra cómo crear un libro de Excel, rellenarlo con datos de ejemplo y generar un gráfico de columnas con Aspose.Cells para .NET. La guía paso a paso abarca la creación de un libro, la adición de datos, la configuración de etiquetas de categoría y la configuración del gráfico. Aspose.Cells proporciona una potente API para automatizar la creación y manipulación de archivos de Excel en aplicaciones .NET, lo que facilita la generación de gráficos dinámicos mediante programación.

## Instrucciones de instalación  

Para utilizar Aspose.Cells para .NET, siga estos pasos:  

1. Instalar a través del Administrador de paquetes NuGet:  
   - Abra su proyecto .NET en Visual Studio.  
   - Vaya a Herramientas → Administrador de paquetes NuGet → Administrar paquetes NuGet para la solución.  
   - Busque Aspose.Cells e instale la última versión.  

   Alternativamente, instálelo usando la consola del administrador de paquetes:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Descargar desde el sitio web de Aspose:  
   - Visita la página oficial de descarga: [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   - Descargue la DLL y agregue una referencia a ella en su proyecto.  

3. Solicitar licencia (opcional):  
   - Obtener una licencia de [Compra de Aspose](https://purchase.aspose.com/buy) o solicitar una [licencia temporal](https://purchase.aspose.com/temporary-license/).  
   - Aplique la licencia en su código:  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
