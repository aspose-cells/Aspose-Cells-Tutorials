# Crear gráfico de Excel con Aspose.Cells para .NET
Este ejemplo demuestra cómo crear un libro de trabajo de Excel, llenarlo con datos de muestra y generar un gráfico de columna utilizando Aspose.Cells para .NET. La guía paso a paso cubre la creación de un libro de trabajo, agregar datos, establecer etiquetas de categoría y configurar el gráfico. Aspose.Cells proporciona una potente API para automatizar la creación y manipulación de archivos de Excel en aplicaciones .NET, lo que facilita la generación de gráficos dinámicos de forma programática.

## Instrucciones de instalación  

Para usar Aspose.Cells para .NET, siga estos pasos:  

1. Instalar a través del administrador de paquetes NuGet:  
   * Abre tu proyecto .NET en Visual Studio.  
   * Ve a Herramientas → Gestión de paquetes NuGet → Gestión de paquetes NuGet para Solución.  
   * Busque Aspose.Cells e instale la última versión.  

   Alternativamente, instale con la consola de gestión de paquetes:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Descarga desde el sitio web de Aspose:  
   * Visite la página oficial de descarga: [Aspose.Cells para .NET](https://releases.aspose.com/cells/net/)  
   * Descargue el DLL y añada una referencia a él en su proyecto.  

3. Licencia de solicitud (opcional):  
   * Obtenga una licencia de [Comprar con el objetivo](https://purchase.aspose.com/buy)o solicitar una [licencia temporal](https://purchase.aspose.com/temporary-license/)¿ Qué ?  
   * Aplica la licencia en tu código:  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
