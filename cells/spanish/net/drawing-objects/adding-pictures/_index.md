# Cree y guarde archivos de Excel con Aspose.Cells
Este tutorial muestra cómo usar Aspose.Cells para .NET para crear un nuevo archivo de Excel mediante programación. Aprenderá a comprobar la existencia de un directorio, instanciar un objeto Workbook, agregar una hoja de cálculo, insertar una imagen en una celda específica y guardar el archivo en la ruta deseada. Siga el sencillo fragmento de código proporcionado para empezar a optimizar sus tareas de automatización de Excel.

## Instrucciones de instalación
1. Instalar Aspose.Cells: Asegúrate de tener instalada la biblioteca Aspose.Cells. Puedes hacerlo usando el Administrador de paquetes NuGet ejecutando:
   ```
   Install-Package Aspose.Cells
   ```
2. Configura tu entorno: Crea un directorio en tu equipo donde quieras almacenar los archivos de Excel generados. Reemplaza `"Your Document Directory"` en el ejemplo con su ruta de directorio actual.
3. Implementar el código: Copie el código de ejemplo proporcionado en su aplicación .NET. Asegúrese de actualizar la ruta de la imagen (`dataDir + "logo.jpg"`) para señalar una imagen existente que desea insertar en el archivo Excel.
4. Ejecutar la aplicación: Ejecute la aplicación. El código creará un nuevo archivo de Excel llamado `output.xls` en el directorio especificado, con una imagen colocada en la celda "F6".
