# Manipulación de la orden Z de las formas en Excel usando Aspose.Cells
 
Este ejemplo demuestra cómo usar Aspose.Cells para .NET para cargar un libro de trabajo de Excel, manipular las posiciones de orden Z de las formas dentro de una hoja de trabajo y guardar el libro de trabajo actualizado. Siguiendo esta guía, los desarrolladores pueden obtener información sobre el acceso a las formas, ajustar su orden de apilamiento visual y garantizar la representación correcta de los elementos en archivos de Excel.

## Instrucciones de instalación  
1. Asegúrese de que tiene un entorno de desarrollo .NET configurado, como Visual Studio.  
2. Instale Aspose.Cells para .NET a través de NuGet Package Manager. Utilice el siguiente comando en la consola de gestión de paquetes:  
   ```
   Install-Package Aspose.Cells
   ```
3. Añadir una referencia al espacio de nombres Aspose.Cells en su proyecto:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Drawing; 
   ```
4. Prepare su archivo de Excel fuente y actualice el `fuenteDir`y `salidaDir`los caminos en los ejemplos de código proporcionados.  
5. Copie y pegar el fragmento de código proporcionado en su proyecto para manipular el orden Z de formas dentro de su archivo Excel.  
6. Construye y ejecute su aplicación para ver los cambios reflejados en el archivo de salida Excel.
