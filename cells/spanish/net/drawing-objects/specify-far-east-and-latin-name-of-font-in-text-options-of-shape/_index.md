# Crear un libro de trabajo Excel con fuentes de Oriente Lejano y latín

Este ejemplo demuestra cómo usar Aspose.Cells para .NET para crear un libro de trabajo Excel, agregar un cuadro de texto a una hoja de trabajo y establecer tanto el Lejano Oriente como los nombres de fuentes latinas para el texto dentro de esa caja de texto. El código muestra la inicialización del espacio de trabajo, la manipulación de elementos de hoja de trabajo y el almacenamiento del libro de trabajo con un formato de fuente específico adaptado a los diversos requisitos del idioma.

## Instrucciones de instalación
1. Instalar Aspose.Cells para .NET: 
   * Utilice el administrador de paquetes NuGet en su proyecto Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   * O descargar el DLL desde el [Aspose el sitio web](https://www.aspose.com/downloads/cells/net)¿ Qué ?

2. Añadir referencias: asegúrese de que su proyecto se refiere a la biblioteca Aspose.Cells.

3. Configurar el directorio de salida: actualizar el `salidaDir`variable en el código con el camino de salida deseado donde se guardará el archivo Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
