# Cree un libro de Excel con fuentes del Lejano Oriente y latinas

Este ejemplo muestra cómo usar Aspose.Cells para .NET para crear un libro de Excel, agregar un cuadro de texto a una hoja de cálculo y establecer nombres de fuentes tanto orientales como latinas para el texto dentro de dicho cuadro. El código muestra cómo inicializar el espacio de trabajo, manipular los elementos de la hoja de cálculo y guardar el libro con un formato de fuente específico, adaptado a los requisitos de diversos idiomas.

## Instrucciones de instalación
1. Instalar Aspose.Cells para .NET: 
   - Utilice el Administrador de paquetes NuGet en su proyecto de Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   - O descargue la DLL desde [Sitio web de Aspose](https://www.aspose.com/downloads/cells/net).

2. Agregar referencias: asegúrese de que su proyecto haga referencia a la biblioteca Aspose.Cells.

3. Configurar el directorio de salida: Actualizar el `outputDir` variable en el código con la ruta de salida deseada donde se guardará el archivo Excel.

4. Ejecutar el ejemplo: Copie el código C# proporcionado en su proyecto, compílelo y ejecútelo. Debería ver la confirmación de salida y se creará el archivo de Excel con el cuadro de texto configurado en el directorio especificado.
