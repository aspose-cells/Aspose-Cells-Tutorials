# Establecer el tamaño de fuente en Excel usando Aspose.Cells

Configurar el tamaño de fuente en un archivo de Excel es crucial para mejorar la legibilidad y resaltar los datos clave. Con Aspose.Cells para .NET, puede modificar fácilmente el tamaño de fuente de cualquier celda mediante programación. Este tutorial muestra cómo crear un nuevo libro de Excel, agregar una hoja de cálculo, rellenar una celda con texto, cambiar su tamaño de fuente y guardar el archivo en formato Excel. Siguiendo esta guía paso a paso, dominará el proceso de personalizar la apariencia del texto en archivos de Excel con C#.

## Instrucciones de instalación

Para comenzar a utilizar Aspose.Cells para .NET, siga estos pasos de instalación:  

1. Instalar a través del Administrador de paquetes NuGet:  
   - Abra Visual Studio.  
   - Vaya a Herramientas > Administrador de paquetes NuGet > Administrar paquetes NuGet para la solución.  
   - Buscar `Aspose.Cells` y haga clic en Instalar.  

2. Instalar a través de la consola del administrador de paquetes NuGet:  
   - Abra la Consola del administrador de paquetes desde Herramientas > Administrador de paquetes NuGet > Consola del administrador de paquetes.  
   - Ejecute el comando:  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. Descargar la DLL manualmente:  
   - Visita el [Página de descarga de Aspose.Cells](https://releases.aspose.com/cells/net/).  
   - Descargue y extraiga el paquete.  
   - Añadir una referencia a `Aspose.Cells.dll` en su proyecto.
