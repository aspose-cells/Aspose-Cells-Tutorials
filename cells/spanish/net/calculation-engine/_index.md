# Motor de cálculo de Excel para aplicaciones .NET

## Descripción general

Aspose.Cells para .NET ofrece un potente motor de cálculo que permite a los desarrolladores gestionar cálculos de fórmulas complejas en hojas de cálculo de Excel. Este potente componente permite configurar el comportamiento de los cálculos, implementar lógica de cálculo personalizada y optimizar el rendimiento de libros de gran tamaño.

## Características principales

- **Motores de cálculo personalizados**:Implemente su propia lógica de cálculo para fórmulas especializadas
- **Monitoreo de cálculos**: Cálculos de fórmulas de seguimiento y control para la optimización del rendimiento
- **Opciones de cálculo recursivo**:Configure cómo se manejan las referencias circulares
- **Configuración del sistema de fechas**: Cambiar entre los sistemas de fechas de 1900 y 1904
- **Detección de referencia circular**:Identificar y resolver dependencias circulares en fórmulas
- **Optimización del rendimiento**: Ajuste la configuración de cálculo para libros de trabajo grandes
