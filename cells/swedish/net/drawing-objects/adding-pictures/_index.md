# Skapa och spara Excel-filer med Aspose.Cells
Den här handledningen visar hur man använder Aspose.Cells för .NET för att skapa en ny Excel-fil programmatiskt. Du lär dig att kontrollera om det finns en katalog, instansiera ett Workbook-objekt, lägga till ett kalkylblad, infoga en bild i en specifik cell och spara filen till önskad sökväg. Följ det enkla kodavsnittet som medföljer för att komma igång med att förbättra dina Excel-automatiseringsuppgifter.

## Installationsanvisningar
1. Installera Aspose.Cells: Se till att du har Aspose.Cells-biblioteket installerat. Du kan göra detta med hjälp av NuGet Package Manager genom att köra:
   ```
   Install-Package Aspose.Cells
   ```
2. Konfigurera din miljö: Skapa en katalog på din dator där du vill lagra de genererade Excel-filerna. Ersätt `"Your Document Directory"` i exemplet med din faktiska katalogsökväg.
3. Implementera koden: Kopiera den medföljande exempelkoden till din .NET-applikation. Se till att du uppdaterar sökvägen till bilden (`dataDir + "logo.jpg"`) för att peka på en befintlig bild som du vill infoga i Excel-filen.
4. Kör programmet: Kör ditt program. Koden skapar en ny Excel-fil med namnet `output.xls` i den angivna katalogen, med en bild placerad i cell "F6".
