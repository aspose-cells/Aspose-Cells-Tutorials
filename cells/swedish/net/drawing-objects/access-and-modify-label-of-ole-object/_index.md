# Ändra Ole Object-etiketter i Excel med hjälp av Aspose.Cells

Detta exempel visar hur man får tillgång till och ändrar etiketten på ett Ole- objekt i en Excel-fil med hjälp av biblioteket Aspose.Cells för .NET. Genom att följa de tillhandahållna kodsnycklarna kan utvecklare enkelt manipulera Ole-objekt, uppdatera sina etiketter och spara ändringar effektivt. Denna förmåga är nödvändig för att dynamiskt förbättra Excel-filer i applikationer.

## Installationsinstruktioner
1. Installa Aspose.Cells för .NET: Använd NuGet Package Manager för att installera biblioteket Aspose.Cells i ditt projekt.
   ```
   Install-Package Aspose.Cells
   ```
2. Lägga till användningsdirektiv: Ange nödvändiga namnrum i kodfilen.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Lägg till exempelkod: Kopira och klistra den angivna koden i ditt projekt, för att se till att du anger dokumentkatalogsvägen.
4. Kör koden: Utföra ditt projekt för att få tillgång till och ändra Ole objektetiketten i din Excel-fil. Kontrollera konsolen för utgångsmeddelanden som bekräftar att processen genomförs framgångsrikt.
