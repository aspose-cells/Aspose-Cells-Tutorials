# Ändra Ole-objektetiketter i Excel med hjälp av Aspose.Cells

Det här exemplet visar hur man kommer åt och ändrar etiketten för ett Ole-objekt i en Excel-fil med hjälp av Aspose.Cells-biblioteket för .NET. Genom att följa de medföljande kodavsnitten kan utvecklare enkelt manipulera Ole-objekt, uppdatera deras etiketter och spara ändringar effektivt. Denna funktion är avgörande för att förbättra Excel-filer dynamiskt i applikationer.

## Installationsanvisningar
1. Installera Aspose.Cells för .NET: Använd NuGet Package Manager för att installera Aspose.Cells-biblioteket i ditt projekt.
   ```
   Install-Package Aspose.Cells
   ```
2. Lägg till Använda direktiv: Inkludera nödvändiga namnrymder i din kodfil.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Lägg till exempelkod: Kopiera och klistra in den angivna koden i ditt projekt och se till att ange sökvägen till dokumentkatalogen.
4. Kör koden: Kör ditt projekt för att komma åt och ändra Ole-objektetiketten i din Excel-fil. Kontrollera konsolen för utdatameddelanden som bekräftar att processen har körts.
