# Ändra tabellmärkningsskickning i Excel med hjälp av Aspose.Cells 

Detta exempel visar hur man ändrar riktningen på diagrammäklar i ett Excel- arbete med hjälp av Aspose.Cells för .NET. Genom att justera textriktningen på kategoriaxelmaklar kan du förbättra läsbarheten av diagramdata. Koden laddar en befintlig Excel-arbetebok, går till det första arbetsbladet, hämtar diagrammet, ställer in riktningen på etiketten till horisontell och sparar sedan den ändrade arbetsboken. Detta sätt säkerställer exakt kontroll över orientering av diagrammäklar i dina .NET-applikationer.

## Installationsinstruktioner  
För att köra detta exempel, följ dessa steg:  

1. Installa Aspose.Cells för .NET  
   * Ladda ner från [Aspose.Cells för .NET](https://releases.aspose.com/cells/net/)  
   * Eller installera via NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Ställ in ditt projekt  
   * Skapa ett nytt .NET-projekt.  
   * Lägg till en hänvisning till Aspose.Cells.dll.  

3. Kör koden  
   * Ersätt`"Din dokumentkatalog"`med den faktiska filvägen.  
   * Utföra programmet för att uppdatera riktningen på klocketiketten i diagrammet.
