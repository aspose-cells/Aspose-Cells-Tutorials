# Skapa PDF från Excel-diagram med angiven sidstorlek

Att exportera diagram från Excel till en PDF-fil med en exakt sidlayout är ett avgörande krav för rapportering och datavisualisering. Den här handledningen visar hur man använder Aspose.Cells för .NET för att generera en PDF från ett Excel-diagram samtidigt som man anger sidstorlek och justering. Du lär dig hur du laddar en Excel-fil, öppnar ett kalkylblad och dess diagram och konverterar det till en välformaterad PDF-fil.

## Installationsanvisningar  
För att använda Aspose.Cells för .NET, följ dessa steg:  

1. Installera via NuGet:  
   Öppna NuGet-pakethanteraren i Visual Studio och kör:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Ladda ner DLL-filen:  
   Hämta den senaste versionen från [Nedladdningssida för Aspose.Cells för .NET](https://releases.aspose.com/cells/net/).  

3. Inkludera namnrymden:  
   Lägg till följande i din C#-fil:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
