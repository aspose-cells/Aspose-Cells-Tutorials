# Skapa PDF från Excel- diagram med angivna sidstorlekar

Exporterar diagram från Excel till en PDF-fil med en exakt sidlayout är ett viktigt krav för rapportering och datavisualisering. Denna handledning visar hur du använder Aspose.Cells för .NET för att generera en PDF från ett Excel- diagram samtidigt som du anger sidstorleken och justeringen. Du lär dig hur du laddar en Excel-fil, åtkomst till ett arbetsblad och dess diagram och konverterar den till en välformattad PDF-fil.

## Installationsinstruktioner  
För att använda Aspose.Cells för .NET, följ dessa steg:  

1. Installa via NuGet:  
   Öppna NuGet Package Manager i Visual Studio och kör:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Ladda ner DLL:  
   Få den senaste versionen från [Aspose.Cells för .NET- nedladdningssida](https://releases.aspose.com/cells/net/)\- Jag är inte rädd.  

3. Inkludera Namnrummet:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
