# Ändra formen på diagramdataetiketten i Excel med hjälp av Aspose.Cells
I den här handledningen ska vi utforska hur man ändrar formtypen för dataetiketter i ett Excel-diagram med hjälp av Aspose.Cells för .NET. Dataetiketter i diagram ger bättre visualisering, och att ändra deras form kan förbättra läsbarheten. Det här exemplet visar hur man kommer åt ett befintligt Excel-diagram, hämtar seriedataetiketter och ändrar deras formtyp till en pratbubbla med oval form (Wedge Ellipse Callout). Med några rader C#-kod kan du effektivt förbättra dina Excel-diagrams visuella attraktionskraft.

## Installationsanvisningar:  

För att använda Aspose.Cells för .NET i ditt projekt, följ dessa steg:  

1. Installera via NuGet:  
   - Öppna ditt Visual Studio-projekt.  
   - Gå till Verktyg > NuGet-pakethanterare > Hantera NuGet-paket för lösningen.  
   - Sök efter Aspose.Cells och installera den senaste versionen.  

   Eller använd pakethanterarkonsolen:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Ladda ner från Asposes webbplats:  
   - Besök [Aspose.Cells för .NET nedladdning](https://releases.aspose.com/cells/net/) och hämta de senaste DLL-filerna.  
   - Lägg till den nedladdade DLL-referensen manuellt i ditt projekt.  

3. Använd licensen (valfritt men rekommenderat):  
   - Om du har en licens, ansök om den med hjälp av:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Cells.lic");
     ```
