## कार्यान्वयन मार्गदर्शिका

### विशेषता: कार्यपुस्तिका निर्माण

एक बनाने के `Workbook` अपनी एक्सेल फ़ाइल का प्रतिनिधित्व करने के लिए ऑब्जेक्ट:

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // अपने वास्तविक निर्देशिका पथ से बदलें
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

### विशेषता: वर्कशीट तक पहुँचना

किसी विशिष्ट कार्यपत्रक तक उसके अनुक्रमणिका या नाम का उपयोग करके पहुँचें:

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Workbook;

Worksheet worksheet = workbook.getWorksheets().get(0); // पहली वर्कशीट तक पहुंचें
```

### विशेषता: ऑटो-फिट रो

किसी पंक्ति की ऊंचाई को उसकी सामग्री के आधार पर स्वचालित रूप से फ़िट करें `autoFitRow`:

```java
// दूसरी पंक्ति (सूचकांक 1) को समायोजित करता है
worksheet.autoFitRow(1);
```

### विशेषता: ऑटो-फिट कॉलम

सभी सामग्री को क्लिपिंग के बिना दृश्यमान बनाने के लिए कॉलम को स्वचालित रूप से फिट करें:

```java
// पहले कॉलम की चौड़ाई समायोजित करता है (सूचकांक 0)
worksheet.autoFitColumn(0);
```

### विशेषता: कार्यपुस्तिका सहेजें

समायोजन के बाद कार्यपुस्तिका सहेजें:

```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // वास्तविक आउटपुट निर्देशिका पथ से प्रतिस्थापित करें
workbook.save(outDir + "AutoFitRowsandColumns_out.xls");
```
