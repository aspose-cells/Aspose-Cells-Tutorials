## Java में Excel वर्कबुक कैसे लोड करें?
`Workbook` क्लास एक Excel फ़ाइल को मेमोरी में लोड किए हुए दर्शाता है, जो उसकी वर्कशीट्स, सेल्स, और चार्ट्स तक पहुँच प्रदान करता है।

`new Workbook("path/to/file.xlsx")` के साथ वर्कबुक लोड करें – यह एकल लाइन पूरी स्प्रेडशीट को मेमोरी में पढ़ लेती है, जिससे आपको सभी वर्कशीट्स, सेल्स, और एम्बेडेड चार्ट्स तक प्रोग्रामेटिक एक्सेस मिलती है। Aspose.Cells स्वचालित रूप से फ़ाइल फ़ॉर्मेट का पता लगाता है, इसलिए आपको स्पष्ट रूप से XLSX, XLS, या CSV निर्दिष्ट करने की आवश्यकता नहीं है।

## फ़ाइल से वर्कबुक लोड करें
**सारांश:**  
पहला कदम Excel वर्कबुक को लोड करना है। यह चार्ट्स तक पहुँच के लिए वातावरण सेट करता है।

```java
import com.aspose.cells.Workbook;
// Load an Excel workbook from a specified directory.
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

**व्याख्या:**  
- `Workbook` क्लास शीर्ष‑स्तर का ऑब्जेक्ट है जो मेमोरी में एकल Excel फ़ाइल को दर्शाता है।  
- `dataDir` वैरिएबल या पूर्ण पाथ के माध्यम से अपने Excel फ़ाइल का पूरा पाथ प्रदान करें।

## विशिष्ट वर्कशीट और चार्ट तक कैसे पहुँचें?
`Worksheet` ऑब्जेक्ट वर्कबुक के भीतर एकल शीट से मेल खाता है, जिसमें पंक्तियाँ, कॉलम, और एम्बेडेड ऑब्जेक्ट्स होते हैं।

`Chart` ऑब्जेक्ट वर्कशीट पर डेटा का ग्राफ़िकल प्रतिनिधित्व दर्शाता है, जिसे रेंडर या एक्सपोर्ट किया जा सकता है।

`workbook.getWorksheets().get(0)` से वर्कशीट प्राप्त करें और फिर `getCharts().get(0)` कॉल करके पहला चार्ट ऑब्जेक्ट प्राप्त करें – यह सीधा तरीका किसी भी चार्ट इंडेक्स के लिए काम करता है जिसकी आपको आवश्यकता है। API एक `Chart` इंस्टेंस रिटर्न करता है जो रेंडरिंग या डेटा एक्सट्रैक्शन के लिए तैयार है।

## वर्कशीट और चार्ट तक पहुँचें
**सारांश:**  
लोड करने के बाद, उस विशिष्ट वर्कशीट और चार्ट तक पहुँचें जिसे आप कनवर्ट करना चाहते हैं।

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;
// Access the first worksheet and its first chart.
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```

**व्याख्या:**  
- `worksheet` प्रकार `Worksheet` का ऑब्जेक्ट है।  
- `chart` वर्कशीट के चार्ट कलेक्शन से प्राप्त किया गया है।

## चार्ट को SVG इमेज में कैसे कनवर्ट करें?
`ImageOrPrintOptions` क्लास रेंडरिंग सेटिंग्स को परिभाषित करता है जैसे आउटपुट फ़ॉर्मेट, रिज़ॉल्यूशन, और क्वालिटी, जो चार्ट्स या वर्कशीट्स को इमेज फ़ाइलों में कनवर्ट करने के लिए उपयोग होती हैं।

`ImageOrPrintOptions` का एक इंस्टेंस बनाएं, उसका `setSaveFormat(SaveFormat.SVG)` सेट करें, फिर `chart.toImage(options, "output.svg")` कॉल करें। यह एक‑लाइन कॉल एक पूर्णतः मानक SVG फ़ाइल लिखती है जो रंग, फ़ॉन्ट, और डेटा लेबल्स को ठीक उसी तरह संरक्षित रखती है जैसा कि Excel में दिखता है।

## चार्ट को SVG इमेज में कनवर्ट करें
**सारांश:**  
अंतिम चरण में चार्ट को उच्च‑गुणवत्ता वाले डिस्प्ले के लिए SVG इमेज में कनवर्ट किया जाता है।

```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.SaveFormat;
// Convert and save the chart as an SVG image.
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setSaveFormat(SaveFormat.SVG);
String outDir = "YOUR_OUTPUT_DIRECTORY";
chart.toImage(outDir + "CCToImageinSVGFormat_out.svg", options);
```

**व्याख्या:**  
- `ImageOrPrintOptions` चार्ट को कैसे सेव किया जाए, इसे कॉन्फ़िगर करता है।  
- फ़ॉर्मेट को SVG सेट करने से Aspose.Cells को वेक्टर ग्राफ़िक जेनरेट करने का निर्देश मिलता है।  
- परिणामी फ़ाइल को सीधे HTML या CSS बैकग्राउंड में एम्बेड किया जा सकता है।
