# Excel में Aspose.Cells for Java के साथ चार्ट कैसे जोड़ें: एक व्यापक गाइड

## परिचय

आज के डेटा‑चालित विश्व में, Excel वर्कबुक में **how to add chart** जल्दी जोड़ना स्थिर रिपोर्ट और प्रभावशाली दृश्य कहानी के बीच अंतर बन सकता है। डायनेमिक चार्ट आपको कच्चे आंकड़ों को स्पष्ट अंतर्दृष्टियों में बदलने की अनुमति देते हैं बिना मैन्युअल कॉपी‑पेस्ट के। यह ट्यूटोरियल आपको Aspose.Cells for Java के साथ चार्ट निर्माण को स्वचालित करने के चरण दिखाता है, ताकि आप कोड से सीधे पेशेवर‑दिखावट वाले ग्राफ़िक्स बना सकें।

## त्वरित उत्तर

- **प्राथमिक लाइब्रेरी क्या है?** Aspose.Cells for Java.
- **कौन सा Maven आर्टिफैक्ट आवश्यक है?** `com.aspose:aspose-cells:25.3`.
- **क्या मुझे विकास के लिए लाइसेंस चाहिए?** एक मुफ्त ट्रायल परीक्षण के लिए काम करता है; एक पेड लाइसेंस मूल्यांकन सीमाओं को हटाता है।
- **क्या मैं लाइन, पाई, और बार चार्ट बना सकता हूँ?** हाँ—20 से अधिक चार्ट प्रकार बॉक्स से बाहर समर्थित हैं।
- **क्या बड़े‑फ़ाइल हैंडलिंग कुशल है?** Aspose.Cells 500‑पेज वर्कबुक को < 200 MB मेमोरी उपयोग के साथ प्रोसेस करता है।

## Aspose.Cells for Java क्या है?

`Aspose.Cells` लाइब्रेरी एक Java API है जो Microsoft Office के बिना Excel फ़ाइलों का निर्माण, हेरफेर और रूपांतरण सक्षम करती है। यह वर्कशीट, सेल और चार्ट के लिए एक समृद्ध ऑब्जेक्ट मॉडल प्रदान करती है, जिससे आप **how to add chart** प्रोग्रामेटिक रूप से कर सकते हैं, साथ ही स्टाइलिंग और डेटा बाइंडिंग पर पूर्ण नियंत्रण रख सकते हैं।

Aspose.Cells को एकीकृत करने के लिए अपने `pom.xml` फ़ाइल में निम्नलिखित डिपेंडेंसी शामिल करें:

## इम्प्लीमेंटेशन गाइड

अब चलिए मुख्य प्रश्न का उत्तर देते हैं: **how to add chart** एक Java वर्कबुक में।

### Java में प्रोग्रामेटिक रूप से चार्ट कैसे जोड़ें?

एक `Workbook` लोड या बनाएं, एक वर्कशीट में डेटा जोड़ें, फिर उस डेटा रेंज से जुड़ा एक `Chart` ऑब्जेक्ट इंस्टैंशिएट करें। अंत में, वर्कबुक को सहेजें। यह एंड‑टू‑एंड फ्लो केवल कुछ लाइनों के कोड में पूरा हो जाता है और समर्थित किसी भी चार्ट प्रकार के लिए काम करता है।  
एक `Workbook` मेमोरी में एक Excel फ़ाइल का प्रतिनिधित्व करता है।  
एक `Chart` ऑब्जेक्ट वर्कशीट के भीतर डेटा का दृश्य प्रतिनिधित्व परिभाषित करता है।

### अपने वर्कशीट में डेटा जोड़ना

पहले, हम वर्कशीट को नमूना डेटा से भरेंगे:

```java
// Obtain a reference to the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);
Cells cells = worksheet.getCells();

// Populate data in A1:B4 and C1:C4 as values and categories
cells.get("A1").setValue(50); // Add value to A1
cells.get("B1").setValue(60); // Add value to B1, etc.
```

### चार्ट सम्मिलित करना

अब, वर्कशीट में एक चार्ट जोड़ें:

```java
// Access the charts collection of the worksheet
ChartCollection charts = worksheet.getCharts();

// Add a new chart (e.g., Column type) to the worksheet
int chartIndex = charts.add(ChartType.COLUMN, 5, 0, 15, 5);
Chart chart = charts.get(chartIndex);

// Configure the chart's data source and category
SeriesCollection nSeries = chart.getNSeries();
nSeries.add("A1:B4", true); // Specify data range
nSeries.setCategoryData("C1:C4"); // Set category range

System.out.println("Chart added successfully.");
```

### अपने वर्कबुक को सहेजना

अंत में, अपने वर्कबुक को फ़ाइल में सहेजें:

```java
workbook.save("SettingChartsData_out.xls");
System.out.println("Workbook with chart is created successfully.");
```
