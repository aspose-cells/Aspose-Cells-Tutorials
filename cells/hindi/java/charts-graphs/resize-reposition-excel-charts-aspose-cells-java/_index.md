- वर्कशीट पर चार्ट को पुनः स्थान देने के तरीके

### कार्यान्वयन मार्गदर्शिका
इस अनुभाग में, हम Java के लिए Aspose.Cells का उपयोग करके चार्ट को लोड करने, आकार बदलने और पुनः स्थिति देने का तरीका जानेंगे।
#### चार्ट लोड करें और उसका आकार बदलें
चार्ट का आकार बदलने से उसका स्वरूप आपकी डेटा प्रस्तुति आवश्यकताओं के अनुरूप हो जाता है। यहाँ बताया गया है कि कैसे:
##### चरण 1: कार्यपुस्तिका इंस्टेंस बनाएँ
का एक उदाहरण बनाकर मौजूदा एक्सेल फ़ाइल लोड करें `Workbook`.
```java
String filePath = "YOUR_DATA_DIRECTORY/book1.xls";
Workbook workbook = new Workbook(filePath);
```
##### चरण 2: पहली वर्कशीट तक पहुँचें
हम पहली वर्कशीट के साथ काम करेंगे, जो कई उपयोग मामलों में आम है।
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```
##### चरण 3: चार्ट लोड करें
उस चार्ट तक पहुँचें जिसका आकार आप बदलना चाहते हैं। इस उदाहरण में, हम शीट पर पहले चार्ट के साथ काम कर रहे हैं।
```java
Chart chart = worksheet.getCharts().get(0);
```
##### चरण 4: चार्ट का आकार बदलें
अपने चार्ट की चौड़ाई और ऊंचाई के लिए नए आयाम सेट करें.
```java
chart.getChartObject().setWidth(400); // चार्ट की चौड़ाई 400 इकाई पर सेट करें
chart.getChartObject().setHeight(300); // चार्ट की ऊंचाई 300 इकाई पर सेट करें

// परिवर्तन सहेजें
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "ResizeChart_out.xls");
```
#### चार्ट को पुनः स्थानित करें
चार्ट को फिर से पोजिशन करने से लेआउट और पठनीयता बेहतर होती है। यहाँ बताया गया है कि कैसे:
##### चरण 1: एक्सेल फ़ाइल लोड करें
अपनी कार्यपुस्तिका लोड करें.
```java
String filePath = "YOUR_DATA_DIRECTORY/book1.xls";
Workbook workbook = new Workbook(filePath);
```
##### चरण 2: वर्कशीट और चार्ट तक पहुंचें
आकार बदलने के समान, आवश्यक वर्कशीट और चार्ट तक पहुंचें।
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```
##### चरण 3: चार्ट को पुनः स्थानित करें
अपने चार्ट को वर्कशीट के भीतर ले जाने के लिए X और Y निर्देशांक समायोजित करें।
```java
chart.getChartObject().setX(250); // क्षैतिज स्थिति को 250 इकाई पर सेट करें
chart.getChartObject().setY(150); // ऊर्ध्वाधर स्थिति को 150 इकाई पर सेट करें

// परिवर्तनों को नई फ़ाइल में सहेजें
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "RepositionChart_out.xls");
```
