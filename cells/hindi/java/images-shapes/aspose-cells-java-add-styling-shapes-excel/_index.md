# Aspose.Cells Java का उपयोग करके Excel में आकृतियाँ कैसे जोड़ें और स्टाइल करें

## परिचय

प्रोग्रामेटिक रूप से कस्टम आकार जोड़कर अपने एक्सेल वर्कशीट को बेहतर बनाएं `Aspose.Cells` जावा के लिए। यह ट्यूटोरियल आपको आयताकार आकार जोड़ने, इसकी लाइन शैलियों को कॉन्फ़िगर करने और ग्रेडिएंट फिल लगाने के बारे में मार्गदर्शन करता है।

**आप क्या सीखेंगे:**
- अपने जावा प्रोजेक्ट में Aspose.Cells सेट अप करना।
- एक्सेल वर्कशीट में एक आयताकार आकार जोड़ना।
- आकृतियों के लिए रेखा शैलियाँ और ग्रेडिएंट कॉन्फ़िगर करना.
- संशोधित कार्यपुस्तिका को सहेजा जा रहा है.

आइये सबसे पहले यह सुनिश्चित करें कि आप सभी पूर्वापेक्षाएँ पूरी करते हैं।

## आवश्यक शर्तें

कोड में आगे बढ़ने से पहले, सुनिश्चित करें:
- **पुस्तकालय:** Aspose.Cells लाइब्रेरी (संस्करण 25.3 या बाद का) आपके प्रोजेक्ट में शामिल है।
- **पर्यावरण:** निर्भरता प्रबंधन के लिए मावेन या ग्रेडल जैसे जावा विकास वातावरण से परिचित होना।
- **ज्ञान:** जावा प्रोग्रामिंग और एक्सेल फ़ाइल हेरफेर की बुनियादी समझ।

## Java के लिए Aspose.Cells सेट अप करना

अपने बिल्ड टूल का उपयोग करके Aspose.Cells को अपने Java प्रोजेक्ट में एकीकृत करें:

**मावेन:**
अपने में जोड़ें `pom.xml`:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**ग्रेडेल:**
अपने में शामिल करें `build.gradle` फ़ाइल:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

### लाइसेंस अधिग्रहण

आप Aspose.Cells को बिना किसी सीमा के परीक्षण करने के लिए एक अस्थायी लाइसेंस प्राप्त कर सकते हैं या इसे दीर्घकालिक उपयोग के लिए खरीद सकते हैं। [एक निःशुल्क परीक्षण](https://releases.aspose.com/cells/java/) और एक अधिग्रहण पर विचार करें [अस्थायी लाइसेंस](https://purchase.aspose.com/temporary-license/) यदि ज़रूरत हो तो।

### मूल आरंभीकरण

निर्भरता जोड़ने के बाद, अपने जावा प्रोजेक्ट में Aspose.Cells को आरंभ करें:
```java
import com.aspose.cells.Workbook;

public class ExcelShapeDemo {
    public static void main(String[] args) throws Exception {
        Workbook excelBook = new Workbook();
        // आगे की कार्यवाही यहीं होगी।
    }
}
```

## कार्यान्वयन मार्गदर्शिका

### एक्सेल वर्कशीट में आयत आकार जोड़ना

**अवलोकन:** Aspose.Cells का उपयोग करके अपने वर्कशीट में आयत आकार जोड़ने और उसकी स्थिति निर्धारित करने का तरीका जानें।

#### चरण 1: नई कार्यपुस्तिका बनाएँ
```java
Workbook excelBook = new Workbook();
```
यह एक नई कार्यपुस्तिका इंस्टैंस आरंभ करता है, जहां आप आकृतियां जोड़ेंगे।

#### चरण 2: एक आयताकार आकार जोड़ें
```java
import com.aspose.cells.RectangleShape;
import com.aspose.cells.MsoDrawingType;

RectangleShape rectangle = (RectangleShape) excelBook.getWorksheets().get(0)
        .getShapes().addShape(MsoDrawingType.RECTANGLE, 3, 2, 0, 0, 70, 130);
```
यहाँ, पहली वर्कशीट में एक आयत जोड़ा गया है। पैरामीटर इसके प्रकार, स्थिति और आकार को निर्दिष्ट करते हैं।

#### चरण 3: प्लेसमेंट सेट करें
```java
rectangle.setPlacement(com.aspose.cells.PlacementType.FREE_FLOATING);
```
यह आकृति को किसी विशिष्ट सेल श्रेणी में स्थिर रखने के बजाय मुक्त-तैरने योग्य बनाता है।

### किसी आकृति की रेखा शैली कॉन्फ़िगर करना

**अवलोकन:** अपने आयताकार आकार के लिए रेखा शैली और ग्रेडिएंट भरण को अनुकूलित करें।

#### चरण 1: लाइन शैली कॉन्फ़िगर करें
```java
import com.aspose.cells.LineFormat;
import com.aspose.cells.MsoLineStyle;

LineFormat linestyle = rectangle.getLine();
linestyle.setDashStyle(MsoLineStyle.THICK_THIN);
linestyle.setWeight(4);
```
यह लाइन शैली को मोटे-पतले डैश पैटर्न पर सेट करता है और इसके वजन को समायोजित करता है।

#### चरण 2: ग्रेडिएंट भरण लागू करें
```java
import com.aspose.cells.FillFormat;
import com.aspose.cells.GradientStyleType;

FillFormat fillformat = rectangle.getFill();
fillformat.setOneColorGradient(com.aspose.cells.Color.getBlue(), 1, 
    GradientStyleType.HORIZONTAL, 1);
```
दृश्य वृद्धि के लिए आयत के भरण पर ग्रेडिएंट प्रभाव लागू किया जाता है।

### कार्यपुस्तिका को सहेजना

अंत में, अपनी कार्यपुस्तिका को सभी कॉन्फ़िगरेशन के साथ सहेजें:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
excelBook.save(outDir + "/StyledRectangle_out.xls");
```

- **टेम्पलेट डिजाइनिंग:** विशिष्ट ग्राफिकल तत्वों की आवश्यकता वाले रिपोर्ट या चालान के लिए टेम्पलेट बनाएं।
- **स्वचालित रिपोर्ट निर्माण:** प्रोग्रामेटिक रूप से आकृतियों को जोड़कर और स्टाइल देकर स्वचालित प्रक्रियाओं को बेहतर बनाएँ।
