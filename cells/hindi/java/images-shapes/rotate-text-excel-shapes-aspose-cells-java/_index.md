# Aspose.Cells Java में महारत हासिल करना: Excel में आकृतियों के साथ टेक्स्ट को घुमाना

## परिचय

एक्सेल स्प्रेडशीट के साथ काम करते समय, आपको ऐसे परिदृश्यों का सामना करना पड़ सकता है जहाँ किसी आकृति के भीतर के टेक्स्ट को पूरी आकृति को घुमाए बिना सटीक रूप से संरेखित करने की आवश्यकता होती है। यह ट्यूटोरियल आपको इसका उपयोग करने में मार्गदर्शन करेगा **जावा के लिए Aspose.Cells** इस कार्यक्षमता को प्राप्त करने के लिए। साथ चलते हुए, आप सीखेंगे कि आकृतियों को स्थिर रखते हुए आकृतियों के भीतर पाठ को कुशलतापूर्वक कैसे घुमाया जाए - यह आपके एक्सेल दस्तावेज़ की पठनीयता और प्रस्तुति को बढ़ाने के लिए एकदम सही है।

### आप क्या सीखेंगे:
- Aspose.Cells के साथ एक मौजूदा Excel फ़ाइल लोड करें।
- वर्कशीट कक्षों और आकृतियों तक पहुंचें और उनमें परिवर्तन करें।
- आकृतियों के अंदर पाठ को उनके अभिविन्यास में परिवर्तन किए बिना घुमाएँ।
- परिवर्तनों को नई एक्सेल फ़ाइल में सहेजें.

आइये उन पूर्वापेक्षाओं पर नजर डालें जिनकी आपको शुरुआत करने के लिए आवश्यकता होगी।

## कार्यान्वयन मार्गदर्शिका

### सुविधा 1: नमूना एक्सेल फ़ाइल लोड करें

#### अवलोकन
मौजूदा एक्सेल फ़ाइल को लोड करना हमारी प्रक्रिया का पहला चरण है।

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook wb = new Workbook(dataDir + "/sampleRotateTextWithShapeInsideWorksheet.xlsx");
```

**स्पष्टीकरण**: द `Workbook` क्लास आपकी पूरी स्प्रेडशीट का प्रतिनिधित्व करता है। फ़ाइल पथ पास करके, आप एक्सेल दस्तावेज़ को मेमोरी में लोड करते हैं।

### फ़ीचर 2: पहली वर्कशीट तक पहुँचें

#### अवलोकन
विशिष्ट कार्यपत्रकों तक पहुंचने से हमें पाठ और आकार हेरफेर के लिए सटीक क्षेत्रों को लक्षित करने की अनुमति मिलती है।

```java
import com.aspose.cells.Worksheet;

Worksheet ws = wb.getWorksheets().get(0);
```

**स्पष्टीकरण**: `getWorksheets()` सभी शीटों का संग्रह लौटाता है, जबकि `get(0)` प्रथम कार्यपत्रक तक पहुँचता है.

### फ़ीचर 3: सेल में संदेश जोड़ें

#### अवलोकन
Aspose.Cells के साथ कक्षों में पाठ जोड़ना सरल है।

```java
import com.aspose.cells.Cell;

Cell b4 = ws.getCells().get("B4");
b4.putValue("Text is not rotating with shape because RotateTextWithShape is false.");
```

**स्पष्टीकरण**: `getCells()` सभी सेल ऑब्जेक्ट्स को लाता है, और `putValue` किसी विशिष्ट सेल को पाठ निर्दिष्ट करता है.

### फ़ीचर 4: वर्कशीट में पहले आकार तक पहुँचें

#### अवलोकन
आकृतियों में हेरफेर करने में पाठ संरेखण को समायोजित करने के लिए उनके गुणों तक पहुँच शामिल है।

```java
import com.aspose.cells.Shape;
import com.aspose.cells.ShapeTextAlignment;

Shape sh = ws.getShapes().get(0);
ShapeTextAlignment shapeTextAlignment = sh.getTextBody().getTextAlignment();
shapeTextAlignment.setRotateTextWithShape(false);
```

**स्पष्टीकरण**: द `getShapes()` विधि सभी आकृतियों को पुनः प्राप्त करती है, और हम सेटिंग द्वारा पाठ संरेखण को संशोधित करते हैं `setRotateTextWithShape` झूठा.

### फ़ीचर 5: एक्सेल फ़ाइल को आउटपुट डायरेक्टरी में सेव करें

#### अवलोकन
अंत में, अपने परिवर्तनों को एक नई फ़ाइल में सहेजें।

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
wb.save(outDir + "/outputRotateTextWithShapeInsideWorksheet.xlsx");
```

**स्पष्टीकरण**: द `save()` विधि सभी संशोधनों को निर्दिष्ट आउटपुट निर्देशिका में लिखती है।

## व्यावहारिक अनुप्रयोगों

1. **रिपोर्ट पीढ़ी**: उन रिपोर्टों को अनुकूलित करें जहां ग्राफ़िक्स को विकृत किए बिना पाठ लेबल महत्वपूर्ण हैं।
2. **डैशबोर्ड अनुकूलन**: वर्णनात्मक पाठ को घुमाते हुए व्यवसाय डैशबोर्ड में स्थिर दृश्य बनाए रखें।
3. **शिक्षण सामग्री**: स्पष्ट, सुव्यवस्थित टिप्पणियों के साथ शैक्षिक सामग्री बनाएं।
4. **विपणन संपार्श्विक**: ऐसे विपणन शीट डिज़ाइन करें जिनमें भिन्न पाठ दिशाओं के बावजूद एक समान आकार अभिविन्यास की आवश्यकता हो।

## प्रदर्शन संबंधी विचार

- **फ़ाइल लोडिंग अनुकूलित करें**: मेमोरी उपयोग को कम करने के लिए केवल आवश्यक कार्यपत्रक लोड करें।
