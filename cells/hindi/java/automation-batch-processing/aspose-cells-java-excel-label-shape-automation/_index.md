# Aspose.Cells for Java के साथ Excel Workbook कैसे बनाएं: लेबल शैप जोड़ना

## परिचय

यदि आपको Java में प्रोग्रामेटिक रूप से **create excel workbook** बनाना है, तो Aspose.Cells for Java इसे तेज़, विश्वसनीय और Microsoft Office से पूरी तरह स्वतंत्र बनाता है। इस ट्यूटोरियल में आप देखेंगे कि लाइब्रेरी कैसे सेटअप करें, **aspose cells license** लागू करें, लेबल शैप जोड़ें, और अंत में **save excel workbook** को डिस्क पर सहेजें। अंत तक आप **java generate excel** फ़ाइलों के मुख्य चरणों में सहज हो जाएंगे और एक सामान्य प्रोजेक्ट में **how to use aspose** को ठीक से जानेंगे।

## त्वरित उत्तर
- **कौनसी लाइब्रेरी आवश्यक है?** Aspose.Cells for Java (available via Maven or Gradle).  
- **क्या मैं मुफ्त ट्रायल उपयोग कर सकता हूँ?** Yes – download from Aspose’s website and apply a temporary license.  
- **लेबल शैप कैसे जोड़ें?** Use `sheet.getShapes().addShape(MsoDrawingType.LABEL, …)`.  
- **कौनसा संस्करण लेबल शैप को सपोर्ट करता है?** Version 25.3 or later.  
- **वर्कबुक को कैसे सहेजें?** Call `workbook.save("path/filename.xls")`.

## Aspose.Cells के साथ “create excel workbook” क्या है?

Excel workbook बनाना मतलब Java कोड से प्रोग्रामेटिक रूप से `.xls` या `.xlsx` फ़ाइल उत्पन्न करना है। Aspose.Cells लो‑लेवल फ़ाइल फ़ॉर्मेट विवरणों को एब्स्ट्रैक्ट करता है, जिससे आप फ़ाइल हैंडलिंग के बजाय बिज़नेस लॉजिक पर ध्यान केंद्रित कर सकते हैं। यह वर्कशीट, सेल, स्टाइल और शैप के लिए ऑब्जेक्ट्स प्रदान करता है, जिससे आप पूरी तरह से कोड में फुल‑फ़ीचर स्प्रेडशीट बना सकते हैं।

## इम्प्लीमेंटेशन गाइड

### नया Workbook बनाना

`Workbook` क्लास मेमोरी में एक पूरा Excel फ़ाइल दर्शाता है और सभी ऑपरेशन्स के लिए एंट्री पॉइंट है।

```java
import com.aspose.cells.Workbook;
// Create an empty workbook
Workbook workbook = new Workbook();
```

### पहली Worksheet तक पहुंचना

`Worksheet` क्लास आपको वर्कबुक के अंदर एक सिंगल शीट तक पहुंच देता है जहाँ आप डेटा, चार्ट या शैप जोड़ सकते हैं।

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Worksheets;
// Get the first worksheet from the workbook
Worksheet sheet = workbook.getWorksheets().get(0);
```

### लेबल शैप जोड़ना

`Shape` क्लास (विशेष रूप से लेबल शैप) आपको टेक्स्ट‑बेस्ड ग्राफ़िक्स डालने की अनुमति देता है जो सेल्स के ऊपर फ्लोट करते हैं।

```java
import com.aspose.cells.Label;
import com.aspose.cells.MsoDrawingType;
// Add a label shape to the worksheet
Label label = (Label) sheet.getShapes().addShape(MsoDrawingType.LABEL, 2, 2, 2, 0, 60, 120);
```

### लेबल टेक्स्ट सेट करना

`Label` ऑब्जेक्ट की `setText` मेथड शैप के अंदर दिखने वाली स्ट्रिंग को परिभाषित करती है।

```java
// Set text for the label
label.setText("This is a Label");
```

### लेबल प्लेसमेंट टाइप कॉन्फ़िगर करना

`PlacementType` एनेमरेशन यह नियंत्रित करता है कि रो या कॉलम इन्सर्ट या डिलीट होने पर लेबल कैसे व्यवहार करता है।

```java
import com.aspose.cells.PlacementType;
// Configure label placement
label.setPlacement(PlacementType.FREE_FLOATING);
```

### ग्रेडिएंट के साथ फ़िल कलर सेट करना

`FillFormat` क्लास ग्रेडिएंट फ़िल्स को सक्षम करता है, जिससे आप लेबल बैकग्राउंड पर स्मूद कलर ट्रांज़िशन लागू कर सकते हैं।

```java
import com.aspose.cells.Color;
import com.aspose.cells.GradientStyleType;
// Set one-color gradient as fill for the label
label.getFill().setOneColorGradient(Color.getYellow(), 1, GradientStyleType.HORIZONTAL, 1);
```

### वर्कबुक को सेव करना

`save` मेथड इन‑मेमोरी वर्कबुक को फिजिकल फ़ाइल में लिखता है। आप XLS, XLSX, CSV, या PDF आउटपुट फ़ॉर्मेट चुन सकते हैं।

```java
// Define output directory and save the workbook
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/AddingLabelControl_out.xls");
```
