# Aspose.Cells Java के साथ पिवट टेबल प्रदर्शित करें: कॉम्पैक्ट, आउटलाइन और टेबुलर फॉर्म

## परिचय

क्या आप हर बार सही लेआउट के लिए पिवट टेबल को मैन्युअल रूप से समायोजित करने में संघर्ष कर रहे हैं? Aspose.Cells for Java के साथ, विभिन्न रूपों में पिवट टेबल प्रदर्शित करना - कॉम्पैक्ट, आउटलाइन और सारणीबद्ध - सरल है। यह मार्गदर्शिका आपको दिखाएगी कि Aspose.Cells Java का उपयोग करके आसानी से अपने डेटा प्रस्तुति को कैसे बदला जाए।

**आप क्या सीखेंगे:**
- पिवट टेबल को कॉम्पैक्ट रूप में कैसे प्रदर्शित करें
- पिवट तालिकाओं को रूपरेखा रूप में दिखाने की तकनीकें
- पिवट तालिकाओं को सारणीबद्ध रूप में प्रस्तुत करने के चरण

इस ट्यूटोरियल के अंत तक, आप Aspose.Cells Java का उपयोग करके विभिन्न रूपों में पिवट टेबल प्रदर्शित करने में निपुण हो जाएँगे। आइए जानें कि आरंभ करने के लिए आपको क्या चाहिए।

## Java के लिए Aspose.Cells सेट अप करना

जावा के लिए Aspose.Cells का उपयोग करने के लिए, आपको इसे अपने प्रोजेक्ट में शामिल करना होगा। आपके पास दो विकल्प हैं: Maven या Gradle.

### मावेन
अपने में निम्नलिखित निर्भरता जोड़ें `pom.xml` फ़ाइल:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### ग्रैडल
इसे अपने में शामिल करें `build.gradle` फ़ाइल:
```gradle

```

## कार्यान्वयन मार्गदर्शिका

हम कार्यान्वयन को तीन खंडों में विभाजित करेंगे: कॉम्पैक्ट, आउटलाइन और सारणीबद्ध फॉर्म।

### पिवट तालिका को संक्षिप्त रूप में दिखाएं

**अवलोकन:** पिवट तालिका को संक्षिप्त रूप में प्रदर्शित करने से स्पष्टता बनाए रखते हुए स्थान बचाने में मदद मिलती है।

#### चरण 1: एक्सेल फ़ाइल लोड करें
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```
*क्यों?* यह आपकी स्रोत एक्सेल फ़ाइल को मेमोरी में लोड करता है।

#### चरण 2: वर्कशीट और पिवट टेबल तक पहुंचें
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### चरण 3: कॉम्पैक्ट फ़ॉर्म सेट करें
```java
pivotTable.showInCompactForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/CompactForm.xlsx");
```
*क्यों?* यह कॉन्फ़िगरेशन पिवट तालिका को संक्षिप्त रूप में प्रदर्शित करता है और उसे सहेजता है।

### पिवट तालिका को आउटलाइन रूप में दिखाएं

**अवलोकन:** रूपरेखा प्रपत्र पदानुक्रमित डेटा के लिए आदर्श है, जो उपयोगकर्ताओं को विवरण को विस्तारित या संक्षिप्त करने की अनुमति देता है।

#### चरण 1: कार्यपुस्तिका लोड करें
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### चरण 2: आवश्यक घटकों तक पहुंचें
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### चरण 3: आउटलाइन फ़ॉर्म कॉन्फ़िगर करें
```java
pivotTable.showInOutlineForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/OutlineForm.xlsx");
```
*क्यों?* यह चरण पिवट तालिका को रूपरेखा स्वरूप में सेट करता है और यह सुनिश्चित करता है कि डेटा अद्यतन हो।

### पिवट तालिका को सारणीबद्ध रूप में दिखाएं

**अवलोकन:** सारणीबद्ध प्रारूप सभी डेटा को पंक्तियों में प्रदर्शित करता है, जो विस्तृत विश्लेषण के लिए आदर्श है।

#### चरण 1: कार्यपुस्तिका आरंभ करें
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### चरण 2: घटकों तक पहुँचें
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### चरण 3: सारणीबद्ध प्रारूप सेट करें
```java
pivotTable.showInTabularForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/TabularForm.xlsx");
```
*क्यों?* यह कॉन्फ़िगरेशन पिवट तालिका को सारणीबद्ध रूप में प्रस्तुत करता है।

## व्यावहारिक अनुप्रयोगों

पिवट तालिकाओं को विभिन्न रूपों में प्रदर्शित करने के लिए यहां कुछ वास्तविक उपयोग के मामले दिए गए हैं:

1. **वित्तीय रिपोर्ट:** वित्तीय डेटा को शीघ्रता से सारांशित करने के लिए कॉम्पैक्ट फॉर्म का उपयोग करें।
2. **बिक्री विश्लेषण:** रूपरेखा प्रपत्र विक्रय डेटा को पदानुक्रमिक रूप से समझने में मदद कर सकता है।
3. **सूची प्रबंधन:** सारणीबद्ध प्रपत्र विस्तृत आइटम सूची प्रदान करता है।

एकीकरण संभावनाओं में उन्नत डेटा विज़ुअलाइज़ेशन के लिए BI टूल और डैशबोर्ड से जुड़ना शामिल है।

## प्रदर्शन संबंधी विचार

Aspose.Cells के साथ काम करते समय, निम्नलिखित पर विचार करें:

- **मेमोरी उपयोग अनुकूलित करें:** सुनिश्चित करें कि आपके जावा अनुप्रयोग में बड़ी एक्सेल फ़ाइलों को संभालने के लिए पर्याप्त मेमोरी आवंटन है।
- **कुशल डेटा रिफ्रेशिंग:** उपयोग `refreshData()` और `calculateData()` प्रदर्शन को बनाए रखने के लिए विवेकपूर्ण तरीके से काम करना चाहिए।
- **सर्वोत्तम प्रथाएं:** प्रदर्शन सुधार का लाभ उठाने के लिए अपनी Aspose.Cells लाइब्रेरी को नियमित रूप से अपडेट करें।
