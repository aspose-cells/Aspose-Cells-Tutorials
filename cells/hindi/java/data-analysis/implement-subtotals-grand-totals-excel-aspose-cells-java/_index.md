# जावा के लिए Aspose.Cells का उपयोग करके Excel में उप-योग और कुल योग को कैसे लागू करें

## परिचय

Excel में बड़े डेटासेट के साथ काम करते समय, डेटा को प्रभावी ढंग से सारांशित करना गेम-चेंजर हो सकता है। यह लेख आपको Aspose.Cells for Java का उपयोग करके अपने Excel शीट में उप-योग और कुल योग को लागू करने के बारे में बताता है - एक शक्तिशाली लाइब्रेरी जो स्प्रेडशीट स्वचालन को सरल बनाती है।

इस ट्यूटोरियल के अंत तक आप सीखेंगे कि कैसे:

- अपने विकास परिवेश में Java के लिए Aspose.Cells सेट अप करें
- उप-योग और महायोग को आसानी से लागू करें
- अपनी स्थानीयकरण आवश्यकताओं के अनुरूप उप-योग लेबल अनुकूलित करें

क्या आप अपनी डेटा विश्लेषण प्रक्रिया को सरल बनाने के लिए तैयार हैं? आइये इसके लिए आवश्यक बातों पर गौर करें।

## कार्यान्वयन मार्गदर्शिका

### अवलोकन

यह अनुभाग आपको Java के लिए Aspose.Cells का उपयोग करके अपने Excel कार्यपत्रकों में उप-योग लागू करने और लेबल को अनुकूलित करने में मार्गदर्शन करेगा।

### चरण-दर-चरण निर्देश

#### 1. अपनी स्रोत कार्यपुस्तिका लोड करें

सबसे पहले, अपना डेटा युक्त एक्सेल फ़ाइल लोड करें:

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = Utils.getSharedDataDir(ImplementSubtotalGrandTotallabels.class) + "TechnicalArticles/";

// अपनी स्रोत कार्यपुस्तिका लोड करें
Workbook wb = new Workbook(dataDir + "sample.xlsx");
```

#### 2. उप-योग और कुल योग लेबल को अनुकूलित करें

इन लेबलों को स्थानीयकृत करने के लिए, वैश्वीकरण सेटिंग्स सेट करें:

```java
// उप-योग और कुल योग नाम बदलने के लिए वैश्वीकरण सेटिंग सेट करें
GlobalizationSettings gsi = new GlobalizationSettingsImp();
wb.getSettings().setGlobalizationSettings(gsi);
```

#### 3. अपनी वर्कशीट तक पहुँचें

उस विशिष्ट कार्यपत्रक तक पहुँचें जहाँ आप उप-योग लागू करना चाहते हैं:

```java
// पहली वर्कशीट तक पहुंचें
Worksheet ws = wb.getWorksheets().get(0);
```

#### 4. सबटोटल फ़ंक्शन लागू करें

उपयोग `subtotal` अपनी इच्छित सेल श्रेणी पर विधि, यह निर्दिष्ट करना कि किन स्तंभों का उप-योग करना है और एक समेकन फ़ंक्शन का उपयोग करना जैसे `SUM`:

```java
// स्तंभ 2, 3, और 4 के लिए A1:B10 पर उप-योग लागू करें (अनुक्रमण 0 से शुरू होता है)
CellArea ca = CellArea.createCellArea("A1", "B10");
ws.getCells().subtotal(ca, 0, ConsolidationFunction.SUM, new int[] { 2, 3, 4 });
```

#### 5. कॉलम की चौड़ाई समायोजित करें

बेहतर दृश्यता के लिए, आप कॉलम की चौड़ाई समायोजित कर सकते हैं:

```java
// पहले कॉलम की चौड़ाई निर्धारित करें
ws.getCells().setColumnWidth(0, 40);
```

#### 6. अपनी कार्यपुस्तिका सहेजें

अंत में, अपनी कार्यपुस्तिका को सभी परिवर्तनों के साथ सहेजें:

```java
// आउटपुट एक्सेल फ़ाइल को सेव करें
wb.save(dataDir + "ImplementTotallabels_out.xlsx");
```

### समस्या निवारण युक्तियों

- सुनिश्चित करें कि आपका एक्सेल फ़ाइल पथ सही है.
- उप-योग लागू करते समय जाँच लें कि आप स्तंभों के लिए सही अनुक्रमणिका का उपयोग कर रहे हैं।
- यदि आपको किसी सुविधा संबंधी सीमा का सामना करना पड़े तो अपने लाइसेंस सेटअप को सत्यापित करें।
