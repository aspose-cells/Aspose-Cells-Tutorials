इस गाइड में हम निम्नलिखित विषयों पर चर्चा करेंगे:

- स्क्रैच से एक साझा कार्यपुस्तिका बनाना।
- विशिष्ट दिनों के इतिहास को संरक्षित करने के लिए साझा कार्यपुस्तिका में संशोधन लॉग कॉन्फ़िगर करना।
