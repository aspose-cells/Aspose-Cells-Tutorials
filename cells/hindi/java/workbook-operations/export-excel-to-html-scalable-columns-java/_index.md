# जावा के लिए Aspose.Cells का उपयोग करके स्केलेबल कॉलम के साथ Excel को HTML में कैसे निर्यात करें

## परिचय

क्या आपको कभी एक्सेल वर्कबुक को वेब-फ्रेंडली फॉर्मेट में बदलने की ज़रूरत पड़ी है, और आपने पाया है कि आपके ब्राउज़र में देखने पर कॉलम सिकुड़ते या अजीब तरह से फैलते हैं? यह ट्यूटोरियल ठीक इसी समस्या का समाधान करके बताता है कि इसका उपयोग कैसे करें **जावा के लिए Aspose.Cells** एक्सेल फ़ाइलों को स्केलेबल कॉलम चौड़ाई के साथ HTML के रूप में निर्यात करने के लिए। यह दृष्टिकोण सुनिश्चित करता है कि आपका डेटा किसी भी डिवाइस पर सही दिखता है।

Aspose.Cells के साथ, आप Excel संचालन को स्वचालित कर सकते हैं और अपने Java अनुप्रयोगों में सहज एकीकरण प्राप्त कर सकते हैं। इस गाइड में, हम HtmlSaveOptions को कॉन्फ़िगर करने की प्रक्रिया से गुजरेंगे ताकि यह सुनिश्चित हो सके कि आउटपुट फ़ाइल में सीधे Base64 एनकोडेड स्ट्रिंग के रूप में छवियों को एम्बेड करते समय कॉलम गतिशील रूप से समायोजित हो। 

**आप क्या सीखेंगे:**
- HTML निर्यात के लिए स्केलेबल कॉलम चौड़ाई को कैसे कॉन्फ़िगर करें
- बेस 64 एनकोडिंग का उपयोग करके छवियों को एम्बेड कैसे करें
- Java के लिए Aspose.Cells का चरण-दर-चरण सेटअप और उपयोग

आइये शुरू करने से पहले आवश्यक शर्तों पर गौर करें।

## कार्यान्वयन मार्गदर्शिका

आइए बेहतर समझ के लिए कार्यान्वयन को अलग-अलग विशेषताओं में विभाजित करें।

### सुविधा 1: फ़ाइल से कार्यपुस्तिका लोड करें

**अवलोकन**: इस चरण में Aspose.Cells का उपयोग करके एक Excel फ़ाइल लोड करना शामिल है। यह सुनिश्चित करना महत्वपूर्ण है कि आपका डेटा स्रोत सही ढंग से निर्दिष्ट किया गया है।

#### चरण-दर-चरण कार्यान्वयन:
1. **कार्यपुस्तिका वर्ग आयात करें**

   ```java
   import com.aspose.cells.Workbook;
   ```

2. **कार्यपुस्तिका लोड करें**

   यहां, हम अपनी एक्सेल फ़ाइल का पथ निर्दिष्ट करते हैं:

   ```java
   String dataDir = "YOUR_DATA_DIRECTORY";
   Workbook wb = new Workbook(dataDir + "/sampleForScalableColumns.xlsx");
   ```

   *क्यों?* कार्यपुस्तिका को लोड करना आवश्यक है क्योंकि यह आपके डेटा को हेरफेर और रूपांतरण के लिए तैयार करता है।

### फ़ीचर 2: स्केलेबल कॉलम चौड़ाई के लिए HtmlSaveOptions कॉन्फ़िगर करें

**अवलोकन**: यह सुविधा हमें HTML सेव विकल्पों को कॉन्फ़िगर करने की अनुमति देती है, जो कॉलम की चौड़ाई को स्केलेबल बनाने और बेस 64 प्रारूप में छवियों को एम्बेड करने पर ध्यान केंद्रित करती है।

#### चरण-दर-चरण कार्यान्वयन:
1. **HtmlSaveOptions क्लास आयात करें**

   ```java
   import com.aspose.cells.HtmlSaveOptions;
   ```

2. **सहेजें विकल्प सेट करें**

   कॉन्फ़िगर करें `HtmlSaveOptions` यह सुनिश्चित करने के लिए कि कॉलम स्वचालित रूप से समायोजित हो जाएं:

   ```java
   HtmlSaveOptions options = new HtmlSaveOptions();
   options.setWidthScalable(true); // स्तंभों के लिए स्केलेबल चौड़ाई सक्षम करें
   options.setExportImagesAsBase64(true); // छवियों को बेस64 स्ट्रिंग के रूप में एम्बेड करें
   ```

   *क्यों?* स्केलेबल कॉलम चौड़ाई सुनिश्चित करती है कि आपका HTML विभिन्न स्क्रीन आकारों पर अच्छी तरह से प्रस्तुत हो। Base64 में छवियों को एम्बेड करने से बाहरी फ़ाइलों की आवश्यकता समाप्त हो जाती है, जिससे वितरण सरल हो जाता है।

### फ़ीचर 3: कॉन्फ़िगर किए गए विकल्पों के साथ HTML फ़ॉर्मेट में कार्यपुस्तिका सहेजें

**अवलोकन**अंतिम चरण कॉन्फ़िगर किए गए विकल्पों का उपयोग करके हमारी कार्यपुस्तिका को HTML फ़ाइल के रूप में सहेजना है।

#### चरण-दर-चरण कार्यान्वयन:
1. **पुनः कॉन्फ़िगर करें और सहेजें**

   यदि आवश्यक हो तो कार्यपुस्तिका को पुनः लोड करें, कॉन्फ़िगरेशन पुनः लागू करें, और सहेजें:

   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY";
   Workbook wb = new Workbook(dataDir + "/sampleForScalableColumns.xlsx");
   HtmlSaveOptions options = new HtmlSaveOptions();
   options.setWidthScalable(true);
   options.setExportImagesAsBase64(true);

   // कॉन्फ़िगर किए गए विकल्पों के साथ कार्यपुस्तिका को HTML प्रारूप में सहेजें
   wb.save(outDir + "/outsampleForScalableColumns.html", options);
   ```

   *क्यों?* कार्यपुस्तिका को सहेजने से यह सुनिश्चित होता है कि सभी कॉन्फ़िगरेशन लागू हो गए हैं, और आपका डेटा इच्छित रूप से निर्यात हो गया है।

1. **रिपोर्टिंग**: गतिशील रिपोर्ट बनाएं जो विभिन्न डिवाइसों पर खूबसूरती से समायोजित हो जाएं।
2. **डेटा साझा करना**: अतिरिक्त सॉफ्टवेयर इंस्टॉलेशन की आवश्यकता के बिना आसानी से सारणीबद्ध डेटा वितरित करें।
3. **डैशबोर्ड एकीकरण**: एक्सेल-आधारित डैशबोर्ड को वेब अनुप्रयोगों में सहजता से एकीकृत करें।
