## कार्यान्वयन मार्गदर्शिका
### कस्टम दशमलव संख्या स्वरूपण का अवलोकन
कस्टम दशमलव संख्या स्वरूपण आपको एक्सेल के संख्या प्रारूप का पालन करते हुए, आपके एप्लिकेशन में संख्याओं को प्रदर्शित करने के तरीके को नियंत्रित करने की अनुमति देता है। यह सुविधा सटीक डेटा प्रतिनिधित्व की आवश्यकता वाले अनुप्रयोगों के लिए महत्वपूर्ण है।

#### चरण-दर-चरण कार्यान्वयन
**1. वर्कबुक और एक्सेस वर्कशीट बनाएं**
सबसे पहले, इसका एक उदाहरण बनाएं `Workbook` क्लास और इसकी पहली वर्कशीट तक पहुँचें:
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class Main {
    public static void main(String[] args) throws Exception {
        // कार्यपुस्तिका आरंभ करें
        Workbook wb = new Workbook();
        
        // पहली वर्कशीट तक पहुंचें
        Worksheet ws = wb.getWorksheets().get(0);
    }
}
```
**2. मान डालें और प्रारूपित करें**
सेल A1 में मान डालें और कस्टम दशमलव संख्या स्वरूपण लागू करें:
```java
import com.aspose.cells.Cell;
import com.aspose.cells.Style;

public class Main {
    public static void main(String[] args) throws Exception {
        // पिछला कोड...

        // सेल A1 पर पहुँचें और मान 123 डालें
        Cell cell = ws.getCells().get("A1");
        cell.putValue(123);

        // कस्टम दशमलव संख्या स्वरूपण लागू करें
        Style st = cell.getStyle();
        st.setCustom("[DBNum2][$-804]General");

        // सेल के लिए स्वरूपित शैली सेट करें
        cell.setStyle(st);
    }
}
```
**3. कार्यपुस्तिका को PDF के रूप में सहेजें**
अंत में, अपनी कार्यपुस्तिका को PDF प्रारूप में सहेजें:
```java
import com.aspose.cells.SaveFormat;

public class Main {
    public static void main(String[] args) throws Exception {
        // पिछला कोड...

        // कॉलम की चौड़ाई सेट करें और कार्यपुस्तिका सहेजें
        ws.getCells().setColumnWidth(0, 30);
        wb.save("outputDecimalNumberFormatting.pdf", SaveFormat.PDF);

        System.out.println("Custom Decimal Number Formatting Done Successfully");
    }
}
```
