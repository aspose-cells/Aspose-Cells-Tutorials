## कार्यान्वयन मार्गदर्शिका

यह अनुभाग आपको Java के लिए Aspose.Cells का उपयोग करके PowerQuery सूत्रों तक पहुंचने और उन्हें प्रिंट करने के बारे में बताएगा।

### PowerQuery सूत्रों तक पहुँचना

#### अवलोकन
इस सुविधा में, हम यह पता लगाएंगे कि Excel कार्यपुस्तिका के डेटा मैशअप में एम्बेड किए गए PowerQuery सूत्रों को कैसे पढ़ा जाए।

#### कोड कार्यान्वयन
1. **कार्यपुस्तिका लोड करें**
   अपनी एक्सेल फ़ाइल को एक में लोड करके शुरू करें `Workbook` वस्तु:

   ```java
   Workbook workbook = new Workbook(dataDir + "/ODataSample.xlsx");
   ```

2. **PowerQuery फ़ॉर्मूला संग्रह तक पहुँचें**
   उपयोग `getDataMashup()` सूत्रों तक पहुंचने की विधि:

   ```java
   PowerQueryFormulaCollection PQFcoll = workbook.getDataMashup().getPowerQueryFormulas();
   ```

3. **सूत्रों पर पुनरावृत्ति करें**
   प्रत्येक सूत्र को लूप करें और उसका विवरण प्रिंट करें:

   ```java
   for (Object obj : PQFcoll) {
       PowerQueryFormula PQF = (PowerQueryFormula)obj;
       System.out.println("Connection Name: " + PQF.getName());
       
       PowerQueryFormulaItemCollection PQFIcoll = PQF.getPowerQueryFormulaItems();
       
       for (Object obj2 : PQFIcoll) {
           PowerQueryFormulaItem PQFI = (PowerQueryFormulaItem)obj2;
           System.out.println("Name: " + PQFI.getName());
           System.out.println("Value: " + PQFI.getValue());
       }
   }
   ```

### पैरामीटर्स और विधियों को समझना
- **`getName()`**: कनेक्शन या सूत्र आइटम का नाम पुनर्प्राप्त करता है.
- **`getValue()`**: PowerQuery फ़ॉर्मूला आइटम से संबद्ध मान लौटाता है.
