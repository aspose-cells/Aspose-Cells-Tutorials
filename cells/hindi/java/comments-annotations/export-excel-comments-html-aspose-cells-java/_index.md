## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में, हम आपकी फ़ाइलों को HTML के रूप में सहेजते समय Excel टिप्पणियों को निर्यात करने के बारे में बताएंगे।

### टिप्पणियाँ निर्यात करने का अवलोकन

इसका लक्ष्य यह सुनिश्चित करना है कि एक्सेल फ़ाइल में मौजूद सभी टिप्पणियाँ परिणामी HTML में शामिल हों। यह सुविधा ऑनलाइन डेटा देखने वाले उपयोगकर्ताओं के लिए स्पष्टता और संदर्भ को बढ़ा सकती है।

#### चरण 1: अपनी एक्सेल फ़ाइल लोड करें

सबसे पहले, उस एक्सेल वर्कबुक को लोड करें जिसे आप कनवर्ट करना चाहते हैं:

```java
import com.aspose.cells.Workbook;

// अपने स्रोत निर्देशिका पथ के साथ कार्यपुस्तिका आरंभ करें
String srcDir = "/path/to/your/source/";
Workbook wb = new Workbook(srcDir + "sampleExportCommentsHTML.xlsx");
```

#### चरण 2: HTML सहेजें विकल्प कॉन्फ़िगर करें

सेट करें `IsExportComments` संपत्ति को `true` में `HtmlSaveOptions`:

```java
import com.aspose.cells.HtmlSaveOptions;

// HtmlSaveOptions का एक उदाहरण बनाएं और निर्यात टिप्पणियाँ सेट करें
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.setExportComments(true);
```

#### चरण 3: HTML के रूप में सहेजें

अंत में, कॉन्फ़िगर किए गए विकल्पों के साथ अपनी कार्यपुस्तिका को HTML फ़ाइल के रूप में सहेजें:

```java
import java.io.IOException;

// HTML को सहेजने के लिए आउटपुट निर्देशिका पथ
String outDir = "/path/to/your/output/";

try {
    // एक्सेल फ़ाइल को टिप्पणियों सहित HTML प्रारूप में सहेजें
    wb.save(outDir + "outputExportCommentsHTML.html", opts);
} catch (IOException e) {
    System.out.println("Error occurred while saving the file.");
}
```

**समस्या निवारण सुझाव:** सुनिश्चित करें कि आपकी आउटपुट डायरेक्टरी लिखने योग्य है और उसमें पर्याप्त स्थान है।
