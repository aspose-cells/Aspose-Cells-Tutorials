## कार्यान्वयन मार्गदर्शिका
टिप्पणियों में पाठ दिशा परिवर्तन लागू करने के लिए इन चरणों का पालन करें।

### टिप्पणियाँ जोड़ना और संशोधित करना
#### अवलोकन
कक्षों में टिप्पणियाँ जोड़कर और उनके पाठ की दिशा को अनुकूलित करके दस्तावेज़ की स्पष्टता को बढ़ाएँ, यह बहुभाषी दस्तावेज़ों या विशिष्ट स्वरूपण की आवश्यकता वाली प्रणालियों के लिए विशेष रूप से उपयोगी है।

**चरण 1: कार्यपुस्तिका को इंस्टैंसिएट करें**
इसका एक उदाहरण बनाएं `Workbook` कक्षा:
```java
// नई कार्यपुस्तिका बनाएँ
Workbook wb = new Workbook();
```

**चरण 2: वर्कशीट तक पहुंचें**
उस वर्कशीट तक पहुंचें जहां आप टिप्पणियां जोड़ना चाहते हैं:
```java
// कार्यपुस्तिका में पहली कार्यपत्रिका प्राप्त करें
Worksheet sheet = wb.getWorksheets().get(0);
```

**चरण 3: एक टिप्पणी जोड़ें**
सेल "A1" में एक टिप्पणी जोड़ें:
```java
// सेल A1 में टिप्पणी जोड़ें
Comment comment = sheet.getComments().get(sheet.getComments().add("A1"));
```
टिप्पणी ऑब्जेक्ट को प्रथम सेल में जोड़कर पुनः प्राप्त करें।

**चरण 4: टेक्स्ट संरेखण और दिशा कॉन्फ़िगर करें**
पाठ संरेखण को अनुकूलित करें और दाएं से बाएं दिशा निर्धारित करें:
```java
// ऊर्ध्वाधर और क्षैतिज पाठ संरेखण सेट करें
comment.getCommentShape().setTextVerticalAlignment(TextAlignmentType.CENTER);
comment.getCommentShape().setTextHorizontalAlignment(TextAlignmentType.RIGHT);

// पाठ की दिशा को दाएं से बाएं बदलें
comment.getCommentShape().setTextDirection(TextDirectionType.RIGHT_TO_LEFT);
```
The `TextAlignmentType` और `TextDirectionType` enums आपकी टिप्पणियों के स्वरूप को कॉन्फ़िगर करने में सहायता करते हैं।

**चरण 5: टिप्पणी नोट सेट करें**
टिप्पणी की विषय-वस्तु परिभाषित करें:
```java
// टिप्पणी में एक नोट जोड़ें
comment.setNote("This is my Comment Text. This is test");
```

### अपना कार्य सहेजना
परिवर्तन करने के बाद, अद्यतन टिप्पणियों के साथ कार्यपुस्तिका को सहेजें:
```java
// आउटपुट निर्देशिका और फ़ाइल नाम परिभाषित करें
String dataDir = Utils.getSharedDataDir(ChangeTextDirectionofComment.class) + "TechnicalArticles/";

// संशोधित टिप्पणियों के साथ Excel फ़ाइल सहेजें
wb.save(dataDir + "CTDOfComment_out.xlsx");
```
