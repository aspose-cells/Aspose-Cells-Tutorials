# .NET के लिए Aspose.Cells का उपयोग करके एक्सेल शीट को एकल टेक्स्ट फ़ाइल में कैसे संयोजित करें

## परिचय

कई एक्सेल शीट में डेटा को मैनेज करना बोझिल हो सकता है, खासकर तब जब आपको विश्लेषण या रिपोर्टिंग के लिए उन्हें एक ही टेक्स्ट फ़ाइल में समेकित करना हो। यह ट्यूटोरियल दर्शाता है कि इसका उपयोग कैसे करें **.NET के लिए Aspose.Cells** एक्सेल वर्कबुक को लोड करने के लिए, प्रत्येक वर्कशीट को टैब-सेपरेटेड फॉर्मेट में परिवर्तित करें, और उन्हें एक व्यापक टेक्स्ट फ़ाइल में मर्ज करें।

इस गाइड में आप सीखेंगे:
- अपने .NET वातावरण में Aspose.Cells कैसे सेट करें।
- किसी निर्देशिका से कार्यपुस्तिका को आसानी से लोड करना।
- डेटा निर्यात के लिए पाठ सहेजने के विकल्प कॉन्फ़िगर करना.
- एकाधिक कार्यपत्रकों को एकल बाइट सरणी में संयोजित करना।
- संयुक्त डेटा को एकीकृत पाठ फ़ाइल के रूप में सहेजना.

आइये देखें कि आप इस प्रक्रिया को कैसे सरल बना सकते हैं!

## कार्यान्वयन मार्गदर्शिका

स्पष्टता के लिए आइये इस प्रक्रिया को अलग-अलग चरणों में विभाजित करें।

### कार्यपुस्तिका लोड करें

#### अवलोकन
किसी निर्दिष्ट निर्देशिका से Excel कार्यपुस्तिका लोड करें.

#### कार्यान्वयन चरण
1. **स्रोत निर्देशिका सेट करें**
   वह पथ निर्धारित करें जहां आपकी एक्सेल फ़ाइल स्थित है.
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   ```
2. **कार्यपुस्तिका लोड करें**
   एक नया बनाएँ `Workbook` अपनी एक्सेल फ़ाइल लोड करने के लिए ऑब्जेक्ट का उपयोग करें।
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "/book1.xls");
   ```

### पाठ सहेजें विकल्प आरंभ करें

#### अवलोकन
कॉन्फ़िगर करें कि टैब-पृथक मान (TSV) का उपयोग करके प्रत्येक कार्यपत्रक को पाठ प्रारूप में कैसे सहेजा जाएगा।

#### कार्यान्वयन चरण
1. **TxtSaveOptions बनाएं**
   इन्स्तांत करना `TxtSaveOptions` विभाजक निर्दिष्ट करने के लिए.
   ```csharp
   TxtSaveOptions opts = new TxtSaveOptions();
   opts.Separator = '\t'; // TSV प्रारूप के लिए विभाजक के रूप में टैब का उपयोग करें
   ```

### वर्कशीट को टेक्स्ट प्रारूप में परिवर्तित और संयोजित करें

#### अवलोकन
प्रत्येक वर्कशीट को टेक्स्ट प्रारूप में परिवर्तित करें और उन्हें एकल बाइट सरणी में संयोजित करें।

#### कार्यान्वयन चरण
1. **बाइट ऐरे आरंभ करें**
   सभी कार्यपत्रकों से संयुक्त डेटा रखने के लिए एक खाली बाइट सरणी तैयार करें।
   ```csharp
   byte[] workbookData = new byte[0];
   ```
2. **कार्यपत्रकों के माध्यम से पुनरावृति करें**
   प्रत्येक वर्कशीट पर लूप करें, उसे टेक्स्ट के रूप में सहेजें और आउटपुट को संयोजित करें।
   ```csharp
   for (int idx = 0; idx < workbook.Worksheets.Count; idx++) {
       workbook.Worksheets.ActiveSheetIndex = idx;
       
       using (MemoryStream ms = new MemoryStream()) {
           workbook.Save(ms, opts);
           
           ms.Position = 0;
           byte[] sheetData = ms.ToArray();
           
           byte[] combinedArray = new byte[workbookData.Length + sheetData.Length];
           Array.Copy(workbookData, 0, combinedArray, 0, workbookData.Length);
           Array.Copy(sheetData, 0, combinedArray, workbookData.Length, sheetData.Length);
           
           workbookData = combinedArray;
       }
   }
   ```

### संयुक्त कार्यपुस्तिका डेटा को फ़ाइल में सहेजें

#### अवलोकन
सभी कार्यपत्रकों से संयुक्त पाठ डेटा को एक एकल फ़ाइल में सहेजें।

#### कार्यान्वयन चरण
1. **आउटपुट निर्देशिका सेट करें**
   निर्धारित करें कि आपकी आउटपुट टेक्स्ट फ़ाइल कहाँ सहेजी जाएगी.
   ```csharp
   string OutputDir = "YOUR_OUTPUT_DIRECTORY";
   ```
2. **फ़ाइल में लिखें**
   उपयोग `File.WriteAllBytes` बाइट सरणी को एक के रूप में सहेजने के लिए `.txt` फ़ाइल।
   ```csharp
   File.WriteAllBytes(OutputDir + "/out.txt", workbookData);
   ```

यह विधि निम्नलिखित परिदृश्यों में उपयोगी है:
1. **डेटा समेकन**विभिन्न रिपोर्टों से डेटा को एक व्यापक दस्तावेज़ में संयोजित करें।
2. **रिपोर्टिंग स्वचालन**: आसान विश्लेषण और रिपोर्टिंग के लिए एकीकृत पाठ फ़ाइलें उत्पन्न करें।
3. **प्रवासन परियोजनाएं**: एक्सेल डेटा को अन्य प्रणालियों में स्थानांतरित करने की सुविधा प्रदान करना जो पाठ इनपुट स्वीकार करते हैं।
4. **सहयोगात्मक वर्कफ़्लो**जटिल स्प्रेडशीट को सरल, सर्वत्र सुलभ प्रारूप में परिवर्तित करके साझाकरण को सरल बनाना।
