# .NET (ODF 1.1 और 1.2) में Aspose.Cells का उपयोग करके ODS फ़ाइलें बनाएँ और सहेजें

## परिचय

आज की डेटा-संचालित दुनिया में, स्प्रेडशीट फ़ाइलों को प्रोग्रामेटिक रूप से बनाने और उनमें हेरफेर करने की क्षमता अमूल्य है। चाहे आप रिपोर्ट को स्वचालित कर रहे हों या बड़े डेटासेट को प्रोसेस कर रहे हों, एक विश्वसनीय टूल होने से समय की बचत हो सकती है और त्रुटियाँ कम हो सकती हैं। यह ट्यूटोरियल आपको ODF 1.2 और ODF 1.1 दोनों विनिर्देशों के साथ ODS फ़ाइलें बनाने और सहेजने के लिए Aspose.Cells for .NET का उपयोग करने के बारे में मार्गदर्शन करेगा।

**आप क्या सीखेंगे:**
- अपने विकास परिवेश में .NET के लिए Aspose.Cells सेट अप करना
- नई कार्यपुस्तिका बनाना और डेटा जोड़ना
- डिफ़ॉल्ट ODF 1.2 सेटिंग्स का उपयोग करके ODS फ़ाइल सहेजना
- ODF 1.1 अनुपालन के लिए सहेजने के विकल्प कॉन्फ़िगर करना

आइये शुरू करने से पहले कुछ पूर्वापेक्षाओं पर नजर डाल लें।

## कार्यान्वयन मार्गदर्शिका

हम इस गाइड को दो मुख्य विशेषताओं में विभाजित करेंगे: डिफ़ॉल्ट ODF 1.2 विनिर्देशों के साथ ODS फ़ाइलें बनाना और सहेजना, और ODF 1.1 अनुपालन को कॉन्फ़िगर करना।

### डिफ़ॉल्ट ODF 1.2 विनिर्देशों के साथ ODS फ़ाइल बनाएँ और सहेजें

#### अवलोकन

यह सुविधा आपको डिफ़ॉल्ट ODF 1.2 विनिर्देश सेटिंग्स के साथ Aspose.Cells का उपयोग करके एक सरल ODS फ़ाइल बनाने देती है।

#### चरण-दर-चरण कार्यान्वयन

##### चरण 1: निर्देशिका पथ सेट करें

अपनी स्रोत और आउटपुट निर्देशिकाएँ परिभाषित करें:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY"; // अपना स्रोत निर्देशिका पथ यहां सेट करें
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // अपना आउटपुट डायरेक्टरी पथ यहां सेट करें
```

##### चरण 2: नई कार्यपुस्तिका बनाएँ

एक नई कार्यपुस्तिका इंस्टैंस आरंभ करें:
```csharp
Workbook workbook = new Workbook();
```

##### चरण 3: वर्कशीट तक पहुंचें और उसे संशोधित करें

पहली वर्कशीट तक पहुँचें और सेल A1 में डेटा डालें:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Welcome to Aspose!");
```

##### चरण 4: सहेजें विकल्प कॉन्फ़िगर करें और फ़ाइल सहेजें

डिफ़ॉल्ट ODF 1.2 विनिर्देश के लिए ODS सहेजें विकल्प सेट करें और फ़ाइल सहेजें:
```csharp
OdsSaveOptions options = new OdsSaveOptions();
workbook.Save(outputDir + "/ODF1.2_out.ods", options);
```

### ODF 1.1 विनिर्देशों के साथ ODS फ़ाइल बनाएँ और सहेजें

#### अवलोकन

यह सुविधा दर्शाती है कि ODF 1.1 विनिर्देश का सख्ती से पालन करते हुए Aspose.Cells का उपयोग करके ODS फ़ाइल को कैसे सहेजा जाए।

#### चरण-दर-चरण कार्यान्वयन

##### चरण 1: निर्देशिका पथ सेट करें

सुनिश्चित करें कि आपकी स्रोत और आउटपुट निर्देशिकाएं सही ढंग से परिभाषित हैं:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY"; // अपना स्रोत निर्देशिका पथ यहां सेट करें
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // अपना आउटपुट डायरेक्टरी पथ यहां सेट करें
```

##### चरण 2: नई कार्यपुस्तिका बनाएँ

कार्यपुस्तिका इंस्टैंस को पहले की तरह आरंभ करें:
```csharp
Workbook workbook = new Workbook();
```

##### चरण 3: वर्कशीट तक पहुंचें और उसे संशोधित करें

वर्कशीट तक पहुंचें और सेल A1 में डेटा डालें:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Welcome to Aspose!");
```

##### चरण 4: ODF 1.1 के लिए सेव विकल्प कॉन्फ़िगर करें और फ़ाइल सेव करें

ODS सेव विकल्पों को सख्त ODF 1.1 अनुपालन के साथ सेट करें:
```csharp
OdsSaveOptions options = new OdsSaveOptions();
options.IsStrictSchema11 = true;
workbook.Save(outputDir + "/ODF1.1_out.ods", options);
```
