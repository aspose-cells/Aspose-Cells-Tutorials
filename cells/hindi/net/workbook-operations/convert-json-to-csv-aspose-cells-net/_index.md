## कार्यान्वयन मार्गदर्शिका

### रूपांतरण सुविधा का अवलोकन

Aspose.Cells का उपयोग करके JSON को CSV में बदलने में JSON फ़ाइल को पढ़ना और CSV के रूप में सहेजने से पहले उसके डेटा को Excel कार्यपुस्तिका में आयात करना शामिल है। यह प्रक्रिया सुनिश्चित करती है कि JSON की पदानुक्रमित संरचना एक सपाट, तालिका-जैसे प्रारूप में बनी रहे।

#### चरण 1: JSON फ़ाइल पढ़ें

```csharp
// स्रोत निर्देशिका जहाँ आपकी JSON फ़ाइल स्थित है
string sourceDir = RunExamples.Get_SourceDirectory();
string jsonFilePath = sourceDir + "SampleJson.json";

// JSON फ़ाइल की सामग्री पढ़ना
string jsonString = File.ReadAllText(jsonFilePath);
```

यहाँ, `File.ReadAllText` संपूर्ण JSON सामग्री को एक स्ट्रिंग में पढ़ता है। यह रूपांतरण की दिशा में हमारा पहला कदम है।

#### चरण 2: कार्यपुस्तिका बनाएं और कॉन्फ़िगर करें

```csharp
// रिक्त कार्यपुस्तिका आरंभ करें
Workbook workbook = new Workbook();

// प्रथम कार्यपत्रक के कक्ष संग्रह तक पहुँचना
Cells cells = workbook.Worksheets[0].Cells;

// आयात सेटिंग के लिए JsonLayoutOptions कॉन्फ़िगर करना
JsonLayoutOptions options = new JsonLayoutOptions
{
    ConvertNumericOrDate = true,
    ArrayAsTable = true,
    IgnoreArrayTitle = true,
    IgnoreObjectTitle = true
};
```

The `JsonLayoutOptions` क्लास रूपांतरण प्रक्रिया को अनुकूलित करने के लिए विभिन्न सेटिंग्स प्रदान करता है। उदाहरण के लिए, `ConvertNumericOrDate` यह सुनिश्चित करता है कि संख्यात्मक और दिनांक मानों की सही व्याख्या की गई है।

#### चरण 3: JSON डेटा आयात करें

```csharp
// पंक्ति 0, स्तंभ 0 से शुरू करके कार्यपुस्तिका कक्षों में JSON स्ट्रिंग से डेटा आयात करना
JsonUtility.ImportData(jsonString, cells, 0, 0, options);
```

`JsonUtility.ImportData` विधि प्रदान की गई कॉन्फ़िगरेशन का उपयोग करके JSON डेटा को निर्दिष्ट वर्कशीट और सेल श्रेणी में आयात करती है।

#### चरण 4: CSV के रूप में सहेजें

```csharp
// CSV फ़ाइल को सहेजने के लिए आउटपुट निर्देशिका निर्धारित करें
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "SampleJson_out.csv");
```

अंत में, अपनी कार्यपुस्तिका को CSV प्रारूप में सहेजें। `Save` विधि बहुमुखी है और CSV सहित विभिन्न प्रारूपों का समर्थन करती है।

### समस्या निवारण युक्तियों

- **फ़ाइल प्राप्त नहीं हुई**: सुनिश्चित करें कि आपकी JSON फ़ाइल का पथ सही है.
- **अनुमति संबंधी समस्याएं**: जाँच करें कि आपके अनुप्रयोग में सम्मिलित निर्देशिकाओं के लिए पढ़ने/लिखने की अनुमति है।
- **डेटा दूषण**: रूपांतरण से पहले अपने JSON डेटा की अखंडता को सत्यापित करें।
