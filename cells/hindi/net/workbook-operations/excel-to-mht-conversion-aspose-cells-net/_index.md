## कार्यान्वयन मार्गदर्शिका

### चरण 1: फ़ाइल पथ परिभाषित करें
अपनी स्रोत Excel फ़ाइल और आउटपुट MHT फ़ाइल के लिए पथ निर्दिष्ट करें।
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

string filePath = SourceDir + "/Book1.xlsx"; // एक्सेल फ़ाइल पथ इनपुट करें
string outputPath = outputDir + "/Book1.out.mht"; // आउटपुट MHT फ़ाइल पथ
```

### चरण 2: HTML सहेजें विकल्प कॉन्फ़िगर करें
अपनी Excel फ़ाइल को MHTML प्रारूप में परिवर्तित करने के लिए सहेजें विकल्प कॉन्फ़िगर करें।
```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.MHTML);
```
The `HtmlSaveOptions` क्लास HTML-आधारित प्रारूपों में कार्यपुस्तिकाओं को सहेजने के लिए कॉन्फ़िगरेशन प्रदान करता है। सेटिंग `SaveFormat.MHTML` सभी संसाधनों (छवियां, सीएसएस) को एक एकल फ़ाइल में संयोजित करता है।

### चरण 3: एक्सेल वर्कबुक लोड करें
पहले परिभाषित पथ का उपयोग करके अपनी Excel कार्यपुस्तिका लोड करें.
```csharp
Workbook workbook = new Workbook(filePath);
```
The `Workbook` Aspose.Cells में क्लास एक संपूर्ण Excel दस्तावेज़ का प्रतिनिधित्व करता है। इसे लोड करने से डेटा में हेरफेर करने की अनुमति मिलती है।

### चरण 4: MHT के रूप में सहेजें
कॉन्फ़िगर किए गए विकल्पों का उपयोग करके कार्यपुस्तिका को अपने इच्छित आउटपुट पथ पर सहेजें।
```csharp
workbook.save(outputPath, saveOptions);
```
यह चरण आपकी एक्सेल फ़ाइल को MHTML प्रारूप में परिवर्तित और सहेजता है, तथा वेब उपयोग के लिए इसके लेआउट और स्टाइलिंग को संरक्षित करता है।

### समस्या निवारण युक्तियों
- **फ़ाइल नहीं मिली त्रुटि:** सुनिश्चित करें कि आपके स्रोत निर्देशिका पथ सही हैं और फ़ाइलें मौजूद हैं।
- **लाइसेंस संबंधी मुद्दे:** लाइसेंस सेटअप को दोबारा जांचें। अनुपलब्ध या गलत लाइसेंस के कारण मूल्यांकन में बाधा आ सकती है।
