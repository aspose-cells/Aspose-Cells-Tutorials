## कार्यान्वयन मार्गदर्शिका

यह अनुभाग आपको एक्सेल वर्कशीट से दृश्यमान पंक्तियों को निर्यात करने के माध्यम से मार्गदर्शन करता है **.NET के लिए Aspose.Cells**.

### चरण 1: कार्यपुस्तिका लोड करें और कार्यपत्रक तक पहुँचें

अपनी एक्सेल वर्कबुक लोड करें और इसकी पहली वर्कशीट तक पहुँचें:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleExportVisibleRowsData.xlsx");
Worksheet worksheet = workbook.Worksheets[0]; // पहली वर्कशीट तक पहुँचें
```

### चरण 2: विकल्पों के साथ दृश्यमान पंक्तियाँ निर्यात करें

केवल दृश्यमान पंक्तियों को निर्यात करने और स्तंभ नामों को शामिल करने पर ध्यान दें:

```csharp
// केवल दृश्यमान पंक्तियों और शीर्षकों को शामिल करने के लिए निर्यात विकल्प सेट करें
ExportTableOptions exportOptions = new ExportTableOptions();
exportOptions.PlotVisibleRows = true; // केवल दृश्यमान पंक्तियाँ शामिल करें
exportOptions.ExportColumnName = true; // निर्यात में स्तंभ शीर्षलेख शामिल करें

// A1 से शुरू करते हुए, कक्षों की निर्दिष्ट श्रेणी निर्यात करें
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, 10, 4, exportOptions);
```
