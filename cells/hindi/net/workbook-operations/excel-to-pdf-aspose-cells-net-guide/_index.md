## कार्यान्वयन मार्गदर्शिका
हम कार्यपुस्तिका निर्माण और पीडीएफ रूपांतरण को व्यवधान प्रबंधन के साथ कवर करेंगे।

### एक्सेल वर्कबुक बनाना और पीडीएफ में परिवर्तित करना
यह सुविधा दिखाती है कि कार्यपुस्तिका कैसे बनाएं, उसमें पाठ जोड़कर उसे कैसे संशोधित करें, तथा उसे पीडीएफ में कैसे परिवर्तित करें।

#### चरण 1: घटकों को आरंभ करें
निर्देशिकाएँ सेट करें:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// रुकावटों से निपटने के लिए एक InterruptMonitor ऑब्जेक्ट बनाएँ
InterruptMonitor im = new InterruptMonitor();
```

#### चरण 2: कार्यपुस्तिका बनाएं और संशोधित करें
एक कार्यपुस्तिका इंस्टैंस बनाएं, इंटरप्टमॉनिटर असाइन करें, और एक सेल संशोधित करें:
```csharp
Workbook wb = new Workbook();
wb.InterruptMonitor = im;

Worksheet ws = wb.Worksheets[0];
Cell cell = ws.Cells["J1000000"];
cell.PutValue("This is text.");
```

#### चरण 3: पीडीएफ में कनवर्ट करें
कार्यपुस्तिका को PDF के रूप में सहेजने का प्रयास करें और व्यवधानों को संभालें:
```csharp
try {
    wb.Save(outputDir + "/output_InterruptMonitor.pdf");
} catch (Aspose.Cells.CellsException ex) {
    Console.WriteLine("Process Interrupted - Message: " + ex.Message);
}
```

### थ्रेड्स का उपयोग करके प्रक्रिया व्यवधानों का प्रबंधन करना
यह सुविधा थ्रेडिंग का उपयोग करके किसी प्रक्रिया को बाधित करने का प्रदर्शन करती है।

#### चरण 1: व्यवधान तर्क को परिभाषित करें
व्यवधान से पहले प्रतीक्षा करने की विधि बनाएं:
```csharp
void WaitForWhileAndThenInterrupt() {
    // 10 सेकंड के लिए सोएं (1000 मिलीसेकंड * 10)
    Thread.Sleep(1000 * 10);
    
    // 10 सेकंड के बाद प्रक्रिया को बाधित करें
    im.Interrupt();
}
```

#### चरण 2: थ्रेडिंग सेट अप करें
कार्यपुस्तिका निर्माण और व्यवधानों को प्रबंधित करने के लिए थ्रेड्स का उपयोग करें:
```csharp
InterruptMonitor im = new InterruptMonitor();

ThreadStart ts1 = new ThreadStart(() => {
    Workbook wb = new Workbook();
    wb.InterruptMonitor = im;
    Worksheet ws = wb.Worksheets[0];
    Cell cell = ws.Cells["J1000000"];
    cell.PutValue("This is text.");
    
    try {
        wb.Save(outputDir + "/output_InterruptMonitor.pdf");
    } catch (Aspose.Cells.CellsException ex) {
        Console.WriteLine("Process Interrupted - Message: " + ex.Message);
    }
});

ThreadStart ts2 = new ThreadStart(WaitForWhileAndThenInterrupt);

Thread t1 = new Thread(ts1);
Thread t2 = new Thread(ts2);
t1.Start();
t2.Start();
t1.Join();
t2.Join();
```
