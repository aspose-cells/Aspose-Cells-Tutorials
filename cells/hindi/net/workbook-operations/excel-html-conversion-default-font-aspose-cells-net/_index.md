# .NET के लिए Aspose.Cells के साथ Excel में डिफ़ॉल्ट फ़ॉन्ट सेटिंग को HTML में रूपांतरित करने में महारत हासिल करें

## परिचय

एक समान टाइपोग्राफी बनाए रखते हुए एक एक्सेल वर्कबुक को HTML फॉर्मेट में बदलना चुनौतीपूर्ण हो सकता है। यह ट्यूटोरियल आपको .NET के लिए Aspose.Cells का उपयोग करके एक डिफ़ॉल्ट फ़ॉन्ट सेट करने के माध्यम से मार्गदर्शन करता है, यह सुनिश्चित करता है कि आपके परिवर्तित दस्तावेज़ पॉलिश और पेशेवर दिखें। इस सुविधा में महारत हासिल करके, आप रूपांतरण प्रक्रिया में अज्ञात या अनुपलब्ध फ़ॉन्ट से संबंधित चुनौतियों पर काबू पा लेंगे।

**आप क्या सीखेंगे:**
- एक्सेल फ़ाइल को HTML में परिवर्तित करते समय डिफ़ॉल्ट फ़ॉन्ट कैसे सेट करें।
- .NET के लिए Aspose.Cells का उपयोग करने पर चरण-दर-चरण मार्गदर्शन।
- रेंडरिंग के दौरान अज्ञात फ़ॉन्ट्स को सुंदर ढंग से संभालने की तकनीकें।

आइए अपना परिवेश सेट अप करें और इस सुविधा का अन्वेषण शुरू करें!

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- **.NET वातावरण**: .NET का संगत संस्करण स्थापित (जैसे, .NET Core या .NET Framework).
- **.NET लाइब्रेरी के लिए Aspose.Cells**: NuGet के माध्यम से Aspose.Cells स्थापित करें।
- **बुनियादी C# ज्ञान**C# प्रोग्रामिंग अवधारणाओं से परिचित होना उपयोगी होगा।

## कार्यान्वयन मार्गदर्शिका

### रेंडरिंग करते समय डिफ़ॉल्ट फ़ॉन्ट सेट करना

यह सुविधा सुनिश्चित करती है कि HTML में कनवर्ट करते समय Excel कार्यपुस्तिका एक विशिष्ट डिफ़ॉल्ट फ़ॉन्ट के साथ रेंडर हो। यह उन मामलों को संभालने के लिए विशेष रूप से उपयोगी है जहाँ कुछ फ़ॉन्ट लक्ष्य सिस्टम पर उपलब्ध नहीं हो सकते हैं।

#### चरण 1: कार्यपुस्तिका बनाएं और उस तक पहुंचें

इसका एक नया उदाहरण बनाएं `Workbook` और इसकी पहली वर्कशीट तक पहुंचें:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// कार्यपुस्तिका ऑब्जेक्ट बनाएं और पहली कार्यपत्रक तक पहुंचें।
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```

#### चरण 2: सेल शैली संशोधित करें

किसी विशिष्ट सेल तक पहुंचें, टेक्स्ट जोड़ें, तथा प्रदर्शन के लिए फ़ॉन्ट को अज्ञात पर सेट करें:
```csharp
// सेल B4 तक पहुंचें और उसके अंदर कुछ पाठ जोड़ें।
Cell cell = ws.Cells["B4"];
cell.PutValue("This text has some unknown or invalid font which does not exist.");

// सेल B4 का फ़ॉन्ट अज्ञात फ़ॉन्ट पर सेट करें.
Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist";
st.Font.Size = 20;
cell.SetStyle(st);
```

#### चरण 3: HTML सेव विकल्प परिभाषित करें

अपने HTML आउटपुट में डिफ़ॉल्ट फ़ॉन्ट सेट करें। यहाँ, हम तीन अलग-अलग फ़ॉन्ट के साथ प्रदर्शन करते हैं:

**कूरियर नया:**
```csharp
// कार्यपुस्तिका को HTML प्रारूप में सहेजें, जिसमें डिफ़ॉल्ट फ़ॉन्ट कूरियर न्यू पर सेट हो।
HtmlSaveOptions optsCourierNew = new HtmlSaveOptions();
optsCourierNew.DefaultFontName = "Courier New";
wb.Save(outputDir + "/out_courier_new_out.htm", optsCourierNew);
```

**एरियल:**
```csharp
// कार्यपुस्तिका को HTML प्रारूप में सहेजें, डिफ़ॉल्ट फ़ॉन्ट को Arial पर सेट करें।
HtmlSaveOptions optsArial = new HtmlSaveOptions();
optsArial.DefaultFontName = "Arial";
wb.Save(outputDir + "/out_arial_out.htm", optsArial);
```

**टाइम्स न्यू रोमन:**
```csharp
// कार्यपुस्तिका को HTML प्रारूप में सहेजें, डिफ़ॉल्ट फ़ॉन्ट को टाइम्स न्यू रोमन पर सेट करें।
HtmlSaveOptions optsTimesNewRoman = new HtmlSaveOptions();
optsTimesNewRoman.DefaultFontName = "Times New Roman";
wb.Save(outputDir + "/times_new_roman_out.htm", optsTimesNewRoman);
```

### कार्यपुस्तिका निर्माण और सेल स्टाइलिंग

इस अनुभाग में कार्यपुस्तिका बनाना, कार्यपत्रकों, कक्षों तक पहुँचना और शैलियाँ लागू करना शामिल है:

#### चरण 1: कार्यपुस्तिका आरंभ करें
एक नया बनाएँ `Workbook` उदाहरण:
```csharp
// एक कार्यपुस्तिका ऑब्जेक्ट बनाएँ.
Workbook wb = new Workbook();
```

#### चरण 2: वर्कशीट और सेल तक पहुंचें
टेक्स्ट जोड़ने और उसे स्टाइल देने के लिए पहले वर्कशीट और सेल B4 तक पहुँचें:
```csharp
// कार्यपुस्तिका में प्रथम कार्यपत्रक तक पहुँचें.
Worksheet ws = wb.Worksheets[0];

// सेल B4 तक पहुंचें और उसके अंदर कुछ पाठ जोड़ें।
Cell cell = ws.Cells["B4"];
cell.PutValue("This text has some unknown or invalid font which does not exist.");

// सेल B4 का फ़ॉन्ट अज्ञात फ़ॉन्ट पर सेट करें.
Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist";
st.Font.Size = 20;
cell.SetStyle(st);
```

- **सुसंगत ब्रांडिंग**: सुनिश्चित करें कि निर्यात किए गए HTML दस्तावेज़ों में ब्रांड फ़ॉन्ट लगातार लागू किए गए हैं।
- **दस्तावेज़ पोर्टेबिलिटी**: ऐसे परिदृश्यों को संभालें जहां लक्ष्य वातावरण में विशिष्ट फ़ॉन्ट का अभाव हो।
- **स्वचालित रिपोर्टिंग**: सुसंगत टाइपोग्राफी के साथ स्वचालित रिपोर्ट तैयार करने के लिए इस सुविधा का उपयोग करें।
