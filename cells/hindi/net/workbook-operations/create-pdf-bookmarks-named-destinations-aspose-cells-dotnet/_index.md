**आप क्या सीखेंगे:**

## कार्यान्वयन मार्गदर्शिका

### नामित गंतव्यों के साथ पीडीएफ बुकमार्क्स का अवलोकन

पीडीएफ बुकमार्क दस्तावेजों के माध्यम से कुशल नेविगेशन की अनुमति देते हैं। इस गाइड में, हम नामित गंतव्य बनाएंगे जो पीडीएफ के रूप में निर्यात करते समय एक्सेल शीट में विशिष्ट कोशिकाओं से सीधे लिंक करते हैं। यह गतिशील रिपोर्ट और दस्तावेज़ बनाने के लिए विशेष रूप से उपयोगी है।

#### चरण-दर-चरण कार्यान्वयन

##### 1. अपनी कार्यपुस्तिका सेट करना

अपनी स्रोत एक्सेल फ़ाइल लोड करके प्रारंभ करें:

```csharp
// स्रोत एक्सेल फ़ाइल लोड करें
tWorkbook wb = new Workbook("samplePdfBookmarkEntry_DestinationName.xlsx");
```

##### 2. वर्कशीट और सेल्स तक पहुँचना

इच्छित वर्कशीट और कक्षों तक पहुँचें जहाँ आप बुकमार्क बनाना चाहते हैं।

```csharp
// पहली वर्कशीट तक पहुंचें
tWorksheet ws = wb.Worksheets[0];

// सेल C5 तक पहुंचें
tCell cellC5 = ws.Cells["C5"];
```

##### 3. बुकमार्क प्रविष्टियाँ बनाना

परिभाषित करें `PdfBookmarkEntry` पाठ और गंतव्य के साथ प्रत्येक बुकमार्क के लिए:

```csharp
// सेल C5 के लिए बुकमार्क और गंतव्य बनाएँ
tPdfBookmarkEntry bookmarkEntry = new PdfBookmarkEntry();
bookmarkEntry.Text = "Main Section";
bookmarkEntry.Destination = cellC5;
bookmarkEntry.DestinationName = "AsposeCells--" + cellC5.Name;

// इसी तरह अन्य कक्षों तक पहुंचें और उप-बुकमार्क बनाएं
tCell cellG56 = ws.Cells["G56"];
tPdfBookmarkEntry subbookmark1 = new PdfBookmarkEntry();
subbookmark1.Text = "Subsection 1";
subbookmark1.Destination = cellG56;
subbookmark1.DestinationName = "AsposeCells--" + cellG56.Name;

// आवश्यकतानुसार अतिरिक्त कोशिकाओं के लिए दोहराएँ
```

##### 4. उप-बुकमार्क व्यवस्थित करना

अपने उप-बुकमार्क को एक सूची में जोड़ें और उसे मुख्य बुकमार्क में असाइन करें:

```csharp
ArrayList list = new ArrayList { subbookmark1 /*, अन्य उपबुकमार्क यहां जोड़ें */ };
bookmarkEntry.SubEntry = list;
```

##### 5. पीडीएफ सेव विकल्प कॉन्फ़िगर करना

स्थापित करना `PdfSaveOptions` बुकमार्क शामिल करने और अपनी कार्यपुस्तिका को PDF के रूप में सहेजने के लिए:

```csharp
// PdfSaveOptions कॉन्फ़िगर करें
tPdfSaveOptions opts = new PdfSaveOptions();
opts.Bookmark = bookmarkEntry;

// कार्यपुस्तिका को बुकमार्क के साथ PDF प्रारूप में सहेजें
wb.Save("outputPdfBookmarkEntry_DestinationName.pdf", opts);
```
