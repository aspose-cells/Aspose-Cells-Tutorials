# .NET के लिए Aspose.Cells का उपयोग करके कस्टम फ़ॉन्ट के साथ Excel फ़ाइलों को PNG, TIFF और PDF में प्रस्तुत करें

## परिचय

एक्सेल फ़ाइलों को छवियों या पीडीएफ में बदलने के दौरान फ़ॉन्ट की अखंडता बनाए रखना ब्रांड की स्थिरता के लिए महत्वपूर्ण है। Aspose.Cells for .NET आपको अपने दस्तावेज़ रूपांतरणों में कस्टम डिफ़ॉल्ट फ़ॉन्ट निर्दिष्ट करने की अनुमति देकर एक मजबूत समाधान प्रदान करता है।

इस ट्यूटोरियल में, हम आपको निर्दिष्ट कस्टम डिफ़ॉल्ट फ़ॉन्ट के साथ .NET के लिए Aspose.Cells का उपयोग करके PNG, TIFF और PDF फ़ॉर्मेट में Excel फ़ाइलों को रेंडर करने के बारे में मार्गदर्शन करेंगे। यह आदर्श है यदि आप:
- प्रस्तुत दस्तावेजों में सुसंगत टाइपोग्राफी का लक्ष्य रखें।
- रूपांतरण के दौरान फ़ॉन्ट सेटिंग को अनुकूलित करने की आवश्यकता है।
- .NET के लिए Aspose.Cells के भीतर कॉन्फ़िगरेशन विकल्पों का पता लगाना चाहते हैं।

आइए अपना परिवेश तैयार करें और इन सुविधाओं को निर्बाध रूप से क्रियान्वित करें।

## कार्यान्वयन मार्गदर्शिका

### कस्टम डिफ़ॉल्ट फ़ॉन्ट के साथ PNG में रेंडरिंग

कस्टम डिफ़ॉल्ट फ़ॉन्ट सेट करते समय Excel वर्कशीट को PNG में रेंडर करना दृश्य स्थिरता सुनिश्चित करता है। यहाँ बताया गया है कि कैसे:

#### चरण 1: छवि विकल्प कॉन्फ़िगर करें

अपने छवि आउटपुट के लिए रेंडरिंग विकल्प कॉन्फ़िगर करें.
```csharp
using Aspose.Cells;
using Aspose.Cells.Rendering;

// निर्देशिकाएँ निर्दिष्ट करें.
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// एक एक्सेल फ़ाइल खोलें.
Workbook workbook = new Workbook(SourceDir + "sampleSetDefaultFontPropertyOfPdfSaveOptionsAndImageOrPrintOptions.xlsx");

// छवि रेंडरिंग विकल्प सेट करें.
ImageOrPrintOptions imgOpt = new ImageOrPrintOptions();
imgOpt.ImageType = Drawing.ImageType.Png;
imgOpt.CheckWorkbookDefaultFont = false; // कार्यपुस्तिका में लुप्त फ़ॉन्ट के लिए कस्टम फ़ॉन्ट का उपयोग करें।
imgOpt.DefaultFont = "Times New Roman";
```

#### चरण 2: रेंडर करें और सेव करें

इन सेटिंग्स का उपयोग करके अपनी वर्कशीट को एक छवि फ़ाइल में प्रस्तुत करें।
```csharp
// प्रथम वर्कशीट को PNG छवि में प्रस्तुत करें।
SheetRender sr = new SheetRender(workbook.Worksheets[0], imgOpt);
sr.ToImage(0, outputDir + "out1_imagePNG.png");
```

### कस्टम डिफ़ॉल्ट फ़ॉन्ट के साथ TIFF में रेंडरिंग

TIFF प्रारूप उच्च गुणवत्ता वाली छवियों के लिए आदर्श है। यहां बताया गया है कि आप संपूर्ण कार्यपुस्तिका को TIFF फ़ाइल के रूप में कैसे प्रस्तुत कर सकते हैं:

#### चरण 3: TIFF के लिए छवि विकल्प सेट करें

TIFF आउटपुट के लिए विशेष रूप से रेंडरिंग विकल्प कॉन्फ़िगर करें।
```csharp
// पहले से परिभाषित निर्देशिकाओं का पुनः उपयोग करें और एक्सेल फ़ाइल खोलें।
Workbook workbook = new Workbook(SourceDir + "sampleSetDefaultFontPropertyOfPdfSaveOptionsAndImageOrPrintOptions.xlsx");

// TIFF के लिए छवि रेंडरिंग विकल्प कॉन्फ़िगर करें.
imgOpt.ImageType = Drawing.ImageType.Tiff;
```

#### चरण 4: संपूर्ण कार्यपुस्तिका को TIFF में प्रस्तुत करें

संपूर्ण कार्यपुस्तिका को एकल TIFF फ़ाइल में परिवर्तित करें।
```csharp
// कार्यपुस्तिका को TIFF छवि के रूप में प्रस्तुत करें।
WorkbookRender wr = new WorkbookRender(workbook, imgOpt);
wr.ToImage(outputDir + "out1_imageTIFF.tiff");
```

### कस्टम डिफ़ॉल्ट फ़ॉन्ट के साथ PDF में रेंडर करना

व्यावसायिक दस्तावेज़ीकरण के लिए फ़ॉन्ट की एकरूपता सुनिश्चित करते हुए एक्सेल वर्कबुक को पीडीएफ के रूप में सहेजना महत्वपूर्ण है।

#### चरण 5: पीडीएफ सेव विकल्प कॉन्फ़िगर करें

अपनी फ़ाइल को PDF के रूप में सहेजने के लिए आवश्यक विकल्प सेट करें।
```csharp
using Aspose.Cells;

// कार्यपुस्तिका पुनः खोलें.
Workbook workbook = new Workbook(SourceDir + "sampleSetDefaultFontPropertyOfPdfSaveOptionsAndImageOrPrintOptions.xlsx");

// पीडीएफ सेव विकल्प सेट करें.
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.DefaultFont = "Times New Roman";
saveOptions.CheckWorkbookDefaultFont = false; // कार्यपुस्तिका में लुप्त फ़ॉन्ट के लिए कस्टम फ़ॉन्ट का उपयोग करें।
```

#### चरण 6: PDF के रूप में सहेजें

अपनी कार्यपुस्तिका को PDF दस्तावेज़ में निर्यात करें.
```csharp
// कार्यपुस्तिका को PDF फ़ाइल के रूप में सहेजें.
workbook.Save(outputDir + "out1_pdf.pdf", saveOptions);
```

## व्यावहारिक अनुप्रयोगों

- **व्यापार रिपोर्ट**: कस्टम फ़ॉन्ट का उपयोग करके सभी निर्यातित रिपोर्ट में सुसंगत ब्रांडिंग सुनिश्चित करें।
- **दस्तावेज़ संग्रहण**: समान टाइपोग्राफी के साथ आसान साझाकरण और संग्रह के लिए विरासत एक्सेल फ़ाइलों को पीडीएफ में परिवर्तित करें।
- **ग्राफ़िक डिज़ाइन**प्रस्तुतियों या डिज़ाइन परियोजनाओं के लिए Excel डेटा की उच्च-रिज़ॉल्यूशन TIFF छवियाँ बनाएँ।

अन्य प्रणालियों, जैसे कि CRM प्लेटफॉर्म या दस्तावेज़ प्रबंधन समाधान, के साथ एकीकरण, विशिष्ट ट्रिगर्स या घटनाओं के आधार पर निर्यात को स्वचालित करके इन उपयोग मामलों को और बढ़ा सकता है।

- **स्मृति प्रबंधन**: बचना `Workbook`, `SheetRender`, और `WorkbookRender` संसाधनों को मुक्त करने के लिए तुरंत कार्रवाई करें।
- **प्रचय संसाधन**यदि एकाधिक फाइलों पर काम करना हो तो कुशल संचालन के लिए बैच प्रोसेसिंग लागू करें।
- **अतुल्यकालिक संचालन**अनुप्रयोगों में प्रत्युत्तरशीलता में सुधार करने के लिए जहां संभव हो, अतुल्यकालिक विधियों का उपयोग करें।

**3. मैं रेंडरिंग के दौरान गायब फ़ॉन्ट्स को कैसे संभालूँ?**

**5. बड़े पैमाने के अनुप्रयोगों में Aspose.Cells का उपयोग करने के लिए कुछ सर्वोत्तम अभ्यास क्या हैं?**
