### बुनियादी आरंभीकरण और सेटअप

Aspose.Cells का एक उदाहरण बनाकर उसे आरंभ करें `Workbook` कक्षा।

## कार्यान्वयन मार्गदर्शिका

अपनी एक्सेल फ़ाइल बनाने और अनुकूलित करने के लिए इन चरणों का पालन करें:

### नई कार्यपुस्तिका बनाना
एक नई Excel कार्यपुस्तिका इस प्रकार बनाएं:
```csharp
// वर्कबुक (एक्सेल फ़ाइल) का एक उदाहरण बनाएँ
Workbook workbook = new Workbook();
```

### डिफ़ॉल्ट वर्कशीट हटाना
यदि इसकी आवश्यकता न हो तो डिफ़ॉल्ट वर्कशीट हटाएँ:
```csharp
// नई कार्यपुस्तिका के इंस्टैंसिएट होने पर बनाई गई डिफ़ॉल्ट कार्यपत्रक को हटाएँ
workbook.Worksheets.RemoveAt(0);
```

### एकाधिक शीट जोड़ना और नाम देना
अपनी कार्यपुस्तिका में पाँच कार्यपत्रक जोड़ें और उन्हें क्रमिक रूप से नाम दें।
```csharp
// 5 वर्कशीट जोड़ें और उन्हें नाम दें
for (int i = 0; i < 5; i++) {
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = "Sheet" + (i + 1).ToString();
}
```

### शीट में डेटा भरना
प्रत्येक वर्कशीट को ग्रिड में डेटा से भरें।
```csharp
// शीट को डेटा से भरें
for (int i = 0; i < workbook.Worksheets.Count; i++) {
    Worksheet ws = workbook.Worksheets[i];
    for (int row = 0; row < 150; row++) {
        for (int col = 0; col < 56; col++) {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

### कार्यपुस्तिका को सहेजना
अपनी कार्यपुस्तिका को निर्दिष्ट निर्देशिका में सहेजें.
```csharp
// कार्यपुस्तिका सहेजें
string outputFilePath = System.IO.Path.Combine(outputDir, "ACellsSample_out.xlsx");
workbook.Save(outputFilePath);
```
