# एक्सेल में कस्टम सॉर्ट सूची के साथ कॉलम में डेटा सॉर्ट करें

## परिचय

यह ट्यूटोरियल आपको अपनी परियोजना को सेट अप करने, एक्सेल फ़ाइल लोड करने और कस्टम सॉर्ट ऑर्डर का उपयोग करके निर्दिष्ट सीमा के भीतर डेटा सॉर्ट करने की प्रक्रिया के माध्यम से मार्गदर्शन करेगा। इस गाइड का पालन करके, आपको व्यावहारिक अनुभव प्राप्त होगा जो आपके डेटा प्रबंधन कौशल और Aspose.Cells लाइब्रेरी की उपयोगिता को बढ़ा सकता है।

## आवश्यक शर्तें

ट्यूटोरियल में आगे बढ़ने से पहले, आइए एक सुचारू शिक्षण अनुभव सुनिश्चित करने के लिए कुछ पूर्वापेक्षाओं की रूपरेखा तैयार करें।

### C# का बुनियादी ज्ञान

यद्यपि यह ट्यूटोरियल आपको प्रत्येक चरण में मार्गदर्शन करने के लिए डिज़ाइन किया गया है, C# की आधारभूत समझ होने से प्रस्तुत अवधारणाओं को समझना आसान हो जाएगा।

### .NET विकास वातावरण

सुनिश्चित करें कि आपके पास एक कार्यशील .NET विकास वातावरण सेट अप है। आप Visual Studio या किसी अन्य IDE का उपयोग कर सकते हैं जो .NET विकास का समर्थन करता है।

### .NET NuGet पैकेज के लिए Aspose.Cells

आपको अपने प्रोजेक्ट में .NET के लिए Aspose.Cells लाइब्रेरी इंस्टॉल करनी होगी। आप इसे NuGet पैकेज मैनेजर के ज़रिए आसानी से जोड़ सकते हैं। 

इसे करने का तरीका इस प्रकार है:

1. अपना प्रोजेक्ट Visual Studio में खोलें.
2. "टूल्स" > "NuGet पैकेज मैनेजर" > "समाधान के लिए NuGet पैकेज प्रबंधित करें" पर जाएं।
3.  निम्न को खोजें`Aspose.Cells` और नवीनतम संस्करण स्थापित करें.

### परीक्षण के लिए मूल एक्सेल फ़ाइल

आपको काम करने के लिए एक नमूना एक्सेल फ़ाइल की आवश्यकता होगी। आप यादृच्छिक देश के नाम और उनके कोड के साथ एक सरल एक्सेल फ़ाइल बना सकते हैं।

## पैकेज आयात करें

आरंभ करने के लिए, आइए अपने प्रोजेक्ट में आवश्यक पैकेज आयात करें। अपना कोड सेट अप करने का तरीका यहां दिया गया है:

```csharp
using System;
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

पैकेज आयातित होने के साथ, हम आगे बढ़ने के लिए तैयार हैं।

## चरण 1: स्रोत और आउटपुट निर्देशिकाएँ परिभाषित करें 

पहला चरण यह निर्धारित करना है कि आपकी इनपुट फ़ाइल कहाँ स्थित है और आप आउटपुट (सॉर्ट की गई फ़ाइल) को कहाँ सहेजना चाहते हैं। आपको दो पथ निर्दिष्ट करने होंगे: एक स्रोत एक्सेल फ़ाइल के लिए और दूसरा सॉर्टिंग के बाद आउटपुट को सहेजने के लिए।

```csharp
string sourceDir = "Your Document Directory\\";
string outputDir = "Your Document Directory\\";
```

## चरण 2: स्रोत एक्सेल फ़ाइल लोड करें

इसके बाद, हम एक्सेल फ़ाइल लोड करेंगे जिसमें वह डेटा होगा जिसे आप सॉर्ट करना चाहते हैं। यह एक इंस्टेंस बनाकर किया जाता है`Workbook` क्लास में जाकर अपने स्रोत फ़ाइल का पथ पास करें।

```csharp
Workbook wb = new Workbook(sourceDir + "sampleSortData_CustomSortList.xlsx");
```

## चरण 3: पहली वर्कशीट तक पहुँचें 

फ़ाइल लोड होने के बाद, हमें उस विशिष्ट वर्कशीट तक पहुँचने की आवश्यकता होती है जिसमें वह डेटा होता है जिसे हम सॉर्ट करना चाहते हैं। इस मामले में, हम पहली वर्कशीट को लक्षित कर रहे हैं।

```csharp
Worksheet ws = wb.Worksheets[0];
```

## चरण 4: सॉर्ट करने के लिए सेल क्षेत्र निर्दिष्ट करें

 हमें उन कोशिकाओं की श्रेणी निर्धारित करने की आवश्यकता है जिन्हें हम सॉर्ट करेंगे। इस उदाहरण में, हम A1 से A40 तक की कोशिकाओं को सॉर्ट करेंगे।`CellArea.CreateCellArea` सेल क्षेत्र को परिभाषित करने की विधि.

```csharp
CellArea ca = CellArea.CreateCellArea("A1", "A40");
```

## चरण 5: एक कस्टम सॉर्ट सूची बनाएँ

सॉर्टिंग से पहले, हमें अपने कस्टम सॉर्ट के लिए उपयोग किए जाने वाले मानदंड स्थापित करने की आवश्यकता है। आप सॉर्ट सूची को स्ट्रिंग्स की एक सरणी के रूप में परिभाषित कर सकते हैं। कस्टम सॉर्ट सूची सॉर्टिंग के क्रम को निर्धारित करेगी।

```csharp
string[] customSortList = new string[] { "USA,US", "Brazil,BR", "China,CN", "Russia,RU", "Canada,CA" };
```

## चरण 6: सॉर्ट कुंजी जोड़ें और सॉर्ट करें

अब सॉर्ट करने का समय है! हम इसके लिए DataSorter क्लास का उपयोग करेंगे। हमारी कस्टम सूची के आधार पर सॉर्टिंग के लिए एक कुंजी बनाएँ और सॉर्ट ऑपरेशन निष्पादित करें।

```csharp
wb.DataSorter.AddKey(0, SortOrder.Ascending, customSortList);
wb.DataSorter.Sort(ws.Cells, ca);
```

## चरण 7: आउटपुट एक्सेल फ़ाइल को सेव करें

सॉर्टिंग पूरी होने के बाद, अंतिम चरण परिवर्तनों को एक नई एक्सेल फ़ाइल में सहेजना है। आउटपुट फ़ाइल नाम निर्दिष्ट करें और कार्यपुस्तिका को सहेजें।

```csharp
wb.Save(outputDir + "outputSortData_CustomSortList.xlsx");
```

## चरण 8: सफल निष्पादन की पुष्टि करें

यह सुनिश्चित करने के लिए कि सब कुछ सुचारू रूप से काम कर रहा है, आप कंसोल पर एक पुष्टिकरण संदेश प्रिंट कर सकते हैं। यह डिबगिंग में मदद करता है और आपको संतुष्टि देता है कि ऑपरेशन सफल रहा।

```csharp
Console.WriteLine("SortDataInColumnWithCustomSortList executed successfully.\r\n");
```

## निष्कर्ष

और अब आप इसे प्राप्त कर चुके हैं! आपने .NET के लिए Aspose.Cells के साथ कस्टम सॉर्ट सूची का उपयोग करके Excel कॉलम में डेटा को सफलतापूर्वक सॉर्ट किया है। सॉर्टिंग आपके डेटा में संरचना और स्पष्टता लाने में मदद करता है, जिससे इसका विश्लेषण और व्याख्या करना आसान हो जाता है। मुझे उम्मीद है कि यह मार्गदर्शिका आपके कौशल को अगले स्तर तक ले जाएगी और आपको यह समझने में मदद करेगी कि आपके Excel-संबंधित कार्यों के लिए Aspose.Cells कितना शक्तिशाली हो सकता है।

हाँ! यदि आवश्यक हो तो आप कई कॉलम के आधार पर छाँटने के लिए अतिरिक्त कुंजियाँ जोड़ सकते हैं, बस प्रत्येक कुंजी के लिए समान प्रक्रिया का पालन करें।

### क्या मुझे Aspose.Cells का उपयोग करने के लिए C# का पूर्व ज्ञान होना आवश्यक है?
हालाँकि यह मददगार है, आप इस ट्यूटोरियल का अनुसरण कर सकते हैं और आगे बढ़ते हुए सीख सकते हैं! C# की कुछ बुनियादी समझ होने से आपका सीखने का अनुभव बेहतर होगा।

बिल्कुल! यदि आप बिना किसी प्रतिबंध के लाइब्रेरी की सभी सुविधाओं का परीक्षण करना चाहते हैं तो आप एक अस्थायी लाइसेंस प्राप्त कर सकते हैं।
