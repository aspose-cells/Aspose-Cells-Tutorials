# .NET के लिए Aspose.Cells के साथ Excel में कस्टम उप-योग कैसे लागू करें

## परिचय

क्या आप अपनी एक्सेल फ़ाइलों में विशिष्ट उप-योग लेबल के साथ अनुकूलित रिपोर्ट बनाना चाहते हैं? यह मार्गदर्शिका आपको बताएगी कि .NET के लिए शक्तिशाली Aspose.Cells लाइब्रेरी का उपयोग करके इसे कैसे प्राप्त किया जाए। हम आपकी ज़रूरतों के अनुरूप औसत उप-योग बनाने पर ध्यान केंद्रित करेंगे।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Cells को सेट अप करना और उसका उपयोग करना

- एक्सेल शीट में कस्टम उप-योग जोड़ना
- सूत्रों की गणना करना और स्तंभ की चौड़ाई को स्वचालित रूप से समायोजित करना

## कार्यान्वयन मार्गदर्शिका

हम इस कार्यान्वयन को चरणों में विभाजित करेंगे ताकि आपको प्रक्रिया के प्रत्येक भाग को समझने में मदद मिल सके।

### चरण 1: कस्टम सेटिंग क्लास बनाएँ
सबसे पहले, एक कस्टम क्लास बनाएं जो विस्तारित हो `GlobalizationSettings`:
```csharp
class CustomSettings : GlobalizationSettings
{
    public override string GetTotalName(ConsolidationFunction functionType)
    {
        switch (functionType)
        {
            case ConsolidationFunction.Average:
                return "AVG";
            default:
                return base.GetTotalName(functionType);
        }
    }

    public override string GetGrandTotalName(ConsolidationFunction functionType)
    {
        switch (functionType)
        {
            case ConsolidationFunction.Average:
                return "GRD AVG";
            default:
                return base.GetGrandTotalName(functionType);
        }
    }
}
```
**स्पष्टीकरण:** यह वर्ग विभिन्न कार्यों के लिए उप-योगों के नामकरण को अनुकूलित करता है, जैसे औसत।

### चरण 2: अपनी कार्यपुस्तिका लोड करें
अपनी मौजूदा Excel कार्यपुस्तिका लोड करें जिसमें वह डेटा हो जिसे आप संशोधित करना चाहते हैं:
```csharp
Workbook book = new Workbook("sampleCustomLabelsSubtotals.xlsx");
```
**स्पष्टीकरण:** प्रतिस्थापित करें `"sampleCustomLabelsSubtotals.xlsx"` अपने फ़ाइल पथ के साथ। यह आरंभ करता है `Workbook` वस्तु।

### चरण 3: कस्टम ग्लोबलाइज़ेशन सेटिंग्स सेट करें
कार्यपुस्तिका को हमारी कस्टम सेटिंग्स असाइन करें:
```csharp
book.Settings.GlobalizationSettings = new CustomSettings();
```
**स्पष्टीकरण:** यह सुनिश्चित करता है कि कोई भी उप-योग गणना हमारे अनुकूलित लेबल का उपयोग करती है `CustomSettings`.

### चरण 4: उप-योग कार्यक्षमता जोड़ें
औसत फ़ंक्शन का उपयोग करके निर्दिष्ट सीमा के भीतर अपने वर्कशीट में उप-योग जोड़ें:
```csharp
Worksheet sheet = book.Worksheets[0];
sheet.Cells.Subtotal(CellArea.CreateCellArea("A2", "B9"), 0, ConsolidationFunction.Average, new int[] { 1 });
```
**स्पष्टीकरण:** यह A2 से B9 तक की कोशिकाओं को लक्षित करता है और पहले कॉलम (सूचकांक 1) के आधार पर औसत उप-योग जोड़ता है।

### चरण 5: सूत्रों की गणना करें और कॉलम समायोजित करें
उप-योग जोड़ने के बाद, किसी भी सूत्र की गणना करें और कॉलम को स्वचालित रूप से फिट करें:
```csharp
book.CalculateFormula();
sheet.AutoFitColumns();
```
**स्पष्टीकरण:** `CalculateFormula()` यह सुनिश्चित करता है कि सभी गणनाएं अद्यतन हैं। `AutoFitColumns()` सामग्री को फिट करने के लिए कॉलम की चौड़ाई को समायोजित करता है.

### चरण 6: अपनी कार्यपुस्तिका सहेजें
अपने परिवर्तनों को नई फ़ाइल में सहेजें:
```csharp
book.Save("outputCustomLabelsSubtotals.xlsx");
```
**स्पष्टीकरण:** यह आपकी संशोधित कार्यपुस्तिका को कस्टम उप-योग और समायोजित स्तंभों के साथ सहेजता है।

## व्यावहारिक अनुप्रयोगों
यहां कुछ वास्तविक दुनिया परिदृश्य दिए गए हैं जहां कस्टम उप-योग अमूल्य हो सकते हैं:
1. **वित्तीय रिपोर्टिंग**"शुद्ध औसत" या "कुल समायोजित राजस्व" जैसे विशिष्ट वित्तीय शब्दों को प्रतिबिंबित करने के लिए उप-योग लेबल को अनुकूलित करें।
2. **सूची प्रबंधन**: अपनी इन्वेंट्री रिपोर्ट में विभिन्न श्रेणियों या आपूर्तिकर्ताओं के लिए अनुकूलित उप-योग का उपयोग करें।
3. **बिक्री डेटा विश्लेषण**: औसत गणना लागू करें जो नई बिक्री डेटा प्रविष्टियों के साथ स्वचालित रूप से अपडेट हो जाती है।
4. **शैक्षिक ग्रेडिंग प्रणालियाँ**: विभिन्न विषयों में छात्रों के औसत अंकों को दर्शाने के लिए लेबल को अनुकूलित करें।
5. **बिजनेस इंटेलिजेंस डैशबोर्ड**बेहतर स्पष्टता के लिए विशिष्ट KPI या मीट्रिक से मिलान करने के लिए उप-योग लेबल तैयार करें।
