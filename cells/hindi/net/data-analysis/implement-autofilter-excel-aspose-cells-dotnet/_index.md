# .NET के लिए Aspose.Cells का उपयोग करके Excel में ऑटोफ़िल्टर कैसे लागू करें

## परिचय

क्या आप एक्सेल फ़ाइलों में पंक्तियों को प्रोग्रामेटिक रूप से फ़िल्टर करके डेटा विश्लेषण को सुव्यवस्थित करना चाहते हैं? शक्तिशाली **.NET के लिए Aspose.Cells** लाइब्रेरी में, आप आसानी से वर्कबुक में बदलाव कर सकते हैं और ऑटो फ़िल्टर लगा सकते हैं। यह ट्यूटोरियल आपको अपना वातावरण सेट करने, वर्कबुक को आरंभ करने, वर्कशीट तक पहुँचने, कस्टम ऑटो फ़िल्टर बनाने और परिवर्तनों को सहेजने के लिए उन्हें रिफ्रेश करने में मार्गदर्शन करेगा।

### आप क्या सीखेंगे:
- .NET के लिए Aspose.Cells कैसे स्थापित करें
- Excel फ़ाइल से वर्कबुक ऑब्जेक्ट को आरंभ करना
- किसी कार्यपुस्तिका में विशिष्ट कार्यपत्रकों तक पहुँचना
- कस्टम ऑटो फ़िल्टर लागू करना और लागू करना
- फ़िल्टर ताज़ा करना और अद्यतन कार्यपुस्तिका सहेजना

इससे पहले कि हम आगे बढ़ें, आइए सुनिश्चित करें कि आपके पास वह सब कुछ है जो आपको चाहिए।

## .NET के लिए Aspose.Cells सेट अप करना

### इंस्टालेशन

आप अपने प्रोजेक्ट में Aspose.Cells पैकेज को निम्न में से किसी एक का उपयोग करके जोड़ सकते हैं **नुगेट पैकेज मैनेजर** या **.NET सीएलआई**:

**.NET सीएलआई**
```bash
dotnet add package Aspose.Cells
```

**पैकेज प्रबंधक कंसोल**
```powershell
PM> NuGet\Install-Package Aspose.Cells
```

- **खरीदना**दीर्घकालिक उपयोग के लिए, लाइसेंस खरीदने पर विचार करें।

### मूल आरंभीकरण

एक बार इंस्टॉल हो जाने पर, इसका एक उदाहरण बनाकर शुरू करें `Workbook` क्लास पर जाएँ और अपनी एक्सेल फ़ाइल लोड करें:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";

// निर्दिष्ट स्रोत निर्देशिका से नमूना डेटा के साथ कार्यपुस्तिका लोड करें
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
```

## कार्यान्वयन मार्गदर्शिका

### 1. कार्यपुस्तिका आरंभीकरण और खोलना

#### अवलोकन
यह अनुभाग बताता है कि एक्सेल फ़ाइल को कैसे लोड किया जाए `Workbook` Aspose.Cells का उपयोग करके ऑब्जेक्ट.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// निर्दिष्ट स्रोत निर्देशिका से नमूना डेटा के साथ कार्यपुस्तिका लोड करें
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
```

**स्पष्टीकरण**: द `Workbook` क्लास एक संपूर्ण एक्सेल फ़ाइल का प्रतिनिधित्व करता है। पथ निर्दिष्ट करके, आप हेरफेर के लिए मौजूदा फ़ाइलों को लोड कर सकते हैं।

### 2. वर्कबुक में वर्कशीट तक पहुँचना

#### अवलोकन
फ़िल्टरिंग जैसे विशिष्ट ऑपरेशन लागू करने के लिए अपनी कार्यपुस्तिका के भीतर अलग-अलग कार्यपत्रकों तक पहुँचें।

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// स्रोत निर्देशिका से कार्यपुस्तिका लोड करें
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");

// इंडेक्स द्वारा पहली वर्कशीट तक पहुँचें
Worksheet worksheet = workbook.Worksheets[0];
```

**स्पष्टीकरण**: द `Worksheets` संग्रह आपको प्रत्येक शीट तक पहुंचने की अनुमति देता है। इंडेक्स 0 पहली वर्कशीट से मेल खाता है।

### 3. ऑटोफ़िल्टर बनाना और लागू करना

#### अवलोकन
प्रासंगिक डेटा दिखाने के लिए कस्टम मानदंड लागू करते हुए, कक्षों की निर्दिष्ट श्रेणी के लिए स्वचालित फ़िल्टर सेट करें.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// कार्यपुस्तिका लोड करें और पहली कार्यपत्रक तक पहुँचें
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
Worksheet worksheet = workbook.Worksheets[0];

// स्वचालित फ़िल्टर के लिए सीमा निर्धारित करें (उदाहरणार्थ, A1:A18)
worksheet.AutoFilter.Range = "A1:A18";

// 'Ba' से शुरू होने वाले मान वाली पंक्तियाँ दिखाने के लिए कस्टम फ़िल्टर लागू करें
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");
```

**स्पष्टीकरण**: द `AutoFilter` प्रॉपर्टी रेंज को परिभाषित करने और फ़िल्टर लागू करने की अनुमति देती है। शर्तों को निर्दिष्ट करने के लिए कस्टम विधियों का उपयोग किया जा सकता है।

### 4. कार्यपुस्तिका को ताज़ा करना और सहेजना

#### अवलोकन
परिवर्तन लागू करने और कार्यपुस्तिका को नए फ़ाइल स्थान पर सहेजने के लिए अपने फ़िल्टर को ताज़ा करें.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// कार्यपुस्तिका लोड करें, कार्यपत्रक तक पहुंचें, और स्वचालित फ़िल्टर सेट करें
Workbook workbook = new Workbook(SourceDir + "/sourceSampleCountryNames.xlsx");
Worksheet worksheet = workbook.Worksheets[0];
worksheet.AutoFilter.Range = "A1:A18";
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");

// परिवर्तन लागू करने के लिए ऑटो फ़िल्टर को रीफ़्रेश करें
worksheet.AutoFilter.Refresh();

// अद्यतन की गई कार्यपुस्तिका को निर्दिष्ट आउटपुट निर्देशिका में सहेजें
workbook.Save(outputDir + "/outSourceSampleCountryNames.xlsx");
```

**स्पष्टीकरण**: फ़िल्टर लगाने के बाद, उपयोग करें `Refresh()` वर्कशीट को अपडेट करने के लिए। अंत में, अपने परिवर्तनों को सेव करें `Save()` तरीका।

## व्यावहारिक अनुप्रयोगों

1. **डेटा रिपोर्टिंग**: केवल विशिष्ट देशों या क्षेत्रों को शामिल करने वाली रिपोर्ट के लिए डेटा को स्वचालित रूप से फ़िल्टर करें.
2. **सूची प्रबंधन**: विशेष अक्षरों से शुरू होने वाले आइटम नामों या श्रेणियों के आधार पर इन्वेंट्री सूचियों को फ़िल्टर करें।
3. **वित्तीय विश्लेषण**: कुछ निश्चित मानदंडों को पूरा करने वाले वित्तीय रिकॉर्डों पर ध्यान केंद्रित करने के लिए स्वचालित फ़िल्टर का उपयोग करें, जैसे कि किसी विशिष्ट विक्रेता के नाम से शुरू होने वाले लेनदेन।
