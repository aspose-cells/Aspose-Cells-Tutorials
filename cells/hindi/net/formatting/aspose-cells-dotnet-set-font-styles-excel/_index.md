एक नई कार्यपुस्तिका बनाकर और एक कार्यपत्रक जोड़कर आरंभ करें:

```csharp
// एक नई वर्कबुक ऑब्जेक्ट को इन्स्टेन्सिएट करें.
```

### सेल शैलियों तक पहुँचना और उन्हें संशोधित करना

इस ट्यूटोरियल का मुख्य उद्देश्य फ़ॉन्ट शैली में बदलाव करना है। यहाँ बताया गया है कि कैसे:

#### फ़ॉन्ट वेट को बोल्ड पर सेट करना

पाठ को बोल्ड बनाने के लिए, इच्छित सेल के स्टाइल ऑब्जेक्ट तक पहुँचें:

```csharp
// सेल "A1" पर पहुँचें.
Aspose.Cells.Cell cell = worksheet.Cells["A1"];

// सेल में मान जोड़ें.
cell.PutValue("Hello Aspose!");

// सेल से संबद्ध स्टाइल ऑब्जेक्ट प्राप्त करें.
Style style = cell.GetStyle();

// फ़ॉन्ट वज़न को बोल्ड पर सेट करें.
style.Font.IsBold = true;

// शैली को वापस सेल पर लागू करें.
cell.SetStyle(style);
```

#### कोड का स्पष्टीकरण
- **गेटस्टाइल()**: किसी सेल की वर्तमान शैली सेटिंग्स को पुनर्प्राप्त करता है।
- **फ़ॉन्ट.इज़बोल्ड**: वह प्रॉपर्टी जो टेक्स्ट के बोल्डनेस को नियंत्रित करती है। इसे इस पर सेट करना `true` बोल्ड फ़ॉर्मेटिंग लागू करता है.
