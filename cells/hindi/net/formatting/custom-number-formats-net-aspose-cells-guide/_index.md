## कार्यान्वयन मार्गदर्शिका

हम Aspose.Cells का उपयोग करके संख्या प्रारूपों को अनुकूलित करने के लिए प्रमुख विशेषताओं को कवर करेंगे।

### कस्टम दिनांक प्रारूप जोड़ना
**अवलोकन**: कस्टम शैली के साथ एक्सेल कक्षों में दिनांकों को प्रारूपित करना सीखें।
1. **वर्कशीट बनाएं या एक्सेस करें**
   ```csharp
   int sheetIndex = workbook.Worksheets.Add();
   Worksheet worksheet = workbook.Worksheets[sheetIndex];
   ```
2. **कस्टम प्रारूप के साथ वर्तमान सिस्टम दिनांक सेट करें**
   सेल "A1" में वर्तमान दिनांक जोड़ें और एक कस्टम प्रदर्शन प्रारूप लागू करें।
   ```csharp
   // A1 में वर्तमान सिस्टम दिनांक डालें
   worksheet.Cells["A1"].PutValue(DateTime.Now);

   // अनुकूलन के लिए शैली ऑब्जेक्ट पुनः प्राप्त करें
   Style style = worksheet.Cells["A1"].GetStyle();

   // कस्टम संख्या प्रारूप को "d-mmm-yy" पर सेट करें
   style.Custom = "d-mmm-yy";

   // अनुकूलित शैली को वापस सेल A1 पर लागू करें
   worksheet.Cells["A1"].SetStyle(style);
   ```

### संख्यात्मक मानों को प्रतिशत के रूप में स्वरूपित करना
**अवलोकन**: संख्यात्मक मानों को प्रतिशत प्रारूप में प्रदर्शित करें.
1. **मान डालें और प्रारूपित करें**
   ```csharp
   // सेल A2 में संख्यात्मक मान जोड़ें
   worksheet.Cells["A2"].PutValue(20);

   // स्वरूपण के लिए शैली प्राप्त करें
   Style style = worksheet.Cells["A2"].GetStyle();

   // कस्टम संख्या प्रारूप को प्रतिशत के रूप में लागू करें
   style.Custom = "0.0%";

   // स्वरूपित शैली को वापस सेल A2 पर सेट करें
   worksheet.Cells["A2"].SetStyle(style);
   ```

### मुद्रा प्रारूप लागू करना
**अवलोकन**: संख्याओं को मुद्रा प्रारूप में दिखाएं, ऋणात्मक मानों के लिए विशिष्ट स्वरूपण के साथ।
1. **मुद्रा मान डालें और शैली दें**
   ```csharp
   // सेल A3 में मान जोड़ें
   worksheet.Cells["A3"].PutValue(2546);

   // स्टाइल ऑब्जेक्ट तक पहुंचें
   Style style = worksheet.Cells["A3"].GetStyle();

   // कस्टम मुद्रा प्रारूप सेट करें
   style.Custom = "\u00a3#,##0;[Red]$-#,##0";

   // सेल A3 पर लागू करें
   worksheet.Cells["A3"].SetStyle(style);
   ```

## व्यावहारिक अनुप्रयोगों

कस्टम नंबर फ़ॉर्मेटिंग निम्नलिखित परिदृश्यों में अमूल्य है:
1. **वित्तीय रिपोर्ट**स्पष्टता के लिए मुद्रा मानों को प्रारूपित करना.
2. **बिक्री डैशबोर्ड**: प्रदर्शन मीट्रिक्स को उजागर करने के लिए बिक्री के आंकड़ों को प्रतिशत के रूप में प्रदर्शित करना।
3. **ईवेंट की योजना बनाना**: ईवेंट शेड्यूल को व्यवस्थित और प्रस्तुत करने के लिए दिनांक प्रारूपों का उपयोग करना।
