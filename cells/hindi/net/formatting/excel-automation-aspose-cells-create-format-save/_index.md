**आप क्या सीखेंगे:**

## कार्यान्वयन मार्गदर्शिका

### कार्यपुस्तिका बनाएँ और कार्यपत्रक तक पहुँचें

**अवलोकन:** यह सुविधा एक नई एक्सेल वर्कबुक बनाने और उसकी पहली वर्कशीट तक पहुंचने का प्रदर्शन करती है।

#### चरण 1: कार्यपुस्तिका आरंभ करें और कार्यपत्रक तक पहुँचें
आरंभ करके शुरू करें `Workbook` ऑब्जेक्ट और उसके डिफ़ॉल्ट वर्कशीट तक पहुँचें.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

### कक्षों में डेटा जोड़ें

**अवलोकन:** जानें कि वर्कशीट में विशिष्ट कक्षों में डेटा कैसे भरें।

#### चरण 2: वर्कशीट कक्षों को भरें
वर्कशीट में कुछ कॉलम में मान जोड़ने के लिए लूप का उपयोग करें।
```csharp
for (int i = 2; i <= 15; i++)
{
    worksheet.Cells["A" + i].PutValue(i);
    worksheet.Cells["D" + i].PutValue(i);
}
```
यह स्निपेट सेल A2 से A15 और D2 से D15 तक क्रमिक संख्याएँ रखता है।

### दो-रंग स्केल सशर्त स्वरूपण जोड़ें

**अवलोकन:** श्रेणी A2:A15 में डेटा विविधताओं को दृश्य रूप से दर्शाने के लिए दो-रंग स्केल सशर्त स्वरूपण लागू करें।

#### चरण 3: सेल क्षेत्र निर्धारित करें
सशर्त स्वरूपण लागू करने के लिए कक्ष क्षेत्र निर्दिष्ट करें.
```csharp
CellArea ca = CellArea.CreateCellArea("A2", "A15");
```

#### चरण 4: फ़ॉर्मेटिंग नियम जोड़ें
दो-रंग स्केल प्रारूप शर्त जोड़ें और कॉन्फ़िगर करें.
```csharp
int idx = worksheet.ConditionalFormattings.Add();
FormatConditionCollection fcc = worksheet.ConditionalFormattings[idx];
fcc.AddCondition(FormatConditionType.ColorScale);
fcc.AddArea(ca);

FormatCondition fc = worksheet.ConditionalFormattings[idx][0];
fc.ColorScale.Is3ColorScale = false;
fc.ColorScale.MaxColor = Color.LightBlue;
fc.ColorScale.MinColor = Color.LightGreen;
```

### तीन-रंग स्केल सशर्त स्वरूपण जोड़ें

**अवलोकन:** श्रेणी D2:D15 के लिए तीन-रंग स्केल सशर्त स्वरूपण के साथ डेटा विज़ुअलाइज़ेशन को बढ़ाएं।

#### चरण 5: अन्य सेल क्षेत्र निर्धारित करें
तीन-रंग पैमाने के लिए एक और सेल क्षेत्र स्थापित करें।
```csharp
CellArea ca = CellArea.CreateCellArea("D2", "D15");
```

#### चरण 6: तीन-रंग स्केल फ़ॉर्मेटिंग नियम जोड़ें
तीन-रंग सशर्त स्वरूपण नियम कॉन्फ़िगर करें.
```csharp
int idx = worksheet.ConditionalFormattings.Add();
FormatConditionCollection fcc = worksheet.ConditionalFormattings[idx];
fcc.AddCondition(FormatConditionType.ColorScale);
fcc.AddArea(ca);

FormatCondition fc = worksheet.ConditionalFormattings[idx][0];
fc.ColorScale.Is3ColorScale = true;
fc.ColorScale.MaxColor = Color.LightBlue;
fc.ColorScale.MidColor = Color.Yellow;
fc.ColorScale.MinColor = Color.LightGreen;
```

### कार्यपुस्तिका सहेजें

**अवलोकन:** परिवर्तन लागू करने के बाद, कार्यपुस्तिका को निर्दिष्ट स्थान पर सहेजें.

#### चरण 7: संशोधित कार्यपुस्तिका सहेजें
अंत में, का उपयोग करें `Save` अपने संशोधनों को बनाए रखने की विधि.
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "output_out.xlsx");
```
