## कार्यान्वयन मार्गदर्शिका

### दशमलव और समूह विभाजकों को अनुकूलित करना

**अवलोकन:**
दशमलव और समूह विभाजकों को अनुकूलित करने से डेटा की पठनीयता बढ़ जाती है और विभिन्न क्षेत्रों या व्यवसायों द्वारा आवश्यक विशिष्ट स्वरूपण मानकों की पूर्ति होती है।

#### चरण 1: सेटिंग्स कॉन्फ़िगर करें
संपूर्ण कार्यपुस्तिका के लिए अपने इच्छित संख्या प्रारूप निर्दिष्ट करके प्रारंभ करें:
```csharp
// कस्टम दशमलव और समूह विभाजक परिभाषित करें
workbook.Settings.NumberDecimalSeparator = '.';
workbook.Settings.NumberGroupSeparator = ' ';
```
**स्पष्टीकरण:** The `NumberDecimalSeparator` कई क्षेत्रों में आमतौर पर इस्तेमाल किए जाने वाले एक अवधि (.) पर सेट किया गया है। `NumberGroupSeparator` को एक स्थान (' ') के रूप में कॉन्फ़िगर किया गया है, जिसे क्षेत्रीय प्राथमिकताओं के आधार पर अनुकूलित किया जा सकता है।

#### चरण 2: कस्टम शैलियाँ लागू करें
एक बार विभाजक परिभाषित हो जाने पर, अपने कक्षों पर कस्टम शैली लागू करें:
```csharp
Worksheet worksheet = workbook.Worksheets[0];

// सेल मान सेट करें और शैली लागू करें
Cell cell = worksheet.Cells["A1"];
cell.PutValue(123456.789);

Style style = cell.GetStyle();
style.Custom = "#,##0.000;[Red]#,##0.000"; // कस्टम प्रारूप स्ट्रिंग
cell.SetStyle(style);
```
**स्पष्टीकरण:** कस्टम प्रारूप `#,##0.000` तीन दशमलव स्थानों को सुनिश्चित करता है और परिभाषित विभाजकों का उपयोग करके अंकों को समूहीकृत करता है।

#### चरण 3: कॉलम स्वचालित रूप से फ़िट करें
यह सुनिश्चित करने के लिए कि आपका डेटा अच्छी तरह से प्रस्तुत किया गया है, कॉलम को स्वचालित रूप से फ़िट करें:
```csharp
worksheet.AutoFitColumns();
```
यह विधि स्तंभ की चौड़ाई को उनकी विषय-वस्तु के अनुरूप स्वचालित रूप से समायोजित कर देती है।

#### चरण 4: PDF के रूप में सहेजें
अंत में, अपनी कस्टम सेटिंग्स के साथ कार्यपुस्तिका को PDF के रूप में सहेजें:
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY/CustomSeparator_out.pdf");
```
