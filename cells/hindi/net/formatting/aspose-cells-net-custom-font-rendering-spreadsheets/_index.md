**आप क्या सीखेंगे:**
- स्प्रेडशीट रेंडरिंग के लिए डिफ़ॉल्ट फ़ॉन्ट सेट करना।

## कार्यान्वयन मार्गदर्शिका

### फ़ीचर 1: स्प्रेडशीट रेंडर करते समय डिफ़ॉल्ट फ़ॉन्ट सेट करें

#### अवलोकन
यह सुविधा स्प्रेडशीट फ़ॉन्ट्स की सुसंगत प्रस्तुति सुनिश्चित करती है, भले ही निर्दिष्ट फ़ॉन्ट गायब हों या अज्ञात हों।

#### चरण-दर-चरण कार्यान्वयन
**चरण 1: अपनी कार्यपुस्तिका तैयार करें**
कार्यपुस्तिका ऑब्जेक्ट बनाएं और उसकी डिफ़ॉल्ट शैली सेट करें:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook();
Style s = wb.DefaultStyle;
s.Font.Name = "Arial"; // प्रारंभिक डिफ़ॉल्ट फ़ॉन्ट सेट करें.
wb.DefaultStyle = s;
```
**चरण 2: अपनी वर्कशीट कॉन्फ़िगर करें**
अपनी वर्कशीट तक पहुँचें, सेल मान सेट करें, और शैलियाँ लागू करें:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell cell = ws.Cells["A4"];
cell.PutValue("This text uses a custom default font.");

Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist"; // जानबूझकर अनुपलब्ध फ़ॉन्ट का उपयोग करें।
st.Font.Size = 20;
st.IsTextWrapped = true;
cell.SetStyle(st);

// बेहतर दृश्य के लिए कॉलम की चौड़ाई और पंक्ति की ऊंचाई समायोजित करें:
ws.Cells.SetColumnWidth(0, 80);
ws.Cells.SetRowHeight(3, 60);
```
**चरण 3: कस्टम फ़ॉन्ट के साथ रेंडर करें**
विभिन्न डिफ़ॉल्ट फ़ॉन्ट का उपयोग करके अपनी वर्कशीट को रेंडर करने के लिए छवि विकल्प सेट करें:
```csharp
using Aspose.Cells.Rendering;

ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.OnePagePerSheet = true;
opts.ImageType = Drawing.ImageType.Png;

// डिफ़ॉल्ट फ़ॉन्ट के रूप में 'Arial' के साथ रेंडर करें।
opts.DefaultFont = "Arial";
SheetRender sr = new SheetRender(ws, opts);
sr.ToImage(0, System.IO.Path.Combine(outputDir, "out_a.png"));

// 'टाइम्स न्यू रोमन' में बदलें।
opts.DefaultFont = "Times New Roman";
sr = new SheetRender(ws, opts);
sr.ToImage(0, System.IO.Path.Combine(outputDir, "times_new_roman_out.png"));
```
### सुविधा 2: कॉलम की चौड़ाई और पंक्ति की ऊंचाई सेट करें

#### अवलोकन
स्तंभ की चौड़ाई और पंक्ति की ऊंचाई को समायोजित करने से स्पष्ट और पेशेवर डेटा प्रदर्शन सुनिश्चित होता है।

**चरण-दर-चरण कार्यान्वयन**
**चरण 1: आयाम समायोजित करें**
वर्कशीट तक पहुंचें और विशिष्ट आयाम निर्धारित करें:
```csharp
Worksheet ws = wb.Worksheets[0];
ws.Cells.SetColumnWidth(0, 80); // प्रथम कॉलम की चौड़ाई निर्धारित करें.
ws.Cells.SetRowHeight(3, 60);   // चौथी पंक्ति की ऊंचाई निर्धारित करें.
```
