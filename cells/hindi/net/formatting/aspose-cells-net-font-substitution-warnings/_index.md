## कार्यान्वयन मार्गदर्शिका
यह मार्गदर्शिका फ़ॉन्ट प्रतिस्थापन के लिए चेतावनी कॉलबैक सेट अप करने और इन चेतावनियों के साथ Excel कार्यपुस्तिका को PDF में परिवर्तित करने के बारे में बताती है।

### फ़ॉन्ट प्रतिस्थापन चेतावनी कॉलबैक का कार्यान्वयन
#### अवलोकन
यहां लक्ष्य एक ऐसी प्रणाली बनाना है जो रूपांतरण के दौरान जब भी लाइब्रेरी कोई फ़ॉन्ट प्रतिस्थापित करती है तो आपको सचेत कर दे, जिससे यह सुनिश्चित हो सके कि आपका आउटपुट अपेक्षाओं के अनुरूप है।

#### चरण-दर-चरण कार्यान्वयन
**कॉलबैक क्लास बनाएं**
कार्यान्वयन करने वाला वर्ग परिभाषित करें `IWarningCallback` रूपांतरण जैसे कार्यों के दौरान चेतावनियों को संभालने के लिए:
```csharp
using Aspose.Cells;
using System.Diagnostics;

public class GetWarningsForFontSubstitution : IWarningCallback
{
    // फ़ॉन्ट प्रतिस्थापन चेतावनियों को कैप्चर और लॉग करने की विधि।
    public void Warning(WarningInfo info)
    {
        if (info.WarningType == WarningType.FontSubstitution)
        {
            Debug.WriteLine("WARNING INFO: " + info.Description);
        }
    }
}
```

**स्पष्टीकरण:** यह वर्ग रूपांतरण के दौरान चेतावनी घटनाओं को सुनता है। यदि घटना का प्रकार है `FontSubstitution`, यह एक विस्तृत संदेश लॉग करता है `Debug.WriteLine`.

### फ़ॉन्ट प्रतिस्थापन चेतावनियों के साथ कार्यपुस्तिका को PDF में रूपांतरित करना
#### अवलोकन
हमारी चेतावनी कॉलबैक तैयार होने के साथ, आइए इसका उपयोग फ़ॉन्ट प्रतिस्थापन चेतावनियों को कैप्चर करते हुए Excel कार्यपुस्तिका को PDF फ़ाइल में परिवर्तित करने के लिए करें।

**रूपांतरण को क्रियान्वित करना**
रूपांतरण प्रक्रिया को संभालने के लिए एक स्थैतिक वर्ग और विधि बनाएँ:
```csharp
using Aspose.Cells;
using System.IO;

public static class ConvertWorkbookToPdfWithWarnings
{
    public static void Run()
    {
        // अपनी स्रोत और आउटपुट निर्देशिकाएँ परिभाषित करें.
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string OutputDir = "YOUR_OUTPUT_DIRECTORY";

        // निर्दिष्ट निर्देशिका से Excel कार्यपुस्तिका लोड करें.
        Workbook workbook = new Workbook(SourceDir + "sampleGetWarningsForFontSubstitution.xlsx");

        // बचत विकल्पों को अनुकूलित करने के लिए PdfSaveOptions का एक उदाहरण बनाएं।
        PdfSaveOptions options = new PdfSaveOptions();

        // फ़ॉन्ट प्रतिस्थापन चेतावनियों को संभालने के लिए हमारी चेतावनी कॉलबैक असाइन करें।
        options.WarningCallback = new GetWarningsForFontSubstitution();

        // निर्दिष्ट विकल्पों का उपयोग करके कार्यपुस्तिका को पीडीएफ फाइल के रूप में सहेजें।
        workbook.Save(OutputDir + "outputGetWarningsForFontSubstitution.pdf", options);
    }
}
```

**स्पष्टीकरण:** यह कोड एक एक्सेल फ़ाइल लोड करता है और सेट करता है `PdfSaveOptions` हमारे कस्टम चेतावनी कॉलबैक का उपयोग करने के लिए। कॉल करते समय `workbook.Save`, किसी भी फ़ॉन्ट प्रतिस्थापन चेतावनी को कॉलबैक द्वारा कैप्चर किया जाता है, जिससे आपके आउटपुट गुणवत्ता पर बेहतर नियंत्रण की अनुमति मिलती है।
