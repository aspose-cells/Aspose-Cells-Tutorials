## कार्यान्वयन गाइड (H2)

### OLE ऑब्जेक्ट्स निकालना

यह सुविधा आपको अपनी एक्सेल शीट में एम्बेडेड विभिन्न प्रकार की फाइलों को निकालने और उनके प्रारूप प्रकार के आधार पर उन्हें सहेजने में सक्षम बनाती है।

#### चरण 1: अपनी कार्यपुस्तिका लोड करें
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/book1.xls");
```

#### चरण 2: OLE ऑब्जेक्ट तक पहुँचें
```csharp
Aspose.Cells.Drawing.OleObjectCollection oles = workbook.Worksheets[0].OleObjects;
```

#### चरण 3: प्रारूप के आधार पर पुनरावृति करें और सहेजें

प्रत्येक एम्बेडेड ऑब्जेक्ट को उसके फ़ाइल प्रारूप प्रकार के आधार पर नियंत्रित किया जाता है।

```csharp
for (int i = 0; i < oles.Count; i++)
{
    Aspose.Cells.Drawing.OleObject ole = oles[i];
    string fileName = "YOUR_OUTPUT_DIRECTORY/ole_" + i + ".";

    switch (ole.FileFormatType)
    {
        case FileFormatType.Doc:
            fileName += "doc";
            break;
        case FileFormatType.Xlsx:
            fileName += "Xlsx";
            break;
        case FileFormatType.Ppt:
            fileName += "Ppt";
            break;
        case FileFormatType.Pdf:
            fileName += "Pdf";
            break;
        default:
            fileName += "Jpg";  // अज्ञात प्रारूपों को छवियों के रूप में संभालें
            break;
    }

    if (ole.FileFormatType == FileFormatType.Xlsx)
    {
        MemoryStream ms = new MemoryStream();
        ms.Write(ole.ObjectData, 0, ole.ObjectData.Length);
        
        Workbook oleBook = new Workbook(ms);
        oleBook.Settings.IsHidden = false; // सुनिश्चित करें कि कार्यपुस्तिका छिपी हुई न हो
        oleBook.Save("YOUR_OUTPUT_DIRECTORY/Excel_File" + i + ".out.xlsx");
    }
    else
    {
        using (FileStream fs = File.Create(fileName))
        {
            fs.Write(ole.ObjectData, 0, ole.ObjectData.Length);
        }
    }
}
```

### मुख्य भागों का स्पष्टीकरण

- **फ़ाइल प्रारूप प्रकार**: यह निर्धारित करता है कि निकाले गए ऑब्जेक्ट को कैसे सहेजा जाए। प्रत्येक केस एक प्रासंगिक फ़ाइल एक्सटेंशन जोड़ता है।
- **मेमोरीस्ट्रीम**: उनकी जटिल संरचना के कारण एक्सेल फ़ाइलों को संभालने के लिए उपयोग किया जाता है।

## व्यावहारिक अनुप्रयोग (H2)

OLE ऑब्जेक्ट्स को निकालने का तरीका समझने से विभिन्न व्यावहारिक अनुप्रयोग सामने आ सकते हैं:

1. **डेटा संग्रहण**: आसान अभिलेखीकरण या समीक्षा प्रक्रियाओं के लिए एम्बेडेड दस्तावेजों के निष्कर्षण को स्वचालित करें।
2. **दस्तावेज़ प्रबंधन प्रणालियों के साथ एकीकरण**: निकाले गए ऑब्जेक्ट्स को अपने दस्तावेज़ प्रबंधन वर्कफ़्लो में सहजता से एकीकृत करें।
3. **सामग्री का पुनःप्रयोजन**: विभिन्न प्लेटफार्मों या प्रारूपों के लिए प्रस्तुतियों, पीडीएफ और अन्य मीडिया प्रकारों को पुन: उपयोग करें।
