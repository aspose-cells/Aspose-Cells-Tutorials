# .NET के लिए Aspose.Cells के साथ Excel OLE ऑब्जेक्ट निष्कर्षण और सहेजना स्वचालित करें

## परिचय

क्या आप अपनी एक्सेल फाइलों में एम्बेडेड ऑब्जेक्ट्स के निष्कर्षण को स्वचालित करके अपने वर्कफ़्लो को सुव्यवस्थित करना चाहते हैं? चाहे आप डेवलपर हों या डेटा विश्लेषक, **.NET के लिए Aspose.Cells** मैन्युअल प्रयास और त्रुटियों को काफी हद तक कम कर सकता है। यह ट्यूटोरियल आपको एक्सेल वर्कबुक से उनके फ़ाइल स्वरूपों के आधार पर ऑब्जेक्ट लिंकिंग और एम्बेडिंग (OLE) ऑब्जेक्ट्स को निकालने और सहेजने के बारे में मार्गदर्शन करेगा।

### आप क्या सीखेंगे:
- Aspose.Cells का उपयोग करके Excel कार्यपुस्तिका खोलना और लोड करना।
- किसी कार्यपत्रक में OLE ऑब्जेक्ट्स के संग्रह तक पहुँचना।
- OLE ऑब्जेक्ट्स को उनके विशिष्ट प्रारूपों के अनुसार निकालना और सहेजना।

आइए अपना वातावरण तैयार करें और इस कुशल सुविधा को क्रियान्वित करें!

## कार्यान्वयन मार्गदर्शिका

### विशेषता 1: कार्यपुस्तिका खोलें और लोड करें

आइए एक निर्दिष्ट निर्देशिका से एक एक्सेल कार्यपुस्तिका लोड करें।

#### चरण-दर-चरण कार्यान्वयन:

**स्रोत निर्देशिका परिभाषित करें:**
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
```

**कार्यपुस्तिका इंस्टेंस बनाएँ:**
```csharp
Workbook workbook = new Workbook(sourceDir + "sampleExtractOLEObjects.xlsx");
```
यह चरण आपकी एक्सेल फ़ाइल को लोड करता है `Workbook` ऑब्जेक्ट, आपको इसकी सामग्री को प्रोग्रामेटिक रूप से हेरफेर करने की अनुमति देता है।

### फ़ीचर 2: वर्कशीट में OleObject संग्रह तक पहुँचें

अब, कार्यपुस्तिका की पहली वर्कशीट में एम्बेडेड OLE ऑब्जेक्ट तक पहुँचें।

#### चरण-दर-चरण कार्यान्वयन:

**प्रथम कार्यपत्रक तक पहुंचें:**
```csharp
OleObjectCollection oles = workbook.Worksheets[0].OleObjects;
```
यह स्निपेट आगे की प्रक्रिया के लिए निर्दिष्ट वर्कशीट से सभी OLE ऑब्जेक्ट्स को पुनर्प्राप्त करता है।

### विशेषता 3: प्रारूप के आधार पर OLE ऑब्जेक्ट्स निकालें और सहेजें

इसके बाद, प्रत्येक OLE ऑब्जेक्ट को पुनरावृत्त करके उसका डेटा निकालें और उसे उसके प्रारूप के अनुसार सेव करें।

#### चरण-दर-चरण कार्यान्वयन:

**OLE ऑब्जेक्ट्स के माध्यम से पुनरावृति करें:**
```csharp
using System.IO;

for (int i = 0; i < oles.Count; i++)
{
    OleObject ole = oles[i];
    byte[] oleData = ole.ObjectData;
    string fileName = outputDir + "outputExtractOLEObjects" + (i+1) + ".";

    switch (ole.FileFormatType)
    {
        case FileFormatType.Doc:
            fileName += "doc";
            break;
        case FileFormatType.Docx:
            fileName += "docx";
            break;
        case FileFormatType.Excel97To2003:
            fileName += "xls";
            break;
        case FileFormatType.Xlsx:
            // XLSX प्रारूपों के लिए विशेष हैंडलिंग
            using (MemoryStream ms = new MemoryStream())
            {
                ms.Write(ole.ObjectData, 0, ole.ObjectData.Length);
                Workbook oleBook = new Workbook(ms);
                oleBook.Settings.IsHidden = false;

                ms.SetLength(0); // स्ट्रीम साफ़ करें
                oleBook.Save(ms, SaveFormat.Xlsx);

                ms.Position = 0;
                byte[] bts = new byte[ms.Length];
                ms.Read(bts, 0, (int)ms.Length);
                oleData = bts;
            }
            fileName += "xlsx";
            break;
        case FileFormatType.Ppt:
            fileName += "ppt";
            break;
        case FileFormatType.Pdf:
            fileName += "pdf";
            break;
        case FileFormatType.Unknown:
            Guid g = new Guid(ole.ClassIdentifier);
            if (g.ToString() == "b801ca65-a1fc-11d0-85ad-444553540000")
            {
                fileName += "pdf";
            }
            else
            {
                fileName += "jpg";
            }                      
            break;
        default:
            // अन्य प्रारूपों को संभालें या अपवाद फेंकें
            break;
    }

    File.WriteAllBytes(fileName, oleData);
}
```
यह अनुभाग दर्शाता है कि विभिन्न फ़ाइल स्वरूपों को गतिशील रूप से कैसे प्रबंधित किया जाए और उन्हें उचित रूप से कैसे सहेजा जाए।

## व्यावहारिक अनुप्रयोगों

एक्सेल फाइलों से OLE ऑब्जेक्ट्स निकालने के लिए यहां कुछ वास्तविक उपयोग के मामले दिए गए हैं:
1. **स्वचालित डेटा रिपोर्टिंग:** डेटा रिपोर्टिंग प्रक्रिया के भाग के रूप में एम्बेडेड दस्तावेज़ों या छवियों को स्वचालित रूप से निकालें।
2. **डेटा संग्रहण प्रणालियाँ:** अनुपालन उद्देश्यों के लिए एम्बेड की गई सामग्री को स्प्रेडशीट में संग्रहित करें।
3. **दस्तावेज़ प्रबंधन प्रणालियों के साथ एकीकरण:** निकाले गए OLE ऑब्जेक्ट्स को अन्य दस्तावेज़ प्रबंधन प्लेटफ़ॉर्म में निर्बाध रूप से एकीकृत करें।

## अक्सर पूछे जाने वाले प्रश्न अनुभाग

1. **मैं असमर्थित OLE ऑब्जेक्ट प्रारूपों को कैसे संभालूँ?**
   - अज्ञात या असमर्थित प्रारूपों के लिए, का उपयोग करें `FileFormatType.Unknown` मामले को समझें और आवश्यकतानुसार कस्टम तर्क लागू करें।

2. **क्या Aspose.Cells बड़ी Excel फ़ाइलों को कुशलतापूर्वक संभाल सकता है?**

3. **यदि मेरी निकाली गई फ़ाइल का प्रारूप ग़लत है तो क्या होगा?**

5. **मैं निकाले गए OLE ऑब्जेक्ट्स को अन्य सिस्टम में कैसे एकीकृत करूं?**
