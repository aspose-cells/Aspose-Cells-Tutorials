## कार्यान्वयन मार्गदर्शिका

यह अनुभाग आपको एम्बेडेड OLE ऑब्जेक्ट्स के लिए वर्ग पहचानकर्ता प्राप्त करने और सेट करने की प्रक्रिया से परिचित कराता है।

### एम्बेडेड OLE ऑब्जेक्ट से क्लास आइडेंटिफ़ायर प्राप्त करें

**अवलोकन**यह सुविधा आपको अपनी एक्सेल फ़ाइल में किसी विशिष्ट एम्बेडेड ऑब्जेक्ट का विशिष्ट पहचानकर्ता (GUID) प्राप्त करने की अनुमति देती है।

#### चरण 1: अपनी कार्यपुस्तिका लोड करें
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook wb = new Workbook(sourceDir + "sampleGetSetClassIdentifierEmbedOleObject.xls");
```

#### चरण 2: वर्कशीट और OLE ऑब्जेक्ट तक पहुँचें
```csharp
Worksheet ws = wb.Worksheets[0];
OleObject oleObj = ws.OleObjects[0];
```

#### चरण 3: GUID में कनवर्ट करें और प्रिंट करें
```csharp
Guid guid = new Guid(oleObj.ClassIdentifier);
Console.WriteLine(guid.ToString().ToUpper());
```

### नया क्लास पहचानकर्ता सेट करें

**अवलोकन**यदि आवश्यक हो तो किसी मौजूदा OLE ऑब्जेक्ट के वर्ग पहचानकर्ता को संशोधित करें।

#### चरण 1: नया GUID परिभाषित करें
```csharp
string newClassId = "Your-New-GUID-Here"; // वास्तविक GUID स्ट्रिंग से प्रतिस्थापित करें
Guid newGuid = new Guid(newClassId);
```

#### चरण 2: परिवर्तन असाइन करें और सहेजें
```csharp
oleObj.ClassIdentifier = newGuid.ToByteArray();
wb.Save("updatedWorkbook.xls");
```
