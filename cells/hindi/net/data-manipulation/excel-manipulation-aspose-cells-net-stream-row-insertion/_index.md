## कार्यान्वयन मार्गदर्शिका

### सुविधा 1: फ़ाइल स्ट्रीम निर्माण और कार्यपुस्तिका आरंभीकरण

यह सुविधा दर्शाती है कि एक्सेल फ़ाइल को पढ़ने के लिए फ़ाइल स्ट्रीम कैसे बनाएं, एक फ़ाइल को इंस्टेंटाइज़ करें `Workbook` ऑब्जेक्ट पर क्लिक करें, और पहले वर्कशीट तक पहुंचें।

#### चरण 1: फ़ाइलस्ट्रीम बनाएँ

एक बनाकर शुरू करें `FileStream` अपनी एक्सेल फ़ाइल खोलने के लिए। यह महत्वपूर्ण है क्योंकि यह आपको कार्यपुस्तिका में मौजूद डेटा को पढ़ने की अनुमति देता है।

```csharp
using System.IO;
using Aspose.Cells;

// स्रोत निर्देशिका को परिभाषित करें और फ़ाइल स्ट्रीम बनाएं
string SourceDir = "YOUR_SOURCE_DIRECTORY";
using (FileStream fstream = new FileStream(SourceDir + "/book1.xls", FileMode.Open)) {
```

#### चरण 2: कार्यपुस्तिका को इंस्टैंशिएट करें

निर्मित फ़ाइल स्ट्रीम का उपयोग करके, एक इंस्टैंसिएट करें `Workbook` ऑब्जेक्ट। यहीं से आपके सभी डेटा हेरफेर शुरू होते हैं।

```csharp
    // फ़ाइल स्ट्रीम का उपयोग करके वर्कबुक ऑब्जेक्ट को इंस्टैंशिएट करना
    Workbook workbook = new Workbook(fstream);
```

#### चरण 3: वर्कशीट तक पहुंचें

डेटा पढ़ने या संशोधित करने जैसे कार्य करने के लिए पहली वर्कशीट तक पहुँचें।

```csharp
    // एक्सेल वर्कबुक में पहली वर्कशीट तक पहुँचना
    Worksheet worksheet = workbook.Worksheets[0];
}
```

### फ़ीचर 2: फ़ॉर्मेटिंग विकल्पों के साथ पंक्ति सम्मिलित करना

विशिष्ट स्वरूपण विकल्पों का उपयोग करके किसी Excel शीट में निर्दिष्ट स्थान पर पंक्ति सम्मिलित करना सीखें।

#### चरण 1: कार्यपुस्तिका लोड करें और कार्यपत्रक तक पहुँचें

अपनी मौजूदा कार्यपुस्तिका खोलें और उस कार्यपत्रक तक पहुँचें जहाँ आप परिवर्तन करना चाहते हैं।

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
// किसी मौजूदा फ़ाइल से वर्कबुक ऑब्जेक्ट को इंस्टैंशिएट करना
Workbook workbook = new Workbook(SourceDir + "/book1.xls");

// एक्सेल फ़ाइल में पहली वर्कशीट तक पहुँचना
Worksheet worksheet = workbook.Worksheets[0];
```

#### चरण 2: InsertOptions सेटअप करें

पंक्तियाँ सम्मिलित करते समय एकरूपता सुनिश्चित करने के लिए स्वरूपण विकल्प परिभाषित करें।

```csharp
using Aspose.Cells;

// पंक्तियाँ सम्मिलित करने के लिए स्वरूपण विकल्प सेट करना
InsertOptions insertOptions = new InsertOptions {
    CopyFormatType = CopyFormatType.SameAsAbove
};
```

#### चरण 3: पंक्ति सम्मिलित करें

निर्दिष्ट स्थान पर एक पंक्ति डालें, इस मामले में, तीसरी पंक्ति (सूचकांक 2)।

```csharp
// वर्कशीट में तीसरे स्थान पर एक पंक्ति सम्मिलित करना (सूचकांक 2)
worksheet.Cells.InsertRows(2, 1, insertOptions);

// संशोधित Excel फ़ाइल को आउटपुट निर्देशिका में सहेजना
workbook.Save("YOUR_OUTPUT_DIRECTORY/InsertingARowWithFormatting.out.xls");
```

### समस्या निवारण युक्तियों

- **फ़ाइल प्राप्त नहीं हुई**: सुनिश्चित करें कि आपका `SourceDir` रास्ता सही और सुलभ है.
- **स्म्रति से रिसाव**: उपयोग के बाद हमेशा स्ट्रीम बंद करें `using` उचित निपटान सुनिश्चित करने के लिए बयान जारी किए गए।
