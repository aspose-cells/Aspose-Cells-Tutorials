# .NET के लिए Aspose.Cells के साथ Excel में HTML-स्वरूपित डेटाटेबल्स को कैसे आयात करें

## परिचय

क्या आप Excel में आयातित वेब पेज या डेटाबेस डेटा को मैन्युअल रूप से फ़ॉर्मेट करने में संघर्ष कर रहे हैं? आप अकेले नहीं हैं! डेवलपर्स को अक्सर बोल्ड और इटैलिक जैसी टेक्स्ट शैलियों को बनाए रखने की आवश्यकता होती है, जो पठनीयता के लिए महत्वपूर्ण है। .NET के लिए Aspose.Cells के साथ, स्टाइलिंग को संरक्षित करते हुए HTML-स्वरूपित स्ट्रिंग्स वाले डेटाटेबल को Excel वर्कबुक में आयात करना आसान हो जाता है।

इस ट्यूटोरियल में, आप सीखेंगे कि Aspose.Cells का उपयोग करके डेटाटेबल से HTML-स्वरूपित डेटा को Excel में कैसे आयात किया जाए, जिससे यह सुनिश्चित हो सके कि आपका डेटा स्प्रेडशीट में ठीक वैसा ही दिखाई दे जैसा कि अपेक्षित है।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Cells को सेट अप और कॉन्फ़िगर करना
- Aspose.Cells का उपयोग करके HTML फ़ॉर्मेटिंग के साथ डेटाटेबल्स आयात करना
- सामग्री को फिट करने के लिए पंक्ति और स्तंभ आकार को स्वचालित रूप से समायोजित करना
- कार्यपुस्तिकाओं को XLSX और ODS जैसे अनेक प्रारूपों में सहेजना

आइये सबसे पहले यह सुनिश्चित करें कि आपके पास आवश्यक पूर्वापेक्षाएँ हैं!

## कार्यान्वयन मार्गदर्शिका

आइए HTML फ़ॉर्मेटिंग के साथ डेटाटेबल्स को आयात करने को स्पष्ट चरणों में विभाजित करें।

### अपना डेटा स्रोत तैयार करना

**अवलोकन:**
Aspose.Cells की स्टाइलिंग क्षमता को प्रदर्शित करने के लिए HTML स्वरूपित स्ट्रिंग्स को शामिल करने वाले नमूना डेटा के साथ एक DataTable सेट अप करके प्रारंभ करें।
```csharp
using System.Data;

// अपनी स्रोत और आउटपुट निर्देशिकाएँ यहाँ सेट करें
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// कुछ HTML स्वरूपित मानों के साथ एक डेटाटेबल तैयार करें
dataTable = new DataTable("Products");
dataTable.Columns.Add("Product ID", typeof(Int32));
dataTable.Columns.Add("Product Name", typeof(string));
dataTable.Columns.Add("Units In Stock", typeof(Int32));

// HTML स्वरूपण के साथ पंक्तियाँ जोड़ना
DataRow dr = dataTable.NewRow();
dr[0] = 1;
dr[1] = "<i>Aniseed</i> Syrup"; // उत्पाद नाम के लिए HTML इटैलिक
dr[2] = 15;
dataTable.Rows.Add(dr);

dr = dataTable.NewRow();
dr[0] = 2;
dr[1] = "<b>Boston Crab Meat</b>"; // उत्पाद नाम के लिए HTML बोल्ड
dr[2] = 123;
dataTable.Rows.Add(dr);
```

### आयात विकल्प सेट करना

**आयात तालिका विकल्प कॉन्फ़िगर करें:**
उपयोग `ImportTableOptions` यह निर्दिष्ट करने के लिए कि सेल मानों को HTML स्ट्रिंग के रूप में व्याख्या किया जाना चाहिए।
```csharp
// HTML स्वरूपित स्ट्रिंग्स को संभालने के लिए आयात विकल्प बनाएँ
ImportTableOptions importOptions = new ImportTableOptions();
importOptions.IsFieldNameShown = true; // आयात में स्तंभ शीर्षलेख शामिल करें
importOptions.IsHtmlString = true; // सेल मानों को HTML स्ट्रिंग के रूप में समझें
```

### एक्सेल में डेटा आयात करना

**अवलोकन:**
एक कार्यपुस्तिका और कार्यपत्रक बनाएं, फिर उपयोग करें `ImportData` अपने डेटाटेबल को सभी स्वरूपण के साथ एक्सेल में लाने के लिए।
```csharp
// कार्यपुस्तिका बनाएं और पहली कार्यपत्रक प्राप्त करें
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];

// पंक्ति 0, स्तंभ 0 से शुरू होने वाली डेटाटेबल को आयात करें
worksheet.Cells.ImportData(dataTable, 0, 0, importOptions);

// बेहतर पठनीयता के लिए पंक्ति और स्तंभ आकार समायोजित करें
worksheet.AutoFitRows();
worksheet.AutoFitColumns();
```

### अपनी कार्यपुस्तिका को सहेजना

अंत में, विभिन्न स्प्रेडशीट अनुप्रयोगों में संगतता सुनिश्चित करने के लिए अपनी कार्यपुस्तिका को XLSX और ODS दोनों प्रारूपों में सहेजें।
```csharp
string output1Path = OutputDir + "Output.out.xlsx";
string output2Path = OutputDir + "Output.out.ods";

// कार्यपुस्तिका को दो प्रारूपों में सहेजें
workbook.Save(output1Path);
workbook.Save(output2Path);
```

- **रिपोर्टिंग:** वित्तीय रिपोर्टों पर स्वचालित रूप से शैलियाँ लागू करना।
- **डेटा माइग्रेशन:** HTML स्वरूपण को बनाए रखते हुए वेब-स्क्रैप किए गए डेटा को Excel में ले जाना।
- **सूची प्रबंधन:** महत्वपूर्ण विशेषताओं पर जोर देते हुए उत्पाद विवरण प्रदर्शित करना।

इस कार्यक्षमता को एकीकृत करने से व्यवसाय विश्लेषण और रिपोर्टिंग कार्यों में प्रक्रियाओं को काफी हद तक सुव्यवस्थित किया जा सकता है।
