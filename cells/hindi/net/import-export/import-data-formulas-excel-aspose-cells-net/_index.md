## कार्यान्वयन मार्गदर्शिका

यह अनुभाग डेटा आइटम निर्दिष्ट करने और उन्हें सूत्रों के साथ एक्सेल वर्कशीट में आयात करने को कवर करता है।

### डेटा आइटम निर्दिष्ट करना

#### अवलोकन

आयात करने से पहले कस्टम डेटा ऑब्जेक्ट बनाना और व्यवस्थित करना महत्वपूर्ण है। यह सुविधा C# क्लास का उपयोग करके इन ऑब्जेक्ट को परिभाषित करने पर केंद्रित है।

#### चरण-दर-चरण कार्यान्वयन

**उपयोगकर्ता-परिभाषित क्लास परिभाषित करें**

```csharp
using System;
using System.Collections.Generic;

class FeatureSpecifyDataItems
{
    class DataItems
    {
        public int Number1 { get; set; }
        public int Number2 { get; set; }
        public string Formula1 { get; set; }
        public string Formula2 { get; set; }
    }

    public static void Run()
    {
        List<DataItems> dis = new List<DataItems>();

        // डेटा आइटम परिभाषित करें
        DataItems di = new DataItems();
        di.Number1 = 2005;
        di.Number2 = 3505;
        di.Formula1 = "+=SUM(A5,B5)"; // A5 और B5 का योग करने का सूत्र
        di.Formula2 = "+=HYPERLINK(\"https://www.aspose.com\", \"Aspose वेबसाइट\")";

        dis.Add(di);
    }
}
```

**स्पष्टीकरण**: 
- The `DataItems` वर्ग पूर्णांक और सूत्र रखता है.
- आयात के दौरान लचीलेपन के लिए सूत्रों को स्ट्रिंग के रूप में परिभाषित किया जाता है।

### सूत्रों के साथ वर्कशीट में डेटा आयात करना

#### अवलोकन

यह सुविधा पहले से निर्मित डेटा आइटम को एक्सेल वर्कशीट में आयात करने का प्रदर्शन करती है, तथा निर्दिष्ट करती है कि किन फ़ील्ड को सूत्रों के रूप में माना जाना चाहिए।

#### चरण-दर-चरण कार्यान्वयन

**कस्टम ऑब्जेक्ट आयात करें**

```csharp
using Aspose.Cells;

class FeatureImportDataWithFormulas
{
    string outputDir = "YOUR_OUTPUT_DIRECTORY";

    public static void Run()
    {
        Workbook wb = new Workbook();
        Worksheet ws = wb.Worksheets[0];

        ImportTableOptions opts = new ImportTableOptions();
        opts.IsFormulas = new bool[] { false, false, true, true };

        List<DataItems> dis = new List<DataItems>(); // मान लीजिए कि यह सूची ऊपर दिखाए अनुसार भरी गई है।
        
        ws.Cells.ImportCustomObjects(dis, 0, 0, opts);
        wb.CalculateFormula();
        ws.AutoFitColumns();

        wb.Save(outputDir + "/outputSpecifyFormulaFieldsWhileImportingDataToWorksheet.xlsx");
    }
}
```

**स्पष्टीकरण**: 
- `ImportTableOptions` निर्दिष्ट करता है कि कौन से फ़ील्ड सूत्र हैं.
- सूत्रों की गणना निम्न प्रकार से की जाती है `wb.CalculateFormula()`.
- बेहतर पठनीयता के लिए कॉलम स्वचालित रूप से फिट किए जाते हैं।
