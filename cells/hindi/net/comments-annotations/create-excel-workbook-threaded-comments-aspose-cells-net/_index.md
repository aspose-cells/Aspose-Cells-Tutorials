# Aspose.Cells .NET का उपयोग करके थ्रेडेड टिप्पणियों के साथ एक कार्यपुस्तिका बनाएँ

## परिचय

- नई कार्यपुस्तिका इंस्टैंस बनाना
- थ्रेडेड टिप्पणियों के लिए लेखक जोड़ना
- कोशिकाओं के भीतर थ्रेडेड टिप्पणियाँ लागू करना

आइए अपने Excel-संबंधित प्रोजेक्ट्स को कारगर बनाने के लिए .NET के लिए Aspose.Cells का लाभ उठाएं।

## कार्यान्वयन मार्गदर्शिका

### थ्रेडेड टिप्पणियों के साथ कार्यपुस्तिका बनाना और सहेजना

#### अवलोकन
इस अनुभाग में, हम .NET के लिए Aspose.Cells का उपयोग करके एक Excel कार्यपुस्तिका बनाएंगे और थ्रेडेड टिप्पणियाँ जोड़ेंगे।

#### चरण-दर-चरण निर्देश
**1. कार्यपुस्तिका को आरंभ करें**
एक नया उदाहरण बनाकर शुरू करें `Workbook`:
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// एक नई कार्यपुस्तिका इंस्टेंस बनाएँ
Workbook workbook = new Workbook();
```

**2. थ्रेडेड टिप्पणियों के लिए एक लेखक जोड़ें**
टिप्पणी लेखक को परिभाषित करें और जोड़ें `ThreadedCommentAuthors` संग्रह:
```csharp
// थ्रेडेड टिप्पणी के लिए लेखक जोड़ें
int authorIndex = workbook.Worksheets.ThreadedCommentAuthors.Add("Aspose Test", "", "");
ThreadedCommentAuthor author = workbook.Worksheets.ThreadedCommentAuthors[authorIndex];
```

**3. थ्रेडेड टिप्पणी डालें**
प्रथम कार्यपत्रक में सेल A1 में थ्रेडेड टिप्पणी जोड़ें:
```csharp
// पहली वर्कशीट में सेल A1 में थ्रेडेड टिप्पणी जोड़ें
workbook.Worksheets[0].Comments.AddThreadedComment("A1", "Test Threaded Comment", author);
```

**4. कार्यपुस्तिका सहेजें**
अंत में, अपनी कार्यपुस्तिका को इच्छित आउटपुट निर्देशिका में सहेजें:
```csharp
// कार्यपुस्तिका को आउटपुट निर्देशिका में सहेजें
workbook.Save(outputDir + "/AddThreadedComments_out.xlsx");
```
