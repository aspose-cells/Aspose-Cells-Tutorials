## कार्यान्वयन मार्गदर्शिका
अब, आइए प्रत्येक सुविधा को चरण-दर-चरण देखें।

### कार्यपुस्तिका और कार्यपत्रक बनाना
**अवलोकन**यह अनुभाग कार्यपुस्तिका बनाने और कार्यपत्रक जोड़ने का तरीका बताता है।
1. **कार्यपुस्तिका आरंभ करें**
   - एक खाली जगह बनाकर शुरू करें `Workbook` वस्तु।
   ```csharp
   Workbook workbook = new Workbook();
   ```
2. **एक नई वर्कशीट जोड़ें**
   - उपयोग `Worksheets.Add()` एक नई शीट जोड़ने की विधि.
   ```csharp
   int sheetIndex = workbook.Worksheets.Add();
   Worksheet worksheet = workbook.Worksheets[sheetIndex];
   // कार्यपुस्तिका में अब एक कार्यपत्रक है।
   ```

### किसी सेल में टिप्पणी जोड़ना
**अवलोकन**: विशिष्ट कक्षों में टिप्पणियाँ सम्मिलित करना सीखें।
1. **एक टिप्पणी जोड़ने**
   - उपयोग `Comments.Add()` सेल "F5" में टिप्पणी डालने की विधि।
   ```csharp
   int commentIndex = worksheet.Comments.Add("F5");
   Comment comment = worksheet.Comments[commentIndex];
   ```
2. **टिप्पणी नोट सेट करें**
   - अपनी टिप्पणी में पाठ निर्दिष्ट करने के लिए निम्न का उपयोग करें: `Note` संपत्ति।
   ```csharp
   comment.Note = "Hello Aspose!";
   ```

### टिप्पणी स्वरूपण
**अवलोकन**: बेहतर पठनीयता के लिए टिप्पणियों के स्वरूप को अनुकूलित करें।
1. **फ़ॉन्ट आकार और शैली समायोजित करें**
   - फ़ॉन्ट का आकार बदलें और बोल्ड फ़ॉर्मेटिंग लागू करें.
   ```csharp
   comment.Font.Size = 14;
   comment.Font.IsBold = true;
   ```
2. **सेंटीमीटर में आयाम सेट करें**
   - दृश्य स्थान को नियंत्रित करने के लिए ऊँचाई और चौड़ाई निर्दिष्ट करें.
   ```csharp
   comment.HeightCM = 10;
   comment.WidthCM = 2;
   ```

### कार्यपुस्तिका को सहेजना
**अवलोकन**: कार्यपुस्तिका को सहेजकर अपने परिवर्तनों को बनाए रखें।
1. **परिवर्तनों को सुरक्षित करें**
   - उपयोग `Workbook.Save()` किसी फ़ाइल में परिवर्तन लिखने की विधि.
   ```csharp
   workbook.Save(outputDir + "book1.out.xls");
   ```

यहां कुछ वास्तविक परिदृश्य दिए गए हैं जहां टिप्पणियां जोड़ना और प्रारूपित करना उपयोगी हो सकता है:
- **डेटा समीक्षा**टीमों के बीच साझा की गई स्प्रेडशीट में ध्यान देने योग्य क्षेत्रों को हाइलाइट करें।
- **प्रलेखन**: भावी उपयोगकर्ताओं के लिए स्पष्टीकरण या संदर्भ के साथ कक्षों पर टिप्पणी करें।
- **लेखा परीक्षा**: डेटा प्रोसेसिंग के दौरान किए गए परिवर्तनों पर नोट्स प्रदान करें।
