# .NET में पिवट टेबल डेटा प्रदर्शन प्रारूप रैंकिंग

## परिचय
जब डेटा विश्लेषण की बात आती है, खासकर एक्सेल में, तो पिवट टेबल आपके सबसे अच्छे दोस्त हैं। वे आपको डेटा को सारांशित करने, एक्सप्लोर करने और विज़ुअलाइज़ करने में मदद करते हैं, जो साधारण टेबल नहीं कर सकते। यदि आप .NET वातावरण में काम कर रहे हैं और पिवट टेबल की शक्ति का दोहन करना चाहते हैं, तो Aspose.Cells एक आदर्श लाइब्रेरी है। इसके उपयोगकर्ता-अनुकूल API और व्यापक सुविधाओं के साथ, यह आपको एक प्रो की तरह एक्सेल फ़ाइलों में हेरफेर करने में सक्षम बनाता है। इस ट्यूटोरियल में, हम Aspose.Cells का उपयोग करके .NET में पिवट टेबल डेटा डिस्प्ले फ़ॉर्मेट रैंकिंग सेट अप करने का तरीका जानेंगे, इसे स्पष्ट समझ के लिए चरण दर चरण विभाजित करेंगे।
## आवश्यक शर्तें
इससे पहले कि हम विवरण में आगे बढ़ें, आइए सुनिश्चित करें कि आपके पास अनुसरण करने के लिए सब कुछ सेट है। आपको निम्न चीज़ों की आवश्यकता होगी:
1. विकास पर्यावरण: सुनिश्चित करें कि आपके पास एक कार्यशील .NET विकास पर्यावरण है। यह Visual Studio या कोई अन्य संगत IDE हो सकता है।
2. Aspose.Cells लाइब्रेरी: आपको Aspose.Cells लाइब्रेरी की आवश्यकता होगी। आप इसे यहाँ से डाउनलोड कर सकते हैं[साइट](https://releases.aspose.com/cells/net/). बिना किसी तत्काल लागत के आरंभ करने के लिए आपके लिए एक निःशुल्क परीक्षण भी उपलब्ध है।
3.  नमूना डेटा: इस ट्यूटोरियल के लिए, हम नामक एक एक्सेल फ़ाइल का उपयोग करेंगे`PivotTableSample.xlsx`. सुनिश्चित करें कि पिवट टेबल बनाने के लिए इस फ़ाइल में आपका डेटा सही ढंग से संरचित है।
अब जबकि हमने आवश्यक बातें समझ ली हैं, तो चलिए कोड पर आते हैं!
## पैकेज आयात करें
आरंभ करने के लिए, आपको अपने .NET प्रोजेक्ट में आवश्यक नामस्थान आयात करने होंगे। यह सुनिश्चित करने के लिए एक महत्वपूर्ण कदम है कि आपका एप्लिकेशन Aspose.Cells कार्यक्षमता तक पहुँच सकता है। यहाँ बताया गया है कि आप इसे कैसे करते हैं:
### Aspose.Cells नामस्थान आयात करें
```csharp
using System;
using Aspose.Cells.Pivot;
```
अपनी C# फ़ाइल के शीर्ष पर इस पंक्ति के साथ, आप Excel फ़ाइलों के साथ काम करने के लिए आवश्यक सभी सुविधाओं तक पहुँच सकेंगे।
## चरण 1: निर्देशिकाएँ सेट करें
अपने एक्सेल दस्तावेज़ को लोड करने से पहले, आपको यह निर्दिष्ट करना होगा कि आपका स्रोत डेटा कहाँ स्थित है और आप आउटपुट को कहाँ सहेजना चाहते हैं। यहाँ उन निर्देशिकाओं को सेट अप करने का तरीका बताया गया है:
```csharp
// निर्देशिका
string sourceDir = "Your Document Directory"; // अपनी वास्तविक निर्देशिका के साथ अद्यतन करें
string outputDir = "Your Document Directory"; // अपनी वास्तविक निर्देशिका के साथ अद्यतन करें
```
 प्रतिस्थापित करना सुनिश्चित करें`"Your Document Directory"` वास्तविक पथ के साथ जहां आपकी फ़ाइलें संग्रहीत हैं.
## चरण 2: कार्यपुस्तिका लोड करें
इसके बाद, आपको एक्सेल फ़ाइल लोड करनी होगी जिसमें आपकी पिवट टेबल शामिल है। यहाँ बताया गया है कि कैसे:
```csharp
// टेम्पलेट फ़ाइल लोड करें
Workbook workbook = new Workbook(sourceDir + "PivotTableSample.xlsx");
```
`Workbook` क्लास एक्सेल फ़ाइलों के साथ काम करने का आपका गेटवे है। अपनी इनपुट फ़ाइल का पथ पास करके, आप Aspose.Cells को उस फ़ाइल को मेमोरी में लोड करने के लिए कह रहे हैं।
## चरण 3: वर्कशीट तक पहुंचें
कार्यपुस्तिका लोड करने के बाद, आपको उस विशिष्ट कार्यपत्रक तक पहुंचने की आवश्यकता है जिसमें आपकी पिवट तालिका है:
```csharp
// पहली वर्कशीट प्राप्त करें
Worksheet worksheet = workbook.Worksheets[0];
```
यह कोड स्निपेट आपकी कार्यपुस्तिका से पहली कार्यपत्रक प्राप्त करता है। यदि आपकी पिवट तालिका किसी भिन्न शीट पर स्थित है, तो बस इंडेक्स को तदनुसार समायोजित करें।
## चरण 4: पिवट तालिका तक पहुंचें
अब समय आ गया है मामले के मूल तक पहुँचने का - पिवट टेबल। आइये इसे देखें:
```csharp
int pivotIndex = 0; // पिवट तालिका का सूचकांक
PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
```
इस परिदृश्य में, हम पहली पिवट तालिका तक पहुँचते हैं। यदि आपके पास कई पिवट तालिकाएँ हैं, तो समायोजित करें`pivotIndex`.
## चरण 5: डेटा फ़ील्ड तक पहुँचें
पिवट टेबल तक पहुँचने के बाद, अगला कदम इसके डेटा फ़ील्ड में जाना है। यहाँ बताया गया है कि कैसे:
```csharp
// डेटा फ़ील्ड तक पहुँचना.
PivotFieldCollection pivotFields = pivotTable.DataFields;
```
इस संग्रह में पिवट तालिका से संबद्ध सभी डेटा फ़ील्ड शामिल हैं।
## चरण 6: डेटा प्रदर्शन प्रारूप कॉन्फ़िगर करें
अब आता है मज़ेदार हिस्सा—रैंकिंग के लिए डेटा डिस्प्ले फ़ॉर्मेट सेट करना। यह वह जगह है जहाँ आप पिवट टेबल को बताते हैं कि आप डेटा को किस तरह से दिखाना चाहते हैं:
```csharp
// डेटा फ़ील्ड में पहले डेटा फ़ील्ड तक पहुँचना.
PivotField pivotField = pivotFields[0];
// डेटा प्रदर्शन प्रारूप सेट करना
pivotField.DataDisplayFormat = PivotFieldDataDisplayFormat.RankLargestToSmallest;
```
ऐसा करके, आप पिवट टेबल को पहले डेटा फ़ील्ड को अवरोही क्रम में प्रदर्शित करने का निर्देश दे रहे हैं। यदि आप आरोही क्रम में जाना चाहते हैं, तो आप डिस्प्ले फ़ॉर्मेट को तदनुसार बदल सकते हैं।
## चरण 7: डेटा की गणना करें
पिवट टेबल में किए गए बदलाव तब तक प्रभावी नहीं होंगे जब तक आप डेटा की पुनर्गणना नहीं करते। यहाँ बताया गया है कि कैसे:
```csharp
pivotTable.CalculateData();
```
यह पंक्ति पिवट तालिका को ताज़ा करती है, तथा आपके द्वारा किए गए सभी परिवर्तन लागू करती है।
## चरण 8: आउटपुट सहेजें
अंत में, अपनी संशोधित कार्यपुस्तिका को निर्दिष्ट आउटपुट निर्देशिका में सहेजें:
```csharp
// एक्सेल फ़ाइल को सहेजना
workbook.Save(outputDir + "PivotTableDataDisplayFormatRanking_out.xlsx");
```
इससे लागू प्रदर्शन प्रारूप के साथ एक नई एक्सेल फ़ाइल बन जाएगी। 
## चरण 9: पुष्टिकरण संदेश
यह पुष्टि करना हमेशा अच्छा होता है कि सब कुछ उम्मीद के मुताबिक काम कर रहा है। आप यह जानने के लिए एक सरल कंसोल आउटपुट जोड़ सकते हैं:
```csharp
Console.WriteLine("PivotTableDataDisplayFormatRanking executed successfully.");
```
