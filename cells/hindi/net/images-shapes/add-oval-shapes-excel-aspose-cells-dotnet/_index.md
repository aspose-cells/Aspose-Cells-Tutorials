## Implementation Guide

### Adding an Oval Shape

This feature guides you through adding a basic oval shape to an Excel worksheet.

#### Overview
Adding ovals can enhance the visual appeal of your data presentation. In this section, we'll add and configure an oval in the first worksheet of our Excel file using Aspose.Cells.

#### Steps:

##### Step 1: Define Directory for Output

First, define where you want to save your output files:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
string dataDir = Path.Combine(outputDir, "OvalShapeExample");

if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
```

##### चरण 2: कार्यपुस्तिका को इंस्टैंसिएट करें

इसका एक उदाहरण बनाएं `Workbook` एक्सेल फाइलों के साथ काम करना शुरू करने के लिए क्लास:

```csharp
Workbook excelbook = new Workbook();
```

##### चरण 3: अंडाकार आकार जोड़ें

उपयोग `AddOval` वर्कशीट में अंडाकार आकृति रखने की विधि:

```csharp
// निर्दिष्ट निर्देशांक और आकार पर एक अंडाकार जोड़ें
Oval oval1 = excelbook.Worksheets[0].Shapes.AddOval(2, 0, 2, 0, 130, 160);
```

##### चरण 4: प्लेसमेंट कॉन्फ़िगर करें

प्लेसमेंट प्रकार को इस पर सेट करें `FreeFloating` स्थिति निर्धारण पर अधिक नियंत्रण के लिए:

```csharp
oval1.Placement = PlacementType.FreeFloating;
```

##### चरण 5: लाइन गुण सेट करें

रेखा भार और डैश शैली निर्धारित करके अंडाकार की रूपरेखा के स्वरूप को अनुकूलित करें:

```csharp
// लाइन वजन और डैश शैली सेट करें
oval1.Line.Weight = 1;
oval1.Line.DashStyle = MsoLineDashStyle.Solid;
```

##### चरण 6: कार्यपुस्तिका सहेजें

अंत में, अपनी कार्यपुस्तिका को निर्दिष्ट निर्देशिका में एक फ़ाइल में सहेजें:

```csharp
excelbook.Save(Path.Combine(dataDir, "OvalShapeExample.xls"));
```

#### समस्या निवारण युक्तियों:
- फ़ाइल नहीं मिली त्रुटि को रोकने के लिए सुनिश्चित करें कि सभी निर्देशिका पथ सही ढंग से सेट किए गए हैं।
- यदि आप परीक्षण सीमाओं से परे सुविधाओं का उपयोग कर रहे हैं तो जांच लें कि Aspose.Cells उचित रूप से लाइसेंस प्राप्त है।

### एक और अंडाकार आकार (वृत्त) जोड़ना

अब आइए एक और अंडाकार आकृति जोड़ें, जिसे भिन्न गुणों के साथ वृत्त के रूप में कॉन्फ़िगर किया गया हो।

#### अवलोकन
कई आकृतियाँ जोड़ने से अधिक जटिल विज़ुअलाइज़ेशन बनाने में मदद मिल सकती है। यहाँ, हम आपके वर्कशीट में एक गोलाकार अंडाकार जोड़ना प्रदर्शित करेंगे।

#### चरण:

##### चरण 1: सुनिश्चित करें कि निर्देशिका मौजूद है

यह चरण पिछले अनुभाग के समान है; सुनिश्चित करें कि आपकी निर्देशिका सही ढंग से सेट की गई है।

```csharp
if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
```

##### चरण 2: कार्यपुस्तिका को इंस्टैंशिएट करें

एक नया बनाएँ `Workbook` इस आकृति जोड़ के लिए उदाहरण:

```csharp
Workbook excelbook = new Workbook();
```

##### चरण 3: वृत्त आकार जोड़ें

इसे वृत्त के रूप में दिखाने के लिए आयामों के साथ एक और अंडाकार जोड़ें:

```csharp
// विभिन्न निर्देशांकों और आकार पर एक गोलाकार आकृति जोड़ें
Oval oval2 = excelbook.Worksheets[0].Shapes.AddOval(9, 0, 2, 15, 130, 130);
```

##### चरण 4: प्लेसमेंट कॉन्फ़िगर करें

नये आकार के लिए प्लेसमेंट प्रकार सेट करें:

```csharp
oval2.Placement = PlacementType.FreeFloating;
```

##### चरण 5: लाइन गुण सेट करें

अनुकूलन के लिए लाइन वजन और डैश शैली परिभाषित करें:

```csharp
// लाइन गुण अनुकूलित करें
oval2.Line.Weight = 1;
oval2.Line.DashStyle = MsoLineDashStyle.Solid;
```

##### चरण 6: कार्यपुस्तिका को नए आकार के साथ सहेजें

कार्यपुस्तिका को पुनः सेव करें, इस बार दोनों आकृतियों को शामिल करें:

```csharp
excelbook.Save(Path.Combine(dataDir, "OvalShapeExampleWithCircle.xls"));
```
