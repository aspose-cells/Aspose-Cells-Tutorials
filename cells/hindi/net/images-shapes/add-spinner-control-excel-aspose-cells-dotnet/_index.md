## कार्यान्वयन मार्गदर्शिका

### टेक्स्ट जोड़ना और सेल को स्टाइल करना

स्पिनर नियंत्रण जोड़ने से पहले अपने कक्षों को लेबल के साथ तैयार करें।

#### चरण 1: लेबल और शैलियाँ इनपुट करें

**अवलोकन**स्पिनर नियंत्रण के लिए उपयोगकर्ता मार्गदर्शन लेबल के साथ अपनी एक्सेल शीट सेट करें।

```csharp
Cells cells = worksheet.Cells;

// A1 सेल में एक लेबल जोड़ें.
cells["A1"].PutValue("Select Value:");
Style style = cells["A1"].GetStyle();
style.Font.Color = Color.Red;
style.Font.IsBold = true;
cells["A1"].SetStyle(style);

// स्पिनर नियंत्रण के लिए लिंक किए गए सेल (A2) को तैयार करें।
cells["A2"].PutValue(0);
style = cells["A2"].GetStyle();
style.ForegroundColor = Color.Black;
style.Pattern = BackgroundType.Solid;
style.Font.Color = Color.White;
style.Font.IsBold = true;
cells["A2"].SetStyle(style);
```

#### चरण 2: स्पिनर नियंत्रण जोड़ें

**अवलोकन**: स्पिनर नियंत्रण को अपनी वर्कशीट में एकीकृत करें, इसे विशिष्ट डेटा से लिंक करें।

```csharp
// सेल A2 से जुड़ा स्पिनर नियंत्रण जोड़ना।
Aspose.Cells.Drawing.Spinner spinner = excelbook.Worksheets[0].Shapes.AddSpinner(1, 0, 1, 0, 20, 18);
spinner.Placement = PlacementType.FreeFloating;
spinner.LinkedCell = "A2";
spinner.Max = 10;
spinner.Min = 0;
spinner.IncrementalChange = 2;
spinner.Shadow = true;
```

### स्पष्टीकरण

- **प्लेसमेंट**स्पिनर को इस पर सेट किया गया है `FreeFloating`, लचीली स्थिति की अनुमति देता है।
- **लिंक्ड सेल**: स्पिनर को सेल A2 से जोड़ता है, जिससे यह सुनिश्चित होता है कि स्पिनर में हुए परिवर्तन इस सेल में प्रतिबिंबित हों।
- **रेंज और वृद्धि**: स्पिनर की रेंज को 0 से 10 तक 2 की वृद्धि के साथ कॉन्फ़िगर करता है।
