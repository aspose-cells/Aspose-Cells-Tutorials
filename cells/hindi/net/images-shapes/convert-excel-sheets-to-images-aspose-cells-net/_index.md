**आप क्या सीखेंगे:**
- किसी मौजूदा Excel कार्यपुस्तिका को कैसे लोड करें और खोलें
- किसी कार्यपुस्तिका के भीतर विशिष्ट कार्यपत्रकों तक पहुँचना
- रूपांतरण के लिए छवि प्रिंट विकल्प कॉन्फ़िगर करना
- Aspose.Cells .NET का उपयोग करके कार्यपत्रकों को छवियों के रूप में प्रस्तुत करना
- रेंडर की गई छवियों को कुशलतापूर्वक सहेजना

आइए देखें कि आप इस कार्यक्षमता का लाभ कैसे उठा सकते हैं, सबसे पहले अपने परिवेश को सेट अप करें।

## कार्यान्वयन मार्गदर्शिका

हम तार्किक अनुभागों का उपयोग करके एक्सेल शीट को छवियों में परिवर्तित करने की प्रत्येक सुविधा का विश्लेषण करेंगे।

### Excel कार्यपुस्तिका लोड करें और खोलें

**अवलोकन:**
हमारी प्रक्रिया का पहला चरण किसी निर्दिष्ट निर्देशिका से मौजूदा एक्सेल वर्कबुक को लोड करना है। यह हमें उस डेटा तक पहुँचने की अनुमति देता है जिसे हम छवियों में बदलना चाहते हैं।

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Excel फ़ाइल को वर्कबुक ऑब्जेक्ट में लोड करें
Workbook book = new Workbook(SourceDir + "sampleConvertWorksheettoImageFile.xlsx");
```

**स्पष्टीकरण:**
- `Workbook`संपूर्ण कार्यपुस्तिका का प्रतिनिधित्व करता है और इसके कार्यपत्रकों तक पहुंच प्रदान करता है।
- कन्स्ट्रक्टर एक्सेल फ़ाइल के पथ को तर्क के रूप में लेता है, तथा उसे मेमोरी में लोड करता है।

### कार्यपुस्तिका से कार्यपत्रक तक पहुँचना

**अवलोकन:**
कार्यपुस्तिका खोलने के बाद, हमें यह निर्दिष्ट करना होगा कि हम किस कार्यपत्रक को परिवर्तित करना चाहते हैं। यह अनुभाग कार्यपुस्तिका के भीतर एक विशिष्ट शीट तक पहुँचने का प्रदर्शन करता है।

```csharp
// Excel फ़ाइल को वर्कबुक ऑब्जेक्ट में खोलें
Workbook book = new Workbook(SourceDir + "sampleConvertWorksheettoImageFile.xlsx");

// कार्यपुस्तिका से प्रथम कार्यपत्रक तक पहुँचना
Worksheet sheet = book.Worksheets[0];
```

**स्पष्टीकरण:**
- `Worksheets`: के भीतर एक संग्रह `Workbook` जो सभी शीटों को संग्रहीत करता है।
- `sheet.Worksheets[0]`: कार्यपुस्तिका में प्रथम कार्यपत्रक (सूचकांक 0) को पुनः प्राप्त करता है।

### छवि प्रिंट विकल्प कॉन्फ़िगर करना

**अवलोकन:**
रेंडरिंग से पहले, हम कॉन्फ़िगर करते हैं कि वर्कशीट को इमेज में कैसे बदला जाएगा। इसमें आउटपुट फ़ॉर्मेट और पेज विकल्प सेट करना शामिल है।

```csharp
// रेंडरिंग के लिए छवि या प्रिंट विकल्प कॉन्फ़िगर करें
ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
imgOptions.OnePagePerSheet = true; // संपूर्ण वर्कशीट को एक पृष्ठ पर प्रस्तुत करें
imgOptions.ImageType = Drawing.ImageType.Jpeg; // आउटपुट छवि प्रकार को JPEG पर सेट करें
```

**स्पष्टीकरण:**
- `OnePagePerSheet`यह सुनिश्चित करता है कि संपूर्ण शीट एक ही छवि पर प्रस्तुत हो।
- `ImageType`: आउटपुट छवि का प्रारूप निर्दिष्ट करता है, इस मामले में, JPEG.

### वर्कशीट को छवि के रूप में प्रस्तुत करना

**अवलोकन:**
अब हम पहले से निर्धारित विकल्पों का उपयोग करके निर्दिष्ट वर्कशीट को एक छवि में परिवर्तित करते हैं।

```csharp
// वर्कशीट को छवि के रूप में प्रस्तुत करने के लिए SheetRender ऑब्जेक्ट बनाएँ
SheetRender sr = new SheetRender(sheet, imgOptions);
Bitmap bitmap = sr.ToImage(0); // शीट के पहले पृष्ठ को छवि में प्रस्तुत करें
```

**स्पष्टीकरण:**
- `SheetRender`: कार्यपत्रकों के लिए रेंडरिंग संचालन को संभालता है।
- `ToImage(int pageIndex)`: निर्दिष्ट कार्यपत्रक पृष्ठ को छवि में परिवर्तित करता है।

### रेंडर की गई छवि को सहेजना

**अवलोकन:**
अंत में, उत्पन्न छवि को अपनी इच्छित आउटपुट निर्देशिका में सहेजें।

```csharp
// रेंडर की गई छवि को आउटपुट निर्देशिका में सहेजें
bitmap.Save(outputDir + "outputConvertWorksheettoImageFile.jpg");
```

**स्पष्टीकरण:**
- `Save(string path)`: छवि फ़ाइल को निर्दिष्ट स्थान पर डिस्क पर लिखता है।
