**आप क्या सीखेंगे:**

## कार्यान्वयन मार्गदर्शिका

### साझा कार्यपुस्तिकाओं में इतिहास को संरक्षित करने के लिए दिन निर्धारित करना

साझा कार्यपुस्तिकाओं में, सहयोगात्मक संपादन के लिए संशोधनों का ट्रैक रखना आवश्यक है। Aspose.Cells के साथ, आप निर्दिष्ट कर सकते हैं कि इन लॉग को कितने समय तक संरक्षित किया जाना चाहिए।

#### साझा कार्यपुस्तिका बनाना और कॉन्फ़िगर करना

**चरण 1: एक खाली कार्यपुस्तिका बनाएँ**

```csharp
// एक नई कार्यपुस्तिका इंस्टेंस बनाएँ
Workbook wb = new Workbook();
```

**चरण 2: कार्यपुस्तिका साझा करें**

एकाधिक उपयोगकर्ताओं को संपादित करने की अनुमति देने के लिए साझाकरण सक्षम करें:

```csharp
// साझा सेटिंग सक्षम करें
wb.Settings.Shared = true;
```

**चरण 3: संशोधन लॉग का इतिहास संरक्षित करते हुए दिन अपडेट करें**

निर्दिष्ट करें कि आप संशोधन इतिहास को कितने दिनों तक संरक्षित रखना चाहते हैं:

```csharp
// संशोधन लॉग को संरक्षित करने के लिए दिन निर्धारित करें
wb.Worksheets.RevisionLogs.DaysPreservingHistory = 7;
```

यह सेटिंग सुनिश्चित करती है कि केवल पिछले सात दिनों के परिवर्तन ही रिकॉर्ड किए जाएं, जिससे आपके लॉग संक्षिप्त और प्रासंगिक बने रहें।

**चरण 4: कार्यपुस्तिका सहेजें**

अंत में, अद्यतन सेटिंग्स के साथ कार्यपुस्तिका को सहेजें:

```csharp
// आउटपुट निर्देशिका परिभाषित करें
string outputDir = RunExamples.Get_OutputDirectory();

// फ़ाइल सहेजें
wb.Save(outputDir + "outputShared_DaysPreservingHistory.xlsx");
```

#### समस्या निवारण युक्तियों

- **सुनिश्चित करें कि कार्यपुस्तिका साझा की गई है**: यदि परिवर्तन प्रतिबिंबित नहीं होते हैं, तो सत्यापित करें `wb.Settings.Shared` सत्य पर सेट है.
- **दिनों का मूल्य जांचें**: सुनिश्चित करना `DaysPreservingHistory` एक धनात्मक पूर्णांक है.
