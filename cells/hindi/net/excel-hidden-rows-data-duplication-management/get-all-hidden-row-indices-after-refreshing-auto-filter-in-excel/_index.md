# एक्सेल में ऑटो फ़िल्टर को रिफ्रेश करने के बाद छुपे हुए पंक्ति सूचकांक प्राप्त करें

## चरण 1: अपना वातावरण सेट करें

कोडिंग में गोता लगाने से पहले, आइए अपना वातावरण सेट करें और आवश्यक चर घोषित करें। यह सेटअप सब कुछ आपकी सैंपल एक्सेल फ़ाइल में निर्देशित करेगा और वर्कबुक तैयार करेगा।

```csharp
string sourceDir = "Your Document Directory"; // अपनी निर्देशिका निर्दिष्ट करें
```

## चरण 2: नमूना एक्सेल फ़ाइल लोड करें

इसके बाद, हमें आपकी एक्सेल फ़ाइल को वर्कबुक ऑब्जेक्ट में लोड करना होगा। इससे हम इसे प्रोग्रामेटिक रूप से मैनिपुलेट कर सकते हैं। 

```csharp
Workbook wb = new Workbook(sourceDir + "sampleGetAllHiddenRowsIndicesAfterRefreshingAutoFilter.xlsx");
```

यहाँ, हम एक नया निर्माण कर रहे हैं `Workbook` ऑब्जेक्ट जो निर्दिष्ट एक्सेल फ़ाइल को लोड करता है.

## चरण 3: इच्छित वर्कशीट तक पहुंचें

अब, हम वर्कबुक की पहली वर्कशीट के साथ काम करने जा रहे हैं। यह चरण उस शीट को अलग करता है जिसमें वह डेटा है जिसे हम फ़िल्टर करना चाहते हैं।

```csharp
Worksheet ws = wb.Worksheets[0]; // प्रथम कार्यपत्रक तक पहुँचना
```

## चरण 4: ऑटो फ़िल्टर लागू करें

ऑटो फ़िल्टर लागू करना वह जगह है जहाँ जादू शुरू होता है! हम निर्दिष्ट करेंगे कि हम किस कॉलम को फ़िल्टर करना चाहते हैं और अपना मानदंड निर्धारित करेंगे। यहाँ, हम "नारंगी" के लिए फ़िल्टर कर रहे हैं। 

```csharp
ws.AutoFilter.AddFilter(0, "Orange"); // पहले कॉलम के लिए ऑटोफ़िल्टर लागू करें
```

## चरण 5: ऑटो फ़िल्टर को रिफ्रेश करें और छिपी हुई पंक्तियाँ प्राप्त करें

निम्न पंक्ति ऑटो फ़िल्टर को रिफ्रेश करती है। यह उन पंक्तियों के इंडेक्स लौटाएगी जो हमारे फ़िल्टर को लागू करने के बाद छिपी हुई हैं। पैरामीटर को true पर सेट करने से फ़िल्टर प्रभावी रूप से रिफ्रेश हो जाता है।

```csharp
int[] rowIndices = ws.AutoFilter.Refresh(true);
```

## चरण 6: छिपी हुई पंक्ति अनुक्रमणिका प्रिंट करें

अब जब हमारे पास छिपी हुई पंक्ति अनुक्रमणिकाएँ हैं, तो उन्हें कंसोल पर आउटपुट करें। इससे यह स्पष्ट हो जाएगा कि हमारे ऑटो फ़िल्टर के कारण क्या छिपा हुआ था।

```csharp
Console.WriteLine("Printing Rows Indices, Cell Names and Values Hidden By AutoFilter.");
Console.WriteLine("--------------------------");

for (int i = 0; i < rowIndices.Length; i++)
{
    int r = rowIndices[i];
    Cell cell = ws.Cells[r, 0];
    Console.WriteLine(r + "\t" + cell.Name + "\t" + cell.StringValue);
}

Console.WriteLine("GetAllHiddenRowsIndicesAfterRefreshingAutoFilter executed successfully.");
```
