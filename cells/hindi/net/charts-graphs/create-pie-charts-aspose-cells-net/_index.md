## कार्यान्वयन मार्गदर्शिका

### कार्यपुस्तिका और कार्यपत्रक बनाना
1. **कार्यपुस्तिका आरंभ करें**
   XLSX प्रारूप में एक नई कार्यपुस्तिका बनाएँ:
   ```csharp
   Workbook workbook = new Workbook(FileFormatType.Xlsx);
   ```

2. **प्रथम वर्कशीट तक पहुँचना**
   डेटा इनपुट करने के लिए पहली वर्कशीट का उपयोग करें:
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **पाई चार्ट के लिए डेटा जोड़ना**
   अपनी वर्कशीट को श्रेणियों और मानों से भरें:
   ```csharp
   worksheet.Cells["A1"].PutValue("Retail");
   // शेष श्रेणी नाम जोड़ें...
   worksheet.Cells["B1"].PutValue(10.4);
   // संगत मान जोड़ें...
   ```

### वर्कशीट में पाई चार्ट जोड़ना
1. **पाई चार्ट बनाएं**
   एक पाई चार्ट बनाएं और उसे अपने वर्कशीट के चार्ट संग्रह में जोड़ें:
   ```csharp
   int id = worksheet.Charts.Add(ChartType.Pie, 3, 3, 23, 13);
   ```

2. **श्रृंखला और श्रेणियाँ डेटा कॉन्फ़िगर करें**
   श्रृंखला और श्रेणियों के लिए डेटा लिंक करें:
   ```csharp
   Chart chart = worksheet.Charts[id];
   chart.NSeries.Add("B1:B16", true);
   chart.NSeries.CategoryData = "A1:A16";
   ```

3. **डेटा लेबल अनुकूलित करें**
   लेजेंड प्रदर्शन बंद करें, श्रेणी नाम और प्रतिशत दिखाने के लिए डेटा लेबल सेट करें:
   ```csharp
   chart.ShowLegend = false;
   DataLabels dataLabels = chart.NSeries[0].DataLabels;
   dataLabels.ShowCategoryName = true;
   dataLabels.ShowPercentage = true;
   dataLabels.Position = LabelPositionType.OutsideEnd;
   ```

### लीडर लाइन्स का क्रियान्वयन
1. **लीडर लाइन चालू करें**
   स्पष्ट दृश्य कनेक्शन के लिए लीडर लाइन सक्षम करें:
   ```csharp
   chart.NSeries[0].HasLeaderLines = true;
   ```

2. **डेटा लेबल की स्थिति समायोजित करें**
   लेबल की स्थिति समायोजित करके दृश्यता सुनिश्चित करें:
   ```csharp
   int DELTA = 100;
   foreach (var point in chart.NSeries[0].Points)
   {
       int X = point.DataLabels.X;
       if (X > 2000) 
           point.DataLabels.X += DELTA;
       else 
           point.DataLabels.X -= DELTA;
   }
   ```

### चार्ट और कार्यपुस्तिका को सहेजना
1. **छवि के रूप में सहेजें**
   चार्ट को छवि फ़ाइल में प्रस्तुत करें:
   ```csharp
   ImageOrPrintOptions options = new ImageOrPrintOptions { ImageType = Drawing.ImageType.Png, HorizontalResolution = 200, VerticalResolution = 200 };
   chart.ToImage("output_out.png", options);
   ```

2. **कार्यपुस्तिका सहेजें**
   Excel में चार्ट देखने के लिए कार्यपुस्तिका सहेजें:
   ```csharp
   workbook.Save("output_out.xlsx");
   ```
