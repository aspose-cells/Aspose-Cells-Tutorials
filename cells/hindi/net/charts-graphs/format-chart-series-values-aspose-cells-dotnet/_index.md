# Aspose.Cells .NET का उपयोग करके Excel में चार्ट श्रृंखला मानों को कैसे प्रारूपित करें

## परिचय

क्या आपको Excel में चार्ट श्रृंखला मानों को प्रोग्रामेटिक रूप से फ़ॉर्मेट करने की आवश्यकता है? यह ट्यूटोरियल चार्ट श्रृंखला के लिए फ़ॉर्मेट कोड सेट करने के लिए .NET के लिए Aspose.Cells का उपयोग करके प्रदर्शित करता है। चाहे रिपोर्ट जनरेशन को स्वचालित करना हो या वित्तीय प्रस्तुतियों को मानकीकृत करना हो, मूल्य फ़ॉर्मेट को नियंत्रित करने से डेटा की पठनीयता और संगति में बहुत सुधार हो सकता है।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Cells को स्थापित और आरंभ करना
- कार्यपुस्तिका लोड करना और उसके घटकों जैसे कार्यपत्रकों और चार्टों तक पहुँचना
- चार्ट में श्रृंखला जोड़ना और उनके मान प्रारूप कोड सेट करना
- परिवर्तनों को वापस Excel फ़ाइल में सहेजना

सबसे पहले, आइए पूर्वापेक्षाओं की समीक्षा करें।

## कार्यान्वयन मार्गदर्शिका

आइये, आसान कार्यान्वयन के लिए इस प्रक्रिया को अलग-अलग चरणों में विभाजित करें।

### निर्देशिका से कार्यपुस्तिका लोड करें

**अवलोकन:** अपनी निर्दिष्ट निर्देशिका से एक्सेल कार्यपुस्तिका लोड करके प्रारंभ करें।

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
// स्रोत एक्सेल फ़ाइल लोड करें 
Workbook wb = new Workbook(SourceDir + "sampleSeries_ValuesFormatCode.xlsx");
```

**स्पष्टीकरण:**
- `SourceDir` यह आपके इनपुट फ़ाइलों का पथ है.
- The `Workbook` कन्स्ट्रक्टर निर्दिष्ट फ़ाइल को खोलता है.

### कार्यपुस्तिका से कार्यपत्रक तक पहुँचें

**अवलोकन:** वह वर्कशीट पुनः प्राप्त करें जिस पर आपको काम करना है।

```csharp
// पहली वर्कशीट तक पहुंचें
Worksheet worksheet = wb.Worksheets[0];
```

**स्पष्टीकरण:**
- वर्कबुक में कई वर्कशीट हो सकती हैं। यहाँ, हम इंडेक्स का उपयोग करके पहले वाले तक पहुँचते हैं `0`.

### वर्कशीट से चार्ट एक्सेस करें

**अवलोकन:** हेरफेर करने के लिए अपने चयनित वर्कशीट के भीतर चार्ट का पता लगाएं।

```csharp
// पहले चार्ट तक पहुंचें
Chart ch = worksheet.Charts[0];
```

**स्पष्टीकरण:**
- वर्कशीट की तरह, एक वर्कशीट में कई चार्ट हो सकते हैं। यह कोड पहले चार्ट तक पहुँचता है।

### चार्ट में श्रृंखला जोड़ें

**अवलोकन:** मानों की एक सरणी का उपयोग करके अपने चार्ट में डेटा श्रृंखला जोड़ें.

```csharp
// मानों की सरणी का उपयोग करके श्रृंखला जोड़ें
ch.NSeries.Add("{10000, 20000, 30000, 40000}", true);
```

**स्पष्टीकरण:**
- `NSeries.Add` संख्याओं का एक स्ट्रिंग प्रतिनिधित्व और एक बूलियन लेता है जो यह बताता है कि क्या सीमा अनन्य है। यहाँ, यह समावेशी है।

### श्रृंखला मान प्रारूप कोड सेट करें

**अवलोकन:** अपने चार्ट श्रृंखला में मानों को प्रारूपित करने का तरीका अनुकूलित करें.

```csharp
// श्रृंखला तक पहुंचें और इसके मान प्रारूप कोड सेट करें
Series srs = ch.NSeries[0];
srs.ValuesFormatCode = "$#,##0";
```

**स्पष्टीकरण:**
- `ValuesFormatCode` आपको कस्टम संख्या प्रारूप परिभाषित करने की अनुमति देता है, जैसे इस उदाहरण में मुद्रा (`"$#,##0"`).

### कार्यपुस्तिका को निर्देशिका में सहेजें

**अवलोकन:** कार्यपुस्तिका को आउटपुट निर्देशिका में सहेजकर अपने परिवर्तनों को बनाए रखें।

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
// आउटपुट एक्सेल फ़ाइल को सहेजें
wb.Save(outputDir + "outputSeries_ValuesFormatCode.xlsx");
```

**स्पष्टीकरण:**
- The `Save` विधि आपके परिवर्तनों को संरक्षित करते हुए संशोधित कार्यपुस्तिका को एक नई फ़ाइल में लिखती है।

## व्यावहारिक अनुप्रयोगों

यहां कुछ परिदृश्य दिए गए हैं जहां यह कार्यक्षमता उपयोगी है:
1. **वित्तीय रिपोर्टिंग:** वित्तीय डैशबोर्ड के लिए चार्ट में मुद्रा मानों को स्वचालित रूप से प्रारूपित करें।
2. **स्वचालित डेटा विश्लेषण:** कच्चे डेटासेट से उत्पन्न कई एक्सेल रिपोर्टों में डेटा प्रस्तुति को मानकीकृत करें।
3. **शैक्षिक उपकरण:** सुसंगत स्वरूपित डेटा विज़ुअलाइज़ेशन के साथ अनुदेशात्मक सामग्री बनाएं।
