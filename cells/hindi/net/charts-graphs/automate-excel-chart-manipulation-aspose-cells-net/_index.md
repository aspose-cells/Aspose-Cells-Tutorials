## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में, हम Excel 2016 फ़ाइल में चार्ट को पढ़ने और उसमें बदलाव करने के लिए आवश्यक चरणों पर चर्चा करेंगे।

### वर्कशीट में चार्ट तक पहुँचना

हम अपनी स्रोत कार्यपुस्तिका को लोड करके और इसकी पहली वर्कशीट तक पहुंचकर शुरुआत करते हैं, जिसमें हमारे चार्ट होते हैं:

```csharp
// एक्सेल फ़ाइल लोड करें
Workbook book = new Workbook("sampleReadAndManipulateExcel2016Charts.xlsx");

// पहली वर्कशीट तक पहुँचें
Worksheet sheet = book.Worksheets[0];
```

### रीडिंग चार्ट के प्रकार

इसके बाद, हम वर्कशीट में प्रत्येक चार्ट को दोहराकर उसका प्रकार पढ़ते हैं और उसका प्रिंट निकालते हैं:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    // वर्तमान चार्ट प्राप्त करें
    Chart ch = sheet.Charts[i];

    // चार्ट प्रकार प्रिंट करें
    Console.WriteLine(ch.Type);
}
```

### चार्ट शीर्षक संशोधित करना

हम प्रत्येक चार्ट के शीर्षक को उसके प्रकार के अनुसार बदल सकते हैं:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    Chart ch = sheet.Charts[i];

    // चार्ट शीर्षक अपडेट करें
    ch.Title.Text = "Chart Type is " + ch.Type.ToString();
}
```

### परिवर्तन सहेजना

अंत में, अपने संशोधनों को एक नई एक्सेल फ़ाइल में सहेजें:

```csharp
book.Save("outputReadAndManipulateExcel2016Charts.xlsx");
Console.WriteLine("Manipulation completed successfully.");
```
