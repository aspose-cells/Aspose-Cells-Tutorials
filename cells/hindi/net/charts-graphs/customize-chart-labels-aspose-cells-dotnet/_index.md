## कार्यान्वयन मार्गदर्शिका
### लोकेल के आधार पर चार्ट लेबल को अनुकूलित करना
प्राथमिक उद्देश्य संस्कृति-विशिष्ट सेटिंग्स का उपयोग करके पाई चार्ट लेबल के लिए डिफ़ॉल्ट टेक्स्ट को ओवरराइड करना है। यहां बताया गया है कि आप इसे कैसे प्राप्त कर सकते हैं:

#### 1. अपनी कार्यपुस्तिका लोड करें और चार्ट तक पहुँचें
पाई चार्ट वाली मौजूदा एक्सेल फ़ाइल लोड करके प्रारंभ करें:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleCustomTextForLabels.xlsx");
```

उस वर्कशीट और चार्ट तक पहुंचें जिसे आप अनुकूलित करना चाहते हैं:
```csharp
Worksheet sheet = book.Worksheets[0];
Chart chart = sheet.Charts[0];
```

#### 2. वैश्वीकरण सेटिंग्स सेट करें
ओवरराइड करें `GetOtherName` सिस्टम के लोकेल के आधार पर कस्टम लेबल प्रदान करने की विधि:

```csharp
GlobalizationSettings globalSettings = new GlobalizationSettings();
globalSettings.ChartSettings = new CustomSettings();
book.Settings.GlobalizationSettings = globalSettings;
```

अपना कस्टम सेटिंग वर्ग परिभाषित करें:
```csharp
class CustomSettings : ChartGlobalizationSettings
{
    public override string GetOtherName()
    {
        int lcid = System.Globalization.CultureInfo.CurrentCulture.LCID;
        switch (lcid)
        {
            case 1033: // अंग्रेज़ी
                return "Other";
            case 1036: // फ्रेंच
                return "Autre";
            case 1031: // जर्मन
                return "Andere";
            default:
                return base.GetOtherName();
        }
    }
}
```

#### 3. चार्ट को रिफ्रेश और रेंडर करें
अपने परिवर्तनों को लागू करने के लिए, चार्ट को रिफ्रेश करें और उसे एक छवि फ़ाइल में प्रस्तुत करें:

```csharp
chart.Calculate();
chart.ToImage(outputDir + "outputCustomTextForLabels.png", new ImageOrPrintOptions());
Console.WriteLine("CustomTextForLabels executed successfully.");
```

- **संस्कृति बेमेल**सत्यापित करें कि आपके सिस्टम की स्थानीय सेटिंग आपके लक्ष्यित स्थानीय सेटिंग से मेल खाती है।
