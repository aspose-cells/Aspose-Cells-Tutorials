## कार्यान्वयन मार्गदर्शिका

पिरामिड चार्ट बनाने के लिए इन चरणों का पालन करें।

### कार्यपुस्तिका और कार्यपत्रक आरंभ करें

**अवलोकन:**
हम एक्सेल वर्कबुक बनाकर और उसकी पहली वर्कशीट तक पहुंचकर शुरुआत करेंगे।

#### चरण 1: कार्यपुस्तिका इंस्टेंस बनाएँ

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// एक नई कार्यपुस्तिका ऑब्जेक्ट आरंभ करें
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

### कक्षों में नमूना डेटा जोड़ें

**अवलोकन:**
इसके बाद, हमारे चार्ट के लिए नमूना डेटा के साथ वर्कशीट भरें।

#### चरण 2: कोशिकाओं को भरें

```csharp
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```

### वर्कशीट में पिरामिड चार्ट जोड़ें

**अवलोकन:**
अब, डेटा को देखने के लिए एक पिरामिड चार्ट जोड़ें।

#### चरण 3: पिरामिड चार्ट डालें

```csharp
using Aspose.Cells.Charts;

// वर्कशीट में पिरामिड चार्ट जोड़ें
int chartIndex = worksheet.Charts.Add(ChartType.Pyramid, 5, 0, 25, 10);
Chart chart = worksheet.Charts[chartIndex];
```

### चार्ट डेटा स्रोत सेट करें

**अवलोकन:**
परिभाषित करें कि हमारे पिरामिड चार्ट के लिए कौन सी डेटा रेंज का उपयोग किया जाएगा।

#### चरण 4: चार्ट डेटा कॉन्फ़िगर करें

```csharp
// चार्ट के लिए डेटा स्रोत श्रेणी सेट करें
chart.NSeries.Add("A1:B3", true);
```

### कार्यपुस्तिका को फ़ाइल में सहेजें

**अवलोकन:**
अंत में, अपनी कार्यपुस्तिका को नए बनाए गए पिरामिड चार्ट के साथ सेव करें।

workbook.Save(outputDir + "outputHowToCreatePyramidChart.xlsx");
