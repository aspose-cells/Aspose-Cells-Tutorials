## कार्यान्वयन मार्गदर्शिका
हम प्रमुख विशेषताओं का पता लगाएंगे: कार्यपुस्तिका मैनिपुलेशन और फ़ाइल पथ प्रबंधन।
### कार्यपुस्तिका हेरफेर
एक्सेल फ़ाइल खोलने, वर्कशीट में पंक्तियाँ डालने, और संशोधित वर्कबुक को सहेजने पर ध्यान केंद्रित करें।
#### चरण 1: FileStream का उपयोग करके मौजूदा Excel फ़ाइल खोलें
मौजूदा Excel फ़ाइल को खोलें `FileStream`, सीधे पढ़ने या लिखने की अनुमति देता है:
```csharp
// स्रोत एक्सेल फ़ाइल खोलें
FileStream fstream = new FileStream(dataDir, FileMode.Open);
```
#### चरण 2: फ़ाइल स्ट्रीम से वर्कबुक ऑब्जेक्ट बनाएँ
एक बनाने के `Workbook` मेमोरी में संपूर्ण Excel कार्यपुस्तिका का प्रतिनिधित्व करने के लिए ऑब्जेक्ट:
```csharp
// फ़ाइल स्ट्रीम का उपयोग करके कार्यपुस्तिका लोड करें
Workbook workbook = new Workbook(fstream);
```
#### चरण 3: कार्यपुस्तिका में प्रथम कार्यपत्रक तक पहुँचें
विशिष्ट कार्यपत्रकों तक पहुँच कर संशोधनों को सटीक रूप से लक्षित करें:
```csharp
// कार्यपुस्तिका से पहली कार्यपत्रिका प्राप्त करें
Worksheet worksheet = workbook.Worksheets[0];
```
#### चरण 4: वर्कशीट में पंक्तियाँ डालें
निर्दिष्ट इंडेक्स पर एकाधिक पंक्तियाँ डालें, विद्यमान डेटा को अधिलेखित किए बिना उसे नीचे ले जाएँ:
```csharp
// पंक्ति सूचकांक 2 (तीसरी पंक्ति) से शुरू करके 10 पंक्तियाँ डालें
worksheet.Cells.InsertRows(2, 10);
```
#### चरण 5: संशोधित एक्सेल फ़ाइल को नए स्थान पर सहेजें
अपने परिवर्तनों को एक नए फ़ाइल स्थान पर सहेजें, मूल डेटा को सुरक्षित रखें और संशोधनों को अलग से संग्रहीत करें:
```csharp
// संशोधित कार्यपुस्तिका को आउटपुट निर्देशिका में सहेजें
workbook.Save(outputFilePath);
```
#### चरण 6: संसाधन जारी करने के लिए फ़ाइलस्ट्रीम बंद करें
सिस्टम संसाधनों को खाली करने के लिए ऑपरेशन के बाद हमेशा स्ट्रीम बंद करें:
```csharp
// फ़ाइल स्ट्रीम बंद करेंstream.Close();
```
