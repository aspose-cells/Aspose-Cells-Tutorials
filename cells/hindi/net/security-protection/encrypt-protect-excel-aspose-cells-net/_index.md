### .NET के लिए Aspose.Cells के साथ Excel फ़ाइलों को एन्क्रिप्ट करना
**अवलोकन:**
अनधिकृत पहुँच से संवेदनशील जानकारी की सुरक्षा के लिए अपनी Excel फ़ाइलों को एन्क्रिप्ट करें। यह अनुभाग दर्शाता है कि Aspose.Cells का उपयोग करके विभिन्न एन्क्रिप्शन प्रकारों को कैसे लागू किया जाए।

#### चरण 1: अपना प्रोजेक्ट सेट करें और कार्यपुस्तिका लोड करें
```csharp
// सुनिश्चित करें कि आपने अपने वातावरण में इन निर्देशिका पथों को सही ढंग से सेट किया है।
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

Workbook workbook = new Workbook(SourceDir + "/Book1.xls");
```

#### चरण 2: एन्क्रिप्शन विकल्प निर्दिष्ट करें
XOR और सशक्त क्रिप्टोग्राफ़िक प्रदाता एन्क्रिप्शन प्रकारों के बीच चयन करें:
```csharp
// 40 की कुंजी लंबाई के साथ XOR एन्क्रिप्शन का उपयोग करें।
workbook.SetEncryptionOptions(EncryptionType.XOR, 40);

// वैकल्पिक रूप से, 128-बिट कुंजी लंबाई के साथ मजबूत RC4 एन्क्रिप्शन का उपयोग करें।
workbook.SetEncryptionOptions(EncryptionType.StrongCryptographicProvider, 128);
```

#### चरण 3: फ़ाइल पासवर्ड सेट करें
```csharp
// पासवर्ड सेट करके अपनी एक्सेल फ़ाइल को सुरक्षित रखें।
workbook.Settings.Password = "1234";
```

#### चरण 4: एन्क्रिप्टेड कार्यपुस्तिका को सहेजें
```csharp
// अपनी एन्क्रिप्टेड कार्यपुस्तिका को आउटपुट निर्देशिका में सहेजें.
workbook.Save(OutputDir + "/encryptedBook1.out.xls");
```

### Aspose.Cells के साथ संशोधन के लिए पासवर्ड सुरक्षा
**अवलोकन:**
संपादन के लिए आवश्यक पासवर्ड सेट करके अनधिकृत संशोधनों को रोकें।

#### चरण 1: मौजूदा कार्यपुस्तिका लोड करें
```csharp
Workbook workbook = new Workbook(SourceDir + "/Book1.xls");
```

#### चरण 2: लेखन-सुरक्षा पासवर्ड सेट करें
```csharp
// एक्सेल फ़ाइल को संशोधित करने के लिए आवश्यक पासवर्ड निर्धारित करें।
workbook.Settings.WriteProtection.Password = "1234";
```

#### चरण 3: संरक्षित कार्यपुस्तिका को सहेजें
```csharp
// अपनी कार्यपुस्तिका को संशोधन सुरक्षा सक्षम करके सहेजें.
workbook.Save(OutputDir + "/SpecifyPasswordToModifyOption.out.xls");
```
