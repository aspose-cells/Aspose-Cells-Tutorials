## कार्यान्वयन मार्गदर्शिका

### कार्यपुस्तिकाएँ बनाना और कॉन्फ़िगर करना (विशेषता 1)

यह सुविधा दर्शाती है कि नई कार्यपुस्तिका कैसे बनाई जाए और कार्यपत्रक शैलियाँ कैसे सेट की जाएं।

#### अवलोकन
एक्सेल फ़ाइलों को प्रोग्रामेटिक रूप से प्रबंधित करने में वर्कबुक बनाना पहला कदम है। आप इसे स्टाइल लागू करके, सेल लॉक करके या सुरक्षा स्तर सेट करके कॉन्फ़िगर कर सकते हैं।

#### चरण-दर-चरण कार्यान्वयन

##### नई कार्यपुस्तिका बनाएँ

आरंभ करके प्रारंभ करें `Workbook` वस्तु:

```csharp
// एक नई कार्यपुस्तिका आरंभ करें.
Workbook wb = new Workbook();
```

##### पहली वर्कशीट प्राप्त करें

संशोधन शुरू करने के लिए पहले वर्कशीट पर पहुँचें:

```csharp
// पहली वर्कशीट प्राप्त करें.
Worksheet sheet = wb.Worksheets[0];
```

##### शैलियाँ लागू करें और कॉलम अनलॉक करें

स्तंभों को अनलॉक करने के लिए शैलियाँ परिभाषित करें और लागू करें, जिससे आपकी कार्यपुस्तिका डिज़ाइन में लचीलापन सुनिश्चित हो:

```csharp
Style style = new Style { IsLocked = false };
StyleFlag styleflag = new StyleFlag { Locked = true };

// सभी कॉलम अनलॉक करें.
for (int i = 0; i <= 255; i++) {
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, styleflag);
}
```

##### विशिष्ट कक्षों को लॉक करें

संवेदनशील जानकारी की सुरक्षा के लिए विशिष्ट कक्षों को लॉक करें:

```csharp
sheet.Cells["A1"].SetStyle(new Style { IsLocked = true });
sheet.Cells["B1"].SetStyle(new Style { IsLocked = true });
sheet.Cells["C1"].SetStyle(new Style { IsLocked = true });
```

##### वर्कशीट को सुरक्षित रखें

अंत में, अपने डेटा को सुरक्षित करने के लिए वर्कशीट सुरक्षा लागू करें:

```csharp
// पूर्ण सुरक्षा लागू करें.
sheet.Protect(ProtectionType.All);

// कार्यपुस्तिका सहेजें.
wb.Save(outputDir + "/output.xls", SaveFormat.Excel97To2003);
```

### कोशिकाओं को लॉक करना और अनलॉक करना (फीचर 2)

यह सुविधा बताती है कि वर्कशीट के भीतर चुनिंदा कक्षों को कैसे लॉक या अनलॉक किया जाए।

#### अवलोकन
सेल एक्सेस को नियंत्रित करके, आप डेटा अखंडता का प्रबंधन कर सकते हैं और आवश्यकतानुसार संशोधन की अनुमति दे सकते हैं।

#### चरण-दर-चरण कार्यान्वयन

##### प्रारंभ में सभी कॉलम अनलॉक करें

अधिकतम लचीलेपन के लिए सभी कॉलम अनलॉक करके शुरुआत करें:

```csharp
Style unlockStyle = new Style { IsLocked = false };
StyleFlag unlockStyleFlag = new StyleFlag { Locked = true };

// सभी स्तंभों पर अनलॉक शैली लागू करें.
for (int i = 0; i <= 255; i++) {
    sheet.Cells.Columns[(byte)i].ApplyStyle(unlockStyle, unlockStyleFlag);
}
```

##### विशिष्ट कक्षों को लॉक करें

विशेष कक्षों को लॉक करने के लिए शैलियाँ परिभाषित करें और लागू करें:

```csharp
Style lockStyle = new Style { IsLocked = true };

// विशिष्ट कोशिकाओं को लॉक करें.
sheet.Cells["A1"].SetStyle(lockStyle);
sheet.Cells["B1"].SetStyle(lockStyle);
sheet.Cells["C1"].SetStyle(lockStyle);

// संशोधित कार्यपुस्तिका को सहेजें.
wb.Save(outputDir + "/output_locked.xls", SaveFormat.Excel97To2003);
```
