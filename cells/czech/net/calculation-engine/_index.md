# Výpočtový engine Excel pro aplikace .NET

## Přehled

Aspose.Cells pro .NET nabízí robustní výpočetní engine, který umožňuje vývojářům zpracovávat složité výpočty vzorců v tabulkách aplikace Excel. Tato výkonná komponenta umožňuje konfigurovat chování výpočtů, implementovat vlastní výpočetní logiku a optimalizovat výkon pro velké sešity.

## Klíčové vlastnosti

- **Vlastní výpočetní nástroje**Implementujte si vlastní výpočetní logiku pro specializované vzorce
- **Monitorování výpočtů**Výpočty vzorců pro sledování a řízení pro optimalizaci výkonu
- **Možnosti rekurzivního výpočtu**: Konfigurace způsobu zpracování cyklických odkazů
- **Konfigurace systému data**Přepínání mezi datovými systémy 1900 a 1904
- **Detekce kruhových referencí**Identifikace a řešení cyklických závislostí ve vzorcích
- **Optimalizace výkonu**Doladění nastavení výpočtů pro velké sešity
