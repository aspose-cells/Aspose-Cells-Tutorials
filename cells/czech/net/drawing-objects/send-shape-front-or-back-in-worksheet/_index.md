Tento příklad ukazuje, jak používat Aspose.Cells pro .NET k načítání Excelové pracovní knihy, manipulovat s pozicemi tvarů v Z-pořadí v pracovním listu a ukládat aktualizované pracovní knižku.

## Příkazy na instalaci  
1. Ujistěte se, že máte nastavené vývojové prostředí .NET, jako je Visual Studio.  
2. Instalujte Aspose.Cells pro .NET prostřednictvím programu NuGet Package Manager. Použijte následující příkaz v konzole programu Package Manager:  
   ```
   Install-Package Aspose.Cells
   ```
3. Přidejte odkaz na název Aspose.Cells do projektu:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Drawing; 
   ```
4. Připravte zdrojový soubor Excel a aktualizujte `zdrojDir`a `VýstupDir`cesty v uvedených příkladech kódu.  
5. Kopírovat a vložit poskytnutý snippet kódu do projektu, abyste manipulovali s Z řádem tvarů v souboru Excel.  
6. Vytvořte a spuštějte aplikaci, abyste viděli změny, které se odrážejí v výstupním souboru Excel.
