# Vytvořte a ukrýt soubory Excel pomocí Aspose.Cells
Tento tutoriál ukazuje, jak využít Aspose.Cells pro .NET k programové tvorbě nového souboru Excel. Naučíte se ověřovat existenci adresáře, instanciovat objekt Workbook, přidat pracovní list, vložit obrázek do konkrétní buňky a uložit soubor na požadovanou cestu.

## Příkazy na instalaci
1. Instalujte Aspose.Cells: Ujistěte se, že máte nainstalovanou knihovnu Aspose.Cells. Můžete to udělat pomocí NuGet Package Manager:
   ```
   Install-Package Aspose.Cells
   ```
2. Nastavte prostředí: Vytvořte adresář na počítači, kde chcete uložit vytvořené soubory Excel.`"Váš adresář dokumentů"`v příkladu s vaší skutečnou adresářovou cestou.
3. Implementovat kód: Kopírovat poskytnutý příkladový kód do aplikace .NET. Ujistěte se, že aktualizujete cestu obrazu (`dataDir + "logo.jpg"`) ukazovat na existující obrázek, který chcete vložit do souboru Excel.
4. Vypravte aplikaci: Vypravte aplikaci. Kód vytvoří nový soubor Excel s názvem `output.xls`v uvedeném seznamu s obrázkem umístěným v buňce "F6".
