# Vytvořte Excel Pracovní knižku s Far East a Latinsky písmen

Tento příklad ukazuje, jak používat Aspose.Cells pro .NET k vytvoření pracovního seznamu Excel, přidat textové skříňky do pracovního listu a nastavit jak Daleký východ, tak latinské názvy písmen pro text v této textové listu. Kód zobrazuje inicializaci pracovního prostoru, manipulaci s prvky pracovního listu a ukládání pracovního listu s specifickým formátováním písmen přizpůsobeným různým jazykovým požadavkům.

## Příkazy na instalaci
1. Instalujte Aspose.Cells pro .NET: 
   * Použijte v projektu Visual Studio NuGet Package Manager:
     ```
     Install-Package Aspose.Cells
     ```
   * Nebo stahujte DLL z [Webové stránky](https://www.aspose.com/downloads/cells/net)\- Ne , ne .

2. Přidání odkazů: Ujistěte se, že vaše projektová odkaza je v knihovně Aspose.Cells.

3. Nastavte výstupní adresář: Aktualizujte `VýstupDir`Vyměnná v kódu s požadovanou výstupní cestou, kde bude ukládán soubor Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
