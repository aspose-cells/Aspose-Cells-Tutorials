# Změna velikosti a umístění grafu v Excelu pomocí Aspose.Cells 
Aspose.Cells pro .NET poskytuje výkonné API pro programovou práci s excelovými soubory. Tento příklad ukazuje, jak změnit velikost a umístění grafu v excelovém sešitu pomocí Aspose.Cells. Ať už potřebujete upravit rozměry grafu nebo jej přesunout na konkrétní místo, tento tutoriál vás provede celým procesem pomocí snadno srozumitelného kódu C#. Využitím Aspose.Cells můžete bezproblémově manipulovat s excelovými grafy a zajistit tak přesné formátování vašich sestav a vizuálních datových prezentací.

## Pokyny k instalaci:  

Chcete-li použít Aspose.Cells pro .NET, postupujte takto:  

1. Instalace pomocí Správce balíčků NuGet  
- Otevřete Visual Studio.  
- Přejděte do nabídky Nástroje → Správce balíčků NuGet → Spravovat balíčky NuGet pro řešení.  
- Hledat Aspose.Cells.  
- Kliknutím na tlačítko Instalovat přidáte balíček do projektu.  

Nebo jej nainstalujte pomocí konzole Správce balíčků NuGet:  

```powershell
Install-Package Aspose.Cells
```

2. Přidání jmenného prostoru do kódu C#  
```csharp
using Aspose.Cells;
```

3. Aktivace licence (volitelné, pro plné funkce)  
Pokud máte licenci, použijte ji pomocí následujícího kódu:  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
