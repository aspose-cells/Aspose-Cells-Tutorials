# Přeměnit velikost a položit graf v programu Excel pomocí Aspose.Cells 
Aspose.Cells pro .NET poskytuje výkonný API pro programatickou práci s soubory Excel. Tento příklad ukazuje, jak změnit velikost a přemístění grafu v pracovní knize Excel pomocí Aspose.Cells. Ať už potřebujete upravit rozměry grafu nebo ji přesunout do konkrétní pozice, tento příruční program vás provede proces pomocí snadno sledovatelného kódu C#. Použitím Aspose.Cells můžete bezproblémově manipulovat s grafickými graficemi Excel, zajišťující, že vaše zprávy a prezentace vizuálních dat jsou přesně formátovány.

## Příkazy na instalaci:  

Chcete-li použít Aspose.Cells pro .NET, postupujte podle těchto kroků:  

1. Instalace prostřednictvím programu NuGet Package Manager  
* Otevři Visual Studio.  
* Přejďte na nástroje → Správce balíček NuGet → Správte balíčky NuGet pro řešení.  
* Hledejte Aspose.Cells.  
* Klikněte na položku Instalit, abyste balíček přidali do projektu.  

Alternativně ji nainstalovat pomocí konzole NuGet Manager:  

```powershell
Install-Package Aspose.Cells
```

2. Přidejte prostor pro názvy do svého kódu C#  
```csharp
using Aspose.Cells;
```

3. Aktivovat licenci (nepotřební pro plné funkce)  
Pokud máte licenci, použijte ji pomocí následujícího kódu:  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
