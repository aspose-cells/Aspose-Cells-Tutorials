# Přeměnit velikost a položit graf v programu Excel pomocí Aspose.Cells 
Aspose.Cells for .NET provides a powerful API for working with Excel files programmatically. This example demonstrates how to resize and reposition a chart in an Excel workbook using Aspose.Cells. Whether you need to adjust chart dimensions or move it to a specific position, this tutorial walks you through the process with easy-to-follow C# code. By leveraging Aspose.Cells, you can manipulate Excel charts seamlessly, ensuring your reports and visual data presentations are precisely formatted.

## Příkazy na instalaci:  

Chcete-li použít Aspose.Cells pro .NET, postupujte podle těchto kroků:  

1. Instalace prostřednictvím programu NuGet Package Manager  
- Otevři Visual Studio.  
- Přejďte na nástroje → Správce balíček NuGet → Správte balíčky NuGet pro řešení.  
- Hledejte Aspose.Cells.  
- Klikněte na položku Instalit, abyste balíček přidali do projektu.  

Alternativně ji nainstalovat pomocí konzole NuGet Manager:  

```powershell
Install-Package Aspose.Cells
```

2. Add Namespace in Your C# Code  
```csharp
using Aspose.Cells;
```

3. Aktivovat licenci (nepotřební pro plné funkce)  
Pokud máte licenci, použijte ji pomocí následujícího kódu:  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
