# Získejte titulky grafu ze souboru ODS pomocí Aspose.Cells  
Tento příklad ukazuje, jak pomocí Aspose.Cells pro .NET načíst soubor ODS, přistupovat ke konkrétnímu grafu v listu a načíst jeho podtitul. Dodržováním této příručky mohou vývojáři efektivně číst a extrahovat informace o grafu z tabulek ODS pomocí jazyka C#. Příklad poskytuje jasný, podrobný přístup k práci s grafy v souborech Excelu a zajišťuje tak hladkou implementaci v aplikacích .NET.

## Pokyny k instalaci:  
Chcete-li použít tento příklad, postupujte takto:  

1. Nainstalujte Aspose.Cells pro .NET pomocí NuGetu:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Přidejte požadovaný jmenný prostor do svého projektu C#:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Spuštěním poskytnutého kódu načtěte soubor ODS, zpřístupněte graf a vytiskněte jeho podtitulky do konzole.
