# Vytvořit PDF z Excel Chart s specifikovaným velikostí stránky

Exportování grafů z programu Excel do souboru PDF s přesným rozložením stránek je zásadní požadavkem pro reportování a vizualizaci dat. Tento výukový program ukazuje, jak používat Aspose.Cells pro .NET k vytvoření PDF z programu Excel, zatímco specifikuje velikost a uspořádání stránek. Naučíte se načítat soubor Excel, získat přístup k pracovní listě a jeho grafu a přeměnit ho na dobře formátovaný soubor PDF.

## Příkazy na instalaci  
Chcete-li použít Aspose.Cells pro .NET, postupujte podle těchto kroků:  

1. Instalace prostřednictvím NuGet:  
   Otevři správu balíček NuGet v Visual Studio a spuštěj:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Snímek DLL:  
   Získáte nejnovější verzi z [Aspose.Cells pro stránku s stažením .NET](https://releases.aspose.com/cells/net/)\- Ne , ne .  

3. Vkládejte prostor pro názvy:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
