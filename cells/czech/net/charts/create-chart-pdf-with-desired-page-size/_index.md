# Vytvoření PDF z tabulky v Excelu se zadanou velikostí stránky

Export grafů z Excelu do souboru PDF s přesným rozvržením stránky je klíčovým požadavkem pro vytváření sestav a vizualizaci dat. Tento tutoriál ukazuje, jak pomocí Aspose.Cells for .NET vygenerovat PDF z grafu v Excelu se zadáním velikosti stránky a zarovnání. Naučíte se, jak načíst soubor v Excelu, jak přistupovat k listu a jeho grafu a jak jej převést do dobře formátovaného souboru PDF.

## Pokyny k instalaci  
Chcete-li použít Aspose.Cells pro .NET, postupujte takto:  

1. Instalace přes NuGet:  
   Otevřete Správce balíčků NuGet ve Visual Studiu a spusťte:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Stáhněte si knihovnu DLL:  
   Získejte nejnovější verzi z [Stránka ke stažení Aspose.Cells pro .NET](https://releases.aspose.com/cells/net/).  

3. Zahrnout jmenný prostor:  
   Přidejte do souboru C# následující kód:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
