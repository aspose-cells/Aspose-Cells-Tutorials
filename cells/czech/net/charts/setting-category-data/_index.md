## Příkazy na instalaci  

Chcete-li použít Aspose.Cells pro .NET, postupujte podle těchto kroků:  

1. Instalace prostřednictvím programu NuGet Package Manager:  
   * Otevři svůj projekt .NET v Visual Studio.  
   * Jděte do nástroje → NuGet Package Manager → Správte pakety NuGet pro řešení.  
   * Hledejte Aspose.Cells a nainstalujte nejnovější verzi.  

   Alternativně ji nainstalovat pomocí konzole pro správu balíček:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Stáhněte si z webové stránky Aspose:  
   * Navštivte oficiální stránku pro stažení: [Aspose.Cells pro .NET](https://releases.aspose.com/cells/net/)  
   * Snímejte DLL a přidejte do projektu odkaz na něj.  

3. Žádost o licenci (nezveřejněná):  
   * Získám povolení od [Předpokládej nákup](https://purchase.aspose.com/buy)nebo požádat o [dočasná licence](https://purchase.aspose.com/temporary-license/)\- Ne , ne .  
   * Použijte licenci ve svém kódu:  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
