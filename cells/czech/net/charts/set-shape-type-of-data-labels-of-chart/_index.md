# Změnit tvar etikety datového grafu v programu Excel pomocí Aspose.Cells
V tomto tutoriálu zkoumáme, jak upravit tvar datových štítků v Excel tabulce pomocí Aspose.Cells pro .NET. Datové štítky v tabulkách pomáhají poskytovat lepší vizualizaci a změna jejich tvaru může zlepšit čitelnost. Tento příklad ukazuje, jak získat přístup k existujícímu Excelovému tabulce, získat data štítky série a změnit jejich tvar na řečový bublový oval (Wedge Ellipse Callout).

## Příkazy na instalaci:  

Chcete-li v projektu používat Aspose.Cells pro .NET, postupujte podle těchto kroků:  

1. Instalace prostřednictvím NuGet:  
   * Otevři svůj projekt Visual Studio.  
   * Jděte do nástrojů > Správce balíček NuGet > Správte balíčky NuGet pro řešení.  
   * Hledejte Aspose.Cells a nainstalujte nejnovější verzi.  

   Nebo použijte konzoli pro správu balíček:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Stáhněte si z webové stránky Aspose:  
   * Navštěvovat [Aspose.Cells pro .NET Download](https://releases.aspose.com/cells/net/)a získat nejnovější DLL.  
   * Přidejte si stáhnutý odkaz na DLL do projektu ručně.  

3. Využijte licenci (nepotřební, ale doporučeno):  
   * Pokud máte licenci, použijte ji:  
     ```csharp
     License license = new License();
     ```
