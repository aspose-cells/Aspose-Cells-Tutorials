## Istruzioni per l'installazione
Per iniziare a utilizzare Aspose.Cells per .NET, seguire questi passaggi:  

1. Scarica Aspose.Cells per .NET dal repository ufficiale: [Scarica Aspose.Cells](https://releases.aspose.com/cells/net/)  
2. Installa tramite NuGet Package Manager:  
   - Apri Visual Studio  
   - Eseguire il seguente comando nella console di Package Manager:  
     ```powershell
     Install-Package Aspose.Cells
     ```
3. Includi lo spazio dei nomi Aspose.Cells nel tuo progetto C#:  
   ```csharp
   using Aspose.Cells;
   ```  
4. Inizia a programmare utilizzando l'esempio fornito per creare e salvare una tabella pivot in formato ODS.
