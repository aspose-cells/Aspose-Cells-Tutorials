# Crea Excel Workbook con Fonti Latino e Orientale

Questo esempio dimostra come utilizzare Aspose.Cells per .NET per creare un libro di lavoro Excel, aggiungere una scatola di testo a un foglio di lavoro e impostare sia i nomi di font in Far East che in latino per il testo all'interno di tale scatola di testo. Il codice mostra l'inizializzazione dello spazio di lavoro, la manipolazione degli elementi di foglio di lavoro e il salvataggio del libro di lavoro con una formattazione di font specifica su misura per i vari requisiti linguistici.

## Instruzioni di installazione
1. Installare Aspose.Cells per .NET: 
   * Utilizzare NuGet Package Manager nel tuo progetto Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   * O scaricare il DLL dal [Sito Web di destinazione](https://www.aspose.com/downloads/cells/net)\- Non è vero .

2. Aggiungere riferimenti: Assicurati che il tuo progetto faccia riferimento alla libreria Aspose.Cells.

3. Configurazione della directory di output: aggiornare la `uscitaDir`variabile nel codice con il percorso di uscita desiderato in cui verrà salvato il file Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
