# Crea una cartella di lavoro Excel con caratteri latini e dell'Estremo Oriente

Questo esempio illustra come utilizzare Aspose.Cells per .NET per creare una cartella di lavoro di Excel, aggiungere una casella di testo a un foglio di lavoro e impostare nomi di font sia orientali che latini per il testo all'interno di tale casella. Il codice illustra l'inizializzazione dell'area di lavoro, la manipolazione degli elementi del foglio di lavoro e il salvataggio della cartella di lavoro con una formattazione di font specifica, adattata alle diverse esigenze linguistiche.

## Istruzioni per l'installazione
1. Installa Aspose.Cells per .NET: 
   - Utilizza NuGet Package Manager nel tuo progetto Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   - Oppure scarica la DLL da [Sito web di Aspose](https://www.aspose.com/downloads/cells/net).

2. Aggiungi riferimenti: assicurati che il tuo progetto faccia riferimento alla libreria Aspose.Cells.

3. Imposta directory di output: aggiorna `outputDir` variabile nel codice con il percorso di output desiderato in cui verrà salvato il file Excel.

4. Esegui l'esempio: copia il codice C# fornito nel tuo progetto, compilalo ed eseguilo. Dovresti visualizzare la conferma dell'output e il file Excel con la casella di testo configurata verrà creato nella directory specificata.
