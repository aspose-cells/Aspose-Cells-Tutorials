## Instruzioni di installazione
1. Installare Aspose.Cells: assicurati di avere installato la libreria Aspose.Cells. Puoi farlo utilizzando NuGet Package Manager eseguendo:
   ```
   Install-Package Aspose.Cells
   ```
2. Configurazione dell'ambiente: Crea una directory sulla tua macchina in cui vuoi archiviare i file Excel generati. Rimpiazzare `"La tua directory di documenti"`nell'esempio con il tuo percorso di directory effettivo.
3. Implementare il codice: copiare il codice di esempio fornito nella tua applicazione .NET.`dataDir + "logo.jpg"`) per indicare un'immagine esistente che si desidera inserire nel file Excel.
4. Eseguire l'applicazione: eseguire l'applicazione. Il codice creerà un nuovo file Excel chiamato `output.xls`nella directory specificata, con un'immagine inserita nella cella "F6".
