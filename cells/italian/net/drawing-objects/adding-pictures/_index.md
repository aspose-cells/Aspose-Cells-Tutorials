# Crea e salva file Excel con Aspose.Cells
Questo tutorial illustra come utilizzare Aspose.Cells per .NET per creare un nuovo file Excel a livello di codice. Imparerai a verificare l'esistenza di una directory, a creare un'istanza di un oggetto Workbook, ad aggiungere un foglio di lavoro, a inserire un'immagine in una cella specifica e a salvare il file nel percorso desiderato. Segui il semplice frammento di codice fornito per iniziare a migliorare le tue attività di automazione in Excel.

## Istruzioni per l'installazione
1. Installa Aspose.Cells: assicurati di aver installato la libreria Aspose.Cells. Puoi farlo utilizzando NuGet Package Manager eseguendo:
   ```
   Install-Package Aspose.Cells
   ```
2. Imposta il tuo ambiente: crea una directory sul tuo computer in cui desideri archiviare i file Excel generati. Sostituisci `"Your Document Directory"` nell'esempio con il percorso effettivo della directory.
3. Implementa il codice: copia il codice di esempio fornito nella tua applicazione .NET. Assicurati di aggiornare il percorso dell'immagine (`dataDir + "logo.jpg"`) per puntare a un'immagine esistente che si desidera inserire nel file Excel.
4. Esegui l'applicazione: esegui l'applicazione. Il codice creerà un nuovo file Excel denominato `output.xls` nella directory specificata, con un'immagine inserita nella cella "F6".
