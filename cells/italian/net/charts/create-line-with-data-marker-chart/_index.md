## Instruzioni di installazione:  

Per utilizzare Aspose.Cells per .NET nel tuo progetto, segui questi passaggi:  

1. Installazione tramite NuGet:  
   Aprire la Console di gestione pacchetti in Visual Studio e eseguire:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Scarico DLL:  
   In alternativa, è possibile scaricare l'ultima versione di .NET Aspose.Cells da [Aspose download](https://releases.aspose.com/cells/net/)\- Non è vero .

3. Aggiungere riferimento:  
   Se hai scaricato il DLL manualmente, aggiungi un riferimento a `Aspose.Cells.`nel tuo progetto .NET.

4. Includere il namespace:  
   In your C# file, add:  
   ```csharp
   using Aspose.Cells;
   using System;
   using System.Drawing;
   ```

5. Eseguire l' Esempio:  
   Copy and paste the provided C# code into your project, update the output directory, and execute the script to generate an Excel file with a line chart.
