## Istruzioni per l'installazione:  

Per utilizzare Aspose.Cells per .NET nel tuo progetto, segui questi passaggi:  

1. Installazione tramite NuGet:  
   Aprire la console di Gestione pacchetti in Visual Studio ed eseguire:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Scarica DLL:  
   In alternativa, puoi scaricare l'ultima versione di Aspose.Cells per .NET da [Download di Aspose](https://releases.aspose.com/cells/net/).

3. Aggiungi riferimento:  
   Se hai scaricato manualmente la DLL, aggiungi un riferimento a `Aspose.Cells.dll` nel tuo progetto .NET.

4. Includi spazio dei nomi:  
   Nel tuo file C#, aggiungi:  
   ```csharp
   using Aspose.Cells;
   using System;
   using System.Drawing;
   ```

5. Esegui l'esempio:  
   Copia e incolla il codice C# fornito nel tuo progetto, aggiorna la directory di output ed esegui lo script per generare un file Excel con un grafico a linee.
