# Crea PDF da Excel Grafico con dimensioni specificate di pagina

L'esportazione di grafici da Excel a un file PDF con un layout di pagina preciso è un requisito cruciale per la segnalazione e la visualizzazione dei dati. Questo tutorial mostra come utilizzare Aspose.Cells per .NET per generare un PDF da un grafico Excel specificando la dimensione e l'allineamento della pagina. Imparerai a caricare un file Excel, accedere a un foglio di lavoro e alla sua tabella e convertirlo in un file PDF ben formattato.

## Instruzioni di installazione  
Per utilizzare Aspose.Cells per .NET, seguire questi passaggi:  

1. Installazione tramite NuGet:  
   Aprire il NuGet Package Manager in Visual Studio e eseguire:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Scarica il DLL:  
   Prendi l' ultima versione del [Aspose.Cells per la pagina di download .NET](https://releases.aspose.com/cells/net/)\- Non è vero .  

3. Includere il namespace:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
