# Motore di calcolo Excel per applicazioni .NET

## Panoramica

Aspose.Cells per .NET offre un robusto motore di calcolo che consente agli sviluppatori di gestire calcoli complessi con formule all'interno di fogli di calcolo Excel. Questo potente componente consente di configurare il comportamento di calcolo, implementare logiche di calcolo personalizzate e ottimizzare le prestazioni per cartelle di lavoro di grandi dimensioni.

## Caratteristiche principali

- **Motori di calcolo personalizzati**: Implementa la tua logica di calcolo per formule specializzate
- **Monitoraggio del calcolo**: Calcoli delle formule di tracciamento e controllo per l'ottimizzazione delle prestazioni
- **Opzioni di calcolo ricorsivo**: Configura come vengono gestiti i riferimenti circolari
- **Configurazione del sistema di data**: Passare dal sistema di data del 1900 a quello del 1904
- **Rilevamento del riferimento circolare**: Identificare e risolvere le dipendenze circolari nelle formule
- **Ottimizzazione delle prestazioni**: Ottimizza le impostazioni di calcolo per cartelle di lavoro di grandi dimensioni
