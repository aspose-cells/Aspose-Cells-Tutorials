# Maak een Excel-werkmap met Verre Oosten- en Latijnse lettertypen

Dit voorbeeld laat zien hoe u Aspose.Cells voor .NET kunt gebruiken om een Excel-werkmap te maken, een tekstvak aan een werkblad toe te voegen en zowel Verre Oosten- als Latijnse lettertypen in te stellen voor de tekst in dat tekstvak. De code laat zien hoe u de werkruimte initialiseert, werkbladelementen bewerkt en de werkmap opslaat met een specifieke lettertypeopmaak die is afgestemd op diverse taalvereisten.

## Installatie-instructies
1. Installeer Aspose.Cells voor .NET: 
   - Gebruik NuGet Package Manager in uw Visual Studio-project:
     ```
     Install-Package Aspose.Cells
     ```
   - Of download de DLL van de [Aspose-website](https://www.aspose.com/downloads/cells/net).

2. Verwijzingen toevoegen: zorg ervoor dat uw project verwijst naar de Aspose.Cells-bibliotheek.

3. Uitvoermap instellen: de uitvoermap bijwerken `outputDir` variabele in de code met het gewenste uitvoerpad waar het Excel-bestand wordt opgeslagen.

4. Voer het voorbeeld uit: Kopieer de meegeleverde C#-code naar uw project, compileer en voer deze uit. U zou de uitvoerbevestiging moeten zien en het Excel-bestand met het geconfigureerde tekstvak wordt aangemaakt in de opgegeven map.
