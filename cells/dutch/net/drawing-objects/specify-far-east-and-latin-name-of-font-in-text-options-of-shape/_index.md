## Installatie-instructies
1. Installeer Aspose.Cells voor .NET: 
   * Gebruik NuGet Package Manager in uw Visual Studio-project:
     ```
     Install-Package Aspose.Cells
     ```
   * Of download de DLL van de [Webpagina's opzetten](https://www.aspose.com/downloads/cells/net)\- Ik heb het niet gedaan .

2. Voeg Referenties toe: zorg ervoor dat uw project verwijst naar de bibliotheek Aspose.Cells.

3. Instel van het output directory: Update de `outputDir`variabele in de code met het gewenste outputpad waar het Excel-bestand wordt opgeslagen.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
