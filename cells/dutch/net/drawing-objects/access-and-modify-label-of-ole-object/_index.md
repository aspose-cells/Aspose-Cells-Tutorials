## Installatie-instructies
1. Installeer Aspose.Cells voor .NET: Gebruik NuGet Package Manager om de bibliotheek Aspose.Cells in je project te installeren.
   ```
   Install-Package Aspose.Cells
   ```
2. Voeg Gebruiksaanwijzingen toe: Voeg de nodige naamruimten in uw codebestand.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Voeg voorbeeldcode toe: Kopieer en plak de verstrekte code in je project, zodat je het document directory pad kunt specificeren.
4. Run the Code: Voer het project uit om het Ole objectetiket in het Excel- bestand te gebruiken en te wijzigen.
