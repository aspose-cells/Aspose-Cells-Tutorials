## Installatie-instructies
1. Installeer Aspose.Cells voor .NET: gebruik NuGet Package Manager om de Aspose.Cells-bibliotheek in uw project te installeren.
   ```
   Install-Package Aspose.Cells
   ```
2. Voeg richtlijnen toe: Neem de benodigde naamruimten op in uw codebestand.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Voorbeeldcode toevoegen: Kopieer en plak de meegeleverde code in uw project. Zorg ervoor dat u het pad naar uw documentmap opgeeft.
4. Voer de code uit: voer uw project uit om het label van het OLE-object in uw Excel-bestand te openen en te wijzigen. Controleer de console op uitvoerberichten die de succesvolle uitvoering van het proces bevestigen.
