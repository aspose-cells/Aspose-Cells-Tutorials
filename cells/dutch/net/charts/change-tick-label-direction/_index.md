# Wijzig de richting van het label van een grafiek in Excel met Aspose.Cells 

Dit voorbeeld laat zien hoe u de richting van de maatstreeplabels in een Excel-werkblad kunt aanpassen met Aspose.Cells voor .NET. Door de tekstrichting van de maatstreeplabels op de categorie-as aan te passen, kunt u de leesbaarheid van grafiekgegevens verbeteren. De code laadt een bestaande Excel-werkmap, opent het eerste werkblad, haalt de grafiek op, stelt de richting van de maatstreeplabels in op horizontaal en slaat vervolgens de gewijzigde werkmap op. Deze aanpak garandeert nauwkeurige controle over de richting van de grafieklabels in uw .NET-toepassingen.

## Installatie-instructies  
Om dit voorbeeld uit te voeren, volgt u deze stappen:  

1. Aspose.Cells voor .NET installeren  
   - Downloaden van [Aspose.Cells voor .NET](https://releases.aspose.com/cells/net/)  
   - Of installeer via NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Stel uw project in  
   - Maak een nieuw .NET-project.  
   - Voeg een verwijzing naar Aspose.Cells.dll toe.  

3. Voer de code uit  
   - Vervangen `"Your Document Directory"` met het werkelijke bestandspad.  
   - Voer het programma uit om de richting van het vinkje in de grafiek bij te werken.
