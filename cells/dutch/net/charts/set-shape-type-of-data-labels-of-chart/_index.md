# Wijzig de vorm van grafiekgegevenslabels in Excel met Aspose.Cells
In deze tutorial laten we zien hoe je het vormtype van gegevenslabels in een Excel-grafiek kunt aanpassen met Aspose.Cells voor .NET. Gegevenslabels in grafieken zorgen voor een betere visualisatie en het aanpassen van hun vorm kan de leesbaarheid verbeteren. Dit voorbeeld laat zien hoe je een bestaande Excel-grafiek opent, de gegevenslabels van de reeks ophaalt en hun vormtype wijzigt in een ovale tekstballon (een ellipsvormige tekstballon). ## Installatie-instructies:  

Volg deze stappen om Aspose.Cells voor .NET in uw project te gebruiken:  

1. Installeren via NuGet:  
   - Open uw Visual Studio-project.  
   - Ga naar Extra > NuGet Package Manager > NuGet-pakketten beheren voor oplossing.  
   - Zoek naar Aspose.Cells en installeer de nieuwste versie.  

   Of gebruik de Package Manager Console:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Downloaden van Aspose-website:  
   - Bezoek [Aspose.Cells voor .NET downloaden](https://releases.aspose.com/cells/net/) en download de nieuwste DLL's.  
   - Voeg de gedownloade DLL-referentie handmatig toe aan uw project.  

3. Pas de licentie toe (optioneel, maar aanbevolen):  
   - Als u een licentie heeft, kunt u deze als volgt aanvragen:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Cells.lic");
     ```
