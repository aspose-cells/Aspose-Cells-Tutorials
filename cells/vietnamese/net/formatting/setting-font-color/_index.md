## Hướng dẫn cài đặt  
Để sử dụng Aspose.Cells cho .NET, hãy làm theo các bước sau:  

1. Cài đặt thông qua Trình quản lý gói NuGet:  
   - Mở Visual Studio.  
   - Vào Công cụ > Trình quản lý gói NuGet > Quản lý gói NuGet cho Solution.  
   - Tìm kiếm Aspose.Cells và cài đặt phiên bản mới nhất.  

2. Cài đặt thông qua Package Manager Console:  
   - Mở Bảng điều khiển quản lý gói trong Visual Studio.  
   - Chạy lệnh sau:  
     ```powershell
     Install-Package Aspose.Cells
     ```

3. Tải xuống từ trang web Aspose:  
   - Thăm nom [Tải xuống Aspose.Cells cho .NET](https://releases.aspose.com/cells/net/).  
   - Tải xuống và cài đặt thủ công các DLL mới nhất.  

4. Thêm tham chiếu đến Aspose.Cells vào dự án của bạn:  
   - Trong Solution Explorer, nhấp chuột phải vào References.  
   - Nhấp vào Thêm tham chiếu và duyệt tìm tệp Aspose.Cells.dll.  
   - Nhấp vào OK để thêm vào dự án của bạn.
