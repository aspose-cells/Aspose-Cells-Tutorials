1. Thiết lập thông qua NuGet:  
   * Open Visual Studio.  
   * Đi đến Công cụ > Quản lý gói NuGet > Quản lý gói NuGet cho giải pháp.  
   * Tìm kiếm Aspose.Cells và cài đặt nó.  

   Ngoài ra, cài đặt qua NuGet Package Manager Console:  
   ```powershell
   Install-Package Aspose.Cells
   ```  
