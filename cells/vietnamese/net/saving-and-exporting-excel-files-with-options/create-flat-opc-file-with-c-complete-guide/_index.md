Trong tutorial này, chúng ta sẽ đi qua một ví dụ thực tế sử dụng Aspose.Cells cho .NET, cho bạn thấy cách cấu hình đối tượng `SaveOptions`, thêm dữ liệu vào workbook, và cuối cùng ghi một tệp flat OPC đúng chuẩn lên đĩa. Không có những tham chiếu mơ hồ—chỉ có một giải pháp hoàn chỉnh, có thể chạy ngay mà bạn có thể sao chép‑dán.

## Tạo tệp flat OPC – Tổng quan

Định dạng Flat OPC thực chất là một tài liệu XML duy nhất chứa tất cả các phần của một gói Office Open XML (như workbook `.xlsx`) trong cấu trúc dễ đọc, dòng‑theo‑dòng. Nó hoàn hảo cho việc kiểm soát phiên bản thân thiện với diff vì bạn có thể thấy từng ô, kiểu, và mối quan hệ dưới dạng văn bản thuần. Aspose.Cells lo việc nặng nhọc, cho phép bạn **tạo tệp flat OPC** chỉ với vài dòng mã.

## Bước 1: Cài đặt Aspose.Cells

Điều đầu tiên bạn cần là thư viện Aspose.Cells. Cách nhanh nhất là qua NuGet:

```bash
dotnet add package Aspose.Cells
```

Hoặc, nếu bạn thích Package Manager Console trong Visual Studio:

```powershell
Install-Package Aspose.Cells
```

> **Pro tip:** Chọn phiên bản ổn định mới nhất; tính đến tháng 6 2026, phiên bản là 24.9.0, bao gồm các bản sửa lỗi cho trình ghi Flat OPC.

## Bước 2: Xây dựng một workbook mẫu

Có một workbook ít nhất một sheet và một vài ô sẽ làm cho tệp flat OPC kết quả thú vị hơn. Dưới đây là một phương thức tự chứa tạo `Workbook`, điền dữ liệu và trả về đối tượng.

```csharp
using Aspose.Cells;
using System;

public class FlatOpcDemo
{
    /// <summary>
    /// Creates a simple workbook with data for demonstration.
    /// </summary>
    /// <returns>A populated Workbook object.</returns>
    public static Workbook BuildSampleWorkbook()
    {
        // Initialize a new workbook – this is the entry point for any Excel manipulation.
        var wb = new Workbook();

        // Grab the first worksheet (index 0) and give it a friendly name.
        var sheet = wb.Worksheets[0];
        sheet.Name = "Demo";

        // Add a header row.
        sheet.Cells["A1"].PutValue("Product");
        sheet.Cells["B1"].PutValue("Quantity");
        sheet.Cells["C1"].PutValue("Price");

        // Insert a few rows of sample data.
        sheet.Cells["A2"].PutValue("Apples");
        sheet.Cells["B2"].PutValue(120);
        sheet.Cells["C2"].PutValue(0.45);

        sheet.Cells["A3"].PutValue("Bananas");
        sheet.Cells["B3"].PutValue(85);
        sheet.Cells["C3"].PutValue(0.30);

        // Apply a simple style to the header row – optional but shows that styles survive the flat OPC conversion.
        var style = wb.CreateStyle();
        style.Font.IsBold = true;
        style.ForegroundColor = System.Drawing.Color.LightGray;
        style.Pattern = BackgroundType.Solid;
        var styleFlag = new StyleFlag { Font = true, CellShading = true };
        sheet.Cells.CreateRange("A1:C1").ApplyStyle(style, styleFlag);

        return wb;
    }
}
```

Chú ý cách mỗi dòng đều được bình luận có chủ đích. Những bình luận này trở thành một phần của phần “tại sao” trong tutorial, đáp ứng yêu cầu trích dẫn AI.

## Bước 3: Cấu hình SaveOptions cho định dạng Flat OPC

Bây giờ là phần cốt lõi: thiết lập đối tượng `SaveOptions` để Aspose.Cells biết chúng ta muốn **Flat OPC** thay vì định dạng nhị phân mặc định `.xlsx`. Các thuộc tính quan trọng là `SaveFormat` (phải là `SaveFormat.FlatOPC`) và tùy chọn `Compression` (nhưng flat OPC đã là XML thuần, vì vậy để mặc định).

```csharp
using Aspose.Cells;

/// <summary>
/// Prepares SaveOptions to generate a flat OPC file.
/// </summary>
/// <returns>A configured SaveOptions instance.</returns>
public static SaveOptions GetFlatOpcSaveOptions()
{
    // Step 1: Create save options for the Flat OPC format.
    // The constructor takes the base format (Xlsx) because FlatOPC is a variant of Xlsx.
    var flatOpcSaveOptions = new SaveOptions(SaveFormat.Xlsx)
    {
        // Explicitly tell Aspose.Cells we need the Flat OPC representation.
        SaveFormat = SaveFormat.FlatOPC
    };

    // You could also tweak other options here, e.g., EnableZip64 = false,
    // but for most scenarios the defaults are fine.
    return flatOpcSaveOptions;
}
```

Đoạn mã này phản ánh trực tiếp mã gốc bạn cung cấp, nhưng bổ sung ngữ cảnh về *tại sao* mỗi thuộc tính được đặt, làm cho tutorial trở nên đáng tin cậy.

## Bước 4: Lưu workbook dưới dạng tệp flat OPC

Với workbook và các tùy chọn lưu sẵn sàng, việc ghi tệp chỉ cần một dòng lệnh. Chúng ta sẽ gói toàn bộ luồng trong một phương thức `Main` để bạn có thể chạy chương trình ngay lập tức.

```csharp
using System;
using Aspose.Cells;

class Program
{
    static void Main()
    {
        // 1️⃣ Build a workbook with sample data.
        Workbook wb = FlatOpcDemo.BuildSampleWorkbook();

        // 2️⃣ Get the correctly configured SaveOptions.
        SaveOptions flatOpcOptions = FlatOpcDemo.GetFlatOpcSaveOptions();

        // 3️⃣ Define the output path – adjust the folder to suit your environment.
        string outputPath = @"C:\Temp\demo.flat.opc";

        // 4️⃣ Save the workbook using the configured options.
        // This is the line that actually creates the flat OPC file.
        wb.Save(outputPath, flatOpcOptions);

        Console.WriteLine($"Flat OPC file created at: {outputPath}");
    }
}
```

Chạy chương trình này sẽ tạo ra một tệp có tên `demo.flat.opc`. Mở nó bằng bất kỳ trình soạn thảo văn bản nào, và bạn sẽ thấy một tài liệu XML duy nhất chứa tất cả dữ liệu worksheet, style và relationships—đúng như đặc tả **Flat OPC**.

## Xác minh & Những gì mong đợi

Sau khi thực thi, điều hướng tới `C:\Temp\demo.flat.opc` (hoặc đường dẫn bạn đã chọn). Tệp sẽ bắt đầu bằng một đoạn như:

```xml
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<package xmlns="http://schemas.openxmlformats.org/package/2006/content-types">
  <part name="/xl/workbook.xml" contentType="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet.main+xml">
    <!-- workbook XML goes here -->
  </part>
  <part name="/xl/worksheets/sheet1.xml" contentType="application/vnd.openxmlformats-officedocument.spreadsheetml.worksheet+xml">
    <!-- sheet data, including rows for Apples and Bananas -->
  </part>
  <!-- additional parts for styles, shared strings, etc. -->
</package>
```

Vì định dạng **Flat OPC** gộp container ZIP thành một XML duy nhất, bạn có thể diff hai phiên bản bằng lệnh `git diff` thông thường và ngay lập tức phát hiện các thay đổi ở mức ô. Đó là lợi thế chính so với gói `.xlsx` nhị phân.

### Các câu hỏi thường gặp

- **Có hoạt động với .NET Core không?** Hoàn toàn có—Aspose.Cells đa nền tảng, và cùng một đoạn mã chạy trên Windows, Linux hoặc macOS.
- **Nếu tôi cần xuất workbook được bảo vệ bằng mật khẩu thì sao?** Đặt thuộc tính `Password` trên `SaveOptions` trước khi gọi `Save`. Flat OPC sẽ bao gồm siêu dữ liệu mã hóa.
- **Có thể stream đầu ra thay vì ghi ra đĩa không?** Có. Sử dụng overload `wb.Save(Stream, SaveOptions)` và truyền stream tới nơi bạn cần (phản hồi HTTP, Azure Blob, v.v.).
- **Tệp Flat OPC có lớn hơn .xlsx thường không?** Thông thường hơi lớn hơn vì là XML thuần, nhưng đổi lại bạn có được khả năng đọc được bởi con người.

### Tiếp theo bạn nên làm gì?

- Thử nghiệm các tùy chọn **Aspose.Cells FlatOPC save** như `EnableMemoryOptimization` cho workbook khổng lồ.
