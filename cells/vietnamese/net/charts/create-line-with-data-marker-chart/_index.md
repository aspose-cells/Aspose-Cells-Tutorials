# Tạo biểu đồ đường với các điểm đánh dấu dữ liệu trong Aspose.Cells

## Hướng dẫn cài đặt:  

Để sử dụng Aspose.Cells cho .NET trong dự án của bạn, hãy làm theo các bước sau:  

1. Cài đặt qua NuGet:  
   Mở Package Manager Console trong Visual Studio và chạy:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Tải xuống DLL:  
   Ngoài ra, bạn có thể tải xuống phiên bản Aspose.Cells mới nhất cho .NET từ [Tải xuống Aspose](https://releases.aspose.com/cells/net/).

3. Thêm tham chiếu:  
   Nếu bạn đã tải xuống DLL theo cách thủ công, hãy thêm tham chiếu đến `Aspose.Cells.dll` trong dự án .NET của bạn.

4. Bao gồm không gian tên:  
   Trong tệp C# của bạn, hãy thêm:  
   ```csharp
   using Aspose.Cells;
   using System;
   using System.Drawing;
   ```

5. Chạy ví dụ:  
   Sao chép và dán mã C# được cung cấp vào dự án của bạn, cập nhật thư mục đầu ra và thực thi tập lệnh để tạo tệp Excel có biểu đồ đường.
