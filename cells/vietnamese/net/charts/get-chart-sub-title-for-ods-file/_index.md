## Hướng dẫn cài đặt:  
Để sử dụng ví dụ này, hãy làm theo các bước sau:  

1. Cài đặt Aspose.Cells cho .NET thông qua NuGet:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Thêm không gian tên cần thiết vào dự án C# của bạn:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Chạy mã được cung cấp để tải tệp ODS, truy cập biểu đồ và in phụ đề của biểu đồ vào bảng điều khiển.
