## Các hướng dẫn cài đặt  

Để sử dụng Aspose.Cells cho .NET trong dự án của bạn, hãy làm theo các bước sau:  

1. Thiết lập thông qua quản lý gói NuGet  
- Mở Visual Studio và di chuyển đến Công cụ → NuGet Package Manager → Quản lý NuGet Packages for Solution.  
- Tìm kiếm Aspose.Cells và nhấp vào cài đặt.  

Hoặc, cài đặt bằng cách sử dụng NuGet Package Manager Console:  

```powershell
Install-Package Aspose.Cells
```  

2. Thiết lập qua .NET CLI  
Nếu bạn đang sử dụng CLI .NET, chạy:  

```powershell
dotnet add package Aspose.Cells
```  

3. Thêm tài liệu tham khảo vào dự án của bạn  
Once installed, include the following namespace in your C# file:  

```csharp
using Aspose.Cells;
using Aspose.Cells.Charts;
```
