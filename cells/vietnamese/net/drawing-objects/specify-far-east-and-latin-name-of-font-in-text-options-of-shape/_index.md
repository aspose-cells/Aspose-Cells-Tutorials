## Các hướng dẫn cài đặt
1. Thiết lập Aspose.Cells cho .NET: 
   * Sử dụng NuGet Package Manager trong dự án Visual Studio của bạn:
     ```
     Install-Package Aspose.Cells
     ```
   * Hoặc tải xuống DLL từ [Đặt trang web](https://www.aspose.com/downloads/cells/net)\- Tôi không biết.

2. Thêm References: Đảm bảo dự án của bạn tham khảo thư viện Aspose.Cells.

3. Thiết lập thư mục đầu ra: Cập nhật `outputDir`biến trong mã với con đường đầu ra mong muốn nơi tệp Excel sẽ được lưu.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
