# Tạo Sổ làm việc Excel có Định dạng trong Java – Hướng dẫn đầy đủ

Bạn đã bao giờ cần **create styled excel workbook** từ Java nhưng cảm thấy bối rối vì các chi tiết định dạng? Bạn không phải là người duy nhất. Dù bạn đang lấy dữ liệu từ cơ sở dữ liệu hay xuất báo cáo, việc có được giao diện đúng—ký hiệu tiền tệ, định dạng ngày, tiêu đề in đậm—có thể cảm giác như một công việc nặng nhọc.  

Trong tutorial này chúng ta sẽ đi qua một giải pháp thực tế mà **adds custom formatting to excel**, cho phép bạn **set date format excel column**, và cuối cùng **save workbook as xlsx** mà không gặp khó khăn. Sẵn sàng xem nó hoạt động chưa?

## Những gì bạn sẽ học

- Cách khởi tạo một `Workbook` Aspose.Cells từ đầu.  
- Các cách áp dụng một mảng style bao gồm các cột General, Currency và Date.  
- Lệnh chính xác để **create styled excel workbook** với tên cột đã được chèn sẵn.  
- Cách lưu file bằng phương thức **save workbook as xlsx**.  

Không cần file cấu hình bên ngoài, không có thủ thuật XML khó hiểu—chỉ cần đoạn mã Java thuần túy mà bạn có thể copy‑paste, chạy và tùy chỉnh.

### Yêu cầu trước

- Java 8+ (mã chạy trên bất kỳ JDK hiện đại nào).  
- Thư viện Aspose.Cells for Java (bạn có thể tải JAR mới nhất từ Maven Central).  
- Một chút kiến thức về collections (`List`, `Map`).  

Nếu bất kỳ mục nào trên còn lạ, hãy tạm dừng và cài đặt JAR Aspose.Cells; phần còn lại sẽ tự nhiên chảy.

## Tạo Styled Excel Workbook – Cài đặt dự án

Điều đầu tiên cần làm: tạo một dự án Maven (hoặc Gradle) mới và thêm Aspose.Cells làm dependency.

```xml
<!-- pom.xml snippet -->
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>23.12</version> <!-- use the latest version -->
</dependency>
```

> **Pro tip:** Nếu bạn dùng Gradle, cùng một coordinates cũng hoạt động với `implementation 'com.aspose:aspose-cells:23.12'`.

Tại sao bắt đầu bằng Maven? Nó tự động kéo tất cả các dependency transitive, vì vậy bạn sẽ không phải tốn thời gian tìm kiếm các JAR phù hợp sau này.

## Thêm Định dạng Tùy chỉnh cho Các Cột Excel

Bây giờ dự án đã sẵn sàng, hãy định nghĩa các style mà chúng ta muốn. Trong Aspose.Cells bạn có thể truyền một mảng các chuỗi định dạng mà ánh xạ trực tiếp tới từng cột.

```java
// Define column styles: General, Currency, Date
String[] columnStyles = {
    "General",          // Column A – plain text or numbers
    "$#,##0.00",        // Column B – currency with two decimals
    "mm/dd/yyyy"        // Column C – US‑style date
};
```

Bước **add custom formatting to excel** đơn giản chỉ là xây dựng mảng chuỗi này.  
Mỗi mục tuân theo cùng một mẫu mà bạn sẽ dùng trong hộp thoại “Custom” của Excel, vì vậy bạn không cần học cú pháp mới.

---

## Đặt Định dạng Ngày cho Cột Excel

Nếu dữ liệu của bạn có ngày, bạn sẽ muốn chúng hiển thị một cách nhất quán. Mục thứ ba trong `columnStyles` (`"mm/dd/yyyy"`) làm đúng điều đó—nó bảo Aspose.Cells xử lý cột thứ ba như ngày và hiển thị theo thứ tự tháng/ngày/năm.

```java
// Example: Adding a row with a Date object
Map<String, Object> row = new HashMap<>();
row.put("Product", "Widget");
row.put("Price", 19.99);
row.put("ReleaseDate", new java.util.Date()); // Today’s date
dataRows.add(row);
```

Khi workbook được tạo, lệnh **set date format excel column** đảm bảo mọi ô `ReleaseDate` đều hiển thị như `04/27/2026` (hoặc ngày hiện tại). Không cần xử lý sau.

## Nhập Dữ liệu và Áp dụng Các Style

Với các style đã được định nghĩa, bước tiếp theo là đưa dữ liệu vào worksheet. Aspose.Cells cung cấp `importCustomObjects`, không chỉ chèn dữ liệu mà còn áp dụng mảng style mà chúng ta vừa tạo.

```java
// Step 1: Simulate fetching rows from a DB
List<Map<String, Object>> dataRows = fetchRows(); // see helper below

// Step 2: Create a fresh workbook and grab the first sheet
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);

// Step 3: Import data with custom styles; true = first row has headers
worksheet.getCells().importCustomObjects(
        dataRows,        // data source
        0,               // start row (0‑based)
        0,               // start column (0‑based)
        columnStyles,    // style array from above
        true);           // first row contains column names
```

### Tại sao lại dùng `importCustomObjects`?

- **Magic một lần gọi:** Nó ghi dữ liệu *và* áp dụng định dạng trong một bước.  
- **Nhận thức tiêu đề:** Truyền `true` cho API biết rằng khóa đầu tiên của map là tiêu đề cột, rất phù hợp cho các trường hợp **create styled excel workbook** khi bạn muốn tiêu đề cột dễ đọc.  
- **Hiệu năng:** Bên trong, Aspose batch các thao tác ghi, nhanh hơn so với việc lặp qua từng ô.

> **Lưu ý:** Thứ tự các khóa trong mỗi `Map` quan trọng. Dùng `LinkedHashMap` nếu bạn cần thứ tự cột cố định.

## Lưu Workbook dưới dạng File XLSX

Mọi công sức sẽ vô nghĩa nếu bạn không thể lưu file. Dòng cuối cùng là một câu lệnh một dòng **save workbook as xlsx**.

```java
// Step 5: Persist the workbook
String outputPath = "output/DataTableStyled.xlsx";
workbook.save(outputPath);
System.out.println("Workbook saved to " + outputPath);
```

Phương thức `save` tự động nhận diện phần mở rộng file. Khi chỉ định `.xlsx`, bạn bảo đảm định dạng Office Open XML hiện đại, mà hầu hết các công cụ downstream đều mong đợi.

## Chạy và Kiểm tra Kết quả

Biên dịch và chạy lớp `ImportDataTableWithStyle`. Sau khi thực thi, bạn sẽ thấy một file tại `output/DataTableStyled.xlsx`. Mở nó trong Excel (hoặc LibreOffice) và bạn sẽ nhận thấy:

- Cột **B** hiển thị số với dấu đô la và hai chữ số thập phân (`$19.99`).

## Ví dụ Đầy đủ, Có Thể Chạy

Dưới đây là lớp hoàn chỉnh mà bạn có thể sao chép thẳng vào IDE. Nó bao gồm một phương thức mock `fetchRows` trả về dữ liệu mẫu, vì vậy bạn không cần cơ sở dữ liệu thực để thử.

```java
import com.aspose.cells.*;
import java.util.*;

public class ImportDataTableWithStyle {
    public static void main(String[] args) throws Exception {
        // ---------- Step 1: Get data ----------
        List<Map<String, Object>> dataRows = fetchRows();

        // ---------- Step 2: Create workbook ----------
        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // ---------- Step 3: Define column styles ----------
        String[] columnStyles = { "General", "$#,##0.00", "mm/dd/yyyy" };

        // ---------- Step 4: Import data with styles ----------
        // true = first row contains column names
        worksheet.getCells().importCustomObjects(dataRows, 0, 0, columnStyles, true);

        // ---------- Step 5: Save as XLSX ----------
        String outPath = "output/DataTableStyled.xlsx";
        workbook.save(outPath);
        System.out.println("Workbook saved to " + outPath);
    }

    // Mock method – replace with real DB call if needed
    private static List<Map<String, Object>> fetchRows() {
        List<Map<String, Object>> rows = new ArrayList<>();

        Map<String, Object> row1 = new LinkedHashMap<>();
        row1.put("Product", "Gadget");
        row1.put("Price", 49.95);
        row1.put("ReleaseDate", new Date()); // today
        rows.add(row1);

        Map<String, Object> row2 = new LinkedHashMap<>();
        row2.put("Product", "Widget");
        row2.put("Price", 19.99);
        Calendar cal = Calendar.getInstance();
        cal.set(2025, Calendar.DECEMBER, 15);
        row2.put("ReleaseDate", cal.getTime());
        rows.add(row2);

        return rows;
    }
}
```

Chạy nó, mở file đã tạo, và bạn sẽ thấy kết quả **create styled excel workbook** như mong đợi.

## Câu hỏi Thường gặp & Trường hợp Cạnh

**Nếu tôi cần hơn ba cột thì sao?**  
Chỉ cần mở rộng mảng `columnStyles`—mỗi mục ánh xạ tới một cột theo thứ tự. Đối với các cột bạn muốn để nguyên, dùng `"General"`.

**Có thể áp dụng phông chữ hoặc màu sắc không?**  
Có. Thay vì chuỗi định dạng, bạn có thể tạo một đối tượng `Style`, gán nó cho `StyleFlag`, và truyền một mảng `Style[]` vào `importCustomObjects`. Đó là một bước sâu hơn, nhưng nguyên tắc vẫn giống.

**Còn với bộ dữ liệu lớn thì sao?**  
Aspose.Cells xử lý hàng triệu dòng một cách hiệu quả, nhưng bạn có thể muốn tăng heap JVM (`-Xmx2...`).
