# Erstellen Sie PDF aus einem Excel-Diagramm mit angegebener Seitengröße

Der Export von Diagrammen aus Excel in eine PDF-Datei mit präzisem Seitenlayout ist eine wichtige Voraussetzung für Reporting und Datenvisualisierung. Dieses Tutorial zeigt, wie Sie mit Aspose.Cells für .NET ein PDF aus einem Excel-Diagramm generieren und dabei Seitengröße und Ausrichtung festlegen. Sie lernen, wie Sie eine Excel-Datei laden, auf ein Arbeitsblatt und dessen Diagramm zugreifen und es in eine gut formatierte PDF-Datei konvertieren.

## Installationsanweisungen  
Um Aspose.Cells für .NET zu verwenden, führen Sie die folgenden Schritte aus:  

1. Installation über NuGet:  
   Öffnen Sie den NuGet-Paket-Manager in Visual Studio und führen Sie Folgendes aus:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Laden Sie die DLL herunter:  
   Holen Sie sich die neueste Version von der [Aspose.Cells für .NET-Downloadseite](https://releases.aspose.com/cells/net/).  

3. Schließen Sie den Namespace ein:  
   Fügen Sie Ihrer C#-Datei Folgendes hinzu:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
