# Erstellen Sie PDFs aus einem Excel-Diagramm mit einer angegebenen Seitengröße

Die Export von Grafiken von Excel zu einer PDF-Datei mit einem genauen Seitenaufbau ist eine entscheidende Voraussetzung für die Berichterstattung und Datenvisualisierung. Dieses Tutorial zeigt, wie man Aspose.Cells für .NET verwendet, um eine PDF aus einem Excel-Diagramm zu erzeugen, während die Seitegröße und Ausrichtung angegeben werden. Sie lernen, wie man eine Excel- Datei lädt, auf ein Arbeitsblatt und sein Diagramm zugreift und in eine gut formatierte PDF-Datei konvertiert.

## Installationsanweisungen  
Um Aspose.Cells für .NET zu verwenden, folgen Sie diesen Schritten:  

1. Installieren Sie über NuGet:  
   Öffnen Sie den NuGet Package Manager in Visual Studio und laufen Sie:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Laden Sie die DLL herunter:  
   Holen Sie sich die neueste Version von der [Aspose.Cells für die .NET- Downloadseite](https://releases.aspose.com/cells/net/)\- Ich weiß .  

3. Fügen Sie den Namensbereich ein:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
