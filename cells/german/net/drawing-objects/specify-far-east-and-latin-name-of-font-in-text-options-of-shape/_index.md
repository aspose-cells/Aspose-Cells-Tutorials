# Erstellen Sie Excel-Arbeitsbuch mit Fernöstlichen und Latein-Schriftzeichen

Dieses Beispiel zeigt, wie man Aspose.Cells für .NET verwendet, um ein Excel- Workbuch zu erstellen, eine Textbox zu einem Arbeitsblatt hinzuzufügen und sowohl Fernöstliche als auch lateinische Schriftnamen für den Text innerhalb dieses Textplatzes festzulegen. Der Code zeigt die Initialisierung des Arbeitsplatzes, die Manipulation von Arbeitsblattelementen und die Speicherung des Arbeitsbüchens mit einer speziellen Schriftformatierung, die auf verschiedene Sprachanforderungen zugeschnitten ist.

## Installationsanweisungen
1. Installieren Sie Aspose.Cells für .NET: 
   * Nutzen Sie NuGet Package Manager in Ihrem Visual Studio-Projekt:
     ```
     Install-Package Aspose.Cells
     ```
   * Oder die DLL aus dem [Webseite zu verfolgen](https://www.aspose.com/downloads/cells/net)\- Ich weiß .

2. Hinzufügen von Referenzen: Stellen Sie sicher, dass Ihr Projekt die Bibliothek Aspose.Cells verweist.

3. Setzen Sie das Ausgabeverzeichnis ein: Aktualisieren Sie die `AusgabeDir`Variable im Code mit dem gewünschten Ausgangsweg, auf dem die Excel-Datei gespeichert wird.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
