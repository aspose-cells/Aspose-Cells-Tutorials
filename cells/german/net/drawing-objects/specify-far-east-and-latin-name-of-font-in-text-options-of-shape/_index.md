# Erstellen Sie eine Excel-Arbeitsmappe mit fernöstlichen und lateinischen Schriftarten

Dieses Beispiel zeigt, wie Sie mit Aspose.Cells für .NET eine Excel-Arbeitsmappe erstellen, ein Textfeld zu einem Arbeitsblatt hinzufügen und sowohl fernöstliche als auch lateinische Schriftarten für den Text darin festlegen. Der Code zeigt die Initialisierung des Arbeitsbereichs, die Bearbeitung von Arbeitsblattelementen und das Speichern der Arbeitsmappe mit spezifischer Schriftformatierung, die auf unterschiedliche Sprachanforderungen zugeschnitten ist.

## Installationsanweisungen
1. Installieren Sie Aspose.Cells für .NET: 
   - Verwenden Sie den NuGet-Paket-Manager in Ihrem Visual Studio-Projekt:
     ```
     Install-Package Aspose.Cells
     ```
   - Oder laden Sie die DLL von der [Aspose-Website](https://www.aspose.com/downloads/cells/net).

2. Referenzen hinzufügen: Stellen Sie sicher, dass Ihr Projekt auf die Aspose.Cells-Bibliothek verweist.

3. Ausgabeverzeichnis einrichten: Aktualisieren Sie das `outputDir` Variable im Code mit dem gewünschten Ausgabepfad, in dem die Excel-Datei gespeichert wird.

4. Beispiel ausführen: Kopieren Sie den bereitgestellten C#-Code in Ihr Projekt, kompilieren Sie ihn und führen Sie ihn aus. Sie sollten die Ausgabebestätigung sehen, und die Excel-Datei mit dem konfigurierten Textfeld wird im angegebenen Verzeichnis erstellt.
