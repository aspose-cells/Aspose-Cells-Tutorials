# Excel munkafüzetet készítsen a Távoli- keleti és latin írásszövegekkel

Ez a példa bemutatja, hogyan lehet használni az Aspose.Cells a .NET-hez egy Excel munkafüzet létrehozásához, egy szöveges doboz hozzáadásához egy munkafüzethez, és a textboxban lévő szövegre mind a Távoli-Kelet, mind a latin betűnéveket beállítja. A kód bemutatja a munkahelyi tér kezdeményezését, a munkafüzet elemek manipulálását, és a munkafüzetet különféle nyelvi követelményekre szabott betűformázással való mentését.

## A telepítési utasítások
1. A .NET-re telepítse az Aspose.Cells-t: 
   * Használja a NuGet csomagkezelőt a Visual Studio projektében:
     ```
     Install-Package Aspose.Cells
     ```
   * Vagy töltsd le a DLL-t a [Célszerű weboldal](https://www.aspose.com/downloads/cells/net)\- Nem.

2. Adj hivatkozásokat: A projektednek a Aspose.Cells könyvtárához kell hivatkoznia.

3. Beállítsa a kimeneti direktoriumot: frissítse a `KibocsátásDir`A kódban lévő változó a kívánt kimeneti útvonalon, ahol az Excel fájlt tároljuk.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
