## Руководство по внедрению

В этом разделе вы узнаете, как задать и получить размеры страницы с помощью Aspose.Cells для .NET.

### Установка размеров страницы

Настройка размеров бумаги имеет важное значение при подготовке документов для печати или цифрового распространения. Давайте рассмотрим эту функцию:

#### Шаг 1: Доступ к рабочему листу
Откройте рабочий лист, на котором вы хотите изменить параметры страницы:
```csharp
// Доступ к первому рабочему листу
Worksheet sheet = book.Worksheets[0];
```

#### Шаг 2: Настройка размера бумаги
Вы можете установить различные размеры бумаги, изменив `PaperSize` свойство:

- **Установите размер бумаги на A2**
    ```csharp
    // Установите размер бумаги на A2 и напечатайте ширину и высоту бумаги в дюймах.
    sheet.PageSetup.PaperSize = PaperSizeType.PaperA2;
    Console.WriteLine("PaperA2: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

- **Установить размер бумаги на A3**
    ```csharp
    // Установите размер бумаги на A3 и распечатайте ширину и высоту бумаги в дюймах.
    sheet.PageSetup.PaperSize = PaperSizeType.PaperA3;
    Console.WriteLine("PaperA3: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

- **Установить размер бумаги на A4**
    ```csharp
    // Установите размер бумаги на A4 и распечатайте ширину и высоту бумаги в дюймах.
    sheet.PageSetup.PaperSize = PaperSizeType.PaperA4;
    Console.WriteLine("PaperA4: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

- **Установить размер бумаги на Letter**
    ```csharp
    // Установите размер бумаги на Letter и напечатайте ширину и высоту бумаги в дюймах.
    sheet.PageSetup.PaperSize = PaperSizeType.PaperLetter;
    Console.WriteLine("PaperLetter: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

### Получение размеров страницы
