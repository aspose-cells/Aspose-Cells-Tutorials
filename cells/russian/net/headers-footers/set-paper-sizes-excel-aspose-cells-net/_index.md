В этом руководстве вы узнаете, как устанавливать и настраивать размеры бумаги в Excel с помощью Aspose.Cells для .NET — мощной библиотеки для управления электронными таблицами.

**Что вы узнаете:**
- Настройте среду разработки с помощью Aspose.Cells для .NET.
- Настройте пользовательские размеры бумаги, такие как A2, A3, A4 и Letter, в книге Excel.

## Руководство по внедрению
Давайте рассмотрим процесс настройки размеров бумаги для различных форматов.

### Установка размера бумаги на A2
#### Обзор
Настройте рабочий лист Excel для использования формата бумаги А2, подходящего для больших отпечатков и плакатов.

#### Шаги
**1. Создайте новый экземпляр рабочей книги**
```csharp
Workbook wb = new Workbook();
```

**2. Доступ к первому рабочему листу**
```csharp
Worksheet ws = wb.Worksheets[0];
```

**3. Установите размер бумаги на A2.**
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA2;
```

**4. Размеры дисплея в дюймах**
```csharp
Console.WriteLine("PaperA2: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
*Объяснение*: `PageSetup.PaperSize` свойство регулирует размер бумаги, в то время как `PaperWidth` и `PaperHeight` укажите размеры.
