## Руководство по внедрению
### Указание названий заданий при печати рабочих книг
#### Обзор
В этом разделе вы узнаете, как распечатать всю книгу Excel и указать имя задания, чтобы отличать его от других задач печати.

#### Шаги
**1. Создать объект рабочей книги**
Сначала загрузите исходный файл Excel:
```csharp
// Путь к исходному каталогу
string sourceDir = RunExamples.Get_SourceDirectory();

// Загрузить книгу из файла
Workbook workbook = new Workbook(sourceDir + "sampleSpecifyJobWhilePrinting.xlsx");
```

**2. Настройте принтер и имя задания**
Определите имя принтера и название задания для идентификации:
```csharp
string printerName = "doPDF 8"; // Измените на установленный принтер
string jobName = "My Job Name";
```

**3. Визуализация и печать рабочей книги**
Использовать `WorkbookRender` для управления печатью:
```csharp
// Настройте параметры рендеринга (здесь можно добавить дополнительные конфигурации)
ImageOrPrintOptions options = new ImageOrPrintOptions();

// Инициализируйте визуализацию рабочей книги с рабочей книгой и параметрами
WorkbookRender wr = new WorkbookRender(workbook, options);

try
{
    // Печать с использованием указанного принтера и имени задания
    wr.ToPrinter(printerName, jobName);
}
catch (Exception ex)
{
    Console.WriteLine("Error during printing: " + ex.Message);
}
```
### Печать определенных рабочих листов
#### Обзор
Если вам необходимо распечатать определенный рабочий лист с индивидуальным названием задания, выполните следующие действия.

**1. Доступ к рабочему листу**
Выберите рабочий лист из вашей рабочей тетради:
```csharp
// Доступ к первому рабочему листу
Worksheet worksheet = workbook.Worksheets[0];
```

**2. Визуализация и печать рабочего листа**
Использовать `SheetRender` для целевой печати:
```csharp
// Инициализируйте SheetRender с определенным рабочим листом и параметрами.
SheetRender sr = new SheetRender(worksheet, options);

try
{
    // Выполнить печать на указанном принтере с именем задания
    sr.ToPrinter(printerName, jobName);
}
catch (Exception ex)
{
    Console.WriteLine("Worksheet print error: " + ex.Message);
}
```
