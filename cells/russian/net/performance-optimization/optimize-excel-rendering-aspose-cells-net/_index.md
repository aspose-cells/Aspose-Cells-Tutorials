## Руководство по внедрению

Давайте разобьем реализацию на логические разделы, чтобы ее было легче отслеживать.

### Загрузка файла Excel

Начните с загрузки вашего шаблона файла Excel. Здесь вы примените настройки рендеринга:

```csharp
// Загрузить шаблон файла Excel
Workbook workbook = new Workbook(sourceDir + "sampleCrossType.xlsx");
```

### Настройка рендеринга PDF с помощью TextCrossType

Начнем с настройки параметров сохранения PDF-файла, чтобы обеспечить точность текста.

#### Инициализировать параметры сохранения PDF

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.TextCrossType = TextCrossType.StrictInCell;
```
*Здесь, `TextCrossType.StrictInCell` обеспечивает строгое выравнивание текста в пределах границ ячейки.*

### Сохранение файла Excel как PDF

Конвертируйте и сохраните ваш документ как файл PDF:

```csharp
using (FileStream pdfStream = new FileStream(outputDir + "outputCrossType.pdf", FileMode.Create))
{
    workbook.Save(pdfStream, pdfSaveOptions);
}
```

### Настройка рендеринга изображения с помощью TextCrossType

Далее настройте параметры рендеринга изображений, чтобы сохранить целостность текста на изображениях.

#### Инициализация параметров изображения или печати

```csharp
ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
imgOptions.TextCrossType = TextCrossType.StrictInCell;
```
*Одинаковый `TextCrossType` настройка обеспечивает единообразие в различных выходных форматах.*

### Рендеринг и сохранение в формате PNG-изображения

Преобразуйте лист Excel в изображение:

```csharp
SheetRender renderer = new SheetRender(workbook.Worksheets[0], imgOptions);
System.Drawing.Bitmap bitmap = renderer.ToImage(0);

using (FileStream pngStream = new FileStream(outputDir + "outputCrossType.png", FileMode.Create))
{
    bitmap.Save(pngStream, ImageFormat.Png);
}
```

### Советы по устранению неполадок

- **Отсутствующие файлы**: Убедитесь, что исходный и выходной каталоги настроены правильно.
- **Проблемы с рендерингом**: Проверьте, если `TextCrossType` настроен правильно, чтобы избежать смещения текста.
