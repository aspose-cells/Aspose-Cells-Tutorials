### Загружать большие файлы Excel

**Обзор:**
Загрузите большой файл Excel и измерьте время, необходимое для обработки его содержимого, перебирая рабочие листы и обновляя значения ячеек.

#### Шаг 1: Создание класса бенчмаркинга

Настройте свой класс с помощью необходимых методов:

```csharp
using System;
using Aspose.Cells;

namespace Aspose.Cells.Examples.CSharp.KnowledgeBase.Benchmarking
{
    public class LoadLargeExcelScenario1
    {
        public static void Run()
        {
            string dataDir = GetDataDirectory();
            try
            {
                CreateAsposeCellsFile(dataDir + "Sample.xls", dataDir + "output_out.xls");
            }
            catch (Exception ex)
            {
                Console.WriteLine(ex.Message);
            }
        }

        private static void CreateAsposeCellsFile(string filename_in, string filename_out)
        {
            DateTime start = DateTime.Now;
            Workbook workbook = new Workbook(filename_in);

            for (int i = 0; i < workbook.Worksheets.Count; i++)
            {
                Worksheet ws = workbook.Worksheets[i];
                ws.Cells[0, 0].PutValue("Data" + i.ToString());
            }

            workbook.Save(filename_out);
            DateTime end = DateTime.Now;
            TimeSpan time = end - start;

            Console.WriteLine($"File Updated!\nTime consumed (Seconds): {time.TotalSeconds}");
        }
        
        private static string GetDataDirectory()
        {
            return "path_to_your_data_directory\\";
        }
    }
}
```

#### Шаг 2: Поймите ключевые компоненты

- **Рабочая тетрадь:** Представляет файл Excel. Инициализируйте его с путем к вашему файлу.
- **Рабочий лист:** Доступ к каждому листу в рабочей книге осуществляется по его индексу.
- **Ячейки[0, 0].PutValue():** Обновляет ячейку A1 каждого рабочего листа.

#### Шаг 3: Измерение производительности

Использовать `DateTime` объекты для регистрации времени до и после обработки, расчета общего затраченного времени для сравнительного анализа производительности.
