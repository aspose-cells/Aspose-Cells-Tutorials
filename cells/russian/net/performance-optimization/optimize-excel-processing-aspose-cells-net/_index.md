## Руководство по внедрению

Это руководство разделено на три основные части: загрузка рабочей книги, настройка параметров расчета и расчетные формулы.

### Функция 1: Загрузка рабочей книги

Загрузка существующего файла Excel в `Workbook` объект прост. Это позволяет вам манипулировать данными программно.

#### Пошаговая реализация:

**3.1 Настройка исходного каталога**
Определите исходный каталог, в котором находится ваш шаблон рабочей книги.

```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
```

**3.2 Загрузка файла Excel**
Создать `Workbook` экземпляр и открыть существующий файл.

```csharp
// Загрузить книгу по указанному пути
Workbook workbook = new Workbook(sourceDir + "book1.xls");
```

### Функция 2: Настройка параметров расчета

Оптимизация расчетов формул имеет решающее значение для производительности, особенно в больших книгах. Вот как отключить настройку цепочки расчетов.

#### Пошаговая реализация:

**3.3 Доступ к FormulaSettings**
Доступ и изменение `FormulaSettings` в настройках рабочей книги.

```csharp
// Отключите цепочку вычислений для оптимизации производительности
workbook.Settings.FormulaSettings.EnableCalculationChain = false;
```

### Функция 3: Расчет формул рабочей книги

После настройки убедитесь, что все формулы рассчитаны правильно.

#### Пошаговая реализация:

**3.4 Формулы расчета**
Вызовите метод для вычисления всех формул в рабочей книге.

```csharp
// Обработать все формулы в рабочей книге
workbook.CalculateFormula();
```
