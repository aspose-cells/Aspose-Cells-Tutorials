## Руководство по внедрению

Теперь, когда у вас есть базовые настройки, давайте перейдем к реализации различных уровней сжатия.

### Регулировка уровней сжатия

#### Обзор

Сжатие в файлах Excel помогает уменьшить размер файла, что упрощает его хранение и распространение. Aspose.Cells обеспечивает несколько уровней сжатия, от уровня 1 (самый быстрый) до уровня 9 (максимальное сжатие).

#### Пошаговая реализация

##### Шаг 1: Загрузите свою рабочую книгу

```csharp
using Aspose.Cells;
using System.Diagnostics;

// Укажите исходный и выходной каталоги
cstring sourceDir = "your_source_directory_path";
cstring outDir = "your_output_directory_path";

Workbook workbook = new Workbook(sourceDir + "LargeSampleFile.xlsx");
```

##### Шаг 2: Установите уровень сжатия

Чтобы отрегулировать уровень сжатия, используйте `XlsbSaveOptions`:

```csharp
XlsbSaveOptions options = new XlsbSaveOptions();
options.CompressionType = OoxmlCompressionType.Level1;
```

##### Шаг 3: Сохранение со сжатием

Измерьте и сохраните файл, используя указанный тип сжатия:

```csharp
var watch = Stopwatch.StartNew();
workbook.Save(outDir + "LargeSampleFile_level_1_out.xlsb", options);
watch.Stop();

Console.WriteLine("Level 1 Elapsed Time: " + watch.ElapsedMilliseconds);
```

Повторите эти шаги для других уровней (Уровень 6 и Уровень 9), регулируя `options.CompressionType` соответственно.

#### Объяснение параметров
- **Тип сжатия:** Определяет уровень сжатия. Более высокие уровни уменьшают размер больше, но требуют больше времени для обработки.
- **СохранитьПараметры:** Настройте дополнительные параметры сохранения, такие как формат и параметры шифрования.

### Советы по устранению неполадок

- Убедитесь, что путь к исходному каталогу указан правильно.
- Если размеры файлов существенно не уменьшаются, проверьте сложность данных и попробуйте разные уровни сжатия.
