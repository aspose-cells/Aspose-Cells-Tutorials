#### Создание и настройка рабочей книги с помощью Aspose.Cells

**Обзор**

Создание рабочей книги — ваш первый шаг к автоматизации Excel. В этом разделе вы узнаете, как инициализировать рабочую книгу, получить доступ к ее рабочим листам и сохранить ее в соответствующем формате.

##### Шаг 1: Инициализация рабочей книги

```csharp
using System;
using Aspose.Cells;

string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Создать новый экземпляр Workbook
Workbook workbook = new Workbook();
```

The `Workbook` class представляет ваш файл Excel. Создавая экземпляр, вы по сути готовитесь к программной работе с этим файлом.

##### Шаг 2: Доступ к первому рабочему листу

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

Каждая рабочая книга содержит набор рабочих листов. Здесь мы получаем доступ к первому рабочему листу по индексу `0`.

##### Шаг 3: Сохраните рабочую книгу

```csharp
// Сохраните книгу в формате xlsx.
workbook.Save(outputDir + "outputCreateWorkbook.xlsx");
```

На этом этапе ваши изменения будут записаны в файл Excel.

#### Добавить и настроить фигуру TextBox с текстом

**Обзор**

Добавление таких фигур, как текстовые поля, может улучшить визуальную привлекательность ваших электронных таблиц. В этом разделе показано добавление формы текстового поля и настройка его содержимого и размера шрифта.

##### Шаг 1: Создание текстового поля

```csharp
using Aspose.Cells.Drawing;

// Добавить текстовое поле на рабочий лист
TextBox textbox = worksheet.Shapes.AddTextBox(0, 0, 0, 0, 100, 700);
textbox.Text = "Aspose File Format APIs";
textbox.Font.Size = 44;
```

The `AddTextBox` Метод позволяет указать позицию и размер. Здесь мы задаем пользовательский текст и размер шрифта.

##### Шаг 2: Сохраните рабочую книгу

```csharp
// Сохраните изменения с добавленным текстовым полем.
workbook.Save(outputDir + "outputAddTextbox.xlsx");
```

После добавления фигур обязательно сохраните изменения.

#### Применить предустановленный стиль WordArt к тексту TextBox

**Обзор**

Улучшите презентацию текста, применив предустановленные стили, такие как WordArt. В этом разделе показано, как применить стиль к тексту в форме текстового поля.

##### Шаг 1: Установите стиль WordArt

```csharp
FontSetting fntSetting = textbox.GetCharacters()[0] as FontSetting;
fntSetting.SetWordArtStyle(PresetWordArtStyle.WordArtStyle3);
```

Использовать `SetWordArtStyle` применять предопределенные стили, улучшая эстетику текста.

##### Шаг 2: Сохраните рабочую книгу

```csharp
// Сохраните книгу с примененным стилем WordArt.
workbook.Save(outputDir + "outputSetPresetWordArtStyle.xlsx");
```

Завершите изменения, сохранив книгу.
