## Руководство по внедрению

Мы разобьем процесс создания пузырьковой диаграммы на логические этапы.

### Создание и заполнение данных для серии диаграмм
Перед добавлением диаграммы заполните рабочий лист данными:
1. **Создание экземпляра объекта Workbook**
   ```csharp
   // Создать экземпляр объекта Workbook
   Workbook workbook = new Workbook();
   ```
2. **Получите ссылку на первый рабочий лист**
   ```csharp
   // Доступ к первому рабочему листу в рабочей книге
   Worksheet worksheet = workbook.Worksheets[0];
   ```
3. **Заполните данные для серии диаграммы**
   Заполните столбцы данных значениями Y, размером пузырька и значениями X:
   
   - **Значения Y**: Номера 2, 4 и 6.
   - **Размер пузырька**: Размеры обозначены цифрами 2, 3 и 1.
   - **Значения X**: Последовательность 1, 2 и 3.

   ```csharp
   // Заполните значения Y
   worksheet.Cells[0, 0].PutValue("Y Values");
   worksheet.Cells[0, 1].PutValue(2);
   worksheet.Cells[0, 2].PutValue(4);
   worksheet.Cells[0, 3].PutValue(6);

   // Заполните размер пузыря
   worksheet.Cells[1, 0].PutValue("Bubble Size");
   worksheet.Cells[1, 1].PutValue(2);
   worksheet.Cells[1, 2].PutValue(3);
   worksheet.Cells[1, 3].PutValue(1);

   // Заполните значения X
   worksheet.Cells[2, 0].PutValue("X Values");
   worksheet.Cells[2, 1].PutValue(1);
   worksheet.Cells[2, 2].PutValue(2);
   worksheet.Cells[2, 3].PutValue(3);
   ```

### Добавление и настройка пузырьковой диаграммы
Добавьте пузырьковую диаграмму на свой рабочий лист:
4. **Добавить диаграмму**
   ```csharp
   // Добавить новую пузырьковую диаграмму в указанное место на рабочем листе
   int chartIndex = worksheet.Charts.Add(Aspose.Cells.Charts.ChartType.Bubble, 5, 0, 25, 10);
   ```
5. **Доступ к диаграмме и ее настройка**
   Настройте источники данных для пузырьковой диаграммы:
   
   ```csharp
   // Доступ к недавно добавленному экземпляру диаграммы
   Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];

   // Добавить SeriesCollection (источник данных) в диапазон диаграммы
   chart.NSeries.Add("B1:D1", true);

   // Установите значения Y
   chart.NSeries[0].Values = "B1:D1";

   // Назначить размеры пузырьков
   chart.NSeries[0].BubbleSizes = "B2:D2";

   // Определить значения оси X
   chart.NSeries[0].XValues = "B3:D3";
   ```
6. **Сохраните файл Excel**
   Сохраните книгу, чтобы сохранить все изменения:
   
   ```csharp
   // Сохраните полученный файл Excel.
   workbook.Save(outputDir + "outputHowToCreateBubbleChart.xlsx");
   ```

1. **Что такое пузырьковая диаграмма?**
