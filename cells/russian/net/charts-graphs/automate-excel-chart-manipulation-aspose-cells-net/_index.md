Чтобы установить Aspose.Cells, используйте либо **.NET CLI** или **Консоль менеджера пакетов**:

## Руководство по внедрению

В этом разделе мы рассмотрим шаги, необходимые для чтения и обработки диаграмм в файле Excel 2016.

### Доступ к диаграммам на рабочем листе

Начнем с загрузки нашей исходной рабочей книги и доступа к ее первому рабочему листу, содержащему наши диаграммы:

```csharp
// Загрузите файл Excel
Workbook book = new Workbook("sampleReadAndManipulateExcel2016Charts.xlsx");

// Доступ к первому рабочему листу
Worksheet sheet = book.Worksheets[0];
```

### Типы диаграмм чтения

Далее мы проходим по каждой диаграмме на рабочем листе, чтобы прочитать ее тип и распечатать его:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    // Получить текущий график
    Chart ch = sheet.Charts[i];

    // Распечатать тип диаграммы
    Console.WriteLine(ch.Type);
}
```

### Изменение названий диаграмм

Мы можем изменить название каждой диаграммы, чтобы оно отражало ее тип:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    Chart ch = sheet.Charts[i];

    // Обновить заголовок диаграммы
    ch.Title.Text = "Chart Type is " + ch.Type.ToString();
}
```

### Сохранение изменений

Наконец, сохраните ваши изменения в новом файле Excel:

```csharp
book.Save("outputReadAndManipulateExcel2016Charts.xlsx");
Console.WriteLine("Manipulation completed successfully.");
```
