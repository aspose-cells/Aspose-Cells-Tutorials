## Руководство по внедрению

### Определить ось на диаграмме

Основная цель здесь — определить, какие оси присутствуют в диаграмме. Это может быть критически важно для настройки и точной интерпретации ваших данных.

#### Доступ к рабочему листу и диаграмме

Сначала загрузите рабочую книгу и откройте ее рабочий лист:

```csharp
// Исходный каталог
string sourceDir = "path_to_directory";

// Загрузить существующий файл Excel
Workbook workbook = new Workbook(sourceDir + "sampleDetermineAxisInChart.xlsx");

// Доступ к первому рабочему листу в рабочей книге
Worksheet worksheet = workbook.Worksheets[0];
```

#### Проверка наличия осей

Теперь определим, какие оси присутствуют:

```csharp
// Доступ к первой диаграмме на рабочем листе
Chart chart = worksheet.Charts[0];

// Проверьте наличие осей первичной и вторичной категории
bool hasPrimaryCategoryAxis = chart.HasAxis(AxisType.Category, true);
Console.WriteLine("Has Primary Category Axis: " + hasPrimaryCategoryAxis);

bool hasSecondaryCategoryAxis = chart.HasAxis(AxisType.Category, false);
Console.WriteLine("Has Secondary Category Axis: " + hasSecondaryCategoryAxis);

// Проверка осей значений
bool hasPrimaryValueAxis = chart.HasAxis(AxisType.Value, true);
Console.WriteLine("Has Primary Value Axis: " + hasPrimaryValueAxis);

bool hasSecondaryValueAxis = chart.HasAxis(AxisType.Value, false);
Console.WriteLine("Has Secondary Value Axis: " + hasSecondaryValueAxis);
```

**Объяснение:** 
- `chart.HasAxis(AxisType.Category, true/false)` проверки осей первичной/вторичной категории.
- `chart.HasAxis(AxisType.Value, true/false)` проверяет наличие осей значений.

## Раздел часто задаваемых вопросов

**В: Как проверить обе оси диаграммы с помощью Aspose.Cells?**
А: Использовать `chart.HasAxis(AxisType.Category, true/false)` и `chart.HasAxis(AxisType.Value, true/false)`.

**В: Есть ли способ работать с несколькими диаграммами в одной книге?**
