## Руководство по внедрению
### Создание диаграммы PDF с пользовательским размером страницы
В этом разделе объясняется, как экспортировать диаграммы в формат PDF, указав желаемый размер страницы.

#### Шаг 1: Загрузите файл Excel
Загрузите пример файла Excel, содержащий диаграмму, которую вы хотите экспортировать:
```csharp
Workbook wb = new Workbook("sampleCreateChartPDFWithDesiredPageSize.xlsx");
```

#### Шаг 2: Доступ к рабочему листу и диаграмме
Доступ к рабочему листу и диаграмме из вашей рабочей книги. Обычно вы начинаете с доступа к первому рабочему листу и диаграмме.
```csharp
Worksheet ws = wb.Worksheets[0];	Chart ch = ws.Charts[0];
```

#### Шаг 3: Экспортируйте диаграмму в PDF с пользовательским размером страницы
Используйте `ToPdf` метод экспорта диаграммы в PDF, указав пользовательские размеры. Здесь мы устанавливаем ширину и высоту на 7 дюймов.
```csharp
ch.ToPdf("outputCreateChartPDFWithDesiredPageSize.pdf", 7, 7, 	PageLayoutAlignmentType.Center, PageLayoutAlignmentType.Center);
```

**Объясняемые параметры:**
- **Путь к файлу:** Место назначения выходного PDF-файла.
- **Ширина и высота:** Размеры указаны в дюймах.
- **Типы выравнивания макета страницы:** Укажите параметры выравнивания для центрирования.

### Советы по устранению неполадок
- Убедитесь, что у вас есть соответствующие разрешения на чтение/запись файлов.
- Убедитесь, что ваш файл Excel содержит хотя бы одну диаграмму.
