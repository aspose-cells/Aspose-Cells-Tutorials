## Руководство по внедрению
### Настройка меток диаграммы на основе локали
Основная цель — переопределить текст по умолчанию для меток круговой диаграммы с использованием настроек, зависящих от языка и региональных параметров. Вот как этого можно добиться:

#### 1. Загрузите рабочую книгу и получите доступ к диаграмме.
Начните с загрузки существующего файла Excel, содержащего круговую диаграмму:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleCustomTextForLabels.xlsx");
```

Получите доступ к рабочему листу и диаграмме, которые вы хотите настроить:
```csharp
Worksheet sheet = book.Worksheets[0];
Chart chart = sheet.Charts[0];
```

#### 2. Установите параметры глобализации
Переопределить `GetOtherName` метод предоставления пользовательских меток на основе локали системы:

```csharp
GlobalizationSettings globalSettings = new GlobalizationSettings();
globalSettings.ChartSettings = new CustomSettings();
book.Settings.GlobalizationSettings = globalSettings;
```

Определите свой класс пользовательских настроек:
```csharp
class CustomSettings : ChartGlobalizationSettings
{
    public override string GetOtherName()
    {
        int lcid = System.Globalization.CultureInfo.CurrentCulture.LCID;
        switch (lcid)
        {
            case 1033: // Английский
                return "Other";
            case 1036: // Французский
                return "Autre";
            case 1031: // немецкий
                return "Andere";
            default:
                return base.GetOtherName();
        }
    }
}
```

#### 3. Обновите и отобразите диаграмму
Чтобы применить изменения, обновите диаграмму и преобразуйте ее в файл изображения:

```csharp
chart.Calculate();
chart.ToImage(outputDir + "outputCustomTextForLabels.png", new ImageOrPrintOptions());
Console.WriteLine("CustomTextForLabels executed successfully.");
```

### Советы по устранению неполадок
- **Отсутствует диаграмма**: Убедитесь, что на первом листе вашего файла Excel есть диаграмма.
- **Несоответствие культур**: Убедитесь, что региональные настройки вашей системы соответствуют выбранным вами.
