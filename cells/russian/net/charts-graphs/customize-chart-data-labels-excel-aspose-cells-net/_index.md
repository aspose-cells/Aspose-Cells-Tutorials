Инициализируйте Aspose.Cells и загрузите файл Excel:

## Руководство по внедрению

### Настройка типа формы меток данных (H2)

Настройте формы меток данных, чтобы улучшить визуальные эффекты диаграммы.

#### Шаг 1: Доступ к диаграмме и рядам (H3)

Доступ к нужному рабочему листу и диаграмме:
```csharp
// Доступ к первому рабочему листу в рабочей книге
Worksheet ws = wb.Worksheets[0];

// Доступ к первой диаграмме на рабочем листе
Chart ch = ws.Charts[0];
```

#### Шаг 2: Изменение формы метки данных (H3)

Установите тип формы меток данных на WedgeEllipseCallout:
```csharp
// Доступ к первой серии в диаграмме
Series srs = ch.NSeries[0];

// Установите тип формы меток данных
srs.DataLabels.ShapeType = DataLabelShapeType.WedgeEllipseCallout;
```
The `DataLabelShapeType` параметр предлагает различные формы для улучшения визуального повествования.

#### Шаг 3: Сохранение изменений (H3)

Сохраните изменения в новом файле:
```csharp
// Сохраните измененный файл Excel.
wb.Save(outputDir + "/outputSetShapeTypeOfDataLabelsOfChart.xlsx");
```
**Советы по устранению неполадок:**
- Проверьте пути и существование каталогов.
- Проверьте права доступа к файлу при сохранении.
