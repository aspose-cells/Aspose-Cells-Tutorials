# Как создать Workbook в C# – Записать значение и отформатировать число

Создание workbook в C# — распространённая задача, когда необходимо генерировать файлы Excel «на лету». В этом руководстве мы пошагово покажем, как записать значение в ячейку и отформатировать число в ячейке, чтобы итоговый лист выглядел аккуратно.

Если вы когда‑нибудь смотрели на пустую таблицу и задавались вопросом, почему числа отображаются с слишком большим количеством знаков после запятой, вы не одиноки. Мы рассмотрим всё — от инициализации объекта workbook до установки пользовательского формата числа, а также поделимся несколькими советами для крайних случаев, с которыми вы можете столкнуться позже.

## Что вы узнаете

- **Инициализировать** новый экземпляр `Workbook`.  
- **Записать значение в ячейку** с помощью метода `PutValue`.  
- **Установить числовой формат ячейки** с объектом `Style`, получая чистое отображение с двумя знаками после запятой.  
- Проверить результат, считав значение ячейки обратно или открыв файл в Excel.  

Никакие внешние библиотеки, кроме стандартного Aspose.Cells (или любого аналогичного API), не требуются, и код работает на .NET 6+ без дополнительной настройки.

---

## Как создать Workbook – инициализация объекта

Для начала вам нужен объект workbook, который будет содержать листы. Представьте `Workbook` как весь файл Excel, а каждый `Worksheet` — как отдельную вкладку.

```csharp
// Step 1: Create a new workbook instance
Workbook workbook = new Workbook();
```

*Почему это важно:* Создание workbook выделяет внутренние структуры, которые позже будут хранить строки, столбцы и форматирование. Без этого объекта некуда записывать значение в ячейку.

> **Pro tip:** Если вы планируете работать с существующим файлом, замените `new Workbook()` на `new Workbook("template.xlsx")`, чтобы загрузить шаблон и сохранить его стили.

## Записать значение в ячейку

Теперь, когда у нас есть workbook, давайте запишем число в ячейку **A1** первого листа.

```csharp
// Step 2: Access cell A1 in the first worksheet
Cell cellA1 = workbook.Worksheets[0].Cells["A1"];

// Step 3: Insert a numeric value into the cell
cellA1.PutValue(123.456789);
```

*Почему мы используем `PutValue`*: Этот метод автоматически определяет тип данных, поэтому вам не нужно вручную приводить или конвертировать тип. Он также сохраняет существующий стиль ячейки, что удобно, когда позже вы **устанавливаете числовой формат ячейки**.

### Быстрая проверка

Если вы считаете значение ячейки обратно, вы увидите исходное значение:

```csharp
double raw = cellA1.DoubleValue; // raw == 123.456789
```

Это число до применения любого форматирования.

## Установить числовой формат ячейки

Отображение двойного числа с множеством знаков после запятой не всегда удобно для пользователя. Ограничим его двумя знаками после запятой.

```csharp
// Step 4: Apply a style that formats the number with two significant digits
cellA1.SetStyle(new Style() { Number = 2 });
```

`Свойство` `Number` соответствует встроенным идентификаторам числовых форматов Excel. `2` означает «Число с двумя знаками после запятой». Если нужен другой формат — например, валюта или дата — используйте другой ID или строку пользовательского формата.

### Альтернатива: строка пользовательского формата

```csharp
Style customStyle = workbook.CreateStyle();
customStyle.Custom = "#,##0.00"; // forces two decimals with thousand separator
cellA1.SetStyle(customStyle);
```

*Почему выбирать пользовательский стиль?* Он дает полный контроль, особенно когда встроенные ID не покрывают ваши региональные настройки.

## Проверка вывода (необязательно, но рекомендуется)

После применения стиля вы можете сохранить workbook и открыть его в Excel, чтобы убедиться в правильном отображении.

```csharp
// Save the workbook to a file
workbook.Save("FormattedWorkbook.xlsx");

// Or, for quick verification in code:
string displayed = cellA1.StringValue; // "123.46"
Console.WriteLine($"Displayed value: {displayed}");
```

Вы должны увидеть **123.46** в ячейке A1 — ровно два знака после запятой, благодаря установленному формату.

### Полный рабочий пример

Собрав всё вместе, представляем автономную программу, которую вы можете скопировать и вставить в консольное приложение.

```csharp
using System;
using Aspose.Cells;   // Ensure you have the Aspose.Cells NuGet package

class Program
{
    static void Main()
    {
        // Initialize the workbook
        Workbook workbook = new Workbook();

        // Access the first worksheet and cell A1
        Cell cellA1 = workbook.Worksheets[0].Cells["A1"];

        // Write a numeric value
        cellA1.PutValue(123.456789);

        // Apply a two‑decimal number format
        cellA1.SetStyle(new Style() { Number = 2 });

        // Save to disk (optional)
        workbook.Save("FormattedWorkbook.xlsx");

        // Output the displayed text for verification
        Console.WriteLine($"Cell A1 shows: {cellA1.StringValue}");
    }
}
```

**Ожидаемый вывод при запуске программы:**

```
Cell A1 shows: 123.46
```

Откройте `FormattedWorkbook.xlsx` в Excel, и вы увидите то же отформатированное значение.

## Распространённые варианты и крайние случаи

### 1. Разные числовые форматы

| Цель | ID формата | Фрагмент кода |
|------|-----------|--------------|
| Currency (two decimals) | 5 | `cellA1.SetStyle(new Style() { Number = 5 });` |
| Percentage (no decimals) | 10 | `cellA1.SetStyle(new Style() { Number = 10 });` |
| Scientific notation | 11 | `cellA1.SetStyle(new Style() { Number = 11 });` |

Если ни один из встроенных ID не подходит, используйте пользовательскую строку, как показано ранее.

### 2. Культурно‑специфичные десятичные разделители

В некоторых локалях в качестве десятичного разделителя используется запятая. Вы можете задать формат, учитывающий культуру:

```csharp
Style cultureStyle = workbook.CreateStyle();
cultureStyle.Custom = "#,##0.00"; // works for most European locales
cellA1.SetStyle(cultureStyle);
```

### 3. Запись текста вместо чисел

Когда вам нужно **как записать ячейку** строкой, просто передайте строку в `PutValue`:

```csharp
cellA1.PutValue("Total Revenue");
```

Формат числа не требуется, но вы всё равно можете применить стили шрифта.

### 4. Большие наборы данных

Если вы заполняете тысячи строк, пакетная вставка (`Cells.ImportArray`) быстрее, чем цикл с `PutValue`. Подход к форматированию остаётся тем же; просто примените стиль к диапазону:

```csharp
Range range = workbook.Worksheets[0].Cells.CreateRange("B2:B1001");
range.ApplyStyle(new Style() { Number = 2 });
```

## Часто задаваемые вопросы

**Q: Работает ли это с .NET Core?**  
A: Абсолютно. Aspose.Cells поддерживает .NET Standard 2.0 и более новые версии, поэтому вы можете нацеливаться на .NET 5, .NET 6 или .NET 7 без изменений.

**Q: Что делать, если нужно более двух знаков после запятой?**  
A: Измените свойство `Number` на соответствующий встроенный ID (например, `3` для трёх знаков) или поправьте строку пользовательского формата (`"#,##0.000"`).

**Q: Можно ли применить формат ко всей колонке сразу?**  
A: Да. Используйте `Cells["A:A"]`, чтобы получить всю колонку, а затем `SetStyle`.
