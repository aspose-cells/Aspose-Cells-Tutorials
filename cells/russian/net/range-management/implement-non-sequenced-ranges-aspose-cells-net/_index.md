### Создание непоследовательных диапазонов в Excel

**Обзор**
Непоследовательные диапазоны позволяют ссылаться на несколько отдельных групп ячеек в таблице Excel. Эта функция особенно полезна при работе с наборами данных, которые не являются смежными, но логически сгруппированы вместе.

#### Пошаговая реализация

1. **Создание экземпляра объекта Workbook**

   Начните с создания нового экземпляра рабочей книги:

   ```csharp
   using Aspose.Cells;

   // Создать новый объект Workbook
   Workbook workbook = new Workbook();
   ```

2. **Добавить имя для неупорядоченного диапазона**

   Присвойте диапазону имя, которое позволит легко ссылаться на него в формулах и скриптах.

   ```csharp
   int index = workbook.Worksheets.Names.Add("NonSequencedRange");
   Name name = workbook.Worksheets.Names[index];
   ```

3. **Определите несеквенированные диапазоны ячеек**

   Используйте синтаксис формулы для указания групп ячеек. Вот как можно определить диапазоны, например `A1:B3` и `D5:E6` на Листе1:

   ```csharp
   // Определить непоследовательный диапазон
   name.RefersTo = "=Sheet1!$A$1:$B$3,Sheet1!$D$5:$E$6";
   ```

4. **Сохранить рабочую книгу**

   Наконец, сохраните вашу рабочую книгу в желаемом выходном каталоге.

   ```csharp
   string outputDir = RunExamples.Get_OutputDirectory();
   workbook.Save(outputDir + "outputImplementingNonSequencedRanges.xlsx");

   Console.WriteLine("Non-Sequenced Ranges implementation executed successfully.");
   ```

### Советы по устранению неполадок

- Убедитесь, что названия листов и ссылки на ячейки указаны правильно.
- Проверьте наличие синтаксических ошибок в `RefersTo` нить.
