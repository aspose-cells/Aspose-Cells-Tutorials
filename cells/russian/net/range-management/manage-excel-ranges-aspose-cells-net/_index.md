## Руководство по внедрению

### Создайте и назовите диапазон ячеек на листе Excel
Эта функция покажет вам, как создать определенный диапазон на рабочем листе и присвоить ему имя для удобства использования.

#### Обзор
Вы узнаете, как определить диапазон ячеек от A1 до C10 и дать этому диапазону имя с указанием ссылки на лист, что сделает ваши данные более доступными.

#### Этапы внедрения

##### Шаг 1: Инициализация рабочей книги
Создать экземпляр `Workbook` который представляет собой файл Excel.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создать новый объект Workbook
Workbook workbook = new Workbook();
```

##### Шаг 2: Доступ к коллекции рабочих листов и ячеек
Получите доступ к первому листу вашей рабочей книги и его коллекции ячеек.
```csharp
// Получить первый рабочий лист рабочей книги
Worksheet sheet = workbook.Worksheets[0];

// Доступ к коллекции ячеек рабочего листа
Cells cells = sheet.Cells;
```

##### Шаг 3: Создайте диапазон ячеек
Определите диапазон внутри ячеек, указав желаемые начальную и конечную позиции.
```csharp
// Создайте диапазон ячеек от A1 до C10
Range localRange = cells.CreateRange("A1", "C10");
```

##### Шаг 4: Присвойте имя с помощью ссылки на лист
Дайте созданному диапазону имя для более легкой идентификации и ссылки в формулах или скриптах.
```csharp
// Присвойте созданному диапазону имя со ссылкой на лист.
localRange.Name = "Sheet1!local";
```

##### Шаг 5: Сохраните свою рабочую книгу
Сохраните изменения, сохранив книгу в указанном каталоге.
```csharp
// Сохраните книгу в указанном выходном каталоге.
workbook.Save(Path.Combine(outputDir, "outputWorksheetNamedRange.xlsx"));
```
