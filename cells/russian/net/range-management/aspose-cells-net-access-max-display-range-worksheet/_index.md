## Руководство по внедрению

Давайте шаг за шагом рассмотрим максимальный диапазон отображения рабочего листа.

### Обзор

Доступ к максимальному диапазону отображения позволяет понять, какая часть листа Excel видна. Это полезно для больших наборов данных, где только подмножество может отображаться в любой момент времени.

#### Шаг 1: Создание экземпляра объекта Workbook

Создайте экземпляр `Workbook` класс для загрузки вашего файла Excel:

```csharp
// Исходный каталог
total_sourceDir = RunExamples.Get_SourceDirectory();

// Создать экземпляр объекта Workbook
Workbook workbook = new Workbook(sourceDir + "sampleAccessingMaximumDisplayRangeofWorksheet.xlsx");
```

#### Шаг 2: Доступ к рабочему листу

Получите рабочий лист, с которым вы хотите работать. Обычно это первый лист:

```csharp
// Доступ к первой рабочей книге
Worksheet worksheet = workbook.Worksheets[0];
```

#### Шаг 3: Получите максимальный диапазон отображения

Используйте `MaxDisplayRange` собственность `Cells` сбор для получения диапазона:

```csharp
// Доступ к максимальному диапазону отображения
Range range = worksheet.Cells.MaxDisplayRange;
```

#### Шаг 4: Вывод результата

Распечатайте или используйте информацию о максимальном диапазоне отображения по мере необходимости:

```csharp
// Распечатать свойство RefersTo максимального диапазона отображения
Console.WriteLine("Maximum Display Range: " + range.RefersTo);
Console.WriteLine("AccessingMaximumDisplayRangeofWorksheet executed successfully.");
```

### Советы по устранению неполадок
- **Файл не найден**: Проверьте правильность пути к исходному каталогу.
- **Исключение нулевой ссылки**: Убедитесь, что индекс рабочего листа существует.
