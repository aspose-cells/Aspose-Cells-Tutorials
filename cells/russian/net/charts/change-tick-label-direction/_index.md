# Изменить направление таблицы на тесте на этикетке в Excel с помощью Aspose.Cells 

Этот пример демонстрирует, как изменить направление табличных теток в рабочем листе Excel с использованием Aspose.Cells для .NET. При помощи регулирования текстового направления теток категории оси можно улучшить читаемость данных графики. Код загружает существующую рабочую книгу Excel, получает доступ к первому рабочему листу, восстанавливает график, устанавливает направление теток на горизонтальную, а затем сохраняет измененную рабочую книгу. Этот подход обеспечивает точный контроль над ориентацией теток на графике в ваших приложениях .NET.
