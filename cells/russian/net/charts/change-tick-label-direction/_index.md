# Изменение направления меток делений диаграммы в Excel с помощью Aspose.Cells 

В этом примере показано, как изменить направление меток делений диаграммы на листе Excel с помощью Aspose.Cells for .NET. Изменяя направление текста меток делений осей категорий, можно улучшить читаемость данных диаграммы. Код загружает существующую книгу Excel, обращается к первому листу, извлекает диаграмму, устанавливает горизонтальное направление меток делений, а затем сохраняет измененную книгу. Такой подход обеспечивает точный контроль над ориентацией меток диаграммы в ваших приложениях .NET.

## Инструкция по установке  
Чтобы запустить этот пример, выполните следующие действия:  

1. Установить Aspose.Cells для .NET  
   - Скачать с [Aspose.Cells для .NET](https://releases.aspose.com/cells/net/)  
   - Или установите через NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Настройте свой проект  
   - Создайте новый проект .NET.  
   - Добавьте ссылку на Aspose.Cells.dll.  

3. Запустить код  
   - Заменять `"Your Document Directory"` с фактическим путем к файлу.  
   - Запустите программу, чтобы обновить направление меток делений на графике.
