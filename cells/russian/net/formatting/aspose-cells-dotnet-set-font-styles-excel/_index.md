### Доступ к стилям ячеек и их изменение

Суть этого урока — управление стилем шрифта. Вот как:

#### Установка жирного шрифта

Чтобы сделать текст жирным, обратитесь к объекту стиля нужной ячейки:

```csharp
// Доступ к ячейке «А1».
Aspose.Cells.Cell cell = worksheet.Cells["A1"];

// Добавьте значение в ячейку.
cell.PutValue("Hello Aspose!");

// Получить объект стиля, связанный с ячейкой.
Style style = cell.GetStyle();

// Установите жирный шрифт.
style.Font.IsBold = true;

// Примените стиль обратно к ячейке.
cell.SetStyle(style);
```

#### Пояснение к коду
- **ПолучитьСтиль()**: Извлекает текущие настройки стиля ячейки.
- **Шрифт.IsBold**: Свойство, которое управляет жирностью текста. Установка его в `true` применяется полужирное форматирование.

### Сохранение файла Excel

Наконец, сохраните книгу, чтобы сохранить изменения:

```csharp
string outputPath = "Path_to_output_directory\\styledWorkbook.xls";
workbook.Save(outputPath, SaveFormat.Excel97To2003);
```
