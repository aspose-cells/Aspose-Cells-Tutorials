### Указание пользовательского форматирования шаблона в Aspose.Cells

Применяйте пользовательские форматы с помощью шаблонов форматирования DBNum для индивидуального представления данных.

#### Обзор

Пользовательское форматирование шаблонов может улучшить внешний вид данных, например, отображение валюты или процентное форматирование.

#### Этапы внедрения (H3)
1. **Создать рабочую книгу**
   Инициализируйте новый объект рабочей книги:
   ```csharp
   Workbook wb = new Workbook();
   ```
2. **Доступ к ячейкам и их изменение**
   Откройте первый рабочий лист и измените ячейку A1:
   ```csharp
   Worksheet ws = wb.Worksheets[0];
   Cell cell = ws.Cells["A1"];
   cell.PutValue(123);
   ```
3. **Применить форматирование пользовательского шаблона**
   Получить и установить пользовательский стиль:
   ```csharp
   Style st = cell.GetStyle();
   st.Custom = "[DBNum2][$-804]General";
   cell.SetStyle(st);
   ```
   *Объяснение*: `Custom` свойство позволяет устанавливать определенные коды форматирования. Здесь, `[DBNum2][$-804]General` применяет формат валюты.
4. **Сохранить как PDF**
   Отрегулируйте ширину столбца для видимости и сохраните книгу:
   ```csharp
   ws.Cells.SetColumnWidth(0, 30);
   wb.Save("outputDBNumCustomFormatting.pdf", SaveFormat.Pdf);
   ```

#### Советы по устранению неполадок
- Убедитесь, что используются правильные коды формата. `st.Custom`.
- Убедитесь, что Aspose.Cells правильно указан и лицензирован.

### Манипулирование рабочими тетрадями и листами (H2)

В этом разделе рассматриваются создание, доступ и изменение рабочих книг и листов программным способом.

#### Обзор

Программное управление рабочими книгами и листами обеспечивает гибкость для автоматизации задач Excel.

#### Этапы внедрения (H3)
1. **Инициализировать новую рабочую книгу**
   Начните с создания экземпляра `Workbook` сорт:
   ```csharp
   Workbook wb = new Workbook();
   ```
2. **Доступ к рабочим книгам и рабочим листам**
   Используйте индексацию листов для доступа к определенным листам:
   ```csharp
   Worksheet ws = wb.Worksheets[0];
   ```
3. **Изменить ячейки**
   При необходимости задайте значения в ячейках:
   ```csharp
   Cell cell = ws.Cells["A1"];
   cell.PutValue(123);
   ```
4. **Сохранить изменения**
   Сохраните изменения, сохранив книгу:
   ```csharp
   wb.Save("ModifiedWorkbook.pdf", SaveFormat.Pdf);
   ```
