### Установка размера шрифта в ячейке (H2)
#### Обзор:
Настройка внешнего вида текста путем установки определенных размеров шрифта в ячейке Excel может повысить ясность. Вот как этого добиться с помощью Aspose.Cells для .NET.
##### Шаг 1: Подготовьте среду
Начните с объявления исходного и выходного каталогов.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создать экземпляр объекта Workbook
Workbook workbook = new Workbook();
```
##### Шаг 2: Добавьте рабочий лист и получите доступ к ячейкам
Добавьте новый рабочий лист в свою книгу и получите доступ к нужной ячейке.
```csharp
int i = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[i];
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
cell.PutValue("Hello Aspose!");
```
##### Шаг 3: Установите размер шрифта
Получите стиль ячейки, измените размер шрифта и примените его обратно.
```csharp
Style style = cell.GetStyle();
style.Font.Size = 14; // Установите желаемый размер шрифта здесь
cell.SetStyle(style);
```
##### Шаг 4: Сохраните свою рабочую книгу
Наконец, сохраните свою рабочую книгу, чтобы увидеть изменения.
```csharp
workbook.Save(outputDir + "SetFontSizeExample.out.xls", SaveFormat.Excel97To2003);
```
