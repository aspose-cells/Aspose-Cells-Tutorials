### Функция автоматического подгонки строки

Эта функция позволяет вам автоматически регулировать высоту определенной строки в зависимости от ее содержимого. Вот как:

#### Шаг 1: Загрузите файл Excel

Откройте существующий файл Excel с помощью FileStream, который обеспечивает эффективные способы чтения и записи файлов в .NET.
```csharp
using System.IO;
using Aspose.Cells;

// Определите путь к исходному каталогу.
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// Создайте файловый поток для файла Excel.
FileStream fstream = new FileStream(SourceDir + "/Book1.xlsx", FileMode.Open);

// Откройте книгу с помощью потока файлов.
Workbook workbook = new Workbook(fstream);
```

#### Шаг 2: Доступ к строке и ее автоматическая подгонка

Получите доступ к определенному рабочему листу и используйте `AutoFitRow` метод регулировки высоты строки.
```csharp
// Откройте первый рабочий лист в рабочей книге.
Worksheet worksheet = workbook.Worksheets[0];

// Автоматически подгонять третью строку (индекс начинается с 0).
worksheet.AutoFitRow(1); // Регулирует высоту в зависимости от содержимого
```

#### Шаг 3: Сохранить и закрыть

После внесения изменений сохраните их в новом файле и убедитесь, что ресурсы освобождены должным образом, закрыв FileStream.
```csharp
// Определите путь к выходному каталогу.
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Сохраните книгу с измененной высотой строк.
workbook.Save(outputDir + "/output.xlsx");

// Всегда закрывайте поток, чтобы освободить все ресурсы.
fstream.Close();
```
