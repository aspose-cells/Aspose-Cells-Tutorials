## Руководство по внедрению

### Настройка десятичных и групповых разделителей

**Обзор:**
Настройка десятичных и групповых разделителей улучшает читаемость данных и соответствует определенным стандартам форматирования, требуемым различными регионами или предприятиями.

#### Шаг 1: Настройка параметров
Начните с указания желаемых числовых форматов для всей книги:
```csharp
// Определите пользовательские десятичные и групповые разделители
workbook.Settings.NumberDecimalSeparator = '.';
workbook.Settings.NumberGroupSeparator = ' ';
```
**Объяснение:** The `NumberDecimalSeparator` установлен на точку (.), как это обычно используется во многих регионах. `NumberGroupSeparator` настраивается как пробел (' '), который можно адаптировать в зависимости от региональных предпочтений.

#### Шаг 2: Применение пользовательских стилей
После определения разделителей примените к ячейкам пользовательский стиль:
```csharp
Worksheet worksheet = workbook.Worksheets[0];

// Установить значение ячейки и применить стиль
Cell cell = worksheet.Cells["A1"];
cell.PutValue(123456.789);

Style style = cell.GetStyle();
style.Custom = "#,##0.000;[Red]#,##0.000"; // Пользовательская строка формата
cell.SetStyle(style);
```
**Объяснение:** Пользовательский формат `#,##0.000` обеспечивает три десятичных знака и группирует цифры, используя определенные разделители.

#### Шаг 3: Автоматическая подгонка столбцов
Чтобы обеспечить хорошее представление данных, используйте функцию автоматического подбора столбцов:
```csharp
worksheet.AutoFitColumns();
```
Этот метод автоматически регулирует ширину столбцов в соответствии с их содержимым.

#### Шаг 4: Сохранить как PDF
Наконец, сохраните рабочую книгу в формате PDF с вашими пользовательскими настройками:
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY/CustomSeparator_out.pdf");
```
