### Функция: Создание и сохранение рабочей книги
**Обзор**
Эта функция демонстрирует, как создать новую книгу Excel, применить стили, автоматически подогнать строки/столбцы и сохранить в различных форматах.

#### Шаг 1: Создайте новую рабочую книгу

```csharp
using System;
using Aspose.Cells;

public class FeatureWorkbookCreation
{
    public static void Run()
    {
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string output1Path = SourceDir + "Output.xlsx";
        string output2Path = SourceDir + "Output.out.ods";

        // Создать новый экземпляр рабочей книги
        Workbook workbook = new Workbook();
```

#### Шаг 2: Доступ к первому рабочему листу и его оформление

```csharp
        // Доступ к первому рабочему листу в рабочей книге
        Worksheet worksheet = workbook.Worksheets[0];

        // Применить встроенный стиль «Заголовок» к ячейке A1
        Style style = workbook.CreateBuiltinStyle(BuiltinStyleType.Title);
        Cell cell = worksheet.Cells["A1"];
        cell.PutValue("Aspose");
        cell.SetStyle(style);

        // Автоматически подогнать первый столбец и строку
        worksheet.AutoFitColumn(0);
        worksheet.AutoFitRow(0);
```

#### Шаг 3: Сохранение в нескольких форматах

```csharp
        // Сохранить как формат Excel (.xlsx)
        workbook.Save(output1Path);

        // Сохранить как формат электронной таблицы OpenDocument (.ods)
        workbook.Save(output2Path);
    }
}
```

### Функция: стилизация ячеек с помощью встроенных стилей
**Обзор**
Узнайте, как применять встроенные стили, повышая визуальную привлекательность ваших ячеек.

#### Шаг 1: Создание и применение стиля

```csharp
using Aspose.Cells;

public class FeatureCellStyling
{
    public static void Run()
    {
        string SourceDir = "YOUR_SOURCE_DIRECTORY";

        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.Worksheets[0];

        // Создайте встроенный стиль «Заголовок» и примените его к ячейке A1.
        Style style = workbook.CreateBuiltinStyle(BuiltinStyleType.Title);
        Cell cell = worksheet.Cells["A1"];
        cell.PutValue("Aspose");
        cell.SetStyle(style);
    }
}
```
