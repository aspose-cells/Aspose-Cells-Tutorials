## Руководство по внедрению

### Создание и форматирование ячейки Excel

#### Обзор

Узнайте, как создать ячейку в книге Excel, вставить значения, применить числовые форматы для лучшей читаемости и извлечь как отформатированные, так и неотформатированные данные ячеек.

**Шаг 1: Создание рабочей книги и рабочего листа Access**

Создать новый `Workbook` объект и доступ к первому рабочему листу:

```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**Шаг 2: Вставьте значение в ячейку**

Откройте ячейку A1 и введите числовое значение:

```csharp
Cell cell = worksheet.Cells["A1"];
cell.PutValue(0.012345);
```

**Шаг 3: Примените форматирование чисел**

Отформатируйте ячейку так, чтобы отображалось только два десятичных знака, используя `Style`:

```csharp
Style style = cell.GetStyle();
style.Number = 2; // Формат «0.00»
cell.SetStyle(style);
```

**Шаг 4: Извлечение форматированных и неформатированных значений**

Получите обе версии значения ячейки для сравнения:

```csharp
string formattedValue = cell.GetStringValue(CellValueFormatStrategy.CellStyle);
string unformattedValue = cell.GetStringValue(CellValueFormatStrategy.None);
```

### Управление рабочими книгами и рабочими листами

#### Обзор

Изучите, как создавать, открывать и управлять рабочими листами в книге Excel.

**Шаг 1: Создайте новую рабочую книгу**

Инициализируйте `Workbook` объект, как показано ранее.

**Шаг 2: Доступ к рабочему листу по индексу**

Доступ к первому рабочему листу осуществляется с помощью его индекса:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
Console.WriteLine("Accessed Worksheet: " + worksheet.Name);
```

**Шаг 3: Манипулирование ячейками на рабочем листе**

Создайте и задайте значения для новых ячеек, например, поместите «Hello World» в ячейку A2:

```csharp
cell = worksheet.Cells["A2"];
cell.PutValue("Hello World");
```
