### Реализация обратного вызова предупреждения о замене шрифта
#### Обзор
Целью здесь является создание механизма, который оповещает вас всякий раз, когда библиотека заменяет шрифт во время преобразования, гарантируя, что ваш вывод соответствует ожиданиям.

#### Пошаговая реализация
**Создать класс обратного вызова**
Определить класс, реализующий `IWarningCallback` для обработки предупреждений во время таких операций, как преобразования:
```csharp
using Aspose.Cells;
using System.Diagnostics;

public class GetWarningsForFontSubstitution : IWarningCallback
{
    // Метод сбора и регистрации предупреждений о замене шрифтов.
    public void Warning(WarningInfo info)
    {
        if (info.WarningType == WarningType.FontSubstitution)
        {
            Debug.WriteLine("WARNING INFO: " + info.Description);
        }
    }
}
```

**Объяснение:** Этот класс прослушивает предупреждающие события во время преобразования. Если тип события — `FontSubstitution`, он регистрирует подробное сообщение, используя `Debug.WriteLine`.

### Преобразование рабочей книги в PDF с предупреждениями о замене шрифтов
#### Обзор
Теперь, когда наш обратный вызов предупреждения готов, давайте воспользуемся им для преобразования книги Excel в файл PDF, одновременно фиксируя предупреждения о замене шрифтов.

**Реализация преобразования**
Создайте статический класс и метод для обработки процесса преобразования:
```csharp
using Aspose.Cells;
using System.IO;

public static class ConvertWorkbookToPdfWithWarnings
{
    public static void Run()
    {
        // Определите исходный и выходной каталоги.
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string OutputDir = "YOUR_OUTPUT_DIRECTORY";

        // Загрузите книгу Excel из указанного каталога.
        Workbook workbook = new Workbook(SourceDir + "sampleGetWarningsForFontSubstitution.xlsx");

        // Создайте экземпляр PdfSaveOptions для настройки параметров сохранения.
        PdfSaveOptions options = new PdfSaveOptions();

        // Назначьте наш обратный вызов предупреждения для обработки предупреждений о замене шрифтов.
        options.WarningCallback = new GetWarningsForFontSubstitution();

        // Сохраните книгу как файл PDF, используя указанные параметры.
        workbook.Save(OutputDir + "outputGetWarningsForFontSubstitution.pdf", options);
    }
}
```

**Объяснение:** Этот код загружает файл Excel и настраивает `PdfSaveOptions` использовать наш пользовательский обратный вызов предупреждения. При вызове `workbook.Save`, любые предупреждения о замене шрифтов фиксируются функцией обратного вызова, что позволяет лучше контролировать качество вывода.
