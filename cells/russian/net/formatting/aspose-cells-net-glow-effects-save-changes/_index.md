- Загрузка книги Excel

- Сохранение измененной рабочей книги

## Руководство по внедрению

### Функция 1: Загрузка и доступ к файлу Excel

#### Обзор
Первый шаг — загрузка файла Excel. Этот пример демонстрирует открытие книги и доступ к ее первому листу.

**Шаг 1**: Инициализировать `Workbook` объект
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook wb = new Workbook(SourceDir + "sampleGlowEffectOfShape.xlsx");
```

**Шаг 2**: Доступ к первому рабочему листу
```csharp
Worksheet ws = wb.Worksheets[0];
// «ws» теперь ссылается на первый рабочий лист в рабочей книге.
```

### Функция 2: Доступ и изменение свойств фигуры

#### Обзор
Эта функция позволяет получить доступ к фигуре на листе Excel и изменить ее свойства, например применить эффект свечения.

**Шаг 1**: Получить первую форму
```csharp
using Aspose.Cells.Drawing;

Shape sh = ws.Shapes[0];
```

**Шаг 2**: Изменить свойства эффекта свечения
```csharp
GlowEffect ge = sh.Glow;
ge.Size = 30; // Установка размера эффекта свечения.
ge.Transparency = 0.4; // Регулировка уровня прозрачности.
// «sh» теперь имеет обновленные свойства свечения.
```

### Функция 3: Сохранение рабочей книги с изменениями

#### Обзор
После внесения изменений в файл Excel крайне важно сохранить эти изменения.

**Шаг 1**: Сохраните измененную рабочую книгу
```csharp
using Aspose.Cells;

wb.Save(outputDir + "outputGlowEffectOfShape.xlsx");
// Измененная рабочая книга сохраняется под новым именем в выходном каталоге.
```
