## Руководство по внедрению
Давайте разберем реализацию на управляемые части, сосредоточившись на применении встроенных числовых форматов к различным типам данных.

### Настройка вашей рабочей тетради

#### Обзор
Начните с создания нового файла Excel и получения ссылок на его рабочие листы. Этот шаг имеет решающее значение для эффективного управления стилями ячеек.

**Создание рабочей книги**
```csharp
// Создать новый экземпляр рабочей книги
Workbook workbook = new Workbook();

// Доступ к первому рабочему листу в рабочей книге
Worksheet worksheet = workbook.Worksheets[0];
```

### Форматирование дат

#### Обзор
Отображение дат в удобном для пользователя формате необходимо для ясности. Давайте применим формат "д-ммм-гг" к ячейке.

**Применение формата даты**
```csharp
// Вставьте текущую дату в ячейку A1.
worksheet.Cells["A1"].PutValue(DateTime.Now);

// Извлечь и изменить стиль ячейки
Style style = worksheet.Cells["A1"].GetStyle();
style.Number = 15; // Встроенный формат для "д-ммм-гг"
worksheet.Cells["A1"].SetStyle(style);
```

### Форматирование процентов

#### Обзор
Преобразование числовых значений в проценты может улучшить интерпретацию данных, особенно в финансовых отчетах.

**Применение процентного формата**
```csharp
// Вставьте числовое значение в ячейку A2.
worksheet.Cells["A2"].PutValue(20);

// Измените стиль отображения процентов
style = worksheet.Cells["A2"].GetStyle();
style.Number = 9; // Встроенный формат для процентов
worksheet.Cells["A2"].SetStyle(style);
```

### Форматирование валюты

#### Обзор
Финансовые данные часто требуют форматирования валюты для обеспечения согласованности в отчетах.

**Применение формата валюты**
```csharp
// Вставьте числовое значение в ячейку A3.
worksheet.Cells["A3"].PutValue(2546);

// Установить стиль отображения валюты
style = worksheet.Cells["A3"].GetStyle();
style.Number = 6; // Встроенный формат для валюты
worksheet.Cells["A3"].SetStyle(style);
```

### Сохранение вашей рабочей книги
Наконец, сохраните вашу рабочую книгу в файле Excel:
```csharp
// Сохраните книгу в формате Excel97To2003.
workbook.Save("path/to/your/book1.out.xls", SaveFormat.Excel97To2003);
```
