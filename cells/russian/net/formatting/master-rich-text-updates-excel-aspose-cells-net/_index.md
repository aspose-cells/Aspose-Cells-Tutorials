## Руководство по внедрению

Теперь давайте реализуем функцию обновления расширенного текста. Мы разобьем это руководство на логические разделы, чтобы вам было легче следовать.

### Загрузка и доступ к ячейкам форматированного текста

#### Обзор
Чтобы обновить ячейку с форматированным текстовым содержимым в файле Excel, сначала загрузите рабочую книгу и откройте конкретный рабочий лист и ячейку, где требуются обновления.
```csharp
// Определить исходные и выходные каталоги
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// Загрузите рабочую книгу, содержащую ваш файл Excel.
Workbook workbook = new Workbook(sourceDir + "sampleUpdateRichTextCells.xlsx");

// Доступ к первому рабочему листу
Worksheet worksheet = workbook.Worksheets[0];

// Получить ячейку A1, содержащую форматированный текст
Cell cell = worksheet.Cells["A1"];
```

#### Объяснение
- **Рабочая тетрадь:** Представляет собой целый файл Excel.
- **Рабочий лист:** Отдельный лист в вашей рабочей книге, доступ к которому осуществляется по индексу или имени.
- **Клетка:** Конкретная ячейка, в которую вы хотите внести обновления.

### Обновление настроек шрифта в ячейках форматированного текста

#### Обзор
Чтобы изменить настройки шрифта содержимого форматированного текста в ячейке, извлеките и измените `FontSetting` объекты.
```csharp
Console.WriteLine("Before updating the font settings....");

// Получить все символы в ячейке как массив FontSettings
FontSetting[] fnts = cell.GetCharacters();

// Пройдитесь по каждому FontSetting, чтобы вывести текущее имя шрифта
for (int i = 0; i < fnts.Length; i++)
{
    Console.WriteLine(fnts[i].Font.Name);
}

// Обновить имя шрифта первого FontSetting
fnts[0].Font.Name = "Arial";

// Применить изменения обратно к ячейке
cell.SetCharacters(fnts);

Console.WriteLine();

Console.WriteLine("After updating the font settings....");

// Получить обновленные FontSettings
fnts = cell.GetCharacters();

// Распечатайте новые названия шрифтов.
for (int i = 0; i < fnts.Length; i++)
{
    Console.WriteLine(fnts[i].Font.Name);
}
```

#### Объяснение
- **ПолучитьСимволы():** Извлекает массив `FontSetting` объекты, представляющие части форматированного текста внутри ячейки.
- **SetCharacters(НастройкаШрифта[]):** Применяет измененные настройки шрифта обратно к ячейке.
- **Совет по устранению неполадок:** Убедитесь, что вы применили изменения, используя `SetCharacters()`; в противном случае изменения не сохранятся.
