### Установка значащих цифр в файле Excel

Эта функция гарантирует, что ваши числовые данные будут храниться с указанным количеством значащих цифр, имитируя поведение MS-Excel.

#### Пошаговая реализация:

1. **Инициализировать CellsHelper**: Установите глобальную точность для числовых значений.
   ```csharp
   using Aspose.Cells;

   // Установить количество значащих цифр глобально
   CellsHelper.SignificantDigits = 15;
   ```

2. **Создать экземпляр рабочей книги**:
   ```csharp
   // Создать экземпляр класса Workbook
   Workbook workbook = new Workbook();
   ```

3. **Доступ и управление рабочим листом**:
   Откройте первый рабочий лист и измените ячейку A1, чтобы продемонстрировать значимые цифры.
   ```csharp
   // Доступ к первому рабочему листу
   Worksheet worksheet = workbook.Worksheets[0];

   // Откройте ячейку A1 и установите двойное значение с 15 значащими цифрами.
   Cell cell = worksheet.Cells["A1"];
   cell.PutValue(1234567890.123451711);
   ```

4. **Сохранить рабочую книгу**:
   Определите выходной каталог и сохраните книгу.
   ```csharp
   string temp_outputDir = "YOUR_OUTPUT_DIRECTORY";
   workbook.Save(temp_outputDir + "/outputNumberOfSignificantDigits.xlsx");
   ```

### Создать и получить доступ к рабочему листу

Создание новой рабочей книги или доступ к существующему рабочему листу осуществляется с помощью Aspose.Cells.

#### Пошаговая реализация:

1. **Создать новую рабочую книгу**:
   ```csharp
   // Создать новый объект Workbook
   Workbook workbook = new Workbook();
   ```

2. **Доступ к первому рабочему листу**:


3. **Сохраните изменения (необязательно)**:
   При необходимости сохраните изменения.
   ```csharp

   workbook.Save(temp_outputDir + "/exampleWorkbook.xlsx");
   ```
