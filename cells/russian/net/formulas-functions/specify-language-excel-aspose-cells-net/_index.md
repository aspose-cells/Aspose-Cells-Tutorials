## Руководство по внедрению
Мы сосредоточимся на указании языка файла Excel с помощью встроенных свойств документа. Эта функция позволяет пользователям определять основные языки, используемые в их документах, для лучшей доступности и локализации.

### Шаг 1: Создание объекта рабочей книги
Начните с создания нового объекта рабочей книги, представляющего ваш файл Excel.

```csharp
// Инициализируйте библиотеку Aspose.Cells
Workbook wb = new Workbook();
```

Эта строка создает пустую книгу, в которую вы можете добавлять данные, листы или свойства по мере необходимости.

### Шаг 2: Доступ к встроенным свойствам документа
Чтобы изменить языковые настройки, откройте встроенную коллекцию свойств документа вашей рабочей книги:

```csharp
// Доступ к встроенным свойствам документа
Aspose.Cells.Properties.BuiltInDocumentPropertyCollection bdpc = wb.BuiltInDocumentProperties;
```

Здесь, `bdpc` представляет собой коллекцию, содержащую различные свойства документа, такие как имя автора, название и язык.

### Шаг 3: Установите язык
Укажите языки, используемые в вашем файле Excel. Это помогает пользователям с экранными дикторами или инструментами перевода лучше понимать содержимое:

```csharp
// Установка языка на немецкий и французский
bdpc.Language = "German, French";
```

На этом этапе мы устанавливаем немецкий и французский в качестве основных языков для нашего документа.

### Шаг 4: Сохраните свою рабочую книгу
Наконец, сохраните вашу книгу с этими свойствами. Это гарантирует сохранение всех настроек:

```csharp
// Сохраните книгу по указанному пути.
wb.Save(outputDir + "outputSpecifyLanguageOfExcelFileUsingBuiltInDocumentProperties.xlsx", SaveFormat.Xlsx);
```

Этот шаг записывает изменения в `.xlsx` файл, готовый к использованию или распространению.
