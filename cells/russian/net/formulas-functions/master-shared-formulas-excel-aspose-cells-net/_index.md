## Руководство по внедрению (H2)

Выполните следующие действия, чтобы создать и применить общие формулы с помощью Aspose.Cells для .NET.

### Шаг 1: Инициализация рабочей книги (H3)

Создайте экземпляр рабочей книги, представляющий ваш файл Excel:

```csharp
// Создать новый экземпляр рабочей книги
Workbook workbook = new Workbook();
```

### Шаг 2: Доступ и изменение ячеек рабочего листа (H3)

Откройте первый рабочий лист для работы с его ячейками:

```csharp
// Доступ к первому рабочему листу в рабочей книге
Worksheet worksheet = workbook.Worksheets[0];

// Вставьте образец данных в столбец A.
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["A4"].PutValue(30);
worksheet.Cells["A5"].PutValue(65);
worksheet.Cells["A6"].PutValue(49);
```

### Шаг 3: Примените общую формулу (H3)

Использовать `SetSharedFormula` чтобы применить формулу к нескольким ячейкам:

```csharp
// Получите коллекцию клеток из рабочего листа.
Cells cells = worksheet.Cells;

// Установить общую формулу для диапазона C1:C6
cells["C1"].SetSharedFormula("=A1*2", 6, 1);
```
**Объяснение:** The `SetSharedFormula` Метод принимает три параметра:
- Формула в виде строки (например, "=A1*2")
- Количество строк для применения формулы
- Количество столбцов для применения формулы

### Шаг 4: Сохраните свою рабочую книгу (H3)

Сохраните вашу рабочую книгу с изменениями:

```csharp
// Определить выходной каталог
string outputDir = RunExamples.Get_OutputDirectory();

// Сохраните измененный файл Excel.
workbook.Save(outputDir + "outputSettingSharedFormula.xlsx", SaveFormat.Xlsx);

Console.WriteLine("SettingSharedFormula executed successfully.");
```
