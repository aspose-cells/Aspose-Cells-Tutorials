## Руководство по внедрению

### Загрузка рабочей книги

Загрузите вашу рабочую книгу, чтобы определить файл Excel:
```csharp
// Загрузить существующую книгу из указанного пути
Workbook workbook = new Workbook("Book1.xlsx");
```
#### Обзор
Это инициализирует `Workbook` объект, обеспечивающий доступ к рабочим листам и ячейкам.

### Доступ к ячейкам и отслеживание зависимостей
Выберите рабочий лист и ячейку для отслеживания зависимости:
```csharp
// Получить первый рабочий лист в рабочей тетради
Worksheet worksheet = workbook.Worksheets[0];

// Доступ к определенной ячейке
Cell targetCell = worksheet.Cells["B2"];
```
#### Обзор
Доступ к `Cells` сбор указанного рабочего листа для точного определения целевой ячейки.

### Получение иждивенцев
Используйте `GetDependents` метод извлечения зависимых ячеек:
```csharp
// Получить все зависимые ячейки для «B2»
Cell[] dependents = targetCell.GetDependents(true);

foreach (Cell c in dependents)
{
    Console.WriteLine(c.Name); // Выводит имена зависимых ячеек
}
```
#### Обзор
`GetDependents(true)` возвращается `Cell` объекты, затронутые изменениями в указанной ячейке.
