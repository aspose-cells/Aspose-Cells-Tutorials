## Руководство по внедрению
### Функция: загрузка и изменение файла Excel
Эта функция демонстрирует загрузку существующей книги Excel, вставку столбцов в каждый лист и сохранение обновленной версии.

#### Обзор
Мы рассмотрим, как загружать рабочую книгу, перебирать ее листы, вставлять новые столбцы, задавать значения заголовков и эффективно сохранять изменения с помощью Aspose.Cells для .NET.

#### Шаг 1: Загрузите рабочую книгу
Начните с создания экземпляра `Workbook` с путем к исходному файлу Excel:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string inputFile = SourceDir + "/Sample.xls";

// Создайте объект Workbook для загрузки файла Excel.
Workbook workbook = new Workbook(inputFile);
```

#### Шаг 2: Вставьте столбцы и задайте заголовки
Пройдитесь по каждому рабочему листу и вставьте столбцы:
```csharp
for (int i = 0; i < workbook.Worksheets.Count; i++)
{
    Worksheet ws = workbook.Worksheets[i];
    Cells cells = ws.Cells;

    // Вставьте десять новых столбцов в начало каждого рабочего листа.
    for (int c = 0; c < 10; c++)
    {
        cells.InsertColumn(c); // Вставить новый столбец
        cells[0, c].PutValue("Column" + c.ToString()); // Установить имена заголовков
    }
}
```
**Почему именно такой подход?**
Вставка столбцов перед установкой значений гарантирует, что все заголовки будут правильно выровнены и их будет легко идентифицировать.

#### Шаг 3: Сохраните измененную рабочую книгу.
После завершения изменений сохраните рабочую книгу в новом файле:
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
string outputFile = outputDir + "/output_out.xls";

// Сохраните измененный файл Excel.
workbook.Save(outputFile);
```
