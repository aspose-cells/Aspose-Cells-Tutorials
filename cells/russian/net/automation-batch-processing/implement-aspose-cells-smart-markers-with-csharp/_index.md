## Руководство по внедрению
Это руководство проведет вас через создание каталога и настройку `WorkbookDesigner` использовать умные маркеры.

### Настройка каталога
#### Обзор:
Программное создание каталогов необходимо для динамического хранения файлов, обеспечения их организованности и легкого доступа.
##### Шаг 1: Проверьте, существует ли каталог
```csharp
string dataDir = "YOUR_SOURCE_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
```
##### Шаг 2: Создайте каталог, если его не существует
```csharp
if (!isExists)
{
    Directory.CreateDirectory(dataDir);
}
```
**Объяснение**: Этот фрагмент кода проверяет, существует ли указанный вами каталог, и создает его, если его нет, обеспечивая плавный процесс настройки.

### Создание и настройка WorkbookDesigner
#### Обзор:
The `WorkbookDesigner` класс играет ключевую роль в обработке шаблонов Excel с помощью интеллектуальных маркеров, позволяя вам легко создавать динамические отчеты.
##### Шаг 1: Определите DesignerFile и набор данных
```csharp
public static Stream DesignerFile { get; set; }
public static System.Data.SqlClient.SqlConnection Dataset { get; set; }
```
**Объяснение**: Эти свойства являются заполнителями для вашего файла шаблона и подключения к базе данных соответственно.
##### Шаг 2: Реализация метода запуска
```csharp
public static void Run()
{
    if (DesignerFile != null && Dataset != null)
    {
        WorkbookDesigner designer = new WorkbookDesigner();
        designer.Workbook = new Workbook(DesignerFile);
        designer.SetDataSource(Dataset);
        designer.Process();
    }
}
```
**Объяснение**: этот метод обеспечивает доступность как шаблона, так и источника данных, а затем обрабатывает интеллектуальные маркеры для создания окончательного документа.
