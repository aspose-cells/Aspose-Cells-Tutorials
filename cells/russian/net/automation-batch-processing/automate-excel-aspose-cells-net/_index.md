## Руководство по внедрению

Давайте разберем реализацию на ключевые разделы.

### Функция: Манипулирование рабочими книгами и рабочими листами

В этом разделе демонстрируется создание рабочей книги, доступ к рабочим листам и настройка значений ячеек.

#### Шаг 1: Создайте новую рабочую книгу и получите доступ к рабочим листам

```csharp
// Инициализируйте WorkbookDesigner
WorkbookDesigner designer = new WorkbookDesigner();
Cells cells = designer.Workbook.Worksheets[0].Cells;

// Установить начальные заголовки в A1 и B1
cells["A1"].PutValue("Name");
cells["B1"].PutValue("Age");
```

Этот фрагмент создает рабочую книгу с заголовками «Имя» и «Возраст».

#### Шаг 2: Использование анонимных пользовательских объектов с WorkbookDesigner

Здесь мы будем использовать пользовательские объекты в качестве источников данных в нашей рабочей книге.

##### Определить маркеры

```csharp
// Определите маркеры в ячейках для использования пользовательских объектов.
cells["A2"].PutValue("&=Person.Name");
cells["B2"].PutValue("&=Person.Age");
```

Маркеры, такие как `&=Person.Name` выступают в качестве заполнителей для динамических данных из пользовательских объектов.

##### Создать и добавить источник данных

```csharp
// Создайте ArrayList объектов Person
ArrayList list = new ArrayList();
list.Add(new Person("Simon", 30));
list.Add(new Person("Johnson", 33));
// Дополнительные лица...
designer.SetDataSource("Person", list); // Привязать источник данных к дизайнеру
```

### Обработать и сохранить рабочую книгу

```csharp
// Замените маркеры реальными данными
designer.Process();

// Сохранить в выходной файл
string outputPath = @"YOUR_OUTPUT_DIRECTORY/outputAddingAnonymousCustomObject.xlsx";

```
