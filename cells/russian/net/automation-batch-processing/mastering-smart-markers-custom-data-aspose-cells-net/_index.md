## Руководство по внедрению

### Определить пользовательский класс для источника данных

**Обзор:**
Создайте пользовательский класс с именем `Person` со свойствами имени и возраста, служащими источником данных для интеллектуальных маркеров.

#### Шаг 1: Создание класса Person
```csharp
using System;

public class Person
{
    private string m_Name;
    
    public string Name
    {
        get { return m_Name; }
        set { m_Name = value; }
    }
    
    private int m_Age;
    
    public int Age
    {
        get { return m_Age; }
        set { m_Age = value; }
    }
    
    internal Person(string name, int age)
    {
        this.m_Name = name;
        this.m_Age = age;
    }
}
```

**Объяснение:** Этот класс определяет `Name` и `Age` как приватные поля с публичными свойствами для доступа. Конструктор инициализирует эти свойства.

### Использование интеллектуальных маркеров с пользовательским источником данных

**Обзор:**
Изучите использование интеллектуальных маркеров с Aspose.Cells, интегрируя наши пользовательские `Person` источник данных в шаблон Excel.

#### Шаг 2: Настройте рабочую тетрадь и назначьте смарт-маркеры
```csharp
using System.IO;
using Aspose.Cells;
using System.Collections.Generic;

public class UseSmartMarkersWithCustomData
{
    public static void Run()
    {
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string outputDir = "YOUR_OUTPUT_DIRECTORY";

        WorkbookDesigner report = new WorkbookDesigner();
        Worksheet sheet = report.Workbook.Worksheets[0];

        // Определить заголовки для смарт-маркеров
        sheet.Cells["A1"].PutValue("Name");
        sheet.Cells["B1"].PutValue("Age");

        // Настройте значения интеллектуальных маркеров
        sheet.Cells["A2"].PutValue("&=MyProduct.Name");
        sheet.Cells["B2"].PutValue("&=MyProduct.Age");

        IList<Person> peopleList = new List<Person>
        {
            new Person("Simon", 30),
            new Person("Johnson", 33)
        };

        report.SetDataSource("MyProduct", peopleList);
        report.Process(false);

        string outputPath = Path.Combine(outputDir, "SmartMarkerCustomObjects.xls");
        report.Workbook.Save(outputPath);
    }
}
```

**Объяснение:** Этот код настраивает конструктор рабочих книг и использует интеллектуальные маркеры (`&=MyProduct.Name` и `&=MyProduct.Age`) для отображения данных из `Person` класс. `SetDataSource` метод связывает наш пользовательский список с «MyProduct» для удобства использования.

### Советы по устранению неполадок
- **Распространенная проблема:** Убедитесь, что пути к каталогам указаны правильно; в противном случае операции сохранения могут завершиться неудачей.
- **Отладка интеллектуальных маркеров:** Используйте ведение журнала для проверки обработки маркеров, если значения не заполняются ожидаемым образом.
