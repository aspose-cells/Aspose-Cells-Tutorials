# Печать комментариев Excel в формате PDF с помощью Aspose.Cells для .NET: подробное руководство

## Руководство по внедрению
Давайте разберем шаги по печати комментариев при сохранении файла Excel в формате PDF.

### Шаг 1: Загрузите свою рабочую книгу
Создайте и загрузите книгу Excel. Убедитесь, что у вас есть путь к исходному файлу Excel.

```csharp
// Исходный каталог
string sourceDir = RunExamples.Get_SourceDirectory();

// Создать рабочую книгу из исходного файла Excel
Workbook workbook = new Workbook(sourceDir + "samplePrintCommentWhileSavingToPdf.xlsx");
```

### Шаг 2: Доступ к рабочему листу и настройка комментариев
Получите доступ к рабочему листу, с которым вы хотите работать. Здесь мы сосредоточимся на печати комментариев в конце каждого листа.

```csharp
// Доступ к первому рабочему листу
Worksheet worksheet = workbook.Worksheets[0];

// Установите PrintCommentsType на PrintSheetEnd для включения комментариев в PDF
worksheet.PageSetup.PrintComments = PrintCommentsType.PrintSheetEnd;
```

### Шаг 3: Сохранить как PDF
Сохраните вашу рабочую книгу в формате PDF с помощью Aspose.Cells `Save` метод.

```csharp
// Выходной каталог
string outputDir = RunExamples.Get_OutputDirectory();

// Сохранить книгу в формате PDF
workbook.Save(outputDir + "outputPrintCommentWhileSavingToPdf.pdf");

Console.WriteLine("PrintCommentWhileSavingToPdf executed successfully.");
```

### Советы по устранению неполадок
- **Отсутствующие комментарии**: Гарантировать `PrintCommentsType` установлен правильно.
- **Проблемы с путями к файлам**: Еще раз проверьте пути к исходному и выходному каталогам.
