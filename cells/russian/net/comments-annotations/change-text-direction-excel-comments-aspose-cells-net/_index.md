# Изменение направления текста в комментариях Excel с помощью Aspose.Cells .NET

Это руководство проведет вас через изменение направления текста комментариев с слева направо (LTR) на справа налево (RTL) и наоборот.

## Руководство по внедрению

### Создание рабочей книги и добавление комментариев

Начнем с создания новой книги Excel и добавления текста в ячейку.

**Обзор:**
В этом разделе показано, как создать экземпляр рабочей книги, добавить текст на рабочий лист и добавить комментарии.

```csharp
// Создать новую рабочую книгу
var wb = new Workbook();

// Получить первый рабочий лист
var sheet = wb.Worksheets[0];

// Добавьте текст в ячейку A1.
sheet.Cells["A1"].PutValue("Here");
```

### Добавление и настройка комментариев

Теперь давайте добавим комментарий в нашу ячейку и настроим выравнивание его текста.

**Добавление комментария:**
```csharp
// Добавить комментарий к ячейке A1
var comment = sheet.Comments[sheet.Comments.Add("A1"]);
```

**Настройка выравнивания и направления текста:**

- **Вертикальное выравнивание**: Выровнять текст по центру вертикально.
- **Горизонтальное выравнивание**: Выровнять текст по правому краю.
- **Направление текста**: Установите слева направо (LTR) на справа налево (RTL).

```csharp
// Установить вертикальное выравнивание
comment.CommentShape.TextVerticalAlignment = TextAlignmentType.Center;

// Установить горизонтальное выравнивание
comment.CommentShape.TextHorizontalAlignment = TextAlignmentType.Right;

// Изменить направление текста на справа налево
comment.CommentShape.TextDirection = TextDirectionType.RightToLeft;
```

**Совет по устранению неполадок:** Убедитесь, что ячейка, в которую вы добавляете комментарии, не заблокирована и не защищена, так как это может помешать внесению изменений.

### Сохранение вашей рабочей книги

Наконец, сохраните изменения, чтобы увидеть их отражение в файле Excel:

```csharp
// Сохраните файл Excel.
wb.Save("outputChangeTextDirection.xlsx");

Console.WriteLine("ChangeTextDirection executed successfully.\r\n");
```
