## Руководство по внедрению

В этом разделе описывается процесс получения и установки идентификаторов классов для встроенных объектов OLE.

### Получить идентификатор класса из встроенного объекта OLE

**Обзор**: эта функция позволяет вам получить уникальный идентификатор (GUID) определенного внедренного объекта в вашем файле Excel.

#### Шаг 1: Загрузите свою рабочую книгу
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook wb = new Workbook(sourceDir + "sampleGetSetClassIdentifierEmbedOleObject.xls");
```

#### Шаг 2: Доступ к рабочему листу и объекту OLE
```csharp
Worksheet ws = wb.Worksheets[0];
OleObject oleObj = ws.OleObjects[0];
```

#### Шаг 3: Преобразование в GUID и печать
```csharp
Guid guid = new Guid(oleObj.ClassIdentifier);
Console.WriteLine(guid.ToString().ToUpper());
```

### Установить новый идентификатор класса

**Обзор**: При необходимости измените идентификатор класса существующего объекта OLE.

#### Шаг 1: Определите новый GUID
```csharp
string newClassId = "Your-New-GUID-Here"; // Заменить фактической строкой GUID
Guid newGuid = new Guid(newClassId);
```

#### Шаг 2: Назначьте и сохраните изменения
```csharp
oleObj.ClassIdentifier = newGuid.ToByteArray();
wb.Save("updatedWorkbook.xls");
```
