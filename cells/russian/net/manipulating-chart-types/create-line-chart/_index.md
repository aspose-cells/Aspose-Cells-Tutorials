# Создать линейную диаграмму

## Введение

Вы готовы визуализировать свои данные с потрясающей ясностью? Линейные диаграммы — это фантастический способ отображения тенденций с течением времени или взаимосвязи между двумя переменными. Независимо от того, управляете ли вы данными для бизнес-проекта или анализируете личные показатели, возможность создания линейных диаграмм программным способом может сэкономить вам время и обеспечить большую гибкость. В этом руководстве мы проведем вас через каждый шаг создания линейной диаграммы с помощью Aspose.Cells для .NET. Готовы погрузиться? Давайте начнем!

## Предпосылки

Прежде чем мы перейдем к тонкостям создания линейного графика, давайте убедимся, что вы готовы к дальнейшему изучению:

1. Visual Studio: убедитесь, что на вашем компьютере установлена Visual Studio, так как это одна из самых популярных сред IDE для разработки .NET.
2.  Библиотека Aspose.Cells для .NET: Вам понадобится библиотека Aspose.Cells, которую можно загрузить с сайта[здесь](https://releases.aspose.com/cells/net/).
3. Базовые знания C#: знакомство с языком программирования C# поможет вам лучше понять примеры и фрагменты кода.
4. .NET Framework или .NET Core: базовая настройка любой из этих платформ, поскольку она станет основой для наших приложений.

Как только вы выполните все эти предварительные условия, вы будете готовы создать несколько диаграмм!

## Импортные пакеты

Теперь, когда мы настроили нашу среду, нам нужно импортировать необходимые пакеты в наш код C#. Так же, как вы собираете инструменты перед началом проекта, импорт пакетов необходим, чтобы у вас было все необходимое.

Вот как это сделать:

```csharp
using System;
using System.IO;

using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
using Aspose.Cells.Charts;
```

 Эта строка импортирует`Aspose.Cells` Пространство имен, содержащее все классы и методы, которые мы будем использовать для создания нашей линейной диаграммы.

Теперь давайте разобьем весь процесс на простые, усваиваемые шаги. Каждый шаг проведет вас через логический поток создания линейной диаграммы с помощью Aspose.Cells для .NET.

## Шаг 1: Настройте выходной каталог

Первый шаг — определить, где вы хотите сохранить выходной файл. Это похоже на настройку рабочего пространства, прежде чем вы начнете пачкать руки. 

```csharp
// Выходной каталог
string outputDir = "Your Output Directory";
```
 Заменять`"Your Output Directory"`на фактический путь, по которому вы хотите сохранить созданный файл Excel.

## Шаг 2: Создание экземпляра объекта Workbook

Далее нам нужно создать новый экземпляр рабочей книги. Думайте о рабочей книге как о холсте, на котором будет струиться ваше творчество. 

```csharp
// Создание объекта Workbook
Workbook workbook = new Workbook();
```
Эта строка инициализирует новую рабочую книгу, которая будет содержать все ваши данные и визуальные материалы.

## Шаг 3: Доступ к рабочему листу

В нашей новой книге нам нужно получить ссылку на лист, куда мы будем вводить наши данные. Если книга — это наш холст, то лист — это наша палитра.

```csharp
// Получение ссылки на недавно добавленный рабочий лист путем передачи его индекса листа
Worksheet worksheet = workbook.Worksheets[0];
```
 Здесь мы получаем доступ к первому рабочему листу (индекс`0`).

## Шаг 4: Добавьте выборочные значения в ячейки

А теперь самое интересное! Мы собираемся ввести несколько выборочных значений в наш рабочий лист. Эти данные послужат основой для нашей линейной диаграммы. 

```csharp
// Добавление выборочных значений в ячейки
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```
В этом фрагменте мы добавляем значения в ячейки столбцов A и B. Столбец A представляет значения оси X, а столбец B представляет значения оси Y.

## Шаг 5: Добавьте линейную диаграмму на рабочий лист.

Далее мы собираемся ввести нашу линейную диаграмму в рабочий лист. Вот где ваши данные действительно оживут!

```csharp
// Добавление диаграммы на рабочий лист
int chartIndex = worksheet.Charts.Add(Aspose.Cells.Charts.ChartType.Line, 5, 0, 25, 10);
```
Здесь мы добавляем линейную диаграмму в указанное место. Параметры (5, 0, 25, 10) определяют положение и размер диаграммы на рабочем листе.

## Шаг 6: Доступ к новому экземпляру диаграммы

После того, как мы добавили нашу диаграмму, пришло время заняться только что созданным объектом диаграммы. 

```csharp
// Доступ к экземпляру недавно добавленной диаграммы
Aspose.Cells.Charts.Chart chart = worksheet.Charts[chartIndex];
```
Этот код подключает нас к диаграмме, чтобы мы могли манипулировать ею дальше.

## Шаг 7: Добавьте SeriesCollection в диаграмму

Теперь нам нужно указать нашей диаграмме, какие данные отображать. Здесь мы определяем источник данных для нашей линейной диаграммы, добавляя SeriesCollection.

```csharp
// Добавление SeriesCollection (источник данных диаграммы) к диаграмме в диапазоне от ячейки «A1» до «B3»
chart.NSeries.Add("A1:B3", true);
```
В этом примере мы указываем диаграмме использовать значения в ячейках A1–B3.

## Шаг 8: Сохраните файл Excel.

Грандиозный финал! После всей вашей тяжелой работы пришло время сохранить файл Excel и увидеть вашу линейную диаграмму в действии.

```csharp
// Сохранение файла Excel
workbook.Save(outputDir + "outputHowToCreateLineChart.xlsx");
```
 Эта строка сохраняет вашу рабочую книгу в указанном выходном каталоге под именем`outputHowToCreateLineChart.xlsx`.

## Шаг 9: Выполнение и проверка

Наконец, теперь вы можете запустить свой код и убедиться, что линейная диаграмма была успешно создана в вашем выходном каталоге! 

```csharp
Console.WriteLine("HowToCreateLineChart executed successfully.");
```
Это выведет сообщение на вашу консоль, сообщающее, что все прошло гладко.
