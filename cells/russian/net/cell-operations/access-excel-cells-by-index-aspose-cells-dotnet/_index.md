### Доступ к ячейке с использованием индекса ее строки и столбца
#### Обзор
Доступ к определенным ячейкам, важным для таких задач, как извлечение или обработка данных. Эта функция позволяет программно определить любую ячейку рабочего листа.

#### Этапы внедрения
##### Шаг 1: Загрузите рабочую книгу
Откройте существующую рабочую книгу из исходного каталога:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleAccessCellUsingCellIndexInCellsCollection.xlsx");
```

##### Шаг 2: Доступ к рабочему листу
Доступ к любому листу по его индексу. Для этого примера используйте первый лист (индекс 0):

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

##### Шаг 3: Извлеките ячейку
Используйте `GetCell` метод доступа к ячейке с использованием индексов строк и столбцов:

```csharp
Cell cell = worksheet.Cells.GetCell(5, 2);
```

#### Параметры Пояснение
- **Индекс строки**: Индекс строки, отсчитываемый от нуля.
- **Индекс столбца**: Индекс столбца, отсчитываемый от нуля.
Этот метод возвращает `Cell` объект для извлечения или изменения его значения по мере необходимости. 

#### Советы по устранению неполадок
- Убедитесь, что пути к файлам верны и доступны.
- Убедитесь, что индексы находятся в пределах размеров рабочего листа, чтобы избежать исключений.
