## Руководство по внедрению
В этом разделе вы познакомитесь с реализацией решения для обнаружения максимального количества строк и столбцов в форматах XLS и XLSX с использованием C#.

### Обзор
Наша цель — создать программу, которая выводит максимальное количество строк и столбцов, поддерживаемых как Excel 97-2003 (XLS), так и современными файлами Excel (XLSX). Мы достигнем этого, используя Aspose.Cells `WorkbookSettings` характеристики.

#### Пошаговая реализация
**1. Создание и настройка рабочей книги для формата XLS**
```csharp
using System;
using Aspose.Cells;

namespace DiscoverMaxRowsColumns
{
    class Program
    {
        public static void Main()
        {
            // Инициализировать сообщение о формате XLS.
            Console.WriteLine("Maximum Rows and Columns supported by XLS format.");

            // Создайте рабочую книгу в формате XLS.
            Workbook wb = new Workbook(FileFormatType.Excel97To2003);

            // Определите максимальное количество строк и столбцов для XLS.
            int maxRowsXls = wb.Settings.MaxRow + 1;
            int maxColsXls = wb.Settings.MaxColumn + 1;

            // Выведите результаты.
            Console.WriteLine("Maximum Rows: " + maxRowsXls);
            Console.WriteLine("Maximum Columns: " + maxColsXls);
        }
    }
}
```
**Объяснение:**
- `FileFormatType.Excel97To2003`: Указывает, что мы работаем со старым форматом Excel, XLS.
- `wb.Settings.MaxRow` и `wb.Settings.MaxColumn`: Эти свойства обеспечивают максимально поддерживаемые значения индекса. Добавление 1 преобразует их в понятные человеку числа.

**2. Создание и настройка рабочей книги для формата XLSX**
```csharp
// Распечатать сообщение о формате XLSX.
Console.WriteLine("Maximum Rows and Columns supported by XLSX format.");

// Воссоздать рабочую книгу в формате XLSX.
wb = new Workbook(FileFormatType.Xlsx);

// Определите максимальное количество строк и столбцов для XLSX.
int maxRowsXlsx = wb.Settings.MaxRow + 1;
int maxColsXlsx = wb.Settings.MaxColumn + 1;

// Выведите результаты.
Console.WriteLine("Maximum Rows: " + maxRowsXlsx);
Console.WriteLine("Maximum Columns: " + maxColsXlsx);

Console.WriteLine("DiscoverMaxRowsColumns executed successfully.");
```
**Объяснение:**
- Переключение на `FileFormatType.Xlsx` позволяет нам изучить возможности современного Excel, который, как правило, поддерживает больше строк и столбцов, чем старый формат XLS.
