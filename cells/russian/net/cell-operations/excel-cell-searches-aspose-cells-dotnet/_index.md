## Руководство по внедрению

В этом разделе мы разобьем реализацию на выполнимые этапы.

### Создание экземпляра рабочей книги и доступ к ячейкам

Сначала давайте получим доступ к ячейкам на листе:

#### Обзор
Эта функция позволяет загружать файл Excel и взаимодействовать с данными его ячеек с помощью Aspose.Cells.

#### Фрагмент кода
```csharp
using Aspose.Cells;

string sourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(sourceDir + "sampleFindingCellsContainingStringValueOrNumber.xlsx");

// Доступ к ячейкам первого рабочего листа
Cells cells = workbook.Worksheets[0].Cells;
```

### Поиск ячеек по точному совпадению целого или двойного значения

#### Обзор
Эта функция демонстрирует, как найти ячейку, содержащую точное целое значение.

#### Фрагмент кода
```csharp
using Aspose.Cells;

// Определите параметры поиска для поиска точного соответствия
FindOptions optsExactMatch = new FindOptions();
optsExactMatch.LookInType = LookInType.Values;
optsExactMatch.LookAtType = LookAtType.EntireContent;

// Выполнить поиск ячейки с целочисленным значением (205)
Cell cell1 = cells.Find(205, null, optsExactMatch);

if (cell1 != null)
{
    Console.WriteLine($"Found at: {cell1.Name}");
}
else
{
    Console.WriteLine("Record not found.");
}
```

### Поиск ячеек по точному совпадению строкового значения

#### Обзор
Найдите ячейку с точным совпадением строки, используя методы, аналогичные целочисленному поиску.

#### Фрагмент кода
```csharp
using Aspose.Cells;

// Повторно используйте FindOptions для поиска точного соответствия, никаких изменений не требуется.
Cell cell2 = cells.Find("Items A", null, optsExactMatch);

if (cell2 != null)
{
    Console.WriteLine($"Found at: {cell2.Name}");
}
else
{
    Console.WriteLine("Record not found.");
}
```

### Поиск ячеек по частичному совпадению строкового значения

#### Обзор
Найдите ячейки, содержащие часть строки, используя `Contains` опция в FindOptions.

#### Фрагмент кода
```csharp
using Aspose.Cells;

// Измените FindOptions для поиска частичного совпадения (содержит)
FindOptions optsPartialMatch = new FindOptions();
optsPartialMatch.LookInType = LookInType.Values;
optsPartialMatch.LookAtType = LookAtType.Contains;

// Выполнить поиск ячейки со строковым значением, которое может содержаться в других строках («Данные»).
Cell cell3 = cells.Find("Data", null, optsPartialMatch);

if (cell3 != null)
{
    Console.WriteLine($"Found at: {cell3.Name}");
}
else
{
    Console.WriteLine("Record not found.");
}
```
