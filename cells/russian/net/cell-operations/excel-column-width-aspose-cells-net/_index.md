## Руководство по внедрению

### Установка ширины столбцов

#### Обзор

Вы можете стандартизировать ширину всех столбцов, используя простой метод. Это повышает единообразие на ваших листах, делая данные более доступными и профессионально выглядящими.

#### Пошаговое руководство:

##### 1. **Настройка вашей среды**

Убедитесь, что вы создали соответствующий каталог для обработки файлов:

```csharp
// ExStart:1
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

##### 2. **Загрузка файла Excel**

Откройте нужный вам файл Excel с помощью `FileStream`:

```csharp
using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
{
    Workbook workbook = new Workbook(fstream);
}
```

- **Параметры:** `dataDir + "book1.xls"` указывает путь к файлу.
- **Цель метода:** Открывает файл Excel для редактирования.

##### 3. **Доступ к рабочему листу и его изменение**

Выберите рабочий лист, который вы хотите изменить:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Cells.StandardWidth = 20.5;
```

- **Конфигурация ключа:** `StandardWidth` устанавливает для всех столбцов одинаковую ширину 20,5.

##### 4. **Сохранение и закрытие ресурсов**

Не забудьте сохранить изменения и закрыть поток файлов:

```csharp
workbook.Save(dataDir + "output.out.xls");
fstream.Close();
```

- **Совет по устранению неполадок:** Всегда проверяйте, закрыты ли потоки, чтобы предотвратить утечку ресурсов.
