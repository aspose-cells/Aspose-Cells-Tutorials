## Руководство по внедрению

### Настройка каталогов
#### Обзор
Создание исходных и выходных каталогов необходимо для эффективной организации файлов. В этом разделе показано, как настроить эти каталоги с помощью C#.

#### Пошаговая реализация
**1. Определите пути к каталогам**
```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY"; // Укажите здесь ваш исходный каталог
string OutputDir = "YOUR_OUTPUT_DIRECTORY"; // Укажите здесь ваш выходной каталог
```

**2. Проверка и создание каталогов**
```csharp
if (!Directory.Exists(SourceDir))
{
    Directory.CreateDirectory(SourceDir); // Создайте исходный каталог, если он не существует.
}

if (!Directory.Exists(OutputDir))
{
    Directory.CreateDirectory(OutputDir); // Создайте выходной каталог, если он не существует.
}
```

### Создание и управление рабочей книгой
#### Обзор
Узнайте, как создать книгу Excel, добавить рабочие листы и обрабатывать изображения с помощью Aspose.Cells.

**3. Создать экземпляр рабочей книги**
```csharp
using Aspose.Cells;

Workbook workbook = new Workbook();
```

**4. Добавить рабочий лист**
```csharp
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

**5. Вставьте изображение на рабочий лист.**
Убедитесь, что у вас есть файл изображения с именем `logo.jpg` в исходном каталоге.
```csharp
string imagePath = Path.Combine(SourceDir, "logo.jpg");
int pictureIndex = worksheet.Pictures.Add(5, 5, imagePath);
Aspose.Cells.Drawing.Picture picture = worksheet.Pictures[pictureIndex];

// Расположите изображение пропорционально
picture.UpperDeltaX = 200; // Горизонтальное смещение
picture.UpperDeltaY = 200; // Вертикальное смещение
```

**6. Сохраните рабочую книгу.**
```csharp
string outputPath = Path.Combine(OutputDir, "book1.out.xls");
workbook.Save(outputPath);
```
