#### Загрузка и доступ к рабочей книге Excel

Понимание того, как загружать рабочие книги и получать к ним доступ, имеет важное значение для манипулирования данными.

**Обзор:**  
В этом разделе демонстрируется загрузка книги Excel из определенного каталога и доступ к ее первому листу.

##### Шаг 1: Определите каталоги и инициализируйте рабочую книгу

```csharp
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Загрузите исходный файл Excel
Workbook workbook = new Workbook(sourceDir + "/Book1.xlsx");
```

- **`sourceDir`:** Каталог, в котором хранятся ваши файлы Excel.
- **`workbook`:** Представляет загруженную книгу Excel.

##### Шаг 2: Доступ к первому рабочему листу

```csharp
// Доступ к первому рабочему листу в рабочей книге
Worksheet worksheet = workbook.Worksheets[0];
```
На этом шаге извлекается первый рабочий лист из рабочей книги с помощью индекса. `[0]`.
