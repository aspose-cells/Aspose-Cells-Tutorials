### Создание и сохранение рабочей книги в формате PDF
**Обзор**
Эта функция демонстрирует, как создать экземпляр `Workbook` объект и сохраните его как PDF-документ с помощью Aspose.Cells для .NET.

#### Шаг 1: Инициализация рабочей книги

```csharp
// Импортировать необходимые пространства имен
using Aspose.Cells;

// Укажите путь к исходному каталогу
string SourceDir = "YOUR_SOURCE_DIRECTORY";
// Укажите путь к выходному каталогу
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте новый экземпляр класса Workbook.
Workbook workbook = new Workbook();
```

#### Шаг 2: Сохранить как PDF

```csharp
// Сохраните книгу в формате PDF в указанном месте.
workbook.Save(OutputDir + "/output.pdf", SaveFormat.Pdf);
```

**Объяснение**: 
- `SaveFormat.Pdf` указывает, что вы хотите сохранить файл в формате PDF. Убедитесь, что путь для доступного для записи каталога вашего приложения установлен правильно.

### Работа с HttpResponse для загрузки файлов
**Обзор**
В этом разделе показано, как использовать `HttpResponse` объект для запуска загрузки файла, в частности, для PDF-файлов, созданных с помощью Aspose.Cells.

#### Шаг 1: Подготовка объекта ответа

```csharp
// Импортировать необходимые пространства имен
using System.Web;
using Aspose.Cells;

// Предположим, что объект HttpResponse доступен в вашем контексте ASP.NET.
HttpResponse response = HttpContext.Current.Response;

// Создайте или используйте существующую рабочую книгу
Workbook workbook = new Workbook();
```

#### Шаг 2: Настройте размещение контента и сохраните его в ответе

```csharp
if (response != null)
{
    // Настройте HTTP-заголовок для загрузки файла
    response.AddHeader("Content-Disposition", "attachment; filename=\"output.pdf\"");

    // Сохраните рабочую книгу напрямую в HttpResponse OutputStream
    workbook.Save(response.OutputStream, new PdfSaveOptions());
    
    // Завершите процесс ответа
    response.End();
}
```

**Объяснение**: 
- `response.AddHeader` гарантирует, что браузеры будут обрабатывать вывод как загрузку файла.
- `PdfSaveOptions` предоставляет дополнительные настройки для сохранения PDF-файлов.
