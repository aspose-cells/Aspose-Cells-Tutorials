## Руководство по внедрению

### Конвертировать рабочую книгу в PDF/A

#### Обзор

В этом разделе мы покажем, как преобразовать книгу Excel в совместимый файл PDF. Мы сосредоточимся на установке уровня соответствия PDF/A-1b для долгосрочного архивирования.

#### Пошаговая реализация

**Шаг 1: Создание и заполнение рабочей книги**

Начните с создания экземпляра `Workbook` класс, представляющий ваш файл Excel:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // Замените на ваш исходный каталог
string outputDir = @"YOUR_OUTPUT_DIRECTORY"; // Замените на желаемый выходной каталог

// Создать новый объект рабочей книги
Workbook workbook = new Workbook();

// Вставка значения в ячейку A1 первого рабочего листа
workbook.Worksheets[0].Cells[0, 0].PutValue("Testing PDF/A");
```

**Шаг 2: Настройте параметры сохранения PDF-файла**

Далее настройте параметры сохранения, чтобы указать параметры соответствия:

```csharp
using Aspose.Cells.Rendering;

// Создайте экземпляр PdfSaveOptions
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

// Установите уровень соответствия для выходного PDF-файла
pdfSaveOptions.Compliance = PdfCompliance.PdfA1b;
```

**Шаг 3: Сохранить как PDF**

Наконец, сохраните свою рабочую книгу в файле PDF, используя следующие параметры:

```csharp
// Сохраните книгу в формате PDF с указанными параметрами.
workbook.Save(outputDir + "/output.pdf", pdfSaveOptions);
```

### Объяснение ключевых элементов

- **Рабочая тетрадь**: Представляет файл Excel. Вы можете манипулировать листами и ячейками внутри этого объекта.
- **PdfSaveOptions**: Позволяет устанавливать определенные параметры для сохранения файлов в формате PDF, такие как уровни соответствия.
- **Согласие**: Определяет стандарт PDF, которого следует придерживаться; здесь мы используем `PdfA1b` для архивного качества.
