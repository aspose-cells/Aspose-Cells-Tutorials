## Руководство по внедрению

### Функция: настройка параметров изображения для преобразования в HTML

Эта функция направлена на улучшение качества изображения при преобразовании электронных таблиц Excel в формат HTML.

#### Шаг 1: Определите пути к файлам

Сначала укажите пути к исходному и выходному каталогам:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Загрузите электронную таблицу

Загрузите файл электронной таблицы, который вы собираетесь преобразовать:

```csharp
Workbook book = new Workbook($"{SourceDir}/Book1.xlsx");
```

#### Шаг 3: Настройте параметры сохранения HTML

Создать экземпляр `HtmlSaveOptions` и настройте параметры изображения:

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Html);
// Для лучшего качества установите формат изображения на PNG.
saveOptions.ImageOptions.ImageType = Drawing.ImageType.Png;
// Включите AntiAlias для сглаживания изображений и текста
saveOptions.ImageOptions.SmoothingMode = SmoothingMode.AntiAlias;
saveOptions.ImageOptions.TextRenderingHint = TextRenderingHint.AntiAlias;
```

#### Шаг 4: Сохраните преобразованный HTML-код.

Наконец, сохраните вашу рабочую книгу как HTML-файл со следующими настройками:

```csharp
book.Save($"{OutputDir}/output.html", saveOptions);
```

### Советы по устранению неполадок

- **Проблемы с качеством изображения**: Гарантировать `SmoothingMode` установлен на `AntiAlias`.
- **Ошибки «Файл не найден»**: Еще раз проверьте пути к исходному и выходному каталогам.
