## Руководство по внедрению

### Функция: Создание и обработка рабочих тетрадей
В этом разделе показано, как создать книгу Excel, управлять ее листами, добавлять значения в ячейки, вставлять изображения со ссылками на ячейки и сохранять книгу.

#### Создание новой рабочей книги
Начните с создания нового `Workbook` объект. Это будет ваш холст для всех операций:

```csharp
// Создать новую рабочую книгу
Workbook workbook = new Workbook();
```

#### Доступ к рабочим листам и добавление значений
Чтобы начать ввод данных, перейдите к коллекции ячеек первого рабочего листа:

```csharp
// Получить коллекцию ячеек первого рабочего листа
Cells cells = workbook.Worksheets[0].Cells;

// Добавить строковые значения в определенные ячейки
cells["A1"].PutValue("A1");
cells["C10"].PutValue("C10");
```

#### Вставка изображения со ссылками на ячейки
Добавьте изображение на лист и ссылайтесь на него с помощью формул ячеек:

```csharp
// Добавьте пустую картинку в позицию D1
Picture pic = workbook.Worksheets[0].Shapes.AddPicture(0, 3, 10, 6, null);

// Укажите формулу для рисунка, ссылающегося на ячейки A1:C10
cells["D1"].Formula = "=OFFSET($A$1:$C$10, ROW()-ROW(A1), COLUMN()-COLUMN(A1))";
pic.Formula = "=OFFSET($A$1:$C$10, 0, 3)";

// Обновите выбранное значение фигуры, чтобы отразить изменения.
table.Links[2].LinkSource = "path_to_your_image.jpg";
workbook.Worksheets[0].Shapes.UpdateSelectedValue();
```

#### Сохранение рабочей книги
Сохраните вашу рабочую книгу в указанном месте:

```csharp
// Сохраните книгу в выходном каталоге.
workbook.Save(outputDir + "/output.out.xls");
```
