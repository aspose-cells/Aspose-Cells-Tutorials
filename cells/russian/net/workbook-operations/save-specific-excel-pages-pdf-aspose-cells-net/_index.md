## Руководство по внедрению
Давайте разобьем нашу задачу на логические шаги, чтобы реализовать сохранение определенных страниц документа Excel в формате PDF.

### Функция 1: Открытие файла Excel
#### Обзор
На этом этапе открывается существующий файл Excel с помощью Aspose.Cells, который служит основой для дальнейших операций, таких как преобразование.
##### Шаг 1: Загрузите файл Excel

```csharp
using System;
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
// Откройте файл Excel.
Workbook workbook = new Workbook(sourceDir + "/sampleLimitNumberOfPagesGenerated.xlsx");

Console.WriteLine("Excel file opened successfully.");
```

*Объяснение*: `Workbook` объект представляет собой загруженный документ Excel, необходимый для доступа к данным в нем и управления ими.

### Функция 2: Настройка параметров сохранения PDF-файла
#### Обзор
Чтобы сохранить определенные страницы из книги Excel в формате PDF, настройте `PdfSaveOptions`.
##### Шаг 1: Настройка параметров сохранения PDF

```csharp
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте экземпляр объекта PdfSaveOption
PdfSaveOptions options = new PdfSaveOptions();

// Укажите, какие страницы следует включить в PDF-файл
options.PageIndex = 3; // Начать с индекса страницы 3
options.PageCount = 4; // Включить в общей сложности 4 страницы, начиная с PageIndex

Console.WriteLine("PDF save options configured.");
```

*Объяснение*: `PageIndex` и `PageCount` ключевые параметры, определяющие, какая часть документа Excel будет преобразована в PDF.

### Функция 3: Сохранение файла Excel в формате PDF с указанием определенных страниц
#### Обзор
Используйте настроенные параметры PdfSaveOptions для сохранения определенных страниц файла Excel в формате PDF.
##### Шаг 1: Сохраните документ

```csharp
using Aspose.Cells;
using System.IO;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Откройте файл Excel для обработки.
Workbook workbook = new Workbook(sourceDir + "/sampleLimitNumberOfPagesGenerated.xlsx");

// Настройте параметры сохранения PDF-файла, чтобы указать, какие страницы следует сохранять.
PdfSaveOptions options = new PdfSaveOptions();
options.PageIndex = 3; // Начать с индекса страницы 3
options.PageCount = 4; // Включить в общей сложности 4 страницы, начиная с PageIndex

// Сохраните указанные страницы как PDF-файл в выходном каталоге.
workbook.Save(outputDir + "/outputLimitNumberOfPagesGenerated.pdf", options);

Console.WriteLine("Excel document saved as PDF with specific pages.");
```

*Объяснение*: `Save` метод берет целевой путь и `PdfSaveOptions` для создания желаемого PDF-файла.
