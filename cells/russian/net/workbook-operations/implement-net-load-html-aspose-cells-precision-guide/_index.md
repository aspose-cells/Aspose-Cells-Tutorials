### Загрузить HTML-файл с определенными параметрами

Эта функция позволяет вам сохранять точность данных при конвертации HTML-документа в книгу Excel. Вот как этого можно добиться:

#### Обзор
Установив `KeepPrecision` в `HtmlLoadOptions`Aspose.Cells гарантирует, что числа не округляются и не форматируются во время преобразования, сохраняя их исходное значение.

#### Пошаговая реализация

**1. Задайте параметры загрузки HTML:**
```csharp
// Инициализируйте HtmlLoadOptions и укажите формат HTML
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
```

**2. Загрузите исходный HTML-файл:**
Заменять `YOUR_SOURCE_DIRECTORY` с фактическим путем к вашему каталогу.
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(sourceDir + "sampleSelfClosingTags.html", loadOptions);
```
- **Параметры**Конструктор принимает путь к файлу и параметры загрузки, чтобы указать, как следует интерпретировать HTML.

**3. Сохраните рабочую книгу:**
Заменять `YOUR_OUTPUT_DIRECTORY` с желаемым выходным каталогом.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outsampleSelfClosingTags.xlsx");
```
- **Метод Цель**: `Save()` Метод записывает рабочую книгу в указанный файл, в данном случае в формате Excel.
