# Как создать закладки PDF в Excel с помощью Aspose.Cells для .NET

## Введение

Создание навигируемых PDF-документов из сложных электронных таблиц Excel имеет важное значение при обмене отчетами или финансовыми отчетами. С Aspose.Cells for .NET вы можете легко создавать закладки PDF, упрощая пользователям переход непосредственно к определенным разделам в документе. В этом руководстве будет показано, как создавать как одиночные, так и множественные вложенные записи закладок PDF в файле Excel с помощью Aspose.Cells for .NET.

**Что вы узнаете:**
- Как настроить Aspose.Cells для .NET
- Создание одной записи закладки PDF для рабочего листа
- Реализация нескольких вложенных закладок PDF для различных рабочих листов
- Практическое применение этих функций

Давайте рассмотрим, как можно улучшить возможности навигации по документам с помощью Aspose.Cells для .NET.

## Руководство по внедрению

### Создание отдельной записи закладки PDF для рабочего листа

Эта функция позволяет создать закладку, указывающую на определенное место на рабочем листе.

#### Шаг 1: Загрузите файл Excel

Начните с загрузки исходного файла Excel:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook wb = new Workbook(SourceDir + "sampleCreatePdfBookmarkEntryForChartSheet.xlsx");
```

#### Шаг 2: Получите доступ к нужному рабочему листу

Откройте рабочий лист, который вы хотите добавить в закладки:

```csharp
Worksheet sheet1 = wb.Worksheets[0];
```

#### Шаг 3: Создание и настройка записи закладки

Настройте запись закладки PDF с указанием места назначения и текстовой метки:

```csharp
PdfBookmarkEntry ent1 = new PdfBookmarkEntry();
ent1.Destination = sheet1.Cells["A1"];
ent1.Text = "Bookmark-I";
```

#### Шаг 4: Сохраните параметры и выведите PDF-файл

Создавать `PdfSaveOptions` и сохраните рабочую книгу в формате PDF с закладкой:

```csharp
PdfSaveOptions opts = new PdfSaveOptions();
opts.Bookmark = ent1;
wb.Save(@"YOUR_OUTPUT_DIRECTORY" + "outputCreateSingleSheetBookmark.pdf", opts);
```

### Создание нескольких вложенных записей закладок PDF для рабочих листов

Эта функция демонстрирует создание вложенных закладок, позволяющих более организованно осуществлять навигацию по нескольким листам.

#### Шаг 1: Загрузите файл Excel

Загрузите исходный файл, как и прежде:

```csharp
Workbook wb = new Workbook(SourceDir + "sampleCreatePdfBookmarkEntryForChartSheet.xlsx");
```

#### Шаг 2: Доступ к нескольким рабочим листам

Доступ ко всем необходимым рабочим листам:

```csharp
Worksheet sheet1 = wb.Worksheets[0];
Worksheet sheet2 = wb.Worksheets[1];
Worksheet sheet3 = wb.Worksheets[2];
Worksheet sheet4 = wb.Worksheets[3];
```

#### Шаг 3: Создайте записи закладок для каждого рабочего листа

Инициализируйте и настройте каждую запись закладки:

```csharp
PdfBookmarkEntry ent1 = new PdfBookmarkEntry { Destination = sheet1.Cells["A1"], Text = "Bookmark-I" };
PdfBookmarkEntry ent2 = new PdfBookmarkEntry { Destination = sheet2.Cells["A1"], Text = "Bookmark-II-Chart1" };
PdfBookmarkEntry ent3 = new PdfBookmarkEntry { Destination = sheet3.Cells["A1"], Text = "Bookmark-III" };
PdfBookmarkEntry ent4 = new PdfBookmarkEntry { Destination = sheet4.Cells["A1"], Text = "Bookmark-IV-Chart2" };
```

#### Шаг 4: Вложите записи закладок

Создайте вложенный список для организации этих записей:

```csharp
ArrayList lst = new ArrayList();
ent1.SubEntry = lst;
lst.Add(ent2);
lst.Add(ent3);
lst.Add(ent4);
```

#### Шаг 5: Сохранение с помощью вложенных закладок

Настройте и сохраните ваш PDF-файл:

```csharp
PdfSaveOptions opts = new PdfSaveOptions { Bookmark = ent1 };
wb.Save(@"YOUR_OUTPUT_DIRECTORY" + "outputCreateMultipleNestedBookmarks.pdf", opts);
```
