## Руководство по внедрению

В этом разделе вы познакомитесь с двумя основными функциями: управлением пользовательскими свойствами и обработкой рабочих книг без изменений.

### Функция 1: Инициализация рабочей книги и удаление пользовательских свойств

#### Обзор

В этой функции мы инициализируем объект Workbook из файла Excel, извлечем его пользовательские свойства, удалим определенное свойство («Издатель») и сохраним обновленную рабочую книгу.

#### Пошаговая реализация

##### Инициализировать рабочую книгу

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sample-document-properties.xlsx");
```
*Почему этот шаг?* Загрузка существующего файла Excel в `Workbook` объект необходим для программного доступа и манипулирования его содержимым.

##### Получить пользовательские свойства документа

```csharp
documentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```
*Цель:* Доступ к коллекции пользовательских свойств позволяет вам проверять или изменять их по мере необходимости. Эти свойства хранят метаданные о ваших файлах Excel, например, информацию об авторе или примечания к версии.

##### Удалить определенное свойство

```csharp
customProperties.Remove("Publisher");
```
*Объяснение:* Удаление ненужных или конфиденциальных свойств гарантирует сохранение только релевантных метаданных, что повышает безопасность и организацию данных.

##### Сохранить рабочую книгу

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/out_sample-document-properties.xlsx");
```
*Функциональность:* Этот шаг сохраняет ваши изменения обратно в новый файл Excel. Это важно для сохранения изменений, сделанных во время выполнения.

### Функция 2: Инициализация и сохранение рабочей книги без изменений

#### Обзор

Иногда вам нужно просто загрузить файл Excel в приложение, не изменяя его содержимое. Эта функция демонстрирует, как это сделать.

#### Этапы внедрения

##### Загрузить существующий файл

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sample-document-properties.xlsx");
```
*Почему?* Загрузка рабочей книги без изменений полезна, когда вам необходимо отобразить ее содержимое или сослаться на него в других частях вашего приложения.

##### Сохранить без изменений

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/saved-sample-document-properties.xlsx");
```
*Цель:* Эта операция гарантирует, что исходные данные останутся нетронутыми, при этом возможен последующий доступ или распространение без внесения изменений.
