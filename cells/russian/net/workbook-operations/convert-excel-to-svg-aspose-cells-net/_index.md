### Инициализация и настройка рабочей книги (H2)

Перед конвертацией листа в SVG необходимо правильно настроить рабочую книгу. Это включает создание листов и заполнение их данными.

#### 1. Создайте новую рабочую книгу
Начните с создания нового экземпляра `Workbook` объект:
```csharp
// Создать экземпляр рабочей книги
class Workbook()
```
Эта строка программно инициализирует пустой файл Excel.

#### 2. Добавьте образцы данных на рабочие листы
Добавьте текст в ячейки рабочего листа:
```csharp
// Поместите образец текста в первую ячейку первого рабочего листа.
workbook.Worksheets[0].Cells["A1"].Value = "DEMO TEXT ON SHEET1";

// Добавьте второй рабочий лист и задайте его содержимое.
workbook.Worksheets.Add(SheetType.Worksheet);
workbook.Worksheets[1].Cells["A1"].Value = "DEMO TEXT ON SHEET2";
```
Здесь мы добавляем демонстрационный текст, который поможет визуализировать данные в нашем SVG.

#### 3. Установить активный рабочий лист
Чтобы отобразить определенный рабочий лист в формате SVG:
```csharp
// Активировать второй лист
class Workbook.Worksheets.ActiveSheetIndex(1)
```
Этот шаг гарантирует, что в формат SVG будет преобразован только активный лист.
