## Руководство по внедрению

### Загрузка файла Excel
**Обзор**: Узнайте, как загрузить существующий файл Excel в `Workbook` объект. Это необходимо для обработки и манипулирования данными, хранящимися в файлах Excel.

#### Пошаговая реализация
1. **Настройте исходный каталог**
   Определите, где находится исходный файл Excel:
   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   ```
2. **Загрузите файл Excel**
   Инициализировать `Workbook` объект, загрузив существующий файл Excel:
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "sampleExpandTextFromRightToLeft.xlsx");
   ```
   - **Объяснение**: `Workbook` конструктор берет путь к файлу и загружает его в память, позволяя вам программно взаимодействовать с его содержимым.

### Сохранение книги как HTML
**Обзор**: В этом разделе показано сохранение книги Excel в формате HTML. Это особенно полезно для веб-приложений, которым требуется представление данных вне программного обеспечения Excel.

#### Пошаговая реализация
1. **Настройте выходной каталог**
   Определите, где сохранить выходной HTML-файл:
   ```csharp
   string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
   ```
2. **Сохраните рабочую книгу в формате HTML**
   Сохраните книгу, добавив к имени файла информацию о версии:
   ```csharp
   workbook.Save(OutputDir + "outputExpandTextFromRightToLeft_" + CellsHelper.GetVersion() + ".html", SaveFormat.Html);
   ```
   - **Объяснение**: `Save` Метод записывает содержимое рабочей книги в HTML-файл. `CellsHelper.GetVersion()` предоставляет сведения о версии для различения различных выходных данных.
