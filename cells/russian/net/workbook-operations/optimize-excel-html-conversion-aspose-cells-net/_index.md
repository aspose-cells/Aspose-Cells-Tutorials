## Руководство по внедрению

### Обзор
В этом разделе вы узнаете, как настроить и выполнить преобразование Excel в HTML с масштабируемыми столбцами с помощью Aspose.Cells для .NET.

#### Шаг 1: Загрузите свою рабочую книгу
Начните с загрузки исходной книги Excel, которую вы хотите преобразовать. Это включает в себя настройку входных и выходных каталогов:

```csharp
// Входной каталог
string sourceDir = RunExamples.Get_SourceDirectory();

// Выходной каталог
string outputDir = RunExamples.Get_OutputDirectory();
```

#### Шаг 2: Настройте параметры сохранения HTML
Создать экземпляр `HtmlSaveOptions` для управления тем, как файл Excel сохраняется как HTML. Это включает в себя включение масштабируемых столбцов и экспорт изображений как Base64.

```csharp
// Укажите параметры сохранения Html
HtmlSaveOptions options = new HtmlSaveOptions();

// Установите свойство для масштабируемой ширины
options.WidthScalable = true;

// Экспортируйте изображения в формате Base64 для встраивания их в HTML
options.ExportImagesAsBase64 = true;
```

#### Шаг 3: Выполнение преобразования
Наконец, сохраните книгу в HTML-файле с настроенными вами параметрами:

```csharp
// Загрузить исходный файл образца
Workbook wb = new Workbook(sourceDir + "sampleForScalableColumns.xlsx");

// Сохраните книгу в формате HTML.
wb.Save(outputDir + "outsampleForScalableColumns.html", options);
```
