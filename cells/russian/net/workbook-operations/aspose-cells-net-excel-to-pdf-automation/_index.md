## Руководство по внедрению

Давайте разберем каждую функцию шаг за шагом:

### Загрузить рабочую книгу

**Обзор:**
Загрузка книги Excel — первый шаг в ее программной обработке. Мы воспользуемся Aspose.Cells, чтобы открыть пример книги.

#### Шаг 1: Определите исходный каталог
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
```

#### Шаг 2: Инициализация объекта Workbook
```csharp
Workbook workbook = new Workbook(SourceDir + "/sampleMINIFSAndMAXIFS.xlsx");
```
*Объяснение:* Эта линия создает `Workbook` объект, указав путь к файлу Excel. Библиотека Aspose.Cells считывает и загружает данные рабочей книги для дальнейшей обработки.

### Формулы расчета

**Обзор:**
После загрузки выполните необходимые вычисления по формулам в рабочей книге. Это гарантирует обновление всех динамических значений перед экспортом.

#### Шаг 1: Загрузите рабочую книгу
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleMINIFSAndMAXIFS.xlsx");
```

#### Шаг 2: Формулы расчета
```csharp
workbook.CalculateFormula();
```
*Объяснение:* Этот метод пересчитывает все формулы в рабочей книге. Это важно для получения точных данных перед экспортом в PDF.

### Сохранить книгу как PDF

**Обзор:**
Наконец, сохраните рассчитанную рабочую книгу в виде файла PDF с определенными параметрами, например, по одной странице на листе.

#### Шаг 1: Определите выходной каталог
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Инициализация и расчет рабочей книги
```csharp
Workbook workbook = new Workbook("YOUR_SOURCE_DIRECTORY/sampleMINIFSAndMAXIFS.xlsx");
workbook.CalculateFormula();
```

#### Шаг 3: Настройте параметры сохранения PDF-файла
```csharp
PdfSaveOptions options = new PdfSaveOptions() {
    OnePagePerSheet = true
};
```
*Объяснение:* `OnePagePerSheet` гарантирует, что каждый лист Excel сохраняется как отдельная страница в PDF-файле, что может иметь решающее значение для удобства чтения и представления.

#### Шаг 4: Сохраните рабочую книгу в формате PDF
```csharp
workbook.Save(outputDir + "/outputMINIFSAndMAXIFS.pdf", options);
```
*Объяснение:* Эта команда экспортирует рабочую книгу в PDF-файл, расположенный в указанном вами выходном каталоге, применяя все настроенные параметры сохранения.

### Советы по устранению неполадок

- **Проблемы с путем к файлу:** Убедитесь, что пути проложены правильно и доступны.
- **Ошибки расчета формулы:** Перед расчетом проверьте правильность формул.
- **Проблемы с экспортом PDF:** Проверьте наличие неподдерживаемых функций или настроек в рабочей книге, которые могут повлиять на экспорт в PDF.
