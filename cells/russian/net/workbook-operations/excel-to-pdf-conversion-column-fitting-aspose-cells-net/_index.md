## Руководство по внедрению

### Функция 1: Загрузка рабочей книги из исходного каталога
#### Обзор
Загрузка книги Excel в память — это первый шаг в ее обработке и преобразовании.
##### Шаг 1: Настройте каталоги и пути к файлам
```csharp
using System;
using Aspose.Cells;
// Определите пути для исходных и выходных каталогов
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string excelFileName = "sampleFitAllWorksheetColumns.xlsx";
// Загрузить книгу из указанного пути файла
Workbook book = new Workbook(SourceDir + "/" + excelFileName);
```
**Объяснение**: Заменять `YOUR_SOURCE_DIRECTORY` с вашим реальным путем к каталогу. Этот фрагмент кода инициализирует `Workbook` объект, загрузив файл Excel и подготовив его к дальнейшей обработке.

### Функция 2: Настройка параметров сохранения PDF-файла
#### Обзор
Настройте способ сохранения книги в формате PDF, чтобы все столбцы помещались на одной странице на каждом листе.
##### Шаг 2: Настройка параметров сохранения PDF
```csharp
// Инициализировать PdfSaveOptions
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.AllColumnsInOnePagePerSheet = true;
```
**Объяснение**: Устанавливая `AllColumnsInOnePagePerSheet` значение true, вы указываете Aspose.Cells настроить ширину столбцов таким образом, чтобы все столбцы помещались на одной странице каждого листа в PDF-файле.

### Функция 3: Сохранение рабочей книги в формате PDF с настраиваемыми параметрами
#### Обзор
Сохраните загруженную книгу в файл PDF, используя настроенные параметры.
##### Шаг 3: Укажите выходные данные и сохраните
```csharp
using System.IO;
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
string pdfFileName = "outputFitAllWorksheetColumns.pdf";
// Сохраните книгу в формате PDF с указанными параметрами сохранения.
book.Save(OutputDir + "/" + pdfFileName, saveOptions);
```
**Объяснение**: Заменять `YOUR_OUTPUT_DIRECTORY` с желаемым выходным путем. Этот код сохраняет книгу в формате PDF, применяя конфигурации для размещения всех столбцов на одной странице.
