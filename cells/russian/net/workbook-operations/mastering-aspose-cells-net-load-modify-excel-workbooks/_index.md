## Руководство по внедрению

### Функция 1: Загрузка и сохранение рабочей книги

#### Обзор
Эта функция демонстрирует, как загрузить существующую книгу Excel, внести изменения и сохранить ее как новый файл с помощью Aspose.Cells для .NET.

#### Пошаговая реализация

##### Загрузка рабочей книги
Для начала создайте `Workbook` объект, указав путь к исходному файлу Excel. Это загрузит всю книгу Excel в память.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Загрузить существующую книгу из указанного каталога
Workbook workbook = new Workbook(SourceDir + "Book1.xls");
```

##### Сохранение рабочей книги
После загрузки вы можете сохранить книгу в другом месте или с изменениями. Этот шаг записывает изменения обратно в файл Excel.
```csharp
// Сохраните загруженную книгу как новый файл в выходном каталоге.
workbook.Save(outputDir + "output.xls");
```

### Функция 2: Доступ и изменение ячеек рабочего листа

#### Обзор
Эта функция показывает, как получить доступ к определенным рабочим листам в рабочей книге и изменить содержимое ячеек, включая добавление формул.

#### Пошаговая реализация

##### Доступ к рабочему листу
Вы можете получить доступ к отдельным рабочим листам по их индексу. Здесь мы сосредоточимся на первом рабочем листе:
```csharp
// Загрузите файл Excel еще раз, если он еще не загружен.
Workbook workbook = new Workbook(SourceDir + "Book1.xls");

// Доступ к первому рабочему листу в рабочей книге
Worksheet worksheet = workbook.Worksheets[0];
```

##### Изменение содержимого ячеек с помощью формул
Aspose.Cells поддерживает нотацию R1C1 для формул, что позволяет использовать относительные ссылки. Вот как задать формулу для ячейки A11:
```csharp
// Установите формулу R1C1 в ячейке A11
worksheet.Cells["A11"].R1C1Formula = ";=SUM(R[-10]C[0]:R[-7]C[0])";
```

##### Сохранение рабочей книги с изменениями
После внесения изменений сохраните книгу, как и прежде:
```csharp
// Сохраните измененную книгу в новом файле.
tworkbook.Save(outputDir + "output_with_formula.xls");
```
