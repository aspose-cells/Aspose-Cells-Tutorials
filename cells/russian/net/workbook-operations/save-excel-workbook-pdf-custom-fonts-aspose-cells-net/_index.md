### Сохранение рабочей книги в формате PDF с пользовательскими шрифтами
Эта функция позволяет вам настраивать способ отображения ваших рабочих книг Excel в PDF-файлах, указывая индивидуальные настройки шрифтов. Это гарантирует, что все шрифты, используемые в вашем документе, будут отображаться правильно в выходном файле.

#### Настройте пользовательские параметры шрифта
Сначала создайте каталог для пользовательских шрифтов и настройте Aspose.Cells для использования этих шрифтов:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

IndividualFontConfigs fontConfigs = new IndividualFontConfigs();
fontConfigs.SetFontFolder(SourceDir + "/CustomFonts", false); // Настройте папку, в которой будут храниться ваши пользовательские шрифты.
```
#### Параметры загрузки с пользовательскими шрифтами
Примените эти конфигурации для загрузки параметров при открытии книги:
```csharp
LoadOptions opts = new LoadOptions(LoadFormat.Xlsx);
opts.FontConfigs = fontConfigs; // Назначьте настроенные параметры шрифта параметрам загрузки.

Workbook wb = new Workbook(SourceDir + "/sampleSpecifyIndividualOrPrivateSetOfFontsForWorkbookRendering.xlsx", opts); // Загрузите файл Excel с пользовательскими шрифтами.
```
#### Сохранить как PDF
Наконец, сохраните загруженную книгу в формате PDF, убедившись, что используются все указанные шрифты:
```csharp
wb.Save(outputDir + "/outputSpecifyIndividualOrPrivateSetOfFontsForWorkbookRendering.pdf", SaveFormat.Pdf);
```
**Советы по устранению неполадок**: Если ваши пользовательские шрифты отображаются неправильно:
- Убедитесь, что файлы шрифтов имеют поддерживаемые форматы (например, .ttf, .otf).
- Проверьте правильность пути к каталогу пользовательских шрифтов.
