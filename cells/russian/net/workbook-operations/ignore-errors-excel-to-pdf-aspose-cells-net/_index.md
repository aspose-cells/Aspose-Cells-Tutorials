## Руководство по внедрению

### Обзор функции игнорирования ошибок

Основная цель здесь — преобразовать файл Excel в PDF, игнорируя ошибки рендеринга. Это гарантирует, что незначительные проблемы в вашей таблице не прерывают весь процесс преобразования.

### Реализация параметров сохранения PDF-файлов

Давайте разберем каждый шаг, необходимый для реализации этой функции:

#### Шаг 1: Загрузите свою рабочую книгу

Сначала загрузите книгу Excel, которую вы хотите преобразовать:
```csharp
string sourceDir = "path/to/your/source/directory";
Workbook wb = new Workbook(sourceDir + "sampleErrorExcel2Pdf.xlsx");
```

#### Шаг 2: Настройте параметры сохранения PDF-файла

Настройте свой `PdfSaveOptions` объект для игнорирования ошибок во время преобразования:
```csharp
// Укажите параметры сохранения PDF-файла — Игнорировать ошибку
PdfSaveOptions opts = new PdfSaveOptions();
opts.IgnoreError = true;
```

- **Параметры**:
  - `IgnoreError`: Логический флаг, который при установке в значение `true`, позволяет библиотеке обходить ошибки, возникающие во время рендеринга.

#### Шаг 3: Конвертировать и сохранить

Наконец, сохраните свою рабочую книгу в формате PDF, используя следующие параметры:
```csharp
string outputDir = "path/to/your/output/directory";
wb.Save(outputDir + "outputErrorExcel2Pdf.pdf", opts);
```
