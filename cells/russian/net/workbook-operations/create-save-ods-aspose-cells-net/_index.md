### Создайте и сохраните ODS-файл с характеристиками ODF 1.2 по умолчанию

#### Обзор

Эта функция позволяет создать простой ODS-файл с помощью Aspose.Cells с настройками спецификации ODF 1.2 по умолчанию.

#### Пошаговая реализация

##### Шаг 1: Настройте пути к каталогам

Определите исходный и выходной каталоги:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY"; // Укажите здесь путь к исходному каталогу
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Укажите здесь путь к выходному каталогу
```

##### Шаг 2: Создайте новую рабочую книгу

Инициализируйте новый экземпляр рабочей книги:
```csharp
Workbook workbook = new Workbook();
```

##### Шаг 3: Доступ к рабочему листу и его изменение

Откройте первый рабочий лист и вставьте данные в ячейку A1:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Welcome to Aspose!");
```

##### Шаг 4: Настройте параметры сохранения и сохраните файл.

Настройте параметры сохранения ODS для спецификации ODF 1.2 по умолчанию и сохраните файл:
```csharp
OdsSaveOptions options = new OdsSaveOptions();
workbook.Save(outputDir + "/ODF1.2_out.ods", options);
```

### Создайте и сохраните ODS-файл с использованием спецификаций ODF 1.1

#### Обзор

Эта функция демонстрирует, как сохранить ODS-файл с помощью Aspose.Cells, строго придерживаясь спецификации ODF 1.1.

#### Пошаговая реализация


##### Шаг 2: Создайте новую рабочую книгу

Инициализируйте экземпляр рабочей книги так же, как и раньше:
```csharp
Workbook workbook = new Workbook();
```

##### Шаг 3: Доступ к рабочему листу и его изменение

Откройте рабочий лист и вставьте данные в ячейку A1:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Welcome to Aspose!");
```

Настройте параметры сохранения ODS со строгим соответствием ODF 1.1:


## Раздел часто задаваемых вопросов

1. **Что такое ОРВ?**
   - ODS (OpenDocument Spreadsheet) — открытый формат файлов XML, используемый приложениями для работы с электронными таблицами, особенно основанными на LibreOffice и OpenOffice.

2. **Как установить Aspose.Cells для .NET?**
   - Используйте диспетчер пакетов NuGet или .NET CLI, как показано в этом руководстве.

3. **Каковы спецификации ODF?**
   - ODF (формат OpenDocument) — стандарт для файлов документов, включая электронные таблицы, текстовые документы и презентации.

4. **Могу ли я использовать Aspose.Cells с другими форматами электронных таблиц?**
   - Да, Aspose.Cells поддерживает множество форматов, таких как XLSX, CSV, PDF и т. д.

5. **Что делать, если мой ODS-файл сохраняется неправильно?**
   - Убедитесь, что пути к каталогам указаны правильно и у вас есть необходимые разрешения на запись. Проверьте наличие исключений в вашем коде.
