## Руководство по внедрению

Настроив Aspose.Cells, давайте экспортируем файл Excel как CSV, сохранив разделители для пустых строк.

### Открытие и настройка рабочей книги (H2)

#### Шаг 1: Загрузите файл Excel
Укажите путь к исходному каталогу, где находится файл Excel. Создайте `Workbook` объект, чтобы открыть его:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string filePath = System.IO.Path.Combine(SourceDir, "Book1.xlsx");
Workbook wb = new Workbook(filePath);
```

#### Шаг 2: Настройте параметры сохранения
Настраивать `TxtSaveOptions` чтобы настроить сохранение CSV и гарантировать сохранение разделителей для пустых строк:
```csharp
TxtSaveOptions options = new TxtSaveOptions();
options.KeepSeparatorsForBlankRow = true; // Сохраняет разделитель в пустых строках.
```

#### Шаг 3: Сохраните книгу как CSV
Сохраните вашу рабочую книгу с настроенными параметрами в указанном выходном каталоге:
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
wb.Save(System.IO.Path.Combine(outputDir, "output.csv"), options);
```

### Советы по устранению неполадок
- **Проблемы с путями к файлам**: Убедитесь, что пути к файлам верны и доступны.
