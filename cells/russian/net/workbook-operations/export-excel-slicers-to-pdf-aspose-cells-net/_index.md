## Руководство по внедрению
### Экспорт среза Excel в PDF
Эта функция позволяет вам преобразовать диаграмму среза Excel непосредственно в документ PDF. Вот как это работает:
#### Шаг 1: Определите пути к каталогам
Сначала настройте каталоги для исходных и выходных файлов. Заменить `YOUR_SOURCE_DIRECTORY` и `YOUR_OUTPUT_DIRECTORY` с реальными путями в вашей системе.
```csharp
// ФУНКЦИЯ: Настройка путей к каталогам
string SourceDir = @"C:\\Path\\To\\Your\\ExcelFile";
string OutputDir = @"C:\\Path\\To\\Save\\PDF";
```
#### Шаг 2: Загрузите рабочую книгу
Далее загрузите файл Excel с помощью Aspose.Cells. Убедитесь, что путь к файлу правильный и доступный.
```csharp
// Загрузить существующую книгу из указанного каталога
Workbook workbook = new Workbook(SourceDir + "SampleSlicerChart.xlsx");
```
#### Шаг 3: Сохранить как PDF
Наконец, сохраните загруженную книгу как документ PDF в желаемом месте вывода.
```csharp
// Сохраните книгу как файл PDF в указанном выходном каталоге.
workbook.Save(OutputDir + "SampleSlicerChart.pdf", SaveFormat.Pdf);
```
### Объяснение фрагментов кода
- **Рабочая тетрадь**Представляет файл Excel. Этот объект позволяет вам манипулировать файлами и сохранять их.
- **СохранитьФормат.Pdf**: Указывает, что документ следует сохранить в формате PDF.
Этот простой процесс эффективно экспортирует вашу диаграмму среза в PDF-файл, делая ее готовой к распространению или архивированию.
