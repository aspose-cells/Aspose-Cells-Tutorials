### Открытие файла SpreadsheetML
Aspose.Cells упрощает чтение и обработку файлов SpreadsheetML. Вот как это можно сделать:

#### Обзор функции
Эта функция позволяет разработчикам загружать файлы SpreadsheetML в `Workbook` объект, облегчающий извлечение данных и манипулирование ими.

#### Пошаговая реализация
**1. Настройте исходный каталог**
Сначала определите путь, по которому находится ваш файл SpreadsheetML:
```csharp
string SourceDir = "/path/to/your/source/directory";
```

**2. Укажите параметры загрузки для формата SpreadsheetML**
Создавать `LoadOptions` предназначен для обработки файлов SpreadsheetML.
```csharp
LoadOptions loadOptions = new LoadOptions(LoadFormat.SpreadsheetML);
```

**3. Создайте и откройте объект «Рабочая книга»**
Используйте `Workbook` класс для открытия вашего файла:
```csharp
Workbook workbook = new Workbook(SourceDir + "/Book3.xml", loadOptions);
```
*Параметры Пояснение:*
- **Исходный каталог**: Путь, где хранится «Book3.xml».
- **LoadOptions**: Указывает, что мы имеем дело с форматом SpreadsheetML.
