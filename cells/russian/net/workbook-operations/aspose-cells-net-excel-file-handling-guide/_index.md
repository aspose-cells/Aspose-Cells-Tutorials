## Руководство по внедрению

### Открытие файлов Excel через путь

**Обзор:**
Открытие файла Excel с использованием прямого пути — простая задача. Этот метод идеально подходит для сценариев, где у вас есть доступ к файловой системе.

**Пошаговая реализация:**

#### Шаг 1: Определите путь к файлу
```csharp
// Определите каталог, содержащий ваши файлы.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Укажите путь к файлу Excel.
string filePath = dataDir + "Book1.xls";
```

#### Шаг 2: Создание объекта рабочей книги
```csharp
// Создайте новый объект Workbook и откройте указанный файл Excel.
Workbook workbook = new Workbook(filePath);
Console.WriteLine("Workbook opened using path successfully!");
```
**Объяснение:** The `Workbook` class представляет файл Excel. Передав путь к файлу, вы можете легко загрузить его в память.

### Открытие файлов Excel через Stream

**Обзор:**
Использование потоков полезно при работе с файлами, к которым нет прямого доступа на диске, или для сетевых приложений.

#### Шаг 1: Создание FileStream
```csharp
// Откройте поток в существующем файле Excel.
using (FileStream fstream = new FileStream(dataDir + "Book2.xls", FileMode.Open))
{
    // Загрузите рабочую книгу из потока.
    Workbook workbookStream = new Workbook(fstream);
    Console.WriteLine("Workbook opened using stream successfully!");
}
```
**Объяснение:** Потоки обеспечивают более гибкий способ обработки файловых операций, особенно при работе с большими файлами или сетевыми ресурсами.

### Открытие зашифрованных файлов Excel

**Обзор:**
Обработка зашифрованных файлов Excel требует указания пароля во время загрузки.

#### Шаг 1: Установите параметры загрузки
```csharp
// Определите параметры загрузки и установите пароль.
LoadOptions loadOptions = new LoadOptions();
loadOptions.Password = "1234";

// Откройте зашифрованную книгу, используя указанный пароль.
Workbook wbEncrypted = new Workbook(dataDir + "encryptedBook.xls", loadOptions);
Console.WriteLine("Encrypted excel file opened successfully!");
```
**Объяснение:** `LoadOptions` позволяет указать необходимые параметры, такие как пароли, обеспечивая безопасный доступ к защищенным файлам.
