## Руководство по внедрению

### Обзор функции преобразования

Преобразование JSON в CSV с помощью Aspose.Cells включает чтение файла JSON и импорт его данных в книгу Excel перед сохранением в виде CSV. Этот процесс гарантирует, что иерархическая структура JSON поддерживается в плоском табличном формате.

#### Шаг 1: Чтение файла JSON

```csharp
// Исходный каталог, где находится ваш JSON-файл
string sourceDir = RunExamples.Get_SourceDirectory();
string jsonFilePath = sourceDir + "SampleJson.json";

// Чтение содержимого JSON-файла
string jsonString = File.ReadAllText(jsonFilePath);
```

Здесь, `File.ReadAllText` считывает весь контент JSON в строку. Это наш первый шаг к конвертации.

#### Шаг 2: Создание и настройка рабочей книги

```csharp
// Инициализировать пустую книгу
Workbook workbook = new Workbook();

// Доступ к коллекции ячеек первого рабочего листа
Cells cells = workbook.Worksheets[0].Cells;

// Настройка JsonLayoutOptions для параметров импорта
JsonLayoutOptions options = new JsonLayoutOptions
{
    ConvertNumericOrDate = true,
    ArrayAsTable = true,
    IgnoreArrayTitle = true,
    IgnoreObjectTitle = true
};
```

The `JsonLayoutOptions` класс предоставляет различные настройки для настройки процесса преобразования. Например, `ConvertNumericOrDate` обеспечивает правильную интерпретацию числовых значений и значений дат.

#### Шаг 3: Импорт данных JSON

```csharp
// Импорт данных из строки JSON в ячейки рабочей книги, начиная со строки 0, столбца 0
JsonUtility.ImportData(jsonString, cells, 0, 0, options);
```

`JsonUtility.ImportData` Метод импортирует данные JSON в указанный рабочий лист и диапазон ячеек, используя предоставленную конфигурацию.

#### Шаг 4: Сохранить как CSV

```csharp
// Определите выходной каталог для сохранения CSV-файла.
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "SampleJson_out.csv");
```

Наконец, сохраните вашу рабочую книгу в формате CSV. `Save` Метод универсален и поддерживает различные форматы, включая CSV.
