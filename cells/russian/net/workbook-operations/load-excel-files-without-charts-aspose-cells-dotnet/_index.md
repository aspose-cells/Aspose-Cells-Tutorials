## Руководство по внедрению

### Обзор функций
Эта функция позволяет загружать книги Excel, специально исключая данные диаграмм. Это особенно полезно при работе с большими наборами данных, где данные диаграмм могут потреблять ненужные ресурсы и время обработки.

### Пошаговая реализация

#### **1. Определите исходные и выходные каталоги (H3)**

Начните с установки каталогов для исходного файла и выходного назначения:

```csharp
// Укажите пути для ваших файлов
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();
```
**Объяснение**: Эти строки определяют, где находится ваш входной файл Excel и где вы хотите сохранить обработанный вывод.

#### **2. Настройте параметры загрузки (H3)**

Настройте параметры загрузки для фильтрации данных диаграммы:

```csharp
// Создайте параметры загрузки с определенным фильтром для данных
LoadOptions loadOptions = new LoadOptions();
loadOptions.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.Chart);
```
**Объяснение**: Здесь мы создаем `LoadOptions` и применить `LoadFilter` для исключения данных диаграммы (`~LoadDataFilterOptions.Chart`). Это гарантирует, что диаграммы не будут загружены в память.

#### **3. Загрузите рабочую книгу (H3)**

Теперь загрузите свою рабочую книгу, используя следующие параметры:

```csharp
// Используйте параметры загрузки, чтобы открыть файл Excel без загрузки диаграмм
Workbook workbook = new Workbook(sourceDir + "sampleLoadExcelFileWithoutChart.xlsx", loadOptions);
```
**Объяснение**: `Workbook` конструктор принимает путь и `LoadOptions`, загружая только те данные, которые указаны вашим фильтром.

#### **4. Сохраните обработанный файл (H3)**

Наконец, сохраните обработанную книгу в желаемом формате:

```csharp
// Сохраните книгу в формате PDF без диаграмм.
workbook.Save(outputDir + "outputLoadExcelFileWithoutChart.pdf", SaveFormat.Pdf);
```
**Объяснение**: `Save` Метод выводит файл в указанный каталог и формат. Здесь мы конвертируем его в PDF.
