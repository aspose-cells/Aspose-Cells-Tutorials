## Руководство по внедрению

### Функция: загрузка рабочей книги с определенными параметрами

Эта функция позволяет загружать книгу Excel, исключая данные диаграммы, тем самым оптимизируя процесс загрузки.

#### Шаг 1: Определите исходные и выходные каталоги

Начните с указания каталогов для исходных файлов и вывода:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Настройте параметры загрузки

Создать экземпляр `LoadOptions` и установите фильтр для исключения данных диаграммы с помощью побитовых операций:

```csharp
LoadOptions options = new LoadOptions();
options.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.Chart);
```

- **Почему?** Такая конфигурация гарантирует загрузку только необходимых данных (за исключением диаграмм), что сокращает использование памяти и время загрузки.

#### Шаг 3: Загрузите рабочую книгу

Используйте указанные параметры для загрузки вашей рабочей книги:

```csharp
Workbook workbook = new Workbook(SourceDir + "sampleLoadTemplateWithoutCharts.xlsx", options);
```

- **Что происходит?** Рабочая книга открывается с определенными ограничениями, игнорируя любые встроенные в нее данные диаграмм.

#### Шаг 4: Сохраните рабочую книгу

После загрузки сохраните книгу в желаемом формате, например, PDF:

```csharp
workbook.Save(OutputDir + "outputLoadTemplateWithoutCharts.pdf", SaveFormat.Pdf);
```

- **Выгода:** Этот шаг гарантирует, что вы сможете легко обмениваться или распространять данные без ненужной информации о диаграммах.
