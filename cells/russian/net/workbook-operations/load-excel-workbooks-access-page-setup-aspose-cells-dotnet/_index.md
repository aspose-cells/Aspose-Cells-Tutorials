### Загрузка рабочих книг с определенными настройками
Эта функция демонстрирует загрузку книг Excel с использованием Aspose.Cells, уделяя особое внимание `PageSetup.IsAutomaticPaperSize` свойство.

#### Обзор
Загрузите две разные рабочие книги — в одной из которых автоматический размер бумаги установлен на значение false, а в другой — на значение true, — а затем откройте их свойства PageSetup.

#### Пошаговая реализация
1. **Загрузите рабочую книгу, установив для параметра «Автоматический размер бумаги» значение «Ложь»**
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   // Загрузите книгу, в которой автоматический размер бумаги установлен на значение false
   Workbook wb1 = new Workbook(SourceDir + "/samplePageSetupIsAutomaticPaperSize-False.xlsx");

   // Доступ к первому рабочему листу
   Worksheet ws11 = wb1.Worksheets[0];

   // Распечатать свойство IsAutomaticPaperSize
   Console.WriteLine("First Worksheet of First Workbook - IsAutomaticPaperSize: " + ws11.PageSetup.IsAutomaticPaperSize);
   ```
2. **Загрузите рабочую книгу с автоматическим размером бумаги, установленным на True**
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   // Загрузите книгу, в которой автоматический размер бумаги установлен на значение true
   Workbook wb2 = new Workbook(SourceDir + "/samplePageSetupIsAutomaticPaperSize-True.xlsx");

   // Доступ к первому рабочему листу
   Worksheet ws12 = wb2.Worksheets[0];

   // Распечатать свойство IsAutomaticPaperSize
   Console.WriteLine("First Worksheet of Second Workbook - IsAutomaticPaperSize: " + ws12.PageSetup.IsAutomaticPaperSize);
   ```

#### Объяснение
- **Параметры**: `Workbook` конструктор принимает путь к файлу для загрузки книги Excel.
- **Возвращаемые значения**: `PageSetup.IsAutomaticPaperSize` Свойство возвращает логическое значение, указывающее, устанавливается ли размер бумаги автоматически.
