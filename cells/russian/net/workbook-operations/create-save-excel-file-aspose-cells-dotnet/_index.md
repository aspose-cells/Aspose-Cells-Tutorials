## Руководство по внедрению

Чтобы создать и настроить файл Excel, выполните следующие действия:

### Создание новой рабочей книги
Создайте новую книгу Excel следующим образом:
```csharp
// Создать экземпляр Workbook (файл Excel)
Workbook workbook = new Workbook();
```

### Удаление рабочего листа по умолчанию
Удалите рабочий лист по умолчанию, если он не нужен:
```csharp
// Удалить рабочий лист по умолчанию, который создается при создании новой рабочей книги.
workbook.Worksheets.RemoveAt(0);
```

### Добавление и наименование нескольких листов
Добавьте в свою рабочую книгу пять рабочих листов и дайте им последовательные названия.
```csharp
// Добавьте 5 рабочих листов и дайте им названия.
for (int i = 0; i < 5; i++) {
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = "Sheet" + (i + 1).ToString();
}
```

### Заполнение листов данными
Заполните каждый рабочий лист данными в виде сетки.
```csharp
// Заполните листы данными
for (int i = 0; i < workbook.Worksheets.Count; i++) {
    Worksheet ws = workbook.Worksheets[i];
    for (int row = 0; row < 150; row++) {
        for (int col = 0; col < 56; col++) {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

### Сохранение рабочей книги
Сохраните вашу рабочую книгу в указанном каталоге.
```csharp
// Сохраните рабочую книгу
string outputFilePath = System.IO.Path.Combine(outputDir, "ACellsSample_out.xlsx");
workbook.Save(outputFilePath);
```
