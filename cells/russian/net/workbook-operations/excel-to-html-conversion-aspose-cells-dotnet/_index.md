## Руководство по внедрению

### Функция 1: Загрузка и сохранение книги Excel

Эта функция демонстрирует, как загрузить книгу Excel из указанного исходного каталога и сохранить ее в формате HTML с пользовательскими параметрами.

#### Обзор
Эффективная загрузка и сохранение рабочих книг обеспечивает бесперебойный обмен данными между приложениями в различных форматах.

#### Шаги:

**Шаг 1**: Определите исходный и выходной каталоги.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**Шаг 2**: Загрузите книгу Excel с помощью Aspose.Cells.
```csharp
// Загрузить существующую книгу из файла
Workbook workbook = new Workbook(SourceDir + "sampleChangeHtmlLinkTarget.xlsx");
```
*Объяснение*: `Workbook` класс используется для загрузки и обработки файлов Excel.

**Шаг 3**: Настройте параметры сохранения HTML с указанием конкретных целевых ссылок.
```csharp
// Инициализируйте HtmlSaveOptions и задайте LinkTargetType
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.LinkTargetType = HtmlLinkTargetType.Self; // Ссылки открываются в том же окне/вкладке
```
*Конфигурация ключа*: `HtmlLinkTargetType.Self` гарантирует, что все ссылки в HTML-файле будут открываться в текущей вкладке браузера.

**Шаг 4**: Сохраните книгу как HTML-файл.
```csharp
// Сохраните книгу с указанными параметрами HTML
workbook.Save(OutputDir + "outputChangeHtmlLinkTarget.html", opts);
```
*Цель*: `Save` Метод записывает рабочую книгу в указанный формат, в данном случае HTML.

### Функция 2: Настройка параметров сохранения HTML

Эта функция предназначена для настройки параметров сохранения HTML-файлов для книги Excel.

#### Обзор
Настройка параметров сохранения позволяет получать индивидуальные выходные данные, соответствующие требованиям конкретного приложения.

#### Шаги:

**Шаг 1**: Создать и настроить `HtmlSaveOptions`.
```csharp
// Создать экземпляр HtmlSaveOptions
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.LinkTargetType = HtmlLinkTargetType.Self;
```
*Объяснение*: Настройка параметров сохранения HTML, таких как `LinkTargetType` обеспечивает контроль над тем, как ваши данные отображаются в браузере.

**Шаг 2**: Сохранить с настроенными параметрами.
```csharp
// Предположим, что рабочая книга уже загружена как «рабочая книга»
workbook.Save(OutputDir + "outputChangeHtmlLinkTarget.html", opts);
```
