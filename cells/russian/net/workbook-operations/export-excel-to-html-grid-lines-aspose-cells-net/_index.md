## Руководство по внедрению

### Экспорт рабочей книги в HTML с линиями сетки

В этом разделе мы сосредоточимся на экспорте рабочей книги и обеспечении включения линий сетки в выходной HTML-файл.

#### Инициализация рабочей книги и рабочего листа

Сначала создайте новый `Workbook` объект и получить доступ к его первому рабочему листу:

```csharp
// Создать новый объект Workbook
Workbook wb = new Workbook();

// Доступ к первому рабочему листу
Worksheet ws = wb.Worksheets[0];
```

#### Заполнение данных для демонстрации

Чтобы смоделировать реальный сценарий, давайте заполним рабочий лист образцами данных:

```csharp
// Заполните рабочий лист целыми значениями.
for (int r = 0; r < 10; r++) {
    for (int c = 0; c < 10; c++) {
        ws.Cells[r, c].PutValue(r * 1);
    }
}
```

#### Настройка параметров экспорта HTML

Настройте `HtmlSaveOptions` чтобы включить линии сетки в ваш HTML-вывод:

```csharp
// Настройте параметры сохранения HTML
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.ExportGridLines = true;
```

#### Сохранение в формате HTML с линиями сетки

Наконец, сохраните книгу как HTML-файл, используя указанные параметры:

```csharp
// Сохраните книгу в формате HTML с линиями сетки.
wb.Save("YOUR_OUTPUT_DIRECTORY/outputExportToHTMLWithGridLines.html", opts);
```
