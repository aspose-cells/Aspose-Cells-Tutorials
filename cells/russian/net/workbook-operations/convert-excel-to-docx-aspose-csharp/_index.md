## Руководство по внедрению
В этом разделе мы рассмотрим преобразование файла Excel в формат DOCX.

### Обзор
Преобразование файла Excel в документ Word оптимизирует представление данных и улучшает совместную работу, упрощая редактирование и обмен информацией на разных платформах.

### Пошаговый процесс преобразования
**1. Откройте файл шаблона.**
Начните с загрузки исходного файла Excel:
```csharp
// Путь к исходному каталогу
string sourceDir = RunExamples.Get_SourceDirectory();

// Инициализируйте объект Workbook с путем к файлу Excel
Workbook workbook = new Workbook(sourceDir + "Book1.xlsx");
```
*Объяснение*: `Workbook` класс представляет файл Excel. Передавая путь к файлу его конструктору, вы загружаете электронную таблицу в память.

**2. Сохранить как DOCX**
Конвертируйте и сохраните загруженный файл Excel как документ Word:
```csharp
// Путь к выходному каталогу
string outputDir = RunExamples.Get_OutputDirectory();

// Сохраните книгу в формате DOCX.
workbook.Save(outputDir + "Book1.docx", SaveFormat.Docx);
```
*Объяснение*: `Save` Метод экспортирует текущую книгу в различные форматы. Здесь, `SaveFormat.Docx` указывает, что нам нужен документ Word.
