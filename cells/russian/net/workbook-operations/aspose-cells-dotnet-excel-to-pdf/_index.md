## Руководство по внедрению
### Загрузить и сохранить рабочую книгу в формате PDF
**Обзор:**
Мы загрузим существующий файл Excel из исходного каталога, при необходимости обработаем его и сохраним как PDF в выходном каталоге. Эта функция имеет решающее значение для приложений, требующих создания отчетов или распространения данных в универсальном формате, таком как PDF.

#### Шаг 1: Определите пути к файлам
Использование заполнителей обеспечивает гибкость при указании каталогов во время выполнения:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

// Построить полные пути
string sourceFilePath = SourceDir + "sampleRenderCustomDateFormat.xlsx";
string outputFilePath = OutputDir + "outputRenderCustomDateFormat.pdf";
```

#### Шаг 2: Загрузите файл Excel
Загрузите вашу рабочую книгу с помощью `Workbook` класс, представляющий целый файл Excel:

```csharp
// Загрузите файл Excel
Workbook workbook = new Workbook(sourceFilePath);
```

Этот шаг инициализирует наш процесс конвертации путем загрузки данных, которые мы преобразуем в PDF-файл.

#### Шаг 3: Сохранить как PDF
Конвертируйте и сохраните загруженный файл Excel в PDF с помощью `Save` метод, явно указывающий формат:

```csharp
// Сохраните его как PDF-файл в выходном каталоге.
workbook.Save(outputFilePath, SaveFormat.Pdf);
```

На этом этапе преобразование завершается, и результат сохраняется в указанном вами месте.

### Обработка путей к файлам
**Обзор:**
Динамическая обработка путей к файлам позволяет создавать более адаптивный код. Создавая пути во время выполнения, вы можете сделать свои приложения более гибкими и пригодными для повторного использования в различных средах или наборах данных.

#### Построение динамических путей
Использование заполнителей для каталогов повышает гибкость:

```csharp
string SourceDirPlaceholder = @"YOUR_SOURCE_DIRECTORY";
string OutputDirPlaceholder = @"YOUR_OUTPUT_DIRECTORY";

// Пример использования при построении пути
string sourceFilePath = SourceDirPlaceholder + "yourFile.xlsx";
string outputFilePath = OutputDirPlaceholder + "outputFile.pdf";
```

Такой подход гарантирует, что ваше приложение сможет адаптироваться к изменениям в местах хранения файлов без необходимости внесения изменений в код.
