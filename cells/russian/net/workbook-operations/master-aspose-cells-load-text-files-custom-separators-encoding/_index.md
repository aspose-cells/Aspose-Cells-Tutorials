### Загрузка текстовых файлов с пользовательским разделителем и кодировкой

#### Обзор

Эта функция позволяет вам указать пользовательский разделитель для вашего текстового файла (например, запятую для CSV) и определить тип кодировки, например UTF8. Это особенно полезно при работе с международными наборами данных или нестандартными форматами файлов.

#### Этапы внедрения

1. **Определить исходные и выходные каталоги**
   Укажите, где находятся ваши исходные текстовые файлы и куда вы хотите сохранить обработанные данные:

   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Создать экземпляр LoadOptions**
   Создать `TxtLoadOptions` объект для указания пользовательских настроек загрузки:

   ```csharp
   TxtLoadOptions txtLoadOptions = new TxtLoadOptions();
   ```

3. **Установить пользовательский разделитель и кодировку**
   Назначьте разделитель и тип кодировки:

   ```csharp
   // Укажите разделитель (например, запятую для CSV-файлов)
   txtLoadOptions.Separator = Convert.ToChar(",");

   // Укажите тип кодировки (например, UTF8)
   txtLoadOptions.Encoding = Encoding.UTF8;
   ```

4. **Создать и загрузить рабочую книгу**
   Использовать `Workbook` для загрузки текстового файла с указанными параметрами:

   ```csharp
   Workbook wb = new Workbook(SourceDir + "/Book11.csv", txtLoadOptions);
   ```

5. **Сохранить обработанные данные**
   Сохраните рабочую книгу в желаемом выходном каталоге:

   ```csharp
   wb.Save(outputDir + "/output.txt");
   ```

#### Советы по устранению неполадок
- Убедитесь, что пути проложены правильно и доступны.
- Проверьте соответствие разделителя и кодировки спецификациям файла, чтобы избежать ошибок синтаксического анализа.
