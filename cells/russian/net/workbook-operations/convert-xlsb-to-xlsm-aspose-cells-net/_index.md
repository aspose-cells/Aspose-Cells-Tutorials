## Руководство по внедрению

Чтобы преобразовать файл XLSB в XLSM с помощью Aspose.Cells, выполните следующие действия.

### Открыть и загрузить рабочую книгу

**Обзор:** Загрузите существующий файл XLSB в память, указав исходный каталог.

#### Шаг 1: Укажите исходный каталог
```csharp
// Замените «YOUR_SOURCE_DIRECTORY» на фактический путь, где находятся ваши исходные файлы.
string sourceDir = "YOUR_SOURCE_DIRECTORY";
```

#### Шаг 2: Определите и загрузите путь к рабочей книге
```csharp
// Укажите путь к файлу книги XLSB, которую необходимо преобразовать.
string filePath = Path.Combine(sourceDir, "sampleConvertXLSBToXLSM.xlsb");
// Загрузите книгу по указанному пути.
Workbook workbook = new Workbook(filePath);
```

### Сохранить книгу в другом формате

**Обзор:** После загрузки рабочей книги сохраните ее как файл XLSM.

#### Шаг 1: Установка выходного каталога
```csharp
// Замените «YOUR_OUTPUT_DIRECTORY» на фактический путь, по которому вы хотите сохранить выходные файлы.
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Сохраните книгу как XLSM
```csharp
// Определите новое имя файла и путь для книги XLSM.
string outputFilePath = Path.Combine(outputDir, "outputConvertXLSBToXLSM.xlsm");
// Сохраните книгу в формате XLSM.
workbook.Save(outputFilePath, SaveFormat.Xlsm);
```
