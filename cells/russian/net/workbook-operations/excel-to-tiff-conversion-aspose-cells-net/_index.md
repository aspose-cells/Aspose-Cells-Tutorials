### Преобразовать лист Excel в изображение TIFF
В этом разделе демонстрируется преобразование листа Excel в многостраничное изображение TIFF с помощью Aspose.Cells для .NET.

#### Шаг 1: Загрузите свою рабочую книгу
Начните с загрузки книги Excel:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleConvertWorksheetToImageByPage.xlsx");
```
Здесь мы загружаем файл Excel из указанного каталога.

#### Шаг 2: Получите доступ к своему рабочему листу
Получите доступ к рабочему листу, который вы хотите преобразовать. В этом примере мы используем первый рабочий лист:
```csharp
Worksheet sheet = book.Worksheets[0];
```

#### Шаг 3: Настройте параметры изображения
Настройте параметры преобразования изображения, включая разрешение и формат:
```csharp
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.HorizontalResolution = 200;
options.VerticalResolution = 200;
options.ImageType = Drawing.ImageType.Tiff; // Установить вывод в формате TIFF
```
Эти конфигурации гарантируют, что генерируемые изображения имеют высокое разрешение и находятся в формате TIFF.

#### Шаг 4: Преобразование страниц рабочего листа в изображения
Использовать `SheetRender` чтобы преобразовать каждую страницу вашего рабочего листа в изображение:
```csharp
SheetRender sr = new SheetRender(sheet, options);
for (int j = 0; j < sr.PageCount; j++)
{
    sr.ToImage(j, outputDir + "outputConvertWorksheetToImageByPage_" + (j + 1) + ".tif");
}
```
Этот цикл проходит по каждой странице рабочего листа и сохраняет ее как изображение TIFF.
