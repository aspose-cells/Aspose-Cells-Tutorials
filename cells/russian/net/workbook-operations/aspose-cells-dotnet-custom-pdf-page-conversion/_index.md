## Руководство по внедрению

### Пользовательские обратные вызовы для преобразования страниц PDF

#### Обзор
Настройка преобразования страниц имеет решающее значение для создания определенных отчетов или сводок. `IPageSavingCallback` Интерфейс позволяет управлять каждой страницей во время конвертации.

#### Пошаговая реализация

**1. Определите класс обратного вызова**
Создайте класс, реализующий `IPageSavingCallback`:
```csharp
using Aspose.Cells.Rendering;

public class TestPageSavingCallback : IPageSavingCallback
{
    public void PageStartSaving(PageStartSavingArgs args)
    {
        Console.WriteLine("Starting to save page index {0} of pages {1}", args.PageIndex, args.PageCount);
        
        // Исключить страницы до второй.
        if (args.PageIndex < 2)
        {
            args.IsToOutput = false;
        }
    }

    public void PageEndSaving(PageEndSavingArgs args)
    {
        Console.WriteLine("Ending save of page index {0} of pages {1}", args.PageIndex, args.PageCount);

        // Прекратите обработку после восьмой страницы.
        if (args.PageIndex >= 8)
        {
            args.HasMorePages = false;
        }
    }
}
```

**2. Настройте параметры сохранения PDF-файла**
Настройте свой `PdfSaveOptions` и назначьте свой обратный вызов:
```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.PageSavingCallback = new TestPageSavingCallback();
```

**3. Выполнить преобразование**
Конвертируйте свою рабочую книгу в PDF с указанными параметрами:
```csharp
workbook.Save("output-file-path.pdf", pdfSaveOptions);
```
