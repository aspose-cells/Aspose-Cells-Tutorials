## Руководство по внедрению

### Экспорт Excel в HTML без фреймовых скриптов

#### Обзор
Мы покажем вам, как экспортировать файл Excel в HTML, отключив скрипты фреймов. Это гарантирует легкий и чистый вывод HTML.

#### Пошаговая реализация
**Шаг 1: Откройте нужную рабочую книгу**
Сначала укажите исходный каталог и откройте книгу, которую вы хотите преобразовать.
```csharp
// Исходный каталог
string sourceDir = RunExamples.Get_SourceDirectory();

// Откройте нужную книгу
Workbook wb = new Workbook(sourceDir + "sampleHtmlExportFrameScripts.xlsx");
```

**Шаг 2: Настройте параметры сохранения HTML**
Далее настройте `HtmlSaveOptions` для отключения фреймовых скриптов.
```csharp
// Отключить экспорт сценариев фрейма и свойств документа
HtmlSaveOptions options = new HtmlSaveOptions();
options.ExportFrameScriptsAndProperties = false;
```

**Шаг 3: Сохраните книгу как HTML**
Наконец, сохраните вашу рабочую книгу в формате HTML, используя настроенные параметры.
```csharp
// Выходной каталог
string outputDir = RunExamples.Get_OutputDirectory();

// Сохранить книгу как HTML
wb.Save(outputDir + "outputHtmlExportFrameScripts.html", options);

Console.WriteLine("HtmlExportFrameScripts executed successfully.");
```

### Советы по устранению неполадок
- **Распространенная проблема**: Если выходной файл не соответствует ожидаемому, проверьте, `options.ExportFrameScriptsAndProperties` установлено значение false.
- Убедитесь, что пути в вашем коде для исходных и выходных каталогов заданы правильно.
