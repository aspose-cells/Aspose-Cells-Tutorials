### Открытие файлов Excel через Stream

Открытие файлов с использованием потоков может быть полезным для приложений, где размер файла или ограничения памяти являются проблемой. Вот как можно реализовать эту функциональность:

#### Шаг 1: Создание объекта FileStream

Первый шаг — создать `FileStream` объект, позволяющий считывать данные файла в ваше приложение.
```csharp
// Путь к каталогу документов.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
// Создание объекта Stream
using (FileStream fstream = new FileStream(dataDir + "Book2.xls", FileMode.Open))
{
```
#### Шаг 2: Откройте рабочую книгу из Stream

Теперь, когда ваш поток готов, вы можете открыть файл Excel с помощью Aspose.Cells.
```csharp
    // Создание объекта Workbook и открытие файла из потока
    Workbook workbook2 = new Workbook(fstream);
    Console.WriteLine("Workbook opened using stream successfully!");
}
```
**Параметры и методы:**
- `FileStream`: Используется для чтения данных. Убедитесь, что он поддерживает поиск для эффективного чтения.
- `Workbook`: Основной объект Aspose.Cells, представляющий файл Excel.

#### Советы по устранению неполадок

Если у вас возникли проблемы, убедитесь, что:
- Путь к файлу правильный и доступный.
- Ваш поток поддерживает поиск, который может потребоваться для Aspose.Cells.
