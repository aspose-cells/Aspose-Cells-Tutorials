### Отображение пустого рабочего листа в виде изображения PNG

Эта функция полезна для создания визуальных представлений рабочих листов без данных. Вот как это реализовать:

#### Шаг 1: Создание и настройка рабочей книги

Создайте новый экземпляр рабочей книги, включающий один рабочий лист по умолчанию.
```csharp
// Инициализировать новый экземпляр рабочей книги
Workbook wb = new Workbook();

// Доступ к первому (по умолчанию) рабочему листу
Worksheet ws = wb.Worksheets[0];
```

#### Шаг 2: Настройте параметры изображения

Настроить `ImageOrPrintOptions` чтобы указать PNG в качестве выходного формата и обеспечить генерацию изображения для пустых листов.
```csharp
// Настройте параметры изображения или печати
ImageOrPrintOptions opts = new ImageOrPrintOptions {
    // Формат вывода установлен на PNG
    ImageType = Drawing.ImageType.Png,
    
    // Убедитесь, что изображение создается даже для пустых листов.
    OutputBlankPageWhenNothingToPrint = true
};
```

#### Шаг 3: Визуализация рабочего листа

Использовать `SheetRender` для создания изображения и сохранения его в указанном вами выходном каталоге.
```csharp
// Сконвертируйте рабочий лист в файл PNG.
SheetRender sr = new SheetRender(ws, opts);
sr.ToImage(0, "YOUR_OUTPUT_DIRECTORY\OutputBlankPageWhenNothingToPrint.png");
```

Этот фрагмент кода создает изображение пустого рабочего листа и сохраняет его как `OutputBlankPageWhenNothingToPrint.png` в вашем выходном каталоге.
