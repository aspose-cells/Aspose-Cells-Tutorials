## Руководство по внедрению

Мы разобьем руководство на логические разделы на основе функций.

### Создание рабочей книги

Рабочая книга служит базовым объектом, представляющим файл Excel. Вот как ее создать:

#### Шаг 1: Создание новой рабочей книги
```csharp
// Создайте новый экземпляр класса Workbook.
Workbook workbook = new Workbook();
```

Этот `Workbook` объект представляет ваш файл Excel, позволяя вам добавлять рабочие листы и данные.

### Добавление рабочего листа

Добавление рабочего листа становится простым, если у вас есть экземпляр рабочей книги:

#### Шаг 2: Добавьте новый рабочий лист
```csharp
// Добавить новый рабочий лист в рабочую книгу
int sheetIndex = workbook.Worksheets.Add();
```

Этот метод возвращает индекс для доступа к недавно добавленному рабочему листу.

#### Шаг 3: Получите доступ к рабочему листу и дайте ему имя

Чтобы работать с новым рабочим листом, откройте его по индексу и присвойте ему имя:

```csharp
// Получить ссылку на вновь созданный рабочий лист
Worksheet newSheet = workbook.Worksheets[sheetIndex];

// Задайте имя для легкой идентификации.
newSheet.Name = "My Worksheet";
```

### Сохранение рабочей книги

Наконец, сохраните ваши изменения обратно в файл Excel:

#### Шаг 4: Определите выходной путь и сохраните

```csharp
using System.IO;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Перед сохранением убедитесь, что выходной каталог существует.
bool isExists = Directory.Exists(outputDir);
if (!isExists)
    Directory.CreateDirectory(outputDir);

// Определите путь для выходного файла
string outputPath = Path.Combine(outputDir, "output.out.xls");

// Сохраните книгу в указанном месте.
workbook.Save(outputPath);
```

Этот шаг гарантирует сохранение вашей рабочей книги в нужном формате и месте.
