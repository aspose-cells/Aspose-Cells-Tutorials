## Руководство по внедрению: настройка метаданных рабочей книги

Давайте разобьем процесс на управляемые этапы.

### 1. Инициализируйте рабочую книгу и задайте параметры метаданных

Сначала вам нужно указать, с какими свойствами метаданных вы хотите работать. В этом примере мы сосредоточимся на свойствах документа:

```csharp
using Aspose.Cells;
using Aspose.Cells.Metadata;

// Определите каталоги для исходных и выходных файлов
string sourceDir = "path_to_source_directory";
string outputDir = "path_to_output_directory";

// Инициализировать параметры метаданных
MetadataOptions options = new MetadataOptions(MetadataType.DocumentProperties);

// Загрузите рабочую книгу с указанными параметрами метаданных
WorkbookMetadata meta = new WorkbookMetadata(sourceDir + "sampleUsingWorkbookMetadata.xlsx", options);
```

### 2. Добавить пользовательские свойства документа

Пользовательские свойства полезны для добавления конкретной информации, относящейся к вашей организации или проекту:

```csharp
// Добавить пользовательское свойство документа
meta.CustomDocumentProperties.Add("MyTest", "This is My Test");
```

**Почему это важно:** Установив пользовательские метаданные, вы можете отслеживать дополнительный контекст содержимого рабочей книги, например сведения об авторстве, версии и многое другое.

### 3. Сохраните обновленные метаданные

После настройки свойств сохраните их, чтобы изменения сохранились:

```csharp
// Сохраните обновленные метаданные обратно в новый файл.
meta.Save(outputDir + "outputUsingWorkbookMetadata.xlsx");
```

### 4. Чтение и отображение метаданных

Чтобы проверить изменения, откройте книгу и прочитайте пользовательское свойство:

```csharp
// Откройте книгу с обновленными метаданными.
Workbook w = new Workbook(outputDir + "outputUsingWorkbookMetadata.xlsx");

// Отображение пользовательского свойства документа
Console.WriteLine("Metadata Custom Property MyTest: " + w.CustomDocumentProperties["MyTest"]);
```
