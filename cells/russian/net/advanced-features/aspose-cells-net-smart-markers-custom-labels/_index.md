## Руководство по внедрению

Давайте разобьем реализацию на удобные для управления разделы, чтобы помочь вам эффективно использовать смарт-маркеры и настраивать метки.

### Шаг 1: Подготовка рабочей тетради

Сначала мы подготовим наш шаблон рабочей книги, содержащий Smart Markers. Эти маркеры выступают в качестве заполнителей в вашем файле Excel, которые будут заменены фактическими данными во время обработки.

```csharp
// Путь к каталогу документов.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Загрузите рабочую книгу, содержащую смарт-маркеры
Workbook designer = new Workbook(dataDir + "SmartMarker_Designer.xlsx");
```

### Шаг 2: Экспорт данных

Нам нужны данные для заполнения нашего шаблона. Здесь мы экспортируем их из существующего файла Excel.

```csharp
// Создать новый объект Workbook для исходного файла
Workbook workbook = new Workbook(dataDir + "Book1.xlsx");

// Экспортировать данные из первого рабочего листа в DataTable
DataTable dt = workbook.Worksheets[0].Cells.ExportDataTable(0, 0, 11, 5, true);

// Присвойте имя DataTable
dt.TableName = "Report";
```

### Шаг 3: Настройка WorkbookDesigner

Далее используйте `WorkbookDesigner` для привязки данных к вашим смарт-маркерам.

```csharp
// Создать экземпляр класса WorkbookDesigner
WorkbookDesigner d = new WorkbookDesigner();

// Установите рабочую тетрадь дизайнера
d.Workbook = designer;

// Назначить DataTable в качестве источника данных
d.SetDataSource(dt);

// Обработать смарт-маркеры в шаблоне
d.Process();
```

### Шаг 4: Сохранение результата

После обработки сохраните файл, чтобы завершить автоматизацию.

```csharp
// Сохраните выходной файл.
designer.Save(dataDir + "output.xlsx", SaveFormat.Xlsx);
```

**Совет по устранению неполадок:** Убедитесь, что синтаксис вашего Smart Marker в шаблоне соответствует структуре источника данных. Распространенные проблемы включают несоответствующие имена или неправильные форматы заполнителей.
