## Руководство по внедрению

Для ясности и простоты понимания мы разберем реализацию на отдельные функции.

### Настройка каталога

#### Обзор
Первый шаг — убедиться, что существуют каталоги для ввода и вывода. Это позволяет избежать ошибок времени выполнения при попытке чтения или записи по несуществующим путям.

#### Этапы внедрения
**1. Определить каталоги**
Настройте пути к исходному и выходному каталогам:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";
```

**2. Проверка и создание каталогов**
Используйте следующий фрагмент кода, чтобы проверить, существуют ли каталоги, и создать их, если их нет:
```csharp
if (!Directory.Exists(SourceDir))
{
    Directory.CreateDirectory(SourceDir);
}

if (!Directory.Exists(OutputDir))
{
    Directory.CreateDirectory(OutputDir);
}
```

### Создание рабочей книги и доступ к рабочим листам

#### Обзор
Создание рабочей книги и доступ к ее рабочим листам просты с Aspose.Cells. В этом разделе показано, как создать экземпляр новой рабочей книги и получить рабочий лист по умолчанию.

#### Этапы внедрения
**1. Создание новой рабочей книги**
Создайте новый экземпляр `Workbook`:
```csharp
Workbook book = new Workbook();
```

**2. Доступ к рабочему листу по умолчанию**
Откройте первый рабочий лист в рабочей книге:
```csharp
Worksheet sheet = book.Worksheets[0];
```

### Разрешить редактирование диапазонов управления

#### Обзор
Защита определенных диапазонов в ваших рабочих листах имеет решающее значение для целостности данных. Эта функция позволяет вам определять и защищать эти области.

#### Этапы внедрения
**1. Извлечь диапазоны разрешенного редактирования**
Доступ к коллекции разрешенных диапазонов редактирования:
```csharp
ProtectedRangeCollection allowRanges = sheet.AllowEditRanges;
```

**2. Создайте и защитите диапазон**
Определите защищенный диапазон, установите его пароль и примените защиту ко всему рабочему листу:
```csharp
int idx = allowRanges.Add("r2", 1, 1, 3, 3);
ProtectedRange protectedRange = allowRanges[idx];
protectedRange.Password = "123";
sheet.Protect(ProtectionType.All);
```

### Сохранение рабочей книги

#### Обзор
После настройки рабочей книги сохраните ее в указанном каталоге.

#### Этапы внедрения
**1. Определите путь к выходному файлу**
Объедините путь к выходному каталогу с желаемым именем файла:
```csharp
string outputFilePath = Path.Combine(OutputDir, "protectedrange.out.xls");
```

**2. Сохраните рабочую книгу.**
Сохраните файл Excel с помощью `Save` метод:
```csharp
book.Save(outputFilePath);
```
