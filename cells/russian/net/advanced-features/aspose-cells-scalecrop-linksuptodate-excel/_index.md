## Руководство по внедрению

В этом разделе вы узнаете, как настроить `ScaleCrop` и `LinksUpToDate` свойства в документах Excel с помощью Aspose.Cells.

### Настройка свойства ScaleCrop

The `ScaleCrop` свойство гарантирует, что изображения вписываются в границы ячеек без искажений. Вот как это установить:

#### Шаг 1: Создание экземпляра объекта Workbook
```csharp
// Создайте новый экземпляр класса Workbook.
Workbook workbook = new Workbook();
```

#### Шаг 2: Настройка ScaleCrop
```csharp
// Включите ScaleCrop для сохранения пропорций изображения в ячейках
workbook.BuiltInDocumentProperties.ScaleCrop = true;
```

### Настройка свойства LinksUpToDate

The `LinksUpToDate` свойство проверяет, являются ли гиперссылки документа актуальными. Чтобы установить это:

#### Шаг 1: Настройте LinksUpToDate
```csharp
// Установите LinksUpToDate, чтобы обеспечить действительность гиперссылки
workbook.BuiltInDocumentProperties.LinksUpToDate = true;
```

### Сохранение вашей рабочей книги

Наконец, сохраните настроенную вами рабочую книгу, применив следующие настройки:
```csharp
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "outputSettingScaleCropAndLinksUpToDateProperties.xlsx", SaveFormat.Xlsx);
Console.WriteLine("SettingScaleCropAndLinksUpToDateProperties executed successfully.");
```

### Советы по устранению неполадок

- **Файл не найден**: Обеспечить `outputDir` правильно настроен и доступен.
- **Ошибки лицензии**: Проверьте путь к файлу лицензии и его действительность, если возникнут соответствующие ошибки.
