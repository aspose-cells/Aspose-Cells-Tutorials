### Обзор обнаружения пустых рабочих листов

Обнаружение пустых рабочих листов помогает управлять и оптимизировать большие наборы данных. Эта функциональность имеет решающее значение для таких задач, как очистка данных и создание отчетов.

#### Шаг 1: Загрузите свою рабочую книгу
Сначала создайте экземпляр `Workbook` класс для загрузки файла электронной таблицы:

```csharp
// Загрузить существующую рабочую книгу
string sourceDir = RunExamples.Get_SourceDirectory();
var book = new Workbook(sourceDir + "sampleDetectEmptyWorksheets.xlsx");
```

#### Шаг 2: Просмотрите рабочие листы

Просмотрите каждый рабочий лист в рабочей книге и проверьте его содержимое.

##### Проверить наличие заселенных ячеек
Если какие-либо ячейки заполнены, лист не пуст:

```csharp
for (int i = 0; i < book.Worksheets.Count; i++)
{
    Worksheet sheet = book.Worksheets[i];
    
    if (sheet.Cells.MaxDataRow != -1)
    {
        Console.WriteLine(sheet.Name + " is not Empty because one or more Cells are Populated");
    }
}
```

##### Проверьте наличие фигур
Листы могут содержать фигуры, что делает их непустыми:

```csharp
else if (sheet.Shapes.Count > 0)
{
    Console.WriteLine(sheet.Name + " is not Empty because there are one or more Shapes");
}
```

##### Проверка инициализированных ячеек

Для полностью пустых листов проверьте инициализированные ячейки:

```csharp
else
{
    Aspose.Cells.Range range = sheet.Cells.MaxDisplayRange;
    var rangeIterator = range.GetEnumerator();
    
    if (rangeIterator.MoveNext())
    {
        Console.WriteLine(sheet.Name + " is not Empty because one or more cells are Initialized");
```
