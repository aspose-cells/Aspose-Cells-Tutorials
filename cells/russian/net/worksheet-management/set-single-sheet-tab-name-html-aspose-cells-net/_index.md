## Руководство по внедрению
Давайте настроим имя вкладки отдельного листа в HTML с помощью Aspose.Cells для .NET. Этот процесс включает загрузку вашего файла Excel, указание параметров экспорта и сохранение его как файла HTML с пользовательскими настройками.

### Загрузите образец файла Excel
Начните с загрузки книги Excel, содержащей только один лист:
```csharp
// Укажите исходный каталог
string sourceDir = "Your source directory path";
Workbook wb = new Workbook(sourceDir + "sampleSingleSheet.xlsx");
```
Здесь мы загружаем одностраничный файл Excel в `Workbook` объект. Убедитесь, что путь к файлу указан правильно.

### Настройте параметры сохранения HTML
Чтобы настроить экспорт листа Excel в HTML, используйте `HtmlSaveOptions` сорт:
```csharp
// Укажите параметры сохранения HTML
Aspose.Cells.HtmlSaveOptions options = new Aspose.Cells.HtmlSaveOptions();
options.Encoding = System.Text.Encoding.UTF8;
options.ExportImagesAsBase64 = true; // Встраивайте изображения непосредственно в HTML-файл
options.ExportGridLines = true;      // Экспортируйте линии сетки для сохранения структуры
options.ExportSimilarBorderStyle = true;
options.ExportBogusRowData = true;   // Включить скрытые данные строк и столбцов
options.ExcludeUnusedStyles = true;  // Уменьшить размер, исключив неиспользуемые стили
options.ExportHiddenWorksheet = false; // Экспортировать только видимые рабочие листы
```
### Экспортировать рабочую книгу в HTML
Установив нужные параметры, вы теперь можете сохранить книгу в формате HTML:
```csharp
// Укажите выходной каталог
string outputDir = "Your output directory path";
wb.Save(outputDir + "outputSampleSingleSheet.htm", options);
Console.WriteLine("Export executed successfully.");
```
Этот код сохраняет ваш одностраничный файл Excel как HTML-документ со всеми указанными настройками.
