## Руководство по внедрению

### Функция 1: Объединение нескольких файлов Excel

Эта функция демонстрирует, как объединить несколько файлов .xls в один вывод с помощью Aspose.Cells.

#### Шаг 1: Определите исходные и выходные каталоги

Укажите пути для исходного и целевого каталогов:

```csharp
string YOUR_SOURCE_DIRECTORY = "YOUR_SOURCE_DIRECTORY";
string YOUR_OUTPUT_DIRECTORY = "YOUR_OUTPUT_DIRECTORY";
```

#### Шаг 2: Укажите файлы для слияния

Создайте массив путей к файлам, которые вы хотите объединить:

```csharp
String[] files = new String[2];
files[0] = YOUR_SOURCE_DIRECTORY + "/sampleMergeFiles_Book1.xls";
files[1] = YOUR_SOURCE_DIRECTORY + "/sampleMergeFiles_Book2.xls";
```

#### Шаг 3: Выполнение слияния

Использовать `CellsHelper.MergeFiles` чтобы объединить файлы Excel в одну книгу:

```csharp
string cacheFile = YOUR_OUTPUT_DIRECTORY + "/cacheMergeFiles.txt";
string dest = YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls";

CellsHelper.MergeFiles(files, cacheFile, dest);
```

### Функция 2: Переименование листов в объединенном файле Excel

После объединения файлов вам, возможно, захочется переименовать каждый лист для лучшей организации.

#### Шаг 1: Загрузите рабочую книгу

Загрузите книгу, в которой будут переименованы листы:

```csharp
Workbook workbook = new Workbook(YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls");
```

#### Шаг 2: Последовательное переименование листов

Пройдитесь по каждому рабочему листу и присвойте новое имя:

```csharp
int i = 1;
foreach (Worksheet sheet in workbook.Worksheets)
{
    sheet.Name = "Sheet" + i++;
}
```

#### Шаг 3: Сохраните рабочую книгу

Наконец, сохраните изменения, чтобы сохранить переименованные листы:

```csharp
workbook.Save(YOUR_OUTPUT_DIRECTORY + "/outputMergeFiles.xls");
```
