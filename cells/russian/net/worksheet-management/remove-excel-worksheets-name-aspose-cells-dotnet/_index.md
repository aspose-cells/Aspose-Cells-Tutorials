### Удаление рабочих листов с использованием имен листов
Удаление определенных листов может иметь решающее значение для управления данными. Давайте посмотрим, как это работает:

#### Шаг 1: Загрузите файл Excel
Начните с загрузки файла Excel с помощью `FileStream`.

```csharp
string dataDir = "your_directory_path_here";

// Создайте FileStream для открытия файла Excel.
using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
{
    // Создайте объект Workbook и загрузите файл через поток.
    Workbook workbook = new Workbook(fstream);
}
```
*Зачем использовать `FileStream`?* Он позволяет эффективно управлять файлами, гарантируя высвобождение ресурсов после завершения операций.

#### Шаг 2: Удалите рабочий лист
Теперь давайте удалим рабочий лист по его имени:

```csharp
// Удалить рабочий лист, используя его имя
workbook.Worksheets.RemoveAt("Sheet1");
```
Этот метод позволяет напрямую выбрать и удалить указанный лист, что упрощает задачи управления файлами.

#### Шаг 3: Сохраните изменения.
Наконец, сохраните книгу, чтобы сохранить изменения:

```csharp
// Сохраните обновленную рабочую книгу.
using (FileStream fstream = new FileStream(dataDir + "output.out.xls", FileMode.Create))
{
    workbook.Save(fstream);
}
```

### Советы по устранению неполадок
- **Файл не найден:** Убедитесь, что путь к файлу правильный и доступный.
  
- **Несоответствие имени листа:** Еще раз проверьте название листа, учитывая регистр.
