### Руководство по внедрению
Мы разобьем реализацию на три основных раздела: создание группового поля, добавление переключателей и группировка фигур.

#### Создание элемента управления Group Box
Групповое поле служит контейнером для связанных элементов управления. Вот как можно добавить его на лист Excel:

**Шаг 1**: Инициализируйте свою рабочую книгу и откройте первый рабочий лист.
```csharp
using Aspose.Cells;
using Aspose.Cells.Drawing;

string outputDir = "/YOUR_OUTPUT_DIRECTORY";
Workbook excelbook = new Workbook();
Worksheet sheet = excelbook.Worksheets[0];
```

**Шаг 2**: Добавить на рабочий лист групповой блок с указанными размерами.
```csharp
GroupBox box = sheet.Shapes.AddGroupBox(1, 0, 300, 250);
box.Text = "Age Groups";
box.Placement = PlacementType.FreeFloating;
box.Shadow = false;

excelbook.Save(outputDir + "/GroupBoxControl.xls");
```

**Объяснение**: `AddGroupBox` метод размещает групповой блок в указанных индексах строки и столбца с шириной 300 единиц и высотой 250 единиц. Размещение установлено как свободное плавающее, что позволяет независимое перемещение.

#### Добавление радиокнопок
Переключатели полезны для выбора одного варианта из нескольких вариантов в групповом поле.

**Шаг 1**: Создайте переключатели на рабочем листе.
```csharp
RadioButton radio1 = sheet.Shapes.AddRadioButton(3, 0, 30, 110);
radio1.Text = "20-29";
radio1.LinkedCell = "A1"; // Ссылки на ячейку A1 для извлечения данных
radio1.Shadow = true;
radio1.Line.Weight = 4;
radio1.Line.DashStyle = MsoLineDashStyle.Solid;

RadioButton radio2 = sheet.Shapes.AddRadioButton(6, 0, 30, 110);
radio2.Text = "30-39";
radio2.LinkedCell = "A1";

RadioButton radio3 = sheet.Shapes.AddRadioButton(9, 0, 30, 110);
radio3.Text = "40-49";
radio3.LinkedCell = "A1";

excelbook.Save(outputDir + "/RadioButtons123.xls");
```

**Объяснение**: Каждый `AddRadioButton` вызов создает новую кнопку в указанных позициях. `LinkedCell` Свойство привязывает переключатель к ячейке, что позволяет легко извлекать данные.

#### Группировка фигур
Группировка фигур упрощает манипуляции и организацию рабочего листа.
```csharp
Shape[] shapeobjects = new Shape[] { box, radio1, radio2, radio3 };
GroupShape group = sheet.Shapes.Group(shapeobjects);

excelbook.Save(outputDir + "/GroupedShapes.xls");
```

**Объяснение**Используя `sheet.Shapes.Group`, вы можете объединить несколько фигур в одну сущность. Это особенно полезно для поддержания пространственных отношений между элементами управления.
