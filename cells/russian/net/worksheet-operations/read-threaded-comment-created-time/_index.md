# Прочитать время создания связанных комментариев в рабочем листе

## Введение
При работе с файлами Excel управление комментариями может быть важным аспектом совместной работы с данными и обратной связи. Если вы используете Aspose.Cells для .NET, вы найдете его невероятно мощным для обработки различных функций Excel, включая потоковые комментарии. В этом руководстве мы сосредоточимся на том, как читать созданное время потоковых комментариев на рабочем листе. Независимо от того, являетесь ли вы опытным разработчиком или только начинаете, это руководство проведет вас через процесс шаг за шагом.
## Предпосылки
Прежде чем погрузиться в код, давайте убедимся, что у вас есть все необходимое для начала работы:
1. Aspose.Cells для .NET: Убедитесь, что у вас установлена библиотека Aspose.Cells. Вы можете загрузить ее с [Сайт Aspose](https://releases.aspose.com/cells/net/).
2. Visual Studio: рабочая установка Visual Studio или любой другой .NET IDE, в которой вы можете писать и выполнять свой код C#.
3. Базовые знания C#: знакомство с программированием на C# поможет вам лучше понимать фрагменты кода.
4. Файл Excel: Подготовьте файл Excel с несколькими цепочками комментариев. Для этого примера мы будем использовать файл с именем `ThreadedCommentsSample.xlsx`.
Теперь, когда все необходимые условия выполнены, давайте импортируем необходимые пакеты.
## Импортные пакеты
Чтобы начать работу с Aspose.Cells, вам нужно импортировать требуемые пространства имен. Вот как это сделать:
### Импорт пространства имен Aspose.Cells
Откройте свой проект C# в Visual Studio и добавьте следующую директиву using в начало файла кода:
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```
Это пространство имен позволяет получить доступ ко всем классам и методам, предоставляемым библиотекой Aspose.Cells.
Теперь, когда мы подготовили почву, давайте разобьем процесс чтения созданного времени ветвящихся комментариев на управляемые этапы.
## Шаг 1: Определите исходный каталог
Во-первых, вам нужно указать каталог, в котором находится ваш файл Excel. Это важно, поскольку программа должна знать, где искать файл.
```csharp
// Исходный каталог
string sourceDir = "Your Document Directory";
```
Заменять `"Your Document Directory"` с фактическим путем к вашему файлу Excel. Это может быть что-то вроде `"C:\\Documents\\"`.
## Шаг 2: Загрузите рабочую книгу
Далее вы загрузите книгу Excel, содержащую цепочку комментариев. Вот как это сделать:
```csharp
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```
Эта строка кода создает новый `Workbook` объект, загрузив указанный файл Excel. Если файл не найден, будет выдано исключение, поэтому убедитесь, что путь указан правильно.
## Шаг 3: Доступ к рабочему листу
После загрузки рабочей книги следующим шагом будет доступ к конкретному рабочему листу, содержащему комментарии. В нашем случае мы получим доступ к первому рабочему листу:
```csharp
// Доступ к первому рабочему листу
Worksheet worksheet = workbook.Worksheets[0];
```
Эта строка извлекает первый рабочий лист (индекс 0) из рабочей книги. Если ваши комментарии находятся на другом рабочем листе, измените индекс соответствующим образом.
## Шаг 4: Получите цепочку комментариев
Теперь пришло время извлечь цепочку комментариев из определенной ячейки. В этом примере мы получим комментарии из ячейки A1:
```csharp
// Получить древовидные комментарии
ThreadedCommentCollection threadedComments = worksheet.Comments.GetThreadedComments("A1");
```
Эта строка извлекает все связанные комментарии, связанные с ячейкой A1. Если комментариев нет, коллекция будет пустой.
## Шаг 5: Просмотрите комментарии
Теперь, когда все связанные комментарии извлечены, мы можем просмотреть их и отобразить подробную информацию, включая время создания:
```csharp
foreach (ThreadedComment comment in threadedComments)
{
    Console.WriteLine("Comment: " + comment.Notes);
    Console.WriteLine("Author: " + comment.Author.Name);
    Console.WriteLine("Created Time: " + comment.CreatedTime);
}
```
Этот цикл проходит по каждому комментарию в `threadedComments` собирает и распечатывает текст комментария, имя автора и время создания комментария.
## Шаг 6: Подтверждающее сообщение
Наконец, после выполнения логики чтения комментариев всегда полезно предоставить сообщение с подтверждением. Это помогает в отладке и гарантирует, что код был выполнен успешно:
```csharp
Console.WriteLine("ReadThreadedCommentCreatedTime executed successfully.");
```

Поздравляем! Вы успешно научились читать созданное время цепочек комментариев в таблице Excel с помощью Aspose.Cells for .NET. Эта функция может быть невероятно полезна для отслеживания отзывов и совместной работы в документах Excel. С помощью всего нескольких строк кода вы можете извлечь ценную информацию, которая может улучшить ваши процессы анализа данных и отчетности.
