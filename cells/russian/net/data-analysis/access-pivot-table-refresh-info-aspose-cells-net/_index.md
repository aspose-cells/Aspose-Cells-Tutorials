## Руководство по внедрению

### Доступ к информации об обновлении сводной таблицы

#### Обзор

Эта функция позволяет программно получить информацию о том, кто и когда последний раз обновлял сводную таблицу, предоставляя ценную информацию о целостности ваших данных.

#### Настройка вашего проекта
1. **Загрузите рабочую книгу:**
   Загрузите книгу Excel, содержащую целевую сводную таблицу, с помощью `Workbook` сорт.
   ```csharp
   Workbook workbook = new Workbook("sourcePivotTable.xlsx");
   ```
2. **Доступ к рабочему листу и сводной таблице:**
   Откройте рабочий лист, а затем конкретную сводную таблицу внутри него.
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   PivotTable pivotTable = worksheet.PivotTables[0];
   ```
3. **Получить обновленную информацию:**
   Использовать `RefreshedByWho` и `RefreshDate` чтобы получить подробную информацию об обновлении.
   ```csharp
   string refreshByWho = pivotTable.RefreshedByWho;
   DateTime refreshDate = pivotTable.RefreshDate;
   
   Console.WriteLine("Pivot table refreshed by: " + refreshByWho);
   Console.WriteLine("Last refresh date: " + refreshDate);
   ```

#### Объяснение
- **`RefreshedByWho`:** Возвращает имя пользователя, который последним обновил сводную таблицу.
- **`RefreshDate`:** Предоставляет временную метку последнего обновления сводной таблицы.

### Советы по устранению неполадок

- Убедитесь, что путь к файлу Excel указан правильно и доступен для вашего приложения.
- Убедитесь, что указанные индексы рабочего листа и сводной таблицы действительны в вашей рабочей книге.
