## Руководство по внедрению
В этом разделе рассматривается, как изменить макеты сводных таблиц с помощью Aspose.Cells .NET.

### Изменение макета на компактную форму
Компактная форма идеально подходит для быстрых обзоров. Вот как это реализовать:

#### Шаг 1: Загрузите файл Excel
```csharp
// Загрузить существующую рабочую книгу
Workbook workbook = new Workbook("sampleChangingLayoutOfPivotTable.xlsx");
```

#### Шаг 2: Доступ к сводной таблице
```csharp
Worksheet worksheet = workbook.Worksheets[0];
PivotTable pivotTable = worksheet.PivotTables[0];
```

#### Шаг 3: Установите компактную форму и обновите данные
```csharp
// Изменить на компактную форму
pivotTable.ShowInCompactForm();

// Обновите данные, чтобы применить изменения.
pivotTable.RefreshData();
pivotTable.CalculateData();

// Сохраните рабочую книгу
workbook.Save("outputChangingLayoutOfPivotTable_CompactForm.xlsx");
```

### Изменение макета на контурную форму
Форма структуры расширяет вашу сводную таблицу для детального анализа.

#### Шаг 1: Доступ и настройка
```csharp
// Изменить на контурную форму
pivotTable.ShowInOutlineForm();

// Обновите данные, чтобы применить изменения.
pivotTable.RefreshData();
pivotTable.CalculateData();

// Сохраните рабочую книгу
workbook.Save("outputChangingLayoutOfPivotTable_OutlineForm.xlsx");
```

### Изменение макета на табличную форму
Для традиционного табличного представления используйте табличную форму.

#### Шаг 1: Установка и обновление
```csharp
// Изменить на табличную форму
pivotTable.ShowInTabularForm();

// Обновите данные, чтобы применить изменения.
pivotTable.RefreshData();
pivotTable.CalculateData();

// Сохраните рабочую книгу
workbook.Save("outputChangingLayoutOfPivotTable_TabularForm.xlsx");
```

### Советы по устранению неполадок:
- Убедитесь, что путь к файлу Excel указан правильно.
- Убедитесь, что сводные таблицы правильно проиндексированы на вашем листе.
