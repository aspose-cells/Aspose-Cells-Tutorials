## Настройка Aspose.Cells для .NET
Давайте заложим основу для нашей задачи управления сценарием. Начнем с загрузки существующей книги Excel и доступа к ее рабочим листам.

### Загрузка рабочей книги и доступ к рабочему листу
Начните с создания нового экземпляра `Workbook` класс и загрузите нужный вам файл:

```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// Загрузить существующую рабочую книгу
Workbook workbook = new Workbook(sourceDir + "sampleCreateManipulateRemoveScenarios.xlsx");
Worksheet worksheet = workbook.Worksheets[0];
```

### Создание сценария
Загрузив рабочую книгу, вы теперь можете добавлять сценарии и управлять ими:

```csharp
// Добавить новый сценарий
int i = worksheet.Scenarios.Add("MyScenario");

// Доступ к недавно созданному сценарию
Scenario scenario = worksheet.Scenarios[i];

// Добавляем комментарий к нашему сценарию для лучшего понимания
scenario.Comment = "Test scenario is created.";
```

### Управление ячейками входных данных сценария
Чтобы сделать ваши сценарии динамичными, вам необходимо эффективно управлять ячейками ввода:

```csharp
// Получить коллекцию входных ячеек
ScenarioInputCellCollection sic = scenario.InputCells;

// Добавьте ячейку ввода в B4 со значением по умолчанию.
sic.Add(3, 1, "1100000");
```

### Сохранение вашей работы
Не забудьте сохранить изменения в файле:

```csharp
workbook.Save(outputDir + "outputCreateManipulateRemoveScenarios.xlsx");
Console.WriteLine("Scenario management executed successfully.\r\n");
```
