### Настройка параметров отображения сводной таблицы

В этом разделе вы узнаете, как настроить отображение данных в сводных таблицах с помощью Aspose.Cells для .NET.

#### Указание пустых значений ячеек

Чтобы контролировать, отображаются ли пустые ячейки в сводной таблице, используйте `DisplayNullString` свойство:
```csharp
// Доступ к первому рабочему листу и его первой сводной таблице
PivotTable pt = wb.Worksheets[0].PivotTables[0];

// Установите значение true, чтобы отображать пустые строки для пустых ячеек.
pt.DisplayNullString = true;
```

#### Настройка пустых строк

Укажите, какую строку показывать, если ячейка пуста. `NullString`:
```csharp
// Установка пользовательского текста для нулевых значений
pt.NullString = "null";
pt.CalculateData();
```

#### Обновить данные при открытии файла

Управляйте тем, должна ли сводная таблица обновлять данные при открытии файла, используя:
```csharp
pt.RefreshDataOnOpeningFile = false;
```

### Сохранение вашей рабочей книги

Наконец, сохраните вашу книгу с обновленными настройками сводной таблицы:
```csharp
wb.Save("outputSettingPivotTableOption.xlsx");
Console.WriteLine("Pivot table options set successfully.");
```
