## Руководство по внедрению

В этом разделе мы шаг за шагом рассмотрим сортировку данных по цвету ячеек.

### Создание и загрузка рабочей книги

Начните с создания экземпляра `Workbook` класс и загрузка вашего файла Excel:
```csharp
// Создать объект рабочей книги и загрузить файл шаблона
Workbook workbook = new Workbook(sourceDir + "sampleBackGroundFile.xlsx");
```
Этот код инициализирует новую рабочую книгу и загружает данные из существующего файла Excel, расположенного в исходном каталоге.

### Инициализация DataSorter

Далее создайте экземпляр `DataSorter` класс для подготовки к сортировке:
```csharp
// Создать экземпляр объекта сортировщика данных
DataSorter sorter = workbook.DataSorter;
```
The `DataSorter` необходим для определения и выполнения операций сортировки ваших данных.

### Добавление ключа сортировки по цвету ячейки

Укажите, как вы хотите сортировать данные. Здесь мы добавляем ключ на основе цвета ячейки:
```csharp
// Добавить ключ для второго столбца для красного цвета
csorter.AddKey(1, SortOnType.CellColor, SortOrder.Descending, Color.Red);
```
На этом этапе сортировщику сообщается, что необходимо отдать приоритет строкам, в которых ячейки во втором столбце имеют красный фон, и отсортировать их в порядке убывания.

### Выполнение операции сортировки

После настройки ключей выполните сортировку:
```csharp
// Сортировать данные по ключу
sorter.Sort(workbook.Worksheets[0].Cells, CellArea.CreateCellArea("A2", "C6"));
```
Эта команда сортирует строки в пределах заданной области ячеек (от A2 до C6) на основе наших критериев.

### Сохранение отсортированных данных

Наконец, сохраните отсортированную книгу:
```csharp
// Сохраните выходной файл.
workbook.Save(outputDir + "outputsampleBackGroundFile.xlsx");
```
Приведенный выше код сохраняет обработанные данные в новый файл Excel в указанном вами выходном каталоге.
