## Руководство по внедрению

Теперь, когда все настроено, давайте реализуем функцию «Отключить ленту сводной таблицы».

### Обзор отключения ленты сводной таблицы

Отключение ленты сводной таблицы не позволяет пользователям получать доступ к определенным функциям напрямую из пользовательского интерфейса Excel. Это может быть полезно для сценариев, требующих настраиваемых интерфейсов или ограниченных функций.

#### Пошаговая реализация

##### 1. Загрузите рабочую книгу

Сначала загрузите рабочую книгу, содержащую сводные таблицы:

```csharp
// Открыть файл-образец
Workbook wb = new Workbook("samplePivotTableTest.xlsx");
```

##### 2. Доступ к сводной таблице

Получите доступ к конкретной сводной таблице, которую вы хотите изменить. Здесь мы работаем с первой сводной таблицей первого листа.

```csharp
// Получить сводную таблицу с первого листа
PivotTable pt = wb.Worksheets[0].PivotTables[0];
```

##### 3. Отключить ленту сводной таблицы

Установите `EnableWizard` свойство false:

```csharp
// Отключить мастер сводных таблиц
pt.EnableWizard = false;
```

##### 4. Сохраните рабочую книгу.

Сохраните изменения в новом файле:

```csharp
// Вывести измененную рабочую книгу
wb.Save("outputSamplePivotTableTest.xlsx");
```

#### Основные параметры конфигурации

- **`EnableWizard`**Это логическое свойство управляет включением или отключением ленты сводной таблицы.
