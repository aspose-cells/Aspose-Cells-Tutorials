## Руководство по внедрению

### Создание и настройка сводной таблицы (H2)
В этом разделе мы рассмотрим создание и настройку сводной таблицы с помощью Aspose.Cells.

#### Шаг 1: Подготовка данных (H3)
Сначала создайте или откройте книгу в формате Excel и добавьте данные, необходимые для сводной таблицы:

```csharp
// Создать новый объект Workbook
Workbook workbook = new Workbook();

// Доступ к первому рабочему листу в рабочей книге
Worksheet sheet = workbook.Worksheets[0];

// Получить коллекцию ячеек рабочего листа
Cells cells = sheet.Cells;

// Заполните рабочий лист образцами данных о продажах в спорте.
cells["A1"].PutValue("Sport");
cells["B1"].PutValue("Quarter");
cells["C1"].PutValue("Sales");

cells["A2"].PutValue("Golf");    cells["B2"].PutValue("Qtr3");  cells["C2"].PutValue(1500);
cells["A3"].PutValue("Golf");    cells["B3"].PutValue("Qtr4");  cells["C3"].PutValue(2000);
cells["A4"].PutValue("Tennis");  cells["B4"].PutValue("Qtr3");  cells["C4"].PutValue(600);
// Продолжайте для других записей...
```

#### Шаг 2: Добавление сводной таблицы (H3)
Затем добавьте на свой рабочий лист сводную таблицу:

```csharp
PivotTableCollection pivotTables = sheet.PivotTables;

// Добавьте новую сводную таблицу в ячейку «E3» на основе диапазона данных «A1:C8».
int index = pivotTables.Add("=A1:C8", "E3", "PivotTable2");

// Доступ к недавно созданному экземпляру PivotTable
PivotTable pivotTable = pivotTables[index];

// Настройте сводную таблицу
pivotTable.RowGrand = false; // Скрыть общие итоги для строк

// Добавляйте поля в разные области сводной таблицы
pivotTable.AddFieldToArea(PivotFieldType.Row, 0);   // Спортивное поле в районе Роу
pivotTable.AddFieldToArea(PivotFieldType.Column, 1); // Поле четверти в область столбца
pivotTable.AddFieldToArea(PivotFieldType.Data, 2);   // Поле продаж в область данных

// Рассчитать данные для сводной таблицы
pivotTable.CalculateData();
```

#### Шаг 3: Сохранение как ODS-файла (H3)
Наконец, сохраните вашу рабочую книгу в формате ODS:

```csharp
string outputDir = "your/output/directory/";
workbook.Save(outputDir + "PivotTableSaveInODS_out.ods");
Console.WriteLine("PivotTableSaveInODS executed successfully.");
```
