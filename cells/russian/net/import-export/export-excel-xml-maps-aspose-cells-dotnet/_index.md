## Руководство по внедрению

### Функция: создание экземпляра рабочей книги и экспорт XML-карты
#### Обзор
Эта функция включает загрузку книги Excel и экспорт связанных с ней карт XML. Это особенно полезно для миграции данных или при работе с системами, которым требуются определенные форматы XML, полученные из таблиц Excel.

#### Шаг 1: Создание экземпляра объекта Workbook
Начните с создания `Workbook` объект. Это загрузит указанный вами файл Excel, сделав его готовым к дальнейшей обработке.
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Загрузить книгу из указанного каталога
Workbook workbook = new Workbook(SourceDir + "/sampleExportXmlMapFromWorkbook.xlsx");
```
**Объяснение:** The `Workbook` конструктор принимает путь к файлу в качестве аргумента, загружая документ Excel в память.

#### Шаг 2: Итерация по картам XML
Далее, пройдитесь по всем доступным картам XML в вашей рабочей книге. Это позволит вам получить доступ и экспортировать каждую карту по отдельности.
```csharp
// Доступ и итерация по всем XmlMaps в рабочей книге
global for (int i = 0; i < workbook.Worksheets.XmlMaps.Count; i++) {
    // Получить каждую XmlMap по ее индексу
    XmlMap map = workbook.Worksheets.XmlMaps[i];
    
    // Экспортируйте XML-данные из этого XmlMap в отдельный файл.
    workbook.ExportXml(map.Name, outputDir + "/" + map.Name + ".xml");
}
```
**Объяснение:** `workbook.Worksheets.XmlMaps` предоставляет коллекцию всех XML-карт. Каждая карта может быть доступна по ее индексу и экспортирована с помощью `ExportXml` метод.

### Советы по устранению неполадок
- **Проблемы с путем к файлу:** Убедитесь, что пути к исходному и выходному каталогам указаны правильно.
