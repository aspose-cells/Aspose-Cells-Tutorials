## Руководство по внедрению

### Обзор
Мы рассмотрим, как загрузить книгу Excel и экспортировать ее в формат HTML, сохранив при этом комментарии.

### Пошаговые инструкции

#### Загрузите рабочую тетрадь
Начните с загрузки исходного файла Excel:

```csharp
// Исходный каталог
string sourceDir = RunExamples.Get_SourceDirectory();

// Загрузить образец файла Excel
Workbook wb = new Workbook(sourceDir + "sampleExportCommentsHTML.xlsx");
```
Здесь, `RunExamples.Get_SourceDirectory()` — это служебная функция для получения пути к исходным файлам.

#### Настройте параметры сохранения HTML
Чтобы экспортировать комментарии, установите `IsExportComments` свойство:

```csharp
// Экспортировать комментарии — установите свойство IsExportComments в значение true
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.IsExportComments = true;
```
Такая конфигурация гарантирует, что все комментарии в файле Excel будут включены в вывод HTML.

#### Сохранить как HTML
Наконец, сохраните книгу как HTML-файл:

```csharp
// Выходной каталог
string outputDir = RunExamples.Get_OutputDirectory();

// Сохраните файл Excel в HTML
wb.Save(outputDir + "outputExportCommentsHTML.html", opts);

Console.WriteLine("ExportCommentsWhileSavingExcelFileToHtml executed successfully.\r\n");
```

### Советы по устранению неполадок
- Убедитесь, что пути к исходным каталогам указаны правильно.
- Убедитесь, что предоставлены все необходимые разрешения на чтение и запись файлов.
