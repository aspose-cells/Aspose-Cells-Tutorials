## Руководство по внедрению

Давайте разберем реализацию импорта пользовательских объектов в таблицу Excel.

### Шаг 1: Определите свой пользовательский объект
Начните с создания класса, представляющего вашу структуру данных. Для этого примера мы будем использовать `Person` класс со свойствами для `Name` и `Age`.
```csharp
class Person
{
    int _age;
    string _name;

    public int Age 
    { 
        get => _age; 
        set => _age = value; 
    }
    
    public string Name 
    {
        get => _name;  
        set => _name = value; 
    }

    public Person(string name, int age)
    {
        Age = age;
        Name = name;
    }
}
```
### Шаг 2: Подготовьте данные
Создайте список пользовательских объектов, которые вы хотите импортировать в Excel.
```csharp
List<Person> people = new List<Person>
{
    new Person("Mike", 25),
    new Person("Steve", 30),
    new Person("Billy", 35)
};
```
### Шаг 3: Импорт пользовательских объектов
Настройте `ImportTableOptions` чтобы указать, как следует импортировать данные, а затем использовать `ImportCustomObjects` метод.
```csharp
// Создайте новую рабочую книгу и получите первый рабочий лист.
Workbook book = new Workbook();
Worksheet sheet = book.Worksheets[0];

// Настройте параметры импорта
ImportTableOptions options = new ImportTableOptions { InsertRows = true };

// Импортировать только выбранные столбцы («Имя» и «Возраст»)
sheet.Cells.ImportCustomObjects((System.Collections.ICollection)people,
    new string[] { "Name", "Age" }, 
    true, 0, 0, people.Count, true, null, false);

// Автоматически подгонять все столбцы под их содержимое
book.Worksheets[0].AutoFitColumns();
```
### Шаг 4: Сохраните свою рабочую книгу
Наконец, сохраните вашу рабочую книгу в файле Excel.
```csharp
string dataDir = "path/to/your/directory";
book.Save(dataDir + "ImportedCustomObjects.xlsx");
```
