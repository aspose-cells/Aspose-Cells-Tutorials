## Руководство по внедрению

В этом разделе мы покажем вам, как импортировать массив в Excel с помощью Aspose.Cells.

### Создание новой рабочей книги

Начните с создания нового экземпляра `Workbook`:

```csharp
// Создать новую рабочую книгу
Workbook workbook = new Workbook();
```

Это представляет ваш файл Excel. Далее, получите первый рабочий лист из этой рабочей книги:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

### Импорт данных массива

Вот как импортировать массив на рабочий лист:

#### Шаг 1: Определите свой массив

Создайте массив строк с данными, которые вы хотите импортировать:

```csharp
// Создать массив имен
string[] names = new string[] { "Laurence Chen", "Roman Korchagin", "Kyle Huang" };
```

#### Шаг 2: Импорт данных

Использовать `ImportArray` метод для вставки данных в ваш рабочий лист. Этот метод размещает данные вертикально, начиная с указанной строки и столбца:

```csharp
// Импортировать массив на рабочий лист, начиная с ячейки A1 (0, 0)
worksheet.Cells.ImportArray(names, 0, 0, true);
```

- **Параметры**:
  - `names`: Исходный массив.
  - `0`: Начальный индекс строки.
  - `0`: Начальный индекс столбца.
  - `true`: Указывает, следует ли вставлять данные вертикально.

### Сохранение рабочей книги

После импорта данных сохраните книгу как файл Excel:

```csharp
// Определите путь к каталогу для сохранения файла
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Проверьте и создайте каталог, если необходимо.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);

// Сохранить книгу в файл
workbook.Save(dataDir + "DataImport.out.xls");
```
