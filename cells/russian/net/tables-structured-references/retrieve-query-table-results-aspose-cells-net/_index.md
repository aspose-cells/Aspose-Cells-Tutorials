### Получить результаты таблицы запросов

**Обзор:**
Получение результатов таблицы запросов подразумевает доступ к данным, связанным с внешними источниками в рабочей книге Excel. В этом разделе вы узнаете, как получить доступ к этим результатам с помощью C# и Aspose.Cells.

#### Шаг 1: Загрузите рабочую книгу

Сначала загрузите исходный файл Excel в `Workbook` объект.

```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook wb = new Workbook(sourceDir + "sampleQueryTXT.xlsx");
```

*Объяснение*: Приведенный выше код инициализирует рабочую книгу с указанным путем к файлу Excel. Крайне важно убедиться, что файл существует в указанном вами каталоге.

#### Шаг 2: Доступ к результату таблицы запроса

Далее получите доступ к диапазону результатов вашей таблицы запроса.

```csharp
Console.WriteLine(wb.Worksheets[0].QueryTables[0].ResultRange.Address);
```

*Объяснение*: Эта строка извлекает и печатает адрес диапазона результатов из первой таблицы запроса на первом рабочем листе. `Address` Свойство предоставляет диапазон ячеек, в котором хранятся данные, связанные запросом.

#### Советы по устранению неполадок
- **Отсутствующие таблицы запросов**: Убедитесь, что ваш файл Excel содержит таблицы запросов.
- **Проблемы с путями к файлам**: Еще раз проверьте путь к исходному каталогу и имя файла.
