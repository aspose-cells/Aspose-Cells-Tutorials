### Преобразовать таблицу в диапазон с параметрами

Эта функция помогает преобразовать таблицу в книге Excel в обычный диапазон с использованием определенных конфигураций.

#### Обзор

Преобразуя таблицы в диапазоны, вы получаете большую гибкость в манипулировании данными и можете применять различные методы .NET, требующие простых диапазонов. Давайте рассмотрим шаги реализации:

**Загрузите свою рабочую тетрадь:**

Начните с загрузки существующей рабочей книги с помощью Aspose.Cells.

```csharp
using System;
using Aspose.Cells;

string SourceDir = "/path/to/your/source/directory";
string outputDir = "/path/to/your/output/directory";

// Загрузить существующую рабочую книгу
Workbook workbook = new Workbook(SourceDir + "/book1.xlsx");
```

**Настройте параметры преобразования:**

Определите параметры преобразования, используя `TableToRangeOptions` сорт.

```csharp
using Aspose.Cells.Tables;

// Создать экземпляр TableToRangeOptions для настройки
TableToRangeOptions options = new TableToRangeOptions();
options.LastRow = 5; // Настройте, чтобы указать последнюю строку диапазона
```

**Конвертировать и сохранить:**

Выполните преобразование в указанной таблице, затем сохраните книгу.

```csharp
// Преобразовать первую таблицу на рабочем листе в обычный диапазон
workbook.Worksheets[0].ListObjects[0].ConvertToRange(options);

// Сохраните измененную книгу.
workbook.Save(outputDir + "/output.xlsx");
```

**Совет по устранению неполадок:** Если у вас возникли проблемы с путями к каталогам, убедитесь, что они правильно заданы и доступны.
