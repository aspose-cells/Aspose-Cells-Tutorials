## Руководство по внедрению

### Чтение книги Excel
**Обзор:** Эта функция демонстрирует, как загрузить файл Excel и получить доступ к его рабочим листам.

#### Шаг 1: Загрузите рабочую книгу
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(sourceDir + "sampleReadingAndWritingQueryTable.xlsx");
```

#### Шаг 2: Доступ к рабочим листам
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

### Доступ к таблице запросов на рабочем листе
**Обзор:** Узнайте, как получить доступ к определенным таблицам запросов на листе Excel.

#### Шаг 1: Инициализация рабочей книги и рабочего листа
```csharp
Workbook workbook = new Workbook(sourceDir + "sampleReadingAndWritingQueryTable.xlsx");
Worksheet worksheet = workbook.Worksheets[0];
```

#### Шаг 2: Доступ к таблице запросов
```csharp
QueryTable qt = worksheet.QueryTables[0];
```

### Чтение свойств таблицы запросов
**Обзор:** Эта функция демонстрирует такие свойства чтения, как `AdjustColumnWidth` и `PreserveFormatting`.

```csharp
bool adjustColumnWidth = qt.AdjustColumnWidth;
bool preserveFormatting = qt.PreserveFormatting;

// Пояснение: AdjustColumnWidth автоматически изменяет размер столбцов, PreserveFormatting сохраняет исходный формат.
```

### Изменение свойств таблицы запросов
**Обзор:** Узнайте, как изменить свойства таблицы запросов.

#### Шаг 1: Установите сохранение форматирования
```csharp
qt.PreserveFormatting = true;
```

### Сохранение книги Excel
**Обзор:** Эта функция показывает, как сохранить изменения, внесенные в книгу Excel.

#### Шаг 1: Сохраните рабочую книгу
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outputReadingAndWritingQueryTable.xlsx");
```
