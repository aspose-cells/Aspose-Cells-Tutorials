### Создание и настройка объектов списка

Эта функция позволяет добавлять структурированные таблицы данных с такими функциями, как сортировка, фильтрация и подсчет итогов.

#### Шаг 1: Настройте свою рабочую книгу и рабочий лист

```csharp
// Путь, по которому находятся ваши входные файлы.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Загрузите существующую книгу или создайте новую
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

#### Шаг 2: Доступ и добавление объектов списка

```csharp
// Доступ к первому рабочему листу из рабочей книги
Worksheet sheet = workbook.Worksheets[0];

// Получить коллекцию объектов списка на этом листе
Aspose.Cells.Tables.ListObjectCollection listObjects = sheet.ListObjects;
```

#### Шаг 3: Создайте новый объект списка

Определите диапазон и добавьте заголовки в новую таблицу.

```csharp
// Добавить объект списка с указанными размерами, начиная со строки 1, столбца 1
listObjects.Add(1, 1, 7, 5, true); // Включает заголовки, устанавливая последний параметр в значение «true»
```

#### Шаг 4: Настройка расчета итогов

Включите и настройте итоги для столбцов списка.

```csharp
// Включить отображение общей строки
listObjects[0].ShowTotals = true;

// Установите метод расчета «Сумма» для пятого столбца (индекс 4)
listObjects[0].ListColumns[4].TotalsCalculation = Aspose.Cells.Tables.TotalsCalculation.Sum;
```

#### Шаг 5: Сохраните свою рабочую книгу

Убедитесь, что ваши изменения сохранены в файле Excel.

```csharp
// Сохраните книгу по указанному пути.
workbook.Save(dataDir + "output.xls");
```
