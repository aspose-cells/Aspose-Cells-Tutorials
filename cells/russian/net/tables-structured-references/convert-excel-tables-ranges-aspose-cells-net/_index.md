## Руководство по внедрению

Давайте разобьем процесс конвертации на управляемые этапы.

### Шаг 1: Загрузите файл Excel

Для начала загрузите существующий файл Excel, содержащий таблицу, которую вы хотите преобразовать:

```csharp
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
Workbook workbook = new Workbook(dataDir + "book1.xlsx");
```

**Объяснение:** Здесь, `dataDir` содержит путь к каталогу ваших документов. `Workbook` объект инициализируется с помощью существующего файла.

### Шаг 2: Преобразование таблицы в диапазон

Далее найдите и преобразуйте первую таблицу на вашем рабочем листе:

```csharp
workbook.Worksheets[0].ListObjects[0].ConvertToRange();
```

**Объяснение:** Выбираем первый рабочий лист (`Worksheets[0]`) и первый объект списка (или таблица) внутри него (`ListObjects[0]`). `ConvertToRange()` метод выполняет преобразование.

### Шаг 3: Сохраните изменения.

Наконец, сохраните вашу книгу с преобразованным диапазоном:

```csharp
workbook.Save(dataDir + "output.xlsx");
```

**Объяснение:** На этом этапе все изменения записываются в новый файл с именем `output.xlsx`.
