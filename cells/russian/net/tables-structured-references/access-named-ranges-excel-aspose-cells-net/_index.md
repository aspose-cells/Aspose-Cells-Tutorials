### Доступ к именованному диапазону в Excel
**Обзор:** Мы загрузим файл Excel и извлечем указанный именованный диапазон с именем «MyRangeTwo».
1. **Загрузите рабочую тетрадь**
   Начните с загрузки вашей книги Excel с помощью `Workbook`:
   ```csharp
   string sourceDir = RunExamples.Get_SourceDirectory();
   Workbook workbook = new Workbook(sourceDir + "sampleAccessSpecificNamedRange.xlsx");
   ```
2. **Получить именованный диапазон**
   Использовать `GetRangeByName()` для доступа к именованному диапазону:
   ```csharp
   Range range = workbook.Worksheets.GetRangeByName("MyRangeTwo");

   if (range != null)
       Console.WriteLine("Named Range: " + range.RefersTo);
   ```
3. **Подтверждение вывода**
   Подтвердите успешное выполнение с помощью сообщения на консоли:
   ```csharp
   Console.WriteLine("AccessSpecificNamedRange executed successfully.");
   ```

**Параметры и назначение:**
- `GetRangeByName(string name)`: Извлекает именованный диапазон по его идентификатору, возвращая `null` если не найдено.
- `RefersTo`: Предоставляет строковое представление ссылки на диапазон в Excel.
