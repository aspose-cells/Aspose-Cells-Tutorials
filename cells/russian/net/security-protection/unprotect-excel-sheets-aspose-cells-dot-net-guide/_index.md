## Руководство по внедрению

Чтобы получить доступ к рабочему листу и снять с него защиту, выполните следующие действия:

### Доступ к рабочему листу и снятие защиты

#### Шаг 1: Создание экземпляра объекта Workbook

Начните с создания `Workbook` объект из вашего существующего файла:

```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Загрузить книгу с диска
Workbook workbook = new Workbook(sourceDir + "/book1.xls");
```

**Объяснение:** Эта строка инициализирует рабочую книгу с указанным вами файлом Excel.

#### Шаг 2: Доступ к рабочему листу

Извлеките рабочий лист, с которого вы хотите снять защиту:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

**Объяснение:** The `Worksheets[0]` метод обращается к первому рабочему листу в рабочей книге.

#### Шаг 3: Снимите защиту с рабочего листа.

Снимите защиту с рабочего листа:

```csharp
// Снять защиту листа без указания пароля
worksheet.Unprotect();
```

**Объяснение:** Это действие снимает любую существующую защиту, предоставляя полный доступ к рабочему листу.

#### Шаг 4: Сохраните рабочую книгу

Сохраните изменения обратно на диск:

```csharp
workbook.Save(outputDir + "/output.xls", SaveFormat.Excel97To2003);
```

**Объяснение:** The `save` Метод записывает обновленную книгу в формате Excel 97-2003.

### Загрузка и сохранение рабочей книги

Загрузите рабочую книгу, измените ее и сохраните изменения:

#### Шаг 1: Загрузите существующую рабочую книгу

```csharp
// Перезагрузите рабочую книгу для демонстрационных целей.
tWorkbook = new Workbook(sourceDir + "/book1.xls");
```

**Объяснение:** Это гарантирует, что вы работаете с последней версией файла.

#### Шаг 2: Снимите защиту и сохраните

Повторите снятие защиты и сохранение, как описано ранее, чтобы изменения вступили в силу.
