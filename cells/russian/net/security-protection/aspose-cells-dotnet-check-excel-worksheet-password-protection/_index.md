## Руководство по внедрению

### Проверка защиты рабочего листа паролем

Эта функция позволяет определить, защищен ли пароль каким-либо листом в файле Excel.

#### Шаг 1: Загрузите свою рабочую книгу
Загрузите рабочую книгу, защиту которой вы хотите проверить:
```csharp
// Исходный каталог
string sourceDir = RunExamples.Get_SourceDirectory();

// Создайте экземпляр Workbook и загрузите электронную таблицу.
var book = new Workbook(sourceDir + "sampleCheckIfPasswordProtected.xlsx");
```

#### Шаг 2: Доступ к рабочему листу
Откройте рабочий лист, который вы хотите проверить на предмет защиты:
```csharp
// Доступ к защищенному рабочему листу
var sheet = book.Worksheets[0];
```

#### Шаг 3: Проверьте наличие защиты паролем
Определите, защищен ли рабочий лист паролем, используя `IsProtectedWithPassword`:
```csharp
if (sheet.Protection.IsProtectedWithPassword)
{
    Console.WriteLine("Worksheet is Password Protected");
}
else
{
    Console.WriteLine("Worksheet is Not Password Protected");
}

Console.WriteLine("CheckIfPasswordProtected executed successfully.");
```

**Объяснение:**
- **Параметры**: `Workbook` и `Worksheets` классы управляют содержимым файла Excel.
- **Возвращаемые значения**: Логическое значение, указывающее состояние защиты паролем.

### Советы по устранению неполадок
- Убедитесь, что путь к исходному каталогу указан правильно, чтобы избежать ошибок загрузки.
- Убедитесь, что индекс рабочего листа, к которому вы обращаетесь, существует в вашей рабочей книге.
