# Сохранить файл Excel в формате 97-2003

## Введение
Создание и управление файлами Excel программным способом может стать переломным моментом, особенно для предприятий, которые в значительной степени зависят от обработки данных. Aspose.Cells — один из замечательных инструментов, доступных разработчикам .NET. Он универсален и эффективен, помогая вам оптимизировать рабочие процессы и автоматизировать задачи с помощью электронных таблиц. Если вы хотите сохранить файлы Excel в классическом формате 97-2003, вы попали по адресу! Давайте углубимся.
## Предпосылки
Прежде чем мы углубимся в детали, вам нужно будет отметить в своем списке несколько предварительных условий:
1. Базовые знания .NET: знакомство с C# или VB.NET будет чрезвычайно полезным.
2. Aspose.Cells для .NET: Убедитесь, что в вашем проекте установлена библиотека Aspose.Cells. Если вы еще этого не сделали, вы можете [скачать здесь](https://releases.aspose.com/cells/net/).
3. Visual Studio: Среда разработки, такая как Visual Studio или любая совместимая с .NET IDE, облегчит кодирование и отладку.
4. Менеджер пакетов NuGet: для максимально простой установки Aspose.Cells в вашем проекте. 
Как только вы выполните все эти предварительные условия, мы готовы приступить к работе!
## Импортные пакеты
Чтобы начать работу с Aspose.Cells, вам сначала нужно импортировать необходимые пространства имен в ваш проект. Это даст вам доступ к классам и методам, необходимым для работы с файлами Excel. Вот как это сделать:
### Откройте свой проект
Откройте свой проект .NET в Visual Studio.
### Установить Aspose.Cells
Если вы еще не установили пакет Aspose.Cells, вы можете сделать это через NuGet. 
1. Перейдите в Инструменты -> Диспетчер пакетов NuGet -> Управление пакетами NuGet для решения.
2. Найдите Aspose.Cells.
3. Нажмите «Установить».
### Импорт пространства имен
В верхней части файла C# добавьте следующую строку:
```csharp
using System.IO;
using Aspose.Cells;
```
Теперь вы готовы приступить к программированию!
В этом разделе мы проведем вас через процесс сохранения файла Excel в формате 97-2003 (.xls) с помощью Aspose.Cells. Давайте разобьем его на простые шаги.
## Шаг 1: Настройте каталог документов
Сначала самое главное! Вам нужно будет указать каталог, в котором будет сохранен ваш файл Excel.
```csharp
string dataDir = "Your Document Directory";
```
- `"Your Document Directory"`: Замените эту строку-заполнитель на фактический путь, по которому вы хотите сохранить файл Excel. Это может быть что-то вроде `"C:\\ExcelFiles\\"`.
## Шаг 2: Создание нового объекта рабочей книги
Далее давайте создадим новый экземпляр `Workbook` класс. Вот где происходит вся магия!
```csharp
Workbook workbook = new Workbook();
```
- `Workbook`: Этот класс представляет файл Excel, с которым вы работаете. Создавая его экземпляр, вы по сути создаете новую пустую книгу.
## Шаг 3: Сохраните рабочую книгу в формате 97-2003
Вот тот момент, которого вы ждали! Пришло время сохранить вашу рабочую книгу. Есть два способа сделать это.
### Простое сохранение
Используйте следующий код, чтобы сохранить файл непосредственно по указанному пути.
```csharp
workbook.Save(dataDir + "output.xls");
```
### Сохранить в указанном формате
Вы также можете явно указать формат сохранения:
```csharp
workbook.Save(dataDir + "output.xls", SaveFormat.Excel97To2003);
```
- `output.xls`: Это имя файла, который вы сохраняете. Вы можете переименовать его по своему усмотрению.
- `SaveFormat.Excel97To2003`: Это гарантирует, что ваш файл будет сохранен в формате Excel 97-2003.
