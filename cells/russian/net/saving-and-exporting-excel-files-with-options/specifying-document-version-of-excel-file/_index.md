# Указание версии документа файла Excel программным способом в .NET

## Введение
Aspose.Cells для .NET — это мощная библиотека, которая позволяет разработчикам с легкостью программно манипулировать файлами Excel. Независимо от того, хотите ли вы создать файлы Excel с нуля или изменить существующие, Aspose.Cells предлагает комплексный API для достижения ваших целей. Одной из таких функций является указание свойств документа, таких как версия, автор или заголовок. В этом руководстве вы узнаете, как указать версию документа файла Excel программным способом с помощью Aspose.Cells для .NET.
## Предпосылки
Прежде чем углубляться в детали, давайте убедимся, что у вас есть все необходимое для выполнения этого руководства:
1. Aspose.Cells для .NET: Вы можете загрузить последнюю версию[здесь](https://releases.aspose.com/cells/net/) . Если вы еще не приобрели лицензию, вы можете выбрать[временная лицензия](https://purchase.aspose.com/temporary-license/) для изучения особенностей.
2. Среда разработки .NET: можно использовать Visual Studio или любую совместимую с .NET IDE.
3. Базовые знания C#: понимание программирования на C# облегчит понимание материала.
## Импортные пакеты
Прежде чем начать кодирование, вам нужно импортировать необходимые пространства имен из библиотеки Aspose.Cells. Это даст вам доступ к классам и методам, необходимым для работы с файлами Excel.
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```
Эти два пространства имен будут необходимы для взаимодействия с рабочей книгой и ее встроенными свойствами документа.
Теперь давайте разберем процесс указания свойств документа в файле Excel, включая версию, название и автора.
## Шаг 1: Инициализация объекта Workbook
 Первый шаг — создать новый экземпляр`Workbook` объект. Этот объект представляет собой весь файл Excel, с которым вы будете работать.
```csharp
Workbook wb = new Workbook();
```
 The`Workbook`класс предоставляет представление файла Excel. Создавая его экземпляр, мы создаем пустую книгу Excel, которой можем манипулировать.
## Шаг 2: Доступ к встроенным свойствам документа
 Aspose.Cells предлагает встроенные свойства документа, которые включают такие поля, как заголовок, автор и версия документа. Вы можете получить доступ к этим свойствам через`BuiltInDocumentProperties`коллекция.
```csharp
Aspose.Cells.Properties.BuiltInDocumentPropertyCollection bdpc = wb.BuiltInDocumentProperties;
```
 The`BuiltInDocumentPropertyCollection` Класс предоставляет доступ к набору встроенных свойств документа, таких как заголовок, автор и другие метаданные, обычно связанные с документом.
## Шаг 3: Задайте заголовок документа Excel.
Далее мы установим заголовок документа Excel. Эти метаданные помогут в идентификации и управлении файлом в дальнейшем.
```csharp
bdpc.Title = "Aspose File Format APIs";
```
Настройка заголовка важна для организации документа. Эти метаданные можно увидеть в свойствах файла, и они могут использоваться внешними системами для более эффективной каталогизации или идентификации документа.
## Шаг 4: Укажите автора
Также можно указать автора документа, чтобы отразить, кто создал или изменил файл.
```csharp
bdpc.Author = "Aspose APIs Developers";
```
Этот шаг помогает идентифицировать создателя документа, предоставляя дополнительные метаданные для управления документами или сценариев совместной работы.
## Шаг 5: Укажите версию документа
Одним из наиболее важных свойств, которые мы рассматриваем в этом руководстве, является версия документа. Этот шаг позволяет указать версию документа, что полезно при работе в средах, требующих контроля версий.
```csharp
bdpc.DocumentVersion = "Aspose.Cells Version - 18.3";
```
Установка версии документа обеспечивает ясность относительно того, какая версия документа или библиотеки использовалась для создания файла. Это особенно важно в средах, где необходимо отслеживать версии файлов или совместимость с различными версиями библиотек.
## Шаг 6: Сохраните файл Excel.
 Наконец, вы можете сохранить файл Excel со всеми свойствами, которые вы только что установили. Aspose.Cells позволяет вам сохранять файл в различных форматах, но для этого примера мы будем придерживаться`.xlsx` формат.
```csharp
wb.Save("outputSpecifyDocumentVersionOfExcelFile.xlsx", SaveFormat.Xlsx);
```
 The`Save` Метод используется для сохранения файла в указанном вами каталоге. Здесь мы сохраняем его как файл Excel в`.xlsx`формат. При необходимости Aspose.Cells также поддерживает такие форматы, как`.xls`, `.csv` , и`.pdf`, обеспечивая гибкость в зависимости от потребностей вашего проекта.

Aspose.Cells — чрезвычайно гибкий и мощный инструмент, позволяющий программно манипулировать файлами Excel, что делает его отличным приобретением для любого разработчика .NET, работающего с электронными таблицами.
