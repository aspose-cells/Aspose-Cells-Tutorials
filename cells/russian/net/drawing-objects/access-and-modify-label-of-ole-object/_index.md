# Изменение меток объектов OLE в Excel с помощью Aspose.Cells

В этом примере показано, как получить доступ и изменить метку объекта Ole в файле Excel с помощью библиотеки Aspose.Cells для .NET. Следуя предоставленным фрагментам кода, разработчики могут легко манипулировать объектами Ole, обновлять их метки и эффективно сохранять изменения. Эта возможность необходима для динамического улучшения файлов Excel в приложениях.

## Инструкция по установке
1. Установите Aspose.Cells для .NET: используйте диспетчер пакетов NuGet для установки библиотеки Aspose.Cells в свой проект.
   ```
   Install-Package Aspose.Cells
   ```
2. Добавьте директивы Using: включите необходимые пространства имен в свой файл кода.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Добавьте пример кода: скопируйте и вставьте предоставленный код в свой проект, обязательно указав путь к каталогу документов.
4. Запустите код: выполните свой проект, чтобы получить доступ и изменить метку объекта Ole в вашем файле Excel. Проверьте консоль на наличие выходных сообщений, подтверждающих успешное выполнение процесса.
