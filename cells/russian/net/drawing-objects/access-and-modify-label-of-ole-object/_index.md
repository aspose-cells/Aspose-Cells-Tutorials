## Инструкции по установке
1. Установка Aspose.Cells для .NET: Используйте NuGet Package Manager для установки библиотеки Aspose.Cells в вашем проекте.
   ```
   Install-Package Aspose.Cells
   ```
2. Добавить инструкции по использованию: включите необходимые пространства имен в файл кода.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Добавьте код образца: Копируйте и вставьте указанный код в проект, чтобы указать путь каталога документов.
4. Запустить код: выполните свой проект для доступа и изменения этикетки объекта Ole в файле Excel. Проверьте консоль на выходное сообщение, подтверждающее успешное выполнение процесса.
