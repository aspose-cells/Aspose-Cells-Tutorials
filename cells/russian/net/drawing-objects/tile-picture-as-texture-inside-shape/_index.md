## Инструкции по установке
1. Убедитесь, что у вас установлена среда .NET на вашем компьютере.
2. Скачать и установить библиотеку Aspose.Cells с официального сайта Aspose или с помощью NuGet Package Manager:
   ```
   Install-Package Aspose.Cells
   ```
3. Create a new C# project or open your existing .NET project.
4. Добавьте в свой проект ссылку на библиотеку Aspose.Cells.
5. Копируйте приведенный примерный код в свой проект, заменяя по мере необходимости пути входного/выходного каталога.
6. Составьте и запущите свой проект, чтобы увидеть наполнение текстуры плитки, примененное к формам в файле Excel.
