## Инструкция по установке
1. Убедитесь, что на вашем компьютере настроена среда .NET.
2. Загрузите и установите библиотеку Aspose.Cells с официального сайта Aspose или с помощью диспетчера пакетов NuGet:
   ```
   Install-Package Aspose.Cells
   ```
3. Создайте новый проект C# или откройте существующий проект .NET.
4. Добавьте ссылку на библиотеку Aspose.Cells в свой проект.
5. Скопируйте предоставленный пример кода в свой проект, при необходимости заменив пути к каталогам ввода/вывода.
6. Скомпилируйте и запустите свой проект, чтобы увидеть текстурную заливку плитки, примененную к фигурам в файле Excel.
