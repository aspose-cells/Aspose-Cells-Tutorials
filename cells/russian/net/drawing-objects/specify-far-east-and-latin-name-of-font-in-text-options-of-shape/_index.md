# Создать рабочий стол Excel с помощью дальневосточных и латинских шрифтов

Этот пример демонстрирует, как использовать Aspose.Cells для .NET для создания рабочей книги Excel, добавления текстового ящика в рабочий лист и установки как Дальневосточных, так и латинских названий шрифтов для текста в этом текстовом ящике.

## Инструкции по установке
1. Установка Aspose.Cells для .NET: 
   * Используйте менеджер пакетов NuGet в проекте Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   * Или скачать DLL с [Назначить сайт](https://www.aspose.com/downloads/cells/net)\- Я не знаю .

2. Добавить ссылки: убедитесь, что ваш проект содержит ссылки на библиотеку Aspose.Cells.

3. Настройка каталога выхода: обновление `выпускDir`переменная в коде с желаемым выходном пути, где будет сохранен файл Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
