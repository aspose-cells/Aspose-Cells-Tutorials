# Создать книгу Excel с дальневосточными и латинскими шрифтами

В этом примере показано, как использовать Aspose.Cells для .NET для создания книги Excel, добавления текстового поля на лист и установки имен шрифтов Far East и Latin для текста в этом текстовом поле. Код демонстрирует инициализацию рабочего пространства, управление элементами листа и сохранение книги с определенным форматированием шрифта, адаптированным под различные языковые требования.

## Инструкция по установке
1. Установите Aspose.Cells для .NET: 
   - Используйте диспетчер пакетов NuGet в своем проекте Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   - Или загрузите DLL с сайта [Сайт Aspose](https://www.aspose.com/downloads/cells/net).

2. Добавьте ссылки: убедитесь, что ваш проект ссылается на библиотеку Aspose.Cells.

3. Настройте выходной каталог: обновите `outputDir` переменная в коде с желаемым выходным путем, по которому будет сохранен файл Excel.

4. Запустите пример: скопируйте предоставленный код C# в свой проект, скомпилируйте и запустите его. Вы должны увидеть подтверждение вывода, и файл Excel с настроенным текстовым полем будет создан в указанном каталоге.
