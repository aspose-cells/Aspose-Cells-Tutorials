## Руководство по внедрению

Теперь мы рассмотрим создание нескольких файлов Excel с помощью Aspose.Cells для .NET.

### Создание нескольких файлов Excel

Эта функция позволяет эффективно создавать несколько рабочих книг Excel:

#### Шаг 1: Инициализация рабочей книги

Начните с создания нового экземпляра `Workbook`. Этот объект представляет собой целый файл Excel.

```csharp
Workbook workbook = new Workbook();
```

#### Шаг 2: Настройка рабочих листов

Каждая рабочая книга может содержать несколько рабочих листов. Здесь мы добавим пять листов в каждую рабочую книгу:

```csharp
// Удалить рабочий лист по умолчанию.
workbook.Worksheets.RemoveAt(0);

for (int sheetIndex = 0; sheetIndex < 5; sheetIndex++)
{
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = sheetIndex.ToString();
    
    for (int row = 0; row < 150; row++)
    {
        for (int col = 0; col < 50; col++)
        {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

#### Шаг 3: Сохраните рабочую книгу

Наконец, сохраните каждую рабочую книгу как файл Excel в желаемом выходном каталоге:

```csharp
string filePath = System.IO.Path.Combine(outputDir, $"{fileIndex}_out.xls");
workbook.Save(filePath);
```
