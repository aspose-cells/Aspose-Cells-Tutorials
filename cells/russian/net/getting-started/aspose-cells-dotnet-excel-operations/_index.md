## Руководство по внедрению

### Функция 1: Создание рабочей книги и вычисление формул

Эта функция позволяет вам загружать файл Excel и вычислять любые содержащиеся в нем формулы.

#### Шаг 1: Создание экземпляра объекта Workbook

Сначала создайте `Workbook` объект из указанного вами пути к файлу Excel:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleFindingCellsWithStringOrNumber.xlsx");
```

#### Шаг 2: Вычислите формулы в загруженной рабочей книге

Позвоните `CalculateFormula` метод обработки всех формул в рабочей книге:

```csharp
workbook.CalculateFormula();
```

### Функция 2: Найти ячейку с целым или двойным значением

Эта функция показывает, как находить ячейки, содержащие целые или двойные значения.

#### Шаг 1: Доступ к коллекции клеток

Получите ячейки с первого листа вашей рабочей книги:

```csharp
Cells cells = workbook.Worksheets[0].Cells;
```

#### Шаг 2: Настройка FindOptions и поиск ячейки

Создавать `FindOptions` чтобы указать критерии поиска, затем найдите ячейку с указанным значением (например, целое число 224):

```csharp
FindOptions opts = new FindOptions();
opts.LookInType = LookInType.Values;
opts.LookAtType = LookAtType.EntireContent;

Cell cell1 = cells.Find(224, null, opts);
if (cell1 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell1.Name);
}
else
{
    Console.WriteLine("Record not found");
}
```

### Функция 3: Найти ячейку с точным строковым значением

Найдите ячейку, которая точно соответствует заданной строке.

#### Шаг 1: Настройте FindOptions для точного соответствия

С использованием `LookAtType` установлен на `EntireContent`найдите точное строковое значение:

```csharp
opts.LookInType = LookInType.Values;
opts.LookAtType = LookAtType.EntireContent;

Aspose.Cells.Cell cell2 = cells.Find("Items E", null, opts);
if (cell2 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell2.Name);
}
else
{
    Console.WriteLine("Record not found");
}
```

### Функция 4: Найти ячейку со строкой, содержащей определенную подстроку

Найдите ячейки, содержащие определенную подстроку в своем содержимом.

#### Шаг 1: Настройка FindOptions для поиска подстрок

Набор `LookAtType` к `Contains` и найдите подстроку «Данные»:

```csharp
opts.LookInType = LookInType.Values;
opts.LookAtType = LookAtType.Contains;

Cell cell3 = cells.Find("Data", null, opts);
if (cell3 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell3.Name);
}
else
{
    Console.WriteLine("Record not found");
}
```
