## Руководство по внедрению

### Функция 1: Инициализация рабочей книги и проверка данных для одной ячейки

#### Обзор

Научитесь инициализировать рабочую книгу и проверять данные в определенных ячейках с помощью Aspose.Cells.

**Шаг 1: Импорт необходимых библиотек**

Убедитесь, что вы импортировали необходимые библиотеки Aspose.Cells:

```java
import com.aspose.cells.*;
```

**Шаг 2: Инициализация рабочей книги**

Загрузите файл Excel в объект рабочей книги.

```csharp
Workbook workbook = new Workbook("YOUR_SOURCE_DIRECTORY/sampleDataValidationRules.xlsx");
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("C1");
```

**Шаг 3: Проверка данных ячейки**

Проверьте, соответствуют ли данные в определенной ячейке критериям проверки.

```csharp
// Значение 3 выходит за пределы допустимого диапазона (от 10 до 20)
cell.putValue(3);
System.out.println("Is 3 a Valid Value for this Cell: " + cell.getValidationValue());

// Значение 15 находится в пределах допустимого диапазона (от 10 до 20)
cell.putValue(15);
System.out.println("Is 15 a Valid Value for this Cell: " + cell.getValidationValue());

// Значение 30 выходит за пределы допустимого диапазона (от 10 до 20)
cell.putValue(30);
System.out.println("Is 30 a Valid Value for this Cell: " + cell.getValidationValue());
```

### Функция 2: Проверка данных для другой ячейки с другим диапазоном правил

#### Обзор

Примените другие правила проверки данных к другой ячейке.

**Шаг 1: Инициализация рабочей книги и целевой ячейки**

Загрузите рабочую книгу и выберите новую целевую ячейку:

```csharp
Workbook workbook2 = new Workbook("YOUR_SOURCE_DIRECTORY/sampleDataValidationRules.xlsx");
Worksheet worksheet2 = workbook2.getWorksheets().get(0);
Cell cell2 = worksheet2.getCells().get("D1");
```

**Шаг 2: Проверка данных**

Введите значение и проверьте, соответствует ли оно критериям проверки.

```csharp
// Введите большое число 12345678901 в ячейку D1, которое должно пройти проверку из-за своего диапазона (от 1 до 999999999999).
cell2.putValue(12345678901);
System.out.println("Is 12345678901 a Valid Value for this Cell: " + cell2.getValidationValue());
```

**Советы по устранению неполадок:**
- Убедитесь, что в вашем файле Excel правильно установлены правила проверки.
- Еще раз проверьте диапазон и критерии, указанные в ваших проверках.
