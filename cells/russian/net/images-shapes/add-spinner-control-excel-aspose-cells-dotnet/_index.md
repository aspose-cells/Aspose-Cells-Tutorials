## Руководство по внедрению

### Добавление текста и стилизация ячеек

Перед добавлением элемента управления «спиннер» подготовьте ячейки с метками.

#### Шаг 1: Введите метки и стили

**Обзор**: Настройте таблицу Excel с метками руководства пользователя для элемента управления счетчиком.

```csharp
Cells cells = worksheet.Cells;

// Добавьте метку в ячейку A1.
cells["A1"].PutValue("Select Value:");
Style style = cells["A1"].GetStyle();
style.Font.Color = Color.Red;
style.Font.IsBold = true;
cells["A1"].SetStyle(style);

// Подготовьте связанную ячейку (A2) для управления прядильщиком.
cells["A2"].PutValue(0);
style = cells["A2"].GetStyle();
style.ForegroundColor = Color.Black;
style.Pattern = BackgroundType.Solid;
style.Font.Color = Color.White;
style.Font.IsBold = true;
cells["A2"].SetStyle(style);
```

#### Шаг 2: Добавьте элемент управления Spinner

**Обзор**: Интегрируйте элемент управления «Счетчик» в свой рабочий лист, связав его с определенными данными.

```csharp
// Добавление элемента управления «Счетчик», связанного с ячейкой A2.
Aspose.Cells.Drawing.Spinner spinner = excelbook.Worksheets[0].Shapes.AddSpinner(1, 0, 1, 0, 20, 18);
spinner.Placement = PlacementType.FreeFloating;
spinner.LinkedCell = "A2";
spinner.Max = 10;
spinner.Min = 0;
spinner.IncrementalChange = 2;
spinner.Shadow = true;
```

### Объяснение

- **Размещение**Счетчик установлен на `FreeFloating`, что обеспечивает гибкое позиционирование.
- **Связанная ячейка**: Связывает счетчик с ячейкой A2, гарантируя, что изменения в счетчике будут отражаться в этой ячейке.
- **Диапазон и приращение**: Настраивает диапазон счетчика от 0 до 10 с шагом 2.
