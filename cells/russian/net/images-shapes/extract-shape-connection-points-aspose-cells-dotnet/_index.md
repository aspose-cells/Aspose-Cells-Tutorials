## Руководство по внедрению
### Извлечение точек соединения форм
В этом разделе вы узнаете, как извлекать точки соединения из фигур с помощью Aspose.Cells для .NET.
#### Шаг 1: Создайте новую рабочую книгу и получите доступ к рабочему листу
Начните с создания экземпляра `Workbook` объект, представляющий файл Excel. Затем откройте первый рабочий лист, где находится ваша форма.
```csharp
// Создайте новую рабочую книгу.
Workbook workbook = new Workbook();

// Получите первый рабочий лист в книге.
Worksheet worksheet = workbook.Worksheets[0];
```
#### Шаг 2: Добавьте и получите доступ к фигуре
Добавьте текстовое поле (или любую другую фигуру) в коллекцию, затем извлеките его из коллекции фигур.
```csharp
// Добавьте новое текстовое поле в коллекцию.
int textboxIndex = worksheet.TextBoxes.Add(2, 1, 160, 200);

// Получите доступ к текстовому полю, которое также является объектом-фигурой из коллекции фигур.
Shape shape = workbook.Worksheets[0].Shapes[textboxIndex];
```
#### Шаг 3: Извлечение точек подключения
Используйте `GetConnectionPoints` метод для извлечения всех точек соединения фигуры.
```csharp
// Получите все точки соединения в этой форме
var connectionPoints = shape.GetConnectionPoints();

// Отобразить все точки формы
foreach (var pt in connectionPoints)
{
    System.Console.WriteLine(string.Format("X = {0}, Y = {1}", pt[0], pt[1]));
}
```
