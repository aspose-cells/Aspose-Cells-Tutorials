### Добавление элемента управления «Кнопка» на рабочий лист
#### Обзор
Добавление интерактивных элементов, таких как кнопки, может сделать ваши листы Excel более удобными для пользователя. В этом разделе вы узнаете, как создать новую кнопку на листе Excel.

#### Пошаговая реализация
1. **Создать или открыть рабочую книгу**
   Начните с инициализации `Workbook` объект, представляющий файл Excel.
    
   ```csharp
   // Инициализируйте новый объект Workbook
   Workbook workbook = new Workbook();
   ```

2. **Доступ к рабочему листу**
   Откройте первый рабочий лист, на котором вы разместите кнопку.
    
   ```csharp
   // Получить первый рабочий лист в рабочей тетради
   Worksheet sheet = workbook.Worksheets[0];
   ```

3. **Добавить кнопку управления**
   Используйте `Shapes.AddButton` метод вставки новой кнопки на рабочий лист.
    
   ```csharp
   // Добавить новую кнопку на рабочий лист
   Aspose.Cells.Drawing.Button button = sheet.Shapes.AddButton(2, 0, 2, 0, 28, 80);
   ```

4. **Настроить свойства кнопки**
   Задайте различные свойства кнопки, такие как текст, шрифт и гиперссылку.
    
   ```csharp
   // Настроить свойства кнопки
   button.Text = "Aspose";
   button.Placement = PlacementType.FreeFloating;
   button.Font.Name = "Tahoma";
   button.Font.IsBold = true;
   button.Font.Color = Color.Blue;
   button.AddHyperlink("http://www.aspose.com/");
   ```

5. **Сохранить рабочую книгу**
   После настройки сохраните книгу, чтобы завершить изменения.
    
   ```csharp
   // Сохраните файл под новым именем.
   string dataDir = "path/to/save/directory/";
   workbook.Save(dataDir + "book1.out.xls");
   ```

### Советы по устранению неполадок
- **Файл не сохраняется**: Убедитесь, что путь к каталогу существует или создан правильно.
- **Проблемы со шрифтами**: Убедитесь, что нужный вам шрифт установлен в вашей системе.
