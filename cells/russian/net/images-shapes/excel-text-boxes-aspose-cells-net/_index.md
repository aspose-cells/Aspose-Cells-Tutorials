### Создание и настройка текстового поля (H2)

#### Обзор
Создание и настройка текстового поля позволяет добавлять интерактивные элементы в листы Excel. Мы настроим свойства шрифта, типы размещения и другие настройки.

##### Шаг 1: Инициализация рабочей книги и рабочего листа
```java
// Импортируйте необходимые классы Aspose.Cells.
import com.aspose.cells.*;

String SourceDir = "YOUR_SOURCE_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте новый экземпляр рабочей книги.
Workbook workbook = new Workbook();

// Откройте первый рабочий лист.
Worksheet worksheet = workbook.getWorksheets().get(0);
```

##### Шаг 2: Добавьте и настройте TextBox
```java
// Добавьте текстовое поле в коллекцию по указанным координатам.
int textboxIndex = worksheet.getTextBoxes().add(2, 1, 160, 200);

// Откройте только что созданное текстовое поле.
TextBox textbox0 = (TextBox)worksheet.getTextBoxes().get(textboxIndex);

// Задайте текстовое содержимое со стилями и гиперссылками.
textbox0.setText("ASPOSE______The .NET & JAVA Component Publisher!");
textbox0.setPlacement(PlacementType.FREE_FLOATING);
textbox0.getFont().setColor(Color.getBlue());
textbox0.getFont().setBold(true);
textbox0.getFont().setSize(14);
textbox0.getFont().setItalic(true);

// Добавьте гиперссылку на сайт Aspose.
textbox0.addHyperlink("http://www.aspose.com/");

// Настройте форматы линий и заливок для лучшей видимости.
LineFormat lineformat = textbox0.getLine();
lineformat.setWeight(6);
lineformat.setDashStyle(MsoLineDashStyle.SQUARE_DOT);
FillFormat fillformat = textbox0.getFill();

// Сохраните книгу в выходном каталоге.
workbook.save(outputDir + "book1.out.xls");
```

#### Основные параметры конфигурации
- **Тип размещения**: FREE_FLOATING позволяет текстовым полям свободно перемещаться, в то время как MOVE_AND_SIZE подстраивается под ячейки.
- **Настройка шрифта**: Измените цвет, размер и стили для лучшей читабельности.
- **Добавление гиперссылки**: Повысьте интерактивность, добавив ссылки на внешние ресурсы.

### Добавление еще одного текстового поля (H2)

#### Обзор
Добавьте дополнительные текстовые поля, чтобы предоставить больше информации или расширить функциональные возможности вашего рабочего листа.

##### Шаг 1: Добавьте новое текстовое поле
```java
// Создайте еще одно текстовое поле с другими координатами.
int textboxIndex = worksheet.getTextBoxes().add(15, 4, 85, 120);

// Извлеките недавно добавленный объект текстового поля.
TextBox textbox1 = (TextBox)worksheet.getTextBoxes().get(textboxIndex);
```

##### Шаг 2: Настройте размещение и сохраните
```java
// Задайте текстовое содержимое и измените его размер вместе с ячейками.
textbox1.setText("This is another simple text box");
textbox1.setPlacement(PlacementType.MOVE_AND_SIZE);

// Сохраните изменения в новом файле.
workbook.save(outputDir + "book2.out.xls");
```

#### Советы по устранению неполадок
