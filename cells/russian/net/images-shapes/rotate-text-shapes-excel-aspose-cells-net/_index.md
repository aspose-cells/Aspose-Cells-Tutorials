## Руководство по внедрению
В этом разделе мы покажем вам процесс поворота текста внутри фигуры на листе Excel.

### Шаг 1: Загрузите файл Excel
Начните с загрузки образца файла Excel:
```csharp
Workbook wb = new Workbook("sampleRotateTextWithShapeInsideWorksheet.xlsx");
```
На этом шаге инициализируется объект рабочей книги, представляющий ваш документ Excel.

### Шаг 2: Доступ к рабочему листу и его изменение
Откройте нужный рабочий лист, на котором вы хотите манипулировать фигурами и текстом:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell b4 = ws.Cells["B4"];
b4.PutValue("Text is not rotating with shape because RotateTextWithShape is false.");
```

### Шаг 3: Настройка свойств фигуры
Откройте первую фигуру на рабочем листе, чтобы изменить ее текстовые свойства:
```csharp
Shape sh = ws.Shapes[0];
Aspose.Cells.Drawing.Texts.ShapeTextAlignment shapeTextAlignment = sh.TextBody.TextAlignment;
shapeTextAlignment.RotateTextWithShape = false; // Установите значение true, если вы хотите, чтобы текст вращался вместе с формой.
```
Эта конфигурация определяет, будет ли текст вращаться вместе с фигурой.

### Шаг 4: Сохраните изменения.
После внесения изменений сохраните книгу:
```csharp
wb.Save("outputRotateTextWithShapeInsideWorksheet.xlsx");
Console.WriteLine("Rotation executed successfully.");
```
