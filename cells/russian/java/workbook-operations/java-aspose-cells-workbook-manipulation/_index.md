Чтобы инициализировать библиотеку, просто начните с создания новой `Workbook` объект. Это ваша точка входа для работы с файлами Excel:

## Руководство по внедрению

В этом руководстве каждая функция разбита на простые шаги, что позволяет вам понять их и эффективно реализовать.

### Инициализация нового объекта рабочей книги
**Обзор**: Этот шаг включает в себя создание `Workbook` экземпляр с использованием существующего файла Excel. Это ваша отправная точка для любых дальнейших манипуляций.
1. **Импортируйте необходимые классы**
   ```java
   import com.aspose.cells.Workbook;
   ```
2. **Создайте экземпляр рабочей книги**
   - The `Workbook` конструктор может загружать файлы из различных форматов, таких как `.xls`.
   ```java
   String dataDir = "YOUR_DATA_DIRECTORY";
   Workbook workbook = new Workbook(dataDir + "/BookStyles.xls");
   ```
   - Эта строка кода создает новый `Workbook` объект на основе существующего файла Excel.

### Доступ к рабочему листу и ячейкам
**Обзор**: Здесь мы демонстрируем, как получить доступ к определенным рабочим листам и их ячейкам для манипулирования.
1. **Импорт дополнительных классов**
   ```java
   import com.aspose.cells.Worksheet;
   import com.aspose.cells.Cells;
   ```
2. **Извлечь первый рабочий лист и его ячейки**
   - Доступ к первому рабочему листу осуществляется с помощью `getWorksheets().get(0)`.
   ```java
   Worksheet worksheet = workbook.getWorksheets().get(0);
   Cells cells = worksheet.getCells();
   ```
   - Это позволит извлечь все ячейки из выбранного рабочего листа для дальнейших операций.

### Разгруппировать строки
**Обзор**: эта функция позволяет удалить группировку из указанных строк на листе.
1. **Доступ к необходимым классам**
   ```java
   import com.aspose.cells.Cells;
   ```
2. **Разгруппировать строки на рабочем листе**
   - Использовать `ungroupRows(int firstRow, int totalRows)` разгруппировать.
   ```java
   Cells cells = workbook.getWorksheets().get(0).getCells();
   cells.ungroupRows(0, 5);
   ```
   - Эта команда удаляет группировку из строк с индексами от 0 до 5.

### Разгруппировать столбцы
**Обзор**: Подобно строкам, с помощью этой функции можно также разгруппировать столбцы.
1. **Доступ к необходимым классам**
   ```java
   import com.aspose.cells.Cells;
   ```
2. **Разгруппировать столбцы на рабочем листе**
   - Использовать `ungroupColumns(int firstColumn, int totalColumns)` для этой задачи.
   ```java
   Cells cells = workbook.getWorksheets().get(0).getCells();
   cells.ungroupColumns(0, 2);
   ```
   - Это приведет к разгруппировке столбцов с индексом от 0 до 2.

### Сохранить рабочую книгу
**Обзор**: После внесения изменений в файл Excel вам необходимо сохранить его правильно.
1. **Импортировать требуемый класс**
   ```java
   import com.aspose.cells.Workbook;
   ```
2. **Сохраните измененную рабочую книгу**
   - Использовать `workbook.save(String outputPath)` для экономии.
   ```java
   workbook.save("YOUR_OUTPUT_DIRECTORY/SummaryRowRight_out.xls");
   ```
   - Это сохранит ваши изменения в формате Excel 2003.
