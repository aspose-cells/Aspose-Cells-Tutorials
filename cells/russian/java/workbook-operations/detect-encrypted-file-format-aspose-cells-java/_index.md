### Определение формата файла (H2)

#### Обзор

Используя `FileFormatUtil` класс в Aspose.Cells, вы можете определить формат зашифрованного файла, указав правильный пароль. Эта функциональность жизненно важна при работе с различными типами файлов, надежно хранящихся с помощью шифрования.

#### Пошаговая реализация (подзаголовки H3)

1. **Подготовьте свою среду:**

   Убедитесь, что ваш проект включает необходимые зависимости, как описано ранее.

2. **Настройте каталог и путь к файлу:**

   Определите путь к каталогу, где находятся ваши зашифрованные файлы.

   ```java
   String dataDir = "path_to_your_directory/";
   String filename = dataDir + "encryptedBook1.out.tmp";
   ```

3. **Определить формат файла:**

   Использовать `FileFormatUtil.detectFileFormat` определить формат файла, указав путь к файлу и пароль.

   ```java
   FileFormatInfo fileFormatInfo = FileFormatUtil.detectFileFormat(filename, "1234");
   ```

   - **Параметры:** 
     - `filename`: Путь к зашифрованному файлу.
     - `"1234"`: Пароль для расшифровки информации о формате файла.

   - **Возвращаемое значение:** А `FileFormatInfo` объект, содержащий сведения об обнаруженном формате файла.

4. **Определить тип формата файла:**

   Оцените возвращаемый тип формата файла, используя условные операторы:

   ```java
   if (fileFormatInfo.getFileFormatType() == FileFormatType.EXCEL_97_TO_2003) {
       System.out.println("File Format: EXCEL_97_TO_2003");
   } else if (fileFormatInfo.getFileFormatType() == FileFormatType.PPTX) {
       System.out.println("File Format: PPTX");
   } else if (fileFormatInfo.getFileFormatType() == FileFormatType.DOCX) {
       System.out.println("File Format: DOCX");
   }
   ```

#### Советы по устранению неполадок

- **Распространенные проблемы:** 
  - Неправильный путь к файлу или пароль могут привести к ошибкам.
  - Убедитесь, что библиотека Aspose.Cells правильно подключена и обновлена.
