## Настройка Aspose.Cells для Java

### Создание и сохранение рабочей книги в формате PDF

#### Обзор
Эта функция демонстрирует, как создать книгу Excel, задать значения ячеек, настроить параметры сохранения PDF-файла с уровнями соответствия и, наконец, сохранить его как файл PDF. 

##### Шаг 1: Создание нового экземпляра рабочей книги

Начните с создания нового экземпляра `Workbook` сорт:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
// Создать новый экземпляр Workbook
Workbook workbook = new Workbook();
Cells cells = workbook.getWorksheets().get(0).getCells();

// Установить значение в ячейке A12
cells.get("A12").setValue("Test PDF");
```

**Объяснение:** Мы инициализируем `Workbook` объект, представляющий файл Excel. `Cells` Коллекция позволяет нам получать доступ к отдельным ячейкам и изменять их.

##### Шаг 2: Настройте параметры сохранения PDF-файла

Далее настройте параметры сохранения, чтобы указать соответствие определенным стандартам PDF/A:

```java
// Настройте параметры сохранения PDF-файла с учетом уровня соответствия
PdfSaveOptions pdfOptions = new PdfSaveOptions();
pdfOptions.setCompliance(PdfCompliance.PDF_A_1_B);
```

**Объяснение:** `PdfSaveOptions` позволяет вам определить, как рабочая книга сохраняется в формате PDF. Настройка соответствия гарантирует, что ваш вывод будет соответствовать таким стандартам, как PDF/A, что повышает долговечность и совместимость документа.

##### Шаг 3: Сохраните рабочую книгу в формате PDF.

Наконец, сохраните книгу, используя настроенные параметры:

```java
// Сохраните книгу как файл PDF
workbook.save(dataDir + "ACToPdf_out.pdf", pdfOptions);
```

**Объяснение:** The `save` метод записывает вашу книгу на диск в указанном формате. Здесь он использует наш ранее определенный `pdfOptions`.

### Настройка соответствия рабочей книги для преобразования в PDF

Эта функция фокусируется на установке уровней соответствия в пределах `PdfSaveOptions`, что имеет решающее значение для обеспечения целостности документа.

#### Обзор
Установка определенного уровня соответствия гарантирует, что ваш PDF-файл будет соответствовать отраслевым стандартам, таким как PDF/A-1b, широко используемым для долгосрочного хранения электронных документов.

##### Шаг 1: Настройте уровень соответствия

Чтобы установить желаемый уровень соответствия:

```java
PdfSaveOptions pdfOptions = new PdfSaveOptions();
pdfOptions.setCompliance(PdfCompliance.PDF_A_1_B);
```

**Объяснение:** The `setCompliance` метод в `PdfSaveOptions` указывает, какой стандарт PDF/A следует соблюдать. Варианты включают PDF/A-1a, PDF/A-1b и другие.
