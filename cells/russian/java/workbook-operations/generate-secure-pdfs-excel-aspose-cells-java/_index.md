Чтобы включить Aspose.Cells в ваш проект с использованием Maven, добавьте следующую зависимость в ваш `pom.xml`:


### Руководство по внедрению
#### Функция: создание безопасных PDF-документов
Эта функция включает преобразование файла Excel в документ PDF с улучшенными настройками безопасности. Давайте рассмотрим процесс шаг за шагом.

##### Шаг 1: Откройте файл Excel.
Сначала загрузите книгу Excel с помощью Aspose.Cells. Убедитесь, что вы указали правильный путь к входному файлу:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "input.xlsx");
```

Здесь, `Workbook` представляет собой документ Excel, который вы хотите преобразовать.

##### Шаг 2: Настройте параметры безопасности PDF-файла
Далее настройте параметры безопасности с помощью `PdfSaveOptions`. Этот класс позволяет указывать различные атрибуты защиты:

```java
// Создайте экземпляр PdfSaveOptions для управления атрибутами безопасности PDF-файла.
PdfSaveOptions saveOption = new PdfSaveOptions();

// Настройте параметры безопасности для PDF-документа.
saveOption.setSecurityOptions(new PdfSecurityOptions());
```

##### Шаг 3: Определите пароли и разрешения
Установите пароли пользователя и владельца, а также ограничения прав доступа:

```java
// Определите пароль пользователя, необходимый для открытия PDF-файла.
saveOption.getSecurityOptions().setUserPassword("user");

// Определите пароль владельца для полных прав доступа.
saveOption.getSecurityOptions().setOwnerPassword("owner");

// Отключить извлечение содержимого из документа.
saveOption.getSecurityOptions().setExtractContentPermission(false);

// Ограничьте разрешения на печать.
saveOption.getSecurityOptions().setPrintPermission(false);
```

- **Пароль пользователя**: Требуется для открытия и просмотра PDF-файла.
- **Пароль владельца**: Предоставляет полный доступ, позволяя изменять настройки безопасности.

##### Шаг 4: Сохранить как защищенный PDF-файл
Наконец, сохраните свою рабочую книгу как защищенный PDF-документ:

```java
// Определите путь к выходному каталогу для сохранения защищенного PDF-файла.
String outDir = "YOUR_OUTPUT_DIRECTORY";

// Сохраните книгу Excel как защищенный PDF-файл в указанном месте вывода.
workbook.save(outDir + "SecurePDFDocuments_out.pdf", saveOption);
```
