## Руководство по внедрению

Для ясности мы разобьем каждую функцию на выполнимые шаги. 

### Функция 1: Загрузка файла Excel

**Обзор**

Эта функция позволяет загрузить существующую книгу Excel с помощью Aspose.Cells, подготовив ее к дальнейшей обработке или анализу.

#### Шаг 1: Создание нового объекта рабочей книги

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook wb = new Workbook(dataDir + "/sampleInlineCharts.xlsx");
```

**Объяснение:**  
- **`Workbook` Сорт**: Представляет файл Excel. Загружает книгу из указанного вами пути в память.
- **Параметры**: Конструктор принимает строковый аргумент, указывающий местоположение файла.

### Функция 2: Установка области печати на рабочем листе

**Обзор**

Здесь мы определяем конкретный диапазон ячеек на рабочем листе, который следует учитывать при выполнении задач печати или экспорта.

#### Шаг 1: Загрузите рабочую книгу

```java
Workbook wb = new Workbook(dataDir + "/sampleInlineCharts.xlsx");
```

#### Шаг 2: Доступ к первому рабочему листу

```java
Worksheet ws = wb.getWorksheets().get(0);
```

**Объяснение:**  
- **`getWorksheets()`**: Возвращает коллекцию всех рабочих листов в рабочей книге.
- **Метод доступа**: Использовать `get(index)` для доступа к определенным листам по индексу.

#### Шаг 3: Установка области печати

```java
ws.getPageSetup().setPrintArea("D2:M20");
```

**Объяснение:**  
- **`setPrintArea(String range)`**: Указывает диапазон ячеек для операций печати/экспорта.
- **Параметр диапазона**: Строка, указывающая, какие ячейки следует включить, например, `"D2:M20"`.

### Функция 3: Экспорт области печати в HTML

**Обзор**

Эта функция экспортирует в HTML-файл только определенную область печати рабочего листа.

#### Шаг 1: Загрузка и настройка области печати (как описано ранее)

```java
Workbook wb = new Workbook(dataDir + "/sampleInlineCharts.xlsx");
Worksheet ws = wb.getWorksheets().get(0);
ws.getPageSetup().setPrintArea("D2:M20");
```

#### Шаг 2: Инициализация HtmlSaveOptions

```java
HtmlSaveOptions options = new HtmlSaveOptions();
options.setExportPrintAreaOnly(true);
```

**Объяснение:**  
- **`HtmlSaveOptions`**: Настраивает способ сохранения файлов Excel в формате HTML.
- **`setExportPrintAreaOnly(boolean value)`**: Если установлено значение `true`, экспортирует только указанную область печати.

#### Шаг 3: Сохранить как HTML

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
wb.save(outDir + "/outputInlineCharts.html", options);
```

**Объяснение:**  
- **`save(String filePath, HtmlSaveOptions options)`**: Сохраняет книгу в соответствии с указанными параметрами.
- **Выходной каталог**: Убедитесь, что выходной каталог настроен правильно.
