## Настройка Aspose.Cells для Java

Чтобы интегрировать Aspose.Cells в свой проект, выполните следующие действия:

### Установка Maven
Добавьте эту зависимость в свой `pom.xml` файл для включения Aspose.Cells для Java:
```xml
<dependency>
  <groupId>com.aspose</groupId>
  <artifactId>aspose-cells</artifactId>
  <version>25.3</version>
</dependency>
```

### Установка Gradle
Для проектов, использующих Gradle, добавьте следующую строку в свой файл `build.gradle`:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Руководство по внедрению

В этом разделе мы рассмотрим, как отключить экспорт сценариев фрейма и свойств документа с помощью Aspose.Cells для Java.

### Отключение экспорта сценариев фрейма и свойств документа
Эта функция позволяет контролировать вывод HTML, предотвращая включение сценариев фреймов и свойств документа.

#### Шаг 1: Загрузите существующую рабочую книгу
Загрузите вашу книгу Excel в `Workbook` объект:
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook w = new Workbook(dataDir + "Sample1.xlsx");
```

#### Шаг 2: Установите параметр «Отключить экспорт сценариев фрейма и свойств документа»
Чтобы отключить экспорт сценариев фрейма, используйте соответствующий метод или класс, предоставляемый Aspose.Cells:
```java
// Пример использования гипотетического IStreamProvider в демонстрационных целях.
IStreamProvider options = new ImplementingIStreamProvider();
options.setExportFrameScriptsAndProperties(false);
w.saveOptions(options);
```
*Примечание: этот шаг предполагает наличие специальных методов или классов для обработки этих настроек, что типично для таких API.*

#### Шаг 3: Сохранить как HTML
Наконец, сохраните вашу рабочую книгу как HTML-файл:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
w.save(outDir + "DisableExporting_out.html");
```

Загрузка рабочей книги для обработки проста:

#### Откройте нужную рабочую книгу
Загрузите книгу, используя ее путь:
```java
Workbook w = new Workbook(dataDir + "Sample1.xlsx");
```
