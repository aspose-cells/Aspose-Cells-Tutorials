## Руководство по внедрению
Мы разберем реализацию на три основные функции: преобразование HTML-строк в Excel, автоматическое подгонка строк и столбцов и сохранение книги в формате XLSX.

### Преобразовать HTML-строку в рабочую книгу
Эта функция позволяет преобразовать строку HTML, содержащую вложенные теги, в структурированную книгу Excel. Вот как:

**1. Подготовьте HTML-строку**
Начните с определения вашего HTML-контента в Java. Например:
```java
String export_html = "<html><body>...</body></html>";  // Ваш HTML здесь
```

**2. Преобразовать HTML-строку в рабочую книгу**
Загрузите ваш HTML в Aspose.Cells `Workbook` объект:
```java
import com.aspose.cells.HtmlLoadOptions;
import java.io.ByteArrayInputStream;

public class SupportthelayoutofDIVtags {
    public static void main(String[] args) throws Exception {
        byte[] bts = export_html.getBytes();
        ByteArrayInputStream bis = new ByteArrayInputStream(bts);

        HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.HTML);
        loadOptions.setSupportDivTag(true);  // Включить поддержку тегов div

        Workbook wb = new Workbook(bis, loadOptions);
    }
}
```
- **`HtmlLoadOptions`**Этот класс предоставляет параметры для управления загрузкой HTML-контента в рабочую книгу.
- **`setSupportDivTag(true)`**: Позволяет обрабатывать `<div>` элементы, имеющие решающее значение для вложенных структур.

### Автоматическая подгонка строк и столбцов
Чтобы обеспечить отображение всех данных без ручной корректировки:
```java
public class AutoFitRowsAndColumns {
    public static void main(String[] args) throws Exception {
        Workbook wb = new Workbook("YOUR_INPUT_FILE_PATH");
        Worksheet ws = wb.getWorksheets().get(0);

        ws.autoFitRows();
        ws.autoFitColumns();
    }
}
```
- **`autoFitRows()`**: Изменяет высоту строк в соответствии с их содержимым.
- **`autoFitColumns()`**: Регулирует ширину столбцов для размещения данных.

### Сохранить книгу как XLSX
Наконец, сохраните вашу книгу в формате Excel:
```java

```
- **`SaveFormat.XLSX`**: Указывает формат файла для сохранения.
