### Функция 1: Извлечение информации о версии
**Обзор**: Извлечь и отобразить версию Aspose.Cells для Java. Это может быть полезно для ведения журнала или обеспечения совместимости.

#### Пошаговая реализация:
**Получить версию**
```java
import com.aspose.cells.*;

public class VersionInfo {
    public static void main(String[] args) throws Exception {
        // Получить и распечатать информацию о версии
        String version = CellsHelper.getVersion();
        System.out.println("Aspose.Cells for Java Version: " + version);
    }
}
```
**Объяснение**: 
- `CellsHelper.getVersion()` извлекает текущую версию библиотеки.
- Эта функция проста, но важна для отладки и проверки совместимости.

### Функция 2: Создание рабочей книги и стилизация ячеек
**Обзор**: Узнайте, как создать рабочую книгу, получить доступ к рабочим листам, изменить содержимое ячеек и применить стили, например изменить цвет шрифта.

#### Пошаговая реализация:
**Создать рабочую книгу и получить доступ к рабочему листу**
```java
import com.aspose.cells.*;

public class WorkbookAndCellStyling {
    public static void main(String[] args) throws Exception {
        // Создать экземпляр объекта Workbook
        Workbook wb = new Workbook();
        
        // Доступ к первому рабочему листу в рабочей книге
        Worksheet ws = wb.getWorksheets().get(0);
```
**Изменить содержимое и стиль ячейки**
```java
        // Извлечь ячейку B5 из рабочего листа
        Cell cell = ws.getCells().get("B5");
        
        // Установите значение «Это какой-то текст» в ячейке B5.
        cell.putValue("This is some text.");
        
        // Получить стиль ячейки, установить цвет шрифта на красный
        Style st = cell.getStyle();
        st.getFont().setColor(Color.getRed());
        
        // Применить стилизованные настройки обратно к ячейке
        cell.setStyle(st);
    }
}
```
**Объяснение**: 
- `Workbook` и `Worksheet` объекты используются для управления файлами Excel.
- Стилизация ячеек достигается с помощью `Style` класс, позволяющий настраивать такие параметры, как цвет шрифта.

### Функция 3: Экспорт CSS-файла рабочего листа отдельно в HTML
**Обзор**: Экспортируйте лист Excel как HTML-файл с разделенными стилями (CSS). Эта функция улучшает визуальное представление ваших данных на веб-платформах.

#### Пошаговая реализация:
**Создать рабочую книгу и стилизовать ячейку**
```java
import com.aspose.cells.*;

public class ExportWorksheetCSSSeparatelyInHTML {
    public static void main(String[] args) throws Exception {
        // Создать объект Workbook
        Workbook wb = new Workbook();
        
        // Доступ к первому рабочему листу в рабочей книге
        Worksheet ws = wb.getWorksheets().get(0);
        
        // Откройте ячейку B5 и введите в нее значение.
        Cell cell = ws.getCells().get("B5");
        cell.putValue("This is some text.");
        
        // Установить стиль ячейки - цвет шрифта красный
        Style st = cell.getStyle();
        st.getFont().setColor(Color.getRed());
        
        // Применить стилизованные настройки обратно к ячейке
        cell.setStyle(st);
```
**Экспортировать как HTML с отдельным CSS**
```java
        // Укажите параметры сохранения HTML с CSS, экспортированным отдельно
        HtmlSaveOptions opts = new HtmlSaveOptions();
        opts.setExportWorksheetCSSSeparately(true);
        
        // Сохраните книгу как HTML-файл с указанными параметрами.
        String outDir = "YOUR_OUTPUT_DIRECTORY";
        wb.save(outDir + "/outputExportWorksheetCSSSeparately.html", opts);
    }
}
```
**Объяснение**: 
- `HtmlSaveOptions` позволяет настраивать способ сохранения файлов Excel в формате HTML.
- Параметр `setExportWorksheetCSSSeparately(true)` обеспечивает отдельный экспорт CSS для лучшего контроля стиля.
