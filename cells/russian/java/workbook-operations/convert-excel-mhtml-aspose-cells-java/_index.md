### Настройка Aspose.Cells для Java
Чтобы использовать Aspose.Cells в своем проекте, выполните следующие действия:

Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml

```

### Руководство по внедрению
#### Загрузка файла Excel
Чтобы загрузить файл Excel, выполните следующие действия:
1. **Настройте свой каталог данных**: Определите путь, по которому хранятся ваши файлы Excel.
   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // Замените на фактический путь к каталогу данных.
   ```
2. **Создание экземпляра объекта Workbook**: Этот объект представляет вашу книгу Excel.
   ```java
   String filePath = dataDir + "Book1.xlsx"; // Путь к файлу Excel
   Workbook wb = new Workbook(filePath); // Загружает файл Excel
   ```
3. **Зачем использовать `Workbook`?** The `Workbook` класс необходим, поскольку он инкапсулирует все листы и их данные, позволяя легко манипулировать ими.

#### Преобразование файла Excel в формат MHTML
Теперь, когда мы загрузили наш файл Excel, давайте преобразуем его в MHTML:
1. **Настроить выходной каталог**: Определите, где вы хотите сохранить преобразованный файл.
   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY"; // Замените на фактический путь к выходному каталогу.
   ```
2. **Укажите параметры сохранения HTML**: Использовать `HtmlSaveOptions` для настройки формата преобразования.
   ```java
   HtmlSaveOptions sv = new HtmlSaveOptions(SaveFormat.M_HTML); // MHTML — это формат веб-архива.
   ```
3. **Выполнить преобразование**: Сохраните вашу рабочую книгу в желаемом формате.
   ```java
   wb.save(outDir + "/CToMHTMLFiles_out.mht", sv);
   ```
4. **Почему `SaveFormat.M_HTML`?** Эта опция гарантирует, что ваш файл Excel будет сохранен в формате MHTML, подходящем для веб-просмотра и архивирования.
