## Руководство по внедрению

### Создать и заполнить рабочую книгу

#### Обзор
Создание рабочей книги с образцами данных включает инициализацию объекта рабочей книги, добавление рабочих листов и заполнение ячеек текстом.

**Шаг 1: Создание рабочей книги**

```java
import com.aspose.cells.Workbook;

String outputDir = "YOUR_OUTPUT_DIRECTORY";
Workbook workbook = new Workbook();
```
*Объяснение:* Это инициализирует пустой экземпляр рабочей книги. `outputDir` переменная должна указывать на желаемый каталог для сохранения файлов.

**Шаг 2: Добавьте и заполните рабочие листы**

- **Добавить образец текста на первый рабочий лист**

```java
workbook.getWorksheets().get(0).getCells().get("A1").setValue("DEMO TEXT ON SHEET1");
```
*Объяснение:* Этот код устанавливает значение ячейки A1 на первом листе, проверяя вставку данных.

- **Добавить второй рабочий лист и заполнить**

```java
import com.aspose.cells.SheetType;

workbook.getWorksheets().add(SheetType.WORKSHEET);
workbook.getWorksheets().get(1).getCells().get("A1").setValue("DEMO TEXT ON SHEET2");
```
*Объяснение:* Добавление второго рабочего листа и заполнение его текстом демонстрирует, как управлять несколькими листами.

### Установить активный рабочий лист

#### Обзор
Установка активного рабочего листа позволяет указать, какой лист в данный момент находится в фокусе для таких операций, как рендеринг или сохранение.

```java
// Предположим, что «рабочая книга» уже создана и содержит несколько рабочих листов...
workbook.getWorksheets().setActiveSheetIndex(1);
```
*Объяснение:* Это устанавливает второй рабочий лист (индекс 1) в качестве активного, что имеет решающее значение при выполнении действий, специфичных для этого листа, таких как его рендеринг в SVG.

### Сохранить книгу как SVG

#### Обзор
Сохранение книги в формате SVG подразумевает указание того, что должен отображаться только активный лист, оптимизацию размера файла и сосредоточение внимания на релевантных данных.

```java
// Предположим, что «рабочая книга» уже создана и для нее установлен активный рабочий лист...
workbook.save(outputDir + "/ConvertActiveWorksheetToSVG_out.svg");
```
*Объяснение:* Этот код сохраняет только активный лист как файл SVG. Убедитесь, что выходной путь настроен правильно для правильного сохранения.

**Советы по устранению неполадок:**
- Убедитесь, что `outputDir` является допустимым каталогом с правами записи.
- Перед попыткой сохранения убедитесь, что установлен индекс активного рабочего листа.
