## Руководство по внедрению

### Обзор: настройка параметров изображения
Управление тем, как визуализируются изображения, имеет решающее значение для сохранения целостности документа во время преобразования. В этом разделе вы узнаете, как эффективно настроить эти параметры.

#### Шаг 1: Загрузите свою рабочую книгу
Начните с загрузки книги Excel:
```java
import com.aspose.cells.Workbook;
import AsposeCellsExamples.Utils;

public class SettingImagePreferencesForHTML {
    public static void main(String[] args) throws Exception {
        // Путь к каталогу документов.
        String dataDir = Utils.getSharedDataDir(SettingImagePreferencesForHTML.class)
                + "LoadingSavingConvertingAndManaging/";

        // Создать экземпляр объекта Workbook по пути к файлу Excel
        Workbook workbook = new Workbook(dataDir + "Book1.xlsx");
    }
}
```

#### Шаг 2: Настройте параметры сохранения HTML
Далее настройте `HtmlSaveOptions` чтобы задать настройки изображения:
```java
import com.aspose.cells.HtmlSaveOptions;

// Создать объект HtmlSaveOptions
HtmlSaveOptions options = new HtmlSaveOptions();

// Установить параметры для изображений
options.setExportImagesAsBase64(true); // Встраивание изображений в виде строк base64

// Сохраните документ в формате HTML с указанными параметрами.
workbook.save(dataDir + "Output.html", options);
```
**Объяснение:**
- `setExportImagesAsBase64(true)`: этот параметр встраивает изображения непосредственно в HTML-файл с использованием кодировки Base64, гарантируя их корректное отображение без необходимости использования отдельных файлов изображений.

#### Шаг 3: Сохраните книгу как HTML
Наконец, сохраните вашу книгу с настроенными параметрами:
```java
// Сохраните документ в формате HTML с указанными параметрами.
workbook.save(dataDir + "Output.html", options);

System.out.println("PDF Creation Time successfully.");
```
**Советы по устранению неполадок:**
- Убедитесь, что пути к файлам указаны правильно, чтобы избежать `FileNotFoundException`.
- Если изображения не отображаются, проверьте, включено ли кодирование Base64.
