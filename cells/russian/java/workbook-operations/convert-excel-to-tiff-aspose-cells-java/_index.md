Чтобы интегрировать Aspose.Cells с помощью Maven, добавьте следующую зависимость в ваш `pom.xml` файл:

```xml

```

## Руководство по внедрению (H2)

### Инициализация Aspose.Cells для преобразования TIFF (H3)

Начните с настройки основных функций для работы с Aspose.Cells.

#### Шаг 1: Настройка параметров рабочей книги и изображения (H3)

Загрузите книгу Excel, которую вы хотите преобразовать в изображение. Настройте `ImageOrPrintOptions` чтобы указать TIFF в качестве желаемого выходного формата:

```java
import com.aspose.cells.*;

// Загрузите исходную рабочую книгу
Workbook wb = new Workbook("path/to/sampleUseWorkbookRenderForImageConversion.xlsx");

// Настройте параметры для преобразования TIFF
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.setImageType(ImageType.TIFF);
```

#### Шаг 2: Реализация обратного вызова прогресса (H3)

Для отслеживания процесса преобразования документа реализуйте пользовательский класс, который соответствует `IPageSavingCallback` Интерфейс. Позволяет определить действия в начале и конце процесса сохранения каждой страницы:

```java
class TestTiffPageSavingCallback implements IPageSavingCallback {
    public void pageStartSaving(PageStartSavingArgs args) {
        System.out.println("Start saving page index " + args.getPageIndex() + \
                           " of pages " + args.getPageCount());
        // Исключить страницы до индекса 2
        if (args.getPageIndex() < 2) {
            args.setToOutput(false);
        }
    }

    public void pageEndSaving(PageEndSavingArgs args) {
        System.out.println("End saving page index " + args.getPageIndex() + \
                           " of pages " + args.getPageCount());
        // Остановить преобразование после индекса страницы 8
        if (args.getPageIndex() >= 8) {
            args.setHasMorePages(false);
        }
    }
}
```

#### Шаг 3: Выполнение процесса преобразования (H3)

Использовать `WorkbookRender` чтобы преобразовать вашу рабочую книгу в изображение TIFF, применяя настроенные параметры и обратный вызов хода выполнения:

```java
// Установите обратный вызов сохранения страницы для отслеживания хода конверсии
opts.setPageSavingCallback(new TestTiffPageSavingCallback());

// Преобразовать рабочую книгу в файл изображения с указанными параметрами
WorkbookRender wr = new WorkbookRender(wb, opts);
wr.toImage("path/to/DocumentConversionProgressForTiff_out.tiff");
```

### Советы по устранению неполадок (H2)

- **Обеспечьте правильные пути**: Убедитесь, что все пути к файлам (исходный Excel и выходной TIFF) определены правильно.
- **Мониторинг использования памяти Java**: Aspose.Cells может быть ресурсоемким; выделите достаточно памяти для вашей JVM.
