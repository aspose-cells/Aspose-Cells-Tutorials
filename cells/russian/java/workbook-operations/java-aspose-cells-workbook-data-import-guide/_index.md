## Руководство по внедрению
### Функция 1: Таблица данных ячеек
Эта функция позволяет вам определять и управлять таблицами данных с предустановленными столбцами и строками. Вот как это работает:

#### Обзор
The `CellsDataTable` Класс предоставляет структурированный способ обработки табличных данных с использованием Aspose.Cells, позволяя получать доступ по индексу столбца или имени.

#### Этапы внедрения
##### 1. Определите структуру данных
Создайте класс для инкапсуляции структуры вашей таблицы данных:
```java
import java.util.Arrays;

class CellsDataTable {
    private int m_index = -1;
    private String[] colsNames = new String[]{"Pet", "Fruit", "Country", "Color"};
    private String[][] colsData = {
        {"Dog", "Cat", "Duck"},
        {"Apple", "Pear", "Banana"},
        {"UK", "USA", "China"},
        {"Red", "Green", "Blue"}
    };

    public void beforeFirst() {
        m_index = -1;
    }

    public Object get(int columnIndex) {
        return (m_index >= 0 && m_index < colsData[columnIndex].length)
            ? colsData[columnIndex][m_index] : null;
    }

    public String[] getColumns() {
        return colsNames;
    }

    public int getCount() {
        return colsData[0].length;
    }

    public boolean next() {
        if (m_index + 1 < colsData[0].length) {
            m_index++;
            return true;
        } else {
            return false;
        }
    }
}
```
##### 2. Траверсные данные
Используйте `beforeFirst`, `next`, и `get` методы для эффективной итерации по таблице данных.

### Функция 2: Манипулирование рабочей книгой с параметрами импорта данных
Эта функция демонстрирует, как импортировать структурированные данные в книгу Excel с помощью Aspose.Cells, предлагая контроль над поведением импорта, таким как сдвиг строк.

#### Обзор
The `WorkbookDataImport` В классе демонстрируется импорт предопределенной структуры данных в файл Excel с сохранением определенных конфигураций.

#### Этапы внедрения
##### 1. Настройка рабочей книги и рабочего листа
Загрузите целевую книгу и выберите лист, на который вы хотите импортировать данные:
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;
import com.aspose.cells.ImportTableOptions;

public class WorkbookDataImport {
    public void run() throws Exception {
        String dataDir = "YOUR_DATA_DIRECTORY";
        String outDir = "YOUR_OUTPUT_DIRECTORY";

        CellsDataTable cellsDataTable = new CellsDataTable();
        Workbook wb = new Workbook(dataDir + "/sampleImportTableOptionsShiftFirstRowDown.xlsx");
        Worksheet ws = wb.getWorksheets().get(0);
```
##### 2. Настройте параметры импорта
Настройте `ImportTableOptions` для управления импортом данных:
```java
        ImportTableOptions opts = new ImportTableOptions();
        opts.setShiftFirstRowDown(false);

        // Импорт данных, начиная с ячейки (2, 2)
        ws.getCells().importData(cellsDataTable, 2, 2, opts);
```
##### 3. Сохранить книгу
После настройки и импорта сохраните книгу, чтобы сохранить изменения:
```java
        wb.save(outDir + "/outputImportTableOptionsShiftFirstRowDown-False.xlsx");
    }
}
```
