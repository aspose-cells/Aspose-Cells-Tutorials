## Руководство по внедрению
Давайте рассмотрим процесс создания и настройки пузырьковых диаграмм с помощью Aspose.Cells.

### Инициализация объекта рабочей книги
А `Workbook` представляет собой целый файл Excel, позволяющий вам манипулировать листами, ячейками и т. д. Инициализируйте его следующим образом:
```java
import com.aspose.cells.Workbook;

// Создать новый экземпляр Workbook
Workbook workbook = new Workbook();
```

### Доступ к рабочим листам и управление ими
Доступ к рабочим листам для подготовки данных для построения диаграмм:
```java
import com.aspose.cells.WorksheetCollection;
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;

// Получить коллекцию рабочих листов
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet sheet = worksheets.get(0);
Cells cells = sheet.getCells();

// Установите значения в определенных ячейках, чтобы подготовить данные для построения диаграммы.
cells.get("A1").setValue(50);
cells.get("A2").setValue(100);
cells.get("A3").setValue(150);
cells.get("B1").setValue(4);
cells.get("B2").setValue(20);
cells.get("B3").setValue(180);
cells.get("C1").setValue(320);
cells.get("C2").setValue(110);
cells.get("C3").setValue(180);
cells.get("D1").setValue(40);
cells.get("D2").setValue(120);
cells.get("D3").setValue(250);
```

### Создание и настройка пузырьковых диаграмм
Создайте пузырьковую диаграмму, добавив ее на рабочий лист и указав источники данных:
```java
import com.aspose.cells.ChartCollection;
import com.aspose.cells.Chart;
import com.aspose.cells.SeriesCollection;
import com.aspose.cells.ChartType;

// Доступ к коллекции диаграмм на листе
ChartCollection charts = sheet.getCharts();
int chartIndex = charts.add(ChartType.BUBBLE, 5, 0, 15, 5);
Chart chart = charts.get(chartIndex);

// Добавьте ряды в диаграмму и задайте источники данных
SeriesCollection serieses = chart.getNSeries();
serieses.add("A1:B3", true);

// Установите размеры пузырьков, значения X и значения Y для диаграммы
chart.getNSeries().get(0).setBubbleSizes("B2:D2");
chart.getNSeries().get(0).setXValues("B3:D3");
chart.getNSeries().get(0).setValues("B1:D1");
```

### Сохранение рабочей книги
Сохраните книгу, чтобы сохранить все изменения:
```java
import com.aspose.cells.SaveFormat;

// Определите каталог для сохранения файла
String dataDir = "YOUR_DATA_DIRECTORY";
workbook.save(dataDir + "/HToCrBChart_out.xls", SaveFormat.EXCEL_97_TO_2003);
```
