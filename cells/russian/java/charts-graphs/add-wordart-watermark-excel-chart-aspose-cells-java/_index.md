## Что такое конфиденциальный водяной знак в Excel?
Конфиденциальный водяной знак — это полупрозрачный текст или графика, размещённые за данными диаграммы, чтобы указать, что содержимое является чувствительным. Он остаётся видимым при печати и на экране, не закрывая базовые данные.

## Почему использовать Aspose.Cells для добавления водяного знака?
Aspose.Cells предоставляет мощный API для работы с файлами Excel без необходимости установки Microsoft Office. Он поддерживает формы WordArt, точный контроль прозрачности и работает на всех платформах Java.

### Добавление водяного знака WordArt в диаграмму
1. **Откройте существующий файл Excel**  
   Загрузите ваш файл Excel, в который вы хотите добавить водяной знак:  
```java
String dataDir = Utils.getSharedDataDir(AddWordArtWatermarkToChart.class) + "TechnicalArticles/";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

2. **Получите диаграмму**  
   Получите диаграмму с первого листа, который вы хотите изменить:  
```java
Chart chart = workbook.getWorksheets().get(0).getCharts().get(0);
```

3. **Добавьте форму WordArt**  
   Вставьте новую форму WordArt в область построения вашей диаграммы:  
```java
Shape wordart = chart.getShapes().addTextEffectInChart(
    MsoPresetTextEffect.TEXT_EFFECT_1,
    "CONFIDENTIAL",
    "Arial Black", 66, false, false, 
    1200, 500, 2000, 3000);
```

4. **Настройте заливку и формат линии**  
   Установите прозрачность, чтобы водяной знак был ненавязчивым:  
```java
// Configure transparency.
FillFormat wordArtFormat = wordart.getFill();
wordArtFormat.setTransparency(0.9);

// Make line format invisible.
LineFormat lineFormat = wordart.getLine();
lineFormat.setWeight(0.0);
```

5. **Сохраните книгу**  
   Сохраните изменения в новый файл:  
```java
workbook.save(dataDir + "AWArtWToC_out.xlsx");
```

## Часто задаваемые вопросы

**В: Появляется ли водяной знак на печатных листах Excel?**  
О: Да, форма WordArt является частью диаграммы и печатается вместе с данными диаграммы.

**В: Можно ли автоматически применить один и тот же водяной знак к нескольким диаграммам?**  
О: Пройдитесь по `workbook.getWorksheets().get(i).getCharts()` и примените те же шаги к каждой диаграмме.

**В: Можно ли изменить цвет водяного знака?**  
О: Конечно — используйте `wordArtFormat.getSolidFill().setColor(Color.getRGB(255,0,0))` для установки пользовательского цвета.

**В: Увеличит ли добавление водяного знака размер файла существенно?**  
О: Увеличение минимально, так как добавляется только один объект формы.

**В: Как позже удалить водяной знак?**  
О: Найдите форму по её имени или индексу в `chart.getShapes()` и вызовите `shape.delete()`.
