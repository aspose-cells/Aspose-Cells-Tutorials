## Руководство по внедрению

Мы рассмотрим три основные функции: загрузку и обновление диаграмм, настройку параметров глобализации и отображение диаграмм в виде изображений.

### Загрузка и обновление диаграммы из файла Excel

**Обзор**: эта функция позволяет загрузить существующий файл Excel с диаграммой и обновить его для обеспечения точности данных.

#### Пошаговая реализация

##### 1. Загрузите рабочую книгу
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook book = new Workbook(dataDir + "sample.xlsx");
```
The `Workbook` класс загружает ваш файл Excel из указанного пути.

##### 2. Доступ к рабочему листу и диаграмме
```java
Worksheet sheet = book.getWorksheets().get(0);
Chart chart = sheet.getCharts().get(0);
```
Откройте первый рабочий лист и диаграмму, чтобы работать с ними программно.

##### 3. Обновите диаграмму.
```java
chart.calculate();
```
The `calculate` метод обновляет данные и макет диаграммы, гарантируя, что она отражает любые изменения в исходных данных.

### Настройте параметры глобализации для диаграмм

**Обзор**: Настройте глобальные параметры, такие как форматы дат или стили чисел, в соответствии с региональными предпочтениями.

#### Пошаговая реализация

##### 1. Загрузите рабочую книгу
```java
Workbook book = new Workbook(dataDir + "sample.xlsx");
```

##### 2. Установите пользовательские параметры глобализации
```java
CustomSettings customGlobalizationSettings = new CustomSettings();
customGlobalizationSettings.setChartSettings(new CustomChartGlobalizationSettings());
book.getSettings().setGlobalizationSettings(customGlobalizationSettings);
```
В этом фрагменте показано, как применять пользовательские настройки, обеспечивая соответствие диаграмм конкретным региональным стандартам.

### Преобразовать диаграмму в изображение

**Обзор**: Преобразуйте диаграмму в формат изображения для удобства публикации или встраивания в документы.

#### Пошаговая реализация

##### 1. Загрузите рабочую книгу и получите доступ к диаграмме.
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
Workbook book = new Workbook(dataDir + "sample.xlsx");
Worksheet sheet = book.getWorksheets().get(0);
Chart chart = sheet.getCharts().get(0);
chart.calculate();
```

##### 2. Рендеринг в изображение
```java
ImageOrPrintOptions options = new ImageOrPrintOptions();
chart.toImage(outDir + "CustomTextforOtherLabelofPieChart_out.png", options);
```
Это преобразует диаграмму в изображение PNG, которое можно использовать в различных контекстах.
