## Руководство по внедрению

### Загрузка рабочей книги и доступ к фигурам

#### Обзор
Чтобы обнаружить фигуры SmartArt, сначала необходимо загрузить книгу Excel и получить доступ к ее содержимому.

#### Шаги:

**1. Загрузите образец рабочей книги**

```java
import com.aspose.cells.*;

public class DetermineIfShapeIsSmartArtShape {
    static String srcDir = Utils.Get_SourceDirectory();

    public static void main(String[] args) throws Exception {
        // Загрузите образец смарт-арт-фигуры — файл Excel
        Workbook wb = new Workbook(srcDir + "sampleSmartArtShape.xlsx");
    }
}
```

- **Параметры**: `Workbook` конструктор принимает строковый параметр, представляющий путь к файлу документа Excel.

**2. Доступ к первому рабочему листу**

```java
// Доступ к первому рабочему листу
Worksheet ws = wb.getWorksheets().get(0);
```

- **Цель**: извлекает первый рабочий лист в рабочей книге для дальнейших операций.

**3. Доступ к фигуре и обнаружение SmartArt**

```java
// Доступ к первой форме
Shape sh = ws.getShapes().get(0);

// Определите, является ли форма умным искусством
System.out.println("Is Smart Art Shape: " + sh.isSmartArt());
```

- **Объяснение метода**: `isSmartArt()` метод проверяет, является ли заданная фигура графическим элементом SmartArt.
  
**Советы по устранению неполадок**:
- Убедитесь, что ваш файл Excel содержит хотя бы один рабочий лист и форму.
- Проверьте путь, указанный в `srcDir` указывает на правильное местоположение вашего файла Excel.
