## Руководство по внедрению
### Расширение класса Person
В этом разделе мы расширим `Person` класс для создания `Individual` класс, который управляет дополнительными атрибутами и поведением.
#### Обзор:
The `Individual` класс расширяется `Person`, демонстрирующий наследование в Java для расширения функциональности путем добавления определенных характеристик, таких как информация о супруге.
##### Шаг 1: Определите индивидуальный класс
Начните с создания `Individual` класс, включающий закрытые члены и конструкторы для инициализации объектов:
```java
import java.util.ArrayList;
class Person {
    // Упрощенная версия базового класса, например Aspose.Person
    protected String name;
    protected int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
// Индивидуальный класс, расширяющий Персону
class Individual extends Person {
    private Person m_Wife; // Частный участник для информации о супруге

    // Конструктор для класса Individual
    public Individual(String name, int age, Person wife) {
        super(name, age); // Вызов конструктора суперкласса
        this.m_Wife = wife; // Инициализируйте m_Wife с указанным значением
    }

    // Метод получения для m_Wife
    public Person getWife() {
        return m_Wife;
    }
}
```
**Объяснение**: 
- **Конструктор суперкласса**: `super(name, age)` инициализирует суперкласс `Person` атрибуты.
- **Частный член**: `m_Wife` хранит информацию о супруге, демонстрируя инкапсуляцию.
##### Шаг 2: Используйте индивидуальный класс
Создайте экземпляры вашего нового класса и используйте его функциональность:
```java
public class Main {
    public static void main(String[] args) {
        Person wife = new Person("Jane", 30);
        Individual person = new Individual("John", 35, wife);

        System.out.println("Person's Wife: " + person.getWife().name); // Вывод: Джейн
    }
}
```
**Объяснение**: 
- Это демонстрирует создание `Person` объект для представления супруга и передачи его при построении `Individual`.
