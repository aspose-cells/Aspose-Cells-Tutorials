## Руководство по внедрению

### Установить для комментариев к печати значение «Нет комментариев»
Эта функция гарантирует, что комментарии не будут распечатаны, и распечатываемый документ будет сосредоточен на данных.

#### Обзор
Установив `PrintCommentsType` к `PRINT_NO_COMMENTS`, вы запрещаете включение каких-либо комментариев в PDF-вывод вашего файла Excel.

#### Этапы внедрения
**Шаг 1: Загрузите свою рабочую книгу**
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

**Шаг 2: Доступ к рабочему листу**
```java
Worksheet worksheet = workbook.getWorksheets().get(0); // Первый рабочий лист
```

**Шаг 3: Установите параметр «Печать комментариев»**
```java
import com.aspose.cells.PrintCommentsType;
worksheet.getPageSetup().setPrintComments(PrintCommentsType.PRINT_NO_COMMENTS);
```

**Шаг 4: Сохранить как PDF**
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "PrintNoComments_out.pdf");
```

### Распечатать комментарии на месте
Печать комментариев непосредственно там, где они находятся, обеспечивает четкое представление аннотаций вместе с соответствующими данными.

#### Обзор
Установите `PrintCommentsType` к `PRINT_IN_PLACE` чтобы достичь этого.

#### Этапы внедрения
**Шаг 1: Загрузите свою рабочую книгу**
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

**Шаг 2: Доступ к рабочему листу**
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**Шаг 3: Настройте комментарии к печати на месте**
```java
worksheet.getPageSetup().setPrintComments(PrintCommentsType.PRINT_IN_PLACE);
```

**Шаг 4: Сохранить как PDF**
```java
workbook.save(outDir + "PrintInPlace_out.pdf");
```

### Распечатать комментарии в конце листа
Соберите все комментарии и распечатайте их в конце листа для получения консолидированного представления.

#### Обзор
Использовать `PRINT_SHEET_END` для настройки этого параметра.

#### Этапы внедрения
**Шаг 1: Загрузите свою рабочую книгу**
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

**Шаг 2: Доступ к рабочему листу**
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**Шаг 3: Установите комментарии к печати в конце листа**
```java
worksheet.getPageSetup().setPrintComments(PrintCommentsType.PRINT_SHEET_END);
```

**Шаг 4: Сохранить как PDF**
```java
workbook.save(outDir + "PrintSheetEnd_out.pdf");
```
