### Добавление горизонтальных разрывов страниц
**Обзор**: Настройка отображения данных на страницах повышает читабельность. Давайте посмотрим, как добавить горизонтальные разрывы страниц на рабочем листе.

#### Пошаговое руководство
1. **Доступ к первому рабочему листу**
2. **Добавить горизонтальный разрыв страницы**

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.WorksheetCollection;
import com.aspose.cells.Worksheet;
import com.aspose.cells.HorizontalPageBreakCollection;

public class AddHorizontalPageBreak {
    public static void main(String[] args) throws Exception {
        // Создать новый экземпляр рабочей книги
        Workbook workbook = new Workbook();
        
        // Доступ к первому рабочему листу в рабочей книге
        WorksheetCollection worksheets = workbook.getWorksheets();
        Worksheet worksheet = worksheets.get(0);
        
        // Получить коллекцию горизонтальных разрывов страниц на рабочем листе
        HorizontalPageBreakCollection hPageBreaks = worksheet.getHorizontalPageBreaks();
        
        // Добавить горизонтальный разрыв страницы в ячейке «Y30»
        hPageBreaks.add("Y30");
    }
}
```

### Добавление вертикальных разрывов страниц
**Обзор**: Подобно горизонтальным разрывам, вертикальные разрывы страниц могут помочь более эффективно организовать данные.

#### Пошаговое руководство
1. **Получить первый рабочий лист**
2. **Добавить вертикальный разрыв страницы**

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.WorksheetCollection;
import com.aspose.cells.Worksheet;
import com.aspose.cells.VerticalPageBreakCollection;

public class AddVerticalPageBreak {
    public static void main(String[] args) throws Exception {
        // Создать новый объект рабочей книги
        Workbook workbook = new Workbook();
        
        // Извлечь первый рабочий лист из рабочей книги
        WorksheetCollection worksheets = workbook.getWorksheets();
        Worksheet worksheet = worksheets.get(0);
        
        // Доступ к коллекции вертикальных разрывов страниц на рабочем листе
        VerticalPageBreakCollection vPageBreaks = worksheet.getVerticalPageBreaks();
        
        // Добавить вертикальный разрыв страницы в ячейке «Y30»
        vPageBreaks.add("Y30");
    }
}
```
