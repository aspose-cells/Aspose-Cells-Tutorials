## Руководство по внедрению
Теперь давайте рассмотрим основную функцию копирования настроек страниц между рабочими листами.

### Обзор
Копирование настроек страницы подразумевает дублирование настроек, таких как размер бумаги и ориентация, с одного листа на другой. Это обеспечивает единообразие на нескольких листах в рабочей книге.

#### Создание рабочих книг и рабочих листов (H3)
Начните с создания новой рабочей книги и добавления двух тестовых рабочих листов:

```java
import com.aspose.cells.*;

public class CopyPageSetupSettingsFromSourceWorksheetToDestinationWorksheet {
    public static void main(String[] args) throws Exception {
        // Инициализировать рабочую книгу
        Workbook wb = new Workbook();

        // Добавить рабочие листы
        wb.getWorksheets().add("TestSheet1");
        wb.getWorksheets().add("TestSheet2");

        System.out.println("Workbooks and worksheets created successfully.");
    }
}
```

#### Настроить размер бумаги (H3)
Определите размер бумаги для `TestSheet1` для демонстрации копирования настроек:

```java
// Доступ к тестовому листу1
Worksheet TestSheet1 = wb.getWorksheets().get("TestSheet1");

// Установите размер бумаги TestSheet1 на PAPER_A_3_EXTRA_TRANSVERSE
TestSheet1.getPageSetup().setPaperSize(PaperSizeType.PAPER_A_3_EXTRA_TRANSVERSE);

System.out.println("Paper size set for TestSheet1.");
```

#### Копировать параметры страницы (H3)
Теперь скопируйте настройки страницы из `TestSheet1` к `TestSheet2`:

```java
// Доступ к TestSheet2
Worksheet TestSheet2 = wb.getWorksheets().get("TestSheet2");

// Скопируйте PageSetup из TestSheet1 в TestSheet2
TestSheet2.getPageSetup().copy(TestSheet1.getPageSetup(), new CopyOptions());

System.out.println("Page setup copied successfully.");
```
