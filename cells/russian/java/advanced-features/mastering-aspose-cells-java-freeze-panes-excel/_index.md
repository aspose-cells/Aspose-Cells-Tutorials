## Почему использовать Aspose.Cells Freeze Panes?
- **Consistent Reporting** – Заголовки никогда не исчезают, улучшая читаемость печатных или общих отчётов.  
- **Automation Friendly** – Применяйте одинаковый макет к десяткам сгенерированных книг одной строкой кода.

## aspose cells freeze panes – Основные концепции
### Загрузка и доступ к файлу Excel
**Обзор**: Этот раздел поможет вам загрузить существующий файл Excel и получить доступ к его первому листу с помощью Aspose.Cells Java.

#### Шаг 1: Импорт необходимых классов
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.WorksheetCollection;
import com.aspose.cells.Worksheet;
```

#### Шаг 2: Загрузка книги
Создайте экземпляр `Workbook`, указав путь к вашему файлу Excel. Это необходимо для доступа к содержимому и его изменения.
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "book.xls");
```
**Объяснение**: Конструктор `new Workbook(filePath)` инициализирует объект книги, позволяя выполнять над ним операции.

#### Шаг 3: Доступ к первому листу
Получите первый лист из книги, используя её коллекцию листов. 
```java
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet worksheet = worksheets.get(0);
```
**Объяснение**: Метод `getWorksheets()` возвращает все листы, а обращение к индексу `0` даёт нам первый лист.

## Как применить Freeze Panes в Aspose.Cells
### Установка Freeze Panes на листе
**Обзор**: Узнайте, как сохранять определённые строки и столбцы видимыми при прокрутке листа, применяя настройки закрепления областей.

#### Шаг 4: Установка Freeze Panes
Примените закрепление областей с помощью метода `freezePanes`.
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
worksheet.freezePanes(3, 2, 3, 2);
```
**Объяснение**: Параметры `(rowSplitIndex, columnSplitIndex, frozenRowCount, frozenColumnCount)` определяют, какие строки и столбцы остаются видимыми при прокрутке.

## Как сохранить Excel Workbook Java
### Сохранение изменений
**Обзор**: После внесения изменений сохраните книгу, чтобы зафиксировать ваши правки.

#### Шаг 5: Сохранение книги
Запишите обновлённую книгу обратно на диск, указав путь.
```java
workbook.save(outDir + "FreezePanes_out.xls");
```
**Объяснение**: Метод `save(filePath)` фиксирует все изменения, сделанные в книге, гарантируя их постоянное сохранение в файле Excel.
