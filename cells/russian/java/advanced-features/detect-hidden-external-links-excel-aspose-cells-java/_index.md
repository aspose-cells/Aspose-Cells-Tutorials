## Руководство по внедрению

### Обнаружение скрытых внешних ссылок

Давайте рассмотрим, как можно обнаружить скрытые внешние ссылки в книгах Excel с помощью Aspose.Cells для Java.

#### Обзор

Этот раздел проведет вас через загрузку рабочей книги, доступ к ее внешним ссылкам и проверку их статуса видимости. Это имеет решающее значение для аудита целостности данных в ваших электронных таблицах.

#### Загрузка рабочей книги

Сначала убедитесь, что у вас есть доступ к нужному каталогу, где находится ваша рабочая книга:
```java
import com.aspose.cells.Workbook;
import AsposeCellsExamples.Utils;

public class CheckWorkbookContainsHiddenExternalLinks {
    public static void main(String[] args) throws Exception {
        // Определите путь к вашей рабочей книге
        String dataDir = Utils.getSharedDataDir(CheckWorkbookContainsHiddenExternalLinks.class) + "TechnicalArticles/";
        
        // Загрузите книгу, содержащую внешние ссылки
        Workbook workbook = new Workbook(dataDir + "CheckWorkbookContainsHiddenExternalLinks_in.xlsx");
    }
}
```

#### Доступ к внешним ссылкам

После загрузки вашей рабочей книги откройте ее коллекцию внешних ссылок:
```java
import com.aspose.cells.ExternalLinkCollection;

public class CheckWorkbookContainsHiddenExternalLinks {
    public static void main(String[] args) throws Exception {
        // Загрузите рабочую книгу (как показано ранее)
        
        // Доступ к коллекции внешних ссылок
        ExternalLinkCollection links = workbook.getWorksheets().getExternalLinks();
    }
}
```

#### Проверка видимости ссылки

Пройдитесь по каждой ссылке, чтобы определить ее статус видимости:
```java
public class CheckWorkbookContainsHiddenExternalLinks {
    public static void main(String[] args) throws Exception {
        // Загрузите рабочую книгу и получите доступ к внешним ссылкам (как показано ранее)
        
        // Пройдитесь по каждой ссылке и выведите подробности
        for (int i = 0; i < links.getCount(); i++) {
            System.out.println("Data Source: " + links.get(i).getDataSource());
            System.out.println("Is Referred: " + links.get(i).isReferred());
            System.out.println("Is Visible: " + links.get(i).isVisible());
            System.out.println();
        }
    }
}
```

**Объяснение:**
- `links.get(i).getDataSource()` извлекает URL-адрес источника данных каждой внешней ссылки.
- `links.get(i).isReferred()` проверяет, активно ли ссылка упоминается в рабочей книге.
- `links.get(i).isVisible()` указывает, является ли ссылка видимой или скрытой.
