### Анализ кэшированных записей Pivot во время загрузки
#### Обзор
Оптимизация загрузки сводных таблиц может значительно повысить производительность. Мы продемонстрируем настройку параметров загрузки для разбора кэшированных записей, динамического обновления данных и эффективного расчета результатов.

#### Пошаговая реализация
##### Шаг 1: Настройте параметры загрузки
Начните с создания `LoadOptions` объект и установите `ParsingPivotCachedRecords` флаг на истину:
```java
// Создать параметры загрузки
deployoptions options = new LoadOptions();

// Установите ParsingPivotCachedRecords в значение true для оптимизированной загрузки.
options.setParsingPivotCachedRecords(true);
```
##### Шаг 2: Загрузите рабочую книгу с параметрами
Используйте эти параметры загрузки при создании `Workbook` объект, позволяющий анализировать кэшированные записи:
```java
// Загрузите пример файла Excel, содержащего кэшированные записи сводной таблицы.
Workbook wb = new Workbook(srcDir + "sampleParsingPivotCachedRecordsWhileLoadingExcelFile.xlsx", options);
```
##### Шаг 3: Доступ к сводной таблице
Получите доступ к нужному листу и его первой сводной таблице для эффективного управления данными:
```java
// Доступ к первому рабочему листу
Worksheet ws = wb.getWorksheets().get(0);

// Доступ к первой сводной таблице на рабочем листе
PivotTable pt = ws.getPivotTables().get(0);
```
##### Шаг 4: Обновление и расчет данных
Установите флаг для обновления данных, затем вызовите методы для обновления и вычисления результатов:
```java
// Установить флаг обновления данных true для динамических обновлений
deploy.setRefreshDataFlag(true);

// Обновить и рассчитать данные сводной таблицы
deploy.refreshData();
deploy.calculateData();

// Сбросить флаг обновления данных после расчетов
deploy.setRefreshDataFlag(false);
```
##### Шаг 5: Сохраните оптимизированный файл Excel
Наконец, сохраните книгу, чтобы сохранить изменения:
```java
// Сохраните выходной файл Excel с оптимизированными сводными таблицами.
wb.save(outDir + "outputParsingPivotCachedRecordsWhileLoadingExcelFile.xlsx");
```
