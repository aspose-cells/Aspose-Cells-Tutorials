## Настройка Aspose.Cells для Java

### Установка Maven

Чтобы включить Aspose.Cells в ваш проект Maven, добавьте следующую зависимость в ваш проект `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Установка Gradle

Для пользователей Gradle включите это в свой файл сборки:

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Руководство по внедрению

Мы разделим это на две основные функции: настройка параметров глобализации сводной таблицы и экспорт в PDF.

### Настройки глобализации пользовательских сводных таблиц

#### Обзор

Эта функция позволяет определять специальные метки для различных компонентов сводной таблицы, обеспечивая больший контроль над ее внешним видом в различных локалях или настраиваемых форматах.

#### Этапы внедрения
1. **Определить пользовательские метки**
   Создайте класс, расширяющий `GlobalizationSettings`:

   ```java
   import com.aspose.cells.*;

   public class CustomPivotTableGlobalizationSettings extends GlobalizationSettings {
       public String getPivotTotalName() { return "AsposeGetPivotTotalName"; }
       // Определите другие методы, аналогичные указанным выше, для каждой метки, которую вы хотите настроить.
   }
   ```

2. **Применить настройки**
   Загрузите рабочую книгу и примените следующие настройки:

   ```java
   Workbook wb = new Workbook("YOUR_DATA_DIRECTORY/samplePivotTableGlobalizationSettings.xlsx");
   wb.getSettings().setGlobalizationSettings(new CustomPivotTableGlobalizationSettings());
   ```

### Экспорт в PDF

#### Обзор

После настройки сводной таблицы вы можете захотеть экспортировать ее в формате PDF. В этом разделе показано, как эффективно сохранить настроенную книгу Excel.

#### Этапы внедрения
1. **Скрыть лист данных**
   Если в конечном результате лист данных не нужен:

   ```java
   wb.getWorksheets().get(0).setVisible(false);
   ```

2. **Обновить и рассчитать сводную таблицу**
   Убедитесь, что сводная таблица отражает последние данные:

   ```java
   Worksheet ws = wb.getWorksheets().get(1);
   PivotTable pt = ws.getPivotTables().get(0);

   pt.setRefreshDataFlag(true);
   pt.refreshData();
   pt.calculateData();
   pt.setRefreshDataFlag(false);
   ```

3. **Сохранить как PDF**
   Настройте параметры сохранения и экспорта:

   ```java
   PdfSaveOptions options = new PdfSaveOptions();
   options.setOnePagePerSheet(true);

   wb.save("YOUR_OUTPUT_DIRECTORY/outputPivotTableGlobalizationSettings.pdf", options);
   ```
