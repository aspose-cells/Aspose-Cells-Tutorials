# Как разгруппировать строки и столбцы в Excel с помощью Aspose.Cells Java

## Введение

Вы боретесь с группированными данными в таблицах Excel, что усложняет анализ или представление? Группировка строк и столбцов — это обычная функция в Excel для упрощения представлений, но иногда ее нужно отменять. Это руководство поможет вам без труда разгруппировать эти строки и столбцы с помощью Aspose.Cells для Java.

К концу этого руководства вы узнаете:
- Как настроить среду с помощью Aspose.Cells.
- Пошаговые инструкции по разгруппировке строк и столбцов в файлах Excel.
- Практическое применение этих функций.

Давайте рассмотрим необходимые предварительные условия, прежде чем начать.

## Предпосылки

Прежде чем приступить к кодированию, убедитесь, что у вас есть следующее:

- **Необходимые библиотеки**: Требуется Aspose.Cells для Java версии 25.3 или более поздней.

## Руководство по внедрению

### Разгруппировка строк в Excel

Разгруппировка строк поможет вам вернуться к исходному макету, если они сгруппированы для детального анализа. Выполните следующие шаги, чтобы разгруппировать строки.

#### Загрузите вашу рабочую книгу и рабочий лист
Сначала загрузите вашу рабочую книгу из файла:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "BookStyles.xls");
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### Разгруппировать строки
Определите строки, которые вы хотите разгруппировать, и выполните:

```java
Cells cells = worksheet.getCells();
cells.ungroupRows(0, 5); // Разгруппировывает строки с индексом от 0 до 5
```

### Разгруппировка столбцов в Excel
Аналогичным образом разгруппируйте столбцы, если они были сгруппированы, для лучшего управления данными или их представления.

#### Загрузите вашу рабочую книгу и рабочий лист
Убедитесь, что ваша рабочая книга загружена:

```java
Workbook workbook = new Workbook(dataDir + "BookStyles.xls");
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### Разгруппировать столбцы
Укажите диапазон индексов столбцов для разгруппировки:

```java
Cells cells = worksheet.getCells();
cells.ungroupColumns(0, 2); // Разгруппировать столбцы с индексом от 0 до 2
```

### Сохраните изменения
После внесения изменений сохраните книгу:

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "UngroupingRows_out.xls"); // Для рядов
workbook.save(outDir + "UngroupingColumns_out.xls"); // Для столбцов
```
