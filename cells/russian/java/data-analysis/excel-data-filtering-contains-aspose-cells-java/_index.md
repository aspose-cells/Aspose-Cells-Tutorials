## Настройка Aspose.Cells для Java

Чтобы использовать Aspose.Cells в вашем проекте, вам нужно включить его в качестве зависимости. Вот как это можно сделать с помощью Maven или Gradle:

### Знаток

Добавьте следующую зависимость к вашему `pom.xml` файл:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Градл

Включите это в свой `build.gradle` файл:

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Руководство по внедрению

В этом разделе мы рассмотрим создание настраиваемого фильтра, который идентифицирует строки, содержащие определенный текст, с помощью Aspose.Cells для Java.

### Шаг 1: Загрузите рабочую книгу

Начните с загрузки существующего файла Excel. Это ваш источник данных.

```java
Workbook workbook = new Workbook(srcDir + "sourseSampleCountryNames.xlsx");
```

### Шаг 2: Доступ к рабочему листу

Откройте рабочий лист, содержащий данные, которые вы хотите отфильтровать:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

### Шаг 3: Настройка автофильтра

Создайте `AutoFilter` указав диапазон. Это определяет область вашего листа, которая будет отфильтрована:

```java
worksheet.getAutoFilter().setRange("A1:A18");
```

### Шаг 4: Применение пользовательского фильтра

Используйте `custom()` метод применения фильтра, который ищет строки, содержащие определенный текст, в данном случае «Ba»:

```java
worksheet.getAutoFilter().custom(0, FilterOperatorType.CONTAINS, "Ba");
```

**Объяснение**: 
- Первый параметр (`0`) указывает индекс столбца.
- `FilterOperatorType.CONTAINS` применяется логика фильтрации.
- `"Ba"` — это искомая строка в указанном диапазоне.

### Шаг 5: Обновите и сохраните

После настройки фильтра обновите его, чтобы применить изменения и сохранить книгу:

```java
worksheet.getAutoFilter().refresh();
workbook.save(outDir + "outSourseSampleCountryNames.xlsx");
```

**Советы по устранению неполадок**: Если у вас возникли проблемы с путями к файлам или диапазонами данных, убедитесь, что они правильно указаны относительно структуры каталогов вашего проекта.
