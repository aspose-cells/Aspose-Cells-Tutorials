## Руководство по внедрению
### Функция 1: Загрузка и обновление сводной таблицы
#### Обзор
Эта функция демонстрирует загрузку книги Excel, доступ к сводной таблице, обновление ее данных и пересчет для получения актуальной информации.

**Шаги:**

1. **Загрузите рабочую тетрадь**
   ```java
   String dataDir = "YOUR_DATA_DIRECTORY";
   Workbook workbook = new Workbook(dataDir + "/PivotTableHideAndSortSample.xlsx");
   ```

2. **Доступ к сводной таблице**
   ```java
   Worksheet worksheet = workbook.getWorksheets().get(0);
   PivotTable pivotTable = worksheet.getPivotTables().get(0);
   ```

3. **Обновить и пересчитать данные**
   ```java
   pivotTable.refreshData();
   pivotTable.calculateData();
   ```
   
Обновление гарантирует, что данные отражают все изменения, внесенные в исходный набор данных.

### Функция 2: Сортировка полей строк сводной таблицы в порядке убывания
#### Обзор
Автоматически сортировать поле строки в порядке убывания, чтобы отдать приоритет более высоким значениям.

**Шаги:**

1. **Установить автоматическую сортировку и направление**
   ```java
   PivotField field = pivotTable.getRowFields().get(0);
   field.setAutoSort(true);
   field.setAscendSort(false); // ложь для убывания
   field.setAutoSortField(0);
   ```

2. **Обновление данных Сортировка постов**
   ```java
   pivotTable.refreshData();
   pivotTable.calculateData();
   ```
   
Эта конфигурация позволяет выполнять динамическую сортировку на основе ваших критериев.

### Функция 3: Скрыть строки с оценкой менее 60
#### Обзор
Скройте строки в сводной таблице, где оценка ниже порогового значения, например 60, чтобы сосредоточиться только на значимых данных.

**Шаги:**

1. **Итерация по диапазону тел данных**
   ```java
   CellArea dataBodyRange = pivotTable.getDataBodyRange();
   int currentRow = 3;
   int rowsUsed = dataBodyRange.getEndRow();

   while (currentRow < rowsUsed) {
       Cell cell = worksheet.getCells().get(currentRow, 1);
       double score = (double) cell.getValue();
       if (score < 60) {
           worksheet.getCells().hideRow(currentRow);
       }
       currentRow++;
   }
   ```

2. **Обновить данные после скрытия строк**
   ```java
   pivotTable.refreshData();
   pivotTable.calculateData();
   ```
   
Эта логика помогает эффективно отфильтровывать менее релевантные точки данных.

### Функция 4: Сохранение файла Excel
#### Обзор
Сохраните изменения, сохранив измененную книгу в указанном каталоге.

**Шаги:**

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/PivotTableHideAndSort_out.xlsx");
```

Этот шаг гарантирует, что все изменения будут сохранены для будущего использования или распространения.
