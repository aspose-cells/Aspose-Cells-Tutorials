## Настройка Aspose.Cells для Java

Чтобы использовать Aspose.Cells для Java, вам нужно включить его в свой проект. У вас есть два варианта: Maven или Gradle.

### Знаток
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Градл
Включите это в свой `build.gradle` файл:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Руководство по внедрению

Мы разобьем реализацию на три раздела: компактная, структурная и табличная формы.

### Показать сводную таблицу в компактном виде

**Обзор:** Отображение сводной таблицы в компактном виде помогает сэкономить место, сохраняя при этом ясность.

#### Шаг 1: Загрузите файл Excel
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```
*Почему?* Это загрузит ваш исходный файл Excel в память.

#### Шаг 2: Доступ к рабочему листу и сводной таблице
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### Шаг 3: Установите компактную форму
```java
pivotTable.showInCompactForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/CompactForm.xlsx");
```
*Почему?* Данная конфигурация отображает сводную таблицу в компактном виде и сохраняет ее.

### Показать сводную таблицу в виде структуры

**Обзор:** Форма структуры идеально подходит для иерархических данных, позволяя пользователям разворачивать или сворачивать детали.

#### Шаг 1: Загрузите рабочую книгу
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### Шаг 2: Получите доступ к необходимым компонентам
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### Шаг 3: Настройте форму структуры
```java
pivotTable.showInOutlineForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/OutlineForm.xlsx");
```
*Почему?* На этом этапе сводная таблица приобретает структурную форму и обеспечивает обновление данных.

### Показать сводную таблицу в табличной форме

**Обзор:** Табличная форма отображает все данные в строках, что идеально подходит для детального анализа.

#### Шаг 1: Инициализация рабочей книги
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### Шаг 2: Доступ к компонентам
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### Шаг 3: Установка табличной формы
```java
pivotTable.showInTabularForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/TabularForm.xlsx");
```
*Почему?* Данная конфигурация представляет сводную таблицу в табличной форме.
