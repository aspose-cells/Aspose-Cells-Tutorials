## Руководство по внедрению
Выполните следующие действия для реализации изменения направления текста в комментариях.

### Добавление и изменение комментариев
#### Обзор
Повысьте ясность документа, добавив комментарии к ячейкам и настроив направление их текста. Это особенно полезно для многоязычных документов или систем, требующих особого форматирования.

**Шаг 1: Создание рабочей книги**
Создайте экземпляр `Workbook` сорт:
```java
// Создать новую рабочую книгу
Workbook wb = new Workbook();
```

**Шаг 2: Доступ к рабочему листу**
Откройте рабочий лист, на который вы хотите добавить комментарии:
```java
// Получить первый рабочий лист в рабочей тетради
Worksheet sheet = wb.getWorksheets().get(0);
```

**Шаг 3: Добавьте комментарий**
Добавьте комментарий к ячейке «A1»:
```java
// Добавить комментарий к ячейке A1
Comment comment = sheet.getComments().get(sheet.getComments().add("A1"));
```
Извлеките объект комментария, добавив его в первую ячейку.

**Шаг 4: Настройте выравнивание и направление текста**
Настройте выравнивание текста и задайте направление справа налево:
```java
// Установите вертикальное и горизонтальное выравнивание текста
comment.getCommentShape().setTextVerticalAlignment(TextAlignmentType.CENTER);
comment.getCommentShape().setTextHorizontalAlignment(TextAlignmentType.RIGHT);

// Изменить направление текста на справа налево
comment.getCommentShape().setTextDirection(TextDirectionType.RIGHT_TO_LEFT);
```
The `TextAlignmentType` и `TextDirectionType` Перечисления помогают настроить внешний вид ваших комментариев.

**Шаг 5: Настройте комментарий**
Определите содержание комментария:
```java
// Добавить примечание к комментарию
comment.setNote("This is my Comment Text. This is test");
```

### Сохранение вашей работы
После внесения изменений сохраните книгу с обновленными комментариями:
```java
// Определите выходной каталог и имя файла
```
