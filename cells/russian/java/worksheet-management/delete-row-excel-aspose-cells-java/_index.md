## Настройка Aspose.Cells для Java

Чтобы начать работу с Aspose.Cells, выполните следующие действия:

### Настройка Maven

Добавьте следующую зависимость к вашему `pom.xml` файл:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Настройка Gradle

Если вы используете Gradle, включите это в свой `build.gradle` файл:

```gradle

```

## Руководство по внедрению

Давайте разберем процесс удаления строк в листе Excel с помощью Aspose.Cells Java. Мы пойдем по шагам, чтобы обеспечить ясность.

### Создание экземпляра объекта Workbook

Начните с создания `Workbook` объект, представляющий ваш файл Excel:

```java
// Загрузите существующий файл Excel
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

Эта строка загружает ваш файл Excel в память, подготавливая его к обработке.

### Доступ к рабочему листу

Затем откройте рабочий лист, на котором вы хотите удалить строку:

```java
// Доступ к первому листу в файле Excel.
Worksheet worksheet = workbook.getWorksheets().get(0);
```

Здесь мы нацеливаемся на первый рабочий лист. Вы можете настроить это, если ваш целевой лист находится в другом месте.

### Удаление строк

Теперь давайте удалим определенные строки из рабочего листа:

```java
// Удалить 3-ю строку (индекс 2) и сместить ячейки вверх
worksheet.getCells().deleteRows(2, 1, true);
```

**Объяснение:**
- **`deleteRows(startIndex, totalRows, updateReference)`**: Этот метод удаляет строки, начиная с `startIndex`. Параметр `totalRows` указывает, сколько строк следует удалить. `updateReference` к `true` обеспечивает соответствующее обновление ссылок на ячейки.

### Сохранение измененного файла

Наконец, сохраните изменения:

```java
// Сохраните файл Excel с изменениями.
workbook.save(dataDir + "DeleteARow_out.xls");
```

На этом этапе все изменения записываются обратно в выходной файл, что сохраняет ваши изменения.
