## Руководство по внедрению

В этом разделе мы рассмотрим реализацию функции задания имени вкладки отдельного листа при экспорте файла Excel в формате HTML.

### Загрузка и настройка рабочей книги

Сначала загрузите книгу Excel, содержащую только один лист. Такая настройка обеспечивает ясность в экспортируемом HTML:

#### Загрузите рабочую тетрадь
```java
// Инициализируйте новый объект Workbook с помощью пути к исходному каталогу.
Workbook wb = new Workbook(srcDir + "sampleSingleSheet.xlsx");
```

### Настройка параметров сохранения HTML

Настройте `HtmlSaveOptions` для управления способом сохранения рабочей книги в виде HTML-файла.

#### Настроить HtmlSaveOptions
```java
HtmlSaveOptions options = new HtmlSaveOptions();

// Установите различные параметры экспорта для лучшей настройки вывода.
options.setEncoding(Encoding.getUTF8()); // Использовать кодировку UTF-8
options.setExportImagesAsBase64(true);   // Экспорт изображений в формате Base64
options.setExportGridLines(true);        // Включить линии сетки в вывод HTML
options.setExportSimilarBorderStyle(true);
options.setExportBogusRowData(true);     // Сохраните целостность данных, экспортируя фиктивные данные строк
options.setExcludeUnusedStyles(true);    // Исключите неиспользуемые стили CSS, чтобы уменьшить размер файла
options.setExportHiddenWorksheet(true);  // При необходимости экспортируйте скрытые рабочие листы.
```

#### Сохранить книгу как HTML

Наконец, сохраните книгу в формате HTML с указанными вами параметрами:

```java
// Определите выходной каталог и сохраните HTML-файл.
wb.save(outDir + "outputSampleSingleSheet.htm", options);
```

### Основные параметры конфигурации:
- **Кодировка:** Обеспечьте правильное представление символов, используя UTF-8.
- **Изображения Base64:** Встраивание изображений непосредственно в HTML помогает избежать внешних зависимостей.
- **Линии и стили сетки:** Они сохраняют визуальную структуру данных Excel в выходном HTML-формате.
