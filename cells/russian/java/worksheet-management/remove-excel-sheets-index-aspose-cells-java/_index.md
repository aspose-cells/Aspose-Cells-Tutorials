## Руководство по внедрению

### Удаление рабочего листа с помощью индекса листов
#### Обзор
Эта функция позволяет удалить определенный лист из книги Excel, указав его индекс, что идеально подходит для динамических наборов данных, где порядок и количество листов могут меняться.
#### Пошаговая реализация
##### 1. Настройте пути к файлам
Сначала определите каталоги для входных и выходных файлов:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";
```
##### 2. Откройте файл Excel из Stream
Используйте `FileInputStream` для чтения книги Excel:
```java
FileInputStream fstream = new FileInputStream(dataDir + "book.xls");
Workbook workbook = new Workbook(fstream);
```
*Почему?*: На этом этапе инициализируется объект рабочей книги, что позволяет вам управлять его содержимым.
##### 3. Удалить рабочий лист по индексу
Удалить рабочий лист с определенным индексом (например, первый лист с индексом `0`):
```java
workbook.getWorksheets().removeAt(0);
```
##### 4. Сохраните изменения.
Сохраните измененную книгу:
```java
workbook.save(outDir + "RWUsingSheetIndex_out.xls");
```
*Почему?*: Сохранение изменений имеет решающее значение для обеспечения сохранения ваших модификаций.
##### 5. Очистите ресурсы
Закройте файловый поток, чтобы освободить системные ресурсы:
```java
fstream.close();
```
#### Советы по устранению неполадок
- **Файл не найден**: Обеспечить пути в `dataDir` и `outDir` верны.
- **Индекс вне границ**: Проверьте индекс рабочего листа перед попыткой удаления.
