
### Функция 2: Создание рабочей книги и интеллектуальная обработка маркеров с помощью формул

Эта функция позволяет динамически создавать рабочие книги Excel, включая интеллектуальные маркеры для автоматизации вставки данных с использованием формул.

#### Обзор
Умные маркеры в Aspose.Cells для Java обеспечивают беспрепятственную интеграцию внешних данных в электронные таблицы, упрощая выполнение повторяющихся задач.

#### Этапы внедрения

**Шаг 1: Определите каталоги данных**
```java
String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";
```

**Шаг 2: Создание массива формул**
```java
String[] TestFormula = {
    "= \"01-This \" & \"is \" & \"concatenation\"",
    "= \"02-This \" & \"is \" & \"concatenation\"",
    "= \"03-This \" & \"is \" & \"concatenation\"",
    "= \"04-This \" & \"is \" & \"concatenation\"",
    "= \"05-This \" & \"is \" & \"concatenation\""
};
```

**Шаг 3: Инициализация рабочей книги и рабочего листа**
```java
Workbook wb = new Workbook();
Worksheet ws = wb.getWorksheets().get(0);
Cells cells = ws.getCells();
Cell cell = cells.get("A1");
cell.putValue("&=$Test(formula)");
```
- **Параметры:** Поле интеллектуального маркера `&=$Test(formula)` используется для указания места, куда следует вставить данные.
- **Конфигурация ключа:** Убедитесь, что ваши формулы правильно отформатированы для обработки Aspose.Cells.

**Шаг 4: Настройка WorkbookDesigner и обработка интеллектуальных маркеров**
```java
WorkbookDesigner wd = new WorkbookDesigner(wb);
wd.setDataSource("Test", TestFormula);
wd.process();
```

**Шаг 5: Сохраните рабочую книгу**

- **Возврат:** Обработанная рабочая книга сохранена в формате Excel.
