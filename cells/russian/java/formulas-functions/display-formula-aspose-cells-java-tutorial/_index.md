## Настройка Aspose.Cells для Java

Интеграция Aspose.Cells в ваш проект Java может быть легко выполнена с помощью Maven или Gradle. Вот как это настроить:

**Мейвен:**
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Градл:**
Включите это в свой `build.gradle` файл:
```gradle
implementation 'com.aspose:aspose-cells:25.3'
```

### Базовая инициализация и настройка
После добавления Aspose.Cells в проект инициализируйте его в своем приложении Java следующим образом:
```java
// Импорт необходимых классов из Aspose.Cells
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class ShowFormulas {
    public static void main(String[] args) throws Exception {
        // Определите путь, по которому находятся ваши файлы Excel.
        String dataDir = "path/to/your/excel/files/";

        // Загрузить существующую книгу с диска
        Workbook workbook = new Workbook(dataDir + "source.xlsx");
        
        // Доступ к первому рабочему листу в рабочей книге
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // Показать формулы на этом листе
        worksheet.setShowFormulas(true);
        
        // Сохраните изменения обратно в файл.
        workbook.save(dataDir + "ShowFormulas_out.xlsx");
    }
}
```

1. **Загрузите исходную рабочую книгу**: Начните с загрузки существующего файла Excel с помощью `Workbook`.
2. **Доступ к рабочему листу**:
   - Использовать `workbook.getWorksheets().get(0)` для доступа к первому рабочему листу.
3. **Формулы отображения**:
   - Вызов `worksheet.setShowFormulas(true);` для переключения отображения формул вместо их результатов.

### Сохранить изменения
После внесения изменений обязательно сохраните книгу, используя `workbook.save()`Этот шаг имеет решающее значение, поскольку он записывает все изменения обратно в файл Excel на диске.
