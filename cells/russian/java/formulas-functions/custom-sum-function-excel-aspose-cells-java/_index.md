## Руководство по внедрению

### Функция: Пользовательский механизм расчета
Эта функция позволяет вам изменять функции Excel, например: `SUM` работают в Aspose.Cells.

#### Обзор
Расширяя вычислительный движок, вы можете настроить поведение для определенных функций. В этом руководстве основное внимание уделяется изменению `SUM` функция для добавления дополнительного постоянного значения.

#### Пошаговая реализация
##### Расширение AbstractCalculationEngine
1. **Создать класс CustomEngine**
   Начните с создания класса, который расширяет `AbstractCalculationEngine`.
   
   ```java
   import com.aspose.cells.AbstractCalculationEngine;
   import com.aspose.cells.CalculationData;

   public class CustomEngine extends AbstractCalculationEngine {
       @Override
       public void calculate(CalculationData data) {
           // Проверьте, является ли вычисляемая функция «СУММ».
           if (data.getFunctionName().toUpperCase().equals("SUM")) {
               // Извлечь и изменить текущее расчетное значение.
               double val = (double) data.getCalculatedValue();
               val += 30;  // Добавление постоянного значения 30
               data.setCalculatedValue(val);
           }
       }
   }
   ```
2. **Объяснение параметров**
   - `data.getFunctionName()`: Возвращает имя вычисляемой функции.
   - `data.getCalculatedValue()`: Извлекает текущий вычисленный результат.
   - `data.setCalculatedValue(double)`: Обновляет данные расчета новым значением.
3. **Советы по устранению неполадок**
   Убедитесь, что имена методов и логика проверки функций нечувствительны к регистру, чтобы предотвратить возникновение ошибок во время выполнения.
