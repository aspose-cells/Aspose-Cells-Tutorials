### Руководство по внедрению

#### Поиск ячеек с помощью регулярных выражений

В этом разделе вы узнаете, как реализовать поиск ячеек на основе регулярных выражений в таблице Excel.

**Шаг 1: Доступ к рабочему листу**

Получите ссылку на рабочий лист, на котором вы хотите выполнить поиск:

```java
// Доступ к первому листу в файле Excel
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**Шаг 2: Настройте параметры поиска**

Настройте свой `FindOptions` для использования регулярных выражений для поиска по шаблону:

```java
// Создайте экземпляр FindOptions и установите RegexKey в значение true
FindOptions opt = new FindOptions();
opt.setRegexKey(true); // Позволяет искать по регулярным выражениям
opt.setLookAtType(LookAtType.ENTIRE_CONTENT);
```

**Шаг 3: Выполните поиск**

Используйте `find` метод с вашим шаблоном регулярного выражения:

```java
// Поиск ячеек, соответствующих указанному регулярному выражению
Cells cells = worksheet.getCells();
cells.find("abc[\\s]*$", null, opt); // Поиск «abc» с последующими пробелами в конце содержимого ячейки.
```

**Объяснение:**
- `setRegexKey(true)`: Активирует поиск по регулярному выражению.
- `LookAtType.ENTIRE_CONTENT`: Гарантирует, что во время поиска учитывается все содержимое ячейки, а не только ее части.
