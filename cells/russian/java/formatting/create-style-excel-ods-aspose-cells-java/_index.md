## Руководство по внедрению

### Функция: создание и настройка новой книги Excel
Эта функция позволяет вам создать новую книгу Excel, открыть ее первый лист и заполнить его данными.

#### Шаг 1: Создание нового экземпляра рабочей книги
Создать экземпляр `Workbook` которая представляет собой всю электронную таблицу:
```java
import com.aspose.cells.Workbook;
Workbook workbook = new Workbook();
```

#### Шаг 2: Откройте первый рабочий лист из рабочей книги.
Используйте `getWorksheets().get(0)` метод доступа к первому листу в вашей рабочей книге:
```java
import com.aspose.cells.Worksheet;
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### Шаг 3: Заполните рабочий лист данными
Выполните итерацию по ячейкам и установите значения, демонстрируя базовый ввод данных на рабочий лист:
```java
for (int i = 0; i < 6; i++) {
    // Установите значения в первом столбце
    worksheet.getCells().get(i, 0).setValue(i + 1);
    
    // Установите значения во втором столбце
    worksheet.getCells().get(i, 1).setValue(i + 7);
}
```

### Функция: настройка цвета фона страницы ODS
Эта функция позволяет вам установить цветной фон для страницы ODS с помощью Aspose.Cells.

#### Шаг 1: Создание нового экземпляра рабочей книги
Инициализируйте рабочую книгу, как это было сделано ранее:
```java
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### Шаг 2: Настройте цвет фона страницы ODS
Доступ `OdsPageBackground` и установите его цвет:
```java
import com.aspose.cells.OdsPageBackground;
import com.aspose.cells.Color;
import com.aspose.cells.OdsPageBackgroundType;

OdsPageBackground background = worksheet.getPageSetup().getODSPageBackground();
background.setColor(Color.getAzure());
background.setType(OdsPageBackgroundType.COLOR);
```

#### Шаг 3: Сохраните рабочую книгу в формате ODS
Укажите выходной каталог и сохраните книгу:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "/ColoredBackground.ods", com.aspose.cells.SaveFormat.ODS);
```
