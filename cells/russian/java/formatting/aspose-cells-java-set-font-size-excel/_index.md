## Руководство по внедрению

Теперь давайте рассмотрим, как можно задать размер шрифта в ячейке Excel с помощью Aspose.Cells для Java.

### Создание рабочей книги и доступ к ячейкам
**Обзор:**
Начните с создания экземпляра `Workbook` объект. Затем перейдите на рабочий лист, где вы хотите изменить размер шрифта.
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class SetFontSize {
    public static void main(String[] args) throws Exception {
        // Создать экземпляр объекта Workbook
        Workbook workbook = new Workbook();
        
        // Доступ к добавленному листу в файле Excel
        int sheetIndex = workbook.getWorksheets().add();
        Worksheet worksheet = workbook.getWorksheets().get(sheetIndex);
    }
}
```

### Установка размера шрифта
**Обзор:**
Измените размер шрифта определенной ячейки, открыв и изменив ее `Style`.
```java
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;
import com.aspose.cells.Style;
import com.aspose.cells.Font;

public class SetFontSize {
    public static void main(String[] args) throws Exception {
        Workbook workbook = new Workbook();
        int sheetIndex = workbook.getWorksheets().add();
        Worksheet worksheet = workbook.getWorksheets().get(sheetIndex);
        Cells cells = worksheet.getCells();

        // Получить доступ к ячейке и установить ее значение
        Cell cell = cells.get("A1");
        cell.setValue("Hello Aspose!");

        // Извлечь и изменить стиль ячейки, чтобы настроить размер шрифта.
        Style style = cell.getStyle();
        Font font = style.getFont();
        font.setSize(14);  // Установите желаемый размер шрифта
        cell.setStyle(style);

        // Сохраните измененную книгу.
        String dataDir = "path/to/save/";
        workbook.save(dataDir + "SetFontSize_out.xls");
    }
}
```
**Объяснение:**
- **`Font.setFontSize(int size)`**: Устанавливает размер шрифта. Здесь мы используем `14`, но вы можете выбрать любое другое целочисленное значение.
- **Сохранение рабочей книги**: `workbook.save()` метод записывает изменения в файл в вашей системе.
