## Руководство по внедрению
### Обзор пользовательского форматирования десятичных чисел
Форматирование пользовательских десятичных чисел позволяет вам контролировать, как числа отображаются в вашем приложении, следуя числовому формату Excel. Эта функция имеет решающее значение для приложений, требующих точного представления данных.

#### Пошаговая реализация
**1. Создайте рабочую книгу и рабочий лист Access**
Сначала создайте экземпляр `Workbook` класс и откройте его первый рабочий лист:
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class Main {
    public static void main(String[] args) throws Exception {
        // Инициализировать книгу
        Workbook wb = new Workbook();
        
        // Доступ к первому рабочему листу
        Worksheet ws = wb.getWorksheets().get(0);
    }
}
```
**2. Вставьте и отформатируйте значение**
Вставьте значение в ячейку A1 и примените пользовательский формат десятичных чисел:
```java
import com.aspose.cells.Cell;
import com.aspose.cells.Style;

public class Main {
    public static void main(String[] args) throws Exception {
        // Предыдущий код...

        // Откройте ячейку A1 и введите значение 123.
        Cell cell = ws.getCells().get("A1");
        cell.putValue(123);

        // Применить пользовательское форматирование десятичных чисел
        Style st = cell.getStyle();
        st.setCustom("[DBNum2][$-804]General");

        // Установить форматированный стиль для ячейки
        cell.setStyle(st);
    }
}
```
**3. Сохранить книгу в формате PDF**
Наконец, сохраните свою рабочую книгу в формате PDF:
```java
import com.aspose.cells.SaveFormat;

public class Main {
    public static void main(String[] args) throws Exception {
        // Предыдущий код...

        // Установите ширину столбца и сохраните книгу.
        ws.getCells().setColumnWidth(0, 30);
        wb.save("outputDecimalNumberFormatting.pdf", SaveFormat.PDF);

        System.out.println("Custom Decimal Number Formatting Done Successfully");
    }
}
```
**Советы по устранению неполадок**
