## Руководство по внедрению

### Объединение ячеек

Объединение ячеек объединяет несколько смежных ячеек в одну, что полезно для создания заголовков или эффективной организации данных. Вот как это сделать с помощью Aspose.Cells.

#### Пошаговый процесс:
**1. Создайте новую рабочую книгу:**
Начните с создания экземпляра `Workbook` класс, представляющий ваш файл Excel.
```java
// Инициализировать рабочую книгу
Workbook workbook = new Workbook();
```

**2. Откройте рабочий лист:**
Для выполнения операций перейдите к первому листу рабочей книги.
```java
// Получить первый рабочий лист
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**3. Определите диапазон ячеек:**
Укажите диапазон, который вы хотите объединить, например `A1:D4`.
```java
// Создать диапазон ячеек
Range range = worksheet.getCells().createRange("A1:D4");
```

**4. Объедините определенный диапазон:**
Вызовите `merge()` метод на определенном диапазоне для объединения ячеек.
```java
// Объединить диапазон в одну ячейку
range.merge();
```

**5. Сохраните рабочую книгу:**
Сохраните изменения, указав выходной каталог и имя файла.
```java
// Укажите выходной каталог
String outDir = "YOUR_OUTPUT_DIRECTORY";

// Сохраните рабочую книгу
workbook.save(outDir + "MURangeofCells_out.xlsx");
```

### Разделение ячеек

Разъединение ячеек важно, особенно когда вам нужно отменить изменения или скорректировать макеты данных. Выполните следующие шаги, чтобы разъединить ранее объединенные ячейки.

#### Пошаговый процесс:
**1. Загрузите рабочую книгу:**
Загрузите существующую книгу, содержащую объединенный диапазон ячеек.
```java
// Загрузите книгу с объединенными ячейками
Workbook workbook = new Workbook(outDir + "MURangeofCells_out.xlsx");
```

**2. Снова откройте рабочий лист:**
Повторно откройте первый рабочий лист, чтобы выполнить операции по разделению.
```java
// Получить первый рабочий лист
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**3. Определите тот же диапазон ячеек:**
Снова укажите диапазон, который вы ранее объединили.
```java
// Создать диапазон ячеек
Range range = worksheet.getCells().createRange("A1:D4");
```

**4. Разделить диапазон:**
Позвоните `unMerge()` Метод, позволяющий вернуть клетки в исходное состояние.
```java
// Разъединить диапазон
range.unMerge();
```

**5. Сохраните изменения:**
Сохраните книгу с необъединенными ячейками.
```java
// Сохраните книгу с необъединенными изменениями.
workbook.save(outDir + "UnMURangeofCells_out.xlsx");
```
