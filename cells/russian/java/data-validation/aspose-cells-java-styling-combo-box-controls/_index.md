### Создание и стилизация ячейки с помощью Aspose.Cells Java
**Обзор:**
В этом разделе показано, как создать новую ячейку на листе Excel, ввести текст и применить полужирное начертание с помощью Aspose.Cells для Java.
#### Шаг 1: Инициализация рабочей книги и рабочего листа
```java
Workbook workbook = new Workbook();
Worksheet sheet = workbook.getWorksheets().get(0);
Cells cells = sheet.getCells();
```
*Объяснение:* Начнем с создания `Workbook` экземпляр, представляющий файл Excel. Затем мы получаем доступ к первому листу и его коллекции ячеек.
#### Шаг 2: Ввод данных и применение стиля
```java
cells.get("B3").setValue("Employee:");
Style style = cells.get("B3").getStyle();
style.getFont().setBold(true);
cells.get("B3").setStyle(style);
```
*Объяснение:* Здесь мы вводим текст "Сотрудник:" в ячейку B3. Затем мы извлекаем и изменяем его `Style` объект, чтобы установить шрифт жирным.
#### Шаг 3: Сохраните книгу
```java
workbook.save(outDir + "CreateAndStyleCell_out.xls");
```
*Объяснение:* Наконец, мы сохраняем книгу в указанном каталоге вместе с нашими изменениями.

ShapeCollection shapes = sheet.getShapes();
ComboBox comboBox = (ComboBox) shapes.addShape(MsoDrawingType.COMBO_BOX, 3, 0, 1, 0, 20, 100);
