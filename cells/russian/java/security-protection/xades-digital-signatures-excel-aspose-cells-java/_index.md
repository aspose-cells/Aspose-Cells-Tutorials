## Настройка Aspose.Cells для Java

Aspose.Cells — это основа нашей реализации. Давайте настроим ее.

**Зависимость Maven**

Чтобы интегрировать Aspose.Cells с помощью Maven, добавьте это в свой `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Зависимость Gradle**

Для пользователей Gradle включите следующее в свой `build.gradle`:

```gradle
implementation 'com.aspose:aspose-cells:25.3'
```

### Этапы получения лицензии

Aspose.Cells предлагает различные варианты лицензирования:
- **Бесплатная пробная версия**: Начните с 30-дневной бесплатной пробной версии, чтобы протестировать все возможности.
- **Временная лицензия**: При необходимости получите временную лицензию для расширенной оценки.
- **Покупка**: Для долгосрочного использования рассмотрите возможность приобретения лицензии.

Получив файл лицензии, инициализируйте Aspose.Cells следующим образом:

```java
com.aspose.cells.License license = new com.aspose.cells.License();
license.setLicense("path/to/your/license/file.lic");
```

## Руководство по внедрению

### Добавить подпись XAdES в файл Excel

В этом разделе мы рассмотрим шаги по добавлению цифровой подписи XAdES в книгу Excel.

#### Шаг 1: Загрузите свою рабочую тетрадь и сертификат

Сначала загрузите файл Excel и подготовьте сертификат для подписания:

```java
// Определить каталоги и пути
double sourceDir = Utils.Get_SourceDirectory();
double outputDir = Utils.Get_OutputDirectory();

Workbook workbook = new Workbook(sourceDir + "sourceFile.xlsx");
String password = "pfxPassword";
String pfxPath = sourceDir + "pfxFile.pfx";

InputStream inStream = new FileInputStream(pfxPath);
java.security.KeyStore inputKeyStore = java.security.KeyStore.getInstance("PKCS12");
inputKeyStore.load(inStream, password.toCharArray());
```

Здесь мы загружаем файл Excel (`sourceFile.xlsx`) и сертификат PKCS#12 (`pfxFile.pfx`). `password` используется для разблокировки вашего сертификата.

#### Шаг 2: Создание и настройка цифровой подписи

Теперь давайте создадим цифровую подпись:

```java
digitalSignature = new DigitalSignature(inputKeyStore, password, "testXAdES", com.aspose.cells.DateTime.getNow());
signature.setXAdESType(XAdESType.X_AD_ES);
```

The `DigitalSignature` Объект инициализируется с вашим KeyStore и временной меткой. Метод `setXAdESType` настраивает подпись в соответствии со стандартами XAdES.

#### Шаг 3: Добавьте подпись в рабочую книгу

Наконец, добавьте цифровую подпись в книгу:

```java
digitalSignatureCollection = new DigitalSignatureCollection();
digitalSignatureCollection.add(signature);
workbook.setDigitalSignature(digitalSignatureCollection);

// Сохраните подписанный файл Excel.
workbook.save(outputDir + "XAdESSignatureSupport_out.xlsx");
```

The `DigitalSignatureCollection` содержит нашу подпись, которая затем связывается с рабочей книгой с помощью `setDigitalSignature`.

### Советы по устранению неполадок
- **Проблемы с сертификатами**: Убедитесь, что путь к сертификату и пароль верны.
- **Сохранить ошибки пути**: Убедитесь, что у вас есть права на запись в выходной каталог.
