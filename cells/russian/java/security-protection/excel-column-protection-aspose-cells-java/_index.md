# Освоение защиты столбцов Excel с помощью Aspose.Cells для Java

В этом руководстве вы узнаете, как разблокировать все столбцы Excel, заблокировать определенные столбцы и защитить рабочие листы.

**Что вы узнаете:**
- Как разблокировать все столбцы в таблице Excel с помощью Aspose.Cells.

## Руководство по внедрению

В этом руководстве реализация разделена на разделы по функциям: разблокировка столбцов, блокировка определенных столбцов и защита рабочих листов.

### Разблокировать все столбцы в Excel

Разблокировка столбцов позволяет пользователям свободно редактировать данные по всему рабочему листу.

#### Обзор
Следующий код перебирает все столбцы (до 255) и разблокирует их:

```java
// Создайте новую рабочую книгу.
Workbook wb = new Workbook();
// Возьмите первый лист из рабочей тетради.
Worksheet sheet = wb.getWorksheets().get(0);

// Определите объекты стиля и флага стиля.
Style style;
StyleFlag flag;

// Пройдитесь по всем столбцам и разблокируйте их.
for (int i = 0; i <= 255; i++) {
    // Получить текущий стиль столбца.
    style = sheet.getCells().getColumns().get(i).getStyle();
    // Для разблокировки установите свойство locked в значение false.
    style.setLocked(false);
    flag = new StyleFlag();
    flag.setLocked(true);
    // Примените разблокированный стиль обратно к столбцу.
    sheet.getCells().getColumns().get(i).applyStyle(style, flag);
}

// Сохраните изменения во временном файле.
wb.save(dataDir + "TempUnlockColumns_out.xls");
```

**Объяснение:**
- **Стиль и СтильФлаг**: Объекты, определяющие визуальные и поведенческие свойства столбцов.
- **Цикл**: Итерирует по каждому столбцу, чтобы настроить заблокированный статус.

### Заблокировать первый столбец

Блокировка определенного столбца может защитить важные данные от изменения пользователями.

#### Обзор
Этот фрагмент блокирует только первый столбец на вашем листе:

```java
// Создайте новую рабочую книгу.
Workbook wb = new Workbook();
// Возьмите первый лист из рабочей тетради.
Worksheet sheet = wb.getWorksheets().get(0);

// Получить стиль первого столбца и заблокировать его.
Style style = sheet.getCells().getColumns().get(0).getStyle();
style.setLocked(true);
StyleFlag flag = new StyleFlag();
flag.setLocked(true);

// Примените заблокированный стиль к первому столбцу.
sheet.getCells().getColumns().get(0).applyStyle(style, flag);

// Сохраните изменения во временном файле.
wb.save(dataDir + "TempLockFirstColumn_out.xls");
```

**Объяснение:**
- **Закрытая собственность**: Установить на `true` чтобы предотвратить любые правки.

### Защитить рабочий лист

Защита всего рабочего листа не позволяет пользователям вносить изменения без разрешения.

#### Обзор
Чтобы защитить весь рабочий лист, используйте:

```java
// Создайте новую рабочую книгу.
Workbook wb = new Workbook();
// Возьмите первый лист из рабочей тетради.
Worksheet sheet = wb.getWorksheets().get(0);

// Защитите рабочий лист всеми типами защиты.
sheet.protect(ProtectionType.ALL);

// Сохраните последнюю защищенную книгу.
wb.save(dataDir + "PColumnWorksheet_out.xls");
```

**Объяснение:**
- **ProtectionType.ALL**: Обеспечивает максимальную безопасность, отключая все параметры редактирования.
