## Sık Sorulan Sorular

**Q: Can I modify chart data labels after the chart is created?**  
A: Yes, you can access the chart’s `DataLabels` collection and update text, position, or formatting at any time. → **Soru: Grafik oluşturulduktan sonra veri etiketlerini değiştirebilir miyim?**  
Cevap: Evet, grafiğin `DataLabels` koleksiyonuna erişerek metni, konumu veya biçimlendirmeyi istediğiniz zaman güncelleyebilirsiniz.

**Q: How do I set custom legend text for a chart?**  
A: Use `chart.getLegend().setText("Your Legend Text")` after creating the chart object. → **Soru: Bir grafik için özel lejant metni nasıl ayarlarım?**  
Cevap: Grafik nesnesini oluşturduktan sonra `chart.getLegend().setText("Your Legend Text")` metodunu kullanın.

**Q: Is it possible to apply a custom theme color palette to all charts in a workbook?**  
A: Absolutely. Load or create a `Theme` object, modify its color scheme, and assign it to the workbook before adding charts. → **Soru: Çalışma kitabındaki tüm grafiklere özel bir tema renk paleti uygulamak mümkün mü?**  
Cevap: Kesinlikle. Bir `Theme` nesnesi yükleyin veya oluşturun, renk şemasını değiştirin ve grafikleri eklemeden önce çalışma kitabına atayın.

**Q: Do I need a license to export charts as SVG or PDF?**  
A: A trial works for evaluation, but a commercial license is required for production‑grade exports. → **Soru: Grafiklerin SVG veya PDF olarak dışa aktarılması için lisansa ihtiyacım var mı?**  
Cevap: Değerlendirme için deneme sürümü yeterli, ancak üretim‑seviyesinde dışa aktarmalar için ticari bir lisans gereklidir.

**Q: Which chart types are supported by Aspose.Cells for Java?**  
A: Over 50 chart types, including column, line, pie, bubble, radar, stock, and pivot charts. → **Soru: Aspose.Cells for Java hangi grafik türlerini destekliyor?**  
Cevap: Sütun, çizgi, pasta, balon, radar, hisse senedi ve pivot grafikler dahil olmak üzere 50’den fazla grafik türü desteklenir.
