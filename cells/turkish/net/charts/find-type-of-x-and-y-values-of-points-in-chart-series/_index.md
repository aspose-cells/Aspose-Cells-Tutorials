## Kurulum talimatları  

Projeinizde .NET için Aspose.Cells kullanmak için, bu adımları takip edin:  

1. NuGet Paket Yöneticisi üzerinden yükle  
- Visual Studio'yu açın ve Araçlara git → NuGet Paket Yöneticisi → Solution için NuGet Paketleri Yönetin.  
- Aspose.Cells için arama yapın ve Kurulum'u tıklayın.  

Ya da NuGet Paket Yöneticisi Konsolu kullanarak yükleyin:  

```powershell
Install-Package Aspose.Cells
```  

2. .NET CLI üzerinden yükle  
.NET CLI kullanıyorsanız:  

```powershell
dotnet add package Aspose.Cells
```  

3. Projeye İpucu Ekle  
Once installed, include the following namespace in your C# file:  

```csharp
using Aspose.Cells;
using Aspose.Cells.Charts;
```
