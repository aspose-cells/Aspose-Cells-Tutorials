## Kurulum Talimatları:  
Bu örneği kullanmak için şu adımları izleyin:  

1. NuGet aracılığıyla .NET için Aspose.Cells'i yükleyin:  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. C# projenize gerekli ad alanını ekleyin:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Verilen kodu çalıştırarak bir ODS dosyasını yükleyin, bir grafiğe erişin ve alt yazısını konsola yazdırın.
