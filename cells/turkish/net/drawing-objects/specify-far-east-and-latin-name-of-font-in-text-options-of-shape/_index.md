## Kurulum talimatları
1. .NET için Aspose.Cells yükle: 
   * Visual Studio projenizdeki NuGet Paket Yöneticisi kullanın:
     ```
     Install-Package Aspose.Cells
     ```
   * Ya da DLL'i [Web sitesini hedefleme](https://www.aspose.com/downloads/cells/net)\- Evet .

2. İpuçları Ekle: Projenizin Aspose.Cells kütüphanesine ilişkin referanslarını sağlayın.

3. Çıktı Dizini Oluştur: `Çıkış`Excel dosyasının kaydedileceği istenen çıkış yolu ile koddaki değişken.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
