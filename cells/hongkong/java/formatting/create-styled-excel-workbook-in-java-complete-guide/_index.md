# 在 Java 中建立具樣式的 Excel 工作簿 – 完整指南

是否曾需要從 Java **create styled excel workbook**（建立具樣式的 Excel 工作簿），卻被格式細節搞得頭疼？你並不孤單。無論是從資料庫提取資料或匯出報告，想要取得正確的外觀——貨幣符號、日期格式、粗體標題——都可能感覺像是一項繁瑣的工作。  

在本教學中，我們將逐步說明一個實用的解決方案，該方案 **adds custom formatting to excel**（為 Excel 添加自訂格式），讓你 **set date format excel column**（設定 Excel 欄位的日期格式），最後 **save workbook as xlsx**（將工作簿儲存為 xlsx）而不費吹灰之力。準備好看看實際效果了嗎？

## 你將學到什麼

- 如何從頭開始建立 Aspose.Cells `Workbook`。  
- 如何套用涵蓋 General、Currency 與 Date 欄位的樣式陣列。  
- 精確的呼叫方式，可 **creates styled excel workbook**（建立具樣式的 Excel 工作簿）並內嵌欄位名稱。  
- 如何使用 **save workbook as xlsx** 方法將檔案永久保存。  

不需要外部設定檔，也不需要晦澀的 XML 技巧——只要純粹的 Java 程式碼，你可以直接複製貼上、執行並自行調整。

### 前置條件

- Java 8 以上（程式碼在任何近期的 JDK 都可執行）。  
- Aspose.Cells for Java 函式庫（可從 Maven Central 取得最新的 JAR）。  
- 對集合（`List`、`Map`）有一點基本了解。  

如果上述任一項你不熟悉，請先暫停一下並安裝 Aspose.Cells JAR；其餘部分自然就能跟上。

## 建立具樣式的 Excel 工作簿 – 專案設定

首先，建立一個新的 Maven（或 Gradle）專案，並將 Aspose.Cells 加入為相依性。

```xml
<!-- pom.xml snippet -->
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>23.12</version> <!-- use the latest version -->
</dependency>
```

> **專業提示：** 若使用 Gradle，請使用相同的座標 `implementation 'com.aspose:aspose-cells:23.12'`。

為什麼要從 Maven 設定開始？它會自動拉入所有傳遞相依性，讓你不必再花時間去尋找正確的 JAR 檔案。

## 為 Excel 欄位添加自訂格式

專案已就緒後，讓我們定義所需的樣式。在 Aspose.Cells 中，你可以傳入一個格式字串陣列，直接對應到每一個欄位。

```java
// Define column styles: General, Currency, Date
String[] columnStyles = {
    "General",          // Column A – plain text or numbers
    "$#,##0.00",        // Column B – currency with two decimals
    "mm/dd/yyyy"        // Column C – US‑style date
};
```

**add custom formatting to excel** 步驟只需要建立此字串陣列即可。  
每個條目遵循 Excel「自訂」格式對話框中使用的相同模式，因此無需學習新語法。

## 設定 Excel 欄位的日期格式

如果資料中包含日期，你會希望它們顯示一致。`columnStyles` 中的第三個條目（`"mm/dd/yyyy"`）正是如此——它告訴 Aspose.Cells 將第三欄視為日期，並以月/日/年的順序顯示。

```java
// Example: Adding a row with a Date object
Map<String, Object> row = new HashMap<>();
row.put("Product", "Widget");
row.put("Price", 19.99);
row.put("ReleaseDate", new java.util.Date()); // Today’s date
dataRows.add(row);
```

當工作簿產生時，**set date format excel column** 指令會確保每個 `ReleaseDate` 儲存格顯示為 `04/27/2026`（或今天的日期），不需要後續處理。

## 匯入資料並套用樣式

樣式已定義後，下一步是將資料寫入工作表。Aspose.Cells 提供 `importCustomObjects`，不僅插入資料，還會套用剛才建立的樣式陣列。

```java
// Step 1: Simulate fetching rows from a DB
List<Map<String, Object>> dataRows = fetchRows(); // see helper below

// Step 2: Create a fresh workbook and grab the first sheet
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);

// Step 3: Import data with custom styles; true = first row has headers
worksheet.getCells().importCustomObjects(
        dataRows,        // data source
        0,               // start row (0‑based)
        0,               // start column (0‑based)
        columnStyles,    // style array from above
        true);           // first row contains column names
```

### 為什麼使用 `importCustomObjects`？

- **Single‑call magic**：一次呼叫即可寫入資料 *並* 套用格式。  
- **Header awareness**：傳入 `true` 會告訴 API 將第一個 Map 的鍵視為欄位標題，這在 **create styled excel workbook** 場景中非常適合需要可讀欄位名稱的情況。  
- **Performance**：在底層 Aspose 會批次寫入操作，比逐格迴圈更快。  

> **注意：** 每個 `Map` 中鍵的順序很重要。若需要確定的欄位順序，請使用 `LinkedHashMap`。

## 將工作簿儲存為 XLSX 檔案

如果無法將檔案永久保存，前面的工作就毫無意義。最後一行是一行程式碼即可 **save workbook as xlsx**。

```java
// Step 5: Persist the workbook
String outputPath = "output/DataTableStyled.xlsx";
workbook.save(outputPath);
System.out.println("Workbook saved to " + outputPath);
```

`save` 方法會自動偵測檔案副檔名。指定 `.xlsx` 後，即可確保使用現代的 Office Open XML 格式，這也是大多數下游工具所期望的。

## 完整、可執行範例

以下是完整的類別程式碼，你可以直接複製到 IDE 中。它包含一個模擬的 `fetchRows` 方法，回傳範例資料，無需真實資料庫即可測試。

```java
import com.aspose.cells.*;
import java.util.*;

public class ImportDataTableWithStyle {
    public static void main(String[] args) throws Exception {
        // ---------- Step 1: Get data ----------
        List<Map<String, Object>> dataRows = fetchRows();

        // ---------- Step 2: Create workbook ----------
        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // ---------- Step 3: Define column styles ----------
        String[] columnStyles = { "General", "$#,##0.00", "mm/dd/yyyy" };

        // ---------- Step 4: Import data with styles ----------
        // true = first row contains column names
        worksheet.getCells().importCustomObjects(dataRows, 0, 0, columnStyles, true);

        // ---------- Step 5: Save as XLSX ----------
        String outPath = "output/DataTableStyled.xlsx";
        workbook.save(outPath);
        System.out.println("Workbook saved to " + outPath);
    }

    // Mock method – replace with real DB call if needed
    private static List<Map<String, Object>> fetchRows() {
        List<Map<String, Object>> rows = new ArrayList<>();

        Map<String, Object> row1 = new LinkedHashMap<>();
        row1.put("Product", "Gadget");
        row1.put("Price", 49.95);
        row1.put("ReleaseDate", new Date()); // today
        rows.add(row1);

        Map<String, Object> row2 = new LinkedHashMap<>();
        row2.put("Product", "Widget");
        row2.put("Price", 19.99);
        Calendar cal = Calendar.getInstance();
        cal.set(2025, Calendar.DECEMBER, 15);
        row2.put("ReleaseDate", cal.getTime());
        rows.add(row2);

        return rows;
    }
}
```

執行它，開啟產生的檔案，你會看到預期的 **create styled excel workbook** 結果。

## 常見問題與邊緣情況

**如果需要超過三個欄位該怎麼辦？**  
只要延伸 `columnStyles` 陣列——每個條目依序對應到欄位。對於不想變更的欄位，使用 `"General"` 即可。

**我可以套用字型或顏色嗎？**  
可以。你可以建立 `Style` 物件，指定給 `StyleFlag`，再將 `Style[]` 陣列傳入 `importCustomObjects`。雖然較為進階，但原理相同。
