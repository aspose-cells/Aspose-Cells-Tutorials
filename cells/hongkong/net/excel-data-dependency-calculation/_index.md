# Excel 資料依賴與計算


## 介紹
Excel 自動化最重要的方面之一是了解資料依賴性——一個單元格的值如何受到其他單元格的影響。 Aspose.Cells for .NET 提供了出色的功能來處理這些場景。無論您處理的是簡單計算還是複雜的資料驅動依賴關係，這些教學課程都可以為您提供協助。

## [在 Excel 中追蹤依賴儲存格](./tracing-dependent-cells-in-excel/)
