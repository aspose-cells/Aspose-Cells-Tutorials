# Excel資料依賴與計算


## 介紹
Excel 自動化最重要的方面之一是了解資料依賴性——一個單元格的值如何受到其他單元格的影響。 Aspose.Cells for .NET 提供了處理這些場景的出色功能。無論您正在處理簡單的計算還是複雜的資料驅動依賴項，這些教學課程都可以為您提供協助。
