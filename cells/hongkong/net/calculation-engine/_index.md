# 適用於 .NET 應用程式的 Excel 計算引擎

## 概述

Aspose.Cells for .NET 提供了一個強大的計算引擎，讓開發人員可以處理 Excel 電子表格中的複雜公式計算。這個強大的元件使您能夠配置計算行為，實現自訂計算邏輯，並優化大型工作簿的效能。

## 主要特點

- **自訂計算引擎**：針對專門的公式實現自己的計算邏輯
- **計算監控**：追蹤和控制公式計算以實現效能最佳化
- **遞歸計算選項**：配置如何處理循環引用
- **日期系統配置**：在 1900 和 1904 日期系統之間切換
- **循環引用檢測**：識別並解決公式中的循環依賴關係
- **效能最佳化**：微調大型工作簿的計算設置
