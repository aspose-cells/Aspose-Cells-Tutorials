## 安裝說明
若要開始使用 Aspose.Cells for .NET，請依照下列步驟操作：  

1. 從官方儲存庫下載 Aspose.Cells for .NET： [Aspose.Cells 下載](https://releases.aspose.com/cells/net/)  
2. 透過 NuGet 套件管理器安裝：  
   - 開啟 Visual Studio  
   - 在程式包管理器控制台中執行以下命令：  
     ```powershell
     Install-Package Aspose.Cells
     ```
3. 在您的 C# 專案中包含 Aspose.Cells 命名空間：  
   ```csharp
   using Aspose.Cells;
   ```  
4. 使用提供的範例開始編碼以建立並儲存 ODS 格式的資料透視表。
