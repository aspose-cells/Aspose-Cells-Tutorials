## 設置說明
請使用以下步:  

1. 請從官方資料庫下載Aspose.Cells[請使用Aspose.Cells下載](https://releases.aspose.com/cells/net/)  
2. 透過NuGet包管理器安裝:  
   * 打開視覺工作室  
   * 在包管理器控制台執行下列命令:  
     ```powershell
     Install-Package Aspose.Cells
     ```
3. 請將Aspose.Cells命名空間加入您的C#項目:  
   ```csharp
   using Aspose.Cells;
   ```  
4. 開始使用所提供的示例編碼, 以創建和保存 ODS 格式的 PivotTable.
