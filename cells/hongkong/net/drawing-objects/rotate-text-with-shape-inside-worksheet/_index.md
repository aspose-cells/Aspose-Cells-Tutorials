## 安裝說明：  

若要在您的專案中使用 Aspose.Cells for .NET，請依照下列步驟操作：  

1. 透過 NuGet 套件管理器安裝：  
   - 開啟 Visual Studio。  
   - 前往工具 → NuGet 套件管理器 → 管理解決方案的 NuGet 套件。  
   - 搜尋 Aspose.Cells 並安裝最新版本。  

2. 透過套件管理器控制台安裝：  
   - 在 Visual Studio 中開啟套件管理器控制台。  
   - 運行以下命令：  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. 從 Aspose 網站下載：  
   - 從下載最新版本 [Aspose.Cells for .NET 下載](https://releases。aspose.com/cells/net/).  
   - 將 DLL 新增至您的專案引用。
