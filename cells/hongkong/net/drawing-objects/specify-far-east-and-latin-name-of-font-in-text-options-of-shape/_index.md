## 設置說明
1. 設置Aspose.Cells於 .NET: 
   * 在您的 Visual Studio 項目中使用 NuGet 包管理器:
     ```
     Install-Package Aspose.Cells
     ```
   * 或是從[網站網站](https://www.aspose.com/downloads/cells/net)我們都在等.

2. 請確保您的項目引用Aspose.Cells圖書館.

3. 設定輸出目錄: 更新`輸出方向`該變量包含 Excel 檔案將會保存的所需輸出路徑.

4. 執行示例:將提供 C# 代碼複製到您的項目中,編譯並執行.您應該看到輸出確認,並在指定目錄中創建配置文本框的 Excel 檔案.
