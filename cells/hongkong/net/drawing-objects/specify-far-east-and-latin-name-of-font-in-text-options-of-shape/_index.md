# 使用遠東和拉丁字體建立 Excel 工作簿

此範例示範如何使用 Aspose.Cells for .NET 建立 Excel 工作簿、為工作表新增文字方塊以及為該文字方塊中的文字設定遠東和拉丁字體名稱。該程式碼展示了初始化工作區、操作工作表元素以及使用針對不同語言要求自訂的特定字體格式儲存工作簿。

## 安裝說明
1. 安裝 Aspose.Cells for .NET： 
   - 在 Visual Studio 專案中使用 NuGet 套件管理器：
     ```
     Install-Package Aspose.Cells
     ```
   - 或者從 [Aspose 網站](https://www。aspose.com/downloads/cells/net).

2. 新增引用：確保您的專案引用 Aspose.Cells 庫。

3. 設定輸出目錄：更新 `outputDir` 程式碼中的變數具有保存 Excel 檔案的所需輸出路徑。

4. 運行範例：將提供的 C# 程式碼複製到您的專案中，編譯並執行它。您應該會看到輸出確認，並且將在指定目錄中建立具有配置的文字方塊的 Excel 檔案。
