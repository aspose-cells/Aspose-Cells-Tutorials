## 安裝說明
1. 安裝 Aspose.Cells for .NET：使用 NuGet 套件管理器在您的專案中安裝 Aspose.Cells 函式庫。
   ```
   Install-Package Aspose.Cells
   ```
2. 新增使用指令：在程式碼檔案中包含必要的命名空間。
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. 新增範例程式碼：將提供的程式碼複製並貼上到您的專案中，確保指定您的文件目錄路徑。
4. 運行程式碼：執行您的專案以存取和修改 Excel 檔案中的 Ole 物件標籤。檢查控制台是否有輸出訊息確認該程序已成功執行。
