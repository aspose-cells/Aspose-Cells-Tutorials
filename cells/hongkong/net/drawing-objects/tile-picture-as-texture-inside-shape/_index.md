## 安裝說明
1. 確保您的機器上已設定 .NET 環境。
2. 從 Aspose 官方網站或使用 NuGet 套件管理器下載並安裝 Aspose.Cells 庫：
   ```
   Install-Package Aspose.Cells
   ```
3. 建立一個新的 C# 專案或開啟現有的 .NET 專案。
4. 在您的專案中新增對 Aspose.Cells 函式庫的參考。
5. 將提供的範例程式碼複製到您的專案中，根據需要替換輸入/輸出目錄路徑。
6. 編譯並執行您的專案以查看套用於 Excel 檔案中的形狀的平鋪紋理填充。
