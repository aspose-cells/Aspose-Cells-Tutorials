## 設置說明
1. 確保您在電腦上設定了.
2. 請下載並安裝Aspose.Cells庫從官方網站下載或使用NuGet包管理器:
   ```
   Install-Package Aspose.Cells
   ```

3. 創建新的C#項目或開啟現有的.
4. 請加入您的項目中的Aspose.Cells圖書館參考.

6. 編輯並執行您的項目, 以看到在 Excel 檔案中的形狀應用質感填充.
