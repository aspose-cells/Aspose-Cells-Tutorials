## 安裝說明

若要開始使用 Aspose.Cells for .NET，請依照下列安裝步驟操作：  

1. 透過 NuGet 套件管理器安裝：  
   - 開啟 Visual Studio。  
   - 前往工具>NuGet 套件管理器>管理解決方案的 NuGet 套件。  
   - 搜尋 `Aspose.Cells` 並點選“安裝”。  

2. 透過 NuGet 套件管理器控制台安裝：  
   - 從「工具」>「NuGet 套件管理器」>「套件管理器控制台」開啟「套件管理器控制台」。  
   - 運行以下命令：  
     ```powershell
     Install-Package Aspose.Cells
     ```  

3. 手動下載 DLL：  
   - 訪問 [Aspose.Cells下載頁面](https://releases。aspose.com/cells/net/).  
   - 下載並解壓縮該套件。  
   - 新增引用 `Aspose.Cells.dll` 在你的專案中。
