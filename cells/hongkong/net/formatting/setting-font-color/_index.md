## 安裝說明  
若要使用 Aspose.Cells for .NET，請依照下列步驟操作：  

1. 透過 NuGet 套件管理器安裝：  
   - 開啟 Visual Studio。  
   - 前往工具>NuGet 套件管理器>管理解決方案的 NuGet 套件。  
   - 搜尋 Aspose.Cells 並安裝最新版本。  

2. 透過套件管理器控制台安裝：  
   - 在 Visual Studio 中開啟套件管理器控制台。  
   - 運行以下命令：  
     ```powershell
     Install-Package Aspose.Cells
     ```

3. 從 Aspose 網站下載：  
   - 訪問 [Aspose.Cells for .NET 下載](https://releases。aspose.com/cells/net/).  
   - 手動下載並安裝最新的 DLL。  

4. 在您的專案中加入 Aspose.Cells 的引用：  
   - 在解決方案資源管理器中，以滑鼠右鍵按一下「引用」。  
   - 點擊新增引用並瀏覽 Aspose.Cells.dll 檔案。  
   - 按一下「確定」將其新增至您的專案。
