## 安裝說明
1. 從下載並安裝 Aspose.Cells for .NET [Aspose 發佈頁面](https://releases。aspose.com/cells/net/).  
2. 在您的 C# 專案中新增對 Aspose.Cells.dll 的參考。  
3. 使用 NuGet 套件管理器安裝 Aspose.Cells：  
   ```sh
   Install-Package Aspose.Cells
   ```
4. 在 C# 檔案中匯入所需的命名空間：  
   ```csharp
   using Aspose.Cells;
   ```
