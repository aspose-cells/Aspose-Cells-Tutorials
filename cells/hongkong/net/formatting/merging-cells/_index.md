## 設置說明  
請使用以下步:  

1. 透過NuGet安裝  
   打開視覺工作室中的包管理器控制台,  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. 下載DLL  
   另行選擇是下載最新版本[網頁下載的Aspose.Cells](https://releases.aspose.com/cells/net/)請使用您的網站.  

3. 在您的項目中添加參考  
   * 在 Visual Studio 中,右鍵按您的項目.  
   * 點擊 添加參考資料> 覽> 選擇下載的DLL.  
   * 點擊OK,然後添加參考.  

4. 開始使用Aspose.Cells  
   在您的C#檔案開始時添加以下命名空間:  
   ```csharp
   using Aspose.Cells;
   ```
