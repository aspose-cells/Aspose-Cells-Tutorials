## 安裝說明：  

若要使用 Aspose.Cells for .NET，請依照下列步驟操作：  

1. 透過 NuGet 套件管理器安裝  
- 開啟 Visual Studio。  
- 導覽至工具 → NuGet 套件管理器 → 管理解決方案的 NuGet 套件。  
- 搜尋 Aspose.Cells。  
- 按一下「安裝」將套件新增至您的專案。  

或者，使用 NuGet 套件管理器控制台安裝它：  

```powershell
Install-Package Aspose.Cells
```

2. 在 C# 程式碼中新增命名空間  
```csharp
using Aspose.Cells;
```

3. 啟動許可證（可選，用於完整功能）  
如果您有許可證，請使用以下代碼申請：  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
