## 安裝說明  
若要使用 Aspose.Cells for .NET，請依照下列步驟操作：  

1. 透過 NuGet 安裝：  
   在 Visual Studio 中開啟 NuGet 套件管理器並執行：  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. 下載 DLL：  
   從 [Aspose.Cells for .NET下載頁面](https://releases。aspose.com/cells/net/).  

3. 包括命名空間：  
   在您的 C# 檔案中加入以下內容：  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
