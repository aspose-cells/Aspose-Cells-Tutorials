## 設置說明  
請使用以下步:  

1. 透過NuGet安裝:  
   在 Visual Studio 打開 NuGet 包管理器,然後執行:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. 下載DLL:  
   獲得最新版本[網頁下載頁面的Aspose.Cells](https://releases.aspose.com/cells/net/)我們都在等.  

3. 包含名稱空間:  
   請將以下內容加入您的C#檔案:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
