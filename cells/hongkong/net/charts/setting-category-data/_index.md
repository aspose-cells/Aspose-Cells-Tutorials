## 設置說明  

請使用以下步:  

1. 透過NuGet包管理器安裝:  
   * 打開您的.NET 項目在視覺工作室.  
   * 請前往工具 → NuGet包管理器 → 管理 NuGet包.  
   * 搜尋Aspose.Cells並安裝最新版本.  

   另行選擇是使用包管理器控制台安裝:  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. 請從Aspose網站下載:  
   * 請訪問官方下載頁面:[必須使用Aspose.Cells](https://releases.aspose.com/cells/net/)  
   * 請下載DLL, 在您的項目中添加參考.  

3. 申請許可證 (可選擇):  
   * 請獲得免許[該國政府將對此做出決定.](https://purchase.aspose.com/buy)或要求一個[暫時許可證](https://purchase.aspose.com/temporary-license/)我們都在等.  
   * 應用您的代碼中的授權:  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
