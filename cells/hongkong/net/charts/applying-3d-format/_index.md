# 使用 Aspose.Cells 在 Excel 中建立和格式化 3D 圖表

## 安裝說明  
若要使用 Aspose.Cells for .NET，請依照下列步驟操作：  

1. 透過 NuGet 套件管理器安裝：  
   - 開啟 Visual Studio 並前往工具 > NuGet 套件管理器 > 管理解決方案的 NuGet 套件。  
   - 搜尋 Aspose.Cells 並安裝它。  

   或者，透過 NuGet 套件管理器控制台安裝：  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. 下載最新版本：  
   - 從 Aspose 網站取得最新版本：  
     [下載 Aspose.Cells for .NET](https://releases.aspose.com/cells/net/)  

3. 在您的專案中加入引用：  
   - 安裝後，在 C# 程式碼中包含以下命名空間：  
     ```csharp
     using Aspose.Cells;
     using Aspose.Cells.Charts;
     using Aspose.Cells.Drawing;
     using System;
     using System.Drawing;
     ```
