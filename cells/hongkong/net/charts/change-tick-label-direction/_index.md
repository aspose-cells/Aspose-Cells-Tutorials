# 使用 Aspose.Cells 在 Excel 中變更圖表刻度標籤方向 

此範例示範如何使用 Aspose.Cells for .NET 修改 Excel 工作表中圖表刻度標籤的方向。透過調整分類軸刻度標籤的文字方向，可以增強圖表資料的可讀性。程式碼載入現有的 Excel 工作簿，存取第一個工作表，擷取圖表，將刻度標籤方向設為水平，然後儲存修改後的工作簿。這種方法可確保在 .NET 應用程式中精確控制圖表標籤的方向。

## 安裝說明  
若要執行此範例，請按照以下步驟操作：  

1. 安裝 Aspose.Cells for .NET  
   - 下載地址 [Aspose.Cells for .NET](https://releases.aspose.com/cells/net/)  
   - 或透過 NuGet 安裝：  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. 設定你的項目  
   - 建立一個新的 .NET 專案。  
   - 新增對 Aspose.Cells.dll 的引用。  

3. 運行程式碼  
   - 代替 `"Your Document Directory"` 使用實際文件路徑。  
   - 執行程式以更新圖表中的刻度標籤方向。
