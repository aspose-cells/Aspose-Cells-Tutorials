# 使用Aspose.Cells修改Excel中的圖表數據標籤形狀
這篇教學將探討如何使用Aspose.Cells來修改Excel圖表中的數據標籤的形狀類型. 圖表中的數據標籤有助於提供更好的可視化,更改其形狀可以改善可讀性. 這篇示例展示如何访问現有Excel圖表,取回系列數據標籤,並將其形狀類型變為Speech Bubble Oval (圆形呼叫).

## 設置說明:  

請使用以下步:  

1. 透過NuGet安裝:  
   * 打開您的視覺工作室項目.  
   * 請前往工具> NuGet包管理器> 管理NuGet包.  
   * 搜尋Aspose.Cells並安裝最新版本.  

   或使用包管理器控制台:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. 請從Aspose網站下載:  
   * 訪問[請使用Aspose.Cells](https://releases.aspose.com/cells/net/)獲得最新的DLL.
