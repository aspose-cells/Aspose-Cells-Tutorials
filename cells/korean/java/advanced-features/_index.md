## Why Freeze Panes Matters

행이나 열을 고정하면 사용자가 방대한 데이터 세트를 스크롤할 때도 헤더가 계속 표시됩니다. 재무 보고서, 대시보드, 재고 목록 등에서 이 간단한 UI 개선은 사용자가 컨텍스트를 잃지 않게 하여 스프레드시트가 보다 세련되고 전문적으로 보이게 합니다.
