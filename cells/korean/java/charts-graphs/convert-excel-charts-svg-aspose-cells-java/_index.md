# Aspose.Cells Java를 사용하여 Excel 차트에서 SVG 변환하는 방법

## 소개

Excel 워크북의 데이터 분석 결과를 웹에 품질 저하 없이 표시하는 것은 매우 중요합니다. **SVG 변환**은 대시보드, 보고서 또는 이메일 템플릿에 선명하고 해상도에 독립적인 그래픽이 필요할 때 큰 장점이 됩니다. 이 가이드에서는 Excel 워크북을 로드하고, 차트를 찾아, Aspose.Cells for Java를 사용해 SVG 이미지로 내보내는 방법을 배웁니다. 단계는 간단하며, 라이브러리가 모든 렌더링 세부 사항을 처리해 줍니다.

**학습 내용**
- 파일에서 Excel 워크북을 로드하는 방법
- 워크시트와 특정 차트에 접근하는 방법
- 몇 줄의 코드만으로 Excel 차트를 SVG로 내보내는 방법

코드 작성을 시작하기 전에 개발 환경을 준비해 보겠습니다.

## 빠른 답변
- **라이선스 없이 차트를 내보낼 수 있나요?** 무료 체험판을 사용할 수 있지만, 실제 운영에서는 유효한 라이선스가 필요합니다.  
- **Aspose.Cells가 지원하는 내보내기 형식은?** SVG, PNG, JPEG, PDF 등 다양한 형식을 지원합니다.  
- **SVG가 진짜 벡터인가요?** 네 – SVG 파일은 어떤 화면 크기에서도 픽셀화 없이 확대·축소됩니다.  
- **특별한 IDE가 필요하나요?** IntelliJ, Eclipse, VS Code 등 모든 Java IDE에서 사용 가능합니다.  
- **변환 소요 시간은?** 일반적인 차트는 보통 1초 미만에 변환됩니다.

## “how to convert svg”란 무엇인가요?
“how to convert svg”는 래스터 이미지나 Excel 차트를 Scalable Vector Graphics(SVG) 파일로 변환하는 과정을 의미합니다. SVG는 XML 기반 벡터 포맷으로, 크기에 관계없이 시각적 정확성을 유지해 웹 페이지, 보고서, 반응형 디자인에 적합한 선명한 그래픽을 제공합니다.

## 왜 Aspose.Cells for Java를 사용하여 차트를 내보내나요?
Aspose.Cells는 **50개 이상의** 입력·출력 포맷(XLSX, CSV, PDF, SVG, HTML 등)을 지원하며, 전체 파일을 메모리에 로드하지 않고도 수백 페이지 워크북을 처리합니다. 라이브러리의 렌더링 엔진은 차트 스타일, 그라디언트, 데이터 레이블을 **시각적 정확도 99 %**로 재현해 기업용 애플리케이션에 신뢰할 수 있는 선택이 됩니다.

## 전제 조건
- Java Development Kit (JDK 8 이상) 설치
- IntelliJ IDEA 또는 Eclipse와 같은 IDE
- 기본적인 Java 프로그래밍 지식
- Aspose.Cells for Java 접근 권한(체험판 또는 정식 라이선스)

Maven 프로젝트에 Aspose.Cells를 종속성으로 추가하려면 `pom.xml` 파일에 다음을 삽입합니다:
```xml
<dependency>
    <groupId>com.aspose</groupId>

</dependency>
```


## Java에서 Excel 워크북을 로드하는 방법은?

`Workbook` 클래스는 메모리로 로드된 Excel 파일을 나타내며, 워크시트, 셀 및 차트에 접근할 수 있게 해줍니다.

`new Workbook("path/to/file.xlsx")` 로 워크북을 로드하면 전체 스프레드시트가 메모리로 읽혀져 모든 워크시트, 셀, 내장 차트에 프로그래밍 방식으로 접근할 수 있습니다. Aspose.Cells는 파일 형식을 자동으로 감지하므로 XLSX, XLS, CSV 등을 명시적으로 지정할 필요가 없습니다.

## 파일에서 워크북 로드
**개요:**  
첫 번째 단계는 Excel 워크북을 로드하는 것입니다. 이는 차트에 접근하기 위한 환경을 설정합니다.

```java
import com.aspose.cells.Workbook;
// Load an Excel workbook from a specified directory.
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

**설명:**  
- `Workbook` 클래스는 메모리 내 단일 Excel 파일을 나타내는 최상위 객체입니다.  
- `dataDir` 변수 또는 절대 경로를 통해 Excel 파일의 전체 경로를 제공하십시오.

## 특정 워크시트와 차트에 접근하는 방법은?

`Worksheet` 객체는 워크북 내의 단일 시트를 나타내며, 행·열·내장 객체를 포함합니다.  
`Chart` 객체는 워크시트에 있는 데이터의 그래픽 표현으로, 렌더링하거나 내보낼 수 있습니다.

`workbook.getWorksheets().get(0)` 로 워크시트를 가져온 뒤 `getCharts().get(0)` 을 호출하면 첫 번째 차트 객체를 얻을 수 있습니다 – 이 직접적인 접근 방식은 필요한 차트 인덱스에 모두 적용됩니다. API는 렌더링 또는 데이터 추출을 위한 `Chart` 인스턴스를 반환합니다.

## 워크시트 및 차트 접근
**개요:**  
로드가 완료되면 변환하려는 특정 워크시트와 차트에 접근합니다.

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;
// Access the first worksheet and its first chart.
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```

**설명:**  
- `worksheet` 은 `Worksheet` 타입의 객체입니다.  
- `chart` 는 워크시트의 차트 컬렉션에서 가져온 객체입니다.

## 차트를 SVG 이미지로 변환하는 방법은?

`ImageOrPrintOptions` 클래스는 차트 또는 워크시트를 이미지 파일로 변환할 때 출력 형식, 해상도, 품질 등의 렌더링 설정을 정의합니다.

`ImageOrPrintOptions` 인스턴스를 생성하고 `setSaveFormat(SaveFormat.SVG)` 로 형식을 지정한 뒤 `chart.toImage(options, "output.svg")` 를 호출합니다. 이 한 줄 호출로 Excel에 표시된 색상, 폰트, 데이터 레이블을 정확히 보존하는 완전한 SVG 파일이 생성됩니다.

## 차트를 SVG 이미지로 변환
**개요:**  
마지막 단계에서는 차트를 고품질 SVG 이미지로 변환합니다.

```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.SaveFormat;
// Convert and save the chart as an SVG image.
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setSaveFormat(SaveFormat.SVG);
String outDir = "YOUR_OUTPUT_DIRECTORY";
chart.toImage(outDir + "CCToImageinSVGFormat_out.svg", options);
```

**설명:**  
- `ImageOrPrintOptions` 는 차트 저장 방식을 구성합니다.  
- 형식을 SVG 로 설정하면 Aspose.Cells가 벡터 그래픽을 생성합니다.  
- 생성된 파일은 HTML이나 CSS 배경에 바로 삽입할 수 있습니다.

## 문제 해결 팁
- 제공한 파일 경로가 실행 중인 JVM에서 접근 가능한지 확인하십시오.  
- “Unsupported format” 오류가 발생하면 최신 Aspose.Cells 버전을 사용하고 있는지 확인하십시오.  
- 대용량 워크북은 힙 메모리를 늘려야 할 수 있으니 JVM `-Xmx` 설정을 조정하십시오.

## 실제 적용 사례
1. **웹 분석:** 대시보드에 SVG 차트를 삽입해 모든 디바이스에서 선명하고 확대 가능한 시각화를 제공합니다.  
2. **보고서 생성:** PDF 또는 Word 보고서에 SVG 이미지를 삽입해 전문가 수준의 프레젠테이션을 구현합니다.  
3. **BI 도구 통합:** 벡터 그래픽을 지원하는 비즈니스 인텔리전스 플랫폼에 SVG 출력을 제공합니다.
