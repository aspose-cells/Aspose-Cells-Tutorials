# 극동 및 라틴 글꼴을 사용하여 Excel 통합 문서 만들기

이 예제는 Aspose.Cells for .NET을 사용하여 Excel 통합 문서를 만들고, 워크시트에 텍스트 상자를 추가하고, 해당 텍스트 상자 내 텍스트에 극동 및 라틴 글꼴 이름을 설정하는 방법을 보여줍니다. 이 코드는 작업 영역 초기화, 워크시트 요소 조작, 다양한 언어 요구 사항에 맞는 특정 글꼴 서식으로 통합 문서를 저장하는 방법을 보여줍니다.

## 설치 지침
1. .NET용 Aspose.Cells 설치: 
   - Visual Studio 프로젝트에서 NuGet 패키지 관리자를 사용하세요.
     ```
     Install-Package Aspose.Cells
     ```
   - 또는 DLL을 다운로드하세요 [Aspose 웹사이트](https://www.aspose.com/downloads/cells/net).

2. 참조 추가: 프로젝트에서 Aspose.Cells 라이브러리를 참조하는지 확인하세요.

3. 출력 디렉토리 설정: 업데이트 `outputDir` Excel 파일이 저장될 원하는 출력 경로를 코드에 입력합니다.

4. 예제 실행: 제공된 C# 코드를 프로젝트에 복사하고 컴파일한 후 실행합니다. 출력 확인 메시지가 표시되고, 구성된 텍스트 상자가 포함된 Excel 파일이 지정된 디렉터리에 생성됩니다.
