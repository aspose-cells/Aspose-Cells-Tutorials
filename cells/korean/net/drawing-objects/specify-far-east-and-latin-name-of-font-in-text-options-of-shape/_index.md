# 극동 및 라틴 글꼴을 사용하여 Excel 작업본을 생성

이 예제에서는 Excel 작업책을 생성하고, 작업장에 텍스트 박스를 추가하고, 그 텍스트 상의 텍스트에 대한 극동 및 라틴 글꼴 이름을 설정하는 데 .NET에 대한 Aspose.Cells를 사용하는 방법을 보여줍니다. 코드에서는 작업 공간을 초기화하고, 작업 페이지 요소를 조작하고, 다양한 언어 요구 사항에 맞게 맞춤형 글꼴 형식을 사용하여 작업책서를 저장합니다.

## 설치 지침
1. .NET에 Aspose.Cells를 설치합니다. 
   * Visual Studio 프로젝트에서 NuGet 패키지 관리자를 사용하십시오:
     ```
     Install-Package Aspose.Cells
     ```
   * 또는 DLL를 다운로드[웹사이트를 설정](https://www.aspose.com/downloads/cells/net)\- 그래

2. 참고자료 추가: 프로젝트 참조를 Aspose.Cells 라이브러리로 확인하세요.

3. 출력 디렉토리를 설정: 업데이트`출력Dir`Excel 파일의 저장되는 원하는 출력 경로와 함께 코드의 변수

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
