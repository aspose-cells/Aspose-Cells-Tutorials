# Moteur de calcul Excel pour les applications .NET

## Aperçu

Aspose.Cells pour .NET offre un moteur de calcul robuste permettant aux développeurs de gérer des formules complexes dans des feuilles de calcul Excel. Ce puissant composant vous permet de configurer le comportement des calculs, d'implémenter une logique de calcul personnalisée et d'optimiser les performances des classeurs volumineux.

## Caractéristiques principales

- **Moteurs de calcul personnalisés**: Implémentez votre propre logique de calcul pour les formules spécialisées
- **Suivi des calculs**:Suivre et contrôler les calculs de formules pour l'optimisation des performances
- **Options de calcul récursif**: Configurer la manière dont les références circulaires sont gérées
- **Configuration du système de dates**: Basculer entre les systèmes de date 1900 et 1904
- **Détection de référence circulaire**: Identifier et résoudre les dépendances circulaires dans les formules
- **Optimisation des performances**: Affiner les paramètres de calcul pour les grands classeurs
