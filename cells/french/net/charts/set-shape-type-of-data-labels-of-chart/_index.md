# Modifier la forme de l'étiquette des données du graphique dans Excel à l'aide d'Aspose.Cells
Dans ce tutoriel, nous allons découvrir comment modifier le type de forme des étiquettes de données dans un graphique Excel à l'aide d'Aspose.Cells pour .NET. Les étiquettes de données dans les graphiques améliorent la visualisation, et modifier leur forme peut améliorer la lisibilité. Cet exemple montre comment accéder à un graphique Excel existant, récupérer les étiquettes de données de la série et modifier leur forme en forme de bulle ovale (ellipse en coin). Avec quelques lignes de code C#, vous pouvez améliorer efficacement l'esthétique de vos graphiques Excel.

## Instructions d'installation :  

Pour utiliser Aspose.Cells pour .NET dans votre projet, suivez ces étapes :  

1. Installer via NuGet :  
   - Ouvrez votre projet Visual Studio.  
   - Accédez à Outils > Gestionnaire de packages NuGet > Gérer les packages NuGet pour la solution.  
   - Recherchez Aspose.Cells et installez la dernière version.  

   Ou utilisez la console du gestionnaire de packages :  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Télécharger depuis le site Web d'Aspose :  
   - Visite [Téléchargement d'Aspose.Cells pour .NET](https://releases.aspose.com/cells/net/) et obtenez les dernières DLL.  
   - Ajoutez manuellement la référence DLL téléchargée à votre projet.  

3. Appliquer la licence (facultatif mais recommandé) :  
   - Si vous avez une licence, appliquez-la en utilisant :  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Cells.lic");
     ```
