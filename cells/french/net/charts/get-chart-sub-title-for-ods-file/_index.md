# Obtenir le sous-titre du graphique à partir du fichier ODS à l'aide d'Aspose.Cells  
Cet exemple montre comment utiliser Aspose.Cells pour .NET pour charger un fichier ODS, accéder à un graphique spécifique dans une feuille de calcul et récupérer son sous-titre. En suivant ce guide, les développeurs peuvent lire et extraire efficacement les informations graphiques des feuilles de calcul ODS en C#. Cet exemple propose une approche claire et étape par étape pour travailler avec des graphiques dans des fichiers Excel, garantissant une implémentation fluide dans les applications .NET.

## Instructions d'installation :  
Pour utiliser cet exemple, suivez ces étapes :  

1. Installez Aspose.Cells pour .NET via NuGet :  
   ```shell
   Install-Package Aspose.Cells
   ```  
2. Ajoutez l’espace de noms requis dans votre projet C# :  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Charts;
   ```  
3. Exécutez le code fourni pour charger un fichier ODS, accéder à un graphique et imprimer son sous-titre sur la console.
