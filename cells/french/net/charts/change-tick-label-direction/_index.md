# Modifier la direction des graduations d'un graphique dans Excel à l'aide d'Aspose.Cells 

Cet exemple montre comment modifier l'orientation des graduations d'un graphique dans une feuille de calcul Excel à l'aide d'Aspose.Cells pour .NET. En ajustant l'orientation du texte des graduations de l'axe des catégories, vous pouvez améliorer la lisibilité des données du graphique. Le code charge un classeur Excel existant, accède à la première feuille de calcul, récupère le graphique, définit l'orientation horizontale des graduations, puis enregistre le classeur modifié. Cette approche garantit un contrôle précis de l'orientation des étiquettes du graphique dans vos applications .NET.

## Instructions d'installation  
Pour exécuter cet exemple, suivez ces étapes :  

1. Installer Aspose.Cells pour .NET  
   - Télécharger depuis [Aspose.Cells pour .NET](https://releases.aspose.com/cells/net/)  
   - Ou installez via NuGet :  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Configurez votre projet  
   - Créez un nouveau projet .NET.  
   - Ajoutez une référence à Aspose.Cells.dll.  

3. Exécuter le code  
   - Remplacer `"Your Document Directory"` avec le chemin d'accès réel au fichier.  
   - Exécutez le programme pour mettre à jour la direction de l’étiquette de graduation dans le graphique.
