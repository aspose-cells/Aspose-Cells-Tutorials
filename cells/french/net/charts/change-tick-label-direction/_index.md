# Modifier la direction de l'étiquette de balise du graphique dans Excel en utilisant Aspose.Cells 

Cet exemple montre comment modifier la direction des étiquettes de balises de graphique dans une feuille de calcul Excel en utilisant Aspose.Cells pour .NET. En ajustant la direction du texte des étiquettes de balises d'axe de catégorie, vous pouvez améliorer la lisibilité des données du graphique. Le code charge un classeur Excel existant, accède à la première classeur, récupère le tableau, définit la direction de la balise de balise à l'horizontale, puis enregistre le classeur modifié. Cette approche garantit un contrôle précis sur l'orientation des étiquettes de graphique dans vos applications .NET.

## Instructions d'installation  
Pour exécuter cet exemple, suivez ces étapes:  

1. Installez Aspose.Cells pour .NET  
   * Télécharger à partir de [Aspose.Cells pour .NET](https://releases.aspose.com/cells/net/)  
   * Ou installer par NuGet:  
     ```shell
     Install-Package Aspose.Cells
     ```  

2. Définissez votre projet  
   * Créer un nouveau projet .NET.  
   * Ajouter une référence à Aspose.Cells.dll.  

3. Exécutez le code  
   * Remplacez`"Votre répertoire de documents"`avec le chemin du fichier réel.  
   * Exécutez le programme pour mettre à jour la direction de l'étiquette de la balise dans le graphique.
