# Redimensionner et repositionner un graphique dans Excel à l'aide d'Aspose.Cells 
Aspose.Cells pour .NET fournit une API puissante pour manipuler des fichiers Excel par programmation. Cet exemple montre comment redimensionner et repositionner un graphique dans un classeur Excel avec Aspose.Cells. Que vous ayez besoin d'ajuster les dimensions d'un graphique ou de le déplacer vers une position spécifique, ce tutoriel vous guide pas à pas avec du code C# facile à suivre. Grâce à Aspose.Cells, vous pouvez manipuler les graphiques Excel en toute fluidité, garantissant ainsi la mise en forme précise de vos rapports et présentations visuelles.

## Instructions d'installation :  

Pour utiliser Aspose.Cells pour .NET, suivez ces étapes :  

1. Installer via le gestionnaire de packages NuGet  
- Ouvrez Visual Studio.  
- Accédez à Outils → Gestionnaire de packages NuGet → Gérer les packages NuGet pour la solution.  
- Rechercher Aspose.Cells.  
- Cliquez sur Installer pour ajouter le package à votre projet.  

Vous pouvez également l'installer à l'aide de la console du gestionnaire de packages NuGet :  

```powershell
Install-Package Aspose.Cells
```

2. Ajoutez un espace de noms dans votre code C#  
```csharp
using Aspose.Cells;
```

3. Activer la licence (facultatif, pour toutes les fonctionnalités)  
Si vous avez une licence, appliquez-la en utilisant le code suivant :  

```csharp
License license = new License();
license.SetLicense("Aspose.Cells.lic");
```
