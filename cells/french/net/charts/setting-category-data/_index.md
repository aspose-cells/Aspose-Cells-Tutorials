# Créer un graphique Excel avec Aspose.Cells pour .NET
Cet exemple montre comment créer un classeur Excel, le remplir avec des données d'exemple et générer un histogramme avec Aspose.Cells pour .NET. Ce guide pas à pas décrit la création d'un classeur, l'ajout de données, la définition des étiquettes de catégorie et la configuration du graphique. Aspose.Cells fournit une API puissante pour automatiser la création et la manipulation de fichiers Excel dans les applications .NET, facilitant ainsi la génération de graphiques dynamiques par programmation.

## Instructions d'installation  

Pour utiliser Aspose.Cells pour .NET, suivez ces étapes :  

1. Installer via le gestionnaire de packages NuGet :  
   - Ouvrez votre projet .NET dans Visual Studio.  
   - Accédez à Outils → Gestionnaire de packages NuGet → Gérer les packages NuGet pour la solution.  
   - Recherchez Aspose.Cells et installez la dernière version.  

   Vous pouvez également l'installer à l'aide de la console du gestionnaire de packages :  
   ```powershell
   Install-Package Aspose.Cells
   ```

2. Télécharger depuis le site Web d'Aspose :  
   - Visitez la page de téléchargement officielle : [Aspose.Cells pour .NET](https://releases.aspose.com/cells/net/)  
   - Téléchargez la DLL et ajoutez-y une référence dans votre projet.  

3. Appliquer la licence (facultatif) :  
   - Obtenir une licence auprès de [Achat Aspose](https://purchase.aspose.com/buy) ou demander un [permis temporaire](https://purchase.aspose.com/temporary-license/).  
   - Appliquez la licence dans votre code :  
     ```csharp
     Aspose.Cells.License license = new Aspose.Cells.License();
     license.SetLicense("Aspose.Cells.lic");
     ```
