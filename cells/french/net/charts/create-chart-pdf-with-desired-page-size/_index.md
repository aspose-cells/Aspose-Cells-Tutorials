# Créer un PDF à partir d' un graphique Excel avec une taille de page spécifiée

L'exportation de graphiques d'Excel vers un fichier PDF avec une mise en page précise est une exigence cruciale pour la déclaration et la visualisation des données. Ce tutoriel montre comment utiliser Aspose.Cells pour .NET pour générer un PDF à partir d'un graphique Excel tout en spécifiant la taille et l'alignement de la page. Vous apprendrez à charger un fichier Excel, accéder à une feuille de calcul et à son graphique, et le convertir en un fichier PDF bien formaté.

## Instructions d'installation  
Pour utiliser Aspose.Cells pour .NET, suivez ces étapes:  

1. Installer par NuGet:  
   Ouvrez le gestionnaire de paquets NuGet dans Visual Studio et exécutez:  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Télécharger le DLL:  
   Obtenez la dernière version de la [Aspose.Cells pour la page de téléchargement .NET](https://releases.aspose.com/cells/net/)\- Je suis désolé .  

3. Inclure l' espace de noms:  
   Add the following in your C# file:  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
