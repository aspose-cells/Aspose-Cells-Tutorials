# Créer un PDF à partir d'un graphique Excel avec une taille de page spécifiée

Exporter des graphiques Excel vers un fichier PDF avec une mise en page précise est essentiel pour la création de rapports et la visualisation de données. Ce tutoriel montre comment utiliser Aspose.Cells pour .NET pour générer un PDF à partir d'un graphique Excel tout en spécifiant la taille et l'alignement de la page. Vous apprendrez à charger un fichier Excel, à accéder à une feuille de calcul et à son graphique, et à le convertir en un fichier PDF correctement formaté.

## Instructions d'installation  
Pour utiliser Aspose.Cells pour .NET, suivez ces étapes :  

1. Installer via NuGet :  
   Ouvrez le gestionnaire de packages NuGet dans Visual Studio et exécutez :  
   ```sh
   Install-Package Aspose.Cells
   ```  

2. Téléchargez la DLL :  
   Obtenez la dernière version du [Page de téléchargement d'Aspose.Cells pour .NET](https://releases.aspose.com/cells/net/).  

3. Inclure l'espace de noms :  
   Ajoutez ce qui suit dans votre fichier C# :  
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Rendering;
   ```
