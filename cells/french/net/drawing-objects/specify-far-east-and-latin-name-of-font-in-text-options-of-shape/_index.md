# Créer un classeur Excel avec des polices d'Extrême-Orient et latines

Cet exemple montre comment utiliser Aspose.Cells pour .NET pour créer un classeur Excel, ajouter une zone de texte à une feuille de calcul et définir les polices de caractères orientales et latines pour le texte contenu dans cette zone. Le code illustre l'initialisation de l'espace de travail, la manipulation des éléments de la feuille de calcul et l'enregistrement du classeur avec une mise en forme de police spécifique, adaptée aux différentes langues.

## Instructions d'installation
1. Installer Aspose.Cells pour .NET : 
   - Utilisez le gestionnaire de packages NuGet dans votre projet Visual Studio :
     ```
     Install-Package Aspose.Cells
     ```
   - Ou téléchargez la DLL à partir du [Site Web d'Aspose](https://www.aspose.com/downloads/cells/net).

2. Ajouter des références : assurez-vous que votre projet référence la bibliothèque Aspose.Cells.

3. Configurer le répertoire de sortie : mettre à jour le `outputDir` variable dans le code avec le chemin de sortie souhaité où le fichier Excel sera enregistré.

4. Exécuter l'exemple : copiez le code C# fourni dans votre projet, compilez-le et exécutez-le. Vous devriez voir la confirmation du résultat et le fichier Excel contenant la zone de texte configurée sera créé dans le répertoire spécifié.
