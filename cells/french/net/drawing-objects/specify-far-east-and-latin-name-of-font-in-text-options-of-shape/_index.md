# Créer un manuel Excel avec des polices de l' Extrême-Orient et du latin

Cet exemple montre comment utiliser Aspose.Cells pour .NET pour créer un carnet de travail Excel, ajouter une boîte de texte à une feuille de calcul, et définir à la fois le Far East et les noms de polices latines pour le texte dans cette boîte de calcul. Le code montre l'initialisation de l'espace de travail, la manipulation des éléments de la feuille de calcul, et le stockage du carnet de travail avec un formatage de polices spécifique adapté aux exigences linguistiques diverses.

## Instructions d'installation
1. Installez Aspose.Cells pour .NET: 
   * Utilisez le gestionnaire de paquets NuGet dans votre projet Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   * Ou télécharger le DLL de la [Site web à viser](https://www.aspose.com/downloads/cells/net)\- Je suis désolé .

2. Ajouter des références: Assurez-vous que votre projet renvoie à la bibliothèque Aspose.Cells.

3. Configurer le répertoire de sortie: mettre à jour le `Résultats`variable dans le code avec le chemin de sortie souhaité où le fichier Excel sera enregistré.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
