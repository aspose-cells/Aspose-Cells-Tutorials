# Modifier les étiquettes des objets Ole dans Excel à l'aide d'Aspose.Cells

Cet exemple montre comment accéder et modifier l'étiquette d'un objet Ole dans un fichier Excel à l'aide de la bibliothèque Aspose.Cells pour .NET. En suivant les extraits de code fournis, les développeurs peuvent facilement manipuler les objets Ole, mettre à jour leurs étiquettes et enregistrer efficacement les modifications. Cette fonctionnalité est essentielle pour optimiser dynamiquement les fichiers Excel dans les applications.

## Instructions d'installation
1. Installer Aspose.Cells pour .NET : utilisez le gestionnaire de packages NuGet pour installer la bibliothèque Aspose.Cells dans votre projet.
   ```
   Install-Package Aspose.Cells
   ```
2. Ajouter des directives d'utilisation : incluez les espaces de noms nécessaires dans votre fichier de code.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Ajouter un exemple de code : copiez et collez le code fourni dans votre projet, en veillant à spécifier le chemin du répertoire de votre document.
4. Exécutez le code : exécutez votre projet pour accéder à l'étiquette de l'objet Ole dans votre fichier Excel et la modifier. Consultez la console pour obtenir des messages confirmant la réussite du processus.
