# Modifier les étiquettes d'objets Ole dans Excel en utilisant Aspose.Cells

Cet exemple montre comment accéder et modifier l'étiquette d'un objet Ole dans un fichier Excel en utilisant la bibliothèque Aspose.Cells pour .NET. En suivant les extraits de code fournis, les développeurs peuvent facilement manipuler les objets Ole, mettre à jour leurs étiquettes et enregistrer les changements efficacement. Cette capacité est essentielle pour améliorer dynamiquement les fichiers Excel dans les applications.

## Instructions d'installation
1. Installez Aspose.Cells pour .NET: Utilisez NuGet Package Manager pour installer la bibliothèque Aspose.Cells dans votre projet.
   ```
   Install-Package Aspose.Cells
   ```
2. Ajouter des directives d'utilisation: Inclure les espaces de noms nécessaires dans votre fichier de code.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Ajouter le code d'échantillon: Copiez et collez le code fourni dans votre projet, en veillant à spécifier votre chemin de répertoire de documents.
4. Exécutez le code: Exécutez votre projet pour accéder et modifier l'étiquette d'objet Ole dans votre fichier Excel. Vérifiez dans la console pour les messages de sortie confirmant l'exécution réussie du processus.
