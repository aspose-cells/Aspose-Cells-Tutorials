# Créer et enregistrer des fichiers Excel avec Aspose.Cells
Ce tutoriel explique comment utiliser Aspose.Cells pour .NET pour créer un fichier Excel par programmation. Vous apprendrez à vérifier l'existence d'un répertoire, à instancier un objet Workbook, à ajouter une feuille de calcul, à insérer une image dans une cellule spécifique et à enregistrer le fichier à l'emplacement souhaité. Suivez l'extrait de code simple fourni pour commencer à optimiser vos tâches d'automatisation Excel.

## Instructions d'installation
1. Installer Aspose.Cells : Assurez-vous que la bibliothèque Aspose.Cells est installée. Vous pouvez le faire via le gestionnaire de packages NuGet en exécutant :
   ```
   Install-Package Aspose.Cells
   ```
2. Configurez votre environnement : créez un répertoire sur votre machine où vous souhaitez stocker les fichiers Excel générés. Remplacez `"Your Document Directory"` dans l'exemple avec votre chemin de répertoire réel.
3. Implémentez le code : copiez l’exemple de code fourni dans votre application .NET. Assurez-vous de mettre à jour le chemin d’accès à l’image (`dataDir + "logo.jpg"`) pour pointer vers une image existante que vous souhaitez insérer dans le fichier Excel.
4. Exécuter l'application : exécutez votre application. Le code créera un fichier Excel nommé `output.xls` dans le répertoire spécifié, avec une image placée dans la cellule « F6 ».
