# Créer et enregistrer des fichiers Excel avec Aspose.Cells
Ce tutoriel montre comment utiliser Aspose.Cells pour .NET pour créer un nouveau fichier Excel de manière programmatique. Vous apprendrez à vérifier l'existence d'un répertoire, instantaner un objet de Workbook, ajouter une feuille de calcul, insérer une image dans une cellule spécifique et enregistrer le fichier sur votre cheminement souhaité. Suivez le simple extrait de code fourni pour commencer à améliorer vos tâches d'automatisation Excel.

## Instructions d'installation
1. Installez Aspose.Cells: Assurez-vous d'avoir la bibliothèque Aspose.Cells installée. Vous pouvez le faire en utilisant NuGet Package Manager en exécutant:
   ```
   Install-Package Aspose.Cells
   ```
2. Configurer votre environnement: créez un répertoire sur votre machine où vous souhaitez stocker les fichiers Excel générés.`"Votre répertoire de documents"`dans l'exemple avec votre chemin réel de répertoire.
3. Implémenter le code: Copiez le code d'exemple fourni dans votre application .NET. Assurez-vous de mettre à jour le chemin d'image (`donnéesDir + "logo.jpg"`) pour indiquer une image existante que vous souhaitez insérer dans le fichier Excel.
4. Exécuter l'application: Exécuter votre application. Le code créera un nouveau fichier Excel nommé `dégagement.xls`dans le répertoire spécifié, avec une image placée dans la cellule "F6".
