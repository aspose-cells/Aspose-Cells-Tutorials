# Alignement horizontal du texte dans Excel avec Aspose.Cells

Cet exemple montre comment utiliser Aspose.Cells pour .NET pour créer un fichier Excel et définir l'alignement horizontal du texte dans une cellule spécifique. Ce tutoriel vous guide dans l'initialisation d'un classeur, la modification des propriétés d'une cellule et l'enregistrement du fichier au format Excel 97-2003. Idéal pour les développeurs souhaitant améliorer leurs compétences en manipulation de fichiers Excel avec Aspose.Cells.

## Instructions d'installation
1. Téléchargez et installez Aspose.Cells pour .NET à partir du [Page de publication d'Aspose](https://releases.aspose.com/cells/net/).  
2. Ajoutez une référence à Aspose.Cells.dll dans votre projet C#.  
3. Utilisez le gestionnaire de packages NuGet pour installer Aspose.Cells :  
   ```sh
   Install-Package Aspose.Cells
   ```
4. Importez les espaces de noms requis dans votre fichier C# :  
   ```csharp
   using Aspose.Cells;
   ```
