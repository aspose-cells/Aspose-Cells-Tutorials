We start by creating a new workbook, accessing a worksheet, and merging specific cells to format the document efficiently. ## Instructions d'installation  
Pour utiliser Aspose.Cells pour .NET dans votre projet, suivez ces étapes:  

1. Installez par NuGet  
   Ouvrez la console de gestion de paquets dans Visual Studio et exécutez:  
   ```powershell
   Install-Package Aspose.Cells
   ```  

2. Télécharger le DLL  
   Alternativement, téléchargez la dernière version à partir de [Aspose.Cells pour les téléchargements .NET](https://releases.aspose.com/cells/net/)et de faire référence au DLL dans votre projet.  

3. Ajouter une référence à votre projet  
   * Dans Visual Studio, cliquez à droite sur votre projet dans Solution Explorer.  
   * Cliquez sur Ajouter une référence > Navigation > Sélectionnez le DLL téléchargé.  
   * Cliquez sur OK pour ajouter la référence.  

4. Commencez à utiliser Aspose.Cells  
   Add the following namespace at the beginning of your C# file:  
   ```csharp
   using Aspose.Cells;
   ```
