## Οδηγός Εφαρμογής
### Βήμα 1: Φόρτωση του αρχείου Excel
Αρχικά, φορτώστε το αρχείο Excel χρησιμοποιώντας το Aspose.Cells δημιουργώντας ένα `Workbook` αντικείμενο.

#### Φόρτωση βιβλίου εργασίας
```java
Workbook wb = new Workbook("path/to/sampleImageOrPrintOptions_PageIndexPageCount.xlsx");
```
Εδώ, χρησιμοποιούμε `new Workbook()` για να ανοίξετε ένα υπάρχον αρχείο στην καθορισμένη διαδρομή.

### Βήμα 2: Πρόσβαση σε φύλλα εργασίας
Στη συνέχεια, αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας που θέλετε να αποδώσετε.

#### Φύλλο εργασίας της Access
```java
Worksheet ws = wb.getWorksheets().get(0);
```
Αυτή η γραμμή ανακτά το πρώτο φύλλο εργασίας στο βιβλίο εργασίας. Τροποποιήστε την ώστε να στοχεύει οποιοδήποτε φύλλο με βάση το ευρετήριο ή το όνομά του.

### Βήμα 3: Ρύθμιση επιλογών εικόνας/εκτύπωσης
Ρυθμίστε τις παραμέτρους απόδοσης, καθορίζοντας ποιες σελίδες θέλετε να αποδοθούν ως εικόνες.

#### Ρύθμιση παραμέτρων επιλογών απόδοσης
```java
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.setPageIndex(3); // Ξεκινώντας από τη σελίδα 4 (ευρετήριο με βάση το 0)
opts.setPageCount(4); // Απόδοση τεσσάρων διαδοχικών σελίδων
opts.setImageType(ImageType.PNG);
```
- `setPageIndex`: Ορίστε την αρχική σελίδα.
- `setPageCount`Καθορίστε πόσες σελίδες θα αποδοθούν.
- `setImageType`: Επιλέξτε τη μορφή για τις εικόνες εξόδου.

### Βήμα 4: Απόδοση σελίδων
Δημιουργήστε ένα `SheetRender` αντικείμενο και χρησιμοποιήστε το για να μετατρέψετε σελίδες σε εικόνες.

#### Απόδοση σελίδων
```java
SheetRender sr = new SheetRender(ws, opts);

for (int i = opts.getPageIndex(); i < sr.getPageCount(); i++) {
    sr.toImage(i, "outputPath/outputImage-" + (i+1) + ".png");
}
```
Εδώ, κάνουμε επανάληψη μέσα από το καθορισμένο εύρος σελίδων και μετατρέπουμε την καθεμία σε εικόνα.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Ευρετήριο σελίδων εκτός εύρους**Βεβαιωθείτε ότι `setPageIndex` και `setPageCount` βρίσκονται εντός του συνολικού αριθμού σελίδων.
- **Σφάλματα διαδρομής αρχείου**Ελέγξτε ξανά τις διαδρομές αρχείων τόσο για τα αρχεία Excel εισόδου όσο και για τις εικόνες εξόδου.
