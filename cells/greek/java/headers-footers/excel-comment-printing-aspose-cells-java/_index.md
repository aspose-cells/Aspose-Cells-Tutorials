## Οδηγός Εφαρμογής

### Ορισμός Εκτύπωση σχολίων σε Χωρίς σχόλια
Αυτή η λειτουργία διασφαλίζει ότι δεν θα εκτυπωθούν σχόλια, διατηρώντας την εκτύπωση του εγγράφου σας εστιασμένη στα δεδομένα.

#### Επισκόπηση
Ρυθμίζοντας το `PrintCommentsType` να `PRINT_NO_COMMENTS`, αποτρέπετε την συμπερίληψη σχολίων στο αποτέλεσμα PDF του αρχείου Excel σας.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση του βιβλίου εργασίας σας**
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

**Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας**
```java
Worksheet worksheet = workbook.getWorksheets().get(0); // Πρώτο φύλλο εργασίας
```

**Βήμα 3: Ορισμός επιλογής εκτύπωσης σχολίων**
```java
import com.aspose.cells.PrintCommentsType;
worksheet.getPageSetup().setPrintComments(PrintCommentsType.PRINT_NO_COMMENTS);
```

**Βήμα 4: Αποθήκευση ως PDF**
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "PrintNoComments_out.pdf");
```

### Εκτύπωση σχολίων στη θέση τους
Η εκτύπωση σχολίων απευθείας εκεί που βρίσκονται παρέχει μια σαφή εικόνα των σχολιασμών μαζί με τα σχετικά δεδομένα.

#### Επισκόπηση
Ορίστε το `PrintCommentsType` να `PRINT_IN_PLACE` για να επιτευχθεί αυτό.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση του βιβλίου εργασίας σας**
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

**Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας**
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**Βήμα 3: Ρύθμιση παραμέτρων εκτύπωσης σχολίων στη θέση τους**
```java
worksheet.getPageSetup().setPrintComments(PrintCommentsType.PRINT_IN_PLACE);
```

**Βήμα 4: Αποθήκευση ως PDF**
```java
workbook.save(outDir + "PrintInPlace_out.pdf");
```

### Εκτύπωση σχολίων στο τέλος του φύλλου
Συλλέξτε όλα τα σχόλια και εκτυπώστε τα στο τέλος του φύλλου σας για μια ενοποιημένη προβολή.

#### Επισκόπηση
Χρήση `PRINT_SHEET_END` για να διαμορφώσετε αυτήν τη ρύθμιση.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση του βιβλίου εργασίας σας**
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

**Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας**
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**Βήμα 3: Ορισμός σχολίων εκτύπωσης στο τέλος του φύλλου**
```java
worksheet.getPageSetup().setPrintComments(PrintCommentsType.PRINT_SHEET_END);
```

**Βήμα 4: Αποθήκευση ως PDF**
```java
workbook.save(outDir + "PrintSheetEnd_out.pdf");
```
