## Οδηγός Εφαρμογής

Ας αναλύσουμε πώς να ενεργοποιήσετε την προεπισκόπηση αλλαγής σελίδας στα βιβλία εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για Java.

### Επισκόπηση της λειτουργίας προεπισκόπησης αλλαγής σελίδας

Η λειτουργία προεπισκόπησης αλλαγής σελίδας σάς επιτρέπει να δείτε πώς θα κατανεμηθεί το περιεχόμενο του φύλλου εργασίας σας στις σελίδες κατά την εκτύπωση. Αυτό είναι ιδιαίτερα χρήσιμο για να διασφαλίσετε ότι ενότητες όπως πίνακες και επικεφαλίδες δεν θα χωρίζονται μεταξύ σελίδων.

### Βήμα προς βήμα εφαρμογή

#### Φόρτωση του βιβλίου εργασίας
Αρχικά, φορτώστε το αρχείο Excel χρησιμοποιώντας το Aspose.Cells:
```java
// Η διαδρομή προς τον κατάλογο εγγράφων.
String dataDir = Utils.getSharedDataDir(PageBreakPreview.class) + "Worksheets/";

// Δημιουργία αντικειμένου βιβλίου εργασίας μέσω διαδρομής αρχείου excel
Workbook workbook = new Workbook(dataDir + "book1.xls");
```
**Εξήγηση:** 
- `Utils.getSharedDataDir(...)` βοηθά στην ανάκτηση του σωστού καταλόγου όπου βρίσκονται τα αρχεία σας.
- Η φόρτωση ενός βιβλίου εργασίας το προετοιμάζει για περαιτέρω λειτουργίες.

#### Πρόσβαση και τροποποίηση του φύλλου εργασίας

```java
// Λήψη του πρώτου φύλλου εργασίας από τη συλλογή
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet worksheet = worksheets.get(0);

// Ενεργοποίηση προεπισκόπησης αλλαγής σελίδας
worksheet.setPageBreakPreview(true);
```
**Εξήγηση:** 
- `getWorksheets()` ανακτά όλα τα φύλλα μέσα στο βιβλίο εργασίας.
- `getPageBreakPreview(true)` ενεργοποιεί τον οπτικό οδηγό για την εκτύπωση.

#### Αποθήκευση αλλαγών

```java
// Αποθήκευση του τροποποιημένου αρχείου Excel στην προεπιλεγμένη μορφή
workbook.save(dataDir + "PageBreakPreview_out.xls");
```
**Εξήγηση:**
- Οι αλλαγές αποθηκεύονται σε ένα νέο αρχείο, διατηρώντας τις προσαρμογές σας.
