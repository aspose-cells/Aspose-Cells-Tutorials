# Εισαγωγή δεδομένων από το Excel


Σε αυτόν τον ολοκληρωμένο οδηγό, θα σας καθοδηγήσουμε στη διαδικασία εισαγωγής δεδομένων από αρχεία Excel χρησιμοποιώντας την ισχυρή βιβλιοθήκη Aspose.Cells για Java. Είτε εργάζεστε σε ανάλυση δεδομένων, δημιουργία αναφορών είτε σε οποιαδήποτε εφαρμογή Java που απαιτεί ενσωμάτωση δεδομένων Excel, το Aspose.Cells απλοποιεί την εργασία. Ας ξεκινήσουμε.

## Ανάγνωση δεδομένων από το Excel

Τώρα, ας γράψουμε τον κώδικα Java για να διαβάσουμε δεδομένα από ένα αρχείο Excel χρησιμοποιώντας το Aspose.Cells. Ακολουθεί ένα απλό παράδειγμα:

```java
import com.aspose.cells.*;
import java.io.*;

public class ExcelDataImport {
    public static void main(String[] args) throws Exception {
        // Φόρτωση του αρχείου Excel
        Workbook workbook = new Workbook("input.xlsx");

        // Πρόσβαση στο φύλλο εργασίας
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // Πρόσβαση σε δεδομένα κελιού (π.χ., A1)
        Cell cell = worksheet.getCells().get("A1");
        System.out.println("Data in cell A1: " + cell.getStringValue());

        // Πρόσβαση και επανάληψη σε γραμμές και στήλες
        for (int row = 0; row < worksheet.getCells().getMaxDataRow() + 1; row++) {
            for (int col = 0; col < worksheet.getCells().getMaxDataColumn() + 1; col++) {
                Cell dataCell = worksheet.getCells().get(row, col);
                System.out.print(dataCell.getStringValue() + "\t");
            }
            System.out.println();
        }
    }
}
```

Σε αυτόν τον κώδικα, φορτώνουμε ένα βιβλίο εργασίας του Excel, έχουμε πρόσβαση σε ένα συγκεκριμένο κελί (A1) και επεξεργαζόμαστε όλες τις γραμμές και τις στήλες για να διαβάσουμε και να εμφανίσουμε τα δεδομένα.

## Εκτέλεση του Κώδικα

Μεταγλωττίστε και εκτελέστε τον κώδικα Java στο IDE σας. Βεβαιωθείτε ότι έχετε ένα αρχείο Excel με το όνομα "input.xlsx" στον κατάλογο του έργου σας. Ο κώδικας θα εμφανίσει τα δεδομένα στο κελί A1 και όλα τα δεδομένα στο φύλλο εργασίας.
