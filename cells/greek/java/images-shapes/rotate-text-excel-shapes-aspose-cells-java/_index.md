```xml
<dependency>
```

## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Φόρτωση δείγματος αρχείου Excel

#### Επισκόπηση
Η φόρτωση ενός υπάρχοντος αρχείου Excel είναι το πρώτο βήμα στη διαδικασία μας.

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook wb = new Workbook(dataDir + "/sampleRotateTextWithShapeInsideWorksheet.xlsx");
```

**Εξήγηση**: Το `Workbook` Η κλάση αντιπροσωπεύει ολόκληρο το υπολογιστικό σας φύλλο. Περνώντας τη διαδρομή αρχείου, φορτώνετε το έγγραφο Excel στη μνήμη.

### Χαρακτηριστικό 2: Φύλλο εργασίας Access First

#### Επισκόπηση
Η πρόσβαση σε συγκεκριμένα φύλλα εργασίας μάς επιτρέπει να στοχεύουμε ακριβείς περιοχές για χειρισμό κειμένου και σχήματος.

```java
import com.aspose.cells.Worksheet;

Worksheet ws = wb.getWorksheets().get(0);
```

**Εξήγηση**: `getWorksheets()` επιστρέφει μια συλλογή όλων των φύλλων, ενώ `get(0)` αποκτά πρόσβαση στο πρώτο φύλλο εργασίας.

### Λειτουργία 3: Προσθήκη μηνύματος σε κελί

#### Επισκόπηση
Η προσθήκη κειμένου σε κελιά είναι απλή με το Aspose.Cells.

```java
import com.aspose.cells.Cell;

Cell b4 = ws.getCells().get("B4");
b4.putValue("Text is not rotating with shape because RotateTextWithShape is false.");
```

**Εξήγηση**: `getCells()` ανακτά όλα τα αντικείμενα κελιού και `putValue` αντιστοιχίζει κείμενο σε ένα συγκεκριμένο κελί.

### Λειτουργία 4: Πρόσβαση στο πρώτο σχήμα στο φύλλο εργασίας

#### Επισκόπηση
Ο χειρισμός σχημάτων περιλαμβάνει την πρόσβαση στις ιδιότητές τους για την προσαρμογή της ευθυγράμμισης κειμένου.

```java
import com.aspose.cells.Shape;
import com.aspose.cells.ShapeTextAlignment;

Shape sh = ws.getShapes().get(0);
ShapeTextAlignment shapeTextAlignment = sh.getTextBody().getTextAlignment();
shapeTextAlignment.setRotateTextWithShape(false);
```

**Εξήγηση**: Το `getShapes()` η μέθοδος ανακτά όλα τα σχήματα και τροποποιούμε την ευθυγράμμιση του κειμένου ορίζοντας `setRotateTextWithShape` σε ψευδή.

### Λειτουργία 5: Αποθήκευση αρχείου Excel στον κατάλογο εξόδου

#### Επισκόπηση
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο.

```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
wb.save(outDir + "/outputRotateTextWithShapeInsideWorksheet.xlsx");
```

**Εξήγηση**: Το `save()` Η μέθοδος γράφει όλες τις τροποποιήσεις στον καθορισμένο κατάλογο εξόδου.
