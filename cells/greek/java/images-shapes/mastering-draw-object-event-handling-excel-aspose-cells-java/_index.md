## Ρύθμιση του Aspose.Cells για Java

Η έναρξη με το Aspose.Cells για Java είναι απλή, χάρη στην υποστήριξη Maven και Gradle.

### Χρησιμοποιώντας το Maven

Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Χρησιμοποιώντας το Gradle

Συμπεριλάβετε αυτό στο δικό σας `build.gradle` αρχείο:

```gradle
implementation 'com.aspose:aspose-cells:25.3'
```

## Οδηγός Εφαρμογής

### Χειρισμός συμβάντων αντικειμένου σχεδίασης

Αυτή η λειτουργία σάς επιτρέπει να διαχειρίζεστε συμβάντα που σχετίζονται με τη σχεδίαση αντικειμένων σε ένα αρχείο Excel. Ας αναλύσουμε τον τρόπο υλοποίησης αυτής της λειτουργικότητας.

#### Προσαρμοσμένη κλάση χειριστή συμβάντων

Ξεκινήστε δημιουργώντας μια προσαρμοσμένη κλάση χειριστή συμβάντων που επεκτείνεται `DrawObjectEventHandler`:

```java
import com.aspose.cells.*;

class clsDrawObjectEventHandler extends DrawObjectEventHandler {
    @Override
    public void draw(DrawObject drawObject, float x, float y, float width, float height) {
        if (drawObject.getType() == DrawObjectEnum.CELL) {
            System.out.println("[X]: " + x +
                               " [Y]: " + y +
                               " [Width]: " + width +
                               " [Height]: " + height +
                               " [Cell Value]: " + drawObject.getCell().getStringValue());
        }

        if (drawObject.getType() == DrawObjectEnum.IMAGE) {
            System.out.println("[X]: " + x +
                               " [Y]: " + y +
                               " [Width]: " + width +
                               " [Height]: " + height +
                               " [Shape Name]: " + drawObject.getShape().getName());
        }

        System.out.println("----------------------");
    }
}
```

#### Μετατροπή βιβλίου εργασίας και PDF

Στη συνέχεια, εφαρμόστε τη λειτουργικότητα για να φορτώσετε ένα αρχείο Excel, να ρυθμίσετε τον χειριστή συμβάντων και να το αποθηκεύσετε ως PDF:

```java
void Run() throws Exception {
    String dataDir = "YOUR_DATA_DIRECTORY"; 
    String outDir = "YOUR_OUTPUT_DIRECTORY";

    // Φόρτωση του βιβλίου εργασίας από έναν καθορισμένο κατάλογο
    Workbook wb = new Workbook(dataDir + "sampleGetDrawObjectAndBoundUsingDrawObjectEventHandler.xlsx");

    PdfSaveOptions opts = new PdfSaveOptions();
    
    // Αντιστοιχίστε τον προσαρμοσμένο χειριστή συμβάντων αντικειμένου σχεδίασης
    opts.setDrawObjectEventHandler(new clsDrawObjectEventHandler());
    
    // Αποθήκευση του βιβλίου εργασίας ως PDF με τις καθορισμένες επιλογές
    wb.save(outDir + "outputGetDrawObjectAndBoundUsingDrawObjectEventHandler.pdf", opts);
}
```
