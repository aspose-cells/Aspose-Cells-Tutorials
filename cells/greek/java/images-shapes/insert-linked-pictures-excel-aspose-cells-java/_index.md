## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία εισαγωγής συνδεδεμένων εικόνων στα αρχεία Excel.

### Εισαγωγή συνδεδεμένης εικόνας από μια διεύθυνση ιστού

#### Βήμα 1: Ρύθμιση του βιβλίου εργασίας
Δημιουργήστε μια νέα παρουσία βιβλίου εργασίας όπου θα εισαγάγετε τη συνδεδεμένη εικόνα σας.

```java
import com.aspose.cells.Workbook;

Workbook workbook = new Workbook();
```

#### Βήμα 2: Προσθήκη συνδεδεμένης εικόνας
Χρησιμοποιήστε το `addLinkedPicture` μέθοδος για την προσθήκη μιας εικόνας από μια διεύθυνση ιστού στο κελί B2. Οι παράμετροι καθορίζουν τη γραμμή, τη στήλη και το μέγεθος της εικόνας.

```java
import com.aspose.cells.Picture;
import com.aspose.cells.Worksheet;

Worksheet worksheet = workbook.getWorksheets().get(0);
int pictureIndex = worksheet.getShapes().addLinkedPicture(1, 1, 100, 100,
        "http://www.aspose.com/Images/aspose-logo.jpg");
Picture pic = worksheet.getShapes().get(pictureIndex) instanceof Picture ? (Picture) worksheet.getShapes().get(pictureIndex) : null;
```

#### Βήμα 3: Ρύθμιση παραμέτρων της πηγής εικόνας
Ορίστε τη διεύθυνση URL της πηγής εικόνας για να βεβαιωθείτε ότι είναι δυναμικά συνδεδεμένη.

```java
pic.setSourceFullName("http://www.aspose.com/images/aspose-logo.gif");
```

#### Βήμα 4: Προσαρμογή διαστάσεων εικόνας
Προσαρμόστε το ύψος και το πλάτος για καλύτερη εμφάνιση στο αρχείο Excel.

```java
pic.setHeightInch(1.04);
pic.setWidthInch(2.6);
```

#### Βήμα 5: Αποθήκευση του βιβλίου εργασίας σας
Αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε τις αλλαγές, διασφαλίζοντας ότι περιλαμβάνεται η συνδεδεμένη εικόνα.

```java
workbook.save("ILPfromWebAddress_out.xlsx");
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Η εικόνα δεν εμφανίζεται**Βεβαιωθείτε ότι η διεύθυνση URL είναι σωστή και προσβάσιμη.
- **Προβλήματα μνήμης**Βελτιστοποιήστε το μέγεθος της εικόνας για καλύτερη απόδοση με μεγάλα αρχεία Excel.
