# Πώς να προσθέσετε και να διαμορφώσετε σχήματα στο Excel χρησιμοποιώντας το Aspose.Cells Java

## Εισαγωγή

Βελτιώστε τα φύλλα εργασίας του Excel προσθέτοντας προσαρμοσμένα σχήματα μέσω προγραμματισμού με `Aspose.Cells` για Java. Αυτό το σεμινάριο σας καθοδηγεί στην προσθήκη ενός ορθογωνίου σχήματος, στη διαμόρφωση των στυλ γραμμών του και στην εφαρμογή γεμισμάτων με διαβάθμιση.

**Τι θα μάθετε:**
- Ρύθμιση του Aspose.Cells στο έργο Java σας.
- Προσθήκη ορθογωνίου σχήματος σε ένα φύλλο εργασίας του Excel.
- Ρύθμιση παραμέτρων στυλ γραμμών και διαβαθμίσεων για σχήματα.
- Αποθήκευση του τροποποιημένου βιβλίου εργασίας.

Ας ξεκινήσουμε διασφαλίζοντας ότι πληροίτε όλες τις προϋποθέσεις.

## Προαπαιτούμενα

Πριν εμβαθύνετε στον κώδικα, βεβαιωθείτε ότι:
- **Βιβλιοθήκες:** Η βιβλιοθήκη Aspose.Cells (έκδοση 25.3 ή νεότερη) περιλαμβάνεται στο έργο σας.
- **Περιβάλλο:** Εξοικείωση με περιβάλλοντα ανάπτυξης Java όπως το Maven ή το Gradle για διαχείριση εξαρτήσεων.
- **Γνώση:** Βασική κατανόηση προγραμματισμού Java και χειρισμού αρχείων Excel.

## Ρύθμιση του Aspose.Cells για Java

Ενσωματώστε το Aspose.Cells στο έργο Java σας χρησιμοποιώντας το εργαλείο δημιουργίας:

**Maven:**
Προσθήκη στο δικό σας `pom.xml`:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Βαθμός:**
Συμπεριλάβετε στο δικό σας `build.gradle` αρχείο:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Οδηγός Εφαρμογής

### Προσθήκη ορθογωνίου σχήματος σε φύλλο εργασίας Excel

**Επισκόπηση:** Μάθετε πώς να προσθέτετε και να τοποθετείτε ένα ορθογώνιο σχήμα στο φύλλο εργασίας σας χρησιμοποιώντας το Aspose.Cells.

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας
```java
Workbook excelBook = new Workbook();
```
Αυτό προετοιμάζει μια νέα παρουσία βιβλίου εργασίας όπου θα προσθέτετε τα σχήματα.

#### Βήμα 2: Προσθήκη ορθογωνίου σχήματος
```java
import com.aspose.cells.RectangleShape;
import com.aspose.cells.MsoDrawingType;

RectangleShape rectangle = (RectangleShape) excelBook.getWorksheets().get(0)
        .getShapes().addShape(MsoDrawingType.RECTANGLE, 3, 2, 0, 0, 70, 130);
```
Εδώ, ένα ορθογώνιο προστίθεται στο πρώτο φύλλο εργασίας. Οι παράμετροι καθορίζουν τον τύπο, τη θέση και το μέγεθός του.

#### Βήμα 3: Ορισμός τοποθέτησης
```java
rectangle.setPlacement(com.aspose.cells.PlacementType.FREE_FLOATING);
```
Αυτό ρυθμίζει το σχήμα ώστε να είναι ελεύθερα κινούμενο αντί να είναι αγκυρωμένο σε μια συγκεκριμένη περιοχή κελιών.

### Ρύθμιση παραμέτρων στυλ γραμμής ενός σχήματος

**Επισκόπηση:** Προσαρμόστε το στυλ γραμμής και το γέμισμα με διαβάθμιση για το σχήμα ορθογωνίου σας.

#### Βήμα 1: Ρύθμιση παραμέτρων στυλ γραμμής
```java
import com.aspose.cells.LineFormat;
import com.aspose.cells.MsoLineStyle;

LineFormat linestyle = rectangle.getLine();
linestyle.setDashStyle(MsoLineStyle.THICK_THIN);
linestyle.setWeight(4);
```
Αυτό ορίζει το στυλ γραμμής σε ένα παχύ-λεπτό μοτίβο παύλας και προσαρμόζει το πάχος του.

#### Βήμα 2: Εφαρμογή γεμίσματος με διαβάθμιση
```java
import com.aspose.cells.FillFormat;
import com.aspose.cells.GradientStyleType;

FillFormat fillformat = rectangle.getFill();
fillformat.setOneColorGradient(com.aspose.cells.Color.getBlue(), 1, 
    GradientStyleType.HORIZONTAL, 1);
```
Ένα εφέ διαβάθμισης εφαρμόζεται στο γέμισμα του ορθογωνίου για οπτική βελτίωση.

### Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με όλες τις ρυθμίσεις παραμέτρων:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
excelBook.save(outDir + "/StyledRectangle_out.xls");
```
