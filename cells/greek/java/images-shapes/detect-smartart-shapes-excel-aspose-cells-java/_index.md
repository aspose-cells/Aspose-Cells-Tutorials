## Οδηγός Εφαρμογής

### Φόρτωση του βιβλίου εργασίας και πρόσβαση σε σχήματα

#### Επισκόπηση
Για να εντοπίσετε σχήματα SmartArt, πρέπει πρώτα να φορτώσετε ένα βιβλίο εργασίας του Excel και να αποκτήσετε πρόσβαση στα περιεχόμενά του.

#### Βήματα:

**1. Φορτώστε το Δείγμα Βιβλίου Εργασίας**

```java
import com.aspose.cells.*;

public class DetermineIfShapeIsSmartArtShape {
    static String srcDir = Utils.Get_SourceDirectory();

    public static void main(String[] args) throws Exception {
        // Φόρτωση του δείγματος σχήματος έξυπνης τέχνης - αρχείο Excel
        Workbook wb = new Workbook(srcDir + "sampleSmartArtShape.xlsx");
    }
}
```

- **Παράμετροι**: Το `Workbook` Ο κατασκευαστής δέχεται μια παράμετρο συμβολοσειράς που αντιπροσωπεύει τη διαδρομή αρχείου του εγγράφου Excel σας.

**2. Πρόσβαση στο Πρώτο Φύλλο Εργασίας**

```java
// Πρώτο φύλλο εργασίας της Access
Worksheet ws = wb.getWorksheets().get(0);
```

- **Σκοπός**: Αυτό ανακτά το πρώτο φύλλο εργασίας μέσα στο βιβλίο εργασίας για περαιτέρω λειτουργίες.

**3. Πρόσβαση στο σχήμα και ανίχνευση SmartArt**

```java
// Πρόσβαση στο πρώτο σχήμα
Shape sh = ws.getShapes().get(0);

// Προσδιορίστε εάν το σχήμα είναι έξυπνη τέχνη
System.out.println("Is Smart Art Shape: " + sh.isSmartArt());
```

- **Εξήγηση μεθόδου**: Το `isSmartArt()` Η μέθοδος ελέγχει εάν το δεδομένο σχήμα είναι ένα γραφικό SmartArt.
  
**Συμβουλές αντιμετώπισης προβλημάτων**:
- Βεβαιωθείτε ότι το αρχείο Excel σας περιέχει τουλάχιστον ένα φύλλο εργασίας και ένα σχήμα.
- Επαληθεύστε τη διαδρομή που καθορίζεται στο `srcDir` δείχνει στη σωστή θέση του αρχείου Excel σας.
