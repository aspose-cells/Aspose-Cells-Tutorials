## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να εφαρμόζουμε τη λειτουργία ορισμού ονόματος καρτέλας σε ένα μόνο φύλλο κατά την εξαγωγή ενός αρχείου Excel ως HTML.

### Φόρτωση και ρύθμιση παραμέτρων βιβλίου εργασίας

Αρχικά, φορτώστε το βιβλίο εργασίας του Excel που περιέχει μόνο ένα φύλλο. Αυτή η ρύθμιση διασφαλίζει σαφήνεια στην εξαγόμενη HTML:

#### Φόρτωση του βιβλίου εργασίας
```java
// Αρχικοποίηση ενός νέου αντικειμένου βιβλίου εργασίας με τη διαδρομή του καταλόγου προέλευσης
Workbook wb = new Workbook(srcDir + "sampleSingleSheet.xlsx");
```

### Ρύθμιση επιλογών αποθήκευσης HTML

Διαμορφώστε το `HtmlSaveOptions` για να ελέγξετε τον τρόπο αποθήκευσης του βιβλίου εργασίας ως αρχείο HTML.

#### Ρύθμιση παραμέτρων HtmlSaveOptions
```java
HtmlSaveOptions options = new HtmlSaveOptions();

// Ορίστε διάφορες επιλογές εξαγωγής για καλύτερη προσαρμογή της εξόδου
options.setEncoding(Encoding.getUTF8()); // Χρήση κωδικοποίησης UTF-8
options.setExportImagesAsBase64(true);   // Εξαγωγή εικόνων σε μορφή Base64
options.setExportGridLines(true);        // Συμπερίληψη γραμμών πλέγματος στην έξοδο HTML
options.setExportSimilarBorderStyle(true);
options.setExportBogusRowData(true);     // Διατηρήστε την ακεραιότητα των δεδομένων εξάγοντας ψευδή δεδομένα γραμμών
options.setExcludeUnusedStyles(true);    // Εξαιρέστε τα αχρησιμοποίητα στυλ CSS για να μειώσετε το μέγεθος του αρχείου
options.setExportHiddenWorksheet(true);  // Εξαγωγή κρυφών φύλλων εργασίας, εάν χρειάζεται
```

#### Αποθήκευση βιβλίου εργασίας ως HTML

Τέλος, αποθηκεύστε το βιβλίο εργασίας σε μορφή HTML με τις καθορισμένες επιλογές σας:

```java
// Ορίστε τον κατάλογο εξόδου και αποθηκεύστε το αρχείο HTML
wb.save(outDir + "outputSampleSingleSheet.htm", options);
```

### Βασικές επιλογές διαμόρφωσης:
- **Κωδικοποίηση:** Διασφαλίστε την ορθή αναπαράσταση χαρακτήρων χρησιμοποιώντας UTF-8.
- **Εικόνες Base64:** Η ενσωμάτωση εικόνων απευθείας μέσα στο HTML βοηθά στην αποφυγή εξωτερικών εξαρτήσεων.
- **Γραμμές και στυλ πλέγματος:** Αυτά διατηρούν την οπτική δομή των δεδομένων του Excel σας στην έξοδο HTML.
