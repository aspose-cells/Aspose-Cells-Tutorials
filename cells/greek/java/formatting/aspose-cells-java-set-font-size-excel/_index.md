## Οδηγός Εφαρμογής

Τώρα, ας εξερευνήσουμε πώς μπορείτε να ορίσετε το μέγεθος γραμματοσειράς σε ένα κελί του Excel χρησιμοποιώντας το Aspose.Cells για Java.

### Δημιουργία βιβλίου εργασίας και πρόσβαση σε κελιά
**Επισκόπηση:**
Ξεκινήστε δημιουργώντας ένα `Workbook` αντικείμενο. Στη συνέχεια, αποκτήστε πρόσβαση στο φύλλο εργασίας όπου θέλετε να τροποποιήσετε το μέγεθος της γραμματοσειράς.
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class SetFontSize {
    public static void main(String[] args) throws Exception {
        // Δημιουργία αντικειμένου βιβλίου εργασίας
        Workbook workbook = new Workbook();
        
        // Πρόσβαση στο φύλλο εργασίας που προστέθηκε στο αρχείο Excel
        int sheetIndex = workbook.getWorksheets().add();
        Worksheet worksheet = workbook.getWorksheets().get(sheetIndex);
    }
}
```

### Ρύθμιση μεγέθους γραμματοσειράς
**Επισκόπηση:**
Τροποποιήστε το μέγεθος γραμματοσειράς ενός συγκεκριμένου κελιού αποκτώντας πρόσβαση και τροποποιώντας το `Style`.
```java
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;
import com.aspose.cells.Style;
import com.aspose.cells.Font;

public class SetFontSize {
    public static void main(String[] args) throws Exception {
        Workbook workbook = new Workbook();
        int sheetIndex = workbook.getWorksheets().add();
        Worksheet worksheet = workbook.getWorksheets().get(sheetIndex);
        Cells cells = worksheet.getCells();

        // Αποκτήστε πρόσβαση στο κελί και ορίστε την τιμή του
        Cell cell = cells.get("A1");
        cell.setValue("Hello Aspose!");

        // Ανάκτηση και τροποποίηση του στυλ του κελιού για προσαρμογή του μεγέθους της γραμματοσειράς
        Style style = cell.getStyle();
        Font font = style.getFont();
        font.setSize(14);  // Ορίστε το επιθυμητό μέγεθος γραμματοσειράς
        cell.setStyle(style);

        // Αποθήκευση του τροποποιημένου βιβλίου εργασίας
        String dataDir = "path/to/save/";
        workbook.save(dataDir + "SetFontSize_out.xls");
    }
}
```
**Εξήγηση:**
- **`Font.setFontSize(int size)`**: Ορίζει το μέγεθος της γραμματοσειράς. Εδώ, χρησιμοποιούμε `14`, αλλά μπορείτε να επιλέξετε οποιαδήποτε άλλη ακέραια τιμή.
- **Αποθήκευση του βιβλίου εργασίας**: Το `workbook.save()` Η μέθοδος γράφει αλλαγές σε ένα αρχείο στο σύστημά σας.
