## Οδηγός Εφαρμογής

Τώρα, ας δούμε πώς αλλάζουμε τα χρώματα του θέματος χρησιμοποιώντας το Aspose.Cells:

### Ανάκτηση τρεχόντων χρωμάτων θέματος

#### Επισκόπηση
Ξεκινήστε ανοίγοντας ένα υπάρχον αρχείο Excel και ανακτώντας τα τρέχοντα χρώματα θέματος. Αυτό σας βοηθά να κατανοήσετε τη γραμμή βάσης πριν κάνετε οποιεσδήποτε αλλαγές.

#### Απόσπασμα κώδικα

```java
import com.aspose.cells.Color;
import com.aspose.cells.ThemeColorType;
import com.aspose.cells.Workbook;

public class GetSetThemeColors {
    public static void main(String[] args) throws Exception {
        // Διαδρομή προς το αρχείο Excel σας
        String dataDir = "path_to_your_directory/";
        
        // Άνοιγμα ενός υπάρχοντος αρχείου Excel
        Workbook workbook = new Workbook(dataDir + "book1.xlsx");
        
        // Ανάκτηση και εκτύπωση χρώματος θέματος Background1
        Color background1Color = workbook.getThemeColor(ThemeColorType.BACKGROUND_1);
        System.out.println("Current Background1 Theme Color: " + background1Color);
        
        // Ανάκτηση και εκτύπωση χρώματος θέματος Accent2
        Color accent2Color = workbook.getThemeColor(ThemeColorType.ACCENT_1);
        System.out.println("Current Accent2 Theme Color: " + accent2Color);
    }
}
```

Αυτός ο κώδικας ανοίγει ένα αρχείο Excel και εκτυπώνει τα τρέχοντα χρώματα θέματος για `BACKGROUND_1` και `ACCENT_1`.

### Αλλαγή χρωμάτων θέματος

#### Επισκόπηση
Στη συνέχεια, τροποποιήστε αυτά τα χρώματα θέματος ώστε να ταιριάζουν στις ανάγκες σας. Θα τα αλλάξουμε εμείς `BACKGROUND_1` στο κόκκινο και `ACCENT_2` σε μπλε.

#### Απόσπασμα κώδικα

```java
import com.aspose.cells.Color;
import com.aspose.cells.ThemeColorType;

public class GetSetThemeColors {
    public static void main(String[] args) throws Exception {
        // Διαδρομή προς το αρχείο Excel σας
        String dataDir = "path_to_your_directory/";
        
        // Άνοιγμα ενός υπάρχοντος αρχείου Excel
        Workbook workbook = new Workbook(dataDir + "book1.xlsx");
        
        // Αλλαγή χρώματος θέματος Background1 σε κόκκινο
        workbook.setThemeColor(ThemeColorType.BACKGROUND_1, Color.getRed());
        System.out.println("Background1 Theme Color changed to: Red");
        
        // Αλλαγή χρώματος θέματος Accent2 σε μπλε
        workbook.setThemeColor(ThemeColorType.ACCENT_1, Color.getBlue());
        System.out.println("Accent2 Theme Color changed to: Blue");
        
        // Αποθήκευση του ενημερωμένου αρχείου
        workbook.save(dataDir + "GetSetThemeColors_out.xlsx");
    }
}
```

Αυτός ο κώδικας δείχνει πώς να αλλάξετε και να επιβεβαιώσετε τις τροποποιήσεις χρώματος θέματος.
