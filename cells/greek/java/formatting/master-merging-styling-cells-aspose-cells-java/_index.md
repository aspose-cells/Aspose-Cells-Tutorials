## Οδηγός Εφαρμογής

### Συγχώνευση κελιών

**Επισκόπηση:** Αυτή η λειτουργία συνδυάζει γειτονικά κελιά σε μία ενιαία οντότητα, ιδανική για τη δημιουργία τίτλων ή κεφαλίδων που εκτείνονται σε πολλαπλές στήλες.

#### Βήμα προς βήμα:

**1. Δημιουργία και συγχώνευση εύρους**

```java
import com.aspose.cells.Range;
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

Workbook workbook = new Workbook();
Worksheet _worksheet = workbook.getWorksheets().get(0);
Range range = _worksheet.getCells().createRange(0, 0, 1, 2); // Α1:Β1
range.merge(); // Συγχώνευση κελιών A1 και B1
_worksheet.getCells().get(0, 0).setValue("A quick brown fox...");
workbook.save(outDir + "MergedCells.xlsx");
```
- **Επεξήγηση παραμέτρων:** `createRange(0, 0, 1, 2)` Καθορίζει την επάνω αριστερή γωνία (γραμμή 0, στήλη 0) και εκτείνεται σε μία γραμμή πάνω από δύο στήλες.
- **Σκοπός:** Η συγχώνευση κελιών βοηθά στην ενοποίηση δεδομένων για καλύτερη οπτικοποίηση.

### Εφαρμογή στυλ σε κελιά

**Επισκόπηση:** Βελτιώστε την παρουσίαση κελιών εφαρμόζοντας στυλ όπως η αναδίπλωση κειμένου, διασφαλίζοντας ότι το περιεχόμενο ταιριάζει άψογα μέσα στα συγχωνευμένα κελιά.

#### Βήμα προς βήμα:

**1. Ενεργοποίηση αναδίπλωσης κειμένου**

```java
import com.aspose.cells.Style;

Worksheet _worksheet = workbook.getWorksheets().get(0);
Style style = _worksheet.getCells().get(0, 0).getStyle();
style.setTextWrapped(true); // Ενεργοποίηση αναδίπλωσης κειμένου
_worksheet.getCells().get(0, 0).setStyle(style);
```
- **Διαμόρφωση κλειδιού:** `setTextWrapped(true)` διασφαλίζει ότι τα μεγάλα κείμενα δεν υπερχειλίζουν εκτός των ορίων των κελιών.

### Αυτόματη προσαρμογή γραμμών για συγχωνευμένα κελιά

**Επισκόπηση:** Αυτόματη προσαρμογή του ύψους της γραμμής ώστε να χωράει το περιεχόμενο μέσα στα συγχωνευμένα κελιά, διατηρώντας μια καθαρή και ευανάγνωστη μορφή.

#### Βήμα προς βήμα:

**1. Ρύθμιση παραμέτρων επιλογών αυτόματης προσαρμογής**

```java
import com.aspose.cells.AutoFitMergedCellsType;
import com.aspose.cells.AutoFitterOptions;

AutoFitterOptions options = new AutoFitterOptions();
options.setAutoFitMergedCellsType(AutoFitMergedCellsType.EACH_LINE); // Τοποθετήστε κάθε γραμμή ξεχωριστά
_worksheet.autoFitRows(options);
```
- **Σκοπός της μεθόδου:** `autoFitRows` προσαρμόζει τις γραμμές με βάση το ύψος του περιεχομένου, βελτιστοποιώντας την αναγνωσιμότητα.
