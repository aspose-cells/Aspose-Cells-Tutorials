Αυτός ο οδηγός θα σας βοηθήσει να ρυθμίσετε και να διαμορφώσετε αποτελεσματικά τις πηγές γραμματοσειρών, διασφαλίζοντας ότι οι εφαρμογές σας χρησιμοποιούν την ακριβή τυπογραφία που απαιτείται.

## Οδηγός Εφαρμογής

Ας δούμε πώς να ρυθμίσετε προσαρμοσμένες γραμματοσειρές στην εφαρμογή Java σας χρησιμοποιώντας το Aspose.Cells.

### Ρύθμιση φακέλων προσαρμοσμένων γραμματοσειρών

#### Επισκόπηση
Μπορείτε να καθορίσετε καταλόγους όπου το Aspose.Cells θα αναζητά αρχεία γραμματοσειρών. Αυτό διασφαλίζει ότι χρησιμοποιούνται οι σωστές γραμματοσειρές κατά τη δημιουργία εγγράφων Excel.

##### Βήμα 1: Ορισμός διαδρομών φακέλων γραμματοσειρών

Αρχικά, ορίστε διαδρομές προς τους φακέλους προσαρμοσμένων γραμματοσειρών σας:

```java
String dataDir = Utils.getSharedDataDir(SetCustomFontFolders.class) + "TechnicalArticles/";
String fontFolder1 = dataDir + "/Arial";
String fontFolder2 = dataDir + "/Calibri";
```

##### Βήμα 2: Ορισμός φακέλου γραμματοσειρών

Χρησιμοποιήστε το `setFontFolder` μέθοδος για τον καθορισμό ενός φακέλου. Η δεύτερη παράμετρος επιτρέπει την αναδρομική αναζήτηση εντός υποκαταλόγων:

```java
FontConfigs.setFontFolder(fontFolder1, true);
```

##### Βήμα 3: Ορισμός πολλαπλών φακέλων γραμματοσειρών

Για να ορίσετε πολλούς φακέλους ταυτόχρονα χωρίς επανάληψη, χρησιμοποιήστε `setFontFolders`:

```java
FontConfigs.setFontFolders(new String[] { fontFolder1, fontFolder2 }, false);
```

### Ρύθμιση παραμέτρων πηγών γραμματοσειρών

#### Επισκόπηση
Μπορούν να οριστούν διαφορετικές πηγές γραμματοσειρών για την ενίσχυση της ευελιξίας. Αυτές περιλαμβάνουν πηγές που βασίζονται σε φακέλους, αρχεία και στη μνήμη.

##### Βήμα 4: Ορισμός FolderFontSource

Δημιουργήστε ένα `FolderFontSource` αντικείμενο για γραμματοσειρές που βασίζονται σε καταλόγους:

```java
FolderFontSource sourceFolder = new FolderFontSource(fontFolder1, false);
```

##### Βήμα 5: Ορισμός FileFontSource

Καθορίστε ένα μεμονωμένο αρχείο γραμματοσειράς χρησιμοποιώντας `FileFontSource`:

```java
String fontFile = dataDir + "/Arial/arial.ttf";
FileFontSource sourceFile = new FileFontSource(fontFile);
```

##### Βήμα 6: Ορισμός MemoryFontSource

Για γραμματοσειρές στη μνήμη, διαβάστε τον πίνακα byte και δημιουργήστε έναν `MemoryFontSource`:

```java
byte[] bytes = Files.readAllBytes(new File(fontFile).toPath());
MemoryFontSource sourceMemory = new MemoryFontSource(bytes);
```

##### Βήμα 7: Ορισμός πηγών γραμματοσειράς

Συνδυάστε όλες τις πηγές χρησιμοποιώντας `setFontSources`:

```java
FontConfigs.setFontSources(new FontSourceBase[] { sourceFolder, sourceFile, sourceMemory });
```
