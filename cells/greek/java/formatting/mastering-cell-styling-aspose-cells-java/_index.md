## Οδηγός Εφαρμογής

### Δημιουργία και πρόσβαση στο βιβλίο εργασίας

#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει τον τρόπο δημιουργίας ενός βιβλίου εργασίας και πρόσβασης στο πρώτο φύλλο εργασίας του.

**Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας**
Ξεκινήστε δημιουργώντας μια παρουσία του `Workbook`, το οποίο αντιπροσωπεύει το αρχείο Excel σας:
```java
// Καθορίστε καταλόγους για την εισαγωγή και την έξοδο δεδομένων
String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργία νέου βιβλίου εργασίας από ένα υπάρχον αρχείο
Workbook workbook = new Workbook(dataDir + "/book1.xls");
```
**Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας**
Η πρόσβαση σε φύλλα εργασίας σάς επιτρέπει να χειρίζεστε απευθείας κελιά:
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Cells cells = worksheet.getCells();
```

### Τροποποίηση στυλ κελιών

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τον τρόπο τροποποίησης στυλ κελιών, συμπεριλαμβανομένης της στοίχισης κειμένου και της προσαρμογής γραμματοσειράς.

**Βήμα 1: Πρόσβαση στο κελί "A1"**
Εντοπίστε ένα συγκεκριμένο κελί που θέλετε να διαμορφώσετε:
```java
Cell cell = cells.get("A1");
cell.setValue("Hello Aspose!");
```
**Βήμα 2: Δημιουργία και εφαρμογή στυλ**
Δημιουργήστε ένα νέο `Style` αντικείμενο, διαμορφώστε το και εφαρμόστε το στο κελί σας:
```java
Style style = workbook.createStyle();
style.setVerticalAlignment(TextAlignmentType.CENTER);
style.setHorizontalAlignment(TextAlignmentType.CENTER);

Font font = style.getFont();
font.setColor(Color.getGreen());
style.setShrinkToFit(true);
style.setBorder(BorderType.BOTTOM_BORDER, CellBorderType.MEDIUM, Color.getRed());

cell.setStyle(style);
```
**Βήμα 3: Αποθήκευση του βιβλίου εργασίας**
Μετά το στυλ, αποθηκεύστε τις αλλαγές σας σε ένα αρχείο Excel:
```java
workbook.save(outDir + "/FCUsingStyleObject_out.xls");
```
