## Οδηγός Εφαρμογής
### Επισκόπηση της προσαρμοσμένης μορφοποίησης δεκαδικών αριθμών
Η προσαρμοσμένη μορφοποίηση δεκαδικών αριθμών σάς επιτρέπει να ελέγχετε τον τρόπο εμφάνισης των αριθμών στην εφαρμογή σας, ακολουθώντας τη μορφή αριθμών του Excel. Αυτή η λειτουργία είναι κρίσιμη για εφαρμογές που απαιτούν ακριβή αναπαράσταση δεδομένων.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργήστε ένα βιβλίο εργασίας και ένα φύλλο εργασίας της Access**
Αρχικά, δημιουργήστε μια παρουσία του `Workbook` τάξη και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας της:
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class Main {
    public static void main(String[] args) throws Exception {
        // Αρχικοποίηση βιβλίου εργασίας
        Workbook wb = new Workbook();
        
        // Πρώτο φύλλο εργασίας της Access
        Worksheet ws = wb.getWorksheets().get(0);
    }
}
```
**2. Εισαγωγή και μορφοποίηση τιμής**
Εισαγάγετε μια τιμή στο κελί A1 και εφαρμόστε προσαρμοσμένη μορφοποίηση δεκαδικού αριθμού:
```java
import com.aspose.cells.Cell;
import com.aspose.cells.Style;

public class Main {
    public static void main(String[] args) throws Exception {
        // Προηγούμενος κώδικας...

        // Αποκτήστε πρόσβαση στο κελί A1 και εισάγετε την τιμή 123
        Cell cell = ws.getCells().get("A1");
        cell.putValue(123);

        // Εφαρμογή προσαρμοσμένης μορφοποίησης δεκαδικού αριθμού
        Style st = cell.getStyle();
        st.setCustom("[DBNum2][$-804]General");

        // Ορισμός του μορφοποιημένου στυλ στο κελί
        cell.setStyle(st);
    }
}
```
**3. Αποθήκευση βιβλίου εργασίας ως PDF**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε μορφή PDF:
```java
import com.aspose.cells.SaveFormat;

public class Main {
    public static void main(String[] args) throws Exception {
        // Προηγούμενος κώδικας...

        // Ορισμός πλάτους στήλης και αποθήκευση του βιβλίου εργασίας
        ws.getCells().setColumnWidth(0, 30);
        wb.save("outputDecimalNumberFormatting.pdf", SaveFormat.PDF);

        System.out.println("Custom Decimal Number Formatting Done Successfully");
    }
}
```
