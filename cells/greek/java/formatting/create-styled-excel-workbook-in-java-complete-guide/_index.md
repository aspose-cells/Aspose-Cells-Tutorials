# Δημιουργία Στυλιζαρισμένου Βιβλίου Εργασίας Excel σε Java – Πλήρης Οδηγός

Έχετε ποτέ χρειαστεί να **create styled excel workbook** από τη Java αλλά να νιώσατε υπερφορτωμένοι από τις λεπτομέρειες μορφοποίησης; Δεν είστε μόνοι. Είτε εξάγετε δεδομένα από μια βάση δεδομένων είτε εξάγετε μια αναφορά, η απόκτηση της σωστής εμφάνισης—σύμβολα νομισμάτων, μάσκες ημερομηνίας, έντονες κεφαλίδες—μπορεί να φαίνεται σαν μια εργασία.

Σε αυτό το σεμινάριο θα περάσουμε βήμα‑βήμα μια πρακτική λύση που **adds custom formatting to excel**, σας επιτρέπει να **set date format excel column**, και τελικά **save workbook as xlsx** χωρίς καμία δυσκολία. Έτοιμοι να το δείτε σε δράση;

## Τι Θα Μάθετε

- Πώς να δημιουργήσετε ένα Aspose.Cells `Workbook` από το μηδέν.  
- Τρόποι εφαρμογής ενός πίνακα στυλ που καλύπτει στήλες General, Currency και Date.  
- Η ακριβής κλήση που **creates styled excel workbook** με ενσωματωμένα ονόματα στηλών.  
- Πώς να αποθηκεύσετε το αρχείο χρησιμοποιώντας τη μέθοδο **save workbook as xlsx**.  

### Προαπαιτούμενα

- Java 8+ (ο κώδικας λειτουργεί σε οποιοδήποτε πρόσφατο JDK).  
- Βιβλιοθήκη Aspose.Cells for Java (μπορείτε να κατεβάσετε το τελευταίο JAR από το Maven Central).  
- Μια μικρή εξοικείωση με συλλογές (`List`, `Map`).  

Αν κάποιο από αυτά σας φαίνεται άγνωστο, κάντε ένα διάλειμμα και εγκαταστήστε το Aspose.Cells JAR· το υπόλοιπο θα κυλήσει φυσικά.

## Δημιουργία Στυλιζαρισμένου Βιβλίου Εργασίας Excel – Ρύθμιση του Έργου

Πρώτα απ' όλα: δημιουργήστε ένα νέο έργο Maven (ή Gradle) και προσθέστε το Aspose.Cells ως εξάρτηση.

```xml
<!-- pom.xml snippet -->
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>23.12</version> <!-- use the latest version -->
</dependency>
```

> **Pro tip:** Αν χρησιμοποιείτε Gradle, οι ίδιες συντεταγμένες λειτουργούν με `implementation 'com.aspose:aspose-cells:23.12'`.

Γιατί να ξεκινήσετε με μια ρύθμιση Maven; Φέρνει αυτόματα όλες τις διαμεσολαβημένες εξαρτήσεις, ώστε να μην χάνετε χρόνο ψάχνοντας τα σωστά JARs αργότερα.

## Προσθήκη Προσαρμοσμένης Μορφοποίησης σε Στήλες Excel

Τώρα που το έργο είναι έτοιμο, ας ορίσουμε τα στυλ που θέλουμε. Στο Aspose.Cells μπορείτε να περάσετε έναν πίνακα συμβολοσειρών μορφοποίησης που αντιστοιχούν απευθείας σε κάθε στήλη.

```java
// Define column styles: General, Currency, Date
String[] columnStyles = {
    "General",          // Column A – plain text or numbers
    "$#,##0.00",        // Column B – currency with two decimals
    "mm/dd/yyyy"        // Column C – US‑style date
};
```

Το βήμα **add custom formatting to excel** είναι τόσο απλό όσο η δημιουργία αυτού του πίνακα συμβολοσειρών.  
Κάθε καταχώρηση ακολουθεί το ίδιο μοτίβο που θα χρησιμοποιούσατε στον διάλογο μορφοποίησης “Custom” του Excel, οπότε δεν χρειάζεται να μάθετε νέα σύνταξη.

## Ορισμός Μορφής Ημερομηνίας σε Στήλη Excel

Εάν τα δεδομένα σας περιλαμβάνουν ημερομηνίες, θα θέλετε να βεβαιωθείτε ότι εμφανίζονται σταθερά. Η τρίτη καταχώρηση στο `columnStyles` (`"mm/dd/yyyy"`) κάνει ακριβώς αυτό—λέει στο Aspose.Cells να αντιμετωπίζει την τρίτη στήλη ως ημερομηνία και να την εμφανίζει με τη σειρά μήνας/ημέρα/έτος.

```java
// Example: Adding a row with a Date object
Map<String, Object> row = new HashMap<>();
row.put("Product", "Widget");
row.put("Price", 19.99);
row.put("ReleaseDate", new java.util.Date()); // Today’s date
dataRows.add(row);
```

Όταν το βιβλίο εργασίας δημιουργηθεί, η οδηγία **set date format excel column** εξασφαλίζει ότι κάθε κελί `ReleaseDate` φαίνεται όπως `04/27/2026` (ή όποια είναι η σημερινή ημερομηνία). Δεν απαιτείται καμία επεξεργασία μετά.

## Εισαγωγή Δεδομένων και Εφαρμογή των Στυλ

Με τα στυλ ορισμένα, το επόμενο βήμα είναι να τροφοδοτήσουμε τα δεδομένα μας στο φύλλο εργασίας. Το Aspose.Cells παρέχει τη μέθοδο `importCustomObjects`, η οποία όχι μόνο εισάγει τα δεδομένα αλλά και εφαρμόζει τον πίνακα στυλ που μόλις δημιουργήσαμε.

```java
// Step 1: Simulate fetching rows from a DB
List<Map<String, Object>> dataRows = fetchRows(); // see helper below

// Step 2: Create a fresh workbook and grab the first sheet
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);

// Step 3: Import data with custom styles; true = first row has headers
worksheet.getCells().importCustomObjects(
        dataRows,        // data source
        0,               // start row (0‑based)
        0,               // start column (0‑based)
        columnStyles,    // style array from above
        true);           // first row contains column names
```

### Γιατί `importCustomObjects`;

- **Single‑call magic:** Γράφει τα δεδομένα *και* εφαρμόζει τη μορφοποίηση σε μία κλήση.  
- **Header awareness:** Η μεταβίβαση του `true` λέει στο API να αντιμετωπίζει τα κλειδιά του πρώτου χάρτη ως κεφαλίδες στηλών, κάτι που είναι ιδανικό για σενάρια **create styled excel workbook** όπου θέλετε αναγνώσιμους τίτλους στηλών.  
- **Performance:** Στο παρασκήνιο, το Aspose ομαδοποιεί τις λειτουργίες εγγραφής, κάνοντάς το ταχύτερο από την επανάληψη κελιού‑κατά‑κελί.

> **Watch out:** Η σειρά των κλειδιών σε κάθε `Map` έχει σημασία. Χρησιμοποιήστε ένα `LinkedHashMap` εάν χρειάζεστε καθορισμένη σειρά στηλών.

## Αποθήκευση Βιβλίου Εργασίας ως Αρχείο XLSX

Όλη αυτή η δουλειά θα ήταν μάταιη αν δεν μπορείτε να αποθηκεύσετε το αρχείο. Η τελευταία γραμμή είναι μια εντολή μιας γραμμής που **save workbook as xlsx**.

```java
// Step 5: Persist the workbook
String outputPath = "output/DataTableStyled.xlsx";
workbook.save(outputPath);
System.out.println("Workbook saved to " + outputPath);
```

Η μέθοδος `save` εντοπίζει αυτόματα την επέκταση του αρχείου. Καθορίζοντας `.xlsx`, εξασφαλίζετε τη σύγχρονη μορφή Office Open XML, η οποία είναι αυτή που αναμένουν τα περισσότερα εργαλεία downstream.

## Πλήρες, Εκτελέσιμο Παράδειγμα

Παρακάτω βρίσκεται η πλήρης κλάση που μπορείτε να αντιγράψετε απευθείας στο IDE σας. Περιλαμβάνει μια ψεύτικη μέθοδο `fetchRows` που επιστρέφει δείγμα δεδομένων, ώστε να μην χρειάζεστε πραγματική βάση δεδομένων για δοκιμή.

```java
import com.aspose.cells.*;
import java.util.*;

public class ImportDataTableWithStyle {
    public static void main(String[] args) throws Exception {
        // ---------- Step 1: Get data ----------
        List<Map<String, Object>> dataRows = fetchRows();

        // ---------- Step 2: Create workbook ----------
        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // ---------- Step 3: Define column styles ----------
        String[] columnStyles = { "General", "$#,##0.00", "mm/dd/yyyy" };

        // ---------- Step 4: Import data with styles ----------
        // true = first row contains column names
        worksheet.getCells().importCustomObjects(dataRows, 0, 0, columnStyles, true);

        // ---------- Step 5: Save as XLSX ----------
        String outPath = "output/DataTableStyled.xlsx";
        workbook.save(outPath);
        System.out.println("Workbook saved to " + outPath);
    }

    // Mock method – replace with real DB call if needed
    private static List<Map<String, Object>> fetchRows() {
        List<Map<String, Object>> rows = new ArrayList<>();

        Map<String, Object> row1 = new LinkedHashMap<>();
        row1.put("Product", "Gadget");
        row1.put("Price", 49.95);
        row1.put("ReleaseDate", new Date()); // today
        rows.add(row1);

        Map<String, Object> row2 = new LinkedHashMap<>();
        row2.put("Product", "Widget");
        row2.put("Price", 19.99);
        Calendar cal = Calendar.getInstance();
        cal.set(2025, Calendar.DECEMBER, 15);
        row2.put("ReleaseDate", cal.getTime());
        rows.add(row2);

        return rows;
    }
}
```

Τρέξτε το, ανοίξτε το παραγόμενο αρχείο, και θα δείτε το αποτέλεσμα **create styled excel workbook** που περιμένατε.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις

**What if I need more than three columns?**  
Απλώς επεκτείνετε τον πίνακα `columnStyles`—κάθε καταχώρηση αντιστοιχεί σε μια στήλη με τη σειρά. Για στήλες που θέλετε να αφήσετε αμετάβλητες, χρησιμοποιήστε `"General"`.

**Can I apply fonts or colors?**  
Ναι. Αντί για μια συμβολοσειρά μορφοποίησης, μπορείτε να δημιουργήσετε ένα αντικείμενο `Style`, να το αναθέσετε σε ένα `StyleFlag`, και να περάσετε έναν πίνακα `Style[]` στη `importCustomObjects`. Αυτό είναι πιο προχωρημένο, αλλά η αρχή παραμένει η ίδια.
