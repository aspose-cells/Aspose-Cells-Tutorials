## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία πρόσβασης και ενημέρωσης γραμματοσειρών σε συγκεκριμένα τμήματα ενός κελιού στο φύλλο Excel.

### Πρόσβαση στις ρυθμίσεις γραμματοσειράς
Για να αποκτήσουμε πρόσβαση στις ρυθμίσεις γραμματοσειράς, θα ξεκινήσουμε φορτώνοντας ένα υπάρχον βιβλίο εργασίας και ανακτώντας το επιθυμητό κελί:

**Βήμα 1: Φόρτωση βιβλίου εργασίας και επιλογή κελιού**
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;
import com.aspose.cells.Cell;

Workbook workbook = new Workbook("source.xlsx");
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("A1");

System.out.println("Before updating the font settings....");
```

**Βήμα 2: Ανάκτηση ρυθμίσεων γραμματοσειράς**
```java
import com.aspose.cells.FontSetting;

FontSetting[] fontSettings = cell.getCharacters();

for (int i = 0; i < fontSettings.length; i++) {
    System.out.println(fontSettings[i].getFont().getName());
}
```
Αυτό το βήμα ανακτά και εκτυπώνει τις τρέχουσες γραμματοσειρές που έχουν εφαρμοστεί σε διαφορετικά εύρη χαρακτήρων εντός του καθορισμένου κελιού.

### Ενημέρωση ρυθμίσεων γραμματοσειράς
Μόλις αποκτήσετε πρόσβαση στις ρυθμίσεις γραμματοσειράς, η τροποποίησή τους είναι απλή:

**Βήμα 3: Τροποποίηση γραμματοσειράς**
```java
// Αλλάξτε το όνομα γραμματοσειράς της πρώτης ρύθμισης γραμματοσειράς σε "Arial"
fontSettings[0].getFont().setName("Arial");
```

**Βήμα 4: Εφαρμογή αλλαγών**
```java
cell.setCharacters(fontSettings);
System.out.println("\nAfter updating the font settings....");

for (int i = 0; i < fontSettings.length; i++) {
    System.out.println(fontSettings[i].getFont().getName());
}
```
Εδώ, ενημερώνουμε την πρώτη ρύθμιση γραμματοσειράς σε "Arial" και εφαρμόζουμε αυτές τις αλλαγές ξανά στο κελί.

### Αποθήκευση αλλαγών

**Βήμα 5: Αποθήκευση βιβλίου εργασίας**
```java
workbook.save("AAUPortions_out.xlsx");
System.out.println("Workbook saved successfully.");
```
