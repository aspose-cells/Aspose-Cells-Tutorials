# Βελτιστοποίηση χώρων HTML και αυτόματης προσαρμογής στηλών στο Excel χρησιμοποιώντας το Aspose.Cells για Java

## Εισαγωγή

Δυσκολεύεστε με ακατάστατα δεδομένα HTML που προκαλούν προβλήματα μορφοποίησης κατά την εισαγωγή τους στο Excel; Περιττοί χώροι, ειδικά μετά... `<br>` ετικέτες, μπορούν να διαταράξουν την οπτική συνοχή των δεδομένων σας. Αυτό το σεμινάριο σας καθοδηγεί στη χρήση του Aspose.Cells για Java για να βελτιστοποιήσετε το περιεχόμενο HTML σας αφαιρώντας περιττά κενά και προσαρμόζοντας αυτόματα στήλες σε φύλλα Excel. Κατακτώντας αυτές τις λειτουργίες, εξασφαλίζετε καθαρότερες εισαγωγές δεδομένων και καλύτερη παρουσίαση.

**Τι θα μάθετε:**
- Αφαίρεση περιττών κενών από συμβολοσειρές HTML με το Aspose.Cells.
- Τεχνικές για αυτόματη προσαρμογή στηλών σε φύλλα εργασίας του Excel χρησιμοποιώντας Java.
- Πρακτικά παραδείγματα ενσωμάτωσης αυτών των χαρακτηριστικών στα έργα σας.

Ας ξεκινήσουμε ρυθμίζοντας το περιβάλλον!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

### Απαιτούμενες βιβλιοθήκες
Συμπεριλάβετε το Aspose.Cells για Java ως εξάρτηση. Χρησιμοποιήστε την ενσωμάτωση Maven ή Gradle όπως καθορίζεται παρακάτω.

### Απαιτήσεις Ρύθμισης Περιβάλλοντος
Βεβαιωθείτε ότι έχετε εγκαταστήσει ένα Java Development Kit (JDK) στον υπολογιστή σας, κατά προτίμηση έκδοση 8 ή νεότερη.

### Προαπαιτούμενα Γνώσεων
Η βασική εξοικείωση με τον προγραμματισμό Java και η κατανόηση των λειτουργιών με αρχεία Excel είναι ωφέλιμες για αυτό το σεμινάριο.

## Ρύθμιση του Aspose.Cells για Java

Για να χρησιμοποιήσετε το Aspose.Cells στο έργο Java σας, συμπεριλάβετέ το ως εξάρτηση:

**Maven**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Γκράντλ**
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα καλύψουμε δύο βασικές λειτουργίες: τη διαγραφή περιττών κενών από την HTML και την αυτόματη προσαρμογή στηλών σε φύλλα εργασίας του Excel.

### Λειτουργία 1: Διαγραφή πλεοναζόντων κενών από HTML

**Επισκόπηση:**
Αυτή η λειτουργία βελτιστοποιεί το περιεχόμενο HTML αφαιρώντας τα περιττά κενά που συχνά ακολουθούν `<br>` ετικέτες. Εξασφαλίζει καθαρότερα δεδομένα κατά τη μετατροπή HTML σε βιβλίο εργασίας Excel χρησιμοποιώντας το Aspose.Cells.

#### Βήμα προς βήμα εφαρμογή

**Εισαγωγή απαιτούμενων βιβλιοθηκών**
Αρχικά, εισαγάγετε τις απαραίτητες κλάσεις:
```java
import com.aspose.cells.*;
```

**Ορισμός καταλόγων εισόδου και εξόδου**
Ορίστε τους καταλόγους δεδομένων και εξόδου σας ανάλογα:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";
```

**Προετοιμασία περιεχομένου HTML**
Ορίστε ένα δείγμα συμβολοσειράς HTML με πλεονάζοντα κενά:
```java
String html = "<html><body><table><tr><td><br>\t\tThis is sample data<br>\t\tThis is sample data<br>\t\tThis is sample data</td></tr></table></body></html>";
```

**Μετατροπή HTML σε πίνακα byte**
Για επεξεργασία, μετατρέψτε τη συμβολοσειρά HTML σε έναν πίνακα byte:
```java
byte[] byteArray = html.getBytes();
```

**Αρχικοποίηση HtmlLoadOptions**
Στήνω `HtmlLoadOptions` με ενεργοποιημένη τη διαγραφή πλεονάζοντος χώρου:
```java
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.HTML);
loadOptions.setDeleteRedundantSpaces(true);
```

**Φόρτωση περιεχομένου HTML στο βιβλίο εργασίας**
Δημιουργήστε ένα InputStream και φορτώστε το περιεχόμενο σε ένα `Workbook`:
```java
java.io.ByteArrayInputStream stream = new java.io.ByteArrayInputStream(byteArray);
Workbook workbook = new Workbook(stream, loadOptions);
```

**Αυτόματη Προσαρμογή Στηλών και Αποθήκευση Εξόδου**
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και στις στήλες αυτόματης προσαρμογής για καλύτερη ορατότητα:
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
worksheet.autoFitColumns();
workbook.save(outDir + "/DRSFromHtml_out-" + loadOptions.getDeleteRedundantSpaces() + ".xlsx", SaveFormat.XLSX);
```

### Χαρακτηριστικό 2: Αυτόματη προσαρμογή στηλών βιβλίου εργασίας

**Επισκόπηση:**
Αυτή η λειτουργία προσαρμόζει τα πλάτη των στηλών σε ένα φύλλο εργασίας του Excel ώστε να ταιριάζουν στο περιεχόμενό τους, εξασφαλίζοντας μια καθαρή και οργανωμένη εμφάνιση.

#### Βήμα προς βήμα εφαρμογή

**Φόρτωση υπάρχοντος βιβλίου εργασίας**
Φορτώστε το υπάρχον αρχείο βιβλίου εργασίας σας:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

Workbook workbook = new Workbook(dataDir + "/sample.xlsx");
```

**Φύλλο εργασίας της Access και Αυτόματη Προσαρμογή Στηλών**
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και στις στήλες αυτόματης προσαρμογής:
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
worksheet.autoFitColumns();
workbook.save(outDir + "/AutoFitSample_out.xlsx", SaveFormat.XLSX);
```
