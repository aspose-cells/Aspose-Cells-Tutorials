## Οδηγός Εφαρμογής

### Δημιουργία βιβλίου εργασίας και πρόσβαση σε κελιά

Αρχικά, δημιουργήστε μια παρουσία βιβλίου εργασίας και αποκτήστε πρόσβαση στα κελιά της:

```java
// Δημιουργία νέου αντικειμένου βιβλίου εργασίας

// Άνοιγμα του πρώτου φύλλου εργασίας στο βιβλίο εργασίας

// Λήψη της συλλογής κελιών από το φύλλο εργασίας

```

### Ρύθμιση παραμέτρων πλάτους στήλης και ύψους γραμμής

Προσαρμόστε το πλάτος της στήλης και το ύψος της γραμμής για να διασφαλίσετε ότι το κείμενο ταιριάζει άψογα:

```java
// Αύξηση του πλάτους της πρώτης στήλης
cells.setColumnWidth(0, 35);

// Αυξήστε το ύψος της πρώτης σειράς
cells.setRowHeight(0, 65);
```

### Προσθήκη κειμένου και εφαρμογή στυλ αναδίπλωσης

Προσθήκη κειμένου σε ένα κελί και ενεργοποίηση αναδίπλωσης κειμένου:

```java
// Προσθήκη κειμένου στο πρώτο κελί

// Αποκτήστε το στυλ του κελιού

// Ενεργοποίηση αναδίπλωσης κειμένου για το περιεχόμενο του κελιού
style.setTextWrapped(true);

// Εφαρμογή του στυλ πίσω στο κελί
cells.get(0, 0).setStyle(style);
```
