- Δημιουργία νέου αντικειμένου βιβλίου εργασίας

## Ρύθμιση του Aspose.Cells για Java

Για να ενσωματώσετε το Aspose.Cells στα έργα Java σας, ρυθμίστε τη βιβλιοθήκη χρησιμοποιώντας το Maven ή το Gradle:

**Maven:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Βαθμός:**
```gradle
implementation(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Δημιουργία βιβλίου εργασίας

Δημιουργήστε ένα `Workbook` αντικείμενο για την αναπαράσταση του αρχείου Excel σας:

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου σας
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

### Χαρακτηριστικό: Πρόσβαση σε φύλλο εργασίας

Αποκτήστε πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας χρησιμοποιώντας το ευρετήριο ή το όνομά του:

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Workbook;

Worksheet worksheet = workbook.getWorksheets().get(0); // Πρώτο φύλλο εργασίας της Access
```

### Χαρακτηριστικό: Αυτόματη προσαρμογή σειράς

Αυτόματη προσαρμογή ύψους μιας γραμμής με βάση το περιεχόμενό της χρησιμοποιώντας `autoFitRow`:

```java
// Προσαρμόζει τη δεύτερη γραμμή (δείκτης 1)
worksheet.autoFitRow(1);
```

### Χαρακτηριστικό: Αυτόματη προσαρμογή στήλης

Αυτόματη προσαρμογή στηλών για να διασφαλίσετε ότι όλα τα περιεχόμενα είναι ορατά χωρίς περικοπή:

```java
// Προσαρμόζει το πλάτος της πρώτης στήλης (δείκτης 0)
worksheet.autoFitColumn(0);
```

### Χαρακτηριστικό: Αποθήκευση βιβλίου εργασίας

Αποθήκευση του βιβλίου εργασίας μετά τις προσαρμογές:

```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικατάσταση με την πραγματική διαδρομή καταλόγου εξόδου
workbook.save(outDir + "AutoFitRowsandColumns_out.xls");
```
