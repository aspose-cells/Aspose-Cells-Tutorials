## Οδηγός Εφαρμογής
### Εισαγωγή προσαρμοσμένων αντικειμένων σε συγχωνευμένα κελιά
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να εισάγετε δεδομένα από προσαρμοσμένα αντικείμενα σε προκαθορισμένα πρότυπα Excel με συγχωνευμένα κελιά, διατηρώντας τη μορφή και τη δομή.

#### Βήμα προς βήμα εφαρμογή
**1. Φόρτωση του βιβλίου εργασίας σας**
Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο Excel που περιέχει ένα πρότυπο με συγχωνευμένα κελιά:
```java
import com.aspose.cells.Workbook;
import AsposeCellsExamples.Utils;

String sourceDir = Utils.Get_SourceDirectory();
Workbook workbook = new Workbook(sourceDir + "sampleMergedTemplate.xlsx");
```

**2. Προετοιμάστε τα δεδομένα σας**
Δημιουργήστε μια λίστα με προσαρμοσμένα αντικείμενα που θέλετε να εισαγάγετε. Εδώ, χρησιμοποιούμε ένα απλό `Product` τάξη:
```java
import java.util.ArrayList;
import AsposeCellsExamples.Data.Product;

ArrayList<Product> productList = new ArrayList<>();
for (int i = 0; i < 3; i++) {
    productList.add(new Product("Test Product - " + i, i * 2));
}
```

**3. Διαμόρφωση επιλογών εισαγωγής**
Στήνω `ImportTableOptions` για να διαχειριστείτε τον τρόπο εισαγωγής δεδομένων, ειδικά όσον αφορά τα συγχωνευμένα κελιά:
```java
import com.aspose.cells.ImportTableOptions;

ImportTableOptions tableOptions = new ImportTableOptions();
tableOptions.setCheckMergedCells(true); // Διασφαλίζει ότι τα δεδομένα σέβονται τη συγχώνευση κελιών
```

**4. Εκτέλεση εισαγωγής δεδομένων**
Χρήση `importCustomObjects` για να συμπληρώσετε το πρότυπο Excel με δεδομένα:
```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.SaveFormat;

Worksheet worksheet = workbook.getWorksheets().get(0);
worksheet.getCells().importCustomObjects(productList, 1, 0, tableOptions);

String outDir = Utils.Get_OutputDirectory();
workbook.save(outDir + "sampleMergedTemplate_out.xlsx", SaveFormat.XLSX);
```
### Συμβουλές αντιμετώπισης προβλημάτων
- **Λανθασμένη ευθυγράμμιση δεδομένων:** Εξασφαλίζω `setCheckMergedCells(true)` έχει οριστεί.
- **Λείπουν δεδομένα:** Επαληθεύστε ότι οι δείκτες αρχικών γραμμών και στηλών είναι σωστοί.
