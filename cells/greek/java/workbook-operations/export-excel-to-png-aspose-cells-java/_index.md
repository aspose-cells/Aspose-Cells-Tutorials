## Ρύθμιση του Aspose.Cells για Java
Το Aspose.Cells για Java είναι μια ισχυρή βιβλιοθήκη σχεδιασμένη για χειρισμό υπολογιστικών φύλλων. Παρακάτω ακολουθούν τα βήματα για να την ενσωματώσετε στο έργο σας χρησιμοποιώντας το Maven ή το Gradle:

### **Maven**
Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:

```xml
<dependency>
  <groupId>com.aspose</groupId>
  <artifactId>aspose-cells</artifactId>
  <version>25.3</version>
</dependency>
```

### **Γκράντλ**
Συμπεριλάβετε αυτήν τη γραμμή στο δικό σας `build.gradle` αρχείο:

```gradle

```

## Οδηγός Εφαρμογής

### Εξαγωγή φύλλου εργασίας ως εικόνα PNG
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε οποιοδήποτε φύλλο εργασίας στο αρχείο Excel σας σε εικόνα PNG υψηλής ποιότητας.

#### **Βήμα 1: Φόρτωση του βιβλίου εργασίας**
Ξεκινήστε φορτώνοντας το αρχείο Excel χρησιμοποιώντας το Aspose.Cells:

```java
String dataDir = "YOUR_DATA_DIRECTORY/"; // Καθορισμός διαδρομής καταλόγου πηγής
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα `Workbook` αντικείμενο, που αντιπροσωπεύει ολόκληρο το αρχείο Excel.

#### **Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας**
Αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε να εξαγάγετε:

```java
Worksheet worksheet = workbook.getWorksheets().get(0); // Λήψη πρώτου φύλλου εργασίας
```

Εδώ έχουμε πρόσβαση στο πρώτο φύλλο εργασίας του βιβλίου εργασίας. Μπορείτε να κάνετε επανάληψη στα φύλλα εργασίας, εάν χρειάζεται.

#### **Βήμα 3: Ρύθμιση παραμέτρων επιλογών εικόνας**
Στήνω `ImageOrPrintOptions` για να ορίσετε πώς θα αποδοθεί η εικόνα σας:

```java
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.setOnePagePerSheet(true); // Εξασφαλίζει μία σελίδα ανά φύλλο
opts.setImageType(ImageType.PNG); // Μορφή εξόδου ως PNG
opts.setDesiredSize(400, 400); // Ορίστε τις επιθυμητές διαστάσεις (πλάτος και ύψος)
```

Αυτές οι επιλογές διασφαλίζουν ότι το φύλλο εργασίας αποδίδεται σε μια εικόνα μίας σελίδας με καθορισμένες διαστάσεις.

#### **Βήμα 4: Απόδοση και αποθήκευση της εικόνας**
Χρήση `SheetRender` για την απόδοση του φύλλου εργασίας:

```java
SheetRender sr = new SheetRender(worksheet, opts);
sr.toImage(0, "YOUR_OUTPUT_DIRECTORY/EWSheetToImage_out.png");
```

Αυτός ο κώδικας δημιουργεί μια εικόνα της πρώτης σελίδας του φύλλου εργασίας σας και την αποθηκεύει ως αρχείο PNG στον καθορισμένο κατάλογο εξόδου.
