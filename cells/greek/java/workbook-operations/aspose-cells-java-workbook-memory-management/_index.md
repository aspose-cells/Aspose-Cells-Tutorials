# Mastering Aspose.Cells Java: Αποτελεσματική διαχείριση βιβλίων εργασίας στη μνήμη

## Εισαγωγή

Το Aspose.Cells για Java παρέχει μια ισχυρή λύση για τη δημιουργία, αποθήκευση και φόρτωση βιβλίων εργασίας Excel απευθείας από τη μνήμη, εξαλείφοντας την ανάγκη για ενδιάμεση αποθήκευση σε δίσκο και βελτιώνοντας την απόδοση.

- Τοποθετήστε χαρτί συγκεκριμένων μεγεθών (A5 ή A3)
- Μετατρέψτε το σε μορφή PDF

Ενσωματώστε το Aspose.Cells στο έργο σας χρησιμοποιώντας είτε το Maven είτε το Gradle:

**Maven:**
```xml

    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Βαθμός:**

compile group: 'com.aspose', name: 'aspose-cells', version: '25.3'

## Οδηγός Εφαρμογής

### Δημιουργία και αποθήκευση βιβλίου εργασίας στη μνήμη

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να δημιουργήσετε ένα βιβλίο εργασίας του Excel και να το αποθηκεύσετε απευθείας σε μια ροή στη μνήμη, παρακάμπτοντας τον χώρο αποθήκευσης στο δίσκο. Αυτό είναι ιδιαίτερα χρήσιμο για εφαρμογές που πρέπει να επεξεργάζονται δεδομένα γρήγορα ή να χειρίζονται ευαίσθητες πληροφορίες με ασφάλεια.

**Βήμα προς βήμα εφαρμογή:**
1. **Αρχικοποίηση του Βιβλίου Εργασίας:**
   Ξεκινήστε δημιουργώντας ένα νέο `Workbook` αντικείμενο.
    ```java
    import com.aspose.cells.Workbook;
    import java.io.ByteArrayOutputStream;

    Workbook workbook = new Workbook();
    ```
2. **Αποθήκευση σε ροή στη μνήμη:**
   Χρησιμοποιήστε ένα `ByteArrayOutputStream` για να αποθηκεύσετε το αρχείο Excel στη μνήμη.
    ```java
    ByteArrayOutputStream baout = new ByteArrayOutputStream();
    workbook.save(baout, com.aspose.cells.SaveFormat.XLSX);
    ```

### Φόρτωση βιβλίου εργασίας από τη μνήμη με συγκεκριμένο μέγεθος χαρτιού

#### Επισκόπηση
Η φόρτωση βιβλίων εργασίας με συγκεκριμένα μεγέθη χαρτιού είναι ζωτικής σημασίας κατά τη μετατροπή εγγράφων σε PDF για εκτύπωση ή παρουσίαση. Αυτή η ενότητα καλύπτει τη φόρτωση του βιβλίου εργασίας στη μνήμη και τη ρύθμιση προσαρμοσμένων μεγεθών χαρτιού όπως A5 ή A3.

**Βήμα προς βήμα εφαρμογή:**
1. **Διαμόρφωση επιλογών φόρτωσης (A5):**
   Ρύθμιση παραμέτρων `LoadOptions` για να ορίσετε το μέγεθος χαρτιού σε A5 πριν από τη φόρτωση του βιβλίου εργασίας από έναν πίνακα byte.
    ```java
    import com.aspose.cells.LoadOptions;
    import com.aspose.cells.LoadFormat;
    import com.aspose.cells.PaperSizeType;
    import java.io.ByteArrayInputStream;

    byte[] bts = baout.toByteArray();
    ByteArrayInputStream bain = new ByteArrayInputStream(bts);

    LoadOptions optsA5 = new LoadOptions(LoadFormat.XLSX);
    optsA5.setPaperSize(PaperSizeType.PAPER_A_5);
    Workbook workbookA5 = new Workbook(bain, optsA5);
    ```
2. **Αποθήκευση ως PDF:**
   Μετατρέψτε το φορτωμένο βιβλίο εργασίας σε αρχείο PDF με το καθορισμένο μέγεθος χαρτιού.
    ```java
    String outDir = "YOUR_OUTPUT_DIRECTORY";
    workbookA5.save(outDir + "/output-a5.pdf");
    ```
3. **Τοποθέτηση χαρτιού μεγέθους A3:**
   Ομοίως, ορίστε το μέγεθος χαρτιού σε A3 και αποθηκεύστε:
    ```java
    LoadOptions optsA3 = new LoadOptions(LoadFormat.XLSX);
    optsA3.setPaperSize(PaperSizeType.PAPER_A_3);
    Workbook workbookA3 = new Workbook(bain, optsA3);
    workbookA3.save(outDir + "/LWorkbook_out.pdf");
    ```
