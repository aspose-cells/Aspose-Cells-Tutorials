
### Οδηγός Εφαρμογής

Τώρα, ας εμβαθύνουμε στην εφαρμογή των βασικών λειτουργιών του Aspose.Cells για τη δημιουργία και αποθήκευση αρχείων ODS.

#### Δημιουργία βιβλίου εργασίας και φύλλου εργασίας της Access

**Επισκόπηση**Ξεκινήστε δημιουργώντας ένα νέο βιβλίο εργασίας και αποκτώντας πρόσβαση στο πρώτο φύλλο εργασίας του. Αυτό αποτελεί τη βάση για τυχόν λειτουργίες που σχετίζονται με υπολογιστικά φύλλα.

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

public class CreateWorkbook {
    public static void main(String[] args) {
        // Αρχικοποίηση ενός νέου αντικειμένου βιβλίου εργασίας
        Workbook workbook = new Workbook();

        // Πρόσβαση στο πρώτο φύλλο εργασίας
        Worksheet worksheet = workbook.getWorksheets().get(0);

        System.out.println("Workbook and worksheet created!");
    }
}
```

#### Τροποποίηση τιμής κελιού

**Επισκόπηση**: Αλλάξτε εύκολα τις τιμές κελιών μέσα στο υπολογιστικό σας φύλλο. Αυτό το βήμα είναι κρίσιμο για τη δυναμική συμπλήρωση δεδομένων.

```java
import com.aspose.cells.Cell;

public class ModifyCellValue {
    public static void main(String[] args) {
        // Υποθέτοντας ότι το `φύλλο εργασίας` έχει ήδη αρχικοποιηθεί
        Cell cell = worksheet.getCells().get("A1");
        cell.putValue("Welcome to Aspose!");

        System.out.println("Cell value modified successfully!");
    }
}
```

#### Αποθήκευση αρχείου ODS με προεπιλεγμένες επιλογές

**Επισκόπηση**Μάθετε πώς να αποθηκεύετε το βιβλίο εργασίας σας ως αρχείο ODS χρησιμοποιώντας τις προεπιλεγμένες ρυθμίσεις, κατάλληλες για τις περισσότερες περιπτώσεις γενικής χρήσης.

```java
import com.aspose.cells.OdsSaveOptions;

public class SaveOdsFile {
    public static void main(String[] args) {
        String outDir = "YOUR_OUTPUT_DIRECTORY"; // Ορισμός καταλόγου εξόδου

        // Αποθήκευση βιβλίου εργασίας με τις προεπιλεγμένες επιλογές ODS
        OdsSaveOptions options = new OdsSaveOptions();
        workbook.save(outDir + "/SaveODSfile1_out.ods", options);

        System.out.println("File saved with default options!");
    }
}
```

#### Αποθήκευση αρχείου ODS με Strict Schema 1.1

**Επισκόπηση**Για σενάρια που απαιτούν αυστηρή τήρηση του σχήματος ODF 1.1, ρυθμίστε και αποθηκεύστε το αρχείο ODS σας ανάλογα.

```java
public class SaveOdsStrictSchema {
    public static void main(String[] args) {
        String outDir = "YOUR_OUTPUT_DIRECTORY"; // Ορισμός καταλόγου εξόδου

        // Ρύθμιση παραμέτρων επιλογών για αυστηρή συμμόρφωση με το ODF 1.1
        OdsSaveOptions options = new OdsSaveOptions();
        options.setStrictSchema11(true);
        workbook.save(outDir + "/SaveODSfile2_out.ods", options);

        System.out.println("File saved with strict schema!");
    }
}
```
