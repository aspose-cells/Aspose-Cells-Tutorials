## Οδηγός Εφαρμογής

### Εμφάνιση έκδοσης Aspose.Cells

#### Επισκόπηση
Η ανάκτηση και η εμφάνιση των πληροφοριών έκδοσης είναι ζωτικής σημασίας για τον εντοπισμό σφαλμάτων και τη διασφάλιση της συμβατότητας με διαφορετικές μορφές αρχείων Excel.

#### Βήμα προς βήμα εφαρμογή

**1. Εισαγωγή απαιτούμενων κλάσεων**
```java
import com.aspose.cells.*;
```

**2. Ανάκτηση και εκτύπωση πληροφοριών έκδοσης**
Δημιουργήστε μια τάξη `ShowAsposeCellsVersion` για να εκτελέσετε αυτήν την εργασία.
```java
public class ShowAsposeCellsVersion {
    public static void main(String[] args) throws Exception {
        // Βήμα 1: Αποκτήστε την έκδοση του Aspose.Cells για Java
        String version = CellsHelper.getVersion();
        
        // Βήμα 2: Εμφάνιση της έκδοσης στην κονσόλα
        System.out.println("Aspose.Cells for Java Version: " + version);
    }
}
```
- **Εξήγηση:** `CellsHelper.getVersion()` ανακτά την τρέχουσα έκδοση της βιβλιοθήκης, η οποία εκτυπώνεται χρησιμοποιώντας `System.out.println`.

### Δημιουργία κοινόχρηστου βιβλίου εργασίας

#### Επισκόπηση
Τα κοινόχρηστα βιβλία εργασίας επιτρέπουν σε πολλούς χρήστες να έχουν πρόσβαση και να επεξεργάζονται ένα αρχείο Excel ταυτόχρονα. Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη για συνεργατικά έργα.

#### Βήμα προς βήμα εφαρμογή

**1. Εισαγωγή απαιτούμενων κλάσεων**
```java
import com.aspose.cells.*;
```

**2. Δημιουργία και ρύθμιση παραμέτρων του βιβλίου εργασίας**
Δημιουργήστε μια τάξη `CreateSharedWorkbook` με την ακόλουθη υλοποίηση:
```java
public class CreateSharedWorkbook {
    // Διαδρομή κράτησης θέσης για τον κατάλογο εξόδου
    static String outDir = "YOUR_OUTPUT_DIRECTORY";

    public static void main(String[] args) throws Exception {
        // Βήμα 1: Αρχικοποίηση ενός νέου αντικειμένου βιβλίου εργασίας
        Workbook wb = new Workbook();
        
        // Βήμα 2: Ενεργοποίηση κοινόχρηστων ρυθμίσεων
        wb.getSettings().setShared(true);
        
        // Βήμα 3: Αποθήκευση του βιβλίου εργασίας ως αρχείο Excel
        wb.save(outDir + "outputSharedWorkbook.xlsx");
    }
}
```
- **Εξήγηση:** Ο `setShared(true)` Η μέθοδος διαμορφώνει το βιβλίο εργασίας ως κοινόχρηστο έγγραφο, το οποίο στη συνέχεια αποθηκεύεται χρησιμοποιώντας την καθορισμένη διαδρομή.
