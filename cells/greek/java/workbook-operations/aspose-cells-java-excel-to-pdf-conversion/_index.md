## Οδηγός Εφαρμογής

### Εμφάνιση έκδοσης Aspose.Cells

**Επισκόπηση:** Ο έλεγχος της έκδοσης του Aspose.Cells διασφαλίζει τη συμβατότητα και βοηθά στην αποσφαλμάτωση.

1. **Εισαγωγή απαραίτητων κλάσεων:**

   ```java
   import com.aspose.cells.CellsHelper;
   ```

2. **Εκτύπωση της έκδοσης:**

   Χρήση `CellsHelper.getVersion()` για να ανακτήσετε και να εμφανίσετε την τρέχουσα έκδοση:

   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου προέλευσης

   System.out.println("Aspose.Cells for Java Version: " + CellsHelper.getVersion());
   ```

### Φόρτωση αρχείου Excel

**Επισκόπηση:** Η φόρτωση ενός αρχείου Excel στο Aspose.Cells σάς επιτρέπει να το χειριστείτε και να το μετατρέψετε.

1. **Ρύθμιση μεταβλητών διαδρομής:**

   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου προέλευσης
   ```

2. **Φόρτωση του βιβλίου εργασίας:**

   Δημιουργήστε ένα `Workbook` αντικείμενο χρησιμοποιώντας τη διαδρομή αρχείου:

   ```java
   Workbook wb = new Workbook(dataDir + "/sampleRenderOfficeAdd-Ins.xlsx");
   ```

### Μετατροπή και αποθήκευση Excel ως PDF

**Επισκόπηση:** Η μετατροπή ενός βιβλίου εργασίας του Excel σε μορφή PDF είναι απλή με το Aspose.Cells.

1. **Ορισμός καταλόγου εξόδου:**

   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εξόδου σας
   ```

2. **Αποθήκευση του Βιβλίου Εργασίας ως PDF:**

   Αποθηκεύστε το φορτωμένο βιβλίο εργασίας σε μορφή PDF, ενσωματώνοντας πληροφορίες έκδοσης:

   ```java
   wb.save(outDir + "/output-" + CellsHelper.getVersion() + ".pdf");
   ```
