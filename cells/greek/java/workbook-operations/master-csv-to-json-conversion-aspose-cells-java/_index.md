## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή CSV σε JSON

Αυτή η λειτουργία επιτρέπει τη μετατροπή ενός αρχείου CSV σε μορφή JSON, διευκολύνοντας την επεξεργασία δεδομένων και την ενσωμάτωση με εφαρμογές ιστού.

#### Βήμα 1: Δημιουργήστε LoadOptions για τη μορφή CSV

Ξεκινήστε με τη ρύθμιση `LoadOptions` για να υποδείξετε ότι εργάζεστε με ένα αρχείο CSV:

```java
LoadOptions loadOptions = new LoadOptions(LoadFormat.CSV);
```
Ο `LoadFormat.CSV` διασφαλίζει ότι το Aspose.Cells ερμηνεύει σωστά τη δομή του αρχείου εισόδου.

#### Βήμα 2: Φόρτωση του αρχείου CSV σε ένα αντικείμενο βιβλίου εργασίας

Φορτώστε τα δεδομένα CSV σε ένα `Workbook` αντικείμενο:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "/SampleCsv.csv", loadOptions);
```
Ο `Workbook` Η κλάση διαχειρίζεται τη φόρτωση αρχείων, επιτρέποντας περαιτέρω λειτουργίες στα δεδομένα.

#### Βήμα 3: Ρύθμιση παραμέτρων ExportRangeToJsonOptions

Ορίστε επιλογές για την εξαγωγή μιας περιοχής κελιών σε JSON:

```java
ExportRangeToJsonOptions options = new ExportRangeToJsonOptions();
Cell lastCell = workbook.getWorksheets().get(0).getCells().getLastCell();
Range range = workbook.getWorksheets().get(0).getCells().createRange(0, 0, lastCell.getRow() + 1, lastCell.getColumn() + 1);
```
Εδώ, `ExportRangeToJsonOptions` και `Range` έχουν ρυθμιστεί ώστε να ορίζουν την περιοχή δεδομένων για μετατροπή.

#### Βήμα 4: Μετατροπή του καθορισμένου εύρους σε μορφή JSON

Μετατρέψτε το εύρος σε JSON:

```java
String data = JsonUtility.exportRangeToJson(range, options);
system.out.println(data);
```
Ο `JsonUtility.exportRangeToJson()` Η μέθοδος επεξεργάζεται το καθορισμένο εύρος και εξάγει δεδομένα σε μορφή JSON. Αυτό το βήμα είναι κρίσιμο για τη μετατροπή του CSV σας σε μια ευέλικτη δομή JSON.
