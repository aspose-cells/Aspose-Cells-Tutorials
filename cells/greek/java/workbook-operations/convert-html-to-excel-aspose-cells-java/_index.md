### Μετατροπή συμβολοσειράς HTML σε βιβλίο εργασίας
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε μια συμβολοσειρά HTML που περιέχει ένθετες ετικέτες σε ένα δομημένο βιβλίο εργασίας του Excel. Δείτε πώς:

**1. Προετοιμάστε τη συμβολοσειρά HTML σας**
Ξεκινήστε ορίζοντας το περιεχόμενο HTML σας σε Java. Για παράδειγμα:
```java
String export_html = "<html><body>...</body></html>";  // Το HTML σας εδώ
```

**2. Μετατρέψτε τη συμβολοσειρά HTML σε βιβλίο εργασίας**
Φορτώστε το HTML σας σε ένα Aspose.Cells `Workbook` αντικείμενο:
```java
import com.aspose.cells.HtmlLoadOptions;
import java.io.ByteArrayInputStream;

public class SupportthelayoutofDIVtags {
    public static void main(String[] args) throws Exception {
        byte[] bts = export_html.getBytes();
        ByteArrayInputStream bis = new ByteArrayInputStream(bts);

        HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.HTML);
        loadOptions.setSupportDivTag(true);  // Ενεργοποίηση υποστήριξης για ετικέτες div

        Workbook wb = new Workbook(bis, loadOptions);
    }
}
```
- **`HtmlLoadOptions`**Αυτή η κλάση παρέχει επιλογές για τον έλεγχο του τρόπου φόρτωσης περιεχομένου HTML στο βιβλίο εργασίας.
- **`setSupportDivTag(true)`**: Επιτρέπει την επεξεργασία `<div>` στοιχεία, κρίσιμα για τις ένθετες δομές.

### Αυτόματη προσαρμογή γραμμών και στηλών
Για να διασφαλίσετε ότι όλα τα δεδομένα είναι ορατά χωρίς χειροκίνητες προσαρμογές:
```java
public class AutoFitRowsAndColumns {
    public static void main(String[] args) throws Exception {
        Workbook wb = new Workbook("YOUR_INPUT_FILE_PATH");
        Worksheet ws = wb.getWorksheets().get(0);

        ws.autoFitRows();
        ws.autoFitColumns();
    }
}
```
- **`autoFitRows()`**: Προσαρμόζει το ύψος των γραμμών ώστε να ταιριάζει με το περιεχόμενό τους.
- **`autoFitColumns()`**: Προσαρμόζει το πλάτος των στηλών για να φιλοξενήσει δεδομένα.

### Αποθήκευση βιβλίου εργασίας ως XLSX
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε μορφή Excel:
```java
public class SaveWorkbookAsXlsx {
    public static void main(String[] args) throws Exception {
        Workbook wb = new Workbook("YOUR_INPUT_FILE_PATH");
        String outDir = "YOUR_OUTPUT_DIRECTORY";
        
        wb.save(outDir + "/SThelayoutofDIVtags_out.xlsx", SaveFormat.XLSX);
    }
}
```
- **`SaveFormat.XLSX`**: Καθορίζει τη μορφή αρχείου για αποθήκευση.
