## Οδηγός Εφαρμογής

Τώρα, ας εμβαθύνουμε στην εφαρμογή του `FilePathProvider` κλάση για τη διαχείριση των εξαγωγών HTML.

### Υλοποίηση του FilePathProvider

Ο σκοπός αυτής της δυνατότητας είναι να παρέχει έναν απρόσκοπτο τρόπο εξαγωγής κάθε φύλλου εργασίας ως μεμονωμένο αρχείο HTML, διατηρώντας παράλληλα τις μεταξύ τους σχέσεις. 

#### Βήμα 1: Δημιουργήστε την κλάση FilePathProvider

```java
package AsposeCellsExamples.TechnicalArticles;

import AsposeCellsExamples.Utils;

public class FilePathProvider {
    // Επιστρέφει την πλήρη διαδρομή του αρχείου κατά όνομα φύλλου εργασίας κατά την εξαγωγή σε HTML.
    public String getFullName(String sheetName) {
        String dataDir = Utils.getSharedDataDir(FilePathProvider.class) + "TechnicalArticles/";
        
        if ("Sheet2".equals(sheetName)) {
            return dataDir + "Sheet2.html";
        } else if ("Sheet3".equals(sheetName)) {
            return dataDir + "Sheet3.html";
        }
        
        return "";
    }
}
```

#### Εξήγηση
- **Utils.getSharedDataΚατάλογος:** Ανακτά τη διαδρομή καταλόγου για την αποθήκευση των εξαγόμενων αρχείων HTML.
- **Όνομα φύλλου:** Καθορίζει ποιο φύλλο εργασίας θα εξαχθεί, διατηρώντας σωστά τις αναφορές.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι όλα τα ονόματα φύλλων έχουν καθοριστεί σωστά στο `getFullName` μέθοδος.
- Επαληθεύστε ότι το `dataDir` Η διαδρομή υπάρχει και είναι εγγράψιμη.
