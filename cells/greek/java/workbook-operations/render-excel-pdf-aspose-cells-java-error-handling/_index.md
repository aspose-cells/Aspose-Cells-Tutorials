## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την εφαρμογή χειρισμού σφαλμάτων κατά τη μετατροπή από Excel σε PDF χρησιμοποιώντας το Aspose.Cells για Java. 

### Αγνόηση σφαλμάτων κατά τη μετατροπή
Για να χειριστείτε ομαλά τα σφάλματα στο αρχείο προέλευσης, ρυθμίστε τις παραμέτρους του Aspose.Cells ώστε να τα αγνοεί κατά τη μετατροπή. 

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας
Φορτώστε το βιβλίο εργασίας του Excel με το `Workbook` τάξη:
```java
import com.aspose.cells.Workbook;
import AsposeCellsExamples.Utils;

String srcDir = Utils.Get_SourceDirectory();
Workbook wb = new Workbook(srcDir + "sampleErrorExcel2Pdf.xlsx");
```
#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF
Δημιουργήστε ένα `PdfSaveOptions` instance και ορίστε το να αγνοεί τα σφάλματα:
```java
import com.aspose.cells.PdfSaveOptions;

PdfSaveOptions opts = new PdfSaveOptions();
opts.setIgnoreError(true); // Το Aspose.Cells θα αγνοεί πλέον τα σφάλματα κατά τη μετατροπή.
```
#### Βήμα 3: Αποθήκευση του βιβλίου εργασίας ως PDF
Χρησιμοποιήστε αυτές τις επιλογές για να αποθηκεύσετε το βιβλίο εργασίας σε μορφή PDF:
```java
import java.io.IOException;

String outDir = Utils.Get_OutputDirectory();

try {
    wb.save(outDir + "outputErrorExcel2Pdf.pdf", opts);
    System.out.println("Conversion executed successfully, ignoring errors.");
} catch (IOException e) {
    e.printStackTrace();
}
```
