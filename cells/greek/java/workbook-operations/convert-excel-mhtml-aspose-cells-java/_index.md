### Οδηγός Εφαρμογής
#### Φόρτωση αρχείου Excel
Για να φορτώσετε ένα αρχείο Excel, ακολουθήστε τα εξής βήματα:
1. **Ρύθμιση του καταλόγου δεδομένων σας**: Ορίστε τη διαδρομή όπου αποθηκεύονται τα αρχεία Excel.
   ```java
   String dataDir = "YOUR_DATA_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου δεδομένων σας
   ```
2. **Δημιουργία αντικειμένου βιβλίου εργασίας**Αυτό το αντικείμενο αντιπροσωπεύει το βιβλίο εργασίας του Excel σας.
   ```java
   String filePath = dataDir + "Book1.xlsx"; // Διαδρομή προς το αρχείο Excel
   Workbook wb = new Workbook(filePath); // Φορτώνει το αρχείο Excel
   ```
3. **Γιατί να χρησιμοποιήσετε `Workbook`;** Ο `Workbook` Η κλάση είναι απαραίτητη καθώς ενσωματώνει όλα τα φύλλα και τα δεδομένα τους, επιτρέποντας εύκολο χειρισμό.

#### Μετατροπή αρχείου Excel σε μορφή MHTML
Τώρα που έχουμε φορτώσει το αρχείο Excel μας, ας το μετατρέψουμε σε MHTML:
1. **Ρύθμιση καταλόγου εξόδου**: Ορίστε πού θέλετε να αποθηκεύσετε το αρχείο που έχει μετατραπεί.
   ```java
   String outDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου εξόδου σας
   ```
2. **Καθορισμός επιλογών αποθήκευσης HTML**: Χρήση `HtmlSaveOptions` για τον ορισμό της μορφής μετατροπής.
   ```java
   HtmlSaveOptions sv = new HtmlSaveOptions(SaveFormat.M_HTML); // Η MHTML είναι μια μορφή αρχειοθέτησης ιστού
   ```
3. **Εκτελέστε τη μετατροπή**Αποθηκεύστε το βιβλίο εργασίας σας στην επιθυμητή μορφή.
   ```java
   wb.save(outDir + "/CToMHTMLFiles_out.mht", sv);
   ```
4. **Γιατί `SaveFormat.M_HTML`;** Αυτή η επιλογή διασφαλίζει ότι το αρχείο Excel σας αποθηκεύεται ως MHTML, μια μορφή κατάλληλη για προβολή και αρχειοθέτηση στο web.
