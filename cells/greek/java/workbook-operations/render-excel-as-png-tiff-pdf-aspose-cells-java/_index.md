# Μετατροπή αρχείων Excel σε PNG, TIFF και PDF χρησιμοποιώντας το Aspose.Cells για Java

Στο σημερινό επιχειρηματικό περιβάλλον που βασίζεται σε δεδομένα, η μετατροπή αρχείων Excel σε διαφορετικές μορφές, όπως εικόνες ή PDF, είναι απαραίτητη για τη βελτίωση της ποιότητας των αναφορών που κοινοποιούνται στα ενδιαφερόμενα μέρη. Αυτό το ολοκληρωμένο σεμινάριο θα σας καθοδηγήσει στον απρόσκοπτο μετασχηματισμό των υπολογιστικών φύλλων Excel σε μορφές εικόνας, όπως PNG και TIFF, ή στην αποθήκευσή τους ως PDF χρησιμοποιώντας το Aspose.Cells για Java.

- Αποθήκευση δεδομένων Excel ως PDF με προσαρμοσμένες ρυθμίσεις γραμματοσειράς.
- Η σημασία του ορισμού προεπιλεγμένων γραμματοσειρών για χαρακτήρες που λείπουν σε έγγραφα.
- Τεχνικές για τη βελτιστοποίηση της απόδοσης κατά τη χρήση του Aspose.Cells.

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο απόδοσης αρχείων Excel σε μορφές PNG, TIFF και PDF χρησιμοποιώντας το Aspose.Cells για Java.

### Απόδοση Excel σε PNG με προεπιλεγμένη γραμματοσειρά
**Επισκόπηση:** Μετατρέψτε ένα φύλλο Excel σε εικόνα PNG, ορίζοντας παράλληλα προεπιλεγμένες γραμματοσειρές για τυχόν χαρακτήρες που λείπουν στο βιβλίο εργασίας.

#### Οδηγός βήμα προς βήμα:
1. **Δημιουργία εικόνας ή εκτύπωσης Επιλογές:**
   Αυτό το αντικείμενο σάς επιτρέπει να καθορίσετε ρυθμίσεις όπως τον τύπο εικόνας και τις επιλογές γραμματοσειράς.

   ```java
   import com.aspose.cells.ImageOrPrintOptions;
   import com.aspose.cells.ImageType;

   ImageOrPrintOptions imgOpt = new ImageOrPrintOptions();
   imgOpt.setImageType(ImageType.PNG);
   imgOpt.setCheckWorkbookDefaultFont(false); // Παράβλεψη προεπιλεγμένων γραμματοσειρών βιβλίου εργασίας
   imgOpt.setDefaultFont("Times New Roman"); // Προεπιλεγμένη γραμματοσειρά για χαρακτήρες που λείπουν
   ```

2. **Απόδοση του πρώτου φύλλου εργασίας:**
   Χρήση `SheetRender` για να μετατρέψετε το πρώτο φύλλο εργασίας του αρχείου Excel σε εικόνα PNG.

   ```java
   import com.aspose.cells.SheetRender;
   import com.aspose.cells.Workbook;

   Workbook workbook = new Workbook(dataDir + "/sample.xlsx");
   SheetRender sr = new SheetRender(workbook.getWorksheets().get(0), imgOpt);
   sr.toImage(0, "YOUR_OUTPUT_DIRECTORY/output.png"); // Αποθήκευση του αρχείου PNG
   ```

### Απόδοση Excel σε TIFF με προεπιλεγμένη γραμματοσειρά
**Επισκόπηση:** Μετατρέψτε ένα ολόκληρο βιβλίο εργασίας του Excel σε μια εικόνα TIFF πολλαπλών σελίδων, διασφαλίζοντας ότι όλοι οι χαρακτήρες εμφανίζονται χρησιμοποιώντας μια προεπιλεγμένη γραμματοσειρά.

#### Οδηγός βήμα προς βήμα:
1. **Ρύθμιση παραμέτρων ImageOrPrintOptions για TIFF:**

   ```java
   ImageOrPrintOptions imgOpt = new ImageOrPrintOptions();
   imgOpt.setImageType(ImageType.TIFF);
   imgOpt.setCheckWorkbookDefaultFont(false); // Παράβλεψη προεπιλεγμένων γραμματοσειρών βιβλίου εργασίας
   imgOpt.setDefaultFont("Times New Roman"); // Προεπιλεγμένη γραμματοσειρά για χαρακτήρες που λείπουν
   ```

2. **Απόδοση ολόκληρου του βιβλίου εργασίας:**
   Χρήση `WorkbookRender` για να μετατρέψετε ολόκληρο το βιβλίο εργασίας του Excel σε εικόνα TIFF.

   ```java
   import com.aspose.cells.WorkbookRender;

   Workbook workbook = new Workbook(dataDir + "/sample.xlsx");
   WorkbookRender wr = new WorkbookRender(workbook, imgOpt);
   wr.toImage("YOUR_OUTPUT_DIRECTORY/output.tiff"); // Αποθήκευση του αρχείου TIFF
   ```

### Αποθήκευση Excel ως PDF με προεπιλεγμένη γραμματοσειρά
**Επισκόπηση:** Αποθηκεύστε το βιβλίο εργασίας του Excel ως έγγραφο PDF, καθορίζοντας παράλληλα μια προεπιλεγμένη γραμματοσειρά για τυχόν γραμματοσειρές που λείπουν.

#### Οδηγός βήμα προς βήμα:
1. **Ρύθμιση παραμέτρων PdfSaveOptions:**

   ```java
   import com.aspose.cells.PdfSaveOptions;

   PdfSaveOptions saveOptions = new PdfSaveOptions();
   saveOptions.setDefaultFont("Times New Roman"); // Προεπιλεγμένη γραμματοσειρά για χαρακτήρες που λείπουν
   saveOptions.setCheckWorkbookDefaultFont(false); // Παράβλεψη προεπιλεγμένων γραμματοσειρών βιβλίου εργασίας
   ```

2. **Αποθήκευση του βιβλίου εργασίας ως PDF:**
   Χρησιμοποιήστε το `save` μια μέθοδο για να μετατρέψετε το αρχείο Excel σε PDF.

   ```java
   Workbook workbook = new Workbook(dataDir + "/sample.xlsx");
   workbook.save("YOUR_OUTPUT_DIRECTORY/output.pdf", saveOptions); // Αποθήκευση του εγγράφου PDF
   ```
