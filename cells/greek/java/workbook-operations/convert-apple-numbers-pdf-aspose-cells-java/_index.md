## Οδηγός Εφαρμογής

### Φόρτωση υπολογιστικού φύλλου αριθμών

Το πρώτο βήμα στη διαδικασία μετατροπής μας είναι η φόρτωση ενός αρχείου Apple Numbers χρησιμοποιώντας το Aspose.Cells για Java. Αυτή η λειτουργία σάς επιτρέπει να χειρίζεστε δεδομένα υπολογιστικών φύλλων αποτελεσματικά.

#### Επισκόπηση
Η φόρτωση ενός αρχείου Numbers περιλαμβάνει τον καθορισμό των σωστών επιλογών φόρτωσης και τη χρήση του `Workbook` τάξη.

#### Βήμα προς βήμα εφαρμογή

**1. Εισαγωγή απαιτούμενων κλάσεων:**
```java
import com.aspose.cells.LoadOptions;
import com.aspose.cells.Workbook;
```

**2. Καθορίστε τις επιλογές φόρτωσης:**
Δημιουργήστε μια παρουσία του `LoadOptions`, καθορίζοντας ότι φορτώνετε ένα αρχείο Numbers:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
String outDir = "YOUR_OUTPUT_DIRECTORY";

// Ορισμός επιλογών φόρτωσης για τη μορφή Numbers
LoadOptions opts = new LoadOptions(com.aspose.cells.LoadFormat.NUMBERS);
```

**3. Φορτώστε το Βιβλίο Εργασίας:**
Χρησιμοποιήστε το `Workbook` κλάση για να φορτώσετε το υπολογιστικό φύλλο Numbers με συγκεκριμένες επιλογές:
```java
Workbook wb = new Workbook(dataDir + "/sampleNumbersByAppleInc.numbers", opts);
```

### Αποθήκευση βιβλίου εργασίας ως PDF

Μετά τη φόρτωση, μπορείτε να μετατρέψετε και να αποθηκεύσετε το βιβλίο εργασίας σε μορφή PDF.

#### Επισκόπηση
Η αποθήκευση ενός αρχείου περιλαμβάνει τον καθορισμό του καταλόγου εξόδου και της επιθυμητής μορφής χρησιμοποιώντας το Aspose.Cells. `save` μέθοδος.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση βιβλίου εργασίας:**
Βεβαιωθείτε ότι το δικό σας `Workbook` παράδειγμα (`wb`) φορτώνεται όπως φαίνεται προηγουμένως.

**2. Αποθήκευση ως PDF:**
Μετατροπή και αποθήκευση του βιβλίου εργασίας σε αρχείο PDF:
```java
// Αποθήκευση του βιβλίου εργασίας σε μορφή PDF
wb.save(outDir + "/outputNumbersByAppleInc.pdf", com.aspose.cells.SaveFormat.PDF);
```

#### Βασικές επιλογές διαμόρφωσης
- **Κατάλογος εξόδου**: Βεβαιωθείτε `outDir` έχει ρυθμιστεί σωστά για την αποθήκευση αρχείων.
- **Ονομασία αρχείων**: Προσαρμόστε τα ονόματα των αρχείων εξόδου όπως απαιτείται.
