## Οδηγός Εφαρμογής
Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη δημιουργία ενός προσαρμοσμένου `IStreamProvider` και μετατροπή αρχείων Excel σε PDF χρησιμοποιώντας αυτόν τον πάροχο.
### Υλοποίηση του IStreamProvider για Διαχείριση Πόρων
#### Επισκόπηση
Ο `MyStreamProvider` η κλάση υλοποιεί το `IStreamProvider` διεπαφή, που σας επιτρέπει να διαχειρίζεστε εξωτερικούς πόρους, όπως εικόνες, κατά τη διάρκεια των διαδικασιών μετατροπής. Αυτό είναι ιδιαίτερα χρήσιμο όταν το αρχείο Excel σας περιλαμβάνει εξωτερικά μέσα που πρέπει να ενσωματωθούν στο PDF που προκύπτει.
#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργήστε την κλάση MyStreamProvider**
```java
import java.io.*;
import com.aspose.cells.IStreamProvider;
import com.aspose.cells.StreamProviderOptions;

class MyStreamProvider implements IStreamProvider {
    public void closeStream(StreamProviderOptions options) throws Exception {
        // Κλείστε την παρεχόμενη ροή εάν είναι απαραίτητο
    }

    public void initStream(StreamProviderOptions options) throws Exception {
        String dataDir = "YOUR_DATA_DIRECTORY";
        File imgFile = new File(dataDir + "/newPdfSaveOptions_StreamProvider.png");

        byte[] bts = new byte[(int) imgFile.length()];

        try (FileInputStream fin = new FileInputStream(imgFile)) {
            fin.read(bts);
        }

        ByteArrayOutputStream baout = new ByteArrayOutputStream();
        baout.write(bts);

        // Αντιστοιχίστε τη ροή μνήμης στην ιδιότητα Stream
        options.setStream(baout);
    }
}
```
- **Μέθοδος initStream**: Αρχικοποιεί μια ροή με ένα εξωτερικό αρχείο εικόνας. Αυτό το βήμα είναι κρίσιμο για να διασφαλίσετε ότι το PDF σας περιέχει όλους τους απαραίτητους πόρους.
### Μετατροπή Excel σε PDF με προσαρμοσμένη υπηρεσία παροχής πόρων
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει τον τρόπο μετατροπής ενός βιβλίου εργασίας Excel σε PDF χρησιμοποιώντας την προσαρμοσμένη `IStreamProvider` για τη διαχείριση πόρων, διασφαλίζοντας ότι οι εξωτερικές εικόνες ή άλλα μέσα χειρίζονται σωστά κατά τη μετατροπή.
#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση και ρύθμιση παραμέτρων βιβλίου εργασίας**
```java
import com.aspose.cells.*;

void Run() throws Exception {
    String dataDir = "YOUR_DATA_DIRECTORY";
    String outDir = "YOUR_OUTPUT_DIRECTORY";

    // Φόρτωση του βιβλίου εργασίας του Excel από ένα αρχείο
    Workbook wb = new Workbook(dataDir + "/samplePdfSaveOptions_StreamProvider.xlsx");

    // Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF
    PdfSaveOptions opts = new PdfSaveOptions();
    opts.setOnePagePerSheet(true);

    // Ορίστε τον πάροχο προσαρμοσμένης ροής για τη διαχείριση εξωτερικών πόρων
    wb.getSettings().setResourceProvider(new MyStreamProvider());

    // Αποθήκευση του βιβλίου εργασίας ως αρχείο PDF
    wb.save(outDir + "/outputΕπιλογές Αποθήκευσης Pdf_StreamProvider.pdf", opts);
}
```
- **PdfSaveOptions**: Ρυθμίζει τον τρόπο αποθήκευσης του αρχείου Excel σε PDF, όπως η ρύθμιση `onePagePerSheet` για πιο καθαρές εξόδους.
- **Μέθοδος setResourceProvider**: Σύνδεσμοι για τις προσαρμοσμένες μας υπηρεσίες `IStreamProvider`, διασφαλίζοντας ότι οι πόροι διαχειρίζονται σωστά κατά τη μετατροπή.
