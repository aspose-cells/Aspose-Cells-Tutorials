## Οδηγός Εφαρμογής
### Φόρτωση αρχείου Excel XLSB
**Επισκόπηση:** Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο XLSB σε ένα `Workbook` αντικείμενο.

#### Βήμα 1: Εισαγωγή απαραίτητων κλάσεων
```java
import com.aspose.cells.Workbook;
```

#### Βήμα 2: Καθορισμός καταλόγου δεδομένων και φόρτωση του αρχείου
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook wb = new Workbook(dataDir + "/sampleExternalConnection_XLSB.xlsb");
```
Αντικαθιστώ `dataDir` με την πραγματική διαδρομή καταλόγου που περιέχει το αρχείο XLSB.

### Ανάγνωση εξωτερικής σύνδεσης βάσης δεδομένων από αρχείο XLSB
**Επισκόπηση:** Στη συνέχεια, διαβάστε την πρώτη εξωτερική σύνδεση βάσης δεδομένων που είναι ενσωματωμένη στο βιβλίο εργασίας.

#### Βήμα 1: Εισαγωγή της κλάσης DBConnection
```java
import com.aspose.cells.DBConnection;
```

#### Βήμα 2: Πρόσβαση και ανάκτηση της πρώτης σύνδεσης βάσης δεδομένων
```java
DBConnection dbCon = (DBConnection) wb.getDataConnections().get(0);
```
Αυτό ανακτά την πρώτη σύνδεση βάσης δεδομένων από τη συλλογή συνδέσεων δεδομένων του βιβλίου εργασίας.

### Τροποποίηση και εμφάνιση λεπτομερειών σύνδεσης βάσης δεδομένων
**Επισκόπηση:** Τώρα, τροποποιήστε το όνομα αυτής της σύνδεσης και εμφανίστε τα στοιχεία της για επαλήθευση.

#### Βήμα 1: Εισαγωγή βοηθητικών προγραμμάτων Java
```java
import java.util.Objects;
```

#### Βήμα 2: Ανάκτηση και εκτύπωση λεπτομερειών τρέχουσας σύνδεσης
```java
System.out.println("Connection Name: " + Objects.requireNonNull(dbCon).getName());
System.out.println("Command: " + Objects.requireNonNull(dbCon).getCommand());
System.out.println("Connection Info: " + Objects.requireNonNull(dbCon).getConnectionInfo());
```

#### Βήμα 3: Τροποποίηση του ονόματος σύνδεσης
```java
dbCon.setName("NewCust");
```
Αυτό αλλάζει το όνομα της σύνδεσης σε "NewCust".

### Αποθήκευση τροποποιημένου αρχείου Excel XLSB
**Επισκόπηση:** Τέλος, αποθηκεύστε τις τροποποιήσεις σας σε ένα αρχείο XLSB.

#### Βήμα 1: Εισαγωγή κλάσης SaveFormat
```java
import com.aspose.cells.SaveFormat;
```

#### Βήμα 2: Ορισμός καταλόγου εξόδου και αποθήκευση βιβλίου εργασίας
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";
wb.save(outDir + "/outputExternalConnection_XLSB.xlsb", SaveFormat.AUTO);
```
Αντικαθιστώ `outDir` με την πραγματική διαδρομή του καταλόγου εξόδου σας.
