## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο αξιοποίησης βασικών χαρακτηριστικών του Aspose.Cells για Java.

### Φόρτωση βιβλίου εργασίας και επανάληψη κελιών (H2)
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα βιβλίο εργασίας του Excel και να κάνετε επανάληψη σε όλα τα κελιά ενός φύλλου εργασίας.

**Βήμα 1: Φόρτωση του βιβλίου εργασίας**
```java
// Φόρτωση ενός υπάρχοντος βιβλίου εργασίας
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook book = new Workbook(dataDir + "sample.xlsx");
```

**Βήμα 2: Επανάληψη σε κελιά**
```java
import java.util.Iterator;
import com.aspose.cells.Cell;

Iterator cellIterator = book.getWorksheets().get(0).getCells().iterator();
while (cellIterator.hasNext()) {
    Cell cell = (Cell) cellIterator.next();
    // Παράδειγμα διαδικασίας: Εκτύπωση ονόματος και τιμής κελιού
    System.out.println("Name: " + cell.getName() + ", Value: " + cell.getValue());
}
```

**Εξήγηση:** Χρησιμοποιούμε ένα `Iterator` να διασχίσει όλα τα κελιά, ανακτώντας τα ονόματα και τις τιμές τους.

### Επανάληψη γραμμής (H2)
#### Επισκόπηση
Επαναλάβετε την εκτέλεση κελιών σε μια συγκεκριμένη γραμμή μέσα στο φύλλο Excel σας.

**Βήμα 1: Λήψη του Iterator για μια συγκεκριμένη γραμμή**
```java
Iterator rowIterator = book.getWorksheets().get(0).getCells().getRows().get(0).iterator();
```

**Βήμα 2: Διασχίστε τα κελιά στη σειρά**
```java
while (rowIterator.hasNext()) {
    Cell cell = (Cell) rowIterator.next();
    System.out.println("Name: " + cell.getName() + ", Value: " + cell.getValue());
}
```
Αυτή η μέθοδος είναι χρήσιμη για λειτουργίες που επικεντρώνονται σε συγκεκριμένες γραμμές.

### Επανάληψη εύρους (H2)
#### Επισκόπηση
Επιτρέπει την επανάληψη σε ένα συγκεκριμένο εύρος κελιών, ιδανικό για στοχευμένη επεξεργασία δεδομένων.

**Βήμα 1: Ορισμός της περιοχής κελιών**
```java
Iterator rangeIterator = book.getWorksheets().get(0).getCells().createRange("A1:B10").iterator();
```

**Βήμα 2: Διασχίστε το καθορισμένο εύρος**
```java
while (rangeIterator.hasNext()) {
    Cell cell = (Cell) rangeIterator.next();
    System.out.println("Name: " + cell.getName() + ", Value: " + cell.getValue());
}
```
Αυτή η προσέγγιση είναι ιδανική για τον χειρισμό καθορισμένων ενοτήτων του βιβλίου εργασίας σας.
