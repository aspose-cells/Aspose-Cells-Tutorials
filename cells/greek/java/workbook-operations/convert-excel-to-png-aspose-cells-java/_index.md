## Οδηγός Εφαρμογής
### Φόρτωση βιβλίου εργασίας Excel
Για να ξεκινήσετε τη μετατροπή του αρχείου Excel σε εικόνες, πρέπει πρώτα να το φορτώσετε χρησιμοποιώντας το Aspose.Cells:
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY";
Workbook book = new Workbook(dataDir + "/MyTestBook1.xlsx");
```
**Εξήγηση:**
- `Workbook` είναι μια κλάση στο Aspose.Cells που αντιπροσωπεύει ένα αρχείο Excel.
- Ο κατασκευαστής φορτώνει το βιβλίο εργασίας από την καθορισμένη διαδρομή.
### Ρύθμιση παραμέτρων επιλογών εικόνας για μετατροπή φύλλου εργασίας
Μόλις φορτωθεί το βιβλίο εργασίας σας, ρυθμίστε τον τρόπο με τον οποίο θέλετε να το μετατρέψετε σε εικόνες:
```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.ImageType;

ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
imgOptions.setImageType(ImageType.PNG);
```
**Εξήγηση:**
- `ImageOrPrintOptions` επιτρέπει την προσαρμογή της εξόδου εικόνας.
- `setImageType(ImageType.PNG)` Καθορίζει ότι η έξοδος πρέπει να είναι σε μορφή PNG.
### Φύλλο εργασίας Access και Render ως εικόνες
Αφού ορίσετε τις επιλογές εικόνας, μπορείτε πλέον να αποδώσετε κάθε φύλλο εργασίας σε εικόνες:
```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.SheetRender;

String outDir = "YOUR_OUTPUT_DIRECTORY";
Worksheet sheet = book.getWorksheets().get(0);
SheetRender sr = new SheetRender(sheet, imgOptions);

for (int j = 0; j < sr.getPageCount(); j++) {
    // Δημιουργήστε μια εικόνα για το φύλλο εργασίας
    sr.toImage(j, outDir + "/WToImage-out" + j + ".png");
}
```
**Εξήγηση:**
- `SheetRender` Χρησιμοποιείται για την απόδοση ενός φύλλου εργασίας σε εικόνες.
- Ο βρόχος επαναλαμβάνεται σε κάθε σελίδα του φύλλου εργασίας και το αποθηκεύει ως αρχείο PNG.
