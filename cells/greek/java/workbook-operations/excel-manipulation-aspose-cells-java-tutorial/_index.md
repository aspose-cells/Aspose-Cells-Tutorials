## Οδηγός Εφαρμογής

### Δημιουργία Βιβλίου Εργασίας και Πρόσβαση σε Φύλλα Εργασίας (H2)

#### Επισκόπηση
Η δημιουργία ενός βιβλίου εργασίας του Excel και η πρόσβαση στα φύλλα εργασίας του είναι βασικές εργασίες κατά την εργασία με υπολογιστικά φύλλα. Δείτε πώς μπορείτε να τις πετύχετε χρησιμοποιώντας το Aspose.Cells για Java:

##### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας

Δημιουργήστε μια νέα παρουσία του `Workbook` κλάση για να ξεκινήσετε το έγγραφο Excel σας.

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου σας εδώ
dataDir += "/Data/";

// Δημιουργία νέου βιβλίου εργασίας
Workbook workbook = new Workbook();
```

##### Βήμα 2: Προσθήκη και πρόσβαση σε φύλλα εργασίας

Μπορείτε να προσθέσετε νέα φύλλα εργασίας στο βιβλίο εργασίας σας και να έχετε πρόσβαση σε αυτά μέσω προγραμματισμού.

```java
import com.aspose.cells.Worksheet;

int sheetIndex = workbook.getWorksheets().add(); // Προσθήκη φύλλου εργασίας
Worksheet worksheet = workbook.getWorksheets().get(sheetIndex); // Πρόσβαση στο φύλλο εργασίας που προστέθηκε
```

### Ορισμός τιμής κελιού και στυλ με υπογράμμιση γραμματοσειράς (H2)

#### Επισκόπηση
Η τροποποίηση τιμών κελιών και η εφαρμογή στυλ όπως η υπογράμμιση γραμματοσειράς μπορεί να βελτιώσει την αναγνωσιμότητα του υπολογιστικού σας φύλλου. Ας δούμε πώς γίνεται αυτό:

##### Βήμα 1: Ορισμός τιμής κελιού

Αποκτήστε πρόσβαση σε συγκεκριμένα κελιά στο φύλλο εργασίας για να ορίσετε τις τιμές τους.

```java
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;

Cells cells = worksheet.getCells(); // Λήψη συλλογής κελιών
cell = cells.get("A1"); // Πρόσβαση στο κελί "A1"
cell.setValue("Hello Aspose!"); // Ορισμός τιμής στο κελί
```

##### Βήμα 2: Εφαρμογή στυλ υπογράμμισης γραμματοσειράς

Χρησιμοποιήστε το `Style` και `Font` κλάσεις για την τροποποίηση της εμφάνισης των κελιών.

```java
import com.aspose.cells.Style;
import com.aspose.cells.Font;
import com.aspose.cells.FontUnderlineType;

// Λήψη τρέχοντος στυλ του κελιού
Style style = cell.getStyle();
Font font = style.getFont();

// Εφαρμογή στυλ υπογράμμισης
font.setUnderline(FontUnderlineType.SINGLE);
style.setFont(font);

// Ενημέρωση του κελιού με νέο στυλ
cell.setStyle(style);
```

#### Αποθήκευση του βιβλίου εργασίας σας

Μην ξεχάσετε να αποθηκεύσετε τις αλλαγές σας σε ένα αρχείο.

```java
String outDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εξόδου
dataDir += "/SFUnderlineType_out.xls";
workbook.save(dataDir); // Αποθήκευση βιβλίου εργασίας με αλλαγές
```
