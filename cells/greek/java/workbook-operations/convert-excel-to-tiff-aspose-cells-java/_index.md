## Οδηγός Εφαρμογής (H2)

### Αρχικοποίηση του Aspose.Cells για μετατροπή TIFF (H3)

Ξεκινήστε ρυθμίζοντας τις βασικές λειτουργίες για να λειτουργήσουν με το Aspose.Cells.

#### Βήμα 1: Ρύθμιση βιβλίου εργασίας και επιλογών εικόνας (H3)

Φορτώστε το βιβλίο εργασίας του Excel που θέλετε να μετατρέψετε σε εικόνα. Ρύθμιση παραμέτρων `ImageOrPrintOptions` για να καθορίσετε το TIFF ως την επιθυμητή μορφή εξόδου:

```java
import com.aspose.cells.*;

// Φόρτωση του βιβλίου εργασίας προέλευσης
Workbook wb = new Workbook("path/to/sampleUseWorkbookRenderForImageConversion.xlsx");

// Ρύθμιση παραμέτρων επιλογών για μετατροπή TIFF
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.setImageType(ImageType.TIFF);
```

#### Βήμα 2: Υλοποίηση Επανακλήσης Προόδου (H3)

Για να παρακολουθήσετε την πρόοδο της μετατροπής εγγράφων, εφαρμόστε μια προσαρμοσμένη κλάση που πληροί τις προϋποθέσεις `IPageSavingCallback` διεπαφή. Αυτό σας επιτρέπει να ορίσετε ενέργειες στην αρχή και στο τέλος της διαδικασίας αποθήκευσης κάθε σελίδας:

```java
class TestTiffPageSavingCallback implements IPageSavingCallback {
    public void pageStartSaving(PageStartSavingArgs args) {
        System.out.println("Start saving page index " + args.getPageIndex() + \
                           " of pages " + args.getPageCount());
        // Εξαίρεση σελίδων πριν από το ευρετήριο 2
        if (args.getPageIndex() < 2) {
            args.setToOutput(false);
        }
    }

    public void pageEndSaving(PageEndSavingArgs args) {
        System.out.println("End saving page index " + args.getPageIndex() + \
                           " of pages " + args.getPageCount());
        // Διακοπή μετατροπής μετά την ευρετηρίαση σελίδας 8
        if (args.getPageIndex() >= 8) {
            args.setHasMorePages(false);
        }
    }
}
```

#### Βήμα 3: Εκτέλεση της Διαδικασίας Μετατροπής (H3)

Χρήση `WorkbookRender` για να μετατρέψετε το βιβλίο εργασίας σας σε εικόνα TIFF, εφαρμόζοντας τις διαμορφωμένες επιλογές και συνεχίζοντας την επανάκληση:

```java
// Ορίστε την επανακλήση αποθήκευσης σελίδας για την παρακολούθηση της προόδου μετατροπής
opts.setPageSavingCallback(new TestTiffPageSavingCallback());

// Απόδοση του βιβλίου εργασίας σε αρχείο εικόνας με καθορισμένες επιλογές
WorkbookRender wr = new WorkbookRender(wb, opts);
wr.toImage("path/to/DocumentConversionProgressForTiff_out.tiff");
```
