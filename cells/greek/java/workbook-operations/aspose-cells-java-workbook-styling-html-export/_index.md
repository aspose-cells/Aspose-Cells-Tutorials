### Χαρακτηριστικό 2: Δημιουργία βιβλίου εργασίας και στυλ κελιών
**Επισκόπηση**Μάθετε πώς να δημιουργείτε ένα βιβλίο εργασίας, να έχετε πρόσβαση σε φύλλα εργασίας, να τροποποιείτε το περιεχόμενο των κελιών και να εφαρμόζετε στυλ, όπως αλλαγή χρώματος γραμματοσειράς.

#### Βήμα προς βήμα εφαρμογή:
**Δημιουργία βιβλίου εργασίας και φύλλου εργασίας της Access**
```java
import com.aspose.cells.*;

public class WorkbookAndCellStyling {
    public static void main(String[] args) throws Exception {
        // Δημιουργήστε μια παρουσία του αντικειμένου Βιβλίου εργασίας
        Workbook wb = new Workbook();
        
        // Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
        Worksheet ws = wb.getWorksheets().get(0);
```
**Τροποποίηση περιεχομένου και στυλ κελιού**
```java
        // Ανάκτηση κελιού B5 από το φύλλο εργασίας
        Cell cell = ws.getCells().get("B5");
        
        // Ορίστε την τιμή "Αυτό είναι κείμενο." στο κελί B5
        cell.putValue("This is some text.");
        
        // Λήψη του στυλ του κελιού, ορισμός χρώματος γραμματοσειράς σε κόκκινο
        Style st = cell.getStyle();
        st.getFont().setColor(Color.getRed());
        
        // Εφαρμογή των ρυθμίσεων με στυλ πίσω στο κελί
        cell.setStyle(st);
    }
}
```
**Εξήγηση**: 
- `Workbook` και `Worksheet` Τα αντικείμενα χρησιμοποιούνται για τον χειρισμό αρχείων Excel.
- Το styling των κελιών επιτυγχάνεται χρησιμοποιώντας το `Style` κλάση, επιτρέποντας την προσαρμογή όπως το χρώμα της γραμματοσειράς.

### Χαρακτηριστικό 3: Εξαγωγή φύλλου εργασίας CSS ξεχωριστά σε HTML
**Επισκόπηση**Εξαγωγή ενός φύλλου εργασίας Excel ως αρχείο HTML με διαχωρισμένα στυλ (CSS). Αυτή η λειτουργία βελτιώνει την οπτική παρουσίαση των δεδομένων σας σε πλατφόρμες ιστού.

#### Βήμα προς βήμα εφαρμογή:
**Δημιουργία βιβλίου εργασίας και διαμόρφωση κελιού**
```java
import com.aspose.cells.*;

public class ExportWorksheetCSSSeparatelyInHTML {
    public static void main(String[] args) throws Exception {
        // Δημιουργία αντικειμένου βιβλίου εργασίας
        Workbook wb = new Workbook();
        
        // Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
        Worksheet ws = wb.getWorksheets().get(0);
        
        // Αποκτήστε πρόσβαση στο κελί B5 και εισάγετε τιμή μέσα σε αυτό
        Cell cell = ws.getCells().get("B5");
        cell.putValue("This is some text.");
        
        // Ορισμός του στυλ του κελιού - χρώμα γραμματοσειράς σε κόκκινο
        Style st = cell.getStyle();
        st.getFont().setColor(Color.getRed());
        
        // Εφαρμογή των ρυθμίσεων με στυλ πίσω στο κελί
        cell.setStyle(st);
```
**Εξαγωγή ως HTML με ξεχωριστό CSS**
```java
        // Καθορίστε επιλογές αποθήκευσης HTML με ξεχωριστή εξαγωγή CSS
        HtmlSaveOptions opts = new HtmlSaveOptions();
        opts.setExportWorksheetCSSSeparately(true);
        
        // Αποθήκευση του βιβλίου εργασίας ως αρχείο HTML με καθορισμένες επιλογές
        String outDir = "YOUR_OUTPUT_DIRECTORY";
        wb.save(outDir + "/outputExportWorksheetCSSSeparately.html", opts);
    }
}
```
**Εξήγηση**: 
- `HtmlSaveOptions` επιτρέπει την προσαρμογή του τρόπου αποθήκευσης των αρχείων Excel ως HTML.
- Σύνθεση `setExportWorksheetCSSSeparately(true)` διασφαλίζει ότι το CSS εξάγεται ξεχωριστά για καλύτερο έλεγχο του στυλ.
