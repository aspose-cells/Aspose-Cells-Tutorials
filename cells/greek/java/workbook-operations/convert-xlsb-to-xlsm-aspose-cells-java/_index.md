## Οδηγός Εφαρμογής

### Μετατροπή XLSB σε XLSM

Ας αναλύσουμε τη διαδικασία μετατροπής βήμα προς βήμα.

#### Βήμα 1: Φόρτωση του αρχείου XLSB
Για να ξεκινήσετε, φορτώστε το αρχείο XLSB σε ένα `Workbook` αντικείμενο χρησιμοποιώντας το απλό API του Aspose.Cells:

```java
import com.aspose.cells.Workbook;

public class ConvertExample {
    public static void main(String[] args) throws Exception {
        // Καθορίστε τη διαδρομή προς τον κατάλογο εγγράφων
        String dataDir = "path/to/your/documents/";

        // Φόρτωση ενός υπάρχοντος βιβλίου εργασίας XLSB
        Workbook workbook = new Workbook(dataDir + "book1.xlsb");
    }
}
```
**Γιατί:** Η φόρτωση του αρχείου το αρχικοποιεί στη μνήμη, προετοιμάζοντάς το για τροποποιήσεις ή μετατροπές.

#### Βήμα 2: Αποθήκευση ως XLSM
Μόλις φορτωθεί, η αποθήκευσή του σε διαφορετική μορφή είναι απρόσκοπτη:

```java
import com.aspose.cells.SaveFormat;

public class ConvertExample {
    public static void main(String[] args) throws Exception {
        String dataDir = "path/to/your/documents/";
        
        // Φόρτωση του βιβλίου εργασίας
        Workbook workbook = new Workbook(dataDir + "book1.xlsb");
        
        // Αποθήκευση ως XLSM
        workbook.save(dataDir + "CROfXLSBtoXLSM_out.xlsm", SaveFormat.XLSM);
    }
}
```
**Γιατί:** Καθορισμός `SaveFormat.XLSM` διασφαλίζει ότι το αρχείο εξόδου περιλαμβάνει λειτουργίες μακροεντολών, καθιστώντας το κατάλληλο για προηγμένες λειτουργίες.
