# Αλλαγή του συστήματος ημερομηνίας του Excel σε 1904 με το Aspose.Cells Java

Η διαχείριση ιστορικών δεδομένων στο Excel μπορεί να είναι δύσκολη επειδή το Excel υποστηρίζει δύο διαφορετικά συστήματα ημερομηνίας. **Σε αυτό το σεμινάριο θα μάθετε πώς να αλλάξετε το σύστημα ημερομηνίας του Excel σε μορφή 1904 χρησιμοποιώντας το Aspose.Cells για Java**, το οποίο καθιστά την επεξεργασία παλαιών ημερομηνιών εύκολη. Θα περάσουμε από τη δημιουργία ενός βιβλίου εργασίας, την ενεργοποίηση του συστήματος ημερομηνίας 1904 και τη διατήρηση της αλλαγής.

## Γρήγορες Απαντήσεις
- **Τι κάνει το σύστημα ημερομηνίας 1904;** Ξεκινά την μέτρηση των ημερών από την 1 Ιανουαρίου 1904, μετατοπίζοντας όλες τις ημερομηνίες κατά 1462 ημέρες σε σύγκριση με το προεπιλεγμένο σύστημα 1900.  
- **Γιατί να χρησιμοποιήσετε το Aspose.Cells για την αλλαγή του συστήματος ημερομηνίας;** Παρέχει ένα απλό API που λειτουργεί χωρίς εγκατεστημένο Excel και υποστηρίζει μεγάλα αρχεία.  
- **Ποιες εκδόσεις της Java υποστηρίζονται;** JDK 8 ή νεότερη.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· μια άδεια αφαιρεί τους περιορισμούς χρήσης.  
- **Μπορώ να επαναφέρω το σύστημα 1900 αργότερα;** Ναι, απλώς ορίστε `setDate1904(false)`.

## Τι είναι το σύστημα ημερομηνίας 1904 στο Excel;
Το σύστημα ημερομηνίας 1904 χρησιμοποιήθηκε αρχικά από τις πρώτες εκδόσεις του Excel για Macintosh. Μετρά τις ημέρες από την 1 Ιανουαρίου 1904, κάτι που είναι χρήσιμο για συμβατότητα με παλαιά λογιστικά φύλλα και ορισμένα οικονομικά μοντέλα.

## Γιατί να αλλάξετε το σύστημα ημερομηνίας του Excel με το Aspose.Cells;
- **Συμβατότητα μεταξύ πλατφορμών** – λειτουργεί σε Windows, Linux και macOS.  
- **Δεν απαιτείται εγκατάσταση του Excel** – ιδανικό για επεξεργασία στο διακομιστή.  
- **Υψηλή απόδοση** – διαχειρίζεται μεγάλα βιβλία εργασίας με ελάχιστη χρήση μνήμης.  

## Προαπαιτούμενα
- Java Development Kit (JDK) 8 ή νεότερο.  
- Maven ή Gradle για διαχείριση εξαρτήσεων.  
- Βασικές γνώσεις προγραμματισμού Java.

## Αλλαγή του συστήματος ημερομηνίας του Excel χρησιμοποιώντας το Aspose.Cells Java

Παρακάτω βρίσκεται ο οδηγός βήμα‑βήμα που πραγματικά **αλλάζει το σύστημα ημερομηνίας του Excel**. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση ακολουθούμενη από τον ακριβή κώδικα που χρειάζεστε.

### Βήμα 1: Αρχικοποίηση και φόρτωση του βιβλίου εργασίας
Αρχικά, δημιουργήστε μια παρουσία `Workbook` που δείχνει στο υπάρχον αρχείο Excel σας.

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ensure the path to your Excel file is correct
// Initialize a Workbook object with the path to your Excel file
Workbook workbook = new Workbook(dataDir + "/Mybook.xlsx");
```

### Βήμα 2: Ενεργοποίηση του συστήματος ημερομηνίας 1904
Χρησιμοποιήστε τις ρυθμίσεις του βιβλίου εργασίας για να αλλάξετε το σύστημα ημερομηνίας.

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ensure the path to your Excel file is correct
// Load the workbook from your specified directory
Workbook workbook = new Workbook(dataDir + "/Mybook.xlsx");

// Enable the 1904 date system
workbook.getSettings().setDate1904(true);
```

**Συμβουλή:** Μπορείτε επίσης να καλέσετε `setDate1904(false)` αργότερα αν χρειαστεί να επαναφέρετε.

### Βήμα 3: Αποθήκευση του τροποποιημένου βιβλίου εργασίας
Τέλος, γράψτε τις αλλαγές σε ένα νέο αρχείο (ή αντικαταστήστε το αρχικό).

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ensure the path to your Excel file is correct
String outDir = "YOUR_OUTPUT_DIRECTORY"; // Specify where you want to save the modified workbook

// Load and modify your workbook as shown in previous steps
tWorkbook workbook = new Workbook(dataDir + "/Mybook.xlsx");
workbook.getSettings().setDate1904(true);

// Save the changes to a new file
workbook.save(outDir + "/I1904DateSystem_out.xls");
```

> **Σημείωση:** Ο παραπάνω κώδικας χρησιμοποιεί το όνομα κλάσης `tWorkbook` όπως δόθηκε αρχικά. Βεβαιωθείτε ότι αυτό το τυπογραφικό λάθος ταιριάζει με τις συμβάσεις ονοματοδοσίας του έργου σας ή διορθώστε το σε `Workbook` αν χρειάζεται.

## Ορισμός ημερομηνίας Excel προγραμματιστικά (δευτερεύον κλειδί)
Αν χρειάζεται να προσαρμόσετε μεμονωμένες τιμές κελιών μετά την αλλαγή του συστήματος, μπορείτε να χρησιμοποιήσετε `Cells.get(i, j).putValue(Date)` όπου η ημερομηνία θα ερμηνευτεί σύμφωνα με το ενεργό σύστημα ημερομηνίας.

## Μετατροπή του συστήματος Excel 1904 πίσω στο 1900 (δευτερεύον κλειδί)
Για επαναφορά, απλώς καλέστε:

```java
workbook.getSettings().setDate1904(false);
```

Στη συνέχεια αποθηκεύστε ξανά το βιβλίο εργασίας.

## Συχνές Ερωτήσεις

**Q: Ποια είναι η διαφορά μεταξύ των συστημάτων ημερομηνίας 1900 και 1904;**  
A: Το σύστημα 1900 ξεκινά την 1 Ιανουαρίου 1900, ενώ το σύστημα 1904 ξεκινά την 1 Ιανουαρίου 1904, μετατοπίζοντας όλες τις ημερομηνίες κατά 1462 ημέρες.

**Q: Μπορώ να αλλάξω το σύστημα ημερομηνίας ενός βιβλίου εργασίας που είναι ανοιχτό στο Excel;**  
A: Ναι, αλλά πρέπει πρώτα να κλείσετε το αρχείο στο Excel· διαφορετικά η αποθήκευση θα αποτύχει.

**Q: Χρειάζομαι άδεια για τη χρήση του `setDate1904`;**  
A: Η μέθοδος λειτουργεί στη δωρεάν δοκιμή, αλλά μια πλήρης άδεια αφαιρεί τους περιορισμούς αξιολόγησης.

**Q: Είναι δυνατόν να αλλάξω το σύστημα ημερομηνίας μόνο για ένα φύλλο εργασίας;**  
A: Όχι, το σύστημα ημερομηνίας είναι ρύθμιση επιπέδου βιβλίου εργασίας· ισχύει για όλα τα φύλλα.

**Q: Πώς μπορώ να επαληθεύσω ότι το σύστημα ημερομηνίας άλλαξε;**  
A: Ανοίξτε το αποθηκευμένο αρχείο στο Excel, μεταβείτε στο **File → Options → Advanced** και τσεκάρετε το πλαίσιο **"Use 1904 date system"**.
