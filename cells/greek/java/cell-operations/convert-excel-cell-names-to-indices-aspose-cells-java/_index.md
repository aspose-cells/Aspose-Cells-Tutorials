## Οδηγός Εφαρμογής

### Μετατροπή ονόματος κελιού σε ευρετήριο

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ονόματα κελιών του Excel (όπως "C6") στους αντίστοιχους δείκτες γραμμών και στηλών τους. Ας αναλύσουμε τα βήματα:

#### Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων

Ξεκινήστε εισάγοντας τις απαραίτητες κλάσεις από το Aspose.Cells:

```java
import com.aspose.cells.CellsHelper;
```

#### Βήμα 2: Υλοποίηση Λογικής Μετατροπής

Χρησιμοποιήστε το `CellsHelper.cellNameToIndex` μέθοδος για την εκτέλεση της μετατροπής:

```java
public class NameToIndex {
    public static void main(String[] args) throws Exception {
        // Μετατροπή ονόματος κελιού "C6" σε δείκτες
        int[] cellIndices = CellsHelper.cellNameToIndex("C6");
        
        // Έξοδος των αποτελεσμάτων
        System.out.println("Row Index of Cell C6: " + cellIndices[0]);
        System.out.println("Column Index of Cell C6: " + cellIndices[1]);
    }
}
```

**Εξήγηση**: 
- `CellsHelper.cellNameToIndex` δέχεται μια συμβολοσειρά που αντιπροσωπεύει ένα όνομα κελιού Excel και επιστρέφει έναν πίνακα όπου το πρώτο στοιχείο είναι ο δείκτης γραμμής και το δεύτερο είναι ο δείκτης στήλης.

#### Βήμα 3: Εκτελέστε τον κώδικά σας

Μεταγλωττίστε και εκτελέστε την εφαρμογή Java για να δείτε τη μετατροπή εν δράσει. Θα πρέπει να δείτε ένα αποτέλεσμα παρόμοιο με αυτό:

```
Row Index of Cell C6: 5
Column Index of Cell C6: 2
```
