```java
    }
}
```

### Μετατροπή κειμένου σε αριθμητικές τιμές

Επαναλάβετε κάθε φύλλο εργασίας και μετατρέψτε αριθμούς με μορφοποίηση κειμένου σε αριθμητικές τιμές χρησιμοποιώντας `convertStringToNumericValue()`Αυτή η μέθοδος χειρίζεται αυτόματα τη διαδικασία μετατροπής.

```java
for (int i = 0; i < workbook.getWorksheets().getCount(); i++) {
    workbook.getWorksheets().get(i).getCells().convertStringToNumericValue();
}
```

Μετά τη μετατροπή, αποθηκεύστε τις αλλαγές σε ένα αρχείο Excel. Αυτό διασφαλίζει ότι τα δεδομένα σας ενημερώνονται και αποθηκεύονται σωστά.
