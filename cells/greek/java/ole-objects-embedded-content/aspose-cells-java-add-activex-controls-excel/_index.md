## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη στοιχείου ελέγχου ActiveX σε φύλλο εργασίας

#### Επισκόπηση
Η προσθήκη ενός στοιχείου ελέγχου ActiveX, όπως ένα κουμπί εναλλαγής, περιλαμβάνει τη δημιουργία του μέσα στη συλλογή σχημάτων του φύλλου εργασίας. Αυτή η ενότητα σας καθοδηγεί σε αυτήν τη διαδικασία.

#### Οδηγός βήμα προς βήμα
**1. Δημιουργία βιβλίου εργασίας και φύλλου εργασίας Access First**
Αρχικοποιήστε το βιβλίο εργασίας σας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας του:

```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;

// Αρχικοποίηση του βιβλίου εργασίας
Workbook wb = new Workbook();

// Αποκτήστε το πρώτο φύλλο εργασίας
Worksheet sheet = wb.getWorksheets().get(0);
```

**2. Προσθήκη κουμπιού εναλλαγής ελέγχου ActiveX**
Προσθέστε ένα κουμπί εναλλαγής στο φύλλο εργασίας σας:

```java
import com.aspose.cells.ControlType;
import com.aspose.cells.Shape;

// Προσθήκη κουμπιού εναλλαγής μέσα στη συλλογή σχημάτων σε καθορισμένη θέση και μέγεθος
Shape s = sheet.getShapes().addActiveXControl(
    ControlType.TOGGLE_BUTTON, 4, 0, 4, 0, 100, 30);
```

**3. Ρύθμιση παραμέτρων ελέγχου ActiveX**
Ορίστε ιδιότητες όπως η σύνδεση κελιών για να βελτιώσετε την αλληλεπίδραση:

```java
import com.aspose.cells.ActiveXControl;

// Πρόσβαση στο αντικείμενο ελέγχου ActiveX
ActiveXControl c = s.getActiveXControl();

// Σύνδεση του στοιχείου ελέγχου με ένα κελί
c.setLinkedCell("A1");
```

**4. Αποθήκευση βιβλίου εργασίας**
Αποθηκεύστε το βιβλίο εργασίας σας στην επιθυμητή μορφή:

```java
import com.aspose.cells.SaveFormat;

// Ορίστε τον κατάλογο εξόδου
String dataDir = "path/to/your/directory/";

// Αποθήκευση του βιβλίου εργασίας ως αρχείο Excel
wb.save(dataDir + "AAXControl_out.xlsx", SaveFormat.XLSX);
```
