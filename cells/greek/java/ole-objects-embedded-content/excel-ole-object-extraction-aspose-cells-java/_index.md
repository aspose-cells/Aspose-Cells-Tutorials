# Εξαγωγή αντικειμένων OLE από το Excel με Aspose.Cells σε Java

### Εισαγωγή

Η διαχείριση σύνθετων αρχείων Excel που είναι ενσωματωμένα σε έγγραφα, υπολογιστικά φύλλα ή παρουσιάσεις μπορεί να είναι δύσκολη. Αυτό το σεμινάριο θα σας καθοδηγήσει στην εξαγωγή αντικειμένων OLE (Σύνδεση και Ενσωμάτωση Αντικειμένων) από ένα φύλλο εργασίας Excel χρησιμοποιώντας το Aspose.Cells Java.

### Οδηγός Εφαρμογής

#### Εξαγωγή αντικειμένων OLE από το Excel

Αυτή η λειτουργία δείχνει πώς να εξαγάγετε ενσωματωμένα αντικείμενα OLE από ένα φύλλο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells Java.

##### Επισκόπηση

Θα μάθετε πώς να αποκτάτε πρόσβαση και να κάνετε επανάληψη σε αντικείμενα OLE μέσα σε ένα βιβλίο εργασίας και να τα αποθηκεύετε ως ξεχωριστά αρχεία με βάση τον τύπο μορφοποίησής τους.

##### Οδηγός βήμα προς βήμα

**1. Φόρτωση του Βιβλίου Εργασίας**

Ξεκινήστε φορτώνοντας το αρχείο Excel:

```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY/";
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

**2. Πρόσβαση σε αντικείμενα OLE**

Αποκτήστε πρόσβαση στη συλλογή αντικειμένων OLE στο πρώτο φύλλο εργασίας:

```java
import com.aspose.cells.OleObjectCollection;
import com.aspose.cells.MsoDrawingType;

OleObjectCollection oles = workbook.getWorksheets().get(0).getOleObjects();
```

**3. Επανάληψη και εξαγωγή**

Επαναλάβετε κάθε αντικείμενο OLE, ελέγξτε τον τύπο του και αποθηκεύστε το:

```java
for (int i = 0; i < oles.getCount(); i++) {
    if (oles.get(i).getMsoDrawingType() == MsoDrawingType.OLE_OBJECT) {
        OleObject ole = (OleObject) oles.get(i);

        String fileName = dataDir + "tempBook1ole" + i + ".";
        switch (ole.getFileFormatType()) {
            case FileFormatType.DOC:
                fileName += "doc";
                break;
            case FileFormatType.EXCEL_97_TO_2003:
                fileName += "Xls";
                break;
            case FileFormatType.PPT:
                fileName += "Ppt";
                break;
            case FileFormatType.PDF:
                fileName += "Pdf";
                break;
            case FileFormatType.UNKNOWN:
                fileName += "Jpg";
                break;
            default:
                fileName += "data";
                break;
        }

        try (FileOutputStream fos = new FileOutputStream(fileName)) {
            byte[] data = ole.getObjectData();
            fos.write(data);
        }
    }
}
```

**Εξήγηση:**
- **Ανίχνευση μορφής αρχείου**Προσδιορίστε τη μορφή του αντικειμένου OLE για να δημιουργήσετε ένα κατάλληλο όνομα αρχείου.
- **Χειρισμός ροής byte**: Χρήση `FileOutputStream` για την εγγραφή εξαγόμενων δεδομένων, διασφαλίζοντας ότι οι πόροι διαχειρίζονται σωστά με τη μέθοδο try-with-resources.
