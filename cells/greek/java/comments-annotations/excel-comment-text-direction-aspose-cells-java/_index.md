### Προσθήκη και τροποποίηση σχολίων
#### Επισκόπηση
Βελτιώστε τη σαφήνεια του εγγράφου προσθέτοντας σχόλια στα κελιά και προσαρμόζοντας την κατεύθυνση του κειμένου τους, κάτι ιδιαίτερα χρήσιμο για πολύγλωσσα έγγραφα ή συστήματα που απαιτούν συγκεκριμένη μορφοποίηση.

**Βήμα 1: Δημιουργία ενός βιβλίου εργασίας**
Δημιουργήστε μια παρουσία του `Workbook` τάξη:
```java
// Δημιουργία νέου βιβλίου εργασίας
Workbook wb = new Workbook();
```

**Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας**
Αποκτήστε πρόσβαση στο φύλλο εργασίας όπου θέλετε να προσθέσετε σχόλια:
```java
// Λήψη του πρώτου φύλλου εργασίας στο βιβλίο εργασίας
Worksheet sheet = wb.getWorksheets().get(0);
```

**Βήμα 3: Προσθήκη σχολίου**
Προσθήκη σχολίου στο κελί "A1":
```java
// Προσθήκη σχολίου στο κελί A1
Comment comment = sheet.getComments().get(sheet.getComments().add("A1"));
```
Ανακτήστε το αντικείμενο σχολίου προσθέτοντάς το στο πρώτο κελί.

**Βήμα 4: Ρύθμιση παραμέτρων στοίχισης και κατεύθυνσης κειμένου**
Προσαρμόστε την ευθυγράμμιση κειμένου και ορίστε την κατεύθυνση από δεξιά προς τα αριστερά:
```java
// Ορισμός κάθετης και οριζόντιας στοίχισης κειμένου
comment.getCommentShape().setTextVerticalAlignment(TextAlignmentType.CENTER);
comment.getCommentShape().setTextHorizontalAlignment(TextAlignmentType.RIGHT);

// Αλλαγή κατεύθυνσης κειμένου από δεξιά προς τα αριστερά
comment.getCommentShape().setTextDirection(TextDirectionType.RIGHT_TO_LEFT);
```
Ο `TextAlignmentType` και `TextDirectionType` Οι enums βοηθούν στη διαμόρφωση της εμφάνισης των σχολίων σας.

**Βήμα 5: Ορισμός σχολίου**
Ορίστε το περιεχόμενο του σχολίου:
```java
// Προσθήκη σημείωσης στο σχόλιο
comment.setNote("This is my Comment Text. This is test");
```
