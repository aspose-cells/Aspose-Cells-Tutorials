## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να εξάγετε σχόλια στο Excel κατά την αποθήκευση των αρχείων σας ως HTML.

### Επισκόπηση της εξαγωγής σχολίων

Στόχος είναι να διασφαλιστεί ότι όλα τα σχόλια που υπάρχουν σε ένα αρχείο Excel περιλαμβάνονται στο HTML που προκύπτει. Αυτή η λειτουργία μπορεί να βελτιώσει τη σαφήνεια και το περιεχόμενο για τους χρήστες που προβάλλουν δεδομένα στο διαδίκτυο.

#### Βήμα 1: Φόρτωση του αρχείου Excel

Αρχικά, φορτώστε το βιβλίο εργασίας του Excel που θέλετε να μετατρέψετε:

```java
import com.aspose.cells.Workbook;

// Αρχικοποίηση βιβλίου εργασίας με τη διαδρομή του καταλόγου προέλευσης
String srcDir = "/path/to/your/source/";
Workbook wb = new Workbook(srcDir + "sampleExportCommentsHTML.xlsx");
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

Ορίστε το `IsExportComments` ιδιοκτησία σε `true` στο `HtmlSaveOptions`:

```java
import com.aspose.cells.HtmlSaveOptions;

// Δημιουργήστε μια παρουσία του HtmlSaveOptions και ορίστε σχόλια εξαγωγής
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.setExportComments(true);
```

#### Βήμα 3: Αποθήκευση ως HTML

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας ως αρχείο HTML με τις διαμορφωμένες επιλογές:

```java
import java.io.IOException;

// Διαδρομή καταλόγου εξόδου για την αποθήκευση του HTML
String outDir = "/path/to/your/output/";

try {
    // Αποθηκεύστε το αρχείο Excel σε μορφή HTML με σχόλια
    wb.save(outDir + "outputExportCommentsHTML.html", opts);
} catch (IOException e) {
    System.out.println("Error occurred while saving the file.");
}
```

**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι ο κατάλογος εξόδου σας είναι εγγράψιμος και έχει επαρκή χώρο.
