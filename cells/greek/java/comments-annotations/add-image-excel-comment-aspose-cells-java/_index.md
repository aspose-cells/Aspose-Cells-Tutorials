
## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε λογικές ενότητες ανά χαρακτηριστικό. Κάθε ενότητα θα σας καθοδηγήσει στον κώδικα και τον σκοπό του.

### Προσθήκη σχολίου με κείμενο σε ένα κελί του Excel

#### Επισκόπηση
Το πρώτο βήμα είναι να προσθέσετε κείμενο μέσα σε ένα σχόλιο στο φύλλο Excel σας, το οποίο μπορεί να είναι χρήσιμο για την παροχή πρόσθετων πληροφοριών ή εξηγήσεων.

#### Βήματα Υλοποίησης
**1. Δημιουργία αρχικού βιβλίου εργασίας και συλλογής σχολίων της Access**
```java
Workbook workbook = new Workbook();
CommentCollection comments = workbook.getWorksheets().get(0).getComments();
```

**2. Προσθήκη σχολίου στο κελί A1**
```java
int commentIndex = comments.add(0, 0);
Comment comment = comments.get(commentIndex);
comment.setNote("First note.");
```
Εδώ, `comments.add(0, 0)` προσθέτει ένα νέο σχόλιο στο πρώτο κελί (A1). Το `setNote` Η μέθοδος ορίζει το κείμενο του σχολίου σας.

**3. Προσαρμόστε τη γραμματοσειρά σχολίων**
```java
comment.getFont().setName("Times New Roman");
```
Η προσαρμογή των ρυθμίσεων γραμματοσειράς βελτιώνει την αναγνωσιμότητα και την παρουσίαση.

### Φόρτωση και ορισμός εικόνας σε σχήμα σχολίου

#### Επισκόπηση
Η προσθήκη εικόνων σε σχόλια μπορεί να επισημάνει οπτικά πληροφορίες ή στοιχεία επωνυμίας, όπως λογότυπα.

#### Βήματα Υλοποίησης
**1. Φόρτωση δεδομένων εικόνας**
Βεβαιωθείτε ότι η διαδρομή του αρχείου εικόνας σας έχει οριστεί σωστά:
```java
String dataDir = "YOUR_DATA_DIRECTORY";
FileInputStream inFile = new FileInputStream(dataDir + "/school.jpg");
byte[] picData = new byte[inFile.available()];
inFile.read(picData);
inFile.close();
```
Αυτός ο κώδικας διαβάζει την εικόνα σε έναν πίνακα byte, ο οποίος στη συνέχεια μπορεί να εφαρμοστεί στο σχήμα σχολίου.

**2. Ορισμός δεδομένων εικόνας**
```java
comment.getCommentShape().getFill().setImageData(picData);
```
Ο `setImageData` Η μέθοδος ενσωματώνει την εικόνα που έχετε φορτώσει απευθείας στην οπτική αναπαράσταση του σχολίου.
