### Οδηγός Εφαρμογής
#### Εξαγωγή του Excel Slicer σε PDF
**Επισκόπηση**
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο Excel με ένα γράφημα αναλυτή σε ένα έγγραφο PDF, διατηρώντας τα διαδραστικά του στοιχεία και τη διάταξή του.

**Οδηγίες βήμα προς βήμα**
1. **Φόρτωση του βιβλίου εργασίας**: Φορτώστε το αρχείο Excel χρησιμοποιώντας `Workbook`, καθορίζοντας τη διαδρομή στην οποία βρίσκεται.
   
   ```java
   // Φόρτωση του βιβλίου εργασίας από τον καθορισμένο κατάλογο
   Workbook workbook = new Workbook(dataDir + "/SampleSlicerChart.xlsx");
   ```

2. **Αποθήκευση ως PDF**: Χρησιμοποιήστε το `save` μέθοδος `Workbook` για να εξαγάγετε το αρχείο σε μορφή PDF, καθορίζοντας τον κατάλογο εξόδου και τη μορφή.
   
   ```java
   // Αποθηκεύστε το βιβλίο εργασίας σε μορφή PDF στον κατάλογο εξόδου
   workbook.save(outDir + "/SampleSlicerChart.pdf", SaveFormat.PDF);
   ```

**Βασικές επιλογές διαμόρφωσης**
- **ΑποθήκευσηΜορφή.PDF**Αυτή η σταθερά διασφαλίζει ότι το αρχείο αποθηκεύεται ως έγγραφο PDF, διατηρώντας την οπτική πιστότητα των γραφημάτων του Excel.
