## Οδηγός Εφαρμογής

### Φόρτωση του βιβλίου εργασίας

Φόρτωση ενός βιβλίου εργασίας Excel χρησιμοποιώντας το `Workbook` τάξη:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή καταλόγου εισαγωγής σας
Workbook book = new Workbook(dataDir + "/book1.xlsx"); // Φόρτωση του βιβλίου εργασίας
```
**Εξήγηση**: Το `Workbook` Το αντικείμενο είναι κρίσιμο για την πρόσβαση και τον χειρισμό αρχείων Excel. Εδώ, φορτώνουμε ένα αρχείο με όνομα `book1.xlsx`.

### Ρύθμιση παραμέτρων επιλογών απόδοσης εικόνας

Ρύθμιση παραμέτρων απόδοσης χρησιμοποιώντας `ImageOrPrintOptions`:
```java
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setImageType(ImageType.TIFF); // Ορισμός μορφής εξόδου σε TIFF
options.setOnePagePerSheet(true); // Απόδοση κάθε φύλλου σε μία μόνο σελίδα
```
**Εξήγηση**: `ImageOrPrintOptions` σας επιτρέπει να καθορίσετε παραμέτρους όπως τον τύπο και τη διάταξη εικόνας. Εδώ, χρησιμοποιούμε τη μορφή TIFF με μία εικόνα ανά φύλλο Excel.

### Απόδοση του Βιβλίου Εργασίας

Αποδώστε το βιβλίο εργασίας ως εικόνα:
```java
WorkbookRender render = new WorkbookRender(book, options); // Αρχικοποίηση του renderer με επιλογές
render.toImage("YOUR_OUTPUT_DIRECTORY/CWorkbooktoImage_out.tiff"); // Αποθήκευση εικόνας εξόδου
```
**Εξήγηση**: `WorkbookRender` παίρνει ένα `Workbook` και `ImageOrPrintOptions`, αποδίδοντας το αρχείο Excel ως εικόνα. Καθορίστε εδώ την τοποθεσία αποθήκευσης και το όνομα αρχείου.
