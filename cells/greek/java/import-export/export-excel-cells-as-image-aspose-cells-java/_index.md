## Οδηγός Εφαρμογής

### Εξαγωγή περιοχής κελιών ως εικόνα

Αυτή η ενότητα εξηγεί πώς να εξαγάγετε συγκεκριμένα κελιά από το φύλλο εργασίας του Excel σε μια εικόνα JPEG χρησιμοποιώντας το Aspose.Cells για Java.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το αρχείο Excel όπου θέλετε να εξαγάγετε την περιοχή κελιών:

```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "book1.xlsx");
```

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας

Αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει τα επιθυμητά κελιά:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### Βήμα 3: Ορισμός περιοχής εκτύπωσης

Καθορίστε ποιο μέρος του φύλλου εργασίας θα εξαχθεί ορίζοντας μια περιοχή εκτύπωσης:

```java
worksheet.getPageSetup().setPrintArea("E8:H10");
```

Αυτό το βήμα διασφαλίζει ότι μόνο το καθορισμένο εύρος λαμβάνεται υπόψη για εξαγωγή.

#### Βήμα 4: Ρύθμιση παραμέτρων επιλογών εικόνας

Ορίστε τον τρόπο με τον οποίο θέλετε να εξάγεται η εικόνα σας χρησιμοποιώντας `ImageOrPrintOptions`:

```java
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setOnePagePerSheet(true);
options.setImageType(ImageType.JPEG);
```

Εδώ, ρυθμίζουμε την εξαγωγή ώστε να δημιουργεί μία εικόνα JPEG ανά φύλλο.

#### Βήμα 5: Δημιουργία και χρήση του SheetRender

Χρησιμοποιώ `SheetRender` για να διαχειριστείτε την απόδοση του φύλλου εργασίας σας ως εικόνα:

```java
SheetRender sr = new SheetRender(worksheet, options);
sr.toImage(0, "YOUR_OUTPUT_DIRECTORY" + "ERangeofCells_out.jpg");
```

Αυτό το τελικό βήμα εμφανίζει την καθορισμένη περιοχή κελιών ως αρχείο JPEG.

### Ορισμός περιοχής εκτύπωσης ενός φύλλου εργασίας

Η προσαρμογή της περιοχής εκτύπωσης είναι ζωτικής σημασίας για τον καθορισμό του τμήματος του φύλλου εργασίας με το οποίο θα εργαστείτε. Δείτε πώς:

```java
worksheet.getPageSetup().setPrintArea("E8:H10");
```

Ορίζοντας αυτήν την επιλογή, εστιάζετε στο εύρος E8:H10 κατά την εξαγωγή ή την εκτύπωση.

### Προσαρμογή περιθωρίων φύλλου εργασίας

Για να αφαιρέσετε τα περιττά κενά γύρω από την περιοχή εκτύπωσης, ορίστε όλα τα περιθώρια στο μηδέν:

```java
worksheet.getPageSetup()
    .setLeftMargin(0)
    .setRightMargin(0)
    .setTopMargin(0)
    .setBottomMargin(0);
```

Αυτό εξασφαλίζει καθαρή εικόνα χωρίς επιπλέον περιθώρια.
