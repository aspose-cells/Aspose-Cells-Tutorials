### Ρύθμιση του Aspose.Cells για Java

#### Εγκατάσταση μέσω Maven
Συμπεριλάβετε αυτό στο δικό σας `pom.xml`:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

#### Εγκατάσταση μέσω Gradle
Προσθέστε τα παρακάτω στο δικό σας `build.gradle`:
```gradle
implementation 'com.aspose:aspose-cells:25.3'
```

## Οδηγός Εφαρμογής
### Λειτουργία 1: Δημιουργία και αρχικοποίηση βιβλίου εργασίας
**Επισκόπηση**Ξεκινήστε δημιουργώντας ένα αντικείμενο βιβλίου εργασίας που χρησιμεύει ως βάση για όλες τις επόμενες λειτουργίες.

#### Βήμα προς βήμα:
1. **Εισαγωγή απαιτούμενων κλάσεων**
    ```java
    import com.aspose.cells.Workbook;
    ```
2. **Αρχικοποίηση παρουσίας βιβλίου εργασίας**
    ```java
    String dataDir = "YOUR_DATA_DIRECTORY";
    Workbook wb = new Workbook();
    ```
Αυτό το βήμα δημιουργεί ένα κενό βιβλίο εργασίας, έτοιμο για περαιτέρω τροποποιήσεις, όπως η προσθήκη αντιστοιχίσεων XML.

### Δυνατότητα 2: Προσθήκη αντιστοίχισης XML στο βιβλίο εργασίας
**Επισκόπηση**Ενσωματώστε ένα εξωτερικό αρχείο XML στο βιβλίο εργασίας σας για προηγμένο χειρισμό και επεξεργασία δεδομένων.

#### Βήμα προς βήμα:
1. **Φόρτωση του αρχείου XML**
    ```java
    wb.getWorksheets().getXmlMaps().add(dataDir + "sample.xml");
    ```
Αυτή η γραμμή προσθέτει το `sample.xml` από τον καθορισμένο κατάλογο στη συλλογή αντιστοιχίσεων XML του βιβλίου εργασίας, επιτρέποντας την εξαγωγή δεδομένων και τις εργασίες μετασχηματισμού.

### Λειτουργία 3: Αποθήκευση βιβλίου εργασίας ως XLSX
**Επισκόπηση**Εξαγωγή του τροποποιημένου βιβλίου εργασίας σας σε αρχείο XLSX, διατηρώντας όλες τις αλλαγές που έγιναν κατά την επεξεργασία.

#### Βήμα προς βήμα:
1. **Ορισμός καταλόγου εξόδου**
    ```java
    String outDir = "YOUR_OUTPUT_DIRECTORY";
    ```
2. **Αποθήκευση του βιβλίου εργασίας σε μορφή XLSX**
    ```java
    wb.save(outDir + "AddXMLMapInsideWorkbook_out.xlsx");
    ```
Αυτό το βήμα διασφαλίζει ότι το βιβλίο εργασίας σας, το οποίο είναι πλέον εμπλουτισμένο με αντιστοιχίσεις δεδομένων XML, αποθηκεύεται σωστά για κοινή χρήση ή περαιτέρω ανάλυση.
