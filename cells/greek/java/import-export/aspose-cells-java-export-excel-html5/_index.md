
## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να εξάγουμε δεδομένα κελιών σε HTML5 χρησιμοποιώντας το Aspose.Cells για Java.

### Δημιουργία βιβλίου εργασίας και πρόσβαση σε κελιά

**Επισκόπηση:**
Ξεκινάμε δημιουργώντας μια παρουσία ενός βιβλίου εργασίας, αποκτώντας πρόσβαση σε φύλλα εργασίας και χειριζόμενοι κελιά.

1. **Αρχικοποίηση του Βιβλίου Εργασίας:**
   ```java
   // Δημιουργήστε ένα νέο βιβλίο εργασίας.
   Workbook wb = new Workbook();
   ```

2. **Φύλλο εργασίας και κελί της Access:**
   ```java
   // Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.
   Worksheet ws = wb.getWorksheets().get(0);

   // Πάρτε το κελί A1 και ορίστε την τιμή του.
   Cell cell = ws.getCells().get("A1");
cell.putValue("Αυτό είναι κείμενο.");
   ```

**Εξήγηση:**
- `Workbook` represents an Excel file.
- Accessing the first worksheet allows you to manipulate data within it.
- The `Cell` object represents a specific cell, where we input our desired content.

### Exporting Cell Data as HTML5

3. **Retrieve Normal and HTML5 Strings:**
   ```java
   // Get HTML strings from the cell.
   String strNormal = cell.getHtmlString(false);
   String strHtml5 = cell.getHtmlString(true);
   
   // Print both versions to understand differences.
   System.out.println("Normal:\r\n" + strNormal);
   System.out.println();
   System.out.println("HTML5:\r\n" + strHtml5);
   ```

**Explanation:**
- `getHtmlString(false)` ανακτά την τυπική αναπαράσταση HTML του περιεχομένου των κελιών.
- `getHtmlString(true)` παράγει μια έκδοση HTML5, εξασφαλίζοντας συμβατότητα με τον σύγχρονο ιστό.
