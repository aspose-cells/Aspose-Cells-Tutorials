```java
        // Αποθηκεύστε το έγγραφο σε μορφή HTML

    }
}
```

## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε πώς μπορείτε να αλλάξετε τους στόχους συνδέσμων HTML χρησιμοποιώντας το Aspose.Cells για Java.

### Επισκόπηση της αλλαγής στόχων συνδέσμων

Κατά την εξαγωγή αρχείων Excel σε HTML, οι υπερσύνδεσμοι μπορούν να ρυθμιστούν ώστε να ανοίγουν στο ίδιο παράθυρο ή σε νέα καρτέλα. Αυτή η λειτουργία είναι ζωτικής σημασίας για τον έλεγχο της πλοήγησης των χρηστών και τη βελτίωση του SEO.

#### Βήμα 1: Ρύθμιση του βιβλίου εργασίας σας

Ξεκινήστε φορτώνοντας το αρχείο Excel σε ένα Aspose.Cells `Workbook` αντικείμενο:

```java
String dataDir = "your/data/directory/";
String inputPath = dataDir + "Sample1.xlsx";
Workbook workbook = new Workbook(inputPath);
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

Χρησιμοποιήστε το `HtmlSaveOptions` κλάση για να καθορίσετε τον τρόπο με τον οποίο θα πρέπει να συμπεριφέρονται οι υπερσύνδεσμοι στο εξαγόμενο έγγραφο HTML:

```java
// Εισαγωγή HtmlLinkTargetType για τον ορισμό στόχων συνδέσμων
import com.aspose.cells.HtmlLinkTargetType;

HtmlSaveOptions opts = new HtmlSaveOptions();
opts.setLinkTargetType(HtmlLinkTargetType.SELF);  // Οι σύνδεσμοι ανοίγουν στο ίδιο παράθυρο/καρτέλα
```

#### Βήμα 3: Αποθήκευση του βιβλίου εργασίας ως HTML

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τις διαμορφωμένες επιλογές:

```java
String outputPath = dataDir + "CHLinkTarget.html";
workbook.save(outputPath, opts);

System.out.println("File saved at " + outputPath);
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνηθισμένο πρόβλημα:** Οι σύνδεσμοι δεν ενημερώνονται όπως αναμενόταν. Βεβαιωθείτε ότι οι σύνδεσμοι δεν ενημερώνονται όπως αναμενόταν. `HtmlSaveOptions` έχει ρυθμιστεί και εφαρμοστεί σωστά κατά τη διάρκεια της διαδικασίας αποθήκευσης.
- **Ανησυχίες σχετικά με την απόδοση:** Για μεγάλα αρχεία, σκεφτείτε να βελτιστοποιήσετε τον κώδικά σας για να χειρίζεστε αποτελεσματικά τη μνήμη.
