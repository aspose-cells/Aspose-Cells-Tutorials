## Οδηγός Εφαρμογής
### Φόρτωση βιβλίου εργασίας πηγής
#### Επισκόπηση
Το πρώτο βήμα είναι να φορτώσετε ένα βιβλίο εργασίας του Excel από έναν καθορισμένο κατάλογο. Αυτό σας επιτρέπει να έχετε πρόσβαση και να χειρίζεστε τα δεδομένα που περιέχει.

#### Απόσπασμα κώδικα
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY/TechnicalArticles/";
Workbook wb = new Workbook(dataDir + "sample_Export-Xml-Data-linked.xlsx");
```
**Εξήγηση**: 
- `dataDir` καθορίζει πού βρίσκεται το βιβλίο εργασίας σας.
- Ο `Workbook` Το αντικείμενο φορτώνει το αρχείο Excel, καθιστώντας το έτοιμο για περαιτέρω λειτουργίες.

### Εξαγωγή δεδομένων XML χαρτών XML
#### Επισκόπηση
Τώρα που το βιβλίο εργασίας έχει φορτωθεί, μπορείτε να εξαγάγετε όλα τα δεδομένα XML που σχετίζονται με τις αντιστοιχίσεις XML που περιέχει. Αυτή η λειτουργία είναι κρίσιμη για την ενσωμάτωση ή την ανάλυση δεδομένων XML μέσω προγραμματισμού.

#### Απόσπασμα κώδικα
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.XmlMap;

String outDir = "YOUR_OUTPUT_DIRECTORY/";
for (int i = 0; i < wb.getWorksheets().getXmlMaps().getCount(); i++) {
    XmlMap map = wb.getWorksheets().getXmlMaps().get(i);
    wb.exportXml(map.getName(), outDir + map.getName() + ".xml");
}
```
**Εξήγηση**: 
- Επαναλάβετε κάθε `XmlMap` στο βιβλίο εργασίας.
- Χρήση `exportXml()` μέθοδος για την εξαγωγή δεδομένων από κάθε αντιστοίχιση XML σε έναν καθορισμένο κατάλογο εξόδου.

#### Βασικές επιλογές διαμόρφωσης
- Τροποποιήστε το `dataDir` και `outDir` διαδρομές σύμφωνα με τη δομή του αρχείου σας.
- Βεβαιωθείτε ότι οι καθορισμένοι κατάλογοι υπάρχουν πριν εκτελέσετε το πρόγραμμα για να αποφύγετε εξαιρέσεις.
