### Ρύθμιση του Aspose.Cells για Java

**1. Προσθέστε την εξάρτηση:**

Ενσωματώστε το Aspose.Cells στο εργαλείο δημιουργίας σας (Maven/Gradle) όπως φαίνεται παραπάνω.

**2. Αρχικοποίηση του Aspose.Cells:**

```java
import com.aspose.cells.*;

public class ChartLocalizationSetup {
    public static void main(String[] args) {
        // Φόρτωση ενός δείγματος αρχείου Excel για εργασία με γραφήματα
        Workbook workbook = new Workbook("sample.xlsx");

        // Πρόσβαση στο πρώτο φύλλο εργασίας του βιβλίου
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // Δημιουργήστε ένα αντικείμενο γραφήματος
        int chartIndex = worksheet.getCharts().add(ChartType.COLUMN, 5, 0, 15, 5);
        Chart chart = worksheet.getCharts().get(chartIndex);

        System.out.println("Aspose.Cells setup complete. Ready to localize charts.");
    }
}
```

### Οδηγός Εφαρμογής

#### Προσαρμοσμένη Τοπική Προσαρμογή Γραφήματος

**Επισκόπηση:**
Η προσαρμογή της τοπικής προσαρμογής γραφημάτων περιλαμβάνει την προσαρμογή των ετικετών και των τίτλων στα γραφήματά σας σύμφωνα με τις τοπικές ρυθμίσεις συστήματος του χρήστη.

**Βήμα 1: Ανάκτηση τοπικών ρυθμίσεων συστήματος**

Ανάκτηση της τρέχουσας ρύθμισης γλώσσας του συστήματος χρησιμοποιώντας Java `Locale` τάξη:

```java
import java.util.Locale;

String getOtherName() {
    String language = Locale.getDefault().getLanguage();
    switch (language) {
        case "en":
            return "Other"; // Αγγλική τοπική ρύθμιση
        case "fr":
            return "Autre"; // Γαλλική τοπική κουζίνα
        case "de":
            return "Andere"; // Γερμανική τοπική κουζίνα
        default:
            return "Other"; // Προεπιλογή στα Αγγλικά εάν δεν βρεθεί αντιστοιχία
    }
}
```

**Βήμα 2: Εφαρμογή τοπικής προσαρμογής στο γράφημα**

Τροποποίηση στοιχείων γραφήματος με βάση την ανακτημένη γλώσσα:

```java
public void localizeChart(Chart chart) {
    String otherLabel = getOtherName();
    
    // Υποθέτοντας ότι η σειρά στον δείκτη 0 χρειάζεται εντοπισμό
    SeriesCollection nSeries = chart.getNSeries();
    if (nSeries.getCount() > 0) {
        nSeries.get(0).setName(otherLabel + " Data");
    }
}
```

**Παράμετροι και τιμές επιστροφής:**
- `Locale.getDefault().getLanguage()` επιστρέφει τον κωδικό γλώσσας δύο πεζών γραμμάτων.
- `chart.getNSeries().get(index)` ανακτά σειρές για να ορίσει ονόματα.

#### Συμβουλές αντιμετώπισης προβλημάτων

- **Λείπουν μεταφράσεις:** Βεβαιωθείτε ότι όλες οι απαιτούμενες τοπικές ρυθμίσεις χειρίζονται στη λογική της θήκης διακόπτη.
- **Το γράφημα δεν ενημερώνεται:** Επαληθεύστε ότι οι δείκτες γραφημάτων ταιριάζουν με αυτούς που χρησιμοποιούνται κατά τη ρύθμιση σειρών δεδομένων.
