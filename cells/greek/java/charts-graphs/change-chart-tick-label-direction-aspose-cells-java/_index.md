## Οδηγός Εφαρμογής
Τώρα, ας εξερευνήσουμε πώς να αλλάξουμε την κατεύθυνση της ετικέτας tick σε ένα γράφημα χρησιμοποιώντας το Aspose.Cells για Java.

### Φόρτωση του βιβλίου εργασίας και του γραφήματος
1. **Αρχικοποίηση του βιβλίου εργασίας**Φορτώστε το αρχείο Excel που περιέχει γραφήματα.
2. **Πρόσβαση στο Φύλλο Εργασίας**Αποκτήστε το φύλλο εργασίας που περιέχει το επιθυμητό διάγραμμα.
3. **Ανάκτηση του αντικειμένου γραφήματος**: Εξαγάγετε το συγκεκριμένο γράφημα που θέλετε να τροποποιήσετε.

```java
import com.aspose.cells.*;

public class ChangeTickLabelDirection {
    public static void main(String[] args) throws Exception {
        // Ρύθμιση καταλόγων για αρχεία προέλευσης και εξόδου
        String sourceDir = Utils.Get_SourceDirectory();
        String outputDir = Utils.Get_OutputDirectory();

        // Φόρτωση του βιβλίου εργασίας από ένα αρχείο
        Workbook workbook = new Workbook(sourceDir + "SampleChangeTickLabelDirection.xlsx");
        
        // Λήψη του πρώτου φύλλου εργασίας στο βιβλίο εργασίας
        Worksheet worksheet = workbook.getWorksheets().get(0);
        
        // Ανάκτηση του γραφήματος από το φύλλο εργασίας
        Chart chart = worksheet.getCharts().get(0);
    }
}
```

### Αλλαγή κατεύθυνσης ετικέτας τικ
4. **Τροποποίηση προσανατολισμού ετικέτας τικ**: Ορίστε τον τύπο κατεύθυνσης για τις ετικέτες υποδιαίρεσης άξονα κατηγορίας.

```java
// Αλλαγή προσανατολισμού ετικέτας υποδιαίρεσης σε οριζόντιο
chart.getCategoryAxis().getTickLabels().setDirectionType(ChartTextDirectionType.HORIZONTAL);
```

5. **Αποθήκευση αλλαγών**: Εξαγωγή του τροποποιημένου βιβλίου εργασίας σε ένα νέο αρχείο.

```java
// Αποθηκεύστε τις αλλαγές σε ένα νέο αρχείο Excel
workbook.save(outputDir + "outputChangeTickLabelDirection.xlsx");
```

**Παράμετροι και Διαμόρφωση**
- `ChartTextDirectionType.HORIZONTAL`: Καθορίζει ότι οι ετικέτες υποδιαίρεσης θα πρέπει να εμφανίζονται οριζόντια.
  
### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η έκδοση της βιβλιοθήκης Aspose.Cells ταιριάζει με αυτήν που δηλώσατε στο Maven/Gradle για να αποφύγετε προβλήματα συμβατότητας.
