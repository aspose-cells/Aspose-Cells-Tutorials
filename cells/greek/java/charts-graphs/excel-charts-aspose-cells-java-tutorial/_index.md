
## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε ξεχωριστά χαρακτηριστικά.

### Ρύθμιση βιβλίου εργασίας και φύλλου εργασίας

#### Επισκόπηση
Η δημιουργία ενός βιβλίου εργασίας είναι απαραίτητη για την προετοιμασία δεδομένων που θα χρησιμοποιηθούν στα γραφήματα του Excel. Αυτή η ενότητα παρουσιάζει τη δημιουργία ενός αρχικού βιβλίου εργασίας και τη συμπλήρωσή του με τιμές-δείγματα.

##### Δημιουργία νέου βιβλίου εργασίας

```java
Workbook workbook = new Workbook();
WorksheetCollection worksheets = workbook.getWorksheets();

// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας.
Worksheet worksheet = worksheets.get(0);
Cells cells = worksheet.getCells();
```

##### Προσθήκη δειγμάτων δεδομένων για γράφημα

Συμπληρώστε συγκεκριμένα κελιά για να προετοιμάσετε δεδομένα για δημιουργία γραφήματος:

```java
cells.get("A1").setValue(50);
cells.get("A2").setValue(100);
cells.get("A3").setValue(150);
cells.get("B1").setValue(60);
cells.get("B2").setValue(32);
cells.get("B3").setValue(50);
```

### Προσθήκη γραφήματος στο φύλλο εργασίας

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στην προσθήκη ενός γραφήματος στηλών και στον ορισμό της πηγής δεδομένων του.

##### Πρόσβαση σε συλλογή γραφημάτων και προσθήκη γραφήματος στηλών

```java
ChartCollection charts = worksheet.getCharts();
int chartIndex = charts.add(ChartType.COLUMN, 5, 0, 15, 7);
Chart chart = charts.get(chartIndex);

// Ορίστε το εύρος δεδομένων για τη σειρά.
SeriesCollection nSeries = chart.getNSeries();
nSeries.add("A1:B3", true);
```

### Προσαρμογή χρωμάτων γραφήματος

#### Επισκόπηση
Η προσαρμογή των χρωμάτων του γραφήματος βελτιώνει την οπτική αναπαράσταση και βοηθά στη διάκριση διαφορετικών στοιχείων.

##### Προσαρμόστε τα χρώματα της περιοχής σχεδίασης και της περιοχής γραφήματος

```java
ChartFrame plotArea = chart.getPlotArea();
Area area = plotArea.getArea();
area.setForegroundColor(Color.getBlue());

ChartArea chartArea = chart.getChartArea();
area = chartArea.getArea();
area.setForegroundColor(Color.getYellow());
```

##### Προσαρμόστε τα χρώματα σειρών και σημείων

```java
Series aSeries = nSeries.get(0);
area = aSeries.getArea();
area.setForegroundColor(Color.getRed());

ChartPointCollection chartPoints = aSeries.getPoints();
ChartPoint point = chartPoints.get(0);
point.getArea().setForegroundColor(Color.getCyan());
```
