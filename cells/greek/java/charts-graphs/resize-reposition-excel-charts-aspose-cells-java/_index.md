#### Φόρτωση και αλλαγή μεγέθους γραφήματος
Η αλλαγή μεγέθους ενός γραφήματος προσαρμόζει την εμφάνισή του στις ανάγκες παρουσίασης δεδομένων σας. Δείτε πώς:
##### Βήμα 1: Δημιουργία μιας παρουσίας βιβλίου εργασίας
Φορτώστε το υπάρχον αρχείο Excel δημιουργώντας μια παρουσία του `Workbook`.
```java
String filePath = "YOUR_DATA_DIRECTORY/book1.xls";
Workbook workbook = new Workbook(filePath);
```
##### Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας
Θα εργαστούμε με το πρώτο φύλλο εργασίας, το οποίο είναι συνηθισμένο σε πολλές περιπτώσεις χρήσης.
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```
##### Βήμα 3: Φόρτωση του γραφήματος
Αποκτήστε πρόσβαση στο γράφημα του οποίου το μέγεθος θέλετε να αλλάξετε. Σε αυτό το παράδειγμα, εργαζόμαστε με το πρώτο γράφημα στο φύλλο.
```java
Chart chart = worksheet.getCharts().get(0);
```
##### Βήμα 4: Αλλαγή μεγέθους του γραφήματος
Ορίστε νέες διαστάσεις για το πλάτος και το ύψος του γραφήματός σας.
```java
chart.getChartObject().setWidth(400); // Ορισμός πλάτους γραφήματος σε 400 μονάδες
chart.getChartObject().setHeight(300); // Ορισμός ύψους γραφήματος σε 300 μονάδες

// Αποθήκευση των αλλαγών
String outDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outDir + "ResizeChart_out.xls");
```
#### Επανατοποθέτηση ενός γραφήματος
Η επανατοποθέτηση γραφημάτων βελτιστοποιεί τη διάταξη και την αναγνωσιμότητα. Δείτε πώς:
##### Βήμα 1: Φόρτωση του αρχείου Excel
Φορτώστε το βιβλίο εργασίας σας.
```java
String filePath = "YOUR_DATA_DIRECTORY/book1.xls";
Workbook workbook = new Workbook(filePath);
```
##### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας και στο Διάγραμμα
Αποκτήστε πρόσβαση στο απαραίτητο φύλλο εργασίας και γράφημα, όπως ακριβώς κάνετε με την αλλαγή μεγέθους.
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```
##### Βήμα 3: Επανατοποθέτηση του γραφήματος
Προσαρμόστε τις συντεταγμένες X και Y για να μετακινήσετε το γράφημά σας μέσα στο φύλλο εργασίας.
```java
chart.getChartObject().setX(250); // Ορίστε την οριζόντια θέση στις 250 μονάδες
chart.getChartObject().setY(150); // Ορίστε την κατακόρυφη θέση στις 150 μονάδες

// Αποθηκεύστε τις αλλαγές σε ένα νέο αρχείο

workbook.save(outDir + "RepositionChart_out.xls");
```
