## Οδηγός Υλοποίησης

### Προσθήκη Υδατογραφήματος WordArt σε Διάγραμμα
1. **Ανοίξτε ένα Υπάρχον Αρχείο Excel**  
   Φορτώστε το αρχείο Excel όπου θέλετε να προσθέσετε το υδατογράφημα:
```java
String dataDir = Utils.getSharedDataDir(AddWordArtWatermarkToChart.class) + "TechnicalArticles/";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

2. **Πρόσβαση στο Διάγραμμα**  
   Αποκτήστε το διάγραμμα από το πρώτο φύλλο εργασίας που θέλετε να τροποποιήσετε:
```java
Chart chart = workbook.getWorksheets().get(0).getCharts().get(0);
```

3. **Προσθήκη Σχήματος WordArt**  
   Εισάγετε ένα νέο σχήμα WordArt στην περιοχή σχεδίασης του διαγράμματος:
```java
Shape wordart = chart.getShapes().addTextEffectInChart(
    MsoPresetTextEffect.TEXT_EFFECT_1,
    "CONFIDENTIAL",
    "Arial Black", 66, false, false, 
    1200, 500, 2000, 3000);
```

4. **Διαμόρφωση Γέμισης και Μορφής Γραμμής**  
   Ορίστε τη διαφάνεια ώστε το υδατογράφημα να είναι διακριτικό:
```java
// Configure transparency.
FillFormat wordArtFormat = wordart.getFill();
wordArtFormat.setTransparency(0.9);

// Make line format invisible.
LineFormat lineFormat = wordart.getLine();
lineFormat.setWeight(0.0);
```

5. **Αποθήκευση του Βιβλίου Εργασίας**  
   Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:
```java
workbook.save(dataDir + "AWArtWToC_out.xlsx");
```

**Q: Εμφανίζεται το υδατογράφημα στα εκτυπωμένα φύλλα Excel;**  
A: Ναι, το σχήμα WordArt είναι μέρος του διαγράμματος και εκτυπώνεται μαζί με τα δεδομένα του διαγράμματος.

**Q: Μπορώ να εφαρμόσω το ίδιο υδατογράφημα σε πολλαπλά διαγράμματα αυτόματα;**  
A: Επανάληψη μέσω `workbook.getWorksheets().get(i).getCharts()` και εφαρμογή των ίδιων βημάτων σε κάθε διάγραμμα.

**Q: Είναι δυνατόν να αλλάξω το χρώμα του υδατογραφήματος;**  
A: Απόλυτα—χρησιμοποιήστε `wordArtFormat.getSolidFill().setColor(Color.getRGB(255,0,0))` για να ορίσετε προσαρμοσμένο χρώμα.

**Q: Θα αυξήσει η προσθήκη υδατογραφήματος το μέγεθος του αρχείου σημαντικά;**  
A: Η αύξηση είναι ελάχιστη, καθώς προστίθεται μόνο ένα αντικείμενο σχήματος.

**Q: Πώς μπορώ να αφαιρέσω το υδατογράφημα αργότερα;**  
A: Εντοπίστε το σχήμα με το όνομά του ή το δείκτη στο `chart.getShapes()` και καλέστε `shape.delete()`.
