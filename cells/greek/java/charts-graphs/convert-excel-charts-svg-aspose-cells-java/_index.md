## Τι είναι το “how to convert svg”;
Το “how to convert svg” αναφέρεται στη διαδικασία μετατροπής μιας ραστερ εικόνας ή ενός διαγράμματος Excel σε αρχείο Scalable Vector Graphics (SVG). Το SVG είναι μια μορφή διανύσματος βασισμένη σε XML που διατηρεί την οπτική πιστότητα σε οποιοδήποτε μέγεθος, επιτρέποντας στα γραφικά να κλιμακώνονται χωρίς εικονοστοιχίες. Αυτή η μετατροπή παρέχει καθαρά, ανεξάρτητα από την ανάλυση οπτικά στοιχεία κατάλληλα για ιστοσελίδες, αναφορές και ανταποκρινόμενα σχέδια.

## Πώς να φορτώσετε ένα βιβλίο εργασίας Excel σε Java;
Η κλάση `Workbook` αντιπροσωπεύει ένα αρχείο Excel που έχει φορτωθεί στη μνήμη, παρέχοντας πρόσβαση στα φύλλα εργασίας, τα κελιά και τα διαγράμματα.

Φορτώστε το βιβλίο εργασίας με `new Workbook("path/to/file.xlsx")` – αυτή η μοναδική γραμμή διαβάζει ολόκληρο το φύλλο σε μνήμη, δίνοντάς σας προγραμματιστική πρόσβαση σε όλα τα φύλλα εργασίας, τα κελιά και τα ενσωματωμένα διαγράμματα. Το Aspose.Cells ανιχνεύει αυτόματα τη μορφή του αρχείου, οπότε δεν χρειάζεται να καθορίσετε ρητά XLSX, XLS ή CSV.

## Φόρτωση Βιβλίου Εργασίας από Αρχείο
**Επισκόπηση:**  
Το πρώτο βήμα είναι η φόρτωση ενός βιβλίου εργασίας Excel. Αυτό προετοιμάζει το περιβάλλον για πρόσβαση στα διαγράμματα.

```java
import com.aspose.cells.Workbook;
// Load an Excel workbook from a specified directory.
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "sample.xlsx");
```

**Εξήγηση:**  
- Η κλάση `Workbook` είναι το αντικείμενο ανώτερου επιπέδου που αντιπροσωπεύει ένα μόνο αρχείο Excel στη μνήμη.  
- Παρέχετε το πλήρες μονοπάτι στο αρχείο Excel μέσω της μεταβλητής `dataDir` ή ενός απόλυτου μονοπατιού.

## Πώς να έχετε πρόσβαση σε συγκεκριμένο φύλλο εργασίας και διάγραμμα;
Ένα αντικείμενο `Worksheet` αντιστοιχεί σε ένα μόνο φύλλο εντός του βιβλίου εργασίας, περιέχοντας σειρές, στήλες και ενσωματωμένα αντικείμενα.  
Ένα αντικείμενο `Chart` αντιπροσωπεύει μια γραφική αναπαράσταση δεδομένων σε ένα φύλλο εργασίας, η οποία μπορεί να αποδοθεί ή να εξαχθεί.

Ανακτήστε το φύλλο εργασίας με `workbook.getWorksheets().get(0)` και στη συνέχεια καλέστε `getCharts().get(0)` για να λάβετε το πρώτο αντικείμενο διαγράμματος – αυτή η άμεση προσέγγιση λειτουργεί για οποιονδήποτε δείκτη διαγράμματος χρειάζεστε. Το API επιστρέφει μια παρουσία `Chart` έτοιμη για απόδοση ή εξαγωγή δεδομένων.

## Πρόσβαση σε Φύλλο Εργασίας και Διάγραμμα
**Επισκόπηση:**  
Μετά τη φόρτωση, αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας και διάγραμμα που θέλετε να μετατρέψετε.

```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Chart;
// Access the first worksheet and its first chart.
Worksheet worksheet = workbook.getWorksheets().get(0);
Chart chart = worksheet.getCharts().get(0);
```

**Εξήγηση:**  
- `worksheet` είναι ένα αντικείμενο τύπου `Worksheet`.  
- `chart` ανακτάται από τη συλλογή διαγραμμάτων του φύλλου εργασίας.

## Πώς να μετατρέψετε ένα διάγραμμα σε εικόνα SVG;
Η κλάση `ImageOrPrintOptions` ορίζει τις ρυθμίσεις απόδοσης όπως μορφή εξόδου, ανάλυση και ποιότητα για τη μετατροπή διαγραμμάτων ή φύλλων εργασίας σε αρχεία εικόνας.

Δημιουργήστε μια παρουσία `ImageOrPrintOptions`, ορίστε το `setSaveFormat(SaveFormat.SVG)`, στη συνέχεια καλέστε `chart.toImage(options, "output.svg")`. Αυτή η κλήση μίας γραμμής γράφει ένα πλήρως συμβατό αρχείο SVG που διατηρεί τα χρώματα, τις γραμματοσειρές και τις ετικέτες δεδομένων ακριβώς όπως εμφανίζονται στο Excel.

## Μετατροπή Διαγράμματος σε Εικόνα SVG
**Επισκόπηση:**  
Το τελικό βήμα περιλαμβάνει τη μετατροπή του διαγράμματος σε εικόνα SVG για υψηλής ποιότητας προβολή.

```java
import com.aspose.cells.ImageOrPrintOptions;
import com.aspose.cells.SaveFormat;
// Convert and save the chart as an SVG image.
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setSaveFormat(SaveFormat.SVG);
String outDir = "YOUR_OUTPUT_DIRECTORY";
chart.toImage(outDir + "CCToImageinSVGFormat_out.svg", options);
```

**Εξήγηση:**  
- Το `ImageOrPrintOptions` διαμορφώνει τον τρόπο αποθήκευσης του διαγράμματος.  
- Ορίζοντας τη μορφή σε SVG, το Aspose.Cells δημιουργεί ένα διανυσματικό γραφικό.  
- Το παραγόμενο αρχείο μπορεί να ενσωματωθεί απευθείας σε HTML ή φόντο CSS.
