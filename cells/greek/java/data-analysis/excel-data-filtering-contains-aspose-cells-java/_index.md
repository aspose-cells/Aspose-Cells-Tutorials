## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς δημιουργείται ένα προσαρμοσμένο φίλτρο που αναγνωρίζει γραμμές που περιέχουν συγκεκριμένο κείμενο χρησιμοποιώντας το Aspose.Cells για Java.

### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο Excel. Αυτή είναι η πηγή δεδομένων σας.

```java
Workbook workbook = new Workbook(srcDir + "sourseSampleCountryNames.xlsx");
```

### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας

Αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει τα δεδομένα που θέλετε να φιλτράρετε:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

### Βήμα 3: Ρύθμιση του Αυτόματου Φίλτρου

Δημιουργήστε ένα `AutoFilter` καθορίζοντας ένα εύρος. Αυτό καθορίζει την περιοχή του φύλλου σας που θα φιλτραριστεί:

```java
worksheet.getAutoFilter().setRange("A1:A18");
```

### Βήμα 4: Εφαρμογή του προσαρμοσμένου φίλτρου

Χρησιμοποιήστε το `custom()` μέθοδος για την εφαρμογή ενός φίλτρου που αναζητά γραμμές που περιέχουν συγκεκριμένο κείμενο, στην προκειμένη περίπτωση "Ba":

```java
worksheet.getAutoFilter().custom(0, FilterOperatorType.CONTAINS, "Ba");
```

**Εξήγηση**: 
- Η πρώτη παράμετρος (`0`) καθορίζει τον δείκτη της στήλης.
- `FilterOperatorType.CONTAINS` είναι η εφαρμοζόμενη λογική φιλτραρίσματος.
- `"Ba"` είναι η συμβολοσειρά που αναζητάτε εντός του καθορισμένου εύρους.

### Βήμα 5: Ανανέωση και αποθήκευση

Αφού ρυθμίσετε το φίλτρο, ανανεώστε το για να εφαρμόσετε τις αλλαγές και να αποθηκεύσετε το βιβλίο εργασίας σας:

```java
worksheet.getAutoFilter().refresh();
workbook.save(outDir + "outSourseSampleCountryNames.xlsx");
```

**Συμβουλές αντιμετώπισης προβλημάτων**Εάν αντιμετωπίσετε προβλήματα με διαδρομές αρχείων ή περιοχές δεδομένων, βεβαιωθείτε ότι έχουν καθοριστεί σωστά σε σχέση με τη δομή καταλόγου του έργου σας.
