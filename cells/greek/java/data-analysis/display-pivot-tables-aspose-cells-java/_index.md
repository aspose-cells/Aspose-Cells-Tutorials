- Βήματα για την παρουσίαση συγκεντρωτικών πινάκων σε μορφή πίνακα

## Ρύθμιση του Aspose.Cells για Java

Για να χρησιμοποιήσετε το Aspose.Cells για Java, πρέπει να το συμπεριλάβετε στο έργο σας. Έχετε δύο επιλογές: Maven ή Gradle.

### Maven
Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Γκράντλ
Συμπεριλάβετε αυτό στο δικό σας `build.gradle` αρχείο:
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

Το Aspose.Cells προσφέρει δωρεάν δοκιμαστική περίοδο, προσωρινή άδεια χρήσης για σκοπούς αξιολόγησης και επιλογές αγοράς για μακροχρόνια χρήση. Επισκεφθείτε την ιστοσελίδα [Αγορά Aspose](https://purchase.aspose.com/buy) για να εξερευνήσετε τις επιλογές αδειοδότησης που έχετε.

## Οδηγός Εφαρμογής

Θα χωρίσουμε την υλοποίηση σε τρία μέρη: Συμπαγή, Περίγραμμα και Πινακοποιημένες Μορφές.

### Εμφάνιση Συγκεντρωτικού Πίνακα σε Συμπαγή Μορφή

**Επισκόπηση:** Η εμφάνιση ενός συγκεντρωτικού πίνακα σε συμπαγή μορφή βοηθά στην εξοικονόμηση χώρου, διατηρώντας παράλληλα την ευκρίνεια.

#### Βήμα 1: Φόρτωση του αρχείου Excel
```java
String dataDir = "YOUR_DATA_DIRECTORY";
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```
*Γιατί;* Αυτό φορτώνει το αρχείο προέλευσης Excel στη μνήμη.

#### Βήμα 2: Φύλλο εργασίας και Συγκεντρωτικός πίνακας Access
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### Βήμα 3: Ορισμός Συμπαγούς Φόρμας
```java
pivotTable.showInCompactForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/CompactForm.xlsx");
```
*Γιατί;* Αυτή η διαμόρφωση εμφανίζει τον συγκεντρωτικό πίνακα σε συμπυκνωμένη μορφή και τον αποθηκεύει.

### Εμφάνιση Συγκεντρωτικού Πίνακα σε Μορφή Διάρθρωσης

**Επισκόπηση:** Η φόρμα διάρθρωσης είναι ιδανική για ιεραρχικά δεδομένα, επιτρέποντας στους χρήστες να επεκτείνουν ή να συμπτύξουν λεπτομέρειες.

#### Βήμα 1: Φόρτωση βιβλίου εργασίας
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### Βήμα 2: Πρόσβαση στα απαραίτητα στοιχεία
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### Βήμα 3: Ρύθμιση παραμέτρων φόρμας διάρθρωσης
```java
pivotTable.showInOutlineForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/OutlineForm.xlsx");
```
*Γιατί;* Αυτό το βήμα ορίζει τον συγκεντρωτικό πίνακα σε μορφή διάρθρωσης και διασφαλίζει ότι τα δεδομένα ενημερώνονται.

### Εμφάνιση Συγκεντρωτικού Πίνακα σε μορφή πίνακα

**Επισκόπηση:** Η μορφή πίνακα εμφανίζει όλα τα δεδομένα σε γραμμές, ιδανική για λεπτομερή ανάλυση.

#### Βήμα 1: Αρχικοποίηση βιβλίου εργασίας
```java
Workbook workbook = new Workbook(dataDir + "source.xlsx");
```

#### Βήμα 2: Πρόσβαση σε Στοιχεία
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

#### Βήμα 3: Ορισμός μορφής πίνακα
```java
pivotTable.showInTabularForm();
pivotTable.refreshData();
pivotTable.calculateData();
workbook.save("YOUR_OUTPUT_DIRECTORY/TabularForm.xlsx");
```
*Γιατί;* Αυτή η διαμόρφωση παρουσιάζει τον συγκεντρωτικό πίνακα σε μορφή πίνακα.
