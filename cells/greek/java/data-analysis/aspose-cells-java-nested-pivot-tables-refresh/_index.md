### Λειτουργία 4: Ανανέωση και υπολογισμός δεδομένων για θυγατρικούς συγκεντρωτικούς πίνακες

**Επισκόπηση:** Η ενημέρωση των δεδομένων σας είναι ζωτικής σημασίας για την ακριβή ανάλυση και αναφορά.

#### 6.1 Επανάληψη σε Συγκεντρωτικούς Πίνακες Υπό-Θυγατρικών
```java
for (int idx = 0; idx < ptChildren.length; idx++) {
    PivotTable ptChild = ptChildren[idx];
    
    // Ανανέωση των δεδομένων κάθε θυγατρικού συγκεντρωτικού πίνακα.
    ptChild.refreshData();
    
    // Υπολογίστε ξανά τα δεδομένα με βάση το ανανεωμένο περιεχόμενο.
    ptChild.calculateData();
}
```
- **Σκοπός:** Διασφαλίζει ότι όλα τα δεδομένα σε ένθετους συγκεντρωτικούς πίνακες είναι ενημερωμένα και ακριβή.
