## Οδηγός Εφαρμογής

### Επισκόπηση

Αυτή η ενότητα θα σας καθοδηγήσει στην εφαρμογή μερικών αθροισμάτων και στην προσαρμογή των ετικετών στα φύλλα εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για Java.

### Οδηγίες βήμα προς βήμα

#### 1. Φόρτωση του βιβλίου εργασίας πηγαίου κώδικα

Αρχικά, φορτώστε το αρχείο Excel που περιέχει τα δεδομένα σας:

```java
// Η διαδρομή προς τον κατάλογο εγγράφων.
String dataDir = Utils.getSharedDataDir(ImplementSubtotalGrandTotallabels.class) + "TechnicalArticles/";

// Φόρτωση του βιβλίου εργασίας πηγαίου κώδικα
Workbook wb = new Workbook(dataDir + "sample.xlsx");
```

#### 2. Προσαρμόστε τις ετικέτες Μερικού και Γενικού Συνόλου

Για να προσαρμόσετε τοπικά αυτές τις ετικέτες, ορίστε τις ρυθμίσεις παγκοσμιοποίησης:

```java
// Ορίστε τη ρύθμιση παγκοσμιοποίησης για να αλλάξετε τα ονόματα των μερικών και των γενικών αθροισμάτων
GlobalizationSettings gsi = new GlobalizationSettingsImp();
wb.getSettings().setGlobalizationSettings(gsi);
```

#### 3. Πρόσβαση στο φύλλο εργασίας σας

Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας όπου θέλετε να εφαρμόσετε μερικά αθροίσματα:

```java
// Πρώτο φύλλο εργασίας της Access
Worksheet ws = wb.getWorksheets().get(0);
```

#### 4. Εφαρμογή συνάρτησης μερικού αθροίσματος

Χρήση `subtotal` μέθοδος στην επιθυμητή περιοχή κελιών, καθορίζοντας ποιες στήλες θα μερικό άθροισμα και χρησιμοποιώντας μια συνάρτηση ενοποίησης όπως `SUM`:

```java
// Εφαρμογή μερικού αθροίσματος στο A1:B10 για τις στήλες 2, 3 και 4 (η δημιουργία ευρετηρίου ξεκινά από το 0)
CellArea ca = CellArea.createCellArea("A1", "B10");
ws.getCells().subtotal(ca, 0, ConsolidationFunction.SUM, new int[] { 2, 3, 4 });
```

#### 5. Προσαρμογή πλάτους στήλης

Για καλύτερη ορατότητα, μπορείτε να προσαρμόσετε τα πλάτη των στηλών:

```java
// Ορίστε το πλάτος της πρώτης στήλης
ws.getCells().setColumnWidth(0, 40);
```

#### 6. Αποθηκεύστε το βιβλίο εργασίας σας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με όλες τις αλλαγές που έχουν εφαρμοστεί:

```java
// Αποθηκεύστε το αρχείο excel εξόδου
wb.save(dataDir + "ImplementTotallabels_out.xlsx");
```
