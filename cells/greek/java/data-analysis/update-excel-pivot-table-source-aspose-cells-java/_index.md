## Ρύθμιση του Aspose.Cells για Java
Για χρήση **Aspose.Cells για Java**, συμπεριλάβετέ το ως εξάρτηση στο έργο σας:

**Εξάρτηση Maven:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

**Εξάρτηση Gradle:**
```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα δούμε πώς να αλλάζουμε τα δεδομένα προέλευσης ενός συγκεντρωτικού πίνακα χρησιμοποιώντας το Aspose.Cells για Java.

### Βήμα 1: Φόρτωση ενός υπάρχοντος αρχείου Excel
Αρχικά, φορτώστε το υπάρχον αρχείο Excel που περιέχει τον συγκεντρωτικό πίνακα.

**Επεξήγηση Κώδικα:**
```java
import com.aspose.cells.Workbook;
import AsposeCellsExamples.Utils;

public class ChangeSourceData {
    public static void main(String[] args) throws Exception {
        // Ορίστε τη διαδρομή προς τον κατάλογο δεδομένων σας.
        String dataDir = Utils.getSharedDataDir(ChangeSourceData.class) + "PivotTables/";
        
        // Φορτώστε το βιβλίο εργασίας με έναν υπάρχοντα συγκεντρωτικό πίνακα.
        Workbook workbook = new Workbook(dataDir + "PivotTable.xls");
    }
}
```
- **`Workbook workbook = new Workbook(...)`**: Δημιουργεί ένα `Workbook` αντικείμενο, που αντιπροσωπεύει το αρχείο Excel σας.

### Βήμα 2: Πρόσβαση και τροποποίηση δεδομένων φύλλου εργασίας
Αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει τον συγκεντρωτικό σας πίνακα και ενημερώστε τα δεδομένα του.

**Επεξήγηση Κώδικα:**
```java
import com.aspose.cells.Worksheet;
import com.aspose.cells.Cells;
import com.aspose.cells.Cell;

public class ChangeSourceData {
    public static void main(String[] args) throws Exception {
        // Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας.
        Worksheet worksheet = workbook.getWorksheets().get(0);

        // Λήψη συλλογής κελιών και ενημέρωση συγκεκριμένων τιμών κελιών.
        Cells cells = worksheet.getCells();
        
        Cell cell = cells.get("A9");
        cell.setValue("Golf");

        cell = cells.get("B9");
        cell.setValue("Qtr4");

        cell = cells.get("C9");
        cell.setValue(7000);
    }
}
```
- **`cells.get("A9").setValue(...)`**: Πρόσβαση και τροποποίηση της τιμής συγκεκριμένων κελιών.

### Βήμα 3: Ενημέρωση ονομασμένης περιοχής
Αλλάξτε την ονομασμένη περιοχή που χρησιμεύει ως πηγή για τον συγκεντρωτικό σας πίνακα.

**Επεξήγηση Κώδικα:**
```java
import com.aspose.cells.Range;

public class ChangeSourceData {
    public static void main(String[] args) throws Exception {
        // Δημιουργήστε ένα νέο εύρος και ορίστε το ως προέλευση δεδομένων.
        Range range = cells.createRange(0, 0, 8, 2);
        range.setName("DataSource");
    }
}
```
- **`cells.createRange(...)`**: Ορίζει μια περιοχή κελιών και ενημερώνει το όνομά της ώστε να ταιριάζει με την πηγή δεδομένων του συγκεντρωτικού πίνακα.

### Βήμα 4: Αποθήκευση αλλαγών
Τέλος, αποθηκεύστε τις τροποποιήσεις σας σε ένα αρχείο Excel.

**Επεξήγηση Κώδικα:**
```java
public class ChangeSourceData {
    public static void main(String[] args) throws Exception {
        // Αποθηκεύστε το βιβλίο εργασίας με τις αλλαγές.
        workbook.save(dataDir + "ChangeSourceData_out.xls");
    }
}
```
- **`workbook.save(...)`**: Εγγράφει τις αλλαγές σας σε ένα νέο αρχείο Excel.
