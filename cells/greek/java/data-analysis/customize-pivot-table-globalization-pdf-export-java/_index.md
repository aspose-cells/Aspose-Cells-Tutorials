# Προσαρμόστε την παγκοσμιοποίηση του Συγκεντρωτικού Πίνακα και την εξαγωγή PDF σε Java με το Aspose.Cells

## Εισαγωγή

Δυσκολεύεστε με την προσαρμογή ετικετών συγκεντρωτικού πίνακα ή την εξαγωγή τους σε PDF; Αυτό το σεμινάριο σας καθοδηγεί στην εφαρμογή μιας ισχυρής λύσης χρησιμοποιώντας την ισχυρή βιβλιοθήκη Aspose.Cells για Java. Μάθετε πώς να προσαρμόζετε τις ρυθμίσεις παγκοσμιοποίησης συγκεντρωτικού πίνακα και να αποθηκεύετε τα αποτελέσματά σας ως PDF, διασφαλίζοντας ότι οι παρουσιάσεις δεδομένων σας είναι ακριβείς και οπτικά ελκυστικές.

### Τι θα μάθετε:
- Προσαρμόστε τις ετικέτες του συγκεντρωτικού πίνακα με συγκεκριμένα ονόματα
- Εφαρμογή προσαρμοσμένων ρυθμίσεων παγκοσμιοποίησης σε βιβλία εργασίας του Excel
- Εξαγωγή προσαρμοσμένων συγκεντρωτικών πινάκων σε μορφή PDF
- Βελτιστοποιήστε τη βιβλιοθήκη Aspose.Cells για αποτελεσματικές εφαρμογές Java

Είστε έτοιμοι να βελτιώσετε τις δεξιότητές σας στην παρουσίαση δεδομένων; Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:
- **Βιβλιοθήκη Aspose.Cells**Έκδοση 25.3 ή νεότερη.
- **Κιτ ανάπτυξης Java (JDK)**Το JDK θα πρέπει να εγκατασταθεί και να ρυθμιστεί στο σύστημά σας.
- **Ρύθμιση IDE**Χρησιμοποιήστε ένα IDE όπως το IntelliJ IDEA ή το Eclipse για ευκολότερη διαχείριση κώδικα.

## Ρύθμιση του Aspose.Cells για Java

### Εγκατάσταση Maven

Για να συμπεριλάβετε το Aspose.Cells στο έργο Maven σας, προσθέστε την ακόλουθη εξάρτηση στο `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-cells</artifactId>
    <version>25.3</version>
</dependency>
```

### Εγκατάσταση Gradle

Για τους χρήστες του Gradle, συμπεριλάβετε τα εξής στο αρχείο δημιουργίας:

```gradle
compile(group: 'com.aspose', name: 'aspose-cells', version: '25.3')
```

## Οδηγός Εφαρμογής

Θα το αναλύσουμε σε δύο κύριες λειτουργίες: την προσαρμογή των ρυθμίσεων παγκοσμιοποίησης του συγκεντρωτικού πίνακα και την εξαγωγή σε PDF.

### Ρυθμίσεις παγκοσμιοποίησης προσαρμοσμένου συγκεντρωτικού πίνακα

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να ορίσετε συγκεκριμένες ετικέτες για διάφορα στοιχεία ενός συγκεντρωτικού πίνακα, παρέχοντας μεγαλύτερο έλεγχο στην εμφάνισή του σε διαφορετικές τοπικές ρυθμίσεις ή προσαρμοσμένες μορφές.

#### Βήματα Υλοποίησης
1. **Ορισμός προσαρμοσμένων ετικετών**
   Δημιουργήστε μια κλάση που επεκτείνεται `GlobalizationSettings`:

   ```java
   import com.aspose.cells.*;

   public class CustomPivotTableGlobalizationSettings extends GlobalizationSettings {
       public String getPivotTotalName() { return "AsposeGetPivotTotalName"; }
       // Ορίστε άλλες μεθόδους παρόμοιες με τις παραπάνω για κάθε ετικέτα που θέλετε να προσαρμόσετε
   }
   ```

2. **Εφαρμογή ρυθμίσεων**
   Φορτώστε το βιβλίο εργασίας σας και εφαρμόστε αυτές τις ρυθμίσεις:

   ```java
   Workbook wb = new Workbook("YOUR_DATA_DIRECTORY/samplePivotTableGlobalizationSettings.xlsx");
   wb.getSettings().setGlobalizationSettings(new CustomPivotTableGlobalizationSettings());
   ```

### Εξαγωγή σε PDF

#### Επισκόπηση

Αφού ρυθμίσετε τον συγκεντρωτικό σας πίνακα, ίσως θελήσετε να τον εξαγάγετε ως PDF. Αυτή η ενότητα δείχνει πώς να αποθηκεύσετε αποτελεσματικά το προσαρμοσμένο βιβλίο εργασίας του Excel.

#### Βήματα Υλοποίησης
1. **Απόκρυψη φύλλου δεδομένων**
   Εάν το φύλλο δεδομένων δεν χρειάζεται στο τελικό αποτέλεσμα:

   ```java
   wb.getWorksheets().get(0).setVisible(false);
   ```

2. **Ανανέωση και Υπολογισμός Συγκεντρωτικού Πίνακα**
   Βεβαιωθείτε ότι ο συγκεντρωτικός πίνακας αντικατοπτρίζει τα πιο πρόσφατα δεδομένα:

   ```java
   Worksheet ws = wb.getWorksheets().get(1);
   PivotTable pt = ws.getPivotTables().get(0);

   pt.setRefreshDataFlag(true);
   pt.refreshData();
   pt.calculateData();
   pt.setRefreshDataFlag(false);
   ```

3. **Αποθήκευση ως PDF**
   Ρυθμίστε τις επιλογές αποθήκευσης και εξαγωγής:

   ```java
   PdfSaveOptions options = new PdfSaveOptions();
   options.setOnePagePerSheet(true);

   wb.save("YOUR_OUTPUT_DIRECTORY/outputPivotTableGlobalizationSettings.pdf", options);
   ```
