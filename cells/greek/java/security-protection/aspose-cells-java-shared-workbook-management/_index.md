#### Βήμα 1: Αρχικοποίηση του Aspose.Cells και δημιουργία νέου βιβλίου εργασίας

**Επισκόπηση**Προσαρμόστε τον αριθμό ημερών για τις οποίες ένα κοινόχρηστο βιβλίο εργασίας διατηρεί το ιστορικό αναθεωρήσεών του.

#### Βήμα 1: Αρχικοποίηση του Aspose.Cells και δημιουργία νέου βιβλίου εργασίας
