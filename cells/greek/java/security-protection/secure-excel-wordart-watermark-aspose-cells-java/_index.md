## Οδηγός Εφαρμογής
### Προσθήκη υδατογραφήματος WordArt
#### Επισκόπηση:
Ένα υδατογράφημα WordArt στο Excel μπορεί να αποτρέψει αποτελεσματικά τη μη εξουσιοδοτημένη διανομή των εγγράφων σας. Σε αυτήν την ενότητα, θα μάθουμε πώς να προσθέτουμε και να προσαρμόζουμε ένα εφέ WordArt "ΕΜΠΙΣΤΕΥΤΙΚΟ" στο υπολογιστικό σας φύλλο.

**Βήμα 1: Δημιουργία ή φόρτωση βιβλίου εργασίας**
```java
// Δημιουργία ενός νέου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```
Αυτό το βήμα αρχικοποιεί ένα νέο αρχείο Excel όπου μπορείτε να ξεκινήσετε την προσθήκη περιεχομένου και υδατογραφημάτων.

**Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας**
```java
Worksheet sheet = workbook.getWorksheets().get(0);
```
Η πρόσβαση στο πρώτο φύλλο εργασίας είναι απαραίτητη, καθώς συνήθως εκεί θέλετε να προσθέσετε το υδατογράφημά σας.

**Βήμα 3: Προσθήκη σχήματος WordArt**
```java
Shape wordart = sheet.getShapes().addTextEffect(
    MsoPresetTextEffect.TEXT_EFFECT_1, 
    "CONFIDENTIAL", 
    "Arial Black", 
    50, 
    false, 
    true, 
    18, 
    8, 
    1, 
    1, 
    130, 
    800
);
```
Αυτό το απόσπασμα προσθέτει ένα σχήμα WordArt με το κείμενο "ΕΜΠΙΣΤΕΥΤΙΚΟ". `MsoPresetTextEffect.TEXT_EFFECT_1` χρησιμοποιείται για styling.

**Βήμα 4: Προσαρμογή εμφάνισης**
```java
FillFormat wordArtFormat = wordart.getFill();
wordArtFormat.setOneColorGradient(Color.getRed(), 0.2, GradientStyleType.HORIZONTAL, 2);
wordArtFormat.setTransparency(0.9);
```
Εδώ ορίζουμε μια κόκκινη διαβάθμιση και προσαρμόζουμε τη διαφάνεια για να κάνουμε το υδατογράφημα διακριτικό αλλά αισθητό.

**Βήμα 5: Κάντε τη γραμμή αόρατη**
```java
wordart.setHasLine(false);
```
Αν βεβαιωθείτε ότι δεν υπάρχει περίγραμμα γύρω από το WordArt σας, αυτό θα φαίνεται πιο καθαρό.

**Βήμα 6: Κλείδωμα όψεων σχήματος**
```java
wordart.setLocked(true);
wordart.setLockedProperty(ShapeLockType.SELECTION, true);
wordart.setLockedProperty(ShapeLockType.SHAPE_TYPE, true);
wordart.setLockedProperty(ShapeLockType.MOVE, true);
wordart.setLockedProperty(ShapeLockType.RESIZE, true);
wordart.setLockedProperty(ShapeLockType.TEXT, true);
```
Αυτές οι γραμμές διασφαλίζουν ότι το υδατογράφημα δεν μπορεί να τροποποιηθεί ή να αφαιρεθεί εύκολα.
