## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: LoadOptions για φιλτράρισμα μακροεντολών VBA
Αυτή η δυνατότητα σάς επιτρέπει να καθορίσετε επιλογές φόρτωσης που εξαιρούν μακροεντολές VBA κατά το άνοιγμα ενός βιβλίου εργασίας.

#### Επισκόπηση
Ρυθμίζοντας `LoadFilter` με `~LoadDataFilterOptions.VBA`, μπορείτε να αποτρέψετε τη φόρτωση στοιχείων VBA στα βιβλία εργασίας του Excel, βελτιώνοντας την ασφάλεια και την απόδοση.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Ορισμός επιλογών φόρτωσης**

```java
// Εισαγωγή απαιτούμενων κλάσεων Aspose.Cells
import com.aspose.cells.*;

public class ExcludeVbaMacros {
    public static void main(String[] args) {
        String dataDir = "YOUR_DATA_DIRECTORY";
        
        // Δημιουργήστε επιλογές φόρτωσης με τις επιθυμητές ρυθμίσεις φίλτρου
        LoadOptions loadOptions = new LoadOptions(LoadFormat.AUTO);
        loadOptions.setLoadFilter(new LoadFilter(LoadDataFilterOptions.ALL & ~LoadDataFilterOptions.VBA));

        System.out.println("Load options configured to exclude VBA macros.");
    }
}
```
**Εξήγηση:** 
Ο `LoadOptions` η κλάση αρχικοποιείται με τη μορφή να έχει οριστεί σε αυτόματη ανίχνευση. `setLoadFilter()` Η μέθοδος καθορίζει ότι όλα τα δεδομένα εκτός από την VBA θα πρέπει να φορτωθούν.

### Δυνατότητα 2: Φόρτωση βιβλίου εργασίας με φιλτραρισμένες μακροεντολές VBA
Τώρα, ας φορτώσουμε ένα βιβλίο εργασίας του Excel χρησιμοποιώντας αυτές τις φιλτραρισμένες επιλογές.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Φόρτωση του βιβλίου εργασίας**

```java
public class ExcludeVbaMacros {
    public static void main(String[] args) {
        String dataDir = "YOUR_DATA_DIRECTORY";
        
        // Ορισμός επιλογών φόρτωσης για την εξαίρεση μακροεντολών VBA
        LoadOptions loadOptions = new LoadOptions(LoadFormat.AUTO);
        loadOptions.setLoadFilter(new LoadFilter(LoadDataFilterOptions.ALL & ~LoadDataFilterOptions.VBA));

        // Φόρτωση του βιβλίου εργασίας με καθορισμένες επιλογές φόρτωσης
        Workbook book = new Workbook(dataDir + "/sampleMacroEnabledWorkbook.xlsm", loadOptions);

        System.out.println("Workbook loaded without VBA macros.");
    }
}
```
**Εξήγηση:** 
Ο `Workbook` Ο κατασκευαστής παίρνει μια διαδρομή αρχείου και `LoadOptions`Αυτή η ρύθμιση διασφαλίζει ότι το βιβλίο εργασίας φορτώνεται χωρίς τα στοιχεία VBA του.

### Δυνατότητα 3: Αποθήκευση βιβλίου εργασίας σε μορφή XLSM
Αφού εξαιρέσετε τις μακροεντολές VBA, αποθηκεύστε το τροποποιημένο βιβλίο εργασίας για να διατηρήσετε τις αλλαγές.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Αποθήκευση του τροποποιημένου βιβλίου εργασίας**

```java
public class ExcludeVbaMacros {
    public static void main(String[] args) {
        String dataDir = "YOUR_DATA_DIRECTORY";
        String outDir = "YOUR_OUTPUT_DIRECTORY";
        
        // Φόρτωση επιλογών για εξαίρεση μακροεντολών VBA
        LoadOptions loadOptions = new LoadOptions(LoadFormat.AUTO);
        loadOptions.setLoadFilter(new LoadFilter(LoadDataFilterOptions.ALL & ~LoadDataFilterOptions.VBA));

        // Φόρτωση του βιβλίου εργασίας
        Workbook book = new Workbook(dataDir + "/sampleMacroEnabledWorkbook.xlsm", loadOptions);

        // Αποθήκευση του βιβλίου εργασίας σε μορφή XLSM χωρίς μακροεντολές VBA
        book.save(outDir + "/OutputSampleMacroEnabledWorkbook.xlsm", SaveFormat.XLSM);

        System.out.println("Workbook saved successfully.");
    }
}
```
**Εξήγηση:** 
Ο `save()` Η μέθοδος γράφει το τροποποιημένο βιβλίο εργασίας στον δίσκο. Χρησιμοποιώντας `SaveFormat.XLSM` διατηρεί τη δομή της με δυνατότητα μακροεντολών μείον τα στοιχεία VBA.
