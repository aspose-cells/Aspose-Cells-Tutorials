## Οδηγός Εφαρμογής

Τώρα, ας εφαρμόσουμε τη λειτουργικότητα για την κατάργηση της προστασίας ενός φύλλου εργασίας του Excel.

### Επισκόπηση της κατάργησης προστασίας ενός φύλλου εργασίας

Αυτή η λειτουργία σάς επιτρέπει να καταργήσετε την προστασία από ένα φύλλο εργασίας που ήταν προηγουμένως ασφαλισμένο. Αυτό είναι χρήσιμο όταν χρειάζεται να κάνετε αλλαγές ή να κοινοποιήσετε δεδομένα χωρίς περιορισμούς.

#### Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας

Αρχικά, δημιουργήστε ένα `Workbook` αντικείμενο και φορτώστε το προστατευμένο αρχείο Excel:

```java
String dataDir = Utils.getSharedDataDir(UnprotectingSimplyProtectedWorksheet.class) + "Worksheets/";
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

#### Βήμα 2: Πρόσβαση στη Συλλογή Φύλλων Εργασίας

Ανακτήστε τη συλλογή φύλλων εργασίας από το βιβλίο εργασίας και επιλέξτε αυτήν που θέλετε να καταργήσετε την προστασία:

```java
WorksheetCollection worksheets = workbook.getWorksheets();
Worksheet worksheet = worksheets.get(0);
```

#### Βήμα 3: Τροποποίηση ρυθμίσεων προστασίας (για Excel 2000 και παλαιότερες εκδόσεις)

Εάν έχετε να κάνετε με παλαιότερες μορφές Excel, προσαρμόστε τις ρυθμίσεις προστασίας:

```java
Protection protection = worksheet.getProtection();
protection.setAllowEditingContent(false);
protection.setAllowEditingObject(false);
protection.setAllowEditingScenario(false);
```

#### Βήμα 4: Κατάργηση προστασίας του φύλλου εργασίας

Αφαιρέστε την προστασία χρησιμοποιώντας το `unprotect()` μέθοδος. Αυτό το βήμα δεν απαιτεί παραμέτρους εάν δεν έχει οριστεί κωδικός πρόσβασης:

```java
worksheet.unprotect();
```

#### Βήμα 5: Αποθήκευση αλλαγών σε νέο αρχείο

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```java
workbook.save(dataDir + "USPWorksheet_out.xls");
```
