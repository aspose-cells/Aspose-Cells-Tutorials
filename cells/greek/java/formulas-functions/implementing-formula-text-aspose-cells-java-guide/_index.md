## Οδηγός Εφαρμογής

### Εξαγωγή κειμένου τύπου χρησιμοποιώντας `FormulaText`

#### Επισκόπηση
Ο `FormulaText` Η συνάρτηση σάς επιτρέπει να ανακτήσετε το κείμενο ενός τύπου μέσα σε ένα κελί του Excel, κάτι που είναι χρήσιμο για σκοπούς ελέγχου ή καταγραφής.

#### Βήμα προς βήμα εφαρμογή
1. **Δημιουργία αντικειμένου βιβλίου εργασίας**
   Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Workbook` τάξη:
   
   ```java
   import com.aspose.cells.Workbook;
   import com.aspose.cells.Worksheet;
   import com.aspose.cells.Cell;

   public class UsingFormulaTextFunction {
       public static void main(String[] args) throws Exception {
           Workbook workbook = new Workbook();
   ```

2. **Πρόσβαση στο Πρώτο Φύλλο Εργασίας**
   Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας:
   
   ```java
   // Αποκτήστε το πρώτο φύλλο εργασίας
   Worksheet worksheet = workbook.getWorksheets().get(0);
   ```

3. **Εισαγωγή τύπου σε κελί**
   Εισαγάγετε έναν τύπο, όπως π.χ. `SUM`, στο κελί A1:
   
   ```java
   // Προσθήκη ενός τύπου SUM στο κελί A1
   Cell cellA1 = worksheet.getCells().get("A1");
   cellA1.setFormula("=Sum(B1:B10)");
   ```

4. **Ανάκτηση κειμένου τύπου χρησιμοποιώντας `FormulaText`**
   Χρησιμοποιήστε το `FormulaText` συνάρτηση για την εξαγωγή και εμφάνιση του κειμένου του τύπου στο κελί A2:
   
   ```java
   // Ανάκτηση και ορισμός του κειμένου του τύπου στο κελί A2
   Cell cellA2 = worksheet.getCells().get("A2");
   cellA2.setFormula("=FormulaText(A1)");

   // Υπολογισμός τύπων βιβλίου εργασίας
   workbook.calculateFormula();

   // Εξαγωγή του κειμένου του τύπου από το A2
   System.out.println(cellA2.getStringValue());
       }
   }
   ```

### Επεξήγηση Παραμέτρων και Μεθόδων
- **`setFormula(String formula)`**: Ορίζει έναν τύπο στο καθορισμένο κελί.
- **`getStringValue()`**: Ανακτά την αναπαράσταση συμβολοσειράς της τιμής του κελιού, χρήσιμη για την επαλήθευση της εξόδου.
