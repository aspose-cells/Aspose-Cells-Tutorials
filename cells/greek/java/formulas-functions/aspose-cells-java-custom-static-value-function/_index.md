## Οδηγός Εφαρμογής

### Επέκταση του AbstractCalculationEngine για προσαρμοσμένες στατικές τιμές
Σε αυτήν την ενότητα, θα δημιουργήσουμε μια προσαρμοσμένη συνάρτηση που επιστρέφει στατικές τιμές. Αυτό είναι χρήσιμο όταν χρειάζεστε προκαθορισμένες απαντήσεις κατά τη διάρκεια των υπολογισμών.

#### Βήμα 1: Δημιουργήστε μια κλάση προσαρμοσμένης συνάρτησης
Αρχικά, δημιουργήστε μια νέα κλάση που επεκτείνεται `AbstractCalculationEngine`:
```java
import com.aspose.cells.AbstractCalculationEngine;
import com.aspose.cells.CalculationData;
import com.aspose.cells.DateTime;

public class CustomFunctionStaticValue extends AbstractCalculationEngine {
    @Override
    public void calculate(CalculationData calculationData) {
        // Ορισμός στατικών υπολογισμένων τιμών για τα δεδομένα κελιά
        calculationData.setCalculatedValue(new Object[][] { 
            new Object[] { new DateTime(2015, 6, 12, 10, 6, 30), 2 },
            new Object[] { 3.0, "Test" }
        });
    }
}
```
**Εξήγηση:**
- **`calculate(CalculationData calculationData)`:** Αυτή η μέθοδος παρακάμπτεται για να οριστεί ο τρόπος με τον οποίο η προσαρμοσμένη συνάρτηση υπολογίζει τις τιμές.
- **Στατικές τιμές:** Χρήση `setCalculatedValue(Object[][])` για να ορίσετε προκαθορισμένα αποτελέσματα για συγκεκριμένα κελιά.

#### Βήμα 2: Καταχωρίστε την προσαρμοσμένη λειτουργία σας
Για να κάνετε διαθέσιμη τη νέα σας συνάρτηση, καταχωρίστε την σε ένα βιβλίο εργασίας:
```java
import com.aspose.cells.*;

public class Main {
    public static void main(String[] args) throws Exception {
        Workbook workbook = new Workbook();
        
        // Πρόσβαση στο μητρώο της μηχανής υπολογισμών
        CalculationEngineManager manager = workbook.getSettings().getCalculationEngineManager();
        manager.addCustomFunction("MyStaticFunc", new CustomFunctionStaticValue());
        
        // Χρήση της προσαρμοσμένης συνάρτησης σε έναν τύπο
        Worksheet worksheet = workbook.getWorksheets().get(0);
        worksheet.getCells().get("A1").setFormula("=MyStaticFunc()");
        workbook.calculateFormula();

        // Αποθηκεύστε το αποτέλεσμα για να επαληθεύσετε την εφαρμογή
        workbook.save("output.xlsx");
    }
}
```
**Εξήγηση:**
- **Εγγραφή προσαρμοσμένης συνάρτησης:** Χρήση `addCustomFunction` για να καταχωρήσετε την προσαρμοσμένη μηχανή υπολογισμών σας.
- **Χρήση σε έναν τύπο:** Εφαρμόστε το ως τύπο μέσα σε οποιοδήποτε κελί, όπως `"=MyStaticFunc()"`.

Ακολουθούν ορισμένες περιπτώσεις χρήσης στον πραγματικό κόσμο όπου οι προσαρμοσμένες στατικές τιμές θα μπορούσαν να είναι επωφελείς:
1. **Αυτοματοποιημένη αναφορά:** Χρησιμοποιήστε στατικές τιμές σε αναφορές που χρειάζονται συνεπή μορφοποίηση ή προκαθορισμένες μετρήσεις.
2. **Έλεγχοι επικύρωσης δεδομένων:** Υλοποιήστε ελέγχους με προκαθορισμένες απαντήσεις για την επικύρωση της ακεραιότητας των δεδομένων κατά την ανάλυση.
3. **Εκπαιδευτικά Εργαλεία:** Δημιουργήστε ενότητες μάθησης με σταθερές απαντήσεις για ασκήσεις και κουίζ.
