## Οδηγός Εφαρμογής
### Υλοποίηση προσαρμοσμένων ρυθμίσεων παγκοσμιοποίησης
Αυτή η λειτουργία σάς επιτρέπει να προσαρμόσετε τα ονόματα συναρτήσεων σε τύπους με βάση τις ρυθμίσεις τοπικής προσαρμογής.
#### Βήμα 1: Ορισμός μιας προσαρμοσμένης κλάσης που επεκτείνεται `GlobalizationSettings`
```java
import com.aspose.cells.*;

class GS extends GlobalizationSettings {
    // Μέθοδος για την απόκτηση ενός μεταφρασμένου ονόματος για τυπικές συναρτήσεις.
    public String getLocalFunctionName(String standardName) {
        if (standardName.equals("SUM")) { 
            return "UserFormulaLocal_SUM";
        }
        if (standardName.equals("AVERAGE")) { 
            return "UserFormulaLocal_AVERAGE";
        }
        return standardName;  // Επιστροφή αρχικού ονόματος για άλλες συναρτήσεις
    }
}
```
**Εξήγηση**: Αυτή η κλάση παρακάμπτει `getLocalFunctionName` για να επιστρέψει μεταφρασμένα ονόματα συναρτήσεων για `SUM` και `AVERAGE`Επιστρέφει το αρχικό όνομα για συναρτήσεις που δεν έχουν παρακαμφθεί ρητά.
### Επίδειξη δημιουργίας βιβλίου εργασίας και τοπικής προσαρμογής τύπου
Αυτή η ενότητα παρουσιάζει τον τρόπο ρύθμισης ενός βιβλίου εργασίας με προσαρμοσμένες ρυθμίσεις παγκοσμιοποίησης.
#### Βήμα 2: Ρύθμιση του βιβλίου εργασίας και εφαρμογή ρυθμίσεων παγκοσμιοποίησης
```java
import com.aspose.cells.*;

public class WorkbookFormulaLocalization {
    public void demonstrate() throws Exception {
        // Δημιουργία νέας παρουσίας βιβλίου εργασίας
        Workbook wb = new Workbook();
        
        // Ορίστε τις προσαρμοσμένες ρυθμίσεις παγκοσμιοποίησης στο βιβλίο εργασίας
        wb.getSettings().setGlobalizationSettings(new GS());
        
        // Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
        Worksheet ws = wb.getWorksheets().get(0);
        
        // Πρόσβαση σε ένα συγκεκριμένο κελί όπου θα οριστούν οι τύποι
        Cell cell = ws.getCells().get("C4");
        
        // Ορισμός ενός τύπου SUM και ανάκτηση της μεταφρασμένης έκδοσής του
        cell.setFormula("SUM(A1:A2)");
        String sumLocal = cell.getFormulaLocal();
        
        // Ορίστε έναν τύπο AVERAGE και ανακτήστε την μεταφρασμένη έκδοσή του
        cell.setFormula("=AVERAGE(B1:B2, B5)");
        String averageLocal = cell.getFormulaLocal();
    }
}
```
**Εξήγηση**: Ο κώδικας αρχικοποιεί ένα βιβλίο εργασίας, ορίζει την προσαρμοσμένη `GlobalizationSettings`και εφαρμόζει τύπους για να δείξει τον εντοπισμό.
