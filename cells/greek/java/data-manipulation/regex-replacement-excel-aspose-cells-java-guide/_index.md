## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία υλοποίησης σε διαχειρίσιμα μέρη:

### Αρχικοποίηση βιβλίου εργασίας και εκτέλεση αντικατάστασης Regex

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να φορτώσετε ένα βιβλίο εργασίας του Excel, να εκτελέσετε μια αντικατάσταση κειμένου που βασίζεται σε regex και να αποθηκεύσετε τις αλλαγές.

#### Αρχικοποίηση βιβλίου εργασίας
Ξεκινήστε φορτώνοντας το αρχείο Excel:
```java
String dataDir = "YOUR_DATA_DIRECTORY"; // Ενημέρωση με τη διαδρομή του καταλόγου προέλευσης

// Φόρτωση του βιβλίου εργασίας από τον καθορισμένο κατάλογο
Workbook workbook = new Workbook(dataDir + "/SampleRegexReplace.xlsx");
```
**Γιατί;** Η φόρτωση του βιβλίου εργασίας είναι απαραίτητη για την πρόσβαση στο περιεχόμενό του για τροποποίηση.

#### Ρύθμιση παραμέτρων επιλογών αντικατάστασης
Ρύθμιση επιλογών για αντικατάσταση κειμένου:
```java
ReplaceOptions replace = new ReplaceOptions();
replace.setCaseSensitive(false);  // Η αντικατάσταση δεν εξαρτάται από την ευαισθησία πεζών-κεφαλαίων
replace.setMatchEntireCellContents(false);  // Επιτρέπει μερικές αντιστοιχίσεις εντός των περιεχομένων των κελιών
replace.setRegexKey(true);  // Ενεργοποιεί την αντιστοίχιση μοτίβου regex
```
**Γιατί;** Η ρύθμιση αυτών των επιλογών διασφαλίζει την ακριβή αντικατάσταση κειμένου σύμφωνα με τις απαιτήσεις σας.

#### Εκτέλεση αντικατάστασης βάσει Regex
Εκτελέστε την αντικατάσταση κειμένου:
```java
// Αντικαταστήστε όλες τις εμφανίσεις του "\\bKIM\\b" με "^^^TIM^^^"
workbook.replace("\\bKIM\\b", "^^^TIM^^^", replace);
```
**Γιατί;** Αυτό το βήμα χρησιμοποιεί regex για να βρει και να αντικαταστήσει συγκεκριμένα μοτίβα στο βιβλίο εργασίας σας.

#### Αποθήκευση του τροποποιημένου βιβλίου εργασίας
Τέλος, αποθηκεύστε τις αλλαγές σας:
```java
String outDir = "YOUR_OUTPUT_DIRECTORY";  // Ενημέρωση με τη διαδρομή του καταλόγου εξόδου σας

// Αποθήκευση του τροποποιημένου βιβλίου εργασίας σε νέο αρχείο
workbook.save(outDir + "/RegexReplace_out.xlsx");
```
**Γιατί;** Η αποθήκευση διασφαλίζει ότι όλες οι τροποποιήσεις αποθηκεύονται και μπορούν να ελεγχθούν ή να κοινοποιηθούν.
