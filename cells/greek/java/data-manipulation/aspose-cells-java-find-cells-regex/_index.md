### Οδηγός Εφαρμογής

#### Εύρεση κελιών με κανονικές εκφράσεις

Αυτή η ενότητα σας καθοδηγεί στην εφαρμογή αναζητήσεων κελιών που βασίζονται σε regex σε ένα φύλλο Excel.

**Βήμα 1: Πρόσβαση στο Φύλλο Εργασίας**

Βρείτε μια αναφορά στο φύλλο εργασίας όπου θέλετε να εκτελέσετε την αναζήτηση:

```java
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Worksheet worksheet = workbook.getWorksheets().get(0);
```

**Βήμα 2: Ρύθμιση επιλογών εύρεσης**

Διαμορφώστε το `FindOptions` για να χρησιμοποιήσετε regex για αναζητήσεις που βασίζονται σε μοτίβα:

```java
// Δημιουργήστε ένα στιγμιότυπο FindOptions και ορίστε το RegexKey σε true
FindOptions opt = new FindOptions();
opt.setRegexKey(true); // Ενεργοποιεί την αναζήτηση με κανονικές εκφράσεις
opt.setLookAtType(LookAtType.ENTIRE_CONTENT);
```

**Βήμα 3: Εκτελέστε την αναζήτηση**

Χρησιμοποιήστε το `find` μέθοδος με το μοτίβο regex σας:

```java
// Εύρεση κελιών που ταιριάζουν με το καθορισμένο regex
Cells cells = worksheet.getCells();
cells.find("abc[\\s]*$", null, opt); // Αναζητά τον όρο 'abc' ακολουθούμενο από τυχόν χαρακτήρες κενού διαστήματος στο τέλος του περιεχομένου ενός κελιού
```

**Εξήγηση:**
- `setRegexKey(true)`Ενεργοποιεί την αναζήτηση με κανονικές εκφράσεις.
- `LookAtType.ENTIRE_CONTENT`: Διασφαλίζει ότι κατά την αναζήτηση λαμβάνεται υπόψη ολόκληρο το περιεχόμενο του κελιού, όχι μόνο μέρος αυτού.
