## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην πρόσβαση και την εκτύπωση τύπων PowerQuery χρησιμοποιώντας το Aspose.Cells για Java.

### Πρόσβαση σε τύπους PowerQuery

#### Επισκόπηση
Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο ανάγνωσης των τύπων PowerQuery που είναι ενσωματωμένοι στο mashup δεδομένων ενός βιβλίου εργασίας του Excel.

#### Υλοποίηση κώδικα
1. **Φόρτωση του βιβλίου εργασίας**
   Ξεκινήστε φορτώνοντας το αρχείο Excel σε ένα `Workbook` αντικείμενο:

   ```java
   Workbook workbook = new Workbook(dataDir + "/ODataSample.xlsx");
   ```

2. **Συλλογή τύπων PowerQuery της Access**
   Χρησιμοποιήστε το `getDataMashup()` μέθοδος για την πρόσβαση στους τύπους:

   ```java
   PowerQueryFormulaCollection PQFcoll = workbook.getDataMashup().getPowerQueryFormulas();
   ```

3. **Επανάληψη πάνω από τύπους**
   Επαναλάβετε κάθε τύπο και εκτυπώστε τις λεπτομέρειες του:

   ```java
   for (Object obj : PQFcoll) {
       PowerQueryFormula PQF = (PowerQueryFormula)obj;
       System.out.println("Connection Name: " + PQF.getName());
       
       PowerQueryFormulaItemCollection PQFIcoll = PQF.getPowerQueryFormulaItems();
       
       for (Object obj2 : PQFIcoll) {
           PowerQueryFormulaItem PQFI = (PowerQueryFormulaItem)obj2;
           System.out.println("Name: " + PQFI.getName());
           System.out.println("Value: " + PQFI.getValue());
       }
   }
   ```

### Κατανόηση Παραμέτρων και Μεθόδων
- **`getName()`**: Ανακτά το όνομα της σύνδεσης ή του στοιχείου τύπου.
- **`getValue()`**Επιστρέφει την τιμή που σχετίζεται με ένα στοιχείο τύπου PowerQuery.
