## Οδηγός Εφαρμογής
### Επέκταση της Κλάσης Ατόμων
Σε αυτήν την ενότητα, θα επεκτείνουμε το `Person` κλάση για να δημιουργήσετε ένα `Individual` κλάση που διαχειρίζεται πρόσθετα χαρακτηριστικά και συμπεριφορές.
#### Επισκόπηση:
Ο `Individual` η τάξη επεκτείνεται `Person`, παρουσιάζοντας την κληρονομικότητα σε Java για τη βελτίωση της λειτουργικότητας προσθέτοντας συγκεκριμένα χαρακτηριστικά, όπως πληροφορίες για τον/την σύζυγο.
##### Βήμα 1: Ορίστε την Ατομική Κλάση
Ξεκινήστε με τη δημιουργία του `Individual` κλάση, συμπεριλαμβανομένων ιδιωτικών μελών και κατασκευαστών για την αρχικοποίηση αντικειμένων:
```java
import java.util.ArrayList;
class Person {
    // Απλοποιημένη έκδοση μιας βασικής κλάσης όπως το Aspose.Person
    protected String name;
    protected int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
// Άτομο που επεκτείνει την ατομική τάξη
class Individual extends Person {
    private Person m_Wife; // Ιδιωτικό μέλος για πληροφορίες συζύγου

    // Κατασκευαστής για την κλάση Individual
    public Individual(String name, int age, Person wife) {
        super(name, age); // Κλήση κατασκευαστή υπερκλάσης
        this.m_Wife = wife; // Αρχικοποίηση m_Wife με την παρεχόμενη τιμή
    }

    // Μέθοδος Getter για m_Wife
    public Person getWife() {
        return m_Wife;
    }
}
```
**Εξήγηση**: 
- **Κατασκευαστής υπερκλάσης**: `super(name, age)` αρχικοποιεί την υπερκλάση `Person` γνωρίσματα.
- **Ιδιωτικό Μέλος**: `m_Wife` αποθηκεύει πληροφορίες συζύγου, παρουσιάζοντας την ενθυλάκωση.
##### Βήμα 2: Χρησιμοποιήστε την ατομική τάξη
Δημιουργήστε στιγμιότυπα της νέας σας κλάσης και αξιοποιήστε τη λειτουργικότητά της:
```java
public class Main {
    public static void main(String[] args) {
        Person wife = new Person("Jane", 30);
        Individual person = new Individual("John", 35, wife);

        System.out.println("Person's Wife: " + person.getWife().name); // Έξοδος: Τζέιν
    }
}
```
**Εξήγηση**: 
- Αυτό καταδεικνύει τη δημιουργία ενός `Person` αντίρρηση να εκπροσωπήσει τον/την σύζυγο και να το μεταβιβάσει κατά την κατασκευή ενός `Individual`.
