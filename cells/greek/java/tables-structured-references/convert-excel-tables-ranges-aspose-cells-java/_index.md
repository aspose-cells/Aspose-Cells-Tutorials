## Οδηγός Εφαρμογής

Ας χωρίσουμε τη διαδικασία σε διαχειρίσιμα βήματα:

### 1. Αρχικοποίηση και φόρτωση βιβλίου εργασίας
#### Επισκόπηση
Ξεκινήστε αρχικοποιώντας ένα `Workbook` αντικείμενο και φόρτωση ενός υπάρχοντος αρχείου Excel.

**Απόσπασμα κώδικα:**
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου δεδομένων σας
Workbook wb = new Workbook(dataDir + "book1.xlsx");
```

**Εξήγηση:** 
- `Workbook` είναι η κύρια κλάση για τη διαχείριση αρχείων Excel.
- Ο κατασκευαστής φορτώνει ένα υπάρχον αρχείο που καθορίζεται από τη διαδρομή του.

### 2. Μετατροπή αντικειμένου πίνακα/λίστας σε εύρος
#### Επισκόπηση
Μετατρέψτε ένα αντικείμενο πίνακα ή λίστας στο φύλλο εργασίας σας σε κανονική περιοχή για πιο προηγμένους χειρισμούς.

**Απόσπασμα κώδικα:**
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου δεδομένων σας
Workbook wb = new Workbook(dataDir + "book1.xlsx");
wb.getWorksheets().get(0).getListObjects().get(0).convertToRange();
```

**Εξήγηση:**
- Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και στα αντικείμενα λίστας του.
- Χρήση `convertToRange()` για να αλλάξετε τον πίνακα σε ένα τυπικό εύρος δεδομένων.

### 3. Αποθήκευση του τροποποιημένου βιβλίου εργασίας
#### Επισκόπηση
Μετά τις τροποποιήσεις, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε τις αλλαγές.

**Απόσπασμα κώδικα:**
```java
import com.aspose.cells.Workbook;

String dataDir = "YOUR_DATA_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου δεδομένων σας
String outDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εξόδου σας
Workbook wb = new Workbook(dataDir + "book1.xlsx");
wb.getWorksheets().get(0).getListObjects().get(0).convertToRange();
wb.save(outDir + "ConvertTableToRange_out.xlsx");
```

**Εξήγηση:**
- Κλήση `save()` για να γράψετε ξανά το βιβλίο εργασίας στον δίσκο.
- Βεβαιωθείτε ότι ο κατάλογος εξόδου έχει οριστεί σωστά.
