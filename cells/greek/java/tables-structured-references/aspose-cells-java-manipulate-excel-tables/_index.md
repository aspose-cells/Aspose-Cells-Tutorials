## Οδηγός Εφαρμογής

Τώρα που το περιβάλλον σας έχει ρυθμιστεί, ας αποκτήσουμε πρόσβαση και ας χειριστούμε έναν πίνακα Excel μέσω ενός κελιού.

### Πρόσβαση σε έναν πίνακα Excel από ένα κελί

#### 1. Φόρτωση του Βιβλίου Εργασίας

Αρχικά, δημιουργήστε ένα `Workbook` παράδειγμα φορτώνοντας ένα υπάρχον αρχείο Excel:

```java
String dataDir = "path/to/your/directory/";
Workbook workbook = new Workbook(dataDir + "ATFromCell.xlsx");
```

#### 2. Πρόσβαση στο Φύλλο Εργασίας και στο Κελί

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και στο συγκεκριμένο κελί μέσα στον πίνακα που σας ενδιαφέρει:

```java
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("D5");
```

#### 3. Χειρισμός δεδομένων πίνακα

Χρησιμοποιήστε το `TableObject` για να τροποποιήσετε δεδομένα χρησιμοποιώντας μετατοπίσεις γραμμών και στηλών:

```java
// Τοποθετήστε μια τιμή μέσα στο κελί D5
cell.putValue("D5 Data");

// Πρόσβαση στον πίνακα από αυτό το κελί
ListObject table = cell.getTable();

// Προσθέστε κάποια τιμή σε καθορισμένη θέση μετατόπισης
table.putCellValue(1, 1, "Offset [2,2]");
```

#### 4. Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα αρχείο Excel:

```java
workbook.save(dataDir + "ATFromCell_out.xlsx");
```
