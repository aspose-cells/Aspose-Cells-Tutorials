# Εξαγωγή γραφήματος σε PNG και προσθήκη σειράς δεδομένων για συνδυαστικό γράφημα

Σε αυτό το tutorial θα **προσθέσετε σειρά δεδομένων** σε ένα βιβλίο εργασίας Excel, θα **συνδυάσετε στοιχεία γραφήματος γραμμής και στήλης**, και θα μάθετε πώς να **εξάγετε το γράφημα σε PNG** χρησιμοποιώντας το Aspose.Cells for Java. Θα περάσουμε από κάθε βήμα — από τη ρύθμιση του βιβλίου εργασίας, την προσθήκη του γραφήματος σε ένα φύλλο εργασίας, την προσαρμογή του υπομνήματος, έως το **αποθήκευση του βιβλίου εργασίας ως xlsx** και τη δημιουργία μιας εικόνας PNG του γραφήματος. Στο τέλος, θα έχετε ένα έτοιμο συνδυαστικό γράφημα που μπορείτε να ενσωματώσετε σε αναφορές ή πίνακες ελέγχου.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη δημιουργεί συνδυαστικά γραφήματα;** Aspose.Cells for Java  
- **Πώς προσθέτω σειρά δεδομένων;** Χρησιμοποιήστε `chart.getNSeries().add(...)`  
- **Πώς μπορώ να εξάγω το γράφημα σε png;** Καλέστε `chart.toImage("file.png", ImageFormat.getPng())`  
- **Σε ποια μορφή αρχείου μπορώ να αποθηκεύσω το βιβλίο εργασίας;** Standard `.xlsx` (save workbook as xlsx)  
- **Χρειάζομαι άδεια για παραγωγή;** A valid Aspose.Cells license is required  

## Τι είναι η **εξαγωγή γραφήματος σε PNG** στο Aspose.Cells;
Η εξαγωγή ενός γραφήματος σε PNG δημιουργεί μια ραστερ εικόνα του γραφήματος Excel που μπορεί να εμφανιστεί σε ιστοσελίδες, αναφορές ή email χωρίς να απαιτείται η εφαρμογή Excel.

## Γιατί να δημιουργήσετε ένα **συνδυαστικό γράφημα γραμμής-στήλης**;
Ένα συνδυαστικό γράφημα σας επιτρέπει να εμφανίσετε διαφορετικά σύνολα δεδομένων με διαφορετικές οπτικές αναπαραστάσεις (π.χ., μια σειρά γραμμής πάνω σε μια σειρά στήλης) σε μία ενιαία προβολή. Είναι ιδανικό για σύγκριση τάσεων με συνολικά ποσά, ανάδειξη συσχετίσεων ή παροχή πιο πλούσιων πληροφοριών σε συμπαγή μορφή.

## Προαπαιτούμενα
- Java Development Kit (JDK) 8 ή νεότερο  
- Βιβλιοθήκη Aspose.Cells for Java (λήψη από τον παρακάτω σύνδεσμο)  
- Βασική εξοικείωση με τη σύνταξη Java και τις έννοιες του Excel  

## Έναρξη

Αρχικά, κατεβάστε τη βιβλιοθήκη Aspose.Cells for Java από την επίσημη ιστοσελίδα:

[Download Aspose.Cells for Java](https://releases.aspose.com/cells/java/)

Μόλις το JAR προστεθεί στο classpath του έργου σας, μπορείτε να αρχίσετε να δημιουργείτε το γράφημα.

### Βήμα 1: Εισαγωγή κλάσεων Aspose.Cells
```java
import com.aspose.cells.*;
```

### Βήμα 2: Δημιουργία νέου βιβλίου εργασίας
```java
Workbook workbook = new Workbook();
```

### Βήμα 3: Πρόσβαση στο πρώτο φύλλο εργασίας
```java
Worksheet worksheet = workbook.getWorksheets().get(0);
```

### Βήμα 4: Προσθήκη αντικειμένου συνδυαστικού γραφήματος στο φύλλο εργασίας  
Θα ξεκινήσουμε με ένα γράφημα γραμμής και αργότερα θα προσθέσουμε μια σειρά στήλης για να επιτύχουμε το εφέ **συνδυαστικού γραφήματος γραμμής-στήλης**.
```java
int chartIndex = worksheet.getCharts().add(ChartType.LINE, 0, 0, 20, 10);
Chart chart = worksheet.getCharts().get(chartIndex);
```

## Προσθήκη Δεδομένων στο Γράφημα

Τώρα που υπάρχει το κοντέινερ του γραφήματος, πρέπει να το τροφοδοτήσουμε με δεδομένα.

### Βήμα 5: Ορισμός περιοχών δεδομένων και **προσθήκη σειράς δεδομένων**
```java
Cells cells = worksheet.getCells();
chart.getNSeries().add("A1:A5", true);
chart.getNSeries().add("B1:B5", true);
```
> **Συμβουλή:** Η πρώτη παράμετρος (`"A1:A5"`) είναι η περιοχή για την πρώτη σειρά, και η δεύτερη (`"B1:B5"`) δημιουργεί μια δεύτερη σειρά που θα συνδυαστεί με την πρώτη.

### Βήμα 6: Ορισμός δεδομένων κατηγορίας (άξονας X)
```java
chart.getNSeries().setCategoryData("C1:C5");
```

## Προσαρμογή του Γραφήματος

Ένα καλό γράφημα λέει μια ιστορία. Ας του δώσουμε τίτλους, ετικέτες αξόνων και ένα σαφές υπόμνημα.

### Βήμα 7: **Ορισμός ετικετών αξόνων γραφήματος** και τίτλου
```java
chart.getTitle().setText("Combined Chart Example");
chart.getCategoryAxis().getTitle().setText("Categories");
chart.getValueAxis().getTitle().setText("Values");
```

### Βήμα 8: **Προσθήκη υπομνήματος γραφήματος** και προσαρμογή της θέσης του
```java
chart.getLegend().setPosition(LegendPositionType.BOTTOM);
chart.getLegend().setOverlay(true);
```

## Αποθήκευση και Εξαγωγή του Γραφήματος

Μετά την προσαρμογή, θα θέλετε να **αποθηκεύσετε το βιβλίο εργασίας ως xlsx** και επίσης να δημιουργήσετε μια εικόνα.

### Βήμα 9: Αποθήκευση του βιβλίου εργασίας ως αρχείο Excel (xlsx)
```java
workbook.save("CombinedChart.xlsx");
```

### Βήμα 10: **Εξαγωγή γραφήματος σε PNG**
```java
chart.toImage("CombinedChart.png", ImageFormat.getPng());
```
> Η μέθοδος `chart.toImage` **δημιουργεί εικόνες γραφήματος Excel** που μπορούν να χρησιμοποιηθούν σε ιστοσελίδες, αναφορές ή email.

## Συχνά Προβλήματα & Επίλυση

| Πρόβλημα | Λύση |
|-------|----------|
| **Δεν εμφανίζονται δεδομένα** | Επαληθεύστε ότι οι περιοχές κελιών (`A1:A5`, `B1:B5`, `C1:C5`) περιέχουν πραγματικά δεδομένα πριν δημιουργήσετε το γράφημα. |
| **Το υπόμνημα επικαλύπτει το γράφημα** | Ορίστε `chart.getLegend().setOverlay(false)` ή μετακινήστε το υπόμνημα σε διαφορετική θέση (π.χ., `RIGHT`). |
| **Το αρχείο εικόνας είναι κενό** | Βεβαιωθείτε ότι το γράφημα έχει τουλάχιστον μία σειρά και ότι η `chart.toImage` καλείται μετά από όλες τις προσαρμογές. |
| **Η αποθήκευση προκαλεί εξαίρεση** | Ελέγξτε ότι έχετε δικαιώματα εγγραφής στον προορισμό και ότι το αρχείο δεν είναι ανοιχτό στο Excel. |
