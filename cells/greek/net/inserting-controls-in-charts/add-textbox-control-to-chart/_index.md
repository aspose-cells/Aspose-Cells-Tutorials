# Προσθήκη στοιχείου ελέγχου πλαισίου κειμένου σε γράφημα

## Εισαγωγή

Η δημιουργία δυναμικών και οπτικά ελκυστικών γραφημάτων στο Excel είναι ένας φανταστικός τρόπος για την αποτελεσματική αναπαράσταση δεδομένων. Μια έξυπνη λειτουργία που μπορείτε να χρησιμοποιήσετε είναι η προσθήκη ενός TextBox σε ένα γράφημα. Με το Aspose.Cells για .NET, αυτή η εργασία γίνεται εύκολη και διασκεδαστική! Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε βήμα προς βήμα στη διαδικασία ενσωμάτωσης ενός TextBox στο γράφημά σας. Είτε είστε έμπειρος προγραμματιστής είτε μόλις ξεκινάτε, αυτό το σεμινάριο θα σας δώσει όλα τα εργαλεία που χρειάζεστε για να βελτιώσετε τα γραφήματά σας στο Excel. Λοιπόν, είστε έτοιμοι να ξεκινήσετε;

## Προαπαιτούμενα

Πριν ξεκινήσουμε τον προγραμματισμό, υπάρχουν μερικά πράγματα που πρέπει να έχετε υπόψη σας:

- Βασική Κατανόηση της C#: Μια βασική κατανόηση του προγραμματισμού C# θα είναι χρήσιμη. Μην ανησυχείτε. Δεν χρειάζεται να είστε ειδικός, απλώς να μπορείτε να πλοηγηθείτε άνετα στη σύνταξη.
- Εγκατεστημένη βιβλιοθήκη Aspose.Cells: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Cells για .NET. Μπορείτε να την κατεβάσετε από [εδώ](https://releases.aspose.com/cells/net/) αν δεν το έχετε κάνει ήδη.
- Visual Studio: Η εξοικείωση με το Visual Studio ή οποιοδήποτε IDE που προτιμάτε να χρησιμοποιείτε για το .NET framework είναι απαραίτητη.
- Ένα υπάρχον αρχείο Excel: Για αυτό το παράδειγμα, θα εργαστούμε με ένα υπάρχον αρχείο Excel με το όνομα "sampleAddingTextBoxControlInChart.xls". Μπορείτε να δημιουργήσετε ένα ή να κατεβάσετε ένα δείγμα.

Τώρα που έχουμε όλα έτοιμα, ας περάσουμε στο κομμάτι του προγραμματισμού!

## Εισαγωγή πακέτων

Πρώτα απ 'όλα, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων Aspose.Cells στο έργο μας C#. Μπορείτε να το κάνετε αυτό εύκολα συμπεριλαμβάνοντας τις ακόλουθες γραμμές στην αρχή του αρχείου κώδικά σας:

```csharp
using System;
using System.IO;

using Aspose.Cells;
using System.Drawing;
```

## Βήμα 1: Ορίστε τους καταλόγους προέλευσης και εξόδου

Πριν ξεκινήσουμε να εργαζόμαστε με το αρχείο Excel, είναι σημαντικό να ορίσουμε πού βρίσκεται το αρχείο εισόδου και πού θέλετε να αποθηκεύσετε το αρχείο εξόδου. Αυτό βοηθά στην οργάνωση του έργου σας.

```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory";

// Κατάλογος εξόδου
string outputDir = "Your Output Directory";
```
Αντικαθιστώ `"Your Document Directory"` και `"Your Output Directory"` με τις πραγματικές διαδρομές στο σύστημά σας.

## Βήμα 2: Ανοίξτε το υπάρχον αρχείο Excel

Στη συνέχεια, πρέπει να ανοίξουμε το αρχείο Excel που περιέχει το γράφημα που θέλουμε να τροποποιήσουμε. Αυτό θα μας επιτρέψει να ανακτήσουμε το γράφημα και να κάνουμε αλλαγές.

```csharp
// Ανοίξτε το υπάρχον αρχείο.
Workbook workbook = new Workbook(sourceDir + "sampleAddingTextBoxControlInChart.xls");
```
Αυτή η γραμμή αρχικοποιεί ένα νέο αντικείμενο Βιβλίου Εργασίας με το καθορισμένο αρχείο μας.

## Βήμα 3: Πρόσβαση στο Διάγραμμα στο Φύλλο Εργασίας

Δεδομένου ότι τα γραφήματα στο Excel αποθηκεύονται μέσα σε ένα φύλλο εργασίας, πρέπει πρώτα να αποκτήσουμε πρόσβαση στο φύλλο εργασίας και στη συνέχεια να λάβουμε το επιθυμητό γράφημα. Για αυτό το παράδειγμα, θα αποκτήσουμε πρόσβαση στο πρώτο γράφημα στο πρώτο φύλλο εργασίας.

```csharp
// Αποκτήστε το διάγραμμα σχεδιαστή στο πρώτο φύλλο.
Worksheet sheet = workbook.Worksheets[0];
Aspose.Cells.Charts.Chart chart = sheet.Charts[0];
```
Αλλάζοντας την τιμή του δείκτη, μπορείτε να επιλέξετε διαφορετικά φύλλα εργασίας ή γραφήματα εάν το αρχείο σας έχει περισσότερα.

## Βήμα 4: Προσθήκη νέου πλαισίου κειμένου στο γράφημα

Τώρα, είμαστε έτοιμοι να προσθέσουμε το TextBox μας. Θα καθορίσουμε τη θέση και το μέγεθός του κατά τη δημιουργία του.

```csharp
// Προσθέστε ένα νέο πλαίσιο κειμένου στο γράφημα.
Aspose.Cells.Drawing.TextBox textbox0 = chart.Shapes.AddTextBoxInChart(400, 1100, 350, 2550);
```
Σε αυτήν την εντολή, οι παράμετροι καθορίζουν τη θέση (x, y) και το μέγεθος (πλάτος, ύψος) του TextBox στο γράφημα. Προσαρμόστε αυτές τις τιμές με βάση τις συγκεκριμένες ανάγκες διάταξης.

## Βήμα 5: Ορίστε το κείμενο για το πλαίσιο κειμένου

Μόλις τοποθετηθεί το πλαίσιο κειμένου, ήρθε η ώρα να το γεμίσετε με περιεχόμενο. Μπορείτε να προσθέσετε οποιοδήποτε κείμενο κρίνετε απαραίτητο για το γράφημά σας.

```csharp
// Συμπληρώστε το κείμενο.
textbox0.Text = "Sales By Region";
```
Μη διστάσετε να αντικαταστήσετε την επιλογή "Πωλήσεις ανά περιοχή" με οποιοδήποτε κείμενο σχετικό με τα δεδομένα σας.

## Βήμα 6: Προσαρμογή ιδιοτήτων πλαισίου κειμένου

Τώρα, ας κάνουμε το TextBox μας να φαίνεται όμορφο! Μπορείτε να προσαρμόσετε διάφορες ιδιότητες όπως το χρώμα, το μέγεθος και το στυλ της γραμματοσειράς.

```csharp
// Ορίστε το χρώμα της γραμματοσειράς.
textbox0.Font.Color = Color.Maroon; // Αλλάξτε στο επιθυμητό χρώμα

// Ορίστε τη γραμματοσειρά σε έντονη γραφή.
textbox0.Font.IsBold = true;

// Ορίστε το μέγεθος της γραμματοσειράς.
textbox0.Font.Size = 14;

// Ορισμός της ιδιότητας γραμματοσειράς σε πλάγια γραφή.
textbox0.Font.IsItalic = true;
```

Κάθε μία από αυτές τις γραμμές τροποποιεί την εμφάνιση του κειμένου μέσα στο TextBox σας, βελτιώνοντας την ορατότητα και την ελκυστικότητα.

## Βήμα 7: Μορφοποίηση της εμφάνισης του πλαισίου κειμένου

Είναι επίσης απαραίτητο να μορφοποιήσετε το φόντο και το περίγραμμα του TextBox. Αυτό το κάνει να ξεχωρίζει στο γράφημα.

```csharp
// Λάβετε τη μορφή συμπλήρωσης του πλαισίου κειμένου.
Aspose.Cells.Drawing.FillFormat fillformat = textbox0.Fill;

// Λάβετε τον τύπο μορφής γραμμής του πλαισίου κειμένου.
Aspose.Cells.Drawing.LineFormat lineformat = textbox0.Line;

// Ορίστε το πάχος της γραμμής.
lineformat.Weight = 2;

// Ορίστε το στυλ παύλας σε συμπαγές.
lineformat.DashStyle = Aspose.Cells.Drawing.MsoLineDashStyle.Solid;
```

Αυτές οι επιλογές σάς επιτρέπουν να ορίσετε το γέμισμα φόντου του TextBox και να προσαρμόσετε το περίγραμμά του.

## Βήμα 8: Αποθήκευση του τροποποιημένου αρχείου Excel

Το τελευταίο βήμα είναι να αποθηκεύσετε τις αλλαγές που έχετε κάνει σε ένα νέο αρχείο Excel. Αυτό θα διασφαλίσει ότι το αρχικό σας αρχείο θα παραμείνει ανέπαφο.

```csharp
// Αποθηκεύστε το αρχείο excel.
workbook.Save(outputDir + "outputAddingTextBoxControlInChart.xls");
```
Αντικαθιστώ `"outputAddingTextBoxControlInChart.xls"` με όποιο όνομα αρχείου προτιμάτε.
