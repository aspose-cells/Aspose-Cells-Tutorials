# Εξαγωγή ημερομηνίας από το Excel – Πλήρης Προγραμματιστική Επίδειξη

Κάποτε χρειάστηκε να **εξάγετε ημερομηνία από το Excel** αλλά δεν ήξερες πώς να διαχειριστείς μορφές που εξαρτώνται από τον πολιτισμό; Δεν είσαι μόνος/η. Είτε εξάγεις μια οικονομική περίοδο από ένα ιαπωνικό φύλλο εργασίας είτε απλώς κανονικοποιείς ημερομηνίες για μια αλυσίδα αναφορών, το κόλπο είναι να φορτώσεις σωστά το βιβλίο εργασίας, να διαβάσεις το σωστό κελί και να πεις στο .NET ποιος πολιτισμός θα χρησιμοποιηθεί.

Σε αυτόν τον οδηγό θα σου δείξουμε ακριβώς πώς να **εξάγεις ημερομηνία από το Excel** χρησιμοποιώντας C#. Θα καλύψουμε **πώς να φορτώσεις το βιβλίο εργασίας**, πώς να **διαβάσεις κελί Excel**, και ακόμη **πώς να διαβάσεις ιαπωνική ημερομηνία** χωρίς εικασίες. Στο τέλος θα έχεις ένα έτοιμο κομμάτι κώδικα που μπορείς να ενσωματώσεις σε οποιοδήποτε έργο .NET.

---

## Τι θα χρειαστείς

- .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί επίσης σε .NET Framework 4.6+)  
- Μια αναφορά στο **Aspose.Cells** (ή οποιαδήποτε συμβατή βιβλιοθήκη που παρέχει αντικείμενα `Workbook` και `Cell`)  
- Ένα αρχείο Excel (`japan.xlsx`) που αποθηκεύει μια ημερομηνία στο κελί **A1** με τη μορφή του ιαπωνικού ημερολογίου  

Αυτό είναι σχεδόν όλο—χωρίς επιπλέον υπηρεσίες, χωρίς COM interop, μόνο μερικά πακέτα NuGet και λίγες γραμμές κώδικα.

## Βήμα 3: Ανάγνωση του στόχου κελιού (Read Excel Cell)

Με το βιβλίο εργασίας στη μνήμη, μπορούμε να **διαβάσουμε το κελί Excel** A1. Η αναφορά `Worksheets[0]` παίρνει το πρώτο φύλλο· μπορείς να το αντικαταστήσεις με όνομα αν χρειάζεται.

```csharp
        // Step 3: Access cell A1 in the first worksheet
        Cell targetCell = workbook.Worksheets[0].Cells["A1"];
```

> **Κοινό λάθος:** Κάποιοι προγραμματιστές ξεχνούν ότι οι στήλες του Excel είναι 1‑βάσιμες, ενώ η συλλογή `Cells` της βιβλιοθήκης είναι 0‑βάσιμη όταν χρησιμοποιούνται αριθμητικοί δείκτες. Η σημειογραφία `["A1"]` παρακάμπτει αυτή τη σύγχυση.

## Βήμα 4: Ανάκτηση της τιμής ως DateTime (Read Japanese Date)

Το Excel αποθηκεύει τις ημερομηνίες ως σειριακούς αριθμούς, αλλά η οπτική αναπαράσταση μπορεί να διαφέρει ανάλογα με την τοπική ρύθμιση. Με τη μεταβίβαση ενός αντικειμένου `CultureInfo` λέμε στο Aspose.Cells πώς να ερμηνεύσει τον αριθμό. Να πώς να **διαβάσεις ιαπωνική ημερομηνία** σωστά:

```csharp
        // Step 4: Retrieve the cell value as a DateTime using Japanese culture
        // The "ja-JP" culture knows about the Japanese calendar and date separators
        DateTime japaneseDate = targetCell.GetDateTimeValue(new CultureInfo("ja-JP"));
        
        Console.WriteLine($"Extracted date: {japaneseDate:yyyy-MM-dd}");
    }
}
```

**Αναμενόμενη έξοδος** (υποθέτοντας ότι το A1 περιέχει “2023/04/01” σε ιαπωνική μορφή):

```
Extracted date: 2023-04-01
```

> **Γιατί χρησιμοποιούμε `CultureInfo`;** Αν παραλείψεις τον πολιτισμό, το Aspose θα υποθέσει τον τρέχοντα πολιτισμό του νήματος (συχνά en‑US). Αυτό μπορεί να οδηγήσει σε ανταλλαγές μήνα/ημέρας ή εντελώς λανθασμένα έτη όταν δουλεύεις με ιαπωνικά ονόματα εποχών.

## Βήμα 5: Προστασία ενάντια σε κενά ή μη‑ημερομηνιακά κελιά (How to Read Excel Date Safely)

Τα πραγματικά φύλλα εργασίας δεν είναι πάντα τακτοποιημένα. Ας προσθέσουμε έναν γρήγορο έλεγχο ώστε ο κώδικας να μην πετάξει εξαίρεση αν το A1 είναι κενό ή περιέχει κείμενο.

```csharp
        // Optional safety net
        if (targetCell.Type != CellValueType.IsDateTime)
        {
            Console.WriteLine("Cell A1 does not contain a valid date.");
            return;
        }
```

Μπορείς επίσης να επιστρέψεις σε `DateTime.TryParse` με συγκεκριμένο format string αν το κελί αποθηκεύει την ημερομηνία ως συμβολοσειρά αντί για πραγματική ημερομηνία Excel.

## Πλήρες Παράδειγμα Λειτουργικού Κώδικα

Συνδυάζοντας τα παραπάνω, εδώ είναι το **πλήρες, εκτελέσιμο πρόγραμμα** που δείχνει πώς να **εξάγεις ημερομηνία από το Excel**, **να διαβάσεις κελί Excel**, και **να διαβάσεις ιαπωνική ημερομηνία** σε μια ομαλή ροή.

```csharp
using Aspose.Cells;
using System;
using System.Globalization;

class ExcelDateExtractor
{
    static void Main()
    {
        // ---- 1️⃣ Load the workbook -------------------------------------------------
        string filePath = @"C:\Data\japan.xlsx";          // adjust as needed
        Workbook workbook = new Workbook(filePath);

        // ---- 2️⃣ Grab the target cell ------------------------------------------------
        Cell targetCell = workbook.Worksheets[0].Cells["A1"];

        // ---- 3️⃣ Validate the cell content -----------------------------------------
        if (targetCell.Type != CellValueType.IsDateTime)
        {
            Console.WriteLine("Cell A1 does not contain a valid date.");
            return;
        }

        // ---- 4️⃣ Extract the date using Japanese culture ----------------------------
        DateTime extractedDate = targetCell.GetDateTimeValue(new CultureInfo("ja-JP"));

        // ---- 5️⃣ Show the result ----------------------------------------------------
        Console.WriteLine($"Extracted date: {extractedDate:yyyy-MM-dd}");
    }
}
```

**Τρέξε το** (`dotnet run`) και θα δεις την μορφοποιημένη ημερομηνία να εμφανίζεται στην κονσόλα. Ανάθεσε διαφορετική διαδρομή αρχείου, δείκτη φύλλου ή αναφορά κελιού ώστε να ταιριάζει στο δικό σου βιβλίο εργασίας, και το ίδιο μοτίβο θα λειτουργεί.

## Ακραίες Περιπτώσεις & Παραλλαγές

| Κατάσταση                              | Τι πρέπει να αλλάξει                                                            |
|----------------------------------------|---------------------------------------------------------------------------------|
| **Το κελί περιέχει συμβολοσειρά** (π.χ., “2023‑04‑01”) | Χρησιμοποίησε `DateTime.TryParseExact(targetCell.StringValue, "yyyy-MM-dd", new CultureInfo("ja-JP"), DateTimeStyles.None, out var dt)` |
| **Πολλά φύλλα**                        | Αντικατέστησε `Worksheets[0]` με `Worksheets["SheetName"]` ή κάνε βρόχο μέσω `workbook.Worksheets` |
| **Διαφορετικός πολιτισμός** (π.χ., Γαλλικός) | Πέρασε `new CultureInfo("fr-FR")` αντί για `"ja-JP"`                           |
| **Μεγάλο αρχείο** ( > 10 000 γραμμές)  | Σκέψου να χρησιμοποιήσεις `Workbook.LoadOptions` με `MemorySetting` για μείωση χρήσης RAM |

## Συχνές Ερωτήσεις

**Ε: Λειτουργεί αυτό με αρχεία .xls;**  
Α: Ναι. Το Aspose.Cells ανιχνεύει αυτόματα τη μορφή, οπότε μπορείς να δώσεις στο `Workbook` ένα παλιό `.xls` και ο ίδιος κώδικας ισχύει.

**Ε: Πώς να εμφανίσω την ημερομηνία στην ιαπωνική εποχή (π.χ., Reiwa 5);**  
Α: Χρησιμοποίησε `japaneseDate.ToString("gg y年M月d日", new CultureInfo("ja-JP"))` για μορφοποίηση με σύμβολα εποχής.

Α: Φυσικά. Κάνε βρόχο πάνω σε μια περιοχή—`Cells["A1:A100"]`—και εφάρμοσε την ίδια λογική `GetDateTimeValue` μέσα στον βρόχο.
