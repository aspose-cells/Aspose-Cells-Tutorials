# Πώς να δημιουργήσετε βιβλίο εργασίας σε C# – Εγγραφή τιμής & μορφοποίηση αριθμού

Η δημιουργία βιβλίου εργασίας σε C# είναι μια κοινή εργασία όταν χρειάζεται να δημιουργήσετε αρχεία Excel σε πραγματικό χρόνο. Σε αυτόν τον οδηγό θα σας δείξουμε πώς να γράψετε τιμή σε κελί και να μορφοποιήσετε τον αριθμό του κελιού ώστε το τελικό φύλλο να φαίνεται επαγγελματικό.

Αν ποτέ έχετε κολλήσει σε ένα κενό φύλλο εργασίας και αναρωτηθείτε γιατί οι αριθμοί εμφανίζονται με πάρα πολλά δεκαδικά ψηφία, δεν είστε μόνοι. Θα καλύψουμε τα πάντα, από την αρχικοποίηση του αντικειμένου workbook μέχρι τον ορισμό προσαρμοσμένης μορφής αριθμού, και θα προσθέσουμε μερικές συμβουλές για ειδικές περιπτώσεις που μπορεί να συναντήσετε αργότερα.

## Τι θα μάθετε

- **Initialize** ένα νέο αντικείμενο `Workbook`.  
- **Write value to cell** χρησιμοποιώντας τη μέθοδο `PutValue`.  
- **Set cell number format** με ένα αντικείμενο `Style`, επιτυγχάνοντας μια καθαρή εμφάνιση με δύο δεκαδικά ψηφία.  
- Επαληθεύστε το αποτέλεσμα διαβάζοντας το κελί ξανά ή ανοίγοντας το αρχείο στο Excel.  

Δεν απαιτούνται εξωτερικές βιβλιοθήκες πέρα από το τυπικό Aspose.Cells (ή οποιοδήποτε παρόμοιο API), και ο κώδικας εκτελείται σε .NET 6+ χωρίς επιπλέον ρυθμίσεις.

## Πώς να δημιουργήσετε Workbook – Αρχικοποίηση του αντικειμένου

Πρώτα απ' όλα: χρειάζεστε ένα αντικείμενο workbook για να κρατάει τα φύλλα σας. Σκεφτείτε το `Workbook` ως ολόκληρο το αρχείο Excel, ενώ κάθε `Worksheet` είναι μια ξεχωριστή καρτέλα.

```csharp
// Step 1: Create a new workbook instance
Workbook workbook = new Workbook();
```

*Γιατί είναι σημαντικό:* Η δημιουργία του workbook διανέμει τις εσωτερικές δομές που αργότερα κρατούν γραμμές, στήλες και μορφοποιήσεις. Χωρίς αυτό το αντικείμενο, δεν υπάρχει που να γράψετε μια τιμή σε κελί.

> **Pro tip:** Αν σκοπεύετε να εργαστείτε με ένα υπάρχον αρχείο, αντικαταστήστε το `new Workbook()` με το `new Workbook("template.xlsx")` για να φορτώσετε ένα πρότυπο και να διατηρήσετε τις μορφές του.

## Εγγραφή τιμής σε κελί

Τώρα που έχουμε ένα workbook, ας τοποθετήσουμε έναν αριθμό στο κελί **A1** του πρώτου φύλλου εργασίας.

```csharp
// Step 2: Access cell A1 in the first worksheet
Cell cellA1 = workbook.Worksheets[0].Cells["A1"];

// Step 3: Insert a numeric value into the cell
cellA1.PutValue(123.456789);
```

*Γιατί χρησιμοποιούμε το `PutValue`*: Αυτή η μέθοδος ανιχνεύει αυτόματα τον τύπο δεδομένων, ώστε να μην χρειάζεται να κάνετε cast ή μετατροπή χειροκίνητα. Επίσης σέβεται το υπάρχον στυλ του κελιού, κάτι χρήσιμο όταν αργότερα **ορίσετε μορφή αριθμού κελιού**.

### Γρήγορος έλεγχος

Αν διαβάσετε το κελί ξανά, θα δείτε την ακατέργαστη τιμή:

```csharp
double raw = cellA1.DoubleValue; // raw == 123.456789
```

Αυτός είναι ο αριθμός πριν εφαρμοστεί οποιαδήποτε μορφοποίηση.

## Ορισμός μορφής αριθμού κελιού

Η εμφάνιση ενός ακατέργαστου double με πολλά δεκαδικά δεν είναι πάντα φιλική προς τον χρήστη. Ας το περιορίσουμε σε δύο σημαντικά ψηφία.

```csharp
// Step 4: Apply a style that formats the number with two significant digits
cellA1.SetStyle(new Style() { Number = 2 });
```

Η ιδιότητα `Number` αντιστοιχεί στα ενσωματωμένα IDs μορφής αριθμού του Excel. Το `2` σημαίνει “Αριθμός με δύο δεκαδικά ψηφία”. Αν χρειάζεστε διαφορετική μορφή—π.χ. νόμισμα ή ημερομηνία—θα χρησιμοποιήσετε άλλο ID ή μια προσαρμοσμένη συμβολοσειρά μορφής.

### Εναλλακτική: Προσαρμοσμένη συμβολοσειρά μορφής

```csharp
Style customStyle = workbook.CreateStyle();
customStyle.Custom = "#,##0.00"; // forces two decimals with thousand separator
cellA1.SetStyle(customStyle);
```

*Γιατί να επιλέξετε προσαρμοσμένο στυλ;* Σας δίνει πλήρη έλεγχο, ειδικά όταν τα ενσωματωμένα IDs δεν καλύπτουν τις περιφερειακές σας ρυθμίσεις.

## Επαλήθευση εξόδου (Προαιρετικό αλλά Συνιστάται)

Αφού εφαρμόσετε το στυλ, μπορείτε να αποθηκεύσετε το workbook και να το ανοίξετε στο Excel για να επιβεβαιώσετε την εμφάνιση.

```csharp
// Save the workbook to a file
workbook.Save("FormattedWorkbook.xlsx");

// Or, for quick verification in code:
string displayed = cellA1.StringValue; // "123.46"
Console.WriteLine($"Displayed value: {displayed}");
```

Θα πρέπει να δείτε **123.46** στο κελί A1—ακριβώς δύο δεκαδικά ψηφία, χάρη στη μορφή που ορίσαμε.

### Πλήρες λειτουργικό παράδειγμα

Συνδυάζοντας όλα, εδώ είναι ένα αυτόνομο πρόγραμμα που μπορείτε να αντιγράψετε‑επικολλήσετε σε μια εφαρμογή κονσόλας.

```csharp
using System;
using Aspose.Cells;   // Ensure you have the Aspose.Cells NuGet package

class Program
{
    static void Main()
    {
        // Initialize the workbook
        Workbook workbook = new Workbook();

        // Access the first worksheet and cell A1
        Cell cellA1 = workbook.Worksheets[0].Cells["A1"];

        // Write a numeric value
        cellA1.PutValue(123.456789);

        // Apply a two‑decimal number format
        cellA1.SetStyle(new Style() { Number = 2 });

        // Save to disk (optional)
        workbook.Save("FormattedWorkbook.xlsx");

        // Output the displayed text for verification
        Console.WriteLine($"Cell A1 shows: {cellA1.StringValue}");
    }
}
```

**Αναμενόμενη έξοδος όταν εκτελέσετε το πρόγραμμα:**

```
Cell A1 shows: 123.46
```

Ανοίξτε το `FormattedWorkbook.xlsx` στο Excel και θα δείτε την ίδια μορφοποιημένη τιμή.

## Συνηθισμένες παραλλαγές & ειδικές περιπτώσεις

### 1. Διαφορετικές μορφές αριθμού

| Στόχος | ID μορφής | Κώδικας |
|------|-----------|--------------|
| Νόμισμα (δύο δεκαδικά) | 5 | `cellA1.SetStyle(new Style() { Number = 5 });` |
| Ποσοστό (χωρίς δεκαδικά) | 10 | `cellA1.SetStyle(new Style() { Number = 10 });` |
| Επιστημονική σημειογραφία | 11 | `cellA1.SetStyle(new Style() { Number = 11 });` |

Αν κανένα από τα ενσωματωμένα IDs δεν ταιριάζει, επιστρέψτε σε μια προσαρμοσμένη συμβολοσειρά όπως φαίνεται παραπάνω.

### 2. Πολιτισμικά‑συγκεκριμένοι διαχωριστές δεκαδικών

Ορισμένες τοπικές ρυθμίσεις χρησιμοποιούν κόμματα για τα δεκαδικά. Μπορείτε να επιβάλετε μια μορφή που λαμβάνει υπόψη τον πολιτισμό:

```csharp
Style cultureStyle = workbook.CreateStyle();
cultureStyle.Custom = "#,##0.00"; // works for most European locales
cellA1.SetStyle(cultureStyle);
```

### 3. Εγγραφή κειμένου αντί για αριθμούς

Όταν χρειάζεται να **γράψετε κελί** με μια συμβολοσειρά, απλώς περάστε μια συμβολοσειρά στο `PutValue`:

```csharp
cellA1.PutValue("Total Revenue");
```

Δεν απαιτείται μορφή αριθμού, αλλά μπορείτε ακόμη να εφαρμόσετε στυλ γραμματοσειράς.

### 4. Μεγάλα σύνολα δεδομένων

Αν γεμίζετε χιλιάδες γραμμές, η εισαγωγή σε παρτίδες (`Cells.ImportArray`) είναι πιο γρήγορη από την επανάληψη `PutValue`. Η προσέγγιση μορφοποίησης παραμένει η ίδια· απλώς εφαρμόζετε το στυλ σε μια περιοχή:

```csharp
Range range = workbook.Worksheets[0].Cells.CreateRange("B2:B1001");
range.ApplyStyle(new Style() { Number = 2 });
```

## Συχνές Ερωτήσεις

**Q: Λειτουργεί αυτό με .NET Core;**  
A: Απόλυτα. Το Aspose.Cells υποστηρίζει .NET Standard 2.0 και μεταγενέστερα, ώστε να μπορείτε να στοχεύσετε .NET 5, .NET 6 ή .NET 7 χωρίς αλλαγές.

**Q: Τι γίνεται αν χρειάζομαι περισσότερα από δύο δεκαδικά ψηφία;**  
A: Αλλάξτε την ιδιότητα `Number` στο κατάλληλο ενσωματωμένο ID (π.χ., `3` για τρία δεκαδικά) ή προσαρμόστε τη προσαρμοσμένη συμβολοσειρά μορφής (`"#,##0.000"`).

**Q: Μπορώ να εφαρμόσω τη μορφή σε ολόκληρη τη στήλη ταυτόχρονα;**  
A: Ναι. Χρησιμοποιήστε `Cells["A:A"]` για να πάρετε ολόκληρη τη στήλη και μετά `SetStyle`.
