## Βήμα 2: Φόρτωση του βιβλίου εργασίας

## Βήμα 3: Επανάληψη σε φύλλα εργασίας

Τώρα που το βιβλίο εργασίας μας έχει φορτωθεί, ήρθε η ώρα να το ψάξουμε πιο βαθιά. Πρέπει να κάνετε επανάληψη σε κάθε φύλλο εργασίας στο βιβλίο εργασίας για να βρείτε τυχόν ενσωματωμένα αντικείμενα:

```csharp

    // Συνέχεια επεξεργασίας αντικειμένων OLE...

```

Με αυτό το απόσπασμα, χρησιμοποιούμε ένα `foreach` βρόχο για να περιηγηθούμε σε κάθε φύλλο στο βιβλίο εργασίας μας. Αποκτώντας πρόσβαση στο `OleObjects` συλλογή, μπορούμε να έχουμε πρόσβαση σε όλα τα ενσωματωμένα αντικείμενα σε αυτό το συγκεκριμένο φύλλο. 

## Βήμα 4: Εξαγωγή αντικειμένων OLE

Εδώ είναι που συμβαίνει η μαγεία! Πρέπει να επαναλάβετε κάθε αντικείμενο OLE για να εξαγάγετε και να αποθηκεύσετε τα αρχεία MOL:

```csharp
var index = 1;
foreach (OleObject ole in oles)
{
    string fileName = outputDir + "OleObject" + index + ".mol";
    FileStream fs = File.Create(fileName);
    fs.Write(ole.ObjectData, 0, ole.ObjectData.Length);
    fs.Close();
    index++;
}
```

Σε αυτήν την προσέγγιση:
- Παρακολουθούμε το ευρετήριο για να ονομάσουμε τα αρχεία εξόδου διαδοχικά.
- Για κάθε αντικείμενο OLE, δημιουργούμε ένα νέο αρχείο χρησιμοποιώντας το FileStream.
- Στη συνέχεια, γράφουμε τα ενσωματωμένα δεδομένα σε αυτό το αρχείο και κλείνουμε τη ροή.
