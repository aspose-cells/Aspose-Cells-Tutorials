Παρουσιάζουμε το Aspose.Cells για .NET—ένα ισχυρό API σχεδιασμένο για εργασία με αρχεία Excel μέσω προγραμματισμού. Ένα από τα εξαιρετικά χαρακτηριστικά του έγκειται στην ενσωμάτωση του OData, επιτρέποντας στους χρήστες να αλληλεπιδρούν απρόσκοπτα με σύνθετες πηγές δεδομένων. Σε αυτόν τον οδηγό, θα περιηγηθούμε στη διαδικασία βήμα προς βήμα για την εξαγωγή λεπτομερειών OData χρησιμοποιώντας το Aspose.Cells για .NET.

Μόλις έχετε έτοιμα αυτά τα στοιχεία, θα είστε έτοιμοι να ξεκινήσετε την εξαγωγή λεπτομερειών OData χωρίς κόπο!

## Βήμα 1: Ρύθμιση του βιβλίου εργασίας

Σε αυτό το αρχικό βήμα, θα δημιουργήσετε μια παρουσία του `Workbook` κλάση και φορτώστε το αρχείο Excel σας:

```csharp
// Ορισμός του καταλόγου πηγής
string SourceDir = "Your Document Directory";
Workbook workbook = new Workbook(SourceDir + "ODataSample.xlsx");
```

## Βήμα 2: Πρόσβαση σε τύπους Power Query

Στη συνέχεια, θα έχετε πρόσβαση στους τύπους Power Query στο βιβλίο εργασίας σας, οι οποίοι περιέχουν τις λεπτομέρειες του OData:

```csharp
PowerQueryFormulaCollction PQFcoll = workbook.DataMashup.PowerQueryFormulas;
```

Αυτή η γραμμή αρχικοποιεί μια συλλογή τύπων Power Query, προετοιμάζοντάς μας για επανάληψη και ανάκτηση των απαραίτητων λεπτομερειών.

## Βήμα 3: Επανάληψη των τύπων

Τώρα, χρησιμοποιήστε έναν βρόχο για να περάσετε από κάθε τύπο Power Query, ανακτώντας το όνομά του και τα σχετικά στοιχεία:

```csharp
foreach (PowerQueryFormula PQF in PQFcoll)
{
    Console.WriteLine("Connection Name: " + PQF.Name);
    PowerQueryFormulaItemCollection PQFIcoll = PQF.PowerQueryFormulaItems;
    
    foreach (PowerQueryFormulaItem PQFI in PQFIcoll)
    {
        Console.WriteLine("Name: " + PQFI.Name);
        Console.WriteLine("Value: " + PQFI.Value);
    }
}
```

Σε αυτό το μπλοκ, εμείς:
- Εκτυπώστε το όνομα σύνδεσης κάθε τύπου Power Query.
- Αποκτήστε πρόσβαση στα στοιχεία μέσα σε κάθε τύπο και εκτυπώστε τα ονόματα και τις τιμές τους.

## Βήμα 4: Εκτέλεση & Επαλήθευση

Τέλος, πρέπει να βεβαιωθείτε ότι ο κώδικας εκτελείται σωστά και επιστρέφει την αναμενόμενη έξοδο. Προσθέστε την ακόλουθη γραμμή στο τέλος του `Main` μέθοδος:
