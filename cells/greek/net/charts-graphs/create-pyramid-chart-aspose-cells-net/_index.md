## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε ένα διάγραμμα πυραμίδας.

### Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας

**Επισκόπηση:**
Θα ξεκινήσουμε δημιουργώντας ένα βιβλίο εργασίας του Excel και αποκτώντας πρόσβαση στο πρώτο φύλλο εργασίας του.

#### Βήμα 1: Δημιουργία στιγμιότυπου βιβλίου εργασίας

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Αρχικοποίηση ενός νέου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

### Προσθήκη δειγμάτων δεδομένων σε κελιά

**Επισκόπηση:**
Στη συνέχεια, συμπληρώστε το φύλλο εργασίας με δείγματα δεδομένων για το γράφημά μας.

#### Βήμα 2: Συμπλήρωση κελιών

```csharp
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```

### Προσθήκη γραφήματος πυραμίδας στο φύλλο εργασίας

**Επισκόπηση:**
Τώρα, προσθέστε ένα διάγραμμα πυραμίδας για να οπτικοποιήσετε τα δεδομένα.

#### Βήμα 3: Εισαγωγή γραφήματος πυραμίδας

```csharp
using Aspose.Cells.Charts;

// Προσθήκη γραφήματος πυραμίδας στο φύλλο εργασίας
int chartIndex = worksheet.Charts.Add(ChartType.Pyramid, 5, 0, 25, 10);
Chart chart = worksheet.Charts[chartIndex];
```

### Ορισμός πηγής δεδομένων γραφήματος

**Επισκόπηση:**
Ορίστε ποιο εύρος δεδομένων θα χρησιμοποιηθεί για το διάγραμμα πυραμίδας μας.

#### Βήμα 4: Ρύθμιση παραμέτρων δεδομένων γραφήματος

```csharp
// Ορισμός του εύρους πηγής δεδομένων για το γράφημα
chart.NSeries.Add("A1:B3", true);
```

### Αποθήκευση βιβλίου εργασίας σε αρχείο

**Επισκόπηση:**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με το νέο πυραμιδικό διάγραμμα.

#### Βήμα 5: Αποθήκευση αρχείου Excel

```csharp
workbook.Save(outputDir + "outputHowToCreatePyramidChart.xlsx");
```
