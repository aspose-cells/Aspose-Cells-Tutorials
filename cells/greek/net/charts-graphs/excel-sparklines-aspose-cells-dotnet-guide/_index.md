## Οδηγός Εφαρμογής

### Ανάγνωση πληροφοριών Sparkline

#### Επισκόπηση
Η ανάγνωση πληροφοριών από γραφήματα sparkline περιλαμβάνει την πρόσβαση σε υπάρχουσες ομάδες και τις λεπτομέρειες τους μέσα σε ένα φύλλο εργασίας.

**Βήμα 1: Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας**

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook book = new Workbook(SourceDir + "/sampleUsingSparklines.xlsx");
Worksheet sheet = book.Worksheets[0];
```

**Βήμα 2: Επαναλάβετε μέσω ομάδων Sparkline**

```csharp
foreach (SparklineGroup g in sheet.SparklineGroups)
{
    Console.WriteLine("sparkline group: type:" + g.Type + ", sparkline items count:" + g.Sparklines.Count);
    
    foreach (Sparkline s in g.Sparklines)
    {
        Console.WriteLine("sparkline: row:" + s.Row + ", col:" + s.Column + ", dataRange:" + s.DataRange);
    }
}
```

Σε αυτόν τον κώδικα, `g.Type` και `g.Sparklines.Count` δώστε τον τύπο ομάδας και τον αριθμό των sparkline. Για κάθε sparkline, μπορείτε να έχετε πρόσβαση στη θέση του (`Row`, `Column`) και `DataRange`.

### Προσθήκη γραφημάτων σπινθήρων σε ένα φύλλο εργασίας

#### Επισκόπηση
Η προσθήκη γραφημάτων sparkline σάς επιτρέπει να οπτικοποιήσετε τις τάσεις των δεδομένων μέσω προγραμματισμού.

**Βήμα 1: Ορισμός CellArea για Sparklines**

```csharp
CellArea ca = new CellArea();
ca.StartColumn = 4;
ca.EndColumn = 4;
ca.StartRow = 1;
ca.EndRow = 7;
```

**Βήμα 2: Προσθήκη νέας ομάδας Sparkline**

```csharp
int idx = sheet.SparklineGroups.Add(SparklineType.Column, "Sheet1!B2:D8", false, ca);
SparklineGroup group = sheet.SparklineGroups[idx];
```

Εδώ, `SparklineType.Column` Καθορίζει τον τύπο των γραφημάτων sparkline που θα προστεθούν. Το εύρος δεδομένων και η περιοχή εμφάνισης ορίζονται από αναφορές κελιών.

**Βήμα 3: Προσαρμόστε την εμφάνιση του Sparkline**

```csharp
CellsColor clr = book.CreateCellsColor();
clr.Color = Color.Orange;
group.SeriesColor = clr;
```

Μπορείτε να προσαρμόσετε το χρώμα χρησιμοποιώντας `CellsColor`, ενισχύοντας την οπτική διάκριση.

**Βήμα 4: Αποθήκευση του βιβλίου εργασίας**

```csharp
book.Save(outputDir + "/outputUsingSparklines.xlsx");
```

Αυτό αποθηκεύει τις αλλαγές σας, διατηρώντας τα πρόσφατα προστιθέμενα sparkline στον καθορισμένο κατάλογο εξόδου.
