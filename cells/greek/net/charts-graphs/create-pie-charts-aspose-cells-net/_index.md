# Δημιουργία γραφημάτων πίτας με γραμμές οδηγού χρησιμοποιώντας το Aspose.Cells .NET

## Οδηγός Εφαρμογής

### Δημιουργία του Βιβλίου Εργασίας και του Φύλλου Εργασίας
1. **Αρχικοποίηση του βιβλίου εργασίας**
   Δημιουργήστε ένα νέο βιβλίο εργασίας σε μορφή XLSX:
   ```csharp
   Workbook workbook = new Workbook(FileFormatType.Xlsx);
   ```

2. **Πρόσβαση στο Πρώτο Φύλλο Εργασίας**
   Χρησιμοποιήστε το πρώτο φύλλο εργασίας για να εισαγάγετε δεδομένα:
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **Προσθήκη δεδομένων για γράφημα πίτας**
   Συμπληρώστε το φύλλο εργασίας σας με κατηγορίες και τιμές:
   ```csharp
   worksheet.Cells["A1"].PutValue("Retail");
   // Προσθήκη ονομάτων κατηγοριών που απομένουν...
   worksheet.Cells["B1"].PutValue(10.4);
   // Προσθέστε αντίστοιχες τιμές...
   ```

### Προσθήκη κυκλικού γραφήματος στο φύλλο εργασίας
1. **Δημιουργήστε το γράφημα πίτας**
   Δημιουργήστε ένα γράφημα πίτας και προσθέστε το στη συλλογή γραφημάτων του φύλλου εργασίας σας:
   ```csharp
   int id = worksheet.Charts.Add(ChartType.Pie, 3, 3, 23, 13);
   ```

2. **Ρύθμιση παραμέτρων δεδομένων σειρών και κατηγοριών**
   Συνδέστε τα δεδομένα για τη σειρά και τις κατηγορίες:
   ```csharp
   Chart chart = worksheet.Charts[id];
   chart.NSeries.Add("B1:B16", true);
   chart.NSeries.CategoryData = "A1:A16";
   ```

3. **Προσαρμογή ετικετών δεδομένων**
   Απενεργοποίηση εμφάνισης υπομνήματος, ορισμός ετικετών δεδομένων ώστε να εμφανίζουν ονόματα κατηγοριών και ποσοστά:
   ```csharp
   chart.ShowLegend = false;
   DataLabels dataLabels = chart.NSeries[0].DataLabels;
   dataLabels.ShowCategoryName = true;
   dataLabels.ShowPercentage = true;
   dataLabels.Position = LabelPositionType.OutsideEnd;
   ```

### Υλοποίηση γραμμών Leader
1. **Ενεργοποίηση γραμμών οδηγού**
   Ενεργοποιήστε τις γραμμές οδηγού για πιο καθαρές οπτικές συνδέσεις:
   ```csharp
   chart.NSeries[0].HasLeaderLines = true;
   ```

2. **Προσαρμογή θέσης ετικετών δεδομένων**
   Εξασφαλίστε ορατότητα προσαρμόζοντας τις θέσεις των ετικετών:
   ```csharp
   int DELTA = 100;
   foreach (var point in chart.NSeries[0].Points)
   {
       int X = point.DataLabels.X;
       if (X > 2000) 
           point.DataLabels.X += DELTA;
       else 
           point.DataLabels.X -= DELTA;
   }
   ```

### Αποθήκευση του γραφήματος και του βιβλίου εργασίας
1. **Αποθήκευση ως εικόνα**
   Αποδώστε το γράφημα σε αρχείο εικόνας:
   ```csharp
   ImageOrPrintOptions options = new ImageOrPrintOptions { ImageType = Drawing.ImageType.Png, HorizontalResolution = 200, VerticalResolution = 200 };
   chart.ToImage("output_out.png", options);
   ```

2. **Αποθήκευση βιβλίου εργασίας**
   Αποθηκεύστε το βιβλίο εργασίας για να δείτε το γράφημα μέσα στο Excel:
   ```csharp
   workbook.Save("output_out.xlsx");
   ```
