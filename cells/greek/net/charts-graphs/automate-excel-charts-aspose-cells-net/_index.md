## Οδηγός Εφαρμογής

Θα καλύψουμε τα βασικά χαρακτηριστικά με βήματα και αποσπάσματα κώδικα για να διευκολύνουμε την εφαρμογή.

### Λειτουργία 1: Φόρτωση αρχείου Excel

Φόρτωση ενός υπάρχοντος αρχείου Excel χρησιμοποιώντας το `Workbook` κλάση από το Aspose.Cells.

**Βήμα 1:** Ορίστε τον κατάλογο πηγής σας:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
```

**Βήμα 2:** Φόρτωση του βιβλίου εργασίας:
```csharp
Workbook wb = new Workbook(SourceDir + "/sampleReadManipulateExcel2016Charts.xlsx");
```

### Χαρακτηριστικό 2: Φύλλα εργασίας και γραφήματα της Access

Αποκτήστε πρόσβαση σε συγκεκριμένα φύλλα εργασίας και στα γραφήματά τους για χειρισμό.

**Βήμα 1:** Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας:
```csharp
Worksheet ws = wb.Worksheets[0];
```

**Βήμα 2:** Επαναλάβετε όλα τα γραφήματα σε αυτό το φύλλο εργασίας:
```csharp
for (int i = 0; i < ws.Charts.Count; i++)
{
    Chart ch = ws.Charts[i];
}
```

### Λειτουργία 3: Ανάγνωση και τροποποίηση ιδιοτήτων γραφήματος

Προσαρμόστε τα γραφήματα Excel σας ενημερώνοντας τους τίτλους με βάση τον τύπο του γραφήματος.

**Βήμα 1:** Επαναλάβετε κάθε διάγραμμα:
```csharp
for (int i = 0; i < ws.Charts.Count; i++)
{
    Chart ch = ws.Charts[i];
```

**Βήμα 2:** Ενημερώστε τον τίτλο ώστε να περιλαμβάνει τον τύπο γραφήματος:
```csharp
string chartType = ch.Type.ToString();
ch.Title.Text = "Chart Type is " + chartType;
}
```

### Λειτουργία 4: Αποθήκευση τροποποιημένου βιβλίου εργασίας

Διατηρήστε τις αλλαγές αποθηκεύοντας το βιβλίο εργασίας σας.

**Βήμα 1:** Ορίστε τον κατάλογο εξόδου:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**Βήμα 2:** Αποθήκευση του τροποποιημένου βιβλίου εργασίας:
```csharp
wb.Save(outputDir + "/outputReadManipulateExcel2016Charts.xlsx");
```
