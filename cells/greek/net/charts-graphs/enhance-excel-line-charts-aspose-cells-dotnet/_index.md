### Οδηγός Εφαρμογής
#### Προσθήκη σειράς δεδομένων σε υπάρχον γράφημα
##### Επισκόπηση
Η βελτίωση των γραφημάτων με νέες σειρές δεδομένων μπορεί να προσφέρει βαθύτερες πληροφορίες. Δείτε πώς μπορείτε να το κάνετε χρησιμοποιώντας το Aspose.Cells.

##### Βήματα για την προσθήκη μιας νέας σειράς
**1. Φόρτωση του βιβλίου εργασίας σας**
Ξεκινήστε φορτώνοντας το αρχείο Excel που περιέχει το γράφημά σας:
```csharp
Workbook workbook = new Workbook("sampleModifyLineChart.xlsx");
```

**2. Πρόσβαση στο Διάγραμμα**
Προσδιορίστε και αποκτήστε πρόσβαση στο συγκεκριμένο γράφημα όπου θέλετε να προσθέσετε σειρές δεδομένων:
```csharp
Chart chart = workbook.Worksheets[0].Charts[0];
```

**3. Προσθήκη νέας σειράς δεδομένων**
Χρήση `NSeries.Add` για την εισαγωγή νέων σειρών δεδομένων:
```csharp
// Προσθήκη τρίτης σειράς δεδομένων
chart.NSeries.Add("{60, 80, 10}", true);

// Προσθήκη τέταρτης σειράς δεδομένων
chart.NSeries.Add("{0.3, 0.7, 1.2}", true);
```

**4. Ρύθμιση παραμέτρων ιδιοτήτων σειράς**
Προσαρμόστε την εμφάνιση της νέας σας σειράς:
```csharp
// Ορισμός χρώματος περιγράμματος για τη δεύτερη και τρίτη σειρά
chart.NSeries[1].Border.Color = Color.Green;
chart.NSeries[2].Border.Color = Color.Red;

// Σχεδιάστε την τέταρτη σειρά δεδομένων σε έναν δευτερεύοντα άξονα
chart.NSeries[3].PlotOnSecondAxis = true;

// Κάντε τον δευτερεύοντα άξονα τιμών ορατό
chart.SecondValueAxis.IsVisible = true;
```

**5. Αποθηκεύστε το βιβλίο εργασίας σας**
Αποθηκεύστε το τροποποιημένο βιβλίο εργασίας σας:
```csharp
workbook.Save("outputModifyLineChart.xlsx");
```
