## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει κάθε λειτουργία σε διαχειρίσιμα βήματα, βοηθώντας σας να δημιουργείτε και να διαμορφώνετε αποτελεσματικά συγκεντρωτικά γραφήματα.

### Δημιουργία και άνοιγμα βιβλίου εργασίας

#### Επισκόπηση
Δημιουργώντας ένα νέο `Workbook` Το αντικείμενο είναι το πρώτο βήμα για τον προγραμματιστικό χειρισμό ενός αρχείου Excel.

**Βήμα 1: Φόρτωση ενός υπάρχοντος βιβλίου εργασίας**
```csharp
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string fileName = "sampleCreatePivotChart.xlsx";

// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας με τη διαδρομή προς το αρχείο Excel σας
Workbook workbook = new Workbook(sourceDir + fileName);
```

- **Παράμετροι:** Ο κατασκευαστής παίρνει τη διαδρομή αρχείου του εγγράφου Excel.
- **Σκοπός:** Αυτό το βήμα προετοιμάζει το βιβλίο εργασίας για περαιτέρω λειτουργίες, όπως η προσθήκη φύλλων ή γραφημάτων.

### Προσθήκη και ονομασία νέου φύλλου

#### Επισκόπηση
Η προσθήκη ενός φύλλου γραφήματος είναι απαραίτητη για τη φιλοξενία συγκεντρωτικών γραφημάτων. Δείτε πώς μπορείτε να το κάνετε:

**Βήμα 2: Δημιουργήστε ένα νέο φύλλο γραφήματος**
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Προσθήκη νέου φύλλου γραφήματος με το όνομα «Συγκεντρωτικό Γράφημα»
Worksheet sheet3 = workbook.Worksheets[workbook.Worksheets.Add(SheetType.Chart)];
sheet3.Name = "PivotChart";
```

- **Παράμετροι:** `SheetType.Chart` καθορίζει τον τύπο του φύλλου.
- **Σκοπός:** Αυτό το βήμα προσθέτει έναν ειδικό χώρο για το συγκεντρωτικό σας διάγραμμα, ο οποίος ονομάζεται για εύκολη αναγνώριση.

### Προσθήκη και ρύθμιση παραμέτρων γραφήματος στηλών

#### Επισκόπηση
Για να προσθέσετε ένα γράφημα στηλών που χρησιμεύει ως συγκεντρωτικό γράφημα, ακολουθήστε τα εξής βήματα:

**Βήμα 3: Εισαγωγή και διαμόρφωση του Συγκεντρωτικού Γραφήματος**
```csharp
Worksheet sheet3 = workbook.Worksheets[0];

// Προσθήκη γραφήματος στηλών σε καθορισμένη θέση στο φύλλο εργασίας
int index = sheet3.Charts.Add(ChartType.Column, 0, 5, 28, 16);

// Ορισμός της πηγής δεδομένων για το συγκεντρωτικό γράφημα σε 'PivotTable1'
sheet3.Charts[index].PivotSource = "PivotTable!PivotTable1";

// Ρύθμιση παραμέτρων για απόκρυψη των κουμπιών του pivot field (ορίζεται σε false εδώ)
sheet3.Charts[index].HidePivotFieldButtons = false;
```

- **Παράμετροι:** Ο `Add` Η μέθοδος απαιτεί τον τύπο και τη θέση του γραφήματος.
- **Σκοπός:** Αυτό δημιουργεί ένα γράφημα συνδεδεμένο με τον συγκεντρωτικό σας πίνακα, επιτρέποντας τη δυναμική αναπαράσταση δεδομένων.

### Αποθήκευση του βιβλίου εργασίας

#### Επισκόπηση
Τέλος, αποθηκεύστε τις αλλαγές σας για να τις διατηρήσετε σε ένα αρχείο Excel.

**Βήμα 4: Αποθήκευση του βιβλίου εργασίας σας**
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Αποθήκευση του τροποποιημένου βιβλίου εργασίας σε έναν καθορισμένο κατάλογο
workbook.Save(outputDir + "outputCreatePivotChart.xlsx");
```

- **Παράμετροι:** Ο `Save` Η μέθοδος ακολουθεί τη διαδρομή όπου θέλετε να αποθηκεύσετε το αρχείο Excel.
- **Σκοπός:** Αυτό το βήμα διασφαλίζει ότι όλες οι τροποποιήσεις σας αποθηκεύονται και είναι δυνατή η πρόσβαση σε αυτές ή η κοινοποίησή τους, όπως απαιτείται.
