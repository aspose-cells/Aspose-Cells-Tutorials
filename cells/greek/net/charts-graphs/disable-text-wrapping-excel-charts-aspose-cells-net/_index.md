## Οδηγός Εφαρμογής

Αφού όλα ρυθμιστούν, ας απενεργοποιήσουμε την αναδίπλωση κειμένου στις ετικέτες δεδομένων γραφήματος του Excel χρησιμοποιώντας το Aspose.Cells για .NET.

### Φόρτωση και πρόσβαση στο βιβλίο εργασίας
Φορτώστε το αρχείο Excel σας σε ένα `Workbook` αντικείμενο:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του δείγματος αρχείου Excel μέσα στο αντικείμενο του βιβλίου εργασίας
Workbook workbook = new Workbook(SourceDir + "/sampleDisableTextWrappingForDataLabels.xlsx");
```

### Πρόσβαση στο Φύλλο Εργασίας και στο Διάγραμμα
Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας και στο γράφημα που θέλετε να τροποποιήσετε:
```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];

// Πρόσβαση στο πρώτο γράφημα στο φύλλο εργασίας
Chart chart = worksheet.Charts[0];
```

### Απενεργοποίηση αναδίπλωσης κειμένου για ετικέτες δεδομένων
Απενεργοποίηση αναδίπλωσης κειμένου ρυθμίζοντας `IsTextWrapped` σε ψευδές:
```csharp
foreach (var series in chart.NSeries)
{
    // Ορίστε την τιμή IsTextWrapped σε false για να απενεργοποιήσετε την αναδίπλωση κειμένου
    series.DataLabels.IsTextWrapped = false;
}
```

### Αποθήκευση του τροποποιημένου βιβλίου εργασίας
Αποθηκεύστε τις αλλαγές σας γράφοντας το τροποποιημένο βιβλίο εργασίας σε ένα νέο αρχείο:
```csharp
// Αποθήκευση του βιβλίου εργασίας με τις αλλαγές σε νέο αρχείο

```
