# Πώς να μορφοποιήσετε τιμές σειράς γραφημάτων στο Excel χρησιμοποιώντας το Aspose.Cells .NET

## Εισαγωγή

Χρειάζεται να μορφοποιήσετε μέσω προγραμματισμού τιμές σειρών γραφημάτων στο Excel; Αυτό το σεμινάριο παρουσιάζει τη χρήση του Aspose.Cells για .NET για τον ορισμό κωδικών μορφοποίησης για σειρές γραφημάτων. Είτε αυτοματοποιείτε τη δημιουργία αναφορών είτε τυποποιείτε οικονομικές παρουσιάσεις, ο έλεγχος των μορφών τιμών μπορεί να βελτιώσει σημαντικά την αναγνωσιμότητα και τη συνέπεια των δεδομένων.

**Τι θα μάθετε:**
- Εγκατάσταση και αρχικοποίηση του Aspose.Cells για .NET
- Φόρτωση βιβλίου εργασίας και πρόσβαση στα στοιχεία του, όπως φύλλα εργασίας και γραφήματα
- Προσθήκη σειρών σε ένα γράφημα και ορισμός κώδικα μορφοποίησης τιμών τους
- Αποθήκευση αλλαγών σε αρχείο Excel

Αρχικά, ας εξετάσουμε τις προϋποθέσεις.

## Οδηγός Εφαρμογής

Ας χωρίσουμε τη διαδικασία σε ξεχωριστά βήματα για ευκολότερη εφαρμογή.

### Φόρτωση βιβλίου εργασίας από τον κατάλογο

**Επισκόπηση:** Ξεκινήστε φορτώνοντας ένα βιβλίο εργασίας του Excel από τον καθορισμένο κατάλογο.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
// Φόρτωση του αρχείου προέλευσης Excel 
Workbook wb = new Workbook(SourceDir + "sampleSeries_ValuesFormatCode.xlsx");
```

**Εξήγηση:**
- `SourceDir` είναι η διαδρομή προς τα αρχεία εισόδου σας.
- Ο `Workbook` Ο κατασκευαστής ανοίγει το καθορισμένο αρχείο.

### Φύλλο εργασίας Access από βιβλίο εργασίας

**Επισκόπηση:** Ανακτήστε το φύλλο εργασίας με το οποίο πρέπει να εργαστείτε.

```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet worksheet = wb.Worksheets[0];
```

**Εξήγηση:**
- Τα βιβλία εργασίας μπορούν να περιέχουν πολλά φύλλα εργασίας. Εδώ, έχουμε πρόσβαση στο πρώτο χρησιμοποιώντας ένα ευρετήριο `0`.

### Γράφημα πρόσβασης από φύλλο εργασίας

**Επισκόπηση:** Εντοπίστε το διάγραμμα μέσα στο επιλεγμένο φύλλο εργασίας που θέλετε να επεξεργαστείτε.

```csharp
// Πρόσβαση στο πρώτο διάγραμμα
Chart ch = worksheet.Charts[0];
```

**Εξήγηση:**
- Όπως και με τα φύλλα εργασίας, ένα φύλλο εργασίας μπορεί να έχει πολλά γραφήματα. Αυτός ο κώδικας έχει πρόσβαση στο πρώτο γράφημα.

### Προσθήκη σειράς στο γράφημα

**Επισκόπηση:** Προσθέστε σειρές δεδομένων στο γράφημά σας χρησιμοποιώντας έναν πίνακα τιμών.

```csharp
// Προσθήκη σειρών χρησιμοποιώντας έναν πίνακα τιμών
ch.NSeries.Add("{10000, 20000, 30000, 40000}", true);
```

**Εξήγηση:**
- `NSeries.Add` δέχεται μια αναπαράσταση συμβολοσειράς αριθμών και μια λογική τιμή που υποδεικνύει εάν το εύρος είναι αποκλειστικό. Εδώ, είναι συμπεριληπτικό.

### Ορισμός Κώδικα Μορφής Τιμών Σειράς

**Επισκόπηση:** Προσαρμόστε τον τρόπο μορφοποίησης των τιμών στη σειρά γραφημάτων σας.

```csharp
// Αποκτήστε πρόσβαση στη σειρά και ορίστε τον κώδικα μορφοποίησης τιμών της
Series srs = ch.NSeries[0];
srs.ValuesFormatCode = "$#,##0";
```

**Εξήγηση:**
- `ValuesFormatCode` σας επιτρέπει να ορίσετε μια προσαρμοσμένη μορφή αριθμού, όπως το νόμισμα σε αυτό το παράδειγμα (`"$#,##0"`).

### Αποθήκευση βιβλίου εργασίας στον κατάλογο

**Επισκόπηση:** Διατηρήστε τις αλλαγές σας αποθηκεύοντας το βιβλίο εργασίας σε έναν κατάλογο εξόδου.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
// Αποθηκεύστε το αρχείο Excel που προκύπτει
wb.Save(outputDir + "outputSeries_ValuesFormatCode.xlsx");
```

**Εξήγηση:**
- Ο `Save` Η μέθοδος γράφει το τροποποιημένο βιβλίο εργασίας σε ένα νέο αρχείο, διατηρώντας τις αλλαγές σας.
