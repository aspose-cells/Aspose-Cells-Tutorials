## Οδηγός Εφαρμογής

### Αλλαγή πηγής δεδομένων γραφήματος

Αυτή η λειτουργία σάς επιτρέπει να τροποποιήσετε εύκολα την πηγή δεδομένων ενός γραφήματος σε ένα βιβλίο εργασίας του Excel.

#### Επισκόπηση
Σε αυτήν την ενότητα, θα δείξουμε πώς να αλλάξετε την πηγή δεδομένων χρησιμοποιώντας το Aspose.Cells. Θα μάθετε πώς να φορτώνετε υπάρχοντα βιβλία εργασίας, να έχετε πρόσβαση σε φύλλα εργασίας και να ενημερώνετε γραφήματα.

**Βήμα 1: Φόρτωση του βιβλίου εργασίας**

Αρχικά, αρχικοποιήστε το `Workbook` αντικείμενο φορτώνοντας ένα υπάρχον αρχείο:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook(SourceDir + "/sampleChangeChartDataSource.xlsx");
```

**Βήμα 2: Πρόσβαση και ρύθμιση παραμέτρων φύλλων εργασίας**

Αποκτήστε πρόσβαση στο φύλλο εργασίας προέλευσης από το οποίο θα αντιγράψετε δεδομένα:
```csharp
Worksheet source = wb.Worksheets[0];
Worksheet destination = wb.Worksheets.Add("DestSheet");

CopyOptions options = new CopyOptions();
options.ReferToDestinationSheet = true;

destination.Cells.CopyRows(source.Cells, 0, 0, source.Cells.MaxDisplayRange.RowCount, options);
```

**Βήμα 3: Αποθήκευση του βιβλίου εργασίας**

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τα ενημερωμένα δεδομένα:
```csharp
wb.Save(outputDir + "/outputChangeChartDataSource.xlsx", SaveFormat.Xlsx);
```

### Φόρτωση και πρόσβαση σε βιβλίο εργασίας Excel
Η πρόσβαση σε υπάρχοντα βιβλία εργασίας είναι απλή με το Aspose.Cells.

**Βήμα 1: Φόρτωση ενός υπάρχοντος βιβλίου εργασίας**
Φορτώστε ένα βιβλίο εργασίας για να αποκτήσετε πρόσβαση στα φύλλα εργασίας του:
```csharp
Workbook wb = new Workbook(SourceDir + "/sampleChangeChartDataSource.xlsx");
Worksheet sourceSheet = wb.Worksheets[0];
```

### Προσθήκη και διαμόρφωση φύλλου εργασίας
Η προσθήκη και η διαμόρφωση φύλλων εργασίας είναι ζωτικής σημασίας για τη διαχείριση δεδομένων.

**Βήμα 1: Δημιουργία νέου βιβλίου εργασίας**
Αρχικοποίηση μιας νέας παρουσίας βιβλίου εργασίας:
```csharp
Workbook wb = new Workbook();
Worksheet destination = wb.Worksheets.Add("DestSheet");
```

**Βήμα 2: Αντιγραφή δεδομένων με τις Επιλογές**
Χρησιμοποιώ `CopyOptions` για να διαχειριστείτε τον τρόπο αντιγραφής των δεδομένων:
```csharp
CopyOptions options = new CopyOptions();
options.ReferToDestinationSheet = true;
destination.Cells.CopyRows(source.Cells, 0, 0, source.Cells.MaxDisplayRange.RowCount, options);
```

**Βήμα 3: Αποθήκευση του νέου βιβλίου εργασίας**
Αποθηκεύστε τις αλλαγές σας σε ένα αρχείο:
```csharp
wb.Save(outputDir + "/outputWorkbook.xlsx", SaveFormat.Xlsx);
```
