## Οδηγός Εφαρμογής
### Προσαρμογή ετικετών γραφημάτων με βάση τις τοπικές ρυθμίσεις
Ο κύριος στόχος είναι να παρακαμφθεί το προεπιλεγμένο κείμενο για τις ετικέτες γραφήματος πίτας χρησιμοποιώντας ρυθμίσεις που αφορούν συγκεκριμένες πολιτισμικές παραμέτρους. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### 1. Φορτώστε το βιβλίο εργασίας σας και αποκτήστε πρόσβαση στο διάγραμμα
Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο Excel που περιέχει ένα γράφημα πίτας:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleCustomTextForLabels.xlsx");
```

Αποκτήστε πρόσβαση στο φύλλο εργασίας και στο διάγραμμα που θέλετε να προσαρμόσετε:
```csharp
Worksheet sheet = book.Worksheets[0];
Chart chart = sheet.Charts[0];
```

#### 2. Ορισμός ρυθμίσεων παγκοσμιοποίησης
Παράκαμψη του `GetOtherName` μέθοδος για την παροχή προσαρμοσμένων ετικετών με βάση τις τοπικές ρυθμίσεις του συστήματος:

```csharp
GlobalizationSettings globalSettings = new GlobalizationSettings();
globalSettings.ChartSettings = new CustomSettings();
book.Settings.GlobalizationSettings = globalSettings;
```

Ορίστε την κλάση προσαρμοσμένων ρυθμίσεων:
```csharp
class CustomSettings : ChartGlobalizationSettings
{
    public override string GetOtherName()
    {
        int lcid = System.Globalization.CultureInfo.CurrentCulture.LCID;
        switch (lcid)
        {
            case 1033: // αγγλικός
                return "Other";
            case 1036: // Γάλλος
                return "Autre";
            case 1031: // Γερμανός
                return "Andere";
            default:
                return base.GetOtherName();
        }
    }
}
```

#### 3. Ανανέωση και απόδοση του γραφήματος
Για να εφαρμόσετε τις αλλαγές σας, ανανεώστε το γράφημα και αποδώστε το σε αρχείο εικόνας:

```csharp
chart.Calculate();
chart.ToImage(outputDir + "outputCustomTextForLabels.png", new ImageOrPrintOptions());
Console.WriteLine("CustomTextForLabels executed successfully.");
```
