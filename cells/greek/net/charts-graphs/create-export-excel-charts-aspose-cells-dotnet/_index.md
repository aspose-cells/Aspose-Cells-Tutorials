## Οδηγός Εφαρμογής

### Δημιουργία και συμπλήρωση βιβλίου εργασίας

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα νέο βιβλίο εργασίας, να προσθέσετε φύλλα εργασίας και να τα συμπληρώσετε με δείγματα δεδομένων.

#### Βήμα προς βήμα εφαρμογή:

**1. Αρχικοποιήστε το Βιβλίο Εργασίας:**
```csharp
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργία αντικειμένου βιβλίου εργασίας (δημιουργία αρχείου Excel)
Workbook workbook = new Workbook();
```

**2. Προσθήκη και διαμόρφωση φύλλου εργασίας:**
```csharp
// Προσθήκη νέου φύλλου εργασίας στο Βιβλίο εργασίας
int sheetIndex = workbook.Worksheets.Add();

// Λάβετε αναφορά στο φύλλο εργασίας που μόλις προστέθηκε, περνώντας τον δείκτη του.
Worksheet worksheet = workbook.Worksheets[sheetIndex];

// Συμπλήρωση κελιών με δείγματα δεδομένων
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["B1"].PutValue(4);
worksheet.Cells["B2"].PutValue(20);
worksheet.Cells["B3"].PutValue(50);
```

### Προσθήκη και διαμόρφωση γραφήματος

**Επισκόπηση:**
Μάθετε πώς να προσθέσετε ένα γράφημα στο φύλλο εργασίας σας, να το ρυθμίσετε και να ορίσετε την πηγή δεδομένων του.

#### Προσθήκη του γραφήματος:
```csharp
using Aspose.Cells.Charts;

// Προσθήκη γραφήματος στηλών στο φύλλο εργασίας σε καθορισμένη θέση
int chartIndex = worksheet.Charts.Add(ChartType.Column, 5, 0, 15, 5);

// Πρόσβαση στην πρόσφατα προστιθέμενη παρουσία γραφήματος
Chart chart = worksheet.Charts[chartIndex];

// Ορισμός εύρους δεδομένων για τη συλλογή σειρών του γραφήματος (A1:B3)
chart.NSeries.Add("A1:B3", true);
```

### Μετατροπή γραφήματος σε μορφές εικόνας

**Επισκόπηση:**
Αυτή η λειτουργία καλύπτει τη μετατροπή γραφημάτων σε διάφορες μορφές εικόνας, συμπεριλαμβανομένων των EMF και Bitmap.

#### Μετατροπή και αποθήκευση εικόνων:
```csharp
using System.Drawing;
using Aspose.Cells.Rendering;

// Μετατρέψτε το γράφημα σε μορφή EMF και αποθηκεύστε το
chart.ToImage(outputDir + "/outputChartRendering.emf", Imaging.ImageFormat.Emf);

// Μετατρέψτε το γράφημα σε μορφή Bitmap και αποθηκεύστε το
Bitmap bitmap = chart.ToImage();
bmp.Save(outputDir + "/outputChartRendering.bmp", Imaging.ImageFormat.Bmp);
```

### Προηγμένες επιλογές μετατροπής εικόνας

**Επισκόπηση:**
Βελτιώστε την ποιότητα της εικόνας σας ορίζοντας προηγμένες επιλογές κατά τη μετατροπή.

#### Απόδοση υψηλής ποιότητας:
```csharp
using System.Drawing.Imaging;
using System.Drawing.Drawing2D;

// Δημιουργήστε μια παρουσία του ImageOrPrintOptions και ορίστε ιδιότητες για απόδοση υψηλής ποιότητας
ImageOrPrintOptions options = new ImageOrPrintOptions
{
    VerticalResolution = 300,
    HorizontalResolution = 300,
    SmoothingMode = SmoothingMode.AntiAlias
};

// Μετατροπή γραφήματος σε εικόνα με πρόσθετες ρυθμίσεις, αποθήκευση σε μορφή PNG
chart.ToImage(outputDir + "/outputChartRendering.png", options);
```

### Μετατροπή γραφήματος σε PDF

**Επισκόπηση:**
Μετατρέψτε τα γραφήματά σας απευθείας σε αρχείο PDF για εύκολη κοινή χρήση και εκτύπωση.

#### Αποθήκευση ως PDF:
```csharp
chart.ToPdf(outputDir + "/outputChartRendering.pdf");
```
