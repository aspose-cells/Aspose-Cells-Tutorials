## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα περιγράψουμε τα βήματα που απαιτούνται για την ανάγνωση και τον χειρισμό γραφημάτων σε ένα αρχείο Excel 2016.

### Πρόσβαση σε γραφήματα σε ένα φύλλο εργασίας

Ξεκινάμε φορτώνοντας το βιβλίο εργασίας πηγαίου κώδικα και αποκτώντας πρόσβαση στο πρώτο φύλλο εργασίας του, το οποίο περιέχει τα γραφήματά μας:

```csharp
// Φόρτωση του αρχείου Excel
Workbook book = new Workbook("sampleReadAndManipulateExcel2016Charts.xlsx");

// Πρόσβαση στο πρώτο φύλλο εργασίας
Worksheet sheet = book.Worksheets[0];
```

### Τύποι γραφημάτων ανάγνωσης

Στη συνέχεια, εξετάζουμε επανειλημμένα κάθε γράφημα στο φύλλο εργασίας για να διαβάσουμε τον τύπο του και να τον εκτυπώσουμε:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    // Λήψη του τρέχοντος γραφήματος
    Chart ch = sheet.Charts[i];

    // Εκτύπωση του τύπου γραφήματος
    Console.WriteLine(ch.Type);
}
```

### Τροποποίηση τίτλων γραφημάτων

Μπορούμε να αλλάξουμε τον τίτλο κάθε γραφήματος ώστε να αντικατοπτρίζει τον τύπο του:

```csharp
for (int i = 0; i < sheet.Charts.Count; i++)
{
    Chart ch = sheet.Charts[i];

    // Ενημέρωση τίτλου γραφήματος
    ch.Title.Text = "Chart Type is " + ch.Type.ToString();
}
```

### Αποθήκευση αλλαγών

Τέλος, αποθηκεύστε τις τροποποιήσεις σας σε ένα νέο αρχείο Excel:

```csharp
book.Save("outputReadAndManipulateExcel2016Charts.xlsx");
Console.WriteLine("Manipulation completed successfully.");
```
