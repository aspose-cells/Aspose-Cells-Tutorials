## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε βήμα προς βήμα την αντιγραφή σχημάτων μεταξύ φύλλων εργασίας.

### Βήμα 1: Άνοιγμα ενός υπάρχοντος βιβλίου εργασίας
Ξεκινήστε δημιουργώντας ένα αντικείμενο βιβλίου εργασίας από το αρχείο προέλευσης Excel. Από εδώ θα έχετε πρόσβαση στα σχήματα που θα αντιγραφούν.
```csharp
// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας και ανοίξτε το αρχείο προτύπου
Workbook workbook = new Workbook(sourceDir + "sampleCopyControls.xlsx");
```

### Βήμα 2: Πρόσβαση σε σχήματα στο φύλλο εργασίας πηγαίου κώδικα
Αποκτήστε πρόσβαση στη συλλογή σχημάτων από το φύλλο εργασίας προέλευσης. Εδώ, στοχεύουμε στο φύλλο εργασίας "Φύλλο1" για να ανακτήσουμε τα σχήματά του.
```csharp
// Λήψη των σχημάτων από το φύλλο εργασίας "Έλεγχος"
Aspose.Cells.Drawing.ShapeCollection shapes = workbook.Worksheets["Sheet1"].Shapes;
```

### Βήμα 3: Αντιγραφή συγκεκριμένων σχημάτων
Τώρα, ας αντιγράψουμε συγκεκριμένα σχήματα (όπως ένα πλαίσιο κειμένου ή ένα οβάλ) σε ένα άλλο φύλλο εργασίας. Θα προσθέσουμε αυτά τα αντίγραφα σε συγκεκριμένες θέσεις.
```csharp
// Αντιγραφή του πλαισίου κειμένου στο φύλλο εργασίας αποτελεσμάτων
workbook.Worksheets["Result"].Shapes.AddCopy(shapes[0], 5, 0, 2, 0);

// Αντιγράψτε το οβάλ σχήμα στο φύλλο εργασίας αποτελεσμάτων
workbook.Worksheets["Result"].Shapes.AddCopy(shapes[1], 10, 0, 2, 0);
```
- **Παράμετροι**: Το `AddCopy` Η μέθοδος λαμβάνει παραμέτρους για τη θέση και το μέγεθος. Προσαρμόστε τες με βάση τις ανάγκες σας.

### Βήμα 4: Αποθήκευση του βιβλίου εργασίας
Τέλος, αποθηκεύστε το βιβλίο εργασίας για να διατηρήσετε τις αλλαγές σας.
```csharp
// Αποθήκευση του Φύλλου Εργασίας
workbook.Save(outputDir + "outputCopyControls.xlsx");
```
