## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε τρία κύρια χαρακτηριστικά: προσθήκη κειμένου, σχολίων και εικόνων σε κελιά του Excel.

### Προσθήκη κειμένου σε ένα κελί του Excel

**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα νέο βιβλίο εργασίας και να προσθέσετε κείμενο στο κελί A1.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργία αντικειμένου βιβλίου εργασίας**

```csharp
// Δημιουργήστε μια νέα παρουσία της κλάσης Βιβλίου Εργασίας
Workbook workbook = new Workbook();
```

**2. Προσθήκη κειμένου στο κελί A1**

```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και εισαγάγετε κείμενο στο κελί A1
workbook.Worksheets[0].Cells["A1"].PutValue("Here");
```

**3. Αποθήκευση του βιβλίου εργασίας**

```csharp
// Αποθήκευση του βιβλίου εργασίας σας ως αρχείο Excel
workbook.Save(outputDir + "outputAddTextToCell.xlsx", Aspose.Cells.SaveFormat.Xlsx);
```

### Προσθήκη σχολίου στο κελί A1

**Επισκόπηση:** Μάθετε πώς να προσθέτετε και να προσαρμόζετε σχόλια στα φύλλα εργασίας σας.

#### Βήμα προς βήμα εφαρμογή

**1. Αποκτήστε πρόσβαση στη Συλλογή Σχολίων**

```csharp
// Πρόσβαση σε σχόλια του πρώτου φύλλου εργασίας
CommentCollection comments = workbook.Worksheets[0].Comments;
```

**2. Προσθήκη σχολίου στο κελί A1**

```csharp
// Εισαγωγή νέου σχολίου στο κελί A1 και ορισμός του κειμένου σημείωσής του
int commentIndex = comments.Add(0, 0);
Comment comment = comments[commentIndex];
comment.Note = "First note.";
comment.Font.Name = "Times New Roman";
```

**3. Αποθήκευση του βιβλίου εργασίας**

```csharp
// Αποθήκευση του βιβλίου εργασίας με το νέο σχόλιο
workbook.Save(outputDir + "outputAddCommentToCell.xlsx", Aspose.Cells.SaveFormat.Xlsx);
```

### Προσθήκη εικόνας σε σχόλιο του Excel

**Επισκόπηση:** Αυτή η λειτουργία δείχνει την προσθήκη μιας εικόνας ως φόντο στο σχόλιο ενός κελιού.

#### Βήμα προς βήμα εφαρμογή

**1. Φορτώστε την εικόνα σε μια ροή**

```csharp
// Φορτώστε το αρχείο εικόνας σας σε μια ροή (βεβαιωθείτε ότι έχετε τη σωστή διαδρομή)
Bitmap bmp = new Bitmap(SourceDir + "sampleAddPictureToExcelComment.jpg");
MemoryStream ms = new MemoryStream();
bmp.Save(ms, ImageFormat.Png);
```

**2. Ορισμός εικόνας ως φόντο σχολίου**

```csharp
// Αντιστοίχιση των δεδομένων εικόνας που έχουν φορτωθεί στο φόντο του σχήματος σχολίου
comment.CommentShape.Fill.ImageData = ms.ToArray();
```

**3. Αποθήκευση του βιβλίου εργασίας**

```csharp
// Αποθηκεύστε το βιβλίο εργασίας σας με την εικόνα που προσθέσατε στο σχόλιο
workbook.Save(outputDir + "outputAddPictureToExcelComment.xlsx", Aspose.Cells.SaveFormat.Xlsx);
```
