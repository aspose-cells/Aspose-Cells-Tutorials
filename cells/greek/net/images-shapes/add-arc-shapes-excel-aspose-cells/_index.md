Εάν χρειάζεται να βεβαιωθείτε ότι υπάρχει ένας κατάλογος εξόδου πριν από την αποθήκευση αρχείων, χρησιμοποιήστε αυτήν την απλή μέθοδο:

### Λειτουργία 2: Προσθήκη σχήματος τόξου στο Excel

Για να προσθέσετε ένα βασικό σχήμα τόξου στο βιβλίο εργασίας του Excel, ακολουθήστε τα εξής βήματα:
```csharp
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;

// Δημιουργήστε ένα νέο Βιβλίο Εργασίας.
Workbook excelbook = new Workbook();

// Προσθέστε ένα σχήμα τόξου στο πρώτο φύλλο εργασίας.
ArcShape arc1 = excelbook.Worksheets[0].Shapes.AddArc(2, 0, 2, 0, 130, 130);

// Ορισμός ιδιοτήτων του τόξου
arc1.Fill.FillType = FillType.Solid;
arс1.Fill.SolidFill.Color = Color.Blue;

c1.Placement = PlacementType.FreeFloating;
c1.Line.Weight = 1; // Πάχος γραμμής
c1.Line.DashStyle = MsoLineDashStyle.Solid; // Στυλ ταμπλό
```

**Βασικές επιλογές διαμόρφωσης:**
- **`AddArc`:** Προσθέτει ένα τόξο με καθορισμένες διαστάσεις και γωνίες.
- **Ιδιότητες πλήρωσης:** Χρήση `FillType.Solid` για ένα συμπαγές χρώμα γεμίσματος.
- **Τύπος τοποθέτησης:** `FreeFloating` επιτρέπει στο σχήμα να κινείται ελεύθερα μέσα στο φύλλο εργασίας.

### Χαρακτηριστικό 3: Προσθήκη άλλου σχήματος τόξου με προσαρμοσμένες ιδιότητες γραμμής

// Προσθήκη άλλου σχήματος τόξου
