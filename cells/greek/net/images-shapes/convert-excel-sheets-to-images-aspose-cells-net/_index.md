## Οδηγός Εφαρμογής

Θα αναλύσουμε κάθε χαρακτηριστικό της μετατροπής φύλλων Excel σε εικόνες χρησιμοποιώντας λογικές ενότητες.

### Φόρτωση και άνοιγμα ενός βιβλίου εργασίας του Excel

**Επισκόπηση:**
Το πρώτο βήμα στη διαδικασία μας είναι η φόρτωση ενός υπάρχοντος βιβλίου εργασίας του Excel από έναν καθορισμένο κατάλογο. Αυτό μας επιτρέπει να έχουμε πρόσβαση στα δεδομένα που θέλουμε να μετατρέψουμε σε εικόνες.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του αρχείου Excel σε ένα αντικείμενο βιβλίου εργασίας
Workbook book = new Workbook(SourceDir + "sampleConvertWorksheettoImageFile.xlsx");
```

**Εξήγηση:**
- `Workbook`Αντιπροσωπεύει ολόκληρο το βιβλίο εργασίας και παρέχει πρόσβαση στα φύλλα εργασίας του.
- Ο κατασκευαστής λαμβάνει τη διαδρομή του αρχείου Excel ως όρισμα, φορτώνοντάς το στη μνήμη.

### Πρόσβαση σε φύλλο εργασίας από βιβλίο εργασίας

**Επισκόπηση:**
Αφού ανοίξουμε το βιβλίο εργασίας, πρέπει να καθορίσουμε ποιο φύλλο εργασίας θέλουμε να μετατρέψουμε. Αυτή η ενότητα δείχνει την πρόσβαση σε ένα συγκεκριμένο φύλλο μέσα στο βιβλίο εργασίας.

```csharp
// Άνοιγμα του αρχείου Excel σε ένα αντικείμενο βιβλίου εργασίας
Workbook book = new Workbook(SourceDir + "sampleConvertWorksheettoImageFile.xlsx");

// Πρόσβαση στο πρώτο φύλλο εργασίας από το βιβλίο εργασίας
Worksheet sheet = book.Worksheets[0];
```

**Εξήγηση:**
- `Worksheets`: Μια συλλογή εντός του `Workbook` που αποθηκεύει όλα τα φύλλα.
- `sheet.Worksheets[0]`Ανακτά το πρώτο φύλλο εργασίας (ευρετήριο 0) στο βιβλίο εργασίας.

### Ρύθμιση παραμέτρων επιλογών εκτύπωσης εικόνας

**Επισκόπηση:**
Πριν από την απόδοση, ρυθμίζουμε τον τρόπο με τον οποίο το φύλλο εργασίας θα μετατραπεί σε εικόνα. Αυτό περιλαμβάνει τον ορισμό μορφών εξόδου και επιλογών σελίδας.

```csharp
// Ρύθμιση παραμέτρων εικόνας ή επιλογής εκτύπωσης για απόδοση
ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
imgOptions.OnePagePerSheet = true; // Απόδοση ολόκληρου του φύλλου εργασίας σε μία σελίδα
imgOptions.ImageType = Drawing.ImageType.Jpeg; // Ορίστε τον τύπο εικόνας εξόδου σε JPEG
```

**Εξήγηση:**
- `OnePagePerSheet`Εξασφαλίζει ότι ολόκληρο το φύλλο αποδίδεται σε μία μόνο εικόνα.
- `ImageType`: Καθορίζει τη μορφή της εικόνας εξόδου, σε αυτήν την περίπτωση, JPEG.

### Απόδοση ενός φύλλου εργασίας ως εικόνας

**Επισκόπηση:**
Τώρα μετατρέπουμε το καθορισμένο φύλλο εργασίας σε εικόνα χρησιμοποιώντας τις επιλογές που ορίσαμε προηγουμένως.

```csharp
// Δημιουργήστε ένα αντικείμενο SheetRender για να αποδώσετε το φύλλο εργασίας ως εικόνα
SheetRender sr = new SheetRender(sheet, imgOptions);
Bitmap bitmap = sr.ToImage(0); // Απόδοση της πρώτης σελίδας του φύλλου σε εικόνα
```

**Εξήγηση:**
- `SheetRender`: Χειρίζεται λειτουργίες απόδοσης για φύλλα εργασίας.
- `ToImage(int pageIndex)`Μετατρέπει μια συγκεκριμένη σελίδα φύλλου εργασίας σε εικόνα.

### Αποθήκευση της εικόνας που έχει αποδοθεί

**Επισκόπηση:**
Τέλος, αποθηκεύστε την εικόνα που δημιουργήθηκε στον επιθυμητό κατάλογο εξόδου.

```csharp
// Αποθηκεύστε την εικόνα που αποδόθηκε στον κατάλογο εξόδου
bitmap.Save(outputDir + "outputConvertWorksheettoImageFile.jpg");
```

**Εξήγηση:**
- `Save(string path)`: Εγγράφει το αρχείο εικόνας στον δίσκο στην καθορισμένη θέση.
