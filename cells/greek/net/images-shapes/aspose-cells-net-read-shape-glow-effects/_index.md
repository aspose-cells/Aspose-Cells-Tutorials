## Οδηγός Εφαρμογής
Ας χωρίσουμε τη διαδικασία υλοποίησης σε εύπεπτα μέρη.

### Εφέ λάμψης σχήματος ανάγνωσης
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε και να αναλύσετε το εφέ λάμψης που εφαρμόζεται σε σχήματα μέσα σε ένα αρχείο Excel. 

#### Βήμα 1: Διαβάστε το αρχείο προέλευσης Excel
Ξεκινήστε φορτώνοντας το έγγραφο Excel:
```csharp
string sourceDir = "YourDirectoryPath";
Workbook book = new Workbook(sourceDir + "sampleReadColorOfShapesGlowEffect.xlsx");
```

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας και το Σχήμα
Μεταβείτε στο συγκεκριμένο φύλλο εργασίας και σχήμα που θέλετε να εξετάσετε:
```csharp
Worksheet sheet = book.Worksheets[0];
Shape shape = sheet.Shapes[0];
```

#### Βήμα 3: Ιδιότητες εφέ λάμψης εξαγωγής
Αποκτήστε πρόσβαση στις ιδιότητες του εφέ λάμψης του σχήματός σας:
```csharp
GlowEffect effect = shape.Glow;
CellsColor color = effect.Color;

Console.WriteLine("Color: " + color.Color);
Console.WriteLine("ColorIndex: " + color.ColorIndex);
Console.WriteLine("IsShapeColor: " + color.IsShapeColor);
Console.WriteLine("Transparency: " + color.Transparency);
Console.WriteLine("Type: " + color.Type);
```

**Εξήγηση**Αυτός ο κώδικας ανακτά τις λεπτομέρειες χρώματος του εφέ λάμψης, συμπεριλαμβανομένης της τιμής RGB, του δείκτη, του επιπέδου διαφάνειας και του τύπου του.
