### Χαρακτηριστικό 1: Φόρτωση και πρόσβαση σε αρχείο Excel

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο Excel και να αποκτήσετε πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας χρησιμοποιώντας το Aspose.Cells για .NET.

### Λειτουργία 2: Εξαγωγή κειμένου σχήματος SmartArt

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στην εξαγωγή κειμένου από σχήματα SmartArt τύπου γραναζιού μέσα σε ένα αρχείο Excel.

#### Βήματα:
**1. Εισαγωγή απαιτούμενων κλάσεων**
```java
import com.aspose.cells.Workbook;
import com.aspose.cells.Worksheet;
import com.aspose.cells.Shape;
import com.aspose.cells.GroupShape;
```

**2. Φόρτωση του Βιβλίου Εργασίας και του Φύλλου Εργασίας της Access**
Όπως και στη Λειτουργία 1, φορτώστε το βιβλίο εργασίας σας και αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε.

**3. Πρόσβαση στο σχήμα ομάδας SmartArt**
Υποθέτοντας ότι το πρώτο σχήμα είναι μια ομάδα SmartArt:
```java
Shape sh = ws.getShapes().get(0);
GroupShape gs = (GroupShape)sh.getResultOfSmartArt();
```

**4. Εξαγωγή κειμένου από σχήματα τύπου γραναζιού**
Επαναλάβετε τα σχήματα για να εξαγάγετε κείμενο από τύπους γραναζιών:
```java
Shape[] shps = gs.getGroupedShapes();

for (int i = 0; i < shps.length; i++) {
    Shape s = shps[i];
    
    if (s.getType() == com.aspose.cells.AutoShapeType.GEAR9 || 
        s.getType() == com.aspose.cells.AutoShapeType.GEAR6) {
        System.out.println("Gear Type Shape Text: " + s.getText());
    }
}
```
