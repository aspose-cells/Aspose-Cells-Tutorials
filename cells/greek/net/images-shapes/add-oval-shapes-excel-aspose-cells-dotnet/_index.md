## Implementation Guide

### Adding an Oval Shape

This feature guides you through adding a basic oval shape to an Excel worksheet.

#### Overview
Adding ovals can enhance the visual appeal of your data presentation. In this section, we'll add and configure an oval in the first worksheet of our Excel file using Aspose.Cells.

#### Steps:

##### Step 1: Define Directory for Output

First, define where you want to save your output files:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
string dataDir = Path.Combine(outputDir, "OvalShapeExample");

if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
```

##### Βήμα 2: Δημιουργία ενός βιβλίου εργασίας

Δημιουργήστε μια παρουσία του `Workbook` κλάση για να ξεκινήσετε να εργάζεστε με αρχεία Excel:

```csharp
Workbook excelbook = new Workbook();
```

##### Βήμα 3: Προσθήκη οβάλ σχήματος

Χρησιμοποιήστε το `AddOval` μέθοδος για την τοποθέτηση ενός οβάλ σχήματος στο φύλλο εργασίας:

```csharp
// Προσθήκη οβάλ σε καθορισμένες συντεταγμένες και μέγεθος
Oval oval1 = excelbook.Worksheets[0].Shapes.AddOval(2, 0, 2, 0, 130, 160);
```

##### Βήμα 4: Διαμόρφωση τοποθέτησης

Ορίστε τον τύπο τοποθέτησης σε `FreeFloating` για περισσότερο έλεγχο της τοποθέτησης:

```csharp
oval1.Placement = PlacementType.FreeFloating;
```

##### Βήμα 5: Ορισμός ιδιοτήτων γραμμής

Προσαρμόστε την εμφάνιση του περιγράμματος του οβάλ ορίζοντας το πάχος γραμμής και το στυλ παύλας:

```csharp
// Ορισμός πάχους γραμμής και στυλ παύλας
oval1.Line.Weight = 1;
oval1.Line.DashStyle = MsoLineDashStyle.Solid;
```

##### Βήμα 6: Αποθήκευση βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε ένα αρχείο στον καθορισμένο κατάλογο:

```csharp
excelbook.Save(Path.Combine(dataDir, "OvalShapeExample.xls"));
```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι όλες οι διαδρομές καταλόγων έχουν οριστεί σωστά για να αποτρέψετε σφάλματα "δεν βρέθηκε αρχείο".
- Ελέγξτε ότι το Aspose.Cells διαθέτει την κατάλληλη άδεια χρήσης, εάν χρησιμοποιείτε λειτουργίες πέρα από τους περιορισμούς της δοκιμαστικής έκδοσης.

### Προσθήκη ενός ακόμη οβάλ σχήματος (κύκλος)

Τώρα ας προσθέσουμε ένα άλλο οβάλ σχήμα, διαμορφωμένο ως κύκλος, με διαφορετικές ιδιότητες.

#### Επισκόπηση
Η προσθήκη πολλαπλών σχημάτων μπορεί να βοηθήσει στη δημιουργία πιο σύνθετων απεικονίσεων. Εδώ, θα δείξουμε την προσθήκη ενός κυκλικού οβάλ στο φύλλο εργασίας σας.

#### Βήματα:

##### Βήμα 1: Βεβαιωθείτε ότι υπάρχει κατάλογος

Αυτό το βήμα είναι παρόμοιο με την προηγούμενη ενότητα. Βεβαιωθείτε ότι ο κατάλογός σας έχει ρυθμιστεί σωστά.

```csharp
if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
```

##### Βήμα 2: Δημιουργία βιβλίου εργασίας

Δημιουργήστε ένα νέο `Workbook` παράδειγμα για αυτήν την πρόσθεση σχήματος:

```csharp
Workbook excelbook = new Workbook();
```

##### Βήμα 3: Προσθήκη σχήματος κύκλου

Προσθέστε ένα άλλο οβάλ με διαστάσεις για να φαίνεται ως κύκλος:

```csharp
// Προσθέστε ένα κυκλικό σχήμα σε διαφορετικές συντεταγμένες και μέγεθος
Oval oval2 = excelbook.Worksheets[0].Shapes.AddOval(9, 0, 2, 15, 130, 130);
```

##### Βήμα 4: Διαμόρφωση τοποθέτησης

Ορίστε τον τύπο τοποθέτησης για το νέο σχήμα:

```csharp
oval2.Placement = PlacementType.FreeFloating;
```

##### Βήμα 5: Ορισμός ιδιοτήτων γραμμής

Ορίστε το πάχος γραμμής και το στυλ παύλας για προσαρμογή:

```csharp
// Προσαρμογή ιδιοτήτων γραμμής
oval2.Line.Weight = 1;
oval2.Line.DashStyle = MsoLineDashStyle.Solid;
```

##### Βήμα 6: Αποθήκευση βιβλίου εργασίας με νέο σχήμα

Αποθηκεύστε ξανά το βιβλίο εργασίας, αυτή τη φορά συμπεριλαμβάνοντας και τα δύο σχήματα:

```csharp
excelbook.Save(Path.Combine(dataDir, "OvalShapeExampleWithCircle.xls"));
```
