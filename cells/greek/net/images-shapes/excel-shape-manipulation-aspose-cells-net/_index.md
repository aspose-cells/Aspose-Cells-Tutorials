### Χειρισμός της θέσης Z-Order των σχημάτων

#### Επισκόπηση
Η κατανόηση και ο χειρισμός της θέσης σειράς Z σάς επιτρέπει να ελέγχετε ποια σχήματα εμφανίζονται στην κορυφή σε επικαλυπτόμενα σενάρια. Αυτή η λειτουργία είναι κρίσιμη όταν ασχολείστε με σύνθετα φύλλα εργασίας που περιέχουν πολλά γραφικά αντικείμενα.

#### Πρόσβαση και ρύθμιση θέσεων σχήματος (H3)

Για να στείλετε ένα σχήμα μπροστά ή πίσω, ακολουθήστε τα εξής βήματα:

```csharp
// Φόρτωση αρχείου Excel πηγής
Workbook workbook = new Workbook("sampleToFrontOrBack.xlsx");

// Πρώτο φύλλο εργασίας της Access
Worksheet sheet = workbook.Worksheets[0];

// Πρόσβαση σε συγκεκριμένα σχήματα μέσω ευρετηρίου
Shape shape1 = sheet.Shapes[0];
Shape shape4 = sheet.Shapes[3];

// Εκτύπωση της τρέχουσας θέσης Z-Order του σχήματος
Console.WriteLine("Z-Order Shape 1: " + shape1.ZOrderPosition);

// Μετακίνηση αυτού του σχήματος προς τα εμπρός
shape1.ToFrontOrBack(2);

// Επαλήθευση νέας θέσης Z-Order
Console.WriteLine("New Z-Order Shape 4: " + shape4.ZOrderPosition);

// Στείλτε ένα άλλο σχήμα στο πίσω μέρος
shape4.ToFrontOrBack(-2);
```

**Εξήγηση**: 
- `ToFrontOrBack(int value)`Αυτή η μέθοδος προσαρμόζει την τάξη Z με βάση την παράμετρο. Ένας θετικός ακέραιος αριθμός μετακινεί το σχήμα προς τα εμπρός, ενώ ένας αρνητικός το στέλνει προς τα πίσω.

#### Αποθήκευση αλλαγών (H3)

Αφού χειριστείτε τα σχήματα, αποθηκεύστε τις αλλαγές σας για να βεβαιωθείτε ότι θα διατηρηθούν:

```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
workbook.Save("outputToFrontOrBack.xlsx");
```
