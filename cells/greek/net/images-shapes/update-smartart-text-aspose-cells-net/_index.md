### Λειτουργία 1: Φόρτωση βιβλίου εργασίας και επανάληψη σε φύλλα εργασίας
**Επισκόπηση:**
Μάθετε πώς να φορτώνετε ένα αρχείο Excel και να έχετε πρόσβαση σε κάθε φύλλο εργασίας για να χειρίζεστε το περιεχόμενό του.

#### Βήμα προς βήμα εφαρμογή:
##### Φόρτωση του βιβλίου εργασίας
Ξεκινήστε δημιουργώντας ένα `Workbook` αντικείμενο με τη διαδρομή του αρχείου προέλευσης:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook wb = new Workbook(SourceDir + "SmartArt.xlsx");
```

##### Επανάληψη μέσω φύλλων εργασίας και σχημάτων
Χρησιμοποιήστε ένθετους βρόχους για να αποκτήσετε πρόσβαση σε κάθε φύλλο εργασίας και στα σχήματά του, ορίζοντας εναλλακτικό κείμενο για προσαρμογή:

```csharp
foreach (Worksheet worksheet in wb.Worksheets)
{
    foreach (Shape shape in worksheet.Shapes)
    {
        shape.AlternativeText = "ReplacedAlternativeText";
        
        if (shape.IsSmartArt)
        {
            // Χειριστείτε εδώ τη λογική που αφορά συγκεκριμένα το SmartArt.
        }
    }
}
```

### Δυνατότητα 2: Χειρισμός σχημάτων SmartArt
**Επισκόπηση:**
Βυθιστείτε στην επεξεργασία και ενημέρωση κειμένου μέσα σε σχήματα SmartArt μέσω προγραμματισμού.

#### Βήμα προς βήμα εφαρμογή:
##### Επανάληψη μέσω σχημάτων SmartArt
Εντός των προηγουμένως δημιουργημένων βρόχων, εστιάστε σε σχήματα SmartArt για να τροποποιήσετε το περιεχόμενό τους:

```csharp
foreach (Worksheet worksheet in wb.Worksheets)
{
    foreach (Shape shape in worksheet.Shapes)
    {
        if (shape.IsSmartArt)
        {
            foreach (Shape smartart in shape.GetResultOfSmartArt().GetGroupedShapes())
            {
                smartart.Text = "ReplacedText"; // Ενημερώστε το κείμενο
            }
        }
    }
}
```

### Δυνατότητα 3: Αποθήκευση βιβλίου εργασίας με ενημερωμένα κείμενα SmartArt
**Επισκόπηση:**
Βεβαιωθείτε ότι οι αλλαγές σας αποθηκεύονται ρυθμίζοντας και αποθηκεύοντας σωστά το βιβλίο εργασίας.

#### Βήμα προς βήμα εφαρμογή:
##### Αποθήκευση του βιβλίου εργασίας
Χρήση `OoxmlSaveOptions` για να διευκρινιστεί ότι θα πρέπει να ληφθούν υπόψη οι ενημερώσεις SmartArt:
```csharp
Aspose.Cells.OoxmlSaveOptions options = new Aspose.Cells.OoxmlSaveOptions();
options.UpdateSmartArt = true;
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
wb.Save(OutputDir + "outputSmartArt.xlsx", options);
```
