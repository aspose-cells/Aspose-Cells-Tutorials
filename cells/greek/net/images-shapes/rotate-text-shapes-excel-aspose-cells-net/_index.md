- Πρακτικές εφαρμογές της περιστροφής κειμένου μέσα σε σχήματα

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη διαδικασία περιστροφής κειμένου μέσα σε ένα σχήμα σε ένα φύλλο εργασίας του Excel.

### Βήμα 1: Φόρτωση του αρχείου Excel
Ξεκινήστε φορτώνοντας το δείγμα αρχείου Excel:
```csharp
Workbook wb = new Workbook("sampleRotateTextWithShapeInsideWorksheet.xlsx");
```
Αυτό το βήμα αρχικοποιεί το αντικείμενο βιβλίου εργασίας που αντιπροσωπεύει το έγγραφο Excel σας.

### Βήμα 2: Πρόσβαση και τροποποίηση φύλλου εργασίας
Αποκτήστε πρόσβαση στο επιθυμητό φύλλο εργασίας όπου θέλετε να χειριστείτε σχήματα και κείμενο:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell b4 = ws.Cells["B4"];
b4.PutValue("Text is not rotating with shape because RotateTextWithShape is false.");
```

### Βήμα 3: Ρύθμιση παραμέτρων ιδιοτήτων σχήματος
Αποκτήστε πρόσβαση στο πρώτο σχήμα στο φύλλο εργασίας σας για να τροποποιήσετε τις ιδιότητες κειμένου του:
```csharp
Shape sh = ws.Shapes[0];
Aspose.Cells.Drawing.Texts.ShapeTextAlignment shapeTextAlignment = sh.TextBody.TextAlignment;
shapeTextAlignment.RotateTextWithShape = false; // Ορίστε αυτήν την τιμή σε true εάν θέλετε περιστροφή κειμένου με το σχήμα.
```
Αυτή η ρύθμιση παραμέτρων καθορίζει εάν το κείμενο περιστρέφεται μαζί με το σχήμα.

### Βήμα 4: Αποθήκευση των αλλαγών σας
Αφού κάνετε τις αλλαγές σας, αποθηκεύστε το βιβλίο εργασίας:
```csharp
wb.Save("outputRotateTextWithShapeInsideWorksheet.xlsx");
Console.WriteLine("Rotation executed successfully.");
```
