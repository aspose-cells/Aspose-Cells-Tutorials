## Οδηγός Εφαρμογής

### Ανανέωση συνδεδεμένων σχημάτων σε γραφήματα Excel

Η ανανέωση συνδεδεμένων σχημάτων περιλαμβάνει την ενημέρωση των πηγών δεδομένων για τα γραφήματα. Αυτή η ενότητα παρέχει έναν λεπτομερή οδηγό υλοποίησης.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας
Ξεκινήστε φορτώνοντας το αρχείο Excel που περιέχει το γράφημα και τα συνδεδεμένα σχήματα.

```csharp
// Κατάλογος πηγής όπου βρίσκεται το αρχείο δείγματος
string sourceDir = RunExamples.Get_SourceDirectory();

// Δημιουργία βιβλίου εργασίας από το αρχείο προέλευσης
Workbook workbook = new Workbook(sourceDir + "sampleRefreshValueOfLinkedShapes.xlsx");
```

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας
Αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει το διάγραμμά σας.

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

#### Βήμα 3: Ενημέρωση τιμών κελιών
Αλλάξτε την τιμή ενός κελιού που είναι συνδεδεμένο με το σχήμα ή το γράφημα.

```csharp
// Αλλαγή της τιμής του κελιού B4
Cell cell = worksheet.Cells["B4"];
cell.PutValue(100);
```

#### Βήμα 4: Ανανέωση συνδεδεμένων σχημάτων
Ενημερώστε την τιμή της συνδεδεμένης εικόνας χρησιμοποιώντας τις μεθόδους Aspose.Cells.

```csharp
// Ενημέρωση της τιμής της συνδεδεμένης εικόνας που συνδέεται με το κελί B4
worksheet.Shapes.UpdateSelectedValue();
```

#### Βήμα 5: Αποθήκευση του βιβλίου εργασίας
Αποθηκεύστε τις αλλαγές σας και εκτυπώστε τις σε διαφορετική μορφή, εάν χρειάζεται, όπως PDF.

```csharp
// Κατάλογος εξόδου για την αποθήκευση αρχείων
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση του βιβλίου εργασίας σε μορφή PDF
workbook.Save(outputDir + "outputRefreshValueOfLinkedShapes.pdf", SaveFormat.Pdf);
```
