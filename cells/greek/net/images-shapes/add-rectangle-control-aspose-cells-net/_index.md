## Οδηγός Εφαρμογής
Τώρα που καλύψαμε τη ρύθμιση, ας υλοποιήσουμε την προσθήκη ενός ορθογωνίου στοιχείου ελέγχου μέσα σε ένα βιβλίο εργασίας του Excel χρησιμοποιώντας C#.

### Δημιουργία και διαμόρφωση ενός ορθογώνιου στοιχείου ελέγχου
#### Επισκόπηση
Η προσθήκη ενός στοιχείου ελέγχου ορθογωνίου περιλαμβάνει τη δημιουργία ενός νέου σχήματος στο φύλλο εργασίας και την προσαρμογή των ιδιοτήτων του, όπως η τοποθέτηση, το μέγεθος, το πάχος γραμμής και το στυλ παύλας.

#### Οδηγός βήμα προς βήμα
**1. Δημιουργήστε ένα βιβλίο εργασίας**
Ξεκινήστε δημιουργώντας μια παρουσία του `Workbook` τάξη:
```csharp
// Δημιουργία νέας παρουσίας βιβλίου εργασίας
Workbook excelbook = new Workbook();
```

**2. Προσθήκη ορθογωνίου σχήματος**
Χρησιμοποιήστε το `AddRectangle` Μέθοδος για την εισαγωγή ενός ορθογωνίου σχήματος στο φύλλο εργασίας σας:
```csharp
// Προσθήκη ενός ορθογωνίου ελέγχου σε καθορισμένη θέση και μέγεθος
Aspose.Cells.Drawing.RectangleShape rectangle = excelbook.Worksheets[0].Shapes.AddRectangle(3, 0, 2, 0, 70, 130);
```
- **Παράμετροι**: Οι παράμετροι `(3, 0, 2, 0, 70, 130)` ορίστε τον δείκτη γραμμής, τον δείκτη στήλης, το πλάτος και το ύψος του ορθογωνίου σε σημεία.

**3. Ορισμός τοποθέτησης**
Ορίστε πού θα τοποθετηθεί το ορθογώνιο σας μέσα στο φύλλο εργασίας:
```csharp
// Ορισμός τοποθέτησης σε ελεύθερη κυμαινόμενη ρύθμιση
rectangle.Placement = Τύπος τοποθέτησης.FreeFloating;
```
- **PlacementType**Το FreeFloating επιτρέπει την κίνηση χωρίς ευθυγράμμιση με τα κελιά.

**4. Προσαρμόστε την εμφάνιση**
Ρυθμίστε τις οπτικές ιδιότητες όπως το πάχος γραμμής και το στυλ παύλας για καλύτερη ορατότητα:
```csharp
// Τροποποίηση της εμφάνισης του ορθογωνίου
rectangle.Line.Weight = 4; // Ορίστε το πάχος γραμμής
rectangle.Line.DashStyle = MsoLineDashStyle.Solid; // Ορίστε το στυλ παύλας ως συμπαγές
```
- **Βάρος**: Καθορίζει το πάχος του περιγράμματος του σχήματος.
- **Στυλ Dash**: Ορίζει το μοτίβο των παυλών και των κενών που χρησιμοποιούνται για την επισήμανση διαδρομών.

**5. Αποθήκευση του βιβλίου εργασίας**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με το νέο στοιχείο ελέγχου ορθογωνίου που προστέθηκε:
```csharp
// Αποθήκευση αλλαγών σε νέο αρχείο
excelbook.Save(dataDir + "book1.out.xls");
```
