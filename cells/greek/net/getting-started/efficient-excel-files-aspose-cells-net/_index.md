## Οδηγός Εφαρμογής

Θα δούμε τώρα πώς να δημιουργούμε πολλά αρχεία Excel χρησιμοποιώντας το Aspose.Cells για .NET.

### Δημιουργία πολλαπλών αρχείων Excel

Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε αποτελεσματικά πολλά βιβλία εργασίας του Excel:

#### Βήμα 1: Αρχικοποίηση του βιβλίου εργασίας

Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Workbook`Αυτό το αντικείμενο αντιπροσωπεύει ένα ολόκληρο αρχείο Excel.

```csharp
Workbook workbook = new Workbook();
```

#### Βήμα 2: Ρύθμιση παραμέτρων φύλλων εργασίας

Κάθε βιβλίο εργασίας μπορεί να περιέχει πολλά φύλλα εργασίας. Εδώ, θα προσθέσουμε πέντε φύλλα εργασίας σε κάθε βιβλίο εργασίας:

```csharp
// Καταργήστε το προεπιλεγμένο φύλλο εργασίας.
workbook.Worksheets.RemoveAt(0);

for (int sheetIndex = 0; sheetIndex < 5; sheetIndex++)
{
    Worksheet ws = workbook.Worksheets[workbook.Worksheets.Add()];
    ws.Name = sheetIndex.ToString();
    
    for (int row = 0; row < 150; row++)
    {
        for (int col = 0; col < 50; col++)
        {
            ws.Cells[row, col].PutValue($"row{row} col{col}");
        }
    }
}
```

#### Βήμα 3: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε κάθε βιβλίο εργασίας ως αρχείο Excel στον επιθυμητό κατάλογο εξόδου:

```csharp
string filePath = System.IO.Path.Combine(outputDir, $"{fileIndex}_out.xls");
workbook.Save(filePath);
```
