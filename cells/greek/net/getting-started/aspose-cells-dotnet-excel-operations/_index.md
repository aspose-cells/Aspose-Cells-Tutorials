## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία Βιβλίου Εργασίας και Υπολογισμός Τύπου

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο Excel και να υπολογίσετε τυχόν τύπους που περιέχει.

#### Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας

Αρχικά, δημιουργήστε ένα `Workbook` αντικείμενο από την καθορισμένη διαδρομή αρχείου Excel:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleFindingCellsWithStringOrNumber.xlsx");
```

#### Βήμα 2: Υπολογισμός τύπων στο φορτωμένο βιβλίο εργασίας

Καλέστε το `CalculateFormula` μέθοδος για την επεξεργασία όλων των τύπων στο βιβλίο εργασίας:

```csharp
workbook.CalculateFormula();
```

### Χαρακτηριστικό 2: Εύρεση κελιού με ακέραια ή διπλή τιμή

Αυτή η λειτουργία δείχνει πώς να εντοπίσετε κελιά που περιέχουν ακέραιες ή διπλές τιμές.

#### Βήμα 1: Πρόσβαση στη Συλλογή Κελιών

Λάβετε τα κελιά από το πρώτο φύλλο εργασίας του βιβλίου εργασίας σας:

```csharp
Cells cells = workbook.Worksheets[0].Cells;
```

#### Βήμα 2: Ρύθμιση FindOptions και Αναζήτηση για κελί

Δημιουργώ `FindOptions` για να καθορίσετε κριτήρια αναζήτησης και, στη συνέχεια, βρείτε το κελί με μια καθορισμένη τιμή (π.χ., ακέραιος αριθμός 224):

```csharp
FindOptions opts = new FindOptions();
opts.LookInType = LookInType.Values;
opts.LookAtType = LookAtType.EntireContent;

Cell cell1 = cells.Find(224, null, opts);
if (cell1 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell1.Name);
}
else
{
    Console.WriteLine("Record not found");
}
```

### Χαρακτηριστικό 3: Εύρεση κελιού με ακριβή τιμή συμβολοσειράς

Βρείτε ένα κελί που ταιριάζει ακριβώς με μια δεδομένη συμβολοσειρά.

#### Βήμα 1: Ρύθμιση του FindOptions για Ακριβή αντιστοίχιση

Χρησιμοποιώντας `LookAtType` στρώνομαι στη δουλειά `EntireContent`αναζητήστε μια ακριβή τιμή συμβολοσειράς:

```csharp
opts.LookInType = LookInType.Values;
opts.LookAtType = LookAtType.EntireContent;

Aspose.Cells.Cell cell2 = cells.Find("Items E", null, opts);
if (cell2 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell2.Name);
}
else
{
    Console.WriteLine("Record not found");
}
```

### Χαρακτηριστικό 4: Εύρεση κελιού με συμβολοσειρά που περιέχει μια συγκεκριμένη δευτερεύουσα συμβολοσειρά

Εντοπίστε κελιά που περιέχουν μια συγκεκριμένη δευτερεύουσα συμβολοσειρά μέσα στο περιεχόμενό τους.

#### Βήμα 1: Ρύθμιση παραμέτρων FindOptions για αναζήτηση δευτερευουσών συμβολοσειρών

Σειρά `LookAtType` να `Contains` και αναζητήστε τη δευτερεύουσα συμβολοσειρά "Δεδομένα":

```csharp
opts.LookInType = LookInType.Values;
opts.LookAtType = LookAtType.Contains;

Cell cell3 = cells.Find("Data", null, opts);
if (cell3 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell3.Name);
}
else
{
    Console.WriteLine("Record not found");
}
```
