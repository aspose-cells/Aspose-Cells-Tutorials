# Πώς να Ενσωματώσετε Γραμματοσειρές σε HTML από το Excel – Πλήρης Οδηγός

Έχετε αναρωτηθεί ποτέ **πώς να ενσωματώσετε γραμματοσειρές** σε ένα αρχείο HTML που δημιουργείται από ένα βιβλίο εργασίας Excel; Δεν είστε οι μόνοι. Πολλοί προγραμματιστές αντιμετωπίζουν το πρόβλημα όταν το εξαγόμενο HTML φαίνεται σωστό στον υπολογιστή τους, αλλά χάνει την αρχική τυπογραφία σε άλλη συσκευή. Τα καλά νέα; Η λύση είναι αρκετά απλή με το Aspose.Cells, και μπορείτε να έχετε τις γραμματοσειρές ενσωματωμένες απευθείας στο HTML αποτέλεσμα.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τις ακριβείς ενέργειες για **ενσωμάτωση γραμματοσειρών σε html**, θα σας δείξουμε πώς να **εξάγετε Excel σε html**, και τέλος θα αποδείξουμε πώς να **αποθηκεύσετε το βιβλίο εργασίας ως html** με όλες τις απαραίτητες ρυθμίσεις. Στο τέλος θα έχετε ένα έτοιμο αρχείο HTML που αποδίδει ακριβώς όπως το αρχικό φύλλο εργασίας—χωρίς ελλιπείς χαρακτήρες, χωρίς εναλλακτικές γραμματοσειρές.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί επίσης με .NET Framework)
- Aspose.Cells for .NET (δωρεάν δοκιμή ή αδειοδοτημένη έκδοση)
- Ένα δείγμα αρχείου Excel (`sample.xlsx`) που χρησιμοποιεί τουλάχιστον μία προσαρμοσμένη γραμματοσειρά
- Visual Studio 2022 ή οποιονδήποτε επεξεργαστή C# προτιμάτε

Δεν απαιτούνται επιπλέον πακέτα NuGet εκτός από το Aspose.Cells.

## Βήμα 1: Ρύθμιση του Έργου και Φόρτωση του Βιβλίου Εργασίας

Πρώτα απ' όλα—δημιουργήστε μια νέα εφαρμογή console και προσθέστε την αναφορά στο Aspose.Cells.

```csharp
using System;
using Aspose.Cells;

namespace ExcelToHtmlWithFonts
{
    class Program
    {
        static void Main(string[] args)
        {
            // Load an existing Excel workbook
            string excelPath = @"C:\Temp\sample.xlsx";
            Workbook workbook = new Workbook(excelPath);
            
            // We'll configure the export options in the next step
        }
    }
}
```

**Γιατί είναι σημαντικό:** Η φόρτωση του βιβλίου εργασίας είναι το θεμέλιο. Αν το βιβλίο εργασίας δεν φορτωθεί σωστά, καμία από τις επόμενες ρυθμίσεις ενσωμάτωσης γραμματοσειρών δεν θα έχει αποτέλεσμα. Επίσης, σημειώστε ότι το Aspose.Cells διαβάζει αυτόματα τις πληροφορίες γραμματοσειράς που αποθηκεύονται στο αρχείο, οπότε δεν χρειάζεται να καθορίσετε χειροκίνητα τα ονόματα των γραμματοσειρών.

## Βήμα 2: Δημιουργία HtmlSaveOptions και Ενεργοποίηση Ενσωμάτωσης Γραμματοσειρών

Τώρα δημιουργούμε ένα αντικείμενο `HtmlSaveOptions` και ενεργοποιούμε τη σημαία `EmbedAllFonts`. Αυτό λέει στο Aspose.Cells να ενσωματώσει κάθε γραμματοσειρά που αναφέρεται στο βιβλίο εργασίας απευθείας στο παραγόμενο HTML.

```csharp
// Step 2: Create HTML save options
HtmlSaveOptions htmlSaveOptions = new HtmlSaveOptions();

// Enable embedding of all fonts in the output HTML
htmlSaveOptions.EmbedAllFonts = true;

// Optional: Reduce the size of the generated HTML by using base64 encoding
htmlSaveOptions.ExportEmbeddedImages = true;
```

**Γιατί ενεργοποιούμε το `EmbedAllFonts`:** Όταν εξάγετε το Excel σε HTML χωρίς αυτή τη σημαία, το HTML αναφέρεται στις γραμματοσειρές μόνο με το όνομα. Αν το σύστημα του χρήστη δεν έχει αυτές τις γραμματοσειρές εγκατεστημένες, ο περιηγητής θα επιστρέψει σε μια γενική οικογένεια, χαλώντας τη διάταξη. Η ενσωμάτωση εγγυάται ότι τα ακριβή γλυφά θα μεταφερθούν μαζί με το αρχείο HTML.

**Συμβουλή επαγγελματία:** Αν χρειάζεστε μόνο ένα υποσύνολο γραμματοσειρών (π.χ. ξέρετε ότι το βιβλίο εργασίας χρησιμοποιεί μόνο *Calibri* και *Arial*), μπορείτε να ορίσετε το `htmlSaveOptions.FontsList` σε μια προσαρμοσμένη συλλογή. Αυτό μπορεί να μειώσει δραματικά το τελικό μέγεθος του αρχείου.

## Βήμα 3: Αποθήκευση του Βιβλίου Εργασίας ως HTML με Ενσωματωμένες Γραμματοσειρές

Τέλος, καλέστε τη μέθοδο `Save` στο αντικείμενο `Workbook`, περνώντας τη διαδρομή και τις επιλογές που μόλις διαμορφώσαμε.

```csharp
// Step 3: Save the workbook as an HTML file with embedded fonts
string htmlPath = @"C:\Temp\embedded.html";
workbook.Save(htmlPath, htmlSaveOptions);

Console.WriteLine($"HTML file with embedded fonts saved to: {htmlPath}");
```

Αυτό είναι—το `embedded.html` σας περιέχει τώρα μπλοκ `<style>` με ορισμούς `@font-face` και δεδομένα γραμματοσειράς κωδικοποιημένα σε base64. Ανοίξτε το σε οποιονδήποτε σύγχρονο περιηγητή και θα δείτε την ίδια τυπογραφία όπως στο `sample.xlsx`.

### Αναμενόμενο Αποτέλεσμα

Όταν ανοίξετε το `embedded.html`:

- Η προσαρμοσμένη γραμματοσειρά εμφανίζεται ακριβώς όπως στο Excel.
- Δεν ζητούνται εξωτερικά αρχεία γραμματοσειρών (ελέγξτε την καρτέλα Network στα dev tools—δεν πρέπει να φορτωθεί τίποτα).
- Το μέγεθος της σελίδας μπορεί να είναι μεγαλύτερο από μια απλή εξαγωγή HTML, αλλά η οπτική πιστότητα είναι ακριβής.

## Εξαγωγή Excel σε HTML – Πλήρες Παράδειγμα

Συνδυάζοντας όλα τα παραπάνω, ορίστε το πλήρες, εκτελέσιμο πρόγραμμα:

```csharp
using System;
using Aspose.Cells;

namespace ExcelToHtmlWithFonts
{
    class Program
    {
        static void Main(string[] args)
        {
            // 1️⃣ Load the workbook
            string excelPath = @"C:\Temp\sample.xlsx";
            Workbook workbook = new Workbook(excelPath);
            
            // 2️⃣ Configure HTML export options
            HtmlSaveOptions htmlOptions = new HtmlSaveOptions
            {
                EmbedAllFonts = true,          // ✅ Embed every used font
                ExportEmbeddedImages = true,   // ✅ Include images as base64
                ExportChartImageFormat = ImageFormat.Png,
                ExportImagesAsBase64 = true    // ✅ Keep everything in one file
            };
            
            // 3️⃣ Save as HTML
            string htmlPath = @"C:\Temp\embedded.html";
            workbook.Save(htmlPath, htmlOptions);
            
            Console.WriteLine($"✅ HTML with embedded fonts saved at: {htmlPath}");
        }
    }
}
```

**Γιατί αυτό λειτουργεί:** Το αντικείμενο `HtmlSaveOptions` είναι ένας ισχυρός κοντέινερ. Με την εναλλαγή του `EmbedAllFonts`, υποδεικνύετε στο Aspose.Cells να σαρώσει τη συλλογή στυλ του βιβλίου εργασίας, να πάρει τα αρχεία γραμματοσειράς από το OS και να τα ενσωματώσει. Οι σημαίες `ExportEmbeddedImages` και `ExportImagesAsBase64` κρατούν το HTML αυτόνομο, κάτι που είναι χρήσιμο όταν χρειάζεται να στείλετε το αρχείο μέσω email ή να το αποθηκεύσετε σε βάση δεδομένων.

## Συνηθισμένα Προβλήματα Κατά την Ενσωμάτωση Γραμματοσειρών σε HTML

Ακόμη και με τον σωστό κώδικα, μερικά μικρά ζητήματα μπορούν να σας εμποδίσουν. Ας τα αντιμετωπίσουμε πριν γίνουν πηγή κεφαλαλγίας.

| Πρόβλημα | Γιατί Συμβαίνει | Πώς να Διορθώσετε |
|----------|----------------|-------------------|
| **Λείπει η γραμματοσειρά στον διακομιστή** | Ο διακομιστής όπου εκτελείται ο κώδικας μπορεί να μην έχει εγκατεστημένη τη προσαρμοσμένη γραμματοσειρά. | Εγκαταστήστε τις απαιτούμενες γραμματοσειρές στον διακομιστή ή αντιγράψτε τα αρχεία `.ttf/.otf` σε έναν γνωστό φάκελο και ορίστε το `htmlSaveOptions.FontsLocation` σε αυτή τη διαδρομή. |
| **Μεγάλο αρχείο HTML** | Η ενσωμάτωση πολλών βαριών γραμματοσειρών μπορεί να φουσκώσει το HTML (μερικές φορές >5 MB). | Χρησιμοποιήστε το `htmlSaveOptions.FontsList` για να ενσωματώσετε μόνο τις απαραίτητες γραμματοσειρές, ή εξετάστε το υποσύνολο των γραμματοσειρών με εργαλείο όπως το FontForge πριν την ενσωμάτωση. |
| **Περιορισμοί αδειοδότησης** | Ορισμένες εμπορικές γραμματοσειρές απαγορεύουν την ενσωμάτωση. | Ελέγξτε την άδεια χρήσης (EULA) της γραμματοσειράς. Αν η ενσωμάτωση δεν επιτρέπεται, χρησιμοποιήστε μια εναλλακτική web‑safe ή μετατρέψτε το φύλλο σε PDF. |
| **Συμβατότητα περιηγητών** | Πολύ παλιοί περιηγητές (IE 8) μπορεί να αγνοούν το `@font-face` με δεδομένα base64. | Παρέχετε έναν εναλλακτικό κανόνα CSS ή εξυπηρετήστε ξεχωριστό αρχείο CSS για παλαιούς περιηγητές. |
| **Λανθασμένο εύρος Unicode** | Η ενσωματωμένη γραμματοσειρά μπορεί να μην περιέχει όλους τους χαρακτήρες που χρησιμοποιούνται (π.χ. ασιατικά γλυφά). | Βεβαιωθείτε ότι η πηγαία γραμματοσειρά υποστηρίζει τα απαιτούμενα μπλοκ Unicode, ή ενσωματώστε μια δευτερεύουσα γραμματοσειρά που καλύπτει το κενό. |

## Προχωρημένο: Ενσωμάτωση Μόνο Επιλεγμένων Γραμματοσειρών

Αν γνωρίζετε ότι το βιβλίο εργασίας σας χρησιμοποιεί μόνο *Calibri* και *Times New Roman*, μπορείτε να περιορίσετε την ενσωμάτωση ως εξής:

```csharp
htmlSaveOptions.FontsList = new string[] { "Calibri", "Times New Roman" };
```

Αυτό μειώνει δραματικά το μέγεθος του HTML ενώ διατηρεί την εμφάνιση.

## Δοκιμή του Αποτελέσματος

Αφού δημιουργήσετε το `embedded.html`, εκτελέστε τους παρακάτω γρήγορους ελέγχους:

1. Ανοίξτε το αρχείο σε Chrome/Edge/Firefox.  
2. Ανοίξτε τα Developer Tools → Network → φιλτράρετε κατά **font**. Δεν πρέπει να δείτε εξωτερικά αιτήματα.  
3. Εξετάστε το μπλοκ `<style>`· θα βρείτε κανόνες `@font-face` με `src: url(data:font/ttf;base64,…)`.  
4. Συγκρίνετε το εμφανιζόμενο κείμενο με την αρχική προβολή του Excel—αν η στοίχιση είναι pixel‑perfect, πετύχατε.

Δημιουργώντας ένα αντικείμενο `HtmlSaveOptions`, ορίζοντας `EmbedAllFonts = true` και καλώντας `Workbook.Save`, λαμβάνετε ένα αυτόνομο αρχείο HTML που αναπαράγει πιστά την τυπογραφία του αρχικού φύλλου. Επίσης, εξετάσαμε κοινά προβλήματα, τεχνικές βελτιστοποίησης και έναν γρήγορο τρόπο για να ενσωματώσετε μόνο τις γραμματοσειρές που χρειάζεστε.
