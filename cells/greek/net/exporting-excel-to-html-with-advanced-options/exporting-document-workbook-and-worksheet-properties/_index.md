# Εξαγωγή ιδιοτήτων βιβλίου εργασίας και φύλλου εργασίας εγγράφου σε HTML

## Εισαγωγή

Όσον αφορά τον χειρισμό υπολογιστικών φύλλων, συχνά χρειάζεται να μετατρέψουμε αρχεία Excel σε διαφορετικές μορφές για κοινή χρήση, διατήρηση ή παρουσίαση. Μια συνηθισμένη εργασία είναι η εξαγωγή ιδιοτήτων βιβλίου εργασίας και φύλλου εργασίας σε μορφή HTML. Σε αυτό το άρθρο, θα σας καθοδηγήσουμε στον τρόπο με τον οποίο μπορείτε να το πετύχετε αυτό χρησιμοποιώντας το Aspose.Cells για .NET. Μην ανησυχείτε αν είστε νέοι στον προγραμματισμό ή στη βιβλιοθήκη Aspose. Θα το αναλύσουμε βήμα προς βήμα για να σας διευκολύνουμε!

## Προαπαιτούμενα

Πριν εμβαθύνουμε στον κώδικα, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ξεκινήσετε:

1. .NET Framework: Βεβαιωθείτε ότι το περιβάλλον ανάπτυξής σας έχει ρυθμιστεί με .NET Framework. Το Aspose.Cells είναι συμβατό με εκδόσεις του .NET Framework έως και 4.8.
   
2. Aspose.Cells για .NET: Θα χρειαστεί να έχετε εγκατεστημένο το Aspose.Cells. Μπορείτε να κατεβάσετε τη βιβλιοθήκη από το [σελίδα λήψεων](https://releases.aspose.com/cells/net/). 

3. IDE: Ένα κατάλληλο Ολοκληρωμένο Περιβάλλον Ανάπτυξης (IDE) όπως το Visual Studio θα απλοποιήσει την εμπειρία προγραμματισμού σας.

4. Δείγμα αρχείου Excel: Για σκοπούς δοκιμής, βεβαιωθείτε ότι έχετε ένα αρχείο Excel με το όνομα `sampleExportDocumentWorkbookAndWorksheetPropertiesInHTML.xlsx` στον κατάλογο εργασίας σας.

## Εισαγωγή πακέτων

Τώρα που καλύψαμε τις προϋποθέσεις, ας ξεκινήσουμε εισάγοντας τα απαραίτητα πακέτα στο έργο μας σε C#. Δείτε πώς μπορείτε να το κάνετε αυτό:

### Δημιουργία νέου έργου

- Ανοίξτε το IDE σας και δημιουργήστε ένα νέο έργο C#. Μπορείτε να επιλέξετε μια εφαρμογή κονσόλας, η οποία είναι ιδανική για την εκτέλεση αυτού του τύπου εργασίας.

### Προσθέστε το πακέτο Aspose.Cells NuGet

Για να προσθέσετε το πακέτο Aspose.Cells, ακολουθήστε τα εξής βήματα:

- Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων και επιλέξτε "Διαχείριση πακέτων NuGet".
- Στο NuGet Package Manager, αναζητήστε το "Aspose.Cells" και εγκαταστήστε το.
- Αυτό το πακέτο θα παρέχει τις απαραίτητες κλάσεις και μεθόδους για την εργασία με αρχεία Excel.

### Εισαγωγή χώρων ονομάτων

Στην κορυφή του κύριου αρχείου προγράμματος, βεβαιωθείτε ότι έχετε συμπεριλάβει τους ακόλουθους χώρους ονομάτων:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

Αυτό θα μας δώσει πρόσβαση στο `Workbook` και `HtmlSaveOptions` κλάσεις, τις οποίες θα χρησιμοποιήσουμε στο παράδειγμά μας.

Τώρα που είστε έτοιμοι, ας αναλύσουμε τη διαδικασία σε απλά βήματα.

## Βήμα 1: Ρύθμιση των καταλόγων αρχείων σας

Αρχικά, πρέπει να καθορίσουμε πού θα βρίσκονται τα αρχεία εισόδου και εξόδου. Στον κώδικά σας, αρχικοποιήστε τους καταλόγους ως εξής:

```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory/";  // Ενημέρωση με την πραγματική σας διαδρομή

// Κατάλογος εξόδου
string outputDir = "Your Document Directory/";  // Ενημέρωση με την πραγματική σας διαδρομή
```

- Κατάλογος πηγών: Εδώ είναι το αρχείο Excel εισόδου σας (`sampleExportDocumentWorkbookAndWorksheetPropertiesInHTML.xlsx`) αποθηκεύεται.
- Κατάλογος εξόδου: Αυτή είναι η διαδρομή όπου θέλετε να αποθηκευτεί το αρχείο HTML εξόδου.

## Βήμα 2: Φόρτωση του αρχείου Excel

Τώρα πρέπει να φορτώσουμε το αρχείο Excel χρησιμοποιώντας το `Workbook` τάξη:

```csharp
// Φόρτωση του δείγματος αρχείου Excel
Workbook workbook = new Workbook(sourceDir + "sampleExportDocumentWorkbookAndWorksheetPropertiesInHTML.xlsx");
```

- Παρουσία βιβλίου εργασίας: Το `Workbook` Ο κατασκευαστής παίρνει τη διαδρομή αρχείου προς το αρχείο Excel και δημιουργεί μια νέα παρουσία την οποία μπορείτε να χειριστείτε.

## Βήμα 3: Ρύθμιση επιλογών αποθήκευσης HTML

Στη συνέχεια, καθορίζουμε πώς θέλουμε να αποθηκεύσουμε τα δεδομένα του Excel σε μορφή HTML:

```csharp
// Καθορισμός επιλογών αποθήκευσης HTML
HtmlSaveOptions options = new HtmlSaveOptions();

// Αποτροπή εξαγωγής ιδιοτήτων εγγράφου, βιβλίου εργασίας και φύλλου εργασίας
options.ExportDocumentProperties = false;
options.ExportWorkbookProperties = false;
options.ExportWorksheetProperties = false;
```

- HtmlSaveOptions: Αυτή η κλάση βοηθά στη διαχείριση του τρόπου με τον οποίο το αρχείο Excel θα μετατραπεί σε HTML.
- Ορίσαμε αρκετές επιλογές για να `false` επειδή δεν θέλουμε να συμπεριλάβουμε ιδιότητες βιβλίου εργασίας και φύλλου εργασίας στην έξοδο HTML.

## Βήμα 4: Εξαγωγή όλων των στοιχείων σε HTML

Τώρα είμαστε έτοιμοι να αποθηκεύσουμε το βιβλίο εργασίας μας σε μορφή HTML:

```csharp
// Εξαγωγή του αρχείου Excel σε Html με επιλογές αποθήκευσης HTML
workbook.Save(outputDir + "outputExportDocumentWorkbookAndWorksheetPropertiesInHTML.html", options);
```

- Ο `Save` Η μέθοδος λαμβάνει δύο παραμέτρους: τη διαδρομή αρχείου για το αρχείο HTML εξόδου και τις επιλογές που έχουμε ορίσει. Η εκτέλεση αυτής της μεθόδου θα δημιουργήσει το αρχείο HTML σας στον καθορισμένο κατάλογο εξόδου.
