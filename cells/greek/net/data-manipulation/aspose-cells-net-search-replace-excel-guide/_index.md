## Οδηγός Εφαρμογής

### Αναζήτηση και αντικατάσταση σε μια περιοχή

Αυτή η λειτουργία σάς επιτρέπει να αναζητάτε αποτελεσματικά συγκεκριμένα δεδομένα εντός ενός καθορισμένου εύρους σε ένα φύλλο Excel και να τα αντικαθιστάτε με νέα δεδομένα. Ας αναλύσουμε τα βήματα υλοποίησης.

#### Επισκόπηση

Θα διαμορφώσετε μια περιοχή κελιών, θα ορίσετε επιλογές εύρεσης, θα κάνετε επανάληψη στα κελιά για αναζήτηση και αντικατάσταση τιμών και θα αποθηκεύσετε το τροποποιημένο βιβλίο εργασίας.

#### Υλοποίηση κώδικα

1. **Ορισμός καταλόγων και φόρτωση βιβλίου εργασίας**
   Ξεκινήστε ορίζοντας τους καταλόγους προέλευσης και εξόδου. Στη συνέχεια, φορτώστε το αρχείο Excel χρησιμοποιώντας `Workbook`.

   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string OutputDir = "YOUR_OUTPUT_DIRECTORY";

   Workbook workbook = new Workbook(SourceDir + "sampleSearchReplaceDataInRange.xlsx");
   Worksheet worksheet = workbook.Worksheets[0];
   ```

2. **Καθορίστε το εύρος και ρυθμίστε τις επιλογές εύρεσης**
   Δημιουργήστε ένα `CellArea` για να ορίσετε πού θέλετε να κάνετε αναζήτηση και να διαμορφώσετε τις επιλογές εύρεσης.

   ```csharp
   CellArea area = CellArea.CreateCellArea("E9", "H15");

   FindOptions opts = new FindOptions();
   opts.LookInType = LookInType.Values;
   opts.LookAtType = LookAtType.EntireContent;
   opts.SetRange(area);
   ```

3. **Αναζήτηση και αντικατάσταση δεδομένων**
   Χρησιμοποιήστε έναν βρόχο για να βρείτε κάθε εμφάνιση του όρου αναζήτησης εντός του εύρους, αντικαθιστώντας τον με νέα δεδομένα.

   ```csharp
   Cell cell = null;

   while (true)
   {
       cell = worksheet.Cells.Find("search", cell, opts);
       if (cell == null) break;
       cell.PutValue("replace");
   }
   ```

4. **Αποθήκευση του τροποποιημένου βιβλίου εργασίας**
   Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο στον κατάλογο εξόδου.

   ```csharp
   workbook.Save(OutputDir + "outputSearchReplaceDataInRange.xlsx");
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι όλες οι διαδρομές καταλόγου είναι σωστές και προσβάσιμες.
- Ελέγξτε ξανά τους ορισμούς εύρους κελιών στο `CellArea.CreateCellArea`.

### Χειρισμός βιβλίου εργασίας και φύλλων εργασίας
Αυτή η λειτουργία εστιάζει στη φόρτωση ενός αρχείου Excel και στην πρόσβαση στο πρώτο φύλλο εργασίας του.

#### Επισκόπηση
Φορτώστε ένα βιβλίο εργασίας, αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε και εκτελέστε τις απαραίτητες λειτουργίες.

#### Υλοποίηση κώδικα
1. **Φόρτωση του βιβλίου εργασίας**
   Αρχικοποιήστε το βιβλίο εργασίας από τον κατάλογο προέλευσης.

   ```csharp

   ```

2. **Πρόσβαση στο Πρώτο Φύλλο Εργασίας**
   Άμεση πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.

   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```
