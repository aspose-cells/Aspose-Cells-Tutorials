## Οδηγός Εφαρμογής

### Ρύθμιση καταλόγων
#### Επισκόπηση
Η δημιουργία καταλόγων προέλευσης και εξόδου είναι απαραίτητη για την αποτελεσματική οργάνωση των αρχείων. Αυτή η ενότητα δείχνει πώς να ρυθμίσετε αυτούς τους καταλόγους χρησιμοποιώντας C#.

#### Βήμα προς βήμα εφαρμογή
**1. Ορισμός διαδρομών καταλόγου**
```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY"; // Καθορίστε τον κατάλογο πηγής σας εδώ
string OutputDir = "YOUR_OUTPUT_DIRECTORY"; // Καθορίστε τον κατάλογο εξόδου σας εδώ
```

**2. Έλεγχος και δημιουργία καταλόγων**
```csharp
if (!Directory.Exists(SourceDir))
{
    Directory.CreateDirectory(SourceDir); // Δημιουργήστε τον κατάλογο πηγής εάν δεν υπάρχει
}

if (!Directory.Exists(OutputDir))
{
    Directory.CreateDirectory(OutputDir); // Δημιουργήστε τον κατάλογο εξόδου εάν δεν υπάρχει
}
```

### Δημιουργία και χειρισμός ενός βιβλίου εργασίας
#### Επισκόπηση
Μάθετε πώς να δημιουργείτε ένα βιβλίο εργασίας του Excel, να προσθέτετε φύλλα εργασίας και να χειρίζεστε εικόνες χρησιμοποιώντας το Aspose.Cells.

**3. Δημιουργία Βιβλίου Εργασίας**
```csharp
using Aspose.Cells;

Workbook workbook = new Workbook();
```

**4. Προσθήκη φύλλου εργασίας**
```csharp
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

**5. Εισαγωγή εικόνας στο φύλλο εργασίας**
Βεβαιωθείτε ότι έχετε ένα αρχείο εικόνας με το όνομα `logo.jpg` στον κατάλογο πηγής σας.
```csharp
string imagePath = Path.Combine(SourceDir, "logo.jpg");
int pictureIndex = worksheet.Pictures.Add(5, 5, imagePath);
Aspose.Cells.Drawing.Picture picture = worksheet.Pictures[pictureIndex];

// Τοποθετήστε την εικόνα αναλογικά
picture.UpperDeltaX = 200; // Οριζόντια μετατόπιση
picture.UpperDeltaY = 200; // Κάθετη μετατόπιση
```

**6. Αποθήκευση του βιβλίου εργασίας**
```csharp
string outputPath = Path.Combine(OutputDir, "book1.out.xls");
workbook.Save(outputPath);
```
