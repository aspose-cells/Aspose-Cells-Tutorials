## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην εισαγωγή και διαγραφή γραμμών χρησιμοποιώντας το Aspose.Cells με σαφή βήματα.

### Φόρτωση βιβλίου εργασίας
**Επισκόπηση:**
Η φόρτωση ενός βιβλίου εργασίας του Excel είναι το πρώτο σας βήμα για τον χειρισμό του περιεχομένου του με το Aspose.Cells.

#### Οδηγός βήμα προς βήμα:
1. **Αρχικοποίηση παρουσίας βιβλίου εργασίας**
   Χρησιμοποιήστε το `Workbook` κλάση για να φορτώσετε ένα υπάρχον αρχείο.
   ```csharp
   using Aspose.Cells;

   string sourceDir = @"YOUR_SOURCE_DIRECTORY";
   Workbook workbook = new Workbook(sourceDir + "/sampleInsertDeleteRows.xlsx");
   ```
   - Ο κατασκευαστής του `Workbook` Η κλάση ακολουθεί μια διαδρομή προς το αρχείο Excel σας.

### Εισαγωγή γραμμών
**Επισκόπηση:**
Η προσθήκη γραμμών είναι ζωτικής σημασίας για την προσθήκη πληροφοριών ή την προσαρμογή συνόλων δεδομένων.

#### Οδηγός βήμα προς βήμα:
1. **Φόρτωση βιβλίου εργασίας και φύλλου εργασίας της Access**
   ```csharp
   string sourceDir = @"YOUR_SOURCE_DIRECTORY";
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";

   Workbook workbookInsert = new Workbook(sourceDir + "/sampleInsertDeleteRows.xlsx");
   Worksheet sheetInsert = workbookInsert.Worksheets[0];
   ```
2. **Εισαγωγή γραμμών**
   Χρησιμοποιήστε το `InsertRows` μέθοδος.
   ```csharp
   // Εισαγάγετε 10 γραμμές ξεκινώντας από τον δείκτη γραμμής 2.
   sheetInsert.Cells.InsertRows(2, 10);
   ```
3. **Αποθήκευση αλλαγών**
   Αποθηκεύστε το βιβλίο εργασίας σας με τις τροποποιήσεις.
   ```csharp
   workbookInsert.Save(outputDir + "/outputInsertRows.xlsx");
   ```

### Διαγραφή γραμμών
**Επισκόπηση:**
Η κατάργηση περιττών γραμμών βοηθά στην απλοποίηση των δεδομένων και στη βελτίωση της αναγνωσιμότητας.

#### Οδηγός βήμα προς βήμα:
1. **Φόρτωση βιβλίου εργασίας και φύλλου εργασίας της Access**
   ```csharp
   string sourceDir = @"YOUR_SOURCE_DIRECTORY";
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";

   Workbook workbookDelete = new Workbook(sourceDir + "/sampleInsertDeleteRows.xlsx");
   Worksheet sheetDelete = workbookDelete.Worksheets[0];
   ```
2. **Διαγραφή γραμμών**
   Χρησιμοποιήστε το `DeleteRows` μέθοδος.
   ```csharp
   // Διαγράψτε 5 γραμμές ξεκινώντας από τον δείκτη γραμμής 17.
   sheetDelete.Cells.DeleteRows(17, 5);
   ```
3. **Αποθήκευση αλλαγών**
   Αποθηκεύστε το βιβλίο εργασίας σας με τις διαγραφές που έχουν εφαρμοστεί.
   ```csharp
   workbookDelete.Save(outputDir + "/outputDeleteRows.xlsx");
   ```
