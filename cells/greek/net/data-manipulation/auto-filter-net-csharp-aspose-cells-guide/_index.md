## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε βήμα προς βήμα στη δημιουργία ενός Αυτόματου Φίλτρου για αρχεία Excel.

### Δημιουργία βιβλίου εργασίας και πρόσβαση σε φύλλα

**Επισκόπηση:**
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας του Excel και αποκτώντας πρόσβαση στο φύλλο εργασίας όπου θα εφαρμόσετε φίλτρα.

**Βήματα Υλοποίησης:**
1. **Δημιουργία αντικειμένου βιβλίου εργασίας:**
   ```csharp
   string dataDir = "your_directory_path";
   Workbook workbook = new Workbook(dataDir + "book1.xls");
   ```
2. **Πρόσβαση στο Φύλλο Εργασίας:**
   Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel:
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

### Εφαρμογή αυτόματου φίλτρου

**Επισκόπηση:**
Ρύθμιση και εφαρμογή ενός Αυτόματου Φίλτρου σε μια καθορισμένη περιοχή κελιών.

**Βήματα Υλοποίησης:**
3. **Δημιουργία και εφαρμογή αυτόματου φίλτρου:**
   Ορίστε την περιοχή κελιών για το φίλτρο σας:
   ```csharp
   worksheet.AutoFilter.Range = "A1:B1";
   ```
4. **Αποθήκευση του βιβλίου εργασίας σας:**
   Αποθηκεύστε τις αλλαγές σε ένα νέο αρχείο μετά την εφαρμογή του φίλτρου:
