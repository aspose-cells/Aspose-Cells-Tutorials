## Οδηγός Εφαρμογής
Ας δούμε τα βήματα για να διαγράψετε πολλές γραμμές σε ένα φύλλο εργασίας του Excel.
### Βήμα 1: Άνοιγμα ή δημιουργία αρχείου Excel
Ανοίξτε ένα υπάρχον αρχείο Excel ή δημιουργήστε ένα νέο. Εδώ, ανοίγουμε `Book1.xlsx`:
```csharp
// Διαδρομή προς τον κατάλογο δεδομένων σας
string dataDir = "YourPath/"; 

// Χρησιμοποιήστε το FileStream για να ανοίξετε ένα υπάρχον αρχείο Excel
FileStream fstream = new FileStream(dataDir + "Book1.xlsx", FileMode.OpenOrCreate);
```
### Βήμα 2: Φόρτωση του βιβλίου εργασίας
Φορτώστε το αρχείο Excel σε ένα `Workbook` αντικείμενο:
```csharp
// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας με το FileStream
Workbook workbook = new Workbook(fstream);

// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας σας
Worksheet worksheet = workbook.Worksheets[0];
```
### Βήμα 3: Διαγραφή πολλαπλών γραμμών
Τώρα, διαγράψτε πολλές γραμμές. Εδώ, αφαιρούμε 10 γραμμές ξεκινώντας από τον δείκτη γραμμής 2:
```csharp
// Διαγραφή 10 γραμμών ξεκινώντας από την 3η γραμμή (δείκτης 2)
worksheet.Cells.DeleteRows(2, 10);
```
### Βήμα 4: Αποθήκευση και κλείσιμο
Αποθηκεύστε το βιβλίο εργασίας για να διατηρηθούν οι αλλαγές και να κλείσετε τη ροή αρχείων:
```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
workbook.Save(dataDir + "output.xlsx");

// Να θυμάστε πάντα να κλείνετε το FileStream
fstream.Close();
```

- **Σφάλματα διαδρομής αρχείου**Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές.
