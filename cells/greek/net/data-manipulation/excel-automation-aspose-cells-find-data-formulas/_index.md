### Χαρακτηριστικό 2: Πρόσβαση σε συλλογή κινητών
#### Επισκόπηση
Η πρόσβαση σε συλλογές κελιών ενός φύλλου εργασίας σάς επιτρέπει να χειρίζεστε δεδομένα αποτελεσματικά.

**Βήματα:**
##### Συλλογή κελιών πρόσβασης
```csharp
Cells cells = workbook.Worksheets[0].Cells;
```
- **Εξήγηση:** Ανακτά τη συλλογή κελιών από το πρώτο φύλλο εργασίας, ενεργοποιώντας τις λειτουργίες δεδομένων σε συγκεκριμένα κελιά.

### Χαρακτηριστικό 3: Ρύθμιση παραμέτρων FindOptions
#### Επισκόπηση
Η ρύθμιση παραμέτρων των επιλογών αναζήτησης σάς επιτρέπει να ορίσετε ακριβή κριτήρια για την εύρεση δεδομένων εντός ενός καθορισμένου εύρους.

**Βήματα:**
##### Ρύθμιση παραμέτρων επιλογών εύρεσης
```csharp
FindOptions findOptions = new FindOptions();
CellArea ca = new CellArea { StartRow = 8, StartColumn = 2, EndRow = 17, EndColumn = 13 };
findOptions.SetRange(ca);
findOptions.SearchBackward = false;
findOptions.SearchOrderByRows = true;
findOptions.LookInType = LookInType.Values;
findOptions.LookAtType = LookAtType.EntireContent;
```
- **Σκοπός:** Ορίζει το εύρος και τα κριτήρια για αναζήτηση εντός κελιών, βελτιστοποιώντας την αποτελεσματικότητα της αναζήτησης.

### Λειτουργία 4: Εύρεση δεδομένων ή τύπων σε κελιά
#### Επισκόπηση
Χρησιμοποιήστε διαμορφωμένες επιλογές για να εντοπίσετε συγκεκριμένα δεδομένα ή τύπους μέσα στο βιβλίο εργασίας σας.

**Βήματα:**
##### Υλοποίηση λειτουργικότητας αναζήτησης
```csharp
Cell cell = cells.Find(276, null, findOptions);

if (cell != null)
{
    Console.WriteLine("Found at " + cell.Name);
}
else
{
    Console.WriteLine("Value not found.");
}
```
- **Εξήγηση:** Αναζητά μια καθορισμένη τιμή εντός του καθορισμένου εύρους. Εάν βρεθεί, εμφανίζει το όνομα του κελιού. Διαφορετικά, υποδεικνύει ότι η τιμή δεν βρέθηκε.
