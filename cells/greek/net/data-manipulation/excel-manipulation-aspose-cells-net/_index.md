## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση και αποθήκευση αρχείου Excel

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον αρχείο Excel, να εκτελέσετε τις απαραίτητες λειτουργίες και να το αποθηκεύσετε ξανά. Δείτε πώς:

#### Βήμα 1: Ορίστε τις διαδρομές αρχείων σας
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση του βιβλίου εργασίας
Φορτώστε το αρχείο Excel χρησιμοποιώντας το Aspose.Cells.
```csharp
Workbook wb = new Workbook(SourceDir + "sampleSetMarginsOfCommentOrShapeInsideTheWorksheet.xlsx");
```

#### Βήμα 3: Αποθήκευση του βιβλίου εργασίας
Αποθηκεύστε το τροποποιημένο βιβλίο εργασίας σε μια καθορισμένη θέση.
```csharp
wb.Save(OutputDir + "outputSetMarginsOfCommentOrShapeInsideTheWorksheet.xlsx");
```

### Λειτουργία 2: Πρόσβαση και τροποποίηση σχημάτων σε ένα φύλλο εργασίας

Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση σε σχήματα μέσα σε ένα φύλλο εργασίας του Excel και να προσαρμόζετε τις ιδιότητες ευθυγράμμισης κειμένου τους για ακριβή έλεγχο μορφοποίησης.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας σας όπως έχει αποδειχθεί προηγουμένως.
```csharp
Workbook wb = new Workbook(SourceDir + "sampleSetMarginsOfCommentOrShapeInsideTheWorksheet.xlsx");
```

#### Βήμα 2: Πρόσβαση σε σχήματα σε ένα φύλλο εργασίας
Αποκτήστε πρόσβαση σε σχήματα χρησιμοποιώντας τον ακόλουθο κώδικα:
```csharp
Worksheet ws = wb.Worksheets[0];

foreach (Shape sh in ws.Shapes)
{
    // Ανάκτηση ιδιοτήτων στοίχισης κειμένου
    Aspose.Cells.Drawing.Texts.ShapeTextAlignment txtAlign = sh.TextBody.TextAlignment;

    // Απενεργοποίηση αυτόματου περιθωρίου για προσαρμοσμένες ρυθμίσεις
    txtAlign.IsAutoMargin = false;
    
    // Ορίστε προσαρμοσμένα περιθώρια
    txtAlign.TopMarginPt = 10;
    txtAlign.LeftMarginPt = 10;
    txtAlign.BottomMarginPt = 10;
    txtAlign.RightMarginPt = 10;
}
```

#### Βήμα 3: Αποθήκευση των αλλαγών
Αφού τροποποιήσετε τα σχήματα, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε τις αλλαγές.
```csharp
wb.Save(OutputDir + "outputSetMarginsOfCommentOrShapeInsideTheWorksheet.xlsx");
```
