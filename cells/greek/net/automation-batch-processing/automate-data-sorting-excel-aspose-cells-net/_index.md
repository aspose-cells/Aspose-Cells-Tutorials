using Aspose.Cells;

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να ρυθμίζουμε και να εκτελούμε την ταξινόμηση δεδομένων χρησιμοποιώντας το Aspose.Cells.

### Βήμα 1: Προετοιμασία του βιβλίου εργασίας σας

Ξεκινήστε φορτώνοντας το αρχείο Excel σε ένα `Workbook` αντικείμενο. Αυτό το αντικείμενο αντιπροσωπεύει ολόκληρο το βιβλίο εργασίας μέσα στην εφαρμογή σας.

```csharp
// Φόρτωση ενός υπάρχοντος αρχείου Excel.
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "book1.xls");
```

### Βήμα 2: Ρύθμιση παραμέτρων του DataSorter

Στη συνέχεια, διαμορφώστε το `DataSorter` αντικείμενο. Αυτό θα καθορίσει πώς και ποια δεδομένα θα ταξινομηθούν.

```csharp
// Αποκτήστε πρόσβαση στο DataSorter από το βιβλίο εργασίας.
DataSorter sorter = workbook.DataSorter;

// Ορίστε τη σειρά ταξινόμησης για το πρώτο κλειδί σε φθίνουσα.
sorter.Order1 = SortOrder.Descending;
sorter.Key1 = 0; // Ευρετήριο πρώτης στήλης

// Ορίστε τη σειρά ταξινόμησης για το δεύτερο κλειδί σε αύξουσα.
sorter.Order2 = SortOrder.Ascending;
sorter.Key2 = 1; // Ευρετήριο δεύτερης στήλης
```

### Βήμα 3: Ορισμός περιοχής κελιού

Ορίστε την περιοχή κελιών που θέλετε να ταξινομήσετε χρησιμοποιώντας ένα `CellArea` αντικείμενο.

```csharp
// Καθορίστε την περιοχή κελιών για ταξινόμηση.
CellArea ca = new CellArea();
ca.StartRow = 0;
ca.EndRow = 13; // Περιλαμβάνει τις σειρές 0-13
ca.StartColumn = 0;
ca.EndColumn = 1; // Περιλαμβάνει τις στήλες 0 και 1
```

### Βήμα 4: Εκτέλεση ταξινόμησης

Εκτελέστε τη λειτουργία ταξινόμησης στο καθορισμένο φύλλο εργασίας.

```csharp
// Εφαρμόστε ταξινόμηση στην καθορισμένη περιοχή κελιών στο πρώτο φύλλο.
sorter.Sort(workbook.Worksheets[0].Cells, ca);
```
