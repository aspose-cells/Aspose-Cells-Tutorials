## Οδηγός Εφαρμογής

### Ρύθμιση βιβλίου εργασίας και φύλλου εργασίας

**Επισκόπηση:**
Το πρώτο βήμα είναι να δημιουργήσετε ένα νέο βιβλίο εργασίας και να αποκτήσετε πρόσβαση στα φύλλα εργασίας του. Αυτό αποτελεί τη βάση των εργασιών αυτοματοποίησης του Excel.

#### Δημιουργία νέου βιβλίου εργασίας
```csharp
Workbook workbook = new Workbook(); // Αρχικοποίηση ενός νέου αντικειμένου βιβλίου εργασίας
```

Εδώ, δημιουργούμε ένα παράδειγμα `Workbook`, το οποίο αντιπροσωπεύει ένα ολόκληρο αρχείο Excel.

#### Πρόσβαση στο Πρώτο Φύλλο Εργασίας
```csharp
Worksheet sheet = workbook.getWorksheets().get(0); // Ανάκτηση του πρώτου φύλλου εργασίας
```

Η πρόσβαση στο πρώτο φύλλο εργασίας σάς επιτρέπει να ξεκινήσετε τη συμπλήρωσή του με δεδομένα και στοιχεία ελέγχου.

#### Λήψη συλλογής κελιών
```csharp
Cells cells = sheet.getCells(); // Πρόσβαση σε όλα τα κελιά στο φύλλο εργασίας
```

Αυτή η συλλογή μας επιτρέπει να χειριζόμαστε μεμονωμένα κελιά ή περιοχές κελιών μέσα στο φύλλο.

### Προσθήκη κειμένου και μορφοποίηση κελιών

**Επισκόπηση:**
Βελτιώστε τα φύλλα Excel σας προσθέτοντας κείμενο σε κελιά και εφαρμόζοντας στυλ όπως η έντονη γραφή για έμφαση.

#### Εισαγωγή κειμένου σε ένα κελί
```csharp
cells.get("B3").putValue("Choose Dept:");
```

Αυτός ο κώδικας εισάγει τη συμβολοσειρά "Choose Dept:" στο κελί B3.

#### Ορισμός στυλ κελιού σε έντονη γραφή
```csharp
Style style = cells.get("B3").getStyle();
style.getFont().setBold(true);
cells.get("B3").setStyle(style);
```

Εδώ, ανακτούμε και τροποποιούμε το στυλ του κελιού B3 για να κάνουμε το κείμενό του έντονο, βελτιώνοντας την ορατότητα.

### Εισαγωγή τιμών λίστας και προσθήκη στοιχείου ελέγχου ListBox

**Επισκόπηση:**
Συμπληρώστε τα κελιά με τιμές λίστας που μπορούν να επιλεγούν μέσω ενός στοιχείου ελέγχου ListBox, προσθέτοντας διαδραστικότητα στο φύλλο σας.

#### Εισαγωγή τιμών λίστας σε κελιά
```csharp
cells.get("A2").putValue("Sales");
cells.get("A3").putValue("Finance");
// Συνέχεια για άλλα τμήματα...
```

Αυτό γεμίζει τα κελιά με ονόματα τμημάτων, ορίζοντας επιλογές για το ListBox.

#### Προσθήκη και ρύθμιση παραμέτρων ενός στοιχείου ελέγχου ListBox
```csharp
Aspose.Cells.Drawing.ListBox listBox = sheet.getShapes().addListBox(2, 0, 3, 0, 122, 100);
listBox.setPlacement(PlacementType.FreeFloating);
cells.get("A1").setValue(listBox.getName());
string tempLinkedCell = "A1";
listBox.setLinkedCell(tempLinkedCell);
listBox.setInputRange("A2:A7");
cells.get(tempLinkedCell).setValue(listBox.getName());
string tempInputRange = "A2:A7";
listBox.setInputRange(tempInputRange);
cells.get("A1").setFormula(RangeUtility.getReferenceFromHSSFRangeName(tempLinkedCell));
listBox.setSelectionType(SelectionType.Single);
listBox.setShadow(true);
```

Το ListBox προστίθεται στο φύλλο εργασίας, συνδέεται με το κελί A1 για έξοδο και διαμορφώνεται με μια σειρά επιλογών.

### Αποθήκευση βιβλίου εργασίας

**Επισκόπηση:**
Βεβαιωθείτε ότι η εργασία σας δεν θα χαθεί αποθηκεύοντας το βιβλίο εργασίας σε έναν καθορισμένο κατάλογο.

#### Αποθήκευση του βιβλίου εργασίας
```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/book1.out.xls";
workbook.save(outputFilePath);
```

Αυτό αποθηκεύει το αρχείο Excel σας με όλες τις αλλαγές που έχουν εφαρμοστεί, χρησιμοποιώντας μια καθορισμένη διαδρομή.
