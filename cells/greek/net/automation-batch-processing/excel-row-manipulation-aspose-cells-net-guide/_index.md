## Οδηγός Εφαρμογής

### Δυνατότητα: Αντιγραφή γραμμής σε φύλλο εργασίας Excel

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να αντιγράψετε μια γραμμή από τη μία θέση στην άλλη μέσα στο ίδιο φύλλο εργασίας, διασφαλίζοντας ότι περιλαμβάνονται όλα τα στοιχεία όπως δεδομένα, μορφοποίηση, εικόνες και αντικείμενα σχεδίασης.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση του βιβλίου εργασίας σας**
Ξεκινήστε φορτώνοντας το υπάρχον αρχείο Excel:
```csharp
// Αντικαταστήστε με την πραγματική διαδρομή του καταλόγου προέλευσης
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; 
Workbook excelWorkbook1 = new Workbook(SourceDir + "book1.xls");
```

**2. Πρόσβαση στο Φύλλο Εργασίας**
Αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε να χειριστείτε, π.χ. το πρώτο φύλλο:
```csharp
Worksheet wsTemplate = excelWorkbook1.Worksheets[0];
```

**3. Αντιγράψτε τη γραμμή**
Χρησιμοποιήστε το `CopyRow` μέθοδος για την αντιγραφή δεδομένων από τη μία γραμμή στην άλλη. Εδώ, αντιγράφουμε τη δεύτερη γραμμή (ευρετήριο 1) στη δέκατη έκτη γραμμή (ευρετήριο 15):
```csharp
wsTemplate.Cells.CopyRow(wsTemplate.Cells, 1, 15);
```

**4. Αποθηκεύστε το βιβλίο εργασίας σας**
Τέλος, αποθηκεύστε τις αλλαγές σας:
```csharp
excelWorkbook1.Save(SourceDir + "output.xls");
```
