## Οδηγός Εφαρμογής

### Διαχείριση καταλόγου

**Επισκόπηση**: Αυτή η λειτουργία ελέγχει την ύπαρξη ενός καταλόγου και τον δημιουργεί εάν λείπει.

#### Βήμα προς βήμα εφαρμογή

##### Έλεγχος εάν υπάρχει κατάλογος

```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
bool isExists = Directory.Exists(SourceDir);
```

Εδώ, `Directory.Exists` ελέγχει εάν η καθορισμένη διαδρομή υπάρχει. Αυτή η μέθοδος επιστρέφει μια λογική τιμή.

##### Δημιουργία καταλόγου εάν δεν υπάρχει

```csharp
if (!isExists)
{
    Directory.CreateDirectory(SourceDir);
}
```

`Directory.CreateDirectory` δημιουργεί τον κατάλογο και όλους τους απαραίτητους υποκαταλόγους κατά μήκος της διαδρομής.

### Χειρισμός ροής αρχείων

**Επισκόπηση**: Δείχνει πώς να ανοίξετε ένα αρχείο Excel χρησιμοποιώντας το FileStream και να διασφαλίσετε ότι οι πόροι έχουν αποδεσμευτεί σωστά.

#### Βήμα προς βήμα εφαρμογή

##### Δημιουργήστε ένα FileStream για το αρχείο Excel

```csharp
string SourceFile = Path.Combine("YOUR_SOURCE_DIRECTORY", "book1.xls");
FileStream fstream = new FileStream(SourceFile, FileMode.Open);
```

`FileStream` χρησιμοποιείται για το άνοιγμα του αρχείου σε `Open` τρόπος.

##### Κλείσιμο του FileStream

```csharp
fstream.Close();
```

Το κλείσιμο της ροής απελευθερώνει τους πόρους του συστήματος που είναι συνδεδεμένοι με αυτήν, αποτρέποντας διαρροές μνήμης.

### Λειτουργίες βιβλίου εργασίας με το Aspose.Cells

**Επισκόπηση**Αυτή η λειτουργία δείχνει τη φόρτωση ενός βιβλίου εργασίας του Excel, την τροποποίηση ιδιοτήτων όπως το πλάτος των στηλών και την αποθήκευση των αλλαγών.

#### Βήμα προς βήμα εφαρμογή

##### Φόρτωση και άνοιγμα βιβλίου εργασίας

```csharp
using (FileStream fstream = new FileStream(inputFilePath, FileMode.Open))
{
    Workbook workbook = new Workbook(fstream);
}
```

Ο `Workbook` Ο κατασκευαστής αρχικοποιεί ένα αντικείμενο για λειτουργίες αρχείων Excel. Χρησιμοποιώντας ένα `using` Η δήλωση διασφαλίζει ότι η ροή κλείνει αυτόματα.

##### Πρόσβαση και τροποποίηση ιδιοτήτων φύλλου εργασίας

```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Cells.StandardWidth = 20.5;
```

Η πρόσβαση στο πρώτο φύλλο εργασίας σάς επιτρέπει να τροποποιήσετε τα πλάτη των στηλών, βελτιώνοντας την αναγνωσιμότητα.

##### Αποθήκευση του βιβλίου εργασίας

```csharp
workbook.Save(outputFilePath);
```

Ο `Save` Η μέθοδος γράφει όλες τις αλλαγές πίσω σε μια καθορισμένη θέση στο αρχείο Excel.
