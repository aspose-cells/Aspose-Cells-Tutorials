### Οδηγός Εφαρμογής
Ας αναλύσουμε κάθε χαρακτηριστικό και ας δούμε πώς να το εφαρμόσουμε χρησιμοποιώντας **Aspose.Cells για .NET**.

#### Χαρακτηριστικό: Αρχικοποίηση βιβλίου εργασίας και φόρτωση προτύπου
##### Επισκόπηση
Αυτό το βήμα περιλαμβάνει την αρχικοποίηση ενός `WorkbookDesigner` αντικείμενο και φόρτωση ενός προτύπου Excel. Αυτό είναι κρίσιμο καθώς θέτει τα θεμέλια για τη συμπλήρωση δεδομένων.
##### Βήματα
1. **Αρχικοποίηση του WorkbookDesigner**
   ```csharp
   WorkbookDesigner designer = new WorkbookDesigner();
   ```

2. **Φόρτωση προτύπου**
   Καθορίστε τον κατάλογο προέλευσης όπου βρίσκεται το αρχείο προτύπου `SM_NestedObjects.xlsx` κατοικεί.
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   designer.Workbook = new Workbook(SourceDir + "SM_NestedObjects.xlsx");
   ```

#### Χαρακτηριστικό: Δημιουργία αντικειμένων και συμπλήρωση δεδομένων
##### Επισκόπηση
Εδώ, θα δημιουργήσετε προσαρμοσμένες κλάσεις για να διατηρήσετε τα δεδομένα σας και να τα συμπληρώσετε με τιμές. Αυτό το βήμα είναι απαραίτητο για την προσομοίωση πραγματικών σεναρίων όπου τα δεδομένα προέρχονται από διάφορες πηγές.
##### Βήματα
1. **Ορισμός κλάσεων**

   Δημιουργώ `Individual` και `Wife` κλάσεις για την αναπαράσταση ένθετων αντικειμένων.
   ```csharp
Ατομική τάξη {
    δημόσια συμβολοσειρά Όνομα { get; set; }
    public int Ηλικία { get; set; }
    εσωτερικό Individual(όνομα συμβολοσειράς, ηλικία ακέραιου αριθμού) {
        αυτό.Όνομα = όνομα;
        αυτό.Ηλικία = ηλικία;
    }
    δημόσια Σύζυγος Σύζυγος { get; set; }
}

δημόσια τάξη Σύζυγος {
    δημόσια συμβολοσειρά Όνομα { get; set; }
    public int Ηλικία { get; set; }
    δημόσια Σύζυγος(όνομα συμβολοσειράς, ηλικία ακέραιος) {
        αυτό.Όνομα = όνομα;
        αυτό.Ηλικία = ηλικία;
    }
}
```

2. **Create Instances**
   Populate instances of these classes with data.
   ```csharp
Individual p1 = new Individual("Damian", 30);
p1.Wife = new Wife("Dalya", 28);
Individual p2 = new Individual("Mack", 31);
p2.Wife = new Wife("Maaria", 29);
```

3. **Προετοιμασία συλλογής**
   Αποθηκεύστε αυτά τα αντικείμενα σε μια συλλογή για να τα χρησιμοποιήσετε ως πηγή δεδομένων.
   ```csharp
Λίστα<Individual> λίστα = νέα λίστα<Individual>();
λίστα.Προσθήκη(p1);
λίστα.Προσθήκη(p2);
```

#### Feature: Setting Data Source and Processing Markers
##### Overview
In this section, you'll set up your data source in `WorkbookDesigner` and process markers to generate the final Excel file.
##### Steps
1. **Set DataSource**
   Link the data collection with the template.
   ```csharp
designer.SetDataSource("Individual", list);
```

2. **Δείκτες Διαδικασίας**
   Επεξεργαστείτε όλους τους καθορισμένους δείκτες στο πρότυπο ώστε να αντικατοπτρίζουν τα δεδομένα σας.
   ```csharp
σχεδιαστής.Διαδικασία(ψευδής);
```

3. **Save Output**
   Save the processed workbook to an output directory.
   ```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
designer.Workbook.Save(outputDir + "output.xlsx");
```
