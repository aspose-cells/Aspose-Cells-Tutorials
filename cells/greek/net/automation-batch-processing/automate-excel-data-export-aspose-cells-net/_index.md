## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία και φόρτωση βιβλίου εργασίας

#### Επισκόπηση
Ξεκινήστε δημιουργώντας ένα `Workbook` αντικείμενο για να φορτώσετε το αρχείο Excel σας, επιτρέποντάς σας να χειρίζεστε δεδομένα μέσω προγραμματισμού.

**Βήμα προς βήμα εφαρμογή**

##### Βήμα 1: Ορίστε τον κατάλογο πηγών
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
```
*Εξήγηση:* Καθορίστε τον κατάλογο όπου βρίσκεται το αρχείο προέλευσης Excel.

##### Βήμα 2: Δημιουργία και φόρτωση του βιβλίου εργασίας
```csharp
Workbook workbook = new Workbook(sourceDir + "/sampleNamesTable.xlsx");
```
*Εξήγηση:* Αυτή η γραμμή δημιουργεί ένα `Workbook` και φορτώνει το 'sampleNamesTable.xlsx'. Η διαδρομή αρχείου συνδυάζει τον καθορισμένο κατάλογο με το όνομα αρχείου.

### Δυνατότητα 2: Πρόσβαση σε μια ονομασμένη περιοχή σε ένα φύλλο εργασίας του Excel

#### Επισκόπηση
Αποκτήστε πρόσβαση σε συγκεκριμένες επώνυμες περιοχές μέσα στο βιβλίο εργασίας του Excel για να εκτελέσετε λειτουργίες σε στοχευμένες ενότητες δεδομένων.

**Βήμα προς βήμα εφαρμογή**

##### Βήμα 1: Αρχικοποίηση του WorkbookDesigner
```csharp
WorkbookDesigner designer = new WorkbookDesigner(workbook);
```
*Εξήγηση:* Ο `WorkbookDesigner` Η κλάση επιτρέπει τον προηγμένο χειρισμό βιβλίων εργασίας, όπως την πρόσβαση σε ονομασμένες περιοχές.

##### Βήμα 2: Ανάκτηση του ονομασμένου εύρους
```csharp
var range = designer.Workbook.Worksheets.GetRangeByName("Names");
```
*Εξήγηση:* Χρησιμοποιήστε αυτήν τη μέθοδο για να αποκτήσετε πρόσβαση στην ονομασμένη περιοχή 'Ονόματα' μέσα στο βιβλίο εργασίας σας. Αυτή η περιοχή είναι τώρα έτοιμη για περαιτέρω επεξεργασία.

### Λειτουργία 3: Εξαγωγή δεδομένων από μια ονομαστική περιοχή με επιλογές

#### Επισκόπηση
Εξάγετε δεδομένα αποτελεσματικά παρακάμπτοντας τις κεφαλίδες και διαμορφώνοντας τις επιλογές εξαγωγής χρησιμοποιώντας `ExportTableOptions`.

**Βήμα προς βήμα εφαρμογή**

##### Βήμα 1: Ρύθμιση παραμέτρων επιλογών εξαγωγής
```csharp
ExportTableOptions options = new ExportTableOptions();
options.ExportColumnName = true;
```
*Εξήγηση:* Ρυθμίζοντας `ExportColumnName` να `true`, η πρώτη γραμμή (που υποτίθεται ότι είναι κεφαλίδες) θα παραλειφθεί κατά την εξαγωγή.

##### Βήμα 2: Εξαγωγή δεδομένων από ονομασμένο εύρος
```csharp
var dataTable = range.ExportDataTable(options);
```
*Εξήγηση:* Αυτή η μέθοδος εξάγει δεδομένα σε ένα `DataTable`, παραλείποντας τα ονόματα στηλών ως κεφαλίδες, καθιστώντας το ιδανικό για περαιτέρω επεξεργασία ή ανάλυση.
