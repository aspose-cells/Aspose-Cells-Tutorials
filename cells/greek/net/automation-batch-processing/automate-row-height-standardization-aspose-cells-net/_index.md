- Βήματα για την προγραμματιστική προσαρμογή των υψών γραμμών σε ολόκληρο το φύλλο εργασίας
- Συμβουλές για τη βελτιστοποίηση των εργασιών χειρισμού αρχείων Excel

Ας δούμε πώς μπορείτε να απλοποιήσετε αυτήν τη διαδικασία. Πριν ξεκινήσουμε, ας καλύψουμε τις απαραίτητες προϋποθέσεις για να παρακολουθήσετε αυτό το σεμινάριο.

## Οδηγός Εφαρμογής

### Ορισμός ύψους γραμμών σε ένα φύλλο εργασίας

Αυτή η λειτουργία σάς επιτρέπει να τυποποιήσετε τα ύψη των γραμμών σε όλες τις γραμμές ενός φύλλου εργασίας. Ας αναλύσουμε πώς να την εφαρμόσετε βήμα προς βήμα:

#### Βήμα 1: Φόρτωση του αρχείου Excel
Αρχικά, ανοίξτε το αρχείο Excel που θέλετε χρησιμοποιώντας ένα `FileStream`Αυτή η ροή θα χρησιμοποιηθεί για τη δημιουργία του `Workbook` αντικείμενο.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Δημιουργία ροής αρχείων που περιέχει το αρχείο Excel που θα ανοιχτεί
using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
{
    // Δημιουργία στιγμιότυπου ενός αντικειμένου βιβλίου εργασίας ανοίγοντας το αρχείο μέσω της ροής αρχείων
    Workbook workbook = new Workbook(fstream);
```

Εδώ, `RunExamples.GetDataDir` χρησιμοποιείται για την ανάκτηση της διαδρομής καταλόγου του αρχείου Excel. Βεβαιωθείτε ότι το αρχείο "book1.xls" υπάρχει σε αυτήν τη θέση.

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας
Αποκτήστε πρόσβαση στο φύλλο εργασίας όπου θέλετε να ορίσετε τα ύψη των γραμμών χρησιμοποιώντας:

```csharp
    // Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
    Worksheet worksheet = workbook.Worksheets[0];
```

Αυτός ο κώδικας έχει πρόσβαση στο πρώτο φύλλο μέσω ευρετηρίου. Μπορείτε να τον τροποποιήσετε για να αποκτήσετε πρόσβαση σε διαφορετικό φύλλο, εάν χρειάζεται.

#### Βήμα 3: Ορισμός ύψους γραμμών
Χρησιμοποιήστε το `StandardHeight` ιδιότητα για να ορίσετε το ύψος για όλες τις γραμμές:

```csharp
    // Ορισμός του ύψους όλων των γραμμών στο φύλλο εργασίας σε 15 σημεία
    worksheet.Cells.StandardHeight = 15;
```

Εδώ, το ύψος κάθε σειράς τυποποιείται σε 15 πόντους. Μπορείτε να προσαρμόσετε αυτήν την τιμή ανάλογα με τις απαιτήσεις σας.

#### Βήμα 4: Αποθήκευση και κλείσιμο
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο και κλείστε τη ροή:

```csharp
    // Αποθήκευση του τροποποιημένου αρχείου Excel
    workbook.Save(dataDir + "output.out.xls");

    // Το κλείσιμο της ροής αρχείων γίνεται με τη χρήση εντολής
}
```

Ο `using` Η δήλωση διασφαλίζει ότι οι πόροι διατίθενται σωστά μόλις ολοκληρωθούν οι λειτουργίες.
