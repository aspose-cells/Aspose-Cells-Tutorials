## Οδηγός Εφαρμογής
Θα εξερευνήσουμε βασικά χαρακτηριστικά: Χειρισμός βιβλίου εργασίας και Διαχείριση διαδρομής αρχείου.
### Χειρισμός βιβλίου εργασίας
Εστιάστε στο άνοιγμα ενός αρχείου Excel, στην εισαγωγή γραμμών σε ένα φύλλο εργασίας και στην αποθήκευση του τροποποιημένου βιβλίου εργασίας.
#### Βήμα 1: Ανοίξτε ένα υπάρχον αρχείο Excel χρησιμοποιώντας το FileStream
Ανοίξτε το υπάρχον αρχείο Excel χρησιμοποιώντας `FileStream`, επιτρέποντας άμεσες λειτουργίες ανάγνωσης ή εγγραφής:
```csharp
// Άνοιγμα του αρχείου πηγαίου κώδικα Excel
FileStream fstream = new FileStream(dataDir, FileMode.Open);
```
#### Βήμα 2: Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας από τη ροή αρχείων
Δημιουργήστε ένα `Workbook` αντικείμενο για την αναπαράσταση ενός ολόκληρου βιβλίου εργασίας του Excel στη μνήμη:
```csharp
// Φόρτωση του βιβλίου εργασίας χρησιμοποιώντας τη ροή αρχείων
Workbook workbook = new Workbook(fstream);
```
#### Βήμα 3: Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Στοχεύστε με ακρίβεια τις τροποποιήσεις αποκτώντας πρόσβαση σε συγκεκριμένα φύλλα εργασίας:
```csharp
// Λήψη του πρώτου φύλλου εργασίας από το βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```
#### Βήμα 4: Εισαγωγή γραμμών στο φύλλο εργασίας
Εισαγάγετε πολλές γραμμές σε ένα καθορισμένο ευρετήριο, μετατοπίζοντας τα υπάρχοντα δεδομένα προς τα κάτω χωρίς να τα αντικαταστήσετε:
```csharp
// Εισαγωγή 10 γραμμών ξεκινώντας από τον δείκτη γραμμής 2 (τρίτη γραμμή)
worksheet.Cells.InsertRows(2, 10);
```
#### Βήμα 5: Αποθήκευση του τροποποιημένου αρχείου Excel σε νέα τοποθεσία
Αποθηκεύστε τις αλλαγές σας σε μια νέα θέση αρχείου, διατηρώντας τα αρχικά δεδομένα και αποθηκεύοντας τις τροποποιήσεις ξεχωριστά:
```csharp
// Αποθήκευση του τροποποιημένου βιβλίου εργασίας στον κατάλογο εξόδου
workbook.Save(outputFilePath);
```
#### Βήμα 6: Κλείστε το FileStream για να απελευθερώσετε πόρους
Να κλείνετε πάντα τις ροές μετά από λειτουργίες για να ελευθερώνετε πόρους συστήματος:
```csharp
// Κλείσιμο της ροής αρχείωνstream.Close();
```
