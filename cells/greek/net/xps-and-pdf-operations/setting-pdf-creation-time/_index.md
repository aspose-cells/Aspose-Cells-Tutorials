# Ρύθμιση χρόνου δημιουργίας PDF στο .NET

## Εισαγωγή
Στη σημερινή ψηφιακή εποχή, η δυνατότητα μετατροπής εγγράφων σε διαφορετικές μορφές είναι ζωτικής σημασίας για πολλές εφαρμογές. Μια κοινή ανάγκη είναι η μετατροπή υπολογιστικών φύλλων Excel σε αρχεία PDF. Αυτό όχι μόνο διατηρεί τη μορφοποίηση, αλλά κάνει επίσης την κοινή χρήση και την εκτύπωση πολύ πιο εύκολη. Εάν είστε προγραμματιστής που εργάζεται με .NET, το Aspose.Cells είναι μια φανταστική βιβλιοθήκη που απλοποιεί αυτήν τη διαδικασία. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στον τρόπο ορισμού του χρόνου δημιουργίας PDF κατά τη μετατροπή ενός αρχείου Excel σε PDF χρησιμοποιώντας το Aspose.Cells για .NET.
## Προαπαιτούμενα
Πριν προχωρήσουμε στις λεπτομέρειες του κώδικα, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ξεκινήσετε.
### Τι χρειάζεστε
1. Visual Studio: Βεβαιωθείτε ότι έχετε εγκατεστημένο το Visual Studio στον υπολογιστή σας. Αυτό θα είναι το περιβάλλον ανάπτυξής σας.
2. Aspose.Cells για .NET: Κατεβάστε τη βιβλιοθήκη Aspose.Cells από το [δικτυακός τόπος](https://releases.aspose.com/cells/net/)Μπορείτε επίσης να ξεκινήσετε με μια δωρεάν δοκιμαστική περίοδο για να δοκιμάσετε τις λειτουργίες του.
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.
4. Αρχείο Excel: Να έχετε ένα αρχείο Excel έτοιμο για μετατροπή. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε ένα αρχείο με το όνομα `Book1.xlsx`.
Τώρα που έχετε τακτοποιήσει τις προϋποθέσεις, ας περάσουμε στο διασκεδαστικό κομμάτι—την εισαγωγή των απαραίτητων πακέτων και τη σύνταξη του κώδικα!
## Εισαγωγή πακέτων
Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαιτούμενους χώρους ονομάτων στο αρχείο C# σας. Αυτό είναι κρίσιμο, καθώς σας επιτρέπει να έχετε πρόσβαση στις κλάσεις και τις μεθόδους που παρέχονται από τη βιβλιοθήκη Aspose.Cells.
### Ανοίξτε το έργο σας C#
Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο ή ανοίξτε ένα υπάρχον όπου θέλετε να εφαρμόσετε τη λειτουργία μετατροπής PDF.
### Προσθήκη αναφοράς Aspose.Cells
Μπορείτε να προσθέσετε τη βιβλιοθήκη Aspose.Cells στο έργο σας κάνοντας δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων, επιλέγοντας "Διαχείριση πακέτων NuGet" και αναζητώντας το "Aspose.Cells". Εγκαταστήστε το πακέτο.
### Εισαγωγή χώρων ονομάτων
Στο επάνω μέρος του αρχείου C#, συμπεριλάβετε τους ακόλουθους χώρους ονομάτων:
```csharp
using System;
using Aspose.Cells;
using Aspose.Cells.Charts;
```
Αυτοί οι χώροι ονομάτων θα σας δώσουν πρόσβαση στην κλάση Βιβλίου εργασίας και σε άλλες βασικές λειτουργίες.

Τώρα που έχουμε εισάγει τα πακέτα μας, ας αναλύσουμε τη διαδικασία μετατροπής ενός αρχείου Excel σε PDF, ορίζοντας παράλληλα την ώρα δημιουργίας.
## Βήμα 1: Ορισμός του καταλόγου εγγράφων
Αρχικά, πρέπει να καθορίσετε τον κατάλογο στον οποίο αποθηκεύονται τα έγγραφά σας. Εκεί βρίσκεται το αρχείο Excel και εκεί θα αποθηκευτεί το PDF που θα προκύψει.
```csharp
string dataDir = "Your Document Directory"; // Καθορίστε τον κατάλογο εγγράφων σας
```
Αντικαθιστώ `"Your Document Directory"` με την πραγματική διαδρομή όπου εσύ `Book1.xlsx` βρίσκεται το αρχείο. Αυτή η διαδρομή θα βοηθήσει την εφαρμογή να εντοπίσει το αρχείο για επεξεργασία.
## Βήμα 2: Φόρτωση του αρχείου Excel
Στη συνέχεια, θα φορτώσετε το αρχείο Excel σε ένα `Workbook` αντικείμενο. Εδώ είναι που το Aspose.Cells ξεχωρίζει, καθώς σας επιτρέπει να εργάζεστε με αρχεία Excel χωρίς κόπο.
```csharp
string inputPath = dataDir + "Book1.xlsx"; // Διαδρομή προς το αρχείο Excel σας
Workbook workbook = new Workbook(inputPath); // Φόρτωση του αρχείου Excel
```
Ο `Workbook` Η κλάση χρησιμοποιείται για τη φόρτωση και τον χειρισμό αρχείων Excel. Περνώντας τη διαδρομή εισόδου, υποδεικνύετε στην εφαρμογή με ποιο αρχείο να εργαστεί.
## Βήμα 3: Δημιουργία PdfSaveOptions
Τώρα, ήρθε η ώρα να δημιουργήσουμε μια παρουσία του `PdfSaveOptions`Αυτή η κλάση σάς επιτρέπει να καθορίσετε διάφορες επιλογές για την αποθήκευση του βιβλίου εργασίας σας ως PDF, συμπεριλαμβανομένης της ώρας δημιουργίας.
```csharp
PdfSaveOptions options = new PdfSaveOptions(); // Δημιουργία στιγμιότυπου PdfSaveOptions
options.CreatedTime = DateTime.Now; // Ορίστε την ώρα δημιουργίας σε τώρα
```
Ρυθμίζοντας `options.CreatedTime` να `DateTime.Now`, διασφαλίζετε ότι το PDF θα αντικατοπτρίζει την τρέχουσα ημερομηνία και ώρα δημιουργίας του.
## Βήμα 4: Αποθήκευση του βιβλίου εργασίας ως PDF
Τέλος, θα αποθηκεύσετε το βιβλίο εργασίας ως αρχείο PDF χρησιμοποιώντας τις επιλογές που μόλις ορίσατε.
```csharp
workbook.Save(dataDir + "output.pdf", options); // Αποθήκευση ως PDF
```
Αυτή η γραμμή κώδικα παίρνει το βιβλίο εργασίας και το αποθηκεύει σε μορφή PDF στην καθορισμένη θέση. `options` Η παράμετρος μεταβιβάζεται για να συμπεριληφθεί η ώρα δημιουργίας στα μεταδεδομένα PDF.
