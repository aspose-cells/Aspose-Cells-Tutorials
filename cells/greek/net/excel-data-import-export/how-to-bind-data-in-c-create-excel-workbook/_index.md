# Πώς να Συνδέσετε Δεδομένα σε C# – Δημιουργία Βιβλίου Εργασίας Excel

Έχετε αναρωτηθεί ποτέ **πώς να συνδέσετε δεδομένα** σε ένα γράφημα σε C# χωρίς να τρελαίνεστε; Δεν είστε οι μόνοι. Πολλοί προγραμματιστές συναντούν δυσκολίες όταν πρέπει να δημιουργήσουν προγραμματιστικά αρχεία Excel που πραγματικά *μοιάζουν* με αυτά που θα φτιάχνανε χειροκίνητα.  

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πλήρες, έτοιμο‑για‑εκτέλεση παράδειγμα που δημιουργεί ένα βιβλίο εργασίας Excel, το γεμίζει με δεδομένα, συνδέει αυτά τα δεδομένα με ένα γράφημα Waterfall και τελικά αποθηκεύει το αρχείο ως `.xlsx`. Στο τέλος θα γνωρίζετε ακριβώς **πώς να αποθηκεύσετε βιβλίο εργασίας ως XLSX**, **πώς να προσθέσετε γράφημα** σε ένα φύλλο και **πώς να εξάγετε Excel με γράφημα** για αναφορές.

> **Προαπαιτούμενα** – Χρειάζεστε Aspose.Cells για .NET (η δωρεάν δοκιμή λειτουργεί) και ένα περιβάλλον ανάπτυξης .NET όπως το Visual Studio 2022. Δεν απαιτούνται άλλα πακέτα NuGet.

---

## Τι Καλύπτει Αυτός Ο Οδηγός

- **Δημιουργία βιβλίου εργασίας Excel C#** – δημιουργήστε ένα νέο `Workbook` και ένα φύλλο εργασίας.  
- **Πώς να συνδέσετε δεδομένα** – αντιστοιχίστε τις αριθμητικές σειρές και τις ετικέτες κατηγοριών στην πηγή δεδομένων του γραφήματος.  
- **Πώς να προσθέσετε γράφημα** – εισάγετε ένα γράφημα Waterfall και ρυθμίστε τον τίτλο του.  
- **Αποθήκευση βιβλίου εργασίας ως XLSX** – αποθηκεύστε το αρχείο στο δίσκο ώστε όποιος να μπορεί να το ανοίξει στο Excel.  
- **Εξαγωγή Excel με γράφημα** – το τελικό προϊόν είναι ένα πλήρως λειτουργικό βιβλίο εργασίας που μπορείτε να μοιραστείτε.

Αν είστε άνετοι με τη βασική σύνταξη C#, θα βρείτε αυτό το tutorial παιχνιδάκι. Ας ξεκινήσουμε.

## Βήμα 1: Δημιουργία Βιβλίου Εργασίας Excel σε C#  

Πρώτα απ' όλα – χρειαζόμαστε ένα αντικείμενο βιβλίου εργασίας για να δουλέψουμε. Σκεφτείτε την κλάση `Workbook` ως το κενό σημειωματάριο που θα γεμίσετε αργότερα με σελίδες (φύλλα εργασίας) και περιεχόμενο.

```csharp
using Aspose.Cells;
using Aspose.Cells.Charts;

class WaterfallDemo
{
    static void Main()
    {
        // Initialize a new workbook – this is your blank Excel file.
        Workbook workbook = new Workbook();

        // Grab the first worksheet (index 0). It’s already created for us.
        Worksheet worksheet = workbook.Worksheets[0];
```

> **Συμβουλή:** Αν χρειαστείτε πολλαπλά φύλλα, απλώς καλέστε `workbook.Worksheets.Add()` και κρατήστε μια αναφορά σε κάθε νέο `Worksheet`.

---

## Βήμα 2: Συμπλήρωση του Φύλλου Εργασίας με Κατηγορίες και Τιμές  

Τώρα θα **δημιουργήσουμε δεδομένα σε στυλ excel workbook c#**. Το παράδειγμα χρησιμοποιεί ένα κλασικό σενάριο Waterfall: start, revenue, cost, profit, και end.  

```csharp
        // Add header labels.
        worksheet.Cells["A1"].PutValue("Category");
        worksheet.Cells["B1"].PutValue("Amount");

        // Sample data – you can replace these with your own source (database, API, etc.).
        string[] categoryLabels = { "Start", "Revenue", "Cost", "Profit", "End" };
        double[] values = { 0, 150, -70, 0, 80 };

        // Fill rows 2‑6 with the data.
        for (int i = 0; i < categoryLabels.Length; i++)
        {
            worksheet.Cells[i + 1, 0].PutValue(categoryLabels[i]); // Column A
            worksheet.Cells[i + 1, 1].PutValue(values[i]);       // Column B
        }
```

Γιατί βάζουμε `0` για το “Start” και το “Profit”; Σε ένα γράφημα Waterfall αυτά τα μηδενικά λειτουργούν ως *συνδέσμοι* που κάνουν τη ροή του γραφήματος σωστή. Αν τα παραλείψετε, το γράφημα θα φαίνεται σπασμένο.

---

## Βήμα 3: Πώς να Προσθέσετε Γράφημα – Εισαγωγή Γραφήματος Waterfall  

Με τα δεδομένα στη θέση τους, ήρθε η ώρα να **πώς να προσθέσετε γράφημα**. Το Aspose.Cells το κάνει τόσο εύκολο όσο η κλήση του `Charts.Add`.

```csharp
        // Insert a Waterfall chart starting at row 7, column 0 and spanning to row 25, column 10.
        int chartIndex = worksheet.Charts.Add(ChartType.Waterfall, 7, 0, 25, 10);
        Chart waterfallChart = worksheet.Charts[chartIndex];

        // Give the chart a meaningful title.
        waterfallChart.Title.Text = "Quarterly Waterfall";
```

Οι συντεταγμένες `(7,0,25,10)` ορίζουν το αριστερό‑πάνω κελί και το δεξιό‑κάτω κελί του πλαισίου του γραφήματος. Προσαρμόστε τις ώστε να ταιριάζουν με τη διάταξή σας.

---

## Βήμα 4: Πώς να Συνδέσετε Δεδομένα – Σύνδεση Σειρών και Κατηγοριών  

Εδώ είναι η καρδιά του tutorial: **πώς να συνδέσετε δεδομένα** στο γράφημα. Η μέθοδος `NSeries.Add` παίρνει την περιοχή των τιμών Y, ενώ το `CategoryData` δείχνει στις ετικέτες του άξονα X.

```csharp
        // Bind the numeric series (values) – the second parameter “true” tells Aspose to treat it as a series.
        waterfallChart.NSeries.Add("B2:B6", true);

        // Bind the category (X‑axis) labels.
        waterfallChart.NSeries.CategoryData = "A2:A6";
```

Παρατηρήστε ότι αναφερόμαστε στα ίδια κελιά που γεμίσαμε νωρίτερα (`A2:A6` για κατηγορίες, `B2:B6` για ποσά). Αν αλλάξετε τη διάταξη των δεδομένων, απλώς ενημερώστε αυτές τις περιοχές ανάλογα.

## Βήμα 5: Αποθήκευση Βιβλίου Εργασίας ως XLSX – Διατήρηση του Αρχείου  

Τέλος, **αποθηκεύουμε το βιβλίο εργασίας ως XLSX**. Η μέθοδος `Save` επιλέγει αυτόματα τη σωστή μορφή βάσει της επέκτασης του αρχείου.

```csharp
        // Save the workbook to disk. Replace YOUR_DIRECTORY with an actual path.
        workbook.Save("YOUR_DIRECTORY/WaterfallChart.xlsx");
    }
}
```

Όταν ανοίξετε το `WaterfallChart.xlsx` στο Excel, θα δείτε ένα ωραία αποδομένο γράφημα Waterfall που αντικατοπτρίζει τα δεδομένα που εισάγαμε. Αυτό ολοκληρώνει το **export excel with chart**.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις  

### Τι γίνεται αν η πηγή δεδομένων μου είναι δυναμική;  
Αντικαταστήστε τους στατικούς πίνακες με έναν βρόχο που διαβάζει από βάση δεδομένων ή API. Εφόσον γράφετε τις τιμές στην ίδια περιοχή κελιών, ο κώδικας σύνδεσης παραμένει αμετάβλητος.

### Μπορώ να αλλάξω τον τύπο του γραφήματος;  
Φυσικά. Αντικαταστήστε το `ChartType.Waterfall` με `ChartType.Column`, `ChartType.Line`, κ.λπ. Θυμηθείτε να προσαρμόσετε τα δεδομένα σειράς αν το νέο γράφημα απαιτεί διαφορετική διάταξη.

### Πώς ορίζω τα χρώματα του γραφήματος;  
Χρησιμοποιήστε `waterfallChart.NSeries[0].Format.Fill.ForeColor = Color.Yellow;` (ή οποιοδήποτε `System.Drawing.Color`). Αυτό είναι χρήσιμο όταν θέλετε η στήλη “Profit” να ξεχωρίζει.

### Τι κάνω αν χρειάζεται να εξάγω σε PDF αντί για XLSX;  
Καλείτε `workbook.Save("Report.pdf", SaveFormat.Pdf);`. Το γράφημα θα αποδοθεί αυτόματα στο PDF.
