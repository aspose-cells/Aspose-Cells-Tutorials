## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε λογικά βήματα για να διασφαλίσουμε τη σαφήνεια και την ευκολία κατανόησης. 
### 1. Ρύθμιση επιλογών μετατροπής
#### Επισκόπηση
Θα ξεκινήσουμε ρυθμίζοντας το `ImageOrPrintOptions` κλάση, η οποία παρέχει ρυθμίσεις ειδικά για εργασίες απόδοσης εικόνας.
**Οδηγός βήμα προς βήμα:**
##### Ορισμός τύπου εικόνας
Ορίστε τη μορφή εξόδου σε TIFF:
```csharp
ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.ImageType = ImageType.Tiff;
```
##### Προσθήκη προόδου επανάκλησης
Συνδέστε έναν χειριστή επανάκλησης για να παρακολουθείτε την πρόοδο αποθήκευσης σελίδας:
```csharp
opts.PageSavingCallback = new TestTiffPageSavingCallback();
```
### 2. Υλοποίηση της Επανάκλησης Αποθήκευσης Σελίδας
#### Επισκόπηση
Προσαρμόστε τις σελίδες που αποδίδονται και παρακολουθήστε την πρόοδο της απόδοσης με επανακλήσεις.
**Οδηγός βήμα προς βήμα:**
##### Δημιουργία προσαρμοσμένης κλάσης επανάκλησης
Ορίστε την κλάση επανάκλησής σας υλοποιώντας `IPageSavingCallback`:
```csharp
public class TestTiffPageSavingCallback : IPageSavingCallback
{
    public void PageStartSaving(PageStartSavingArgs args)
    {
        Console.WriteLine("Start saving page index {0} of pages {1}", args.PageIndex, args.PageCount);
        
        // Μην εμφανίζετε σελίδες πριν από το ευρετήριο 2
        if (args.PageIndex < 2)
        {
            args.IsToOutput = false;
        }
    }

    public void PageEndSaving(PageEndSavingArgs args)
    {
        Console.WriteLine("End saving page index {0} of pages {1}", args.PageIndex, args.PageCount);

        // Διακοπή εξόδου μετά τον δείκτη σελίδας 8
        if (args.PageIndex >= 8)
        {
            args.HasMorePages = false;
        }
    }
}
```
### 3. Εκτέλεση της Διαδικασίας Μετατροπής
#### Επισκόπηση
Τέλος, αποδώστε το βιβλίο εργασίας σας σε εικόνα TIFF χρησιμοποιώντας `WorkbookRender`.
**Οδηγός βήμα προς βήμα:**
##### Βιβλίο εργασίας απόδοσης
Μετατρέψτε και αποθηκεύστε το έγγραφο με τις διαμορφωμένες επιλογές:
```csharp
WorkbookRender wr = new WorkbookRender(workbook, opts);
wr.ToImage("DocumentConversionProgressForTiff_out.tiff");
```
