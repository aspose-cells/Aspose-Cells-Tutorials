## Οδηγός Εφαρμογής

### Επισκόπηση
Αυτή η ενότητα θα σας καθοδηγήσει στη διαμόρφωση και την εκτέλεση μετατροπής από Excel σε HTML με κλιμακούμενες στήλες χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας σας
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας του Excel που θέλετε να μετατρέψετε. Αυτό περιλαμβάνει τη ρύθμιση των καταλόγων εισόδου και εξόδου:

```csharp
// Κατάλογος εισαγωγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Κατάλογος εξόδου
string outputDir = RunExamples.Get_OutputDirectory();
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML
Δημιουργήστε μια παρουσία του `HtmlSaveOptions` για να διαχειριστείτε τον τρόπο αποθήκευσης του αρχείου Excel ως HTML. Αυτό περιλαμβάνει την ενεργοποίηση κλιμακούμενων στηλών και την εξαγωγή εικόνων ως Base64.

```csharp
// Καθορισμός επιλογών αποθήκευσης HTML
HtmlSaveOptions options = new HtmlSaveOptions();

// Ορίστε την ιδιότητα για κλιμακωτό πλάτος
options.WidthScalable = true;

// Εξαγωγή εικόνων ως Base64 για ενσωμάτωσή τους σε HTML
options.ExportImagesAsBase64 = true;
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Τέλος, αποθηκεύστε το βιβλίο εργασίας σε ένα αρχείο HTML με τις διαμορφωμένες επιλογές σας:

```csharp
// Φόρτωση δείγματος αρχείου πηγαίου κώδικα
Workbook wb = new Workbook(sourceDir + "sampleForScalableColumns.xlsx");

// Αποθήκευση του βιβλίου εργασίας σε μορφή Html
wb.Save(outputDir + "outsampleForScalableColumns.html", options);
```
