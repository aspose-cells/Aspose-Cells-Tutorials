## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε συγκεκριμένα χαρακτηριστικά:
### Δημιουργία ενός βιβλίου εργασίας
**Επισκόπηση:** Η δημιουργία ενός βιβλίου εργασίας του Excel είναι απλή με το Aspose.Cells. Αυτή η λειτουργία δείχνει πώς να αρχικοποιήσετε ένα νέο βιβλίο εργασίας.
#### Βήμα προς βήμα εφαρμογή
**Δημιουργία νέου βιβλίου εργασίας:**
```csharp
using Aspose.Cells;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Δημιουργία ενός νέου βιβλίου εργασίας
var workBook = new Workbook();
```
**Εξήγηση:** Εδώ, `Workbook` δημιουργείται με τις προεπιλεγμένες ρυθμίσεις, έτοιμο για χειρισμό.
### Τροποποίηση κελιών σε φύλλα εργασίας
**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση και να τροποποιείτε κελιά μέσα στα φύλλα εργασίας ενός βιβλίου εργασίας του Excel.
#### Βήμα προς βήμα εφαρμογή
**Φύλλο εργασίας Access First:**
```csharp
var sheet1 = workBook.Worksheets[0];
sheet1.Cells["A1"].Value = "DEMO TEXT ON SHEET1";
```
**Προσθήκη και τροποποίηση νέου φύλλου εργασίας:**
```csharp
// Προσθήκη νέου φύλλου εργασίας στο βιβλίο εργασίας
workBook.Worksheets.Add(SheetType.Worksheet);

var sheet2 = workBook.Worksheets[1];
sheet2.Cells["A1"].Value = "DEMO TEXT ON SHEET2";
```
**Εξήγηση:** Η πρόσβαση στα κελιά γίνεται χρησιμοποιώντας τόσο ευρετήρια όσο και κλειδιά. Μπορείτε να προσθέσετε φύλλα εργασίας δυναμικά και να ορίσετε τιμές όπως απαιτείται.
### Ορισμός ευρετηρίου ενεργού φύλλου εργασίας
**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε ποιο φύλλο εργασίας είναι ενεργό εντός του βιβλίου εργασίας τη δεδομένη στιγμή.
#### Βήμα προς βήμα εφαρμογή
**Ορισμός ενεργού φύλλου εργασίας:**
```csharp
workBook.Worksheets.Add(SheetType.Worksheet);
// Ορίστε τον δείκτη του ενεργού φύλλου σε 1, καθιστώντας το Φύλλο2 το τρέχον ενεργό φύλλο εργασίας
workBook.Worksheets.ActiveSheetIndex = 1;
```
**Εξήγηση:** Ο `ActiveSheetIndex` ορίζεται χρησιμοποιώντας έναν ακέραιο αριθμό με βάση το μηδέν που αντιστοιχεί στη θέση του φύλλου εργασίας.
### Αποθήκευση βιβλίου εργασίας ως SVG
**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να αποθηκεύσετε ένα βιβλίο εργασίας του Excel σε μορφή SVG, αποδίδοντας μόνο το ενεργό φύλλο εργασίας.
#### Βήμα προς βήμα εφαρμογή
**Αποθήκευση ενεργού φύλλου εργασίας ως SVG:**
```csharp
workBook.Worksheets.Add(SheetType.Worksheet);
workBook.Worksheets.ActiveSheetIndex = 1;

// Αποθήκευση του βιβλίου εργασίας ως SVG
workBook.Save(outputDir + "Demo.svg");
```
**Εξήγηση:** Ο `Save` μέθοδος με `.svg` Η μορφή αποδίδει μόνο το ενεργό φύλλο εργασίας σε αρχείο SVG.
