Σε αυτό το σεμινάριο, εμβαθύνουμε στον τρόπο προστασίας και κατάργησης της προστασίας κοινόχρηστων βιβλίων εργασίας με κωδικό πρόσβασης χρησιμοποιώντας το Aspose.Cells για .NET. ## Εισαγωγή πακέτων
Για να ξεκινήσετε με το Aspose.Cells για .NET, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Προσθέστε τις ακόλουθες γραμμές στην αρχή του αρχείου C#:
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```
Αυτές οι εισαγωγές σάς δίνουν πρόσβαση στις κλάσεις και τις μεθόδους που θα χρησιμοποιήσετε για να χειριστείτε βιβλία εργασίας του Excel.
## Βήμα 1: Ορισμός του καταλόγου εξόδου
Πριν δημιουργήσουμε το βιβλίο εργασίας μας, πρέπει να καθορίσουμε πού θα αποθηκευτεί. Εδώ ορίζουμε τη διαδρομή προς τον κατάλογο εξόδου μας.
```csharp
// Κατάλογος εξόδου
string outputDir = "Your Document Directory"; // Ορίστε αυτό στην επιθυμητή διαδρομή εξόδου
```
Η χορδή `outputDir` θα πρέπει να υποδεικνύει έναν έγκυρο κατάλογο στον υπολογιστή σας όπου θέλετε να αποθηκευτεί το αρχείο Excel εξόδου. Μην ξεχάσετε να αντικαταστήσετε το `"Your Document Directory"` με την πραγματική διαδρομή του φακέλου σας.
## Βήμα 2: Δημιουργήστε ένα κενό αρχείο Excel
Στη συνέχεια, ας δημιουργήσουμε μια νέα παρουσία βιβλίου εργασίας. Αυτό είναι το βασικό βήμα όπου δηλώνουμε ένα κενό αρχείο Excel το οποίο θα επεξεργαστούμε αργότερα. 
```csharp
// Δημιουργία κενού αρχείου Excel
Workbook wb = new Workbook();
```
Εδώ, δημιουργούμε μια νέα παρουσία του `Workbook` κλάση, δημιουργώντας ουσιαστικά ένα κενό αρχείο Excel έτοιμο για προσαρμογή.
## Βήμα 3: Προστατέψτε το κοινόχρηστο βιβλίο εργασίας με κωδικό πρόσβασης
Τώρα έρχεται το διασκεδαστικό κομμάτι! Θα ορίσουμε έναν κωδικό πρόσβασης για να προστατεύσουμε το κοινόχρηστο βιβλίο εργασίας μας, διασφαλίζοντας ότι μόνο εξουσιοδοτημένοι χρήστες θα έχουν πρόσβαση στο περιεχόμενο.
```csharp
// Προστασία του κοινόχρηστου βιβλίου εργασίας με κωδικό πρόσβασης
wb.ProtectSharedWorkbook("1234");
```
Ο `ProtectSharedWorkbook` χρησιμοποιείται εδώ η μέθοδος, με κωδικό πρόσβασης `"1234"` ανατεθεί. Αυτό σημαίνει ότι για να επεξεργαστείτε το κοινόχρηστο βιβλίο εργασίας, πρέπει να γνωρίζετε αυτόν τον κωδικό πρόσβασης. Θεωρήστε τον ως την ψηφιακή σας κλειδαριά!
## Βήμα 4: (Προαιρετικό) Κατάργηση προστασίας του κοινόχρηστου βιβλίου εργασίας
Ας υποθέσουμε ότι αργότερα θα χρειαστεί να αποκτήσετε πρόσβαση στο κοινόχρηστο βιβλίο εργασίας χωρίς περιορισμούς. Μπορείτε εύκολα να καταργήσετε την προστασία του καταργώντας τα σχόλια στην παρακάτω γραμμή:
```csharp
// Αποσχολιάστε αυτήν τη γραμμή για να καταργήσετε την προστασία του κοινόχρηστου βιβλίου εργασίας
// wb.UnprotectSharedWorkbook("1234");
```
Χρησιμοποιώντας το `UnprotectSharedWorkbook` Η μέθοδος, με τον ίδιο κωδικό πρόσβασης, θα καταργήσει τυχόν περιορισμούς, επιτρέποντας την ελεύθερη πρόσβαση στο βιβλίο εργασίας. Αυτό το βήμα είναι απαραίτητο εάν θέλετε να επαναφέρετε τις αλλαγές μετά τη συνεργασία σε ένα έγγραφο.
## Βήμα 5: Αποθήκευση του αρχείου Excel εξόδου
Τέλος, αφού ολοκληρώσετε όλες τις τροποποιήσεις σας, ήρθε η ώρα να αποθηκεύσετε αυτό το λαμπερό νέο αρχείο Excel.
```csharp
// Αποθηκεύστε το αρχείο Excel που προκύπτει
wb.Save(outputDir + "outputProtectSharedWorkbook.xlsx");
```
Ο `Save` Η μέθοδος αποθηκεύει το βιβλίο εργασίας στον καθορισμένο κατάλογο εξόδου και ονομάζει το αρχείο `outputProtectSharedWorkbook.xlsx`Τώρα μπορείτε να εντοπίσετε το αρχείο σας εκεί που θέλατε!
## Βήμα 6: Επιβεβαίωση εκτέλεσης
Συνοψίζοντας, ας δώσουμε κάποια σχόλια για να ενημερώσουμε τον χρήστη ότι όλα εκτελέστηκαν με επιτυχία.
```csharp

```
Αυτή η γραμμή απλώς εκτυπώνει ένα μήνυμα στην κονσόλα, επιβεβαιώνοντας ότι η διαδικασία έχει ολοκληρωθεί. Είναι μια τελευταία πινελιά για να διασφαλίσουμε ότι η λειτουργία μας ήταν όχι μόνο λειτουργική αλλά και φιλική προς το χρήστη.
