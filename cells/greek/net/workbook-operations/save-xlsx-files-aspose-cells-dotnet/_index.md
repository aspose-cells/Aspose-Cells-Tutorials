## Οδηγός Εφαρμογής

Ας δούμε βήμα προς βήμα την αποθήκευση ενός αρχείου XLSX.

### Βήμα 1: Ρύθμιση του βιβλίου εργασίας σας

Αρχικά, δημιουργήστε ή φορτώστε ένα βιβλίο εργασίας όπου θα γίνεται χειρισμός ή αποθήκευση δεδομένων. Δείτε πώς:

#### Δημιουργία νέου βιβλίου εργασίας
```csharp
// Αρχικοποίηση μιας νέας παρουσίας βιβλίου εργασίας
Workbook workbook = new Workbook();
```
Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα κενό βιβλίο εργασίας που μπορείτε να συμπληρώσετε με δεδομένα.

### Βήμα 2: Ρύθμιση παραμέτρων της διαδικασίας αποθήκευσης

Τώρα, ρυθμίστε τον τρόπο αποθήκευσης του αρχείου σας:

#### Ρύθμιση παραμέτρων απόκρισης HTTP για λήψη αρχείου
Εάν χρησιμοποιείτε ASP.NET και χρειάζεται να στείλετε το αρχείο ως απάντηση με δυνατότητα λήψης, αρχικοποιήστε `HttpResponse`:
```csharp
HttpResponse Response = HttpContext.Current.Response;
```

#### Αποθήκευση βιβλίου εργασίας ως XLSX
Χρησιμοποιήστε τον ακόλουθο κώδικα για να αποθηκεύσετε το βιβλίο εργασίας σε μορφή xlsx του Excel 2007:
```csharp
// Βεβαιωθείτε ότι η απάντησή σας δεν είναι null πριν την αποθήκευση
if (Response != null)
{
    // Καθορισμός διάταξης περιεχομένου και επιλογών αποθήκευσης για τη μορφή Excel 2007
    workbook.Save(Response, "output.xlsx", ContentDisposition.Attachment, new OoxmlSaveOptions());
    Response.End(); // Τερματισμός της ροής απόκρισης HTTP
}
```

### Επεξήγηση παραμέτρων κώδικα
- **`HttpResponse`**: Διαχειρίζεται τον τρόπο αποστολής του αρχείου στους πελάτες.
- **`ContentDisposition.Attachment`**: Δίνει εντολή στο πρόγραμμα περιήγησης να αντιμετωπίσει το έγγραφο ως αρχείο με δυνατότητα λήψης αντί να το εμφανίσει ενσωματωμένα.
- **`OoxmlSaveOptions`**Παρέχει επιλογές ειδικά για αποθήκευση σε μορφές OLE2 όπως XLSX.

### Συμβουλές αντιμετώπισης προβλημάτων
Συνήθη προβλήματα που ενδέχεται να αντιμετωπίσετε περιλαμβάνουν:
- **Εξαίρεση Μηδενικής Αναφοράς**: Βεβαιωθείτε `HttpResponse` έχει αρχικοποιηθεί σωστά πριν από τη χρήση.
- **Δεν γίνεται λήψη του αρχείου**Ελέγξτε ότι η διαδρομή αρχείου και οι κεφαλίδες HTTP έχουν ρυθμιστεί σωστά για λήψεις από υπολογιστές-πελάτες.
