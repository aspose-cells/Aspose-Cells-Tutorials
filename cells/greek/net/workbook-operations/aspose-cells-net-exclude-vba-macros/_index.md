## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο ρύθμισης παραμέτρων επιλογών φόρτωσης και χειρισμού βιβλίων εργασίας Excel χρησιμοποιώντας το Aspose.Cells για .NET.

### Χαρακτηριστικό 1: Ρύθμιση παραμέτρων LoadOptions

#### Επισκόπηση
Η πρώτη λειτουργία εστιάζει στη ρύθμιση παραμέτρων των επιλογών φόρτωσης για την εξαίρεση μακροεντολών VBA κατά τη φόρτωση ενός βιβλίου εργασίας του Excel. Αυτό είναι ιδιαίτερα χρήσιμο εάν χρειάζεται να επεξεργαστείτε δεδομένα χωρίς την επιβάρυνση των ενσωματωμένων σεναρίων.

**Βήμα προς βήμα εφαρμογή**

1. **Δημιουργήστε μια νέα παρουσία του LoadOptions**
   Ξεκινήστε δημιουργώντας ένα `LoadOptions` αντικείμενο, ρυθμίζοντάς το να ανιχνεύει αυτόματα τις μορφές αρχείων.
   
    ```csharp
    LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
    ```

2. **Εξαίρεση μακροεντολών VBA χρησιμοποιώντας το LoadFilter**
   Ρυθμίστε τις παραμέτρους του φίλτρου ώστε να εξαιρεί μακροεντολές VBA ενώ επιτρέπει άλλους τύπους δεδομένων.

    ```csharp
    loadOptions.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.VBA);
    ```

### Λειτουργία 2: Φόρτωση βιβλίου εργασίας χωρίς VBA

#### Επισκόπηση
Στη συνέχεια, θα δείξουμε πώς να χρησιμοποιήσετε το διαμορφωμένο `LoadOptions` για να ανοίξετε ένα βιβλίο εργασίας εξαιρώντας τα στοιχεία VBA του.

**Βήμα προς βήμα εφαρμογή**

1. **Ορισμός καταλόγων προέλευσης και εξόδου**
   Βεβαιωθείτε ότι έχετε καθορίσει τις διαδρομές καταλόγων όπου αποθηκεύονται τα αρχεία Excel και πού θα πρέπει να αποθηκεύεται η έξοδος.
   
    ```csharp
    string sourceDir = "YOUR_SOURCE_DIRECTORY";
    string outputDir = "YOUR_OUTPUT_DIRECTORY";
    ```

2. **Φόρτωση του βιβλίου εργασίας με εξαιρούμενη VBA**

    ```csharp
    Workbook workbook = new Workbook(sourceDir + "/sampleMacroEnabledWorkbook.xlsm", loadOptions);
    ```
   Το βιβλίο εργασίας φορτώνεται τώρα χωρίς τις μακροεντολές VBA, χάρη στις διαμορφωμένες μακροεντολές μας. `loadOptions`.

### Δυνατότητα 3: Αποθήκευση βιβλίου εργασίας σε συγκεκριμένη μορφή

#### Επισκόπηση
Τέλος, θα αποθηκεύσουμε το τροποποιημένο βιβλίο εργασίας σε μια συγκεκριμένη μορφή, διατηρώντας παράλληλα τις δυνατότητες που δεν είναι VBA.

**Βήμα προς βήμα εφαρμογή**

1. **Αποθήκευση βιβλίου εργασίας σε μορφή XLSM**
   Χρησιμοποιήστε το `Save` μέθοδος για την αποθήκευση του βιβλίου εργασίας σας με τις επιθυμητές ρυθμίσεις.
   
    ```csharp
    workbook.Save(outputDir + "/OutputSampleMacroEnabledWorkbook.xlsm", SaveFormat.Xlsm);
    ```
