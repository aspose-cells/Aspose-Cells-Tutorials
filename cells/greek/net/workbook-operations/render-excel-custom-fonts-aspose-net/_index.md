## Οδηγός Εφαρμογής

### Απόδοση σε PNG με προσαρμοσμένη προεπιλεγμένη γραμματοσειρά

Η απόδοση ενός φύλλου εργασίας του Excel σε μορφή PNG, ενώ παράλληλα ορίζεται μια προσαρμοσμένη προεπιλεγμένη γραμματοσειρά, διασφαλίζει την οπτική συνέπεια. Δείτε πώς:

#### Βήμα 1: Ρύθμιση παραμέτρων επιλογών εικόνας

Ρυθμίστε τις παραμέτρους απόδοσης για την έξοδο της εικόνας σας.
```csharp
using Aspose.Cells;
using Aspose.Cells.Rendering;

// Καθορίστε καταλόγους.
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Ανοίξτε ένα αρχείο Excel.
Workbook workbook = new Workbook(SourceDir + "sampleSetDefaultFontPropertyOfPdfSaveOptionsAndImageOrPrintOptions.xlsx");

// Ρυθμίστε τις επιλογές απόδοσης εικόνας.
ImageOrPrintOptions imgOpt = new ImageOrPrintOptions();
imgOpt.ImageType = Drawing.ImageType.Png;
imgOpt.CheckWorkbookDefaultFont = false; // Χρησιμοποιήστε μια προσαρμοσμένη γραμματοσειρά για γραμματοσειρές που λείπουν στο βιβλίο εργασίας.
imgOpt.DefaultFont = "Times New Roman";
```

#### Βήμα 2: Απόδοση και αποθήκευση

Αποδώστε το φύλλο εργασίας σας σε αρχείο εικόνας χρησιμοποιώντας αυτές τις ρυθμίσεις.
```csharp
// Αποδώστε το πρώτο φύλλο εργασίας σε εικόνα PNG.
SheetRender sr = new SheetRender(workbook.Worksheets[0], imgOpt);
sr.ToImage(0, outputDir + "out1_imagePNG.png");
```

### Απόδοση σε TIFF με προσαρμοσμένη προεπιλεγμένη γραμματοσειρά

Η μορφή TIFF είναι ιδανική για εικόνες υψηλής ποιότητας. Δείτε πώς μπορείτε να αποδώσετε ένα ολόκληρο βιβλίο εργασίας ως αρχείο TIFF:

#### Βήμα 3: Ρύθμιση επιλογών εικόνας για TIFF

Ρυθμίστε τις παραμέτρους απόδοσης ειδικά για την έξοδο TIFF.
```csharp
// Επαναχρησιμοποιήστε προηγουμένως ορισμένους καταλόγους και ανοίξτε το αρχείο Excel.
Workbook workbook = new Workbook(SourceDir + "sampleSetDefaultFontPropertyOfPdfSaveOptionsAndImageOrPrintOptions.xlsx");

// Ρυθμίστε τις παραμέτρους απόδοσης εικόνας για TIFF.
imgOpt.ImageType = Drawing.ImageType.Tiff;
```

#### Βήμα 4: Απόδοση ολόκληρου του βιβλίου εργασίας σε μορφή TIFF

Μετατρέψτε ολόκληρο το βιβλίο εργασίας σε ένα μόνο αρχείο TIFF.
```csharp
// Αποδώστε το βιβλίο εργασίας ως εικόνα TIFF.
WorkbookRender wr = new WorkbookRender(workbook, imgOpt);
wr.ToImage(outputDir + "out1_imageTIFF.tiff");
```

### Απόδοση σε PDF με προσαρμοσμένη προεπιλεγμένη γραμματοσειρά

Η αποθήκευση ενός βιβλίου εργασίας Excel ως PDF, διασφαλίζοντας παράλληλα τη συνέπεια της γραμματοσειράς, είναι ζωτικής σημασίας για την επαγγελματική τεκμηρίωση.

#### Βήμα 5: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF

Ορίστε τις απαραίτητες επιλογές για την αποθήκευση του αρχείου σας ως PDF.
```csharp
using Aspose.Cells;

// Ανοίξτε ξανά το βιβλίο εργασίας.
Workbook workbook = new Workbook(SourceDir + "sampleSetDefaultFontPropertyOfPdfSaveOptionsAndImageOrPrintOptions.xlsx");

// Ρυθμίστε τις επιλογές αποθήκευσης PDF.
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.DefaultFont = "Times New Roman";
saveOptions.CheckWorkbookDefaultFont = false; // Χρησιμοποιήστε μια προσαρμοσμένη γραμματοσειρά για γραμματοσειρές που λείπουν στο βιβλίο εργασίας.
```

#### Βήμα 6: Αποθήκευση ως PDF

Εξαγάγετε το βιβλίο εργασίας σας σε έγγραφο PDF.
```csharp
// Αποθηκεύστε το βιβλίο εργασίας ως αρχείο PDF.
workbook.Save(outputDir + "out1_pdf.pdf", saveOptions);
```
