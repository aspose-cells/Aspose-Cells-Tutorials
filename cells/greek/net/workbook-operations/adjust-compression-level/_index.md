# Προσαρμογή επιπέδου συμπίεσης στο βιβλίο εργασίας

## Εισαγωγή
Όσον αφορά τη διαχείριση μεγάλων αρχείων Excel, η συμπίεση αλλάζει τα δεδομένα. Όχι μόνο εξοικονομεί χώρο αποθήκευσης, αλλά κάνει επίσης τις μεταφορές αρχείων ταχύτερες και πιο αποτελεσματικές. Εάν εργάζεστε με το Aspose.Cells για .NET, μπορείτε εύκολα να προσαρμόσετε το επίπεδο συμπίεσης των βιβλίων εργασίας σας. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στη διαδικασία βήμα προς βήμα, διασφαλίζοντας ότι κατανοείτε κάθε μέρος του κώδικα και τον τρόπο λειτουργίας του.
## Προαπαιτούμενα
Πριν προχωρήσετε στον κώδικα, υπάρχουν μερικές προϋποθέσεις που πρέπει να έχετε στη διάθεσή σας:
1. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.
2. Βιβλιοθήκη Aspose.Cells: Πρέπει να έχετε εγκατεστημένη τη βιβλιοθήκη Aspose.Cells. Μπορείτε να την κατεβάσετε από [εδώ](https://releases.aspose.com/cells/net/).
3. Visual Studio: Ένα περιβάλλον ανάπτυξης όπως το Visual Studio θα είναι απαραίτητο για την εκτέλεση του κώδικα.
4. .NET Framework: Βεβαιωθείτε ότι το έργο σας έχει ρυθμιστεί με μια συμβατή έκδοση του .NET Framework.
## Εισαγωγή πακέτων
Για να ξεκινήσετε, πρέπει να εισαγάγετε τα απαραίτητα πακέτα στο έργο σας σε C#. Δείτε πώς μπορείτε να το κάνετε:
```csharp
using Aspose.Cells.Rendering;
using Aspose.Cells.WebExtensions;
using System;
```
Αυτά τα πακέτα είναι απαραίτητα για την εργασία με αρχεία Excel χρησιμοποιώντας τη βιβλιοθήκη Aspose.Cells. `Aspose.Cells` Ο χώρος ονομάτων περιέχει όλες τις κλάσεις που χρειάζεστε για να χειριστείτε αρχεία Excel, ενώ `Aspose.Cells.Xlsb` παρέχει τις επιλογές για την αποθήκευση αρχείων σε μορφή XLSB.
Τώρα, ας αναλύσουμε τη διαδικασία προσαρμογής του επιπέδου συμπίεσης σε ένα βιβλίο εργασίας σε διαχειρίσιμα βήματα.
## Βήμα 1: Ορισμός καταλόγων προέλευσης και εξόδου
Αρχικά, πρέπει να καθορίσετε πού βρίσκονται τα αρχεία προέλευσης και πού θέλετε να αποθηκεύσετε τα αρχεία εξόδου. Αυτό είναι κρίσιμο για να διασφαλίσετε ότι το πρόγραμμά σας γνωρίζει πού να βρει τα αρχεία με τα οποία χρειάζεται να εργαστεί.
```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory";
string outDir = "Your Document Directory";
```
Αντικαθιστώ `"Your Document Directory"` με την πραγματική διαδρομή προς τους καταλόγους σας. Αυτό θα βοηθήσει το πρόγραμμα να εντοπίσει τα αρχεία που θέλετε να συμπιέσετε.
## Βήμα 2: Φόρτωση του βιβλίου εργασίας
Στη συνέχεια, θα φορτώσετε το βιβλίο εργασίας που θέλετε να συμπιέσετε. Εδώ ξεκινά η μαγεία!
```csharp
Workbook workbook = new Workbook(sourceDir + "LargeSampleFile.xlsx");
```
Σε αυτήν τη γραμμή, δημιουργούμε μια νέα παρουσία του `Workbook` κλάση και φορτώστε ένα υπάρχον αρχείο Excel. Βεβαιωθείτε ότι το όνομα του αρχείου ταιριάζει με αυτό που έχετε στον κατάλογο προέλευσης.
## Βήμα 3: Ρύθμιση επιλογών αποθήκευσης
Τώρα είναι η ώρα να διαμορφώσουμε τις επιλογές αποθήκευσης. Θα ορίσουμε τον τύπο συμπίεσης για το αρχείο εξόδου. 
```csharp
XlsbSaveOptions options = new XlsbSaveOptions();
```
Ο `XlsbSaveOptions` Η κλάση σάς επιτρέπει να καθορίσετε διάφορες επιλογές κατά την αποθήκευση του βιβλίου εργασίας σας σε μορφή XLSB, συμπεριλαμβανομένων των επιπέδων συμπίεσης.
## Βήμα 4: Μέτρηση χρόνου συμπίεσης για το Επίπεδο 1
Ας ξεκινήσουμε με το πρώτο επίπεδο συμπίεσης. Θα μετρήσουμε πόσο χρόνο χρειάζεται για να αποθηκευτεί το βιβλίο εργασίας με αυτό το επίπεδο συμπίεσης.
```csharp
options.CompressionType = OoxmlCompressionType.Level1;
var watch = Stopwatch.StartNew();
workbook.Save(outDir + "LargeSampleFile_level_1_out.xlsb", options);
watch.Stop();
var elapsedMs = watch.ElapsedMilliseconds;
Console.WriteLine("Level 1 Elapsed Time: " + elapsedMs);
```
Εδώ, ορίζουμε τον τύπο συμπίεσης σε Επίπεδο 1, αποθηκεύουμε το βιβλίο εργασίας και, στη συνέχεια, μετράμε τον χρόνο που έχει παρέλθει. Αυτό μας δίνει μια ιδέα για το πόσο διαρκεί η διαδικασία.
## Βήμα 5: Μέτρηση χρόνου συμπίεσης για το Επίπεδο 6
Στη συνέχεια, ας δούμε πώς αποδίδει η συμπίεση Επιπέδου 6.
```csharp
watch = Stopwatch.StartNew();
options.CompressionType = OoxmlCompressionType.Level6;
workbook.Save(outDir + "LargeSampleFile_level_6_out.xlsb", options);
watch.Stop();
elapsedMs = watch.ElapsedMilliseconds;
Console.WriteLine("Level 6 Elapsed Time: " + elapsedMs);
```
Αυτό το βήμα είναι παρόμοιο με το προηγούμενο, αλλά αλλάζουμε το επίπεδο συμπίεσης σε Επίπεδο 6. Θα παρατηρήσετε ότι ο χρόνος που απαιτείται μπορεί να διαφέρει ανάλογα με την πολυπλοκότητα του βιβλίου εργασίας.
## Βήμα 6: Μέτρηση χρόνου συμπίεσης για το Επίπεδο 9
Τέλος, ας δούμε την απόδοση με το υψηλότερο επίπεδο συμπίεσης.
```csharp
watch = Stopwatch.StartNew();
options.CompressionType = OoxmlCompressionType.Level9;
workbook.Save(outDir + "LargeSampleFile_level_9_out.xlsb", options);
watch.Stop();
elapsedMs = watch.ElapsedMilliseconds;
Console.WriteLine("Level 9 Elapsed Time: " + elapsedMs);
```
Σε αυτό το βήμα, ορίζουμε το επίπεδο συμπίεσης στο Επίπεδο 9. Εδώ συνήθως θα δείτε τη σημαντικότερη μείωση στο μέγεθος του αρχείου, αλλά η επεξεργασία της μπορεί να διαρκέσει περισσότερο.
## Βήμα 7: Τελικό αποτέλεσμα
Αφού εκτελέσετε όλα τα επίπεδα συμπίεσης, μπορείτε να εμφανίσετε ένα μήνυμα που υποδεικνύει ότι η διαδικασία ολοκληρώθηκε με επιτυχία.
```csharp
Console.WriteLine("AdjustCompressionLevel executed successfully.");
```
Αυτή η απλή γραμμή κώδικα επιβεβαιώνει ότι το πρόγραμμά σας έχει ολοκληρωθεί χωρίς προβλήματα.
## Σύναψη
Η προσαρμογή του επιπέδου συμπίεσης των βιβλίων εργασίας σας χρησιμοποιώντας το Aspose.Cells για .NET είναι μια απλή διαδικασία που μπορεί να οδηγήσει σε σημαντικά οφέλη όσον αφορά το μέγεθος και την απόδοση του αρχείου. Ακολουθώντας τα βήματα που περιγράφονται σε αυτόν τον οδηγό, μπορείτε εύκολα να εφαρμόσετε συμπίεση στις εφαρμογές σας και να βελτιώσετε την αποτελεσματικότητα της διαχείρισης αρχείων Excel.
