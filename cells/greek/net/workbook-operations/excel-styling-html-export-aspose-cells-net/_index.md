## Οδηγός Εφαρμογής

### Δημιουργία και διαμόρφωση κελιού

Μάθετε πώς να δημιουργείτε ένα βιβλίο εργασίας του Excel, να αποκτάτε πρόσβαση σε συγκεκριμένα κελιά και να εφαρμόζετε προσαρμοσμένα στυλ.

#### Επισκόπηση

Θα ξεκινήσουμε δημιουργώντας ένα βιβλίο εργασίας, αποκτώντας πρόσβαση στο κελί "B5", προσθέτοντας περιεχόμενο κειμένου και διαμορφώνοντάς το με κόκκινο χρώμα γραμματοσειράς.

#### Βήμα προς βήμα εφαρμογή

1. **Δημιουργία βιβλίου εργασίας και κελιού πρόσβασης**
   
   Αρχικοποιήστε το βιβλίο εργασίας σας και επιλέξτε το φύλλο εργασίας:
   
   ```csharp
   using Aspose.Cells;
   using System.Drawing;
   
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";
   
   Workbook wb = new Workbook();
   Worksheet ws = wb.Worksheets[0];
   Cell cell = ws.Cells["B5"];
   ```

2. **Ορισμός τιμής και στυλ κελιού**
   
   Προσθήκη κειμένου στο κελί και εφαρμογή κόκκινου χρώματος γραμματοσειράς:
   
   ```csharp
   cell.PutValue("This is some text.");
   Style st = cell.GetStyle();
   st.Font.Color = Color.Red;
   cell.SetStyle(st);
   ```

#### Βασικές επιλογές διαμόρφωσης
- **Χρώμα γραμματοσειράς**: Προσαρμόστε με οποιοδήποτε `System.Drawing.Color` αξία.
- **Τιμή κελιού**: Χρήση `.PutValue()` για διάφορους τύπους δεδομένων.

### Εξαγωγή βιβλίου εργασίας ως HTML με ξεχωριστό CSS

Μάθετε πώς να εξάγετε ένα βιβλίο εργασίας με στυλ σε μορφή HTML, επιτρέποντας ξεχωριστό στυλ CSS για κάθε φύλλο εργασίας.

#### Επισκόπηση

Θα εξαγάγουμε το στυλιζαρισμένο βιβλίο εργασίας σε μορφή HTML και θα το ρυθμίσουμε ώστε να έχει το CSS διαχωρισμένο από το περιεχόμενο.

#### Βήμα προς βήμα εφαρμογή

1. **Εξαγωγή βιβλίου εργασίας**
   
   Αφού ρυθμίσετε το στυλ κελιού σας, χρησιμοποιήστε `HtmlSaveOptions` για να ορίσετε πώς θέλετε να εμφανίζεται η HTML έξοδος:
   
   ```csharp
   HtmlSaveOptions opts = new HtmlSaveOptions();
   opts.ExportWorksheetCSSSeparately = true;
   wb.Save(outputDir + "outputExportWorksheetCSSSeparately.html", opts);
   ```

#### Βασικές επιλογές διαμόρφωσης
- **ΕξαγωγήφύλλουεργασίαςCSSSξεχωριστά**: Ορισμός σε `true` για ξεχωριστά αρχεία CSS.
