# Εξαγωγή ενσωματωμένου αρχείου Mol από βιβλίο εργασίας

## Εισαγωγή
Όσον αφορά τη διαχείριση δεδομένων σε βιβλία εργασίας του Excel, μερικές φορές συναντάτε διάφορα ενσωματωμένα αντικείμενα που δεν έχουν τυπική μορφή. Μια τέτοια μορφή είναι το MOL (Molecular Structure File), το οποίο χρησιμοποιείται συνήθως στη χημεία για την αναπαράσταση μοριακών πληροφοριών. Εάν θέλετε να εξαγάγετε αυτά τα αρχεία MOL από ένα βιβλίο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET, έχετε βρει τον σωστό οδηγό. Σε αυτό το άρθρο, θα σας καθοδηγήσουμε στη διαδικασία βήμα προς βήμα, απομυθοποιώντας κάθε μέρος στην πορεία.
## Προαπαιτούμενα
Πριν ξεκινήσετε να ασχολείστε με τον κώδικα, είναι σημαντικό να βεβαιωθείτε ότι διαθέτετε τις απαραίτητες δεξιότητες και εργαλεία. Δείτε τι θα χρειαστείτε:
1. Βασική Κατανόηση Προγραμματισμού .NET: Θα πρέπει να είστε εξοικειωμένοι με την C# και το .NET framework.
2. Aspose.Cells για .NET: Βεβαιωθείτε ότι έχετε τη βιβλιοθήκη Aspose.Cells. Μπορείτε να [κατεβάστε το εδώ](https://releases.aspose.com/cells/net/).
3. Ένα IDE: Μπορείτε να χρησιμοποιήσετε το Visual Studio ή οποιοδήποτε άλλο IDE συμβατό με .NET.
4. Βιβλίο εργασίας Excel με ενσωματωμένα αρχεία MOL: Για αυτό το σεμινάριο, χρειάζεστε ένα αρχείο Excel που περιέχει αντικείμενα MOL. Μπορείτε να δημιουργήσετε το δικό σας ή να χρησιμοποιήσετε οποιοδήποτε αρχείο δείγματος.
## Εισαγωγή πακέτων
Για να ξεκινήσετε, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας. Αυτό είναι κρίσιμο για την πρόσβαση στις λειτουργίες του Aspose.Cells. Δείτε πώς μπορείτε να το κάνετε:

```csharp
using Aspose.Cells.Drawing;
using Aspose.Cells.WebExtensions;
using System;
using System.IO;
```

Αυτοί οι χώροι ονομάτων θα σας επιτρέψουν να χειρίζεστε βιβλία εργασίας, να έχετε πρόσβαση σε φύλλα εργασίας και να εργάζεστε με αρχεία γενικά.
Τώρα που έχουμε τακτοποιήσει τις προϋποθέσεις μας, ας εμβαθύνουμε στον κώδικα και ας κατανοήσουμε κάθε βήμα που εμπλέκεται στην εξαγωγή ενσωματωμένων αρχείων MOL από ένα βιβλίο εργασίας του Excel. 
## Βήμα 1: Ρύθμιση των καταλόγων σας
Το πρώτο βήμα είναι να ορίσετε πού βρίσκεται το έγγραφο προέλευσης και πού θέλετε να αποθηκεύσετε τα εξαγόμενα αρχεία MOL. Ας ρυθμίσουμε αυτούς τους καταλόγους.
```csharp
string SourceDir = "Your Document Directory"; // Αντικαταστήστε με τη διαδρομή καταλόγου σας
string outputDir = "Your Document Directory"; // Αντικαταστήστε με τη διαδρομή εξόδου σας
```
Εδώ, αντικαθιστάς `"Your Document Directory"` με τη διαδρομή προς τους πραγματικούς καταλόγους σας. Είναι σημαντικό η εφαρμογή σας να έχει πρόσβαση τόσο στον κατάλογο προέλευσης όσο και στον κατάλογο εξόδου.
## Βήμα 2: Φόρτωση του Βιβλίου Εργασίας
Μόλις ρυθμίσετε τους καταλόγους σας, η επόμενη εργασία είναι να φορτώσετε το βιβλίο εργασίας του Excel. Ας το κάνουμε τώρα.

```csharp
Workbook workbook = new Workbook(SourceDir + "EmbeddedMolSample.xlsx");
```

Δημιουργούμε μια παρουσία του `Workbook` κλάση και περνώντας στη διαδρομή προς το αρχείο Excel με το όνομα `EmbeddedMolSample.xlsx`Αυτό το βήμα αρχικοποιεί το βιβλίο εργασίας, επιτρέποντάς σας να αποκτήσετε πρόσβαση στο περιεχόμενό του.
## Βήμα 3: Επανάληψη σε φύλλα εργασίας
Τώρα που το βιβλίο εργασίας σας έχει φορτωθεί, πρέπει να κάνετε επανάληψη σε κάθε φύλλο εργασίας μέσα στο βιβλίο εργασίας. Αυτό σας επιτρέπει να εξετάσετε κάθε φύλλο για ενσωματωμένα αντικείμενα.

```csharp
var index = 1; // Χρησιμοποιείται για την ονομασία εξαγόμενων αρχείων MOL
foreach (Worksheet sheet in workbook.Worksheets)
{
    OleObjectCollection oles = sheet.OleObjects;
    // Περαιτέρω λογική εξαγωγής πηγαίνει εδώ
}
```

Εδώ, χρησιμοποιείτε ένα `foreach` βρόχο για να περιηγηθείτε στα φύλλα εργασίας. Για κάθε φύλλο εργασίας, έχετε πρόσβαση στο `OleObjects` συλλογή, η οποία περιέχει όλα τα ενσωματωμένα αντικείμενα.
## Βήμα 4: Εξαγωγή αρχείων MOL
Τώρα έρχεται το κρίσιμο μέρος—η εξαγωγή των αρχείων MOL από τα αντικείμενα OLE. Αυτό απαιτεί έναν ακόμη βρόχο μέσα στον βρόχο του φύλλου εργασίας.

```csharp
foreach (OleObject ole in oles)
{
    string fileName = outputDir + "OleObject" + index + ".mol ";
    FileStream fs = File.Create(fileName);
    fs.Write(ole.ObjectData, 0, ole.ObjectData.Length);
    fs.Close();
    index++;
}
```

Για κάθε αντικείμενο OLE που έχετε βρει, δημιουργείτε ένα νέο αρχείο στον κατάλογο εξόδου. `ObjectData` ιδιοκτησία του `OleObject` διατηρεί τα δεδομένα του ενσωματωμένου αντικειμένου, τα οποία γράφετε σε ένα νεοδημιουργημένο αρχείο χρησιμοποιώντας ένα `FileStream`Το αρχείο ονομάζεται διαδοχικά (`OleObject1.mol`, `OleObject2.mol`, κ.λπ.) με βάση το `index` μεταβλητός.
## Βήμα 5: Επιβεβαίωση ολοκλήρωσης διαδικασίας
Τέλος, αφού εξαχθούν όλα τα αρχεία MOL, είναι καλή πρακτική να ενημερώσετε τον χρήστη ότι η διαδικασία ολοκληρώθηκε με επιτυχία.

```csharp
Console.WriteLine("ExtractEmbeddedMolFile executed successfully.");
```

Αυτή η γραμμή απλώς εμφανίζει ένα μήνυμα στην κονσόλα που σας ενημερώνει ότι η εξαγωγή ήταν επιτυχής. Είναι μια ωραία πινελιά για τα σχόλια των χρηστών.

Αυτή η διαδικασία ενσωματώνει μερικά βασικά βήματα, εξασφαλίζοντας μια δομημένη προσέγγιση για τον χειρισμό ενσωματωμένων αντικειμένων.
