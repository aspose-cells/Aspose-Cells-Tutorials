## Οδηγός Εφαρμογής

Θα χωρίσουμε το σεμινάριο σε λογικές ενότητες με βάση τα χαρακτηριστικά.

### Δημιουργία βιβλίου εργασίας

Ένα βιβλίο εργασίας χρησιμεύει ως το βασικό αντικείμενο που αντιπροσωπεύει ένα αρχείο Excel. Δείτε πώς μπορείτε να δημιουργήσετε ένα:

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας
```csharp
// Δημιουργήστε μια νέα παρουσία της κλάσης Βιβλίου Εργασίας
Workbook workbook = new Workbook();
```

Αυτό `Workbook` Το αντικείμενο αντιπροσωπεύει το αρχείο Excel σας, επιτρέποντάς σας να προσθέσετε φύλλα εργασίας και δεδομένα.

### Προσθήκη φύλλου εργασίας

Η προσθήκη ενός φύλλου εργασίας είναι απλή μόλις έχετε μια παρουσία βιβλίου εργασίας:

#### Βήμα 2: Προσθήκη νέου φύλλου εργασίας
```csharp
// Προσθήκη νέου φύλλου εργασίας στο βιβλίο εργασίας
int sheetIndex = workbook.Worksheets.Add();
```

Αυτή η μέθοδος επιστρέφει ένα ευρετήριο για την πρόσβαση στο φύλλο εργασίας που μόλις προστέθηκε.

#### Βήμα 3: Πρόσβαση και ονομασία του φύλλου εργασίας σας

Για να εργαστείτε με το νέο σας φύλλο εργασίας, αποκτήστε πρόσβαση σε αυτό από το ευρετήριό του και εκχωρήστε ένα όνομα:

```csharp
// Λάβετε μια αναφορά στο φύλλο εργασίας που μόλις δημιουργήσατε
Worksheet newSheet = workbook.Worksheets[sheetIndex];

// Ορίστε ένα προσαρμοσμένο όνομα για εύκολη αναγνώριση
newSheet.Name = "My Worksheet";
```

### Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε τις τροποποιήσεις σας σε ένα αρχείο Excel:

#### Βήμα 4: Ορισμός διαδρομής εξόδου και αποθήκευση

```csharp
using System.IO;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει πριν από την αποθήκευση
bool isExists = Directory.Exists(outputDir);
if (!isExists)
    Directory.CreateDirectory(outputDir);

// Ορίστε τη διαδρομή για το αρχείο εξόδου
string outputPath = Path.Combine(outputDir, "output.out.xls");

// Αποθήκευση του βιβλίου εργασίας σε μια καθορισμένη θέση
workbook.Save(outputPath);
```

Αυτό το βήμα διασφαλίζει ότι το βιβλίο εργασίας σας αποθηκεύεται στην επιθυμητή μορφή και θέση.
