## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση HTML στο βιβλίο εργασίας

Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια συμβολοσειρά HTML σε ένα βιβλίο εργασίας χρησιμοποιώντας το Aspose.Cells για .NET.

#### Επισκόπηση
Ο κώδικας μετατρέπει έναν πίνακα HTML σε έναν `MemoryStream`, το οποίο στη συνέχεια φορτώνεται ως `Workbook` αντικείμενο σε μορφή Excel.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1:** Ορίστε τον κατάλογο πηγής και το περιεχόμενο HTML.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string sampleHtml = "<html><body><table><tr><td>This is sample text.</td><td>Some text.</td></tr><tr><td>This is another sample text.</td><td>Some text.</td></tr></table></body></html>";
```
**Βήμα 2:** Μετατρέψτε τη συμβολοσειρά HTML σε `MemoryStream`.
```csharp
MemoryStream ms = new MemoryStream(Encoding.UTF8.GetBytes(sampleHtml));
```
**Βήμα 3:** Φόρτωση της ροής μνήμης σε ένα Aspose.Cells `Workbook` αντικείμενο.
```csharp
Workbook wb = new Workbook(ms);
```
**Βήμα 4:** Αποθηκεύστε το βιβλίο εργασίας σε μορφή XLSX.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
wb.Save(Path.Combine(outputDir, "outputWithout_AutoFitColsAndRows.xlsx"));
```

### Λειτουργία 2: Φόρτωση HTML σε βιβλίο εργασίας με αυτόματη προσαρμογή στηλών και γραμμών

Βελτιώστε την προηγούμενη λειτουργικότητα προσαρμόζοντας αυτόματα στήλες και γραμμές για καλύτερη παρουσίαση.

#### Επισκόπηση
Αυτή η επέκταση χρησιμοποιεί `HtmlLoadOptions` για αυτόματη προσαρμογή του πλάτους των στηλών και του ύψους των γραμμών με βάση το μέγεθος του περιεχομένου.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1:** Επαναχρησιμοποιήστε τον κατάλογο πηγής και τους ορισμούς περιεχομένου HTML από τη Λειτουργία 1.
**Βήμα 2:** Μετατρέψτε τη συμβολοσειρά HTML σε `MemoryStream`.
```csharp
MemoryStream ms = new MemoryStream(Encoding.UTF8.GetBytes(sampleHtml));
```
**Βήμα 3:** Δημιουργώ `HtmlLoadOptions` με ενεργοποιημένες τις ρυθμίσεις αυτόματης προσαρμογής.
```csharp
HtmlLoadOptions opts = new HtmlLoadOptions();
opts.AutoFitColsAndRows = true;
```
**Βήμα 4:** Φορτώστε τη ροή μνήμης σε ένα αντικείμενο βιβλίου εργασίας χρησιμοποιώντας τις καθορισμένες επιλογές.
```csharp
Workbook wb = new Workbook(ms, opts);
```
**Βήμα 5:** Αποθηκεύστε το βιβλίο εργασίας με εφαρμοσμένες τις προσαρμογές αυτόματης προσαρμογής.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
wb.Save(Path.Combine(outputDir, "outputWith_AutoFitColsAndRows.xlsx"));
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα:** Λανθασμένες διαδρομές καταλόγου. Βεβαιωθείτε ότι `SourceDir` και `OutputDir` έχουν ρυθμιστεί σωστά.
- **Σφάλματα MemoryStream:** Επιβεβαιώστε ότι η συμβολοσειρά HTML έχει κωδικοποιηθεί σωστά σε UTF-8.
