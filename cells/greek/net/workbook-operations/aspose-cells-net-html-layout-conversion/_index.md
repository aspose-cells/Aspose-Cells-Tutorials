## Οδηγός Εφαρμογής

### Μετατροπή διατάξεων HTML σε βιβλία εργασίας του Excel

#### Βήμα 1: Προετοιμασία του πηγαίου κώδικα HTML σας
Δημιουργήστε μια συμβολοσειρά HTML που αντιπροσωπεύει τη διάταξη των δεδομένων σας. Το παρακάτω παράδειγμα δείχνει τη δομή ενός τμήματος HTML με ένθετες ετικέτες div.

```csharp
var export_html = @"<html>
                    <body>
                        <table>
                            <tr>
                                <td>
                                    <div>This is some Text.</div>
                                    <!-- Nested divs for additional text and data -->
                                    <div><span>This is more Text</span></div>
                                    <div><span>abc@abc.com</span></div>
                                    <div><span>1234567890</span></div>
                                    <div><span>ABC DEF</span></div>
                                    <div>Generated On May 30, 2016 02:33 PM<br />
                                        Time Call Received from Jan 01, 2016 to May 30, 2016
                                    </div>
                                </td>
                                <td>
                                    <!-- Image integration -->
                                    <img src='" + sourceDir + "sampleDivTagsLayout_ASpose_logo_100x100.png' />
                                </td>
                            </tr>
                        </table>
                    </body>
                    </html>";
```

#### Βήμα 2: Φόρτωση HTML στο βιβλίο εργασίας Aspose.Cells
Χρήση `MemoryStream` για να φορτώσετε το περιεχόμενο HTML και να καθορίσετε ότι θα πρέπει να υποστηρίζονται οι ετικέτες div.

```csharp
var ms = new MemoryStream(Encoding.UTF8.GetBytes(export_html));

// Δημιουργία βιβλίου εργασίας χρησιμοποιώντας επιλογές φόρτωσης
Workbook wb = new Workbook(ms, new HtmlLoadOptions(LoadFormat.Html)
{
    SupportDivTag = true // Ενεργοποίηση υποστήριξης για διατάξεις ετικετών div
});
```

#### Βήμα 3: Αυτόματη Προσαρμογή Γραμμών και Στηλών
Η αυτόματη προσαρμογή γραμμών και στηλών εξασφαλίζει βέλτιστη εμφάνιση στο φύλλο Excel σας.

```csharp
Worksheet ws = wb.Worksheets[0];
ws.AutoFitRows();
ws.AutoFitColumns();
```

#### Βήμα 4: Αποθήκευση ως αρχείο XLSX
Αποθηκεύστε το βιβλίο εργασίας σε μορφή αρχείου .xlsx για περαιτέρω χρήση ή διανομή.

```csharp
wb.Save(outputDir + "outputDivTagsLayout.xlsx", Aspose.Cells.SaveFormat.Xlsx);
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**Το περιεχόμενο HTML δεν αποδίδεται σωστά. Βεβαιωθείτε ότι `SupportDivTag` έχει οριστεί σε true.
- **Προβλήματα MemoryStream**Επαληθεύστε ότι ο τύπος κωδικοποίησης ταιριάζει με το σύνολο χαρακτήρων του περιεχομένου HTML.
