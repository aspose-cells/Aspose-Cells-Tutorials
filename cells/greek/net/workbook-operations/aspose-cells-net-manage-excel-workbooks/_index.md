## Οδηγός Εφαρμογής
Αυτός ο οδηγός θα σας καθοδηγήσει στην εφαρμογή βασικών λειτουργιών του Aspose.Cells για .NET.

### Χαρακτηριστικό 1: Δημιουργία βιβλίου εργασίας και διαχείριση φύλλων
**Επισκόπηση**: Αυτή η ενότητα δείχνει πώς να δημιουργήσετε ένα βιβλίο εργασίας, να προσθέσετε φύλλα και να διαχειριστείτε τιμές κελιών.

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας
```csharp
Workbook wb = new Workbook(); // Δημιουργεί μια νέα παρουσία βιβλίου εργασίας
```

#### Βήμα 2: Προσθήκη Φύλλων
```csharp
wb.Worksheets.Add("Sheet2"); // Προσθέτει ένα δεύτερο φύλλο με το όνομα 'Φύλλο2'
```

#### Βήμα 3: Διαχείριση τιμών κελιών
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και ορίστε τιμές κελιών:
```csharp
Worksheet sht1 = wb.Worksheets[0]; // Πρόσβαση στο πρώτο φύλλο εργασίας
sht1.Cells["C1"].PutValue(4); // Τοποθετήστε μια ακέραια τιμή στο κελί C1
sht1.Cells["K30"].PutValue(4); // Προσθήκη τιμής για την αύξηση των κενών γραμμών και στηλών
```

### Χαρακτηριστικό 2: Προσθήκη τύπων και υπολογισμός βιβλίου εργασίας
**Επισκόπηση**Μάθετε πώς να προσθέτετε τύπους σε κελιά και να υπολογίζετε τα αποτελέσματα του βιβλίου εργασίας.

#### Βήμα 1: Προσθήκη τύπων
Αποκτήστε πρόσβαση στο δεύτερο φύλλο εργασίας και αντιστοιχίστε έναν τύπο:
```csharp
Worksheet sht2 = wb.Worksheets[1]; // Πρόσβαση στο δεύτερο φύλλο εργασίας
sht2.Cells["E3"].Formula = "'Sheet1'!C1"; // Προσθέτει έναν τύπο που αναφέρεται στο 'Φύλλο1'!C1
```

#### Βήμα 2: Υπολογισμός βιβλίου εργασίας
Υπολογίστε όλους τους τύπους στο βιβλίο εργασίας:
```csharp
wb.CalculateFormula(); // Υπολογίζει όλους τους τύπους
```

### Λειτουργία 3: Ενημέρωση αναφορών με επιλογές διαγραφής
**Επισκόπηση**: Αυτή η ενότητα δείχνει πώς να ενημερώνετε αναφορές κατά τη διαγραφή κενών γραμμών και στηλών.

#### Βήμα 1: Ορισμός επιλογής αναφοράς ενημέρωσης
Χρήση `DeleteOptions` για να διασφαλίσετε ότι οι αναφορές ενημερώνονται κατά τη διαγραφή:
```csharp
DeleteOptions opts = new DeleteOptions();
opts.UpdateReference = true; // Εξασφαλίζει ενημερώσεις αναφοράς
```

#### Βήμα 2: Διαγραφή κενών γραμμών και στηλών
Εκτελέστε διαγραφές κατά την ενημέρωση αναφορών:
```csharp
sht1.Cells.DeleteBlankColumns(opts); // Διαγράφει κενές στήλες με επιλογές
sht1.Cells.DeleteBlankRows(opts); // Διαγράφει κενές γραμμές με επιλογές
wb.CalculateFormula(); // Υπολογίζει ξανά τους τύπους μετά από τροποποιήσεις
```
