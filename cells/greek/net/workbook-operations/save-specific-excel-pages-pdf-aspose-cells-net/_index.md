### Λειτουργία 1: Άνοιγμα αρχείου Excel
#### Επισκόπηση
Αυτό το βήμα περιλαμβάνει το άνοιγμα ενός υπάρχοντος αρχείου Excel χρησιμοποιώντας το Aspose.Cells, το οποίο χρησιμεύει ως βάση για περαιτέρω λειτουργίες όπως η μετατροπή.
##### Βήμα 1: Φόρτωση του αρχείου Excel

```csharp
using System;
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
// Άνοιγμα αρχείου Excel
Workbook workbook = new Workbook(sourceDir + "/sampleLimitNumberOfPagesGenerated.xlsx");

Console.WriteLine("Excel file opened successfully.");
```

*Εξήγηση*: Το `Workbook` Το αντικείμενο αντιπροσωπεύει το φορτωμένο έγγραφο Excel, απαραίτητο για την πρόσβαση και τον χειρισμό δεδομένων που περιέχει.

### Λειτουργία 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF
#### Επισκόπηση
Για να αποθηκεύσετε συγκεκριμένες σελίδες από ένα βιβλίο εργασίας του Excel ως PDF, ρυθμίστε τις παραμέτρους του `PdfSaveOptions`.
##### Βήμα 1: Ρύθμιση του PdfSaveOptions

```csharp
using Aspose.Cells;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε ένα αντίγραφο του αντικειμένου PdfSaveOption
PdfSaveOptions options = new PdfSaveOptions();

// Καθορίστε ποιες σελίδες θα συμπεριληφθούν στο PDF
options.PageIndex = 3; // Έναρξη από το ευρετήριο σελίδας 3
options.PageCount = 4; // Συμπεριλάβετε συνολικά 4 σελίδες ξεκινώντας από το PageIndex

Console.WriteLine("PDF save options configured.");
```

*Εξήγηση*: `PageIndex` και `PageCount` είναι βασικές παράμετροι που καθορίζουν ποιο τμήμα του εγγράφου Excel θα μετατραπεί σε PDF.

### Δυνατότητα 3: Αποθήκευση αρχείου Excel ως PDF με συγκεκριμένες σελίδες
#### Επισκόπηση
Χρησιμοποιήστε τις διαμορφωμένες επιλογές PdfSaveOptions για να αποθηκεύσετε συγκεκριμένες σελίδες του αρχείου Excel σας ως PDF.
##### Βήμα 1: Αποθήκευση του εγγράφου

```csharp
using Aspose.Cells;
using System.IO;

string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Άνοιγμα του αρχείου Excel για επεξεργασία
Workbook workbook = new Workbook(sourceDir + "/sampleLimitNumberOfPagesGenerated.xlsx");

// Ρυθμίστε τις παραμέτρους αποθήκευσης PDF για να καθορίσετε ποιες σελίδες θα αποθηκευτούν.
PdfSaveOptions options = new PdfSaveOptions();
options.PageIndex = 3; // Έναρξη από το ευρετήριο σελίδας 3
options.PageCount = 4; // Συμπεριλάβετε συνολικά 4 σελίδες ξεκινώντας από το PageIndex

// Αποθηκεύστε τις καθορισμένες σελίδες ως αρχείο PDF στον κατάλογο εξόδου.
workbook.Save(outputDir + "/outputLimitNumberOfPagesGenerated.pdf", options);

Console.WriteLine("Excel document saved as PDF with specific pages.");
```

*Εξήγηση*: Το `Save` η μέθοδος ακολουθεί τη διαδρομή-στόχο και `PdfSaveOptions` για να δημιουργήσετε το επιθυμητό PDF.
