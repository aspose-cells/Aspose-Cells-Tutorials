## Οδηγός Εφαρμογής

### Λειτουργία 1: Άνοιγμα και ανάγνωση ενός αρχείου Excel
**Επισκόπηση:** Αυτή η λειτουργία δείχνει το άνοιγμα ενός αρχείου Excel χρησιμοποιώντας το Aspose.Cells για να αποκτήσετε πρόσβαση στο αντικείμενο του βιβλίου εργασίας του.

#### Οδηγός βήμα προς βήμα
1. **Ρύθμιση ροής αρχείων**: Χρήση `FileStream` για να ανοίξετε το αρχείο Excel που επιθυμείτε.
   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   FileStream fstream = new FileStream(SourceDir + "/Book1.xlsx", FileMode.Open);
   Workbook workbook = new Workbook(fstream);
   ```
2. **Πρόσβαση στο βιβλίο εργασίας**: Το παραπάνω απόσπασμα κώδικα αρχικοποιεί ένα `Workbook` αντικείμενο, το οποίο παρέχει πρόσβαση στα περιεχόμενα του αρχείου Excel.

#### Βασικές Έννοιες
- **FileStream**: Χειρίζεται λειτουργίες εισόδου/εξόδου αρχείων.
- **Βιβλίο ασκήσεων**: Αντιπροσωπεύει ένα ολόκληρο έγγραφο του Excel.

### Δυνατότητα 2: Πρόσβαση σε ένα φύλλο εργασίας στο βιβλίο εργασίας
**Επισκόπηση:** Μάθετε πώς να στοχεύετε και να εργάζεστε με συγκεκριμένα φύλλα εργασίας μέσα στο βιβλίο εργασίας σας.

#### Οδηγός βήμα προς βήμα
1. **Φόρτωση του βιβλίου εργασίας**:
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "/Book1.xlsx");
   ```
2. **Φύλλο εργασίας της Access**: Πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας χρησιμοποιώντας το ευρετήριό του.
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

### Λειτουργία 3: Αντιγραφή μορφοποίησης υπό όρους από ένα κελί σε άλλο
**Επισκόπηση:** Αυτή η λειτουργία καλύπτει την αντιγραφή ρυθμίσεων μορφοποίησης υπό όρους μεταξύ περιοχών κελιών.

#### Οδηγός βήμα προς βήμα
1. **Αρχικοποίηση βιβλίου εργασίας και φύλλων εργασίας**:
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "/Book1.xlsx");
   Worksheet worksheet = workbook.Worksheets[0];
   int TotalRowCount = 0;
   ```
2. **Βρόχος μορφοποίησης αντιγραφής**: Επαναλάβετε όλα τα φύλλα εργασίας για να αντιγράψετε τη μορφοποίησή τους υπό όρους.
   ```csharp
   for (int i = 0; i < workbook.Worksheets.Count; i++)
   {
       Worksheet sourceSheet = workbook.Worksheets[i];
       Range sourceRange = sourceSheet.Cells.MaxDisplayRange;
       Range destRange = worksheet.Cells.CreateRange(sourceRange.FirstRow + TotalRowCount, 
           sourceRange.FirstColumn, sourceRange.RowCount, sourceRange.ColumnCount);
       destRange.Copy(sourceRange);
       TotalRowCount += sourceRange.RowCount;
   }
   ```

#### Βασικές Έννοιες
- **Σειρά**: Αντιπροσωπεύει ένα μπλοκ κελιών στο βιβλίο εργασίας.
- **Αντίγραφο**: Μέθοδος για την αντιγραφή ρυθμίσεων μορφοποίησης.

### Λειτουργία 4: Αποθήκευση του τροποποιημένου αρχείου Excel
**Επισκόπηση:** Μάθετε πώς να αποθηκεύετε τις τροποποιήσεις σας σε ένα αρχείο Excel.

#### Οδηγός βήμα προς βήμα
1. **Εκτέλεση τροποποιήσεων**Χρησιμοποιήστε τα βήματα από τις προηγούμενες λειτουργίες για να τροποποιήσετε το βιβλίο εργασίας σας.
   ```csharp
   int TotalRowCount = 0;
   for (int i = 0; i < workbook.Worksheets.Count; i++)
   {
       Worksheet sourceSheet = workbook.Worksheets[i];
       Range sourceRange = sourceSheet.Cells.MaxDisplayRange;
       Range destRange = workbook.Worksheets[0].Cells.CreateRange(sourceRange.FirstRow + TotalRowCount, 
           sourceRange.FirstColumn, sourceRange.RowCount, sourceRange.ColumnCount);
       destRange.Copy(sourceRange);
       TotalRowCount += sourceRange.RowCount;
   }
   ```
2. **Αποθήκευση βιβλίου εργασίας**:
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "/output.xls");
   ```
