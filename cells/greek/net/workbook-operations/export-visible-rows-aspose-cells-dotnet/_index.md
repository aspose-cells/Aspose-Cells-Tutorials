## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στην εξαγωγή ορατών γραμμών από ένα φύλλο εργασίας του Excel χρησιμοποιώντας **Aspose.Cells για .NET**.

### Βήμα 1: Φόρτωση του βιβλίου εργασίας και του φύλλου εργασίας της Access

Φορτώστε το βιβλίο εργασίας του Excel και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας του:

```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleExportVisibleRowsData.xlsx");
Worksheet worksheet = workbook.Worksheets[0]; // Πρόσβαση στο πρώτο φύλλο εργασίας
```

### Βήμα 2: Εξαγωγή ορατών γραμμών με επιλογές

Εστίαση στην εξαγωγή μόνο ορατών γραμμών και στη συμπερίληψη ονομάτων στηλών:

```csharp
// Ορίστε τις επιλογές εξαγωγής ώστε να περιλαμβάνουν μόνο ορατές γραμμές και κεφαλίδες
ExportTableOptions exportOptions = new ExportTableOptions();
exportOptions.PlotVisibleRows = true; // Συμπερίληψη μόνο ορατών γραμμών
exportOptions.ExportColumnName = true; // Συμπερίληψη κεφαλίδων στηλών στην εξαγωγή

// Εξαγωγή μιας καθορισμένης περιοχής κελιών, ξεκινώντας από το A1
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, 10, 4, exportOptions);
```
