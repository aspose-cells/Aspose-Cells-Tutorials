## Οδηγός Εφαρμογής

### Εξαγωγή Excel σε HTML χωρίς δέσμες ενεργειών πλαισίων

#### Επισκόπηση
Θα σας δείξουμε πώς να εξάγετε ένα αρχείο Excel σε HTML απενεργοποιώντας παράλληλα τα σενάρια πλαισίων. Αυτό εξασφαλίζει ένα ελαφρύ και καθαρό αποτέλεσμα HTML.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Ανοίξτε το Απαιτούμενο Βιβλίο Εργασίας**
Αρχικά, καθορίστε τον κατάλογο προέλευσης και ανοίξτε το βιβλίο εργασίας που θέλετε να μετατρέψετε.
```csharp
// Κατάλογος πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Άνοιγμα του απαιτούμενου βιβλίου εργασίας
Workbook wb = new Workbook(sourceDir + "sampleHtmlExportFrameScripts.xlsx");
```

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML**
Στη συνέχεια, διαμορφώστε το `HtmlSaveOptions` για να απενεργοποιήσετε τα σενάρια πλαισίων.
```csharp
// Απενεργοποίηση εξαγωγής σεναρίων πλαισίων και ιδιοτήτων εγγράφου
HtmlSaveOptions options = new HtmlSaveOptions();
options.ExportFrameScriptsAndProperties = false;
```

**Βήμα 3: Αποθήκευση βιβλίου εργασίας ως HTML**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε μορφή HTML χρησιμοποιώντας τις διαμορφωμένες επιλογές.
```csharp
// Κατάλογος εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση βιβλίου εργασίας ως HTML
wb.Save(outputDir + "outputHtmlExportFrameScripts.html", options);

Console.WriteLine("HtmlExportFrameScripts executed successfully.");
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**: Εάν το αρχείο εξόδου δεν είναι όπως αναμένεται, επαληθεύστε ότι `options.ExportFrameScriptsAndProperties` έχει οριστεί σε false.
- Βεβαιωθείτε ότι οι διαδρομές στον κώδικά σας έχουν οριστεί σωστά για τους καταλόγους προέλευσης και εξόδου.
