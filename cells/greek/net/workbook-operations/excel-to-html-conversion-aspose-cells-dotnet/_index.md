## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση και αποθήκευση βιβλίου εργασίας Excel

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα βιβλίο εργασίας του Excel από έναν καθορισμένο κατάλογο προέλευσης και να το αποθηκεύσετε ως HTML με προσαρμοσμένες επιλογές.

#### Επισκόπηση
Η αποτελεσματική φόρτωση και αποθήκευση βιβλίων εργασίας διασφαλίζει την απρόσκοπτη ανταλλαγή δεδομένων μεταξύ εφαρμογών σε διαφορετικές μορφές.

#### Βήματα:

**Βήμα 1**: Ορίστε τους καταλόγους προέλευσης και εξόδου.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**Βήμα 2**Φόρτωση του βιβλίου εργασίας του Excel χρησιμοποιώντας το Aspose.Cells.
```csharp
// Φόρτωση ενός υπάρχοντος βιβλίου εργασίας από ένα αρχείο
Workbook workbook = new Workbook(SourceDir + "sampleChangeHtmlLinkTarget.xlsx");
```
*Εξήγηση*: Το `Workbook` Η κλάση χρησιμοποιείται για τη φόρτωση και τον χειρισμό αρχείων Excel.

**Βήμα 3**: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML με συγκεκριμένους προορισμούς συνδέσμων.
```csharp
// Αρχικοποιήστε το HtmlSaveOptions και ορίστε το LinkTargetType
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.LinkTargetType = HtmlLinkTargetType.Self; // Οι σύνδεσμοι ανοίγουν στο ίδιο παράθυρο/καρτέλα
```
*Διαμόρφωση κλειδιού*: `HtmlLinkTargetType.Self` διασφαλίζει ότι όλοι οι σύνδεσμοι μέσα στο αρχείο HTML ανοίγουν στην τρέχουσα καρτέλα του προγράμματος περιήγησης.

**Βήμα 4**Αποθήκευση του βιβλίου εργασίας ως αρχείο HTML.
```csharp
// Αποθήκευση του βιβλίου εργασίας με καθορισμένες επιλογές HTML
workbook.Save(OutputDir + "outputChangeHtmlLinkTarget.html", opts);
```
*Σκοπός*: Το `Save` Η μέθοδος γράφει το βιβλίο εργασίας σε μια καθορισμένη μορφή, στην προκειμένη περίπτωση, HTML.

### Λειτουργία 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

Αυτή η δυνατότητα εστιάζει στην προσαρμογή των ρυθμίσεων αποθήκευσης HTML για ένα βιβλίο εργασίας του Excel.

#### Επισκόπηση
Η προσαρμογή των επιλογών αποθήκευσης επιτρέπει την προσαρμογή της εξόδου που ανταποκρίνεται στις συγκεκριμένες απαιτήσεις της εφαρμογής.

#### Βήματα:

**Βήμα 1**: Δημιουργία και διαμόρφωση `HtmlSaveOptions`.
```csharp
// Δημιουργία στιγμιότυπου HtmlSaveOptions
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.LinkTargetType = HtmlLinkTargetType.Self;
```
*Εξήγηση*: Προσαρμογή επιλογών αποθήκευσης HTML όπως `LinkTargetType` παρέχει έλεγχο στον τρόπο με τον οποίο παρουσιάζονται τα δεδομένα σας στο πρόγραμμα περιήγησης.

**Βήμα 2**: Αποθήκευση με τις διαμορφωμένες επιλογές.
```csharp
// Υποθέτοντας ότι το βιβλίο εργασίας έχει ήδη φορτωθεί ως «βιβλίο εργασίας»
workbook.Save(OutputDir + "outputChangeHtmlLinkTarget.html", opts);
```
