## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση και αποθήκευση βιβλίου εργασίας

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον βιβλίο εργασίας του Excel, να κάνετε τροποποιήσεις και να το αποθηκεύσετε ξανά ως νέο αρχείο χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα προς βήμα εφαρμογή

##### Φόρτωση του βιβλίου εργασίας
Για να ξεκινήσετε, δημιουργήστε ένα `Workbook` αντικείμενο καθορίζοντας τη διαδρομή προς το αρχείο προέλευσης του Excel. Αυτό φορτώνει ολόκληρο το βιβλίο εργασίας του Excel στη μνήμη.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του υπάρχοντος βιβλίου εργασίας από τον καθορισμένο κατάλογο
Workbook workbook = new Workbook(SourceDir + "Book1.xls");
```

##### Αποθήκευση του βιβλίου εργασίας
Μετά τη φόρτωση, μπορείτε να αποθηκεύσετε το βιβλίο εργασίας σε διαφορετική θέση ή με τροποποιήσεις. Αυτό το βήμα εγγράφει τις αλλαγές ξανά σε ένα αρχείο Excel.
```csharp
// Αποθήκευση του φορτωμένου βιβλίου εργασίας ως νέου αρχείου στον κατάλογο εξόδου
workbook.Save(outputDir + "output.xls");
```

### Λειτουργία 2: Πρόσβαση και τροποποίηση κελιών φύλλου εργασίας

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να αποκτήσετε πρόσβαση σε συγκεκριμένα φύλλα εργασίας μέσα σε ένα βιβλίο εργασίας και να τροποποιήσετε τα περιεχόμενα των κελιών, συμπεριλαμβανομένης της προσθήκης τύπων.

#### Βήμα προς βήμα εφαρμογή

##### Πρόσβαση σε φύλλο εργασίας
Μπορείτε να αποκτήσετε πρόσβαση σε μεμονωμένα φύλλα εργασίας με βάση τον δείκτη τους. Εδώ, εστιάζουμε στο πρώτο φύλλο εργασίας:
```csharp
// Φορτώστε ξανά το αρχείο Excel, εάν δεν έχει ήδη φορτωθεί
Workbook workbook = new Workbook(SourceDir + "Book1.xls");

// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

##### Τροποποίηση περιεχομένων κελιών με τύπους
Το Aspose.Cells υποστηρίζει τη σημειογραφία R1C1 για τύπους, η οποία σας επιτρέπει να χρησιμοποιείτε σχετικές αναφορές. Δείτε πώς μπορείτε να ορίσετε έναν τύπο στο κελί A11:
```csharp
// Ορισμός τύπου R1C1 στο κελί A11
worksheet.Cells["A11"].R1C1Formula = ";=SUM(R[-10]C[0]:R[-7]C[0])";
```

##### Αποθήκευση του βιβλίου εργασίας με αλλαγές
Αφού κάνετε τις αλλαγές, αποθηκεύστε το βιβλίο εργασίας όπως πριν:
```csharp
// Αποθήκευση του τροποποιημένου βιβλίου εργασίας σε νέο αρχείο
tworkbook.Save(outputDir + "output_with_formula.xls");
```
