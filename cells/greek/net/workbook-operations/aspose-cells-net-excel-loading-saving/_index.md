## Οδηγός Εφαρμογής

### Φόρτωση αρχείου Excel
**Επισκόπηση**: Μάθετε πώς να φορτώσετε ένα υπάρχον αρχείο Excel σε ένα `Workbook` αντικείμενο. Αυτό είναι απαραίτητο για την επεξεργασία και τον χειρισμό δεδομένων που είναι αποθηκευμένα σε αρχεία Excel.

#### Βήμα προς βήμα εφαρμογή
1. **Ρύθμιση του καταλόγου πηγών**
   Ορίστε πού βρίσκεται το αρχείο προέλευσης Excel:
   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   ```
2. **Φόρτωση του αρχείου Excel**
   Αρχικοποίηση ενός `Workbook` αντικείμενο φορτώνοντας ένα υπάρχον αρχείο Excel:
   ```csharp
   Workbook workbook = new Workbook(SourceDir + "sampleExpandTextFromRightToLeft.xlsx");
   ```
   - **Εξήγηση**: Το `Workbook` Ο κατασκευαστής παίρνει μια διαδρομή αρχείου και τη φορτώνει στη μνήμη, επιτρέποντάς σας να αλληλεπιδράσετε με τα περιεχόμενά του μέσω προγραμματισμού.

### Αποθήκευση βιβλίου εργασίας ως HTML
**Επισκόπηση**Αυτή η ενότητα παρουσιάζει την αποθήκευση ενός βιβλίου εργασίας του Excel σε μορφή HTML. Είναι ιδιαίτερα χρήσιμη για εφαρμογές web που απαιτούν παρουσίαση δεδομένων εκτός του λογισμικού του Excel.

#### Βήμα προς βήμα εφαρμογή
1. **Ρύθμιση του καταλόγου εξόδου**
   Ορίστε πού θα αποθηκευτεί το αρχείο HTML εξόδου:
   ```csharp
   string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
   ```
2. **Αποθήκευση του βιβλίου εργασίας σε μορφή HTML**
   Αποθηκεύστε το βιβλίο εργασίας με τις πληροφορίες έκδοσης προσαρτημένες στο όνομα αρχείου:
   ```csharp
   workbook.Save(OutputDir + "outputExpandTextFromRightToLeft_" + CellsHelper.GetVersion() + ".html", SaveFormat.Html);
   ```
   - **Εξήγηση**: Το `Save` Η μέθοδος γράφει το περιεχόμενο του βιβλίου εργασίας σε ένα αρχείο HTML. `CellsHelper.GetVersion()` Παρέχει λεπτομέρειες έκδοσης για τη διάκριση διαφορετικών εξόδων.
