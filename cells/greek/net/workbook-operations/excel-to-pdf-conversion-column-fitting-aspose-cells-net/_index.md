## Οδηγός Εφαρμογής

### Λειτουργία 1: Φόρτωση βιβλίου εργασίας από τον κατάλογο προέλευσης
#### Επισκόπηση
Η φόρτωση ενός βιβλίου εργασίας του Excel στη μνήμη είναι το πρώτο βήμα για την επεξεργασία και τη μετατροπή του.
##### Βήμα 1: Ρύθμιση καταλόγων και διαδρομής αρχείου
```csharp
using System;
using Aspose.Cells;
// Ορίστε διαδρομές για τους καταλόγους προέλευσης και εξόδου
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string excelFileName = "sampleFitAllWorksheetColumns.xlsx";
// Φόρτωση του βιβλίου εργασίας από την καθορισμένη διαδρομή αρχείου
Workbook book = new Workbook(SourceDir + "/" + excelFileName);
```
**Εξήγηση**: Αντικατάσταση `YOUR_SOURCE_DIRECTORY` με την πραγματική διαδρομή καταλόγου σας. Αυτό το απόσπασμα κώδικα αρχικοποιεί ένα `Workbook` αντικείμενο φορτώνοντας ένα αρχείο Excel, καθιστώντας το έτοιμο για περαιτέρω επεξεργασία.

### Λειτουργία 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF
#### Επισκόπηση
Προσαρμόστε τον τρόπο αποθήκευσης του βιβλίου εργασίας ως PDF για να διασφαλίσετε ότι όλες οι στήλες χωράνε σε μία σελίδα ανά φύλλο.
##### Βήμα 2: Ρύθμιση του PdfSaveOptions
```csharp
// Αρχικοποίηση PdfSaveOptions
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.AllColumnsInOnePagePerSheet = true;
```
**Εξήγηση**: Με ρύθμιση `AllColumnsInOnePagePerSheet` Σε τιμή true, δίνετε εντολή στο Aspose.Cells να προσαρμόσει τα πλάτη των στηλών έτσι ώστε όλες οι στήλες να χωράνε σε μία σελίδα ανά φύλλο στο PDF.

### Δυνατότητα 3: Αποθήκευση βιβλίου εργασίας ως PDF με διαμορφωμένες επιλογές
#### Επισκόπηση
Αποθηκεύστε το φορτωμένο βιβλίο εργασίας σε ένα αρχείο PDF χρησιμοποιώντας τις διαμορφωμένες επιλογές.
##### Βήμα 3: Καθορισμός εξόδου και αποθήκευση
```csharp
using System.IO;
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
string pdfFileName = "outputFitAllWorksheetColumns.pdf";
// Αποθήκευση του βιβλίου εργασίας ως PDF με καθορισμένες επιλογές αποθήκευσης
book.Save(OutputDir + "/" + pdfFileName, saveOptions);
```
**Εξήγηση**: Αντικατάσταση `YOUR_OUTPUT_DIRECTORY` με την επιθυμητή διαδρομή εξόδου. Αυτός ο κώδικας αποθηκεύει το βιβλίο εργασίας σε μορφή PDF, εφαρμόζοντας διαμορφώσεις για να χωρέσουν όλες οι στήλες σε μία σελίδα.
