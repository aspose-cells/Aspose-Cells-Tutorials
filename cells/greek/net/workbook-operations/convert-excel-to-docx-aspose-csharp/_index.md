## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα δούμε πώς να μετατρέψετε ένα αρχείο Excel σε μορφή DOCX.

### Επισκόπηση
Η μετατροπή ενός αρχείου Excel σε έγγραφο του Word βελτιστοποιεί την παρουσίαση δεδομένων και βελτιώνει τη συνεργασία, διευκολύνοντας την επεξεργασία και την κοινή χρήση πληροφοριών σε όλες τις πλατφόρμες.

### Διαδικασία μετατροπής βήμα προς βήμα
**1. Ανοίξτε το Αρχείο Προτύπου**
Ξεκινήστε φορτώνοντας το αρχείο προέλευσης Excel:
```csharp
// Διαδρομή καταλόγου πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Αρχικοποίηση ενός αντικειμένου βιβλίου εργασίας με τη διαδρομή του αρχείου Excel
Workbook workbook = new Workbook(sourceDir + "Book1.xlsx");
```
*Εξήγηση*: Το `Workbook` Η κλάση αντιπροσωπεύει ένα αρχείο Excel. Μεταβιβάζοντας τη διαδρομή του αρχείου στον κατασκευαστή του, φορτώνετε το υπολογιστικό φύλλο στη μνήμη.

**2. Αποθήκευση ως DOCX**
Μετατρέψτε και αποθηκεύστε το φορτωμένο αρχείο Excel ως έγγραφο του Word:
```csharp
// Διαδρομή καταλόγου εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση του βιβλίου εργασίας σε μορφή DOCX
workbook.Save(outputDir + "Book1.docx", SaveFormat.Docx);
```
*Εξήγηση*: Το `Save` η μέθοδος εξάγει το τρέχον Βιβλίο Εργασίας σε διάφορες μορφές. Εδώ, `SaveFormat.Docx` καθορίζει ότι θέλουμε ένα έγγραφο του Word.
