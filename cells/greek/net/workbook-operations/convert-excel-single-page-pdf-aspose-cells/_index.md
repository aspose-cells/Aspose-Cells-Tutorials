## Οδηγός Εφαρμογής

Θα χωρίσουμε αυτήν τη διαδικασία σε διαχειρίσιμα μέρη για λόγους σαφήνειας.

### Άνοιγμα αρχείου Excel

Αυτή η λειτουργία σάς επιτρέπει να ανοίξετε ένα υπάρχον βιβλίο εργασίας του Excel χρησιμοποιώντας το `Workbook` τάξη που παρέχεται από το Aspose.Cells. Δείτε πώς λειτουργεί:

**Βήμα 1**: Ορίστε τον κατάλογο προέλευσης και το όνομα αρχείου.

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string fileName = "sampleRenderOnePdfPagePerExcelWorksheet.xlsx";
```

**Βήμα 2**: Φόρτωση του βιβλίου εργασίας του Excel.

```csharp
Workbook workbook = new Workbook(SourceDir + fileName);
```

### Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF

Για να διασφαλίσετε ότι κάθε φύλλο εργασίας αποδίδεται σε μία μόνο σελίδα στο PDF σας, διαμορφώστε το `PdfSaveOptions`.

**Βήμα 1**: Δημιουργήστε μια παρουσία του `PdfSaveOptions` και ορίστε το `OnePagePerSheet` ιδιοκτησία.

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.OnePagePerSheet = true;
```

### Αποθήκευση Excel ως PDF με συγκεκριμένες επιλογές

Αφού φορτώσετε το βιβλίο εργασίας σας και έχετε ρυθμίσει τις επιλογές σας, αποθηκεύστε το ως αρχείο PDF χρησιμοποιώντας αυτές τις ρυθμίσεις.

**Βήμα 1**: Ορίστε τον κατάλογο εξόδου και το όνομα αρχείου για το PDF που προκύπτει.

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
string pdfFileName = "outputRenderOnePdfPagePerExcelWorksheet.pdf";
```

**Βήμα 2**Αποθηκεύστε το βιβλίο εργασίας με τις καθορισμένες επιλογές αποθήκευσης.

```csharp
workbook.Save(outputDir + pdfFileName, pdfSaveOptions);
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Σφάλμα "Δεν βρέθηκε αρχείο"**: Βεβαιωθείτε ότι το `SourceDir` και η διαδρομή αρχείου έχουν οριστεί σωστά.
- **Προβλήματα εξόδου PDF**: Επαληθεύστε ότι `OnePagePerSheet` έχει ρυθμιστεί σωστά στο `PdfSaveOptions`.
