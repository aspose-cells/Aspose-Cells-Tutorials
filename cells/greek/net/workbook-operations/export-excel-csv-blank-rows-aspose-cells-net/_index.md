Με το Aspose.Cells ρυθμισμένο, ας εξαγάγουμε ένα αρχείο Excel ως CSV διατηρώντας παράλληλα τα διαχωριστικά για τις κενές γραμμές.

### Άνοιγμα και διαμόρφωση του βιβλίου εργασίας (H2)

#### Βήμα 1: Φόρτωση του αρχείου Excel
Καθορίστε τη διαδρομή προς τον κατάλογο προέλευσης όπου βρίσκεται το αρχείο Excel. Δημιουργήστε ένα `Workbook` αντίρρηση για να το ανοίξετε:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string filePath = System.IO.Path.Combine(SourceDir, "Book1.xlsx");
Workbook wb = new Workbook(filePath);
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης
Στήνω `TxtSaveOptions` για να προσαρμόσετε την αποθήκευση CSV και να διασφαλίσετε ότι διατηρούνται τα διαχωριστικά για τις κενές γραμμές:
```csharp
TxtSaveOptions options = new TxtSaveOptions();
options.KeepSeparatorsForBlankRow = true; // Διατηρεί το διαχωριστικό στις κενές γραμμές.
```

#### Βήμα 3: Αποθήκευση του βιβλίου εργασίας ως CSV
Αποθηκεύστε το βιβλίο εργασίας σας με τις διαμορφωμένες επιλογές σε έναν καθορισμένο κατάλογο εξόδου:
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
wb.Save(System.IO.Path.Combine(outputDir, "output.csv"), options);
```
