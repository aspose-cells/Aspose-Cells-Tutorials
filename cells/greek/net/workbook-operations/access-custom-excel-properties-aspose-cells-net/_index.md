## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στην πρόσβαση σε προσαρμοσμένες ιδιότητες εγγράφων στο Excel χρησιμοποιώντας το Aspose.Cells.

### Πρόσβαση σε προσαρμοσμένες ιδιότητες εγγράφου κατά όνομα και ευρετήριο

#### Επισκόπηση

Η πρόσβαση σε προσαρμοσμένες ιδιότητες εγγράφων σάς επιτρέπει να εξαγάγετε μεταδεδομένα που είναι κρίσιμα για τη διαχείριση εγγράφων. Θα δείξουμε πώς να ανακτήσετε αυτές τις ιδιότητες τόσο με βάση το όνομά τους όσο και με το ευρετήριό τους.

#### Βήμα προς βήμα εφαρμογή

**1. Ανάκτηση ιδιοτήτων προσαρμοσμένου εγγράφου**

Αρχικά, φορτώστε το βιβλίο εργασίας και αποκτήστε πρόσβαση στις προσαρμοσμένες ιδιότητές του:

```csharp
Workbook workbook = new Workbook(SourceDir + "sample-document-properties.xlsx");
Aspose.Cells.Properties.DocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```

**2. Πρόσβαση με όνομα**

Για να ανακτήσετε ένα ακίνητο χρησιμοποιώντας το όνομά του:

```csharp
Aspose.Cells.Properties.DocumentProperty customProperty1 = customProperties["ContentTypeId"];
Console.WriteLine($"Name: {customProperty1.Name}, Value: {customProperty1.Value}");
```

**3. Πρόσβαση μέσω Ευρετηρίου**

Εναλλακτικά, ανακτήστε την ίδια ιδιότητα με βάση το ευρετήριο:

```csharp
Aspose.Cells.Properties.DocumentProperty customProperty2 = customProperties[0];
Console.WriteLine($"Name: {customProperty2.Name}, Value: {customProperty2.Value}");
```
