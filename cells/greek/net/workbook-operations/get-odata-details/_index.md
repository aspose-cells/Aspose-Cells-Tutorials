## Εισαγωγή πακέτων
 Για να εργαστείτε με το Aspose.Cells στο έργο σας C#, πρέπει πρώτα να εισαγάγουμε τα σχετικά πακέτα. Βεβαιωθείτε ότι έχετε συμπεριλάβει τα ακόλουθα χρησιμοποιώντας οδηγίες στο επάνω μέρος της σελίδας σας`.cs` αρχείο:

## Βήμα 1: Ορίστε τον Κατάλογο προέλευσης
Αρχικά, πρέπει να πούμε στο πρόγραμμά μας πού θα βρει το αρχείο Excel που θέλουμε να επεξεργαστούμε. Αυτό περιλαμβάνει τη ρύθμιση μιας μεταβλητής για να αντιπροσωπεύει τον κατάλογο προέλευσης. Δείτε πώς μπορείτε να το κάνετε:
```csharp
string SourceDir = "Your Document Directory";
```
 Σε αυτή τη γραμμή, αντικαταστήστε`"Your Document Directory"` με την πραγματική διαδρομή όπου σας`ODataSample.xlsx` αρχείο βρίσκεται. Αυτή η διαδρομή είναι ζωτικής σημασίας επειδή παρέχει στο πρόγραμμα τα μέσα για να εντοπίσει και να ανοίξει το αρχείο Excel.
## Βήμα 2: Δημιουργήστε μια παρουσία βιβλίου εργασίας
Τώρα ήρθε η ώρα να φορτώσετε το βιβλίο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells. Μπορείτε να το κάνετε αυτό με μία μόνο γραμμή κώδικα!
```csharp
Workbook workbook = new Workbook(SourceDir + "ODataSample.xlsx");
```
 Εδώ, δημιουργούμε ένα νέο παράδειγμα του`Workbook` τάξη δείχνοντας το αρχείο μας Excel. Ο κατασκευαστής παίρνει τη διαδρομή του αρχείου ως είσοδο και φορτώνει το βιβλίο εργασίας στη μνήμη, καθιστώντας το έτοιμο για αλληλεπίδραση μαζί του.
## Βήμα 3: Πρόσβαση στους τύπους Power Query
Τώρα που έχουμε φορτώσει το βιβλίο εργασίας μας, ας βουτήξουμε στο περιεχόμενό του. Συγκεκριμένα, θέλουμε να αποκτήσουμε πρόσβαση στη συλλογή τύπων Power Query:
```csharp
PowerQueryFormulaCollction PQFcoll = workbook.DataMashup.PowerQueryFormulas;
```
 Με αυτή τη γραμμή, ανακτούμε το`PowerQueryFormulaCollection`από τη δυνατότητα Data Mashup του βιβλίου εργασίας. Αυτή η συλλογή περιέχει όλους τους τύπους Power Query που υπάρχουν στο αρχείο Excel. Εάν έχετε εργαστεί με ερωτήματα στο Excel, γνωρίζετε πόσο πολύτιμες μπορεί να είναι αυτές οι πληροφορίες!
## Βήμα 4: Κάντε βρόχο μέσω των τύπων Power Query
Ας ρίξουμε μια πιο προσεκτική ματιά σε κάθε τύπο Power Query που μόλις χρησιμοποιήσαμε. Θα περιηγηθούμε στη συλλογή και θα εκτυπώσουμε το όνομα κάθε ερωτήματος και τα στοιχεία του:
```csharp
foreach (PowerQueryFormula PQF in PQFcoll)
{
    Console.WriteLine("Connection Name: " + PQF.Name);
    PowerQueryFormulaItemCollection PQFIcoll = PQF.PowerQueryFormulaItems;
    foreach (PowerQueryFormulaItem PQFI in PQFIcoll)
    {
        Console.WriteLine("Name: " + PQFI.Name);
        Console.WriteLine("Value: " + PQFI.Value);
    }
}
```
1.  Εξωτερικός βρόχος: Εδώ, περνάμε μέσα από το καθένα`PowerQueryFormula` σε`PQFcoll`. Για κάθε τύπο, εκτυπώνουμε το όνομα της σύνδεσης.
  
2.  Εσωτερικός βρόχος: Μέσα στον εξωτερικό βρόχο, δημιουργούμε έναν άλλο βρόχο για ανάκτηση`PowerQueryFormulaItems` από κάθε τύπο. Για κάθε είδος, εκτυπώνουμε το όνομα και την αξία του.
Αυτό σας δίνει μια βαθιά εικόνα της δομής των τύπων Power Query σας. Είναι σαν να ξεφλουδίζουμε τα πίσω στρώματα σε ένα κρεμμύδι. όσο περισσότερο σκάβεις, τόσο περισσότερα ανακαλύπτεις!
