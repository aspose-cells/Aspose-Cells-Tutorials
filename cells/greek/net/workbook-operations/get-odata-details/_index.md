## Εισαγωγή πακέτων
Για να εργαστείτε με το Aspose.Cells στο έργο σας σε C#, πρέπει πρώτα να εισαγάγουμε τα σχετικά πακέτα. Βεβαιωθείτε ότι έχετε συμπεριλάβει τις ακόλουθες οδηγίες χρησιμοποιώντας το πάνω μέρος του `.cs` αρχείο:

Αυτά τα πακέτα σάς παρέχουν πρόσβαση σε λειτουργίες χειρισμού του Excel και δυνατότητες ανάκτησης δεδομένων που παρέχονται από το Aspose.Cells. Τώρα, ας εμβαθύνουμε στη διαδικασία βήμα προς βήμα για την ανάκτηση λεπτομερειών OData από ένα βιβλίο εργασίας!
## Βήμα 1: Ορίστε τον κατάλογο πηγών σας
Αρχικά, πρέπει να πούμε στο πρόγραμμά μας πού θα βρει το αρχείο Excel που θέλουμε να επεξεργαστούμε. Αυτό περιλαμβάνει τον ορισμό μιας μεταβλητής που θα αντιπροσωπεύει τον κατάλογο προέλευσης. Δείτε πώς μπορείτε να το κάνετε:
```csharp
string SourceDir = "Your Document Directory";
```
Σε αυτήν τη γραμμή, αντικαταστήστε `"Your Document Directory"` με την πραγματική διαδρομή όπου εσύ `ODataSample.xlsx` βρίσκεται το αρχείο. Αυτή η διαδρομή είναι κρίσιμη επειδή παρέχει στο πρόγραμμα τα μέσα για να εντοπίσει και να ανοίξει το αρχείο Excel.
## Βήμα 2: Δημιουργία μιας παρουσίας βιβλίου εργασίας
Τώρα είναι η ώρα να φορτώσετε το βιβλίο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells. Μπορείτε να το κάνετε αυτό με μία μόνο γραμμή κώδικα!
```csharp
Workbook workbook = new Workbook(SourceDir + "ODataSample.xlsx");
```
Εδώ, δημιουργούμε μια νέα παρουσία του `Workbook` κλάσης δείχνοντας στο αρχείο Excel μας. Ο κατασκευαστής λαμβάνει τη διαδρομή του αρχείου ως είσοδο και φορτώνει το βιβλίο εργασίας στη μνήμη, καθιστώντας το έτοιμο για να αλληλεπιδράσουμε μαζί του.
## Βήμα 3: Πρόσβαση σε τύπους Power Query
Τώρα που έχουμε φορτώσει το βιβλίο εργασίας μας, ας εμβαθύνουμε στα περιεχόμενά του. Συγκεκριμένα, θέλουμε να έχουμε πρόσβαση στη συλλογή τύπων του Power Query:
```csharp
PowerQueryFormulaCollction PQFcoll = workbook.DataMashup.PowerQueryFormulas;
```
Με αυτήν τη γραμμή, ανακτούμε το `PowerQueryFormulaCollection` από τη λειτουργία Data Mashup του βιβλίου εργασίας. Αυτή η συλλογή περιέχει όλους τους τύπους Power Query που υπάρχουν στο αρχείο Excel. Εάν έχετε εργαστεί με ερωτήματα στο Excel, γνωρίζετε πόσο πολύτιμες μπορούν να είναι αυτές οι πληροφορίες!
## Βήμα 4: Επανάληψη των τύπων του Power Query
Ας ρίξουμε μια πιο προσεκτική ματιά σε κάθε τύπο Power Query στον οποίο μόλις αποκτήσαμε πρόσβαση. Θα επανέλθουμε στη συλλογή και θα εκτυπώσουμε το όνομα κάθε ερωτήματος και τα στοιχεία του:
```csharp
foreach (PowerQueryFormula PQF in PQFcoll)
{
    Console.WriteLine("Connection Name: " + PQF.Name);
    PowerQueryFormulaItemCollection PQFIcoll = PQF.PowerQueryFormulaItems;
    foreach (PowerQueryFormulaItem PQFI in PQFIcoll)
    {
        Console.WriteLine("Name: " + PQFI.Name);
        Console.WriteLine("Value: " + PQFI.Value);
    }
}
```
1. Εξωτερικός βρόχος: Εδώ, κάνουμε βρόχο μέσα από κάθε `PowerQueryFormula` σε `PQFcoll`Για κάθε τύπο, εκτυπώνουμε το όνομα της σύνδεσης.
  
2. Εσωτερικός βρόχος: Μέσα στον εξωτερικό βρόχο, δημιουργούμε έναν άλλο βρόχο για να ανακτήσουμε `PowerQueryFormulaItems` από κάθε τύπο. Για κάθε στοιχείο, εκτυπώνουμε το όνομα και την τιμή του.
Αυτό σας δίνει μια βαθιά εικόνα για τη δομή των τύπων του Power Query σας. Είναι σαν να ξεφλουδίζετε στρώσεις από ένα κρεμμύδι. Όσο περισσότερο ψάχνετε, τόσο περισσότερα ανακαλύπτετε!
