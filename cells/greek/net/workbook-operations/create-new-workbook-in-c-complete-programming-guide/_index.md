# Δημιουργία νέου βιβλίου εργασίας σε C# – Πλήρης Οδηγός Προγραμματισμού

Ποτέ χρειάστηκε να **δημιουργήσετε νέο βιβλίο εργασίας** σε C# αλλά δεν ήξερες από πού να ξεκινήσεις; Δεν είσαι μόνος/η. Είτε αυτοματοποιείς μια αλυσίδα αναφορών είτε απλώς πειραματίζεσαι με τύπους Excel στον κώδικα, η δυνατότητα να δημιουργήσεις ένα βιβλίο εργασίας, να προσθέσεις τύπους όπως `EXPAND` ή `COT`, και στη συνέχεια **να αποθηκεύσετε το βιβλίο εργασίας σε αρχείο** είναι βασική δεξιότητα για κάθε προγραμματιστή .NET.

Σε αυτό το tutorial θα περάσουμε από ένα πραγματικό παράδειγμα που κάνει ακριβώς αυτό: θα δημιουργήσουμε ένα νέο βιβλίο εργασίας, θα χρησιμοποιήσουμε τη λειτουργία `EXPAND` για να μετατρέψουμε έναν στατικό πίνακα σε δυναμική στήλη, θα υπολογίσουμε την συνεφαπτομένη με τη λειτουργία `COT`, και τέλος **θα αποθηκεύσουμε το βιβλίο εργασίας σε αρχείο** ως `.xlsx`. Στο τέλος θα έχετε ένα έτοιμο προς εκτέλεση snippet, θα καταλάβετε *γιατί* κάθε κλήση είναι σημαντική, και θα δείτε μερικές χρήσιμες παραλλαγές για ειδικές περιπτώσεις.

> **Pro tip:** Όλος ο κώδικας παρακάτω λειτουργεί με την πιο πρόσφατη έκδοση του Aspose.Cells for .NET (από Μάρτιο 2026). Αν χρησιμοποιείτε παλαιότερη έκδοση, η επιφάνεια του API είναι κατά κύριο λόγο η ίδια, αλλά ελέγξτε ξανά τις εισαγωγές namespaces.

## Τι θα χρειαστείτε

- .NET 6.0 ή νεότερο (το δείγμα στοχεύει στο .NET 6, αλλά λειτουργεί και με .NET 5)  
- Aspose.Cells for .NET εγκατεστημένο μέσω NuGet (`Install-Package Aspose.Cells`)  
- Μια βασική γνώση C# (το έχετε)  

Αυτό είναι όλο—χωρίς επιπλέον DLLs, χωρίς COM interop, και σίγουρα χωρίς εγκατεστημένο Excel στο μηχάνημα. Έτοιμοι; Ας βουτήξουμε.

## Βήμα 1: Δημιουργία νέου βιβλίου εργασίας

Το πρώτο που πρέπει να κάνετε είναι να δημιουργήσετε ένα αντικείμενο της κλάσης `Workbook`. Σκεφτείτε το σαν το άνοιγμα ενός κεντρικού αρχείου Excel στη μνήμη. Αυτό το αντικείμενο περιέχει μια συλλογή φύλλων εργασίας, στυλ, και όλα τα άλλα που θα χρειαστείτε αργότερα.

```csharp
using Aspose.Cells;

class ExcelDemo
{
    static void Main()
    {
        // Step 1: Create a new workbook and grab the first worksheet
        Workbook workbook = new Workbook();               // <-- creates an empty .xlsx structure
        Worksheet ws = workbook.Worksheets[0];            // default sheet is named "Sheet1"
```

Γιατί να πάρουμε αμέσως το πρώτο φύλλο εργασίας; Τα περισσότερα παραδείγματα γρήγορης εκκίνησης δουλεύουν με ένα μόνο φύλλο, και ο προσπελάστης `Worksheets[0]` είναι ο γρηγορότερος τρόπος για να αποκτήσετε μια αναφορά χωρίς βρόχο. Αν χρειαστείτε πολλαπλά φύλλα αργότερα, μπορείτε να τα προσθέσετε με `workbook.Worksheets.Add()`.

## Βήμα 2: Πώς να χρησιμοποιήσετε το EXPAND για δημιουργία δυναμικών περιοχών

`EXPAND` είναι μια νεότερη λειτουργία του Excel που παίρνει έναν πίνακα και τον επεκτείνει σε καθορισμένο μέγεθος. Στον κώδικά μας θα επεκτείνουμε τον κυριολεκτικό πίνακα `{1,2,3}` σε **στήλη 5‑γραμμών** που ξεκινά από το κελί `A1`. Η σύνταξη μέσα στο string είναι ακριβώς αυτή που θα πληκτρολογούσατε στο Excel, ώστε να μπορείτε να την αντιγράψετε‑επικολλήσετε απευθείας σε ένα κελί αργότερα αν το θέλετε.

```csharp
        // Step 2: Apply EXPAND to turn {1,2,3} into a 5‑row vertical range
        ws.Cells["A1"].Formula = "=EXPAND({1,2,3},5,1)"; // rows=5, cols=1
```

### Τι συμβαίνει στο παρασκήνιο;

- `{1,2,3}` είναι ένας οριζόντιος κυριολεκτικός πίνακας.  
- Το δεύτερο όρισμα (`5`) λέει στο Excel να επεκτείνει τον πίνακα σε **5 γραμμές**.  
- Το τρίτο όρισμα (`1`) εξαναγκάζει την έξοδο να είναι **μία στήλη**.  

Αν παραλείψετε το τρίτο όρισμα, το Excel θα προσπαθήσει να διατηρήσει το αρχικό σχήμα, κάτι που μπορεί να σας δώσει ένα μπλοκ 5×3 αντί για μία στήλη. Αυτό είναι ένα κοινό λάθος όταν πειραματίζεστε για πρώτη φορά με το `EXPAND`.

#### Παραλλαγές που μπορεί να χρειαστείτε

| Επιθυμητό σχήμα | Παράδειγμα τύπου |
|---------------|-----------------|
| Μπλοκ 3‑γραμμών, 2‑στηλών | `=EXPAND({1,2,3},3,2)` |
| Μόνο γέμισμα προς τα κάτω (ίδια στήλη) | `=EXPAND({10,20},10,1)` |
| Επέκταση σε μεγαλύτερο αριθμό στηλών | `=EXPAND({5},5,4)` |

Αλλάξτε ελεύθερα τα κυριολεκτικά ή τις διαστάσεις ώστε να ταιριάζουν με τη λογική δημιουργίας δεδομένων σας.

## Βήμα 3: Πώς να υπολογίσετε τη συνεφαπτομένη με τη λειτουργία COT

Η λειτουργία `COT` επιστρέφει τη συνεφαπτομένη μιας γωνίας εκφρασμένης σε ακτίνια. Στο παράδειγμά μας υπολογίζουμε τη συνεφαπτομένη των 45° (π/4 ακτίνια). Το αποτέλεσμα, `1`, τοποθετείται στο κελί `B1`.

```csharp
        // Step 3: Use COT to calculate cotangent of 45 degrees (π/4 radians)
        ws.Cells["B1"].Formula = "=COT(PI()/4)"; // PI() returns π, divided by 4 = 45°
```

### Γιατί να χρησιμοποιήσετε το COT αντί για χειροκίνητο υπολογισμό;

Το Excel ήδη γνωρίζει πώς να χειριστεί τη τριγωνομετρική μετατροπή, έτσι αποφεύγετε σφάλματα στρογγυλοποίησης που μπορεί να εμφανιστούν αν προσπαθήσετε `1 / TAN(angle)`. Επιπλέον, ο τύπος παραμένει αναγνώσιμος για όποιον αργότερα εξετάσει το φύλλο.

#### Ειδική περίπτωση: γωνίες πέρα από 0‑360°

Αν δώσετε γωνία μεγαλύτερη από `2*PI()` (ή αρνητική), το Excel θα την τυλίξει αυτόματα, αλλά το αποτέλεσμα μπορεί να είναι απρόσμενο. Για να είστε σίγουροι, ίσως θέλετε πρώτα να κανονικοποιήσετε τη γωνία:

```csharp
        // Normalize angle to 0‑2π range before applying COT
        ws.Cells["C1"].Formula = "=COT(MOD(PI()*3, 2*PI()))";
```

Αυτό το snippet δείχνει πώς να συνδυάσετε το `MOD` με το `COT` για αξιόπιστους υπολογισμούς.

## Βήμα 4: Πώς να αποθηκεύσετε το βιβλίο εργασίας σε αρχείο (Excel)

Τώρα που οι τύποι είναι στη θέση τους, το τελευταίο βήμα είναι **να αποθηκεύσετε το βιβλίο εργασίας σε αρχείο**. Μπορείτε να επιλέξετε οποιοδήποτε μονοπάτι—απλώς βεβαιωθείτε ότι ο φάκελος υπάρχει και έχετε δικαιώματα εγγραφής.

```csharp
        // Step 4 (optional): Save the workbook so you can inspect the results
        string outputPath = Path.Combine(
            Environment.GetFolderPath(Environment.SpecialFolder.Desktop),
            "output.xlsx");

        workbook.Save(outputPath);
        Console.WriteLine($"Workbook saved to {outputPath}");
    }
}
```

### Τι αποθηκεύεται στην πραγματικότητα;

Όταν ανοίξετε το `output.xlsx` στο Excel, θα δείτε:

| A | B |
|---|---|
| 1 | 1 |
| 2 |   |
| 3 |   |
|   |   |
|   |   |

- Η στήλη **A** περιέχει τον επεκταμένο πίνακα `{1,2,3}` ακολουθούμενο από δύο κενά κελιά (επειδή ζητήσαμε 5 γραμμές).  
- Το κελί **B1** δείχνει `1`, τη συνεφαπτομένη των 45°.  

Αν ανανεώσετε το βιβλίο εργασίας (πατήστε `F9` ή ενεργοποιήστε τον αυτόματο υπολογισμό), το Excel θα αξιολογήσει τους τύπους και θα εμφανίσει τα αποτελέσματα. Το Aspose.Cells προσφέρει επίσης τη μέθοδο `CalculateFormula` αν χρειάζεστε τις τιμές χωρίς να ανοίξετε το Excel:

```csharp
        workbook.CalculateFormula();
        double cotResult = ws.Cells["B1"].DoubleValue; // should be 1.0
```

## Συχνές Ερωτήσεις & Παγίδες

| Ερώτηση | Απάντηση |
|----------|--------|
| **Πρέπει να ενεργοποιήσω τον υπολογισμό χειροκίνητα;** | Όχι. Από προεπιλογή το Aspose.Cells αποθηκεύει τους τύπους όπως είναι· το Excel θα τους υπολογίσει κατά το άνοιγμα. Χρησιμοποιήστε `workbook.CalculateFormula()` για προ‑υπολογισμό. |
| **Μπορώ να γράψω τύπους σε πολλά κελιά ταυτόχρονα;** | Σίγουρα. Χρησιμοποιήστε `ws.Cells["D1:D5"].Formula = "=RAND()"` για να γεμίσετε μια περιοχή με τυχαίους αριθμούς. |
| **Τι γίνεται αν ο φάκελος προορισμού δεν υπάρχει;** | Δημιουργήστε τον πρώτα: `Directory.CreateDirectory(Path.GetDirectoryName(outputPath));` |
| **Υποστηρίζεται το `EXPAND` σε παλαιότερες εκδόσεις του Excel;** | Το `EXPAND` εμφανίστηκε με το Excel 365/2019. Αν χρειάζεστε συμβατότητα με παλαιότερα αρχεία, σκεφτείτε συνδυασμούς `INDEX`/`SEQUENCE`. |
| **Πώς κρύβω την προβολή τύπου;** | Ορίστε `ws.Cells["A1"].FormulaHidden = true;` και προστατέψτε το φύλλο αν δεν θέλετε οι χρήστες να βλέπουν τον υποκείμενο τύπο. |

## Συμπεράσματα

Το πλήρες, εκτελέσιμο παράδειγμα βρίσκεται στα αποσπάσματα κώδικα παραπάνω—αντιγράψτε το σε μια εφαρμογή console, πατήστε `F5`, και ανοίξτε το παραγόμενο `output.xlsx` για να δείτε τη μαγεία.

### Τι ακολουθεί;

- **Ενσωματώστε πηγές δεδομένων** (SQL, CSV) και τροφοδοτήστε αυτές τις τιμές στους τύπους προγραμματιστικά.
