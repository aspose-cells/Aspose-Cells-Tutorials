## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Υλοποίηση παρόχου ροής

Υλοποίηση μιας προσαρμοσμένης `IStreamProvider` σας επιτρέπει να χειρίζεστε εξωτερικούς πόρους αποτελεσματικά κατά τη μετατροπή. Δείτε πώς μπορείτε να το ρυθμίσετε:

#### Επισκόπηση του Custom IStreamProvider

ΕΝΑ `MyStreamProvider` Η κλάση θα σας βοηθήσει στη φόρτωση εικόνων ή άλλων δυαδικών δεδομένων στις μετατροπές σας από Excel σε PDF.

#### Βήμα προς βήμα εφαρμογή

**1. Ορίστε την Κλάση Παρόχου Ροής**

Δημιουργήστε μια νέα κλάση C# που υλοποιεί `IStreamProvider`Αυτός ο πάροχος αρχικοποιεί ροές με δεδομένα εικόνας:

```csharp
using System.IO;
using Aspose.Cells.Rendering;

class MyStreamProvider : IStreamProvider
{
    // Αρχικοποιεί τη ροή με δεδομένα εικόνας από έναν καθορισμένο κατάλογο πηγής.
    public void InitStream(StreamProviderOptions options)
    {
        string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή του καταλόγου προέλευσης
        
        // Ανάγνωση ενός αρχείου εικόνας σε έναν πίνακα byte και στη συνέχεια σε ένα MemoryStream
        byte[] bts = File.ReadAllBytes(SourceDir + "newPdfSaveOptions_StreamProvider.png");
        MemoryStream ms = new MemoryStream(bts);
        options.Stream = ms; // Αντιστοιχίστε τη ροή μνήμης στην ιδιότητα Stream των επιλογών
    }
    
    // Μέθοδος για το κλείσιμο της ροής, αφήνεται κενή ως σύμβολο κράτησης θέσης.
    public void CloseStream(StreamProviderOptions options)
    {
        // Δεν απαιτείται υλοποίηση για αυτό το παράδειγμα
    }
}
```

**2. Ρύθμιση παραμέτρων μετατροπής PDF**

Στη συνέχεια, θα μετατρέψουμε ένα αρχείο Excel σε PDF χρησιμοποιώντας την προσαρμοσμένη υπηρεσία παροχής ροής μας:

```csharp
using System.IO;
using Aspose.Cells;

class ConvertExcelToPdfWithCustomProvider
{
    // Κύρια μέθοδος για την εκτέλεση της διαδικασίας μετατροπής
    public static void Run()
    {
        string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή του καταλόγου προέλευσης
        string OutputDir = @"YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου εξόδου σας
        
        // Φόρτωση ενός αρχείου Excel από τον καθορισμένο κατάλογο προέλευσης
        Workbook wb = new Workbook(SourceDir + "samplePdfSaveOptions_StreamProvider.xlsx");

        // Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF
        PdfSaveOptions opts = new PdfSaveOptions();
        opts.OnePagePerSheet = true; // Ορίστε κάθε φύλλο εργασίας να αποθηκεύεται ως μία σελίδα στο PDF που προκύπτει
        
        // Ανάθεση προσαρμοσμένου παρόχου ροής για τη διαχείριση εξωτερικών πόρων
        wb.Settings.StreamProvider = new MyStreamProvider();
        
        // Αποθήκευση του βιβλίου εργασίας ως αρχείο PDF στον καθορισμένο κατάλογο εξόδου
        wb.Save(OutputDir + "outputPdfSaveOptions_StreamProvider.pdf", opts);
    }
}
```
