## Οδηγός Εφαρμογής

Αυτός ο οδηγός θα σας καθοδηγήσει στη φόρτωση ενός αρχείου Excel και στην αποθήκευσή του ως HTML με συγκεκριμένες διαμορφώσεις για την απόκρυψη του επικαλυπτόμενου περιεχομένου.

### Φόρτωση αρχείου Excel χρησιμοποιώντας το Aspose.Cells

Ξεκινήστε ρυθμίζοντας τον κατάλογο προέλευσης και φορτώνοντας το επιθυμητό βιβλίο εργασίας του Excel:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου πηγής σας
Workbook wb = new Workbook(SourceDir + "sampleHidingOverlaidContentWithCrossHideRightWhileSavingToHtml.xlsx");
```

### Ρύθμιση παραμέτρων HtmlSaveOptions

Ο `HtmlSaveOptions` Η κλάση σάς επιτρέπει να καθορίσετε τον τρόπο με τον οποίο το περιεχόμενο του Excel μετατρέπεται και εμφανίζεται σε μορφή HTML. Εδώ, θα τη διαμορφώσουμε ώστε να χειρίζεται επικαλυπτόμενο κείμενο:

```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εξόδου σας
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.HtmlCrossStringType = HtmlCrossType.CrossHideRight;
```

### Αποθήκευση Excel ως HTML με Επιλογές

Τέλος, αποθηκεύστε το βιβλίο εργασίας σε ένα αρχείο HTML χρησιμοποιώντας το διαμορφωμένο `HtmlSaveOptions`:

```csharp
wb.Save(outputDir + "outputHidingOverlaidContentWithCrossHideRightWhileSavingToHtml.html", opts);
```
