## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε την υλοποίηση σε διαχειρίσιμα μέρη για μια πιο σαφή κατανόηση.

### Επισκόπηση της λειτουργίας
Αυτή η λειτουργία σάς επιτρέπει να φορτώνετε βιβλία εργασίας του Excel, εξαιρώντας συγκεκριμένα δεδομένα γραφήματος. Αυτό είναι ιδιαίτερα χρήσιμο όταν ασχολείστε με μεγάλα σύνολα δεδομένων, όπου τα δεδομένα γραφήματος μπορούν να καταναλώσουν περιττούς πόρους και χρόνο επεξεργασίας.

### Βήμα προς βήμα εφαρμογή

#### **1. Ορισμός καταλόγων προέλευσης και εξόδου (H3)**

Ξεκινήστε ορίζοντας τους καταλόγους για το αρχείο προέλευσης και τον προορισμό εξόδου:

```csharp
// Καθορίστε τις διαδρομές για τα αρχεία σας
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();
```
**Εξήγηση**Αυτές οι γραμμές καθορίζουν πού βρίσκεται το αρχείο Excel εισόδου σας και πού θέλετε να αποθηκεύσετε το επεξεργασμένο αποτέλεσμα.

#### **2. Ρύθμιση παραμέτρων επιλογών φόρτωσης (H3)**

Ρύθμιση επιλογών φόρτωσης για φιλτράρισμα δεδομένων γραφήματος:

```csharp
// Δημιουργήστε επιλογές φόρτωσης με ένα συγκεκριμένο φίλτρο για δεδομένα
LoadOptions loadOptions = new LoadOptions();
loadOptions.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.Chart);
```
**Εξήγηση**: Εδώ, δημιουργούμε `LoadOptions` και εφαρμόστε ένα `LoadFilter` για να εξαιρέσετε δεδομένα γραφήματος (`~LoadDataFilterOptions.Chart`). Αυτό διασφαλίζει ότι τα γραφήματα δεν φορτώνονται στη μνήμη.

#### **3. Φόρτωση του Βιβλίου Εργασίας (H3)**

Τώρα, φορτώστε το βιβλίο εργασίας σας χρησιμοποιώντας αυτές τις επιλογές:

```csharp
// Χρήση επιλογών φόρτωσης για άνοιγμα ενός αρχείου Excel χωρίς φόρτωση γραφημάτων
Workbook workbook = new Workbook(sourceDir + "sampleLoadExcelFileWithoutChart.xlsx", loadOptions);
```
**Εξήγηση**: Το `Workbook` Ο κατασκευαστής δέχεται μια διαδρομή και `LoadOptions`, φορτώνοντας μόνο τα δεδομένα που καθορίζονται από το φίλτρο σας.

#### **4. Αποθήκευση του επεξεργασμένου αρχείου (H3)**

Τέλος, αποθηκεύστε το επεξεργασμένο βιβλίο εργασίας σας στην επιθυμητή μορφή:

```csharp
// Αποθήκευση του βιβλίου εργασίας ως PDF χωρίς γραφήματα
workbook.Save(outputDir + "outputLoadExcelFileWithoutChart.pdf", SaveFormat.Pdf);
```
**Εξήγηση**: Το `Save` Η μέθοδος εξάγει το αρχείο σε έναν καθορισμένο κατάλογο και μορφή. Εδώ, το μετατρέπουμε σε PDF.
