## Οδηγός Εφαρμογής

### Άνοιγμα και αποθήκευση αρχείου Excel

Αυτή η ενότητα παρουσιάζει τον τρόπο ανοίγματος ενός υπάρχοντος αρχείου Excel και αποθήκευσής του μετά από τροποποιήσεις.

**Επισκόπηση:**

Το άνοιγμα ενός αρχείου είναι το πρώτο βήμα στην επεξεργασία οποιωνδήποτε δεδομένων στο Excel. Μόλις τροποποιηθεί, η αποθήκευση διασφαλίζει ότι όλες οι αλλαγές διατηρούνται.

**Βήματα Υλοποίησης:**

1. **Άνοιγμα ενός υπάρχοντος αρχείου Excel:**

   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   
   // Φόρτωση του υπάρχοντος βιβλίου εργασίας
   Workbook workbook = new Workbook(SourceDir + "sampleSortData.xlsx");
   ```

2. **Αποθήκευση του βιβλίου εργασίας σε έναν καθορισμένο κατάλογο:**

   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";

   // Αποθήκευση του τροποποιημένου βιβλίου εργασίας
   workbook.Save(outputDir + "outputSaveExcel.xlsx");
   ```

### Φύλλο εργασίας της Access και συλλογή κελιών

Η πρόσβαση σε συγκεκριμένα φύλλα εργασίας και τα κελιά τους είναι ζωτικής σημασίας για τον στοχευμένο χειρισμό δεδομένων.

**Επισκόπηση:**

Πλοηγηθείτε σε ένα βιβλίο εργασίας για να αποκτήσετε πρόσβαση σε μεμονωμένα φύλλα και να χειριστείτε τις τιμές των κελιών όπως απαιτείται.

**Βήματα Υλοποίησης:**

1. **Ανοίξτε το Βιβλίο Εργασίας:**

   ```csharp
   string SourceDir = @"YOUR_SOURCE_DIRECTORY";
   
   Workbook workbook = new Workbook(SourceDir + "sampleSortData.xlsx");
   ```

2. **Συγκεκριμένο φύλλο εργασίας και συλλογή κελιών της Access:**

   ```csharp
   // Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
   Worksheet worksheet = workbook.Worksheets[0];
   
   // Λήψη όλων των κελιών από το φύλλο εργασίας
   Cells cells = worksheet.Cells;
   ```

### Αρχικοποίηση και ρύθμιση παραμέτρων DataSorter

Η ρύθμιση παραμέτρων ταξινόμησης επιτρέπει την αποτελεσματική οργάνωση δεδομένων.

**Επισκόπηση:**

Ο `DataSorter` Το αντικείμενο σάς επιτρέπει να ταξινομείτε δεδομένα με βάση καθορισμένα κλειδιά και προτιμήσεις σειράς.

**Βήματα Υλοποίησης:**

1. **Αρχικοποίηση αντικειμένου DataSorter:**

   ```csharp
   Workbook workbook = new Workbook();
   DataSorter sorter = workbook.DataSorter; // Λήψη του αντικειμένου DataSorter
   ```

2. **Ορισμός παραμέτρων ταξινόμησης:**

   ```csharp
   // Ορίστε τη σειρά ταξινόμησης και τα κλειδιά για τις στήλες
   sorter.Order1 = SortOrder.Ascending;
   sorter.Key1 = 0; // Ευρετήριο πρώτης στήλης
   sorter.Order2 = SortOrder.Ascending;
   sorter.Key2 = 1; // Ευρετήριο δεύτερης στήλης
   ```

### Καθορισμός περιοχής κελιού και εκτέλεση ταξινόμησης

Ο ορισμός μιας περιοχής κελιών για την ταξινόμηση δεδομένων εντός καθορισμένων ορίων είναι απαραίτητος για την ακριβή ταξινόμηση.

**Επισκόπηση:**

Χρησιμοποιώντας `CellArea`, μπορείτε να καθορίσετε το ακριβές εύρος που χρειάζεται ταξινόμηση, διασφαλίζοντας την ακρίβεια στις εργασίες χειρισμού δεδομένων σας.

**Βήματα Υλοποίησης:**

1. **Ορισμός Περιοχής Κελιού (Εύρος):**

   ```csharp
   Workbook workbook = new Workbook();
   DataSorter sorter = workbook.DataSorter;

   // Δημιουργήστε μια περιοχή κελιών για ταξινόμηση
   CellArea ca = new CellArea { StartRow = 1, EndRow = 9, StartColumn = 0, EndColumn = 2 };
   ```

2. **Εκτέλεση ταξινόμησης:**

   ```csharp
   // Ταξινόμηση του καθορισμένου εύρους στο πρώτο φύλλο εργασίας
   sorter.Sort(workbook.Worksheets[0].Cells, ca);
   ```
