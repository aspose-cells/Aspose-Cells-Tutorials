## Οδηγός Εφαρμογής

### Δημιουργία καταλόγου

#### Επισκόπηση
Αυτή η λειτουργία διασφαλίζει ότι ο κατάλογος προορισμού σας υπάρχει πριν από την εκτέλεση λειτουργιών αρχείων, αποτρέποντας σφάλματα κατά την αποθήκευση αρχείων.

##### Βήμα 1: Έλεγχος και δημιουργία καταλόγου
```csharp
using System.IO;

string SourceDir = "YOUR_SOURCE_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου πηγής σας
bool IsExists = Directory.Exists(SourceDir);
if (!IsExists)
{
    Directory.CreateDirectory(SourceDir); 
}
```
- **Εξήγηση:** Αυτός ο κώδικας ελέγχει αν υπάρχει ένας συγκεκριμένος κατάλογος και τον δημιουργεί χρησιμοποιώντας `Directory.CreateDirectory` αν όχι.

### Δημιουργία και αποθήκευση βιβλίου εργασίας με το Aspose.Cells

#### Επισκόπηση
Μάθετε πώς να δημιουργείτε ένα βιβλίο εργασίας του Excel, να το συμπληρώνετε με δεδομένα και να το αποθηκεύετε στην επιθυμητή θέση.

##### Βήμα 2: Δημιουργία αντικειμένου βιβλίου εργασίας
```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εξόδου σας

Workbook workbook = new Workbook(); 
Worksheet worksheet = workbook.Worksheets[0];
```
- **Εξήγηση:** Ένα νέο `Workbook` Το αντικείμενο δημιουργείται και έχουμε πρόσβαση στο πρώτο φύλλο.

##### Βήμα 3: Προσθήκη δεδομένων σε κελιά
```csharp
// Προσθήκη διαφόρων τύπων τιμών σε κελιά
worksheet.Cells["A1"].PutValue("Hello World"); // Τιμή συμβολοσειράς
worksheet.Cells["A2"].PutValue(20.5);          // Διπλή αξία
worksheet.Cells["A3"].PutValue(15);            // Ακέραιη τιμή
worksheet.Cells["A4"].PutValue(true);          // Λογική τιμή

// Προσθήκη τιμής ημερομηνίας/ώρας και ρύθμιση της μορφής εμφάνισής της
DateTime now = DateTime.Now;
worksheet.Cells["A5"].PutValue(now);
Style style = worksheet.Cells["A5"].GetStyle();
style.Number = 15;                             // Μορφή αριθμού για ημερομηνία
worksheet.Cells["A5"].SetStyle(style);
```
- **Εξήγηση:** Ο κώδικας συμπληρώνει διαφορετικούς τύπους δεδομένων σε κελιά, συμπεριλαμβανομένης μιας μορφοποιημένης ημερομηνίας.

##### Βήμα 4: Αποθήκευση του αρχείου Excel
```csharp
workbook.Save(Path.Combine(outputDir, "output.out.xls"));
```
- **Εξήγηση:** Αυτό αποθηκεύει το βιβλίο εργασίας σας στον καθορισμένο κατάλογο. Βεβαιωθείτε ότι `outputDir` ορίζεται σωστά.
