## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει τη διαδικασία σε δύο κύρια χαρακτηριστικά: τη φόρτωση ενός αρχείου HTML και τη ρύθμιση παραμέτρων των επιλογών φόρτωσης.

### Λειτουργία 1: Φόρτωση και αποθήκευση αρχείου HTML ως Excel

#### Επισκόπηση:
Μετατρέψτε ένα υπάρχον αρχείο HTML σε ένα πλήρως λειτουργικό βιβλίο εργασίας Excel χρησιμοποιώντας το Aspose.Cells για .NET. 

##### Βήμα προς βήμα εφαρμογή:

**1. Ορίστε τους καταλόγους προέλευσης και εξόδου:**
Ξεκινήστε ορίζοντας τους καταλόγους όπου βρίσκεται το αρχείο HTML προέλευσης και όπου θέλετε να αποθηκεύσετε το αρχείο Excel εξόδου.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**2. Ορίστε τη διαδρομή προς το αρχείο HTML σας:**
Δημιουργήστε μια διαδρομή για το αρχείο HTML πηγής χρησιμοποιώντας `System.IO.Path.Combine`.

```csharp
string filePath = System.IO.Path.Combine(SourceDir, "Book1.html");
```

**3. Διαμόρφωση επιλογών φόρτωσης:**
Δημιουργήστε ένα στιγμιότυπο του `HtmlLoadOptions` τάξη με `LoadFormat.Html`Αυτό το βήμα καθορίζει ότι φορτώνετε ένα έγγραφο HTML.

```csharp
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
```

**4. Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας:**
Χρησιμοποιήστε το `Workbook` κατασκευαστή για να ανοίξετε το αρχείο σας χρησιμοποιώντας την καθορισμένη διαδρομή και τις επιλογές φόρτωσης.

```csharp
Workbook wb = new Workbook(filePath, loadOptions);
```

**5. Αποθήκευση ως αρχείο Excel:**
Τέλος, αποθηκεύστε το βιβλίο εργασίας στον επιθυμητό κατάλογο εξόδου.

```csharp
string outputFilePath = System.IO.Path.Combine(outputDir, "output.xlsx");
wb.Save(outputFilePath);
```

### Λειτουργία 2: Ρύθμιση παραμέτρων επιλογών φόρτωσης για αρχεία HTML

#### Επισκόπηση:
Μάθετε πώς να τροποποιείτε τις επιλογές φόρτωσης για να προσαρμόσετε τον τρόπο αντιμετώπισης του αρχείου HTML κατά τη μετατροπή του σε βιβλίο εργασίας του Excel.

##### Βήμα προς βήμα εφαρμογή:

**1. Ορισμός καταλόγου πηγής:**
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
```

**2. Ορίστε τις επιλογές διαδρομής και φόρτωσης με διαμορφώσεις:**
Χρησιμοποιήστε την ίδια ρύθμιση διαδρομής όπως πριν, αλλά ρυθμίστε πρόσθετες επιλογές φόρτωσης εάν χρειάζεται, όπως η αναγνώριση περιεχομένου HTML ως πλήρους βιβλίου εργασίας.

```csharp
string filePath = System.IO.Path.Combine(SourceDir, "Book1.html");
HtmlLoadOptions loadOptions = new HtmlLoadOptions(LoadFormat.Html);
loadOptions.IsRecognizeAsSingleFile = true;  // Παράδειγμα επιλογής διαμόρφωσης
```

**3. Δημιουργία και αποθήκευση βιβλίου εργασίας:**
Δημιουργήστε το βιβλίο εργασίας με αυτές τις διαμορφωμένες επιλογές και αποθηκεύστε το.

```csharp
Workbook wb = new Workbook(filePath, loadOptions);
string outputFilePath = System.IO.Path.Combine(SourceDir, "output.xlsx");
b.Save(outputFilePath);
```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή του αρχείου HTML έχει καθοριστεί σωστά.
- Ελέγξτε για τυχόν προβλήματα αδειοδότησης που ενδέχεται να επηρεάσουν τη διαδικασία φόρτωσης.
