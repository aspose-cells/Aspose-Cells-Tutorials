# Εξοικείωση με την προεπιλεγμένη ρύθμιση γραμματοσειράς στη μετατροπή Excel σε HTML με το Aspose.Cells για .NET

Η μετατροπή ενός βιβλίου εργασίας του Excel σε μορφή HTML, διατηρώντας παράλληλα σταθερή την τυπογραφία, μπορεί να είναι μια δύσκολη διαδικασία. Αυτό το σεμινάριο σάς καθοδηγεί στον ορισμό μιας προεπιλεγμένης γραμματοσειράς χρησιμοποιώντας το Aspose.Cells για .NET, διασφαλίζοντας ότι τα μετατρεπόμενα έγγραφά σας θα φαίνονται κομψά και επαγγελματικά. Κατακτώντας πλήρως αυτήν τη λειτουργία, θα ξεπεράσετε τις προκλήσεις που σχετίζονται με άγνωστες ή μη διαθέσιμες γραμματοσειρές στη διαδικασία μετατροπής.

**Τι θα μάθετε:**
- Πώς να ορίσετε μια προεπιλεγμένη γραμματοσειρά κατά τη μετατροπή αρχείων Excel σε HTML.
- Οδηγίες βήμα προς βήμα σχετικά με τη χρήση του Aspose.Cells για .NET.
- Τεχνικές για την ομαλή διαχείριση άγνωστων γραμματοσειρών κατά την απόδοση.

Ας ξεκινήσουμε τη ρύθμιση του περιβάλλοντός σας και ας ξεκινήσουμε την εξερεύνηση αυτής της λειτουργίας!

## Οδηγός Εφαρμογής

### Ορισμός προεπιλεγμένης γραμματοσειράς κατά την απόδοση

Αυτή η λειτουργία διασφαλίζει ότι ένα βιβλίο εργασίας του Excel αποδίδεται με μια συγκεκριμένη προεπιλεγμένη γραμματοσειρά κατά τη μετατροπή σε HTML. Είναι ιδιαίτερα χρήσιμο για τον χειρισμό περιπτώσεων όπου ορισμένες γραμματοσειρές ενδέχεται να μην είναι διαθέσιμες στο σύστημα-στόχο.

#### Βήμα 1: Δημιουργία και πρόσβαση σε βιβλίο εργασίας

Δημιουργήστε μια νέα παρουσία του `Workbook` και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας του:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας.
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```

#### Βήμα 2: Τροποποίηση στυλ κελιού

Αποκτήστε πρόσβαση σε ένα συγκεκριμένο κελί, προσθέστε κείμενο και ορίστε τη γραμματοσειρά σε άγνωστη για επίδειξη:
```csharp
// Αποκτήστε πρόσβαση στο κελί B4 και προσθέστε κείμενο μέσα σε αυτό.
Cell cell = ws.Cells["B4"];
cell.PutValue("This text has some unknown or invalid font which does not exist.");

// Ορίστε τη γραμματοσειρά του κελιού B4 σε άγνωστη γραμματοσειρά.
Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist";
st.Font.Size = 20;
cell.SetStyle(st);
```

#### Βήμα 3: Ορισμός επιλογών αποθήκευσης HTML

Ορίστε την προεπιλεγμένη γραμματοσειρά στην έξοδο HTML. Εδώ, παρουσιάζουμε τρεις διαφορετικές γραμματοσειρές:

**Νέος Ταχυμεταφορέας:**
```csharp
// Αποθηκεύστε το βιβλίο εργασίας σε μορφή HTML με την προεπιλεγμένη γραμματοσειρά να έχει οριστεί σε Courier New.
HtmlSaveOptions optsCourierNew = new HtmlSaveOptions();
optsCourierNew.DefaultFontName = "Courier New";
wb.Save(outputDir + "/out_courier_new_out.htm", optsCourierNew);
```

**Arial:**
```csharp
// Αποθηκεύστε το βιβλίο εργασίας σε μορφή HTML με την προεπιλεγμένη γραμματοσειρά να έχει οριστεί σε Arial.
HtmlSaveOptions optsArial = new HtmlSaveOptions();
optsArial.DefaultFontName = "Arial";
wb.Save(outputDir + "/out_arial_out.htm", optsArial);
```

**Times New Roman:**
```csharp
// Αποθηκεύστε το βιβλίο εργασίας σε μορφή HTML με την προεπιλεγμένη γραμματοσειρά Times New Roman.
HtmlSaveOptions optsTimesNewRoman = new HtmlSaveOptions();
optsTimesNewRoman.DefaultFontName = "Times New Roman";
wb.Save(outputDir + "/times_new_roman_out.htm", optsTimesNewRoman);
```

### Δημιουργία βιβλίου εργασίας και στυλ κελιών

Αυτή η ενότητα καλύπτει τη δημιουργία ενός βιβλίου εργασίας, την πρόσβαση σε φύλλα εργασίας, κελιά και την εφαρμογή στυλ:

#### Βήμα 1: Αρχικοποίηση βιβλίου εργασίας
Δημιουργήστε ένα νέο `Workbook` παράδειγμα:
```csharp
// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας.
Workbook wb = new Workbook();
```

#### Βήμα 2: Φύλλο εργασίας και κελί πρόσβασης
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και στο κελί B4 για να προσθέσετε κείμενο και να το διαμορφώσετε:
```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.
Worksheet ws = wb.Worksheets[0];

// Αποκτήστε πρόσβαση στο κελί B4 και προσθέστε κείμενο μέσα σε αυτό.
Cell cell = ws.Cells["B4"];
cell.PutValue("This text has some unknown or invalid font which does not exist.");

// Ορίστε τη γραμματοσειρά του κελιού B4 σε άγνωστη γραμματοσειρά.
Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist";
st.Font.Size = 20;
cell.SetStyle(st);
```
