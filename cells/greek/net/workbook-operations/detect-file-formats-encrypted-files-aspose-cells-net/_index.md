## Οδηγός Εφαρμογής

Η ανίχνευση της μορφής των κρυπτογραφημένων αρχείων είναι απλή με το Aspose.Cells. Αυτή η λειτουργία σάς επιτρέπει να προσδιορίσετε τη μορφή ενός αρχείου Excel χωρίς να το αποκρυπτογραφήσετε πλήρως, εξασφαλίζοντας ασφάλεια και αποτελεσματικότητα.
#### Επισκόπηση:
Αυτή η λειτουργικότητα επιτρέπει την αποτελεσματική ανίχνευση μορφών αρχείων από κρυπτογραφημένα έγγραφα.
### Βήμα 1: Ρύθμιση του περιβάλλοντος σας
Βεβαιωθείτε ότι το έργο σας αναφέρει την απαραίτητη συναρμολόγηση Aspose.Cells.

### Βήμα 2: Άνοιγμα και ανάγνωση του κρυπτογραφημένου αρχείου
Ανοίξτε το κρυπτογραφημένο αρχείο σας χρησιμοποιώντας μια ροή. Εδώ, θα χρησιμοποιήσουμε ένα δείγμα ονόματος αρχείου `encryptedBook1.out.tmp`.
```csharp
public static void Run()
{
    string sourceDir = "Your Source Directory Path";
    var filename = sourceDir + "encryptedBook1.out.tmp";

    // Άνοιγμα του αρχείου σε λειτουργία μόνο για ανάγνωση
    using (Stream stream = File.Open(filename, FileMode.Open))
    {
        // Εντοπισμός μορφής με γνωστό κωδικό πρόσβασης
        FileFormatInfo fileFormatInfo = FileFormatUtil.DetectFileFormat(stream, "1234"); 

        Console.WriteLine("File Format: " + fileFormatInfo.FileFormatType);
    }
}
```
### Εξήγηση:
- **Ρεύμα**Μια ροή παρέχει έναν τρόπο ανάγνωσης των δεδομένων του αρχείου. Εδώ, ανοίγουμε το αρχείο χρησιμοποιώντας `File.Open`.
- **FileFormatUtil.DetectΜορφή αρχείου**: Αυτή η μέθοδος λαμβάνει τη ροή και τον κωδικό πρόσβασης (`"1234"`), ανιχνεύοντας τη μορφή χωρίς να την αποκρυπτογραφήσει πλήρως.
#### Παράμετροι:
- **ρεύμα**: Η ροή αρχείων του κρυπτογραφημένου εγγράφου σας.
- **σύνθημα**: Μια συμβολοσειρά που αντιπροσωπεύει τον κωδικό πρόσβασης που χρησιμοποιείται για την κρυπτογράφηση του εγγράφου. Είναι απαραίτητο το Aspose.Cells να προσδιορίσει σωστά τη μορφή αρχείου.
### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή προς τον κατάλογο προέλευσης είναι σωστή και προσβάσιμη.
- Επαληθεύστε ότι ο κωδικός πρόσβασης που παρέχεται ταιριάζει με αυτόν που χρησιμοποιήθηκε κατά την κρυπτογράφηση. Διαφορετικά, η ανίχνευση θα αποτύχει.
