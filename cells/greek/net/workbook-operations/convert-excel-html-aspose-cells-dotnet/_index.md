## Οδηγός Εφαρμογής

### Βήμα 1: Φόρτωση του αρχείου Excel
Ξεκινήστε φορτώνοντας το αρχείο προέλευσης Excel σε ένα `Workbook` αντικείμενο. Αυτό το βήμα είναι κρίσιμο καθώς ρυθμίζει τα δεδομένα που θα μετατραπούν.

```csharp
// Καθορίστε τους καταλόγους σας
string sourceDir = "path_to_source_directory";
string outputDir = "path_to_output_directory";

// Φόρτωση αρχείου Excel
Workbook workbook = new Workbook(sourceDir + "sampleUsePresentationPreferenceOption.xlsx");
```

### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML
Για να βελτιώσουμε την παρουσίαση, θα χρησιμοποιήσουμε `HtmlSaveOptions`Αυτό μας επιτρέπει να ορίσουμε προτιμήσεις που βελτιώνουν τον τρόπο εμφάνισης των δεδομένων σε HTML.

```csharp
// Δημιουργία αντικειμένου HtmlSaveOptions
HtmlSaveOptions options = new HtmlSaveOptions();

// Ορισμός επιλογής προτίμησης παρουσίασης
options.PresentationPreference = true;
```

### Βήμα 3: Αποθηκεύστε το αρχείο Excel ως HTML
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε ένα αρχείο HTML με βελτιωμένες ρυθμίσεις παρουσίασης.

```csharp
// Αποθήκευση του αρχείου Excel σε HTML με καθορισμένη επιλογή
workbook.Save(outputDir + "outputUsePresentationPreferenceOption.html", options);

Console.WriteLine("Conversion executed successfully.");
```
