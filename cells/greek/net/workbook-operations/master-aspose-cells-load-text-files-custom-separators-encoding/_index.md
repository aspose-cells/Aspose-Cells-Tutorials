## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη φόρτωση αρχείων κειμένου με προσαρμοσμένους διαχωριστές και κωδικοποίηση και τη διαμόρφωση διαδρομών καταλόγων δεδομένων.

### Φόρτωση αρχείων κειμένου με προσαρμοσμένο διαχωριστικό και κωδικοποίηση

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε ένα προσαρμοσμένο διαχωριστικό για το αρχείο κειμένου σας (όπως ένα κόμμα για CSV) και να ορίσετε τον τύπο κωδικοποίησης, όπως UTF8. Αυτό είναι ιδιαίτερα χρήσιμο όταν πρόκειται για διεθνή σύνολα δεδομένων ή μη τυπικές μορφές αρχείων.

#### Βήματα Υλοποίησης

1. **Ορισμός καταλόγων προέλευσης και εξόδου**
   Καθορίστε πού βρίσκονται τα αρχεία κειμένου πηγής και πού θέλετε να αποθηκεύσετε τα επεξεργασμένα δεδομένα:

   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Δημιουργία αρχικών επιλογών φόρτωσης**
   Δημιουργήστε ένα `TxtLoadOptions` αντικείμενο για να καθορίσετε προσαρμοσμένες ρυθμίσεις φόρτωσης:

   ```csharp
   TxtLoadOptions txtLoadOptions = new TxtLoadOptions();
   ```

3. **Ορισμός προσαρμοσμένου διαχωριστή και κωδικοποίησης**
   Αντιστοιχίστε τον διαχωριστή και τον τύπο κωδικοποίησης:

   ```csharp
   // Καθορίστε το διαχωριστικό (π.χ., κόμμα για αρχεία CSV)
   txtLoadOptions.Separator = Convert.ToChar(",");

   // Καθορίστε τον τύπο κωδικοποίησης (π.χ., UTF8)
   txtLoadOptions.Encoding = Encoding.UTF8;
   ```

4. **Δημιουργία και φόρτωση βιβλίου εργασίας**
   Χρήση `Workbook` για να φορτώσετε το αρχείο κειμένου σας με τις καθορισμένες επιλογές:

   ```csharp
   Workbook wb = new Workbook(SourceDir + "/Book11.csv", txtLoadOptions);
   ```

5. **Αποθήκευση επεξεργασμένων δεδομένων**
   Αποθηκεύστε το βιβλίο εργασίας στον επιθυμητό κατάλογο εξόδου:

   ```csharp
   wb.Save(outputDir + "/output.txt");
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές είναι σωστά καθορισμένες και προσβάσιμες.
- Επαληθεύστε ότι ο διαχωριστής και η κωδικοποίηση ταιριάζουν με τις προδιαγραφές του αρχείου για να αποφύγετε σφάλματα ανάλυσης.

### Χειρισμός διαμόρφωσης διαδρομής καταλόγου δεδομένων

#### Επισκόπηση
Η αποτελεσματική ρύθμιση παραμέτρων των καταλόγων προέλευσης και εξόδου μπορεί να βελτιστοποιήσει τη ροή εργασίας επεξεργασίας δεδομένων, ειδικά όταν πρόκειται για μεγάλα σύνολα δεδομένων ή πολλά αρχεία.

#### Βήματα Υλοποίησης
1. **Ορισμός διαδρομών**
   Ορίστε placeholders για τις διαδρομές καταλόγου σας:

   ```csharp
   string SourceDir = "YOUR_SOURCE_DIRECTORY";
   string OutputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Χρήση στην εφαρμογή**
   Ενσωματώστε αυτές τις διαδρομές στη λογική της εφαρμογής σας για να διαχειρίζεστε απρόσκοπτα τις λειτουργίες αρχείων.
