## Οδηγός Εφαρμογής
Αφού ρυθμιστεί το περιβάλλον, ας προχωρήσουμε στη διαδικασία μετατροπής.

### Μετατροπή φύλλου εργασίας Excel σε εικόνα TIFF
Αυτή η ενότητα παρουσιάζει τη μετατροπή ενός φύλλου εργασίας Excel σε μια εικόνα TIFF πολλαπλών σελίδων χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας σας
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας του Excel:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook book = new Workbook(sourceDir + "sampleConvertWorksheetToImageByPage.xlsx");
```
Εδώ, φορτώνουμε ένα αρχείο Excel από έναν καθορισμένο κατάλογο.

#### Βήμα 2: Πρόσβαση στο φύλλο εργασίας σας
Αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε να μετατρέψετε. Σε αυτό το παράδειγμα, χρησιμοποιούμε το πρώτο φύλλο εργασίας:
```csharp
Worksheet sheet = book.Worksheets[0];
```

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών εικόνας
Ρυθμίστε τις επιλογές μετατροπής εικόνας, συμπεριλαμβανομένης της ανάλυσης και της μορφής:
```csharp
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.HorizontalResolution = 200;
options.VerticalResolution = 200;
options.ImageType = Drawing.ImageType.Tiff; // Ορισμός εξόδου ως TIFF
```
Αυτές οι διαμορφώσεις διασφαλίζουν ότι οι εικόνες που δημιουργούνται έχουν υψηλή ανάλυση και είναι σε μορφή TIFF.

#### Βήμα 4: Μετατροπή σελίδων φύλλου εργασίας σε εικόνες
Χρήση `SheetRender` για να μετατρέψετε κάθε σελίδα του φύλλου εργασίας σας σε εικόνα:
```csharp
SheetRender sr = new SheetRender(sheet, options);
for (int j = 0; j < sr.PageCount; j++)
{
    sr.ToImage(j, outputDir + "outputConvertWorksheetToImageByPage_" + (j + 1) + ".tif");
}
```
Αυτός ο βρόχος επαναλαμβάνεται σε κάθε σελίδα του φύλλου εργασίας και την αποθηκεύει ως εικόνα TIFF.
