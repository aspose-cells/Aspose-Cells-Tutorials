## Οδηγός Εφαρμογής

### Βήμα 1: Ορισμός διαδρομών αρχείων
Καθορίστε τις διαδρομές για το αρχείο προέλευσης Excel και το αρχείο MHT εξόδου.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

string filePath = SourceDir + "/Book1.xlsx"; // Εισαγωγή διαδρομής αρχείου Excel
string outputPath = outputDir + "/Book1.out.mht"; // Διαδρομή αρχείου MHT εξόδου
```

### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML
Ρυθμίστε τις παραμέτρους αποθήκευσης για να μετατρέψετε το αρχείο Excel σε μορφή MHTML.
```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.MHTML);
```
Ο `HtmlSaveOptions` Η κλάση παρέχει διαμορφώσεις για την αποθήκευση βιβλίων εργασίας σε μορφές που βασίζονται σε HTML. Ρύθμιση `SaveFormat.MHTML` συνδυάζει όλους τους πόρους (εικόνες, CSS) σε ένα μόνο αρχείο.

### Βήμα 3: Φόρτωση του βιβλίου εργασίας του Excel
Φορτώστε το βιβλίο εργασίας του Excel χρησιμοποιώντας τη διαδρομή που ορίστηκε προηγουμένως.
```csharp
Workbook workbook = new Workbook(filePath);
```
Ο `Workbook` Η κλάση στο Aspose.Cells αντιπροσωπεύει ένα ολόκληρο έγγραφο Excel. Η φόρτωσή της επιτρέπει τον χειρισμό των δεδομένων που περιέχει.

### Βήμα 4: Αποθήκευση ως MHT
Αποθηκεύστε το βιβλίο εργασίας στην επιθυμητή διαδρομή εξόδου χρησιμοποιώντας τις διαμορφωμένες επιλογές.
```csharp
workbook.save(outputPath, saveOptions);
```
Αυτό το βήμα μετατρέπει και αποθηκεύει το αρχείο Excel σε μορφή MHTML, διατηρώντας τη διάταξη και το στυλ του για χρήση στο web.
