### Οδηγός Εφαρμογής

#### Δημιουργία και Διαχείριση Καταλόγου

Αυτή η λειτουργία διασφαλίζει ότι η εφαρμογή σας μπορεί να δημιουργήσει καταλόγους με ασφάλεια χωρίς σφάλματα.

##### Έλεγχος ύπαρξης καταλόγου και δημιουργία του

Για να διαχειριστείτε αποτελεσματικά τους καταλόγους, ακολουθήστε τα παρακάτω βήματα:

1. **Ελέγξτε αν υπάρχει κατάλογος:**

    ```csharp
    using System.IO;

    string SourceDir = "YOUR_SOURCE_DIRECTORY";
    bool IsExists = System.IO.Directory.Exists(SourceDir);
    ```

   - `Directory.Exists`Ελέγχει εάν η καθορισμένη διαδρομή αναφέρεται σε έναν υπάρχοντα κατάλογο.

2. **Δημιουργήστε τον κατάλογο εάν δεν υπάρχει:**

    ```csharp
    if (!IsExists)
        System.IO.Directory.CreateDirectory(SourceDir);
    ```

   - `Directory.CreateDirectory`Δημιουργεί όλους τους καταλόγους και τους υποκαταλόγους στην καθορισμένη διαδρομή, εκτός εάν υπάρχουν ήδη.

#### Δημιουργία και διαχείριση βιβλίου εργασίας Excel

Με το Aspose.Cells, μπορείτε να δημιουργήσετε σύνθετα βιβλία εργασίας του Excel μέσω προγραμματισμού. Ας εξερευνήσουμε πώς να προσθέτετε φύλλα εργασίας, να εισάγετε δεδομένα, να εφαρμόζετε τύπους και να αποθηκεύετε το βιβλίο εργασίας σας.

##### Δημιουργία στιγμιαίου αντικειμένου βιβλίου εργασίας

Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Workbook` τάξη:

```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY";
Workbook workbook = new Workbook();
```

- Ο `Workbook` Το αντικείμενο είναι η βασική οντότητα που αντιπροσωπεύει ένα αρχείο Excel στο Aspose.Cells.

##### Προσθήκη Φύλλων Εργασίας και Συμπλήρωση Κελιών

1. **Προσθήκη νέου φύλλου εργασίας:**

    ```csharp
    int sheetIndex = workbook.Worksheets.Add();
    Worksheet worksheet = workbook.Worksheets[0];
    ```

   - Χρήση `Worksheets.Add()` για να προσθέσετε ένα νέο φύλλο εργασίας στο τέλος της συλλογής.

2. **Εισαγωγή δεδομένων σε κελιά:**

    ```csharp
    worksheet.Cells["A1"].PutValue(1);
    worksheet.Cells["A2"].PutValue(2);
    worksheet.Cells["A3"].PutValue(3);
    ```

   - `PutValue`: Ορίζει την τιμή ενός συγκεκριμένου κελιού.

##### Εφαρμογή τύπων και υπολογισμός αποτελεσμάτων

Για να αυτοματοποιήσετε τους υπολογισμούς, εφαρμόστε τύπους σε κελιά:

```csharp
worksheet.Cells["A4"].Formula = "=SUM(A1:A3)";
workbook.CalculateFormula();
```

- `CalculateFormula()`: Αξιολογεί όλους τους τύπους στο βιβλίο εργασίας.

Ανακτήστε τις υπολογισμένες τιμές όπως απαιτείται:

```csharp
string value = worksheet.Cells["A4"].Value.ToString();
```

##### Αποθήκευση του αρχείου Excel

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε έναν καθορισμένο κατάλογο:

```csharp
workbook.Save(outputDir + "/output.xls");
```

- `Save`Εγγράφει αλλαγές σε ένα αρχείο Excel στη δεδομένη διαδρομή.
