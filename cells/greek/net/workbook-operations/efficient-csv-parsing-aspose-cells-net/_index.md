## Οδηγός Εφαρμογής

### Υλοποίηση προσαρμοσμένου αναλυτή

#### Επισκόπηση

Η δημιουργία προσαρμοσμένων αναλυτών σάς επιτρέπει να χειρίζεστε συγκεκριμένους τύπους δεδομένων πιο αποτελεσματικά κατά τη φόρτωση αρχείων CSV. Αυτή η ενότητα δείχνει πώς να εφαρμόσετε το `ICustomParser` διεπαφή για ανάλυση κειμένου και ημερομηνίας.

##### Υλοποίηση της κλάσης TextParser

Αυτή η κλάση επιστρέφει κείμενο ως έχει, διατηρώντας την αρχική του μορφή στο σύνολο δεδομένων σας:

```csharp
using Aspose.Cells;

public class TextParser : ICustomParser
{
    public object ParseObject(string value)
    {
        return value; // Επιστρέψτε τη συμβολοσειρά ως έχει
    }
    
    public string GetFormat()
    {
        return "";
    }
}
```

##### Υλοποίηση της κλάσης DateParser

Αυτός ο αναλυτής μετατρέπει τις συμβολοσειρές ημερομηνίας σε `DateTime` αντικείμενα, μορφοποιημένα ως `dd/MM/yyyy`.

```csharp
using Aspose.Cells;

public class DateParser : ICustomParser
{
    public object ParseObject(string value)
    {
        DateTime myDate = DateTime.ParseExact(value, "dd/MM/yyyy", System.Globalization.CultureInfo.InvariantCulture);
        return myDate;
    }
    
    public string GetFormat()
    {
        return "dd/MM/yyyy";
    }
}
```

### Φόρτωση CSV με Προτιμώμενους Αναλυτές

#### Επισκόπηση

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο CSV χρησιμοποιώντας το Aspose.Cells κατά την εφαρμογή προσαρμοσμένων αναλυτών για δεδομένα κειμένου και ημερομηνίας.

##### Ρύθμιση της κλάσης Loader

Δείτε πώς μπορείτε να διαμορφώσετε τον φορτωτή σας ώστε να χρησιμοποιεί τους προτιμώμενους αναλυτές:

```csharp
using System.IO;
using Aspose.Cells;

namespace CsvLoadingExample
{
    public class CsvLoaderWithPreferredParsers
    {
        static string SourceDir = @"YOUR_SOURCE_DIRECTORY";
        static string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

        public void LoadCsv()
        {
            // Αρχικοποίηση LoadFormat για αρχεία CSV
            LoadFormat oLoadFormat = LoadFormat.Csv;

            // Δημιουργήστε TxtLoadOptions με την καθορισμένη μορφή φόρτωσης
            TxtLoadOptions oTxtLoadOptions = new TxtLoadOptions(oLoadFormat);

            // Ορισμός διαχωριστικού ως κόμμα και κωδικοποίηση σε UTF-8
            oTxtLoadOptions.Separator = ',';
            oTxtLoadOptions.Encoding = System.Text.Encoding.UTF8;

            // Ενεργοποίηση μετατροπής δεδομένων ημερομηνίας/ώρας κατά τη φόρτωση
            oTxtLoadOptions.ConvertDateTimeData = true;

            // Αντιστοίχιση προσαρμοσμένων αναλυτών για τη διαχείριση συγκεκριμένων τύπων δεδομένων σε CSV
            oTxtLoadOptions.PreferredParsers = new ICustomParser[] { new TextParser(), new DateParser() };

            // Φόρτωση του αρχείου CSV σε ένα αντικείμενο βιβλίου εργασίας χρησιμοποιώντας καθορισμένες επιλογές φόρτωσης
            Workbook oExcelWorkBook = new Workbook(SourceDir + "samplePreferredParser.csv", oTxtLoadOptions);

            // Πρόσβαση και εμφάνιση πληροφοριών από συγκεκριμένα κελιά για επαλήθευση της ανάλυσης
            Cell oCell = oExcelWorkBook.Worksheets[0].Cells["A1"];
            Console.WriteLine($"Value in A1: {oCell.Value}, Type: {oCell.Value.GetType()}");

            oCell = oExcelWorkBook.Worksheets[0].Cells["B1"];
            Console.WriteLine($"Value in B1: {oCell.Value}, Type: {oCell.Value.GetType()}");

            // Αποθήκευση του βιβλίου εργασίας στον καθορισμένο κατάλογο εξόδου
            oExcelWorkBook.Save(OutputDir + "outputsamplePreferredParser.xlsx");
        }
    }
}
```
