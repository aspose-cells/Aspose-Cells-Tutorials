## Οδηγός Εφαρμογής
Ας δούμε βήμα προς βήμα την εφαρμογή της λειτουργίας μας.
### Φόρτωση βιβλίου εργασίας από ροή μνήμης και ρύθμιση μεγέθους χαρτιού
#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να φορτώσετε ένα βιβλίο εργασίας του Excel στη μνήμη και να ορίσετε προσαρμοσμένα μεγέθη εκτυπωτή πριν το εξαγάγετε ως αρχείο PDF.
##### Βήμα 1: Δημιουργία και αποθήκευση βιβλίου εργασίας στη μνήμη
Αρχικά, δημιουργήστε ένα βιβλίο εργασίας με δείγματα δεδομένων και αποθηκεύστε το σε ένα `MemoryStream`.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε ένα νέο βιβλίο εργασίας και φύλλο εργασίας
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Cells["P30"].PutValue("This is sample data.");

// Αποθήκευση στη ροή μνήμης
MemoryStream ms = new MemoryStream();
workbook.Save(ms, SaveFormat.Xlsx);
ms.Position = 0;
```
##### Βήμα 2: Τοποθέτηση βιβλίου εργασίας με προσαρμοσμένο μέγεθος χαρτιού
Φόρτωση του βιβλίου εργασίας από το `MemoryStream` και ορίστε ένα συγκεκριμένο μέγεθος χαρτιού.
```csharp
// Ορίστε το μέγεθος χαρτιού σε A5 και τοποθετήστε το βιβλίο εργασίας
LoadOptions opts = new LoadOptions(LoadFormat.Xlsx);
opts.SetPaperSize(PaperSizeType.PaperA5);
workbook = new Workbook(ms, opts);

// Αποθήκευση ως PDF με ρύθμιση A5
workbook.Save(outputDir + "outputLoadWorkbookWithPrinterSize-A5.pdf");
```
##### Βήμα 3: Αλλαγή μεγέθους χαρτιού και εξαγωγή ξανά
Επαναφέρετε τη θέση της ροής για να φορτώσετε ξανά το βιβλίο εργασίας με διαφορετικό μέγεθος χαρτιού.
```csharp
ms.Position = 0;

// Ορίστε το μέγεθος χαρτιού σε A3 και επανατοποθετήστε το
opts.SetPaperSize(PaperSizeType.PaperA3);
workbook = new Workbook(ms, opts);

// Αποθήκευση ως PDF με ρύθμιση A3
workbook.Save(outputDir + "outputLoadWorkbookWithPrinterSize-A3.pdf");
```
**Συμβουλές αντιμετώπισης προβλημάτων:**
- Εξασφαλίζω `ms.Position` επαναφέρεται στο 0 πριν από την επαναφόρτωση της ροής.
- Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές κατά την αποθήκευση αρχείων.
