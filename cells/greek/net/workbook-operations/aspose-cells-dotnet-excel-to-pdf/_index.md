## Οδηγός Εφαρμογής
### Φόρτωση και αποθήκευση βιβλίου εργασίας ως PDF
**Επισκόπηση:**
Θα φορτώσουμε ένα υπάρχον αρχείο Excel από έναν κατάλογο προέλευσης, θα το επεξεργαστούμε εάν χρειάζεται και θα το αποθηκεύσουμε ως PDF σε έναν κατάλογο εξόδου. Αυτή η λειτουργία είναι κρίσιμη για εφαρμογές που απαιτούν δημιουργία αναφορών ή διανομή δεδομένων σε καθολική μορφή όπως το PDF.

#### Βήμα 1: Ορισμός διαδρομών αρχείων
Η χρήση placeholders εξασφαλίζει ευελιξία στον καθορισμό καταλόγων κατά τη διάρκεια του χρόνου εκτέλεσης:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

// Κατασκευή πλήρων διαδρομών
string sourceFilePath = SourceDir + "sampleRenderCustomDateFormat.xlsx";
string outputFilePath = OutputDir + "outputRenderCustomDateFormat.pdf";
```

#### Βήμα 2: Φόρτωση του αρχείου Excel
Φορτώστε το βιβλίο εργασίας σας χρησιμοποιώντας το `Workbook` κλάση, η οποία αντιπροσωπεύει ένα ολόκληρο αρχείο Excel:

```csharp
// Φόρτωση του αρχείου Excel
Workbook workbook = new Workbook(sourceFilePath);
```

Αυτό το βήμα ξεκινά τη διαδικασία μετατροπής φορτώνοντας τα δεδομένα που θα μετατρέψουμε σε PDF.

#### Βήμα 3: Αποθήκευση ως PDF
Μετατρέψτε και αποθηκεύστε το αρχείο Excel που έχετε φορτώσει σε PDF χρησιμοποιώντας το `Save` μέθοδος, καθορίζοντας ρητά τη μορφή:

```csharp
// Αποθηκεύστε το ως PDF στον κατάλογο εξόδου
workbook.Save(outputFilePath, SaveFormat.Pdf);
```

Αυτό το βήμα ολοκληρώνει τη μετατροπή, αποθηκεύοντας το αποτέλεσμα στην καθορισμένη τοποθεσία σας.

### Χειρισμός διαδρομών αρχείων
**Επισκόπηση:**
Η δυναμική διαχείριση διαδρομών αρχείων επιτρέπει την προσαρμογή κώδικα σε μεγαλύτερο βαθμό. Κατασκευάζοντας διαδρομές κατά τον χρόνο εκτέλεσης, μπορείτε να κάνετε τις εφαρμογές σας πιο ευέλικτες και επαναχρησιμοποιήσιμες σε διαφορετικά περιβάλλοντα ή σύνολα δεδομένων.

#### Κατασκευή Δυναμικών Διαδρομών
Η χρήση placeholders για καταλόγους ενισχύει την ευελιξία:

```csharp
string SourceDirPlaceholder = @"YOUR_SOURCE_DIRECTORY";
string OutputDirPlaceholder = @"YOUR_OUTPUT_DIRECTORY";

// Παράδειγμα χρήσης στην κατασκευή μονοπατιού
string sourceFilePath = SourceDirPlaceholder + "yourFile.xlsx";
string outputFilePath = OutputDirPlaceholder + "outputFile.pdf";
```

Αυτή η προσέγγιση διασφαλίζει ότι η εφαρμογή σας μπορεί να προσαρμοστεί στις αλλαγές στις τοποθεσίες αποθήκευσης αρχείων χωρίς να απαιτούνται τροποποιήσεις κώδικα.
