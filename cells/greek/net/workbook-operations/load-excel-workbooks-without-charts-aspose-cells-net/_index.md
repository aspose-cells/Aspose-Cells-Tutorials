## Οδηγός Εφαρμογής

### Δυνατότητα: Φόρτωση βιβλίου εργασίας με συγκεκριμένες επιλογές

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα βιβλίο εργασίας του Excel εξαιρώντας δεδομένα γραφήματος, βελτιστοποιώντας έτσι τη διαδικασία φόρτωσης.

#### Βήμα 1: Ορισμός καταλόγων προέλευσης και εξόδου

Ξεκινήστε καθορίζοντας τους καταλόγους σας για τα αρχεία προέλευσης και την έξοδο:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών φόρτωσης

Δημιουργήστε μια παρουσία του `LoadOptions` και ορίστε ένα φίλτρο για να εξαιρέσετε δεδομένα γραφήματος χρησιμοποιώντας λειτουργίες bitwise:

```csharp
LoadOptions options = new LoadOptions();
options.LoadFilter = new LoadFilter(LoadDataFilterOptions.All & ~LoadDataFilterOptions.Chart);
```

- **Γιατί;** Αυτή η διαμόρφωση διασφαλίζει ότι φορτώνονται μόνο τα απαραίτητα δεδομένα (εκτός από τα γραφήματα), μειώνοντας τη χρήση μνήμης και τον χρόνο φόρτωσης.

#### Βήμα 3: Φόρτωση του βιβλίου εργασίας

Χρησιμοποιήστε τις καθορισμένες επιλογές για να φορτώσετε το βιβλίο εργασίας σας:

```csharp
Workbook workbook = new Workbook(SourceDir + "sampleLoadTemplateWithoutCharts.xlsx", options);
```

- **Τι συμβαίνει;** Το βιβλίο εργασίας ανοίγει με συγκεκριμένους περιορισμούς, αγνοώντας τυχόν δεδομένα γραφήματος που είναι ενσωματωμένα σε αυτό.

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας

Μετά τη φόρτωση, αποθηκεύστε το βιβλίο εργασίας στην επιθυμητή μορφή, όπως PDF:

```csharp
workbook.Save(OutputDir + "outputLoadTemplateWithoutCharts.pdf", SaveFormat.Pdf);
```

- **Οφελος:** Αυτό το βήμα διασφαλίζει ότι μπορείτε εύκολα να κοινοποιείτε ή να διανέμετε δεδομένα χωρίς περιττές πληροφορίες γραφήματος.

### Συμβουλές αντιμετώπισης προβλημάτων
