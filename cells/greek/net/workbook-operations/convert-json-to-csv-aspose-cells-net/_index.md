## Οδηγός Εφαρμογής

### Επισκόπηση της λειτουργίας μετατροπής

Η μετατροπή JSON σε CSV χρησιμοποιώντας το Aspose.Cells περιλαμβάνει την ανάγνωση ενός αρχείου JSON και την εισαγωγή των δεδομένων του σε ένα βιβλίο εργασίας του Excel πριν από την αποθήκευσή του ως CSV. Αυτή η διαδικασία διασφαλίζει ότι η ιεραρχική δομή του JSON διατηρείται σε επίπεδη μορφή πίνακα.

#### Βήμα 1: Ανάγνωση αρχείου JSON

```csharp
// Ο κατάλογος προέλευσης όπου βρίσκεται το αρχείο JSON σας
string sourceDir = RunExamples.Get_SourceDirectory();
string jsonFilePath = sourceDir + "SampleJson.json";

// Ανάγνωση του περιεχομένου του αρχείου JSON
string jsonString = File.ReadAllText(jsonFilePath);
```

Εδώ, `File.ReadAllText` διαβάζει ολόκληρο το περιεχόμενο JSON σε μια συμβολοσειρά. Αυτό είναι το πρώτο μας βήμα προς τη μετατροπή.

#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων βιβλίου εργασίας

```csharp
// Αρχικοποίηση ενός κενού βιβλίου εργασίας
Workbook workbook = new Workbook();

// Πρόσβαση στη συλλογή κελιών του πρώτου φύλλου εργασίας
Cells cells = workbook.Worksheets[0].Cells;

// Ρύθμιση παραμέτρων JsonLayoutOptions για ρυθμίσεις εισαγωγής
JsonLayoutOptions options = new JsonLayoutOptions
{
    ConvertNumericOrDate = true,
    ArrayAsTable = true,
    IgnoreArrayTitle = true,
    IgnoreObjectTitle = true
};
```

Ο `JsonLayoutOptions` Η κλάση παρέχει διάφορες ρυθμίσεις για την προσαρμογή της διαδικασίας μετατροπής. Για παράδειγμα, `ConvertNumericOrDate` διασφαλίζει ότι οι αριθμητικές τιμές και οι τιμές ημερομηνίας ερμηνεύονται σωστά.

#### Βήμα 3: Εισαγωγή δεδομένων JSON

```csharp
// Εισαγωγή δεδομένων από συμβολοσειρά JSON σε κελιά βιβλίου εργασίας ξεκινώντας από τη γραμμή 0, τη στήλη 0
JsonUtility.ImportData(jsonString, cells, 0, 0, options);
```

`JsonUtility.ImportData` Η μέθοδος εισάγει τα δεδομένα JSON στο καθορισμένο φύλλο εργασίας και στην καθορισμένη περιοχή κελιών χρησιμοποιώντας τη ρύθμιση παραμέτρων που παρέχεται.

#### Βήμα 4: Αποθήκευση ως CSV

```csharp
// Ορίστε τον κατάλογο εξόδου για την αποθήκευση του αρχείου CSV
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "SampleJson_out.csv");
```

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε μορφή CSV. `Save` Η μέθοδος είναι ευέλικτη και υποστηρίζει διάφορες μορφές, συμπεριλαμβανομένου του CSV.
