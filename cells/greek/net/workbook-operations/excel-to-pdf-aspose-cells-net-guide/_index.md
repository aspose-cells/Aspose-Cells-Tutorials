## Οδηγός Εφαρμογής
Θα καλύψουμε τη δημιουργία βιβλίου εργασίας και τη μετατροπή PDF με διαχείριση διακοπών.

### Δημιουργία βιβλίων εργασίας Excel και μετατροπή σε PDF
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα βιβλίο εργασίας, να το τροποποιήσετε προσθέτοντας κείμενο και να το μετατρέψετε σε PDF.

#### Βήμα 1: Αρχικοποίηση στοιχείων
Ρύθμιση καταλόγων:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε ένα αντικείμενο InterruptMonitor για τον χειρισμό διακοπών
InterruptMonitor im = new InterruptMonitor();
```

#### Βήμα 2: Δημιουργία και τροποποίηση βιβλίου εργασίας
Δημιουργήστε μια παρουσία βιβλίου εργασίας, αντιστοιχίστε το InterruptMonitor και τροποποιήστε ένα κελί:
```csharp
Workbook wb = new Workbook();
wb.InterruptMonitor = im;

Worksheet ws = wb.Worksheets[0];
Cell cell = ws.Cells["J1000000"];
cell.PutValue("This is text.");
```

#### Βήμα 3: Μετατροπή σε PDF
Δοκιμάστε να αποθηκεύσετε το βιβλίο εργασίας ως PDF και να διαχειριστείτε τις διακοπές:
```csharp
try {
    wb.Save(outputDir + "/output_InterruptMonitor.pdf");
} catch (Aspose.Cells.CellsException ex) {
    Console.WriteLine("Process Interrupted - Message: " + ex.Message);
}
```

### Διαχείριση διακοπών διεργασιών χρησιμοποιώντας νήματα
Αυτή η λειτουργία καταδεικνύει τη διακοπή μιας διεργασίας χρησιμοποιώντας νήματα.

#### Βήμα 1: Ορισμός της λογικής διακοπής
Δημιουργήστε μια μέθοδο αναμονής πριν από τη διακοπή:
```csharp
void WaitForWhileAndThenInterrupt() {
    // Ύπνος για 10 δευτερόλεπτα (1000 χιλιοστά του δευτερολέπτου * 10)
    Thread.Sleep(1000 * 10);
    
    // Διακόψτε τη διαδικασία μετά από 10 δευτερόλεπτα
    im.Interrupt();
}
```

#### Βήμα 2: Ρύθμιση νήματος
Χρησιμοποιήστε νήματα για να διαχειριστείτε τη δημιουργία βιβλίων εργασίας και τις διακοπές:
```csharp
InterruptMonitor im = new InterruptMonitor();

ThreadStart ts1 = new ThreadStart(() => {
    Workbook wb = new Workbook();
    wb.InterruptMonitor = im;
    Worksheet ws = wb.Worksheets[0];
    Cell cell = ws.Cells["J1000000"];
    cell.PutValue("This is text.");
    
    try {
        wb.Save(outputDir + "/output_InterruptMonitor.pdf");
    } catch (Aspose.Cells.CellsException ex) {
        Console.WriteLine("Process Interrupted - Message: " + ex.Message);
    }
});

ThreadStart ts2 = new ThreadStart(WaitForWhileAndThenInterrupt);

Thread t1 = new Thread(ts1);
Thread t2 = new Thread(ts2);
t1.Start();
t2.Start();
t1.Join();
t2.Join();
```
