## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και ρύθμιση παραμέτρων ενός βιβλίου εργασίας
Αυτή η λειτουργία περιλαμβάνει τη δημιουργία ενός βιβλίου εργασίας του Excel, την προσθήκη δεδομένων σε αυτό και την αποθήκευση του αρχείου.

#### Επισκόπηση
Θα ξεκινήσουμε δημιουργώντας ένα νέο βιβλίο εργασίας και συμπληρώνοντάς το με κεφαλίδα και δεδομένα υπαλλήλου.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1: Αρχικοποίηση βιβλίου εργασίας**
Δημιουργήστε μια νέα παρουσία του `Workbook`.

```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργία νέας παρουσίας βιβλίου εργασίας
Workbook workbook = new Workbook();
```

**Βήμα 2: Πρόσβαση και συμπλήρωση κελιών φύλλου εργασίας**
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και συμπληρώστε το με κεφαλίδες.

```csharp
Worksheet sheet = workbook.Worksheets[0];
Cells cells = sheet.Cells;

// Ορισμός γραμμής κεφαλίδας
string[] headers = { "Employee", "Quarter", "Product", "Continent", "Country", "Sale" };
for (int i = 0; i < headers.Length; i++)
{
    // Ορισμός τιμής για κάθε κελί κεφαλίδας στην πρώτη γραμμή
    cells["A1"].Offset[0, i].PutValue(headers[i]);
}
```

**Βήμα 3: Προσθήκη γραμμών δεδομένων**
Συμπληρώστε τις γραμμές δεδομένων με πληροφορίες υπαλλήλων.

```csharp
string[,] employeeData = {
    { "David", "China", "Asia", "2000" },
    // ...επιπλέον δεδομένα...
};

for (int i = 0; i < employeeData.GetLength(0); i++)
{
    for (int j = 0; j < employeeData.GetLength(1); j++)
    {
        cells["A" + (i + 2)].Offset[0, j].PutValue(employeeData[i, j]);
    }
}
```

**Βήμα 4: Ρύθμιση παραμέτρων αντικειμένου λίστας**
Δημιουργήστε και διαμορφώστε έναν πίνακα μέσα στο φύλλο εργασίας.

```csharp
Aspose.Cells.Tables.ListObject listObject = sheet.ListObjects[sheet.ListObjects.Add("A1", "F" + (employeeData.GetLength(0) + 1), true)];
listObject.TableStyleType = Aspose.Cells.Tables.TableStyleType.TableStyleMedium10;
listObject.ShowTotals = true;

// Ορισμός υπολογισμού συνόλων για τη στήλη «Τρίμηνο»
listObject.ListColumns[1].TotalsCalculation = Aspose.Cells.Tables.TotalsCalculation.Count;
```

**Βήμα 5: Αποθήκευση βιβλίου εργασίας**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε έναν καθορισμένο κατάλογο.

```csharp
workbook.Save(Path.Combine(outputDir, "output.xlsx"));
```

### Λειτουργία 2: Προσθήκη δεδομένων και διαμόρφωση στυλ πίνακα
Αυτή η ενότητα βελτιώνει την προηγούμενη λειτουργία εφαρμόζοντας συγκεκριμένα στυλ για βελτιωμένη αισθητική.

#### Επισκόπηση
Όπως και με την πρώτη λειτουργία, θα συμπληρώσουμε τα κελιά αλλά με πρόσθετες διαμορφώσεις στυλ για μια πιο κομψή εμφάνιση.

#### Βήμα προς βήμα εφαρμογή
**Βήματα 1-4**
Τα βήματα είναι παρόμοια με τη ρύθμιση της Λειτουργίας 1. Εστιάστε στη διαμόρφωση `TableStyleType` και `ShowTotals`.

```csharp
// Προσθήκη αντικειμένου λίστας (πίνακας) με στυλ
Aspose.Cells.Tables.ListObject listObject = sheet.ListObjects.Add("A1", "F" + (employeeData.GetLength(0) + 1), true);
listObject.TableStyleType = Aspose.Cells.Tables.TableStyleType.TableStyleMedium10;
listObject.ShowTotals = true;

// Ρύθμιση παραμέτρων στήλης «Τρίμηνο» για σύνολα
table.ListColumns[1].TotalsCalculation = Aspose.Cells.Tables.TotalsCalculation.Count;
```

**Βήμα 5: Αποθήκευση βιβλίου εργασίας**
Όπως και πριν, αποθηκεύστε το βιβλίο εργασίας.

```csharp
workbook.Save(Path.Combine(outputDir, "styled_output.xlsx"));
```
