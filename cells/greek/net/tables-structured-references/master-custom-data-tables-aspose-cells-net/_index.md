Είτε είστε προγραμματιστής που εργάζεται σε εργαλεία επιχειρηματικής ευφυΐας είτε δημιουργεί οικονομικά μοντέλα, η εκμάθηση του τρόπου χειρισμού αρχείων Excel μέσω προγραμματισμού μπορεί να βελτιώσει σημαντικά την παραγωγικότητα. Αυτό το σεμινάριο θα σας καθοδηγήσει στην υλοποίηση προσαρμοσμένων πινάκων δεδομένων χρησιμοποιώντας το Aspose.Cells για .NET, δίνοντάς σας τη δυνατότητα να ενσωματώσετε απρόσκοπτα αυτήν τη λειτουργικότητα στα έργα σας.

## Οδηγός Εφαρμογής

Θα εφαρμόσουμε δύο βασικές λειτουργίες: τη δημιουργία ενός προσαρμοσμένου πίνακα δεδομένων και την εισαγωγή του σε ένα βιβλίο εργασίας του Excel με συγκεκριμένες επιλογές.

### Χαρακτηριστικό 1: Υλοποίηση προσαρμοσμένου πίνακα δεδομένων

Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε έναν προσαρμοσμένο πίνακα δεδομένων εφαρμόζοντας το `ICellsDataTable` διεπαφή.

#### Επισκόπηση

Ο `ICellsDataTable` Η διεπαφή σάς επιτρέπει να παρέχετε προσαρμοσμένα δεδομένα για λειτουργίες εισαγωγής. Θα ορίσουμε μια κλάση που υλοποιεί αυτήν τη διεπαφή, επιτρέποντάς μας να διαχειριζόμαστε δυναμικά πίνακες δεδομένων.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός ονομάτων δεδομένων και στηλών**

Ξεκινήστε ορίζοντας τα ονόματα του πίνακα δεδομένων και των στηλών:

```csharp
string[][] colsData = new string[][
{
    new string[] { "Dog", "Cat", "Duck" },
    new string[] { "Apple", "Pear", "Banana" },
    new string[] { "UK", "USA", "China" },
    new string[] { "Red", "Green", "Blue" }
};

string[] colsNames = new string[] { "Pet", "Fruit", "Country", "Color" };
```

**2. Υλοποιήστε το `ICellsDataTable` Διεπαφή**

Δημιουργήστε μια κλάση που υλοποιεί αυτήν τη διεπαφή για τη διαχείριση των προσαρμοσμένων δεδομένων σας:

```csharp
class CellsDataTable : ICellsDataTable
{
    int m_index = -1;

    // Επιστρέφει ονόματα στηλών
    string[] ICellsDataTable.Columns => colsNames;

    // Επιστρέφει τον αριθμό των στοιχείων (γραμμές)
    int ICellsDataTable.Count => colsData[0].Length;

    // Επαναφέρει το ευρετήριο πριν ξεκινήσει η επανάληψη
    void ICellsDataTable.BeforeFirst() => m_index = -1;

    // Προχωρά στην επόμενη σειρά
    bool ICellsDataTable.Next()
    {
        m_index++;
        return true;
    }

    // Ανακτά δεδομένα από μια συγκεκριμένη στήλη στο τρέχον ευρετήριο
    object ICellsDataTable.this[int columnIndex] => colsData[columnIndex][m_index];
}
```

### Δυνατότητα 2: Εισαγωγή δεδομένων βιβλίου εργασίας με προσαρμοσμένες επιλογές

Αυτή η ενότητα εστιάζει στην εισαγωγή προσαρμοσμένων πινάκων δεδομένων σε ένα βιβλίο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells και στη ρύθμιση παραμέτρων επιλογών όπως η μετατόπιση γραμμών.

#### Επισκόπηση

Θα μάθετε πώς να εισάγετε δεδομένα χωρίς να διαταράξετε το υπάρχον περιεχόμενο, ελέγχοντας τις μετατοπίσεις γραμμών κατά τη διάρκεια της διαδικασίας εισαγωγής.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργήστε μια παρουσία βιβλίου εργασίας**

Φόρτωση ενός υπάρχοντος βιβλίου εργασίας ή δημιουργία ενός νέου:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook(SourceDir + "/sampleImportTableOptionsShiftFirstRowDown.xlsx");
Worksheet ws = wb.Worksheets[0];
```

**2. Διαμόρφωση επιλογών εισαγωγής**

Ορίστε επιλογές για τον έλεγχο της συμπεριφοράς εισαγωγής, όπως εάν θα μετατοπιστούν οι υπάρχουσες γραμμές:

```csharp
ImportTableOptions opts = new ImportTableOptions { ShiftFirstRowDown = false };
```

**3. Εισαγωγή προσαρμοσμένου πίνακα δεδομένων**

Χρησιμοποιήστε την κλάση προσαρμοσμένου πίνακα δεδομένων και τις καθορισμένες επιλογές για την εισαγωγή δεδομένων ξεκινώντας από ένα συγκεκριμένο κελί:

```csharp
CellsDataTable cellsDataTable = new CellsDataTable();
ws.Cells.ImportData(cellsDataTable, 1, 1, opts);
```

**4. Αποθήκευση του βιβλίου εργασίας**

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τις τροποποιήσεις:

```csharp
wb.Save(OutputDir + "/outputImportTableOptionsShiftFirstRowDown-False.xlsx");
```
