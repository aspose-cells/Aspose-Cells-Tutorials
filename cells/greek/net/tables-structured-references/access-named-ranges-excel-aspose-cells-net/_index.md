### Πρόσβαση σε μια ονομασμένη περιοχή στο Excel
**Επισκόπηση:** Θα φορτώσουμε ένα αρχείο Excel και θα ανακτήσουμε ένα καθορισμένο ονομαστικό εύρος που ονομάζεται "MyRangeTwo".
1. **Φόρτωση του βιβλίου εργασίας**
   Ξεκινήστε φορτώνοντας το βιβλίο εργασίας του Excel χρησιμοποιώντας `Workbook`:
   ```csharp
   string sourceDir = RunExamples.Get_SourceDirectory();
   Workbook workbook = new Workbook(sourceDir + "sampleAccessSpecificNamedRange.xlsx");
   ```
2. **Ανάκτηση του ονομασμένου εύρους**
   Χρήση `GetRangeByName()` για να αποκτήσετε πρόσβαση στο ονομασμένο εύρος:
   ```csharp
   Range range = workbook.Worksheets.GetRangeByName("MyRangeTwo");

   if (range != null)
       Console.WriteLine("Named Range: " + range.RefersTo);
   ```
3. **Επιβεβαίωση εξόδου**
   Επιβεβαιώστε την επιτυχή εκτέλεση με ένα μήνυμα κονσόλας:
   ```csharp
   Console.WriteLine("AccessSpecificNamedRange executed successfully.");
   ```

**Παράμετροι και Σκοπός:**
- `GetRangeByName(string name)`: Ανακτά το ονομασμένο εύρος με βάση το αναγνωριστικό του, επιστρέφοντας `null` αν δεν βρεθεί.
- `RefersTo`Παρέχει μια αναπαράσταση συμβολοσειράς της αναφοράς εύρους στο Excel.
