## Οδηγός υλοποίησης: Ορισμός σειράς σελίδων στο Excel με το Aspose.Cells .NET

### Εισαγωγή στη διαμόρφωση της σελίδας

Η ρύθμιση της σειράς σελίδων είναι ζωτικής σημασίας για συγκεκριμένες διατάξεις εκτύπωσης, όπως η εκτύπωση σε πολλές σελίδες ή ο ορισμός προσαρμοσμένων ακολουθιών. Αυτή η ενότητα δείχνει πώς να ορίσετε τη σειρά σελίδων σε "Πάνω και Κάτω".

#### Βήμα 1: Δημιουργία και ρύθμιση παραμέτρων βιβλίου εργασίας

```csharp
using Aspose.Cells;
using System;

namespace PageOrderExample
{
    public class SetPageOrder
    {
        public static void Run()
        {
            // Ορίστε τον κατάλογο για τα έγγραφα
            string dataDir = "YourDataDirectoryPathHere"; // Ενημέρωση αυτής της διαδρομής

            // Δημιουργία νέου αντικειμένου βιβλίου εργασίας
            Workbook workbook = new Workbook();

            // Πρόσβαση στο PageSetup του πρώτου φύλλου εργασίας
            PageSetup pageSetup = workbook.Worksheets[0].PageSetup;
            
            // Ορίστε τη σειρά εκτύπωσης σε Πάνω και Κάτω
            pageSetup.Order = PrintOrderType.OverThenDown;

            // Αποθήκευση του τροποποιημένου βιβλίου εργασίας
            workbook.Save(dataDir + "SetPageOrder_out.xls");
        }
    }
}
```

#### Επεξήγηση βασικών στοιχείων
- **Αρχικοποίηση βιβλίου εργασίας**: Αντιπροσωπεύει το αρχείο Excel σας.
- **Πρόσβαση σε Ρύθμιση Σελίδας**: Χρησιμοποιείται για την τροποποίηση των ρυθμίσεων εκτύπωσης σε επίπεδο φύλλου εργασίας.
- **Διαμόρφωση παραγγελίας εκτύπωσης**: `PrintOrderType.OverThenDown` Καθορίζει ότι οι σελίδες θα εκτυπωθούν πάνω και μετά προς τα κάτω κατά μήκος των φύλλων.
