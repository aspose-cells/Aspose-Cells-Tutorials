## Οδηγός Εφαρμογής

### Υπολογισμός συντελεστή κλιμάκωσης διαμόρφωσης σελίδας

Αυτή η λειτουργία σάς βοηθά να προσδιορίσετε πόσο κλιμακώνεται το περιεχόμενο ενός φύλλου εργασίας ώστε να ταιριάζει στη σελίδα κατά την εκτύπωση.

#### Βήμα 1: Πρόσβαση και τροποποίηση ιδιοτήτων φύλλου εργασίας

Αρχικά, αποκτήστε πρόσβαση στο φύλλο εργασίας που επιθυμείτε και κάντε τις απαραίτητες προσαρμογές:
```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet worksheet = workbook.Worksheets[0];

// Τοποθετήστε ορισμένα δεδομένα σε συγκεκριμένα κελιά για επίδειξη
worksheet.Cells["A4"].PutValue("Test");
worksheet.Cells["S4"].PutValue("Test");

// Ορισμός μεγέθους χαρτιού σε A4
worksheet.PageSetup.PaperSize = PaperSizeType.PaperA4;

// Ρύθμιση παραμέτρων φύλλου εργασίας ώστε να χωράει το περιεχόμενο σε μία σελίδα
worksheet.PageSetup.FitToPagesWide = 1;
```

#### Βήμα 2: Δημιουργία αντικειμένου SheetRender

Χρησιμοποιήστε το `SheetRender` κλάση για τη διαχείριση των ρυθμίσεων απόδοσης:
```csharp
// Αρχικοποίηση SheetRender με τις προεπιλεγμένες επιλογές εκτύπωσης
SheetRender sr = new SheetRender(worksheet, new ImageOrPrintOptions());
```

#### Βήμα 3: Υπολογισμός και εμφάνιση συντελεστή κλιμάκωσης

Μετατρέψτε τον συντελεστή κλιμάκωσης από διπλή τιμή σε ποσοστιαία μορφή για εύκολη ερμηνεία:
```csharp
// Μετατροπή κλίμακας σελίδας σε αναγνώσιμη συμβολοσειρά ποσοστού
string strPageScale = sr.PageScale.ToString("0%");
Console.WriteLine($"Scaling Factor: {strPageScale}");
```

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι όλες οι διαδρομές (`SourceDir`, `outputDir`) έχουν ρυθμιστεί σωστά.
- Εάν η κλιμάκωση δεν είναι η αναμενόμενη, ελέγξτε ξανά `FitToPagesWide` και άλλες διαμορφώσεις ρύθμισης σελίδας.
