## Οδηγός Εφαρμογής
Ας εξερευνήσουμε τη διαδικασία ορισμού μεγεθών χαρτιού για διάφορες μορφές.

### Ρύθμιση μεγέθους χαρτιού σε A2
#### Επισκόπηση
Διαμορφώστε ένα φύλλο εργασίας του Excel ώστε να χρησιμοποιεί μέγεθος χαρτιού A2, κατάλληλο για μεγάλες εκτυπώσεις και αφίσες.

#### Βήματα
**1. Δημιουργήστε μια νέα παρουσία βιβλίου εργασίας**
```csharp
Workbook wb = new Workbook();
```

**2. Πρόσβαση στο Πρώτο Φύλλο Εργασίας**
```csharp
Worksheet ws = wb.Worksheets[0];
```

**3. Ορίστε το μέγεθος χαρτιού σε A2**
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA2;
```

**4. Διαστάσεις οθόνης σε ίντσες**
```csharp
Console.WriteLine("PaperA2: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
*Εξήγηση*: Το `PageSetup.PaperSize` η ιδιότητα προσαρμόζει το μέγεθος χαρτιού, ενώ `PaperWidth` και `PaperHeight` δώστε διαστάσεις.
