## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στον ορισμό και την ανάκτηση διαστάσεων σελίδας χρησιμοποιώντας το Aspose.Cells για .NET.

### Ρύθμιση διαστάσεων σελίδας

Η ρύθμιση των μεγεθών χαρτιού είναι απαραίτητη κατά την προετοιμασία εγγράφων για έντυπη ή ψηφιακή διανομή. Ας εξερευνήσουμε αυτήν τη λειτουργία:

#### Βήμα 1: Πρόσβαση στο Φύλλο Εργασίας
Αποκτήστε πρόσβαση στο φύλλο εργασίας όπου θέλετε να αλλάξετε τη διαμόρφωση σελίδας:
```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet sheet = book.Worksheets[0];
```

#### Βήμα 2: Ρύθμιση μεγέθους χαρτιού
Μπορείτε να ορίσετε διαφορετικά μεγέθη χαρτιού τροποποιώντας το `PaperSize` ιδιοκτησία:

- **Ορισμός μεγέθους χαρτιού σε A2**
    ```csharp
    // Ορίστε το μέγεθος χαρτιού σε A2 και εκτυπώστε το πλάτος και το ύψος του χαρτιού σε ίντσες
    sheet.PageSetup.PaperSize = PaperSizeType.PaperA2;
    Console.WriteLine("PaperA2: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

- **Ορισμός μεγέθους χαρτιού σε A3**
    ```csharp
    // Ορίστε το μέγεθος χαρτιού σε A3 και εκτυπώστε το πλάτος και το ύψος του χαρτιού σε ίντσες
    sheet.PageSetup.PaperSize = PaperSizeType.PaperA3;
    Console.WriteLine("PaperA3: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

- **Ορισμός μεγέθους χαρτιού σε A4**
    ```csharp
    // Ορίστε το μέγεθος χαρτιού σε A4 και εκτυπώστε το πλάτος και το ύψος του χαρτιού σε ίντσες
    sheet.PageSetup.PaperSize = PaperSizeType.PaperA4;
    Console.WriteLine("PaperA4: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

- **Ορισμός μεγέθους χαρτιού σε Letter**
    ```csharp
    // Ορίστε το μέγεθος χαρτιού σε Letter και εκτυπώστε το πλάτος και το ύψος του χαρτιού σε ίντσες
    sheet.PageSetup.PaperSize = PaperSizeType.PaperLetter;
    Console.WriteLine("PaperLetter: " + sheet.PageSetup.PaperWidth + "x" + sheet.PageSetup.PaperHeight);
    ```

### Ανάκτηση διαστάσεων σελίδας
