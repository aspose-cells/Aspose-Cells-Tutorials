## Οδηγός Εφαρμογής

### Εκκαθάριση αλλαγών σελίδας: Μια επισκόπηση
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη διαγραφή όλων των οριζόντιων και κάθετων αλλαγών σελίδας από ένα φύλλο εργασίας του Excel. Αυτό είναι κρίσιμο για την προετοιμασία εγγράφων χωρίς διακοπές στη διάταξη.

#### Βήμα 1: Ορισμός διαδρομών καταλόγου
Ξεκινήστε ορίζοντας τις διαδρομές καταλόγου προέλευσης και εξόδου:
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Δημιουργία αντικειμένου βιβλίου εργασίας
Δημιουργήστε ένα υπόδειγμα `Workbook` αντικείμενο για την αναπαράσταση του αρχείου Excel σας:
```csharp
// Δημιουργία αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```

#### Βήμα 3: Εκκαθάριση οριζόντιων αλλαγών σελίδας
Διαγράψτε τις οριζόντιες αλλαγές σελίδας από το πρώτο φύλλο εργασίας χρησιμοποιώντας αυτήν τη μέθοδο:
```csharp
// Διαγραφή όλων των οριζόντιων αλλαγών σελίδας στο πρώτο φύλλο εργασίας
workbook.Worksheets[0].HorizontalPageBreaks.Clear();
```
**Εξήγηση:** Αυτό καταργεί όλες τις οριζόντιες αλλαγές σελίδας, εξασφαλίζοντας μια απρόσκοπτη διάταξη σε όλες τις σελίδες.

#### Βήμα 4: Εκκαθάριση κάθετων αλλαγών σελίδας
Διαγραφή κάθετων αλλαγών σελίδας με παρόμοιο τρόπο:
```csharp
// Διαγραφή όλων των κάθετων αλλαγών σελίδας στο πρώτο φύλλο εργασίας
workbook.Worksheets[0].VerticalPageBreaks.Clear();
```
**Εξήγηση:** Αυτό διασφαλίζει ότι δεν θα υπάρχουν κάθετες διακοπές, διατηρώντας μια συνεπή ροή δεδομένων.

#### Βήμα 5: Αποθήκευση του αρχείου Excel
Αποθηκεύστε τις αλλαγές σας στον καθορισμένο κατάλογο εξόδου:
```csharp
// Αποθηκεύστε το αρχείο Excel στον καθορισμένο κατάλογο εξόδου
workbook.Save(OutputDir + "/ClearAllPageBreaks_out.xls");
```
**Συμβουλές αντιμετώπισης προβλημάτων:** Εάν προκύψουν προβλήματα αποθήκευσης, βεβαιωθείτε ότι `OutputDir` είναι σωστό και προσβάσιμο.
