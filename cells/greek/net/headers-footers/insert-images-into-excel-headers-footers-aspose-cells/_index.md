## Οδηγός Εφαρμογής

### Επισκόπηση της λειτουργίας

Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε εικόνες όπως λογότυπα στις κεφαλίδες ή τα υποσέλιδα ενός φύλλου εργασίας του Excel. Είναι ιδιαίτερα χρήσιμη για σκοπούς εμπορικής προώθησης σε όλα τα φύλλα ενός βιβλίου εργασίας.

#### Βήμα 1: Ρύθμιση του έργου και του χώρου ονομάτων σας

Αρχικά, συμπεριλάβετε τα απαραίτητα namespaces στο αρχείο σας:

```csharp
using System.IO;
using Aspose.Cells;
```

#### Βήμα 2: Δημιουργία βιβλίου εργασίας και φόρτωση καταλόγου δεδομένων

Ξεκινήστε δημιουργώντας μια παρουσία του `Workbook` κλάση. Στη συνέχεια, καθορίστε τον κατάλογο δεδομένων όπου αποθηκεύονται οι εικόνες σας.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Δημιουργία αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```

#### Βήμα 3: Ανάγνωση δεδομένων εικόνας

Για να εισαγάγετε μια εικόνα, πρέπει να την διαβάσετε σε έναν πίνακα byte. Χρησιμοποιήστε `FileStream` για την πρόσβαση στο αρχείο.

```csharp
string logo_url = dataDir + "aspose-logo.jpg";
using (FileStream inFile = new FileStream(logo_url, FileMode.Open, FileAccess.Read))
{
    // Δημιουργία στιγμιαίας εικόνας του πίνακα byte του μεγέθους του αντικειμένου FileStream
    byte[] binaryData = new Byte[inFile.Length];
    
    // Διαβάζει ένα μπλοκ bytes από τη ροή σε έναν πίνακα.
    long bytesRead = inFile.Read(binaryData, 0, (int)inFile.Length);
```

#### Βήμα 4: Ρύθμιση παραμέτρων σελίδας και εισαγωγή εικόνας

Πρόσβαση στο `PageSetup` αντικείμενο για να καθορίσετε πού θα πρέπει να εμφανίζεται η εικόνα στην κεφαλίδα.

```csharp
// Λήψη των ρυθμίσεων διαμόρφωσης σελίδας του πρώτου φύλλου εργασίας
PageSetup pageSetup = workbook.Worksheets[0].PageSetup;

// Ορισμός του λογότυπου/εικόνας στο κεντρικό τμήμα της κεφαλίδας σελίδας
pageSetup.SetHeaderPicture(1, binaryData);
```

#### Βήμα 5: Ορισμός σεναρίων κεφαλίδας

Ρυθμίστε σενάρια για να αυτοματοποιήσετε τμήματα των κεφαλίδων σας, όπως ημερομηνία, όνομα φύλλου κ.λπ.

```csharp
// Ρύθμιση παραμέτρων κεφαλίδας με εικόνα και άλλα στοιχεία
pageSetup.SetHeader(1, "&G"); // Σενάριο εικόνας
pageSetup.SetHeader(2, "&A"); // Σενάριο ονόματος φύλλου
```

#### Βήμα 6: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να δείτε τις αλλαγές.

```csharp
workbook.Save(dataDir + "InsertImageInHeaderFooter_out.xls");
```

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι τα αρχεία εικόνας είναι προσβάσιμα και οι διαδρομές έχουν οριστεί σωστά.
- Επαληθεύστε ότι `SetHeaderPicture` λαμβάνει έναν πίνακα byte που δεν είναι null.
- Ελέγξτε για σωστά σύμβολα σεναρίου (`&G` για εικόνες).
