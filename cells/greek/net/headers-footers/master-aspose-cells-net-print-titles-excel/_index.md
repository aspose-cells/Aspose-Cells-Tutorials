## Οδηγός Εφαρμογής

### Ορισμός τίτλων εκτύπωσης σε φύλλα εργασίας Excel

Σε αυτήν την ενότητα, θα σας δείξουμε πώς να ορίσετε μέσω προγραμματισμού συγκεκριμένες στήλες και γραμμές ως τίτλους εκτύπωσης χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Δημιουργία νέας παρουσίας βιβλίου εργασίας

Αρχικά, αρχικοποιήστε ένα νέο βιβλίο εργασίας. Αυτό αντιπροσωπεύει ένα κενό αρχείο Excel στη μνήμη το οποίο μπορείτε να χειριστείτε:

```csharp
Workbook workbook = new Workbook();
```

#### Βήμα 2: Λήψη του αντικειμένου PageSetup του πρώτου φύλλου εργασίας

Στη συνέχεια, αποκτήστε πρόσβαση στο `PageSetup` αντικείμενο από το πρώτο σας φύλλο εργασίας για να προσαρμόσετε τις ρυθμίσεις διάταξης σελίδας.

```csharp
PageSetup pageSetup = workbook.Worksheets[0].PageSetup;
```

#### Βήμα 3: Ορισμός στηλών ως τίτλων Στήλες για εκτύπωση

Για να διασφαλίσετε ότι συγκεκριμένες στήλες επαναλαμβάνονται σε κάθε εκτυπωμένη σελίδα, χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
pageSetup.PrintTitleColumns = "$A:$B";
```
Εδώ, `$A:$B` Καθορίζει ότι οι στήλες Α και Β θα εμφανίζονται στο επάνω μέρος κάθε εκτύπωσης.

#### Βήμα 4: Ορισμός γραμμών ως γραμμών τίτλου για εκτύπωση

Ομοίως, ορίστε γραμμές που θα επαναλαμβάνονται σε κάθε σελίδα ορίζοντας:

```csharp
pageSetup.PrintTitleRows = "$1:$2";
```
Αυτή η διαμόρφωση διασφαλίζει ότι οι γραμμές 1 και 2 εκτυπώνονται στο επάνω μέρος κάθε σελίδας.

#### Βήμα 5: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τις ρυθμίσεις τίτλου εκτύπωσης που έχουν εφαρμοστεί:

```csharp
workbook.Save(outputDir + "/SetPrintTitle_out.xls");
```
