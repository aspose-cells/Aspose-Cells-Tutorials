## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα περιηγηθούμε στη διαδικασία αντιγραφής των ρυθμίσεων διαμόρφωσης σελίδας από ένα φύλλο εργασίας σε ένα άλλο.

### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να αντιγράψετε διάφορες παραμέτρους διαμόρφωσης σελίδας, όπως το μέγεθος χαρτιού και την περιοχή εκτύπωσης. Είναι ιδιαίτερα χρήσιμη κατά τη διαχείριση μεγάλων αρχείων Excel που απαιτούν ομοιόμορφη μορφοποίηση.

#### Βήμα 1: Δημιουργία βιβλίου εργασίας και προσθήκη φύλλων εργασίας (H3)
Ξεκινήστε αρχικοποιώντας ένα βιβλίο εργασίας και προσθέτοντας δύο φύλλα εργασίας:

```csharp
using Aspose.Cells;

namespace CopyPageSetupSettings
{
    public class Program
    {
        public static void Main()
        {
            // Αρχικοποίηση του βιβλίου εργασίας
            Workbook wb = new Workbook();

            // Προσθήκη δύο φύλλων εργασίας
            wb.Worksheets.Add("TestSheet1");
            wb.Worksheets.Add("TestSheet2");

            Worksheet TestSheet1 = wb.Worksheets["TestSheet1"];
            Worksheet TestSheet2 = wb.Worksheets["TestSheet2"];

            Console.WriteLine("Worksheets added successfully.");
        }
    }
}
```

#### Βήμα 2: Ορισμός διαμόρφωσης σελίδας για το φύλλο εργασίας πηγαίου κώδικα (H3)
Διαμορφώστε τις ρυθμίσεις διαμόρφωσης σελίδας για το φύλλο εργασίας προέλευσης:

```csharp
// Ρύθμιση μεγέθους χαρτιού για το TestSheet1
TestSheet1.PageSetup.PaperSize = PaperSizeType.PaperA3ExtraTransverse;

Console.WriteLine("Page setup configured for TestSheet1.");
```

#### Βήμα 3: Αντιγραφή Ρύθμισης Σελίδας από την Προέλευση στον Προορισμό (H3)
Χρησιμοποιήστε το `Copy` μέθοδος μεταφοράς ρυθμίσεων:

```csharp
// Αντιγραφή διαμόρφωσης σελίδας από το TestSheet1 στο TestSheet2
TestSheet2.PageSetup.Copy(TestSheet1.PageSetup, new CopyOptions());

Console.WriteLine("Page setup copied successfully.");
```

#### Βήμα 4: Επαλήθευση αλλαγών (H3)
Τέλος, επιβεβαιώστε ότι οι αλλαγές έχουν εφαρμοστεί σωστά:

```csharp
// Μέγεθος χαρτιού εκτύπωσης και για τα δύο φύλλα εργασίας
Console.WriteLine($"After Paper Size: {TestSheet1.PageSetup.PaperSize}");
Console.WriteLine($"After Paper Size: {TestSheet2.PageSetup.PaperSize}");
```
