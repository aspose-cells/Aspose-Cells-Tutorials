## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Προσθήκη υδατογραφήματος σε φύλλο Excel
#### Επισκόπηση
Η προσθήκη ενός υδατογραφήματος περιλαμβάνει τη δημιουργία εφέ κειμένου που επικαλύπτουν διακριτικά τα δεδομένα σας, σηματοδοτώντας την κατάσταση του εγγράφου όπως "ΕΜΠΙΣΤΕΥΤΙΚΟ".

#### Βήμα προς βήμα εφαρμογή
##### Δημιουργήστε ένα βιβλίο εργασίας και ένα φύλλο εργασίας
```csharp
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

##### Προσθήκη εφέ κειμένου ως υδατογράφημα
Δημιουργήστε το σχήμα εφέ κειμένου με συγκεκριμένα χαρακτηριστικά όπως στυλ γραμματοσειράς, μέγεθος, θέση και εμφάνιση.

```csharp
Shape wordart = sheet.Shapes.AddTextEffect(
    MsoPresetTextEffect.TextEffect1,
    "CONFIDENTIAL", 
    "Arial Black",
    50,   // Μέγεθος γραμματοσειράς
    false, // Είναι πλάγια γραφή;
    true, // Είναι έντονο
    18,   // Αριστερή θέση
    8,    // Κορυφαία θέση
    1,    // Πλάτος
    1,    // Υψος
    130,  // Γωνία περιστροφής
    800   // Συντελεστής κλίμακας
);
```

##### Προσαρμογή εμφάνισης
Ορίστε το χρώμα διαβάθμισης και τη διαφάνεια για μια γυαλιστερή εμφάνιση.
```csharp
FillFormat wordArtFormat = wordart.Fill;
wordArtFormat.SetOneColorGradient(Color.Red, 0.2, GradientStyleType.Horizontal, 2); 
wordArtFormat.Transparency = 0.9; // Κάντε το ελαφρώς διαφανές

wordart.HasLine = false; // Αφαιρέστε τη γραμμή περιγράμματος για πιο καθαρή εμφάνιση
```

##### Αποθήκευση του βιβλίου εργασίας σας
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY\output_out.xlsx");
```

### Χαρακτηριστικό 2: Κλείδωμα πτυχών σχήματος σε φύλλο Excel
#### Επισκόπηση
Το κλείδωμα σχημάτων εμποδίζει τους μη εξουσιοδοτημένους χρήστες να τροποποιήσουν το υδατογράφημα ή άλλα σχήματα, διασφαλίζοντας την ακεραιότητα του εγγράφου.

#### Βήμα προς βήμα εφαρμογή
##### Κλείδωμα διαφόρων ιδιοτήτων του υδατογραφήματος
Ασφαλίστε το υδατόσημό σας κλειδώνοντας τις όψεις του.
```csharp
wordart.IsLocked = true;
wordart.SetLockedProperty(ShapeLockType.Selection, true);
wordart.SetLockedProperty(ShapeLockType.ShapeType, true);
wordart.SetLockedProperty(ShapeLockType.Move, true);
wordart.SetLockedProperty(ShapeLockType.Resize, true);
wordart.SetLockedProperty(ShapeLockType.Text, true);
```

##### Αποθήκευση αλλαγών
Βεβαιωθείτε ότι οι αλλαγές έχουν αποθηκευτεί στο βιβλίο εργασίας σας.
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY\output_out.xlsx");
```
