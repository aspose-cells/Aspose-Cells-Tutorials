## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Αρχικοποίηση βιβλίου εργασίας και έλεγχος επικύρωσης δεδομένων για ένα μόνο κελί

#### Επισκόπηση

Μάθετε πώς να αρχικοποιείτε ένα βιβλίο εργασίας και να επικυρώνετε δεδομένα σε συγκεκριμένα κελιά χρησιμοποιώντας το Aspose.Cells.

**Βήμα 1: Εισαγωγή των απαραίτητων βιβλιοθηκών**

Βεβαιωθείτε ότι έχετε εισαγάγει τις απαιτούμενες βιβλιοθήκες Aspose.Cells:

```java
import com.aspose.cells.*;
```

**Βήμα 2: Αρχικοποίηση του βιβλίου εργασίας**

Φορτώστε το αρχείο Excel σε ένα αντικείμενο βιβλίου εργασίας.

```csharp
Workbook workbook = new Workbook("YOUR_SOURCE_DIRECTORY/sampleDataValidationRules.xlsx");
Worksheet worksheet = workbook.getWorksheets().get(0);
Cell cell = worksheet.getCells().get("C1");
```

**Βήμα 3: Επικύρωση δεδομένων κελιού**

Ελέγξτε εάν τα δεδομένα σε ένα συγκεκριμένο κελί πληρούν τα κριτήρια επικύρωσης.

```csharp
// Η τιμή 3 είναι εκτός του εύρους επικύρωσης (10 έως 20)
cell.putValue(3);
System.out.println("Is 3 a Valid Value for this Cell: " + cell.getValidationValue());

// Η τιμή 15 βρίσκεται εντός του εύρους επικύρωσης (10 έως 20)
cell.putValue(15);
System.out.println("Is 15 a Valid Value for this Cell: " + cell.getValidationValue());

// Η τιμή 30 είναι εκτός του εύρους επικύρωσης (10 έως 20)
cell.putValue(30);
System.out.println("Is 30 a Valid Value for this Cell: " + cell.getValidationValue());
```

### Λειτουργία 2: Έλεγχος επικύρωσης δεδομένων για άλλο κελί με διαφορετικό εύρος κανόνων

#### Επισκόπηση

Εφαρμόστε διαφορετικούς κανόνες επικύρωσης δεδομένων σε άλλο κελί.

**Βήμα 1: Αρχικοποίηση βιβλίου εργασίας και κελιού προορισμού**

Φορτώστε το βιβλίο εργασίας και επιλέξτε ένα νέο κελί προορισμού:

```csharp
Workbook workbook2 = new Workbook("YOUR_SOURCE_DIRECTORY/sampleDataValidationRules.xlsx");
Worksheet worksheet2 = workbook2.getWorksheets().get(0);
Cell cell2 = worksheet2.getCells().get("D1");
```

**Βήμα 2: Επικύρωση των δεδομένων**

Εισαγάγετε μια τιμή και ελέγξτε αν πληροί τα κριτήρια επικύρωσης.

```csharp
// Εισαγάγετε τον μεγάλο αριθμό 12345678901 στο κελί D1, ο οποίος θα πρέπει να περάσει την επικύρωση λόγω του εύρους του (1 έως 999999999999)
cell2.putValue(12345678901);
System.out.println("Is 12345678901 a Valid Value for this Cell: " + cell2.getValidationValue());
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι το αρχείο Excel σας έχει ορίσει σωστά τους κανόνες επικύρωσης.
- Ελέγξτε ξανά το εύρος και τα κριτήρια που καθορίζονται στις επικυρώσεις σας.
