## Οδηγός Εφαρμογής

### Φόρτωση βιβλίου εργασίας και φύλλου εργασίας της Access

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα βιβλίο εργασίας του Excel και να αποκτήσετε πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας με το όνομά του χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Αρχικοποίηση του βιβλίου εργασίας
Ξεκινήστε δημιουργώντας ένα `Workbook` αντικείμενο, καθορίζοντας τη διαδρομή προς το αρχείο Excel σας.

```csharp
using Aspose.Cells;

string sourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του βιβλίου εργασίας από τον καθορισμένο κατάλογο
Workbook book = new Workbook(sourceDir + "sampleValidation.xlsx");
```

#### Βήμα 2: Πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας

Για να αποκτήσετε πρόσβαση σε ένα φύλλο εργασίας, χρησιμοποιήστε το όνομά του:

```csharp
// Αποκτήστε πρόσβαση στο φύλλο εργασίας 'Φύλλο1' με το όνομά του
Worksheet sheet = book.Worksheets["Sheet1"];
Cells cells = sheet.Cells; // Λήψη όλων των κελιών στο φύλλο εργασίας που έχετε πρόσβαση
```

### Έλεγχος επικύρωσης για ένα συγκεκριμένο κελί

**Επισκόπηση:**
Αυτή η λειτουργία ελέγχει εάν ένα συγκεκριμένο κελί έχει επικύρωση και προσδιορίζει εάν περιλαμβάνει ένα αναπτυσσόμενο μενού εντός του κελιού.

#### Βήμα 3: Ανάκτηση και επαλήθευση αντικειμένου επικύρωσης

Για οποιοδήποτε δεδομένο κελί, ανακτήστε το `Validation` αντικείμενο για έλεγχο ρυθμίσεων αναπτυσσόμενου μενού εντός κελιού:

```csharp
string cellName = "A2";
Cell targetCell = cells[cellName];
Validation validationObj = targetCell.GetValidation(); // Λήψη της επικύρωσης του καθορισμένου κελιού
bool isInDropdown = validationObj.InCellDropDown; // Ελέγξτε εάν υπάρχει αναπτυσσόμενο μενού εντός κελιού

// Χρησιμοποιήστε το `isInDropdown` για να διαχειριστείτε εάν το κελί είναι αναπτυσσόμενο μενού
```

### Χειρισμός ελέγχων επικύρωσης πολλαπλών κελιών

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να επαναλάβετε την εκτέλεση πολλαπλών κελιών, ελέγχοντας το καθένα για την κατάσταση επικύρωσης σχετικά με τα αναπτυσσόμενα μενού εντός κελιού.

#### Βήμα 4: Επανάληψη σε πολλά κελιά

Επαναλάβετε μια επανάληψη σε έναν πίνακα καθορισμένων κελιών και επαληθεύστε την επικύρωσή τους:

```csharp
foreach (var name in cellNames)
{
    Cell targetCell = cells[name];
    Validation validationObj = targetCell.GetValidation();
    bool isInDropdown = validationObj.InCellDropDown;

}
```
