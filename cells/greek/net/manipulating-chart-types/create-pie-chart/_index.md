# Δημιουργία κυκλικού γραφήματος

## Εισαγωγή

Η δημιουργία γραφημάτων είναι απαραίτητη για την οπτική αναπαράσταση δεδομένων και τα γραφήματα πίτας είναι ένας από τους πιο δημοφιλείς τρόπους για να δείξετε πώς τα μέρη αποτελούν ένα σύνολο. Με το Aspose.Cells για .NET, μπορείτε εύκολα να αυτοματοποιήσετε τη δημιουργία γραφημάτων πίτας σε αρχεία Excel. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στο πώς να δημιουργήσετε ένα γράφημα πίτας από την αρχή χρησιμοποιώντας το Aspose.Cells για .NET, με έναν οδηγό βήμα προς βήμα για να κάνετε τη διαδικασία ομαλή και απλή. Είτε είστε νέοι στο εργαλείο είτε θέλετε να βελτιώσετε τις δεξιότητές σας στον αυτοματισμό του Excel, αυτός ο οδηγός σας καλύπτει!

1. Aspose.Cells για τη βιβλιοθήκη .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει το Aspose.Cells στο έργο σας. Εάν δεν το έχετε εγκαταστήσει ακόμα, μπορείτε να το κατεβάσετε από [εδώ](https://releases.aspose.com/cells/net/).
2. Περιβάλλον ανάπτυξης .NET: Βεβαιωθείτε ότι το έργο σας έχει ρυθμιστεί για χρήση του .NET Framework ή του .NET Core.
3. Βασικές γνώσεις C#: Θα πρέπει να είστε εξοικειωμένοι με τον προγραμματισμό C#, ιδιαίτερα με τον αντικειμενοστρεφή προγραμματισμό (OOP).

Για προχωρημένους χρήστες, μπορεί να εφαρμοστεί μια προσωρινή άδεια χρήσης για να ξεκλειδωθούν όλες οι δυνατότητες του Aspose.Cells. Μπορείτε να ζητήσετε μία από [εδώ](https://purchase.aspose.com/temporary-license/).

## Εισαγωγή πακέτων

Για να ξεκινήσετε, εισαγάγετε τους απαραίτητους χώρους ονομάτων και τα πακέτα που απαιτούνται για αυτό το σεμινάριο. Αυτά περιλαμβάνουν βασικές λειτουργίες εισόδου/εξόδου και το πακέτο Aspose.Cells.

```csharp
using System;
using System.IO;

using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
using Aspose.Cells.Charts;
```

## Βήμα 1: Δημιουργία νέου βιβλίου εργασίας

Αρχικά, πρέπει να δημιουργήσουμε μια παρουσία του `Workbook` κλάση, η οποία αντιπροσωπεύει το αρχείο Excel. Ένα βιβλίο εργασίας περιέχει πολλά φύλλα εργασίας και, για το παράδειγμά μας, θα εργαστούμε με δύο φύλλα εργασίας—ένα για δεδομένα και ένα για το κυκλικό διάγραμμα.

```csharp
Workbook workbook = new Workbook();
```

Αυτό αρχικοποιεί ένα νέο βιβλίο εργασίας του Excel. Αλλά πού πηγαίνουν τα δεδομένα; Ας το φροντίσουμε αυτό στο επόμενο βήμα.

## Βήμα 2: Προσθήκη δεδομένων στο φύλλο εργασίας

Μόλις δημιουργηθεί το βιβλίο εργασίας, πρέπει να αποκτήσουμε πρόσβαση στο πρώτο φύλλο εργασίας και να του δώσουμε ένα όνομα. Εδώ θα εισαγάγουμε τα δεδομένα που απαιτούνται για το κυκλικό διάγραμμα.

```csharp
Worksheet sheet = workbook.Worksheets[0];
sheet.Name = "Data";
Cells cells = sheet.Cells;
```

Τώρα, μπορούμε να εισαγάγουμε ορισμένα εικονικά δεδομένα πωλήσεων που αντιπροσωπεύουν διαφορετικές περιοχές:

```csharp
cells["A1"].PutValue("Region");
cells["A2"].PutValue("France");
cells["A3"].PutValue("Germany");
cells["A4"].PutValue("England");
cells["A5"].PutValue("Sweden");
cells["A6"].PutValue("Italy");
cells["A7"].PutValue("Spain");
cells["A8"].PutValue("Portugal");

cells["B1"].PutValue("Sales");
cells["B2"].PutValue(70000);
cells["B3"].PutValue(55000);
cells["B4"].PutValue(30000);
cells["B5"].PutValue(40000);
cells["B6"].PutValue(35000);
cells["B7"].PutValue(32000);
cells["B8"].PutValue(10000);
```

Εδώ, προσθέτουμε δύο στήλες: μία για τις περιοχές και μία άλλη για τα στοιχεία πωλήσεων. Αυτά τα δεδομένα θα αναπαρασταθούν στο κυκλικό διάγραμμα.

## Βήμα 3: Προσθήκη φύλλου γραφήματος

Στη συνέχεια, ας προσθέσουμε ένα ξεχωριστό φύλλο εργασίας για να φιλοξενήσουμε το κυκλικό διάγραμμα.

```csharp
int sheetIndex = workbook.Worksheets.Add(SheetType.Chart);
Worksheet chartSheet = workbook.Worksheets[sheetIndex];
chartSheet.Name = "Chart";
```

Αυτό το νέο φύλλο θα φιλοξενεί το κυκλικό γράφημα. Δίνοντάς του ένα όνομα, όπως "Γράφημα", διασφαλίζεται ότι οι χρήστες γνωρίζουν τι να περιμένουν όταν ανοίγουν το αρχείο.

## Βήμα 4: Δημιουργήστε το γράφημα πίτας

Τώρα είναι η ώρα να δημιουργήσουμε το πραγματικό γράφημα. Θα καθορίσουμε ότι θέλουμε ένα γράφημα πίτας και θα ορίσουμε τη θέση του στο φύλλο.

```csharp
int chartIndex = chartSheet.Charts.Add(Aspose.Cells.Charts.ChartType.Pie, 5, 0, 25, 10);
Aspose.Cells.Charts.Chart chart = chartSheet.Charts[chartIndex];
```

Η μέθοδος `Add()` δέχεται παραμέτρους για τον τύπο γραφήματος (σε αυτήν την περίπτωση, `ChartType.Pie`), και τη θέση του στο φύλλο εργασίας. Οι αριθμοί αντιπροσωπεύουν τις θέσεις γραμμών και στηλών.

## Βήμα 5: Προσαρμόστε την εμφάνιση του γραφήματος

Ένα γράφημα πίτας δεν θα ήταν ολοκληρωμένο χωρίς κάποια προσαρμογή! Ας κάνουμε το γράφημά μας οπτικά ελκυστικό τροποποιώντας τα χρώματα, τις ετικέτες και τον τίτλο.

### Ορισμός τίτλου γραφήματος
```csharp
chart.Title.Text = "Sales By Region";
chart.Title.Font.Color = Color.Blue;
chart.Title.Font.IsBold = true;
chart.Title.Font.Size = 12;
```

### Προσαρμογή περιοχής γραφήματος
```csharp
chart.PlotArea.Area.ForegroundColor = Color.Coral;
chart.PlotArea.Area.FillFormat.SetTwoColorGradient(Color.Yellow, Color.White, GradientStyleType.Vertical, 2);
chart.PlotArea.Border.IsVisible = false;
```

Ορίζουμε το γέμισμα με διαβάθμιση για την περιοχή του γραφήματος και κρύβουμε το περίγραμμα για πιο καθαρή εμφάνιση.

## Βήμα 6: Ορισμός δεδομένων γραφήματος

Ήρθε η ώρα να συνδέσουμε το διάγραμμα με τα δεδομένα μας. `NSeries` Η ιδιότητα του γραφήματος συνδέει τα στοιχεία πωλήσεων και τις περιοχές με το κυκλικό γράφημα.

```csharp
chart.NSeries.Add("Data!B2:B8", true);
chart.NSeries.CategoryData = "Data!A2:A8";
chart.NSeries.IsColorVaried = true;
```

Η πρώτη γραμμή καθορίζει ότι χρησιμοποιούμε τα δεδομένα πωλήσεων από τα κελιά `B2:B8`Επίσης, λέμε στο διάγραμμα να χρησιμοποιήσει τα ονόματα περιοχών από `A2:A8` ως ετικέτες κατηγορίας.

## Βήμα 7: Προσθήκη ετικετών δεδομένων

Η προσθήκη ετικετών απευθείας στα τμήματα του γραφήματος μπορεί να διευκολύνει την κατανόηση. Ας συμπεριλάβουμε τα ονόματα περιοχών και τις τιμές πωλήσεων μέσα στα τμήματα του γραφήματος πίτας.

```csharp
for (int i = 0; i < chart.NSeries.Count; i++)
{
    DataLabels labels = chart.NSeries[i].DataLabels;
    labels.ShowCategoryName = true;
    labels.ShowValue = true;
    labels.Position = LabelPositionType.InsideBase;
}
```

## Βήμα 8: Προσαρμογή περιοχής γραφήματος και υπομνήματος

Τέλος, ας δώσουμε μερικές τελευταίες πινελιές στην περιοχή του γραφήματος και στο υπόμνημα. Αυτό βελτιώνει τη συνολική παρουσίαση του γραφήματος.

### Περιοχή γραφήματος
```csharp
ChartArea chartArea = chart.ChartArea;
chartArea.Area.Formatting = FormattingType.Custom;
chartArea.Area.FillFormat.Texture = TextureType.BlueTissuePaper;
```

### Θρύλος
```csharp
Legend legend = chart.Legend;
legend.Position = LegendPositionType.Left;
legend.Font.IsBold = true;
legend.Border.Color = Color.Blue;
legend.Area.FillFormat.Texture = TextureType.Bouquet;
```

## Βήμα 9: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύουμε το βιβλίο εργασίας σε ένα αρχείο Excel. Μπορείτε να καθορίσετε τον κατάλογο εξόδου και το όνομα αρχείου, όπως απαιτείται.

```csharp
workbook.Save(outputDir + "outputHowToCreatePieChart.xlsx");
```
