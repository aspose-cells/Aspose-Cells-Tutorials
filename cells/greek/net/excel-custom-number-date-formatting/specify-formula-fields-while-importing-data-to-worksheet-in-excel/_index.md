# Καθορισμός πεδίων τύπου κατά την εισαγωγή δεδομένων σε φύλλο Excel

## Εισαγωγή

Όσον αφορά τον προγραμματιστικό χειρισμό αρχείων Excel, το Aspose.Cells για .NET είναι ένα ανεκτίμητο εργαλείο. Παρέχει ισχυρή λειτουργικότητα για τη δημιουργία, την τροποποίηση και τον χειρισμό υπολογιστικών φύλλων Excel με ευκολία. Ένα από τα ενδιαφέροντα χαρακτηριστικά που προσφέρει είναι η δυνατότητα καθορισμού πεδίων τύπων κατά την εισαγωγή δεδομένων σε ένα φύλλο Excel. Φανταστείτε ότι εργάζεστε σε μια οικονομική αναφορά και πρέπει να υπολογίζετε αυτόματα τα σύνολα με βάση την εισαγωγή δεδομένων από τον χρήστη. Αυτό το σεμινάριο θα σας καθοδηγήσει βήμα προς βήμα για να το πετύχετε αυτό με μια καθαρή και απλή προσέγγιση.

## Προαπαιτούμενα

Πριν εμβαθύνουμε στον κώδικα, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε. 

1. Visual Studio ή οποιοδήποτε ενσωματωμένο περιβάλλον ανάπτυξης (IDE) .NET: Βεβαιωθείτε ότι έχετε ένα κατάλληλο IDE για να γράψετε και να εκτελέσετε τον κώδικα C#.
2. Aspose.Cells για .NET: Θα χρειαστεί να κατεβάσετε και να αναφέρετε τη βιβλιοθήκη Aspose.Cells στο έργο σας. Μπορείτε να την κατεβάσετε από το [Απελευθερώσεις Aspose](https://releases.aspose.com/cells/net/).
3. Βασικές γνώσεις C#: Η εξοικείωση με την C# και τις έννοιες του αντικειμενοστρεφούς προγραμματισμού θα σας βοηθήσει να κατανοήσετε καλύτερα τα παραδείγματα.
4. .NET Framework: Αυτό το σεμινάριο προϋποθέτει ότι χρησιμοποιείτε .NET Framework 4.5 ή νεότερη έκδοση.

Μόλις διευθετήσετε τις προϋποθέσεις, ας προχωρήσουμε στην εισαγωγή ορισμένων δεδομένων σε ένα φύλλο Excel με καθορισμένα πεδία τύπων.

## Εισαγωγή πακέτων

Πριν ξεκινήσετε να γράφετε τον κώδικά σας, θα χρειαστεί να εισαγάγετε τον απαραίτητο χώρο ονομάτων Aspose.Cells. Αυτό γίνεται συνήθως στην αρχή του αρχείου C#:

```csharp
using Aspose.Cells;
using System;
using System.Collections.Generic;
```

Αυτό σας επιτρέπει να χρησιμοποιείτε τις κλάσεις και τις μεθόδους που παρέχονται από τη βιβλιοθήκη Aspose.Cells χωρίς να χρειάζεται να τους προσθέτετε το πρόθεμα του χώρου ονομάτων κάθε φορά.

Ας χωρίσουμε ολόκληρη τη διαδικασία σε διαχειρίσιμα βήματα:

## Βήμα 1: Ορίστε τον κατάλογο εξόδου

Αρχικά, πρέπει να καθορίσετε πού θέλετε να αποθηκεύσετε το αρχείο Excel. Δείτε πώς μπορείτε να το κάνετε:

```csharp
static string outputDir = "Your Document Directory"; // καθορίστε τον κατάλογο εγγράφων σας εδώ
```

Αντικαθιστώ `"Your Document Directory"` με την πραγματική διαδρομή του αρχείου σας. Εδώ θα αποθηκευτεί το αρχείο Excel που δημιουργήθηκε.

## Βήμα 2: Δημιουργήστε μια κλάση που ορίζεται από τον χρήστη για στοιχεία δεδομένων

Στη συνέχεια, θα ορίσουμε μια κλάση για να δομήσουμε τα δεδομένα που σκοπεύουμε να εισαγάγουμε.

```csharp
class DataItems
{
    public int Number1 { get; set; }
    public int Number2 { get; set; }
    public string Formula1 { get; set; }
    public string Formula2 { get; set; }
}
```

Αυτό `DataItems` Η κλάση θα περιέχει τους ακατέργαστους ακέραιους αριθμούς και τους τύπους που θα γράψουμε στο φύλλο Excel. 

## Βήμα 3: Αρχικοποίηση λίστας για τη διατήρηση στοιχείων δεδομένων

Θα χρησιμοποιήσουμε μια λίστα για να διατηρήσουμε πολλαπλές εμφανίσεις του `DataItems` τάξη.

```csharp
List<DataItems> dis = new List<DataItems>();
```

## Βήμα 4: Προσθήκη στοιχείων δεδομένων στη λίστα

Τώρα, ας προσθέσουμε μερικές καταχωρήσεις στη λίστα μας. Κάθε καταχώρηση θα περιέχει δύο αριθμούς και δύο τύπους.

```csharp
// Ορισμός και προσθήκη κάθε στοιχείου δεδομένων
DataItems di = new DataItems();
di.Number1 = 2002;
di.Number2 = 3502;
di.Formula1 = "=SUM(A2,B2)";
di.Formula2 = "=HYPERLINK(\"https://www.aspose.com\",\"Ιστότοπος Aspose\")";
dis.Add(di);

// Επαναλάβετε για επιπλέον στοιχεία δεδομένων
```

Φροντίστε να προσαρμόσετε το καθένα `DataItems` παράδειγμα με μοναδικές τιμές και τύπους.

## Βήμα 5: Δημιουργία βιβλίου εργασίας και φύλλου εργασίας της Access

Στη συνέχεια, δημιουργήστε το βιβλίο εργασίας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας όπου τελικά θα εισαγάγουμε τα δεδομένα.

```csharp
Workbook wb = new Workbook(); // δημιουργία νέου βιβλίου εργασίας
Worksheet ws = wb.Worksheets[0]; // πρόσβαση στο πρώτο φύλλο εργασίας
```

## Βήμα 6: Καθορισμός επιλογών εισαγωγής πίνακα

Εδώ ακριβώς συμβαίνει η μαγεία. Πρέπει να καθορίσετε ποια πεδία στα δεδομένα σας αντιστοιχούν σε τύπους. 

```csharp
ImportTableOptions opts = new ImportTableOptions();
opts.IsFormulas = new bool[] { false, false, true, true };
```

Σε αυτό το παράδειγμα, τα δύο τελευταία πεδία περιέχουν τύπους, κάτι που υποδεικνύεται από `true`, ενώ τα δύο πρώτα πεδία έχουν οριστεί σε `false`.

## Βήμα 7: Εισαγωγή προσαρμοσμένων αντικειμένων

Τώρα που όλα έχουν ρυθμιστεί, ας εισαγάγουμε τη λίστα με τα στοιχεία δεδομένων μας στο φύλλο εργασίας.

```csharp
ws.Cells.ImportCustomObjects(dis, 0, 0, opts);
```

Αυτή η γραμμή ουσιαστικά εισάγει τα δεδομένα ξεκινώντας από το κελί A1.

## Βήμα 8: Υπολογισμός τύπων

Δεδομένου ότι έχουμε εισαγάγει ορισμένους τύπους, είναι ζωτικής σημασίας να τους υπολογίσουμε.

```csharp
wb.CalculateFormula();
```

Αυτή η μέθοδος διασφαλίζει ότι οι τύποι σας αξιολογούνται με βάση τις εξαρτήσεις τους.

## Βήμα 9: Αυτόματη Προσαρμογή Στηλών

Για να βεβαιωθείτε ότι τα δεδομένα σας είναι εύκολα στην προβολή, μπορείτε να προσαρμόσετε αυτόματα τις στήλες με βάση το περιεχόμενο.

```csharp
ws.AutoFitColumns();
```

Αυτό το βήμα βελτιστοποιεί τη διάταξη του αρχείου Excel. 

## Βήμα 10: Αποθηκεύστε το αρχείο Excel σας

Τέλος, ήρθε η ώρα να αποθηκεύσετε το νέο σας αρχείο Excel. 

```csharp
wb.Save(outputDir + "outputSpecifyFormulaFieldsWhileImportingDataToWorksheet.xlsx");
```

Βεβαιωθείτε ότι το όνομα αρχείου εξόδου σας είναι σχετικό και περιγραφικό!

## Βήμα 11: Έλεγχος εκτέλεσης

Ως ένας απλός τρόπος για να επιβεβαιώσετε ότι όλα εκτελέστηκαν σωστά, ίσως θελήσετε να εκτυπώσετε ένα μήνυμα.

```csharp
Console.WriteLine("SpecifyFormulaFieldsWhileImportingDataToWorksheet executed successfully.");
```

Αυτό σας δίνει άμεση ανατροφοδότηση ότι ο κώδικας λειτούργησε χωρίς προβλήματα.
