## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στην υποβολή ερωτημάτων σε αντιστοιχίσεις XML χρησιμοποιώντας το Aspose.Cells για .NET. Το παράδειγμα κώδικα που παρέχεται θα δείξει την υποβολή ερωτημάτων σε συγκεκριμένες διαδρομές μέσα σε έναν χάρτη XML και την ανάκτηση των αντιστοιχισμένων περιοχών κελιών.

### Βήμα 1: Φόρτωση του αρχείου Excel
Ξεκινήστε φορτώνοντας το αρχείο Excel που περιέχει τον χάρτη XML:
```csharp
// Ορισμός διαδρομής καταλόγου πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Φόρτωση δείγματος αρχείου Excel με XmlMap
Workbook workbook = new Workbook(sourceDir + "sampleXmlMapQuery.xlsx");
```

### Βήμα 2: Πρόσβαση στον χάρτη XML
Αποκτήστε πρόσβαση στον πρώτο χάρτη XML στο βιβλίο εργασίας σας. Αυτό το παράδειγμα προϋποθέτει ότι έχει οριστεί τουλάχιστον ένας χάρτης XML:
```csharp
// Ανάκτηση του πρώτου χάρτη XML από τη συλλογή
XmlMap xmlMap = workbook.Worksheets.XmlMaps[0];
```

### Βήμα 3: Υποβολή ερωτημάτων για συγκεκριμένες διαδρομές εντός του χάρτη XML
Μπορείτε να υποβάλετε ερώτημα σε συγκεκριμένες διαδρομές για να ανακτήσετε αντιστοιχισμένες περιοχές κελιών. Δείτε πώς μπορείτε να το κάνετε:

#### Υποβολή ερωτήματος για μια γενική διαδρομή
```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet worksheet = workbook.Worksheets[0];

// Χάρτης Xml ερωτήματος από διαδρομή - /MiscData
Console.WriteLine("Query Xml Map from Path - /MiscData");
ArrayList results = worksheet.XmlMapQuery("/MiscData", xmlMap);

// Εκτύπωση επιστρεφόμενων τιμών ArrayList
foreach (var item in results)
{
    Console.WriteLine(item);
}
```

#### Υποβολή ερωτήματος σε μια ένθετη διαδρομή
```csharp
// Χάρτης Xml ερωτήματος από διαδρομή - /MiscData/row/Color
Console.WriteLine("Query Xml Map from Path - /MiscData/row/Color");
results = worksheet.XmlMapQuery("/MiscData/row/Color", xmlMap);

// Εκτύπωση επιστρεφόμενων τιμών ArrayList
foreach (var item in results)
{
    Console.WriteLine(item);
}
```
