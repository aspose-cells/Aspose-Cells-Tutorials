### Χαρακτηριστικό 1: Φόρτωση βιβλίου εργασίας και έλεγχος υπογραφής VBA (H2)

#### Επισκόπηση
Η φόρτωση ενός βιβλίου εργασίας για την επαλήθευση της υπογραφής του έργου VBA διασφαλίζει την ακεραιότητα και την ασφάλεια στις εργασίες αυτοματισμού.

#### Βήμα προς βήμα εφαρμογή

##### H3. Φόρτωση του βιβλίου εργασίας
Καθορίστε τη διαδρομή καταλόγου του αρχείου Excel:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "sampleCheckVbaSignatureIsValid.xlsm");
```

##### H3. Έλεγχος εγκυρότητας υπογραφής VBA
Προσδιορίστε εάν η υπογραφή VBA είναι έγκυρη:
```csharp
bool isValidSigned = workbook.VbaProject.IsValidSigned;
Console.WriteLine("Is VBA signed: " + isValidSigned);
```

#### Εξήγηση
- **Βιβλίο ασκήσεων**: Αντιπροσωπεύει το αρχείο Excel σας.
- **ΕίναιΈγκυροΥπογεγραμμένο**: Μια λογική τιμή που υποδεικνύει εάν η υπογραφή του έργου VBA είναι έγκυρη.

### Λειτουργία 2: Τροποποίηση και αποθήκευση κώδικα VBA (H2)

#### Επισκόπηση
Η τροποποίηση κώδικα VBA περιλαμβάνει την τροποποίηση συγκεκριμένου περιεχομένου λειτουργικής μονάδας, την αποθήκευση αλλαγών σε μια ροή και την επαναφόρτωση του βιβλίου εργασίας.

#### Βήμα προς βήμα εφαρμογή

##### H3. Τροποποίηση περιεχομένου ενότητας VBA
Πρόσβαση και τροποποίηση της πρώτης ενότητας VBA:
```csharp
string code = workbook.VbaProject.Modules[1].Codes;
code = code.Replace("Welcome to Aspose", "Welcome to Aspose.Cells");
workbook.VbaProject.Modules[1].Codes = code;
```

##### H3. Αποθήκευση σε ροή μνήμης
Αποθηκεύστε το τροποποιημένο βιβλίο εργασίας σε ένα `MemoryStream`:
```csharp
using System.IO;
MemoryStream ms = new MemoryStream();
workbook.Save(ms, SaveFormat.Xlsm);
```

##### H3. Επαναφόρτωση βιβλίου εργασίας από τη ροή
Επαναφορτώστε και επαληθεύστε ξανά την υπογραφή VBA:
```csharp
ms.Position = 0;
Workbook reloadedWorkbook = new Workbook(ms, new LoadOptions(LoadFormat.Xlsx));
bool isReloadedSignatureValid = reloadedWorkbook.VbaProject.IsValidSigned;
Console.WriteLine("Is reloaded VBA signed: " + isReloadedSignatureValid);
```

#### Εξήγηση
- **Ενότητες[1]**Αναφέρεται στην πρώτη ενότητα στο έργο VBA του βιβλίου εργασίας.
- **Ροή μνήμης**: Χρησιμοποιείται για την αποθήκευση και την επαναφόρτωση βιβλίων εργασίας χωρίς εγγραφή στο δίσκο.
