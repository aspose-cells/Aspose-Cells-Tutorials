## Οδηγός Εφαρμογής

### Ανάκτηση υπερσυνδέσμων από μια περιοχή

#### Επισκόπηση

Η ανάκτηση υπερσυνδέσμων εντός μιας περιοχής του Excel σάς επιτρέπει να αυτοματοποιήσετε τη διαδικασία ανάλυσης ή τροποποίησής τους. Αυτό το παράδειγμα δείχνει την εξαγωγή υπερσυνδέσμων από τα κελιά A2 έως B3.

#### Βήματα Υλοποίησης

1. **Ρύθμιση διαδρομών καταλόγου**
   Ορίστε διαδρομές για τους καταλόγους προέλευσης και εξόδου.
   ```csharp
   string sourceDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
   string outputDir = RunExamples.Get_OutputDirectory();
   ```

2. **Φόρτωση του βιβλίου εργασίας**
   Ανοίξτε ένα υπάρχον αρχείο Excel που περιέχει υπερσυνδέσμους.
   ```csharp
   Workbook workbook = new Workbook(sourceDir + "HyperlinksSample.xlsx");
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **Δημιουργία εύρους και ανάκτηση υπερσυνδέσμων**
   Ορίστε την περιοχή κελιών και εξαγάγετε υπερσυνδέσμους από αυτήν.
   ```csharp
   Range range = worksheet.Cells.CreateRange("A2", "B3");
   Hyperlink[] hyperlinks = range.Hyperlinks;
   
   foreach (Hyperlink link in hyperlinks)
   {
       Console.WriteLine(link.Area + " : " + link.Address);
       // Προαιρετικά: Διαγράψτε τον υπερσύνδεσμο.
       link.Delete();
   }
   ```

4. **Αποθήκευση αλλαγών**
   Αποθηκεύστε το βιβλίο εργασίας με τις αλλαγές σε ένα νέο αρχείο.
   ```csharp
   workbook.Save(outputDir + "HyperlinksSample_out.xlsx");
   ```

### Διαγραφή υπερσυνδέσμων

Ο `Delete()` Η μέθοδος χρησιμοποιείται για την αφαίρεση υπερσυνδέσμων από το καθορισμένο εύρος, απλοποιώντας τις διαδικασίες καθαρισμού δεδομένων ή προετοιμάζοντας αρχεία για περαιτέρω ανάλυση χωρίς εξωτερικούς συνδέσμους.
