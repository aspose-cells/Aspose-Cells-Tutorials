## Οδηγός Εφαρμογής
Θα αναλύσουμε τα βήματα για την αποτελεσματική διαχείριση εξωτερικών συνδέσμων.

### Αναγνώριση και ανάκτηση εύρους με εξωτερικούς συνδέσμους
#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να κάνετε επανάληψη σε ονομασμένες περιοχές σε ένα αρχείο Excel για να εντοπίσετε εκείνες που είναι συνδεδεμένες εξωτερικά.

#### Βήμα προς βήμα εφαρμογή
**1. Φόρτωση του βιβλίου εργασίας σας**
Φορτώστε το αρχείο προέλευσης Excel:
```csharp
string sourceDir = "YourSourceDirectoryPath";
Workbook workbook = new Workbook(sourceDir + "SampleExternalReferences.xlsx");
```
**2. Επαναλάβετε μέσω ονομασμένων περιοχών**
Αποκτήστε πρόσβαση σε κάθε ονομασμένη περιοχή και ελέγξτε για εξωτερικούς συνδέσμους:
```csharp
foreach (Name namedRange in workbook.Worksheets.Names)
{
    ReferredArea[] referredAreas = namedRange.GetReferredAreas(true);
    
    if (referredAreas != null)
    {
        foreach (var referredArea in referredAreas)
        {
            // Εκτύπωση λεπτομερειών για κάθε εξωτερικό σύνδεσμο
            Console.WriteLine("IsExternalLink: " + referredArea.IsExternalLink);
            Console.WriteLine("SheetName: " + referredArea.SheetName);
            Console.WriteLine("ExternalFileName: " + referredArea.ExternalFileName);
            // Πρόσθετες πληροφορίες μπορούν να εκτυπωθούν εδώ, όπως απαιτείται
        }
    }
}
```
**Επεξήγηση βασικών παραμέτρων:**
- **`GetReferredAreas(true)`**: Ανακτά περιοχές που συνδέονται με την καθορισμένη περιοχή. `true` Η παράμετρος διασφαλίζει ότι περιλαμβάνονται εξωτερικές αναφορές.
- **`IsExternalLink`**: Υποδεικνύει εάν η αναφερόμενη περιοχή είναι εξωτερικός σύνδεσμος.
