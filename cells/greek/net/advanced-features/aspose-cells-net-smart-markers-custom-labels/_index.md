## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε διαχειρίσιμες ενότητες για να σας βοηθήσουμε να χρησιμοποιήσετε αποτελεσματικά τους Έξυπνους Δείκτες και να προσαρμόσετε τις ετικέτες.

### Βήμα 1: Προετοιμασία του βιβλίου εργασίας σας

Αρχικά, θα προετοιμάσουμε το πρότυπο βιβλίου εργασίας μας που περιέχει Έξυπνους Δείκτες. Αυτοί οι δείκτες λειτουργούν ως placeholders στο αρχείο Excel σας, οι οποίοι θα αντικατασταθούν με πραγματικά δεδομένα κατά την επεξεργασία.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Φόρτωση του βιβλίου εργασίας που περιέχει τους Έξυπνους Δείκτες
Workbook designer = new Workbook(dataDir + "SmartMarker_Designer.xlsx");
```

### Βήμα 2: Εξαγωγή δεδομένων

Χρειαζόμαστε δεδομένα για να συμπληρώσουμε το πρότυπό μας. Εδώ, θα τα εξαγάγουμε από ένα υπάρχον αρχείο Excel.

```csharp
// Δημιουργήστε ένα νέο αντικείμενο Βιβλίου Εργασίας για το αρχείο προέλευσης
Workbook workbook = new Workbook(dataDir + "Book1.xlsx");

// Εξαγωγή δεδομένων από το πρώτο φύλλο εργασίας σε έναν Πίνακα Δεδομένων
DataTable dt = workbook.Worksheets[0].Cells.ExportDataTable(0, 0, 11, 5, true);

// Αντιστοίχιση ονόματος στον Πίνακα Δεδομένων
dt.TableName = "Report";
```

### Βήμα 3: Ρύθμιση παραμέτρων του WorkbookDesigner

Στη συνέχεια, χρησιμοποιήστε `WorkbookDesigner` για να συνδέσετε δεδομένα με τους Έξυπνους Δείκτες σας.

```csharp
// Δημιουργήστε μια παρουσία της κλάσης WorkbookDesigner
WorkbookDesigner d = new WorkbookDesigner();

// Ορισμός του βιβλίου εργασίας σχεδιαστή
d.Workbook = designer;

// Αντιστοίχιση DataTable ως πηγής δεδομένων
d.SetDataSource(dt);

// Επεξεργαστείτε τους έξυπνους δείκτες στο πρότυπο
d.Process();
```

### Βήμα 4: Αποθήκευση της εξόδου σας

Μετά την επεξεργασία, αποθηκεύστε το αρχείο σας για να ολοκληρώσετε την αυτοματοποίηση.

```csharp
// Αποθήκευση του αρχείου εξόδου
designer.Save(dataDir + "output.xlsx", SaveFormat.Xlsx);
```

**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι η σύνταξη του Έξυπνου Δείκτη στο πρότυπο ταιριάζει με τη δομή της πηγής δεδομένων. Συνηθισμένα προβλήματα περιλαμβάνουν αναντιστοιχία ονομάτων ή λανθασμένες μορφές placeholder.
