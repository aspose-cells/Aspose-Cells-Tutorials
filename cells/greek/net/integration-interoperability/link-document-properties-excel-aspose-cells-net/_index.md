## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Ρύθμιση συνδέσμων ιδιοτήτων εγγράφου

Αυτή η λειτουργία δείχνει πώς να συνδέσετε προσαρμοσμένες ιδιότητες εγγράφου σε ένα αρχείο Excel με συγκεκριμένα εύρη περιεχομένου.

#### Επισκόπηση

Η σύνδεση ιδιοτήτων εγγράφου σάς επιτρέπει να δημιουργείτε δυναμικές αναφορές μέσα στα υπολογιστικά φύλλα σας, καθιστώντας τη διαχείριση δεδομένων πιο διαισθητική και αυτοματοποιημένη. Αυτό μπορεί να είναι ιδιαίτερα χρήσιμο για την παρακολούθηση του κατόχου ή της έκδοσης ενός συνόλου δεδομένων απευθείας από το περιεχόμενό του.

#### Βήμα προς βήμα εφαρμογή

##### 1. Ρύθμιση παραμέτρων καταλόγων

Ορίστε τους καταλόγους προέλευσης και εξόδου όπου θα βρίσκονται τα αρχεία Excel:

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```

**Εξήγηση**Αυτά τα placeholders θα πρέπει να αντικατασταθούν με τις πραγματικές διαδρομές προς το σύστημα αρχείων του έργου σας.

##### 2. Φόρτωση βιβλίου εργασίας

Δημιουργήστε ένα υπόδειγμα `Workbook` αντικείμενο για εργασία με ένα υπάρχον αρχείο Excel:

```csharp
Workbook workbook = new Workbook(SourceDir + "sample-document-properties.xlsx");
```

**Σκοπός**Αυτό φορτώνει το έγγραφο Excel στη μνήμη, επιτρέποντάς σας να χειριστείτε τις ιδιότητες και το περιεχόμενό του μέσω προγραμματισμού.

##### 3. Ανάκτηση Προσαρμοσμένων Ιδιοτήτων

Αποκτήστε πρόσβαση στη συλλογή προσαρμοσμένων ιδιοτήτων εγγράφου μέσα στο βιβλίο εργασίας:

```csharp
CustomDocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```

**Λειτουργικότητα**: `customProperties` παρέχει πρόσβαση σε όλα τα μεταδεδομένα που ορίζονται από τον χρήστη και σχετίζονται με το αρχείο Excel σας.

##### 4. Προσθήκη συνδέσμου προς περιεχόμενο

Συνδέστε μια ιδιότητα με ένα συγκεκριμένο εύρος στο φύλλο εργασίας σας:

```csharp
customProperties.AddLinkToContent("Owner", "MyRange");
```

**Παράμετροι**:
- `"Owner"`: Όνομα της ιδιότητας του προσαρμοσμένου εγγράφου.
- `"MyRange"`: Η αναφορά κελιού ή η περιοχή εντός της οποίας συνδέεται αυτή η ιδιότητα.

##### 5. Επαλήθευση συνδέσμου

Ελέγξτε εάν η προσαρμοσμένη ιδιότητα έχει συνδεθεί με επιτυχία:

```csharp
DocumentProperty customProperty1 = customProperties["Owner"];
bool isLinkedToContent = customProperty1.IsLinkedToContent;
string source = customProperty1.Source; // π.χ., "A1"
```

**Επαλήθευση**: `isLinkedToContent` επιβεβαιώνει εάν η σύνδεση έχει δημιουργηθεί και `source` σας δίνει την ακριβή αναφορά κελιού ή περιοχής.

##### 6. Αποθήκευση τροποποιημένου αρχείου

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```csharp
workbook.Save(outputDir + "out_sample-document-properties.xlsx");
```

**Σπουδαιότητα**Αυτό το βήμα διασφαλίζει ότι όλες οι τροποποιήσεις διατηρούνται σε ένα αρχείο Excel εξόδου.

#### Συμβουλές αντιμετώπισης προβλημάτων

- **Σφάλμα "Δεν βρέθηκε αρχείο"**: Επαληθεύστε τη διαδρομή που καθορίζεται στο `SourceDir` είναι σωστό.
- **Σύνδεση αποτυχιών**Βεβαιωθείτε ότι η περιοχή στην οποία συνδέεστε υπάρχει και ταιριάζει με τη δομή του βιβλίου εργασίας σας.
