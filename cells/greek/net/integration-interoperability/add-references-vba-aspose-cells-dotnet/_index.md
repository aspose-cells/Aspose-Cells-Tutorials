## Οδηγός Εφαρμογής

### Προσθήκη καταχωρημένων αναφορών στο έργο VBA
Αυτή η ενότητα θα σας καθοδηγήσει στην προσθήκη καταχωρισμένων αναφορών όπως `stdole` και `Office` βιβλιοθήκες που χρησιμοποιούν Aspose.Cells για .NET.

#### Επισκόπηση
Η προσθήκη καταχωρημένων αναφορών είναι ζωτικής σημασίας όταν οι μακροεντολές σας εξαρτώνται από εξωτερικά DLL, διασφαλίζοντας ότι όλα τα στοιχεία είναι διαθέσιμα κατά την εκτέλεση.

#### Βήματα Υλοποίησης
1. **Αρχικοποίηση του βιβλίου εργασίας**
   Δημιουργήστε μια νέα παρουσία του `Workbook` κλάση που αντιπροσωπεύει ένα αρχείο Excel:
   ```csharp
   Workbook workbook = new Workbook();
   ```
2. **Πρόσβαση στο έργο VBA**
   Ανακτήστε το έργο VBA που σχετίζεται με το βιβλίο εργασίας σας:
   ```csharp
   VbaProject vbaProj = workbook.VbaProject;
   ```
3. **Προσθήκη καταχωρημένων αναφορών**
   Προσθήκη καταχωρημένης αναφοράς σε `stdole` και `Office` βιβλιοθήκες χρησιμοποιώντας τα αντίστοιχα GUID τους.
   - **Για τη βιβλιοθήκη stdole:**
     ```csharp
     vbaProj.References.AddRegisteredReference(
         "stdole",
         "*\\G{00020430-0000-0000-C000-000000000046}#2.0#0#C:\\Windows\\system32\\stdole2.tlb#OLE Automation"
     );
     ```
   - **Για τη Βιβλιοθήκη του Office:**
     ```csharp
     vbaProj.References.AddRegisteredReference(
         "Office",
         "*\\G{2DF8D04C-5BFA-101B-BDE5-00AA0044DE52}#2.0#0#C:\\Program Files\\Common Files\\Microsoft Shared\\OFFICE14\\MSO.DLL#Microsoft Office 14.0 Object Library"
     );
     ```
   Κάθε `AddRegisteredReference` Η κλήση απαιτεί το όνομα της βιβλιοθήκης, το GUID της, τον αριθμό έκδοσης και τη διαδρομή αρχείου.
4. **Αποθήκευση του βιβλίου εργασίας**
   Αποθηκεύστε το βιβλίο εργασίας σας με το ενημερωμένο έργο VBA:
   ```csharp
   string outputPath = "YOUR_OUTPUT_DIRECTORY" + "Output_out.xlsm";
   workbook.Save(outputPath);
   ```
#### Συμβουλές αντιμετώπισης προβλημάτων
- Επαληθεύστε ότι οι διαδρομές προς τα αρχεία DLL ή TLB είναι σωστές και προσβάσιμες.
- Βεβαιωθείτε ότι το GUID ταιριάζει με την έκδοση της βιβλιοθήκης σας.
