## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την ανάκτηση δεδομένων σύνδεσης SQL χρησιμοποιώντας το Aspose.Cells για .NET.

### Επισκόπηση

Στόχος μας είναι να εξαγάγουμε τις ιδιότητες μιας σύνδεσης βάσης δεδομένων που ορίζεται σε ένα βιβλίο εργασίας του Excel, συμπεριλαμβανομένων των λεπτομερειών της εντολής, των διαπιστευτηρίων και των παραμέτρων ερωτήματος.

### Βήμα προς βήμα εφαρμογή

#### 1. Πρόσβαση σε εξωτερικές συνδέσεις

Φορτώστε το αρχείο Excel και αποκτήστε πρόσβαση στις εξωτερικές του συνδέσεις:
```csharp
// Κατάλογος πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Φόρτωση βιβλίου εργασίας από το αρχείο προέλευσης
Workbook workbook = new Workbook(sourceDir + "sampleRetrievingSQLConnectionData.xlsx");

// Πρόσβαση σε εξωτερικές συλλογές
ExternalConnectionCollection connections = workbook.DataConnections;
```

#### 2. Επαναληπτική Μετάβαση Μέσω Συνδέσεων

Επαναλάβετε τη διαδικασία μέσω των διαθέσιμων συνδέσεων δεδομένων και εντοπίστε τις συνδέσεις βάσης δεδομένων:
```csharp
for (int i = 0; i < connections.Count; i++)
{
    ExternalConnection connection = connections[i];
    
    // Έλεγχος για τύπο DBConnection
    if (connection is DBConnection)
    {
        ProcessDBConnection((DBConnection)connection);
    }
}
```

#### 3. Ανάκτηση ιδιοτήτων σύνδεσης

Ορίστε μια μέθοδο για την επεξεργασία κάθε σύνδεσης βάσης δεδομένων και την ανάκτηση των ιδιοτήτων της:
```csharp
private static void ProcessDBConnection(DBConnection dbConn)
{
    // Ανάκτηση διαφόρων ιδιοτήτων σύνδεσης βάσης δεδομένων
    Console.WriteLine("Command: " + dbConn.Command);
    Console.WriteLine("Command Type: " + dbConn.CommandType);
    Console.WriteLine("Description: " + dbConn.ConnectionDescription);
    Console.WriteLine("ID: " + dbConn.ConnectionId);
    Console.WriteLine("Credentials Method: " + dbConn.CredentialsMethodType);
    Console.WriteLine("Name: " + dbConn.Name);

    // Παράμετροι σύνδεσης διεργασίας
    foreach (ConnectionParameter param in dbConn.Parameters)
    {
        Console.WriteLine($"Cell Reference: {param.CellReference}");
        Console.WriteLine($"Parameter Name: {param.Name}");
        Console.WriteLine($"Prompt: {param.Prompt}");
        Console.WriteLine($"SQL Type: {param.SqlType}");
        Console.WriteLine($"Param Value: {param.Value}");
    }
}
```
