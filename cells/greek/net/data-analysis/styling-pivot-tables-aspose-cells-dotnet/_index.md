## Οδηγός Εφαρμογής (H2)

Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο δημιουργίας και διαμόρφωσης κελιών συγκεντρωτικού πίνακα χρησιμοποιώντας το Aspose.Cells για .NET.

### Πρόσβαση στον Συγκεντρωτικό Πίνακα

Αρχικά, φορτώστε το υπάρχον βιβλίο εργασίας σας που περιέχει τον συγκεντρωτικό πίνακα που θέλετε να τροποποιήσετε.

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sampleFormatPivotTableCells.xlsx");
Worksheet worksheet = workbook.getWorksheets().get(0);
PivotTable pivotTable = worksheet.getPivotTables().get(0);
```

### Εφαρμογή στυλ σε κελιά συγκεντρωτικού πίνακα (H3)

#### Στυλιζάρισμα όλων των κελιών

Δημιουργήστε ένα αντικείμενο στυλ και εφαρμόστε το σε ολόκληρο τον συγκεντρωτικό πίνακα.

```csharp
// Δημιουργήστε ένα νέο στυλ για όλα τα κελιά
Style styleAll = workbook.createStyle();
styleAll.setPattern(BackgroundType.SOLID);
styleAll.setBackgroundColor(Color.LIGHT_BLUE);

pivotTable.formatAll(styleAll);
```

#### Στυλιζάρισμα συγκεκριμένων γραμμών

Για να επισημάνετε συγκεκριμένες γραμμές, δημιουργήστε ένα άλλο στυλ και εφαρμόστε το στα επιλεγμένα κελιά.

```csharp
// Δημιουργήστε ένα νέο στυλ για κελιά γραμμών
Style styleRow = workbook.createStyle();
styleRow.setPattern(BackgroundType.SOLID);
styleRow.setBackgroundColor(Color.YELLOW);

string[] cellsNames = { "H6", "I6", "J6", "K6", "L6", "M6" };

foreach (string cellName in cellsNames) {
    Cell cell = worksheet.getCells().get(cellName);
    pivotTable.format(cell.getRow(), cell.getColumn(), styleRow);
}
```

### Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το στυλιζαρισμένο βιβλίο εργασίας σας στην επιθυμητή θέση.

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outputDir + "/outputFormatPivotTableCells.xlsx");
```
