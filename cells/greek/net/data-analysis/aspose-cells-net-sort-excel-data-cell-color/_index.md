## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε βήμα προς βήμα την ταξινόμηση δεδομένων κατά χρώμα κελιού.

### Δημιουργία και φόρτωση βιβλίου εργασίας

Ξεκινήστε δημιουργώντας μια παρουσία του `Workbook` τάξη και φόρτωση του αρχείου Excel σας:
```csharp
// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας και φορτώστε το αρχείο προτύπου
Workbook workbook = new Workbook(sourceDir + "sampleBackGroundFile.xlsx");
```
Αυτός ο κώδικας αρχικοποιεί ένα νέο βιβλίο εργασίας και φορτώνει δεδομένα από ένα υπάρχον αρχείο Excel που βρίσκεται στον κατάλογο προέλευσης.

### Αρχικοποίηση DataSorter

Στη συνέχεια, δημιουργήστε ένα παράδειγμα του `DataSorter` τάξη για προετοιμασία για τη διαλογή:
```csharp
// Δημιουργία αντικειμένου ταξινόμησης δεδομένων
DataSorter sorter = workbook.DataSorter;
```
Ο `DataSorter` είναι απαραίτητο για τον ορισμό και την εκτέλεση λειτουργιών ταξινόμησης στα δεδομένα σας.

### Προσθήκη κλειδιού ταξινόμησης κατά χρώμα κελιού

Καθορίστε πώς θέλετε να ταξινομηθούν τα δεδομένα. Εδώ, προσθέτουμε ένα κλειδί με βάση το χρώμα του κελιού:
```csharp
// Προσθήκη κλειδιού για τη δεύτερη στήλη για κόκκινο χρώμα
csorter.AddKey(1, SortOnType.CellColor, SortOrder.Descending, Color.Red);
```
Αυτό το βήμα λέει στον ταξινομητή να ιεραρχήσει τις γραμμές όπου τα κελιά στη δεύτερη στήλη έχουν κόκκινο φόντο και να τα ταξινομήσει κατά φθίνουσα σειρά.

### Εκτέλεση της λειτουργίας ταξινόμησης

Αφού ορίσετε τα κλειδιά, εκτελέστε την ταξινόμηση:
```csharp
// Ταξινόμηση δεδομένων με βάση το κλειδί
sorter.Sort(workbook.Worksheets[0].Cells, CellArea.CreateCellArea("A2", "C6"));
```
Αυτή η εντολή ταξινομεί γραμμές εντός της καθορισμένης περιοχής κελιών (από A2 έως C6) με βάση τα κριτήριά μας.

### Αποθήκευση των ταξινομημένων δεδομένων

Τέλος, αποθηκεύστε το ταξινομημένο βιβλίο εργασίας σας:
```csharp
// Αποθήκευση του αρχείου εξόδου
workbook.Save(outputDir + "outputsampleBackGroundFile.xlsx");
```
Ο παραπάνω κώδικας αποθηκεύει τα επεξεργασμένα δεδομένα σε ένα νέο αρχείο Excel στον καθορισμένο κατάλογο εξόδου.
