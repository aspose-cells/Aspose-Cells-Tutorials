## Οδηγός Εφαρμογής

Με το Aspose.Cells για .NET ρυθμισμένο, ας ανανεώσουμε τους ένθετους συγκεντρωτικούς πίνακες βήμα προς βήμα. Αυτό περιλαμβάνει τον εντοπισμό και την ενημέρωση θυγατρικών συγκεντρωτικών πινάκων μέσα σε έναν γονικό πίνακα.

### Φόρτωση του αρχείου Excel
Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο Excel που περιέχει τους συγκεντρωτικούς σας πίνακες:
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook wb = new Workbook(sourceDir + "sampleFindAndRefreshNestedOrChildrenPivotTables.xlsx");
```

### Συγκεντρωτικοί πίνακες της Access στο φύλλο εργασίας
Για να ανανεώσετε τους ένθετους πίνακες, αποκτήστε πρόσβαση στο φύλλο εργασίας και εντοπίστε τον γονικό συγκεντρωτικό πίνακα:
```csharp
Worksheet ws = wb.Worksheets[0];
PivotTable ptParent = ws.PivotTables[2];  // Παράδειγμα: Access τρίτος συγκεντρωτικός πίνακας
```

### Ανανέωση θυγατρικών Συγκεντρωτικών Πινάκων
Αφού εντοπίσετε τον γονικό συγκεντρωτικό πίνακα, ανακτήστε τα παιδιά του και ανανεώστε τα:
```csharp
// Λήψη όλων των θυγατρικών συγκεντρωτικών πινάκων του γονικού πίνακα
PivotTable[] ptChildren = ptParent.GetChildren();

// Κάντε επανάληψη σε κάθε θυγατρικό συγκεντρωτικό πίνακα για να τον ανανεώσετε.
foreach (var ptChild in ptChildren)
{
    ptChild.RefreshData();
    ptChild.CalculateData();  // Διασφαλίζει τον υπολογισμό των ενημερωμένων δεδομένων
}
```
#### Εξήγηση
- **GetChildren()**: Ανακτά όλους τους ένθετους συγκεντρωτικούς πίνακες που βρίσκονται κάτω από τον γονικό πίνακα.
- **RefreshData() & CalculateData()**Ενημερώνει και επανυπολογίζει τα δεδομένα σε κάθε θυγατρικό συγκεντρωτικό πίνακα, διασφαλίζοντας την ακρίβεια.
