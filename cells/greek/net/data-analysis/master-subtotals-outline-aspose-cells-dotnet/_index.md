## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία σε διαχειρίσιμα μέρη για την εφαρμογή μερικών αθροισμάτων και τον έλεγχο της κατεύθυνσης του περιγράμματος.

### Βήμα 1: Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας

Αρχικά, δημιουργήστε μια παρουσία του `Workbook` φορτώνοντας ένα αρχείο Excel και αποκτώντας πρόσβαση στο πρώτο φύλλο εργασίας του:

```csharp
// Δημιουργία βιβλίου εργασίας από το αρχείο προέλευσης του Excel
Workbook workbook = new Workbook(sourceDir + "sampleApplyingSubtotalChangeSummaryDirection.xlsx");

// Πρόσβαση στο πρώτο φύλλο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

### Βήμα 2: Ορισμός περιοχής κελιού για μερικά αθροίσματα

Προσδιορίστε την περιοχή κελιών όπου θέλετε να εφαρμόσετε μερικά αθροίσματα. Εδώ, καθορίζουμε `A2:B11`:

```csharp
// Λήψη της συλλογής Cells στο πρώτο φύλλο εργασίας
Cells cells = worksheet.Cells;

// Δημιουργήστε μια περιοχή κελιών, π.χ. A2:B11
CellArea ca = CellArea.CreateCellArea("A2", "B11");
```

### Βήμα 3: Εφαρμογή μερικών αθροισμάτων

Χρησιμοποιήστε το `Subtotal` μέθοδος για την εφαρμογή μερικών αθροισμάτων, καθορίζοντας στήλες και συναρτήσεις ενοποίησης:

```csharp
// Εφαρμογή μερικού αθροίσματος με τη συνάρτηση Sum στη στήλη B
cells.Subtotal(ca, 0, ConsolidationFunction.Sum, new int[] { 1 }, true, false, true);
```
- **Λειτουργία Ενοποίησης**: Ορίζει την πράξη (π.χ., Άθροισμα).
- **Ευρετήρια στηλών**: Καθορίζει ποιες στήλες θα συμπεριληφθούν.

### Βήμα 4: Ορισμός κατεύθυνσης περιγράμματος

Ελέγξτε πού εμφανίζονται οι γραμμές σύνοψης με το `SummaryRowBelow` ιδιοκτησία:

```csharp
// Ορισμός της κατεύθυνσης της σύνοψης διάρθρωσης
worksheet.Outline.SummaryRowBelow = true;
```

Αυτή η ρύθμιση διασφαλίζει ότι οι γραμμές σύνοψης τοποθετούνται κάτω από τα στοιχεία της ομάδας, βελτιώνοντας την αναγνωσιμότητα.

### Βήμα 5: Αποθήκευση αλλαγών

Τέλος, αποθηκεύστε το τροποποιημένο βιβλίο εργασίας σας σε ένα νέο αρχείο:

```csharp
// Αποθήκευση του αρχείου Excel
workbook.Save(outputDir + "outputApplyingSubtotalChangeSummaryDirection.xlsx");
```
