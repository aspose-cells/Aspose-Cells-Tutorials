# Πώς να δημιουργήσετε και να μορφοποιήσετε Συγκεντρωτικούς Πίνακες χρησιμοποιώντας το Aspose.Cells για .NET: Ένας πλήρης οδηγός

## Οδηγός Εφαρμογής

Ας αναλύσουμε κάθε λειτουργία σε διαχειρίσιμα βήματα.

### Χαρακτηριστικό: Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας (H2)

#### Επισκόπηση

Αυτό το βήμα δημιουργεί ένα νέο βιβλίο εργασίας του Excel και έχει πρόσβαση στο πρώτο φύλλο εργασίας, το οποίο θα ονομάσουμε "Δεδομένα".

**Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας Access First**
```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY";
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
sheet.Name = "Data";
```

### Χαρακτηριστικό: Συμπλήρωση φύλλου εργασίας με δεδομένα (H2)

#### Επισκόπηση

Θα συμπληρώσουμε το φύλλο εργασίας με δείγματα δεδομένων για να δείξουμε πώς μπορούν να χρησιμοποιηθούν οι Συγκεντρωτικοί Πίνακες για ανάλυση.

**Συμπλήρωση κεφαλίδων**
```csharp
Cells cells = sheet.Cells;
cells["A1"].PutValue("Employee");
cells["B1"].PutValue("Quarter");
cells["C1"].PutValue("Product");
cells["D1"].PutValue("Continent");
cells["E1"].PutValue("Country");
cells["F1"].PutValue("Sale");
```

**Προσθήκη Δεδομένων Υπαλλήλου**
```csharp
string[] employees = { "David", "James", "Miya", "Elvis", "Jean", "Ada" };
for (int i = 0; i < employees.Length; i++)
{
    cells[$"A{i + 2}"].PutValue(employees[i]);
}
```

**Προσθήκη δεδομένων τριμήνου, προϊόντος και πωλήσεων**
```csharp
string[] quarters = { "1", "2", "3", "4" };
for (int i = 0; i < 30; i++)
{
    cells[$"B{i + 2}"].PutValue(quarters[i % 4]);
}

string[] products = { /* Λίστα χωρών */ };
for (int i = 0; i < products.Length; i++)
{
    cells[$"E{i + 2}"].PutValue(products[i]);
}

int[] salesData = { 2000, 500, /* Περισσότερα δεδομένα */ };
for (int i = 0; i < salesData.Length; i++)
{
    cells[$"F{i + 2}"].PutValue(salesData[i]);
}
```

### Δυνατότητα: Προσθήκη και ρύθμιση παραμέτρων Συγκεντρωτικού Πίνακα (H2)

#### Επισκόπηση

Αυτή η ενότητα περιλαμβάνει την προσθήκη ενός νέου φύλλου εργασίας για τον Συγκεντρωτικό Πίνακα, τη δημιουργία του και τη διαμόρφωση των ρυθμίσεών του.

**Προσθήκη νέου φύλλου εργασίας για Συγκεντρωτικό πίνακα**
```csharp
Worksheet sheet2 = workbook.Worksheets[workbook.Worksheets.Add()];
sheet2.Name = "PivotTable";
```

**Δημιουργία και ρύθμιση παραμέτρων Συγκεντρωτικού Πίνακα**
```csharp
Aspose.Cells.Pivot.PivotTableCollection pivotTables = sheet2.PivotTables;
int index = pivotTables.Add("=Data!A1:F30", "B3", "PivotTable1");
Aspose.Cells.Pivot.PivotTable pivotTable = pivotTables[index];

pivotTable.RowGrand = true;
pivotTable.ColumnGrand = true;
pivotTable.IsAutoFormat = true;
pivotTable.AutoFormatType = Aspose.Cells.Pivot.PivotTableAutoFormatType.Report6;

pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Row, 0);
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Row, 2);
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Row, 1);
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Column, 3);
pivotTable.AddFieldToArea(Aspose.Cells.Pivot.PivotFieldType.Data, 5);

pivotTable.DataFields[0].NumberFormat = "$#,##0.00";
```

### Αποθήκευση του αρχείου Excel (H2)

Μόλις ρυθμιστεί η διαμόρφωση, αποθηκεύστε το βιβλίο εργασίας σας σε ένα αρχείο εξόδου:

```csharp
workbook.Save(outputDir + "outputCreatePivotTableWithFormatting.xlsx");
```
