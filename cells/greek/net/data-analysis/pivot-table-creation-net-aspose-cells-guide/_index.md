## Οδηγός Εφαρμογής
### Δημιουργία Συγκεντρωτικού Πίνακα με το Aspose.Cells
Ας δούμε βήμα προς βήμα τη δημιουργία ενός Συγκεντρωτικού Πίνακα χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Αρχικοποίηση του βιβλίου εργασίας σας
Αρχικά, δημιουργήστε μια παρουσία του `Workbook` κλάση. Αυτό αντιπροσωπεύει το αρχείο Excel σας:

```csharp
// Δημιουργία στιγμιαίου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```

#### Βήμα 2: Προετοιμασία δεδομένων στο φύλλο εργασίας
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και συμπληρώστε το με τα δεδομένα που είναι απαραίτητα για τον Συγκεντρωτικό Πίνακα σας:

```csharp
// Λήψη της αναφοράς του νέου φύλλου εργασίας που προστέθηκε
Worksheet sheet = workbook.Worksheets[0];
Cells cells = sheet.Cells;

// Ορισμός τιμών στα κελιά
cells["A1"].PutValue("Sport");
cells["B1"].PutValue("Quarter");
cells["C1"].PutValue("Sales");

// Προσθήκη δειγματοληπτικών δεδομένων
string[] sports = { "Golf", "Golf", "Tennis", "Tennis", "Tennis", "Tennis", "Golf" };
string[] quarters = { "Qtr3", "Qtr4", "Qtr3", "Qtr4", "Qtr3", "Qtr4", "Qtr3" };
int[] sales = { 1500, 2000, 600, 1500, 4070, 5000, 6430 };

for (int i = 0; i < sports.Length; i++)
{
    cells[$"A{i + 2}"].PutValue(sports[i]);
cells[$"B{i + 2}"].PutValue(quarters[i]);
cells[$"C{i + 2}"].PutValue(sales[i]);
}
```

#### Βήμα 3: Δημιουργία και ρύθμιση παραμέτρων του Συγκεντρωτικού Πίνακα
Τώρα, προσθέστε έναν Συγκεντρωτικό Πίνακα στο φύλλο εργασίας σας:

```csharp
// Προσθήκη Συγκεντρωτικού Πίνακα στο φύλλο εργασίας
PivotTableCollection pivotTables = sheet.PivotTables;
int index = pivotTables.Add("=A1:C8", "E3", "PivotTable2");

// Πρόσβαση στην παρουσία του πρόσφατα προστεθέντος Συγκεντρωτικού Πίνακα
PivotTable pivotTable = pivotTables[index];

// Ρύθμιση παραμέτρων Συγκεντρωτικού Πίνακα
pivotTable.RowGrand = false; // Απόκρυψη γενικών συνόλων για γραμμές

// Σύρσιμο πεδίων στις κατάλληλες περιοχές
pivotTable.AddFieldToArea(PivotFieldType.Row, 0);   // Αθλητικό γήπεδο στην περιοχή κωπηλασίας
pivotTable.AddFieldToArea(PivotFieldType.Column, 1); // Πεδίο τετάρτου στην περιοχή της στήλης
pivotTable.AddFieldToArea(PivotFieldType.Data, 2);   // Πεδίο πωλήσεων στην περιοχή δεδομένων
```

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να δείτε τα αποτελέσματα:

```csharp
// Αποθήκευση του αρχείου Excel
cells.Workbook.Save("pivotTable_test_out.xls");
```
