## Οδηγός Εφαρμογής

### Δημιουργία και διαμόρφωση ενός Συγκεντρωτικού Πίνακα (H2)
Σε αυτήν την ενότητα, θα δούμε πώς να δημιουργούμε και να ρυθμίζουμε έναν συγκεντρωτικό πίνακα χρησιμοποιώντας το Aspose.Cells.

#### Βήμα 1: Προετοιμασία των δεδομένων σας (H3)
Αρχικά, δημιουργήστε ή ανοίξτε το βιβλίο εργασίας σας τύπου Excel και προσθέστε τα δεδομένα που απαιτούνται για τον συγκεντρωτικό πίνακα:

```csharp
// Δημιουργία ενός νέου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();

// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet sheet = workbook.Worksheets[0];

// Λήψη της συλλογής κελιών του φύλλου εργασίας
Cells cells = sheet.Cells;

// Συμπληρώστε το φύλλο εργασίας με δείγματα δεδομένων πωλήσεων αθλητικών ειδών
cells["A1"].PutValue("Sport");
cells["B1"].PutValue("Quarter");
cells["C1"].PutValue("Sales");

cells["A2"].PutValue("Golf");    cells["B2"].PutValue("Qtr3");  cells["C2"].PutValue(1500);
cells["A3"].PutValue("Golf");    cells["B3"].PutValue("Qtr4");  cells["C3"].PutValue(2000);
cells["A4"].PutValue("Tennis");  cells["B4"].PutValue("Qtr3");  cells["C4"].PutValue(600);
// Συνεχίστε για άλλες καταχωρήσεις...
```

#### Βήμα 2: Προσθήκη του Συγκεντρωτικού Πίνακα (H3)
Στη συνέχεια, προσθέστε έναν συγκεντρωτικό πίνακα στο φύλλο εργασίας σας:

```csharp
PivotTableCollection pivotTables = sheet.PivotTables;

// Προσθήκη νέου Συγκεντρωτικού Πίνακα στο "E3" με βάση την περιοχή δεδομένων "A1:C8"
int index = pivotTables.Add("=A1:C8", "E3", "PivotTable2");

// Πρόσβαση στην πρόσφατα δημιουργημένη παρουσία Συγκεντρωτικού Πίνακα
PivotTable pivotTable = pivotTables[index];

// Ρύθμιση παραμέτρων του Συγκεντρωτικού Πίνακα
pivotTable.RowGrand = false; // Απόκρυψη γενικών συνόλων για γραμμές

// Προσθήκη πεδίων σε διαφορετικές περιοχές του Συγκεντρωτικού Πίνακα
pivotTable.AddFieldToArea(PivotFieldType.Row, 0);   // Αθλητικό γήπεδο προς περιοχή Row
pivotTable.AddFieldToArea(PivotFieldType.Column, 1); // Πεδίο τετάρτου σε περιοχή στήλης
pivotTable.AddFieldToArea(PivotFieldType.Data, 2);   // Πεδίο πωλήσεων προς την περιοχή δεδομένων

// Υπολογισμός δεδομένων για τον Συγκεντρωτικό Πίνακα
pivotTable.CalculateData();
```

#### Βήμα 3: Αποθήκευση ως αρχείο ODS (H3)
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε μορφή ODS:

```csharp
string outputDir = "your/output/directory/";
workbook.Save(outputDir + "PivotTableSaveInODS_out.ods");
Console.WriteLine("PivotTableSaveInODS executed successfully.");
```
