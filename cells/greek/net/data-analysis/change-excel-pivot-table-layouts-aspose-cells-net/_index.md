**Χρήση του Διαχειριστή Πακέτων:**

## Οδηγός Εφαρμογής
Αυτή η ενότητα καλύπτει τον τρόπο αλλαγής των διατάξεων Συγκεντρωτικού Πίνακα χρησιμοποιώντας το Aspose.Cells .NET.

### Αλλαγή διάταξης σε συμπαγή μορφή
Η συμπαγής μορφή είναι ιδανική για γρήγορες επισκοπήσεις. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Βήμα 1: Φόρτωση του αρχείου Excel
```csharp
// Φόρτωση ενός υπάρχοντος βιβλίου εργασίας
Workbook workbook = new Workbook("sampleChangingLayoutOfPivotTable.xlsx");
```

#### Βήμα 2: Πρόσβαση στον Συγκεντρωτικό Πίνακα
```csharp
Worksheet worksheet = workbook.Worksheets[0];
PivotTable pivotTable = worksheet.PivotTables[0];
```

#### Βήμα 3: Ορισμός Συμπυκνωμένης Φόρμας και Ανανέωση Δεδομένων
```csharp
// Αλλαγή σε συμπαγή μορφή
pivotTable.ShowInCompactForm();

// Ανανέωση δεδομένων για εφαρμογή αλλαγών
pivotTable.RefreshData();
pivotTable.CalculateData();

// Αποθήκευση του βιβλίου εργασίας
workbook.Save("outputChangingLayoutOfPivotTable_CompactForm.xlsx");
```

### Αλλαγή διάταξης σε μορφή περιγράμματος
Η φόρμα διάρθρωσης επεκτείνει τον Συγκεντρωτικό Πίνακα σας για λεπτομερή ανάλυση.

#### Βήμα 1: Πρόσβαση και διαμόρφωση
```csharp
// Αλλαγή σε μορφή περιγράμματος
pivotTable.ShowInOutlineForm();

// Ανανέωση δεδομένων για εφαρμογή αλλαγών
pivotTable.RefreshData();
pivotTable.CalculateData();

// Αποθήκευση του βιβλίου εργασίας
workbook.Save("outputChangingLayoutOfPivotTable_OutlineForm.xlsx");
```

### Αλλαγή διάταξης σε μορφή πίνακα
Για μια παραδοσιακή προβολή τύπου πίνακα, χρησιμοποιήστε τη μορφή πίνακα.

#### Βήμα 1: Ορισμός και ανανέωση
```csharp
// Αλλαγή σε μορφή πίνακα
pivotTable.ShowInTabularForm();

// Ανανέωση δεδομένων για εφαρμογή αλλαγών
pivotTable.RefreshData();
pivotTable.CalculateData();

// Αποθήκευση του βιβλίου εργασίας
workbook.Save("outputChangingLayoutOfPivotTable_TabularForm.xlsx");
```
