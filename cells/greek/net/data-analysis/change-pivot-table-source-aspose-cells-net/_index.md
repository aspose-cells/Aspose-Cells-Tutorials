## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει το περιβάλλον, ας αλλάξουμε τα δεδομένα προέλευσης για έναν συγκεντρωτικό πίνακα.

### Επισκόπηση

Αυτή η ενότητα σας καθοδηγεί στην τροποποίηση των δεδομένων προέλευσης ενός υπάρχοντος συγκεντρωτικού πίνακα σε ένα αρχείο Excel. Θα φορτώσουμε το βιβλίο εργασίας, θα αποκτήσουμε πρόσβαση στα φύλλα εργασίας του, θα ενημερώσουμε συγκεκριμένα κελιά με νέα δεδομένα και θα αποθηκεύσουμε τις αλλαγές.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το αρχείο Excel σε ένα `Workbook` αντικείμενο:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
string InputPath = dataDir + "Book1.xlsx";

// Δημιουργία FileStream για το αρχείο Excel
FileStream fstream = new FileStream(InputPath, FileMode.Open);

// Άνοιγμα του αρχείου Excel χρησιμοποιώντας το FileStream
Workbook workbook = new Workbook(fstream);
```

#### Βήμα 2: Πρόσβαση και τροποποίηση δεδομένων

Αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει το εύρος δεδομένων του συγκεντρωτικού σας πίνακα. Ενημερώστε το με νέες τιμές, όπως απαιτείται:

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Worksheet worksheet = workbook.Worksheets[0];

// Ενημέρωση κελιών με νέα δεδομένα για την πηγή pivot
worksheet.Cells["A9"].PutValue("Golf");
worksheet.Cells["B9"].PutValue("Qtr4");
worksheet.Cells["C9"].PutValue(7000);
```

#### Βήμα 3: Ενημέρωση ονομασμένης περιοχής

Τροποποιήστε το ονομασμένο εύρος ώστε να αντικατοπτρίζει τα ενημερωμένα δεδομένα σας:

```csharp
// Ενημέρωση του εύρους με όνομα "DataSource"
Range range = worksheet.Cells.CreateRange(0, 0, 9, 3);
range.Name = "DataSource";
```

#### Βήμα 4: Αποθήκευση αλλαγών

Τέλος, αποθηκεύστε το βιβλίο εργασίας με τα ενημερωμένα δεδομένα προέλευσης:

```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
workbook.Save(dataDir + "output.xls");

// Κλείσιμο του FileStream για δωρεάν πόρους
fstream.Close();
```
