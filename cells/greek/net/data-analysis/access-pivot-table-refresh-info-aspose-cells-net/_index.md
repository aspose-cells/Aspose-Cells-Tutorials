## Οδηγός Εφαρμογής

### Πρόσβαση σε πληροφορίες ανανέωσης συγκεντρωτικού πίνακα

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να ανακτάτε μέσω προγραμματισμού ποιος ανανέωσε τελευταία φορά έναν συγκεντρωτικό πίνακα και πότε ανανεώθηκε, παρέχοντας πολύτιμες πληροφορίες σχετικά με την ακεραιότητα των δεδομένων σας.

#### Ρύθμιση του έργου σας
1. **Φόρτωση του βιβλίου εργασίας:**
   Φορτώστε ένα βιβλίο εργασίας Excel που περιέχει τον συγκεντρωτικό πίνακα προορισμού σας χρησιμοποιώντας το `Workbook` τάξη.
   ```csharp
   Workbook workbook = new Workbook("sourcePivotTable.xlsx");
   ```
2. **Πρόσβαση στο Φύλλο εργασίας και στον Συγκεντρωτικό Πίνακα:**
   Αποκτήστε πρόσβαση στο φύλλο εργασίας και, στη συνέχεια, στον συγκεκριμένο συγκεντρωτικό πίνακα που περιέχει.
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   PivotTable pivotTable = worksheet.PivotTables[0];
   ```
3. **Ανάκτηση πληροφοριών ανανέωσης:**
   Χρήση `RefreshedByWho` και `RefreshDate` για να λάβετε λεπτομερείς πληροφορίες ανανέωσης.
   ```csharp
   string refreshByWho = pivotTable.RefreshedByWho;
   DateTime refreshDate = pivotTable.RefreshDate;
   
   Console.WriteLine("Pivot table refreshed by: " + refreshByWho);
   Console.WriteLine("Last refresh date: " + refreshDate);
   ```

#### Εξήγηση
- **`RefreshedByWho`:** Επιστρέφει το όνομα χρήστη του ατόμου που ανανέωσε τελευταία φορά τον συγκεντρωτικό πίνακα.
- **`RefreshDate`:** Παρέχει τη χρονική σήμανση για την τελευταία ενημέρωση του συγκεντρωτικού πίνακα.
