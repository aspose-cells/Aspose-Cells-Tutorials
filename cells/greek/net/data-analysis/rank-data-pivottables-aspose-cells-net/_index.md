## Οδηγός Εφαρμογής

### Επισκόπηση της κατάταξης σε Συγκεντρωτικούς Πίνακες

Αυτή η λειτουργία σάς επιτρέπει να κατατάξετε δεδομένα μέσα σε έναν συγκεντρωτικό πίνακα, παρέχοντας πληροφορίες σχετικά με τη σχετική τοποθέτηση των τιμών από τη μεγαλύτερη στη μικρότερη.

#### Φόρτωση και πρόσβαση στο βιβλίο εργασίας

Αρχικά, φορτώστε ένα υπάρχον αρχείο Excel που περιέχει τον συγκεντρωτικό σας πίνακα:

```csharp
// Κατάλογοι για αρχεία προέλευσης και εξόδου
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// Φόρτωση βιβλίου εργασίας με πρότυπο Συγκεντρωτικού Πίνακα
Workbook workbook = new Workbook(sourceDir + "PivotTableSample.xlsx");
```

#### Πρόσβαση στον Συγκεντρωτικό Πίνακα

Αποκτήστε πρόσβαση στον συγκεκριμένο συγκεντρωτικό πίνακα στον οποίο θέλετε να εφαρμόσετε την κατάταξη:

```csharp
// Λήψη του πρώτου φύλλου εργασίας που περιέχει τον Συγκεντρωτικό Πίνακα
Worksheet worksheet = workbook.Worksheets[0];

// Ας υποθέσουμε ότι ο Συγκεντρωτικός Πίνακας βρίσκεται στο ευρετήριο 0
int pivotIndex = 0;
PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
```

#### Ρύθμιση παραμέτρων μορφής εμφάνισης δεδομένων

Ρυθμίστε την κατάταξη των πεδίων δεδομένων στον συγκεντρωτικό σας πίνακα:

```csharp
// Πρόσβαση στη συλλογή πεδίων δεδομένων από τον Συγκεντρωτικό Πίνακα
PivotFieldCollection pivotFields = pivotTable.DataFields;

// Λήψη του πρώτου πεδίου δεδομένων για την εφαρμογή μορφοποίησης κατάταξης
PivotField pivotField = pivotFields[0];

// Ορίστε τη μορφή εμφάνισης για την κατάταξη από το μεγαλύτερο στο μικρότερο
pivotField.DataDisplayFormat = PivotFieldDataDisplayFormat.RankLargestToSmallest;
```

#### Αποθήκευση αλλαγών

Μετά τη διαμόρφωση, αποθηκεύστε το βιβλίο εργασίας σας:

```csharp
// Υπολογισμός δεδομένων και αποθήκευση του βιβλίου εργασίας με τις αλλαγές
pivotTable.CalculateData();
workbook.Save(outputDir + "PivotTableDataDisplayFormatRanking_out.xlsx");
```
