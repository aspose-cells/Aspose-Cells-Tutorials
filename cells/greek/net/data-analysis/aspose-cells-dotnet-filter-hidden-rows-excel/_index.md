### Εφαρμογή αυτόματων φίλτρων σε αρχεία Excel χρησιμοποιώντας C#

#### Επισκόπηση
Αυτή η ενότητα εστιάζει στη φόρτωση ενός αρχείου Excel και στην εφαρμογή ενός αυτόματου φίλτρου. Στη συνέχεια, θα ανακτήσουμε τους δείκτες των γραμμών που είναι κρυμμένοι μετά την ανανέωση του φίλτρου.

#### Βήματα

**Βήμα 1: Φόρτωση του αρχείου Excel**

```csharp
// Ορίστε τον κατάλογο προέλευσης και φορτώστε το δείγμα αρχείου Excel
string sourceDir = "PathToYourDirectory\\";
Workbook wb = new Workbook(sourceDir + "sampleGetAllHiddenRowsIndicesAfterRefreshingAutoFilter.xlsx");
```

- **Εξήγηση**: Εδώ, αρχικοποιούμε ένα `Workbook` αντικείμενο με τη διαδρομή προς το δείγμα αρχείου Excel.

**Βήμα 2: Πρόσβαση και εφαρμογή αυτόματου φίλτρου**

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet ws = wb.Worksheets[0];

// Εφαρμογή αυτόματου φίλτρου στον δείκτη στήλης 0 (πρώτη στήλη)
ws.AutoFilter.AddFilter(0, "Orange");
```

- **Εξήγηση**: Αποκτούμε πρόσβαση στο πρώτο φύλλο εργασίας και εφαρμόζουμε ένα φίλτρο για να εμφανίζουμε μόνο γραμμές όπου η πρώτη στήλη περιέχει την τιμή "Πορτοκαλί".

**Βήμα 3: Ανανέωση αυτόματου φίλτρου και ανάκτηση κρυφών γραμμών**

```csharp
// Ανανέωση αυτόματου φίλτρου και λήψη ευρετηρίων κρυφών γραμμών
int[] rowIndices = ws.AutoFilter.Refresh(true);

Console.WriteLine("Printing Rows Indices, Cell Names, and Values Hidden By AutoFilter.");
```

- **Εξήγηση**: Το `Refresh(true)` Η μέθοδος ενημερώνει το φίλτρο και επιστρέφει έναν πίνακα δεικτών γραμμών που είναι κρυμμένοι λόγω του φίλτρου.

**Βήμα 4: Εκτύπωση λεπτομερειών κρυφής γραμμής**

```csharp
for (int i = 0; i < rowIndices.Length; i++)
{
    int r = rowIndices[i];
    Cell cell = ws.Cells[r, 0];
    Console.WriteLine($"{r}\t{cell.Name}\t{cell.StringValue}");
}
```

- **Εξήγηση**: Περιηγηθείτε στους κρυφούς δείκτες γραμμών και εκτυπώστε λεπτομέρειες όπως τον δείκτη γραμμής, το όνομα κελιού και την τιμή.
