## Οδηγός Εφαρμογής

Θα αναλύσουμε αυτήν την υλοποίηση σε βήματα για να σας βοηθήσουμε να κατανοήσετε κάθε μέρος της διαδικασίας.

### Βήμα 1: Δημιουργήστε μια κλάση προσαρμοσμένων ρυθμίσεων
Αρχικά, δημιουργήστε μια προσαρμοσμένη κλάση που επεκτείνεται `GlobalizationSettings`:
```csharp
class CustomSettings : GlobalizationSettings
{
    public override string GetTotalName(ConsolidationFunction functionType)
    {
        switch (functionType)
        {
            case ConsolidationFunction.Average:
                return "AVG";
            default:
                return base.GetTotalName(functionType);
        }
    }

    public override string GetGrandTotalName(ConsolidationFunction functionType)
    {
        switch (functionType)
        {
            case ConsolidationFunction.Average:
                return "GRD AVG";
            default:
                return base.GetGrandTotalName(functionType);
        }
    }
}
```
**Εξήγηση:** Αυτή η κλάση προσαρμόζει τον τρόπο με τον οποίο ονομάζονται τα μερικά αθροίσματα για διαφορετικές συναρτήσεις, όπως ο Μέσος Όρος.

### Βήμα 2: Φόρτωση του βιβλίου εργασίας σας
Φορτώστε το υπάρχον βιβλίο εργασίας του Excel που περιέχει τα δεδομένα που θέλετε να χειριστείτε:
```csharp
Workbook book = new Workbook("sampleCustomLabelsSubtotals.xlsx");
```
**Εξήγηση:** Αντικαθιστώ `"sampleCustomLabelsSubtotals.xlsx"` με τη διαδρομή του αρχείου σας. Αυτό αρχικοποιεί το `Workbook` αντικείμενο.

### Βήμα 3: Ορισμός ρυθμίσεων προσαρμοσμένης παγκοσμιοποίησης
Αντιστοιχίστε τις προσαρμοσμένες ρυθμίσεις μας στο βιβλίο εργασίας:
```csharp
book.Settings.GlobalizationSettings = new CustomSettings();
```
**Εξήγηση:** Αυτό διασφαλίζει ότι τυχόν υπολογισμοί μερικού συνόλου χρησιμοποιούν τις προσαρμοσμένες ετικέτες μας από `CustomSettings`.

### Βήμα 4: Προσθήκη λειτουργικότητας μερικού αθροίσματος
Προσθέστε ένα μερικό άθροισμα στο φύλλο εργασίας σας εντός ενός καθορισμένου εύρους χρησιμοποιώντας τη συνάρτηση μέσου όρου:
```csharp
Worksheet sheet = book.Worksheets[0];
sheet.Cells.Subtotal(CellArea.CreateCellArea("A2", "B9"), 0, ConsolidationFunction.Average, new int[] { 1 });
```
**Εξήγηση:** Αυτό στοχεύει κελιά από το A2 έως το B9 και προσθέτει ένα μέσο μερικό άθροισμα με βάση την πρώτη στήλη (δείκτης 1).

### Βήμα 5: Υπολογισμός τύπων και προσαρμογή στηλών
Αφού προσθέσετε μερικά αθροίσματα, υπολογίστε τυχόν τύπους και στήλες αυτόματης προσαρμογής:
```csharp
book.CalculateFormula();
sheet.AutoFitColumns();
```
**Εξήγηση:** `CalculateFormula()` διασφαλίζει ότι όλοι οι υπολογισμοί είναι ενημερωμένοι. `AutoFitColumns()` προσαρμόζει το πλάτος της στήλης ώστε να ταιριάζει στο περιεχόμενο.

### Βήμα 6: Αποθήκευση του βιβλίου εργασίας σας
Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:
```csharp
book.Save("outputCustomLabelsSubtotals.xlsx");
```
**Εξήγηση:** Αυτό αποθηκεύει το τροποποιημένο βιβλίο εργασίας σας με προσαρμοσμένα μερικά αθροίσματα και προσαρμοσμένες στήλες.
