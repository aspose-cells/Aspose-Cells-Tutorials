## Οδηγός Εφαρμογής
Εξερευνήστε κάθε χαρακτηριστικό λεπτομερώς παρακάτω.

### Φόρτωση και πρόσβαση σε Συγκεντρωτικό Πίνακα
#### Επισκόπηση
Η πρόσβαση σε έναν Συγκεντρωτικό Πίνακα είναι απαραίτητη για τον χειρισμό δεδομένων. Δείτε πώς μπορείτε να φορτώσετε ένα αρχείο Excel και να ανακτήσετε έναν συγκεκριμένο Συγκεντρωτικό Πίνακα.

#### Βήμα-βήμα
**1. Φορτώστε το Βιβλίο Εργασίας:**
   ```csharp
   using Aspose.Cells;
   using Aspose.Cells.Pivot;
   
   string sourceDir = "YOUR_SOURCE_DIRECTORY";
   Workbook workbook = new Workbook(sourceDir + "Book1.xls");
   ```
**2. Πρόσβαση σε ένα φύλλο εργασίας και έναν Συγκεντρωτικό πίνακα:**
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   int pivotIndex = 0;
   PivotTable pivotTable = worksheet.PivotTables[pivotIndex];
   ```
### Ορισμός Γενικών Συνόλων και Μερικών Συνόλων Σειρών
#### Επισκόπηση
Η ρύθμιση παραμέτρων γενικών και μερικών αθροισμάτων γραμμών διασφαλίζει την αποτελεσματική σύνοψη δεδομένων.

#### Βήμα-βήμα
**1. Πεδία Γραμμής Πρόσβασης:**
   ```csharp
   PivotFieldCollection pivotFields = pivotTable.RowFields;
   PivotField pivotField = pivotFields[0];
   ```
**2. Διαμόρφωση Συνόλων και Μερικών Συνόλων:**
   ```csharp
   // Ενεργοποίηση γενικών συνόλων
   pivotTable.RowGrand = true;

   // Ορισμός μερικών αθροισμάτων για το άθροισμα και το πλήθος
   pivotField.SetSubtotals(PivotFieldSubtotalType.Sum, true);
   pivotField.SetSubtotals(PivotFieldSubtotalType.Count, true);
   ```
### Ρύθμιση παραμέτρων επιλογών αυτόματης ταξινόμησης
#### Επισκόπηση
Η αυτόματη ταξινόμηση οργανώνει τα δεδομένα δυναμικά. Δείτε πώς μπορείτε να διαμορφώσετε αυτήν τη λειτουργία.

#### Βήμα-βήμα
**1. Ενεργοποίηση αυτόματης ταξινόμησης:**
   ```csharp
   PivotField pivotField = pivotTable.RowFields[0];
   pivotField.IsAutoSort = true;
   pivotField.IsAscendSort = true; // Ορισμός σειράς ταξινόμησης σε αύξουσα
   ```
**2. Ορίστε τον Δείκτη Πεδίου Ταξινόμησης:**
   ```csharp
   pivotField.AutoSortField = -5;
   ```
### Ρύθμιση παραμέτρων επιλογών αυτόματης εμφάνισης
#### Επισκόπηση
Η λειτουργία αυτόματης εμφάνισης εμφανίζει αυτόματα μόνο τα σχετικά δεδομένα.

#### Βήμα-βήμα
**1. Ενεργοποιήστε τις Ρυθμίσεις Αυτόματης Εμφάνισης:**
   ```csharp
   PivotField pivotField = pivotTable.RowFields[0];
   pivotField.IsAutoShow = true;
   ```
**2. Ρύθμιση παραμέτρων εμφάνισης συνθηκών:**
   ```csharp
   pivotField.AutoShowField = 0; // Με βάση ένα συγκεκριμένο δείκτη πεδίου δεδομένων
   ```
### Αποθήκευση του αρχείου Excel
#### Επισκόπηση
Αφού κάνετε τις αλλαγές, αποθηκεύστε ξανά το βιβλίο εργασίας σας στον δίσκο.

#### Βήμα-βήμα
**1. Αποθήκευση βιβλίου εργασίας:**
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "output.xls");
   ```
