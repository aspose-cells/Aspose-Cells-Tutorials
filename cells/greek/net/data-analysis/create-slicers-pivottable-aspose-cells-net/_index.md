## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει τα πάντα, ας υλοποιήσουμε αναλυτές σε έναν συγκεντρωτικό πίνακα χρησιμοποιώντας το Aspose.Cells για .NET.

### Φόρτωση και πρόσβαση στο βιβλίο εργασίας

Αρχικά, φορτώστε το αρχείο Excel που περιέχει τον συγκεντρωτικό πίνακα:

```csharp
// Διαδρομή καταλόγου πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Φόρτωση του βιβλίου εργασίας
Workbook wb = new Workbook(sourceDir + "sampleCreateSlicerToPivotTable.xlsx");
```

#### Πρόσβαση σε φύλλα εργασίας και συγκεντρωτικούς πίνακες

Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας και στον συγκεντρωτικό πίνακα:

```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet ws = wb.Worksheets[0];

// Πρόσβαση στον πρώτο συγκεντρωτικό πίνακα στο φύλλο εργασίας
Aspose.Cells.Pivot.PivotTable pt = ws.PivotTables[0];
```

### Προσθήκη ενός αναλυτή στον Συγκεντρωτικό Πίνακα

Τώρα, προσθέστε έναν αναλυτή που σχετίζεται με τον συγκεντρωτικό σας πίνακα:

```csharp
// Προσθήκη αναλυτή στο κελί B22 με το πρώτο βασικό πεδίο του συγκεντρωτικού πίνακα
int idx = ws.Slicers.Add(pt, "B22", pt.BaseFields[0]);

// Αποκτήστε πρόσβαση στον πρόσφατα προστιθέμενο αναλυτή από τη συλλογή αναλυτών
Aspose.Cells.Slicers.Slicer slicer = ws.Slicers[idx];
```

#### Εξήγηση:
- **`ws.Slicers.Add()`**Αυτή η μέθοδος προσθέτει έναν αναλυτή στο φύλλο εργασίας. 
  - `pt`: Το αντικείμενο του συγκεντρωτικού πίνακα.
  - "B22": Θέση όπου θα τοποθετηθεί ο κόφτης.
  - `pt.BaseFields[0]`: Το βασικό πεδίο που χρησιμοποιείται από τον αναλυτή.

### Αποθήκευση του βιβλίου εργασίας σας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας στις επιθυμητές μορφές:

```csharp
// Ορισμός διαδρομής καταλόγου εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση ως μορφή XLSX
wb.Save(outputDir + "outputCreateSlicerToPivotTable.xlsx", SaveFormat.Xlsx);

// Αποθήκευση ως μορφή XLSB
wb.Save(outputDir + "outputCreateSlicerToPivotTable.xlsb", SaveFormat.Xlsb);
```
