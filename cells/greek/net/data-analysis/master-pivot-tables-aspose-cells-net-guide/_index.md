# Κύριοι Συγκεντρωτικοί Πίνακες στο .NET χρησιμοποιώντας το Aspose.Cells: Ένας Πλήρης Οδηγός

## Εισαγωγή

Θέλετε να διαχειριστείτε και να αναλύσετε μεγάλα σύνολα δεδομένων πιο αποτελεσματικά; Οι συγκεντρωτικοί πίνακες είναι ένα ισχυρό εργαλείο που μπορεί να μετατρέψει τα ακατέργαστα δεδομένα σε διορατικές περιλήψεις, αλλά η διαμόρφωσή τους στις εφαρμογές σας μπορεί να είναι δύσκολη. Αυτό το σεμινάριο θα σας καθοδηγήσει στη δημιουργία και προσαρμογή συγκεντρωτικών πινάκων χρησιμοποιώντας το Aspose.Cells για .NET, καθιστώντας τις εργασίες ανάλυσης δεδομένων σας απρόσκοπτες και αποτελεσματικές.

### Τι θα μάθετε
- **Δημιουργία νέου φύλλου εργασίας:** Κατανοήστε τον τρόπο αρχικοποίησης και δημιουργίας νέων φύλλων εργασίας μέσα στο βιβλίο εργασίας σας.
- **Προσθήκη και ρύθμιση παραμέτρων ενός Συγκεντρωτικού Πίνακα:** Μάθετε τα βήματα για να προσθέσετε έναν συγκεντρωτικό πίνακα και να διαμορφώσετε τα πεδία του για βέλτιστη παρουσίαση δεδομένων.
- **Προσαρμογή ρυθμίσεων Συγκεντρωτικού Πίνακα:** Ανακαλύψτε πώς να προσαρμόσετε ρυθμίσεις όπως τα μερικά αθροίσματα και τα γενικά αθροίσματα για να προσαρμόσετε το αποτέλεσμα στις ανάγκες σας.
- **Ανανέωση και Υπολογισμός Δεδομένων:** Λάβετε πληροφορίες σχετικά με την ανανέωση και τον επανυπολογισμό των συγκεντρωτικών πινάκων ώστε να αντικατοπτρίζουν τα πιο πρόσφατα δεδομένα.
- **Προσαρμογή θέσεων αντικειμένων:** Μάθετε να τροποποιείτε τις θέσεις των στοιχείων μέσα σε συγκεντρωτικούς πίνακες για καλύτερη οργάνωση και σαφήνεια.

Ας ξεκινήσουμε ρυθμίζοντας το περιβάλλον σας, διασφαλίζοντας ότι έχετε όλα όσα χρειάζεστε για να ακολουθήσετε αποτελεσματικά αυτόν τον οδηγό.

## Προαπαιτούμενα
Για να ξεκινήσετε τη δημιουργία και τη ρύθμιση παραμέτρων συγκεντρωτικών πινάκων χρησιμοποιώντας το Aspose.Cells για .NET, βεβαιωθείτε ότι έχετε τα εξής:

- **Aspose.Cells για βιβλιοθήκη .NET:** Βεβαιωθείτε ότι έχετε εγκαταστήσει την έκδοση 22.10 ή νεότερη.
- **Περιβάλλον Ανάπτυξης:** Χρησιμοποιήστε ένα περιβάλλον ανάπτυξης C# όπως το Visual Studio.
- **Βασικές γνώσεις C#:** Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε και να εφαρμόσετε τα παρεχόμενα αποσπάσματα κώδικα.

## Ρύθμιση του Aspose.Cells για .NET

### Εγκατάσταση
Ενσωματώστε το Aspose.Cells στο έργο σας χρησιμοποιώντας είτε το .NET CLI είτε την Κονσόλα Διαχείρισης Πακέτων στο Visual Studio:

**.NET CLI**
```bash
dotnet add package Aspose.Cells
```

**Κονσόλα διαχείρισης πακέτων**
```powershell
PM> NuGet\Install-Package Aspose.Cells
```

### Απόκτηση Άδειας
- **Δωρεάν δοκιμή:** Ξεκινήστε με μια δωρεάν δοκιμαστική περίοδο 30 ημερών για να εξερευνήσετε όλες τις λειτουργίες.
- **Προσωρινή Άδεια:** Ζητήστε προσωρινή άδεια χρήσης για εκτεταμένες δοκιμές πριν από την αγορά.
- **Αγορά:** Αν διαπιστώσετε ότι η βιβλιοθήκη ταιριάζει στις ανάγκες σας, προχωρήστε στην αγορά μιας συνδρομής.

Μετά την εγκατάσταση, αρχικοποιήστε το Aspose.Cells στο έργο σας ως εξής:
```csharp
using Aspose.Cells;
```

## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη ενός Συγκεντρωτικού Πίνακα
#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να δημιουργήσετε ένα νέο φύλλο εργασίας και να προσθέσετε έναν συγκεντρωτικό πίνακα. Θα διαμορφώσουμε τα απαραίτητα πεδία για την αναπαράσταση δεδομένων.

**Βήμα 1: Αρχικοποίηση βιβλίου εργασίας**
Δημιουργήστε ένα `Workbook` αντικείμενο καθορίζοντας τον κατάλογο πηγής σας.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook wb = new Workbook(SourceDir + "/sampleSpecifyAbsolutePositionOfPivotItem.xlsx");
```

**Βήμα 2: Προσθήκη νέου φύλλου εργασίας**
Προσθέστε ένα νέο φύλλο εργασίας και προετοιμάστε το για τον συγκεντρωτικό πίνακα.
```csharp
Worksheet wsPivot = wb.Worksheets.Add("pvtNew Hardware");
Worksheet wsData = wb.Worksheets["New Hardware - Yearly"];
```

**Βήμα 3: Δημιουργία Συγκεντρωτικού Πίνακα**
Προσθέστε έναν συγκεντρωτικό πίνακα στο νέο φύλλο εργασίας σας, καθορίζοντας τα εύρη προέλευσης και προορισμού δεδομένων.
```csharp
PivotTableCollection pivotTables = wsPivot.PivotTables;
int index = pivotTables.Add("='New Hardware - Yearly'!A1:D621", "A3", "HWCounts_PivotTable");
PivotTable pvtTable = pivotTables[index];
```

**Βήμα 4: Ρύθμιση παραμέτρων πεδίων Συγκεντρωτικού Πίνακα**
Προσθέστε πεδία στον συγκεντρωτικό πίνακα για γραμμές και δεδομένα.
```csharp
pvtTable.AddFieldToArea(PivotFieldType.Row, "Vendor");
pvtTable.AddFieldToArea(PivotFieldType.Row, "Item");
pvtTable.AddFieldToArea(PivotFieldType.Data, "2014");
```

### Ρύθμιση παραμέτρων συγκεντρωτικού πίνακα
#### Επισκόπηση
Βελτιστοποιήστε τον συγκεντρωτικό σας πίνακα απενεργοποιώντας τα μερικά αθροίσματα και τα γενικά αθροίσματα.

**Βήμα 1: Απενεργοποίηση μερικών αθροισμάτων**
Απενεργοποιήστε τα μερικά αθροίσματα για συγκεκριμένα πεδία, όπως απαιτείται.
```csharp
PivotField pivotField = pvtTable.RowFields["Vendor"];
pivotField.SetSubtotals(PivotFieldSubtotalType.None, true);
```

**Βήμα 2: Απενεργοποίηση Γενικών Συνόλων**
Απενεργοποιήστε τα γενικά σύνολα για να βελτιστοποιήσετε την παρουσίαση δεδομένων.
```csharp
pvtTable.ColumnGrand = false;
```

### Ανανέωση και υπολογισμός δεδομένων για τον Συγκεντρωτικό Πίνακα
#### Επισκόπηση
Βεβαιωθείτε ότι ο συγκεντρωτικός σας πίνακας αντικατοπτρίζει τα πιο ενημερωμένα δεδομένα, ανανεώνοντάς τον και υπολογίζοντάς τον εκ νέου.

**Βήμα 1: Ανανέωση δεδομένων**
Καλέστε τη συνάρτηση ανανέωσης για να ενημερώσετε τον συγκεντρωτικό πίνακα με νέα δεδομένα.
```csharp
pvtTable.RefreshData();
```

**Βήμα 2: Υπολογισμός δεδομένων**
Υπολογίστε τα ενημερωμένα δεδομένα για να αντικατοπτρίσετε με ακρίβεια τις αλλαγές στον συγκεντρωτικό πίνακα.
```csharp
pvtTable.CalculateData();
```

### Προσαρμογή απόλυτης θέσης στοιχείων περιστροφής
#### Επισκόπηση
Αναδιοργανώστε τα στοιχεία μέσα στον συγκεντρωτικό σας πίνακα για σαφήνεια και τάξη.

**Βήμα 1: Ορισμός θέσεων αντικειμένων**
Προσαρμόστε τις θέσεις για να εξασφαλίσετε μια λογική ακολουθία των στοιχείων.
```csharp
pvtTable.RowFields["Item"].PivotItems["4H12"].PositionInSameParentNode = 0;
pvtTable.RowFields["Item"].PivotItems["DIF400"].PositionInSameParentNode = 3;

pvtTable.CalculateData();

pvtTable.RowFields["Item"].PivotItems["CA32"].PositionInSameParentNode = 1;
pvtTable.RowFields["Item"].PivotItems["AAA3"].PositionInSameParentNode = 2;
```

### Αποθήκευση του βιβλίου εργασίας με αλλαγές
#### Επισκόπηση
Αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε όλες τις αλλαγές που έγιναν στον Συγκεντρωτικό Πίνακα.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
wb.Save(outputDir + "/outputSpecifyAbsolutePositionOfPivotItem.xlsx");
```
