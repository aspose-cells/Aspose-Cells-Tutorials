### Αναζήτηση δεδομένων σε υπολογιστικά φύλλα Excel
Για να βρείτε συγκεκριμένα δεδομένα σε ένα φύλλο εργασίας του Excel, θα χρησιμοποιήσετε το `FindOptions` κλάση για να ορίσετε τις παραμέτρους αναζήτησής σας. Ακολουθεί μια αναλυτική περιγραφή:

#### Βήμα 1: Φόρτωση και Υπολογισμός Τύπων
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας σας και υπολογίζοντας τυχόν τύπους που ενδέχεται να επηρεάσουν τις τιμές των κελιών.

```csharp
Workbook workbook = new Workbook("sampleFindingDataOrFormulasUsingFindOptions.xlsx");
workbook.CalculateFormula();
```

#### Βήμα 2: Πρόσβαση στη Συλλογή Κελιών
Ανακτήστε τη συλλογή κελιών από το φύλλο εργασίας όπου θέλετε να εκτελέσετε την αναζήτηση:

```csharp
Cells cells = workbook.Worksheets[0].Cells;
```

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών εύρεσης
Ρυθμίστε το δικό σας `FindOptions` αντικείμενο, καθορίζοντας το εύρος και τον τύπο των δεδομένων που αναζητάτε.

```csharp
FindOptions findOptions = new FindOptions();

// Ορίστε μια περιοχή αναζήτησης μέσα στο φύλλο εργασίας
CellArea ca = new CellArea();
ca.StartRow = 8;
ca.EndRow = 17;
currentColumn = 2;
a.EndColumn = 13;

findOptions.SetRange(ca);
findOptions.SearchBackward = false;
findOptions.SearchOrder = SearchOrder.ByRows;
findOptions.LookInType = LookInType.Values;
findOptions.LookAtType = LookAtType.EntireContent;
```

#### Βήμα 4: Εκτελέστε τη λειτουργία εύρεσης
Χρησιμοποιήστε το `Find` μέθοδος για την αναζήτηση συγκεκριμένων τιμών εντός του καθορισμένου εύρους:

```csharp
Cell cell = cells.Find(341, null, findOptions);

if (cell != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell.Name);
}
else
{
    Console.WriteLine("Record not found.");
}
```
