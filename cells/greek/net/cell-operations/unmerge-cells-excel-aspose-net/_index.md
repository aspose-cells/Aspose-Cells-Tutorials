## Οδηγός Υλοποίησης: Κατάργηση Συγχωνευμένων Κελιών

Αφού όλα έχουν ρυθμιστεί, ας επικεντρωθούμε στην κατάργηση της συγχώνευσης συγχωνευμένων κελιών χρησιμοποιώντας το Aspose.Cells.

### Επισκόπηση

Η αποσύνδεση κελιών είναι απαραίτητη για εργασίες χειρισμού δεδομένων όπου απαιτούνται τιμές μεμονωμένων κελιών. Αυτή η διαδικασία είναι απλή με το Aspose.Cells.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το βιβλίο εργασίας του Excel από τον κατάλογο προέλευσης:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook wbk = new Workbook(SourceDir + "/sampleUnMergingtheMergedCells.xlsx");
```

**Γιατί αυτό το βήμα;** Αρχικοποιεί το `Workbook` αντικείμενο με το αρχείο Excel που σκοπεύετε να χειριστείτε.

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας

Στη συνέχεια, αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει τα συγχωνευμένα κελιά:

```csharp
Worksheet worksheet = wbk.Worksheets[0];
```

Αυτή η γραμμή ανακτά το πρώτο φύλλο εργασίας. Προσαρμόστε το ευρετήριο εάν το φύλλο προορισμού σας είναι διαφορετικό.

#### Βήμα 3: Αποσυγχώνευση κελιών

Χρησιμοποιήστε το `UnMerge` μέθοδος για την αποσυγχώνευση μιας συγκεκριμένης περιοχής κελιών:

```csharp
Cells cells = worksheet.Cells;
cells.UnMerge(5, 2, 2, 3);
```

**Επεξήγηση παραμέτρων:**
- **Αρχική σειρά (5)** και **Αρχική στήλη (2)**: Καθορίστε από πού ξεκινά η συγχωνευμένη περιοχή.
- **Σύνολο γραμμών προς αποσυγχώνευση (2)** και **Σύνολο στηλών προς αποσυγχώνευση (3)**: Ορίστε το μέγεθος της περιοχής που θα καταργηθεί η συγχώνευση.

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα αρχείο:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
wbk.Save(outputDir + "/outputUnMergingtheMergedCells.xlsx");
```
