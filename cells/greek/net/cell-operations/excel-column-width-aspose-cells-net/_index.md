## Οδηγός Εφαρμογής

### Ορισμός πλάτους στηλών

#### Επισκόπηση

Μπορείτε να τυποποιήσετε το πλάτος όλων των στηλών χρησιμοποιώντας μια απλή μέθοδο. Αυτό βελτιώνει την ομοιομορφία σε όλα τα φύλλα εργασίας σας, καθιστώντας τα δεδομένα πιο προσβάσιμα και με επαγγελματική εμφάνιση.

#### Οδηγός βήμα προς βήμα:

##### 1. **Ρύθμιση του Περιβάλλοντός σας**

Βεβαιωθείτε ότι έχετε δημιουργήσει έναν κατάλληλο κατάλογο για τη διαχείριση αρχείων:

```csharp
// Εκκίνηση:1
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

##### 2. **Φόρτωση του αρχείου Excel**

Ανοίξτε το επιθυμητό αρχείο Excel χρησιμοποιώντας ένα `FileStream`:

```csharp
using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
{
    Workbook workbook = new Workbook(fstream);
}
```

- **Παράμετροι:** `dataDir + "book1.xls"` καθορίζει τη διαδρομή προς το αρχείο.
- **Σκοπός της μεθόδου:** Ανοίγει το αρχείο Excel για χειρισμό.

##### 3. **Πρόσβαση και τροποποίηση φύλλου εργασίας**

Επιλέξτε το φύλλο εργασίας που θέλετε να τροποποιήσετε:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Cells.StandardWidth = 20.5;
```

- **Διαμόρφωση κλειδιού:** `StandardWidth` ορίζει όλες τις στήλες σε ομοιόμορφο πλάτος 20,5.

##### 4. **Αποθήκευση και Κλείσιμο Πόρων**

Μην ξεχάσετε να αποθηκεύσετε τις αλλαγές σας και να κλείσετε τη ροή αρχείων:

```csharp
workbook.Save(dataDir + "output.out.xls");
fstream.Close();
```

- **Συμβουλή αντιμετώπισης προβλημάτων:** Να διασφαλίζετε πάντα ότι οι ροές είναι κλειστές για την αποφυγή διαρροών πόρων.
