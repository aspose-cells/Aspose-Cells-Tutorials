## Οδηγός Εφαρμογής

### Δημιουργία στιγμιαίου αντικειμένου βιβλίου εργασίας
#### Επισκόπηση
Δημιουργία μιας παρουσίας του `Workbook` Η κλάση είναι το πρώτο βήμα, που αντιπροσωπεύει το αρχείο Excel που θα χειριστείτε.

**Βήμα 1: Φόρτωση αρχείου Excel**
Καθορίστε τον κατάλογο που περιέχει το αρχείο Excel και φορτώστε το σε ένα `Workbook` αντικείμενο:
```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε ένα νέο αντικείμενο βιβλίου εργασίας φορτώνοντας ένα αρχείο Excel.
Workbook workbook = new Workbook(sourceDir + "sampleAccessCellByRowAndColumnIndex.xlsx");
```
Ο παραπάνω κώδικας αρχικοποιεί το `workbook` με δεδομένα από το καθορισμένο αρχείο Excel, έτοιμο για περαιτέρω λειτουργίες.

### Πρόσβαση σε κελιά σε ένα φύλλο εργασίας
#### Επισκόπηση
Μόλις φορτώσετε το βιβλίο εργασίας σας, η πρόσβαση σε συγκεκριμένα κελιά με βάση τους δείκτες τους είναι απλή.

**Βήμα 1: Πρόσβαση στο πρώτο φύλλο εργασίας**
Τα βιβλία εργασίας αποτελούνται από πολλά φύλλα εργασίας. Μπορείτε να αποκτήσετε πρόσβαση σε αυτά χρησιμοποιώντας μηδενική ευρετηρίαση:
```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας.
Worksheet worksheet = workbook.Worksheets[0];
```

**Βήμα 2: Πρόσβαση σε συγκεκριμένο κελί**
Ανάκτηση ενός κελιού με βάση τους δείκτες γραμμών και στηλών του (με μηδενικό δείκτη):
```csharp
// Αποκτήστε πρόσβαση σε ένα συγκεκριμένο κελί χρησιμοποιώντας τους δείκτες γραμμών και στηλών του.
Cell cell = worksheet.Cells[5, 2]; // 6η σειρά, 3η στήλη.

// Εξάγετε το όνομα και την τιμή του κελιού.
Console.WriteLine("Cell Name: " + cell.Name + " Value: " + cell.StringValue);
```
