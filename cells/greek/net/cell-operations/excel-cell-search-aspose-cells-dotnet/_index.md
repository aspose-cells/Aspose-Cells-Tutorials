## Οδηγός Εφαρμογής

### Εύρεση κελιών που περιέχουν συγκεκριμένους αριθμούς

#### Επισκόπηση:
Εντοπίστε κελιά με συγκεκριμένους αριθμούς χρησιμοποιώντας τη λειτουργία αναζήτησης του Aspose.Cells. Αυτό είναι χρήσιμο για την εύρεση σημείων δεδομένων όπως αναγνωριστικά ή μετρήσεις σε μεγάλα υπολογιστικά φύλλα.

**Βήμα 1: Ρύθμιση παραμέτρων επιλογών αναζήτησης**
```csharp
FindOptions opts = new FindOptions();
opts.LookInType = LookInType.Values; // Αναζήτηση εντός τιμών κελιών
opts.LookAtType = LookAtType.EntireContent; // Αντιστοίχιση ολόκληρου του περιεχομένου του κελιού
```

**Βήμα 2: Εκτελέστε την αναζήτηση**
```csharp
Cell cell1 = cells.Find(205, null, opts); // Αναζητώντας τον αριθμό 205

if (cell1 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell1.Name);
}
else
{
    Console.WriteLine("Record not found.");
}
```

### Εύρεση κελιών που περιέχουν συγκεκριμένες συμβολοσειρές

#### Επισκόπηση:
Εξαγάγετε αποτελεσματικά δεδομένα κειμένου, όπως ονόματα προϊόντων ή ετικέτες κατηγοριών, εντοπίζοντας κελιά με συγκεκριμένες συμβολοσειρές.

**Βήμα 1: Ρύθμιση παραμέτρων επιλογών αναζήτησης για συμβολοσειρά**
```csharp
opts.LookAtType = LookAtType.Contains; // Αντιστοίχιση εάν η συμβολοσειρά περιέχεται οπουδήποτε στο κελί
```

**Βήμα 2: Εκτελέστε την αναζήτηση συμβολοσειράς**
```csharp
Cell cell3 = cells.Find("Data", null, opts); // Αναζήτηση για οποιαδήποτε εμφάνιση του όρου "Δεδομένα"

if (cell3 != null)
{
    Console.WriteLine("Name of the cell containing the value: " + cell3.Name);
}
else
{
    Console.WriteLine("Record not found.");
}
```

### Συμβουλές αντιμετώπισης προβλημάτων:
- **Λανθασμένοι τύποι δεδομένων**Βεβαιωθείτε ότι κάνετε αναζήτηση με τον σωστό τύπο δεδομένων (ακέραιος αριθμός για αριθμούς, συμβολοσειρά για κείμενο).
- **Ευαισθησία πεζών-κεφαλαίων**: Από προεπιλογή, οι αναζητήσεις κάνουν διάκριση πεζών-κεφαλαίων. Προσαρμογή `opts.CaseSensitive` αν χρειαστεί.
