## Οδηγός Εφαρμογής

### Ορισμός ημερών για τη διατήρηση του ιστορικού σε κοινόχρηστα βιβλία εργασίας

Στα κοινόχρηστα βιβλία εργασίας, η παρακολούθηση των αναθεωρήσεων είναι απαραίτητη για τη συνεργατική επεξεργασία. Με το Aspose.Cells, μπορείτε να καθορίσετε για πόσο χρονικό διάστημα θα πρέπει να διατηρούνται αυτά τα αρχεία καταγραφής.

#### Δημιουργία και ρύθμιση παραμέτρων ενός κοινόχρηστου βιβλίου εργασίας

**Βήμα 1: Δημιουργήστε ένα κενό βιβλίο εργασίας**

```csharp
// Δημιουργία νέας παρουσίας βιβλίου εργασίας
Workbook wb = new Workbook();
```

**Βήμα 2: Κοινή χρήση του βιβλίου εργασίας**

Ενεργοποιήστε την κοινή χρήση για να επιτρέψετε σε πολλούς χρήστες να επεξεργάζονται:

```csharp
// Ενεργοποίηση κοινόχρηστων ρυθμίσεων
wb.Settings.Shared = true;
```

**Βήμα 3: Ενημέρωση DaysPreservingHistory of RevisionLogs**

Καθορίστε πόσες ημέρες θέλετε να διατηρείται το ιστορικό αναθεωρήσεων:

```csharp
// Ορισμός ημερών για τη διατήρηση των αρχείων καταγραφής αναθεωρήσεων
wb.Worksheets.RevisionLogs.DaysPreservingHistory = 7;
```

Αυτή η ρύθμιση διασφαλίζει ότι καταγράφονται μόνο οι τελευταίες επτά ημέρες αλλαγών, διατηρώντας τα αρχεία καταγραφής σας συνοπτικά και σχετικά.

**Βήμα 4: Αποθήκευση του βιβλίου εργασίας**

Τέλος, αποθηκεύστε το βιβλίο εργασίας με τις ενημερωμένες ρυθμίσεις:

```csharp
// Ορισμός καταλόγου εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση του αρχείου
wb.Save(outputDir + "outputShared_DaysPreservingHistory.xlsx");
```

#### Συμβουλές αντιμετώπισης προβλημάτων

- **Βεβαιωθείτε ότι το βιβλίο εργασίας είναι κοινόχρηστο**: Εάν οι αλλαγές δεν αντικατοπτρίζονται, επαληθεύστε ότι `wb.Settings.Shared` έχει οριστεί σε true.
- **Έλεγχος αξίας ημερών**: Βεβαιωθείτε `DaysPreservingHistory` είναι ένας θετικός ακέραιος αριθμός.
