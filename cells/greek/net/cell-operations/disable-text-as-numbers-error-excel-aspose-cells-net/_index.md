## Οδηγός Εφαρμογής

### Επισκόπηση των επιλογών ελέγχου σφαλμάτων

Σε αυτήν την ενότητα, θα μάθετε πώς να απενεργοποιήσετε την προειδοποίηση "Κείμενο ως αριθμοί" χρησιμοποιώντας το Aspose.Cells για .NET. Αυτή η λειτουργικότητα είναι ιδιαίτερα χρήσιμη εάν το σύνολο δεδομένων σας περιλαμβάνει κείμενο που το Excel ενδέχεται να αντιμετωπίσει εσφαλμένα ως αριθμούς.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας σας

Αρχικά, φορτώστε ένα υπάρχον βιβλίο εργασίας ή δημιουργήστε ένα νέο:

```csharp
// Κατάλογος πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Δημιουργήστε ένα βιβλίο εργασίας και ανοίξτε το υπολογιστικό φύλλο προτύπου
Workbook workbook = new Workbook(sourceDir + "sampleErrorCheckingOptions.xlsx");
```

#### Βήμα 2: Φύλλο εργασίας και επιλογές σφάλματος πρόσβασης

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και στις επιλογές ελέγχου σφαλμάτων:

```csharp
// Αποκτήστε το πρώτο φύλλο εργασίας
Worksheet sheet = workbook.Worksheets[0];

// Δημιουργήστε την αρχική έκδοση της συλλογής επιλογών ελέγχου σφαλμάτων
ErrorCheckOptionCollection opts = sheet.ErrorCheckOptions;
```

#### Βήμα 3: Ρύθμιση παραμέτρων κειμένου ως επιλογή αριθμών

Απενεργοποιήστε την επιλογή "Κείμενο ως αριθμοί" για ένα καθορισμένο εύρος:

```csharp
int index = opts.Add();
ErrorCheckOption opt = opts[index];
opt.SetErrorCheck(ErrorCheckType.TextNumber, false);

// Ορίστε την περιοχή κελιού όπου θα εφαρμοστεί αυτή η ρύθμιση
CellArea ca = CellArea.CreateCellArea("A1", "E20");
opt.AddRange(ca);
```

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας σας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τις ενημερωμένες ρυθμίσεις:

```csharp
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "outputErrorCheckingOptions.xlsx");

Console.WriteLine("ErrorCheckingOptions executed successfully.\r\n");
```
