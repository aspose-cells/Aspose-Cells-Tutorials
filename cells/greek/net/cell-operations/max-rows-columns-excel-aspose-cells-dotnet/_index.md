## Οδηγός Εφαρμογής
Αυτή η ενότητα θα σας καθοδηγήσει στην υλοποίηση μιας λύσης για την ανακάλυψη του μέγιστου αριθμού γραμμών και στηλών σε μορφές XLS και XLSX χρησιμοποιώντας C#.

### Επισκόπηση
Στόχος μας είναι να δημιουργήσουμε ένα πρόγραμμα που να εμφανίζει τον μέγιστο αριθμό γραμμών και στηλών που υποστηρίζονται τόσο από το Excel 97-2003 (XLS) όσο και από τα σύγχρονα αρχεία Excel (XLSX). Θα το πετύχουμε αυτό αξιοποιώντας το Aspose.Cells. `WorkbookSettings` σκηνικά θέατρου.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργία και ρύθμιση παραμέτρων βιβλίου εργασίας για μορφή XLS**
```csharp
using System;
using Aspose.Cells;

namespace DiscoverMaxRowsColumns
{
    class Program
    {
        public static void Main()
        {
            // Αρχικοποίηση μηνύματος σχετικά με τη μορφή XLS.
            Console.WriteLine("Maximum Rows and Columns supported by XLS format.");

            // Δημιουργήστε ένα βιβλίο εργασίας σε μορφή XLS.
            Workbook wb = new Workbook(FileFormatType.Excel97To2003);

            // Προσδιορίστε τον μέγιστο αριθμό γραμμών και στηλών για XLS.
            int maxRowsXls = wb.Settings.MaxRow + 1;
            int maxColsXls = wb.Settings.MaxColumn + 1;

            // Εξάγετε τα αποτελέσματα.
            Console.WriteLine("Maximum Rows: " + maxRowsXls);
            Console.WriteLine("Maximum Columns: " + maxColsXls);
        }
    }
}
```
**Εξήγηση:**
- `FileFormatType.Excel97To2003`: Καθορίζει ότι εργαζόμαστε με μια παλαιότερη μορφή Excel, XLS.
- `wb.Settings.MaxRow` και `wb.Settings.MaxColumn`Αυτές οι ιδιότητες παρέχουν τις μέγιστες υποστηριζόμενες τιμές ευρετηρίου. Η προσθήκη 1 τις μετατρέπει σε μετρήσεις αναγνώσιμες από τον άνθρωπο.

**2. Δημιουργία και ρύθμιση παραμέτρων βιβλίου εργασίας για μορφή XLSX**
```csharp
// Εκτύπωση μηνύματος σχετικά με τη μορφή XLSX.
Console.WriteLine("Maximum Rows and Columns supported by XLSX format.");

// Αναδημιουργήστε ένα βιβλίο εργασίας σε μορφή XLSX.
wb = new Workbook(FileFormatType.Xlsx);

// Προσδιορίστε τον μέγιστο αριθμό γραμμών και στηλών για XLSX.
int maxRowsXlsx = wb.Settings.MaxRow + 1;
int maxColsXlsx = wb.Settings.MaxColumn + 1;

// Εξάγετε τα αποτελέσματα.
Console.WriteLine("Maximum Rows: " + maxRowsXlsx);
Console.WriteLine("Maximum Columns: " + maxColsXlsx);

Console.WriteLine("DiscoverMaxRowsColumns executed successfully.");
```
**Εξήγηση:**
- Μετάβαση σε `FileFormatType.Xlsx` μας επιτρέπει να εξερευνήσουμε τις δυνατότητες του σύγχρονου Excel, οι οποίες γενικά υποστηρίζουν περισσότερες γραμμές και στήλες από την παλαιότερη μορφή XLS.
