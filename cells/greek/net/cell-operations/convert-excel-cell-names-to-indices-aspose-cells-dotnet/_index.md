## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή ονόματος σε ευρετήριο

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα όνομα κελιού όπως 'C4' στους αντίστοιχους δείκτες γραμμών και στηλών.

#### Βήμα 1: Εισαγωγή απαραίτητων βιβλιοθηκών

Εισαγάγετε τον χώρο ονομάτων Aspose.Cells στην αρχή του αρχείου σας:
```csharp
using Aspose.Cells;
```

#### Βήμα 2: Ορισμός καταλόγων προέλευσης και εξόδου

Ορίστε placeholders για καταλόγους όπου θα αποθηκεύονται τα αρχεία εισόδου και τα αποτελέσματα εξόδου.
```csharp
string sourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 3: Αρχικοποίηση του Βοηθού Aspose.Cells

Δημιουργήστε μια παρουσία του `CellsHelper` για να χρησιμοποιήσετε τη λειτουργία μετατροπής:
```csharp
var cellsHelper = new CellsHelper();
```

#### Βήμα 4: Μετατροπή ονόματος κελιού σε δείκτες

Ορίστε το όνομα του κελιού που θέλετε να μετατρέψετε και αρχικοποιήστε μεταβλητές για δείκτες γραμμών και στηλών.
```csharp
string name = "C4";
int row, column;
cellsHelper.CellNameToIndex(name, out row, out column);
```

**Εξήγηση:**
- `CellNameToIndex` είναι μια μέθοδος που λαμβάνει το όνομα του κελιού (π.χ., 'C4') και εξάγει τους αντίστοιχους δείκτες γραμμών και στηλών. Αυτή η μετατροπή είναι κρίσιμη για την πρόσβαση μέσω προγραμματισμού σε συγκεκριμένα κελιά με βάση τα αναγνωριστικά τους στο Excel.
