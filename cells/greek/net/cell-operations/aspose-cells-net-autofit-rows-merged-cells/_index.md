# Κύρια Αυτόματη Προσαρμογή Σειρών σε Συγχωνευμένα Κελιά Χρησιμοποιώντας Aspose.Cells για .NET

Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία αυτόματης προσαρμογής γραμμών σε συγχωνευμένα κελιά χρησιμοποιώντας Aspose.Cells και C#. Στο τέλος, θα κατανοήσετε:
- Τα βασικά της συγχώνευσης κελιών και της αυτόματης προσαρμογής γραμμών.
- Πώς να χρησιμοποιήσετε **Aspose.Cells για .NET** για να βελτιστοποιήσετε τις εργασίες αυτοματοποίησης του Excel.
- Τεχνικές για την εφαρμογή αναδίπλωσης κειμένου και στυλ σε συγχωνευμένα κελιά.

Ας ξεκινήσουμε εξετάζοντας τις προϋποθέσεις.

## Οδηγός Εφαρμογής

Θα σας καθοδηγήσουμε στην αυτόματη προσαρμογή γραμμών σε συγχωνευμένα κελιά χρησιμοποιώντας C#.

### Δημιουργία και συγχώνευση κελιών

#### Επισκόπηση

Αρχικά, δημιουργήστε μια περιοχή κελιών και συγχωνεύστε τα για να ρυθμίσετε το φύλλο εργασίας σας πριν εφαρμόσετε τις ρυθμίσεις αυτόματης προσαρμογής.

**Βήμα 1: Δημιουργία βιβλίου εργασίας και φύλλου εργασίας**

```csharp
// Κατάλογος εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Δημιουργία ενός νέου βιβλίου εργασίας
Workbook wb = new Workbook();

// Λήψη του πρώτου (προεπιλεγμένου) φύλλου εργασίας
Worksheet _worksheet = wb.Worksheets[0];
```

#### Βήμα 2: Δημιουργία εύρους και συγχώνευση

Δημιουργήστε μια περιοχή κελιών που θα συγχωνευθούν για ενοποιημένη αναπαράσταση δεδομένων.

```csharp
// Δημιουργήστε μια περιοχή A1:B1
Range range = _worksheet.Cells.CreateRange(0, 0, 1, 2);

// Συγχώνευση των κελιών
range.Merge();
```

### Εισαγωγή τιμών και κελιών στυλ

#### Επισκόπηση

Μετά τη συγχώνευση, εισαγάγετε κείμενο στο συγχωνευμένο κελί σας και εφαρμόστε στυλ για να διασφαλίσετε την αναγνωσιμότητα.

**Βήμα 3: Προσθήκη κειμένου και στυλ**

Εισαγάγετε μια μακροσκελή πρόταση για να επιδείξετε τις δυνατότητες αυτόματης προσαρμογής. Ενεργοποιήστε την αναδίπλωση κειμένου και ορίστε στυλ για λόγους σαφήνειας.

```csharp
// Εισαγωγή τιμής στο συγχωνευμένο κελί A1
_worksheet.Cells[0, 0].Value = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog....end";

// Δημιουργήστε ένα αντικείμενο στυλ
Aspose.Cells.Style style = _worksheet.Cells[0, 0].GetStyle();

// Ορισμός αναδίπλωσης κειμένου σε
style.IsTextWrapped = true;

// Εφαρμογή του στυλ στο κελί
_worksheet.Cells[0, 0].SetStyle(style);
```

### Αυτόματη προσαρμογή γραμμών

#### Επισκόπηση

Χρήση του Aspose.Cells `AutoFitterOptions` για να προσαρμόσετε τα ύψη των γραμμών για τα συγχωνευμένα κελιά.

**Βήμα 4: Ρύθμιση παραμέτρων και εφαρμογή αυτόματης προσαρμογής**

Ρυθμίστε τις παραμέτρους αυτόματης προσαρμογής προσαρμοσμένες στα συγχωνευμένα κελιά, διασφαλίζοντας ότι κάθε γραμμή κειμένου ταιριάζει απόλυτα μέσα στο κελί.

```csharp
// Δημιουργήστε ένα αντικείμενο για το AutoFitterOptions
AutoFitterOptions options = new AutoFitterOptions();

// Ορισμός αυτόματης προσαρμογής για συγχωνευμένα κελιά
options.AutoFitMergedCellsType = AutoFitMergedCellsType.EachLine;

// Αυτόματη προσαρμογή γραμμών στο φύλλο (συμπεριλαμβανομένων των συγχωνευμένων κελιών)
_worksheet.AutoFitRows(options);
```

### Αποθήκευση και αξιολόγηση

#### Επισκόπηση

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να ελέγξετε τις αλλαγές.

**Βήμα 5: Αποθήκευση βιβλίου εργασίας**

```csharp
// Αποθήκευση του αρχείου Excel
wb.Save(outputDir + "AutofitRowsforMergedCells.xlsx");
Console.WriteLine("AutofitRowsforMergedCells executed successfully.\r\n");
```
