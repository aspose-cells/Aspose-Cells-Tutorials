### Οδηγός Εφαρμογής

#### Συγχώνευση κελιών σε ένα φύλλο εργασίας

**Επισκόπηση:**
Η συγχώνευση κελιών ενοποιεί τα δεδομένα για καλύτερη αναγνωσιμότητα και παρουσίαση. Αυτή η ενότητα σας καθοδηγεί στη συγχώνευση συγκεκριμένων κελιών χρησιμοποιώντας το Aspose.Cells.

1. **Δημιουργία νέου βιβλίου εργασίας**
   Ξεκινήστε δημιουργώντας μια παρουσία του `Workbook` κλάση, η οποία αντιπροσωπεύει ένα αρχείο Excel.
   
   ```csharp
   Workbook workbook = new Workbook();
   ```

2. **Πρόσβαση στο Φύλλο Εργασίας**
   Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας από το βιβλίο εργασίας σας:
   
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   ```

3. **Τροποποίηση και συγχώνευση κελιών**
   Προσθέστε μια τιμή σε ένα συγκεκριμένο κελί και, στη συνέχεια, συγχωνεύστε τα κελιά στο επιθυμητό εύρος.
   
   ```csharp
   // Ορίστε την τιμή του "A1"
   Cell cell = worksheet.Cells["A1"];
   cell.PutValue("Visit Aspose!");

   // Συγχώνευση κελιών από A1 σε C1 (ευρετήριο με βάση το 0)
   worksheet.Cells.Merge(0, 0, 1, 3);
   ```

4. **Αποθήκευση του βιβλίου εργασίας σας**
   Αποθηκεύστε το βιβλίο εργασίας στην επιθυμητή μορφή:
   
   ```csharp
   string outputDir = @"YOUR_OUTPUT_DIRECTORY";
   workbook.Save(outputDir + "/merged_cells_output.xls", SaveFormat.Excel97To2003);
   ```

#### Δημιουργία καταλόγων για λειτουργίες αρχείων

```csharp
}
```
