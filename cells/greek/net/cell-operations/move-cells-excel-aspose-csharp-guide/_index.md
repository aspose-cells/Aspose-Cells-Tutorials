## Οδηγός Εφαρμογής

### Μετακίνηση περιοχής κελιών

Σε αυτήν την ενότητα, θα υλοποιήσουμε την κύρια λειτουργικότητα: μετακίνηση μιας περιοχής κελιών.

#### Επισκόπηση

Ο στόχος είναι η επανατοποθέτηση μιας συγκεκριμένης περιοχής μέσα σε ένα φύλλο εργασίας του Excel. Αυτό μπορεί να είναι χρήσιμο για την οργάνωση δεδομένων ή τη δυναμική προσαρμογή των διατάξεων.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός καταλόγων προέλευσης και εξόδου**

Αρχικά, καθορίστε τον κατάλογο προέλευσης (όπου βρίσκεται το αρχικό αρχείο Excel) και τον κατάλογο εξόδου (όπου θα αποθηκεύσετε το τροποποιημένο αρχείο).

```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();
```

**2. Ανοίξτε το Βιβλίο εργασίας του Excel**

Φορτώστε το βιβλίο εργασίας χρησιμοποιώντας το Aspose.Cells:

```csharp
Workbook workbook = new Workbook(sourceDir + "sampleMoveRangeOfCells.xlsx");
```

**3. Πρόσβαση στα κελιά του φύλλου εργασίας**

Αποκτήστε πρόσβαση στα κελιά από το πρώτο φύλλο εργασίας:

```csharp
Cells cells = workbook.Worksheets[0].Cells;
```

**4. Δημιουργήστε μια περιοχή κελιού και μετακινήστε την**

Καθορίστε το εύρος που θα μετακινηθεί (π.χ., A1:C5) και μετατοπίστε το κατά 7 γραμμές και 5 στήλες.

```csharp
CellArea ca = CellArea.CreateCellArea("A1", "C5");
cells.MoveRange(ca, 7, 5);
```

**5. Αποθήκευση του τροποποιημένου βιβλίου εργασίας**

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```csharp
workbook.Save(outputDir + "outputMoveRangeOfCells.xlsx");
Console.WriteLine("MoveRangeOfCells executed successfully.");
```
