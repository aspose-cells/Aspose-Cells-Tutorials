## Οδηγός Εφαρμογής
### Αυτόματη Προσαρμογή Σειρών σε Συγκεκριμένα Εύρη
Η αυτόματη προσαρμογή γραμμών διασφαλίζει ότι τα δεδομένα σας εμφανίζονται με τάξη, ανεξάρτητα από το μήκος του περιεχομένου. Ας αναλύσουμε τα βήματα:

#### Βήμα 1: Ανοίξτε ένα αρχείο Excel
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας που θέλετε να τροποποιήσετε.
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "path/to/your/files/";

// Δημιουργήστε μια ροή αρχείων που περιέχει το αρχείο Excel που θα ανοίξει
FileStream fstream = new FileStream(dataDir + "Book1.xlsx", FileMode.Open);

// Άνοιγμα του αρχείου Excel μέσω της ροής αρχείων
Workbook workbook = new Workbook(fstream);
```
**Γιατί αυτό το βήμα;** Το άνοιγμα της ροής αρχείων είναι κρίσιμο για την πρόσβαση και την τροποποίηση των δεδομένων σας.

#### Βήμα 2: Πρόσβαση σε ένα φύλλο εργασίας
Στη συνέχεια, αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας όπου θέλετε να προσαρμόσετε αυτόματα τις γραμμές.
```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Worksheet worksheet = workbook.Worksheets[0];
```
Αυτό το βήμα διασφαλίζει ότι εργάζεστε με το σωστό σύνολο δεδομένων.

#### Βήμα 3: Αυτόματη Προσαρμογή Σειρών
Η αυτόματη προσαρμογή μιας γραμμής προσαρμόζει το ύψος της με βάση το περιεχόμενο. Χρησιμοποιήστε `AutoFitRow` για να επιτευχθεί αυτό:
```csharp
// Αυτόματη προσαρμογή της τρίτης γραμμής του φύλλου εργασίας (ο δείκτης ξεκινά από το 0)
worksheet.AutoFitRow(2, 0, 5);
```
**Επεξήγηση παραμέτρων:**
- **Ευρετήριο γραμμής:** Ο δείκτης της γραμμής που θέλετε να προσαρμοστεί αυτόματα.
- **startColumnIndex και endColumnIndex:** Ορίστε το εύρος εντός του οποίου θα εφαρμοστεί η αυτόματη προσαρμογή.

#### Βήμα 4: Αποθήκευση αλλαγών
Αφού κάνετε τις αλλαγές, αποθηκεύστε το βιβλίο εργασίας σας:
```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
tworkbook.Save(dataDir + "output.xlsx");

// Κλείσιμο της ροής αρχείων για την απελευθέρωση όλων των πόρων
fstream.Close();
```
Αυτό το βήμα διασφαλίζει ότι όλες οι τροποποιήσεις θα εγγραφούν ξανά στον δίσκο.
