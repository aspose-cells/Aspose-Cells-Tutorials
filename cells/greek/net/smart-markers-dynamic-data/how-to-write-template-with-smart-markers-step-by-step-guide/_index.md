# Πώς να γράψετε πρότυπο με Smart Markers – Πλήρης Οδηγός  

Έχετε αναρωτηθεί ποτέ **πώς να γράψετε πρότυπο** που επεκτείνεται αυτόματα με βάση τα δεδομένα σας; Δεν είστε μόνοι—πολλοί προγραμματιστές αντιμετωπίζουν πρόβλημα όταν χρειάζονται μια δυναμική αναφορά Excel αλλά δεν ξέρουν ποια δυνατότητα του API να χρησιμοποιήσουν. Τα καλά νέα; Με τα Aspose.Cells Smart Markers μπορείτε να δημιουργήσετε ένα πρότυπο σε ένα μόνο κελί, να συνδέσετε ιεραρχικά δεδομένα και να αφήσετε τη βιβλιοθήκη να επαναλαμβάνει τις γραμμές για εσάς. Σε αυτόν τον οδηγό θα καλύψουμε επίσης **πώς να επαναλάβετε γραμμές**, **πώς να συνδέσετε δεδομένα**, και ακόμη **πώς να δημιουργήσετε αναφορά** χωρίς να κάνετε χειροκίνητο βρόχο στα φύλλα εργασίας.

Στο τέλος αυτού του tutorial θα έχετε ένα πλήρες, εκτελέσιμο παράδειγμα που δείχνει **πώς να δημιουργήσετε πρότυπο** για σενάρια master‑detail, καθώς και συμβουλές για ειδικές περιπτώσεις και τεχνάσματα απόδοσης. Δεν απαιτούνται εξωτερικά έγγραφα—όλα όσα χρειάζεστε είναι εδώ.

---

## Τι θα δημιουργήσετε

Θα δημιουργήσουμε ένα βιβλίο εργασίας Excel που καταγράφει παραγγελίες (το master) και τα στοιχεία τους (το detail). Το πρότυπο βρίσκεται στο κελί **A1**, και τα Smart Markers θα το επεκτείνουν σε έναν ωραία μορφοποιημένο πίνακα. Το τελικό φύλλο θα φαίνεται ως εξής:

```
Order1
   A
   B
Order2
   C
```

Αυτή είναι μια κλασική περίπτωση «πώς να δημιουργήσετε αναφορά», και ο κώδικας λειτουργεί με .NET 6+ και Aspose.Cells 23.x (ή νεότερη έκδοση).

## Βήμα 2: Γράψτε το πρότυπο Smart Marker  

Το πρότυπο χρησιμοποιεί `${Master.Name}` για τον τίτλο της παραγγελίας και `${Detail:Repeat}` για την επανάληψη σε κάθε στοιχείο γραμμής.

```csharp
            // Step 2: Define a Smart Marker template that repeats detail rows for each master record
            string smartMarkerTemplate = @"${Master.Name}
${Detail:Repeat}
   ${Detail.Item}
${/Detail}";
            
            // Write the template into cell A1
            worksheet.Cells["A1"].PutValue(smartMarkerTemplate);
```

> **Συμβουλή**: Κρατήστε το πρότυπο σε ένα μόνο κελί· τα Smart Markers θα το επεκτείνουν αυτόματα σε πολλές γραμμές.  

*Πώς αυτό λύνει το πρόβλημα*: Ενσωματώνοντας το μπλοκ επανάληψης απευθείας στο κελί, αποφεύγετε την χειροκίνητη εισαγωγή γραμμών—η Aspose το διαχειρίζεται για εσάς.

## Βήμα 3: Δημιουργία ιεραρχικών δεδομένων που ταιριάζουν με το πρότυπο  

Τα δεδομένα μας πρέπει να αντικατοπτρίζουν τη δομή του προτύπου: μια συλλογή `Master`, η κάθε μία περιέχει έναν πίνακα `Detail`.

```csharp
            // Step 3: Create hierarchical data matching the template structure
            var orderData = new
            {
                Master = new[]
                {
                    new
                    {
                        Name = "Order1",
                        Detail = new[]
                        {
                            new { Item = "A" },
                            new { Item = "B" }
                        }
                    },
                    new
                    {
                        Name = "Order2",
                        Detail = new[]
                        {
                            new { Item = "C" }
                        }
                    }
                }
            };
```

*Γιατί συνδέουμε τα δεδομένα με αυτόν τον τρόπο*: Τα Smart Markers χρησιμοποιούν σύνδεση τύπου reflection, έτσι τα ονόματα των ιδιοτήτων πρέπει να ταιριάζουν ακριβώς με τα placeholders. Αυτό είναι ο πυρήνας του **πώς να συνδέσετε δεδομένα** για δυναμικές αναφορές.

## Βαθύτερη Εξέταση: Γιατί τα Smart Markers είναι Αλλαγή Παιχνιδιού  

### Πώς να Επαναλάβετε Γραμμές Χωρίς Βρόχο  

Η παραδοσιακή αυτοματοποίηση Excel σας αναγκάζει να υπολογίσετε την τελευταία γραμμή, να εισάγετε νέες γραμμές και να αντιγράψετε στυλ—όλα αυτά είναι επιρρεπή σε σφάλματα. Τα Smart Markers αντικαθιστούν αυτό με ένα δηλωτικό μπλοκ `${Detail:Repeat}`. Η μηχανή αναλύει το μπλοκ, κλωνοποιεί τη γραμμή για κάθε στοιχείο στη συλλογή και εισάγει τις τιμές. Αυτή η προσέγγιση είναι **πώς να επαναλάβετε γραμμές** αποδοτικά.

### Σύνδεση Πολύπλοκων Αντικειμένων  

Μπορείτε να συνδέσετε ένθετα αντικείμενα, συλλογές ή ακόμη και DataTables. Εφόσον τα ονόματα των ιδιοτήτων ταιριάζουν, ο επεξεργαστής θα διασχίσει το γράφημα αντικειμένων. Αυτό είναι η ουσία του **πώς να συνδέσετε δεδομένα**: δίνετε στον επεξεργαστή ένα απλό αντικείμενο CLR (ή έναν ανώνυμο τύπο, όπως κάναμε) και τον αφήνετε να χαρτογραφήσει αυτόματα.

### Δημιουργία Διαφορετικών Μορφών  

Αν και το παράδειγμά μας αποθηκεύει σε XLSX, μπορείτε να αντικαταστήσετε το `SaveFormat.Pdf` ή `SaveFormat.Csv` με μια αλλαγή μίας γραμμής. Αυτό είναι ένας γρήγορος τρόπος για **πώς να δημιουργήσετε αναφορά** σε πολλαπλές μορφές χωρίς να αγγίξετε το πρότυπο.

### Επανάχρηση του Προτύπου  

Αν χρειάζεστε **πώς να δημιουργήσετε πρότυπο** για άλλα φύλλα εργασίας, απλώς αντιγράψτε το περιεχόμενο του κελιού σε άλλο φύλλο ή αποθηκεύστε το σε πόρο συμβολοσειράς. Η ίδια κλήση επεξεργαστή λειτουργεί παντού, κάνοντας τον κώδικά σας DRY και συντηρήσιμο.

## Συχνές Ερωτήσεις & Ειδικές Περιπτώσεις  

| Question | Answer |
|----------|--------|
| *Τι γίνεται αν ένα master δεν έχει γραμμές detail;* | Το μπλοκ `${Detail:Repeat}` θα παραλειφθεί, αφήνοντας μόνο το όνομα του master. Δεν δημιουργούνται κενές γραμμές. |
| *Μπορώ να μορφοποιήσω τις επαναλαμβανόμενες γραμμές;* | Ναι—εφαρμόστε μορφοποίηση στη γραμμή προτύπου (γραμματοσειρά, περιγράμματα κ.λπ.) πριν την επεξεργασία. Το στυλ αντιγράφεται σε κάθε παραγόμενη γραμμή. |
| *Πρέπει να απελευθερώσω το workbook;* | Το `Workbook` υλοποιεί το `IDisposable`. Τυλίξτε το σε ένα μπλοκ `using` για κώδικα παραγωγής, αλλά για μια σύντομη επίδειξη κονσόλας είναι προαιρετικό. |
| *Πόσο μεγάλα μπορούν να είναι τα δεδομένα;* | Τα Smart Markers είναι αποδοτικά στη μνήμη, αλλά εξαιρετικά μεγάλες συλλογές (εκατοντάδες χιλιάδες) μπορεί να απαιτούν σελιδοποίηση ή ροή. |
| *Μπορώ να χρησιμοποιήσω αρχείο JSON αντί για αντικείμενο;* | Απολύτως—αποσαφηνίστε το JSON σε ένα POCO που ταιριάζει με το πρότυπο, και μετά περάστε το στο `Process`. |

## Πλήρες Παράδειγμα Εργασίας (Έτοιμο για Αντιγραφή‑Επικόλληση)

```csharp
using Aspose.Cells;

namespace SmartMarkerDemo
{
    class Program
    {
        static void Main()
        {
            // Initialize workbook
            var workbook = new Workbook();
            var worksheet = workbook.Worksheets[0];

            // Define template
            string smartMarkerTemplate = @"${Master.Name}
${Detail:Repeat}
   ${Detail.Item}
${/Detail}";

            worksheet.Cells["A1"].PutValue(smartMarkerTemplate);

            // Prepare data
            var orderData = new
            {
                Master = new[]
                {
                    new
                    {
                        Name = "Order1",
                        Detail = new[]
                        {
                            new { Item = "A" },
                            new { Item = "B" }
                        }
                    },
                    new
                    {
                        Name = "Order2",
                        Detail = new[]
                        {
                            new { Item = "C" }
                        }
                    }
                }
            };

            // Process template
            worksheet.SmartMarkerProcessor.Process(orderData);

            // Save file
            workbook.Save("SmartMarkerReport.xlsx", SaveFormat.Xlsx);
            System.Console.WriteLine("Report generated: SmartMarkerReport.xlsx");
        }
    }
}
```

Εκτελέστε το πρόγραμμα (`dotnet run`) και ανοίξτε το *SmartMarkerReport.xlsx* – θα δείτε τις γραμμές master‑detail τακτοποιημένες όμορφα.

## Τι Ακολουθεί;  

- **Στυλ εξόδου**: εφαρμόστε στυλ κελιών στη γραμμή προτύπου πριν την επεξεργασία.  
- **Εξαγωγή σε PDF**: αλλάξτε το `SaveFormat.Xlsx` σε `SaveFormat.Pdf` για μια εκτυπώσιμη αναφορά.  
- **Δυναμικές κεφαλίδες**: προσθέστε placeholders `${Headers}` για τη δημιουργία τίτλων στηλών σε πραγματικό χρόνο.  
- **Πολλαπλά φύλλα**: επαναλάβετε τη διαδικασία σε επιπλέον φύλλα εργασίας για αναφορές πολλαπλών ενοτήτων.  

Νιώστε ελεύθεροι να πειραματιστείτε—αλλάξτε την πηγή δεδομένων, προσθέστε περισσότερα ένθετα επίπεδα, ή συνδυάστε με τύπους. Η ευελιξία των Smart Markers σημαίνει ότι ξοδεύετε λιγότερο χρόνο σε βρόχους κώδικα και περισσότερο χρόνο στην παροχή αξίας.
