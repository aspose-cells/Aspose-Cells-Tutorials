## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία εφαρμογής μορφοποίησης υπό όρους χρησιμοποιώντας αντίστροφες διαγώνιες λωρίδες.

### Δημιουργία νέου βιβλίου εργασίας και φύλλου εργασίας

Ξεκινήστε δημιουργώντας μια παρουσία του `Workbook` και πρόσβαση στο πρώτο φύλλο εργασίας του:

```csharp
using Aspose.Cells;

// Δημιουργία νέου βιβλίου εργασίας
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

### Προσθήκη μορφοποίησης υπό όρους

#### Βήμα 1: Ορίστε το εύρος μορφοποίησης

Καθορίστε το εύρος όπου θέλετε να εφαρμόσετε μορφοποίηση υπό όρους:

```csharp
CellArea ca = new CellArea { StartRow = 0, EndRow = 5, StartColumn = 0, EndColumn = 3 };
```

#### Βήμα 2: Ρύθμιση κανόνων μορφοποίησης υπό όρους

Προσθήκη νέου κανόνα μορφοποίησης υπό όρους χρησιμοποιώντας `FormatConditionType` και καθορίστε τον τύπο συνθήκης:

```csharp
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcs = sheet.ConditionalFormattings[index];
fcs.AddArea(ca);

// Ορίστε τη συνθήκη (π.χ., τιμές μεταξύ 50 και 100)
int conditionIndex = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "50", "100");
```

#### Βήμα 3: Εφαρμογή μοτίβου αντίστροφης διαγώνιας λωρίδας

Ρυθμίστε το στυλ ώστε να περιλαμβάνει ένα μοτίβο αντίστροφης διαγώνιας λωρίδας με συγκεκριμένα χρώματα προσκηνίου και φόντου:

```csharp
FormatCondition fc = fcs[conditionIndex];
fc.Style.Pattern = BackgroundType.ReverseDiagonalStripe;
fc.Style.ForegroundColor = Color.FromArgb(255, 255, 0); // Κίτρινος
fc.Style.BackgroundColor = Color.FromArgb(0, 255, 255); // Κυανό
```

### Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να απεικονίσετε τις αλλαγές:

```csharp
workbook.Save("output.xlsx");
```
