## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα περιηγηθούμε στην υλοποίηση της δημιουργίας και της διαμόρφωσης βιβλίων εργασίας του Excel χρησιμοποιώντας το Aspose.Cells.

### Χαρακτηριστικό: Δημιουργία και αποθήκευση βιβλίου εργασίας
**Επισκόπηση**
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα νέο βιβλίο εργασίας του Excel, να εφαρμόσετε στυλ, να προσαρμόσετε αυτόματα γραμμές/στήλες και να αποθηκεύσετε σε διαφορετικές μορφές.

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας

```csharp
using System;
using Aspose.Cells;

public class FeatureWorkbookCreation
{
    public static void Run()
    {
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string output1Path = SourceDir + "Output.xlsx";
        string output2Path = SourceDir + "Output.out.ods";

        // Δημιουργία νέας παρουσίας βιβλίου εργασίας
        Workbook workbook = new Workbook();
```

#### Βήμα 2: Πρόσβαση και διαμόρφωση του πρώτου φύλλου εργασίας

```csharp
        // Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
        Worksheet worksheet = workbook.Worksheets[0];

        // Εφαρμογή ενσωματωμένου στυλ «Τίτλος» στο κελί A1
        Style style = workbook.CreateBuiltinStyle(BuiltinStyleType.Title);
        Cell cell = worksheet.Cells["A1"];
        cell.PutValue("Aspose");
        cell.SetStyle(style);

        // Αυτόματη προσαρμογή της πρώτης στήλης και γραμμής
        worksheet.AutoFitColumn(0);
        worksheet.AutoFitRow(0);
```

#### Βήμα 3: Αποθήκευση σε πολλαπλές μορφές

```csharp
        // Αποθήκευση σε μορφή Excel (.xlsx)
        workbook.Save(output1Path);

        // Αποθήκευση σε μορφή υπολογιστικού φύλλου OpenDocument (.ods)
        workbook.Save(output2Path);
    }
}
```

### Χαρακτηριστικό: Στυλ κελιών με ενσωματωμένα στυλ
**Επισκόπηση**
Μάθετε πώς να εφαρμόζετε ενσωματωμένα στυλ, ενισχύοντας την οπτική ελκυστικότητα των κελιών σας.

#### Βήμα 1: Δημιουργία και εφαρμογή στυλ

```csharp
using Aspose.Cells;

public class FeatureCellStyling
{
    public static void Run()
    {
        string SourceDir = "YOUR_SOURCE_DIRECTORY";

        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.Worksheets[0];

        // Δημιουργήστε ενσωματωμένο στυλ «Τίτλος» και εφαρμόστε το στο κελί A1
        Style style = workbook.CreateBuiltinStyle(BuiltinStyleType.Title);
        Cell cell = worksheet.Cells["A1"];
        cell.PutValue("Aspose");
        cell.SetStyle(style);
    }
}
```

### Χαρακτηριστικό: Αυτόματη προσαρμογή γραμμών και στηλών
**Επισκόπηση**
Αυτή η λειτουργία παρουσιάζει τον τρόπο αυτόματης προσαρμογής των υψών γραμμών και των πλάτων στηλών για καλύτερη αναγνωσιμότητα.

#### Βήμα 1: Αυτόματη προσαρμογή της πρώτης γραμμής και στήλης

```csharp
using Aspose.Cells;

public class FeatureAutoFitRowsAndColumns
{
    public static void Run()
    {
        string SourceDir = "YOUR_SOURCE_DIRECTORY";

        Workbook workbook = new Workbook();
        Worksheet worksheet = workbook.Worksheets[0];

        // Αυτόματη προσαρμογή του πλάτους της πρώτης στήλης και του ύψους της πρώτης γραμμής
        worksheet.AutoFitColumn(0);
        worksheet.AutoFitRow(0);
    }
}
```
