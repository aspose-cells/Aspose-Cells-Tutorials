## Οδηγός Εφαρμογής

### Εξαγωγή χρωμάτων μορφοποίησης υπό όρους

Αυτή η ενότητα σας καθοδηγεί στην εξαγωγή χρωμάτων από κελιά με μορφοποίηση υπό όρους.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας σας

Φορτώστε το αρχείο Excel σας σε ένα `Workbook` αντικείμενο:

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Άνοιγμα του αρχείου προτύπου
Workbook workbook = new Workbook(dataDir + "Book1.xlsx");
```

#### Βήμα 2: Πρόσβαση στο Φύλλο εργασίας και στο Κελί

Μεταβείτε στο συγκεκριμένο φύλλο εργασίας και κελί:

```csharp
// Αποκτήστε το πρώτο φύλλο εργασίας
Worksheet worksheet = workbook.Worksheets[0];

// Αποκτήστε το κελί A1
Cell a1 = worksheet.Cells["A1"];
```

#### Βήμα 3: Εξαγωγή αποτελέσματος μορφοποίησης υπό όρους

Χρησιμοποιήστε τις μεθόδους Aspose.Cells για να ανακτήσετε αποτελέσματα μορφοποίησης υπό όρους και να αποκτήσετε πρόσβαση σε λεπτομέρειες χρωμάτων:

```csharp
// Λήψη του αντικειμένου που προκύπτει από τη μορφοποίηση υπό όρους
ConditionalFormattingResult cfr1 = a1.GetConditionalFormattingResult();

// Λήψη του χρωματικού αντικειμένου που προκύπτει από το ColorScale
Color c = cfr1.ColorScaleResult;

// Διαβάστε και εκτυπώστε το χρώμα
Console.WriteLine(c.ToArgb().ToString());
Console.WriteLine(c.Name);
```

**Εξήγηση**: 
- `GetConditionalFormattingResult()` ανακτά τη μορφοποίηση υπό όρους που εφαρμόζεται σε ένα κελί.
- `ColorScaleResult` παρέχει το ακριβές χρώμα που χρησιμοποιείται στη μορφοποίηση υπό όρους.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι το αρχείο Excel έχει μορφοποιηθεί και αποθηκευτεί σωστά πριν το φορτώσετε.
- Εάν τα χρώματα δεν εξάγονται όπως αναμένεται, επαληθεύστε ότι η μορφοποίηση υπό όρους εφαρμόζεται απευθείας στο κελί αντί να αποτελεί μέρος πιο σύνθετων κανόνων ή εύρους.

## Ενότητα Συχνών Ερωτήσεων

**1. Μπορώ να εξαγάγω χρώματα από όλους τους τύπους μορφοποίησης υπό όρους;**
   - Ναι, εφόσον η μορφοποίηση εφαρμόζεται απευθείας σε ένα κελί και δεν αποτελεί μέρος πιο σύνθετων κανόνων που περιλαμβάνουν πολλά κελιά ή εύρη.

**2. Πώς μπορώ να χειριστώ σφάλματα κατά τη φόρτωση αρχείων Excel;**
   - Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές και ότι το βιβλίο εργασίας δεν είναι κατεστραμμένο. Χρησιμοποιήστε μπλοκ try-catch για καλύτερο χειρισμό σφαλμάτων.

**3. Τι γίνεται αν η μορφοποίηση υπό όρους περιλαμβάνει διαβαθμίσεις;**
   - Το Aspose.Cells μπορεί να χειριστεί κλίμακες χρώματος διαβάθμισης, αλλά να εξαγάγει το χρώμα κάθε στάσης ξεχωριστά χρησιμοποιώντας `ColorScaleResult`.

**4. Υπάρχει όριο στον αριθμό των μορφοποιήσεων υπό όρους που μπορώ να επεξεργαστώ ταυτόχρονα;**
   - Δεν υπάρχει εγγενές όριο, αλλά η απόδοση μπορεί να διαφέρει ανάλογα με το μέγεθος του βιβλίου εργασίας και τους πόρους του συστήματος.

**5. Πώς μπορώ να εφαρμόσω αυτά τα εξαγόμενα χρώματα ξανά σε ένα άλλο αρχείο Excel;**
   - Χρήση του Aspose.Cells `SetStyle` μέθοδοι για την εφαρμογή των εξαγόμενων χρωμάτων σε κελιά σε διαφορετικό βιβλίο εργασίας.
