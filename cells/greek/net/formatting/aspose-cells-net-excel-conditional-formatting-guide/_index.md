# Mastering Aspose.Cells .NET για μορφοποίηση υπό όρους Excel

Μεταμορφώστε τα υπολογιστικά φύλλα του Excel σας με δυναμικά και οπτικά ελκυστικά δεδομένα χρησιμοποιώντας το Aspose.Cells για .NET. Αυτός ο ολοκληρωμένος οδηγός θα σας καθοδηγήσει στη διαδικασία εφαρμογής προηγμένων κανόνων μορφοποίησης υπό όρους για να βελτιώσετε τόσο τη χρηστικότητα όσο και την αισθητική στα υπολογιστικά φύλλα σας.

- Δημιουργία στιγμιαίου βιβλίου εργασίας και φύλλου εργασίας Excel
- Προσθήκη κανόνων μορφοποίησης υπό όρους σε κελιά
- Προσαρμογή χρωμάτων φόντου για επισημασμένα δεδομένα
- Αποθήκευση του μορφοποιημένου αρχείου Excel

Είστε έτοιμοι να αναβαθμίσετε την παρουσίαση δεδομένων σας; Ας ρυθμίσουμε το περιβάλλον σας και ας εμβαθύνουμε στον προγραμματισμό!

## Οδηγός Εφαρμογής
Θα αναλύσουμε κάθε χαρακτηριστικό της μορφοποίησης υπό όρους χρησιμοποιώντας το Aspose.Cells για .NET σε διαχειρίσιμα βήματα.

### Δημιουργία αρχικού βιβλίου εργασίας και φύλλου εργασίας
**Επισκόπηση:** Αυτή η ενότητα παρουσιάζει τη δημιουργία ενός νέου βιβλίου εργασίας του Excel και την πρόσβαση στο πρώτο φύλλο εργασίας του.

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας
```csharp
// Αρχικοποιήστε το αντικείμενο του βιβλίου εργασίας.
Workbook workbook = new Workbook();
```
- **Παράμετροι & Σκοπός**: Το `Workbook` Ο κατασκευαστής αρχικοποιεί ένα νέο αρχείο Excel. Από προεπιλογή, δημιουργεί ένα κενό φύλλο εργασίας.

#### Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας
```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.
Worksheet sheet = workbook.Worksheets[0];
```
Ο `Worksheets[0]` Το index έχει πρόσβαση στο αρχικό φύλλο εργασίας που δημιουργήθηκε με το βιβλίο εργασίας.

### Προσθήκη κανόνων μορφοποίησης υπό όρους
**Επισκόπηση:** Μάθετε πώς να ορίζετε κανόνες μορφοποίησης υπό όρους για συγκεκριμένες περιοχές κελιών μέσα σε ένα φύλλο εργασίας.

#### Βήμα 1: Προσθήκη νέου κανόνα μορφοποίησης υπό όρους
```csharp
// Προσθήκη νέου κανόνα μορφοποίησης υπό όρους.
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcs = sheet.ConditionalFormattings[index];
```
- **Σκοπός**: `ConditionalFormattings.Add()` δημιουργεί έναν νέο κανόνα και επιστρέφει τον δείκτη του.

#### Βήμα 2: Ορίστε την περιοχή κελιού
```csharp
// Ορίστε περιοχές κελιών για την εφαρμογή μορφοποίησης υπό όρους.
CellArea ca = new CellArea();
ca.StartRow = 0;
c.EndRow = 0;
ca.StartColumn = 0;
c.EndColumn = 0;
fcs.AddArea(ca);

c = new CellArea();
ca.StartRow = 1;
c.EndRow = 1;
c.StartColumn = 1;
c.EndColumn = 1;
fcs.AddArea(c);
```
- **Σκοπός**: `CellArea` Τα αντικείμενα καθορίζουν πού θα εφαρμοστεί η μορφοποίηση υπό όρους.

#### Βήμα 3: Προσθήκη συνθηκών
```csharp
// Ορίστε συνθήκες για τον κανόνα μορφοποίησης.
int conditionIndex = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "=A2", "100");
```
- **Σκοπός**: `AddCondition()` προσθέτει έναν νέο κανόνα που βασίζεται σε τιμές κελιών.

### Ορισμός χρώματος φόντου για μορφοποίηση υπό όρους
**Επισκόπηση:** Προσαρμόστε την εμφάνιση των κελιών που πληρούν συγκεκριμένες συνθήκες αλλάζοντας το χρώμα φόντου τους.

#### Βήμα 1: Ορισμός χρώματος φόντου
```csharp
// Αλλάξτε το χρώμα φόντου σε κόκκινο εάν πληρούται η συνθήκη.
FormatCondition fc = fcs[conditionIndex];
fc.Style.BackgroundColor = Color.Red;
```
- **Σκοπός**: `Style.BackgroundColor` Ορίζει το χρώμα φόντου για τα κελιά που πληρούν τον κανόνα υπό όρους.

### Αποθήκευση του αρχείου Excel
**Επισκόπηση:** Μάθετε πώς να αποθηκεύετε το βιβλίο εργασίας σας αφού εφαρμόσετε όλους τους κανόνες μορφοποίησης.

#### Βήμα 1: Αποθήκευση του βιβλίου εργασίας
```csharp
// Καθορίστε τον κατάλογο εξόδου και το όνομα αρχείου.
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "output.xls");
```
- **Σκοπός**: `Save()` γράφει το βιβλίο εργασίας σε μια καθορισμένη διαδρομή με ένα δεδομένο όνομα αρχείου.
