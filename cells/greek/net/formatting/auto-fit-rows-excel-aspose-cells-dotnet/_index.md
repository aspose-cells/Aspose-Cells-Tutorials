## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να υλοποιούμε τη λειτουργία αυτόματης προσαρμογής χρησιμοποιώντας το Aspose.Cells για .NET.

### Λειτουργία αυτόματης προσαρμογής γραμμής

Αυτή η λειτουργικότητα σάς επιτρέπει να προσαρμόζετε αυτόματα το ύψος μιας συγκεκριμένης γραμμής με βάση το περιεχόμενό της. Δείτε πώς:

#### Βήμα 1: Φόρτωση του αρχείου Excel

Ανοίξτε ένα υπάρχον αρχείο Excel χρησιμοποιώντας ένα FileStream, το οποίο παρέχει αποτελεσματικούς τρόπους ανάγνωσης και εγγραφής αρχείων σε .NET.
```csharp
using System.IO;
using Aspose.Cells;

// Ορίστε τη διαδρομή του καταλόγου προέλευσης.
string SourceDir = "YOUR_SOURCE_DIRECTORY";

// Δημιουργήστε μια ροή αρχείων για το αρχείο Excel.
FileStream fstream = new FileStream(SourceDir + "/Book1.xlsx", FileMode.Open);

// Ανοίξτε το βιβλίο εργασίας χρησιμοποιώντας τη ροή αρχείων.
Workbook workbook = new Workbook(fstream);
```

#### Βήμα 2: Πρόσβαση και αυτόματη προσαρμογή της σειράς

Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας και χρησιμοποιήστε το `AutoFitRow` μέθοδος για τη ρύθμιση του ύψους της σειράς.
```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.
Worksheet worksheet = workbook.Worksheets[0];

// Αυτόματη προσαρμογή της τρίτης γραμμής (ο δείκτης ξεκινά από το 0).
worksheet.AutoFitRow(1); // Ρυθμίζει το ύψος με βάση το περιεχόμενό του
```

#### Βήμα 3: Αποθήκευση και κλείσιμο

Αφού κάνετε τις προσαρμογές, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο και βεβαιωθείτε ότι οι πόροι έχουν ελευθερωθεί σωστά κλείνοντας το FileStream.
```csharp
// Ορίστε τη διαδρομή του καταλόγου εξόδου σας.
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Αποθηκεύστε το βιβλίο εργασίας με τα προσαρμοσμένα ύψη γραμμών.
workbook.Save(outputDir + "/output.xlsx");

// Να κλείνετε πάντα τη ροή για να απελευθερώσετε όλους τους πόρους.
fstream.Close();
```
