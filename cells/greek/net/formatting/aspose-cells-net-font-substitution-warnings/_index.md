Αυτός ο οδηγός καλύπτει τη ρύθμιση μιας προειδοποίησης επιστροφής κλήσης για αντικατάσταση γραμματοσειράς και τη μετατροπή ενός βιβλίου εργασίας του Excel σε PDF με αυτές τις προειδοποιήσεις στη θέση τους.

### Υλοποίηση προειδοποίησης αντικατάστασης γραμματοσειράς με επανάκληση
#### Επισκόπηση
Ο στόχος εδώ είναι να δημιουργηθεί ένας μηχανισμός που θα σας ειδοποιεί κάθε φορά που η βιβλιοθήκη αντικαθιστά μια γραμματοσειρά κατά τη μετατροπή, διασφαλίζοντας ότι η έξοδος ανταποκρίνεται στις προσδοκίες.

#### Βήμα προς βήμα εφαρμογή
**Δημιουργήστε την κλάση επανάκλησης**
Ορίστε μια κλάση που υλοποιεί `IWarningCallback` για τη διαχείριση προειδοποιήσεων κατά τη διάρκεια λειτουργιών όπως οι μετατροπές:
```csharp
using Aspose.Cells;
using System.Diagnostics;

public class GetWarningsForFontSubstitution : IWarningCallback
{
    // Μέθοδος για την καταγραφή και την καταγραφή προειδοποιήσεων αντικατάστασης γραμματοσειρών.
    public void Warning(WarningInfo info)
    {
        if (info.WarningType == WarningType.FontSubstitution)
        {
            Debug.WriteLine("WARNING INFO: " + info.Description);
        }
    }
}
```

**Εξήγηση:** Αυτή η κλάση παρακολουθεί για προειδοποιητικά συμβάντα κατά τη μετατροπή. Εάν ο τύπος συμβάντος είναι `FontSubstitution`, καταγράφει ένα λεπτομερές μήνυμα χρησιμοποιώντας `Debug.WriteLine`.

### Μετατροπή βιβλίου εργασίας σε PDF με προειδοποιήσεις αντικατάστασης γραμματοσειράς
#### Επισκόπηση
Έχοντας έτοιμη την επανάκληση προειδοποίησης, ας τη χρησιμοποιήσουμε για να μετατρέψουμε ένα βιβλίο εργασίας του Excel σε αρχείο PDF, καταγράφοντας παράλληλα προειδοποιήσεις αντικατάστασης γραμματοσειράς.

**Υλοποίηση της μετατροπής**
Δημιουργήστε μια στατική κλάση και μια μέθοδο για τον χειρισμό της διαδικασίας μετατροπής:
```csharp
using Aspose.Cells;
using System.IO;

public static class ConvertWorkbookToPdfWithWarnings
{
    public static void Run()
    {
        // Ορίστε τους καταλόγους προέλευσης και εξόδου.
        string SourceDir = "YOUR_SOURCE_DIRECTORY";
        string OutputDir = "YOUR_OUTPUT_DIRECTORY";

        // Φορτώστε το βιβλίο εργασίας του Excel από τον καθορισμένο κατάλογο.
        Workbook workbook = new Workbook(SourceDir + "sampleGetWarningsForFontSubstitution.xlsx");

        // Δημιουργήστε μια παρουσία του PdfSaveOptions για να προσαρμόσετε τις επιλογές αποθήκευσης.
        PdfSaveOptions options = new PdfSaveOptions();

        // Αντιστοιχίστε την επανάκληση προειδοποίησης για τη διαχείριση των προειδοποιήσεων αντικατάστασης γραμματοσειράς.
        options.WarningCallback = new GetWarningsForFontSubstitution();

        // Αποθηκεύστε το βιβλίο εργασίας ως αρχείο PDF, χρησιμοποιώντας τις καθορισμένες επιλογές.
        workbook.Save(OutputDir + "outputGetWarningsForFontSubstitution.pdf", options);
    }
}
```

**Εξήγηση:** Αυτός ο κώδικας φορτώνει ένα αρχείο Excel και το ρυθμίζει `PdfSaveOptions` για να χρησιμοποιήσετε την προσαρμοσμένη μας προειδοποίηση επιστροφής κλήσης. Όταν καλείτε `workbook.Save`, τυχόν προειδοποιήσεις αντικατάστασης γραμματοσειράς καταγράφονται από την επανάκληση, επιτρέποντας καλύτερο έλεγχο της ποιότητας της εξόδου σας.
