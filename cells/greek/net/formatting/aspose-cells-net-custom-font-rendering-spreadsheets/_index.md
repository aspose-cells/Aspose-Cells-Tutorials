## Οδηγός Εφαρμογής

### Λειτουργία 1: Ορισμός προεπιλεγμένης γραμματοσειράς κατά την απόδοση υπολογιστικού φύλλου

#### Επισκόπηση
Αυτή η λειτουργία διασφαλίζει την συνεπή απόδοση των γραμματοσειρών των υπολογιστικών φύλλων, ακόμα και αν οι καθορισμένες γραμματοσειρές λείπουν ή είναι άγνωστες.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Προετοιμασία του βιβλίου εργασίας σας**
Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας και ορίστε το προεπιλεγμένο στυλ του:
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook();
Style s = wb.DefaultStyle;
s.Font.Name = "Arial"; // Ορίστε μια αρχική προεπιλεγμένη γραμματοσειρά.
wb.DefaultStyle = s;
```
**Βήμα 2: Διαμόρφωση του φύλλου εργασίας σας**
Αποκτήστε πρόσβαση στο φύλλο εργασίας σας, ορίστε τιμές κελιών και εφαρμόστε στυλ:
```csharp
Worksheet ws = wb.Worksheets[0];
Cell cell = ws.Cells["A4"];
cell.PutValue("This text uses a custom default font.");

Style st = cell.GetStyle();
st.Font.Name = "UnknownNotExist"; // Χρησιμοποιήστε σκόπιμα μια μη διαθέσιμη γραμματοσειρά.
st.Font.Size = 20;
st.IsTextWrapped = true;
cell.SetStyle(st);

// Προσαρμόστε το πλάτος της στήλης και το ύψος της γραμμής για καλύτερη οπτικοποίηση:
ws.Cells.SetColumnWidth(0, 80);
ws.Cells.SetRowHeight(3, 60);
```
**Βήμα 3: Απόδοση με προσαρμοσμένες γραμματοσειρές**
Ορίστε επιλογές εικόνας για την απόδοση του φύλλου εργασίας σας χρησιμοποιώντας διαφορετικές προεπιλεγμένες γραμματοσειρές:
```csharp
using Aspose.Cells.Rendering;

ImageOrPrintOptions opts = new ImageOrPrintOptions();
opts.OnePagePerSheet = true;
opts.ImageType = Drawing.ImageType.Png;

// Απόδοση με την προεπιλεγμένη γραμματοσειρά 'Arial'.
opts.DefaultFont = "Arial";
SheetRender sr = new SheetRender(ws, opts);
sr.ToImage(0, System.IO.Path.Combine(outputDir, "out_a.png"));

// Αλλαγή σε 'Times New Roman'.
opts.DefaultFont = "Times New Roman";
sr = new SheetRender(ws, opts);
sr.ToImage(0, System.IO.Path.Combine(outputDir, "times_new_roman_out.png"));
```
### Λειτουργία 2: Ορισμός πλάτους στήλης και ύψους γραμμής

#### Επισκόπηση
Η ρύθμιση του πλάτους των στηλών και του ύψους των γραμμών διασφαλίζει την καθαρή και επαγγελματική εμφάνιση δεδομένων.

**Βήμα προς βήμα εφαρμογή**
**Βήμα 1: Προσαρμογή διαστάσεων**
Αποκτήστε πρόσβαση στο φύλλο εργασίας και ορίστε συγκεκριμένες διαστάσεις:
```csharp
Worksheet ws = wb.Worksheets[0];
ws.Cells.SetColumnWidth(0, 80); // Ορίστε το πλάτος της πρώτης στήλης.
ws.Cells.SetRowHeight(3, 60);   // Ορίστε το ύψος της τέταρτης σειράς.
```
