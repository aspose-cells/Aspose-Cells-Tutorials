### Πρόσβαση και τροποποίηση στυλ κελιών

Ο πυρήνας αυτού του σεμιναρίου είναι ο χειρισμός του στυλ γραμματοσειράς. Δείτε πώς:

#### Ορισμός πάχους γραμματοσειράς σε έντονη γραφή

Για να κάνετε κείμενο με έντονη γραφή, αποκτήστε πρόσβαση στο αντικείμενο στυλ του επιθυμητού κελιού:

```csharp
// Πρόσβαση στο κελί "A1".
Aspose.Cells.Cell cell = worksheet.Cells["A1"];

// Προσθήκη τιμής στο κελί.
cell.PutValue("Hello Aspose!");

// Λάβετε το αντικείμενο στυλ που σχετίζεται με το κελί.
Style style = cell.GetStyle();

// Ορισμός πάχους γραμματοσειράς σε έντονη γραφή.
style.Font.IsBold = true;

// Εφαρμόστε το στυλ πίσω στο κελί.
cell.SetStyle(style);
```

#### Επεξήγηση Κώδικα
- **GetStyle()**: Ανακτά τις τρέχουσες ρυθμίσεις στυλ ενός κελιού.
- **Γραμματοσειρά.ΕίναιΈντονη**: Ιδιότητα που ελέγχει την έντονη γραφή του κειμένου. Ορισμός της σε `true` εφαρμόζει έντονη γραφή.
