## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε διαχειρίσιμα μέρη, εστιάζοντας στην εφαρμογή ενσωματωμένων μορφών αριθμών σε διαφορετικούς τύπους δεδομένων.

### Ρύθμιση του βιβλίου εργασίας σας

#### Επισκόπηση
Ξεκινήστε δημιουργώντας ένα νέο αρχείο Excel και λάβετε αναφορές στα φύλλα εργασίας του. Αυτό το βήμα είναι κρίσιμο για τον αποτελεσματικό χειρισμό των στυλ κελιών.

**Δημιουργία βιβλίου εργασίας**
```csharp
// Δημιουργία νέας παρουσίας βιβλίου εργασίας
Workbook workbook = new Workbook();

// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

### Μορφοποίηση ημερομηνιών

#### Επισκόπηση
Η εμφάνιση ημερομηνιών σε φιλική προς το χρήστη μορφή είναι απαραίτητη για λόγους σαφήνειας. Ας εφαρμόσουμε τη μορφή "η-μμμ-εε" σε ένα κελί.

**Εφαρμογή μορφής ημερομηνίας**
```csharp
// Εισαγάγετε την τρέχουσα ημερομηνία στο κελί A1
worksheet.Cells["A1"].PutValue(DateTime.Now);

// Ανάκτηση και τροποποίηση του στυλ του κελιού
Style style = worksheet.Cells["A1"].GetStyle();
style.Number = 15; // Ενσωματωμένη μορφή για "d-mmm-yy"
worksheet.Cells["A1"].SetStyle(style);
```

### Μορφοποίηση ποσοστών

#### Επισκόπηση
Η μετατροπή αριθμητικών τιμών σε ποσοστά μπορεί να βελτιώσει την ερμηνεία των δεδομένων, ειδικά στις οικονομικές αναφορές.

**Εφαρμογή μορφής ποσοστού**
```csharp
// Εισαγωγή αριθμητικής τιμής στο κελί A2
worksheet.Cells["A2"].PutValue(20);

// Τροποποίηση του στυλ για την εμφάνιση ποσοστού
style = worksheet.Cells["A2"].GetStyle();
style.Number = 9; // Ενσωματωμένη μορφή για ποσοστά
worksheet.Cells["A2"].SetStyle(style);
```

### Μορφοποίηση Νομίσματος

#### Επισκόπηση
Τα οικονομικά δεδομένα συχνά απαιτούν μορφοποίηση νομισμάτων για να διασφαλιστεί η συνέπεια μεταξύ των αναφορών.

**Εφαρμογή μορφής νομίσματος**
```csharp
// Εισαγωγή αριθμητικής τιμής στο κελί A3
worksheet.Cells["A3"].PutValue(2546);

// Ορισμός του στυλ εμφάνισης νομίσματος
style = worksheet.Cells["A3"].GetStyle();
style.Number = 6; // Ενσωματωμένη μορφή για νόμισμα
worksheet.Cells["A3"].SetStyle(style);
```

### Αποθήκευση του βιβλίου εργασίας σας
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε ένα αρχείο Excel:
```csharp
// Αποθήκευση του βιβλίου εργασίας σε μορφή Excel97To2003
workbook.Save("path/to/your/book1.out.xls", SaveFormat.Excel97To2003);
```
