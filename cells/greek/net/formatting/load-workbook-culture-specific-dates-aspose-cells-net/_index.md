## Οδηγός Εφαρμογής
Αυτή η ενότητα σάς καθοδηγεί στη φόρτωση βιβλίων εργασίας με μορφές ημερομηνίας που αφορούν συγκεκριμένες καλλιέργειες χρησιμοποιώντας το Aspose.Cells.

### Ρύθμιση παραμέτρων μορφών ημερομηνίας ανάλογα με τον πολιτισμό
Για να διασφαλίσετε ότι η εφαρμογή σας ερμηνεύει σωστά τις ημερομηνίες από διαφορετικές τοπικές ρυθμίσεις, διαμορφώστε το `CultureInfo` ρυθμίσεις ώστε να ταιριάζουν με την αναμενόμενη μορφή.

#### Ρύθμιση επιλογών φόρτωσης με το CultureInfo
1. **Δημιουργήστε ένα MemoryStream για δεδομένα εισόδου**Προσομοίωση ανάγνωσης δεδομένων από ένα αρχείο HTML.
2. **Σύνταξη περιεχομένου HTML με ημερομηνίες**Συμπεριλάβετε μια ημερομηνία σε μορφή που ταιριάζει σε κάθε πολιτισμό.
3. **Ρύθμιση παραμέτρων πολιτισμού**:
   - Σειρά `NumberDecimalSeparator`, `DateSeparator`, και `ShortDatePattern`.
4. **Χρησιμοποιήστε το LoadOptions για να καθορίσετε το CultureInfo**:

```csharp
using System;
using System.IO;
using System.Globalization;
using Aspose.Cells;

class LoadWorkbookWithSpecificCultureInfoDateFormat
{
    public static void Run()
    {
        using (var inputStream = new MemoryStream())
        {
            using (var writer = new StreamWriter(inputStream))
            {
                // Γράψτε περιεχόμενο HTML με ημερομηνία στη μορφή "ηη-μμ-εεεε"
                writer.WriteLine("<html><head><title>Test Culture</title></head><body><table><tr><td>10-01-2016</td></tr></table></body></html>");
                writer.Flush();
                
                // Ρύθμιση παραμέτρων πολιτισμού για τη μορφή ημερομηνίας Ηνωμένου Βασιλείου
                var culture = new CultureInfo("en-GB");
                culture.NumberFormat.NumberDecimalSeparator = ",";
                culture.DateTimeFormat.DateSeparator = "-";
                culture.DateTimeFormat.ShortDatePattern = "dd-MM-yyyy";

                // Δημιουργήστε LoadOptions με την καθορισμένη κουλτούρα
                LoadOptions options = new LoadOptions(LoadFormat.Html);
                options.CultureInfo = culture;

                // Φόρτωση βιβλίου εργασίας χρησιμοποιώντας InputStream και LoadOptions
                using (var workbook = new Workbook(inputStream, options))
                {
                    var cell = workbook.Worksheets[0].Cells["A1"];
                    
                    // Επιβεβαιώστε ότι η ημερομηνία ερμηνεύεται σωστά ως Ημερομηνία/Ώρα
                    Console.WriteLine("Date Type: " + cell.Type == CellValueType.IsDateTime);
                    Console.WriteLine("Parsed Date: " + cell.DateTimeValue.ToString(culture));
                }
            }
        }
        
        Console.WriteLine("LoadWorkbookWithSpecificCultureInfoDateFormat executed successfully.");
    }
}
```

**Παράμετροι και Σκοπός:**
- **Ροή μνήμης**: Προσομοιώνει την ανάγνωση δεδομένων σαν να προέρχεται από αρχείο.
- **Πληροφορίες Πολιτισμού**: Ρυθμίζει την εφαρμογή ώστε να ερμηνεύει ημερομηνίες σε `dd-MM-yyyy` μορφή, κρίσιμη για τον χειρισμό ημερομηνιών στο Ηνωμένο Βασίλειο.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε για τις πολιτιστικές σας ρυθμίσεις (`DateSeparator`, `ShortDatePattern`) ταιριάζουν με αυτά που χρησιμοποιούνται στο βιβλίο εργασίας.
- Επαληθεύστε ότι η εισαγωγή HTML έχει σωστή μορφοποίηση και είναι προσβάσιμη από το MemoryStream.
