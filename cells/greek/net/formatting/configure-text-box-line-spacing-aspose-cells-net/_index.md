# Ρύθμιση παραμέτρων απόστασης γραμμών πλαισίου κειμένου με το Aspose.Cells .NET: Οδηγός βήμα προς βήμα

## Εισαγωγή
Όταν εργάζεστε με υπολογιστικά φύλλα Excel μέσω προγραμματισμού, η βελτίωση της αναγνωσιμότητας μέσω προσαρμοσμένης μορφοποίησης κειμένου είναι ζωτικής σημασίας. **Aspose.Cells για .NET** επιτρέπει στους προγραμματιστές να δημιουργούν και να χειρίζονται αρχεία Excel χωρίς κόπο. Αυτό το σεμινάριο σας καθοδηγεί στη διαμόρφωση της απόστασης γραμμών σε ένα πλαίσιο κειμένου μέσα σε ένα φύλλο εργασίας Excel χρησιμοποιώντας το Aspose.Cells για .NET. Είτε δημιουργείτε αναφορές είτε αυτοματοποιείτε τη δημιουργία εγγράφων, αυτές οι τεχνικές μπορούν να βελτιώσουν σημαντικά την αισθητική του υπολογιστικού σας φύλλου.

**Τι θα μάθετε:**
- Δημιουργήστε και αποκτήστε πρόσβαση σε ένα νέο βιβλίο εργασίας και τα φύλλα εργασίας του.
- Προσθήκη σχήματος πλαισίου κειμένου σε ένα φύλλο εργασίας.
- Ορίστε και μορφοποιήστε το κείμενο μέσα στο σχήμα, συμπεριλαμβανομένων των προσαρμογών της απόστασης μεταξύ γραμμών.
- Αποθήκευση τροποποιήσεων σε μορφή Excel.

## Οδηγός Εφαρμογής
Ας χωρίσουμε την υλοποίηση σε σαφή τμήματα με βάση τη λειτουργικότητα.

### Δημιουργία και πρόσβαση σε ένα βιβλίο εργασίας
**Επισκόπηση**Ξεκινήστε δημιουργώντας ένα βιβλίο εργασίας του Excel και αποκτώντας πρόσβαση στο πρώτο φύλλο εργασίας του. Αυτό χρησιμεύει ως καμβάς μας για περαιτέρω λειτουργίες.

#### Βήμα 1: Αρχικοποίηση βιβλίου εργασίας
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";

Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```
Εδώ, αρχικοποιούμε ένα `Workbook` αντικείμενο και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας του χρησιμοποιώντας `ws = wb.Worksheets[0]`.

### Προσθήκη πλαισίου κειμένου σε φύλλο εργασίας
**Επισκόπηση**Βελτιώστε το φύλλο εργασίας σας προσθέτοντας ένα σχήμα πλαισίου κειμένου.

#### Βήμα 2: Προσθήκη σχήματος πλαισίου κειμένου
```csharp
using Aspose.Cells.Drawing;

Shape shape = ws.Shapes.AddTextBox(2, 0, 2, 0, 100, 200);
```
Προσθέτουμε ένα `TextBox` στο φύλλο εργασίας στις καθορισμένες διαστάσεις (x, y, πλάτος, ύψος).

### Ορισμός κειμένου σε σχήμα
**Επισκόπηση**: Συμπληρώστε το πλαίσιο κειμένου σας με περιεχόμενο και αποκτήστε πρόσβαση σε παραγράφους για μορφοποίηση.

#### Βήμα 3: Ορισμός περιεχομένου κειμένου
```csharp
shape.Text = "Sign up for your free phone number.\nCall and text online for free.";
TextParagraph p = shape.TextBody.TextParagraphs[1];
```
Αυτό το απόσπασμα ορίζει το κείμενο στο σχήμα και επιλέγει μια παράγραφο για περαιτέρω προσαρμογή.

### Ρύθμιση απόστασης μεταξύ παραγράφων
**Επισκόπηση**Προσαρμόστε την απόσταση μεταξύ γραμμών, το διάστημα πριν και το διάστημα μετά μέσα στο πλαίσιο κειμένου για να βελτιώσετε την αναγνωσιμότητα.

#### Βήμα 4: Ορισμός απόστασης μεταξύ γραμμών
```csharp
using Aspose.Cells.Drawing.Texts;

p.LineSpaceSizeType = LineSpaceSizeType.Points; // Χρησιμοποιήστε σημεία για ακριβή έλεγχο
p.LineSpace = 20; // Διάστιχο 20 σημείων

// Ρύθμιση παραμέτρων διαστήματος μετά την παράγραφο
p.SpaceAfterSizeType = LineSpaceSizeType.Points;
p.SpaceAfter = 10;

// Ρύθμιση παραμέτρων κενού πριν από την παράγραφο
p.SpaceBeforeSizeType = LineSpaceSizeType.Points;
p.SpaceBefore = 10;
```
Αυτές οι ρυθμίσεις βελτιώνουν την εμφάνιση του κειμένου σας, βελτιώνοντας την αναγνωσιμότητα.

### Αποθήκευση βιβλίου εργασίας
**Επισκόπηση**: Μόλις ολοκληρωθεί η διαμόρφωση, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε τις αλλαγές.

#### Βήμα 5: Αποθήκευση αλλαγών
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
wb.Save(outputDir + "/outputSetTextboxOrShapeParagraphLineSpacing.xlsx", SaveFormat.Xlsx);
```
Αυτή η εντολή γράφει ξανά το τροποποιημένο βιβλίο εργασίας σε ένα αρχείο Excel σε μορφή XLSX.
