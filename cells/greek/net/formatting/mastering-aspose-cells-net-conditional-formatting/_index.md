## Οδηγός Εφαρμογής
### Βασικά στοιχεία μορφοποίησης υπό όρους
Η μορφοποίηση υπό όρους στο Aspose.Cells σάς επιτρέπει να αναπαραστήσετε οπτικά μοτίβα δεδομένων και τάσεις εφαρμόζοντας κανόνες όπως κλίμακες χρωμάτων, σύνολα εικονιδίων και λίστες με τις δέκα κορυφαίες λίστες.
#### Μορφοποίηση κλίμακας χρωμάτων
**Επισκόπηση:**
Εφαρμόστε μια διαβάθμιση χρωμάτων με βάση τις τιμές των κελιών χρησιμοποιώντας μια κλίμακα τριών χρωμάτων.
```csharp
// Δημιουργήστε ένα βιβλίο εργασίας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];

// Ορισμός δεδομένων για επίδειξη
sheet.Cells["A1"].PutValue(10);
sheet.Cells["A2"].PutValue(20);
sheet.Cells["A3"].PutValue(30);

// Προσθήκη μορφοποίησης υπό όρους κλίμακας χρώματος σε ένα εύρος
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcc = sheet.ConditionalFormattings[index];
fcc.AddArea(new CellArea(0, 0, 2, 0)); // Εύρος: A1:A3

// Ορίστε την πρώτη συνθήκη (ελάχιστη τιμή)
StyleFlag styleFlag = new StyleFlag { All = true };
Style lowerStyle = workbook.CreateStyle();
lowerStyle.ForegroundColor = Color.Red;
lowerStyle.Pattern = BackgroundType.Solid;

int conditionIndex = fcc.AddCondition(FormatConditionType.ColorScale);
FormatCondition fc = fcc[conditionIndex];
fc.FirstValue = 10; // Ελάχ
fc.SecondValue = 20; // Στα μέσα
fc.Type = FormatConditionType.ColorScale;
fc.ColorScale.MinColor = Color.Red;
fc.ColorScale.MidColor = Color.Yellow;
fc.ColorScale.MaxColor = Color.Green;

fcc[0].Style = lowerStyle;
fcc.SetStyle(styleFlag);

// Αποθήκευση του βιβλίου εργασίας
workbook.Save("ColorScaleConditionalFormatting.xlsx");
```
**Εξήγηση:**
- **Περιοχή Κυψέλης(0, 0, 2, 0)** ορίζει το εύρος από A1 έως A3.
- Η κλίμακα χρωμάτων εφαρμόζεται χρησιμοποιώντας τρία χρώματα για ελάχιστες, μεσαίες και μέγιστες τιμές.
#### Μορφοποίηση συνόλου εικονιδίων
**Επισκόπηση:**
Βελτιώστε την αναγνωσιμότητα των δεδομένων εφαρμόζοντας σύνολα εικονιδίων που υποδεικνύουν οπτικά εύρη τιμών ή τάσεις.
```csharp
// Δημιουργήστε ένα βιβλίο εργασίας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];

// Προσθήκη δειγμάτων δεδομένων σε κελιά
sheet.Cells["B1"].PutValue(100);
sheet.Cells["B2"].PutValue(200);
sheet.Cells["B3"].PutValue(300);

// Προσθήκη μορφοποίησης υπό όρους σε ένα σύνολο εικονιδίων σε ένα εύρος
int index = sheet.ConditionalFormattings.Add();
FormatConditionCollection fcc = sheet.ConditionalFormattings[index];
fcc.AddArea(new CellArea(0, 1, 2, 1)); // Εύρος: B1:B3

// Ορίστε τη συνθήκη για το σύνολο εικονιδίων
int conditionIndex = fcc.AddCondition(FormatConditionType.IconSet);
FormatCondition fc = fcc[conditionIndex];
fc.SetIconSet(IconSetType.TenArrows); // Ορισμός σε ένα προκαθορισμένο σύνολο εικονιδίων

fcc[0].Style = workbook.CreateStyle();
sheet.Cells["B1"].AddComment("Lower values", "author");

// Αποθήκευση του βιβλίου εργασίας
workbook.Save("IconSetConditionalFormatting.xlsx");
```
**Εξήγηση:**
- **IconSetType.TenArrows** Εφαρμόζει μια σειρά από δέκα διαφορετικά εικονίδια με βάση τα εύρη τιμών κελιών.
