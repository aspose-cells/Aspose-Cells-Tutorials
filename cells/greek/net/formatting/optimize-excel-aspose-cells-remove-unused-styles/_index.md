## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να υλοποιούμε τη λειτουργία κατάργησης αχρησιμοποίητων στυλ από ένα βιβλίο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET.

### Φόρτωση και κατάργηση αχρησιμοποίητων στυλ σε βιβλία εργασίας του Excel

Αυτή η λειτουργία βοηθά στη μείωση του μεγέθους του αρχείου εξαλείφοντας τα αχρησιμοποίητα στυλ, βελτιώνοντας την απόδοση της εφαρμογής σας.

#### Βήμα 1: Ρύθμιση του περιβάλλοντος σας

Ξεκινήστε καθορίζοντας διαδρομές για τους καταλόγους προέλευσης και εξόδου. Αντικαταστήστε `YOUR_SOURCE_DIRECTORY` και `YOUR_OUTPUT_DIRECTORY` με τις πραγματικές διαδρομές στο σύστημά σας.

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση του βιβλίου εργασίας

Δημιουργήστε μια νέα παρουσία του `Workbook` κλάση, φορτώνοντας ένα αρχείο Excel που περιέχει αχρησιμοποίητα στυλ:

```csharp
// Φόρτωση του βιβλίου εργασίας από τον κατάλογο προέλευσης
Workbook workbook = new Workbook(SourceDir + "/sampleRemoveUnusedStyles.xlsx");
```

#### Βήμα 3: Αφαίρεση αχρησιμοποίητων στυλ

Επικαλέστε το `RemoveUnusedStyles()` μέθοδος για τον καθαρισμό του βιβλίου εργασίας. Αυτή η λειτουργία καταργεί τυχόν ορισμούς στυλ που δεν χρησιμοποιούνται στο βιβλίο εργασίας, βελτιστοποιώντας το μέγεθός του:

```csharp
// Καθαρισμός αχρησιμοποίητων στυλ από το βιβλίο εργασίας
workbook.RemoveUnusedStyles();
```

#### Βήμα 4: Αποθήκευση του Βελτιστοποιημένου Βιβλίου Εργασίας

Τέλος, αποθηκεύστε το βελτιστοποιημένο βιβλίο εργασίας στον καθορισμένο κατάλογο εξόδου:

```csharp
// Εξαγωγή του καθαρισμένου βιβλίου εργασίας
workbook.Save(outputDir + "/outputRemoveUnusedStyles.xlsx");
```
