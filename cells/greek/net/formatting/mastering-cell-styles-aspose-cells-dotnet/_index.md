## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα για την εφαρμογή στυλ κελιών στο Excel.

### Δημιουργία και διαμόρφωση φύλλου εργασίας Excel

Θα ξεκινήσουμε δημιουργώντας ένα νέο φύλλο εργασίας και εφαρμόζοντας προσαρμοσμένα στυλ στα κελιά του.

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας
Ξεκινήστε δημιουργώντας ένα στιγμιότυπο του `Workbook` αντικείμενο. Αυτό θα είναι το κύριο κοντέινερ για όλες τις λειτουργίες.

```csharp
Workbook workbook = new Workbook();
```

#### Βήμα 2: Προσθήκη φύλλου εργασίας
Προσθέστε ένα νέο φύλλο εργασίας όπου μπορείτε να εφαρμόσετε διάφορα στυλ για να επιδείξετε ευελιξία.

```csharp
int sheetIndex = workbook.Worksheets.Add(); // Προσθέτει ένα νέο φύλλο εργασίας και επιστρέφει τον δείκτη του
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

#### Βήμα 3: Ορισμός στυλ για κελιά

Κάθε διαμόρφωση στυλ κελιού σάς επιτρέπει να ορίσετε χρώματα προσκηνίου και φόντου, καθώς και μοτίβα όπως κάθετες ρίγες.

##### Εφαρμογή στυλ στο κελί A1

Ας ξεκινήσουμε ορίζοντας ένα κίτρινο χρώμα με μοτίβο κάθετης λωρίδας στο κελί A1.

```csharp
Style styleA1 = worksheet.Cells["A1"].GetStyle();
styleA1.ForegroundColor = Color.Yellow;
styleA1.Pattern = BackgroundType.VerticalStripe;
worksheet.Cells["A1"].SetStyle(styleA1);
```

##### Εφαρμογή στυλ στο κελί A2

Στη συνέχεια, διαμορφώστε το κελί A2 με μπλε προσκήνιο και κίτρινο φόντο.

```csharp
Style styleA2 = worksheet.Cells["A2"].GetStyle();
styleA2.ForegroundColor = Color.Blue;
styleA2.BackgroundColor = Color.Yellow;
styleA2.Pattern = BackgroundType.VerticalStripe;
worksheet.Cells["A2"].SetStyle(styleA2);
```

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε όλες τις αλλαγές.

```csharp
workbook.Save("StyledExcelFile.xls", SaveFormat.Excel97To2003);
```
