### Οδηγός Εφαρμογής
Θα σας καθοδηγήσουμε βήμα προς βήμα στη ρύθμιση του ύψους των γραμμών.

#### Επισκόπηση της ρύθμισης ύψους σειράς
Η προσαρμογή του ύψους της γραμμής βελτιώνει την ορατότητα και την παρουσίαση των δεδομένων, ειδικά όταν το περιεχόμενο διαφέρει μεταξύ των κελιών.

##### Βήμα 1: Ανοίξτε το βιβλίο εργασίας σας
Φορτώστε το αρχείο Excel σας σε ένα `Workbook` αντικείμενο χρησιμοποιώντας μια ροή αρχείων.
```csharp
using System.IO;
using Aspose.Cells;

namespace AsposeCellsExamples
{
    public class SettingHeightOfRowExample
    {
        public static void Run()
        {
            // Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας
            string dataDir = "path_to_your_directory";
            
            // Άνοιγμα ροής αρχείων για το έγγραφο Excel σας
            using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
            {
                // Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας με την ανοιχτή ροή αρχείων
                Workbook workbook = new Workbook(fstream);

                // Πρόσβαση και τροποποίηση του φύλλου εργασίας...
            }
        }
    }
}
```

##### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας
Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας όπου θέλετε να προσαρμόσετε το ύψος της γραμμής.
```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Worksheet worksheet = workbook.Worksheets[0];
```

##### Βήμα 3: Ορισμός ύψους γραμμής
Χρησιμοποιήστε το `SetRowHeight` μέθοδος για να αλλάξετε το ύψος μιας συγκεκριμένης γραμμής. Εδώ, ορίζουμε το ύψος της δεύτερης γραμμής σε 13 σημεία.
```csharp
// Ορισμός του ύψους της δεύτερης γραμμής (δείκτης 1) σε 13 σημεία
worksheet.Cells.SetRowHeight(1, 13);
```

##### Βήμα 4: Αποθήκευση του βιβλίου εργασίας σας
Αφού κάνετε αλλαγές, αποθηκεύστε το βιβλίο εργασίας σας ξανά σε ένα αρχείο ή μεταδώστε το σε ροή, όπως απαιτείται.
```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
workbook.Save(dataDir + "output.out.xls");
```
