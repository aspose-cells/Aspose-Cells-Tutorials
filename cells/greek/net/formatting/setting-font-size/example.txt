Ξεκινά ένα βιβλίο εργασίας, προσθέτει ένα φύλλο εργασίας, γεμίζει ένα κελί με κείμενο, τροποποιεί το μέγεθος γραμματοσειράς του κελίου και αποθηκεύει το βιβλίο εργασίας σε ένα συγκεκριμένο κατάλογο.
//ExStepSummary:0: Αυτό το βήμα δείχνει πώς να καθοριστεί η διαδρομή προς τον κατάλογο εγγράφων.
//ExStepSummary:1: Αυτό το βήμα δείχνει πώς να δημιουργήσετε έναν κατάλογο αν δεν υπάρχει ήδη.
//ExStepSummary:2: Αυτό το βήμα δείχνει την εκτοπισμό ενός αντικειμένου Workbook.
//ExStepSummary:3: Αυτό το βήμα δείχνει πώς να προσθέσετε ένα νέο φύλλο εργασίας στο βιβλίο εργασίας και να το αποκτήσετε με αναφορά.
//ExStepSummary:4: Αυτό το βήμα περιγράφει λεπτομερώς τον τρόπο πρόσβασης σε ένα συγκεκριμένο κύτταρο και πληθυσμού του με μια τιμή.
//ExStepSummary:5: Αυτό το βήμα εξηγεί πώς να αποκτήσετε το στυλ του κυττάρου και να τροποποιήσετε το μέγεθος της γραμματοσειράς και δείχνει πώς να εφαρμόσετε το τροποποιημένο στυλ στο κυττάριο.
//ExStepSummary:6: Αυτό το βήμα δείχνει πώς να αποθηκεύσετε το βιβλίο εργασίας σε μια καθορισμένη διαδρομή αρχείου.
//ExStepImage:4:images/1.png

//ExStep:0-
// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων
string dataDir = "Your Document Directory";

//ExStep:1-
// Ελέγξτε αν υπάρχει το κατάλογο
bool IsExists = System.IO.Directory.Exists(dataDir);
// Αν δεν υπάρχει
if (!IsExists)
	// Δημιουργήστε τον κατάλογο
    System.IO.Directory.CreateDirectory(dataDir);

//ExStep:2-
// Εναρμηνεύστε ένα αντικείμενο Workbook
Workbook workbook = new Workbook();

//ExStep:3-
// Προσθήκη νέου φύλλου εργασίας στο βιβλίο εργασίας
int i = workbook.Worksheets.Add();
// Βρείτε την αναφορά του πρόσφατα προστίθεται δελτίου εργασίας
Worksheet worksheet = workbook.Worksheets[i];

//ExStep:4-
// Πρόσβαση στην κυτταρία "A1" από το φύλλο εργασίας
Aspose.Cells.Cell cell = worksheet.Cells["A1"];
// Γενικεύστε το κελί "A1" με τιμή
cell.PutValue("Hello Aspose!");

//ExStep:5-
// Να αποκτήσεις το στυλ του κελίου
Style style = cell.GetStyle();
// Εγκαταστήστε το μέγεθος γραμματοσειράς σε 14
style.Font.Size = 14;
// Εφαρμόστε το τροποποιημένο στυλ στο κελί
cell.SetStyle(style);

//ExStep:6-
// Κράτησε το βιβλίο εργασίας στο καθορισμένο μονοπάτι αρχείου
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
