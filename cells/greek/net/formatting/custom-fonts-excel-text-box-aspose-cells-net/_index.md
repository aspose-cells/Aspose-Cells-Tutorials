## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει το περιβάλλον μας, ας εμβαθύνουμε στην εφαρμογή προσαρμοσμένων ρυθμίσεων γραμματοσειράς για πλαίσια κειμένου.

### Προσθήκη πλαισίου κειμένου σε φύλλο εργασίας Excel

**Επισκόπηση**Θα προσθέσουμε ένα πλαίσιο κειμένου και θα διαμορφώσουμε τις γραμματοσειρές του χρησιμοποιώντας το Aspose.Cells. Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε διαφορετικές γραμματοσειρές για σύνολα χαρακτήρων Λατινικής και Άπω Ανατολής στο ίδιο πλαίσιο κειμένου.

#### Βήμα 1: Δημιουργήστε ένα κενό βιβλίο εργασίας

Ξεκινήστε δημιουργώντας ένα νέο βιβλίο εργασίας και αποκτώντας πρόσβαση στο πρώτο φύλλο εργασίας του:

```csharp
// Δημιουργήστε ένα νέο βιβλίο εργασίας.
Workbook wb = new Workbook();

// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας.
Worksheet ws = wb.Worksheets[0];
```

#### Βήμα 2: Προσθήκη πλαισίου κειμένου στο φύλλο εργασίας

Στη συνέχεια, προσθέστε ένα πλαίσιο κειμένου σε καθορισμένες συντεταγμένες μέσα στο φύλλο εργασίας.

```csharp
// Προσθέστε ένα πλαίσιο κειμένου μέσα στο φύλλο εργασίας.
int idx = ws.TextBoxes.Add(5, 5, 50, 200);
Aspose.Cells.Drawing.TextBox tb = ws.TextBoxes[idx];
```

#### Βήμα 3: Ορισμός ονομάτων κειμένου και γραμματοσειράς

Ορίστε το κείμενο του πλαισίου κειμένου και καθορίστε προσαρμοσμένες γραμματοσειρές τόσο για χαρακτήρες Άπω Ανατολής όσο και για λατινικούς χαρακτήρες.

```csharp
// Ορίστε το κείμενο του πλαισίου κειμένου.
tb.Text = "こんにちは世界";

// Καθορίστε τα ονόματα των γραμματοσειρών.
tb.TextOptions.LatinName = "Comic Sans MS";
tb.TextOptions.FarEastName = "KaiTi";
```

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας σας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε ένα αρχείο εξόδου.

```csharp
// Αποθηκεύστε το αρχείο Excel που προκύπτει.
wb.Save("outputSpecifyFarEastAndLatinNameOfFontInTextOptionsOfShape.xlsx", SaveFormat.Xlsx);
```
