## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία εξαγωγής δεδομένων θέματος από ένα βιβλίο εργασίας του Excel σε διαχειρίσιμα βήματα.

### Εξαγωγή ονόματος θέματος βιβλίου εργασίας

**Επισκόπηση:**
Το πρώτο βήμα είναι να εξαγάγετε το συνολικό όνομα θέματος που εφαρμόζεται σε ολόκληρο το βιβλίο εργασίας. Αυτό σας δίνει μια πλήρη κατανόηση του στυλ που χρησιμοποιείται στο έγγραφό σας.

#### Βήματα Υλοποίησης:
1. **Φόρτωση του βιβλίου εργασίας σας**
   Ξεκινήστε δημιουργώντας ένα `Workbook` αντικείμενο με τη διαδρομή προς το αρχείο Excel σας.
    ```csharp
    string sourceDir = RunExamples.Get_SourceDirectory();
    Workbook workbook = new Workbook(sourceDir + "sampleExtractThemeData.xlsx");
    ```
2. **Ανάκτηση πληροφοριών θέματος**
   Χρησιμοποιήστε το `Theme` ιδιοκτησία του `Workbook` κλάση για να λάβετε το όνομα του θέματος.
    ```csharp
    Console.WriteLine(workbook.Theme);
    ```

### Πρόσβαση σε στυλ και θέματα κελιών

**Επισκόπηση:**
Μόλις ανακτήσετε το θέμα του βιβλίου εργασίας, αποκτήστε πρόσβαση σε συγκεκριμένα στυλ κελιών και στα σχετικά χρώματα θέματος.

#### Βήματα Υλοποίησης:
1. **Φύλλο εργασίας και κελιά της Access**
   Μεταβείτε στο φύλλο εργασίας που επιθυμείτε και επιλέξτε ένα συγκεκριμένο κελί για λεπτομερή ανάλυση.
    ```csharp
    Worksheet worksheet = workbook.Worksheets[0];
    Cell cell = worksheet.Cells["A1"];
    ```
2. **Ανάκτηση πληροφοριών στυλ**
   Αποκτήστε το στυλ που εφαρμόζεται στο κελί και ελέγξτε για χρώματα θέματος.
    ```csharp
    Style style = cell.GetStyle();

    if (style.ForegroundThemeColor != null)
    {
        Console.WriteLine(style.ForegroundThemeColor.ColorType);
    }
    else
    {
        Console.WriteLine("Theme has no Foreground Color defined.");
    }
    ```
3. **Έλεγχος χρωμάτων θέματος περιγράμματος**
   Ομοίως, αναλύστε τα χρώματα θέματος που εφαρμόζονται στα περιγράμματα των κελιών.
    ```csharp
    Border bot = style.Borders[BorderType.BottomBorder];
    if (bot.ThemeColor != null)
    {
        Console.WriteLine(bot.ThemeColor.ColorType);
    }
    else
    {
        Console.WriteLine("Theme has no Border Color defined.");
    }
    ```
