## Οδηγός Εφαρμογής

### Ρύθμιση του βιβλίου εργασίας

#### Επισκόπηση

Αρχικά, θα δημιουργήσουμε ένα νέο βιβλίο εργασίας του Excel και θα το συμπληρώσουμε με δείγματα δεδομένων.

**Βήμα 1: Δημιουργία νέου βιβλίου εργασίας**
```csharp
using Aspose.Cells;

namespace ExcelFormattingGuide
{
    class Program
    {
        static void Main(string[] args)
        {
            // Αρχικοποίηση νέου βιβλίου εργασίας
            var workbook = new Workbook();
            
            // Πρόσβαση στο πρώτο φύλλο εργασίας
            var sheet = workbook.Worksheets[0];
            
            // Προσθήκη δειγμάτων δεδομένων σε κελιά
            sheet.Cells["A1"].PutValue("Month");
            sheet.Cells["B1"].PutValue("Sales");

            for (int i = 2; i <= 13; i++)
            {
                sheet.Cells[$"A{i}"].PutValue($"Month {i-1}");
                sheet.Cells[$"B{i}"].PutValue(i * 1000);
            }
        }
    }
}
```

**Εξήγηση:** Αυτός ο κώδικας αρχικοποιεί ένα νέο βιβλίο εργασίας και προσθέτει δείγματα μηνιαίων δεδομένων πωλήσεων. `PutValue` Η μέθοδος εισάγει τιμές σε συγκεκριμένα κελιά.

### Μορφοποίηση κελιών

#### Επισκόπηση

Στη συνέχεια, θα εφαρμόσουμε διάφορα στυλ για να βελτιώσουμε την αναγνωσιμότητα των δεδομένων μας.

**Βήμα 2: Εφαρμογή στυλ**
```csharp
// Δημιουργήστε ένα αντικείμενο στυλ για κεφαλίδες
Style headerStyle = workbook.CreateStyle();
headerStyle.ForegroundColor = System.Drawing.Color.FromArgb(124, 199, 72);
headerStyle.Pattern = BackgroundType.Solid;
headerStyle.Font.IsBold = true;
headerStyle.HorizontalAlignment = TextAlignmentType.Center;

// Εφαρμογή του στυλ στην πρώτη γραμμή (κεφαλίδες)
Range headerRange = sheet.Cells.CreateRange("A1", "B1");
headerRange.ApplyStyle(headerStyle, new StyleFlag() { All = true });
```

**Εξήγηση:** Αυτό το απόσπασμα δημιουργεί ένα έντονο, κεντρικό στυλ με πράσινο φόντο για τις κεφαλίδες. `ApplyStyle` Η μέθοδος εφαρμόζει αυτό το στυλ στο καθορισμένο εύρος.

### Μορφοποίηση υπό όρους

#### Επισκόπηση

Για να επισημάνουμε εξαιρετικά στοιχεία πωλήσεων, θα χρησιμοποιήσουμε μορφοποίηση υπό όρους.

**Βήμα 3: Εφαρμογή μορφοποίησης υπό όρους**
```csharp
// Ορίστε έναν κανόνα για την επισήμανση κελιών που είναι μεγαλύτερα από 10.000 $
int index = sheet.ConditionalFormattings.Add();
var cfRule = sheet.ConditionalFormattings[index].AddCondition(FormatConditionType.CellValue, OperatorType.GreaterThan, "10000");
cfRule.Style.ForegroundColor = System.Drawing.Color.FromArgb(255, 192, 0);
cfRule.Style.Pattern = BackgroundType.Solid;
cfRule.Formula1 = "10000";

// Εφαρμογή του κανόνα σε δεδομένα πωλήσεων
var range = sheet.Cells.CreateRange("B2", "B13");
sheet.ConditionalFormattings[index].AddArea(range);
```

**Εξήγηση:** Αυτός ο κώδικας ορίζει έναν κανόνα μορφοποίησης υπό όρους που επισημαίνει τα κελιά με πωλήσεις άνω των 10.000 $ με πορτοκαλί χρώμα.
