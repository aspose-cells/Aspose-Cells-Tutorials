## Οδηγός Εφαρμογής

### Επισκόπηση: Ορισμός ορίων υπό όρους
Αυτή η ενότητα καλύπτει την εφαρμογή μορφοποίησης υπό όρους με διακεκομμένα περιγράμματα χρησιμοποιώντας το Aspose.Cells. Θα ορίσετε εύρη και συνθήκες και, στη συνέχεια, θα εφαρμόσετε προσαρμοσμένα στυλ περιγράμματος.

#### Βήμα 1: Ορίστε το εύρος μορφοποίησης υπό όρους
Καθορίστε ποια κελιά θα πρέπει να μορφοποιηθούν υπό όρους:
```csharp
// Ορίστε μια περιοχή κελιού (CellArea) για το εύρος.
CellArea ca = new CellArea();
ca.StartRow = 0;
c.EndRow = 5;
ca.StartColumn = 0;
ca.EndColumn = 3;

// Προσθέστε αυτήν την περιοχή στη συλλογή μορφοποίησης υπό όρους.
FormatConditionCollection fcs = sheet.ConditionalFormattings.Add();
fcs.AddArea(ca);
```

#### Βήμα 2: Ορισμός του κανόνα μορφοποίησης υπό όρους
Ορίστε μια συνθήκη που ενεργοποιείται όταν οι τιμές των κελιών πέφτουν μεταξύ 50 και 100:
```csharp
int conditionIndex = fcs.AddCondition(FormatConditionType.CellValue, OperatorType.Between, "50", "100");
```

#### Βήμα 3: Προσαρμογή στυλ περιγράμματος
Εφαρμόστε διακεκομμένα περιγράμματα σε κελιά που πληρούν την προϋπόθεση για γρήγορη αναγνώριση σχετικών δεδομένων.
```csharp
// Αποκτήστε πρόσβαση στη συγκεκριμένη συνθήκη μορφοποίησης.
FormatCondition fc = fcs[conditionIndex];

// Ορίστε στυλ και χρώματα περιγράμματος.
fc.Style.Borders[BorderType.LeftBorder].LineStyle = CellBorderType.Dashed;
fc.Style.Borders[BorderType.RightBorder].LineStyle = CellBorderType.Dashed;
fc.Style.Borders[BorderType.TopBorder].LineStyle = CellBorderType.Dashed;
fc.Style.Borders[BorderType.BottomBorder].LineStyle = CellBorderType.Dashed;

// Ορίστε χρώματα περιγράμματος.
fc.Style.Borders[BorderType.LeftBorder].Color = Color.FromArgb(0, 255, 255);
fc.Style.Borders[BorderType.RightBorder].Color = Color.FromArgb(0, 255, 255);
fc.Style.Borders[BorderType.TopBorder].Color = Color.FromArgb(0, 255, 255);
fc.Style.Borders[BorderType.BottomBorder].Color = Color.FromArgb(255, 255, 0);
```

Αποθηκεύστε τις αλλαγές σας σε ένα αρχείο εξόδου:
