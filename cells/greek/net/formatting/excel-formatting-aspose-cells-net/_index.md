## Οδηγός Εφαρμογής

### Δημιουργία και μορφοποίηση κελιού Excel

#### Επισκόπηση

Μάθετε πώς να δημιουργείτε ένα κελί στο βιβλίο εργασίας του Excel, να εισάγετε τιμές, να εφαρμόζετε μορφές αριθμών για καλύτερη αναγνωσιμότητα και να ανακτάτε δεδομένα κελιών τόσο σε μορφοποιημένο όσο και σε μη μορφοποιημένο επίπεδο.

**Βήμα 1: Δημιουργία βιβλίου εργασίας και φύλλου εργασίας της Access**

Δημιουργήστε ένα νέο `Workbook` αντικείμενο και πρόσβαση στο πρώτο φύλλο εργασίας:

```csharp
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
```

**Βήμα 2: Εισαγωγή τιμής στο κελί**

Αποκτήστε πρόσβαση στο κελί A1 και εισαγάγετε μια αριθμητική τιμή:

```csharp
Cell cell = worksheet.Cells["A1"];
cell.PutValue(0.012345);
```

**Βήμα 3: Εφαρμογή μορφοποίησης αριθμών**

Μορφοποιήστε το κελί ώστε να εμφανίζει μόνο δύο δεκαδικά ψηφία χρησιμοποιώντας `Style`:

```csharp
Style style = cell.GetStyle();
style.Number = 2; // Μορφή '0,00'
cell.SetStyle(style);
```

**Βήμα 4: Ανάκτηση μορφοποιημένων και μη μορφοποιημένων τιμών**

Λάβετε και τις δύο εκδόσεις της τιμής του κελιού για σύγκριση:

```csharp
string formattedValue = cell.GetStringValue(CellValueFormatStrategy.CellStyle);
string unformattedValue = cell.GetStringValue(CellValueFormatStrategy.None);
```

### Διαχείριση Βιβλίων Εργασίας και Φύλλων Εργασίας

#### Επισκόπηση

Εξερευνήστε πώς να δημιουργείτε, να έχετε πρόσβαση και να χειρίζεστε φύλλα εργασίας μέσα σε ένα βιβλίο εργασίας του Excel.

**Βήμα 1: Δημιουργία νέου βιβλίου εργασίας**

Αρχικοποίηση του `Workbook` αντικείμενο όπως φαίνεται νωρίτερα.

**Βήμα 2: Πρόσβαση στο φύλλο εργασίας ανά ευρετήριο**

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας χρησιμοποιώντας το ευρετήριό του:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
Console.WriteLine("Accessed Worksheet: " + worksheet.Name);
```

**Βήμα 3: Χειρισμός κελιών σε ένα φύλλο εργασίας**

Δημιουργήστε και ορίστε τιμές για νέα κελιά, όπως τοποθετώντας το "Hello World" στο κελί A2:

```csharp
cell = worksheet.Cells["A2"];
cell.PutValue("Hello World");
```
