## Οδηγός Εφαρμογής

### Ορισμός χρώματος γραμματοσειράς σε κελιά του Excel

Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στην αλλαγή του χρώματος γραμματοσειράς για κείμενο μέσα σε ένα κελί του Excel.

#### Βήμα 1: Δημιουργία νέου βιβλίου εργασίας

Ξεκινήστε δημιουργώντας ένα νέο `Workbook` αντικείμενο. Αυτό αντιπροσωπεύει ολόκληρο το αρχείο Excel σας.

```csharp
// Δημιουργία στιγμιαίου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```

#### Βήμα 2: Προσθήκη φύλλου εργασίας

Προσθέστε ένα φύλλο εργασίας στο βιβλίο εργασίας σας όπου θα εφαρμόσετε τις αλλαγές στο χρώμα της γραμματοσειράς.

```csharp
// Προσθήκη νέου φύλλου εργασίας στο βιβλίο εργασίας
int sheetIndex = workbook.Worksheets.Add();
Worksheet worksheet = workbook.Worksheets[sheetIndex];
```

#### Βήμα 3: Πρόσβαση και τροποποίηση στυλ κελιού

Αποκτήστε πρόσβαση στο επιθυμητό κελί, τροποποιήστε το στυλ του και ορίστε το χρώμα της γραμματοσειράς. Εδώ θα αλλάξουμε το χρώμα της γραμματοσειράς του κελιού "A1" σε μπλε.

```csharp
// Πρόσβαση στο κελί "A1" από το φύλλο εργασίας
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Hello Aspose!");

// Λήψη του αντικειμένου στυλ για το κελί
Style style = cell.GetStyle();

// Ρύθμιση του χρώματος γραμματοσειράς σε μπλε
style.Font.Color = Color.Blue;

// Εφαρμογή του στυλ πίσω στο κελί
cell.SetStyle(style);
```

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τις αλλαγές που πραγματοποιήσατε.

```csharp
// Αποθήκευση του αρχείου Excel
string dataDir = "path_to_save_directory";
workbook.Save(dataDir + "StyledWorkbook.xls", SaveFormat.Excel97To2003);
```
