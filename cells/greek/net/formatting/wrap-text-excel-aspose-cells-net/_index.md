## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει το απαραίτητο περιβάλλον, ας εφαρμόσουμε τη λειτουργία αναδίπλωσης κειμένου στο Excel.

### Δημιουργήστε ένα νέο αρχείο Excel και ορίστε την αναδίπλωση κειμένου

#### Επισκόπηση:
Σε αυτήν την ενότητα, θα δημιουργήσουμε ένα αρχείο Excel και θα διαμορφώσουμε την αναδίπλωση κειμένου για ένα συγκεκριμένο κελί.

**Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας**
Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Workbook` κλάση. Αυτό αντιπροσωπεύει το αρχείο Excel σας.
```csharp
// Αρχικοποίηση βιβλίου εργασίας.
Workbook workbook = new Workbook();
```

**Βήμα 2: Λήψη αναφοράς φύλλου εργασίας**
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας, το οποίο δημιουργείται από προεπιλογή όταν δημιουργείτε την αρχική του μορφή. `Workbook`.
```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας.
Worksheet worksheet = workbook.Worksheets[0];
```

**Βήμα 3: Πρόσβαση και τροποποίηση περιεχομένου κελιού**
Αποκτήστε πρόσβαση σε ένα συγκεκριμένο κελί (π.χ., "A1") και ορίστε την τιμή του.
```csharp
// Λάβετε μια αναφορά κελιού και εισάγετε μια τιμή σε αυτήν.
Cell cell = worksheet.Cells["A1"];
cell.PutValue("Visit Aspose!");
```

**Βήμα 4: Ενεργοποίηση αναδίπλωσης κειμένου**
Αναδίπλωση κειμένου ορίζοντας το `IsTextWrapped` η ιδιότητα να είναι true εντός της διαμόρφωσης στυλ του κελιού.
```csharp
// Ανάκτηση και διαμόρφωση του στυλ για την αναδίπλωση κειμένου.
Style style = cell.GetStyle();
style.IsTextWrapped = true;
cell.SetStyle(style);
```

**Βήμα 5: Αποθήκευση του βιβλίου εργασίας**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας. Μπορείτε να καθορίσετε διαφορετικές μορφές όπως Excel97To2003 ή Xlsx.
```csharp
// Ορίστε τη διαδρομή αρχείου και αποθηκεύστε το βιβλίο εργασίας σε μορφή Excel.
string dataDir = "your_directory_path";
workbook.Save(dataDir + "WrappedTextExample.xls", SaveFormat.Excel97To2003);
```
