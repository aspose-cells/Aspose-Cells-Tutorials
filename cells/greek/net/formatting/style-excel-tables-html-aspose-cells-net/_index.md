## Οδηγός Εφαρμογής

### Στυλιζάρισμα πινάκων Excel
Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας που θα περιέχει τα δεδομένα του Excel:
```csharp
// Δημιουργία στιγμιότυπου βιβλίου εργασίας
Workbook wb = new Workbook();
```
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και διαμορφώστε τα κελιά του:
```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet ws = wb.Worksheets[0];

// Προσθήκη κειμένου στο κελί B5
Cell cell = ws.Cells["B5"];
cell.PutValue("This is some text.");

// Στυλ κελιού - αλλαγή χρώματος γραμματοσειράς σε κόκκινο
Style st = cell.GetStyle();
st.Font.Color = Color.Red;
cell.SetStyle(st);
```
### Αποθήκευση ως HTML με προσαρμοσμένο CSS
Χρήση `HtmlSaveOptions` για να καθορίσετε προσαρμοσμένα στυλ:
```csharp
// Ρύθμιση παραμέτρων HtmlSaveOptions και καθορισμός του αναγνωριστικού CSS του πίνακα
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.TableCssId = "MyTest_TableCssId";

// Αποθήκευση του βιβλίου εργασίας ως αρχείο HTML με στυλιζαρισμένους πίνακες
wb.Save("outputTableCssId.html", opts);
```
