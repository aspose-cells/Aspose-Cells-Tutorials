## Οδηγός Εφαρμογής

### Προσαρμογή διαχωριστικών δεκαδικών και ομάδων

**Επισκόπηση:**
Η προσαρμογή των διαχωριστών δεκαδικών ψηφίων και ομάδων βελτιώνει την αναγνωσιμότητα των δεδομένων και πληροί συγκεκριμένα πρότυπα μορφοποίησης που απαιτούνται από διάφορες περιοχές ή επιχειρήσεις.

#### Βήμα 1: Διαμόρφωση ρυθμίσεων
Ξεκινήστε καθορίζοντας τις επιθυμητές μορφές αριθμών για ολόκληρο το βιβλίο εργασίας:
```csharp
// Ορίστε προσαρμοσμένους δεκαδικούς και διαχωριστές ομάδων
workbook.Settings.NumberDecimalSeparator = '.';
workbook.Settings.NumberGroupSeparator = ' ';
```
**Εξήγηση:** Ο `NumberDecimalSeparator` έχει οριστεί σε μια τελεία (.) όπως χρησιμοποιείται συνήθως σε πολλές περιοχές. Το `NumberGroupSeparator` έχει διαμορφωθεί ως κενό (' '), το οποίο μπορεί να προσαρμοστεί με βάση τις τοπικές προτιμήσεις.

#### Βήμα 2: Εφαρμογή προσαρμοσμένων στυλ
Μόλις οριστούν τα διαχωριστικά, εφαρμόστε ένα προσαρμοσμένο στυλ στα κελιά σας:
```csharp
Worksheet worksheet = workbook.Worksheets[0];

// Ορισμός τιμής κελιού και εφαρμογή στυλ
Cell cell = worksheet.Cells["A1"];
cell.PutValue(123456.789);

Style style = cell.GetStyle();
style.Custom = "#,##0.000;[Red]#,##0.000"; // Προσαρμοσμένη συμβολοσειρά μορφής
cell.SetStyle(style);
```
**Εξήγηση:** Η προσαρμοσμένη μορφή `#,##0.000` εξασφαλίζει τρία δεκαδικά ψηφία και ομαδοποιεί τα ψηφία χρησιμοποιώντας τα καθορισμένα διαχωριστικά.

#### Βήμα 3: Αυτόματη Προσαρμογή Στηλών
Για να διασφαλίσετε ότι τα δεδομένα σας παρουσιάζονται σωστά, προσαρμόστε αυτόματα τις στήλες:
```csharp
worksheet.AutoFitColumns();
```
Αυτή η μέθοδος προσαρμόζει αυτόματα το πλάτος των στηλών ώστε να ταιριάζει στο περιεχόμενό τους.

#### Βήμα 4: Αποθήκευση ως PDF
Τέλος, αποθηκεύστε το βιβλίο εργασίας ως PDF με τις προσαρμοσμένες ρυθμίσεις σας:
```csharp
workbook.Save("YOUR_OUTPUT_DIRECTORY/CustomSeparator_out.pdf");
```
