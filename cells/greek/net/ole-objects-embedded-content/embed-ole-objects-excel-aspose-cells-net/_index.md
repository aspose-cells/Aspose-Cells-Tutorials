## Οδηγός Εφαρμογής (H2)

Τώρα που έχετε ρυθμίσει το περιβάλλον σας, ας εφαρμόσουμε την εισαγωγή αντικειμένων OLE.

### Επισκόπηση: Εισαγωγή αντικειμένου OLE στο Excel

Αυτή η λειτουργία επιτρέπει την ενσωμάτωση εικόνων ή άλλων αρχείων απευθείας στα υπολογιστικά φύλλα του Excel χρησιμοποιώντας C#. Δείτε πώς μπορείτε να το πετύχετε βήμα προς βήμα:

#### Βήμα 1: Προετοιμασία των αρχείων σας (H3)

Αρχικά, βεβαιωθείτε ότι η εικόνα και το αρχείο που θέλετε να ενσωματώσετε είναι προσβάσιμα. Για αυτό το παράδειγμα, χρησιμοποιούμε μια εικόνα λογότυπου και ένα αρχείο Excel.

```csharp
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Δημιουργία καταλόγου εάν δεν υπάρχει
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

#### Βήμα 2: Φόρτωση δεδομένων εικόνας και αντικειμένου (H3)

Διαβάστε τα δεδομένα του αρχείου εικόνας και αντικειμένου σε πίνακες byte.

```csharp
// Διαβάστε την εικόνα σε μια ροή και στη συνέχεια σε έναν πίνακα byte
string ImageUrl = dataDir + "logo.jpg";
FileStream fs = File.OpenRead(ImageUrl);
byte[] imageData = new Byte[fs.Length];
fs.Read(imageData, 0, imageData.Length);
fs.Close();

// Διαβάστε το αρχείο αντικειμένου (π.χ., ένα άλλο αρχείο Excel) με παρόμοιο τρόπο
string path = dataDir + "book1.xls";
fs = File.OpenRead(path);
byte[] objectData = new Byte[fs.Length];
fs.Read(objectData, 0, objectData.Length);
fs.Close();
```

#### Βήμα 3: Προσθήκη του αντικειμένου OLE στο φύλλο εργασίας (H3)

Ενσωματώστε την εικόνα και το αρχείο σας στο φύλλο εργασίας.

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας
Worksheet sheet = workbook.Worksheets[0];

// Προσθέστε ένα αντικείμενο Ole στο φύλλο εργασίας με την εικόνα που εμφανίζεται στο MS Excel
sheet.OleObjects.Add(14, 3, 200, 220, imageData);

// Ορισμός ενσωματωμένων δεδομένων αντικειμένου ole
sheet.OleObjects[0].ObjectData = objectData;
```

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας (H3)

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να αντικατοπτρίσετε αυτές τις αλλαγές.

```csharp
workbook.Save(dataDir + "output.out.xls");
```
