## Οδηγός Εφαρμογής

### Εισαγωγή αρχείου WAV ως αντικειμένου OLE

Θα εξετάσουμε κάθε βήμα για την εισαγωγή ενός αρχείου WAV στο Excel χρησιμοποιώντας το Aspose.Cells.

#### 1. Προετοιμάστε τα αρχεία σας

Βεβαιωθείτε ότι έχετε έτοιμα τα απαραίτητα αρχεία εικόνας και ήχου:
- `sampleInsertOleObject_WAVFile.jpg` (Αναπαράσταση εικόνας του αντικειμένου OLE σας)
- `sampleInsertOleObject_WAVFile.wav` (Το πραγματικό αρχείο ήχου)

#### 2. Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας

Δημιουργήστε ένα νέο βιβλίο εργασίας του Excel και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας του.
```csharp
// Δημιουργήστε ένα νέο Βιβλίο Εργασίας.
Workbook workbook = new Workbook();
Worksheet sheet = workbook.Worksheets[0];
```

#### 3. Προσθέστε το αντικείμενο OLE

Χρησιμοποιήστε το Aspose.Cells για να προσθέσετε ένα αντικείμενο OLE που ενσωματώνει το αρχείο WAV σας:
```csharp
// Ορισμός πινάκων byte για δεδομένα εικόνας και ήχου
byte[] imageData = File.ReadAllBytes("sampleInsertOleObject_WAVFile.jpg");
byte[] objectData = File.ReadAllBytes("sampleInsertOleObject_WAVFile.wav");

// Προσθήκη του αντικειμένου Ole στο φύλλο εργασίας στο καθορισμένο κελί
int idx = sheet.OleObjects.Add(3, 3, 200, 220, imageData);
OleObject ole = sheet.OleObjects[idx];
```

#### 4. Ρύθμιση παραμέτρων ιδιοτήτων OLE

Ορίστε διάφορες ιδιότητες για το ενσωματωμένο αντικείμενο για να διασφαλίσετε ότι λειτουργεί σωστά:
```csharp
// Ορισμός μορφής αρχείου και άλλων βασικών ιδιοτήτων
ole.FileFormatType = FileFormatType.Ole10Native;
ole.ObjectData = objectData;
ole.ObjectSourceFullName = "sample.wav";
ole.ProgID = "Packager Shell Object";

Guid gu = new Guid("0003000c-0000-0000-c000-000000000046");
ole.ClassIdentifier = gu.ToByteArray();
```

#### 5. Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρηθούν οι αλλαγές:
```csharp
// Αποθήκευση του αρχείου Excel
workbook.Save("outputInsertOleObject_WAVFile.xlsx");
Console.WriteLine("InsertOleObject_WAVFile executed successfully.");
```
