## Οδηγός Εφαρμογής

### Αφαίρεση στοιχείων ελέγχου ActiveX από βιβλία εργασίας του Excel
Αυτή η ενότητα σας καθοδηγεί στην κατάργηση των στοιχείων ελέγχου ActiveX χρησιμοποιώντας C# και Aspose.Cells.

#### Βήμα 1: Φόρτωση του αρχείου Excel
Φορτώστε το βιβλίο εργασίας σας που περιέχει το στοιχείο ελέγχου ActiveX. Αντικαταστήστε `sourceDir` με τη διαδρομή προς το αρχείο σας:
```csharp
// Κατάλογος πηγής
string sourceDir = "path_to_your_source_directory";

// Δημιουργία βιβλίου εργασίας από ένα υπάρχον αρχείο
Workbook wb = new Workbook(sourceDir + "sampleUpdateActiveXComboBoxControl.xlsx");
```

#### Βήμα 2: Πρόσβαση και κατάργηση του στοιχείου ελέγχου ActiveX
Αποκτήστε πρόσβαση στο σχήμα που περιέχει το στοιχείο ελέγχου ActiveX και, στη συνέχεια, καταργήστε το.
```csharp
// Πρόσβαση στο πρώτο σχήμα από το πρώτο φύλλο εργασίας
Shape shape = wb.Worksheets[0].Shapes[0];

if (shape.ActiveXControl != null)
{
    // Αφαίρεση στοιχείου ελέγχου Shape ActiveX
    shape.RemoveActiveXControl();
}
```
**Επεξήγηση παραμέτρων:**
- `Workbook`: Αντιπροσωπεύει το βιβλίο εργασίας του Excel.
- `Worksheet.Shapes`Προσπελαύνει σχήματα, συμπεριλαμβανομένων των στοιχείων ελέγχου ActiveX, σε ένα φύλλο εργασίας.

#### Βήμα 3: Αποθήκευση του τροποποιημένου βιβλίου εργασίας
Αποθηκεύστε το βιβλίο εργασίας σας για να διατηρηθούν οι αλλαγές:
```csharp
// Κατάλογος εξόδου
string outputDir = "path_to_your_output_directory";

// Αποθήκευση του τροποποιημένου βιβλίου εργασίας
wb.Save(outputDir + "RemoveActiveXControl_our.xlsx");
```
**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή του αρχείου είναι σωστή και προσβάσιμη.
- Επαληθεύστε ότι δεν υπάρχουν προβλήματα με δικαιώματα εγγραφής στον κατάλογο αποθήκευσης.
