## Εισαγωγή πακέτων
Το πρώτο βήμα στο ταξίδι μας περιλαμβάνει την εισαγωγή βασικών χώρων ονομάτων. Αυτό είναι κρίσιμο, καθώς μας επιτρέπει την πρόσβαση στις κλάσεις και τις μεθόδους που θα χρησιμοποιήσουμε για τον χειρισμό αρχείων Excel. Δείτε πώς το κάνετε:
```csharp
using System;
using System.IO;
using Aspose.Cells;
```
Βεβαιωθείτε ότι έχετε συμπεριλάβει αυτήν τη γραμμή στην αρχή του αρχείου .cs. Τώρα που έχουμε έτοιμες τις εισαγωγές, ας προχωρήσουμε στη δημιουργία του βιβλίου εργασίας μας και στην πρόσβαση στο φύλλο εργασίας.
## Βήμα 1: Δημιουργήστε το βιβλίο εργασίας σας
Ξεκινάμε δημιουργώντας μια παρουσία του `Workbook` κλάση. Αυτό αποτελεί τη βάση του χειρισμού αρχείων Excel.
```csharp
Workbook wb = new Workbook();
```
Αυτή η γραμμή λέει στο πρόγραμμα να αρχικοποιήσει ένα νέο βιβλίο εργασίας, ρυθμίζοντάς μας να εμβαθύνουμε στα φύλλα εργασίας μας.
## Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας
Στη συνέχεια, θα έχουμε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας που μόλις δημιουργήσαμε. Είναι αρκετά απλό:
```csharp
Worksheet ws = wb.Worksheets[0];
```
Εδώ, έχουμε πρόσβαση στο πρώτο φύλλο (με ευρετήριο 0) στο βιβλίο εργασίας μας. Εδώ θα ορίσουμε τα μεγέθη χαρτιού.
## Ρύθμιση μεγέθους χαρτιού και ανάκτηση διαστάσεων
Τώρα μπαίνουμε στην ουσία της λειτουργίας—ορίζοντας το μέγεθος του χαρτιού και ανακτώντας τις διαστάσεις του! Ας το αναλύσουμε βήμα προς βήμα.
## Βήμα 3: Ορίστε το μέγεθος χαρτιού σε A2
Ας ορίσουμε πρώτα το μέγεθος χαρτιού μας σε A2 και ας εκτυπώσουμε τις διαστάσεις του.
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA2;
Console.WriteLine("PaperA2: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
Μετά από αυτή τη ρύθμιση, χρησιμοποιούμε `Console.WriteLine` για να εμφανίσετε τις διαστάσεις. Όταν εκτελέσετε αυτήν την ενέργεια, θα δείτε το πλάτος και το ύψος σε ίντσες για μέγεθος χαρτιού A2.
## Βήμα 4: Ορίστε το μέγεθος χαρτιού σε A3
Τώρα είναι η ώρα για το A3! Απλώς επαναλαμβάνουμε τη διαδικασία:
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA3;
Console.WriteLine("PaperA3: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
Ιδού! Η δήλωση θα εκτυπώσει το συγκεκριμένο ύψος και πλάτος για χαρτί A3.
## Βήμα 5: Ορίστε το μέγεθος χαρτιού σε A4
Ακολουθώντας το ίδιο μοτίβο, ας δούμε πώς τα πάει το A4:
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperA4;
Console.WriteLine("PaperA4: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
Αυτό μας δίνει τις διαστάσεις για A4—ένα από τα πιο συχνά χρησιμοποιούμενα μεγέθη χαρτιού.
## Βήμα 6: Ορισμός μεγέθους χαρτιού σε Letter
Για να ολοκληρώσουμε την εξερεύνησή μας σχετικά με το μέγεθος χαρτιού, ας το ορίσουμε σε μέγεθος Letter:
```csharp
ws.PageSetup.PaperSize = PaperSizeType.PaperLetter;
Console.WriteLine("PaperLetter: " + ws.PageSetup.PaperWidth + "x" + ws.PageSetup.PaperHeight);
```
Και πάλι, θα δούμε το συγκεκριμένο πλάτος και ύψος για το μέγεθος Letter.
