## Οδηγός Εφαρμογής

Τώρα που καλύψαμε τη ρύθμιση, ας προχωρήσουμε στην εφαρμογή της λύσης.

### Φόρτωση συγκεκριμένων φύλλων

Ο στόχος είναι να φορτώσετε μόνο συγκεκριμένα φύλλα από ένα αρχείο Excel, αγνοώντας άλλα. Δείτε πώς μπορείτε να το πετύχετε:

#### Βήμα 1: Ορισμός επιλογών φόρτωσης

Αρχικά, δημιουργήστε ένα `LoadOptions` αντικείμενο που καθορίζει τη μορφή του βιβλίου εργασίας σας και αντιστοιχίζει ένα προσαρμοσμένο φίλτρο φόρτωσης.

```csharp
LoadOptions loadOptions = new LoadOptions(LoadFormat.Xlsx);
loadOptions.LoadFilter = new CustomLoad();
```

**Εξήγηση**: Το `LoadOptions` Η κλάση παρέχει ρυθμίσεις για τη φόρτωση αρχείων Excel. Ορίζοντας το `LoadFilter`, εσείς ελέγχετε ποια φύλλα θα φορτωθούν με βάση τα κριτήριά σας.

#### Βήμα 2: Δημιουργήστε ένα προσαρμοσμένο φίλτρο φόρτωσης

Ορίστε ένα προσαρμοσμένο φίλτρο κληρονομώντας από `LoadFilter`Αυτό θα καθορίσει τον τρόπο επεξεργασίας κάθε φύλλου.

```csharp
class CustomLoad : LoadFilter
{
    public override void StartSheet(Worksheet sheet)
    {
        if (sheet.Name == "Sheet2")
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.All;
        }
        else
        {
            this.LoadDataFilterOptions = LoadDataFilterOptions.Structure;
        }
    }
}
```

**Εξήγηση**: Το `StartSheet` Η μέθοδος παρακάμπτεται για να καθορίσει ότι μόνο το "Sheet2" θα πρέπει να φορτωθεί με όλα τα δεδομένα, ενώ τα άλλα φύλλα αγνοούνται πέρα από τη δομή τους.

#### Βήμα 3: Φόρτωση του βιβλίου εργασίας

Χρησιμοποιήστε τις καθορισμένες επιλογές φόρτωσης για να δημιουργήσετε μια παρουσία βιβλίου εργασίας και να φορτώσετε το φύλλο που επιθυμείτε.

```csharp
Workbook workbook = new Workbook(sourceDir + "sampleLoadSpecificSheets.xlsx", loadOptions);
```

**Εξήγηση**: Το `Workbook` Ο κατασκευαστής δέχεται επιλογές τόσο διαδρομής αρχείου όσο και φόρτωσης, επιτρέποντάς σας να καθορίσετε ποια φύλλα θα πρέπει να φορτωθούν με βάση τη λογική του προσαρμοσμένου φίλτρου.

#### Βήμα 4: Αποθήκευση του αποτελέσματος

Μετά την επεξεργασία, αποθηκεύστε το βιβλίο εργασίας σας με τυχόν τροποποιήσεις, εάν χρειάζεται:

```csharp
workbook.Save(outputDir + "outputLoadSpecificSheets.xlsx");
```
