Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στον τρόπο με τον οποίο μπορείτε να συνδυάσετε πολλά φύλλα εργασίας σε ένα.

### Επισκόπηση

Αυτή η λειτουργία επιτρέπει την αποτελεσματική συγχώνευση δεδομένων από πολλά φύλλα εργασίας σε ένα μόνο φύλλο, κάτι που είναι χρήσιμο για την ενοποίηση αναφορών ή τη σύνταξη δεδομένων σε διάφορα φύλλα.

#### Βήμα προς βήμα εφαρμογή

##### Αρχικοποίηση των αντικειμένων του βιβλίου εργασίας

Αρχικά, φορτώστε το βιβλίο εργασίας προέλευσης και δημιουργήστε ένα βιβλίο εργασίας προορισμού όπου θα αποθηκευτούν τα συγχωνευμένα δεδομένα:
```csharp
// Διαδρομή καταλόγου πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Διαδρομή καταλόγου εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

Workbook workbook = new Workbook(sourceDir + "sampleCombineMultipleWorksheetsSingleWorksheet.xlsx");
Workbook destWorkbook = new Workbook();
```

##### Συγχώνευση Φύλλων Εργασίας

Επαναλάβετε κάθε φύλλο εργασίας στο βιβλίο εργασίας προέλευσης και αντιγράψτε το περιεχόμενό του σε ένα μόνο φύλλο προορισμού:
```csharp
Worksheet destSheet = destWorkbook.Worksheets[0];
int TotalRowCount = 0;

for (int i = 0; i < workbook.Worksheets.Count; i++)
{
    Worksheet sourceSheet = workbook.Worksheets[i];
    
    Range sourceRange = sourceSheet.Cells.MaxDisplayRange;
    Range destRange = destSheet.Cells.CreateRange(sourceRange.FirstRow + TotalRowCount, 
                      sourceRange.FirstColumn, sourceRange.RowCount, sourceRange.ColumnCount);
    
    // Αντιγραφή δεδομένων από την περιοχή προέλευσης στην περιοχή προορισμού
    destRange.Copy(sourceRange);
    
    // Ενημέρωση συνολικού αριθμού γραμμών
    TotalRowCount += sourceRange.RowCount;
}
```

##### Αποθήκευση του συγχωνευμένου φύλλου εργασίας

Τέλος, αποθηκεύστε το βιβλίο εργασίας με όλα τα φύλλα εργασίας συνδυασμένα σε ένα:
```csharp
destWorkbook.Save(outputDir + "outputCombineMultipleWorksheetsSingleWorksheet.xlsx");
Console.WriteLine("CombineMultipleWorksheetsSingleWorksheet executed successfully.\r\n");
```
