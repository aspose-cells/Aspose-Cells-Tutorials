### Φόρτωση και τροποποίηση αρχείου Excel

**Επισκόπηση**: Αυτή η ενότητα σάς δείχνει πώς να φορτώσετε ένα μεγάλο βιβλίο εργασίας, να επανεξετάσετε τα φύλλα εργασίας του, να εισαγάγετε γραμμές στην αρχή κάθε φύλλου και να συμπληρώσετε αυτές τις νέες γραμμές με δεδομένα.

#### Βήμα 1: Ορισμός διαδρομών εισόδου και εξόδου

Καθορίστε καταλόγους για το αρχείο προέλευσης και την έξοδο. Αντικαταστήστε `"YOUR_SOURCE_DIRECTORY"` και `"YOUR_OUTPUT_DIRECTORY"` με πραγματικές διαδρομές στο μηχάνημά σας:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

string inputFile = SourceDir + "/Sample.xls";
string outputFile = outputDir + "/output_out.xls";
```

#### Βήμα 2: Φόρτωση του βιβλίου εργασίας

Χρησιμοποιήστε το Aspose.Cells για να φορτώσετε ένα υπάρχον αρχείο Excel. Αυτό το βήμα αρχικοποιεί ένα `Workbook` αντικείμενο:

```csharp
try {
    Workbook workbook = new Workbook(inputFile);
    DateTime start = DateTime.Now;
    
    // Προχωρήστε με τις τροποποιήσεις...
} catch (Exception ex) {
    // Χειρισμός εξαιρέσεων εδώ
}
```

#### Βήμα 3: Εισαγωγή και συμπλήρωση γραμμών

Επαναλάβετε κάθε φύλλο εργασίας, εισάγοντας 100 γραμμές στην αρχή. Στη συνέχεια, συμπληρώστε αυτές τις γραμμές με προσαρμοσμένα δεδομένα:

```csharp
for (int i = 0; i < workbook.getWorksheets().getCount(); i++) {
    Worksheet worksheet = workbook.getWorksheets().get(i);
    Cells cells = worksheet.getCells();

    // Εισαγάγετε 100 γραμμές στο δείκτη 0.
    cells.insertRows(0, 100);

    for (int r = 0; r < 100; r++) {
        cells.get(r, 0).putValue("This is testing row #: " + r.ToString());
    }
}
```

#### Βήμα 4: Αποθήκευση του τροποποιημένου βιβλίου εργασίας

Αφού κάνετε τροποποιήσεις, αποθηκεύστε το βιβλίο εργασίας σε ένα νέο αρχείο:

```csharp
workbook.save(outputFile);
DateTime end = DateTime.Now;
TimeSpan time = end - start;

// Προαιρετικά, καταγραφή του χρόνου επεξεργασίας.
```
