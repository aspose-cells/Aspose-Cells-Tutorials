## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία εισαγωγής μιας γραμμής σε ένα φύλλο εργασίας του Excel.
### Βήμα 1: Ανοίξτε το αρχείο Excel (H3)
Αρχικά, πρέπει να ανοίξετε το αρχείο Excel χρησιμοποιώντας ένα `FileStream`Αυτό το βήμα περιλαμβάνει την ανάγνωση του υπάρχοντος εγγράφου Excel σας:
```csharp
using System.IO;

// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "your_data_directory_path/";

// Δημιουργία ροής αρχείων που περιέχει το αρχείο Excel που θα ανοιχτεί
FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open);

// Άνοιγμα του αρχείου Excel μέσω της ροής αρχείων
Workbook workbook = new Workbook(fstream);
```
### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας (H3)
Στη συνέχεια, αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας που θέλετε να τροποποιήσετε. Αυτό το παράδειγμα έχει πρόσβαση στο πρώτο φύλλο εργασίας:
```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Worksheet worksheet = workbook.Worksheets[0];
```
### Βήμα 3: Εισαγωγή μιας γραμμής στο φύλλο εργασίας (H3)
Τώρα, εισαγάγετε μια γραμμή στην επιθυμητή θέση. Ο ακόλουθος κώδικας εισάγει μια γραμμή στην τρίτη θέση (ευρετήριο 2):
```csharp
// Εισαγωγή μιας γραμμής στο φύλλο εργασίας στην 3η θέση
worksheet.Cells.InsertRow(2);
```
### Βήμα 4: Αποθήκευση και κλείσιμο της ροής αρχείων (H3)
Τέλος, αποθηκεύστε τις τροποποιήσεις σας και κλείστε τη ροή αρχείων στους ελεύθερους πόρους:
```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
workbook.Save(dataDir + "output.out.xls");

// Κλείσιμο της ροής αρχείων
fstream.Close();
```
