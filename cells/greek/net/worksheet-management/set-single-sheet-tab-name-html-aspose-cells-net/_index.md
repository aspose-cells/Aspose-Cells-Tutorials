## Οδηγός Εφαρμογής
Ας προσαρμόσουμε το όνομα της καρτέλας ενός φύλλου σε HTML χρησιμοποιώντας το Aspose.Cells για .NET. Αυτή η διαδικασία περιλαμβάνει τη φόρτωση του αρχείου Excel, τον καθορισμό επιλογών εξαγωγής και την αποθήκευσή του ως αρχείο HTML με προσαρμοσμένες ρυθμίσεις.

### Φόρτωση του δείγματος αρχείου Excel
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας του Excel που περιέχει μόνο ένα φύλλο:
```csharp
// Καθορισμός καταλόγου πηγής
string sourceDir = "Your source directory path";
Workbook wb = new Workbook(sourceDir + "sampleSingleSheet.xlsx");
```
Εδώ, φορτώνουμε ένα αρχείο Excel ενός φύλλου σε ένα `Workbook` αντικείμενο. Βεβαιωθείτε ότι η διαδρομή προς το αρχείο σας είναι σωστή.

### Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML
Για να προσαρμόσετε τον τρόπο εξαγωγής του φύλλου Excel σε HTML, χρησιμοποιήστε το `HtmlSaveOptions` τάξη:
```csharp
// Καθορισμός επιλογών αποθήκευσης HTML
Aspose.Cells.HtmlSaveOptions options = new Aspose.Cells.HtmlSaveOptions();
options.Encoding = System.Text.Encoding.UTF8;
options.ExportImagesAsBase64 = true; // Ενσωματώστε εικόνες απευθείας στο αρχείο HTML
options.ExportGridLines = true;      // Εξαγωγή γραμμών πλέγματος για διατήρηση της δομής
options.ExportSimilarBorderStyle = true;
options.ExportBogusRowData = true;   // Συμπερίληψη κρυφών δεδομένων γραμμών και στηλών
options.ExcludeUnusedStyles = true;  // Μειώστε το μέγεθος εξαιρώντας τα αχρησιμοποίητα στυλ
options.ExportHiddenWorksheet = false; // Εξαγωγή μόνο ορατών φύλλων εργασίας
```
### Εξαγωγή του βιβλίου εργασίας σε HTML
Αφού ορίσετε τις επιλογές σας, μπορείτε πλέον να αποθηκεύσετε το βιβλίο εργασίας σε μορφή HTML:
```csharp
// Καθορισμός καταλόγου εξόδου
string outputDir = "Your output directory path";
wb.Save(outputDir + "outputSampleSingleSheet.htm", options);
Console.WriteLine("Export executed successfully.");
```
Αυτός ο κώδικας αποθηκεύει το αρχείο Excel ενός φύλλου ως έγγραφο HTML με όλες τις καθορισμένες ρυθμίσεις.
