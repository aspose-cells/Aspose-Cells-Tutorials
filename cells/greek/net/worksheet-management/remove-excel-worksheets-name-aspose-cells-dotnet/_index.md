### Αφαίρεση φύλλων εργασίας χρησιμοποιώντας ονόματα φύλλων
Η κατάργηση συγκεκριμένων φύλλων μπορεί να είναι κρίσιμη για τη διαχείριση δεδομένων. Ας δούμε πώς λειτουργεί:

#### Βήμα 1: Φόρτωση του αρχείου Excel
Ξεκινήστε φορτώνοντας το αρχείο Excel χρησιμοποιώντας ένα `FileStream`.

```csharp
string dataDir = "your_directory_path_here";

// Δημιουργήστε ένα FileStream για να ανοίξετε το αρχείο Excel
using (FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open))
{
    // Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας και φορτώστε το αρχείο μέσω της ροής
    Workbook workbook = new Workbook(fstream);
}
```
*Γιατί να χρησιμοποιήσετε `FileStream`;* Σας επιτρέπει να διαχειρίζεστε αρχεία αποτελεσματικά, διασφαλίζοντας ότι οι πόροι απελευθερώνονται μετά την ολοκλήρωση των εργασιών.

#### Βήμα 2: Αφαίρεση του Φύλλου Εργασίας
Τώρα, ας αφαιρέσουμε ένα φύλλο εργασίας με το όνομά του:

```csharp
// Κατάργηση ενός φύλλου εργασίας χρησιμοποιώντας το όνομα του φύλλου του
workbook.Worksheets.RemoveAt("Sheet1");
```
Αυτή η μέθοδος στοχεύει και διαγράφει απευθείας το καθορισμένο φύλλο, βελτιώνοντας τις εργασίες διαχείρισης αρχείων.

#### Βήμα 3: Αποθήκευση των αλλαγών
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρηθούν οι αλλαγές:

```csharp
// Αποθήκευση του ενημερωμένου βιβλίου εργασίας
using (FileStream fstream = new FileStream(dataDir + "output.out.xls", FileMode.Create))
{
    workbook.Save(fstream);
}
```
