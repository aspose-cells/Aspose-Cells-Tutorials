## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στη χρήση `CreateSafeSheetName` για την αποτελεσματική διαχείριση των ονομάτων των φύλλων.

### Περικοπή και αντικατάσταση μη έγκυρων χαρακτήρων
1. **Επισκόπηση:**
   - Διασφαλίζει τη συμμόρφωση με τους κανόνες ονομασίας του Excel, αφαιρώντας μη έγκυρους χαρακτήρες και περικόπτοντας μεγάλα ονόματα.
2. **Περικοπή μακροσκελών ονομάτων:**
Η μέθοδος περιορίζει αυτόματα τα ονόματα σε 31 χαρακτήρες:
```csharp
string name1 = CellsHelper.CreateSafeSheetName("this is first name which is created using CellsHelper.CreateSafeSheetName and truncated to 31 characters");
```
3. **Αντικατάσταση μη έγκυρων χαρακτήρων:**
Αντικαθιστά τους μη έγκυρους χαρακτήρες με μια υπογράμμιση (`_`):
```csharp
string name2 = CellsHelper.CreateSafeSheetName("<> + (adj.Private ? \" Private\" : \")", '_');
```
4. **Εμφάνιση αποτελεσμάτων:**
Επαληθεύστε τα αποτελέσματα χρησιμοποιώντας `Console.WriteLine()`:
```csharp
Console.WriteLine(name1);  // Εξάγει περικομμένο όνομα
Console.WriteLine(name2);  // Εξάγει το αποστειρωμένο όνομα με υπογράμμιση
Console.WriteLine("CreateSafeSheetNames executed successfully.");
```
### Συμβουλές αντιμετώπισης προβλημάτων
- **Έλεγχος μήκους ονόματος:** Βεβαιωθείτε ότι τα ονόματα είναι εντός του ορίου του Excel.
- **Επικύρωση χαρακτήρων:** Ελέγξτε τους μη έγκυρους χαρακτήρες στο Excel για να προεπικυρώσετε τα ονόματα των φύλλων.

**Ε1: Τι συμβαίνει εάν το όνομα του φύλλου μου υπερβαίνει τους 31 χαρακτήρες;**
