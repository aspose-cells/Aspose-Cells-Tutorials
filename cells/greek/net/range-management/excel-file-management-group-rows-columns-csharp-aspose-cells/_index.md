### Δυνατότητα 1: Ροή αρχείων και λειτουργίες βιβλίου εργασίας

#### Άνοιγμα αρχείου Excel χρησιμοποιώντας το FileStream

Για να ξεκινήσετε, ανοίξτε το αρχείο Excel χρησιμοποιώντας ένα `FileStream`Αυτή η μέθοδος διαβάζει αποτελεσματικά μεγάλα αρχεία χωρίς να τα φορτώνει εξ ολοκλήρου στη μνήμη.

```csharp
using System.IO;
using Aspose.Cells;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";

// Δημιουργήστε ένα FileStream για το αρχείο Excel
using (FileStream fstream = new FileStream(SourceDir + "/book1.xls", FileMode.Open))
{
    // Άνοιγμα του βιβλίου εργασίας με τη ροή αρχείων
    Workbook workbook = new Workbook(fstream);

    // Πρόσβαση στο πρώτο φύλλο εργασίας
    Worksheet worksheet = workbook.Worksheets[0];

    // Εκτελέστε λειτουργίες στο φύλλο εργασίας εδώ
}
```

**Γιατί να χρησιμοποιήσετε το FileStream;**

Το FileStream είναι ωφέλιμο για τον χειρισμό μεγάλων αρχείων, καθώς σας επιτρέπει να εργάζεστε με δεδομένα σε τμήματα αντί να φορτώνετε τα πάντα ταυτόχρονα.

### Χαρακτηριστικό 2: Ομαδοποίηση και απόκρυψη γραμμών

#### Ομαδοποίηση γραμμών στο Excel

Για να απλοποιήσετε την παρουσίαση δεδομένων, μπορείτε να ομαδοποιήσετε γραμμές. Δείτε πώς:

```csharp
using (FileStream fstream = new FileStream(SourceDir + "/book1.xls", FileMode.Open))
{
    Workbook workbook = new Workbook(fstream);
    Worksheet worksheet = workbook.Worksheets[0];

    // Ομαδοποιήστε τις πρώτες έξι σειρές και αποκρύψτε τις
    worksheet.Cells.GroupRows(0, 5, true);

    // Αποθήκευση των αλλαγών σε νέο αρχείο
    string outputDir = @"YOUR_OUTPUT_DIRECTORY";
    workbook.Save(outputDir + "/row_grouped_output.xls");
}
```

**Εξήγηση**: Το `GroupRows` Η μέθοδος ομαδοποιεί τις γραμμές μεταξύ των δεικτών 0 και 5. Η τρίτη παράμετρος `true` υποδεικνύει ότι αυτές οι γραμμές πρέπει να είναι κρυφές.

### Χαρακτηριστικό 3: Ομαδοποίηση και Απόκρυψη Στηλών

#### Ομαδοποίηση στηλών στο Excel

Όπως και με την ομαδοποίηση γραμμών, μπορείτε επίσης να ομαδοποιήσετε στήλες:

```csharp
using (FileStream fstream = new FileStream(SourceDir + "/book1.xls", FileMode.Open))
{
    Workbook workbook = new Workbook(fstream);
    Worksheet worksheet = workbook.Worksheets[0];

    // Ομαδοποιήστε τις τρεις πρώτες στήλες και αποκρύψτε τις
    worksheet.Cells.GroupColumns(0, 2, true);

    // Αποθήκευση των αλλαγών σε νέο αρχείο
    string outputDir = @"YOUR_OUTPUT_DIRECTORY";
    workbook.Save(outputDir + "/column_grouped_output.xls");
}
```

**Εξήγηση**: Το `GroupColumns` Η μέθοδος ομαδοποιεί στήλες από τον δείκτη 0 έως τον 2. Ορίζοντας την τελευταία παράμετρο σε `true` αποκρύπτει αυτές τις στήλες.
