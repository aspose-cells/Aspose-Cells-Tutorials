## Οδηγός Εφαρμογής

Ας δούμε βήμα προς βήμα το μέγιστο εύρος εμφάνισης ενός φύλλου εργασίας.

### Επισκόπηση

Η πρόσβαση στο μέγιστο εύρος εμφάνισης επιτρέπει την κατανόηση του μέρους ενός φύλλου Excel που είναι ορατό. Αυτό είναι χρήσιμο για μεγάλα σύνολα δεδομένων όπου μπορεί να εμφανίζεται μόνο ένα υποσύνολο ανά πάσα στιγμή.

#### Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας

Δημιουργήστε μια παρουσία του `Workbook` κλάση για να φορτώσετε το αρχείο Excel:

```csharp
// Κατάλογος πηγής
total_sourceDir = RunExamples.Get_SourceDirectory();

// Δημιουργία αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook(sourceDir + "sampleAccessingMaximumDisplayRangeofWorksheet.xlsx");
```

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας

Ανακτήστε το φύλλο εργασίας με το οποίο θέλετε να εργαστείτε. Συνήθως, αυτό είναι το πρώτο φύλλο:

```csharp
// Πρόσβαση στο πρώτο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

#### Βήμα 3: Ανάκτηση μέγιστου εύρους εμφάνισης

Χρησιμοποιήστε το `MaxDisplayRange` ιδιοκτησία του `Cells` συλλογή για να αποκτήσετε την γκάμα:

```csharp
// Πρόσβαση στο μέγιστο εύρος εμφάνισης
Range range = worksheet.Cells.MaxDisplayRange;
```

#### Βήμα 4: Εξαγωγή του αποτελέσματος

Εκτυπώστε ή χρησιμοποιήστε τις πληροφορίες μέγιστης εμβέλειας εμφάνισης, όπως απαιτείται:

```csharp
// Εκτύπωση της ιδιότητας Μέγιστο εύρος εμφάνισης RefersTo
Console.WriteLine("Maximum Display Range: " + range.RefersTo);
Console.WriteLine("AccessingMaximumDisplayRangeofWorksheet executed successfully.");
```
