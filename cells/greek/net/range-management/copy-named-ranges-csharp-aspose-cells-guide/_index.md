## Οδηγός Εφαρμογής

### Δημιουργία και ονομασία περιοχών

#### Επισκόπηση
Η δημιουργία ονομασμένων περιοχών επιτρέπει ευκολότερη αναφορά και χειρισμό μέσα στα φύλλα του Excel σας.

**Βήματα:**
1. **Δημιουργία ενός βιβλίου εργασίας**
   ```csharp
   Workbook workbook = new Workbook();
   ```
2. **Συλλογή φύλλων εργασίας της Access**
   ```csharp
   WorksheetCollection worksheets = workbook.Worksheets;
   ```
3. **Δημιουργήστε το πρώτο εύρος και ονομάστε το**
   ```csharp
   Worksheet worksheet = workbook.Worksheets[0];
   Range range1 = worksheet.Cells.CreateRange("E12", "I12");
   range1.Name = "MyRange";
   ```
4. **Ορισμός περιγράμματος για οπτική διάκριση**
   ```csharp
   range1.SetOutlineBorder(BorderType.TopBorder, CellBorderType.Medium, Color.FromArgb(0, 0, 128));
   // Επαναλάβετε για άλλα περιγράμματα...
   ```
5. **Εισαγωγή δεδομένων στο εύρος**
   ```csharp
   range1[0, 0].PutValue("Test");
   range1[0, 4].PutValue("123");
   ```

### Αντιγραφή ονομασμένων περιοχών

#### Επισκόπηση
Η αντιγραφή εύρους με όνομα σάς επιτρέπει να αντιγράφετε δεδομένα και να μορφοποιείτε αποτελεσματικά.

**Βήματα:**
1. **Δημιουργήστε μια δεύτερη περιοχή**
   ```csharp
   Range range2 = worksheet.Cells.CreateRange("B3", "F3");
   range2.Name = "testrange";
   ```
2. **Αντιγραφή δεδομένων από το πρώτο εύρος**
   ```csharp
   range2.Copy(range1);
   ```
