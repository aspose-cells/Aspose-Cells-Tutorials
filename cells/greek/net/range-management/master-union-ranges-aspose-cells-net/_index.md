## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη διαδικασία ενοποίησης εύρους και εφαρμογής στυλ.

### Ανάκτηση ονομασμένων περιοχών
Αρχικά, αποκτήστε πρόσβαση σε ονομασμένες περιοχές μέσα στο βιβλίο εργασίας του Excel:
```csharp
// Ανοίξτε ένα υπάρχον αρχείο excel.
Workbook workbook = new Workbook("sampleUnionOfRanges.xlsx");

// Λάβετε τα ονομασμένα εύρη από το πρώτο φύλλο εργασίας.
Range[] ranges = workbook.Worksheets[0].GetNamedRanges();
```
**Εξήγηση**: Το `GetNamedRanges` Η μέθοδος ανακτά όλα τα ονομασμένα εύρη που ορίζονται στο καθορισμένο φύλλο εργασίας, επιτρέποντας τον χειρισμό.

### Δημιουργία και εφαρμογή στυλ
Για να διαφοροποιήσετε οπτικά τα ενοποιημένα εύρη, εφαρμόστε ένα προσαρμοσμένο στυλ:
```csharp
// Δημιουργήστε ένα νέο αντικείμενο στυλ.
Style style = workbook.CreateStyle();

// Ορίστε το χρώμα φόντου σε κόκκινο με συμπαγές μοτίβο.
style.ForegroundColor = Color.Red;
style.Pattern = BackgroundType.Solid;

// Αρχικοποιήστε το StyleFlag για να καθορίσετε ποια στοιχεία του κελιού θα μορφοποιηθούν.
StyleFlag flag = new StyleFlag();
flag.CellShading = true; // Εφαρμόζουμε σκίαση
```

### Εκτέλεση Λειτουργίας Ένωσης
Τώρα, εκτελέστε την πράξη ένωσης στις ονομασμένες περιοχές σας:
```csharp
// Δημιουργήστε ένα ArrayList για να αποθηκεύσετε το αποτέλεσμα της λειτουργίας ένωσης.
ArrayList al = ranges[0].Union(ranges[1]);
```
**Εξήγηση**: Το `Union` η μέθοδος συνδυάζει πολλαπλά εύρη σε μία μόνο συλλογή εύρους. Χρησιμοποιούμε ένα `ArrayList` εδώ για λόγους απλότητας, αλλά προσαρμόστε το ανάλογα με τις ανάγκες.

### Εφαρμογή στυλ σε ενοποιημένες περιοχές
Μόλις ενοποιηθούν, εφαρμόστε τα στυλ:
```csharp
foreach (Range rng in al)
{
    // Εφαρμόστε το προηγουμένως δημιουργημένο στυλ σε κάθε εύρος.
    rng.ApplyStyle(style, flag);
}
```
**Εξήγηση**: Το `ApplyStyle` Η μέθοδος χρησιμοποιεί το προσαρμοσμένο αντικείμενο στυλ και τις σημαίες μας για να μορφοποιήσει κάθε κελί εντός των ενοποιημένων περιοχών.

### Αποθήκευση του βιβλίου εργασίας
Τέλος, αποθηκεύστε τις αλλαγές σας:
```csharp
// Αποθηκεύστε το βιβλίο εργασίας με στυλιζαρισμένα εύρη.
workbook.Save("outputUnionOfRanges.xlsx");
```
