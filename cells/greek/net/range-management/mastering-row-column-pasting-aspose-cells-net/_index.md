## Οδηγός Εφαρμογής
Αυτή η ενότητα σας καθοδηγεί στην υλοποίηση της αντιγραφής γραμμών και στηλών με το Aspose.Cells.

### Επισκόπηση της επικόλλησης γραμμών/στηλών
Ο στόχος είναι η αντιγραφή δεδομένων από ένα φύλλο εργασίας σε ένα άλλο, προσαρμόζοντας παράλληλα τη συμπεριφορά επικόλλησης. Θα χρησιμοποιήσουμε `CopyOptions` και `PasteOptions` για τον σκοπό αυτό.

#### Βήμα 1: Φόρτωση του αρχείου Excel προέλευσης
Ξεκινήστε φορτώνοντας το αρχείο προέλευσης Excel:

```csharp
// Ορισμός καταλόγων
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// Φόρτωση του βιβλίου εργασίας
Workbook wb = new Workbook(sourceDir + "SamplePasteOptions.xlsx");
```

#### Βήμα 2: Φύλλα εργασίας προέλευσης και προορισμού πρόσβασης
Αποκτήστε πρόσβαση τόσο στο φύλλο εργασίας προέλευσης που περιέχει τα δεδομένα σας όσο και στη δημιουργία ενός φύλλου προορισμού:

```csharp
// Λήψη του πρώτου φύλλου εργασίας ως πηγής
Worksheet source = wb.Worksheets[0];

// Προσθήκη ενός ακόμη φύλλου για επικόλληση
Worksheet destination = wb.Worksheets.Add("DestSheet");
```

#### Βήμα 3: Ρύθμιση παραμέτρων των επιλογών αντιγραφής
Σειρά `CopyOptions` για να παραπέμψετε τις πηγές δεδομένων στο φύλλο προορισμού:

```csharp
// Ορισμός επιλογών αντιγραφής
CopyOptions options = new CopyOptions();
options.ReferToDestinationSheet = true;
```

#### Βήμα 4: Ορισμός Επιλογών Επικόλλησης
Ρύθμιση παραμέτρων `PasteOptions` για προσαρμοσμένη συμπεριφορά επικόλλησης:

```csharp
// Ορισμός Επιλογών Επικόλλησης
PasteOptions pasteOptions = new PasteOptions();
pasteOptions.PasteType = PasteType.Values; // Επικόλληση μόνο τιμών
pasteOptions.OnlyVisibleCells = true;      // Συμπερίληψη μόνο ορατών κελιών
```

#### Βήμα 5: Αντιγραφή γραμμών με επιλογές
Εκτελέστε τη λειτουργία αντιγραφής χρησιμοποιώντας τις καθορισμένες επιλογές:

```csharp
// Εκτέλεση αντιγραφής γραμμών
destination.Cells.CopyRows(source.Cells, 0, 0, source.Cells.MaxDisplayRange.RowCount, options, pasteOptions);
```
