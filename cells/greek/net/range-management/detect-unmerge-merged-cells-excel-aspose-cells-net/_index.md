## Οδηγός Εφαρμογής
Αυτή η ενότητα περιγράφει λεπτομερώς τη διαδικασία ανίχνευσης και κατάργησης της συγχώνευσης συγχωνευμένων κελιών χρησιμοποιώντας το Aspose.Cells. Θα αναλύσουμε κάθε βήμα για λόγους σαφήνειας.

### Ανίχνευση συγχωνευμένων κελιών
Αρχικά, ανοίξτε ένα αρχείο Excel που περιέχει συγχωνευμένα κελιά:

```csharp
// Δημιουργήστε ένα νέο αντικείμενο Βιβλίου εργασίας με τη διαδρομή αρχείου Excel
Workbook workbook = new Workbook("path_to_your_file/sampleDetectMergedCellsAndUnmerge.xlsx");
```

Αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε να τροποποιήσετε με βάση το όνομα ή το ευρετήριο:

```csharp
Worksheet worksheet = workbook.Worksheets["Sheet1"];
```

Ανάκτηση μιας λίστας συγχωνευμένων κελιών από αυτό το φύλλο εργασίας:

```csharp
ArrayList mergedCellsList = worksheet.Cells.MergedCells;
```

### Κατάργηση συγχώνευσης συγχωνευμένων κελιών
Βρόχος μέσα από κάθε μία `CellArea` για να τα αποσυνδέσετε:

```csharp
for (int i = 0; i < mergedCellsList.Count; i++)
{
    CellArea cellArea = (CellArea)mergedCellsList[i];
    
    int startRow = cellArea.StartRow;
    int startColumn = cellArea.StartColumn;
    int totalRows = cellArea.EndRow - startRow + 1;
    int totalColumns = cellArea.EndColumn - startColumn + 1;

    // Αποσυγχώνευση των κελιών
    worksheet.Cells.UnMerge(startRow, startColumn, totalRows, totalColumns);
}
```

### Αποθήκευση αλλαγών
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να διατηρήσετε τις αλλαγές:

```csharp
workbook.Save("outputDetectMergedCellsAndUnmerge.xlsx");
Console.WriteLine("Successfully detected and unmerged merged cells.");
```
