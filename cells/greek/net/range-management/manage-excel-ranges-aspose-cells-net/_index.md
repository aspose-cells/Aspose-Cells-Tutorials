## Οδηγός Εφαρμογής

### Δημιουργία και ονομασία περιοχής κελιών σε φύλλο εργασίας Excel
Αυτή η λειτουργία θα σας δείξει πώς να δημιουργήσετε μια συγκεκριμένη περιοχή μέσα σε ένα φύλλο εργασίας και να της εκχωρήσετε ένα όνομα για εύκολη αναφορά.

#### Επισκόπηση
Θα μάθετε πώς να ορίζετε μια περιοχή κελιών από A1 έως C10 και να ονομάζετε αυτήν την περιοχή με αναφορά φύλλου, καθιστώντας τα δεδομένα σας πιο προσβάσιμα.

#### Βήματα Υλοποίησης

##### Βήμα 1: Αρχικοποίηση του βιβλίου εργασίας
Δημιουργήστε μια παρουσία του `Workbook` το οποίο αντιπροσωπεύει ένα αρχείο Excel.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργία νέου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```

##### Βήμα 2: Πρόσβαση στο Φύλλο εργασίας και στη συλλογή κελιών
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας σας και στη συλλογή κελιών του.
```csharp
// Λήψη του πρώτου φύλλου εργασίας του βιβλίου εργασίας
Worksheet sheet = workbook.Worksheets[0];

// Πρόσβαση στη συλλογή κελιών του φύλλου εργασίας
Cells cells = sheet.Cells;
```

##### Βήμα 3: Δημιουργήστε μια περιοχή κελιών
Ορίστε μια περιοχή μέσα στα κελιά, καθορίζοντας τις επιθυμητές θέσεις έναρξης και λήξης.
```csharp
// Δημιουργήστε μια περιοχή κελιών από A1 έως C10
Range localRange = cells.CreateRange("A1", "C10");
```

##### Βήμα 4: Ορισμός ονόματος με αναφορά φύλλου
Ονομάστε το δημιουργημένο εύρος για ευκολότερη αναγνώριση και αναφορά σε τύπους ή σενάρια.
```csharp
// Αντιστοίχιση ονόματος με αναφορά φύλλου στην περιοχή που δημιουργήθηκε
localRange.Name = "Sheet1!local";
```

##### Βήμα 5: Αποθήκευση του βιβλίου εργασίας σας
Διατηρήστε τις αλλαγές σας αποθηκεύοντας το βιβλίο εργασίας σε έναν καθορισμένο κατάλογο.
```csharp
// Αποθήκευση του βιβλίου εργασίας στον καθορισμένο κατάλογο εξόδου
workbook.Save(Path.Combine(outputDir, "outputWorksheetNamedRange.xlsx"));
```
