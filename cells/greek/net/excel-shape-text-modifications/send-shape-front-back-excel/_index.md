# Αποστολή σχήματος εμπρός ή πίσω στο Excel

## Εισαγωγή
Όταν εργάζεστε με αρχεία Excel, μπορεί να χρειαστείτε περισσότερο έλεγχο στα οπτικά στοιχεία του υπολογιστικού σας φύλλου. Τα σχήματα, όπως οι εικόνες και τα γραφικά, μπορούν να βελτιώσουν την παρουσίαση των δεδομένων σας. Τι συμβαίνει όμως όταν αυτά τα σχήματα επικαλύπτονται ή πρέπει να αναδιαταχθούν; Εδώ ακριβώς ξεχωρίζει το Aspose.Cells για .NET. Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στα βήματα για τον χειρισμό σχημάτων σε ένα φύλλο εργασίας του Excel, συγκεκριμένα στέλνοντας σχήματα μπροστά ή πίσω από άλλα σχήματα. Αν είστε έτοιμοι να βελτιώσετε το παιχνίδι σας στο Excel, ας ξεκινήσουμε αμέσως!
## Προαπαιτούμενα
Πριν ξεκινήσουμε, θα πρέπει να έχετε μερικά πράγματα στη θέση τους:
1. Εγκατάσταση της βιβλιοθήκης Aspose.Cells: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Cells για .NET. Μπορείτε να τη βρείτε. [εδώ](https://releases.aspose.com/cells/net/).
2. Περιβάλλον ανάπτυξης: Βεβαιωθείτε ότι έχετε ρυθμίσει ένα περιβάλλον ανάπτυξης με υποστήριξη .NET, όπως το Visual Studio.
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.
Εντάξει, έχετε τσεκάρει όλα τα κουτάκια στη λίστα προαπαιτούμενων; Τέλεια! Ας προχωρήσουμε στο διασκεδαστικό κομμάτι - τη σύνταξη κώδικα!
## Εισαγωγή πακέτων
Πριν εμβαθύνουμε στην πραγματική κωδικοποίηση, ας εισαγάγουμε τα απαραίτητα πακέτα. Απλώς προσθέστε τα ακόλουθα χρησιμοποιώντας την οδηγία στην κορυφή του αρχείου C#:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System;
```
Αυτοί οι χώροι ονομάτων είναι κρίσιμοι, καθώς περιέχουν τις κλάσεις και τις μεθόδους που θα χρησιμοποιήσουμε για να χειριστούμε αρχεία και σχήματα του Excel.
## Βήμα 1: Ορίστε τις διαδρομές αρχείων σας
Σε αυτό το πρώτο βήμα, πρέπει να καθορίσουμε τους καταλόγους προέλευσης και εξόδου. Εκεί βρίσκεται το αρχείο Excel και εκεί θέλετε να αποθηκεύσετε το τροποποιημένο αρχείο.
```csharp
//Κατάλογος πηγής
string sourceDir = "Your Document Directory";
//Κατάλογος εξόδου
string outputDir = "Your Document Directory";
```
Αντικαθιστώ `"Your Document Directory"` με την πραγματική διαδρομή όπου είναι αποθηκευμένα τα αρχεία Excel σας.
## Βήμα 2: Φόρτωση του βιβλίου εργασίας
Τώρα που έχουμε ορίσει τους καταλόγους μας, ας φορτώσουμε το βιβλίο εργασίας (το αρχείο Excel) που περιέχει τα σχήματα που θέλουμε να χειριστούμε.
```csharp
//Φόρτωση αρχείου Excel πηγής
Workbook wb = new Workbook(sourceDir + "sampleToFrontOrBack.xlsx");
```
Αυτή η γραμμή κώδικα αρχικοποιεί ένα νέο `Workbook` αντικείμενο, φορτώνοντας το καθορισμένο αρχείο Excel στη μνήμη ώστε να μπορέσουμε να εργαστούμε με αυτό.
## Βήμα 3: Πρόσβαση στο Φύλλο Εργασίας 
Στη συνέχεια, πρέπει να έχουμε πρόσβαση στο συγκεκριμένο φύλλο εργασίας όπου βρίσκονται τα σχήματά μας. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε το πρώτο φύλλο εργασίας.
```csharp
//Πρώτο φύλλο εργασίας της Access
Worksheet ws = wb.Worksheets[0];
```
Με αναφορά `Worksheets[0]`, στοχεύουμε στο πρώτο φύλλο του βιβλίου εργασίας μας. Εάν τα σχήματά σας βρίσκονται σε διαφορετικό φύλλο, προσαρμόστε το ευρετήριο ανάλογα.
## Βήμα 4: Πρόσβαση στα σχήματα
Έχοντας έτοιμη την πρόσβαση στο φύλλο εργασίας, ας πάρουμε τα σχήματα που μας ενδιαφέρουν. Για αυτό το παράδειγμα, θα έχουμε πρόσβαση στο πρώτο και στο τέταρτο σχήμα.
```csharp
//Πρόσβαση στο πρώτο και τέταρτο σχήμα
Shape sh1 = ws.Shapes[0];
Shape sh4 = ws.Shapes[3];
```
Αυτές οι γραμμές λαμβάνουν τα συγκεκριμένα σχήματα από το φύλλο εργασίας με βάση τον δείκτη τους.
## Βήμα 5: Εκτύπωση της θέσης των σχημάτων σε σειρά Z
Πριν μετακινήσουμε οποιαδήποτε σχήματα, ας εκτυπώσουμε την τρέχουσα θέση τους σε σειρά Z. Αυτό μας βοηθά να παρακολουθούμε τη θέση τους πριν κάνουμε αλλαγές.
```csharp
//Εκτύπωση της θέσης Z-Order του σχήματος
Console.WriteLine("Z-Order Shape 1: " + sh1.ZOrderPosition);
```
Καλώντας `ZOrderPosition`, μπορούμε να δούμε πού βρίσκεται κάθε σχήμα στη σειρά σχεδίασης.
## Βήμα 6: Στείλτε το πρώτο σχήμα μπροστά
Τώρα είναι ώρα για δράση! Ας στείλουμε το πρώτο σχήμα στο μπροστινό μέρος της Ζ-Τάξης.
```csharp
//Στείλτε αυτό το σχήμα μπροστά
sh1.ToFrontOrBack(2);
```
Περνώντας `2` να `ToFrontOrBack`, δίνουμε οδηγίες στο Aspose.Cells να φέρει αυτό το σχήμα στο μπροστινό μέρος. 
## Βήμα 7: Εκτυπώστε τη θέση Z-Order του δεύτερου σχήματος
Πριν στείλουμε το δεύτερο σχήμα στο πίσω μέρος, ας ελέγξουμε πού βρίσκεται.
```csharp
//Εκτύπωση της θέσης Z-Order του σχήματος
Console.WriteLine("Z-Order Shape 4: " + sh4.ZOrderPosition);
```
Αυτό μας δίνει μια εικόνα για τη θέση του τέταρτου σχήματος πριν κάνουμε οποιεσδήποτε αλλαγές.
## Βήμα 8: Στείλτε το τέταρτο σχήμα προς τα πίσω
Τέλος, θα στείλουμε το τέταρτο σχήμα στο πίσω μέρος της στοίβας Z-Order.
```csharp
//Στείλτε αυτό το σχήμα πίσω
sh4.ToFrontOrBack(-2);
```
Χρησιμοποιώντας `-2` καθώς η παράμετρος στέλνει το σχήμα προς το πίσω μέρος της στοίβας, διασφαλίζοντας ότι δεν θα εμποδίσει άλλα σχήματα ή κείμενο.
## Βήμα 9: Αποθήκευση του βιβλίου εργασίας 
Το τελευταίο βήμα είναι να αποθηκεύσετε το βιβλίο εργασίας σας με τα σχήματα που τοποθετήθηκαν πρόσφατα.
```csharp
//Αποθηκεύστε το αρχείο Excel που προκύπτει
wb.Save(outputDir + "outputToFrontOrBack.xlsx");
```
Αυτή η εντολή αποθηκεύει το τροποποιημένο βιβλίο εργασίας στον καθορισμένο κατάλογο εξόδου.
## Βήμα 10: Μήνυμα επιβεβαίωσης
Τέλος, ας δώσουμε μια απλή επιβεβαίωση για να μας ενημερώσετε ότι η εργασία μας ολοκληρώθηκε με επιτυχία.
```csharp
Console.WriteLine("SendShapeFrontOrBackInWorksheet executed successfully.\r\n");
```
Και αυτό ολοκληρώνει τον κώδικα για το σεμινάριό μας!
