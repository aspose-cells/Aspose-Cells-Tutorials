### Συμπεριλάβετε τη χρήση οδηγιών
Στο επάνω μέρος του αρχείου C#, συμπεριλάβετε τους ακόλουθους χώρους ονομάτων:

Τώρα έρχεται το συναρπαστικό κομμάτι: η πραγματική εφαρμογή! Ακολουθεί μια αναλυτική περιγραφή του ορισμού περιθωρίων για σχόλια ή σχήματα μέσα σε ένα φύλλο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells.
## Βήμα 1: Ορίστε τους καταλόγους σας
Πριν κάνουμε οτιδήποτε με το αρχείο Excel σας, πρέπει να προσδιορίσουμε πού βρίσκεται και πού θα αποθηκεύσουμε το τροποποιημένο αρχείο μας.
```csharp
//Κατάλογος πηγής
string sourceDir = "Your Document Directory";
//Κατάλογος εξόδου
string outputDir = "Your Document Directory";
```
Βεβαιωθείτε ότι έχετε αντικαταστήσει `"Your Document Directory"` με την πραγματική διαδρομή όπου είναι αποθηκευμένα τα αρχεία σας.
## Βήμα 2: Φόρτωση του αρχείου Excel
Σε αυτό το βήμα, θα ανοίξουμε το αρχείο Excel στο οποίο σκοπεύουμε να εργαστούμε. Ας αξιοποιήσουμε τη δύναμη του `Workbook` τάξη.
```csharp
Workbook wb = new Workbook(sourceDir + "sampleSetMarginsOfCommentOrShapeInsideTheWorksheet.xlsx");
```
Αυτή η γραμμή κώδικα φορτώνει το αρχείο Excel στη μνήμη, προετοιμάζοντας το έδαφος για τροποποιήσεις.
## Βήμα 3: Πρόσβαση στο Φύλλο Εργασίας
Στη συνέχεια, πρέπει να έχουμε πρόσβαση στο συγκεκριμένο φύλλο εργασίας που περιέχει τα σχήματα ή τα σχόλια. Θα εργαστούμε με το πρώτο φύλλο εργασίας για λόγους απλότητας.
```csharp
Worksheet ws = wb.Worksheets[0];
```
Αυτός ο κώδικας στοχεύει στο πρώτο φύλλο εργασίας, το οποίο έχει δείκτη 0.
## Βήμα 4: Επανάληψη μέσω σχημάτων
Τώρα πρέπει να επαναλάβουμε όλα τα σχήματα που υπάρχουν στο φύλλο εργασίας. Αυτό θα μας επιτρέψει να εφαρμόσουμε ρυθμίσεις περιθωρίου σε κάθε σχήμα που βρίσκουμε.
```csharp
foreach (Shape sh in ws.Shapes)
```
Χρησιμοποιούμε εδώ έναν βρόχο foreach. Είναι ένας απλός τρόπος για να χειριζόμαστε κάθε σχήμα ξεχωριστά.
## Βήμα 5: Προσαρμογή στοίχισης κειμένου
Κάθε σχήμα μπορεί να έχει ήδη μια ρύθμιση στοίχισης που πρέπει να τροποποιήσουμε. Εδώ, έχουμε πρόσβαση στη στοίχιση κειμένου του σχήματος και καθορίζουμε ότι θα ορίσουμε τα περιθώρια χειροκίνητα.
```csharp
Aspose.Cells.Drawing.Texts.ShapeTextAlignment txtAlign = sh.TextBody.TextAlignment;
txtAlign.IsAutoMargin = false;
```
Ρυθμίζοντας `IsAutoMargin` σε ψευδές, τώρα έχουμε τον έλεγχο των περιθωρίων.
## Βήμα 6: Ορίστε τα περιθώρια
Αυτό είναι το κρίσιμο βήμα όπου ορίζουμε τα περιθώρια. Μπορείτε να προσαρμόσετε αυτές τις τιμές ανάλογα με τις ανάγκες σας.
```csharp
txtAlign.TopMarginPt = 10;
txtAlign.LeftMarginPt = 10;
txtAlign.BottomMarginPt = 10;
txtAlign.RightMarginPt = 10;
```
Σε αυτό το παράδειγμα, ορίζουμε ομοιόμορφα όλα τα περιθώρια σε 10 σημεία. Μπορείτε να προσαρμόσετε αυτές τις τιμές. 
## Βήμα 7: Αποθήκευση του τροποποιημένου αρχείου Excel
Αφού κάνουμε τις αλλαγές μας, ήρθε η ώρα να αποθηκεύσουμε το αρχείο Excel. Ας το κάνουμε!
```csharp
wb.Save(outputDir + "outputSetMarginsOfCommentOrShapeInsideTheWorksheet.xlsx");
```
Αυτή η γραμμή θα αποθηκεύσει το τροποποιημένο αρχείο σας στον κατάλογο εξόδου που ορίσατε νωρίτερα.
## Βήμα 8: Έξοδος επιβεβαίωσης
Τέλος, είναι πάντα καλό να γνωρίζουμε ότι όλα πήγαν ομαλά. Μια απλή έξοδος από την κονσόλα θα επιβεβαιώσει ότι η λειτουργία σας ήταν επιτυχής.
