## Βήμα 1: Ρυθμίστε τον Κατάλογο εξόδου
Ξεκινάμε καθορίζοντας πού θέλουμε να αποθηκεύσουμε το αρχείο μας Excel. Αυτό είναι κρίσιμο γιατί θέλουμε να διασφαλίσουμε ότι το αρχείο εξόδου μας αποθηκεύεται σε μια τοποθεσία που είναι εύκολα προσβάσιμη.
```csharp
// Κατάλογος εξόδου
string outputDir = "Your Document Directory";
```
## Βήμα 2: Δημιουργήστε ένα κενό βιβλίο εργασίας
Τώρα που έχουμε ρυθμίσει τον κατάλογό μας, ας δημιουργήσουμε ένα νέο βιβλίο εργασίας όπου θα προσθέσουμε το περιεχόμενό μας. Αυτό είναι παρόμοιο με το να ξεκινάς με έναν φρέσκο καμβά πριν από τη ζωγραφική.
```csharp
// Δημιουργία κενού βιβλίου εργασίας.
Workbook wb = new Workbook();
```
## Βήμα 3: Πρόσβαση στο Πρώτο φύλλο εργασίας
Στη συνέχεια, θέλουμε να δουλέψουμε με ένα φύλλο εργασίας από το βιβλίο εργασίας μας. Σκεφτείτε ένα φύλλο εργασίας ως μια σελίδα στο βιβλίο σας όπου συμβαίνει όλη η μαγεία.
```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας.
Worksheet ws = wb.Worksheets[0];
```
## Βήμα 4: Προσθέστε ένα πλαίσιο κειμένου
Τώρα, θα προσθέσουμε ένα πλαίσιο κειμένου στο φύλλο εργασίας μας. Εδώ θα πληκτρολογήσουμε το κείμενό μας. Φανταστείτε ότι δημιουργείτε ένα πλαίσιο κειμένου μέσα σε μια διαφάνεια μιας παρουσίασης.
```csharp
// Προσθήκη πλαισίου κειμένου μέσα στο φύλλο εργασίας.
int idx = ws.TextBoxes.Add(5, 5, 50, 200);
Aspose.Cells.Drawing.TextBox tb = ws.TextBoxes[idx];
```
## Βήμα 5: Ορίστε το κείμενο του πλαισίου κειμένου
Ας πληκτρολογήσουμε κάποιο κείμενο. Σε αυτό το παράδειγμα, θα εισάγουμε ιαπωνικούς χαρακτήρες για να δείξουμε τη γραμματοσειρά της Άπω Ανατολής. Είναι τόσο απλό όσο το γράψιμο σε ένα πλαίσιο κειμένου στον υπολογιστή σας!
```csharp
// Ορίστε το κείμενο του πλαισίου κειμένου.
tb.Text = "こんにちは世界"; //Αυτό σημαίνει "Hello World" στα Ιαπωνικά.
```
## Βήμα 6: Καθορίστε τις γραμματοσειρές
Τώρα έρχεται το συναρπαστικό μέρος! Θα ορίσουμε τόσο τη λατινική όσο και τη γραμματοσειρά Άπω Ανατολής για το κείμενο. Αυτό μοιάζει με την επιλογή της τέλειας γραμματοσειράς για ένα φανταχτερό προσκλητήριο γάμου!
```csharp
// Καθορίστε την Άπω Ανατολή και το Λατινικό όνομα της γραμματοσειράς.
tb.TextOptions.LatinName = "Comic Sans MS"; // Αυτή είναι η λατινική γραμματοσειρά που επιλέξαμε.
tb.TextOptions.FarEastName = "KaiTi"; // Αυτή είναι η γραμματοσειρά της Άπω Ανατολής που επιθυμούμε.
```
## Βήμα 7: Αποθηκεύστε το αρχείο εξόδου Excel
Επιτέλους, ας αποθηκεύσουμε το βιβλίο εργασίας μας! Αυτό το βήμα ολοκληρώνει την εργασία μας και διασφαλίζει ότι όλη η σκληρή δουλειά που έχουμε κάνει αποθηκεύεται σωστά. 
```csharp
// Αποθηκεύστε το αρχείο εξόδου Excel.
wb.Save(outputDir + "outputSpecifyFarEastAndLatinNameOfFontInTextOptionsOfShape.xlsx", SaveFormat.Xlsx);
```
