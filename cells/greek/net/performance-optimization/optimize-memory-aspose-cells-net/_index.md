## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στον ορισμό προτιμήσεων μνήμης για βιβλία εργασίας και μεμονωμένα φύλλα εργασίας.

### Ορισμός προτιμήσεων μνήμης σε επίπεδο βιβλίου εργασίας

#### Επισκόπηση

Ρύθμιση παραμέτρων του `MemorySetting` Η ιδιότητα βελτιστοποιεί τη χρήση μνήμης του βιβλίου εργασίας σας, κάτι που είναι ιδιαίτερα χρήσιμο με μεγάλα αρχεία ή πολλαπλές λειτουργίες δεδομένων.

#### Βήματα για την εφαρμογή
1. **Ορισμός προτίμησης μνήμης σε επίπεδο βιβλίου εργασίας:**
    ```csharp
    // Ορισμός της προτίμησης μνήμης σε επίπεδο βιβλίου εργασίας
    wb.Settings.MemorySetting = MemorySetting.MemoryPreference;
    ```
   - **Εξήγηση:** Σύνθεση `MemorySetting` να `MemoryPreference` βελτιστοποιεί τη συνολική χρήση μνήμης του βιβλίου εργασίας.

### Ορισμός προτιμήσεων μνήμης για μεμονωμένα φύλλα εργασίας

#### Επισκόπηση

Η προσαρμογή των προτιμήσεων μνήμης μεμονωμένων φύλλων εργασίας επιτρέπει τον λεπτομερή έλεγχο της χρήσης πόρων.

#### Βήματα για την εφαρμογή
1. **Πρόσβαση σε κελιά και ορισμός προτίμησης μνήμης σε επίπεδο φύλλου εργασίας:**
    ```csharp
    // Πρόσβαση σε κελιά ενός υπάρχοντος φύλλου εργασίας και ορισμός της προτίμησης μνήμης του
    Cells cells = wb.Worksheets[0].Cells;
    cells.MemorySetting = MemorySetting.MemoryPreference;
    ```
   - **Εξήγηση:** Αυτό ορίζει `MemoryPreference` για το πρώτο φύλλο εργασίας, μειώνοντας το αποτύπωμα μνήμης του.

2. **Προσθήκη νέου φύλλου εργασίας με κληρονομημένες ρυθμίσεις:**
    ```csharp
    // Προσθήκη νέου φύλλου εργασίας με προεπιλεγμένες ρυθμίσεις που έχουν κληρονομηθεί από το Βιβλίο εργασίας
    Cells newSheetCells = wb.Worksheets.Add("Sheet2").Cells;
    ```
   - **Εξήγηση:** Το φύλλο εργασίας που προστέθηκε πρόσφατα κληρονομεί τις προτιμήσεις μνήμης από το βιβλίο εργασίας, διασφαλίζοντας συνεπή βελτιστοποίηση.
