## Οδηγός Εφαρμογής

### Δημιουργία αρχείου Excel με το Aspose.Cells
#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να δημιουργήσετε ένα απλό αρχείο Excel χρησιμοποιώντας το Aspose.Cells, γεμίζοντάς το με δείγματα δεδομένων.

**Βήματα Υλοποίησης:**
1. **Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας**
   ```csharp
   Workbook workbook = new Workbook();
   Worksheet ws = workbook.Worksheets[0];
   ```
2. **Συμπλήρωση φύλλου εργασίας με δεδομένα**
   - Χρησιμοποιήστε ένθετους βρόχους για να γεμίσετε κελιά:
     ```csharp
     for (int row = 0; row < 10000; row++)
     {
         for (int col = 0; col < 30; col++)
         {
             ws.Cells[row, col].PutValue(row.ToString() + "," + col.ToString());
         }
     }
     ```
3. **Αποθήκευση του βιβλίου εργασίας**
   - Καθορίστε τον κατάλογο εξόδου και αποθηκεύστε:
     ```csharp
     string outputDir = "YOUR_OUTPUT_DIRECTORY";
     workbook.Save(outputDir + "CellsSample_out.xlsx"); // Ενημερώθηκε σε .xlsx για σύγχρονες μορφές Excel
     ```

### Χρόνος δημιουργίας αρχείου συγκριτικής αξιολόγησης
#### Επισκόπηση
Μετρήστε πόσο χρόνο χρειάζεται για να δημιουργήσετε ένα αρχείο Excel, κάτι που σας βοηθά να βελτιστοποιήσετε την απόδοση.

**Βήματα Υλοποίησης:**
1. **Χρονισμός έναρξης**
   - Καταγράψτε την ώρα έναρξης πριν από τη δημιουργία του βιβλίου εργασίας:
     ```csharp
     DateTime start = DateTime.Now;
     ```
2. **Δημιουργία βιβλίου εργασίας και συμπλήρωση δεδομένων**
   - Ακολουθήστε παρόμοια βήματα όπως παραπάνω για τη συμπλήρωση δεδομένων.
3. **Χρονισμός λήξης**
   - Καταγράψτε την ώρα λήξης μετά την αποθήκευση του αρχείου:
     ```csharp
     DateTime end = DateTime.Now;
     TimeSpan time = end - start;
     Console.WriteLine("Time taken: " + time.TotalSeconds + " seconds"); // Έξοδος στην κονσόλα για ορατότητα
     ```
