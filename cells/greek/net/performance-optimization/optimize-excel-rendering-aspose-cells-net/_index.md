## Οδηγός Εφαρμογής

Ας χωρίσουμε την υλοποίηση σε λογικά μέρη για να είναι εύκολη η παρακολούθησή της.

### Φόρτωση του αρχείου Excel σας

Ξεκινήστε φορτώνοντας το αρχείο Excel του προτύπου σας. Εδώ θα εφαρμόσετε τις ρυθμίσεις απόδοσης:

```csharp
// Φόρτωση αρχείου προτύπου Excel
Workbook workbook = new Workbook(sourceDir + "sampleCrossType.xlsx");
```

### Ρύθμιση απόδοσης PDF με το TextCrossType

Θα ξεκινήσουμε ρυθμίζοντας τις επιλογές αποθήκευσης PDF για να διασφαλίσουμε την ακρίβεια του κειμένου.

#### Αρχικοποίηση επιλογών αποθήκευσης PDF

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.TextCrossType = TextCrossType.StrictInCell;
```
*Εδώ, `TextCrossType.StrictInCell` διασφαλίζει ότι το κείμενο ευθυγραμμίζεται αυστηρά εντός των ορίων των κελιών.*

### Αποθήκευση του αρχείου Excel ως PDF

Μετατρέψτε και αποθηκεύστε το έγγραφό σας ως αρχείο PDF:

```csharp
using (FileStream pdfStream = new FileStream(outputDir + "outputCrossType.pdf", FileMode.Create))
{
    workbook.Save(pdfStream, pdfSaveOptions);
}
```

### Ρύθμιση παραμέτρων απόδοσης εικόνας με το TextCrossType

Στη συνέχεια, ρυθμίστε τις επιλογές απόδοσης εικόνας για να διατηρήσετε την ακεραιότητα του κειμένου στις εικόνες.

#### Αρχικοποίηση επιλογών εικόνας ή εκτύπωσης

```csharp
ImageOrPrintOptions imgOptions = new ImageOrPrintOptions();
imgOptions.TextCrossType = TextCrossType.StrictInCell;
```
*Το ίδιο `TextCrossType` Η ρύθμιση διασφαλίζει τη συνέπεια σε διαφορετικές μορφές εξόδου.*

### Απόδοση και αποθήκευση ως εικόνα PNG

Αποδώστε το φύλλο Excel σας σε εικόνα:

```csharp
SheetRender renderer = new SheetRender(workbook.Worksheets[0], imgOptions);
System.Drawing.Bitmap bitmap = renderer.ToImage(0);

using (FileStream pngStream = new FileStream(outputDir + "outputCrossType.png", FileMode.Create))
{
    bitmap.Save(pngStream, ImageFormat.Png);
}
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Λείπουν αρχεία**Βεβαιωθείτε ότι οι κατάλογοι προέλευσης και εξόδου έχουν οριστεί σωστά.
- **Προβλήματα απόδοσης**: Ελέγξτε αν το `TextCrossType` έχει ρυθμιστεί σωστά για να αποφευχθεί η λανθασμένη ευθυγράμμιση κειμένου.
