## Οδηγός Εφαρμογής

Αυτός ο οδηγός χωρίζεται σε τρία κύρια χαρακτηριστικά: φόρτωση βιβλίου εργασίας, ρύθμιση παραμέτρων υπολογισμού και υπολογισμός τύπων.

### Χαρακτηριστικό 1: Φόρτωση βιβλίου εργασίας

Φόρτωση ενός υπάρχοντος αρχείου Excel σε ένα `Workbook` Το αντικείμενο είναι απλό. Αυτό σας επιτρέπει να χειρίζεστε τα δεδομένα μέσω προγραμματισμού.

#### Βήμα προς βήμα εφαρμογή:

**3.1 Ρύθμιση καταλόγου πηγαίου κώδικα**
Ορίστε τον κατάλογο προέλευσης όπου βρίσκεται το βιβλίο εργασίας του προτύπου σας.

```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY";
```

**3.2 Φόρτωση αρχείου Excel**
Δημιουργήστε ένα `Workbook` ένα instance και ανοίξτε ένα υπάρχον αρχείο.

```csharp
// Φόρτωση του βιβλίου εργασίας από την καθορισμένη διαδρομή
Workbook workbook = new Workbook(sourceDir + "book1.xls");
```

### Λειτουργία 2: Ρύθμιση παραμέτρων υπολογισμού

Η βελτιστοποίηση των υπολογισμών τύπων είναι ζωτικής σημασίας για την απόδοση, ειδικά σε μεγάλα βιβλία εργασίας. Δείτε πώς μπορείτε να απενεργοποιήσετε τη ρύθμιση της αλυσίδας υπολογισμών.

#### Βήμα προς βήμα εφαρμογή:

**3.3 Πρόσβαση στις Ρυθμίσεις Τύπου**
Πρόσβαση και τροποποίηση του `FormulaSettings` εντός των ρυθμίσεων του βιβλίου εργασίας.

```csharp
// Απενεργοποιήστε την αλυσίδα υπολογισμών για βελτιστοποίηση της απόδοσης
workbook.Settings.FormulaSettings.EnableCalculationChain = false;
```

### Χαρακτηριστικό 3: Υπολογισμός τύπων βιβλίου εργασίας

Μετά τη διαμόρφωση, βεβαιωθείτε ότι όλοι οι τύποι έχουν υπολογιστεί σωστά.

#### Βήμα προς βήμα εφαρμογή:

**3.4 Υπολογισμός Τύπων**
Καλέστε τη μέθοδο για να υπολογίσετε όλους τους τύπους στο βιβλίο εργασίας.

```csharp
// Επεξεργασία όλων των τύπων στο βιβλίο εργασίας
workbook.CalculateFormula();
```
