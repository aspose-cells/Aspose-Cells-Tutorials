## Οδηγός Εφαρμογής
Στόχος μας είναι να διαβάζουμε τιμές κελιών από ένα μεγάλο φύλλο Excel χρησιμοποιώντας πολλαπλά νήματα ταυτόχρονα. Αυτή η προσέγγιση μπορεί να μειώσει δραστικά τους χρόνους ανάγνωσης για τεράστια σύνολα δεδομένων.

### Αρχικοποίηση βιβλίου εργασίας και κελιών
Αρχικά, θα δημιουργήσουμε ένα βιβλίο εργασίας και θα το συμπληρώσουμε με δείγματα δεδομένων:
```csharp
Workbook testWorkbook = new Workbook();
testWorkbook.Worksheets.Clear();
Worksheet sheet = testWorkbook.Worksheets.Add("Sheet1");

for (var row = 0; row < 10000; row++)
{
    for (var col = 0; col < 100; col++)
    {
        sheet.Cells[row, col].Value = $"R{row}C{col}";
    }
}
```

Αυτό το τμήμα κώδικα αρχικοποιεί ένα βιβλίο εργασίας και συμπληρώνει το πρώτο φύλλο εργασίας με δεδομένα σε μορφή `R<RowNumber>C<ColumnNumber>`.

### Δημιουργία νημάτων για την ανάγνωση τιμών κελιών
Δείτε πώς μπορούμε να ρυθμίσουμε τα νήματα ώστε να διαβάζουν αυτές τις τιμές ταυτόχρονα:
```csharp
public static void ThreadLoop()
{
    Random random = new Random();
    while (Thread.CurrentThread.IsAlive)
    {
        try
        {
            int row = random.Next(0, 10000);
            int col = random.Next(0, 100);
            string s = testWorkbook.Worksheets[0].Cells[row, col].StringValue;
            if (s != $"R{row}C{col}")
            {
                Console.WriteLine("This message will show up when cells read values are incorrect.");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error: {ex.Message}"); // Σφάλμα καταγραφής για εντοπισμό σφαλμάτων
        }
    }
}

public static void TestMultiThreadingRead()
{
    Thread myThread1 = new Thread(new ThreadStart(ThreadLoop));
    myThread1.Start();
    Thread myThread2 = new Thread(new ThreadStart(ThreadLoop));
    myThread2.Start();

    System.Threading.Thread.Sleep(5000);
    myThread1.Abort();
    myThread2.Abort();

    Console.WriteLine("ReadingCellValuesInMultipleThreadsSimultaneously executed successfully.");
}
```

#### Διαμόρφωση κλειδιού
- **Ανάγνωση πολλαπλών νημάτων**: Αποσχολιασμός `testWorkbook.Worksheets[0].Cells.MultiThreadReading = true;` για να ενεργοποιήσετε την ανάγνωση σε πολλαπλά νήματα.
- Χρησιμοποιήστε μπλοκ try-catch για τον ομαλό χειρισμό των εξαιρέσεων, ειδικά στην παραγωγή.
