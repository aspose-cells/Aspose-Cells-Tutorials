## Οδηγός Εφαρμογής

### Ορισμός μέγιστου αριθμού γραμμών για κοινόχρηστους τύπους

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να ορίσετε ένα όριο στον αριθμό των γραμμών που μπορούν να καλυφθούν από έναν κοινόχρηστο τύπο σε ένα φύλλο εργασίας του Excel.

**Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας**

Ξεκινήστε αρχικοποιώντας ένα νέο `Workbook` αντικείμενο, το οποίο αντιπροσωπεύει το αρχείο Excel σας.

```csharp
using Aspose.Cells;

string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε τον κατάλογο εξόδου σας

// Αρχικοποίηση του βιβλίου εργασίας
Workbook wb = new Workbook();
```

**Βήμα 2: Ρύθμιση παραμέτρων μέγιστου αριθμού γραμμών για κοινόχρηστους τύπους**

Χρήση `wb.Settings.MaxRowsOfSharedFormula` για να ορίσετε τον μέγιστο αριθμό γραμμών που μπορεί να καλύψει ένας κοινόχρηστος τύπος. Εδώ, τον ρυθμίζουμε σε 5.

```csharp
// Ορίστε τον μέγιστο αριθμό γραμμών για κοινόχρηστους τύπους
wb.Settings.MaxRowsOfSharedFormula = 5;
```

**Βήμα 3: Αποθήκευση του βιβλίου εργασίας σας**

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας για να εφαρμόσετε αυτές τις ρυθμίσεις.

```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και αποθηκεύστε το
Worksheet ws = wb.Worksheets[0];
wb.Save(outputDir + "outputMaxRowsSharedFormula.xlsx");
```

### Εφαρμογή κοινόχρηστου τύπου σε πολλές γραμμές

#### Επισκόπηση
Μάθετε πώς να χρησιμοποιείτε το `SetSharedFormula` μέθοδος για την αποτελεσματική εφαρμογή τύπων σε πολλά κελιά.

**Βήμα 1: Ρύθμιση του βιβλίου εργασίας και του φύλλου εργασίας σας**

Όπως και πριν, αρχικοποιήστε το βιβλίο εργασίας σας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας του.

```csharp
Workbook wb = new Workbook();
Worksheet ws = wb.Worksheets[0];
```

**Βήμα 2: Εφαρμογή κοινόχρηστου τύπου**

Για να το δείξουμε, ας εφαρμόσουμε το `Sum` τύπος από A1 σε A2 σε 100 γραμμές ξεκινώντας από το κελί D1.

```csharp
Cell cell = ws.Cells["D1"];
cell.SetSharedFormula("=Sum(A1:A2)", 100, 1);
```

**Βήμα 3: Αποθήκευση του βιβλίου εργασίας σας**

Βεβαιωθείτε ότι έχετε αποθηκεύσει τις αλλαγές σας για να δείτε τα αποτελέσματα της εφαρμογής κοινόχρηστου τύπου.

```csharp
wb.Save(outputDir + "outputApplySharedFormula.xlsx");
```
