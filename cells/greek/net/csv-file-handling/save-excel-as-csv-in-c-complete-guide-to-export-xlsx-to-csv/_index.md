# Αποθήκευση Excel ως CSV – Πλήρης Οδηγός C#

Έχετε χρειαστεί ποτέ να **αποθηκεύσετε Excel ως CSV** αλλά δεν ήξερες ποια κλήση API κάνει τη δουλειά; Δεν είστε οι μόνοι. Είτε χτίζετε μια ροή δεδομένων, τροφοδοτείτε ένα παλιό σύστημα, είτε απλώς χρειάζεστε μια γρήγορη εξαγωγή κειμένου, η μετατροπή ενός αρχείου `.xlsx` σε `.csv` είναι ένα κοινό εμπόδιο για πολλούς προγραμματιστές.

Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία: από το **φόρτωμα ενός βιβλίου εργασίας Excel** μέχρι τη διαμόρφωση της εξαγωγής, και τελικά το **αποθήκευση του βιβλίου εργασίας ως CSV**. Καθ' όλη τη διάρκεια θα δούμε επίσης πώς να **εξάγετε xlsx σε CSV** με προσαρμοσμένη μορφοποίηση, και γιατί μπορεί να θέλετε να **μετατρέψετε Excel σε CSV** αντί να χρησιμοποιήσετε το ενσωματωμένο UI του Excel. Ας ξεκινήσουμε—χωρίς περιττές εξηγήσεις, μόνο μια πρακτική λύση που μπορείτε να αντιγράψετε‑επικολλήσετε σήμερα.

## Τι Θα Χρειαστείτε

Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε τα εξής:

- **Aspose.Cells for .NET** (οποιαδήποτε πρόσφατη έκδοση· το API που χρησιμοποιούμε λειτουργεί με 23.x και νεότερες).  
- Ένα περιβάλλον ανάπτυξης .NET (Visual Studio, VS Code, Rider—ό,τι προτιμάτε).  
- Ένα αρχείο Excel (`numbers.xlsx`) που θέλετε να μετατρέψετε σε CSV.  
- Βασική εξοικείωση με τη σύνταξη C#· δεν απαιτούνται προχωρημένα κόλπα.

Αυτό είναι όλο. Αν έχετε ήδη αυτά, είστε έτοιμοι να εξάγετε Excel σε CSV μέσα σε λίγα λεπτά.

## Βήμα 1: Φόρτωση του Βιβλίου Εργασίας Excel

Το πρώτο που πρέπει να κάνετε είναι **να φορτώσετε το βιβλίο εργασίας Excel** στη μνήμη. Το Aspose.Cells το κάνει με μία γραμμή κώδικα, αλλά αξίζει να καταλάβετε γιατί το κάνουμε έτσι: το φόρτωμα σας δίνει πρόσβαση στα φύλλα, τα στυλ, τους τύπους και—το πιο σημαντικό για CSV—τις τιμές των κελιών.

```csharp
using Aspose.Cells;

// Step 1: Load the Excel workbook
Workbook workbook = new Workbook(@"C:\Data\numbers.xlsx");
```

> **Γιατί είναι σημαντικό:**  
> Το *φόρτωμα* του αρχείου μετατρέπει το πακέτο `.xlsx` σε ένα αντικειμενοστραφές μοντέλο που μπορείτε να επεξεργαστείτε προγραμματιστικά. Επίσης επικυρώνει το αρχείο, οπότε θα λάβετε μια σαφή εξαίρεση αν η διαδρομή είναι λανθασμένη ή το αρχείο είναι κατεστραμμένο—κάτι που η UI αγνοεί σιωπηλά.

### Γρήγορη συμβουλή
Αν δουλεύετε με ροή (π.χ. ένα αρχείο που ανέβηκε μέσω API), μπορείτε να αντικαταστήσετε τη διαδρομή αρχείου με ένα `MemoryStream`:

```csharp
using (var stream = new MemoryStream(uploadedBytes))
{
    Workbook workbook = new Workbook(stream);
}
```

Με αυτόν τον τρόπο **φορτώνετε το βιβλίο εργασίας Excel** απευθείας από τη μνήμη, κρατώντας τον κώδικά σας φιλικό στο cloud.

## Βήμα 2: Διαμόρφωση Επιλογών Αποθήκευσης CSV (Προαιρετικό Στρογγυλοποίηση)

Όταν **εξάγετε xlsx σε CSV**, μπορεί να θέλετε να ελέγξετε πώς εμφανίζονται οι αριθμοί. Η κλάση `TxtSaveOptions` σας δίνει λεπτομερή έλεγχο, όπως στρογγυλοποίηση σε συγκεκριμένο αριθμό σημαντικών ψηφίων. Παρακάτω στρογγυλοποιούμε τα πάντα σε τέσσερα σημαντικά ψηφία—μια κοινή απαίτηση για οικονομικές αναφορές.

```csharp
// Step 2: Configure CSV save options to round numbers to 4 significant digits
TxtSaveOptions csvOptions = new TxtSaveOptions(SaveFormat.Csv)
{
    // Keep only 4 significant digits (e.g., 12345 → 1.235E+04)
    SignificantDigits = 4,

    // Optional: Force all numbers to use the invariant culture (dot as decimal separator)
    CultureInfo = System.Globalization.CultureInfo.InvariantCulture
};
```

> **Γιατί μπορεί να το χρειαστείτε:**  
> Ορισμένα downstream συστήματα δυσκολεύονται με υπερβολικά ακριβείς τιμές κινητής υποδιαστολής. Περιορίζοντας τα σε τέσσερα σημαντικά ψηφία μειώνετε το μέγεθος του αρχείου και αποφεύγετε σφάλματα ανάλυσης χωρίς να χάνετε ουσιώδη ακρίβεια.

### Ακραία περίπτωση
Αν το βιβλίο εργασίας σας περιέχει τύπους που επιστρέφουν κείμενο, η ρύθμιση `SignificantDigits` **δεν** τους επηρεάζει. Μόνο τα αριθμητικά κελιά στρογγυλοποιούνται. Αν χρειάζεστε μορφοποίηση ημερομηνιών, χρησιμοποιήστε `CsvSaveOptions` (υποκλάση) για να ορίσετε μια συμβολοσειρά μορφής ημερομηνίας.

## Βήμα 3: Αποθήκευση του Βιβλίου Εργασίας ως CSV

Τώρα που το βιβλίο εργασίας είναι φορτωμένο και οι επιλογές έχουν οριστεί, το τελευταίο βήμα είναι μια κλήση στο `Save`. Εδώ **αποθηκεύουμε το βιβλίο εργασίας ως CSV**.

```csharp
// Step 3: Save the workbook as a CSV file using the configured options
workbook.Save(@"C:\Data\rounded.csv", csvOptions);
```

Αυτό είναι κυριολεκτικά όλο. Μετά το τέλος της κλήσης, θα βρείτε το `rounded.csv` δίπλα στο αρχικό αρχείο, έτοιμο για εισαγωγή από οποιοδήποτε εργαλείο κειμένου.

### Pro tip
Αν χρειάζεστε **μετατροπή Excel σε CSV** για πολλά φύλλα, κάντε βρόχο πάνω από `workbook.Worksheets` και καλέστε `Save` για κάθε φύλλο ξεχωριστά, περνώντας `csvOptions` και ένα όνομα αρχείου ανά φύλλο.

```csharp
foreach (Worksheet sheet in workbook.Worksheets)
{
    string csvPath = $@"C:\Data\{sheet.Name}.csv";
    sheet.Save(csvPath, csvOptions);
}
```

## Βήμα 4: Επαλήθευση του Αποτελέσματος (Προαιρετικό αλλά Συνιστώμενο)

Μια γρήγορη επιβεβαίωση μπορεί να σας εξοικονομήσει ώρες εντοπισμού σφαλμάτων αργότερα. Ανοίξτε το παραγόμενο CSV σε έναν επεξεργαστή κειμένου (Notepad, VS Code) και ελέγξτε:

1. Οι στήλες διαχωρίζονται με κόμματα (ή το διαχωριστικό που ορίσατε στο `CsvSaveOptions`).  
2. Οι αριθμητικές τιμές σέβονται τη στρογγυλοποίηση των τεσσάρων ψηφίων που ρυθμίσατε.  
3. Δεν εμφανίζονται περιττά BOM ή κρυφά χαρακτήρες στην αρχή του αρχείου.

Αν όλα φαίνονται εντάξει, ολοκληρώσατε με επιτυχία την **εξαγωγή xlsx σε CSV** με προσαρμοσμένη στρογγυλοποίηση.

## Πλήρες Παράδειγμα Εργασίας

Παρακάτω υπάρχει ένα αυτόνομο πρόγραμμα που μπορείτε να ενσωματώσετε σε μια εφαρμογή console και να τρέξετε αμέσως. Δείχνει όλη τη ροή—από το φόρτωμα του βιβλίου εργασίας μέχρι την αποθήκευση του CSV.

```csharp
using System;
using System.Globalization;
using Aspose.Cells;

namespace ExcelToCsvDemo
{
    class Program
    {
        static void Main()
        {
            // Path to the source Excel file
            string sourcePath = @"C:\Data\numbers.xlsx";

            // Path where the CSV will be saved
            string csvPath = @"C:\Data\rounded.csv";

            // 1️⃣ Load the Excel workbook
            Workbook workbook = new Workbook(sourcePath);

            // 2️⃣ Configure CSV options (4 significant digits, invariant culture)
            TxtSaveOptions csvOptions = new TxtSaveOptions(SaveFormat.Csv)
            {
                SignificantDigits = 4,
                CultureInfo = CultureInfo.InvariantCulture
            };

            // 3️⃣ Save as CSV
            workbook.Save(csvPath, csvOptions);

            Console.WriteLine($"✅ Successfully saved '{sourcePath}' as CSV to '{csvPath}'.");
        }
    }
}
```

**Αναμενόμενη έξοδος** (στο console):

```
✅ Successfully saved 'C:\Data\numbers.xlsx' as CSV to 'C:\Data\rounded.csv'.
```

Και το παραγόμενο `rounded.csv` θα περιέχει γραμμές όπως:

```
Name,Amount,Date
Alice,1.235E+03,2024-01-15
Bob,9.876E+02,2024-01-16
```

Παρατηρήστε πώς οι αριθμοί στρογγυλοποιούνται στα τέσσερα σημαντικά ψηφία, ακριβώς όπως ζητήσαμε.

## Συχνές Ερωτήσεις & Πιθανά Προβλήματα

| Ερώτηση | Απάντηση |
|----------|--------|
| *Μπορώ να αλλάξω το διαχωριστικό;* | Ναι. Χρησιμοποιήστε `CsvSaveOptions` αντί για `TxtSaveOptions` και ορίστε `Separator` (π.χ. `Separator = ';'`). |
| *Τι γίνεται αν τα φύλλα μου έχουν τύπους που πρέπει να μείνουν ως τύποι;* | Το CSV είναι μορφή απλού κειμένου· οι τύποι πάντα αξιολογούνται στις **τιμές εμφάνισης** πριν την αποθήκευση. |
| *Χρειάζομαι άδεια για το Aspose.Cells;* | Μια δωρεάν αξιολόγηση λειτουργεί, αλλά προσθέτει υδατογράφημα. Για παραγωγή, αποκτήστε άδεια για να αφαιρέσετε το banner και να ξεκλειδώσετε όλες τις δυνατότητες. |
| *Η μετατροπή είναι ασφαλής για Unicode;* | Από προεπιλογή το Aspose γράφει UTF‑8 με BOM. Μπορείτε να αλλάξετε την ιδιότητα `Encoding` στο `CsvSaveOptions` αν χρειάζεστε ANSI ή UTF‑16. |
| *Πώς να διαχειριστώ μεγάλα αρχεία (> 500 MB);* | Χρησιμοποιήστε `LoadOptions` με `MemorySetting = MemorySetting.MemoryOptimized` για να μειώσετε το αποτύπωμα μνήμης κατά το φόρτωμα. |

## Συμβουλές Απόδοσης

- **Επαναχρησιμοποίηση του `TxtSaveOptions`** αν επεξεργάζεστε πολλά αρχεία σε batch· η δημιουργία νέας στιγμής κάθε φορά προσθέτει αμελητέο κόστος, αλλά η επαναχρησιμοποίηση κρατά τον κώδικα πιο καθαρό.  
- **Ροή εξόδου**: Αντί να γράφετε απευθείας στο δίσκο, περάστε ένα `Stream` στο `Save`. Αυτό είναι χρήσιμο για web API που επιστρέφει το CSV ως λήψη.  

```csharp
using (var outStream = new MemoryStream())
{
    workbook.Save(outStream, csvOptions);
    // Return outStream.ToArray() to the client
}
```

- **Παράλληλη επεξεργασία**: Αν έχετε δεκάδες αρχεία Excel, σκεφτείτε το `Parallel.ForEach`. Βεβαιωθείτε ότι κάθε νήμα λαμβάνει τη δική του παρουσία `Workbook`· τα αντικείμενα Aspose **δεν είναι thread‑safe**.

## Επόμενα Βήματα

Τώρα που μπορείτε να **αποθηκεύσετε Excel ως CSV**, ίσως θέλετε να εξερευνήσετε σχετικά θέματα:

- **Εξαγωγή Xlsx σε CSV με προσαρμοσμένα διαχωριστικά** – ιδανικό για ευρωπαϊκές τοπικές ρυθμίσεις που προτιμούν το ερωτηματικό.  
- **Μετατροπή Excel σε CSV σε web service** – δημιουργήστε ένα endpoint που δέχεται ένα ανεβασμένο `.xlsx` και επιστρέφει ένα ρεύμα CSV.  
- **Φόρτωση βιβλίου εργασίας Excel από BLOB βάσης δεδομένων** – συνδυάστε ADO.NET με την τεχνική `MemoryStream` που δείξαμε νωρίτερα.  

Κάθε ένα από αυτά βασίζεται στις βασικές έννοιες που καλύψαμε, ενισχύοντας την ιδέα ότι μόλις μάθετε πώς να **φορτώνετε excel workbook** και **να αποθηκεύετε workbook as csv**, το υπόλοιπο είναι μόνο θέμα ρύθμισης επιλογών.
