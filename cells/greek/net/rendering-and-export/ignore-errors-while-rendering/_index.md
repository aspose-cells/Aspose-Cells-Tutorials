using System;

## Βήμα 1: Ρύθμιση του καταλόγου
Αποφασίστε για τους καταλόγους όπου θα αποθηκευτούν τα αρχεία προέλευσης του Excel και τα PDF εξόδου. Αντικατάσταση `"Your Document Directory"` με την πραγματική διαδρομή στο μηχάνημά σας.
```csharp
// Κατάλογος πηγής
string sourceDir = "C:\\Your\\Path\\Here\\";
// Κατάλογος εξόδου
string outputDir = "C:\\Your\\Path\\Here\\Output\\";
```
Έχοντας όλα τα βασικά στοιχεία στη θέση τους, ας τα δούμε όλα μαζί σε έναν οδηγό βήμα προς βήμα.
## Βήμα 2: Φόρτωση του βιβλίου εργασίας του Excel
Εδώ μπορείτε να πείτε στο Aspose.Cells ποιο αρχείο Excel θέλετε να μετατρέψετε. Αυτό το παράδειγμα υποθέτει ότι χρησιμοποιείτε ένα δείγμα αρχείου με το όνομα `sampleErrorExcel2Pdf.xlsx` που ενδέχεται να έχουν σφάλματα που εμποδίζουν την ομαλή μετατροπή.
```csharp
// Φόρτωση του Δείγματος Βιβλίου Εργασίας που εμφανίζει σφάλμα κατά τη μετατροπή Excel2Pdf
Workbook wb = new Workbook(sourceDir + "sampleErrorExcel2Pdf.xlsx");
```
## Βήμα 3: Ορισμός επιλογών αποθήκευσης PDF
Στη συνέχεια, πρέπει να δημιουργήσουμε ένα `PdfSaveOptions` αντικείμενο. Αυτό το αντικείμενο μας επιτρέπει να καθορίσουμε διαφορετικές ρυθμίσεις, όπως να αγνοήσουμε σφάλματα κατά τη μετατροπή.
```csharp
// Καθορισμός επιλογών αποθήκευσης PDF - Παράβλεψη σφάλματος
PdfSaveOptions opts = new PdfSaveOptions();
opts.IgnoreError = true;  // Αυτό είναι το χρυσό εισιτήριο!
```
## Βήμα 4: Αποθήκευση του βιβλίου εργασίας ως PDF
Τώρα, ήρθε η ώρα να αποθηκεύσουμε το φορτωμένο βιβλίο εργασίας ως αρχείο PDF. Θα χρησιμοποιήσουμε το προηγουμένως διαμορφωμένο `PdfSaveOptions`.
```csharp
// Αποθήκευση του βιβλίου εργασίας σε μορφή PDF με τις επιλογές αποθήκευσης PDF
wb.Save(outputDir + "outputErrorExcel2Pdf.pdf", opts);
```
## Βήμα 5: Επιβεβαίωση επιτυχίας
Για να ενημερώσουμε τον χρήστη ότι όλα λειτούργησαν σωστά, ας εκτυπώσουμε μια απλή επιβεβαίωση στην κονσόλα.
```csharp
Console.WriteLine("IgnoreErrorsWhileRenderingExcelToPdf executed successfully.\r\n");
```
