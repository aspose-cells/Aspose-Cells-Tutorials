# Έλεγχος εξωτερικών πόρων από το Excel σε PDF στο Aspose.Cells

## Εισαγωγή
Στη σημερινή ψηφιακή εποχή, η μετατροπή υπολογιστικών φύλλων Excel σε έγγραφα PDF είναι μια συνηθισμένη εργασία. Είτε πρόκειται για την προετοιμασία αναφορών, οικονομικών δεδομένων ή υλικού παρουσίασης, θέλετε να διασφαλίσετε ότι τα PDF σας θα φαίνονται ακριβώς όπως τα θέλετε. Το Aspose.Cells για .NET είναι μια ισχυρή βιβλιοθήκη που σας επιτρέπει να ελέγχετε αυτήν τη διαδικασία μετατροπής μέχρι την τελευταία λεπτομέρεια, ειδικά όταν χειρίζεστε εξωτερικούς πόρους όπως εικόνες που συνοδεύουν τα αρχεία Excel σας. Σε αυτόν τον οδηγό, εμβαθύνουμε στον τρόπο ελέγχου των εξωτερικών πόρων κατά τη διάρκεια της διαδικασίας μετατροπής από Excel σε PDF χρησιμοποιώντας το Aspose.Cells. Πάρτε λοιπόν το αγαπημένο σας ρόφημα και ας ξεκινήσουμε!
## Προαπαιτούμενα
Πριν μπούμε στα πιο σημαντικά, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ξεκινήσετε. Ακολουθεί μια γρήγορη λίστα ελέγχου:
1. Visual Studio ή οποιοδήποτε IDE συμβατό με .NET: Θα χρειαστείτε ένα περιβάλλον για να γράψετε και να δοκιμάσετε τον κώδικά σας.
2. Aspose.Cells για .NET: Εάν δεν το έχετε εγκαταστήσει ακόμα, μεταβείτε στο [Λήψεις Aspose](https://releases.aspose.com/cells/net/) σελίδα και κατεβάστε την πιο πρόσφατη έκδοση.
3. Βασικές γνώσεις C#: Η εξοικείωση με τη γλώσσα προγραμματισμού C# θα είναι χρήσιμη. Εάν δεν είστε σίγουροι για κάποια έννοια, μη διστάσετε να την αναζητήσετε.
4. Ένα δείγμα αρχείου Excel: Προετοιμάστε ένα αρχείο Excel με τυχόν εξωτερικούς πόρους που θέλετε να μετατρέψετε. Μπορείτε να χρησιμοποιήσετε το παρεχόμενο δείγμα αρχείου "samplePdfSaveOptions_StreamProvider.xlsx".
5. Ένα αρχείο εικόνας για δοκιμή: Αυτό θα χρησιμοποιηθεί ως εξωτερικός πόρος κατά τη μετατροπή. Το αρχείο εικόνας "newPdfSaveOptions_StreamProvider.png" είναι ένα καλό σύμβολο κράτησης θέσης.
## Εισαγωγή πακέτων
Για να ξεκινήσετε, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων από τη βιβλιοθήκη Aspose.Cells. Αυτό είναι κρίσιμο για την πρόσβαση στις λειτουργίες της. Βεβαιωθείτε ότι έχετε προσθέσει τα ακόλουθα χρησιμοποιώντας οδηγίες στην αρχή του αρχείου σας:
```csharp
using System.IO;
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using Aspose.Cells.Rendering;
using System;
```
Αυτά τα πακέτα θα παρέχουν όλες τις απαραίτητες κλάσεις και μεθόδους που θα χρειαστείτε για να εκτελέσετε τις εργασίες σας.
## Βήμα 1: Δημιουργήστε την κλάση παροχής ροής σας
Η πρώτη σειρά εργασιών είναι η δημιουργία μιας κλάσης παρόχου ροής που υλοποιεί το `IStreamProvider` διεπαφή. Αυτή η κλάση θα σας επιτρέψει να ελέγχετε τον τρόπο φόρτωσης των εξωτερικών πόρων.
```csharp
class MyStreamProvider : IStreamProvider
{
    public void CloseStream(StreamProviderOptions options)
    {
        Debug.WriteLine("-----Close Stream-----");
    }
    public void InitStream(StreamProviderOptions options)
    {
        string sourceDir = "Your Document Directory";
        Debug.WriteLine("-----Init Stream-----");
        // Διαβάστε τη νέα εικόνα σε μια ροή μνήμης και αντιστοιχίστε την στην ιδιότητα Stream
        byte[] bts = File.ReadAllBytes(sourceDir + "newPdfSaveOptions_StreamProvider.png");
        MemoryStream ms = new MemoryStream(bts);
        options.Stream = ms;
    }
}
```
Σε αυτή την τάξη:
- CloseStream: Αυτή η μέθοδος θα κληθεί όταν η ροή κλείσει. Προς το παρόν, απλώς γράφουμε ένα μήνυμα εντοπισμού σφαλμάτων για την παρακολούθηση.
- InitStream: Εδώ ξεκινά η μαγεία. Εδώ, θα διαβάσετε την εξωτερική σας εικόνα ως πίνακα byte, θα την μετατρέψετε σε ροή μνήμης και θα την αντιστοιχίσετε στο `options.Stream` ιδιοκτησία.
## Βήμα 2: Ρύθμιση καταλόγων προέλευσης και εξόδου
Τώρα που ο πάροχος ροής σας είναι έτοιμος, ήρθε η ώρα να προσδιορίσετε πού βρίσκεται το αρχείο Excel και πού θέλετε να αποθηκεύσετε το PDF σας.
```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory";
// Κατάλογος εξόδου
string outputDir = "Your Document Directory";
```
Απλώς αντικαταστήστε `"Your Document Directory"` με την πραγματική διαδρομή στον υπολογιστή σας όπου βρίσκονται τα αρχεία σας. Η οργάνωση των αρχείων σας είναι το κλειδί!
## Βήμα 3: Φόρτωση του αρχείου Excel
Στη συνέχεια, θα φορτώσετε το αρχείο Excel από το οποίο θέλετε να δημιουργήσετε το PDF.
```csharp
// Φόρτωση αρχείου Excel πηγαίου κώδικα που περιέχει εξωτερικές εικόνες
Workbook wb = new Workbook(sourceDir + "samplePdfSaveOptions_StreamProvider.xlsx");
```
Χρησιμοποιούμε το `Workbook` κλάση από το Aspose.Cells, το οποίο αντιπροσωπεύει το αρχείο Excel σας. Το αρχείο μπορεί να περιλαμβάνει διάφορους εξωτερικούς πόρους, όπως εικόνες, τις οποίες θέλετε να ελέγξετε κατά τη μετατροπή.
## Βήμα 4: Ορισμός επιλογών αποθήκευσης PDF
Πριν αποθηκεύσετε το βιβλίο εργασίας ως PDF, ας καθορίσουμε πώς θέλετε να αποθηκευτεί. Μπορείτε να προσαρμόσετε αυτές τις επιλογές σύμφωνα με τις απαιτήσεις σας.
```csharp
// Καθορισμός επιλογών αποθήκευσης PDF - Πάροχος ροής
PdfSaveOptions opts = new PdfSaveOptions();
opts.OnePagePerSheet = true; // Αποθήκευση κάθε φύλλου σε νέα σελίδα
```
Εδώ, δημιουργούμε μια νέα παρουσία του `PdfSaveOptions`το οποίο σας επιτρέπει να προσαρμόσετε τον τρόπο μορφοποίησης του PDF σας. `OnePagePerSheet` Η επιλογή είναι χρήσιμη για να διασφαλίσετε ότι κάθε φύλλο Excel θα έχει τη δική του σελίδα στο τελικό PDF.
## Βήμα 5: Ορίστε τον πάροχο ροής σας
Αφού ορίσετε τις επιλογές PDF, πρέπει να ενημερώσετε το Aspose ώστε να χρησιμοποιεί τον προσαρμοσμένο πάροχο ροής σας για εξωτερικούς πόρους.
```csharp
wb.Settings.StreamProvider = new MyStreamProvider();
```
Αυτή η γραμμή σας συνδέει `Workbook` παράδειγμα με το `MyStreamProvider` κλάση που δημιουργήσατε νωρίτερα. Αυτό σημαίνει ότι κάθε φορά που συναντώνται εξωτερικοί πόροι κατά τη μετατροπή, ο πάροχός σας θα τους χειρίζεται όπως καθορίζεται.
## Βήμα 6: Αποθήκευση του βιβλίου εργασίας ως PDF
Αφού έχετε όλα τα απαραίτητα, ήρθε επιτέλους η ώρα να αποθηκεύσετε το βιβλίο εργασίας του Excel ως PDF.
```csharp
// Αποθήκευση του βιβλίου εργασίας σε μορφή Pdf
wb.Save(outputDir + "outputPdfSaveOptions_StreamProvider.pdf", opts);
```
Καλώντας το `Save` τη μέθοδο στο αντικείμενο του βιβλίου εργασίας και μεταβιβάζοντας στον κατάλογο εξόδου σας μαζί με τις επιλογές PDF, μετατρέπετε το αρχείο Excel σε ένα όμορφα μορφοποιημένο PDF.
