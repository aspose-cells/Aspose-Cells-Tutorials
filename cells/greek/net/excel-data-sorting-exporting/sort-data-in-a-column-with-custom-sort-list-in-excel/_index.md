# Ταξινόμηση δεδομένων σε στήλη με λίστα προσαρμοσμένης ταξινόμησης στο Excel

## Εισαγωγή

Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία ρύθμισης του έργου σας, φόρτωσης ενός αρχείου Excel και ταξινόμησης δεδομένων εντός ενός καθορισμένου εύρους χρησιμοποιώντας μια προσαρμοσμένη σειρά ταξινόμησης. Ακολουθώντας αυτόν τον οδηγό, θα αποκτήσετε πρακτική εμπειρία που μπορεί να βελτιώσει τις δεξιότητες διαχείρισης δεδομένων και τη χρηστικότητα της βιβλιοθήκης Aspose.Cells.

## Προαπαιτούμενα

Πριν βουτήξουμε στο σεμινάριο, ας περιγράψουμε ορισμένες προϋποθέσεις για να εξασφαλίσουμε μια ομαλή εμπειρία εκμάθησης.

### Βασικές γνώσεις C#

Ενώ το σεμινάριο έχει σχεδιαστεί για να σας καθοδηγεί σε κάθε βήμα, η βασική κατανόηση της C# θα διευκολύνει την κατανόηση των εννοιών που παρουσιάζονται.

### .NET Αναπτυξιακό Περιβάλλον

Βεβαιωθείτε ότι έχετε ρυθμίσει ένα λειτουργικό περιβάλλον ανάπτυξης .NET. Μπορείτε να χρησιμοποιήσετε το Visual Studio ή οποιοδήποτε άλλο IDE που υποστηρίζει την ανάπτυξη .NET.

### Aspose.Cells για το πακέτο .NET NuGet

Χρειάζεστε τη βιβλιοθήκη Aspose.Cells για .NET εγκατεστημένη στο έργο σας. Μπορείτε εύκολα να το προσθέσετε μέσω του NuGet Package Manager. 

Δείτε πώς να το κάνετε:

1. Ανοίξτε το έργο σας στο Visual Studio.
2. Μεταβείτε στα "Εργαλεία" > "Διαχείριση πακέτων NuGet" > "Διαχείριση πακέτων NuGet για λύση".
3.  Αναζήτηση για`Aspose.Cells` και εγκαταστήστε την πιο πρόσφατη έκδοση.

### Βασικό αρχείο Excel για δοκιμή

Θα χρειαστείτε ένα δείγμα αρχείου Excel για να εργαστείτε. Μπορείτε να δημιουργήσετε ένα απλό αρχείο Excel με τυχαία ονόματα χωρών και τους κωδικούς τους.

## Εισαγωγή πακέτων

Για να ξεκινήσετε, ας εισάγουμε τα απαραίτητα πακέτα στο έργο σας. Ακολουθεί ένα απόσπασμα του τρόπου ρύθμισης του κώδικά σας:

```csharp
using System;
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

Με τα πακέτα που εισάγονται, είμαστε έτοιμοι να προχωρήσουμε.

## Βήμα 1: Ορίστε τους καταλόγους προέλευσης και εξόδου 

Το πρώτο βήμα είναι να ορίσετε πού βρίσκεται το αρχείο εισόδου και πού θέλετε να αποθηκευτεί η έξοδος (ταξινομημένο αρχείο). Πρέπει να καθορίσετε δύο διαδρομές: μια για το αρχείο προέλευσης Excel και μια άλλη για την αποθήκευση της εξόδου μετά την ταξινόμηση.

```csharp
string sourceDir = "Your Document Directory\\";
string outputDir = "Your Document Directory\\";
```

## Βήμα 2: Φορτώστε το αρχείο προέλευσης Excel

Στη συνέχεια, θα φορτώσουμε το αρχείο Excel που περιέχει τα δεδομένα που θέλετε να ταξινομήσετε. Αυτό γίνεται με τη δημιουργία ενός στιγμιότυπου του`Workbook` τάξη και περνώντας τη διαδρομή του αρχείου προέλευσης.

```csharp
Workbook wb = new Workbook(sourceDir + "sampleSortData_CustomSortList.xlsx");
```

## Βήμα 3: Πρόσβαση στο Πρώτο φύλλο εργασίας 

Μόλις φορτωθεί το αρχείο, πρέπει να αποκτήσουμε πρόσβαση στο συγκεκριμένο φύλλο εργασίας που περιέχει τα δεδομένα που σκοπεύουμε να ταξινομήσουμε. Σε αυτήν την περίπτωση, στοχεύουμε το πρώτο φύλλο εργασίας.

```csharp
Worksheet ws = wb.Worksheets[0];
```

## Βήμα 4: Καθορίστε την περιοχή κελιών προς ταξινόμηση

 Πρέπει να προσδιορίσουμε το εύρος των κελιών που θα ταξινομήσουμε. Σε αυτό το παράδειγμα, θα ταξινομήσουμε τα κελιά από A1 σε A40. Χρησιμοποιήστε το`CellArea.CreateCellArea` μέθοδος καθορισμού της περιοχής κελιού.

```csharp
CellArea ca = CellArea.CreateCellArea("A1", "A40");
```

## Βήμα 5: Δημιουργήστε μια λίστα προσαρμοσμένης ταξινόμησης

Πριν από την ταξινόμηση, πρέπει να καθορίσουμε τα κριτήρια που θα χρησιμοποιήσουμε για την προσαρμοσμένη ταξινόμηση. Μπορείτε να ορίσετε μια λίστα ταξινόμησης ως μια σειρά από συμβολοσειρές. Η λίστα προσαρμοσμένης ταξινόμησης θα υπαγορεύσει τη σειρά ταξινόμησης.

```csharp
string[] customSortList = new string[] { "USA,US", "Brazil,BR", "China,CN", "Russia,RU", "Canada,CA" };
```

## Βήμα 6: Προσθέστε κλειδί ταξινόμησης και εκτελέστε την ταξινόμηση

Τώρα είναι ώρα για τακτοποίηση! Θα χρησιμοποιήσουμε την κλάση DataSorter για αυτό. Δημιουργήστε ένα κλειδί για ταξινόμηση με βάση την προσαρμοσμένη λίστα μας και εκτελέστε τη λειτουργία ταξινόμησης.

```csharp
wb.DataSorter.AddKey(0, SortOrder.Ascending, customSortList);
wb.DataSorter.Sort(ws.Cells, ca);
```

## Βήμα 7: Αποθηκεύστε το αρχείο εξόδου Excel

Αφού ολοκληρωθεί η ταξινόμηση, το τελευταίο βήμα είναι να αποθηκεύσετε τις αλλαγές σε ένα νέο αρχείο Excel. Καθορίστε το όνομα του αρχείου εξόδου και αποθηκεύστε το βιβλίο εργασίας.

```csharp
wb.Save(outputDir + "outputSortData_CustomSortList.xlsx");
```

## Βήμα 8: Επιβεβαιώστε την επιτυχή εκτέλεση

Για να βεβαιωθείτε ότι όλα έχουν λειτουργήσει ομαλά, μπορείτε να εκτυπώσετε ένα μήνυμα επιβεβαίωσης στην κονσόλα. Αυτό βοηθά στον εντοπισμό σφαλμάτων και σας δίνει την ικανοποίηση ότι η λειτουργία ήταν επιτυχής.

```csharp
Console.WriteLine("SortDataInColumnWithCustomSortList executed successfully.\r\n");
```
