# Ταξινόμηση δεδομένων σε στήλη με προσαρμοσμένη λίστα ταξινόμησης στο Excel

## Εισαγωγή

Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία ρύθμισης του έργου σας, φόρτωσης ενός αρχείου Excel και ταξινόμησης δεδομένων εντός ενός καθορισμένου εύρους χρησιμοποιώντας μια προσαρμοσμένη σειρά ταξινόμησης. Ακολουθώντας αυτόν τον οδηγό, θα αποκτήσετε πρακτική εμπειρία που μπορεί να βελτιώσει τις δεξιότητές σας στη διαχείριση δεδομένων και τη χρηστικότητα της βιβλιοθήκης Aspose.Cells.

## Προαπαιτούμενα

Πριν ξεκινήσουμε το σεμινάριο, ας περιγράψουμε ορισμένες προϋποθέσεις για να διασφαλίσουμε μια ομαλή μαθησιακή εμπειρία.

### Βασικές γνώσεις C#

Ενώ το σεμινάριο έχει σχεδιαστεί για να σας καθοδηγήσει σε κάθε βήμα, η βασική κατανόηση της C# θα σας διευκολύνει να κατανοήσετε τις έννοιες που παρουσιάζονται.

### Περιβάλλον ανάπτυξης .NET

Βεβαιωθείτε ότι έχετε ρυθμίσει ένα λειτουργικό περιβάλλον ανάπτυξης .NET. Μπορείτε να χρησιμοποιήσετε το Visual Studio ή οποιοδήποτε άλλο IDE που υποστηρίζει ανάπτυξη .NET.

### Aspose.Cells για το πακέτο .NET NuGet

Χρειάζεστε την βιβλιοθήκη Aspose.Cells για .NET εγκατεστημένη στο έργο σας. Μπορείτε εύκολα να την προσθέσετε μέσω του NuGet Package Manager. 

Δείτε πώς μπορείτε να το κάνετε:

1. Ανοίξτε το έργο σας στο Visual Studio.
2. Μεταβείτε στα "Εργαλεία" > "Διαχειριστής πακέτων NuGet" > "Διαχείριση πακέτων NuGet για λύση".
3. Αναζήτηση για `Aspose.Cells` και εγκαταστήστε την πιο πρόσφατη έκδοση.

### Βασικό αρχείο Excel για δοκιμές

Θα χρειαστείτε ένα δείγμα αρχείου Excel για να εργαστείτε. Μπορείτε να δημιουργήσετε ένα απλό αρχείο Excel με τυχαία ονόματα χωρών και τους κωδικούς τους.

## Εισαγωγή πακέτων

Για να ξεκινήσουμε, ας εισαγάγουμε τα απαραίτητα πακέτα στο έργο σας. Ακολουθεί ένα απόσπασμα για το πώς να ρυθμίσετε τον κώδικά σας:

```csharp
using System;
using System.IO;
using Aspose.Cells;
using System.Drawing;
```

Με τα πακέτα που έχουν εισαχθεί, είμαστε έτοιμοι να προχωρήσουμε.

## Βήμα 1: Ορίστε τους καταλόγους προέλευσης και εξόδου 

Το πρώτο βήμα είναι να ορίσετε πού βρίσκεται το αρχείο εισόδου σας και πού θέλετε να αποθηκευτεί το αποτέλεσμα (ταξινομημένο αρχείο). Πρέπει να καθορίσετε δύο διαδρομές: μία για το αρχείο προέλευσης Excel και μια άλλη για την αποθήκευση του αποτελέσματος μετά την ταξινόμηση.

```csharp
string sourceDir = "Your Document Directory\\";
string outputDir = "Your Document Directory\\";
```

## Βήμα 2: Φόρτωση του αρχείου Excel προέλευσης

Στη συνέχεια, θα φορτώσουμε το αρχείο Excel που περιέχει τα δεδομένα που θέλετε να ταξινομήσετε. Αυτό γίνεται δημιουργώντας μια παρουσία του `Workbook` κλάση και περνώντας τη διαδρομή του αρχείου πηγαίου κώδικα.

```csharp
Workbook wb = new Workbook(sourceDir + "sampleSortData_CustomSortList.xlsx");
```

## Βήμα 3: Πρόσβαση στο πρώτο φύλλο εργασίας 

Μόλις φορτωθεί το αρχείο, πρέπει να έχουμε πρόσβαση στο συγκεκριμένο φύλλο εργασίας που περιέχει τα δεδομένα που σκοπεύουμε να ταξινομήσουμε. Σε αυτήν την περίπτωση, στοχεύουμε στο πρώτο φύλλο εργασίας.

```csharp
Worksheet ws = wb.Worksheets[0];
```

## Βήμα 4: Καθορίστε την περιοχή κελιών για ταξινόμηση

Πρέπει να προσδιορίσουμε το εύρος των κελιών που θα ταξινομήσουμε. Σε αυτό το παράδειγμα, θα ταξινομήσουμε τα κελιά από A1 έως A40. Χρησιμοποιήστε το `CellArea.CreateCellArea` μέθοδος για τον ορισμό της περιοχής του κελιού.

```csharp
CellArea ca = CellArea.CreateCellArea("A1", "A40");
```

## Βήμα 5: Δημιουργήστε μια προσαρμοσμένη λίστα ταξινόμησης

Πριν από την ταξινόμηση, πρέπει να καθορίσουμε τα κριτήρια που θα χρησιμοποιήσουμε για την προσαρμοσμένη ταξινόμηση. Μπορείτε να ορίσετε μια λίστα ταξινόμησης ως έναν πίνακα συμβολοσειρών. Η προσαρμοσμένη λίστα ταξινόμησης θα υπαγορεύσει τη σειρά ταξινόμησης.

```csharp
string[] customSortList = new string[] { "USA,US", "Brazil,BR", "China,CN", "Russia,RU", "Canada,CA" };
```

## Βήμα 6: Προσθήκη κλειδιού ταξινόμησης και εκτέλεση της ταξινόμησης

Τώρα είναι η ώρα για ταξινόμηση! Θα χρησιμοποιήσουμε την κλάση DataSorter για αυτό. Δημιουργήστε ένα κλειδί για ταξινόμηση με βάση την προσαρμοσμένη λίστα μας και εκτελέστε τη λειτουργία ταξινόμησης.

```csharp
wb.DataSorter.AddKey(0, SortOrder.Ascending, customSortList);
wb.DataSorter.Sort(ws.Cells, ca);
```

## Βήμα 7: Αποθήκευση του αρχείου εξόδου Excel

Αφού ολοκληρωθεί η ταξινόμηση, το τελευταίο βήμα είναι να αποθηκεύσετε τις αλλαγές σε ένα νέο αρχείο Excel. Καθορίστε το όνομα του αρχείου εξόδου και αποθηκεύστε το βιβλίο εργασίας.

```csharp
wb.Save(outputDir + "outputSortData_CustomSortList.xlsx");
```

## Βήμα 8: Επιβεβαίωση επιτυχούς εκτέλεσης

Για να βεβαιωθείτε ότι όλα έχουν λειτουργήσει ομαλά, μπορείτε να εκτυπώσετε ένα μήνυμα επιβεβαίωσης στην κονσόλα. Αυτό βοηθά στην αποσφαλμάτωση και σας δίνει την επιβεβαίωση ότι η λειτουργία ήταν επιτυχής.

```csharp
Console.WriteLine("SortDataInColumnWithCustomSortList executed successfully.\r\n");
```
