## Οδηγός Εφαρμογής
Τώρα, ας δούμε κάθε χαρακτηριστικό βήμα προς βήμα.

### Δημιουργία βιβλίου εργασίας και φύλλου εργασίας
**Επισκόπηση**Αυτή η ενότητα καλύπτει τον τρόπο δημιουργίας ενός βιβλίου εργασίας και προσθήκης ενός φύλλου εργασίας.
1. **Αρχικοποίηση του βιβλίου εργασίας**
   - Ξεκινήστε δημιουργώντας ένα κενό `Workbook` αντικείμενο.
   ```csharp
   Workbook workbook = new Workbook();
   ```
2. **Προσθήκη νέου φύλλου εργασίας**
   - Χρησιμοποιήστε το `Worksheets.Add()` μέθοδος για την προσθήκη ενός νέου φύλλου.
   ```csharp
   int sheetIndex = workbook.Worksheets.Add();
   Worksheet worksheet = workbook.Worksheets[sheetIndex];
   // Το βιβλίο εργασίας περιέχει πλέον ένα φύλλο εργασίας.
   ```

### Προσθήκη σχολίου σε ένα κελί
**Επισκόπηση**: Μάθετε πώς να εισάγετε σχόλια σε συγκεκριμένα κελιά.
1. **Προσθήκη σχολίου**
   - Χρησιμοποιήστε το `Comments.Add()` μέθοδος για την τοποθέτηση ενός σχολίου στο κελί "F5".
   ```csharp
   int commentIndex = worksheet.Comments.Add("F5");
   Comment comment = worksheet.Comments[commentIndex];
   ```
2. **Ορισμός της σημείωσης σχολίου**
   - Αντιστοιχίστε κείμενο στο σχόλιό σας χρησιμοποιώντας το `Note` ιδιοκτησία.
   ```csharp
   comment.Note = "Hello Aspose!";
   ```

### Μορφοποίηση εμφάνισης σχολίου
**Επισκόπηση**: Προσαρμόστε την εμφάνιση των σχολίων για καλύτερη αναγνωσιμότητα.
1. **Προσαρμογή μεγέθους και στυλ γραμματοσειράς**
   - Αλλάξτε το μέγεθος της γραμματοσειράς και εφαρμόστε έντονη γραφή.
   ```csharp
   comment.Font.Size = 14;
   comment.Font.IsBold = true;
   ```
2. **Ορισμός διαστάσεων σε εκατοστά**
   - Καθορίστε ύψος και πλάτος για να ελέγξετε τον οπτικό χώρο.
   ```csharp
   comment.HeightCM = 10;
   comment.WidthCM = 2;
   ```

### Αποθήκευση του βιβλίου εργασίας
**Επισκόπηση**: Διατηρήστε τις αλλαγές σας αποθηκεύοντας το βιβλίο εργασίας.
1. **Αποθήκευση αλλαγών**
   - Χρήση `Workbook.Save()` μέθοδος για την εγγραφή αλλαγών σε ένα αρχείο.
   ```csharp
   workbook.Save(outputDir + "book1.out.xls");
   ```
