## Οδηγός Εφαρμογής

### Επισκόπηση της κατάργησης σχολίων σε νήματα

Αυτή η ενότητα εξηγεί πώς να καταργήσετε σχόλια με νήματα από βιβλία εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το αρχείο του βιβλίου εργασίας σας:

```csharp
string sourceDir = "path_to_your_directory";
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```

**Γιατί αυτό έχει σημασία:** Η φόρτωση του βιβλίου εργασίας είναι απαραίτητη για την πρόσβαση και τον χειρισμό των περιεχομένων του.

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας

Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας που περιέχει τα σχόλιά σας:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
CommentCollection comments = worksheet.Comments;
```

**Εξήγηση:** Η στόχευση ενός συγκεκριμένου φύλλου εργασίας επιτρέπει την αποτελεσματική διαχείριση των σχολίων του.

#### Βήμα 3: Κατάργηση σχολίων σε νήματα

Αφαίρεση σχολίων από ένα καθορισμένο κελί, όπως "A1":

```csharp
// Λήψη Συγγραφέα του πρώτου σχολίου στο A1 (προαιρετικό βήμα εάν θέλετε να διαχειριστείτε τους συγγραφείς)
ThreadedCommentAuthor author = worksheet.Comments.GetThreadedComments("A1")[0].Author;

// Κατάργηση σχολίου στο A1
comments.RemoveAt("A1");

// Προαιρετικά, αφαιρέστε και τον συγγραφέα
ThreadedCommentAuthorCollection authors = workbook.Worksheets.ThreadedCommentAuthors;
authors.RemoveAt(authors.IndexOf(author));
```

**Βασική Επισκόπηση:** `RemoveAt` αφαιρεί αποτελεσματικά τα σχόλια από τις αναφορές κελιών τους.

#### Βήμα 4: Αποθήκευση του βιβλίου εργασίας

Τέλος, αποθηκεύστε το τροποποιημένο βιβλίο εργασίας σας:

```csharp
string outDir = "output_directory_path";
workbook.Save(outDir + "ThreadedCommentsSample_Out.xlsx");
```

**Σκοπός:** Η αποθήκευση διασφαλίζει ότι όλες οι αλλαγές διατηρούνται σε ένα νέο ή υπάρχον αρχείο.
