## Οδηγός Εφαρμογής

### Επεξεργασία σχολίων σε νήματα σε φύλλο εργασίας Excel

Αυτή η λειτουργία εστιάζει στην πρόσβαση και την τροποποίηση σχολίων με νήματα μέσα σε ένα συγκεκριμένο κελί ενός φύλλου εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το υπάρχον αρχείο Excel. Αυτό γίνεται χρησιμοποιώντας το `Workbook` κλάση, η οποία αντιπροσωπεύει ένα ολόκληρο βιβλίο εργασίας του Excel:

```csharp
// Ορισμός διαδρομών για τους καταλόγους προέλευσης και εξόδου
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του βιβλίου εργασίας από έναν καθορισμένο κατάλογο
Workbook workbook = new Workbook(SourceDir + "ThreadedCommentsSample.xlsx");
```

#### Βήμα 2: Πρόσβαση σε σχόλια με νήματα

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και ανακτήστε σχόλια σε νήματα για ένα συγκεκριμένο κελί, όπως π.χ. `A1`Μπορείτε να στοχεύσετε οποιοδήποτε κελί αλλάζοντας την αναφορά του:

```csharp
// Λήψη του πρώτου φύλλου εργασίας από το βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];

// Ανάκτηση όλων των σχολίων σε νήματα για το κελί A1
ThreadedComment comment = worksheet.Comments.GetThreadedComments("A1")[0];
```

#### Βήμα 3: Ενημέρωση του σχολίου

Μόλις αποκτήσετε πρόσβαση σε ένα συγκεκριμένο σχόλιο σε νήμα, ενημερώστε το περιεχόμενό του όπως απαιτείται:

```csharp
// Τροποποίηση της σημείωσης του σχολίου σε νήματα
comment.Notes = "Updated Comment";
```

#### Βήμα 4: Αποθήκευση αλλαγών

Αφού κάνετε τις ενημερώσεις σας, αποθηκεύστε το βιβλίο εργασίας για να διατηρηθούν οι αλλαγές. Μπορείτε να καθορίσετε ένα νέο όνομα αρχείου ή να αντικαταστήσετε το αρχικό αρχείο:

```csharp
// Αποθήκευση του ενημερωμένου βιβλίου εργασίας με νέο όνομα αρχείου
workbook.Save(OutputDir + "EditThreadedComments.xlsx");
```

### Φόρτωση και αποθήκευση ενός βιβλίου εργασίας Excel

Αυτή η λειτουργία είναι μια γρήγορη επίδειξη του τρόπου φόρτωσης ενός υπάρχοντος αρχείου Excel, εκτέλεσης λειτουργιών και αποθήκευσής του.

#### Βήμα 1: Φόρτωση ενός υπάρχοντος βιβλίου εργασίας

```csharp
// Καθορισμός καταλόγων για τη φόρτωση και την αποθήκευση βιβλίων εργασίας
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
```

```csharp
// Αποθήκευση του τροποποιημένου βιβλίου εργασίας σε νέο αρχείο
```
