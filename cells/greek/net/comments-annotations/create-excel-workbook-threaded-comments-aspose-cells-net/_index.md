## Οδηγός Εφαρμογής

### Δημιουργία και αποθήκευση βιβλίου εργασίας με σχόλια σε νήματα

#### Επισκόπηση
Σε αυτήν την ενότητα, θα δημιουργήσουμε ένα βιβλίο εργασίας του Excel και θα προσθέσουμε σχόλια με νήματα χρησιμοποιώντας το Aspose.Cells για .NET.

#### Οδηγίες βήμα προς βήμα
**1. Αρχικοποίηση του Βιβλίου Εργασίας**
Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Workbook`:
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργία νέας παρουσίας βιβλίου εργασίας
Workbook workbook = new Workbook();
```

**2. Προσθέστε έναν συγγραφέα για σχόλια σε νήματα**
Ορίστε και προσθέστε έναν συντάκτη σχολίων στο `ThreadedCommentAuthors` συλλογή:
```csharp
// Προσθήκη συντάκτη για το σχόλιο σε νήμα
int authorIndex = workbook.Worksheets.ThreadedCommentAuthors.Add("Aspose Test", "", "");
ThreadedCommentAuthor author = workbook.Worksheets.ThreadedCommentAuthors[authorIndex];
```

**3. Εισαγωγή σχολίου σε νήματα**
Προσθήκη σχολίου με νήματα στο κελί A1 στο πρώτο φύλλο εργασίας:
```csharp
// Προσθήκη σχολίου με νήματα στο κελί A1 στο πρώτο φύλλο εργασίας
workbook.Worksheets[0].Comments.AddThreadedComment("A1", "Test Threaded Comment", author);
```

**4. Αποθήκευση του βιβλίου εργασίας**
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας στον επιθυμητό κατάλογο εξόδου:
```csharp
// Αποθήκευση του βιβλίου εργασίας σε έναν κατάλογο εξόδου
workbook.Save(outputDir + "/AddThreadedComments_out.xlsx");
```
