## Οδηγός Υλοποίησης: Ορισμός Μεταδεδομένων Βιβλίου Εργασίας

Ας χωρίσουμε τη διαδικασία σε διαχειρίσιμα βήματα.

### 1. Αρχικοποίηση βιβλίου εργασίας και ορισμός επιλογών μεταδεδομένων

Αρχικά, πρέπει να καθορίσετε με ποιες ιδιότητες μεταδεδομένων θέλετε να εργαστείτε. Σε αυτό το παράδειγμα, θα επικεντρωθούμε στις ιδιότητες του εγγράφου:

```csharp
using Aspose.Cells;
using Aspose.Cells.Metadata;

// Ορισμός καταλόγων για αρχεία προέλευσης και εξόδου
string sourceDir = "path_to_source_directory";
string outputDir = "path_to_output_directory";

// Αρχικοποίηση επιλογών μεταδεδομένων
MetadataOptions options = new MetadataOptions(MetadataType.DocumentProperties);

// Φόρτωση του βιβλίου εργασίας με καθορισμένες επιλογές μεταδεδομένων
WorkbookMetadata meta = new WorkbookMetadata(sourceDir + "sampleUsingWorkbookMetadata.xlsx", options);
```

### 2. Προσθήκη ιδιοτήτων προσαρμοσμένου εγγράφου

Οι προσαρμοσμένες ιδιότητες είναι χρήσιμες για την προσθήκη συγκεκριμένων πληροφοριών που σχετίζονται με τον οργανισμό ή το έργο σας:

```csharp
// Προσθήκη προσαρμοσμένης ιδιότητας εγγράφου
meta.CustomDocumentProperties.Add("MyTest", "This is My Test");
```

**Γιατί αυτό έχει σημασία:** Ορίζοντας προσαρμοσμένα μεταδεδομένα, μπορείτε να παρακολουθείτε επιπλέον περιβάλλοντα σχετικά με το περιεχόμενο του βιβλίου εργασίας, όπως λεπτομέρειες συντάκτη, διαχείριση εκδόσεων και πολλά άλλα.

### 3. Αποθήκευση ενημερωμένων μεταδεδομένων

Αφού ορίσετε τις ιδιότητές σας, αποθηκεύστε τις για να διασφαλίσετε ότι οι αλλαγές θα διατηρηθούν:

```csharp
// Αποθηκεύστε τα ενημερωμένα μεταδεδομένα σε ένα νέο αρχείο
meta.Save(outputDir + "outputUsingWorkbookMetadata.xlsx");
```

### 4. Ανάγνωση και εμφάνιση μεταδεδομένων

Για να επαληθεύσετε τις αλλαγές σας, ανοίξτε το βιβλίο εργασίας και διαβάστε την προσαρμοσμένη ιδιότητα:

```csharp
// Άνοιγμα του βιβλίου εργασίας με ενημερωμένα μεταδεδομένα
Workbook w = new Workbook(outputDir + "outputUsingWorkbookMetadata.xlsx");

// Εμφάνιση της ιδιότητας προσαρμοσμένου εγγράφου
Console.WriteLine("Metadata Custom Property MyTest: " + w.CustomDocumentProperties["MyTest"]);
```
