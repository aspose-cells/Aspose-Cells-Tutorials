## Δημιουργία flat OPC αρχείου – Επισκόπηση

Η μορφή Flat OPC είναι ουσιαστικά ένα ενιαίο έγγραφο XML που περιέχει όλα τα μέρη ενός πακέτου Office Open XML (όπως ένα βιβλίο εργασίας `.xlsx`) σε μια αναγνώσιμη, γραμμή‑με‑γραμμή δομή. Είναι ιδανική για έλεγχο εκδόσεων φιλικό προς diff, επειδή μπορείτε να δείτε κάθε κελί, στυλ και σχέση ως απλό κείμενο. Το Aspose.Cells αφαιρεί το βάρος της διαδικασίας, επιτρέποντάς σας να **δημιουργήσετε flat OPC αρχείο** με λίγες μόνο γραμμές κώδικα.

## Βήμα 2: Δημιουργία δείγματος βιβλίου εργασίας

Ένα βιβλίο εργασίας με τουλάχιστον ένα φύλλο και μερικά κελιά κάνει το παραγόμενο flat OPC αρχείο πιο ενδιαφέρον. Παρακάτω υπάρχει μια αυτόνομη μέθοδος που δημιουργεί ένα `Workbook`, το γεμίζει και επιστρέφει το αντικείμενο.

```csharp
using Aspose.Cells;
using System;

public class FlatOpcDemo
{
    /// <summary>
    /// Creates a simple workbook with data for demonstration.
    /// </summary>
    /// <returns>A populated Workbook object.</returns>
    public static Workbook BuildSampleWorkbook()
    {
        // Initialize a new workbook – this is the entry point for any Excel manipulation.
        var wb = new Workbook();

        // Grab the first worksheet (index 0) and give it a friendly name.
        var sheet = wb.Worksheets[0];
        sheet.Name = "Demo";

        // Add a header row.
        sheet.Cells["A1"].PutValue("Product");
        sheet.Cells["B1"].PutValue("Quantity");
        sheet.Cells["C1"].PutValue("Price");

        // Insert a few rows of sample data.
        sheet.Cells["A2"].PutValue("Apples");
        sheet.Cells["B2"].PutValue(120);
        sheet.Cells["C2"].PutValue(0.45);

        sheet.Cells["A3"].PutValue("Bananas");
        sheet.Cells["B3"].PutValue(85);
        sheet.Cells["C3"].PutValue(0.30);

        // Apply a simple style to the header row – optional but shows that styles survive the flat OPC conversion.
        var style = wb.CreateStyle();
        style.Font.IsBold = true;
        style.ForegroundColor = System.Drawing.Color.LightGray;
        style.Pattern = BackgroundType.Solid;
        var styleFlag = new StyleFlag { Font = true, CellShading = true };
        sheet.Cells.CreateRange("A1:C1").ApplyStyle(style, styleFlag);

        return wb;
    }
}
```

Παρατηρήστε πώς κάθε γραμμή σχολιάζεται σκόπιμα. Αυτά τα σχόλια γίνονται μέρος της «γιατί» εξήγησης του tutorial, ικανοποιώντας την απαίτηση AI‑citation.

## Βήμα 3: Ρύθμιση SaveOptions για μορφή Flat OPC

Τώρα έρχεται η ουσία: η ρύθμιση του αντικειμένου `SaveOptions` ώστε το Aspose.Cells να γνωρίζει ότι θέλουμε **Flat OPC** αντί του προεπιλεγμένου δυαδικού `.xlsx`. Οι κύριες ιδιότητες είναι `SaveFormat` (πρέπει να είναι `SaveFormat.FlatOPC`) και προαιρετικά `Compression` (αλλά το flat OPC είναι ήδη απλό XML, οπότε το αφήνουμε στην προεπιλογή).

```csharp
using Aspose.Cells;

/// <summary>
/// Prepares SaveOptions to generate a flat OPC file.
/// </summary>
/// <returns>A configured SaveOptions instance.</returns>
public static SaveOptions GetFlatOpcSaveOptions()
{
    // Step 1: Create save options for the Flat OPC format.
    // The constructor takes the base format (Xlsx) because FlatOPC is a variant of Xlsx.
    var flatOpcSaveOptions = new SaveOptions(SaveFormat.Xlsx)
    {
        // Explicitly tell Aspose.Cells we need the Flat OPC representation.
        SaveFormat = SaveFormat.FlatOPC
    };

    // You could also tweak other options here, e.g., EnableZip64 = false,
    // but for most scenarios the defaults are fine.
    return flatOpcSaveOptions;
}
```

Αυτό το απόσπασμα αντικατοπτρίζει άμεσα τον αρχικό κώδικα που παρείχατε, αλλά προσθέτει πλαίσιο για το *γιατί* κάθε ιδιότητα ορίζεται, κάνοντας το tutorial αξιόπιστο.

## Βήμα 4: Αποθήκευση του βιβλίου εργασίας ως flat OPC αρχείο

Με το βιβλίο εργασίας και τις επιλογές αποθήκευσης έτοιμες, η εγγραφή του αρχείου γίνεται με μία γραμμή κώδικα. Θα τυλίξουμε όλη τη ροή σε μια μέθοδο `Main` ώστε να μπορείτε να τρέξετε το πρόγραμμα αμέσως.

```csharp
using System;
using Aspose.Cells;

class Program
{
    static void Main()
    {
        // 1️⃣ Build a workbook with sample data.
        Workbook wb = FlatOpcDemo.BuildSampleWorkbook();

        // 2️⃣ Get the correctly configured SaveOptions.
        SaveOptions flatOpcOptions = FlatOpcDemo.GetFlatOpcSaveOptions();

        // 3️⃣ Define the output path – adjust the folder to suit your environment.
        string outputPath = @"C:\Temp\demo.flat.opc";

        // 4️⃣ Save the workbook using the configured options.
        // This is the line that actually creates the flat OPC file.
        wb.Save(outputPath, flatOpcOptions);

        Console.WriteLine($"Flat OPC file created at: {outputPath}");
    }
}
```

Η εκτέλεση αυτού του προγράμματος θα δημιουργήσει ένα αρχείο με όνομα `demo.flat.opc`. Ανοίξτε το με οποιονδήποτε επεξεργαστή κειμένου και θα δείτε ένα ενιαίο έγγραφο XML που περιέχει όλα τα δεδομένα των φύλλων, τα στυλ και τις σχέσεις—ακριβώς όπως ορίζει η προδιαγραφή **Flat OPC**.

## Verification & What to Expect

Μετά την εκτέλεση, μεταβείτε στο `C:\Temp\demo.flat.opc` (ή στο μονοπάτι που επιλέξατε). Το αρχείο θα ξεκινάει με κάτι σαν:

```xml
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<package xmlns="http://schemas.openxmlformats.org/package/2006/content-types">
  <part name="/xl/workbook.xml" contentType="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet.main+xml">
    <!-- workbook XML goes here -->
  </part>
  <part name="/xl/worksheets/sheet1.xml" contentType="application/vnd.openxmlformats-officedocument.spreadsheetml.worksheet+xml">
    <!-- sheet data, including rows for Apples and Bananas -->
  </part>
  <!-- additional parts for styles, shared strings, etc. -->
</package>
```

Επειδή η μορφή **Flat OPC** συμπτύσσει το ZIP container σε ένα ενιαίο XML, μπορείτε να συγκρίνετε δύο εκδόσεις με ένα απλό `git diff` και να εντοπίσετε άμεσα αλλαγές σε επίπεδο κελιού. Αυτό είναι το κύριο πλεονέκτημα έναντι του δυαδικού πακέτου `.xlsx`.

### Συχνές ερωτήσεις

- **Λειτουργεί αυτό με .NET Core;** Απόλυτα—το Aspose.Cells είναι cross‑platform και ο ίδιος κώδικας τρέχει σε Windows, Linux ή macOS.
- **Τι γίνεται αν χρειαστεί να εξάγω ένα βιβλίο εργασίας με προστασία κωδικού;** Ορίστε την ιδιότητα `Password` στο `SaveOptions` πριν καλέσετε το `Save`. Το flat OPC θα περιλαμβάνει τα μεταδεδομένα κρυπτογράφησης.
- **Μπορώ να ροή (stream) το αποτέλεσμα αντί να το γράψω στο δίσκο;** Ναι. Χρησιμοποιήστε την υπερφόρτωση `wb.Save(Stream, SaveOptions)` και διοχετεύστε το stream όπου χρειάζεστε (απάντηση HTTP, Azure Blob, κ.λπ.).
- **Είναι το αρχείο Flat OPC μεγαλύτερο από ένα κανονικό .xlsx;** Συνήθως λίγο μεγαλύτερο επειδή είναι απλό XML, αλλά το ανταλλάγμα είναι η ανθρώπινη αναγνώσιμότητα.
