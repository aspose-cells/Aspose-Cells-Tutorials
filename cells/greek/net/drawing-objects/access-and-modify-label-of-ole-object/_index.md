## Οδηγίες εγκατάστασης
1. Εγκατάσταση Aspose.Cells για .NET: Χρησιμοποιήστε το NuGet Package Manager για να εγκαταστήσετε τη βιβλιοθήκη Aspose.Cells στο έργο σας.
   ```
   Install-Package Aspose.Cells
   ```
2. Προσθήκη με χρήση οδηγιών: Συμπεριλάβετε τους απαραίτητους χώρους ονομάτων στο αρχείο κώδικά σας.
   ```csharp
   using Aspose.Cells;
   using System.IO;
   ```
3. Προσθήκη δείγματος κώδικα: Αντιγράψτε και επικολλήστε τον παρεχόμενο κώδικα στο έργο σας, φροντίζοντας να καθορίσετε τη διαδρομή του καταλόγου του εγγράφου σας.
4. Εκτελέστε τον Κώδικα: Εκτελέστε το έργο σας για να αποκτήσετε πρόσβαση και να τροποποιήσετε την ετικέτα αντικειμένου Ole στο αρχείο Excel. Ελέγξτε την κονσόλα για μηνύματα εξόδου που επιβεβαιώνουν την επιτυχή εκτέλεση της διαδικασίας.
