# Δημιουργία βιβλίου εργασίας Excel με γραμματοσειρές Άπω Ανατολής και Λατινικής

Αυτό το παράδειγμα δείχνει πώς να χρησιμοποιήσετε το Aspose.Cells για .NET για να δημιουργήσετε ένα βιβλίο εργασίας του Excel, να προσθέσετε ένα πλαίσιο κειμένου σε ένα φύλλο εργασίας και να ορίσετε ονόματα γραμματοσειρών Άπω Ανατολής και Λατινικής γλώσσας για το κείμενο μέσα σε αυτό το πλαίσιο κειμένου. Ο κώδικας παρουσιάζει την αρχικοποίηση του χώρου εργασίας, τον χειρισμό στοιχείων φύλλου εργασίας και την αποθήκευση του βιβλίου εργασίας με συγκεκριμένη μορφοποίηση γραμματοσειράς προσαρμοσμένη για ποικίλες γλωσσικές απαιτήσεις.

## Οδηγίες εγκατάστασης
1. Εγκατάσταση του Aspose.Cells για .NET: 
   - Χρησιμοποιήστε το NuGet Package Manager στο έργο σας στο Visual Studio:
     ```
     Install-Package Aspose.Cells
     ```
   - Ή κατεβάστε το DLL από το [Ιστότοπος Aspose](https://www.aspose.com/downloads/cells/net).

2. Προσθήκη αναφορών: Βεβαιωθείτε ότι το έργο σας αναφέρεται στη βιβλιοθήκη Aspose.Cells.

3. Ρύθμιση καταλόγου εξόδου: Ενημέρωση του `outputDir` μεταβλητή στον κώδικα με την επιθυμητή διαδρομή εξόδου όπου θα αποθηκευτεί το αρχείο Excel.

4. Εκτέλεση του παραδείγματος: Αντιγράψτε τον παρεχόμενο κώδικα C# στο έργο σας, μεταγλωττίστε τον και εκτελέστε τον. Θα πρέπει να δείτε την επιβεβαίωση εξόδου και το αρχείο Excel με το διαμορφωμένο πλαίσιο κειμένου θα δημιουργηθεί στον καθορισμένο κατάλογο.
