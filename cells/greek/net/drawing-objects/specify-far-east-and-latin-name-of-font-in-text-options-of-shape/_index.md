# Δημιουργήστε Excel Workbook με Far East & Latin Font

Αυτό το παράδειγμα δείχνει πώς να χρησιμοποιήσετε το Aspose.Cells για το .NET για να δημιουργήσετε ένα εργασιακό βιβλίο Excel, να προσθέσετε ένα κείμενο σε ένα φύλλο εργασίας και να καθορίσετε τόσο τις Far East όσο και τα λατινικά ονόματα γραμματοσειρών για το κείμενο μέσα σε αυτό το κείμενο. Ο κώδικας παρουσιάζει την αρχιτοποίηση του χώρου εργασίας, την χειραγώγηση των στοιχείων του φύλλου εργασίας και τη διατήρηση του εργασιακού βιβλίου με συγκεκριμένη μορφοποίηση γραμματοσειρών προσαρμοσμένη για τις διάφορες απαιτήσεις γλώσσας.

## Διδασκαλίες εγκατάστασης
1. Εγκαταστήστε Aspose.Cells για .NET: 
   * Χρησιμοποιήστε το NuGet Package Manager στο έργο Visual Studio σας:
     ```
     Install-Package Aspose.Cells
     ```
   * Ή κατεβάστε το DLL από το [Στόχος ιστοσελίδα](https://www.aspose.com/downloads/cells/net)\- Δεν ξέρω .

2. Προσθέστε Αναφορές: Βεβαιωθείτε ότι το έργο σας αναφορές στη βιβλιοθήκη Aspose.Cells.

3. Εγκαταστήστε τον κατάλογο εξόδου: Αναβαθμίστε το `εξόδουDir`μεταβλητή στον κώδικα με την επιθυμητή διαδρομή εξόδου όπου θα αποθηκευτεί το αρχείο Excel.

4. Run the Example: Copy the provided C# code into your project, compile, and run it. You should see the output confirmation, and the Excel file with the configured textbox will be created in the specified directory.
