Αυτό το εγχειρίδιο δείχνει πώς να χρησιμοποιήσετε το Aspose.Cells για το .NET για να δημιουργήσετε ένα PDF από ένα διάγραμμα Excel ενώ καθορίζετε το μέγεθος και την ευθυγράμμιση της σελίδας.
