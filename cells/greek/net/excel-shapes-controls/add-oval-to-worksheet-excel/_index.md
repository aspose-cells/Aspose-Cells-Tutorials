# Προσθήκη οβάλ σε φύλλο εργασίας στο Excel

## Εισαγωγή
Η δημιουργία εκπληκτικών και διαδραστικών αρχείων Excel μπορεί να περιλαμβάνει περισσότερα από απλούς αριθμούς και τύπους. Σχήματα όπως τα οβάλ μπορούν να προσθέσουν μια οπτική γοητεία ή να παρέχουν λειτουργικά στοιχεία στα φύλλα εργασίας σας. Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να χρησιμοποιήσετε το Aspose.Cells για .NET για να προσθέσετε οβάλ σε ένα φύλλο εργασίας Excel μέσω προγραμματισμού. Είτε θέλετε να προσθέσετε λίγη πινελιά είτε λειτουργικότητα, σας καλύπτουμε με έναν οδηγό βήμα προς βήμα που αναλύει τα πάντα.
## Προαπαιτούμενα

2. Περιβάλλον ανάπτυξης: AC# IDE όπως το Visual Studio.

Επίσης, θυμηθείτε να ρυθμίσετε το έργο σας εγκαθιστώντας τη βιβλιοθήκη Aspose.Cells για .NET. ## Εισαγωγή πακέτων
Πριν γράψετε οποιονδήποτε κώδικα, βεβαιωθείτε ότι έχετε συμπεριλάβει τους απαιτούμενους χώρους ονομάτων. Ακολουθεί το απόσπασμα κώδικα C# για να βεβαιωθείτε ότι χρησιμοποιείτε τις σωστές βιβλιοθήκες:
```csharp
using System.IO;
using Aspose.Cells;
using Aspose.Cells.Drawing;
using System.Drawing;
```
## Βήμα 1: Ρύθμιση του καταλόγου σας
Το πρώτο βήμα για την προσθήκη ενός οβάλ σε ένα φύλλο Excel είναι να καθορίσετε πού θα αποθηκευτεί το αρχείο Excel. Ας ορίσουμε τη διαδρομή του καταλόγου και ας βεβαιωθούμε ότι ο κατάλογος υπάρχει πριν αποθηκεύσουμε την εργασία μας.

Θα δημιουργήσουμε μια διαδρομή καταλόγου και θα επαληθεύσουμε αν υπάρχει. Εάν ο φάκελος δεν υπάρχει, θα δημιουργηθεί.
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "Your Document Directory";
// Δημιουργήστε έναν κατάλογο εάν δεν υπάρχει ήδη.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
Αυτό το βήμα είναι κρίσιμο, καθώς διασφαλίζει ότι το αρχείο σας αποθηκεύεται στη σωστή θέση και ότι δεν θα αντιμετωπίσετε προβλήματα με τη διαδρομή του αρχείου αργότερα.
## Βήμα 2: Αρχικοποίηση νέου βιβλίου εργασίας
Στη συνέχεια, πρέπει να δημιουργήσουμε ένα νέο βιβλίο εργασίας στο οποίο θα προσθέσουμε τα οβάλ σχήματά μας. Το βιβλίο εργασίας αντιπροσωπεύει ένα αρχείο Excel και μπορούμε να προσθέσουμε περιεχόμενο ή σχήματα σε αυτό.

Σε αυτό το βήμα, δημιουργούμε ένα νέο `Workbook` αντικείμενο που θα χρησιμεύσει ως κοντέινερ αρχείου Excel.
```csharp
// Δημιουργήστε ένα νέο Βιβλίο Εργασίας.
Workbook excelbook = new Workbook();
```
## Βήμα 3: Προσθέστε το πρώτο οβάλ σχήμα
Τώρα έρχεται το διασκεδαστικό κομμάτι—η προσθήκη ενός οβάλ σχήματος στο φύλλο εργασίας. Αυτό το οβάλ θα μπορούσε να αντιπροσωπεύει ένα οπτικό στοιχείο όπως ένα κουμπί ή μια επισήμανση. Θα ξεκινήσουμε προσθέτοντας το πρώτο οβάλ σχήμα στο πρώτο φύλλο εργασίας του βιβλίου εργασίας μας.

Εδώ, χρησιμοποιούμε το `Shapes.AddOval()` μέθοδος για τη δημιουργία ενός οβάλ στο φύλλο εργασίας σε μια συγκεκριμένη γραμμή και στήλη.
```csharp
// Προσθέστε ένα οβάλ σχήμα.
Aspose.Cells.Drawing.Oval oval1 = excelbook.Worksheets[0].Shapes.AddOval(2, 0, 2, 0, 130, 160);
```
Οι παράμετροι στο εσωτερικό `AddOval()` είναι τα εξής:
- Οι δύο πρώτοι αριθμοί αντιπροσωπεύουν τη γραμμή και τη στήλη για την επάνω αριστερή γωνία του οβάλ.
- Οι επόμενοι δύο αριθμοί αντιπροσωπεύουν το ύψος και το πλάτος του οβάλ.
## Βήμα 4: Ορίστε την τοποθέτηση και το στυλ του οβάλ
Μόλις δημιουργηθεί το οβάλ, μπορούμε να ορίσουμε τη θέση του, το πάχος της γραμμής και το στυλ της παύλας. `Placement` Η ιδιότητα καθορίζει τον τρόπο συμπεριφοράς του οβάλ όταν αλλάζετε το μέγεθος ή μετακινείτε κελιά στο φύλλο εργασίας.

Κάνουμε το οβάλ να αιωρείται ελεύθερα και προσαρμόζουμε την εμφάνισή του.
```csharp
// Ορίστε την τοποθέτηση του οβάλ.
oval1.Placement = PlacementType.FreeFloating;
// Ορίστε το πάχος της γραμμής.
oval1.Line.Weight = 1;
// Ορίστε το στυλ παύλας του οβάλ.
oval1.Line.DashStyle = MsoLineDashStyle.Solid;
```
Αυτό επιτρέπει στο οβάλ να κινείται ελεύθερα μέσα στο φύλλο εργασίας και το πάχος γραμμής και το στυλ του ορίζονται για οπτική συνέπεια.
## Βήμα 5: Προσθέστε ένα ακόμη οβάλ (κύκλο) σχήμα
Γιατί να σταματήσουμε σε ένα; Σε αυτό το βήμα, θα προσθέσουμε ένα ακόμη οβάλ σχήμα, αυτή τη φορά δημιουργώντας έναν τέλειο κύκλο κάνοντας το ύψος και το πλάτος ίδια.

Δημιουργούμε ένα άλλο οβάλ, το τοποθετούμε σε διαφορετική θέση και διασφαλίζουμε ότι έχει κυκλικό σχήμα ορίζοντας ίσο ύψος και πλάτος.
```csharp
// Προσθέστε ένα άλλο οβάλ (κυκλικό) σχήμα.
Aspose.Cells.Drawing.Oval oval2 = excelbook.Worksheets[0].Shapes.AddOval(9, 0, 2, 15, 130, 130);
```
## Βήμα 6: Στυλίστε το δεύτερο οβάλ
Όπως και πριν, θα προσαρμόσουμε την τοποθέτηση, το βάρος και το στυλ της παύλας αυτού του δεύτερου οβάλ (ή κύκλου).

Εφαρμόζουμε παρόμοιες ιδιότητες στο δεύτερο οβάλ για να ταιριάζει με το στυλ του πρώτου.
```csharp
// Ορίστε την τοποθέτηση του οβάλ.
oval2.Placement = PlacementType.FreeFloating;
// Ορίστε το πάχος της γραμμής.
oval2.Line.Weight = 1;
// Ορίστε το στυλ παύλας του οβάλ.
oval2.Line.DashStyle = MsoLineDashStyle.Solid;
```
## Βήμα 7: Αποθήκευση του βιβλίου εργασίας
Τέλος, πρέπει να αποθηκεύσουμε το βιβλίο εργασίας με τα οβάλ που μόλις προσθέσαμε. Η αποθήκευση του αρχείου διασφαλίζει ότι όλες οι αλλαγές μας αποθηκεύονται.

Αποθηκεύουμε το βιβλίο εργασίας στη διαδρομή καταλόγου που ορίσαμε νωρίτερα.
```csharp
// Αποθηκεύστε το αρχείο excel.
excelbook.Save(dataDir + "book1.out.xls");
```
Και αυτό είναι όλο! Προσθέσατε με επιτυχία οβάλ στο φύλλο εργασίας του Excel και αποθηκεύσατε το αρχείο.
