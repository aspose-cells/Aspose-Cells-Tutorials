# Προσθήκη συνδυαστικού πλαισίου σε φύλλο εργασίας στο Excel

## Εισαγωγή
Η δημιουργία διαδραστικών υπολογιστικών φύλλων Excel μπορεί να βελτιώσει σημαντικά την εμπειρία χρήστη, ειδικά όταν προσθέτετε στοιχεία φόρμας όπως συνδυαστικά πλαίσια. Τα συνδυαστικά πλαίσια επιτρέπουν στους χρήστες να επιλέγουν επιλογές από μια προκαθορισμένη λίστα, προσθέτοντας ευκολία και αποτελεσματικότητα στην εισαγωγή δεδομένων. Με το Aspose.Cells για .NET, μπορείτε να δημιουργήσετε συνδυαστικά πλαίσια μέσω προγραμματισμού σε φύλλα Excel χωρίς να χρησιμοποιήσετε απευθείας το Excel. Αυτή η ισχυρή βιβλιοθήκη επιτρέπει στους προγραμματιστές να χειρίζονται αρχεία Excel με διάφορους τρόπους, συμπεριλαμβανομένης της δυνατότητας αυτοματοποίησης των στοιχείων ελέγχου φόρμας.
Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία προσθήκης ενός σύνθετου πλαισίου σε ένα φύλλο εργασίας στο Excel χρησιμοποιώντας το Aspose.Cells για .NET. Αν θέλετε να δημιουργήσετε δυναμικά, φιλικά προς το χρήστη υπολογιστικά φύλλα, αυτός ο οδηγός θα σας βοηθήσει να ξεκινήσετε.

## Εισαγωγή πακέτων
Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαιτούμενους χώρους ονομάτων στο έργο σας. Δείτε τι χρειάζεστε:
```csharp
using System.IO;
using Aspose.Cells;
```
Αυτά είναι απαραίτητα για την αλληλεπίδραση με αρχεία Excel και τον χειρισμό στοιχείων φόρμας, όπως τα σύνθετα πλαίσια στο βιβλίο εργασίας.
Ας αναλύσουμε τη διαδικασία προσθήκης ενός σύνθετου πλαισίου σε πολλά απλά βήματα για εύκολη κατανόηση.
## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων
Το πρώτο βήμα είναι να δημιουργήσετε έναν κατάλογο όπου θα αποθηκευτούν τα αρχεία Excel. Μπορείτε να δημιουργήσετε έναν νέο φάκελο εάν δεν υπάρχει ήδη.
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "Your Document Directory";
// Δημιουργήστε έναν κατάλογο εάν δεν υπάρχει ήδη.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
- dataDir: Καθορίζει τη θέση όπου θα αποθηκευτεί το αρχείο εξόδου.
- System.IO.Directory.Exists: Ελέγχει αν ο κατάλογος υπάρχει ήδη.
- System.IO.Directory.CreateDirectory: Δημιουργεί τον κατάλογο εάν λείπει.
## Βήμα 2: Δημιουργία νέου βιβλίου εργασίας
Τώρα, δημιουργήστε ένα νέο βιβλίο εργασίας του Excel όπου θα προσθέσετε το σύνθετο πλαίσιο.

```csharp
// Δημιουργήστε ένα νέο Βιβλίο Εργασίας.
Workbook workbook = new Workbook();
```

- Βιβλίο εργασίας: Αρχικοποιεί μια νέα παρουσία της κλάσης Βιβλίου εργασίας, που αντιπροσωπεύει ένα αρχείο Excel.
## Βήμα 3: Λήψη του Φύλλου Εργασίας και των Κελιών
Στη συνέχεια, αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας από το βιβλίο εργασίας και ανακτήστε τη συλλογή κελιών όπου θα εισαγάγετε δεδομένα.

```csharp
// Αποκτήστε το πρώτο φύλλο εργασίας.
Worksheet sheet = workbook.Worksheets[0];
// Αποκτήστε τη συλλογή κελιών του φύλλου εργασίας.
Cells cells = sheet.Cells;
```

- Φύλλο εργασίας: Ανακτά το πρώτο φύλλο εργασίας από το βιβλίο εργασίας.
- Κελιά κελιά: Λαμβάνει τη συλλογή κελιών από το φύλλο εργασίας.
## Βήμα 4: Εισαγωγή τιμών για το σύνθετο πλαίσιο
Τώρα, πρέπει να εισαγάγουμε ορισμένες τιμές στα κελιά. Αυτές οι τιμές θα χρησιμεύσουν ως επιλογές για το σύνθετο πλαίσιο.

```csharp
// Εισαγάγετε μια τιμή.
cells["B3"].PutValue("Employee:");
// Ορίστε το με έντονη γραφή.
cells["B3"].GetStyle().Font.IsBold = true;
// Εισαγάγετε ορισμένες τιμές που υποδηλώνουν το εύρος εισόδου για το σύνθετο πλαίσιο.
cells["A2"].PutValue("Emp001");
cells["A3"].PutValue("Emp002");
cells["A4"].PutValue("Emp003");
cells["A5"].PutValue("Emp004");
cells["A6"].PutValue("Emp005");
cells["A7"].PutValue("Emp006");
```

- cells["B3"].PutValue: Τοποθετεί την ετικέτα "Υπάλληλος" στο κελί B3.
- Font.IsBold = true: Ορίζει το κείμενο σε έντονη γραφή για να ξεχωρίζει.
- Εύρος εισαγωγής: Εισάγει πολλά αναγνωριστικά υπαλλήλων στα κελιά A2 έως A7. Αυτά θα εμφανιστούν στο αναπτυσσόμενο μενού του σύνθετου πλαισίου.
## Βήμα 5: Προσθέστε το Σύνθετο Πλαίσιο στο Φύλλο Εργασίας
Το επόμενο βήμα είναι να προσθέσετε το στοιχείο ελέγχου σύνθετου πλαισίου στο φύλλο εργασίας σας. Αυτό το σύνθετο πλαίσιο θα επιτρέψει στους χρήστες να επιλέξουν ένα από τα αναγνωριστικά υπαλλήλων που εισαγάγατε νωρίτερα.

```csharp
// Προσθήκη νέου συνδυαστικού πλαισίου.
Aspose.Cells.Drawing.ComboBox comboBox = sheet.Shapes.AddComboBox(2, 0, 2, 0, 22, 100);
```

- AddComboBox: Προσθέτει ένα νέο σύνθετο πλαίσιο στο φύλλο εργασίας. Οι αριθμοί (2, 0, 2, 0, 22, 100) αντιπροσωπεύουν τη θέση και τις διαστάσεις του σύνθετου πλαισίου.
## Βήμα 6: Συνδέστε το σύνθετο πλαίσιο με ένα κελί και ορίστε το εύρος εισόδου
Για να κάνουμε το σύνθετο πλαίσιο λειτουργικό, πρέπει να το συνδέσουμε με ένα συγκεκριμένο κελί και να ορίσουμε το εύρος των κελιών από τα οποία θα αντλεί τις επιλογές του.

```csharp
// Ορίστε το συνδεδεμένο κελί.
comboBox.LinkedCell = "A1";
// Ορίστε το εύρος εισόδου.
comboBox.InputRange = "A2:A7";
```

- LinkedCell: Συνδέει την επιλογή του σύνθετου πλαισίου με το κελί A1. Η επιλεγμένη τιμή από το σύνθετο πλαίσιο θα εμφανιστεί σε αυτό το κελί.
- InputRange: Ορίζει την περιοχή κελιών (A2:A7) που περιέχει τις τιμές που θα συμπληρώσουν τις επιλογές του σύνθετου πλαισίου.
## Βήμα 7: Προσαρμόστε την εμφάνιση του συνδυαστικού πλαισίου
Μπορείτε να προσαρμόσετε περαιτέρω το σύνθετο πλαίσιο καθορίζοντας τον αριθμό των αναπτυσσόμενων γραμμών και ενεργοποιώντας την τρισδιάστατη σκίαση για καλύτερη αισθητική.

```csharp
// Ορίστε τον αριθμό των γραμμών λίστας που εμφανίζονται στο τμήμα λίστας του σύνθετου πλαισίου.
comboBox.DropDownLines = 5;
// Ορίστε το σύνθετο πλαίσιο με τρισδιάστατη σκίαση.
comboBox.Shadow = true;
```

- Αναπτυσσόμενες Γραμμές: Ελέγχει πόσες επιλογές θα είναι ορατές στο αναπτυσσόμενο μενού του σύνθετου πλαισίου ταυτόχρονα.
- Σκιά: Προσθέτει ένα εφέ σκίασης 3D στο σύνθετο πλαίσιο.
## Βήμα 8: Αυτόματη Προσαρμογή Στηλών και Αποθήκευση του Βιβλίου Εργασίας
Τέλος, ας προσαρμόσουμε αυτόματα τις στήλες για μια καθαρή διάταξη και ας αποθηκεύσουμε το βιβλίο εργασίας.

```csharp
// Αυτόματη προσαρμογή στηλών
sheet.AutoFitColumns();
// Αποθηκεύει το αρχείο.
workbook.Save(dataDir + "book1.out.xls");
```

- Αυτόματη Προσαρμογή Στηλών: Προσαρμόζει αυτόματα το πλάτος των στηλών ώστε να ταιριάζει στο περιεχόμενο.
- Αποθήκευση: Αποθηκεύει το βιβλίο εργασίας ως αρχείο Excel στον καθορισμένο κατάλογο.
