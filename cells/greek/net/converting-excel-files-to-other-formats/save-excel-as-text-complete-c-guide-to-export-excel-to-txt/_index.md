# Αποθήκευση Excel ως κείμενο – Πλήρης Οδηγός C#

Έχετε χρειαστεί ποτέ να **αποθηκεύσετε το Excel ως κείμενο** αλλά δεν ήσασταν σίγουροι ποια κλήση API να χρησιμοποιήσετε; Δεν είστε μόνοι. Πολλοί προγραμματιστές συναντούν δυσκολίες όταν προσπαθούν να **εξάγουν το Excel σε txt** επειδή οι προεπιλεγμένες βιβλιοθήκες interop είναι αργές και ακατάλληλες.  

Σε αυτό το tutorial θα περάσουμε από μια καθαρή, έτοιμη για παραγωγή λύση που μετατρέπει ένα βιβλίο εργασίας *.xlsx* σε ένα απλό αρχείο *.txt*, όλα με λίγες μόνο γραμμές C#. Στο τέλος θα ξέρετε πώς να **μετατρέψετε υπολογιστικό φύλλο σε txt**, να ρυθμίσετε τις επιλογές στρογγυλοποίησης και να αποφύγετε τα πιο συχνά προβλήματα όταν **μετατρέπετε xlsx σε txt**.

> **Τι θα πάρετε:** ένα πλήρες, εκτελέσιμο πρόγραμμα, εξηγήσεις για το *γιατί* κάθε γραμμή είναι σημαντική, και συμβουλές για την επέκταση της λογικής σε μεγαλύτερα βιβλία εργασίας ή προσαρμοσμένους οριοθέτες.

---

## Προαπαιτήσεις

Πριν προχωρήσουμε, βεβαιωθείτε ότι έχετε:

* .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί σε .NET Core και .NET Framework εξίσου).  
* Το πακέτο NuGet **Aspose.Cells for .NET** – περιλαμβάνει τις κλάσεις `Workbook` και `TxtSaveOptions` που θα χρησιμοποιήσουμε.  
* Ένα απλό αρχείο Excel (`nums.xlsx`) τοποθετημένο κάπου που μπορείτε να αναφέρετε με απόλυτη ή σχετική διαδρομή.  

Αν δεν έχετε εγκαταστήσει ακόμη το Aspose.Cells, εκτελέστε:

```bash
dotnet add package Aspose.Cells
```

Αυτό είναι όλο—χωρίς COM interop, χωρίς εγκατάσταση Office.

## Βήμα 1: Φόρτωση του Excel Workbook

Το πρώτο που χρειάζεται είναι μια παρουσία της `Workbook` που δείχνει στο αρχείο προέλευσης. Σκεφτείτε τη `Workbook` ως την αναπαράσταση στη μνήμη ολόκληρου του εγγράφου Excel.

```csharp
using Aspose.Cells;
using System;

class Program
{
    static void Main()
    {
        // 🔹 Load the Excel workbook from disk
        Workbook workbook = new Workbook("YOUR_DIRECTORY/nums.xlsx");
```

**Γιατί είναι σημαντικό:**  
`Workbook` αναλύει το αρχείο μία φορά, δημιουργεί αντικείμενα κελιών και διατηρεί τις πληροφορίες στυλ έτοιμες για οποιαδήποτε επόμενη εξαγωγή. Η προφόρτωση σας επιτρέπει επίσης να ελέγξετε τον αριθμό φύλλων ή να επικυρώσετε δεδομένα πριν γράψετε το αρχείο κειμένου.

---

## Βήμα 2: Διαμόρφωση των επιλογών αποθήκευσης κειμένου (Export Excel to TXT)

Το Aspose.Cells μας παρέχει την κλάση `TxtSaveOptions` όπου μπορούμε να ρυθμίσουμε λεπτομερώς πώς εμφανίζονται οι αριθμοί. Σε αυτό το παράδειγμα περιορίζουμε την έξοδο σε **τέσσερα σημαντικά ψηφία** και στρογγυλοποιούμε, ώστε το αρχείο κειμένου να παραμένει τακτοποιημένο.

```csharp
        // 🔹 Set up how the data will be written to .txt
        TxtSaveOptions saveOptions = new TxtSaveOptions
        {
            // Keep numbers readable – 4 significant digits, rounded
            SignificantDigits = 4,
            DigitsMode = DigitsMode.Round
        };
```

**Γιατί μπορεί να το αλλάξετε:**  
Αν το φύλλο σας περιέχει επιστημονικά δεδομένα, ίσως θέλετε περισσότερα ψηφία ή διαφορετικό τρόπο στρογγυλοποίησης. Το `TxtSaveOptions` υποστηρίζει επίσης προσαρμοσμένους οριοθέτες (tab, comma, semicolon) και κωδικοποίηση—ιδανικό για διεθνή έργα.

---

## Βήμα 3: Αποθήκευση του Workbook ως αρχείο κειμένου (Convert Spreadsheet to TXT)

Τώρα γίνεται η βαριά δουλειά. Παραδίδουμε τη `Workbook` και τις ρυθμισμένες `TxtSaveOptions` στη μέθοδο `Save`, η οποία γράφει μια απλή κειμενική αναπαράσταση του ενεργού φύλλου.

```csharp
        // 🔹 Export the workbook to a .txt file using the options above
        workbook.Save("YOUR_DIRECTORY/nums.txt", saveOptions);

        Console.WriteLine("✅ Excel file has been saved as text!");
    }
}
```

**Τι θα δείτε:** ένα αρχείο `.txt` με οριοθέτη tab, όπου η τιμή κάθε κελιού ακολουθεί τον κανόνα στρογγυλοποίησης τεσσάρων ψηφίων. Ανοίξτε το σε Notepad ή οποιονδήποτε επεξεργαστή και θα δείτε κάτι σαν:

```
12.34	56.78	90.12
3.1416	2.718	1.618
```

Αν ανοίξετε ξανά το αρχείο στο Excel (Data → From Text), οι αριθμοί θα ευθυγραμμιστούν ακριβώς όπως εμφανίζονταν στο αρχικό βιβλίο εργασίας.

## Export Excel to TXT – Επιλογή Οριοθέτη

Από προεπιλογή το Aspose χρησιμοποιεί οριοθέτη **tab** (`\t`), που είναι ιδανικός για τις περισσότερες περιπτώσεις μετατροπής φύλλου σε κείμενο. Ωστόσο, μπορεί να χρειαστείτε **κόμμα** για ροές συμβατές με CSV.

```csharp
        TxtSaveOptions csvOptions = new TxtSaveOptions
        {
            Delimiter = ',',
            SignificantDigits = 6,
            DigitsMode = DigitsMode.Round
        };
        workbook.Save("YOUR_DIRECTORY/nums_comma.txt", csvOptions);
```

**Συμβουλή:** Όταν σκοπεύετε να τροφοδοτήσετε το αρχείο σε άλλο σύστημα (π.χ. φορτωτής βάσης δεδομένων), ελέγξτε ξανά τον απαιτούμενο οριοθέτη και την κωδικοποίηση (`Encoding` property) για να αποφύγετε διαφθορά δεδομένων.

## Convert Xlsx to Txt – Διαχείριση Πολλαπλών Φύλλων

Το παραπάνω παράδειγμα εξάγει μόνο το **ενεργό φύλλο**. Αν το βιβλίο εργασίας σας περιέχει πολλά tabs και χρειάζεστε το καθένα ως ξεχωριστό αρχείο κειμένου, κάντε βρόχο στη συλλογή `Worksheets`:

```csharp
        foreach (Worksheet sheet in workbook.Worksheets)
        {
            // Activate the sheet before saving
            workbook.Worksheets.ActiveSheetIndex = sheet.Index;

            string txtPath = $"YOUR_DIRECTORY/{sheet.Name}.txt";
            workbook.Save(txtPath, saveOptions);
            Console.WriteLine($"📄 Saved sheet '{sheet.Name}' to {txtPath}");
        }
```

**Γιατί είναι χρήσιμο:**  
Μεγάλες αλυσίδες αναφορών συχνά δημιουργούν ένα φύλλο ανά πελάτη ή ανά μήνα. Η αυτοματοποίηση του διαχωρισμού εξοικονομεί ώρες χειροκίνητης αντιγραφής.

## Συνηθισμένα Πιθανά Προβλήματα Κατά τη Μετατροπή Xlsx σε Txt

| Πρόβλημα | Τι Συμβαίνει | Πώς να το Διορθώσετε |
|----------|--------------|----------------------|
| **Missing Aspose.Cells license** | Η βιβλιοθήκη εμφανίζει υδατογράφημα δοκιμής ή περιορίζει τις γραμμές. | Αγοράστε άδεια ή χρησιμοποιήστε τη δωρεάν λειτουργία αξιολόγησης για μικρά αρχεία. |
| **Wrong encoding** | Οι μη‑ASCII χαρακτήρες γίνονται ακατανόητοι (π.χ. τονισμένα γράμματα). | Ορίστε `saveOptions.Encoding = Encoding.UTF8;` |
| **Large worksheets (>1 M rows)** | Η χρήση μνήμης αυξάνεται απότομα, η διαδικασία μπορεί να καταρρεύσει. | Χρησιμοποιήστε `Workbook.LoadOptions` με `MemorySetting` ορισμένο σε `MemorySetting.MemoryPreference` ή επεξεργαστείτε το φύλλο σε τμήματα. |
| **Unexpected delimiter in data** | Tabs μέσα στις τιμές κελιών διασπούν την ευθυγράμμιση των στηλών. | Αλλάξτε σε λιγότερο κοινό οριοθέτη (π.χ. `|`) και αντικαταστήστε τα tabs στα δεδομένα εκ των προτέρων. |

Η αντιμετώπιση αυτών των ζητημάτων εκ των προτέρων κάνει τη λύση **how to save txt** ανθεκτική για περιβάλλον παραγωγής.

## Pro Tip: Επαλήθευση του Αποτελέσματος Προγραμματιστικά

Αντί να ανοίγετε το αρχείο χειροκίνητα, μπορείτε να διαβάσετε τις πρώτες γραμμές πίσω στο C# για να επιβεβαιώσετε ότι η εξαγωγή ολοκληρώθηκε επιτυχώς:

```csharp
using System.IO;

string[] lines = File.ReadAllLines("YOUR_DIRECTORY/nums.txt");
Console.WriteLine("First line of exported text:");
Console.WriteLine(lines.Length > 0 ? lines[0] : "File is empty!");
```

Αυτός ο γρήγορος έλεγχος είναι χρήσιμος σε CI pipelines όπου θέλετε να βεβαιωθείτε ότι η μετατροπή δεν παρήγαγε κενό αρχείο.

## Recap & Next Steps

Έχουμε καλύψει όλη τη ροή εργασίας **save excel as text**:

1. Φορτώστε το βιβλίο εργασίας με `Workbook`.  
2. Διαμορφώστε το `TxtSaveOptions` (σημαντικά ψηφία, στρογγυλοποίηση, οριοθέτης).  
3. Καλέστε `Save` για να δημιουργήσετε ένα αρχείο κειμένου.  

Τώρα ξέρετε πώς να **export Excel to txt**, **convert spreadsheet to txt**, και πώς να αντιμετωπίσετε τις ιδιαιτερότητες του **convert xlsx to txt** για βιβλία εργασίας με πολλά φύλλα.  

**Τι έπεται;**  

* Δοκιμάστε εξαγωγή σε CSV (`CsvSaveOptions`) για εισαγωγές συμβατές με Excel.
