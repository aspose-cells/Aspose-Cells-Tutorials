# Πώς να Εξάγετε Διαγράμματα από το Word – Πλήρης Οδηγός C#

Ποτέ χρειάστηκε **πώς να εξάγετε διαγράμματα** από ένα έγγραφο Word αλλά δεν ήξερατε από πού να ξεκινήσετε; Δεν είστε μόνοι· πολλοί προγραμματιστές αντιμετωπίζουν αυτό το πρόβλημα όταν αυτοματοποιούν αναφορές. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από μια πρακτική, ολοκληρωμένη λύση που όχι μόνο δείχνει **πώς να εξάγετε διαγράμματα**, αλλά εξηγεί επίσης **πώς να συμπεριλάβετε διαγράμματα** στο εξαγόμενο αρχείο. Στο τέλος θα μπορείτε να εξάγετε διαγράμματα από το Word με λίγες μόνο γραμμές C#.

Θα χρησιμοποιήσουμε τη δημοφιλή βιβλιοθήκη **Aspose.Words for .NET** επειδή διαχειρίζεται αντικείμενα διαγραμμάτων εγγενώς και λειτουργεί με .docx, .doc και ακόμη και παλαιότερες μορφές. Χωρίς να ασχοληθείτε με Office Interop, χωρίς COM εφιάλτες. Τα παρακάτω βήματα υποθέτουν ότι έχετε ένα βασικό έργο C# και το πακέτο NuGet Aspose.Words εγκατεστημένο. Αν είστε νέοι στη βιβλιοθήκη, μην ανησυχείτε· θα καλύψουμε γρήγορα τις προαπαιτήσεις.

## Προαπαιτήσεις

- .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί επίσης σε .NET Framework 4.7+)
- Visual Studio 2022 ή οποιοδήποτε IDE προτιμάτε
- Aspose.Words for .NET (εγκαταστήστε το μέσω `dotnet add package Aspose.Words`)

> **Pro tip:** Κρατήστε την έκδοση του Aspose.Words ενημερωμένη· η τελευταία έκδοση (απρίλιο 2026) προσθέτει καλύτερη διαχείριση διαγραμμάτων και βελτιώσεις απόδοσης.

## Βήμα 1: Φόρτωση του Πηγαίου Εγγράφου Word

Το πρώτο που πρέπει να κάνετε είναι να ανοίξετε το αρχείο `.docx` που περιέχει τα διαγράμματα που θέλετε να εξάγετε. Το Aspose.Words το κάνει με μία γραμμή κώδικα.

```csharp
using Aspose.Words;

// Load the source document (replace with your actual path)
Document document = new Document(@"C:\Docs\input.docx");
```

*Γιατί είναι σημαντικό:* Η φόρτωση του εγγράφου δημιουργεί μια αναπαράσταση στη μνήμη για κάθε στοιχείο—παραγράφους, πίνακες και, κυρίως, τα αντικείμενα διαγράμματος. Χωρίς αυτό το βήμα δεν μπορείτε να προσπελάσετε ή να επεξεργαστείτε τα διαγράμματα.

## Βήμα 2: Διαμόρφωση Επιλογών Αποθήκευσης για Διατήρηση Διαγραμμάτων

Από προεπιλογή, ένα απλό `document.Save("output.docx")` διατηρεί τα πάντα, αλλά αν ενεργοποιήσετε `ExportImages` ή παρόμοιες σημαίες, μπορεί να χάσετε τα ενσωματωμένα διαγράμματα. Για να είμαστε σαφείς—και για να απαντήσουμε στο ερώτημα “**πώς να συμπεριλάβετε διαγράμματα**”—ορίζουμε `DocxSaveOptions` με `ExportCharts = true`.

```csharp
// Create save options that ensure charts are included
DocxSaveOptions saveOptions = new DocxSaveOptions
{
    ExportCharts = true          // Guarantees charts are part of the saved file
};
```

*Εξήγηση:* Το `ExportCharts` λέει στη μηχανή να σειριοποιήσει κάθε διάγραμμα ως εγγενές τμήμα Office Open XML. Αυτό είναι ουσιώδες όταν αργότερα ανοίξετε το αρχείο στο Word ή σε άλλους επεξεργαστές· τα διαγράμματα εμφανίζονται ακριβώς όπως ήταν στο αρχικό έγγραφο.

## Βήμα 3: Αποθήκευση του Εγγράφου με τις Διαμορφωμένες Επιλογές

Τώρα γράφουμε το έγγραφο ξανά στο δίσκο, χρησιμοποιώντας τις επιλογές που ορίσαμε. Το αρχείο εξόδου θα περιέχει όλο το αρχικό περιεχόμενο **και** τα διαγράμματα.

```csharp
// Save the document with charts preserved
document.Save(@"C:\Docs\charts.docx", saveOptions);
```

Σε αυτό το σημείο έχετε ένα νέο αρχείο Word (`charts.docx`) που είναι πιστό αντίγραφο του αρχικού, πλήρες με όλα τα γραφικά διαγράμματα. Ανοίξτε το στο Microsoft Word για να το ελέγξετε—τα διαγράμματα θα πρέπει να είναι πλήρως λειτουργικά, επεξεργάσιμα και να φαίνονται ακριβώς όπως πριν.

## Πλήρες Παράδειγμα Λειτουργικού Κώδικα

Παρακάτω βρίσκεται το πλήρες, έτοιμο‑για‑εκτέλεση πρόγραμμα. Αντιγράψτε το σε μια εφαρμογή console, προσαρμόστε τις διαδρομές και πατήστε **F5**.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Saving;

namespace ExportChartsDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // 1️⃣ Load the source document containing charts
            string inputPath = @"C:\Docs\input.docx";
            Document document = new Document(inputPath);
            Console.WriteLine($"Loaded document: {inputPath}");

            // 2️⃣ Set save options to explicitly include charts
            DocxSaveOptions saveOptions = new DocxSaveOptions
            {
                ExportCharts = true   // This ensures charts are not stripped out
            };
            Console.WriteLine("Configured DocxSaveOptions to export charts.");

            // 3️⃣ Save the new file
            string outputPath = @"C:\Docs\charts.docx";
            document.Save(outputPath, saveOptions);
            Console.WriteLine($"Document saved with charts at: {outputPath}");

            // Verification hint
            Console.WriteLine("Open the output file in Word to confirm charts are present.");
        }
    }
}
```

**Αναμενόμενο αποτέλεσμα:** Όταν ανοίξετε το `charts.docx` στο Microsoft Word, κάθε διάγραμμα από το `input.docx` εμφανίζεται αμετάβλητο. Χωρίς ελλιπείς εικόνες, χωρίς σπασμένες αναφορές.

## Διαχείριση Συνηθισμένων Περιπτώσεων

| Κατάσταση | Τι Πρέπει Να Προσέξετε | Προτεινόμενη Λύση |
|-----------|------------------------|-------------------|
| **Το έγγραφο περιέχει ενσωματωμένα φύλλα Excel** | Τα διαγράμματα μπορεί να είναι συνδεδεμένα με εξωτερικά δεδομένα Excel. | Χρησιμοποιήστε `DocxSaveOptions.ExportEmbeddedExcelData = true` (διαθέσιμο σε νεότερες εκδόσεις) για να διατηρήσετε τα δεδομένα. |
| **Μεγάλα έγγραφα (> 100 MB)** | Η χρήση μνήμης αυξάνεται κατά τη φόρτωση. | Ορίστε `LoadOptions.LoadFormat = LoadFormat.Docx` και σκεφτείτε streaming με `DocumentBuilder` για επεξεργασία κατά βήματα. |
| **Χρειάζεστε μόνο συγκεκριμένα διαγράμματα** | Η εξαγωγή ολόκληρου του αρχείου είναι υπερβολική. | Επανάληψη μέσω `document.GetChildNodes(NodeType.Shape, true)` και φιλτράρισμα με `Shape.IsChart`. Στη συνέχεια κλωνοποιήστε αυτά τα σχήματα σε νέο `Document` πριν την αποθήκευση. |
| **Ο προορισμός είναι PDF** | Τα διαγράμματα μπορεί να αποδοθούν διαφορετικά. | Χρησιμοποιήστε `PdfSaveOptions` με `ExportCharts = true` (η σημαία λειτουργεί και για PDF). |

Αυτές οι παραλλαγές απαντούν στο ερώτημα “**εξαγωγή διαγραμμάτων από word**” σε διαφορετικά πλαίσια, διασφαλίζοντας ότι καλύπτεστε είτε αποθηκεύετε ξανά σε DOCX είτε μετατρέπετε σε άλλη μορφή.

## Συχνές Ερωτήσεις

**Ε: Λειτουργεί αυτό με παλαιότερα αρχεία `.doc`;**  
Α: Ναι. Το Aspose.Words μετατρέπει αυτόματα τη παλαιότερη δυαδική μορφή σε σύγχρονη δομή Open XML στη μνήμη, οπότε το `ExportCharts` ισχύει και εδώ.

**Ε: Τι γίνεται αν θέλω μόνο τις εικόνες των διαγραμμάτων, όχι ολόκληρο το έγγραφο;**  
Α: Μπορείτε να εξάγετε κάθε διάγραμμα ως εικόνα χρησιμοποιώντας `ChartRenderer`. Παράδειγμα: `chartRenderer.Save("chart.png", ImageFormat.Png);` Αυτό καλύπτει μια πιο στενή ανάγκη “πώς να εξάγετε διαγράμματα”.

**Ε: Υπάρχει θέμα αδειοδότησης;**  
Α: Το Aspose.Words είναι εμπορική βιβλιοθήκη. Για αξιολόγηση μπορείτε να χρησιμοποιήσετε προσωρινή άδεια· για παραγωγική χρήση απαιτείται πλήρης άδεια ώστε να αποφύγετε το υδατογράφημα αξιολόγησης.
