## Οδηγός Εφαρμογής

### Λειτουργία 1: Επαλήθευση κωδικού πρόσβασης κρυπτογραφημένου αρχείου Excel

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να ελέγξετε αν ο κωδικός πρόσβασης που παρέχεται για ένα κρυπτογραφημένο αρχείο Excel είναι σωστός. Χρησιμοποιεί το `FileFormatUtil.VerifyPassword` μέθοδος από το Aspose.Cells.

#### Βήμα προς βήμα εφαρμογή:

##### Βήμα 1: Ρύθμιση των καταλόγων και της ροής σας
Αρχικά, καθορίστε τον κατάλογο προέλευσης που περιέχει το κρυπτογραφημένο αρχείο Excel.

```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
FileStream fstream = new FileStream(SourceDir + "EncryptedBook1.xlsx", FileMode.Open);
```

##### Βήμα 2: Επαλήθευση του κωδικού πρόσβασης
Χρησιμοποιήστε το `VerifyPassword` μέθοδος για να ελέγξετε αν ο κωδικός πρόσβασης είναι έγκυρος.

```csharp
bool isPasswordValid = FileFormatUtil.VerifyPassword(fstream, "1234");
fstream.Close(); // Να κλείνετε πάντα το FileStream μετά τη χρήση.
```

##### Επεξήγηση παραμέτρων:
- **FileStream**Η ροή του αρχείου Excel σας.
- **σειρά**: Ο κωδικός πρόσβασης που θέλετε να επαληθεύσετε.

##### Επιστρεφόμενη τιμή:
- `true` αν ο κωδικός πρόσβασης είναι σωστός, διαφορετικά, `false`.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή και το όνομα του αρχείου είναι σωστά.
- Χειριστείτε εξαιρέσεις για περιπτώσεις όπως λανθασμένες διαδρομές ή προβλήματα δικαιωμάτων.

### Χαρακτηριστικό 2: Χειρισμός αρχείων με αντικείμενα ροής

#### Επισκόπηση
Η σωστή διαχείριση των αντικειμένων FileStream διασφαλίζει την αποτελεσματική χρήση των πόρων και αποτρέπει τις διαρροές δεδομένων. Αυτή η λειτουργία δείχνει πώς να χειρίζεστε υπεύθυνα τις ροές αρχείων σε εφαρμογές .NET.

#### Βήμα προς βήμα εφαρμογή:

##### Βήμα 1: Ανοίξτε ένα FileStream
Ανοίξτε τη ροή για να διαβάσετε το αρχείο Excel, φροντίζοντας να καθορίσετε το σωστό όνομα αρχείου.

```csharp
FileStream fstream = new FileStream(SourceDir + "EncryptedBook1.xlsx", FileMode.Open);
```

##### Βήμα 2: Υλοποίηση του μπλοκ Try-Finally
Να χρησιμοποιείτε πάντα ένα `try-finally` μπλοκ για να διασφαλιστεί ότι οι πόροι απελευθερώνονται σωστά.

```csharp
try
{
    // Εκτελέστε λειτουργίες στο FileStream.
}
finally
{
    if (fstream != null)
        fstream.Close();
}
```

### Βασικές επιλογές διαμόρφωσης:
- Χρήση `FileMode.Open` για την ανάγνωση υπαρχόντων αρχείων.
- Βεβαιωθείτε ότι τα ρέματα είναι κλειστά σε ένα `finally` μπλοκ για την αποτροπή διαρροών πόρων.
