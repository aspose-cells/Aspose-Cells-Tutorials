## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία ροής αρχείων και αρχικοποίηση βιβλίου εργασίας

Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε ροές αρχείων από αρχεία Excel, διευκολύνοντας την αποτελεσματική διαχείριση μεγάλων συνόλων δεδομένων.

#### Βήμα 1: Δημιουργήστε ένα FileStream
```csharp
using System.IO;
using Aspose.Cells;

string SourceDir = @"YOUR_SOURCE_DIRECTORY";

// Δημιουργία FileStream για το καθορισμένο αρχείο Excel
FileStream fstream = new FileStream(SourceDir + "book1.xls", FileMode.Open);
```
*Γιατί;* Η χρήση του FileStream σάς επιτρέπει να χειρίζεστε αρχεία αποτελεσματικά, ειδικά με μεγάλα σύνολα δεδομένων.

#### Βήμα 2: Αρχικοποίηση αντικειμένου βιβλίου εργασίας
```csharp
// Δημιουργία αντικειμένου βιβλίου εργασίας χρησιμοποιώντας το FileStream
Workbook excel = new Workbook(fstream);

// Κλείσιμο του FileStream για δωρεάν πόρους
fstream.Close();
```
*Εξήγηση*: Το `Workbook` Η κλάση αρχικοποιείται με τη ροή αρχείων, επιτρέποντάς σας να χειρίζεστε αρχεία Excel μέσω προγραμματισμού.

### Λειτουργία 2: Ρυθμίσεις προστασίας φύλλου εργασίας

Η προστασία των φύλλων εργασίας σας διασφαλίζει την ακεραιότητα των δεδομένων και περιορίζει τις μη εξουσιοδοτημένες αλλαγές.

#### Βήμα 1: Φόρτωση βιβλίου εργασίας και φύλλου εργασίας της Access
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";

// Δημιουργία αντικειμένου βιβλίου εργασίας ανοίγοντας το καθορισμένο αρχείο
Workbook excel = new Workbook(SourceDir + "book1.xls");

// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet worksheet = excel.Worksheets[0];
```
*Τι κάνει;* Αυτό το βήμα προετοιμάζει το φύλλο εργασίας σας για την εφαρμογή των ρυθμίσεων προστασίας.

#### Βήμα 2: Εφαρμογή ρυθμίσεων προστασίας
```csharp
// Εφαρμογή διαφόρων ρυθμίσεων προστασίας για τον περιορισμό των ενεργειών των χρηστών
worksheet.Protection.AllowDeletingColumn = false;
worksheet.Protection.AllowDeletingRow = false;
worksheet.Protection.AllowEditingContent = false;

// Επιτρέψτε συγκεκριμένες ενέργειες κατά την προστασία του φύλλου εργασίας
data cell formatting and hyperlink insertion are permitted.
worksheet.Protection.AllowFormattingCell = true;
worksheet.Protection.AllowInsertingHyperlink = true;

// Αποθήκευση βιβλίου εργασίας με ρυθμίσεις προστασίας
excel.Save(@"YOUR_OUTPUT_DIRECTORY\output.xls", SaveFormat.Excel97To2003);
```
*Εξήγηση*Αυτές οι ρυθμίσεις καθορίζουν τι μπορούν και τι δεν μπορούν να κάνουν οι χρήστες, παρέχοντας μια ισορροπία μεταξύ ασφάλειας και χρηστικότητας.
