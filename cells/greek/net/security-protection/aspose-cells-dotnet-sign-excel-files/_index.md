## Οδηγός Εφαρμογής

### Υπογραφή αρχείων Excel με ψηφιακές υπογραφές

Οι ψηφιακές υπογραφές διασφαλίζουν την αυθεντικότητα και την ακεραιότητα των αρχείων Excel σας. Δείτε πώς μπορείτε να εφαρμόσετε ψηφιακή υπογραφή χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Προετοιμασία του πιστοποιητικού σας

Βεβαιωθείτε ότι το πιστοποιητικό σας, το οποίο πρέπει να περιέχει ένα ιδιωτικό κλειδί, είναι έτοιμο. Μπορείτε να χρησιμοποιήσετε ένα `.pfx` ή ανακτήστε το από το Windows Certificate Store. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε ένα αρχείο PFX:
```csharp
X509Certificate2 cert = new X509Certificate2("path_to_your_certificate.pfx", "your_password");
```

#### Βήμα 2: Δημιουργία και εκχώρηση ψηφιακής υπογραφής

Δημιουργήστε ένα `DigitalSignature` αντικείμενο χρησιμοποιώντας το πιστοποιητικό σας και προσθέστε το σε ένα `DigitalSignatureCollection`Στη συνέχεια, εφαρμόστε αυτήν τη συλλογή στο βιβλίο εργασίας σας:
```csharp
// Αρχικοποίηση συλλογής ψηφιακών υπογραφών και υπογραφή του βιβλίου εργασίας
DigitalSignatureCollection dsc = new DigitalSignatureCollection();
DigitalSignature ds = new DigitalSignature(cert, "test for sign", DateTime.Now);
dsc.Add(ds);

Workbook wb = new Workbook(); // Δημιουργήστε ένα νέο βιβλίο εργασίας ή φορτώστε ένα υπάρχον
wb.SetDigitalSignature(dsc);  // Εφαρμογή ψηφιακών υπογραφών

// Αποθήκευση του υπογεγραμμένου βιβλίου εργασίας
wb.Save("output_signed_workbook.xlsx");
```

#### Βήμα 3: Επικύρωση ψηφιακών υπογραφών

Για να επαληθεύσετε εάν το αρχείο Excel σας είναι ψηφιακά υπογεγραμμένο και να επικυρώσετε αυτές τις υπογραφές:
```csharp
Workbook wb = new Workbook("output_signed_workbook.xlsx");

if (wb.IsDigitallySigned)
{
    Console.WriteLine("The workbook is digitally signed.");
}

DigitalSignatureCollection dsc = wb.GetDigitalSignature();
foreach (DigitalSignature dst in dsc)
{
    // Λεπτομέρειες εξόδου κάθε υπογραφής
    Console.WriteLine($"Comments: {dst.Comments}");
    Console.WriteLine($"SignTime: {dst.SignTime}");
    Console.WriteLine($"IsValid: {dst.IsValid}");
}
```
