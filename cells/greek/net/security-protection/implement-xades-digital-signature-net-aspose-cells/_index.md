## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει τα πάντα, ας δούμε πώς να εφαρμόζουμε τις ψηφιακές υπογραφές XAdES στα έγγραφά σας στο Excel.

### Βήμα 1: Φόρτωση του βιβλίου εργασίας σας

Αρχικά, φορτώστε το βιβλίο εργασίας που θέλετε να υπογράψετε χρησιμοποιώντας το Aspose.Cells.

```csharp
// Ορίστε τον κατάλογο και το αρχείο πηγής.
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook workbook = new Workbook(sourceDir + "sourceFile.xlsx");
```

**Εξήγηση**: Αυτό το απόσπασμα αρχικοποιεί ένα `Workbook` αντικείμενο με το αρχείο Excel προορισμού σας. Βεβαιωθείτε ότι η διαδρομή είναι σωστή για να αποφύγετε εξαιρέσεις.

### Βήμα 2: Δημιουργήστε μια ψηφιακή υπογραφή

Στη συνέχεια, δημιουργήστε μια παρουσία του `DigitalSignature`.

```csharp
// Ορίστε τον κωδικό πρόσβασης και τις λεπτομέρειες του αρχείου PFX.
string password = "pfxPassword";
string pfxFile = sourceDir + "pfxFile.pfx";

// Αρχικοποιήστε την ψηφιακή υπογραφή με το πιστοποιητικό σας.
DigitalSignature signature = new DigitalSignature(File.ReadAllBytes(pfxFile), password, "testXAdES", DateTime.Now);
```

**Παράμετροι**: 
- `File.ReadAllBytes(pfxFile)`Διαβάζει το περιεχόμενο του αρχείου PFX.
- `password`: Ο κωδικός πρόσβασης για την πρόσβαση στο αρχείο PFX σας.
- `"testXAdES"`: Μια περιγραφή ή αναγνωριστικό για την υπογραφή.
- `DateTime.Now`: Χρονοσήμανση της ψηφιακής υπογραφής.

### Βήμα 3: Ρύθμιση παραμέτρων και εφαρμογή υπογραφής

Ρυθμίστε τις παραμέτρους του τύπου XAdES και εφαρμόστε τον στο βιβλίο εργασίας.

```csharp
// Ορίστε τον τύπο XAdES και προσθέστε την υπογραφή σε μια συλλογή.
signature.XAdESType = XAdESType.XAdES;
DigitalSignatureCollection dsCollection = new DigitalSignatureCollection();
dsCollection.Add(signature);

// Εφαρμόστε τις ψηφιακές υπογραφές στο βιβλίο εργασίας.
workbook.SetDigitalSignature(dsCollection);
```

**Διαμόρφωση κλειδιού**: Το `XAdESType` μπορεί να προσαρμοστεί ανάλογα με τις ανάγκες συμμόρφωσής σας.

### Βήμα 4: Αποθήκευση του υπογεγραμμένου βιβλίου εργασίας

Τέλος, αποθηκεύστε το υπογεγραμμένο έγγραφο.

```csharp
// Ορίστε τον κατάλογο εξόδου και το όνομα αρχείου.
string outputDir = RunExamples.Get_OutputDirectory();
workbook.Save(outputDir + "XAdESSignatureSupport_out.xlsx");
```

**Σημείωμα**Βεβαιωθείτε ότι η διαδρομή εξόδου είναι προσβάσιμη για να αποφύγετε σφάλματα αποθήκευσης αρχείων.
