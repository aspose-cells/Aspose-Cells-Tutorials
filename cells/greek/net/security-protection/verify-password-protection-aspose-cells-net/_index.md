## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την επαλήθευση της προστασίας με κωδικό πρόσβασης σε φύλλα εργασίας.

### Επαλήθευση προστασίας φύλλου εργασίας

#### Επισκόπηση

Θα ελέγξουμε αν ένα φύλλο εργασίας προστατεύεται από κωδικό πρόσβασης και θα επαληθεύσουμε την ακρίβειά του χρησιμοποιώντας το Aspose.Cells για .NET.

#### Οδηγίες βήμα προς βήμα

**1. Φόρτωση του Βιβλίου Εργασίας**

Ξεκινήστε φορτώνοντας το αρχείο Excel:

```csharp
string sourceDir = "path_to_your_directory";
var book = new Workbook(sourceDir + "sampleVerifyPasswordUsedToProtectWorksheets.xlsx");
```
*Εξήγηση*: Το `Workbook` Η κλάση φορτώνει και χειρίζεται αρχεία Excel.

**2. Πρόσβαση στο Φύλλο Εργασίας**

Αποκτήστε πρόσβαση στο συγκεκριμένο φύλλο εργασίας για να επαληθεύσετε:

```csharp
var sheet = book.Worksheets[0];
```
*Εξήγηση*: Αυτό έχει πρόσβαση στο πρώτο φύλλο εργασίας με βάση το ευρετήριο.

**3. Ελέγξτε την κατάσταση προστασίας**

Προσδιορίστε εάν το φύλλο εργασίας προστατεύεται με κωδικό πρόσβασης:

```csharp
if (sheet.Protection.IsProtectedWithPassword)
{
    // Προχωρήστε στην επαλήθευση του κωδικού πρόσβασης
}
else
{
    Console.WriteLine("Worksheet is not protected.");
}
```
*Εξήγηση*: Το `IsProtectedWithPassword` Η ιδιότητα υποδεικνύει εάν υπάρχει προστασία.

**4. Επαληθεύστε τον κωδικό πρόσβασης**

Εάν προστατεύεται, ελέγξτε τον παρεχόμενο κωδικό πρόσβασης:

```csharp
if (sheet.Protection.VerifyPassword("1234"))
{
    Console.WriteLine("Specified password has matched");
}
else
{
    Console.WriteLine("Specified password has not matched");
}
```
*Εξήγηση*: `VerifyPassword` ελέγχει την ορθότητα του δοθέντος κωδικού πρόσβασης.
