## Οδηγός Εφαρμογής

### Έλεγχος προστασίας με κωδικό πρόσβασης φύλλου εργασίας

Αυτή η λειτουργία σάς επιτρέπει να προσδιορίσετε εάν οποιοδήποτε φύλλο εργασίας μέσα σε ένα αρχείο Excel προστατεύεται με κωδικό πρόσβασης.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας σας
Φορτώστε το βιβλίο εργασίας από το οποίο θέλετε να ελέγξετε την προστασία:
```csharp
// Κατάλογος πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Δημιουργήστε μια παρουσία του Βιβλίου Εργασίας και φορτώστε ένα υπολογιστικό φύλλο
var book = new Workbook(sourceDir + "sampleCheckIfPasswordProtected.xlsx");
```

#### Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας
Αποκτήστε πρόσβαση στο φύλλο εργασίας που θέλετε να ελέγξετε για προστασία:
```csharp
// Πρόσβαση στο προστατευμένο Φύλλο Εργασίας
var sheet = book.Worksheets[0];
```

#### Βήμα 3: Ελέγξτε για προστασία με κωδικό πρόσβασης
Προσδιορίστε εάν το φύλλο εργασίας προστατεύεται με κωδικό πρόσβασης χρησιμοποιώντας `IsProtectedWithPassword`:
```csharp
if (sheet.Protection.IsProtectedWithPassword)
{
    Console.WriteLine("Worksheet is Password Protected");
}
else
{
    Console.WriteLine("Worksheet is Not Password Protected");
}

Console.WriteLine("CheckIfPasswordProtected executed successfully.");
```

**Εξήγηση:**
- **Παράμετροι**: Το `Workbook` και `Worksheets` Οι κλάσεις διαχειρίζονται το περιεχόμενο του αρχείου Excel.
- **Επιστρεφόμενες τιμές**: Μια λογική τιμή που υποδεικνύει την κατάσταση προστασίας με κωδικό πρόσβασης.

### Συμβουλές αντιμετώπισης προβλημάτων

- Επαληθεύστε ότι το ευρετήριο του φύλλου εργασίας στο οποίο αποκτάτε πρόσβαση υπάρχει μέσα στο βιβλίο εργασίας σας.
