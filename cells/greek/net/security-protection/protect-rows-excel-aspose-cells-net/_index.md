## Οδηγός Εφαρμογής

### Ξεκλείδωμα στηλών
Αρχικά, ας ξεκλειδώσουμε όλες τις στήλες εκτός από αυτήν που θέλουμε να προστατεύσουμε. Αυτό διασφαλίζει ότι μόνο συγκεκριμένες γραμμές μπορούν να τροποποιηθούν.

#### Βήμα 1: Επανάληψη και ξεκλείδωμα στηλών

```csharp
// Ορισμός αντικειμένου στυλ για ξεκλείδωμα
Style style;
// Ορισμός σημαίας για την εφαρμογή στυλ
StyleFlag flag;

for (int i = 0; i <= 255; i++)
{
    // Λήψη του στυλ της τρέχουσας στήλης
    style = sheet.Cells.Columns[(byte)i].GetStyle();
    // Ορίστε το χαρακτηριστικό κλειδώματος σε false
    style.IsLocked = false;
    
    // Δημιουργήστε ένα νέο αντικείμενο StyleFlag
    flag = new StyleFlag { Locked = true };
    
    // Εφαρμογή του ξεκλειδωμένου στυλ σε όλες τις στήλες
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
}
```

### Κλείδωμα και προστασία συγκεκριμένων γραμμών
Στη συνέχεια, εστιάζουμε στην προστασία συγκεκριμένων γραμμών, αφήνοντας παράλληλα άλλες προσβάσιμες.

#### Βήμα 2: Κλείδωμα πρώτης σειράς

```csharp
// Αποκτήστε το στυλ της πρώτης σειράς
style = sheet.Cells.Rows[0].GetStyle();
// Ορίστε το χαρακτηριστικό κλειδώματος σε true
style.IsLocked = true;

// Εφαρμόστε τη ρύθμιση κλειδώματος χρησιμοποιώντας ένα StyleFlag
flag.Locked = true;
sheet.Cells.ApplyRowStyle(0, style, flag);
```

### Προστασία του Φύλλου Εργασίας
Τέλος, προστατέψτε το φύλλο εργασίας για να διασφαλίσετε ότι οι μη εξουσιοδοτημένοι χρήστες δεν μπορούν να παρακάμψουν τα κλειδώματα γραμμών.

#### Βήμα 3: Εφαρμογή προστασίας

```csharp
// Κλείδωμα όλων των στοιχείων στο φύλλο
sheet.Protect(ProtectionType.All);

// Αποθήκευση του βιβλίου εργασίας
wb.Save(dataDir + "output.out.xls", SaveFormat.Excel97To2003);
```
