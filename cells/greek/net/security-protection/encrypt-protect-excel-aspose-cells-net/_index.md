### Κρυπτογράφηση αρχείων Excel με Aspose.Cells για .NET
**Επισκόπηση:**
Κρυπτογραφήστε τα αρχεία Excel για να προστατεύσετε τις ευαίσθητες πληροφορίες από μη εξουσιοδοτημένη πρόσβαση. Αυτή η ενότητα δείχνει πώς να εφαρμόσετε διαφορετικούς τύπους κρυπτογράφησης χρησιμοποιώντας το Aspose.Cells.

#### Βήμα 1: Ρύθμιση του έργου σας και φόρτωση του βιβλίου εργασίας
```csharp
// Βεβαιωθείτε ότι έχετε ορίσει σωστά αυτές τις διαδρομές καταλόγου στο περιβάλλον σας.
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
string OutputDir = @"YOUR_OUTPUT_DIRECTORY";

Workbook workbook = new Workbook(SourceDir + "/Book1.xls");
```

#### Βήμα 2: Καθορισμός επιλογών κρυπτογράφησης
Επιλέξτε μεταξύ των τύπων κρυπτογράφησης XOR και Strong Cryptographic Provider:
```csharp
// Χρησιμοποιήστε κρυπτογράφηση XOR με μήκος κλειδιού 40.
workbook.SetEncryptionOptions(EncryptionType.XOR, 40);

// Εναλλακτικά, χρησιμοποιήστε ισχυρή κρυπτογράφηση RC4 με μήκος κλειδιού 128 bit.
workbook.SetEncryptionOptions(EncryptionType.StrongCryptographicProvider, 128);
```

#### Βήμα 3: Ορισμός κωδικού πρόσβασης αρχείου
```csharp
// Προστατέψτε το αρχείο Excel σας ορίζοντας έναν κωδικό πρόσβασης.
workbook.Settings.Password = "1234";
```

#### Βήμα 4: Αποθήκευση του κρυπτογραφημένου βιβλίου εργασίας
```csharp
// Αποθηκεύστε το κρυπτογραφημένο βιβλίο εργασίας σας σε έναν κατάλογο εξόδου.
workbook.Save(OutputDir + "/encryptedBook1.out.xls");
```

### Προστασία με κωδικό πρόσβασης για τροποποίηση με το Aspose.Cells
**Επισκόπηση:**
Αποτρέψτε τις μη εξουσιοδοτημένες τροποποιήσεις ορίζοντας έναν κωδικό πρόσβασης που απαιτείται για την επεξεργασία.

#### Βήμα 1: Φόρτωση του υπάρχοντος βιβλίου εργασίας
```csharp

```

```csharp
// Αποθηκεύστε το βιβλίο εργασίας σας με ενεργοποιημένη την προστασία τροποποίησης.

```
