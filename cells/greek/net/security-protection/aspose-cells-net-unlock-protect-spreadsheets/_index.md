## Οδηγός Εφαρμογής
Τώρα θα εξετάσουμε λεπτομερώς κάθε χαρακτηριστικό.

### Ξεκλείδωμα όλων των στηλών
Το ξεκλείδωμα όλων των στηλών επιτρέπει στους χρήστες να επεξεργάζονται οποιοδήποτε κελί μέσα σε αυτές τις στήλες, παρέχοντας ευελιξία κατά την επεξεργασία μεγάλων συνόλων δεδομένων.

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να ξεκλειδώσετε κάθε στήλη σε ένα φύλλο εργασίας χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήματα Υλοποίησης
**Βήμα 1: Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας**
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

Workbook wb = new Workbook();
Worksheet sheet = wb.Worksheets[0];
```

**Βήμα 2: Ξεκλείδωμα στηλών**
Επαναλάβετε τη διαδικασία σε κάθε στήλη, ορίστε το `IsLocked` ορίστε την ιδιότητα σε false και εφαρμόστε το στυλ.
```csharp
Style style;
StyleFlag flag;

for (int i = 0; i <= 255; i++) {
    style = sheet.Cells.Columns[(byte)i].Style;
    style.IsLocked = false;
    
    flag = new StyleFlag();
    flag.Locked = true;
    
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, flag);
}
```

#### Εξήγηση
- `style.IsLocked` ελέγχει την κατάσταση κλειδώματος της στήλης.
- `StyleFlag` Καθορίζει ποιες ιδιότητες θα εφαρμοστούν κατά τη διαμόρφωση.

### Κλείδωμα συγκεκριμένης γραμμής
Το κλείδωμα συγκεκριμένων γραμμών μπορεί να αποτρέψει τυχαίες επεξεργασίες σε κρίσιμες περιοχές δεδομένων, όπως κεφαλίδες ή τύπους.

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στο κλείδωμα μόνο της πρώτης γραμμής στο φύλλο εργασίας σας.

#### Βήματα Υλοποίησης
**Βήμα 1: Λήψη στυλ πρώτης γραμμής**
```csharp
Style style = sheet.Cells.Rows[0].GetStyle();
style.IsLocked = true;
```

**Βήμα 2: Εφαρμογή κλειδωμένου στυλ στη γραμμή**
```csharp
flag = new StyleFlag();
flag.Locked = true;

sheet.Cells.ApplyRowStyle(0, style, flag);
```

#### Εξήγηση
- Το κλείδωμα επιτυγχάνεται ρυθμίζοντας `IsLocked` να είναι αληθές και να το εφαρμόζουμε με `ApplyRowStyle`.

### Προστασία ενός φύλλου εργασίας
Η προστασία διασφαλίζει ότι η δομή του φύλλου εργασίας παραμένει άθικτη, προστατεύοντας την ακεραιότητα των δεδομένων.

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να προστατεύσετε ένα ολόκληρο φύλλο εργασίας χρησιμοποιώντας διάφορους τύπους προστασίας.

#### Βήματα Υλοποίησης
**Βήμα 1: Εφαρμογή προστασίας**
```csharp
sheet.Protect(ProtectionType.All);
```

**Βήμα 2: Αποθήκευση βιβλίου εργασίας**
```csharp
wb.Save(outputDir + "output.out.xls", SaveFormat.Excel97To2003);
```

#### Εξήγηση
- `Protect` Η μέθοδος προστατεύει το φύλλο εργασίας από μη εξουσιοδοτημένες αλλαγές.
- Επιλέξτε το κατάλληλο `ProtectionType` με βάση τις ανάγκες σας.
