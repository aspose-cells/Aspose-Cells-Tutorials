## Οδηγός Εφαρμογής

### Κατάργηση προστασίας φύλλου εργασίας χωρίς κωδικό πρόσβασης

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο Excel, να αποκτήσετε πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας και να καταργήσετε την προστασία του, ακόμη και αν ο κωδικός πρόσβασης είναι άγνωστος.

#### Βήμα προς βήμα εφαρμογή:

**1. Φορτώστε το αρχείο Excel**

Αρχικά, φορτώστε το βιβλίο εργασίας σας από τον κατάλογο προέλευσης.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/book1.xls");
```
*Εξήγηση:* Αυτή η γραμμή αρχικοποιεί ένα `Workbook` αντικείμενο φορτώνοντας ένα υπάρχον αρχείο Excel.

**2. Πρόσβαση και κατάργηση προστασίας του φύλλου εργασίας**

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας και καταργήστε την προστασία του.
```csharp
Worksheet worksheet = workbook.Worksheets[0];
worksheet.Unprotect();
```
*Εξήγηση:* Με πρόσβαση `Worksheets[0]`, ανακτάτε το πρώτο φύλλο. Το `Unprotect()` Η μέθοδος καταργεί οποιαδήποτε προστασία, επιτρέποντας τροποποιήσεις.

**3. Αποθήκευση του βιβλίου εργασίας**

Τέλος, αποθηκεύστε το μη προστατευμένο βιβλίο εργασίας στον επιθυμητό κατάλογο.
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/output.out.xls", SaveFormat.Excel97To2003);
```
*Εξήγηση:* Αυτή η γραμμή αποθηκεύει το βιβλίο εργασίας σε μορφή Excel 97-2003. Μπορείτε να επιλέξετε άλλες μορφές που υποστηρίζονται από το Aspose.Cells.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές.
- Ελέγξτε για δικαιώματα ανάγνωσης/εγγραφής σε καταλόγους.

### Αποθήκευση αρχείου Excel σε συγκεκριμένη μορφή

**Επισκόπηση:** Μάθετε πώς να αποθηκεύετε ένα αρχείο Excel χρησιμοποιώντας μια συγκεκριμένη μορφή, η οποία είναι ιδιαίτερα χρήσιμη όταν αντιμετωπίζετε παλαιότερα συστήματα ή προβλήματα συμβατότητας.

#### Βήμα προς βήμα εφαρμογή:

**1. Φόρτωση του Βιβλίου Εργασίας**

Παρόμοια με τη λειτουργία κατάργησης προστασίας:
```csharp
Workbook workbook = new Workbook(SourceDir + "/book1.xls");
```

**2. Αποθήκευση στην επιθυμητή μορφή**

Καθορίστε τη μορφή κατά τη λειτουργία αποθήκευσης.
```csharp
workbook.Save(outputDir + "/output.out.xls", SaveFormat.Excel97To2003);
```
*Εξήγηση:* `SaveFormat` καθορίζει τον τύπο αρχείου εξόδου, διασφαλίζοντας τη συμβατότητα με παλαιότερες εκδόσεις του Excel.
