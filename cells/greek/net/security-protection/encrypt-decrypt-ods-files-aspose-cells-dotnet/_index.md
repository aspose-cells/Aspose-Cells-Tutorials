**Τι θα μάθετε:**
- Κρυπτογράφηση ενός αρχείου ODS με κωδικό πρόσβασης.
- Αποκρυπτογράφηση προηγουμένως κρυπτογραφημένων αρχείων ODS.

## Οδηγός Εφαρμογής

### Κρυπτογράφηση αρχείου ODS

Η κρυπτογράφηση ενός αρχείου ODS διασφαλίζει ότι μόνο εξουσιοδοτημένοι χρήστες έχουν πρόσβαση στο περιεχόμενό του. Δείτε πώς μπορείτε να το πετύχετε αυτό χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το αρχείο ODS πηγής σας σε ένα `Workbook` αντικείμενο:

```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/Book1.ods");
```

#### Βήμα 2: Ορισμός προστασίας με κωδικό πρόσβασης

Προστατέψτε το βιβλίο εργασίας με κωδικό πρόσβασης:

```csharp
workbook.Settings.Password = "1234"; // Επιλέξτε τον επιθυμητό κωδικό πρόσβασης
```
Ο `Settings.Password` Η ιδιότητα ορίζει έναν κωδικό πρόσβασης για την προστασία του αρχείου, διασφαλίζοντας ότι οι μη εξουσιοδοτημένοι χρήστες δεν μπορούν να το ανοίξουν.

#### Βήμα 3: Αποθήκευση του κρυπτογραφημένου αρχείου

Τέλος, αποθηκεύστε το κρυπτογραφημένο ODS με ένα νέο όνομα αρχείου:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/encryptedBook1.out.ods");
```

### Αποκρυπτογράφηση αρχείου ODS

Η αποκρυπτογράφηση είναι απαραίτητη όταν χρειάζεται να αποκτήσετε πρόσβαση ή να τροποποιήσετε δεδομένα που είχαν ασφαλιστεί προηγουμένως.

#### Βήμα 1: Ορισμός επιλογών φόρτωσης με κωδικό πρόσβασης

Καθορίστε τις επιλογές φόρτωσης, συμπεριλαμβανομένου του κωδικού πρόσβασης που χρησιμοποιείται κατά την κρυπτογράφηση:

```csharp
OdsLoadOptions loadOptions = new OdsLoadOptions();
loadOptions.Password = "1234"; // Χρησιμοποιήστε τον ίδιο κωδικό πρόσβασης όπως για την κρυπτογράφηση
```
Ο `OdsLoadOptions` Η κλάση διευκολύνει τη φόρτωση κρυπτογραφημένων αρχείων παρέχοντας τα απαραίτητα διαπιστευτήρια αποκρυπτογράφησης.

#### Βήμα 2: Φόρτωση του κρυπτογραφημένου βιβλίου εργασίας

Φορτώστε το κρυπτογραφημένο βιβλίο εργασίας σας χρησιμοποιώντας αυτές τις επιλογές:

```csharp
Workbook encryptedWorkbook = new Workbook(SourceDir + "/encryptedBook1.out.ods", loadOptions);
```

#### Βήμα 3: Κατάργηση προστασίας και κατάργηση κρυπτογράφησης

Καταργήστε την προστασία του αρχείου και καταργήστε τον κωδικό πρόσβασής του:

```csharp
encryptedWorkbook.Unprotect("1234"); // Χρησιμοποιήστε τον ίδιο κωδικό πρόσβασης για να καταργήσετε την προστασία
encryptedWorkbook.Settings.Password = null;
```
Αυτό το βήμα διασφαλίζει ότι οποιαδήποτε επακόλουθη πρόσβαση ή τροποποίηση δεν απαιτεί κωδικό πρόσβασης.

#### Βήμα 4: Αποθηκεύστε το αποκρυπτογραφημένο αρχείο

Αποθηκεύστε το αποκρυπτογραφημένο βιβλίο εργασίας σας με νέο όνομα:

```csharp
encryptedWorkbook.Save(outputDir + "/decryptedBook1.out.ods");
```
