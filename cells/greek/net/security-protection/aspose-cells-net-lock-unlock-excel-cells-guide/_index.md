## Οδηγός Εφαρμογής

### Δημιουργία και διαμόρφωση βιβλίων εργασίας (Λειτουργία 1)

Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα νέο βιβλίο εργασίας και να ορίσετε στυλ φύλλων εργασίας.

#### Επισκόπηση
Η δημιουργία ενός βιβλίου εργασίας είναι το πρώτο βήμα στη διαχείριση αρχείων Excel μέσω προγραμματισμού. Μπορείτε να το ρυθμίσετε εφαρμόζοντας στυλ, κλειδώνοντας κελιά ή ορίζοντας επίπεδα προστασίας.

#### Βήμα προς βήμα εφαρμογή

##### Δημιουργία νέου βιβλίου εργασίας

Ξεκινήστε αρχικοποιώντας ένα `Workbook` αντικείμενο:

```csharp
// Αρχικοποιήστε ένα νέο βιβλίο εργασίας.
Workbook wb = new Workbook();
```

##### Αποκτήστε το πρώτο φύλλο εργασίας

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας για να ξεκινήσετε τις τροποποιήσεις:

```csharp
// Αποκτήστε το πρώτο φύλλο εργασίας.
Worksheet sheet = wb.Worksheets[0];
```

##### Εφαρμογή στυλ και ξεκλείδωμα στηλών

Ορίστε και εφαρμόστε στυλ για να ξεκλειδώσετε στήλες, διασφαλίζοντας ευελιξία στη σχεδίαση του βιβλίου εργασίας σας:

```csharp
Style style = new Style { IsLocked = false };
StyleFlag styleflag = new StyleFlag { Locked = true };

// Ξεκλειδώστε όλες τις στήλες.
for (int i = 0; i <= 255; i++) {
    sheet.Cells.Columns[(byte)i].ApplyStyle(style, styleflag);
}
```

##### Κλείδωμα συγκεκριμένων κελιών

Κλείδωμα συγκεκριμένων κελιών για την προστασία ευαίσθητων πληροφοριών:

```csharp
sheet.Cells["A1"].SetStyle(new Style { IsLocked = true });
sheet.Cells["B1"].SetStyle(new Style { IsLocked = true });
sheet.Cells["C1"].SetStyle(new Style { IsLocked = true });
```

##### Προστατέψτε το φύλλο εργασίας

Τέλος, εφαρμόστε προστασία φύλλου εργασίας για να ασφαλίσετε τα δεδομένα σας:

```csharp
// Εφαρμόστε πλήρη προστασία.
sheet.Protect(ProtectionType.All);

// Αποθηκεύστε το βιβλίο εργασίας.
wb.Save(outputDir + "/output.xls", SaveFormat.Excel97To2003);
```

### Κλείδωμα και ξεκλείδωμα κελιών (Λειτουργία 2)

Αυτή η λειτουργία δείχνει πώς να κλειδώνετε ή να ξεκλειδώνετε επιλεκτικά κελιά μέσα σε ένα φύλλο εργασίας.

#### Επισκόπηση
Ελέγχοντας την πρόσβαση σε κελιά, μπορείτε να διαχειριστείτε την ακεραιότητα των δεδομένων, επιτρέποντας παράλληλα τροποποιήσεις όπου χρειάζεται.

#### Βήμα προς βήμα εφαρμογή

##### Ξεκλείδωμα όλων των στηλών αρχικά

Ξεκινήστε ξεκλειδώνοντας όλες τις στήλες για μέγιστη ευελιξία:

```csharp
Style unlockStyle = new Style { IsLocked = false };
StyleFlag unlockStyleFlag = new StyleFlag { Locked = true };

// Εφαρμόστε το στυλ ξεκλειδώματος σε όλες τις στήλες.
for (int i = 0; i <= 255; i++) {
    sheet.Cells.Columns[(byte)i].ApplyStyle(unlockStyle, unlockStyleFlag);
}
```

##### Κλείδωμα συγκεκριμένων κελιών

Ορίστε και εφαρμόστε στυλ για να κλειδώσετε συγκεκριμένα κελιά:

```csharp
Style lockStyle = new Style { IsLocked = true };

// Κλείδωμα συγκεκριμένων κελιών.
sheet.Cells["A1"].SetStyle(lockStyle);
sheet.Cells["B1"].SetStyle(lockStyle);
sheet.Cells["C1"].SetStyle(lockStyle);

// Αποθηκεύστε το τροποποιημένο βιβλίο εργασίας.
wb.Save(outputDir + "/output_locked.xls", SaveFormat.Excel97To2003);
```
