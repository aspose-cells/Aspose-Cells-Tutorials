Ας εξετάσουμε τις προϋποθέσεις πριν ξεκινήσουμε!

## Οδηγός Εφαρμογής

### Προσθήκη ιδιοτήτων εγγράφου σε αρχείο Excel

**Επισκόπηση:**
Η προσθήκη προσαρμοσμένων ιδιοτήτων επιτρέπει την ενσωμάτωση μεταδεδομένων απευθείας στα αρχεία Excel σας, βελτιώνοντας την οργάνωση και τη χρηστικότητά τους.

#### Βήμα 1: Φόρτωση του υπάρχοντος αρχείου Excel

Φορτώστε το αρχείο Excel σας σε ένα `Workbook` αντικείμενο. Καθορίστε τη διαδρομή του καταλόγου προέλευσης όπου βρίσκεται το αρχείο Excel σας.
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "/sample-document-properties.xlsx");
```

#### Βήμα 2: Πρόσβαση στις Ιδιότητες Προσαρμοσμένου Εγγράφου

Ανάκτηση της συλλογής προσαρμοσμένων ιδιοτήτων εγγράφου από το βιβλίο εργασίας:
```csharp
CustomDocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```

#### Βήμα 3: Προσθήκη νέας ιδιότητας

Προσθέστε μια νέα ιδιότητα με το όνομα "Publisher" με την τιμή "Aspose":
```csharp
customProperties.Add("Publisher", "Aspose");
```

Αυτό το βήμα δείχνει πώς να προσαρμόσετε τα μεταδεδομένα με βάση τις απαιτήσεις σας.

#### Βήμα 4: Αποθήκευση αλλαγών

Τέλος, αποθηκεύστε το τροποποιημένο βιβλίο εργασίας σε έναν κατάλογο εξόδου:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "/out_sample-document-properties.xlsx");
```

### Ανάκτηση ιδιοτήτων εγγράφου από αρχείο Excel

**Επισκόπηση:**
Η ανάκτηση προσαρμοσμένων ιδιοτήτων εγγράφου είναι ζωτικής σημασίας για την εξαγωγή μεταδεδομένων και την κατανόηση του περιβάλλοντος του αρχείου.

#### Βήμα 1: Φόρτωση του αρχείου Excel

Φορτώστε το βιβλίο εργασίας σας, όπως ακριβώς και με την προσθήκη ιδιοτήτων:
```csharp
Workbook workbook = new Workbook(SourceDir + "/sample-document-properties.xlsx");
```

#### Βήμα 2: Πρόσβαση στις Ιδιότητες Προσαρμοσμένου Εγγράφου

Αποκτήστε πρόσβαση στη συλλογή προσαρμοσμένων ιδιοτήτων εγγράφων όπως πριν:
```csharp
CustomDocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;
```

#### Επανάληψη πάνω από ιδιότητες

Επαναλάβετε κάθε ιδιότητα, εμφανίζοντας το όνομα και την τιμή της. Αυτό βοηθά στην κατανόηση των ενσωματωμένων μεταδεδομένων.
```csharp
foreach (var property in customProperties)
{
    Console.WriteLine("Name: " + property.Name);
    Console.WriteLine("Value: " + property.Value);
}
```
