## Φόρτωση βιβλίου εργασίας με κωδικό πρόσβασης
### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να ανοίξετε ένα κρυπτογραφημένο αρχείο Excel χρησιμοποιώντας έναν καθορισμένο κωδικό πρόσβασης. Είναι απαραίτητο κατά τον χειρισμό προστατευμένων βιβλίων εργασίας που περιέχουν ευαίσθητα δεδομένα.
### Βήματα Υλοποίησης:
#### 1. Καθορίστε τον Κατάλογο Πηγών
Προσδιορίστε πού αποθηκεύονται τα αρχεία Excel. Αυτή η διαδρομή καταλόγου θα χρησιμοποιηθεί για τον εντοπισμό και τη φόρτωση του βιβλίου εργασίας.
```csharp
string SourceDir = @"YOUR_SOURCE_DIRECTORY";
```
#### 2. Δημιουργήστε το LoadOptions και ορίστε τον κωδικό πρόσβασης
Αρχικοποίηση `LoadOptions` και ορίστε τον κωδικό πρόσβασης που απαιτείται για το άνοιγμα του κρυπτογραφημένου αρχείου.
```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.Password = "1234"; // Χρησιμοποιήστε τον πραγματικό σας κωδικό πρόσβασης εδώ
```
#### 3. Ανοίξτε το κρυπτογραφημένο αρχείο Excel
Χρησιμοποιήστε το `Workbook` κλάση με τις καθορισμένες επιλογές φόρτωσης για πρόσβαση στο αρχείο.
```csharp
Workbook workbook = new Workbook(SourceDir + "/sampleCheckPasswordToModify.xlsx", loadOptions);
```
**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι ο κωδικός πρόσβασης είναι σωστός και ταιριάζει με αυτόν που χρησιμοποιήθηκε για την κρυπτογράφηση.
- Επαληθεύστε ότι η διαδρομή του αρχείου είναι ακριβής και προσβάσιμη από το περιβάλλον της εφαρμογής σας.
## Επικύρωση κωδικού πρόσβασης για τροποποίηση βιβλίου εργασίας
### Επισκόπηση
Μόλις φορτωθεί ένα βιβλίο εργασίας, ίσως χρειαστεί να ελέγξετε αν ένας δεδομένος κωδικός πρόσβασης επιτρέπει τροποποιήσεις. Αυτή η λειτουργία διασφαλίζει ότι μόνο εξουσιοδοτημένοι χρήστες μπορούν να τροποποιήσουν τα προστατευμένα βιβλία εργασίας.
### Βήματα Υλοποίησης:
#### 1. Ανοίξτε το αρχείο Excel με το LoadOptions
Υποθέτοντας ότι οι επιλογές φόρτωσης έχουν ήδη οριστεί από το προηγούμενο βήμα:
```csharp
Workbook workbook = new Workbook(SourceDir + "/sampleCheckPasswordToModify.xlsx", loadOptions);
```
#### 2. Επικύρωση κωδικών πρόσβασης τροποποίησης
Χρήση `ValidatePassword` για να ελέγξετε εάν συγκεκριμένοι κωδικοί πρόσβασης επιτρέπουν τροποποιήσεις.
```csharp
bool isCorrectPassword1 = workbook.Settings.WriteProtection.ValidatePassword("567");
bool isCorrectPassword2 = workbook.Settings.WriteProtection.ValidatePassword("5678");
```
**Βασικές σκέψεις:**
- Μόνο οι έγκυροι κωδικοί πρόσβασης τροποποίησης θα επιστρέψουν την τιμή true.
- Βεβαιωθείτε ότι η εφαρμογή σας χειρίζεται τις ψευδείς επικυρώσεις με ομαλό τρόπο, για να αποτρέψετε απόπειρες μη εξουσιοδοτημένης πρόσβασης.
