## Οδηγός Εφαρμογής

### Χρόνος Υπολογισμού Δοκιμής με Αναδρομική Επιλογή

Αυτή η λειτουργία δείχνει πώς η ενεργοποίηση ή η απενεργοποίηση των αναδρομικών υπολογισμών επηρεάζει την απόδοση.

#### Επισκόπηση

Η κατανόηση του αντίκτυπου της αναδρομής στις λειτουργίες υπολογισμού μπορεί να βελτιώσει σημαντικά την αποτελεσματικότητα της εφαρμογής σας. Σε αυτήν την ενότητα, θα εξερευνήσουμε τη μέτρηση των χρόνων υπολογισμού χρησιμοποιώντας το Aspose.Cells για .NET.

##### Βήμα 1: Ορισμός καταλόγου πηγής
Ξεκινήστε καθορίζοντας πού βρίσκεται το αρχείο του βιβλίου εργασίας σας:

```csharp
string sourceFilePath = SourceDir + "/sampleDecreaseCalculationTime.xlsx";
```

##### Βήμα 2: Φόρτωση βιβλίου εργασίας
Φόρτωση του βιβλίου εργασίας από την καθορισμένη διαδρομή:

```csharp
Workbook wb = new Workbook(sourceFilePath);
```

##### Βήμα 3: Φύλλο εργασίας Access
Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας σας:

```csharp
Worksheet ws = wb.Worksheets[0];
```

##### Βήμα 4: Ρύθμιση παραμέτρων επιλογών υπολογισμού
Δημιουργήστε μια παρουσία του `CalculationOptions` και ορίστε την αναδρομική επιλογή με βάση την εισαγωγή δεδομένων από τον χρήστη.

```csharp
CalculationOptions opts = new CalculationOptions();
opts.Recursive = rec;
```

Αυτή η παράμετρος καθορίζει εάν οι αλλαγές σε ένα κελί θα ενεργοποιήσουν επανυπολογισμούς των εξαρτώμενων κελιών αναδρομικά.

##### Βήμα 5: Μέτρηση χρόνου υπολογισμού
Χρησιμοποιήστε ένα χρονόμετρο για να μετρήσετε πόσο χρόνο χρειάζεται για να εκτελέσετε υπολογισμούς:

```csharp
Stopwatch sw = new Stopwatch();
sw.Start();

for (int i = 0; i < 1000000; i++)
{
    ws.Cells["A1"].Calculate(opts);
}

sw.Stop();
long estimatedTimeInSeconds = sw.ElapsedMilliseconds / 1000;
```

Αυτός ο βρόχος υπολογίζει ξανά την τιμή του κελιού A1 ένα εκατομμύριο φορές, επιτρέποντάς σας να παρατηρήσετε διαφορές στην απόδοση με ενεργοποιημένους ή απενεργοποιημένους αναδρομικούς υπολογισμούς.
