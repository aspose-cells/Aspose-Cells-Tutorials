## Οδηγός Εφαρμογής

Θα χωρίσουμε αυτόν τον οδηγό σε δύο κύριες ενότητες με βάση τη λειτουργικότητα.

### Αλλαγή συστήματος ημερομηνιών βιβλίου εργασίας Excel

#### Επισκόπηση
Αυτή η δυνατότητα αλλάζει το σύστημα ημερομηνιών ενός βιβλίου εργασίας του Excel από την προεπιλεγμένη ημερομηνία (1899) σε 1904, κάτι που είναι απαραίτητο για λόγους συμβατότητας ή για συγκεκριμένες τοπικές απαιτήσεις.

##### Βήμα προς βήμα εφαρμογή:

**1. Ανοίξτε το αρχείο Excel**
```csharp
string SourceDir = "YOUR_SOURCE_DIRECTORY";
Workbook workbook = new Workbook(SourceDir + "sampleImplement1904DateSystem.xlsx");
```
Εδώ, `Workbook` αρχικοποιείται με μια υπάρχουσα διαδρομή αρχείου για τη φόρτωση του εγγράφου Excel.

**2. Αλλαγή του συστήματος ημερομηνιών**
```csharp
workbook.Settings.Date1904 = true;
```
Αυτή η γραμμή ορίζει το σύστημα ημερομηνιών του βιβλίου εργασίας σε 1904 τροποποιώντας το `Date1904` ιδιοκτησία.

**3. Αποθηκεύστε το ενημερωμένο βιβλίο εργασίας**
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.Save(outputDir + "outputImplement1904DateSystem_1904DateSystem.xlsx");
```
Το βιβλίο εργασίας αποθηκεύεται με νέο όνομα, το οποίο αντικατοπτρίζει την ενημερωμένη διαμόρφωση συστήματος ημερομηνιών.

### Φόρτωση και αποθήκευση βιβλίου εργασίας

#### Επισκόπηση
Μάθετε πώς να φορτώνετε αποτελεσματικά ένα αρχείο Excel από έναν κατάλογο και να το αποθηκεύετε αλλού χρησιμοποιώντας το Aspose.Cells.

##### Βήμα προς βήμα εφαρμογή:

**1. Ανοίξτε το αρχείο Excel**
```csharp
Workbook workbook = new Workbook(SourceDir + "sampleImplement1904DateSystem.xlsx");
```
Αυτό το βήμα είναι παρόμοιο με το προηγούμενο παράδειγμά μας, όπου ανοίγουμε το βιβλίο εργασίας για χειρισμό.

**2. Αποθήκευση του Βιβλίου Εργασίας**
```csharp
workbook.Save(outputDir + "outputSaveWorkbook.xlsx");
```
Εδώ, το βιβλίο εργασίας αποθηκεύεται σε μια νέα τοποθεσία με ένα καθορισμένο όνομα αρχείου.
