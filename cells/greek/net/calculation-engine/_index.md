# Μηχανή υπολογισμού Excel για εφαρμογές .NET

## Επισκόπηση

Το Aspose.Cells για .NET προσφέρει μια ισχυρή μηχανή υπολογισμών που επιτρέπει στους προγραμματιστές να χειρίζονται πολύπλοκους υπολογισμούς τύπων μέσα σε υπολογιστικά φύλλα του Excel. Αυτό το ισχυρό στοιχείο σάς επιτρέπει να διαμορφώνετε τη συμπεριφορά υπολογισμών, να εφαρμόζετε προσαρμοσμένη λογική υπολογισμών και να βελτιστοποιείτε την απόδοση για μεγάλα βιβλία εργασίας.

## Βασικά χαρακτηριστικά

- **Μηχανές Προσαρμοσμένων Υπολογισμών**: Υλοποιήστε τη δική σας λογική υπολογισμού για εξειδικευμένους τύπους
- **Παρακολούθηση Υπολογισμών**: Υπολογισμοί τύπων παρακολούθησης και ελέγχου για βελτιστοποίηση απόδοσης
- **Επιλογές Αναδρομικού Υπολογισμού**: Ρύθμιση παραμέτρων για τον τρόπο χειρισμού των κυκλικών αναφορών
- **Ρύθμιση παραμέτρων συστήματος ημερομηνίας**: Εναλλαγή μεταξύ συστημάτων ημερομηνίας 1900 και 1904
- **Κυκλική ανίχνευση αναφοράς**: Προσδιορισμός και επίλυση κυκλικών εξαρτήσεων σε τύπους
- **Βελτιστοποίηση απόδοσης**: Βελτιστοποίηση ρυθμίσεων υπολογισμού για μεγάλα βιβλία εργασίας
