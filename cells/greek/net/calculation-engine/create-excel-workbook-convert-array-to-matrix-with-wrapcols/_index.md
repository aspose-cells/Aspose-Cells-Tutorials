# Δημιουργία Βιβλίου Εργασίας Excel – Μετατροπή Πίνακα σε Μήτρα με WRAPCOLS

Έχετε χρειαστεί ποτέ να **δημιουργήσετε βιβλίο εργασίας Excel** από το μηδέν και ξαφνικά να κολλήσετε όταν προσπαθείτε να επανασχηματίσετε δεδομένα; Δεν είστε μόνοι. Πολλοί προγραμματιστές ξεκινούν με έναν απλό πίνακα, μόνο για να ανακαλύψουν ότι το Excel απαιτεί μια σωστή 2‑Δ περιοχή.  

Σε αυτό το tutorial θα σας δείξουμε ακριβώς πώς να **δημιουργήσετε βιβλίο εργασίας Excel**, να χρησιμοποιήσετε τη λειτουργία `WRAPCOLS` για **μετατροπή πίνακα σε μήτρα**, **να εξαναγκάσετε τον υπολογισμό του βιβλίου εργασίας**, και τέλος **να αποθηκεύσετε το βιβλίο εργασίας ως XLSX**. Στο τέλος θα έχετε ένα εκτελέσιμο πρόγραμμα C# που κάνει όλα αυτά με λίγες μόνο γραμμές κώδικα.

> **Συμβουλή επαγγελματία:** Το ίδιο μοτίβο λειτουργεί με μεγαλύτερα σύνολα δεδομένων, ώστε μπορείτε να κλιμακώσετε από ένα demo 4‑στοιχείων σε χιλιάδες γραμμές χωρίς να αλλάξετε τη βασική λογική.

## Τι Θα Χρειαστείτε

- .NET 6 ή νεότερο (οποιοδήποτε πρόσφατο runtime .NET λειτουργεί)
- Aspose.Cells for .NET (η βιβλιοθήκη που παρέχει `Workbook`, `Worksheet`, κ.λπ.)
- Έναν επεξεργαστή κώδικα ή IDE (Visual Studio, VS Code, Rider – επιλέξτε το αγαπημένο σας)
- Δικαιώματα εγγραφής σε φάκελο όπου θα αποθηκευτεί το αρχείο εξόδου

Δεν απαιτούνται πρόσθετα πακέτα NuGet πέρα από το Aspose.Cells· το υπόλοιπο του κώδικα είναι καθαρό C#.

## Βήμα 1 – Δημιουργία Βιβλίου Εργασίας Excel (Κύρια Λέξη‑Κλειδί σε Δράση)

Για αρχή, δημιουργούμε ένα νέο αντικείμενο `Workbook` και παίρνουμε το πρώτο φύλλο εργασίας. Αυτό αποτελεί τη βάση για όλα τα επόμενα.

```csharp
using Aspose.Cells;

class Program
{
    static void Main()
    {
        // Step 1: Create a new workbook and get the first worksheet
        Workbook workbook = new Workbook();               // creates a blank Excel file in memory
        Worksheet ws = workbook.Worksheets[0];           // default sheet is named "Sheet1"
```

**Γιατί είναι σημαντικό:**  
Η δημιουργία βιβλίου εργασίας προγραμματιστικά σας δίνει πλήρη έλεγχο πάνω στη μορφοποίηση, τους τύπους και την εισαγωγή δεδομένων πριν το αρχείο αγγίξει το δίσκο. Επίσης σημαίνει ότι μπορείτε να παράγετε αρχεία σε διακομιστή χωρίς ποτέ να ανοίξετε το Excel.

## Βήμα 2 – Εισαγωγή Τύπου WRAPCOLS για Μετατροπή Πίνακα σε Μήτρα

`WRAPCOLS` είναι μια ενσωματωμένη λειτουργία του Excel που επανασχηματίζει έναν μονοδιάστατο πίνακα σε μια μήτρα με καθορισμένο αριθμό στηλών. Εδώ μετατρέπουμε το `{1,2,3,4}` σε διάταξη 2 στηλών.

```csharp
        // Step 2: Insert a WRAPCOLS formula that converts a 1‑D array into a 2‑column matrix
        ws.Cells["A1"].Formula = "=WRAPCOLS({1,2,3,4},2)";
```

**Πώς λειτουργεί:**  
- Το πρώτο όρισμα `{1,2,3,4}` είναι ένας ενσωματωμένος κυριολεκτικός πίνακας.  
- Το δεύτερο όρισμα `2` λέει στο Excel να «τυλίξει» τις τιμές σε δύο στήλες, με αποτέλεσμα:

| A | B |
|---|---|
| 1 | 2 |
| 3 | 4 |

Αν χρειάζεστε διαφορετικό σχήμα, απλώς αλλάξτε το δεύτερο παράμετρο – `WRAPCOLS({1,2,3,4,5,6},3)` θα σας δώσει τρεις στήλες.

## Βήμα 3 – Εξαναγκασμός Υπολογισμού του Βιβλίου Εργασίας ώστε ο Τύπος να Υλοποιηθεί

Από προεπιλογή, το Aspose.Cells αξιολογεί τους τύπους αργά. Για να βεβαιωθούμε ότι η μήτρα εμφανίζεται στο αρχείο, καλούμε ρητά τη μέθοδο `Calculate()`.

```csharp
        // Step 3: Force calculation so the formula result is materialized
        workbook.Calculate();   // forces evaluation of all formulas in the workbook
```

**Γιατί να εξαναγκάσουμε τον υπολογισμό;**  
Αν παραλείψετε αυτό το βήμα, το αποθηκευμένο αρχείο θα περιέχει ακόμη τον τύπο, αλλά τα κελιά θα φαίνονται κενά μέχρι ο χρήστης να ανοίξει το βιβλίο εργασίας και το Excel να επαναϋπολογίσει. Σε αυτοματοποιημένες αλυσίδες συνήθως θέλετε τις τιμές ήδη ενσωματωμένες.

## Βήμα 4 – Αποθήκευση του Βιβλίου Εργασίας ως XLSX (Δευτερεύουσα Λέξη‑Κλειδί Συμπεριλαμβανόμενη)

Τώρα που τα δεδομένα είναι έτοιμα, γράφουμε το βιβλίο εργασίας στο δίσκο. Η μέθοδος `Save` εντοπίζει αυτόματα τη μορφή αρχείου από την επέκταση.

```csharp
        // Step 4: (Optional) Save the workbook to inspect the result
        string outputPath = @"C:\Temp\output.xlsx";   // adjust folder as needed
        workbook.Save(outputPath);                    // creates a .xlsx file on disk
        System.Console.WriteLine($"Workbook saved to {outputPath}");
    }
}
```

Όταν ανοίξετε το `output.xlsx` θα δείτε τη μήτρα τοποθετημένη ακριβώς όπως φαίνεται παραπάνω. Δεν απαιτούνται επιπλέον βήματα.

## Bonus: Μετατροπή Μεγαλύτερων Πινάκων – Πραγματικές Περιπτώσεις Χρήσης

Φανταστείτε ότι λαμβάνετε μια επίπεδη λίστα JSON με 100 αριθμούς από ένα API και χρειάζεστε τα σε έναν πίνακα 10 στηλών. Μπορείτε να επαναχρησιμοποιήσετε το ίδιο μοτίβο:

```csharp
int[] numbers = Enumerable.Range(1, 100).ToArray();
string arrayLiteral = "{" + string.Join(",", numbers) + "}";
ws.Cells["A1"].Formula = $"=WRAPCOLS({arrayLiteral},10)";
workbook.Calculate();
```

**Περιπτώσεις Όριων που Πρέπει να Προσέξετε**

- **Πάρα πολλές στήλες:** Το Excel περιορίζει τον αριθμό στηλών στα 16 384. Αν ζητήσετε από το WRAPCOLS περισσότερες, η λειτουργία επιστρέφει σφάλμα `#VALUE!`.
- **Μη‑αριθμητικά δεδομένα:** Το WRAPCOLS λειτουργεί και με κείμενο, αλλά πρέπει να περικλείσετε τις συμβολοσειρές σε διπλά εισαγωγικά μέσα στον κυριολεκτικό πίνακα (π.χ., `{"Apple","Banana","Cherry"}`).
- **Απόδοση:** Για πολύ μεγάλους πίνακες, η δημιουργία της κυριολεκτικής συμβολοσειράς μπορεί να γίνει bottleneck. Σε τέτοιες περιπτώσεις, σκεφτείτε να γράψετε τις τιμές απευθείας στα κελιά αντί να χρησιμοποιήσετε τύπο.

## Συχνές Ερωτήσεις (FAQ)

**Λειτουργεί αυτό με παλαιότερες εκδόσεις του Excel;**  
Ναι. Το `WRAPCOLS` εισήχθη στο Excel 365 και Excel 2019, αλλά το Aspose.Cells μπορεί να το προσομοιώσει για παλαιότερες μορφές αρχείων (π.χ., `.xls`). Το παραγόμενο αρχείο θα ανοίξει, αν και ο τύπος μπορεί να εμφανίζεται ως απλό κείμενο αν ο θεατής δεν το υποστηρίζει.

**Τι γίνεται αν θέλω να διατηρήσω τον τύπο για μελλοντικές ενημερώσεις;**  
Απλώς παραλείψτε το `workbook.Calculate()`. Το αποθηκευμένο αρχείο θα διατηρήσει τον τύπο `WRAPCOLS`, επιτρέποντας στους τελικούς χρήστες να επεξεργαστούν τον αρχικό πίνακα και να δουν αυτόματα την ενημέρωση της μήτρας.

**Μπορώ να εφαρμόσω μορφοποίηση μετά την εμφάνιση της μήτρας;**  
Απολύτως. Μετά το `Calculate()`, μπορείτε να αναφερθείτε στην πλημμυρισμένη περιοχή (`A1:B2` στο demo) και να εφαρμόσετε γραμματοσειρές, περιγράμματα ή μορφές αριθμών όπως σε οποιαδήποτε άλλη περιοχή κελιών.

## Πλήρες Παράδειγμα – Έτοιμο για Αντιγραφή‑Επικόλληση

Παρακάτω βρίσκεται το πλήρες πρόγραμμα που μπορείτε να τοποθετήσετε σε μια εφαρμογή κονσόλας και να τρέξετε αμέσως (μην ξεχάσετε να προσθέσετε το πακέτο NuGet Aspose.Cells).

```csharp
using System;
using Aspose.Cells;

class Program
{
    static void Main()
    {
        // 1️⃣ Create a new workbook and get the first worksheet
        Workbook workbook = new Workbook();
        Worksheet ws = workbook.Worksheets[0];

        // 2️⃣ Insert WRAPCOLS formula to convert a 1‑D array into a 2‑column matrix
        ws.Cells["A1"].Formula = "=WRAPCOLS({1,2,3,4},2)";

        // 3️⃣ Force calculation so the result is materialized
        workbook.Calculate();

        // 4️⃣ Save the workbook as XLSX
        string outputPath = @"C:\Temp\output.xlsx";
        workbook.Save(outputPath);
        Console.WriteLine($"✅ Workbook created and saved to {outputPath}");
    }
}
```

**Αναμενόμενο αποτέλεσμα:**  
- Ένα αρχείο `output.xlsx` στο `C:\Temp\`.  
- Τα κελιά `A1:B2` γεμάτα με `1, 2, 3, 4` διατεταγμένα σε δύο στήλες.  
- Δεν θα υπάρχουν υπόλοιποι τύποι αν κάλεσατε το `Calculate()`· διαφορετικά ο τύπος θα παραμείνει ορατός.

3. **Αυτοματοποίηση μορφοποίησης** – εφαρμόστε στυλ πινάκων, υπό συνθήκη μορφοποίηση ή γραφήματα στη δημιουργημένη μήτρα.
4. **Εξαγωγή σε άλλες μορφές** – το Aspose.Cells μπορεί επίσης να αποθηκεύσει ως CSV, PDF ή ακόμη και HTML αν χρειαστεί να μοιραστείτε τα δεδομένα εκτός Excel.
