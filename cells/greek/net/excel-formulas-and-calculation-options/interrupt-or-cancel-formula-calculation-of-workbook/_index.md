# Διακοπή ή ακύρωση υπολογισμού τύπου βιβλίου εργασίας

## Εισαγωγή
Έχετε κουραστεί από τους υπολογισμούς σας στο Excel που εκτελούνται περισσότερο από όσο θα έπρεπε; Υπάρχουν φορές που μπορεί να θέλετε να διακόψετε ή να διακόψετε έναν μακροσκελή υπολογισμό τύπου στο βιβλίο εργασίας σας. Είτε έχετε να κάνετε με εκτεταμένα σύνολα δεδομένων είτε με σύνθετους τύπους, η γνώση του τρόπου ελέγχου αυτής της διαδικασίας μπορεί να σας εξοικονομήσει πολύ χρόνο και ταλαιπωρία. Σε αυτό το άρθρο, θα σας καθοδηγήσουμε στον τρόπο χρήσης του Aspose.Cells για .NET για την αποτελεσματική διακοπή ή ακύρωση υπολογισμών τύπων στα βιβλία εργασίας του Excel. 
## Προαπαιτούμενα
Πριν ξεκινήσουμε το σεμινάριό μας, ας βεβαιωθούμε ότι έχετε ρυθμίσει τα πάντα:
1. Visual Studio: Πρέπει να έχετε εγκατεστημένο το Visual Studio στον υπολογιστή σας. Οποιαδήποτε έκδοση υποστηρίζει ανάπτυξη .NET είναι κατάλληλη.
2. Aspose.Cells για .NET: Λήψη και εγκατάσταση της βιβλιοθήκης Aspose.Cells από [εδώ](https://releases.aspose.com/cells/net/).
3. Βασικές γνώσεις C#: Η εξοικείωση με τη γλώσσα προγραμματισμού C# θα είναι ωφέλιμη, καθώς θα γράφουμε μαζί αποσπάσματα κώδικα.
4. Ένα αρχείο Excel: Για αυτό το σεμινάριο, θα αναφερθούμε σε ένα δείγμα αρχείου Excel με το όνομα `sampleCalculationMonitor.xlsx`Βεβαιωθείτε ότι το έχετε διαθέσιμο στον κατάλογο εργασιών σας.
Μόλις τα έχετε όλα αυτά στη θέση τους, μπορούμε να προχωρήσουμε κατευθείαν στον κώδικα!
## Εισαγωγή πακέτων
Στο έργο σας στο Visual Studio, θα χρειαστεί να εισαγάγετε αρκετούς χώρους ονομάτων που σχετίζονται με το Aspose.Cells. Ακολουθούν τα πακέτα που θα θέλετε να συμπεριλάβετε στην αρχή του αρχείου κώδικά σας:
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```
Συμπεριλαμβάνοντας αυτούς τους χώρους ονομάτων, θα αποκτήσετε πρόσβαση στις απαραίτητες κλάσεις και μεθόδους για τον χειρισμό βιβλίων εργασίας του Excel.
Τώρα που είστε έτοιμοι με τις προϋποθέσεις και τα πακέτα, ας χωρίσουμε την εργασία σε διαχειρίσιμα βήματα. Κάθε βήμα θα έχει μια επικεφαλίδα και μια συνοπτική εξήγηση.
## Βήμα 1: Ρύθμιση του βιβλίου εργασίας σας
Αρχικά, πρέπει να φορτώσετε το βιβλίο εργασίας σας. Αυτό είναι το αρχείο που περιέχει τους υπολογισμούς που ίσως θέλετε να διακόψετε. Δείτε πώς:
```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory"; // Ενημερώστε με την πραγματική διαδρομή καταλόγου σας.
Workbook wb = new Workbook(sourceDir + "sampleCalculationMonitor.xlsx");
```
Σε αυτό το βήμα, δημιουργούμε ένα `Workbook` για παράδειγμα, κατευθύνοντάς το στο αρχείο Excel μας. Αυτό θέτει το σκηνικό για όλες τις περαιτέρω ενέργειες.
## Βήμα 2: Δημιουργία επιλογών υπολογισμού
Στη συνέχεια, θα δημιουργήσουμε μια επιλογή υπολογισμού και θα τη συνδυάσουμε με μια κλάση παρακολούθησης υπολογισμών. Αυτό είναι κρίσιμο για τον έλεγχο του τρόπου εκτέλεσης των υπολογισμών μας.
```csharp
CalculationOptions opts = new CalculationOptions();
opts.CalculationMonitor = new clsCalculationMonitor();
```
Εδώ, δημιουργούμε ένα υπόδειγμα `CalculationOptions` και να αναθέσετε `clsCalculationMonitor` — μια προσαρμοσμένη κλάση που θα ορίσουμε στη συνέχεια. Αυτό θα μας επιτρέψει να παρακολουθούμε τους υπολογισμούς και να εφαρμόζουμε διακοπές.
## Βήμα 3: Υλοποίηση της Παρακολούθησης Υπολογισμών
Τώρα, ας δημιουργήσουμε το δικό μας `clsCalculationMonitor` κλάση. Αυτή η κλάση θα κληρονομήσει από `AbstractCalculationMonitor` και θα περιέχει τη λογική μας για να διακόπτει τους υπολογισμούς.
```csharp
class clsCalculationMonitor : AbstractCalculationMonitor
{
    public override void BeforeCalculate(int sheetIndex, int rowIndex, int colIndex)
    {
        // Βρείτε το όνομα του κελιού
        string cellName = CellsHelper.CellIndexToName(rowIndex, colIndex);
        // Εκτυπώστε το φύλλο, τον δείκτη γραμμών και στηλών καθώς και το όνομα του κελιού
        System.Diagnostics.Debug.WriteLine(sheetIndex + "----" + rowIndex + "----" + colIndex + "----" + cellName);
        // Εάν το όνομα κελιού είναι B8, διακόψτε/ακυρώστε τον υπολογισμό του τύπου
        αν (cellName == "B8")
        {
            this.Interrupt("Interrupt/Cancel the formula calculation");
        } // if
    } // ΠρινΥπολογίστε
} // clsCalculationMonitor
```
Σε αυτήν την κλάση, παρακάμπτουμε το `BeforeCalculate` μέθοδος, η οποία ενεργοποιείται πριν από οποιονδήποτε υπολογισμό κελιού. Ελέγχουμε αν το τρέχον κελί είναι `B8`Αν είναι, καλούμε `this.Interrupt()` για να σταματήσει ο υπολογισμός.
## Βήμα 4: Υπολογίστε τον τύπο με επιλογές
Με τις επιλογές και την οθόνη μας στη θέση τους, ήρθε η ώρα να εκτελέσουμε τον υπολογισμό:
```csharp
wb.CalculateFormula(opts);
```
Αυτή η εντολή θα εκτελέσει τους υπολογισμούς ενώ θα παρακολουθεί για διακοπές. Εάν ο υπολογισμός φτάσει στο B8, θα σταματήσει σύμφωνα με την προηγούμενη λογική μας.
