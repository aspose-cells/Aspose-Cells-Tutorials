# Χρήση ενσωματωμένων μορφών αριθμών στο Excel μέσω προγραμματισμού

## Εισαγωγή
Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στον τρόπο χρήσης ενσωματωμένων μορφών αριθμών στο Excel χρησιμοποιώντας το Aspose.Cells για .NET. Θα καλύψουμε τα πάντα, από τη ρύθμιση του περιβάλλοντός σας έως την εφαρμογή διαφορετικών μορφών όπως ημερομηνίες, ποσοστά και νομίσματα. Είτε είστε έμπειρος επαγγελματίας είτε απλώς βουτάτε τα δάχτυλά σας στο οικοσύστημα .NET, αυτός ο οδηγός θα σας βοηθήσει να μορφοποιήσετε κελιά του Excel σαν παιχνίδι.
## Προαπαιτούμενα
Πριν βουτήξετε, βεβαιωθείτε ότι έχετε τα εξής:
- Εγκατεστημένο το Aspose.Cells για τη βιβλιοθήκη .NET. Μπορείτε να [κατεβάστε το εδώ](https://releases.aspose.com/cells/net/).
- Πρακτική γνώση C# και βασικού προγραμματισμού .NET.
- Visual Studio ή οποιοδήποτε .NET IDE εγκατεστημένο στον υπολογιστή σας.
- Μια έγκυρη άδεια Aspose ή [προσωρινή άδεια](https://purchase.aspose.com/temporary-license/).
- Εγκατεστημένο .NET framework (έκδοση 4.0 ή νεότερη).
  
Αν σας λείπει κάποιο από τα παραπάνω, ακολουθήστε τους συνδέσμους που παρέχονται για να τα ρυθμίσετε όλα. Είστε έτοιμοι; Ας περάσουμε στο διασκεδαστικό κομμάτι!
## Εισαγωγή πακέτων
Πριν ξεκινήσουμε με το σεμινάριο, φροντίστε να εισαγάγετε τους απαραίτητους χώρους ονομάτων για την εργασία με το Aspose.Cells για .NET:
```csharp
using System.IO;
using Aspose.Cells;
using System;
```
Μόλις τα εισαγάγετε, είστε έτοιμοι να χειριστείτε αρχεία Excel μέσω προγραμματισμού. Τώρα, ας δούμε τον οδηγό βήμα προς βήμα!
## Βήμα 1: Δημιουργία ή πρόσβαση στο βιβλίο εργασίας του Excel
Σε αυτό το βήμα, θα δημιουργήσετε ένα νέο βιβλίο εργασίας. Σκεφτείτε το σαν να ανοίγετε ένα νέο αρχείο Excel, εκτός από το ότι το κάνετε μέσω κώδικα!
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "Your Document Directory";
// Δημιουργήστε έναν κατάλογο εάν δεν υπάρχει ήδη.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
	System.IO.Directory.CreateDirectory(dataDir);
// Δημιουργία στιγμιαίου αντικειμένου βιβλίου εργασίας
Workbook workbook = new Workbook();
```
Εδώ, απλώς δημιουργούμε ένα νέο `Workbook` αντικείμενο. Αυτό λειτουργεί ως το αρχείο Excel σας, έτοιμο για χειρισμό δεδομένων. Μπορείτε επίσης να φορτώσετε ένα υπάρχον αρχείο παρέχοντας τη διαδρομή του.
## Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας
Τα βιβλία εργασίας του Excel μπορούν να περιέχουν πολλά φύλλα εργασίας. Σε αυτό το βήμα, θα έχουμε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας σας:
```csharp
Worksheet worksheet = workbook.Worksheets[0];
```
Τώρα έχουμε πρόσβαση στο πρώτο φύλλο εργασίας του βιβλίου εργασίας. Εάν χρειάζεται να χειριστείτε επιπλέον φύλλα, μπορείτε να τα αναφέρετε χρησιμοποιώντας το ευρετήριο ή το όνομά τους.
## Βήμα 3: Προσθήκη δεδομένων σε κελιά
Ας ξεκινήσουμε την προσθήκη ορισμένων δεδομένων σε συγκεκριμένα κελιά. Αρχικά, θα εισαγάγουμε την τρέχουσα ημερομηνία συστήματος στο κελί "A1":
```csharp
worksheet.Cells["A1"].PutValue(DateTime.Now);
```
Αυτή η γραμμή εισάγει την τρέχουσα ημερομηνία στο κελί A1. Πολύ ωραίο, σωστά; Φανταστείτε να το κάνετε αυτό χειροκίνητα για εκατοντάδες κελιά—θα ήταν ένας εφιάλτης. Τώρα, θα προχωρήσουμε στη μορφοποίηση!
## Βήμα 4: Μορφοποίηση ημερομηνίας στο κελί "A1"
Στη συνέχεια, ας μορφοποιήσουμε αυτήν την ημερομηνία σε μια πιο ευανάγνωστη μορφή, όπως "15-Οκτ-24". Εδώ είναι που το Aspose.Cells πραγματικά λάμπει:
1. Ανάκτηση του στυλ του κελιού:
```csharp
Style style = worksheet.Cells["A1"].GetStyle();
```
Εδώ, υιοθετούμε το στυλ του κελιού A1. Σκεφτείτε το σαν να υιοθετούμε τη "μόδα" του κελιού πριν κάνουμε οποιεσδήποτε τροποποιήσεις.
2. Ορίστε τη μορφή ημερομηνίας:
```csharp
style.Number = 15;
```
Ρύθμιση του `Number` Η ιδιότητα έως 15 εφαρμόζει την επιθυμητή μορφή ημερομηνίας. Αυτός είναι ένας ενσωματωμένος κώδικας μορφής αριθμού για την εμφάνιση ημερομηνιών στη μορφή "η-μμμ-εε".
3. Εφαρμόστε το στυλ στο κελί:
```csharp
worksheet.Cells["A1"].SetStyle(style);
```
Αυτή η γραμμή εφαρμόζει τις αλλαγές στυλ στο κελί. Τώρα, αντί για μια προεπιλεγμένη μορφή ημερομηνίας, θα δείτε κάτι πολύ πιο φιλικό προς το χρήστη, όπως "15-Οκτ-24".
## Βήμα 5: Προσθήκη και μορφοποίηση ποσοστού στο κελί "A2"
Ας προχωρήσουμε στη μορφοποίηση ποσοστών. Φανταστείτε ότι θέλετε να εισαγάγετε μια τιμή και να την εμφανίσετε ως ποσοστό. Σε αυτό το βήμα, θα προσθέσουμε μια αριθμητική τιμή στο κελί "A2" και θα τη μορφοποιήσουμε ως ποσοστό:
1. Εισαγωγή αριθμητικής τιμής:
```csharp
worksheet.Cells["A2"].PutValue(20);
```
Αυτό εισάγει τον αριθμό 20 στο κελί A2. Ίσως σκέφτεστε: "Αυτός είναι απλώς ένας απλός αριθμός—πώς μπορώ να τον μετατρέψω σε ποσοστό;" Λοιπόν, σύντομα θα φτάσουμε σε αυτό.
2. Ανακτήστε το στυλ και ορίστε τη μορφή ποσοστού:
```csharp
style = worksheet.Cells["A2"].GetStyle();
style.Number = 9;  // Μορφοποίηση ως ποσοστό
worksheet.Cells["A2"].SetStyle(style);
    ```
Setting the `Number` property to 9 applies the built-in percentage format. Now the value in A2 will be displayed as "2000%." (Yes, 20 is treated as 2000% in percentage formatting).
## Step 6: Add and Format Currency in Cell "A3"
Now, let’s add a numeric value in cell A3 and format it as currency. This is a common use case for financial reports.
1. Insert Numeric Value:
```csharp
worksheet.Cells["A3"].PutValue(2546);
```
Εδώ, προσθέτουμε τον αριθμό 2546 στο κελί A3. Στη συνέχεια, θα μορφοποιήσουμε αυτόν τον αριθμό ώστε να εμφανίζεται ως νόμισμα.
2. Ανακτήστε το στυλ και ορίστε τη μορφή νομίσματος:
```csharp
style = worksheet.Cells["A3"].GetStyle();
style.Number = 6;  // Μορφοποίηση ως νόμισμα
worksheet.Cells["A3"].SetStyle(style);
```
Ρύθμιση του `Number` Η ιδιότητα έως 6 εφαρμόζει τη μορφή νομίσματος. Τώρα η τιμή στο κελί A3 θα εμφανίζεται ως "2.546,00", συμπληρώνοντας με κόμματα και δύο δεκαδικά ψηφία.
## Βήμα 7: Αποθήκευση του αρχείου Excel
Τώρα που έχουμε εφαρμόσει όλη τη μαγεία της μορφοποίησης, ήρθε η ώρα να αποθηκεύσουμε το αρχείο:
```csharp
// Αποθήκευση του αρχείου Excel
workbook.Save(dataDir + "book1.out.xls", SaveFormat.Excel97To2003);
```
Αυτή η γραμμή αποθηκεύει το αρχείο Excel στη μορφή Excel 97-2003. Μπορείτε να αλλάξετε το `SaveFormat` για να ταιριάζει στις ανάγκες σας. Και έτσι απλά, δημιουργήσατε και μορφοποιήσατε ένα αρχείο Excel μέσω προγραμματισμού!
## Σύναψη
Συγχαρητήρια! Μάθατε με επιτυχία πώς να χρησιμοποιείτε το Aspose.Cells για .NET για να εφαρμόζετε ενσωματωμένες μορφές αριθμών σε κελιά σε ένα αρχείο Excel. Από ημερομηνίες έως ποσοστά και νομίσματα, καλύψαμε μερικές από τις πιο συνηθισμένες ανάγκες μορφοποίησης για την επεξεργασία δεδομένων Excel. Τώρα, αντί να μορφοποιείτε κελιά χειροκίνητα, μπορείτε να αυτοματοποιήσετε ολόκληρη τη διαδικασία, εξοικονομώντας σας χρόνο και μειώνοντας τα σφάλματα.
## Συχνές ερωτήσεις
### Μπορώ να εφαρμόσω προσαρμοσμένες μορφές αριθμών χρησιμοποιώντας το Aspose.Cells για .NET;
Ναι! Εκτός από τις ενσωματωμένες μορφές, το Aspose.Cells υποστηρίζει επίσης προσαρμοσμένες μορφές αριθμών. Μπορείτε να δημιουργήσετε πολύ συγκεκριμένες μορφές χρησιμοποιώντας το `Custom` ιδιοκτησία στο `Style` τάξη.
### Πώς μπορώ να μορφοποιήσω ένα κελί ως νόμισμα με ένα συγκεκριμένο σύμβολο;
Για να εφαρμόσετε ένα συγκεκριμένο σύμβολο νομίσματος, μπορείτε να χρησιμοποιήσετε προσαρμοσμένη μορφοποίηση ορίζοντας το `Style.Custom` ιδιοκτησία.
### Μπορώ να μορφοποιήσω ολόκληρες γραμμές ή στήλες;
Απολύτως! Μπορείτε να εφαρμόσετε στυλ σε ολόκληρες γραμμές ή στήλες χρησιμοποιώντας το `Rows` ή `Columns` συλλογές στο `Worksheet` αντικείμενο.
### Πώς μπορώ να μορφοποιήσω πολλά κελιά ταυτόχρονα;
Μπορείτε να χρησιμοποιήσετε το `Range` αντιταχθείτε στην επιλογή πολλών κελιών και στην εφαρμογή στυλ σε όλα ταυτόχρονα.
### Χρειάζεται να έχω εγκατεστημένο το Microsoft Excel για να χρησιμοποιήσω το Aspose.Cells;
Όχι, το Aspose.Cells λειτουργεί ανεξάρτητα από το Microsoft Excel, επομένως δεν χρειάζεται να έχετε εγκατεστημένο το Excel στον υπολογιστή σας.
