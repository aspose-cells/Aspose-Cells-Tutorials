# πώς να μορφοποιήσετε νόμισμα σε στήλη Grid – Πλήρες Πρόγραμμα Εκμάθησης

Έχετε αναρωτηθεί ποτέ **πώς να μορφοποιήσετε νόμισμα** σε μια στήλη πλέγματος χωρίς να τρελαίνεστε; Δεν είστε ο μόνος. Όταν κοιτάζετε έναν απλό αριθμό όπως `1234.5` και θα θέλατε να εμφανίζεται μαγικά ως `$1,234.50`, η απάντηση είναι συνήθως μόνο μερικές γραμμές ρύθμισης.  

Σε αυτόν τον οδηγό θα **ανακτήσουμε τη στήλη με το όνομα**, **ορίσουμε τη μορφή αριθμού της στήλης**, και **εφαρμόσουμε προσαρμοσμένη αριθμητική μορφή** που σέβεται τη τυπική λογιστική διάταξη. Καθ' όλη τη διάρκεια θα **ορίσουμε την ευθυγράμμιση της στήλης του πλέγματος** και θα προσθέσουμε ένα διακριτικό περίγραμμα ώστε η διεπαφή να φαίνεται επαγγελματική.

> **TL;DR** – Στο τέλος θα έχετε ένα έτοιμο‑για‑εκτέλεση απόσπασμα κώδικα που μετατρέπει ακατέργαστα δεκαδικά σε όμορφα μορφοποιημένες τιμές νομίσματος μέσα σε οποιονδήποτε έλεγχο τύπου `GridJs`.

---

## Τι Θα Χρειαστείτε

- Ένα .NET project (οποιαδήποτε έκδοση που υποστηρίζει C# 8.0+ – το Visual Studio 2022 λειτουργεί εξαιρετικά).  
- Ένα στοιχείο πλέγματος που εκθέτει μια συλλογή `Columns` (το παράδειγμα χρησιμοποιεί μια φανταστική κλάση `GridJs`, αλλά οι έννοιες μεταφράζονται σε πλέγματα DevExpress, Telerik ή Syncfusion).  
- Βασική εξοικείωση με τη σύνταξη C# – δεν απαιτούνται προχωρημένα κόλπα.

Αν τα έχετε ήδη, υπέροχα. Αν όχι, δημιουργήστε μια εφαρμογή κονσόλας· το πλέγμα μπορεί να προσομοιωθεί για επεξήγηση.

---

## Υλοποίηση Βήμα‑βήμα

Κάτω από κάθε βήμα θα δείτε ένα συμπαγές μπλοκ κώδικα, μια σύντομη εξήγηση του **γιατί** η γραμμή είναι σημαντική, και μια συμβουλή για να αποφύγετε κοινά προβλήματα.

### ## Βήμα 1 – Ανάκτηση της στήλης “Amount” με το όνομα

```csharp
// Step 1: Retrieve the "Amount" column from the grid
var amountColumn = gridJs.Columns["Amount"];
if (amountColumn == null)
{
    throw new InvalidOperationException("Column 'Amount' does not exist. Verify the column name or check the grid's schema.");
}
```

**Γιατί είναι σημαντικό:**  
Οι περισσότερες API πλέγματος εκθέτουν τις στήλες μέσω ενός δείκτη τύπου λεξικού. Η ανάκτηση της στήλης με το όνομα της κεφαλίδας (`"Amount"`) σας επιτρέπει να χειριστείτε την εμφάνισή της χωρίς να αγγίξετε την υποκείμενη πηγή δεδομένων.  

**Συμβουλή:** Πάντα να προστατεύετε από επιστροφή `null` – ένα τυπογραφικό λάθος στο όνομα της στήλης ή μια δυναμική αλλαγή σχήματος μπορεί διαφορετικά να προκαλέσει `NullReferenceException` κατά την εκτέλεση.

---

### ## Βήμα 2 – Ορισμός μορφής αριθμού στήλης χρησιμοποιώντας προσαρμοσμένη μάσκα νομίσματος

```csharp
// Step 2: Apply a custom numeric format for currency values
amountColumn.NumberFormat = "_(* #,##0.00_);_(* (#,##0.00);_(* \"-\"??_);_(@_)";
```

**Γιατί είναι σημαντικό:**  
Η συμβολοσειρά μορφής ακολουθεί τις συμβάσεις μορφοποίησης λογιστικής του Excel:

- `_(* #,##0.00_)` → Θετικοί αριθμοί, ευθυγραμμισμένοι δεξιά με ένα αρχικό κενό για το σύμβολο νομίσματος.  
- `_(* (#,##0.00)` → Αρνητικοί αριθμοί σε παρενθέσεις.  
- `_(* \"-\"??_)` → Τιμές μηδέν εμφανίζονται ως παύλα.  
- `_(@_)` → Τιμές κειμένου παραμένουν αμετάβλητες.

Η χρήση του **apply custom numeric format** σας δίνει πλήρη έλεγχο πάνω στους διαχωριστές χιλιάδων, τα δεκαδικά ψηφία και τη θέση του συμβόλου νομίσματος.  

**Περίπτωση άκρης:** Αν η εφαρμογή σας πρέπει να σέβεται διαφορετική τοπική ρύθμιση (π.χ., Ευρώ αντί για USD), αντικαταστήστε το αρχικό κενό με το κατάλληλο σύμβολο ή χρησιμοποιήστε μορφοποίηση που λαμβάνει υπόψη το `CultureInfo` στην πηγή δεδομένων.

---

### ## Βήμα 3 – Ευθυγράμμιση του περιεχομένου της στήλης δεξιά για ευκολία ανάγνωσης

```csharp
// Step 3: Align the column contents to the right for better readability
amountColumn.Alignment = GridAlignment.Right;
```

**Γιατί είναι σημαντικό:**  
Οι τιμές νομίσματος είναι πιο εύκολες στην ανάγνωση όταν ευθυγραμμίζονται στον δεκαδικό διαχωριστή. Ορίζοντας το **set grid column alignment** σε `Right` αντικατοπτρίζει τον τρόπο που τα φύλλα εργασίας εμφανίζουν χρηματικά δεδομένα.  

**Προειδοποίηση:** Ορισμένα πλέγματα αγνοούν την ευθυγράμμιση σε κελιά που περιέχουν προσαρμοσμένα πρότυπα. Αν παρατηρήσετε ότι η ευθυγράμμιση δεν εφαρμόζεται, ελέγξτε ξανά ότι η στήλη δεν χρησιμοποιεί προσαρμοσμένο renderer κελιού.

---

### ## Βήμα 4 – Προσθήκη λεπτού γκρι περιγράμματος γύρω από τα κελιά της στήλης

```csharp
// Step 4: Add a thin gray border around the column cells
amountColumn.Border = new GridBorder
{
    Color = Color.Gray,
    Style = BorderLineStyle.Thin
};
```

**Γιατί είναι σημαντικό:**  
Ένα διακριτικό περίγραμμα διαχωρίζει τη στήλη “Amount” από τις γειτονικές, ειδικά όταν το πλέγμα έχει εναλλασσόμενα χρώματα γραμμών. Είναι ένα οπτικό σήμα ότι τα δεδομένα αντιπροσωπεύουν ξεχωριστό οικονομικό στοιχείο.  

**Συμβουλή:** Αν χρειάζεστε πιο παχύ γραμμή για εκτύπωση, αυξήστε το `BorderLineStyle` σε `Medium` ή αλλάξτε το `Color` σε `Color.Black`.

## Πλήρες Παράδειγμα Λειτουργίας

Ακολουθεί ολόκληρο το απόσπασμα κώδικα που μπορείτε να ενσωματώσετε σε ένα έργο WinForms ή WPF που χρησιμοποιεί έναν έλεγχο τύπου `GridJs`. Το παράδειγμα εκτυπώνει επίσης τις μορφοποιημένες τιμές στην κονσόλα ώστε να μπορείτε να επαληθεύσετε το αποτέλεσμα χωρίς UI.

```csharp
using System;
using System.Drawing;   // For Color
using GridLibrary;      // Hypothetical namespace for GridJs

namespace GridCurrencyDemo
{
    class Program
    {
        static void Main()
        {
            // Create a mock grid and add a sample column
            var gridJs = new GridJs();
            gridJs.Columns.Add(new GridColumn
            {
                Name = "Amount",
                Header = "Amount",
                DataType = typeof(decimal)
            });

            // Populate some sample data
            gridJs.Rows.Add(new { Amount = 1234.5m });
            gridJs.Rows.Add(new { Amount = -567.89m });
            gridJs.Rows.Add(new { Amount = 0m });

            // ---- Formatting steps ------------------------------------------------
            // 1️⃣ Retrieve the "Amount" column
            var amountColumn = gridJs.Columns["Amount"]
                ?? throw new InvalidOperationException("Column 'Amount' not found.");

            // 2️⃣ Apply custom numeric format for currency
            amountColumn.NumberFormat = "_(* #,##0.00_);_(* (#,##0.00);_(* \"-\"??_);_(@_)";

            // 3️⃣ Right‑align the values
            amountColumn.Alignment = GridAlignment.Right;

            // 4️⃣ Add a thin gray border
            amountColumn.Border = new GridBorder
            {
                Color = Color.Gray,
                Style = BorderLineStyle.Thin
            };
            // -----------------------------------------------------------------------

            // Render the grid (in a real UI you would call gridJs.Render() or similar)
            Console.WriteLine("Formatted Currency Grid:");
            foreach (var row in gridJs.Rows)
            {
                var rawValue = (decimal)row.Amount;
                // The grid library would automatically apply NumberFormat when displaying.
                // For console demo we mimic the formatting:
                string formatted = rawValue.ToString("#,##0.00", System.Globalization.CultureInfo.InvariantCulture);
                if (rawValue < 0)
                    formatted = $"({formatted.TrimStart('-')})";
                else if (rawValue == 0)
                    formatted = "-";

                Console.WriteLine($"| {formatted,15} |");
            }

            // Keep console open
            Console.WriteLine("\nPress any key to exit...");
            Console.ReadKey();
        }
    }
}
```

**Αναμενόμενη έξοδος κονσόλας**

```
Formatted Currency Grid:
|        1,234.50 |
|       (567.89) |
|               - |
```

Παρατηρήστε πώς ο θετικός αριθμός είναι ευθυγραμμισμένος δεξιά, ο αρνητικός εμφανίζεται σε παρενθέσεις, και το μηδέν δείχνει παύλα – ακριβώς όπως ορίζει η προσαρμοσμένη συμβολοσειρά μορφής.

## Συχνές Ερωτήσεις & Περιπτώσεις Άκρων

| Ερώτηση | Απάντηση |
|----------|--------|
| *Τι γίνεται αν το πλέγμα χρησιμοποιεί διαφορετικό πολιτισμό (π.χ., € αντί για $);* | Αντικαταστήστε το αρχικό κενό στη συμβολοσειρά μορφής με το επιθυμητό σύμβολο ή αφήστε την πηγή δεδομένων να εκδώσει μια προ‑μορφοποιημένη συμβολοσειρά χρησιμοποιώντας `CultureInfo.CurrentCulture`. |
| *Μπορώ να επαναχρησιμοποιήσω την ίδια μορφή για πολλαπλές στήλες;* | Απολύτως. Αποθηκεύστε τη συμβολοσειρά μορφής σε μια σταθερά (`const string CurrencyMask = "...";`) και αναθέστε την όπου χρειάζεστε νόμισμα. |
| *Τι συμβαίνει αν η στήλη περιέχει τιμή κειμένου;* | Η συμβολοσειρά μορφής επηρεάζει μόνο αριθμητικούς τύπους. Τα κείμενα περνούν αμετάβλητα, γι' αυτό υπάρχει το τελευταίο μέρος της μάσκας (`_(@_)`) – διατηρεί το μη‑αριθμητικό περιεχόμενο. |
| *Υπάρχει επίπτωση στην απόδοση;* | Αμελητέα. Η μορφή εφαρμόζεται κατά το rendering, όχι κατά την ανάκτηση των δεδομένων. Εκτός αν αποδίδετε χιλιάδες γραμμές ανά καρέ, δεν θα παρατηρήσετε καθυστέρηση. |
| *Πώς κάνω το περίγραμμα πιο παχύ για εκτυπωμένες αναφορές;* | Αλλάξτε το `BorderLineStyle.Thin` με `BorderLineStyle.Medium` ή `BorderLineStyle.Thick`. Ορισμένες βιβλιοθήκες επιτρέπουν επίσης τον καθορισμό του πλάτους σε pixel απευθείας. |
