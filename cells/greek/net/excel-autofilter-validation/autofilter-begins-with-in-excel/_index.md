# Το αυτόματο φίλτρο ξεκινά με στο Excel

## Εισαγωγή

Όσον αφορά την εργασία με δεδομένα, το Excel έχει καθιερωθεί ως μια εφαρμογή που χρησιμοποιείται για αμέτρητους κλάδους και σκοπούς. Ένα από τα πιο ισχυρά χαρακτηριστικά του είναι το Αυτόματο Φίλτρο, το οποίο κάνει την περιήγηση σε εκτεταμένα σύνολα δεδομένων παιχνιδάκι. Εάν χρησιμοποιείτε το Aspose.Cells για .NET, μπορείτε να αξιοποιήσετε αυτήν τη λειτουργικότητα μέσω προγραμματισμού και να βελτιώσετε σημαντικά τις εργασίες διαχείρισης δεδομένων σας. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στη διαδικασία εφαρμογής μιας λειτουργίας που φιλτράρει τις γραμμές του Excel με βάση το αν ξεκινούν με μια συγκεκριμένη συμβολοσειρά.

## Εισαγωγή πακέτων

Για να ξεκινήσετε, εισαγάγετε τους απαραίτητους χώρους ονομάτων στην κορυφή του αρχείου C#:

```csharp
using System.IO;
using Aspose.Cells;
using System;
```

Αυτό εισάγει την βασική λειτουργικότητα του Aspose.Cells μαζί με τις βασικές δυνατότητες του συστήματος στις οποίες θα βασιστούμε για την αλληλεπίδραση με την κονσόλα.

Τώρα που έχετε ρυθμίσει το περιβάλλον σας και έχετε εισαγάγει τα απαραίτητα πακέτα, ας αναλύσουμε τη λειτουργία Αυτόματου φίλτρου σε διαχειρίσιμα βήματα. Θα εφαρμόσουμε ένα φίλτρο που εξάγει γραμμές που αρχίζουν με "Ba".

## Βήμα 1: Ορισμός καταλόγων προέλευσης και εξόδου

Αρχικά, ας ορίσουμε πού βρίσκεται το αρχείο εισόδου Excel, καθώς και πού θέλουμε να αποθηκεύσουμε το φιλτραρισμένο αποτέλεσμα:

```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory\\";

// Κατάλογος εξόδου
string outputDir = "Your Document Directory\\";
```

Εξήγηση: Εδώ, αντικαταστήστε `"Your Document Directory\\"` με την πραγματική διαδρομή προς τους καταλόγους σας. Βεβαιωθείτε ότι έχετε ολοκληρώσει τις διαδρομές καταλόγων με διπλή ανάστροφη κάθετο (`\\`) για να αποφύγετε τυχόν προβλήματα διαδρομής.

## Βήμα 2: Δημιουργία αντικειμένου βιβλίου εργασίας

Στη συνέχεια, θα δημιουργήσουμε ένα αντικείμενο Workbook που δείχνει στο αρχείο Excel μας:

```csharp
// Δημιουργία αντικειμένου βιβλίου εργασίας που περιέχει δείγματα δεδομένων
Workbook workbook = new Workbook(sourceDir + "sourseSampleCountryNames.xlsx");
```

Εξήγηση: Αυτή η γραμμή αρχικοποιεί μια νέα παρουσία Βιβλίου Εργασίας χρησιμοποιώντας την καθορισμένη διαδρομή αρχείου. `Workbook` Η κλάση είναι θεμελιώδης καθώς αντιπροσωπεύει ολόκληρο το αρχείο Excel.

## Βήμα 3: Πρόσβαση στο πρώτο φύλλο εργασίας

Τώρα, πρέπει να έχουμε πρόσβαση στο συγκεκριμένο φύλλο εργασίας με το οποίο θέλουμε να εργαστούμε:

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο αρχείο Excel
Worksheet worksheet = workbook.Worksheets[0];
```

Εξήγηση: Το `Worksheets` η συλλογή μας επιτρέπει την πρόσβαση σε μεμονωμένα φύλλα. Η χρήση `[0]` αναφέρεται στο πρώτο φύλλο εργασίας στο αρχείο Excel σας, κάτι που είναι γενικά μια συνηθισμένη πρακτική όταν εργάζεστε με ένα αρχείο ενός φύλλου.

## Βήμα 4: Ρύθμιση του Αυτόματου Φίλτρου

Εδώ ξεκινά η μαγεία! Θα δημιουργήσουμε ένα εύρος αυτόματου φιλτραρίσματος για τα δεδομένα μας:

```csharp
// Δημιουργία Αυτόματου Φίλτρου δίνοντας εύρος στα κελιά
worksheet.AutoFilter.Range = "A1:A18";
```

Εξήγηση: Το `AutoFilter.Range` Η ιδιότητα σάς επιτρέπει να καθορίσετε ποιες γραμμές θα φιλτράρετε. Σε αυτήν την περίπτωση, φιλτράρουμε γραμμές εντός του εύρους A1 έως A18, οι οποίες θεωρείται ότι περιέχουν τα δεδομένα μας.

## Βήμα 5: Εφαρμογή συνθήκης φίλτρου

Το επόμενο βήμα είναι να ορίσετε τη συνθήκη φίλτρου. Θέλουμε να εμφανίσουμε μόνο εκείνες τις γραμμές των οποίων οι τιμές της πρώτης στήλης ξεκινούν με "Ba":

```csharp
// Αρχικοποίηση φίλτρου για γραμμές που ξεκινούν με συμβολοσειρά "Ba"
worksheet.AutoFilter.Custom(0, FilterOperatorType.BeginsWith, "Ba");
```

Εξήγηση: Το `Custom` η μέθοδος ορίζει τη λογική φιλτραρίσματος. Το πρώτο όρισμα (`0`) υποδεικνύει ότι φιλτράρουμε με βάση την πρώτη στήλη (A) και το `FilterOperatorType.BeginsWith` καθορίζει την συνθήκη μας για αναζήτηση γραμμών που ξεκινούν με "Ba".

## Βήμα 6: Ανανέωση του φίλτρου

Αφού εφαρμόσουμε τη συνθήκη φίλτρου μας, πρέπει να βεβαιωθούμε ότι το Excel ανανεώνεται ώστε να αντικατοπτρίζει τις αλλαγές:

```csharp
// Ανανέωση του φίλτρου για εμφάνιση/απόκρυψη φιλτραρισμένων γραμμών
worksheet.AutoFilter.Refresh();
```

Εξήγηση: Αυτή η γραμμή ενεργοποιεί μια ανανέωση στο Αυτόματο Φίλτρο για να διασφαλίσει ότι οι ορατές γραμμές αντιστοιχούν στα εφαρμοσμένα κριτήρια φίλτρου. Είναι παρόμοιο με το πάτημα του κουμπιού ανανέωσης στο Excel.

## Βήμα 7: Αποθήκευση του τροποποιημένου αρχείου Excel

Τώρα είναι η ώρα να αποθηκεύσουμε τις αλλαγές που έχουμε κάνει:

```csharp
// Αποθήκευση του τροποποιημένου αρχείου Excel
workbook.Save(outputDir + "outSourseSampleCountryNames.xlsx");
```

Εξήγηση: Το `Save` Η μέθοδος γράφει το τροποποιημένο Βιβλίο Εργασίας πίσω στην καθορισμένη διαδρομή εξόδου. Αυτό εμπίπτει στην εγγραφή των καθορισμένων φίλτρων σας σε ένα νέο αρχείο, έτσι ώστε τα αρχικά σας δεδομένα να παραμείνουν άθικτα.

## Βήμα 8: Επιβεβαίωση εξόδου

Τέλος, ας επιβεβαιώσουμε ότι η επέμβασή μας ήταν επιτυχής:

```csharp
Console.WriteLine("AutofilterBeginsWith executed successfully.\r\n");
```

Εξήγηση: Αυτή η απλή γραμμή εμφανίζει ένα μήνυμα επιβεβαίωσης στην κονσόλα, ενημερώνοντάς σας ότι η διαδικασία φιλτραρίσματος ολοκληρώθηκε χωρίς σφάλματα.
