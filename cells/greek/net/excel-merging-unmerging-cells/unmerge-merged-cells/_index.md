# Κατάργηση συγχώνευσης συγχωνευμένων κελιών στο Excel

## Εισαγωγή

Έχετε κουραστεί να ασχολείστε με συγχωνευμένα κελιά στα υπολογιστικά φύλλα του Excel σας; Δεν είστε οι μόνοι! Τα συγχωνευμένα κελιά μπορούν να αποτελέσουν μια χρήσιμη λειτουργία για τη μορφοποίηση, αλλά συχνά μπορούν να οδηγήσουν σε πονοκεφάλους όσον αφορά τον χειρισμό και την ανάλυση δεδομένων. Αλλά μαντέψτε τι; Η κατάργηση της συγχώνευσης αυτών των ενοχλητικών κελιών είναι ευκολότερη από ό,τι νομίζετε, ειδικά όταν χρησιμοποιείτε το Aspose.Cells για .NET. Σε αυτό το άρθρο, θα σας καθοδηγήσω βήμα προς βήμα στον τρόπο κατάργησης της συγχώνευσης συγχωνευμένων κελιών, διασφαλίζοντας ότι τα δεδομένα σας είναι τακτοποιημένα, καθαρά και έτοιμα για δράση! Πάρτε λοιπόν το καπέλο σας στον προγραμματισμό και ας βυθιστούμε στον κόσμο του Aspose.Cells.

Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Cells στο περιβάλλον .NET σας. Μπορείτε εύκολα να την αποκτήσετε μεταβαίνοντας στο [Σελίδα λήψης Aspose.Cells](https://releases.aspose.com/cells/net/).

Αποκτήστε ένα δείγμα αρχείου Excel που περιέχει ορισμένα συγχωνευμένα κελιά—θα χρησιμοποιήσετε αυτό το αρχείο για να εξασκηθείτε στην αποσυγχώνευση.

Έχοντας τακτοποιήσει όλες αυτές τις προϋποθέσεις, μπορούμε τώρα να προχωρήσουμε στο συναρπαστικό κομμάτι—την κωδικοποίηση της λύσης μας!

## Εισαγωγή πακέτων

Πρώτα απ' όλα, ας εισαγάγουμε τα απαραίτητα πακέτα. Με το Aspose.Cells, θα αλληλεπιδράτε με διάφορες κλάσεις για να διαχειρίζεστε αποτελεσματικά τα αρχεία Excel σας. Δείτε τι πρέπει να συμπεριλάβετε στην αρχή του αρχείου C#:

```csharp
using System;
using System.IO;

using Aspose.Cells;
```

Συμπεριλαμβάνοντας αυτό το πακέτο, θα έχετε πρόσβαση σε όλες τις λειτουργίες που προσφέρει το Aspose.Cells.

Ας αναλύσουμε τη διαδικασία αποσυγχώνευσης σε διαχειρίσιμα βήματα. Κάθε βήμα θα ορίζεται με σαφήνεια, ώστε να μπορείτε να την παρακολουθείτε εύκολα.

## Βήμα 1: Ορισμός καταλόγων

Το πρώτο βήμα είναι να ορίσετε τους καταλόγους όπου βρίσκονται το αρχείο Excel εισόδου (αυτό με τα συγχωνευμένα κελιά) και το αρχείο εξόδου (αυτό όπου θα αποθηκευτούν τα μη συγχωνευμένα δεδομένα). Δείτε πώς μπορείτε να το ρυθμίσετε:

```csharp
// Κατάλογος πηγής
string sourceDir = "Your Document Directory"; 

// Κατάλογος εξόδου
string outputDir = "Your Document Directory"; 
```

Φροντίστε να αντικαταστήσετε `"Your Document Directory"` με την πραγματική διαδρομή προς τα αρχεία σας.

## Βήμα 2: Δημιουργία βιβλίου εργασίας

Τώρα που έχετε ορίσει τους καταλόγους, ήρθε η ώρα να δημιουργήσετε ένα αντικείμενο Workbook. Αυτό το αντικείμενο θα σας επιτρέψει να χειριστείτε το αρχείο Excel. Μπορείτε να το κάνετε αυτό με τον ακόλουθο κώδικα:

```csharp
// Δημιουργία βιβλίου εργασίας
Workbook wbk = new Aspose.Cells.Workbook(sourceDir + "sampleUnMergingtheMergedCells.xlsx");
```

Αυτή η γραμμή κώδικα διαβάζει το δείγμα αρχείου Excel και το προετοιμάζει για επεξεργασία. 

## Βήμα 3: Πρόσβαση στο Φύλλο Εργασίας

Κάθε βιβλίο εργασίας αποτελείται από φύλλα. Πρέπει να έχετε πρόσβαση στο συγκεκριμένο φύλλο εργασίας όπου θέλετε να καταργήσετε τη συγχώνευση των κελιών. Δείτε πώς μπορείτε να το κάνετε αυτό:

```csharp
// Δημιουργήστε ένα Φύλλο Εργασίας και λάβετε το πρώτο φύλλο
Worksheet worksheet = wbk.Worksheets[0];
```

Αυτός ο κώδικας καταγράφει το πρώτο φύλλο εργασίας. Εάν τα συγχωνευμένα κελιά σας βρίσκονται σε διαφορετικό φύλλο εργασίας, ενημερώστε το ευρετήριο ανάλογα.

## Βήμα 4: Πρόσβαση σε κελιά στο φύλλο εργασίας

Στη συνέχεια, θα χρειαστεί να βρείτε μια αναφορά στα κελιά στο φύλλο εργασίας σας. Αυτό μπορεί να επιτευχθεί χρησιμοποιώντας:

```csharp
// Δημιουργήστε ένα αντικείμενο Cells για να ανακτήσετε όλα τα κελιά
Cells cells = worksheet.Cells;
```

Με αυτήν τη γραμμή, έχετε πλέον πρόσβαση σε όλα τα κελιά του φύλλου εργασίας, επιτρέποντάς σας να τα χειριστείτε όπως απαιτείται.

## Βήμα 5: Αποσυγχώνευση των κελιών

Εδώ έρχεται το κρίσιμο βήμα—η κατάργηση της συγχώνευσης των κελιών! Θα πρέπει να καθορίσετε το εύρος των συγχωνευμένων κελιών που θέλετε να καταργήσετε τη συγχώνευση. Χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
// Αποσυγχώνευση των κελιών
cells.UnMerge(5, 2, 2, 3);
```

Σε αυτό το παράδειγμα, το `UnMerge` Η μέθοδος λαμβάνει τέσσερις παραμέτρους: τον δείκτη αρχικής γραμμής (5), τον δείκτη αρχικής στήλης (2), τον αριθμό των γραμμών προς κατάργηση της συγχώνευσης (2) και τον αριθμό των στηλών προς κατάργηση της συγχώνευσης (3). Προσαρμόστε αυτές τις παραμέτρους ώστε να ταιριάζουν με τα συγκεκριμένα συγχωνευμένα κελιά στο αρχείο Excel σας.

## Βήμα 6: Αποθήκευση του βιβλίου εργασίας

Μετά την αποσυγχώνευση, θα πρέπει να αποθηκεύσετε τις αλλαγές σας σε ένα νέο αρχείο Excel. Δείτε πώς μπορείτε να το κάνετε αυτό:

```csharp
// Αποθήκευση του αρχείου
wbk.Save(outputDir + "outputUnMergingtheMergedCells.xlsx");
```

Αυτή η γραμμή αποθηκεύει τα μη συγχωνευμένα δεδομένα σας στον καθορισμένο κατάλογο εξόδου. Τόσο απλά!

## Βήμα 7: Επιβεβαίωση της διαδικασίας

Τέλος, είναι καλή ιδέα να επιβεβαιώσετε ότι όλα πήγαν ομαλά. Μπορείτε να εκτυπώσετε ένα μήνυμα στην κονσόλα που θα σας ενημερώνει ότι η λειτουργία εκτελέστηκε με επιτυχία:
