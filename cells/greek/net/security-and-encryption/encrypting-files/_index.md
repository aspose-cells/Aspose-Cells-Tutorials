Στο επάνω μέρος του κύριου αρχείου του προγράμματος, προσθέστε την ακόλουθη γραμμή για να συμπεριλάβετε τον χώρο ονομάτων Aspose.Cells:

using System.IO;

Τώρα, ας φτάσουμε στην ουσία της εργασίας μας: την κρυπτογράφηση ενός αρχείου Excel. Ακολουθήστε αυτά τα λεπτομερή βήματα για να δημιουργήσετε ένα κρυπτογραφημένο αρχείο Excel.
## Βήμα 1: Ορίστε τον κατάλογο εγγράφων σας
Πρώτα απ 'όλα, ας προετοιμάσουμε μια διαδρομή για τα έγγραφά σας στο Excel. Εδώ θα αποθηκεύσετε τα αρχεία εισόδου και εξόδου.
```csharp
string dataDir = "Your Document Directory";
```
Εδώ, αντικαταστήστε `"Your Document Directory"` με μια πραγματική διαδρομή όπου βρίσκεται το αρχείο Excel σας και όπου θέλετε να αποθηκεύσετε το κρυπτογραφημένο αρχείο.
## Βήμα 2: Δημιουργία αντικειμένου βιβλίου εργασίας
Τώρα, ας δημιουργήσουμε ένα αντικείμενο βιβλίου εργασίας για να λειτουργήσει με το αρχείο Excel σας.
```csharp
Workbook workbook = new Workbook(dataDir + "Book1.xls");
```
Αυτή η γραμμή κώδικα ανοίγει το καθορισμένο αρχείο Excel (`Book1.xls`) ώστε να μπορείτε να ξεκινήσετε να κάνετε αλλαγές. Σκεφτείτε το σαν να ανοίγετε ένα βιβλίο που θέλετε να επεξεργαστείτε.
## Βήμα 3: Καθορισμός επιλογών κρυπτογράφησης
Στη συνέχεια, ήρθε η ώρα να ορίσετε τις επιλογές κρυπτογράφησης. Δείτε πώς μπορείτε να το κάνετε:

Έχετε επιλογές όσον αφορά την κρυπτογράφηση στο Aspose.Cells. Για αυτό το παράδειγμα, θα ορίσετε κρυπτογράφηση XOR και Strong Cryptographic Provider. 
```csharp
// Καθορίστε τον τύπο κρυπτογράφησης XOR.
workbook.SetEncryptionOptions(EncryptionType.XOR, 40);
// Καθορίστε τον τύπο Ισχυρής Κρυπτογράφησης (RC4, Microsoft Strong Cryptographic Provider).
workbook.SetEncryptionOptions(EncryptionType.StrongCryptographicProvider, 128);
```
Σκεφτείτε αυτές τις επιλογές όπως το είδος των κλειδαριών που μπορείτε να χρησιμοποιήσετε—μερικές είναι μικρότερες και πιο εύκολες στην κλειδαριά (XOR), ενώ άλλες είναι πολύ πιο απαιτητικές (Strong Cryptographic Provider).
## Βήμα 4: Προστασία του αρχείου με κωδικό πρόσβασης
Τώρα, ας προσθέσουμε έναν κωδικό πρόσβασης στο αρχείο σας. Αυτό είναι το μυστικό κλειδί που θα κλειδώσει την πόρτα:
```csharp
workbook.Settings.Password = "1234";
```
Μη διστάσετε να αλλάξετε `"1234"` σε οποιονδήποτε κωδικό πρόσβασης προτιμάτε. Απλώς να θυμάστε, όσο ισχυρότερος είναι ο κωδικός πρόσβασης, τόσο καλύτερη είναι η προστασία!
## Βήμα 5: Αποθήκευση του κρυπτογραφημένου αρχείου Excel
Τέλος, ας αποθηκεύσουμε τις αλλαγές για να δημιουργήσουμε το κρυπτογραφημένο αρχείο σας.
```csharp
workbook.Save(dataDir + "encryptedBook1.out.xls");
```
Αυτή η γραμμή κώδικα αποθηκεύει το βιβλίο εργασίας ως `encryptedBook1.out.xls` στον καθορισμένο κατάλογό σας. Είναι σαν να βάζετε το βιβλίο πίσω στο ράφι, κλειδωμένο με ασφάλεια!
