# Πώς να εισαγάγετε πίνακες δεδομένων με μορφοποίηση HTML στο Excel με το Aspose.Cells για .NET

## Εισαγωγή

Δυσκολεύεστε με τη μη αυτόματη μορφοποίηση εισαγόμενων δεδομένων ιστοσελίδας ή βάσης δεδομένων στο Excel; Δεν είστε μόνοι! Οι προγραμματιστές συχνά χρειάζεται να διατηρούν στυλ κειμένου όπως έντονη και πλάγια γραφή, τα οποία είναι απαραίτητα για την αναγνωσιμότητα. Με το Aspose.Cells για .NET, η εισαγωγή ενός DataTable που περιέχει συμβολοσειρές με μορφοποίηση HTML σε ένα βιβλίο εργασίας του Excel, διατηρώντας παράλληλα το στυλ, γίνεται πανεύκολη.

Σε αυτό το σεμινάριο, θα μάθετε πώς να εισάγετε δεδομένα σε μορφή HTML από έναν Πίνακα Δεδομένων στο Excel χρησιμοποιώντας το Aspose.Cells, διασφαλίζοντας ότι τα δεδομένα σας εμφανίζονται ακριβώς όπως προορίζονται στα υπολογιστικά φύλλα.

**Τι θα μάθετε:**
- Ρύθμιση και ρύθμιση παραμέτρων του Aspose.Cells για .NET
- Εισαγωγή πινάκων δεδομένων με μορφοποίηση HTML χρησιμοποιώντας το Aspose.Cells
- Αυτόματη προσαρμογή μεγέθους γραμμών και στηλών ώστε να ταιριάζει στο περιεχόμενο
- Αποθήκευση βιβλίων εργασίας σε πολλαπλές μορφές, όπως XLSX και ODS

Ας ξεκινήσουμε διασφαλίζοντας ότι έχετε τις απαραίτητες προϋποθέσεις!

## Οδηγός Εφαρμογής

Ας αναλύσουμε την εισαγωγή DataTables με μορφοποίηση HTML σε σαφή βήματα.

### Προετοιμασία της πηγής δεδομένων σας

**Επισκόπηση:**
Ξεκινήστε ρυθμίζοντας έναν Πίνακα Δεδομένων με δείγματα δεδομένων που περιλαμβάνουν συμβολοσειρές με μορφοποίηση HTML για να δείξετε τη δυνατότητα στυλιζαρίσματος του Aspose.Cells.
```csharp
using System.Data;

// Ορίστε τους καταλόγους προέλευσης και εξόδου εδώ
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string OutputDir = "YOUR_OUTPUT_DIRECTORY";

// Προετοιμάστε έναν Πίνακα Δεδομένων με ορισμένες τιμές σε μορφή HTML
dataTable = new DataTable("Products");
dataTable.Columns.Add("Product ID", typeof(Int32));
dataTable.Columns.Add("Product Name", typeof(string));
dataTable.Columns.Add("Units In Stock", typeof(Int32));

// Προσθήκη γραμμών με μορφοποίηση HTML
DataRow dr = dataTable.NewRow();
dr[0] = 1;
dr[1] = "<i>Aniseed</i> Syrup"; // Πλάγια γραφή HTML για το όνομα του προϊόντος
dr[2] = 15;
dataTable.Rows.Add(dr);

dr = dataTable.NewRow();
dr[0] = 2;
dr[1] = "<b>Boston Crab Meat</b>"; // HTML έντονη γραφή για το όνομα του προϊόντος
dr[2] = 123;
dataTable.Rows.Add(dr);
```

### Ρύθμιση επιλογών εισαγωγής

**Ρύθμιση παραμέτρων επιλογών εισαγωγής πίνακα:**
Χρήση `ImportTableOptions` για να καθορίσετε ότι οι τιμές των κελιών θα πρέπει να ερμηνεύονται ως συμβολοσειρές HTML.
```csharp
// Δημιουργήστε επιλογές εισαγωγής για τη διαχείριση συμβολοσειρών με μορφή HTML
ImportTableOptions importOptions = new ImportTableOptions();
importOptions.IsFieldNameShown = true; // Συμπερίληψη κεφαλίδων στηλών στην εισαγωγή
importOptions.IsHtmlString = true; // Ερμηνεία τιμών κελιών ως συμβολοσειρές HTML
```

### Εισαγωγή δεδομένων στο Excel

**Επισκόπηση:**
Δημιουργήστε ένα βιβλίο εργασίας και ένα φύλλο εργασίας και, στη συνέχεια, χρησιμοποιήστε `ImportData` για να μεταφέρετε τον Πίνακα Δεδομένων σας στο Excel με ανέπαφη τη μορφοποίηση.
```csharp
// Δημιουργήστε ένα βιβλίο εργασίας και λάβετε το πρώτο φύλλο εργασίας
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];

// Εισαγάγετε τον Πίνακα Δεδομένων ξεκινώντας από τη γραμμή 0, τη στήλη 0
worksheet.Cells.ImportData(dataTable, 0, 0, importOptions);

// Προσαρμόστε τα μεγέθη γραμμών και στηλών για καλύτερη αναγνωσιμότητα
worksheet.AutoFitRows();
worksheet.AutoFitColumns();
```

### Αποθήκευση του βιβλίου εργασίας σας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε μορφή XLSX και ODS για να διασφαλίσετε τη συμβατότητα σε διαφορετικές εφαρμογές υπολογιστικών φύλλων.
```csharp
string output1Path = OutputDir + "Output.out.xlsx";
string output2Path = OutputDir + "Output.out.ods";

// Αποθήκευση του βιβλίου εργασίας σε δύο μορφές
workbook.Save(output1Path);
workbook.Save(output2Path);
```
