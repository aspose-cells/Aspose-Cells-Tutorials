## Οδηγός Εφαρμογής

Αυτή η ενότητα χωρίζεται σε δύο κύριες λειτουργίες: τη ρύθμιση του DataTable και του DataGrid, ακολουθούμενη από την εισαγωγή αυτών των δεδομένων σε ένα αρχείο Excel.

### Ρύθμιση DataTable και DataGrid

**Επισκόπηση**Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε έναν Πίνακα Δεδομένων, να τον συμπληρώσετε με δείγματα δεδομένων και να τον συνδέσετε με ένα Πλέγμα Δεδομένων για περαιτέρω χειρισμό ή εμφάνιση στην εφαρμογή σας.

#### Βήμα 1: Δημιουργία και συμπλήρωση ενός αντικειμένου DataTable
```java
DataTable dataTable = new DataTable("Products");
dataTable.Columns.Add("Product ID", Integer.class);
dataTable.Columns.Add("Product Name", String.class);
dataTable.Columns.Add("Units In Stock", Integer.class);

DataRow dr = dataTable.NewRow();
dr[0] = 1;
dr[1] = "Aniseed Syrup";
dr[2] = 15;
dataTable.Rows.Add(dr);

// Προσθήκη άλλης γραμμής στον Πίνακα Δεδομένων
dr = dataTable.NewRow();
dr[0] = 2;
dr[1] = "Boston Crab Meat";
dr[2] = 123;
dataTable.Rows.Add(dr);
```

#### Βήμα 2: Σύνδεση του DataTable με ένα DataGrid
```java
DataGrid dg = new DataGrid();
dg.setDataSource(dataTable);
dg.DataBind();
```

### Εισαγωγή DataGrid σε βιβλίο εργασίας Excel

**Επισκόπηση**Αυτή η λειτουργία δείχνει πώς να λαμβάνετε δεδομένα από το DataGrid σας και να τα εξάγετε σε ένα φύλλο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET.

#### Βήμα 1: Δημιουργήστε ένα νέο βιβλίο εργασίας και αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας
```java
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);
```

#### Βήμα 2: Εισαγωγή περιεχομένων DataGrid στο φύλλο εργασίας
```java
Cells cells = worksheet.getCells();
cells.importDataGrid(dg, 0, 0, false); // Ξεκινώντας από το κελί A1
```

#### Βήμα 3: Αποθήκευση του βιβλίου εργασίας σε έναν καθορισμένο κατάλογο
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
workbook.save(outputDir + "/output.xlsx");
```
