### Εξαγωγή δεδομένων σε DataTable

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να λαμβάνετε συγκεκριμένες περιοχές κελιών από ένα φύλλο εργασίας του Excel και να τις εξάγετε ως Πίνακα Δεδομένων, επιτρέποντας πιο ευέλικτο χειρισμό δεδομένων σε εφαρμογές .NET.

**Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας**
Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Workbook` κλάση χρησιμοποιώντας την καθορισμένη διαδρομή αρχείου. Αυτό το βήμα αποκτά πρόσβαση στο αρχείο Excel μέσω προγραμματισμού.

```csharp
using Aspose.Cells;
string SourceDir = "YOUR_SOURCE_DIRECTORY";
string filePath = SourceDir + "Book1.xlsx";

// Δημιουργία νέας παρουσίας της κλάσης Βιβλίου Εργασίας.
Workbook workbook = new Workbook(filePath);
```

**Βήμα 2: Πρόσβαση στο Φύλλο Εργασίας**
Στη συνέχεια, αποκτήστε πρόσβαση στο φύλλο εργασίας που περιέχει τα δεδομένα που θέλετε να εξαγάγετε. Εδώ έχουμε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

**Βήμα 3: Εξαγωγή δεδομένων από κελιά**
Τέλος, μετατρέψτε μια περιοχή κελιών σε έναν Πίνακα Δεδομένων. Αυτό το παράδειγμα εξάγει 11 γραμμές και 2 στήλες ξεκινώντας από το πρώτο κελί (με δείκτη 0).

```csharp
using System.Data;

// Εξαγωγή των δεδομένων σε έναν Πίνακα Δεδομένων.
DataTable dataTable = worksheet.Cells.ExportDataTableAsString(0, 0, 11, 2, true);

// Επανάληψη κάθε γραμμής στον Πίνακα Δεδομένων.
foreach (DataRow r in dataTable.Rows)
{
    foreach (DataColumn c in dataTable.Columns)
    {
        string value = r.Field<string>(c);
        // Επεξεργαστείτε την τιμή κελιού όπως απαιτείται
    }
}
```

- **Διασφαλίστε την ακρίβεια της διαδρομής αρχείου**: Οι λανθασμένες διαδρομές θα οδηγήσουν σε `FileNotFoundException`.
