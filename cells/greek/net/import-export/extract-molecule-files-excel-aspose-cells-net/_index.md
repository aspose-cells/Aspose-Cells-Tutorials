## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει το Aspose.Cells, ας εξαγάγουμε αυτά τα ενσωματωμένα αρχεία μορίων.

### Εξαγωγή ενσωματωμένων αρχείων μορίων από το Excel

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να ανακτάτε μέσω προγραμματισμού `.mol` αρχεία που είναι αποθηκευμένα ως OleObjects μέσα σε ένα βιβλίο εργασίας του Excel χρησιμοποιώντας το Aspose.Cells για .NET. Δείτε πώς μπορείτε να το κάνετε:

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας
Ξεκινήστε φορτώνοντας το βιβλίο εργασίας σας που περιέχει ενσωματωμένα μόρια.

```csharp
string sourceDir = @"YOUR_SOURCE_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου προέλευσης
string outputDir = @"YOUR_OUTPUT_DIRECTORY";  // Αντικαταστήστε με τη διαδρομή του καταλόγου εξόδου σας

Workbook workbook = new Workbook(sourceDir + "EmbeddedMolSample.xlsx");
```

#### Βήμα 2: Επαναλάβετε τα φύλλα εργασίας και τα OleObjects
Περιηγηθείτε σε κάθε φύλλο εργασίας στο βιβλίο εργασίας για να αποκτήσετε πρόσβαση σε ενσωματωμένα αντικείμενα.

```csharp
var index = 1;
foreach (Worksheet sheet in workbook.Worksheets)
{
    OleObjectCollection oles = sheet.OleObjects; // Λήψη όλων των αντικειμένων Ole από το φύλλο εργασίας
    
    foreach (OleObject ole in oles)
    {
        string fileName = outputDir + "OleObject" + index + ".mol";
        
        using (FileStream fs = File.Create(fileName))
        {
            fs.Write(ole.ObjectData, 0, ole.ObjectData.Length); // Εγγραφή των δεδομένων του ενσωματωμένου αντικειμένου σε ένα αρχείο
        }
        index++;
    }
}
```

#### Εξήγηση
- **Βιβλίο ασκήσεων**: Αντιπροσωπεύει το βιβλίο εργασίας του Excel και λειτουργεί ως σημείο εισόδου για χειρισμό.
- **Συλλογή OleObject**Μια συλλογή αντικειμένων OLE σε κάθε φύλλο εργασίας.
- **FileStream**: Χρησιμοποιείται για τη δημιουργία αρχείων όπου εξάγεται `.mol` τα δεδομένα γράφονται.

### Συμβουλές αντιμετώπισης προβλημάτων
