### Εξαγωγή τύπων Power Query
Το Power Query στο Excel επιτρέπει στους χρήστες να συνδέονται σε ένα ευρύ φάσμα προελεύσεων δεδομένων. Με το Aspose.Cells, μπορείτε να αποκτήσετε πρόσβαση σε αυτές τις συνδέσεις μέσω προγραμματισμού.

#### Βήμα 1: Φόρτωση του βιβλίου εργασίας
Αρχικά, φορτώστε το βιβλίο εργασίας σας που περιέχει τις συνδέσεις OData:

```csharp
string SourceDir = RunExamples.Get_SourceDirectory();
Workbook workbook = new Workbook(SourceDir + "ODataSample.xlsx");
```
Εδώ, `SourceDir` είναι μια μέθοδος για να λάβετε τη διαδρομή του καταλόγου πηγής σας.

#### Βήμα 2: Πρόσβαση σε τύπους Power Query
Στη συνέχεια, αποκτήστε πρόσβαση στη συλλογή τύπων Power Query:

```csharp
PowerQueryFormulaCollection PQFcoll = workbook.DataMashup.PowerQueryFormulas;
```
Αυτό σας δίνει πρόσβαση σε όλα τα Power Queries που έχουν οριστεί στο αρχείο Excel σας.

#### Βήμα 3: Επαναλάβετε τις συνδέσεις
Εκτελέστε επανάληψη σε κάθε σύνδεση για να εξαγάγετε λεπτομέρειες:

```csharp
foreach (PowerQueryFormula PQF in PQFcoll)
{
    Console.WriteLine("Connection Name: " + PQF.Name);
    
    PowerQueryFormulaItemCollection PQFIcoll = PQF.PowerQueryFormulaItems;
    foreach (PowerQueryFormulaItem PQFI in PQFIcoll)
    {
        Console.WriteLine("Name: " + PQFI.Name);
        Console.WriteLine("Value: " + PQFI.Value);
    }
}
```
Αυτός ο κώδικας εκτυπώνει το όνομα κάθε σύνδεσης και τα σχετικά στοιχεία τύπου.
