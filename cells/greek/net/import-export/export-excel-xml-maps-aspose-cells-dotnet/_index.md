### Χαρακτηριστικό: Δημιουργία στιγμιαίου βιβλίου εργασίας και εξαγωγή αντιστοίχισης XML
#### Επισκόπηση
Αυτή η λειτουργία περιλαμβάνει τη φόρτωση ενός βιβλίου εργασίας του Excel και την εξαγωγή των σχετικών αντιστοιχίσεων XML. Αυτό είναι ιδιαίτερα χρήσιμο για τη μετεγκατάσταση δεδομένων ή κατά την εργασία με συστήματα που απαιτούν συγκεκριμένες μορφές XML που προέρχονται από φύλλα Excel.

#### Βήμα 1: Δημιουργία αντικειμένου βιβλίου εργασίας
Ξεκινήστε δημιουργώντας ένα `Workbook` αντικείμενο. Αυτό θα φορτώσει το καθορισμένο αρχείο Excel, καθιστώντας το έτοιμο για περαιτέρω χειρισμό.
```csharp
using Aspose.Cells;

string SourceDir = "YOUR_SOURCE_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Φόρτωση του βιβλίου εργασίας από τον καθορισμένο κατάλογο
Workbook workbook = new Workbook(SourceDir + "/sampleExportXmlMapFromWorkbook.xlsx");
```
**Εξήγηση:** Ο `Workbook` Ο κατασκευαστής δέχεται μια διαδρομή αρχείου ως όρισμα, φορτώνοντας το έγγραφο Excel στη μνήμη.

#### Βήμα 2: Επανάληψη μέσω χαρτών XML
Στη συνέχεια, κάντε επανάληψη σε όλες τις διαθέσιμες αντιστοιχίσεις XML μέσα στο βιβλίο εργασίας σας. Αυτό σας επιτρέπει να έχετε πρόσβαση και να εξάγετε κάθε αντιστοιχία ξεχωριστά.
```csharp
// Πρόσβαση και επανάληψη σε όλα τα XmlMaps στο Βιβλίο Εργασίας
global for (int i = 0; i < workbook.Worksheets.XmlMaps.Count; i++) {
    // Ανάκτηση κάθε XmlMap με βάση τον δείκτη του
    XmlMap map = workbook.Worksheets.XmlMaps[i];
    
    // Εξαγωγή των δεδομένων XML από αυτό το XmlMap σε ένα μεμονωμένο αρχείο
    workbook.ExportXml(map.Name, outputDir + "/" + map.Name + ".xml");
}
```
**Εξήγηση:** `workbook.Worksheets.XmlMaps` παρέχει μια συλλογή όλων των χαρτών XML. Κάθε χάρτης μπορεί να προσπελαστεί από το ευρετήριό του και να εξαχθεί χρησιμοποιώντας το `ExportXml` μέθοδος.
