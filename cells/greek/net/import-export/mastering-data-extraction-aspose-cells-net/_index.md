### Ανάκτηση δεδομένων από κελιά
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να κάνετε επανάληψη σε κάθε κελί σε ένα φύλλο εργασίας και να ανακτήσετε την τιμή του με βάση τον τύπο—συμβολοσειρά, αριθμητική, λογική τιμή ή ημερομηνία/ώρα. 

#### Βήματα Υλοποίησης
**Βήμα 1: Ρύθμιση του έργου σας**
Βεβαιωθείτε ότι το έργο σας αναφέρεται στο Aspose.Cells και ότι το περιβάλλον σας έχει ρυθμιστεί σύμφωνα με την ενότητα προαπαιτούμενων.

**Βήμα 2: Αρχικοποίηση βιβλίου εργασίας και φύλλου εργασίας της Access**
Δημιουργήστε μια νέα παρουσία του `Workbook` για να ανοίξετε ένα υπάρχον αρχείο Excel και να αποκτήσετε πρόσβαση στο πρώτο φύλλο εργασίας του:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = RunExamples.GetDataDir(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);

// Άνοιγμα ενός υπάρχοντος βιβλίου εργασίας
Workbook workbook = new Workbook(dataDir + "book1.xls");

// Πρόσβαση στο πρώτο φύλλο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```

**Βήμα 3: Ανάκτηση και εκτύπωση δεδομένων κελιού**
Εκτελέστε επανάληψη σε κάθε κελί στο φύλλο εργασίας, προσδιορίστε τον τύπο δεδομένων του και εκτυπώστε την τιμή ανάλογα:

```csharp
foreach (Cell cell1 in worksheet.Cells)
{
    // Μεταβλητές για την αποθήκευση τιμών διαφορετικών τύπων δεδομένων
    string stringValue;
    double doubleValue;
    bool boolValue;
    DateTime dateTimeValue;

    // Διαβίβαση του τύπου των δεδομένων που περιέχονται στο κελί για αξιολόγηση
    switch (cell1.Type)
    {
        case CellValueType.IsString:
            stringValue = cell1.StringValue;
            Console.WriteLine("String Value: " + stringValue);
            break;

        case CellValueType.IsNumeric:
            doubleValue = cell1.DoubleValue;
            Console.WriteLine("Double Value: " + doubleValue);
            break;

        case CellValueType.IsBool:
            boolValue = cell1.BoolValue;
            Console.WriteLine("Bool Value: " + boolValue);
            break;

        case CellValueType.IsDateTime:
            dateTimeValue = cell1.DateTimeValue;
            Console.WriteLine("DateTime Value: " + dateTimeValue);
            break;

        case CellValueType.IsUnknown:
            stringValue = cell1.StringValue;
            Console.WriteLine("Unknown Value: " + stringValue);
            break;

        case CellValueType.IsNull:
            // Δεν απαιτείται ενέργεια για null κελιά
            break;
    }
}
```
