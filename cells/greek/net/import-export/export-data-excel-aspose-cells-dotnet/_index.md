## Οδηγός Εφαρμογής (H2)

### Εξαγωγή δεδομένων από το Excel (H2)

Ο κύριος στόχος είναι η αποτελεσματική εξαγωγή δεδομένων από ένα φύλλο εργασίας του Excel και η εξαγωγή τους σε έναν Πίνακα Δεδομένων. Ας δούμε πώς μπορεί να επιτευχθεί αυτό με το Aspose.Cells.

#### Βήμα 1: Ρύθμιση του περιβάλλοντος σας

Ορίστε διαδρομές και δημιουργήστε μια ροή αρχείων για το αρχείο Excel:
```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας.
string dataDir = "path/to/your/excel/files/";

// Δημιουργήστε ένα FileStream για να ανοίξετε ένα αρχείο Excel.
FileStream fstream = new FileStream(dataDir + "book1.xls", FileMode.Open);

// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας με τη ροή αρχείων.
Workbook workbook = new Workbook(fstream);
```

#### Βήμα 2: Πρόσβαση στο φύλλο εργασίας και εξαγωγή δεδομένων

Αποκτήστε πρόσβαση στο φύλλο εργασίας σας και εξαγάγετε το επιθυμητό εύρος δεδομένων σε έναν Πίνακα Δεδομένων:
```csharp
// Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας.
Worksheet worksheet = workbook.Worksheets[0];

// Εξαγωγή περιεχομένων καθορισμένων γραμμών και στηλών σε έναν Πίνακα Δεδομένων.
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, 7, 2, true);

System.Console.WriteLine("Number of Rows in Data Table: " + dataTable.Rows.Count);
```

#### Εξήγηση
- **Μέθοδος ExportDataTable**Αυτή η μέθοδος εξάγει δεδομένα από το καθορισμένο εύρος (αρχική γραμμή, αρχική στήλη, σύνολο γραμμών, σύνολο στηλών) σε έναν Πίνακα Δεδομένων.
- **Παράμετροι**:
  - `startRow`Ο δείκτης αρχικής γραμμής.
  - `startColumn`: Ο δείκτης αρχικής στήλης.
  - `totalRows`: Αριθμός γραμμών προς εξαγωγή.
  - `totalColumns`: Αριθμός στηλών προς εξαγωγή.
  - `convertStringToNumeric`Μετατρέπει συμβολοσειρές που αναπαριστούν αριθμούς σε αριθμητικούς τύπους δεδομένων.

#### Βήμα 3: Καθαρισμός πόρων

Να βεβαιώνεστε πάντα ότι κλείνετε τυχόν ανοιχτές ροές αρχείων σε ελεύθερους πόρους:
```csharp
// Κλείστε το FileStream μετά τη χρήση.
fstream.Close();
```
