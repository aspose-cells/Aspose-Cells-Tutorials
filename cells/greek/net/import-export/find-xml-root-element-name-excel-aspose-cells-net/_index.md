## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία εύρεσης του ονόματος του ριζικού στοιχείου ενός χάρτη XML σε διαχειρίσιμα βήματα.

### Φόρτωση του αρχείου Excel

Ξεκινήστε φορτώνοντας το αρχείο Excel που περιέχει τον χάρτη XML:

```csharp
// Διαδρομή καταλόγου πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Φόρτωση του δείγματος αρχείου Excel
Workbook workbook = new Workbook(sourceDir + "sampleRootElementNameOfXmlMap.xlsx");
```

**Γιατί:** Η φόρτωση του βιβλίου εργασίας είναι απαραίτητη για την πρόσβαση στο περιεχόμενό του, συμπεριλαμβανομένων τυχόν συσχετισμένων αντιστοιχίσεων XML.

### Πρόσβαση στον χάρτη XML

Στη συνέχεια, ανακτήστε την πρώτη αντιστοίχιση XML από το βιβλίο εργασίας:

```csharp
// Λήψη του πρώτου αντικειμένου XmlMap από τη συλλογή
XmlMap xmlMap = workbook.Worksheets.XmlMaps[0];
```

**Γιατί:** Το Excel μπορεί να περιέχει πολλαπλές αντιστοιχίσεις XML. Η πρόσβαση σε αυτές απαιτεί την προσθήκη ευρετηρίου στη συλλογή τους.

### Εξαγωγή του ονόματος του ριζικού στοιχείου

Τέλος, εκτυπώστε το όνομα του ριζικού στοιχείου της αντιστοίχισης XML:

```csharp
// Εκτυπώστε το όνομα του ριζικού στοιχείου στην κονσόλα
Console.WriteLine("Root Element Name Of Xml Map: " + xmlMap.RootElementName);
```

**Γιατί:** Ο `RootElementName` Η ιδιότητα παρέχει έναν γρήγορο τρόπο για να αναγνωρίσετε τον κύριο κόμβο στη δομή XML σας, κάτι που είναι χρήσιμο για περαιτέρω επεξεργασία.
