## Οδηγός Εφαρμογής
### Εξαγωγή γραφήματος Excel ως SVG
Ο κύριος στόχος είναι η εξαγωγή ενός γραφήματος από ένα βιβλίο εργασίας του Excel σε ένα αρχείο SVG χρησιμοποιώντας το Aspose.Cells. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### 1. Φορτώστε το Βιβλίο Εργασίας και αποκτήστε πρόσβαση στο Φύλλο Εργασίας
Ξεκινήστε φορτώνοντας το αρχείο Excel σε ένα `Workbook` αντικείμενο και αποκτήστε πρόσβαση στο επιθυμητό φύλλο εργασίας που περιέχει το γράφημα.
```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
string outputDir = RunExamples.Get_OutputDirectory();

// Δημιουργία βιβλίου εργασίας από ένα υπάρχον αρχείο Excel
Workbook workbook = new Workbook(sourceDir + "sampleExportChartToSvgWithViewBox.xlsx");

// Πρόσβαση στο πρώτο φύλλο εργασίας
Worksheet worksheet = workbook.Worksheets[0];
```
#### 2. Πρόσβαση και ρύθμιση παραμέτρων επιλογών εξαγωγής γραφήματος
Προσδιορίστε το γράφημα που θέλετε να εξαγάγετε και, στη συνέχεια, διαμορφώστε το χρησιμοποιώντας `ImageOrPrintOptions`.
```csharp
Aspose.Cells.Charts.Chart chart = worksheet.Charts[0];

// Ρύθμιση επιλογών εικόνας ή εκτύπωσης με ενεργοποιημένο το SVGFitToViewPort
Aspose.Cells.Rendering.ImageOrPrintOptions opts = new Aspose.Cells.Rendering.ImageOrPrintOptions();
opts.SaveFormat = SaveFormat.Svg;
opts.SVGFitToViewPort = true; // Διασφαλίζει ότι το γράφημα ταιριάζει στο παράθυρο προβολής
```
#### 3. Εξαγωγή του γραφήματος σε SVG
Τέλος, αποθηκεύστε το διάγραμμα ως αρχείο SVG.
```csharp
// Αποθήκευση του γραφήματος σε μορφή SVG
cart.ToImage(outputDir + "outputExportChartToSvgWithViewBox.svg", opts);

Console.WriteLine("ExportChartToSvgWithViewBox executed successfully.");
```
### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του αρχείου προέλευσης Excel είναι σωστή.
- Ελέγξτε αν `SVGFitToViewPort` έχει οριστεί σε true για σωστή κλιμάκωση.
