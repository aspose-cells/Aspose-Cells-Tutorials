## Οδηγός Εφαρμογής

### Επισκόπηση
Θα εξερευνήσουμε πώς να φορτώσετε ένα βιβλίο εργασίας του Excel και να το εξαγάγετε σε μορφή HTML, διασφαλίζοντας παράλληλα ότι τα σχόλια διατηρούνται.

### Οδηγίες βήμα προς βήμα

#### Φόρτωση του βιβλίου εργασίας
Ξεκινήστε φορτώνοντας το αρχείο προέλευσης Excel:

```csharp
// Κατάλογος πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Φόρτωση δείγματος αρχείου Excel
Workbook wb = new Workbook(sourceDir + "sampleExportCommentsHTML.xlsx");
```
Εδώ, `RunExamples.Get_SourceDirectory()` είναι μια βοηθητική συνάρτηση για την ανάκτηση της διαδρομής των αρχείων πηγής σας.

#### Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML
Για να εξαγάγετε σχόλια, ορίστε το `IsExportComments` ιδιοκτησία:

```csharp
// Εξαγωγή σχολίων - ορισμός της ιδιότητας IsExportComments σε true
HtmlSaveOptions opts = new HtmlSaveOptions();
opts.IsExportComments = true;
```
Αυτή η ρύθμιση παραμέτρων διασφαλίζει ότι τυχόν σχόλια στο αρχείο Excel σας περιλαμβάνονται στην έξοδο HTML.

#### Αποθήκευση ως HTML
Τέλος, αποθηκεύστε το βιβλίο εργασίας ως αρχείο HTML:

```csharp
// Κατάλογος εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση του αρχείου Excel σε μορφή HTML
wb.Save(outputDir + "outputExportCommentsHTML.html", opts);

Console.WriteLine("ExportCommentsWhileSavingExcelFileToHtml executed successfully.\r\n");
```
