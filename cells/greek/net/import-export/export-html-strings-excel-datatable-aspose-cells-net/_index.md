## Οδηγός Εφαρμογής
### Φόρτωση του αρχείου Excel
Φορτώστε το δείγμα αρχείου Excel χρησιμοποιώντας το `Workbook` τάξη.
**Βήμα 1: Φόρτωση δείγματος αρχείου Excel**
```csharp
// Κατάλογος πηγής
string sourceDir = RunExamples.Get_SourceDirectory();

// Φόρτωση δείγματος αρχείου Excel
Workbook wb = new Workbook(sourceDir + "sampleExportTableAsHtmlString.xlsx");
```
### Πρόσβαση στο Φύλλο Εργασίας
Αποκτήστε πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας στο βιβλίο εργασίας του Excel ως εξής:
**Βήμα 2: Πρόσβαση στο πρώτο φύλλο εργασίας**
```csharp
// Πρώτο φύλλο εργασίας της Access
Worksheet ws = wb.Worksheets[0];
```
### Ρύθμιση παραμέτρων επιλογών εξαγωγής
Ρυθμίστε τις παραμέτρους εξαγωγής για να καθορίσετε την εξαγωγή δεδομένων ως συμβολοσειρές HTML.
**Βήμα 3: Ρύθμιση παραμέτρων ExportTableOptions**
```csharp
// Καθορίστε τις επιλογές πίνακα εξαγωγής και ορίστε την τιμή ExportAsHtmlString σε true
ExportTableOptions opts = new ExportTableOptions();
opts.ExportColumnName = false;
opts.ExportAsHtmlString = true;
```
### Εξαγωγή δεδομένων
Εξαγωγή δεδομένων από την καθορισμένη περιοχή κελιών σε έναν Πίνακα Δεδομένων.
**Βήμα 4: Εξαγωγή κελιών σε DataTable**
```csharp
// Εξαγωγή των δεδομένων κελιών σε πίνακα δεδομένων με τις καθορισμένες επιλογές πίνακα εξαγωγής
DataTable dt = ws.Cells.ExportDataTable(0, 0, 3, 3, opts);
```
### Εμφάνιση τιμών συμβολοσειράς HTML
Εκτυπώστε την τιμή συμβολοσειράς HTML από ένα συγκεκριμένο κελί στον Πίνακα Δεδομένων.
**Βήμα 5: Εκτύπωση τιμής συμβολοσειράς HTML κελιού**
```csharp
// Εκτυπώστε την τιμή συμβολοσειράς html του κελιού που βρίσκεται στην τρίτη γραμμή και τη δεύτερη στήλη 
Console.WriteLine(dt.Rows[2][1].ToString());
```
