## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει τα βήματα για την υλοποίηση της λειτουργίας προσαρμοσμένης υπηρεσίας παροχής ροής χρησιμοποιώντας διαχειρίσιμες εργασίες.

### Υλοποίηση παρόχου ροής

#### Επισκόπηση
Ένας προσαρμοσμένος πάροχος ροής διαχειρίζεται εξωτερικούς πόρους, όπως εικόνες, μέσα σε ένα βιβλίο εργασίας του Excel. Αυτό περιλαμβάνει τη δημιουργία μιας κλάσης που υλοποιεί `IStreamProvider`.

#### Βήματα για την Υλοποίηση
**1. Ορίστε την κλάση παρόχου προσαρμοσμένης ροής**
Δημιουργήστε μια νέα κλάση με το όνομα `StreamProvider` εφαρμογή `IStreamProvider`Εδώ, θα χειρίζεστε το άνοιγμα και το κλείσιμο ροών αρχείων για εξωτερικούς πόρους.
```csharp
using System;
using System.IO;
using Aspose.Cells.Rendering;

class StreamProvider : IStreamProvider
{
    public void CloseStream(StreamProviderOptions options)
    {
        // Εφαρμόστε λογική για να κλείσετε τη ροή, εάν είναι απαραίτητο.
    }

    public void InitStream(StreamProviderOptions options)
    {
        FileStream fi = new FileStream(SourceDir + "sampleControlExternalResourcesUsingWorkbookSetting_StreamProvider.png", FileMode.OpenOrCreate, FileAccess.Read);
        options.Stream = fi;
    }
}
```

**2. Έλεγχος εξωτερικών πόρων σε ένα βιβλίο εργασίας**
Χρησιμοποιήστε την προσαρμοσμένη υπηρεσία παροχής ροής για να χειριστείτε εξωτερικούς πόρους μέσα στο βιβλίο εργασίας του Excel:
```csharp
using Aspose.Cells;

void ControlExternalResources()
{
    Workbook wb = new Workbook(SourceDir + "sampleControlExternalResourcesUsingWorkbookSetting_StreamProvider.xlsx");
    wb.Settings.StreamProvider = new StreamProvider();

    Worksheet ws = wb.Worksheets[0];

    ImageOrPrintOptions opts = new ImageOrPrintOptions
    {
        OnePagePerSheet = true,
        ImageType = Drawing.ImageType.Png
    };

    SheetRender sr = new SheetRender(ws, opts);
    sr.ToImage(0, OutputDir + "outputControlExternalResourcesUsingWorkbookSetting_StreamProvider.png");
}
```
