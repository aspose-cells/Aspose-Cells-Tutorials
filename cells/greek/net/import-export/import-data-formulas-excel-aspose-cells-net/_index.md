## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει τον καθορισμό στοιχείων δεδομένων και την εισαγωγή τους σε ένα φύλλο εργασίας του Excel με τύπους.

### Καθορισμός στοιχείων δεδομένων

#### Επισκόπηση

Η δημιουργία και η οργάνωση προσαρμοσμένων αντικειμένων δεδομένων είναι ζωτικής σημασίας πριν από την εισαγωγή. Αυτή η λειτουργία εστιάζει στον ορισμό αυτών των αντικειμένων χρησιμοποιώντας κλάσεις C#.

#### Βήμα προς βήμα εφαρμογή

**Ορισμός κλάσης που ορίζεται από τον χρήστη**

```csharp
using System;
using System.Collections.Generic;

class FeatureSpecifyDataItems
{
    class DataItems
    {
        public int Number1 { get; set; }
        public int Number2 { get; set; }
        public string Formula1 { get; set; }
        public string Formula2 { get; set; }
    }

    public static void Run()
    {
        List<DataItems> dis = new List<DataItems>();

        // Ορισμός ενός στοιχείου δεδομένων
        DataItems di = new DataItems();
        di.Number1 = 2005;
        di.Number2 = 3505;
        di.Formula1 = "+=SUM(A5,B5)"; // Τύπος για το άθροισμα των A5 και B5
        di.Formula2 = "+=HYPERLINK(\"https://www.aspose.com\", \"Ιστότοπος Aspose\")";

        dis.Add(di);
    }
}
```

**Εξήγηση**: 
- Ο `DataItems` Η κλάση περιέχει ακέραιους αριθμούς και τύπους.
- Οι τύποι ορίζονται ως συμβολοσειρές για ευελιξία κατά την εισαγωγή.

### Εισαγωγή δεδομένων σε φύλλο εργασίας με τύπους

#### Επισκόπηση

Αυτή η λειτουργία επιδεικνύει την εισαγωγή των στοιχείων δεδομένων που δημιουργήθηκαν προηγουμένως σε ένα φύλλο εργασίας του Excel, καθορίζοντας ποια πεδία θα πρέπει να αντιμετωπίζονται ως τύποι.

#### Βήμα προς βήμα εφαρμογή

**Εισαγωγή προσαρμοσμένων αντικειμένων**

```csharp
using Aspose.Cells;

class FeatureImportDataWithFormulas
{
    string outputDir = "YOUR_OUTPUT_DIRECTORY";

    public static void Run()
    {
        Workbook wb = new Workbook();
        Worksheet ws = wb.Worksheets[0];

        ImportTableOptions opts = new ImportTableOptions();
        opts.IsFormulas = new bool[] { false, false, true, true };

        List<DataItems> dis = new List<DataItems>(); // Ας υποθέσουμε ότι αυτή η λίστα έχει συμπληρωθεί όπως φαίνεται παραπάνω.
        
        ws.Cells.ImportCustomObjects(dis, 0, 0, opts);
        wb.CalculateFormula();
        ws.AutoFitColumns();

        wb.Save(outputDir + "/outputSpecifyFormulaFieldsWhileImportingDataToWorksheet.xlsx");
    }
}
```

**Εξήγηση**: 
- `ImportTableOptions` καθορίζει ποια πεδία είναι τύποι.
- Οι τύποι υπολογίζονται χρησιμοποιώντας `wb.CalculateFormula()`.
- Οι στήλες προσαρμόζονται αυτόματα για καλύτερη αναγνωσιμότητα.
