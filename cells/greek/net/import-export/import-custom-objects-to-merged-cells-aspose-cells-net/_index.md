## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία εισαγωγής προσαρμοσμένων αντικειμένων σε συγχωνευμένα κελιά.

### Ρύθμιση του έργου σας

Ξεκινήστε δημιουργώντας ένα `Product` κλάση για να αναπαραστήσει το μοντέλο δεδομένων σας. Αυτή θα περιέχει τις ιδιότητες που σκοπεύετε να εισαγάγετε:

```csharp
public class Product
{
    public int ProductId { get; set; }
    public string ProductName { get; set; }
}
```

### Εισαγωγή προσαρμοσμένων αντικειμένων

Δείτε πώς μπορείτε να εφαρμόσετε τη λειτουργικότητα για την εισαγωγή προσαρμοσμένων αντικειμένων σε μια συγχωνευμένη περιοχή σε ένα πρότυπο Excel.

#### Φόρτωση του βιβλίου εργασίας σας

Φορτώστε το βιβλίο εργασίας σας χρησιμοποιώντας το `Workbook` τάξη:

```csharp
string sourceDir = RunExamples.Get_SourceDirectory();
Workbook workbook = new Workbook(sourceDir + "sampleMergedTemplate.xlsx");
```

#### Δημιουργία λίστας προϊόντων

Δημιουργήστε μια λίστα προϊόντων για εισαγωγή:

```csharp
List<Product> productList = new List<Product>();
for (int i = 0; i < 3; i++)
{
    Product product = new Product
    {
        ProductId = i,
        ProductName = "Test Product - " + i
    };
    productList.Add(product);
}
```

#### Ρύθμιση παραμέτρων επιλογών εισαγωγής

Διαμορφώστε το `ImportTableOptions` για τη διαχείριση συγχωνευμένων κελιών:

```csharp
ImportTableOptions tableOptions = new ImportTableOptions();
tableOptions.CheckMergedCells = true;
tableOptions.IsFieldNameShown = false;
```

#### Εισαγωγή δεδομένων

Τέλος, εισαγάγετε τα δεδομένα σας στο φύλλο εργασίας:

```csharp
workbook.Worksheets[0].Cells.ImportCustomObjects((ICollection)productList, 1, 0, tableOptions);
workbook.Save("outputDirectory/sampleMergedTemplate_out.xlsx", SaveFormat.Xlsx);
```
