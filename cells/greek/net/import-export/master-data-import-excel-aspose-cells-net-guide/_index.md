## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση της εισαγωγής προσαρμοσμένων αντικειμένων σε ένα φύλλο Excel.

### Βήμα 1: Ορίστε το προσαρμοσμένο αντικείμενό σας
Ξεκινήστε δημιουργώντας μια κλάση που αντιπροσωπεύει τη δομή δεδομένων σας. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε ένα `Person` κλάση με ιδιότητες για `Name` και `Age`.
```csharp
class Person
{
    int _age;
    string _name;

    public int Age 
    { 
        get => _age; 
        set => _age = value; 
    }
    
    public string Name 
    {
        get => _name;  
        set => _name = value; 
    }

    public Person(string name, int age)
    {
        Age = age;
        Name = name;
    }
}
```
### Βήμα 2: Προετοιμασία των δεδομένων σας
Δημιουργήστε μια λίστα με προσαρμοσμένα αντικείμενα που θέλετε να εισαγάγετε στο Excel.
```csharp
List<Person> people = new List<Person>
{
    new Person("Mike", 25),
    new Person("Steve", 30),
    new Person("Billy", 35)
};
```
### Βήμα 3: Εισαγωγή προσαρμοσμένων αντικειμένων
Διαμορφώστε το `ImportTableOptions` για να καθορίσετε τον τρόπο εισαγωγής των δεδομένων και στη συνέχεια να χρησιμοποιήσετε το `ImportCustomObjects` μέθοδος.
```csharp
// Δημιουργήστε ένα νέο Βιβλίο Εργασίας και λάβετε το πρώτο φύλλο εργασίας
Workbook book = new Workbook();
Worksheet sheet = book.Worksheets[0];

// Ρύθμιση παραμέτρων επιλογών εισαγωγής
ImportTableOptions options = new ImportTableOptions { InsertRows = true };

// Εισαγωγή μόνο επιλεγμένων στηλών ("Όνομα" και "Ηλικία")
sheet.Cells.ImportCustomObjects((System.Collections.ICollection)people,
    new string[] { "Name", "Age" }, 
    true, 0, 0, people.Count, true, null, false);

// Αυτόματη προσαρμογή όλων των στηλών στο περιεχόμενό τους
book.Worksheets[0].AutoFitColumns();
```
### Βήμα 4: Αποθήκευση του βιβλίου εργασίας σας
Τέλος, αποθηκεύστε το βιβλίο εργασίας σας σε ένα αρχείο Excel.
```csharp
string dataDir = "path/to/your/directory";
book.Save(dataDir + "ImportedCustomObjects.xlsx");
```
