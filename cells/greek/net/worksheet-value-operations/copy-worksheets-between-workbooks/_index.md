## Εισαγωγή πακέτων
Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στον κώδικά σας. Δείτε πώς μπορείτε να το κάνετε:
```csharp
using System.IO;
using Aspose.Cells;
using System;
```
Αυτή η μοναδική γραμμή σάς παρέχει πρόσβαση σε όλες τις ισχυρές λειτουργίες του Aspose.Cells.
Σε αυτό το σεμινάριο, θα αναλύσουμε την εργασία σε διαχειρίσιμα βήματα. Κάθε βήμα βασίζεται στο προηγούμενο, επομένως θα έχετε ένα ολοκληρωμένο, λειτουργικό απόσπασμα κώδικα στο τέλος.
## Βήμα 1: Ορισμός του καταλόγου εγγράφων
Αρχικά, ας καθορίσουμε τη διαδρομή όπου αποθηκεύονται τα αρχεία του βιβλίου εργασίας μας. Αυτή η διαδρομή θα υποδείξει στο πρόγραμμα πού να βρει το βιβλίο εργασίας προέλευσης και πού να αποθηκεύσει το αντιγραμμένο αρχείο.
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "Your Document Directory";
```
Εδώ, αντικαταστήστε `"Your Document Directory"` με την πραγματική διαδρομή όπου αποθηκεύονται τα αρχεία σας.
## Βήμα 2: Ορίστε τη διαδρομή εισόδου αρχείου
Σε αυτό το βήμα, θα ορίσουμε τη διαδρομή προς το αρχικό βιβλίο εργασίας που περιέχει το φύλλο εργασίας που θέλουμε να αντιγράψουμε. Για επίδειξη, ας υποθέσουμε ότι το αρχείο έχει το όνομα `book1.xls`.
```csharp
string inputPath = dataDir + "book1.xls";
```
Αυτή η γραμμή συνδυάζει `dataDir` με το όνομα του αρχείου, δημιουργώντας μια πλήρη διαδρομή προς `book1.xls`Αυτό είναι το βιβλίο εργασίας που περιέχει το φύλλο που θα αντιγράψουμε.
## Βήμα 3: Ανοίξτε το Βιβλίο εργασίας πηγαίου κώδικα
Τώρα, ας ανοίξουμε το βιβλίο εργασίας πηγαίου κώδικα (`book1.xls`) δημιουργώντας ένα `Workbook` αντικείμενο και περνώντας μέσα `inputPath` ως επιχείρημα.
```csharp
// Δημιουργήστε ένα Βιβλίο Εργασίας.
// Ανοίξτε ένα αρχείο στο πρώτο βιβλίο.
Workbook sourceWorkbook = new Workbook(inputPath);
```
Εδώ, αρχικοποιούμε `sourceWorkbook` για να αναπαραστήσει το βιβλίο εργασίας πηγαίου κώδικα. Αυτό το αντικείμενο μας δίνει πρόσβαση σε όλα τα φύλλα εργασίας μέσα στο αρχείο.
## Βήμα 4: Δημιουργία του βιβλίου εργασίας προορισμού
Σε αυτό το βήμα, θα δημιουργήσουμε ένα νέο βιβλίο εργασίας που θα χρησιμεύσει ως προορισμός για το αντιγραμμένο φύλλο εργασίας μας. Αυτό θα χρησιμεύσει ως κενή πλάκα όπου θα επικολλήσουμε το αντιγραμμένο φύλλο.
```csharp
// Δημιουργήστε ένα άλλο Βιβλίο Εργασίας.
Workbook destinationWorkbook = new Workbook();
```
Μας `destinationWorkbook` είναι κενό από προεπιλογή, και περιέχει μόνο ένα φύλλο εργασίας.
## Βήμα 5: Αντιγραφή του φύλλου εργασίας στο νέο βιβλίο εργασίας
Τώρα έρχεται η ουσία αυτού του σεμιναρίου—η αντιγραφή του φύλλου εργασίας. Θα αντιγράψουμε το πρώτο φύλλο εργασίας από το βιβλίο εργασίας προέλευσης και θα το επικολλήσουμε στην πρώτη υποδοχή φύλλου εργασίας του βιβλίου εργασίας προορισμού.
```csharp
// Αντιγράψτε το πρώτο φύλλο του βιβλίου εργασίας προέλευσης στο βιβλίο εργασίας προορισμού.
destinationWorkbook.Worksheets[0].Copy(sourceWorkbook.Worksheets[0]);
```
Σε αυτόν τον κώδικα:
- `sourceWorkbook.Worksheets[0]` αντιπροσωπεύει το πρώτο φύλλο εργασίας στο βιβλίο εργασίας πηγαίου κώδικα.
- `destinationWorkbook.Worksheets[0]` αναφέρεται στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας προορισμού.
- Ο `.Copy` Η μέθοδος κάνει τη δύσκολη δουλειά, μεταφέροντας απρόσκοπτα το φύλλο εργασίας από το ένα βιβλίο εργασίας στο άλλο.
## Βήμα 6: Αποθήκευση του βιβλίου εργασίας προορισμού
Τέλος, ας αποθηκεύσουμε το βιβλίο εργασίας προορισμού μας. Αυτό θα ολοκληρώσει τη διαδικασία αντιγραφής και θα δημιουργήσει ένα αρχείο εξόδου που περιέχει το αντιγραμμένο φύλλο εργασίας.
```csharp
// Αποθηκεύστε το αρχείο.
destinationWorkbook.Save(dataDir + "CopyWorksheetsBetweenWorkbooks_out.xls");
```
Αντικαθιστώ `"CopyWorksheetsBetweenWorkbooks_out.xls"` με το όνομα αρχείου εξόδου που προτιμάτε. Τώρα, θα έχετε ένα νέο αρχείο στον καθορισμένο κατάλογο με το αντιγραμμένο φύλλο εργασίας.
