## Οδηγός Εφαρμογής (H2)

Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε και να εφαρμόσετε κοινόχρηστους τύπους χρησιμοποιώντας το Aspose.Cells για .NET.

### Βήμα 1: Αρχικοποίηση του Βιβλίου Εργασίας (H3)

Δημιουργήστε μια παρουσία βιβλίου εργασίας που αντιπροσωπεύει το αρχείο Excel σας:

```csharp
// Δημιουργία νέας παρουσίας βιβλίου εργασίας
Workbook workbook = new Workbook();
```

### Βήμα 2: Πρόσβαση και τροποποίηση κελιών φύλλου εργασίας (H3)

Αποκτήστε πρόσβαση στο πρώτο φύλλο εργασίας για να εργαστείτε με τα κελιά του:

```csharp
// Πρόσβαση στο πρώτο φύλλο εργασίας στο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];

// Εισαγωγή δεδομένων δείγματος στη στήλη Α
worksheet.Cells["A1"].PutValue(50);
worksheet.Cells["A2"].PutValue(100);
worksheet.Cells["A3"].PutValue(150);
worksheet.Cells["A4"].PutValue(30);
worksheet.Cells["A5"].PutValue(65);
worksheet.Cells["A6"].PutValue(49);
```

### Βήμα 3: Εφαρμογή του Κοινόχρηστου Τύπου (H3)

Χρήση `SetSharedFormula` για να εφαρμόσετε έναν τύπο σε πολλά κελιά:

```csharp
// Λήψη της συλλογής κελιών από το φύλλο εργασίας
Cells cells = worksheet.Cells;

// Ορισμός κοινόχρηστου τύπου για το εύρος C1:C6
cells["C1"].SetSharedFormula("=A1*2", 6, 1);
```
**Εξήγηση:** Ο `SetSharedFormula` Η μέθοδος δέχεται τρεις παραμέτρους:
- Τύπος ως συμβολοσειρά (π.χ., "=A1*2")
- Αριθμός γραμμών για την εφαρμογή του τύπου
- Αριθμός στηλών για την εφαρμογή του τύπου

### Βήμα 4: Αποθήκευση του βιβλίου εργασίας σας (H3)

Αποθηκεύστε το βιβλίο εργασίας σας με τις αλλαγές:

```csharp
// Ορισμός καταλόγου εξόδου
string outputDir = RunExamples.Get_OutputDirectory();

// Αποθήκευση του τροποποιημένου αρχείου Excel

Console.WriteLine("SettingSharedFormula executed successfully.");
```
