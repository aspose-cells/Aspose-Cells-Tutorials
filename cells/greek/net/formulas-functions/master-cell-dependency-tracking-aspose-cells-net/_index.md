## Οδηγός Εφαρμογής

### Φόρτωση του βιβλίου εργασίας

Φορτώστε το βιβλίο εργασίας σας για να ορίσετε το αρχείο Excel:
```csharp
// Φόρτωση ενός υπάρχοντος βιβλίου εργασίας από μια καθορισμένη διαδρομή
Workbook workbook = new Workbook("Book1.xlsx");
```
#### Επισκόπηση
Αυτό αρχικοποιεί το `Workbook` αντικείμενο, παρέχοντας πρόσβαση σε φύλλα εργασίας και κελιά.

### Πρόσβαση σε κελιά και ανίχνευση εξαρτήσεων
Επιλέξτε το φύλλο εργασίας και το κελί για την ανίχνευση εξαρτήσεων:
```csharp
// Λήψη του πρώτου φύλλου εργασίας στο βιβλίο εργασίας
Worksheet worksheet = workbook.Worksheets[0];

// Πρόσβαση σε συγκεκριμένο κελί
Cell targetCell = worksheet.Cells["B2"];
```
#### Επισκόπηση
Πρόσβαση στο `Cells` συλλογή του καθορισμένου φύλλου εργασίας για τον εντοπισμό του κελιού-στόχου.

### Λήψη εξαρτώμενων ατόμων
Χρησιμοποιήστε το `GetDependents` μέθοδος για την ανάκτηση εξαρτημένων κελιών:
```csharp
// Λήψη όλων των εξαρτώμενων κελιών για το 'B2'
Cell[] dependents = targetCell.GetDependents(true);

foreach (Cell c in dependents)
{
    Console.WriteLine(c.Name); // Εξάγει τα ονόματα των εξαρτώμενων κελιών
}
```
#### Επισκόπηση
`GetDependents(true)` επιστρέφει `Cell` αντικείμενα που επηρεάζονται από αλλαγές στο καθορισμένο κελί.
