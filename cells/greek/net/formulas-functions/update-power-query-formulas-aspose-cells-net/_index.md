### Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα περιηγηθούμε στην ενημέρωση τύπων Power Query χρησιμοποιώντας το Aspose.Cells.

#### Επισκόπηση: Ενημέρωση τύπων Power Query
Η ενημέρωση των τύπων του Power Query μέσω προγραμματισμού βοηθά στην αυτοματοποίηση και τη διασφάλιση της συνέπειας στις συνδέσεις δεδομένων σε όλα τα βιβλία εργασίας του Excel. Δείτε πώς μπορείτε να το πετύχετε με το Aspose.Cells για .NET.

##### Βήμα 1: Φόρτωση του βιβλίου εργασίας

Ξεκινήστε φορτώνοντας το βιβλίο εργασίας που περιέχει τους τύπους του Power Query:

```csharp
using System;
using Aspose.Cells;

namespace Aspose.Cells.Examples.CSharp._Workbook
{
    public class UpdatePowerQueryFormulaItem
    {
        public static void Run()
        {
            string SourceDir = RunExamples.Get_SourceDirectory();
            string outputDir = RunExamples.Get_OutputDirectory();

            // Φορτώστε το βιβλίο εργασίας με τύπους Power Query.
            Workbook workbook = new Workbook(SourceDir + "SamplePowerQueryFormula.xlsx");
```

##### Βήμα 2: Πρόσβαση και ενημέρωση τύπων Power Query

Αποκτήστε πρόσβαση σε κάθε τύπο στη συλλογή DataMashup του βιβλίου εργασίας. Ελέγξτε για συγκεκριμένες συνθήκες ή ονόματα προς ενημέρωση:

```csharp
            // Επαναλάβετε όλους τους τύπους power query.
            DataMashup mashupData = workbook.DataMashup;
            foreach (PowerQueryFormula formula in mashupData.PowerQueryFormulas)
            {
                foreach (PowerQueryFormulaItem item in formula.PowerQueryFormulaItems)
                {
                    if (item.Name == "Source")
                    {
                        // Ενημερώστε τον τύπο ώστε να δείχνει σε μια νέα προέλευση δεδομένων.
                        item.Value = $"Excel.Workbook(File.Contents(\"{SourceDir}SamplePowerQueryFormulaSource.xlsx\"), null, true)";
                    }
                }
            }
```

##### Βήμα 3: Αποθήκευση του ενημερωμένου βιβλίου εργασίας

Μόλις ενημερωθούν οι τύποι, αποθηκεύστε το βιβλίο εργασίας για να διατηρηθούν οι αλλαγές:

```csharp
            // Αποθηκεύστε το βιβλίο εργασίας εξόδου με ενημερωμένους τύπους Power Query.
            workbook.Save(outputDir + "SamplePowerQueryFormula_out.xlsx");
        }
    }
}
Console.WriteLine("UpdatePowerQueryFormulaItem executed successfully.");
```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές αρχείων έχουν καθοριστεί σωστά και είναι προσβάσιμες.
- Βεβαιωθείτε ότι έχετε τα απαραίτητα δικαιώματα για την ανάγνωση/εγγραφή των αρχείων.
- Ελέγξτε για τυχόν σφάλματα στη σύνταξη τύπων, εάν οι ενημερώσεις δεν αντικατοπτρίζουν τα αναμενόμενα.
