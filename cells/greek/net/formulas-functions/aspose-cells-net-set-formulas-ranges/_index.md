## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας εφαρμόσουμε ονομασμένα εύρη και τύπους.

### Δημιουργία ονομασμένων περιοχών

**Επισκόπηση:**
Οι ονομασμένες περιοχές βελτιώνουν την αναγνωσιμότητα και τη συντηρησιμότητα, αναφέροντας κελιά με ουσιαστικά ονόματα αντί για συντεταγμένες.

#### Βήμα 1: Συλλογή φύλλων εργασίας Access

Ανακτήστε τη συλλογή φύλλων εργασίας μέσα στο βιβλίο εργασίας σας:

```csharp
// Πρόσβαση στη συλλογή φύλλων εργασίας
WorksheetCollection worksheets = book.Worksheets;
```

#### Βήμα 2: Προσθήκη εύρους με όνομα

Προσθέστε μια ονομασμένη περιοχή με το όνομα "NewNamedRange" που να αναφέρεται στο κελί A3 στο Φύλλο1.

```csharp
// Προσθήκη νέου εύρους με όνομα
int index = worksheets.Names.Add("NewNamedRange");
Name name = worksheets.Names[index];
name.RefersTo = "+=Sheet1!$A$3";
```

#### Βήμα 3: Ορισμός τύπου χρησιμοποιώντας ονομασμένο εύρος

Αντιστοιχίστε τον τύπο στο κελί A1 χρησιμοποιώντας την ονομασμένη περιοχή.

```csharp
// Αντιστοίχιση τύπου στο κελί A1
worksheets[0].Cells["A1"].Formula = "NewNamedRange";
```

#### Βήμα 4: Εισαγωγή τιμής αναφοράς

Εισαγάγετε την τιμή στην οποία αναφέρεται το ονομαστικό εύρος σας, διασφαλίζοντας ακριβείς υπολογισμούς.

```csharp
// Ορισμός της τιμής για το κελί αναφοράς
worksheets[0].Cells["A3"].PutValue("This is the value of A3");
```

### Υπολογισμός τύπων

Υπολογίστε όλους τους τύπους στο βιβλίο εργασίας:

```csharp
// Υπολογισμός τύπων
book.CalculateFormula();
```

### Αποθήκευση του βιβλίου εργασίας σας

Τέλος, αποθηκεύστε το βιβλίο εργασίας σας με τις αλλαγές.

```csharp
// Αποθήκευση του βιβλίου εργασίας σε αρχείο
book.Save("outputSettingSimpleFormulaWithRange.xlsx");
```
